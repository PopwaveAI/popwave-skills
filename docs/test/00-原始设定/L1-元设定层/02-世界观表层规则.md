# 02-世界观表层规则

> 普通人眼中的世界长什么样？他们知道什么？不知道什么？误解了什么？

---

## 一、战斗力坐标系（1-9星制）

D&D世界通用的战力评级，从最简单的"能不能打"出发。普通人只知道2-4星的范围，5星以上就是传说。

| 星级 | 普通人认知 | 对应战力 |
|:----:|:-----------|:---------|
| 1星 | 职业士兵 | 特种兵级别 |
| 2星 | 精锐冒险者 | 黑铁战甲秦川 |
| 3星 | 矮人老兵铁砧·石炉 | 精英战士 |
| 4星 | 需要一个小队才能对付 | 武士虫 |
| 5星 | 传说级（没见过） | 虫巢暴君 |
| 6-7星 | 神话（不存在于普通人世界） | 纳垢化身 |

## 二、金钱与交易

南方联盟通用金币。帝国装备无法用金币购买——秦川的"货币"是功勋。

土著对帝国装备的误解：
- 黑铁战甲 = "某种失传的上古附魔板甲"
- 秦弩光矢 = "魔法箭"
- 空投莲花舱 = "天降神物"

**普通人不知道的是**：这些"附魔"本质是维级纳米工程，不是魔法。但对读者来说，这不需要解释——让土著说"这是神器"就够了。

## 三、普通人知道的 vs 不知道的

| 他们知道 | 他们不知道 |
|:---------|:-----------|
| 北方虫潮已经打了三年 | 虫族是被纳垢催化的，不是自然繁殖 |
| 南方联盟议会能管大部分城邦 | 议会里有人被奸奇渗透了 |
| 冒险者可以去自由城接任务 | 主神空间的"轮回者"曾在这里活动 |
| 法师塔能施法 | 魔网另一端连着什么他们想都不敢想 |

---

## 四、日常生活的几个画面

- **银盾堡酒馆**：墙上挂着虫壳装饰品，老板按"击杀等级"给老客人打折
- **自由城集市**：有专门收"异界遗物"的二手商——主神空间轮回者死前丢下的装备
- **圣罗兰城领主府**：议会的讨论总是"出兵不出兵"，每次讨论完都没出兵——读者从这里能看出联盟的低效
