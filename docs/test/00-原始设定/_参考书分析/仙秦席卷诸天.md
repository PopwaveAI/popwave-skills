# 参考书分析 — 对标拆解

> 差异化决策备忘录（嵌入PRD尾部）

---

## 分析对象

基于核心创意"仙秦+泰伦虫族+D&D"，筛选3本对标书。

### 1. 《诡异人生模拟器》— 系统流+无限流标杆

**节奏地图**：前3章直接展示金手指全文，同步展开第一场战斗。读者在第3章结束时已经知道"主角有什么挂"。

| 萃取 | 内容 |
|:-----|:------|
| ✅ 借鉴 | 开局3章同步展示金手指+战斗，降低"我要不要继续看"的决策成本 |
| ❌ 偏离 | 不做任务制循环（做任务→拿奖励→下一个任务），太机械 |
| 🆕 空白 | 没有势力扩张线——主角全程独行侠。我们补这个 |

### 2. 《诡秘之主》— 设定深度标杆

**节奏地图**：前100章慢热铺设定，不适合番茄。但序列体系的分层设计是天花板级。

| 萃取 | 内容 |
|:-----|:------|
| ✅ 借鉴 | 22条途径×10级的分层设计，每升一级有新能力+新代价 |
| ❌ 偏离 | 不做固定升级路径——我们做军衔+装备双线，更灵活 |
| 🆕 空白 | 诡秘的序列是"职业路径"。我们做"帝国陆军装备体系"——读者看得懂而且有工业美学 |

### 3. 《海贼法典》— 跨IP混搭标杆

| 萃取 | 内容 |
|:-----|:------|
| ✅ 借鉴 | 战锤+海贼的混搭法——用读者熟悉的IP降低认知成本 |
| ❌ 偏离 | 不做同人——我们套D&D的框架但内容全部原创 |
| 🆕 空白 | 没有"文明差"的设定设计。仙秦vsD&D的文明冲突，没有对标作品做过 |

## 差异化决策三元组

| 类型 | 内容 |
|:-----|:------|
| **借鉴点** | 开局金手指展示方式（诡异人生）、分层升级体系（诡秘）、IP框架套原创内容（海贼法典） |
| **偏离点** | 不做任务制循环、不做固定升级路径、不做同人 |
| **空白创新点** | 仙秦vsD&D的文明冲突、「帝国装备体系」的工业美学、版图扩张的政治逻辑 |
