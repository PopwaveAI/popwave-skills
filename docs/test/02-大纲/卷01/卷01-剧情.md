# 卷01-剧情.md — 银盾堡的陷落与崛起

> 章数：1-30 | 定位：落地·求存·建据点
> 情绪弧线：CH1-5高张力→CH6-15探索适应→CH16-25建立信任→CH26-30首胜

---

## 核心冲突链

```
截胡落地（1-3）→ 银盾堡守卫战（4-8）→ 撤退至自由城（9-12）
    → 银羽佣兵团加入（13-17）→ 第一次反攻（18-22）
    → 据点建立（23-27）→ 虫巢坐标确认（28-30）
```

---

## 三幕展开

### Act 1：落地（1-10）

- **1**: 截胡落地，战场初体验
- **2**: 黑铁战甲+秦弩首秀
- **3**: 帝国战术协议首秀+功勋+30
- **4-5**: 银盾堡掩护平民撤退
- **6-8**: 接触南方联盟斥候，获得情报
- **9-10**: 到达自由城，遇到银羽佣兵团

**核心爽点**：装备从无到有、从生涩到熟练、从独狼到有队友

### Act 2：扎根（11-20）

- **11-13**: 自由城站住脚，功勋换取秦血强化
- **14-15**: 城外遭遇战——首次配合佣兵团作战
- **16-17**: 发现虫巢信号，决定反攻
- **18-20**: 小规模反攻，夺回银盾堡周边阵地

**核心爽点**：武力成长（铜皮→秦血）、势力扩张（有据点）、情报解锁（发现混沌痕迹）

### Act 3：首胜（21-30）

- **21-23**: 银盾堡收复战——秦川指挥帝国兵俑首次参战
- **24-26**: 追击虫巢母体
- **27-28**: 解锁兵俑之心召唤权限
- **29-30**: 第一次虫巢攻坚，在巢穴深处发现纳垢的印记——**真正的敌人浮出水面**

**核心爽点**：首次指挥战斗（从执行者变成指挥官）、解锁核心能力、世界观反转（虫族≠最终敌人）
