# 卷01-剧情.md — 银盾堡的陷落与崛起

> 核心冲突：一个现代大学生在异界战场求生→扎根→建立据点
> 情绪弧线：CH1-5 高张力→CH6-15 探索适应→CH16-25 建立信任→CH26-30 首胜

---

## 什么在驱动角色行动？

秦川的行动驱动：**不是"作者让他做"——是他自己想要。** 他想活着，想证明截胡自己的人没选错人，想看帝国的力量在D&D世界能做到什么程度。

- CH1-10：求生本能——"我不跑就死了"
- CH11-20："这地方我能做点什么不一样的东西"
- CH21-30：主动规划——"把银盾堡变成帝国在D&D世界的第一个据点"
