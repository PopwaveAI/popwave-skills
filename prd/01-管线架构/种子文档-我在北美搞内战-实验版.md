## 压力机制

**系统性斩杀线。** 北美的社会制度本身就是一台吃人机器——主角面对的不是单一倒计时，而是一整张压力矩阵，每个压力源都能独立杀死他，多个压力源交叉时产生困境。

**压力矩阵：**

| 压力源 | 机制 | 杀人方式 | 显性表现 |
|:-------|:-----|:---------|:---------|
| 医疗缺失 | 无医保，药价极高 | 病了不治就死 | 血量斩杀线（HP≤5，每小时-1，ch1-3显性驱动） |
| 帮派勒索 | MS-13/Aryan控制街区，收保护费 | 不交钱=被打死 | 巡逻搜人、暴力清场 |
| 前身欠债 | 催债人持续追索，利滚利 | 还不上=被烧帐篷/打断腿 | 催债人上门（ch3末） |
| 贷款信用卡 | 高利贷陷阱，破产即流浪 | 利滚利永远还不清 | 前身三婚三离破产的根因 |
| 离婚斩杀 | 美国离婚法对男方极度不利 | 财产分割导致一无所有 | 前身三次离婚的背景 |
| 邪教器官 | 巫毒教/地下组织需要人体器官 | 被盯上=被献祭 | 老约翰之死（巫毒献祭痕迹） |
| 强化剂 | 地下流通的非法药物，短期增强长期毁灭 | 依赖=奴隶 | 后续章节浮现 |
| 系统压制 | 警察/社工/收容所，身份追踪 | 被录入系统=失去自由 | 身份暴露风险 |

叙事功能：这张矩阵不是"设定清单"——是剧情发电机群。每章至少有一个压力源在逼近，不同压力源的交叉产生道德困境（比如：要保护身边的人就会暴露自己→帮派盯上→医疗资源更难获取）。血量斩杀线只是压力矩阵在开局的显性表现，随着故事推进，其他压力源逐一浮现，形成层层加码的系统性压迫。

> **注：压力矩阵的构建需要种子搜索法**——调研目标社会的"人吃人"制度维度，而非凭空设计。种子设计阶段应包含对北美社会制度的系统性搜索，识别可用的压力源。这与 creative skill 的生态调研有相通之处，但聚焦点从"角色群生态"扩展到"制度性压迫维度"。

## 主角引擎

韦恩。前世中国刑警。

- **驱动力（三层）**：
  - 表层：活下去，活好——直接应对压力矩阵
  - 中层：保护身边的人——天然与压力机制冲突（保护别人消耗自己资源、暴露自己位置、招来更多压力源），产生道德困境和决策压力
  - 深层：推翻腐朽的北美制度——从"在被压榨中求生"到"向压榨者开战"，这个驱动力前期不显性，随着主角能力成长和身边的人不断被制度碾碎而逐渐浮现
- **缺陷**：身体极弱（大病虚弱，体力仅常人30%，HP=4开局）+ 无社会身份（流浪汉）+ 前身欠债（前身"布鲁斯·韦恩"：童星→娈童丑闻→三婚三离→流浪，催债人持续追索）
- **决策风格**：实用主义+杀伐果断。什么有用就用什么，该出手时不含糊。生饮海鸥血回HP、卖尸体换钱、用演技吓退流浪汉——不是因为"创意"，是因为"能活"。当面决断不犹豫，事后不内耗。

## 金手指

**地下城冒险系统。** 核心创意：系统把美利坚当成魔域地下城，一切以地下城冒险逻辑运行。

**能力（按模块解锁）：**
- L1（ch1-3）：基础面板 + 战斗评估 + 尸体搜刮（触碰尸体获得技能/物品）+ 斩杀线警告
- L1+（ch3-7）：经验值系统 + 商店系统（1金币≈1万刀）+ 被动技能 + 战宠进化 + 羁绊系统
- L2（ch12-23）：技能融合 + 空间背包 + 技能拾取 + 职业选择（巫医）
- L3（ch41+）：庇护所建造 + 史诗任务 + 效果外放 + 圣遗物

**限制：**
- 精神力=技能使用次数上限，耗尽则无法使用技能
- 体力极低时技能效果打折
- 搜刮必须物理接触尸体
- 2级后每次技能使用仅+1经验，升级极慢

**系统错位（核心创意来源）：**
系统把现实"错误翻译"为地下城——流浪汉=哥布林，警察=堕落执法者，催债人=双头食人魔，收尸人=神秘商贩。这个错位本身是金手指：**表演式打倒=真打倒**（用演技吓退哥布林，系统判定为战斗胜利并给经验）。理解错位的荒诞性并善用漏洞，才是真正的生存之道。

**数值风格**：戏剧优先。HP=4触发危机，HP=6脱离危机，每次数字变化都对应剧情转折，不做精确数值模拟。

## 冲突轴

**底层流浪汉 vs 整个美利坚系统。**

四层生态：
- **帮派层**（MS-13 / Aryan Brotherhood）：暴力威胁，控制街区，搜人清场
- **流浪者层**（老约翰 / 收尸人 / 醉汉 / 其他流浪汉）：信息/资源/灰色经济/尸体交易
- **系统层**（警察 / 催债人 / 社工）：规则压制，身份追踪，债务追索
- **地下层**（巫毒献祭 / 裂隙异常 / 超自然实体）：隐藏威胁，系统真相的线索

交叉设计：帮派争地盘 × 流浪者求生 × 系统管控 × 地下势力渗透。每层都能独立产生危机，多层交叉产生复杂困境。

## 目的地

1. [ ] 脱离斩杀线，建立稳定生存（~ch3-4）
2. [ ] 第一次系统模块大解锁——职业选择（~ch20-25）
3. [ ] 与催债人/前身过去正面冲突并解决（~ch4 / ~ch20-25）
4. [ ] 建立势力——庇护所（~ch41+）
5. [ ] 发现裂隙本质——系统为什么把美国当地下城（~ch50-60）
6. [ ] 终局：美国内战爆发，主角站在风暴中心（~ch80+）

目的地是模糊路标，不指定章节。故事在涌现过程中自然到达。允许调整。

种子五要素能完整生成前3章的全部剧情走向。关键变化：压力矩阵比单一斩杀线能驱动更多维度的剧情——ch1同时有医疗缺失和邪教器官两个压力源在作用，ch3同时有医疗缺失和前身欠债两个压力源在交叉。驱动力三层结构让主角不只是"活下去"——中层"保护身边的人"会在后续章节（收留流浪者、建立庇护所）自然涌现，深层"推翻制度"会在更后期浮现。

## 种子规模

- 种子文档：约600字
- v2管线的等价前置文档（creative+world+character+plot）：约31000字
- 压缩比：50:1

信息损失的减少：从3次压缩（设定→大纲→章纲→正文）变为1次（种子→正文）。写作agent拿到的是种子的原始驱动力，不是第三手摘要。
