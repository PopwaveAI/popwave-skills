## 一、文档概要

本文档是写作管线的架构设计说明，涵盖了从 creative → world → plot → chapter-design → prose-render → qa 的完整管线。

## 二、核心内容结构

### 1. 管线总览
- **管线路径**：creative → world → plot → chapter-design → prose-render → qa
- **有对标书路径**：download → deconstruct Lv1 → creative（加载拆书）→ world → plot → ...

### 2. Skill 拆分说明

| Skill | 职责 | 上游 | 下游 |
|-------|------|------|------|
| pop-novel-creative | 创意打磨、跨域素材、方向碰撞、样品试读 | deconstructor | world |
| pop-novel-world | L1设定、角色卡、数值体系、起点快照、宪法 | creative | plot |
| pop-novel-plot | 全书架构、卷设计、幕纲编排、Canvas矩阵 | world | chapter-design |
| pop-novel-chapter-design | 事件链设计、设计包产出 | plot | prose-render |
| pop-novel-prose-render | 正文渲染、文风DNA应用 | chapter-design | qa |
| pop-novel-qa | 门禁检查、质量审计 | prose-render | — |
| pop-novel-deconstructor | 拆书分析（Lv1-Lv4） | — | creative |
| pop-novel-character-schema | 角色分级标准（Lv1-Lv4） | — | world |

### 3. 核心架构决策

- **创意与工程分离**：creative 做"什么叫好"，world 做"能不能做"
- **样品试读**：story-engine 的描述验证不可靠，必须用户看到文字确认方向
- **宪法约束**：story-engine 从描述型改为约束型（immutables/flexibles/forbidden）
- **宪法审计**：plot Step 0 自动校验 world 产出 vs story-engine
- **character-schema 独立**：标准（定义角色卡长什么样）≠ 执行（具体角色卡填写）

### 4. 管线闸门

| 闸门 | 位置 | 条件 |
|------|------|------|
| 样品确认 | creative → world | 用户确认样品试读 |
| 宪法审计 | world → plot | plot Step 0 发现冲突 → 退回 world |
| 里程碑确认 | plot Step 3 | 用户确认里程碑 |
| 场景卡确认 | plot Step 10 | 用户确认场景卡 |

### 5. 典型路径速查

| 场景 | 管线路径 | 关键闸门 |
|------|---------|---------|
| 新书启动 | creative → world → plot → design → render → qa | 样品确认/里程碑/场景卡 |
| 有对标书新书 | download → deconstruct Lv1 → creative → world → plot → ... | Lv1 拆完后可选继续深拆或直接开书 |
| 拆解参考书 | download → deconstruct（Phase S 或 Phase 0-4）→ pop-dna | TXT 质量验证 |
| 已有项目续写 | bookstrap REV → plot → design → render → qa | 精读闸门 |
| 正文修改（骨架级） | design → render → qa | — |
| 正文修改（渲染级） | render → qa | — |
| 文风分析→应用 | pop-dna → render | style_executed 验证 |

### 6. 跨Skill共享文件清单

核心共享文件包括（S=静态/D=动态）：

- **story-engine.yaml** (S) — creative 产出，creative → world → plot
- **样品试读.md** (S) — creative 产出，world + 用户
- **L1-01~06.md** (S) — world 产出，plot
- **状态/角色/{角色}-角色卡.md** (D) — world 初版 → plot 卷间回写
- **数值体系×4** (S) — world 产出，plot
- **世界宪法.md** (S) — world 产出，plot Step 0 审计
- **设计/卷/volume-XX.md** (S) — plot 产出，chapter-design
- **设计/幕/act-XX.yaml** (D) — plot 产出，chapter-design + qa
- **chXXX-设计包.md** (D) — chapter-design 产出，prose-render
- **正文/chXXX.md** (D) — prose-render 产出，qa + html-renderer
- **entity-snapshot.yaml** (D) — chapter-design，跨章追踪
