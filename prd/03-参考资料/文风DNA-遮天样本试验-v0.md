# 文风 DNA 产出试验：《遮天》样本 v0

## 0. 这次试验先修正的问题

不能把早期章节、特殊场景、单个角色语气、作者口癖直接上升为全书通用 DNA。

本试验将 DNA 拆成四层：

- 全书稳定骨架：跨阶段、跨场景都反复存在的组织方式。
- 阶段性机制：只在前期、中期、后期、终局显著的商业体验。
- 场景卡机制：战斗、遗迹、对峙、过渡、章末钩子等各自的写法。
- 角色/口癖隔离：区分主角风格、丑角风格、旁观者反应、作者高频表达。

## 2. 先不要抽成通用公式的内容

以下内容只能作为场景或角色机制，不能直接写进全书通用 DNA：

- 第 001 章的航天证据链：这是开篇专用，不是每章都有的叙事方式。
- 第 100 章的同学受辱打脸：这是情绪补偿场景，不是全书主循环。
- 第 156 章的连续肉身打击：这是近身反杀场景，不是所有战斗公式。
- 庞博/黑皇式插科打诨：这是调节气氛的角色功能，不代表全书说话风格。
- 旁观者的震惊台词：这是外部标尺，不能当成主角语气。
- 高频感叹号、四字词、"震惊/骇然/变色"：需要标注为表达习惯，不等于结构 DNA。

## 3. 全书稳定骨架候选

这些机制跨多个阶段出现，可以进入全书稳定 DNA，但仍需后续扩大采样验证。

### 3.1 位阶差驱动

《遮天》的冲突常常不是单纯的敌我冲突，而是位阶差冲突：

- 低境界对高境界。
- 散修对圣地/世家。
- 当代人对太古/大帝遗留。
- 人间修士对禁区/仙路/时间。

生成约束：

- 每个核心冲突必须明确高位存在或高位秩序。
- 主角行动的意义来自"在不被允许的位置上强行进入牌桌"。
- 压迫感优先来自位阶差，而不是单纯数量差。

### 3.2 外部标尺

《遮天》经常用旁观者、传闻、历史、距离、天地异象来给力量定级。

生成约束：

- 高强度事件不能只写主角感受。
- 必须至少使用一种外部标尺：旁观者震动、历史比较、天地变化、远距离影响、古老记录。
- 标尺的作用是让读者知道"这件事超过常规等级"。

### 3.3 压迫后的反向越界

爽点不是普通胜利，而是主角在被压低位置后反向越界：

- 被轻视后反击。
- 被追杀后反杀。
- 被制度排斥后强入。
- 被大势碾压后找到缝隙。

生成约束：

- 爆点前必须先建立限制。
- 爆点不是"更用力"，而是打穿原本的等级判断。
- 爆点后需要有外部确认：敌人失态、旁观者震惊、局势变化、名声扩散。

### 3.4 神话尺度递进

尺度常从具体物象递进到神话秩序：

- 物：棺、鼎、药、源、兵器。
- 地：废墟、禁地、古路、星域。
- 人：圣子、大能、祖王、大帝。
- 秩序：道、劫、禁区、仙路、时间。

生成约束：

- 不要一开始就抽象宏大。
- 先给可触摸物象，再抬升到历史/神话/秩序。
- 读者的认知要经历"看见物 → 知道来历 → 意识到位阶"。

## 4. 阶段性机制候选

### 4.1 前期：陌生世界与资源争夺

核心体验：

- 低位进入高危世界。
- 每个资源都有具体用途。
- 灵药、源、法器、秘术等收益能迅速改变处境。

不要误判：

- 前期的"寻药/开苦海/同学重逢"不是全书通用结构。
- 它们属于低位成长阶段的商业机制。

### 4.2 中期：势力博弈与名声扩散

核心体验：

- 主角不再只是求生，而是利用规则反制势力。
- 布局、借刀、引乱、公开挑战开始增多。
- 名声成为战力的一部分。

### 4.3 后期：历史债务与大势对抗

核心体验：

- 冲突对象从具体敌人变成历史遗留、禁区秩序、帝级因果。
- 战斗更常用天地异象、道则、古史比较来定级。
- 个体选择与时代命运绑定。

### 4.4 终局：回环与兑现

核心体验：

- 早期意象回收。
- 长线承诺兑现。
- 时间跨度极大，人物与旧物承载情感重量。

## 5. 场景卡机制草案

### 5.1 开篇/世界颠覆

适用：

- 第一章或重大世界观揭露。

场景公式：

1. 现实/常识基底。
2. 可信证据链。
3. 第一处异常。
4. 权威或群体确认。
5. 神话物象降临。
6. 章末悬置。

禁用：

- 直接解释全部神话设定。
- 只写主角内心震惊。

### 5.2 遗迹/禁地探索

适用：

- 荒古禁地、废墟、古殿、未知地带。

场景公式：

1. 异常环境先行。
2. 生物/地貌反常。
3. 资源或危险露头。
4. 角色判断收益与风险。
5. 深处更大变化牵引下一场。

### 5.3 打脸/羞辱反击

适用：

- 敌人居高临下、旧怨重逢、弱者被辱。

场景公式：

1. 弱势方受辱。
2. 敌人语言压迫。
3. 主角克制或冷处理。
4. 突然出手。
5. 物理羞辱复刻敌人原话或姿态。
6. 敌人失态。

注意：

- 这是情绪补偿场景，不是所有冲突都该这么写。

### 5.4 越级反杀/生死战

适用：

- 主角被强者追杀、境界处于低位。

场景公式：

1. 明确位阶差。
2. 主角承受压制。
3. 破局变量出现：法器、地形、体质、敌人轻视、规则漏洞。
4. 短句动作链爆发。
5. 胜利后快速收束。
6. 后果追来：更高势力震怒或追杀。

### 5.5 高位战/圣级战

适用：

- 远超主角当前层级的战斗。

场景公式：

1. 先写气势与位阶。
2. 用外部标尺说明不可接近。
3. 动作不必细拆，重点写天地、道则、旁观者承受。
4. 胜负可以极快分出。
5. 结尾留下更大压迫或下一个高位目标。

### 5.6 章末钩子

常见类型：

- 继续深入型：更深处有变化。
- 后果追索型：杀完后更高势力将至。
- 大势引乱型：一个计划将撬动全局。
- 回环兑现型：早期意象重新出现。

生成约束：

- 章末钩子必须改变读者对下一章的期待。
- 不只是"他继续往前走"，而是"往前走会触发更高层级的事"。

## 6. 角色/口癖隔离规则

### 6.1 主角叶凡

候选特征：

- 低位时冷静、能忍、能算。
- 出手时果断，常伴随物理压制。
- 对敌时可以狠，对友时有情义，但抒情不长。

注意：

- 叶凡的狠话不能扩展成所有角色的说话方式。

### 6.2 调节角色

庞博、黑皇等角色承担：

- 插科打诨。
- 降低连续高压造成的疲劳。
- 把宏大事件拉回口语和损友关系。

注意：

- 这类台词不可当成全书通用对话 DNA。

### 6.3 旁观者/群众

功能：

- 定级。
- 震惊。
- 传播名声。
- 放大事件后果。

注意：

- 旁观者不是人物深描，而是商业标尺。

### 6.4 作者高频表达

需要单独标注为"表达习惯"，而不是剧情结构：

- 惊骇、震动、变色、不可思议。
- 感叹号密集。
- 四字结构和古意词。
- "像是/仿佛/宛若"类比。

## 7. 第三层 DNA 的遮天式候选定义

第三层不是"像辰东"，而是复现《遮天》的商业体验：

- 读者持续感到世界位阶比主角高。
- 主角不断从低位撬动高位秩序。
- 每次爆点都要改变外部世界对主角的定级。
- 资源、名声、境界、古史、禁区都能转化为期待。
- 章末不是停顿，而是把下一章推向更高位阶。

## 8. 用于跨题材强嫁接的翻译规则

如果把《遮天》嫁接到现代异端审判庭，不迁移词汇，迁移结构：

| 遮天结构 | 现代异端审判庭翻译 |
|---|---|
| 圣地/世家 | 教廷、审判庭、军方、财阀、秘密机构 |
| 古族/禁区 | 古老异端谱系、不可公开的神性污染区 |
| 大帝遗留 | 历史上某位圣徒/审判官/异端神明的遗物 |
| 越级反杀 | 人类调查员以低权限撬动神性实体 |
| 天劫/道则 | 动用权柄后的代价、注视、污染反噬 |
| 旁观者震惊 | 专业机构重新给主角定级 |
| 九龙拉棺式物象 | 现代仪器捕捉到无法归类的神话证据 |

强嫁接检查：

- 是否保留位阶差？
- 是否保留外部标尺？
- 是否保留爆点后的重新定级？
- 是否避免直接搬运玄幻名词？
