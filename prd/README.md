## 命名规范

- `01-` 前缀表示目录内排序，**不代表优先级**
- 文件名用中英文混排，保留原始 PRD 标题
- 问题诊断类文件按日期命名，便于时间线追溯

## 新增文档规则

1. 新 PRD 放在对应分类目录
2. 首屏包含元数据块（版本/日期/状态/分类/关联）
3. 更新本 README 的分类导航表
