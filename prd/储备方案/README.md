## 入库标准

- **已有 PRD 方案文档**（不是口头想法）
- **方案已评审通过**，但因依赖上下游稳定性、优先级排期等原因**尚未开始实现**
- **方案明确描述了"做什么"和"为什么做"**，接手即可执行

- 依赖项稳定 → 进入迭代计划
- 或用户明确要求当前推进此方向
