## 前提大事件（故事开始前已发生/正在进行）

### 1. 混乱时空爆发（前提核心事件）
一股无法被诸神抵抗的可怕力量在混乱时空内爆发，直接导致：诸神暂时失去神力→被迫以"圣者"（avatar）形态降临凡间→神术体系开始不稳定→凡人首次获得击杀神灵的机会。这是一个改变世界底层规则的事件。

### 2. 万神殿崩塌
万神殿内三分之一神灵陨落——包括魔法女神、秩序之主、黑夜女士、阴影主宰、恐惧魔王等强大神灵。神国从天界坠落化为凡间的特殊位面——"诸神的国度"。死后留下的空位成为所有凡人觊觎的目标。

### 3. 魔法女神陨落
魔法女神的死亡直接影响了全世界的法术系统：学习法术的智力门槛从18提升到20。魔网（魔法能量网络）可能开始不稳定。

### 4. 《诸神之战》游戏世界与真实世界重合
索伦前世的VR游戏《诸神之战》——100%真实度的虚拟世界——其设定（职业/面板/地图/神系）与这个"真实世界"完全一致。核心谜题：是"游戏预测了世界"还是"世界就是游戏"？（原文未显式说明，推断留作全书悬疑）

## 核心矛盾（→ plot M1 线）

```
core_conflict: "旧神陨落、新神未立——凡人、恶魔、圣者谁将主宰这个世界？"
```

这个世界当前面临的根本性对抗：

1. **凡人 vs 神灵**：神灵以圣者形态行走凡间，失去了绝对权威和保护伞。凡人第一次拥有了"杀死神灵"的可能性和动机——权力真空等待被填补。

2. **深渊 vs 凡间**：趁诸神失势，深渊恶魔大规模入侵。蛇魔/巫妖/食人魔等黑暗生物不再被神灵压制，凡间变成了狩猎场。

3. **凡人 vs 凡人**：权力真空意味着所有野心家都在争抢。阿伦黛尔的女王继承权之争（ch150）只是缩影——全世界都在经历权力洗牌。

4. **秩序 vs 混沌**：DND经典对抗。琥珀城还有"三位守序神灵"坐镇——但能守多久？随着动荡之年的深入，守序力量在节节败退。

5. **游戏 vs 现实**（核心谜题）：索伦的穿越和面板的出现意味着什么？诸神之战的游戏世界和这个"真实世界"的关系是什么？前150章中尚未解答——这是全书的终极叙事悬念。
