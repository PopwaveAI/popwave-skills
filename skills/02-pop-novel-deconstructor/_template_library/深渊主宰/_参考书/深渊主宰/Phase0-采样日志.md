## 【初步故事骨架】

### 整体结构

```
卷1(ch1-20): 琥珀城篇 —— 从濒死苏醒 → 击杀蛇魔 → 离开城市
  幕1(ch1-10): 贫民区觉醒 —— 接受现实，开始杀人
  幕2(ch11-17): 藏宝洞 + 蛇魔 —— 第一个地下城，第一次Boss战
  幕3(ch18-20): 离开前夜 —— 神之子事件，剥皮事件，离别

卷2(ch21-100): 流浪与立足篇 —— 加入商队 → 白马城 → 野外系列 → 落叶城
  幕1(ch21-40): 商队旅程 —— 离开琥珀城，进入荒野
  幕2(ch41-70): 白马城区域 —— 城市立足，法术学习
  幕3(ch71-100): 远征与落叶 —— 食人魔战役，北上落叶城

卷3(ch101-150): 冰雪与神性篇 —— 雪原 → 冰封谷 → 阿伦黛尔
  幕1(ch101-120): 雪原旅程 —— 背妹妹穿越雪原
  幕2(ch121-140): 冰封谷 —— 定制武器，猎杀者，神性触碰
  幕3(ch141-150): 冰雪国度 —— 长公主，神灵之战，女王继承者
```

### 核心冲突

**M1（世界级）**：动荡之年——诸神陨落、圣者降临、深渊入侵。凡人有机会屠神，世界秩序全面崩塌。
  - ch1-20: M1仅暗示（ch3提及、ch14单体蛇魔出现）——M1裸露度约20%
  - ch21-100: M1逐步推进——更多恶魔/怪物出现，世界设定密度高(ch83 世界设定章)
  - ch101-150: M1全面爆发——神灵之战(ch145)、神术不稳定(ch127)、冰霜翼龙(ch147)

**M2（个人级）**：索伦从凡人到半神的成长——活下来→变强→保护妹妹→触碰神性
