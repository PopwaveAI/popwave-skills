# Phase 0.4：主角设计

> 该文件同时作为执行指令和参考指南，无需加载外部 ref 文件。

## 前置条件
参考书分析完成。story-engine 方向已锁定。

**核心定位**：主角不是"有什么能力的人"——是"站在碰撞点上、被迫在两套（或多套）不相容的规则之间做选择的人"。**唯一性是必选的**——每一个主角都必须回答"为什么他是主角，而不是摄像头正好对着他"。

---

## 执行步骤

### 1. 从碰撞点反推主角

```
不是在"创造"主角——是从碰撞点推导出主角的必然形状。

碰撞点：修真 × 克苏鲁 — 飞升 = 变成旧日支配者的容器
  → 主角必须是一个即将接近飞升门槛的人
  → 她必须知道飞升的真相（否则没有冲突）
  → 她不能是一个局外人——她必须是修士，否则不痛
  → 推导结果：一个资质顶尖的修士，在飞升前发现真相

碰撞点：修真 × 公务员 — 编制体系 vs 逆天之道
  → 主角必须同时深处两套体系
  → 他的能力在两套体系里的评价必须相反
  → 他不是"不知道选哪个"——是"选哪个都对也都错"
```

### 2. 主角设计四维

| 维度 | 问什么 | 示例（修真×克苏鲁） |
|:-----|:------|:------------------|
| **身份** | 他在碰撞点上的位置是什么？这个位置为什么让他无处可逃？ | 她是修真界最有希望飞升的修士。十年前师父飞升——"成功"飞升。但她看见了。 |
| **驱力** | 他要什么？他怕什么？两套规则各给了他什么不能放弃的东西？ | 要：让修真这条路有意义。怕：飞升的真相=她毕生的信仰是谎言。 |
| **缺陷** | 什么让他注定做错选择？这个缺陷和碰撞点什么关系？ | 她太聪明了——聪明到无法假装不知道。如果她笨一点，可以像其他人一样心怀希望地飞升。 |
| **唯一性** | 他凭什么脱颖而出？换一个人站在这儿行不行？<br>★ 必填。没有唯一性的主角 = 摄像头 | 她能在飞升前看到飞升的真相——别的修士要到最后一刻才知道。代价：她和所有飞升者共享一个秘密。"知道"本身是她最大的能力也是最大的诅咒。 |

### 3. 唯一性展开 — 为什么他是主角 ★ 必填

> **不选"有没有金手指"——选"唯一性以什么形式存在"。** 每一个主角都有唯一性，只是载体不同。

#### 3a. 唯一性类型判断

| 题材 | 唯一性的典型载体 | 示例 |
|:-----|:---------------|:------|
| 修真/玄幻/系统 | 特殊体质/血脉/金手指/传承 | 荒古圣体 — 肉身无敌但被天地压制 |
| 都市/历史/权谋 | 认知/信息/人际关系 | 《庆余年》范闲 — 一个现代人的认知活在古代 |
| 悬疑/推理 | 智力/观察力/过往经历 | 福尔摩斯 — 观察力 ≠ 超能力，但仍然是唯一性 |
| 末世/生存 | 体质/特殊知识/位置 | 《吞噬星空》罗峰 — 能在废土里感知灵石 |
| 恋爱/日常 | 性格/缺陷/经历形成的独特应对方式 | 一个从不相信爱情的人被迫相亲 × 200 次 |
| 群像/多主角 | 每个主角的某种执念/缺陷/立场 | 《冰与火之歌》提利昂 — 智慧 + 被家族放逐的视角 |

**核心判断**：删掉这个特性，主角还是不是同一个人？如果不是 → 这就是唯一性。

#### 3b. 唯一性质量检验（四级）

无论载体是什么，所有唯一性都要经过这四级检验。**必走，不可跳过。**

| 级别 | 问什么 | 不合格示例 |
|:-----|:------|:---------|
| L1 **核心差异** | 一句话——他跟同世界其他人最大的不同是什么？ | "他很努力" — 所有人都可以是"努力"的 |
| L2 **约束条件** | 这个差异的代价/限制是什么？≥3 条。**没有约束的唯一性 = 作弊器** | 系统签到流 — 没有任何代价 |
| L3 **不可替代性** | 如果不给这个主角删掉这个特性换一个正常人，故事还成立吗？ | 成立 → 唯一性是假的，他只是镜头对着的人 |
| L4 **叙事驱动** | 这个唯一性是否在驱动剧情？是否把主角逼到两难处境？ | 唯一性只加数值不驱动剧情 → 不是引擎，是补丁 |

**四级走完的判断**：
- 全部通过 → 这个主角是真的不得不由他做。进产出。
- L1 不通过 → 唯一性定义太模糊。回到四维重写。
- L2 不通过 → 唯一性没有代价。加约束。
- L3 不通过 → 换一个人也行。唯一性是假的。回到碰撞点重新推导主角。

#### 3c. 无"金手指"题材示例

> 不是没有唯一性——是没有超自然金手指。唯一性以其他形式存在。

```
都市职场：
  L1 核心差异：她是全公司唯一一个敢在全员大会上反驳 CEO 的人
  L2 约束：但她是三个孩子的单亲妈妈，跳槽自由度为零
  L3 不可替代：换一个不敢说话的人 → 整个故事不成立。她必须在"保护饭碗"和"说真话"之间选
  L4 叙事驱动：每一次她说真话都在赌一次被开除

悬疑推理：
  L1 核心差异：他是唯一一个相信那起"已结案"的案子是冤案的人——因为被害者的女儿是他的学生
  L2 约束：他没有警徽、没有资源、没有时间（凶手还在杀人）
  L3 不可替代：换一个不相干的私家侦探 → 读者不痛
  L4 叙事驱动：他每靠近真相一步，就在离"合法公民"的身份更远一步
```
