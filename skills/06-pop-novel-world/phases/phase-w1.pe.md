# Phase W1：拆书融合

> 该文件同时作为执行指令和参考指南。
> 
> v1.1.0: 从 pop-novel-creative Phase 0.7 移入。有对标书时强制执行。

## 前置条件
- Phase W0 跨域素材聚合已完成
- deconstructor 已对锚点书产出至少 T1-T7 + 卷1起点快照 + 卷1终点快照
- creative Phase 0.3 的"观察清单+差异化"已产出（知道重点看什么）

## 强制规则

```
有对标书 → Phase W1 强制执行（不看 = 假对标，退回 creative）
无对标书 → 跳过。标注"无锚点书分析，跳过拆书融合"。
```

## 目的
将 deconstructor 对锚点书的产出物进行结构化融合，提取可复用的规则注入后续设定。**核心动作是提取参考值**——标注"可以是这个幅度""这个结构的原理在此"等（而非照搬）。

## 执行步骤

### 1. 加载 creative 的观察清单
从 `_参考书分析/观察清单+差异化.md` 读取 creative Phase 0.3 产出的验证需求和关键指标。这些决定了"重点看什么"。

### 2. 定位 deconstructor 产出
读取 `_参考书/` 目录下对应锚点书的文件，确认存在：
- T1-力量体系规则手册
- T2-世界观展开
- T4-剧情全貌（含卷1起点/终点快照）
- T5-人物阵营
- T6-叙事设计技法
- T7-文风DNA指纹

### 3. 逐 T 提取参考值

- **T1 力量体系**: 阶层差/通胀幅度/瓶颈设计 → 作为后续力量体系的参考幅度
- **T1 升级年表**: 提取前 50 章的升级节奏密度 → "原著平均几章一次成长反馈？前期密度和后期密度差多少？" → 标注为「节奏参考，非本项目排期」
- **T2 世界观展开**: 信息释放节奏/世界观展开速度
- **T4 剧情全貌**: 卷1起点/终点快照作为参考锚点 → 帮助理解"卷边界怎么设"
- **T5 人物阵营**: 角色密度/阵营分裂方式
- **T6 叙事设计**: 节奏模板/悬念链路
- **T7 文风DNA**: 叙事哲学原则

### 4. 产出融合摘要

存入 `_参考书分析/拆书融合摘要.md`：

```markdown
# 拆书融合摘要
> 来源：《{书名}》 | 日期：YYYY-MM-DD

## 力量体系参考
- 阶层差：[参考幅度]
- 突破节奏：[参考值]

## 世界观展开参考
- 释放节奏：[参考值]

## 卷边界参考
- 卷1起点：[关键点]
- 卷1终点：[关键点]

## 叙事设计参考
- 高潮间隔：[参考值]
- 悬念链路：[模式]

## 角色密度参考
- 核心角色：[≤N人]
- 配角：[≤N人]
```

### 5. M1/M2 转译

**为什么需要**：deconstructor T4 的 M1/M2 是对锚点书的分析——同名但不同源，直接复用会导致因果错位。

**执行**：
- 继承：节奏规律（密度/露出方式/交汇频率）
- 不继承：具体事件
- 自检：如果转译后的 M1 事件和锚点书有 3 个以上相同 → 退回

### 6. 红线
❌ 有对标书但拆书成果不读取 → 退回 creative
