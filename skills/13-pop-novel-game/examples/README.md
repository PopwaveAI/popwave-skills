# 13-pop-novel-game · 使用示例

> 这个目录存放**AI文游产出示例**

## 如何运行

1. 调用本 skill 时，告诉 pop 你有哪些世界观/人设/剧情资料
2. 提供资料文件或说明资料存放位置
3. 指定想要什么类型的玩法（恋爱/冒险/探索等）
4. 产出物会出现在 `<小说名>-AI文游/` 目录下

## 示例产出结构

```
<小说名>-AI文游/
├── 文游指令.md              ← 核心产物！丢给AI模型即可开玩
├── 游戏世界.html            ← 交互式HTML游玩界面（双击即开）
├── 世界观设定集.md           ← 结构化世界摘要
└── 角色档案.md              ← 完整角色档案
```

## 使用流程示意

```
老板：把这个仙侠小说的世界观做成AI文游
pop：好的，请提供资料

[用户提供资料]
  ↓
Phase A → 解析世界观/角色/剧情
Phase B → 设计穿越者模式 + 修炼冒险玩法
Phase C → 生成指令文件 + 交互式HTML
  ↓

产出：
  ├─ 文游指令.md    → 丢给Claude/DeepSeek即可开玩
  └─ 游戏世界.html  → 双击即可进入仙侠世界
```
