# 轻松调剂场景模板

> 适用于：节奏切换、角色魅力展示、暗线埋设
> 核心原则：**轻松不等于水字数——轻松之下必须有东西在推进**

---

## 段落结构

```
┌─ 段1 · 切入（1句） ───────────────────────────────┐
│  功能：告诉读者"这个场景的画风变了"                   │
│  内容：一个不同于当前基调的画面/动作/对话             │
│  示例：上一章是战斗，这一章从"食堂阿姨多给了他一勺肉"开始 │
└────────────────────────────────────────────────────┘
                      ↓
┌─ 段2 · 展开（2-4段） ──────────────────────────────┐
│  功能：节奏放缓，但信息不空                          │
│  内容（至少选1）：                                   │
│    ▶ 角色魅力展现 —— 让读者更喜欢这个角色           │
│    ▶ 关系升温/降温 —— 角色之间的关系发生变化        │
│    ▶ 暗线埋设 —— 看似日常的场景中放入一个伏笔        │
│    ▶ 世界观侧写 —— 通过日常场景展示世界的另一面      │
│  写法：轻松的语气，但结构不松弛                      │
│  节奏：每段至少有一个推进（信息/关系/伏笔）          │
└────────────────────────────────────────────────────┘
                      ↓
┌─ 段3 · 落地（1段） ───────────────────────────────┐
│  功能：让轻松不变成"没用"                            │
│  内容：轻松场景产生的一个小后果                      │
│  示例：不是场景播完了就切——                        │
│        是"他多吃了那口肉，后来跑路时多撑了半天。"     │
└────────────────────────────────────────────────────┘
```

---

## 关键约束

- **不超过500字**——轻松场景不需要长，长了就是在水
- **至少有1个"可追溯"的内容**——读完全书回头看，这个场景里至少有一句话在后续被验证
- **不纯功能性提醒**——拒绝"啊，对了，明天要交报告"这种为推进而推进
- **节奏切换不等于信息暂停**——轻松场景的信息密度可以降低，但不能归零

---

## 节奏标记

```
切入：快 → 展开：慢（但每段有推进） → 落地：中速
总字数：300-500字
```
