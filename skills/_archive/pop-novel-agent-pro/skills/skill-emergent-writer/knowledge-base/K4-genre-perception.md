# K4 · 题材感知

> 按章节场景类型选择性注入。每次只注入当前类型对应的内容。
> 选择依据：事实骨架头部的场景权重 → 取最高权重的场景类型。

---

## 【战斗/对抗】

- 读者通过主角的视线看世界——描写主角眼中的对手，不直接描写对手
- 战斗的本质是信息差博弈，不是数值比拼
- 不写招式名称和动作细节，写决策后果和损伤结果
- 先写"结果已经发生了"，再写"过程是怎样的"（结果前置）

## 【对话/博弈】

- 每句对话必须推进至少一个维度（信息/关系/情绪）
- 角色不说出口的话比说出口的更重要
- 连续3段纯对话后必须有断裂式补白（动作/环境/内心）
- 谈判场景中，亮底牌的节奏控制——什么时候给多少信息

## 【悬疑/揭示】

- 让读者自己拼出真相比直接告诉真相好十倍
- 信息释放的节奏：先给现象→再给线索→最后给解释
- 读者知道得比角色少 = 好奇；读者和角色同步发现 = 参与感

## 【过渡/铺垫】

- 过渡不是休息，是带着读者往前走
- 每段过渡必须推进至少一个维度（位置/信息/情绪/关系）
- 过渡场景是埋伏笔的最佳位置——读者不会太警觉

---

## 【通用·否定句变体识别】

以下 3 种否定式结构在正文中禁止出现。如果要写对比，用肯定句直接写结果。

### ❌ 变体1：三段否定（不是A不是B是一种C）
```
❌ 不是痛，不是冷，是一种从骨头里渗出来的空。
✅ 骨头里的空渗出来了，像有什么东西在体内醒来。
```

### ❌ 变体2：否定式铺垫（不是X是Y的缩排）
```
❌ 不是恐惧时的那种狂跳——是一种很平静的咚。
✅ 心跳声传来，咚。咚。咚。——平静得不像他自己的心跳。
```

### ❌ 变体3：否定式开头（以"不是"引段）
```
❌ 不是被推开的。不是被砸开的。黑暗从门的另一侧长进来。
✅ 黑暗从门的另一侧长进来，门板从中间裂开。
```

### ✅ 正面写法对照

否定式绕过方式：
1. 先说结果，再描述状态——跳过中间的否定对比
2. 用一个感官动词替换否定结构（"看见/听见/感觉到"强于"不是X而是Y"）
3. 用动作替代判断——"他举起印" 强于 "他不是在恐惧"

## 【通用·角色台词差异化】

每个角色的台词必须有可辨别的节奏特征：

| 角色类型 | 台词特征 |
|:---------|:---------|
| 沉默型主角 | 话少，每句都推进剧情或暴露信息。不闲聊、不评价。 |
| 话多型队友 | 有标志性口癖/句式/话题偏好。但每多一次死亡就少一段功能——用台词长度暗示消耗。 |
| 分析型角色 | 每句都包含观察结论。不说"这很奇怪"——说"天花板上的灰尘分布不均，说明有人最近上去过"。 |
| BOSS/反派 | 有自己的说话节奏（慢/快/停顿位置）。每次出场先说同一句标志性开场白。 |


## 【通用·文化锚定渲染约束】

当作品的核心能力/装备有明确的文化或传说来源时，正文中至少体现以下数量的文化源特征：

- 3 个以上的视觉/行为特征可追溯至文化原型
- 至少 1 次在战斗中自然提及文化源流（通过角色认知而非作者旁白）
- 能力进化方向不应偏离文化原型的行为范式

例：钟馗印
- 文化原型特征：怒目圆睁 / 嫉恶如仇 / 捉鬼吃鬼 / 怕狗 / 酒后真性情
- 正文应体现：印在战斗时"发怒"（烫到握不住）/ 印主动"吃掉"对手而不是毁灭 / 印对某些东西视而不见而对另一些东西反应剧烈
