# Phase 1：L1 设定层推演

**目标**：从创意层落到元设定层，产出六份结构化设定文档。

**条件**：金手指设计完成，跨域素材聚合完成。

## 执行步骤

依次产出六个文件：

| # | 文件 | 覆盖内容 |
|:-:|:-----|:---------|
| 01 | `00-原始设定/L1-元设定层/01-世界观底层逻辑.md` | 世界基本法则、时空结构、力量终极来源 |
| 02 | `00-原始设定/L1-元设定层/02-世界观表层规则.md` | 社会结构、力量体系公开规则、日常生活、文化禁忌 |
| 03 | `00-原始设定/L1-元设定层/03-种族.md` | 主要智慧种族、标志性怪物、传说生物 |
| 04 | `00-原始设定/L1-元设定层/04-金手指.md` | 如在 Phase 0.4 已完成则跳过 |
| 05 | `00-原始设定/L1-元设定层/05-势力组织.md` | 主要势力结构/目标/资源/关系图谱、隐藏势力 |
| 06 | `00-原始设定/L1-元设定层/06-关键物品和资源.md` | 力量体系物品、经济资源、剧情关键物品 |

## 原则

L1 设定的存在目的是**服务于 L0 灵魂层**。每个元设定都能回答"它服务什么爽点"。
