# Lv2 · 阿金 — 案例卡

> 📍 **快照标记：第 1-4 章 | 海贼船在小镇劫掠期间**
> 小说：《海贼法典》 | 卡状态：首次登场·功能性角色
> 截取原因：阿金在小说前 4 章中的功能集中而明确——作为威廉的忠诚部下提供武力保护和执行功能

---

## 设计目的

为特定叙事功能服务。阿金的存在让威廉这个穿越者有一个"可信任的执行者"，在陌生的海贼船上形成一个微型权力支点。他不承担独立的情节推动作用，而是放大威廉的决策效力。

## 叙事能力

- **催化剂**：威廉的指令通过阿金获得执行力——他帮威廉稳住场面、提供武力威慑
- **场面注入**：阿金"又黑又瘦、痨病鬼外貌"与"一言不合就抽实心铁棒"的反差，制造压迫感
- **镜像对照**：阿金的"凶悍忠诚"映衬威廉的"有道德底线的冷静领导者"形象

## 二、标签 ← 新增

```
黑瘦 · 痨病鬼外貌 · 腰间插铁棒 · 忠诚 · 不爱说话但行动力强 · 从不动摇 · 不把自己的命当一回事
```

## 三、驱动与功能 ← 新增

| 项目 | 内容 |
|:-----|:------|
| 他/她想要什么 | 跟随威廉活下来。既是报恩（威廉救过他一命），也是信任 |
| 在故事中做什么 | 作为威廉小团伙的核心武力执行者：保护威廉安全、震慑其他海贼、执行脏活判断 |
| 离开后去哪 | 阿金的功能是为威廉提供忠诚的武力支持，只要威廉还在船上他就会一直跟着。※ 后续如需单独展开（如阿金独自执行任务或因某事被俘），可升 Lv3 |

## 四、说法风格 ← Lv1 升级

| 字段 | 填写 |
|:-----|:------|
| 对外语气 | 对威廉低声但干脆；对外人冷硬；对敌人几乎不废话 |
| 句式习惯 | 先结论，后补半句；能用两个字说完就不用十个字 |
| 口头禅/常用词 | 行、我去、交给我 |
| 信息释放方式 | 有用信息直接抛出，不解释情绪 |
| 情绪上头时怎么说 | 越怒越短，像在下砍人的前置命令 |
| 示例台词 | “人我盯住了。” / “你退后，这里我来。” |
