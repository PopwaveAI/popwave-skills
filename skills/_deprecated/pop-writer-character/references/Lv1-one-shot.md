# Lv1 · 路人/一次性 — 模板

> 适用：只出现 1 个场景的角色、背景板、路人。一句话概括即可。
> 颗粒度：一句话摘要 + 说法风格 | 不对后续情节承担责任。

---

## 设计目的

> 填充世界观，营造场景真实感。让读者感觉到"这个世界不只有主角在动，还有无数普通人在生活"。

## 叙事能力

> 承载单个场景功能：传递一句关键信息、制造一个瞬间氛围、提供一个背景对比。用后即弃。

---

## 一、一句话摘要

```
一个[外貌/身份]的[人/生物]，他/她[在此场景中的行为/说的话]。
```

---

## 二、说法风格 ← 新增

> 即使是路人，也要知道“他一句话会怎么说”。至少给正文一个可直接拿去写对白的声音锚点。

| 字段 | 填写 |
|:-----|:------|
| 语气 | 直白 / 紧张 / 讨好 / 木讷 / 油滑 / 怯生生 / 其他 |
| 句长 | 短句 / 正常句 / 碎句 |
| 常用词 | 1-2 个口头词，如“师兄”“官爷”“哎”“这位” |
| 避免点 | 不会说什么，如“不讲术语 / 不会长篇解释” |
| 示例台词 | 一句不超过 25 字的代表性台词 |

---

## 三、★ 复用例（可选）

> 如果这个一次性角色有可能在后续场景再次出现，标注潜在展开方向。

```
★ 可复用：[这个角色如果再次出现，可以往哪个方向展开？]
```

---

## 最小样例

```markdown
## 一、一句话摘要
- 一个满头汗的年轻衙役，他拦住主角，压低声音说“前头别去，刚抬出来一具尸首。”

## 二、说法风格
- 语气：紧张、压低声音
- 句长：短句
- 常用词：别去、刚刚
- 避免点：不会讲完整来龙去脉
- 示例台词：前头别去，刚抬出来一具尸首。
```
