# Lv2 · 功能角色 — 模板

> 适用：为特定叙事功能服务、出现 1-3 幕的角色。
> 颗粒度：Lv1 全部 + 身份/标签/驱动功能 + 升级后的说法风格 | 不需要弧线，任务完成后退场。

---

## 设计目的

> 为特定叙事功能服务。作者需要一个人物来完成某个固定的叙事动作（递情报、触发决策、制造冲突），完成后角色退场或降为背景。

## 叙事能力

> 推动单幕情节：作为信息管道释放情报、作为催化剂触发主角决策、作为冲突放大器制造阶段性紧张。

---

## 一、摘要 / 身份 ← Lv1 升级

| 字段 | 填写 |
|:-----|:------|
| 姓名 | |
| 一句话印象 | "一个[特质]的[身份]，他/她[做了什么/给主角什么]" |

---

## 二、标签 ← 新增

```
[3-5 个关键词，覆盖性格、外貌、说话方式、行为模式]

例如：急性子 · 烟不离手 · 说话带刺但心软 · 穿褪色军大衣 · 敲桌子
```

---

## 三、驱动与功能 ← 新增

| 项目 | 内容 |
|:-----|:------|
| 他/她想要什么 | 在此场景/幕中的目标 |
| 在故事中做什么 | 具体承担的叙事功能 |
| 离开后去哪 | 任务完成后角色去向（不需要后续追踪） |

---

## 四、说法风格 ← Lv1 升级

> Lv2 开始，“怎么说话”已经影响功能完成效率。必须能支撑 2-4 句连续对白。

| 字段 | 填写 |
|:-----|:------|
| 对外语气 | 对主角/对陌生人/对上位者分别什么口气 |
| 句式习惯 | 短句推进 / 反问多 / 先结论后解释 / 碎碎念 |
| 口头禅/常用词 | 1-3 个 |
| 信息释放方式 | 直接说 / 挤牙膏 / 先试探再说 / 用抱怨带出信息 |
| 情绪上头时怎么说 | 发怒 / 害怕 / 着急时语言怎么变形 |
| 示例台词 | 2 句，分别体现日常与任务状态 |

---

## 五、★ 复用例（可选） ← Lv1 继承

```
★ 可复用：[如果这个角色在后续场景再次出现，可以往哪个方向展开？]
```

---

## 最小样例

```markdown
## 四、说法风格
- 对外语气：对主角带点熟络；对陌生人戒备；对上位者短促应答
- 句式习惯：先报结果，再补细节
- 口头禅/常用词：成、我去、别磨叽
- 信息释放方式：有用信息会直接甩出来
- 情绪上头时怎么说：越急越短，像下命令
- 示例台词：
  - “人我给你带来了，后门那边暂时没人盯。”
  - “别磨叽，再晚半刻咱们都得倒霉。”
```
