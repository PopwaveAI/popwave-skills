## 维度递增体系

Lv1 → Lv4 维度严格做加法。每级 = 上一级**全部维度** + 本 Lv **新增维度**。

```
Lv1 ── 路人/一次性
  ├─ 一句话摘要
  ├─ 说法风格
  └─ [可选] ★ 复用例

Lv2 ── 功能角色 ────── Lv1 全部 + 新增功能层
  ├─ 摘要/身份（升级）
  ├─ 标签（新增）
  ├─ 驱动·功能（新增）
  ├─ 说法风格（升级）
  └─ [可选] ★ 复用例（继承）

Lv3 ── 重要配角 ────── Lv2 全部 + 新增弧线层
  ├─ 身份（升级：姓名/年龄/关系）
  ├─ 标签（继承）
  ├─ 驱力（升级：想要/恐惧/背叛条件）
  ├─ 叙事功能（新增）
  ├─ 弧线（新增）
  ├─ 登场场景表（新增）
  └─ 说法风格（升级：形成可识别角色声音）

Lv4 ── 主角/核心对立 ── Lv3 全部 + 新增主轴层
  ├─ 身份（完整版：种族/职业/原型/出场状态）
  ├─ 标签（继承）
  ├─ 心理（新增：性格特质/决策逻辑/核心缺陷）
  ├─ 驱力（升级：表层/深层/黑暗/恐惧四层 + 冲突分析）
  ├─ 能力（新增：优势/中等/弱项 + 成长路径）
  ├─ 演化（新增：阶段弧线 + 弧线类型 + 不可逆点）
  ├─ 网络（新增：关系图谱 + 演化摘要）
  ├─ 说法风格（升级：对白圣经 / 独白风格 / 分对象声线）
  ├─ 叙事功能（升级：全局叙事定位）
  └─ 弧线（升级：完整类型 + 不可逆点）
```

## 选级快判四问

1. **这人只服务一个场景，还是会跨幕持续存在？**
2. **这人有没有自己的欲望和代价，不只是替别人办事？**
3. **如果删掉这个人，剧情是少一段功能，还是会塌一条关系/弧线？**
4. **这个人值不值得单独维护说话声音？**

> 快判原则：只要第 4 问答案是“值得”，通常至少从 `Lv2` 起步；如果读者能仅凭对白认出这个人，通常应落在 `Lv3+`。

## 每级设计目的 / 叙事能力 / 角色声音目标

### Lv1 · 路人/一次性

| 字段 | 内容 |
|:-----|:------|
| **设计目的** | 填充世界观，营造场景真实感。让读者感觉到"这个世界不只有主角在动，还有无数普通人在生活" |
| **叙事能力** | 承载单个场景功能：传递一句关键信息、制造一个瞬间氛围、提供一个背景对比。用后即弃，不为后续情节承担责任 |
| **角色声音目标** | 给出一句可以直接放进正文的台词，让这个路人不至于像匿名播报器 |

### Lv2 · 功能角色

| 字段 | 内容 |
|:-----|:------|
| **设计目的** | 为特定叙事功能服务。作者需要一个人物来完成某个固定的叙事动作，完成后角色退场或降为背景 |
| **叙事能力** | 推动单幕情节：作为信息管道释放情报、作为催化剂触发主角决策、作为冲突放大器制造阶段性紧张 |
| **角色声音目标** | 能稳定支撑 2-4 句连续对白，让功能推进时带人物味，而不是工具提示 |

### Lv3 · 重要配角

| 字段 | 内容 |
|:-----|:------|
| **设计目的** | 承载独立动机，跨幕存在，形成次级弧线。读者对这个角色有情感期待，角色自己的选择会产生有分量的后果 |
| **叙事能力** | 与主角产生深度互动（伙伴/导师/恋人/宿敌）；承载读者次级情感投射（安心/心疼/愤怒）；为关键情节提供不可替代的推动力 |
| **角色声音目标** | 读者不看名字，也能从 2-3 句对白里大致认出这个人 |

### Lv4 · 主角/核心对立

| 字段 | 内容 |
|:-----|:------|
| **设计目的** | 贯穿全文的灵魂。承载故事核心主题，读者通过这个角色体验整个世界的意义 |
| **叙事能力** | 驱动全篇情节走向——角色弧线 = 故事主梗；承载读者全部情感投射；所有关键转折由这个角色的决策触发；也是读者的最终认同锚点 |
| **角色声音目标** | 形成稳定的“对白圣经”与“独白圣经”，供 `chapter/prose` 长线复用 |
