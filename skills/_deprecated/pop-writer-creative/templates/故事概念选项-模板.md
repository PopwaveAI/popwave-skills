# 故事概念选项模板

> Step 2.2 合成故事概念选项时使用。这不是填空模板，是质量标准+吸引力设计+差异化要求。
> 目标：让用户看到选项时"想立刻开写"，而不是"就这？"

---

## 一、质量标准

### 什么是好的故事概念

| 维度 | ✅ 好 | ❌ Low |
|:-----|:------|:------|
| **一句话定位** | 让人脑中放电影，3 秒内产生画面感 | 类型标签堆砌（"一本黑暗流修仙文"） |
| **世界观结构** | 有具体的"吃人规则"，读者能想象人在里面怎么活 | "有修仙体系，有宗门，有境界"（万能设定） |
| **核心冲突** | 主角和世界的矛盾是结构性的，不是"有个坏人" | "主角要变强，有人阻挡他" |
| **金手指** | 有明确的代价和限制，代价本身制造戏剧性 | "获得系统，可以升级"（无代价开挂） |
| **对标参考** | 说得出和谁像、和谁不像、为什么不像 | "类似XX但又不一样"（说不清不一样在哪） |
| **差异化** | 2-3 个选项从不同维度切入，不是同一套路的微调 | 3 个选项都是"修仙+系统+宗门"，只是金手指名字不同 |

### 选项差异化矩阵（强制）

> 2-3 个选项必须在至少 2 个维度上不同，不能是同一套路的微调。

| 差异维度 | 说明 | 示例 |
|:---------|:-----|:-----|
| **世界观结构** | 单世界 vs 双世界 vs 多位面 | 选项A=单世界修仙，选项B=双世界武道+修仙 |
| **冲突引擎** | 信息差驱动 vs 阶级对抗 vs 智斗博弈 | 选项A=信息差碾压，选项B=阶级革命 |
| **金手指类型** | 系统 vs 血脉 vs 道具 vs 契约 | 选项A=死后复活契约，选项B=感知破绽能力 |
| **主角起点** | 底层 vs 中层 vs 陨落者 | 选项A=黔首底层，选项B=没落世家子 |
| **基调** | 黑暗博弈 vs 热血逆袭 vs 荒诞解构 | 选项A=黑暗流，选项B=黑色幽默流 |

> ⚠️ 如果 2-3 个选项在所有维度上都相同，只有金手指名字不同 = 不合格，重新合成。

## 二、选项格式模板（增强版）

> 比原版增加了「吸引力设计」和「一句话钩子」两个质量模块。

```markdown
### 选项 {N}：{概念名称}

**一句话钩子**：{不是定位，是钩子——读完这句话必须想"然后呢？"。
  公式：{反常识/反预期} + {具体画面} + {未解悬念}
  ❌ Low：「一个穿越者在修仙世界崛起」
  ✅ 好：「一个每次死亡都会换一个身份的穿越者，死到第七次才发现——每个死掉的旧身份都还活着，而且都在找他」}

**全书故事 brief**（150-300字）：
{不是设定摘要，是故事本身。用户读完必须知道"这本书到底讲什么"。
  必须包含：主角是谁（1句）→ 世界怎么吃人的（1-2句）→ 主角的核心困境（1句）→ 主角怎么破局（1-2句）→ 故事的走向弧线（1句）
  ❌ Low：「在一个有修仙体系的世界，主角获得系统后不断变强，最终成为最强者」（万能模板，换本书名也能用）
  ✅ 好：「林渊是黔首出身的信息差修士——他能看见每个人头顶的因果线，但每次读取都会被因果反噬。在一个灵根世袭、凡人如草的修仙世界，他靠因果线在宗门权谋中步步为营。当他发现宗门高层用凡人炼制力士的真相后，他必须在暴露和隐忍之间做选择——暴露意味着死，隐忍意味着成为共犯。故事从他被迫卷入宗门内斗开始，到组织底层修士反抗，最终动摇灵根世袭制度本身。」}

**世界观结构**：
- {世界的基本形态——单世界/双世界/多位面，用1句话描述物理质感}
- {世界的"吃人规则"——不是设定列表，是"这个世界怎么吃人的"1-2句}
- {历史/社会原型——从搜索B提取，说明骨架来源}

**核心冲突模式**：
- {主角vs世界的结构性矛盾——不是"有人阻挡"，是"世界规则本身和主角的存在冲突"}
- {主角的初始位置→上升路径——用1句话画出弧线}
- {冲突的升级逻辑——从小规模到大规模，冲突怎么放大}

### 金手指设计
- **库参考**：{匹配的金手指库设计卡名，或"库缺声明"}
- **名称+机制**：{金手指名称} — {一句话核心机制}
- **代价规则**：{代价类型+戏剧性等级+叙事功能}
- **成长路径**：{4阶段概要+天花板}
- **赛道适配**：{本书赛道的适配度+改写建议}
- **库缺声明**：{如有，说明缺什么类型的设计卡，降级方案}

**对标参考**：
- {1-2本结构相似的作品 + 核心差异化}
- {差异化必须是结构性的，不是"我们文笔更好"}

**域碰撞设计**（v5.0新增）：
- **域清单**：本选项涉及哪些域（域A×域B×域C）
- **叙事语法声明**：{哪个域是操作系统}——因为{判定理由}
- **素材贡献表**：

  | 域 | 贡献什么 | 具体元素 |
  |:---|:---------|:---------|
  | 域A（操作系统） | 叙事语法 | {节奏/爽点/结构} |
  | 域B | 核心机制 | {力量体系/规则} |
  | 域C | 美学基因+经典元素 | {风格/功法/设定} |

- **跨域联动清单**（≥2个）：
  1. {联动点1} — 类型：{元素借用/结构移植/意象蒸馏/功能反转} — 效果：{产生什么新东西}
  2. {联动点2} — ...
- **原创空间**：{碰撞产生的核心创新，标注"本选项独有"}

**风险提示**：
- {这个方向的常见坑——从搜索A的作品缺点提取}
- {本书的应对策略——不是"我们会注意"，是具体的设计对策}

**吸引力自检**：
- 一句话钩子能让人想"然后呢"？ ☐
- 全书故事 brief 让用户知道"这本书到底讲什么"？不是万能模板？ ☐
- 世界有具体的"吃人规则"而不是万能设定？ ☐
- 金手指有代价且代价制造戏剧性？ ☐
- 和其他选项在至少2个维度上不同？ ☐
```

## 三、吸引力设计技巧

### 一句话钩子公式

| 技巧 | 公式 | 示例 |
|:-----|:-----|:-----|
| **反常识** | "所有人都以为X，其实是Y" | "所有人都以为死了就完了，其实死亡只是换身份的过场" |
| **悖论** | "A的前提必然导致B，但B会毁灭A" | "他的金手指越强，欠的债越多，而还债的唯一方式是变得更强" |
| **未解悬念** | "发生了X，但X不可能发生" | "他死过七次，每次都换了身份复活——但第一个身份突然出现在他面前" |
| **极端处境** | "在X条件下，只有Y能活" | "在一个说谎就会变强的世界，他天生不会说谎" |
| **身份错位** | "他是X，但被当成了Y" | "一个现代会计穿越成了魔教教主，手下以为他的每句话都是深谋远虑" |

### 世界观"吃人规则"设计

> 好的世界观不是"有什么"，是"怎么吃人的"。

| ❌ Low（设定列表） | ✅ 好（吃人规则） |
|:---|:---|
| 有九个境界，每个境界有不同能力 | 灵根决定一切——没有灵根的人连修炼的资格都没有，而灵根是世袭的 |
| 有宗门、有散修、有凡人 | 修仙者收割凡人——死人税、人牲、力士炼制，凡人是修仙者的原材料 |
| 有系统可以升级 | 系统是贷款——提前支取力量，但利息是寿命，还不上就死 |

### 金手指代价设计

> 代价不是限制，是戏剧性来源。

| 代价类型 | 戏剧性 | 示例 |
|:---------|:------|:-----|
| 资源消耗 | ⭐⭐ | 消耗灵石/气运——可预测，戏剧性低 |
| 时间限制 | ⭐⭐⭐ | CD冷却——制造紧迫感 |
| 身份暴露 | ⭐⭐⭐⭐ | 越级使用会暴露自己——制造 secrecy tension |
| 不可逆损失 | ⭐⭐⭐⭐⭐ | 每次使用失去一段记忆/一个关系——制造情感代价 |
| 悖论代价 | ⭐⭐⭐⭐⭐ | 变强=变不像人/变强=欠更多债——代价本身是叙事引擎 |

## 四、质量自检清单

合成选项后，逐条自检：

| 检查项 | 标准 |
|:-------|:-----|
| 一句话钩子 | 每个选项的钩子能让人想"然后呢"？不是类型标签堆砌？ |
| 吃人规则 | 每个选项的世界有具体的"吃人规则"，不是万能设定？ |
| 金手指代价 | 每个选项的金手指有代价，且代价制造戏剧性（≥⭐⭐⭐）？ |
| 差异化 | 2-3个选项在至少2个维度上不同？不是同一套路的微调？ |
| 特征参数关联 | 每个选项的世界观/冲突/金手指从Step 1特征参数推导，不是通用模板？ |
| 对标参考 | 每个选项说得出和谁像、和谁不像、为什么不像？ |
| 风险应对 | 每个选项的风险有具体应对策略，不是"我们会注意"？ |
| 域碰撞设计 | 每个选项有叙事语法声明？跨域联动≥2个？有原创空间？ |
| 碰撞非缝合 | 联动点是否产生了新东西（不是简单放在一起）？ |
| 吸引力 | 如果你是读者，看到这3个选项会想选一个继续吗？ |
| 金手指查库 | □ 金手指已查库（金手指库/00-索引.md），有库参考或库缺声明 |
| 金手指代价 | □ 金手指有代价设计（不是无代价变强） |
| 金手指成长 | □ 金手指有成长路径（不是一步到位） |
```
