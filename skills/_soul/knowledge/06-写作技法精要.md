# 写作技法精要

> 领域：内容创作 → 写作技法
> 适用：需要提升文字表现力的写作场景
> 来源：整合自知乎文章 10 篇 + ibiling 描写技法 + 52 个网文技巧 + 100 条核心技巧
> 目的：独立可读。读完能写出画面感强、节奏好、不拖沓的正文。

---

## 一、否定式描写：最隐蔽的写作癌症

### 问题

用否定句来描述事物（"不是A而是B"），而不是直接呈现。读者读到"不是X"时需要先理解X是什么，再理解"不是X"意味着什么——信息传递效率降低 50%。

### 禁止的 6 种模式

| 变体 | 示例（禁止） | 替代（推荐） |
|:-----|:------------|:------------|
| 不是A而是B | "那不是恐惧，是一种说不清的压迫" | "恐惧攥住了他的喉咙" |
| 不是A不是B是一种C | "不是风不是震动是一种频率" | "空气在低频震颤" |
| 说不清的X | "一种说不清的感觉" | "寒意从脚底爬到脊椎" |
| 直觉告诉 | "直觉告诉他不能回头" | "他停住了脚步，没有回头" |
| 他没有/她没有问 | "他没有问为什么" | "他想问为什么，张了张嘴，没出声" |
| 不知道为什么 | "不知道为什么，他感到不安" | "不安毫无来由地攥紧了他的胃" |

**铁律**：任何形式的"否定式描写"在正文中出现即标记为 WARN。

## 二、留白四招

| 招式 | 做法 | 效果 |
|:-----|:-----|:------|
| **句末留白** | 关键信息说完就断句，不加解释 | 读者自己脑补，比自己写更震撼 |
| **对话留白** | 角色没说完的话用"……"或动作打断 | 制造张力 |
| **情节留白** | 高潮后不给立刻反应，切到另一条线 | 悬念叠加 |
| **情绪留白** | 激烈情绪后，但"不让角色哭出来" | 读者先替角色哭 |

### 案例：留白 vs 填满

```
❌ 填满：
  他感到巨大的悲伤涌上心头。
  他想起了小时候和爷爷一起度过的那些日子。
  那些日子再也回不去了。
  他的眼泪终于掉了下来。

✅ 留白：
  他站在爷爷的坟前，风很大。
  墓碑上的字他已经看了很久。
  "走吧。"他说。
```

## 三、五感描写——骗过读者的大脑

读者"感受"场景靠的不是逻辑，是感官。在关键位置调用至少两种感官：

| 感官 | 形容词方向 | 效果 |
|:-----|:----------|:-----|
| **视觉** | 颜色/亮度/形状/运动 | 建立场景的基本画面 |
| **听觉** | 音量/音调/节奏/远近 | 制造氛围和紧张感 |
| **触觉** | 温度/质地/压力/风吹 | 让读者"感觉自己在场" |
| **嗅觉** | 气味强度/类型 | 最快唤起记忆的感官 |
| **味觉** | 苦/甜/金属味/血腥味 | 最具冲击力的感官 |

**原则**：关键场景调用至少 2 种感官。不要每段都写"他看到的"——只写最关键的 2-3 个细节，让读者脑补剩下的。

## 四、节奏控制

### 张弛有度黄金法则

```
高潮 → 缓冲（短） → 蓄力（中） → 高潮 → 缓冲 → 蓄力 → 大高潮
```

| 段 | 节奏 | 字数 | 写法 |
|:---|:-----|:----|:-----|
| 高潮 | 快 | 500-800 | 短句、动词密集、少修饰、少副词 |
| 缓冲 | 中 | 300-500 | 情绪回落、环境描写为主 |
| 蓄力 | 慢 | 800-1500 | 细节铺陈、心理活动、信息展示 |

### 语言辅助节奏

| 节奏 | 句长 | 标点 | 修饰 |
|:-----|:-----|:-----|:-----|
| 快 | 8-15 字/句 | 多用句号，少分句 | 形容词少，动词多 |
| 中 | 15-25 字/句 | 正常 | 中等修饰 |
| 慢 | 25-40 字/句 | 多用逗号、破折号 | 细致铺陈 |

### 高强度节奏压缩技巧

当某个情节本身不够"重"但需要快节奏时：

1. **压缩铺垫**——"他训练了三年"（不描写三年过程）
2. **加速重复**——"第一个、第二个、第三个……"（快速带过同类事件）
3. **跳刀叙述**——不是逐字按时间写，只写关键帧

## 五、对话技巧

### 对话占比基线

| 章节类型 | 对话占比 | 说明 |
|:---------|:---------|:-----|
| 战斗/动作章 | 15-25% | 动作多，对话自然少 |
| 推进/过渡章 | 25-35% | 对话推进信息+调节节奏 |
| 情感/互动章 | 35-50% | 大量角色互动 |

**底线**：全文对话占比 ≥ 25%。连续 3 章低于 25% → 告警。

### 让对话有戏的方法

1. **每句对话都要有"潜台词"** — 不只是传递信息，还要透露角色的情绪/目的/隐瞒
2. **用动作/环境代替引号** — "他推开门"比"他说"更有画面感
3. **打断对话** — 当角色说到一半时,让另一个角色打断/让环境打断，制造真实感

## 六、章节级六大钩子

| 类型 | 定义 | 写法 |
|:-----|:-----|:------|
| **事件断裂** | 中断在事件关键节点 | "他的手指碰到了扳机——"（章末） |
| **悬念收束** | 揭示线索但留疑问 | "那封信的落款，是一个已死之人" |
| **代价提示** | 展示选择的代价 | "裂缝到了第四道" |
| **信息差** | 读者知道角色不知道 | 读者知道埋伏，角色往前走 |
| **新威胁** | 新的危机浮现 | "远方传来脚步声" |
| **过渡缓冲** | 情绪承接 | "后天就到了" |

**铁律**：不允许连续 2 章使用同一种钩子类型。不确定时默认用"事件断裂"。

## 七、画面感三原则

### 1. 展示不告知

```
❌ "他很害怕" → 告知
✅ "他的手指在发抖，声音比平时高了半个调" → 展示
```

### 2. 具体代替抽象

```
❌ "那是一种说不清的感觉" → 抽象
✅ "空气像粘稠的糖浆一样包裹着他" → 具体
```

### 3. 动作比心理更有效

```
❌ "他意识到自己惹了麻烦" → 心理
✅ "他的笑容僵在脸上，慢慢放下了酒杯" → 动作
```

## 八、叙事视角控制

### 不要随意切换视角

**铁律**：在同一章内，不切换视角。如果必须切换，用空行 + 明显标记隔开。

### 视角决定信息量

| 视角 | 读者知道 | 读者不知道 |
|:-----|:---------|:-----------|
| **主角限制视角**（推荐） | 主角看到听到的一切 | 其他人的想法和视角外的信息 |
| **上帝视角** | 一切 | 悬疑感下降 |
| **多视角交替** | 多条线索 | 需要管理信息一致性 |

**番茄平台推荐**：主角限制视角。读者跟主角同步发现世界，代入感最强。

## 九、52 个网文技巧速查

按领域分类的核心技巧：

### 开头

1. 前三行不无聊——第一个词要有分量
2. 第一章必须有钩子——不是目标，是硬性要求
3. 拒绝大段旁白介绍——用行动展示设定

### 人物

4. 主角要有缺点——至少一个可以通过"丢人"体现的
5. 配角要有功能——不只是"站在那边"
6. 反派要有立场——他不是"坏"，是"站在主角对立面"

### 情节

7. 每章一个小悬念+一个大悬念
8. 不要让主角顺利太久——每 3 章制造一次新的挫折
9. 伏笔要在 10 章内回收——读者记不住太久

### 叙述

10. 短句 = 紧张感，长句 = 铺陈感
11. "突然"不要频繁用——真正的突发事件不需要"突然"
12. 情绪最高点时用短句——一个字一行都可以

### 心态

13. 不要等灵感——写就对了，80% 分之后改
14. 卡文时跳过——去写你知道的内容
15. 这章写的不好——你改了,读者不知道,但你不改,读者知道
