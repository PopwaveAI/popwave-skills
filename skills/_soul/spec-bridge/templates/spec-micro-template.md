## Assumptions

## Acceptance Criteria

### AC-1: 正文合规
- **Given**: 本章正文已完成
- **When**: 逐项检查内容
- **Then**:
  1. 字数在目标范围 ±15% 内
  2. 覆盖了 FR-1 中所有场景
  3. 未涉及 Non-Goals 中排除的内容
  4. {实体名} 计数 ≥ 约束要求
- **Verification**: programmatic

### AC-2: 爽点落地
- **Given**: 正文已完成
- **When**: QC 正文层介入
- **Then**: 纯感受报告中提及了 FR-2 设计的爽点类型，无"弃书"标记
- **Verification**: human-judgment + programmatic

### AC-3: 钩子有效
- **Given**: 正文已完成
- **When**: 检查章末 3 行
- **Then**: 符合 FR-5 定义的钩子类型，答案未被提前释放
- **Verification**: programmatic

### AC-4: 风格一致
- **Given**: 正文已完成
- **When**: 对比参考片段的写法
- **Then**: 画面密度和叙事效率接近锚定章水平
- **Verification**: human-judgment

### AC-5: 无越界实现
- **Given**: 正文已完成
- **When**: 检查 Non-Goals 列表
- **Then**: 没有涉及 Non-Goals 中定义的排除内容
- **Verification**: programmatic
