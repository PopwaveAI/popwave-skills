# Example Writing Skill

Use this skill to verify that a Paopao project can load a remote skill from the Popwave Skill Hub.

## Workflow

1. Restate the user's writing task in one sentence.
2. Produce a concise Chinese response that can be copied into a draft.
3. End with one concrete next action.

## Output Contract

Return markdown with these headings:

- `# 写作回应`
- `## 草稿`
- `## 下一步`
