# 衍生/二创作品 — 设定与设计 Gap 分析工作流

> 适用场景：用户基于已有作品（有 deconstructor 产出）做同人二创/跨界融合/角色替换
> 触发时机：pop-writer-creative Phase 0 完成后、pop-writer-plot 开工前
> 核心原则：设定层 1:1 对齐补全 → 然后才推进剧情设计

---

## 一、前置条件

进入 gap 分析前必须确保：
1. ✅ 源作品的 **deconstructor 产出** 存在（否则无基准可对比）
   - 路径通常是 `D:/workspace/{源作品名}-拆解试水/`
   - 需要的核心文件：L1-世界观/六件套、世界宪法、全书架构.md、volume-01.md、act-XX.yaml
2. ✅ 新作品的 **L1 元设定层** 六件套已产出
3. ✅ 新作品的 **世界宪法** 已有初版

---

## 二、双层 Gap 分类法

所有差异分为两个层次，**处理方式不同**：

### 2.1 设定层 Gaps

| 判定器 | 属于设定层 |
|:-------|:-----------|
| 问"这个差异**是否改变了源作品的底层规则**" | 如果答案是"是" → 设定层 gap |
| 具体范围 | 位面法则、力量体系、属性规则、种族生物法则、世界宪法条款 |
| **处理方式** | **必须 1:1 对齐补全**。不可跳过或"以后再改" |

### 2.2 设计层 Gaps

| 判定器 | 属于设计层 |
|:-------|:-----------|
| 问"这个差异**是否改变了剧情的走向/节奏/角色关系**" | 如果答案是"是" → 设计层 gap |
| 具体范围 | 章节顺序、角色出场时机、剧情线分配、情感锚点设置、升级节奏 |
| **处理方式** | 逐项标注后提交用户决策。用户可能接受差异、也可能要求调整 |

## 三、执行流程

```
Step 1: 加载源作品 deconstructor 产出
  → 读 L1-01~06（设定层基准）
  → 读 世界宪法（规则基准）
  → 读 全书架构.md + volume-XX.md（设计层基准，便于后续对齐）

Step 2: 逐维扫描设定层差异
  → 比对新旧作品的 L1 六件套
  → 对比世界宪法条款
  → 输出表单：每项标注"已对齐 ❎ 有gap"、"gap 说明"
  
Step 3: 逐项扫描设计层差异
  → 对比全书架构（卷拆分/主线定义）
  → 对比首卷 volume-01.md（剧情线/角色池/势力动机）
  → 输出表单：标注 "与源作品不同，已适配新设定"

Step 4: 生成 Gap 报告
  ┌────────────────────────────────────────────┐
  │ 一、设定层 Gap（必须1:1补全）               │
  │   ├ ✅ 位面法则 — 已对齐                    │
  │   └ ❎ 属性体系 — 源作品有六维+超凡/传奇门槛 │
  │      → 需补充                              │
  │                                            │
  │ 二、设计层 Gap（用户决策）                  │
  │   ├ 源作品有薇薇安式情感锚点                │
  │   └ 新作品暂缺对应角色                     │
  └────────────────────────────────────────────┘

Step 5: 补全设定层 Gaps
  → 逐项修改 L1 文件
  → 更新世界宪法追加条款
  → 更新数值体系追加属性表/法则
  → 新建缺失的地点设定文件（对标源作品粒度）

Step 6: 补全后验证
  → 世界稳定性检验（7项跑一次）
  → 确认宪法条款与实际设定无矛盾
  → 告知用户"设定层已对齐"，等待用户确认后再推进 plot
```

---

## 四、常见 Gap 类型速查

| 类型 | 典型表现 | 处理 |
|:-----|:---------|:-----|
| **缺少属性体系** | 新作品只有等级/段位，源作品有六维+阈值 | 补 DND 3.5e 六维属性 + 超凡门槛(20) + 传奇门槛(25) |
| **缺少属性成长规则** | 新作品升级不涉及属性分配 | 补"每2级1自由属性点"规则 |
| **缺少属性压制规则** | 战斗描写中高属性无压制效果 | 补"差距≥5压制/≥10免疫"规则 |
| **地点粒度不足** | 源作品有琥珀城（5街区/经济/势力/时间线），新作品只有地名 | 对标补地点设定文件 |
| **世界宪法偏推导** | 宪法条款从设定推导而非从故事验证 | 接受暂缺，标注"待故事产出后验证修订" |

---

## 五、同步推进 vs 分步推进

- **设定层 gaps 必须全部补完再进 plot**（否则 plot 产出的地图/角色/战力线会因设定不匹配而返工）
- **设计层 gaps 可以带着进 plot**，但在卷设计（volume-XX.md）的剧情线确认闸门前必须与用户达成一致
