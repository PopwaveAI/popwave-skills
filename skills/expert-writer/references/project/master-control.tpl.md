## 🗺️ 所属管线

### 理想全流程

```
种子设计(pop-writer-v3-seed) → L2卡设计(pop-writer-v3-plot) → 涌现写作环(expert-writer 5步循环) ↔ 弧线校准(pop-writer-v3-arc)
                                                                ↑ 按需：pop-research
```

### 5步循环（每章执行）

| Step | 名称 | 执行者 | 人工check |
|:-----|:-----|:-------|:---------|
| Step0 | 导演意图提取 | 主会话 | CHECK 1：用户确认 |
| Step1 | 状态快照投影 | 主会话 | 无 |
| Step2 | 信息获取 | 主会话 | 无 |
| Step3 | 子agent创作 | 子agent | CHECK 2：用户验收 |
| Step4 | receipt检查 | 主会话 | 无 |
| Step5 | 活记忆更新+落盘 | 主会话 | 无 |

### 弧线触发

- **每个L2单元结束时触发arc**（不再等每10-20章）
- 触发条件：L2单元最后一章Step5完成
