# Step 0 - 项目初始化

仅在新项目或关键目录缺失时执行。

## 最小目录

```text
项目根/
  项目总控.md
  卷纲/
    卷1-战略锚.md
    卷1-幕序列.md
    运行/
  library/
    单元剧情卡/
    设定账本/
    文风DNA/
  正文/
  活记忆/
    活记忆.yaml
```

## 初始化原则

- `项目总控.md` 用 `references/project/master-control.tpl.md` 的结构。
- `卷纲/卷N-战略锚.md` 存本卷商业承诺、主压力、主收益、禁区和章节预算；它是用户确认文件，不是剧情细纲。
- `卷纲/卷N-幕序列.md` 只存本卷 5-7 个幕的功能位，不写逐章剧情。
- `卷纲/运行/` 存卷级尺度契约、卷级设计意图、当前幕切片、案例消化、候选竞技、审计、章节运行日志与状态快照。
- `卷纲/幕纲-幕XXX-{标题}.md` 存 plot 产出的唯一可用幕纲。
- `library/单元剧情卡/` 只存用户明确确认后需要沉淀的单元卡副本；未通过质量门的草案不得进入这里。
- `library/设定账本/` 存 seed 初始化和 arc/plot 增量冻结的设定。
- `活记忆/活记忆.yaml` 只放当前写作必须知道的短状态，不替代设定账本。
