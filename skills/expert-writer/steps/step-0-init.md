# Step 0：项目初始化

> 创建 Pop 写作项目目录骨架。初始化只建结构，不设计剧情。

## 初始化操作

1. 如果 `项目总控.md` 已存在，视为已初始化，跳过。
2. 创建上述目录。
3. 创建 `写作参考/索引.md` 初始骨架。
4. 从 `references/project/master-control.tpl.md` 创建 `项目总控.md`。
5. 初始化完成后，路由到 `pop-writer-v3-seed`。
