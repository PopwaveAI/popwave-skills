# Step 3：调度修订子skill

> 参考: expert-writer 涌现写作环
> 类型: 主会话调度（调用 pop-writer-v3-revise）
> 消费: Step 2正文初稿 + 文风DNA + 种子文件夹 + 活记忆
> 产出: 重写稿 + 修订记录
> 红线: ❌1 文风DNA缺失=修订层硬阻塞终止；❌7 子skill调度context隔离

## 目的

emerge作为调度器，组装精简context调用pop-writer-v3-revise子skill执行**完全重写（基于正文+文风DNA）**，并承担v3.4下沉的全部质检职责：文风DNA逐条终验+事实信息一致性检查+字数终检+AI观感词清理。revise是最终正文交付前的最后一道关卡，重写稿产出后交付用户check（CHECK 2）。v3.4剔除qa环节后，原qa的文风验证职责全部下沉到revise层。

## 输入context组装（精简，不传会话历史）

| 输入项 | 来源 | 内容 |
|:-------|:-----|:-----|
| 正文初稿 | Step 2 create子skill产出 | 待修订的正文全文 |
| 文风DNA | `写作资产/文风库/{书名}.md` | 句式/叙事距离/感官密度/对话留白锚定（硬阻塞，红线❌1） |
| 种子六要素 | `种子/文件夹`（当前版本） | 主角引擎（含行为准则）/金手指/冲突轴+活跃线索（人设丰富+bug修复的依据） |
| 活记忆七组件 | `活记忆/活记忆.yaml` | 角色状态/世界规则/种子追踪（bug修复+人设丰富的依据） |
| 创作决策记录 | Step 2 create产出 | 用于事实信息一致性比对的骨架来源 |
| 修订checklist | `pop-writer-v3-revise/templates/修订checklist-模板.md` | 5项checklist（含行为准则对齐2.3）+修订记录格式 |

**不传入的内容：** 主会话历史、Step 0/1/2的执行过程、质检模板。

## 调用

加载 `pop-writer-v3-revise` SKILL.md → 按其 `steps/step-1-revise.md` 执行

子skill执行内容：
- 文风DNA硬阻塞检查（红线❌1）
- `Get-Content -Raw` 完整加载文风DNA（禁止limit截断）
- 基于正文骨架+文风DNA完全重写
- 事实信息一致性检查（与创作决策记录骨架比对）
- 文风DNA逐条终验（每项附正文证据）
- 人设丰富（主角引擎对齐+**行为准则对齐2.3**+心理层次）
- 爽点验证（即时奖励+养成感+情绪闭环）
- bug修复（世界规则+伏笔+金手指+时间线）
- AI观感词清理（≤3种命中）
- 字数终检（2500-3500区间）

## 输出收集

| 输出项 | 格式 | 传递给 |
|:-------|:-----|:-------|
| 重写稿 | markdown | 交付用户check（CHECK 2：最终正文验收）→验收后传Step 4 |
| 修订记录 | YAML | Step 4（记忆+生长） |

## 子agent失败降级策略（方案B兜底）

优先使用子agent（sessions_spawn）调度revise。子agent失败时降级为主会话执行，但必须：

1. **重读完整种子context（`Get-Content -Raw`）**：主角引擎+金手指+冲突轴
2. **重读文风DNA完整文件（`Get-Content -Raw`，禁止limit截断）**——v3.3事故根因
3. **独立执行文风DNA终验**：降级不豁免终验，每项附正文证据
4. **降级标记**：修订记录标注 `degraded_master_execution: true`
5. **重试限制**：最多1次，第2次失败才降级

> 降级≠跳过门禁。6步门禁在降级模式下同样必须全部通过。

## 门禁

| 检查项 | 失败动作 |
|:-------|:---------|
| **传入了会话历史（红线❌7）** | **退回，重组精简context** |
| **文风DNA缺失（红线❌1）** | **硬阻塞终止** |
| 子skill未产出修订稿 | 终止，报错 |
| AI观感词命中>3种 | 退回create子skill重写 |
| 8020比例超标 | 退回重写 |
| 文风DNA用read limit截断加载 | 退回 `Get-Content -Raw` 重载 |
| 重写稿事实信息与初稿不一致 | 退回重写 |
| 文风DNA终验未逐条附正文证据 | 退回补验 |
| 字数<2500或>3500 | 退回重写/精简 |
