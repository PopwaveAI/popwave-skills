# Step 2-5：正文落盘与活记忆更新

> create 或 revise 产出正文后执行。

## 1. 正文落盘

写入：

```text
正文/chXXX.md
```

落盘后确认：

- 文件存在。
- 非空。
- UTF-8 编码。

## 2. 活记忆追加

读取：

```powershell
Get-Content -Encoding UTF8 -Raw '活记忆/活记忆.yaml'
```

追加一条自然语言 event：

```yaml
events:
  - chapter: chXXX
    summary: >
      {一段话概括本章实际发生的不可逆变化：人物、资源/能力、关系/声望、
      威胁/情报、章末新压力。不要写复盘长文。}
```

活记忆只保存下一章需要知道的状态，不写完整剧情分析。

## 3. 项目总控更新

如存在 `项目总控.md`，更新：

- 当前章号。
- 已完成章节。
- 当前单元进度。
- 最近更新内容。

## 4. 单元结束检查

根据单元卡章节范围判断当前章是否为单元最后一章。

如果是：

- 提示进入 `pop-writer-v3-arc`。
- arc 将负责事后反拆、设定账本维护、真实剧情线命名和活记忆压缩。

如果否：

- 提示下一章可继续。
