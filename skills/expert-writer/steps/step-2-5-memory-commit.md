# Step 2.5 - 结果写入

在执行 skill 完成后，只做必要写入，不扩写历史。

## 正文完成

- 正文保存到 `正文/chXXX.md` 或用户指定路径。
- 将本章真实发生的变化追加到 `活记忆/活记忆.yaml`。
- 如果出现新设定或设定变更，标注“待 arc/plot 冻结”，不要在正文阶段私自改总设定。

## 幕纲完成

- 最终幕纲保存到 `卷纲/幕纲-幕XXX-{标题}.md` 或用户指定路径。
- 过程文件必须保存到 `卷纲/运行/`，至少包括卷级尺度、卷级设计意图、当前幕切片、案例消化、方案仿真、审计压缩。
- 必须存在 `卷纲/运行/卷N-00a-上游宪法消费确认.md`。若 PRD、L1世界包、动态升级表、主角引擎、金手指或文风DNA缺失，幕纲只能标为草案。
- 必须存在 `卷纲/运行/卷N-00-全书卷级尺度契约.md` 和 `卷纲/运行/卷N-00b-卷级设计意图契约.md`。若没有先对齐当前卷体量、读者承诺、主角变化、禁区和不可牺牲项，幕纲只能标为草案。
- 必须存在 `卷纲/运行/幕XXX-01-当前幕切片确认.md`。若没有先确认本轮深推演哪一幕、该幕功能位和边界，幕纲只能标为草案。
- 必须存在 `卷纲/运行/幕XXX-03-候选竞技.md` 和 `03a/03b/03c` 三套完整候选文件。若只有单方案、A/B 弱对比、B/C 低颗粒度概览，幕纲只能标为草案。
- 必须存在 `卷纲/运行/幕XXX-02b-外部母版扫描.md`，并说明外部类型母版如何进入事件矿池或为什么被淘汰。若用户明确禁止外部参考，则记录该禁用声明。
- 必须生成 `卷纲/运行/幕XXX-产物索引.md`，标明每个文件是“过程资料/草案/唯一可用稿”，并标明能否给 create。
- 章节运行日志保存到最终幕纲内，同时可另存 `卷纲/运行/幕XXX-运行日志.md` 便于 create 读取。
- 章级导演包必须保存到最终幕纲内，或另存 `卷纲/章节设计包/幕XXX-chYYY-设计包.md`。缺章级导演包时，不得称为可直接正文的完成稿。
- 设定新增项写入 `卷纲/运行/幕XXX-设定账本.md` 草案区。
- 未通过质量门的幕纲不得写入 `library/单元剧情卡/`，也不得在回报中称为“完成稿”。

## Seed 立项完成

- seed 正式产物保存到 `library/设定账本/`。
- seed 研究档案保存到 `library/设定账本/研究档案/`，过程文件保存到 `library/设定账本/运行/`，不得保存到 `卷纲/运行/`。
- 必须存在 `library/设定账本/研究档案/00-研究索引.md`、`library/设定账本/研究档案/01-种子展开图.md`、`library/设定账本/研究档案/02-外部母版卡.md`。
- 必须存在 `library/设定账本/运行/00-故事概念候选PK.md`。若没有 2-3 个互斥故事概念和选择理由，只能标为草案。
- 必须存在 `library/设定账本/seed-产物索引.md`、`library/设定账本/新书立项PRD.md`、`library/设定账本/seed-to-world交接包.md`。
- seed 完成后只允许推进到 world-foundation，不得直接推进到 plot。

## World 底盘完成

- world-foundation 正式产物保存到 `library/设定账本/` 和 `library/文风DNA/`。
- 设定涌现全过程保存到 `library/设定账本/设定涌现/`。
- 必须存在 `library/设定账本/设定涌现/00-涌现索引.md`、`01-PRD消费确认.md`、`02-设定问题清单.md`、`03-候选规则池.md`、`04-设定碰撞矩阵.md`、`05-L1方案竞技.md`、`06-冻结决策日志.md`、`07-设定债务与待验证.md`、`08-金手指候选竞技.md`。
- 必须存在 `library/设定账本/L1-世界蓝图.md`、`library/设定账本/L1-力量体系.md`、`library/设定账本/L1-势力格局.md`、`library/设定账本/L1-资源物品.md`、`library/设定账本/L1-术语与文明底色.md`、`library/设定账本/起点快照.md`、`library/设定账本/世界宪法.md`、`library/设定账本/动态升级表.md`、`library/设定账本/主角引擎.md`、`library/设定账本/金手指.md`、`library/设定账本/开局契约.md`、`library/文风DNA/文风DNA.md`、`library/设定账本/world-产物索引.md`。
- 缺设定涌现目录或 world 产物索引时，不得把项目阶段推进到 plot。
- `项目总控.md` 只记录正式底盘文件路径，不粘贴长设定。

## 复盘完成

- arc 输出的剧情线、设定账本、活记忆压缩结果，按其指定位置写入。
- 不保留废弃文档，但保留本轮可复盘的过程文件。过程文件不是废弃物，是用户干预和复盘入口。
