# Step 3: LLM 逐章清洗

> **方向**：去除每章中的广告、网站导航、版权声明等非正文内容。
> **核心约束**：仅移除，不增补，不改写。每章独立清洗。

---

## I/O 注解

| 维度 | 内容 |
|:-----|:-----|
| **读什么** | `_temp/chapters/chXXX.txt`（原始分章文件） |
| **做什么** | LLM 逐章扫描，移除所有非正文内容 |
| **产出** | 清洗后的 `_temp/chapters/chXXX.txt`（覆盖写入） |
| **门禁** | 广告未去净 → 退回 |

---

## 操作步骤

### 1. 读取分章文件

逐章读取 `_temp/chapters/ch001.txt` → `chNNN.txt`。

对每章执行以下 LLM 清洗。

### 2. LLM 清洗指令

```
你是一个网文章节清洗助手。你的任务是移除以下非正文内容，**仅保留纯正文**：

需要移除的内容类型（请逐行检查）：
1. **广告** — "下载本书最新txt电子书请点击"、"书籍介绍"、"本站所有txt" 等
2. **网站导航** — "上一页"、"返回目录"、"下一页"、"章节目录" 等
3. **版权声明** — "版权归作者所有"、"本站所有小说" 等
4. **作者说/题外话** — 以"作者说"、"PS"、"题外话" 开头的段落（去除即可，不删前后文）
5. **重复的章节标题** — 如果文件首尾都出现了同一章节标题，保留开头一次
6. **空白行** — 连续超过 2 个空行的缩减为 2 个

清洗规则：
- 不移除正文对话和叙事内容
- 不移除角色名称、地名、技能名等正常内容
- 正文中如果混入广告短句在同一段内，移除该短句，保留正文
- 每一行输出都与原文对应，仅标记移除行的状态

输出格式：逐行输出清洗结果。在文件末尾标注：
```
[清洗统计]
- 移除段落数: N
- 保留段落数: M
- 疑是广告段落（未确认）: K
```

输入文件内容：
{在此粘贴 chXXX.txt 的全文}
```

### 3. 验证清洗结果

每章清洗后验证：

- 正文核心情节是否完整保留
- 是否有明显的广告段落残留
- 角色对话和叙事是否完整

### 4. 门禁检查（通过/退回）

```
□ 每章检查是否有明显的网站域名 / APP 下载链接残留
□ 每章检查是否有"章节导航"类段落残留
□ 每章检查末尾是否有版权声明残留
□ 每章核心情节完整未受损
→ 全部通过 → 进入 Step 4（逐章结构化 JSON）
→ 未通过 → 退回补充清洗
```

---

## 质量红线

| # | 红线 |
|:-:|:-----|
| ❌1 | **广告残留** — 任何包含网站域名、下载链接、APP 推广的内容未移除 |
| ❌2 | **误删正文** — 将正常对话/叙事内容当作广告删除 |
| ❌3 | **章节标题格式损坏** — 修改了章节标题（如将"第一章 觉醒"改为"1. 觉醒"）|
