# CHANGELOG — pop-decon-design-pack

## v1.0.0 — 2026-06-15

### 初始版本

- **定位**：Phase 2 of deconstruction pipeline：逐章 JSON → 章节设计包
- **三步流程**：读 JSON → 填充事件链 → 写入设计包
- **5 条质量红线**：覆盖 Phase 1 前置校验、凭空发明事件、无证据状态变化、抄袭写作话术、产出只留摘要
- **精简设计包格式**：仅含基础信息、登场人物、事件链、事件类型标签——不产出写作端专属字段
- **边界条件**：chapter-data 目录缺失、JSON 数据过少、事件类型模糊、角色出场无变化
