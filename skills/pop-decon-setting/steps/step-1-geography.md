# Step 1：地理蓝图 → L1-01-世界蓝图.md

> **骨架锚点**：本 step 是血肉层，必须先读 `L1-02-力量体系.md` 的境界金字塔，以力量层级为组织原则绘制地图。

**数据来源**：`world-data.json#geography.entries[]` — 所有地名及其出现章节 + `L1-02-力量体系.md` 境界金字塔

**归纳逻辑**：
1. 将 geography 条目按出现章节排序 → 形成地理扩展时间线
2. 按出现频次分组 → 核心区域(≥10次) / 常规区域(2-9次) / 提及区域(1次)
3. 从 Phase 2 的卷地图扩展顺序中提炼"叙事空间节奏"
4. **（骨架锚点）读 L1-02 境界金字塔，将地理区域按力量层级分组——哪些区域是 T0-T2 普通区，哪些是 T4 生肖领地，哪些是 T5/T6 统治区。填写「力量层级空间投影」表格**
5. **（骨架锚点）识别力量真空事件——如生肖被调走导致的空间崩塌，标注 chXX**

**产出**：按 `templates/L1-01-世界蓝图.tpl.md` 填写 → 地理总览 / **力量层级空间投影** / 位面规则 / 核心法则 / 力量来源（与L1-02交叉验证）/ 世界基调
