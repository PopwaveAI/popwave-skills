# Step 12：伏笔与悬念追踪表 → 伏笔与悬念追踪表.md

**数据来源**：Phase 1.5 全书剧情白描 + Phase 2 L2/L3 单元卡 + L1-05 剧情冲突表（迁入）

**归纳逻辑**：
1. 从 Phase 1.5 白描中提取所有"埋下"和"回收"的伏笔/悬念
2. 从 L2/L3 单元卡中提取伏笔标记（如有）
3. 从 L1-05 剧情冲突表迁入冲突项（标注是否已解决+悬念残留）
4. 对每条伏笔标注状态：✅已回收 / ⏳悬念中（部分揭示）/ ❌未回收
5. 对未回收伏笔提出续写接入建议
6. **禁止行为**：原文没有的伏笔 = 不准编造。原文只提了一次但没回收的 = 标注「未回收」

**续写价值**：未回收伏笔是续写的金矿——原著埋了但没填的坑，续写可以直接接入。

**产出**：按 `templates/伏笔与悬念追踪表.tpl.md` 填写
