# Step 14：经典场景拆解 → 经典场景拆解.md

**数据来源**：Phase 2 叙事技法 + L2/L3 单元卡 + 原文

**归纳逻辑**：
1. 从 Phase 2 叙事技法中识别"名场面"候选
2. 从 L2/L3 单元卡中提取有重大转折意义的场景
3. 对每个名场面拆解：前置铺垫 → 场景结构（起承转合）→ 为什么有效 → 核心技巧
4. 分析全书场景节奏（名场面分布密度和间隔）
5. 提出续写场景设计建议（在什么节点设计什么类型的场景）
6. **禁止行为**：原文没有的场景 = 不准编造。场景分析必须基于原文实际内容

**与 Phase 2 叙事技法的区别**：叙事技法分析"怎么写的"（写作技巧），经典场景拆解分析"为什么这个场景让人记住"（读者心理+场景设计）。

**续写/同人价值**：续写时复刻场景节奏，同人写作时致敬或变奏。

**产出**：按 `templates/经典场景拆解.tpl.md` 填写
