# Step 7：世界观弹性边界 → 世界观弹性边界.md

> **v3.0.0 改造说明**：原"世界宪法"产出"叙事规律"（基调边界/世界底线/视角节奏），但对续写/同人帮助有限。改造为"世界观弹性边界"，明确"哪些设定能改、哪些不能改"，直接服务续写/同人。

**数据来源**：L1-01~07 全套设定 + 原文

**归纳逻辑**：
1. 遍历 L1-01~07 所有设定维度
2. 对每个维度判定"硬度"：
   - **硬核规则**：改了世界逻辑崩塌 → 不可更改
   - **弹性区域**：可调整但有限度 → 标注可调范围+不可触碰底线
   - **灰色地带**：原文未明确 → 给出续写建议+风险
3. 提取角色行为边界（绝对不会做/可能会做+触发条件）
4. 制定世界观扩展指南（新职业/新种族/新神职的兼容性原则）
5. **禁止行为**：硬度判定必须有原文证据支撑。不能凭感觉标注"不可更改"

**改造前对比**：
- 旧"世界宪法"：回答"这本书的叙事规律是什么" → 对分析有用，对写作帮助有限
- 新"世界观弹性边界"：回答"哪些设定能改哪些不能改" → 直接服务续写/同人

**产出**：按 `templates/世界观弹性边界.tpl.md` 填写（替代原 `世界宪法.md`）
