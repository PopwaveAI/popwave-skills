# Step 2：L3 剧情线追踪

> **方向**：从 L2 单元卡中追踪跨单元持续演进的主题线，产出 L3 剧情线卡。
> **级别覆盖**：前N章 / 全书

---

## 数据来源

- **Step 1 产出的 L2 单元卡**（`设计/L2单元卡/L2-*.md`）— 唯一数据源
- L2 单元卡中的「嵌套子线」标注「延伸」的子线 → L3 剧情线追踪的种子
- L2 单元卡中的「结构分析」表 → L3 阶段推进的事件证据

> **❗ 核心约束**：L3 剧情线从 L2 单元卡中追踪，不从原子事件直接聚类。
> 旧版从 `_temp/baseline-data.json` 的 events[] 直接聚类三条主线的方式已废弃。

## 追踪规则

### 1. 从 L2 单元卡的「延伸子线」提取种子

扫描所有 L2 单元卡的嵌套子线表，提取标注为「延伸」的子线：
```
L2-001 嵌套子线：
  - 子线1：武道修炼线（延伸）→ 种子
  - 子线2：演武令金手指线（延伸）→ 种子
  - 子线3：王成/王印仇怨线（闭合）→ 不追踪（已闭合）
  - 子线4：穿越者心态线（闭合·第一次蜕变）→ 部分追踪（心态线本身是全书级）
```

### 2. 跨单元追踪同一主题的演进

对每个种子，在后续 L2 单元卡中寻找同主题的子线或事件：
- 同一承压者/压力源持续施压 → 同一条 L3 线
- 压力源变化但主题延续（如"复仇"→"复仇后的反思"→"新的复仇目标"）→ 同一条 L3 线
- 主题完全切换 → 新的 L3 线

### 3. 允许自定义剧情线

**取消旧版「3条固定主线（世界危机/主角成长/主角行动）」限制。**

L3 剧情线的数量和类型由原文结构决定，不由模板预设：
- 如果原文只有 2 条持续演进的线 → 只产出 2 条 L3 线卡
- 如果原文有 6 条交织的线 → 产出 6 条 L3 线卡
- 线型分类：主线 / 支线 / 设定线 / 背景线 / 关系线 / 成长线 / 反派线

### 4. 阶段推进标注「承载单元」

L3 剧情线的每个阶段必须标注它由哪个 L2 单元承载：
```
L3-001 · 武道修炼线
  阶段1（触发）：承载单元 L2-001（ch001-ch011）— 穿越附身，获得演武令
  阶段2（推进）：承载单元 L2-002（ch012-ch025）— 入霍山蛰伏修炼
  阶段3（突破）：承载单元 L2-003（ch026-ch040）— 首次穿越副本世界
  ...
```

> **❗ 红线**：L3 的阶段推进必须引用 L2 单元编号。不得跳过 L2 直接从事件聚类。

## L3 剧情线识别信号

| 信号 | 说明 |
|:-----|:-----|
| 有承压者/行动者 | 这条线围绕谁被逼迫、诱惑、误导或推动 |
| 有持续压力源 | 不是一次性敌人，而是制度、反派、资源缺口、关系债务或认知缺口持续施压 |
| 有阶段变化 | 触发、推进、复杂化、反转/代价、高潮、收束至少 4 段清晰 |
| 有叙事债务 | 前文递送的信息/关系/规则/道具/选择，后文必须回收 |
| 能脱离单元复用 | 不依赖原书固定章数和单元名，也能给新书 plot 提供结构参考 |

## 产出

| 文件 | 内容 |
|:-----|:-----|
| `设计/L3剧情线/L3-001-{线名}.md` | 按 `templates/L3-剧情线.tpl.md` 填写 |
| `设计/L3剧情线/L3-002-{线名}.md` | ...（覆盖全部已识别剧情线） |

每个 L3 剧情线卡必须包含：
- **物理坐标**：时间/地点/人物/力量跃迁/世界冲突质变
- **核心驱动力**：承压者/压力源/行动方向/失败代价/终局变化
- **阶段推进**：阶段/承载单元（L2编号）/事件brief/不可逆变化/读者收益
- **起点/终点快照**：维度/线起点/线终点/变化幅度
- **叙事债务与回收**：债务/前置递送/为什么必须回收/回收方式
- **与其他线的咬合**：关联线/咬合方式/本线给对方什么/对方给本线什么

## 红线

| # | 红线 |
|:-:|:-----|
| 1 | **L3 阶段推进必须引用 L2 单元** — 每个阶段标注「承载单元」（L2 编号），不得跳过 L2 直接从事件聚类 |
| 2 | **不从原子事件直接聚类** — 数据源是 L2 单元卡，不是 `_temp/baseline-data.json` 的 events[] |
| 3 | **不编造剧情线** — 线来自 L2 单元卡中「延伸」子线的跨单元追踪，不是 agent 自拟 |
| 4 | **缺核心驱动力不入库** — 没有压力源和失败代价的线不能服务写书 |
| 5 | **缺叙事债务不入库** — 没有递送与回收，说明它不是长期剧情线 |
| 6 | **禁止多线混合** — 一条主线/支线/设定线/关系线单独成文 |
