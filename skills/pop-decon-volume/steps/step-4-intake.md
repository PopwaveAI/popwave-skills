# Step 4：L2卡+卷纲双轨入库

> **前置**：Step 1-3 全部完成（L2 单元卡 × 卷纲 × 跨卷主题线均已产出）
> **方向**：从完整的结构分析中，提取可迁移的 L2 单元卡和卷纲，入库到公共剧情库。
> **产出路径**：`写作资产/剧情库/{标签}/{书}-L2-{编号}-{名称}.md` + `...-卷纲-{卷号}.md`

## 为什么需要双轨入库

公共 `剧情库` 服务写书侧，需要两种粒度的可迁移资产：

| 轨道 | 粒度 | 服务场景 | 入库条件 |
|:-----|:-----|:---------|:---------|
| **L2 轨道** | 单元级（一个完整行动弧） | 为新书 plot 提供单元级结构参考（如"蛰伏-突破-因果-余波"四拍子） | 有可迁移的结构模式 + 有可迁移的情感设计 |
| **卷纲轨道** | 卷级（含溯源燃料台） | 为新书 plot 提供卷级设计参考（如卷级幕序列+溯源来源） | 有溯源燃料台（4类溯源完整）+ 有幕序列含L2卡映射 |

> 核心原则：L2 入可迁移单元，卷纲入可迁移卷级结构。原书切片（不可迁移部分）不入库。

## 区分「原书切片」与「可迁移资产」

### 原书切片（不入库）

| 类型 | 说明 |
|:-----|:-----|
| 原书专属设定 | 少林寺九图六坐像身法、阿罗汉神功、演武令的具体机制 |
| 原书专属角色 | 王成/王印（复仇对象）、澄智（师父）、封羽（追捕者） |
| 原书专属因果 | 前身被王印害死的具体情节 |
| 原书专属篇幅 | 章数、幕数、卷长（写入不可迁移部分，不作为入库主资产） |

### 可迁移资产（入库）

| 类型 | 说明 |
|:-----|:-----|
| L2 结构模式 | 蛰伏-突破-因果-余波 四拍子（可迁移到任何"新手村"场景） |
| L2 情感设计 | 压抑→爽→冷→反思（可迁移到任何"复仇后反思"场景） |
| L2 节奏控制 | 5慢+1爆+1收（可迁移到任何"积累→爆发→收束"周期） |
| L2 嵌套策略 | 4条子线，2闭合2延伸（可迁移到任何多线交织的单元） |
| 卷纲幕序列 | 幕的功能位+主爽点+L2卡映射（可迁移到任何同类型卷） |
| 卷纲溯源燃料台 | 原文段落→猜测来源→置信度（为写书提供灵感来源参考） |
| 卷纲舞台人物表 | 7列人物表（可迁移到任何同类型卷的角色设计） |

## L2 入库流程

### 1. 筛选可迁移 L2 单元

扫描所有 L2 单元卡，筛选满足以下条件的单元：
- 有可迁移的结构模式（非原书专属）
- 有可迁移的情感设计（非原书专属）
- 有可迁移的节奏控制（非原书专属）
- 有至少 1 条可迁移的嵌套子线策略

### 2. 提取可迁移部分

从 L2 单元卡中提取可迁移要点，剥离原书专属内容：
```
原 L2-001 单元卡：
  可迁移：蛰伏-突破-因果-余波 四拍子结构
  可迁移：压抑→爽→冷→反思 情感设计
  可迁移：5慢+1爆+1收 节奏控制
  可迁移：4条子线 2闭合2延伸 嵌套策略
  不可迁移：少林寺设定、王成/王印角色、演武令机制
```

### 3. 入库

将可迁移部分写入剧情库：
```
写作资产/剧情库/{标签}/{书}-L2-{编号}-{名称}.md
```

标签根据单元类型分配：成长型/对抗型/探索型/关系型/建置型/复合型

## 卷纲入库流程

### 1. 筛选可迁移卷纲

扫描所有卷纲，筛选满足以下条件的卷纲：
- 有溯源燃料台（4类溯源完整：剧情/设定/创意/质感）
- 有幕序列含 L2 卡映射
- 有舞台人物7列表
- 能脱离原书复用（幕序列的功能位和主爽点具有通用性）

### 2. 提取可迁移部分

从卷纲中提取可迁移要点，剥离原书专属内容：
```
原 卷1-卷纲.md：
  可迁移：幕序列的功能位+主爽点+L2卡映射
  可迁移：溯源燃料台（原文段落→猜测来源→置信度）
  可迁移：舞台人物7列表的结构
  可迁移：事件种子池的结构
  不可迁移：少林寺设定、王成/王印角色、演武令机制
```

### 3. 入库

将可迁移部分写入剧情库：
```
写作资产/剧情库/{标签}/{书}-卷纲-{卷号}.md
```

标签根据卷的类型分配：成长型/对抗型/探索型/关系型/建置型/复合型

## 红线

| # | 红线 |
|:-:|:-----|
| 1 | **禁止原书切片入库** — 原书专属设定/角色/因果/篇幅不入公共库 |
| 2 | **禁止照搬原书篇幅** — 章数、幕数、卷长必须写入「不可迁移部分」 |
| 3 | **L2 入库必须有可迁移结构模式** — 没有可迁移结构模式的 L2 卡不能服务写书 |
| 4 | **卷纲入库必须有溯源燃料台** — 没有溯源燃料台的卷纲不入库 |
| 5 | **L2卡/卷纲双轨分离** — L2 单元和卷纲分别入库，不合并为同一文件 |
| 6 | **卷纲入库幕序列必须含 L2 卡映射** — 入库卷纲的幕序列必须保留对应L2卡列 |
