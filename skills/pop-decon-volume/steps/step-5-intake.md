# Step 5：L2/L3 双轨入库

> **前置**：Step 1-4 全部完成（L2 单元卡 × L3 剧情线 × L4 全书事件 × 套路聚合均已产出）
> **方向**：从完整的结构分析中，提取可迁移的 L2 单元卡和 L3 剧情线卡，入库到公共剧情库。
> **产出路径**：`写作资产/剧情库/{标签}/{书}-L2-{编号}-{名称}.md` + `...-L3-{编号}-{名称}.md`

## 为什么需要双轨入库

公共 `剧情库` 服务写书侧，需要两种粒度的可迁移资产：

| 轨道 | 粒度 | 服务场景 | 入库条件 |
|:-----|:-----|:---------|:---------|
| **L2 轨道** | 单元级（一个完整行动弧） | 为新书 plot 提供单元级结构参考（如"蛰伏-突破-因果-余波"四拍子） | 有可迁移的结构模式 + 有可迁移的情感设计 |
| **L3 轨道** | 线级（跨单元持续演进） | 为新书 plot 提供线级设计参考（如"穿越者心态蜕变线"的完整阶段推进） | 有核心驱动力 + 有叙事债务与回收 |

> 核心原则：L2 入可迁移单元，L3 入可迁移线。原书切片（不可迁移部分）不入库。

## 区分「原书切片」与「可迁移单元」

### 原书切片（不入库）

| 类型 | 说明 |
|:-----|:-----|
| 原书专属设定 | 少林寺九图六坐像身法、阿罗汉神功、演武令的具体机制 |
| 原书专属角色 | 王成/王印（复仇对象）、澄智（师父）、封羽（追捕者） |
| 原书专属因果 | 前身被王印害死的具体情节 |
| 原书专属篇幅 | 章数、幕数、卷长（写入不可迁移部分，不作为入库主资产） |

### 可迁移单元/线（入库）

| 类型 | 说明 |
|:-----|:-----|
| 结构模式 | 蛰伏-突破-因果-余波 四拍子（可迁移到任何"新手村"场景） |
| 情感设计 | 压抑→爽→冷→反思（可迁移到任何"复仇后反思"场景） |
| 节奏控制 | 5慢+1爆+1收（可迁移到任何"积累→爆发→收束"周期） |
| 嵌套策略 | 4条子线，2闭合2延伸（可迁移到任何多线交织的单元） |
| 核心驱动力 | 承压者/压力源/行动方向/失败代价/终局变化（可迁移到任何同类型线） |
| 阶段推进 | 触发→推进→复杂化→反转→高潮→收束（可迁移到任何长期剧情线） |

## L2 入库流程

### 1. 筛选可迁移 L2 单元

扫描所有 L2 单元卡，筛选满足以下条件的单元：
- 有可迁移的结构模式（非原书专属）
- 有可迁移的情感设计（非原书专属）
- 有可迁移的节奏控制（非原书专属）
- 有至少 1 条可迁移的嵌套子线策略

### 2. 提取可迁移部分

从 L2 单元卡中提取可迁移要点，剥离原书专属内容：
```
原 L2-001 单元卡：
  可迁移：蛰伏-突破-因果-余波 四拍子结构
  可迁移：压抑→爽→冷→反思 情感设计
  可迁移：5慢+1爆+1收 节奏控制
  可迁移：4条子线 2闭合2延伸 嵌套策略
  不可迁移：少林寺设定、王成/王印角色、演武令机制
```

### 3. 入库

将可迁移部分写入剧情库：
```
写作资产/剧情库/{标签}/{书}-L2-{编号}-{名称}.md
```

标签根据单元类型分配：成长型/对抗型/探索型/关系型/建置型/复合型

## L3 入库流程

### 1. 筛选可迁移 L3 剧情线

扫描所有 L3 剧情线卡，筛选满足以下条件的线：
- 有完整的核心驱动力（承压者/压力源/行动方向/失败代价/终局变化）
- 有完整的阶段推进（至少 4 段）
- 有叙事债务与回收
- 能脱离原书单元复用

### 2. 提取可迁移部分

从 L3 剧情线卡中提取可迁移要点，剥离原书专属内容：
```
原 L3-001 剧情线卡：
  可迁移：武道修炼线的阶段推进模式
  可迁移：金手指加速修炼的驱动力结构
  可迁移：修炼膨胀→心态危机→蜕变的内在冲突
  不可迁移：演武令的具体机制、少林寺的修炼体系
```

### 3. 入库

将可迁移部分写入剧情库：
```
写作资产/剧情库/{标签}/{书}-L3-{编号}-{名称}.md
```

标签根据线型分配：主线/支线/设定线/背景线/关系线/成长线/反派线

## 红线

| # | 红线 |
|:-:|:-----|
| 1 | **禁止原书切片入库** — 原书专属设定/角色/因果/篇幅不入公共库 |
| 2 | **禁止多线混合** — 一条 L3 线单独成文，不混合多条线 |
| 3 | **禁止照搬原书篇幅** — 章数、幕数、卷长必须写入「不可迁移部分」 |
| 4 | **缺核心驱动力不入库** — 没有压力源和失败代价的 L3 线不能服务写书 |
| 5 | **缺叙事债务不入库** — 没有递送与回收，说明它不是长期剧情线 |
| 6 | **L2/L3 双轨分离** — L2 单元和 L3 线分别入库，不合并为同一文件 |
