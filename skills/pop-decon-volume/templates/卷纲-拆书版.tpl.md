> 融合六维拆文法维度4（正常时间线）+维度5（成品剧情节奏）的分析能力。

### 7.1 时间线对照

将本卷事件按**真实发生先后顺序**排列，与成品章节顺序对照：

| # | 真实时间线事件 | 对应章 | 成品顺序位置 | 是否打乱 | 打乱类型 |
|:-:|:-------------|:------:|:-----------:|:--------:|:---------|
| 1 | {真实最早发生的事件} | chXXX | 第N章 | 否 | — |
| 2 | {真实第二早的事件} | chXXX | 第M章 | 是 | 提前/延后/倒叙/插叙 |
| ... | | | | | |

**打乱类型定义**：
- **提前**：真实时间晚但成品排前 → 制造悬念
- **延后**：真实时间早但成品排后 → 制造反转
- **倒叙**：整段倒序讲述
- **插叙**：在主线中插入过去的事件片段
