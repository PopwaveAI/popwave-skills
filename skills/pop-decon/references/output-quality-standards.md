# 拆书产出物质量标准

> **来源**: 2026-06-15 深渊主宰第一卷(187章)全量拆解后质量评估
> **用途**: 每个 Phase 完成后，agent 必须对照此表自检，不达标不准进下一 Phase

---

## 一、质量标准总表

| Phase | 产出物 | 最低标准 | 良好标准 | 优秀标准 |
|:------|:-------|:---------|:---------|:---------|
| 1 | 设计包 | 每章≥3事件，事件有类型分类+TAG，情绪节拍完整 | + 每事件有原文证据引用 | + 每事件含感官锚点+对话序列 |
| 2 | L2单元卡(L2-*.md) | 结构分析≥4字段 + 每章7字段 | + 有结构分析+副线活跃度 | + 有紧张曲线(1-10)+章节类型分类 |
| 2 | 卷纲(volume-*.md) | 3条主线各≥5阶段 | + 有分支线跟踪表 | + 有伏笔-回收对照表+未解之谜 |
| 2 | 全书架构.md | ≥40行，每幕≥3行描述 | + 有主题宣言+结构角色 | + 有剧情曲线+多线交织图 |
| 3 | L1世界观六件套 | 覆盖率≥80%，每条目有chXX证据 | + 跨文档一致性校验通过 | + 有ch05X类占位符零容忍 |

## 二、各产出物详细标准

### 2.1 Phase 1 — 设计包

**当前已知缺陷（深渊主宰评估，2026-06-15）**：
- 事件粒度：前4包良好(5-9事件/章)，第5包起退化(格式丢失+事件类型分类缺失)
- 动作序列：有但缺节奏感和肢体感受
- **对话：严重不足** — 仅摘1-2句引文，无完整对话序列，无语气/语调标注
- **感官锚点：致命缺口** — 无系统化视觉/听觉/嗅觉/触觉记录
- POV/文风指引：无叙事视角/内心独白密度标注

**最低标准**(L1)：
- 每章≥3个原子事件
- 每个事件有类型分类（交锋/社交/探索/成长/披露/转折/日常）
- 情绪节拍分段完整（起点→推进→高潮→终点或等价的子段）
- 智慧实体状态轨迹每5章聚合一次

**良好标准**(L2，在L1基础上追加)：
- 每个事件的"证据"字段指向具体原文行或段落
- 关键数据锚点（属性面板、技能数值、装备属性）完整保留
- 每章有章末钩子类型标注（悬念/信息/情绪）

**优秀标准**(L3，在L2基础上追加)：
- 每事件含 ≥3个感官细节（场景看/听/嗅/触至少2种维度）
- 关键对话场景保留 ≥3句连续对话，标注说话人语气
- 每章有叙事视角标注（POV角色 + 内心独白密度）
- 战斗章节有节奏标记（快写段落 / 慢写段落 / 中速对话段）

### 2.2 Phase 2 — L2单元卡(L2-*.md)

**当前已知缺陷（深渊主宰评估）**：
- **幕级元数据完全缺失** — 5个act文件全部只有章级字段，无幕级信息
- 副线在YAML中零体现（56%章节只填了3条主线，其余4+条副线无字段记录）
- 无紧张曲线量化等级
- 无章节类型分类

**最低标准**(L1) — **每章字段**：
| 字段 | 必要性 | 说明 |
|:-----|:------:|:------|
| chapter | ✅ 必需 | 章号 |
| title | ✅ 必需 | 章标题 |
| main1_world | ✅ 必需 | 世界危机线进展 |
| main2_growth | ✅ 必需 | 主角成长线进展 |
| main3_action | ⚠️ 建议 | 守护/行动线进展 |
| emotional_goal | ✅ 必需 | 本情绪目标（如：紧张/压抑/轻松） |
| end_hook | ✅ 必需 | 章末钩子类型+内容 |
| payoff_summary | ✅ 必需 | 本爽点/信息释放 |
| setting_release | ⚠️ 建议 | 本设定/世界观增量 |

**良好标准**(L2) — **追加幕级元数据**：
| 字段 | 必要性 | 说明 |
|:-----|:------:|:------|
| dramatic_function | ✅ 必需 | 建置/上升/中点/高潮/余波 |
| emotional_arc | ✅ 必需 | 本幕从什么情绪到什么情绪 |
| opening_state | ✅ 必需 | 本幕开始时主角力量/关系/认知三轴 |
| closing_state | ✅ 必需 | 本幕结束时三轴变化 |
| core_conflict | ✅ 必需 | 本幕核心冲突（如：生存vs死亡） |
| key_decision_point | ⚠️ 建议 | 主角在本幕做出的关键选择 |
| subplots_active | ✅ 必需 | 本幕活跃的副线清单 |
| word_count_planned | ⚠️ 建议 | 计划字数 |
| word_count_actual | ⚠️ 建议 | 实际字数（后续更新） |

**优秀标准**(L3) — **追加量化分析**：
- tension_curve: 每章1-10紧张值数组（如 [2,3,4,5,3,4,6,7,4,5]）
- chapter_type_distribution: 章节类型统计（action/exposition/emotional/turning_point各多少章）

### 2.3 Phase 2 — 卷纲(volume-*.md)

**当前已知缺陷（深渊主宰评估）**：
- 三大剧情线仅罗列等级变动阶段，缺人格成长轨迹
- 分支线完全未统计（一卷有7条活跃副线）
- 无剧情密度/节奏分析
- 无卷末未解之谜提示

**最低标准**(L1)：
- 3条主线各≥5个阶段，每阶段≥2行描述
- 每个阶段标注起止章号
- 本卷总字数/总章数统计

**良好标准**(L2，在L1基础上追加)：
- **分支线跟踪表**：每条副线记录 起止章/活跃状态/活跃章数/所属幕
- 主角阵营漂移轨迹（关键选择→阵营变化）
- 情感节奏表：每幕的情感基调

**优秀标准**(L3，在L2基础上追加)：
- 伏笔-回收对照表：本卷埋了多少根线，收了多少根，留了多少根
- 未解之谜清单：卷末遗留悬念（至少明确列出）
- 角色出场密度统计：主要角色在各幕的出场权重

### 2.4 Phase 2 — 全书架构

**当前已知缺陷**：
- 仅13行/848字节，每幕仅一句话概述（~30字/幕）
- 无顶层架构判断（核心主题弧/多卷传承/结构定位）
- 无剧情密度评估（爆发幕vs铺垫幕）

**最低标准**(L1)：
- ≥3行/幕的描述（幕名+章范围+核心事件摘要）
- 每幕标注字数/章数

**良好标准**(L2，在L1基础上追加)：
- 全书主题宣言：一句话+三段式展开
- 各幕的结构角色（第一幕建置/中点转折/终局高潮）
- 副线分布概览

**优秀标准**(L3，在L2基础上追加)：
- 全剧紧张曲线（低→高→低→更高的数值曲线）
- 多卷预留接口（本卷终点与下一卷起点的连接点）
- 每幕核心问题（dramatic question）

### 2.5 Phase 3 — L1世界观六件套

**当前状态（深渊主宰评估）**：✅ 优秀级别，覆盖率、准确性、跨文档一致性均表现良好。

**门禁检查点**：
| 检查项 | 说明 |
|:-------|:------|
| ✅ 全部条目有chXX证据出处 | 必须执行 |
| ✅ 无编造内容 | 跨文档一致性抽查≥3项 |
| 🔴 无ch05X类占位符 | 零容忍——发现即退回 |
| 🔴 无问号存疑名未处理 | 命运女神? 类必须修复或标注脚注 |
| ⚠️ prose-render可消费性 | 建议增加世界观总纲作为入口文档 |

## 三、跨产出物一致性校验

不同Phase的产出物之间数据可能冲突，必须在全部产出后做至少3项交叉校验：

| 校验 | 源1 | 源2 | 失败后果 |
|:-----|:----|:----|:---------|
| 主角起始属性 | L1-02力量体系 | 起点终点快照 | prose-render用错数据 |
| 等级体系 | L1-02力量体系 | combat_capability | 战斗数值不一致 |
| 重要物品归属 | L1-06资源与物品 | design-pack事件链 | 物品-角色归属错误 |
| 核心角色名称 | 所有产出 | 所有产出 | 名称不一致→穿帮 |
| 卷-幕-章关系 | 全书架构 | volume/L2单元卡 | 章节归属错位 |
