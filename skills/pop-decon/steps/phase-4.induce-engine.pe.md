# Phase 4：归纳故事引擎

> **方向**：逆 `creative` Phase 0。从所有已归纳的数据中提炼——这本书的"核心假说"是什么。
> **级别覆盖**：仅 Lv3（全书）
> **前置条件**：Phase 1 + Phase 2 + Phase 3 全部完成
> **Lv1/Lv2 不执行此 Phase** — 故事引擎是归纳的终点，不是起点。数据不足时归纳 = 编造。

---

## 🚨 为什么故事引擎是最后一个 Phase

```
写作方向：故事引擎 → L1设定 → 卷大纲 → 事实骨架 → 正文
         "我想写什么"                                   "我写了什么"

拆书方向：正文 → 事实骨架 → 卷大纲 → L1设定 → 故事引擎
         "原文写了什么"                                   "所以这书是什么"

故事引擎不是分析的第一步，而是全文分析的终点。
只有走完了 Phase 1-3（事实→结构→世界观），才有资格回答"这本书是什么"。
```

**反例（当前错误做法）**：
- 看完前 20 章就写「故事引擎.md」→ 写出"守序邪恶的少年坚定地走向黑暗"
  → 这句是对的，但它其实来自简介（元数据），不是来自归纳
  → 真正的故事引擎应该能从全文数据中推导出来，而不是复述简介

## 步骤详解

### Step 1：提炼核心假说 → 「一、这本书是什么」

**数据来源**：

| 要归纳的 | 取什么数据 | 怎么归纳 |
|:---------|:----------|:---------|
| **一句话** | Phase 2 Canvas 三条主线的推进模式 + Phase 3 世界宪法 | 从三条主线的本质冲突中提取"这本书最核心的一句话设定" |
| **碰撞** | 世界宪法 + 角色卡四层驱力 | 世界的规则 × 主角的规则 = 在哪互否？ |
| **世界锚点** | L1-01 世界蓝图 + L1-03 历史驱动力 | 时代/地域/力量层级/与常见设定最大不同 |
| **主角** | 角色卡（Lv4 九节完整） | 从九节中提取核心段落：站在碰撞点哪一侧？要什么？怕什么？ |

**归纳规则**：
```
✅ "核心假说：一个带着前世 DND 传奇盗贼记忆的穿越者，在诸神陨落的动荡之年，从贫民区走向深渊" → 来自 ch2 面板+ch3 动荡之年设定
❌ "核心假说：守序邪恶的少年坚定地走向黑暗" → 复述简介，不是归纳
```

### Step 2：提炼卖点+读者 → 「二、读者为什么追」

**从原文前三章事件摘要中回答**：
- **核心爽点**：前三章做了什么让读者必须看下去？（不是"升级爽"—是具体的情感命题）
- **对标书差异**：与同赛道作品比，这本书的差异是什么？
- **前三章承诺**：读者看完前三章应该期待什么？

### Step 3：归纳宪法 → 「三、绝对不能做的事」

**从 Phase 3 世界宪法 + 角色卡设计目的中提取**：
- 基调边界：从叙事底线反推
- 主角边界：从角色卡"绝不会做"反向提取
- 世界底线：从世界宪法提取
- 叙事底线：从 chapter-index 的节奏规律反推

### Step 4：确认开篇 → 「四、第一页」

**从 Phase 1 ch1 事件摘要 + 原文首段**：
- 开篇画面：ch1 前 3 段是什么？（原文引用）
- 签名画面：全书最标志性的一个画面是什么？（从 canvas 高潮章中找）

### Step 5：确认结构预期 → 「五、结构预期」

**从 Phase 2 全书架构 + 卷起点/终点快照**：
- 每卷的状态 → 核心追问

## ❌ 质量红线

| # | 标准 |
|:-:|:-----|
| ❌1 | **所有前序 Phase 产出必须存在** — Phase 1/2/3 任一缺失 → 退回。故事引擎不能凭空归纳 |
| ❌2 | **核心假说必须能从 Canvas 主线和世界宪法中推导** — 不能复述简介/标题/书评 |
| ❌3 | **每个断言必须能追溯到前序产出的某个具体段落** — 标注 `@Phase{N}-XX` 来源 |
| ❌4 | **不准把"我们认为"写成"作者意图"** — 归纳是"从数据看本书呈现了什么样的结构"，不是"作者想表达什么" |
| ❌5 | **结构预期必须与 Phase 2 全书架构一致** — 卷数/起止状态不能矛盾 |
| ❌6 | **产出只留摘要** — 写入后对话中不粘贴全文。格式：「已写入 {路径}。核心假说：{一句话}」 |

## 归纳纪律

```
故事引擎归纳 = 最高级别的归纳。不允许任何文本中没有证据的断言。

每次写一段前，执行：
1. 这段的核心论断是什么？
2. 这个论断来自前序哪个产出的哪一段？
3. 标注 @Phase{N} 来源
4. 如果找不到来源 → 不写

故事引擎的可回源性：
  核心假说 ← Canvas 主线 + 世界宪法
  碰撞     ← 世界宪法 × 角色卡驱力
  卖点     ← 前 3 章事件 + 全卷情绪曲线
  宪法     ← 世界宪法 + 叙事规律
  结构预期 ← 全书架构 + 卷快照
```
