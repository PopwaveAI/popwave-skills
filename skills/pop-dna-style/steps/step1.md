# Step 1: 场景定向采样

> 从参考书全书中按场景类型定向采样25-30章，为笔触DNA提取准备覆盖全场景类型的样本。

---

## 采样目标

- **总采样量**：25-30章
- **场景类型覆盖**：≥8种（9种场景类型中至少选8种）
- **采样原则**：按场景类型定向选章，禁止用"前10+中5+后5"固定时间分段

---

## 采样流程

### 1.1 解析全书章节列表

读取参考书txt文件，解析章节列表（章号+章名）。

**中文小说章节识别**：匹配 `第\d+章` / `第[一二三四五六七八九十百千]+章` 等格式。

产出中间结果：`章号 | 章名` 列表。

### 1.2 章名关键词初分类

按9种场景类型，用章名关键词做初步分类：

| 场景类型 | 章名关键词线索 | 默认采样量 |
|:--|:--|:--|
| 日常对话 | 叙旧/来访/闲谈/夜话/茶/酒 | 2-3章 |
| 环境引入 | 初到/抵达/新地/入城/进入 | 2-3章 |
| 早期遭遇战 | （前30章内的战斗章） | 2-3章 |
| 中坚boss战 | 单挑/对决/招式名对抗 | 2-3章 |
| 群战/军队战 | 围杀/战场/军阵/万军/千军 | 2-3章 |
| 暗杀/背刺 | 刺杀/潜行/暗影/偷袭/暗算 | 2-3章 |
| 揭秘/心理戏 | 真相/发现/回忆/秘密/心魔 | 2-3章 |
| 过渡缓冲 | 战后/清晨/赶路/休整/疗伤 | 2-3章 |
| 情感高潮 | 告别/重逢/告白/泪/死/牺牲 | 2-3章 |

### 1.3 不确定章确认

对分类不确定的章，读前50行确认场景类型：
- 看前50行有没有明显的场景标志（战斗描写/对话密度/环境铺陈/内心独白）
- 不确定的章先标"待定"，最后统一归位

### 1.4 采样清单生成

每种场景类型选2-3个代表章（优先选该场景类型的典型章，避免边缘章）。

汇总采样清单，格式：

```markdown
# 采样清单

| 章号 | 章名 | 场景类型 | 采样理由 |
|:--|:--|:--|:--|
| ch003 | 初入山门 | 环境引入 | 新场景首出场，典型环境铺陈 |
| ch012 | 夜话旧事 | 日常对话 | 章名含"夜话"，典型闲谈章 |
| ... | ... | ... | ... |
```

**门禁**：
- 场景类型覆盖 ≥8 种
- 总采样章数 25-30 章
- 每种场景类型至少2章

门禁不通过 → 补采样到达标。

---

## 边界条件

| 场景 | 处理 |
|:--|:--|
| 参考书章节 < 30 章 | 全书采样，场景类型按实际分布选 |
| 参考书章节 > 500 章 | 只采样前300章（覆盖前期成长线） |
| 章名无关键词线索（如纯数字章名） | 间隔采样（每20章抽1章）+ 读前50行分类 |
| 某场景类型全书都没有 | 跳过该类型，在采样清单标注"本书无此场景"，总覆盖数仍需≥8 |

---

## 产出

- **采样清单**（25-30章，覆盖≥8种场景类型）
- 传递给 Step 2 做笔触DNA提取
