# Step 2: 笔触DNA提取（档位A·必做）

> 对采样清单的25-30章，按 pop-shared-dna v4模板提取笔触DNA。这是必做档位，产出 write skill 的笔触层供给。

---

## 提取目标

- **通用维度**：≥4个（从8个必选维度中选）
- **场景卡矩阵**：≥8个（覆盖采样清单的主要场景类型）
- **全书稳定特征**：2条
- **每个维度/场景卡格式**：观察(1-2句) + 原文(≥500字) + 时间演变(1-2句)，三者不可缺一

---

## v4模板格式

### 通用维度卡格式

```markdown
### 维度：{维度名}

**观察**：（1-2句描述该维度的写作风格特征，如"短句为主，平均每段2-3行，对话段独立成段"）

**原文样本**：（≥500字原文摘录，体现该维度特征）

**时间演变**：（1-2句描述全书该维度是否有演变，如"前期短句密集，后期逐渐出现长段叙事"）
```

### 场景卡格式

```markdown
### 场景卡：{场景类型}

**观察**：（1-2句描述该场景类型的笔触特征）

**原文样本**：（≥500字原文摘录）

**时间演变**：（1-2句描述该场景类型在全书的笔触演变）
```

---

## 提取流程

### 2.1 通用维度提取（≥4个必选）

从8个必选维度中选≥4个（优先选该参考书最有特色的）：

| 维度 | 观察什么 |
|:--|:--|
| 叙事者距离 | 近距跟随主角/中距旁观/全知视角？视角切换频率？ |
| 情绪传递 | 直述情绪/动作外化/物象外化/环境烘托？ |
| 对话引导词 | "道"/"说"/省略引导词/动作带出？ |
| 外貌描写 | 什么时候写外貌？写多少？静态铺陈还是动态带出？ |
| 数据嵌入 | 系统面板/数值/属性怎么呈现？内联【】还是独立段？ |
| 环境展开 | 环境描写长度？感官覆盖？功能化还是氛围化？ |
| 段落呼吸 | 段落长度分布？长短交替规律？节奏感？ |
| 物品描写 | 物品命名风格？描写详略？是否功能化？ |

> 每个维度读3-5个采样章，归纳特征。

### 2.2 场景卡矩阵提取（≥8个必选）

按采样清单的场景类型，每种类型做1张场景卡：

**候选场景卡**（按采样覆盖选）：
- 战斗·早期遭遇战
- 战斗·中坚boss战
- 战斗·群战/军队战
- 战斗·暗杀/背刺
- 对话·日常
- 对话·情感高潮
- 对话·信息交换
- 描写·环境引入
- 描写·角色外貌
- 叙述·孤独紧张
- 叙述·过渡缓冲
- 叙述·揭秘/心理戏

> 每张场景卡读该类型的2-3个采样章，归纳该场景的笔触特征。

### 2.3 全书稳定特征（2条必选）

归纳全书贯穿始终的稳定笔触特征，不随场景变化：

例如：
- "全书对话段独立成段，不与动作段混合"
- "全书无内心独白大段，情绪用动作或物象外化"

---

## 笔触DNA红线自检

提取完成后，逐条自检：

| 自检项 | 标准 | 不合格处理 |
|:--|:--|:--|
| v4模板完整性 | 每个维度/场景卡都有 观察+原文≥500字+时间演变 三段 | 补全缺失段 |
| 场景覆盖 | 场景卡矩阵 ≥8 个 | 补充采样和场景卡 |
| 只学笔触不学内容 | 观察字段只描述写作风格，不描述剧情/战斗/角色思维 | 重写观察字段 |
| 原文样本长度 | 每段原文 ≥500字 | 补充摘录 |

> 红线3（只学笔触不学内容）特别重要：观察字段禁止出现"主角用XX技能打败敌人"这类剧情描述，只描述"短句密集+动作动词连续+无心理描写"这类笔触特征。

---

## 产出

- **笔触DNA文档**（通用维度≥4 + 场景卡≥8 + 全书稳定特征2条）
- 传递给 Step 4 部署到 `素材/文风锚定.md`
- 如启用档位B，传递给 Step 3 做剧情DNA brief提取
