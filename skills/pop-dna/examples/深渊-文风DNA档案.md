# 文风DNA档案

> 目标作者/书籍：诸生浮屠 · 《深渊主宰》
> 样本来源：CH1-CH3
> 分析日期：2026-06-05
> 分析工具：pop-dna v1.0（7维分析框架）

## 一、风格摘要

冷峻克制的西幻生存叙事。叙事者保持克制距离，不跳出来评价角色。苦难和恐惧全部压在细节里让读者自己感受。情感外化为动作和身体反应，不做内心直述。世界观通过日常叙事自然带出，不做教材式说明。温情高光和外部威胁高频交替，不让读者在单一情绪里待太久。

**一句话定位**：冷峻克制、动作外化、留白信任读者的西幻生存叙事

## 二、7维分析详情

### 维度1：词汇与语料

**分析结果**：

| 检查点 | 发现 |
|--------|------|
| 词汇丰富度 | 中等偏口语化。很少生僻词，但用词精准 |
| 特色词库 | "缩进""抽泣""决然""打杀""低语"等动作+声音相关词 |
| 语气标记 | "了"字使用中等。感叹号偶用。问号极少 |
| 否定/肯定偏好 | 以肯定陈述为主。几乎不用"不是A而是B"否定结构 |

**原文证据**：
```
"她慌忙缩回了手，抱紧了怀里的老狗。"
"碰到后背上的伤口，不由疼得抽泣了一下。"
```

**量化指标**：
- 平均每千字生僻词数：3-5
- 感叹号密度：1-2次/千字
- "了"字密度：15-20次/千字

### 维度2：句式与节奏

**分析结果**：

| 检查点 | 发现 |
|--------|------|
| 句长分布 | 混合。中句为主，情绪高点转短句 |
| 句式变化 | 主谓宾简单结构为主。偶有插入修饰语 |
| 排比/重复 | 几乎不用排比 |
| 标点偏好 | 句号为主。逗号用于短分句连接。偶用破折号做插入 |

**原文证据**：
```
"她坚强地擦去泪水。"
"眼角露出了一丝决然，俯身轻轻吻了吻老狗的额头。"
```

**量化指标**：
- 平均句长：18-25字
- 最短句长：4字 | 最长句长：45字
- 短句(<15字)占比：30%
- 中句(15-30字)占比：50%
- 长句(>30字)占比：20%

**句式特征示例**：
- 短句做情绪收束："恐惧。"
- 动作串接用逗号分隔："她慌忙缩回了手，抱紧了怀里的老狗。"
- 细节补充用破折号："生火对她来说很困难——她今年才八岁。"

### 维度3：叙事者姿态

**分析结果**：

| 检查点 | 发现 |
|--------|------|
| 侵入频率 | 中低度侵入。每千字约2-3次 |
| POV 距离 | 近距到零距离交替。紧贴角色认知但不跳入内心 |
| 因果关系 | 叙事者偶尔主动解释（"她今年才八岁"），大部分留给读者 |
| 情绪标签 | 偶用直接情绪标签（"恐惧""决然"），但更多通过动作暗示 |

**原文证据**：
```
叙事者侵入示例：
"她今年才八岁，生火对她来说很困难。"（解释因果关系）
"它有这样的资格。"（评价）

动作暗示示例（无标签）：
"缩进了被窝里面。"（→恐惧）
"眼角露出了一丝决然。"（→决意）
```

**叙事者侵入分类统计**（取样3000字）：

| 侵入类型 | 次数 | 示例 |
|---------|------|------|
| 解释因果关系 | 2 | "她今年才八岁""它有这样的资格" |
| 评价角色行为 | 1 | "她坚强地" |
| 直接情绪标签 | 2 | "恐惧""决然" |
| 预告未来 | 0 | — |

### 维度4：信息释放策略

**分析结果**：

| 检查点 | 发现 |
|--------|------|
| 释放节奏 | 层层递进 + 埋伏笔 |
| 设定密度 | 中高。前三章释放了全书级信息量 |
| 悬念构造 | 信息差（读者不知道门外是什么）+ 力量差距 + 道德困境 |
| 章末钩子 | 悬念截断 / 情绪总结交替使用 |

**原文证据**：
```
"脚步声由远及近，停在门外。"
——悬念建立，不解释谁来了

"这个世界，远比她想象的要大。"
——世界观预告，埋伏笔
```

**信息释放模式识别**：
- 开篇500字内的新概念引入数：3（琥珀城、贫民区、人贩子）
- 新概念首次出现后的解释方式：**场景带出为主**，不单独做说明段
- 10%位置新概念数：4 | 50%位置：7 | 90%位置：10

### 维度5：对话策略

**分析结果**：

| 检查点 | 发现 |
|--------|------|
| 对话密度 | 低（<10%） |
| 引导词风格 | 极简——"XX道"或动作+对白 |
| 对话功能 | 释放情绪 > 推进信息 |
| 独白比例 | 少量内心独白，主要用于回忆带背景 |

**原文证据**：
```
"它把吃的都给我了。"薇薇安低声道。
"它自己没吃。"
```

**量化指标**：
- 对话占全文比例：~8%
- 引导词使用率：偶用
- 每轮对话平均轮次：2-3轮
- 内心独白占比：~5%

### 维度6：修辞与描写

**分析结果**：

| 检查点 | 发现 |
|--------|------|
| 修辞密度 | 克制。无比喻无排比 |
| 修辞类型 | 不使用修辞。用平实物/事本身触达情绪 |
| 感官维度 | 视觉+听觉为主。触觉偶用 |
| 环境描写 | 简洁，不超过段落的20%。环境服务于叙事 |


**原文证据**：
```
"烈火熊熊燃烧，将半边天空都映成了赤红色。"
——平实描写，不修饰不渲染
```

**量化指标**：
- 每千字修辞手法数：0-1
- 主要感官通道：视觉 > 听觉 > 触觉
- 环境描写占比：15%

### 维度7：情绪与张力

**分析结果**：

| 检查点 | 发现 |
|--------|------|
| 情感传递 | 动作外化为主，叙事者直述为辅 |
| 节奏模式 | 震荡型——温情与威胁高频交替 |
| 铺垫-释放比 | 铺垫约占60%，释放约占40% |
| 文风"气质" | 冷峻、克制、偶尔透出冷幽默 |

**原文证据**：
```
节奏序列（CH1）：
做饭（日常）→ 脚步声（威胁）→ 狗吠（紧张）
→ 远去（放下）→ 喂粥（温情）→ 关门回忆（低沉）
→ 决意卖身（凄凉）→ 打杀（恐惧）→ 醒来（希望）
```

**情感传递方式分布**（取样20处）：

| 方式 | 占比 | 示例 |
|------|------|------|
| 动作外化 | 50% | "缩进了被窝""抱紧老狗" |
| 内心独白 | 15% | 回忆索西亚、回忆母亲 |
| 环境投射 | 10% | "烈火映红了天空" |
| 对话潜文本 | 15% | "它把吃的都给我了" |
| 叙事者直述 | 10% | "恐惧。""她坚强地" |

## 三、关键洞察

1. **叙事者既克制又侵入的矛盾统一**：作者表面上是克制的（不渲染不评价），但实际上叙事者声音非常主动——频繁跳出来解释因果、直接给情绪标签。这是一种"叙事者存在感强但不啰嗦"的独特平衡。

2. **动作外化是核心机制**：所有内心活动都外化为动作/身体反应。内心OS是最后手段。这让文本有极强的"画面感"——情绪让读者自己从画面里读出来，而非直接告诉读者。

3. **信息密度远超叙事简单度**：虽然文本读感简单直接（无修辞、少对话），但前三章释放的信息量非常大。信息通过密集的日常场景串联释放，而非通过说明段落堆砌。

## 四、风格红线清单

```
□ 叙事者跳出来长篇大论评价角色或场景
□ 用"不是A而是B"的否定式描写
□ 内心OS代替动作
□ 把世界观写成说明文段落
□ 把所有信息都解释得明明白白，不给读者留空间
□ 连续平铺叙事，没有节奏交替
□ 比喻/排比等修辞泛滥
```
