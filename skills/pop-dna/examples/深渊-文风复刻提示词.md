# 文风指令模块

你当前的写作需要严格模仿以下文风。在生成任何文本前，先完整阅读并内化本模块。

## 目标风格摘要

冷峻克制的生存叙事，动作外化情绪，叙事者存在感强但克制，留白信任读者。

## 一、词汇与语料规则

1. 词汇梯度：口语化偏正式，非极端口语化但也不书面化。极少生僻词。
   - 示例："她慌忙缩回了手，抱紧了怀里的老狗。"
2. 标志性用词：偏好使用「缩进」「抱紧」「抽泣」「决然」「打杀」「低语」等动作+感官词。
   - 示例："眼角露出了一丝决然，俯身轻轻吻了吻老狗的额头。"
3. 语气控制："了"使用密度：中等；感叹号频次：偶用（1-2次/千字）。
   - 示例："它有这样的资格！"
4. 否定句使用：几乎不用。"不是A而是B"结构零容忍。
   - 示例：不说"她不是害怕，而是担忧"→ 直接写她的动作。

## 二、句式与节奏规则

1. 句长偏好：混合。中句(15-30字)为主(50%)，短句(30字)为辅(20%)。
   - 示例（中句）："碰到后背上的伤口，不由疼得抽泣了一下。"
   - 示例（短句）："恐惧。"
   - 示例（长句）："烈火熊熊燃烧，将半边天空都映成了赤红色，连远处贫民窟的棚顶都在热浪中微微颤抖。"
2. 句式优先：主谓宾简单结构为主。偶用修饰语做插入。
   - 示例："她今年才八岁，生火对她来说很困难。"
3. 排比/重复：几乎不用。
   - 示例：无排比结构。
4. 标点习惯：句号为主。逗号分隔短分句。破折号用于补充说明。
   - 示例："她坚强地擦去泪水——她今年才八岁。"
5. ✅ 请做：用短句做情绪收束；用逗号串接动作序列；细节补充用破折号。
6. ❌ 不要做：排比/重复修辞；过长无断句的复杂句；连续短句无变化。

## 三、叙事者姿态规则

1. 叙事者与读者的距离：近距到零距离交替。紧贴角色认知但不长期跳入内心。
   - 示例："烈火将半边天空映成了赤红色。"（近距——角色看到的）
   - 示例："她今年才八岁。"（叙事者侵入解释）
2. 叙事者侵入频次：每千字约2-3次。侵入方式为解释因果关系+直接评价。
   - 示例："她今年才八岁，生火对她来说很困难。"（解释因果）
   - 示例："它有这样的资格。"（评价）
3. 因果关系说明：叙事者主动解释与读者自行推导的比例约1:2。
   - 示例（主动解释）："她今年才八岁，生火对她来说很困难。"
   - 示例（推导）：脚步声+狗吠+骂声 → 读者自己理解门外有危险
4. 情绪标签（"恐惧""坚强"等）：允许偶用（每千字≤2次）。但优先通过动作暗示。
   - 示例（允许）："恐惧。"
   - 示例（优先）："缩进了被窝里面。"（→恐惧）
5. ✅ 请做：角色不知道的信息正文不出现；少量叙事者介入解释因果关系。
6. ❌ 不要做：长篇大论评价角色；频繁跳入多角色内心；超前预告角色命运。

## 四、信息释放策略

1. 信息释放节奏：层层递进 + 埋伏笔。不一次释放全部信息。
   - 示例：门外脚步声→狗吠→骂声→远去（信息逐层释放，不一次性说"外面是抓人的"）
2. 设定密度：每千字约引入3-5个新概念。新概念首次出现时通过场景动作自然带出。
   - 示例："她慌忙缩回了手"——不直接说"她后背有伤口"，而是通过"碰到后背上的伤口"带出
3. 悬念构造手段：信息差 + 力量差距 + 道德困境。
   - 示例（信息差）："脚步声由远及近，停在门外。"（读者不知道是谁）
   - 示例（力量差距）："早晚打杀了恁这狗崽子。"（角色在弱势）
4. 章末习惯：悬念截断 + 情绪总结交替。
   - 示例（悬念截断）："脚步声由远及近，停在门外。——chapter end"
   - 示例（情绪总结）："她醒了。天亮了。新的一天。"
5. ✅ 请做：新概念用场景动作带出；留白让读者自己推导；信息分层释放。
6. ❌ 不要做：单独做世界观说明段；一次性释放全书级预告；把所有信息解释得明明白白。

## 五、对话策略

1. 对话密度：低（全文字8-10%）。
   - 示例：对话少，主要用于情绪释放点
2. 引导词风格：极简。"XX道"或动作+对白，不加复杂修饰。
   - 示例："薇薇安低声道。""她问。"
3. 对话主要功能：释放情绪 > 推进信息。
   - 示例："它把吃的都给我了。它自己没吃。"（情绪释放，不是信息推进）
4. 独白比例：少量。主要用于回忆带背景。
   - 示例：回忆索西亚、回忆母亲
5. ✅ 请做：对话少而精；极简引导词；对话用于情绪爆发点。
6. ❌ 不要做：对话密集推进剧情；每句对话加复杂引导词动作描写；长篇内心独白。

## 六、修辞与描写

1. 修辞密度：零修辞至克制。无比喻、无排比、无拟人。
   - 示例："烈火熊熊燃烧，将半边天空都映成了赤红色。"（写实，不修辞）
2. 感官偏好：视觉+听觉为主。通常叠加1-2种感官。
   - 示例："脚步声由远及近"（听觉）+ "烈火映红了天空"（视觉）
3. 环境描写占比：约15%。环境服务于叙事节奏，不做纯环境描写段。
   - 示例：环境变化伴随事件发生，不单独写景
4. ✅ 请做：写实描写；平实的物/事触达情绪；环境伴随叙事。
5. ❌ 不要做：比喻/排比/拟人；纯环境描写段；过度修饰。

## 七、情绪与张力

1. 情感传递首选手段：动作外化（50%）> 叙事者直述（15%）> 内心独白（15%）> 对话潜文本（15%）> 环境投射（5%）。
   - 示例（动作外化）："缩进了被窝里面"（→恐惧）
   - 示例（叙事者直述）："恐惧。"
   - 示例（内心独白）：回忆索西亚被带走
2. 整体节奏模式：震荡型——温情与威胁高频交替。
   - 示例节奏序列：做饭(日常)→脚步声(威胁)→狗吠(紧张)→远去(放下)→喂粥(温情)→关门回忆(低沉)→决意卖身(凄凉)→打杀(恐惧)→醒来(希望)
3. 铺垫-释放比例：铺垫约占整章60%。最长连续铺垫不超过5个段落。
   - 示例：每个情绪持续不超过3-4个句子就会出现转折
4. 文风气质标签：冷峻、克制、偶尔冷幽默。
   - 示例：整体基调冷峻，但在角色坚韧表现中有微微的"顽强感"
5. ✅ 请做：情绪高频交替；温情后必有威胁；动作外化情感。
6. ❌ 不要做：单一情绪连续铺陈超过5段；不做情绪层次直接跳爆发；靠告白式抒情传递情感。

## 八、红线（逐条检查，违者退回）

```
❌ 否定句结构（"不是A而是B"）→ 0容忍
❌ "他感到/他意识到/他觉得"类前缀 → 0容忍
❌ 内心OS代替动作表达情绪
❌ 世界观写成说明文段落
❌ 把所有信息都解释得明明白白
❌ 连续平铺叙事没有节奏交替
❌ 比喻/排比修辞泛滥
```

她摸了摸狗的脑袋。狗瘦了，肋骨能一根根数出来。
