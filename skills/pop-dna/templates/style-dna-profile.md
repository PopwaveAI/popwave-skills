# 文风DNA档案 v3.0

> ⚠️ **v3 与 v1/v2 的根本区别**：不统计平均句长/词频/对话占比。不浸泡原文让 Agent "感受"。只提取**作者在每个场景里做了什么选择**——每条规则 = 结构描述 + 原文证据 + 可执行指令。

---

## 产出结构（4 段）

```
§1  作者方法论摘要      ≤5 条·贯穿全场景的写作哲学
§2  场景型决策规则      按场景类型组织·每条配原文证据+可执行指令
§3  跨赛道便携性标注    骨架 vs 肉身·用变量代替硬码
§4  风格红线            禁令式·本风格绝对不会做的事
```

---

## §1 作者方法论摘要

```
### 1. {一句话写作哲学}
→ "{这段哲学在写作中的实际表现。不要统计数字——描写作家怎么做选择的。}"
证据: ch{XX}: "{一句原文}" {↔ 如果有跨作品对比，chYY: "对应原文"}
```

≤5 条。每条一条核心思想、一句原文证据。不列统计数字。

---

## §2 场景型决策规则

按场景类型逐个提取。格式：

```
### 场景 {N}：{场景类型}

规则: {作者在这个场景里做了什么结构选择——不是描述原文内容，是提取可复用的结构}
文A: "{原文}"（{作品·语境}）
{文B: "{跨作品对比例句}"（如有）}
可执行指令: {给 Agent 的一句话指令。不是"注意xx"——是"做xx，不做yy"}

{重复，覆盖: 外貌/群像/对话/决策/战斗/世界观植入/段落呼吸}
```

### 场景清单（必须覆盖的 6 个）

| # | 场景类型 | 要提取的问题 |
|:-:|:--------|:----------|
| 2.1 | 人物外貌描写 | 作者怎么介绍一个人？材质/版型/配饰三层？动作带出外貌？整体印象一句？ |
| 2.2 | 城市/群像 | 怎么让读者感觉到"这里人多、阶层分明"？清单枚举还是直接说结论？ |
| 2.3 | 角色决策推理 | 角色做重大决定时——条件句前置？直觉跳转？内心独白？ |
| 2.4 | 战斗 | 战斗节奏序列是什么？接触→被压制→触底→转折→反制？感官怎么切换？ |
| 2.5 | 市井/对话 | 对话语感是叙事者语感还是角色自己的语感？市井场景用不用俗语/黑话？ |
| 2.6 | 世界观植入 | 新概念怎么介绍？场景带出？角色对话？叙述者说明段落？教材式？ |

## §3 跨赛道便携性标注

```yaml
portable（跨赛道通用——骨架·永久保留）:
  - {规则名}   → 可用到任何题材/文化
  - ...

genre_specific（换赛道换变量——肉身·取值替换）:
  - {规则名}   → 变量: {变量名}
    {赛道A}: {取值A} / {赛道B}: {取值B}
  - ...
```
