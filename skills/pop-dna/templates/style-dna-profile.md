# 文风DNA档案 · 模板

> **格式规则**：平铺，无章节编号，无四段结构，无禁令。每个维度一节 H2 标题，配 2-3 处原文引用。每个维度标注开篇/中期/末期变化（如样本足够）。不写绝对化结论——有全书搜索数字的标注数字，没有的标注"采样范围内"。

---

## 标题

```
# {书名} · 文风DNA
> {作者}《{书名}》| 采样 {章段范围} + 全书搜索验证
```

## 维度覆盖（强制 9 个）

| # | 维度 | 要回答的问题 |
|:-:|------|------------|
| 1 | 叙事者距离 | 叙事者离角色多近？进入内心还是只跟视线？随时间怎么变？ |
| 2 | 情绪传递 | 情绪主要通过动作/对话/直述哪一个？全书搜索直述情绪的数量。 |
| 3 | 外貌描写 | 外貌通过身体状态传递还是形容词直述？全书搜索"英俊""清秀"等的数量。 |
| 4 | 环境展开 | 每章从什么感官进入？空间认知从哪里开始？开篇和末期的差异？ |
| 5 | 对话 | 引导词密度？市井角色怎么说话？与开篇的差异？ |
| 6 | 战斗 | 有招式名吗？搜索全书战斗相关词汇数量。招式名用什么语言包装（武侠/游戏/系统）？战后数据处理方式？ |
| 7 | 数据嵌入 | 面板/装备/击杀反馈在正文中的呈现格式？用自然段落还是代码块？从开篇到末期的格式是否一致？ |
| 8 | 段落呼吸 | 独句段的使用频率和功能？开篇/中期/末期的差异？ |
| 9 | {场景特有发现} | 根据实际阅读发现新增——如性描写、幽默、作者语调变化等。必须基于原文证据。 |

> 第 9 个维度非固定——取决于实际采样中发现的**全书范围内的显著特征**。没有发现就不写。

## 每条维度的格式

```
## {维度名}

{1-2句描述发现的模式}

{随时间变化的说明——如"开篇做到极致，中期保留，末期转为宏观视角"}

> "{原文}"（{章节标注·简短说明}）
> "{原文}"（{章节标注·简短说明}）
> "{原文}"（{章节标注·简短说明}）

{如有全书搜索数字——"全书搜索：'XX' N次"——放在原文引用后面}

---
```

## 文件末尾

```
## 全书稳定特征

1. {特征1}——从 CH{X} "{简述}" 到 CH{Y} "{简述}"，{说明为什么稳定}
2. {特征2}——{同上格式}

---

*{书名} v3 | 采样 ~{行数}行 + 全文搜索 | {日期}*
```

## ❌ 禁止事项

- 不写禁令清单（"绝对不要做 X"）——只写发现了什么
- 不写可执行指令（"写 X 时先 Y 再 Z"）——发现本身就是指引
- 不写便携性标注——不属于文风DNA，属于下游 writer 的判断
- 不统计词频/句长/修辞密度——那不是发现，是症状
- 不把单个场景的偶然细节当全书规律——搜全书验证
