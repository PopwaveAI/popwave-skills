# 修仙流剧情模式

> 待拆解爆款案例后补充完善。

## 赛道特征

| 维度 | 说明 |
|------|------|
| 世界观 | 修仙界 + 宗门/境界/法宝/丹药体系 |
| 主角身份 | 外门弟子/废柴/重生者/穿越者 |
| 核心爽点 | 越级碾压/打脸宗门/炼丹炼器碾压/宗门政治翻盘 |
| 节奏偏好 | 长弧蓄力（3-5章一个大爽点），修炼段节奏可慢 |
| 冲突来源 | 宗门政治/资源争夺/道统之争/师门恩怨 |

## 爽点结构

| 爽点类型 | 修仙特色实现 |
|---------|-------------|
| 打脸循环 | 被师兄/长老轻视→修炼突破→大比/实战碾压→全宗门震惊 |
| 压力突破 | 被围杀/秘境绝境→突破境界→反杀 |
| 未知揭示 | 功法秘密/宗门历史/天劫真相/大道之谜 |
| 弱→强反转 | 废柴/低境界→金手指/秘宝→越级碾压 |

## 事件分量标准

- 改变命运：境界突破/拜师/被逐出宗门/获得传承
- 打开可能：解锁新功法/进入秘境/接触新势力
- 制造追读：宗门阴谋/传承秘密/天劫将至/仇人身份

## 低压区设计

- 被当废物（灵根差/修为低/外门弟子）
- 被宗门规则压制（辈分/门规/资源分配不公）
- 被高境界碾压（境界差距到绝望）
- 被师兄/长老针对（身份地位压制）

## 冲突来源

| 冲突类型 | 主角欲望 | 敌方欲望 | 不可调和原因 |
|---------|---------|---------|-------------|
| 资源争夺 | 想获得修炼资源 | 敌方也想获得 | 资源稀缺 |
| 宗门政治 | 想安稳修炼 | 长老/派系想利用他 | 权力斗争 |
| 道统之争 | 想走自己的道 | 宗门/正统不许 | 道不同 |

## 特有节奏要求

- 长弧蓄力：3-5章修炼铺垫→1章大爽点爆发
- 修炼段可以慢，但必须有信息推进（功法领悟/瓶颈突破线索）
- 大比/拍卖/秘境是爽点密集区
- 宗门日常是低压区喘息场景
