## 赛道特征

| 维度 | 说明 |
|------|------|
| 世界观 | 末日后世界 + 生存/变异/资源稀缺 |
| 主角身份 | 普通幸存者/重生者/变异者 |
| 核心爽点 | 生存逆袭/资源掠夺/人性考验中胜出 |
| 节奏偏好 | 紧凑生存压力（每章生死攸关） |
| 冲突来源 | 人性考验/资源稀缺/怪物威胁/安全区政治 |

## 爽点结构

| 爽点类型 | 末世特色实现 |
|---------|-------------|
| 打脸循环 | 被当累赘→生存能力碾压→队友/敌人震惊 |
| 压力突破 | 被丧尸/变异体围困→发现弱点/解锁能力→反杀 |
| 未知揭示 | 灾难原因/变异真相/安全区秘密/更大威胁 |
| 弱→强反转 | 普通人→获得变异能力→碾压变异体 |

## 事件分量标准

- 改变命运：获得变异能力/进入安全区/失去同伴/成为领导者
- 打开可能：发现新避难所/解锁新能力/接触军方/发现物资
- 制造追读：灾难真相/安全区阴谋/变异升级/食物将尽

## 低压区设计

- 被当累赘（战斗力低/需要保护/消耗物资）
- 资源匮乏到绝望（食物/水/药品将尽）
- 被变异体围困（无路可逃）
- 被同伴背叛（末世人性）

## 冲突来源

| 冲突类型 | 主角欲望 | 敌方欲望 | 不可调和原因 |
|---------|---------|---------|-------------|
| 资源争夺 | 想活下去 | 他人也想活下去 | 资源不够分 |
| 人性考验 | 想保持人性 | 他人放弃底线 | 末世逼人变恶 |
| 安全区政治 | 想安稳 | 管理层想控制 | 权力vs自由 |

## 特有节奏要求

- 每章有生存压力（不是每章打架，但每章有生存威胁）
- 安全区喘息场景用于人际互动和情感建立
- 物资/资源的可数收益极其重要（读者爱看"又获得了X天口粮"）
- 人性考验比怪物战斗更制造追读
