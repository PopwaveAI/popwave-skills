## 文件定位

本文件覆盖D&D数据面板流中所有多线叙事技法。多线叙事是在主角视角为主体的前提下，通过POV切换、多地点快切、世界级事件呈现等手法，扩展叙事广度、制造信息不对称悬念、侧面定级主角实力、建立配角情感锚点。

**核心原则**：多线叙事不等于平均分配视角——主角视角仍占主体。所有POV切换必须有叙事目的（信息不对称/侧面定级/情感锚点），不是为切而切。

prose-render在Step 2c加载本文件后，根据施工卡标注的多线叙事类型从下方6种类型中选取对应技法渲染。

## 二、多线叙事类型详解

### 类型1：多地点快切（超凡期+）

**适用场景**：同一事件在不同地点同时产生反应，需要跨空间串联因果链。

**等级段限制**：超凡期+（11级以上）。凡人期叙事范围有限，不使用多地点快切。

**切换标记**：

多地点快切必须使用明确的切换标记，引导读者完成空间跳跃。

| 切换标记格式 | 使用场景 |
|-------------|---------|
| "与此同时。" | 同一时间不同地点的平行事件 |
| "千里之外。XX城。" | 远距离切换，强调空间跨度 |
| "……"+地点名 | 省略号过渡后接地点名 |
| "就在这一刻，XX的XX" | 因果串联式切换 |

**核心技法**：
- **同一事件在不同地点同时产生反应**：一个事件如石子入水，涟漪向多个地点扩散
- **因果链跨空间串联**：A地的事件触发B地的反应，B地的反应又影响C地
- **电影蒙太奇效果**：快速切换多个地点的片段，拼凑出事件全貌

**原作锚点**：

> 恐惧神国/现实世界/摩多城大神殿三线并行——同一事件（神国变动）同时在三个空间产生反应，通过快切串联因果链，呈现事件的世界级影响。

**渲染要点**：
1. 每次地点切换必须有明确标记（"与此同时""千里之外"等），不得无标记切换
2. 每个地点的片段控制在1-3段，不宜过长（保持蒙太奇节奏）
3. 各地点片段之间必须有因果关联——不是随机拼凑，而是同一事件的不同切面
4. 快切顺序按因果链排列：事件源头 → 直接影响区域 → 间接影响区域
5. 超凡期+才可使用此技法——凡人期主角的影响力不足以支撑跨空间叙事

**节奏模板**：
```
[地点A：事件发生] → "与此同时。" → [地点B：直接反应] → "千里之外。地点C。" → [地点C：间接反应] → 切回主角
```

### 类型2：世界级事件叙事（reveal_hook章）

**适用场景**：影响多个区域/势力/阶层的世界级事件，在reveal_hook章中呈现。

**章节限制**：仅用于reveal_hook章。其他章节不使用世界级事件叙事。

**事件规模**：
- 影响多个区域（跨城市/跨国家/跨位面）
- 影响多个势力（诸神/超凡者/凡人组织/平民）
- 影响多个阶层（从最高神灵到最底层平民）

**多层递进展示影响**：

影响按层级从上到下递进展示，每一层呈现不同的反应。

| 层级 | 反应类型 | 描写侧重 |
|------|---------|---------|
| 诸神 | 力量变化/神职波动 | 概念级描写，宏观 |
| 超凡者 | 实力波动/适应/不适应 | 数据化描写，个体 |
| 凡人组织 | 运转受影响/秩序变化 | 组织视角，机构化 |
| 平民 | 生活变化/恐慌/无知 | 群像描写，情绪化 |

**世界事件面板格式**：

```
【世界事件】XX
地点：XX
影响：XX
```

**面板示例**：

> 天空中魔网的纹路突然扭曲，如同被无形巨手撕扯的蛛网。施法者们的咒语在舌尖变得滞涩。
>
> 【世界事件】魔网崩溃
> 地点：全域（费伦大陆所有魔网覆盖区域）
> 影响：魔网能量紊乱，施法者法术效率下降50%-80%，高阶法术无法施放。

**"新闻播报"式笔法**：

用类似新闻播报的笔法呈现多地点灾情——冷静、客观、信息密集。

**示例**：

> 摩多城大神殿的牧师们发现他们的神术变得迟钝，治愈轻伤的祈祷需要三倍的时间。深海城的法师塔中，三位大法师的传送术同时失效，将他们困在了实验室。北境的德鲁伊圈感受到了自然魔力的枯竭，橡树守卫的仪式被迫中断。而在更远的东方，一个依靠法术维持防御的城邦，城墙上的魔法屏障正在一层层地剥落……

**原作锚点**：

> 魔网崩溃 → 诸神力量衰减 → 施法者不适应 → 女巫团被废。事件从最高层（诸神）逐级向下传导，每一层的影响方式不同，但因果链清晰。

**渲染要点**：
1. 世界级事件叙事仅用于reveal_hook章，其他章节不用
2. 影响展示必须按层级递进（诸神→超凡者→凡人组织→平民），不得跳层
3. 世界事件面板弹出后，该事件必须在后续5章内对主角产生直接影响（参照面板叙事文件红线）
4. 多地点灾情用"新闻播报"式笔法——信息密集、冷静客观，不用情绪化描写
5. 世界事件面板格式固定：【世界事件】+ 事件名 / 地点 / 影响，三行
6. 世界级事件的因果链必须清晰：A导致B，B导致C，不得断裂

### 类型3：敌人POV插入

**适用场景**：切到敌人视角，呈现敌人不知道自己被盯上/敌人第一次感到恐惧的心理状态。

**等级段限制**：全等级段通用。

**核心功能**：制造信息不对称的悬念——读者通过敌人视角知道了敌人不知道的信息（主角已经盯上他了/主角已经设好了陷阱），产生紧张感。

**插入模式**：
- 切到敌人视角写"他不知道自己被盯上"
- 或切到敌人视角写"他第一次感到恐惧"
- 3-5行后切回主角

**原作锚点**：

> 希瑞克在暗杀巫妖——"不过不知道为什么，他此刻突然有点不安起来！"敌人视角中呈现不安感，读者知道这是主角逼近的信号，但敌人自己不知道原因——信息不对称的悬念。

**信息不对称类型**：

| 类型 | 敌人视角内容 | 读者知道的信息 | 悬念效果 |
|------|-------------|--------------|---------|
| 不知被盯上 | 敌人日常/自信/放松 | 主角已经锁定他 | 紧张期待（何时出手？） |
| 隐约不安 | 敌人感到不安但不知原因 | 主角正在逼近 | 悬疑恐惧（会怎样？） |
| 第一次恐惧 | 敌人遭遇主角后恐惧 | 主角实力远超预期 | 爽感（碾压确认） |

**渲染要点**：
1. 敌人POV插入控制在3-5行，快速切回主角——不宜过长
2. 敌人视角的内容必须与主角行动有因果关联（主角的潜行/逼近导致敌人不安）
3. 敌人视角的语感更主观、更情绪化，与主角视角的冷静数据化形成对比
4. 切回主角时不需要切换标记（直接切回），但切入敌人视角时建议有场景转换引导
5. 敌人POV插入不是展示敌人有多强，而是制造信息不对称——敌人视角的"安全感"反衬即将到来的危险

### 类型4：旁观者视角插入

**适用场景**：主角行为后切到旁观者视角，用旁观者的恐惧/震惊来侧面定级主角实力。

**等级段限制**：全等级段通用。

**核心功能**：侧面定级——不直接说主角有多强，而是通过旁观者的反应让读者感受到主角的强度。

**插入模式**：
- 主角完成某个动作（击杀/速度展示/力量展示）后
- 切到旁观者视角
- 旁观者的震惊反应侧面定级
- 2-3行后切回主角

**旁观者反应类型**：

| 反应类型 | 示例句式 | 定级效果 |
|---------|---------|---------|
| 速度震惊 | "那是什么速度？" | 主角速度超出认知 |
| 能力质疑 | "这不是人类能做到的" | 主角超出凡人范畴 |
| 恐惧反应 | 旁观者退后/发抖/失语 | 主角威慑力 |
| 经验判断 | "我打了三十年仗，从没见过……" | 用老兵经验衬托 |

**渲染要点**：
1. 旁观者视角插入控制在2-3行，比敌人POV更短——旁观者不是叙事主体
2. 旁观者的反应必须是"震惊/恐惧/不可置信"等强烈情绪，不是冷静分析
3. 旁观者用疑问句/感叹句表达震惊，不用陈述句——"那是什么速度？"比"他的速度很快"更有冲击力
4. 旁观者视角不需要切换标记，直接在主角行动后另起段落切入
5. 同一章内旁观者视角插入不超过2次——用多了会打断主角叙事节奏

---

### 类型5：配角独立行动线（超凡期+）

**适用场景**：关键NPC的独立行动需要切POV展示，建立配角情感锚点。

**等级段限制**：超凡期+（11级以上）。凡人期配角独立性有限，不单独开行动线。

**核心结构**：
- 配角视角开场（章节前1/3用配角视角建立情感锚点）
- 然后切回主角
- 配角行动线与主角行动线在章节后段交汇

**配角视角开场的作用**：
1. **建立情感锚点**：让读者对配角产生情感投入，关心配角的命运
2. **信息铺垫**：配角视角提供主角不在场的信息
3. **平行叙事**：配角行动与主角行动平行推进，最终交汇

**原作锚点**：

> 爵士视角——"是时候让那些杂碎们看看我们的实力！"配角视角的热血宣言建立情感锚点，读者对配角产生期待，随后配角在关键时刻登场与主角交汇。

**渲染要点**：
1. 配角视角开场控制在章节前1/3，不宜超过——章节主体仍是主角视角
2. 配角视角必须建立情感锚点——不是单纯的信息传递，而是让读者关心这个配角
3. 配角行动线必须与主角行动线有交汇点——不能完全独立无关
4. 配角视角的语感与主角区分——配角可以更情绪化/更热血/更恐惧，取决于配角性格
5. 超凡期+才使用此技法——凡人期配角不够独立，不需要单独行动线

**结构模板**：
```
[配角视角开场：建立情感锚点 + 信息铺垫] → [切回主角：主角行动推进] → [双线交汇：配角登场/行动影响主角]
```

### 类型6：神灵视角感知（神明级）

**适用场景**：神灵级别的感知方式——非五官感知，而是神识/心生感应。

**等级段限制**：神明级（25级+）。超凡期及以下不使用神灵视角感知。

**感知方式**：

| 感知类型 | 描写方式 | 与凡人感知差异 |
|---------|---------|--------------|
| 神识 | "神识扫过""神识覆盖" | 非视觉，概念级感知 |
| 心生感应 | "心生感应""冥冥中感知到" | 被动触发，非主动搜索 |
| 信仰网络 | "信仰网络中传来波动" | 通过信徒感知事件 |
| 概念感知 | "死亡三神的力量""神性的气息" | 感知概念本身，非物理实体 |

**核心特征**：
- **非五官而是"神识""心生感应"**——神灵不依赖眼睛耳朵，而是用概念级感知
- **概念级感知**：感知的不是物理现象，而是概念本身——"死亡三神的力量""神性的气息"
- **被动感知**：信仰网络内的异常自动引起注意，不需要主动搜索
- **千里之外感应事件**：空间距离对神灵感知不构成障碍

**感知描写示例**：

> 索伦端坐在神座之上，神识如潮水般向四面八方蔓延。信仰网络中，数以万计的信徒祈祷声如同海浪般起伏。突然，一道异样的波动从极远处传来——那是死亡三神的力量在某个位面汇聚的痕迹。

**渲染要点**：
1. 神灵视角感知用"神识""心生感应""信仰网络"等专属词汇，不用"看到""听到"
2. 感知内容是概念级的——"力量的汇聚""气息的变化"，不是物理描写
3. 神灵感知是被动触发的——信仰网络内的异常自动引起注意，不需要主动搜索
4. 神灵视角可以感知千里之外的事件，空间距离不构成障碍
5. 仅神明级使用此技法——超凡期及以下的角色不具备神识/信仰网络
6. 神灵视角感知的面板可呈现神性信息（信仰值变化/神职波动/领域异常），参照神明级面板格式

## 三、POV切换硬规则

以下场景**必须**进行POV切换，不是可选项：

| 场景 | 必须切换的POV | 切换时长 | 目的 |
|------|-------------|---------|------|
| 高潮章Boss战 | Boss视角 | 3-5行 | 呈现Boss的恐惧/暴怒/误判 |
| 新势力登场 | 该势力视角 | 1段 | 建立势力印象/定调 |
| 主角不在场的关键事件 | 在场者视角 | 事件全程 | 呈现主角不在场时发生的事 |

**切换标志要求**：
- POV切换需有明确标志（场景转换/地点标记/人物焦点转移）
- 快速切回：3-5行后切回主角（超凡期+可延长至1/3章节）
- 切回主角不需要标志（直接切回），但切入非主角视角时建议有引导

**POV切换检查清单**：
- [ ] 高潮章Boss战是否切了Boss视角？
- [ ] 新势力登场是否切了该势力视角？
- [ ] 主角不在场的关键事件是否切了在场者视角？
- [ ] 每次POV切换是否有明确叙事目的？
- [ ] POV切换后是否在规定行数内切回主角？

## 四、多线叙事与面板的结合

多线叙事中的面板使用规则：

| 多线类型 | 面板使用 | 格式来源 |
|---------|---------|---------|
| 多地点快切 | 各地点可弹简版状态面板 | 等级段面板文件 |
| 世界级事件叙事 | 必弹世界事件面板 | 面板叙事文件第六节 |
| 敌人POV插入 | 敌人视角可弹敌人属性简版 | 数据化分析文件 |
| 旁观者视角插入 | 不弹面板（纯情绪反应） | — |
| 配角独立行动线 | 配角视角可弹配角属性简版 | 等级段面板文件 |
| 神灵视角感知 | 可弹神性信息面板 | 神明级面板文件 |

**面板结合红线**：
- 旁观者视角插入不弹面板——旁观者的震惊是纯情绪反应，不需要数据化
- 世界级事件叙事必弹世界事件面板——这是世界级事件的标志
- 敌人POV插入的敌人面板是简版——不需要完整属性，只需关键信息
- 非主角视角的面板用第三人称（"目标""他"），主角面板用第二人称（"你"）

## 五、各等级段多线叙事能力对照

| 能力 | 凡人期(1-10) | 超凡期(11-25) | 神明级(25+) |
|------|-------------|--------------|------------|
| 多地点快切 | 不可用 | 可用 | 可用 |
| 世界级事件叙事 | 不可用（非reveal_hook章） | 可用（reveal_hook章） | 可用（reveal_hook章） |
| 敌人POV插入 | 可用 | 可用 | 可用 |
| 旁观者视角插入 | 可用 | 可用 | 可用 |
| 配角独立行动线 | 不可用 | 可用 | 可用 |
| 神灵视角感知 | 不可用 | 不可用 | 可用 |
| POV切换频率 | 低（偶尔切敌人/旁观者） | 中（每3章至少1次非主角视角） | 高（多POV常态化） |

**递进逻辑**：
- 凡人期：主角视角为主，偶尔切入敌人/旁观者视角制造悬念
- 超凡期：主角视角仍为主体，但多地点快切和配角独立行动线开始使用，POV切换频率上升
- 神明级：多POV常态化，神灵视角感知提供概念级叙事广度

## 六、红线

1. **主角主体红线**：多线叙事不等于平均分配视角——主角视角仍占主体。非主角视角的总篇幅不得超过章节篇幅的1/3（超凡期+可放宽至1/2，但主角视角必须仍是主体）。
2. **叙事目的红线**：POV切换必须有叙事目的（信息不对称/侧面定级/情感锚点），不是为切而切。如果切 POV 对叙事没有贡献，则不该切。
3. **世界事件面板红线**：世界级事件面板只用于reveal_hook章。其他章节不得使用世界事件面板格式。
4. **快速切回红线**：敌人POV插入3-5行后必须切回主角，旁观者视角2-3行后切回。不得在非主角视角停留过久（超凡期+配角独立行动线除外，但章节前1/3后必须切回主角）。
5. **切换标志红线**：POV切换需有明确标志（场景转换/地点标记/人物焦点转移）。无标志的POV跳跃=不合格。切回主角时不需要标志。
6. **必切场景红线**：高潮章Boss战必须切Boss视角，新势力登场必须切该势力视角1段，主角不在场的关键事件必须切在场者视角。漏切=不合格。
7. **等级段匹配红线**：多地点快切仅超凡期+使用，配角独立行动线仅超凡期+使用，神灵视角感知仅神明级使用。低等级段使用高等级段技法=不合格。
8. **信息不对称红线**：敌人POV插入的核心价值是信息不对称——敌人视角必须呈现"不知道自己被盯上"或"不知道危险来源"的状态。如果敌人视角与主角视角信息完全对称，则失去POV切换的意义。
