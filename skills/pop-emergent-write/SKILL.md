# Pop Emergent Write

这是“把 X 写成 Y”的短文/正文执行 skill，不是路由级专家，也不是正式长篇管线。默认由 `pop-emergent` 调用。目标是把一个素材、桥段、参考对象或设定母题，直接转化成可读的短篇、开篇、单章、爽文场面或风格样稿。

## 核心原则

- 先让模型进入作家状态，再让编辑审稿；不要先把创作压成表格施工。
- 前置只给短燃料包，不做卷纲、幕纲、章级施工卡、微beat表。
- 正文一口气写完，不在写作中插入解释、评分、流程说明。
- 审稿在正文之后，只检查爽文兑现、AI味、可读性、明显bug。
- 只做短文、单场景、试写、开篇样章；不要把它当成长篇正式管线替代品。
- 不输出正式管线 execution YAML、loaded_skills、缺口清单；这是涌现短文，不是 create 降级稿。

## 适用场景

- 用户说“把 X 写成 Y”：把一个点子、桥段、人物关系、设定、参考作品、战斗场景写成短文/开篇/单章/爽文场面。
- 用户要快速试写一篇短文、开篇、战斗、打脸、悬疑、奇观、爽点场面。
- 用户明确说不走完整 skill 管线、不要复杂约束、想看直出效果。
- 用户要比较“涌现式写作”和“结构化施工”的质量差异。
- 用户只给了一个灵感、母题、桥段、角色或题材，希望先出一版有阅读感的文本。

不适用：

- 正式长篇立项、世界底盘、卷纲、幕纲、连续章节生产。
- 需要严格追踪设定账本、角色卡、数值表、跨章承诺的任务。
- 用户要求按已有 plot/create skill 正式落盘。
- 路由判断任务；路由由 `pop-emergent` 负责。

## 输入包

开始写之前，只整理一个短输入包。核心句式是：`把【X】写成【Y】`。不要超过 12 行。

```text
把 X 写成 Y：
题材/场景：
主角：
主爽点：
压力/敌人：
关键燃料：
禁区：
目标字数：
输出位置：
```

规则：

- `X` 是素材源：点子、桥段、参考书、角色关系、战斗、世界观元素、用户随口想法。
- `Y` 是目标成品：短篇、开篇、单章、爽文场面、战斗短文、打脸桥段、黑暗升级流样稿、克苏鲁爽文开篇等。
- 用户没给的信息可以合理补，不要为了补齐表格频繁追问。
- `关键燃料` 可以是参考作品、桥段、人物原型、世界观元素、用户随口想法；只保留最能刺激创作的 3-5 个。
- `禁区` 默认包含：严肃文学空镜、抽象总结、假紧张、只压迫不兑现、主角旁观。
- 如果用户没有目标字数，默认 1500-3000 字。
- 如果用户没有输出位置，直接在回复中给正文；如果给项目路径，写入项目下 `试写/` 或根目录的试写文件。

## 写作模式

写正文时遵守这些约束，但不要把它们显性写进正文：

- 第一屏尽快出现异常、冲突、压力或欲望，不用慢热铺陈。
- 主角必须主动判断、下注、行动或反击，不能只被事件推着走。
- 爽点要发生在场面里：误判、反转、打脸、收益、震惊、战书、升级至少兑现一个。
- 细节服务动作和信息，不用细节替代关键信息。
- 可以使用旁白、心理活动和直白解释，只要能让读者更快理解爽点。
- 结尾要有明确钩子：新敌人、新收益、新误判、新目标或更大的局。

## 后验审稿

正文完成后，给一个短审稿，不超过正文长度的 15%。只看四项：

| 项 | 检查 |
| --- | --- |
| 爽文兑现 | 主角主动、压迫、误判/反转、收益/反馈、钩子是否成立 |
| AI味 | 抽象总结、情绪标签、假紧张、机械转折、严肃文学空镜、说明书对白 |
| 可读性 | 开场抓人、场面清楚、信息密度、节奏是否顺 |
| 明显bug | 只有发现明显设定/逻辑/OOC问题才写 |

评分用 `强 / 可用 / 偏弱 / 失败`。每项必须给一句证据，不写长篇理论。

## 返修规则

- 如果只有局部 AI味或句子问题，只列 2-4 条修正建议，不立刻重写，除非用户要求。
- 如果爽文兑现失败，建议重写，并指出应该补哪个环节：压力、主角主动、误判、反馈、收益、钩子。
- 不为了“修得更高级”而削弱爽点。
- 不把一次短文试验沉淀为正式设定，除非用户确认“这个版本纳入项目”。

## 输出格式

默认输出：

```markdown
# 试写 · {标题}

{正文}

---

## 短审稿

| 项 | 判断 | 证据/建议 |
| --- | --- | --- |
| 爽文兑现 |  |  |
| AI味 |  |  |
| 可读性 |  |  |
| 明显bug |  |  |
```

如果写入文件，文件名使用 `试写-{标题}.md`。不要创建额外过程文件。

## 触发示例

- 把“雨夜客栈毒杀局，少年剑客反破局并下战书”写成一篇 2500 字武侠爽文短篇。
- 把“收尸人捡到神格碎片”写成一篇黑暗升级流开篇。
- 把“DND神明降临现代北美城市，主角被借眼看到高维战争”写成克苏鲁爽文开篇。
- 把“退婚现场，所有人以为男主是废物，结果他拿出宗门黑账”写成小白打脸流第一章。
