# CHANGELOG

## v5.0.0 (2026-07-15)

### 从按需辅助重构为单元剧蓝图设计器

**根因**：top20拆解发现单元剧模式是番茄爆款的主流结构。旧版plot"按需辅助"不设计任何结构，导致write每章都在即兴发挥，缺乏可复用的骨架。新版plot设计可复用模具（单元模板+长线种子+变体池），一次设计反复使用。

**改动**：
- 从"按需辅助（搜素材+建议下一章）"重构为"单元剧蓝图设计器"
- Step 1：单元模板设计（输入→识别→破解→奖励→钩子，可复用骨架）
- Step 2：长线种子设计（top20四种埋法+推进节奏）
- Step 3：变体池设计（10-15个可取用变体，类型多样+痛感递增+长线碎片）
- Step 4：燃料搜索（按需触发，丰富变体池）
- 产出从"口述不落盘"改为`单元蓝图.md`

## v4.0.0 (2026-07-15)

### 大幅简化：从故事推导引擎改为按需辅助

**根因**：实测发现plot的危机阶梯+四象限+场景卡在梗和写作之间加了抽象层，稀释了梗的力量。项目c不走plot直接写，质量更好。

**改动**：
- 砍掉Step 0全局危机阶梯
- 砍掉Step 2四象限散弹枪
- 砍掉场景卡产出
- 砍掉所有中间文件产出
- 只保留两个按需功能：燃料搜索（write卡住时）+下一章建议（不确定时）
- 不落盘任何文件
