# CHANGELOG

## v3.1.0 (2026-07-15)

### 新增Step 0全局危机阶梯

**根因**：v3.0.0的plot虽有燃料注入+四象限，但底层素材仍来自seed的100章推导——那是一堆无结构的孤立场景列表，没有危机递增、没有因果链、没有情绪阶梯。四象限在无结构素材上展开4个方向，4个方向都是同一个低痛感场景的变体。验证番茄top20后发现：75%作品有危机递增结构，80%有因果链，这些是全局结构属性，不是单章属性。

**改动**：
- 新增 Step 0 全局危机阶梯：从概念推导3-5个危机阶段（量级递增）+因果链+情绪曲线+部分解决比例(30-40%)
- 燃料注入改为针对当前阶段危机类型搜索（不再泛泛搜索）
- 草案5问升级为6问：新增"因果"（本章变化如何触发下章处境）
- 场景卡元数据新增：危机阶段、解决程度、因果指向
- 新增红线：Step 0必须先做、连续2章不得使用相同解题方式、部分解决占比≥30%
- seed的100章推导职责由plot Step 0接管

## v3.0.0 (2026-07-14)

### 重构：从"场景标签生成器"改为"故事推导引擎"

**改动**：
- 新增燃料注入+四象限散弹枪+用户选定
- 草案5问：处境/行动/转折/变化/读者幻想
- 场景卡格式改为故事优先
- 每批从5-10章减为3章
