## 1a. 加载创意文档

| 产出 | 提取内容 |
|------|---------|
| 一句话故事+草纲 | 方向已定，plot不重复 |
| 金手指 | 功能/入口能力+触发条件+限制边界+进化方向含衍生分支+升级路径 |
| 主角设定卡 | 职业/主性格/记忆点/缺点/主角危机/成长弧线+主角深度设计+核心羁绊角色 |
| 世界观骨架 | 世界格局/力量体系/核心规则+世界自洽四问答案/世界危机/势力格局/进度框架 |
| 长线线索 | 主线悬念+长线伏笔 |
| 番茄简介 | 基调参考 |

创意文档已定方向，plot做的是**完整世界构筑+第一卷执行方案**——不是重新设计方向。

## 1b. 加载黄金首章

读取 `2-正文/ch001.md`（seed的Step 5产出），提取：

| 要素 | 提取内容 |
|------|---------|
| ch001核心事件 | 首章发生了什么（3-5句） |
| ch001章末钩子 | 首章留下的悬念——ch002必须回收 |

| ch001主角状态 | 首章结束时主角的处境/位置 |
| ch001已出场角色 | 首章已经出场的角色——不能在plot中冲突 |

**黄金首章是plot推演的起点**——ch02开始的剧情白描必须从ch001的章末钩子延续。
