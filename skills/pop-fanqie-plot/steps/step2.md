# Step 2: 本卷设定快照+分支剧情线+分幕设计+叙事流剧情白描

> **v3.3.0：统一幕/卷格式+重构四层结构。** 幕=8~12章起承转合单元，卷=40~60章4~5幕。plot先从骨架全书设定圈出本卷涉及的设定并充实（2a本卷设定快照），再列出≥6条分支剧情线（2b），然后基于分支线分幕（2c），最后按幕写叙事流白描（2d）。

## 2a. 本卷设定快照（从骨架全书设定圈出+充实）

> **v3.3.0：从"切片"升级为"快照"。** 不只是从骨架圈范围，还要往地图里填充势力和彼此矛盾——不是空地图，是可直接写白描的实地图。

### 2a-1. 确定本卷卷号

从用户输入或项目状态确定当前要写第几卷。第一卷从ch001（seed黄金首章）之后开始；后续卷从上一卷卷末钩子延续。

### 2a-2. 从骨架.md全书设定圈出本卷范围

查骨架.md的"各卷设定切片预览"（world Step 3c产出），取本卷对应的范围：
- 力量体系范围（从全书5-7层中取本卷涉及的2-3层）
- 涉及地图区域（从全书所有区域中取本卷涉及的2-3个）
- 势力范围（从全书4层中取本卷涉及的层级）
- 危机范围（从全书终极+阶段性危机中取本卷爆发的）

### 2a-3. 充实为本卷设定快照

圈出范围后，**充实**成可直接写白描的实地图。不发明新设定，但要把骨架的设定充实成本卷可写的密度：

```markdown
### 本卷（第X卷）设定快照

#### 力量体系范围
- **主角起点快照**：阶位N（开卷时主角的力量状态——具体到能做什么/不能做什么）
- **主角终点快照**：阶位M（卷末主角的力量状态——本卷成长目标）
- **卷级敌人的力量范围**：阶位X-阶位Y（本卷主要敌人的力量分布，和主角的差距在哪）

#### 登场势力/角色
- **本卷碰撞势力**：势力A（第N层）+势力B（第M层）——每个势力标注：控制什么资源/弱点/和上层关系
- **本卷登场角色**：主角+配角A+配角B+反派A+反派B——每个角色标注：核心欲望/势力归属/和主角的关系张力

#### 涉及地图（填充势力和矛盾，不是空地图）
- **区域A**：[具体描述——这个区域长什么样/谁盘踞在这里/这里有什么资源]
  - 盘踞势力：势力A（控制XXX资源）
  - 内部矛盾：势力A内部XXX派vsXXX派的冲突
  - 和区域B的矛盾：势力A和势力B在XXX上的利益冲突
- **区域B**：[具体描述]
  - 盘踞势力：势力B
  - 内部矛盾：...
  - 和区域A的矛盾：...
- **区域间矛盾网络**：区域A的势力A ↔ 区域B的势力B（矛盾核心：XXX）——这是本卷白描的冲突地基

#### 整卷主要矛盾/主要剧情走向
- **主要矛盾**：（一句话——本卷主角要解决的核心问题是什么）
- **主要剧情走向**：（3-5句话——本卷从开头到结尾的大致走向，不是分幕，是方向）
```

### 2a-4. 快照自检

| 自检项 | 标准 | 不通过处理 |
|:--|:--|:--|
| 力量范围从骨架取？ | 不能自己发明力量层级 | 回骨架取 |
| 地图是空的吗？ | 每个区域必须填充盘踞势力+内部矛盾+区域间矛盾 | 补充实 |
| 势力矛盾网络有吗？ | 区域间矛盾必须标注矛盾核心 | 补矛盾 |
| 主角起点/终点快照有吗？ | 必须具体到能做什么/不能做什么 | 补具体 |
| 卷级敌人力量范围标注了吗？ | 必须和主角力量有差距 | 补差距 |
| 整卷主要矛盾一句话能说清吗？ | 不能模糊 | 重新提炼 |

**快照通过后，后续2b分支剧情线、2c分幕、2d白描都基于本卷设定快照，不再直接引用全书设定。**

## 2b. 分支剧情线（≥6条）

> **v3.3.0新增。** 先有线网，再组织成幕。每幕内部多线交织，不是单线跑。从2a本卷设定快照推导分支线。

### 2b-1. 列出≥6条分支剧情线

比例分布：主线1~2条 + 支线3~4条 + 暗线1~2条。

```markdown
### 分支剧情线

#### 主线（1~2条）
1. **[主线名]**（主角成长线）
   - 类型：主线
   - 起：[本卷开始时主角的状态/目标]
   - 止：[本卷结束时主角的状态/目标]
   - 涉及势力：势力A / 势力B
   - 涉及地图：区域A / 区域B
   - 和主要矛盾的关系：[这条线如何推进整卷主要矛盾]

2. **[主线名]**（卷级危机线，如有第二条主线）
   - ...

#### 支线（3~4条）
3. **[支线名]**（配角线/势力博弈线/悬念揭示线）
   - 类型：支线
   - 起：...
   - 止：...
   - 涉及势力：...
   - 涉及地图：...
   - 和主线的关系：[这条支线如何影响主线]

4. **[支线名]**
   - ...

5. **[支线名]**
   - ...

6. **[支线名]**
   - ...

#### 暗线（1~2条）
7. **[暗线名]**（为后续卷埋的伏笔）
   - 类型：暗线
   - 起：[本卷埋下的种子]
   - 止：[本卷末暗示，但不揭示——留给后续卷]
   - 涉及势力：...
   - 涉及地图：...
   - 预计揭示卷：第X卷

8. **[暗线名]**（如有第二条暗线）
   - ...
```

### 2b-2. 分支线自检

| 自检项 | 标准 | 不通过处理 |
|:--|:--|:--|
| 总数≥6条？ | 主线1~2+支线3~4+暗线1~2 | 补线 |
| 每条有起止吗？ | 起止必须具体到状态/目标 | 补具体 |
| 每条标注涉及势力/地图？ | 必须从2a快照取 | 回快照取 |
| 支线影响主线吗？ | 支线不能独立于主线——必须标注和主线的关系 | 补关系 |
| 暗线不揭示吗？ | 暗线本卷只埋种子不揭示，标注预计揭示卷 | 确认不揭示 |

## 2c. 分幕设计（卷40~60章/幕8~12章/4~5幕）

> **v3.3.0：基于分支线分幕。** 不再单线分幕——先把分支线分配到幕，每幕内部多线交织。每幕是完整起承转合单元，8~12章。

### 2c-1. 确定本卷章数和幕数

本卷总章数=40~60章（通常第一卷25章可放宽，但目标40~60章）。幕数=4~5幕，每幕8~12章。

```markdown
### 本卷结构
- 总章数：ch002-ch0XX（共XX章，ch001已由seed产出）
- 幕数：N幕
- 幕1：ch002-ch0XX（XX章）
- 幕2：ch0XX-ch0XX（XX章）
- ...
```

### 2c-2. 分配分支线到幕

把2b的分支线分配到每幕——每幕至少推进2~3条分支线（不是只推主线）。

```markdown
### 幕1：「幕名」（ch002-ch0XX，XX章）
- **起承转合**：起（XXX）→承（XXX）→转（XXX）→合（XXX）
- **本幕推进的分支线**：
  - 主线1：[本幕推进到哪]
  - 支线3：[本幕推进到哪]
  - 暗线7：[本幕埋下什么种子]
- **对赌难度**：阶位N（从2a快照取）
- **碰撞势力**：势力A（从2a快照取）
- **场景地标**：区域A（从2a快照取）
- **幕末钩子**：本幕结束时读者在想什么？
- **战斗场景配额**：每幕至少3场战斗场景（遇敌/越级/Boss），缺战斗场景的幕必须补齐。战斗场景在2d白描的场景表中体现。
- **出场角色**：{角色名}({定位}·{势力归属}), {角色名}({定位}·{势力归属})——定位取值：主角/配角/反派·{层级}；势力归属从骨架.md势力格局取

### 幕2：「幕名」（ch0XX-ch0XX，XX章）
- **起承转合**：...
- **本幕推进的分支线**：
  - 主线1：[本幕推进到哪]
  - 主线2：[本幕推进到哪]
  - 支线4：[本幕推进到哪]
- ...

### 幕3~幕N同上格式
```

每幕的设计原则：
- **对赌难度必须逐幕升级**
- **每幕必须有完整起承转合**（不是只有"弧线"）
- **每幕至少推进2~3条分支线**（不是只推主线）

### 2c-3. 分幕自检

| 自检项 | 标准 | 不通过处理 |
|:--|:--|:--|
| 每幕8~12章？ | 不能太短（<8）或太长（>12） | 调整章数 |
| 每幕有起承转合？ | 四段缺一不可 | 补完整 |
| 每幕推进≥2条分支线？ | 不能只推主线 | 分配更多线 |
| 对赌难度逐幕升级？ | 不能平级或倒退 | 调整难度曲线 |
| 总章数40~60？ | 卷的格式要求 | 调整幕数或幕章数 |

## 2d. 按幕白描（每幕一个单元）

> **v3.4.0：从"按场景拆+氛围描写"改为"按幕白描"。** 根因：深渊主宰白描是先有正文后拆书，按卷没问题；我们是推演，按卷无法保证全程精彩，按幕（8~12章）是推演能保证密度的合理粒度。参考假面僧匪幕纲格式——每幕一个单元，含五段式叙事梗概+场景表+幕末钩子。

### 2d-1. 每幕白描的格式

每幕一个完整单元，三部分：

```markdown
### 第X幕：「幕名」（ch0XX-ch0XX，XX章）

#### 五段式叙事梗概

【建置段】[2-4句叙事梗概——幕的起点：主角面对什么处境/做什么事。要有画面感，不是提纲]
【冲突段】[2-4句叙事梗概——冲突升级：主角遇到什么阻力/矛盾激化。要有画面感]
【转折段】[2-4句叙事梗概——转折点：局势如何反转/主角做出什么选择。要有画面感]
【高潮段】[2-4句叙事梗概——高潮爆发：主角如何爆发/敌人如何反击。要有画面感]
【收束段】[2-4句叙事梗概——收尾：高潮后的格局变化/为下一幕留什么。要有画面感]

#### 场景表

| 场景 | 地点 | 人物 | 事件 | 冲突 | 爽点 | 情绪 |
|:--|:--|:--|:--|:--|:--|:--|
| 1 | [地点] | [人物] | [事件] | [冲突] | [爽点/无] | [情绪关键词] |
| 2 | ... | ... | ... | ... | ... | ... |
| 3 | ... | ... | ... | ... | ... | ... |

#### 幕末钩子
[1-2句——本幕结束时读者在想什么？主角面对什么新困境/新悬念？]

#### 本幕推进的分支线
- 主线X：[本幕推进到哪]
- 支线X：[本幕推进到哪]
- 暗线X：[本幕埋下什么种子]

#### 主角力量成长线
- 起点：{本幕开始时主角的能力/阶位}
- 获得：{本幕中主角获得的新能力/技能/装备}
- 越级击杀：{本幕中主角越级击杀的敌人（如有）}
- 终点：{本幕结束时主角的能力/阶位}
```

### 2d-2. 五段式叙事梗概的写法

**参考假面僧匪的格式**——每段2-4句叙事梗概，有画面感，不是提纲。

| 禁止 | 要求 |
|:--|:--|
| "ch010，卡森动手了。切断了补给线。" | "公会断粮的第三天。委托板空了，墙角的漏雨滴在空椅子上。亚力克斯把最后一壶茶倒进杯子——没客人了，用不着泡两壶。'被掐住了。'" |
| "主角去丹峰谈合作。被拒绝了。" | "陈渡拿着《百草订单优化方案》走进丹峰采购接待室。程万里扫了一眼封面，把文件推回来——'你的杂物科，管不到丹峰的采购。'陈渡没走，他从袖子里抽出第二份文件：《丹峰采购成本对比分析》。" |
| 纯提纲式（"主角升级→打怪→获得装备"） | 叙事梗概式（有画面+有动作+有转折，2-4句说完一段剧情） |

**核心规则**：
- 每段2-4句，不是1句也不是10句
- 每段必须有画面感（读者能在脑子里看到的）
- 每段必须有事件推进（不是纯氛围描写——氛围留给write）
- 五段之间有因果链（建置→冲突→转折→高潮→收束，不是五个独立场景）

### 2d-3. 场景表的写法

场景表给write agent导航——每章/每场景有明确的地点/人物/事件/冲突/爽点/情绪，write照着写就行。

**每幕必须产出场景表**——不只是第一幕，2-5幕也必须有完整场景表（6~10场景/幕）。

**字段说明**：
- **地点**：从2a快照地理范围取（用充实的实地图，不是空区域名）
- **人物**：本场景出场的角色（主角+配角+反派）
- **事件**：本场景发生的核心事件（一句话）
- **冲突**：本场景的冲突点（主角vs谁/矛盾核心）
- **爽点**：本场景的爽点类型（压力→突破/弱→强/未知→揭示/被欺→反击/无）
- **情绪**：本场景的情绪基调（冷寂/压抑/热血/温馨/紧张...）

**场景数量**：每幕8~12章，场景表通常6~10个场景（有些章共用场景，有些章一个场景跨多章）。

### 2d-4. 推演流程

基于2a本卷设定快照+2b分支剧情线+2c分幕设计，**按幕逐个写白描**。每幕一个完整单元（五段式+场景表+钩子+分支线）。

### 推演步骤

1. 从2c-2取第一幕的起承转合+本幕推进的分支线
2. 查2a快照地理范围→确定场景发生的地标
3. 查2a快照力量范围→确定主角当前对赌难度
4. 查2a快照势力矛盾网络→确定碰撞的势力和矛盾核心
5. 查2a快照危机范围→确定推进的倒计时
6. 写五段式叙事梗概（建置→冲突→转折→高潮→收束，每段2-4句）
7. 写场景表（6~10个场景，每个标注地点/人物/事件/冲突/爽点/情绪）
8. 写幕末钩子
9. 标注本幕推进的分支线
10. **进入下一幕，重复1-9**

**关键**：每幕独立推演，不一口气写整卷。推演一幕的密度远高于推演整卷。

---

### 2d-5. 白描质量自检

每幕白描写完后，逐条自检：

| 自检项 | 标准 | 不通过处理 |
|:--|:--|:--|
| 五段式完整吗？ | 建置/冲突/转折/高潮/收束缺一不可 | 补完整 |
| 每段2-4句吗？ | 不是1句提纲，也不是10句水文 | 调整长度 |
| 有画面感吗？ | 读完能浮现场景——不是"谁做了什么" | 补画面 |
| 场景表完整吗？ | 6~10个场景，每个6个字段都填 | 补字段 |
| 幕末钩子有吗？ | 1-2句，读者有追读欲望 | 补钩子 |
| 分支线标注了吗？ | 本幕推进≥2条分支线 | 补交织 |
| 多线交织吗？ | 五段式内部推进≥2条分支线——不是单线跑 | 补交织 |
