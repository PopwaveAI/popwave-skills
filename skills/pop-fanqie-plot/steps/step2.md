# Step 2: 世界构筑

> 基于创意文档展开完整故事世界。这是plot的核心环节——四张地图+人物包是Step 4剧情白描推演的前置输入。
> 世界构筑的质量直接决定剧情白描的质量。

---

## 2a. 四张地图（**红线·独立产出**）

四张地图分别独立产出，禁止散落。每张地图必须与创意文档的世界观骨架自洽。

### 地理地图

摄像头位置——主角的POV活动范围。

| 要素 | 内容 |
|------|------|
| 地标清单（5-15个） | 每个地标有名+一句话功能描述 |
| 地标归属 | 哪个地标属于哪个势力？（与势力地图挂钩） |
| 进度锚点 | 第1幕/第2幕/第3幕分别在哪些地标发生？ |
| 信息差标注 | 哪些地标主角一开始看不到？（紫色势力所在地等） |

### 力量地图

主角成长曲线+反派强度梯度。

| 要素 | 内容 |
|------|------|
| 主角等级阶梯（5-10级） | LV1起点 → LV5第一卷终点 → LV10全书终点。每级有触发条件+新增能力 |
| 反派强度对照 | 4层敌人分别对应什么等级（街头层LV2-3/地方势力LV4-5/执法系统LV6-7/幕后金主LV8-10） |
| 金手指升级路径 | 与主角等级阶梯对齐——什么时候解锁什么新功能 |
| 成长压力点 | 哪一关主角必须越级挑战？（爽点制造点） |

### 势力地图

势力碰撞和演变。

| 要素 | 内容 |
|------|------|
| 势力金字塔（3-4层） | 底层街头帮派→中层地方势力→上层执法系统→顶层幕后金主 |
| 势力关系图 | 谁对抗谁？谁利用谁？谁暗中操控谁？ |
| 势力变化轨迹 | 第一卷结束时势力关系如何变化？（主角从棋子→变量） |
| 主角位置 | 在势力博弈中的位置+如何迁移 |

### 危机地图

时间压力源和倒计时。

| 要素 | 内容 |
|------|------|
| 压力源矩阵（3-4层） | 金手指内部压力/外部势力压力/世界级危机/长线悬念压力 |
| 倒计时清单 | 每个倒计时有触发条件+到期后果+推进节奏 |
| 危机叠加节点 | 哪几个倒计时会在同一章叠加？（高潮制造点） |
| 危机与世界危机挂钩 | 表层金手指压力如何指向深层世界危机 |

## 2b. 完整人物包

从创意文档继承主角+核心羁绊角色，补充完整人物包。

### 继承自seed

- 主角设定卡+主角深度设计（seed产出4a+4b）
- 核心羁绊角色（seed产出4c，至少2人）

### 补充·第一卷敌人4层梯度（**红线**）

第一卷必须设计4层反派梯度，每层至少1个具名反派。这是Step 4推演剧情白描的核心前置输入。

| 层级 | 定位 | 出场时机 | 设计要素 |
|------|------|---------|---------|
| 街头层 | 第一卷前1/4对手（ch02-05） | ch01已登场或ch02登场 | 名字/核心欲望/致命弱点/与主角的第一次冲突 |
| 地方势力层 | 第一卷中段对手（ch06-15） | 第一幕末或第二幕登场 | 名字/势力规模/核心欲望/致命弱点/与街头层的关系 |
| 执法系统层 | 第一卷后段对手（ch16-25） | 中后段登场 | 名字/系统权力/核心欲望/致命弱点/与地方势力的勾连 |
| 幕后金主层 | 第一卷终极boss+全书暗线 | 卷末揭示，本卷只露冰山一角 | 名字/家族或组织/真实威胁等级（远超主角当前能力）/核心欲望/与前三层的操控关系 |

**每层反派都必须给主角成长空间**——不是被主角正面硬刚打倒，而是通过弱点/信息差/势力博弈被破解。

**幕后金主层的特殊设计**：第一卷只揭示其存在（"紫色势力"），主角意识到自己打掉的只是棋子——为卷2预留长线。

## 2c. 四图自洽检查

四张地图+人物包产出后，必须做自洽检查：

1. **地理地图与势力地图自洽？** — 每个地标都有归属势力
2. **力量地图与人物包自洽？** — 4层敌人等级梯度与主角成长阶梯对齐
3. **势力地图与危机地图自洽？** — 每层势力都有自己的倒计时/压力源
4. **危机地图与世界危机自洽？** — 表层金手指压力能追溯到深层世界危机

任何一项不自洽→重设计。
