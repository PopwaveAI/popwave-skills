# Step 2: 第一卷详规

---

## 2a. 第一卷终点和boss

第一卷的终点事件（一段话）+ boss是谁（一句话+核心欲望+致命弱点）。

boss优先从seed的分层反派低级或中期层选取。

---

## 2b. 配角设计（2-4个）

主角设定卡已由seed产出，plot只设计配角。每个配角：
- 名字
- 核心欲望（一句话）
- 功能分类（情感锚点/终极对手/镜像对手/逆徒/助力者）
- 与主角的关系张力（一句话——什么让他们冲突或绑定）

---

## 2c. 幕序列（3-5幕）

每幕：
- 幕名
- 功能位（建置/冲突升级/高潮/余波）
- 一句话弧线（这幕发生了什么核心事件）
- 章节预算（约X章）

第一幕参考——黄金三章法则（番茄通用，给write的方向参考）：
- ch01：核心冲突爆发——读者知道"发生了什么事"
- ch02：金手指激活——读者知道"主角凭什么翻盘"
- ch03：短期目标明确——读者知道"接下来看什么"
- ch04-07：第一次公开打脸——兑现第一个爽点
- ch08-10：绑定强力助攻——埋下长线悬念

---

## 2d. 悬念分层设计

分3层：

| 悬念层级 | 说明 | 数量 |
|---------|------|------|
| 核心悬念 | 全书要解答的大问题（继承seed主线悬念） | 1条 |
| 场景悬念 | 某个场景中的具体障碍/未解问题 | 2-3条 |
| 段落悬念 | 一连串场景的共同未知/伏笔 | 1-2条 |

每条标注触发章（大约第几章揭示）。

---

## 2e. 核心高潮点（2-3个，三段式）

每个高潮点三段式设计：
- **低谷**：主角遭遇什么困境/压制（一句话）
- **高潮**：主角如何爆发/反转（一句话）
- **变化**：高潮后主角/格局发生了什么变化（一句话）

---

## 2f. 分层反派设计（3层）

优先继承seed世界观骨架中的设定，补充细节：

| 层级 | 定位 | 解锁时机 | 核心欲望 | 致命弱点 |
|------|------|---------|---------|---------|
| 低级 | 第一卷前半段对手 | 第一卷前半段 | | |
| 中期 | 第一卷后半段-第二卷对手 | 第一卷后半段 | | |
| 终极 | 全书终局黑手 | 全书终局 | | |
