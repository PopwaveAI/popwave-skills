# Step 3: 第一卷详规

基于世界构筑（Step 2）设计第一卷结构详规。

---

## 3a. 第一卷终点和boss

第一卷的终点事件（一段话）+ boss是谁（一句话+核心欲望+致命弱点）。

boss优先从Step 2的敌人4层梯度中选取——第一卷boss通常是地方势力层或执法系统层，幕后金主层只露冰山一角。

终点事件必须与世界危机挂钩——本卷结束时，主角应第一次感知到世界危机的存在（不是完全揭示，而是"水面下的暗流"）。

---

## 3b. 配角设计（2-4个）

主角设定卡已由seed产出，核心羁绊角色已由seed产出，plot只设计配角。每个配角：
- 名字
- 核心欲望（一句话）
- 功能分类（情感锚点/终极对手/镜像对手/逆徒/助力者）
- 与主角的关系张力（一句话——什么让他们冲突或绑定）

---

## 3c. 幕序列（3-5幕）

每幕：
- 幕名
- 功能位（建置/冲突升级/高潮/余波）
- 一句话弧线（这幕发生了什么核心事件）
- 章节预算（约X章）

**注意**：ch01已由seed的黄金首章产出。第一幕从ch02开始：

- ch01：黄金首章（seed产出，已建立冲突+金手指+钩子）
- ch02-03：延续ch01钩子，短期目标明确——读者知道"接下来看什么"
- ch04-07：第一次公开打脸——兑现第一个爽点
- ch08-10：绑定强力助攻——埋下长线悬念

---

## 3d. 悬念分层设计

分3层：

| 悬念层级 | 说明 | 数量 |
|---------|------|------|
| 核心悬念 | 全书要解答的大问题（继承seed主线悬念） | 1条 |
| 场景悬念 | 某个场景中的具体障碍/未解问题 | 2-3条 |
| 段落悬念 | 一连串场景的共同未知/伏笔 | 1-2条 |

每条标注触发章（大约第几章揭示）。

---

## 3e. 核心高潮点（2-3个，三段式）

每个高潮点三段式设计：
- **低谷**：主角遭遇什么困境/压制（一句话）
- **高潮**：主角如何爆发/反转（一句话）
- **变化**：高潮后主角/格局发生了什么变化（一句话）

---

## 3f. 势力格局（从世界构筑继承+展开）

继承Step 2的势力地图，在本卷具体化：

| 势力 | 掌握资源 | 本卷角色 | 与主角关系 |
|------|---------|---------|-----------|
| 势力A | | 前期对手/幕后操纵 | |
| 势力B | | 潜在盟友/竞争者 | |
| 主角位置 | | 被利用的棋子→独立变量 | |

势力变化：本卷结束时势力关系如何变化？

---

## 3g. 分层反派设计（从世界构筑继承）

继承Step 2的敌人4层梯度，补充本卷细节：

| 层级 | 定位 | 解锁时机 | 核心欲望 | 致命弱点 |
|------|------|---------|---------|---------|
| 街头层 | 第一卷前半段对手 | ch01-05 | | |
| 地方势力层 | 第一卷后半段-第二卷对手 | ch06-15 | | |
| 执法系统层 | 第一卷后段对手 | ch16-25 | | |
| 幕后金主层 | 全书终局黑手 | 卷末揭示 | | |

---

## 3h. 世界危机映射

继承创意文档的世界危机，标注在本卷的体现：

| 危机层级 | 本卷体现 | 触发章 |
|---------|---------|--------|
| 表层危机 | 主角直接面对的威胁（具体事件） | |
| 深层危机 | 暗线暗示——读者隐约感觉"世界不对劲" | |
| 终极危机 | 本卷不揭示，但埋下伏笔 | |

**原则**：第一卷只展示表层危机+暗线暗示深层危机，终极危机留到后续卷。
