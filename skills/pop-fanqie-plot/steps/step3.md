# Step 3: 落盘+章锚点表

> 产出两个文件给write层消费。

---

## 3a. 落盘剧情白描.md（最核心产出）

格式：
```markdown
# 《书名》· 第一卷剧情白描（ch001-ch025）

> ch001已由seed的黄金首章产出，本白描从ch02开始。
> ch001锚点：[核心事件摘要] + [章末钩子]

（Step 2产出的整卷叙事流白描，分场景标注信息差/伏笔/钩子）

---
卷末钩子：[本卷最后的悬念]
```

---

## 3b. 提取章锚点表.md

从剧情白描中提取每章的锚点信息。每章一行：

```markdown
# 《书名》· 第一卷章锚点表

| 章号 | 场景 | 章型 | 核心事件（1-2句） | 爽点 | 钩子 | 预期回收章 |
|:--|:--|:--|:--|:--|:--|:--|
| ch001 | 赌场 | opening_shift | （从seed黄金首章提取） | 赌局翻盘 | 巷子里有眼睛盯着 | ch002 |
| ch002 | 贫民区·家 | confrontation_pressure | 灰蛇派人堵门→亚伦翻屋顶逐个摸掉，用命运卡牌逼秃鹫带路 | 6人反杀 | 系统弹出失衡值警告 | ch003 |
| ... | ... | ... | ... | ... | ... | ... |
```

**章型映射**（从白描的氛围+事件推导，不硬塞4种）：

| 白描特征 | 映射章型 |
|:--|:--|
| 遭遇新势力/被堵门/正面冲突 | confrontation_pressure |
| 揭秘/发现真相/信息释放 | reveal_hook |
| 战斗/绝境翻盘 | combat_reversal |
| 场景切换/新地图/身份变化 | opening_shift |

```

📄 章锚点表.md —— [N]章，每章锚点就位

下一步：进入 write 管线从 ch002 开始逐章渲染正文
```
