# CHANGELOG

## v3.0.0 (2026-07-16)

### 从评分模型重构为检查+沉淀

**根因**：R4验证打分模型占用创作空间。时停邪神拆解揭示白描卡格式（事件白描+关键数据+钩子追踪）比评分更有效。

**改动**：
- 从5维度评分→定性判断（不打分）
- 从概念审核→AI味检查+正文精彩度检查
- 从事实快照（概念状态/单元进度/变体追踪）→剧情快照（事件白描+关键数据+钩子追踪+下章建议）
- 钩子追踪从单栏升级为三栏闭环（埋设/回收/待回收）

## v2.1.0 (2026-07-15)

### 适配单元剧模式

**根因**：plot v5.0.0重构为单元剧蓝图设计器后，review需要追踪单元进度和长线碎片。

**改动**：
- 1a概念兑现度检查中"概念变体"改为"单元变体（来自变体池）"
- 快照更新新增「单元进度」（当前单元编号+节拍位置+本单元变体）和「长线追踪」（碎片累计+距下次推进）
- 下章建议从"概念变体池+节奏模型"改为"单元模板+变体池+长线追踪"
