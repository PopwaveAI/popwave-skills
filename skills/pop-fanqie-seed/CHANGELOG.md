# CHANGELOG

## v10.0.0 (2026-07-16)

### 世界自洽+金手指限制取代价+推导SOP

**根因**：R19测试发现世界不自洽（灵债数值对不上/从何来/如何还/为什么扣记忆都答不上来）；金手指加了代价（灵债涨=扣寿命）让读者不爽；1e立项展开只有格式没有SOP——主角/世界观/长线全是agent即兴填空。

**改动**：
- 金手指铁律：只加限制不加代价（使用条件/边界/冷却可以，扣寿命/扣记忆等负反馈禁止）
- 1e立项展开每个产出加推导逻辑：主角从金手指反推、世界观从金手指推导规则、长线从进化方向推导
- 1b搜索改为标签三层搜索（主题/角色/情节标签组合），找5-8本不只3本头部
- 1c创意发散新增换元法（一度置换/二度置换/极端法/相反法）
- Phase 2新增2c世界自洽检查（四问：能力从何来/触发条件/作用边界/世界规则如何执行）
- Phase 2新增2d金手指代价检查（有代价打回重设计为限制）
- 落盘格式更新：金手指从"创意×机制×限制"改为"功能/触发条件/限制边界/进化方向"
- showcase.md更新：示例表加触发条件+限制边界列，新增限制vs代价对比表
- 红线从4条改为5条
- 吸收外部PE参考：升级流指令金手指框架/换元法置换思路/番茄过稿指令创新点
