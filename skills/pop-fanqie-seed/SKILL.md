# pop-fanqie-seed · 番茄梗搜索引擎

> 自由出创意，结构做打磨。创意火花在最无约束的环境里才烧得最旺。
> 两阶段：Phase 1 纯自由产出5-6个方向→用户选→Phase 2 结构化打磨落盘。

---

## SOP骨架

### Phase 1: 纯自由发散

**核心原则：不加任何约束。** R9测试验证——纯自由模式（11字符system+一句话user）产出的创意质量远高于注入搜索数据/格式模板/结构引导的模式。"中医把脉诊断股票""殡葬直播间全网等投胎"这些最佳梗都是纯自由模式下诞生的。

**1a. 问需求**

第一步问用户想写什么。根据回答分路径：

| 用户回答 | 路径 |
|---------|------|
| 直接给了梗/画面/一句话 | 跳过Phase 1，直接进Phase 2打磨 |
| 给了方向/偏好（"想写都市异能""想写神医文"） | 进1b纯自由发散 |
| 完全开放（"你看着来"） | 进1c扫榜定方向再发散 |

**1b. 纯自由发散**

拿到用户方向后，**不给API/agent任何额外约束**——不注入搜索数据，不给格式模板，不给结构引导，不提示"3选1"。只给一句话方向，让模型自由发挥。

**发散指令**（这是给子agent/API的全部输入）：

```
system: 你是一个网文创意编辑。
user: 我想写{用户方向}，请帮我生成{N}个完全不同方向的开书创意种子。
```

N = 5-6（不是3。R9证明模型在多产出时创意发散性更强，3个太少容易同质化）

**关键约束（给agent的，不给API的）**：
- 5-6个方向必须差异最大化——不同切入点、不同主角类型、不同情绪基调
- 不套框架，不加限制，不想机制——只管有没有意思
- 参考"给怪谈写员工手册""中医把脉诊断股票"这种好梗的特质：一个具体行为×一个反差场景，一句话就能让人脑补画面

**1c. 扫榜定方向（用户完全开放时）**

用户完全开放时，先扫番茄/起点热门榜单找到当前火的题材，再用1b纯自由发散。

搜索3轮：
- "番茄小说/起点中文网" + "热门/排行榜/2025" → 找当前什么在火
- "B站/抖音/知乎" + "小说推荐/出圈" + "2025" → 找正在出圈的概念
- "网文" + "新题材/新趋势" + "2025" → 找正在冒头的新方向

扫榜结果**只用于确定方向**，不注入发散prompt。确定方向后走1b。

**1d. 用户选择**

把5-6个方向呈现给用户，让用户选1个。格式：

```
## 创意方向

### 创意1：[名称]
一句话描述：[主角是谁，在做什么，遇到了什么]
画面：[脑补的场景，最好带一句台词]
为什么有意思：[吸引人的点]

### 创意2：...
...
### 创意6：...
```

用户选1个，或要求重新发散。

**无用户交互时**（API直调/自动化测试）：自动选最有画面感和反差感的1个，继续Phase 2。

### Phase 2: 结构化打磨

用户选定方向后，进入结构化打磨。这一阶段的作用不是改创意，而是**给选定的创意装上引擎、验证质量、写成简介**。

**2a. 行为引擎检查**

检查选定创意是否已有行为引擎——主角每章在做一件具体的事（谈判/把脉/写手册），还是在操作一个系统（抽卡/升级/兑换）。

- **已有行为引擎** → 跳过碰撞，直接进2b
- **没有行为引擎** → 用行为框架碰撞补上

行为框架来源——与赛道无关的日常领域，自带流程、甲乙方关系、可博弈规则：

| 素材池 | 示例行为 |
|-------|---------|
| 职场行为 | 签合同/写手册/定KPI/走流程 |
| 社会服务 | 接投诉/送快递/做客服 |
| 法律框架 | 维权/执法/合规审查 |
| 民间习俗 | 讨封问话/红白事流程 |
| 日常生活 | 做饭/装修/带娃/相亲 |

碰撞方法：把行为框架的流程步骤逐条映射到创意场景。例——"殡葬直播间"碰撞"客服"框架：接单→亡魂提需求→走流程完成→满意度评分→获得功德值。每条映射=一个章节。

**2b. 合成金手指**

```
金手指 = 梗（提供形式——怎么表现）
       × 机制（提供动力——为什么有效果）
       × 限制（提供张力——为什么不能随便用）
```

**合成示例库（14个showcase）：**

*用户原创型（4个）：*

| 梗 | 机制 | 限制 | 合成金手指 |
|----|------|------|-----------|
| 大周礼时代（模仿古人说话） | 言出法随（说话有效果） | 每人一生仅生效一次 | 用周礼体说话能引人向善，但每个人只生效一次——要选对时机 |
| 黄鼠狼讨封（民间讨封梗） | 话术成真（说出口就当真） | 只能用公文套话格式 | 被妖怪拦住问话时用红头文件格式回答，对方必须照办——但格式错了也照办 |
| 全县每人每天给我一块钱 | 价值交换（给什么得什么） | 只有328元本金 | 花1块钱买1年修为，但穷到只有328元——每笔钱要精算 |
| 反女拳复仇 | 因果倒转（恶行反弹） | 只反弹女拳行为不反弹其他 | 被女拳攻击时伤害反弹——但必须先承受一次攻击才能反弹 |

*番茄top20逆推型（6个）：*

| 梗 | 机制 | 限制 | 合成金手指 | 来源 |
|----|------|------|-----------|------|
| 死亡游戏大逃杀 | 通关获得规则能力 | 每关规则不同必须变换解法 | 在死亡游戏中通关获得规则能力，但每关规则不同必须变换解法 | 十日终焉 |
| 工地实习挖出怪物 | 击杀吸收（杀A获得A能力） | 能力链依赖（必须用A能力杀B） | 杀怪物获得该能力，但必须用当前能力杀下一个 | 大一749 |
| 每天一分钟时间静止 | 命运轮盘赌命收容禁忌物 | 只有一分钟+赌命 | 每天时停一分钟收容禁忌物，时间极短且赌命 | 时停起手 |
| 重生抢在内测前 | 盗神天赋偷NPC东西带入现实 | 内测只有7小时不可重复 | 重生回内测前抢S级天赋，7小时内偷的东西带入现实 | 游戏入侵 |
| 诸神降临世界变信仰游戏 | 试炼通关获得资源 | 7天强制试炼不参加就死 | 通关试炼获得资源，7天强制参加不打就死 | 诸神愚戏 |
| 五百年前风水师醒来 | 风水术法有效 | 现代社会规则不同+对手也有术法 | 五百年前风水师在现代社会用风水术，现代规则不同且对手也有术法 | 风水之王 |

*新创作型（4个）：*

| 梗 | 机制 | 限制 | 合成金手指 |
|----|------|------|-----------|
| 中医把脉×金融分析 | 望闻问切诊断一切 | 主角不懂金融只懂中医 | 用中医把脉诊断股票走势，准确但不懂金融术语无法解释 |
| 扶不扶（老人摔倒社会梗） | 善行值兑换 | 只有扶了被讹的人才有值 | 扶被讹的老人获得善行值兑换能力，越被骂越强 |
| 打工人魂穿老板 | 魂穿控制 | 只能穿24小时且对方必须是压榨过自己的人 | 魂穿压榨过自己的老板24小时，只限仇人 |
| 全网黑转粉 | 黑粉值转化 | 必须真的被误解不能自导自演 | 被全网黑时获得能力加成，自导自演无效 |

**反面教材（不叫化学反应，叫贴标签）：**
- ❌ "周礼×都市×搞笑" → 没有机制没有限制，不是金手指
- ❌ "体制内×规则怪谈" → 两个赛道标签拼在一起，没有故事引擎
- ❌ "重生+都市+复仇" → 三个赛道标签，缺机制缺限制
- ❌ "每天随机天赋+三合一合成" → 纯系统机制，无行为引擎，主角在操作系统而非做事

**2c. 四眼法验证**

验证对象是合成后的金手指：

**2c-1. 有没有画面？**
- ✅ "用周礼体说话引人向善，每人一次" → 画面立刻浮现
- ❌ "获得成长型系统" → 没画面

**2c-2. 限制条件是不是天然存在？**
- ✅ "每人一生一次" → 限制天然存在
- ❌ "无敌系统" → 没限制

**2c-3. 能不能想象出十几种不同场景？**
- ✅ "周礼体引人向善" → 校园怼霸凌者、商战怼奸商、直播怼喷子……
- ❌ "获得读心术" → 第3个场景就腻

**2c-4. 有没有行为引擎？**
- ✅ "给怪谈写员工手册" → 每个新怪物=新合同谈判=新冲突，行为驱动
- ❌ "每天随机天赋掉落" → 每天抽卡=换数值，机制驱动
- 判断标准：主角每章在做一件具体的事（谈判/投诉/写手册），还是在操作一个系统（抽卡/升级）？

**验证结果**：
- 四项全过 → 放行，进Step 3
- 任何一项不过 → 打回，换机制或调整限制，重新合成

### Phase 3: 落盘

**只产出一个文件**：`梗.md`

落盘格式 = 结构化字段 + **散文体番茄简介**。简介不是概述，是**小说开头**——第一人称，带对话，带情绪弧线，带章末钩子。

```
# 梗

## 金手指
- 梗：（一句话）
- 梗来源：（用户选定的方向 + Phase 2打磨过程）
- 行为引擎：（主角每章在做什么具体行为）
- 机制：（一句话）
- 限制：（一句话）
- 合成金手指：（一句话——梗×机制×限制的产物）

## 番茄简介
【关键词1】+【关键词2】+【关键词3】+【关键词4】

（150-300字散文体简介，必须包含：
- 梗展示：用故事而非描述展示金手指
- 限制暗示：通过情节让读者感受到限制的存在
- 第一个爽点的期待感：结尾留钩子
- 对话：至少2句角色台词，展现人物性格
- 情绪弧线：主角从被动到主动的转变）
```

**简介写法参考**（来自R7非skill组+R9纯自由组的高质量产出）：

> 我叫林北，考研失败后回到十八线小县城，继承了爷爷留给我的殡葬铺。
>
> 爷爷临终前说："北北，这铺子连着阴间，你要好好干。"
>
> 我以为是老人家糊涂了，直到我打开了直播。
>
> 第一单，来了一位穿红裙子的女鬼，她哭着说："主播，我死得太冤了，帮我找到凶手。"……

这段好在哪里：第一人称代入、对话展现人设、从日常到异常的转折自然、结尾留悬念。**不是描述创意，是在讲创意的故事。**

不写种子文档。不写100章推导。不写世界规则。不写节奏框架。不写笔触风格。
**梗就是种子。金手指就是引擎。拿到梗直接去写。**

## 红线

1. Phase 1发散阶段必须纯自由——不注入搜索数据/格式模板/结构引导，只给一句话方向
2. Phase 1必须产出≥5个差异化方向——不得只给3个，不得同赛道
3. Phase 2只在用户选定后启动——不在发散阶段做结构化
4. 行为引擎检查是Phase 2第一步——有则跳过碰撞，无则才补
5. 金手指三要素齐全——梗×机制×限制缺一不可
6. 四眼法四项全过才放行
7. 只产出`梗.md`——不写种子文档/世界规则/节奏框架/笔触风格
8. 梗不得与番茄Top20核心机制雷同
9. 番茄简介必须是散文体——带对话、带情绪弧线、带钩子，不是概述
10. 无用户交互时自动选最优方向继续——不得卡在确认门
