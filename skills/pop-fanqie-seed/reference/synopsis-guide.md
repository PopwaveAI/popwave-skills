# 番茄简介写法参考

## 要求
- 150-300字，第一人称散文体（不是概述）
- 必须包含：创意展示（用故事展示金手指）、限制暗示（通过情节让读者感受到限制）、第一个爽点的期待感（结尾留钩子）
- 至少2句角色台词，展现人物性格
- 情绪弧线：主角从被动到主动的转变

## 优秀示例

> 我叫林北，考研失败后回到十八线小县城，继承了爷爷留给我的殡葬铺。
> 爷爷临终前说："北北，这铺子连着阴间，你要好好干。"
> 我以为是老人家糊涂了，直到我打开了直播。
> 第一单，来了一位穿红裙子的女鬼，她哭着说："主播，我死得太冤了，帮我找到凶手。"……

## 这段好在哪里
1. 第一人称代入——读者立刻进入主角视角
2. 对话展现人设——爷爷的叮嘱和女鬼的哭诉，两句话两个人物
3. 从日常到异常的转折自然——殡葬铺→直播→女鬼，层层递进
4. 结尾留悬念——读者想知道凶手是谁

**核心原则：不是描述创意，是在讲创意的故事。**
