## 1c. 市场调研（WebSearch + 借鉴点/避雷点分析）

> 如1b-2并发中③已启动，等待返回后消费落盘文件，跳过1c-1~1c-3，直接确认1c-4落盘。
> **产出是"借鉴点+避雷点清单"，不是搜索数据本身。搜索数据消化完后不进入发散prompt。**

### 1c-1. WebSearch（3轮）

- 第1轮：`"番茄小说/起点中文网" + "{赛道} + 热门/排行榜/2026"` → 找榜单热门
- 第2轮：`"B站/抖音/知乎" + "{赛道} + 小说推荐/出圈" + "2026"` → 找出圈概念
- 第3轮：`"网文" + "{赛道} + 新题材/新趋势/审美疲劳" + "2026"` → 找新方向+疲劳模式

### 1c-2. 市场分析 + 产出校准清单

对每部热门作品分两层标注（借鉴：可借鉴的机制/爽感/节奏；避雷：必须避开的已有设定/金手指/开局），产出校准清单。

## 1d-0. 改编策略分析

> **核心：在5+5发散前，先用参考书DNA+赛道调研+用户元素+情绪+爽点类型组装3-5个"改编策略"，用户选1-2个策略后，1d双轨发散在策略框架下产出10个创意。策略是生成器，创意是产物。**

### 1d-0-A. 素材盘点

| 素材 | 来源 | 1d-0用途 |
|:--|:--|:--|
| 赛道借鉴点+避雷点 | 1c市场调研 | 策略的爽点类型选择 |
| 参考书·力量体系DNA | decon-lite表1 | 策略的"借什么力量体系" |
| 参考书·势力格局DNA | decon-lite表6 | 策略的"借什么势力结构" |
| 参考书·金手指DNA | decon-lite表7 | 策略的"借什么金手指机制" |
| 参考书·世界危机DNA | decon-lite表8 | 策略的"借什么危机引擎" |
| 用户元素 | 七维底牌+用户意图 | 策略的"融入什么元素" |

**缺失项检查**：情绪基调（1a已收集，未收集则追问）；全书最大卖点（1a已收集，未收集则追问）；爽点类型清单（从1c借鉴点提炼，未提炼则提取）。

### 1d-0-B. 策略组装卡（3-5个）

→ **组装卡结构见 `templates/Phase1-模板.md`·策略组装卡**

**组装原则**：3-5个策略的改编维度必须差异化；策略是"生成器"不是"成品"；必须和参考书DNA结合；无参考书时从赛道调研提炼切入策略。

### 1d-0-C. 用户选策略

用户选1-2个策略，可混搭（策略A的改编维度+策略B的爽点类型）。无用户交互时默认选第1个策略。

### 1d-0-D. 策略→发散的传递

用户选定策略后，1d双轨发散prompt注入策略框架：

**A轨（王道）**：`system: 你是一个网文创意编辑。 user: 赛道：{方向}。改编策略：{改编维度+借什么+改什么}。已验证卖点：{卖点列表}。避雷点：{避雷点}。爽点偏好：{策略选定爽点}。情绪基调：{策略选定情绪}。用融合策略产出5个王道方向开书创意。`

**B轨（猎奇）**：`system: 你是一个网文创意编辑。 user: 我想写{方向}，改编策略：{改编维度+借什么+改什么}。爽点偏好：{策略选定爽点}。情绪基调：{策略选定情绪}。生成5个完全不同方向的开书创意种子，形式自由。`

**关键约束**：策略定方向不定形式；B轨在策略框架内仍自由发散；不破坏红线2（纯自由发散）——B轨system只有一句话。

## 1d. 双轨发散：王道×猎奇

> 在1d-0选定的策略框架下，A轨扫榜→提取卖点→融合新元素产出5个，B轨纯自由发散产出5个。共10个。

### 1d-A. 王道赛道（产出5个）

#### 1d-A-1. 扫榜
> 如1b-2中④已启动扫榜子agent，直接消费返回结果。否则执行2轮搜索：
- `"番茄小说" + "{赛道}" + "排行榜/热销/2026"` → top 5
- `"{赛道}" + "起点" + "精品/万订/2026"` → 补top 3

#### 1d-A-2. 提取核心卖点
每本top书提取1句话"可迁移的核心卖点"（不是抄设定，是理解读者为什么买单）。只提取卖点机制，不提取具体设定。

#### 1d-A-3. 融合发散
5种融合策略：卖点平移/卖点叠加/反向操作/设定替换/情绪基调换挡。基于提取的5-8个卖点产出5个方向。

#### 1d-A-4. 质量门禁
每个王道方向必须：有卖点基底+有差异化+不踩避雷点+有画面。不通过→重新发散。

### 1d-B. 猎奇赛道（产出5个）

**发散指令**：`system: 你是一个网文创意编辑。 user: 我想写{用户方向}，请帮我生成5个完全不同方向的开书创意种子。`

**关键约束（agent自检，不给API）**：
- 5个方向差异最大化——不同切入点/主角类型/情绪基调/创意类型
- **创意类型强制多样化**：10个创意（王道5+猎奇5）必须覆盖≥3种创意类型（反转型/阶级攀爬型/生存竞技型/重建型/身份揭秘型），禁止全偏一种
- **情绪基调强制多样化**：10个必须覆盖≥5种情绪基调（热血逆袭/暗黑复仇/悬疑烧脑/都市恐怖/情感治愈/荒诞轻喜剧/末世求生/权谋博弈）
- 画面优先——每句话读完脑子里立刻浮现场景

**好创意六标准**（全过才放行）：
- 读者三标准：有画面+有张力（反差/攀爬压力/生存压力/揭秘悬念任一）+有意思
- 作者三标准：有世界（力量体系+隐藏规则）+有舞台（世界危机驱动100章）+有主角（特殊性含金手指+力量路径在世界力量体系内）

**反面教材**：纯事件无特殊性（穿越→系统→升级）/纯机制无舞台/纯势力操盘无金手指。

### 1d-C. 双轨市场校准

用1c校准清单统合校验：踩避雷点=淘汰；用到借鉴点=加分。王道额外校验卖点基底+差异化（最多淘汰2个）；猎奇额外校验画面+张力+有意思（重新发散补足5个）。最终呈现10个。
