# Step 1: 参考书摸底 + 市场调研 + 双轨发散（Phase 1）

> **核心原则：双轨发散——A轨王道赛道扫描排行榜借鉴+融合微调（5个），B轨猎奇赛道纯自由发散（5个）。共10个立项。**
> R9验证纯自由发散创意质量高但偏向猎奇（"殡葬直播间""中医诊断股票"），R13/R14验证借鉴排行榜是出好创意的重要校准。两者互补：王道给安全牌，猎奇给出圈牌。

## 1a. 七维底牌摸底

> **如果由 pipeline Phase 0 调起**：pipeline 已完成用户意图深问（`素材/用户意图.md` 存在 + project-state.md phase=phase1），seed 跳过七维摸底，直接进 1b 闸门消费已有底牌，然后进 1b-2 并发推进（如底牌未就绪）。
> **如果 seed 独立启动**（无 pipeline）：完整执行以下七维摸底。

第一步同时摸排**方向**和**七类底牌**。七类底牌必须在进入1b闸门前全部回答完毕。

### 方向摸底

| 用户回答 | 路径 |
|---------|------|
| 直接给了创意/画面/一句话 | 跳过Phase 1发散，直接进Step 2打磨 |
| 给了方向/偏好（"想写都市异能""想写神医文"） | 进1b底牌闸门→1c市场调研→1d双轨发散 |
| 完全开放（"你看着来"） | 进1b底牌闸门→1c市场调研（含扫榜定方向）→1d双轨发散 |

### 七维底牌摸底表

方向确定后，给用户一张摸底表，七类底牌一次性问：

```
开工前，先确认几件底牌（有就填，没有就跳过）：

1️⃣ 笔触参考书 —— 有没有觉得哪本书的写法/节奏/笔触对胃口？
  （有 → 我下载→提取笔触DNA，写正文时笔触层从那本书取）
  （没有 → 跳过，写正文时你手动指定风格方向）

2️⃣ 力量/体系参考书 —— 有没有哪本书的力量体系/升级节奏可以参考？
  （有 → 我下载→跑decon-lite拆八表：力量体系规则级+剧情节奏+情绪弧线+人称视角+开篇解剖+势力格局+金手指深度+世界危机引擎，给plot做世界构筑参考+给seed做改编策略素材）
  （没有 → 跳过，plot从赛道惯例和创意要素推导力量体系）

3️⃣ 赛道经典参考 —— 同赛道有没有标杆书想避开或借鉴？
  （有 → 我纳入市场校准，确保创意不和它撞车）
  （没有 → 跳过，市场调研时我帮你从排行榜找）

4️⃣ 现有设定 —— 你心里已经有想好的力量体系/世界规则吗？
  （有 → 告诉我，我记录到创意文档，plot直接继承，不重设计）
  （没有 → 跳过，plot三源合流从零推导）

5️⃣ 情绪基调偏好 —— 你想写什么情绪基调的故事？
  （热血逆袭/暗黑复仇/悬疑烧脑/都市恐怖/情感治愈/荒诞轻喜剧/末世求生/权谋博弈/还没想好）
  （有 → 我记录到创意文档，1d-0改编策略组装时用）
  （没有 → 跳过，1d-0时我根据赛道推荐）

6️⃣ 全书最大卖点 —— 这本书靠什么吸引读者翻页？
  故事驱动型：恐怖烧脑的故事/剧情反转/悬疑揭秘——读者为故事翻页
  设定驱动型：有趣精彩的设定/力量体系精巧/世界观宏大——读者为设定翻页
  处置驱动型：主角如何处置诡/敌人/问题——处置方式本身是爽点（封印/谈判/利用/转化/共生/驱使，每个问题不同处置）
  （有 → 记录到创意文档，约束1d发散方向——选了处置驱动型，发散时必须回答"怎么处置"，不能逃避到"吞噬变强"）
  （没有 → 跳过，1d-0时我根据赛道推荐）

7️⃣ 还没想清楚，需要推荐 → 我调研同赛道热门，出推荐书单
```

**红约束**：必须等用户回答完七维摸底表才能进1b闸门。用户说"全跳过"的可以直接放行。

## 1c. 市场调研（WebSearch + 借鉴点/避雷点分析）

> 如1b-2并发推进中③已启动市场调研子agent，等待返回后直接消费落盘文件（`素材/市场校准.md`），跳过以下1c-1到1c-3，直接进1c-4确认落盘。
> 如市场调研未由子agent执行（并发推进时③被跳过），则完整执行以下1c-1到1c-4。

> R13/R14核心环节。**这一步的产出是"借鉴点+避雷点清单"，不是搜索数据本身**。搜索数据消化完后不进入发散prompt。

### 1c-1. WebSearch（3轮）

搜索3轮，找当前赛道什么在火+已有模式是什么：

- 第1轮：`"番茄小说/起点中文网" + "{赛道} + 热门/排行榜/2026"` → 找当前榜单热门作品
- 第2轮：`"B站/抖音/知乎" + "{赛道} + 小说推荐/出圈" + "2026"` → 找正在出圈的概念
- 第3轮：`"网文" + "{赛道} + 新题材/新趋势/审美疲劳" + "2026"` → 找正在冒头的新方向+读者已疲劳的老模式

**用户完全开放时**：先扫番茄/起点总榜定方向，再用方向做1c。

### 1c-2. 市场分析（借鉴点 + 避雷点）

对每部热门作品/每种热门模式，分两层标注：

```markdown
1. 《书名/模式》
   借鉴：[可借鉴的机制/爽感/节奏/读者期待]
   避雷：[必须避开的已有设定/金手指/开局模式]
```

**借鉴点示例**（R13诡异流分析）：
- 《诡舍》：借鉴"规则怪谈+无限轮回的生死游戏张力"，避雷"密室逃生+规则怪谈"已有开局
- 《捞尸人》：借鉴"冷门行当的新鲜感+中式民俗恐怖"，避雷"捞尸行当+克苏鲁"已有设定
- 趋势借鉴："主动对抗规则"是已验证爽感方向（"被动求生"已审美疲劳）


**这份清单给agent看，不进入发散prompt**。agent在发散后用这张清单做校验。

## 1c-5. 题材深度调研（条件触发）

> **触发条件**：1a底牌"全书最大卖点"选了"处置驱动型"或"设定驱动型"，且赛道涉及需要专业素材的题材（诡异/修仙/末世/系统等）。
> **不触发**：卖点选了"故事驱动型"（故事驱动靠剧情不靠体系深度），或赛道是都市言情/职场等不需专业素材的题材。

当触发时，执行1轮WebSearch做**题材深度调研**（不是赛道级调研，是题材级的素材库建设）：

| 赛道 | 题材深度调研搜什么 | 产出 |
|:--|:--|:--|
| 诡异/灵异 | 处置方式素材库：民俗驱鬼/佛教超度/道教镇煞/克苏鲁封印/现代收容——每种1-2句话讲核心机制 | 素材/题材深度调研.md |
| 修仙 | 修炼体系素材库：丹道/符箓/阵法/雷劫/心魔——每种1-2句话 | 素材/题材深度调研.md |
| 末世 | 生存体系素材库：变异/进化/资源/避难所——每种1-2句话 | 素材/题材深度调研.md |

**搜索关键词**：`"{题材} + 处置方式/体系设定 + 民俗/宗教/学术" + "2026"`

**产出格式**：
```markdown

> seed Phase 1 · {timestamp}
> 触发原因：全书最大卖点={卖点类型}，赛道={赛道}需要专业素材

...

## 可迁移到本项目的机制
1. [机制1] —— 从哪个方式提取 + 怎么适配本项目
2. [机制2] —— ...
```

**落盘**：`素材/题材深度调研.md`

**消费规则**：
- 1d-0策略组装时，从素材库选择处置方式作为策略的"借什么"

- seed故事纲领力量体系层必须消费素材库，给出处置方式+力量体系核心机制

## 1d-0. 改编策略分析（v13.15.0新增）

> **核心：在5+5发散前，先用参考书DNA+赛道调研+用户元素+情绪+爽点类型组装3-5个"改编策略"，用户选1-2个策略后，1d双轨发散在策略框架下产出10个创意。策略是生成器，创意是产物。**
>
> 设计依据：agent环境测试发现，agent给的3个改编思路（时间线改编/势力替换/地域替换）比直接给5个立项更有意思——因为策略是"怎么切入"的方法论，立项是"切入成什么"的成品。策略一个撬动N种玩法，立项创造空间被封死。先选策略再发散，用户做"决策"而不是"选选项"。

### 1d-0-A. 素材盘点（发散前手上的牌）

进入1d-0前，确认手上有以下6类素材：

| 素材 | 来源 | 1d-0用途 |
|:--|:--|:--|
| 赛道借鉴点+避雷点 | 1c市场调研（`素材/市场校准.md`） | 策略的爽点类型选择 |
| 参考书·力量体系DNA | decon-lite表1（规则级：主养成线+子养成线） | 策略的"借什么力量体系" |
| 参考书·势力格局DNA | decon-lite表6 | 策略的"借什么势力结构" |
| 参考书·金手指DNA | decon-lite表7 | 策略的"借什么金手指机制" |
| 参考书·世界危机DNA | decon-lite表8 | 策略的"借什么危机引擎" |
| 用户元素 | 七维底牌+用户意图 | 策略的"融入什么元素" |

**缺失项检查**：
- **情绪基调**：七维底牌摸底时已收集用户偏好（1a"情绪基调偏好"问询）。如未收集，1d-0时追问
- **全书最大卖点**：七维底牌摸底时已收集用户偏好（1a"全书最大卖点"问询）。如未收集，1d-0时追问。处置驱动型/设定驱动型→确认1c-5题材深度调研已触发
- **爽点类型清单**：从1c借鉴点中显式提炼"可选择的爽点类型"（如：主动对抗型/越级碾压型/信息差型/反转揭秘型/...），单独成段。如1c未提炼，1d-0时从借鉴点中提取

### 1d-0-B. 策略组装卡（3-5个）

基于6类素材，组装3-5个**改编策略组装卡**。每张卡是一个"怎么切入"的方法论，不是一个具体创意。

**组装卡结构**：

```markdown
### 策略X：「策略名」

**改编维度**：[从以下维度选1-2个——时间线/势力/地域/主角身份/金手指形式/世界危机/力量体系]

**借参考书什么**：[从decon-lite表1/6/7/8选1-2项——借力量体系规则？借势力结构？借金手指机制？借危机引擎？]

**改参考书什么**：[选1-2项做改编——时间线前移？势力替换？地域转移？主角身份不同？金手指换限制？危机换驱动机制？]

**用什么爽点**：[从1c爽点类型清单选1-2种——主动对抗？越级碾压？信息差？反转揭秘？]

**融入用户什么元素**：[用户方向+预设设定+情绪基调]

**情绪基调**：[1种主情绪——热血逆袭/暗黑复仇/悬疑烧脑/...]

**2-3种展开玩法**：
1. [玩法1——同一策略的不同展开，每个1-2句话有画面]
2. [玩法2]
3. [玩法3]

**为什么这个策略有意思**：[1句话讲清楚这个策略的张力来源]
```

**组装原则**：
- 3-5个策略的改编维度必须差异化——不能3个都是"时间线改编"
- 策略是"生成器"不是"成品"——每张卡要能撬动多个具体创意，不是已经写死了主角是谁
- 策略必须和参考书DNA结合——不是凭空想策略，是从参考书的结构中提炼"借什么+改什么"
- 无参考书时：从赛道调研提炼"切入策略"（从哪个角度切入这个赛道），同样组装3-5张卡

### 1d-0-C. 用户选策略

把3-5个策略组装卡呈现给用户，让用户选1-2个策略。用户也可以"策略A的改编维度+策略B的爽点类型"混搭。

**无用户交互时**（API直调/自动化测试）：默认选第1个策略，继续1d双轨发散。

### 1d-0-D. 策略→发散的传递

用户选定策略后，1d双轨发散的prompt注入策略框架：

**王道赛道（A轨）**：
```
system: 你是一个网文创意编辑，擅长从热门作品提炼可迁移的卖点并微调出新方向。
user: 赛道：{用户方向}。改编策略：{选定策略的改编维度+借什么+改什么}。已验证的核心卖点（可迁移）：{卖点列表}。避雷点（必须避开）：{避雷点列表}。爽点类型偏好：{策略选定的爽点类型}。情绪基调：{策略选定的情绪}。请用融合策略（卖点平移/卖点叠加/反向操作/设定替换/情绪基调换挡）产出5个王道方向的开书创意。每个创意必须符合选定策略的改编维度。
```

**猎奇赛道（B轨）**：
```
system: 你是一个网文创意编辑。
user: 我想写{用户方向}，改编策略：{选定策略的改编维度+借什么+改什么}。爽点偏好：{策略选定的爽点类型}。情绪基调：{策略选定的情绪}。请帮我生成5个完全不同方向的开书创意种子，每个必须符合选定策略的改编维度但形式自由。
```

**关键约束**：
- 策略定方向（改编维度+借什么+改什么+爽点+情绪），不定形式（主角是谁/在哪/做什么由发散决定）
- B轨猎奇在策略框架内仍然自由发散——策略是"赛道"，发散是"跑法"
- 不破坏红线1（纯自由发散）：策略是给用户选的方向选项，不是注入发散prompt的格式模板。B轨的system仍然只有一句话，user给了策略方向但不给格式/结构/借鉴点

## 1d. 双轨发散：王道×猎奇

> **核心：在1d-0选定的策略框架下，拆为双轨——A轨"王道赛道"扫描排行榜top书提取核心卖点→融合新元素产出5个方向，B轨"猎奇赛道"在策略框架内纯自由发散产出5个方向。共10个立项，用户从两条赛道里选。**
>
> 设计依据：纯自由发散优势是反差/画面/原创性，劣势是容易全部偏向猎奇（"殡葬直播间""中医诊断股票"），缺少王道选项。用户选不到靠谱的安全牌会全部推翻重来。排行榜借鉴弥补这个缺陷——用已验证的卖点做基底，加新元素做微调，产出"安全但有趣"的选项。
>
> v13.15.0改动：1d在1d-0选定的策略框架下发散，不再无方向发散。

---

### 1d-A. 王道赛道：排行榜借鉴→融合新元素（产出5个）

> **目标**：5个"王道"方向——有排行榜验证的卖点做基底，但加了新元素不撞车。

#### 1d-A-1. 扫榜（WebSearch 2轮，找top书）

> 如1b-2并发推进中④已启动赛道扫榜子agent，直接消费返回的top书列表+核心卖点，跳过本轮搜索，直接进1d-A-2提取核心卖点。
> 如赛道扫榜未由子agent执行，则完整执行以下2轮搜索。

搜索目标赛道的番茄/起点排行榜top作品：

- 第1轮：`"番茄小说" + "{赛道}" + "排行榜/热销/2026"` → 找top 5
- 第2轮：`"{赛道}" + "起点" + "精品/万订/2026"` → 补起点top 3（如果番茄结果不够）

**用户完全开放时**：用1c市场调研时已经扫过总榜，直接复用那个方向。

#### 1d-A-2. 提取核心卖点（每本书1句话提炼为什么能卖）

对每本top书，提取**可迁移的核心卖点**（不是抄设定，是理解为什么读者买单）：

| top书 | 书名 | 核心卖点（1句话） | 可迁移到什么方向 |
|:--|:--|:--|:--|
| 1 | 《XXX》 | 冷门行当的新鲜感+中式民俗恐怖 | 其他冷门行当+任意类型 |
| 2 | 《XXX》 | 主动对抗规则的爽感+生死游戏张力 | 任意规则体系+主动对抗 |
| ... | ... | ... | ... |

**提炼原则**：
- 只提取"可迁移的卖点机制"，不提取"具体设定"（如《诡舍》的卖点是"规则怪谈+主动对抗"，不是"密室逃生场景"）
- 每本书最多提取1-2个卖点，只提取最核心的那个
- 卖点必须能用1句话讲清楚"读者为什么买单"

#### 1d-A-3. 融合发散（取卖点→加新元素→产出5个方向）

基于提取的5-8个核心卖点，用以下融合策略产出5个方向：

**融合策略**：
1. **卖点平移**：把A书的卖点机制平移到B书的场景/行当里（如"规则怪谈+主动对抗"平移到"外卖骑手+规则怪谈"）
2. **卖点叠加**：把两本书的不同卖点叠在一起（如"冷门行当新鲜感"+"主动对抗规则"→"殡葬师+规则怪谈+主动对抗"）
3. **反向操作**：把某本书的核心卖点翻过来用（如"被动求生"翻成"主动狩猎诡异"）
4. **设定替换**：保留卖点机制，换成全新设定（如"规则怪谈"改为"合同陷阱"→"公司法务给妖怪审合同"）
5. **情绪基调换挡**：同一个卖点机制换一种情绪基调（恐怖→爽文，悬疑→喜剧，黑暗→温馨）

**发散指令**（给子agent的全部输入）：

```
system: 你是一个网文创意编辑，擅长从热门作品提炼可迁移的卖点并微调出新方向。
user: 赛道：{用户方向}。已验证的核心卖点（可迁移）：{卖点列表}。避雷点（必须避开）：{避雷点列表}。请用融合策略（卖点平移/卖点叠加/反向操作/设定替换/情绪基调换挡）产出5个王道方向的开书创意。每个方向保留已验证卖点做基底，但加上新元素做差异化。
```

#### 1d-A-4. 王道方向质量门禁

每个王道方向必须满足：
- **有卖点基底**：能说清楚借了哪个top书的哪个已验证机制
- **有差异化**：加了至少1个新元素（新场景/新行当/新情绪基调），不是换皮
- **不踩避雷点**：核心设定不和已有热门撞车
- **有画面**：读完脑子里能浮现场景

不通过→重新融合发散。

### 1d-B. 猎奇赛道：纯自由发散（产出5个）

> **目标**：5个"猎奇"方向——不借鉴排行榜，纯自由发散，追求反差/画面/原创性。保持R9验证的最佳实践。

**发散指令**（给子agent/API的全部输入）：

```
system: 你是一个网文创意编辑。
user: 我想写{用户方向}，请帮我生成5个完全不同方向的开书创意种子。
```

**关键约束（给agent的，不给API的）**：
- 5个方向必须差异最大化——不同切入点、不同主角类型、不同情绪基调、不同创意类型
- **创意类型强制多样化**：10个创意（王道5+猎奇5）必须覆盖≥3种不同创意类型，禁止全偏一种类型（R43验证：10个全是反转型=失败）。创意类型清单（不限于）：
  - 反转型：普通人/现实世界→发现金手指/隐藏世界→反转日常认知（驱动力=揭秘）
  - 阶级攀爬型：世界内成员→靠特殊性往上爬→打破阶级天花板（驱动力=野心/生存，如《深渊主宰》——主角生在世界里，不是发现世界）
  - 生存竞技型：世界危机/竞技场→求生→胜出（驱动力=活下去）
  - 重建型：世界崩坏后→重建文明/势力→从废墟到秩序（驱动力=建设）
  - 身份揭秘型：主角身世是谜→一层层揭开→真相改变命运（驱动力=我是谁）
- **情绪基调强制多样化**：10个创意（王道5+猎奇5）必须覆盖≥5种不同情绪基调，禁止全偏一种调性。情绪基调清单（不限于）：
  - 热血逆袭（弱→强，打脸翻盘）
  - 暗黑复仇（被背叛→复仇，冷酷烧脑）
  - 悬疑烧脑（解谜推理，反转不断）
  - 都市恐怖（民俗/规则恐怖，细思极恐）
  - 情感治愈（温情救赎，以情动人）
  - 荒诞轻喜剧（反差幽默，荒谬对话）
  - 末世求生（资源管理，生存压力）
  - 权谋博弈（势力操盘，智斗）
- 不套框架，不加限制，不想机制——只管有没有意思
- **画面优先**：每句话读完脑子里立刻浮现场景。没有画面的创意直接淘汰

**好创意标准（agent自检，不通过重新发散）**：

读者三标准（验证吸引力）：
- **有画面**：读完脑子里立刻浮现场景，不用想
- **有张力**：有反差/有攀爬压力/有生存压力/有揭秘悬念——任何一种张力都算，不只有反差（反转型靠反差，攀爬型靠攀爬压力，生存型靠生存压力）
- **有意思**：读者看完会不会"卧槽这个我想看"？

作者三标准（验证可立项）：
- **有世界**：创意能定义出什么样的世界？**世界的力量体系是什么**（修仙境界/系统规则/规则怪谈/异能等级/...）？有什么隐藏规则/可生长设定？（world能推导力量体系/地图/势力）
- **有舞台**：创意自带什么样的世界危机？什么驱动剧情持续运转100章？（world能推导敌人梯度/压力源/倒计时）
- **有主角**：主角有什么特殊性（含金手指）？特殊性有升级空间吗？**主角的力量路径在世界力量体系内吗**？（world能推导成长路径/幕序列）

**六项全过才放行**。读者三标准筛掉"无聊"，作者三标准筛掉"跑不远"。

**反面教材（禁止产出）**：
- "主角穿越到XX世界，获得XX系统，开始升级" — 纯事件，没有主角特殊性
- "主角有XX能力，限制是XX，通过XX变强" — 纯机制，没有故事舞台
- "主角在XX势力中崛起，对抗XX敌人"但没有金手指/特殊性/成长路径 — 纯势力操盘（注：阶级攀爬型本身不是问题，有主角特殊性+成长路径的攀爬型是合格创意，没有特殊性才是问题）

### 1d-C. 双轨市场校准（用1c清单统合校验）

A轨和B轨各产出5个创意后，用1c的市场校准清单做**统合校验**：

| 轨道 | 创意 | 踩避雷点？ | 用到借鉴点？ | 校验结果 |
|:--|:--|:--|:--|:--|
| 王道 | 创意A1 | ✗ | ✓ | 放行 |
| ... | ... | ... | ... | ... |
| 猎奇 | 创意B1 | ✗ | ✗ | 放行 |
| ... | ... | ... | ... | ... |

**校验规则**：
- **踩避雷点=淘汰**（两轨统一规则）
- **用到借鉴点=加分**（两轨统一规则）
- 王道赛道额外校验：**有卖点基底+有差异化**，不通过淘汰不补（最多淘汰2个）
- 猎奇赛道额外校验：**有画面+有张力+有意思**，不通过重新发散补足5个
- 最终呈现：王道5个+猎奇5个=10个

### 1d-D. 双轨要点

| | 王道赛道（A） | 猎奇赛道（B） |
|:--|:--|:--|
| 产出数 | 5个 | 5个 |
| 发散方式 | 扫榜→提取卖点→融合新元素 | 纯自由发散（不注入任何数据） |
| 发散prompt注入 | 卖点列表+避雷点+融合策略 | 只给赛道方向 |
| 质量侧重 | 安全有趣（有卖点基底+差异化） | 原创有趣（画面+张力+有意思） |
| 用户场景 | "不想踩雷，想要靠谱的" | "要出圈的，赌一把" |

## 1e. 用户选择（10选1，双轨展示）

把校验通过的10个创意呈现给用户，让用户选1个。分两条赛道展示：

```
## 创意方向（共10个）

### 🔷 王道赛道（排行榜验证 + 微调，适合稳扎稳打）

#### 王道1：[名称]
一句话描述：[主角是谁，在做什么，要做什么]
画面：[脑补的场景，最好带一句台词]
为什么有意思：[吸引人的点——读者视角]
市场定位：[借鉴了《XXX》的[卖点]机制，做了[差异化]]
**创意类型**：[反转型/阶级攀爬型/生存竞技型/重建型/身份揭秘型]
**情绪基调**：[热血逆袭/暗黑复仇/悬疑烧脑/都市恐怖/情感治愈/荒诞轻喜剧/末世求生/权谋博弈]
**立项评估**：
- 世界：[什么样的世界——**力量体系**+隐藏规则/可生长设定]
- 舞台：[什么世界危机驱动剧情——敌人梯度/压力源/倒计时]
- 主角：[主角什么特殊性（含金手指）——成长路径/升级空间]

###（王道2-5同上格式，5个王道必须覆盖≥3种不同情绪基调、≥2种不同创意类型）

### 🔶 猎奇赛道（纯自由发散，适合博出圈）

#### 猎奇1：[名称]
一句话描述：[主角是谁，在做什么，要做什么]
画面：[脑补的场景，最好带一句台词]
```
