## 产出3·金手指

### 铁律：只加限制不加代价

- 限制 = 使用条件/边界（只能在子时用/只能对低于自己境界的人生效/每次间隔三天）
- 代价 = 用一次扣一年寿命/用一次忘一段记忆 ← **禁止**。代价让读者不爽
- 限制随剧情逐渐解锁——初始限制大，随主角成长逐步解除

### 6要素

1. 创意：一句话
2. 功能：能做什么（入口能力——最劲爆的那个，写进简介）
3. 触发条件：什么情况下能用（自然限制，不是惩罚）
4. 限制边界：使用条件/范围/冷却——初始限制严格，随成长解锁
5. 进化方向：含≥3种衍生分支（质变型，不是量的变化）
6. **本质**：金手指与世界底层逻辑的关系——不只是"能做什么"，还有"为什么这个能力存在"。进化到尽头会触及什么世界真相？

**进化方向怎么设计** → `reference/design-guide.md` "金手指进化方向设计方法"
**金手指本质怎么设计** → `reference/design-guide.md` "金手指本质设计方法"

## 产出4·主角设定卡 + 角色深度设计

从金手指反推主角（不是凭空编）。

**怎么反推** → `reference/design-guide.md` "主角设定卡设计方法"

### 4a. 主角设定卡（6要素）

- 职业/身份：什么人最适合这个金手指
- 主性格
- 记忆点：视觉化特征（习惯动作/口头禅/外貌标记）
- 缺点：从金手指限制推导——限制会诱发什么弱点
- **主角危机**：存在性危机——不解决就xxx，不是任务式倒计时。主角为什么非干不可？如果他不干会怎样？
- 成长弧线：从"被金手指驱动"→"驾驭金手指"→"超越金手指"

### 4b. 主角深度设计（S级，200-300字）

- 表层欲望：主角表面上想做什么
- 深层需要：主角真正需要什么（通常他自己都不知道）
- **致命缺陷**：驱动主角矛盾的核心缺陷——这是"好看"的关键
- 行为模式：5种情境下的行为（面对敌人/弱者/背叛/选择/资源）
- **人物弧光**：5阶段演变（起点→转变期→成长期→巅峰期→终极期）

### 4c. 核心羁绊角色（S/A级，120-200字/人，至少2人）

- 与主角关系
- 核心能力/功能
- 表层欲望+深层需要
- **致命缺陷**
- 关系变化轨迹（如"被保护者→并肩作战"）
- **人物弧光**3-5阶段

> **注意**：seed只设计主角+核心羁绊角色（2人）。完整的反派梯度（街头层/地方势力层/执法系统层/幕后金主层）由plot的Step 2世界构筑环节设计——因为反派需要与四张地图自洽。

### 4d. 金手指深度设计（增强产出3）

在原6要素基础上，补充以下深度设计：

- **金手指类型判定**：组合型占比（如面板流60%+杀戮流30%+血统流10%）
- **功能分类**：信息类/成长类/资源类/战斗类/社交类——金手指属于哪类
- **升级路径**：5-7阶段，每阶段有触发条件+新增能力
- **限制条件深度**：使用代价（如有）/冷却时间（如有）/**副作用**/使用前提/成长上限
- **与剧情的驱动关系**：金手指触发哪些关键剧情（3-5个节点）

## 产出5·世界观骨架

从金手指推导世界规则（不是凭空设计世界）。

**怎么推导** → `reference/design-guide.md` "世界观骨架设计方法"

### 6要素

1. **世界格局**：世界有多大？谁掌控？普通人在哪？——不是一条街，而是一个有结构的世界
2. **力量体系**：至少2条并行路径，不是单一线性。金手指属于哪条路径？
3. **核心规则**（2-3层互相制约）：金手指在哪层？世界自洽四问必须回答
4. **世界危机**：世界级威胁正在发生——世界不是静态背景，有推动剧情的大事。主角的金手指如何与世界危机挂钩？
5. **势力格局**：谁掌握核心资源？谁对抗谁？主角在哪个位置？（至少2个对立势力）
6. **进度框架**（3段式）：主角在世界中的位置变化——不是抽象阶段，而是在世界格局中的具体位置迁移

### 世界自洽四问（核心规则必须回答）

- 能力从何来？（世界规则支撑）
- 触发条件是什么？（世界内在逻辑）
- 作用边界在哪？（明确天花板）
- 世界规则如何执行？（世界机制运转，不是主角自己扣自己）

> **注意**：seed产出的是世界观骨架——方向性的世界规则。四张地图（地理/力量/势力/危机的具体地图数据）由plot的Step 2世界构筑环节展开——因为四张地图需要与反派梯度、幕序列、剧情白描自洽。
