如果Step 0产出了`参考书DNA.md`，立项展开时必须加载它。DNA中的元素已分为两类：

### 核心方法论 → 作为设计约束

只迁移DNA中标注`[核心]`的元素（跨章节复现的方法论）：

| DNA维度 | 约束什么 | 怎么约束 |
|---------|---------|---------|
| 系统设计模式·核心 | 金手指设计 | 系统需要匹配参考书的深度——非战斗功能（商店/任务/被动技能）+ 世界观转译机制 |
| 剧情节奏样本·单元剧循环 | plot骨架 | 单元剧循环模式（遇到NPC→系统扫描→交互→结算→新危机）作为plot骨架的节奏基础 |
| 剧情节奏样本·长线推进 | plot骨架 | 长线在单元剧之间怎么穿插、boss什么时候暗示/出场 |
| 社会交互模式 | 世界观+骨架 | 必须设计非战斗社会交互场景+NPC系统标注机制 |
| 资源经济系统 | 世界观骨架 | 资源稀缺+可转换链+替代品经济 |
| 世界观转译机制 | 金手指+世界观 | 系统把现实世界转译成游戏术语——印度背景下警察/黑帮/贫民/华裔→什么术语？ |
| 主角设计 | 主角设定卡 | 前世身份赋能/初始弱点/核心生存策略（不是战斗力）/成长方向（建势力网不是变强） |
| Boss设计 | plot骨架 | boss首次暗示时机/出场前铺垫层数/设计维度/第一次交互方式（谈判不是战斗） |
| 笔触特征 | write心法参考 | 系统面板格式/句式节奏作为write心法的补充约束 |

### 前期钩子 → 只参考不复制

DNA中标注`[钩子]`的元素（只在开篇出现后续不复现的），不作为设计约束，只在"番茄简介"设计时参考其方法论：

| 钩子元素 | 参考书怎么用的 | 我们怎么参考 |
|---------|-------------|------------|
| （如：斩杀线） | ch01用系统机制制造紧迫感，让读者觉得"主角随时会死" | 我们的开篇也需要一个系统级紧迫感钩子——但设计自己的机制，不复制斩杀线 |

**关键原则**：DNA是设计约束不是内容模板——学习参考书"怎么做到好"，不是复制"做了什么"。`[钩子]`元素学的是"为什么这个钩子有效"，不是"复制这个钩子"。

## 产出3·金手指

### 铁律：只加限制不加代价

- 限制 = 使用条件/边界（只能在子时用/只能对低于自己境界的人生效/每次间隔三天）
- 代价 = 用一次扣一年寿命/用一次忘一段记忆 ← **禁止**。代价让读者不爽
- 限制随剧情逐渐解锁——初始限制大，随主角成长逐步解除

### 6要素

1. 创意：一句话
2. 功能：能做什么（入口能力——最劲爆的那个，写进简介）
3. 触发条件：什么情况下能用（自然限制，不是惩罚）
4. 限制边界：使用条件/范围/冷却——初始限制严格，随成长解锁
5. 进化方向：含≥3种衍生分支（质变型，不是量的变化）
6. **本质**：金手指与世界底层逻辑的关系——不只是"能做什么"，还有"为什么这个能力存在"。进化到尽头会触及什么世界真相？

**进化方向怎么设计** → `reference/design-guide.md` "金手指进化方向设计方法"
**金手指本质怎么设计** → `reference/design-guide.md` "金手指本质设计方法"

## 产出4·主角设定卡

从金手指反推主角（不是凭空编）。

**怎么反推** → `reference/design-guide.md` "主角设定卡设计方法"

6要素：
- 职业/身份：什么人最适合这个金手指
- 主性格
- 记忆点：视觉化特征（习惯动作/口头禅/外貌标记）
- 缺点：从金手指限制推导——限制会诱发什么弱点
- **主角危机**：存在性危机——不解决就xxx，不是任务式倒计时。主角为什么非干不可？如果他不干会怎样？（不是"被迫接单"，而是"不做就消失/被吞噬/成为另一个自己"）
- 成长弧线：从"被金手指驱动"→"驾驭金手指"→"超越金手指"

## 产出5·世界观骨架

从金手指推导世界规则（不是凭空设计世界）。

**怎么推导** → `reference/design-guide.md` "世界观骨架设计方法"

### 6要素（从4要素升级）

1. **世界格局**（从"世界地图"升级）：世界有多大？谁掌控？普通人在哪？——不是一条街，而是一个有结构的世界
2. **力量体系**（从"等级体系"升级）：至少2条并行路径，不是单一线性。金手指属于哪条路径？
3. **核心规则**（2-3层互相制约）：金手指在哪层？世界自洽四问必须回答
4. **世界危机**：世界级威胁正在发生——世界不是静态背景，有推动剧情的大事。主角的金手指如何与世界危机挂钩？
5. **势力格局**：谁掌握核心资源？谁对抗谁？主角在哪个位置？（至少2个对立势力）
6. **进度框架**（3段式）：主角在世界中的位置变化——不是"保命/经营/破局"的抽象阶段，而是在世界格局中的具体位置迁移

### 世界自洽四问（核心规则必须回答）

- 能力从何来？（世界规则支撑）
- 触发条件是什么？（世界内在逻辑）
- 作用边界在哪？（明确天花板）
- 世界规则如何执行？（世界机制运转，不是主角自己扣自己）
