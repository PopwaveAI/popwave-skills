## 七维底牌摸底表

```
开工前，先确认几件底牌（有就填，没有就跳过）：

1️⃣ 笔触参考书 —— 有没有觉得哪本书的写法/节奏/笔触对胃口？
  （有 → 我下载→提取笔触DNA，写正文时笔触层从那本书取）
  （没有 → 跳过，写正文时你手动指定风格方向）

2️⃣ 力量/体系参考书 —— 有没有哪本书的力量体系/升级节奏可以参考？
  （有 → 我下载→跑decon-lite拆八表：力量体系规则级+剧情节奏+情绪弧线+人称视角+开篇解剖+势力格局+金手指深度+世界危机引擎，给plot做世界构筑参考+给seed做改编策略素材）
  （没有 → 跳过，plot从赛道惯例和创意要素推导力量体系）

3️⃣ 赛道经典参考 —— 同赛道有没有标杆书想避开或借鉴？
  （有 → 我纳入市场校准，确保创意不和它撞车）
  （没有 → 跳过，市场调研时我帮你从排行榜找）

4️⃣ 现有设定 —— 你心里已经有想好的力量体系/世界规则吗？
  （有 → 告诉我，我记录到创意文档，plot直接继承，不重设计）
  （没有 → 跳过，plot三源合流从零推导）

5️⃣ 情绪基调偏好 —— 你想写什么情绪基调的故事？
  （热血逆袭/暗黑复仇/悬疑烧脑/都市恐怖/情感治愈/荒诞轻喜剧/末世求生/权谋博弈/还没想好）
  （有 → 我记录到创意文档，1d-0改编策略组装时用）
  （没有 → 跳过，1d-0时我根据赛道推荐）

6️⃣ 全书最大卖点 —— 这本书靠什么吸引读者翻页？
  故事驱动型：恐怖烧脑的故事/剧情反转/悬疑揭秘——读者为故事翻页
  设定驱动型：有趣精彩的设定/力量体系精巧/世界观宏大——读者为设定翻页
  处置驱动型：主角如何处置诡/敌人/问题——处置方式本身是爽点（封印/谈判/利用/转化/共生/驱使，每个问题不同处置）
  （有 → 记录到创意文档，约束1d发散方向——选了处置驱动型，发散时必须回答"怎么处置"，不能逃避到"吞噬变强"）
  （没有 → 跳过，1d-0时我根据赛道推荐）

7️⃣ 还没想清楚，需要推荐 → 我调研同赛道热门，出推荐书单
```
