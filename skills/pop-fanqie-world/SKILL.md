## 定位

world 帮助用户把创意从 idea 变成一个**有根的世界**。seed 产出"创意文档+黄金首章+市场校准"，world 负责从创意反向推导整个世界——不是填表，是生长。

**与 plot 的分工**：world 专注设定设计（理性推导），产出骨架.md；plot 专注叙事创作（感性创作），消费骨架.md 产出剧情白描.md + 章锚点表.md。

# Step 1 · 加载三件底牌（内化）

| 底牌 | 来源 | 路径 | world用途 |
|:--|:--|:--|:--|
| 参考书DNA | pop-dna-style / pop-decon | `涌现/文风锚定.md` + `涌现/剧情DNA-brief.md` | 参考书的世界构筑DNA（力量体系风格/势力结构/危机设计模式）——不是抄设定，是学设计思路 |
| 创意+首章 | pop-fanqie-seed | `0-立项/创意.md` + `2-正文/ch001.md` | 创意/金手指/行为引擎/主角轮廓——世界从这些条款生长出来 |
| 市场校准 | pop-fanqie-seed 1c | `写作参考/市场校准.md` | 赛道上什么在火、什么烂大街——避免力量体系/势力格局撞车已有热门 |

### 从创意.md提取种子条款

| 种子条款 | 提取内容 | 世界构筑用途 |
|:--|:--|:--|
| 创意 | 一句话创意——一个具体行为×一个反差场景 | 世界必须让这个创意的行为有意义 |
| 金手指·机制+限制 | 激活条件/胜利判定/限制（6条） | 每一条限制都是力量体系中的**张力设计点** |
| 行为引擎 | 主动型/赌性/底线 | 地图和势力必须支持这个行为 |
| 主角轮廓 | 名字/身份/核心动机/记忆点 | 主角在世界中的初始位置 |
| 前期成长路线 | 街头期/起家期/扩张期/立足期 | 第一卷章节预算和敌人梯度的蓝图 |
| 故事纲领五维 | 世界底色/危机基调/主角弧线/最大钩子/即时兑现感 | 世界构筑的方向锚——力量体系和世界规则必须服务于这五维 |

### 从ch001提取起点

核心事件 → 章末钩子( **ch002必须从这里延续** ) → 金手指激活状态 → 已出场角色(不可冲突)

缺失底牌标注缺失，不阻塞流程。

# Step 2 · 从创意生长世界（内化）

> **核心原则：世界不是填表填出来的，是从创意核心要素反向推导出来的。**
>
> **力量体系是骨架，地图/势力/危机是血肉——血肉塞入骨架。**（详见 pop-decon-setting v3.3.0 的骨架/血肉两层结构设计）

## 推导链

```
创意.md核心要素 + 赛道已验证模式 + 参考书DNA
        ↓
    力量体系 ──→ 地图 ──→ 势力 ──→ 危机
```

## 力量体系设计起点判定

| 本项目类型 | 设计起点 | 例 |
|:--|:--|:--|
| 系统流 | 金手指机制是主角优势，体系来自世界规则或赛道惯例 | 深渊主宰：DND体系是世界观自带 |
| 修炼流 | 世界观自带的修炼体系，金手指只是加速器 | 凡人修仙传：炼气→筑基→金丹→元婴 |
| 规则系 | 世界规则就是力量体系，金手指只是让你看规则 | 诡舍：规则怪谈世界的规则本身 |
| 无金手指 | 世界观/赛道惯例直接决定 | — |

## 2a. 三源合流设计力量体系

### 第一源：赛道参考（`写作参考/市场校准.md`，如有）
- top作品的力量体系类型（等级制/修炼境/规则系/百分比/职业树）
- 读者疲劳的模式（什么已被写烂了，必须避开）
- 正在冒头的变体（赛道演变方向）

### 第二源：参考书DNA（`涌现/文风锚定.md` + `涌现/剧情DNA-brief.md`，如有）
- 文风锚定：参考书的力量体系风格
- 剧情DNA-brief·Boss梯度：每层boss的强度分布+出场节奏
- 剧情DNA-brief·主角成长曲线：破局手段演变（前期靠什么/中期靠什么/后期靠什么）

**消费规则**：学设计思路和强度分布节奏，不抄具体等级名。

### 第三源：创意.md
- 金手指（如有）：机制和限制决定主角在体系中的**优势方式和成长节奏**（不是决定体系本身）
- 行为引擎：决定**触发升级的条件**（不是靠刷怪，靠对赌/经营/探索？）
- 前期成长路线：决定每卷的力量跨度
- 世界观规则（如有）：规则系优先于金手指

### 三源合流产出：力量层级表

展开5-7个力量层级，每个层级必须：

| 字段 | 含义 | 要求 |
|:--|:--|:--|
| 名称 | ≤6字，贴合本项目世界观 | 不套"DND/炼气筑基"模板 |
| 这个世界长什么样 | MMO式的等级区描述 | 新手区→城镇区→高级区 |
| 主角在这个层级面对什么 | 不是"解锁什么技能" | 这个层级的世界是什么样的 |
| 升级触发条件 | 从行为引擎推导 | 不是"打够怪就升级" |
| 参考源 | 标注参考了哪个源 | 赛道·X / 参考书·X / 创意·X |

**力量跨度**：第一卷从阶位1到阶位N（对齐创意.md前期成长路线的街头期上限）

## 2b. 从力量体系+行为引擎生长地图

推导逻辑（MMO式）：
```
力量层级 → 每个阶位的"区域主题" → 城市/世界地图
  阶位1："新手区" → 贫民区+赌场
  阶位2："城镇区" → 码头+商业街+公会
  阶位3："高级区" → 贵族区+神殿区+地下城
```

每个区域回答：
- 对应哪个力量阶位？
- 主角的**核心行为**是什么？（从行为引擎推导）
- 有什么**信息差**可以利用？

输出：城市结构表（区域/对应阶位/主角行为/信息差来源/势力归属/出现幕）

## 2c. 从力量体系+地图生长势力

推导逻辑：
```
力量阶位1→贫民区→谁控制这里的资源？→街头帮派
力量阶位2→码头+商业街→谁控制贸易？→地方商会+走私网
力量阶位3→贵族区→谁控制制度？→贵族家族+执法系统
力量阶位4→隐藏区域→谁在幕后？→神秘组织
```

每层势力必须回答：
1. **为什么主角必须和这个势力冲突？**（不解决它就无法进入下一阶位）
2. **主角靠什么赢？**（阶位1靠硬刚→阶位2靠信息差→阶位3靠制度漏洞）
3. **赢了得到什么？**（地盘/人脉/信息/身份——进入下一阶位的钥匙）

## 2d. 危机体系设计（四源）

| 危机来源 | 说明 | 必选 |
|:--|:--|:--|
| **世界固有危险** | 创意.md设定的世界级威胁——全书暗线 | ✓ |
| **力量体系内压** | 进入更高阶位必然面对的压力（强者注目/势力反弹/制度反噬） | ✓ |
| **金手指反噬** | 金手指限制条款触发的危机 | 如有金手指 |
| **赛道惯例压力** | 赛道读者期待的特定压力类型（如诡异流必然有"规则杀"） | 可选 |

**倒计时设计**：从危机四源推导2-3条倒计时（触发条件→到期章→后果）

## 红线

1. **必须加载三件底牌**——参考书DNA（如存在）+创意+首章+市场校准（如存在），禁止凭空编造世界观。缺失标注缺失，不阻塞
2. **每项世界要素必须回溯创意.md条款**——金手指/行为引擎/主角轮廓/前期成长路线，至少回溯一条。禁止塞DND模板/通用西幻设定
3. **力量体系必须三源合流**——从赛道已验证模式+参考书DNA+创意.md要素三源推导，不能凭空编造
4. **第一卷敌人必须4层梯度**——街头层/地方势力层/执法系统层/幕后金主层，每层至少1个具名反派
5. **力量体系是骨架，地图/势力/危机是血肉**——血肉必须塞入骨架，地图按力量层级空间化，势力按金字塔tier分层
6. **Step文件链式加载**——Step1-2内化在本文件直接执行→读取steps/step3.md→读取steps/step4.md
