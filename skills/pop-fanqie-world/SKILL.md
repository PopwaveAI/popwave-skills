## 幕/卷统一格式定义（world+plot共用）

| 单元 | 定义 | 章数 |
|:--|:--|:--|
| **幕** | 完整起承转合的剧情单元 | 8~12章 |
| **卷** | 多个幕，主角/世界完成一次大的推进 | 40~60章（即4~5幕） |

**换算**：一卷=4~5幕（8~12章×4~5=32~60章，落在40~60区间）。

**world视角**：全书3-8卷×40~60章=120~480章（200万字级）。
**plot视角**：每卷分4~5幕写白描，每幕8~12章的起承转合。

## 定位

world 帮助用户把创意从 idea 变成一个**有根的世界**。seed 产出"创意文档+黄金首章+市场校准"，world 负责从创意反向推导整个世界——不是填表，是生长。

**与 plot 的分工（v1.3.0明确边界）**：
- world 专注**设定设计**（理性推导）——世界是什么样的（力量体系/地图/势力/危机/全书设定），产出设计/全书设定/（多文件）
- plot 专注**叙事创作**（感性创作）——故事怎么讲（本卷设定切片/幕序列弧线/高潮场景展开/悬念分层/剧情白描），从设计/全书设定/中切片消费，产出设计/第一卷剧情/剧情白描.md
- **world不做**：幕序列弧线（每幕的弧线/钩子）、高潮场景展开（三段式高潮）、悬念分层（触发章/揭示章）、主角靠什么赢（剧情问题）

# Step 2 · 从创意生长世界（内化）

> **核心原则：世界不是填表填出来的，是从创意核心要素反向推导出来的。**
>
> **力量体系是骨架，地图/势力/危机是血肉——血肉塞入骨架。**（详见 pop-decon-setting v3.3.0 的骨架/血肉两层结构设计）

## 推导链

```
创意.md核心要素 + 赛道已验证模式 + 参考书DNA
        ↓
    力量体系 ──→ 地图 ──→ 势力 ──→ 危机
```

## 力量体系设计起点判定

> **v1.1.0核心改动：world不再自己发明力量体系，消费seed已明确的力量体系。** seed v13.13.0+的故事纲领"什么样的世界"必须包含"力量体系层"，world拿到后直接展开为力量层级表，不再自己猜。

| 本项目类型 | 设计起点 | 例 |
|:--|:--|:--|
| seed已明确力量体系 | **直接消费seed的力量体系**，展开为力量层级表。不自己发明，不替换，不修改 | seed说力量体系是修仙境界→world展开炼气→筑基→金丹→元婴的层级表 |
| seed未明确力量体系（v13.13.0前） | 从赛道惯例+创意.md要素推导，标注"seed缺失，world推导" | — |

**硬约束**：world的力量层级表必须和seed故事纲领的力量体系对齐。如果seed说力量体系是修仙境界，world不能展开成"管理权限层级"——不对齐=world错误，不是seed错误。

## 2a. 三源合流设计力量体系

### 第一源：赛道参考（`素材/市场校准.md`，如有）
- top作品的力量体系类型（等级制/修炼境/规则系/百分比/职业树）
- 读者疲劳的模式（什么已被写烂了，必须避开）
- 正在冒头的变体（赛道演变方向）

### 第二源：参考书DNA（`素材/文风锚定.md` + `素材/剧情DNA-brief.md`，如有）
- 文风锚定：参考书的力量体系风格
- 剧情DNA-brief·Boss梯度：每层boss的强度分布+出场节奏
- 剧情DNA-brief·主角成长曲线：破局手段演变（前期靠什么/中期靠什么/后期靠什么）

**消费规则**：照搬参考书的结构规则（子养成线/交叉规则/阶位名渊源），学设计思路和强度分布节奏。可以照搬具体概念名（如"咒印/系缚/意根藏"）做本项目化微调，禁止标"参考decon·XX"但实际自己另编一套不同规则——假消费=废设定。

### 第三源：创意.md
- 金手指（如有）：机制和限制决定主角在体系中的**优势方式和成长节奏**（不是决定体系本身）
- 行为引擎：决定**触发升级的条件**（不是靠刷怪，靠对赌/经营/探索？）
- 前期成长路线：决定每卷的力量跨度
- 世界观规则（如有）：规则系优先于金手指

### 三源合流产出：力量体系四层结构

> 力量体系不是单张层级表，是四层嵌套结构。world必须逐层设计，每层消费decon-lite对应内容（如有参考书）。

#### 第一层：世界源力量

> 力量从哪来？源力量如何转化为主角可用的力量？源力量和世界观挂钩——为什么世界有这种力量？

| 字段 | 含义 | 要求 |
|:--|:--|:--|
| 源力量名称 | 这个世界的力量本源叫什么 | 贴合世界观——诡异题材是"诡韵/阴气/煞气"，修仙是"灵气"，末世是"火泉/变异因子" |
| 源力量来源 | 源力量为什么存在？和世界观什么关系？ | 必须和创意.md世界观挂钩——"裂缝存在→诡异滋生→诡韵弥漫" / "天地灵气复苏→灵气滋养万物" |
| 源力量→主养成线转化链 | 源力量如何驱动主养成线成长？ | 源力量→转化通道→成长输出。如"诡韵→被吞噬→承载力提升" / "灵气→修炼→修为增长" |
| 源力量→诡/怪/敌分级标准 | 源力量如何给世界中的超自然存在分级？ | 给诡异/妖兽/敌人分级——如"低阶诡（诡韵浓度<10%）→中阶诡（10-30%）→高阶诡（>30%）"，主角的主养成线阶位对应能处置哪级 |

#### 第二层：主养成线（位阶）

> 主养成线是主角的核心成长轴。每个阶位必须有文化渊源+社会地位参照。

展开5-7个阶位，每个阶位必须：

| 字段 | 含义 | 要求 |
|:--|:--|:--|
| 阶位名称 | ≤6字，**必须有文化渊源** | 不是"觉醒者/承载者"这种通用词，是"咒印修行者/系缚修行者/佛子"这种有佛教/道教/民俗渊源的词。阶位名要沾题材——诡异题材用民俗/道教/佛教词汇，修仙用境界名，末世用变异等级 |
| 这个世界长什么样 | MMO式的等级区描述 | 新手区→城镇区→高级区 |
| 主角在这个层级面对什么 | 不是"解锁什么技能" | 这个层级的世界是什么样的 |
| 升级触发条件 | 从行为引擎推导 | 不是"打够怪就升级" |
| **社会地位参照** | 这个阶位有多少人？什么社会地位？ | 稀有度+社会角色——"全城约50人，地下势力中层，被普通人恐惧但被势力拉拢" / "万人中无一，城市实际掌控者之一"。这一层决定世界的真实感和社会结构 |
| 预计展开卷 | 哪卷展开 | 第一卷通常覆盖前2-3层 |

**阶位名称硬约束**：如果参考书decon-lite拆出的阶位名有文化渊源（如《我的诡异人生》的咒印修行者/系缚修行者/佛子），world可以照搬或微调，不能降级为通用词。

#### 第三层：子养成线

> 子养成线是主养成线内的平行成长维度。不能是乱设计的平行进度条——每条必须有自身阶位+和主养成线挂钩。

从decon-lite表1逐条消费子养成线（如有参考书），每条子养成线必须：

| 字段 | 含义 | 要求 |
|:--|:--|:--|
| 子养成线名称 | ≤6字 | 从decon-lite照搬或微调，如"咒印系统/诡异契合度/意根藏" |
| 自身阶位 | 这条子养成线自身的成长阶位 | 不能是无阶位的进度条——必须有≥2级自身阶位，如"咒印系统：初印→深印→真印" |
| 和主养成线挂钩关系 | 子养成线阶位与主养成线阶位的对应 | 主养成线阶位决定子养成线上限/解锁条件——如"主养成线3阶才能解锁咒印系统2级"，或"契合度每提升1级，主养成线容纳上限+20%" |
| 规则 | 获取方式/消耗/上限 | 从decon-lite照搬或微调 |
| 参考源 | 标注参考了decon-lite哪条 | decon·表1·子养成线N |

**子养成线硬约束**：decon-lite拆出N条子养成线，world必须逐条处理——照搬转化or标注"本项目不需要因为XX"。禁止丢条自己另编。

#### 第四层：交叉规则

> 子养成线之间的协同/制约关系 + 源力量流转链。

从decon-lite表1逐条消费交叉规则（如有参考书），每条交叉规则必须：

| 字段 | 含义 | 要求 |
|:--|:--|:--|
| 规则名称 | ≤10字 | 从decon-lite照搬或微调 |
| 涉及哪些子养成线 | 2条以上的子养成线 | 标注涉及哪几条 |
| 协同/制约类型 | 协同（A↑→B↑）还是制约（A↑→B↓） | 必须明确 |
| 具体规则 | 1-2句话讲清机制 | 从decon-lite照搬或微调，不能自己另编 |
| 参考源 | 标注参考了decon-lite哪条 | decon·表1·交叉规则N |

**交叉规则硬约束**：decon-lite拆出N条交叉规则，world必须逐条映射到本项目——照搬转化or标注"本项目不需要因为XX"。禁止标"参考decon·XX"但实际自己另编一套不同规则。

## 2b. 从力量体系+行为引擎生长地图

推导逻辑（MMO式）：
```
力量层级 → 每个阶位的"区域主题" → 城市/世界地图
  阶位1："新手区" → 贫民区+赌场
  阶位2："城镇区" → 码头+商业街+公会
  阶位3："高级区" → 贵族区+神殿区+地下城
```

每个区域回答：
- 对应哪个力量阶位？
- 主角的**核心行为**是什么？（从行为引擎推导）
- 有什么**信息差**可以利用？

输出：城市结构表（区域/对应阶位/主角行为/信息差来源/势力归属/出现卷——标注哪卷进入此区域）

## 2c. 从力量体系+地图生长势力

推导逻辑：
```
力量阶位1→贫民区→谁控制这里的资源？→街头帮派
力量阶位2→码头+商业街→谁控制贸易？→地方商会+走私网
力量阶位3→贵族区→谁控制制度？→贵族家族+执法系统
力量阶位4→隐藏区域→谁在幕后？→神秘组织
```

每层势力必须回答（设定视角，不回答剧情怎么赢）：
1. **这个势力控制什么资源？**（资源=进入下一阶位的钥匙）
2. **这个势力有什么弱点/裂缝？**（弱点=world给plot提供的可利用设定，不是剧情怎么利用）
3. **和上层势力什么关系？**（从属/同盟/对抗——决定主角打这层时上层什么态度）

**全书要求**：4层势力全部具名设计，每层标注哪卷碰撞（第一卷通常碰撞前2层，后2层暗示；后续卷逐层展开）。不是只做第一卷的——全书设定/要给plot提供全书4层的设定。

## 2d. 危机体系设计（四源）

| 危机来源 | 说明 | 必选 |
|:--|:--|:--|
| **世界固有危险** | 创意.md设定的世界级威胁——全书暗线 | ✓ |
| **力量体系内压** | 进入更高阶位必然面对的压力（强者注目/势力反弹/制度反噬） | ✓ |
| **金手指反噬** | 金手指限制条款触发的危机 | 如有金手指 |
| **赛道惯例压力** | 赛道读者期待的特定压力类型（如诡异流必然有"规则杀"） | 可选 |

**倒计时设计**：从危机四源推导全书危机体系——1条终极危机（全书终卷爆发）+ 2-3条阶段性危机（分卷爆发）。每条标注：触发条件→预计爆发卷→后果。plot按卷切片消费时只取本卷涉及的危机。

## 红线

1. **必须加载三件底牌**——参考书DNA（如存在）+创意+首章+市场校准（如存在），禁止凭空编造世界观。缺失标注缺失，不阻塞
2. **每项世界要素必须回溯创意.md条款**——金手指/行为引擎/主角轮廓/前期成长路线，至少回溯一条。禁止塞DND模板/通用西幻设定。**阶位名称必须有文化渊源，禁止通用词（觉醒者/承载者/压制者）。**
3. **力量体系必须三源合流**——从赛道已验证模式+参考书DNA+创意.md要素三源推导，不能凭空编造。**decon-lite拆出的主养成线N阶/子养成线N条/交叉规则N条必须逐条处理（照搬转化or标注不需要原因），禁止丢条自编。**
4. **全书势力必须4层梯度**——街头层/地方势力层/执法系统层/幕后金主层，每层至少1个具名反派。不是只做第一卷的——全书设定要覆盖4层
5. **力量体系是骨架，地图/势力/危机是血肉**——血肉必须塞入骨架，地图按力量层级空间化，势力按金字塔tier分层
