## 做什么

输入：项目上下文（**剧情白描.md**+骨架.md+current-state.md+前章正文+**设定库精选**）+ 本章锚点 + **笔触DNA（标准插槽，三态协议）**
输出：2000-2500字的章节正文，**不准超过2500字**，附交付面板

**定位**：write从ch002开始写正文（ch001已由seed的黄金首章产出）。write消费plot产出的剧情白描+章锚点表，逐章渲染正式正文。

文件架构：
- **SKILL.md（本文件）**：4种章型骨架 + SOP（含笔触DNA标准插槽三态协议） + 红线 + 速查表
- **references/章型定义.md**：4种章型的7节拍骨架 + 密度约束
- **references/爽点引擎.md**：5种爽感引擎（打脸/压力突破/未知揭示/小爽沿途/即时奖励）
- **references/technique.md**：写作技法参考（卡壳时翻阅）
- **dna/**：笔触DNA文件（由 pop-dna-style 产出）——**标准插槽加载路径**：`项目空间/涌现/文风锚定.md`（第一优先级，10万字裁剪时也不裁剪）

## 4种章型骨架

番茄适配的4种章型：

| 章型 | 使用条件 | 读者期待 | 主动行为类型 |
|:----|:--------|:--------|:------------|
| opening_shift | 第一章/世界规则松动/金手指激活 | 先信现实→被打开更大世界 | 收获/质变级反馈 |
| confrontation_pressure | 对峙/打脸/信息差博弈/势力试探 | 看打脸反转 | 宏观层面解决问题 |
| combat_reversal | 战斗/绝境翻盘/首杀/暗杀 | 看绝境翻盘 | 战斗 |
| reveal_hook | 揭秘/新敌人/升级/章末钩子 | 看真相炸裂 | 获得反馈/质变级反馈 |

> 探索章和高位展示章合入confrontation_pressure或reveal_hook，番茄读者不需要太细的章型区分。

每种章型7节拍骨架详见 `references/章型定义.md`。

## 红线

1. **每章必须全文加载剧情白描.md+骨架.md+current-state.md**——禁止用current-state.md替代立项包
2. **设定库禁止全文注入**——只精选与本章相关的2个维度（≤1000字），设定库是"查字典"不是"读全书"
3. **第一章(ch001)由seed产出**——write从ch002开始，ch002必须回收ch001黄金首章的章末钩子
4. **篇幅硬限制**：2000-2500字，**不准超过2500字**。超2500字=废章，必须裁剪
5. **章型必选**：每章必须从4种章型中选1个，按7节拍骨架组织结构
6. **爽感≥1/章**：每章至少1个爽感触发，用爽点引擎设计
7. **章名**：每章正文开头必须有正常章节名（贴合内容命名，不刻意制造噱头/反差/玩梗）
8. **笔触DNA标准插槽**（三态协议）：
    - **启用态**（`涌现/文风锚定.md` 存在）：笔触层**必须从DNA取**，禁止凭空发挥
    - **缺失态**（`涌现/文风锚定.md` 不存在）：Step1必须提示用户部署笔触DNA，用户拒绝部署则进入trial模式
    - **trial模式**（用户拒绝部署）：不内置任何默认风格，用户须显式声明基础风格倾向（或接受API自由发挥的质量波动）
9. **章末有钩子**：每章必须以主动钩子或威胁逼近收尾
10. **不写思考过程/检查表**——直接写正文

## 速查表

### 节奏约束速查（结构层·非风格）

| 指标 | 约束 |
|:----|:----|
| 事件间隔 | ≤20行 |
| 事件密度 | 8-12个/章 |
| 蓄力上限 | ≤2段 |
| 信息释放 | ≤3段 |
| 微钩子 | 每3-5行 |

> 注：感官连续/情绪连续的字数限制属于**风格层**，由DNA决定。trial模式不内置。

### 爽感速查

| 机制 | 要求 |
|:----|:----|
| 打脸循环 | 轻视≥2层，翻盘≤3行，对方有可见失态 |
| 小爽链 | 每3-5事件≥1个小爽，大爽前≥2个铺垫 |
| 爆发后三段式 | 爆发→反馈≤2段→余韵≥1段→下一事件 |
| 即时奖励 | 每章≥1次行动→可见回报闭环 |

### 篇幅控制速查

| 字数区间 | 判断 | 处理 |
|---------|------|------|
| 2000-2500字 | 合格 | 完成 |
| 1800-2000字 | 偏短 | 检查信息密度 |
| 2500-3000字 | 超标 | 必须裁剪描述性文字 |
| 超过3000字 | 废章 | 重写，保留剧情推进和对话 |
