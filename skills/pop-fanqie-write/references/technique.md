## 一、代入感六大支柱

### 1. 基础信息标签化
读者先认知主角，再认知世界。一句话交代地点、人物、特质、身份。
> "小爷我乃镇南府世子林峰，不过是区区六百两酒钱，醉红楼我欠了三千两都不敢这么对我，还不快给小爷我松绑？"

### 2. 具体化与可视化
不写"一辆车" → 写"白色面包车"
不写"他愤怒" → 写"他捏碎了茶杯，滚烫的茶水流过指缝"
不写"一个漂亮女人" → 写"穿着洗得发白的牛仔外套，头发扎在脑后，指甲掐进木门漆皮"

### 3. 熟悉感
用读者经历过的场景——高考后分手/被克扣工资/被老师冤枉/被室友欺负

### 4. 情绪共鸣
让主角的选择和读者重叠——读者觉得"我也会这么做"

### 5. 五感代入
每个场景选1-2种感官落地，不堆叠。

### 6. 人设通过事件立
不靠外貌描述——在事件中看角色的行动、思考、反应

## 二、避免流水账

### 方法1：增加冲突
给平淡情节加前因后果。"他下班回家吃饭看手机有人敲门" → "催债电话打了三次——两天内不还就上门。他啃着硬包子，砰砰砰有人敲门"

### 方法2：情绪升级链
倒霉→更倒霉→最倒霉→爆发

### 方法3：从"炸楼"开始
不从砌砖开始。开篇1-3个场景、1-3个角色

## 三、语言去油

### 高疲劳词（单章只用1次）
冷笑 / 蝼蚁 / 轰然炸裂 / 倒吸一口凉气 / 瞳孔骤缩 / 满场死寂 / 嘴角勾起一抹弧度 / 令人窒息的压迫感 / 不怒自威

### 替代
- "全场震惊" → 1-2个具体角色的身体反应
- "他感到愤怒" → "他捏碎了茶杯"
- "冷冷一笑" → 具体面部动作或说话方式

### 句式
- 长短句交替，避免连续相同主语开头
- 段落适合手机阅读
- 减少"了"字和转折词
