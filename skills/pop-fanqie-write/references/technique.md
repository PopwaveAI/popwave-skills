## 一、代入感六大支柱

### 1. 基础信息标签化
读者先认知主角，再认知世界。百十来个字概括外貌、衣着、性格、特征。
一句话可以交代很多：地点、人物、特质、身份。
> "小爷我乃镇南府世子林峰，不过是区区六百两酒钱，醉红楼我欠了三千两都不敢这么对我，还不快给小爷我松绑？"

### 2. 具体化与可视化
描述要具体到读者脑海中能浮现的东西。
- 不写"一辆车" → 写"白色面包车"
- 不写"他愤怒" → 写"他捏碎了茶杯，滚烫的茶水流过指缝，但他像没感觉一样"
- 不写"一个漂亮女人" → 写"穿着洗得发白的牛仔外套，头发扎在脑后，指甲掐进木门漆皮"

### 3. 熟悉感
用读者经历过的场景——天然代入感。
- 高考后分手 / 被克扣工资 / 被老师冤枉 / 被室友欺负
- 接地气不等于土——白话反而能快速拉近和读者的距离

### 4. 情绪共鸣
让主角的选择和读者重叠——读者会觉得"我也会这么做"。
- 认知共鸣：主角的认知和读者保持同频
- 情绪共鸣：用亲情/不公/被欺压等天然纽带制造强情绪
- 效果：读者从"在看故事"变成"在经历故事"

### 5. 五感代入
每个场景选1-2种感官落地，不堆叠。
- 视觉：夕阳的火烧云 / 水渍形状像张嘴的鱼
- 听觉：雨水砸在铁皮瓦上 / 猫叫像婴儿哭
- 嗅觉：消毒水的味道 / 焦油味
- 触觉：潮湿的短袖黏在后背 / 铜壳的凉意
- 味觉：嘴里有股苦味 / 豆浆的甜味

### 6. 人设与代入感
- 人设不需要十全十美——缺点拉近距离
- 表面光鲜的总裁回家也丢衣服在沙发上 / 下班买淀粉肠
- 人设标签提前构思：癖好/习惯/缺点/特殊外貌
- 通过事件立人设——不是外貌描述，是在事件中看角色的行动、思考、反应

## 二、避免流水账方法

### 方法1：增加冲突
给平淡情节加前因后果，增强期待感。
- 流水账："他下班回家，吃了一口饭，看了手机，有人敲门"
- 改后："催债电话打了三次——两天内不还就上门。他啃着早上剩下的硬包子，手机里放着赚钱课的直播，砰砰砰有人敲门"

### 方法2：加入强情绪
让读者跟着着急。情绪升级链：倒霉→更倒霉→最倒霉→爆发。
- 流水账："他等公交"
- 改后："公交走了。领导看到他像疯牛一样冲出来，一把扯过去臭骂。手机被小孩撞掉进下水道。直播课结束了，没有回放"

### 方法3：从"炸楼"开始
不从砌砖开始写，从炸楼开始。
- 不是"主角考上大学，第一天入住寝室，介绍六位室友"
- 而是"主角入住寝室，发现下铺被胖子霸占，床板被弄坏，行李和锅碗瓢盆堆满"
- 开篇1-3个场景、1-3个角色——超过三个读者会混乱

## 三、语言去油清单

### 高疲劳词（单章只用1次）
冷笑 / 蝼蚁 / 轰然炸裂 / 倒吸一口凉气 / 瞳孔骤缩 / 满场死寂 / 嘴角勾起一抹弧度 / 令人窒息的压迫感 / 不怒自威

### 替代写法
- "全场震惊" → 改写1-2个具体角色的身体反应或利益震荡
- "他感到非常愤怒" → "他捏碎了茶杯"
- "气势惊人" → 用动作、器物反应、局部感官制造压迫感
- "冷冷一笑" → 写具体的面部动作或说话方式

### 句式多样化
- 长短句交替
- 避免连续相同主语开头
- 段落长度适中，适合手机阅读
- 每段聚焦1个核心信息点
- 减少"了"字和转折词（虽然…但是…/然而/却）——用多了有强行让读者认同的感觉

## 四、情绪节奏公式

### 爽点节奏
- 每3章至少1个小爽点
- 每10章1个中型高潮
- 每50章1个大型主线高潮
- 连续憋屈不超过2章——超过2章读者直接弃书

### 爽感闭环（触发→爆发→后果）
所有爽感类型均需三段式：
1. **触发**：压力/被欺/弱于对手/未知谜题
2. **爆发**：突破/反击/变强/揭示真相
3. **后果**：可见代价/地位变化/资源收益
缺一段=废章

### 钩子设计
- 每章末尾留钩子——像电视剧结尾悬念
- 钩子类型：未解问题 / 新角色出场 / 旧线索浮现 / 反转型信息
- 回收节奏：本章埋钩，下章或近章回收
- 钩子重要性递增——早期钩子要简单易回收，避免过于复杂

### 黄金三章法则（plot参考，write执行）
- ch01：核心冲突爆发——读者知道"发生了什么事"
- ch02：金手指激活——读者知道"主角凭什么翻盘"
- ch03：短期目标明确——读者知道"接下来看什么"
