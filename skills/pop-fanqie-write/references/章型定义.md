# 章型定义

> 番茄适配的4种章型，每种7节拍骨架。plot选章型时锁定密度约束，write按7节拍骨架渲染。

---

## 4种章型速查

| 章型 | 使用条件 | 读者期待 | 主动行为类型 |
|:----|:--------|:--------|:------------|
| opening_shift | 第一章/世界规则松动/金手指激活 | 先信现实→被打开更大世界 | 收获/质变级反馈 |
| confrontation_pressure | 对峙/打脸/信息差博弈/势力试探 | 看打脸反转 | 宏观层面解决问题 |
| combat_reversal | 战斗/绝境翻盘/首杀/暗杀 | 看绝境翻盘 | 战斗 |
| reveal_hook | 揭秘/新敌人/升级/章末钩子 | 看真相炸裂 | 获得反馈/质变级反馈 |

---

## 密度约束

每种章型自带密度约束。plot选类型时自动锁定，write必须遵守。

| 章型 | 战斗密度 | 系统交互密度 | 冲突密度 |
|:----|:--------|:--------|:--------|
| opening_shift | 无要求 | ≥1次（金手指激活） | ≥1次外部压力 |
| confrontation_pressure | 无要求 | ≥1次（主角用系统分析） | ≥1次信息差博弈 |
| combat_reversal | 必须有战斗（占比≥30%） | 战斗后必须弹系统反馈 | ≥1次正面交锋 |
| reveal_hook | 无要求 | 可弹升级/伏笔/异常面板 | ≥1次信息揭露一半 |

**密度分配原则**：前5章不允许连续2章无战斗类章型。前5章至少1章是combat_reversal。

---

## 7节拍骨架

每种章型7个节拍。节拍功能定义是通用的，笔触绑定在笔触DNA模板中。

### opening_shift（开篇/世界松动章）

| 节拍 | 功能 |
|:----|:----|
| 1 起笔 | 现实常识/日常/机构秩序 |
| 2 第一信息变化 | 不合常理的数据/痕迹/目击 |
| 3 压力/诱因 | 权威或群体无法解释 |
| 4 主角动作或判断 | 接近异常 |
| 5 阻碍升级 | 异常是更大物象的一角（爽感爆发点） |
| 6 可见反馈 | 群体失语/机构失效/常识崩裂 |
| 7 章末钩子 | 异常继续逼近或呼叫下一层真相 |

### confrontation_pressure（对峙/压力章）

| 节拍 | 功能 |
|:----|:----|
| 1 敌方/高位者给压力 | 出场即轻视 |
| 2 对方暴露目的/误判 | 筹码出现 |
| 3 语言羞辱/身份压制 | 被欺蓄力 |
| 4 沉默/反问/亮筹码 | 暗蓄 |
| 5 对方加码/旁人围观 | 爽感爆发点（被欺→反击） |
| 6 对方失态/局势微反转 | 打脸反馈 |
| 7 矛盾升级到下一场行动 | 章末钩子 |

### combat_reversal（战斗/反击章）

| 节拍 | 功能 |
|:----|:----|
| 1 明确对方优势/主角劣势 | 出场即轻视 |
| 2 主角发现缝隙/变量 | 物象出现 |
| 3 对方攻击/羞辱/封路 | 压制蓄力 |
| 4 不解释，直接执行 | 暗蓄 |
| 5 对方后手/位阶压制超预期 | 爽感爆发点（压力→突破）。内部=压制段≤3段+爆发段≤3行 |
| 6 身体损伤/法器破碎/敌人失态 | 代价必须可见且不可逆 |
| 7 反击引来更高后果 | 章末钩子 |

### reveal_hook（揭秘/章末钩子章）

| 节拍 | 功能 |
|:----|:----|
| 1 上一场结果/异常余波 | 余波开场 |
| 2 新解释出现 | 信息句清楚 |
| 3 解释带来更大危险 | 短句施压 |
| 4 选择追查/隐瞒/反击 | 主角动作 |
| 5 新信息牵出更高层 | 爽感爆发点（未知→揭示） |
| 6 关系/组织判断改变 | 反馈 |
| 7 下一章必须进入新场面 | 章末钩子 |

---

## 技法推荐组合

每种章型推荐的微观技法组合（可调整，但不超过3类）：

| 章型 | 推荐组合 |
|------|---------|
| opening_shift | 信息差博弈 + 感官锚点 + 预期违背 |
| confrontation_pressure | 信息差博弈 + 人设微操 + 节奏微操 |
| combat_reversal | 节奏微操 + 感官锚点 + 后果链设计 |
| reveal_hook | 预期违背 + 后果链设计 + 信息差博弈 |

---

## 关于升级和情感

升级面板和情感锤击不单独成章型：
- **升级面板**：嵌在combat_reversal的节拍6（代价/反馈）或reveal_hook的节拍6（关系/判断改变）中。
- **情感锤击**：嵌在任何章型的节拍中，不单独成章。

---

## 章型选择原则

1. **第一章必选opening_shift**——黄金开篇
2. **战斗场景选combat_reversal**——绝境翻盘
3. **对峙/谈判/打脸选confrontation_pressure**——信息差博弈
4. **揭秘/升级/新敌人选reveal_hook**——真相炸裂
5. **探索/新地点/资源发现**——合入confrontation_pressure（用信息差博弈）或reveal_hook（用未知揭示）
