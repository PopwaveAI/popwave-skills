# 章型定义（番茄版）

> pop-fanqie-plot 和 pop-fanqie-write 共用的章型体系。
> 每种章型定义7节拍骨架+概念兑现点+番茄密度约束。

---

## 13种章型速查

### 原始6种（战斗/探索流）

| 章型 | 使用条件 | 读者期待 | 概念兑现点 |
|:----|:--------|:--------|:----------|
| opening_shift | 第一章/概念首次展示 | 先信现实→被概念打开 | 节拍2-3：概念首次激活 |
| exploration_gain | 进入新场景/发现资源 | 跟着主角探索未知 | 节拍4-6：概念在新场景中的应用 |
| confrontation_pressure | 强弱对话/势力试探 | 看打脸反转 | 节拍5-6：概念提供信息差优势 |
| combat_reversal | 被压制/追杀/首杀 | 看绝境翻盘 | 节拍4-6：概念是翻盘关键 |
| high_tier_display | 远超主角层级的存在出现 | 看格局碾压+立志 | 节拍3-4：概念展示层级差距 |
| reveal_hook | 规则揭露/新敌人/升级 | 看真相炸裂 | 节拍5-6：概念揭示隐藏信息 |

### 新增7种（悬疑/情感/知识/搞笑流）

| 章型 | 使用条件 | 读者期待 | 概念兑现点 |
|:----|:--------|:--------|:----------|
| mystery_deduction | 推理博弈/规则解谜 | 看主角智斗规则 | 节拍4-5：概念提供关键线索 |
| atmosphere_horror | 氛围营造/恐惧蓄力 | 看诡异氛围+恐惧感 | 节拍3-4：概念展示诡异面 |
| emotional_pressure | 虐心蓄力/情感压迫 | 看虐心后爆发 | 节拍5-6：概念/身世带来反转 |
| sweet_interaction | 甜宠互动/反差萌 | 看心动瞬间 | 节拍4-5：概念驱动互动 |
| knowledge_display | 知识展示/智力碾压 | 看知识打脸 | 节拍4-6：概念=专业知识 |
| comedic_subversion | 搞笑反差/反套路 | 看反差笑点 | 节拍3-5：概念使用出错或反转 |
| time_skip_montage | 时间跳跃/成长蒙太奇 | 看积累式成长 | 节拍2-3：概念年度/周期使用 |

## 番茄密度约束

### 原始6种

| 章型 | 事件密度 | 爽感 | 概念交互 | 面板/金手指 |
|:----|:--------|:------|:--------|:-----------|
| opening_shift | 8-10 | ≥1（概念激活） | ≥1次（首次展示） | 概念首次弹出 |
| exploration_gain | 8-12 | ≥1（资源获取） | ≥1次（辅助探索） | 概念辅助检定 |
| confrontation_pressure | 8-12 | ≥1（打脸/反转） | ≥1次（信息差） | 概念分析数据 |
| combat_reversal | 10-12 | ≥1（翻盘爆发） | ≥1次（战斗关键） | 概念结算反馈 |
| high_tier_display | 8-10 | ≥1（格局震撼） | 1次（对比差距） | 概念显示差距 |
| reveal_hook | 8-10 | ≥1（真相揭示） | ≥1次（揭示信息） | 概念弹出伏笔 |

### 新增7种

| 章型 | 事件密度 | 爽感 | 概念交互 | 替代密度指标 |
|:----|:--------|:------|:--------|:-----------|
| mystery_deduction | 4-6 | ≥1（悬疑反转） | ≥1次（线索推进） | 悬念密度≥3 |
| atmosphere_horror | 4-6 | ≥1（恐惧释放） | 1次（诡异展示） | 悬念密度≥3 |
| emotional_pressure | 5-7 | ≥1（虐心释放） | 1次（概念影响情感） | 情感密度≥2 |
| sweet_interaction | 5-7 | ≥1（甜宠反差） | 1次（概念驱动互动） | 情感密度≥2 |
| knowledge_display | 3-5 | ≥1（知识碾压） | ≥1次（专业展示） | 信息密度≥4 |
| comedic_subversion | 5-7 | ≥1（搞笑反转） | 1次（概念出错/反转） | 笑点密度≥2 |
| time_skip_montage | 3-5 | ≥1（积累爆发） | ≥1次（周期使用） | 成长节点≥3 |

密度分配原则：前5章不允许连续2章无战斗类或推理类章型。前5章至少1章是combat_reversal或mystery_deduction。

## 7节拍骨架

### opening_shift（开篇/概念激活章）

| 节拍 | 功能 |
|:----|:----|
| 1 起笔 | 现实常识/日常/主角处境（前300字必出冲突） |
| 2 第一信息变化 | 概念激活——不合常理的数据/能力出现 |
| 3 压力/诱因 | 概念的限制条件暴露（无限制=无张力） |
| 4 主角动作或判断 | 主动使用概念 |
| 5 阻碍升级 | 概念使用引发更大问题（爽感爆发点） |
| 6 可见反馈 | 概念反馈——数据/物品/信息到手 |
| 7 章末钩子 | 更大的威胁或可能性出现 |

### exploration_gain（探索/资源露头章）

| 节拍 | 功能 |
|:----|:----|
| 1 起笔 | 环境反常或新场景入场（感官序列：嗅→听→触→视） |
| 2 资源/线索露头 | 概念辅助发现目标 |
| 3 危险阻挡 | 风险评估——概念限制条件施压 |
| 4 判断收益，冒险 | 主动选择深入 |
| 5 危险扩大或深处异动 | 爽感爆发点 |
| 6 资源到手/能力变化 | 概念结算反馈 |
| 7 更深处还有更大变化 | 章末钩子 |

### confrontation_pressure（对峙/压力章）

| 节拍 | 功能 |
|:----|:----|
| 1 敌方/高位者给压力 | 出场即轻视 |
| 2 对方暴露目的/误判 | 筹码出现 |
| 3 语言羞辱/身份压制 | 被欺蓄力 |
| 4 沉默/反问/亮筹码 | 概念提供信息差 |
| 5 对方加码/旁人围观 | 爽感爆发点（被欺→反击） |
| 6 对方失态/局势微反转 | 打脸反馈 |
| 7 矛盾升级到下一场行动 | 章末钩子 |

### combat_reversal（战斗/反击章）

| 节拍 | 功能 |
|:----|:----|
| 1 明确对方优势/主角劣势 | 出场即轻视 |
| 2 主角发现缝隙/变量 | 概念提供关键信息 |
| 3 对方攻击/羞辱/封路 | 压制蓄力 |
| 4 不解释，直接执行 | 概念驱动战术 |
| 5 对方后手/位阶压制超预期 | 爽感爆发点（压力→突破） |
| 6 身体损伤/敌人失态 | 代价可见+概念结算反馈 |
| 7 反击引来更高后果 | 章末钩子 |

### high_tier_display（高位展示章）

| 节拍 | 功能 |
|:----|:----|
| 1 高位存在出场/气息压场 | 威压即事件 |
| 2 普通判断失效 | 独句段 |
| 3 旁观者/环境承受不住 | 概念显示差距——侧面定级 |
| 4 观察/避让/借势/立志 | 主角动作 |
| 5 高位行为造成大范围后果 | 爽感爆发点（未知→揭示） |
| 6 天地/组织/群众影响变化 | 反馈 |
| 7 主角未来必须抵达此层级 | 章末钩子 |

### reveal_hook（揭秘/章末钩子章）

| 节拍 | 功能 |
|:----|:----|
| 1 上一场结果/异常余波 | 余波开场 |
| 2 新解释出现 | 概念揭示隐藏信息 |
| 3 解释带来更大危险 | 短句施压 |
| 4 选择追查/隐瞒/反击 | 主角动作 |
| 5 新信息牵出更高层 | 爽感爆发点（未知→揭示） |
| 6 关系/组织判断改变 | 反馈 |
| 7 下一章必须进入新场面 | 章末钩子 |

### mystery_deduction（推理博弈章）— 新增

| 节拍 | 功能 |
|:----|:----|
| 1 规则发布/困境呈现 | 生存压力开场 |
| 2 信息收集/观察异常 | 概念提供隐蔽线索 |
| 3 初步判断被打破 | 预期违背 |
| 4 重新推理/发现关键 | 概念提供关键信息差 |
| 5 推理验证/真相揭示 | 爽感爆发点（悬疑反转） |
| 6 对手失态/规则反噬 | 反馈 |
| 7 更大的谜题出现 | 章末钩子 |

## 技法推荐组合

每种章型推荐的微观技法组合（不超过3类）：

| 章型 | 推荐组合 |
|------|---------|
| opening_shift | 信息差博弈 + 感官锚点 + 预期违背 |
| exploration_gain | 世界观渐进揭示 + 感官锚点 + 后果链设计 |
| confrontation_pressure | 信息差博弈 + 人设微操 + 节奏微操 |
| combat_reversal | 节奏微操 + 感官锚点 + 后果链设计 |
| high_tier_display | 高位视角插入 + 恐怖奇观美学化 + 感官锚点 |
| reveal_hook | 预期违背 + 后果链设计 + 信息差博弈 |
| mystery_deduction | 信息差博弈 + 预期违背 + 节奏微操 |
| atmosphere_horror | 恐怖奇观美学化 + 感官锚点 + 情感锚点设计 |
| emotional_pressure | 情感锚点设计 + 后果链设计 + 人设微操 |
| sweet_interaction | 人设微操 + 预期违背 + 节奏微操 |
| knowledge_display | 信息差博弈 + 世界观渐进揭示 + 预期违背 |
| comedic_subversion | 预期违背 + 人设微操 + 节奏微操 |
| time_skip_montage | 节奏微操 + 后果链设计 + 人设微操 |
