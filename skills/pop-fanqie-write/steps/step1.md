## 1b. 设定库精选注入（禁止全文注入）

**红线：设定库不全文注入，只精选与本章相关的设定**

检测路径：`0-立项/参考书拆书/设定库/` 或 decon产出的Phase3 L1六件套

### 精选规则

根据本章白描卡/场景卡的章型+事件，只注入本章会用到的设定维度：

| 本章特征 | 注入的设定维度 |
|---------|--------------|
| 战斗章 | 力量体系（属性/技能/专长）+ 物种与天赋（种族特征） |
| 探索章 | 世界蓝图（地理）+ 势力格局（组织） |
| 交易章 | 资源与物品（价格/物品） |
| 对话章 | 术语与文明底色（专有名词）+ 历史与驱动力（背景） |
| 成长章 | 力量体系（升级/职业）+ 物种与天赋（天赋觉醒） |

### 注入方式

- 每个维度只提取与本章事件直接相关的段落（≤500字/维度）
- 最多注入2个维度（≤1000字总设定）
- 禁止注入"与本章无关的设定历史/势力详情/物品清单"

**原则：设定库是write层的"查字典"，不是"读全书"。write需要什么查什么，不查的就不注入。**

**如设定库不存在**：跳过本步，依赖剧情白描（剧情白描应该已经消化了设定库的关键信息）。

## 1c. 加载笔触DNA（如存在）

**检测路径**：`0-立项/参考书拆书/笔触DNA.md`

如存在，加载并提取以下维度（write时必须绑定）：
- **叙事距离**：近/中/远，何时切换
- **句式特征**：长句/短句节奏，对话引导词
- **物象系统**：反复出现的物象，象征意义
- **对话风格**：角色语言特征，方言/俚语
- **情绪外化**：通过什么外化情绪（动作/物象/环境）
- **章型节拍绑定**：参考书怎么用章型节拍

如不存在（用户未给参考书），跳过本步，write时用通用笔触约束。
