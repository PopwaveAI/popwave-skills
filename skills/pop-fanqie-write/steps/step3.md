# Step 3: 写正文

> 三层指导：结构 / 节奏 / 爽感
> 笔触层：从DNA取（启用态）或用户显式声明（trial模式）
> 目标：2000-2500字，不准超过2500字

---

## 笔触层（从DNA取，不在本step约束）

**启用态**（`涌现/文风锚定.md` 存在）：
- 叙事距离/句式节奏/物象/对话引导词/情绪外化/感官顺序/段落呼吸/系统面板格式，全部从DNA取
- write skill不内置任何笔触约束

**trial模式**（无DNA）：
- write skill不内置任何默认风格
- 用户须显式声明基础风格倾向，或接受API自由发挥
- 交付面板标注"trial模式·无DNA驱动"

---

## 三层指导

### 第1层·结构层

**按章型7节拍骨架组织结构**。每节拍有明确功能，不要混乱。

详见 `references/章型定义.md`。

**字数分配建议**：

| 段落 | 字数 | 内容 |
|------|------|------|
| 节拍1-2 | 300-400字 | 起笔+第一信息变化 |
| 节拍3-4 | 400-500字 | 压力/诱因+主角动作 |
| 节拍5 | 500-700字 | 阻碍升级/爽感爆发点 |
| 节拍6 | 400-500字 | 可见反馈 |
| 节拍7 | 200-300字 | 章末钩子 |
| **合计** | **2000-2500字** | |

### 第2层·节奏层

遵守节奏物理量（番茄12岁小学生读者模型）：

| 指标 | 约束 |
|:----|:----|
| 事件间隔 | ≤20行 |
| 事件密度 | 8-12个/章 |
| 蓄力上限 | ≤2段 |
| 信息释放 | ≤3段 |
| 微钩子 | 每3-5行 |

> 注：感官连续/情绪连续的字数限制属于**风格层**，由DNA决定。trial模式不内置。

### 第3层·爽感层

- **主推爽感必须触发**（章型绑定的主推爽感）
- **爽感闭环**：触发→爆发→后果
- **爆发后三段式**：爆发≤3行 → 反馈≤2段 → 余韵≥1段
- **小爽沿途**：每3-5事件1个小爽
- **即时奖励**：每章≥1次行动→可见回报

## 写作禁区

禁止：
- 写思考过程/检查表/分析框架
- 章末没有钩子
- 超过2500字
