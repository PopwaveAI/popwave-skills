# Step 3: 写正文

> 五层指导：笔触 / 节奏 / 格局 / 爽感 / 微观
> 目标：2000-2500字，不准超过2500字

---

## 五层指导

### 第1层·笔触层

**如启用笔触DNA**：叙事距离/句式/物象/对话/情绪外化从DNA中来。

**如未启用**：用通用笔触约束：
- 叙事距离：近为主，关键场景切近
- 句式：短句为主，长句用于铺垫
- 对话引导词只用"道"
- 系统面板用【】内联格式，禁止markdown代码块
- 情绪外化：通过动作和物象，禁止直接写"他感到..."
- 角色思考碎片化，禁止完整分析句

### 第2层·节奏层

遵守节奏物理量（番茄12岁小学生读者模型）：

| 指标 | 约束 |
|:----|:----|
| 事件间隔 | ≤20行 |
| 感官连续 | ≤3行 |
| 情绪连续 | ≤2行 |
| 事件密度 | 8-12个/章 |
| 蓄力上限 | ≤2段 |
| 信息释放 | ≤3段 |
| 微钩子 | 每3-5行 |

**按章型7节拍骨架组织结构**。每节拍有明确功能，不要混乱。

### 第3层·格局层

- 本章在长线中的位置（推进了哪条L2剧情线？）
- 本章的势力格局变化（主角的位置变化？）
- 本章的世界观信息释放（释放了什么设定？）

### 第4层·爽感层

- **主推爽感必须触发**（章型绑定的主推爽感）
- **爽感闭环**：触发→爆发→后果
- **爆发后三段式**：爆发≤3行 → 反馈≤2段 → 余韵≥1段
- **小爽沿途**：每3-5事件1个小爽
- **即时奖励**：每章≥1次行动→可见回报

### 第5层·微观层（v7.2.0增强）

落地Step 2c选的3-4类微观技法：

**基础6类技法**：
- **信息差博弈**：围绕"谁知道什么"设计至少1次博弈
- **感官锚点**：每场景至少1个具体生理感官细节
- **预期违背**：建立1个预期，然后打破
- **节奏微操**：一章内快慢交替至少2次
- **人设微操**：主角+1配角各有1个小动作/小习惯展示
- **后果链设计**：章内事件产生至少1个可见后果涟漪

**v7.2.0新增技法**：
- **动作链** ⭐：战斗/日常/情绪动作必须拆成2-3个细动作，禁止概括式。每章至少3处，战斗章至少5处
- **多感官优先级** ⭐：每场景至少覆盖3种感官（视觉+听觉+触觉/嗅觉），不堆叠（每种≤3行）。每章至少2处完整多感官描写，战斗章至少3处

**动作链落地检查**（写作时自检）：
- [ ] 是否有"他做了XX"这种概括式动作？
- [ ] 战斗动作是否拆成2-3个细动作？
- [ ] 情绪动作是否通过身体细节外化？

**多感官落地检查**（写作时自检）：
- [ ] 开篇场景是否覆盖3种感官？
- [ ] 战斗场景是否覆盖触觉+视觉+听觉？
- [ ] 是否有连续超过3行的同一种感官？

## 写作禁区

禁止：
- 写思考过程/检查表/分析框架
- 用markdown代码块写系统面板（用【】内联）
- 对话引导词用"说/问/答"（只用"道"）
- 直接写"他感到悲伤/愤怒"（用动作/物象外化）
- 角色大段内心独白（碎片化表达）
- 章末没有钩子
- 超过2500字
- **用1个动词概括连续动作**（v7.2.0新增）
- **场景只覆盖1种感官**（v7.2.0新增）

| 节拍5 | 500-700字 | 阻碍升级/爽感爆发点 |
| 节拍6 | 400-500字 | 可见反馈 |
| 节拍7 | 200-300字 | 章末钩子 |
| **合计** | **2000-2500字** | |
