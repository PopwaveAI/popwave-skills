### 动力引擎全书演化表（v1.0.0新增）

> **核心新增**：标注每卷的引擎状态 + 演化节点。引擎不是静态的——长篇中引擎会演化（飞轮加速/压迫升级/探索扩展/范式转换），演化节点必须与全书卷纲对齐。
>
> **v1.1.0强化**：每卷必须明确标注"本卷演化触发（是/否）"——不能留空或只标注部分卷。

| 卷 | 引擎范式 | 引擎状态 | 本卷演化触发（是/否） | 触发事件+演化方向（如有） | 众生攀登方式变化 | 代价结构变化 | 主角在引擎中的位置 |
|:--|:--|:--|:--|:--|:--|:--|:--|
| 第一卷 | （从Step 2b组成5取） | 飞轮启动/压迫初现/探索起步 | 是——引擎建立 | 引擎建立：众生攀登逻辑首次展示 | 众生攀登方式分层确立 | 通用代价显现 | 主角作为众生之一进入攀登闭环 |
| 第二卷 | | 飞轮加速/压迫升级/探索扩展 | 是/否（根据设计） | （如是）触发事件XX→演化方向XX / （如否）引擎维持XX状态加速 | 新群体登场/旧群体变化 | 代价累积开始 | 主角凭金手指加速，触及第一层代价 |
| 第三卷 | | | 是/否 | | | 代价突变 | 主角到达中阶位，引擎阻力显现 |
| 第四卷 | | | 是/否 | | | | |
| 终卷 | | 引擎终极状态——闭环完成或断裂 | 是/否 | | 众生攀登格局终态 | 代价清算 | 主角到达坐标系顶层 |

**要求**：
- 每卷标注引擎状态（飞轮加速/压迫升级/探索扩展/范式转换/闭环断裂）
- **每卷必须标注"本卷演化触发（是/否）"**——无演化的卷标注"否，引擎维持XX状态"；有演化的卷标注"是"+触发事件+演化方向。禁止只标注首次演化而后续卷留空。
- 如有演化节点，标注触发事件+演化方向（对齐Step 2b组成6）
- 众生攀登方式变化——新群体登场/旧群体变化（直接关联势力全书展开表）
- 代价结构变化——代价累积/突变（直接关联危机全书展开表的"引擎代价"层面）
- 主角在引擎中的位置——主角作为众生之一，每卷在攀登闭环中的位置变化

## 3c. 各卷设定切片预览

**v1.0.0新增**：每卷设定切片增加"动力引擎切片"——标注本卷的引擎状态+演化节点。

```markdown

- **力量范围**：阶位1-阶位2

### 第二卷设定切片

- **卷末设定边界**：...

### 第三卷及后续同上格式
```

**注意**：这是给plot的导航索引，plot每卷执行时从设计/全书设定/各卷切片.md取对应切片做本卷设定切片（plot v3.2.0新增步骤）。
