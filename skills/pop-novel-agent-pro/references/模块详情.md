# 模块详细描述 & 链路信息

> 归档内容，按需查阅。根 SKILL.md 不包含以下信息。

## 模块详细描述

### A. 基础设施层

**web-access v2.4.3** — 所有联网操作唯一入口
- WebSearch / WebFetch / curl / Chrome CDP
- 胶水需求：无

**cnovel-research / book-opinion-tracker** — 平台调研 + 舆情追踪
- 胶水需求：无

### B. 开书启动层

**project-bootstrap v2.9**
- core_profile 嵌入、数值体系模板（combat_capability/monster_rank_map/act_rank_schedule）、超越性硬检查

**book-deconstructor v4.8** — 五模式拆书（A/B/C/D/E）

### C. 剧情设计层

**plot-architecture v2.7**
- 事件数≥3/章、scene_type 多样性（同类型≤2章连续）、字数基线2000-2500
- 输出：act-XX.yaml（含 fun_level / scene_time / event_density）

### D. 正文写作层

**emergent-writer v9.3** — 六阶段管线
- Director→QC→骨架→QC→ESM before→Pass2→QC→ESM after
- 数据层：v3.db（characters/weirds/skills/items + state_changelog）
- 知识注入：K1核心理论 / K2平台画像 / K3量化基准 / K4题材感知
- 依赖：project.yaml、v3.db、锚定章库、拆解报告

### E. 质检层

**qa-payoff v0.4.1** — 纯感受型质检
- 不评分、不提建议、不规律验证
- 红线："想跳过"≥2处 或 "会弃书"=是 → 退回重写
