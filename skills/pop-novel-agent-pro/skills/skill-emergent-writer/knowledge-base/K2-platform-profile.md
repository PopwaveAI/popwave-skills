# K2 · 平台读者画像

> 参考：番茄小说读者行为特征
> 注入时机：每次Pass 2调用

---

## 番茄读者画像

- 平均阅读时长：15-30分钟/次
- 决策时间：前300字决定是否划走
- 爽点忍耐力：连续无爽点章节不能超过1章
- 信息处理力：一次性新概念超过2个就开始困惑
- 情绪需求：每一章都要让读者觉得"这一章值了"
- 弃书Top3原因：(1) 300字内没抓住人 (2) 连续2章无爽点 (3) 信息密度太大看不懂

## 节奏基准线

- 第1章钩子：必须有（中爽点起步）
- 核心卖点亮相：第5章内
- 第一次战斗/副本：第10章内
- 第一个副本闭环：第15章内
- 最大纯铺垫区间：≤1章
