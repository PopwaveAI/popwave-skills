# K3 · 量化检测基准

> 注入时机：每次Pass 2调用
> 如果发现偏离，直接在写作中修正，不需要停下来报告。

---

## 基准线

| 指标 | 基准值 | 检测方式 |
|:-----|:-------|:---------|
| 对话占比 | ≥25% | 肉眼估计 |
| 新概念/章 | ≤2个 | 写作时注意 |
| 章末钩子 | 必须 | 三种之一 |
| 连续无爽点章节 | ≤1章 | 读设计说明中的情绪时刻表 |
| 连续独白段 | ≤15行 | 插入威胁/悬念/冲突/感官异常 |
| "不是…而是…"句式 | 0容忍 | 写作时注意避免 |
| "他感到/他意识到/他评估" | 尽可能减少 | 写作时注意避免 |
