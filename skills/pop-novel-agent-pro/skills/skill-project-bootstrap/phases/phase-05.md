# Phase 0.5：跨域素材聚合 ★ 强制 | 禁止跳过

**目标**：从 L0 灵魂对齐 + 核心卖点出发，跨越多个知识领域搜索素材，将外部研究转化为 L1 设定层的设计养分。

> ⚠️ **设定浅的根因就在于这一步被跳过。** Phase 1 的 L1 文件只靠 LLM 内部知识——那是"泛常识"。跨域搜索把"泛常识"变成"有根系的设定"。

**条件**：PRD 已定稿，金手指设计完成。

**强制规则**：禁止跳过。L1 设定层的丰富度依赖此阶段素材质量。

## 执行步骤

1. **卖点词拆解**
   - 从 PRD 核心卖点中提取 3-5 个关键词/短语

2. **跨域搜索**（每个关键词 1-2 次 WebSearch，至少覆盖 4 个领域）

| 类别 | 方向 | 示例 |
|------|------|------|
| 地理/环境 | 真实地理/建筑/生态 | 深海热泉、喀斯特地貌 |
| 生物/怪物 | 古生物/神话/生态位 | 寒武纪奇虾、中庭巨蛇 |
| 社会/组织 | 历史组织/秘密社团 | 共济会、中世纪行会 |
| 技术/魔法 | 科技/伪科学/神秘学 | 卡巴拉生命之树、炼金术 |
| 文化/哲学 | 哲学概念/文化原型 | 尼采永恒轮回、英雄之旅 |
| 心理/情绪 | 认知科学/行为模式 | 斯德哥尔摩综合症、峰终定律 |

3. **素材蒸馏** — 每方向提取≥2个设定种子，标注来源/关联度/可转化性
4. **设定转化** — 映射到 L1 六个维度，最少 3 个种子转化到具体段落

**产出**：
- `00-原始设定/_素材聚合/跨域素材蒸馏.md`
- L1 各文件的「素材来源注释」

**量化标准（自动检查，不达标补充）**：
| 检查项 | 门槛 | 不达标处理 |
|--------|:---:|:----------:|
| 覆盖领域数 | ≥ 4 | 补充未覆盖领域 |
| 设定种子总数 | ≥ 8 | 补充搜索 |
| 转化到 L1 种子数 | ≥ 3 | 补充转化映射 |
| 蒸馏报告字数 | ≥ 1500字 | 补充素材描述 |
