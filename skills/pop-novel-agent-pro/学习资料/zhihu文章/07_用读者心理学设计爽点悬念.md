# 用读者心理学设计网文爽点+悬念

> 来源：头条号（网文写作心理学）
> 主题：爽点设计、悬念设计、读者心理学

## 一、3个核心读者心理

1. **补偿心理**：读者现实缺什么，网文就要给什么
2. **攀比优越心理**：人天生喜欢"比别人强"
3. **情绪蓄力效应**：情绪先压抑，再爆发才最爽

## 二、7大心理学爽点模板

1. **反差爽**：身份/实力反差（扮猪吃虎）
2. **打脸爽**：精准打击看不起你的人
3. **特权爽**：获得别人得不到的东西
4. **守护爽**：保护在意的人
5. **碾压爽**：降维打击，智商/实力全面压制
6. **选择爽**：别人求而不得，主角随便选
7. **逆袭爽**：从低谷到巅峰

爽点节奏：每3000-5000字一个小爽点，每章结尾一个微爽点。

## 三、4个心理学悬念原理

1. 未完成执念：人对未完成的事记忆最深
2. 怕失去心理：害怕主角死亡/失败/失去
3. 信息差悬念：知道一部分，不知道另一部分
4. 代入焦虑：把读者绑在主角身上

## 四、5类悬念模板

1. 危机悬念（最常用，结尾绝杀）
2. 身份悬念（长线钩子）
3. 选择悬念
4. 反转悬念
5. 信息悬念

## 五、爽点+悬念黄金搭配公式

**单章节奏**：铺垫压抑→小爽点反击→升级危机→结尾抛悬念

**卷节奏**：开局困境→多次小爽点逆袭→大危机悬念→终极大爽点→长线悬念

核心：先抑后扬，爽完立刻抛悬念，悬念解开给爽点，循环往复。
