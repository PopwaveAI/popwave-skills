## 一、静官角色五大设计技法

> 来源：`/s9062233`

### 技法1：绰号——第一时间吸引眼球

"经典作品除了出彩的主角，一定有特别出彩的配角。而配角最容易让人记住的方式就是绰号。"

```
五虎上将 → 关二爷、张三爷、赵子龙（绰号比本名流传更广）
网文中 → "大猩猩""胖子""小妖精""老鬼""疯狗"
```

**原则**：绰号要有记忆点，符合角色特征，一叫就知道是谁。

### 技法2：口头禅——让角色开口就活

- 要醒目、有哲理，符合人物身份
- 例：痞子类角色每句带一个口头禅→读者一看到这个句式就知道是谁在说话

### 技法3：出场动作——心理暗示广告

"一个固定的动作可以成为角色的身份标签。"

例：《古惑仔》陈浩南——掐烟、抓耳梢
一旦建立，读者看到这个动作就自动联想到角色的气场。

### 技法4：缺点设计——丰富角色的关键

"没有个性的人就是平庸的人。"

- 太完美的主角读者不喜欢——因为"我不完美"
- 给主角设计2-3个缺点（冲动/优柔/自负/抠门/好色）
- 缺点不能是致命硬伤（不能是真正的人品问题）

### 技法5：个性反差——让人物立体

"一个只有一面的角色是纸片人。"

**反差公式**：
```
表面性格 + 关键时刻的相反表现 = 立体角色

例：
  野蛮的女主角 → 关键时候柔情似水
  冷血的杀手 → 对一个小孩子网开一面
  抠门的配角 → 为了兄弟倾家荡产
```

## 二、柳下挥的人物塑造核心

> 来源：`/s9062619`

### 主角设定的关键

**"人活着，都要有个盼头。要给读者盼头。"**

读者追一本书的动力来源：
1. 想看主角什么时候翻身（爽点）
2. 想看主角什么时候收了这个妹子（暧昧）
3. 想看主角什么时候解决了那个对手（打脸）

### 角色三要素

| 要素 | 作用 | 操作 |
|:---|:---|:---|
| **女人** | 让读者有念想 | 暧昧阶梯：调情→触碰→牵手→亲吻→……但别太快 |
| **对手** | 让读者有期待 | 分级：小对手10章解决，大对手纠缠全书 |
| **朋友** | 让读者有代入感 | 朋友的忠诚/背叛都会牵动读者情绪 |

**金句**："一本小说，有两点非常重要：一、女人；二、对手。"

### 暧昧升级阶梯

```
1. 语言调情
2. 无意间的触碰
3. 牵手
4. 亲吻
5. 赤裸相对
6. 叉叉OO
```

**关键原则**：不要走太快。"邻家女主直到三百多万字还没扑倒，订阅反而越来越好。"——因为有盼头。

### 关于女性角色的写作

**静官补充**：
- 女人写得好是"拖字数骗稿费的不二绝学"
- 但写不好女性角色宁可干脆不写，也不要写花瓶和工具人
- 梦入神机很少在女性角色上浪费笔墨，一样红得发紫

## 三、罗霸道代入感法则

> 来源：`/s9241276`

### 代入感的核心公式

**"写书的终极奥义：设定一个有缺陷的主角，把缺陷放大，制造冲突，解决之。"**

```
主角有缺陷 → 缺陷被放大 → 产生冲突 → 解决冲突 → 主角成长 → 新缺陷出现（循环）
```

**大多数新人犯的错误**：急着掩盖缺陷，而不是放大缺陷。

```
❌ 正确操作（放大缺陷）：
  主角怕血 → 必须参加一场血腥的战斗 → 克服恐惧 → 读者爽

❌ 错误操作（掩盖缺陷）：
  主角怕血 → 但他是天才，一上战场就不怕了 → 毫无张力 → 读者无感
```

### 用主角性格推动情节

"用主角性格推动情节，而不是用情节推动。性格推动无法预测，更有活力。"

```
❌ 情节推动：主角接到一个任务（外部驱动）→ 去做
✅ 性格推动：主角是个执着的 → 一定要查出真相（内部驱动）→ 去做
   → 读者能理解"为什么他要这么做"
```

### 周边人物的真实性

"龙套真实了，对手真实了，女人真实了，主角自然就真实了。"

**方法**：
- 龙套不要只写"路人甲"——给他们一个小特征（"那个老是在抠鼻子的保安"）
- 对手不要写"坏"——写他们有合理性（"他这么做是因为……"）
- 女人不要写"美"——写她们有性格（"她虽然好看，但一开口就能把人气死"）

## 四、人物塑造常见错误（ibiling综合）

### 错误1：主角出场太晚

**表现**：前500字在写天气、背景、祖先历史，主角还没出现。
**修正**：第一段就让主角在行动中登场。

### 错误2：主角太完美

**表现**：英俊、聪明、强大、人缘好、家世好——读者不喜欢，因为"我不可能是这样"。
**修正**：给主角2-3个明显的缺点，让读者觉得"我也是这样"。

### 错误3：配角工具化

**表现**：配角只在需要时出现，用完就消失。
**修正**：给配角一个小标签（绰号/口头禅/职业特征），每次出现都维持这个标签。

### 错误4：反派智商不在线

**表现**：反派为了坏而坏，做蠢事被打脸。
**修正**：反派要有合理性。"反派做的事如果换一个角度看，他也觉得自己是对的。"
