## 一、用好这四招，烂文笔也能写出好故事

> 来源：`/s8957903`

### 第一招：学会留白

**话不说太满，留白比情绪渲染更重要。**

| 层级 | 写法 | 案例 |
|:---:|:---|:---|
| 普通 | 直接写情绪 | "我酗酒、抽烟、哭泣……" |
| 中级 | 隐藏情绪让读者揣摩 | "银行卡放在桌子下第二个抽屉，房门钥匙要再配一把……" |
| 高级 | 用抽象物讲人话 | 王家卫的凤梨罐头——"不知道从什么时候开始，什么东西上面都有一个日期" |

**做减法**：
- 去掉"的"、"觉得"、"感觉"、"简直"等虚词
- 去掉重复表述
- 精简前：凌晨的火车站，寒冷、凄清……
- 精简后（木心）：清早上火车站，长街黑暗无行人

**核心金句**："好文就像山水，留白足够，让读者思考，大道至简、返璞归真。"

### 第二招：增加文字张力

**操作方法**：
1. 切掉无实际意义的虚词（"的""觉得""感觉""简直"）
2. 切掉重复表述
3. 精准用词、删繁就简、重新组合

**示例**：
```
原版：他感到非常愤怒，简直无法控制自己的情绪，他觉得整个世界都在与他作对
精简版：他攥紧了拳头。无所谓了。

原版：这一幕让他感觉到前所未有的恐惧
精简版：腿在抖。不是冷的。
```

### 第三招：让物说话

用物体表达观点，产生喜剧性和冲突性：
- 不写"他很穷"，写"他翻遍了所有口袋，找到三块钱和一个过期的优惠券"
- 不写"她很伤心"，写"她撕了又写，写了又撕，纸篓里全是揉成团的信纸"

### 第四招：满足猎奇

利用读者对未知事物的天生好奇：
- 恐怖故事要打破常规、突破思维界限
- 展示读者没见过的东西（新规则/新生物/新设定）

## 二、罗霸道视角法则

> 来源：`/s9241276`

### 核心法则

**"描写的奥妙是不要描写天空，不要描写马路，不要描写行人走卒，务必始终以主角的视线焦点来看。"**

```
❌ 上帝视角："酒店的装修很豪华，天花板上有水晶灯，大厅里有黑帮在闹事……"
✅ 主角视角："推开门，第一眼就看到了那个光头。他站在大厅中间，
   身后五个人一字排开，衬衫下鼓鼓囊囊的。水晶灯的光打在他亮铮铮的脑门上，
   反射出一道刺眼的光。"
   → 通过描写光头把天花板、酒店装饰"捎带"勾勒出来
```

**原则**：读者不是站在场外看戏，是通过主角的眼睛在看世界。

### 停顿感的营造

用时间和距离交错的写法制造热血沸腾的感觉：

```markdown
一秒，两秒，三秒……主角心里默默计算着……七秒，到了！
```

用段落分开制造节奏：

```markdown
五米！
——更近了！
两米！
——到了！
```

## 三、恐怖小说五大技法

> 来源：`/k2605661`
> 对诡异游戏项目有直接参考价值

### 1. 构建故事背景

| 元素 | 推荐 | 不推荐 |
|:---|:---|:---|
| 地点 | 古老建筑、偏僻之地、超自然空间 | 日常生活场所（除非加了超自然元素） |
| 历史 | 有前科/有传说/发生过怪事 | 纯新建的场所 |
| 规则 | 有一套隐形的恐怖规则 | 无规则的纯吓人 |

### 2. 塑造角色

**主角要有弱点**：
- 胆小/多疑/偏执 → 恐惧放大的载体
- 但要有闪光点（勇敢面对/保护同伴）→ 读者才不会厌烦

**恐怖反派设计**：
- 外形诡异 + 行为诡异 并存
- 例：纸人（外形诡异：纸做的身体）+ 会动（行为诡异）+ 摸着主人的身体在长（行为诡异升级）

### 3. 营造恐怖氛围

**三感综合运用**：
- 视觉：阴暗、潮湿、看不清的角落
- 听觉：滴水声、脚步声、低语声
- 嗅觉：腥臭味、铁锈味、腐朽味

**节奏要张弛有度**：不能一直高强度，要有舒缓间隔。紧张→舒缓→更紧张。

### 4. 设计恐怖情节

- **利用未知与不可解释**：超自然现象、神秘规则——不要解释原理
- **制造惊悚转折**：剧情反转、意外死亡或消失——读者以为安全了，其实更危险

### 5. 细节描写

| 类型 | 作用 | 示例 |
|:---|:---|:---|
| 心理细节 | 深入刻画恐惧焦虑 | 心跳声、手心出汗、呼吸困难 |
| 动作细节 | 用身体语言表现恐惧 | 颤抖的手、沉重的脚步、跌跌撞撞的奔跑 |
| 环境细节 | 暗示危险逼近 | 窗外的影子越来越近、烛火突然熄灭 |

## 四、柳下挥人物出场描写

**人物描写先做人物卡，标清性格特征后"死抓着这一点深写"。**

- 每个人物有多面性，不可能每一面都刻画完美——抓住一点深写即可
- 如果没有烽火那样的风骚文笔，**建议少写心理描写，尽量加快剧情**
- 减少无意义的路人甲镜头：用特征代替名字（"那个光头男""那个痘痘男"）

### 外貌描写的原则（ibiling综合）

**不做全身扫描**：
```
❌ "他身高一米八，剑眉星目，穿着一件黑色风衣……" — 像通缉令
✅ "他的眼睛像鹰一样锐利，盯着人看的时候让人后背发凉。"
   → 一个特征就够
```

## 五、提升文笔的具体做法

### 句式锤炼

1. **长短句交替**：长句描写氛围/铺垫，短句制造节奏/冲击
2. **去冗词**：删掉"的""地""得"的过度使用
3. **动词代替形容词**：
   - ❌ "他很愤怒地踢开了门"
   - ✅ "他一脚踹开了门"
   - ❌ "他悲伤地看着窗外的雨"
   - ✅ "雨打在窗上，他看着，一动不动"

### 画面感的核心

**画面感 = 具体细节 + 感官动词**

```
❌ 抽象：天气很冷
✅ 画面感：呼出的气在面前凝成一团白雾，手指冻得发红
❌ 抽象：他很紧张
✅ 画面感：手心全是汗，他在裤子上擦了擦，又擦了擦
```
