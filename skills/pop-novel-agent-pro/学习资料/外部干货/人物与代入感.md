# 人物与代入感

> 来源：多篇网文人物设计文章综合
> 核心：商业网文的角色不需要"复杂人性"，需要"3秒站队"

---

## 一、主角人设公式

### 欲望 + 困境 + 选择

```
他最想要什么？（欲望）
现在最大的障碍是什么？（困境）
他会做出什么决定？（选择）
```

这三个要素决定了读者能不能记住主角。

**案例拆解**：
- 萧炎：欲望（夺回尊严）+ 困境（斗之力三段）+ 选择（三十年河东三十年河西）
- 叶凡：欲望（活下去/强大起来）+ 困境（无法修炼的荒古圣体）+ 选择（偏要逆天而行）

### 主角标签铁律

第一章必须让读者记住主角的 **3 个特质**：

```
例：陈默
  ① 重生者（冷静、预知）—— 有信息优势
  ② 过阴人世家 —— 有特殊能力传承
  ③ 谨慎型性格 —— 遇到异常先观察再行动

3 个特质立刻建立：读者知道主角是谁、会怎么做、为什么能做成
```

**反例**（普通废柴流）：
```
"他很善良、很努力、很坚韧"—— 这些是品质不是特质，任何主角都有
```

## 二、脸谱化角色设计

### 为什么脸谱化是对的

读者的"认知带宽"有限。脸谱化能让读者 3 秒站队，跟着情绪走。

### 角色标签要像《甄嬛传》的华妃

华妃的标签：**"刁蛮跋扈、深爱皇上"** → 读者立刻知道"这是反派，得看甄嬛怎么收拾她"。

### 三要素角色设计法

| 角色类型 | 标签 | 案例 |
|:---|:---|:---|
| 主角 | 核心特质 3 个 | 陈默（重生·过阴人·谨慎） |
| 反派 | 一种可恨 + 一种强大 | "阴险但实力碾压主角" |
| 伙伴 | 一种可爱 + 一种有用 | "呆萌但战力惊人" |
| 导师 | 一种神秘 + 一张底牌 | "古怪但关键时刻总能出手" |

**不用写复杂人性的反派**（有悲惨童年、隐藏阴谋）。在网文里，这叫"自嗨式注水"。

## 三、代入感三要素

### 要素 1：主角不能太完美

完美角色读者代入不了。主角要有：
- 缺点（冲动/自负/自卑）
- 犹豫的时刻
- 恐惧的时刻

### 要素 2：动机要"普通"

```
❌ "我要拯救世界" — 读者没有代入感
✅ "我要活下去" — 谁都想活下去
✅ "我要保护家人" — 谁都想保护家人
✅ "我要证明自己不是废物" — 谁都不想被看不起
```

**离读者越近的动机，代入感越强。**

### 要素 3：用读者的视角写

```
❌ 上帝视角："主角很牛逼，他一拳打死了一条龙"
   → 读者站在外面看着主角

✅ 主角视角："拳峰上全是血，不知道是自己的还是龙的。
   眼前一阵发黑，但对面那个庞然大物终于不动了。"
   → 读者通过主角的眼睛感受世界
```

## 四、配角管理

### 配角的三不原则

1. **不抢戏**：配角再有趣也不能比主角出彩
2. **不废话**：配角的台词必须有推进功能
3. **不死板**：同一类型标签的配角不要超过 1 个

### 配角的功能分类

| 功能 | 定位 | 数量控制 |
|:---|:---|:---:|
| 陪衬 | 展示主角的强/爽 | 可多，但每个只出现一次 |
| 交付信息 | 给主角提供线索/资源 | 用完可退场 |
| 制造障碍 | 推动主角成长 | 阶段性出现 |
| 情感锚点 | 读者关心的"其他人" | 全书 2-3 个 |

**经验法则**：每本书读者能记住的角色不超过 5-7 个。超过的都是工具人，不要投入太多笔墨。
