# 行业实操指南

> 来源：番茄作家助手公开规则 + 多篇作者实战分享
> 核心：知道平台规则，才能写出匹配算法的作品

---

## 一、番茄小说

### 字数策略

| 阶段 | 建议字数 | 原因 |
|:---|:---:|:---|
| 开篇章节 | 2000-2500 字 | 高于 3000 字直接影响完读率 |
| 正常章节 | 2000-3000 字 | 最有利于算法推荐的区间 |
| 高潮章节 | 2500-3500 字 | 可以略长，但需要情绪密度支撑 |
| 单章底线 | ≥ 1500 字 | 低于 1500 字影响流量权重 |

**死守 1000 字底线 → 流量权重极低**  
**盲目写 3000-4000 字大章 → 节奏拖沓读者跑路**

### 算法节奏

番茄的推荐算法高度依赖完读率。
- 前 10 章完读率 ≥ 8%：进入推荐池
- 追更率 ≥ 10%：持续推荐
- 首秀（8 万字-15 万字）：7 天大量曝光测试

### 番茄读者画像

- 碎片化阅读，平均 3 秒划走一本
- 喜欢快节奏、高密度信息释放
- 不喜欢大段环境描写和设定科普
- 对话要多、动作要快、别抒情

## 二、起点中文网

### 签约偏好

| 类型 | 编辑偏好度 | 原因 |
|:---|:---:|:---|
| 玄幻/仙侠 | 高 | 起点基本盘，但竞争激烈 |
| 都市/异能 | 高 | 读者粘性强，付费率高 |
| 系统/游戏 | 中 | 需要差异化，同质化严重 |
| 轻小说 | 中 | 新兴品类，门槛低但天花板也低 |
| 现实/历史 | 低-中 | 需要功底，新人慎入 |

### 起点读者画像

- 老书虫为主，阅读经验丰富
- 对套路极度敏感，一眼看出"模板"
- 愿意等铺垫，但必须值
- 对文笔和节奏要求高于番茄
- 付费意愿高，但弃书成本低

### 起步策略

1. 内投编辑（先投 1-3 万字大纲+开头）
2. 编辑过稿后签约
3. 发书后冲榜（新人榜/分类榜）
4. 日更 4000-6000 字以上才能稳定推荐

## 三、通用规则

### 标题公式

```
好标题 = 主角身份 + 核心冲突 + 爽感暗示

✅ 《开局一座山：从破产领主到帝国之盾》
   ↑主角身份 ↑核心冲突  ↑爽感暗示

❌ 《命运之轮》
   ↑读者不知道这是讲什么的
```

### 简介公式

```
好简介 = 一个具体的场景 + 核心悬念 + 一句话爽点承诺

✅ "穿越到诡异世界的第一天，陈默发现自己变成了纸人。
    纸人七日水火不侵，但第七天过后就会活过来。
    不过这不是坏消息——因为他也活过来了。
    （前作《我的诡异人生》均订过万，新书已签约）"

❌ "这是一个少年在异世界冒险成长的故事。"
```

### 日更策略

| 阶段 | 日更字数 | 策略 |
|:---|:---:|:---|
| 发书期（前 2 周） | 4000-6000 | 快速积累章节，冲推荐 |
| 稳定期（签约后） | 4000 字 | 维持数据，不断更 |
| 瓶颈期 | 3000 字 | 宁可减少字数，不中断更新 |
| 高潮期 | 4000-6000 | 趁读者情绪高时多更 |

**不断更 > 字数多**。断更一天，推荐权重降一周。

## 四、数据驱动写作

### 需要关注的核心指标

| 指标 | 健康值 | 预警值 |
|:---|:---:|:---:|
| 章末读完率 | ≥ 60% | ≤ 50% → 钩子或节奏有问题 |
| 前 3 章留存 | ≥ 30% | ≤ 20% → 开篇需要重写 |
| 追更率 | ≥ 10% | ≤ 5% → 故事进入疲软期 |
| 读者平均阅读时长 | ≥ 60s | ≤ 30s → 内容不吸引人 |

### 根据数据调整写作

```
某章跳出率突然升高
    ↓
检查这章前 300 字开头是否慢了？
这章是否是纯过渡段？
这章末端钩子是否失效？
    ↓
定位问题 → 针对性修改
如果是钩子问题 → 重写章末
如果是内容无法问题 → 接受（不能每章都高能）
```

## 五、新人避坑清单

| # | 坑 | 表现 | 怎么躲 |
|:---:|:---|:---|:---|
| 1 | 幻想一本封神 | 第一本书期待过高 | 第一本书的目标是签约，不是火 |
| 2 | 完美主义卡文 | 因为文笔不好反复改开头 | 先写完 10 万字再回头看改 |
| 3 | 追热点但不擅长 | 哪本书火写哪本 | 写自己擅长的，热点的技术含量很高 |
| 4 | 日更扛不住就断 | 断更一次后持续断 | 前期存稿 10 章再发书 |
| 5 | 不重视数据 | 埋头写不看后台 | 每天看一次完读率，根据数据调 |
| 6 | 开局就写大主线 | 第一章就写"拯救世界" | 从小处切入，让读者先关心主角 |
