# 黄金三章理论

> 来源：起点/番茄编辑公开分享 + 多个爆款拆解文章综合
> 核心观点：前三章是留存生死线，决定算法推荐和签约资格

---

## 一、黄金三章的底层逻辑

黄金三章的本质不是"写得好不好"，而是**预期管理**——给读者画一张"爽点蓝图"，让他们看完前三章就产生追更冲动。

编辑审稿流程：
- 起点编辑：3-5 分钟扫前三章，先看开局、再看节奏、最后看文笔
- 番茄编辑：看前 3000 字完读率 + 前 3 章留存率

读者流失规律：
- 第一章流失超过 50%（平台均值）
- 前 300 字（手机一屏）决定是否划走
- 前 7 行是生死线（"黄金七行"）

## 二、第一章：出场 + 危机 + 钩子

### 必须完成的三件事

| 任务 | 具体做法 | 反例 |
|:---|:---|:---|
| 让主角立刻出场 | 第一段就让主角在行动中登场 | 先写 500 字世界观背景 |
| 给出当前困境 | 300 字内引爆核心冲突 | 写主角的日常流水账 |
| 埋下第一个钩子 | 章末留一个"想知道接下来"的悬念 | 章末是旁白总结 |

### 3000 字的标准分配

```
0-300 字（第一屏）：砸冲突、锚主角、造悬念
  必须出现的三样东西：
  - 冲突/危机/反转 —— 让读者立刻产生"怎么回事"
  - 主角是谁 —— 用行动/对话展现，不用介绍
  - 一个让读者想往下划的钩子

300-2000 字：推进 + 验证
  展示冲突的严重性
  让读者理解"主角为什么必须面对这个"

2000-3000 字：首个小高潮 + 下章悬念
  给出第一次能力/机会展示
  章末留钩子
```

### 起点的第一章 vs 番茄的第一章

| 平台 | 节奏 | 写法 |
|:---|:---|:---|
| 起点 | 相对从容，前三章完成建置 | 可以花一章铺垫，但前 500 字必须钩住 |
| 番茄 | 极快，第一章必须出金手指信号 | 前 3000 字必须有冲突+能力+悬念 |

## 三、第二章：冲突 + 升级 + 期待

### 核心原则

**不要缓一缓。** 读者被第一章的危机钩住后，第二章绝不能放松。

### 必须做的事

| 任务 | 具体做法 |
|:---|:---|
| 压力拉满 | 把第一章的困境推到更糟 |
| 破局伏笔 | 暗示有解法，但不立刻兑现 |
| 金手指亮相 | 第二章末尾必须让读者看到"主角有翻盘的可能" |
| 期待感 | 让读者心里冒出"我倒要看主角怎么翻盘" |

### 常见错误

- ❌ 写日常/写回忆/写世界观补充
- ❌ 把第一章得到的能力立刻用掉
- ❌ 反派直接投降了（冲突消解）

---

## 四、第三章：反击 + 甜头 + 新悬念

### 核心原则

**给一点甜头，再给一个更大的钩子。** 第三章是决定读者追不追更的关键。

### 必须做的事

| 任务 | 具体做法 |
|:---|:---|
| 让主角完成一次小爆发 | 展示新能力的初步效果 |
| 解决眼前的小危机 | 给读者第一次满足感 |
| 结尾抛更大悬念 | 暗示更大的危险/更高的目标 |

### 完美收尾公式

```
主角获得了一个阶段性成果 ✅
    ↓
但发现这只是一个开始 ⚠️
    ↓
更大的威胁/更高的目标出现 ❓
    ↓
读者：我再看一章
```

## 五、具体写法模板

### 模板 A：废柴觉醒型

第一章：(被辱/被打/被退婚) → 隐藏实力暗示 → 章末"我其实不是废柴"
第二章：危机升级（家族危机/宗门考核）→ 金手指露头 → 章末准备爆发
第三章：小试牛刀 → 打脸/证明自己 → 章末更大的麻烦出现

### 模板 B：重生/穿越型

第一章：重生瞬间（前世惨死）→ 意识到重生时机 → 章末"这次我要改变一切"
第二章：利用先知优势获得第一个优势 → 但发现情况有变 → 章末新蝴蝶效应
第三章：第一次成功验证 → 获得新同伴/资源 → 章末本以为掌握了全貌，但上一世忽略的暗线浮现

### 模板 C：系统降临型

第一章：宿主的危机 → 系统激活/金手指提示 → 章末"我能用这个翻盘"
第二章：系统规则揭示 + 第一次使用尝试 → 但系统有代价/限制 → 章末必须在时限内完成某事
第三章：系统带来的第一次小成功 → 副作用/代价出现 → 章末更大的敌人发现了这个系统

## 六、起点 vs 番茄的差异化

| 维度 | 起点 | 番茄 |
|:---|:---|:---|
| 读者画像 | 老书虫、中深度阅读 | 碎片化阅读、高流失率 |
| 首章节奏 | 快但可以有点铺垫 | 极快，3 秒不吸引人就走 |
| 字数偏好 | 2000-4000 字/章 | 2000-3000 字/章 |
| 金手指时机 | 前 3 章出现即可 | 前 2 章必须暗示 |
| 爽点密度 | 允许蓄力 2-3 章 | 每章必须有一个小爽点 |
| 文笔要求 | 更高，讲究节奏感 | 偏向通俗直白 |
