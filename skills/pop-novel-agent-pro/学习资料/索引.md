## 索引使用指南

### 写作前（开书阶段）

```
1. 外部干货/黄金三章理论.md → 了解平台标准
2. 拆书报告/密藏域/... → 了解诡异人生世界观设定的底层逻辑
3. 拆书报告/四本对照/四本50章_横向校准... → 诡异游戏取舍得
```

### 写作中（market-test阶段）

```
1. 外部干货/钩子设计指南.md → 检查章末钩子类型
2. 外部干货/爽点与节奏系统.md → 检查节奏指标
3. 拆书报告/... → 加载赛道对标检查项
```

### 写作后（复盘阶段）

```
1. 自我复盘/诡异游戏/... → 回顾自己踩过的坑
2. 自我复盘/灰骑士/... → 跨项目教训借鉴
```
