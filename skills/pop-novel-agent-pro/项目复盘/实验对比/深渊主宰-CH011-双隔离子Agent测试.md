## 二、关键发现：是"写出来的"，不是"仿出来的"

### 2.1 Agent B从未见过原文

Agent B的生成prompt中**没有任何一句原文**。它只有：
1. 事实骨架（带 {实体} 标记的35行）
2. 实体卡（角色身份/palette/说话风格/状态）

它不知道原文里费尔牧师说了什么，不知道安亚丽穿什么，不知道索伦怎么发现洞口——这些都是它自己从骨架+ palette 中生成的。

**结论：排除了"你读过原文所以在Prompt里偷塞原文"的疑虑。**

### 2.2 同一个骨架，不同"表演"

| 维度 | 原文 | Agent B Pass 2 | 评价 |
|:-----|:-----|:---------------|:-----|
| 晨曦神殿交涉 | 费尔牧师春风满面说长篇台词 | 费尔牧师"掂了掂钱袋，笑容重新舒展开" | 更克制，符合索伦视角 |
| 属性世界观 | 集中大段解释（10-20点力量对照表） | 拆成三处嵌入："大约码头苦力水平"、"接近举重运动员" | 信息密度降低但更可读 |
| 阴影掌握觉醒 | "身影渐渐朦胧，宛若消失在了阴影中" | "冰凉力量从体内涌出，暗影从脚下蔓延包裹全身" | 原文更简洁，Pass 2更详细 |
| 背刺骷髅 | 18(+1)伤害→背刺未秒杀→分析 | 同上+手臂发麻的持续体感 | 增加了物理反馈 |
| 章末 | 属性世界观大段展开+"……" | "十二点力量还是太弱了"→"得先解决这只亡灵" | 钩子更清晰 |

**结论：风格不同，但质量在同一水平。体系不是"原样复制"，是"在骨架约束下有自己的表达"。**

### 2.3 暴露的一个问题

Agent B对晨曦神殿的对话篇幅偏短（原文约40行，Pass 2约20行）。这不是情节缺失——是所有节点都在——而是**渲染深度不足**。Pass 2倾向于"把事讲完就往前走"，而不是"在这个场景里多待一会儿"。

这可能是因为fact skeleton只标注了"发生了什么"，没有标注"这个场景的情绪重量是多少"。Pass 2不知道"晨曦神殿交涉"是本章第一大场景，和"乱葬岗入口"是同一个权重处理的。

**改进建议**：事实骨架中标注每个场景的情绪权重（weights），告诉Pass 2哪个场景该花更多笔墨。

## 四、体系验证结论

### ✅ 双隔离子Agent测试通过。

这套体系的核心假设——**"事实骨架+实体卡→Pass 2渲染能产出可用正文，不需要看过原文"**——在CH011上成立。

### 测试的四层隔离

| 隔离层 | 是否通过 | 说明 |
|:-------|:--------:|:-----|
| Agent B未读原文 | ✅ | 未读取原txt文件 |
| Agent B未收到原文片段 | ✅ | Prompt只有骨架+卡，无原文 |
| 事实骨架不含感官细节 | ✅ | 骨架=事件序列，无修辞描写 |
| 实体卡无原文信息 | ✅ | 身份/状态层，不含具体台词/描写 |
