## 三条复盘主线的关系

```
灰骑士（渊界系统）                   诡异游戏（独立项目）
     │                                      │
     │  v4.0-v7.2 主战场                    │  v2-v7.6 从零设计
     │  管线架构 + DB驱动                    │  技能架构迭代
     │  约束实验 × 4 轮对照                   │  竞品对标驱动
     │                                      │
     └──────────────┬───────────────────────┘
                    │
                    ▼
          novel-agent-pro 融合
          v7.6 当前架构
          (导演层+碎片写作+QC)
                    │
                    ▼
          v9 ESM 管线
          (SQLite数据中台 + 事实/描述双层隔离)
                    │
                    ▼
          Vibe Creating（Docs 当前）
          (全管道创造者操作系统)
```

**灰骑士贡献**：管线稳定性、红线自动化、DB管理、对照实验方法论。
**诡异游戏贡献**：碎片写作、导演层、场景卡资产化、极简约束、竞品全方位对标。
**v9 ESM 贡献**：SQLite全书数据中台、事实/描述双层隔离、before三层加载、after双写闭环。
