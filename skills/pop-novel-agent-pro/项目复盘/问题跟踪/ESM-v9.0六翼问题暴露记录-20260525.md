## 三、角色问题专项（跨章节）

| 角色 | 当前状态 | 核心问题 |
|:-----|:---------|:---------|
| 陈默 | 被动响应型主角 | 没有一句能让人记住的主动台词。全程被事件推着走，不主动制造冲突。读者不知道他胆小还是冷静还是冲动 |
| 老狗 | 登场有辨识度，第4章后停滞 | "卧槽建模太细"有效。但此后没有性格进化——每次死亡没有留下性格变化 |
| 白芷 | 标签（冷静分析型）停留在纸面 | 分析型角色没有展示分析能力的台词/行为。"数钥匙串"是观察不是分析 |
| 叔公 | 永远隔着一层 | 没有当下的对话、没有活着的互动。读者对他的情感连接来自二手材料 |
| 老徐 | ch010 刚登场，未展开 | 有潜质但未验证 |

### 角色问题的根因链条

```
L0 人设契约太薄
  └─→ 只有关键词（3特质+3动机+3缺点），没有行为锚定
  └─→ 正文中不知道角色在压力下会说什么做什么
  └─→ 所有人的反应趋同——沉默/观察/等待
  └─→ 角色之间无法通过台词区分
```
