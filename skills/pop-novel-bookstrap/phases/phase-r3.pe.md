# Phase r3：逆向提取 — L1 元设定层

## 前置条件
Phase r1 全部完成（全本事件日志 + 批次摘要齐全）。

## 加载参考
先加载 `phases/phase-1.pe.md`，了解新六件套的维度和字段标准。

## 执行步骤

### 1. 从事件日志合成 L1
基于全本事件日志 + 批次摘要中的 L1 候选，合并去重后，按新六件套维度输出：

| # | 维度 | 从事件日志提取方向 |
|:-:|:-----|:-----------------|
| 01 | 世界蓝图 | 正文中展现的时空结构、法则、力量来源 |
| 02 | 力量体系 | 正文中出现的境界、修炼路径、职业体系 |
| 03 | 历史与驱动力 | 正文提及的纪元划分、大事件、核心矛盾 |
| 04 | 物种与天赋 | 正文中出现的主要种族、特殊体质/血脉 |
| 05 | 势力格局 | 正文中出现的势力组织、权力关系 |
| 06 | 资源与物品 | 正文中实际使用的重要物品 |

**注意**：提取的是"正文里实际表现了的规则"，不是"设计出来的规则"。如果事件日志中某条规则只出现过 1 次 → 标注为"待确认"。

### 2. 产出
`00-原始设定/L1-元设定层/01-06.md`

### 3. 用户确认
逐文件确认，用户可以修改/补充。
