# Phase r4：逆向提取 — 卷纲/幕地图还原

## 前置条件
Phase r1 + Phase r3 完成。

## 加载参考
先加载 `phases/phase-r1.pe.md` 的事件日志，了解已有正文的章节结构。

## 执行步骤

### 1. 从事件日志归纳卷/幕结构
从 Phase r1 产出的事件日志中归纳：
- 卷划分：正文自然形成的卷边界（通常以重大事件/地点转移/时间跳跃为界）
- 幕划分：每卷内的幕结构（情绪推进单元）
- 每幕的地理锚点：关键场景发生在什么地点

### 2. 产出
- `设计/卷/volume-XX.md`：每卷的地理总览+时空规则
- `设计/幕/vol-XX/act-YY.yaml`：每卷内的幕结构还原

### 3. 用户确认
不可跳过。卷纲和幕结构是后续写作的骨架。
