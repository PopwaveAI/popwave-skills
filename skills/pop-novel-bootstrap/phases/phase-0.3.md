# Phase 0.3：参考书甄别

**目标**：系统筛选对标参考书，为后续设定提供借鉴与差异化锚点。

**条件**：Phase 0 灵魂对齐完成，PRD.md 已存在。

## 执行步骤

1. **参考书粗筛**
   - 基于核心创意+核心爽点，搜索 5-10 本潜在对标作品
   - 排除标准：均订<500、风格差异过大、>10年无翻新
   - 保留 2-4 本进入深度分析

2. **深度分析**（调用 pop-novel-deconstructor skill）
   - 对每本参考书执行模式B「对标拆解」
   - 提取：节奏地图、核心设定规则、爽点触发模式、读者痛点/槽点
   - 输出 `00-原始设定/_参考书分析/{书名}.md`

3. **差异化决策**
   - 对比 PRD 中的差异化优势，确认「我不做什么」
   - 生成差异化清单：明确标注「借鉴点」「偏离点」「空白创新点」
   - 输出：差异化决策备忘录（嵌入 PRD.md 尾部）
