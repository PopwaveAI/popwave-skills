# Phase 0.3：参考书甄别

## 前置条件
story-engine.yaml 已锁定。

## 加载参考
先加载 `phases/phase-0.3.ref.md`。

## 执行步骤

### 1. 参考书粗筛
- 使用 WebSearch 搜索同类作品
- ⚠️ **禁止使用 LLM 内部训练数据中的作品**
- 排除：均订<500、风格差异过大、>10年无翻新
- 保留 2-4 本

### 2. 深度分析
调用 pop-novel-deconstructor 做对标拆解。输出 `_参考书分析/{书名}.md`

### 3. 差异化决策
完成借鉴/偏离/空白三列表，每项至少 2 条。嵌入 story-engine.yaml 的尾部作为 appendix。

**差异化表模板**：

```
✅ 借鉴点 | [一句话描述] | 为什么借鉴 | 我们会怎么做
❌ 偏离点 | [一句话描述] | 我们的判断依据
🆕 空白   | [一句话描述] | 为什么我们认为市场有这个需求
```

**示例**（遮天 vs 凡人修仙传）：

```
✅ 借鉴 | 凡人的人界→灵界→仙界三层升级结构 | 清晰的三段式读者认知锚点 | 我们也可以分三界但改变每界的矛盾主题
❌ 偏离 | 凡人主角的"低调谨慎"人设 | 我们的主角需要更主动制造冲突 | 改为"被迫高调"——每次出手都有代价
🆕 空白 | 二者都没有的"跨文化宗教冲突"设定 | 结合克苏鲁元素的仙侠尚无成功先例 | 在修炼体系中引入"信仰污染"机制
```

## 检查清单

- [ ] 差异化清单完成（每列≥2条）
