# Phase 0.3 参考：分析模板

## 四维度分析

| 维度 | 提取内容 |
|:-----|:---------|
| 节奏地图 | 前100章情绪曲线、爽点间隔、钩子密度 |
| 核心设定规则 | 世界观展开方式、信息释放时机 |
