# Phase 0.4：金手指概念设计

**目标**：设计有约束、有代价、有成长弧线的金手指系统。

**条件**：参考书分析完成，差异化方向已明确。

## 执行步骤

### 节点 A：金手指导演判断

| 作品类型 | 金手指必要性 |
|----------|-------------|
| 系统流/无限流/穿越重生 | **必须** — 金手指是剧情发动机 |
| 都市/现实/历史 | **可选** — 可用天赋/知识/技能替代 |
| 纯文学/悬疑推理 | **不推荐** — 破坏悬念/代入感 |

> 判断为「不推荐」→ 跳至 Phase 0.4.2（无金手指替代路线）

### 四级评估
| 级别 | 评估项 | 说明 |
|------|--------|------|
| L1 | 核心功能 | 金手指做什么？一句话说清楚 |
| L2 | **约束条件** | 使用限制？（冷却/代价/条件/副作用） |
| L3 | **成长弧线** | 金手指自身是否成长/进化？升级条件？ |
| L4 | **叙事功能** | 如何驱动剧情？（解决问题/制造冲突/暴露性格） |

### 三必答问题
1. 代价是什么？— 必须可感知、可累积、不可轻易消除
2. 成长空间是什么？— 是否有明确升级路径？升级伴随新限制？
3. 什么情况下金手指会失效？— 必须有失灵场景作为张力来源

### 文化根系检查
- 金手指的文化原型/隐喻是什么？
- 是否与世界观基调一致？
- 是否与目标读者的文化认知框架兼容？

### 番茄视角验证
- 前3章：金手指是否出现？足够吸引人？
- 前10章：是否持续驱动剧情？制造明确期待？
- 前50章：成长曲线可见？仍有新意？

**产出**：
- `00-原始设定/L1-元设定层/04-金手指.md`（含约束条件矩阵、成长树、失效场景清单）

### Phase 0.4.2（替代路线）：无金手指替代方案

1. 主角的核心驱动力是什么？（欲望/使命感/诅咒/债务……）
2. 剧情推进的发动机是什么？（解谜/追凶/生存/复仇……）
3. 替代金手指的「不对称优势」：
   - 知识差异（穿越者/行业专家）
   - 认知差异（高维度思考方式）
   - 资源差异（人脉/遗产/秘密组织）
   - 体质差异（特殊血统/天赋/变异）

**产出**：`00-原始设定/L1-元设定层/04-无金手指替代方案.md`
