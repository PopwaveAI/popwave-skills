# Phase 0.4：金手指概念设计

## 这份文档存在的意义

金手指不是"给主角开个挂就行"——它是整本书的**剧情发动机**。

金手指设计文档存在的意义是：确保金手指不是一个"想用就用"的工具，而是一套有约束、有代价、有成长弧线的系统。更重要的是——**它必须是"主角的"金手指，不是随便换个人也能用的万能工具**。

**没有金手指设计的后果**：写到中期发现主角已经无敌了（没有约束）/ 金手指变成背景板，主角靠自身能力解决问题（金手指被遗忘）/ 读者觉得"凭什么是他，换我也行"。

---

## 必须回答的问题

1. **为什么是主角被选中？** — 金手指的挑选逻辑是什么？主神空间/帝国/系统为什么偏偏找上他？（这是金手指的"根基设定"，不是写完就忘的东西）
2. **主角凭什么脱颖而出？** — 同样的金手指，第一个人用了变成废物，第二个人用了封神。区别在哪？（主角的个人特质/性格/判断力如何让金手指发挥更大价值）
3. **金手指自身的成长故事是什么？** — 不是"秦川变强了"——是"这套系统自己也在进化"。解锁新功能需要什么条件？
4. **这套系统和本作常规力量体系的区别在哪？** — 两者是什么关系？（替代/互补/冲突/伪装？）读者如何区分"主角自己的能力"和"金手指给的能力"？
5. **什么情况下金手指会失效？** — 必须有明确的「金手指失灵场景」作为剧情张力来源。没有失效场景的挂就是无敌流。

---

## 执行步骤

### 节点 A：金手指导演判断

| 作品类型 | 金手指必要性 |
|----------|-------------|
| 系统流/无限流/穿越重生 | **必须** — 金手指是剧情发动机 |
| 都市/现实/历史 | **可选** — 可用天赋/知识/技能替代 |
| 纯文学/悬疑推理 | **不推荐** — 破坏悬念/代入感 |

> 判断为「不推荐」→ 跳至 Phase 0.4.2（无金手指替代路线）

### 四级评估（围绕上述 5 个必答问题展开）

| 级别 | 评估项 | 对应问题 |
|------|--------|---------|
| L1 | 核心功能 | 金手指一句话做什么？ |
| L2 | **约束条件** | 使用限制？（冷却/代价/条件/副作用）→ 对应 Q5 |
| L3 | **成长弧线** | 金手指自身是否成长/进化？→ 对应 Q3 |
| L4 | **叙事功能** | 如何驱动剧情？→ 对应 Q2（主角的决策让金手指发挥不同价值）|

### 三必答问题
1. 代价是什么？— 必须可感知、可累积、不可轻易消除
2. 成长空间是什么？— 是否有明确升级路径？升级伴随新限制？
3. 什么情况下金手指会失效？— 必须有失灵场景作为张力来源

### 文化根系检查
- 金手指的文化原型/隐喻是什么？（例如：仙秦功勋体系源自中国秦制的军功爵位制）
- 是否与世界观基调一致？
- 是否与目标读者的文化认知框架兼容？

### 番茄视角验证
| 阶段 | 检查点 |
|:-----|:-------|
| 前 3 章 | 金手指是否出现？读者看完能说清"主角有什么挂"吗？ |
| 前 10 章 | 金手指是否持续驱动剧情？是否制造了明确的期待？ |
| 前 50 章 | 成长曲线可见？是否有新意？读者还对金手指有期待吗？ |

- 包含：核心功能定义、约束条件矩阵、成长树、失效场景清单
- ⚠️ 必须回答上述 5 个必答问题，缺一不可

### Phase 0.4.2（替代路线）：无金手指替代方案

如果作品不需要金手指，回答以下问题：
1. 主角的核心驱动力是什么？（欲望/使命感/诅咒/债务……）
2. 剧情推进的发动机是什么？（解谜/追凶/生存/复仇……）
3. 替代金手指的「不对称优势」：
   - 知识差异（穿越者/行业专家）
   - 认知差异（更高维度的思考方式）
   - 资源差异（人脉/遗产/秘密组织）
   - 体质差异（特殊血统/天赋/变异）

**产出**：`00-原始设定/L1-元设定层/04-无金手指替代方案.md`
