# Phase 0.4：金手指概念设计

> 该文件同时作为执行指令和参考指南，无需加载外部 ref 文件。

## 前置条件
参考书分析完成。

## 执行步骤

### 1. 金手指导演判断
系统流/无限流=必须；都市/历史=可选；悬疑/纯文学=不推荐

### 2. 围绕5个必答问题展开
1. 为什么是主角被选中？
2. 主角凭什么脱颖而出？
3. 金手指自身的成长故事？
4. 和常规力量体系的区别？
5. 什么情况下失效？

### 3. 四级评估
用以下框架评估金手指质量。四层全部通过才算合格。

| 级别 | 合格标准 |
|:-----|:---------|
| L1 核心功能 | 一句话说清金手指做什么 |
| L2 约束条件 | 至少3条约束，不可轻易消除 |
| L3 成长弧线 | 有明确升级路径，升级伴随新限制 |
| L4 叙事功能 | 驱动剧情——给主角制造选择，不是加数值 |

**Before/After 参考**：

❌ 坏的金手指：系统签到流
```
每天签到送100经验值，连续签到7天送稀有道具。
主角不需要做任何决策——只要登录就行。
```
问题：L1 清晰，但 L2 约束 = 0，L3 成长 = 0。主角不决策、不付出代价。

✅ 好的金手指：荒古圣体（遮天）
```
荒古圣体——肉身强横、万古无双。但被天地压制，每突破一个境界需要比常人多走九个小台阶。
L1：肉身无敌——同级对手打不破防御
L2：被天地压制 → 每级多走九个小台阶 → 每突破一次都要付出额外时间/资源/冒险代价
L3：随着境界提升，圣体的优势逐步解锁（如不灭金身、圣体异象）
L4：这决定了主角的成长节奏和每一次战斗的策略——他用肉身硬扛是因为他炼不起境界
```

### 4. 番茄视角验证

| 阶段 | 检查点 |
|:-----|:-------|

| 前50章 | 成长曲线可见？读者仍有期待？ |
