# Phase 0.5：跨域素材聚合 ★ 强制 | 禁止跳过

## 这份文档存在的意义

大部分小说的设定"浅"，不是因为作者没创意——是因为**所有信息都来自同一类来源**（其他网文+LLM训练数据）。

跨域素材聚合存在的意义是：把真实的、外部的、不同领域的知识拉进来，转化成设定的养分。这不是"百度百科搬运"——是从地理学、生物学、历史学、神秘学等不同领域找到可用的设定种子，让世界观有根系。

**没有跨域素材的后果**：L1 设定层全是"泛常识"——精灵住在森林里、法师塔有魔法阵、反派想统治世界。读者一眼就能看穿这是"另一本看过的小说的翻版"。

---

## 必须回答的问题

1. **我的核心创意放在哪个领域能找到独特的素材？** — 从 PRD 核心卖点拆出 3-5 个关键词，每个关键词从 4 个不同领域搜索。
2. **这些外部知识如何让世界更真实？** — 不是"我找到了 XX 生物叫这个名字"——是"这个真实存在的生态系统/组织结构/哲学概念如何启发我设计某个设定"。
3. **至少 3 个种子最终转化成了什么具体设定？** — 素材不能停在"收藏夹里"，必须变成 L1 文件的具体段落。

---

## 执行步骤

### 1. 卖点词拆解
从 PRD 的核心卖点中提取 3-5 个关键词/短语。

### 2. 跨域搜索（每个关键词 1-2 次 WebSearch，至少覆盖 4 个领域）

| 类别 | 方向 | 示例 |
|------|------|------|
| 地理/环境 | 真实地理/建筑/生态 | 深海热泉、喀斯特地貌、中世纪城堡构造 |
| 生物/怪物 | 古生物/神话/生态位 | 寒武纪奇虾、北欧中庭巨蛇、深海生物发光机制 |
| 社会/组织 | 历史组织/秘密社团 | 共济会结构、中世纪行会、秦制二十等爵 |
| 技术/魔法 | 科技/伪科学/神秘学 | 卡巴拉生命之树、炼金术原理、纳米工程 |
| 文化/哲学 | 哲学概念/文化原型 | 尼采永恒轮回、佛教六道、英雄之旅原型 |
| 心理/情绪 | 认知科学/行为模式 | 峰终定律、习得性无助、认知失调 |

### 3. 素材蒸馏
每方向提取≥2 个"设定种子"，每个种子标注：
- 来源
- 与核心卖点的关联度
- 可转化性（直接使用/改编使用/仅作灵感）

### 4. 设定转化
将素材种子映射到 L1 六个维度。**最少 3 个种子最终转化到 L1 文件的具体段落。**

**产出**：
- `00-原始设定/_素材聚合/跨域素材蒸馏.md`
- L1 各文件的「素材来源注释」

### 量化标准（自动检查，不达标补充）

| 检查项 | 门槛 | 不达标处理 |
|--------|:---:|:----------:|
| 覆盖领域数 | ≥ 4 | 补充未覆盖领域 |
| 设定种子总数 | ≥ 8 | 补充搜索 |
| 转化到 L1 的种子数 | ≥ 3 | 补充转化映射 |
| 蒸馏报告字数 | ≥ 1500字 | 补充素材描述 |
