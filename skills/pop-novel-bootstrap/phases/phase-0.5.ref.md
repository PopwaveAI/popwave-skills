# Phase 0.5 参考：搜索领域与质量检查

## 六大搜索领域

地理/环境 → 社会/组织 → 生物/怪物 → 技术/魔法 → 文化/哲学 → 心理/情绪

## 质量导向检查表

| 检查层级 | 标准 | 判断方式 |
|:---------|:-----|:---------|
| L1 可定位 | 至少 2 个种子被明确标记为"已嵌入 L1 的哪个文件哪段" | 检查种子末尾是否有文件+字段名标记 |
| L2 有结构 | 至少 1 个种子是该领域独有的结构（不是"魔法=火焰"这种通用知识） | 问自己：这个种子是"谁都知道"还是"只有这个领域的人才知道"？ |
| L3 可融入 | 嵌入 L1 后，想象一段正文试写——种子能不能自然融入场景？ | 产出 2-3 句正文试写 |

## 质量标准
- 数字不是目的，**有结构差异的种子**才是目的
- 一个"该领域专家才知道的冷知识"比 10 个"百度百科通用词"有用
