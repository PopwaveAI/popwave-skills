# Phase 0 参考：PRD 结构与压力测试标准

## PRD 六要素

| 要素 | 说明 |
|:-----|:------|
| 产品名称 | 书名 |
| 一句话卖点 | [核心冲突] + [独特解决方案] + [终极愿景] |
| 目标读者画像 | 年龄/性别/阅读场景/弃书阈值 |
| 核心体验承诺 | 读者读完记得的3个体验 |
| 差异化优势 | 和同类比"我不同在哪" |
| 风险预判 | 诚实面对弱点 |

## 灵魂三问

- Q1 为什么是我来写这本书？不是"我喜欢"——是"我有别人没有的XX"
- Q2 读者为什么非看不可？不是"好看"——是"这本书给XX体验，其他书给不了"
- Q3 核心情绪是什么？只选一个：热血/治愈/悬疑/爽/虐/幽默

## 压力测试判断

| 测试 | 通过条件 |
|:-----|:---------|
| 电梯测试 | 30秒内向陌生人讲清楚 |
| 长篇拉力 | 能支撑200万字以上 |
| 平台适配 | 与目标平台读者画像匹配 |
| 差异化 | 与Top20同类有核心差异 |
