# Phase 1.2：L1 深度展开（新分类）

## 前置条件
六篇 L1 骨架完成（Phase 1），跨域素材就绪。

## 加载参考
先加载 `phases/phase-1.ref.md`。

## 核心原则
逐篇执行，不可并行。每篇完成后对照 ref.md 的必填字段检查是否有缺失。不达标不进下一步。

## 执行步骤
每篇 L1 执行一次深度展开：

1. **字段补齐** — 对照 ref.md 中该维度的必填子字段列表，检查是否有缺失字段
2. **画面锚点** — 每个子字段至少搭配一个"读者能脑中成像"的具体例子（一句场景/一个画面）
3. **素材注入** — 从跨域素材提取相关种子嵌入对应字段
4. `[主角]` 标签检查 — 主角相关内容是否已用 `[主角]` 前缀标注？主角在该维度是否有遗漏的关键信息？

## 检查清单
- [ ] 每篇 ref.md 中列出的必填子字段全部覆盖
- [ ] 每子字段至少1个画面锚点
- [ ] `[主角]` 标签无遗漏
