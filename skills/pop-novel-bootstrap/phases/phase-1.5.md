# Phase 1.5：世界稳定性检验

**目标**：对 L1 设定层进行一致性检查，发现并修复设定漏洞。

**条件**：Phase 1.3 全部交叉关联检查通过。

## 执行步骤

1. **一致性检查**（交叉验证所有 L1 文档）
   - 力量体系是否与世界观底层逻辑一致？
   - 种族设定是否与势力关系一致？
   - 物品体系是否与力量体系/经济体系一致？

2. **逻辑闭环检查**
   - 每个设定可回溯至 L0 PRD？
   - 存在无法解释的矛盾/漏洞？
   - 存在「作者知道但角色不应该知道」的信息暴露？

3. **可扩展性检查**
   - 当前设定限制了未来剧情可能性？
   - 「写死自己」的设定陷阱？
   - 预留了后续卷的新发现空间？

4. **外部一致性验证**（可选）
   - 查阅同类型经典作品的处理方式
   - 确保差异点是有意识的设计

**产出**：
- `00-原始设定/world-stability-check.md`
- 修复方案（如需修改 L1 则标记变更）
