# Phase 1：L1 设定层推演

## 这份文档存在的意义

L1 设定层是 PRD 的落地。它的意义不是"写出六份好看的文档"——是**把核心创意的每一个维度都写得足够具体，让后续的大纲和正文写作时有据可查**。

六份文件的共同根基是：它们必须共同服务于**同一个核心矛盾**。不是六份独立的设定文档，是一份完整世界观的六个切面。

**没有 L1 的后果**：写正文时发现角色没背景可依、世界观前后矛盾、力量体系写到一半不知道怎么升级。

---

## 必须回答的问题（总纲）

六篇 L1 文件必须共同回答这个问题：

> **这个世界的根本矛盾是什么？**

比如在"仙秦诸天"中，根本矛盾是"秩序与混沌的对抗"——01 底层逻辑讲这个矛盾如何体现在世界法则层面，02 表层规则讲普通人如何感知这个矛盾，03 种族讲不同生命体在这个矛盾中的位置，以此类推。

六篇文件可以理解为一个矛盾在六个维度上的展开：

```
01-底层逻辑 → 矛盾的本体（宇宙法则层面）
02-表层规则 → 矛盾的日常（普通人能看到的部分）
03-种族     → 矛盾中的生命体（不同物种如何应对）
04-金手指   → 矛盾的破局者（主角的异类解决方案）
05-势力     → 矛盾的集体表达（组织如何站队）
06-物品     → 矛盾的物质化（什么物品承载了这个矛盾）
```

---

## 执行步骤

依次产出六个文件：

| # | 文件 | 必须回答的问题 |
|:-:|:-----|:--------------|
| 01 | `01-世界观底层逻辑.md` | 这个世界的基本法则是什么？力量从哪里来？规则的限制在哪里？ |
| 02 | `02-世界观表层规则.md` | 普通人眼中的世界长什么样？他们知道什么、不知道什么、误解了什么？ |
| 03 | `03-种族.md` | 每个种族的生存逻辑是什么？他们如何在世界中找到自己的生态位？ |
| 04 | `04-金手指.md` | 如在 Phase 0.4 已完成则跳过 |
| 05 | `05-势力组织.md` | 每个势力想要什么？他们为什么得不到？相互之间的张力关系是什么？ |
| 06 | `06-关键物品和资源.md` | 每个物品为什么存在？谁需要它？稀缺性在哪？ |

---

## 原则

每个 L1 设定在写完之前，问自己一个问题：**这个设定服务 PRD 里的什么爽点？** 如果回答不上来，这个设定就是多余的。
