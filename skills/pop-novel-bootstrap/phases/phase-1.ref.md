## 01 · 世界蓝图

**核心问题**：这个世界的基本面貌是什么？物理/魔法规则如何运转？

### 必填子字段

| 子字段 | 说明 | 字数建议 |
|:-------|:-----|:--------|
| 时空结构 | 世界的地域范围、位面/星域划分、时间流动规则 | ≥300 |
| 核心法则 | 世界运行的底层规则（物理/魔法/天道） | ≥300 |
| 力量来源 | 力量从哪来（灵气/魔力/原能/信仰） | ≥200 |
| 世界基调 | 这个世界的整体氛围（黑暗/热血/神秘/恐惧） | ≥100 |

### 示例：遮天

```
01-世界蓝图.md

## 时空结构
诸天万域格局。北斗星域为核心大星域，包含东荒、西漠、南岭、北原、中州五大区域。
生命禁区散落分布（太初古矿、轮回海、不死山、上苍、葬天岛）。星空古路连接各大星域。
时间流动规则：人界一天、仙界一年。

## 核心法则
大道压制。天地法则会压制超过当前境界的修士，导致古之大帝级强者难以存续太久。
每过数万年一次的"黑暗动乱"——禁区至尊收割众生生命来延续己身。

[主角] 荒古圣体：被天地压制，每突破一个境界需要比常人多走九个小台阶。
  这个设定直接决定了主角的成长节奏——不可能快速升级，必须一个一个台阶走。

## 力量来源
源——天地灵气凝聚的结晶，修士修炼主要靠源。
悟道——通过感悟天地法则获得突破，"道"的境界比灵气积累更重要。

## 世界基调
宏大、苍凉、悲壮。人与天争命，修士在压迫中寻求超脱。
```

### ❌ 坏的写法

```
01-世界蓝图.md

这个世界叫大荒大陆，分为东荒、西荒、南荒、北荒四个区域。
有灵气、有法则。修士可以修炼变强。
```

**为什么坏**：没有具体内容。"修士可以修炼变强"在任何一个修仙世界都成立。没有区分度。

## 02 · 力量体系

**核心问题**：在这个世界里，如何变强？有几个阶段？每个阶段有多大差距？

### 必填子字段

| 子字段 | 说明 | 字数建议 |
|:-------|:-----|:--------|
| 境界总览 | 从入门到巅峰的完整等级链 | ≥200 |
| 核心路径 | 主要修炼/升级方式（功法/冥想/战斗/吞噬） | ≥200 |
| 突破条件 | 每级怎么突破（积累/机缘/领悟/代价） | ≥200 |
| 力量天花板 | 故事结束时能达到的最强级别是什么？ | ≥100 |
| 跨级约束 | 差几级就不可战胜？什么情况可以越级？ | ≥100 |

### 示例：吞噬星空

```
02-力量体系.md

## 境界总览
地球篇：武者→战士→战将→战神→超越战神
宇宙篇：行星级→恒星级→宇宙级→域主→界主→不朽→宇宙之主→宇宙最强者→真神

[主角] 罗峰的晋升路径特殊：前期靠陨墨星传承（突破快），中期靠金角巨兽分身（基因倍数×10），后期靠九劫秘典（打破法则上限）。

## 核心路径
基因原能修炼法——通过引动宇宙能量淬炼身体。
精神念师——少数人拥有的精神力量体系，可与基因原能并行修炼。
本源法则感悟——参悟宇宙本源法则（时间/空间/金/木/水/火/土/风/雷/光）。

## 突破条件
- 学徒→行星级：积累足够原能，精神共鸣突破
- 行星级→恒星级：体内星核壮大
- 每级需本源法则感悟配合，不能只靠能量堆积

## 力量天花板
真神（起源大陆顶端），全书结尾达到。

## 跨级约束
- 同级：五五开
- 越1级：需合理解释（装备/环境/代价）
- 越2级：极特殊情境，不可复用
- 越3级：体系崩了
```

### ❌ 坏的写法

```
02-力量体系.md

修炼分为九段，每段实力比前一段强。
主角是天才，修炼速度很快。
```

**为什么坏**：没有具体等级名、没有突破条件、没有跨级约束。"天才"是标签不是设定。

## 03 · 历史与驱动力

**核心问题**：这个世界之前发生过什么？现在的核心矛盾是什么？

### 必填子字段

| 子字段 | 说明 | 字数建议 |
|:-------|:-----|:--------|
| 纪元/时代划分 | 几个大的历史时期及特征 | ≥150 |
| 前提大事件 | 故事开始前发生的关键事件 | ≥200 |
| 核心矛盾 | 这个世界当前面临的根本性对抗 | ≥200 |

### 示例：深渊主宰

```
03-历史与驱动力.md

## 纪元/时代划分
- 耐瑟帝国时代（远古）：魔法文明巅峰，耐瑟卷轴流传
- 诸神纪元：神祇统治世界，神权高于一切
- 动荡之年/圣者浩劫（当前）：神力风暴，诸神被迫以圣者形态降临凡间

## 前提大事件
动荡之年的神力能量风暴席卷多元宇宙，所有神灵暂时失去神力。
诸神被迫以圣者身份降临凡间，大量神灵陨落，神位出现真空。

## 核心矛盾
秩序vs混沌。圣者浩劫导致权力真空，各方势力争夺神位。
主角索伦在这个混乱中逐步崛起，以守序邪恶的方式走向深渊主宰之路。
```

### ❌ 坏的写法

```
03-历史与驱动力.md

这个世界很久以前很和平，后来发生了大乱。
现在势力很多，互相争斗。
```

## 04 · 物种与天赋

**核心问题**：谁生活在这个世界？什么物种？有什么特殊天赋或血脉？

### 必填子字段

| 子字段 | 说明 | 字数建议 |
|:-------|:-----|:--------|
| 主要种族 | 智慧种族的列表及简要特征 | ≥200 |
| 异兽/怪物 | 非智慧生物的威胁等级和基本生态 | ≥200 |
| 特殊体质/血脉/天赋 | 极少数拥有者才有的天生成就 | ≥150 |

### 示例：遮天

```
04-物种与天赋.md

## 主要种族
- 人族：数量最多，分布在各大星域，是历史的主要创造者
- 太古万族：远古时期统治世界的强大种族，曾与人族大战
- 妖族：万物皆可成妖，从草木到走兽，顶级妖族可与圣地抗衡
- 圣灵：天地孕育的奇异生命，天生强大

## 特殊体质
- 荒古圣体：万古无双的体质，肉身强横。但被天地压制，修炼比常人难九倍
  [主角] 叶凡继承荒古圣体，这决定了他不能像其他人一样快速升级
- 先天道胎：天生亲近大道，悟道速度极快
- 苍天霸体：与荒古圣体并列的顶级体质，对决是万古宿命
```

## 05 · 势力格局

**核心问题**：谁说了算？势力之间的关系如何？

### 必填子字段

| 子字段 | 说明 | 字数建议 |
|:-------|:-----|:--------|
| 主要势力 | 组织/国家/派系列表 | ≥200 |
| 权力分布 | 谁掌控什么资源/地盘 | ≥100 |
| 势力关系 | 各势力间的对抗/合作/中立 | ≥150 |

### 示例：吞噬星空（分两层）

```
05-势力格局.md

## 地球篇
- HR联盟：全球大财阀联合组织，掌控经济命脉
- 极限武馆：全球最大武馆，培养武者
- 雷电武馆：第二大武馆，与极限武馆竞争
- 战神宫：战神级强者的组织

## 宇宙篇
- 虚拟宇宙公司：最大的信息/交易平台
- 巨斧斗武场：以战斗为核心的势力
- 宇宙佣兵联盟：雇佣兵组织
- 宇宙星河银行 / 第一银行：金融势力

[主角] 罗峰先后加入极限武馆→虚拟宇宙公司，成为巨头之一
```

## 06 · 资源与物品

**核心问题**：这个世界的"好东西"有哪些？为什么重要？

### 必填子字段

| 子字段 | 说明 | 字数建议 |
|:-------|:-----|:--------|
| 修炼资源 | 提升力量的消耗品 | ≥150 |
| 关键道具 | 有特殊功能的物品 | ≥150 |
| 货币体系 | 用什么交易？ | ≥50 |

### 示例：遮天

```
06-资源与物品.md

## 修炼资源
- 源：天地灵气结晶，修士修炼必需品。分普通源/异种源/神源/仙源
- 不死神药：可延长寿命、助人突破，如九妙神药、真龙不死药

## 关键道具
- 万物母气鼎（[主角] 叶凡的本命帝兵，以万物母气铸成，可成长）
- 帝兵：大帝级兵器（荒塔、仙钟、绿铜鼎等）
- 仙金：铸造帝兵的材料（龙纹黑金、凰血赤金、永恒蓝金等）
```

## 六件套的切分避坑

| 混淆点 | 正确归属 |
|:-------|:---------|
| 修炼等级和突破条件 | 02-力量体系（不是01-世界蓝图） |
| 历史背景和前提事件 | 03-历史与驱动力（不是01-世界蓝图） |
| 血脉/天赋体质 | 04-物种与天赋（不是02-力量体系，虽然是力量相关的内容） |
| 势力组织的社会规则 | 05-势力格局（不是01-世界蓝图） |
| 金手指 | 不设独立维度，用 `[主角]` 标签在各篇标注 |
