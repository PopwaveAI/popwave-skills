## 三、叙事功能（Narrative Function）

### 角色在故事中做什么

```
□ 情感锚点 — 承载读者对特定情感的投射（安心/心疼/愤怒/…）
□ 冲突放大 — 通过与主角的矛盾推动剧情
□ 信息管道 — 向读者/主角释放关键信息
□ 镜像/对照 — 与主角形成对照，凸显主角的某个面向
□ 催化剂 — 推动主角做出决策，自己不跟进
□ 场面注入 — 提供特定气氛（幽默/紧张/温暖/…）
```
