# Lv3 · 功能角色 — 模板

> 适用：为特定叙事功能服务、出现 1-3 幕的角色
> 颗粒度：3 维 | 不需要弧线，但需要明确其功能

---

## 一、身份（Identity）

| 字段 | 填写 |
|:-----|:------|
| 姓名 | |
| 一句话印象 | "一个[特质]的[身份]，他/她[做了什么/给主角什么]" |

## 二、标签（Tags）

```
[3-5 个关键词，覆盖性格、外貌、说话方式、行为模式]
例如：急性子 · 烟不离手 · 说话带刺但心软 · 穿褪色军大衣 · 敲桌子
```

## 三、驱动与功能（Drive & Function）

| 项目 | 内容 |
|:-----|:------|
| 他/她想要什么 | 在此场景/幕中 |
| 在故事中做什么 | |
| 离开后去哪 | 任务完成后角色去哪里了？（不需要后续追踪） |
