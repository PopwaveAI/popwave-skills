# Lv4 · 主角/核心对立 — 模板

> 适用：故事主角、主要反派、最终对手。贯穿全文的灵魂。
> 颗粒度：Lv3 全部 + 7 维 | 每轮动笔前建议重新审读，确保决策一致性。

---

## 设计目的

> 贯穿全文的灵魂。承载故事核心主题，读者通过这个角色体验整个世界的意义。

## 叙事能力

> 驱动全篇情节走向——角色弧线 = 故事主梗；承载读者全部情感投射；所有关键转折由这个角色的决策触发；是读者的最终认同锚点。

## 三、心理 ← 新增

### 核心性格特质（3 条）

每条格式：
```
特质名：
  表现：在压力下如何显现
  边界：什么情况下该特质被压抑/反转？
  盲点：该特质导致的判断偏差
```

### 决策逻辑

```
日常决策倾向：[谨慎/冲动/直觉/计算/拖延/…]
危机决策倾向：[与日常是否不同？压力下会怎么变？]
决策阈值：[什么事件会打破惯常模式，迫使角色做出非常决策？]
```

### 核心缺陷

```
缺陷：
  对谁/什么事最致命？
  整个故事中改善了多少？（0 → 部分 → 大幅 → 克服）
```

## 四、驱力（四层完整） ← Lv3 升级

| 层级 | 内容 | 故事中何时显露 |
|:-----|:------|:--------------|
| 表层目标 | 角色自己认为想要的 | 开局即显 |
| 深层需要 | 角色真正需要的（与表层可能冲突） | 故事后半才会意识 |
| 黑暗冲动 | 角色最坏情况下可能做的事 | 压力阈值触发时浮现 |
| 恐惧 | 角色最害怕发生的 | 伏笔阶段反复暗示 |

### 驱力冲突

```
哪个驱力层级之间的张力最大？  [如：表层目标与深层需要背道而驰]
这种冲突在故事中以什么形式出现？ [如：角色嘴上说想回家，行为上一直在往前走]
```

## 六、演化 ← 新增

### 弧线阶段（最少 3 个阶段，推荐 4-5）

```
阶段 1：[阶段名]
  起止章节/幕：
  触发事件：
  前心态：
  后心态：
  关键决策（这次选择定义了角色）：
  表象变化 vs 本质变化：

阶段 2：[阶段名]
  ...
```

### 弧线类型（可复选）

```
□ 成长弧  — 从不足到完整（主角经典弧）
□ 堕落弧  — 从光明到黑暗（悲剧/反派弧）
□ 转变弧  — 从 A 到 B 的横向转变（身份/信仰/阵营转换）
□ 揭露弧  — 角色本质不变，但读者逐渐认识其真实面貌
□ 平线弧  — 角色不改变，而是改变周围世界（如完美英雄激励他人）
```

### 不可逆点清单

```
事件 1（约 1/4 处）：
  - 这个事件后角色无法回到原本的生活
  - 故事前 vs 后的心态对比

事件 2（约 1/2 处）：
  - 角色的核心认知被颠覆
  - 旧的驱动力在此失效

事件 3（约 3/4 处）：
  - 最黑暗时刻 / 伪胜利
  - 角色被迫面对最不想面对的事
```
