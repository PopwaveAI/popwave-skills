# T10 文风DNA - 拆解模板

## 核心原则
文风DNA是作者最底层的"写作指纹"——跨章节一致存在的词汇/句式/描写/节奏/叙事偏好。
不是单章的选择，而是全书的稳定模式。每条发现都要能回答"这个特征是否在全书中一致出现"。

---

## 检查清单

### 模块A：叙事哲学（5条核心原则）

```
【核心原则：不是"作者做了什么"，而是"作者为什么这么做"。
 从行为上升到信念层。每条原则必须有跨章验证。】

#### 原则1：[维度名称]

**信念陈述**：[一句话——不是作者做了什么，而是作者为什么这么做]

**证据链**（原文片段或章节引用）：
- CH{X}：{该片段在该维度上的具体表现}
- CH{Y}：{同上，第二个证据}

**跨章验证**：该特征在 CH{..} 和 CH{..} 中一致出现？
- 验证通过 ✅ / 部分一致 ⚠️ / 不一致 ❌

**应用规则**（写作者可直接执行）：
- [可执行规则1]
- [可执行规则2]
- [可执行规则3]
```

**预设维度（可根据赛道调整）**：

| # | 维度 | 核心问题 |
|:-:|:-----|:---------|
| 1 | 信息释放哲学 | 作者对"什么时候给读者什么信息"的根本态度是什么？ |
| 2 | 叙事者姿态哲学 | 叙事者与读者之间的距离关系是什么？全知/受限/客观？ |
| 3 | 情感表达哲学 | 情绪主要靠什么传达？行为/环境/旁白/对话？ |
| 4 | 对话策略哲学 | 对话在小说中扮演什么角色？推进信息/揭示关系/制造冲突？ |
| 5 | 张力-释放哲学 | 节奏控制的底层逻辑是什么？类型切换/嵌套/递进？ |

### 模块B：词汇指纹（定量分析）

```
【核心问题：作者常用的词是一个什么词汇库？高频/低频词有哪些？
 通过原文采样统计，不建议纯手工数，而是通过阅读感知+关键词搜索。】

□ 高频词汇表（全书最常见的20-50个实词）
   - 高频动词：（如：杀/走/看/说/想/感到/觉得）
   - 高频形容词：（如：冰冷/黑暗/强大/可怕）
   - 高频名词：（如：力量/灵魂/深渊/死亡）
   - 高频副词：（如：突然/瞬间/悄然/猛然）

□ 偏好表达模式
   - 四字短语偏好（如：杀气腾腾/血流成河/铺天盖地）
   - 语气词使用频率（如：了/的/啊/吧/吗）
   - 口语化程度（正式/通俗/方言）

□ 禁用/罕见词（作者刻意不用的词）
   - 例如：某些玄幻文不用"爱情"二字，用"缘分/羁绊"

□ 名台词/金句风格
   - 本书的金句有什么共同特征？（短促有力/哲理/押韵？）
   - 示例：[列出3-5条]
```

### 模块C：句式节奏（定量分析）

```
【核心问题：作者的句子是长句驱动还是短句驱动？
 场景/战斗/对话的句式有何不同？】

□ 句长分布
   - 平均句长（字数）
   - 短句（<15字）占比
   - 长句（>50字）占比
   - 极端句（>100字）出现频率

□ 场景句式偏好
   场景类型 | 句长特征 | 标点偏好 | 段落长度
   --------|---------|---------|---------
   战斗/动作 | 短句密集 | 感叹号/句号 | 短段落
   设定说明 | 长句为主 | 逗号/分号 | 中长段落
   内心独白 | 中句为主 | 省略号/问号 | 中段落
   对话 | 极短 | 引号 | 极短段落

□ 段落结构
   - 平均段长（行数）
   - 超短段（1行）占比
   - 超长段（>20行）占比
   - 对话段 vs 叙述段的比例

□ 排比/重复模式
   - 三句式排比频率
   - 递进式重复频率
   - 对仗/对称结构偏好
```

### 模块D：描写策略（定性+定量）

```
【核心问题：作者怎么描写场景/人物/战斗？
 多感官还是单一感官？细节密度高还是低？】

□ 感官调用偏好
   感官 | 使用频率 | 占描写比例 | 典型场景
   ----|---------|-----------|---------
   视觉 | 高/中/低 | % | 场景/人物/战斗
   听觉 | 高/中/低 | % | 战斗/环境
   触觉 | 高/中/低 | % | 战斗/亲密
   嗅觉 | 高/中/低 | % | 环境
   味觉 | 高/中/低 | % | 罕见

□ 描写密度
   - 平均每100字有多少个"描写词"（形容词/副词/动词修饰）
   - 高密度描写的场景（如：战斗/景观/人物外貌）
   - 低密度描写的场景（如：对话/内心独白/过渡段）

□ 人物外貌描写策略
   - 首次出场描写的详细度（详细/简洁/一笔带过）
   - 是否重复描写（每次出场都描写/只第一次/只在关键场景）
   - 描写角度（旁观者视角/镜子/他人评论）

□ 环境描写策略
   - 每个新场景是否必有环境交代？
   - 环境与情绪的关联（风=悲伤/雨=困境/阳光=希望等符号化使用）
   - 环境描写的长度（占章节比例%）

□ 战斗描写策略
   - 动作细节度（招式名/体感/策略）
   - 结果描写方式（致命伤/参数变化/画面定格）
   - 常见战斗词汇使用频率（刺/砍/斩/闪/格挡/轰）
```

### 模块E：对话DNA（定性+定量）

```
【核心问题：作者的对话风格是什么？
 口语化还是书面化？信息密度高还是低？】

□ 对话标签使用
   - "说/道"使用频率
   - "某某说" vs 直接对话的比例
   - 动作替代对话标签的频率（"xxx推开门"代替"xxx说"）

□ 对话长度分布
   - 平均每轮对话行数
   - 单人最长连续对话行数
   - 对话在总章节中的占比（%）

□ 对话功能分布
   功能类型 | 占比 | 典型章节
   --------|-----|---------
   推进信息 | % | 
   揭示性格 | % | 
   制造冲突 | % | 
   情感表达 | % | 
   闲聊填充 | % | → 标记为潜在问题

□ 角色语言区别度
   - 不同角色的说话方式是否可区分？
   - 角色A的典型句式/词汇 vs 角色B的典型句式/词汇
   - 身份/性格/情绪如何通过语言风格体现
```

---

### 模块F：节奏指纹（定量分析）

```
【核心问题：作者怎么控制章节级的阅读节奏？
 场景切换频率、时间跳跃、信息密度变化规律。】

□ 章节长度规律
   - 平均每章字数
   - 最长章节 vs 最短章节
   - 章节长度变异系数（稳定/波动大）

□ 场景切换频率
   - 平均每章场景切换次数
   - 单场景最长持续章节数
   - 场景切换标记方式（***/空行/章节分隔）

□ 时间跳跃粒度
   - 平均每次跳跃的时间跨度（几小时/几天/几个月/几年）
   - 时间流速与剧情密度关系（重要情节慢速/过渡快速）
   - 时间标记方式（章节名/叙事者说明/暗线暗示）

□ 信息密度曲线
   - 高信息密度章节的特征
   - 低信息密度章节的特征（注水/放松/过渡）
   - 信息密度与张力曲线的匹配程度

□ 钩子密度
   - 章末钩子出现频率（每X章出现一次）
   - 章末钩子类型偏好（悬念/警告/预告/冲突新升级）
   - 中段钩子/章内钩子的使用频率
```

### 模块G：跨章一致性验证

```
【核心原则：识别出的DNA特征必须经得起跨章检验。
 一条特征如果只在某一章出现，那是选择不是DNA。】

□ 词汇指纹一致性
   - 前100章 vs 后100章：核心高频词是否一致？
   - 中期 vs 后期：描写用词是否发生了演变？

□ 句式节奏一致性
   - 第一卷 vs 最后一卷：句长分布是否发生变化？
   - 如果变化，是故意的风格演变还是写作习惯漂移？

□ 描写策略一致性
   - 作者在第1章和第1000章描写场景的方式是否一致？
   - 战斗描写公式是跨卷一致还是逐卷演进？

□ 对话风格一致性
   - 主角的说话方式是否随成长发生变化？（角色成长=好的演进）
   - 旁白的语言风格在全书中是否稳定？

□ 可靠的DNA vs 演变的选择
   - ✅ 稳定DNA：跨全书一致出现的特征
   - ⚠️ 阶段DNA：只在某几卷出现的特征（可作为"阶段性标志"）
   - ❌ 单章选择：只出现1-2次，不能算DNA
```

## 参考：赛道常见文风DNA特征

```
玄幻修仙赛道常见DNA：
- 词汇偏好：境界名/功法名/丹药名等专有名词丰富
- 句式：修炼突破场面长句密集，战斗场面短句
- 描写：战斗以"拳/掌/剑/气"等动作为主，感官以视觉+触觉为主
- 对话：信息型对话为主（解释功法/交代背景）

都市职场赛道常见DNA：
- 词汇偏好：行业术语/品牌名/公司名
- 句式：长中句为主，口语化程度中等
- 描写：重视细节真实感（吃喝穿用）
- 对话：功能性+关系型对话混合

悬疑推理赛道常见DNA：
- 词汇偏好：模糊词（似乎/仿佛/可能）高频
- 句式：短句制造紧张感，长句用于推理说明
- 描写：环境描写高频（氛围营造）
- 信息控制：关键信息刻意隐藏，误导词高频

言情耽美赛道常见DNA：
- 词汇偏好：情感词汇高频（思念/心动/温柔/疼）
- 句式：内心独白段落较长
- 描写：外貌描写/微表情/肢体语言优先
- 对话：情感表达+关系推进为主
```
