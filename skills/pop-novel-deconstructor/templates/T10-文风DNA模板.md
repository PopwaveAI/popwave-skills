# T10 文风DNA - 拆解模板

## 核心原则
文风DNA拆解不是为了"研究"，而是为了让另一个Agent能写出这种风格。
**每项产出必须是可执行的写前规则。** 不是数据堆砌，不是调研报告。

## 方法论
按**场景类型**取样分析。每种场景选取3-5个代表性章节，从中提取：
1. **写前规则**（做/不做）— 最核心产出，Agent直接执行
2. **锚定章片段** — 标注"这段好在哪里、怎么复用"
3. **量化参考** — 关键指标（句长/比喻密度/对话占比等），够校准Agent即可，不需要全量统计

## 检查清单

### □ 赛道定位

```
赛道 | 该赛道下本书的文风DNA是什么类型？
特征标签 | 3-5个词概括（如：数据化叙事/黑暗现实主义/力量至上/快速升级）
关键选书理由 | 为什么选这本书作为风格参考？
```

### □ 场景类型1：战斗/动作场面 ⭐ 必做

```
选取3-5个战斗章节，提取写前规则：

【写前规则】
做：
- [规则1，如：战斗前先给全景，再聚焦一个角色/细节]
- [规则2，如：使用"动作+结果+推测"三段式]
- [规则3，如：数据/数值信息嵌入动作描述中]
不做：
- [规则1，如：避免省略策略思考]
- [规则2，如：避免连续长句纯动作堆砌]

【锚定章片段】
> [原文片段200-300字]
标签：[句式特征/描写偏好/节奏特征/数据使用方式]

【量化参考】
- 平均句长：X字
- 动作词密度：X/千字
- 比喻频率：X/千字
- 战斗前准备行数：X行（占比X%）
- 战后处理方式：战利品评估/面板确认/叙事间奏
```

### □ 场景类型2：对话/交流场面 ⭐ 必做

```

- [规则1，如：对话以"XX道"为引导，15~17字/段]
- [规则2，如：情绪强烈处使用"！……"组合]
- [规则3，如：关键对话后紧跟叙述描写]
不做：
- [规则1，如：避免长篇独白]
- [规则2，如：避免纯对话无叙述]

【锚定章片段】
> [原文片段200-300字]
标签：[对话引导词偏好/语气词/情绪表达方式]

【量化参考】
- 对话占章节比例：X%
- 平均每段对话字数：X字
- 引导词偏好（道 vs 说 vs 其他占比）
- "！"在对话中频率：X/千字
```

### □ 场景类型6：高潮/爽点场面 ⭐ 必做


## 模板使用说明

1. **优先度**：▶ 战斗/对话/设定/高潮为必做 ◀ 情感/过渡可选
2. **锚定章数量**：每种场景类型3-5个，够提取规则即可，不需要遍历全书
3. **写前规则是核心产出**：Agent拿到后直接执行，不需要再读量化报告
4. **量化参考的目的是校准**：不是学术统计，够让Agent知道"大概什么范围"就行
5. **按赛道调整**：都市文可以不拆战斗，悬疑文必须拆设定释放
