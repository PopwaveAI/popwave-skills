# T3 角色系统设计 - 拆解模板

> 📌 **消费管线**: 本模板产出可直接供 `bookstrap·Phase 0.6(拆书成果融合)` + `pop-novel-writer(Design阶段角色状态)` + `pop-novel-plot(反派节奏)` 消费。

## 核心原则

角色是小说最重要的元素。当前模板从金手指天赋升级为**角色系统设计**——不只是记录"主角有什么能力"，而是拆解"作者用了什么手法让读者关心这些角色"。

每个维度必须回答：**"这个手法好在哪、我怎么用"**。

[ ] | [...] | [...] | [...] | [...] | [...]

叙事功能分类:
- 引导型: 提供知识/方向/训练（导师/前辈）
- 伴侣型: 情感锚点/分担压力（恋人/挚友）
- 对照型: 通过对比凸显主角特质（镜像/对手/衬托）
- 工具型: 一次性推动剧情（⚠️ 标记为潜在问题）
- 逗趣型: 调节节奏/制造轻松时刻（搞笑担当）
```

```

#### 3.3.2 反派动机合理性

```
反派 | 动机 | 合理性评估
-----|------|----------
[ ] | [ ] | [充分/勉强/不合理]

动机类型:
- 生存驱动（被迫/自卫型）
- 欲望驱动（权力/财富/力量）
- 信念驱动（正义/秩序/变革）

- 混沌驱动（单纯坏/无动机）

优秀反派的特征（如果有）:
- 站在反派视角看，他的选择是否合理？
- 反派和主角是否存在"镜像"关系？
- 反派下线后，压迫感是否由新反派接替？
```

```

- 实力碾压: [有/无] → 维持手段: [等级压制/不可伤/群体碾压]
- 智谋压制: [有/无] → 维持手段: [走一步看三步/信息差/设局]
- 情感绑架: [有/无] → 维持手段: [人质/牵连/道德困境]
- 倒计时: [有/无] → 维持手段: [限时威胁/催化剂]
- 不可战胜性: [有/无] → 维持手段: [规则级碾压/概念级存在]

压迫感不下降的写法:
- 主力反派下线前，是否有新反派顶上来？
- 反派的实力增长是否与主角同步或更快？
- 反派是否有"底牌"在即将被击败时翻出？
```
