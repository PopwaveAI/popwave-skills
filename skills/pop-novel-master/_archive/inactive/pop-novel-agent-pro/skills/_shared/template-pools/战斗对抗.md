# 战斗对抗场景模板

> 适用于：任何形式的对抗冲突（物理战斗/规则对抗/智力博弈）
> 核心原则：**不写招式描写，只写决策后果**

---

## 段落结构

```
┌─ 段1 · 态势建立（1-2段） ────────────────────────┐
│  功能：让读者知道"这一轮谁占优"                      │
│  内容：环境/位置/双方状态（不是"他冲过来"——是"距离    │
│        还剩三步时，他左肩下沉了"）                    │
│  字数：100-150字                                     │
└────────────────────────────────────────────────────┘
                      ↓
┌─ 段2 · 核心回合（1-3轮交替） ──────────────────────┐
│  功能：对抗的核心信息交换                            │
│  每轮结构：                                          │
│    ① 甲方动作（1句）                                │
│    ② 乙方反应/反制（1句）                           │
│    ③ 新态势（1句）                                  │
│  节奏：每轮≤3句，不拉长                              │
└────────────────────────────────────────────────────┘
                      ↓
┌─ 段3 · 转折点（1段） ─────────────────────────────┐
│  功能：改变对抗走向的新信息/新条件                    │
│  内容：底牌揭示 / 第三方介入 / 代价出现 / 规则变化    │
│  位置：第2-3轮之间（不是最后）                        │
│  写法：用事件本身揭示，不用"突然"或"没想到"           │
└────────────────────────────────────────────────────┘
                      ↓
┌─ 段4 · 后果（1段） ───────────────────────────────┐
│  功能：胜负不是重点，读卷后的情绪状态才是             │
│  内容：谁处于什么新位置 + 这个位置对下一章的影响      │
│  注意：不写"他赢了"/"他输了"——写新的局面              │
└────────────────────────────────────────────────────┘
```

---

## 关键约束

- **不写招式名**——不靠"火焰掌"、"天雷诀"凑字数，靠决策推进
- **不写单招描写**——"一剑劈出"不行，"他赌对方会闪左路"行
- **不少于2个回合**——单回合秒杀不是对抗，是碾压（除非幕纲设计为碾压）
- **每回合不超过3句话**——超过就是在注水

---

## 节奏标记

```
态势：中速 → 回合：快-快/快-中-快 → 转折：慢（聚焦） → 后果：中速
总字数：600-1200字（视幕纲等级调整）
```
