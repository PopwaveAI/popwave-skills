# Phase 2：L2 卷级展开

**目标**：从元设定层展开到具体卷级设计，含爽点设计。

**条件**：世界稳定性检验通过。

## 执行步骤

1. **全书规划**
   - 确定总卷数（3-8卷）
   - 每卷核心冲突/主题
   - 全书情绪曲线

2. **卷01 展开**
   - 地理：开局地图（主要场景/势力范围）
   - 势力：开局涉入的势力/组织
   - 角色：主要角色表（主角/重要配角/反派）
   - 剧情：核心冲突链/悬念设置
   - 装备：开局可获得的关键物品/能力
   - 爽点设计 → `02-大纲/卷01/爽点设计.md`

3. **爽点密度标准（番茄参考）**
   - 前3章：第一个小爽点出现
   - 前10章：≥3个小爽点+1个中爽点
   - 前50章：爽点不可中断超过5章
   - 大爽点间隔不超过20章

**产出**：
- `02-大纲/卷01/地理.md`
- `02-大纲/卷01/势力.md`
- `02-大纲/卷01/角色.md`
- `02-大纲/卷01/剧情.md`
- `02-大纲/卷01/装备.md`
- `02-大纲/卷01/爽点设计.md`
- `02-大纲/卷N/...`
