# 本卷登场人物清单模板

> 产出路径: `设计/幕/act-XX-人物.md`
> 下游消费: pop-novel-writer → chXXX-登场人物卡.md

---

## 模板说明

本文件定义「一卷」视角下的角色全景。不是章级（章级是"登场人物卡"），而是**卷级**——哪些角色在本卷登场？他们的初始状态是什么？在本卷结束后变成什么？

**为什么要卷级人物清单？**
- 章纲的"登场人物卡"需要知道"本卷还有谁会出场"——不能靠章纲 Agent 临时编
- 避免角色突然出现或突然消失（"角色遗忘症"）
- 确保角色在本卷中的出场节奏和弧线是可追溯的
