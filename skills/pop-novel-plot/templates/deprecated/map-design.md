## 二、关键地点清单

### 地点名

| 维度 | 说明 |
|:-----|:-----|
| 类型 | 城镇/野外/地下城/建筑 |
| 一级视觉 | 第一眼看到什么 |
| 叙事功能 | 在这个地点会发生的剧情类型 |
| 空间情绪 | 压抑/开阔/神秘/破败/... |
| 位置关系 | 与哪些地点相邻、移动时间 |
| 危险等级 | 安全/关注/警戒/高危 |
| 物资补给 | 能买到什么、不能买到什么 |
| 开放章段 | 从 chX 到 chX |

（重复以上结构，每关键地点一个条目）
