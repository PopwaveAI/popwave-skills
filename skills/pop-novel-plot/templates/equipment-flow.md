# 本卷装备/资源变化模板

> 产出路径: `设计/幕/act-XX-装备.md`
> 下游消费: pop-novel-writer → 正文中装备描写的参考基线

---

## 模板说明

定义「一卷」视角下主角装备和资源的获得、消耗、升级过程。

**为什么需要卷级装备变化表？**
- 防止主角"装备通货膨胀"（获得太多/消耗太少 → 失去成长实感）
- 防止装备遗忘（获得了但后面再也没用过）
- 为数值体系的消耗检验提供数据来源

## 五、装备设计红线

- [ ] 每件获得的装备至少使用 2 次（防止装备遗忘）
- [ ] 获取 ≠ 使用之间的间隔 ≥ 1 章（给读者消化的时间）
- [ ] 本卷获得的装备总数 ≤ 3 件（防止通货膨胀）
- [ ] 每件装备有限制/代价（防止装备万能化）
- [ ] 核心装备的叙事意义不重复（每件装备讲不同的故事）
