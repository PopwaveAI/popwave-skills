## 详细定义

### MK-01：

| 维度 | 说明 |
|:-----|:-----|
| 成就标准 | 什么事件标志着这个里程碑达成？ |
| 预期章节段 | 大致出现在第几章到第几章 |
| 等级要求 | 主角大概需要什么水平才能过这个里程碑 |
| 朝向方向 | M1推进 / M2推进 / M3推进 |
| 如果不达成会怎样 | 跳过这个里程碑会损失什么？ |
| 前置依赖 | 需要先完成哪个 MK 或获得什么资源？ |
| 涉及的角色 | 哪些角色在这个里程碑中出场或起作用？ |
