## 一、四级爽点总表

| 级别 | 是什么 | 密度 | 判定器 | 负责 |
|:-----|:------|:----:|:------|:----:|
| 小 | 干脆利落的斩杀/升级/打脸/梗 | 每章 ≥ 3 | 读者 3 秒读完，反应"嗯爽" | chapter-design |
| 中 | 有铺垫的释放。一条子线收束 | 每章 ≥ 1 | 读者想"终于"或"我就知道" | chapter-design |
| **大** | **剧情层面的重大转折点。格局重画** | 间隔 ≤ 5 章 | **"故事进入新阶段了"** | **plot** |
| **特大** | **全书承诺兑现。作者从第 1 章就在铺垫的那个答案** | 每卷 ≥ 1 | **"原来前面那么多章就是为了这一章"** | **plot** |

## 二、大爽点

### 2.1 定义

> 大爽点 = 一条或多条主线的释放改变了故事的格局。释放后，下一章的语境已经不一样了——地图被重画了。

**判定器**：释放之后，读者能不能清晰地感觉到"从这一章起，故事的语境变了"？

### 2.2 案例（10 个）

| # | 赛道 | 铺垫（一幕的长度） | 释放（1 章） |
|:-:|:-----|:-------------------|:------------|
| 1 | 朝堂 | 主角花了半卷在六部安插自己人，拉拢清流，扳倒一个尚書作探路石 | 早朝上把首辅十八年前的军械贪污旧账摊在金殿——人证物证齐全。首辅跪下的瞬间满朝文武自动分成两列 |
| 2 | 都市商战 | 主角花了一卷低调收购对手放弃了的小公司，不对抗不回应 | 这些小公司突然串成完整产业链，掐住对手三条产品线。主角在财报发布当天开发布会："你们自己会枯死" |
| 3 | 玄幻 | 主角用整卷在遗迹中收集十二块残片——每块都在低语，但他听不懂 | 第十二块嵌入神龛的瞬间他不聋了——听懂了那些低语是在喊他的名字。不是他在找神，是神一直在找他 |
| 4 | 末日废土 | 主角花了一卷从幸存者聚落收集"禁区内部"碎片信息 | 拼出完整地图后带队进去——禁区最深处是完好的旧世界城市，住着三百万在培养皿里长大的"复苏人类" |
| 5 | 战争文 | 第1-8章观察敌军、9-12章发现必经峡谷太窄骑兵展不开、13-15章埋伏三千人 | 第16章——二十万敌军进峡谷，火起，石头封出口。不是围歼是收割。敌军统帅的头挂在瞭望塔上 |
| 6 | 科技文 | 主角从冷聚变实验被嘲笑→拿到第一笔投资→失败三次 | 第四次点火——反应堆稳定运行七十二秒。不是欢呼——有人在哭，有人在笑，主角蹲在地上擦屏幕"是不是数据错了" |
| 7 | 悬疑 | 全书查了六起看似无关的案子，到了第 200 章发现共用一个受害者家属 | 主角翻到第七本案卷——7 不是凶手留的数字。是七个家属在不同城市同一天悼念同一个人——他们互相不知道。那个共同者就是第 7 起受害者 |
| 8 | 异能/恐怖 | 全书主角与"不可名状的存在"对抗，以为它是入侵者 | 走进它的记忆——不是入侵，是三千年前人类主动开了门，邀请它进来。它没有恶意，它在等一个定义。不可名状 = 还没被命名 |
| 9 | 商战/复仇 | 主角被前公司坑走成果后在出租屋里重头开始，忍着三年不发声 | 三年后的发布会上一个专利都没展示——展示了三年前那封被前公司法务拦截的邮件。"我没偷。是你们拦了我证明自己的机会。" |
| 10 | 穿越历史 | 主角穿越成末代皇子，所有人都等他亡国 | 他没有打仗——用一卷时间织了一张贸易网。敌国发现自己的粮食、铁器、马匹全部依赖主角的商路。兵临城下那天，城门开了——但不是投降。是敌国将军下马单膝跪地："我们要的是活着。" |

### 2.3 Canvas 联动设计方法

**核心逻辑**：Canvas 矩阵中某条线的 payoff_level=大 → 那一章就是大爽点章。

```
plot Step 2 设计 Canvas 时逐线标记 payoff_level：
  - 空 = 铺垫/建设/埋伏笔
  - 小 = 干脆利落的斩杀/升级/打脸/梗植入
  - 中 = 这条线在这里有一次可感知的释放（信息揭示/关系转折）
  - 大 = ★ 这条线在这章爆发——前面的蓄力全部释放。故事从这章起进入新阶段

填完 Canvas 后扫描：
  → 所有 payoff_level=大 的格子 → 大爽点章号
  → 验证间隔 ≤ 5 章
  → 大爽点前 2-3 章标记蓄力
```

**大爽点间隔保证**：
- 扫描本幕所有大爽点章号
- 如果相邻大爽点间隔 > 5 章 → 把中间某条线的释放提前
- 大爽点前 2-3 章 Accumlation（叠加压缩蓄力）

## 三、特大爽点

### 3.1 定义

> 特大爽点 = 全卷乃至全书的终极释放。不是一条线收束，不是格局变化——是**作者从第 1 章就开始铺垫的那个承诺，终于兑现了**。

**判定器**：读者读完这一章后会不会在心里把前面整本重新翻一遍？会不会觉得"前面那么多章就是为了这一章"？

### 3.2 案例（6 个）

| # | 赛道 | 全书铺垫 | 释放 | 读者反应 |
|:-:|:-----|:--------|:-----|:--------|
| 11 | 玄幻 | 全书主角寻找"天道的答案"——神为什么抛弃人间。以为是力量是境界是权柄 | 最后一道门——没有神，只有一个空王座。他站了三息坐了下去。不是登基——是他明白了：神从来没来过，一直是人在自己撑 | "原来写了 300 章就是让他发现从来没人会来救" |
| 12 | 战争 | 全书主角从逃兵→统帅，灭国→流亡→重建。以为复仇在驱动他 | 敌国皇城前围了三个月，收到降书后下令退兵。"我不要他的国家。我要他承认他杀的第一个人不是将军，是一个被砍断腿后跪在原地等死的农民" | 战争文最炸裂的一章没有战斗，只有一封信和一百万军队的沉默 |
| 13 | 朝堂 | 全文与首辅斗了 150 章——每一步都是反制→铺垫→收网 | 首辅在天牢求见："你到底想要什么？"主角："我不想坐你的位置。我只想证明做好臣也可以赢" | 整本书的价值观翻过来了——前 150 章的斗争不是复仇是证明 |
| 14 | 科技 | 全文从冷聚变实验被嘲笑→拿到千万投资→失败三次 | 第四次点火——反应堆稳定七十二秒。主角蹲在地上用手擦屏幕问"是不是数据错了"，其实没擦 | "人类触碰了恒星的秘密。而主角只想知道自己有没有看错" |
| 15 | 悬疑 | 第 1 章就埋的伏笔：凶手留下一个数字 7。全书查了六起看似无关的案子 | 第七本案卷不是凶手留的。是六个家属在同一天悼念同一个人——他们互相不知道，但主角知道了那个共同的人是谁 | 读者开始翻前面两百章重看，每个数字每个日期都有解释 |
| 16 | 异能 | 全书与"不可名状的存在"对抗，以为它是入侵者 | 走进它的记忆——三千年前人类主动开了门邀请它进来。它没恶意，在等人类给它一个定义。不可名状 = 还没被命名。主角给了它第一个名字 | "如果你从来没被定义过——你会是恐惧还是一切？" |

### 3.3 Canvas 联动设计方法

**特大爽点 = Canvas 中多条线 payoff_level=大 汇聚到同一章或连续两章 + 全卷情绪极值。**

```
plot 填完 Canvas 后：
1. 扫描所有 payoff_level=大 的格子
2. 找多线汇聚章（≥ 2 条线同为 大 的章）→ 特大爽点候选
3. 幕约 1/2-2/3 处选一个做幕中爆发
4. 幕末最后一章做幕末收束
5. 每卷至少 1 个特大爽点
```

### 3.4 责任归属

| plot 负责 | plot 不负责 |
|:----------|:-----------|
| Canvas 逐线标注 payoff_level | 小爽点+中爽点具体事件（归 chapter-design） |
| Canvas 汇总 payoff_map 交给 design | 具体事件内容 |
| 大爽点间隔 ≤ 5 章 / 特大 ≥ 1 个/卷 | |
| 大爽点前蓄力标记 / 特大爽点多线汇聚设计 | |

## 四、Canvas payoff_map 交付物

```
payoff_map:
  ch1:  total=0 → ⚠️ 自行制造 ≥ 3 小爽点 + ≥ 1 中爽点
  ch3:  total=1 (lines=["M1:中"]) → ✅ 供给 1 中爽点，按线号定位
  ch7:  total=1 (lines=["M1:大"]) → ★ 大爽点章，全幕最高密度事件链
  ch11: total=1 (lines=["M3:大"]) → ★ 大爽点章
  ch14: total=3 (lines=["M1:大","M2:大","M3:中"]) → ★★ 特大爽点候选（2线大汇聚）
```

**消费规则**：
- total=0 → chapter-design 自行制造
- 某线=大 → 大爽点章
- ≥2 线=大 → 特大爽点章
