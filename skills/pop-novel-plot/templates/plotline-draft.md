
图例：■ = 推进 / ■🔫 = 设伏 / ■💥 = 回收 / ☐ = 未出现
**M3 必须拆为两行**：`M3-等级`（职业等级/阶位变化）+ `M3-面板`（属性加点/专长解锁/新能力）
**列数 = 实际卷章数**。22章就画22列，不合并为"ch1-N"分组。

## 四、风险判断

- [ ] 线数不过多（M1≥3子线 + M2 + M3双行 + S线≥2条，每章活跃≤5条）
- [ ] 无孤立线（每条 S 线与至少 1 条 M 线有交叉，标注交叉章号）
- [ ] 交叉节奏有疏有密（有4线汇聚的高潮章 + 2-3线推进的过渡章）
- [ ] M3 逐章摘要无空白行（22章 = 22行，不允许"逐步升级"分组）
- [ ] 每2章至少1次战斗场景（M2行 ■ 间隔不超过1个☐）
- [ ] 每2章至少1次成长反馈（M3行 ■ 间隔不超过1个☐）
