## 一、罗列候选线

### M1 — 世界危机（必选，主线）

| 维度 | 说明 |
|:-----|:-----|
| 一句话定义 | 世界级压力源，回答"世界发生了什么" |
| 在本卷的角色 | |
| 预期频次 | 每2章 / 每3章 |
| 本卷推进幅度 | 危机从什么程度升级到什么程度 |

#### M1-子线①

| 维度 | 说明 |

（M1-② / M1-③ 同理）

### M2 — 主角成长（必选，成长线）

| 维度 | 说明 |
|:-----|:-----|
| 一句话定义 | 主角在本卷"从谁变成谁" |
| 预期频次 | 每章都有 |
| 本卷成长目标 | 从{等级/段位}到{等级/段位} |
| 关键成长节点 | chX: {突破} / chY: {质变} |

### M3 — 外部压力/剧情推进（必选，主线推进）

| 维度 | 说明 |
|:-----|:-----|
| 一句话定义 | 如果 M1 是世界发生了什么，M3 就是"主角被逼做什么" |
| 预期频次 | 每章都有 |
| 本卷推进目标 | 从{起点}到{终点} |
| 关键决策点 | chX: {选择} / chY: {代价} |
