## 一、节奏自检（7+3+3）

### 基础 7 项

- [ ] 连续无爽点章节 ≤ 3 章
- [ ] 连续无新信息章节 ≤ 2 章
- [ ] 高强度章间距 ≥ 3 章
- [ ] 情绪低谷长度 ≤ 2 章
- [ ] 终极爽点预留了整卷铺垫
- [ ] 大爽点前 2-3 章用叠加压缩蓄力了
- [ ] 幕的情绪弧线有"起"

### 核心三要素

- [ ] 无连续 2 章同时缺(战斗/恐怖/成长)
- [ ] 无连续 2 章 fun_level 为零爽点
- [ ] 事件数/多样性/字数基线正常

### 契诃夫枪

- [ ] 已设伏未回收的枪中，无超过 10 章仍未回收的

### 信息释放

- [ ] 连续无新信息章节 ≤ 2 章
- [ ] P0 级信息全部在本幕内分配
- [ ] 第1章新概念 ≤ 2 个
- [ ] 每个 info_release 标注了 source_doc
