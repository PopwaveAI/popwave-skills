## 一、节奏自检（7+3+3）

### 基础 7 项

- [ ] 无中爽点空白章 — 每章 ≥ 1 中爽点
- [ ] 连续无新信息章节 ≤ 2 章
- [ ] 高强度章间距 ≥ 3 章
- [ ] 情绪低谷长度 ≤ 2 章
- [ ] 特大爽点每卷 ≥ 1 个（全卷情绪极值 / ≥2 线大汇聚）

### 核心三要素

- [ ] 无连续 3 章无大或特大爽点 — 大爽点间隔 ≤ 5 章
- [ ] 特大爽点每卷 ≥ 1 个
- [ ] 事件数/多样性/字数基线正常

### 契诃夫枪

- [ ] 已设伏未回收的枪中，无超过 10 章仍未回收的

### 信息释放

- [ ] 连续无新信息章节 ≤ 2 章
- [ ] P0 级信息全部在本幕内分配
- [ ] 第1章新概念 ≤ 2 个
- [ ] 每个 info_release 标注了 source_doc

## 二、番茄平台标准校准

> 统一番茄标准。首卷前 2 章是黄金窗口——错过即弃。

| 节点 | 本幕实际 | 番茄要求 | 通过？ |
|:-----|:--------:|:--------|:------:|
| 第1章钩子 | | **必须有** | |
| 核心卖点亮相 | | **第 1 章内** | |
| 第一次战斗/重大矛盾 | | **第 2 章内** | |
| 第一个副本闭环 | | 第15章内 | |
| 最大纯铺垫区间 | | ≤ 1章 | |

> 第 1 章 = 核心卖点 + 钩子 → 读者知道"这本书是什么"。
> 第 2 章 = 第一次战斗/重大矛盾 → 读者尝到爽感、知道"这本书怎么爽"。
> 违规 = **P0 退回**，重新排布前 2 章事件。
