# 卷N 卷纲

> 卷纲是战略契约和剧情矿场，不是剧情细纲。卷纲只定战略、边界、燃料和功能位，具体剧情交给幕纲环节。

## 篇幅重心

| 模块 | 建议占比 |
| --- | ---: |
| 本卷定位/主爽点/读者期待 | 15% |
| 本卷边界 | 15-20% |
| 本书涌现燃料池 | 25% |
| 外部燃料筛选 | 25% |
| 可选剧情燃料池 | 与上两项合计约50% |
| 幕序列功能位 | 10%以内 |

## 一、本卷定位

- 卷定位（一句话：这卷在全书中干什么）
- 主爽点（1-2个）
- 读者期待（读者读完这卷应该得到什么）
- 剧情brief（本卷大致走向，3-5句话框选方向，不写beat）
