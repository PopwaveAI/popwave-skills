# 风格契约模板

> 产出路径: Phase 1 内存产物（不落盘），供 Phase 2/3 消费
> 生产方式: pop-novel-prose-render v1.0+
> 输入: 文风DNA 档案 + 锚定章特征提炼

---

## 模板说明

风格契约 = 为「本章这场戏」提取的风格执行规则。不是泛泛的「模仿某作者」，而是针对本章场景类型的具体规则。

**关键原则**：
- 不是规则列表——规则必然丢信息。是语感对齐的产物
- 少即是多——选 3-5 条最适合本章场景的规则
- 每条规则必须有"怎么做"和"怎么检查"

---

## 模板正文

```markdown
## 风格契约 — chXXX · 场景类型: {combat/dialogue/discovery}

> 源: {文风DNA 档案路径} + {锚定章路径}

### 1. 叙事哲学
- **叙事者在多远的距离**: (近·紧贴感官 / 中·观察但不解释 / 远·事后回忆)
- **本章的叙事者声音**: (克制/冷峻/戏谑/诗意/锋利/...)
- **情感传递方式**: (靠动作 / 靠环境 / 靠对话潜文本 / 靠内心独白——选一种为主)

### 2. 句式与节奏
- **句子长度偏好**: (短句为主 10-15字 / 中句 15-25字 / 长短交替)
- **段落切换频率**: (快·每2-3句换段 / 中·每4-6句/慢·长段叙事)
- **本章的情绪节拍与句子节奏的对应**:
  - 情绪压低时 → 句子更短、段间距更大
  - 情绪推进时 → 句子略长、动作连写
  - 情绪高潮时 → 短句爆发、单独成段

### 3. 描写策略
- **感官优先级**: (视觉 > 听觉 > 触觉 > 嗅觉 —— 按本章场景调整顺序)
- **环境描写的功能**: (制造氛围 / 呼应情绪 / 提供信息 / 三者都要——选一种为主)
- **战斗描写策略**: (动作写实 / 特效渲染 / 心理博弈 / 慢镜头拉伸 —— 本章只用一种)
- **禁止的描写**: (本章不写什么——如"不写反派内心""不写血腥细节""不写环境抒情")

### 4. 对话策略
- **引导词风格**: (极简·"XX道" / 加动作·"XX道"+ 动作 / 无引导词·直接引语)
- **对话密度**: (对话占本章 %)
- **对话功能**: (推进信息 / 塑造关系 / 控制节奏 / 释放情绪 —— 选一种为主)
- **本章禁止的对话方式**: (如"禁止长篇大论""禁止角色念设定""禁止两个人都说一样语气")

### 5. 本章红线（从文风DNA 提取）
- 不:
- 不:
- 不:
```

---

## 使用说明

1. **Phase 1 产出此契约** → 不落盘，存在 context 里
2. **Phase 2 逐事件渲染时** → 每个事件先看契约 → 再写句子
3. **Phase 3 验证时** → 逐条对照契约检查 → 标偏差

> 契约是针对「本章场景类型」定制的。下一章场景类型变了 → 重新锚定。
