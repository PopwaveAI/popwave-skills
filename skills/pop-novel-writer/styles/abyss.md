## 一、战斗

### 【战斗·暗杀·一对多】第8章 杀人夜

索伦单挑一个小帮派。潜行→暗杀→暴露→正面格杀。短句起势，动作+数据+推测三拍。

```
吱嘎。
一个鹰钩鼻的恶棍走了出来，骂骂咧咧道："臭婊子！"
"让你用嘴也不肯，要不是看你还能赚些钱，非得把你好好教训一下不可！"
他转过栏杆便进入了一个小巷，随后掏出家伙打算尿尿。

"操！"
"早晚要你好看！"

鹰钩鼻的恶棍刚刚说完话，突然感觉后颈有些发凉，还等不及他把武器掏出来，
一只手臂便捂住了他的嘴巴，随即黑暗中的匕首划过寒光，一瞬间切开了他的喉咙。
鲜血喷出去了很远，咕噜咕噜地往外冒，背后的身影卡住他的脖子，一直到尸体停止抽搐才放开。
这时索伦的身影才在阴影中浮现，面无表情宛若死神一般。
一旦进入战斗状态，他便没有丝毫的怜悯之心。

他拖着尸体进入了小巷内，血迹铺满了地面，幸好黑暗掩盖了一切。

"阴影躲藏成功！"
"背刺发动！"
"成功发动背刺*2，对目标造成16点伤害！"
"目标死亡！"
"抽取目标灵魂能量，获得经验值30点。"

连续杀掉了两个人，里面的血腥味已经非常浓郁。
索伦拿起了另外一把匕首，双手同时持有武器走了出来。

蹬蹬！
索伦的身影跳起在墙壁上一接力，便是落在了屋顶上。
高达20点的超凡人类敏捷让他在行走时没有丝毫的声音。

"啊！"
凄厉的惨叫声回荡。
索伦的身影宛若老鹰般从天而降，朝着落后的一个人扑了过去，匕首从他的后背一路切到了腰部，
随即另外一只手划过一道寒光，从对方的下巴缝隙刺入了头骨内。
没有丝毫的犹豫。
索伦抬腿一蹬，将尸体踢飞了出去，接着甩出手中的匕首，当场命中了第二个敌人的心口。

"是你！……"
为首的战士低喝了一声，拔出长剑便是劈落下来。
铛铛铛！
索伦手中的匕首划过妖异的轨迹，连续接住了对方的劈砍，沉声道："你们不该把主意打到薇薇安的身上。"
"所以只能送你们全部下地狱了！"

加里斯是一个警觉的家伙！
自从冒险受伤归来以后，他睡觉身上都穿着皮甲，坚韧牛皮制作的皮甲匕首是没有办法穿透的。
只能攻击他身上没有皮甲覆盖的地方才行。

鲜血渗出。
加里斯伸手捂住腰部，感觉那里一片发麻，脸色异常难看："有毒！"

对付冰霜翼龙最重要的不是造成伤害，而是让它不能够继续飞行。

索伦的脚步划出一道奇异的轨迹，突然从正面转到了敌人的背后，随即刺入了加里斯的颈骨，
接着翻转匕首转了一圈。
扑通！
加里斯的尸体倒地。

索伦提起一具尸体，便是从大门撞了进去。

嗦！
弓弩发射的声音传来。
索伦提着的尸体震了一下，他瞬间松开尸体扑了过去，居然蹬腿凌空在墙壁上横着助跑了三步，
这才落地扑向了眼前的敌人。

啪嗒！
索伦脚尖轻轻一勾，一把弓弩拿在了手中，他飞速地摆弄了一下，一脚踢开房门便是朝着前方一射。
一道弩矢射出。

"索……索伦！……"
科尔是一个阴沉的中年男子，不过此刻他捂着手臂满脸恐惧，不断地哀求道："不要杀我！"
"我以后都听你的。"
一道寒光。
索伦面无表情地割开了他的喉咙，随即皱着眉头看着自己满身的血迹，开始翻找房间内的财物。
```

### 【战斗·经典三段式】第148章 战冰霜翼龙

短句起势 + 动作+数据+推测 + 策略逻辑。深渊主宰战斗的标准模板。

```
一片冰雪飞溅。
索伦一震胳膊抖掉了身上的寒冰，随即便是拔出来标枪朝着眼前的冰霜翼龙投掷了过去。
寒冰吐息给他带来了二十多点的冻伤，如果不是有弱等寒冷抗性的专长在，
恐怕他还得僵硬好一会儿才能恢复知觉。
对付冰霜翼龙最重要的不是造成伤害，而是让它不能够继续飞行，
索伦虽然有攻击对方头部颈脖要害的机会，可还是将标枪投向了冰霜翼龙的翅膀。
```

### 【战斗·死战·越级】第16章 亡命搏杀

索伦重伤状态下反杀深渊蛇魔。升级→属性分配→反杀→战利品。数据面板和决策逻辑穿插在动作中。

```
索伦咬牙从破烂的衣服上撕下来一块布，然后缠在了腿部的伤口上勒死，这样可以暂时压制他身上的伤势，
也可以轻微地减轻活动时的痛楚。

"将所有经验分配到盗贼等级！"
索伦冷静地拔出了长剑，开始分配自己的经验值。

一排文字数据浮现在眼前：
"盗贼职业提升到4级。"
"获得24（敏捷值20+（智力18－10）*0.5）点技能点，生命值提高10点（职业生命6点+（体质18－10）*0.5）。"
"获得1点自由属性。"

他将所有的技能点都分配到了潜行上，一口气将潜行技能提升到了102点。
身影渐渐在黑暗中越来越淡。

至于那宝贵的一点自由属性，索伦咬了咬牙还是分配在了体质上，
20点的超凡敏捷跟21点没有太大的差距，除非是一口气提升到了25点以上，
进而觉醒出来【阴影领域】的传奇能力。

他看到了不远处的蛇魔，对方庞大的身躯变得狼狈无比，地面上到处都是炸裂的血肉鳞片，
蛇魔超过600磅的体型在通道内完全没有办法闪避，爆裂陷阱几乎是直接在它的身体旁炸开。
如果不是蛇魔拥有恶魔类生命强大的坚韧豁免，恐怕当场就会被炸成满地的碎肉。

"嘶！"
蛇类生命的嘶吼声响起，蛇魔一瞬间发现了索伦，将仇恨的目光投了过来。

铛！
索伦轻易地侧身避开，随即反手刺向了敌人的胸口，那里已经有很多的鳞片被炸裂。
噗嗤！一股滚烫的鲜血溅射出来。

"滋滋滋！"
一部分鲜血溅射在了索伦身上，很快便会有一种灼烧般的感觉。
他脸色微变，飞快地拉开距离，凝重道："酸性血液？"

蛇魔回光返照般地疯狂攻击，可是依旧无法命中眼前的敌人。
体能上的消耗让它力量下降，身体上的伤势严重影响它的敏捷，
进入濒死状态下它的战斗还不到原来的三成。

"结束了！"

短暂的休息。
索伦在体能恢复了一部分后，开始打扫自己的战利品。
深渊恶魔的身上很少携带金钱，不过却可以获得一些超凡装备。

一排数据浮现：
"物品种类：蛇魔长剑+1。
物品等级：【一阶超凡武器】
物品说明：使用深渊火岩锻造出来的长剑，拥有很强的劈砍穿刺效果，额外增加1点火焰灼烧伤害。
装备要求：14点力量以上。
特殊效果：对中小体型生物视为【巨剑】种类，力量需求+2。"

力量值不够！
索伦苦笑着将蛇魔长剑放了下来，毕竟是超过600磅体重敌人使用的武器，几乎都相当于人类使用的巨剑了。
他现在的小身板体质是足够，体能耐力也不错，可是力量不够的话，强行用这把武器还不如拿着普通的匕首。

正常的长剑重量只有两三公斤左右，这把剑最少有十五公斤。
两把就是三十公斤。

索伦将两把蛇魔长剑背了起来，三十公斤的重量让他感觉人都沉了一下，
越发觉得自己现在12点的力量值不够用了。
不过这个战利品是必须要带出去的。
一阶超凡武器在市场上的价格基本上都有100枚金德勒以上。
```

## 二、对话·交涉·欺骗

### 【对话·暗语·欺骗】第19章 血裔

索伦遭遇吸血鬼血裔。非战斗解决——通过暗语和欺骗脱身。对话简洁（15字级），叙述者大量介入解释。

```
索伦看到了一个女人。
一个穿着暴露容貌妖艳的女人，有着一头淡金色的长发，裸露的皮肤异常苍白。

索伦的脚步声很轻。
但是那个妖艳的女子还是轻易地发现了他，她抬起头来露出幽绿色的眸子，
伸出舌头轻舔殷红的嘴唇，玩味道："看起来我今天运气很好。"

这是一位血裔。
吸血鬼培养的一种仆人，实力远比低阶的血奴更强。

短暂的死寂。
妖艳女子胸前的年轻男人似乎已经被情欲所笼罩，根本没有觉察到附近的气氛已经有些压抑沉重。

索伦丝毫没有理会那个可怜的猎物，他心中在计算的是自己对抗血裔时的胜算，
对方的敏捷极有可能比自己还高，力量远超于普通人的范围，并且拥有超强的闪避直觉。
论实力一点都不会比前面的深渊蛇魔弱小。

那么！
唯一的办法就只有一个。

索伦微微地俯下身，将左手握拳食指中指交错落在了自己的锁骨上，缓缓道："阴影笼罩世界！"
"黑暗中的假面使者注视着我们！"
"荣耀。"
"归于无形的主宰！"

妖艳女子戏谑的笑意逐渐收敛，她的眉头微微蹙起，深深地打量着眼前的索伦，沉声道：
"黑夜主宰我们的命运！"
"始祖的意志指引我们的方向。"
"夜之女士。"
"庇佑她的信众！"

成功了！
索伦的心中浮现一丝劫后余生的喜悦，不过表情却没有丝毫的变化。
他轻轻点头后退了一步，表达自己没有丝毫恶意，同时俯身道："尊敬的女士。"
"愿黑夜永远眷顾你，请安静享用你的晚餐。"
"没有人会打扰你！"

说完。
他转过身朝着另外一个方向离去。

自始至终。
索伦都没有看那个可怜的男子一眼，一直到离开了乱葬岗后，他这才虚脱般地长舒一口气。
他刚刚说的话是阴影假面的暗语。

当然！
索伦并不是阴影假面的成员，更不是阴影之主的信徒。
他其实是在骗人。
不过这应该是最好的办法了。
解决问题的唯一方式并非只有战斗，如果是只懂得战斗的二逼，索伦根本就活不到传奇领域。
在这个'圣者遍地走，巫妖多如狗'的世界。
比自身强大的存在不知道有多少，甚至遇到神祇的可能性也很高！
```

## 三、发现·探索

### 【探索·陷阱·遗迹】第13章 万象无常牌

盗贼探索地下藏宝洞。陷阱拆除的解密节奏——紧张→拆除→收获。多细节铺陈。

```
短短十几米的距离。
索伦便耗费了半个多小时的时间，陷阱最让人头疼的不是如何解除，而是你无法确定它会隐藏在哪里。
尤其是这种一击毙命的陷阱，很多时候你都是提着脑袋在冒险，哪怕是一丝的不小心都有可能让你毙命。

"发现了！"
15点的感知让索伦对危险的觉察很敏锐，凭借着曾经的经验，他小心翼翼地用匕首从墙角撬开一块不规则的石头，
果然在其中发现了一个复杂的发射装置。

"一、二、三、四、五、六！……"
索伦不由擦了一下冷汗，暗自庆幸自己足够小心，这居然是多重连弩陷阱，
如果不小心中招了直接就得被射成筛子。
除非他身上披着重甲，这才有可能挡住这种强劲的弩矢。
如果他没有记错这种强弩陷阱的单个杀伤力是12点左右，不命中要害的话，六支弩矢也足够在一瞬间秒杀他。

咔嚓！
索伦用手指卡主扳机，接着趴在地上用匕首敲了一下铁杠，这种陷阱用得都是特殊钢材，
经过简单的附魔处理，一般几十年内都不会生锈出错。

啪啪啪。
六只弩矢射了出来，尽根没入另外一边的墙壁。

一排数据浮现：
"解除陷阱成功。"
"你的盗贼技艺获得提升，获得210点职业经验。"

"呜！"
一阵阴风吹过，索伦突然感觉后背发凉，毫不犹豫地朝着后方劈出武器，
只看见一道朦胧的阴影浮现，狰狞的鬼脸上露出凄惨的笑容，怪叫一声消失在了原地。
"幽灵？！"
索伦小心翼翼地靠在了墙壁上，飞速地摸出来了圣水。
幽灵是灵体类生命，常规非附魔武器无法伤害到它们，哪怕是站着让索伦砍一百下也屁事没有。

"桀桀！"
索伦的耳膜一疼，心中并没有丝毫的恐惧，反手就将圣水砸了过去。
哐当！
圣水洒满了幽灵的全身，顿时响起一阵凄厉的惨叫声，幽灵的身影飞速消溶，化做一团阴影消失。

"幸好早有准备。"

一排数据浮现：
"圣水命中幽灵，造成60点伤害。"
"目标死亡！"
"抽取目标灵魂能量，获得经验值120点。"

吱嘎！
索伦推开了密室的石门。

正前方是一个石头桌子，上面摆放着许多零零碎碎的东西，比如说不知名的头骨，断裂的手指骨，
风干的蝙蝠尸体，以及一个黑色的木盒子。

"似乎是铁木？"
索伦看了看四周，将盒子托在手心，远离自己的脑袋，紧接着用匕首撬开。

咔嚓！
索伦撬开了眼前的盒子，里面放着的是三张卡牌。
仿佛是跟周围的一切都格格不入，那三张卡牌竟然是如此的精美，
秘银花边镶嵌着细碎的钻石，中央是一道宛若漩涡般的魔纹……
```

## 四、危机·悬念

### 【危机·预感·酝酿】第10章 超凡觉醒（剥皮事件段）

教团活动升级的危机感。叙事者介入替读者提问，信息的三层递进释放：现象→疑问→留白。

```
码头区游走在灰色地带的人，最近明显开始限制出手的次数。
既然发生了剥皮事件，那么事情没可能就这样结束，根据灰色地带对于暗黑领域的了解，
那些疯子不可能就这样收手的。
后面恐怕还会有血腥事件升级！

当索伦听到这个消息时，心中也是不好的震动了一下，随即便意识到了一点。
这是混乱时代即将来临的征兆！
在诸神黄昏之战爆发前，最先动手的便是地狱恶魔，它们的追随者掀起一场场的血腥杀戮，
其中最常见的便是剥皮事件。

索伦的身影站在码头区的角落中，眼神异常的凝重，喃喃道："琥珀城！"
"这里应该是南方公国的地界。"
"到底发生了什么？"

索伦皱着眉头竭力回忆着，可惜依旧没有足够的线索，当时连续发生的大事实在太多了。
上古红龙都跑出来了一头！
许多人都好像疯了一样打算跑去屠龙，哪里还有人会关注南方公国一个城池发生的事情。

"剥皮事件开始了。"
索伦看了看码头区人来人往的货物，仿佛预见到了即将来临的动乱，喃喃道：
"看起来应该要去一趟隐蔽藏宝洞了。"
力量。
这是动乱时代最重要的东西。
因为只有力量才可以保证自己活下来，也只有力量才可以保护自己珍惜的一切！

诸神即将来临。
属于圣者的时代已经拉开序幕！
```

## 五、后果·情感·恢复

### 【后果·安葬·回忆】第3章 墓园区

希斯之死。悲伤不写悲伤——写泥泞的路、写饥饿的野狗、写"希斯应该被安葬在那里"。情感通过环境和行动折射。

```
雨后的道路一片泥泞。
贫民区里面永远都是一片烂泥路，一脚踩下去便会溅起泥浆污垢，偶尔也会有小巷角落里面的粪便垃圾被雨水冲刷出来，
空气中永远存在着一股恶臭味。
密密麻麻的低矮屋棚里面，可以看到许多表情麻木的人正在忙碌。

索伦抱着老狗已经冰凉的尸体走在泥泞恶臭的道路上，薇薇安神色黯然地牵着他的手，默默地跟在后面。
希斯已经死了！
那条从她出生开始便一只跟在身边的老狗，终于是走到了生命的尽头。

它有这样的资格！

墓园不在贫民区。

生活在这里的绝大部分人都处于饥饿线上。
他们想吃肉。
如果不是吃人肉会堕入地狱深渊，恐怕有些饥饿的流浪汉会做一些令人发指的行为！
```

## 六、世界观·设定释放

```
灵魂就好像是在播种，总能收获到一些粮食。
恶魔不懂得像魔鬼那样精细耕作，从更优秀的其他种族内获取灵魂，
所以深渊军团的炮灰还是来自于那些低等生物。
怯魔就像是另一种的狗头人，总是喜欢一窝蜂的乱窜，
拥有一个实力最强大的首领，害怕强大的敌人但却喜欢欺负弱小的敌人。
最后，在无底深渊优胜劣汰的残酷法则下，渐渐成长为一个个能够踏上战场的炮灰。
```

### 【设定·先果后因】第85章（成熟期风格）

```
魔鬼们终于被彻底赶回了地狱。
但是在经历了这场前所未有的浩劫后，物质位面也同样是伤痕累累。
幸存下来的人们拖着疲惫的身躯回到了已经化作废墟的家园，即便是侍奉诸神的神殿如今也是十不存三。
接下来很长的一段时间，所有人都必须想办法重建家园，而随着人口的剧烈减少，
诸神也必须暂时放下斗争恢复元气。
许多神灵的教会已经被打残了，一部分的神灵甚至面临着神格降级的危险，
接下来近百年的时间他们必须想办法保证自己信徒的数目，否则有可能跌落一个神格等级。
```
