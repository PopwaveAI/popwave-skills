# 文风参考 · abyss（v1.0）

> 以《深渊主宰》（诸生浮屠）为样本提取的文风参考。西幻·黑暗写实。
> 在 `project.yaml` 中配置 `writing_style: abyss` 启用。
>
> **特征**：感官密集型环境描写、对话节制、情感浓度高、残酷现实基调。

---

## 一、锚定章参考模式

> 实际产出时替换为参考书对应场景的原文片段。

### [战斗] — 痛感驱动型

**微观质感**：
- 句长：短句（8-20字），急促
- 描写：高密度，写血写伤写痛感，不数据化
- 对话：轻度引导
- POV：零距离，紧贴主角体感
- 段长：短段（1-3句）

### [对话] — 节制引导型

**微观质感**：
- 引导词密度：轻度，主要通过内容和语境分辨
- 独白比例：对话为主，独白仅短促反应
- 推进维度：情绪+关系优先，信息次之

### [悬疑/氛围] — 感官堆叠型

**微观质感**：
- 感官密度：高——视觉+触觉+嗅觉至少3个维度
- 信息释放：通过现实压迫自然流露
- 留白度：技巧性留白——日常中埋藏伏笔

## 二、叙事策略

### 2.1 时间流速

```
[x] 正常流速——1章≈半天到一天
```

时间处理：顺序叙事

### 2.2 信息释放模式

```
[x] 探索发现型——通过主角经历残酷现实逐步揭示
[x] 埋伏笔型——先给现象（有人想绑架妹妹），隔几章揭示幕后黑手
```

信息密度：中（每章 3-4 个），以现实压迫推进，不堆设定

参考锚定章：悬疑/氛围场景 → 信息释放节奏

### 2.3 叙事声音侵入度

```
[x] 零侵入——纯故事，没有任何叙事者声音
```

### 2.4 开篇模式

```
[x] 场景铺陈——先给环境/氛围再推进，用感官描写定调
```

> 典型开篇：一个感官点定调整个场景（"屋外传来噼里啪啦的雨声"）。

### 2.5 收束模式

```
[x] 悬念截断——在高潮/对话中间截断
[x] 画面静止——用一个定格的画面/意象收尾
```

### 2.6 情绪节奏模式

```
[x] 震荡型——紧张→松弛→紧张→松弛高频交替
```

> 通过叙事对比制造节奏：温暖互动（松弛）→ 外界威胁（紧张）。

### 2.7 余白密度

```
[x] 技巧性留白——日常描写中埋藏信息，留给读者自己拼出全貌
```

参考锚定章：氛围场景 → 留白和暗示

## 三、技法偏好

### 3.1 对话/叙述比

```
[x] 叙述为主：对话占 < 35%
```

> 对话非常节制，大部分信息通过叙述和动作传递。

### 3.2 心理-动作比

```
[x] 动作驱动——写做不写想，通过行为理解角色
```

参考锚定章：情感场景 → 动作代替内心OS展现情绪

### 3.3 过渡处理

```
[x] 空行分隔——用"---"或空行分段
```

### 3.4 金句倾向

```
[x] 偶尔——每 2-3 章 1 个
```

金句类型偏好：意象型

### 3.5 数字使用习惯

```
[x] 模糊表达——"一点点""很多次""很小"
```

> 西幻风格使用模糊表达，保持"吟游诗人叙事"的质感。

## 四、红线清单

### 4.1 通用红线

```
□ 否定句结构 → 0容忍
□ "他感到/他意识到/他觉得/他评估"类前缀 → 0容忍
```

### 4.2 本风格特有红线

```
□ 超过 60 字的句子 → 标记退回
□ 章节无感官描写（至少2个维度：视觉+触觉/嗅觉）→ 标记
□ 西幻名词首次出现不带上下文解释 → 标记
□ 连续 3 段纯对话 → 标记（需要叙事穿插）
□ 连续 3 段无触觉/嗅觉描写 → 标记（风格特征缺失）
□ 西幻题材出现现代口语网络用词 → 0容忍
```

### 4.3 红线触发后的动作

```
[x] 标记项 ≥ 3 → 退回
[x] 西幻题材现代用词红线 → 立即退回
```
