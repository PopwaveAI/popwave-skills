## 一、锚定章原文参考

> 以下原文摘自《深渊主宰》第 1 章，作为本条文风的具体实例。

### [氛围·贫民区夜雨]

**来源章节**：第 1 章

```
屋外传来噼里啪啦的雨声。
薇薇安小心翼翼地将矮脚凳子放在了灶台前，随即弯下腰从旁边的米袋里面抓起了
两把糙米。袋子里面的粮食已经不多了，哥哥还是上个月才从码头偷到了一块银饰，
这才换来这一袋子的糙米。即便她这半个月来已经精打细算了很多次，可是里面
越来越少的粮食还是让她感到一丝恐慌。

没有食物了。
最多再过几天他们就要饿肚子了！

薇薇安眼睛通红地望了一眼破旧房屋内的小床，铺着发霉味道的被褥上可以看到一个
昏迷中的年轻人。哥哥到今天还是没有醒来，那位曾经愿意帮助他们的牧师姐姐似乎
也放弃希望，毕竟费尔主教并不愿意费尽心力去救一位小偷。
```

**微观质感标注**：
- 句长：中短句（10-25字），急促中带有停顿
- 描写密度：**高**——听觉（雨声）+ 视觉（发霉的被褥）+ 嗅觉（霉味）+ 触觉（湿气）
- 对话：无——纯叙述，通过动作展现情绪
- POV：**零距离**——紧贴薇薇安的视野和认知
- 段长：中短（2-4句），雨声开段独立
- 修辞：克制——无比喻，平实叙述

### [情感·兄妹]

**来源章节**：第 1 章

```
"哥哥！……"
"你一定要醒过来啊！……薇薇安很想你！……"
"不要离开我！……"

一滴滴晶莹的泪珠滑落，薇薇安坐在灶台边轻轻地抽泣着，过了一会儿用小手擦了擦
脸颊，被熏得黑漆漆脸上浮现两道白痕。家里面原本还有一些盐，可是上个礼拜已经
吃完了，她想要学哥哥去偷一些，可是差点被那肥胖的店老板抓住，老板娘拿起扫帚
在她的后背上抽了一下，哪怕是现在背上依旧是一片青紫色，稍微碰到便会感觉
火辣辣的疼。
```

**微观质感标注**：
- 引导词密度：**零引导词**——纯对白推进
- 独白比例：几乎零——通过动作和眼泪展现情绪，不写"她想……"
- 推进维度：**情绪优先**——三句话都是情感释放
- 动作代替内心OS：扫帚的后背/青紫色/火辣辣的疼——用身体感受代替心理描写

### [悬疑·深夜脚步声]

**来源章节**：第 1 章

```
屋外传来一阵脚步声，似乎在门口停留了一下，隐约传来说话声。

原本老态龙钟的黑黄色老狗突然睁开了眼睛，一瞬间宛若饿狼凶悍，口中发出凶狠地
呜声，朝着外面大声地吠了一下。

"该死！"
"早晚打杀了你这条老狗炖汤！"

外面传来一个男人气急败坏地声音，随即脚步声渐渐地远去。

薇薇安惊恐的脸色稍微平静了一点点，伸出小小的手掌摸了摸老狗的头，低声道：
"希斯。"
"如果他敢进来你就咬他！"
"哥哥现在昏迷不醒，只有你能够保护我了。"
```

**微观质感标注**：
- 感官密度：高——**听觉为主**（脚步声/说话声/狗吠）+ 视觉（老狗睁眼）
- 信息释放：**埋伏笔型**——给现象（门外有人），给危机（被盯上了），不给解释（是谁？为什么？）
- 留白度：**技巧性留白**——不写门外的人是谁、有什么意图，留下悬念
- 情绪节奏：**震荡型**——平静（雨声）→ 紧张（脚步声）→ 爆发（狗吠）→ 松弛（脚步声远去）→ 克制恐惧（喂粥）

## 二、叙事策略

### 2.1 时间流速

```
[x] 正常流速——1章≈半天到一天
```

时间处理：顺序叙事

### 2.2 信息释放模式

```
[x] 探索发现型——通过主角经历残酷现实逐步揭示
[x] 埋伏笔型——先给现象（有人想绑架妹妹），隔几章揭示幕后黑手
```

信息密度：中（每章 3-4 个），以现实压迫推进，不堆设定

参考锚定章：悬疑/氛围场景 → 信息释放节奏

### 2.3 叙事声音侵入度

```
[x] 零侵入——纯故事，没有任何叙事者声音
```

### 2.4 开篇模式

```
[x] 场景铺陈——先给环境/氛围再推进，用感官描写定调
```

> 典型开篇：一个感官点定调整个场景（"屋外传来噼里啪啦的雨声"）。

### 2.5 收束模式

```
[x] 悬念截断——在高潮/对话中间截断
[x] 画面静止——用一个定格的画面/意象收尾
```

### 2.6 情绪节奏模式

```
[x] 震荡型——紧张→松弛→紧张→松弛高频交替
```

> 通过叙事对比制造节奏：温暖互动（松弛）→ 外界威胁（紧张）。

### 2.7 余白密度

```
[x] 技巧性留白——日常描写中埋藏信息，留给读者自己拼出全貌
```

参考锚定章：氛围场景 → 留白和暗示

## 三、技法偏好

### 3.1 对话/叙述比

```
[x] 叙述为主：对话占 < 35%
```

> 对话非常节制，大部分信息通过叙述和动作传递。

### 3.2 心理-动作比

```
[x] 动作驱动——写做不写想，通过行为理解角色
```

参考锚定章：情感场景 → 动作代替内心OS展现情绪

### 3.3 过渡处理

```
[x] 空行分隔——用"---"或空行分段
```

### 3.4 金句倾向

```
[x] 偶尔——每 2-3 章 1 个
```

金句类型偏好：意象型

### 3.5 数字使用习惯

```
[x] 模糊表达——"一点点""很多次""很小"
```

> 西幻风格使用模糊表达，保持"吟游诗人叙事"的质感。

## 四、红线清单

### 4.1 通用红线

```
□ 否定句结构 → 0容忍
□ "他感到/他意识到/他觉得/他评估"类前缀 → 0容忍
```

### 4.2 本风格特有红线

```
□ 超过 60 字的句子 → 标记退回
□ 章节无感官描写（至少2个维度：视觉+触觉/嗅觉）→ 标记
□ 西幻名词首次出现不带上下文解释 → 标记
□ 连续 3 段纯对话 → 标记（需要叙事穿插）
□ 连续 3 段无触觉/嗅觉描写 → 标记（风格特征缺失）
□ 西幻题材出现现代口语网络用词 → 0容忍
```

### 4.3 红线触发后的动作

```
[x] 标记项 ≥ 3 → 退回
[x] 西幻题材现代用词红线 → 立即退回
```
