# 风格注入包 · abyss（v1.0）

> 以《深渊主宰》（诸生浮屠）为样本提取的文风。西幻·DND 游戏化，厚重质感的场景描写、第一人称及第三人称双视角。
> 在 `project.yaml` 中配置 `writing_style: abyss` 启用。
>
> **特征**：黑暗写实基调、感官密集型环境描写（湿冷/霉味/雨夜）、信息通过残酷现实自然释放而不是对话揭示、情感浓度高（尤其是兄妹/弱势群体的描写）。

---

## 第1层 · 微观质感

### 1.1 句长分布

```
[x] 中句均衡：平均 15-30 字，长短句自然交替
```

**句长变化模式**：

```
[x] 周期型——长→短→长→短有规律交替，长句铺氛围，短句给冲击
```

### 1.2 描写密度与感官偏好

```
[x] 高密度——五感堆叠，环境描写频繁
```

**感官偏好**：

```
[x] 视觉优先——极重光影/颜色/黑暗中的微弱光线
[x] 触觉优先——湿冷/疼痛/温度变化频繁出现
[x] 嗅觉/味觉——霉味/血腥味/食物气味高频出现
```

> 诸生浮屠的风格特点：每一个场景都至少有 3 个感官维度的描写——视觉（你能看到什么）、触觉（你感受到什么）、嗅觉（你闻到什么）。

### 1.3 对话风格

```
[x] 轻度引导——偶用"他说/她说"，主要通过内容和语境分辨说话人
```

**独白 vs 对白比例**：

```
[x] 对话为主——独白仅用于短促内心反应（1-2句）
```

> 角色很少大段独白推演，而是通过行为和简短的对话来展现决策。

### 1.4 POV 距离

```
[x] 零距离——完全贴死在主角大脑里。读者获得的信息=主角获得的信息。
```

**POV 切换频率**：

```
[x] 从不切换——全书锁死一个视角
```

### 1.5 段长与分段模式

```
[x] 中段——每段 2-5 句
```

**分段模式**：

```
[x] 长短交替——长段感官铺陈后接短段动作/对话
```

### 1.6 修辞偏好

```
[x] 克制——比喻只在"不用就说不清楚"时使用
```

**修辞类型偏好**：

```
[x] 比喻/拟人——主要用于描写环境和怪兽
[x] 无偏好——用什么取决于场景
```

### 1.7 词汇层级

```
[x] 平实——中性词汇，西幻名词使用考究但日常叙述平实
```

**术语密度**：

```
[x] 低密度——术语首次出现附带上下文解释
[x] 高密度（特殊：西幻专有名词需要大量出现——职业/法术/种族/神灵名称）
```

> 西幻题材特有的矛盾：专有名词多（费尔主教/哥拉斯猎犬/琥珀城/索伦），但每个都通过上下文自然解释。

## 第2层 · 叙事策略

### 2.1 时间流速

```
[x] 正常流速——1章≈半天到一天
```

**时间处理**：

```
[x] 顺序叙事——完全按时间顺序
```

### 2.2 信息释放模式

```
[x] 探索发现型——信息通过主角经历残酷现实逐步揭示
[x] 埋伏笔型——先给现象（有人想绑架妹妹），隔几章再揭示（幕后黑手）
```

**信息密度**：

```
[x] 中密度——每章 3-4 个，以现实压迫感推进，不堆设定
```

### 2.3 叙事声音侵入度

```
[x] 零侵入——纯故事，没有任何叙事者声音
```

### 2.4 开篇模式

```
[x] 场景铺陈——先给环境/氛围再推进，用感官描写定调
```

> 典型开篇："屋外传来噼里啪啦的雨声。"——一个感官点定调整个场景。

### 2.5 收束模式

```
[x] 悬念截断——在高潮/对话中间截断
[x] 画面静止——用一个定格的画面/意象收尾
```

### 2.6 情绪节奏模式

```
[x] 震荡型——紧张→松弛→紧张→松弛高频交替
```

> 通过叙事对比制造节奏：温暖的兄妹互动（松弛）→ 外界的威胁（紧张）→ 老狗的警觉（紧张升级）→ 安静喂粥（松弛）。

### 2.7 余白密度

```
[x] 技巧性留白——关键信息故意不写，让读者主动猜测
```

> 诸生浮屠擅长在看似日常的描写中埋藏信息（"米袋快空了""上个月的银饰"）留给读者自己拼出全貌。

## 第3层 · 技法偏好

### 3.1 对话/叙述比

```
[x] 叙述为主：对话占 < 35%
```

> 深渊主宰的对话非常节制，大部分信息通过叙述和动作传递。

### 3.2 心理-动作比

```
[x] 动作驱动——写做不写想，通过行为理解角色
```

> 哪怕是小女孩薇薇安的内心也是通过动作和眼泪展现的（"坚强地擦去眼角的泪水""顾不得双眼酸痛"），而不是写"她想……"。

### 3.3 环境-剧情耦合度

```
[x] 强耦合——环境是叙事的参与者（大雨=困境加剧/夜晚=危险增加/火光=希望）
```

### 3.4 过渡处理

```
[x] 空行分隔——用"---"或空行分段
```

### 3.5 感叹号/问号密度

```
[x] 克制——极端情绪时才用
```

### 3.6 金句倾向

```
[x] 偶尔——每 2-3 章 1 个
```

**金句类型偏好**：

```
[x] 意象型——用意象做情绪锚定（"路边的火光""雨中的黑暗"）
```

### 3.7 数字使用习惯

```
[x] 模糊表达——"一点点""很多次""很小"
```

> 西幻风格下通常使用模糊表达而非精确数字，保持"吟游诗人叙事"的质感。

## 第4层 · 红线清单

### 4.1 句式红线

```
□ 否定句结构（不是A而是B / 不是A不是B是一种C / 以"不是"开段）→ 0容忍
□ "他感到/他意识到/他觉得/他评估"类前缀 → 0容忍
□ 超过 60 字的句子 → 标记退回
□ 章节无感官描写（至少维度 2 个：视觉+触觉/嗅觉） → 标记
□ 西幻名词首次出现不带上下文解释 → 标记
```

### 4.2 段落红线

```
□ 连续 3 段纯对话 → 标记（深渊主宰需要叙事穿插）
□ 连续 3 段无触觉/嗅觉描写 → 标记（风格特征缺失）
□ 单段超过 8 句 → 标记
```

### 4.3 技法红线

```
□ "过渡段"（不推进剧情也不推进情绪的段落）→ 0容忍
□ 对话引导词连续 5 句不换 → 标记
□ 连续 3 章无环境描写 → 标记（风格特色缺失）
□ 西幻题材出现现代口语网络用词 → 0容忍
□ 角色行为与西幻世界设定不一致的现代感 → 标记
```

### 4.4 红线触发后的动作

```
[x] 标记项 ≥ 3 → 退回
[x] 任何西幻红线（第4.3条的后两项）→ 立即退回
```

## 场景类型 vs 写法映射

| 场景类型 | 句长 | 描写密度 | 段长 | 节奏 | 特殊要求 |
|:--------|:----|:--------|:----|:----|:--------|
| 战斗对抗 | 短句（8-20） | 高（感官细节） | 短（1-3句） | 快 | 写血写伤写痛感，不数据化。写"被扫帚打到背上留下青紫色" |
| 日常/贫民区 | 中句（15-30） | 极高（五感） | 中（3-5句） | 慢 | 写食物/天气/环境，用感官细节塑造"活着的艰辛" |
| 悬疑/威胁 | 短句（10-25） | 中（听觉为主） | 短（1-2句） | 快 | 写脚步声/狗叫/黑暗中的动静 |
| 情感互动 | 中句（15-25） | 中 | 短（2-3句） | 慢 | 写动作为主的感情表达，不写内心OS |
| 探索/冒险 | 中句（15-30） | 高 | 中（3-5句） | 中 | 写环境变化和角色应对，一步步探索 |
| 升级/学习 | 中句（15-30） | 中 | 中（3-5句） | 中 | 通过实践和代价展现成长，不写系统面板 |
| 过渡推进 | 中句（15-30） | 高 | 中（3-4句） | 中 | 过渡也是氛围——用天气/环境变化暗示时间流逝 |
