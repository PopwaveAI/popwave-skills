# 文风参考 · abyss（v3.0 — 叙事哲学DNA）

> 以《深渊主宰》（诸生浮屠）为样本提取的叙事哲学DNA。西幻·黑暗写实。
> 在 `project.yaml` 中配置 `writing_style: abyss` 启用。
>
> **适用场景**：西幻/DND/黑暗奇幻、实体出版、需要氛围沉浸的长篇
> **不适合**：日更新快节奏、对话驱动型网文、数据化升级流
>
> **取样范围**：CH1-CH3（覆盖日常叙事/系统说明/情感场景）
> **取样章节数**：3章
> **叙事者侵入频率**：约 1.5次/1000字
> **对话占比范围**：10%-15%

---

## 一、锚定章片段

> 以下原文摘自《深渊主宰》CH1-CH3，作为本DNA的具体实例。

### [氛围·贫民区夜雨]

**来源章节**：第 1 章

```
屋外传来噼里啪啦的雨声。
薇薇安小心翼翼地将矮脚凳子放在了灶台前，随即弯下腰从旁边的米袋里面抓起了
两把糙米。袋子里面的粮食已经不多了，哥哥还是上个月才从码头偷到了一块银饰，
这才换来这一袋子的糙米。即便她这半个月来已经精打细算了很多次，可是里面
越来越少的粮食还是让她感到一丝恐慌。

没有食物了。
最多再过几天他们就要饿肚子了！

薇薇安眼睛通红地望了一眼破旧房屋内的小床，铺着发霉味道的被褥上可以看到一个
昏迷中的年轻人。哥哥到今天还是没有醒来，那位曾经愿意帮助他们的牧师姐姐似乎
也放弃希望，毕竟费尔主教并不愿意费尽心力去救一位小偷。
```

**微观质感标注**：
- 句长：中短句（10-25字），急促中带有停顿
- 描写密度：**高**——听觉（雨声）+ 视觉（发霉的被褥）+ 嗅觉（霉味）
- 对话：无——纯叙述，通过动作展现情绪
- POV：**零距离**——紧贴薇薇安的视野和认知
- 段长：中短（2-4句），雨声开段独立
- 修辞：克制——无比喻，平实叙述
- **叙事哲学印证**：原则2（叙事者零侵入）+ 原则3（动作后果传达情绪）

### [情感·兄妹]

**来源章节**：第 1 章

```
"哥哥！……"
"你一定要醒过来啊！……薇薇安很想你！……"
"不要离开我！……"

一滴滴晶莹的泪珠滑落，薇薇安坐在灶台边轻轻地抽泣着，过了一会儿用小手擦了擦
脸颊，被熏得黑漆漆脸上浮现两道白痕。家里面原本还有一些盐，可是上个礼拜已经
吃完了，她想要学哥哥去偷一些，可是差点被那肥胖的店老板抓住，老板娘拿起扫帚
在她的后背上抽了一下，哪怕是现在背上依旧是一片青紫色，稍微碰到便会感觉
火辣辣的疼。
```

**微观质感标注**：
- 引导词密度：**零引导词**——纯对白推进
- 独白比例：几乎零——通过动作和眼泪展现情绪
- 推进维度：**情绪优先**——三句话都是情感释放
- 动作代替内心OS：扫帚的后背/青紫色/火辣辣的疼——用身体感受代替心理描写
- **叙事哲学印证**：原则3（动作后果传达情绪）+ 原则4（对话用于情绪释放）

### [悬疑·深夜脚步声]

**来源章节**：第 1 章

```
屋外传来一阵脚步声，似乎在门口停留了一下，隐约传来说话声。

原本老态龙钟的黑黄色老狗突然睁开了眼睛，一瞬间宛若饿狼凶悍，口中发出凶狠地
呜声，朝着外面大声地吠了一下。

"该死！"
"早晚打杀了你这条老狗炖汤！"

外面传来一个男人气急败坏地声音，随即脚步声渐渐地远去。

薇薇安惊恐的脸色稍微平静了一点点，伸出小小的手掌摸了摸老狗的头，低声道：
"希斯。"
"如果他敢进来你就咬他！"
"哥哥现在昏迷不醒，只有你能够保护我了。"
```

**微观质感标注**：
- 感官密度：高——**听觉为主**（脚步声/说话声/狗吠）+ 视觉（老狗睁眼）
- 信息释放：**埋伏笔型**——给现象不解释→门外的人是谁？
- 留白度：**技巧性留白**——不说门外人的身份意图
- 情绪节奏：**震荡型**——平静→紧张→爆发→松弛
- **叙事哲学印证**：原则1（埋伏笔信息释放）+ 原则5（震荡型张力节奏）

## 二、叙事哲学 DNA（5条原则）

### 原则 1：信息释放哲学

**信念陈述**：

> "作者认为信息应该埋伏笔式释放，先给现象，不给解释。信息在本书中是悬念的工具——留到后续再揭示。"

**证据链**（来自锚定章）：
- 悬疑·脚步声 → 门外有人站定，不提是谁，让读者自己猜
- 氛围·夜雨 → 先写贫穷的环境，后写原因（哥哥重伤、母亲病逝）——不在第一页全交代
- 第2章系统说明 → 先用 700 字一次性释放（教材式），这与原则矛盾——说明深渊的原作者在"必须交代的设定"上会打破自己的哲学

**跨章验证**：
- CH1：埋伏笔为主——贫民区危险通过薇薇安的关门回忆带出
- CH2：集中爆发——700字系统设定一次性倒出来
- CH3：埋伏笔为主——世界展望通过索伦的内心一闪而过

**结论**：深渊的DNA是**混合型**——情感/悬疑类信息埋伏笔，游戏/体系类信息集中爆发。作者在面对不同类型信息时有不同的策略。

**应用规则**（可执行指引）：
- 情感/悬疑类信息 → 先给现象，隔章揭示
- 游戏/体系类信息 → 一次性集中释放（教材式），但不重复
- 每章新概念上限 → 情感章 ≤ 2个，设定章 ≤ 6个
- 信息释放后的读者预期管理 → 埋伏笔信息在 3-5 章内必须给出解释或线索

---

### 原则 2：叙事者姿态哲学

**信念陈述**：

> "叙事者是偶尔介入的讲述者。本书以零距离POV为主，但在需要解释世界观或评价角色时，作者会跳出来对读者说话。"

**证据链**（来自锚定章）：
- 氛围·夜雨 → 叙事者未介入，紧贴薇薇安POV
- 统计结果：约 1.5次/1000字叙事者跳出来（"她今年才八岁，生火对她来说很困难""她愿意做任何事情，因为她就只有这一个亲人了"）

**跨章验证**：
- CH1：叙事者介入 3 次（解释她为什么困难、评价她坚强、总结"她就只有这一个亲人"）
- CH2：叙事者介入 2 次（"这是一个罪恶的世界"、"它的任务完成了"）
- CH3：叙事者介入 1 次（"它有这样的资格"）

**应用规则**（可执行指引）：
- POV 距离默认模式 → **零距离**（紧贴当前视角角色）
- 叙事者介入 → 仅允许在以下场景出现：
  - ⚠️ 需要说明关键背景时（"她今年才八岁"）
  - ⚠️ 情感高潮的总结性陈述（"它的任务完成了"）
- 因果关系解释 → 由叙事者主动说明，不依赖读者推导
- 情绪标签 → 允许有限使用（每千字 ≤ 1次）

---

### 原则 3：情感表达哲学

**信念陈述**：

> "情绪主要通过动作后果和身体反应传达。作者不回避直接写情绪名词，但更多时候依赖动作代替内心OS。"

**证据链**（来自锚定章）：
- 情感·兄妹 → 眼泪/抽泣/擦脸/后背青紫色——全是动作和身体反应
- 悬疑·脚步声 → "惊恐的脸色稍微平静了一点点"——不说"她很怕"，写脸色变化
- CH2的希斯之死 → "眼泪一下子涌出来了"——动作在前，情绪在后

**跨章验证**：
- CH1：主要靠动作后果（生火困难→眼泪/抽泣；后背碰到疼→抽泣）
- CH2：混合——动作+偶尔的内心OS（"不行！我要醒来！"）
- CH3：动作驱动为主（蹲在墓前/低声说话/握手）

**应用规则**（可执行指引）：
- 高强度情绪（恐惧/绝望）→ 写身体反应为主（抖/疼/凉/僵），不写情绪名词
- 动情时刻 → 通过环境呼应（雨声/黑夜/晨光），不单独说明
- 情绪转换 → 需要至少1个外部事件作为转折点
- 允许有限使用内心OS，但仅限于生死攸关的决断时刻

---

### 原则 4：对话策略哲学

**信念陈述**：

> "对话的主要功能是情绪释放而非推进信息。作者倾向于极简引导词、叙述为主的对话风格。"

**证据链**（来自锚定章）：
- 情感·兄妹 → 三句对白全用于情感释放，无信息推进
- 悬疑·脚步声 → 对话短促、无引导词、用于展示危机
- 对话占正文比例估算：10%-15%

**跨章验证**：
- CH1：对话占 ~12%，主要用于情绪释放
- CH2：对话占 ~8%，被系统说明挤压
- CH3：对话占 ~10%，推进剧情+告别

**应用规则**（可执行指引）：
- 每轮对话推进的信息维度上限 → 1 个（要么推关系，要么推情绪，不推信息）
- 引导词使用频率 → 偶用（"XX道"或直接对白）
- 独白比例 → 几乎零（不写"她想……"式独白）
- 对话中的情感表达 → 通过台词+动作（台词简短，动作补充情绪）

---

### 原则 5：张力-释放哲学

**信念陈述**：

> "张力主要通过外部威胁建立，释放通过威胁解除或情感慰藉完成。铺垫与释放呈震荡型交替——不让读者在一个情绪里待太久。"

**证据链**（来自锚定章）：
- 氛围·夜雨 → 做饭(松弛)→脚步声(紧张)→狗吠(爆发)→远去(松弛)→喂粥(温情)→打杀(紧张)
- CH1整体节奏：3次紧张↔松弛交替
- 章内铺垫-释放比例：铺垫不超过整章的 60%

**跨章验证**：
- 整体旋律：震荡型
- 连续张力章节上限：1章（CH2的张力在CH3开头就释放了）

**应用规则**（可执行指引）：
- 章末钩子类型 → 悬念截断 或 画面静止
- 大爽点前的蓄力 → 最多 1 章铺垫
- 连续无爽点章节上限 → 1 章
- 开篇策略 → 场景铺陈（先给感官信号入题）
- 收束策略 → 画面定格（用一个不要说话的画面收尾）

## 三、红线清单

### 3.1 通用红线

```
□ 否定句结构 → 0容忍
□ "他感到/他意识到/他觉得/他评估"类前缀 → 0容忍
```

### 3.2 本风格特有红线

```
□ 超过 60 字的句子 → 标记退回
□ 章节无感官描写（至少2个维度：视觉+触觉/嗅觉）→ 标记
□ 西幻名词首次出现不带上下文解释 → 标记
□ 连续 3 段纯对话 → 标记（需要叙事穿插）
□ 连续 3 段无触觉/嗅觉描写 → 标记（风格特征缺失）
□ 西幻题材出现现代口语网络用词 → 0容忍
```

### 3.3 红线触发后的动作

```
[x] 标记项 ≥ 3 → 退回
[x] 西幻题材现代用词红线 → 立即退回
```
