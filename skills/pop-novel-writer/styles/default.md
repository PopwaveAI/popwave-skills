# 文风参考 · default（v3.0 — 叙事哲学DNA）

> default 是 `pop-novel-writer` 的默认文风，由 K1-K4 + QC 规则的显式化。
> **兜底文风**——当 bootstrap 未提取参考书文风时使用。
> **适用场景**：所有网文题材的通用兜底。

## 一、锚定章片段

> 以下原文来自 default 文风的测试产出（深渊主宰 CH1 测试）。代表管线隐含规则的实例化。

### [战斗·归途警觉]

**来源章节**：ch001 测试产出

```
艾伦背着猎弓走进琥珀镇的栅栏门时，太阳刚刚偏西。他手里提着两只野兔，
收获一般。镇口的老水井立在路边，井沿上的绳结松了，几股麻绳散垂在那儿。
他瞥了一眼，心里记下——回来紧了它。

镇上人少。铁匠铺的炉火还烧着，老汤姆站在门口，目光在他手里的猎物上停了
半秒，点了点头。艾伦也点了点头，继续走。

镇长家门口站着几个人。哈里斯镇长正和几个穿深色斗篷的人说话，声音压得很低。
艾伦扫了一眼，收回目光——镇上的事跟一个猎人距离太远。
```

**微观质感标注**：
- 句长：中句（15-30字），长短自然交替
- 描写密度：中——视觉+轻微体感
- 对话：中等引导
- POV：近距离——紧贴艾伦所见和内心
- 段长：中段（3-5句）
- 修辞：克制——无比喻
- **叙事哲学印证**：原则2（叙事者零侵入）+ 原则1（信息通过观察带出）

### [悬疑·空屋发现]

**来源章节**：ch001 测试产出

```
艾伦把两只野兔换到左手，右手推开了门。

屋里空空荡荡。
灶台上的锅是凉的，盖子掀开了一半，锅底空荡荡的。火早就熄了，
灰烬只剩一片死灰。窗户大敞着，傍晚的风灌进来，吹得窗台上那排干草药
东倒西歪——薄荷叶落了好几片在地上，贴着泥地卷了边。

艾伦站在门口，目光快速扫过整个屋子。
锅碗都在原位，擦得干净。莉莉的旧布鞋整整齐齐地摆在门边——她在家的时候
穿的。挂在门旁木钉上的另一双鞋，她平时出门穿的那双，消失了。
```

**微观质感标注**：
- 感官密度：2-3个维度——视觉+触觉
- 信息释放：先给现象再给线索
- 留白度：中度留白
- **叙事哲学印证**：原则5（渐进型张力）+ 原则1（线索型信息释放）

## 二、叙事哲学 DNA（5条原则）

### 原则 1：信息释放哲学

**信念陈述**：
> "信息通过观察和发现逐步揭示。先给现象，再给线索，让读者跟着主角一起发现。"

**应用规则**：
- 新信息必须通过主角的观察/行动带出
- 每章新概念 3-5 个
- 信息释放方式：对话揭示/探索发现

### 原则 2：叙事者姿态哲学

**信念陈述**：
> "叙事者完全隐身。纯故事，没有任何叙事者声音。"

**应用规则**：
- POV距离：近距离
- 叙事者不介入、不评价、不解释
- 情绪标签：完全禁止

### 原则 3：情感表达哲学

**信念陈述**：
> "情感通过动作和环境传达，不做内心直写。"

**应用规则**：
- 情绪 = 动作后果 + 环境投影
- 高强度情绪写身体反应，不写情绪名词

### 原则 4：对话策略哲学

**信念陈述**：
> "对话均衡使用，推进信息和推进情绪各半。"

**应用规则**：
- 对话占比：40%-60%
- 引导词中等密度
- 每轮对话推进 1-2 个维度

### 原则 5：张力-释放哲学

**信念陈述**：
> "多峰型节奏——每 800-1000 字一个小起伏。弹性时间流速。"

**应用规则**：
- 开篇策略：结果前置（先写结果再写过程）
- 收束策略：悬念截断或画面静止
- 过渡：时间标记

## 三、红线清单

### 3.1 通用红线

```
□ 否定句结构 → 0容忍
□ "他感到/他意识到"类前缀 → 0容忍
```

### 3.2 本风格特有红线

```
□ 超过 60 字的句子 → 标记退回
□ 连续 3 段纯环境描写 → 退回
□ "过渡段"（不推进剧情也不推进情绪）→ 0容忍
□ 连续 3 章同一种开篇模式 → 需修改
```

### 3.3 红线触发后的动作

```
[x] 标记项 ≥ 3 → 退回
```
