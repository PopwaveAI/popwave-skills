# 风格注入包 · default（v9.5）

> 这是 `pop-novel-writer` 的默认文风，由 K1-K4 + QC 规则 + 模板池共同定义的隐含风格显式化。
> 如果 `project.yaml` 未配置 `writing_style`，自动使用此风格。

---

## 核心维度

| 维度 | 设定 | 说明 |
|:---|:---|:---|
| 句长分布 | 中句为主（15-35 字） | 不刻意短也不刻意长，自然节奏。关键动作句可独立成短句。 |
| 描写密度 | 中密度 | 战斗/紧张场景白描（动作+对话驱动），悬疑/氛围场景可拉满五感。由 Director 按场景类型分配。 |
| 对话风格 | 中等 | 有引导词但不泛滥。"他说/她说"在需要区分说话人时用。角色台词需有差异化节奏（沉默型/话多型/分析型/BOSS——见 K4）。 |
| POV 距离 | 近距离 | 以主角视角为主，紧贴主角的所见所感。不写主角"看不到"的信息。不跳入配角内心。 |
| 段长 | 中段（3-5 句） | 关键动作可独占一段。纯对话场景 1-2 句/段。不出现连续 10+ 句的无分段长段。 |
| 修辞偏好 | 克制 | 不用无意义的修辞堆砌。比喻只在"不用就说不清楚"时使用。一个段落不超过 1 个比喻。 |
| 词汇层级 | 中性 | 不刻意口语化也不刻意书面化。术语/设定名词在首次出现时附带上下文解释。 |

---

## 写法铁则（8 条）

1. **结果前置**：先写"发生了什么"，再写"过程是什么"。读者先看到结果再被满足好奇心。
2. **否定句禁止**：5 种否定式结构（不是A而是B/不是A不是B是一种C/说不清的X/直觉告诉/他没有问）正文中零容忍。详见 K4 否定句变体识别。
3. **感到类前缀禁止**："他感到/他意识到/他评估/他觉得"这类冗余前缀在正文中零容忍。直接写感官结果。
4. **碎句独占行红线**：不足 8 字且以句号结尾且不在引号内的独立行，判为碎句。核心意象句（如"两个月亮"）豁免。
5. **每章实体引用 ≥ 8 个**：事实骨架阶段标记的 {实体名} 在正文中至少出现 8 次（含不同场景）。
6. **对话推进约束**：每句对话必须推进至少一个维度（信息/关系/情绪）。连续 3 段纯对话后必须有断裂式补白（动作/环境/内心）。
7. **战斗不写招式名**：写决策后果和损伤结果，不写动作细节和术语堆叠。先写伤害，再写怎么伤的。
8. **文化锚定**：核心能力/装备必须有可追溯的文化原型特征。正文中至少体现 3 个视觉/行为特征和一个战斗中的文化源流提及。

## 场景类型 vs 写法映射

| 场景类型 | 句长 | 描写密度 | 节奏 | 参考模板 |
|:--------|:----|:--------|:----|:--------|
| 战斗对抗 | 短句（8-20 字） | 低（白描） | 快（动作链推进） | `template-pools/战斗对抗.md` |
| 对话博弈 | 中句（15-30 字） | 中（对话+微动作） | 中（信息推进） | `template-pools/对话博弈.md` |
| 悬疑揭示 | 中长句（20-40 字） | 高（五感堆叠） | 慢（信息释放缓急） | `template-pools/悬疑揭示.md` |
| 压抑蓄力 | 短句为主（10-25 字） | 中（感官聚焦） | 慢→快（递进） | `template-pools/压抑蓄力.md` |
| 爽点释放 | 短句（8-20 字） | 低（动作+情绪） | 快（爆发） | `template-pools/爽点释放.md` |
| 轻松调剂 | 中句（15-30 字） | 低（对话+日常） | 中（松弛） | `template-pools/轻松调剂.md` |
| 过渡推进 | 中句（15-30 字） | 低（信息为主） | 中（推进） | `template-pools/过渡推进.md` |

## 字数基线

| 场景 | 正常章节 | 黄金三章（CH1-3） |
|:----|:--------|:----------------|
| 目标字数 | 1800-2500 | 2200-2500 |
| 最低字数 | 1800 | 2000 |
| 最高字数 | 2800 | 2800 |

低于下限 → 合并场景或扩写。
高于上限 → 砍描写或拆章。

## 红线清单（QC 检查项）

```
□ 否定句（5 变体）→ 0 容忍
□ 感到类前缀 → 0 容忍
□ 碎句独占行（<8字，非引号内，非核心意象句）→ 0 容忍
□ 连续 3 章同种 scene_type → 需要替换
□ 字数低于 1800 → 退回
□ 骨架 {实体名} 计数 < 8 → 退回
□ 章末钩子类型与上一章相同 → 需修改
□ "想跳过"≥2 或 "会弃书" → 退回重写
```
