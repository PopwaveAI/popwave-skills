## 一、锚定章原文参考

### [战斗·归途警觉]

**来源章节**：ch001 测试产出

```
艾伦背着猎弓走进琥珀镇的栅栏门时，太阳刚刚偏西。他手里提着两只野兔，
收获一般。镇口的老水井立在路边，井沿上的绳结松了，几股麻绳散垂在那儿。
他瞥了一眼，心里记下——回来紧了它。

镇上人少。铁匠铺的炉火还烧着，老汤姆站在门口，目光在他手里的猎物上停了
半秒，点了点头。艾伦也点了点头，继续走。

镇长家门口站着几个人。哈里斯镇长正和几个穿深色斗篷的人说话，声音压得很低。
艾伦扫了一眼，收回目光——镇上的事跟一个猎人距离太远。
```

**微观质感标注**：
- 句长：中句（15-30字），长短自然交替
- 描写密度：**中**——视觉+轻微体感，基调由 1-2 个感官点定调
- 对话：**中等引导**——"点了点头""扫了一眼"
- POV：**近距离**——紧贴艾伦的所见和内心（"心里记下""距离太远"）
- 段长：中段（3-5句）
- 修辞：克制——无比喻

### [对话·邻居问询]

**来源章节**：ch001 测试产出

```
他拐进通向自家小屋的巷子。巷子很安静，远处传来孩子的嬉闹声。
其中有一个声音听起来像莉莉，他侧耳听了一瞬——那声音更尖细一些。
他继续走。

他家在巷子尽头。低矮的烟囱冒着细烟，白色的烟柱在傍晚的空气中缓缓升腾。
莉莉正在准备晚饭。

艾伦加快脚步。

走到门前，他停住了。
门虚掩着——门板和门框之间留了一道窄缝，里头透出昏暗的光。
```

**微观质感标注**：
- 引导词密度：中等——需要区分时才用
- 独白比例：少量（1-2句），短促内心反应
- 推进维度：**信息+情绪**——位置推进+不安情绪积累
- 对话中断：无对话纯叙述，用环境描写替代情绪

### [悬疑·空屋发现]

**来源章节**：ch001 测试产出

```
艾伦把两只野兔换到左手，右手推开了门。

屋里空空荡荡。
灶台上的锅是凉的，盖子掀开了一半，锅底空荡荡的。火早就熄了，
灰烬只剩一片死灰。窗户大敞着，傍晚的风灌进来，吹得窗台上那排干草药
东倒西歪——薄荷叶落了好几片在地上，贴着泥地卷了边。

艾伦站在门口，目光快速扫过整个屋子。
锅碗都在原位，擦得干净。莉莉的旧布鞋整整齐齐地摆在门边——她在家的时候
穿的。挂在门旁木钉上的另一双鞋，她平时出门穿的那双，消失了。
```

**微观质感标注**：
- 感官密度：2-3个维度——视觉（空锅/草药/布鞋）+ 触觉（凉/风）
- 信息释放：**先给现象→再给线索**——"门虚掩着"→"锅是凉的"→"另一双鞋消失了"
- 留白度：**中度留白**——写关键细节（鞋少了一双），留给读者推测
- 叙事声音：零侵入——纯叙事，不加"他不知道的是……"

## 二、叙事策略

### 2.1 时间流速

```
[x] 弹性——紧张场景放慢、过渡场景加速
```

时间处理：顺序叙事

### 2.2 信息释放模式

```
[x] 对话揭示型——信息主要通过角色对话释出
```

信息密度：中（每章 3-5 个新概念）

参考锚定章：对话场景 → 通过对话释放信息的方式

### 2.3 叙事声音侵入度

```
[x] 零侵入——纯故事，没有任何叙事者声音
```

### 2.4 开篇模式

```
[x] 结果前置——先写"发生了什么"再写"怎么到这一步的"
```

### 2.5 收束模式

```
[x] 悬念截断——在高潮/对话中间截断
[x] 画面静止——用一个定格的画面/意象收尾
```

### 2.6 情绪节奏模式

```
[x] 多峰型——每 800-1000 字一个小起伏
```

### 2.7 余白密度

```
[x] 中度留白——写关键细节，留给读者想象空间
```

参考锚定章：悬疑场景 → 留白处理方式

## 四、红线清单

### 4.1 通用红线

```
□ 否定句结构（不是A而是B / 不是A不是B是一种C / 以"不是"开段）→ 0容忍
□ "他感到/他意识到/他觉得/他评估"类前缀 → 0容忍
```

### 4.2 本风格特有红线

```
□ 超过 60 字的句子 → 标记退回
□ 连续 3 段纯环境描写 → 退回
□ "过渡段"（不推进剧情也不推进情绪的段落）→ 0容忍
□ 连续 3 章同一种开篇模式 → 需修改
```

### 4.3 红线触发后的动作

```
[x] 标记项 ≥ 3 → 退回
```
