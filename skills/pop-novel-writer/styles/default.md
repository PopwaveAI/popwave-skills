# 风格注入包 · default（v9.5）

> 这是 `pop-novel-writer` 的默认文风，由 K1-K4 + QC 规则 + 模板池共同定义的隐含风格显式化。
> 如果 `project.yaml` 未配置 `writing_style`，自动使用此风格。

---

## 第1层 · 微观质感

### 1.1 句长分布

```
[x] 中句均衡：平均 15-30 字，长短句自然交替
```

**句长变化模式**：

```
[x] 混合型——不同场景用不同模式（详见场景映射表）
```

### 1.2 描写密度与感官偏好

```
[x] 中密度——场景基调由 1-2 个感官点定调
```

**感官偏好**：

```
[x] 视觉优先——偏重"看到"什么（颜色/形状/光影）
[x] 体感（本体感）——偏重平衡/位置/内在感受
```

### 1.3 对话风格

```
[x] 中等引导——需要区分说话人时用引导词
```

**独白 vs 对白比例**：

```
[x] 对话为主——独白仅用于短促内心反应（1-2句）
```

### 1.4 POV 距离

```
[x] 近距离——以主角视角为主，但偶尔有选择性信息提示（"他不知道的是……"）
```

**POV 切换频率**：

```
[x] 从不切换——全书锁死一个视角
```

### 1.5 段长与分段模式

```
[x] 中段——每段 3-5 句
```

**分段模式**：

```
[x] 长短交替——长段铺垫后接短段爆发
```

### 1.6 修辞偏好

```
[x] 克制——比喻/拟人只在"不用就说不清楚"时使用
```

**修辞类型偏好**：

```
[x] 比喻/拟人
[x] 无偏好——用什么取决于场景
```

### 1.7 词汇层级

```
[x] 平实——中性词汇，不刻意口语也不刻意书面
```

**术语密度**：

```
[x] 低密度——术语首次出现附带上下文解释
```

## 第2层 · 叙事策略

### 2.1 时间流速

```
[x] 弹性——紧张场景放慢、过渡场景加速
```

**时间处理**：

```
[x] 顺序叙事——完全按时间顺序
```

### 2.2 信息释放模式

```
[x] 对话揭示型——信息主要通过角色对话释出
```

**信息密度**：

```
[x] 中密度——每章 3-5 个
```

### 2.3 叙事声音侵入度

```
[x] 零侵入——纯故事，没有任何叙事者声音
```

### 2.4 开篇模式

```
[x] 结果前置——先写"发生了什么"再写"怎么到这一步的"
```

### 2.5 收束模式

```
[x] 悬念截断——在高潮/对话中间截断
[x] 画面静止——用一个定格的画面/意象收尾
```

### 2.6 情绪节奏模式

```
[x] 多峰型——每 800-1000 字一个小起伏
```

### 2.7 余白密度

```
[x] 中度留白——写关键细节，留给读者想象空间
```

## 第3层 · 技法偏好

### 3.1 对话/叙述比

```
[x] 均衡型：对话和叙述各半
```

### 3.2 心理-动作比

```
[x] 心理辅助——动作在前，心理在后（"他退了半步。他在犹豫。"）
```

### 3.3 环境-剧情耦合度

```
[x] 中耦合——环境暗示情绪/氛围（雨=悲伤/阳光=希望）
```

### 3.4 过渡处理

```
[x] 时间标记——"三小时后""第二天"
```

### 3.5 感叹号/问号密度

```
[x] 克制——极端情绪时才用
```

### 3.6 金句倾向

```
[x] 偶尔——每 2-3 章 1 个
```

**金句类型偏好**：

```
[x] 意象型——用意象做情绪锚定
```

### 3.7 数字使用习惯

```
[x] 精确数字——"17分钟后""300米""3.5秒"
```

## 第4层 · 红线清单

### 4.1 句式红线

```
□ 否定句结构（不是A而是B / 不是A不是B是一种C / 以"不是"开段）→ 0容忍
□ "他感到/他意识到/他觉得/他评估"类前缀 → 0容忍
□ 超过 60 字的句子 → 标记退回
□ 连续 3 句超过 40 字 → 标记
```

### 4.2 段落红线

```
□ 连续 3 段纯环境描写 → 退回
□ 单段超过 8 句 → 标记
□ 连续 3 个短段（<8字，非核心意象句） → 标记
```

### 4.3 技法红线

```
□ "过渡段"（不推进剧情也不推进情绪的段落）→ 0容忍
□ 对话引导词连续 5 句不换 → 标记
□ 连续 3 章同一种开篇模式 → 需修改
□ 连续 3 章同一种收束模式 → 需修改
```

### 4.4 红线触发后的动作

```
[x] 标记项 ≥ 3 → 退回
```

## 场景类型 vs 写法映射

| 场景类型 | 句长 | 描写密度 | 段长 | 节奏 | 特殊要求 |
|:--------|:----|:--------|:----|:----|:--------|
| 战斗对抗 | 短句（8-20） | 低（白描） | 极短（1-2句） | 快 | 不写招式名，写决策后果 |
| 对话博弈 | 中句（15-30） | 低 | 中（3-4句） | 中 | 每句推进信息/关系/情绪 |
| 悬疑揭示 | 中长（20-40） | 高（五感堆叠） | 中长（4-6句） | 缓急 | 先给现象→再给线索→再给解释 |
| 压抑蓄力 | 短句（10-25） | 中 | 短（2-3句） | 递进快 | 压抑感通过事件节奏制造 |
| 爽点释放 | 短句（8-20） | 低 | 极短（1-2句） | 爆发 | 不给读者喘气机会 |
| 轻松调剂 | 中句（15-30） | 低 | 中（3-5句） | 松弛 | 对话推进为主 |
| 过渡推进 | 中句（15-30） | 低 | 中（3-4句） | 稳定 | 每段推进至少一个维度 |
