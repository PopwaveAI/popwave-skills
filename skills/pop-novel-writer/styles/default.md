## 一、锚定章参考模式（通用网文）

> 以下描述 default 风格的典型锚定章特征，供 bootstrap 提取时对照。
> 实际产出时替换为参考书原文片段。

### [战斗] — 中速白描型

**微观质感**：
- 句长：中句为主，紧张处短句密集
- 描写：白描，动作+结果，不写招式术语
- POV：近距离，紧贴主角
- 段长：短段（1-3句），关键动作独占一行

### [对话] — 中等引导型

**微观质感**：
- 引导词密度：需要区分说话人时用
- 独白比例：少量，1-2句短促内心反应
- 推进维度：每句对话推进至少1个维度

### [悬疑] — 感官堆叠型

**微观质感**：
- 感官密度：2-3个维度
- 信息释放：先给现象→再给线索→再给解释
- 留白度：中度留白

## 四、红线清单

### 4.1 通用红线

```
□ 否定句结构（不是A而是B / 不是A不是B是一种C / 以"不是"开段）→ 0容忍
□ "他感到/他意识到/他觉得/他评估"类前缀 → 0容忍
```

### 4.2 本风格特有红线

```
□ 超过 60 字的句子 → 标记退回
□ 连续 3 段纯环境描写 → 退回
□ "过渡段"（不推进剧情也不推进情绪的段落）→ 0容忍
□ 连续 3 章同一种开篇模式 → 需修改
```

### 4.3 红线触发后的动作

```
[x] 标记项 ≥ 3 → 退回
```
