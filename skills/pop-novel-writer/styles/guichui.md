# 文风参考 · guichui（v1.0）

> 以《鬼吹灯》（天下霸唱）为代表样本的悬疑盗墓风格。第一人称·知识科普·层层递进的悬疑氛围。
> 在 `project.yaml` 中配置 `writing_style: guichui` 启用。
>
> **适用场景**：悬疑/盗墓/探险、灵异、知识科普向、第一人称
> **不适合**：爽文/升级流、高产量日更、对话驱动型

---

## 一、锚定章原文参考

> 以下原文为风格模式的合成示例，代表鬼吹灯式悬疑叙述的典型特征。

### [氛围·古墓入口]

**风格特征示例**

```
手电筒的光柱切进黑暗里，只照亮了前方不到三米的范围。
空气是凝固的。不是那种闷热的凝滞——是冷的、沉的，像是几百年来
从没有人呼吸过，连空气自己都忘了该怎么流动。

墙壁上湿漉漉的，一层青苔泛着暗绿色的荧光。这些苔藓不该长在这里——
地下一百多米，没有阳光，没有通风，它们靠什么活？

"不对劲。"胖子在后面说了一声，声音在甬道里反复弹跳，变成了
好几层回音。

我没有回头，把手电往上照了照。穹顶很高，看不到尽头，光线消失在
一片黑雾般的暗影里。但就在那一瞬间——光柱的边缘，似乎有什么东西
缩了回去。

"走。"我说。

其实我知道，那个方向不是出口。
```

**微观质感标注**：
- 句长：中句（15-35字），长句铺氛围
- 描写密度：**高**——视觉（光柱/苔藓）+ 触觉（冷/凝固）+ 听觉（回音）
- 对话：**极简**——一两句话打断叙述，不展开对话
- POV：**近距离**——第一人称"我"的所见所感
- 段长：中段（3-5句），悬念处独立短段
- 修辞：比喻克制——"像是""仿佛"仅用于描绘不可名状之物

### [知识科普·风水格局]

**风格特征示例**

```
我蹲下来，用手指在浮尘里画了一个八卦图。

"你看，这墓室的结构不是随便挖的。"我指着东北角的方向，"坎卦主水，
那边一定有地下暗河。墓主人选在这里下葬，图的是水龙绕棺，
生生不息。但是——"

我站起来，用电筒扫了一遍周围四壁。

"这种格局有一个致命的破绽。水龙绕棺固然是好风水，可如果暗河改道，
水气就会倒灌。墓室里的空气越来越潮，就是这个道理。"

胖子的呼吸声粗了起来："那咱们岂不是……"

"嗯。"我把电筒照向来时的路。进来的石门，已经不知道什么时候，
无声无息地合上了。
```

**微观质感标注**：
- 感官密度：中——视觉（八卦图/电筒光）+ 触觉（潮湿）+ 听觉（呼吸声）
- 信息释放：**知识揭示型**——风水/盗墓知识通过主角的讲解自然释出
- 留白度：**技巧性留白**——关键信息（石门关了）在结尾轻描淡写
- 对话中断：知识段落+紧张对话交替

### [悬疑·不可名状]

**风格特征示例**

```
那声音不是从外面传进来的。

它从我背后的石壁里渗出来——像是什么东西在石头内部移动。
不是老鼠，不可能是老鼠。老鼠不会发出那种……骨骼摩擦的声音。

"你听到了吗？"我问。

胖子没说话。不用看他我也知道，他听到了。

然后墙壁动了。

不是整面墙在动——是墙面上的一块砖。它往外推了一点，又缩回去。
像是在呼吸。
```

**微观质感标注**：
- 感官密度：**极高**——听觉为主（声音/骨骼摩擦）+ 触觉（呼吸感）+ 视觉（砖动）
- 信息释放：**先不给解释**——先给现象（声音/砖动），悬念留到后面
- 留白度：高度留白——不写"是什么"，留给读者想象
- 句长：**递进**——从铺陈到短句碎片（"然后墙壁动了。"）
- 段长：**碎段收束**——先长段后越切越碎

## 二、叙事策略

### 2.1 时间流速

```
[x] 慢镜头——关键场景叙事时间≈故事时间
[x] 弹性——探索时放慢，转移时加速
```

时间处理：顺序叙事，偶尔插叙（回忆相关知识）

### 2.2 信息释放模式

```
[x] 知识揭示型——通过主角的讲解揭示盗墓/风水/历史知识
[x] 埋伏笔型——先给诡异现象，隔很久再给解释（甚至不解释）
```

信息密度：**可变**——探索场景高（新线索+新知识），转移场景低

### 2.3 叙事声音侵入度

```
[x] 低侵入——有叙述者腔调（"后来我才知道""多年后回想起来"）
```

### 2.4 开篇模式

```
[x] 氛围铺陈——先给环境和诡异感，再推进剧情
[x] 知识切入——以风水/盗墓知识开头再引入场景
```

### 2.5 收束模式

```
[x] 悬念截断——在发现/危险逼近时停顿
[x] 反转收尾——最后一句揭示事态已变（石门关了）
```

### 2.6 情绪节奏模式

```
[x] 递进型——紧张不断积累，偶尔用"胖子吐槽"短暂缓和
```

### 2.7 余白密度

```
[x] 技巧性留白——诡异现象不给科学解释，保留神秘感
```

参考锚定章：悬疑场景 → 留白和神秘感处理

## 三、技法偏好

### 3.1 对话/叙述比

```
[x] 叙述为主：对话占 < 30%
```

### 3.2 心理-动作比

```
[x] 心理辅助——动作在前，"我"的判断在后
```

### 3.3 过渡处理

```
[x] 空行分隔
[x] 时间标记——"转过一个弯""走了大约半小时"
```

### 3.4 金句倾向

```
[x] 偶尔——每 3-4 章 1 个
```

金句类型偏好：**哲理型**（关于生死/命运）、**反转型**

### 3.5 数字使用习惯

```
[x] 混合——精确用于盗墓知识（"地下五十米""三合土"），模糊于感受
```

## 四、红线清单

### 4.1 通用红线

```
□ 否定句结构 → 0容忍
□ "他感到/他意识到/他觉得/他评估"类前缀 → 0容忍
```

### 4.2 本风格特有红线

```
□ 超过 70 字的句子 → 标记退回
□ 连续 3 段无感官描写 → 标记（悬疑需要感官密度）
□ 诡异现象立即给出解释 → 标记（破坏了神秘感）
□ 第一人称出现"他不知道的是……" → 0容忍（违背第一人称规则）
□ 过度使用"突然""就在这时" → 标记（悬疑靠氛围不靠词）
□ 胖子式吐槽缺失 → 标记（悬疑需要松弛调剂）
```

### 4.3 红线触发后的动作

```
[x] 标记项 ≥ 3 → 退回
[x] 第一人称红 → 立即退回
```
