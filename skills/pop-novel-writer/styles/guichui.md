# 文风参考 · guichui（v3.0 — 叙事哲学DNA）

> 以《鬼吹灯》（天下霸唱）为代表样本的叙事哲学DNA。悬疑盗墓·第一人称·知识科普型叙事。
> 在 `project.yaml` 中配置 `writing_style: guichui` 启用。
>
> **适用场景**：悬疑/盗墓/探险、灵异、知识科普向、第一人称
> **不适合**：爽文/升级流、高产量日更、对话驱动型
>
> **取样范围**：CH1-CH3 综合特征（覆盖氛围/知识科普/悬疑三种场景）
> **取样章节数**：综合样本
> **叙事者侵入频率**：约 1次/1000字
> **对话占比范围**：15%-25%

## 一、锚定章片段

> 以下原文为风格模式的合成示例，代表鬼吹灯式悬疑叙述的典型特征。

### [氛围·古墓入口]

**风格特征示例**

```
手电筒的光柱切进黑暗里，只照亮了前方不到三米的范围。
空气是凝固的。不是那种闷热的凝滞——是冷的、沉的，像是几百年来
从没有人呼吸过，连空气自己都忘了该怎么流动。

墙壁上湿漉漉的，一层青苔泛着暗绿色的荧光。这些苔藓不该长在这里——
地下一百多米，没有阳光，没有通风，它们靠什么活？

"不对劲。"胖子在后面说了一声，声音在甬道里反复弹跳，变成了
好几层回音。

我没有回头，把手电往上照了照。穹顶很高，看不到尽头，光线消失在
一片黑雾般的暗影里。但就在那一瞬间——光柱的边缘，似乎有什么东西
缩了回去。

"走。"我说。

其实我知道，那个方向不是出口。
```

**微观质感标注**：
- 句长：中句（15-35字），长句铺氛围
- 描写密度：**高**——视觉（光柱/苔藓）+ 触觉（冷/凝固）+ 听觉（回音）
- 对话：**极简**——一两句话打断叙述，不展开对话
- POV：**近距离**——第一人称"我"的所见所感
- 段长：中段（3-5句），悬念处独立短段
- 修辞：比喻克制——"像是""仿佛"仅用于描绘不可名状之物
- **叙事哲学印证**：原则3（感官驱动情感）+ 原则5（层进式递进型张力）

---

### [知识科普·风水格局]

**风格特征示例**

```
我蹲下来，用手指在浮尘里画了一个八卦图。

"你看，这墓室的结构不是随便挖的。"我指着东北角的方向，"坎卦主水，
那边一定有地下暗河。墓主人选在这里下葬，图的是水龙绕棺，
生生不息。但是——"

我站起来，用电筒扫了一遍周围四壁。

"这种格局有一个致命的破绽。水龙绕棺固然是好风水，可如果暗河改道，
水气就会倒灌。墓室里的空气越来越潮，就是这个道理。"

胖子的呼吸声粗了起来："那咱们岂不是……"

"嗯。"我把电筒照向来时的路。进来的石门，已经不知道什么时候，
无声无息地合上了。
```

**微观质感标注**：
- 感官密度：中——视觉（八卦图/电筒光）+ 触觉（潮湿）+ 听觉（呼吸声）
- 信息释放：**知识揭示型**——风水/盗墓知识通过主角的讲解自然释出
- 留白度：**技巧性留白**——关键信息（石门关了）在结尾轻描淡写
- 对话中断：知识段落+紧张对话交替
- **叙事哲学印证**：原则1（知识揭示型信息释放）+ 原则5（技巧性留白收束）

---

### [悬疑·不可名状]

**风格特征示例**

```
那声音不是从外面传进来的。

它从我背后的石壁里渗出来——像是什么东西在石头内部移动。
不是老鼠，不可能是老鼠。老鼠不会发出那种……骨骼摩擦的声音。

"你听到了吗？"我问。

胖子没说话。不用看他我也知道，他听到了。

然后墙壁动了。

不是整面墙在动——是墙面上的一块砖。它往外推了一点，又缩回去。
像是在呼吸。
```

**微观质感标注**：
- 感官密度：**极高**——听觉为主（声音/骨骼摩擦）+ 触觉（呼吸感）+ 视觉（砖动）
- 信息释放：**先不给解释**——先给现象（声音/砖动），悬念留到后面
- 留白度：高度留白——不写"是什么"，留给读者想象
- 句长：**递进**——从铺陈到短句碎片（"然后墙壁动了。"）
- 段长：**碎段收束**——先长段后越切越碎
- **叙事哲学印证**：原则1（埋伏笔式信息释放）+ 原则5（递进型张力+碎段收束）

## 二、叙事哲学 DNA（5条原则）

### 原则 1：信息释放哲学

**信念陈述**：

> "作者认为信息应该埋伏笔式释放，先给诡异现象，不给解释。信息在本书中是制造悬疑的核心工具——先给读者'不可思议的东西'，让读者带着疑问追读。风水/盗墓知识则通过主角讲解一次性揭示（知识科普型），但诡异现象绝不立即解释。"

**证据链**（来自锚定章）：
- 氛围·古墓入口 → "光柱的边缘，似乎有什么东西缩了回去"——给现象不说是什么
- 知识科普·风水格局 → 坎卦/水龙绕棺/暗河改道——风水知识一次性释出
- 悬疑·不可名状 → "骨骼摩擦的声音""砖往外推又缩回去"——只给现象不给答案

**跨章验证**：
- 综合样本：情感/悬疑类信息埋伏笔，风水/盗墓类知识集中揭示

**应用规则**（可执行指引）：
- 悬疑/灵异信息 → 先给现象，隔章/甚至全书不解释（保留神秘感）
- 风水/盗墓/历史知识 → 通过主角讲解一次性揭示（知识科普型）
- 每章新概念上限 → 悬疑类 ≤ 2个，知识类 ≤ 4个
- 埋伏笔信息 → 在 3-5 章内必须给出线索或解释，否则读者会弃
- 连续 3 段无感官描写 → 标记（悬疑需要感官密度）

---

### 原则 2：叙事者姿态哲学

**信念陈述**：

> "叙事者是以第一人称'我'为主的亲身经历者。偶尔有回望型叙述腔调——'后来我才知道''多年后回想起来'——这是作者惯用的技巧，通过拉长时间距离增加宿命感和无力感。"

**证据链**（来自锚定章）：
- 氛围·古墓入口 → "其实我知道，那个方向不是出口"——零距离POV下的内心决断
- 知识科普·风水格局 → "我蹲下来""我站起来""我把电筒"——持续的"我"的第一人称动作
- 悬疑·不可名状 → "不用看他我也知道，他听到了"——近距离POV+对同伴的了解

**跨章验证**：
- 综合样本：第一人称零距离，偶尔有"后来才知道"类回望

**应用规则**（可执行指引）：
- POV 距离默认模式 → **零距离**（第一人称"我"的所见所感）
- 叙事者回望 → 仅允许在章节开头/结尾出现（"多年后回想起来"）
- 因果关系解释 → 通过"我"的知识推理完成，不依赖全知叙事者
- 情绪标签（"我心神不宁""我心里发毛"）→ 允许有限使用（第一人称合理）
- 第一人称出现"他不知道的是……" → 0容忍（违背第一人称规则）

---

### 原则 3：情感表达哲学

**信念陈述**：

> "情绪主要通过多感官堆叠（视觉+触觉+听觉）和环境投射传递。恐惧不是写出来的——是通过'空气是凝固的''冷的、沉的''骨骼摩擦的声音'让读者自己感受到的。胖子式吐槽是悬疑的必要调剂——用插科打诨打破持续紧张。"

**证据链**（来自锚定章）：
- 氛围·古墓入口 → "空气是凝固的""冷的、沉的""湿漉漉的青苔"——多感官堆叠制造压抑
- 悬疑·不可名状 → "骨骼摩擦的声音""砖像在呼吸"——听觉+视觉+触觉三维度建立恐惧
- 知识科普·风水格局 → "胖子的呼吸声粗了起来"——同伴的生理反应侧面衬托"我"的紧张

**跨章验证**：
- 综合样本：情感通过感官堆叠+环境投射+同伴反应三重传递

**应用规则**（可执行指引）：
- 高强度情绪（恐惧/紧张）→ 写身体反应+环境投射（"手电的光切进黑暗""呼吸变粗"） 
- 动情时刻 → 通过环境呼应（黑暗加深/回声变远/温度下降），不单独说明
- 胖子式吐槽 → 悬疑需要松弛调剂，每 1-2 个紧张场景安排一次
- 情绪转换 → 需要至少1个外部事件作为转折点（发现新线索/新危险）

---

### 原则 4：对话策略哲学

**信念陈述**：

> "对话的主要功能是打断紧张叙述和释放基础信息。作者倾向于极简对话——'走。''不对劲。''你听到了吗？'——每次对话不超过两回合，用胖子式吐槽调剂悬疑氛围。知识科普时对话带讲解性质。"

**证据链**（来自锚定章）：
- 氛围·古墓入口 → "不对劲。""走。"——共两句话，打断悬念段
- 知识科普·风水格局 → 大段讲解+"那咱们岂不是……""嗯"——知识+简短回应
- 悬疑·不可名状 → "你听到了吗？""……"——短促、以未完成句收尾
- 对话占正文比例估算：15%-25%

**跨章验证**：
- 综合样本：对话占比 15%-25%，主要用于打断紧张+知识讲解

**应用规则**（可执行指引）：
- 每轮对话推进的维度 → 1 个（要么推进情绪，要么打断气氛）
- 引导词使用频率 → 偶用（"XX道"或直接对白）
- 对话中断频率 → 每次对话不超过 2 回合
- 胖子式吐槽 → 每章至少出现 1 次（悬疑需要松弛调剂）
- 知识讲解时 → 使用较长对话+小动作穿插

---

### 原则 5：张力-释放哲学

**信念陈述**：

> "张力主要通过环境压迫（黑暗/封闭/未知）和谜题积累建立，释放通过知识揭示（破解风水格局/理解墓室结构）或危险化解完成。节奏呈单峰递进型——紧张不断积累，在关键发现处释放，但释放后立刻有新谜题。铺垫与释放的比例约为3:2。"

**证据链**（来自锚定章）：
- 氛围·古墓入口 → 黑暗寂静（铺垫）→ 发现不对劲（累积）→ 光柱边缘有东西缩回去（小释放+新悬念）
- 知识科普·风水格局 → 画八卦图讲解（铺陈知识）→ 揭示暗河改道破绽（释放）→ 石门关上了（新悬念）
- 悬疑·不可名状 → 石壁里有声音（铺垫）→ 骨骼摩擦（积累）→ 墙砖动了（释放+新恐惧）

**跨章验证**：
- 整体旋律：**递进型**——紧张不断积累，偶尔用"胖子吐槽"短暂缓和
- 章内铺垫-释放比：铺垫不超过整章的 60%（悬疑需要足够铺垫才能释放）

**应用规则**（可执行指引）：
- 章末钩子类型 → **悬念截断**（在发现/危险逼近时停顿）或 **反转收尾**（最后一句揭示事态已变）
- 大爽点前的蓄力 → 需要至少 1-2 章的铺垫（悬疑需要酝酿）
- 连续无悬疑发现章节上限 → 1 章（超过则丧失悬疑风格）
- 开篇策略 → **氛围铺陈**（先给环境和诡异感，再推进剧情）或 **知识切入**（以风水/盗墓知识开头）
- 收束策略 → 反转收尾（最后一句揭示事态已变）
- 诡异现象立即给出解释 → 标记（破坏了神秘感）

## 三、红线清单

### 3.1 通用红线

```
□ 否定句结构 → 0容忍
□ "他感到/他意识到/他觉得/他评估"类前缀 → 0容忍
```

### 3.2 本风格特有红线

```
□ 超过 70 字的句子 → 标记退回
□ 连续 3 段无感官描写 → 标记（悬疑需要感官密度）
□ 诡异现象立即给出解释 → 标记（破坏了神秘感）
□ 第一人称出现"他不知道的是……" → 0容忍（违背第一人称规则）
□ 过度使用"突然""就在这时" → 标记（悬疑靠氛围不靠词）
□ 胖子式吐槽缺失 → 标记（悬疑需要松弛调剂）
```

### 3.3 红线触发后的动作

```
[x] 标记项 ≥ 3 → 退回
[x] 第一人称红 → 立即退回
```
