# 文风参考 · nvpin（v1.0）

> 以《知否？知否？应是绿肥红瘦》为代表的女频古言风格。细腻日常，慢生活，情感推进。
> 在 `project.yaml` 中配置 `writing_style: nvpin` 启用。
>
> **适用场景**：女频古言/宅斗/宫斗、细腻日常向、慢热情感线
> **不适合**：打斗/升级流、快节奏平台、男频热血型

---

## 一、锚定章原文参考

> 以下原文为风格模式的合成示例，代表女频古言的典型特征。

### [日常·晨起梳妆]

**风格特征示例**

```
天刚蒙蒙亮的时候，外头的鸟雀便叽叽喳喳地叫了起来。
她翻了个身，被褥间还残留着昨晚熏过的茉莉清香。隔着帐子，
听见外间侍女轻手轻脚收拾的声音——瓷器碰撞的细响，水倒入铜盆的哗啦声。

"姑娘醒了？"翠儿的声音隔着帐子传来，带着刚起的微哑。

她应了一声。帐子被撩开，晨光涌进来，带着院子里青草和露水的气味。
翠儿捧着一件藕荷色的褙子，低头候在一旁。

"今日穿这件吧。"她说。

翠儿应了声是，手脚麻利地服侍她起身梳洗。铜镜里映出一张疲惫的脸，
眼下有些发青——昨夜又没睡好。
```

**微观质感标注**：
- 句长：中句（15-30字），柔和平缓
- 描写密度：高——视觉+听觉+嗅觉+触觉多感官
- 对话：**轻声细语**——引导词轻柔（"轻声道""低低地说"）
- POV：**零距离**——紧贴女主认知和感知
- 段长：中段（3-5句）
- 修辞：克制——少量比喻，以温和为主

### [对话·含蓄试探]

**风格特征示例**

```
"老太太那边……可有什么吩咐？"她端起茶盏，用盖碗轻轻拨了拨浮叶，
目光落在茶水漾开的涟漪上，并不抬头看他。

"母亲说，让你明日过去用饭。"他坐在对面的椅子上，声音听不出情绪，
"大嫂嫂也要来。"

她端茶的顿了顿，片刻后才应道："是。"

屋子里安静了一瞬。窗外传来棋子落下的清脆声响——是他大哥在廊下
与幕僚对弈。

"你……没有什么想问的？"他忽然开口。

她抬起眼，对上他的目光，又很快垂下："爷想说的，自然会告诉妾身。"
```

**微观质感标注**：
- 引导词密度：**每句对话带微动作/神态**——端茶/拨浮叶/不抬头
- 独白比例：**少但精**——内心反应通过微动作暗示（端茶手顿了顿）
- 推进维度：**关系+情绪优先**——每句话都在试探/维护/推进关系
- 对话中断频率：隔 1-2 轮一次——用环境/动作描写打断

### [冲突·言语机锋]

**风格特征示例**

```
"妹妹这话，我可不敢当。"她放下筷子，嘴角挂着笑，眼底却没什么温度，
"论理儿，我不过是遵老太太的吩咐做事罢了。"

对面的人脸色微微一变，手里的帕子绞了绞，却仍端着笑意：
"姐姐这话说的，倒像是我在埋怨老太太了？"

"我可没这么说。"她重新拿起筷子，夹了一筷春笋，慢慢地嚼了，
"妹妹多心了。"

一桌子的人都低着头吃饭，仿佛什么都没听见。
只有碗筷碰撞的声音，在忽然变得安静的厅堂里格外清晰。
```

**微观质感标注**：
- 对话密度：**高**——冲突靠对话推进，不动手
- 微表情描写：每个话轮附带面部/手部/动作
- 环境烘托：安静下来的厅堂/碗筷声/低头吃饭的众人
- 信息释放：**埋伏笔型**——对话背后有潜台词

## 二、叙事策略

### 2.1 时间流速

```
[x] 慢镜头——1章≈半天，日常细致描写
[x] 跳跃式——章间可跳数天/数月（用"几日后""过了年"）
```

时间处理：顺序叙事为主

### 2.2 信息释放模式

```
[x] 埋伏笔型——先给日常细节（如一件旧事/一件物品），隔几章揭示意义
[x] 对话揭示型——信息通过侍女/婆子的闲话自然流出
```

信息密度：**低→中**（每章 1-2 个核心信息，日常细节多于剧情推进）

### 2.3 叙事声音侵入度

```
[x] 零侵入——纯故事，叙述者不点评
```

### 2.4 开篇模式

```
[x] 日常切入——从生活细节开始（起床/梳妆/用饭/请安）
[x] 景物代入——先给环境/季节再引入人物
```

### 2.5 收束模式

```
[x] 情绪沉淀——以人物的内心状态/一句话/一个动作结束
[x] 画面静止——以一个定格的日常画面收尾
```

### 2.6 情绪节奏模式

```
[x] 平铺型——稳定的阅读节奏，靠细节积累情绪
```

### 2.7 余白密度

```
[x] 技巧性留白——人物表情/语气背后的意思留给读者去品
```

参考锚定章：对话场景 → 潜台词和微表情

## 三、技法偏好

### 3.1 对话/叙述比

```
[x] 均衡型——对话略少于叙述，但每段对话都有微动作
```

### 3.2 心理-动作比

```
[x] 心理外化——内心活动通过微动作和环境暗示（"茶盏顿了顿""帕子绞了绞"）
```

### 3.3 过渡处理

```
[x] 时间标记——"次日""过了两日"
[x] 情绪桥接——用上章结尾的情绪状态接续下章
```

### 3.4 金句倾向

```
[x] 偶尔——每 3-4 章 1 个
```

金句类型偏好：**意象型**（"日子就像茶，苦了回甘"）、哲理型

### 3.5 数字使用习惯

```
[x] 模糊表达——"一些""几件""略略"
```

> 古言风格避免精确数字，用模糊表达保持古风。

## 四、红线清单

### 4.1 通用红线

```
□ 否定句结构 → 0容忍
□ "他感到/他意识到/他觉得/他评估"类前缀 → 0容忍
```

### 4.2 本风格特有红线

```
□ 超过 60 字的句子 → 标记退回
□ 连续 3 段无动作/微表情 → 标记
□ 对话连续 5 句无动作/神态 → 标记（女频对话必须带微表情）
□ 出现口语/网络用词 → 标记（古言不可有现代感）
□ 直接写内心OS → 标记（要用动作暗示代替"他想"）
□ 连续 2 章无日常细节描写 → 标记
```

### 4.3 红线触发后的动作

```
[x] 标记项 ≥ 3 → 退回
[x] 出现现代口语网络用词 → 立即退回
```
