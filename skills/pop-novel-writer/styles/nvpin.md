# 文风参考 · nvpin（v3.0 — 叙事哲学DNA）

> 以《知否？知否？应是绿肥红瘦》为代表的女频古言叙事哲学DNA。细腻日常，慢生活，情感渐进。
> 在 `project.yaml` 中配置 `writing_style: nvpin` 启用。
>
> **适用场景**：女频古言/宅斗/宫斗、细腻日常向、慢热情感线
> **不适合**：打斗/升级流、快节奏平台、男频热血型
>
> **取样范围**：综合特征样本（覆盖日常/对话/冲突三类场景）
> **取样章节数**：综合样本
> **叙事者侵入频率**：0次/1000字（零侵入）
> **对话占比范围**：40%-50%

## 一、锚定章片段

> 以下原文为风格模式的合成示例，代表女频古言的典型特征。

### [日常·晨起梳妆]

**风格特征示例**

```
天刚蒙蒙亮的时候，外头的鸟雀便叽叽喳喳地叫了起来。
她翻了个身，被褥间还残留着昨晚熏过的茉莉清香。隔着帐子，
听见外间侍女轻手轻脚收拾的声音——瓷器碰撞的细响，水倒入铜盆的哗啦声。

"姑娘醒了？"翠儿的声音隔着帐子传来，带着刚起的微哑。

她应了一声。帐子被撩开，晨光涌进来，带着院子里青草和露水的气味。
翠儿捧着一件藕荷色的褙子，低头候在一旁。

"今日穿这件吧。"她说。

翠儿应了声是，手脚麻利地服侍她起身梳洗。铜镜里映出一张疲惫的脸，
眼下有些发青——昨夜又没睡好。
```

**微观质感标注**：
- 句长：中句（15-30字），柔和平缓
- 描写密度：高——视觉+听觉+嗅觉+触觉多感官
- 对话：**轻声细语**——引导词轻柔（"轻声道""低低地说"）
- POV：**零距离**——紧贴女主认知和感知
- 段长：中段（3-5句）
- 修辞：克制——少量比喻，以温和为主
- **叙事哲学印证**：原则1（埋伏笔型日常信息释放）+ 原则3（微动作外化情绪）

---

### [对话·含蓄试探]

**风格特征示例**

```
"老太太那边……可有什么吩咐？"她端起茶盏，用盖碗轻轻拨了拨浮叶，
目光落在茶水漾开的涟漪上，并不抬头看他。

"母亲说，让你明日过去用饭。"他坐在对面的椅子上，声音听不出情绪，
"大嫂嫂也要来。"

她端茶的顿了顿，片刻后才应道："是。"

屋子里安静了一瞬。窗外传来棋子落下的清脆声响——是他大哥在廊下
与幕僚对弈。

"你……没有什么想问的？"他忽然开口。

她抬起眼，对上他的目光，又很快垂下："爷想说的，自然会告诉妾身。"
```

**微观质感标注**：
- 引导词密度：**每句对话带微动作/神态**——端茶/拨浮叶/不抬头
- 独白比例：**少但精**——内心反应通过微动作暗示（端茶手顿了顿）
- 推进维度：**关系+情绪优先**——每句话都在试探/维护/推进关系
- 对话中断频率：隔 1-2 轮一次——用环境/动作描写打断
- **叙事哲学印证**：原则4（微表情对话策略）+ 原则5（情绪累积型节奏）

---

### [冲突·言语机锋]

**风格特征示例**

```
"妹妹这话，我可不敢当。"她放下筷子，嘴角挂着笑，眼底却没什么温度，
"论理儿，我不过是遵老太太的吩咐做事罢了。"

对面的人脸色微微一变，手里的帕子绞了绞，却仍端着笑意：
"姐姐这话说的，倒像是我在埋怨老太太了？"

"我可没这么说。"她重新拿起筷子，夹了一筷春笋，慢慢地嚼了，
"妹妹多心了。"

一桌子的人都低着头吃饭，仿佛什么都没听见。
只有碗筷碰撞的声音，在忽然变得安静的厅堂里格外清晰。
```

**微观质感标注**：
- 对话密度：**高**——冲突靠对话推进，不动手
- 微表情描写：每个话轮附带面部/手部/动作
- 环境烘托：安静下来的厅堂/碗筷声/低头吃饭的众人
- 信息释放：**埋伏笔型**——对话背后有潜台词
- **叙事哲学印证**：原则1（对话揭示型信息释放）+ 原则3（微表情外化情绪）

## 二、叙事哲学 DNA（5条原则）

### 原则 1：信息释放哲学

**信念陈述**：

> "作者认为信息应该埋伏笔式渐进释放，通过日常细节慢慢揭示。信息在本书中是情绪的载体——一件旧物/一句闲话/一个眼神，看似无意其实都是伏笔。剧情信息通过侍女/婆子的闲话自然流出，不靠教材式说明。"

**证据链**（来自锚定章）：
- 日常·晨起梳妆 → "昨夜又没睡好"——通过眼下发青的细节暗示心事，不直接说
- 对话·含蓄试探 → "老太太那边……可有什么吩咐？"——用问话试探，不直接沟通
- 冲突·言语机锋 → 每句话背后都有潜台词（"妹妹这话，我可不敢当"=你越界了）

**跨章验证**：
- 综合样本：信息通过日常对话和物件细节渐进释放，每章 1-2 个核心信息

**应用规则**（可执行指引）：
- 剧情信息 → 通过对话/闲话/日常细节释放，不做说明段
- 伏笔信息 → 先给日常细节（如一件旧事/一件物品），隔几章揭示意义
- 每章核心信息上限 → 1-2 个（日常细节多于剧情推进）
- 信息释放方式 → 对话揭示型（通过角色对话带出）> 埋伏笔型（先给现象后揭示）
- 直接写内心OS → 标记（要用动作暗示代替）

---

### 原则 2：叙事者姿态哲学

**信念陈述**：

> "叙事者完全隐身。纯故事呈现，没有任何叙事者声音。作者不点评角色、不解释动机、不预判结局——所有信息通过角色的动作/对话/环境传递给读者。读者需要自己从角色的微表情和潜台词中读取信息。"

**证据链**（来自锚定章）：
- 日常·晨起梳妆 → "她应了一声""今日穿这件吧"——纯动作+对话，无叙事者介入
- 对话·含蓄试探 → "她端茶的顿了顿，片刻后才应道"——叙事者只写动作，不做"她犹豫了"的评价
- 冲突·言语机锋 → "只有碗筷碰撞的声音"——环境说话，叙事者不评论局势

**跨章验证**：
- 综合样本：叙事者零侵入，纯故事呈现

**应用规则**（可执行指引）：
- POV 距离默认模式 → **零距离**（紧贴女主认知和感知）
- 叙事者侵入场景 → 零容忍（全风格禁止叙事者评价/解释）
- 因果关系解释 → 由读者通过场景自行推导
- 情绪标签（"她感到恐惧""她心里一紧"）→ 完全禁止

---

### 原则 3：情感表达哲学

**信念陈述**：

> "情绪主要通过微动作和环境暗示外化。作者不写直白情绪名词——不说'她很紧张'，而是写'端茶的手顿了顿''帕子绞了绞''茶杯里泛起涟漪'。内心活动全部通过外部可观察的动作和神态传递。"

**证据链**（来自锚定章）：
- 日常·晨起梳妆 → "眼下有些发青——昨夜又没睡好"——身体状态暗示心事，不写"她很焦虑"
- 对话·含蓄试探 → "她端茶的顿了顿""目光落在茶水漾开的涟漪上，并不抬头"——微动作暗示犹豫和回避
- 冲突·言语机锋 → "嘴角挂着笑，眼底却没什么温度""手里的帕子绞了绞"——表面表情和手部动作暴露真实情绪

**跨章验证**：
- 综合样本：情感通过动作/微表情/神态三重外化

**应用规则**（可执行指引）：
- 高强度情绪（愤怒/紧张/伤心）→ 写微动作（手抖/绞帕子/咬唇/垂眸），不写情绪名词
- 温柔情绪（心动/感动/安心）→ 通过环境呼应（晨光/鸟鸣/茶香），不写内心OS
- 冲突情绪 → 通过面部表情和体态呈现（笑里藏刀/低头/抬眸）
- 连续 3 段无动作/微表情 → 标记（风格特征缺失）
- 直接写内心OS → 标记（要用动作暗示代替"他想"）

---

### 原则 4：对话策略哲学

**信念陈述**：

> "对话是本书最核心的叙事工具。每一句话都附带微动作/神态，引导词密度极高——不说话的时候通过动作暗示，说话的时候通过微表情补充。对话推进关系优先于推进信息——每句话都在试探/维护/推进角色之间的关系。"

**证据链**（来自锚定章）：
- 日常·晨起梳妆 → "姑娘醒了？"（声音隔着帐子）+ "今日穿这件吧"（轻声）——对白简短，引导词柔和
- 对话·含蓄试探 → 每句话带微动作（端茶/拨浮叶/不抬头/抬眼/垂下）
- 冲突·言语机锋 → 对话高密度推进冲突，不动手，每句话有潜台词
- 对话占正文比例估算：40%-50%

**跨章验证**：
- 综合样本：对话占比 40%-50%，每句对话带动作/神态

**应用规则**（可执行指引）：
- 每轮对话推进的维度 → 1-2 个（关系+情绪优先，信息其次）
- 引导词使用频率 → 每句必用动作/神态引导
- 对话中断频率 → 隔 1-2 轮用环境/动作描写打断
- 对话连续 5 句无动作/神态 → 标记（女频对话必须带微表情）
- 出现口语/网络用词 → 标记（古言不可有现代感）

---

### 原则 5：张力-释放哲学

**信念陈述**：

> "张力主要通过人际关系的微妙变化和潜台词积累建立，释放通过一句关键对话或一个细微动作完成。节奏呈平铺型——稳定的阅读节奏，靠细节积累情绪。没有剧烈的戏剧性高潮，而是'温水煮蛙'式的情绪累积。铺垫与释放的比例约为3:1——需要很长时间酝酿才能释放一次。"

**证据链**（来自锚定章）：
- 日常·晨起梳妆 → 起床穿衣（铺陈日常）→ "昨夜又没睡好"（暗示心事，小张力）
- 对话·含蓄试探 → 端茶寒暄（铺垫）→ 提到大嫂嫂要来（关系张力建立）→ "爷想说的，自然会告诉妾身"（释放+新关系定位）
- 冲突·言语机锋 → 表面客气（铺垫）→ 夹枪带棒（张力累积）→ 众人低头吃饭/碗筷声（无声释放）

**跨章验证**：
- 整体旋律：**平铺型**——稳定节奏，靠细节积累情绪
- 章内铺垫-释放比：铺垫不超过整章的 70%（日常细节多于冲突）

**应用规则**（可执行指引）：
- 章末钩子类型 → **情绪沉淀**（以人物的内心状态/一句话/一个动作结束）或 **画面静止**（以一个定格的日常画面收尾）
- 大爽点前的蓄力 → 需要 2-3 章的日常铺垫（慢热是风格特征）
- 连续无日常细节章节上限 → 1 章（连续 2 章无日常细节 → 标记）
- 开篇策略 → **日常切入**（从生活细节开始：起床/梳妆/用饭/请安）或 **景物代入**（先给环境/季节再引入人物）
- 收束策略 → 画面静止（以一个安静的日常画面收尾）

## 三、红线清单

### 3.1 通用红线

```
□ 否定句结构 → 0容忍
□ "他感到/他意识到/他觉得/他评估"类前缀 → 0容忍
```

### 3.2 本风格特有红线

```
□ 超过 60 字的句子 → 标记退回
□ 连续 3 段无动作/微表情 → 标记
□ 对话连续 5 句无动作/神态 → 标记（女频对话必须带微表情）
□ 出现口语/网络用词 → 标记（古言不可有现代感）
□ 直接写内心OS → 标记（要用动作暗示代替"他想"）
□ 连续 2 章无日常细节描写 → 标记
```

### 3.3 红线触发后的动作

```
[x] 标记项 ≥ 3 → 退回
[x] 出现现代口语网络用词 → 立即退回
```
