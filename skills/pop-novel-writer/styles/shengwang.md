# 文风参考 · shengwang（v1.0）

> 以《圣王》（梦入神机）为样本提取的文风参考。狂暴升级流，设定密集型。
> 在 `project.yaml` 中配置 `writing_style: shengwang` 启用。
>
> **适用场景**：玄幻/修仙、升级打怪、战力膨胀型爽文、设定爆发型
> **不适合**：慢热型叙事、情感细腻型、日常向、极简描写风格

---

## 一、锚定章原文参考

### [设定爆发·气功体系]

**来源章节**：第 1 章

```
气，一切生命之本源。
气功，是生命本源的修行。
修炼到达绝顶，就有种种异能。
气功强者，可以"手破岩壁""耳闻蚁斗""足踏波面""口吐碧火"
"刀枪不入""气接云霞""吞气辟谷""不食烟火""念通天地""千里追魂"

杨奇就是这样一个气功修行者。

在丰饶大陆上，高手都修炼气功，气功修炼分为九段。
一段为"养气"。
二段"运气"，搬运体内元气，游走经络。
三段"聚气"，把元气聚集起来藏在气海之中。
四段"炼气"，聚集起来元气之后，进行提炼，百炼成钢。
五段"暴气"。气功外放，隔空伤人。
```

**微观质感标注**：
- 句长：极短→中句（5-25字），每段独立成行
- 描写密度：低——不写环境、不写天气、直接上设定
- 对话：无
- POV：全知视角，直接向读者展示世界观
- 段长：**极短段**——1句话/段，分条罗列
- 修辞：排比句式——"一段""二段""三段"式递进

### [动作·夜中取宝]

**来源章节**：第 1 章

```
砰！

城墙角落一处人迹罕至的地面，突然被震开，出现了深深洞穴。
一个十七八岁的少年全身蒙着黑衣，手里拿着一个精巧的盒子，嗖的一声，
整个人如狸猫一般，从那洞穴中跃出。

这个少年，就是杨奇。

猛烈一运动，全身内部的气流轰隆隆的作响，声势猛烈，夺人心魂。

"看来我很快就要修炼到达气功五段，暴气境。到时候，真气外放，
也是一个高手，一步登天！"

看看高高的城墙，杨奇很是满意，身躯一闪，就进入了城外漆黑的树林之中，
非常之快，简直是一头黑色的豹子，显现出来了非人的体能。
```

**微观质感标注**：
- 句长：短句（8-20字），断裂感强
- 描写密度：中——动作+轻微环境
- 对话：**内心OS为主**——通过独白推进目标和野心
- POV：**全知/中距离**——可以随时跳出写"这个少年，就是杨奇"
- 段长：极短（1-2句），砰！独立成段
- 修辞：比喻直接——"如狸猫""如黑豹""夺人心魂"

### [悬疑·实力对比]

**来源章节**：第 1 章

```
五段"暴气"，才是真正的高手。
所谓"暴气"，是气功修行中的品级，也是一个分水岭。

一般的人修行，气功修行一段到四段——"养，运，聚，炼"——
都是在体内搬运气流，只能够强身健体。

但是到达了五段"暴气"境界，就脱胎换骨。

可以气功爆发，外放，隔空伤人。
```

**微观质感标注**：
- 感官密度：低——信息优先、描写让位
- 信息释放：**设定揭示型**——通过等级体系直接展示，不给读者猜
- 留白度：写满——设定写得清清楚楚，不埋伏笔
- 叙事声音：**中侵入**——有明显叙述者（"所谓""才是"）

## 二、叙事策略

### 2.1 时间流速

```
[x] 弹性——修炼/升级时放慢，战斗时极快推进
```

时间处理：顺序叙事

### 2.2 信息释放模式

```
[x] 集中爆发——每章集中释放 1-2 个重磅等级/功法信息
[x] 探索发现型——信息通过主角修炼/战斗发现
```

信息密度：**高**（每章 ≥6 个新设定）

> 梦入神机：每章至少爆一个新等级/一个新功法/一个新世界观设定。

### 2.3 叙事声音侵入度

```
[x] 中侵入——"所谓""就是""原来如此"等直接叙议
```

### 2.4 开篇模式

```
[x] 设定先行——先给世界观/等级体系，再引入人物
[x] 动作引入——直接上冲突/取宝/战斗
```

### 2.5 收束模式

```
[x] 信息炸弹——最后一句放出新等级/新目标
[x] 悬念截断——在战斗高潮或认知刷新处截断
```

### 2.6 情绪节奏模式

```
[x] 递进型——情绪从低到高持续爬升，等级突破即是情绪高潮
```

### 2.7 余白密度

```
[x] 写满——设定/等级/功法全部写得清清楚楚，不给读者猜
```


```
[x] 时间标记——"三日后""半年后"

```

```
[x] 常规——每章 1-2 个
```

金句类型偏好：**感叹型**（"一步登天！""真正的高手！"）、境界型

## 四、红线清单

### 4.1 通用红线

```
□ 否定句结构 → 0容忍
□ "他感到/他意识到/他觉得/他评估"类前缀 → 0容忍
```

### 4.2 本风格特有红线

```
□ 连续 3 段无新设定/新等级 → 标记（风格特征缺失）
□ 超过 60 字的句子 → 标记退回
□ 战斗无等级对比 → 标记
□ 修炼/升级无"突破感"描述 → 标记
□ 设定出现不解释 → 标记（梦入神机风格设定必须解释清楚）
```

### 4.3 红线触发后的动作

```
[x] 标记项 ≥ 3 → 退回
```
