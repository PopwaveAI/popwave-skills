# 文风参考 · shengwang（v3.0 — 叙事哲学DNA）

> 以《圣王》（梦入神机）为样本提取的叙事哲学DNA。狂暴升级流，设定密集型。
> 在 `project.yaml` 中配置 `writing_style: shengwang` 启用。
>
> **适用场景**：玄幻/修仙、升级打怪、战力膨胀型爽文、设定爆发型
> **不适合**：慢热型叙事、情感细腻型、日常向、极简描写风格
>
> **取样范围**：CH1（覆盖设定爆发/动作/实力对比）
> **取样章节数**：1章
> **叙事者侵入频率**：约 3次/1000字
> **对话占比范围**：< 10%

---

## 一、锚定章片段

> 以下原文摘自《圣王》CH1，作为本DNA的具体实例。

### [设定爆发·气功体系]

**来源章节**：第 1 章

```
气，一切生命之本源。
气功，是生命本源的修行。
修炼到达绝顶，就有种种异能。
气功强者，可以"手破岩壁""耳闻蚁斗""足踏波面""口吐碧火"
"刀枪不入""气接云霞""吞气辟谷""不食烟火""念通天地""千里追魂"

杨奇就是这样一个气功修行者。

在丰饶大陆上，高手都修炼气功，气功修炼分为九段。
一段为"养气"。
二段"运气"，搬运体内元气，游走经络。
三段"聚气"，把元气聚集起来藏在气海之中。
四段"炼气"，聚集起来元气之后，进行提炼，百炼成钢。
五段"暴气"。气功外放，隔空伤人。
```

**微观质感标注**：
- 句长：极短→中句（5-25字），每段独立成行
- 描写密度：低——不写环境、不写天气、直接上设定
- 对话：无
- POV：全知视角，直接向读者展示世界观
- 段长：**极短段**——1句话/段，分条罗列
- 修辞：排比句式——"一段""二段""三段"式递进
- **叙事哲学印证**：原则1（集中爆发型信息释放）+ 原则2（高侵入叙事者姿态）

---

### [动作·夜中取宝]

**来源章节**：第 1 章

```
砰！

城墙角落一处人迹罕至的地面，突然被震开，出现了深深洞穴。
一个十七八岁的少年全身蒙着黑衣，手里拿着一个精巧的盒子，嗖的一声，
整个人如狸猫一般，从那洞穴中跃出。

这个少年，就是杨奇。

猛烈一运动，全身内部的气流轰隆隆的作响，声势猛烈，夺人心魂。

"看来我很快就要修炼到达气功五段，暴气境。到时候，真气外放，
也是一个高手，一步登天！"

看看高高的城墙，杨奇很是满意，身躯一闪，就进入了城外漆黑的树林之中，
非常之快，简直是一头黑色的豹子，显现出来了非人的体能。
```

**微观质感标注**：
- 句长：短句（8-20字），断裂感强
- 描写密度：中——动作+轻微环境
- 对话：**内心OS为主**——通过独白推进目标和野心
- POV：**全知/中距离**——可以随时跳出写"这个少年，就是杨奇"
- 段长：极短（1-2句），砰！独立成段
- 修辞：比喻直接——"如狸猫""如黑豹""夺人心魂"
- **叙事哲学印证**：原则3（等级驱动型情感释放）+ 原则5（递进型张力持续爬升）

---

### [悬疑·实力对比]

**来源章节**：第 1 章

```
五段"暴气"，才是真正的高手。
所谓"暴气"，是气功修行中的品级，也是一个分水岭。

一般的人修行，气功修行一段到四段——"养，运，聚，炼"——
都是在体内搬运气流，只能够强身健体。

但是到达了五段"暴气"境界，就脱胎换骨。

可以气功爆发，外放，隔空伤人。
```

**微观质感标注**：
- 感官密度：低——信息优先、描写让位
- 信息释放：**设定揭示型**——通过等级体系直接展示，不给读者猜
- 留白度：写满——设定写得清清楚楚，不埋伏笔
- 叙事声音：**中侵入**——有明显叙述者（"所谓""才是"）
- **叙事哲学印证**：原则1（写满不埋伏笔）+ 原则5（等级差距即张力来源）

## 二、叙事哲学 DNA（5条原则）

### 原则 1：信息释放哲学

**信念陈述**：

> "作者认为信息应该集中爆发式释放，每章重磅等级/功法设定直接倒给读者。信息在本书中是爽点的燃料——写得越满，读者越爽。"

**证据链**（来自锚定章）：
- 设定爆发·气功体系 → 开篇 500 字内将气功九段体系全部列出，不预留伏笔
- 实力对比·暴气 → 五段"暴气"的完整能力说明一口气倒出，不给读者猜的空间
- 动作·夜中取宝 → 杨奇的内心独白直接披露当前等级和目标（暴气境），零信息隐藏

**跨章验证**：
- CH1：集中爆发——九段体系+暴气标准+取宝伏笔，每段至少一个设定

**应用规则**（可执行指引）：
- 每章集中释放 1-2 个重磅等级/功法信息，不分散不埋伏笔
- 首次出现的设定概念 → 必须附带能力描述（不解释等于0容忍）
- 世界观关键信息 → 每章最多释放 ≥6 个新设定（信息密度高是核心特征）
- 信息释放后的读者预期管理 → 立即验证（设定出现即展示其威力或意义）

---

### 原则 2：叙事者姿态哲学

**信念陈述**：

> "叙事者是主动引导的说书人。作者频繁跳出场景对读者说话——'所谓''就是''这个少年，就是杨奇'等引导句式是常规武器。"

**证据链**（来自锚定章）：
- 设定爆发·气功体系 → "气，一切生命之本源""在丰饶大陆上"——叙事者直接宣告世界观
- 动作·夜中取宝 → "这个少年，就是杨奇"——叙事者向读者介绍角色身份
- 实力对比·暴气 → "所谓''暴气''""一般的人修行""但是到达了"——因果关系由叙事者主动说明

**跨章验证**：
- CH1：叙事者侵入约 3次/1000字，每次都是设定宣告或角色介绍

**应用规则**（可执行指引）：
- POV 距离默认模式 → **中距离/全知视角**（可自由跳出补充设定）
- 叙事者侵入场景 → 允许在设定宣告、角色介绍、战力对比时出现
- 因果关系解释 → 由叙事者主动说明，不依赖读者推导
- 情绪标签（"他在心中怒吼""他心里清楚"）→ 允许使用

---

### 原则 3：情感表达哲学

**信念陈述**：

> "情绪主要通过等级突破和战力膨胀释放。角色在本书中的情感不是通过细腻描摹传达的，而是通过'我很快就要突破X段''一步登天''真正的高手'这类目标宣言直接表达。"

**证据链**（来自锚定章）：
- 动作·夜中取宝 → "看来我很快就要修炼到达气功五段，暴气境。到时候，真气外放，也是一个高手，一步登天！"——情感通过等级目标宣言释放
- 实力对比·暴气 → "五段'暴气'，才是真正的高手"——叙事者替角色表达对力量的渴望
- 修炼场景 → 升级即有"脱胎换骨""一步登天"式情绪高潮

**跨章验证**：
- CH1：情感表达的主要手段=等级突破宣言+战力对比

**应用规则**（可执行指引）：
- 高强度情绪（兴奋/野心/渴望）→ 通过等级目标宣言表达，通过等级突破来释放
- 角色之间的情感（恩怨/忠诚/背叛）→ 通过战力对比来呈现
- 焦虑/紧迫感 → 通过修炼瓶颈/敌人等级压制来建立
- 情绪转换 → 由等级突破或实力提升触发

---

### 原则 4：对话策略哲学

**信念陈述**：

> "对话的主要功能是内心独白和等级宣告。作者倾向于极简的外部对话，大量使用内心OS展示角色目标和野心。对话占正文比例极低（<10%）。"

**证据链**（来自锚定章）：
- 动作·夜中取宝 → 唯一"对话"是内心OS（"看来我很快就要修炼到达气功五段"）
- 设定爆发·气功体系 → 零对话，纯叙事者宣告
- 全章估算：对话占正文比例 < 8%

**跨章验证**：
- CH1：无实质性外部对话，纯叙述+内心独白驱动

**应用规则**（可执行指引）：
- 每轮对话推进的信息维度上限 → 1 个（等级宣告或目标宣言）
- 引导词使用频率 → 极简，甚至无引导词
- 独白比例 → 高（内心OS是核心驱动方式）
- 外部对话 → 仅用于展示等级对比或修炼交流

---

### 原则 5：张力-释放哲学

**信念陈述**：

> "张力主要通过等级差距和力量压制建立，释放通过突破瓶颈和战力跨越完成。情绪呈递进型持续爬升——等级突破即是情绪高潮。铺垫与释放的比例约为1:1，不给读者喘息的间隙。"

**证据链**（来自锚定章）：
- 设定爆发·气功体系 → 先列九段体系（铺垫），然后定位杨奇当前段位（释放）
- 动作·夜中取宝 → 取宝动作（铺垫）→ 内心宣言突破目标（释放）
- 实力对比·暴气 → 一段到四段只是强身健体（铺垫），五段才是真正的高手（释放）

**跨章验证**：
- 整体旋律：**递进型**——情绪从低到高持续爬升
- 章内铺垫-释放比：铺垫不超过整章的 40%
- 连续张力上限：可持续 3 章以上，因为等级突破可以不断递进

**应用规则**（可执行指引）：
- 章末钩子类型 → **信息炸弹**（最后一句放出新等级/新目标）或 **悬念截断**（在战斗高潮截断）
- 大爽点前的蓄力 → 不超过 1 章
- 连续无爽点章节上限 → 0 章（每章必须有等级/战力推进）
- 开篇策略 → **设定先行**（先给世界观/等级体系）或 **动作引入**（直接上冲突/取宝/战斗）
- 收束策略 → 信息炸弹收尾（新等级/新目标）

## 三、红线清单

### 3.1 通用红线

```
□ 否定句结构 → 0容忍
□ "他感到/他意识到/他觉得/他评估"类前缀 → 0容忍
```

### 3.2 本风格特有红线

```
□ 连续 3 段无新设定/新等级 → 标记（风格特征缺失）
□ 超过 60 字的句子 → 标记退回
□ 战斗无等级对比 → 标记
□ 修炼/升级无"突破感"描述 → 标记
□ 设定出现不解释 → 标记（梦入神机风格设定必须解释清楚）
```

### 3.3 红线触发后的动作

```
[x] 标记项 ≥ 3 → 退回
```
