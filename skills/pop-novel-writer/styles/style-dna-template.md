# 文风DNA模板（v1.0）

> 这是定义一种文风的全量模板。覆盖 4 个层次、约 30 个维度。
>
> **使用方式**：复制此文件为 `styles/{你的风格名}.md`，逐维填写。每个维度选一个值，在 `[ ]` 中打 `x`。
> 不填的维度会降级到 `default.md` 的对应值。

---

## 第1层 · 微观质感（语言粒度的可测量特征）

### 1.1 句长分布

写正文时的句子长度偏好。

```
[ ] 短句密集：平均 8-20 字，超过 30 字的句子占比 < 10%
[ ] 中句均衡：平均 15-30 字，长短句自然交替
[ ] 长句铺陈：平均 25-45 字，复杂从句多
[ ] 混合型：按场景切换——战斗短句、描述长句（需在场景映射中具体指定）
```

**句长变化模式**：

```
[ ] 均匀型——所有句子长度波动小
[ ] 递进型——先短后长（蓄力感）或先长后短（冲击感）
[ ] 碎片型——大量超短句穿插，制造断裂节奏
[ ] 周期型——长→短→长→短有规律交替
```

### 1.2 描写密度与感官偏好

```
[ ] 极低密度（白描）——对话+动作驱动，几乎不写环境
[ ] 低密度——只写"角色能注意到"的细节
[ ] 中密度——场景基调由 1-2 个感官点定调
[ ] 高密度——五感堆叠，环境描写独立成段
[ ] 极高密度——每一段都有感官描写
```

**感官偏好（多选，选最主要的 1-2 个）**：

```
[ ] 视觉优先——偏重"看到"什么（颜色/形状/光影）
[ ] 听觉优先——偏重"听到"什么（声音/寂静/音色）
[ ] 触觉优先——偏重"身体感受到"什么（温度/触感/痛觉）
[ ] 嗅觉/味觉——偏重气味和味道
[ ] 体感（本体感）——偏重平衡/位置/内在感受
```

### 1.3 对话风格

```
[ ] 极简——不用引导词，读者通过内容分辨说话人
[ ] 轻度引导——偶用"他说/她说"，每5句对话不超过1次
[ ] 中等引导——需要区分说话人时用引导词
[ ] 高度引导——每句对话带动作/神态/语气
```

**独白 vs 对白比例**：

```
[ ] 几乎零独白——全是人物之间的对话
[ ] 对话为主——独白仅用于短促内心反应（1-2句）
[ ] 均衡——对话和内心独白各占一半左右
[ ] 独白为主——大量内心戏，对话作为点缀
```

### 1.4 POV 距离

```
[ ] 零距离——完全贴死在主角大脑里。读者获得的信息=主角获得的信息。
[ ] 近距离——以主角视角为主，但偶尔有选择性信息提示（"他不知道的是……"）
[ ] 中距离——第三人称限制视角，可描写主角的外在表现但不写内心
[ ] 远距离——可自由切换视角，全知叙事
```

**POV 切换频率**：

```
[ ] 从不切换——全书锁死一个视角
[ ] 每幕切换——幕与幕之间换视角角色
[ ] 每章切换——章与章之间可以换
[ ] 自由切换——同一章内可以跳视角（用空行分隔）
```

### 1.5 段长与分段模式

```
[ ] 极短段——每段 1-2 句，关键动作独占一行
[ ] 短段——每段 1-3 句
[ ] 中段——每段 3-5 句
[ ] 长段——每段 5-8 句
[ ] 混合——按场景类型切换（在场景映射中指定）
```

**分段模式**：

```
[ ] 均匀切分——每段长度接近
[ ] 长短交替——长段铺垫后接短段爆发
[ ] 递进分段——情绪上升时段长递增或递减
[ ] 碎段收束——段落前半部分长，后半部分越切越碎
```

### 1.6 修辞偏好

```
[ ] 零修辞——完全平铺直叙，不用任何修辞格
[ ] 克制——比喻/拟人只在"不用就说不清楚"时使用
[ ] 适度——每 3-5 段可用 1 个修辞
[ ] 密集——大量使用比喻/排比/拟人
[ ] 华丽——修辞是正文的核心质感之一
```

**修辞类型偏好（多选，最多 2 个）**：

```
[ ] 比喻/拟人
[ ] 排比/对仗
[ ] 夸张/反差
[ ] 反复/层递
[ ] 设问/反问
[ ] 无偏好——用什么取决于场景
```

### 1.7 词汇层级

```
[ ] 口语化——日常用词，短缩写法，接近说话
[ ] 平实——中性词汇，不刻意口语也不刻意书面
[ ] 书面化——规范的书面语，避免口语化表达
[ ] 文雅——用词考究，古典词汇与白话混合
[ ] 炫技——大量生僻词/专业词汇/自创词
```

**术语密度**：

```
[ ] 零术语——所有概念用白话解释
[ ] 低密度——术语首次出现附带上下文解释
[ ] 中密度——术语可堆叠但不超过 3 个连续
[ ] 高密度——术语是正文的识别特征，不解释
```

## 第2层 · 叙事策略（故事怎么被讲出来）

### 2.1 时间流速

叙事时间与故事时间的缩放比率。

```
[ ] 慢镜头——叙事时间 ≈ 故事时间，1章≈几分钟
[ ] 正常流速——1章≈半天到一天
[ ] 快速推进——1章≈2-7天
[ ] 跳跃式——章间可跳数天到数月（用"X天后"过渡）
[ ] 弹性——紧张场景放慢、过渡场景加速
```

**时间处理**：

```
[ ] 顺序叙事——完全按时间顺序
[ ] 倒叙/插叙——允许回忆倒叙
[ ] 多线并行——同一时间不同地点切换
```

### 2.2 信息释放模式

新信息（设定/剧情推进）以什么方式给读者。

```
[ ] 集中爆发——每章集中释放 1-2 个重磅信息
[ ] 均匀撒放——信息持续不断地穿插在每段中
[ ] 埋伏笔型——先给现象，隔几章再给解释
[ ] 对话揭示型——信息主要通过角色对话释出
[ ] 探索发现型——信息通过主角探索/观察发现
```

**信息密度**：

```
[ ] 低密度——每章 1-2 个新概念/新事件
[ ] 中密度——每章 3-5 个
[ ] 高密度——每章 ≥6 个
[ ] 可变——按章节节奏调整（在场景映射中指定）
```

### 2.3 叙事声音侵入度

叙述者在正文中是否"刷存在感"。

```
[ ] 零侵入——纯故事，没有任何叙事者声音
[ ] 低侵入——偶尔有"如果你仔细看""当然，事情没那么简单"
[ ] 中侵入——叙事者偶尔点评/剧透/与读者对话
[ ] 高侵入——叙述者是正文的一部分（如第一人称叙事/说书人口吻）
```

### 2.4 开篇模式

每章第一段的写法。

```
[ ] 钩子先行——第一句话就是剧情推进或悬念
[ ] 场景铺陈——先给环境/时间/地点再推进
[ ] 对话切入——以对话开头
[ ] 内心独白——以角色内心活动开头
[ ] 结果前置——先写"发生了什么"再写"怎么到这一步的"
[ ] 混合——按章节类型切换（金手指/战斗/过渡等有不同开篇）
```

### 2.5 收束模式

每章最后两段的写法。

```
[ ] 悬念截断——在高潮/对话中间截断
[ ] 情绪总结——给读者"喘口气"的余地再结束
[ ] 信息炸弹——最后一句放出新的重磅信息
[ ] 画面静止——用一个定格的画面/意象收尾
[ ] 对话收尾——以一句有分量的话结束
```

### 2.6 情绪节奏模式

一章之内张弛交替的规律。

```
[ ] 单峰型——铺垫大半章，最后集中释放
[ ] 多峰型——每 800-1000 字一个小起伏
[ ] 递进型——情绪从低到高持续爬升
[ ] 震荡型——紧张→放松→紧张→放松高频交替
[ ] 平铺型——情绪波动小，稳定的阅读节奏
```

### 2.7 余白密度

给读者自己脑补的空间有多大。

```
[ ] 写满——心理/环境/解释全部写清楚
[ ] 中度留白——写关键细节，留给读者想象空间
[ ] 高度留白——只写动作和对话，心理和环境全靠读者脑补
[ ] 技巧性留白——关键信息故意不写，让读者主动猜测
```

## 第3层 · 技法偏好（区分不同平台/流派的核心变量）

### 3.1 对话/叙述比

```
[ ] 对话驱动：对话占全文 60%+
[ ] 均衡型：对话和叙述各半
[ ] 叙述为主：对话占 < 40%
[ ] 全叙述：几乎零对话
```

### 3.2 心理-动作比

```
[ ] 动作驱动——写做不写想，读者通过行为理解角色
[ ] 心理辅助——动作在前，心理在后（"他退了半步。他在犹豫。"）
[ ] 心理为主——大量内心独白驱动决策
[ ] 纯心理——极简外部动作，以内心活动为主线
```

### 3.3 环境-剧情耦合度

环境描写是否具有叙事功能。

```
[ ] 零耦合——不写环境，只写事件
[ ] 弱耦合——环境作为"背景板"，几笔带过
[ ] 中耦合——环境暗示情绪/氛围（雨=悲伤/阳光=希望）
[ ] 强耦合——环境是叙事的参与者（天气变化=剧情转折）
[ ] 象征性——环境描写本身就是隐喻系统
```

### 3.4 过渡处理

章节与章节之间怎么衔接。

```
[ ] 硬切——直接进入新场景，不用过渡段
[ ] 空行分隔——用"---"或空行分段
[ ] 时间标记——"三小时后""第二天"
[ ] 场景总结——一段总结/过渡后进入新场景
[ ] 情绪桥接——用上章结尾的情绪对接下章开头
```

### 3.5 感叹号/问号密度

情绪表达的外显强度。

```
[ ] 零感叹——全文几乎不用感叹号
[ ] 克制——极端情绪时才用
[ ] 中等——每章 3-5 个
[ ] 密集——每段都有，快节奏章节更密
```

### 3.6 金句倾向

在正文中穿插"可摘抄"的句子的习惯。

```
[ ] 不写金句——叙事完全平实
[ ] 偶尔——每 2-3 章 1 个
[ ] 常规——每章 1-2 个金句
[ ] 密集——关键场景每段都想让人划线
```

**金句类型偏好**：

```
[ ] 哲理型——关于人生/世界的总结
[ ] 感叹型——"这就是XX"
[ ] 反转型——前一句和后一句形成反差
[ ] 意象型——用意象做情绪锚定
```

### 3.7 数字使用习惯

```
[ ] 精确数字——"17分钟后""300米""3.5秒"
[ ] 模糊表达——"一炷香的时间""大约一箭之地"
[ ] 混合——精确用于战斗/数据，模糊用于日常
[ ] 零数字——完全不用具体数字
```

## 第4层 · 红线清单（本条风格下绝对禁止的东西）

### 4.1 句式红线

```
□ 否定句结构（不是A而是B / 不是A不是B是一种C / 以"不是"开段）→ 0容忍
□ "他感到/他意识到/他觉得/他评估"类前缀 → 0容忍
□ 超过 [__] 字的句子 → 标记退回
□ 连续 [__] 句超过 [__] 字 → 标记
```

### 4.2 段落红线

```
□ 连续 [__] 段纯环境描写 → 退回
□ 单段超过 [__] 句 → 标记
□ 连续 [__] 个短段（<8字） → 标记（散句豁免）
```

### 4.3 技法红线

```
□ "过渡段"（不推进剧情也不推进情绪的段落）→ 0容忍
□ 对话引导词连续 [__] 句不换 → 标记
□ 连续 [__] 章同一种开篇模式 → 需修改
□ 连续 [__] 章同一种收束模式 → 需修改
```

### 4.4 红线触发后的动作

```
[ ] 任何红线 → 退回重写
[ ] 标记项 ≥ 3 → 退回
[ ] 标记项 ≥ 5 → 退回至 Director 阶段
[ ] 红线只标记，不自动退回（由 QC 确认）
```
