# 文风参考 · tomato（v9.5）

> 番茄快节奏文风参考。适用于日更短章节平台。
> 在 `project.yaml` 中配置 `writing_style: tomato` 启用。
>
> **与 default 的核心差异**：对话驱动（60%+）、极短段、余白写满、多峰式情绪起伏。

---

## 一、锚定章参考模式

> 实际产出时替换为参考书对应场景的原文片段。

### [战斗] — 极速冲击型

**微观质感**：
- 句长：极短（5-15字），一句一拳
- 描写：极低，不写环境
- 对话：零引导词
- POV：零距离
- 段长：1句/段

### [对话] — 纯对白型

**微观质感**：
- 引导词密度：零。读者通过内容分辨
- 独白比例：几乎零。每500字最多1句，不超过15字
- 推进维度：每句推进信息

### [悬疑] — 快释型

**微观质感**：
- 感官密度：低（1个维度），以视觉为主
- 信息释放：集中爆发，不给读者猜测时间
- 留白度：写满——不给读者停下来想的机会

## 二、叙事策略

### 2.1 时间流速

```
[x] 快速推进——1章≈2-7天
```

> 不写"几分钟内的战斗"。慢镜头只在幕末终极爽点使用。

时间处理：

```
[x] 顺序叙事
[x] 跳跃式——章间可跳数天（用"三天后"硬切）
```

### 2.2 信息释放模式

```
[x] 集中爆发——每章集中释放 1-2 个重磅信息
```

信息密度：高（每章 ≥6 个）

> 番茄读者每刷一屏约5秒。连续2屏无信息推进，读者划走。

### 2.3 叙事声音侵入度

```
[x] 零侵入——纯故事，没有任何叙事者声音
```

### 2.4 开篇模式

```
[x] 钩子先行——第一句话就是剧情推进或悬念
```

> 前300字必须有钩子。不铺环境、不建世界观、不设内心独白。

### 2.5 收束模式

```
[x] 悬念截断——在高潮/对话中间截断
```

> 绝不用情绪总结收尾。最后一句话必须让人想点下一章。

### 2.6 情绪节奏模式

```
[x] 多峰型——每 800-1000 字一个小起伏
```

> 每800-1000字必须有：反转/微爽点/新信息/情绪冲击。

### 2.7 余白密度

```
[x] 写满——心理/环境/解释全部写清楚
```

> 不给读者"自己脑补"的机会。唯一留白的是"下一章会发生什么"。

## 三、技法偏好

### 3.1 对话/叙述比

```
[x] 对话驱动：对话占全文 60%+
```

### 3.2 心理-动作比

```
[x] 动作驱动——写做不写想，读者通过行为理解角色
```

参照锚定章：战斗场景 → 动作驱动，不写内心OS

### 3.3 过渡处理

```
[x] 硬切——直接进入新场景，不用过渡段
[x] 空行分隔——用"---"或空行分段
```

### 3.4 金句倾向

```
[x] 常规——每章 1-2 个金句
```

金句类型偏好：反转型、感叹型

### 3.5 数字使用习惯

```
[x] 精确数字——"17分钟后""300米""3.5秒"
```

## 四、红线清单

### 4.1 通用红线

```
□ 否定句结构 → 0容忍
□ "他感到/他意识到/他觉得/他评估"类前缀 → 0容忍
```

### 4.2 本风格特有红线

```
□ 超过 40 字的句子 → 标记退回
□ 连续 2 段纯环境描写 → 退回
□ 对话引导词 → 0容忍
□ 首 300 字无钩子 → 退回 Director 阶段
□ 内心独白连续超过 2 句 → 退回
□ 出现"像/如同/仿佛"类比喻词 → 标记
□ 出现"然后""接着""就在这时"等口语过渡词 → 标记
□ 战后/事件后的情绪回收段落 → 0容忍
□ 连续 3 段无对话 → 退回
□ 段首出现环境铺陈（"天空是……""空气中……"）→ 标记
```

### 4.3 红线触发后的动作

```
[x] 任何红线 → 退回重写
[x] 标记项 ≥ 2 → 退回
```
