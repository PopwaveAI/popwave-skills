# 文风参考 · tomato（v9.5）

> 番茄快节奏文风参考。适用于日更短章节平台。
> 在 `project.yaml` 中配置 `writing_style: tomato` 启用。
>
> **适用场景**：日更平台（番茄/七猫）、快节奏爽文、升级流、对话驱动型
> **不适合**：需要氛围沉浸的西幻/悬疑、慢热型叙事、出版级文学

---

## 一、锚定章原文参考

> 以下原文来自 tomato 文风的真实产出（深渊主宰测试）。非独立参考书，是番茄平台优化规则的实例化。

### [战斗·镇口对峙]

**来源章节**：ch001 测试产出

```
艾伦背着猎弓走进琥珀镇。

"站住。"

守门的老头抬起眼皮。看清是他，又缩回棚子底下。

"今天收成怎么样？"

"两只兔子。"

"行。"

老头点了点头。

"走了。"

艾伦穿过镇口。手里提着两只野兔。兔子还温着，脖子上的箭口凝了血痂。

他经过老水井。井绳又松了一圈。

"得紧了。"
```

**微观质感标注**：
- 句长：**极短**（3-12字），碎句推进
- 描写密度：**极低**——不写环境、不写五感，只写动作和对话
- 对话：**零引导词**——通过内容分辨谁在说话
- POV：**零距离**——艾伦的所见所做
- 段长：**1句/段**，关键动作独占一行
- 修辞：**零修辞**——无比喻、无拟人

### [对话·铁匠铺前]

**来源章节**：ch001 测试产出

```
铁匠铺的炉火还亮着。老汤姆蹲在铺子门口，擎着烟斗。

"回来了？"

"嗯。"

老汤姆的目光在野兔上停了半秒。

"明天还去？"

"看天气。"

"行。"
```

**微观质感标注**：
- 引导词密度：**零**——纯对白
- 独白比例：零——不写任何内心活动
- 推进维度：**信息推进**——每句推进最小信息量
- 字数：每个话轮不超过 5 个字

### [悬疑·空屋]

**来源章节**：ch001 测试产出

```
他推开门。屋子里是空的。

灶台是凉的。炉膛里只剩死灰。

窗户开着的。风从窗口灌进来，吹得干薄荷叶子满地打转。

他站了一秒。

然后转身出了门。
```

**微观质感标注**：
- 感官密度：低——只写视觉
- 信息释放：**集中爆发**——一推开就给结果（空的/凉的），不给过渡
- 留白度：**写满**——不留给读者猜测时间
- 情绪节奏：**多峰型**——推门（紧张）→ 空（冲击）→ 转身出门（下一个开始）

## 二、叙事策略

### 2.1 时间流速

```
[x] 快速推进——1章≈2-7天
```

> 不写"几分钟内的战斗"。慢镜头只在幕末终极爽点使用。

时间处理：

```
[x] 顺序叙事
[x] 跳跃式——章间可跳数天（用"三天后"硬切）
```

### 2.2 信息释放模式

```
[x] 集中爆发——每章集中释放 1-2 个重磅信息
```

信息密度：高（每章 ≥6 个）

> 番茄读者每刷一屏约5秒。连续2屏无信息推进，读者划走。

### 2.3 叙事声音侵入度

```
[x] 零侵入——纯故事，没有任何叙事者声音
```

### 2.4 开篇模式

```
[x] 钩子先行——第一句话就是剧情推进或悬念
```

> 前300字必须有钩子。不铺环境、不建世界观、不设内心独白。

### 2.5 收束模式

```
[x] 悬念截断——在高潮/对话中间截断
```

> 绝不用情绪总结收尾。最后一句话必须让人想点下一章。

### 2.6 情绪节奏模式

```
[x] 多峰型——每 800-1000 字一个小起伏
```

> 每800-1000字必须有：反转/微爽点/新信息/情绪冲击。

### 2.7 余白密度

```
[x] 写满——心理/环境/解释全部写清楚
```

> 不给读者"自己脑补"的机会。唯一留白的是"下一章会发生什么"。

## 三、技法偏好

### 3.1 对话/叙述比

```
[x] 对话驱动：对话占全文 60%+
```

### 3.2 心理-动作比

```
[x] 动作驱动——写做不写想，读者通过行为理解角色
```

参照锚定章：战斗场景 → 动作驱动，不写内心OS

### 3.3 过渡处理

```
[x] 硬切——直接进入新场景，不用过渡段
[x] 空行分隔——用"---"或空行分段
```

### 3.4 金句倾向

```
[x] 常规——每章 1-2 个金句
```

金句类型偏好：反转型、感叹型

### 3.5 数字使用习惯

```
[x] 精确数字——"17分钟后""300米""3.5秒"
```

## 四、红线清单

### 4.1 通用红线

```
□ 否定句结构 → 0容忍
□ "他感到/他意识到/他觉得/他评估"类前缀 → 0容忍
```

### 4.2 本风格特有红线

```
□ 超过 40 字的句子 → 标记退回
□ 连续 2 段纯环境描写 → 退回
□ 对话引导词 → 0容忍
□ 首 300 字无钩子 → 退回 Director 阶段
□ 内心独白连续超过 2 句 → 退回
□ 出现"像/如同/仿佛"类比喻词 → 标记
□ 出现"然后""接着""就在这时"等口语过渡词 → 标记
□ 战后/事件后的情绪回收段落 → 0容忍
□ 连续 3 段无对话 → 退回
□ 段首出现环境铺陈（"天空是……""空气中……"）→ 标记
```

### 4.3 红线触发后的动作

```
[x] 任何红线 → 退回重写
[x] 标记项 ≥ 2 → 退回
```
