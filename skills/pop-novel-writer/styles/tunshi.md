# 风格注入包 · tunshi（v1.0）

> 以《吞噬星空》（我吃西红柿）为样本提取的文风。科幻升级流，平实流畅，细节扎实，世界观通过日常自然渗透。
> 在 `project.yaml` 中配置 `writing_style: tunshi` 启用。
>
> **特征**：叙事密度高但节奏松弛，技术细节精确（精确数字+装备参数），对话带微动作和神态，慢热型推进。

---

## 第1层 · 微观质感

### 1.1 句长分布

```
[x] 中句均衡：平均 15-35 字，长短句自然交替
```

**句长变化模式**：

```
[x] 均匀型——所有句子长度波动小，稳定的叙述节奏
```

### 1.2 描写密度与感官偏好

```
[x] 中密度——场景基调由 1-3 个感官点定调
```

**感官偏好**：

```
[x] 视觉优先——详细描写视觉细节（颜色/形状/大小）
[x] 听觉优先——环境音效/怪兽吼声/城市噪音皆有描写
```

### 1.3 对话风格

```
[x] 中等引导——需要区分说话人时用引导词，对话带动作/神态（"罗峰一笑""魏文竖起大拇指""壮硕男生忐忑道"）
```

**独白 vs 对白比例**：

```
[x] 均衡——对话和内心独白各占一半左右
```

> 角色内心活动丰富，且经常通过独白做决策推演。

### 1.4 POV 距离

```
[x] 近距离——以主角视角为主，偶尔切换外部视角（"此时宜安区上空……"）
```

**POV 切换频率**：

```
[x] 每章切换——章与章之间可以换，偶尔同一章内切换（用空行分隔）
```

### 1.5 段长与分段模式

```
[x] 中段——每段 3-6 句
```

**分段模式**：

```
[x] 均匀切分——每段长度接近，稳定的叙述流
```

### 1.6 修辞偏好

```
[x] 克制——比喻/'仿佛'适度使用，主要用于环境描写（"天空仿佛一块巨大的蓝色翡翠"）
```

**修辞类型偏好**：

```
[x] 比喻/拟人
```

### 1.7 词汇层级

```
[x] 平实——中性词汇，不刻意口语也不刻意书面
```

**术语密度**：

```
[x] 中密度——术语可堆叠但不超过 3 个连续，术语自带解释（"3.9马赫""威力达到21米"）
```

> 科技/怪兽参数精确给出，形成"硬核感"。

## 第2层 · 叙事策略

### 2.1 时间流速

```
[x] 正常流速——1章≈半天到一天
```

**时间处理**：

```
[x] 顺序叙事——完全按时间顺序
```

### 2.2 信息释放模式

```
[x] 均匀撒放——信息持续不断地穿插在每段中
[x] 对话揭示型——信息通过角色对话和独白双重释出
```

**信息密度**：

```
[x] 中密度——每章 3-5 个新概念/设定，逐步展开世界观
```

### 2.3 叙事声音侵入度

```
[x] 低侵入——偶尔有"现如今""此刻""不得不说"类引导
```

### 2.4 开篇模式

```
[x] 场景铺陈——先给环境/时间/地点再推进剧情
```

> 典型开篇：先描写时间+场景+氛围，再引入人物和对话。

### 2.5 收束模式

```
[x] 情绪总结——给读者"喘口气"的余地再结束
[x] 对话收尾——以一句有分量的话或日常对话结束
```

### 2.6 情绪节奏模式

```
[x] 多峰型——每 1200-1500 字一个小起伏
```

### 2.7 余白密度

```
[x] 中度留白——写关键细节，留给读者想象空间
```

> 战斗和参数写得很满，但日常和情感留有大量余白。

## 第3层 · 技法偏好

### 3.1 对话/叙述比

```
[x] 均衡型：对话和叙述各半
```

### 3.2 心理-动作比

```
[x] 心理辅助——动作在前，心理在后（"罗峰心跳微微加速。心中掠过一个名字——徐欣！"）
```

### 3.3 环境-剧情耦合度

```
[x] 中耦合——环境暗示情绪/氛围（城市环境→秩序/安全，荒野怪兽→危险）
```

### 3.4 过渡处理

```
[x] 时间标记——"第二天""回家的路上""此时"
```

### 3.5 感叹号/问号密度

```
[x] 克制——极端情绪时才用
```

### 3.6 金句倾向

```
[x] 偶尔——每 2-3 章 1 个
```

**金句类型偏好**：

```
[x] 感叹型——"这就是XX"
[x] 哲理型——关于奋斗/成长/梦想的小总结
```

### 3.7 数字使用习惯

```
[x] 精确数字——"3.9马赫""20mm火神炮""36平米""32层"
```

> 这是我吃西红柿风格的重要识别特征——大量精确技术参数。

## 第4层 · 红线清单

### 4.1 句式红线

```
□ 否定句结构（不是A而是B / 不是A不是B是一种C / 以"不是"开段）→ 0容忍
□ "他感到/他意识到/他觉得/他评估"类前缀 → 0容忍
□ 超过 60 字的句子 → 标记退回
□ 首次出现术语不带解释 → 标记
```

### 4.2 段落红线

```
□ 连续 3 段纯环境描写 → 标记
□ 单段超过 10 句 → 标记
□ 战斗场景无精确数字 → 标记（吞噬星空风格下战斗必须附带数据）
```

### 4.3 技法红线

```
□ "过渡段"（不推进剧情也不推进情绪的段落）→ 标记
□ 对话引导词连续 5 句不换 → 标记
□ 连续 3 章无技术参数/数据 → 标记（风格特色缺失）
□ 内心独白无实际行动衔接 → 标记（"想了就得做"）
```

### 4.4 红线触发后的动作

```
[x] 标记项 ≥ 3 → 退回
```

## 场景类型 vs 写法映射

| 场景类型 | 句长 | 描写密度 | 段长 | 节奏 | 特殊要求 |
|:--------|:----|:--------|:----|:----|:--------|
| 战斗对抗 | 中句（15-30） | 中（动作+参数） | 中（3-5句） | 中→快 | 附带精确速度和力量数据。先写动作，再写结果 |
| 对话博弈 | 中句（15-30） | 中（微动作+神态） | 中（3-5句） | 中 | 每句对话附带说话人的动作或表情 |
| 悬疑揭示 | 中长句（20-40） | 高 | 中长（4-6句） | 缓→快 | 先给现象和数据，再给认知刷新 |
| 日常/都市 | 中句（15-30） | 中 | 中（3-5句） | 慢→稳 | 通过日常细节展现世界观，不急推进 |
| 修炼/学习 | 中句（15-30） | 低→中 | 中（3-5句） | 慢 | 写过程不写结果。修炼细节有数据支撑 |
| 情感互动 | 短→中（10-25） | 低 | 短（2-3句） | 慢 | 写微表情和微动作，不写内心OS |
| 过渡推进 | 中句（15-30） | 低 | 中（3-4句） | 稳定 | 时间标记明确，每段推进位置或信息 |
