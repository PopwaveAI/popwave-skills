# 文风参考 · tunshi（v1.0）

> 以《吞噬星空》（我吃西红柿）为样本提取的文风参考。科幻升级流。
> 在 `project.yaml` 中配置 `writing_style: tunshi` 启用。
>
> **适用场景**：科幻/末世/升级流、起点长篇慢热、技术参数向
> **不适合**：纯西幻、古典仙侠、日更快节奏（字数偏高）

---

## 一、锚定章原文参考

> 以下原文摘自《吞噬星空》第 1-2 章，作为本条文风的具体实例。

### [战斗·空中危机]

**来源章节**：第 1 章

```
一道刺耳之际的鸣叫声猛然响起，雷的声音很大，震耳欲聋！而这高亢的鸣叫声却是刺耳，
罗峰只感到耳朵鼓膜都是隐隐疼痛，不由有些难受皱起眉头……
高亢刺耳的鸣叫声音震荡下，旁边街道的一座摩天大楼上一块块巨大的玻璃，发出低沉的
"卡——卡——"声音，不少玻璃龟裂开来，甚至于当场有十多块玻璃从高空掉落了下来……

一块玻璃碎片，仿佛刀片射向罗峰。
罗峰目光一扫，并没有闪躲，而是平静站在原地，右手闪电般伸出，瞬间抓出这迸射过来的
碎玻璃。这玻璃碎块隐约映照出罗峰的样子，掂了两下，随手一扔，碎玻璃仿佛暗器一样
直接射向远处的垃圾箱，哐当一声，准确的从垃圾箱口射进。
```

**微观质感标注**：
- 句长：中句（15-30字），动作处变短
- 描写密度：中——视觉+听觉，先写现象再写人
- 对话：无（战斗不说话）
- POV：近距离，紧贴罗峰的感官
- 段长：中段（3-5句），均匀
- 修辞：克制——"仿佛刀片""仿佛暗器"

### [对话·校门口]

**来源章节**：第 1 章

```
"罗峰师兄！罗峰师兄！"一道粗厚的声音响起。
"阿峰，有人喊你。"

罗峰疑惑转头看去，喊他的是一名男生，身高大概有一米九，长的虎背熊腰。
"你是？"
"罗峰师兄，我，我有事想请罗峰师兄帮忙。"壮硕男忐忑道。
"什么事？"罗峰一笑，说道。
"我练拳的时候，出拳发劲，总是感觉不对，不知道师兄你有没有时间指点一下。"
"哦，是这样……"罗峰略微停顿一下，点点头，"这样吧，这个星期的星期天下午，
在武馆的时候你找我。"
"谢师兄，谢师兄了。"
```

**微观质感标注**：
- 引导词密度：每句对话带引导（"忐忑道""一笑""略微停顿"）
- 独白比例：低——只有动作和对话，不写内心想法
- 推进维度：信息推进——问请求→问原因→给答复
- 对话中断频率：每 2-3 轮中断一次（用动作/神态描写）

### [悬疑·黑冠金雕揭秘]

**来源章节**：第 1 章

```
"黑冠金雕，在雕类怪兽中排第三。"魏文双眸放光，低沉道，"成年体的黑冠金雕，
体长一般达到21米，翼展在36米左右，飞行时极速可以达到3.9马赫。那可是足足
3.9倍音速。音速按照一秒340米计算的话，就是一秒1326米，也就是一小时4774公里。"

罗峰知道黑冠金雕厉害，可是听到一秒1326米的极限速度，依旧感到心中屏息。
一秒钟，也就是眼睛一眨啊。就一千米开外了。

"这黑冠金雕，羽毛比金刚石还要坚硬，应该达到三级克罗合金的硬度。"魏文兴奋连道，
"网上可是有视频的，黑冠金雕曾经随怪兽群冲击过军队，遭受到口径20mm的火神炮轰击，
火神炮可是一分钟7000发子弹……可是……竟然连一片羽毛都没轰击掉下来。"
"后来，还是一名神秘武者强者，手持一柄克罗合金战刀，化作一道流光，将黑冠金雕
一刀劈成两半！"
```

**微观质感标注**：
- 感官密度：中——视觉（魏文的兴奋表情）+ 听觉（魏文的语气）
- 信息释放：**数据揭示型**——精确参数（3.9马赫/21米/20mm）→ 对比（火神炮打不掉）→ 解决（一刀劈成两半）
- 留白度：中度留白——给数据，不给结论，读者自己感受震撼
- 对话/叙述穿插：大段对话带微动作（"双眸放光""兴奋连道"）

## 二、叙事策略

### 2.1 时间流速

```
[x] 正常流速——1章≈半天到一天
```

时间处理：顺序叙事

### 2.2 信息释放模式

```
[x] 均匀撒放——信息持续不断地穿插在每段中
[x] 对话揭示型——通过角色对话和独白双重释出
```

信息密度：中（每章 3-5 个新概念，逐步展开世界观）

参考锚定章：对话场景 → 通过对话释放信息的方式

### 2.3 叙事声音侵入度

```
[x] 低侵入——偶尔有"现如今""此刻"类引导
```

### 2.4 开篇模式

```
[x] 场景铺陈——先给环境/时间/地点再推进剧情
```

> 典型开篇：先描写时间+场景+氛围，再引入人物和对话。

### 2.5 收束模式

```
[x] 情绪总结——给读者"喘口气"的余地再结束
[x] 对话收尾——以一句有分量的话或日常对话结束
```

### 2.6 情绪节奏模式

```
[x] 多峰型——每 1200-1500 字一个小起伏
```

### 2.7 余白密度

```
[x] 中度留白——战斗和参数写满，日常和情感留有余白
```

参考锚定章：悬疑场景 → 留白处理方式

## 四、红线清单

### 4.1 通用红线

```
□ 否定句结构 → 0容忍
□ "他感到/他意识到/他觉得/他评估"类前缀 → 0容忍
```

### 4.2 本风格特有红线

```
□ 超过 60 字的句子 → 标记退回
□ 首次出现术语不带解释 → 标记
□ 连续 3 段纯环境描写 → 标记
□ 战斗场景无精确数字 → 标记（风格特色缺失）
□ 连续 3 章无技术参数/数据 → 标记
□ 内心独白无实际行动衔接 → 标记
```

### 4.3 红线触发后的动作

```
[x] 标记项 ≥ 3 → 退回
```
