# 文风参考 · tunshi（v3.0）

> 以《吞噬星空》（我吃西红柿）为样本提取的文风参考。科幻升级流。
> 在 `project.yaml` 中配置 `writing_style: tunshi` 启用。
>
> **适用场景**：科幻/末世/升级流、起点长篇慢热、技术参数向
> **不适合**：纯西幻、古典仙侠、日更快节奏（字数偏高）

---

## 一、锚定章片段

> 以下原文摘自《吞噬星空》第 1-2 章，作为本条文风的具体实例。
> 每个片段后附 **叙事哲学印证**，标记该片段所体现的叙事哲学原则编号。

### [战斗·空中危机]

**来源章节**：第 1 章

```
一道刺耳之际的鸣叫声猛然响起，雷的声音很大，震耳欲聋！而这高亢的鸣叫声却是刺耳，
罗峰只感到耳朵鼓膜都是隐隐疼痛，不由有些难受皱起眉头……
高亢刺耳的鸣叫声音震荡下，旁边街道的一座摩天大楼上一块块巨大的玻璃，发出低沉的
"卡——卡——"声音，不少玻璃龟裂开来，甚至于当场有十多块玻璃从高空掉落了下来……

一块玻璃碎片，仿佛刀片射向罗峰。
罗峰目光一扫，并没有闪躲，而是平静站在原地，右手闪电般伸出，瞬间抓出这迸射过来的
碎玻璃。这玻璃碎块隐约映照出罗峰的样子，掂了两下，随手一扔，碎玻璃仿佛暗器一样
直接射向远处的垃圾箱，哐当一声，准确的从垃圾箱口射进。
```

**叙事哲学印证**：
- **[原则 2：叙事者姿态]** 低侵入度叙事，POV紧贴罗峰感官（鼓膜疼痛、目光一扫、掂了两下），叙事者隐身为摄像机。
- **[原则 3：情感表达]** 情感通过动作传递——不写"罗峰很紧张"，而是通过"耳朵鼓膜疼痛""皱眉""平静站在原地""闪电般伸手"让读者自行感受。
- **[原则 5：张力-释放]** 战斗场景由精确行为链驱动（鸣叫→玻璃碎裂→碎玻璃射来→抓取→扔掉），节奏紧凑无赘余。

### [对话·校门口]

**来源章节**：第 1 章

```
"罗峰师兄！罗峰师兄！"一道粗厚的声音响起。
"阿峰，有人喊你。"

罗峰疑惑转头看去，喊他的是一名男生，身高大概有一米九，长的虎背熊腰。
"你是？"
"罗峰师兄，我，我有事想请罗峰师兄帮忙。"壮硕男忐忑道。
"什么事？"罗峰一笑，说道。
"我练拳的时候，出拳发劲，总是感觉不对，不知道师兄你有没有时间指点一下。"
"哦，是这样……"罗峰略微停顿一下，点点头，"这样吧，这个星期的星期天下午，
在武馆的时候你找我。"
"谢师兄，谢师兄了。"
```

**叙事哲学印证**：
- **[原则 1：信息释放]** 对话承载信息推进：身份确认→求助原因→具体问题→解决方案→致谢，信息层层递进。
- **[原则 4：对话策略]** 每句对话带引导词（"忐忑道""一笑""略微停顿"），每2-3轮中断一次用于动作/神态，独白为零，完全通过外部行为传递互动。
- **[原则 3：情感表达]** 情感靠对话语气和微动作传达（壮硕男的"忐忑"、罗峰的"一笑"），不写内心想法。

### [悬疑·黑冠金雕揭秘]

**来源章节**：第 1 章

```
"黑冠金雕，在雕类怪兽中排第三。"魏文双眸放光，低沉道，"成年体的黑冠金雕，
体长一般达到21米，翼展在36米左右，飞行时极速可以达到3.9马赫。那可是足足
3.9倍音速。音速按照一秒340米计算的话，就是一秒1326米，也就是一小时4774公里。"

罗峰知道黑冠金雕厉害，可是听到一秒1326米的极限速度，依旧感到心中屏息。
一秒钟，也就是眼睛一眨啊。就一千米开外了。

"这黑冠金雕，羽毛比金刚石还要坚硬，应该达到三级克罗合金的硬度。"魏文兴奋连道，
"网上可是有视频的，黑冠金雕曾经随怪兽群冲击过军队，遭受到口径20mm的火神炮轰击，
火神炮可是一分钟7000发子弹……可是……竟然连一片羽毛都没轰击掉下来。"
"后来，还是一名神秘武者强者，手持一柄克罗合金战刀，化作一道流光，将黑冠金雕
一刀劈成两半！"
```

**叙事哲学印证**：
- **[原则 1：信息释放]** 经典"数据揭示型"信息释放——精确参数（3.9马赫/21米/20mm火神炮）→ 对比参照（火神炮打不掉羽毛）→ 高潮解决（一刀劈成两半）。不给结论，读者自行感受震撼。
- **[原则 4：对话策略]** 大段信息型对话配微动作（"双眸放光""兴奋连道"），引导词使信息对话不显枯燥。
- **[原则 5：张力-释放]** 数据层层递进（排第三→尺寸→速度→硬度→武器→强者一刀），张力逐步攀升至爆发点，形成完整释放弧线。

## 二、叙事哲学 DNA

> 核心哲学：**硬核升级流**——精确参数、日常渗入世界观、冷静客观叙事、数据辅助真实感。

### 原则 1：信息释放哲学

**信念陈述**：
信息即力量，释放节奏即叙事节奏。精确参数和世界观信息是吞噬风格的核心驱动力，信息的层层递进就是读者的期待曲线。不给结论，让数据自己说话。

**应用规则**：
1. 每章释放 3-5 个新概念/技术参数，均匀穿插于全章
2. 优先通过对话释放信息（对话揭示型），次选叙述穿插
3. 信息释放节奏遵循"参数→对比参照→高潮解决"三段式
4. 首次出现的术语必须附带解释（字数优先通过角色之口完成）
5. 连续 3 章无技术参数/数据 → 标记退回

---

### 原则 2：叙事者姿态哲学

**信念陈述**：
叙事者隐身为摄像机，冷静客观记录所见所闻。低侵入度、高透明度，让读者仿佛亲眼目睹而非被动接受讲述。叙事者的存在感降至最低，一切交由画面和行动说话。

**应用规则**：
1. 叙事声音低侵入——偶尔使用"此刻""现如今"类引导，不做主观评述
2. POV紧贴主角感官——只写主角看到、听到、触到的内容
3. 开篇采用场景铺陈模式：先给环境/时间/地点氛围，再引入人物和行动
4. 句长控制在15-30字（中句），动作处缩短以提速
5. 避免"他感到/他意识到/他觉得"类前缀 → 0容忍

---

### 原则 3：情感表达哲学

**信念陈述**：
情感通过动作和数据传递，而非内心独白。不写"他很愤怒"，写他一拳打碎墙壁；不写"他很震撼"，写精确参数给他的冲击。情感是行动和参数的副产品，而非直接描写对象。

**应用规则**：
1. 心理辅助型表达——动作在前、心理在后，先写行为再一笔带过感受
2. 情感高潮集中通过战斗场面和参数对比来爆发（火神炮打不掉羽毛→强者一刀劈开）
3. 修辞克制——偶尔使用"仿佛""如"类比喻（每章≤3处），不滥用修辞堆砌
4. 结尾采用情绪总结或对话收尾，给读者喘口气的余地再结束
5. 内心独白必须有实际行动衔接 → 无行动衔接则标记退回

---

### 原则 4：对话策略哲学

**信念陈述**：
对话是信息传递和人物塑造的双重载体。吞噬风格的对话始终承载双重功能：推进信息 + 刻画人物。对话本身即是动作，不写无信息量的闲聊，每句对话必须有推进意义。

**应用规则**：
1. 每句对话带引导词（含有动作/神情的引导：如"低沉道""双眸放光""一笑"）
2. 对话每 2-3 轮用动作或神态描写中断一次，避免纯对话段落
3. 独白比例极低——只有动作和对话，几乎不写内心想法
4. 对话/Dialogue比控制在均衡型——对话和叙述各半
5. 对话推进维度以信息推进为主（问请求→问原因→给答复），情感推进为辅

---

### 原则 5：张力-释放哲学

**信念陈述**：
张力的构建依赖精确参数和节奏控制。吞噬风格的爽感来自于数据层面的碾压感，而非情绪层面的煽动。多峰型节奏持续给读者产生小高潮，参数精确度越高，战斗场景的真实感和张力越强。

**应用规则**：
1. 多峰型节奏——每 1200-1500 字设置一个小起伏，保持读者持续兴奋
2. 战斗场景必须包含精确数字支撑（武器参数、速度、力量等级、攻击距离等）
3. 中度留白——战斗和参数写满（信息密度高），日常和情感留有余白
4. 收束方式二选一：情绪总结（让读者喘息）或对话收尾（以有分量的话结束）
5. 超过 60 字的句子 → 标记退回；连续 3 段纯环境描写 → 标记退回

## 三、红线清单

### 3.1 通用红线

```
□ 否定句结构 → 0容忍
□ "他感到/他意识到/他觉得/他评估"类前缀 → 0容忍
```

### 3.2 本风格特有红线

```
□ 超过 60 字的句子 → 标记退回
□ 首次出现术语不带解释 → 标记
□ 连续 3 段纯环境描写 → 标记
□ 战斗场景无精确数字 → 标记（风格特色缺失）
□ 连续 3 章无技术参数/数据 → 标记
□ 内心独白无实际行动衔接 → 标记
```

### 3.3 红线触发后的动作

```
[x] 标记项 ≥ 3 → 退回
```
