# 文风参考 · yazhou（v3.0 — 叙事哲学DNA）

> 以《海贼法典》为样本提取的叙事哲学DNA。同人·日漫风·轻小说化叙述。
> 在 `project.yaml` 中配置 `writing_style: yazhou` 启用。
>
> **适用场景**：同人/二次元、穿越、日漫世界框架、轻小说节奏
> **不适合**：古典仙侠、严肃历史、出版级文学追求
>
> **取样范围**：CH1（覆盖氛围/内心独白/世界认知）
> **取样章节数**：1章
> **叙事者侵入频率**：约 2次/1000字
> **对话占比范围**：20%-30%（含内心OS）

---

## 一、锚定章片段

> 以下原文摘自《海贼法典》CH1，作为本DNA的具体实例。

### [氛围·海贼镇夜袭]

**来源章节**：第 1 章

```
海圆历1506年，东海。

小镇平静安宁的生活被一伙外来者轻易撕碎，码头停泊着一艘挂着骷髅旗的
残破多桅帆船，在夜色的笼罩下，海贼船随着海浪起起伏伏，如同拥有生命的
海怪，它宁静的注视着充斥着厮杀与怒吼的小镇，一如往昔，或许早已习惯。

小镇外围已经到处都是海贼进攻过的痕迹，一些因战斗而引燃的零星火光照亮
街道，一行海贼脚步匆匆。
```

**微观质感标注**：
- 句长：中句（15-35字），画面感强
- 描写密度：中——视觉+氛围，有"电影分镜"感
- 对话：未出现，但叙述有叙事者腔调
- POV：**中距离**——全知视角，可以写海贼内心的嘲笑
- 段长：中段（3-5句）
- 修辞：拟人——"如同拥有生命的海怪""注视着小镇"
- **叙事哲学印证**：原则1（探索发现型信息释放）+ 原则5（震荡型节奏建立）

---

### [内心·穿越者独白]

**来源章节**：第 1 章

```
在21世纪的地球，他叫孙博，是一个白领，而且用网上的话来讲，就是即将
当上CEO，迎娶白富美，走上人生巅峰，但是人生就是那么奇妙，在即将当上CEO
的路上，赢取白富美的途中，走上人生巅峰的半道儿，倒霉的孙博被命运突如其来
的给了一次肾击，于是他悲剧了。

没有任何预兆和由头，只是在办公室里小憩了一会儿，他就穿越了，而且还加入了
一个任何在稳定社会成长生活的人，都会觉得毫无前途的行业——海贼。
```

**微观质感标注**：
- 引导词密度：无对话，全内心独白
- 独白比例：**高**——大量穿越者的认知/吐槽/对比
- 推进维度：**情绪+信息**——穿越背景/对当前处境的态度
- 语言风格：**口语化+网络梗**——"CEO""肾击""悲剧了""毫无前途"
- **叙事哲学印证**：原则3（吐槽外化情感）+ 原则4（口语化对话策略）

---

### [悬疑·世界认知]

**来源章节**：第 1 章

```
这种奇异的环境让谨慎的孙博丝毫不敢有暴露自己穿越者事实的念头，
谁知道这里的人会不会在发现他是穿越者后，把他绑到柱子上烧死。

不幸中的万幸是，这座身体原本主人的本能记忆都被他接受下来，如语言、
动作等，而其他记忆叠加在一起就仿佛一堆乱糟糟的照片，不光杂乱缺乏梳理，
情感上也让人有一些隔阂。

不过，这到不妨碍孙博鸠占鹊巢的使用他的名字——摩根.威廉。
```

**微观质感标注**：
- 感官密度：低——主要是心理活动
- 信息释放：**穿越设定揭示型**——直接告诉读者穿越规则
- 留白度：写满——不埋伏笔、不暗示，直接说
- 叙事声音：**低侵入**——有叙事者腔调（"不幸中的万幸""不得不说"）
- **叙事哲学印证**：原则1（写满式信息释放）+ 原则2（弹幕式叙事者介入）

## 二、叙事哲学 DNA（5条原则）

### 原则 1：信息释放哲学

**信念陈述**：

> "作者认为信息应该通过穿越者的认知/吐槽释放，不埋伏笔不隐藏，让读者跟着主角一起了解世界观。信息在同人小说中是认知对比的工具——穿越者用现代认知对比异世界，读者在差异中获得趣味。"

**证据链**（来自锚定章）：
- 内心·穿越者独白 → 穿越背景（21世纪白领/穿越方式）一口气倒出，不保留不铺垫
- 氛围·海贼镇夜袭 → 世界观通过全知视角直接展示（海圆历/东海/海贼船）
- 悬疑·世界认知 → "这座身体原本主人的本能记忆都被他接受下来"——穿越规则直接告知读者

**跨章验证**：
- CH1：信息传递方式——通过穿越者认知对比+叙事者宣告

**应用规则**（可执行指引）：
- 世界观信息 → 通过穿越者吐槽/认知对比自然释出，不做教材式说明
- 穿越设定 → 一次性释放（穿越方式/金手指规则），不保留
- 世界观关键信息 → 每章 3-4 个信息点，通过日常对话和内心OS自然流出
- 信息表达方式 → 口语化+现代网络比喻（"CEO""肾击""悲剧了"）

---

### 原则 2：叙事者姿态哲学

**信念陈述**：

> "叙事者是偶尔介入的弹幕式讲解者。作者在需要补充穿越设定或评价角色处境时跳出来对读者说话——'不幸中的万幸''不得不说''用网上的话来讲'等引导词是风格标记。"

**证据链**（来自锚定章）：
- 内心·穿越者独白 → "而且用网上的话来讲，就是即将当上CEO"——叙事者跳出来对读者说话
- 悬疑·世界认知 → "不幸中的万幸是""不过，这到不妨碍"——叙事者用总结性语句衔接
- 氛围·海贼镇夜袭 → "一如往昔，或许早已习惯"——叙事者借海贼船拟人做评价

**跨章验证**：
- CH1：叙事者侵入约 2次/1000字，特点是用口语化引导词

**应用规则**（可执行指引）：
- POV 距离默认模式 → **中距离**（全知视角，可补充角色内心外的信息）
- 叙事者侵入场景 → 允许在补充穿越背景、吐槽角色处境时出现
- 因果关系解释 → 由叙事者用口语化方式说明，不依赖读者推导
- 引导词风格 → 口语化（"用……的话来说""不得不说""然而""可惜"）

---

### 原则 3：情感表达哲学

**信念陈述**：

> "情绪主要通过吐槽和内心OS直接外化。角色对异世界的惊讶/无奈/兴奋全部通过吐槽表达——不是写'他很害怕'，而是写'谁知道这里的人会不会在发现他是穿越者后，把他绑到柱子上烧死'。"

**证据链**（来自锚定章）：
- 内心·穿越者独白 → "倒霉的孙博被命运突如其来的给了一次肾击，于是他悲剧了"——用吐槽代替悲伤
- 悬疑·世界认知 → "谁知道这里的人会不会在发现他是穿越者后，把他绑到柱子上烧死"——用自嘲代替恐惧
- 认知对比 → 穿越者用现代认知对比异世界时的反差感本身就是情绪表达

**跨章验证**：
- CH1：情感表达的主要手段=吐槽+内心OS+认知对比

**应用规则**（可执行指引）：
- 高强度情绪（惊讶/恐惧/无奈）→ 通过吐槽表达，不写直白情绪名词
- 悲伤/愤怒 → 用自嘲/讽刺的口吻写出（"于是他就悲剧了"）
- 兴奋/期待 → 用网络梗/现代比喻表达（"当上CEO，迎娶白富美，走上人生巅峰"）
- 连续 3 章无穿越者吐槽/内心对比 → 风格特征缺失，标记

---

### 原则 4：对话策略哲学

**信念陈述**：

> "对话和内心OS是双核心驱动。外部对话承担剧情推进和信息释放功能，内心OS承担情绪释放和吐槽功能。两者交替出现，呈日漫分镜式的跳跃感。"

**证据链**（来自锚定章）：
- 内心·穿越者独白 → 无外部对话，全内心独白推进
- 悬疑·世界认知 → 内心OS承担全部信息释放功能
- 全章估算：外部对话+内心OS合计占 40%-50%

**跨章验证**：
- CH1：外部对话较少，以内心OS为主

**应用规则**（可执行指引）：
- 每轮对话推进的维度 → 1-2 个（信息+情绪并进）
- 引导词使用频率 → 中等，偶用口语化引导
- 内心OS比例 → 高，穿越者持续用现代认知对比/吐槽异世界
- 对话风格 → 口语化，带日漫动画感（语气词多、短促、画面感强）
- 战斗需有"动画分镜"感的画面描写

---

### 原则 5：张力-释放哲学

**信念陈述**：

> "张力主要通过穿越者对未知世界的认知差建立，释放通过吐槽和认知更新完成。节奏呈震荡型——紧张场景→内心吐槽放松→紧张→放松，不让读者在一个情绪里待太久。铺垫与释放的比例约为1:1。"

**证据链**（来自锚定章）：
- 氛围·海贼镇夜袭 → 紧张的海贼袭击场景（铺垫）→ 全知视角提供背景信息（释放）
- 内心·穿越者独白 → 穿越者的困惑（铺垫）→ 吐槽化解紧张（释放）
- 悬疑·世界认知 → 被发现是穿越者的恐惧（铺垫）→ 获得记忆的庆幸（释放）

**跨章验证**：
- 整体旋律：**震荡型**——紧张场景↔内心吐槽放松交替
- 章内铺垫-释放比：紧张铺垫不超过整章的 60%

**应用规则**（可执行指引）：
- 章末钩子类型 → **悬念截断**（在发现/决策处截断）或 **内心总结**（以主角的一个结论/决定收束）
- 大爽点前的蓄力 → 不超过 1 章
- 连续张力章节上限 → 1 章（之后需要吐槽调剂）
- 开篇策略 → **场景铺陈**（先给场景/世界观再引入人物）或 **内心切入**（以穿越者认知/吐槽开头）
- 收束策略 → 内心总结（以主角的判断/决定收尾）

## 三、红线清单

### 3.1 通用红线

```
□ 否定句结构 → 0容忍
□ "他感到/他意识到/他觉得/他评估"类前缀 → 0容忍
```

### 3.2 本风格特有红线

```
□ 超过 60 字的句子 → 标记退回
□ 连续 3 章无穿越者吐槽/内心对比 → 标记（风格特征缺失）
□ 同人/二次元题材出现纯古典文风 → 标记
□ 战斗无画面感描写 → 标记（同人需要"动画分镜"感）
□ 日漫角色使用中国网络用词 → 标记
```

### 3.3 红线触发后的动作

```
[x] 标记项 ≥ 3 → 退回
```
