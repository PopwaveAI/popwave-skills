# 文风参考 · zhetian（v1.0）

> 以《遮天》（辰东）为样本提取的文风参考。古典仙侠，宏大叙事，文学化表达。
> 在 `project.yaml` 中配置 `writing_style: zhetian` 启用。
>
> **特征**：古籍引经据典、长句铺陈、意象堆叠、叙事声音明显。

---

## 一、锚定章参考模式

> 实际产出时替换为参考书对应场景的原文片段。

### [战斗] — 意象爆发型

**微观质感**：
- 句长：递进——先长后短，高潮处碎片短句
- 描写：高密度，用意象写战斗（"九龙拉棺""紫气东来"）
- 对话：中等引导，话中有机锋
- POV：中距离，可跳出外部写宏大场面
- 段长：混合，关键处用独立短行冲击

### [对话] — 机锋型

**微观质感**：
- 引导词密度：中等，对话带动作神态
- 独白比例：独白为主，大量内心戏
- 推进维度：信息+关系+情绪多维度推进

### [悬疑] — 埋伏笔型

**微观质感**：
- 感官密度：高，视觉+体感
- 信息释放：先给现象，隔数章再给出解释
- 留白度：技巧性留白——故意不写关键信息

## 二、叙事策略

### 2.1 时间流速

```
[x] 弹性——紧张场景放慢、过渡场景加速
```

时间处理：顺序叙事为主，偶尔大段背景倒叙

### 2.2 信息释放模式

```
[x] 埋伏笔型——先给现象，隔几章再给解释
[x] 探索发现型——通过主角探索/观察/思考发现
```

信息密度：中（每章 3-5 个），设定爆发章极高

参考锚定章：悬疑场景 → 预留伏笔和信息释放节奏

### 2.3 叙事声音侵入度

```
[x] 中侵入——叙事者偶尔点评/剧透（"谁也不知道，这将是……"）
```

### 2.4 开篇模式

```
[x] 场景铺陈——先给宏大的环境/历史/传说再引入人物
```

> 经典方式：建立宏大世界观画面（九龙拉棺/星空古路），再慢慢聚焦人物。

### 2.5 收束模式

```
[x] 悬念截断——在高潮中间截断
[x] 信息炸弹——最后一句放出重磅信息
[x] 画面静止——用宏大画面/意象收尾
```

### 2.6 情绪节奏模式

```
[x] 单峰型——铺垫大半章，最后集中释放
```

### 2.7 余白密度

```
[x] 技巧性留白——关键信息故意不写，让读者主动猜测
```

> 辰东的核心技法：有意的疑惑/未解之谜驱动追更。

参考锚定章：悬疑场景 → 留白方式

## 三、技法偏好

### 3.1 对话/叙述比

```
[x] 叙述为主：对话占 < 40%
```

### 3.2 心理-动作比

```
[x] 心理为主——大量内心独白驱动决策
```

### 3.3 过渡处理

```
[x] 场景总结——一段总结/过渡后进入新场景
```

### 3.4 金句倾向

```
[x] 密集——关键场景每段都想让人划线
```

金句类型偏好：哲理型、感叹型、意象型

### 3.5 数字使用习惯

```
[x] 模糊表达——"九""万""无尽""千百年来"
```

> 古典仙侠偏好象征性数字（九=极数，万=极多）。

## 四、红线清单

### 4.1 通用红线

```
□ 否定句结构 → 0容忍
□ "他感到/他意识到/他觉得/他评估"类前缀 → 0容忍
```

### 4.2 本风格特有红线

```
□ 超过 80 字的句子 → 标记退回
□ 连续 3 段无描写 → 标记
□ 章节无"意象锚定"（无标志性画面/器物/场景）→ 标记
□ 无宏大叙事视角的章节 → 标记
□ 连续 3 章无悬念钩子 → 标记
```

### 4.3 红线触发后的动作

```
[x] 标记项 ≥ 3 → 退回
[x] 标记项 ≥ 5 → 退回至 Director 阶段
```
