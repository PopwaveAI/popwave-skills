# 风格注入包 · zhetian（v1.0）

> 以《遮天》（辰东）为样本提取的文风。古典仙侠，宏大叙事，文学化表达。
> 在 `project.yaml` 中配置 `writing_style: zhetian` 启用。
>
> **特征**：古籍引经据典、长句铺陈、意象堆叠、叙事声音明显（有"说书人"腔调）、高潮处的"片段式断句"。

---

## 第1层 · 微观质感

### 1.1 句长分布

```
[x] 长句铺陈：平均 20-45 字，复杂从句多
```

**句长变化模式**：

```
[x] 递进型——先长后短（高潮处用短句碎片制造冲击感）
```

> 典型节奏：长段铺陈（世界观/氛围）→ 中间长句描述 → 高潮处用超短句反复（"大帝！""怎么可能！""不是……是……"）

### 1.2 描写密度与感官偏好

```
[x] 高密度——五感堆叠，环境描写独立成段
```

**感官偏好**：

```
[x] 视觉优先——极重色彩和光影描写（"九具龙尸""青铜古棺""紫气东来"）
[x] 体感（本体感）——偏重宏大场景带来的压迫感和震撼感
```

### 1.3 对话风格

```
[x] 中等引导——对话带动作和神态，但有时大段对话不加引导
```

**独白 vs 对白比例**：

```
[x] 独白为主——大量内心戏，对话作为点缀
```

> 辰东风格下角色经常有长段内心独白/回忆/思辨，对话反而相对克制。

### 1.4 POV 距离

```
[x] 中距离——第三人称限制视角，可描写主角的外在表现但不写内心
```

**POV 切换频率**：

```
[x] 每幕切换——幕与幕之间换视角角色
```

### 1.5 段长与分段模式

```
[x] 混合——按场景类型切换
```

**分段模式**：

```
[x] 碎段收束——段落前半部分长，后半部分越切越碎
```

> 关键时刻用独立短行制造冲击——一个词/名字独立成一段。

### 1.6 修辞偏好

```
[x] 密集——大量使用比喻/排比/拟人
```

**修辞类型偏好**：

```
[x] 比喻/拟人
[x] 排比/对仗——"不是……也不是……而是……"结构密集
```

### 1.7 词汇层级

```
[x] 文雅——用词考究，古典词汇与白话混合
```

**术语密度**：

```
[x] 高密度——术语是正文的识别特征，不解释
```

> 大量自创修炼术语/境界名称/法器名称，不解释读者自己体会。

## 第2层 · 叙事策略

### 2.1 时间流速

```
[x] 弹性——紧张场景放慢、过渡场景加速
```

**时间处理**：

```
[x] 顺序叙事——完全按时间顺序，但偶尔有大段背景倒叙
```

### 2.2 信息释放模式

```
[x] 埋伏笔型——先给现象，隔几章再给解释
[x] 探索发现型——信息通过主角探索/观察/思考发现
```

**信息密度**：

```
[x] 中密度——每章 3-5 个，但在设定爆发的章节密度极高
```

### 2.3 叙事声音侵入度

```
[x] 中侵入——叙事者偶尔点评/剧透/与读者对话（"谁也不知道，这将是……"）
```

### 2.4 开篇模式

```
[x] 场景铺陈——先给宏大的环境/历史/传说再引入人物
```

> 辰东经典开篇：先建立宏大世界观画面（九龙拉棺/星空古路），再慢慢聚焦到人物。

### 2.5 收束模式

```
[x] 悬念截断——在高潮/对话中间截断
[x] 信息炸弹——最后一句放出新的重磅信息
[x] 画面静止——用一个定格的宏大画面/意象收尾
```

### 2.6 情绪节奏模式

```
[x] 单峰型——铺垫大半章，最后集中释放
```

### 2.7 余白密度

```
[x] 技巧性留白——关键信息故意不写，让读者主动猜测
```

> 辰东擅长"有意的疑惑"——给读者留大量未解之谜追更。

## 第3层 · 技法偏好

### 3.1 对话/叙述比

```
[x] 叙述为主：对话占 < 40%
```

### 3.2 心理-动作比

```
[x] 心理为主——大量内心独白驱动决策
```

### 3.3 环境-剧情耦合度

```
[x] 强耦合——环境是叙事的参与者（"九龙拉棺""青铜古棺"等环境本身就是核心道具）
```

### 3.4 过渡处理

```
[x] 场景总结——一段总结/过渡后进入新场景
```

### 3.5 感叹号/问号密度

```
[x] 密集——关键场景每段都有，情绪高潮处密集使用
```

### 3.6 金句倾向

```
[x] 密集——关键场景每段都想让人划线
```

**金句类型偏好**：

```
[x] 哲理型——关于修炼/大道/命运的总结
[x] 感叹型——"这就是XX""原来如此"
[x] 意象型——用宏大意象做情绪锚定（九龙拉棺/北斗星域）
```

### 3.7 数字使用习惯

```
[x] 模糊表达——"九""万""无尽""千百年来"
```

> 古典仙侠偏好使用象征性数字（九=极数，万=极多），绝不用精确数值。

## 第4层 · 红线清单

### 4.1 句式红线

```
□ 否定句结构（不是A而是B / 不是A不是B是一种C）→ 0容忍（辰东虽然爱用，但网文通用红线）
□ "他感到/他意识到/他觉得/他评估"类前缀 → 0容忍
□ 超过 80 字的句子 → 标记退回
□ 连续 3 段无古文/典故引用 → 标记（风格特色缺失）
```

### 4.2 段落红线

```
□ 连续 3 段无描写 → 标记（遮天风格必须有描写密度）
□ 单段超过 12 句 → 标记
□ 章节无"意象锚定"（没有标志性的画面/器物/场景） → 标记
```

### 4.3 技法红线

```
□ "过渡段"不推进剧情也不推进情绪的段落 → 0容忍
□ 对话引导词连续 5 句不换 → 标记
□ 连续 3 章无悬念钩子 → 标记
□ 无宏大叙事视角的章节 → 标记（辰东风格需有"跳出来看全局"的时刻）
```

### 4.4 红线触发后的动作

```
[x] 标记项 ≥ 3 → 退回
[x] 标记项 ≥ 5 → 退回至 Director 阶段
```

## 场景类型 vs 写法映射

| 场景类型 | 句长 | 描写密度 | 段长 | 节奏 | 特殊要求 |
|:--------|:----|:--------|:----|:----|:--------|
| 战斗对抗 | 中长→短碎（递进） | 高→低 | 中→极短 | 慢→爆发 | 先写环境和气势，再用意象写战斗，不写具体招式 |
| 修炼/悟道 | 长句铺陈（30-50） | 极高 | 长（5-8句） | 慢 | 引用古籍典故，写心境变化，不写升级数据 |
| 悬疑揭示 | 中长句（20-40） | 高 | 中长（4-6句） | 缓急 | 先给现象，引发疑惑，再给解释。保留一定神秘感 |
| 对话博弈 | 中句（15-30） | 中 | 中（3-5句） | 中 | 对话带机锋，话中有话 |
| 场景过渡 | 中长句（20-35） | 高 | 中长（4-6句） | 慢 | 过渡也是描写——用景物变化代替"X天后" |
| 情感/回忆 | 长句（20-45） | 极高 | 长（5-8句） | 慢→缓 | 大量的追忆和内心感慨 |
| 获得机缘 | 中→短（递进） | 中→高 | 中→短 | 缓→快 | 先不说是啥，通过反应来反衬价值 |
