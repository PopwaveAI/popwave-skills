# 文风参考 · zhetian（v3.0）

> 以《遮天》（辰东）为样本提取的文风参考。古典仙侠，宏大叙事，文学化表达。
> 在 `project.yaml` 中配置 `writing_style: zhetian` 启用。
>
> **适用场景**：古典仙侠、宏大世界观、起点精品长篇
> **不适合**：日更短章节、对话驱动型、极简描写风格

**遮天风格的核心哲学** = 文学化叙事：意象先行、修辞密集、重要时刻拉长时间感、环境拟人化、内心层次深挖。

## 一、锚定章片段

> 以下原文摘自《遮天》第 1-2 章，作为本条文风的具体实例。

### [宏大·九龙拉棺]

**来源章节**：第 1 章

```
在黑暗的宇宙中，九具庞大的龙尸静静悬浮，让人感觉无尽的苍凉与久远——
九具龙尸皆长达百米，鳞片密布，乌光闪烁，仿佛刚从太空中打捞上来。
每具龙尸的尾端皆绑缚着粗大的黑色铁索，铁索的另一端连着一口长达二十米的
青铜巨棺。青铜棺椁古朴无华，上面刻满了模糊的古老图案——岁月的沧桑气息
扑面而来，也不知存在了多少万年。

九条百米长的龙尸，青铜巨棺被黑色铁索束缚，静静漂浮在枯寂的宇宙中。
这不是传说中的一幕吗？

国际空间站内的几名宇航员都惊呆了，他们看到了什么？
上帝，我看到了什么？
怎么可能，无法相信！
```

**微观质感标注**：
- 句长：递进——长铺陈（40-60字）→ 短句碎片（"九龙拉棺。""怎么可能！"）
- 描写密度：极高——视觉（龙尸/铁索/铜棺）+ 体感（苍凉/久远/扑面而来）
- 对话：口语化感叹句，与大段叙述形成反差
- POV：中距离——从宇宙全景视角切入
- 段长：长→碎（关键处独立短行）
- 修辞：密集——"乌光闪烁""古朴无华""岁月的沧桑气息"

**叙事哲学印证**：
- **信息释放哲学**：先给宏大视觉奇观（九具龙尸+青铜棺椁），不解释来源，留待后续章节揭露。
- **叙事者姿态哲学**：中侵入——"这不是传说中的一幕吗？"叙事者引导读者情绪；"也不知存在了多少万年"保持神秘感。
- **情感表达哲学**：意象先行——"苍凉""久远""沧桑"通过龙尸、铁索、铜棺的视觉描写自然折射，而非直抒胸臆。
- **对话策略哲学**：宇航员口语化感叹（"上帝，我看到了什么？"）与大段文学叙述形成强烈反差，增强冲击力。
- **张力-释放哲学**：从长镜头铺陈到短句爆发（"怎么可能，无法相信！"），张力积累后集中释放。

### [氛围·稷下学宫梧桐]

**来源章节**：第 2 章

```
学宫内栽有几株梧桐，摇摇曳曳，茂密的枝叶伸展向天穹，投下的绿荫在风中
很温和。叶子凡喜欢一边看书，一边喝清淡的绿茶，开始品读手中的古卷。
《黄帝内经》——一部宝典级的古籍，撰写于五千年前，为中国古代经典之一，
全书虽然仅有十几万字，但却有说不尽的价值。

他心中默诵原文，竟是越来越凝重。

因为这篇文字提到上古，出现了许多常人不应知晓的隐秘……难道历史上真的
有一段不为人知的古史？
```

**微观质感标注**：
- 句长：中长句（20-40字），节奏舒缓
- 描写密度：中——视觉（梧桐/绿荫）+ 体感（温和/凝重）
- 叙事声音：低侵入——"竟""难道"等叙事者引导
- POV：中距离，叙事者在角色和读者之间

**叙事哲学印证**：
- **信息释放哲学**：通过主角读书自然引出上古隐秘——探索发现型信息释放，读者跟随主角一同发现线索。
- **叙事者姿态哲学**：低侵入但存在——"竟是越来越凝重""难道"等措辞，叙事者在关键节点轻推读者注意力。
- **情感表达哲学**：情感通过环境折射——"梧桐摇摇曳曳""绿荫很温和"的舒缓氛围与"越来越凝重"的内心变化形成反差，无需直白说"他感到不安"。
- **对话策略哲学**：此段无直接对话——以内心默诵和叙事思考取代对话，保持叙述主导地位。
- **张力-释放哲学**：从宁静阅读到"难道"的疑问递进，构建轻声悬念，为下文铺垫。

### [悬疑·青铜棺椁]

**来源章节**：第 1 章

```
九龙拉棺——传说中上古天帝出巡的景象？

谁也不知道，这九具龙尸究竟从何而来，它们经历了多少万年的岁月漂流，
在宇宙的黑暗中无声前行。而那具青铜棺椁之中，又埋葬着怎样的存在？
是传说中的仙？还是超越仙的存在？

在九具龙尸之间，在黑暗的衬托下，青铜棺椁上的古老文字显得愈发神秘。
那一笔一划之间，仿佛蕴含着某种古老的力量，让人无法移开目光。
```

**微观质感标注**：
- 感官密度：高——视觉（黑暗/青铜/文字）+ 体感（神秘/强大）
- 信息释放：**埋伏笔型**——先给现象（九龙拉棺），引发疑问（是谁？从何而来？），不给解释
- 留白度：**技巧性留白**——关键信息故意不写，留给读者猜测
- 修辞：排比式提问密集——"谁也不知道""究竟""还是""又"

**叙事哲学印证**：
- **信息释放哲学**：纯粹的埋伏笔型——连续抛出一连串问题（从何而来？埋葬着谁？仙？超越仙？），全部留白，驱动读者追更欲望。
- **叙事者姿态哲学**：中侵入最大化——"谁也不知道"直接表明叙事者知道但不告诉读者，"是传说中的仙？还是超越仙的存在？"叙事者替读者提问。
- **情感表达哲学**：神秘感的营造完全依赖意象——"黑暗的宇宙""古老的文字""古老的力量"，没有抽象的情绪形容词，全部通过物质细节传递。
- **对话策略哲学**：零对话——但排比式提问（"是……？还是……？"）在功能上替代了对话，构成叙事者与读者之间的隐性问答。
- **张力-释放哲学**：以疑问层层叠加张力，最终以"让人无法移开目光"的画面静止收束，不给答案——悬念截断+画面静止复合收束模式。

## 二、叙事哲学 DNA

### 原则 1：信息释放哲学

**信念陈述**：
遮天式的阅读快感来自"谜"而非"答"。先给现象引发好奇，经过数章甚至数十章的铺垫后方才揭晓答案。信息就像剥洋葱——每一层揭开会发现还有下一层。读者不是在读故事，而是在解谜。

**应用规则**：
1. 每章设置 3-5 个信息点/暗示，其中至少 1 个为核心伏笔（"坑"）
2. 关键设定禁止直白说明，必须通过主角探索/观察/战斗/对话自然引出
3. 使用"三重递进"：现象级信息（视觉震惊）→ 疑问（叙事者或角色发问）→ 留白（不解释）
4. 伏笔回收周期：小节内回收 < 章内回收 < 卷内回收 < 跨卷回收，至少有 1 个跨卷坑在持续
5. 设定爆发章可信息密度极高（每段都有新信息），但爆发后必须紧跟一段"沉淀段落"让读者消化

### 原则 2：叙事者姿态哲学

**信念陈述**：
叙事者应该像一位说书人——有自己的存在感，但始终服务于故事。在宏大叙事中，叙事者偶尔露面对读者耳语（"谁也不知道，这将是……"），能极大增强代入感和共情感。关键是要把握好"存在感"和"侵入度"的平衡：太弱则叙述平淡，太强则打断沉浸。

**应用规则**：
1. 使用中侵入度叙事者，关键场景允许"点评式插入"（"竟""难道""谁也不知道"）
2. 禁止第一人称内心独白——所有心理活动通过行动或叙事者转述呈现
3. 保持中距离 POV：视角在角色肩后，既接近角色感受又保留全景视野
4. 高频使用"文学化定格"——在重要意象出现时，叙事者暂停叙事，使用 1-2 句评论性/抒情性句子锁定画面（如"岁月的沧桑气息扑面而来"）
5. 每章至少 1 次叙事者介入，但单次不超过 2 句，全章不超过 5 次

### 原则 3：情感表达哲学

**信念陈述**：
情感的最高表达是不说"情感"二字。遮天的文学化表达核心在于"意象先行"——先让读者看到、听到、感受到环境，情感自然从意象中折射出来。悲伤不写悲伤，写"枯寂的宇宙"；凝重不写凝重，写"梧桐在风中沉默"。环境即心境。

**应用规则**：
1. 情感场景必须由环境/意象开场——至少 2 句描写再进入角色反应
2. 每章必须有 1 个"意象锚定"：一个标志性画面/器物/场景作为本章情感的视觉载体
3. 禁止直白情感词汇：不用"悲伤/愤怒/恐惧/激动"，改用具象动作或环境折射
4. 重要情感时刻使用"时间拉伸"——放慢叙事节奏，增加描写密度，每 100 字处理 1 秒内的情感冲击
5. 内心层次处理：使用"表象动作 → 环境暗示 → 叙事者点破"三层递进，避免"他感到/他意识到/他觉得"类前缀

### 原则 4：对话策略哲学

**信念陈述**：
在文学化叙事中，对话是调味料而非主食。遮天的对话特征鲜明——出现频率低（每章 2-4 段），但每段都是高密度信息或高浓度情感。对话不是用来推进情节的（那是叙述的任务），而是用来定调性的——一句"上帝，我看到了什么？"胜过三页景物描写。

**应用规则**：
1. 全章对话比例控制在 30% 以下（推荐 15%-25%）
2. 角色对话使用口语化、感叹句风格，与叙述的文学化语体形成反差
3. 对话前后必须有动作/环境描写承载——禁止对话孤悬（"XX说"后无任何描写）
4. 关键信息优先通过叙述释放而非对话——对话只用来表达角色情绪或性格
5. 长篇对话（超过 3 轮）中间必须插入至少 1 句环境描写或动作描写打断，避免"话剧式对话"

### 原则 5：张力-释放哲学

**信念陈述**：
追更的核心驱动力是"想知道然后呢"。遮天的章节采用单峰型情绪曲线——前 70%-80% 的篇幅是张力积累（铺陈+埋伏笔+悬念叠加），最后 20%-30% 集中释放（高潮+信息炸弹+悬念截断）。每一章的结尾都在告诉读者：最精彩的部分，在下一章。

**应用规则**：
1. 每章采用单峰型情绪曲线：铺垫缓慢上升（70% 篇幅）→ 接近章节末端快速拉升（20% 篇幅）→ 截断/释放（10% 篇幅）
2. 章节结尾三选一：悬念截断（高潮中间断开）| 信息炸弹（最后一句放出重磅信息）| 画面静止（宏大意象收尾）
3. 每章最后 1-3 句必须承担"悬念钩子"功能——不符合则标记退回
4. 连续 3 章必须有新的悬念钩子（不能连续依赖同一个悬念），否则标记
5. "静→动→静"结构：每章以平缓环境/日常开场 → 中间战斗/冲突/发现 → 结尾以意象定格或悬念截断收束

## 三、红线清单

### 3.1 通用红线

```
□ 否定句结构 → 0容忍
□ "他感到/他意识到/他觉得/他评估"类前缀 → 0容忍
```

### 3.2 本风格特有红线

```
□ 超过 80 字的句子 → 标记退回
□ 连续 3 段无描写 → 标记
□ 章节无"意象锚定"（无标志性画面/器物/场景）→ 标记
□ 无宏大叙事视角的章节 → 标记
□ 连续 3 章无悬念钩子 → 标记
```

### 3.3 红线触发后的动作

```
□ 标记项 ≥ 3 → 退回
□ 标记项 ≥ 5 → 退回至 Director 阶段
```
