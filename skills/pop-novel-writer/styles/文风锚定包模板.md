# 文风锚定包模板（v2.0）

> 这是开书阶段（bootstrap）产出「文风锚定包」的模板。
> 合并锚定章（具体实例）和抽象规则（叙事策略+技法偏好）为一个文件。
>
> **第1层·微观质感不单独定义**——由锚定章片段直接提供实例。
> **第2-3层**的每一条规则后面标注 `参考：{锚定章编号}`，将抽象规则和具体实例绑定。
>
> **产出位置**：`01-写作资产/文风锚定包.md`
>
> **bootstrap 提取流程**：
> ```
> 用户选参考书 → bootstrap 读前 3 章
>     ↓
> 从原文摘录战斗/对话/悬疑三类片段（各 300-400 字）
>     ↓
> 标注每段微观质感（句长/描写/对话/POV/段长/修辞）
>     ↓
> 分析全文提取叙事策略 + 技法偏好（第2-3层）
>     ↓
> 填入此模板 → 产出 01-写作资产/文风锚定包.md
>     ↓
> ESM before 阶段读取 → 作为第10项注入 Pass 2 渲染
> ```
>
> **降级兜底**：如果 bootstrap 未执行或未产出 → styles/default.md

## 二、叙事策略

每条规则标注参考的锚定章编号。

### 2.1 时间流速

```
[ ] 慢镜头——叙事时间≈故事时间，1章≈几分钟
[ ] 正常流速——1章≈半天到一天
[ ] 快速推进——1章≈2-7天
[ ] 跳跃式——章间可跳数天到数月
[ ] 弹性——紧张场景放慢、过渡场景加速
```

参考锚定章：{战斗·{场景名}} → 场景时间与叙事时间的比例
参考锚定章：{过渡·{场景名}} → 加速节奏时的跳转方式

### 2.2 信息释放模式

```
[ ] 集中爆发——每章集中释放 1-2 个重磅信息
[ ] 均匀撒放——信息持续不断地穿插在每段中
[ ] 埋伏笔型——先给现象，隔几章再给解释
[ ] 对话揭示型——信息主要通过角色对话释出
[ ] 探索发现型——信息通过主角探索/观察发现
```

信息密度：{低/中/高/可变}

参考锚定章：{悬疑·{场景名}} → 信息释放的速度和节奏
参考锚定章：{对话·{场景名}} → 通过对话推进信息的方式

### 2.3 叙事声音侵入度

```
[ ] 零侵入——纯故事，没有任何叙事者声音
[ ] 低侵入——偶尔有指引性语言
[ ] 中侵入——叙事者偶尔点评/剧透
[ ] 高侵入——叙述者是正文的一部分
```

### 2.4 开篇模式

```
[ ] 钩子先行——第一句话就是剧情推进或悬念
[ ] 场景铺陈——先给环境/时间/地点再推进
[ ] 对话切入——以对话开头
[ ] 内心独白——以角色内心活动开头
[ ] 结果前置——先写"发生了什么"再写"怎么到这一步的"
```

### 2.5 收束模式

```
[ ] 悬念截断——在高潮/对话中间截断
[ ] 情绪总结——给读者"喘口气"的余地再结束
[ ] 信息炸弹——最后一句放出新的重磅信息
[ ] 画面静止——用一个定格的画面/意象收尾
[ ] 对话收尾——以一句有分量的话结束
```

参考锚定章：{战斗·{场景名}} → 战斗收束方式
参考锚定章：{对话·{场景名}} → 对话截断方式

### 2.6 情绪节奏模式

```
[ ] 单峰型——铺垫大半章，最后集中释放
[ ] 多峰型——每 800-1000 字一个小起伏
[ ] 递进型——情绪从低到高持续爬升
[ ] 震荡型——紧张→放松高频交替
[ ] 平铺型——稳定的阅读节奏
```

参考锚定章：整体连读 3 章后判断

### 2.7 余白密度

```
[ ] 写满——心理/环境/解释全部写清楚
[ ] 中度留白——写关键细节，留给读者想象空间
[ ] 高度留白——只写动作和对话，心理和环境全靠读者脑补
[ ] 技巧性留白——关键信息故意不写，让读者主动猜测
```

参考锚定章：{悬疑/氛围·{场景名}} → 留白的处理方式

## 三、技法偏好

### 3.1 对话/叙述比

```
[ ] 对话驱动：对话占全文 60%+
[ ] 均衡型：对话和叙述各半
[ ] 叙述为主：对话占 < 40%
[ ] 全叙述：几乎零对话
```

参考锚定章：{对话·{场景名}} 中的对话占比

### 3.2 心理-动作比

```
[ ] 动作驱动——写做不写想，读者通过行为理解角色
[ ] 心理辅助——动作在前，心理在后
[ ] 心理为主——大量内心独白驱动决策
[ ] 纯心理——极简外部动作，以内心活动为主线
```

参考锚定章：{战斗·{场景名}} → 战斗中的心理-动作关系

### 3.3 过渡处理

```
[ ] 硬切——直接进入新场景，不用过渡段
[ ] 空行分隔——用"---"或空行分段
[ ] 时间标记——"三小时后""第二天"
[ ] 场景总结——一段总结/过渡后进入新场景
[ ] 情绪桥接——用上章结尾的情绪对接下章开头
```

参考锚定章：{过渡/推进·{场景名}}

### 3.4 金句倾向

```
[ ] 不写金句——叙事完全平实
[ ] 偶尔——每 2-3 章 1 个
[ ] 常规——每章 1-2 个金句
[ ] 密集——关键场景每段都想让人划线
```

金句类型偏好：
```
[ ] 哲理型
[ ] 感叹型
[ ] 反转型
[ ] 意象型
```

### 3.5 数字使用习惯

```
[ ] 精确数字——"17分钟后""300米""3.5秒"
[ ] 模糊表达——"一炷香的时间""大约一箭之地"
[ ] 混合——精确用于战斗/数据，模糊用于日常
[ ] 零数字——完全不用具体数字
```

## 四、红线清单

```

```
