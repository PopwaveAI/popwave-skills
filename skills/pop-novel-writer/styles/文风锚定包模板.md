# 文风锚定包模板（v3.0 — 叙事哲学DNA）

> 这是开书阶段（bootstrap）产出「文风锚定包」的模板。
> 合并锚定章（具体实例）和叙事哲学DNA（抽象原则）为一个文件。
>
> **三层框架（v3.0）的产出定位**：
> ```
> 此模板产出 → Layer 1（锚定章片段：微观质感实例）
>            → Layer 3（文风DNA：5条叙事哲学原则 + 应用规则）
> Layer 2（叙事策略指令）不由此模板产出，由 ESM before 从 DNA 动态生成
> ```
>
> **产出位置**：`01-写作资产/文风锚定包.md`
>
> **bootstrap 提取流程**：
> ```
> 用户选参考书（约3本） → bootstrap 读前 3-5 章
>     ↓
> 从原文摘录战斗/对话/悬疑三类片段（各 300-400 字）
>     ↓  ← 这步不变，保留微观质感实例
> 按模板第二部分分析 5 个叙事哲学维度的原文实际表现
>     ↓
> 填入此模板 → 产出 01-写作资产/文风锚定包.md
>     ↓
> ESM before 阶段读取 → 提取 DNA（Layer 3）+ 锚定章（Layer 1）
>                     → 动态生成 Layer 2（叙事策略指令）
> ```
>
> **降级兜底**：如果 bootstrap 未执行或未产出 → styles/default.md

## 二、叙事哲学 DNA（5条原则）

> 5条叙事哲学原则，从参考书中提取。
> 每条原则回答「作者的叙事信念是什么」——不回答「作者做了什么」，回答「作者为什么这么做」。
> 每条标注【印证锚定章】将抽象原则与具体实例绑定。

### 原则 1：信息释放哲学

**信念陈述**：[一句话描述作者对信息释放的根本态度]

> "作者认为信息应该{集中释放/均匀撒放/埋伏笔}，因为信息在本书中是{爽点的燃料/世界观的载体/悬念的工具/角色成长的刻度}。"

**证据链**（来自锚定章）：
- {战斗·场景名} → 信息在战斗前/中/后释放了什么？释放了多少？
- {悬疑·场景名} → 作者选择在什么时候隐藏/揭示关键信息？
- {过渡·场景名} → 章节间信息如何承接？是否在章末留信息炸弹？

**跨章验证**：该特征在CH{..}和CH{..}中一致出现？
- CH{..}：{一致/不一致，表现为}
- CH{..}：{一致/不一致，表现为}

**应用规则**（可执行指引）：
- 首次出现的设定概念 → 必须经过{实战展示/角色对话/叙事者说明/探索发现}之一
- 世界观关键信息 → 每章最多释放 {N} 个新概念
- 连续无新信息的章节上限 → {N} 章
- 信息释放后的读者预期管理 → {立即验证/留到后续/转化为悬念}

### 原则 2：叙事者姿态哲学

**信念陈述**：[叙事者与读者之间的距离关系]

> "叙事者是{隐身的观察者/偶尔介入的讲述者/主动引导的说书人}，因为作者希望读者在本书中{沉浸式体验/被引导认知/保持清醒的批判距离}。"

**证据链**（来自锚定章）：
- {氛围·场景名} → 叙事者有没有跳出场景给解释/评价？频次如何？
- {战斗·场景名} → 战斗进行中叙事者是否介入？介入时说了什么？
- 原文平均每1000字中叙事者"跳出来"的次数：{N}次

**跨章验证**：
- CH{..}：叙事者侵入频次 {N}次/1000字
- CH{..}：叙事者侵入频次 {N}次/1000字

**应用规则**（可执行指引）：
- POV 距离默认模式 → {零距离/近距离/中距离}
- 叙事者侵入场景 → 仅允许在{开篇/卷末/情感高潮/不侵入}出现
- 因果关系解释 → {由叙事者主动说明 / 由读者通过场景自行推导}
- 情绪标签（"恐惧""坚强""他感到"）→ {允许使用/完全禁止/限制在{N}次/章}

### 原则 3：情感表达哲学

**信念陈述**：[情绪靠什么传达]

> "情绪主要通过{动作后果/感官堆叠/对话潜文本/内心独白}传达。作者{直接写情绪名词/避免写情绪名词/选择性地写}。"

**证据链**（来自锚定章）：
- {战斗·场景名} → 角色在战斗中如何表达恐惧/愤怒/决意？（写身体反应还是写"他很恐惧"？）
- {氛围·场景名} → 悲伤/温暖的情绪场景中，作者用了什么手段？
- {对话·场景名} → 对话中潜藏的情绪如何被读者感知？

**跨章验证**：
- CH{..}：情感表达的主要手段
- CH{..}：情感表达的主要手段

**应用规则**（可执行指引）：
- 高强度情绪（愤怒/恐惧/绝望）→ 写{身体反应/外部动作/环境投射}，不写情绪名词
- 动情时刻 → 环境{呼应/反衬}情绪，不单独说明
- 情绪转换 → 需要{至少1个外部事件/至少1句内心活动/直接切换}作为转折

**信念陈述**：[对话在小说中扮演的角色]

> "对话的主要功能是{推进信息/塑造关系/控制节奏/情绪释放}。作者倾向于{写实/精简/修辞化}的对话风格。"

**证据链**（来自锚定章）：

- 对话占正文比例估算：{N}%

**跨章验证**：
- CH{..}：对话占比 {N}%，主要功能
- CH{..}：对话占比 {N}%，主要功能
- CH{..}：对话占比 {N}%，主要功能

**应用规则**（可执行指引）：
- 每轮对话推进的信息维度上限 → {1/2/3}个
- 引导词使用频率 → {极简/偶用/每句必用}
- 独白比例 → {几乎零/少量/各半}
- 对话中的情感表达 → {直接通过台词/通过台词+动作/通过台词+环境}

### 原则 5：张力-释放哲学

**信念陈述**：[节奏控制的底层逻辑]

> "张力主要通过{时间压迫/信息差/力量差距/道德困境}建立，释放通过{实战验证/信息揭示/关系变化/代价展示}完成。铺垫与释放的比例约为 {N}:1。"

**证据链**（来自锚定章）：
- {战斗·场景名} → 战斗前铺垫了多少？战斗本身写了多少？张力建立和释放的路径？
- {悬疑·场景名} → 悬念的建立→维持→释放周期跨多少章？
- 连读3章后判断整幕的铺垫-释放节奏模式

**跨章验证**：
- 整体旋律：{单峰型/多峰型/递进型/震荡型/平铺型}
- 章内铺垫-释放比：铺垫不超过整章的 {N}%
- 连续张力章节上限：{N} 章

**应用规则**（可执行指引）：
- 章末钩子类型 → {悬念截断/信息炸弹/画面静止/情绪总结/对话收尾}
- 大爽点前的蓄力 → 需要{N}章的铺垫压缩
- 连续无爽点章节上限 → {N} 章
- 开篇策略 → {钩子先行/场景铺陈/对话切入/内心独白/结果前置}

## 三、红线清单

> 本条风格下绝对禁止的东西。每项标注参考锚定章来验证。

### 3.1 通用红线（全风格通用，不可修改）

```
□ 否定句结构（不是A而是B / 不是A不是B是一种C / 以"不是"开段）→ 0容忍
□ "他感到/他意识到/他觉得/他评估"类前缀 → 0容忍
```

```
