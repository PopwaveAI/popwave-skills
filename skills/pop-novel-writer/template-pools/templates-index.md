# 场景模板池索引

> 通用场景类型模板，只约束**形式结构**，不约束**题材内容**。
> 内容填充由项目 L0 + wenfeng-dna + L1 设定 + 参考书锚定决定。

---

## 模板列表

| # | 模板 | 适用场景 | 核心原则 |
|:---:|:---|:---|:---|
| 1 | [爽点释放](爽点释放.md) | 打脸/获得/升级/逆转 | 铺垫短，释放快，锚定长 |
| 2 | [战斗对抗](战斗对抗.md) | 物理战斗/规则对抗/智力博弈 | 不写招式描写，只写决策后果 |

| 5 | [悬疑揭示](悬疑揭示.md) | 信息释放/反转/伏笔回收 | 读者知道得比角色少，比角色想知道的多 |
| 6 | [压抑蓄力](压抑蓄力.md) | 情绪低谷/资源不足/受压 | 压抑是为释放购买的期票 |
| 7 | [轻松调剂](轻松调剂.md) | 节奏切换/角色魅力/暗线埋设 | 轻松不等于水字数 |

## 使用流程

```
导演判断 → 确定本章场景类型
  → 从 template-pools/ 选对应模板
    → 填充项目数据（DNA / L1 / 幕纲切片）
      → 生成场景结构 → 导演思考 → 正文生成
```

---

## 与项目的关系

```
_shared/template-pools/     ← 通用结构模板（版本管理）
         ↓ bootstrap 开书时复制
项目/01-写作资产/template-pools/  ← 项目级副本（可定制修改）
         ↓ 启动时加载
wenfeng-dna + L1 设定      ← 内容层（决定"这个题材怎么写"）
         ↓
正文生成
```

---

## 扩展规则

- 新增模板 → 更新索引文件 + 同步到系统区 `_shared/template-pools/`
- 项目级覆盖 → 在项目 `01-写作资产/template-pools/` 下放同名文件，优先级高于通用模板
- 版本号：`v1.0`（首版7个模板）
