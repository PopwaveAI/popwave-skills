# CH11 藏宝洞 · 遮天文风

琥珀城北区的贫民窟，一片低矮的石屋错落交织，巷陌间弥漫着潮湿腐朽的气息。索伦穿梭在狭窄的巷道中，脚下踏着积水和污泥混杂的路面，像是行走在一片泥泞的沼地深处。他已经在这里转悠了大半个时辰，从一个酒鬼口中零零碎碎地拼凑出了线索——玛瑙河以北一公里处，有片荒草丛生的乱葬岗。

那里，有人见过鬼火。

不止一个人声称，在夜深时分看到幽幽的蓝光在乱葬岗上空飘荡，恍若亡灵的眼睛在黑暗中注视。然而没有谁敢靠近那片区域——哪怕是琥珀城最不要命的醉汉，在说起那片荒地时都会压低嗓音，眼神中透出一抹惧色。索伦却心知肚明：死亡气息浓郁之地，正是巫妖藏宝洞的入口所在。这是这个世界隐秘的规则，每一座城市周边，都会遗留一座巫妖曾用过的工坊，而洞口往往就开在类似乱葬岗这样的地方。

他转身离开贫民窟时，心中已经有了计较。

晨曦神殿坐落在琥珀城东区偏南的位置，是一座不算宏伟但也干净整洁的二层石殿。白色的石墙在午后的阳光下泛着温润的光泽，殿顶竖着一枚银色的晨曦圣徽，反射出耀眼的光芒。索伦站在神殿门前，低头看了一眼身边的小姑娘——薇薇安正紧紧抓着他的衣角，一双清澈的眼睛望着他，满是依赖。

他不能带她去乱葬岗。

藏宝洞中可能存在机关、怪物，甚至真正的亡灵。那不是一个七岁女孩该去的地方。索伦深吸一口气，推开了神殿的橡木大门。

大殿内光线柔和，几排木质长椅整齐排列，最前方是一座朴素的祭台，上面燃着一团银白色的圣焰，散发出温暖的气息。一位中年牧师正站在祭台旁整理典籍，听到脚步声后抬起头来。他的面容平和，眼神却带着一种商人般的精明的审视——这就是费尔牧师，晨曦神殿的主持者。索伦没有绕弯子，直接说明了来意：他想将薇薇安寄养在神殿中几日，为此愿意奉献六十枚银德勒。

费尔牧师沉默了片刻，目光在索伦和薇薇安之间来回扫视。

"六十枚银德勒……"他缓缓重复了一遍这个数字，像是在掂量它的分量。片刻后，费尔牧师点了点头，脸上浮现出一丝务实的笑意。"神殿虽然清贫，但收留一个小姑娘几日的余裕还是有的。"他转身走到祭台下，弯腰取出一个陶瓶，递给索伦："这是经过晨曦祝福的圣水，对付亡灵有奇效。我看得出来，你要去的地方，可能会需要它。"

索伦接过陶瓶，入手温润，瓶中圣水微微晃动，像是感应到了什么。他没有多说什么，只是将圣水收入怀中，点了点头。

就在这时，侧门被推开了，一位年轻的见习牧师走了进来。她身披白色简袍，一头褐色短发利落地别在耳后，面容清秀，但看向索伦时眼神中明显带着一丝不耐——那是安亚丽。她不只一次帮过索伦，却也从不掩饰对这个盗贼的厌恶。然而当她看到薇薇安时，表情却柔和了下来。

"你来照顾她。"费尔牧师简短地吩咐道。

安亚丽轻轻哼了一声，到底没有拒绝。她走到薇薇安面前，蹲下身来，语气出乎意料地温柔："跟我来吧，小家伙。"

薇薇安抬起头，望向索伦。那双清澈的眼睛里没有眼泪，只有一种让人心疼的懂事——她知道索伦有事要做，知道不能哭闹。索伦蹲下身来，轻轻按了按她的肩膀。

"等我回来。"

薇薇安点了点头，然后乖乖地伸出手，握住了安亚丽的手掌。两人穿过侧门，消失在走廊的阴影中。索伦站起身，望着那道门缓缓合上，心中像是有什么东西沉了一下。

他走出晨曦神殿时，午后的阳光已经偏西，将他长长的影子投映在石阶上。

神殿门前的台阶两侧，站着两名全身铠甲的守卫——盔甲擦得锃亮，腰间佩剑，站姿笔挺如松，目光平视前方，一动不动，宛如两尊钢铁铸就的雕像。索伦看了他们一眼，心中忽然涌起一股复杂的思绪。

这就是二阶的神殿守护者。

他的脑海中不由浮现出前世的记忆——那时候他还在地球上，沉浸在各种游戏与数据中。对这个世界而言，前置职业的选择决定了最终成长的上限。一个平民晋升上来的二阶盗贼，与一个从骑士侍从晋升而来的二阶神殿守护者，综合属性差距大得惊人。同样的等级，同样的阶位，后者在力量、耐力、专长数量上都远胜前者，这是准军事训练与野路子成长之间不可逾越的鸿沟。

而他索伦，就是从最差的"平民"走过来的。

平民——没有任何初始技能，没有任何属性加成，唯一的好处就是获得了整整二十五个自由技能点，可以不受约束地选择任何发展方向。这就像是一张白纸，在上面画什么都可以，但前提是你得知道该画什么。

他走在街道上，脚下的石板路渐渐变为泥泞的小径。耳畔传来玛瑙河的水流声——哗哗的水声在傍晚的空气中格外清晰，空气里混合着河泥的腥味和荒草的苦涩气息。他穿过最后一片低矮的灌木丛，前方一片开阔，乱葬岗到了。

夕阳的余晖将这片荒地染成了一片苍茫的暗红色。大大小小的坟包高低起伏，荒草没过膝盖，其间散落着残破的石碑，有的已经歪倒，有的只剩下一截埋在土里。一种阴冷的气息从地底渗上来，无声无息地包裹着这片曾经的安息之地。

索伦站在乱葬岗的边缘，目光投向更深处。

他知道，那个藏宝洞就在前方某处。而那个连环剥皮者的威胁也已经迫在眉睫——混乱的时代即将降临，他必须尽快找到超凡装备，才有活下来的资本。

这是一个冒险者的决断。

而薇薇安还在等他回去。

乱葬岗深处，一阵阴风掠过，荒草沙沙作响。索伦握紧了怀中的陶瓶，迈步走了进去。
