# 游戏入侵流剧情模式

> 在通用剧情模式基础上叠加。游戏入侵赛道专属。
> 基于番茄Top20拆解（游戏入侵/我的诡异人生/时停起手/诸神愚戏等）补充实战技法。

---

## 赛道特征

| 维度 | 说明 |
|------|------|
| 世界观 | 现实世界被游戏规则入侵 + 面板/技能/副本/BOSS |
| 主角身份 | 普通人获得面板/重生者/职业玩家 |
| 核心爽点 | 规则利用/数值碾压/副本首杀/信息差优势 |
| 节奏偏好 | 事件密度极高（每章数值变化/新规则发现） |
| 冲突来源 | 规则博弈/阵营对抗/数值竞赛/信息差 |

## 爽点结构

| 爽点类型 | 游戏入侵特色实现 |
|---------|----------------|
| 打脸循环 | 被当新手/弱者→利用规则碾压→面板差距震惊 |
| 压力突破 | 副本BOSS碾压→发现规则漏洞→击杀 |
| 未知揭示 | 新规则/新副本/游戏真相/入侵原因 |
| 弱→强反转 | 低等级→隐藏职业/独特build→越级碾压 |

## 事件分量标准

- 改变命运：获得面板/转职/获得隐藏职业/加入/创建公会
- 打开可能：解锁新副本/新技能/新区域/新规则
- 制造追读：游戏真相/入侵原因/隐藏BOSS/其他重生者

## 低压区设计

- 被当新手轻视（等级低/装备差/没公会）
- 被数值碾压（等级差距到绝望）
- 副本机制碾压（不懂规则被团灭）
- 被公会/大势力针对（人多吃独食）

## 冲突来源

| 冲突类型 | 主角欲望 | 敌方欲望 | 不可调和原因 |
|---------|---------|---------|-------------|
| 规则博弈 | 想利用规则 | 规则设计者想限制 | 规则vs漏洞 |
| 阵营对抗 | 想自由行动 | 公会/阵营想控制 | 权力vs自由 |
| 数值竞赛 | 想变强 | 他人也想变强 | 资源有限/首杀唯一 |
| 信息差 | 想保持信息优势 | 他人想获取信息 | 信息=力量 |

## 特有节奏要求

- 事件密度极高（12-15个/章），每章有数值变化
- 面板/数值的即时奖励循环极其重要（行动→数值/技能/物质回报）
- 副本是爽点密集区（规则利用+BOSS战+首杀奖励）
- 倒计时/衰减/惩罚等压力机制贯穿（副本时限/世界BOSS刷新/入侵升级）

## 番茄Top20拆解补充

### 金手指巧思（游戏入侵流专项）

番茄Top20中游戏入侵/规则类小说的金手指设计规律：

| 小说 | 金手指 | 巧思维度 |
|------|--------|---------|
| 我的诡异人生 | 人生模拟器 | 决策困境（模拟中死=现实死）+ 自生长（每次模拟=一个事件链） |
| 时停起手 | 一分钟时间静止 | 使用博弈（什么时候用是战略决策）+ 风险代价（用完就没了） |
| 诸神愚戏 | 信仰命途系统 | 决策困境（选命途不可更改）+ 阵营对立（信仰对立互杀） |
| 游戏入侵(金刚明王功) | 重生+天赋抽取 | 成长节奏控制（s级唯一性驱动抢跑）+ 决策困境（删号重抽的成本） |

**核心发现**：游戏入侵流最容易犯的错误是金手指变成"纯翻译器"——面板只展示数据，不产生决策。需要给面板加入至少1个巧思维度：

1. **面板产生决策困境**：升级时不是自动分配属性，而是主角选择分配方向——不同build走向不同路线
2. **面板信息差扩展到社交**：面板不只在战斗中弹——社交场景中评估对方属性→判断对方是否在说谎→产生社交博弈
3. **面板使用有策略空间**：面板弹出有条件（需要专注/需要视线接触/需要目标在范围内），什么时候弹是决策
4. **面板信息可被误用**：正确的数据+错误的推理=灾难性决策——主角看到高属性就跑，结果对方是友军

### 危机多维（游戏入侵流专项）

游戏入侵流的危机叠加模式：

| 危机维度 | 游戏入侵流典型表现 |
|---------|------------------|
| 生存危机 | 副本死亡=现实死亡/世界BOSS刷新倒计时/入侵升级 |
| 情感危机 | 队友在副本中死亡/家人被卷入入侵事件 |
| 社会危机 | 有面板的秘密被发现/被公会追杀/被当成实验体 |
| 认知危机 | 游戏规则和现实不符/前世知识有误/发现世界是假的 |
| 资源危机 | 经验值不够/副本冷却/装备损坏/技能点不足 |

**核心发现**：游戏入侵流的"认知危机"是最独特的维度——主角以为世界是游戏，但世界有自己的运行逻辑。前世知识不是全知全能，20%错误率不只是数值设定，是认知危机的持续来源。

### 成长节奏基准（游戏入侵流）

番茄Top20的游戏入侵/规则类小说前3-5章成长节奏：

| 章 | 应完成的成长节点 | R20对比 |
|----|----------------|---------|
| ch001 | 觉醒+首杀+面板弹出 | ✓ 觉醒+首杀 |
| ch002 | 升级+新技能/新规则发现 | ✗ 还在1级刷怪 |
| ch003 | 越级击杀+质变升级+新区域解锁 | ✓ 越级击杀+升级 |
| ch004 | 新势力/新冲突出现 | ✓ 帮派对峙 |
| ch005 | 碾碎首个上位者+卷末质变 | ✓ 碾碎帮派底层 |

**R20差距**：ch002成长节点滞后——主角ch002还在刷同种怪，没有升级/新技能/新规则发现。成长速度需要加快，特别是前3章要保持"每章都有数值变化+新能力获得"。

### 设定涌现型燃料（游戏入侵流）

番茄Top20中效率最高的燃料全部是设定本身涌现的：

| 燃料 | 来源 | 自生长能力 |
|------|------|-----------|
| 信仰对立→阵营冲突 | 诸神愚戏设定 | 一条设定产生无限社交博弈 |
| 模拟器→预知死亡 | 我的诡异人生设定 | 每次模拟=一个完整事件链 |
| 一分钟时停→战略博弈 | 时停起手设定 | 每次使用都是独立决策事件 |
| 重生+天赋唯一性→抢跑竞赛 | 游戏入侵设定 | 倒计时驱动+竞争压力 |

**核心发现**：游戏入侵流的燃料不应该只来自"原作剧情模式借鉴"（外部燃料），应该优先从设定本身长出燃料（内部涌现）。面板/经验值/职业等级这些设定元素本身就是燃料来源——关键是设计让它们产生决策/博弈/冲突，而不只是展示数据。
