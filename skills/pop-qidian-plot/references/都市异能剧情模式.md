# 都市异能剧情模式

> 在通用剧情模式基础上叠加。都市异能赛道的专属剧情设计指导。

---

## 赛道特征

| 维度 | 说明 |
|------|------|
| 世界观 | 现代都市背景 + 超自然/灵能/异能体系 |
| 主角身份 | 普通人获得异能 / 隐藏身份的异能者 |
| 核心爽点 | 隐藏实力打脸 / 身份反转 / 社会面冲突中的碾压 |
| 节奏偏好 | 紧凑，每章有推进，不长弧蓄力 |
| 冲突来源 | 社会规则 vs 异能者 / 组织 vs 个人 / 隐藏身份 vs 暴露风险 |

## 爽点结构

| 爽点类型 | 都市异能特色实现 |
|---------|----------------|
| 打脸循环 | 被当普通人轻视→用异能碾压→旁观者震惊（社会面打脸比私下打脸更爽） |
| 压力突破 | 异能失控/被组织追杀→发现新用法→反杀 |
| 未知揭示 | 异能来源→组织真相→更大世界 |
| 弱→强反转 | 被高阶异能者碾压→觉醒/解锁→反杀 |

## 事件分量标准

都市异能赛道的事件分量判断：
- 改变命运：异能觉醒/被组织锁定/身份暴露/加入/逃离组织
- 打开可能：解锁新能力/接触新势力/进入新区域
- 制造追读：组织真相/异能来源/幕后黑手/身份即将暴露

## 低压区设计

都市异能的低压区更偏社会面：
- 被当普通人轻视（送外卖的/打零工的/无业游民）
- 被异能者碾压（等级差距明显到绝望）
- 身份暴露风险（异能在公共场合失控/被监控拍到）
- 组织追杀（被专业追杀者锁定，无处可逃）

## 冲突来源

| 冲突类型 | 主角欲望 | 敌方欲望 | 不可调和原因 |
|---------|---------|---------|-------------|
| 社会 vs 异能 | 想正常生活 | 社会排斥异能者 | 异能者不被接纳 |
| 组织 vs 个人 | 想自由 | 组织想控制 | 组织不允许脱离 |
| 隐藏 vs 暴露 | 想隐藏身份 | 敌方想揭穿 | 暴露=危险 |

## 特有节奏要求

- 事件密度偏高（10-12个/章），都市节奏快
- 日常喘息场景用都市元素（送外卖/坐地铁/吃路边摊）
- 社会面冲突优先（有围观群众的打脸比私下打脸更爽）
- 身份隐藏张力贯穿始终（每次用异能都有暴露风险）
