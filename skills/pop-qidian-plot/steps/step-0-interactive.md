# Step 0 - 交互式卷纲决策

> **v4.2.0新增**：在Step 1-3之前，通过5轮交互式决策采集用户偏好，产出`卷纲决策表.md`。
> **核心原则**：给选项不给空白——每轮提供2-3个结构化选项+1个自定义入口，不是开放式提问。
> **交互格式**：每轮Agent输出选项→暂停等待用户选择→确认后进入下一轮。

## 前置条件

- Phase 0-3.5全部完成
- 骨架.md + 主角设计.md + 全书设定/ + 角色库.md 齐全
- 如有research产出（写作燃料/decon-lite/赛道调研），作为选项素材池消费

## 产出

`设计/第一卷剧情/卷纲决策表.md`——后续Step 1-3的硬输入

## 5轮交互决策

### R1 卷终点定位（核心轮·必答）

**底牌消费**：骨架.md（阶位区间+引擎演化节点）+ 创意.md（全书方向）+ 全书设定/各卷切片.md

**Agent动作**：
1. 读取骨架.md，提取本卷可行的阶位区间
2. 读取引擎演化节点，标注本卷对应的节点
3. 生成3个终点方案，每个标注：阶位/大事件/坐标系跃迁/风险

**选项生成规则**：
- 方案A：标准路线（阶位跨度适中，风险均衡）
- 方案B：保守路线（阶位跨度小，风险低但爽感可能不足）
- 方案C：激进路线（阶位跨度大，爽感强但节奏密集）

**输出格式**：
```
## R1 卷终点定位

基于骨架.md，主角当前{当前阶位}，第一卷可行终点：

A) {方案A名}（{阶位}）
   → 卷末高潮：{大事件}
   → 坐标系跃迁：{阶位变化}
   → 风险：{风险描述}

B) {方案B名}（{阶位}）
   → 卷末高潮：{大事件}
   → 坐标系跃迁：{阶位变化}
   → 风险：{风险描述}

C) {方案C名}（{阶位}）
   → 卷末高潮：{大事件}
   → 坐标系跃迁：{阶位变化}
   → 风险：{风险描述}

请选择（A/B/C），或描述你想要的终点。
```

### R2 关底Boss（核心轮·必答）

**底牌消费**：角色库.md（反派卡）+ 危机.md（阶段性危机）+ 骨架.md（坐标系阶位）+ decon-plot Boss战设计（如有）

**Agent动作**：
1. 读取角色库，筛选本卷可用的反派（等级匹配+退场卷匹配）
2. 读取危机体系，标注本卷对应的阶段性危机
3. 生成3个Boss方案，每个标注：Boss名/强度/出场方式/与主角冲突点/击败方式

**选项生成规则**：
- 方案A：标准Boss（角色库中最匹配的反派）
- 方案B：组合Boss（两个反派联手或前后脚）
- 方案C：意外Boss（非角色库中的常规反派，如中立方/盟友反转）

### R3 核心爽感优先级（核心轮·必答）

**底牌消费**：主角设计.md（爽感矛盾公式）+ research-写作燃料.md（爽点素材池）+ decon-plot 爽感场景拆解（如有）

**Agent动作**：
1. 读取爽感公式，提取3个因子的可用组合
2. 读取写作燃料，提取可直接使用的爽点条目
3. 生成4种爽感类型选项，标注每种与坐标系/引擎/金手指的关联+燃料条目数

**选项生成规则**：
- 选项A-D对应4种爽感类型（面板爆发/信息差碾压/越级击杀/打脸反击）
- 用户需排优先级（选2-3个），不是单选

### R4 剧情节奏参考（可选轮·可跳过）

**底牌消费**：decon-lite表2（节奏）+ decon-plot 分幕转折手法（如有）+ 赛道调研.md（读者偏好）

**Agent动作**：
1. 读取decon-lite节奏表，提取参考书的节奏特征
2. 读取赛道调研，标注赛道主流节奏
3. 生成3个节奏方案，每个标注：参考书/节奏特征/分幕密度/风险

**跳过规则**：用户说"跳过"或"默认"→ agent按赛道主流节奏自动选择

### R5 NPC登场+分幕走向（可选轮·可跳过）

**底牌消费**：角色库.md（引入章/退场卷）+ 全书设定/地图.md（区域分布）+ 世界圣经.md（活跃层NPC）+ decon-plot NPC登场效果（如有）

**Agent动作**：
1. 读取角色库，提取本卷出场的角色清单+原定引入章
2. 读取地图，标注各区域适合的剧情场景
3. 生成2-3个NPC登场顺序+分幕走向方案

**跳过规则**：用户说"跳过"或"按原计划"→ agent按角色库原定引入章生成

## 红线

1. **每轮必须给选项**——不能开放式提问"你想要什么终点"，必须给2-3个结构化选项
2. **每轮必须有自定义入口**——选项末尾必须有"或描述你想要的XXX"
3. **选项必须在硬约束内**——不可行的方案不会出现在选项中
4. **核心轮不可跳过**——R1-R3必答，R4-R5可跳过
5. **决策表是Step 1-3的硬输入**——Step 1-3必须遵守决策表的全部决策
