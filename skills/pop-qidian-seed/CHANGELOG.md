# CHANGELOG

## v9.0.0 | 2026-07-23

### 引擎三要素重构

**核心改动**：基于"引擎三要素"（①定位②好坏标准③达成方法）重构S1/S2/S3。每个设计环节从"给字段填空"升级为"先明确定位→再用好坏标准判断→最后用达成方法稳定产出"。

### S1 世界构筑（引擎三要素）
- **定位显式化**：设计世界舞台——表世界+危机引擎+增量吸引力
- **好坏标准显式化**：好舞台3特征（读者想住进去/世界本身有张力/换主角同样精彩）vs 坏舞台3特征（只是角色载具/没主角就没矛盾/换主角讲不下去）
- **增量吸引力升级**：从"至少1个"升级为"至少3个"增量元素，新增主钩子概念（3个中明确1个全书最大差异化卖点）+4种有效组合模式（三重焊合型/骨架-灵魂-皮肤型/现实映射为核心钩子型/类型叠加+文化注入型）
- **抽检验证**：6本好书平均用3个增量元素，2个是下限，3个是最佳值

### S2 力量体系（引擎三要素）
- **定位显式化**：设计"干预这个世界的能力"——力量源+等级制度+子养成线+适配度
- **好坏标准显式化**：好体系3特征（干预方式和世界矛盾深度适配/成长节点密度合理/读者一看就知道这是什么世界）vs 坏体系3特征（和世界矛盾脱节/子境界随意切分/命名脱离世界观）
- **适配度6维判断标准**：①矛盾指向性②力量源同源性③代价内化度④等级映射度⑤矛盾驱动性⑥隐喻一致性。①②为必选项，满足3项以上=高适配。抽检6本好书提炼
- **适配度与增量吸引力协同**：高适配的力量体系本身就是一个增量元素

### S3 主角设计（引擎三要素）
- **定位显式化**：设计主角引擎——三定位+金手指+追读钩子+危机关系
- **好坏标准显式化**：好主角5特征（身份有落差+痛点/欲望具体且与处境强绑定/金手指有明确限制与代价/追读钩子复合型≥3种/与世界危机有弧光）vs 坏主角5问题
- **抽检验证**：6本头部作品（诡秘之主/大奉打更人/凡人修仙传/夜的命名术/捞尸人/模拟长生路）共同骨架
- **三定位**：摄像头定位（消费S1表世界社会结构）+干预者定位（消费S2力量体系+多道途）+攀登者定位（消费S1危机引擎）
- **金手指设计5方法论**：从"3条原则"升级为5条操作方法论（能力与限制并存/代价制造选择/服务于欲望而非替代欲望/与世界设定深度咬合/成长性而非即战力）
- **追读钩子5类型**：代入感/共鸣/好奇/反差/智力博弈。单一钩子留存不稳，头部作品普遍叠加3-4种
- **危机关系4模式**：A完全被动型/B完全主动型/C弧光型/D欲望同向型。最佳模式=C+D结合

### SOP清理 + 决策表消费模式

- 清理历史版本耦合：SKILL.md不再堆叠全部changelog，只保留最近版本，完整历史见CHANGELOG.md

- pipeline同步更新至v3.0.0（Phase路由表seed版本v8.16.0→v9.0.0，清理intro行旧版本引用）
- **7步执行链路显式化**：SKILL.md新增"真实7步执行链路"——①收集底牌→②交互决策（设计大脑）→③骨架展开→④创意发散→⑤故事纲领→⑥黄金首章→⑦主角展开。Step 0定位为"设计大脑"，1d-1f和2a-2c定位为"落盘执行"

## v8.14.0 | 2026-07-23

### 危机引擎设计

**核心改动**：世界危机设计从"一行描述+5标准检查"升级为包含机遇轴+危险轴+咬合机制的"危机引擎设计"。

- **核心洞察（42本验证）**：世界危机=故事主轴。机遇是核心（男频本质=不断攀登向上），危险提供攀登动力+机会+道义。混乱是阶梯
- **机遇轴**：三种"混乱阶梯"机制（秩序崩塌型/制度压迫型/宿命诅咒型）
- **危险轴**：动力（生存压力）+机会（新资源+新体系+新组织三件套）+道义（正道到灰色光谱）
- **咬合机制**：极高（危险即机遇）/高（危险驱动抓机遇）/中低。其中"一体两面"是最高级设计，螺旋上升是最佳结构，周期型危机天然适合超长篇
- **危机引擎5标准**：有运作机制/不可逃避/多层次展开/持续恶化/渗透日常
- 42本验证：危机是上升必要条件（42/42成立）
- v8.14.0框架在西幻、修仙、玄幻、科幻四赛道12个世界方案12个力量体系中全部验证通过
- pipeline同步更新至v2.7.0
