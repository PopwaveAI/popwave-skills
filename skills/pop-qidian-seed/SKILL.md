# seed

> 故事基底构建器。从零构建起点小说项目——种子（世界观/角色/金手指）+ 世界展开（主角引擎/角色储备池/世界圣经）。v7.0.0：轻量燃料注入 + 跨领域张力碰撞 + 四象限散布 + 三眼法判断。

---

## 做什么

输入：用户的创意想法/题材方向/目标读者群
输出：4个文件

| 产出文件 | 阶段 | consumer | 核心内容 |
|---------|------|----------|---------|
| `seed-种子文档.md` | 阶段一 | plot-design | 核心卖点/角色地基/世界观/金手指/流派/三阶段走向/故事弧 |
| `主角引擎.md` | 阶段二 | plot-design | 主角9字段行为模式表 |
| `角色储备池.md` | 阶段二 | plot-design | S/A/B/C四级角色 |
| `世界圣经.md` | 阶段二 | plot-design（按需） | 世界底座——舞台/历史/神系/力量体系/社会结构/势力格局 |

与下游的对接：plot-design加载种子文档+主角引擎+角色储备池+世界圣经，设计卷纲→幕纲→施工卡。

## 核心理念

**种子的质量上限由燃料多样性和碰撞张力决定。** 同赛道排列组合=换皮，不是创意。必须跨领域碰撞，寻找产生结构性矛盾的组合。

不做的事：
- 不编排剧情（plot-design的活）
- 不写笔触风格（流派skill的活）
- 不预设危机线（plot-design的活）

## 怎么操作（SOP骨架）

> **execution.mode**: formal（输入齐全落盘）/ draft（有缺口可补全继续）/ trial（快速试做不落盘）。检测项目文件是否存在判定模式。
> **冷启动加速**：用户给任何方向（哪怕只说"海贼王同人500章"）→ 立即执行Step 0+1，不先问用户问题。输入不够具体时用搜索结果补全。

### 阶段一：种子构建（Step 0-2）

**Step 0：轻量燃料注入** → `steps/step-0-fuel.md`

3次web search，各搜1类燃料：①赛道热门钩子 ②跨赛道机制灵感 ③现实矛盾点。产出3-5条燃料条目（非完整基因库，够碰撞用）。降级：无法搜索时用agent内部知识生成（标注quality:low）。

**Step 1：张力驱动碰撞** → `steps/step-1-collide.md`

用户欲望 × 燃料A(赛道机制) × 燃料B(跨领域元素) × 燃料C(现实矛盾) → 四象限散布(保守/激进 × 最好/最坏) → 三眼法判断(有画面?/有限制?/能跑10个场景?) → 通过的3候选PK → 推荐1个+吸收败者强点。等用户确认。

**Step 2：锁定种子文档** → `steps/step-2-lock.md`

用户确认后，填充7维模板（核心卖点/角色地基/世界观4系统/金手指巧思/流派+三阶段走向/不可牺牲项/体量规划+故事弧）→ 种子压力测试 → 落盘`seed-种子文档.md` + `soul.md`。

### 阶段二：世界展开（Step 3-5）

> 阶段一完成后连续执行，无需用户确认。从种子文档展开为3个设定文件。

**Step 3：主角引擎** — 9字段行为模式表（表层欲望/深层需要/恐惧/底线/行动偏好/第一章主动动作/决策缺陷/第一卷变化/长篇穿透方式）。红线：行动偏好必须具体到方式；第一章主动动作必须是主角发起。输出 `主角引擎.md`。

**Step 4：角色储备池** — S/A/B/C四级角色。S/A角色必须有2-3个原型参考（迁移气质/反应/处境张力，禁止照搬身份/专有设定）+反应模式操作性定义。输出 `角色储备池.md`。

**Step 5：世界圣经（法则层）** — 舞台/历史/神系/力量体系(含三段式骨架+跃迁条件)/敌人生态图/世界矛盾轴。活跃层(社会结构/势力/资源)移至plot按需建。红线：力量体系必须标注各等级段战斗表现+三段式骨架；必须含敌人生态图+世界矛盾轴。输出 `世界圣经.md`。

## 红线

1. **跨领域碰撞红线**：3候选必须从跨领域燃料碰撞产生，每个候选的基因库燃料选取不能完全相同。同赛道排列组合（不同身份同一结构）=不合格。
2. **三眼法红线**：候选必须通过三眼法（有画面/有限制/能跑10个场景）。任一不过=淘汰，不进PK。
3. **金手指红线**：金手指=梗×机制×限制。纯翻译器/纯正向循环/无使用场景限制=不合格。
4. **世界纵深红线**：世界设定覆盖经济/政治/文化/信息不对称。只有地理+势力+力量体系=废设定。
5. **种子深度红线**：种子必须通过压力测试（脑洞张力/冲突密度/世界纵深/角色生态/长线续航）才进阶段二。

## 速查表

| 项 | 合格 | 不合格 |
|----|------|--------|
| 燃料注入 | 3次search覆盖赛道+跨赛道+现实 | 下载10本书(太重不执行) |
| 碰撞 | 跨领域×四象限散布×三眼法 | 同赛道排列组合 |
| 候选差异 | 核心爽感机制实质不同 | 开局身份不同但同一结构 |
| 金手指 | 梗×机制×限制，产生决策困境 | 纯翻译器/纯正向循环 |
| 世界设定 | 经济/政治/文化/信息不对称全覆盖 | 只有地理+势力+力量体系 |
| 角色地基 | 主角+对手+关系动力学齐全 | 只有主角 |
| 种子深度 | 通过压力测试5项 | 薄种子直接进阶段二 |
| 长线伏笔 | ≥5条跨卷伏笔+回声节奏 | 只有3句话走向 |
| 冷启动 | 用户说方向→立即搜索+碰撞 | 先问4个问题再动手 |
| 主角引擎 | 行动偏好具体到方式+第一章主动动作 | "变强"/"战斗"等抽象描述 |
| 角色储备池 | S/A有原型+反应模式操作性定义 | 凭空捏造 |
| 世界圣经 | 法则层完整+敌人生态+矛盾轴 | 追求一步到位的完整世界圣经 |
