## 一、怪物鉴定面板

首次遭遇新怪物类型时弹出。面板以意识流数据流形式嵌入叙述。同类怪物后续不再重复弹（变种/精英/首领视为新类型）。

### 1.1 简版（凡人期）

适用于1-10级（凡人期），面板简洁，5行以内。

**格式**：
```
"[名称]【[种族类型]】（[阶位]）。"
"挑战等级X，[个人/生物]等级X，[生命类型]。"
"最高属性X点，最低属性X点，总属性值X－X点。"
"特殊专长：[专长1]、[专长2]……"
"挑战难度：[字母]+/-。"
```

**示例**：

> "半兽人战士【人型生物】（一阶）。"
> "挑战等级2，生物等级3，人型生命。"
> "最高属性力量17点，最低属性魅力6点，总属性值12－17点。"
> "特殊专长：狂暴、野蛮打击。"
> "挑战难度：D+。"

### 1.2 完整版（超凡期+）

适用于11级以上（超凡期及更高等级段），面板含专属资源条、技能、能力、专长、类法术能力等完整信息。

**格式**：
```
生命：XXX/XXX。
恶魔之力：0/XXX。（或类似专属资源条）
技能：潜行X、学识X、威吓X……
能力：火焰之躯、火焰之鞭、恶魔利爪（+1天生魔法武器）、恶魔皮肤（伤害减免+10）。
专长：多重攻击、武器掌握：长剑。
类法术能力：惊恐术（每天一次）、火球术（每天一次）、召唤恶魔。
```

**示例**：

> "炎魔将军【恶魔】（五阶）。"
> 生命：280/280。
> 恶魔之力：0/150。
> 技能：潜行12、学识18、威吓20。
> 能力：火焰之躯、火焰之鞭、恶魔利爪（+3天生魔法武器）、恶魔皮肤（伤害减免+15/善良）。
> 专长：多重攻击、武器掌握：长剑、精通重击。
> 类法术能力：惊恐术（每天一次）、火球术（每天一次）、召唤恶魔。

### 1.3 简版与完整版对比

| 维度 | 简版（凡人期） | 完整版（超凡期+） |
|------|--------------|-----------------|
| 行数 | 5行 | 7-10行 |
| 生命值 | 无（用生命类型代替） | 有（XXX/XXX格式） |
| 专属资源条 | 无 | 有（恶魔之力/神力/魔力等） |
| 技能 | 无 | 有（含数值） |
| 能力 | 用"特殊专长"概括 | 逐项列举（含具体数值） |
| 类法术能力 | 无 | 有（含使用次数） |
| 适用等级 | 1-10级 | 11级+ |

### 1.4 怪物鉴定写法要点

- 种族类型用方括号【】：【人型生物】【亡灵】【野兽】【异怪】【恶魔】【龙类】
- 阶位用圆括号（）：（一阶）（二阶）……（五阶）……凡人期敌人通常为一至三阶
- 挑战等级为阿拉伯数字，挑战难度为字母+/-：D- < D < D+ < C- < C < C+ < B- < B < B+ < A- < A < A+
- 总属性值格式"X－X"表示最低属性到最高属性的范围
- 完整版的专属资源条根据怪物类型变化：恶魔用"恶魔之力"，亡灵用"负能量"，龙类用"龙息蓄能"等
- 怪物鉴定面板弹出后，后续同类怪物不再重复弹（变种/精英/首领除外）

## 二、属性体感反推

不直接看敌人面板时，通过身体感受反推敌人属性。这是凡人期最常用的"属性感知"写法——主角通过战斗中的体感，将身体反应转化为数值判断。

### 2.1 力量反推

| 体感描写 | 反推属性 | 说明 |
|---------|---------|------|
| "手臂发麻" | 力量16点以上 | 格挡后手臂酸麻，对方力量明显高于常人 |
| "身体沉了一下" | 力量碾压 | 格挡时身体被巨力下压，力量差距极大 |
| "几乎握不住武器" | 力量18点以上 | 武器差点脱手，对方力量达到怪物级 |
| "被震退两步" | 力量20+ | 格挡后身体被迫后退，力量完全不在一个层级 |

**示例**：

> 铛！
>
> 铁剑重重地砸在匕首上，索伦的手臂猛地一沉，一股巨力顺着刀身传来。仅仅是这一击，他的手臂就开始发麻。
>
> 力量16点以上。索伦在心中迅速做出了判断。正面对抗毫无胜算，他必须闪避。

### 2.2 敏捷反推

| 体感描写 | 反推属性 | 说明 |
|---------|---------|------|
| "完全攻击不到他" | 敏捷差距过大 | 对方速度远超主角，攻击全部落空 |
| "只看到一道残影" | 敏捷18+ | 对方移动速度超出视觉追踪能力 |
| "他的动作快得不可思议" | 敏捷16-18 | 对方速度明显高于主角，但勉强能跟上 |

**示例**：

> 索伦的匕首刺出，但对方只是微微侧头，刀锋擦着耳朵飞过。第二轮攻击，对方依然只是侧身——索伦完全攻击不到他。
>
> 敏捷差距太大了。这个人至少有18点以上的敏捷。

### 2.3 体质反推

| 体感描写 | 反推属性 | 说明 |
|---------|---------|------|
| "刀砍上去像砍石头" | 体质16+ | 物理攻击几乎无效，对方皮甲/皮肤极硬 |
| "受了重伤依然行动如常" | 体质18+ | 重伤不减战斗力，体质惊人 |
| "一刀下去没有溅出多少血" | 亡灵/构装体 | 非生命类型，体质反推无效 |

### 2.4 体感反推写法要点

- 反推后必须紧跟战术决策（闪避/游击/撤退），不能只判断不行动
- 反推是主角的"战斗直觉"，不需要面板弹出——是主角的经验判断，可以有误差
- 体感反推用于"来不及弹怪物鉴定面板"的场景（遭遇战中来不及鉴定）
- 同一场战斗中，体感反推最多使用2次（力量/敏捷/体质各算1次），第3次起应弹正式面板或用旁白概括

## 三、伤害类型分离（超凡期+）

超凡期（11级+）开始，伤害不再是一个总数，而是分类型结算。不同伤害类型对应不同的免疫/减伤规则。

### 3.1 基本格式

伤害分类型弹出，每种伤害类型单独结算：

**示例**：

> "你对目标造成了12点的劈斩伤害！……你对目标造成了3点的暗影伤害！"

### 3.2 含免疫/减伤信息

当敌人对某种伤害类型有免疫或减伤时，面板必须反映：

**示例**：

> "半神巫妖可以免疫95%以上的物理伤害！……阴影袭杀可以附加18点的额外伤害，可是现在居然才打出来了3点的暗影伤害。"

**写法要点**：
- 免疫/减伤信息用感叹号+省略号引出，语气带惊讶感（"可是现在居然才打出来了"）
- 免疫百分比用"95%以上"等模糊表述，不用精确到小数
- 减伤后的实际伤害值必须低于理论值，差额即减伤效果
- 伤害类型分离是超凡期核心面板特征，凡人期不用（凡人期伤害是一个总数）

### 3.3 常见伤害类型

| 伤害类型大类 | 具体类型 | 典型免疫者 |
|------------|---------|-----------|
| 物理伤害 | 劈斩、穿刺、钝击 | 巫妖（免疫95%+）、构装体（减伤） |
| 能量伤害 | 火焰、寒冰、闪电、酸蚀、暗影、神圣 | 火元素（免疫火焰）、亡灵（免疫寒冰） |
| 特殊伤害 | 力场、毒素、心灵 | 构装体（免疫心灵）、亡灵（免疫毒素） |

### 3.4 伤害类型分离写法要点

- 物理伤害细分劈斩/穿刺/钝击，与武器类型对应（剑=劈斩，匕首=穿刺，锤=钝击）
- 能量伤害与技能属性对应（暗影系技能=暗影伤害，神圣系技能=神圣伤害）
- 每次攻击的面板中，伤害类型不超过3种（物理+技能附加+特殊效果）
- 免疫/减伤信息只在首次对该敌人造成该类型伤害时弹出，后续同类攻击用旁白概括

## 四、属性标尺锚定

面板弹出属性数值时，必须在同章内用参照物锚定。

### 4.1 锚定规则

- 属性数值弹出后，同章内必须出现至少1个参照物锚定
- 参照物可以是：普通人类水平、具体职业/种族水平、具体已知角色、体感描述
- 同一参照物跨章间隔 ≥ 3章（避免重复使用同一锚点）

### 4.2 锚定方式

| 锚定方式 | 示例 |
|---------|------|
| 直接锚定 | "力量16点以上——这意味着正面对抗毫无胜算。" |
| 比较锚定 | "敏捷18点，已经超越了绝大多数人类能达到的极限。" |
| 体感锚定 | "仅仅是这一击，手臂就开始发麻。力量至少16点。" |
| 参照物锚定 | "挑战等级25——相当于一个全部由高阶传奇组成的冒险者队伍。" |

### 4.3 参照物梯子（完整版）

| 属性值 | 定性 | 力量参照 | 敏捷参照 | 体质参照 |
|--------|------|---------|---------|---------|
| 8-9 | 弱于常人 | 老弱病残 | 迟缓笨拙 | 体弱多病 |
| 10 | 凡人标准 | 普通成年人 | 普通成年人 | 普通成年人 |
| 12-13 | 精英凡人 | 码头苦力/士兵 | 猎人/杂技演员 | 经常锻炼者 |
| 15-16 | 职业者标准 | 资深战士 | 资深盗贼 | 半兽人级别 |
| 18-20 | 凡人极限 | 顶级人类战士 | 超越绝大多数人类 | 难以击杀 |
| 21-23 | 超凡 | 碎石裂壁 | 肉眼难以捕捉 | 法术级恢复 |
| 24-25 | 传奇 | 一拳破城门 | 概念级闪避 | 不死之身 |
| 25+ | 神级 | 改变地形 | 空间级移动 | 神格不朽 |

## 五、挑战等级评估

用"冒险者队伍vs BOSS"的视角反推自身或敌人实力。挑战等级不是单纯的数值对比，而是"需要什么样的队伍才能打赢"的实战评估。

### 5.1 评估格式

**示例**：

> 如果按照前世冒险者们的评估，索伦现在的挑战等级应该是25左右。
> 换句话来说。只有凑齐了一个全部都是高阶传奇的冒险者队伍，才有可能不到50%的几率干掉他。

### 5.2 评估写法要点

- 挑战等级评估用"前世冒险者们的评估"作为信息来源——主角穿越者的视角，用游戏化经验评估世界
- 评估结果用"队伍配置+胜率"的格式表述："全部都是高阶传奇的冒险者队伍"+"不到50%的几率"
- 挑战等级评估用于关键节点：主角实力质变时、遭遇超规格强敌时、幕末/卷末实力盘点时
- 评估可以用于自身（"索伦现在的挑战等级"）也可以用于敌人（"这个BOSS的挑战等级"）

### 5.3 挑战等级与挑战难度的区别

| 维度 | 挑战等级 | 挑战难度 |
|------|---------|---------|
| 表述方式 | 阿拉伯数字（1、2、25……） | 字母+/-（D-、C+、A……） |
| 出现位置 | 怪物鉴定面板第2行 + 独立评估段 | 怪物鉴定面板末行 |
| 含义 | 综合实力数值化 | 对主角个体的难度评级 |
| 使用频率 | 凡人期面板内 + 超凡期独立评估段 | 仅凡人期面板内 |

## 七、红线

1. **怪物鉴定红线**：首次遭遇新怪物类型必弹怪物鉴定面板。同类怪物后续不重复弹（变种/精英/首领除外）。漏弹=不合格。
2. **属性锚定红线**：面板弹出属性数值时，必须在同章内用参照物锚定。同一参照物跨章间隔 ≥ 3章。纯数字无锚定=不合格。
3. **伤害类型分离红线**：伤害类型分离只在超凡期+使用。凡人期伤害为一个总数，不分类型。超凡期+不分离伤害类型=不合格。
4. **体感反推红线**：体感反推后必须紧跟战术决策，不能只判断不行动。同一场战斗中体感反推最多2次。
5. **面板格式匹配红线**：简版怪物鉴定面板仅用于凡人期，完整版用于超凡期+。格式错位=不合格。
6. **挑战等级评估红线**：挑战等级评估仅在关键节点使用（实力质变/遭遇超规格强敌/幕末卷末盘点），不滥用。
