## 文件定位

本文件覆盖超凡期（11-25级）全部战斗模式、面板格式、句式节奏与使用规范。超凡期与凡人期的核心差异：**面板复杂度从5字段暴增到10+字段，战斗规模从1v1/小队升级到军团/弑神/跨位面，法术体系从基础法术进化到瞬发+超魔+领域+神力转化，叙事视角从全主角扩展到多POV**。

## 二、战斗模式详解

### 模式1：法术+近战混合战斗

**适用场景**：主角常规战斗，超凡期最常见的战斗模式。

**核心结构**：瞬发法术 → 近战收割 → 类法术能力，三段式推进。技能名用破折号引出形成快节拍。

**原文锚点**：

> "——火球术！"
> "——剑势【斩首】！"
> "——召唤恶魔！"

**连招sequence写法**（近战→法术→位移→近战→终结）：

> "——回旋斩！"
> "——奥术飞弹*6！"
> "——阴影跳跃！"
> "——剑势【斩首】！"
> "——阴影袭杀！"

**机制说明要点**：
- 法术等级与生物等级挂钩：高阶法术对高等级生物效果递减，低阶法术仅用于控场/试探
- 施法效率比喻：用"人形自走高射炮"式具象比喻呈现施法频率与火力密度
- 瞬发法术（不占用动作）穿插在近战间隙，形成"砍一刀→放一法→砍一刀"的连招节奏

**渲染要点**：
1. 每个技能名独立成行或紧跟破折号引出，形成视觉节拍
2. 法术与近战交替出现，不得连续3次同一类型攻击
3. 连招中间可插入位移技能（阴影跳跃/闪现等）制造空间转换
4. 类法术能力作为收尾或转折，区别于常规法术（不消耗法术位）

**节奏模板**：
```
[破折号引出法术A] → [近战技能B] → [位移技能C] → [破折号引出法术D] → [近战终结技E]
```
每个节点之间可插入1-2句旁白（目标反应/环境变化/主角判断），但不得打断破折号节拍流。

### 模式2：领域碰撞战

**适用场景**：超凡中期（15级+），遭遇拥有领域能力的强敌。

**核心结构**：开启领域 → 视觉化领域效果 → 领域内一切被改变 → 对付领域最好的办法是开启另一个领域让两者碰撞扭曲。

**原文锚点**：

> 领域开启后，周围的一切都变了。灰蒙蒙的色调笼罩下来，温度急剧下降，空气中弥漫着冰霜的气息。
>
> 对付领域最好的办法，就是开启另一个领域，让两者碰撞、扭曲。

**领域效果呈现三维度**：
1. **温度维度**：冰霜领域→温度骤降；火焰领域→灼热难耐
2. **光线维度**：灰蒙蒙/血红/暗影笼罩，色调改变
3. **空间维度**：空间扭曲，传送距离受限，距离感错乱

**神格能力直接改变空间法则**：
- 传送距离限制被领域覆盖/改写
- 领域内物理规则偏移（重力方向/时间流速/伤害类型转换）
- 两个领域碰撞时，交界处出现空间裂缝/法则冲突的视觉化描写

**渲染要点**：
1. 领域开启必须有明确的触发动作（吟唱/姿态/神力释放）
2. 领域效果用物理感受呈现，不用抽象描述——读者要"感觉到"温度和光线的变化
3. 领域碰撞是高潮点，需要至少3句描写碰撞交界处的异象
4. 领域内战斗节奏变慢（法则压制），与领域外的快节奏形成对比

### 模式3：多阶段Boss战

**适用场景**：Boss遭遇战，超凡期核心爽点战斗。

**核心结构**：阶段1试探交锋 → 阶段2法术免疫链 → 阶段3破防 → 阶段4环境崩塌+斩首。**每个阶段必须有明确的攻防转换标志。**

**阶段详解**：

#### 阶段1：试探交锋
- 演技伪装作为战术开篇（主角故意示弱/隐藏实力）
- 常规法术/近战试探，观察Boss反应
- 攻防转换标志：Boss展露第一层实力，主角判断"常规手段无效"

#### 阶段2：法术免疫链
- 主角连续施放法术，Boss逐一免疫
- 用递进式排比呈现：

> "免疫。"
> "免疫。"
> "免疫。"

- 法术免疫用面板呈现：

> "你免疫了该次法术效果！"

- 攻防转换标志：主角意识到需要破防手段，切换战术

#### 阶段3：破防
- 使用特殊手段突破免疫（渎神之语对抗/属性碾压/规则改写）
- 伤害数据化面板弹出，精确到伤害类型分离：

> "你对目标造成了12点的劈斩伤害！……你对目标造成了3点的暗影伤害！"

- 含免疫/减伤信息：

> 半神巫妖可以免疫95%以上的物理伤害！……阴影袭杀可以附加18点的额外伤害，可是现在居然才打出来了3点的暗影伤害。

- 攻防转换标志：Boss受伤后的暴怒形态切换/第二阶段开启

#### 阶段4：环境崩塌+斩首
- 环境成为战斗转折点：墙壁龟裂 → 石钟乳坠落 → 空间不稳
- 突袭打断施法（利用环境碎片/队友配合）
- 神力转化封锁逃跑路线
- 斩首终结，弑神数据链面板弹出

**渲染要点**：
1. 每个阶段转换必须有明确的标志事件（不能模糊过渡）
2. 免疫链用短句排比，每句独立成行，制造"绝望递进"感
3. 环境破坏用拟声词独立成段：

> 轰隆隆！

4. 伤害数据化必须分离伤害类型（劈斩X+暗影Y），体现超凡期的精确度
5. 斩首后必弹弑神数据链面板

### 模式4：召唤物/亡灵军团战斗

**适用场景**：召唤系职业战斗，亡灵法师/恶魔召唤师专属。

**核心结构**：召唤过程视觉化 → 召唤物未出场先制造环境效果 → 召唤物成长机制 → 亡灵军团宏观叙事。

**原文锚点**：

> 白光闪烁，法阵在地面蔓延开来，温度急剧攀升，脚下的沙石开始融化。紧接着，巨大的身影从法阵中冲出——

**召唤过程视觉化三步**：
1. 白光（能量汇聚）
2. 法阵（几何图形蔓延）
3. 生物冲出（体型/气势描写）

**环境前置效果**：
- 召唤物未出场，先改变环境：火系→温度提升/沙石融化；冰系→冰霜覆盖；暗影系→光线吞噬
- 这是召唤物等级的间接暗示——越强的召唤物，环境效果越剧烈

**召唤物成长机制**：
- 召唤物随主人实力提升而增强（等级/体型/能力同步成长）
- 面板可呈现召唤物当前属性（参照怪物鉴定面板格式）

**亡灵军团宏观叙事**：
- 以舰队/方阵形式呈现，不用个体视角
- 宏观笔触：方阵推进/军团碾压/骨海淹没
- 召唤物数量与主人等级挂钩的机制说明

**渲染要点**：
1. 召唤过程不得跳步——白光→法阵→生物，三步缺一不可
2. 环境效果必须在生物出场前出现（悬念+暗示）
3. 军团叙事用宏观笔触，单个召唤物仅在关键击杀时给特写
4. 召唤物面板可简略（比怪物鉴定面板少字段），但必须有等级/体型/核心能力

### 模式5：军团/团战

**适用场景**：大规模战役，多势力交战。

**核心结构**：多兵种交战 → 每种兵种战术特点分析 → 主角以补刀者/指挥者身份参与 → 宏观推进笔触。

**原文锚点**：

> 骑兵冲锋的势头在城镇巷战中受到了限制，狭窄的街道让战马无法展开。这就是绞肉场——双方在巷道里反复争夺，每一寸土地都浸透了鲜血。

**多兵种战术分析**：
- 骑兵：冲锋优势 → 巷战限制（战术转换点）
- 步兵：正面推进/防守线
- 法师：远程支援/范围打击
- 精锐：穿插切割/斩首行动
- 每种兵种给1-2句战术特点分析

**主角定位**：
- 补刀者：不参与正面消耗，专门收割残血/击杀指挥官
- 指挥者：通过命令/信号调度多兵种配合
- 两种身份可切换，但同一战役中身份转换需有触发事件

**绞肉场比喻**：
- 用于描写消耗战的残酷性
- 配合伤害数据化面板（但军团级面板简化，侧重总量/趋势而非个体）

**军团推进宏观笔触**：
- 不聚焦个体战斗，而是描写战线推移/阵型变化/士气波动
- 用地图视角（俯瞰）呈现战场全貌
- 多势力参战时，每个势力给一句战术定位

**渲染要点**：
1. 军团战不得用1v1的细腻笔触写——必须是宏观的、趋势性的
2. 多势力参战，每个势力至少一句定位描写
3. 主角视角切入时，从宏观拉到微观（特写击杀），再拉回宏观
4. "绞肉场"类比喻每场战役最多用1次，不得滥用

### 模式6：仪式/魔法阵战

**适用场景**：仪式类事件、召唤仪式、封印仪式、魔法阵攻防。

**核心结构**：精神融入 → 神力消耗 → 掉级风险 → 神灵威压 → 环境破坏 → 塔灵修复。

**原文锚点**：

> 地面开始龟裂，石柱在威压下倒塌，魔法阵的光芒急剧消散。塔灵疯狂地运转着，修复着被破坏的结构。

**仪式战核心要素**：
1. **精神融入**：施法者意识进入魔法阵/仪式网络，感知范围扩大
2. **神力消耗**：神力值下降，面板呈现消耗量
3. **掉级风险**：仪式失败/中断可能导致等级下降（超凡期独有风险）
4. **神灵威压**：高阶存在降临/被召唤时的威压，直接破坏环境
5. **环境破坏链**：地面龟裂 → 石柱倒塌 → 魔法阵消散
6. **塔灵修复**：环境自我修复机制，提供战斗的拉锯感

**魔法阵大小与召唤物体型挂钩**：
- 法阵直径/面积与召唤物体型正相关
- 大型仪式需要多个施法者协同维持法阵

**渲染要点**：
1. 精神融入用第一人称内视角（感知扩展/信息涌入）
2. 神力消耗必须有面板数值变化（消耗前→消耗后）
3. 掉级风险作为悬念铺垫，不必每次都真的掉级
4. 神灵威压的破坏效果按递进顺序：地面→石柱→法阵
5. 塔灵修复提供"破坏-修复"的拉锯节奏，延长战斗张力

### 模式7：元素相克战

**适用场景**：元素生物交战，多元素混战。

**核心结构**：极短对仗句式呈现元素冲突 → 空气中元素力量剧烈波动 → 主角利用巨兽混战侧面肢解。

**原文锚点**：

> 冰霜的力量弥漫。
> 灼热的火焰肆虐。

**句式特征**：
- 极短对仗句式，每句独立成行
- 句式结构：[元素形容词] + "的" + [力量/元素名] + [动词]
- 对仗工整，字数接近，形成视觉对称

**元素波动描写**：
- 空气中元素力量剧烈波动（不稳定的能量场）
- 元素碰撞交界处产生湮灭/爆炸/元素裂变
- 用环境变化侧面呈现元素强度（金属融化/空气结冰/石块气化）

**侧面肢解战术**：
- 主角不正面参战，利用巨兽混战的混乱
- 在巨兽互相攻击时，趁机肢解受伤目标
- 体现超凡期的战术思维——不硬拼，借力打力

**渲染要点**：
1. 对仗句式仅在元素交锋高潮使用，不得整段使用（否则节奏单调）
2. 侧面肢解需要先铺垫巨兽混战的场景（至少2-3句）
3. 主角每次出手后弹伤害面板，但面板可简化（侧重伤害类型克制关系）
4. 元素相克的伤害加成/减免用面板数值体现（如"火焰伤害对冰霜生物造成200%伤害"）

## 三、面板格式（超凡期）

> **红线**：超凡期面板比凡人期复杂度暴增，从5字段扩展到10+字段。不能凡人期用超凡期面板，反之亦然。

### 3.1 超凡初期升级面板（影武者4级）

```
"影武者职业提升到4级。"
"获得31（敏捷值25+（智力21-10）*0.5）点技能点，生命值提高14点（职业生命8点+（体质22-10）*0.5）。"
"掌握影武者天赋能力【阴影跳跃】！"
"提升为四阶职业者。"
"额外获得一个专长点！……"
```

**字段解析**：
- 职业名+等级
- 技能点（属性公式 + 数值）
- 生命值提升（职业生命 + 体质公式 + 数值）
- 天赋能力习得
- 阶位提升
- 专长点获得

### 3.2 超凡中期升级面板（巫师10级）

```
"巫师职业提升到10级。"
"获得27（智力值21+（智力21-10）*0.5）点技能点，生命值提高10点（职业生命4点+（体质22-10）*0.5）。"
"获得一个自由属性点。"
"获得一个专长点。"
```

**追加超魔专长详细说明**：

```
"法术极效【超魔专长】：该超魔专长可以将法术的威力最大化，以提升两个法术等级为代价，让某一个法术的杀伤力增加100%。"
```

**字段增量**（相比超凡初期）：
- 自由属性点
- 超魔专长（含代价机制说明：提升法术等级=代价，效果=杀伤力增加100%）

### 3.3 半神能力面板（超凡后期巅峰）

```
神格的储存神力极限为200点。
神职：半精灵、游荡者。
领域：海盗。
状态：正常。
基础技能——潜行305，学识290……（20+项技能数值）
圣者形态——超凡入圣、法术免疫1-3环。
半神能力——通晓语言、通晓文字、跨位面沟通。
传奇技能——万能巧手、传奇闪避、传奇战技、史诗敏捷。
特殊专长——弑神者、厄运女神的赐福……
个人专长——灵巧左手、过目不忘……（30+项）
职业专长——奥术记忆、奥术反制……
战斗技能——召唤幽影、阴影跳跃【200米】、阴影袭杀、阴影穿梭【2次】、恐惧目光（神性）……
```

**字段清单（10+字段）**：
1. 神力储存极限
2. 神职
3. 领域
4. 状态
5. 基础技能（20+项，含数值）
6. 圣者形态（含法术免疫环阶）
7. 半神能力
8. 传奇技能
9. 特殊专长
10. 个人专长（30+项）
11. 职业专长
12. 战斗技能（含距离/次数参数）

**渲染要点**：
- 半神面板是超凡期最复杂面板，字段最多
- 技能数值必须列出（潜行305/学识290），体现量化成长
- 战斗技能带参数（【200米】/【2次】），区别于凡人期的无参数技能
- 面板以意识流嵌入叙述，不以弹窗/光屏形式出现

### 3.4 法术免疫面板

```
"你免疫了该次法术效果！"
```

**使用场景**：主角或敌方拥有法术免疫能力时，每次免疫触发弹出。
**渲染要点**：简短有力，独立成行，配合免疫链排比使用。

### 3.5 伤害数据化面板（超凡期精确到伤害类型分离）

```
"你对目标造成了12点的劈斩伤害！……你对目标造成了3点的暗影伤害！"
```

**含免疫/减伤信息**：

```
半神巫妖可以免疫95%以上的物理伤害！……阴影袭杀可以附加18点的额外伤害，可是现在居然才打出来了3点的暗影伤害。
```

**与凡人期差异**：
- 凡人期：单一伤害类型，模糊数值
- 超凡期：伤害类型分离（劈斩+暗影+火焰...），精确数值，含免疫/减伤百分比

**渲染要点**：
- 伤害类型必须分离呈现（劈斩X + 暗影Y），不得合并
- 免疫/减伤用百分比+具体数值双重呈现（95%免疫 + 18点附加仅打出3点）
- "居然才打出来"等语气词体现主角的意外感，增强读者代入

### 3.6 弑神数据链面板

```
"击杀目标→抽取灵魂能量→获得杀戮经验375万→部分被未知目标抽取→获得神性值23点→获得神格值25点→弑神者专长提升"
```

**使用场景**：击杀神性生物/半神/神明时弹出。
**渲染要点**：
- 箭头链式呈现，一气呵成
- "部分被未知目标抽取"制造悬念（伏笔）
- 神性值/神格值/专长提升三者并列，体现弑神的复合收益

### 3.7 神性值晋级面板

```
"能力提升！……"
"你积累的神性值已经达到了100点！……由【弱等神性】晋级成为【中等神性】！……"
"神性力量提升！……"
"神性力量转化！……你的所有属性点+1！……"
```

**使用场景**：神性值达到阈值晋级时弹出。
**渲染要点**：
- 晋级阈值明确（100点=弱等→中等）
- 晋级效果分层呈现：称号变更 → 神性力量提升 → 全属性+1转化
- "……"省略号制造节奏停顿，增强爽感

## 四、数据化怪物分析（超凡期）

> **红线**：首次遭遇新怪物类型时必须弹出怪物鉴定面板。超凡期怪物面板比凡人期更复杂——含完整属性/技能/能力/专长/类法术能力。

### 4.1 简版怪物鉴定（吉斯洋基人）

```
"吉斯洋基人【星界生物】（三阶）。"
"挑战等级12，个人等级15，拥有灵能类法术。"
"最高属性24点，最低属性12点，总属性值95-105点。"
"特殊专长：灵能法术、心灵控制、星界种族。"
"挑战难度：A+。"
```

**字段清单**：
1. 种族名 + 【生物类型】 + （阶位）
2. 挑战等级 + 个人等级 + 类法术标注
3. 属性范围（最高/最低/总值范围）
4. 特殊专长
5. 挑战难度评级

### 4.2 完整版怪物鉴定（火焚魔）

```
生命：XXX/1500。
恶魔之力：0/3000。
技能：潜行125、学识70、威吓70、法术辨识70……
能力：火焰之躯、火焰之鞭、恶魔利爪（+1天生魔法武器）、恶魔皮肤（伤害减免+10）、恶魔抗性（各类抗性+10）。
专长：多重攻击、武器掌握：长剑、异种武器掌握：长鞭。
类法术能力：惊恐术（每天一次）、支配恶魔（每天一次）、火球术（每天一次）、燃烧之手（每天一次）、召唤恶魔、心灵感应。
```

**字段清单（完整版）**：
1. 生命值（当前/上限）
2. 特殊能量条（恶魔之力 0/3000）
3. 技能（含数值）
4. 能力（含参数：+1天生武器/伤害减免+10/抗性+10）
5. 专长（含武器掌握细分）
6. 类法术能力（含使用频率：每天一次/随意施展）

**简版 vs 完整版选择规则**：
- 杂兵/一次性敌人 → 简版（5字段）
- Boss/反复出现的强敌 → 完整版（6+字段）
- 首次遭遇用完整版，后续遭遇可简版

## 五、句式节奏特征

### 5.1 技能名破折号引出（快节拍）

> "——火球术！"
> "——剑势【斩首】！"

**规则**：技能名前加破折号，后加感叹号，独立成行或紧跟句末。用于战斗连招中制造快节拍。

### 5.2 法术免疫链（递进式排比）

> "免疫。"
> "免疫。"
> "免疫。"

**规则**：每次免疫独立成行，用句号收束（非感叹号），制造冷静递进的绝望感。至少3次才构成"链"。

### 5.3 元素相克（极短对仗句式）

> 冰霜的力量弥漫。
> 灼热的火焰肆虐。

**规则**：两句对仗，字数接近，结构一致（[元素形容词]+的+[名词]+[动词]）。仅在元素交锋高潮使用。

### 5.4 环境破坏（拟声词独立成段）

> 轰隆隆！

**规则**：拟声词独立成段，用于环境崩塌/爆炸/巨物坠落的听觉冲击。每场战斗最多用2次。

### 5.5 多POV切换

**规则**：超凡期必须有多POV切换，至少每3章切1次非主角视角。

可用非主角视角：
- 敌方视角（爵士/男爵/恶魔化身视角）：呈现敌人的恐惧/误判/被碾压感
- 友方视角（德鲁伊/队友视角）：呈现对主角实力的侧面评价
- 旁观者视角：呈现战斗的宏观影响

**切换技巧**：
- POV切换在段落间隙进行，用场景转换标志（地点/时间/人物焦点）引导
- 非主角视角的战斗描写更主观、更情绪化，与主角视角的冷静数据化形成对比
- 每次非主角视角持续1-3段，不宜过长（避免偏离主线）

## 六、与凡人期差异总结表

| 维度 | 凡人期（1-10级） | 超凡期（11-25级） |
|------|------------------|-------------------|
| 面板复杂度 | 5字段 | 10+字段 |
| 战斗规模 | 1v1/小队 | 军团/弑神/跨位面 |
| 法术体系 | 基础法术 | 瞬发+超魔+领域+神力转化 |
| 伤害描写 | 模糊（单一类型） | 精确数据化（劈斩X+暗影Y，95%物理免疫） |
| 环境互动 | 背景描写 | 战斗因素（崩塌/融化/法则改变） |
| 叙事视角 | 全主角 | 多POV（至少每3章切1次非主角视角） |

| 升级面板 | 职业名+等级+技能点+生命值 | +天赋能力+阶位+专长点+超魔专长+自由属性点 |

| 召唤物 | 一次性工具 | 成长机制（随主人提升）+环境前置效果 |
| 风险机制 | 死亡 | 掉级风险（仪式失败/中断） |

## 七、使用指引

### 7.1 加载条件

- SKILL.md Step 4b标注"当前等级11-25级"时，加载本文件
- 施工卡标注的战斗类型对应本文件7种模式之一时，查模式详解渲染

### 7.2 模式选择逻辑

```
施工卡战斗类型 → 模式映射：
  法术近战混合     → 模式1
  领域/神格对抗    → 模式2
  Boss遭遇        → 模式3
  召唤/亡灵       → 模式4
  军团/大规模战役  → 模式5
  仪式/魔法阵     → 模式6
  元素生物        → 模式7
```

一场战斗可混合多种模式（如Boss战中场召唤援军=模式3+模式4），但主模式不得连续3章重复。

### 7.3 面板选择逻辑

```
事件 → 面板映射：
  升级（11-15级）  → 3.1 超凡初期升级面板
  升级（16-20级）  → 3.2 超凡中期升级面板（含超魔专长）
  升级（21-25级）  → 3.3 半神能力面板（巅峰）
  法术免疫触发     → 3.4 法术免疫面板
  造成伤害         → 3.5 伤害数据化面板（类型分离）
  击杀神性生物     → 3.6 弑神数据链面板
  神性值达标晋级   → 3.7 神性值晋级面板
  首次遭遇新怪物   → 第四章 数据化怪物分析（简版/完整版）
```

### 7.4 多POV切换检查

每3章检查1次：是否有至少1段非主角视角的战斗叙事？

- 有 → 合格
- 无 → 下一章必须插入非主角视角段落

可选视角：敌方视角 / 友方视角 / 旁观者视角

### 7.5 红线自查清单

- [ ] 面板格式匹配当前等级段（超凡期用10+字段面板，不用凡人期5字段面板）
- [ ] 连续3章未使用同一种战斗模式
- [ ] 多阶段Boss战每个阶段有明确攻防转换标志
- [ ] 至少每3章有1次非主角视角切换
- [ ] 伤害面板类型分离（劈斩X+暗影Y），非单一类型
- [ ] 击杀神性生物弹弑神数据链面板
- [ ] 升级面板格式匹配等级段（初期/中期/后期）
- [ ] 首次遭遇新怪物弹完整版/简版怪物鉴定面板

## 八、模式组合示例

### 示例A：主角遭遇半神巫妖（模式3+模式2+模式1）
