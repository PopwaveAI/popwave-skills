# 海贼王世界·金手指触发判断表

金手指的触发规则不在skill中——由项目的 `systems/金手指系统.md` 定义。write读取该文件后，按其规则判断本章施工卡中的事件是否触发金手指。

| 判断步骤 | 操作 |
|:--|:--|
| 1. 读取金手指系统文件 | 加载项目 `systems/金手指系统.md` |
| 2. 查触发条件表 | 系统文件中应定义"什么事件触发什么效果" |
| 3. 比对施工卡事件 | 本章硬锚点中是否有匹配触发条件的事件 |
| 4. 确定触发内容 | 如果触发，确定金手指在本章的具体表现形式 |
| 5. 标记在执行报告中 | 记录触发的金手指效果和段落位置 |

## 海贼王世界专属红线

1. **世界观红线（同人模式）**：使用《海贼王》原作世界观。原作角色可作为背景人物出场或被提及，但禁止主角与原作核心角色产生深度互动或加入原作主角团。
2. **战斗多样性红线**：连续3章不得使用同一种战斗模式。连续2章不得使用相同章节过渡方式。
3. **悬赏金定级红线**：侧面定级以悬赏金为核心参照物——悬赏金额=实力+威胁度的综合标尺。
