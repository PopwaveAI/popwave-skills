# 爽文剧情设计SOP

> 用途：在矩阵选引擎之后、填链条之前，增加一个"爽感设计"步骤。解决"链条环节全踩中但内容不够爽"的问题。
>
> 定位：不是约束清单，是设计工具。帮助锚点定义者在填链条之前想清楚"这章到底爽什么"。

---

## 流程位置

```
查矩阵选引擎组合 → 【本SOP：爽感设计】 → 填链条 → 定锚点
```

本SOP产出物：一个"爽感设计卡"，嵌入锚点定义文件的爽点链条之前。

---

## 4个设计步骤

### 步骤1：爽感一句话

填链条之前，用一句话写死这章读者读完最爽的那一刻。

格式：`[谁] 在 [什么场合] [做了什么]，导致 [谁] [什么反应]。`

好示范：陆征在秩序之眼据点当着三个灵能者的面，甩出牧师和混沌交易的录音，牧师当场说不出话。

差示范：陆征发现了牧师的秘密。（太模糊，不知道爽在哪）

差示范：陆征和牧师对峙。（没有爽感峰值，只是事件描述）

**判断标准**：把这句话发给一个不看书的人，他听了会不会"哦？然后呢？"——会就过，不会就重想。

### 步骤2：打脸力度公式

翻盘点的力度 = 嚣张程度 × 翻盘反差 × 旁观者数量

三个变量必须全部设计：

**嚣张程度**：对手在翻盘前有多嚣张？设计一个具体的"踩"的动作——当众羞辱/嘲笑/无视/把主角当废物。踩得越狠，翻盘越爽。

**翻盘反差**：翻盘前后的落差有多大？落差 = 翻盘前主角的处境（最低点）和翻盘后主角的处境（最高点）之间的距离。反差越大越爽。

**旁观者数量**：翻盘时有几个人在看？1对1的翻盘只有主角和对手知道 → 爽感有限。有旁观者在场 → 爽感有放大器。旁观者越多、旁观者之前越看不起主角，翻盘越爽。

### 步骤3：低压区设计

爆发前必须有"低压区"——主角被踩到最低点的设计。

低压区的3种类型：
- **地位压制型**：主角被当众贬低/当成下等人/当成废物
- **实力碾压型**：主角的技能/能力被直接否定/嘲笑/当众无效
- **信息差碾压型**：对手当众展示主角不知道的信息差，让主角显得无知

低压区必须当众发生。私下被欺负没有放大器——必须有人看到主角被踩，后面的翻盘才有"打回去"的爽感。

### 步骤4：旁观者反应设计

翻盘时旁观者必须有可见反应。旁观者反应是爽感的放大器——读者通过旁观者的眼睛感受"这一下有多狠"。

旁观者反应的3种类型：
- **沉默型**：全场鸦雀无声。之前嘲笑主角的人闭嘴了。
- **改观型**：之前看不起主角的人重新评估——"这家伙不简单"。
- **恐惧型**：旁观者意识到主角比他们想象的危险——往后退了一步。

至少设计1种旁观者反应，写进锚点定义。写作agent渲染时必须落地。

## 锚点定义中的"爽感设计"部分格式

在爽点链条之前，锚点定义文件新增一个"爽感设计"部分：

```markdown
## 爽感设计

### 爽感一句话
[一句话写死本章最爽的那一刻]

### 打脸力度
- 嚣张程度：[对手怎么踩主角的，具体动作]
- 翻盘反差：[翻盘前最低点 → 翻盘后最高点]
- 旁观者数量：[几个人在场看]

### 低压区
- 类型：[地位压制/实力碾压/信息差碾压]
- 具体场景：[怎么被踩的，当众发生]

### 旁观者反应
- 类型：[沉默/改观/恐惧]
- 具体反应：[旁观者怎么反应的]
```

这个部分在锚点定义中位于"硬锚点"之前、"爽点链条"之前。它是链条填充的设计依据——链条的每个环节都往"爽感一句话"那颗钉子上连。

---

## 与矩阵的关系

矩阵给链条结构（环节模板），本SOP给链条内容的设计方法。

流程：
1. 查矩阵选引擎组合 → 得到链条模板（环节结构）
2. **按本SOP做爽感设计** → 得到爽感一句话+打脸力度+低压区+旁观者反应
3. 把爽感设计的结论填入链条模板 → 得到填好具体剧情的爽点链条
4. 标注硬锚点+不可触发项 → 完成锚点定义

矩阵解决"环节不漏"，本SOP解决"环节够爽"。

## 使用示例

### 示例：combat_reversal章型

**步骤1 爽感一句话**：
陆征在秩序之眼据点当着三个灵能者和Mia的面，一拳打碎了那个全场没人能碰的混沌实体，三个灵能者全跪在地上。

**步骤2 打脸力度**：
- 嚣张程度：三个灵能者当众嘲笑陆征"半成品"，说他是"带枪的猴子"
- 翻盘反差：翻盘前=被当废物看+没人觉得他能帮上忙 → 翻盘后=唯一站着的人+全场最强的存在
- 旁观者数量：5人（3个灵能者+Mia+Kessler）

**步骤3 低压区**：
- 类型：地位压制+实力碾压
- 具体场景：三个灵能者当众展示灵能视角的远程感知能力，然后看陆征——"你连这个都做不到？"陆征沉默。Mia不帮他说话。

**步骤4 旁观者反应**：
- 类型：改观+恐惧
- 具体反应：三个灵能者中叫得最响的那个往后退了一步。Mia的表情第一次出现"重新评估"。Kessler手里的设备掉了。

### 示例：confrontation_pressure章型

**步骤1 爽感一句话**：
陆征在教堂当着二十个教众的面，甩出牧师私下和混沌势力交易的录音，牧师当场说不出话，手在抖。

**步骤2 打脸力度**：
- 嚣张程度：牧师当众骂陆征"撒旦的野狗"，说他不配进教堂，让教众把他赶出去
- 翻盘反差：翻盘前=被当众驱逐+被骂撒旦野狗 → 翻盘后=站在原地不动，牧师反而成了被审判的那个
- 旁观者数量：20+人（教众+牧师+助手）

**步骤3 低压区**：
- 类型：地位压制
- 具体场景：牧师站在台上，居高临下，教众围着陆征。牧师说"这里没有你的位置"。教众开始推搡陆征。

**步骤4 旁观者反应**：
- 类型：沉默型
- 具体反应：全场鸦雀无声。刚才推搡陆征的教众松开了手。牧师的助手往后退了一步。
