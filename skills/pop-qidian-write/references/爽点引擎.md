# 爽点引擎

> 不是配额表，是选型工具。5种引擎各自的原理+原文锚点+适用场景。你选引擎，不是凑数量。

---

## 引擎1：打脸循环

### 原理

番茄第一爽感。不是"主角赢了"，是"看不起主角的人输了"。读者代入的是被轻视的主角，爽的是翻盘后对方的表情。

### 结构

```
轻视（对方嚣张/误判/羞辱）
→ 压制（对方用实际行动碾压）
→ 暗蓄（主角不辩解，默默准备）
→ 翻盘（1-3行，"咔"一下到位）
→ 失态（对方崩溃/哑口/承认）
```

### 原文锚点

> "你这个死老太婆，杀不了你的本体，我还杀不了你的精神印记吗？！"
> "死老太婆你不是喜欢追杀吗？"叶凡冲了上去，喝道："今天我就追杀给你看！"

> "太让我吃惊了……这颠覆了常理！"姬惠依然处在惊憾中，金书一冲而过，将她割裂。

### 关键

- 轻视要厚——至少让对方嚣张2句+碾压1次，读者越愤怒翻盘越爽
- 翻盘要快——≤3行，越绝境配越短
- 翻盘要有智力感——不是蛮力也不是运气，是信息/判断/能力的组合
- 失态要可见——不是"脸色微变"，是具体的崩溃行为+台词

### 适用场景

对峙章/战斗章。对方是智能生物（能嚣张/能失态）。

## 引擎2：压力突破

### 原理

主角被逼到绝境，读者跟主角一起憋着一口气。突破的瞬间，这口气释放出来就是爽。

### 结构

```
绝境（枪无效/能力被封/体力耗尽）
→ 发现（缝隙/变量/关键信息，1句闪回）
→ 突破（1-2行，一招翻盘）
→ 回味（主角站起来/喘气/环视，停1拍）
```

### 原文锚点

> 叶凡抱鼎而上，跳脱出了九位大帝的合围之势，逆冲向九颗神秘的大星。
> 他一拳轰去，这一击的力量发自全身……一拳既出，星域失色。

### 关键

- 绝境要真——读者要相信主角真的可能输
- 突破的钥匙要在前文埋过（笔记里的方法/之前观察到的弱点），不能凭空
- 突破后停一拍——让读者"爽完"再重启压力

### 适用场景

战斗章/探索章。对方可以是非智能体（怪物/机关/环境）。

## 引擎3：未知揭示

### 原理

读者跟着主角累积疑问，到某个瞬间真相炸裂——原来如此！这个"原来如此"就是爽。

### 结构

```
疑问累积（3-5个微钩子制造悬念）
→ 线索收束（2-3条线索在某一刻串联）
→ 真相炸裂（1-2句揭示，颠覆之前的认知）
→ 后果展开（揭示改变了一切）
```

### 原文锚点

> 九具百米长的庞大龙尸，大半截躯体静静地横在山巅上，少半截躯体垂挂在山崖下，犹如黑色的钢铁长城，充满了震撼性的力感，极其冲击人的视觉。

读者从"这是什么"到"龙尸"到"九具"到"百米长"——每一句都在增加信息量，到"钢铁长城"时炸裂。

### 关键

- 疑问要真——不是故弄玄虚，是读者真的想知道
- 揭示要颠覆——不是"答案出来了"，是"原来是这样！"
- 揭示后改变行动——揭示不是终点，是新行动的起点

### 适用场景

开篇章/探索章/揭秘章。

## 引擎4：小爽沿途

### 原理

不是只有大爽才爽。沿途撒小爽让读者"一路爽到大爆"。小爽的本质是"主角牛逼"的微瞬间。

### 5种小爽

| 类型 | 本质 | 示例 |
|:----|:----|:----|
| 信息优势 | 主角知道别人不知道的 | "他看了便签三秒。他知道了。" |
| 判断正确 | 主角判断被验证 | "他选了左边。右边传来碎裂声。" |
| 技能展示 | 主角展示专业能力 | "一眼看出鞋印是警用勤务靴。" |
| 对手失误 | 对手犯错被主角抓住 | "它笑了。它不该笑。" |
| 主动选择 | 主角在压力下主动选择 | "半年没碰的枪。今天他没犹豫。" |

### 关键

- 小爽用1-3句，不展开
- 大爽前撒2-3个小爽做铺垫（让读者"预感"主角要赢）
- 大爽后不再撒小爽（保持纯净回味）
- 小爽之间用压力间隔，不连爽

### 适用场景

所有章型。小爽是调味料不是主菜。

## 引擎5：即时奖励

### 原理

番茄核心机制：行动→可见回报。主角每做一件有价值的事，必须有可见的即时回报。读者需要看到主角在变强。

### 原文锚点

> 叶凡大喝一声，轮海沸腾，五脏通明，四肢璀璨，脊椎骨如大龙冲天。

每次战斗后叶凡的轮海都在变化——这就是即时奖励。读者看到"他在变强"。

### 关键

- 每章至少1次行动→回报闭环
- 回报必须可见、具体——不是"他变强了"，是物理标志（掌心烙印/新能力/新情报）
- 回报有代价但不抵消爽感（失去工具但获得能力）

### 适用场景

所有章型。即时奖励是追读率的跨章驱动力。
