# Pop 读者拆书 — 使用示例

> 这个目录存放**真实的拆书产出示例**

## 如何运行

1. 调用本 skill 时，告诉 pop 你想拆哪本书 + 提供内容来源
2. 指定要哪些模块（默认全部 9 个）
3. 产出物会出现在 `output/<小说名>/` 目录下
