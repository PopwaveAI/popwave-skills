# 章节摘要提取协议 v1.0

> 用于子代理通读小说章节时，保证输出质量一致。
> 所有子代理执行任务时必须严格遵循此协议。

---

## 一、输出格式

每章一条 YAML 条目，固定字段：

```yaml
ch: 101
title: "大旱"
summary:
  events:
    - "山越境内大旱，四个月未降雨"
    - "李通崖深入山越境内侦察"
    - "李项平被咒杀身亡，陈冬河率残部归来"
  new_chars: ["木芽鹿", "沙摩里"]    # 新出场角色，没有则不写
  breakthroughs: []                   # 修为突破
  battles: []                         # 重要战斗
  techs: ["玄巫道术"]                 # 新功法/法器/丹药
  locations: []                       # 新地点
  factions: []                        # 新势力
```

---

## 二、必须扫描的 6 个维度

每章按以下顺序检查，覆盖全部 6 项：

| # | 维度 | 记录规则 | 示例 |
|:-:|:----|:--------|:----|
| 1 | **角色** | 首次出现的人名/妖名 | "安景明（安家天才，练气三层）" |
| 2 | **修为** | 谁突破到第几层、什么功法 | "李通崖突破练气五层" |
| 3 | **战斗** | 谁vs谁、胜负、怎么赢的 | "李清虹以月阙剑弧斩杀练气熊妖" |
| 4 | **功法/法器/丹药** | 名称+品级+来源 | "《金殿煌元诀》（四品古法，第283章得）" |
| 5 | **地域/势力** | 新地点/新势力/势力关系变化 | "宗泉岛（海外据点，第416章建立）" |
| 6 | **设定信息** | 世界观规则、历史真相 | "青池宗以人血炼丹" |

---

## 三、填充规则

### 3.1 每个维度的填充优先级

当一章信息密度高、一个维度有多个条目时：

- **角色**：只记有名字的角色。"一个村民""一个散修"不记。"安景明（练气三层，安家天才）"记。
- **修为**：只记明确说了"突破"、"晋级"的。对话中提到但未确认的不记。
- **战斗**：记参与方+胜负+怎么赢的关键信息。不记过程细节。
- **功法/法器**：有名字就记。无名法器只记品级（如"练气上品法剑"）。
- **地域/势力**：首次出现的地点记。同章节再次出现不重复记。

### 3.2 "什么都没有"的处理

如果一章读完后 6 个维度全空：

```yaml
ch: 105
title: "日常"
summary:
  events: []
  new_chars: []
  breakthroughs: []
  battles: []
  techs: []
  locations: []
  factions: []
  note: "本章为过渡章，无新信息"
```

不允许跳过不写。每章都必须有一条记录，允许空数组。

### 3.3 每章最少记录量

- **events 至少 1 条**（哪怕只是"李玄宣处理族务"）
- 其他维度**有则填、无则空数组**

---

## 四、质量自检

每批读完（约 50-70 章），子代理必须验证：

```
自检清单：
[ ] 每章都有记录，没有跳章
[ ] 每章的 events 至少 1 条
[ ] 新角色有名字+身份说明
[ ] 修为突破标注了境界和章节
[ ] 战斗记录了胜负和关键信息
[ ] 没有遗漏明显的重要事件
```

如果有某章读的时候跳过了（非故意），必须在 `note` 标注 "本章未完整读取"。

---

## 五、输出汇总

读完一批后，除了章节摘要，额外输出一个**批处理摘要**：

```yaml
batch:
  range: "第101-170章"
  summary: "核心剧情线：加泥奚之死→山越归附→矿脉争夺→李尺泾死亡确认"
  key_events:
    - "加泥奚自碎仙基而死"
    - "李通崖融合镜面碎片，修为暴涨至练气五层"
    - "李尺泾被投入大蛟丹炉炼丹"
    - "李通崖吞并华芊山"
  new_major_chars: ["安景明", "郁慕高", "灵岩子"]
  new_techs: ["《越河湍流步》", "《紫雷秘元功》"]
  new_locations: ["华芊山", "望月湖坊市废墟"]
```

## 六、常见错误

| 错误 | 反面示例 | 正面示例 |
|:----|:--------|:--------|
| 无名字的角色 | "一个散修出现" | "灵岩子（紫烟门练气六层）出现" |
| 无胜负的模糊战斗 | "两人打了一架" | "李通崖以一敌二，月阙剑弧压制卢思嗣+安鹧言" |
| 笼统的修为描述 | "修为提升了" | "李通崖从练气三层突破到练气五层" |
| 混入读后感 | "这段写得很好" | 不允许出现任何评价 |
| 跳章不标注 | 直接跳过某章 | 必须在 note 里写"本章未完整读取" |
