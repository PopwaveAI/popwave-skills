# CHANGELOG

## v1.2.0 — 2026-07-20

### 新增采风级调研档位

新增"采风级"期望深度，与种子级/场景级/缺口级并列。采风级走独立流程，跳过三界框架的定界/分界调研/压力源/交叉困境，直接做真实质感采集。

**新增：**
- 采风级流程（7步）：接收采风需求→真实质感采集（4方向并行搜索）→片段整理→质感标注→质量检查→存盘→索引更新+返回
- 采风级4个采集方向：一手资料（从业者口述/纪实文学）+感官细节（视/听/嗅/触/味）+职业黑话（行话/俚语）+场景物理布局
- 采风级产出格式：可复用细节片段（50-200字/片段，3-8个/主题），每片段含质感标签+感官覆盖+适用场景
- 采风级专用门禁（≥3片段+感官覆盖+质感标签完整），不做压力源/交叉困境检查
- 采风级产出路径：`写作参考/知识沉淀/{主题}-采风.md`
- write 加入 pipeline 调用方

**变更：**
- description 更新：增加采风级说明
- 红线1/2/4 增加"采风级除外/采风级不适用"的例外说明
- 调用方表新增 write（采风级）

## v1.0.0 — 2026-06-28

### 初始版本

从 pop-novel-seed v1.6.0 的种子搜索法SOP提取为独立skill，服务于 v3.5 涌现式写作管线。

**新增：**
- 三界框架定界方法论（社会界/自然界/超自然界）
- 分界调研维度（社会界6维度/自然界6维度/超自然界5维度）
- 压力源转化6字段结构化（来源界属/具体描述/显性表现/对主角的压力方式/可量化指标/交叉可能性）
- 交叉困境识别6种类型（同界内3种+跨界3种）
- 质量检查标准（≥3压力源/≥3交叉困境/含具体案例）
- 调研深度标准（≥500字/具体案例/流程步骤/数据数值/多轮搜索≤3轮）
- 退化预警机制（连续3次无需求→标记退化预警）
- 存盘+索引更新+路径指针返回流程

**调用方：**
- seed（step-1）：世界观+压力矩阵调研，传入书型+核心卖点
- plot（step-2）：幕纲设计场景调研，传入事件brief+涉及领域
- pop-novel（Step2）：写作中发现信息缺口，传入缺口描述
