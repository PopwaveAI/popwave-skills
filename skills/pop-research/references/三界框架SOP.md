# 三界框架SOP

> pop-research v1.0.0 核心方法论
> 来源：从 pop-writer-v3-seed 种子搜索法SOP提取，完整重建为独立调研框架

三界框架是 pop-research 的核心调研方法论。将所有压力源归入三个界——社会界、自然界、超自然界——每个界有独立的调研维度体系，确保调研不遗漏关键压力来源。

---

## 1. 三界定义与定界规则

### 1.1 三界定义

| 界 | 定义 | 核心关注 |
|:---|:-----|:---------|
| 社会界 | 人类社会内部的权力/经济/文化/法律/阶层/组织 | 人与人之间的结构性压力 |
| 自然界 | 物理环境的资源/气候/地理/生态/疾病/灾害 | 人与自然之间的生存压力 |
| 超自然界 | 超越物理法则的力量/规则/存在/禁忌 | 人与超自然力量之间的规则压力 |

### 1.2 定界规则

收到调研需求后，判断本次调研涉及哪些界：

```
定界判断：
  1. 从调研需求中提取关键词和领域
  2. 对照三界定义，判断涉及哪些界
  3. 记录定界理由——为什么涉及这些界，为什么不涉及其他界
```

**定界示例：**

| 调研需求 | 涉及界 | 定界理由 |
|:---------|:-------|:---------|
| 修仙世界压力调研 | 社会界+自然界+超自然界 | 修仙涉及宗门权力（社会）、灵气资源（自然）、力量体系（超自然）——三界全涉及 |
| 末世生存压力调研 | 社会界+自然界 | 末世涉及资源争夺（社会）、环境灾害（自然）——无超自然设定 |
| 都市异能压力调研 | 社会界+超自然界 | 都市涉及社会组织（社会）、异能体系（超自然）——自然界为常规现代环境不单独调研 |

### 1.3 无需求处理

某个界被判定为不涉及时，不跳过，而是逐类说明理由：

```
无需求记录格式：
  - {界名}：无需求
    - 理由：{为什么不涉及这个界}
```

> 连续3次无需求 → 标记退化预警（详见 `调研深度标准.md` §3 退化预警机制）。

## 2. 分界调研维度

## 3. 压力源转化（6字段）

调研发现需要转化为结构化压力源，每个压力源包含6个字段：

| 字段 | 含义 | 填写要求 |
|:-----|:-----|:---------|
| 来源界属 | 这个压力源属于哪个界 | 社会界/自然界/超自然界 |
| 具体描述 | 这个压力源是什么 | 从调研中得出的具体内容，非抽象概念 |
| 显性表现 | 读者能看到的场景/现象 | 必须是读者可见的具体场景，不是抽象描述 |
| 对主角的压力方式 | 这个压力源如何逼迫主角行动 | 主角面临的具体困境和选择 |
| 可量化指标 | 用什么标准衡量压力大小/紧迫程度 | 具体的数值/等级/时间限制 |
| 交叉可能性 | 可能与哪些其他压力源产生交叉 | 标注潜在交叉对象的编号 |

### 压力源转化示例

```
压力源 #1：
  1. 来源界属：社会界
  2. 具体描述：修仙世界的宗门垄断了修炼资源（灵石/灵药/秘境），
     散修无法获取修炼所需，只能依附宗门或冒险进入危险区域采药。
  3. 显性表现：散修为争夺一株百年灵药在秘境中互相厮杀；
     宗门弟子携带宗门令牌进入禁地采药，散修被驱赶。
  4. 对主角的压力方式：主角作为散修，每提升一个境界都需要
     冒生命危险获取资源——不修炼就被淘汰，修炼就可能死在采药路上。
  5. 可量化指标：散修平均寿命 vs 宗门弟子平均寿命；
     百年灵药的市场价格 vs 散修年收入。
  6. 交叉可能性：可与#3（资源分布自然界）、#5（阶层关系社会界）交叉。
```

## 4. 交叉困境识别（6种类型）

压力源之间的交叉产生困境。共6种交叉类型：

### 4.1 同界内交叉（3种）

| 交叉类型 | 定义 | 困境特征 |
|:---------|:-----|:---------|
| 社会界内部交叉 | 两个社会界压力源交叉 | 社会规则之间的冲突——如法律与权力的冲突 |
| 自然界内部交叉 | 两个自然界压力源交叉 | 自然力量之间的叠加——如灾害与疾病的连锁 |
| 超自然界内部交叉 | 两个超自然界压力源交叉 | 超自然规则之间的矛盾——如力量与禁忌的冲突 |

### 4.2 跨界交叉（3种）

| 交叉类型 | 定义 | 困境特征 |
|:---------|:-----|:---------|
| 社会×自然跨界 | 社会界与自然界压力源交叉 | 人为因素与自然力量叠加——如资源争夺引发生态灾难 |
| 社会×超自然跨界 | 社会界与超自然界压力源交叉 | 社会秩序与超自然力量冲突——如权力体系与力量体系的对抗 |
| 自然×超自然跨界 | 自然界与超自然界压力源交叉 | 物理法则与超自然规则碰撞——如灵气枯竭与生态崩溃 |

### 4.3 交叉困境记录格式

```
困境 #N：{压力源A} × {压力源B}
  - 交叉类型：{同界/跨界}（{具体标注：社会×社会 / 社会×自然 / 社会×超自然 / 自然×自然 / 自然×超自然 / 超自然×超自然}）
  - 困境描述：{两个压力源交叉产生的两难处境——详细描述}
  - 道德张力：{主角面临的道德选择——两个选项各有代价}
  - 可衍生剧情线：{这个困境可以衍生什么方向的故事}
```

### 4.4 交叉困境识别要点

1. **优先识别跨界困境**：跨界困境通常比同界困境产生更强的道德张力，因为涉及不同维度的力量冲突。
2. **每个交叉困境必须有道德张力**：不是简单的"两个困难叠加"，而是"两个压力源迫使主角在两个都有代价的选项间选择"。
3. **交叉困境数量≥3个**：其中至少1个跨界困境（质量红线❌4）。
4. **交叉困境必须可衍生剧情线**：每个困境标注可衍生的剧情方向，供调用方使用。

### 4.5 交叉困境示例

```
困境 #1：宗门资源垄断（社会界#1） × 灵气资源分布不均（自然界#3）
  - 交叉类型：跨界（社会×自然）
  - 困境描述：宗门占据灵气最浓郁的福地，散修只能在灵气稀薄的
     边缘地带修炼。资源天然分布不均被人为制度固化——散修不仅
     天赋不如宗门弟子，连修炼环境都被制度性剥夺。
  - 道德张力：主角发现一个未被宗门控制的灵气节点，但开采它
     会破坏当地生态平衡——修炼资源 vs 生态责任。
  - 可衍生剧情线：主角成为散修领袖，挑战宗门资源垄断体系。
```

## 5. 调研执行要点

### 5.1 搜索策略

- 多轮搜索（最多3轮），每轮聚焦不同维度
- 第1轮：广度搜索，覆盖所有涉及维度
- 第2轮：深度搜索，针对第1轮发现的关键压力源深挖
- 第3轮：交叉验证，确认数据准确性+补充案例

### 5.2 案例要求

每个压力源和交叉困境必须包含具体案例：
- 真实历史案例（适用于社会界/自然界调研）
- 同类作品参考（适用于超自然界调研）
- 具体的流程步骤（如权力更迭流程、资源争夺模式）
- 可量化的数据数值（如贫富差距系数、灾害频率、力量分级标准）

### 5.3 产出质量

- 每篇知识沉淀≥500字（红线❌2）
- 必须含具体案例/流程步骤/数据数值（红线❌2）
- 压力源≥3个（红线❌4质量检查）
- 交叉困境≥3个，至少1个跨界（红线❌4质量检查）
