## 2. 章内笔触 DNA

### 2.1 叙事距离

适用章型：

- 开篇、高位展示、终局回环可以拉远。
- 反击、探索、对峙应贴近主角行动。

表现方式：

- 拉远时使用群体、历史、空间尺度做背景。
- 贴近时按主角看到、判断、动作、反馈推进。

执行约束：

- 必须：根据章型切换叙事距离。
- 禁止：全章都用主角内心解释宏大设定。

证据章节：

- CH001 从宇宙/航天机构拉远。
- CH156 贴近叶凡动作链。
- CH700 用旁观者和圣级尺度拉远。

### 2.2 句式与段落呼吸

适用章型：

- 战斗/反击：短句、拟声词、独句段。
- 开篇/高位展示：长句铺陈 + 关键独句段。
- 过渡/探索：中等长度段落承接信息。

表现方式：

- 动作段：动作 -> 声音 -> 结果。
- 定级段：长句说明来历/尺度。
- 独句段：用于爆点、转折、钩子。

执行约束：

- 必须：短句只服务动作或转折。
- 禁止：为了像网文而无功能地切碎段落。

### 2.3 物象写法

适用章型：

- 开篇、探索、揭秘、章末钩子。

表现方式：

- 先出现具体物件/地点/异常痕迹。
- 再通过来历、反常、他人反应增加信息量。
- 最后让物象指向下一步行动或压力。

执行约束：

- 必须：物象要能推动剧情或改变判断。
- 禁止：只堆古老、神秘、宏大等形容词。

### 2.4 对话写法

适用章型：

- 对峙章、情绪补偿章、调节过渡章。

表现方式：

- 对峙：短句施压，态度比解释更重要。
- 调节：口语化，缓冲高压。
- 旁观者：负责定级和传播结果。

执行约束：

- 必须：不同功能角色用不同说话方式。
- 禁止：把庞博/黑皇式口吻扩散到所有角色。

### 2.5 情绪外化

适用章型：

- 打脸、反杀、旧友重逢、章末悬置。

表现方式：

- 情绪通过动作、沉默、敌人反应、旁观者变化外化。
- 不长篇抒情，尤其在战斗和反击中。

执行约束：

- 必须：情绪要有可见动作或局势变化。
- 禁止：只写"他很愤怒/震惊/悲伤"。

## 3. 单章剧情套路 DNA

### 3.1 开篇/世界松动章
chapter_type: `opening_shift`

使用条件：

- 第一章，或世界规则发生重大松动。

章内读者期待：

- 先相信现实，再被一个无法解释的物象打开更大世界。

章内节拍：

1. 起笔：现实常识、科学/日常/机构秩序。
2. 第一信息变化：出现不合常理的数据、痕迹或目击。
3. 压力/诱因：权威或群体无法解释。
4. 主角动作或判断：主角/机构接近异常。
5. 阻碍升级：异常不是孤例，而是更大物象的一角。
6. 可见反馈：群体失语、机构失效、常识崩裂。
7. 章末钩子：异常物象继续逼近或呼叫下一层真相。

笔触绑定：

- 叙事距离：前半可拉远，异常出现后聚焦。
- 句式段落：前稳后紧，关键物象独句段。
- 对话方式：少量权威/群体反应即可。

禁止误用：

- 不要每章都用机构证据链。
- 不要搬运九龙拉棺、NASA、修仙词。

### 3.2 探索/资源露头章
chapter_type: `exploration_gain`

使用条件：

- 主角进入新地点、废墟、危险区域，发现资源或线索。

章内读者期待：

- 危险和收益同时出现，读者期待主角敢不敢继续深入。

章内节拍：

1. 起笔：环境反常或生物异常。
2. 第一信息变化：资源/线索露头。
3. 压力/诱因：危险阻挡，或别人不敢拿。
4. 主角动作或判断：判断收益，选择冒险。
5. 阻碍升级：危险扩大或更深处异动。
6. 可见反馈：资源到手、能力变化、信息变化。
7. 章末钩子：更深处还有更大变化。

笔触绑定：

- 叙事距离：贴近主角观察和判断。
- 句式段落：环境中段铺开，动作段收紧。
- 对话方式：同伴问答可用于显化风险。

禁止误用：

- 不能只写观光式环境。
- 资源必须改变下一步选择。

### 3.3 对峙/压力章
chapter_type: `confrontation_pressure`

使用条件：

- 强弱位阶对话、势力试探、旧怨重逢、谈判。

章内读者期待：

- 读者想看主角如何在低位压力下不失控，并拿回主动。

章内节拍：

1. 起笔：敌方/高位者给压力。
2. 第一信息变化：对方暴露目的、筹码或误判。
3. 压力/诱因：语言羞辱、身份压制、条件逼迫。
4. 主角动作或判断：沉默、反问、亮筹码、出手前置。
5. 阻碍升级：对方加码或旁人围观。
6. 可见反馈：对方失态、旁观者改观、局势微反转。
7. 章末钩子：矛盾升级到下一场行动。

笔触绑定：

- 叙事距离：贴近主角反应，穿插旁观者定级。
- 句式段落：对话短，动作/心理少而准。
- 对话方式：强者短句，弱者试探，主角少解释。

禁止误用：

- 没有前置压力，不要强行打脸。
- 不要把旁观者写成连续喊震惊。

### 3.4 战斗/反击章
chapter_type: `combat_reversal`

使用条件：

- 主角被压制、追杀、轻视，或需要打穿误判。

章内读者期待：

- 读者想看到"原本判断被打穿"。

章内节拍：

1. 起笔：明确对方优势或主角劣势。
2. 第一信息变化：主角发现缝隙/变量。
3. 压力/诱因：对方攻击、羞辱、封路。
4. 主角动作或判断：不解释，直接执行。
5. 阻碍升级：对方生命力/后手/位阶压制超预期。
6. 可见反馈：身体损伤、法器破碎、敌人失态、旁观者改判。
7. 章末钩子：反击引来更高后果，或必须立刻撤离。

笔触绑定：

- 叙事距离：贴近动作。
- 句式段落：短句链、拟声词、结果句。
- 对话方式：少说，胜负后再补一句狠话或收束。

禁止误用：

- 不要在动作高峰插长篇哲理。
- 不要没有反馈地写"很强"。

### 3.5 高位展示章
chapter_type: `high_tier_display`

使用条件：

- 远超主角当前层级的战斗、人物、规则或灾难出现。

章内读者期待：

- 读者想看世界上限被抬高。

章内节拍：

1. 起笔：高位存在出场或气息压场。
2. 第一信息变化：普通判断失效。
3. 压力/诱因：旁观者或环境承受不住。
4. 主角动作或判断：观察、避让、借势、立志。
5. 阻碍升级：高位行为造成大范围后果。
6. 可见反馈：天地/组织/群众/远处影响变化。
7. 章末钩子：主角未来必须抵达或对抗此层级。

笔触绑定：

- 叙事距离：拉远，用群体和空间尺度定级。
- 句式段落：长句铺陈，关键处独句段。
- 对话方式：旁观者少量定级。

禁止误用：

- 不要拆成普通拳脚动作。
- 不要让主角强行解释所有规则。

### 3.6 揭秘/章末钩子章
chapter_type: `reveal_hook`

使用条件：

- 规则揭露、旧伏笔变义、新敌人或新地点打开。

章内读者期待：

- 读者想知道"刚才发生的事原来意味着什么"。

章内节拍：

1. 起笔：上一场结果或异常余波。
2. 第一信息变化：新解释出现。
3. 压力/诱因：解释带来更大危险。
4. 主角动作或判断：选择追查/隐瞒/撤离/反击。
5. 阻碍升级：新信息牵出更高层人物或地点。
6. 可见反馈：原有关系、组织判断或资源目标改变。
7. 章末钩子：下一章必须进入新场面。

笔触绑定：

- 叙事距离：可从主角贴身转向更大信息面。
- 句式段落：信息句清楚，钩子句短。
- 对话方式：问答可承担揭秘，但不要百科化。

禁止误用：

- 不要一次解释完所有设定。
- 钩子必须改变下一章行动，不只是制造谜语。

## 4. 角色与口癖隔离

### 主角表达

功能：

- 承担判断、行动、反击。

可迁移：

- 少解释，靠行动改写局势。

不可迁移：

- 原作境界词、招式名、玄幻身份。

### 调节角色表达

功能：

- 高压后的口语缓冲。

可迁移：

- 损友式反应、轻微吐槽。

不可迁移：

- 让所有角色都变成庞博/黑皇。

### 旁观者表达

功能：

- 章内反馈和重新判断。

可迁移：

- 用一句反应说明事件等级变化。

不可迁移：

- 连续喊震惊、骇然、不可思议。

### 作者口癖

功能：

- 可作为轻微气口。

可迁移：

- 四字结构、独句段、短促感叹。

不可迁移：

- 只靠这些制造假宏大。

## 5. 涌现式消费契约

### 写入 soul 的长期笔触偏好

- 可写：叙事距离、句子气口、对话气质、氛围边界、禁止迁移项。
- 不写：单章具体章型、全书位阶结构、原作世界观。

### 写入 current-state 的本章DNA执行包示例

```markdown
## 本章DNA执行包
- DNA源：写作资产/文风库/遮天-章内文风DNA-v5.1.md
- 本章DNA模式：强嫁接
- 本章章型：confrontation_pressure
- 本章笔触目标：对话短、压力强，主角少解释，用动作和旁观者反馈显化重新判断。
- 本章章内套路：
  1. 起笔：高位机构/人物压低主角身份。
  2. 第一信息变化：对方暴露误判或隐藏目的。
  3. 压力/诱因：用权限、身份或规则逼迫主角退让。
  4. 主角动作或判断：主角不争辩，拿出一个事实/行动打破判断。
  5. 阻碍升级：对方加码，试图把主角重新压回低位。
  6. 可见反馈：旁观者或系统记录重新评估主角。
  7. 章末钩子：更高层级的人或异常因此盯上主角。
- 禁止误用：不搬运修仙词；不让所有人连续震惊；不写全书位阶设定。
- 验收点：本章至少一次"外部重新判断"；章末钩子改变下一章行动。
```

## 6. 验收

| 项 | 合格 | 不合格 |
|---|---|---|
| 笔触 | 500 字内能看出叙事距离、句式、对话气口 | 只换宏大词 |
| 章内套路 | 起笔、信息变化、压力、动作、反馈、钩子完整 | 只有氛围，没有推进 |
| 反馈 | 角色/旁观者/局势出现可见变化 | 作者总结"很强" |
| 隔离 | 角色口癖和特殊桥段被限制 | 把开篇或角色台词当通用逻辑 |
