# 名词索引提取协议 v1.1

> Phase A 核心工具：提取名词+首次章节+原文描述，不做判断。
> 机械操作，完美并发。

---

## 一、输出格式

每 100 章一个 YAML 文件，每条 4 个字段：

```yaml
nouns:
  - name: "法鉴"
    type: "器物"
    first_ch: "第1章"
    desc: "一面巴掌大的青灰色铜镜，正面碎裂成七八块，靠铁箍箍着未散架。陆江仙穿越后成为此镜的器灵，可吸收月华修炼。"
```

**核心变化：** `note` 改为 `desc`。`desc` 必须是**原文中第一次定义/描述该名词的句子摘录**。如果原文没有明确的定义句，则用一句话概括原文中的信息。

## 二、13 种名词类型

| 类型 | 含义 | 记什么 | 不记什么 |
|:----|:----|:------|:--------|
| 人物 | 有名字的角色 | 所有有名有姓的修士/凡人/妖物 | "一个散修""一个村民" |
| 器物 | 法器/法宝/阵/特殊物品 | 有名字或有品级的 | 普通兵器（刀剑弓） |
| 功法 | 修炼法门/剑诀/步法 | 有正式名称的功法 | 普通招式 |
| 法术 | 法术/神通 | 有名称的法术 | 无名称的普通攻击 |
| 丹药 | 丹药/灵果/灵植 | 有名字的 | 普通药材 |
| 天地灵气 | 修炼用特殊灵气 | 太阴月华/湖中金秋等 | — |
| 境界 | 修为等级 | 出现时记 | — |
| 地域 | 地点/坊市/洞府 | 有名字的 | 路过的小村庄（除非反复出现） |
| 势力 | 家族/宗门/教派 | 有名字的势力组织 | 临时队伍 |
| 事件 | 有名称的大事件 | 如"仙府传人之乱" | — |
| 设定 | 世界观规则/概念 | 灵窍/符种/箓气等 | — |
| 生物 | 妖物/灵兽 | 有名字或有境界的妖物 | — |
| 果位 | 道果/法则 | 涉及道的概念 | — |

## 三、desc 字段提取规则★★★★★

这是本协议最重要的部分。

### 3.1 desc 的获取方式

**优先方式：摘录原文定义句**

在原文中找到第一次介绍/定义该名词的句子，直接摘录。

示例（原文出现了）：
> 原文："那蛇蛟果乃是一条练气修为异蛇守护的灵根，五年一熟，果成赤红色，生吃可缩短养轮时间。"

输出：
```yaml
  - name: "蛇蛟果"
    type: "丹药"
    first_ch: "第25章"
    desc: "练气修为异蛇守护的灵根，五年一熟，果成赤红色，生吃可缩短养轮时间。"
```

**备选方式：一句话概括**

如果原文没有明确的定义句（名词出现时没有解释），则基于上下文概括一句。

```yaml
  - name: "李通崖"
    type: "人物"
    first_ch: "第2章"
    desc: "李木田次子。提出立即杀李叶盛灭口，性格果决。后修炼太阴吐纳养轮经，成就筑基，仙基浩瀚海。"
```

### 3.2 desc 的写法规范

- **不允许**写"重要角色""关键人物"等主观判断
- **不允许**写"此处埋下伏笔""后来将..."等前瞻信息
- **允许**引用角色原话（加引号）
- **允许**写这个物品/功法在原文中的具体效果

```
好的 desc：
  "一面巴掌大的青灰色铜镜，正面碎裂成七八块"
  "练气修为异蛇守护的灵根，五年一熟，赤红色，生吃可缩短养轮时间"
  "三品步法。李通崖从郁慕剑配剑投影中获得。附《血遁术》（消耗精气寿命逃遁）"

不好的 desc：
  "重要法器"                            ← 缺乏具体信息
  "前期反派"                            ← 主观判断
  "后来成为李家核心人物"                  ← 前瞻信息（还没读到）
  "这里为后续李尺泾之死埋下伏笔"           ← 推测
```

### 3.3 desc 的长度

- **人物**：1-2句话（身份+关键特征）
- **器物/功法/丹药**：1-2句话（外观/品级/效果）
- **地域**：1句话（在哪+什么特征）
- **势力**：1句话（谁+什么级别）
- **设定**：1-2句话（这是什么规则）
- **事件**：1句话（发生了什么）
- **境界**：1句话（第几个大境界+特征）

## 四、提取规则

### 4.1 提取频率

- **首次出现**：必须记，`first_ch` 标注首次章节
- **后续出现**：不记。除非有重要变化（如角色突破修为）
- **每章新增名词数**：一般在 2-10 个之间

### 4.2 去重处理

子代理负责自己那 100 章的去重。同一名词在 100 章内多次出现，只记一次（首次）。

跨子代理的全局去重在 Phase A 完成后由合并步骤处理。

### 4.3 全称优先

用小说中使用的**正式全称**：

```
✓ "《江河一气诀》"
✗ "江河一气诀"（缺书名号）

✓ "青池宗"
✗ "青池"
```

## 五、错误模式

| 错误 | 反面示例 | 正确做法 |
|:----|:--------|:--------|
| 漏记路人但有名字 | 跳过"陈二牛" | 记：陈二牛→人物 |
| 记了无名字的 | "一个青年修士" | 不记 |
| desc 是感想 | "写得很好的一把剑" | 摘录原文描述 |
| desc 是前瞻 | "后来成为李家核心" | 只记当前出现时的信息 |
| 同名词重复记 | 第10章记一次第50章又记 | 只记首次 |
| desc 全是空话 | "一个角色" | 至少写身份+特征 |
