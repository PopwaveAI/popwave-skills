# Phase B -- 跨卷关联

> 多卷读完后打通全貌。

---

## 执行步骤

| 项目 | 内容 |
|:-----|:------|
| 读什么 | 全部卷的叙事笔记 + 结构化 YAML |
| 做什么 | 打通整书的理解：因果链/伏笔线/剧情线/书特有机制 |
| 产出什么 | `{书名}_跨卷关联笔记.md` |
| 详细指令 | `references/phase_b_分析协议.md` |

### 跨卷关联笔记内容

- **跨卷角色轨迹**：同一角色在不同卷的状态变化表
- **跨卷设定演变**：关键设定在多卷中的展开
- **宏观叙事结构**：分卷之间的衔接方式
- **伏笔/未解问题**：尚未回收的伏笔

---

## Phase B+ -- 方案复盘验证

对比 Phase 0 分卷 vs 阅读后的实际感受。

三种结果：
- 分卷准确 -> 进 Phase C
- 边界偏差 -> 微调分卷
- 重大偏差 -> 回到 Phase A 补读
