# pop-snowflake · 雪花式写作方法

> 从一句话到完整小说的迭代式扩展法。10步双轨并行：故事架构与人物塑造交替生长，从25词种子逐步扩展为完整长篇。
> 产出：完整大纲 + 人物宝典 + 场景清单 + 小说正文

## 红线

1. 每一步的产出必须基于前一步的产出扩展——禁止跳步或凭空重写，雪花法的核心是迭代生长
2. 三次灾难性事件缺一不可——第一幕末、第二幕中、第二幕末各一次，是推动主角投入和转变的结构支柱
3. 每个场景必须包含矛盾冲突——无冲突的场景要么加上冲突要么删除，禁止"烘托气氛"型废场景
4. 主动场景（目标→冲突→挫折）与被动场景（反应→困境→决定）必须交替串联——决定即下一幕主动场景的目标

### 核心方法论

**迭代式扩展**：25词→5句→1页→4页→完整小说，每步都是前一步的展开而非重写。

**双轨并行**：故事架构（步骤2→4→6→8→9）与人物塑造（步骤3→5→7）交替推进，互为支撑。

**冲突驱动**：灾难性事件驱动主角投入和转变；矛盾冲突驱动每个场景前进。

**场景循环**：主动场景（目标→冲突→挫折）→ 被动场景（反应→困境→决定）→ 主动场景……决定即下一幕的目标。
