## 场景类型标注

- **A** = 主动场景（目标→冲突→挫折）
- **R** = 被动场景（反应→困境→决定）

## 核心规则

1. 每个场景**必须**有矛盾冲突——无冲突则加冲突或删场景
2. 禁止仅为"烘托气氛""解释背景""阐明动机"而添加场景
3. 视点人物 = 通过他的视角来描述该场景
4. 主动场景与被动场景交替串联（允许部分跳过被动场景）

## 场景筛选标准

| 情况 | 处理 |
|------|------|
| 场景有实质冲突 | 保留 |
| 场景无冲突但可加入 | 加冲突后保留 |
| 场景无冲突且无法加入 | 删除 |
| 场景仅烘托气氛/解释背景 | 删除或合并到相邻场景 |
