# 元爽点 → 套路变体映射表

> 用途：creative 产出爽点引擎.md 后，plot Step 1 配套路时按元爽点筛选变体套路。
>
> 每条 = 一个元爽点 + 其所有变体套路（可查套路库中的具体文档）。
>
> 来源：pop-writer-creative v3.0.0 Phase 爽点引擎 + pop-trope-library 原始素材库。

## 元爽点全景

| 元爽点 | 本质 | 用户信号 | 全书节奏定位 |
|:-------|:-----|:---------|:------------|
| 信息差 | 我知道你不知道→我利用这个赢了 | "喜欢主角布局""读者跟着线索发现真相" | 短周期驱动（每3-5章） |
| 阶级革命 | 底层掀翻上位者→共情快感 | "主角推翻旧世界""和吃人世界对着干" | 长周期主线（每卷一次高潮） |
| 绝境反杀 | 以为要死了→不但没死还赚了 | "主角被逼到墙角再翻盘" | 脉冲节奏（大冲突结尾爆发） |
| 身份跃迁 | 从没人看得起→被万众瞩目 | "主角从底层爬起来""逆袭打脸" | 中周期弧线（幕级成长闭环） |
| 世界观震撼 | 认知刷新→世界比想象的大 | "世界不只两层""力量都是钩子" | 卷级锚点（每卷一次认知刷新） |
| 智斗博弈 | 多势力互相算计→谁棋高一着 | "多股势力互相利用""谁才是棋手" | 幕级高潮（每幕一次格局反转） |
| 情感羁绊 | 关系从对立→信任/从疏远→亲密 | "某个配角让人破防""师徒/战友情" | 跨卷长线（全书级情感锚点） |

## 使用方式

### creative（写爽点引擎时）

从表中找到用户确认的元爽点 → 把对应变体家族写入爽点引擎.md → 为后续套路配套提供锚点。

### plot Step 1（配套路时）

从剧情线设计文档引用的元爽点 → 查本表找到对应变体家族 → 从 pop-trope-library/原始素材/{书}/ 中读具体套路文档 → 选匹配当前剧情阶段和情绪基调的变体。

### chapter-design（设计爽点时）

从幕锚点的元爽点目标 → 查本表确认当前幕可用的变体 → 部署到具体章节的爽点机制表。
