# 剧情库 L2 单元卡分类启发式

> 用于 L2 单元卡入库时判定所属标签。
> 核心原则：**看单元的核心叙事功能是什么，而非它发生在哪一章。**

---

## 现有标签及适用范围

| 标签 | 核心叙事功能 | 典型 L2 特征 | 占比参考 |
|:-----|:------------|:------------|:--------:|
| **崛起征服** | 主角的实力跃迁、身份升级、势力扩张；从底层到上位 | 练级/升级/BOSS战/称号获得/新能力解锁 | ~35% |
| **反派对抗** | 与具体反派的正面冲突、博弈、高潮对决 | BOSS战/暗杀/战争/猎杀/封神阻止 | ~30% |
| **设定揭示** | 世界观底层规则、隐藏势力、力量体系的首次披露 | 信息揭露/世界观设定章/真相大白 | ~10% |
| **势力建设** | 领地建设、团队组建、经济体系搭建 | 基建/经营/招募/贸易/防御战 | ~10% |
| **血脉觉醒** | 角色天赋/血脉/神性的觉醒与质变 | 天赋解锁/双人格/神性觉醒 | ~5% |
| **情感联盟** | 角色关系（恋情/友情/家人）的建立与深化 | 告白/赠礼/羁绊/分离/重逢 | ~5% |
| **阶级揭露** | 社会制度、阶级结构、权力体系的批判性展示 | 制度揭示/社会真相/立场反转 | ~5% |

---

## 判定流程（逐问题过滤）

```
Q1: 本单元的核心叙事是「主角变强/身份升级」吗？
  → 是 → 崛起征服
  → 否 → Q2

Q2: 本单元的核心叙事是「与反派的正面对抗/高潮」吗？
  → 是 → 反派对抗
  → 否 → Q3

Q3: 本单元的核心叙事是「世界观规则的首次揭示」吗？
  → 是 → 设定揭示
  → 否 → Q4

Q4: 本单元的核心叙事是「建设/经营/招募」吗？
  → 是 → 势力建设
  → 否 → Q5

Q5: 本单元的核心叙事是「角色天赋/血脉/神性觉醒」吗？
  → 是 → 血脉觉醒
  → 否 → Q6

Q6: 本单元的核心叙事是「关系建立/情感突破」吗？
  → 是 → 情感联盟
  → 否 → Q7

Q7: 本单元的核心叙事是「社会制度/阶级批判」吗？
  → 是 → 阶级揭露
  → 否 → 复用以上标签中最接近的
```

## 常见边界案例

| 案例 | 归属 | 理由 |
|:-----|:-----|:------|
| 开场建置单元（主角刚穿越/开局弱） | **崛起征服** | 叙事功能是崛起起点，不是其他 |
| BOSS战单元（击杀强敌后实力跃迁） | **崛起征服** 或 **反派对抗** | 看重心：若重心在「战后升级/装备/称号」→崛起征服；若重心在「对抗过程/反派刻画」→反派对抗 |
| 旅行/探索单元（新地图开荒） | **崛起征服** | 成长线功能（通过探索获得经验/资源/认知） |
| 阵营/系统介绍单元 | **设定揭示** | 世界观规则的首次说明 |
| 大规模战争单元（双方势力对抗） | **反派对抗** | 核心矛盾是势力冲突 |
| 割喉/名声扩散单元 | **崛起征服** | 声望升级是身份崛起的一部分 |
| 主角获得特殊天赋/力量觉醒 | **血脉觉醒** 或 **崛起征服** | 若是血统类觉醒→血脉觉醒；若是职业/技能类升级→崛起征服 |

## 多标签决策

一张 L2 卡可能同时具备多个标签特征。此时：

1. 取 **最贴合核心叙事功能** 的标签
2. 在 `00-索引.md` 的备注列标注「含 {其它标签} 要素」
3. 不自建复合标签（如 `崛起征服·反派对抗`）

例：L2-009 碎骨之战（军队战→BOSS击杀→升级）→ 核心叙事是对抗反派食人魔 → **反派对抗**，标注「含崛起征服要素」
