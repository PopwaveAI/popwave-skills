# 20章因果闭环（三层认出→截胡→世界升维→亲手复仇）

> 4章核心爆发段 + 前期4章铺垫。不是"变强回来复仇"——是"知道前世不知道的秘密，用信息差翻盘"。

## 事件链

```
[铺垫 ch009-012]
→ 主角以捕快身份重生回仇人的世界（前世被杀的城）
→ 修炼+世界观建置（兵器五等/外功六要素体系摊开）
→ 丹凤楼百人斩，灵识觉醒（实力锚点——他知道自己能打了）

[一层认出 ch013]
→ 前世师兄弟认出他：全城围杀 → 被一刀斩杀 → 死前认出"被抛弃的小师弟"
→ 前世师兄弟知道"他是谁"——但不知道"他知道了什么"

[二层认出 ch017-018]
→ 前世师父雨中截击 → 自曝死而复生 → "好徒弟，你居然还活着"
→ 主角趁势截胡莲华刀+龙兴县印（用情报差反抢）

[世界升维 ch018-019]
→ 主角在雨中厘清上线十年前布的全局——自己不是棋子，是弃子
→ 同步炸开秋税黑暗真相（查田→假官册→百官印是万民血）

[三层认出 ch020]
→ 前世上线被射杀 → 死前认出"他没死！？"（20章因果闭环完成）
→ 前世下令屠村的人，死前才知道自己杀的人回来了
```

## 结构解析

**三层认出的递进逻辑**：
同一个信息（"主角就是前世那个人"），被三个不同角色逐步认出。每层不是重复——是冲突升级：

| 层 | 认出者 | 新信息 | 冲突升级 |
|:--|:-------|:-------|:---------|
| 1 | 师兄弟 | 主角活着回来了 | 击杀 |
| 2 | 师父 | 主角知道前世秘密 | 拉拢 |
| 3 | 上线 | 主角来复仇了 | 死亡 |

**核心设计模式："信息差复仇"**
主角能复仇不是因为他变强了——是因为他知道了前世不知道的秘密。前世他是白莲教信徒，不知道县衙的计划；这一世他是捕快，能从县衙内部看全局。信息差是武器，力量值是工具。

**因果链**：
```
重生 → 修炼（获得战力）→ 复仇（不是终点）
    → 换身份（获得信息差）→ 看清全局（真正武器）
    → 三层认出（因果闭环）→ 亲手复仇（情绪终局）
```

**情绪设计**：
ch013 高潮(9) → ch014-016 回落(4-5) → ch017-020 连续高潮(8→9→10)
四章连爆不留缓冲——截胡→世界升维→亲手复仇，一口气打完。

## 复用要点

- 保留"三层认出"的递进结构：同一信息被不同角色在不同场景认出，每层有新信息+新冲突
- 保留"信息差 > 力量值"的逻辑：主角靠知道更多赢，不是靠等级高赢
- 保留"世界升维"的位置：复仇高潮之前——让读者先震惊于世界有多大，再震惊于复仇
- 替换：师兄弟/师父/上线 → 任何三层关系链。太平经重生 → 任何重生/轮回机制
