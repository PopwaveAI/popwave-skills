# LEGACY：旧卷幕切片勿作为写书消费入口

> 本目录下早期存在按卷、幕、章级单元入库的文件。
> 这些文件只保留为拆书证据，不再作为 `pop-writer-plot` 或 `pop-writer-chapter` 的直接参考。

## 新规则

剧情库主入口只读：

```text
剧情库/{书名}/剧情线/
剧情库/结构模型/剧情线/
剧情库/角色弧线/
剧情库/章法模型/
```

## 旧文件处理

旧的 `幕*.md`、`卷*.md`、`Uxx-chxxx.md` 文件待重提取为剧情线。

在重提取完成前，任何 writer skill 都不得把旧卷幕切片当作模板、结构参考或剧情规划依据。
