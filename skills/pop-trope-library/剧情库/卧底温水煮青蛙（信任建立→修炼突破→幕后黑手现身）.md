# 卧底温水煮青蛙（信任建立→修炼突破→幕后黑手现身）

> 5章封闭弧线。不是"卧底套情报"——是"帮对方变强，对方主动给"，最后发现对方身后还有更大的局。

## 事件链

```
→ 主角以"家庭教师"身份进入目标家中（卧底开始）
→ 不主动套话——先让对方观察自己

→ 展示实力（帮对方解决修炼瓶颈——不是"我来解决"，是"你自己可以解决"）
→ 对方从克制→尊重→信任（第一阶段：他好像真的有实力）
→ 对方从信任→指导→依赖（第二阶段：他帮我突破了）

→ 对方主动说出秘密（不是被逼问/套话——是自愿的）
→ 秘密引导主角发现对方的修炼法门

→ 主角在对方家中修炼突破（外功六关全破）
→ 对方沦陷告知真名——信任峰值

→ 修炼中触发的法器碎裂——暴露了法门背后还有原主
→ 原主现身（幕后黑手：对方的幕后老板来回收资产）
→ 主角的卧底身份在这种博弈面前无关紧要（不是"被发现了"——是"被更大的局吞了"）

→ 对决：价值观对撞（"黔首也能算人？""但是我拒绝！"）
→ 第三方黄雀现身介入——带走主角（钩子）
```

## 结构解析

**核心设计模式："帮对方变强，而非套话"**
卧底信任建立 = 克制的五阶段：克制→尊重→信任→指导→依赖。主角从头到尾没有套话——他做的一切都是帮对方变强。对方之所以沦陷，是因为太久没有人真心帮她变强。

**身份暴露的非传统设计**：
不是"目标发现你是卧底"——是目标的幕后老板现身，你只是他回收资产时的附带损失。卧底身份在这种级别的博弈里无关紧要。

**因果链**：
```
卧底进入 → 展示能力（非主动套话）→ 对方自愿交出秘密
    → 修炼突破 → 触发性暴露（不是被发现，是被更大的局吞掉）
    → 幕后黑手现身 → 对决 → 第三方介入
```

**情绪设计**：
ch023 好奇(3) → ch024-025 上升(4→6) → ch026 质变(7) → ch026末 反转(8) → ch027 对抗(9) → ch028 高潮→缓冲(9→4)

慢热上升（信任建立2-3章），然后一次反转+一次高潮连爆。

## 金句

> "黔首也能算人？""城外哪里有灾民，只有一群产粮的黔首""但是我拒绝！" — ch027

## 复用要点

- 保留五阶段信任模型（克制→尊重→信任→指导→依赖），不做跳过
- 保留"帮对方变强"而非"套话"的卧底逻辑
- 保留"暴露不是被识破，是被更大局吞掉"的非传统暴露设计
- 替换：家庭教师/苏夫人 → 任何一对一信任建立场景。功体图 → 任何对方持有的核心秘密/法器
