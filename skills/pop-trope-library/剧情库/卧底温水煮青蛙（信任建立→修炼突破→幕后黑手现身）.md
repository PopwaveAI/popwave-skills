# 卧底×温水煮青蛙（信任建立→修炼突破→身份暴露→幕后黑手现身）

> 来源：让魔门再次伟大 ch023-028
> 一个 2-3 章剧情单元。不是传统卧底片的"偷情报"，是"在目标家里修炼变强"，最后发现目标不是目标——她身后还有人。

## 剧情流程

| 段 | 章 | 做什么 | 具体 |
|:---|:---|:-------|:-----|
| **进入** | ch023 | 以"家庭教师"身份进入苏家 | 王平在苏夫人耳边说悄悄话（留白，读者不知道说了什么）。苏夫人从敌对→好奇→接受 |
| **信任** | ch024 | 温水煮青蛙式的信任升级 | 克制→尊重→信任→指导→依赖。苏夫人主动说出城西肉铺藏秘籍。不是王平在逼问——是她自愿给的 |
| **突破** | ch025-026 | 在目标家里修炼到突破 | 一夜七次质变，外功六关全破。苏夫人沦陷告知真名"苏雅"。发簪碎裂露出天鹏擒龙功体图——原来她身后还有人 |
| **暴露** | ch026末 | 功体图原主现身 | 徐秉正现身——他不是来抓王平的，他是来回收功体图的。王平这才知道：苏夫人只是保管人，徐秉正才是功体图的原主+苏家灭门的幕后黑手 |
| **对决** | ch027-028 | 价值观对撞→武力对决→第三方介入 | "黔首也能算人？""但是我拒绝！"。大战。守冲黄雀现身——"好徒弟，我果然没有看错你"——带走王平 |

## 核心设计

### 温水煮青蛙的信任建立

不是"卧底用高超演技骗过目标"——是**目标自己主动给**。

| 阶段 | 苏夫人的变化 | 王平做了什么 |
|:-----|:------------|:-----------|
| 克制 | "这人是谁？" | 不主动，先让对方观察 |
| 尊重 | "他好像真的有实力" | 展示武功，不炫耀 |
| 信任 | "他说的有道理" | 指出苏夫人练功的问题，不是批评，是"你少了一步" |
| 指导 | "你教我" | 帮苏夫人突破瓶颈——不是"我来解决你的问题"，是"你自己可以解决" |
| 依赖 | "这是我的秘密" | 苏夫人主动说出城西肉铺藏秘籍——不是王平逼问出来的 |

**关键**：王平从头到尾没有"套话"。他做的一切都是"帮苏夫人变强"。苏夫人之所以沦陷，是因为**她太久没有人真心帮她变强了**。

### 身份暴露不是"被识破"——是"被更大的局吞掉"

传统卧底片的暴露 = 目标发现你是卧底。这里的暴露 = 目标的幕后老板现身，你只是他回收资产时的"附带损失"。

徐秉正不是来抓卧底的——他是来回收功体图的。王平的卧底身份在这种级别的博弈面前无关紧要。

## 可迁移方向

| 你的书设定 | 怎么用 | 保留什么 |
|:----------|:-------|:---------|
| 卧底/间谍 | 主角潜入某个组织→以"帮助"而非"套话"的方式建立信任→对方主动交出秘密 | 温水煮青蛙的五阶段信任模型 |
| 修仙 | 主角伪装成散修进入某个世家→帮世家小姐突破瓶颈→她主动拿出家传功法 | 目标主动给，不是主角偷 |
| 黑暗流 | 主角被派去"照顾"某个被软禁的人→双方互相同情→被照顾的人主动给出能推翻软禁者的信息 | 囚禁版——两个囚徒的信任建立 |
| 宫廷文 | 主角以"新来的教习"身份进入后宫→帮失宠的嫔妃重新得宠→嫔妃主动给出宫廷秘密 | 后宫版——帮对方获得权力，对方给秘密 |

## 使用红线

| ✅ 必须做 | ❌ 不要做 |
|:---------|:---------|
| 信任是对方主动给的——不是主角骗来/偷来/逼来的。主角做的一切都是让对方变强 | 主角用催眠/魅惑/欺骗手段获取信任（那是反派做法） |
| 暴露不是"被目标识破"——是目标身后更大的局吞掉了主角的卧底身份 | 主角被苏夫人发现"你是卧底！"然后分手（太俗） |
| 目标必须有独立的动机和故事——不是"等待主角拯救的公主" | 目标只是一个"给主角提供秘籍的NPC" |
| 温水煮青蛙的节奏要慢——至少2-3章铺信任建立，不能一跳到底 | 第一面就"我信任你，这是我的秘密" |
