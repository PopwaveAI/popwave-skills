# 无间道排查内鬼（自证→交换→被出卖灭口）

> 来源：让魔门再次伟大 ch001-004
> 一个封闭的 2-3 章剧情单元。可以独立嵌入任何有"卧底/内鬼"设定的开篇或转折点。

## 剧情流程

| 段 | 章 | 做什么 | 具体 |
|:---|:---|:-------|:-----|
| **排查** | ch001 | 组织头目宣布有内鬼。要所有人自证清白 | 守冲把教徒召集到大堂：查内鬼。王平提议"全体自裁以证清白"——反客为主，用极端提议自证。守冲因此任命王平为堂主 |
| **潜伏** | ch002-003 | 内鬼在敌人眼皮底下做事 | 王平拜师守冲学武功。同时暗中去县城接头汇报。上线通知"马上收网"。王平被告知"收网时会有人来救你" |
| **出卖** | ch003末-ch004 | 收网日——上线没有派人来救，反而下令灭口 | 官兵炸门。守冲刀抵王平眉心逼问"为什么要出卖我"。王平按约定"表忠心"——但没人来救。被官兵射杀。濒死时听到上线陈皓彦否认卧底身份，并下令"都杀了"屠村 |
| **收束** | ch004末 | 内鬼死了。但没有结束 | 王平在太平经空间苏醒——他还有第二次机会 |

## 核心设计

### 为什么这个单元"好"

不是"内鬼被发现→被抓"，而是"内鬼被自己的上线出卖"。

主角以为自己在玩双面间谍——但其实两边都抛弃了他。白莲教怀疑他是内鬼，县衙拿他当弃子。他直到临死前才听到上线那句"都杀了"——才知道自己从头到尾都是炮灰。

**读者的愤怒来自"他什么都不知道就被卖了"**——不是他背叛了谁，是所有人都背叛了他。

### 王平的处境递进

```
"我是内鬼"（读者知道，角色知道）
→ "我快暴露了"（紧张）
→ "上线说会派人救我"（暂时安心）
→ "官兵炸门了，救我的人呢？"（不对劲）
→ "没人来"（被卖了）
→ 临死前听到上线说"都杀了"（确认被卖）
→ 他不是在背叛别人——是所有人同时背叛了他
```

### 金句（都是原文）

> "人命关天，指的是那些住在县城里的人，是豪族，是士子，至于城外的那些黔首……他们也算人？" — 陈皓彦，ch004

> "都杀了。" — 陈皓彦，ch004（全书最短、最致命的命令）

## 情绪曲线

```
ch001: 紧张(7) → ch002: 缓和(5) → ch003: 再次紧张(8) → ch004: 绝望(9)
```

关键转折在 ch003→ch004：读者以为收网=主角逃生，实际收网=主角被灭口。

## 可迁移方向

| 你的书设定 | 怎么用 | 保留什么 |
|:----------|:-------|:---------|
| 宗门卧底 | 主角是魔门派到正道的卧底，魔门决定放弃这颗棋子→屠正道外门来"清理痕迹" | 自证→出卖灭口→重生复仇的结构 |
| 官场谍战 | 主角是双面间谍，两边都怀疑他→两边同时决定"清除隐患" | 两边都背叛的绝望感 |
| 黑帮火并 | 主角是帮派卧底，帮派上层为了保住更大的卧底→把他当弃子卖给敌对帮派 | "我以为两边都有人在救我，其实两边都要我死" |
| 系统流 | 主角是系统派到异世界的"任务者"，系统判定"任务失败率过高"→回收主角 | 被自己的"上线"背叛的冲击 |
| 修仙 | 主角是宗门暗子，宗门决定"暗子暴露价值更大"→故意泄露他的身份引发两派火并 | 从棋子到弃子的认知转变 |

## 使用红线

| ✅ 必须做 | ❌ 不要做 |
|:---------|:---------|
| 主角直到最后才确认被卖——读者感受和他同步 | 主角提前知道被卖（那就不是悲剧是蠢） |
| 出卖的决策要"理性"——不是反派情绪化，是"这是最优解" | 反派出卖主角因为他"坏" |
| 死前有一瞬间的确认——听到上线说出那句话 | 在混战中莫名其妙被杀 |
| 死后必须有一个"还能回来"的暗示——否则读者弃书 | 主角死得彻底没有后续——这不是单元是全书终 |
