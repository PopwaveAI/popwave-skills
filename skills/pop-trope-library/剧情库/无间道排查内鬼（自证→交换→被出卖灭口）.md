# 无间道排查内鬼（自证→交换→被出卖灭口）

> 4章封闭弧线。可独立嵌入任何有"卧底/内鬼"设定的开篇。

## 事件链

```
→ 组织头目宣布有内鬼，给自首窗口（建立紧张）
→ 内鬼提议"全体自裁以证清白"，极端提议反向自证（自证段落）
→ 头目表面任命内鬼为堂主，暗地派人盯梢（自证反而加深怀疑——反转1）
→ 内鬼趁接头获上线承诺"收网时会派人接应你"（缓期——但没说接应谁）
→ 收网日。头目刀抵眉心反问"为什么要出卖我"（头目一直知道——反转2）
→ 接应没来。上线下令"都杀了"灭口（内鬼从头到尾是弃子——反转3）
→ 死。但有第二次机会（钩子）
```

## 结构解析

**三层反转递进**：
- 反转1（ch001）：自证成功 → 但自证方式太极端，本身就是破绽。头目在第二层
- 反转2（ch004）：头目早就知道 → 但头目不是最大的威胁
- 反转3（ch004）：上线下令灭口 → 内鬼从头到尾是弃子。最高层

**核心设计模式："自证反成破绽"**
主角最聪明的一步 = 对面看穿他的一步。不是因为做得不够好，是因为做得太好。"全体自裁"这个提议让头目确定了内鬼身份——因为真正的忠臣不会想到用这种方法自证。

**因果链**：
```
极端自证 → 反而暴露 → 被盯梢不自知 → 以为安全 → 收网日被卖 → 死
```

**情绪设计**：
ch001 紧张 → ch002 缓和 → ch003 再次紧张（收网逼近）→ ch004 绝望最低谷

## 金句

> "都杀了。" — 上线，ch004

> "人命关天，指的是那些住在县城里的人，是豪族，是士子，至于城外的那些黔首……他们也算人？" — 上线，ch004

## 复用要点

- 保留"自证反成破绽"的因果：主角最聪明的一步 = 对面看穿他的一步
- 保留"上线出卖"而非"组织发现"的终局反转——敌人不在对面，在自己这边
- 保留"死后有第二次机会"的钩子，不写死结局
- 替换：白莲教/县衙 → 任何有卧底设定的组织，太平经 → 任何重生/续命机制
