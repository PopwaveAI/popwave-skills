# 无间道排查内鬼（自证→交换→被出卖灭口）

> 4章封闭弧线。可独立嵌入任何有"卧底/内鬼"设定的开篇。

## 剧情线

### ch001 · 排查

白莲教前村道观。老道士守冲把二十几个教徒召集到大堂，宣布：**有内鬼**。

不是猜测——是确定。县衙知道他们藏在这里，近期一定会动手。内鬼不揪出来，所有人都得死。

王平站在人群末尾。他知道守冲说的是对的，因为**内鬼就是他**。但他不能慌——慌就是死。

守冲给了一个时间窗：天亮之前自首的，从轻。天亮之后查出来的，全堂处置。

教众开始互相审视。有人提议搜身。有人说查谁最近去过县城。王平听着这些提议，每一个都精准地绕开了他——因为他们查的方向是"谁和县衙有接触"，而王平的接头方式是**老鼠传信**，不走县城。

但王平做了一个所有人没想到的事：他主动站起来，提议**全体自裁以证清白**。

这个提议太极端了。教众哗然，守冲也愣了。但正是因为太极端，反而成了最强的自证——真正的内鬼不会主动提议"全员自裁"，因为内鬼自己也得死。王平用这个极端提议把自己从嫌疑名单里摘了出去。

守冲当场任命王平为堂主。不是因为相信他——是因为**一个敢提议全体自裁的人，要么是忠的，要么是疯的，两种都比懦夫有用**。

### ch002 · 潜伏

守冲开始教王平武功。不是客气地教——是用"飞刀炸假山"的武力展示先镇住他，然后传功。这是典型的"先立威再施恩"——让你知道我有多强，再让你感激我愿意教你。

守冲同时命关门弟子谢石盯梢王平。**他并没有真的信王平**——任命堂主是权宜之计，盯梢才是真正的手段。

王平知道被盯梢。他不在乎。他需要的是时间——上线答应"收网时会派人救你"。他只要撑到收网就行。

这段里王平做了一件聪明事：他故意在谢石面前表现出对守冲的崇拜。不是谄媚——是恰到好处的"弟子对师父的仰慕"。谢石看到的是"这人真心拜师"，不是"这人在演戏"。

### ch003 · 交换

王平找到机会去县城草市接头。接头方式：在米铺唇语传话，上线在对面铁铺假装看货。

上线给了关键信息：**收网定在后日。届时会有人来接应你。你只需要在道观里拖住守冲**。

王平回到道观，开始准备收网日的布局。他需要让守冲在收网时待在道观里不出门——所以他主动汇报"县衙最近有异动，师父要小心"，让守冲觉得"道观是最安全的地方"。

但读者通过ch003的一个细节开始不安：上线说"会有人来接应你"——但**没有说接应的人是谁**。王平没问。他默认上线会安排好。

### ch004 · 出卖

收网日。官兵炸开道观大门。

王平按计划"拖住守冲"——但守冲比他想象的快。守冲一刀架在王平脖子上："**为什么要出卖我？**"

守冲早就知道。不是ch001才发现的——是从王平提议"全体自裁"那一刻就觉得不对。**太极端的自证本身就是破绽**。守冲命王平为堂主是为了把他放在眼皮底下盯。

但守冲没杀王平——因为守冲自己也走不了。官兵围了道观。守冲把莲华刀塞进王平怀里，嫁祸王平是"贼首"——**如果你要死，就当贼首死，总比当内鬼死有价值**。

然后官兵放箭。

王平中箭。倒地。没死透。他听到一个声音——是上线陈皓彦的声音。陈皓彦走进道观，对官兵说了一句话：

**"都杀了。"**

不是"救他"。不是"带走内鬼"。是——都杀了。

王平在这一刻才明白：**他从来不是"被发现的内鬼"，他是从一开始就被设计好的弃子**。上线从来没打算救他。"会有人来接应你"是一句空话——接应他的人就是杀他的人。

陈皓彦还说了另一句话。有官兵问"城外那些村民怎么办？"陈皓彦说：

**"人命关天，指的是那些住在县城里的人。至于城外的黔首……他们也算人？"**

然后是屠村。

一个叫林晟的校尉试图阻止："奉诏讨贼，讨的是贼，不是民！"——但他阻止不了。

王平在尸体堆里断气。最后听到的声音是村外的惨叫。

## 因果链

```
王平提议全体自裁（自证）
  → 守冲反而怀疑（太极端=破绽）
  → 守冲命谢石盯梢（表面信任，实际监控）
  → 王平以为自证成功，继续潜伏
  → 收网日，上线说"都杀了"
  → 王平才明白：从头到尾他就是弃子
  → 死。但太平经给了第二次机会。
```

关键：**主角的自证聪明反被聪明误**。他以为自己用极端提议骗过了守冲，但守冲比他更老辣——"太干净的自证本身就是最大的疑点"。而主角直到死前才知道，他真正的敌人不是守冲的怀疑，是上线从一开始就没打算让他活。


## 复用要点

- 核心设计：主角不是"被发现"——是被自己的上线出卖。上线从一开始就要他死，"会有人接应你"是空话。
- 自证反成破绽：主角以为自己聪明，但对面更老辣。
- 死后必须有"还能回来"的暗示——否则读者弃书。
- 替换世界观时保留：自证→潜伏→出卖→灭口→重生 的因果链。替换：白莲教/县衙 → 任何有卧底设定的组织。
