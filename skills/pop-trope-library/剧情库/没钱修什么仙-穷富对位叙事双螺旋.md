# 没钱修什么仙·穷富对位叙事双螺旋（跨幕设计）

## 原剧情概述
卷1的底层叙事动力来自一种持续的"对比"——穷人和富人（仙都精英）的同台竞技。每个关键事件都有双重视角：穷人视角（张羽/白真真）和富人视角（夜凌霄/云景/乐沐岚/钱深等），形成一种"双螺旋"叙事结构——两条线在同一个事件中并行推进，各自展现其阶级底色。

## 改建思路
**穷富对位不是"反派vs主角"，而是"两种生存哲学的平行展示"**。

六级对位设计（贯穿全卷）：
| 场景 | 穷人表现 | 富人表现 | 对位效果 |
|------|---------|---------|---------|
| 法赛 | 张羽第4，倾尽全力+策略 | 三大名校霸榜前十，用药自由 | "穷人靠智力和拼命才能摸到富人的屁股" |
| 武道竞赛 | 白真真"没用全力"隐藏灵根 | 乐沐岚直播间服药+腿骨断裂硬撑 | "穷人藏着底牌怕暴露，富人在表演拼命" |
| 道心测试 | 张羽百亿不倒→"我要白真真" | 玉星寒合同13亿枷锁，云景50亿家族责任 | "穷人宁愿不要也不交换朋友，富人背负天价代价" |
| 筑基考试·神灵根 | 张羽蹭药到极限（道心89.6%） | 夜凌霄服分神灵双重思维双倍功法 | "穷人用药只能补短板，富人可以扩上限" |
| 筑基考试·法骸 | 张羽79%同步率（道心8级） | 夜凌霄86%同步率（道心10级） | "穷人的身体是自己修的所以抗拒改造，富人的身体是钱堆的所以兼容性好" |
| 缚龙索决斗 | 张羽拖字诀+圣体切换+毒雾爆发 | 夜凌霄分神灵三重思维+双重护体功法 | "穷人走一步算三步，富人的每一步都消耗更多资源" |

**白真真线的对位价值**：白真真作为从富家（妖裔繁育基地）逃出来的"伪穷人"，提供了第三种视角——"从底层爬上来的穷人反而比天生穷人更拼命"。她的"真灵根隐藏九年→一朝爆发碾压"和云景的"第二丹田·外包万个工作人天"形成工业级对比：穷人的灵根是憋出来的，富人的丹田是外包出来的。

**叙事双螺旋的节奏控制**：不每章都做对比，而是每30章左右设置一个"对比场景"——法赛、武道竞赛、道心测试、筑基考试四个核心对比场景分散在全卷不同阶段。中间段不做直接对比，而是各自发展各自的线——穷人线在暗学帮赚钱攒资源，富人线在仙都展示统治力。到下一个对比节点时，读者能清晰看到"双方都在升级，但升级方式完全不同"。

## 改建理由
单纯写穷人的逆袭（"穷人靠努力打败富人"）只是爽文陷阱——读者爽过就忘。穷富对位双螺旋让读者持续思考两个问题：1）穷人真的能靠努力追上资本+天赋+资源的复合碾压吗？2）富人的"痛苦"（夜凌霄的完美血脉代价、玉星寒的13亿合同）真的比穷人的痛苦更轻吗？——前者让读者保持对主角处境的焦虑，后者让反派变得立体，不再是"纯恶"。

## 适用场景
- 阶级冲突主题的校园/竞赛类作品
- 需要"反派也有苦衷"式立体群像的作品
- 主角成长弧线需要参照系（不仅是打脸小人，更是与强者持续对位）的作品
