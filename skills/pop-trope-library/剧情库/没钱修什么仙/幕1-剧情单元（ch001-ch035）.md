# 卷1·幕1 剧情单元卡（ch001-ch035）

## 单元1：面试连环打击与穿越二次绝望（ch001-ch003）

**【结构描述事件链】**
- 面试官连续揭露学霸真相：绝育手术要求、多校极端条件、器官抵押
- 原身以贷养贷欠70万，母亲离开，张羽穿越接手烂摊子
- 白真真高冷学霸跪地借钱，揭示"穷人均贷款修仙"生存现实
- 体育老师王海强制推销补剂+体罚威胁，周天翊揭露王海黑暗史
- 天台仪式掌心符号逐章填充（1/10→1/2→不到1/4），建立24小时倒计时紧迫感

**【结构解析】**
- 绝望建置→穿越二次绝望→微弱曙光（白真真500元关怀），三章完成三重打击+一层温暖
- 三个维度同时建立：世界观（修仙阶级固化）、生存（70万负债）、金手指（仪式倒计时）
- 每章末以符号填充进度做悬念钩子，牵引读者追读到ch004仪式完成
- ch002白真真登场以"高冷学霸跪地借钱"制造首个记忆点角色

**【金句】**
- "学霸只需要睡两个小时就够了"（ch001，面试官揭露睡眠时差真相）
- "只有中考分数在500分以上的'人'，才算是这个城市的人了"（ch001，分数=人权的世界观铁律）

**【复用要点】**
- 面试信息差爽：读者和主角同步被信息冲击，可用于任何需要开局建置的场景
- 连环打击递进：面试→穿越→负债→被歧视，层层加码不喘气
- 穷鬼友情开篇：白真真"跪地叫爹"瞬间打破高冷印象，低成本建立角色记忆点
- 金手指倒计时：以符号填充进度做天然章末钩子，零成本悬念驱动

## 单元2：邪神仪式反转与羽书觉醒（ch004-ch006）

**【结构描述事件链】**
- 历史课揭示昆墟36层结构和十大宗门垄断
- 掌心符号填满→布娃娃邪神登场→愿望反转（不是邪神帮你，是你帮邪神）
- 张羽用逻辑悖论说服邪神"穷出了一个漏洞"，智斗妥协达成协议
- 羽书觉醒：属性面板（法力7.7/肉体0.83）+拖拽技能升级模式
- 仪式24小时监工逼迫修炼，确立"我要自由"的核心动机

**【结构解析】**
- 从绝望谷底→超自然恐惧→智力反杀→金手指觉醒→被迫修炼，五章完成全流程
- 愿望反转是全书最核心套路：贫穷=漏洞，穷比富更让邪神头疼
- 仪式"人工智障"属性建立：可被逻辑漏洞利用，为后续多次智斗打基础
- ch006"我要自由"确立贯穿全书的终极动机，角色弧线从这里正式出发

**【金句】**
- "穷出了一个漏洞啊……"（ch004，张羽用穷胁迫邪神的名场面）
- "MD，不是说要找有钱人吗！"（ch004，邪神内心独白，喜剧反转点）
- "张羽想要……自由！"（ch006，全书核心动机确立）

**【复用要点】**
- 仪式反转：请神是反的，触发者变被触发者，颠覆读者预期
- 穷人压迫金手指：金手指不是白给，24小时逼你练功，外挂=另一种奴役
- 逻辑悖论谈判：张羽用"你帮我实现愿望我才能帮你→我没钱=你没人帮"反制邪神
- 自由动机定位：不开挂不躺平，核心诉求是"不想被任何人控制"

## 单元3：世界观全面碾压与班主任陷阱（ch007-ch009）

**【结构描述事件链】**
- 学生跳楼事件，围观同学发表穷人歧视言论，白真真爆"十几年没有平民考上十大"
- 羽书专精功能发现（一次只能专注一个技能）
- 班主任苏海峰以"关心"姿态骗签债务重组合同（欠款膨胀到100多万），电话暴露真面目
- 计算月需2万的经济压力+修仙者零工生态展示
- 白真真留剩饭"像喂猫一样喂张羽"，建立孤独同类意识

**【结构解析】**
- 世界观快速扩张三连：跳楼歧视→班主任陷阱→打工牛马，每章一个体制级打击
- ch008是首个反转揭露高潮：以为好老师→还是放贷的，建立"这个世界的成人全是猎人"
- 打工生态展示（ch009）第一次展示底层游戏玩家生态，为后续灰色经济铺垫

**【金句】**
- "差生跳楼就死，学霸跳十层才死得掉"（ch007，黑色幽默的世界观定调）
- "穷人嘛……撑不住总会来借的"（ch008，班主任真面目揭露线）
- "毕业前24小时学习，毕业后24小时上班"（ch009，昆墟底层永动机总结）

**【复用要点】**
- 温情穿插：每次世界观碾压后放一段友情互动（白真真留剩饭），调节节奏
- 反转结构：先建立"好老师"形象→细节分化→电话揭露，读者与主角同步被骗
- 经济数字量化：月需2万、债务70万、存款50元，贫穷用数字说话

## 单元4：试功赚钱与周天采气法挂机（ch010-ch012）

**【结构描述事件链】**
- 健体三十六式2→3级质变，解锁法力运转，从"毁肉"变"养肉"
- 抢工失败（肉体2.15门槛不够），锁定高危"入门试功"1万/次
- 仙韵集团试功：张羽唯一完成100%运功路线，获13000（含3000演技加价）
- 周天采气法1级觉醒：24小时被动吐纳，睡觉也得3成效果
- 三天疯狂修炼，周天采气法4级，法力7.7→8.3
- 王海怀疑张羽有"校外进货渠道"，让赵天行跟踪孤立

**【结构解析】**
- 升级正反馈（ch0103级质变）+冒险赚钱（ch011试功）+挂机修炼（ch012被动收益），三连正反馈
- ch011试功建立"张羽的演技/策略"人设：假装重伤讨价还价
- 被动收益系统首次上线：睡觉也在变强，这是穷人vs有钱人最核心的追赶机制

**【金句】**
- "就算死上几百人又如何？"（ch011，资本家台词定调黑心企业）
- "无法反抗，便只有享受"（ch012，被孤立反喜的穷人修炼哲学）

**【复用要点】**
- 试功场景模板：别人全倒→主角唯一成功→讨价还价加钱，三拍爽点
- 被动收益系统：挂机升级是网文读者最爱的"占便宜"快感
- 反向孤立：被孤立→更有时间修炼→更强，化劣势为优势的逻辑

## 单元5：法赛预告与画展机遇（ch013-ch016）

**【结构描述事件链】**
- 走神率0%第一，法力8.3，法赛预告需8天内法力从8.3到10.0
- 白真真展示"筷子剑法"，隐藏天赋首次暗示
- 赵天行跟踪被识破→条件"请我吃炸鸡"化敌为工具人
- 面包车招保安选赵天行拒张羽（"细狗不要"）→张羽捆绑入伙→金丹真人画展机会
- 中心大厦999层俯瞰三重世界，第一次看到不为修为的消费（海鲜"为了好吃"）
- 金色法骸总监、筑基老祖炸鱼、偷听天人演武图情报
- 炼天极（紫云高中）和李星宇（邙山魂修生）相遇

**【结构解析】**
- 从校园到全市舞台的过渡，ch014"细狗"喜剧→ch015三重世界震撼→ch016法骸展世界观
- 中心大厦的"三重分层"是全书阶级隐喻的核心视觉化
- 赵天行从敌人→工具人→朋友的转变在此启动

**【金句】**
- "当保安就是为了摸鱼！"（ch016，保安之道的哲学定调）
- "筑基老祖跑来一层炸鱼装逼打脸"（ch016，黑色幽默的世界观补充）

**【复用要点】**
- 捆绑入伙：被拒后"我们是亲兄弟"强行带入场，急智型解决问题模板
- "细狗"伏笔：当前被嘲笑体力→后续体育考试用实力打脸，长线爽点
- 世界观视觉化展示：999层大厦、三重分层、法骸人，用具体场景替代描述

## 单元6：天人演武图与金丹真人的拒绝（ch017-ch022）

**【结构描述事件链】**
- 画展工作正式上岗，天人演武图绝世武学线索浮现
- 张羽用手机拍下白真真演练视频→独自领悟天武炼心诀
- 星火真人派金丹弟子来看效果→张羽练成但被拒
- 拒收理由："找到了15岁半小时练成天武炼心诀的真正天才"
- 补偿1万元作为"看错了的补偿费"
- 真正的天才身份成谜（全卷最强伏笔之一）

**【结构解析】**
- 希望（天武炼心诀练成→被看中）→冷水（被拒）→安排（补偿1万）
- 星火真人是建立"天才也有层次"的关键角色，拒收的理由比收更有效果
- 补偿1万的双面性：表面上人道的善意，实际上是"我看你有潜力但不选你"的最大侮辱

**【金句】**
- （此单元以事件和伏笔驱动为主，金句密度较低）

**【复用要点】**
- 被拒比收更有效：让读者期待"下次见面时让你后悔"的长线爽点
- 真正的天才伏笔：主角不是唯一天才，世界不是围着主角转
- 补偿的羞辱感：1万块封口费也是"你的潜力就值这么多"的标价

## 单元7：拒绝卖身契与卷王冲刺（ch023-ch026）

**【结构描述事件链】**
- 苏海峰正式摊牌丁级卖身合同，张羽拒绝并提出"考年级第一再谈"
- 班主任全面打压：不给补课、不给优惠、不给请假
- 卷王模式启动：早5点练体→上课学法→午休刷题→晚上实战
- 天灵根体验（借的）+法力快速提升
- 炼体策略转型：从肉体磨损到效率修炼，健体三十六式快速升级
- 赵天行被张羽实力折服，从眼线变跟班

**【结构解析】**
- ch023是本书第一个"拒绝陷阱"的高光时刻，"考年级第一"是建立自信的宣言
- ch024-026密集升级周期，跳过枯燥积累直接显示成果
- 策略转型标志着张羽从"蛮干"到"用脑修炼"的进化

**【金句】**
- （此单元以行动驱动为主，金句密度中等）

**【复用要点】**
- "考年级第一再谈"：用不可能的条件作为拖延策略，同时建立主角野心
- 跳过枯燥：用"几天后/几周后"的时间跳跃展示成果，保持阅读节奏
- 策略转型标志角色成长：从"更努力"到"更聪明"

## 单元8：月考爆发与实战碾压（ch027-ch030）

**【结构描述事件链】**
- 健体三十六式10级→质变，各项数值爆炸增长
- 道术考试不公：穷人分数被压制
- 何大有动用特权挑战张羽→三场实战：秒杀+秒杀+硬碰硬碾压
- 王海体罚威胁被无视，张羽实力碾压所有校内对手
- 全校震惊：穷鬼月考620分，年级第三

**【结构解析】**
- ch027-030是幕1第一个爽点积累兑现窗口
- 三场实战递进：第一场秒杀建立碾压→第二场秒杀巩固→第三场硬碰硬展示"真功夫"
- 不公设定（道术考试压制分数）为后续"体制级打压"定调

**【金句】**
- （此单元以战斗爽感驱动，数值和行动堆叠）

**【复用要点】**
- 三战模版：秒杀+秒杀+硬碰硬，碾压感逐级升温
- 成绩反转：被压分仍然碾压，让体制的不公显得可笑

- 成绩全部揭晓：年级第三，620分

- 功法规划完成，为幕2进阶做准备

- 丰收与悬念并存的收尾，表面胜利（年级第三）下暗流涌动（神秘人电话、姐姐关注）
- ch034-035是幕1到幕2的过渡章节，以胜利为起点引出新问题
- 姐姐线的首次暗示在这里埋下，牵连到卷末才引爆

**【金句】**
- （此单元以伏笔和过渡为主）

**【复用要点】**
- 丰收 + 悬念双结尾：给读者短暂的满足感同时抛出更大的问号
