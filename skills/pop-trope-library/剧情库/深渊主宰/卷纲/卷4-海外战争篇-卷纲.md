# 卷4 卷纲 · 海外战争篇（ch083-167）

> 卷纲是战略契约，不是剧情细纲。卷纲只定战略、定范围、定方向，具体剧情交给幕纲环节。本文件不可直接给 create。

## 一、本卷定位

- 卷定位：索伦从领主到四阶屠龙者+斩首者的蜕变——阴影高塔探索+传奇刺杀+屠龙+星界之战+王者归来+巨头博弈，海外群岛篇的战争与扩张收官
- 主爽点：屠龙跨阶挑战 + 星界位面战争 + 斩首者身份跃迁
- 读者期待：索伦从"有地盘的领主"蜕变为"被巨头关注的势力"的权力跃迁；屠龙+星界战争的战斗密度；斩首127人的冷酷统治力
