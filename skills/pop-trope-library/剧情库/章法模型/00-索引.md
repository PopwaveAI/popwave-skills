# 章法模型 · 索引

> 定位：剧情库的二级资产。收单章如何承接、展开、释放、收钩，服务 chapter。

## 入库范围

| 类型 | 说明 |
|:-----|:-----|
| 章型 | 战斗章、披露章、日常章、社交章、探索章等 |
| 事件链密度 | 一章内事件数、字数预算、scene 切换方式 |
| 章首承接 | 如何接上一章钩子、状态或情绪 |
| 章末钩子 | 钩子类型、强度、预期回收章 |
| 信息释放 | 读者/角色/敌人知道什么，以及何时知道 |
| 场景调度 | 地点、POV、参与角色、感官锚点的组合方式 |

## 不收

抽象桥段爽法进入 `套路库/`；语言句段手法进入 `文风库/`。
