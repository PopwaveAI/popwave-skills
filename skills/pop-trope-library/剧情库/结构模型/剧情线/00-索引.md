# 剧情线结构模型 · 索引

> 定位：跨书提炼后的剧情线模式。服务 `pop-writer-plot`，帮助设计主线/支线/设定线/关系线/成长线/反派线。

## 收录标准

一条剧情线必须回答：

1. 谁是承压者或行动者。
2. 压力源是什么。
3. 这条线如何持续推进，而不是一次性事件。
4. 每个阶段发生了什么不可逆变化。
5. 它如何与其他线咬合。
6. 它最终兑现了什么读者承诺。

## 不收

| 不收内容 | 原因 |
|:---------|:-----|
| 卷纲/幕纲整体 | 容易照搬原书篇幅结构，带偏新书节奏 |
| 章级事件流水账 | 太贴原文，不利于迁移 |
| 单个桥段公式 | 应进入 `套路库/` |
| 世界规则说明 | 应进入 `设定库/` |

## 与书级剧情线的关系

- `剧情库/{书名}/剧情线/`：单书拆出的剧情线。
- `剧情库/结构模型/剧情线/`：多书或复盘后提炼出的跨书剧情线模型。
