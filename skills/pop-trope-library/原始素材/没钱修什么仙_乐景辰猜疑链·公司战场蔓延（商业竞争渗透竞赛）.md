# 套路: 乐景辰猜疑链·公司战场蔓延（商业竞争渗透竞赛）

> 类型: 商业竞争/世界观 | 首次出现: ch105 | 最新使用: ch114

## 定义

体育竞赛不是学生之间的比拼，而是各大公司（紫云药业、仙韵集团、红塔牧业）之间的技术交锋和商业宣传战。当意外的黑马（如嵩阳高中的张羽白真真）表现超出预期时，公司代表立刻开始互相猜疑——是不是对手在暗中布局？前十名中有几个是"我们的人"？竞赛变成了商业战场的情报战。

## 四段式揭露结构

| 阶段 | 功能 | 原文实现（ch114） |
|:----|:-----|:---------|
| 起 | 预期落空 | 乐景辰面色难看——乐沐岚只排第四，风头被嵩阳抢走 |
| 承 | 情报收集 | 周澈尘、蓝岭报告：嵩阳没技术突破+张羽是白龙张翩翩的狗 |
| 转 | 猜疑产生 | 认为张羽白真真用的是仙韵集团的技术 |
| 合 | 推演后果 | 前十五个仙韵集团的人→白龙成为最大赢家 |

| 要素 | 原文锚点 |
|:----|:---------|

用于竞赛/擂台剧情中的商业维度——主角在台上打比赛时，台下的公司代表在暗自角力。每当主角表现超出预期，商业阵营的人就会重新计算"这笔投资值不值"。后续可深化：公司之间互相渗透、挖墙脚、派卧底。
