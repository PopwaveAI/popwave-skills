# 套路：共生型成长搭档（互补搭档）

## 定义
两个角色之间存在相互促进的成长关系，A的成长加速B的成长，B的成长又反哺A，形成正向循环。

## 适用场景
- 双主角配置
- 需要长期绑定的人物关系
- 成长体系中有可共享的核心资源

## 本章应用
- 第153-154章：张羽的10级功法加速真灵根进化
- 白真真灵根进化后获得推演能力→可优化张羽的功法
- 两人交换使用灵根→双方受益

## 核心机制

```
张羽 (功法收集者/快速修炼者)
  ↓ 将一门门10级功法当作"养分"
  ↓ 加速灵根进化
白真真 (功法推演者/深度优化者)
  ↓ 获得推演能力后
  ↓ 优化共享的功法
张羽受益
```

## 写作要点
- 强调"没有A就没有B的今天"的宿命感
- 两人的优势要互补而非重叠
- 最终目标是1+1>2的效果展示
