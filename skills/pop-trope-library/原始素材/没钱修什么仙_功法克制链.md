# 套路：功法/心法克制链

## 定义
不同类型的功法、心法之间存在天然的克制关系，角色需要临场判断并切换功法应对。

## 适用场景
- 多功法储备的主角
- 高级别战斗中的战术博弈
- 展现角色的战斗智慧和功法广度

## 本章应用
- 第151章：天武炼心诀（战意）被霜狱诀（怨念）克制
- 本能切换残牛舍身心诀（习惯绝境→免疫怨念）逆转战局

## 克制链示例
```
天武炼心诀（战场杀伐）← 被克制 ← 霜狱诀（怨念侵蚀）
残牛舍身心诀（麻木绝境）← 克制 ← 霜狱诀（怨念侵蚀）
```

## 写作要点
1. 克制关系需要提前铺垫（让读者知道两种功法的特点）
2. 切换不能太突兀（"本能般的一种反应"）
3. 切换后要有立竿见影的效果对比
