# 套路: 大数据驱动消费（竞争对手下单诱导）

**套路标签**: #大数据 #消费诱导 #心理战术 #反向操作 #黑色幽默 #购物平台

## 定义
修仙世界的购物平台利用大数据实时推送功能，以"您的竞争对手正在下单XX功法"为诱导，刺激用户好奇心+焦虑感，促使用户点入查看甚至下单购买。平台还会根据用户的浏览记录推送"克制"类功法（如看天魔寄生法→推反寄生法），形成双向诱导链条。核心套路在于：用户明知这可能是平台套路，但仍忍不住点进去看。

## 四段式结构

| 段位 | 功能 | 执行要点 |
|:----|:-----|:---------|
| 起 | 诱导触发 | 平台推送"您的竞争对手正在下单"——带具体功法名 |
| 承 | 好奇心驱动 | 主角忍不住点入——功法名越邪门越吸引眼球 |
| 转 | 反制推荐 | 看到克制/反制类功法被推荐——开始怀疑平台 |
| 合 | 用户反制 | 收藏所有相关功法，与可能的对手玩心理战 |

| 段位 | 实现 | 原文锚点 |

| 承 | 狗佛路过都得点进去——张羽点进天魔寄生法详情 | "狗特幺路过都得点进去瞧一瞧" |
| 转 | 页面推荐"观看此功法的人还在看：反寄生法、净化神功诀、斩天魔法" | "观看此功法的人还在看：反寄生法、净化神功诀、斩天魔法" |
| 合 | 张羽收藏所有功法——"最好那个下单的家伙也能看到" | "把反寄生法、净化神功诀、斩天魔之卷诀等功法加入了收藏夹里" |

## 变体用法
- **猜疑链制造**: 主角会开始怀疑现实中的竞争对手（如"何大有嫌疑重大"），真假虚实交织
- **双向暴露**: 主角收藏反制功法后，对方也可能看到"您的竞争对手在看反寄生法"——信息战的子集
- **平台可信度质疑**: 主角内心OS怀疑"会不会这网站瞎扯的"——保留对信息真实性的疑虑

## 底层逻辑
昆墟世界的公司极度资本主义——一切设计都为了刺激消费，包括用户间互相展示、制造焦虑、诱导点入。这是一个比现实世界更赤裸的大数据操控叙事。
