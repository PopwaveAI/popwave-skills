# 套路：学霸三角·良性竞争

> **定义**: 成绩排名揭晓后，年级前三（或前几）的学霸之间形成一种互相认可的竞争关系。排名靠前的对手把主角正式视为"高中三年的对手"，产生"燃起来了"的自我驱动力——良性竞争而非恶意打压。

## 四段式结构

| 段位 | 功能 | 说明 |
|:----|:-----|:-----|
| 起 | 排名揭晓 | 前三名同时登榜，形成三角/多角格局 |
| 承 | 对手确认 | 第二名/第三名把另外两人确认为长期竞争者 |
| 转 | 自我驱动 | 确认对手后反而燃起斗志，拿出书/教材开始卷 |
| 合 | 女主秘密暗示 | 女主对主角的飞速进步产生怀疑——暗示她也有秘密 |

## 核心元素

| 元素 | 本场示例（ch034） | 作用 |
|:----|:------------------|:-----|
| 排名格局 | 白真真651第一、钱深626第二、张羽620第三 | 前三名差距明显但可追赶 |
| 对手确认宣言 | "除了这个白真真之外，这个张羽也会是我在高中三年的对手" | 实力被认可的信号 |
| 燃点表达 | "燃起来了！" | 中二但热血的对手认知 |
| 立刻行动 | 拿出符书开始练 | 用行动回应竞争压力 |
| 被动方的反应 | 张羽没直接回应但得意 | 被强敌当成对手的爽 |
| 女主怀疑 | "羽子难道和我一样，也开了？" | 暗示女主也有开挂/秘密 |

## 适用场景
- 学校/宗门考试后的排名公布环节
- 主角和几个同级天才形成长期竞争关系
- 需要描写良性竞争氛围而非单纯恶意打压

## 变体
- **宗门大比排名**: 宗门大比后前几名形成竞争关系
- **暗面竞争**: 表面友好但暗中较劲
- **三人不同路线**: 一个偏法力、一个偏肉身、一个偏符咒——各有所长的互补竞争
- **被多方盯上**: 不止第二名，第四、第五也把主角当目标
