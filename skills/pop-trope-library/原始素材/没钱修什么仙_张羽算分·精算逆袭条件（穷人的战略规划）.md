# 套路: 张羽算分·精算逆袭条件（穷人的战略规划）

> 类型: 剧情推进/战略思考 | 首次出现: ch114 | 最新使用: ch114

## 定义

主角在总分落后的情况下，提前计算逆袭所需的具体分数条件——不是模糊的"我要赢"，而是精确的"我需要领先对手X分以上，才能从Y分追到Z分"。将竞赛逆转变成一道数学题，让读者清楚知道差距和可行性，制造紧迫感。

## 四段式揭露结构

| 阶段 | 功能 | 原文实现（ch114） |
|:----|:-----|:---------|
| 起 | 了解总分结构 | 四场：强度50+灵敏75+速度75+力量150=总分350 |
| 承 | 查看当前分数 | 玉星寒前三场满分200，张羽191→差9分 |
| 转 | 计算条件 | 拿下力量关第一(150分)+将玉星寒压到141分以下=领先9分以上 |
| 合 | 迎难而上 | 不论可不可能，总要试一下 |

## 关键源码

| 要素 | 原文锚点 |
|:----|:---------|
| 总分说明 | 「体育竞赛的四场比赛，肉体强度占50分，灵敏关卡占75分，速度75分，最后的力量关卡150分」 |
| 当前差距 | 「玉星寒现在前三关加起来满分200分，我则是191分」 |
| 评分机制 | 「评分是根据选手的表现进行打分的，在我拿到第一名，获得150分的情况下，我在下一关的力量比赛上，表现比玉星寒强越多，他得到的评分就会越低」 |
| 精确条件 | 「想要赢他的话，就必须把他压到141分以下，或者说……要领先他9分以上」 |
| 同伴质疑 | 「在总分150分的比赛中，领先玉星寒那个怪胎9分以上？可能吗？」 |
| 主角决心 | 「不论可不可能，总要试一下」 |

## 使用建议
用于竞赛/考试/晋级类剧情中的"逆转倒数"——通过精确的数字计算让读者了解主角需要完成的"不可能任务"的具体难度。关键是数字要真实可感，而不是模糊的"努力就够"。可以配合同伴的"不可能"质疑来烘托难度。
