# 套路：灵界考试·虚拟实境

> **定义**: 修仙世界中利用精神世界（灵界/幻境）进行大规模统一考试的设定体系。考生意识被拉入虚拟实境，经历人为可控的感官体验（痛觉、味觉、触觉等），以意志力/道心强度为评分标准。

## 四段式结构

| 段位 | 功能 | 说明 |
|:----|:-----|:-----|
| 起 | 世界观引入 | 介绍灵界的本质（精神世界/十大宗门制造）、进入方式（意识拖出肉身） |
| 承 | 规则说明 | 考试规则：疼痛/恐惧/幻觉等级越高，维持越久，排名越高；晕厥则0分 |
| 转 | 意志对抗 | 主角逐步提升等级，体验到逼近极限的痛感，现有心法失效 |
| 合 | 觉醒转折 | 主角找到适合自己的意志支撑（天武炼心诀意念/复仇意志/某种思想），突破极限 |

## 核心元素

| 元素 | 本场示例（ch031） | 作用 |
|:----|:------------------|:-----|
| 精神世界的双重性 | "灵界可以说是一个真实的世界，也可以说是一个虚假的世界" | 虚实不分的震撼感 |
| 感官可控性 | "看到的、闻到的、摸到的、感觉到的一切，都是可以被人为控制的" | 细思极恐的设定 |
| 疼痛等级具象化 | 每升一级=多一根钢针刺穿手掌，37级=37根钢针 | 让读者感同身受 |
| 道心/意志/思想三位一体 | "思想为内核，意志为体现，心法为方法" | 这套体系的底层逻辑 |
| 心法匹配性格 | 基础炼心法（静心忍耐）vs天武炼心诀（杀意斗志） | 思想必须对应心法 |

## 适用场景
- 仙侠/修仙作品中需要大规模考生筛选的场景
- 想要具象化"意志力"/"道心"这一抽象概念的场合
- 需要主角在极限中觉醒/突破的关键转折点

## 变体
- **恐惧试炼**: 将疼痛替换为恐惧（如直面内心最深处的恐惧）
- **幻觉试炼**: 将疼痛替换为诱惑/心魔幻象（如看到已故亲人）
- **时光试炼**: 在精神世界中经历数百年，考验时间尺度上的意志
