# 套路库：真灵根·双修叠加修炼法

> **套路名（EN）**: True Spirit Root - Dual Cultivation Stack Method
> **别名**: 灵根共享、双倍修炼、修炼叠加
> **适用阶段**: 中后期·双人合修期

---

## 定义

两人共享一特殊灵根（真灵根），通过「灵根转移→适应新宿主肉体→取回后叠加新效果」的机制，达到「一个人有两个人的修炼效果」。不同功法、不同运功路径的效果可以相互叠加，且随着频繁互借，叠加效果水涨船高。

---

## 核心机制

### 触发条件
- 一方拥有可转移的特殊灵根（非租赁普通灵根）
- 灵根能根据宿主肉体状态进行适应性变化
- 两人物理接触（背靠背、背着等）即可完成转移
- 双方互相信任且住在附近（方便随时交接）

### 双修叠加公式
```
A修炼功法α（灵根在A体内）→ 灵根适应α + A的肉体特征 → A把灵根还给B
→ B收回时灵根带来α的效果 → B修炼时叠加α + 自己的功法β
→ B修炼β时灵根适应β → B把灵根还给A
→ A收回灵根叠加β效果 → A修炼时叠加α + β
→ 循环往复 → 效果指数级上升
```

### 三要素

| 要素 | 说明 | ch099-100示例 |
|------|------|---------------|
| 灵根转移 | 特殊灵根可在两人身体间自由转移，物理接触即可触发 | 「灵根在迅速「生根发芽」之后，便蔓延到了他的浑身上下」「白真真让张羽把自己又背起来……心中不断念叻着灵根快回来的想法，下一刻她便感觉到肚子里突然一暖」 |
| 适应变形 | 灵根进入新宿主体内后，会变成「那个人的形状」——适应对方的肉体结构和运功方式 | 「该不会是进入了羽子体内之后，主动去适应了羽子的肉身，变成了羽子的形状」 |
| 叠加回归 | 收回灵根时，灵根带回了前一位宿主的部分功法效果，叠加到自己的修炼上 | 「等于是把张羽修炼赤髓浑元气时的灵机灌注现象——叠加到了我的身上？」「一个人有两个人的修炼效果」 |

## 变体

1. **「单边转移型」**：只有一方能借出灵根，另一方只能被动接收（ch099初始状态）
2. **「双向互借型」**：双方都可以借出和收回，叠加效果循环增强（ch100成熟状态）
3. **「多人群P型」**：灵根可以在多人之间轮转——每个人都在灵根中叠加自己的功法效果
4. **「灵根＋丹药型」**：灵根叠加效果配合丹药使用，效果倍增（张羽的浑元丹+真灵根）

## 使用建议

- **必须写适应过程**：灵根刚进入新宿主时会有形状适配过程——需要时间来变成「他的形状」——这是重要的设定细节
- **叠加效果要具体化**：不能只说「变强了」——要写清楚「雷真体的神经反应速度+赤髓浑元气的全体增益同时存在」
- **物理接触是必须条件**：转移需要背着/抱着/接触——这既是限制也是增强关系的道具
- **不能无限叠加**：需要设定叠加的上限（如灵根承载不了无限多的信息）或冷却时间
- **互借频次决定关系**：住得越近→交接越方便→效率越高→两人必须捆绑在一起（桥洞合租的必然性）
- **创造「借出去时」的差异**：灵根不在身边时，出借方也会有空虚感/下降感——增加使用成本
