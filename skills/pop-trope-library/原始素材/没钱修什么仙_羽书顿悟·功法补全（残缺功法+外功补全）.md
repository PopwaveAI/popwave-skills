# 套路: 羽书顿悟·功法补全（残缺功法+外功招式=完整功法可学）

**套路标签**: #羽书 #功法补全 #顿悟 #残缺功法 #外功补内功 #白真真点醒

## 定义

主角面对一门残缺的功法（只有法力运转路线，没有外功招式/只有一半内容），羽书不认、无法修炼。但在偶然的机会下，主角从旁人的意外行为（打拳、比划等）中获得了缺失的那一半内容——残缺功法得以补全，羽书浮现完整功法名称和0级熟练度。从此主角只需要练一遍（哪怕是错误百出）就能彻底掌握这门功法。这是羽书系统的核心机制之一。

## 四段式结构

| 段位 | 功能 | 执行要点 |
|:----|:-----|:---------|
| 起 | 残缺功法到手 | 主角获得运功图/功法残篇→羽书没反应→无法修炼 |
| 承 | 困在门外 | 主角试图强行修炼但不得要领→意识到缺少关键信息 |
| 转 | 意外获得补全 | 旁观者做出看似无关的行为（打拳/演练某招式）→主角认出了与残缺功法对应的外功招式 |
| 合 | 功法完整浮现 | 两者合一→羽书浮现完整功法名和0级→只需练一遍即可掌握 |

## 变体用法

1. **反向补全版**：先有外功招式（某人在练武），后得内功运功图——顺序颠倒
2. **多人碎片版**：残缺功法的不同部分分散在不同人身上，主角需要从多人处收集
3. **自悟补全版**：主角自己从残缺部分中推演出了缺失内容（不依赖他人）
4. **白真真感召版（本篇特色）**：白真真总在无意中"点醒"张羽——她的一些看似无意义的行为（打拳、吐槽、自暴自弃）恰恰给了张羽关键启发

## 底层逻辑

羽书的核心机制是"看到完整功法→学会→练一遍即精通"。但实际世界中，功法往往不是完整呈现的——有的是专利保护（只给部分）、有的是加密（需要特殊资质才能阅读）、有的本身就是残缺的。因此主角获得完整功法的过程，就成了一个"拼图"游戏：从多个来源拼凑出完整的信息结构。这与传统的"捡到秘籍直接学"不同，增加了现实感和搜索乐趣。白真真作为"补全者"的角色定位也暗示了她深不可测的背景——她总能无意中出现在功法缺失的环节。
