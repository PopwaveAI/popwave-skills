# 套路：肉身美学·物理评价

> **定义**: 通过精细化、数据化的身体检测+行家眼光的口碑评价，展现主角肉身的完美程度。用旁观者（尤其是想黑主角的人）的专业视角"被迫"称赞，实现"数据碾压+口碑装逼"双管齐下的爽点。

## 四段式结构

| 段位 | 功能 | 说明 |
|:----|:-----|:-----|
| 起 | 数据爆出 | 检测设备反馈主角的肉体力量/结构数据，远超预期 |
| 承 | 口碑堆积 | 多名评委从不同角度（骨架、肌肉、关节）轮流称赞，形成口碑墙 |
| 转 | 黑子被堵 | 原来想打压主角的人，被其他评委的真诚夸奖+专业眼光封住了嘴 |
| 合 | 被迫认可 | 连黑子都不得不承认——「这肉身简直是一件艺术品！」 |

## 核心元素

| 元素 | 本场示例（ch033） | 作用 |
|:----|:------------------|:-----|
| 精细化数据 | "右臂肌肉爆发出443公斤力量，左臂肌肉爆发出422公斤" | 用具体数字制造震撼 |
| 专业行话 | "兽背，妖胸，魔臂" "骨关节经过了千锤百炼的塑形" | 用行家口吻加持可信度 |
| 多角度口碑 | 五位老师一人一句，各有侧重 | 铺陈口碑的厚重感 |
| 想黑被堵 | 王海想打低分→被拍肩膀封口→尴尬笑 | 破坏坏人计划的爽感 |
| 被迫欣赏 | "实在无法对如此璞玉般的肉体打下低分" | 连敌人都不得不服的极致爽 |
| 隐喻比喻 | 评委=巨兽在肉铺挑肉、肉身=艺术品 | 用文学性强化画面感 |

## 适用场景
- 修仙/武道世界中主角的肉身经过千锤百炼后第一次接受专业检验
- 有仇人在场看着主角被公开夸奖的场景
- 考试/比赛/选秀中的实力展示环节

## 变体
- **法力评价版**: 评委用法力探测仪检测主角的法力纯度/运转效率，集体惊叹
- **符咒评价版**: 评委评审主角画符的笔力/灵气/结构，连竞争对手的老师都点头
- **武器锻造版**: 评委对主角锻造的武器的品相/材质/工艺轮流称赞
