# 脑海对练法·穷人的实战驱动修炼

**适用场景：** 没有天赋/装备加持的穷人，通过创造性的低成本方法弥补修炼差距

## 核心机制
在脑海中模拟实战对练，以破招驱动修炼，用过往对战经验代替法骸/天赋

## 本章应用（ch169）
- **主角：** 白真真
- **功法：** 万剑无终诀(10星/149万)
- **困境：** 2小时练一万剑招不可能
- **灵感来源：**
  1. 夜凌霄闭目→联想到脑海演练
  2. 张羽练天武炼心诀招式歪七扭八也能成→剑意比招式更重要
- **解决方案：**
  1. 构建"张羽"对手（基于过往实战记忆）
  2. 在对练中发现自己的破绽
  3. 悟：剑招变化是为了破招
  4. 状态：化身雷霆霹雳，剑意升腾

## 核心要素
1. **低成本：** 不需要法骸/血脉/装备——只需要脑子
2. **实战驱动：** 不是死记硬背，而是在对战中体悟
3. **需要基础：** 主角必须有足够的实战经验来构建对手
4. **效果≈天赋：** 能达到类似夜凌霄/云景等天赋者的修炼速度

## 变体
- 多人对练：脑海中模拟多个对手
- 自我对练：自己打自己（发现自己招式的破绽）
- 历史对战：用过去遇到过的所有对手做素材

## 要点
- 需要前面章节积累的战斗经验作为素材
- 适合有丰富实战经验但没天赋/装备的角色
- 适合在"紧迫时间"场景中使用（考试/竞赛前突击）
