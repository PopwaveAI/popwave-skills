# 套路：身份误会·被捧上神坛的穷人

## 定义
主角因无法解释的原因被外界错误解读为强者/富豪/重要人物，越解释越反效果，身份差距产生悬疑和喜剧效果。

## 覆盖章节
- ch161「小丑位的召唤」
- ch163「深不可测的张羽」

## 适用场景
- 主角有某种"无法解释的表现"（如撑爆道心幻境）
- 旁观者从自身阶级出发，必然选择"有钱"的推理方向
- 主角内心的恐慌和表面镇定形成强烈反差

## 本章应用
- **第161章**：张羽因道心测试"未知估值"被夜凌霄/戴行之推理为"身价不菲的超级富豪"
- **第163章**：张羽在20名挣扎 → 被旁观者误读为"在玩" / "控制排名"
- 每个人物用自己的阶级视角解读张羽：
  - 夜凌霄："涉世未深的有钱人，深不可测"
  - 戴行之："不屑用低端灵根，控制排名炫耀"
  - 云景："躲在后面玩把戏"

## 核心机制
1. **触发条件**：主角做了一件无法用"正常穷人身份"解释的事
2. **放大循环**：旁观者阶级滤镜 → 最有钱方向的解释 → 传播到更多旁观者
3. **张力来源**：主角内心恐惧（怕露馅）vs 表面镇定（只能硬撑）
4. **高潮点**：被误读到极致，甚至开始自我怀疑（如夜凌霄说"玩得不错"，张羽被气到）
