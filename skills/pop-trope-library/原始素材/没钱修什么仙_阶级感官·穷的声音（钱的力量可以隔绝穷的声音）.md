# 套路: 阶级感官·穷的声音（钱的力量可以隔绝穷的声音）

**套路标签**: #阶级感官 #穷的声音 #感官隔离 #隔音耳罩 #钱的力量 #听觉阶级化

## 定义

在修仙赛博社会中，阶级差异不仅体现在分数、金钱、资源等可量化维度上，更体现在感官层面——穷人被疼痛折磨时的痛呼声、被淘汰时的惨叫声，会形成一种"听觉污染"，干扰富人的考试/竞赛环境。而富人可以用钱来隔绝这些声音（戴隔音耳罩、去付费自习室等）。"穷的声音"这个概念的具象化，将阶级差异从经济维度扩展到感官维度——连听觉都成为了阶级分界线。

## 四段式结构

| 段位 | 功能 | 执行要点 |
|:----|:-----|:---------|
| 起 | 听觉干扰 | 穷人被疼痛折磨发出的痛呼/闷哼声——干扰到富人 |
| 承 | 认知觉醒 | 富人意识到"这是穷的声音"——穷人买不起好止痛药 |
| 转 | 抽象概括 | 将具体的痛呼声提炼为"穷的声音"这个概念 |
| 合 | 钱的力量 | 富人用钱（隔音耳罩/贵价空间）隔绝穷的声音 |

## 变体用法

1. **味道版**：穷人身上的"穷人味"（ch049乐沐岚嫌弃张羽第一学历"穷人味儿太重"）——嗅觉阶级化
2. **视觉版**：穷人因衰老/过度劳累的外观（邋遢/枯槁）——视觉阶级化
3. **空间版**：穷人聚集区与富人区的环境差异（噪音/空气/温度）
4. **消费隔离版**：付费才能获得安静——图书馆、自习室、补课班的阶级门槛

## 底层逻辑

"穷的声音"是这个故事中最精妙的黑色幽默之一——它把抽象的阶级差异变成了具体可感知的东西。穷人连痛呼都成了富人的"噪音污染"。乐沐岚的冷漠不是恶意，而是一种"理所当然"——在她的视角里，穷人买不起好止痛药→穷人忍不住痛呼→干扰到她考试→她用耳罩解决问题。每一步都有"合理"的解释，但合在一起就构成了一个残忍的阶级闭环。而这个套路的最大力量在于：它让读者突然意识到，这个世界上还有一群人，连"安静地痛"都做不到。
