# 套路：高低年级等级压制·修仙界学长制度

> **定义**: 修仙高中里每一届之间因一年修炼时间差异造成的绝对实力鸿沟——等同于「炼气前期、中期、后期」的层级差距。高年级学生拥有对低年级学生的全方位压制权（社会地位、修炼资源、实战碾压），低年级学生需对学长主动行礼、让座、让步。三大名校的制度更严酷（如下跪磕头）。

## 四段式结构

| 段位 | 功能 | 说明 |
|:----|:-----|:-----|
| 起 | 规则展示 | 展示学弟见学长的日常规矩（鞠躬问好、让座、让厕所、让打药）——形成惯性服从 |
| 承 | 实例碾压 | 高年级学生对低年级学生进行实战「指导」——以碾压式差距展示实力鸿沟 |
| 转 | 恶意压制 | 指导演变为羞辱/打压——故意下重手、废掉目标、逼出底牌 |
| 合 | 制度反思/反抗 | 有人默默忍受（如双幽朋被打磨成狗），有人试图反抗（如张羽用篆书反击） |

## 核心元素

| 元素 | 本场示例（ch058） | 作用 |
|:----|:------------------|:-----|
| 高低年级＝炼气前中后期 | 「高一、高二、高三……每一届之间都差了整整一年的修炼时间，带来的往往便是绝对的实力差距」 | 用修炼体系类比年级差距，让读者一秒理解 |
| 日常礼仪制度 | 主动打招呼、学长没坐低年级不能坐、上厕所让学长先尿、打药让学长先打 | 无处不在的阶层标记 |
| 三大名校更极端 | 「听说在白龙高中，高年级学生甚至可以让低年级学生下跪磕头」 | 拉开制度梯度——有更狠的 |
| 绝对实力差的具体化 | 白真真肉体1.4级 vs 蓝岭6级以上——全力一脚对方纹丝不动 | 数值化差距的冲击力 |
| 指导=打压 | 以实战指导为名，行羞辱打压之实 | 规则的漏洞被利用 |

## 适用场景
- 主角/主角团同伴有隐藏实力却被高年级打压
- 需要制造阶级对抗的紧张感（穷人 vs 富人的学校版）
- 主角以非常规手段（装备、符咒、智谋而非修炼时间）越级对抗高年级

## 变体
- **宗门版**: 入门早的师兄师姐对新入门的师弟师妹有管教权
- **职场版**: 早入行的前辈对新人训话/压迫——修仙界的职场霸凌
- **修为版**: 不是年级而是大境界差距——筑基对练气有绝对压制
- **体系版**: 反过来——低年级天才用实力赢得高年级的尊重（张羽的潜在路线）
