# 墓志铭·"身在黑暗，心向光明"（主题提炼法）

## 定义
用一章的篇幅为一个角色的死亡铺垫情绪和哲思，最后在实体墓碑上刻下一句简短有力的墓志铭，凝练角色的全部人生和主题。墓志铭不仅是总结死者，也暗示主角自身的命运。

## 适用场景
- 重要配角死亡后的收尾章
- 需要将具体事件升华到哲学/主题层面
- 角色弧线的正式宣告完成
- 为主角后续的命运提供镜像对照

## 核心机制
1. **场景定调**：葬礼场景的安静/美丽/具有象征意味（山谷/小河/来年花开）
2. **物理动作**：竖碑→准备刻字→但先不刻→角色坐下沉思
3. **哲思展开**：沉思死者的一生+思考相关主题（命运/人性/宿命/挣扎）
4. **深挖设定**：通过哲思带出更深层的世界观设定（如鲜血饥渴诅咒的本质）
5. **对照自省**：从死者的命运联想到主角自己的命运（"我是否也会如此？"）
6. **刻字落定**：终于拿起工具在墓碑上刻字→用物理动作强化主题的"最终性"
7. **墓志铭内容**：简短+对仗+一语双关（既形容死者也暗示主角）

## 典型结构
```
①安静美丽的场景（葬礼氛围）
②竖碑→准备刻字→先不刻
③/④坐下沉思→深度剖析死者和相关主题
⑤从死者联想到自己（镜像对照）
⑥终于刻字（物理动作）
⑦墓志铭内容（一句话凝练主题）
⑧角色离开→继续上路
```

## 案例参考
**ch175**：安静山谷→索伦竖碑→准备刻字→坐下沉思→吸血鬼的真实本质分析→"鲜血饥渴"诅咒深度剖析→雅莉安娜输给本能但赢了人性→"她输给了吸血鬼的本能……但是她也没有输"→联想到自己"注定要堕入深渊的人"→终于刻字→"身在黑暗，心向光明"→离开山谷→继续前往财富之城

## 变体
- **无碑版**：没有实体墓地→角色在心中或书本上留下墓志铭
- **集体墓志铭版**：战争/灾难后的集体墓碑→一句概括所有人的牺牲
- **反向版**：恶人的墓志铭→讽刺意味（"这里躺着一个好人"→实际不是）

## 注意事项
- 墓志铭本身必须足够精炼震撼→需要花心思押韵/对仗
- 沉思部分不能太长→否则读者会觉得"死了还要说教"
- 从死者的命运到主角自省需要有自然过渡→不能生硬
- 墓志铭的含义要能延伸到后续剧情（ch175的"身在黑暗，心向光明"也是索伦自己的写照）
- 刻字动作要有仪式感→让读者感受到"这个字落下去，这个故事就结束了"
