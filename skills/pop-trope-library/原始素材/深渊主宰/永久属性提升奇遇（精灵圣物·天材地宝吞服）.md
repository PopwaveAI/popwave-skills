# 永久属性提升奇遇（精灵圣物·天材地宝吞服）

> 套路类型：🌿 奇遇·永久属性·天材地宝·吞服提升
> 强度：⭐⭐⭐⭐⭐
> 适用场景：角色通过高风险抢夺/探险获得永久提升属性的珍稀物品→鉴定门槛高→服用后属性提升+专长升级+后续代价

---

## 核心机制

```
获得天材地宝（高风险抢夺→银月行者眼皮下抢宝+手掌被射穿）
→ 鉴定门槛高（需要3000杀戮经验→比传奇装备还贵）
→ 面板展示双重效果（装备效果：治疗重伤/树皮术/传奇活化植物 + 服用效果：永久属性+血量+专长）
→ 稀有度定位（"全世界加起来还不到十个手指头的数目"）
→ 服用（直接吃掉+属性提升+专长升级）
→ 后续代价（物品有特殊能量印记→可被法术定位→虽吃掉但能量还在体内→被追踪）
```

## 关键信息

- **鉴定成本**: 3000杀戮经验——比传奇晶石还贵
- **物品等级**: 精灵圣物（介于传奇和神器之间）
- **来源**: 神话时代的一片树叶——来自不可预知的古老存在
- **服用效果**: +1体质（上限25）、+10生命值、获得再生专长（初级→中级）
- **装备效果**: 治疗重伤1次/日、树皮术3次/日、传奇活化植物0/1（已使用）
- **稀有度**: 超凡属性20点以上还能继续提升的道具——全世界不到10件
- **后续代价**: 传奇以上物品有特殊能量印记→可被法术定位→吃掉后能量还在体内→银月行者可通过追踪能量找到索伦
- **索伦解法**: 直接吃掉——物理消灭可定位的物体——不搞复杂的反定位法术
