# 诛心之问·"你看自己不就是怪物吗？"

## 定义
反派利用角色自我认同的矛盾进行心理打击——当角色骂反派"怪物"时，反派反问"你自己不也是吗？"——让角色无法反驳，因为角色表面上已经同类化。

## 适用场景
- 被迫转变/堕落角色的身份挣扎
- 角色在"善恶"间的摇摆期
- 需要让读者对角色产生同情（角色被说中了痛处）
- 为后续角色的牺牲/转变做铺垫

## 核心机制
1. **建立对立**：角色骂反派"怪物/恶魔/畜生"
2. **镜像反击**：反派反问"你自己不也是吗？"或"看看你现在的样子"
3. **无法反驳**：因为角色确实做了同类的事（杀了人/变了身/堕落了）
4. **转移责任**：角色被迫用"是你害的"或"我杀的都是该死的"来自我辩护
5. **无效辩护**：这种辩护被反派或读者看穿是自欺欺人→角色的内心挣扎更明显
6. **效果**：读者理解角色的痛苦→产生共情

## 典型结构
```
①角色: "你们这群怪物/畜生！"
②反派（笑）: "难道你就不是怪物吗？看看你自己现在的样子！"
③角色（震住）查看自己→无法反驳
④角色（虚弱）: "都是你害的！是你把我变成这样的！"
⑤反派: "我害的？但你已经做了和他们一样的事。"
⑥角色崩溃或沉默→内心挣扎
```

## 案例参考
**ch173**：雅莉安娜"你们这群怪物"→堂姐娇笑后表情阴冷"难道你就不是怪物吗？你看看自己现在的样子！"→雅莉安娜表情难看→"都是你干的！是你把我变成了这样！"→"我杀的全部都是该死的人！……那些男人没一个是好东西！"→自我安慰但已无力→读者感到她的痛苦→为ch174的牺牲铺垫

## 变体
- **复仇版**：复仇者谴责敌人残忍→敌人"你不是也在做一样的事吗？"
- **战争版**：士兵谴责敌军暴行→敌军反问"你们就没杀平民？"
- **改造版**：被改造的实验体骂科学家→科学家"你现在已经和他们不一样了"

## 注意事项
- 反派的问话必须戳中角色真实的痛处→不能是纯恶意的无的放矢
- 角色的辩护越无力→效果越好→让读者感受到角色的内心矛盾
- 这场对话后角色应该有所变化（开始思考/更愤怒/更消沉）→不能毫无影响
- 适合用于悲剧性角色的转折点→不适合纯喜剧情节
