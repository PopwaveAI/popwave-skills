# 偶遇神秘势力（马戏团·游吟诗人·特殊表演）


> **元爽点家族**：世界观震撼
> **适用章位置**：结尾/高潮

> 套路类型：🎪 偶遇·神秘组织·双向试探·悬念铺垫
> 强度：⭐⭐⭐⭐
> 适用场景：主角在旅途中遇到看似友善但身份不明的群体→通过成员展示+信息试探+悬念为后续剧情铺垫

---

## 核心机制

```
旅途中遇到非预期群体（环境异常→发现痕迹→接近→交涉）
→ 群体展示（种族混杂+全员职业者+特殊徽记+独特文化）
→ 双向试探（双方都在装傻+交换真假信息+相互评估）
→ 关键信息埋设（目的地/传说/暗示/特殊兴趣）
→ 未露面的领袖（马车内/铃铛指挥/一句话决定一切——神秘感）
→ 章末悬念（异常痕迹→蛇一样的湿痕——暗示群体不简单）
```
