# 本地势力驱逐入侵者（"立刻离开！否则处死你！"版）


> **元爽点家族**：世界观震撼
> **适用章位置**：高潮

## 定义
外来者进入某个势力的领地被当地成员拦截并驱逐，通过这场驱逐展示势力间的内部规则和领地意识，同时也间接引出"追捕者已近"的信息。

## 适用场景
- 角色需要离开当前场景推进到下一个地点
- 需要展示种族/势力内部的社会规则
- 用外部压力迫使角色做出行动决策
- 通过驱逐者的判断失误衬托主角或配角的特殊性

## 核心机制
1. **警戒拦截**：本地势力成员现身拦截外来者
2. **强制性警告**：用明确威胁的口吻驱逐（"立刻离开！否则XXX"）
3. **规则说明**：在拦截过程中自然带出势力内部的规则（"我们不能容忍其他家族在领地猎食"等）
4. **对比暗示**：拦截者的判断→可能低估或高估外来者→留下伏笔
5. **驱离+紧迫感**：角色被驱离后→读者感受到"这里呆不住了"→接受角色离开的合理性

## 典型结构
```
①外来者在前一个场景
②本地势力成员现身拦截
③用口头威慑驱逐（"立刻离开！否则处死你！"）
④在对话中自然展示势力内部规则
⑤拦截者的判断（识别正确或错误→影响后续剧情）
⑥被驱离→角色进入下一个场景
```

## 案例参考
**ch173**：雅莉安娜离开沉船海湾→本地美艳吸血鬼拦截→"立刻离开这里！流浪者！否则始祖会处死你！"→揭示吸血鬼内部矛盾：不能容忍其他家族/流浪者在领地猎食→食物领地争夺→雅莉安娜未被识破是新生儿（吸过高阶血→起点高）→被驱离→进入荒野→追捕者正好出现

## 变体
- **黑帮驱逐版**：地下组织拦截→"这不是你的地盘"→展示帮派领地划分
- **贵族驱逐版**：城市贵族领地→"平民/外人不得入内"→展示阶级社会规则
- **种族驱逐版**：种族聚居区→"半精灵/混血不得进入"→展示种族歧视

## 注意事项
- 驱逐本身不能只是"赶人走"→必须在驱逐中展示世界观设定
- 驱逐者不能太弱→否则"被赶走"显得主角怂→要体现"不是打不过而是没必要招惹本地势力"
- 被驱逐后要快速进入下一个场景→不能让读者觉得角色在"逃"
