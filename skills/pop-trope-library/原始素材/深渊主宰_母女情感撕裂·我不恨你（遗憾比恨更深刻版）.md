# 套路卡片：母女情感撕裂·"我不恨你"（遗憾比恨更深刻版）


> **元爽点家族**：世界观震撼/情感羁绊
> **适用章位置**：高潮

## 适用场景
需要展示复杂母女/父子情感撕裂时，用"不恨但模糊"代替激烈冲突，制造更深刻的情感留白。

## 核心机制
1. 先建立角色A做了什么（错误/伤害/疏远）
2. 角色B表达情感立场：不恨（不是原谅——是对你已无情感期待）
3. "你对我来说一直很模糊"——比"我恨你"杀伤力大10倍，因为这是事实（无法反驳）
4. 沉默离开（不需要和解——情感已断）
5. 双方都有"正确"立场（母亲有梦想/女儿想保护母亲）→无法兼容→悲剧感的来源

## ch166案例
歌莉娅vs妖艳女子：对峙→母亲宣告研究成果→歌莉娅劝阻→母亲拒绝→"我不恨你"但"幼年你出现少"→"懂事你带来灾难害死无辜"→"放逐后已不是记忆中的你"→沉默离开→"她拒绝了→感觉与女儿裂缝越来越大"
