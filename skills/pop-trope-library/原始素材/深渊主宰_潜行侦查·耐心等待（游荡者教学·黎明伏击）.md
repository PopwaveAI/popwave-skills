# 潜行侦查·耐心等待（游荡者教学·黎明伏击）


> **元爽点家族**：信息差
> **适用章位置**：结尾/高潮

> 套路类型：🏃 潜行·侦查·耐心
> 强度：⭐⭐⭐⭐
> 适用场景：游荡者面对强敌时的潜行靠近+观察评估+等待时机+黎明突袭

---

## 核心机制

```
潜行靠近（阴影移动/无声/避开嗅觉追踪）
→ 观察评估（敌人人数/职业等级/状态/警惕性）
→ 选择位置（高处/树梢/隐蔽处/安全距离200→80米拉近）
→ 等待时机（识别敌人假寐/警惕松懈/黎明前最暗时刻）
→ 识破伪装（闭眼≠睡着/坐姿保持爆发性/假寐判断）
→ 拔刀靠近（弧线路径/一点一点推进/章末悬停）
```

## 关键信息

- **潜行基本原则**: 必须在阴影中移动/不能暴露在无阴影位置/避开有嗅觉的敌人
- **野兔检测法**: 野兔在一米外无觉察→潜行达到"大自然都不察觉"的水平
- **豺狼人嗅觉**: 天然追踪优势→索伦保持距离→利用风向/阴影遮挡
- **狼人假寐识别**: 闭眼但坐姿保持爆发性蹲坐→可瞬间冲出→不是真睡
- **黎明前时刻**: 战士鼾声→警戒者疲倦→姿势倾斜→精神最松懈→最适合发动突袭
- **弧线路径**: 不走直线→不可预测的接近路线
- **章末悬停**: 最后动作在"一点一点靠近"处停下→悬念留给下一章
