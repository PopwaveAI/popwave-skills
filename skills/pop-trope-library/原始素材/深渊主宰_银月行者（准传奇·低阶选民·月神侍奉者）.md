# 银月行者（准传奇·低阶选民·月神侍奉者）


> **元爽点家族**：世界观震撼
> **适用章位置**：高潮

> 套路类型：🌙 准传奇·六边形战士·月神信仰·神赐力量
> 强度：⭐⭐⭐⭐
> 适用场景：登场的准传奇级NPC——战力全面展示——为后续追捕/合作/冲突铺垫

---

## 核心机制

```
震撼登场（从天而降/数百米高空跳下/弯月徽记）
→ 远程压制（精准射箭→射穿索伦手掌→目标瞄准极准）
→ 战斗力展示（游侠战斗+独挡五六猎杀者+接近传奇闪避）
→ 职业构成展示（游侠+牧师双职业→准传奇→天生接近传奇的进阶职业）
→ 低阶选民设定（神灵赐予额外力量→比普通高阶职业者更强）
→ 追踪能力（捻土闻血→追踪能量→游侠追踪≈游荡者）
→ 后续行动决定（"要不要我追踪那个家伙？"→追踪启动）
```

## 关键信息

- **职业**: 银月行者（准传奇/天生接近传奇的进阶职业）
- **能力组合**: 游侠战斗能力+牧师高阶神术
- **战力水平**: 准传奇——总等级18以上——接近传奇领域
- **闪避专长**: 接近【传奇闪避】——任何攻击差一点才打到
- **冲刺速度**: 拉出残影——超自然领域能力
- **低阶选民**: 侍奉银月女士——神灵赐予的额外力量——比同等级职业者更强
- **追踪能力**: 游侠能力→追踪≈游荡者——很难甩掉
- **性格特点**: 记仇——大部分是雌性精灵——寿命漫长记性好——被得罪容易被追杀很久
- **银月女士**: 她的信仰遍布许多种族——精灵是主要信徒
- **追踪方式**: 物理追踪(闻血迹)+能量追踪(生命之叶能量在索伦体内)
