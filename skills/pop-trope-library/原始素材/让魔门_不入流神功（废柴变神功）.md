# 套路：不入流神功（废柴变神功）

## 元信息

- **套路名称**: 不入流神功（废柴变神功）
- **首次出现章节**: ch009
- **所属母套路**: 反差类/黑色幽默类套路
- **复用性**: 中——适用于需要制造轻松回甘节奏的配套功法设定，不宜频繁使用

## 触发条件

配角主动展示一门「不入流」（品级极低/公认垃圾）的武功 → 主角第一反应是嫌弃/不屑 → 配角得意反驳 · 别急 · 听我说 → 揭示该功法的独特设计——往往有荒诞/下三路/反直觉的特性 → 最后输出金句：「没有没用的功法/道具，只有没用的人」

## 套路公式

```
配角展示「不入流」武功 →
主角嫌弃（不入流=垃圾、要之何用） →
配角得意反驳（「别急，听了再决定也不迟」）→
揭示该功法的独特性/荒诞性（对应命根子/其他下三路部位）→
输出金句总结：「没有没用的功法，只有没用的人！」
```

## 核心张力

以「不入流」为前提交代，先建立低预期，再用该功法的独特设计建立反差——不入流不代表没用，只是在主流评判体系（皮肉筋骨血髓）中没有位置。这种「你以为很废其实有奇用」的反差感，能为高密度信息章提供轻松的笑点回甘，同时传达「体系之外也有价值」的价值观。

## 关键证据

- ch009: 「不入流？」王平有些嫌弃
- ch009: 「我这门武功名为【铁牛功】，圆满之后凝练的兵法叫做【金枪不倒】」
- ch009: 「对应兵器乃是裆下的命根子......」
- ch009: 「没有没用的功法，只有没用的人！」

## 情绪收益

黑色幽默笑果 + 被配角打脸的爽感（主角的嫌弃被反转） + 对配角角色的好感提升 + 「不要以品级论英雄」的价值观输出 + 高强度信息章后的放松回甘

## 变体方向

| 变体方向 | 示例 |
|---------|------|
| 丹药变体（废丹变神丹） | 公认的废丹其实是某冷门传承的圣药 |
| 兵器变体（废铁变神兵） | 生锈的铁剑其实是封印的绝世神兵 |
| 功法变体（禁术/邪功） | 被正道唾弃的魔功其实是上古正道失传秘法 |
| 人物变体（废物变高人） | 大家眼中的废柴老头其实是隐世高人 |

| **元爽点家族** | 绝境反杀 |
| **适用章位置** | 开头/高潮 |
## 下游期待

- 主角是否真的会修炼这门「不入流」神功？
- 金枪不倒在未来剧情中会不会有实际运用？
- 是否还有其他不入流但实用的武功/道具/方法？
- 「没有没用的功法」是否会是本作品的核心理念之一？

## 复用提示

- 不入流神功必须真的有实用性（即使是荒诞的实用性）——如果只是纯搞笑没有实际价值，就浪费了套路张力
- 主角的态度转变要自然——从「嫌弃」到「别急，我再听听」再到「有意思」，不要生硬
- 金句必须简练且可记忆——「没有没用的功法，只有没用的人！」可以作为配角的个人标签
- 这个套路放在章末效果最好——高密度信息后的搞笑收尾能让读者轻松结束一章
