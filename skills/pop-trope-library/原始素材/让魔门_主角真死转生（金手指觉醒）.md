## 定义与触发条件

**触发条件**：主角死亡（真死，非假死）→ 死后进入特殊空间 → 金手指以「规则书/系统」形式正式揭晓 → 说明能力规则和使用限制

**核心张力**：读者以为故事第一章就结束→实则这才是真正的开始——死亡是新生的代价。主角此前一直以为自己是普通人，死后才发现自己是「天地唯一的异数」

## 首次出现原文证据

- 🔒「【太平经】【卷一：域外天魔】」
- 🔒「来自天外的存在，游离于幽冥之境，却并非魂魄之身，乃是天地唯一的异数，超脱所有人的预料。」
- 🔒「死后可复活一次，复活后需要七天时间冷却，复活后可以随机进入新世界，或者进入已去过的世界」
- 🔒「重生后消耗气运可以获得新的身份，气运越多，获得的新身份越高。」
- 🔒「穿越至今二十年，直到大厦崩塌」「爹，您怎么才来啊！」

## 情绪收益

- 穿越者二十年后终获金手指的狂喜释放
- 能力强大带来的无限想象空间（复活+穿越世界+换身份）
- 「真死」的震撼 vs 「复活」的狂喜形成巨大情绪反差

## 变体与扩展方向

| 变体 | 说明 |
|:----|:-----|
| **系统面板更详细** | 规则说明更细致——有经验值、任务系统等 |
| **渐进觉醒** | 不是一次性全给，而是每次死亡解锁新能力 |
| **代价型复活** | 复活有代价（寿命/记忆/重要之物）——增加戏剧冲突 |
| **碎片式金手指** | 太平经有多卷，后续章节逐卷解锁 |

## 使用注意事项

1. 主角真死是高风险操作——必须确保读者在死亡段已经对主角有足够情感投入
2. 金手指规则要清晰、有明确限制（冷却时间/消耗资源/随机性）——无限制的外挂会破坏张力
3. 复活后的身份变化要谨慎处理——换身份可能割裂读者与角色的情感联系
4. 「域外天魔」的设定暗示主角不属于这个世界——这个设定后续要有呼应
