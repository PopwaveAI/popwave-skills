# 仙凡鸿沟（世界观阶梯展示）

- **嵌套型**: 是

## 触发条件
主角（凡间视角）第一次接触仙门体系 → 发现仙门最底层的工具/法术在凡间就是无敌级 → 认知被刷新

## 核心张力
认知刷新带来的震撼 + 对主角成长路径的暗示。"这还只是最底层？那最高层得多恐怖？"的想象空间。

## 关键对白/原文证据
- 🔒「在仙门里只能给弟子做苦力的工具人【力士】，如果放到大顺，有可能轻轻松松打遍天下无敌手？」
- 「这已经不是差距了。完全是鸿沟。就像是永远不会产生交际的天与地」
- 「王平百分百肯定，哪怕是守冲老毕登那足以炸碎假山的飞刀，也不可能在其身上留下哪怕一道划痕」

## 情绪收益
世界观格局打开的震撼感 + 对后续剧情天花板的高度期待 + 读者获得"原来是这样"的认知满足感

## 运作机制
1. 主角以凡间最高战力标准为参照（守冲的飞刀）
2. 仙门最低级工具/人员与之对比（力士）
3. 凡间最高级 < 仙门最低级 → 揭示鸿沟
4. 主角产生三种反应：绝望/好奇/野心（本章是绝望和焦躁）

## 变体方向
- **纵向对比**：凡间vs仙门vs梵国（不同体系的横向交锋）
- **渐进揭示**：每次只揭示上一层台阶，不一次性展示全景
- **反向揭示**：看似普通的仙门物品，在凡间是神器

## 使用注意
- 展示鸿沟后需要给主角留下"跨越可能"的暗示（否则读者绝望）
- 境界阶梯需要留白——不要一次性全部展示完
- 后续主角的成长应与每一级阶梯对应
