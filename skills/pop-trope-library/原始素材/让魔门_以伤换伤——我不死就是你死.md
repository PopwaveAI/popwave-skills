# 我不死就是你死——以伤换伤的战术哲学

> **元爽点家族**：世界观震撼
> **适用章位置**：中间


**套路类型**：战斗策略 / 风格定型
**首次出现**：ch021
**关联套路**：以弱胜强（金手指反杀型）、玉石俱焚（自爆式反杀）

## 四段式拆解

```
【起】主角复盘战斗，意识到自己能赢纯属对方选择了「最稳妥但不够致命」的攻击目标（斩胳膊而非斩头）。
【承】总结核心优势：有锁血/反伤/自愈类能力（气禁+与日偕亡），以伤换伤是最优打法。
【转】清醒认知短板：「以下克上可不是我的风格，以强欺弱才更合我心意。」——不迷恋以弱胜强的浪漫，渴求绝对实力碾压。
【合】结论：需要快速提升武功等级，从「被动的以弱胜强」转向「主动的以强欺弱」。
```

## 核心效果
- 为主角的战斗风格正式定型（以伤换伤+快速升级）
- 避免主角滥用「运气好」——将偶然转化为战术选择
- 为后续「利用金手指打极限战」提供合理性

## 复用条件
- 主角有濒死反弹/锁血/自愈/反伤类能力
- 对手实力明显高于主角
- 战斗需要速战速决（主角耐力差）

## 变体可能
- 「以血换血」——用流血代价换对方更重的伤
- 「以命换命」——一换一的死士打法
- 「以毒换命」——体内藏毒，以中毒代价换对方中毒
- 「以器官换优势」——用断臂/瞎眼等永久代价换致命一击机会

## 本章原文锚点
- 「有【气禁】和【与日偕亡】在，以伤换伤的打法才最适合我」
- 「以下克上可不是我的风格，以强欺弱才更合我心意」
- 「那一刀如果斩的不是胳膊，而是脖颈，那我当场就死了」
