# 军阵显威（量变质变型）

> **元爽点家族**：世界观震撼
> **适用章位置**：中间


**套路类型**：世界观展示 / 战力体系建立
**首次出现**：ch031
**关联套路**：装备飞升（配套展示型）、以弱胜强（金手指反杀型）

## 四段式拆解

```
【起】朝廷基层武装全面亮相——百人捕快私开武库，全套制式装备（兽面铠+偃月刀+战阵）
【承】王平灵识视角下，军阵显化披甲巨人神意——量变引起质变
【转】观战老者点明：唯武道宗师方可匹敌军队，封号武师冲阵必死
【合】结论：朝廷的根基不在某个人，而在制度和武装体系——杀一个总捕、重伤知县，县不乱
```

## 核心效果
- 建立朝廷基层武装的可信威慑力——不是靠单个高手，而是靠体系
- 打破「高手无敌」的错觉——为战力体系设置「军队阈值」天花板
- 世界观自然展开——武库制度、制式装备体系、战阵学一次性带出
- 为后续「宗师vs万人队」的更高战力层次埋下悬念

## 复用条件
- 需要展示国家/组织作为实体的威慑力量（非个人英雄主义）
- 需要为「主角为何不直接冲官府」提供合理解释
- 需要抬高战斗档次：从「人与人」升级为「人与军队」

## 变体可能
- 「武林门派军阵」——宗门也有类似的合击战阵
- 「宗室禁军显威」——更高级别的皇城军阵，更强
- 「异人军阵」——方仙道异人混编入军阵，产生更诡异的效果
- 「神器主阵」——以上乘神物作为军阵阵眼，威力翻倍
