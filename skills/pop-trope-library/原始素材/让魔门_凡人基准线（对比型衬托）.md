# 套路：凡人基准线（对比型衬托）

> **元爽点家族**：世界观震撼
> **适用章位置**：开头

## 元信息

- **套路名称**: 凡人基准线（对比型衬托）
- **首次出现章节**: ch009
- **所属母套路**: 对比衬托/主角塑造类套路
- **复用性**: 极高——适用于任何需要强调主角与众不同、有外挂优势的场景

## 触发条件

公开场景中大佬释放某种震撼世界观信息（难度/门槛极高）→ 普通人的集体反应是绝望/放弃/保守 → 该反应在读者看来是「正常人的合理选择」 → 主角的反应与之形成鲜明反差（不惧反喜）→ 主角的优势（金手指/特殊能力/特殊信息）成为这种反差的合理解释

## 套路公式

```
设定世界观上限（外功圆满=六门武功，缺一不可）→
凡人的合理反应（绝望→计算→放弃）→
高手的降维机制（中乘武功可一门多淬炼，降低门槛）→
凡人的二次反应（有希望但代价太大→继续放弃）→
主角的差异化反应（不惧反喜——因为有挂/有底牌）
```

## 核心张力

通过「凡人」的基准线反应来反衬主角的与众不同。众捕快的每一步反应都是「正常人的合理选择」——六门武功凑不齐→绝望；中乘武功可以但九死一生→不敢接。这些「正常人」的保守选择，恰恰反衬出王平「有挂所以敢冲」的优越感。

读者跟着凡人一起绝望→再跟着主角一起雀跃——这种「从绝望到希望」的体验旅程能最大化爽感。同时，凡人基准线也提供了一种「代入感锚点」——读者知道如果是自己没有外挂也会做出同样的选择，从而更加共情主角的开挂优势。

## 关键证据

- ch009: 「六门武功，还必须是分别对应皮肉筋骨血髓的武功，岂是那么好找的」
- ch009: 「这岂不是意味着，在场绝大部分人，终生都无望外功圆满？」
- ch009: 「一时间，校场重归静谧，无人回应。」
- ch009: 「九死一生？这个任务舍我其谁！」
- ch009: 『冷静，【域外天魔】还没冷却完，等冷却结束了再去接这个任务。』

## 情绪收益

凡人的绝望衬托主角的优越 + 代入主角的爽感（「我有挂我骄傲」） + 对凡人命运的共情感（底层群像塑造） + 对武道体系残酷性的认知 + 从绝望到希望的释放感

## 变体方向

| 变体方向 | 示例 |
|---------|------|
| 竞争型：凡人在努力争取，主角轻松拿到 | 众人拼死拼活抢名额，主角直接被选 |
| 恐惧型：凡人怕得要死，主角毫不在意 | 恐怖副本全员惨叫，主角逛街 |
| 计算型：凡人认为不可能，主角算出了可能 | 修仙突破概率万分之一，主角用道具拉到100% |
| 绝望型：凡人集体绝望，主角给出希望 | 灭世之灾全员等死，主角站了出来 |

## 下游期待

- 主角到底能否超越凡人基准线，达到「超人」级别？
- 是否有「凡人但逆袭」的配角的出现？（刘烨算不算？）
- 更高阶层的武者和凡人之间的差距有多大？还会出现什么新基准线？
- 主角自己的「成长阶段」是否会经历从「基准线之上」到「基准线之下」的再定位？

## 复用提示


- 凡人基准线的建立一定要有「数据化」的支撑——六门武功具体是哪六门、分别对应什么部位，越清晰落差越大
- 凡人的反应不需要太夸张——正常的失望/放弃即可，太过戏剧化反而会削弱真实感
- 主角的差异化反应用内心独白展示最自然——读者可以同时看到「凡人的反应」和「主角的反应」并直接对比
- 凡人基准线可以反复使用于不同场景（不同世界观层级）——每次都能有效衬托主角的超凡
