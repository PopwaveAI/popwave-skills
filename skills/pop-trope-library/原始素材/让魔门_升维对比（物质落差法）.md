# 套路：升维对比（物质落差法）

> **元爽点家族**：世界观震撼
> **适用章位置**：中间


## 元信息
- **套路名称**: 升维对比（物质落差法）
- **首次出现章节**: ch008
- **所属母套路**: 世界观建立/认知刷新类套路
- **复用性**: 极高——适用于任何涉及阶级跨越、地域迁移、世界跳转的剧情，可反复用于不同层级之间的对比

## 触发条件
主角从低阶层/低资源环境来到高阶层/高资源环境 → 经历某种生活方式的直接对比 → 认知被刷新（「原来之前的生活那么差」）

## 套路公式
```
低阶层生活习以为常（建立基准线）→ 意外进入高阶层环境（门槛突破）→ 
接触高阶层物质福利（感官冲击）→ 对比落差引发认知刷新（情绪爆发点）→ 
对更高阶层产生好奇/向往/压力（动力延续）
```

## 核心张力
读者通过主角的视角体验「原来这才是人过的日子」——不同于漫画式的信息描写（直接说「县城更好」），而是通过具体的物质体验（沃焦米、妖兽肉、药田蔬菜）让读者自己得出结论。落差越大，认知刷新越强，读者越能感同身受主角的震撼。

## 情绪收益
代入王平的震撼体验 + "原来可以这么爽"的共情感 + 对更高阶层的向往 + 阶级焦虑（"我还在最底层"）+ "开挂真香"的反权威幽默

## 下游期待
- 更高等级身份（青士/朱服/紫卿/金爵）的物质生活是什么样？
- 大顺王朝的武道体系与物质供给的直接关系——补给越好，练武越快
- 王平是否能通过捕快的身份获取更多资源来加速修炼？
- 县城之内还有哪些超越捕快之上的阶层未被揭示？

## 复用提示
- 升维对比的核心在于**具体化**——不要抽象地说「更好」，而是用具体的食物/住所/器具等细节让读者自行感受落差
- 对比的**倍数**很讲究——乡野生活的「好」vs县城捕快的「好」，差距应该在5-10倍之间，太大会脱离代入感，太小没有震撼效果
- 可以用「阶梯式对比」——每次升维都刷新一次认知（如：乡野→县城捕快→县令→州府→京城），逐步拉高期待
- 升维对比可以与**阶级跃升**套路配合使用——身份跃升带来物质跃升，物质跃升又强化身份跃升的爽感
