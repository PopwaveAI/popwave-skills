# 名号反转（谐正为俗）

## 基本信息
- **名称**: 名号反转（谐正为俗）
- **类型**: 喜剧反差型套路
- **首次出现章节**: ch015
- **嵌套型**: 否

| **元爽点家族** | 身份跃迁/阶级革命 |
| **适用章位置** | 高潮 |
## 触发条件
主角做了值得称道的大事 → 周围人告知主角已「名声大振」 → 主角期待严肃正派的英雄名号 → 实际获得的是低俗/好笑的绰号 → 真相揭露制造喜剧反差

## 核心张力
主角的严肃自我认知 vs 市井大众的消解式解读。观众同时体验到「看主角被整蛊的好笑」和「陪伴主角面对尴尬」的共情。

## 关键对白/原文证据（ch015）
- 🔒「恭喜王兄，你现在可是声威大振啊，名号在全县都传开了。」——刘烨
- 🔒「什么名号？」「这名号可不得了，叫做大龙枪！」——刘烨一字一顿
- 🔒「为何如此？我刀斩白莲反贼，名号不应该是斩夜刀，洗龙刀之类的吗？」
- 🔒「反贼算什么，这年头多了去了！昨夜王兄在丹凤楼以一敌百，枪挑群雌，那才是真的传奇，大龙枪之名也是因此而来」
- 🔒「日尼玛，我还以为你们终于欣赏了我的才华，没想到竟然是垂涎我的肉体。我呸，下贱！」

## 情绪收益
预期落差的喜剧笑果 + 主角内心吐槽的代入感 + 市井叙事对英雄叙事消解的新鲜感

## 运作机制
1. **铺垫期望**：刘烨说「声威大振」「名号传开」→主角（和读者）期待正面名号
2. **反转揭示**：刘烨故意一字一顿「大——龙——枪」→留白让读者先猜
3. **主角困惑**：王平提出正经备选（斩夜刀、洗龙刀）→强化反差
4. **真相揭露**：因青楼百人斩得名，寡妇们暗中打听「大龙枪」
5. **主角吐槽**：嫌弃+否认「我呸，下贱！」

## 变体方向
- **反向变体**：主角期待低俗绰号，结果得了正经名号→同样好笑
- **多层反转**：先被市井误解得俗号→后来用实力正名→获得正经名号
- **连锁变体**：名号引发后续事件（寡妇找上门、其他高手来挑战「大龙枪」）
