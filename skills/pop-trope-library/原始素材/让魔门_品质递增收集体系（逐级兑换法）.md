# 套路：品质递增收集体系（逐级兑换法）

## 元信息
- **套路名称**: 品质递增收集体系（逐级兑换法）
- **首次出现章节**: ch005
- **所属母套路**: 成长资源管理类套路
- **复用性**: 高——适用于任何「收集→兑换→成长」的RPG式系统

## 触发条件
主角获得金手指/系统 → 需要收集特定类型的物品来增强能力 → 物品品质必须逐级递增

## 套路公式
```
系统揭示品质层级（明确梯次）→ 主角分析最优策略 → 
从最低级开始收集（策略执行）→ 每次限制必须高于上次（长期压力）→ 
跳级不可逆转（永久放弃低品质档位）→ 主角必须精打细算（策略博弈）
```

## 核心张力
每件收集物品的品质必须高于上次，导致收集策略必须精打细算——不能贪快，必须从最低级开始稳步爬升。跳级虽然能更快获得经卷，但意味着永久放弃所有低级档位的资源利用效率。

**品质梯次（从低到高）：**
1. 俗世凡物（最低，最容易获得）
2. 地蕴精粹
3. 天养神华
4. 宇内奇珍
5. 唯一真品（最高）

除了增加经卷外，太平经还提供**鉴定**功能——认不得的东西拿给太平经看一下就能得到公正判断。

## 平行体系：身份等级映射（ch008拓展）
太平经的「品质递增」逻辑同样映射到**身份兑换系统**中：

| 青士 | —（尚未出现） | 更高 |
| 朱服 | —（尚未出现） | 更高 |
| 紫卿 | —（尚未出现） | 更高 |

| **元爽点家族** | 绝境反杀 |

身份体系与物品收集体系形成**双轨制**——物品品质递增对应经卷数量，身份等级递增对应气运消耗。二者都遵循「不可跳级或跳级即永失低级档位」的底层逻辑。

## 情绪收益
策略性 + 成长期待 + 收集癖满足 + 精打细算的成就感

## 复用提示
- 品质递增是限制型设计——它强制玩家不能无脑收集，增加了策略深度
- 可以用来制造困难抉择：比如某个珍贵物品近在咫尺，但王平的当前收集档位还很低——是否要牺牲长远效率？
- 品质梯次可以扩展到5阶以上，或者加入同级但不同类别的选择分支
- **身份等级体系**可作为同一设计哲学的社会维度映射——同一个世界内，阶级跃迁也遵循逐级递增逻辑
