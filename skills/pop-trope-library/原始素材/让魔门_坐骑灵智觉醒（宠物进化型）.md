# 套路：坐骑灵智觉醒（宠物进化型）

> **元爽点家族**：情感羁绊
> **适用章位置**：中间

## 元信息

- **套路名称**: 坐骑灵智觉醒（宠物进化型）
- **首次出现章节**: ch026
- **所属母套路**: 伙伴成长类套路
- **复用性**: 中——可用于长期陪伴型坐骑/宠物的进化线，一般一本小说只演化1-2只

## 触发条件

主角长期陪伴/使用的坐骑或动物伙伴 → 随着主角实力的提升或与主角的互动增长，坐骑逐渐表现出超乎寻常的灵性 → 从被动配合变为主动协作 → 甚至能与主角进行「非语言交流」

## 套路公式

```
坐骑/宠物最初只是普通动物
    ↓
与主角长期互动，灵性渐增
    ↓
开始表现出「反常」的智能行为
    ↓
主角从「惊叹」过渡到「习惯」
    ↓
坐骑逐步成为战斗/策略伙伴
```

## 核心张力

**熟悉 x 陌生** — 原本被认为普通的坐骑/宠物，逐步展现出引人注目的灵智。读者跟随主角一起体验「它怎么越来越聪明了？」的惊讶，并在每次「更聪明的行为」中获得乐趣。

## 关键证据

- ch026: 「而看到他过来，卷鬃马似乎也意识到了什么，甚至不用他辅助，只白了他一眼，就主动撅起了屁股。」
- ch026: 「王平也有些犯嘀咕，自己这匹马的灵性是不是有点太足了？一开始好像不是这样的。」

## 情绪收益

- 轻松喜剧效果（马翻白眼、主动配合）
- 情感陪伴价值（主角在孤独的修仙路上并不孤单）
- 成长期待（「这匹马以后会变成什么？」）

## 变体方向

| 变体方向 | 示例 |
|---------|------|
| **战斗坐骑变体** | 灵智觉醒后自动配合战斗走位/战术 |
| **监察伙伴变体** | 灵智觉醒后充当哨兵/观察者 |
| **进化形态变体** | 灵智觉醒后还会伴随形态变化（长出鳞甲/翅膀等） |
| **心灵链接变体** | 灵智觉醒到一定程度后可与主角心灵沟通 |

## 使用注意事项

1. 灵智增长要「渐进」而非「突变」——第一次翻白眼、第二次主动、第三次开始选马草、第四次能认人——每次只多一点点
2. 不要让其灵智超过「宠物」的合理性——如果坐骑比大部分人类还聪明，就需要解释
3. 可与其他伏笔联动（如丹药余效、灵识外溢等）
