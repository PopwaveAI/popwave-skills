# 外挂升级（系统进化法）

## 基本信息
- **名称**: 外挂升级（系统进化法）
- **类型**: 成长驱动型套路
- **首次出现章节**: ch016
- **嵌套型**: 否

| **元爽点家族** | 绝境反杀 |
| **适用章位置** | 中间 |
## 触发条件
主角的金手指/系统/外挂通过「吞噬素材」或「完成特定条件」升级 → 解锁新功能/新页面 → 主角获得新的能力或增益

## 核心张力
金手指本身具备「成长性」——不只是一次性获得后固定不变，而是可以通过持续投入资源来扩充能力。这种可进化的设定让读者期待每一次升级带来的能力突破。

## 关键对白/原文证据
- 🔒「【百炼元铜（俗世凡物）】」— 素材识别
- 🔒「【卷二：空】【待收录】」— 升级成功标志
- 🔒「新的【经卷】只要自己选的素材足够好，相信应该能给自己开出一个更强的外挂吧？」

## 情绪收益
系统成长的满足感 + 升级瞬间的里程碑快感 + 对新增功能的强烈期待 + 「主角在变强的，金手指也在变强」的双重成长感

## 运作机制
1. 设定金手指升级条件（ch016：需要异种铁作为素材）
2. 主角寻找/获取升级素材（ch016：从草市黑市购买百炼元铜）
3. 使用素材触发升级（ch016：太平经吞噬百炼元铜）
4. 升级结果揭示（ch016：第二页开启，【卷二：空】【待收录】）
5. 主角展望新功能（ch016：畅想第二挂的能力）

## 使用注意
- 升级素材需要「有获取难度」，避免变成简单收集任务
- 每次升级后的新功能需要有明确的使用场景
- 避免「升级后就无敌」——可引入「素材等级越高，新功能越强」的梯度
- 建议与主线剧情挂钩：素材的获取途径本身也是世界观延展的一部分（如ch016通过黑市交易链）
