# 套路：实力展示收徒（钓鱼执法式装逼）

> **元爽点家族**：身份跃迁
> **适用章位置**：开头/高潮

## 元信息

- **套路类型**: 社交博弈/实力压制/收服型
- **嵌套型**: 否
- **首次出现**: ch002「无间道」
- **难易度**: 低（效果直接，读者爽感高）

## 触发条件

1. 师傅方有绝对武力优势
2. 徒弟方对武功/超凡力量持怀疑或不屑态度
3. 口头劝说无效 → 需要实物展示证明

## 核心张力

先画饼（口头承诺）+ 当场展示（物理说服）= 徒弟从「不信」到「真香」的跨服打脸。本质是用不可辩驳的武力事实强行修正对方认知。

## 公式

```
画饼（口头描述「绝世武功」）
  → 目标内心不屑（「大人，时代变了！」）
  → 师傅物理展示（飞刀炸假山/一拳碎山石等）
  → 目标认知崩塌/反转
  → 当场拜师/投降/归顺
```

## 章节实例

### ch002「无间道」

**参与角色**: 守冲（展示方）、王平（被收服方）、谢石（旁观）
**场景**: 道观院内

**阶段**:
1. **画饼**：守冲取出银刀，介绍「素月莲华刀」和圆满兵法「一念花开」
2. **内心不屑**：王平心想「飞刀再快，能快得过子弹吗？都是扯淡......」
3. **展示**：守冲挥手飞刀——「轰隆！！！」——假山整个炸碎，飞刀插入照影壁
4. **认知崩塌**：王平呆愣→猛然转身→眼神火热→「『我尼玛......』」
5. **收服**：「弟子拜见师傅！」当场五体投地叩拜

**关键对白**:
- 🔒「大人，时代变了！」→ 展示后内心改为「时代又变回去了！」
- 🔒「弟子拜见师傅！」—— 无犹豫拜师
- 🔒「如何？想学吗？」—— 展示后的从容收网

**情绪收益**: 武力震撼 + 认知刷新 + 爽快拜师（主角不磨叽）+ 读者同步体验世界观击碎

### ch011「我要打十个！」

**参与角色**: 王平（展示方）、八个黑衣人 + 两个地痞（被碾压方）、黑衣人领头者（招揽方）、围观百姓（见证方）
**场景**: 长安街闹市

**阶段**:
1. **被动触发**：王平当街阻止绑架 → 八名黑衣人 + 两个地痞共十人围攻
2. **实力展示**：铁牛功硬扛围攻 + 阎罗三魔刀杀退十人——「我要打十个！！！」
3. **震惊全场**：围观百姓「就……就这？这就打完了？那不是王大头吗？」
4. **招揽抛出**：黑衣人领头者代表背后势力招揽——「我家主人很欣赏你」
5. **拒绝**：王平以「我是官家的人，是不会给你们这些江湖草莽做事的」拒绝

**关键对白**:
- 🔒「我要打十个！！！」—— 标题级台词，叶问梗化用
- 🔒「小子，你的身手不错，我家主人很欣赏你，只要你愿意，随时可以来找我，我保证给你一个锦绣前程」
- 🔒「我是官家的人，是不会给你们这些江湖草莽做事的。」—— 身份跃升后的拒绝底气
- 🔒「就……就这？这就打完了？那不是王大头吗？」—— 围观百姓的反差反应

**情绪收益**: 战力碾压爽 + 被认可（被招揽）的认可爽 + 以体制内身份拒绝黑暗势力的身份跃升爽

**与 ch002 对比**:
| 维度 | ch002（守冲收王平） | ch011（王平被招揽） |
|:----|:-------------------|:-------------------|
| 方向 | 上→下（师傅→徒弟） | 下→平级（江湖势力→官家人） |
| 展示方实力 | 碾压（一招碎假山） | 碾压（1v10） |
| 结果 | 归顺拜师 | 拒绝招揽 |
| 核心动力 | 武力震撼→真香 | 身份认同→拒绝 |
| 读者情绪 | 认知刷新爽 | 身份跃升爽 |

- 王平的「官家人」身份是否能守住这个拒绝带来的潜在敌人？

### ch013「今晚雨大，正好洗刀」——斩杀版

**参与角色**: 王平（展示方）、谢石（被斩杀方/「认证人」）、刘烨（见证方/震惊者）
**场景**: 雨夜县城街道

**阶段**:
1. **被动触发**：谢石逃窜中遇到王平，想抓他当人质——王平「不退反进」
2. **实力展示——碾压级**：阎王帖（三魔刀圆满兵法）+ 真龙筋（龙吟刀鸣）——一刀斩杀外功圆满的谢石
3. **震惊见证者**：刘烨——从「死定了」到「死，死了！？」——认知全面崩塌
4. **身份暗线回收**：谢石灰飞烟灭前回光返照认出王平——「被师傅和自己抛弃的小师弟」
5. **金句收尾**：王平拄刀轻笑——「刘爷放心，不用躲。今晚雨大，正好洗刀。」

**关键对白**:
- 🔒「死，死了！？」——刘烨的震惊
- 🔒『是他！？那个县衙派来的内鬼……小师弟？不可能！绝对不可能！』——谢石（被斩杀者的认知崩塌即是实力认证）
- 🔒「刘爷放心，不用躲。今晚雨大，正好洗刀。」——金句收尾

**情绪收益**: 战力碾压爽（一刀斩杀一群捕快+刘烨都拿不下的谢石）+ 身份暗线回收爽（第一章内鬼伏笔的十三章后回收）+ 见证者震惊爽（刘烨的认知崩塌）+ 金句收尾爽（暴力与诗意的极致反差）

**与 ch002/ch011 对比**:

| 维度 | ch002（守冲收王平） | ch011（王平被招揽） | ch013（王平斩杀谢石） |
|:----|:-------------------|:-------------------|:--------------------|
| 展示方→接收方 | 守冲→王平 | 王平→黑衣人 | 王平→谢石（被斩杀） |
| 方向 | 上→下收徒 | 平级招揽被拒 | 下→上碾压（弱者→强者逆袭） |
| 展示方式 | 主动展示（碎假山） | 被动展示（1v10） | 被动展示（遭遇战→一刀秒杀） |
| 结果 | 王平拜师 | 拒绝招揽 | 斩杀对方 |
| 身份暗线 | 无 | 无 | **有（小师弟→捕快的身份回收）** |
| 核心动力 | 武力震撼→真香 | 身份认同→拒绝 | 因果闭环→碾压杀 |
| 读者情绪 | 认知刷新爽 | 身份跃升爽 | **全书级暗线回收 + 实力碾压 + 因果报应三重爽感** |

**套路演进**: 从 ch002 的「被收服」（被动方）到 ch011 的「被招揽但拒绝」（平级）到 ch013 的「斩杀对方」（上位者）。同样的「实力展示」结构，因为王平的成长阶段不同，结果天差地别——从一个被守护对象，到能拒绝他人，再到能终结他人。三次展示构成了清晰的主角成长里程碑。

**变体说明（斩杀版）**:
本实例是「实力展示收徒」套路的变体——没有了「收徒」（没人归顺），取而代之的是「斩杀」（展示的终结对象是目标的生命）。其实质是「实力展示」的终极形式：让展示的代价成为一个有价值的人的性命。被斩杀者的身份（白莲教重要人物、主角的旧识师兄）本身就是展示的背书。

## 变体

| 变体 | 说明 |
|:----|:-----|
| **反向钓鱼** | 徒弟反向展示实力，师傅求贤收徒 |
| **群攻展示** | 当众展示收服一群人（众目睽睽式收编） |
| **被动展示** | 师傅无意中展示（比如被激怒出手），徒弟主动求教 |
| **身份拒绝变体** ★新增(ch011) | 展示方因身份认同拒绝招揽——从归顺变拒绝 |

## 使用注意事项

1. 展示的威力需显著超越目标原有认知体系
2. 展示后需给目标「合理台阶」（如守冲有秘籍可传、王平可白嫖）
3. 不宜重复使用同级别展示——后续需升级威力级别
4. 展示物（兵器/法宝）最好有后续回收价值
5. ★ 身份拒绝变体中，「拒绝」本身就是最高级的实力展示——能拒绝招揽意味着展示方已有更好的归属
