# 暗赚反转（信息差血赚型）

## 基本信息
- **名称**: 暗赚反转（信息差血赚型）
- **类型**: 智力碾压型套路
- **首次出现章节**: ch015
- **嵌套型**: 否

| **元爽点家族** | 信息差/智斗博弈 |
| **适用章位置** | 高潮 |
## 触发条件
对手提出看似严苛的交易/条件 → 对手基于凡人认知断定主角无法完成 → 主角拥有对手未知的隐藏优势（灵识/仙道/特殊能力）→ 表面吃亏实则血赚

## 核心张力
对手的傲慢预判 vs 主角的隐藏底牌之间的信息差。观众明知主角有底牌，仍然享受「对方以为自己赢了→实则输光」的打脸快感。

## 关键对白/原文证据（ch015）
- 🔒「对我来讲，秘籍内容多？灵识一扫，再多都看完了，事实上我现在就已经将全本武功牢记在心了！」
- 🔒「他才是血赚的那个，几乎什么都没有付出，就收获了一本中乘武功」

## 情绪收益
智力碾压暗爽 + 信息差带来的优越感 + 打脸对方预判的双重快感

## 运作机制
1. **对方出条件**：陈浩彦给穿云震天弓，只可借阅三天，认定王平不可能看完 → 王平「被迫」接受押送莲华刀任务
2. **表面劣势**：王平看上去是被胁迫的一方，三天后九死一生
3. **隐藏优势**：灵识一扫，全本已牢记在心，外加域外天魔不怕死
4. **反转血赚**：零代价获得中乘武功+玄螭弓+「得加钱」加价成功

## 变体方向
- **反向变体**：主角以为自己在暗赚，实则被对方更高维度算计（正中对方下怀）
- **累积变体**：多次使用此套路，每次隐藏优势的层次逐步提升

### ch015 新增实例

**参与角色**: 王平、陈浩彦（间接）
**场景**: 内衙→武库

**关键变化**:
- 此前版本（ch001-ch014）：王平用灵识辅助战斗/修炼
- ch015：灵识用于**信息窃取**——「借阅」模式的秘籍，灵识一扫全记
- 结果：几乎零代价获得穿云震天弓，还额外拿到玄螭弓

**与先前对比**:

| 维度 | 先例（如有） | ch015 |
|:----|:-----------|:------|
| 信息差来源 | 无 | 灵识的「一扫全记」能力首次用于信息窃取 |
| 对手认知 | — | 陈浩彦基于凡人逻辑断定三日看不完 → 完全错误 |
| 获利内容 | — | 中乘武功+禁品兵器 |
| 读者情绪 | — | 暗爽+期待后续修炼成果 |

**情绪收益**: 信息窃取的新鲜感 + 对手被打脸的期待 + 装备双丰收的满足感

**下游期待**:
- 灵识还能用于何种信息窃取场景？（功法/密函/地图/阵法？）
- 穿云震天弓能否在三天内入门？
- 陈浩彦发现真相后会如何？

### ch018 新增实例：夺宝截胡——「没人要我拿走了」

**参与角色**: 王平、守冲（战斗中被牵制）、游神真人（战斗中被牵制）、陈浩彦（幕后/不在现场）
**场景**: 雨中街道/马车内

**关键变化**:
- 此前版本（ch015）：王平用灵识「信息窃取」血赚——几乎零代价获得中乘武功+玄螭弓
- ch018：王平在三大势力混战中被当作弃子 → 不甘心爆发 → 在所有人都忽略马车时一手握住莲华刀、一手打碎刀架取出龙兴县印 → 太平经顷刻炼化 → **「我的了！」**

**与 ch015 对比**:

| 维度 | ch015（信息血赚） | ch018（夺宝截胡） |
|:----|:-----------------|:-----------------|
| 获利方式 | 灵识一扫全记的血赚 | 趁乱截胡的暗赚 |
| 信息差来源 | 对手对灵识能力的认知盲区 | 大人物对小人物「不值一提」的轻视 |
| 获利内容 | 中乘武功+玄螭弓+得加钱 | 莲华刀(异铁帝流浆)+龙兴县印(万民香火礼器) |
| 对手反应 | 陈浩彦不知道被血赚 | 守冲/游神真人/陈浩彦均不知宝物已被吞 |
| 核心情绪 | 暗爽+打脸预期 | 逆袭+截胡+「这口气不咽了」的情绪释放 |

**关键对白/证据**（新增）:
- 🔒「因为，我不值一提。」——被当作弃子的残酷认知
- 🔒「这口气，他咽不下。」——情绪转折起点
- 🔒「开挂前，这口气我咽下去了。开挂后，这口气我还要咽下去？那我不是白穿越了！」——爽点宣言
- 🔒「东西没人要吗？那我可就要拿走了哦。」——黑色幽默截胡
- 🔒「一把抓住两件宝物，顷刻炼化，令其消弭无形。」「我的了！」——成功截胡

**情绪收益**: 小人物截胡大人物的极致爽感 + 暗赚双宝的贪心满足 + 情绪从压抑到爆发的释放感

**下游期待**:
- 三大势力何时发现两件宝物没了？
- 守冲发现自己的莲华刀被「徒弟」吞了——会如何反应？
- 太平经同时吞下帝流浆（材料）和龙兴县印（香火礼器）——两种不同的东西——会融合出什么？
