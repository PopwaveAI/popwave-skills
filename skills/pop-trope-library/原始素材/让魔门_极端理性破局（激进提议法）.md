# 套路：极端理性破局（激进提议法）

> **元爽点家族**：智斗博弈
> **适用章位置**：开头

## 元信息

- **套路类型**: 智力博弈/逆向思维
- **嵌套型**: 是
- **首次出现**: ch001「我们之中出了一个叛徒！」
- **难易度**: 中（需要读者认同「激进行为」的逻辑基础）

## 触发条件

1. 主角陷入无法自证清白的困局（密闭空间+审查+互疑）
2. 常规解释/辩护路径被堵死
3. 对方预期主角会采取防守姿态

## 核心张力

提出一个对方绝对不敢执行的方案，让对方不得不让步。本质是用「更差结果」倒逼「次优结果」。

## 公式

```
激进方案 = 对所有人（包括自己）都极端的提议
          → 对方拒绝方案
          → 让步条件（主角获益）
```

## 章节实例

### ch001「我们之中出了一个叛徒！」

**参与角色**: 王平（提案者）、老道士（决策者）、众教众（利益相关方）
**场景**: 白莲教大堂（大门已被关闭）
**阶段**:
1. **陷入困局**：老道士宣布有叛徒，所有人被怀疑
2. **釜底抽薪**：王平提出「我们所有人自裁，全死了，卧底自然也就死了」
3. **价值包装**：以宗教狂热包装（「死后进真空家乡」「教主比我们重要」）
4. **被拒绝**：老道士无奈制止（「你还是太极端了」）
5. **获益**：王平被任命为堂主+调查负责人

**关键对白**:
- 🔒「我们自裁吧。」
- 🔒「我们所有人自裁，全死了，卧底自然也就死了……何乐而不为呢？」
- 🔒「好了，王小兄弟，你还是太极端了。」

**情绪收益**: 反转/反常识爽感 + 智力碾压 + 晋升奖励

## 复用规则


- 避免连续两章使用同套路
- 后续使用时必须升级赌注（风险更高、涉及面更广）
- 可以变体：主角提议他人执行极端方案（风险转移型）
