# 欲兽驯服·克制碾压型

> **元爽点家族**：战斗爽感
> **适用章位置**：中间


**套路类型**：战力展示型 / 功法优势兑现
**首次出现**：ch034
**关联套路**：功法克制、驯兽、心理掌控、降维打击

## 四段式拆解

```
【起】主角进入被异兽/生物占据的领域——兽房/丛林/异空间。
     异兽具有威胁性——展示异兽的强大：
     「不比内劲武师差！」——用主角熟悉的参照系建立异兽的威胁等级。
     异兽主动攻击主角，主角用常规手段勉强抵挡（但受挫）。

【承】主角转用功法赋予的专属能力——「双眼生出紫色光环」。
     这不是常规战斗——不是靠力量硬拼，而是靠体系克制的「降维」手段。
     功法对异兽存在天生克制：「所有欲兽先天就存在缺漏，六欲天自在成就法对它们可以做到完美克制。」

【转】强大的异兽瞬间屈服——凶戾的眸子泛起紫光，匍匐在地。
     攻击型角色瞬间转为被控制状态。
     主角从「被攻击者」变成「掌控者」——反差极大。

【合】异兽成为主角的修炼资粮/战斗资源。
     成长路径清晰了：收集更多异兽→融合→更强的六欲浑天魔。
     「三天之内驯服了你们，之后还要靠你们修炼呢。」
```

## 核心效果
- **功法特殊性展示**——不是万能碾压，而是「只对特定对象有奇效」
- **专属优势兑现**——告诉读者：练这个功法不只是升级，还带来了对某类对象的绝对掌控
- **成长路线可视化**——异兽=可收集的修炼资源，类似宠物小精灵

## 复用条件
- 功法/能力对特定对象存在克制关系
- 需要展示功法除升级外的「额外福利」
- 有明确的「收集→融合→成长」路径

## 变体可能
- **弱点击破版**：不是克制，而是找到每个对手的独特弱点并击破
- **同化版**：不是控制，而是将异兽同化到主角体内
- **契约版**：建立主从契约，异兽可独立作战
