# 玉石俱焚（自爆式反杀）

## 基本信息
- **名称**: 玉石俱焚（自爆式反杀）
- **类型**: 悲壮反击型套路
- **首次出现章节**: ch007
- **嵌套型**: 是

| **元爽点家族** | 绝境反杀 |
| **适用章位置** | 中间 |
## 触发条件
主角陷入必死绝境 + 反派企图利用主角获取利益 → 主角选择自我毁灭，同时破坏反派的计划

## 核心张力
我不活，你也别想好过——以自身毁灭为代价，给反派造成实质性损失。"宁为玉碎不为瓦全"的爽感。

## 关键对白/原文证据
- 「再无话说，速速动手吧。」→ 接受死亡
- 🔒「轰隆！」→ 自爆
- 🔒「折腾来折腾去，她不仅什么都没有赚到，反而还赔了一具【力士】，以及一本价值不菲的功法经文？」

## 情绪收益
黑色幽默（坏人吃瘪）+ 对主角"宁为玉碎"气节的共鸣 + "死也要带走点什么"的复仇快感

## 运作机制
1. 反派密谋从主角身上获取利益（敲诈、情报、资源等）
2. 主角陷入绝对劣势，无法正面反抗
3. 反派志得意满，开始享用战利品（注入经文）
4. 意外触发——主角的自爆/死亡同时破坏了反派的计划
5. 反派从狂喜到暴怒的反差收尾

## 变体方向
- **真死变体**：主角确实死亡——悲壮收场，适合章节断点
- **假死变体**：主角看似自爆实则转移/重生——后续揭示
- **代价变体**：主角付出沉重代价但未死——失去重要器官/能力

## 使用注意
- 此套路使用次数受限——多次"自爆未死"会削弱张力
- 需要让读者确信主角真死了（至少暂时），否则缺乏冲击力
- 反派损失需可量化（具体物品/金钱/计划），不可模糊

## 下游期待
- 主角是否真的死亡——金手指是否在爆炸瞬间发挥了作用？
- 如果没死，新躯体/新形态是什么？
- 反派后续是否会追查元凶/报复相关方？
