# 套路：穿越者苟且生存（隐忍发展型）

> **元爽点家族**：情感羁绊
> **适用章位置**：开头/中间

## 元信息

- **套路类型**: 成长框架/世界观基础
- **嵌套型**: 否（母结构——全书底层逻辑）
- **首次出现**: ch001「我们之中出了一个叛徒！」
- **难易度**: 低（易上手，适合开篇）

## 触发条件

1. 主角穿越到古代/异世界底层
2. 无系统/无外挂/无金手指
3. 需通过自身努力（体力、智力、社交）生存上升

## 核心张力

现代人灵魂 vs 封建社会/宗法体系的残酷现实。主角需要在「接受现实」和「不甘沉沦」之间挣扎。

## 章节实例

### ch001「我们之中出了一个叛徒！」

**阶段**:
1. **20年穿越史交代**：王平穿越至今二十年，从期待外挂到放弃幻想
2. **生存策略**：加入白莲教、组织教众、左右逢源
3. **资产积累**：从茅草屋到砖石房再到大宅院，从饥寒到三餐有肉
4. **现实清醒**：内心对宗教狂热保持冷漠和嘲讽

**关键对白/段落**:
- 「穿越至今二十年，他也在这个名为『大顺』的破地方苟且偷生了二十年」
- 「刚穿越过来的时候，他还奢望过会有什么外挂。然而事到如今……已经放弃那种不切实际的妄想了」
- 「古代王朝，封建社会，能有什么好日子？能活着，已是邀天之幸。」

**情绪收益**: 真实感/代入感 + 未来成长空间 + 反龙傲天的另类快感

### ch002「无间道」

**阶段**:
1. **卧底身份揭露**：官府招安→成为朝廷在白莲教的卧底，维持双重身份
2. **务实拜师**：从「大人时代变了」→看到武功真实威力后毫不犹豫拜师（生存理性）
3. **表面忠诚维持**：浮夸表忠心（「您就是我心中唯一的太阳」）维持人设
4. **利用修炼需求**：以「需要精米练功」为理由计划去县城找上线接头

**关键对白/段落**:
- 【内心】「虽然在白莲教老老实实地当一个反贼也没什么不好，但对我来说，还是加入朝廷更加海阔天空嘛。」
- 【内心】「大人，时代变了！」→ 展示后改为「时代又变回去了！」
- 【表面】「您就是我心中唯一的太阳！」← 守冲：「……过了过了。」
- 【内心】「当白莲教主的弟子又不妨碍自己继续当朝廷卧底，四舍五入等于白嫖」

**新增人设维度**:
- 双面间谍的生存策略（表面狂热 + 内心算计）
- 务实拜师不磨叽（看到武力价值立刻抓住）
- 利用修炼体系消耗需求作为接头借口（资源需求→剧情驱动）

**情绪收益**: 双面身份的紧张感 + 武力认知刷新 + 白嫖式成长的爽感

### ch006「穿越后的人牲」

**阶段**:
1. **新身份定位**：穿越仙界成为最低级【力士】（人形修炼工具），被放置在库房中等待交付
2. **隐藏意识**：力士不允许拥有独立人格——被发现即灭顶之灾，必须全力伪装成无意识死物
3. **体系内求生**：发现力士记忆中有【气禁】法术可自修，但环境不允许演练，只能忍耐等待被使用
4. **惊险伪装**：被尤道姑审视，邪念突生又强行压制，通过力士记忆伪装自己
5. **被动承受**：在修炼中被当成磨剑石（采阳补阴），忍痛承受精神侵蚀而不反抗

**关键对白/段落**:
- 【内心】『换而言之，【力士】就是一种人形的死物，我现在这种具备独立人格和思考能力的情况无疑是有问题的，一旦被发现，恐怕立刻就是灭顶之灾，所以在【域外天魔】恢复前我绝对不能暴露。』
- 【内心】『无妨，反正我如今是修炼工具，而且好像已经被人订购了，到时候肯定不愁没有演练法术的机会。』——从被动到主动的认知转换：被订购=有演练机会
- 【内心】『日尼玛，居然连平日的保养都要工具自己动？』
- 【内心】『妈的！原来是真的被当成牛马了啊！』/『没有耕坏的田，只有累死的牛！』/『这他妈哪是什么磨剑石，分明就是采阳补阴，一次性的充电宝啊！』

**新增人设维度**:
- 从凡人社会底层穿越者 → 仙门体系底层「死物」身份——压迫升级但隐忍策略延续
- 从主动卧底（ch002）→ 被动伪装（ch006）——伪装难度更高（不是装忠心而是装没意识）
- 自我修复力：利用力士记忆中的知识（真灵傀儡术）在绝境中找到反制出口

**情绪收益**: 更高密度的压迫感（不是社会底层而是直接当死物）+ 随时被发现的悬疑紧张 + 对主角忍痛力的钦佩

### ch011「我要打十个！」

**阶段**:
1. **接受刘烨哲学但不照搬**：王平听完刘烨的「忍耐」哲学，意识到这是底层法则——但自己正在通过身份跃升打破这个天花板
2. **气禁辅助加速**：利用从力士体系获得的【气禁】法术加速铁牛功修炼，进展远超刘烨预期（「第二日小成」vs「至少十七八次」）
3. **身份认同觉醒**：巡街、当街救人、1v10碾压黑衣人——从被动忍耐到主动执法
4. **拒绝江湖招揽**：以「我是官家的人」拒绝神秘势力——身份跃升后的立场宣誓
5. **章末感悟**：「社会地位高果然处处都占便宜」——从「夹缝求生」到「利用身份身份优势」

**关键对白/段落**:
- 【听到刘烨哲学】忍耐...只做远低于自身能力的事情——「我明白了」
- 【内心】『今晚争取再练一百次！』—— 表面配合但不放弃自己的节奏
- 【对白】「我是官家的人，是不会给你们这些江湖草莽做事的。」—— 身份认同的巅峰表达
- 【内心】「真是……社会地位高果然处处都占便宜呀！」—— 章末点题

**新增人设维度**:
- 从「纯苟且」到「利用身份优势主动出击」——生存策略升级
- 表面接受权威教导 vs 内心按自己节奏走——双轨制应对导师
- 官身份认同觉醒——「我是官家的人」成为行动宣言

**情绪收益**: 策略升级的成长爽感 + 身份跃升后行为空间扩展的快感 + 刘烨哲学与王平实践的对比阅读乐趣
