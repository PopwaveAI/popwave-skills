# 脱胎换骨（蜕皮新生型）

**套路类型**：修炼突破型 / 战力飞跃
**首次出现**：ch035
**关联套路**：因祸得福、金手指冷却倒计时

> **元爽点家族**：身份跃迁
> **适用章位置**：高潮

## 四段式拆解

```
【起】主角导入外部能量（灵气/天材地宝/传承）
     ——突破的「催化剂」到位
【承】身体被能量融解重铸——极度痛苦
     「就像是被人埋进了沙里，只露出了一个死人头，
      然后被人从头顶一刀劈了个口子，又导入水银」
     ——用贴近肉体的比喻让读者「感受」痛苦
【转】蜕去旧皮/旧壳——视觉化的「新生」仪式
     腐烂的皮肤剥落、血水淌出、青筋变粗
     ——旧身体被抛弃，新身体从「头顶裂口」钻出
【合】新生体魄的直观展示——不靠旁白，靠行动
     一脚踩碎青砖（不用任何内劲）
     与旧体系的巅峰状态对比（媲美天魔解体状态）
```

## 核心效果
- 让读者「感受」到修炼的代价和回报——痛苦越大突破越震撼
- 视觉化的「蜕皮」过程强化了「物种升级」的质感
- 用旧体系的最高战力作为参照物，让新体系的起点有了具体刻度
- 留下成长空间——初成阶段有外在神异（金色肌肤），完全体返璞归真

## 复用条件
- 主角需要从「旧体系」跨越到「新体系」（如武者→修仙者）
- 需要一次视觉冲击力强的突破场景来标记转折点
- 旧身体的崩溃和新身体的建立之间存在矛盾冲突

## 变体可能
- **浴火重生版**：火焰焚烧→灰烬中重生
- **冰封解冻版**：被封入寒冰→破冰而出
- **蚕蛹化蝶版**：被能量包裹成茧→破茧而出
- **骨骼重铸版**：打断全身骨骼→重铸如玉
