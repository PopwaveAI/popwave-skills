# 自毁禁术（天魔解体型）

> **元爽点家族**：世界观震撼
> **适用章位置**：中间


**套路类型**：战力爆发 / 角色代价塑造
**首次出现**：ch031
**关联套路**：玉石俱焚（自爆式反杀）、以弱胜强（金手指反杀型）

## 四段式拆解

```
【起】主角陷入绝对劣势——面对量级远超自身的军阵，常规战力无法破局
【承】主角不计代价——施展天魔解体大法，炸碎自身功体，换取数倍内劲爆发
【转】爆发成功破阵——但也付出惨痛代价：全身上下没有一块好肉，血流如注
【合】自毁后的主角放弃了气禁疗伤——因为他不是来求活的，是来求胜的
```

## 核心效果
- 为一场本不可能获胜的战斗提供合理破局手段
- 给主角贴上「不惜代价」的标签——从清醒的聪明人升级为可怕的疯子
- 制造永久性代价悬念——功体已毁，后续如何恢复？
- 与「玉石俱焚」的区别：自毁禁术是**牺牲自己换取爆发**（不一定要同归于尽），核心是「一时超频」

## 复用条件
- 主角面对量级碾压的对手，常规手段完全无效
- 需要展示主角「拼命的魄力」——不能每次都靠运气或金手指
- 需要制造「代价悬念」——让读者担心主角的永久性损失

## 变体可能
- 「燃血秘术」——燃烧精血换取战力，代价是缩减寿命
- 「功体逆转」——功法逆行产生毁灭性威力，代价是经脉尽断
- 「元神燃烧」——精神力层面的自毁，代价是痴呆或失忆
- 「修为献祭」——向某种存在献祭修炼成果换取临时力量

## 本章原文锚点
- 「天魔解体大法！」——王平大步向前，眉宇染血，功体轰然炸碎
- 「第一击，就足以定生死了。」
- 「既然来了，他就没打算活着走出来，从一开始，他就是来求胜的。」
- 「他快死了……谁还敢再和他拼命？」
