# 装备飞升（配套展示型）

> **元爽点家族**：身份跃迁
> **适用章位置**：高潮

## 基本信息
- **名称**: 装备飞升（配套展示型）
- **类型**: 成长突破型套路
- **首次出现章节**: ch015
- **嵌套型**: 是（嵌套在「暗赚反转」母结构内）

## 触发条件
主角获得为已有/将有的功法量身定制的顶级兵器 → 兵器亮相（视觉冲击）→ 与功法匹配度说明 → 战力攀升预期被拉满

## 核心张力
装备的视觉震撼 vs 功法成长的可预期性。观众既被兵器的「帅」震撼，又对「装备+功法」组合的成长路径产生强烈期待。

## 关键对白/原文证据（ch015）
- 🔒『好一张宝弓！』
- 🔒「有了此弓，三天之内入门有望！」
- 🔒「入门【血如火】，小成【骨如虎】，大成兵法【一箭霜寒】，圆满兵法【破阵天罡】」
- 🔒「只要我能将这门武功练到圆满，外功六关我就算是走过一半了」

## 情绪收益
装备猎奇爽（视觉） + 成长路径可预期的期待感 + 功法阶梯展示的"开图"快感

## 运作机制
1. **兵器亮相**：玄螭弓——通体玄金，双螭盘绕，龙首咬弦，弦如黑蟒，三支短枪箭矢，散发森然寒气
2. **功法匹配**：穿云震天弓→专精骨、血、筋→玄螭弓完美契合
3. **战力阶梯展示**：入门血如火→小成骨如虎→大成箭霜寒→圆满破阵天罡
4. **长期预期锚定**：「外功六关走过一半」→读者知道这是长期成长路径的一部分

## 变体方向
- **降级变体**：得到的兵器有瑕疵/封印，需要后续修复
- **套装变体**：多件装备组合成套装，集齐后有额外效果

### ch015 新增实例

**参与角色**: 王平
**场景**: 武库

**关键变化**: 首次出现「功法+配套兵器」组合模式

| 维度 | 说明 |
|:----|:-----|
| 兵器 | 玄螭弓——禁品级（暗示威力远超常规兵器） |
| 功法 | 穿云震天弓——中乘武功 |
| 匹配度 | 弓身玄铁抗压（骨）、龙筋弦储能（筋）、三箭爆发（血） |
| 受众预期 | 读者知道主角有灵识+超速学习能力→期待快速入门 |

**情绪收益**: 禁品兵器的稀有度满足收集癖 + 战力阶梯清晰可见的安全感
