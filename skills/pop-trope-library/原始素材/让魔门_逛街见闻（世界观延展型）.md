# 逛街见闻（世界观延展型）

## 基本信息
- **名称**: 逛街见闻（世界观延展型）
- **类型**: 世界观铺设型套路
- **首次出现章节**: ch016
- **嵌套型**: 否

| **元爽点家族** | 世界观震撼 |
| **适用章位置** | 中间 |
## 触发条件
主角因某个需求前往市井/集市/特定区域 → 在「逛街」过程中遇到各种人物和场景 → 人物互动展现世界观细节 → 同时完成需求（购买物资/获取信息）

## 核心张力
「逛街」本身不是目标，而是世界观延展的载体——通过主角的移动路径，将世界观的自然/社会/商业/势力结构逐一展示。同时逛街过程中的「人物互动」也承担了角色立体化的功能（如ch016中刘烨的退钱、砍价等行为展示其另一面）。

## 关键对白/原文证据
- 「刘扒皮又来了！」「夭寿啊，今月怎么早来了？」
- 「什么差不差的，多了！……所以这次就收你六成的例钱。」
- 「『一块是谢石的飞刀，另一块……这特么是一条销赃渠道啊！』」
- 「承惠，五十两银子。」「五两吧。」「老狗，你特么这是砍价！？」

## 情绪收益
市井生活的鲜活感和烟火气 + 世界观自然延展的知识增量（不通过说教，而是在场景中展现） + 人物立体化的惊喜 + 「逛街不白逛」的充实感 + 交易博弈的爽感

## 运作机制
1. 提出需求（ch016：王平要买异种铁升级太平经）
2. 创造逛街的理由（ch016：刘烨以收例钱为名出城）
3. 场景进入——描写市井环境（ch016：草市喧嚣、商贩惊恐）
4. 沿途事件——通过沿途遭遇展示人物/设定（ch016：米铺老板多交→刘烨退钱）
5. 目标达成——完成核心需求（ch016：买百炼元铜）
6. 附加收获——逛街过程中的意外认知（ch016：赃物销赃链的揭露）

## 使用注意
- 逛街不能变成纯水字数——必须包含设定增量或功能产出
- 沿途遭遇的人物需要有代表性——展示世界观的不同侧面
- 逛街的「产出」需要与后续剧情直接关联（ch016的素材=后续金手指升级的前提）
- 人物互动最好能「立住」人物的性格细节（如刘烨退钱、砍价等行为）
