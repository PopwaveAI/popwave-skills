# 套路：采阳补阴反转

> **元爽点家族**：绝境反杀
> **适用章位置**：高潮

## 元信息

- **套路名称**: 采阳补阴反转
- **首次出现章节**: ch006
- **所属母套路**: 反杀类套路 / 功法真相类套路
- **复用性**: 高——类似「吸收→反噬」结构可反复使用，每次变体对象/功法不同即可

## 触发条件

主角被置于「被吸收/被采补/被吞噬」的被动地位 → 对方使用某种功法入侵主角体内 → 主角因特殊体质/穿越者特质/系统漏洞而具备反向吞噬能力

## 套路公式

```
主角沦为修炼/采补的「炉鼎」→ 
对方功法入侵主角体内 → 
主角发现对方无法中途停止（功法绑定）→ 
利用自身特殊条件（穿越者精神力/功法漏洞/权限反制）反向吸收对方 → 
从被榨干者变为吞噬者
```

## 核心张力

这不是普通的战斗反转——它利用了对方功法中「无法中断」的硬锁定机制。对方越贪婪地吸收，就陷得越深；吸收越深入，主角的反制机会越大。关键特征是**功法绑定不可逆**——一旦开始修炼就无法停止。

- ch006: 「所谓用【力士】做磨剑石，就是以力士来修者运功，造成彼此气息共鸣，然后凭借自身精神力入侵力士体内，以力士的自身修为作为养料，来反哺修者自身！」
- ch006: 「尤道姑，因为她的精神力已经入侵了王平的身体，不仅没办法停下，甚至还隐隐感觉到这次修炼似乎尤为顺畅」
- ch006: 「操控力士进行修炼，不断吸收着场域内的精神力量，将它们转化为自己成长的养分。」
- ch006: （尤道姑瘫软在地，浑身香汗淋漓，面色潮红）——被反向吸收的后果呈现

## 情绪收益

震惊（真相揭露）→ 愤怒（被当牛马）→ 绝望（被吞噬）→ 智取快感（发现漏洞）→ 爆发式爽感（反杀）— 情绪弧线呈V型反弹，底部越绝望顶部越爽

## 复用提示


- 此套路的核心在于**信息不对称**——被害者必须比加害者多知道一层信息（如王平知道真灵傀儡术而尤道姑不知道）
- 反转时的「功法绑定不可逆」是确保逻辑自洽的关键——对方不是不想停，是真的停不了
- 被反噬者的身体/精神状态要具体描写（如尤道姑瘫软、香汗、潮红），增加落地感
