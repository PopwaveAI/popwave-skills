# 重诺报恩（信用增值型）

> **元爽点家族**：情感羁绊
> **适用章位置**：中间

## 基本信息
- **名称**: 重诺报恩（信用增值型）
- **类型**: 人设塑造型套路
- **首次出现章节**: ch015
- **嵌套型**: 否

## 触发条件
主角曾受他人帮助并许下回报承诺 → 主角后来获得高价值资源 → 主角主动兑现承诺 → 对方震惊/感动/羞愧 → 关系升级 → 隐性资源回报

## 核心张力
「承诺」的价值 vs 兑现的成本。中乘武功作为回报远超刘烨当初的付出（铁牛功只是下乘），这种「超预期兑现」让双方关系从「生意往来」升级为「生死交情」，同时塑造主角言出必践的信用人设。

## 关键对白/原文证据（ch015）
- 🔒「放心，大人已经允了我一个外传名额，我上无老，下无小，之前就刘老哥你对我最好，我昨夜把丹凤楼弄成那样，也给你添了很多麻烦，这门武功就如约传给你了。」
- 🔒「小子，你玩真的啊？」——刘烨愣住
- 🔒「别人以诚待我，我竟还满脑子生意。老了，当真是老了。」——刘烨自省
- 🔒「王兄你若是信得过我，此事就交给我来处理，保证足以支撑中乘武功的气血消耗。」

## 情绪收益
诚信兑现的满足感 + 对方超出预期的感动带来的「值了」感 + 信用人设的建立带来的读者信任

## 运作机制
1. **旧恩回顾（背景）**：刘烨半卖半送铁牛功，但没指望回报
2. **兑现触发**：王平获得穿云震天弓外传名额 → 主动想起承诺
3. **兑现场景**：看四周无人→拉住刘烨→塞秘籍→真诚说明「你对我最好」
4. **对方反应**：愣住→没开口→「你玩真的啊？」→羞愧自省→主动帮忙
5. **关系升级**：刘烨从「生意伙伴」→「真心兄弟」，主动提供高级食材渠道
6. **隐性回报**：王平获得练功续航资源→「三天之内入门有望」

## 变体方向
- **反向变体**：对方以怨报德，主角从此心硬，不再轻易许诺
- **延迟变体**：承诺数年后才兑现，中间铺垫对方已忘记/放弃期待
- **连锁变体**：A报恩B → B感动后帮助C → C又回头帮助A → 善的循环

### ch015 新增实例

**参与角色**: 王平、刘烨
**场景**: 校场

**关键变化**: 首次出现主角主动兑现长期承诺

| 维度 | ch015 特征 |
|:----|:----------|
| 兑现成本 | 非常高（中乘武功的外传名额） |
| 兑现时机 | 立即（刚获得名额就执行） |
| 兑现方式 | 主动+低调（看四周没人→塞秘籍） |
| 对方预期 | 零（刘烨早就不指望回报） |
| 超额程度 | 远超预期（铁牛功→穿云震天弓） |

**情绪收益**: 
- 主角形象：「靠谱」「言出必践」「重情重义」
- 读者心理：「跟着这样的主角不吃亏」「愿意为这样的主角花钱/花时间」
- 刘烨形象：从计较丹凤楼亏损→因羞愧而加倍补偿→立体化

**下游期待**:
- 刘烨后续会成为王平的重要人脉资源（食补只是开始）
- 王平的「重诺」人设将在后续剧情中反复被验证
- 潜在风险：过于重诺是否会被人利用？
