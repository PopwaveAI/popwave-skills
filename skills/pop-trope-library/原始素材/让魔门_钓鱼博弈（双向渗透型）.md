# 套路：钓鱼博弈（双向渗透型）

## 元信息

- **套路名称**: 钓鱼博弈（双向渗透型）
- **首次出现章节**: ch009
- **所属母套路**: 权谋对抗/信息差类套路
- **复用性**: 高——适用于任何存在多方势力、互相算计的剧情场景

## 触发条件

A方布局（将某物/某人故意送进B方地盘）→ B方其实看穿了A方的计谋但不揭穿 → B方反过来利用同一个事件设下埋伏 → 第三方（中间人/鱼饵）夹在双方之间同时被利用 → 核心对话揭示「钓鱼，鱼饵必死」的残酷真相

## 套路公式

```
表面事件（某物/某人落入敌方之手）→
第一个认知层次（是A方故意送的，要借B方资源达成目标）→
第二个认知层次（B方知道A方的算计，将计就计反向钓鱼）→
真相炸裂（双方都在利用同一个事件，都是钓鱼人，也都在被钓）→
主角/第三方决策（知道真相后是否仍然入场？）
```

## 核心张力

双层套娃博弈——你以为A在算计B，实际上B也在算计A。双方都在钓鱼，钓的是对方，但最先沉底的永远是鱼饵（中间人）。这种「你以为看穿了一层，实际上还有第二层」的信息差解密，是权谋类世界观的经典爽点。

更进一步，当双方都明知对方在钓鱼仍然继续博弈时，比拼的就是谁的底牌更硬、谁更能承受鱼饵被咬钩的代价。

## 关键证据

- ch009: 「守冲想要借县衙的名望，武功更进一步，大人也想着借此机会彻底铲除白莲教。」
- ch009: 「归根结底，还是各凭本事。赢家通吃，这才是人间至理。」
- ch009: 「钓鱼，是需要鱼饵的，而无论鱼儿能不能钓上来，鱼饵都是要死无葬身之地的。」
- ch009: 「王平心中明悟，却依旧没有畏惧，甚至更激动了」

## 情绪收益

信息差解密快感 + 权谋技术含量（不是简单的黑白对立，而是多层次博弈） + 对主角「明知山有虎偏向虎山行」的敬佩 + 对鱼饵身份的代入紧张感 + 对最终博弈结果的期待

## 变体方向

| 变体方向 | 示例 |
|---------|------|
| 三方博弈型：A钓B、B钓C、C钓A——闭环 | 三方势力各怀鬼胎的权谋局 |
| 放长线型：A放鱼饵后不着急收网，等着更大的鱼 | 不抓小反贼，等背后势力出手 |
| 反杀鱼饵型：鱼饵（主角/第三方）知道自己是鱼饵后反杀双方 | 主角利用信息差同时反扑两个钓鱼人 |
| 钓鱼人入局型：钓鱼人最后自己也成了更大的鱼饵 | 以为自己是下棋的人，结果也是棋子 |

## 下游期待

- 谁是最终赢家？（守冲还是陈皓彦？）
- 主角入局后能否利用自己的金手指/信息差破局？
- 鱼饵反杀钓鱼人的反套路是否会发生？
- 当双方都自以为是钓鱼人时，真正在「钓」他们的第三方是否存在？

### ch014 新增实例：钓鱼计划升级——从双人博弈到三人博弈

**参与角色**: 陈浩彦（布局者/主钓者）、守冲（被钓者/昔日钓者）、游神真人（第三方战力/棋子）、王平（鱼饵）
**场景**: 龙兴县衙内院大堂

**阶段**:
1. **局势更新**：莲华刀气运暴跌，守冲急了——ch009的「钓者」反成被钓之势
2. **计划揭晓**：陈浩彦将计就计——派王平押送莲华刀到游神观请游神真人摧毁
3. **三层博弈结构**：游神真人摧毁飞刀→守冲必然出手拦截→鹬蚌相争后陈浩彦渔翁得利
4. **鱼饵确认**：王平验证「完成任务的关键是必须活着把刀送进游神观」——失败即死

**新增维度**: ch009已经揭示了「双层套娃博弈」（守冲钓县衙⇄陈浩彦钓白莲教）。ch014将此升级为「三层博弈」——引入了第三方战力游神真人。陈浩彦的计划本质上是「三角钓鱼」：
- **顶层**：守冲（他以为他在钓？实际上他才是被钓的对象——如果不拦截飞刀，他的本命兵器莲华刀会被毁；如果拦截，就会对上游神真人）
- **中层**：游神真人（被陈浩彦请来当打手——但陈浩彦是否对他也有算计？游神真人是否也有自己的算盘？）
- **底层**：王平（鱼饵——但王平有域外天魔可反杀）

**与 ch009 对比**:

| 维度 | ch009（刘烨解密） | ch014（陈浩彦亲自揭晓） |
|:----|:-----------------|:----------------------|
| 解密者 | 刘烨（民间老捕快视角） | 陈浩彦（布局者本人视角） |
| 信息层次 | 守冲→陈浩彦双向钓鱼 | 守冲→游神真人→陈浩彦三人博弈 |
| 博弈复杂度 | 两层（守冲钓气运⇄陈浩彦钓人） | 三层（守冲钓飞刀→游神真人守观→陈浩彦渔翁得利） |
| 主角身份 | 听解密者讲解（旁观者） | 被布局者纳入计划（鱼饵入局） |
| 权谋设计 | 防御性（将计就计） | 进攻性（主动设局钓鱼） |
| 金句 | 「钓鱼，鱼饵必死」 | 「练了九死一生，不练十死无生」 |

**关键对白/证据**（新增）:
- 🔒「这一口飞刀上的气运突然暴跌。」「守冲明显有些急了」
- 🔒「我打算先将这口飞刀还给守冲，再当众将其击败。」
- 🔒「是【游神观】，我已经请来了那位游神真人」
- 🔒「他这分明是在钓鱼！」

**情绪收益**: 延续ch009的信息差解密快感 + 权谋升级的「从双人到三人」的赌博感 + 对鱼饵命运的紧张感 + 对三方博弈结果的期待 + 对陈浩彦「笑面虎+布局者」人设的深度刻画

## 复用提示

- 钓鱼博弈的核心在于「信息层次」——至少要两层才能形成「原来还有这一层」的惊喜
- 鱼饵必须是真的会死——如果主角知道自己是鱼饵但不会死，张力就碎了
- 刘烨作为「解密人」的设定很关键——由一个懂行但不入局的中间人来解释博弈格局，比主角自己看穿更有说服力
- 「钓鱼，鱼饵必死」这句金句是全套路的点睛之笔——简洁、有力、残酷

### ch017 扩展：刀架藏印后手 + 守冲战略深度

**触发条件**: ch014的「三层钓鱼」（游神真人摧毁飞刀→守冲拦截→渔翁得利）在ch017中迎来两个重大变数——(1) **陈浩彦增设后手**：在刀架底座内藏龙兴县印，王平通过灵识透视发现此秘密；(2) **守冲感知到游神真人出战而色变** → 白莲教众争议（阳谋vs偷梁换柱） → 守冲沉默：「难道.....他和我想到一处去了？」

**核心张力**: 从ch009的双人博弈 → ch014的三人博弈（引入游神真人）→ ch017的**多维博弈升级**。具体增量有三个层面：
1. **陈浩彦的后手加密**：ch014的三层钓鱼（摧毁飞刀→拦截→渔翁得利）已经完善——但ch017新增了「刀架藏印」这一变数。这到底是什么后手？用于制约守冲？制约游神真人？还是用来坑王平的额外机关？悬念密度再次提升。
2. **守冲的战略纵深**：「难道他和我想到一处去了？」——这句话表明守冲也有自己的战略计算，不是单纯的「被钓者」。白莲教众讨论的「阳谋vs偷梁换柱」只是表层，守冲的沉默意味着更深层的博弈认知。
3. **信息差的三层叠加**：读者（知道刀架藏印）> 王平（知道刀架藏印+灵识碾压）> 守冲（仅感知到游神真人）> 白莲教众（仅能看到表面）。信息层级的丰富度达到新高。

**关键对白/证据**（新增）:
- 🔒「他用了什么筹码，【游神观】的异人居然真的愿意出手？」——守冲意识到游神真人出战
- 🔒「师傅，这是县衙的阳谋！」——教众A的解读（对了一半）
- 🔒「要我说，这是偷梁换柱之计！」——教众B的解读（完全错了）
- 🔒「难道.....他和我想到一处去了？」——守冲的沉默思考（暗示守冲也有后手）
- 🔒「果然是空心的！」/「这是.....龙兴县印？」——王平发现刀架藏印（读者独占信息）

**情绪收益**: 陈浩彦后手加密的悬念感 + 守冲并非被动的博弈升级兴奋 + 信息差（读者知道刀架藏印但守冲不知）的碾压爽 + 对谁才是真正「渔翁」的强烈期待

**下游期待**:
- 龙兴县印的用途——控制守冲？制约游神真人？还是王平自身的变数？
- 守冲「想到一处去了」的后手是什么？他真的有自己的伏兵吗？
- 陈浩彦是否知道刀架藏印被王平发现了？王平会不会利用这个信息反制？

### ch018 完整闭环：跨十章博弈收束 + 第四方截胡

**触发条件**: ch017所有悬念兑现——
① 守冲揭穿龙兴县印（猜谜兑现）
② 游神观爆炸——陈浩彦的真正目标是抄游神真人老家
③ 守冲拦住游神真人——「两人的角色，陡然互换」（守冲和陈浩彦想到一处去了）
④ 王平复盘：祸水东引是表象，引蛇出洞才是真（最终答案）
⑤ 王平趁乱吞掉莲华刀+龙兴县印——**意料之外的第四方截胡**

**完整闭环追溯（ch009→ch018）**:
```
ch009: 守冲送刀→借气运→陈浩彦将计就计钓鱼（双人博弈建立）
   ↓
ch014: 气运暴跌→守冲急了→陈浩彦出新计：游神真人+王平押送+鹬蚌相争（三人博弈建立）
   ↓
ch017: 刀架藏印（陈浩彦的后手）→守冲「难道他和我想到一处去了？」（守冲也有战略）
   ↓
ch018: 【闭环】
       守冲揭穿龙兴县印（猜谜兑现）
       游神观爆炸（陈浩彦真正目标→抄游神观）
       守冲拦住游神真人（角色互换→守冲和陈浩彦想到一处去了）
       王平复盘（障眼法真相→引蛇出洞真计划）
       王平吞掉核心双宝（意料之外的第四方截胡）
```

**与 ch009/ch014/ch017 对比**:

| 维度 | ch009（建立） | ch014（升级三人） | ch017（加密后手） | ch018（闭环+截胡） |
|:----|:-------------|:-----------------|:-----------------|:------------------|
| 博弈方 | 守冲↔陈浩彦 | 守冲↔陈浩彦↔游神真人 | 守冲↔陈浩彦↔游神真人+王平(发现者) | 守冲↔陈浩彦↔游神真人↔**王平(截胡者)** |
| 信息层次 | 双层（知道对方在钓） | 三层（引入第三方打手） | 多维（刀架藏印+守冲战略深度） | 闭环（全部兑现）+ 第四方意外 |
| 博弈核心 | 谁在钓鱼谁在鱼饵 | 三人博弈谁是黄雀 | 还有多少后手没掀开？ | 所有人都不是黄雀——小人物才是 |
| 叙事功能 | 建立权谋基调 | 升级博弈复杂度 | 加密悬念，吊胃口 | 收束闭环+开辟新路 |

| **元爽点家族** | 信息差/智斗博弈 |
| **适用章位置** | 开头/结尾/高潮 |
**新增维度**: ch018的闭环实现了两个「超越预期」——
1. **超越第一个预期**——读者从ch017知道「守冲和陈浩彦想到一处去了」，但ch018揭示这处是「守冲猜出龙兴县印+拦住游神真人」——两人虽然没有直接合作，但行动完全撞车，形成了事实默契。这种「敌人默契」比直接合作更有权谋质感。
2. **超越第二个预期**——所有人都以为章末高潮是守冲vs游神真人的战斗结果——但真正的章末高潮是王平趁乱截胡。跨章博弈的最终赢家不是三大势力中的任何一个，而是被所有人视为「不值一提」的小人物。这种「底层反杀顶层」的结构，给钓鱼博弈套路增加了全新的**第四方维度**。

**关键对白/证据**（新增）:
- 🔒「是龙兴县印吧，朝廷在龙兴县的象征，用来收拢万民香火的礼器」——守冲猜谜兑现
- 🔒「……陈浩彦？」「刚刚还镇定自若的游神真人脸色剧变」——陈浩彦的真实目标揭晓
- 🔒「两人的角色，陡然互换。」——守冲拦住游神真人，攻守易势
- 🔒「祸水东引……这只是计划的表象，是故意说，故意做给外面人看的，也就是障眼法。」——真相大白
- 🔒「东西没人要吗？那我可就要拿走了哦。」——第四方截胡宣言
- 🔒「我的了！」——截胡成功

**情绪收益**: 跨十章权谋闭环的结构满足感 + 最终赢家出乎意料的颠覆感 + 第四方截胡的「鲶鱼效应」为新剧情开辟道路的期待

**下游期待**:
- 陈浩彦从游神观抄了什么？他的完整计划和预期收获是什么？
- 守冲和游神真人的战斗结果——何时发现双宝被吞？
- 王平成为第四方后——他会在ch019+如何应对三大势力的反扑？
- 龙兴县印被吞后——陈浩彦的后手彻底报废，他会如何弥补？
