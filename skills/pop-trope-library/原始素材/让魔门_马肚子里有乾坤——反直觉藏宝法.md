# 马肚子里有乾坤——反直觉藏宝法

**套路类型**：智斗 / 藏匿技巧
**首次出现**：ch021
**关联套路**：暗赚反转、苟且生存

> **元爽点家族**：信息差
> **适用章位置**：中间

## 四段式拆解

```
【起】主角拿到贵重物品，常规携带有风险（搜身/失窃/掉落/被抢）。
【承】思考多种藏匿方式，灵光一现——利用灵识控制动物伙伴吞咽。
【转】用灵识调整物品在动物体内的位置，确保不会排出。
【合】「如此，起码能藏一到三天。」——安全窗口期内完成下一步计划。
```

## 核心效果
- 出人意料的机智操作，让读者会心一笑
- 动物伙伴从「坐骑」升级为「移动保险箱」
- 后续可能引发「杀马取物」的紧张桥段

## 复用条件
- 主角有精神控制/灵识/念力类能力
- 身边有动物伙伴（马/狗/鸟等）
- 需要临时藏匿关键物品，且即将进入高风险区域

## 变体可能
- 藏入树洞/水底/粪坑等反直觉地点
- 藏入死人尸体（最危险的地方最安全）
- 藏入公共区域（灯下黑）

## 本章原文锚点

- 「将剩下的丹药包好，密封，然后用灵识辅助自己的卷鬃马一一吞下，藏进了马肚子里。」
- 「王平运转灵识，调整丹药在马肚里的位置，确保不会拉出来后，这才放下心来」
