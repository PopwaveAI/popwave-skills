- **类型**: 世界观解密型套路
- **首次出现章节**: ch019
- **嵌套型**: 否（独立使用）

| **元爽点家族** | 世界观震撼 |
| **适用章位置** | 结尾/高潮 |
## 触发条件
主角通过某种超自然方式（金手指回溯、禁术探查、记忆读取等）看到被隐藏的历史真相 → 真相具有系统性不公特征（不是个人贪腐，而是制度性压迫）→ 揭示过程逐层递进（从具体场景到抽象制度）→ 以核心金句收束 → 触发主角的行动决意

## 套路公式
```
金手指被动触发/主动使用（幻象回溯历史）→
真相场景逐层展开（微观场景→执行层→决策层）→
双视角交错展示（受害者视角+加害者视角）→
剥削数据具象化（隐田/隐户/粮车数据）→
阶级双标揭示（自己人宽待 vs 外人压榨）→
核心金句收束（轻描淡写的最冷酷台词）→
利益闭环展示（加害者如何从中获利）→
主角个人共鸣（親歷或相似记忆激活）→
主题升华（「龙兴县已经死了」「百官印是万民血」）
```

## 核心张力
这不是普通的「揭露贪官」套路——而是通过**系统的全景式展示**来揭示不公的「体制化」特征：
1. **层级分工**：底层执行者（捕快）→中层管理者（陈浩彦）→高层决策者（知县）——每个层级都相信自己在「按规则办事」
2. **自我正当化逻辑**：有看似合理的理由——「良民是自己人所以亲亲相隐」「黔首不懂武功溅不起水花」——不是简单的贪，而是有意识形态的阶层歧视
3. **利益闭环**：执行者获得升迁回报（陈浩彦获神捕门推荐）——系统内每个人都从压迫中获利
4. **温和的冷酷**：最残忍的政策用最平淡的语气说出（「就再苦一苦他们吧」）——比暴政更有冲击力

## 情绪收益
真相逐步揭露的解锁快感 + 对系统性压迫的愤怒 + 多视角共鸣叠加 + 「就再苦一苦他们吧」一句金句引爆情绪 + 从「个人贪腐」到「制度不公」的认知升级

## 运作机制
1. **幻象/回溯机制**：需要一个「上帝视角」来展示历史——太平经的回溯功能是理想载体
2. **递进结构**：微观（老农被逼）→中观（运粮进城）→宏观（知县+陈浩彦对话）——从小到大的视角扩展
3. **双标对照**：必须展示「同一制度对不同阶层的不同待遇」（良民一成 vs 黔首全交）
4. **数据具象化**：抽象的「剥削」需要具体数字（隐田十万亩/隐户三千八百/粮车没地方放）来让读者感受到压迫的规模
5. **核心金句**：需要一句轻描淡写但杀伤力巨大的台词来收束（「就再苦一苦他们吧」）

## 使用注意
- **不能变成说教**：真相必须通过「场景+对话」展示，不能通过旁白直接宣讲
- **需要对照光**：纯粹黑暗会让读者窒息——需要一点「光」（如王平决定复仇）来平衡
- **金句不能太刻意**：太文绉绉或太煽情都会破坏「冷酷」的效果——平淡中见残忍最好
- **不能只揭露不行动**：揭露真相后必须触发主角的某种行动决意，否则读者会感到无力
