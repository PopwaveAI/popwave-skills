# 修仙体系大展开型

**套路类型**：世界观展示型 / 信息释放 / 体系建立
**首次出现**：ch034
**关联套路**：修炼等级体系、传功、修行指南、升级路线图

## 四段式拆解

```
【起】主角接触到一个信息载体（玉简/书籍/传功）。
     这是一个有价值的信息源——来自可靠的背景来源：「仙门八峰之一豢龙峰的秘法。」
     主角正处于转型期（武道→修仙/低级→高级），需要知道新体系的框架。

【承】信息以层次化方式释放——先总纲后分述。
     总纲：「何为修仙？总结下来其实就是九个字——渡劫，炼炁，修真而登仙。」
     分述：渡劫一重人劫→通灵识念/无漏真身/周天吐纳/借炁显法→提炼法力→……
     渡劫二重天劫→风火雷三劫→法术升华为神通→……
     渡劫三重命劫→炼炁→（玄之又玄的描述）→

     用「功法本身没写更高了」来制造未知/期待感。
     同时展示主角卡在哪里：「废灵根大道无望，指的其实就是无法炼炁。」

【转】主角的旧有认知被刷新——之前的瓶颈找到根源。
     「我之前还以为是自己悟性太低。现在看来是因为我没有提炼出法力，硬体不达标。」
     让读者和主角同时获得「串联起来」的快感。

【合】新体系确立，主角明确了自己的路径。
     「不能好高骛远，还是先炼出一道废灵根再说。」
     路标清晰了——读者知道现在第一步是什么，未来要走向哪里。
```

## 核心效果
- **世界观一次性搭建**——避免碎片式信息带来的混乱感
- **读者参与感**——和主角一起「第一次接触」新体系
- **升级清晰度拉满**——读者知道「人劫→天劫→命劫→炼炁」的路径
- **旧伏笔回收**——用新设定解答前文遗留的疑问（气禁为何不能圆满）

## 复用条件
- 故事存在明确的层次/等级升级体系
- 主角即将进入一个全新的领域/体系
- 旧有体系的瓶颈可以和新体系产生关联解释
- 需要一个整理信息的信息载体（玉简/书卷/宗门典籍）

## 变体可能
- **口述版**：导师/师长口述——更有带入感，但信息量受限
- **战斗体验版**：通过实战感受等级差距来让读者理解体系
- **对比版**：同时展示两个体系的差异（武道vs修仙）——更适合体系转换章

## 本章原文锚点
- 🔒「何为修仙？总结下来其实就是九个字：渡劫，炼炁，修真而登仙。」
