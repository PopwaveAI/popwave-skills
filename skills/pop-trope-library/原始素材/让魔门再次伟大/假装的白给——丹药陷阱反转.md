# 假装的白给——丹药陷阱反转

**套路类型**：夺宝反转 / 智商碾压
**首次出现**：ch021
**关联套路**：暗赚反转、因祸得福

## 四段式拆解

```
【起】主角开箱获得大量宝藏（12瓶丹药），读者跟着一起激动。
【承】主角突然起疑：「万一标签是假的？」冷静分析出防盗动机。
【转】用马试药，马吐白沫四脚朝天——果然有毒有诈！愤怒+庆幸双重情绪。
【合】主角反手将毒丹塞进标注正确丹药的瓷瓶，「说不定能起奇效」——把陷阱变成自己的武器。
```

## 核心效果
- 塑造主角多疑、谨慎、机智的正面形象
- 避免「主角傻白甜捡漏」的俗套
- 将一次性陷阱转化为后续暗器/暗算伏笔

## 复用条件
- 主角获得看似丰厚的战利品
- 该战利品来自一个心机深沉的势力/人物（游神观）
- 主角有验证手段（灵识/试药/探测）

## 变体可能
- 宝藏本身无毒，但藏有追踪标记/诅咒/奴役契约
- 宝藏为活物（寄生/封印怪物）
- 宝藏带有记忆读取/魂魄残留

## 本章原文锚点
- 「万一里面放着的其实是【断肠丸】，那又该怎么办？」
- 「所有标注都是乱的……真正用来区分的是瓷瓶上的花纹」
- 「虽然只是随手之举，但说不定能起到奇效」
