# 先贷款后修炼（加速型）

## 基本信息
- **名称**: 先贷款后修炼（加速型）
- **类型**: 修为加速型套路
- **首次出现章节**: ch033
- **嵌套型**: 是——通常嵌套在「经卷结算」或「金手指觉醒」等套路中，作为能力效果的具体应用

## 触发条件
主角获得某种可实现「预支成果—事后补票」的机制 → 快速推演各用法 → 锁定修炼加速为最优解 → 理解该机制对修炼速度的颠覆性提升 → 产生明确的实力跃迁预期

## 套路公式
```
获得预支机制（立承诺→提前支取成果）→
排除错误用法（不能杀人/不能直接突破）→
锁定最优解（修炼加速）→
回顾已有成功案例（气禁修炼的速度验证）→
想象极限用法（如预支圆满境界再回头修行）→
产生明确预期（修炼速度成倍提升）
```

## 核心张力
1. **反直觉逻辑**：不是先修炼后突破——而是「先突破体验→回头修炼→验证实现」，颠覆传统修炼认知
2. **认知爽**：读者和主角一起快速理解「精算师」思维带来的巨大优势——勤奋不如策略
3. **已有验证**：不是凭空画饼——王平已有「气禁」通过力士经验灌注快速提升的先例，增加可信度
4. **代价清晰**：限时完成 + 完不成反噬报废——有明确的机制保障不崩坏

## 关键对白/原文证据
- 🔒「【一诺千金】激活后，我可以在经卷上写下一道【承诺】，然后提前支取【承诺】完成之后的成果。」
- 🔒「之后只要在时限内完成【承诺】，那提前支取来的成果就可以永久保留，反之，如果没能完成【承诺】，那经卷就反噬」
- 🔒「这道经卷……不能用来杀人啊，毕竟人不可能死两次，用这个来作为【承诺】几乎注定了会食言。」
- 🔒「思来想去，只有一个用法性价比最高——修炼。」
- 🔒「武功也好，仙法也罢，只要用这道经卷，就可以提前支取成果，然后再回过头修炼，效率事倍功半。」
- 🔒「就比如他修炼【气禁】，有了作为力士的经验灌注后，这门法术他在极短时间内就修炼到了续骨连筋的层次，只差一步就能做到真气覆体，刀枪不入」

## 情绪收益
新能力的惊艳感 + 最优解锁定后的顿悟快感 + 「先上车后补票」的暗爽 + 对后续快速升级的期待 + 规则理解的智力展示

## 运作机制
1. **预支机制**：立承诺→提前拿到成果体验→回头补修炼票
2. **验证环节**：必须有已有案例证明这个机制的有效性（气禁的先例）
3. **排除法推演**：逐步排除错误用法，最终锁定最优解——展示主角的判断力
4. **边界清晰**：限时完成+反噬代价——避免让读者觉得太无脑强
5. **人死债消**：死亡重置承诺——和主角的复活机制形成完美联动，进一步降低风险

## 使用注意
- **不能无中生有**：预支的成果必须是主角有能力通过正常修炼达到的——不能预支「武道宗师」给一个刚入门的武者
- **限时必须存在**：没有时间限制就变成纯增强，失去张力
- **推演不能太长**：用法推演应在1-2个自然段内完成——拖沓会削弱「认知爽」
- **已有案例要匹配**：「气禁」的案例和「一诺千金」的机制不完全一样（一个是经验灌注，一个是预支成果），需要说明关联但不强求完全一致
