# 套路：功法真相揭露（黑幕型）

## 元信息

- **套路名称**: 功法真相揭露（黑幕型）
- **首次出现章节**: ch006
- **所属母套路**: 世界观解密类套路
- **复用性**: 高——几乎所有高级功法的设定都可以做「先粉饰后揭露」的双层结构

## 触发条件

角色/读者对一门功法产生初步认知（往往是正面的）→ 主角亲自体验/参与修炼 → 发现功法的真实运作机制与表面描述完全不同 → 该真相往往带有道德/伦理上的黑暗面

## 套路公式

```
功法正面描述（磨剑石/辅助修炼） → 
主角参与其中 → 
功法真实运作机制被亲身经历 → 
揭露「美丽包装下的黑暗真相」→ 
刷新对体系/世界的认知
```

## 核心张力

骗局揭露的快感——当一个看似美好的设定被撕开画皮，露出底层的剥削/掠夺/邪恶本质时，读者会经历「被骗→愤怒→信任崩塌」的三段式冲击。这不仅是功法的真相，也是整个仙门体系道德底色的揭示。

## 关键证据

- ch006 表层描述: 「修炼【斩仙葫芦真解】已是绰绰有余了」/「需要【力士】来充当「磨剑石」」
- ch006 深层真相: 「所谓用【力士】做磨剑石，就是以力士来修者运功，造成彼此气息共鸣，然后凭借自身精神力入侵力士体内，以力士的自身修为作为养料，来反哺修者自身！」
- ch006 叙事者解读: 「这他妈哪是什么磨剑石，分明就是采阳补阴，一次性的充电宝啊！」
- ch006 认知落差: 力士记忆中的【气禁】法术是为了「磨损后自我修复」——从「修炼工具」到「一次性消耗品」再到「连保养都要自己动手」，层层递进

## 情绪收益

认知颠覆爽感 + 道德愤怒 + 世界观拓宽 + 对主角处境的深切同情（被迫成为黑幕的受害者）

## 变体方向

| 变体方向 | 示例 |
|---------|------|
| 丹药真相：表面是提升修为 → 实际上是控制服用者的毒药 | 丹药黑幕 |
| 师徒制度：看起来是传承 → 实际上是夺舍/转世容器 | 传承黑幕 |
| 宗门大比：表面是选拔 → 实际上是献祭/筛选养料 | 赛事黑幕 |

## 复用提示

- 真相揭露分三层最佳：①正面描述（好看）②参与体验（有疑点）③真相炸裂（完全颠覆）
- 用主角的现代价值观（牛马/充电宝/采阳补阴）做对比翻译，强化讽刺效果
- 黑幕揭露后要在后续章节产生实际影响——否则就成了「知道真相就完了」的无效揭露
