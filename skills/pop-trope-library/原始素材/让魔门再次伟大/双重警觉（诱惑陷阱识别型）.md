# 双重警觉（诱惑陷阱识别型）

> 本套路属于「智斗反转型」类别。

## 定义

角色面临一个看似完美的机遇/利益诱惑，初始反应是"正合我意"的野心升腾，但在极短时间内（通常是瞬间）凭借敏锐的情报关联能力识破背后的陷阱，情绪从欲望高峰急转直下。核心在于"从欲望中冷静，在诱惑中警觉"。

## 适用条件

- 存在一个显而易见的利益诱惑（升职、宝物、功法、地位）
- 该诱惑的设置方有足够动机和能力设局
- 主角掌握某些旁人不知道的关联信息
- 需要一个情绪急转直下的瞬间

## 基本结构

```
① 抛出诱惑（升职/宝物/机会）
② 主角本能兴奋——「正合我意！」（欲望升腾）
③ 主角突然触发某个关联信息
④ 将诱惑的"条件/时间/方式"与已知信息串联分析
⑤ 识破陷阱——「如坠冰窟」
⑥ 重新评估局势，调整策略
```

## 关键要素

| 要素 | 说明 | ch022示例 |
|------|------|-----------|
| **诱惑物** | 看似诱人的机会 | 一个月后捕头大比 |
| **时间关联** | 诱惑的时间正好匹配某个已知条件 | 一个月正好是消化丹药所需时间 |
| **幕后设局者** | 诱惑的提供方有可疑动机 | 徐秉正可能是在钓鱼 |
| **信息串联** | 主角掌握旁人不知道的信息链 | 陈浩彦强闯游神观的丹药幕后可能是知县 |

## 情绪节奏

```
欲望值
  ▲
  │    ★「正合我意」——欲望峰值
  │    ┃
  │    ┃
  │    ┃   ★「等等，不太对」——警觉萌芽
  │    ┃   ┃
  │    ┃   ┃   ★「如坠冰窟」——情绪谷底
  └────┴───┴───────────────▶ 时间
      t0  t1  t2
```

## 变体

| 变体 | 说明 | 示例 |
|------|------|------|
| **双面诱惑** | 诱惑本身是真的，但也是陷阱（中了有奖，不中则亡） | 大比赢了是捕头，但可能暴露身份 |
| **他人替入** | 主角看穿后，刻意引导他人入局 | 不直接拒绝，而是让别人当替罪羊 |
| **以诱为饵** | 主角将计就计，利用自己"中计"来反制设局者 | 明知是钓鱼，仍假装上钩反向操作 |

## 复用技巧

1. **信息链条必须合理**：主角之所以能识破，是因为他掌握独特信息（非上帝视角）
2. **欲望越真实越好**：诱惑必须是真的让人心动，不能是"一眼假"
3. **反转要在同一段内完成**：从"正合我意"到"如坠冰窟"最好在连续叙述中完成，制造强烈的情绪对冲
4. **留有后路**：识破陷阱不代表放弃，可转为将计就计

## 适用场景

- 上级/敌人突然开出优厚条件
- 看似偶然的"好机会"
- 主角需要做战略选择时
- 权力交接/职位更替的敏感时期
