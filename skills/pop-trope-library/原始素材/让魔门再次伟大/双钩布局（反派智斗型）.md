# 双钩布局（反派智斗型）

## 基本信息
- **名称**: 双钩布局（反派智斗型）
- **类型**: 智力博弈型套路
- **首次出现章节**: ch015
- **嵌套型**: 否

## 触发条件
正反双方都在等待对方露出破绽 → A方设下圈套引蛇出洞 → B方看穿A方计谋 → B方将计就计/调虎离山 → 读者同时知道双方算计但不清楚正面交锋何时爆发

## 核心张力
双线智斗构建的「棋局」感——读者仿佛站在棋盘上方，同时看到黑白双方的布局。双方都以为自己掌控局面，但读者知道真相。

## 关键对白/原文证据（ch015）
- 🔒「死的好！」——守冲对谢石之死满意
- 🔒「他信了，真的信了！不出我所料的话，他应该已经在设法引蛇出洞，想要逼我为了取回【莲华刀】，主动踏入陷阱里。」
- 🔒「【莲华刀】重要吗？当然重要，可狡兔三窟，他当初将其送出去的时候就做好了事不可为即放弃的准备，岂会不留后手？」
- 🔒「他想和我玩引蛇出洞，我就给他来个调虎离山。」
- 🔒「【游神观】......」

## 情绪收益
双线布局的智力快感 + 读者比角色先知道真相的优越感 + 对正面对弈爆发的期待感

## 运作机制
1. **A方布局（明线）**：陈浩彦日夜守在县衙，等守冲来取莲华刀（引蛇出洞）
2. **B方看穿（暗线）**：守冲通过谢石之死确认陈浩彦上钩了
3. **B方评估**：莲华刀价值已降低（白莲教气运散尽），放弃代价可承受
4. **B方反制**：调虎离山——声东击西，目标转移至【游神观】
5. **读者视角**：双方都在精心计算，但交锋还未真正开始 → 悬念拉满

## 变体方向
- **多层变体**：C方也在布局，三钩互相嵌套
- **反转过快变体**：一方以为自己在反制，其实已被更高维度看穿
- **延迟引爆变体**：双钩布局后，实际交锋延迟到3-5章后才爆发

### ch015 新增实例

**参与角色**: 陈浩彦（明线）、守冲（暗线）
**场景**: 县衙 / 龙兴县外乡村

**关键变化**: 首次出现正反派同时布局的智斗模式

| 维度 | 陈浩彦（捕头） | 守冲（白莲教） |
|:----|:--------------|:--------------|
| 策略 | 引蛇出洞 | 调虎离山 |
| 目标 | 莲华刀是诱饵，等守冲来取 | 莲华刀可放弃，目标游神观 |
| 已知信息 | 认为守冲必来取刀 | 确认陈浩彦已上钩 |
| 真实意图 | 抓住守冲 | 趁机袭扰游神观 |
| 当前状态 | 自以为计谋得逞 | 已经把陈浩彦看穿+反制 |

**情绪节奏**: 王平线轻松喜剧→突然切到守冲线→紧张感骤升→信息量冲击（双布局）→悬念感拉满

**情绪收益**: 
- 反派智力在线 → 提升危机感 → 后续冲突含金量更高
- 读者感受到「双方的战斗不是武力对砍，而是智力对弈」
- 世界观深度：白莲教不是无脑反贼，有严密组织与信仰体系

**下游期待**:

- 莲华刀押送任务是否会触发三日后的大战？
