# 反向诱惑（审判者堕落型）

## 套路概述

**核心机制**：负责监管、押送、毁灭某件"禁物"的审判者/执法者，在接触禁物的过程中自己也被其吸引——审判者从「监管者」变成「潜在的堕落者」。

**关键词**：审判者 · 禁物诱惑 · 反向吸引 · 人物弧光 · 欲望软肋

## 触发条件

1. **审判者身份**：角色被设定为负责监管/押送/毁灭某件禁物的人——他的任务就是防止禁物被他人接触
2. **禁物存在内在诱惑力**：该禁物必须具有自身的吸引机制——不是因为外在威胁而危险，而是因为"看了就会忍不住"
3. **脆弱时刻**：审判者在某个特定的脆弱时刻（身心疲惫/意志松动/独处时）接触禁物
4. **清醒与堕落的拉锯**：审判者必须在被吸引和保持清醒之间反复摇摆——最后通常以惊退收尾，但种子已埋下

## 执行节奏

```
① 审判者持禁物（认为自己能免疫诱惑）
② 独处/脆弱时刻 → 禁物散发诱惑
③ "不如，我也看看？"（自我说服）
④ 接触禁物 → 被吸引（开始魔怔）
⑤ 某种刺激使其惊醒（外部声响/内在警觉）
⑥ 惊退 + 愤怒/自省（"妖孽！"）
⑦ 禁物被封存/受损——但种子已经埋下
```

## 适用场景

| 场景类型 | 示例 |
|---------|------|
| 押送禁物途中 | 押送者自己好奇看了禁法秘籍——从此埋下堕落种子 |
| 封印/毁灭禁物时 | 负责毁灭某件邪器的修士，在最后关头被邪器诱惑，不舍得毁掉 |
| 审讯禁术持有者 | 审讯禁术修士的执法者，通过审问了解了禁术内容——开始暗中尝试 |
| 看管禁地/禁书库 | 图书管理员被禁书吸引，偷偷抄录 |

## 变体

1. **渐进式**：不是一次性被吸引，而是在多次接触中逐步沦陷——每次都说"最后一次"
2. **反向拯救型**：审判者被吸引后，反而开始保护禁物不被其他人毁灭
3. **被牺牲型**：审判者最终被禁物反噬——成为下一个怪物/案件的主角
