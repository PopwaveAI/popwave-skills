# 名珍兵器首秀（兵器谱认证型）

**套路类型**：装备展示型 / 世界观拓展
**首次出现**：ch027
**关联套路**：装备飞升、武力展示

## 四段式拆解

```
【起】反派（或高阶角色）祭出已知名兵器——该兵器名列【兵器谱】。
     名称亮相，给出名次：「【兵器谱】，第三百四十二位，【神仙索】徐秉正」
【承】兵器"名不虚传"——主角用手中的常规武器硬抗，
     结果常兵（牛尾刀）直接被崩出缺口，无法破坏兵器分毫。
     "明明只是一张轻薄的网，却重的不可思议。"
【转】兵器的"异常"被揭示——
     它不只是材质特殊，更是因为位列【兵器谱】，
     积累了大量名望和气运，已经超出了"凡兵"的范畴。
     """【利器】……不，是【名珍】！"""——名珍（兵器谱兵器）拥有神异。
【合】主角不得不切换高阶装备或战术应对——
     （玄螭弓+合力一斩）才勉强破开一丝空隙。
     战斗被装备等级重新定义：不是"谁武功高"，而是"谁的兵器强"。
```

## 核心效果
- 世界观拓展——证明存在"有灵/有神"的兵器
- 制造装备压制感——让读者认识到神兵利器的重要性
- 为主角未来的装备获取/升级铺垫期待

## 复用条件
- 故事存在【兵器谱】或类似的"名器排行"设定
- 主角即将/需要获取更强装备
- 需要制造"常规手段无效"的战斗困境

## 变体可能
- **法器版**：修真世界的法器、灵宝登场——"这不是法宝，是灵宝！"
- **名剑版**：名剑谱/剑冢中的名剑——"此剑在名剑谱上排名第七！"
- **神器版**：主角获得/继承神器——但无法完全掌控
