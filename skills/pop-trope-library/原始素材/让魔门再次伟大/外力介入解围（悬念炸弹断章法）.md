| 项目 | 内容 |

| **套路类型** | 结构型 / 断章型 |
| **可复用性** | 高（卷末/章末常用断章手法） |

## 定义与触发条件

**触发条件**：内部对峙陷入僵局/主角面临生死一线 → 外部第三方势力突然介入 → 章末断在介入瞬间 → 下一章回收

**核心张力**：读者既想知道内部危机如何解决，又想知道外部势力来干什么——双重悬念叠加

- 🔒「斩一贼众者，赏银十两，谁能斩了那个白莲贼首，赏银五百两！」

## 情绪收益

- 悬念拉满（两个未解决的问题同时存在）
- 追读驱动（读者必须看下一章）
- 节奏变化（从极静对峙→极动混乱）

## 变体与扩展方向

| 变体 | 说明 |
|:----|:-----|
| **反派内讧** | 第三方是反派势力的另一派系介入 |
| **天灾介入** | 地震/火灾/洪水等自然力量打断对峙 |
| **第三方渔翁** | 第三方势力坐山观虎斗，章末才入场 |
| **误打误撞** | 第三方势力目标不是主角，但意外解围 |

## 使用注意事项

1. 介入势力的动机需在一章内交代清楚——不能留到3章后解释
2. 介入本身要足够猛烈（巨响/爆炸/死亡等）才能打断高强度对峙
3. 不要每卷结尾都用——适当间隔产生最大效果
4. 下一章回收时，内部危机的解决方式需合理（不能因为外力来了大家就握手言和）
