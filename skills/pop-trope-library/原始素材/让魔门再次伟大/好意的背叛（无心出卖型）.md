# 套路：好意的背叛（无心出卖型）

## 元信息

- **套路名称**: 好意的背叛（无心出卖型）
- **首次出现章节**: ch026
- **所属母套路**: 悲剧反转型套路
- **复用性**: 中高——任何「亲友团好心办坏事」的场景均可复用，但不宜在一本书中使用超过2次

## 触发条件

亲近之人（朋友/亲属/伙伴）获得某种向上级/他人展示的「好事」→ 此人出于善意/兴奋/自豪向主角展示成果 → 展示过程中无意间暴露了主角的秘密或引起了敌对势力的注意 → 主角陷入本可避免的危机

## 套路公式

```
亲近之人获得馈赠/晋升（看起来的大好事）
    ↓
此人兴奋地找主角分享/展示
    ↓
出于炫耀或自豪，暴露了某些不该暴露的信息
    ↓
主角发现端倪，但为时已晚
    ↓
更强大的敌人已锁定主角
```

## 核心张力

**善意的背叛 × 无法责怪** — 最残忍的背叛不是来自敌人，而是来自朋友的无心之举。读者和主角都无法对这个「背叛者」产生愤怒（因为他确实出于好意），只能将矛头对准真正的幕后黑手。这种「有怨无处发」的憋屈感是情绪张力的核心来源。

## 关键证据

- ch026: 刘烨兴奋地来找王平，展示了知县灌顶后练成的伥龙劲
- ch026: 「刘烨还在滔滔不绝地说着对未来的畅想」——完全不知道自己闯了多大祸
- ch026: 「王平连呼吸都停了，所有思绪都在看到那本秘籍的瞬间陷入了冻结」

## 情绪收益

- 强烈的悲剧感（最好的朋友害了你，但你不能怪他）
- 对幕后黑手的仇恨值飙升（如果刘烨被利用，那设计这一切的人更可恨）
- 悬念张力（主角要如何处理这段关系？告诉刘烨真相？还是不告诉？）

## 变体方向

| 变体方向 | 示例 |
|---------|------|
| **无意泄密变体** | 朋友聊天时说漏嘴，暴露了主角行踪/计划 |
| **礼物捅娄子变体** | 朋友想送礼给主角，但那礼物恰好是反派追踪的关键 |
| **好心解围变体** | 朋友想帮主角解围，结果反而让情况更糟 |
| **亲情绑架变体** | 亲人打着「我为你好」的旗号做了伤害主角的事 |

## 使用注意事项

1. 「出卖者」必须是真的好心——如果读者能看出他有一丝恶意，套路就失效了
2. 主角不能对此人发火——这是一场主角和读者一起「咽下去」的苦涩
3. 后续需要交代主角如何/是否处理与这个人的关系——不是结束，而是另一段故事的开端
4. 可与「幕后黑手布局」联动——如果好意的背叛是被幕后黑手刻意设计的，效果翻倍（ch026即为此情况）
