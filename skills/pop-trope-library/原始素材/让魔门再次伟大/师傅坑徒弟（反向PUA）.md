# 套路：师傅坑徒弟（反向PUA）

## 元信息
- **套路名称**: 师傅坑徒弟（反向PUA）
- **首次出现章节**: ch008（但原型可追溯至ch002守冲让谢石盯梢王平）
- **所属母套路**: 师徒关系/喜剧型/反派侧面刻画
- **复用性**: 高——可与「双向渗透（套娃卧底）」配合使用，也可独立用于刻画"表面慈师实则利用"的反派形象

## 触发条件
师傅（上位者）面临风险/任务 → 徒弟（下位者）主动关切（表示担忧/想要帮忙）→ 师傅欣然接受徒弟的好意 → 徒弟以为是师傅亲自出马 → 下一秒被告知「你去执行」

## 套路公式
```
风险/任务出现（触发）→ 徒弟关切询问（主动入坑）→ 
师傅表面赞赏（钓鱼）→ 徒弟产生期待（以为师傅亲自出马）→ 
师傅揭底：你去执行（反差打击）→ 徒弟懵逼/绝望（喜剧效果+悬念）
```

## 核心张力
徒弟以为是在表忠心/拍马屁→没想到师傅真的把自己推出去送死→形成强烈的认知反差。这种「你以为你很会做人，其实你是被坑得最惨的」的套路，既是喜剧效果，也是刻画出师傅（守冲）自私深沉人设的高效手法。

**守冲的套路结构：**
1. 徒弟表示关切：「师傅，太危险了」→ 表面是关心，实则是给师傅一个台阶
2. 师傅先肯定徒弟的关心：「为师当然也明白」→ 制造「师傅英明」的印象
3. 师傅暗示有万全之策：「你放心，为师早已备下万全之计」→ 徒弟以为师傅自己有计划
4. 徒弟敬佩：「师傅，您要强闯县衙？」→ 期待值拉满
5. 师傅揭底：「不，是你要强闯县衙。」→ 认知崩坏

## 关键证据
- ch008: 「好徒儿，你放心，为师早已备下万全之计，区区县衙困不住为师。」
- ch008: 「师傅，您要强闯县衙？」「不，是你要强闯县衙。」
- ch008: 谢石：「？？？」
- ch002（原型）: 「盯着王平，我要知道他未来几天的所有行踪」→ 让谢石干脏活

## 情绪收益
喜剧效果（替谢石尴尬/同情）+ 对师傅（守冲）人设的立体化（他不仅是反派，还是坑徒弟专业户）+ 悬念（谢石真的会去？结果如何？）+ 对王平命运的期待（以捕快身份迎战谢石）

## 下游期待
- 谢石能否成功强闯县衙？
- 王平以捕快身份面对昔日的师兄谢石——谢石能否认出王平？
- 守冲是否有其他后手？还是真的只是在消耗谢石的价值？
- 这条师徒线的后续——谢石会在什么时候意识到自己被坑？

## 复用提示
- 师傅坑徒弟的核心在于**反转期待**——先让徒弟（和读者）以为师傅要自己上的时候，突然反转
- 这种套路最好在作品中**重复使用**——每次谢石主动关心师傅时，读者都会本能的警觉「又来了？」，强化喜剧惯性
- 可以与**双向渗透（套娃卧底）** 配合——如果谢石也另有身份，那这坑就变成了多重套娃
- 谢石的「？？？」反应是精髓——不需要任何对白解释，一个懵逼表情就够了
