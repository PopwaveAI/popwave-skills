# 套路：希望→落差双重打击（虚假希望法）

## 元信息
- **套路名称**: 希望→落差双重打击（虚假希望法）
- **首次出现章节**: ch005
- **所属母套路**: 通用情绪操控套路
- **复用性**: 高——可反复用于制造情绪反差

## 触发条件
主角遇到看似极好的机遇（获取希望）→ 迅速发现机遇背后是更可怕的现实（落差打击）

## 套路公式
```
主角听到/看到/遇到表面上极好的事物 → 读者与主角一同狂喜 → 
下一秒细节揭示 → 实际远不如表面美好 → 情绪跌至谷底
```

## 核心张力
先给读者最高期待（修仙世界）→再狠狠摔碎（沦为工具人）→才发现处境比想象中更可怕（人格洗脑）。

**三级落差结构：**
1. ⬆ 修仙世界！长生不老的梦想！
2. ⬇ 我是力士（傀儡/奴仆/工具）
3. ⬇⬇ 还有思想钢印（会被洗脑成忠仆）

## 关键证据
- ch005: 「修仙世界？王平瞬间激动，毕竟谁没有一个修炼成仙，长生不老的梦？」
- ch005: 「终于炼制成功了，这一批的【力士】成功出炉，可以拿去交货了」
- ch005: 「我是，护法院的【力士】」「仙门特地炼制出来服务一众弟子的傀儡，奴仆」

## 情绪收益
强烈情绪反差 + 对主角处境的深度共情 + 对仙门黑暗面的好奇

## 下游期待
- 力士在仙门的日常待遇到底有多惨
- 这个仙门到底是什么性质的势力（正/邪/中间态）
- 其他力士是否也有独立意识？还是王平是唯一的特例

## 复用提示
使用时需注意：每一轮希望→落差的设计必须比上一轮更深、更痛，否则读者会产生免疫。ch005的三级结构（仙门 → 力士 → 钢印) 是良好的递进示范。
