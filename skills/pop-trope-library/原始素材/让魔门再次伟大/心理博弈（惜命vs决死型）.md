# 心理博弈（惜命vs决死型）

**套路类型**：战斗心理 / 智谋破局
**首次出现**：ch031
**关联套路**：以伤换伤（耗命搏杀型）

## 四段式拆解

```
【起】主角身陷绝境，被军阵层层围困——战力不敌，伤势累积
【承】主角放弃气禁疗伤——主动让自己的伤势恶化到濒死边缘
【转】围杀者发现主角快死了——反而无人敢最后一击
        心理逻辑链：①我们人多，他必死→②但他临死前能拉垫背→③谁当这个垫背？→④等他自然死，领赏更安全
【合】主角用「即将到来的死亡」作为武器——不是不能打，是没人敢打
```

## 核心效果
- 用智谋补足战力差距——惜命是人类本能，决死者的最大优势就是对手不想死
- 制造反直觉反转——「越虚弱越安全」的悖论
- 刻画战术级心理博弈的能力——主角不仅武功高，还懂人性
- 制造剧情张力——读者既担心主角的伤势，又被他的智谋折服

## 复用条件
- 主角面对人数远多于己的对手
- 对手有「惜命思维」——体制内人员、雇佣兵、非死士类型敌人
- 主角已有明显不可逆伤势（伪装或真实皆可）

## 变体可能
- 「诈死诱敌」——假装死亡引诱对手靠近，然后偷袭
- 「自残威慑」——主动制造恐怖伤势，让对手产生「这人疯了」的恐惧
- 「毒血反杀」——体内藏毒，以示弱姿态让对手被毒血溅到
- 「同归于尽谈判」——「放我走，否则我引爆/自爆，大家一起死」

## 本章原文锚点
- 「他快死了……谁还敢再和他拼命？」
- 「四个外功圆满老捕快为什么不出手？……宁愿坐视同僚重伤身死，也要躲在最后面」
- 「耐心等一等，等他死了就能安安稳稳去领赏」
- 「烂船还有三斤钉，这个时候要是再上去拼，万一对方临死前把自己带走了」
