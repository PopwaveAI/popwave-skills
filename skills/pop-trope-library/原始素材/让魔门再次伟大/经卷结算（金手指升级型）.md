# 经卷结算（金手指升级型）

## 基本信息
- **名称**: 经卷结算（金手指升级型）
- **类型**: 金手指进化型套路
- **首次出现章节**: ch033
- **嵌套型**: 是——可与「死后结算」「金手指觉醒」等其他金手指套路叠加使用

## 触发条件
主角持有可积累/结算型金手指 → 经历重要事件（战斗、交易、抉择、牺牲等）→ 金手指自动触发结算 → 根据结算行为生成对应效果的新能力/新卷 → 新能力与主角当前困境形成互补 → 主角快速推演新能力的最优用法 → 形成明确的战术/战略预期

## 套路公式
```
金手指持有（已有底层框架）→
经历重大事件（完成结算条件）→
金手指自动结算（画面回放关键节点）→
能力生成（根据事件性质生成对应新卷/模块）→
能力解读（主角+读者同步理解效果）→
用法推演（逐项排除→锁定最优解）→
战略定位（新能力在长期战术中的作用定位）
```

## 核心张力
1. **积累感**：金手指不是一次性给完的——而是不断通过主角的行动积累新的模块/卷轴
2. **事件映射**：生成的能力不是随机——而是根据结算事件的性质（舍身→一诺千金，杀人→杀伐卷等）
3. **认知同步**：新能力揭秘时，读者和主角同步理解效果——不靠旁白解释
4. **推演爽**：主角不是被动接受——而是快速理解并推演最优用法，展示智力
5. **留白钩**：推演时保留可能性——「说不定还能……」，为后续埋钩

## 情绪收益
结算仪式感 + 新能力揭秘的惊喜 + 用法推演的智力展示 + 战略定位后的目标感明确 + 「还能这么用」的认知爽

## 运作机制
1. **结算条件**：需要主角通过特定行为（舍身、诛杀、交易等）来「触发」结算——事件越重大，结算结果越强
2. **结算流程**：画面回放(事件回顾)→文字凝聚(能力命名)→效果说明(能力机制)
3. **推演环节**：必须保留——主角对新能力的思考链条是读者智力参与的通道
4. **能力定位**：新能力必须与当前困境或目标有明确关联——不能是和当前无关的随机能力
5. **进化保留**：旧能力不会消失，新的模块叠加在旧框架之上——积少成多的成长感

## 使用注意
- **不能千篇一律**：每次结算的流程可以相似，但能力效果必须独特且有趣——避免读者审美疲劳
- **不能太强**：新能力需要有明确的边界或代价——一诺千金的「限时完成」和「人死债消」就是限制
- **推演不能废话**：用法推演要简练——王平在一个自然段内就完成了全部分析，不能拖沓
- **画面回放要有意义**：太平经的回放不是废戏——苏夫人的获救画面证明了王平的牺牲没有白费，也有安抚读者的作用
