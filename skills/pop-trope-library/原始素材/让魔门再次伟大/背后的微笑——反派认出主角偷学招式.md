# 背后的微笑——反派认出主角偷学招式

**套路类型**：侧面认可 / 反派视角塑造
**首次出现**：ch021
**关联套路**：实力展示收徒、反派内心戏

## 四段式拆解

```
【起】反派/高人勘查现场，从战斗痕迹精确推演出战斗经过，展现其眼界和经验。
【承】当发现主角使用了自己独门招式的痕迹时，立刻辨识出来。
【转】不但不怒，反而啧啧称奇——欣赏主角的天赋和悟性。
【合】从「损失惋惜」心态转为「这倒也是一次机会」，将主角纳入自己的棋局视角。
```

## 核心效果
- 侧面烘托主角天赋之高（只见过一次就能模仿）
- 塑造反派「惜才但冷酷」的立体形象
- 为后续「反派盯上主角」埋下伏笔

## 复用条件
- 战斗现场有独门招式的痕迹可辨
- 反派有足够的眼力（非菜鸡反派）
- 主角确实偷师了反派（或该招式来自反派的传承）

## 变体可能
- 正面角色发现，引发收徒/提携的念头
- 反派发现后立即追杀（不留后患型）
- 多个高人同时从不同痕迹看出不同信息

- 「我只在他面前出手了一次，仅仅那一次，他竟就有所领悟，甚至现学现用……」
- 「而且……这也是一次机会。」
