# 自省型莽夫（藏智于拙型）

## 基本信息
- **名称**: 自省型莽夫（藏智于拙型）
- **类型**: 人设揭示型套路
- **首次出现章节**: ch033
- **嵌套型**: 否（独立使用）

## 触发条件
主角在前文一直以「莽夫」「头铁」形象示人 → 经历重大损失/失败后进入反思状态 → 主角主动（通过内心独白）揭示自己的真实内核——莽只是伪装，核心是精于算计 → 通过列举前期「莽」中的算计细节来证明 → 人物形象从「单纯莽夫」升级为「有策略的表演型莽夫」

## 套路公式
```
积累期莽夫形象（前文一直表现得很冲动）→
重大失败/损失触发反思（兵器全毁/技能冷却）→
自我定性拆穿（「莽夫只是伪装」→核心是讲章法）→
行为逻辑复盘（历次「莽夫」行为的真实算计）→
人物形象升级（从平面到立体——莽是策略，智是核心）
```

## 核心张力
1. **人设反差**：前文积累的「莽夫」印象被一句自白拆穿——读者的认知被刷新
2. **人物深度**：不是作者旁白介绍——而是角色自己揭示自己的本质，可信度和冲击力更强
3. **行为回顾验证**：读者会回想前文——就会发现王平的莽真的都是「有算计的莽」
4. **区分度**：和其他「扮猪吃虎」套路不同——这里的主角不是装弱，而是装莽（装蠢的变体）

## 关键对白/原文证据
- 🔒「思来想去，王平还是决定稳一手。毕竟他可不是那种头铁的人，莽夫只是伪装，很多时候他的莽其实是很讲章法的，无脑莽那是找死。」
- 🔒「君子报仇，十年不晚。」
- 🔒（前文历次「莽」的行为均可作为验证——比如舍身闯城不是真的送死，而是有复活的后盾；比如单骑破军不是硬莽，而是算计好每步的代价和收益）

## 情绪收益
人设反转的惊喜 + 「原来如此」的恍然大悟 + 人物形象立体的好感度提升 + 对前文行为的重新解读的快感

## 运作机制
1. **前置铺垫**：前文必须有足够的「莽夫行为」积累——否则自省就没有冲击力
2. **自省时机**：必须在重大损失/失败后——此时反思最有说服力，也最符合角色心理
3. **定性明确**：一句明确的自省台词是关键——「莽夫只是伪装」是全文定性句
4. **后置验证**：自省后的主角行为应当表现出更明显的「算计」特征——不能继续无脑莽
5. **不是洗白**：不是把角色从「坏」变成「好」——而是从「平面」变成「立体」

## 使用注意
- **不能太早揭示**：需要足够的莽夫行为积累——太早揭示就失去了反差
- **不能只揭示不验证**：自省后的行为需要表现出算计——如果继续无脑莽，这个揭示就成了废话
- **语气要自然**：太做作的自我揭密会显得刻意——王平是在冷静评估战局时顺带揭示的，不突兀
- **不是全知全能**：揭示的是「核心是讲章法」——不等于主角算无遗策，他也会犯错，只是犯错的原因不是莽而是信息不足
