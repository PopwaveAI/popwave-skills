# 过渡加速（计划急转法）

## 基本信息
- **名称**: 过渡加速（计划急转法）
- **类型**: 节奏控制型套路
- **首次出现章节**: ch016
- **嵌套型**: 是

## 触发条件
主角正在松弛/日常状态中 → 突发外部变量打破原有计划 → 主角瞬间理解局势 → 必须将原定计划提前执行

## 套路公式
```
松弛状态（完成阶段性目标 + 展望未来）→
突发消息（第三方介入 / 敌方提前行动）→
主角理解（瞬间洞察局势——计划不得不加速）→
决策（无选择，立刻行动）→
悬念拉起（任务提前执行，结果未知）
```

## 核心张力
「计划赶不上变化」的叙事逻辑——原本有充足准备时间（ch014的「三天」），但因为外部变量的干扰不得不提前行动。这种加速的逻辑必须合理（如ch016：守冲急了→亲自入城→陈浩彦被迫提前计划→王平的学习准备归零），而不是为了加速而加速。

## 关键对白/原文证据
- 🔒 ch014：「三天.....那还不是九死一生！」——原计划时间
- 🔒 ch016：「就在一刻钟前，白莲教主守冲强闯县城，目前已在城内失去了踪迹。」
- 🔒 ch016：「陈大人要见您！」
- 🔒 ch016（内心）：「守冲那老毕登突然现身……计划赶不上变化。」

## 情绪收益
从松弛到紧张的急转弯刺激感 + 对提前任务的担忧/期待 + 对主角应对能力的信任感考验 + 「原以为有三天，结果现在就要上」的荒诞压迫感

## 运作机制
1. 前章建立「有充足准备时间」的预期（ch014：三天学习穿云震天弓）
2. 本章前段展示主角的松弛/进展状态（ch016前段：练成血如火→逛草市→金手指升级→畅想新挂）
3. 突发消息打断（ch016章末：守冲入城）
4. 主角瞬间理解局势（ch016：王平立刻明白陈浩彦等不下去了）
5. 任务提前执行——悬念拉起（ch016结尾：王平准备立刻押送莲华刀）

## 使用注意
- 突发变量需要在之前有伏笔（ch014已埋下「守冲急了」的伏笔 + ch015的飞刀气运暴跌）
- 主角对局势的理解速度体现其智商（ch016中王平秒懂「计划赶不上变化」）
- 加速后的任务不能太容易通过——否则「加速」没有意义
- 建议与前章的「时间压迫」套路联动（ch014的三天 vs ch016的立即行动）
