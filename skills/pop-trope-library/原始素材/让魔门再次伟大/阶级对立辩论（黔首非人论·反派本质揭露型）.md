# 阶级对立辩论（黔首非人论·反派本质揭露型）

**套路类型**：阶级批判型 / 反派本质揭露
**首次出现**：ch027
**关联套路**：黑暗真相·系统揭秘、阶级跃升·反向坠落

## 四段式拆解

```
【起】主角与地位更高的反派（官员/贵族/宗门长老）产生正面冲突，
     反派占据绝对武力与权力优势，但试图用"大义"来说服/招揽主角。
【承】主角不接受反派的宏大叙事，用具体事实反问——
     「造福苍生？哪个苍生？」"城外灾民遍地"与"天下苍生"的割裂被点破。
【转】反派的反应并非愤怒，而是【真心困惑】——
     他发自内心认为底层百姓不是人，是"前朝余孽"、"杂草"、
     "产粮的黔首"——这不是恶意的侮辱，而是一套完整自洽的世界观。
【合】反派反而责怪主角被"邪教蛊惑"，认为"将黔首当作人来看待，
     反而是对人的亵渎"——用一套扭曲但自洽的逻辑完成了"恶的合理化"。
```

## 核心效果
- 让反派的"恶"超越简单的"我很坏"——给出阶级本位的世界观根基
- 主角在武力弱势下完成道德/逻辑层面的压制——「你错了，但我打不过你」的张力
- 读者获得"看穿系统本质"的认知爽感——反派不是奇葩，而是系统的代言人

## 复用条件
- 故事存在明确的阶级压迫设定
- 反派处于统治阶级/优势阶层
- 主角有底层出身或底层认同
- 需要有直接的对质场景——不能是内心独白，必须当面碰撞

## 变体可能
- **宗门版**：宗门长老将散修/凡人视为"修炼材料"——以"天道不仁"为辩护
- **世家版**：世家家主将寒门视为"血脉杂质"——以"血统纯正论"为辩护
- **神权版**：神职人员将异教徒视为"该被净化"——以"神谕"为辩护
