# 扮猪吃老虎（隐藏实力·认知差碾压）

> **这不是一个特定套路的名称。这是一个抽象模式家族，有N种具体实现。**
> 
> **核心理念**：读者知道主角很强（或很强但没展示），但故事内角色不知道 →
> 展示时机到来时 → 故事内角色震惊 → 读者获得"早就知道你们不知道"的智力优越感爽。
> 
> **为什么爽**：爽感不是来自"主角很强"本身——而是来自**信息差释放的瞬间**。
> 读者已经等了许久"他们什么时候发现主角其实很强"——释放那一刻的落差就是爽感来源。

## 心理学原理

| 元素 | 说明 |
|:-----|:------|
| **信息差** | 读者知道主角真面目 → 故事内角色不知道 → 读者拥有"上帝视角"认知优势 |
| **延迟满足** | 越晚释放反差，释放时的爽感越大（但必须在读者耐心耗尽前释放） |
| **社会地位反转** | 低地位者→高地位者；被轻视者→被敬畏者；低位者的逆袭是人类最基础的爽感模式 |
| **身份认同** | 读者通过"我早就知道他很厉害"的预判获得智力优越感，强化与主角的同盟感 |

## 通用公式

```
前置铺垫期（隐藏实力）
  → 低地位定位（被轻视/被低估/被嘲笑/被无视）
  → 关键时刻（必须展示实力的危机/挑战/赌约）
  → 实力展示（碾压级的、视觉化的、无可辩驳的展示）
  → 旁观者反应（震惊→重新评估→敬畏/恐惧/追随）
  → 后续效应（地位改变/关系重构/权力格局变化）
```

## 变体家族

### 变体A：装弱钓鱼型
**核心**：主角主动装弱，引诱对方上钩。

| 要素 | 说明 |
|:-----|:------|
| 主角状态 | 主动隐藏实力，故意示弱 |
| 对方状态 | 以为主角是肥羊，主动挑衅/攻击 |
| 展示方式 | 在对方最自信的时刻瞬间逆转 |
| 情绪收益 | "你上当了"的智商碾压爽感 |
| 案例 | 让魔门_实力展示收徒（钓鱼执法式装逼） |

### 变体B：被动暴露型
**核心**：主角没有主动装——是其他人低估了主角，然后在危机中被迫展示。

| 要素 | 说明 |
|:-----|:------|
| 主角状态 | 被动隐藏（没人问、没人关心、不需要展示） |
| 对方状态 | 基于表面信息低估主角 |
| 展示方式 | 突发事件（战斗/危机）迫使主角出手 |
| 情绪收益 | "你们根本不知道他是谁"的颠覆感 |
| 案例 | 深渊主宰_以赌约展示实力（三次攻击不后退心理控制版） |

### 变体C：第三方揭示型
**核心**：主角自己不展示——是第三方通过评价/数据/战绩来揭示。

| 要素 | 说明 |
|:-----|:------|
| 主角状态 | 不在场或不在意 |
| 对方状态 | 从怀疑→听到第三方评价→震惊 |
| 展示方式 | 不展示动作——展示"别人怎么看他" |
| 情绪收益 | "原来他这么强？我居然不知道"的间接震撼 |
| 案例 | 深渊主宰_第三方评价揭示主角、让魔门_名号反转（谐正为俗） |

### 变体D：日常化展示型
**核心**：主角展示实力但自己觉得这很普通——通过"不自知的强"来制造反差。

| 要素 | 说明 |
|:-----|:------|
| 主角状态 | 做了很厉害的事但觉得"这没什么" |
| 对方状态 | 看呆了/世界观被刷新 |
| 展示方式 | 日常动作（随手一挥/不经意的一步）却引发了夸张效果 |
| 情绪收益 | "你觉得很难？这不是基本功吗？"的黑色幽默 |
| 案例 | 深渊主宰_拎头咆哮·斩首威吓（展示后轻松收尾） |

## 使用红线

### ✅ 必须做
- **展示必须是"碾压级"的**：不是"险胜"——是"对方全力攻击但主角动都没动"——必须有明确的实力鸿沟
- **展示的视觉化**：不能靠旁白说"他很厉害"——要靠旁观者的反应（表情变化/冷汗/武器落地/集体后退）
- **展示前必须铺够"被低估"的戏份**：没有充分的前置被轻视，反转就没有力度
- **展示后必须改变格局**：展示后主角的地位必须改变——不能明天又回到受气的状态

### ❌ 不要做
- **不要让主角主动炫耀**："看到了吗？我很强"——low了。实力展示后的轻描淡写/无所谓才是高级
- **不要让展示变成"剧情需要"**：展示必须对剧情有实际推动（收了新小弟/震慑了对手/拿到什么资源）
- **不要频繁使用同一变体**：每卷最多用一次——第二次读者就知道了
- **不要让读者等太久**：前置铺垫期超过一卷读者会失去耐心

## 节奏控制

| 篇幅 | 前置期长度 | 展示密度 | 适合位置 |
|:-----|:----------|:---------|:---------|
| 单章 | 同一章前1/3装弱 → 中段展示 | 快速反转 | 最差方案——铺垫不足，反转廉价 |
| 3-5章 | 跨章铺"被低估"细节 → 第4-5章展示 | 中速 | 开篇/新地图出场 |
| 跨幕 | 整幕都在被低估 → 幕高潮处展示 | 慢速但高冲击 | 身份转换节点 |
| 整卷 | 全卷被低估 → 卷末终极展示 | 极限延迟、最高爽感 | 全书的"身份反转"节点 |

## 套路链推荐

```
装弱钓鱼（变体A）→ 地位提升 → 新的更高层级角色出现
                                          ↓
                                    又被低估 → 变体B或C再次展示
```

或：

```
第三方揭示（变体C）→ 引起高层注意 → 新挑战者出现
                                          ↓
                                    日常化展示（变体D）→ 彻底建立地位
```

## 跨书案例对比

| 条目 | 让魔门（实力展示收徒） | 深渊主宰（以赌约展示实力） |
|:-----|:---------------------|:-------------------------|
| 前置类型 | 主角被误认为普通人 | 主角被质疑能力 |
| 展示方式 | 直接出手碾压 | 设立赌约后展示 |
| 旁观者反应 | 震惊→拜师 | 震惊→信服→臣服 |
| 后续效应 | 收徒成功 | 从属关系建立 |
| 核心差异 | 更偏"收服"——展示后获得追随者 | 更偏"威慑"——展示后获得敬畏 |
