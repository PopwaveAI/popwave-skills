## 四、兼容层说明

### 文风路径优先级
```
1. 文风库/{书名}.md          ← canonical（新写入目标）
2. 文风DNA/{书名}.md          ← fallback（现有档案，保持只读）
3. 写作资产/文风DNA/{书名}.md  ← 项目内临时路径
4. styles/{书名}.md           ← 历史路径，不鼓励新逻辑
```

### 设定库历史路径
`设定库/框架/` `设定库/质感/` 中如有遗留文件，**标记为 legacy**，不删除但不在新规范中引用。框架/质感现在是书内维度，不是目录模型。

### 剧情库历史切片
`剧情库/_LEGACY/` 中的卷/幕/章级切片仅作为拆书证据保留，**不作为消费入口**。plot 必须读标准剧情线（`剧情库/{标签}/*.md`）。

## 六、入库快速路由

agent 入库时，按此表定位目标模块：

| 资产类型 | 写入路径 | 索引更新 |
|:---------|:---------|:---------|
| PRD 级经验 | `立项库/{子目录}/{条目}.md` | `立项库/00-索引.md` |
| 整书 L1 设定 | `设定库/{书名}/`（PRD.md + L1-01~06 + 世界宪法） | `设定库/00-索引.md` |
| 文风档案（新） | `文风库/{书名}.md` | `文风库/00-索引.md` |
| 文风档案（兼容） | `文风DNA/{书名}.md` | 保留 |
