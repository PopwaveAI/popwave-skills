# 立项库 · 索引

> 定位：PRD 级知识库，服务 creative / pop-novel。这里沉淀“书怎么立”的经验，而不是具体桥段、设定或文风。

## 目录规划

| 目录 | 用途 | 状态 |
|:-----|:-----|:-----|
| `立项模式/` | 稳定书型与 PRD 架构模式 | 待建设 |
| `元爽点组合/` | 主爽点组合与互斥关系 | 待建设 |
| `题材焊接/` | 跨题材、跨 IP、跨文化素材如何融合 | 待建设 |
| `加工哲学/` | 借机制、借质感、借节奏、借情绪的边界 | 待建设 |
| `失败案例/` | 下游 world / plot / prose 证明难施工的 PRD 案例 | 待建设 |

## 主消费方

| skill | 怎么用 |
|:------|:-------|
| `pop-writer-creative` | 生成故事概念与推导 PRD 前，查立项模式与元爽点组合 |
| `pop-novel` | 开新书时作为 creative 前置知识库 |
| `pop-writer-world` | 当 world 阶段发现 PRD 约束互斥时，回查立项库判断是否为立项层问题 |
| `pop-writer-plot` | 当爽点组合难以持续时，回查元爽点组合与失败案例 |

## 条目建议

```yaml
---
source_type: prd_pattern
track: 修仙
meta_joy: 信息差
stage: creative
status: trial
---
```

## 边界

| 不放这里 | 应放到 |
|:---------|:-------|
| 具体桥段公式 | `套路库/` |
| 长期剧情线 | `剧情库/` |
| 世界规则与 L1 设定 | `设定库/` |
| 文风场景卡 | `文风库/` |
