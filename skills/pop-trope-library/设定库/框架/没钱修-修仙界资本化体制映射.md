# 修仙界资本化体制映射（金钱即力量的完整体制设计）

- **源域**：现代社会阶层固化理论 + 消费社会批判 + 赛博朋克（消费即力量）
- **转化逻辑**：将现实社会阶层困境完整映射到修仙世界观，并使其「物理化」——金钱直接等于力量（法力=钱，灵根=钱，筑基也要钱），阶级差异变成物理屏障（54层空间金字塔，越往上灵气越浓权力越大）。修仙不是能力解放，而是「阶级再生产」——穷人不是在爬梯子，而是在被层层设计的筛子里挣扎。核心命题：「钱不是自由，而是更精致的牢笼」。整个体制围绕「金钱即力量」的铁律运转，每个子系统（空气收费、楼层阶级、医保擂台、灵根租赁）都在强化这一原则。
- **适用场景**：需要在爽文框架中加入社会批判深度的作品；现代阶级困境→奇幻世界观的系统性映射；主角终极对手不是反派而是体制
- **赛道归属**：修仙·校园
- **来源书**：没钱修什么仙
- **核心机制**：金钱=力量（法力/灵根/筑基皆需钱）→ 54层空间金字塔（物理化阶级）→ 阶级再生产（筛子而非梯子）→ 每个子系统强化金钱铁律 → 终极反转：「钱不是自由，而是更精致的牢笼」
