# 修仙金融化·法力贷与借贷体系

- **源域**：现代金融体系（信用贷款/抵押贷款/复利/信贷评级）+ 校园贷现实案例 + 网络流行语「血赚」文化 + 赛博朋克（身体抵押/零件库）
- **转化逻辑**：将修仙时代的「力量=钱」原则系统化为完整的金融借贷体系。法力贷（学费+生活费+修炼费打包贷款，70万起步利息无限滚）是基础产品——穿越第一天就欠70万法力的债，利率按日计息无限复利滚存。法力贷不是选修题：不上学要还贷，上学更要贷。穷人不是在起跑线上落后——是起跑线上就欠债。贷款贯穿全书的每个关键节点：面试要贷、买功法要贷、考筑基证要贷——贷款额度直接等于「仙道潜力」，穷人的「潜力上限」由贷款额度定义。食物贷垄断底层饭碗（分期吃饭=人人负债）；养育贷让生育变成投资（长大了连本带利还）；抵押血肉以身体部位还款（活人剥皮当装备）。整个体系让主角的每一次升级都伴随着「还能还上吗」的紧张感——贷款是踩在脚下的定时炸弹。
- **适用场景**：需要现代制度讽刺+资本主义批判+赛博修仙混搭的修仙/都市/科幻作品；「贫穷」从背景设定升级为持续性叙事引擎
- **赛道归属**：修仙·校园
- **来源书**：没钱修什么仙
- **核心机制**：法力贷（教育贷）→ 食物贷（生存贷）→ 养育贷（人身贷）→ 抵押血肉（身体贷）→ 仙道潜力=信用额度（贷款上限=上升通道）→ 以贷养贷无限循环 → 贷款贯穿每个叙事节点（面试/功法/筑基证）
