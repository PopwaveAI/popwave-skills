# 分数种姓与道心估值体系

- **源域**：现代考试成绩排名文化 + 信用评分系统（芝麻信用/征信）+ 阶级种姓制度 + 手游评分/段位系统
- **转化逻辑**：将考试成绩和修行数值直接编码为阶级身份——「550分以下是NPC」（分数决定人权等级，这不是脏话而是昆墟的基本分类法）。肉体强度排行榜、天眼符评价系统、仙道潜力评分全链条覆盖，将人的价值全面数据化。道心不仅是修炼参数还是自我认知系统，道心等级决定精神承受力（力量超过道心等级→走火入魔）。分数/排名/信用评级三种数据系统共同构成「数据化阶级」——穷人的价值被精确量化并封印在一定阈值内。富二代不需要亲自霸凌——制度已经替他们把穷人踩在脚下（何大有特权调对阵、学生会压迫、富人优越感，全部围绕分数种姓运转）。主角的反抗从「打败某个恶霸」升格为「质疑整个评价体系」。
- **适用场景**：需要社会批判+数据化阶级+校园竞争残酷性的修仙/都市/科幻作品；社会达尔文主义在奇幻世界的制度化体现
- **赛道归属**：修仙·校园
- **来源书**：没钱修什么仙
- **核心机制**：分数种姓（考试排名=人权等级）→ 肉体强度排行榜（战力数据化）→ 天眼符评价系统（大数据监控）→ 道心等级（精神承载力上限）→ 信用评分（贷款上限=上升通道）→ 制度本身即是施暴者（无需具体恶霸）
