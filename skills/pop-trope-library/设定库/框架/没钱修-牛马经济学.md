# 牛马经济学（内卷/996的修仙化表达）

- **源域**：现代996工作制 + 内卷文化 + 打工人困境 + 下沉市场经济
- **转化逻辑**：将996/内卷文化移植为「牛马经济学」体系——专为底层打工者设计的「牛马心法」（性价比最高的功法路线，但也意味着最大程度的消耗），保险公司付费型擂台（医保额度决定战斗续航），暗学帮地下补课市场（穷人用信息差在黑市翻身）。核心反转：牛马功法反而比高端功法更赚钱（下沉市场的大班课一节课几十人）——但当你赚到钱了才发现，你只是从底层的牛马变成了上层的牛马。
- **适用场景**：内卷/996/打工人现实的奇幻化表达；底层修炼体系「性价比困局」设计；黑色幽默式社会批判
- **赛道归属**：修仙·校园
- **来源书**：没钱修什么仙
- **核心机制**：牛马心法（性价比最高但消耗最大）→ 医保付费擂台（医保额度=战斗续航）→ 暗学帮地下补课（信息差黑市）→ 下沉市场大班课盈利模式 → 阶级跃迁后仍是牛马的反转
