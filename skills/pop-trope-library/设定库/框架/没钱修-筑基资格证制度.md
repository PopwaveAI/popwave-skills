# 筑基资格证制度

- **源域**：现代教育制度（学历证书/资格考试/统考）+ 职业资格认证（执照/准入制）+ 公司入职（offer/试用期）+ 高考焦虑与学历歧视
- **转化逻辑**：将修仙「筑基」从个人修为突破转化为需经资格考试的制度化认证——筑基证类似大学毕业证+职业资格证+贷款信用评级的复合体。无证筑基属违法犯罪。考试分三关（神灵根测试+法骸适应性+缚龙索拔河极限对决），唯一由十大宗门和正神系统认证。获证奖励包括300万贷款+专家功法+大学招生资格。筑基后可付费开通「筑基领域」（百米范围，提示周围人注意礼貌）。大学等级即人生等级（十大→36上校→72下校→大专→杂牌），分数500以下被视为NPC。考的分数越高，能借的贷越多——整个考试系统不是能力筛选，而是「信用评级跃升」机制。核心反转：拿到筑基证意味着可以借更多的钱，而不是不再需要钱——「通行证的另一端是屠宰场」。
- **适用场景**：需要教育批判+制度讽刺+赛博修仙氛围的作品；将现实教育焦虑投射到奇幻世界的社会讽刺
- **赛道归属**：修仙·校园
- **来源书**：没钱修什么仙
- **核心机制**：筑基考试（三关制）→ 资格证（唯一认证）→ 附带奖励（贷款/功法/大学）→ 大学等级=人生等级 → 筑基领域（付费功能）→ 无证筑基=犯罪 → 考试=信用评级跃升 → 反转：通行证另一端是屠宰场
