# 阶级物理化设定（54层空间金字塔阶级编码）

- **源域**：现代阶级分层理论 + 赛博朋克（楼层即阶级）+ 空气经济学 + 医疗分级制度
- **转化逻辑**：将阶级从社会经济概念转化为物理世界法则——昆墟54层空间金字塔编码54个阶级，越往上灵气越浓、权力越大，穷人和富人生活在不同的物理维度。空气分三六九等（灵机保护区门票制，空气1999元/立方米），医疗分阶级（医保额度决定擂台续航时间），学校是阶级预制舱（嵩阳高中十几年无平民考十大）。一楼是穷人的牢笼，二楼以上需考试准入。阶级不是隐形的——是被刻在空气里、楼层里、医保额度里的物理事实。
- **适用场景**：需要让「阶级压迫」可视化、可感化的黑暗奇幻/赛博修仙作品；主角的每一次上升也是物理空间跨越的叙事
- **赛道归属**：修仙·校园
- **来源书**：没钱修什么仙
- **核心机制**：54层空间金字塔（楼层=阶级）→ 空气分级收费（灵机保护区门票制）→ 医保额度决定战斗续航 → 学校作为阶级预制舱 → 物理空间准入制（二楼以上需考试）
