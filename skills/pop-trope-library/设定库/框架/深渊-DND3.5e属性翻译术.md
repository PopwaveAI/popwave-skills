# DND 3.5e 六属性 → 西幻力量体系翻译术（外功六要素转化）

- **源域**：DND 3.5e Player's Handbook 六属性体系（力量/敏捷/体质/智力/感知/魅力）+ 专长系统 + 武侠横练概念（金钟罩铁布衫）
- **转化逻辑**：完整沿用 DND 六属性框架作为人物数据面板，但做了「网文化翻译」——将 DND 的专业数值规则（属性值范围、属性修正、DMG 判定规则）转化为可直接嵌入叙事正文的数据面板，让读者直观看到数字变化而非骰子判定过程。同时提取「属性维度化」哲学，将武侠横练「金钟罩铁布衫」的笼统概念拆解为**外功六要素**（皮肉筋骨血髓），每条修炼维度独立量化，配以四品制（普通/超凡/传奇/神器）品级划分。保留DND的「属性→数据面板」可视化特征，但完全替换为东方武侠修炼的叙事语言。专长体系（普通→专家→大师→宗师→传奇五阶）直接引用，职业阶位简化为五阶划分（一阶1-3级→五阶13+传奇）。
- **适用场景**：西幻/游戏异界类作品的底层力量体系设计；系统流/数据流的东方本土化改造；需要「游戏化数据面板」但不想破坏叙事沉浸感的网文
- **赛道归属**：西幻
- **来源书**：深渊主宰
- **核心机制**：六属性作为人物基础框架 → 外功六要素（皮肉筋骨血髓）量化横练 → 四品制（普通/超凡/传奇/神器）品级划分 → 职业体系提供能力路线 → 专长系统五阶细化战斗差异化 → 职业阶位五阶划分提供成长阶梯 → 挑战等级（CR）量化敌我差距
