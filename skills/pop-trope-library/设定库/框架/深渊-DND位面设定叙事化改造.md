# DND位面设定叙事化改造（位面即叙事剧场）

- **源域**：DND被遗忘的国度位面体系（阴影位面/星界/无底深渊/物质位面）+ 克苏鲁式不可名状恐惧
- **转化逻辑**：完整沿用DND位面概念作为空间设定骨架（阴影位面、星界、无底深渊、物质位面结构），但将每个位面改造为承载特定叙事功能的舞台——阴影位面→精神压抑/未知恐惧的探索场所；无底深渊→强大魔神的经营据点和恐惧来源；星界→跨位面冒险的通道。位面不再只是「地图上的格子」，而是角色心理和冲突的投射空间。通过克苏鲁式的「不可名状恐惧」叙事、政治博弈的「神国经营」等手法，让位面从地图升格为叙事空间。
- **适用场景**：从跑团/游戏设定改编为小说时的规则工具→叙事空间升级；需要多世界/多位面架构的奇幻或科幻作品
- **赛道归属**：西幻
- **来源书**：深渊主宰
- **核心机制**：DND位面骨架完整沿用 → 每位面承载特定叙事功能（压抑/恐惧/冒险） → 克苏鲁氛围增强 → 神国经营政治博弈 → 位面从地图升格为叙事剧场
