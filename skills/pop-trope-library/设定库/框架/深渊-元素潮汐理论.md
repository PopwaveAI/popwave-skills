# 元素潮汐理论

- **源域**：DND 3.5e 魔法波动概念 + 作者原创
- **转化逻辑**：将 DND 中模糊的「魔法波动」或「魔网」概念转化为具象的「元素潮汐」理论——元素像海水一样发生潮汐运动，涨潮退潮影响元素能量结构。完全依赖天赋的物种（如龙族）最易受影响：瑟银时代到奥术帝国时期龙族衰退（挑战等级普遍降低2-5级），奥术帝国后龙族复苏。时间跨度数千年，为世界历史提供了自然波动型的底层动力机制。
- **适用场景**：需要解释历史上不同时期战力标准变化、为背景提供自然底层动力的西幻作品
- **赛道归属**：西幻/通用
- **来源书**：深渊主宰
- **核心机制**：元素潮汐周期性波动 → 影响依赖元素天赋的物种能力 → 解释时代间力量差异 → 暗示当前世界元素状态正发生变化（为圣者浩劫铺垫）
