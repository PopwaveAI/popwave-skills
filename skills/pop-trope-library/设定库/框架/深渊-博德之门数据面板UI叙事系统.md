# 博德之门数据面板UI叙事系统

- **源域**：博德之门系列CRPG角色数据面板 + 网文系统流数据面板 + DND数值体系
- **转化逻辑**：借用博德之门式的数据面板可视化理念，将DND体系的数值（属性、职业阶位、挑战等级、经验值）转化为小说中的叙事「仪表盘」。关键创新点：经验值不仅是升级资源，还作为「鉴定费/扫描费」被消耗——增加了经济的流动性和决策维度。称号系统也从被动展示变为主动叙事工具（称号跃迁机制）。数据面板从静态展示升级为动态叙事引擎——读者不仅看数字涨，还要看主角如何分配资源、选择称号。
- **适用场景**：系统流/数据流网文的数值体系创新；需要让面板数据「活起来」、参与剧情决策的作品；CRPG游戏体验转化为小说叙事
- **赛道归属**：西幻
- **来源书**：深渊主宰
- **核心机制**：博德之门式数据面板可视化 → 经验值多出口消耗（鉴定费/扫描费）→ 称号跃迁机制（主动叙事工具）→ 动态叙事引擎（资源分配决策）
