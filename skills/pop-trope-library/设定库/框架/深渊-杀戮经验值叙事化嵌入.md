# 经验值 → 升级的叙事化嵌入

- **源域**：DND 3.5e 经验值规则 + 博德之门系列·经验值货币化 + 网文系统流（打怪升级）
- **转化逻辑**：将 DND 的经验值规则转化为「杀戮经验」——唯一获取途径为杀戮，彻底去除 DND 中任务/探索等非战斗经验来源。经验值不仅是升级资源，更具突破性的设计是「多出口经验值经济」：经验还可用于扫描鉴定目标属性（消耗）、解锁物品信息（鉴定费）、神性赐予机制（作为投资资本）。将网文系统流的数据面板呈现方式（眼中跳出 0 和 1 的数值化描写）与 DND 的底层数值框架融合，形成「游戏化叙事」风格。
- **适用场景**：需要数值化成长体系+资源管理的西幻/游戏异界/升级流作品
- **赛道归属**：西幻
- **来源书**：深渊主宰
- **核心机制**：杀戮获取经验（唯一途径）→ 多出口消耗（升级/扫描/鉴定/神性赐予）→ 积累到阈值释放（等级连跳+专长解锁）→ 神性赐予+经验共享（将神性视为可投资资本）
