# 神术与信仰系统设计

- **源域**：DND 3.5e 神术体系 + 被遗忘的国度·神灵谱系 + 冰雪奇缘·阿伦黛尔命名
- **转化逻辑**：借用 DND 中牧师/神术施法者体系，将神术能力与神灵信仰绑定——牧师通过信仰获得神术而非法术（巫师体系）。神灵按神力等级分微弱/弱等/中等/强大四等，不同等级的神灵提供不同层级的神术。神殿需维持财富以运转法阵和神赐奖励，引入「神殿财政逻辑」——大神殿 vs 小神殿有等级差。神术稀缺且昂贵（高阶牧师极少），强化了宗教势力和世俗权力的交织关系。
- **适用场景**：需要多神系、神权政治、宗教冲突的西幻世界构筑
- **赛道归属**：西幻
- **来源书**：深渊主宰
- **核心机制**：神灵等级（微弱→强大）→ 信仰体系（神殿财政逻辑）→ 神术等级（牧师施法能力）→ 神性体系（选民/圣者/神子）→ 封神路线（跳板神职染指杀戮领域）
