# 兵器谱 × 敕名封号 × 名望转化神意（三位一体名望/名器系统）

> 来源：让魔门再次伟大 · L1-02力量体系 + L1-06资源与物品 + ch009/ch022/ch027/ch028设计包
> 拆书依据：全书最核心的名望/名器联动机制，分散在多个设计包中

## 核心闭环

```
武师打出名号 → 兵器进入兵器谱 → 积累名望气运
    ↓
兵器发生神异进化（名珍→神物）
    ↓
武师借助名望压力觉醒【神意】
    ↓
晋升【封号武师】
```

**这不是三个独立系统——是一条完整的闭环链条。** 兵器、名望、境界三者互相驱动。

## 一、名器系统：兵器五等

| 品级 | 定义 | 实例 | 原文 |
|:-----|:-----|:-----|:-----|
| 凡兵 | 普通兵器 | 牛尾刀 | "天下兵器，共有五等" (ch009) |
| 利器 | 精制兵器 | 玄螭弓（王平从武库获取） | ch015 |
| 名珍 | 有名号、参与大事件 | 素月莲华刀、神仙索、神引药 | "名珍是因为参与了某个大事件，继而流传出了名号的兵器" |
| 神物 | 创造神话、具备神异 | 莲华刀升华目标 | "神物，则是创造了神话" |
| 天工 | 传说级 | — | 全书最高，前37章无实例 |

**跨级碾压规则**：
> "哪怕是外功武者，若是手持一件【神物】，照样能杀内劲如屠狗" (ch031)

## 二、名望系统：兵器谱

兵器谱是大顺朝廷通行的天下名器认证体系——本质上是一份"名望排行榜"。

### 排位与敕名封号

| 原文 | 位置 |
|:-----|:-----|
| "位列【兵器谱】，被天下人熟知，积累了大量名望和气运的兵器，显然已经超出正常兵器的范畴了。它是具备神异的！" | ch027 |
| "【兵器谱】，第三百四十二位，【神仙索】徐秉正" | ch027 |

### 敕名封号的命名规则

**封号既夸赞武师的个人特征，也形容其兵器**：
> "这个封号一方面是夸赞他的轻功高绝，另一方面则是形容其兵器——西域天蚕丝织成的大网" (ch027)

### 名号反差（人设向）
> "王平期待自身外号是'斩夜刀''洗龙刀'等高逼格称号，实际获得'大龙枪'（因丹凤楼百人斩传闻得名）" (世界宪法)

## 三、神意体系：名望→力量的转化

**外功六要素修到圆满之后，上升通道不再是"炼体"，而是"炼名"。**

### 神意的诞生

> "神意源自承受天下名望的心灵重压，质变后反哺肉身，实现五官极致进化（听百米外蚂蚁爬行、危险前心血来潮、放慢世界）" (ch028)

> "封号武师的【神意】就是通过承受天下名望的压力蜕变而来" (ch028)

### 神意的六重进化

| 进化能力 | 表现 |
|:---------|:-----|
| 听觉极致 | 听百米外蚂蚁爬行 |
| 危险直觉 | 危险前心血来潮预警 |
| 时间感知 | 放慢世界的主观时间感 |
| 眼力超绝 | 分辨毫厘之间的破绽 |
| 气场压迫 | 未出手已压敌三分 |
| 名号震慑 | 报出封号=不战而屈人之兵 |

### 晋升门槛

> "至诚之道，可以前知"——成为封号武师的必要条件。不是力量门槛，是**心性门槛**。必须真正经受住"天下名望"的心理重压，不被虚名吞噬。

## 四、完整力量层级链

```
凡人（黔首/平民）
→ 练武入门（铁牛功等）
→ 外功六要素修炼期（皮如牛→肉如象→筋如龙→骨如虎→血如火→髓如霜）
→ 外功圆满（六要素全部大成）
→ 内劲武师（修功体图→开内劲）
→ ★封号武师（觉醒神意，兵器入谱）
→ 见神宗师（全书最高成就，前37章无实例）
→ 武神（传说级，创造神意体系的存在）
```

**平行体系**：
- 兵器：凡兵→利器→名珍→神物→天工
- 武功品级：下乘→中乘→上乘→绝世
- 功体图：天、地、人三等

## 五、太平经身份兑换体系

> "当前身份：流民（白身）""当前气运：一（白莲教堂主）""是否消耗气运，换取新身份？" (ch008)

**六阶递进**：
```
白身 → 皂吏 → 青士 → 朱服 → 紫卿 → 金爵
```

| 等级 | 标识 | 对应身份 | 实例 |
|:-----|:-----|:---------|:-----|
| 白身 | 白 | 平民/流民 | 王平初始身份 |
| 皂吏 | 黑 | 低级吏员 | 王平以气运换得捕快身份 |
| 青士 | 青 | 中级士人 | — |
| 朱服 | 红 | 高阶官员 | — |
| 紫卿 | 紫 | 朝廷大臣级 | — |
| 金爵 | 金 | 最高爵位 | — |

**机制**：消耗气运兑换身份。本质是系统流中的等级解锁变体，但升阶货币是**气运**而非经验值。
