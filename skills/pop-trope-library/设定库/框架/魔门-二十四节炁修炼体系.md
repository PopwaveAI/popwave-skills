# 二十四节炁修炼体系

- **源域**：传统丹道·二十四节气/二十四真（《黄庭经》）+ 中国传统历法 + 仙侠修真灵气设定
- **转化逻辑**：将中国传统历法的二十四节气神秘化为修仙的灵气循环体系——「天地有二十四炁，节制万象之变化」，青黄赤白黑五色轮转。传统丹道有「二十四气」「二十四真」概念但从未系统化，本作将之转化为可通过灵识感知的天地灵气底层循环系统。二十四节炁为修仙世界提供了独立于武道世界的灵气底层架构，是连接历法文化符号与修仙设定的桥梁。
- **适用场景**：需要东方传统文化深度嵌入修炼体系的修仙/仙侠世界构筑
- **赛道归属**：修仙
- **来源书**：让魔门再次伟大
- **核心机制**：二十四节气 → 二十四节炁（灵气循环系统）→ 五色轮转（青黄赤白黑）→ 灵识可感知 → 为废灵根旁门路径提供灵气底层
