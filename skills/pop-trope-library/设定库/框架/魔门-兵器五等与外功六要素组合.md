# 兵器五等 + 外功六要素组合体系

- **源域**：武侠传统（铁布衫/金钟罩/横练外功）+ DND 3.5e 六属性框架（STR/CON/DEX）+ 中国古典兵器等级传统
- **转化逻辑**：将传统武侠中「金钟罩铁布衫」一类笼统的横练概念系统化为六个独立可量化的修炼维度——皮肉筋骨血髓，每维分四品（如血：烈火/炽炎/真焱/至阳）。六要素之间无严格顺序约束，可并行修炼，每要素上限由资源和天赋决定。兵器五等平行存在：凡兵→利器→名珍→神物→天工。武功品级独立分类：下乘→中乘→上乘→绝世。三条平行体系（外功六要素/兵器五等/武功四品）构成完整的武道等级链。
- **适用场景**：需要量化升级感但保持东方武侠味的中式奇幻/武侠作品
- **赛道归属**：武侠/修仙
- **来源书**：让魔门再次伟大
- **核心机制**：外功六要素（皮肉筋骨血髓 × 四品/要素）→ 兵器五等（凡兵→天工）→ 武功四品（下乘→绝世）→ 功体图（天地人三等）→ 内劲/神意（封号武师专属领域技）
