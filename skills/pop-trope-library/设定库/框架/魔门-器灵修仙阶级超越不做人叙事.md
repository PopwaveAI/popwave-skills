# 器灵修仙·阶级超越叙事与「不做人」主题

- **源域**：传统仙侠器灵设定（被炼入法宝的灵魂）+ JOJO「我不做人了」梗 + 身体政治哲学
- **转化逻辑**：将传统器灵从「工具NPC」升级为修仙主体——主角肉身毁灭被炼为六欲天魔旗器灵，反而推演出器灵修仙路径：器灵之身就是宇宙法则的容器（「万化生乎身」），所有神通变化皆从器身生出。JOJO梗「我不做人了」被赋予全新含义：不做人（成为器灵）才能突破灵根限制，不做制度定义的人才能获得真正自由。一个梗同时完成喜剧效果、世界观突破、阶级超越三重功能。
- **适用场景**：需要「废柴逆袭」但不想走传统灵根觉醒老路；将搞笑梗升华为主题表达的进阶玩法；器灵/附身/夺舍等传统设定的颠覆性创新
- **赛道归属**：修仙/武侠
- **来源书**：让魔门再次伟大
- **核心机制**：肉身毁灭→器灵修仙（绕过灵根限制）→ 器身=宇宙法则容器 → 「不做人」从梗升华为主题（不做制度定义的人才能自由）→ 喜剧×世界观×阶级超越三重功能
