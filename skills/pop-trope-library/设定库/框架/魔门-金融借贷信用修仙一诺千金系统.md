# 金融借贷·信用修仙「一诺千金」系统

- **源域**：现代金融体系（信用贷款/抵押贷款/透支/期货/复利/征信）+ 中华传统文化「一诺千金」
- **转化逻辑**：将金融借贷概念移植到仙侠世界观——「一诺千金」从修辞升级为具有制度性约束力的信用货币系统：诺言可计价，承诺即负债。「气运借贷」允许修士预支未来成就换取当前突破，「灵根质押」以修炼资质为抵押品换取短期资源，「透支」允许修士超支未来战力应付当下危机——利息按日息/月息计复利。整个修仙界运行在「暗金融逻辑」之上。同时引入「血赚」不对称收益思维作为角色行为驱动力。
- **适用场景**：想给传统仙侠/武侠加入经济系统深度的作品；「债务驱动」叙事的黑暗奇幻题材；现代金融概念移植到奇幻设定的手法
- **赛道归属**：修仙/武侠
- **来源书**：让魔门再次伟大
- **核心机制**：一诺千金（诺言=信用货币）→ 气运借贷（预支未来成就）→ 灵根质押（资质=抵押品）→ 透支未来战力 → 复利计息 → 暗金融逻辑运行修仙界
