# 阶级压迫三金句序列（制度化压迫叙事结构）

- **源域**：阶级批判文学（压迫话语/异化理论）+ 网文爽文框架
- **转化逻辑**：在网文爽点节奏中系统性地嵌入阶级批判——三句标志性压迫话语构成完整的递进逻辑链：①陈皓彦「公私论」以公家利益合法化屠杀（权力借口）→ ②徐秉正「再苦一苦」将剥削制度化常态化（制度逻辑）→ ③徐秉正「黔首非人论」否定被压迫者人性以彻底合法化压迫（人性否定）。不是孤立台词，而是从「权力借口」到「制度逻辑」到「人性否定」的完整压迫哲学体系，让读者在享受爽文快感的同时被迫面对结构性不公。
- **适用场景**：需要黑暗现实主义社会批判深度、又不放弃爽文节奏的仙侠/武侠/玄幻作品
- **赛道归属**：修仙/武侠/通用
- **来源书**：让魔门再次伟大
- **核心机制**：递进压迫逻辑链（借口→制度化→非人化）→ 爽文外壳+阶级分析内核双层叙事 → 黑色幽默作为情绪缓冲 → 「屠龙者终成恶龙」的开放式结局
