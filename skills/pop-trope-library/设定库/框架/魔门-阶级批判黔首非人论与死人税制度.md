# 阶级批判·黔首非人论与死人税制度

- **源域**：秦代历史称谓「黔首」+ 阶级批判理论（非人化分析）+ 富兰克林「死亡和税收」格言
- **转化逻辑**：「黔首」从秦代平民称谓出发，经阶级批判理论的非人化分析，融入仙门制度设计——仙门不把底层修士当人看：收「死人税」（人死了尸体也要上税）、把人当「人牲」消耗、「黔首非人论」成为统治合法性的论证基础。一个设定同时完成历史×理论×世界观×喜剧的四层输出。引用富兰克林「死亡和税收」经典格言制造荒诞感——「就连死亡都不能避免税收」，将国际格言与仙门制度嫁接。
- **适用场景**：黑暗奇幻中的极端压迫制度设计；需快速让读者感受「这个世界有多黑暗」的建置手法；用极简设定表达复杂阶级批判
- **赛道归属**：修仙/武侠
- **来源书**：让魔门再次伟大
- **核心机制**：黔首非人论（统治合法性论证）→ 死人税（连死亡都逃不掉税收）→ 人牲消耗（底层修士=消耗品）→ 富兰克林格言化用（跨文化荒诞感）→ 历史×理论×制度×喜剧四层输出
