## 一、类人种族

### 1.1 人类（Human）
- **属性范围**：标准10点，进阶职业者可提高
- **特征**：最常发生阵营偏移的种族(ch074)
- **寿命**：40岁开始能力下降，50岁保留一半(ch076)
- **社会地位**：遍布大陆，建立琥珀城、白马城、阿伦黛尔等

### 1.2 半精灵（Half-Elf）
- **代表人物**：索伦（八分之一精灵血统）(ch005)
- **特征**：尖俏耳朵、夜视能力、天生灵巧、敏捷高但力量偏弱(ch005)
- **寿命**：可活一百多岁，但五六十岁后身手开始下降(ch076)
- **社会处境**：受人类和精灵社会双重排斥(ch097)
- **军事特色**：盛产长弓手——综合人类力量+精灵天赋(ch097)
- **信仰**：大多信仰精灵神系(ch121-125)

> **原文证据**：「半精灵可活一百多岁，但五六十岁后身手开始下降；这时会转向神术研究」（ch076）

### 1.3 精灵（Elf）
- **特征**：长耳朵、纤细灵活、俊逸秀丽(ch097)
- **寿命**：漫长（数百年）(ch121-125)
- **文化**：居住在树屋，种植粮食，出口弓箭和精致饰品(ch101-105)
- **军事**：每村有古树守卫（挑战等级12），精灵盛产长弓手(ch101-105)
- **神系**：精灵守护神（雌雄同体传闻）、生命女神、银月女士(ch120)

### 1.4 矮人（Dwarf）
- **特征**：铁脊山脉区域建立城市(ch044)
- **技艺**：矮人泰坦钢锻造工艺，硬度高但爱吹牛(ch121-125)
- **军事**：灰矮人使用原始炼金枪械(18世纪工艺水平)(ch056)
- **文化**：偷偷研究炼金术，被德鲁伊监视(ch058)
- **矮人火枪**：不如侏儒炼金枪械(ch056)

### 1.5 半身人（Halfling）
- **身高**：约一米，体重三四十磅(ch095)
- **特征**：手背脚背有少量绒毛、娃娃脸(ch095)
- **天赋**：天生潜行+20（不穿鞋时无声行走），唯一天生幸运加成种族(ch095)
- **平均力量**：8点（12点已属很高）(ch095)
- **性格**：好脾气、乐观、乐天派、好奇但无争霸心、天字第一号吃货(ch095)
- **特长**：种植烟草、酿造烈酒、追求美食、吹牛逼(ch095-096)
- **人口**：最多几十万，聚居夏尔和姆罗多地区(ch095)
- **守护神**：半身人守护女神——悠坦（强大神力）(ch095)
- **冒险者倾向**：一半以上财富用于享受生活，赚一笔就回老家退休(ch096-097)

> **原文证据**：「半身人是最高明的游荡者，不穿鞋时天生潜行+20」（ch095）

### 1.6 卓尔精灵（Dark Elf / Drow）
- **特征**：黝黑皮肤、尖耳朵、猩红色瞳孔(ch057)
- **社会**：母系社会，信仰蜘蛛神后(ch055)
- **祭祀传统**：自我鞭挞→痛苦→愉悦→更虔诚，蛇首鞭是标志器物(ch057)
- **语言**：懂古神语，可沟通阴影生命(ch057)
- **地表出现原因**：动荡之年位面壁垒松动(ch054-055)

> **原文证据**：「暗黑精灵是不会随意出现在地表世界，索伦可以肯定会有事发生」（ch054-055）

### 1.7 混血蛇人（Nagaji / Snake-like）
- **特征**：布满银色鳞片的类人生物(ch182-185)
- **天赋**：【蛇类感知】天赋专长(ch161-165)
- **出现**：薇薇安在荒野收服的仆人(ch163-164)

## 二、怪物图鉴

### 2.1 地精（Goblin）- 低阶
- **特征**：矮小，绿色皮肤
- **威胁等级**：低阶，群体有术士可能(ch030)
- **社会**：常被食人魔奴役为炮灰/服务(ch079-080)

### 2.2 狗头人（Kobold）- 低阶
- **特征**：矮小爬行类人，鳞片皮肤，身高约3尺/50磅(ch050)
- **属性**：敏捷13-15，天生护甲+1，夜视(ch050)
- **天赋**：矿工、陷阱制作（天生+20陷阱技能）(ch050)
- **语言**：龙语（声音似小狗吠）(ch050)
- **特殊**：容易出现狗头人术士（龙脉），使用燃烧之手(ch050)
- **战利品**：喜好收集亮晶晶物品(ch050)
- **杀戮经验**：普通~15点，战士~50点，术士~300点(ch051)
- **出处**：ch049-051

### 2.3 豺狼人（Gnoll）- 中阶
- **外观**：高2.2米/300磅，土狼头颅，灰皮肤鬃毛(ch059-060)
- **属性**：力量15+/敏捷15+，生物等级3(ch059-060)
- **追踪**：闻血味追踪（1公里/半天记忆）(ch059-060)
- **社会**：首领制，食物充足母狼年产十崽(ch059-060)
- **驯化**：土狼为狩猎帮手+应急储备粮(ch059-060)
- **特殊变体**：外层位面"嗜血者"——追踪10公里血腥味，食谱含低等龙类(ch059-060)
- **杀戮经验**：~200-300点/只，进阶战士更高(ch076)
- **可能信仰**：豺狼人之神（有牧师），几乎无巫师/术士(ch059-060)
- **分析**：比狗头人更危险，狗头人的10倍杀戮经验(ch059-060)

### 2.4 食人魔（Ogre）- 中高阶
- **基础**：生物等级5，挑战等级6，身高3米/500磅(ch065)
- **力量**：天生18+（超凡力量），进阶可达20(ch065)
- **类法术**：嗜血术/嗜血狂暴（2阶自动领悟）(ch065)
- **弱点**：迟钝，攻击速度不快(ch065)
- **战术警告**：绝对不能挨到——「挨到哪里就断哪里」(ch065)
- **双头食人魔**：智力21，可双头同时施法；8级双头食人魔/11级巫师/4级狂暴战士(ch083)
- **碎骨部落**：四五百到八百只，有组织有装备，奴役其他种族(ch079-080)
- **杀戮经验**：普通800点，进阶更高，督军2400点(ch081-084)

### 2.5 蜥蜴人（Lizardfolk）- 中阶
- **基础**：生物等级2，平均力量/敏捷12点(ch063-065)
- **族群**：数百只，战士比例50%+，有弓箭手(ch063-065)
- **特殊兵种**：蜥蜴人潜行者（进阶游荡者，类隐身术）(ch063)
- **施法者**：每族最少1个（术士/德鲁伊），精通驯化沼泽猛兽(ch063-065)
- **威胁比较**：比豺狼人更难对付，人数碾压+配备远程(ch063)
- **德鲁伊·枯萎者**：使用疫病术、缠绕术，堕落德鲁伊分支(ch065)
- **战利品**：首领房间有宝石+金德勒(ch078)
- **出处**：ch062-078

### 2.6 巨魔（Troll）- 高阶
- **再生能力**：惊人——断手断脚砍断脑袋都能长出来(ch076)
- **体质**：天生20点以上，大部分25点(ch076)
- **力量**：比食人魔更高(ch076)
- **弱点**：怕火焰和强酸（不提前准备很难杀死）(ch076)
- **威胁等级**：比食人魔更高级(ch076)

### 2.7 掘地虫（Ankheg）- 三阶
- **体型**：宽5尺(1.5m)，长可达30尺(10m)(ch031)
- **阶级**：挑战等级8，生物等级10(ch031)
- **属性**：最高20，最低10，总值80-90(ch031)
- **特殊能力**：强酸喷涂（每日1次）(ch031)
- **习性**：小型群居(2-5只)，南方潮湿松软泥土区域(ch031)
- **价值**：全身甲≈3000金/套；单只材料600-800金(ch034)
- **出处**：ch031-035

### 2.8 蛇魔（Marilith）- 三阶深渊恶魔
- **挑战等级**：6（双臂），每多一对手臂+5(ch011-015)
- **属性**：力量20+，恶魔皮肤（附魔皮甲级）(ch015)
- **免疫**：对控制法术免疫(ch011-015)
- **社会**：母系体制，雌性更强且有类法术能力(ch011-015)
- **索伦击杀**：利用陷阱+跳斩击杀，获得蛇魔长剑+1(ch016)
- **出处**：ch014-016

### 2.9 冰霜翼龙（Frost Wyvern）- 四阶亚龙
- **阶位**：四阶亚龙种(ch147)
- **挑战等级**：14，生物等级17(ch147)
- **属性**：最高22点，最低10点，总90-100(ch147)
- **体重**：雄性3吨+，雌性2吨+(ch147)
- **特殊**：冰霜吐息(ch147)
- **弱点**：翅膀（限制飞行后威胁大降）(ch147)
- **价值**：龙皮（超凡皮甲原料）、龙血（炼金术）、龙鞭（数千金德勒）(ch147-149)
- **出处**：ch146-150

### 2.10 土元素（Earth Elemental）- 四阶
- **挑战等级**：9，个人等级15(ch104)
- **生命值**：~300（150基础）（ch104）
- **属性**：最高22(力/体)，最低10(敏)，总90-100(ch104)
- **特性**：无致命弱点，免疫背刺，元素装甲免疫强度10以下物理伤害(ch104)
- **价值**：元素核心（南方~500金/块，北地700-800金）(ch104)
- **出处**：ch104-107

### 2.11 海魔/精魂（海外专有）
- 海外有大量**精魂**——图腾信仰产物，有神性无神格(ch187)
- 实力：传奇以上、半神以下(ch187)
- 神性特性：可用作"核燃料"，需特殊能力转化(ch187)
- 出处：ch187

## 三、特殊存在

### 3.1 吸血鬼（Vampire）
| 类型 | 挑战等级 | 个人等级 | 特征 |
|------|----------|----------|------|
| **正统吸血鬼** | 8 | 16 | 白银物种，强大类法术能力(ch019) |
| **血裔** | 6 | 12 | 未继承真血(ch019) |
| **血奴** | - | - | 半亡灵，神智消失(ch019) |
| **新生吸血鬼** | - | - | 需潜藏期培养，叛逃者属严重违规(ch172) |

- **弱点**：鲜血饥渴——意志判定难度30点（戒毒的10点）(ch175)
- **半神吸血鬼**：莱茵哈特——恐惧魔神之子，已晋升半神(ch183-185)

### 3.2 龙族
- **地位**：仅次于诸神的强大存在(ch098)
- **施法能力**：相当于低自身2-3级的术士，成年龙可释放8环法术(ch098)
- **战术**：极度狡猾——变形、偷袭、暗杀、迷宫、陷阱、放逐、怪物围攻(ch098)
- **战斗节奏**：法术甩光→龙息喷吐→龙翼拍击→龙类撕咬→传奇肉搏(ch098)
- **屠龙门槛**：完整传奇团队，索伦团队推演20遍(ch098)
- **上古红龙**：七阶，挑战等级25，生物等级34，属性：力42敏10体35智25感24魅24(ch044)
- **赤铜龙**：成年，精灵国度附近休眠，约一两年后苏醒(ch105)
- **黑龙**：生物等级24，1年后毁灭侏儒城市(ch056)

### 3.3 恐怖骑士（Terror Knight）
- **等级**：五阶传奇亡灵，挑战等级16，生物等级24(ch036-037)
- **属性**：95-110(ch036-037)
- **能力**：斩首剑【传奇】、恐惧灵气、唤起死灵、死亡气息(ch036-037)
- **坐骑**：梦魇（四阶恶魔，挑战等级12）(ch036)

### 3.4 不死鸟（Phoenix）
- **存在时间**：奥术帝国时期(ch097)
- **等级**：比神灵罕见，比巨龙强大(ch097)
- **状态**：已离开这个世界(ch097)
- **神炎**：每秒造成上百点神圣火焰伤害(ch097)
