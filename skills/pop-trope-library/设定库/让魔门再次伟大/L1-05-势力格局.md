## 二、势力关系图谱

### 2.1 白莲教 × 县衙（敌对/渗透关系）

```
白莲教（守冲）
    │
    ├─ 王平（卧底，ch001-ch004）←→ 县衙（陈浩彦上线）
    │
    ├─ 县衙内有白莲教卧底（ch003「有内鬼，终止行动」）
    │
    └─ ch004 陈浩彦出卖王平（过河拆桥）
        ch028 守冲利用王平当鱼饵钓出徐秉正
```

**本质**：双向渗透博弈。白莲教在县衙安插内鬼（截获情报「有内鬼」），县衙在白莲教安插卧底（王平）。双方互相利用、互相算计，是前37章最核心的对抗关系。

### 2.2 县衙 × 异人堂口（合作/利用关系）

```
县衙（陈浩彦/徐秉正）
    │
    ├─ 利用龙兴县印为筹码 → 请动游神真人出手对付守冲（ch018）
    │
    └─ 真正目标不是守冲 → 是调走游神真人后炸毁游神观抢丹药（ch019）
```

**本质**：朝廷对异人的「合作式剥削」。表面上异人独立于朝廷体制外，但朝廷可以用礼器（县印）等条件驱动异人。陈浩彦对游神真人的算计表明——朝廷将异人也视为棋局中的可牺牲棋子。

### 2.3 白莲教 × 异人堂口（独立/潜在对立）

```
守冲（白莲教）←→ 游神真人（异人堂口）
    │                      │
    ├─ ch018 雨中街道冲突  ├─ ch018 守冲拦截游神真人
    ├─ 守冲要取莲华刀     ├─ 游神真人被龙兴县印吸引而来
    └─ 守冲利用了游神真人  └─ 游神观被炸（陈浩彦的算计）
```

**本质**：原本是两个独立势力（白莲教搞底层革命，异人堂口在体制外修炼），被陈浩彦的「三层钓鱼计划」强行卷入同一棋盘。二者之间并无直接深仇大恨，但被第三方操弄成互耗。

### 2.4 仙门 × 赤土世界（收割/隔绝关系）

```
仙门（极乐仙峰等）
    │
    ├─ 力士制度：赤土武者被炼成力士（ch005-ch007）
    ├─ 死人税：北辽长生天对赤土征收「死人税」（ch033）
    ├─ 凡人工具论：仙门视凡人为「人材」「人牲」（ch036）
    └─ 灵根制度：无灵根者在仙门毫无立足之地（ch033）
```

**本质**：仙门对赤土世界是「单向收割」——需要力士时从赤土抓人炼成奴仆，需要税收时通过长生天征收，需要实验材料时将有修为的凡人炼成法器（ch036王平被炼）。两个世界之间存在巨大的「信息不对称」和「权力不对称」。

### 2.5 仙门 × 劫修（正统/叛乱关系）

```
仙门（正统制度）
    │
    ├─ 五贼盗天功法被「盗天贼」开源 → 劫修群体产生
    │
    └─ 劫修杀之不尽、地下化发展
```

**本质**：制度性压迫导致的反叛体系。盗天贼是仙门制度的破壁者——用开源功法对抗知识垄断。劫修与仙门的矛盾是全书深层命题（知识垄断 vs 知识共享/阶级固化 vs 上升通道）的仙门版本。

```
                    └────┬─────┘
                         │
          ┌──────────────┼──────────────┐
          │              │              │
    ┌─────▼────┐  ┌─────▼────┐  ┌─────▼────┐
    │ 陈浩彦   │  │ 游神真人 │  │ 守冲     │

    │ 第一层   │  │ 第二层   │  │ 第二层   │
    └──────────┘  └──────────┘  └──────────┘
                         │
                    ┌────▼────┐
                    │ 王平    │  ← 底层棋子（ch001-ch017）
                    │ (捕快)  │     后被太平经+灵识推至棋盘玩家层

                         │
                         ▼
                  ┌──────────────┐
                  │ 刘烨/普通捕快 │  ← 执行层

                  └──────────────┘
```

**关键特征**：龙兴县的权力结构是「封号武师一元化统治」——徐秉正同时掌握行政权（知县）、最高武力（神意+封号）、政治资源（师承国子监程师）、暴力机器（县衙捕快+军阵）。其余势力（白莲教、异人堂口）只能在他的权力阴影下运作。

### 3.2 仙门世界权力轮廓（ch033-ch037，不完全）

```
                   ┌──────────────┐
                   │ 修仙宗门上层 │   ← 掌握灵根测试、功法垄断、法器炼制
                   │ (未命名)     │     天地阁/神兵院等机构并行
                   └──────┬───────┘
                          │
                    ┌─────▼──────┐
                    │ 极乐仙峰    │   ← 尤道姑所属道峰层级
                    │ (各弟子)    │     真传弟子享受资源、操纵力士
                    └──────┬──────┘
                           │
                    ┌──────▼──────┐
                    │ 劫修群体     │   ← 地下势力（被杀之不尽）
                    │              │     盗天贼开源→五贼盗天功法传播
                    └──────────────┘
                           │
                    ┌──────▼──────┐
                    │ 力士/凡人     │   ← 最底层（比赤土黔首更不如）
                    │ 随侍童子      │     被炼器材料/人牲/工具
                    └──────────────┘
```

**关键特征**：仙门的剥削更直接、更「高效」——凡人不仅是劳动力，更是炼器材料。「在仙门，我们称之为高效」（ch036尤道姑台词）。灵根制度是出身天花板，无灵根者连被剥削的资格都不完整。

### 主角势力标签特征总结

王平在全书前37章完成了三次「势力身份切换」：

1. **ch001-ch004**：白莲教底层卧底 → 被县衙背叛的弃子（双重身份的崩解）
2. **ch008-ch032**：龙兴县捕快 → 孤胆反攻的叛军（体制内→体制外→体制的覆灭者）
3. **ch033-ch037**：尤道姑座下随侍童子 → 六欲天魔旗器灵（从被炼者到器灵修仙的蛰伏者）

每一次切换都是「因祸得福」——旧身份被摧毁后，新身份带来更大的力量空间。这是王平作为「不死小强」的核心叙事模式。
