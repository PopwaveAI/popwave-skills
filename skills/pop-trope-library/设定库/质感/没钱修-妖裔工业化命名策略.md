# 妖裔工业化命名策略

- **源域**：现代工业养殖（繁育基地/孵化室/流水线/开除儿籍）+ 公司人事制度（入职培训/最佳员工/公司魂洗脑/学历分层）+ 商品标价体系（每个妖裔都有标价/种子生意/蛋白质买卖）
- **转化逻辑**：将妖族后裔的生物学出生改造为工业化生产流程——「繁育基地」不是自然生育而是工业化孵化，「开除儿籍」作为终极惩罚（从公司员工名单中除名=从家族中除名），「最佳员工公司魂」洗脑操作。妖裔被商品化——每个妖裔都有标价（出厂价/零售价），龙人转型从妖兽转向人类种子市场（卖蛋白质/种子给人类）。妖裔层层细分学历决定待遇、血脉决定地位，将工业管理的等级体系与种族差异结合。命名策略的核心是「用工业化和商业化的语言重新描述传统妖族」，制造科技伦理的恐怖感。
- **适用场景**：需要工业修仙/生物科技伦理批判的修仙/科幻/赛博作品
- **赛道归属**：修仙·科幻
- **来源书**：没钱修什么仙
- **核心机制**：工业化生育命名（繁育基地/孵化室/流水线）→ 公司化管理（入职培训/最佳员工/开除儿籍）→ 商品标价（每个妖裔有定价/种子买卖）→ 工业术语制造科技伦理恐怖感
