# 克苏鲁氛围化叙事调味（氛围点缀而非核心设定）

- **源域**：克苏鲁神话（洛夫克拉夫特体系/不可名状恐惧/精神污染/San值归零）
- **转化逻辑**：将克苏鲁元素从「核心设定」降级为「叙事调味料」——扭曲阴影的视觉效果、精神压迫的描写方式、不可名状的怪物描述——这些只用于增强氛围，而非构建整个世界观的核心。怪物仍然有清晰的数据面板（挑战等级/阶位/B+评分），不会出现「无法理解的恐怖」导致叙事停滞。既保留了诡异氛围的独特美感，又不破坏游戏化叙事的爽感节奏。
- **适用场景**：在战斗/升级为主的网文中加入恐怖悬疑氛围；需要在特定章节营造「未知恐惧」情绪但全书不改变核心调性；黑暗奇幻作品的氛围点缀
- **赛道归属**：西幻/通用
- **来源书**：深渊主宰
- **核心机制**：克苏鲁元素降级为调味料 → 保留氛围描写（阴影/精神压迫/不可名状）→ 怪物仍可量化（数据面板清晰）→ 不破坏游戏化叙事爽感
