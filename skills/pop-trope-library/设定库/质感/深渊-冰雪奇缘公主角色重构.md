# 冰雪奇缘公主角色重构（命名借用+角色内核重构）

- **源域**：迪士尼《冰雪奇缘》安娜公主/艾莎长公主/阿伦黛尔王国 + 黑暗奇幻政治剧
- **转化逻辑**：借用「安娜公主」「阿伦黛尔王国」的文化命名和角色关系框架（最小的公主+有姐姐是长公主），但彻底重构角色设定——长公主不再是被恐惧驱使的善良女王，而是政治家+掌权者，性格和能力远比艾莎黑暗复杂。冰魔法保留但用法从「情感表达」改为「政治威慑」。安娜的「友好介绍」功能被保留——通过安娜的角色引出阿伦黛尔的政治格局，让读者通过一个「正面角色」的视角理解世界。角色名字是读者入口，角色内核才是叙事武器。
- **适用场景**：将迪士尼/流行文化角色「黑化」或「现实化」改编到严肃题材中；借用经典IP的角色关系结构但替换主题内核；从儿童向素材提取角色原型适配成人向黑暗叙事
- **赛道归属**：西幻
- **来源书**：深渊主宰
- **核心机制**：文化命名借用（降低认知成本）→ 角色内核彻底重构（迪士尼→黑暗政治）→ 冰魔法=政治威慑 → 安娜作为世界观引入窗口角色
