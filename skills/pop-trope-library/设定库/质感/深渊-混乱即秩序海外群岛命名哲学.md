# 「混乱即秩序」的境外之地命名哲学

- **源域**：DND 阵营体系（混乱中立/混乱邪恶）+ 海盗文化（私掠/灰色地带）+ 被遗忘国度·境外之地设定
- **转化逻辑**：海外群岛的命名和氛围设计围绕「混乱即秩序」的核心内核——混乱不是无序，而是另一种不同于大陆的秩序体系。泰罗港「未被通缉/举报的罪犯就是合法平民」的灰色法则，海盗公约的「暴力有定价」，偷窃被抓卫兵不管但弄死人要罚款的「定价公平」。通过具体的命名（泰罗港/沉船海湾/毒蛇岛/孤狼岛）传递「这就是境外之地」的信息，每个地名背后都有政治生态：海洋神殿掌控的港口、海盗割据的海湾、索伦新建的开拓基地。
- **适用场景**：需要设计西部边境/法外之地/灰色地带等区域命名体系的西幻或开放世界作品
- **赛道归属**：西幻/通用
- **来源书**：深渊主宰
- **核心机制**：混乱≠无序 → 暴力有定价 → 灰色地带命名（港口/海湾/营地）→ 地名传递政治生态信息 → 不同地区的秩序规则差异化
