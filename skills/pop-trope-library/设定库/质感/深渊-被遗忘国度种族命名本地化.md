# 被遗忘国度种族命名的本地化策略

- **源域**：被遗忘的国度（Forgotten Realms）种族设定——卓尔精灵社会结构/半身人特征/吉斯洋基人
- **转化逻辑**：完整沿用费伦的种族设定和文化符号，但做了网文语境下的「轻量化本地化」。卓尔精灵（Drow）社会结构（雌性上位/蜘蛛神后信仰/幽暗地域故乡）不加解释直接出场，依靠叙事自然展示而非说明文。半身人（Halfling）保留1.25米矮小体型、嗜酒、美食偏好的费伦特征。吉斯洋基人的银刃神器直接引用。命名策略是「保留费伦风味但不增加阅读门槛」——不刻意解释 Feerûn/Underdark 等异域名，而是将费伦元素转化为读者直觉可理解的奇幻常识。
- **适用场景**：需要 DND/费伦西幻底蕴但面向网文读者的西幻世界命名设计
- **赛道归属**：西幻
- **来源书**：深渊主宰
- **核心机制**：费伦种族特征完整沿用 → 网文化轻量处理（不解释不说明文）→ 种族文化通过叙事自然展示 → 本土读者直觉可理解
