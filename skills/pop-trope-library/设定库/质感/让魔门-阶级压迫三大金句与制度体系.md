# 阶级压迫三大金句 × 制度性压迫体系

> 来源：让魔门再次伟大 · 世界宪法 + 参考域-阶级批判文学 + L1-01世界蓝图
> 拆书依据：全书世界观的地基——不是"这个世界有压迫"，是压迫具体到什么程度

## 三大金句序列

这三句不是偶然修辞——是全书世界观的地基，分别对应三个层面的合法性建构：

### 第一金句：公私论（意识形态合法化）

| 维度 | 内容 |
|:-----|:-----|
| 人物 | 陈皓彦（县尉） |
| 原文 | "人命关天，指的是那些住在县城里的人，是豪族，是士子，至于城外的那些黔首……他们也算人？" |
| 章号 | ch004 |
| 压迫层次 | 意识形态层——给"谁是人谁不是人"划一条线 |
| 叙事功能 | 第一次让读者意识到：在当权者心中，黔首不是人 |

### 第二金句：再苦一苦（经济剥削合法化）

| 维度 | 内容 |
|:-----|:-----|
| 人物 | 徐秉正/陈皓彦 |
| 原文 | "城内良民只要一成就行""城外黔首再苦一苦他们吧" |
| 章号 | ch019 |
| 压迫层次 | 经济层——为了城内1成良民的安稳，城外9成黔首可以压到死 |
| 叙事功能 | 揭露剥削的"合理性"——不是一个人坏，是制度让剥削变成"合理的维稳" |

### 第三金句：黔首非人论（存在论合法化）

| 维度 | 内容 |
|:-----|:-----|
| 人物 | 徐秉正（神捕门封号武师） |
| 原文 | "城外哪里有灾民，只有一群产粮的黔首""每隔一季就需要清理的杂草""黔首也能算人？" |
| 章号 | ch027 |
| 压迫层次 | 存在论层——从"定义非人"到"定期清理" |
| 叙事功能 | 终极揭露——压迫者是真心的，不是演戏 |

**三金句的核心设计**：第一个人说"黔首不是人"（定义），第二个人说"那就再苦一苦"（执行），第三个人说"定期清理杂草"（常态）。步步升级，每一句都在前一句的逻辑上再推一步。

## 六大制度性压迫

| 制度 | 内容 | 原文锚点 |
|:-----|:-----|:---------|
| **秋税制度** | 查田→假官册（八亩→一亩半）→分户剥削 | ch019 |
| **武功缺失制度** | 朝廷刻意剥离武功"炼法"，只传打法+呼吸法——学了也练不成 | ch008 |
| **死人税** | 仙门对凡人征收尸体作为税收，"在仙门，我们称之为高效" | ch033 |
| **人牲制度** | 凡人在仙门眼中被拆解为器官/资质/根骨零件——人=资源 | ch033/ch036 |
| **炼器人材制度** | 随侍童子被当作炼器材料消耗——活人=消耗品 | ch005/ch036 |
| **灵根阶级制** | 废灵根=大道无望=被钉死在底层。不是努力不努力的问题——是生理决定论 | ch033-034 |

## 阶级抽水链条

```
皇族 → 门阀 → 宗门 → 江湖 → 黔首
```

| 层级 | 抽水方式 | 被压榨方 |
|:-----|:---------|:---------|
| 皇族 | 秋税/丁银/加派 | 门阀以下全层级 |
| 门阀 | 地租/高利贷/佃农 | 宗门以下 |
| 宗门 | 供奉/死人税/人牲 | 江湖+黔首 |
| 江湖 | 保护费/地盘/押司 | 黔首 |

每一层抽取剩余价值后留给下一层更少。

## 仙门侧的阶级压迫

仙门用"效率"包装压迫：
> "在仙门，我们称之为高效" (尤道姑，ch036)

| 层次 | 内容 | 实质 |
|:-----|:-----|:-----|
| 死人税 | 尸体上税 | 死后剩余价值也被榨取 |
| 人牲 | 人被拆成器官/资质零件 | 器官级商品化 |
| 力士炼制 | 活人炼成傀儡 | 人格消灭 |

三层递进：死后还要交税→活着被当零件→被炼成傀儡。

---

## 这套设定为什么"好"

不是因为"写出了压迫"——是**压迫被具体化到了制度层面**。读者不是看到"这个世界很黑暗"，而是能看到：
- 黑暗长什么样（假官册怎么改的）
- 黑暗怎么运转的（秋税制度怎么执行的）
- 黑暗为什么不被推翻（灵根生理决定论+武功缺失制度——底层连翻盘的武器都没有）
