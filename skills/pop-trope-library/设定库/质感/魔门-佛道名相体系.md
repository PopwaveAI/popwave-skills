# 佛道名相体系（他化自在天/二十四节炁/六欲天）

- **源域**：佛教佛学名相（他化自在天/六欲天/莲华/菩萨/大德）+ 传统丹道术语（二十四节炁/炼炁/叩仙门/功体图）+ 道教《阴符经》（天有五贼）
- **转化逻辑**：大量借用佛教名相作为功法/经卷/设定名称，但佛教本体教义（因果轮回/慈悲度世）不实质参与叙事。他化自在天（欲界最高天、魔王波旬居所）用作太平经第一卷名和功法名，暗示魔王属性。六欲天（色声香味触法）→ 六欲天自在成就法（以身豢兽的废灵根旁门）。莲华/菩萨等符号保留佛教美学但不深入教义。传统丹道二十四节气→二十四节炁（灵气循环系统）。功体图→焚图解密（传统内景图的创新转化）。采用「名相华丽但本体轻盈」的策略——用古典术语提供东方奇幻美学认同，但所有设定都做了功能转化，不承担宗教教义责任。
- **适用场景**：需要东方奇幻美学深度但不深入宗教教义的修仙/仙侠世界命名
- **赛道归属**：修仙
- **来源书**：让魔门再次伟大
- **核心机制**：佛教名相（他化自在天/六欲天/莲华/大德）→ 功法/设定命名（美学功能）→ 本体教义不参与叙事 → 丹道术语（节炁/功体图/叩仙门）→ 功能化改编为量化修炼体系
