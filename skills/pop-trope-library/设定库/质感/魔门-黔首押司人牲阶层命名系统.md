# 黔首/押司/人牲的阶层命名系统

- **源域**：中国古代社会阶层称呼（黔首/皂吏/青士/朱服/紫卿/金爵）+ 阶级批判文学（非人化命名）+ 佛教概念（人材/人牲）
- **转化逻辑**：用中国历史中真实的阶层称号来标记社会地位和价值——「黔首」不是普通名词而是阶级标签（被统治阶级视为产粮单位和杂草）。身份兑换体系（白身→皂吏→青士→朱服→紫卿→金爵）借用古代官吏服色制度。仙门将凡人称为「人材」（ch036尤道姑「把你炼成法器，也不算浪费了你这份人材」），以工业原料的隐喻标记底层价值。人牲/力士/随侍童子等命名均指向「非人化」——用称呼本身传递阶级压迫信息。押司（押=抵押+司=管理）的名字本身就是金融隐喻。
- **适用场景**：需要深度阶级标注、身份即命运的东方封建社会命名设计
- **赛道归属**：修仙/武侠
- **来源书**：让魔门再次伟大
- **核心机制**：历史阶层名（黔首/皂吏/青士/朱服）→ 身份即价值标签 → 非人化命名（人材/人牲/力士）传递压迫信息 → 姓名含隐喻（押司=管抵押的人）
