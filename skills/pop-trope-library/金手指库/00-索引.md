# 金手指库 · 索引

> **定位**：金手指设计素材库（WHAT——设计什么金手指）。与套路库的"金手指体系与外挂进化"和"隐藏天赋觉醒"互补：套路库回答 HOW（怎么写叙事节奏），金手指库回答 WHAT（设计什么金手指）。
> **消费者**：creative（金手指设计步骤，强制查询）、world（一致性验证参考）、character（金手指结构化记录参考）
> **入库时机**：拆书复盘后提炼 / creative 金手指设计文档产出后（独特设计可入库）
> 更新: 2026-06-24

## 查询路由

| 你的需求 | 查哪里 |
|:---------|:-------|
| 不知道设计什么金手指 | 按赛道筛选 → 按元爽点匹配 → 读设计卡的"核心机制"+"代价设计" |
| 有金手指类型但不知道代价怎么设计 | 按类型筛选 → 读"代价设计"段+"代价设计变体" |
| 有金手指但不知道成长路径怎么设计 | 按类型筛选 → 读"成长路径"段+"成长路径变体" |
| 想知道金手指适配哪些赛道 | 读"赛道适配分析"段 |
| 想知道金手指的叙事节奏怎么写 | → 去套路库查"金手指体系与外挂进化"或"隐藏天赋觉醒"（HOW层） |

## 库缺声明规则

- 按赛道筛选无匹配时 → 标注"本赛道金手指库缺"，降级到通用素材
- 按元爽点筛选无匹配时 → 标注"本元爽点金手指库缺变体"，降级到通用套路
- 道具型/感知型当前无卡 → 标注"本类型库缺，待入库"，降级到联网搜索补充
