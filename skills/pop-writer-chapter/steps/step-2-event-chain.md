## 逐事件设计

从基线 §6 的事件数范围出发，逐事件填写以下字段：

| 字段 | 要求 | 来源 |
|:-----|:-----|:-----|
| 事件序号 | N | — |
| 标题 | 简短概括 | 自行命名 |
| 地点 | 可追溯到剧情线的地点暗示 | 基线 + 卷纲地点描述 |
| 参与角色 | 必须在基线 §4 本章登场角色中 | 基线 §4 |
| scene | DNA 场景卡映射值（combat_stealth / dialogue_daily 等） | 按事件类型推断 |
| POV | 感知锚点角色名。留空 = 默认主角 | 跨角色章必填 |
| 关键对白/数据 | 不可替换的原文。填空 = Render 自行发挥 | 剧情线 + 事件内容推断 |
| 感官锚点 | 启动事件的感官提示（"雨声启动"/"铁锈味启动"等） | 按场景推断 |
| 事件内容 | 「角色做了什么 → 什么结果」。一句话 | 从基线 Canvas 约束推导 |
| **驱动角色** | 这个事件是谁的决策导致的？不是"谁在场"而是"谁的决定推动了这件事"。标注溯源：`角色卡·{角色名}·决策转折点预判·{转折点名}` | 角色卡驱动接口的决策转折点预判 + L2 驱动决策列 |
| **角色决策** | {角色名}决定{决策} → 因为{渴望/缺陷/外部压力}。标注溯源：`L2·7阶段表·{阶段}` | 角色卡驱动接口 + L2 7阶段表驱动决策列 |
| **决策代价**（★DQTC） | 这个决策的代价是什么？从角色卡/L2 映射 | 角色卡决策转折点预判·代价列 + L2 7阶段表·代价列 |
| **不可逆后果**（★DQTC） | 这个决策的不可逆后果是什么？从角色卡/L2 映射 | 角色卡决策转折点预判·不可逆后果列 + L2 7阶段表·不可逆后果列 |
| **决策一致性**（★DQTC） | 事件链的驱动决策与 L2 驱动决策列一致？是/否。否→触发反馈循环2 | 比对事件链角色决策 vs L2 7阶段表驱动决策 |
| 情绪目标 | 一个词（警觉/紧张/冷酷/决断/...） | 参考 emotional-beats 词汇表 |
| 冲突层次 | external / internal / interpersonal（可选 1-3 层） | 按事件判断 |
| 所属线 | 该事件推进哪条剧情线 | 基线 §2 Canvas 活跃线 |
| **剧情推进点** | **是/否** — 该事件是否推进了某条 L2 剧情线（即该事件对应剧情线骨架中的某个节点变动） | 填"是"时必须在"所属线"标注具体L2 |
| 信息释放 | 如触发设定交付，标注信息项 | 基线 Canvas 约束 |
| 估计字数 | 100-250 字 | 自行估算 |

> ★ 以下字段 **Step 2 不填**，留给 Step 3 润色时补充：
> - ◆小爽点 / ★中爽点
> - 契诃夫枪引用
> - 本章套路

**"剧情推进点"定义：** 事件 ≠ 剧情推进点。一个事件可以是过渡/铺垫/氛围渲染（不推进任何线），也可以是剧情推进点（对应某条 L2 骨架中的节点变动）。每章至少 3~5 个事件必须是剧情推进点，否则这一章在所有 L2 上都"空转"。

**设计规则：**
- 事件链有因果——每个事件到下一个有「所以」或「然后」
- 章首事件情绪 ≠ 高潮级（紧张/震撼）——没有铺垫，退回
- 章末事件必须是钩子（悬念/信息/情绪）
- 连续 3 个事件不能同一情绪
- **每章至少推进 3 条不同的 L2**（即至少 3 个不同"所属线"的剧情推进点）
- **每个事件必须有"驱动角色"** — 事件不是自然发生的，是某个人的决策导致的。如果找不到驱动角色 → 这个事件可能是多余的过渡，考虑删除或合并
- **"本章关键决策"从角色卡映射** — 登场人物表的"本章关键决策"字段从角色卡的决策转折点预判映射，不是凭空想

## 门禁

| 检查项 | 失败动作 |
|:-------|:---------|
| 事件数 < 章字数 ÷ 200 | ❌ 退回补事件 |
| **剧情推进点 < 3** | ❌ 退回——至少 3 个事件必须推进某条 L2 |
| **推进的 L2 不足 3 条不同线** | ❌ 退回——至少推进 3 条不同的 L2 |
| scene/POV/关键对白/感官锚点 任缺一 | ❌ 退回 |
| 事件链因果断裂 | ❌ 退回 |
| 章首情绪 = 高潮级 | ❌ 退回 |
| 连续 3 事件同一情绪 | ❌ 退回 |
| 章末不是钩子 | ❌ 退回 |
| 事件无驱动角色 | ❌ 退回——每个事件必须标注驱动角色 |
| 本章关键决策未填 | ❌ 退回——从角色卡驱动接口映射 |
| 驱动角色/角色决策无溯源 | ❌ 退回——必须标注来自角色卡哪个转折点 + L2 哪个阶段 |
| 关键决策无代价/不可逆后果 | ❌ 退回——从角色卡/L2 映射 |
| 决策一致性为"否" | ⚠️ 触发反馈循环2（决策漂移）——标注反馈后暂停，退回 character/plot |
| Step 2 无查询声明 | ❌ 退回补查 |
