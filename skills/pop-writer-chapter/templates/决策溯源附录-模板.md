## 字段说明

| 字段 | 要求 | 来源 |
|:-----|:-----|:-----|
| 驱动角色 | 场景内事件是谁的决策导致的 | 角色卡剧情驱动接口 |
| 角色决策 | 具体决策内容 | 章锚点 Canvas 推导 |
| 决策来源 | 角色卡哪个转折点 + 章锚点哪个阶段 | 角色卡 + 章锚点 |
| 决策代价 | 这个决策牺牲了什么 | 角色卡/L2 映射 |
| 不可逆后果 | 决策后无法回头的结果 | 角色卡/L2 映射 |
| 一致性 | 与角色 core_desire 是否一致 | ✅一致 / ❌需说明 |
