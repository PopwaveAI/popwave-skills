## 快照标记规范

**为什么要快照标记：**

角色是随时间变化的。苏午在第一章获得模拟器时 vs 全书后期是完全不同的人。没有快照标记，角色卡就不知道是在"哪个时间截面的谁"。

**快照标记四要素：**

| 要素 | 示例 | 说明 |
|:-----|:------|:------|
| 章范围 | `第 1 章` | 角色卡对应的章节区间 |
| 剧情节点 | `范闲初入京都前` | 故事内的时间锚点 |
| 卡状态 | `主角卡·第一幕` | 持续更新的卡标注版本 |
| 截取原因 | `基于首次登场时的特征` | 为什么取这个时间截面 |

### 实操规则

- **新角色** → 快照标记 = 首次登场章节
- **阶段性更新角色卡** → 快照标记随阶段推进（如 `→第2版→第3版`）
- **全书已完成** → 快照标记 = `全书完 | 终态卡`

## 如何用这些示例

1. **对照 SKILL.md 看** — 每张卡的维度数 = 对应级别模板要求
2. **注意快照标记** — 同一角色在不同阶段应有不同的角色卡
3. **跨赛道对比** — 4 本不同赛道的小说展示通用模板的适应力
4. **级别差异** — 对比 Lv1→Lv4，感受"复杂度递增"的实际效果
5. **对白驱动** — 每张示例卡都应能直接回答"这个角色在正文里会怎么说话"

> 示例卡严格基于原文信息编写，非原文直接引用的部分以 `※` 标注推断成分。
