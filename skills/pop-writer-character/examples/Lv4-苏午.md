# Lv4 · 苏午 — 案例卡

> 📍 **快照标记：第 1 章 | 获得完美人生模拟器·初始状态**
> 小说：《我的诡异人生》 | 卡状态：主角卡·初始阶段
> 截取原因：苏午的一切从这块爷爷留下的手表开始。本章展示了他获得金手指前的全部初始设定——家庭、性格、初始驱力。后续角色弧线全部从此出发

---

## 设计目的

贯穿全文的灵魂。苏午是一个"命运被神秘力量选中"的普通人——他的世界观在第一章被彻底颠覆：父母的离奇死亡、爷爷传下的手表=完美人生模拟器、天赋"冷静"直接作用于现实。读者通过他的眼睛体验这个诡异的修行世界，从一无所知到逐步揭开谜题的整个历程。

## 叙事能力

驱动全篇情节走向：苏午的每一次模拟选择 = 下一段的剧情走向。他的"冷静"天赋决定了他不会冲动行事，而他对父母死因的追寻是全书最核心的情感驱动力。所有关键转折（模拟器的秘密、诡异世界的真相）都由他的探索决策触发。他是读者在这个诡异世界中的唯一认同锚点。

### 出场状态

```
当前状态：独居。父母三年前在同一天下着小雨的日子去世。他坐在书桌前，拿着已故爷爷传下来、又传到父亲那块手表发呆。
与前情的落差：他以为这只是普通的怀念亲人的一天。但手表的"故障"——表针在三年来第一次停止——标志着他再也不是"普通人"了。
```

## 三、心理 ← 新增

### 核心性格特质（3 条）

```
特质名：超乎常人的冷静
  表现：获得完美人生模拟器这种超自然事件后，他没有恐慌尖叫，而是立刻开始理性分析——"父母离奇死亡与模拟器有没有关系""首充奖励应该很超值"
  边界：冷静不等于没有情绪——当涉及父母死因时会动摇
  盲点：可能因为太冷静而低估了某些需要热血冲动的机缘

特质名：谨慎且逻辑严密
  表现：抽取天赋后立刻分析"绿色品质上面是否还有蓝紫橙"；面对"带物品进游戏"的选项，选择不带任何东西（"都不清楚这游戏是什么，先不冒险"）
  边界：在巨大的诱惑面前（如"亲身体验模拟人生"的高风险高回报选项）会犹豫
  盲点：过分谨慎可能会错过高风险高回报的早期机遇

特质名：对亲人之死的执念
  表现：模拟器出现后，他立刻联想到父母离奇死亡是否与此有关——这是他愿意投入模拟的核心动力
  边界：如果发现真相比他想象的更残酷，不知道他还能不能保持冷静
  盲点：为了真相，他可能会在理性判断之外的领域贸然冒险
```

### 决策逻辑

```
日常决策倾向：观察 → 分析 → 谨慎决定
危机决策倾向：与日常一致——"天赋冷静让他能在混乱局面中细心观察"
决策阈值：当涉及"父母死因线索"或"模拟器核心秘密"时，他的谨慎度会下降
```

### 核心缺陷

```
缺陷：对"父母之死的真相"的执念可能成为被人利用的弱点
  对谁/什么事最致命？任何掌握这条线索的人/势力，可以用真相操纵他的选择
  整个故事中改善了多少？※ 从第一章看，这条缺陷不仅没有改善，反而随着模拟的深入不断强化
```

## 四、驱力（四层完整）← Lv3 升级

| 层级 | 内容 | 故事中何时显露 |
|:-----|:------|:--------------|
| 表层目标 | 弄清楚完美人生模拟器是什么、怎么用 | 第 1 章即显——"要不要开始模拟？" |
| 深层需要 | 查清父母离奇死亡的真相 | 第 1 章暗示——"如果和模拟器有关…" |
| 黑暗冲动 | 如果父母之死是人为，他可能会做出极端报复 | 尚未显露，由读者自行想象 |
| 恐惧 | 最怕真相是自己无法承受的——或者，自己与父母的死有关 | 尚未显露 |

### 驱力冲突

```
哪个驱力层级之间的张力最大？  表层目标（探索模拟器）与恐惧（怕真相无法承受）——他既想深入，又怕深入后发现的东西是自己不想要的
这种冲突在故事中以什么形式出现？  每次模拟前他都会犹豫：选模拟还是不模拟？模拟了可能得到线索也可能得到更可怕的东西
```

## 五、能力 ← 新增

### 当前能力评估

| 维度 | 描述 | 定位 |
|:-----|:------|:-----|
| 优势项 | 天赋"冷静（绿色）" + 逻辑思维能力 | 让他不像其他普通人一样在被超自然冲击时崩溃 |
| 中等项 | 普通人的身体素质 | 学生体格，没有特殊训练背景 |
| 弱项 | 对超凡世界一无所知 | 最致命的短板——他连这个模拟器是做什么的都不清楚就进去了 |

### 成长路径

```
起点能力档位：普通人 + 绿色品质"冷静"天赋
第一阶段目标：通过初次模拟了解游戏规则和世界观
第二阶段目标：积累元玉，抽取更高品质天赋/获得更多资源
...
终点能力档位：※ 预测应能通过模拟获得超凡力量，但代价未知
瓶颈预测：每次模拟都需要消耗元玉，但元玉获取方式和上限不明
```

## 六、演化 ← 新增

### 弧线阶段（初始阶段）

```
阶段 1：觉醒·获得模拟器
  起止章节/幕：第 1 章
  触发事件：手表停走→割破手指→血液渗入→完美人生模拟器启动
  前心态：一个沉浸在父母离世伤痛中的普通青年
  后心态：意识到这个世界和他想的不一样——"我可能需要从模拟器里寻找答案"
  关键决策：消耗 1 元玉抽取永久天赋"冷静（绿色）"
  表象变化 vs 本质变化：表面还是那个独居学生，但内心已经接受了"这个世界有超自然力量"的事实
```

```
☑ 成长弧  — 从普通青年到超凡修行者的经典成长路径

□ 堕落弧
```

## 八、说法风格 ← 新增

| 字段 | 填写 | 说明 |
|:-----|:-----|:-----|
| 核心声线 | 冷静克制，像一边说话一边做判断 | 不抢情绪，先稳住局面 |
| 日常语气 | 低平、简洁、偏陈述句 | 不喜欢空寒暄 |
| 压力语气 | 越危险越平静，句子更短 | 进入“观察—判断—执行”模式 |
| 对不同对象的分层 | 对普通人收着锋芒；对可信同伴给明确结论；对敌人不解释；对掌握线索的人会追问得更细 | 语气变化小，但问法会变 |
| 句式偏好 | 先结论，后补推理；少感叹；常用条件句和反问确认 | “如果……那么……” |
| 高频词与称呼 | 先别急、等一下、你确定、如果是这样、再说一遍 | 高频逻辑连接词多 |
| 禁用表达 | 不会无缘无故热血宣誓，也不会长篇抒情 | 除非被亲人线索刺穿防线 |
| 价值观外露方式 | 不直接喊原则，而是在措辞里优先体现“先保命，再追真相” | 价值判断藏在决策顺序里 |
| 谎言方式 | 撒谎时会说得更完整、更像推演，而不是更简短 | 用逻辑包装情绪 |
| 情绪失控的破音点 | 涉及父母死因、爷爷遗物、模拟器真相时，语速会突然变快 | 冷静外壳会裂一下 |
| 独白风格 | 逻辑化、自我盘问式 | 像在脑内做推演 |
| 示例台词 | “先别碰那块表，再让我看一眼。” / “如果它真和我父母的死有关，那我就必须进去。” / “你现在说的每一句话，我都记着，别前后对不上。” / “我不是不怕，我只是得先知道自己该怕什么。” | 覆盖日常 / 决策 / 冲突 / 情感 |
