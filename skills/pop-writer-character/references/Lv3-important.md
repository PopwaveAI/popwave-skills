# Lv3 · 重要配角 — 模板

> 适用：贯穿 2+ 幕且有独立动机的配角、主角的长期伙伴/导师/宿敌/恋人。
> 颗粒度：Lv2 全部 + 驱力/叙事功能/弧线/登场表 + 可识别角色声音 | 有自己的弧线，跨幕存在。

---

## 设计目的

> 承载独立动机，跨幕存在，形成次级弧线。读者对这个角色有情感期待——角色自己的选择会产生有分量的后果。

## 叙事能力

> 与主角产生深度互动（伙伴/导师/恋人/宿敌）；承载读者次级情感投射（安心/心疼/愤怒）；为关键情节提供不可替代的推动力。

## 四、叙事功能 ← 新增

### 角色在故事中做什么（可复选）

```
□ 情感锚点 — 承载读者对特定情感的投射（安心/心疼/愤怒/…）
□ 冲突放大 — 通过与主角的矛盾推动剧情
□ 信息管道 — 向读者/主角释放关键信息
□ 镜像/对照 — 与主角形成对照，凸显主角的某个面向
□ 催化剂 — 推动主角做出决策，自己不跟进
□ 场面注入 — 提供特定气氛（幽默/紧张/温暖/…）
```

## 七、说法风格 ← 新增

> Lv3 角色必须有稳定“角色声音”。读者不看名字，也能从 2-3 句对白里大致认出是谁。

| 字段 | 填写 |
|:-----|:------|
| 基础语气 | 冷 / 热 / 稳 / 讥诮 / 温和 / 强压情绪 / 其他 |
| 节奏 | 慢条斯理 / 利落 / 跳跃 / 压一句放一句 |
| 常用表达 | 2-4 个高频词、称呼、句尾习惯 |
| 回避表达 | 这个角色基本不会说什么 |
| 对不同对象的口气差 | 对主角 / 对敌人 / 对亲近者的区别 |
| 情绪失控时的变形 | 愤怒、恐惧、心虚时说话会怎么变 |
| 潜台词习惯 | 爱直说 / 爱留半句 / 爱先刺后护 / 爱玩笑掩饰 |
| 示例台词 | 3 句：日常、冲突、情感时刻各 1 句 |

## 最小样例

```markdown
## 七、说法风格
- 基础语气：嘴碎、带点调侃，但关键时刻认真
- 节奏：平时快，遇到危险会突然压低放慢
- 常用表达：你可拉倒吧、先别急、我说真的
- 回避表达：不会讲大义凛然的空话
- 对不同对象的口气差：对主角松弛；对敌人赔笑里带刺；对亲近者更直接
- 情绪失控时的变形：越怕越爱多说两句掩饰
- 潜台词习惯：先吐槽，再把真正关心的话塞进去
- 示例台词：
  - “你可拉倒吧，这事一看就没那么简单。”
  - “我先过去赔个笑脸，你别这会儿跟他硬顶。”
  - “我嘴上损你归损你，真出事我不可能不管。”
```
