# Lv3 · 重要配角 — 模板

> 适用：贯穿 2+ 幕且有独立动机的配角、主角的长期伙伴/导师/宿敌/恋人。
> 颗粒度：Lv2 全部 + 4 维 | 有自己的弧线，跨幕存在。

---

## 设计目的

> 承载独立动机，跨幕存在，形成次级弧线。读者对这个角色有情感期待——角色自己的选择会产生有分量的后果。

## 叙事能力

> 与主角产生深度互动（伙伴/导师/恋人/宿敌）；承载读者次级情感投射（安心/心疼/愤怒）；为关键情节提供不可替代的推动力。

## 四、叙事功能 ← 新增

### 角色在故事中做什么（可复选）

```
□ 情感锚点 — 承载读者对特定情感的投射（安心/心疼/愤怒/…）
□ 冲突放大 — 通过与主角的矛盾推动剧情
□ 信息管道 — 向读者/主角释放关键信息
□ 镜像/对照 — 与主角形成对照，凸显主角的某个面向
□ 催化剂 — 推动主角做出决策，自己不跟进
□ 场面注入 — 提供特定气氛（幽默/紧张/温暖/…）
```

## 五、弧线 ← 新增

### 弧线类型

```
□ 成长弧  — 从不足到完整
□ 堕落弧  — 从光明到黑暗
□ 转变弧  — 从身份A到身份B
□ 揭露弧  — 本质不变，读者逐渐认识其真实面貌
```
