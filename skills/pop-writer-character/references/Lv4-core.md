# Lv4 · 主角/核心对立 — 模板

> 适用：故事主角、主要反派、最终对手。贯穿全文的灵魂。
> 颗粒度：Lv3 全部 + 心理/四层驱力/能力/演化/网络/对白圣经 | 每轮动笔前建议重新审读，确保决策一致性。

---

## 设计目的

> 贯穿全文的灵魂。承载故事核心主题，读者通过这个角色体验整个世界的意义。

## 叙事能力

> 驱动全篇情节走向——角色弧线 = 故事主梗；承载读者全部情感投射；所有关键转折由这个角色的决策触发；是读者的最终认同锚点。

## 四、驱力（四层完整） ← Lv3 升级

| 层级 | 内容 | 故事中何时显露 |
|:-----|:------|:--------------|
| 表层目标 | 角色自己认为想要的 | 开局即显 |
| 深层需要 | 角色真正需要的（与表层可能冲突） | 故事后半才会意识 |
| 黑暗冲动 | 角色最坏情况下可能做的事 | 压力阈值触发时浮现 |
| 恐惧 | 角色最害怕发生的 | 伏笔阶段反复暗示 |

### 驱力冲突

```
哪个驱力层级之间的张力最大？  [如：表层目标与深层需要背道而驰]
这种冲突在故事中以什么形式出现？ [如：角色嘴上说想回家，行为上一直在往前走]
```

## 六、演化 ← 新增

### 弧线阶段（最少 3 个阶段，推荐 4-5）

```
阶段 1：[阶段名]
  起止章节/幕：
  触发事件：
  前心态：
  后心态：
  关键决策（这次选择定义了角色）：
  表象变化 vs 本质变化：

阶段 2：[阶段名]
  ...
```

### 弧线类型（可复选）

```
□ 成长弧  — 从不足到完整（主角经典弧）
□ 堕落弧  — 从光明到黑暗（悲剧/反派弧）
□ 转变弧  — 从 A 到 B 的横向转变（身份/信仰/阵营转换）
□ 揭露弧  — 角色本质不变，但读者逐渐认识其真实面貌
□ 平线弧  — 角色不改变，而是改变周围世界（如完美英雄激励他人）
```

### 不可逆点清单

```
事件 1（约 1/4 处）：
  - 这个事件后角色无法回到原本的生活
  - 故事前 vs 后的心态对比

事件 2（约 1/2 处）：
  - 角色的核心认知被颠覆
  - 旧的驱动力在此失效

事件 3（约 3/4 处）：
  - 最黑暗时刻 / 伪胜利
  - 角色被迫面对最不想面对的事
```

## 八、说法风格 ← 新增

> Lv4 角色必须形成“对白圣经”。后续 prose/chapter 看到这一段，应能稳定写出同一个人的声音。

| 字段 | 填写 | 说明 |
|:-----|:-----|:-----|

| 日常语气 | | 放松状态怎么说 |
| 压力语气 | | 危机、愤怒、恐惧时怎么说 |
| 对不同对象的分层 | | 对弱者 / 同伴 / 爱人 / 敌人 / 上位者分别怎么说 |
| 句式偏好 | | 长短句、停顿、反问、命令、比喻、绕弯等 |

## 剧情驱动接口（★NEW — 供 plot + chapter 消费）

> 角色卡不只是档案。这个段把角色的驱力/缺陷/决策逻辑提炼为"剧情驱动器"，让 plot 和 chapter 可以直接消费。

### 核心驱动力映射（对齐剧情库 L3 卡）

| 维度 | 内容 |
|:-----|:------|
| 承压者/行动者 | {角色名}（{定位}），{起点状态} |
| 压力源 | {来自世界宪法的压力/诱惑/缺口} |
| 行动方向 | {角色决策驱动的行动链条} |
| 失败代价 | {如果失败会失去什么} |
| 终局变化 | {从X→Y的身份/力量/关系跃迁} |

### 驱动的剧情线清单

| 剧情线类型 | 线名（预估） | 驱动方式 | 与其他线的咬合点 |
|:---|:---|:---|:---|
| 主线 | {如：成长线} | {角色的什么决策驱动这条线} | {与哪条线在哪个节点交汇} |
| 副线 | {如：情感线} | {...} | {...} |

### 决策转折点预判（★chapter Step 2 直接消费）

| 转折点 | 触发事件 | 角色决策 | 代价 | 不可逆后果 | 戏剧类型 | 角色特异性 |
|:---|:---|:---|:---|:---|:---|:---|
| 第一个不可逆点（约1/4处） | {事件} | {决策} | {失去什么} | {回不去什么} | 最可能/最戏剧 | {为什么只有这个角色会做这个决策} |
| 核心认知颠覆（约1/2处） | {事件} | {决策} | {代价} | {后果} | 最可能/最戏剧 | {特异性} |
| 最黑暗时刻（约3/4处） | {事件} | {决策} | {代价} | {后果} | 最可能/最戏剧 | {特异性} |

> **戏剧类型说明：** "最可能决策"= 角色基于性格最可能做出的选择；"最戏剧决策"= 出乎意料但合乎逻辑的选择。3个转折点中至少1个必须是"最戏剧"。
> **角色特异性说明：** 换一个角色不会做同样的决策。如果任何角色都会做这个决策 → 决策无特异性 → 退回重新推演。

### 与其他角色的决策化学反应

| 对手角色 | 决策冲突点 | 冲突结果 | 对剧情的影响 |
|:---|:---|:---|:---|
| {角色B} | {两人的决策在哪里冲突} | {谁胜谁负或双输} | {推动什么剧情} |

## 最小样例

```markdown
## 八、说法风格
- 核心声线：冷静克制，先判断后开口
- 日常语气：话不多，偏陈述句
- 压力语气：越危险越平静，句子更短
- 对不同对象的分层：对弱者收着锋芒；对同伴给结论；对敌人不废话；对上位者表面克制但不卑
- 句式偏好：先结论，后补推理；少感叹号；少空话
- 高频词与称呼：先别急、等一下、如果是这样、你确定？
- 禁用表达：不会满嘴热血口号，也不会无缘无故抒情
- 价值观外露方式：不直接喊原则，而是在选择里表现“先保命，再追真相”
- 谎言方式：撒谎时会说得更完整、更像推演
- 情绪失控的破音点：涉及亲人死亡真相时，语速会突然变快
- 独白风格：逻辑化、自我盘问式
- 示例台词：
  - “先别急，把你刚才看到的顺序再说一遍。”
  - “现在进去不是勇敢，是送死。”
  - “你可以骗别人，但别拿这套来试我。”
  - “如果他们真是因这件事而死，我就不可能停在这里。”
```
