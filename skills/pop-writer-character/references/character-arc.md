# 角色弧线指南

> 供消费 skill 在规划角色弧线时参考。简化版——完整方法论见写作理论书籍。

---

## 一、弧线必备元素

每一条角色弧线至少需要：

| 元素 | 定义 | 如果没有会怎样 |
|:-----|:------|:--------------|
| 渴望 | 角色想要什么（表层+深层） | 角色没有方向，读者不知道他要干嘛 |
| 缺陷 | 角色有什么盲点/弱点 | 角色没有成长空间，从头到尾一样 |
| 机会 | 一个事件让角色可以离开现状 | 故事不会真正开始 |
| 代价 | 每个选择都要付出一些东西 | 角色决策没有分量 |
| 不可逆 | 某个事件后回不去了 | 角色弧线可以在任意点终止 |

## 二、常见弧线模式

### 成长弧（最常见）

```
起点：[不足] → 抗拒改变 → 接触[机会] → 尝试但失败 → 
重新调整 → 真正改变 → 终点：[克服缺陷，完成渴望]
```

### 堕落弧

```
起点：[有光明面] → 遭遇[压力/诱惑] → 开始妥协 → 
跨越红线 → 一错再错 → 终点：[失去原本的核心品质]
```

### 转变弧

```
起点：[身份 A] → [事件]使 A 不再可行 → 探索 B/C/D → 
选择 B → 适应 B 的规则 → 终点：[成为身份 B]
```

### 揭露弧

```
起点：[表象] → 读者看到 [暗示与表象矛盾] → 关键曝光事件 → 
角色展示隐藏面 → 终点：[读者认知完全重塑]
```

## 三、弧线密度建议

| 弧线级别 | 跨章节范围 | 每个阶段的最小字数/章节数 |
|:---------|:----------|:------------------------|
| 全书级（主角） | 全卷 | 每 15-25 章一个阶段切换 |
| 卷级（重要配角） | 1-2 卷 | 每卷 1-2 个阶段 |
| 幕级（功能角色） | 1 幕 | 幕内完成，幕结束即闭环 |
| 场景级（一次性） | 1 场景 | 无弧线 |

## 四、弧线与情节的关系

```
角色弧线 → 角色在每个阶段会做什么决策
情节线   → 这些决策导致的后果

两者关系：
  - 角色弧线应该驱动情节，而不是被情节推着走
  - 情节事件应该挑战角色的缺陷，使其不得不改变
  - 第 3 幕的不可逆点 = 角色核心认知被颠覆的时刻
```

## 五、弧线与说法风格的关系

> 角色不是只在“做什么”上变化，也会在“怎么说”上变化。

| 弧线阶段 | 说法风格应该怎么变 |
|:---------|:-------------------|
| 起点 | 最稳定、最日常的声线，作为后续对比基线 |
| 第一次受挫 | 句子开始变短 / 变快 / 变绕，露出压力下的真实习惯 |
| 中段转向 | 对不同对象的口气开始分化，显示关系变化 |
| 不可逆点 | 出现一次明显“失声”或“破音点”，让读者听见角色真的变了 |
| 终点 | 形成新稳定声线，和起点保持可对照差异 |

**使用纪律：**

- `Lv3` 以上角色，弧线变化时应同步检查 `说法风格`
- 不允许“剧情说角色变了，但对白还是老样子”
- `Lv4` 主角/核心对立，至少要能写出**起点声线**与**终态声线**的差别

## 六、决策驱动剧情（★NEW）

> 第四节说"角色弧线应该驱动情节"。本节给出具体机制。

### 决策驱动 vs 情节推动

| 模式 | 特征 | 效果 |
|:---|:---|:---|
| 情节推动（❌当前默认） | 事件发生→角色被动反应 | 角色像弹珠被事件弹来弹去 |
| 决策驱动（✅目标） | 角色做决策→决策导致后果→后果构成情节 | 角色是故事的原动力 |

### 决策设计四问

每个关键剧情节点应能回答：
1. **谁的决策？** — 不是"事件发生了"，而是"某人决定了某事"
2. **为什么？** — 追溯到角色卡的驱力（表层目标/深层需要）或核心缺陷
3. **代价是什么？** — 没有代价的决策没有戏剧分量
4. **不可逆后果？** — 后果如何驱动下一条剧情线

### 角色卡与剧情的接口

角色卡（Lv4）的以下字段是剧情驱动器：
- **驱力四层**（表层目标/深层需要/黑暗冲动/恐惧）→ 渴望 → 驱动主动行动 → 产生主动剧情线
- **核心缺陷** → 导致错误决策 → 产生冲突和转折
- **决策逻辑**（决策阈值）→ 什么事件打破惯常模式 → 转折点设计
- **驱力冲突**（表层 vs 深层）→ 驱动角色弧线 → 成长/堕落/转变

> plot skill 的角色压力测试（Step 0）直接消费这些字段。character skill 产出角色卡时，驱力和缺陷字段必须有具体内容（不允许"待定"）。

### 配角驱动副线

配角（Lv3）不只是"在场的人"。每个关键配角应驱动至少一条副线：
- 配角的渴望/缺陷 → 产生与主线交叉或平行的副线
- 配角的决策 → 在子线咬合点制造化学反应
- 参考剧情库：歌莉娅（配角）驱动情感联盟线，薇薇安（配角）驱动血脉觉醒线——配角线与主线咬合产生双倍满足
