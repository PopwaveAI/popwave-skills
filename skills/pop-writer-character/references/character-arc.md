# 角色弧线指南

> 供消费 skill 在规划角色弧线时参考。简化版——完整方法论见写作理论书籍。

---

## 一、弧线必备元素

每一条角色弧线至少需要：

| 元素 | 定义 | 如果没有会怎样 |
|:-----|:------|:--------------|
| 渴望 | 角色想要什么（表层+深层） | 角色没有方向，读者不知道他要干嘛 |
| 缺陷 | 角色有什么盲点/弱点 | 角色没有成长空间，从头到尾一样 |
| 机会 | 一个事件让角色可以离开现状 | 故事不会真正开始 |
| 代价 | 每个选择都要付出一些东西 | 角色决策没有分量 |
| 不可逆 | 某个事件后回不去了 | 角色弧线可以在任意点终止 |

## 二、常见弧线模式

### 成长弧（最常见）

```
起点：[不足] → 抗拒改变 → 接触[机会] → 尝试但失败 → 
重新调整 → 真正改变 → 终点：[克服缺陷，完成渴望]
```

### 堕落弧

```
起点：[有光明面] → 遭遇[压力/诱惑] → 开始妥协 → 
跨越红线 → 一错再错 → 终点：[失去原本的核心品质]
```

### 转变弧

```
起点：[身份 A] → [事件]使 A 不再可行 → 探索 B/C/D → 
选择 B → 适应 B 的规则 → 终点：[成为身份 B]
```

### 揭露弧

```
起点：[表象] → 读者看到 [暗示与表象矛盾] → 关键曝光事件 → 
角色展示隐藏面 → 终点：[读者认知完全重塑]
```

## 三、弧线密度建议

| 弧线级别 | 跨章节范围 | 每个阶段的最小字数/章节数 |
|:---------|:----------|:------------------------|
| 全书级（主角） | 全卷 | 每 15-25 章一个阶段切换 |
| 卷级（重要配角） | 1-2 卷 | 每卷 1-2 个阶段 |
| 幕级（功能角色） | 1 幕 | 幕内完成，幕结束即闭环 |
| 场景级（一次性） | 1 场景 | 无弧线 |

## 四、弧线与情节的关系

```
角色弧线 → 角色在每个阶段会做什么决策
情节线   → 这些决策导致的后果

两者关系：
  - 角色弧线应该驱动情节，而不是被情节推着走
  - 情节事件应该挑战角色的缺陷，使其不得不改变
  - 第 3 幕的不可逆点 = 角色核心认知被颠覆的时刻
```
