# pop-writer-creative · 创意研究+碰撞引擎 v6.1.0

> 从用户模糊想法到高质量立项PRD+素材储备池的**唯一入口**。核心理念：海明威冰山原则——深度调研是水下的八分之七，PRD是水面的八分之一。没收集够信息不该开PRD。三种调用模式：①新书（全流程）②素材注入（储备卡产出）③主动丰富（缺口补采）

| # | 红线 |
|:-:|:-----|

## 储备池数量级约束

> **储备卡对应具体剧情线，不是主线大类。** plot 阶段的三主线大类（世界大事件/主角成长/主角行动）全书固定；每卷每个主线大类下动态激活 2-3 条具体剧情线。每条具体剧情线对应 ≥1 张剧情储备卡，且对应 plot 的 L3 剧情线文档。支线无主线大类归属，直接是具体剧情线级别。

### 首版储备池下限（被 creative 调用时）

> 首版储备池需要覆盖卷一可能激活的具体剧情线 + 支线 + 设定。不是全书所有具体剧情线的全集——后续卷的具体剧情线可以通过"主动丰富"模式逐步补充。

| 类型 | 采集级别 | 下限 | 对应关系 |
|:-----|:---------|:-----|:---------|
| 剧情储备卡（主线级） | 主线剧情级 | ≥6 张 | 卷一每个主线大类下 ≥2 条具体剧情线，每条 ≥1 张卡 |
| 剧情储备卡（支线级） | 支线剧情级 | ≥3 张 | 支线级剧情线（无主线大类归属），每条 ≥1 张卡 |
| 设定储备卡 | 背景素材级 | ≥3 张 | 世界核心设定的 3 个维度 |
| **总量下限** | | **≥12 张** | |

### 主动丰富模式检查清单

| 检查项 | 不满足时动作 |
|:-------|:------------|
| 卷一每个主线大类下具体剧情线储备卡 ≥2 张 | 补采该大类下的素材 |
| 每个 PRD 元爽点至少有 1 张卡能落地 | 补采对应元爽点的素材 |
| 支线级储备卡 ≥3 张 | 补采支线级素材 |
| 设定储备卡 ≥3 张 | 补采设定级素材 |
| 剧情卡 vs 设定卡比例 ≥ 2:1 | 补采剧情级素材 |
| 新卷启动前具体剧情线储备卡覆盖不足 | 补采该卷计划激活的具体剧情线对应素材 |

### 单张卡质量下限

| 卡类型 | 必须包含 |
|:-------|:---------|
| 剧情储备卡（主线级） | 元卖点 + ≥3方参演势力 + 8~20节点剧情线发展链条 + 绝对之恶(靶子) |
| 剧情储备卡（支线级） | 元卖点 + ≥2方参演势力 + 5~15节点剧情线发展链条 |
| 设定储备卡 | 设定内核 + 兼容性分析 + ≥2个可衍生冲突空间 |
