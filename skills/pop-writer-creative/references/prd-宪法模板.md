# PRD 宪法模板

> Step 8 合成 PRD 时的最终结构模板。推演语气（断言式），宪法层抽象，无具体名/数值/分幕/转译规则。

```
# PRD · {书名}
> 一句话定调

## 〇、核心故事
{一段150-200字的故事梗概。回答「这本书讲了什么」，让人读完能在脑子里放电影。
 不解释设定，只呈现冲突弧线。结尾锚定一个悬念或一个画面。}

---

## 一、爽点锚点

### 锚点①：{名称}
{2-3句爽点描述 —— 读者为什么爽？心理机制是什么？}
- **特征参数**：{触发条件 / 强度峰值 / 衰减曲线}

### 锚点②：{名称}
...

### 锚点③：{名称}
...

---

## 二、核心卖点

### 1. 设定卖点
{1-2条，每条2-4句。这个设定为什么独特？}

### 2. 世界观/理论体系卖点
{1-2条。世界观层面有什么独到洞见？}

### 3. 剧情卖点
{1-2条。剧情上要写什么？读者期待的体验是什么？}

### 4. 主角/金手指卖点
{1-2条。主角有什么吸引力？金手指设计哲学？}

### 5. 全书级卖点
{1-2条。整本书阅读体验有什么独特之处？}

---

## 三、美学基因 + 命名体系

### 美学基因
| 基因 | 定义 | 排他性约束 |
|:----|:-----|:----------|
| 基因1 | {一句话} | {越界即丧失} |
| 基因2 | {一句话} | {越界即丧失} |

### 从基因到世界质感
{物理质感（视觉/听觉/触觉/嗅觉）+ 空间感受。2-4段}

### 从基因到力量美学
{力量外观 + 变强的代价 + 力量的来源。2-3段}

### 从基因到情感光谱
{禁用情感 + 情感节奏 + 情感的特殊质地。2-3段}

### 命名语汇方向
{世界命名语汇 / 力量命名语汇 / 拒绝的语汇 / 一条命名铁律}

### 视觉锚点
{一段150-200字的具体场景片段。把前面全部美学推导收拢为一个可放电影的画面。
 融合质感/力量/情感/命名，但不出现具体名。}

---

## 四、用户画像

- **核心读者**：{谁}——{他们的核心诉求}
- **边缘读者**：{谁}——{为什么也会看}
- **不服务**：{谁}——{明确拒绝的读者类型}

---

## 五、世界宪法（4-6条）

### 第一条：{规则名称}
{1-2句规则描述}
**来源**：{从哪个维度推导} | **违反后果**：{如果这条被打破，书的什么体验会崩塌}

### 第二条：{规则名称}
...

---

## 六、跨域素材指引

- {方向1}：{1-2句指引。只说去哪个领域找什么，不说具体怎么用}
- {方向2}：...
- {方向3}：...
{3-5个方向}

---

## 七、篇幅规划

{目标字数区间 / 预估卷数 / 每章大致爽点密度}

---

## 八、核心承诺 + 防崩阈值

每条元爽点 → 一条对读者的承诺。每条配可量化的防崩信号。

- **承诺①（{对应爽点}）**：{体验描述}
  **防崩**：{什么信号出现说明这条承诺在失效}

- **承诺②**：...
  **防崩**：...

---

## 九、不可妥协 DNA + 边界

{4-6条打死不能改的DNA}
{3-5条硬边界}
```

## 禁止出现在 PRD 中的内容

| 条目 | 原因 | 归到哪 |
|:-----|:-----|:-------|
| 具体角色名 | PRD 是宪法，不是角色设计 | L1 角色设计 |
| 具体地名/势力名 | PRD 定义语汇方向，不定义具体名 | L1 命名设计 |
| 属性数值 | PRD 定义「属性阈值突破触发质变」，不写力量12/敏捷19 | L1 数值设计 |
| 分幕/章节划分 | PRD 定义规模（3-5卷），不写具体幕数 | plot 卷纲 |
| 金手指具体机制 | PRD 定义设计哲学，不写冷却时间/消耗值 | L1 金手指设计 |
| 详细转译规则 | PRD 指引方向，方案留给 L1 | L1 素材设计 |
| 创意溯源交叉表 | PRD「去搜 X 领域」是指引，交叉表是方案 | L1 素材分析 |
| 章节证据引用 | PRD 是推演语气，不是复盘 | 拆书复盘文档 |
