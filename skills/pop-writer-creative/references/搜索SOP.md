## 一、关键词提取规则

> 搜索关键词从 Step 1 Phase 3 的**特征参数表**提取，不从类型模板提取。

### 提取公式

```
搜索关键词 = {元爽点名} + {L1参数} + {L2参数} + {赛道关键词} + {搜索目标关键词}
```

### 示例

| 特征参数表 | ❌ 错误关键词 | ✅ 正确关键词 |
|:---|:---|:---|
| 信息差 L1=感知破绽 L2=实时 L3=有反噬风险 L4=战斗+权谋 | `系统流修仙小说 数据面板` | `感知破绽 代价 实时信息差 小说 战斗 权谋` |
| 阶级革命 L1=制度性压迫 L2=颠覆规则 L3=知识封锁+资源垄断 L4=个人→组织 | `底层逆袭修仙小说` | `知识垄断制度 底层颠覆 历史案例 起义组织结构` |
| 绝境反杀 L1=自身选择 L2=有代价险胜 L3=每幕大绝境 L4=悲壮 | `绝境反杀小说推荐` | `绝境反杀 代价险胜 悲壮叙事 小说 每幕高潮` |

### 关键词数量要求

- 每路搜索至少 **2 个特征参数**作为过滤条件
- 每路搜索关键词总数 **4-7 个**（太少=泛搜噪音大，太多=结果过少）
- 特征参数无法确定时，标记为该路的搜索主攻方向

## 二、三路搜索路径

### 搜索 A：同类作品世界观结构

| 维度 | 要求 |
|:-----|:-----|
| **目标** | 找到 3-5 本同类成功作品，提取世界观结构 |
| **关键词模板** | `{类型} 小说 世界观 结构 设定 亮点` + `{L1参数} {L4参数}` |
| **最低数量** | **5 本**作品（低于 5 本 = 搜索不足，需迭代） |
| **每本记录** | 作品名 + 世界观结构（单世界/双世界/多世界）+ 力量体系设计 + 核心冲突 + 优点 + 缺点 |
| **筛选标准** | ① 完结或 100 万字以上（验证了结构可持续性）② 同赛道但不同金手指（避免直接抄）③ 至少 2 本有拆书/分析文章可参考 |

### 搜索 B：历史/神话/社会结构素材

| 维度 | 要求 |
|:-----|:-----|
| **目标** | 找到 3-5 个可用的历史/社会结构原型，作为世界观骨架 |
| **关键词模板** | 从元爽点出发反推：`{L3参数} 历史案例` / `{L1参数} 社会结构` / `{L4参数} 神话体系` |
| **最低数量** | **3 个**原型（低于 3 个 = 搜索不足） |
| **每个记录** | 原型名 + 核心机制（压迫怎么运作/信息差怎么形成/权力怎么转移）+ 可提取的结构骨架 + 与本书爽点的关联点 |
| **筛选标准** | ① 真实历史/神话（不是另一本小说的设定）② 有足够的结构复杂度（不是简单的好人打坏人）③ 可转译到本书赛道 |

**元爽点→历史素材方向速查**：

| 元爽点 | 搜索方向 |
|:-------|:---------|
| 信息差碾压 | 间谍战/冷战情报网/三十六计/博弈论经典案例 |
| 阶级革命 | 黄巾起义/太平道/法国大革命/奴隶起义/农民战争 |
| 绝境反杀 | 以少胜多战役/绝境翻盘历史/背水一战案例 |
| 身份跃迁 | 平民称帝/阶层流动史/科举制度/冒名顶替史 |
| 世界观震撼 | 大发现时代/科学革命/宇宙学发现/文明崩溃史 |
| 智斗博弈 | 冷战博弈/三国外交/战国合纵连横/宫廷政治 |
| 情感羁绊 | 历史上的著名羁绊/悲剧友情/师徒传统 |

### 搜索 C：金手指原型

| 维度 | 要求 |
|:-----|:-----|
| **目标** | 提取 3-5 种可用的金手指结构，分析优缺点 |
| **关键词模板** | `{L1参数} {L3参数} 金手指 能力 系统` + `{类型} 小说` |
| **最低数量** | **3 种**结构（低于 3 种 = 搜索不足） |
| **每种记录** | 金手指名 + 核心机制 + 代价规则 + 优点 + 缺点 + 契合度评估（与本书特征参数的匹配度） |
| **筛选标准** | ① 有明确的代价/限制（不是无脑开挂）② 与特征参数 L1-L4 匹配度 ≥70% ③ 至少 1 种是跨赛道创新（不是同赛道已有金手指的翻版） |

## 四、迭代搜索策略

> 第一轮搜索不达标时，按以下策略补充。

| 不达标情况 | 补充策略 |
|:-----------|:---------|
| A 不足 5 本 | 扩大赛道范围（如"修仙"→"东方玄幻"→"奇幻"），或搜索 `{类型} 小说推荐 排行` 获取更多候选 |
| B 不足 3 个 | 换元爽点角度搜索（如"阶级革命"搜不到→换"制度性压迫"→换"社会不公历史"） |
| C 不足 3 种 | 跨赛道搜索金手指（如修仙金手指不够→搜科幻/游戏/都市的金手指结构，提取可转译的机制） |
| 搜索结果质量低（全是推荐榜单无分析） | 加关键词 `分析` `拆解` `设定` `世界观构建` 获取深度内容 |
| 搜索结果与特征参数关联弱 | 回到特征参数表，重新提取关键词，确保每路至少 2 个特征参数 |

**最大迭代轮次**：2 轮。第 3 轮仍不达标 → 标注"搜索不足，以下选项基于有限素材推导"，继续合成（不阻塞流程）。

## 五、质量自检

搜索完成后，合成选项前，逐条自检：

| 检查项 | 标准 |
|:-------|:-----|
| 关键词来源 | 每路搜索关键词从特征参数表提取，不是从类型模板硬编码？ |
| 数量达标 | A≥5本、B≥3个、C≥3种？ |
| 质量达标 | 每条记录有结构分析，不是只有作品名/原型名？ |
| 筛选标准 | 筛选标准每条都检查了？ |
| 沉淀完整 | 搜索沉淀格式填写完整，供 §2.2 引用？ |
| trope-library | §2.0 套路参考清单已纳入搜索沉淀？ |
