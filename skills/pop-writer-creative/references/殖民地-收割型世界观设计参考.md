# 殖民地/收割型世界观设计参考

> 适用场景：当作者设定"低武/武侠世界实际上是上界仙侠/修真世界的培养皿/牧场/殖民地"时。
> 这是一个**跨题材的通用世界观模板**，核心特征是"所有向上通道都是陷阱"。

---

## 一、核心模式

```
上界（仙侠/修真/神界）
  │
  ├─ 投放"有毒功法" → 下界修炼者主动成为更好的"材料"
  ├─ 设计"虚假奇遇" → 筛选出资质最好的个体定向诱导
  ├─ 定期"收割" → 以飞升/天劫为名义，将成熟个体取走
  └─ 维持"信息隔离" → 下界永远不知道真相
        │
        ▼
下界（武侠/低魔世界）
  ├─ 所有顶级功法都有"后门"（炼成后自动成为药人/兵器/宝丹）
  ├─ 所有秘境/传承都是"诱饵"（标注每头肥羊的预定收割者）
  ├─ 突破一定境界=触发"收割标记"（上界大能收到通知）
  └─ 历史被"修剪"（每次有人接近真相，就有"天灾"清洗文明）
```

## 二、有毒功法的常见设计

| 类型 | 功法承诺 | 真实代价 | 最终产物 |
|:-----|:---------|:---------|:---------|
| **药人型** | 速成、威力大、进境快 | 修炼者在丹田/经脉中凝练出特殊"药性" | 上界大能的延寿丹药 |
| **兵器型** | 炼体、金刚不坏、肉身成圣 | 身体逐渐金属化/晶体化，失去自我意识 | 上界的神兵胚胎 |
| **宝丹型** | 金丹大道、元婴出窍 | 元婴凝练的全是"杂质/毒素"，丹田特殊化 | 上界的毒丹原料 |
| **灵魂型** | 精神力暴涨、神识千里 | 灵魂结构被改写成可拆解格式 | 上界的魂器/阵眼 |
| **血脉型** | 返祖觉醒、神兽血脉 | 血脉被"定向激活"到特定品种 | 上界的血种/祭品 |

### 关键设计陷阱

1. **不能全是假货** — 有毒功法在初期的确有效（甚至远超正常功法），否则没人会上当
2. **不能永久封印真相** — 主角必须在某个阶段发现"功法不对"，读者等的就是这个反转
3. **必须有人通过假功法"成功"过** — 历史上有人飞升了，而且飞升后"风光无限"（实际被暗中处理，但有假象维持）

## 三、虚假奇遇的设计谱系

| 奇遇类型 | 表面设计 | 真实目的 | 主角可能的利用方式 |
|:---------|:---------|:---------|:------------------|
| 前辈洞府 | 获得大能传承 | 植入收割标记/功法补丁 | 只取资源不修功法 |
| 秘境试炼 | 淬炼肉身神魂 | 筛选高资质个体+注入追踪印记 | 利用秘境规则反向获利 |
| 天降机缘 | 天地灵物/神兵认主 | 绑定灵魂后无法剥离，成定时炸弹 | 用上界知识识别并规避 |
| 上古遗迹 | 发现失传的顶阶功法 | 功法本身就是收割工具的一个版本 | 拆解功法逆向工程 |
| 名师收徒 | 被上界大佬看中收为弟子 | 预定收割名额，近距离监控成长 | 用死亡复活机制骗取情报 |

## 四、收割体系的结构层次

### 4.1 单层收割 （本项目采用）

```
上界（仙侠大能） → 直接管理 → 下界（武侠世界）
                             │
                             ├─ 通过"代理人组织"维持秩序
                             ├─ 通过"天劫/飞升"筛选成熟个体
                             └─ 通过"宗门传承"控制功法流向
```

### 4.2 多层收割（进阶变体）

```
上界（仙界） → 中界（修真界/灵界） → 下界（武侠/凡界）
                                          │
        每层都是上一层的殖民地            │
        每层的"飞升"都是送入虎口          │
        每一层对此的认知不同              │
```

### 4.3 竞争性收割（最黑暗变体）

```
多个上界势力 → 下界（公共牧场）
     │              │
     │        每个势力投放下界自己的
     │        功法/传承/代理人
     │              │
     ▼              ▼
  势力间互相竞争"收成份额"
  下界修炼者=可重复使用的牧场
  不同功法的冲突=势力间的代理人战争
```

## 五、此类设计的标准节奏

| 阶段 | 读者认知 | 剧透级真相 |
|:-----|:---------|:----------|
| 前期（卷1-2） | 普通武侠世界，有江湖恩怨、宗派斗争 | — |
| 中期（卷3-4） | 发现武功巅峰后面还有"武道之上"，似乎不简单 | 主角发现功法的异常 |
| 转折（卷4末-卷5） | 真相揭露——整个世界的武功体系是被人设计的 | 收割体系的冰山一角 |
| 后期（卷6+） | 从上界的耗材开始反杀 | 主角利用双界信息差构建自己的体系 |

## 六、常见陷阱与规避

| 陷阱 | 问题 | 解决方法 |
|:-----|:-----|:---------|
| **真相揭露过早** | 卷1就发现"功法有毒"，江湖斗争瞬间变低幼 | 前期让主角只感觉"有点不对"但说不清是什么，先解决眼前的生存危机 |
| **上界过于降维打击** | 上界大能无所不知，主角无法发育 | 给上界设定约束：监控成本高、大能沉睡/闭关、下界信号有延迟/失真 |
| **收割者动机空洞** | "上界坏蛋就是坏"——缺乏说服力 | 给出供需逻辑：上界的资源衰退/战争消耗/大道缺陷，导致他们需要下界"原料" |
| **主角开挂太容易** | 穿越两次就破解了千年的骗局 | 设定破解需要多次死亡验证+跨世界知识对照；每个真相都是一条命换来的 |
| **下界战力崩塌** | 既然下界只是牧场，那下界所有角色都不值一提 | 设计"深水区"——有前朝/上古的漏网之鱼、有意识到问题并隐藏起来的远古残留势力 |

## 七、与双界穿越的组合模式

当殖民地设定与"双界来回穿越"组合时（就像本会话的项目）：

```
武侠世界（殖民地） ←→ 仙侠世界（上界）

主角优势：
  ├─ 在仙侠世界看到"上界功法的完整版" → 对比发现武侠世界的功法是"阉割毒版"
  ├─ 在武侠世界拿到某"上古奇遇" → 去仙侠世界查这个奇遇的创造者是谁
  ├─ 在仙侠世界的底层市场买到信息 → 回武侠世界改变自己的命运轨迹
  └─ 在一边死亡 → 从另一边复活（打破收割者的标记链）

核心博弈：
  上界大能以为主角只是"一头比较能跑的肥羊"
  主角在利用"收割者视角"来反向收割收割者的情报
```

## 八、参考案例

| 作品 | 模式类型 | 可借鉴元素 |
|:-----|:---------|:----------|
| 《一世之尊》 | 多世界+背后博弈 | 做减求空、彼岸布局、真实界作为博弈棋盘 |
| 《死人经》 | 黑暗生存+信息不对称 | 斩断关系方法论、多重身份、不靠光环靠算计 |
| 《神墓》 | 太古布局 | 天地是囚笼、修炼体系被刻意扭曲 |
| 《牧神记》 | 虚假的诸神体系 | 神祇收割信仰、功法有毒、底层觉醒 |
