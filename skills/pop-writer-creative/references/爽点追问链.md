# 爽点追问链（7 类常见 + 元模板）

> Step 1 Phase 2b 使用。对每个已识别的元爽点，按对应模板自问，产出 L1-L4/L5 特征参数。
> 不匹配 7 类的走元模板。**不要强行归类。**

---

## ① 信息差类

```
L1 感官/媒介：信息差通过什么渠道传递？
   （看见面板数值 / 推演未来 / 感知破绽 / 听见心声 / 特殊联系 / 其他：____）
L2 时效性：信息差是永久能力 / 限时激活 / 一次性道具？
L3 代价：使用信息差要付出什么？
   （免费 / 消耗资源 / 有反噬风险 / 越级使用会暴露自己）
L4 叙事层：信息差在哪个层面发挥作用？
   （战斗层面→看穿破绽 / 权谋层面→预判博弈 / 世界观层面→知道隐藏真相）
```

## ② 阶级革命类

```
L1 压迫主体：压迫来自制度性的结构 / 个体上位者的恶意 / 两者混合？
L2 反抗方式：主角要推翻制度 / 取代上位者 / 逃离系统 / 还是从根本上颠覆游戏规则？
L3 压迫的具体体现：资源垄断 / 人身控制 / 知识封锁 / 身份歧视 / 暴力威慑？
   （至少写 2 个具体维度——这些就是中期冲突的原材料）
L4 革命路径阶段：个人反抗 → 组织底层 → 动摇制度 → 重新定义规则
   （当前定位在第几阶段？每阶段切换的标志是什么？）
```

## ③ 绝境反杀类

```
L1 绝境制造者：谁把主角逼到绝境？（环境 / 具体敌人 / 自身选择 / 系统规则）
L2 反杀代价：反杀需要付出什么？
   （无代价爽杀 / 有代价险胜 / 不可逆损失——影响读者预期）
L3 绝境频率：绝境的节奏是？（每章小险情 / 每幕大绝境 / 卷级生死关）
L4 读者情绪：绝境时读者的预期是什么？
   （紧张未知 / 悲壮 / 笃定主角会赢 / 希望渺茫）
```

## ④ 身份跃迁类

```
L1 跃迁方向：底层→上层 / 边缘→中心 / 隐匿→显赫 / 被否定→被认可？
L2 驱动力：跃迁靠能力展示 / 身份揭示 / 外部赋予 / 还是自我构建？
L3 节奏：匀速上升 / 突进+平台期 / 每步跃迁伴随什么失去？
L4 社会代价：跃迁后面对什么？（被嫉妒 / 被追杀 / 失去旧关系 / 被迫出卖原则）
```

## ⑤ 世界观震撼类

```
L1 震撼类型：力量阶梯刷新 / 历史真相揭露 / 宇宙规则揭秘 / 社会结构反转？
L2 节奏：每卷一次天花板刷新 / 持续碎片拼图 / 一次性大反转？
L3 主角应对：适应新规则 / 利用新规则 / 还是打破新规则？
L4 情感基调：敬畏 / 恐惧 / 兴奋 / 绝望后希望？
```

## ⑥ 智斗博弈类

```
L1 参与方：双边对抗 / 三角制衡 / 多方混战？（具体几个势力？）
L2 信息分布：完全信息 / 不完全信息 / 存在误导信息？
L3 主角优势：计算能力 / 信息渠道 / 人情筹码 / 预知能力？
L4 赌注级别：个人生死 / 势力存亡 / 世界命运？
```

## ⑦ 情感羁绊类

```
L1 关系类型：兄妹 / 师徒 / 战友 / 恋人 / 亦敌亦友 / 族群归属？
L2 羁绊功能：羁绊在叙事中扮演什么角色？
   （情感锚点「不杀的理由」/ 力量源泉「为TA变强」/ 软肋「被利用」/ 身份定义「我是谁」）
L3 叙事密度：全书贯穿 / 阶段性出场 / 只在关键时刻激活？
L4 张力来源：付出不对等 / 认知差 / 身份对立 / 牺牲风险？
```

## ⑧ 元模板（不匹配 7 类时使用）

当 agent 识别出的元爽点不属于以上 7 类，**不要强行归类**。使用元模板生成自定义追问链：

```
自定义追问链 · {爽点名}

L1 感官入口：这个爽点通过哪种读者体验触达？
   （认知层→信息处理 / 情绪层→情感波动 / 生理层→体感反应 / 混合）
L2 触发条件：什么条件成立时这个爽点会爆发？
   （需要铺垫多久？需要哪些前置要素？）
L3 节奏频率：它的爆发节奏是怎样的？
   （脉冲型→来一下就走 / 累积型→慢慢涨然后释放 / 持续性→全程都在）
L4 升级路径：随着剧情推进，这个爽点的表现形态如何演化？
   （从简陋到复杂？从低频到高频？从单一到复合？）
L5 排他性：这个爽点和哪些其他爽点不兼容？
   （例：极致的智力信息差和极致的暴力爽感经常冲突，因为前者需要读者思考，后者需要读者停止思考）
```

> ⚠️ **元模板存在的意义**：7 类追问链是「常见情况的手册」，不是「爽点的全部可能」。当用户提出不在这 7 类中的爽点，agent 必须用元模板自行生成追问链，而非把用户的爽点硬塞进最接近的类别里——后者会导致信息丢失。
