# 种子展开法 · 系统性创意研究方法论

> **用途**：creative Phase 1 执行。从用户输入的种子元素出发，系统性展开为完整的研究档案。
>
> **替代关系**：替代旧版 `域研究SOP.md`（浅层研究）。域研究SOP 只要求每维 1-2 句、每域 2 次搜索，无法支撑概念合成。种子展开法要求四层穿透 + 切向扩展 + 交叉授粉。
>
> **灵感来源**：雪花写作法（Snowflake Method）的分形展开原则，应用于研究阶段而非写作阶段。雪花法解决"从概念到初稿"的展开，种子展开法解决"从种子到概念"的研究。

---

## 一、核心理念

### 为什么要种子展开

用户给的不是完整故事概念，是**种子**——几个离散的元素或灵感。直接从种子合成概念，等于在没有地基的情况下盖楼。

种子展开法的核心是**分形展开**：每个种子像雪花的第一片晶体，通过系统性辐射，生长出完整的素材网络。

### 种子展开 vs 旧版域研究

| | 旧版域研究SOP | 种子展开法 |
|:---|:---|:---|
| 起点 | creative 识别的域 | 用户输入的种子元素 |
| 深度 | 每维 1-2 句 | 四层穿透（L1-L4） |
| 广度 | 只研究明确提到的域 | 切向扩展发现相关但未提及的话题 |
| 交叉 | 无 | 种子间交叉授粉找连接点 |
| 产出 | 浅层域DNA摘要 | 研究档案（种子展开图+域DNA档案+交叉授粉） |

### 海明威冰山原则

> "冰山在海里移动很庄严宏伟，是因为它只有八分之一露出水面上。" — 海明威

种子展开法就是造冰山的水下部分。用户看到的种子是露出水面的八分之一，种子展开法负责建造水下的八分之七。没有足够的调研和创意，不该开启 PRD。

## 二、五阶段工作流

### Phase 1：种子识别

从用户输入中识别种子元素，并分类。

**种子类型**：

| 类型 | 特征 | 示例 |
|:-----|:-----|:-----|
| 内容创作者 | 具体的创作者/IP/频道 | 牢A、李永乐、半佛仙人 |
| 文化现象 | 社会热点/流行文化/亚文化 | 润人潮、中美对比、内卷 |
| 游戏系统 | 游戏机制/TRPG规则 | DND、克苏鲁、黑魂 |
| 叙事套路 | 类型文学/网文流派 | 系统文、无限流、克苏鲁 |
| 社会议题 | 现实社会问题 | 医疗缺陷、阶级固化、移民困境 |
| 地理空间 | 具体地点/区域 | 北美、丁胖子广场、法拉盛 |
| 历史事件 | 历史上的真实事件 | 义和团、淘金潮、排华法案 |

**识别规则**：
- 用户输入的每个名词都可能是种子
- 一个种子可能同时属于多个类型（如"牢A"既是内容创作者又是文化现象）
- 如果用户只给了 1-2 个种子，主动追问"还有什么你想加入的元素？"
- 种子数量影响研究规模：1-2 个种子→精研模式；3-5 个种子→标准模式；6+ 个种子→广研模式

**产出**：种子清单（种子名 + 类型 + 一句话描述）

### Phase 2：直接研究（每个种子独立展开）

对每个种子执行独立研究，提取其核心特征。

**研究内容**：

| 维度 | 问题 | 搜索策略 |
|:-----|:-----|:---------|
| 是什么 | 这个种子是什么？核心内容/风格/机制 | `"{种子名}"` |
| 怎么运作 | 它怎么运作？有什么规则/模式/结构 | `"{种子名} 机制 规则 原理"` |
| 什么风格 | 它的风格/调性/受众是谁 | `"{种子名} 风格 受众 评价"` |
| 什么影响 | 它的影响/衍生/二创 | `"{种子名} 影响 衍生 二创"` |

**同时查 trope-library**：
- `立项库/` — 有没有类似立项模式
- `设定库/` — 有没有对标书的设定包
- `剧情库/` — 有没有可改建的剧情单元
- `套路库/` — 有没有配套套路

**产出**：每个种子的直接研究摘要（是什么/怎么运作/什么风格/什么影响，各 2-3 句）

### Phase 3：切向扩展（每个种子向外辐射）★核心创新

这是种子展开法区别于旧版域研究的核心。不是只研究用户明确提到的种子，而是**发现相关但未提及的话题**。

**五条辐射路径**：

| 路径 | 策略 | 搜索关键词模板 | 示例（牢A） |
|:-----|:-----|:-------------|:-----------|
| 同文化现象 | 搜索同类型/同风格的文化产物 | `"{种子名} 类似 相关 同类"` | 润人笑话、华人自媒体生态、北美荒诞故事 |
| 同社会议题 | 搜索同一社会议题的其他表现 | `"{种子核心议题} 其他案例"` | 小红书中美大对账、美国医疗缺陷、黑工经济 |
| 同地理空间 | 搜索同一地点的相关文化 | `"{地点} 华人 社区 文化"` | 丁胖子广场、法拉盛、蒙特利公园 |
| 同人群 | 搜索同一人群的其他故事 | `"{人群} 故事 经历"` | 偷渡客、留学生、ABC、非法移民 |
| 同机制 | 搜索同一机制的其他应用 | `"{核心机制} 其他应用 变体"` | 跑团文化、TRPG叙事、属性映射现实 |

**切向发现的质量标准**：
- 每个种子至少辐射 3 条路径
- 每条路径至少发现 1 个切向话题
- 切向话题必须标注与原种子的关系（"同文化现象"/"同社会议题"/etc.）
- 切向话题需要快速验证（1 次搜索确认其存在和相关性）

**产出**：种子展开图（每个种子的辐射树 + 切向话题清单）

### Phase 4：深度域研究（吸收采风SOP）

对 Phase 2-3 识别出的每个域执行四层穿透。按 `references/深度域研究SOP.md` 执行。

**域的识别**：种子 + 切向话题 → 归类为域。例如：
- 牢A + 润人笑话 + 华人自媒体 → "北美华人叙事"域
- 北美底层 + 中美大对账 + 黑工经济 → "中美社会对比"域
- DND + 跑团文化 + 属性映射 → "TRPG系统"域
- 穿越/系统 + 系统文变体 → "系统文"域

**四层穿透**（按深度域研究SOP执行）：
- L1 表面层：这个域有什么？代表作/核心设定/经典元素
- L2 结构层：这个域怎么运转？叙事结构/核心机制/美学基因
- L3 系统层：反映了什么更广泛的文化/历史模式？
- L4 叙事语法层：这个域怎么讲故事？节奏控制/爽点释放/读者预期管理

**产出**：域DNA档案（每个域一份）+ 创意溯源记录

### Phase 5：交叉授粉（种子之间找连接点）

在完成所有种子的独立研究和切向扩展后，寻找种子之间的连接点。

**交叉授粉矩阵**：

| | 种子A | 种子B | 种子C | 种子D |
|:---|:---|:---|:---|:---|
| 种子A | — | A×B 连接点 | A×C 连接点 | A×D 连接点 |
| 种子B | — | — | B×C 连接点 | B×D 连接点 |
| 种子C | — | — | — | C×D 连接点 |
| 种子D | — | — | — | — |

**连接点类型**：

| 类型 | 定义 | 示例 |
|:-----|:-----|:-----|
| 机制映射 | 种子A的机制映射到种子B的世界 | DND属性系统 → 北美社会的信用评分/阶层 |
| 意象叠加 | 种子A的意象与种子B的意象叠加 | 牢A的荒诞叙事 × 系统文的金手指 = 系统给的任务都是荒诞的 |
| 结构借用 | 种子A的结构用于种子B的内容 | 中美大对账的结构 → DND阵营轴的善恶对比 |
| 功能反转 | 种子A的功能在种子B中反转 | DND的"英雄冒险" → 底层人民的"NPC待遇" |

**产出**：交叉授粉图（连接点清单 + 叙事可能性）

## 三、素材分级标准

所有研究产出的素材按以下标准分级：

| 层级 | 含义 | 可用于 | 示例（牢A案例） |
|:-----|:-----|:------|:-------------|
| L1 表面事实 | 是什么/谁/什么时候/在哪 | 世界观细节、场景描写 | 牢A是B站创作者，讲北美底层故事 |
| L2 结构理解 | 怎么运作/什么规则/什么模式 | 设定设计、机制设计 | 牢A的叙事模式：真实事件→荒诞化→中式解读 |
| L3 文化模式 | 反映什么社会现象/文化心理 | 主题设计、冲突设计 | 中美想象落差、润人文化、阶层焦虑 |
| L4 叙事语法 | 怎么讲故事/节奏/爽点释放 | 叙事策略、结构设计 | 黑色幽默节奏：铺垫→荒诞揭示→中式吐槽 |

**分级要求**：
- 每个种子至少达到 L2（结构理解）
- 核心域（支撑≥1卷主线的域）必须达到 L4（叙事语法）
- 辅助域至少达到 L3（文化模式）

## 四、研究档案产出格式

```
研究档案/
├── 种子展开图.md          [Phase 1-3 产出]
│   ├── 种子清单（种子名+类型+描述）
│   ├── 直接研究摘要（每个种子4维度）
│   └── 切向扩展树（每个种子的5条辐射路径+切向话题）
│
├── 域DNA档案-{域名}.md    [Phase 4 产出，每个域一份]
│   ├── 叙事语法（2-3句）
│   ├── 核心机制（2-3句）
│   ├── 美学基因（1-2句）
│   ├── 经典元素库（≥5个）
│   └── 可提取素材清单
│
├── 创意溯源记录.md         [Phase 4 产出]
│   └── 每条素材标记来源域+原型+提取内容+拟用转化手法
│
└── 交叉授粉.md            [Phase 5 产出]
    ├── 交叉授粉矩阵
    └── 连接点清单（类型+叙事可能性）
```

## 五、门禁

| 检查项 | 失败动作 |
|:-------|:---------|
| 种子数量 < 1 | 退回——追问用户"还有什么你想加入的元素？" |
| 每个种子未执行直接研究 | 退回——补完4维度研究 |
| 切向扩展未覆盖3条路径 | 退回——补完辐射路径 |
| 切向扩展未发现任何切向话题 | 退回——换关键词重搜 |
| 域DNA档案未达L2 | 退回——深化到L2 |
| 核心域未达L4 | 退回——补充叙事语法层 |
| 交叉授粉矩阵未填满 | 退回——补完所有种子对 |
| 研究档案不完整 | 退回——补完缺失文件 |

## 六、与碰撞引擎的对接

种子展开法的产出（研究档案）传递给碰撞引擎（Phase 2.3）：

| 种子展开法产出 | 传递到碰撞引擎的 | 用途 |
|:-------------|:----------------|:-----|
| 域DNA档案 | 域DNA四维 | 碰撞合成的素材池 |
| 切向扩展树 | 素材贡献表 | 碰撞的素材来源 |
| 交叉授粉图 | 跨域联动清单 | 碰撞的联动设计 |
| 创意溯源记录 | 转化方向 | 碰撞后的转化手法预判 |

> 种子展开法是碰撞引擎的上游。没有种子展开法的深度研究，碰撞引擎只能在 AI 内部知识中合成概念，无法产生有根基的创意。

## 七、实操示例（牢A案例）

### Phase 1：种子识别

| 种子 | 类型 | 描述 |
|:-----|:-----|:-----|
| 牢A | 内容创作者+文化现象 | B站创作者，讲北美底层荒诞故事 |
| 北美底层 | 社会议题+地理空间 | 北美社会底层的生存状态 |
| DND | 游戏系统 | 龙与地下城TRPG系统 |
| 穿越/系统 | 叙事套路 | 网文系统文+穿越设定 |

### Phase 2：直接研究（牢A为例）

| 维度 | 内容 |
|:-----|:-----|
| 是什么 | B站/YouTube创作者，讲北美底层荒诞故事，黑色幽默风格 |
| 怎么运作 | 取材真实事件→荒诞化叙事→中式视角解读→引发共鸣 |
| 什么风格 | 黑色幽默、荒诞、中式吐槽、底层视角 |
| 什么影响 | 在华人社区引发广泛讨论，成为"北美底层"话题的代表性声音 |

### Phase 3：切向扩展（牢A为例）

| 路径 | 切向话题 | 关系 |
|:-----|:---------|:-----|
| 同文化现象 | 润人笑话 | 同一文化现象的不同表现 |
| 同文化现象 | 华人自媒体生态 | 同一传播渠道的创作者群体 |
| 同社会议题 | 小红书中美大对账 | 同一社会议题的不同视角 |
| 同地理空间 | 丁胖子广场 | 同一地理空间的标志性地点 |
| 同人群 | 偷渡客社区 | 同一人群的极端案例 |

### Phase 4：域归类

| 域 | 包含的种子+切向话题 | 采风级别 |
|:---|:-------------------|:---------|
| 北美华人叙事 | 牢A+润人笑话+华人自媒体+丁胖子广场 | 深度采风级 |
| 中美社会对比 | 北美底层+中美大对账+黑工经济+医疗缺陷 | 深度采风级 |
| TRPG系统 | DND+跑团文化+属性映射 | 标准采风级 |
| 系统文 | 穿越/系统+系统文变体+金手指代价 | 标准采风级 |

### Phase 5：交叉授粉

| 连接 | 类型 | 叙事可能性 |
|:-----|:-----|:-----------|
| 牢A荒诞叙事 × 系统文金手指 | 意象叠加 | 系统给的任务都是荒诞的北美底层任务 |
| DND属性系统 × 北美社会阶层 | 机制映射 | 信用评分=属性值，阶层=阵营轴 |
| 中美大对账 × DND阵营轴 | 结构借用 | 中美对比映射为阵营对立 |
| DND英雄冒险 × 底层NPC待遇 | 功能反转 | 穿越者以为自己是英雄，实际是系统NPC |
