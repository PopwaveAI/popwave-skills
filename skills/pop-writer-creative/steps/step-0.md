# Step 0: 故事引擎设计（元素融合 SOP）

> **读什么**：W0 产出 + W1 产出（如有）
> **产出什么**：`创意种子/故事引擎.md`

## 操作步骤

1. **碎片解吸 + 元素联想**：从用户碎片中提取"不可妥协元素"，联想 3-5 个候选碰撞元素
2. **碰撞评级**：★★★ 强碰撞优先，★☆☆ 淘汰。取前 2-3 个
3. **落主角**：一个必须同时遵守两套规则、但两套规则互相否定的普通人
4. **轻量方向 sketches**：每个 ~150 字，含碰撞/钩子/第一画面/基调。不展开三层
5. **用户选择** → 只深化选中那一个 → 故事引擎.md
