# Step PRD：PRD 基本法（A/B/C 三分支共享）

> **定位**：PRD = 这本书的"出厂设置"，是最早锁定的轻量宪法。
> **为什么不是故事引擎**：PRD 只回答"我这本书是什么"（3-5 条），不展开世界锚点、不设计主角、不画卷级结构——那些是故事引擎的事。
> **产出什么**：`创意种子/PRD.md`

---

## 三种撰写模式 + 一种微调模式

### 模式 A：PRD 先行（分支 A）

PRD 在素材采集之前撰写，作为加工坐标系。

**流程**：
1. 从用户核心想法中直接提炼
2. 每个维度的内容是"我决定这本书是X，所以素材必须符合Y"
3. PRD 锁定后，所有 W1 搜索都带筛选条件

**示例对话引导**：
> "你说想写一本'高强度数值+社会学博弈'的玄幻。那我们先把这本书的PRD定下来：它是什么类型、承诺给读者什么、外部元素进来怎么加工、什么绝对不能丢。定好这个，我们再按这套坐标系去搜素材。"

**产出特征**：PRD 语气是**断言式**——"这本书是X，不是Y。遇到Z的时候，加工规则是W。"

---

### 模式 B：PRD 浮现（分支 B）

PRD 在素材大量采集之后，从堆积素材中提炼共同脉络。

**流程**：
1. 把所有素材铺开（W0 归档方案 + W0.5 作者想法 + W1 采集素材 + W2 拆书融合摘要）
2. 找共同脉络："这些素材之间有什么共同的冲突模式/审美取向/叙事基因？"
3. 形成 2-3 个候选 PRD 方向（每个 ~50 字：一句话概括+核心冲突+基调）
4. 用户选一个
5. 锁定后，用 PRD 反审所有素材 → 标记优先级/淘汰

**示例对话引导**：
> "我们搜了一堆素材——战锤的混沌四神、深渊主宰的数值体系、DND的机制深度。这些素材我读下来，发现它们共同的脉络是'规则驱动+黑暗政治'。我提炼了三个可能的PRD方向，你看哪个更接近你的感觉：……"

**产出特征**：PRD 语气是**归纳式**——"从这些素材中，我发现共同的方向是X。你看对吗？"

---

### 模式 C：PRD 提炼（分支 C）

方向选定之后，从选定方向 + 深化后的故事引擎中反向提炼 PRD。

**流程**：
1. 方向选定并深化后，反向提取 PRD
2. 这个 PRD 是"事后总结"的，但内容硬度不变

**产出特征**：PRD 语气是**提炼式**——"现在我们定了方向，回头看，这本书的PRD基本法是这些。"

---

### 模式 D：PRD 微调（Phase Delta）

**触发场景**：Phase Delta 判定新元素为"微调可兼容（⚠️）" → 需要修订 PRD 的某一两条以适应新元素。

**这不是"修改全书PRD"**——是"给 PRD 增加一条例外/扩展一条边界"。

**流程**：
1. 确认原 PRD 中与新材料冲突的具体条目（引用原文）
2. 输出修订方案，格式如下：

```
# PRD 微调方案 — {新元素名}

## 来源
Phase Delta 注入诊断 | {日期}

## 原 PRD 条目
{引用原文}

## 冲突解释
{为什么这条与新材料有张力}

## 建议修订
### 方案一（推荐）：[扩展/放宽]
{修订后的版本}

### 方案二（保守）：[增加例外条款]
{在原条目后追加条款："以下情况例外：……"}

## 影响范围
- 受影响的 L1 维度：[力量体系 / 世界蓝图 / ...]
- 是否需要同步修订世界宪法：[是/否]
- 是否需要回退任何已写正文：[是/否]

---

## 用户决策
- [ ] 接受方案一 → 更新 PRD
- [ ] 接受方案二 → 追加例外条款
- [ ] 拒绝修订 → 新元素降级为剩余焊接点
```

3. **不擅自修改**——必须用户签字后才更新 PRD.md

**修订后标记**：
- PRD.md 头部追加 v2 标记 + 修订日期 + 修订来源（Phase Delta: {元素名}）

## PRD 的六个维度

每个维度 1-3 句话。总长度控制在 1 页以内。

### 1. 一句话定位

> 这本书是什么？用一个让人脑中放电影的句子回答。

```
示例：一本「用DND 3.5e的社会学机制来运转的黑暗玄幻」。
```

### 2. 类型锚点

> 这本书的骨架是什么类型？如果有跨界混合，谁是主谁是辅？

```
- 主类型：[玄幻 / 历史 / 悬疑 / 科幻 / ...]
- 辅类型/跨界元素：[战锤 / 克苏鲁 / DND / ...]
- 混合度：[轻量叠层 / 深度融合 / 完全重构]
```

### 3. 核心承诺

> 读者读这本书得到什么？这是给读者签的合同，不能随口说。

```
示例：「读者将体验到：一个真实的、机制驱动的社会如何被外部威胁逼到崩溃边缘，以及一个人在这种情况下如何从底层爬到决定文明命运的位置。」
```

### 4. 加工哲学

> 当外部元素（跨域IP、网络热梗、现实事件）要融入这本书时，默认的加工规则是什么？

```
我们的加工方式：[写实化/奇观化/消解异质感/放大冲突/机制化/感受化 ...]

示例：「一切元素都要经过『这个机制在这个世界有什么现实后果』的拷问。不接受的加工：降神、机械降神、为冲突而冲突的设定。」
```

### 5. 不可妥协的 DNA

> 这本书什么绝对不能丢？即使外部元素再诱人，也不能牺牲的部分。

```
1. [硬核考据 / 爽感节奏 / 文学性 / 草蛇灰线 / ...]
2. [第二个不可妥协]
3. [第三个不可妥协]
```

### 6. 边界

> 这本书什么不写？这是否定式约束。

```
- ❌ [什么不做：比如"不写后宫""不写主角降智""不写龙傲天"]
- ❌ [第二个边界]
```

---

## PRD 质量标准

| 检查 | 标准 |
|:-----|:-----|
| 一句话定位 | 放出来能让人脑中放电影 |
| 加工哲学 | 不能被误解成"好"（必须具体到可执行） |
| 不可妥协 DNA | 真实约束，不是"写得好"这种空话 |
| 边界 | 至少有一条否定式约束（不写什么） |
| 总长度 | 不超过 1 页（这是基本法，不是说明书） |

---

## PRD 写入后

1. PRD.md 写入 `创意种子/PRD.md`
2. 后续所有 W 阶段的搜索/采集/分析，标注"PRD 加工坐标系"的影响
3. Phase 0 故事引擎撰写时，PRD 作为第一章展开深化
