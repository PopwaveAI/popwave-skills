# Step W1: 跨域素材采集 + 拆书融合

> 本步骤包含两个子任务：
> 1. **跨域素材采集**：从外部（非参考书来源）搜索采集素材
> 2. **拆书融合**（有对标书时强制）：消费 deconstructor Lv1 产出

## Part A：跨域素材采集

> **读什么**：路由记录（确认分支） + 如有 PRD（分支 A）
> **产出什么**：`_素材聚合/跨域素材蒸馏.md`
> **分支感知**：根据 Phase R 分支和 PRD 状态调整采集策略

### 操作步骤

#### 1. 读取路由记录

读取 `_路由记录/分支选择.md` → 确认分支。分支 A 还需读取 `PRD.md` 作为筛选器。

#### 2. 按分支模式采集

##### 分支 A：PRD 定向采集

PRD 已定。**按 PRD 加工坐标系定向搜索**。规则：

- 只搜 PRD 核心承诺需要的素材（"这本书要X，所以需要Y类型的素材"）
- 每个搜到的素材标注"与 PRD 加工哲学的关系"
- 与 PRD 不匹配的素材直接跳过，不花时间记录

**目标**：精准采购，不浪费燃料。

**搜索策略示例**：
| PRD 说"高强度数值+社会学博弈" | → 搜：DND 3.5e 社会分层案例、现实中的政治博弈史料、机制驱动的社会组织模型 |
| PRD 说"硬核历史" | → 搜：对应朝代的官制、兵制、经济制度、社会矛盾 |

##### 分支 B：广度搜索

PRD 尚未浮现。**按用户提到的所有碎片分别广度搜索**。规则：

- 用户提了多少碎片/IP/灵感就搜多少
- 每个素材单独蒸馏，不做融合决策
- 标注素材来源和核心特征，留给 PRD 浮现时做融合

**目标**：覆盖面，所有可能的素材先拉进来。

##### 分支 C：最小广度

方向未定。按用户最模糊的意向（"修仙""玄幻"等）做一个最小化的广度搜索，获取 2-3 个参考方向即可。不要花大量时间在素材采集上——分支 C 的重心在方向碰撞。

#### 3. 产出蒸馏摘要

写入 `_素材聚合/跨域素材蒸馏.md`，格式如下：

```
# 跨域素材蒸馏

## 采集模式

**分支：[A/B/C]**
**策略：[PRD定向 / 广度搜索 / 最小广度]**
**PRD 筛选条件（分支A）：[加工哲学简述]**

## 素材清单

### 素材1：[来源名]

- **采集渠道**：[web_search / 知识库 / ...]
- **核心特征**：[1-2 句话]
- **与本项目的关联**：[分支A：与PRD加工哲学的匹配度 / 分支B：潜在的融合方向]
- **加工预判**：[如果这个素材进入本书，加工方式应该是]
```

## Part B：拆书融合（有对标书时强制）

> **读什么**：deconstructor Lv1 产出
> **产出什么**：`_参考书分析/拆书融合摘要.md`

### 操作步骤

1. 用户提到对标书 → 触发 deconstructor Lv1
2. 读取 deconstructor 产出 → 逐 T 提取参考值
3. 产出融合摘要 + M1/M2 转译（继承规律不继承事件）

### 分支感知

| 分支 | 拆书策略 |
|:-----|:---------|
| A | 按 PRD 决定拆什么书和重点看什么维度。比如 PRD 说"社会学博弈" → 重点看 T1 力量体系的社会分层、T4 势力博弈 |
| B | 广度拆书，所有维度都看。目标是给 PRD 浮现提供更多参考 |
| C | 按方向 sketch 匹配的战略选书，看最相关的维度 |

### 产出格式

```
# 拆书融合摘要

## 对标书信息

**书名**：[书名]
**作者**：[作者]
**拆书模式**：[分支A定向 / 分支B广度 / 分支C战略]

## 提取的参考值

### T1 力量体系
- [提取 2-3 个核心竞争力]

### T2 世界观展开
- [提取 2-3 个节奏参考]

### T3 角色系统
- [提取 2-3 个角色配置参考]

### T4 势力格局
- [提取 2-3 个势力博弈参考]

## M1/M2 转译（继承规律，不继承事件）

**可继承的规律**：
1. [什么设计模式可以照搬]
2. [什么节奏可以模仿]

**必须在本书中不同的**：
1. [什么要避免撞车]
2. [什么要替换成本书的特色]

## 对本项目的融合建议

[1-2 段，根据分支模式给出具体建议]
```

## 强制规则

- 有对标书 → 拆书融合强制执行（不看 = 假对标）
- 无对标书 → 跳过。标注"无锚点书分析，跳过拆书融合"。

## 产出

`_素材聚合/跨域素材蒸馏.md` + `_参考书分析/拆书融合摘要.md`（如有对标书）
