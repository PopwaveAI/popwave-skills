# pop-writer-plot · 剧情架构设计 v7.0.0

> **重大重构**：从"事件链+Canvas矩阵"升级为"卷目标→剧情线设计文档→幕切割→章锚点→契诃夫枪链"。
>
> **核心理念**：每条剧情线是一个**独立 .md 文档**，含数值门槛/植入时间轴/人物/配套套路/契诃夫枪链/起点切片/终点切片。不再用 Canvas 表格塞所有线。
>
> **管线**：`卷战略目标 → 拉取剧情储备卡+配套路 → 剧情线设计文档 → 分幕切割 → 幕内章锚点 → 契诃夫枪链 → chapter-design 消费`
>
> **参考模板**：`D:\workspace\让魔门再次伟大\设计\卷\卷1-架构.md`（套路分析段/剧情线分析/未解之谜） + `设计\幕\卷1\幕1.md`（副线活跃度/紧张曲线/情绪弧线） + `设计\幕\卷1\chekhov-tracker.md`（跨幕伏笔追踪）

## Step 0：卷级宏观目标 / 战略定位

> **读**：PRD.md + 故事引擎.md + world 产出（L1 + 角色卡 + 数值体系）
> **产**：`剧情剧情设计/卷/卷{编号}-战略定位.md`

先回答三个问题：

```
1. 这卷在全书中扮演什么角色？
   例：引入世界观 / 核心冲突引爆 / 主角大幅跃升 / 换地图 / 势力战争 / 收束伏笔
   
2. 读者读完这卷后应该有什么感受？
   例：\"世界观好大\" / \"主角终于站起来了\" / \"这反转把我骗到了\"

3. 这卷的约束边界？
   - 时间跨度（从XX到XX）
   - 地理范围
   - 主角力量区间（起点rank→终点rank）
   - 核心关系变化
```

**产出格式**（参考让魔门 `卷1-架构.md` 的「核心命题」「本卷在全卷定位」）：

```markdown
# 卷{编号} 战略定位 · {书名}

## 核心命题
{一句话}。从XX到XX。{参考让魔门: "从棋盘中的弃子，到掀翻棋盘的人"}

## 本卷在全卷定位
{说明本卷在全书的角色: 奠基卷/冲突引爆卷/高潮前置卷/收尾卷}

## 情感基调与节奏
{整体情绪曲线描述，如让魔门的W型}

## 约束边界
| 维度 | 起点 | 终点 |
|:-----|:-----|:-----|
| 主角力量 | {rank/境界} | {rank/境界} |
| 地理范围 | {在XX活动} | {扩张到XX} |
| 时间跨度 | {起始事件} | {结束事件} |
| 核心关系 | {关系状态} | {关系状态} |
```

## Step 1：拉取剧情储备种子 + 配套路

> **读**：`储备剧情池/素材储备池.md`（reservoir 产出的剧情储备卡）
> **读**：`pop-trope-library/套路库/`（叙事结构模板）+ `pop-trope-library/套路库/00-总索引.md`（按情绪查套路）
> **产**：`剧情设计/卷/卷{编号}-剧情种子拉取清单.md`

### 操作流程

1. **扫描储备池**，找出所有状态=就绪 且 匹配度≠冲突 的剧情储备卡
2. **按本卷战略定位筛选**：
   - 本卷是"引入世界观"→优先拉"世界观延展型/背景素材/仙凡鸿沟"类种子
   - 本卷是"冲突引爆"→优先拉"社会对抗/权力博弈/阶级跃升"类种子
   - 本卷是"主角跃升"→优先拉"金手指升级/装备飞升/身份反转"类种子
3. **每条种子做路线决策**：从储备卡的多路线中选最优的一条
4. **从 pop-trope-library 拉取配套套路**：根据剧情类型，查套路库/ + 套路库/，选择可配合的套路
5. **标记拉取记录**：在池中标注"已拉取-卷X"

### 产出格式

```markdown
# 卷{编号} 剧情种子拉取清单

## 主线索材
| 储备卡 | 路线选择 | 配套套路 | 用途 |
|:-------|:---------|:---------|:-----|
| {编号-名称} | {路线A/B} | {套路1→套路2} | {作为什么类剧情线} |

## 支线索材
| 储备卡 | 路线选择 | 配套套路 | 用途 |
|:-------|:---------|:---------|:-----|
| ... | ... | ... | ... |

## 背景素材
| 储备卡 | 用途 |
|:-------|:-----|
| ... | {嵌入哪个背景维度} |

## 已拉取标记
{储备池中已标注本次拉取的条目}
```

## Step 2：产出剧情线设计文档（每条线独立 .md）

> **读**：Step 1 种子拉取清单 + 战略定位 + 角色池 + 数值体系
> **产**：`剧情剧情设计/剧情线/` 目录下，每条线一个独立 .md 文件

### 剧情线分类

| 线类型 | 命名 | 必选 | 说明 |
|:-------|:-----|:----:|:-----|
| 主线1 - 世界危机 | `主线-01-{名称}.md` | 是 | 外部推动力，全书级冲突 |
| 主线2 - 主角成长 | `主线-02-{名称}.md` | 是 | 内部驱动力，能力/心智/身份 |
| 主线3 - 主角行动 | `主线-03-{名称}.md` | 是 | 主角主动发起的计划/目标 |
| 分支线 - 配角弧 | `支线-{编号}-{名称}.md` | 每卷>=3条 | 配角的独立弧线、次要势力博弈 |
| 世界观揭示线 | `支线-{编号}-{名称}.md` | 视需要 | 信息揭示节奏，分层递进 |

### 每条线的文档格式

## Step 3：分幕切割（剧情线→幕）

> **读**：Step 2 全部剧情线文档 + 卷战略定位
> **产**：`剧情剧情设计/幕/vol-XX/分幕规划.md`

### 操作

1. 确定本卷总幕数（通常 3-6 幕，每幕 15-35 章）
2. 每条剧情线按时间轴分配到各幕
3. 确保每幕至少有 3 条活跃剧情线（1主线+2支线）
4. 标注每幕的戏剧功能（建置/上升/转折/高潮/收束）

### 产出格式

```markdown
# 卷{编号} 分幕规划

| 幕 | 章数 | 戏剧功能 | 活跃主线 | 活跃支线 | 核心冲突 |
|:--|:----:|:---------|:---------|:---------|:---------|
| 1 | chXXX-chXXX | {建置/上升} | {线名} | {线名×N} | {核心冲突描述} |
| ... | ... | ... | ... | ... | ... |
```

## 质量红线

| # | 红线 |
|:-:|:-----|
| 1 | **Step 0 卷目标未确认不进 Step 1** — 没有战略定位就拉种子 = 盲选 |
| 2 | **未读储备池就设计剧情线** — 剧情储备卡就是燃料，不读=浪费 |
| 3 | **未查套路库就定套路链** — pop-trope-library/套路库/ 有现成公式 + 剧情库/ 有改建参考 + 元爽点映射表有变体筛选，不查=重新发明轮子 |
| 4 | **每条剧情线必须独立 .md 文档** — 不再写 Canvas 表格塞所有线 |
| 5 | **每条剧情线缺六件套之一** — 数值门槛/植入时间轴/人物/套路链/契诃夫枪链/起终点切片，缺一不可 |
| 6 | **每条主线至少配套 1 个契诃夫枪** — 没伏笔的线太直白 |
| 7 | **每幕至少 3 条活跃剧情线** — 1主线+2支线，低于此线幕太单薄 |
| 8 | **幕内连续空白限量** — 同一条线连续空白 <= 3章 |
| 9 | **卷套路偏好分析不可跳** — 不做套路分析 = 不知道自己写偏了什么 |
