# 步骤1：创建卷纲 · 战略层

> **产出/更新：** `剧情设计/卷/卷{N}-卷纲.md`（L4 卷纲）
> **模板：** `templates/volume-outline.md`
> **核心约束：** Step1 只写卷纲的战略层，不提前分幕、不预设完整剧情线。

## 读什么

> 全部读取，无例外。括号内是用途说明，不是优先级。

- `全书立项PRD.md`（爽点锚点+加工坐标系）
- `小说世界设定/L1-01~06.md`
- `小说世界设定/数值体系/*.md`
- `小说世界设定/起点快照.md`
- `小说世界设定/终点快照.md`
- `小说世界设定/动态升级表.md`（节奏约束校验）
- `状态/角色/*.md`

## 做什么

1. 锁定本卷命题：这一卷要兑现全书哪个承诺？
2. 锁定读者情绪：本卷主要让读者爽、痛、怕、期待什么？
3. 锁定起点与终点：角色、世界、力量、关系分别从哪里到哪里。
4. 锁定主冲突：谁阻碍谁，争夺什么，失败代价是什么。
5. 锁定剧情线配置：本卷需要几条线，主线/支线/设定线分别承担什么职责。
6. 锁定篇幅预算：本卷预估章数（按 2~3k 字/章）、幕数、各幕章数分配。每幕 ≥ 8 章，卷首/卷尾幕可短至 5 章，幕间章差 ≤ 1.5×。
7. **PRD 爽点逐卷映射**：提取 PRD 中每个元爽点的阶段落地方向，为本卷确定具体落地形式。

### PRD 爽点逐卷映射（Step 1.5）

读什么：`全书立项PRD.md` 的爽点锚点段

| 元爽点 | PRD 方向 | 本卷落地形式 | 兑现章范围 |
|:-------|:---------|:------------|:---------|
| {元爽点1} | {PRD中写的阶段方向} | {本卷具体怎么兑现} | {chX-chY} |
| {元爽点2} | {PRD中写的阶段方向} | {本卷具体怎么兑现} | {chX-chY} |

8. 锁定素材需求：下一步要从素材池/剧情库寻找哪类剧情卡。

## 不做什么

- 不提前把幕切死。
- 不写完整主线/支线。
- 不写章锚点和 Canvas。
- 不写叙事债务链，只允许记录“可能需要回收的承诺/信息/规则/选择”。

## 门禁

| 检查项 | 失败动作 |
|:-------|:---------|
| 本卷起点不清 | 退回补起点快照 |
| 本卷终点不清 | 退回补终点快照 |
| 主元爽点不清 | 回 PRD 确认 |
| 主冲突不清 | 不进入 Step2 |
| 线数配置不清 | 不进入 Step2 |
| 本卷设计章数 > 动态升级表 max_chapters | 退回与 world 协商调整 |
| 本卷能级跨度 < 1 完整小阶 | 退回做节奏协商 |
| PRD 有阶段承诺但卷纲不匹配 | 以 PRD 为更高优先级，退回调整 |
| PRD 爽点逐卷映射未完成 | 不进入 Step2 |
| 本卷线数决策表未填写 | 不进入 Step2——必须明确主线数量（默认3条）+支线/设定线数量 |
| 线型职责判定未做 | 不进入 Step2——每条候选线必须通过删除测试判定线型 |
