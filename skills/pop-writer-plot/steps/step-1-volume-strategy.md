# 步骤1：卷战略定位

> **产出：** `剧情设计/卷/卷{N}-战略定位.md`
> **模板：** `templates/volume-strategy.md`
> **核心约束：** 不确认卷目标，不进入拉种子。

## 读什么

- `全书立项PRD.md`
- `小说世界设定/L1-01世界蓝图.md`
- `小说世界设定/L1-02力量体系.md`
- `小说世界设定/起点快照.md`
- `小说世界设定/终点快照.md`
- `小说世界设定/数值体系/rank_schedule.md`
- `状态/角色/*.md`

## 做什么

1. 锁定本卷在全书中的功能：奠基 / 冲突引爆 / 地图扩张 / 势力战争 / 高潮前置 / 收束。
2. 写清本卷核心命题：读者从什么状态读到什么状态。
3. 约束本卷边界：力量、地理、时间、关系、认知。
4. 明确本卷不可越界项：哪些终局信息、力量等级、关系变化不能提前。

## 门禁

| 检查项 | 失败动作 |
|:-------|:---------|
| 核心命题不清 | 退回重写一句话命题 |
| 起点/终点不对应 | 退回 world 快照或数值体系检查 |
| 角色池缺核心角色 | 标注缺口，不编造角色 |
| 本卷边界过宽 | 收窄到可被 1 卷承载的范围 |

## 输出

按 `templates/volume-strategy.md` 写入 `剧情设计/卷/卷{N}-战略定位.md`。
