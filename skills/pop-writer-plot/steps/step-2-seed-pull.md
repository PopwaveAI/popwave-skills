# 步骤2：拉种子与查库

> **产出：** `剧情设计/卷/卷{N}-剧情种子拉取清单.md`
> **模板：** `templates/seed-pull-list.md`
> **核心约束：** 先查库再定套路链；库缺必须声明。

## 读什么

- `剧情设计/卷/卷{N}-战略定位.md`
- `素材储备池/*.md`
- `pop-trope-library/套路库/00-总索引.md`
- `pop-trope-library/剧情库/`
- `pop-trope-library/references/元爽点-变体映射表.md`

## 做什么

1. 从素材储备池筛选状态可用、与本卷命题匹配的剧情储备卡。
2. 按本卷主元爽点查 `套路库/`，记录可用套路链。
3. 按赛道、元爽点、冲突类型查 `剧情库/`，读取可改建剧情单元。
4. 为每个候选种子写清路线选择、替代方案、放弃理由。
5. 标注已拉取记录，防止后续重复抽同一素材。

## 门禁

| 检查项 | 失败动作 |
|:-------|:---------|
| 未查套路库 | 退回补查 |
| 未查剧情库 | 退回补查 |
| 查无匹配 | 标注“本赛道/元爽点库缺”，降级通用素材 |
| 种子只列名字无用途 | 退回补“用作哪条线” |

## 输出

按 `templates/seed-pull-list.md` 写入 `剧情设计/卷/卷{N}-剧情种子拉取清单.md`。
