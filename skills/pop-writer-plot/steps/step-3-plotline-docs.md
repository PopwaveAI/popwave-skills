# 步骤3：L3 剧情线成文

> **产出：** `剧情设计/剧情线/L3-{编号}-{名称}.md`（L3 剧情线）
> **模板：** `templates/L3-剧情线.tpl.md`
> **核心约束：** Step3 承接卷纲中的候选线雏形，正式写成独立 L3 剧情线文档；Canvas 不定义剧情线内容，只调度章节节奏。

## 读什么

- `剧情设计/卷/卷{N}-卷纲.md`（读取 Step2 候选线雏形）
- `状态/角色/*.md`
- `小说世界设定/数值体系/*.md`
- 已查到的套路库、剧情库条目

## 做什么

每条剧情线单独写一个文档。Step3 不重新筛卡、不重定卷战略，只把候选线雏形扩写成“剧情线设计闭环”：

| 模块 | 要求 |
|:-----|:-----|
| 一句话剧情线 | 谁被什么压力逼迫，必须做什么，终局造成什么不可逆变化 |
| 读者追问 | 至少 1 个主追问 + 2 个阶段追问，回答后能产生新问题 |
| 冲突引擎 | 外部压力、内部矛盾、资源缺口、信息差、升级机制至少 3 项成立 |
| 数值门槛 | 触发、推进、收束分别需要什么能力/等级/条件 |
| 剧情发展骨架 | 触发、首次推进、复杂化、反转/代价、高潮释放、收束/余波 |
| 植入时间轴 | 在卷、幕、章的大致位置、状态和目标 |
| 地点与场域 | 本线在哪些地点发生，地点如何制造冲突、限制行动或触发势力反应 |
| 人物与变化 | 谁推动、谁阻碍、谁见证；关键角色起点到终点有什么变化 |
| 套路链 | 每个套路如何部署、预期情绪收益 |
| 与其他线咬合 | 本线如何给其他线递送信息、制造压力、提供资源或共同爆发 |
| 叙事债务与回收链 | 本线埋下哪些信息、制度、选择、地点规则、资源、道具、关系、反派动作或设定递送，后续如何回收。契诃夫枪只是道具型债务 |
| 起终点切片 | 本线开局状态与本卷末状态 |
| Canvas 投放合同 | 交给 L2 单元卡的出现频率、休息窗口、必须释放/递送位置 |

## 剧情线三层结构

> **核心概念：主线大类 ≠ 具体剧情线。** "世界大事件/主角成长/主角行动"是三个主线大类，不是三条剧情线。主线大类是全书固定的分类框架；具体剧情线是每卷动态激活的剧情链条；剧情节点是每条具体剧情线在具体卷/幕中的事件。
> Step 0 用户确认的结构优先级最高——用户说几个大类就几个。以下为默认值。

### 三层定义

| 名称 | 示例 | 作用 | 对应储备卡 |
|:-----|:-----|:-----|:-----------|
| 主线大类 | 世界大事件、主角成长 | 贯穿全书的核心冲突主题，全书固定 | 不直接对应 |
| 具体剧情线 | 剥皮者渗透、吸血神子封神事件 | 每卷/每幕动态激活的剧情链条 | 每条具体剧情线 ≥1张剧情储备卡 |
| 剧情节点 | 卷2剥皮者潜入王城 | 具体的剧情事件 | 储备卡中的剧情线发展链条节点 |

### 主线 vs 支线的层级差异

```
主线：主线大类（全书固定）→ 具体剧情线（每卷动态2-3条）→ 剧情节点
支线：具体剧情线（直接就是具体线，无主线大类）→ 剧情节点
设定线：具体剧情线 → 剧情节点，无主线大类
```

> 合入不了三大类的剧情线都是独立支线。支线不需要挂主线大类——它们本身就是具体剧情线，直接挂剧情节点。

### 主线大类（默认3个）

| 主线大类 | 职责 | 核心驱动 | 爽点类型 |
|:--------|:-----|:---------|:---------|
| 世界大事件 | 世界层面的事件、真相揭露、势力博弈 | 外部压力持续升级，主角被动卷入 | 探索解谜 + 危机合作 |
| 主角成长 | 主角变强——练物变强，显著变强，不是微进步 | 境界突破、能力获取、资源积累 | 变强（元爽点核心） |
| 主角行动 | 主角主动出击——杀人、夺资源、做决策 | 主角的意志和行动力驱动剧情 | 权力博弈 + 危机合作 |

### 主线大类→具体剧情线 动态绑定规则

- **每个主线大类在每一卷有 2-3 条具体剧情线同时推进**——不是全书固定 2-3 条，是每卷动态选择
- 同一具体剧情线可跨卷推进（如：剥皮者渗透从卷2潜入到卷7暴露），但每卷的具体剧情线组合可以不同
- 每条具体剧情线必须标注所属主线大类
- 具体剧情线之间可以互相咬合（如：吸血神子封神事件触发剥皮者暴露）
- 每幕至少有 1 条具体剧情线处于活跃释放状态

### 示例结构

```
卷一：
  主线大类·世界大事件 → 具体剧情线: 神子觉醒血脉 + 裂隙初现 + 帮会暗流涌动
  主线大类·主角成长   → 具体剧情线: 开脉突破 + 血脉初醒
  主线大类·主角行动   → 具体剧情线: 杀疤脸 + 夺码头资源
  支线（无主线大类）  → 具体剧情线: 老瘸子身世之谜 + 阿七的忠诚考验

卷二：
  主线大类·世界大事件 → 具体剧情线: 剥皮者潜入王城 + 神子献祭筹备
  主线大类·主角成长   → 具体剧情线: 洗髓突破 + 果位碎片初触
  主线大类·主角行动   → 具体剧情线: 渗透帮会 + 夺修炼资源
  支线（无主线大类）  → 具体剧情线: 沈墨的双子星暗线
```

### 定义门禁

**"成长"的定义：** 练物变强才是成长。显著变强才是成长。微进步、口头感悟、心态转变不算成长——必须有可量化的境界突破或能力获取。

**"行动"的定义：** 杀人是行动。夺资源是行动。主动决策并承担后果是行动。被动应对、被追杀、被救人不算行动。

## 门禁

| 检查项 | 失败动作 |
|:-------|:---------|
| 推翻卷纲候选线雏形 | 退回 Step2 重筛，不在 Step3 临场换线 |
| 主线大类少于 3 个且无用户明确确认 | 退回补齐三大类；Step 0 用户确认了其他结构时以用户确认为准 |
| 某卷某主线大类下具体剧情线少于 2 条 | 退回补齐——每卷每个大类至少 2 条具体剧情线活跃 |
| 具体剧情线未标注所属主线大类（主线） | 退回补标注 |
| 只写主线无支线/设定线 | 检查是否能支撑每幕 1 主线 + 2 副线/设定线活跃，不够则补线 |
| 主角成长类（具体剧情线）无显著变强节点 | 退回补境界突破/能力获取/资源积累的具体节点——微进步不算 |
| 主角行动类（具体剧情线）无主动出击节点 | 退回补杀人/夺资源/主动决策节点——被动应对不算 |
| 支线承担卷核心承诺 | 升级为主线或重写支线职责 |
| 设定线只讲背景不事件化 | 退回补"递送 → 验证/应用/代价" |
| 无读者追问 | 退回补主追问和阶段追问 |
| 无冲突引擎 | 退回补外部压力/内部矛盾/资源缺口/信息差/升级机制 |
| 只有事件罗列无变化 | 退回补每个节点的角色/世界变化 |
| 缺数值门槛 | 回读数值体系补齐 |
| 缺地点/场域 | 回读 L1 世界设定、势力格局、资源与物品，补地点功能 |
| 缺角色承载 | 回读角色卡，或标注待 character 补卡 |
| 与其他线无咬合 | 退回补至少 1 条关联线 |
| 无叙事债务 | 主线必须补债务链；设定/背景线必须补"递送→验证/应用/代价"链 |
