# 步骤4：分幕规划

> **产出：** `剧情设计/幕/vol-XX/分幕规划.md`
> **模板：** `templates/act-plan.md`
> **核心约束：** 分幕规划决定“哪些线在哪一幕活跃”，不替代 Canvas。

## 读什么

- `剧情设计/卷/卷{N}-战略定位.md`
- `剧情设计/剧情线/*.md`
- `小说世界设定/数值体系/rank_schedule.md`

## 做什么

1. 确定本卷幕数与章段范围。
2. 为每幕标注戏剧功能：建置 / 上升 / 转折 / 高潮 / 收束。
3. 将每条剧情线分配到幕：萌芽 / 推进 / 高潮 / 收束 / 暂停。
4. 检查每幕剧情密度：至少 1 主线 + 2 支线/副线活跃。
5. 标注每幕的核心冲突与读者情绪承诺。

## 门禁

| 检查项 | 失败动作 |
|:-------|:---------|
| 每幕少于 3 条活跃线 | 补支线或重切幕 |
| 主线整幕缺席 | 说明原因，否则调整 |
| 同一支线空白太长 | 改为间隔式回收，交给 Canvas 精排 |
| 幕功能重复 | 合并或重分配戏剧功能 |

## 输出

按 `templates/act-plan.md` 写入 `剧情设计/幕/vol-XX/分幕规划.md`。
