# 步骤6：契诃夫枪链

> **产出：** `剧情设计/幕/vol-XX/chekhov-tracker.md`
> **模板：** `templates/chekhov-tracker.md`
> **核心约束：** 设伏必须有回收窗口，跨幕伏笔必须持续维护。

## 读什么

- `剧情设计/剧情线/*.md`
- `剧情设计/幕/vol-XX/act-YY.md`
- `剧情设计/幕/vol-XX/分幕规划.md`

## 做什么

1. 汇总所有剧情线文档里的枪链。
2. 将 act 锚点中新设/回收的伏笔登记到 tracker。
3. 标注状态：未设伏 / 已设伏 / 进行中 / 已回收 / 延期 / 废弃。
4. 为跨幕伏笔写回收窗口和风险提示。
5. 检查主线是否至少有一条跨幕枪链。

## 门禁

| 检查项 | 失败动作 |
|:-------|:---------|
| 只设伏无回收窗口 | 退回补预期回收 |
| 回收章早于设伏章 | 调整章锚点 |
| 主线无枪链 | 补至少 1 条核心伏笔 |
| 延期无说明 | 标注延期原因和新窗口 |

## 输出

按 `templates/chekhov-tracker.md` 写入 `剧情设计/幕/vol-XX/chekhov-tracker.md`。
