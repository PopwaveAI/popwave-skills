## 三、Canvas 矩阵（章节×剧情线 爽点释放图）

> 行=剧情线，列=章号，单元格=payoff_level。
> 空=铺垫/埋伏笔，中=子线释放，大=格局重画，特=多线汇聚。
> **设计规则**：每条线连续≥3章空→下一章必须释放；每章至少有1条线=中或以上。

### 线号定义

| 线号 | 名称 | 性质 | 说明 |
|:----:|:-----|:----|:-----|
| M1 | 主线1 | 世界危机 | (本幕具体表现) |
| M2 | 主线2 | 主角成长 | (本幕具体表现) |
| M3 | 主线3 | 主角行动 | (本幕具体表现) |
| B1 | 支线1 | | |
| B2 | 支线2 | | |
| B3 | 支线3 | | |

### Canvas 矩阵

```diagram
 章号 | M1-世界观 | M2-守护 | M3-成长 | B1-帮派 | B2-藏宝洞 | B3-血裔 | 总≥中
------|-----------|---------|---------|---------|-----------|---------|-------
 ch01 |    空     |   中    |   空    |   空    |    —     |    —    |   1
 ch02 |    空     |   大    |   中    |   空    |    —     |    —    |   2
 ...  |           |         |         |         |           |         |
```

- `—` = 该线在本章非活跃（未进入该章节的剧情范围）
- `空` = 活跃但铺垫中（未释放）
- `中` = 子线释放（有可感知的情绪/剧情回报）
- `大` = 格局变化（故事语境改变）
- `特` = 多线大释放汇聚

### Canvas 节奏评估

| 检查项 | 结果 | 说明 |
|:-------|:-----|:------|
| 每条线连续≥3空→释放 | ✅/⚠️/❌ | |
| 每章≥1条中或以上 | ✅/⚠️/❌ | 连续空白章需标注原因（缓冲/过渡章） |
| 大释放间隔≤5章 | ✅/⚠️/❌ | |
| 支线交替活跃 | ✅/⚠️/❌ | 无死线 |

**设计结论**：一段话总结本幕节奏的健康度和特殊处理说明。
