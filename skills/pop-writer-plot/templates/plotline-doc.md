## 核心驱动力

这条线为什么存在？它推动主角、世界或读者情绪发生什么变化？

## 一句话剧情线

> 用一句话写清：谁，因为哪个压力/诱惑/缺口，被迫做什么，结果把局面推向哪里。

| 字段 | 内容 |
|:-----|:-----|
| 主体 | 谁是这条线的主要承压者/行动者 |
| 压力源 | 谁/什么在逼迫这条线前进 |
| 行动方向 | 主体必须做什么 |
| 失败代价 | 如果不推进会失去什么 |
| 终局变化 | 本线结束后，世界/关系/能力/认知发生什么不可逆变化 |

## 线型功能

| 线型 | 必须回答 |
|:-----|:---------|
| 主线 | 本卷核心冲突如何推进、升级、爆发、收束？ |
| 支线 | 它补强主线的哪个压力、情绪或反差？ |
| 设定线/背景线 | 它要递送什么世界规则、历史背景、势力逻辑或社会常识？这些信息如何通过事件被读者吸收？ |
| 关系线 | 哪段关系如何变化，如何影响主线选择？ |
| 成长线 | 主角或关键角色的能力/认知/价值观如何变化？ |
| 反派线 | 反派如何行动、误导、施压或重塑局势？ |

## 冲突引擎

> 冲突引擎不是“有敌人”，而是这条线为什么会自己持续推进。

| 引擎类型 | 本线设计 |
|:---------|:---------|
| 外部压力 | 反派、制度、环境、时间限制如何逼近 |
| 内部矛盾 | 主角/关键角色的欲望、恐惧、误判或底线 |
| 资源缺口 | 缺钱、缺人、缺知识、缺资格、缺时间、缺合法性 |
| 信息差 | 谁知道什么，谁误解什么，读者知道多少 |
| 升级机制 | 每推进一次，压力如何变大、规则如何变严、代价如何变高 |

## 叙事债务与回收链

> 叙事债务 = 前文让读者记住、相信、疑惑或期待的内容，后文必须兑现。契诃夫枪只是其中一种。
> 此时尚未分幕，**不指定具体章号**——只记录债务节点、卷内故事位置和预计回收方式。具体章号在 Step5 Canvas 设计时确定。

| 节点 | 债务类型 | 前置埋点 / 递送节点 | 为什么必须回收 | 故事位置 | 回收方式 |
|:----:|:---------|:--------------------|:---------------|:---------|:---------|
| 1 | 信息差/制度/人物选择/地点规则/资源/道具/关系/反派动作/设定递送 | {一句话简述} | | 卷{前/中/后}段 | — |
| 2 | 回收 | {一句话简述} | | 卷{前/中/后}段 | 揭示/反转/引爆/验证/代价兑现/设定落地 |

> 主线必须有叙事债务链；设定/背景线必须有“递送 → 验证/应用/代价”的回收链；道具型债务才叫契诃夫枪。
