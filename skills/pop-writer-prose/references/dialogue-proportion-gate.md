# 对话比例门禁

> 2026-06-22 新增，源于东方版深渊主宰复盘（ch01 对话 <15% vs PRD 承诺 ≥40%）

## 为什么需要这个门禁

PRD §十 明确承诺「对话比例 ≥ 40%。信息通过对话释放，不靠旁白堆设定」。但 agent 在渲染正文时常见两个错误：

1. **内心独白替代对话** — 主角内心 OS 连篇，"他知道/他意识到/他记得"占了正文 70%，角色之间几乎不说话
2. **叙事者解说替代对话** — 设定信息通过旁白段落释放，而不是"角色 A 告诉角色 B"

这两种写法在文件级验证（无 AI 词/视角一致）中都能通过，但读者体验是灾难性的——读者不是在"看故事"，是在"听解说"。

## 计算方法

```
对白段字数 = 所有被「引号」包裹的对话段落的字数之和
正文总字数 = 全文 UTF-8 字符数（含标点、空格、段落空行）

对话比例 = 对白段字数 ÷ 正文总字数
```

**注意事项：**
- 内心独白（无引号、有「他心想」「他在心里说」等标志）不算对白段
- 一句话对白也算（「好。」——算 2 字对话）
- 旁白段中引用的角色原话算对白段

## 门禁阈值

| 比例 | 判定 | 动作 |
|:----:|:-----|:------|
| **< 0.3** | ❌ 严重不达标 | 退回 Step 1，标注「对话比例严重不足（X% vs 40%）」。审视：哪些内心独白可以改为对白释放？ |
| **0.3 ~ 0.39** | ⚠️ 需改善 | 标注「对话比例偏低（X%），建议增加 1-2 段角色对白替换内心独白」 |
| **≥ 0.4** | ✅ 达标 | 通过 |

## 常见失败模式

| 原始写法（内心独白） | 修正后（对话释放） |
|:--------------------|:-----------------|
| 「江轩知道疤脸背后有人。阿七说过疤脸昨晚去了城西。」 | 「疤脸昨晚去哪儿了？」「城西。上头的人叫他去的。」 |
| 「他意识到这件事比想象中严重——圣者浩劫的规模比他记忆中大。」 | 「你知道圣者浩劫死了多少神吗？」老瘸子顿了顿。「全死了。一个没剩。」 |
| 「老瘸子认得这块瓦片。江轩心里的疑团更大了。」 | 「你那块瓦片——哪来的？」老瘸子的声音变了调。 |

## 案例：东方版深渊主宰 ch01

ch01 正文 217 行，对白仅约 12-15 句，占比 <15%。主要信息通过以下方式释放：

```
❌ 内心独白：「圣者浩劫。他记得自己知道这件事——或者更准确地说，记得"前世知道"这件事。」
❌ 内心独白：「他在心里快速对比了一下——"我记得疤脸只是个普通压迫者，三章就没了。这个版本里他有上线。"」
❌ 内心独白：「"我的剧本有缺页。" 江轩在心里说。"而且我不知道缺了多少。"」
```

正确做法：让疤脸手下在盯梢时说漏嘴、让阿七压低声音说 "疤脸上头的人在查什么东西"、让老瘸子含含糊糊吐几个字。
