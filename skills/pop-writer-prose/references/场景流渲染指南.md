# 场景流渲染指南

> 所属: pop-writer-prose v2.0.0 · references/
> 消费: Step 1 按场景流渲染时参考
> 红线: ❌2 场景流渲染（非逐事件）

## 一、场景内连续流动

v2 核心差异：按场景流渲染，事件在场景内有机融合，**不逐事件独立成段**。

**❌ WRONG（逐事件渲染）：**
```
索伦拔出匕首。他绕到恶棍身后。匕首刺入后心。他捂住对方的嘴。
```

**✅ CORRECT（场景流渲染）：**
```
索伦绕到恶棍身后，匕首无声刺入后心——一只手捂住嘴，另一只手摸走钱袋，全程没发出一点声响。
```

## 二、场景渲染流程

每个场景走 5 步：

| 步骤 | 内容 |
|:-----|:-----|
| A. 理解意图 | 场景指令？情绪走向？推进哪条线？ |
| B. 选POV | 读场景POV字段 → 不进入非锚点角色内心 |
| C. 写场景段落 | 场景内连续流动 + 🔒对白嵌入 + DNA风格对齐 |
| D. 控制比例 | 按场景类型控制展示/内省比例 |
| E. 场景间过渡 | 1-2句自然过渡 |

## 三、开幕段落

- 空间/时间锚定 → 情绪传递（1-2句）
- 如场景有感官锚点字段，第一句以该感官信号开头
- 例：「雨声先到。然后是铁锈味。」

## 四、场景间过渡

- 场景之间 1-2 句自然过渡，不铺陈
- 过渡方式：时空跳转（"三天后"）/ 感官切换（"声音消失了"）/ 情绪转折
- 不写"与此同时"式的平行剪辑——保持单线POV

## 五、对白嵌入规则

| 规则 | 说明 |
|:-----|:-----|
| 🔒标记对白 | 设计包"关键对白"字段的原文必须嵌入，一字不改 |
| 非标记对白 | prose 自行发挥，符合角色台词风格 |
| 对话留白 | 按DNA层A对话留白规则（无引导词/有引导词） |
| 对白占比 | 按场景类型预期（对话场景>行动场景） |

## 六、闭幕段落

章末 = 钩子位置：
- 悬念钩 → 不闭合，留疑问
- 情绪钩 → 安静收尾，情绪余韵
- 行动钩 → 蓄势待发，停在行动起点

## 七、与v1的差异

| 维度 | v1 | v2 |
|:-----|:---|:---|
| 渲染单位 | 逐事件 | 场景流 |
| 事件处理 | 每事件独立成段 | 场景内有机融合 |
| 过渡 | 逐事件间过渡 | 场景间过渡 |
