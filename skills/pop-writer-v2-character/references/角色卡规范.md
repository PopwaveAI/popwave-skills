# 角色卡规范

> v1 完整字段 + v2 新增生态位字段说明。供 Step 2 撰写角色卡时参照。

---

## 一、v1 完整字段（按级别递增）

### Lv1：一次性角色

| 字段 | 说明 |
|:-----|:------|
| 摘要 | 一句话概括角色身份与功能 |
| 代表性台词 | 一句能体现角色特征的对白 |

### Lv2：功能角色（= Lv1 + 新增）

| 字段 | 说明 |
|:-----|:------|
| 标签 | 角色类型标签（如"情报贩子""守门人"） |
| 驱动·功能 | 角色在剧情中的固定功能 |
| 说法风格 | 2-4 句能直接支撑对白的风格锚点 |

### Lv3：重要配角（= Lv2 + 新增）

| 字段 | 说明 |
|:-----|:------|
| 身份 | 角色在世界中的正式身份与社会位置 |
| 驱力四层 | 表层目标 / 深层需要 / 黑暗冲动 / 恐惧 |
| 弧线 | 角色在卷/幕中的变化轨迹 |
| 关系网 | 与其他角色的关系清单 |

### Lv4：主角/核心对立（= Lv3 + 新增）

| 字段 | 说明 |
|:-----|:------|
| 核心缺陷 | 对谁/什么事最致命的盲点或弱点 |
| 决策逻辑 | 日常决策倾向 / 危机决策倾向 / 决策阈值 |
| 剧情驱动接口 | 核心驱动力映射 + 驱动的剧情线 + 决策转折点预判 + 角色化学反应 |
| 角色压力测试 | 角色在世界宪法矛盾中的决策推演 |

### 所有级别必含

| 字段 | 说明 |
|:-----|:------|
| 设计目的 | 这个角色为什么存在，没有他会失去什么 |
| 叙事能力 | 角色能驱动什么叙事效果 |
| 说法风格 | 能直接支撑正文对白的风格锚点 |

---

## 二、v2 新增生态位字段（所有级别必填）

> v2 核心差异：将角色锚定到生态图谱，角色不只是"存在于世界"而是"嵌入生态结构"。

### 1. 生态位置

| 字段 | 说明 |
|:-----|:------|
| 生态图谱节点 | 角色在生态图谱中占据的节点名称 |
| 所属势力/阶层 | 该节点隶属的势力或社会阶层 |
| 位置性质 | 枢纽（多线交汇）/ 中继 / 边缘 / 冲突前沿 |
| 资源掌控 | 该位置掌控什么资源，受制于什么规则 |

### 2. 生态关系

| 字段 | 说明 |
|:-----|:------|
| 关系对象 | 与谁有关系（角色名或势力名） |
| 关系类型 | 合作 / 竞争 / 依附 / 对抗 / 共生 |
| 关系性质 | 具体的关系内容描述 |
| 剧情接口 | 这条关系驱动什么剧情线或副线 |

> 每角色至少 2 条生态关系（红线 ❌3）。

### 3. 生态冲突点

| 字段 | 说明 |
|:-----|:------|
| 生态冲突点 | 角色在生态结构中面临的结构性矛盾 |
| 冲突来源 | 资源争夺 / 规则压制 / 阶层流动 / 势力博弈 |
| 冲突升级路径 | 结构性矛盾如何演变为剧情冲突 |

> 生态冲突点是决策转折点的生态驱动源。Lv4 主角的生态冲突点必须能驱动主线。

---

## 三、字段填写纪律

- 生态位字段不允许"待定"/"？"，必须具体到生态图谱的实际节点
- 生态关系的"剧情接口"必须指向一条可识别的剧情线
- 生态冲突点的"冲突升级路径"必须能解释从结构性矛盾到剧情冲突的转化逻辑
- v1 字段填写纪律不变：说法风格不能只写空标签，驱力四层不能写"待定"
