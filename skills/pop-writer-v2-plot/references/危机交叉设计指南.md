# 危机交叉设计指南

> 5 系统危机分类 + 交叉设计方法。供 Step 2 危机交叉设计参照。

---

## 一、五大危机系统

| 系统 | 说明 | 典型危机 | 情绪效果 |
|:-----|:------|:---------|:---------|
| **金手指系统** | 主角金手指的代价/反噬/限制 | 升级反噬、使用限制暴露、引来觊觎、能量枯竭 | 紧张+期待 |
| **社会环境系统** | 社会结构/阶层规则/制度压力 | 阶层打压、规则束缚、身份歧视、制度陷阱 | 压抑+愤怒 |
| **角色关系系统** | 角色关系网中的冲突/背叛/利益纠葛 | 盟友反目、情感绑架、利益冲突、信任危机 | 痛心+纠结 |
| **世界观系统** | 世界法则/历史遗留/环境灾害 | 法则突变、远古威胁、环境剧变、世界规则限制 | 敬畏+无力 |
| **外部威胁系统** | 外部势力入侵/天灾/不可控事件 | 敌对势力入侵、天灾降临、意外事件、黑天鹅 | 恐惧+紧迫 |

## 二、交叉设计方法

### 原则

1. **每幕 ≥ 3 系统危机**：不同系统提供不同质感的压力
2. **无连续 3 章同源**：避免读者审美疲劳
3. **交叉点制造复合危机**：两个系统危机在同一章交汇，压力倍增

### 交叉点设计

交叉点 = 两个或多个系统的危机在同一章节交汇。

```
交叉点示例：
- 金手指反噬（金手指系统）+ 盟友质疑（角色关系系统）→ 主角在信任危机中独自承担反噬代价
- 阶层打压（社会环境系统）+ 世界法则限制（世界观系统）→ 主角的升级路径被双重封锁
- 外部入侵（外部威胁系统）+ 金手指暴露风险（金手指系统）→ 主角必须在暴露金手指和抵御入侵间抉择
```

### 交叉节奏

| 位置 | 建议交叉 |
|:-----|:---------|
| 幕初 | 单系统危机，建立压力基线 |
| 幕中 | 交叉点（2 系统交汇），压力升级 |
| 幕尾 | 复合交叉（3 系统交汇），推向高潮 |

## 三、危机来源与角色生态冲突映射

每个危机事件应追溯到角色卡的生态冲突点：

| 危机系统 | 对应生态冲突来源 | 映射逻辑 |
|:---|:---|:---|
| 金手指系统 | 角色生态位置的资源掌控与金手指代价的矛盾 | 金手指改变生态位置 → 新矛盾 |
| 社会环境系统 | 角色生态位置的结构性压制 | 阶层/规则对位置的束缚 |
| 角色关系系统 | 生态关系网中的利益纠葛 | 关系变化 → 生态位置动摇 |
| 世界观系统 | 生态图谱底层法则变化 | 法则变 → 所有位置重洗牌 |
| 外部威胁系统 | 外部力量冲击生态图谱 | 入侵 → 生态结构破坏 |

## 四、示例

### 幕一危机交叉设计（示例）

| 章 | 危机系统 | 危机事件 | 交叉 |
|:---|:---|:---|:---|
| ch01 | 社会环境 | 阶层考核，低层被刻意压分 | — |
| ch02 | 角色关系 | 盟友因利益动摇，暗示退缩 | — |
| ch03 | 金手指 | 金手指使用后出现反噬征兆 | — |
| ch04 | 社会环境 + 金手指 | 考核中被迫使用金手指 → 反噬+暴露风险 | ✅ 交叉点 |
| ch05 | 外部威胁 | 敌对势力借考核混乱渗透 | — |
| ch06 | 金手指 + 角色关系 | 反噬加剧时盟友正式离开 | ✅ 交叉点 |
| ch07 | 社会环境 + 外部威胁 + 世界观 | 考核制度+入侵+法则限制三重施压 | ✅ 复合交叉（幕尾高潮） |
