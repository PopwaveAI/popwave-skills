# Step 1：卷战略 + 危机系统分配

> 管线: pop-writer-v2-plot v1.0.0
> 消费: pop-writer-v2-character 产物（角色卡 + 弧线）+ pop-writer-v2-world 产物（世界宪法）
> 产出: `卷/卷{N}-卷纲.md`（卷级战略 + 危机系统分配表）

## 读什么

- `全书立项PRD.md`（爽点锚点 + 加工坐标系）
- `小说世界设定/世界宪法.md`（核心矛盾 + 力量体系）
- `小说世界设定/起点快照.md` + `终点快照.md`（卷级起止状态）
- `状态/角色/*.md`（角色卡 + 生态位字段 + 决策转折点预判）
- `references/危机交叉设计指南.md`（5 系统危机分类）
- `templates/卷大纲-模板.md`（卷大纲模板）

## 做什么

### 一、卷级战略（复用 v1）

1. 锁定本卷命题：这一卷要兑现全书哪个承诺？
2. 锁定读者情绪：本卷主要让读者爽、痛、怕、期待什么？
3. 锁定起点与终点：角色、世界、力量、关系分别从哪里到哪里。
4. 锁定主冲突：谁阻碍谁，争夺什么，失败代价是什么。
5. 锁定剧情线配置：本卷需要几条线，主线/支线/设定线分别承担什么职责。
6. 锁定篇幅预算：本卷预估章数（2~3k 字/章）、幕数、各幕章数分配。每幕 ≥ 8 章。

### 二、危机系统分配（v2 新增）

> v2 核心差异：每幕分配 ≥ 3 个系统危机来源，确保危机多样性。

为每幕分配危机来源系统：

| 系统 | 说明 | 典型危机 |
|:-----|:------|:---------|
| 金手指系统 | 主角金手指带来的代价/反噬/限制 | 升级反噬/使用限制暴露/引来觊觎 |
| 社会环境系统 | 社会结构/阶层规则/制度压力 | 阶层打压/规则束缚/身份歧视 |
| 角色关系系统 | 角色关系网中的冲突/背叛/利益纠葛 | 盟友反目/情感绑架/利益冲突 |
| 世界观系统 | 世界法则/历史遗留/环境灾害 | 法则突变/远古威胁/环境剧变 |
| 外部威胁系统 | 外部势力入侵/天灾/不可控事件 | 敌对势力入侵/天灾降临/意外事件 |

每幕必须从 ≥ 3 个系统中分配危机事件。

|:-------|:---------|:------------|:---------|

## 门禁

| 检查项 | 失败动作 |
|:-------|:---------|
| 角色卡缺失（无生态位字段/无弧线） | ❌ 硬阻塞，退回 character skill（红线 ❌4） |
| 本卷起点/终点不清 | 退回补快照 |
| 主冲突不清 | 不进入 Step 2 |
| 每幕危机系统分配 < 3 个 | 退回补分配 |
| 线数配置不清 | 不进入 Step 2 |
