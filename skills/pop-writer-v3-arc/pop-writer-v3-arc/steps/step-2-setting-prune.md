# Step 2：设定修剪

> 管线: pop-writer-v3-arc v1.4.0
> 消费: 幕纲（嵌套子线+子线咬合） + 写作参考/设定/ + 活记忆 + Step 1检查结果
> 产出: 设定修剪记录（写入写作参考/已废弃/ + 校准报告）

## 读什么

- `卷纲/幕NNN-{名称}.md`（当前L2单元卡：嵌套子线段+子线咬合段）
- `写作参考/设定/压力矩阵全量.md`（压力源列表）
- `写作参考/设定/冲突轴全量.md`（活跃线索列表）
- `写作参考/已废弃/`（已归档的设定要素，版本追溯）
- `活记忆/活记忆.yaml`（本单元events，核对修剪依据）
- Step 1 宏观检查结果（压力推进检查 + 设定一致检查 + 子线闭合检查）
- `references/设定修剪规则.md`（修剪判定标准，权限全给agent）

## 做什么

**核心原则：权限全给agent。** agent自主判断哪些设定要素不再活跃，执行修剪。修剪不删除历史，只标记状态。

修剪对象从v3.4的"种子文档"改为两个目标：
1. **幕纲子线**：关闭已闭合的子线、标记休眠的子线
2. **写作参考/设定/**：维护全局设定文件（压力矩阵/冲突轴），关闭过期线索

### 1. 幕纲子线修剪

逐条检查幕纲嵌套子线的状态：

| 判定条件 | 修剪动作 | 标记状态 |
|:---------|:---------|:---------|
| 子线已闭合（悬念兑现/目标达成） | 在幕纲中标注状态为"闭合"，记录闭合章号 | 已闭合 |
| 子线延伸到后续单元（未完） | 在幕纲中标注状态为"延伸"，记录延伸方向 | 延伸 |
| 子线因剧情发展失效 | 在幕纲中标注状态为"失效"，记录失效原因 | 已失效 |
| 子线整个单元未触及（设计但未推进） | 标注"遗漏"，记入校准报告供下一L2设计参考 | 遗漏 |

agent自主判断：如果子线虽休眠但属于长线伏笔（预计跨多个L2单元回收）→ 保留"延伸"状态，标注"长线"。

### 2. 压力矩阵修剪

逐个检查`写作参考/设定/压力矩阵全量.md`中的压力源：

| 判定条件 | 修剪动作 | 标记状态 |
|:---------|:---------|:---------|
| 倒计时已归零且连续5章未再激活 | 移入已关闭区 | 已关闭 |
| 压力源被剧情解决（如获得对应保障） | 移入已关闭区 | 已解决 |
| 压力源暂时不活跃但可能复发 | 保留在活跃区，标注休眠 | 休眠 |
| 涌现出新压力源未记入 | 追加到压力矩阵 | 新增 |

agent自主判断：如果不确定是否该关闭 → 保留活跃（宁留勿错关，下次arc再判）。

### 3. 活跃线索修剪

逐条检查`写作参考/设定/冲突轴全量.md`中的活跃线索：

| 判定条件 | 修剪动作 | 标记状态 |
|:---------|:---------|:---------|
| 线索悬念已兑现（伏笔已回收） | 移入已关闭区 | 已回收 |
| 连续>=10章未触及且无近期回收计划 | 移入已关闭区 | 休眠 |
| 线索因剧情发展已失效 | 移入已关闭区 | 已失效 |
| 两条线索实际已合并为一条 | 合并保留一条 | 已合并 |

agent自主判断：长线伏笔（预计跨多个L2单元回收）→ 保留活跃，标注"长线休眠"。

### 4. 修剪执行

对每项修剪：

1. **复制原内容**：将要素原内容完整复制到 `写作参考/已废弃/` 的记录中
2. **移除/标记**：从`写作参考/设定/`对应文件中移除该条目，或标注状态变更
3. **幕纲同步**：如涉及幕纲子线，在幕纲嵌套子线段标注状态变更
4. **更新索引**：如`写作参考/索引.md`中有对应条目，同步更新状态

## 门禁

| 检查项 | 失败动作 |
|:-------|:---------|
| 修剪不删除历史，原内容保留在`写作参考/已废弃/` | 检查`写作参考/已废弃/`，缺原内容=补回 |
| 每项修剪有明确理由 | 无理由=撤回该修剪 |
| 幕纲子线状态已同步标注 | 未同步=补标注 |
| 活跃线索<3条时谨慎修剪 | 活跃线索不足=优先保留 |

## 产出

1. 写作参考/设定/更新：
   - `写作参考/设定/压力矩阵全量.md`：移除已关闭压力源 / 标注状态变更
   - `写作参考/设定/冲突轴全量.md`：移除已关闭线索 / 标注状态变更
   - `写作参考/已废弃/`：追加修剪记录（要素|原内容|关闭章号|关闭原因）
2. 幕纲子线状态更新（嵌套子线段标注闭合/延伸/失效）
3. 设定修剪记录表（填入校准报告草稿）：

| 要素 | 类型 | 原内容摘要 | 修剪动作 | 关闭章号 | 理由 |
|:---|:---|:---------|:---------|:---------|:-----|
| | 幕纲子线/压力源/活跃线索 | | 已闭合/已关闭/已回收/休眠/调整 | chXX | |

4. 如无任何要素需要修剪 → 记录"本次校准无需修剪"，直接进入下一步
