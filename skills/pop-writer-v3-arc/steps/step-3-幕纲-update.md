# Step 3：单元卡实际记录回填

> 把 plot 的设计单元卡更新为“实际发生记录”。

## 做什么

- 用正文事实更新物理坐标。
- 用正文事实更新章节导演指令表中的实际事件 brief。
- 用 step-1 提取的真实剧情线替换空线/预想线。
- 用 step-2 更新后的设定账本摘要替换旧摘要。
- 更新单元边界信号，作为下一单元起点。

## 原则

- 不美化失败。实际没发生就写没发生。
- 不保留空子线。
- 不把设计意图当事实。

## 输出

更新 `卷纲/幕NNN-{名称}.md`，并在 arc 报告记录：

```markdown
## 幕纲实际记录回填
- 更新段落：
- 删除空线：
- 新增真实线：
- 下一单元关键输入：
```
