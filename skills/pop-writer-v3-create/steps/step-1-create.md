# Step 1 - 正文创作

## 先核对

写前确认五项材料：

1. 上章末尾是否能接住本章开场。
2. 状态快照是否明确主角处境。
3. 运行日志是否给出本章事件链。
4. 设定账本是否覆盖本章关键规则。
5. 文风DNA是否足够指导语感。

缺失材料能从项目文件补齐就补齐；不能补齐时，在正文中保守处理，不擅自发明硬设定。

## 写法

- 开场承接上章最后一个动作、危险或疑问。
- 每一场都让主角做选择，选择带来代价或收益。
- 设定信息通过冲突、交易、观察、失败来显影，不用百科说明。
- 爽点必须有压迫前置、行动突破、旁人反应、局势收益。
- 章末留下主动钩子：主角做出下一步选择，或威胁逼到眼前。

## 输出

按 `templates/创作-模板.md` 输出正文和简短创作记录。创作记录只写真实偏离点、设定新增风险和下一章衔接点。
