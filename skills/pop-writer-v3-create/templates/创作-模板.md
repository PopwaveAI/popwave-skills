# 创作模板

> create 使用。把当前章运行日志正文化，不重新设计剧情。

## 一、上下文确认

| 确认项 | 来源 | 状态 |
|:-------|:-----|:-----|
| 当前章运行日志 | `卷纲/运行/幕NNN-{名称}-运行日志.md#chXXX` | ☐ 已读 / ☐ 缺失 |
| 当前章导演指令 | `卷纲/幕NNN-{名称}.md#章节导演指令表` | ☐ 已读 / ☐ 缺失 |
| 设定账本 | `卷纲/运行/幕NNN-{名称}-设定账本.md` | ☐ 已读 / ☐ 缺失 |
| 硬设定 | `写作参考/设定/*.md` | ☐ 已读 / ☐ 缺失 |
| 上章末尾 | `正文/ch{上一章}.md` | ☐ 已读 / ☐ 首章无 |
| 活记忆/状态快照 | `活记忆/活记忆.yaml` | ☐ 已读 / ☐ 缺失 |
| 文风 DNA | `写作资产/文风库/{书名}.md` | ☐ 已读 / ☐ 缺失 |

## 六、自检

- [ ] 当前章运行日志已完整覆盖。
- [ ] 没有改变事件链。
- [ ] 没有改变本章结算。
- [ ] 没有新增未入账重大设定。
- [ ] 第一屏有冲突/悬念/反差/行动。
- [ ] 主角有主动行动。
- [ ] 行动有即时反馈。
- [ ] 章末停在指定推力点。
