# Step 2：涌现写作（chapter+prose合并）

> 管线: pop-writer-v3-emerge v1.0.0
> 消费: 种子(当前版本) + 活记忆 + 上一章正文末尾 + 文风DNA + 方向提示 + 增量信息
> 产出: 本章正文初稿（内存）

## 目的

chapter+prose合并——消除章纲→正文的第三次信息损失。agent自由创作，做实时决策，没有章纲约束。文风DNA硬阻塞。80%展示+20%内省。

## 读什么

| # | 文件 | 为什么 |
|:-:|:-----|:-------|
| 1 | `种子文档.md`（当前版本） | 七要素全量驻留：压力矩阵/主角引擎/金手指/冲突轴/活跃线索/成长路径/目的地/文风DNA |
| 2 | `活记忆/活记忆.yaml`（最后baseline+event） | 七组件当前状态：种子追踪/压力状态/角色状态/世界规则/节奏日志/战力曲线/目的地进度 |
| 3 | `正文/ch{上一章}.md` 最后~800字 | 语感衔接+悬念承接 |
| 4 | 文风DNA `写作资产/文风库/{书名}.md` | 句式/叙事距离/感官密度/对话留白锚定（硬阻塞） |
| 5 | 上一章方向提示（活记忆event.direction_hint） | 本章方向感（给奖励/加压/触面线索/即时钩子） |
| 6 | Step 1增量信息（如有） | 制度细节/场景技法/伏笔回收线索 |
| 7 | `references/涌现写作指南.md` | 文风DNA硬阻塞+场景流渲染+80/20规则 |

## 做什么

### 1. 文风DNA硬阻塞检查（红线1）

- 从种子文档解析文风DNA引用路径 `写作资产/文风库/{书名}.md`
- 加载文风DNA文件
- DNA缺失/路径错误/文件空 → **硬阻塞终止**（不降级、不A/B、不空写）
- DNA加载成功 → 建立两层感知：
  - 层A整体风格：句子长短/叙事者距离/战斗节奏/对话留白
  - 层B场景卡：观察句+原文锚定+时间演变

### 2. 上下文整合

将以下信息整合为写作上下文，全量驻留：
- 种子七要素（压力矩阵/主角引擎/金手指/冲突轴+活跃线索/成长路径/目的地/文风DNA）
- 活记忆七组件当前状态
- 上一章正文末尾（语感衔接）
- 方向提示（本章方向感）
- Step 1增量信息（如有）

### 3. 涌现写作（chapter+prose合并，无章纲约束）

agent自由创作，做实时决策。没有章纲约束——agent在写作过程中自主决定：
- 本章推进哪条活跃线索
- 压力矩阵中哪个压力源逼近
- 金手指怎么使用（受限于种子的能力+限制）
- 角色做什么决策（受限于主角引擎的驱动力三层+缺陷+决策风格）
- 章末钩子怎么设置

**合并理由：** 设计决策和渲染执行在人类作者脑中是同时发生的。分离导致prose拿到的是"设计摘要"而非"创作冲动"。合并后agent拿着种子+活记忆直接涌现，信息损失从3次降为1次。

### 4. 场景流渲染（继承v2 prose）

按 `references/涌现写作指南.md`，按场景流渲染（非逐事件）：

| 原则 | 规则 |
|:-----|:-----|
| 场景流渲染 | 事件在场景内有机融合，不逐事件独立成段 |
| 开幕段落 | 空间/时间锚定→情绪传递（1-2句） |
| 场景间过渡 | 1-2句自然过渡，不铺陈 |
| 闭幕段落 | 章末=钩子位置（悬念/情绪/行动） |

### 5. 情感表达比例（红线继承：80%展示+20%内省）

按场景类型控制展示/内省比例：

| 场景类型 | 展示 | 内省 | 内省使用场景 |
|:---------|:----:|:----:|:------------|
| 行动场景 | 90% | 10% | 仅生死瞬间一闪念 |
| 情感场景 | 70% | 30% | 关系转折/情感爆发点 |
| 对话场景 | 85% | 15% | 对话间隙的心理反应 |
| 发现场景 | 80% | 20% | 信息揭示后的认知冲击 |

内省仅用于：关键决策点/情感高潮点/信息揭示后。禁止连续2段以上纯内省。

### 6. AI观感词规避

逐段自检，命中即修正：
- "他感到/他意识到/他仿佛"
- "不是…而是…" ≥ 2次
- 解说员句式（叙事者解释设定）
- 视角越界（限知视角写非锚点内心）

### 7. 压力源逼近（种子消费）

本章正文必须至少推进一个压力源的显性表现。从种子压力矩阵中选取本章活跃的压力源，在正文中体现其逼近。

## 门禁

| 检查项 | 失败动作 |
|:-------|:---------|
| 文风DNA缺失（红线1） | 硬阻塞终止 |
| 逐事件渲染（非场景流） | 退回重写 |
| 内省比例超标 | 退回重写 |
| AI观感词命中>3种 | 退回重写 |
| 本章无压力源逼近 | 退回重写 |
| 种子/活记忆不存在 | 终止，提示先完成上游 |

## 产出

本章正文初稿（内存），等待Step 3反思验证后落盘。

正文格式：
```markdown
# chXXX · {本章标题}

{正文全文}
```
