# 信息获取记录模板

> 所属: pop-writer-v3-emerge v2.0.0 · templates/
> 消费: Step 1 产出记录
> 红线: ❌6 信息获取必须读索引——素材库/索引.md未读取=退回补读
> v3.1改动: 知识库→素材库；索引强制读取；4类检查体系
> v3.2改动: 无需求新增no_need_reasons四项；知识沉淀文件字数≥500字门禁；素材库/知识沉淀路径统一

## 使用说明

Step 1执行时填写本模板。包含3部分：有需求记录、无需求记录、素材库更新记录。

## 素材库目录结构参考

```
素材库/
├── 索引.md                    ← 素材库总索引（每章强制读取，红线❌6）
├── 知识沉淀/                   ← 知识沉淀文件（WebSearch获取的信息）
│   ├── 制度流程/
│   ├── 场景技法/
│   ├── 创意参考/
│   └── 世界设定/
└── 文风库/                     ← 文风DNA文件（由pop-writer-v3-seed蒸馏产出，revise子skill硬阻塞依赖）
    └── {书名}.md
```

## YAML格式：info_acquired

### 有需求示例

```yaml
info_acquired:
  chapter: 5
  index_read: true                    # 红线❌6：索引已读取
  needs:
    - type: 制度流程
      query: "美国催债法FDCPA流程"
      source: "素材库/知识沉淀/制度流程/美国催债法流程.md"  # 素材库已有，直接读取
      result: "催债人不得在早8点前/晚9点后联系；不得向第三方透露债务信息；债务人可30天内要求验证"
      search_depth: "达标"            # v3.2新增：搜索深度标准达标标记（≥500字+具体流程）
    
    - type: 场景技法
      query: "绝境反杀的节奏控制"
      source: "WebSearch→素材库/知识沉淀/创意参考/绝境反杀节奏.md"  # WebSearch获取，已写入素材库
      result: "三拍节奏：绝境确认→反击准备→一击逆转；每拍2-3段"
      search_depth: "达标"            # v3.2新增：≥500字+具体案例
      search_rounds: 2                # v3.2新增：搜索轮数（首次不足→追搜1轮）
    
    - type: 已埋伏笔
      query: "MS-13线索呼应ch1老约翰之死"
      source: "活记忆+前章正文"
      result: "ch1老约翰临终说了'裂隙'，本章主角在帮派档案中发现裂隙关联MS-13"
    
    - type: 创意参考
      query: "帮派对峙的压迫感营造"
      source: "library套路库/压迫感营造.md"
      result: "空间收缩+声音渐弱+主角呼吸感知"
  
  no_need_reasons: null               # 有需求时为null
  
  merged: true                        # 是否已融入写作上下文
  kb_new_files:                       # 本章新建的素材库文件
    - path: "素材库/知识沉淀/创意参考/绝境反杀节奏.md"
      category: 创意参考
      keywords: [反杀, 节奏, 绝境]
      word_count: 650                 # v3.2新增：字数门禁≥500字
      search_depth_met: true          # v3.2新增：搜索深度标准达标
  index_updated: true                 # 素材库/索引.md是否已更新
  kb_dir: "素材库/知识沉淀"           # 素材库知识沉淀目录路径
```

### 无需求示例（v3.2门槛提高：需no_need_reasons四项）

```yaml
info_acquired:
  chapter: 8
  index_read: true                    # 红线❌6：索引已读取（即使无需求也必须读索引）
  needs: []                           # 无需求
  
  no_need_reasons:                    # v3.2新增：无需求时必须逐类说明理由，四项缺一=退回补填
    制度流程: "本章是日常生存场景，不涉及新的制度/流程"
    场景技法: "本章场景类型（室内对话）在ch3已有技法参考（素材库/知识沉淀/场景技法/室内对话节奏.md）"
    已埋伏笔: "本章不推进任何活跃线索，纯角色发展章"
    创意参考: "本章无特殊场景需要灵感参考"
  
  merged: true                        # 无需求但索引已读取确认
  kb_new_files: []                    # 无新文件
  index_updated: false                # 无新文件，索引无需更新
  kb_dir: "素材库/知识沉淀"           # 素材库知识沉淀目录路径
  consecutive_no_need_count: 1        # v3.2新增：连续无需求章数（≥3→标记退化预警）
```

## 字段说明

| 字段 | 类型 | 说明 | 红线/门禁 |
|:-----|:-----|:-----|:---------|
| chapter | int | 章号 | — |
| index_read | bool | 素材库/索引.md是否已读取 | **红线❌6：未读=退回补读** |
| needs | array | 4类检查结果清单（有需求时填写，无需求时为[]） | — |
| needs[].type | string | 需求类型：制度流程/场景技法/已埋伏笔/创意参考 | — |
| needs[].query | string | 查询内容 | — |
| needs[].source | string | 信息来源：素材库文件路径/WebSearch→素材库路径/library套路库/活记忆+前章正文 | — |
| needs[].result | string | 获取的关键信息摘要 | — |
| needs[].search_depth | string | 搜索深度标准达标标记（v3.2新增） | **门禁：不达标=补充至达标** |
| needs[].search_rounds | int | 搜索轮数（v3.2新增，默认1，最多3） | — |
| **no_need_reasons** | **object/null** | **无需求时4类不需求理由（v3.2新增）** | **门禁：四项未全部填写=退回补填** |
| no_need_reasons.制度流程 | string | 不需要查制度流程的理由 | — |
| no_need_reasons.场景技法 | string | 不需要查场景技法的理由 | — |
| no_need_reasons.已埋伏笔 | string | 不需要查已埋伏笔的理由 | — |
| no_need_reasons.创意参考 | string | 不需要查创意参考的理由 | — |
| merged | bool | 是否已融入写作上下文 | — |
| kb_new_files | array | 本章新建的素材库文件列表 | — |
| kb_new_files[].path | string | 文件路径：素材库/知识沉淀/{分类}/{文件名}.md | — |
| kb_new_files[].category | string | 分类：制度流程/场景技法/创意参考/世界设定 | — |
| kb_new_files[].keywords | array | 关键词列表（用于索引匹配） | — |
| kb_new_files[].word_count | int | 文件字数（v3.2新增） | **门禁：≥500字** |
| kb_new_files[].search_depth_met | bool | 搜索深度标准是否达标（v3.2新增） | **门禁：必须true** |
| index_updated | bool | 素材库/索引.md是否已更新 | 有新文件时必须true |
| kb_dir | string | 素材库知识沉淀目录路径 | 固定为"素材库/知识沉淀" |
| consecutive_no_need_count | int | 连续无需求章数（v3.2新增） | **门禁：≥3→标记退化预警** |

## 门禁汇总

| 检查项 | 失败动作 |
|:-------|:---------|
| **素材库/索引.md未读取（红线❌6）** | **退回补读索引** |
| **needs:[]但no_need_reasons未全部填写（四项）** | **退回补填** |
| **知识沉淀文件字数<500字** | **补充搜索结果至达标** |
| **知识沉淀文件搜索深度不达标（无具体案例/流程/数据）** | **追搜补充** |
| WebSearch获取的信息未写入素材库/知识沉淀 | 补写入后再继续 |
| 素材库新文件未更新索引 | 补更新索引后再继续 |
| **连续3章needs:[]** | **标记信息获取退化预警** |
