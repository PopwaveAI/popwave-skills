# 创作模板

> 所属: pop-writer-v3-emerge v1.2.0 · templates/
> 消费: Step 2 创作子agent产出
> 红线: ❌7 3子agent context隔离——传入精简context，不传会话历史

## 使用说明

创作子agent执行时填写本模板。包含4部分：上下文确认表、正文涌现结构、创作决策记录YAML、门禁表。

## 一、上下文确认表

创作子agent启动后，逐项确认输入context已加载：

| 确认项 | 来源 | 状态 |
|:-------|:-----|:-----|
| 种子六要素已加载 | `种子文档.md`（当前版本） | ☐ 已加载 / ☐ 缺失 |
| 活记忆七组件已读取 | `活记忆/活记忆.yaml` | ☐ 已读取 / ☐ 缺失 |
| 上章末尾已读取 | `正文/ch{上一章}.md` 最后~800字 | ☐ 已读取 / ☐ 首章无上章 |
| chapter_plan已加载 | Step 0产出 | ☐ 已加载 / ☐ 缺失 |
| info_acquired已加载 | Step 1产出 | ☐ 已加载 / ☐ 无需求(needs:[]) |
| 创作指南已加载 | `references/创作指南.md` | ☐ 已加载 |

> 注意：创作子agent**不加载文风DNA**。文风DNA在Step 3修订子agent加载。

## 二、正文涌现结构

### 开幕段落

```
{空间/时间锚定 → 情绪传递（1-2句）}
{如有感官锚点，第一句以该感官信号开头}
```

### 场景流

#### 场景1：{场景描述}
```
{场景目的：推进什么线索/什么压力源逼近}
{场景内连续流动——事件有机融合，不逐事件独立成段}
{事件转折1：___}
{事件转折2：___}
{事件转折3：___}
```

#### 场景间过渡
```
{1-2句自然过渡：时空跳转/感官切换/情绪转折}
```

#### 场景2：{场景描述}
```
{场景目的：___}
{场景内连续流动}
{事件转折1：___}
{事件转折2：___}
```

#### 场景间过渡
```
{1-2句自然过渡}
```

### 闭幕段落（章末钩子）

```
{钩子类型：悬念钩 / 情绪钩 / 行动钩}
{钩子内容：___}
{钩子强度：即时危机型 / 远期悬念型}
```

## 三、创作决策记录YAML

```yaml
creation_log:
  chapter: {章号}
  decisions:
    - point: "推进线索"
      choice: "{选择推进哪条线索，推进方式（触面/深化/回收）}"
      seed_basis: "{种子依据：种子冲突轴/活跃线索}"
    
    - point: "压力源选择"
      choice: "{选择哪个压力源逼近}"
      seed_basis: "{种子依据：种子压力矩阵}"
    
    - point: "金手指使用"
      choice: "{金手指怎么使用}"
      seed_basis: "{种子依据：种子金手指L?能力+限制}"
    
    - point: "角色决策"
      choice: "{主角做了什么决策}"
      seed_basis: "{种子依据：主角引擎驱动力?层+缺陷+决策风格}"
    
    - point: "章末钩子"
      choice: "{钩子类型和内容}"
      seed_basis: "{chapter_plan.hook}"
  
  word_count: {正文字数}
  scene_count: {场景数}
  event_count: {事件转折总数}
  pressure_sources_advanced: [{本章逼近的压力源列表}]
  plotlines_advanced: [{本章推进的线索列表}]
  rewards_given: [{本章给读者的爽点列表}]
```

## 四、门禁表

| 门禁项 | 检查结果 | 失败动作 |
|:-------|:---------|:---------|
| 场景流渲染（非逐事件） | ☐ 通过 / ☐ 退回 | 退回重写 |
| 本章有压力源逼近 | ☐ 通过 / ☐ 退回 | 退回重写 |
| 章末有钩子 | ☐ 通过 / ☐ 退回 | 退回重写 |
| 上下文确认表已填 | ☐ 通过 / ☐ 补填 | 补填后再继续 |
| 种子/活记忆存在 | ☐ 通过 / ☐ 终止 | 终止，提示先完成上游 |

> 注意：创作子agent**不做**文风DNA对齐/8020比例/AI观感词清理检查（交给Step 3修订子agent）。
