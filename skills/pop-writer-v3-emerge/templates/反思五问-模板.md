# 反思五问模板

> 所属: pop-writer-v3-emerge v1.0.0 · templates/
> 消费: Step 3 反思五问产出
> 红线: ❌2 反思不通过=回退重写本章

## 使用说明

每章Step 3反思时填写。五个读者体验问题，每个问题必须有具体正文引用作为证据。不接受"是/否"式回答，不接受笼统评价。

任一问题不达标 → 回退Step 2重写本章。

## 填写规范

| 字段 | 说明 |
|:-----|:-----|
| 判断结果 | ✅通过 或 ❌不达标，二选一 |
| 正文引用证据 | 必须引用正文原文片段，不接受笼统评价 |
| 分析 | 基于引用证据的具体分析 |
| 不达标处理 | 不达标时填写处理标记，达标时填"无" |
| 反思总结 | 五问结果汇总表 |
| 传递给Step 5 | 不达标项的处理标记汇总，供方向提示生成 |

## 防应付式填表机制

1. 每问必须有正文引用——不接受"是的，读者获得了奖励"，必须引用原文
2. 事件密度必须逐场景计数——不接受"节奏还行"
3. 伏笔检查必须对照活记忆——不接受"线索正常"
4. 章末钩子必须引用原文——不接受"有钩子"
