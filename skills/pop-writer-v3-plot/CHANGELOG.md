## v1.2.0 | 2026-06-28
- 重构：按 v3.5 涌现式写作管线 PRD（第5章）大改，从"任务链设计"转为"L2剧情单元卡设计"
- 取消种子文档：L2卡吸收种子文档全部运行时功能，成为唯一运行时活文档；plot 不再写入种子文档
- 新增 L2卡标准格式：元数据+物理坐标+一句话单元+结构分析表+嵌套子线+子线咬合+可迁移/不可迁移要点+单元边界信号
- 新增设定引用指针：结构分析表新增"设定引用"列，plot 设计时做信息需求预判，每章标注涉及的设定文件路径
- 新增动作链事件brief：事件brief必须用具体动作+因果链，禁止纯功能描述
- 新增子线因果咬合：子线咬合用因果推导（A→B→C），禁止矩阵填空
- 新增 library 参考：设计 L2 卡时必须从 library 匹配同类 L2卡（≥2张）作为结构标杆（信息注入不足是 v2 失败根因）
- 新增多层信息依赖：前L2卡实际内容（优先级1）+写作参考/设定（优先级2）+library同类L2卡（优先级3）+L3卡（优先级4可选）
- 新增信息需求判断三层流程：盘点本地已有→识别缺口→调用 pop-research 调研补缺
- 明确 plot 不事前设计 L3（L3 由 arc 事后提炼）
- 红线更新：4条核心红线（设定引用列/动作链brief/library参考/活跃子线≤5条）
- 步骤调整：step-1 卷方向锚（消费源改为写作参考/设定）+ step-2 L2卡设计（替代原任务链设计）
- 模板更新：新建 templates/L2卡-模板.md，废弃 任务链-模板.md
