# pop-writer-v3-plot · 任务链设计 v1.0.0

从v2 plot继承卷战略+剧情线理念，大改重构为涌现式任务链设计。不走大纲/章纲，任务链内化大纲功能。核心隐喻：小说世界=MMORPG，种子文档任务表=持续更新的任务日志。

## ❌ 质量红线

| 编号 | 红线 | 检查方式 |
|:-----|:-----|:---------|
| ❌1 | 任务链必须三层映射（终点/当前阶段/当前可执行），锁定阶段不显示可执行路径 | 逐目的地核对三层字段 |
| ❌2 | 每个阶段必须有完成条件+解锁条件 | 阶段完成条件表逐行校验 |
| ❌3 | 当前可执行任务≤5个（防膨胀） | 计数当前阶段可执行任务 |

## 路由

upstream `pop-writer-v3-seed` → **本skill** → downstream `expert-writer`

## 产出

- `卷纲/卷N-方向锚.md`（≤300字）：卷定位+起点终点+主冲突
- `种子文档.md` 主线任务段（写入）：三层映射任务链+金手指任务驱动节点

## 与v2 plot的区别声明

- 砍掉：危机交叉设计 / 分幕章锚点 / 密度审计
- 继承：卷战略（简化版）+ 剧情线 → 任务链（大改）
- 新增：三层映射（终点/当前阶段/当前可执行）+ 阶段完成/解锁条件 + 锁定机制
