# Step 1：卷方向锚设计

> 定位：产出卷方向锚，是方向参考非强制约束。为 step-2 的 L2 卡设计提供卷级坐标。

## 消费
- `写作参考/设定/`（seed 产出的全局设定：主角引擎/金手指/压力矩阵/冲突轴/核心卖点）
- `写作参考/索引.md`（本地已有设定/知识沉淀总目录）
- 用户输入（本卷方向意图）

> v3.5 变化：不再消费种子文档（种子文档已取消）。全局设定由 seed 写入写作参考/设定/。

## 产出
- `卷纲/卷N-方向锚.md`（≤300字）

## 做什么

### 1. 卷定位
本卷兑现全书哪个承诺（1句话）。从 `写作参考/设定/核心卖点.md` 的情绪承诺出发，明确本卷兑现哪一段。

### 2. 起点终点
角色/世界/力量/关系四维度的起点→终点状态变化：

| 维度 | 起点 | 终点 |
|:-----|:-----|:-----|
| 角色 |  |  |
| 世界 |  |  |
| 力量 |  |  |
| 关系 |  |  |

> 起点：参考前序卷终点 / 首卷参考 `写作参考/设定/` 的初始状态。
> 终点：本卷结束时四维度应到达的状态。

### 3. 主冲突
谁阻碍谁，争夺什么，失败代价（1-2句）。从 `写作参考/设定/冲突轴全量.md` 提取本卷激活的冲突线。

## 门禁
- 方向锚≤300字
- 卷定位必须有
- 四维度起点终点表完整
- 主冲突含"谁阻碍谁+争夺什么+失败代价"

## 产出自检
- [ ] 卷定位1句话已写
- [ ] 四维度起点终点表完整
- [ ] 主冲突已写（谁阻碍谁+争夺什么+失败代价）
- [ ] 总字数≤300字
