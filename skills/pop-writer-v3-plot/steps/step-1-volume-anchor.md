# Step 1：卷方向锚设计

> 定位：产出卷方向锚，是方向参考非强制约束

## 消费
- `种子文档.md`（目的地段）
- 用户输入

## 产出
- `卷纲/卷N-方向锚.md`（≤300字）

## 做什么

### 1. 卷定位
本卷兑现全书哪个承诺（1句话）。

### 2. 起点终点
角色/世界/力量/关系四维度的起点→终点状态变化：

| 维度 | 起点 | 终点 |
|:-----|:-----|:-----|
| 角色 |  |  |
| 世界 |  |  |
| 力量 |  |  |
| 关系 |  |  |

### 3. 主冲突
谁阻碍谁，争夺什么，失败代价（1-2句）。

## 门禁
- 方向锚≤300字
- 卷定位必须有

## 产出自检
- [ ] 卷定位1句话已写
- [ ] 四维度起点终点表完整
- [ ] 主冲突已写（谁阻碍谁+争夺什么+失败代价）
- [ ] 总字数≤300字
