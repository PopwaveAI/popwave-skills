# 修订checklist模板

> 所属: pop-writer-v3-revise v1.0.0 · templates/
> 消费: step-1 修订产出
> 红线: ❌1 文风DNA缺失=修订层硬阻塞终止；❌2 context隔离

## 使用说明

修订子skill执行时填写本模板。包含5项checklist和修订记录YAML。每项checklist逐条检查，不通过的项需修正或触发门禁。

## Checklist 1：文风对齐

| # | 检查项 | 标准 | 检查结果 | 修正记录 |
|:-:|:-------|:-----|:---------|:---------|
| 1.1 | DNA加载 | 文风DNA文件已加载（红线❌1硬阻塞） | ☐ 通过 / ☐ 硬阻塞终止 | {修正内容或"终止"} |
| 1.2 | 层A整体风格 | 句式/叙事距离/战斗节奏/对话留白匹配DNA层A | ☐ 通过 / ☐ 需修正 | {修正内容} |
| 1.3 | 层B场景卡 | 场景类型对应的观察句/原文锚定已对齐 | ☐ 通过 / ☐ 需修正 | {修正内容} |
| 1.4 | 8020比例 | 80%展示+20%内省（按场景类型比例） | ☐ 通过 / ☐ **退回** | {比例超标描述} |

**8020比例检查明细：**

| 场景 | 类型 | 展示% | 内省% | 内省次数 | 是否超标 |
|:-----|:-----|:------:|:------:|:--------:|:--------|

| {场景2} | | | | | |

> 门禁：8020超标（连续2段以上纯内省）= 退回重写

## Checklist 5：AI观感词清理

逐段扫描正文，记录命中：

| # | 检查项 | 命中标准 | 命中次数 | 命中段落 | 修正方式 |
|:-:|:-------|:---------|:--------:|:---------|:---------|
| 5.1 | 心理动词 | "他感到/他意识到/他仿佛" | {N}次 | {段落引用} | {改为展示} |
| 5.2 | 对比句式 | "不是…而是…" ≥ 2次 | {N}次 | {段落引用} | {减少或重写} |
| 5.3 | 解说员句式 | 叙事者解释设定 | {N}次 | {段落引用} | {改为对话/行动} |
| 5.4 | 视角越界 | 限知视角写非锚点内心 | {N}次 | {段落引用} | {删除越界} |

**命中类型总计：** {N}种

> 门禁：AI观感词命中 > 3种 = **退回create子skill重写**

## 修订记录YAML

```yaml
revision_log:
  chapter: {章号}
  
  style_alignment:
    dna_loaded: true  # 或 false（硬阻塞终止）
    layer_a_check: "{层A整体风格检查结果}"
    layer_b_check: "{层B场景卡检查结果}"
    ratio_8020: "通过/退回（{比例描述}）"
  
  character_enrichment:
    engine_alignment: "符合/不符合——{说明}"
    behavior_alignment: "符合/不符合——{行为准则对齐说明}"
    psychology_depth: "{决策点心理活动描述}"
  
  reward_verification:
    instant_reward: "{奖励内容或'无'}"
    progression_visible: "{变化内容或'不可见'}"
    emotion_arc: "{起点情绪}→{转折}→{终点情绪}"
  
  bug_fix:
    world_rule: "无违反/{违反内容+修正}"
    foreshadow: "无矛盾/{矛盾内容+修正}"
    golden_finger: "在限制内/{超出内容+修正}"
    timeline: "无矛盾/{矛盾内容+修正}"
  
  ai_word_cleanup:
    psychology_verb: {命中次数}
    contrast_pattern: {命中次数}
    narrator_explain: {命中次数}
    pov_breach: {命中次数}
    total_types: {命中类型总计}
    gate: "通过(≤3种)/退回(>3种)"
  
  word_count_before: {修订前字数}
  word_count_after: {修订后字数}
```
