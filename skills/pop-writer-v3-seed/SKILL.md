# pop-writer-v3-seed · 故事种子与世界生长语法

seed 的职责不是写完整世界观百科，而是给后续 plot 一套能运行故事的初始条件。

核心原则：**seed 不负责写完整世界，seed 负责定义世界的生长语法。**

## 角色边界

| 层级 | seed 要做 | seed 不做 |
|:-----|:---------|:----------|
| 硬设定 | 冻结世界类型、力量系统底层、主角金手指、核心禁忌、主线大压力 | 不展开完整神谱、地图、势力谱系 |
| 主角引擎 | 明确欲望、底线、行为准则、爽感动作模式 | 不替后续章节安排固定剧情线 |
| 设定生长 | 给出设定生成原则、设定缺口分类、冻结规则 | 不预先填满所有软设定 |
| 线索种子 | 放入压力、底线、资源、异常、潜在冲突对象 | 不提前命名完整剧情线 |
| 文风资产 | 蒸馏可模仿原文 DNA | 不在 seed 中写正文 |

## 必须产出

| 文件 | 用途 |
|:-----|:-----|
| `写作参考/设定/故事DNA.md` | 叙事性说明：这是什么故事、主角是谁、读者为什么继续看 |
| `写作参考/设定/世界宪法.md` | 硬设定：世界底层规则、核心禁忌、不可变事实 |
| `写作参考/设定/力量体系.md` | 能力如何获得、升级、反馈、受限 |
| `写作参考/设定/主角引擎.md` | 欲望、底线、决策风格、行动爽点、行为禁忌 |
| `写作参考/设定/金手指.md` | 金手指怎么在事件中制造爽感，含 2-3 个微型场景 |
| `写作参考/设定/核心卖点.md` | 一句话卖点、情绪承诺、主要爽点类型 |
| `写作参考/设定/设定生长语法.md` | 后续 plot 按需孵化软设定的规则 |
| `写作参考/设定/初始种子池.md` | 未解决压力、角色底线、可增长资源、潜在冲突对象、世界异常信号 |
| `活记忆/活记忆.yaml` | baseline#0，自然语言状态快照 |
| `写作参考/索引.md` | 设定文件索引 |
| `写作资产/文风库/{书名}.md` | 文风 DNA 原文摘录和可模仿特征 |

## 设定分层

### 1. 硬设定：必须提前冻结

只写会影响所有章节因果的底层规则：

- 世界基本类型：例如诸神黄昏前夜、修仙王朝崩坏、都市诡异复苏。
- 力量系统底层：例如经验升级、境界突破、污染交换、契约召唤。
- 主角金手指机制：它如何触发、如何反馈、如何升级、有什么禁忌。
- 核心不可逆禁忌：什么不能免费获得，什么代价不能被取消。
- 主线大压力：灾变、追杀、阶层压迫、神明苏醒、战争逼近。

### 2. 软设定：只留下缺口

不要预先补全。写成“剧情需要什么类型的设定”，交给 plot 在事件碰撞中生成。

```markdown
## 设定缺口
- 主角早期练级需要一种低风险但专业感强的经验来源。
- 当前城市需要一个能威胁主角底线的地头势力。
- 世界灾变需要一个早期异常事件露头。
- 主角身边人需要一个社会性压力来源。
```

### 3. 浮现设定：出生后冻结

plot 只有在事件需要时才允许生成软设定。生成后写入设定账本，后续不可随意改。

## 初始种子池

前置不命名完整剧情线，只播种：

```markdown
## 未解决压力
- 贫民区安全屋脆弱，妹妹随时可能被盯上。

## 角色底线
- 谁碰妹妹，主角从回避切换为清算。

## 可增长资源
- 等级、钱、技能、名声、关系、情报。

## 潜在冲突对象
- 本地帮派、人贩子、神殿、邪神信徒。

## 世界异常信号
- 剥皮事件、神殿失灵、怪物越界、官方漠视。
```

plot 会让这些种子碰撞；arc 再把长出来的轨迹命名为剧情线。

## 红线

| 编号 | 红线 |
|:-----|:-----|
| R1 | 不产出百科式全设定。软设定只写缺口，不提前补完。 |
| R2 | 硬设定必须可约束后续因果，不能是氛围描述。 |
| R3 | 初始种子池只能写压力/底线/资源/冲突对象/异常信号，不能提前写完整剧情线。 |
| R4 | 金手指必须用 2-3 个具体爽感微场景说明，不只写功能。 |
| R5 | 文风 DNA 必须有可模仿原文摘录，不能只有抽象风格词。 |

## 交接给 plot

seed 完成后，plot 读取：

1. 硬设定：`世界宪法.md`、`力量体系.md`、`金手指.md`、`主角引擎.md`。
2. 生长材料：`设定生长语法.md`、`初始种子池.md`、`活记忆.yaml`。
3. 叙事方向：`故事DNA.md`、`核心卖点.md`、文风 DNA。

plot 的任务不是执行固定剧情线，而是让这些种子在单元状态中碰撞，跑出可拆书的剧情轨迹。
