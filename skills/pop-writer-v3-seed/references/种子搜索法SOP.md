# 种子搜索法 SOP

> 适用: pop-writer-v3-seed Step 1
> 定位: 三界压力调研标准流程——社会界/自然界/超自然界压力源系统化调研

## 一、为什么需要三界压力框架

涌现式管线的核心是"压力矩阵驱动剧情"。压力矩阵不能是大模型空想——必须来自目标世界真实的压力维度。

不同类型小说的压力来源截然不同：现代现实主义靠制度压迫，末日生存靠自然环境，玄幻修仙靠超自然法则。单一框架无法覆盖所有类型。

**三界压力框架**将所有压力源归入三个域，按需调研：

| 压力界 | 核心问题 | 适用类型 | 典型压力源 |
|:-------|:---------|:---------|:-----------|
| 社会界 | 人吃人的制度是什么？ | 现代/都市/历史/职场 | 法律不公、经济盘剥、暴力威胁、权力倾轧 |
| 自然界 | 环境怎么杀人？ | 末日/生存/荒野/科幻 | 极端气候、疾病瘟疫、猛兽威胁、资源匮乏 |
| 超自然界 | 超自然法则怎么压迫？ | 玄幻/仙侠/灵异/异能 | 魔法诅咒、神灵意志、异种入侵、规则束缚 |

大多数小说混合使用2-3个界。例如都市异能 = 社会界+超自然界，末日生存 = 自然界+社会界。

## 二、调研流程

### 阶段0：界属判定

基于 Step 0 创意共识单的"压力来源方向"，判定本书涉及哪些压力界：

| 判定问题 | 判据 |
|:---------|:-----|
| 主角面对的是人/制度/社会？ | 是→社会界 |
| 主角面对的是自然环境/生存环境？ | 是→自然界 |
| 主角面对的是超自然力量/魔法/神灵？ | 是→超自然界 |

**至少覆盖1个界，通常2-3个界。** 每个界独立调研，交叉时产生跨界困境。

### 阶段1：分界调研

#### 社会界压力调研

| 调研维度 | 调研问题 | 信息源 |
|:---------|:---------|:-------|
| 制度压迫 | 目标社会有什么制度在不公正地压迫弱者？ | WebSearch + library |
| 经济盘剥 | 经济结构怎么榨取底层？高利贷/垄断/失业？ | WebSearch |
| 暴力威胁 | 暴力组织如何控制地盘和人？ | WebSearch + library |
| 权力倾轧 | 权力结构如何挤压主角？官场/职场/家族？ | WebSearch + library |
| 群体排斥 | 群体如何排斥异类？种族/阶级/宗教？ | WebSearch |
| 信息不对称 | 谁在操纵信息？主角被蒙蔽什么？ | 用户设定 |

#### 自然界压力调研

| 调研维度 | 调研问题 | 信息源 |
|:---------|:---------|:-------|
| 生存环境 | 目标环境的致命特征？气候/地形/辐射？ | WebSearch + 用户设定 |
| 疾病瘟疫 | 疾病如何威胁主角？无医疗会怎样？ | WebSearch |
| 猛兽威胁 | 生态链中的掠食者如何威胁人类？ | WebSearch + library |
| 自然灾害 | 什么灾害会突然改变生存条件？ | WebSearch |
| 资源匮乏 | 水/食物/能源/材料的稀缺程度？ | 用户设定 + WebSearch |
| 生理极限 | 饥饿/脱水/疲劳/衰老如何制约行动？ | WebSearch |

#### 超自然界压力调研

| 调研维度 | 调研问题 | 信息源 |
|:---------|:---------|:-------|
| 魔法诅咒 | 什么超自然力量在束缚/威胁主角？ | 用户设定 + library |
| 神灵意志 | 神灵/天道/规则意识如何干预主角？ | 用户设定 + library |
| 异种入侵 | 什么非人类存在在威胁人类？ | 用户设定 + library |
| 规则束缚 | 世界法则/系统规则如何限制主角行动？ | 用户设定 |
| 灵魂侵蚀 | 精神/灵魂层面的威胁如何运作？ | 用户设定 + library |

### 阶段2：压力源转化

每个调研结果转化为压力源：

| 转化步骤 | 问题 |
|:---------|:-----|
| 界属 | 社会界 / 自然界 / 超自然界？ |
| 机制 | 这个压力怎么运作？ |
| 杀人方式 | 不行动/被盯上 = 什么后果？ |
| 显性表现 | 读者看到什么具体场景？ |
| 活跃阶段 | 这个压力源在什么章节显性？ |
| 来源标注 | 来自哪个调研结果？ |

### 阶段3：交叉困境识别

多个压力源交叉时产生道德困境。跨界交叉尤其强力：

| 交叉类型 | 示例 |
|:---------|:-----|
| 社会界 × 社会界 | 医疗缺失 × 帮派控制 = 受伤不能就医，帮派在搜人 |
| 社会界 × 自然界 | 资源匮乏 × 暴力威胁 = 必须冒险进入危险区域获取资源 |
| 社会界 × 超自然界 | 身份暴露 × 魔法诅咒 = 被追杀时诅咒发作无法还手 |
| 自然界 × 自然界 | 极端寒冷 × 猛兽威胁 = 不能生火（引来猛兽）但会冻死 |
| 自然界 × 超自然界 | 疾病瘟疫 × 灵魂侵蚀 = 身体病痛引发精神崩溃 |
| 超自然界 × 超自然界 | 规则束缚 × 神灵意志 = 服从规则被神罚，违反规则被法则反噬 |

## 三、调研质量检查

| 检查项 | 合格标准 |
|:-------|:---------|
| 压力源数量 | ≥3个（红线❌3） |
| 界属标注 | 每个压力源标注所属界 |
| 来源标注 | 每个压力源有来源（WebSearch/library/用户） |
| 交叉困境 | ≥3个交叉困境（至少1个跨界） |
| 显性表现 | 每个压力源有具体的读者可见场景 |

## 四、与素材库的关系

调研过程独立存盘到 `素材库/研究档案/`：
- `种子展开图-{书名}.md`：分界调研详情（制度清单/环境特征/超自然法则）
- `交叉困境分析-{书名}.md`：交叉困境详细分析

种子是引擎索引，研究档案是引擎详情。
