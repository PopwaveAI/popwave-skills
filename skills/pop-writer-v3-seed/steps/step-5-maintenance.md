# Step 5：写作参考维护

> 维护 seed 产出的设定文件、索引和后续 plot 可用的生长材料。

## 维护原则

- `写作参考/设定/` 是硬设定和生长语法的 ground truth。
- 软设定不在 seed 阶段补完；由 plot 在事件运行中孵化，arc 在正文后冻结。
- `写作参考/索引.md` 必须指向所有当前有效设定文件。
- 已废弃设定不删除，移入或标记在 `写作参考/已废弃/`。

## 需要维护的文件

| 文件 | 维护方式 |
|:-----|:---------|
| `故事DNA.md` | 只在核心卖点/主角方向变化时更新 |
| `世界宪法.md` | 极少改动；改动会影响全书因果 |
| `力量体系.md` | 新能力规则经 arc 确认后追加 |
| `主角引擎.md` | 主角底线/行为准则变化需用户确认 |
| `金手指.md` | 新用法经正文验证后追加 |
| `核心卖点.md` | 读者体验承诺变化时更新 |
| `设定生长语法.md` | 调整 plot 孵化软设定的规则 |
| `初始种子池.md` | 只作为开局种子；后续种子由 arc 活记忆和设定账本接力 |
| `写作参考/索引.md` | 每次新增/废弃设定文件后更新 |

## 交接给 plot 的索引格式

```markdown
# 写作参考 · 索引

## 硬设定
| 文件 | 内容 | 状态 |
|:-----|:-----|:-----|
| 设定/世界宪法.md | 世界底层规则 | active |
| 设定/力量体系.md | 能力与成长规则 | active |

## 生长材料
| 文件 | 内容 | 状态 |
|:-----|:-----|:-----|
| 设定/设定生长语法.md | 软设定出生和冻结规则 | active |
| 设定/初始种子池.md | 开局压力/底线/资源/异常/缺口 | active |

## 文风资产
| 文件 | 内容 | 状态 |
|:-----|:-----|:-----|
| ../写作资产/文风库/{书名}.md | 文风 DNA | active |

## 已废弃
| 文件/设定 | 废弃原因 | 替代 |
|:----------|:---------|:-----|
```

## 红线

- 不在维护步骤里新增完整剧情线。
- 不把 plot/arc 的运行结果回写成 seed 初始事实，除非它已经成为全书硬设定。
- 不删除历史设定，避免追溯断裂。
