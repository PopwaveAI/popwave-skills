# 种子文件夹结构模板

> 管线：涌现式写作v3.2 · 活种子（文件夹版）
> 说明：种子不再是单文件，是文件夹。每个要素独立成文件，_index.yaml是索引，_log.md是变更日志。

## 目录结构

```
种子/
├── _index.yaml          # 种子引擎索引（版本+要素清单+last_updated_ch）
├── _log.md              # 变更日志
├── 核心卖点.md           # Step 0产出
├── 压力矩阵.md           # Step 1产出（含三界标注）
├── 主角引擎.md           # Step 4产出（含行为准则）
├── 金手指.md             # Step 2产出
├── 冲突轴.md             # Step 1产出（含活跃线索）
├── 成长路径.md           # Step 4产出
└── 目的地.md             # Step 4产出
```
