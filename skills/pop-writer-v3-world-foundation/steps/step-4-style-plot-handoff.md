## 文风DNA

文风必须服务网文爽点：

| 模块 | 要求 |
| --- | --- |
| 叙述视角 | 主视角/有限视角/信息释放 |
| 句式节奏 | 行动句、反馈句、信息句比例 |
| 爽点外显 | 面板、数字、旁人反应、敌人误判、资源到账 |
| 氛围使用 | 氛围段如何快速回到动作 |
| 禁用倾向 | 严肃文学腔、过度隐喻、长段心理、百科说明 |
| 示例语气 | 3-5 句本书可用语感 |
