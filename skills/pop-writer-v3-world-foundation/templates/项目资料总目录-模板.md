# 项目资料总目录

> 本文件是 agent 找资料的一级入口。它只做路由，不粘贴长设定。

## 总原则

- 正式产物优先于过程文件。
- 过程文件只能用于追溯理由、补证据、查风险，不得直接覆盖正式设定。
- create 读取切片，不读取全量设定。
- review 先读正文原文，再按问题追加资料。

## 任务路由

| 任务 | 必读 | 按需读 | 禁止误读 |
| --- | --- | --- | --- |
| seed 复查 | 新书立项PRD、长篇承诺书、seed-产物索引 | 研究档案、复刻路由表、资料覆盖率声明 | world/plot 后续产物反向改 seed |
| world-foundation | 新书立项PRD、长篇承诺书、seed-to-world交接包、研究索引 | 多元素整合记录、复刻路由表、外部母版卡 | 卷纲/正文反向决定 L1 |
| plot 卷纲 | PRD、Foundation五件套、L1世界包、角色储备池、金手指、开局契约、读者表达偏好、文风DNA、world-产物索引 | 设定涌现目录、Library查询方向、跨媒介母版雷达 | 正文创作记录当剧情来源 |
| plot 幕纲 | 卷纲、本卷选角表、可选剧情燃料池、角色储备池切片、读者表达偏好、文风DNA | 幕级外部母版、设定账本切片 | 跳过选燃料直接写分章 |
| create 正文 | 当前章施工卡、上章末尾、状态快照、本章角色锚点、读者表达偏好、文风DNA切片、相关设定切片 | 幕级外部燃料台、review下一章修正规则 | 整份角色储备池、全量设定涌现目录 |
| review 审稿 | 正文原文、对应章施工卡、创作记录 | 文风DNA、读者表达偏好、角色锚点、设定账本、运行日志 | 不读原文只看摘要 |
