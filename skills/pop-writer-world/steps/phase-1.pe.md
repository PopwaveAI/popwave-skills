# Phase 1：L1 设定层推演（新分类）

> 该文件同时作为执行指令和参考指南，无需加载外部 ref 文件。

## 前置条件
story-engine.yaml 已锁定，金手指概念设计完成，跨域素材聚合完成。

## 新六件套分类（已验证）

| 维度 | 核心问题 |
|:-----|:---------|
| 01 · 世界蓝图 | 世界的基本面貌是什么？时空/法则/基调 |
| 02 · 力量体系 | 如何变强？等级/路径/天花板 |
| 03 · 历史与驱动力 | 前因后果是什么？大事件/核心矛盾 |
| 04 · 物种与天赋 | 谁生活在这里？种族/血脉/天赋 |
| 05 · 势力格局 | 谁说了算？组织/权力/派系 |
| 06 · 资源与物品 | 什么好东西？道具/资源/货币 |

## 核心原则

1. **创作直觉优先**：不需要判断"这是底层逻辑还是表层规则"——直接回答"这个世界有什么力量体系"就好
2. **粒度均衡**：每个维度的信息量大致相当
3. **金手指标签化**：主角专属内容用 `[主角]` 前缀标注在各维度中，不设独立维度

## 执行步骤

依次产出六个文件到 `小说世界设定/L1-`：`01-世界蓝图.md` / `02-力量体系.md` / `03-历史与驱动力.md` / `04-物种与天赋.md` / `05-势力格局.md` / `06-资源与物品.md`

每篇完成后自查该维度的必填子字段列表。每篇标注 `[主角]` 标签并末尾汇总。

## 各维度字段标准与示例

### 01 · 世界蓝图

> **@consumed_by**: plot.Step6(volume-XX.md — 地理总览+时空规则), plot.Step1(act.core_conflict — 世界基调), writer.Step1(chXXX-design块B — 场景空间设计)
> **@fields_for_agent**: 地理总览→plot地图锚点; 世界基调→plot tone_note（取值须为枚举：`暗黑生存/史诗冒险/日常温馨/悬疑惊悚`）

| 子字段 | 说明 | 字数建议 |
|:-------|:-----|:--------|
| 地理总览（→ volume-XX.md） | 世界的地域范围、区域划分、关键地点。本段直接供 plot canvas 消费 | ≥300 |
| 位面/时间规则 | 位面结构、时间流动规则、特殊地域法则 | ≥200 |
| 核心法则 | 世界运行的底层规则（物理/魔法/天道） | ≥300 |
| 力量来源 | 力量从哪来（灵气/魔力/原能/信仰） | ≥200 |
| 世界基调（→ plot tone_note） | 枚举值。取值：`暗黑生存` / `史诗冒险` / `日常温馨` / `悬疑惊悚` | 1 行 |

**示例（遮天）**：
```
01-世界蓝图.md

## 时空结构
诸天万域格局。北斗星域为核心大星域，包含东荒、西漠、南岭、北原、中州五大区域。
生命禁区散落分布（太初古矿、轮回海、不死山、上苍、葬天岛）。
星空古路连接各大星域。时间流动规则：人界一天、仙界一年。

## 核心法则
大道压制。天地法则会压制超过当前境界的修士，导致古之大帝级强者难以存续太久。
每过数万年一次的"黑暗动乱"——禁区至尊收割众生生命来延续己身。
[主角] 荒古圣体：被天地压制，每突破一个境界需要比常人多走九个小台阶。

## 力量来源
源——天地灵气凝聚的结晶，修士修炼主要靠源。
悟道——通过感悟天地法则获得突破，"道"的境界比灵气积累更重要。

## 世界基调
宏大、苍凉、悲壮。人与天争命，修士在压迫中寻求超脱。
```

### 02 · 力量体系

| 子字段 | 说明 | 字数建议 |
|:-------|:-----|:--------|
| 境界总览 | 从入门到巅峰的完整等级链 | ≥200 |
| 核心路径 | 主要修炼/升级方式（功法/冥想/战斗/吞噬） | ≥200 |
| 突破条件 | 每级怎么突破（积累/机缘/领悟/代价） | ≥200 |
| 力量天花板 | 故事结束时能达到的最强级别是什么？ | ≥100 |
| 跨级约束 | 差几级就不可战胜？什么情况可以越级？ | ≥100 |

**示例（吞噬星空）**：
```
02-力量体系.md

## 境界总览
地球篇：武者→战士→战将→战神→超越战神
宇宙篇：行星级→恒星级→宇宙级→域主→界主→不朽→宇宙之主→宇宙最强者→真神
[主角] 罗峰的晋升路径特殊：前期靠陨墨星传承，中期靠金角巨兽分身(基因倍数×10)，
        后期靠九劫秘典(打破法则上限)。

## 核心路径
基因原能修炼法——通过引动宇宙能量淬炼身体。
精神念师——少数人拥有的精神力量体系，可与基因原能并行修炼。
本源法则感悟——参悟宇宙本源法则（时间/空间/金/木/水/火/土/风/雷/光）。

## 突破条件
- 学徒→行星级：积累足够原能，精神共鸣突破
- 每级需本源法则感悟配合，不能只靠能量堆积

## 力量天花板
真神（起源大陆顶端），全书结尾达到。

## 跨级约束
同级五五开 / 越1级需合理解释 / 越2级极特殊情境，不可复用 / 越3级体系崩了
```

### 03 · 历史与驱动力

> **@consumed_by**: plot.Step1(act.core_conflict — 核心矛盾+前提大事件), writer.Step1(背景对话/遗迹描述 — 前提大事件)
> **@fields_for_agent**: 核心矛盾→plot M1 线成因; 前提大事件→writer 角色对话/环境叙事的历史锚点

| 子字段 | 说明 | 字数建议 |
|:-------|:-----|:--------|
| 纪元/时代划分 | 几个大的历史时期及特征 | ≥150 |
| 前提大事件（→ writer 背景对话） | 故事开始前发生的关键事件 | ≥200 |
| **核心矛盾（→ plot M1 线）** | YAML 块：`core_conflict: "秩序vs混沌"` —— 这个世界当前面临的根本性对抗。plot 的 act.core_conflict 直接引用此值 | 1 行 YAML + ≥200 说明 |

**示例（深渊主宰）**：
```
03-历史与驱动力.md

## 纪元/时代划分
- 耐瑟帝国时代（远古）：魔法文明巅峰，耐瑟卷轴流传
- 诸神纪元：神祇统治世界，神权高于一切
- 动荡之年/圣者浩劫（当前）：神力风暴，诸神被迫以圣者形态降临凡间

## 前提大事件
动荡之年的神力能量风暴席卷多元宇宙，所有神灵暂时失去神力。
诸神被迫以圣者身份降临凡间，大量神灵陨落，神位出现真空。

## 核心矛盾
秩序vs混沌。圣者浩劫导致权力真空，各方势力争夺神位。
主角索伦在这个混乱中逐步崛起，以守序邪恶的方式走向深渊主宰之路。
```

### 04 · 物种与天赋

> **@consumed_by**: plot.character-list(NPC 种族标注), writer.Step1(非人类角色出场时读取), deconstructor.T2(对应维度反向分析)
> **@fields_for_agent**: 每条种族 4 字段(name/traits/playable/faction_affiliation)。有下游管线消费的保底字段是 traits。

| 子字段 | 说明 | 字段格式 | 字数建议 |
|:-------|:-----|:--------|:--------|
| 主要种族 | 智慧种族。每条包含: name / traits / playable(是否可选) / faction_affiliation | 结构化列表 | ≥200 |
| 异兽/怪物 | 非智慧生物的威胁等级和基本生态 | 散文 | ≥200 |
| 特殊体质/血脉/天赋 | 极少数拥有者才有的天生成就 | 散文。`[主角]` 标签标注 | ≥150 |

**示例（遮天·结构化）**：
```
04-物种与天赋.md

## 主要种族（→ plot character-list + writer 非人类角色）

| name | traits | playable | faction_affiliation |
|------|--------|:-------:|---------------------|
| 人族 | 数量最多，适应力极强，无种族天赋但职业路径最灵活 | 是 | 各大势力均有分布 |
| 太古万族 | 远古时期统治世界的强大种族 | 否（敌对） | 生命禁区 |
| 妖族 | 万物皆可成妖，顶级妖族可与圣地抗衡 | 是 | 妖族联盟 |

## 异兽/怪物
- CR1-3：狗头人、巨型毒蝎、石肤蜥蜴 — 分布在城市废墟和低阶洞穴
- CR4-6：石魔像、双足飞龙 — 地下遗迹守卫

## 特殊体质/血脉/天赋
- 荒古圣体：万古无双，肉身强横。被天地压制，修炼比常人难九倍。
  [主角] 叶凡继承荒古圣体。
- 先天道胎：天生亲近大道，悟道速度极快。
- 苍天霸体：与荒古圣体并列，对决是万古宿命
```

### 05 · 势力格局

| 子字段 | 说明 | 字数建议 |
|:-------|:-----|:--------|
| 主要势力 | 组织/国家/派系列表 | ≥200 |
| 权力分布 | 谁掌控什么资源/地盘 | ≥100 |
| 势力关系 | 各势力间的对抗/合作/中立 | ≥150 |

**示例（吞噬星空，分两层）**：
```
05-势力格局.md

## 地球篇
- HR联盟：全球大财阀联合组织，掌控经济命脉
- 极限武馆：全球最大武馆，培养武者
- 雷电武馆：第二大武馆，与极限武馆竞争
- 战神宫：战神级强者的组织

## 宇宙篇
- 虚拟宇宙公司：最大的信息/交易平台
- 巨斧斗武场：以战斗为核心的势力
- 宇宙佣兵联盟：雇佣兵组织
- 宇宙星河银行 / 第一银行：金融势力

[主角] 罗峰先后加入极限武馆→虚拟宇宙公司，成为巨头之一
```

### 06 · 资源与物品

| 子字段 | 说明 | 字数建议 |
|:-------|:-----|:--------|
| 修炼资源 | 提升力量的消耗品 | ≥150 |
| 关键道具 | 有特殊功能的物品 | ≥150 |
| 货币体系 | 用什么交易？ | ≥50 |

**示例（遮天）**：
```
06-资源与物品.md

## 修炼资源
源：天地灵气结晶，修士修炼必需品。分普通源/异种源/神源/仙源
不死神药：可延长寿命、助人突破，如九妙神药、真龙不死药

## 关键道具
- 万物母气鼎（[主角] 叶凡的本命帝兵，可成长）
- 帝兵：大帝级兵器（荒塔、仙钟、绿铜鼎等）
- 仙金：铸造帝兵的材料（龙纹黑金、凰血赤金、永恒蓝金等）
```
