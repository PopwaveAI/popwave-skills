## 五、卷级节奏约束（★新增 — world→plot 防崩协议）

> **问题背景：** 2026-06-22 东方版深渊主宰案例中，Phase 7 只写了「全书跃迁路径」，没有约束「每卷的能级跨度」。结果卷一 plot 设计了 50 章，主角从开脉未成→开脉圆满（仅半阶跨度），PRD 承诺「武侠层 ch1-ch30」完全落空。

### 5.1 每卷能级跨度

| 卷号 | 起点能级 | 终点能级 | 最小跨度 | 最大章数 | 门禁说明 |
|:-----|:---------|:---------|:--------|:--------|:---------|
| vol-1 | [起点] | [终点] | ≥1完整小阶 | ≤55 | 单卷跨度<1阶=违规，退回节奏协商 |
| vol-2 | [起点] | [终点] | ≥1完整小阶 | ≤55 | |
| vol-3 | [起点] | [终点] | ≥1.5完整小阶 | ≤55 | 中期跨度适当拉大 |
| ... | ... | ... | ... | ... | |

**门禁规则：**
1. 单卷主角能级跨度 < 1 完整小阶 → 退回做节奏协商
2. plot 阶段接收时先验证：本卷设计章数是否超过约定的 max_chapters → 超过退回
3. 如果 PRD 有明确的阶段承诺（如「武侠层 ch1-ch30」），以 PRD 为更高优先级

### 5.2 子阶拆解规则（防"憋50章才第一次反馈"）

如果某大境界的跨度较大（如后天→先天），建议在境界内拆出明确的感知子阶：

| 大境界 | 建议子阶数 | 反馈节奏目标 |
|:-------|:----------|:-------------|
| 开局最低阶 | ≥3 感知子阶 | 前 15 章内至少给 1 次"变强了"信号 |
| 中间阶 | ≥2 感知子阶 | 每 20 章至少 1 次信号 |
| 高阶 | ≥1 感知子阶 | 每 25 章至少 1 次信号 |

**案例（2026-06-22 东方版深渊主宰）：**
```
改前：开脉未成 → 开脉圆满（半阶，50章无中间反馈）
改后：开脉未成 → 第1条正经打通(ch15) → 功法获+3-4条(ch25) → 
      第10条归墟石突破(ch35) → 12条全通+觉醒(ch50)
      → 15章、25章、35章各有一次成长反馈
```

### 5.3 卷级节奏约束表（产出格式）

```yaml
per-volume-span:
  vol-1:
    start: [起点能级]
    end: [终点能级，门禁≥1阶跨度]
    max_chapters: 55
    sub-stage-breakpoints:
      - [第N章: 第一次子阶突破]
      - [第N章: 第二次子阶突破]
    prd-alignment: [PRD的阶段承诺，如"武侠层ch1-ch30"]
```
