# CHANGELOG

## v5.0.0 — 2026-07-14

### 重磅：解决"种子不足以支撑一本书"+"脑洞不够精彩"两个致命问题

**Step 0 重构：加载赛道文档 → 脑洞碰撞**
- 新增3候选概念生成机制：从用户创意向3个不同方向推演，每个候选必须有差异化的核心钩子
- 新增候选PK环节：兴奋度/可持续性(250章+)/独特性/执行难度逐项对比，推荐+吸收败者强点
- 新增质量门禁：没有任何一个候选让你"想立刻读到正文"→回到0b重新生成
- 根因：v4.0重写时丢失了v3.5 step-1-collide.md的碰撞机制，变成线性提炼

**Step 1 增强：核心卖点定位加深度要求**
- 新增"独特性声明"：同赛道100本书里读者为什么选这本
- 新增"核心悬念"：贯穿全书250章的一个具体谜题
- 根因：原SOP只要求"一句话卖点"，不足以驱动250章

**Step 2 增强：角色地基从只建主角扩展为主角+对手+关系动力学**
- 新增核心对手身份/欲望要求（必须与主角欲望形成结构性对立）
- 新增关系动力学要求（张力来源：利益冲突/理念对立/信息差/信任博弈）
- 根因：只有主角没有对手=250章没有持续施压者

**Step 3 增强：世界观从"与冲突相关的设定"扩展为4系统全覆盖**
- 新增经济系统（财路和分配机制）
- 新增政治系统（谁定规则/规则怎么压迫主角/主角怎么利用规则缝隙）
- 新增文化规范（什么行为正常但读者震惊=代入落差）
- 新增信息不对称（信息分层流动/主角的信息差优势）
- 新增"主角可利用的缝隙"列
- 根因：只有地理+势力+力量体系=废设定，250章会弹尽粮绝

**Step 7 增强：体量规划从"第一卷一句话"升级为全书卷级表+长线伏笔清单**
- 体量规划改为多卷表（方向/卷末质变/新元素引入/悬念递进）
- 新增长线伏笔清单（≥5条，标注埋设卷/预计揭晓卷/回声节奏）
- 根因：只有3句话走向=plot-design没有足够弹药规划250章

**新增种子压力测试（阶段一收尾质量门禁）**
- 6项检查：脑洞兴奋度/冲突密度/世界纵深/角色生态/长线续航/赛道辨识度
- 任一不满足→回到对应Step补强，不进阶段二

**红线从9条扩展到13条**
- 新增红线10-13：脑洞碰撞/种子深度/对手体系/世界纵深/长线伏笔

**速查表新增6行**
- 脑洞碰撞/世界纵深/角色地基(含对手)/种子深度/长线伏笔/体量规划(多卷)

**种子文档输出格式更新**
- 新增脑洞碰撞记录表
- 核心卖点新增独特性声明+核心悬念
- 角色拆分为主角+核心对手+关系动力学
- 世界观新增4系统覆盖要求
- 体量规划改为多卷表+长线伏笔清单

**模板更新**
- `templates/seed-doc.tpl.md` 同步更新为新输出格式
