# Step 3: 分析 + 输出报告

## 输入
第二步采集的全部原始数据。

## 执行
1. 提取关键引用
2. 用户态度分层（激烈反对 → 沉默实用 → 实战派 → 营销号）
3. 痛点拆解（可解决/不可解决/需规避）
4. 输出报告

## 产出格式

```markdown
# <平台名> <调研主题> 调研报告

> **数据来源**：<平台名> — <搜索词/版块>
> **采集方式**：<工具名>
> **数据规模**：<X> 个帖子/视频，<Y> 条回复/评论
> **采集时间**：<日期>

## 一、采集概况
## 二、用户真实态度（分层）
## 三、使用的工具/方法
## 四、痛点拆解
## 五、产品机会
## 六、产品/营销/风险三维建议
```

## 门禁
- [ ] 报告含 >= 3 个平台的数据（多平台调研时）
- [ ] 每类观点附 >= 1 条原文引用
- [ ] 标注了采集方式和数据来源

## 产出
标准化调研报告 Markdown 文件。
