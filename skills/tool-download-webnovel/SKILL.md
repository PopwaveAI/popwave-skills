# tool-download-webnovel · 网文 TXT 直链下载

> **定位：一条路走到底。** 不爬目录页、不逐章抓取、不绕弯路。搜到直链就下，搜不到就报。

---

## 速查表

| 操作 | 命令/动作 | 预期耗时 |
|:-----|:----------|:---------|
| 搜索直链 | `任意搜索引擎 site:pan.baidu.com 书名 txt` | 10-30s |
| 下载 TXT | `curl/wget 直链URL -o 书名.txt` | 10-60s |
| 输出位置 | `d:\popwave-skills\{书名}.txt` | — |
| 下游消费 | pop-decon 自动读取 | — |

## ❌ 质量红线

| # | 规则 |
|:-:|:-----|
| ❌1 | **产出只留摘要** — 下载完成后对话中只说"已下载 {书名}.txt，大小 {N}MB，前 100 字预览：{片段}。"不粘贴全文。 |

## 执行流程（SOP）

> 各步骤详细操作见：[steps/step-1-search.md](steps/step-1-search.md) → [steps/step-2-download.md](steps/step-2-download.md) → [steps/step-3-verify.md](steps/step-3-verify.md)

### 第一步：搜索直链

详见 [steps/step-1-search.md](steps/step-1-search.md)。

**读什么：** 用户输入的书名。

**做什么：** 在搜索引擎中搜索格式如下的直链：
```
{书名} txt 下载
{书名} 全文 txt
{书名} 网盘 txt
```

**门禁：**
- [ ] 搜索结果包含直链（pan.baidu / 网盘 / 直接下载链接）
- ❌ 搜不到任何直链 → **退回**。告知用户"未找到该书的直链，请确认书名是否正确或提供下载来源"。

**产出：** 直链 URL 列表（≥1 条）。

---

### 第二步：下载 TXT

详见 [steps/step-2-download.md](steps/step-2-download.md)。

**读什么：** 第一步产出的直链 URL。

**做什么：** 用 `curl` 或 `wget` 下载 TXT 文件。

```bash
curl -L "直链URL" -o "{书名}.txt"
```

**门禁：**
- [ ] 下载成功，文件大小 > 0
- [ ] 文件编码为 UTF-8（或可转换）
- [ ] 文件内容可读（非乱码/非防盗链页面）
- ❌ 文件大小为 0 或内容明显错误 → **退回**。尝试下一条直链。
- ❌ 所有直链均下载失败 → **退回**。告知用户"所有直链均不可用"。

**产出：** `d:\popwave-skills\{书名}.txt`（TXT 文件）。

---

### 第三步：验证输出

详见 [steps/step-3-verify.md](steps/step-3-verify.md)。

**读什么：** 第二步下载的 TXT 文件。

**做什么：** 验证文件完整性和编码。

```bash
# 检查文件大小
ls -la "d:\popwave-skills\{书名}.txt"

# 检查编码
file "d:\popwave-skills\{书名}.txt"

# 检查前 100 行是否包含小说正文
head -100 "d:\popwave-skills\{书名}.txt"
```

**门禁：**
- [ ] 文件 ≥ 100KB（正常小说长度）
- [ ] 编码为 UTF-8（如非 UTF-8，先转换再保存）
- [ ] 开头非乱码、非防盗链页面
- ❌ 文件 < 100KB → **退回**。疑似不完整，告知用户。
- ❌ 内容为 HTML/防盗链页面 → **退回**。直链已失效。

**产出：** 确认可消费的 TXT 文件。

## ❌ WRONG 示例

| 错误做法 | 问题 | 正确做法 |
|:-----------|:-----|:------------|
| 去小说网站逐章复制粘贴 | 耗时巨大，不符合"只走一条路"原则 | 搜直链，一步到位 |
| 用户说"下txt"却去分析小说内容 | 超出职责范围 | 只执行搜索 + 下载 |
| 下载后不做验证直接交差 | 可能下载了空文件或错误页面 | 验证文件大小 + 编码 + 可读性 |
| 尝试登录/注册网盘账号 | 超出 skill 能力边界 | 告知用户需要手动处理 |

## 落盘检查点

| 检查项 | 规格要求 |
|:-------|:---------|
| 文件大小 | `{书名}.txt` ≥ 100KB |
| 编码 | UTF-8（非 UTF-8 需转码后保存）|
| 内容格式 | 非 HTML、非防盗链页面、可读正文 |

---

## 异常与边界条件

| # | 异常场景 | 处理方式 |
|:-:|:---------|:---------|
| 1 | 搜索引擎返回 0 条结果 | 告知用户"未找到直链"，建议换书名或手动提供链接 |
| 2 | 直链是网盘链接（需登录/提取码） | 告知用户"该直链需手动下载"，提供链接和提取码 |
| 3 | 下载的文件是 HTML（防盗链/404） | 丢弃该文件，尝试下一条直链；全失败则报错 |
| 4 | 下载文件编码非 UTF-8（如 GBK） | 用 `iconv` 或 Python 转码为 UTF-8 后保存 |
| 5 | 文件内容只有目录/分卷信息，无正文 | 告知用户"文件不完整"，尝试其他直链 |
| 6 | 用户提供的书名有错别字/不完整 | 尝试模糊匹配（去掉后缀/修正错字），仍找不到则退回 |
| 7 | 下载中途网络中断 | 断点续传或重试，重试 3 次仍失败则报错 |
| 8 | 文件下载后为 .rar/.zip 压缩包 | 尝试解压，解压失败则告知用户手动解压 |
