# Step 1：确认授权直链来源

> **读什么：** 用户提供的 URL、本地文件路径、书名/输出名。
> **产出什么：** 可直接下载的授权 TXT/ZIP 直链，或可复制的本地文件，或明确的阻塞原因。

## 做什么

1. 判断用户是否提供了可下载 URL 或本地文件。
2. 判断来源是否需要登录、付费、提取码、人工交互或绕过限制。
3. 判断用户是否只给书名；如只有书名，不以盗版站/侵权转载站为目标检索。
4. 确认输出书名，用于 `{书名}.txt`。

## 门禁

| 检查项 | 失败动作 |
|:-------|:---------|
| 只有书名，没有授权直链 | 请用户提供授权 TXT/ZIP 直链或本地文件 |
| 需要登录/提取码/付费/人工点击 | 不绕过；请用户手动下载后提供本地文件 |
| 来源是受保护章节页 | 不逐章抓取；请用户提供合法导出的 TXT/ZIP |
| 输出书名不明确 | 询问书名或用文件名推断 |

## 输出

- `source`: 授权 TXT/ZIP 直链或本地文件路径
- `title`: 输出书名
- `blocked_reason`: 如无法继续，写清原因
