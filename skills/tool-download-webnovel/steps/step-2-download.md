# Step 2: 下载 TXT

> **读什么：** 第一步产出的直链 URL。
> **产出什么：** `d:\popwave-skills\{书名}.txt`（TXT 文件）。

## 操作步骤

用 `curl` 或 `wget` 下载 TXT 文件。

```bash
curl -L "直链URL" -o "{书名}.txt"
```

## 门禁

- [ ] 下载成功，文件大小 > 0
- [ ] 文件编码为 UTF-8（或可转换）
- [ ] 文件内容可读（非乱码/非防盗链页面）
- ❌ 文件大小为 0 或内容明显错误 → **退回**。尝试下一条直链。
- ❌ 所有直链均下载失败 → **退回**。告知用户"所有直链均不可用"。

## 产出

`d:\popwave-skills\{书名}.txt`（TXT 文件）。
