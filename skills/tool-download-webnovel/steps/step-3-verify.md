# Step 3：验证并交付

> **读什么：** Step 2 的脚本输出和保存文件。  
> **产出什么：** 可直接使用的 TXT 文件路径 + 质量信息。可选择性粘贴全文。

---

## 验证项

| 检查项 | 标准 |
|:-------|:-----|
| 文件存在 | `output` 路径存在 |
| 文件大小 | 长篇小说建议 ≥ 100KB；短篇/样章可突破 |
| 编码 | 最终文件为 UTF-8 |
| 内容 | 非 HTTP 错误页（404/403/503） |
| 预览 | 前 120 字可读 |

---

## 交付格式

```
已导入：D:\popwave-skills\downloads\书名.txt
大小：{N} MB
原编码：{encoding}
来源：{搜索路径说明}
预览：{前120字}

可交给 pop-decon 拆解。
```

如需粘贴全文，直接读取文件内容回复即可。

---

## 失败格式

```
未能获取《书名》
搜索结果：{搜索路径说明}
失败原因：{原因}
建议：{换个搜索词 / 换来源 / 用户提供线索}
```

---

## 可选复查命令

```bash
# 检查文件大小
ls -lh /d/popwave-skills/downloads/书名.txt

# 查看前20行
head -n 20 /d/popwave-skills/downloads/书名.txt
```
