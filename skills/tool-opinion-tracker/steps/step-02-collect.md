# Step 2: 平台采集（调用 tool-cnovel-research）

## 输入
第一步确认的平台列表 + 书名。

## 执行
按 tool-cnovel-research 的采集策略逐平台获取数据。

| 采集策略 | 平台 | 工具 |
|:---------|:-----|:-----|
| 全自动采集 | 晋江/贴吧/知乎/B站/微博 | 平台专用 API/爬虫 |
| 混合采集 | 龙空 | AnySearch site: 搜索 + extract |
| 人工兜底 | 豆瓣/阅文/小红书/公众号/抖音/视频号 | AnySearch 扫标题 → 用户粘贴正文 |

## 门禁
- [ ] 每个高可采集平台已采集 >= 10 条相关帖子/内容
- [ ] 已向用户发出人工粘贴请求（涉及低可采集平台时）
- [ ] 某平台采集结果为 0 条 → 检查关键词是否匹配或是否需要人工兜底
- [ ] 用户未完成人工粘贴 → 等待用户粘贴后继续

## 产出
各平台原始采集数据（结构化/原始文本）。
