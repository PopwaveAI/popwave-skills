# Step 3: 分析数据 + 输出报告

## 输入
第二步采集的原始数据。

## 执行
按 `舆情报告模板.md` 输出标准化报告，写入 `.trae/archives/book-opinion-reports/`。

### 报告结构

```
# 《书名》舆情报告

## 一、数据来源总览
## 二、正面评价摘要
## 三、负面评价摘要
## 四、争议焦点
## 五、读者画像特征
## 六、建议
## 附录：原始数据
```

## 门禁
- [ ] 报告包含 >= 3 个平台的独立数据
- [ ] 每类观点至少附 1 条原文引用
- [ ] 标注了数据来源平台和采集时间

## 产出
`{书名}-舆情报告-{日期}.md`
