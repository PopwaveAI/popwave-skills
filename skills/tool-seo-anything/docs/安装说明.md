# SEO SERP 抓取工具 · 安装说明（Windows 版）

## 工具包内容

| 文件 | 说明 |
|------|------|
| `SEOSKILL.md` | SEO 文章写作工作流（给 Cursor AI 用） |
| `google_serp_scraper.py` | Google SERP 核心抓取脚本 |
| `抓取SERP.bat` | 双击运行的快捷脚本（Windows） |
| `安装说明.md` | 本文件 |

---

## 一次性安装（只需做一次）

### 第一步：安装 Python

1. 打开 https://www.python.org/downloads/
2. 点击 **Download Python 3.x.x**（最新版即可）
3. 安装时**务必勾选** `Add Python to PATH`
4. 安装完成后重启电脑

### 第二步：安装 Playwright

按 `Win + R`，输入 `cmd`，回车打开命令提示符，粘贴以下命令：

```
pip install playwright
python -m playwright install chromium
```

等待完成（约 2-5 分钟，会下载 Chromium 浏览器）。

---

## 日常使用

安装完成后，每次使用只需：

1. 双击 `抓取SERP.bat`
2. 输入要查询的关键词（英文），按回车
3. 等待约 30-60 秒
4. 结果自动保存为 JSON 文件在同一目录

---

## 在 Cursor 中使用 SEOSKILL.md

1. 把 `SEOSKILL.md` 放到你的 Cursor 项目的 `.cursor/skills/` 目录下
2. 在 Cursor 中开始 SEO 文章任务时，AI 会自动调用该 skill
3. Step 3 需要你先手动运行 `抓取SERP.bat` 获取 SERP 数据，再粘贴给 AI

---

## 常见问题

**Q: 双击 `.bat` 一闪而过？**

右键点击 `抓取SERP.bat`，选择"以管理员身份运行"。

**Q: 提示 `python 不是内部命令`？**

说明 Python 没有加入 PATH，重新安装 Python 并勾选 `Add Python to PATH`。

**Q: 提示 `playwright` 未找到？**

重新运行：
```
pip install playwright
python -m playwright install chromium
```

**Q: 抓取结果为空或很少？**

Google 有时会触发验证，稍等 1-2 分钟后重试即可。
