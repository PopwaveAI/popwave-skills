# Step 1：任务识别 — SOP

## 第一步：根据输入类型分流

| 输入类型 | 处理方式 |
|:---------|:---------|
| **关键词** | 判断搜索意图（信息型/交易型/导航型/商业调查型）→ 确定目标（排名/教育/转化/旧文更新） |
| **文章 URL** | 抓取正文 → 分析标题/结构/关键词覆盖/内链/信息过时风险 |
| **选题方向** | 拆出主题/受众/语言/地区/商业阶段 → 先产候选关键词和标题方向 |

**❌ 门禁：** URL 页面不可访问 → 要求用户提供原文或截图。不提供 → 退回。

## 第二步：输出任务定义

- 任务定义
- 文章类型（listicle/tutorial/review/comparison/hybrid）
- 缺失信息清单
