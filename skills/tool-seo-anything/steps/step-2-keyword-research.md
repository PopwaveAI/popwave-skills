# Step 2：关键词调研 — SOP

> 用户提供主关键词和次要关键词（从 ahrefs 选），不需要 AI 自行调研。

## 第一步：构建关键词矩阵

至少覆盖：

| 类别 | 说明 |
|:-----|:-----|
| `core_variants` | 主关键词变体 |
| `question_terms` | 疑问型关键词 |
| `commercial_terms` | 商业意图词 |
| `long_tail_scenarios` | 长尾场景词 |
| `semantic_support` | 语义支撑词 |

## 第二步：次要关键词分配

| 使用位置 | 适用场景 |
|:---------|:---------|
| H2/H3 标题 | 适合作为小节主题时 → 优先分配 |
| 正文段落 | 自然融入相关内容 |
| FAQ 问题 | 疑问句式关键词 → 优先做成 FAQ |
| 列表项 | 工具/方法型关键词 → 做成列表/表格 |

**❌ 门禁：** 不是所有次要关键词都必须用到。选最相关的用。不堆词。

## 第三步：输出

- 关键词清单 + 关键词矩阵
- 不建议覆盖的偏移词
- 候选标题角度（2-3个）
