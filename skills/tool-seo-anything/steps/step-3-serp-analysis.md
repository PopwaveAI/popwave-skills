# Step 3：SERP 竞品分析 — SOP

## 第一步：运行 SERP 抓取工具

```bash
# 双击运行 （Windows）
双击 pipeline/抓取SERP.bat → 输入主关键词

# Mac
双击 pipeline/抓取SERP.command
```

**读什么：** Google 美国真实 SERP 前 10 条结果（标题+URL+描述）

## 第二步：竞品结构分析

| 检查项 | 关注点 |
|:-------|:-------|
| 前列结果类型 | 教程/榜单/测评/对比/混合 |
| SERP 特征模块 | FAQ/视频/表格/产品模块 |
| 权威信号 | 作者页/更新时间/来源引用/案例/截图 |
| 适合前置的内容 | 短答/定义段/步骤表/对比表 |

## 第三步：选定框架

| 框架 | 适用场景 |
|:-----|:---------|
| `listicle` | 对比测评+评选 |
| `tutorial` | 操作指南+步骤 |
| `review` | 产品测评+优缺点 |
| `comparison` | 产品对比+选择建议 |
| `hybrid` | 混合型内容 |

## 第四步：输出

- 竞品摘要 + 内容缺口
- 应借鉴结构 + 选定框架
