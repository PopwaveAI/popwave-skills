# Step 00 -- 速查表

> 快速路由：用户说什么 -> 用什么工具。

---

| 用户说什么 | 执行方式 | 使用工具 | 备注 |
|:-----------|:---------|:---------|:-----|
| "搜一下X" / "查资料" / "搜索" | 搜索引擎查询 | WebSearch | 简单信息查找，无需渲染 |
| "打开XX网页" / "看看这个页面" | 网页抓取 | WebFetch | 静态 HTML/Markdown 页面 |
| "抓取小红书/微博/推特的内容" | 真实浏览器 + 登录态 | tool-web-access (CDP Chrome) | 必须登录，走真实浏览器 |
| "帮我登录XX网站" / "表单操作" | 真实浏览器交互 | tool-web-access (CDP Chrome) | 需要模拟人为操作 |
| "这个网站内容加载不出来" / "动态渲染" | 真实浏览器渲染 | tool-web-access (CDP Chrome) | SPA / JS 动态渲染页面 |
| "下载这个PDF/文件" | HTTP 直连下载 | curl / wget | 仅限可公开访问的文件 |
