# Step 01 -- 判断联网类型

> 用户说出联网需求后，先判断属于哪种类型。

---

## 类型判断

| 类型 | 工具 | 条件 |
|:-----|:-----|:-----|
| 简单搜索（查新闻、找资料、单页信息） | WebSearch / WebFetch | 无需登录，页面可公开访问 |
| 真实浏览器（社交媒体、登录态、动态渲染、表单交互） | tool-web-access CDP Chrome | 需要登录 / JS 动态渲染 / 反爬严格 |
| 文件下载（PDF、ZIP等公开文件） | curl / wget | 可公开访问，URL 已知 |

---

## 门禁

| 门禁 |
|:------|
| 判断为"真实浏览器"类型但尝试用 WebFetch/curl 绕过 -- 退回，必须启动真实浏览器 |
| 类型不明确 -- 追问用户确认联网场景 |
