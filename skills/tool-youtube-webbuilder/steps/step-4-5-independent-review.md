## 第二步：质量门禁清单

| 检查项 | 标准 | 结果 |
|:-------|:-----|:-----|
| 零 Emoji | 遍历可见文本节点，无 Emoji Unicode | [ ] |
| 页面充实度 | ≥5 section + About ≥3 段 + Hero ≥4 层 | [ ] |
| 视觉方向一致性 | 与 PRD 一致 | [ ] |
| 工匠精神 | 数字动画 + 入场动画 + hover 反馈 | [ ] |
| 响应式 | 三档测试通过 | [ ] |
| 技术纯净 | 内联 CSS/JS、零依赖、maxresdefault | [ ] |

**❌ 门禁：** 审查未通过 → 最多 2 次重审。审查报告必须公开追加到回复中。
