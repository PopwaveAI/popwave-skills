# 知夏HTML模板说明

## 模板来源

模板只从用户认可的历史终稿提取布局，不直接复制整篇内容。

- `list-9page.html`：来源于眉眼细节 v4、虚拟地名 v1等用户认可的满页清单版。
- `deep-case-9page.html`：来源于6组高张力CP、校园纯爱 v2等一页一组案例版。

## 模板选择

### list-9page.html

用于：50/100条台词、地名、眉眼、伏笔、词汇等短素材。

特征：封面 + 7个分类页 + 检查页；每页10至15条；强调数量和快速浏览。

### deep-case-9page.html

用于：CP设定、人物关系、剧情桥段、写作方法。

特征：封面 + 7个单组内容页 + 检查页；内容页固定“人设/核心 → 写法 → 完整案例”。

## 固定视觉令牌

- 画布：1080×1920。
- 背景：`毒舌台词图文/assets/cover-bg.png`、`content-bg.png`。
- 颜色：正文 `#24292f`，强调 `#c73f60`。
- 封面：两行主标题约156至170px，主标题中心位于页面高度42%至50%。
- 字体：标题楷体/宋体，正文清晰宋体；避免多字体混用。
- 页脚：左侧账号名，右侧页码或内容类型。

## 使用方法

1. 选择母版并复制为业务目录下的新版本。
2. 替换所有 `{{...}}` 槽位。
3. 按需要复制内容页，但总页数默认不超过9页。
4. 删除未使用的占位注释。
5. 检查页面数量、条目数量、重复项、背景路径和打印样式。
6. 新版本不得覆盖用户认可的旧HTML。

## 不要做

- 不从空白HTML重新写整套CSS。
- 不为了塞字临时把正文缩到手机不可读。
- 不把清单版内容强行塞进深写版，也不把完整案例切成15条碎句。
- 不把历史成稿中的人物名、案例和标题保留在母版里。
