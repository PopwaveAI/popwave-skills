# CP 设定资料库

## 来源与使用边界

- 原始资料：`当代全网高人气CP设定大全（含心动名场面完整版）.docx`。
- 文档自注“部分内容可能由 AI 生成”，其中“全网最高热度”“爆款”等说法没有数据来源，只当选题线索，不当事实。
- 原文声称规避未成年暧昧，实际出现大学生与职场年上、高中生年下；另有师徒、伪骨科等高风险关系。对外内容默认改成明确成年、平等、自愿的角色，或不采用。
- 名称、人名和名场面不照搬。先提取关系机制，再重新配置职业、目标、阻力与后果。

## 可复用的 CP 关系机制

### 1. 双强制衡

- 关系：能力相当，做事方式冲突。
- 嗑点：公开争论，关键时刻仍把决定权或后背交给对方。
- 重点：双方各有筹码；合作任务具体；争执能改变方案。
- 避坑：别只写互放狠话，也别让一方突然降智成陪衬。

### 2. 宿敌同归

- 关系：立场相反，却最懂彼此。
- 嗑点：原则与本能冲突，一次破例要付出真实代价。
- 重点：先写不能跨越的原则，再安排救与不救、信与不信的选择。
- 避坑：破例后必须留下后果，不能下一章恢复原状。

### 3. 智性同频

- 关系：思路不同，推理或判断能接上。
- 嗑点：被理解、被跟上、被认真反驳。
- 重点：拆分信息链；一人发现异常，一人补齐逻辑；用细节证明聪明。
- 避坑：少写“他们智商很高”，多写两个人怎样把半句话接完。

### 4. 性格反差

- 关系：嘴硬与软心、冷静与热烈、寡言与直球等外显反差。
- 嗑点：言行错位，只对特定对象破例。
- 重点：关心落在行动；被冒犯的一方保有边界；反差不等于人格突变。
- 避坑：毒舌不能只伤人，小太阳也不能只负责治愈别人。

### 5. 成年旧侣

- 关系：熟悉旧习惯，也面对彼此的新生活。
- 嗑点：本能仍在，边界已经改变。
- 重点：交代当年分开的现实原因；重逢后补谈旧问题；双方都发生过成长。
- 避坑：少用醉酒复合、第三者刺激和单方面苦等。

### 6. 并肩同盟

- 关系：共同目标一致，专业职责发生冲突。
- 嗑点：不要求对方放弃事业，仍愿意为对方调整方案、承担代价。
- 重点：双方各有不可替代的专业能力；意见不合也不互相贬低。
- 避坑：别把其中一人写成战利品、后勤挂件或情绪充电宝。

### 7. 长期羁绊

- 关系：青梅竹马、长期搭档、旧友重逢。
- 嗑点：熟悉带来的默契，与“你已经变了”的陌生感并存。
- 重点：共同记忆要能推动当下选择；旧习惯与新边界同时出现。
- 避坑：认识得久不等于天然相爱，仍要写清当下为什么选择彼此。

## 名场面生成公式

`日常默契或公开冲突 → 突发选择 → 一方为原则或对方付出代价 → 关系发生可见变化`

每个名场面至少回答：

1. 两个人当下各自要什么？
2. 哪个选择会损失证据、职位、名声、机会或安全？
3. 为什么只有这个人能看懂、接住或完成？
4. 场面结束后，权限、称呼、距离或合作方式改变了吗？

## 组 CP 自检

- 双方脱离恋爱线后，是否仍有自己的目标和能力？
- 阻力是否来自具体立场、职责或现实条件？
- 是否明确成年、平等、自愿，并保留拒绝权？
- “偏爱”是否通过一次有成本的选择体现？
- 删掉人设标签后，场景里还能看出两人的差异吗？
- 名场面是否推动关系，而非只负责摆拍和说金句？

## 可做成知夏选题

## 小众邪门 CP：高粘性关系机制

来源补充：`小众邪门高粘性CP设定合集（冷门上头·高死忠·二创爆火专属）.docx`。原文含“爆火、流量密码、粉丝粘性”等无数据支撑的宣传判断，只保留为选题措辞，不当成传播效果事实；部分囚禁、犯罪与失衡关系需重新处理边界和后果。

这类关系的有效部分有三层：

1. **稀缺关系**：同躯双识、人灵殊途、逆向时间、角色与作者、记忆寄居等关系本身就制造阅读问题。
2. **不可替代**：双方掌握对方唯一的语言、记忆、证据或生存条件，但不能因此取消拒绝权。
3. **有价破例**：真正的名场面要损失证据、名声、职位、能力或安全，不能只靠一句“我选你”。

可扩展为七组素材方向：镜像共生、人与非人、天敌站队、生死时差、道德灰区、身份错位、互相驯服。

### 邪门感生成公式

`不可能共存的身份 + 只有对方能完成的事 + 一次有代价的破例 + 关系状态改变`

### 风险校正

- 囚禁与控制：不包装成“极致宠爱”；写清拒绝权、逃离权和越界后果。
- 犯罪与共犯：理解不等于免罪；保留审判、赔偿、揭露或承担责任。
- 同体、镜像与梦境：先明确独立意识和选择，再写亲密，避免只剩概念噱头。
- 人与非人：非人设定不能自动绕过同意；双方都要能理解并表达选择。
- “唯一性”落在具体能力与选择上，不用第三者、雌竞或强迫制造排他感。

### 已生成内容

- `28组小众邪门CP设定｜人设、冲突、名场面都配好了`
- 结构：7 类 × 4 组，第二页直接进入素材；末页用成年、自愿、后果、不可替代和关系变化做检查。

## BG 言情 CP 人设梗

来源补充：`BG 言情小说 CP 人设梗大全.docx`。原文整理了38条标签组合，并使用“热门、爆款、流量最高、读者偏好”等未附数据的判断。入库时只提取关系机制，不把这些传播判断当事实；“正太年下”统一改为明确成年的年龄差关系，“伪骨科”和用偏执控制代替深情的组合不直接采用。

### 七类可复用关系

1. **势均力敌**：宿敌、互相利用、互疑搭档、旧案重逢、公开对手与秘密盟友。
2. **冷热反差**：冰块与热心、阴郁与明亮、寡言与话多、双向反差、真假小太阳。
3. **直球喜剧**：佛系女主与自我攻略男、直球与傲娇、木头与花孔雀、吃货与表演型男主。
4. **事业成长**：事业型女主、成长型男主、目的型接近、阶层冲突、双野心家竞争。
5. **身份阶层**：江湖与世家、将门与纨绔、草根与豪门、君臣、保护关系。
6. **年龄旧情**：默认双方成年；年下、年上、宿敌重逢、前任合作均要写清生活阶段与边界。
7. **奇幻设定**：人鬼、失忆与记忆、穿书觉醒、驱魔与旧神、预知与变数。

### 组合公式

`外显反差 + 各自目标 + 一处现实分歧 + 共同任务 + 一次有代价的选择`

标签只负责让读者迅速认人。故事能不能继续，取决于两个人想要什么、为什么暂时必须合作，以及名场面结束后关系发生了什么变化。

### 已生成内容

- `35组BG言情CP人设梗｜人物关系、冲突、名场面直接套`
- 结构：7类 × 5组，第二页直接进入素材，末页检查目标、差异、冲突、成年自愿和关系变化。

## 校园纯爱 CP 设定

来源补充：`全网高人气CP设定大全｜校园纯爱篇（干净无狗血·青春成长向）.docx`。原稿提供8组校园组合，并使用“流量常青、完播率和点赞率稳居高位、平台推流友好”等未附数据的判断。入库时只保留课桌、操场、社团、晚自习与共同成长等关系机制，不把传播效果写成事实。

### 七类可复用关系

1. **课桌同频**：卷王与学神、毒舌同桌、文理搭档、错题本竞争者、排名对手。
2. **社团赛场**：篮球与画室、主唱与键盘、广播与摄影、辩论与主持、田径与后勤。
3. **青梅邻里**：旧巷同行、邻居同班、竞赛重逢、家属院搭档、早餐摊日常。
4. **新生靠近**：转学生、校园志愿者、复读同桌、被误解的学生、社恐新生；帮助要保留选择权。
5. **竞争成长**：竞赛队、学生会竞选、艺考、体育文化课互助、奖学金竞争。
6. **校园日常**：值日、图书馆、食堂、借伞、运动会后勤；用可观察的小偏心代替空泛告白。
7. **分班毕业**：分科、异地竞赛、志愿分歧、毕业活动、大学重逢；未来方向才是青春拉扯的主要来源。

### 校园纯爱生成公式

`校园共同场景 + 各自成长目标 + 一处现实分歧 + 尊重选择的行动 + 未来计划变化`

### 风险校正

- 校园角色默认按年龄和场景写克制互动，不成人化、性化未成年角色。
- 不把霸凌、暴力与控制包装成“校霸式保护”；需要帮助时保留求助老师、家长和制度渠道。
- “小太阳”不负责治疗另一个人；双方各自面对问题，也能拒绝被拯救。
- 学霸学渣、体育生文艺生只是识别标签，必须继续补共同任务和现实分歧。
- 升学与志愿不为恋爱让位；真正的双向奔赴包含尊重彼此未来。

- `35组校园纯爱CP设定｜关系、矛盾、心动名场面直接套`
- 结构：7类 × 5组，第二页直接进入素材；末页检查目标、冲突、意愿、年龄边界和关系变化。
