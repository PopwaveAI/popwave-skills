## 已收录主题

### 情绪与审美

- 酸涩感：清醒与无力并置；情绪无法被彻底解释后的“剩余物”。
- 破碎感、少女心事、艳鬼系、高智感等“一个词讲透一种气质”的系列逻辑。

### 情节与结构

- 伏笔：名字、玩笑、路人、数字、时间、态度差异、失忆、照片、物件、宠物反应。
- 伏笔层级：浅伏笔负责短程爽感，中伏笔推动人物与关系，深伏笔改变读者对全局的理解。
- 伏笔公式：`日常用途 → 轻微不一致 → 关键选择中改变判断`；资料入口：`foreshadowing.md`。
- 写作流程：随手记录 → 搭建章纲 → 初步填充 → 终稿精修。
- 章纲问题：人物、时间、地点、事件、主线推进、冲突、影响。

### 世界观与地名

- 地名先定层级，再定气质：现代行政、学校、住宅商业、古风州郡、仙侠地貌不能混用后缀。
- 地点生成公式：名称 + 行政层级 + 气候地貌 + 核心产业 + 交通方式 + 一段旧事。
- 虚构名称仍可能撞现实城市、学校、企业和商标，重要作品发布前需要检索查重。
- 资料入口：`worldbuilding-naming.md`。

### 对话写作

- 基础“说、问”可以保留；提示语要安静，关键转折才加重量。
- 对话优化六路：提示语、有效动作、潜台词、快慢节奏、人物声线、环境介入。
- 核心检查：这段说完以后，信息、决定或关系有没有变化？
- 资料入口：`dialogue.md`。

### 眉眼与微表情

- 优先观察视线落点、停留时间、是否回避，以及描写后人物做了什么。
- 眉形、眼型和眉骨只写轮廓，不直接判定善恶或性格。
- 单次突出一至两个变化，不机械堆满眉、眼、睫毛、光影和生理反应。
- 资料入口：`facial-expression.md`。

### 人物与关系

- 女主人设：任性大小姐、绿茶型、疯批型、灵动初恋型。
- 反派动机：嫉妒、背叛、阴谋、欺诈、暴戾、冷血、喜怒无常、病态占有。
- OC 建档：名字、性别、出生、家庭、身份、职业、外貌、可识别特征。
- 霸总题材：克制偏爱型、创伤修复型、反差喜剧型、隐形大佬型；清醒事业女主、单亲母亲、穿书反套路、重生复仇女主。
- CP 关系：双强制衡、宿敌同归、智性同频、性格反差、成年旧侣、并肩同盟、长期羁绊。明确成年、平等、自愿；名场面要有具体选择、代价和关系变化。
- 小众邪门 CP：镜像共生、人与非人、天敌站队、生死时差、道德灰区、身份错位、互相驯服；用稀缺关系、不可替代和有价破例制造粘性，不美化囚禁、犯罪与失衡。
- BG 言情 CP：势均力敌、冷热反差、直球喜剧、事业成长、身份阶层、成年年龄差与旧情、奇幻关系；按“反差 + 目标 + 分歧 + 共同任务 + 有价选择”组合。
- 校园纯爱 CP：课桌同频、社团赛场、青梅邻里、新生靠近、竞争成长、校园日常、分班毕业；用共同任务、现实分歧和未来选择推进，互动符合年龄与校园边界。

### CP 设定

- 核心公式：`对等关系 + 具体阻力 + 只有对方能触发的破例`。
- 场面公式：`日常默契或公开冲突 → 突发选择 → 付出代价 → 关系发生可见变化`。
- 风险校正：不把“全网高热”当事实；规避未成年暧昧、强制关系、依附式恋爱与单方降智。
- 资料入口：`cp.md`。

### 霸总与豪门

- 核心升级：把“霸道”从控制女主，改为处理问题时的决断；把“总裁”从财富标签，改为真实职责与代价。
- 关系底线：尊重、边界、有效沟通、双方都有选择权；偏爱通过破例与承担成本体现。
- 现实校正：公司权力、家族信托、信用卡额度、资产流动性、宴会与住宅都不能写成统一绝对规则。
- 资料入口：`bazong.md`。

### 内容形态

- 理论拆解帖：定义 → 机制 → 核心悖论 → 与相近概念比较 → 写法。
- 数字清单帖：大数字承诺 → 逐条素材 → 使用提示 → 评论征集。
- 步骤教程帖：经验共鸣 → Step 1/2/3/4 → 示例 → 避坑。
- 求助共创帖：具体资料缺口 → 请求真实经历 → 评论沉淀素材。
- 爆文拆解帖：作品名、互动数据、字数、作者、开头钩子、结构复盘。
- SEO 定稿：真实交付 → 省力感承诺 → 1 个核心词 + 1 至 2 个长尾词 → 标题/配文/评论/标签分配 → 去 AI 味。
