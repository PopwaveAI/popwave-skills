# 世界观与虚拟地名资料库

## 来源与边界

- 原始资料：`小说虚拟地名大全.docx`。
- 原文按现代城市、学校、商圈、住宅、休闲场所和生活配套整理了大量名称，并给出“特色字 + 地点后缀”的组合思路。
- 原文声称可生成“不撞现实”的虚拟地名，但其中含现实城市、历史称呼、常见场所或可能存在的商业名称。不能承诺不重名，正式发布前应检索城市、学校、公司、楼盘和商标。
- 入库时重新组合名称，并补充古风行政地名与仙侠秘境；名称只作灵感素材，不代表已完成法律或商标查重。

## 地名生成方法

### 1. 先定地点层级

- 现代行政：市、区、县、镇、街道、路、巷。
- 校园：中学、学院、大学、教学楼、图书馆、体育馆。
- 住宅商业：苑、里、庭、公馆、广场、天地、中心。
- 古风行政：州、郡、府、城、县、镇、关、驿、坊、渡。
- 仙侠地貌：山、谷、泽、川、海、原、墟、岛、秘境、天阶。

### 2. 再定题材气质

- 温柔水系：沚、汀、澜、潮、津、浦。适合治愈、校园、滨海日常。
- 山林清冷：岑、岭、雾、松、砚、崖。适合悬疑、文艺、旧案。
- 都市明亮：照、霁、星、望、临。适合职场、商战、现代言情。
- 古风雅正：雁、朔、晏、蘅、舒、柳。适合州郡、关隘、坊市。
- 仙侠诡奇：烬、昼、坠、镜、冥、魂。适合禁地、秘境和异域。

### 3. 补齐地点生命

`名称 + 行政层级 + 气候地貌 + 核心产业 + 交通方式 + 一段旧事`

地名只负责第一眼。读者真正记住一个地方，通常因为人物在那里生活、失去、选择或重逢。

## 使用检查

1. 城市、区县、街道和学校的层级是否一致？
2. 同一地区是否沿用相近的词根和后缀？
3. 地名是否与当地气候、产业和历史相符？
4. 是否过度使用“云、月、星、清、锦”等高频字，导致全世界同一种气质？
5. 重要名称是否检索过现实重名、商标和机构？
