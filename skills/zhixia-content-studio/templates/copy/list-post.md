# 数量清单帖文案模板

## 封面

```text
前导：{{具体卡点}}
主标题：{{数量}}个{{核心素材词}}
副标题：{{直接动作/收益}}
分类：{{分类1｜分类2｜分类3...}}
```

## 分页

```text
页标题：{{第N类}}
页副标题：{{这一类解决什么问题}}

01 {{素材内容}}
适用：{{场景/情绪/作用}}
```

每页10至15条时优先短句；数量少时增加具体用法，不用重复换词填满。

## 末页

```text
标题：写完以后，查这5项
检查1：{{是否选对场景}}
检查2：{{是否重复}}
检查3：{{是否符合人物}}
检查4：{{是否推动情节}}
检查5：{{是否完成改写}}
```
