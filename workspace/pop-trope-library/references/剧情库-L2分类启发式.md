# 剧情库 L2 分类启发式

> 用途：给 L2 单元卡打粗功能标签与写作任务标签。

---

## 1. 两层标签

L2 分类分两层：

| 标签层 | 作用 | 示例 |
|:-------|:-----|:-----|
| 粗功能标签 | 方便库存统计和大类筛选 | 崛起征服、反派对抗、设定揭示 |
| 写作任务标签 | 方便 plot/create 检索 | 穿越求生、系统觉醒、第一次收益闭环、绝境反杀 |

粗功能标签不能替代写作任务标签。`崛起征服` 只能说明大方向，不能告诉 agent 当前要读哪张开局卡。

---

## 2. 粗功能标签

| 标签 | 核心叙事功能 | 典型特征 |
|:-----|:-------------|:---------|
| 崛起征服 | 主角实力跃迁、身份升级、势力扩张 | 练级、升级、称号、新能力 |
| 反派对抗 | 与具体反派/敌对势力冲突 | BOSS 战、暗杀、战争、猎杀 |
| 设定揭示 | 世界规则、隐藏势力、力量体系披露 | 信息揭露、真相、制度解释 |
| 势力建设 | 领地、团队、资源、经营 | 基建、招募、贸易、防御 |
| 血脉觉醒 | 天赋、血脉、神性、人格质变 | 觉醒、进化、特殊身份 |
| 情感联盟 | 关系建立与深化 | 守护、赠礼、分离、重逢 |
| 阶级揭露 | 社会制度与阶层压迫展示 | 剥削、资格、底层、权力结构 |

---

## 3. 写作任务标签

| 任务域 | 常用标签 |
|:-------|:---------|
| 开局 | 穿越求生 / 系统觉醒 / 低资源生存 / 第一次主动行动 / 新手村逃离 |
| 成长 | 第一次收益闭环 / 技能验证 / 职业转职 / 装备获得 / 金手指认知突破 |
| 冲突 | 绝境反杀 / 追杀逃亡 / 暗杀清算 / 智斗谈判 / BOSS 战 |
| 设定 | 世界观震撼 / 规则揭露 / 制度黑暗 / 力量体系首展 |
| 关系 | 守护之约 / 战宠命名 / 同乡善意 / 追随者建立 / 情感羁绊 |
| 势力 | 据点建立 / 团队招募 / 资源经营 / 声望扩散 |
| 收束 | 债务回收 / 阶段胜利 / 新地图开启 / 更大敌人露面 |

一张 L2 可以有多个任务标签，并可出现在多个任务索引中。

---

## 4. 判定流程

```text
Q1: 这个单元最主要完成什么叙事功能？
  → 打一个粗功能标签

Q2: 写作者什么时候会需要它？
  → 打 2-5 个写作任务标签

Q3: 它最值得学习的爽点是什么？
  → 打 meta_joy

Q4: 它是否值得进入任务索引？
  → 是：更新/新建任务索引
  → 否：只更新剧情库索引
```

## 5. frontmatter 建议

```yaml
---
layer: 机制

  - 信息差
  - 绝境反杀
source_book: 深渊主宰
function_tag: 崛起征服
task_tags:
  - 穿越求生
  - 系统觉醒
  - 第一次收益闭环
  - 贫民区绝境
---
```

## 6. 常见边界

| 案例 | 粗功能标签 | 写作任务标签 |
|:-----|:-----------|:-------------|
| 开场建置单元 | 崛起征服 | 穿越求生 / 系统觉醒 / 低资源生存 |
| 击杀强敌后升级 | 反派对抗 或 崛起征服 | 绝境反杀 / 第一次收益闭环 / BOSS 战 |
| 世界规则首次解释 | 设定揭示 | 世界观震撼 / 力量体系首展 |
| 兄妹/战宠关系确立 | 情感联盟 | 守护之约 / 战宠命名 / 情感羁绊 |
| 建立据点或队伍 | 势力建设 | 据点建立 / 团队招募 / 声望扩散 |
