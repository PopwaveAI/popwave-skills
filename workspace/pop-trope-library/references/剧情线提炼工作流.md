# 剧情线提炼工作流

> 用途：从拆书产物中提炼 L2 单元卡、L3 剧情线，并接入写作任务索引。

---

## 1. 输入

```text
卷纲 / 章节摘要 / L2 单元卡 / L3 剧情线 / 运行日志 / 状态快照
```

---

## 2. L2 提炼

L2 是完整行动弧，必须能回答：

1. 主角承受什么压力？
2. 主角做出什么主动判断？
3. 行动链如何展开？
4. 单元内得到什么收益或付出什么损失？
5. 结尾留下什么新压力？

输出入库：

```text
剧情库/{书名}/L2/{书名}-{编号}-{名称}.md
```

---

## 3. L3 提炼

L3 是跨单元长期线，必须能回答：

1. 这条线从哪里启动？
2. 每阶段发生什么不可逆变化？
3. 哪些债务被递送、延迟、回收？
4. 与其他线在哪里咬合？
5. 终点快照相对起点变化了什么？

输出入库：

```text
剧情库/{书名}/L3/{书名}-{编号}-{名称}.md
```

---

## 4. 任务索引接入

每张高价值 L2/L3 提炼后，检查它服务哪些写作任务：

```text
任务索引/{任务域}/{任务名}.md
```

索引条目必须写：

- 读哪些段。
- 学什么。
- 不要照搬什么。
- 适配哪个赛道/元爽点。

---

## 5. 输出确认

```markdown
## 剧情线提炼完成
- L2 入库：
- L3 入库：
- 剧情库索引更新：
- 任务索引更新：
- 消费摘要状态：
- 库缺或待补：
```
