# PRD · 让魔门再次伟大（鹤守月满池）

> **版本**：PRD v3（创作宪法） | 2026-06-22
> **定位**：这份 PRD 是「让魔门再次伟大」的最高决策坐标。当任何 skill 在管线中面临创作选择时，PRD 是最终仲裁者。
> **来源**：通过用户问答 + 联网搜索 + 同类优秀 PRD 参考推导而成。本书尚未动笔——所有内容均为设计决策，非拆书复盘。

## 〇、本书剧情简述

大顺王朝——一个阶级固化的东方武道世界。赤土底层人命如杂草：功法被垄断、赋税抽走最后一口气、黔首死了连名字都不配留下。而在凡人之上，仙门以「收割」为业——死人税、人牲、力士炼制，赤土不过是仙门的养殖场。

穿越者王平从现代跌入这个把人压成渣的世界。没有灵根，没有背景，只有一颗现代人的脑子和对阶级不公的本能反胃。他从白莲教卧底做起，靠信息差博弈一步步爬上捕快之位，又在绝境中以天魔解体换来器灵重生——每一次「死亡」都不是终点，而是一次阶级跃迁的跳板。

他的金手指叫太平经——不是温和的辅助系统，而是一套以「死后复活 + 身份兑换 + 气运借贷」为核心的残酷机制。金手指不白送力量——它是一份信用契约，提前支取什么，就必须在某个节点还回去。

在赤土与仙门的夹缝中，王平用信息差碾压每一层对手、用穿越者的荒诞笑声嘲讽每一道枷锁、用一次次「旧身份的死亡」掀翻「黔首不能算人」的铁律。

这就是「让魔门再次伟大」——一个穿越者在最残酷的双世界里，用智力和背叛杀出一条阶级革命的血路。

## 一、篇幅规划

> 基于类型（黑暗博弈仙侠）的复杂度 + 双世界结构 + 5-6 卷目标推导。

| 参数 | 数值 | 说明 |
|:-----|:-----|:-----|
| **目标总字数** | ~200 万字 | 5-6 卷 × ~130-160 章/卷 × 2000-3000 字/章（网文章节 2-3k 字基准 + 双世界需要足够空间展开两套力量体系） |
| **预估卷数** | 5-6 卷 | 每卷对应一次力量体系代际跃迁或世界维度切换 |
| **单卷章数（均值）** | ~130-160 章 | 首卷密度较高（身份建立期），后续卷可能拉长 |
| **幕/卷** | 3-4 幕 | 每幕 = 1 次核心冲突的起承转合完成，约 30-50 章/幕 |
| **升级阶梯密度** | 每卷 2-3 次质变 | 设计节奏：凡人身→捕快→天魔解体→器灵→…逐级解锁新维度 |
| **剧情线数量** | 主线 2 条 + 支线 5-8 条 | 主线：身份觉醒线 + 世界观揭示线；支线随卷增删 |
| **角色池规模** | 核心 5-8 人 + 重要 15-20 人 | 每卷引入 2-3 个新重要角色，旧角色退场 2-3 个 |

### 篇幅对结构的约束

- **升级节奏**：平均每 10-15 章一次可感知的力量升级，每 30-50 章一次世界维度质变
- **爽点密度**：每章至少 2 个爽点脉冲（信息差类 + 阶级类/幽默类交替），每幕至少一个高密度高潮区（L5 爽点 ≥8 个/章）
- **角色引入节奏**：每卷引入 2-3 个新重要角色，旧角色退场 2-3 个（首卷已设计：守冲/陈浩彦/徐秉正退场，尤道姑/劫修群体入场）
- **世界观揭示节奏**：每卷翻新 20-30% 的世界认知（首卷设计：武道→异人→修仙→五贼盗天，四层翻新）
- **200 万字的特殊约束**：支线不能失控——每卷结束后必须清理至少 2 条支线（首卷已设计：白莲教线/陈浩彦线/守冲线关闭），防止伏笔堆积到不可管理

## 二、爽点锚点

> 从用户问答中锁定的三个主元爽点，附 Phase 2 自问链产出的特征参数。以下为设计决策，非反向提炼。

### 主元爽点 1：信息差碾压

- **定义**：读者获得智力优越的快感——「我知道主角知道而反派不知道的事情」。核心心理机制是**认知不对称的愉悦**。
- **本书占比**：约 45%，全书最高频爽点
- **变体家族**：
  - 先知先觉（灵识透视/权谋复盘——主角比别人多看一步）
  - 暗赚反转（截胡夺宝/伏击复仇——主角在信息差中血赚）
  - 钓鱼博弈（双向渗透/三层嵌套——你以为你在钓我，其实我在钓你）
  - 身份伪装（双面间谍/器灵化身——没人知道主角的全部身份）
- **兑现方式**：太平经灵识系统提供信息优势基础；多层博弈结构（A 以为 B 不知道 C 但 D 在围观）是叙事引擎；每幕至少一个信息差闭环（设伏→揭示→反转→收割）

**特征参数**：

| 维度 | 参数 |
|:-----|:-----|
| L1 感官/媒介 | 灵识透视 + 权谋复盘（太平经提供的感知优势） |
| L2 时效性 | 永久能力，但需要主动激活（有 CD 和消耗） |
| L3 代价 | 消耗气运 + 越级使用会暴露自己（被更高阶存在察觉） |
| L4 叙事层 | 战斗层面（看穿破绽）+ 权谋层面（预判博弈）+ 世界观层面（知道隐藏真相） |

### 主元爽点 2：阶级革命 / 身份跃迁

- **定义**：读者在阶级固化的残酷世界里，看到主角从最底层一次次掀翻棋盘的心理释放。核心心理机制是**压迫—反抗的正义满足**。
- **本书占比**：约 30%
- **变体家族**：
  - 从弃子到棋手（卧底→截胡→反杀——被操纵者成为操纵者）
  - 以死换升级（死亡不是终点，是跳转新身份/新力量的节点）
  - 蛰伏反噬（器灵化→蛰伏→等待反噬上位者）
  - 阶级嘲讽（「黔首也能算人？」的正面对撞 + 金句反击）
- **兑现方式**：「黔首非人论」「公私论」「再苦一苦」三句金句是全书阶级批判的三根钢钉；主角每一次阶级跨越必须伴随旧身份的毁灭（卧底死→捕快生，王平死→器灵生）；赤土世界的阶级固化是「设计好的」——从功法缺失到税收制度环环相扣

**特征参数**：

| 维度 | 参数 |
|:-----|:-----|
| L1 压迫主体 | 制度性结构压迫为主 + 个体上位者的恶意为辅 |
| L2 反抗方式 | 从根本上颠覆游戏规则——不是推翻某个皇帝，是颠覆「修仙者>凡人」的阶级合法性 |
| L3 压迫体现 | 功法垄断（知识封锁）+ 苛税（经济剥削）+「黔首非人论」（身份歧视）+ 仙门收割（暴力威慑） |
| L4 革命路径 | 个人反抗 → 组织底层（劫修群体） → 动摇制度 → 重新定义规则 |

### 主元爽点 3：穿越者解构 / 黑色幽默

- **定义**：在高压叙事的缝隙中，突然插入现代思维对古代世界的荒诞解构。核心心理机制是**认知错位的喜剧释放**——防止读者被持续压抑逼走。
- **本书占比**：约 15-20%
- **变体家族**：
  - 现代梗吐槽（「大人时代变了」「我尼玛」「不做人了」）
  - 荒诞对比（死后还在算收益 / 决死现场笑扔毒丹逗弄对手）
  - 名场面反转（飞刀炸假山→立刻磕头拜师 / 对手以为胜券在握→主角已经在算计死后结算）
- **兑现方式**：紧张场景后必有 1-2 句穿越者槽点作为情绪泄压阀；幽默从不瓦解主基调（决死=黑色幽默缓冲≠搞笑）；「笑对死亡」是全书标志性的情感姿态

**特征参数**：

| 维度 | 参数 |
|:-----|:-----|
| L1 感官入口 | 情绪层（认知错位的喜剧释放）+ 认知层（现代思维解构古代规则） |
| L2 触发条件 | 在高压场景的临界点触发——不早于紧张的建立，不晚于情绪崩溃前 |
| L3 节奏频率 | 脉冲型——来一下就走，不占叙事主篇幅（单章幽默密度不超过 30%） |
| L4 升级路径 | 从单纯吐槽→黑色幽默→「笑对死亡」的高阶姿态——幽默从调味料进化为角色的生存哲学 |
| L5 排他性 | 与「纯热血欢呼」不兼容，与「绝望哭号」不兼容——幽默是中间态，是「绝望中的不屈」的独特表达 |

## 三、核心卖点

> 这本书和其他书的本质不同，按重要性排列。每条回答：「赛道里常见的做法 → 本书的做法 → 为什么本书更对」。

### 卖点 1：信息差驱动的博弈结构（不是「力量碾压型」）

**别人家**：修仙主角升级→变强→碾压对手。对手越强=主角需更强。

**本书**：主角赢在「信息差更高一层」。多层信息不对称博弈（A 以为 B 不知道 C 但 D 在围观）是全书的核心冲突引擎。主角不是最强的——他是知道最多的人。反派各有各的聪明：守冲会钓鱼、陈浩彦会布局、徐秉正会算账、尤道姑会渗透。主角赢是因为他看到的棋盘比对手多了一层。

**为什么更对**：信息差碾压满足的是「智力优越感」而非「暴力崇拜」。在这个赛道里，读者看腻了越级打脸——本书的「你打不过我，不是因为我不够强，是因为你不知道我知道什么」是更高级的爽点。

### 卖点 2：力量体系的「代价前置」设计（不是白送金手指）

**别人家**：金手指=开挂。系统面板一弹，经验值一加，力量到手。

**本书**：太平经是「先享受后付费」的金融契约。力量从来不是白送的——死后复活有 7 天 CD、气运贷款需要连本带利偿还、每一次阶级跃迁必须伴随旧身份的毁灭。金手指的名字就叫「太平经」——来自黄巾起义的太平道，是造反者的圣经，不是上古大能的馈赠。

**为什么更对**：黑暗流的本质是「一切有代价」——如果金手指白送力量，黑暗流的基调会从内部崩塌。太平经的设计让「代价」从哲学概念变成了可结算的游戏规则。

### 卖点 3：阶级批判作为世界观地基（不是「主题点缀」）

**别人家**：修仙世界有穷人也有富人，但穷只是主角的起点——当主角变强，穷的问题就「自然解决」了。

**本书**：阶级压迫是设计好的——从功法缺失（底层无修炼资格）到税收制度（「黔首再苦一苦」）到意识形态（「黔首非人论」），环环相扣让阶级固化成为「理性的」。主角变强不是因为穷人逆袭——是因为他用信息差+死亡赌注在规则的缝隙里操作。「黔首非人论」「公私论」「再苦一苦」三句金句是全书阶级批判的三根钢钉，不是修辞点缀。

**为什么更对**：市面上大多数「底层崛起」型修仙文，主角变强后就忘了底层——本书不允许这种遗忘。阶级批判不是调味料，是世界运行的基本规则。

### 卖点 4：双世界不对称收割结构

**别人家**：修仙世界只有一个位面，主角在世界内部升级。

**本书**：赤土（武道侧）和仙门（修仙侧）是两个物理隔绝但经济耦合的世界——仙门收割赤土（死人税、人牲、力士炼制）。主角在两套规则的夹缝中操作，穿越不是逃离，是从一种压迫跳进另一种压迫。两个世界质感截然相反但本质相同——都是把底层压榨到极致的世界。

**为什么更对**：双世界结构不是「两个地图」——是两种阶级压迫体系的并置对照。赤土是粗粝的物质压迫，仙门是诡谲的信息压迫。这给了「阶级革命」爽点一个更大的叙事空间。

### 卖点 5：穿越者的现代性作为叙事工具（不是背景设定）

**别人家**：穿越者=带着现代知识在古代装逼。穿越只是开头的设定。

**本书**：穿越者的现代思维是全书的核心叙事工具——现代吐槽是情绪泄压阀、资本式算计（贷款/结算/收益最大化）是金手指的规则骨架、对古代阶级的「局外人视角」是阶级批判的合法性来源。主角王平的每一句话都在提醒你：「这个人是穿越的」——这是他的超能力，也是他的诅咒。

**为什么更对**：大多数穿越文把「穿越」当起点设定用完就丢。本书把穿越者的视角变成了全书情感光谱的定调器——底层视角 + 现代思维 = 一种刻在骨头里的荒诞感。

### 卖点 6：14 域跨域深度融合（不是「缝合怪」）

**本书融合的知识域**：武侠传统/仙侠修真/丹道/阴符经/炼丹/佛教名相/DND 3.5e/系统流/三十六计/金融借贷/策略游戏/阶级批判/古典小说/穿越者梗。

**不是表层叠加**：系统流的面板/CD/结算机制与修仙世界的灵根/渡劫/炼炁体系完全咬合；三十六计的博弈结构与官场/宗门政治完全咬合；阶级批判不是「主题」而是世界运转的基础规则；金融隐喻是金手指的规则骨架而非修辞。

**为什么更对**：深度融合 ≠ 多放几种元素。每一域只在特定层发挥作用（框架层=力量体系/质感层=命名体系/机制层=博弈结构/调味层=幽默笔触），互不越界，互不污染。

## 四、美学基因 + 全书色彩

> **两个基因决定一切：东方黑暗仙侠 + 底层视角。** 以下所有维度——质感、力量感、情感节奏、材料选择——都从这两个基因推演而出，不是并列罗列。

### 美学基因

| 基因 | 含义 | 排他性后果 |
|:-----|:-----|:----------|
| **东方黑暗仙侠** | 根在仙侠而非日轻/西幻；基调是黑暗博弈而非热血/温馨/搞笑 | 拒绝了日系轻小说美学、西方奇幻命名、热血少年漫情绪 |
| **底层视角** | 主角不是天才/世家/天命之子——从最底层一步步往上撕。世界运行的基本规则是阶级压迫 | 拒绝了龙傲天/天选之人叙事、拒绝了「修炼=变美」的审美、拒绝了「制度有道理」的妥协 |

---

### 从基因到世界质感

**东方黑暗仙侠 → 物理质感：**

- 赤土：**粗粝冰冷的物质世界**——寒冬、泥土、铁锈、血、炭火。没有风花雪月，只有刀兵和铁器。大顺王朝是一个在冬天冻死的野兽。
- 仙门：**奇观诡谲的超凡世界**——五彩极光、画门幻景、无边黑暗中一座高台。修仙不是仙境——是另一个同样残酷但美学语言不同的阶级体系。

**底层视角 → 空间感受：**

- 赤土：**逼仄压抑**——县城与乡野的物理鸿沟就是阶级鸿沟的具象化。城内歌舞升平，城外尸横遍野。
- 仙门：**空洞辽阔但同样不对等**——道峰孤悬于异空间，内门是一扇画在墙上的门。无灵根者连门都进不去——空间的辽阔是权力者的辽阔。

**一句话**：两个世界质感截然相反，但本质相同——都是把底层压榨到极致的世界。穿越不是逃离，是从一种压迫跳进另一种压迫。

---

### 从基因到力量美学

**黑暗流 → 力量的外观：**

- 赤土力量 = **功能性的暴力**——没有花哨招式，一刀一刀见血。重甲重刀、军阵如山、兵器决胜负。力量就是杀人的效率。
- 修仙力量 = **诡谲神秘**——不可直视（五贼盗天）、不可理解（玉简的沉寂）、不可信任（说一半藏一半）。力量不是透明的——它总是藏着代价。

**黑暗流 → 变强的代价：**

**每一次突破都伴随旧身份的毁灭。** 死亡不是惩罚，是升级的材料——但每次死亡意味着旧关系网的彻底断裂。在这本书里，变强不是变美——是变得更不像人。

**这条规则直接衍生出金手指的设计方向：**「先享受后付费」的借贷逻辑。力量从来不是白送的——你提前支取了什么，就必须在某个节点还回去。由此自然导向金融隐喻作为金手指的规则骨架——不是拍脑袋的设计选择，是力量美学的必然结果。

**底层视角 → 力量的来源不是天赋：**

主角的力量不能来自「天生异禀」「血脉觉醒」「被高人看中」。只能来自**智力的不对称优势 + 规则的漏洞 + 死亡的赌注**。这直接决定了爽点锚点中的信息差碾压 + 以死换升级。

**一句话**：变强不是变美，是变得更不像人。力量的来源不是天赋，是智力+代价+漏洞。

---

### 从基因到情感光谱

**黑暗流 → 禁用情感：**

- ❌ 纯热血——不写无代价的胜利欢呼
- ❌ 纯温馨——不写治愈系日常
- ❌ 纯绝望——任何绝境后必有黑色幽默缓冲（否则读者被逼走）

**黑暗流 → 情感节奏：脉冲式 W 型**

```
压抑到临界点 → 智力博弈的冷静满足 → 暗赚反转的血腥快感
    → 再度拉紧（更大的危机逼近）
    → 悲壮（决死/身份毁灭）
    → 荒诞黑色幽默（缓冲——「死前还在算收益」）
    → 蛰伏希望（新生/新身份/新维度开启）
```

**底层视角 → 情感的特殊质地：**

底层的人不会在胜利时欢呼「我们赢了！」——他会冷静计算这次胜利值多少钱、欠了多少债、得罪了多少人。底层的情感是**精打细算的快感**，不是热血的快感。这直接决定了：战斗胜利的情绪终点是惨胜/血赚/蛰伏，不是欢呼。

**底层 + 穿越者 → 黑色幽默的来源：**

不是「刻意加搞笑」。底层二十年 + 现代思维 = 一种刻在骨头里的荒诞感。穿越者王平看赤土的一切都带着解构的视角——因为对他来说，这个世界的所有严肃规则（君臣、阶级、修仙）都是「别人的规则」。黑色幽默不是调味料——是底层视角的自然产物。

**一句话**：读者永远在高潮后喘不过气来时被一个穿越者梗逗笑——然后发现幽默背后更大的危机正在逼近。这个节奏本身就来自「黑暗流 + 底层视角」的双重约束。

### 从基因到材料选择

> 以下不是「我们找来了哪些素材」——是「从美学基因出发，必然需要哪些知识领域作为原材料」。

#### 从「东方」出发 → 三套东方知识领域

| 知识域 | 为何被选中 | 基因链 |
|:-------|:----------|:-------|
| **佛教名相**（欲界六天/天魔/菩萨/莲华/业力轮回） | 提供了**层级化的宇宙结构**（欲界六天的层层递进）+ **对抗性概念**（天魔 vs 正道 = 体制外 vs 体制内）+ **欲望体系**（六欲 = 色声香味触法 = 六种可分类的力量来源） | 东方仙侠需要宇宙层级 → 欲界天提供了现成的层级框架；黑暗流需要「正统的对立面」→ 天魔就是体制外的那个 |
| **阴符经**（五贼/盗天/天人/三盗） | 提供了**对抗性的哲学基础**——盗贼杀机四个字就是黑暗流的哲学。道家经典中唯一一部天然含「盗」字的经典 | 东方仙侠需要哲学地基 → 阴符经而非道德经（道德经讲无为，阴符经讲盗窃）；黑暗流需要「逆天」的合法性 → 盗天为己而非顺天应道 |
| **民间直白/江湖切口**（铁器/炭火/刀兵/净街虎/大龙枪） | 提供了**底层世界的物质质感和声音**——不是每个人都活在佛经里。赤土底层的人用铁器命名铁器，用炭火命名炭火 | 底层视角需要粗粝的物质感 → 赤土的所有命名只用民间直白语汇，这是阶级现实的质感化 |

#### 从「黑暗流 + 底层反抗」出发 → 两套反抗传统

| 知识域 | 为何被选中 | 基因链 |
|:-------|:----------|:-------|
| **金融隐喻**（贷款/结算/抵押/利息/套利/信用崩盘） | 提供了**精确的代价-收益模型**——黑暗流的「力量有代价」不能停留在口号，需要用一套可量化的语言描述代价如何运作。同时提供了「剥削如何运作」的语言——贷款/抵押/利息 = 统治阶级榨取底层的方法 | 黑暗流需要代价精确化 → 金融隐喻把「代价」从哲学概念变成可结算的规则；爽点（信息差碾压）需要「不对称收益」框架；阶级批判需要剥削语言 |
| **民间反抗传统**（黄巾起义·太平道、白莲教、劫修/无产阶级革命） | 提供了**底层反抗的叙事原型**——起义、秘密教派、生产资料公有化（开源功法）。这不是「佛教名相」的子类——是独立的叙事传统：造反者的圣经、被剥夺者的反噬 | 黑暗流 + 底层视角 → 必须有一种「体制外的力量源泉」→ 金手指不叫「系统」或「老爷爷」，叫「太平经」（黄巾起义的太平道的回声）；阶级革命爽点 → 劫修群体 = 修仙界的无产阶级，五贼盗天功法的开源 = 生产资料公有化 |

**为什么金手指叫太平经？** 这个选择本身就是两个基因的共同产物——「底层反抗」决定了它不能是修仙界正统传承，必须是造反者的圣经；「黑暗流」决定了它不能是温和的辅助系统，必须是结算死人、借贷未来、把死亡当材料的残酷机制。黄巾起义的「苍天已死，黄天当立」→ 太平经的「旧身份已死，新维度当立」。

### 从基因到命名语汇

- **赤土世界语汇**：全部使用民间直白/江湖切口——铁器、炭火、刀兵、净街虎、大龙枪。粗粝 = 阶级现实的质感化
- **仙门世界语汇**：佛教名相 + 阴符经哲学 + 丹道术语——欲界六天、天魔、五贼盗天、炼炁渡劫
- **金手指语汇**：金融隐喻——贷款、结算、抵押、利息、套利、一诺千金、信用崩盘
- **拒绝的语汇**：❌ 日系翻译腔；❌ 西方奇幻命名（奥术/魔法/龙骑士）；❌ 赤土功法中出现佛教/道家术语（越界）；❌ 金融术语出现在功法命名中（「复利大法」）
- **一条铁律**：**听角色怎么说话，就知道他在食物链的哪一层。唯独王平——他每句话都在提醒你：这个人是穿越的。**

### 视觉锚点场景

寒冬。赤土县城外，一个穿捕快服的年轻人蹲在尸体旁边——不是在哀悼，是在算账。死者的太平经余额、未偿还的气运贷款、可继承的身份点数。他的手指冻得发白，但眼睛在发光——因为这笔「死后结算」刚好够他换下一个身份。

远处，仙门的极光在夜空中无声流淌。五彩的光芒照在尸体的脸上，像是一种嘲讽：你连这光的门都进不去。

年轻人站起来，拍了拍膝盖上的泥，嘴角扯出一个不该属于这个世界的笑——「行吧，这波血赚。」

他转身走向县城。城墙上，文武双圣的画像在风中猎猎作响。没人注意到画像的眼睛似乎在往下看——又或者只是错觉。

## 五、世界宪法（4 条不可违反规则）

### 第一条：信息差是冲突的唯一驱动力

**规则**：任何核心冲突的胜负不能由「纯力量等级」决定——必须由「认知不对称」（信息差）决定。如果主角对某对手的力量等级完全碾压，叙事处理必须以信息差为入口（「他知道你不知道的是……」，而非「他比你强所以碾了你」）。

**来源**：美学基因「黑暗博弈」+ 核心爽点「信息差碾压」 | **违反后果**：如果某场关键冲突变成「主角靠升级后的力量碾压对手」，这本书会滑向龙傲天文——信息差的智力优越感崩塌。

### 第二条：一切力量都有可量化的代价

**规则**：太平经的所有功能（复活、身份兑换、气运贷款、属性提升）必须有精确的代价锚点——CD 周期、消耗气运数额、偿还机制。不接受「境界到了自然就懂了」式的模糊突破。不接受任何「无代价白送」的力量增长。

**来源**：力量美学「变强的代价」+ 金融隐喻规则骨架 | **违反后果**：如果太平经出现无代价功能，金手指的「先享受后付费」逻辑崩塌，读者会意识到「所谓的代价不过是装饰」→ 全书最核心的「黑暗流」承诺失信。

### 第三条：阶级压迫不可逆（结构性，非个体性）

**规则**：压迫来自制度设计，而非某个「坏皇帝」。功法垄断、赋税结构、意识形态（黔首非人论）、仙门收割机制——四条腿同时撑起压迫体系。主角不能「当上皇帝后一切就好起来了」——制度性压迫不能靠个体皇权解除。

**来源**：基因「底层视角」+ 核心爽点「阶级革命」 | **违反后果**：如果压迫被简化为「某个反派在作恶」、主角通过「取代反派」解决问题，全书的阶级批判地基会退化为「好皇帝换坏皇帝」的童话——爽点 2「阶级革命」的深度崩塌。

### 第四条：穿越者视角不可磨灭

**规则**：王平的穿越者身份必须持续在叙事中发挥作用——不是背景设定，是每一章的叙事工具。具体表现为三种形态至少出现一种：（1）穿越者吐槽/黑色幽默（2）现代思维解构古代规则（3）资本式算计（成本-收益分析）。如果连续 5 章没有出现任何穿越者视角信号，说明叙事在滑向「普通修仙主角」。

**来源**：穿越者解构爽点 + 「现代性」核心卖点 | **违反后果**：如果穿越者视角淡化，王平会和其他修仙主角没有区别——爽点 3（黑色幽默）和本书最独特的「局外人视角」同时崩塌。

## 六、核心承诺 + 防崩阈值

### 承诺 1：智力碾压的暗爽——「我比所有人都多看一步」

- **体验描述**：读者在每场冲突中都能感受到「主角知道某件事而对手不知道」的认知优越。这不是力量碾压——是信息碾压。主角赢不是因为更强，是因为他看到了对手没看到的棋盘。
- **对应的元爽点**：信息差碾压（主元爽点 1）
- **兑现方式**：先知先觉（灵识透视+权谋复盘）、暗赚反转（截胡夺宝+伏击复仇）、钓鱼博弈（多层嵌套的钓鱼结构）
- **防崩信号**：如果连续 3 章没有出现「主角比对手多知道一层」的结构，说明叙事滑向力量碾压——立即回头注入信息差。反派不能降智——主角赢是因为信息差更高一层，不是对手蠢。

### 承诺 2：阶级压迫下的反抗快感——「黔首也能算人？」

- **体验描述**：在一个把底层人命当杂草的世界里，看穿越者用智力、金手指和「死亡—重生」机制一步步掀翻压迫系统。每一次旧身份的死亡都是一次阶级跃迁——而每一次跃迁都在打脸那些说过「黔首非人」的人。
- **对应的元爽点**：阶级革命/身份跃迁（主元爽点 2）
- **兑现方式**：从弃子到棋手、以死换升级、阶级嘲讽（三句金句正面对撞）
- **防崩信号**：如果连续 5 章没有触及阶级压迫议题，说明叙事正在脱离世界观地基。王平不能洗白压迫系统——他可以利用系统规则但不等于认同系统。如果王平开始说「这个制度也有它的道理」，核心承诺崩。

### 承诺 3：穿越者的荒诞笑声——死前还在算收益

- **体验描述**：在全书最黑暗的场景中，读者突然被一个穿越者梗逗笑——然后发现这个「笑」不是脱节，是主角对死亡的最后一次嘲讽。这不是搞笑文——是「绝望中的不屈」用黑色幽默的形式呈现。
- **对应的元爽点**：穿越者解构/黑色幽默（主元爽点 3）
- **兑现方式**：现代梗吐槽、荒诞对比（死前算收益）、情绪泄压阀（每章紧张场景后 1-2 句槽点）
- **防崩信号**：幽默不能瓦解主基调——决死场景的黑色幽默是「笑对死亡」，不是「这只是一场游戏」。如果穿越者梗让读者觉得「这场战斗不严重」，用错了。反过来，如果连续 10 章没有黑色幽默，读者会被压抑逼走。

## 七、不可妥协 DNA + 边界

### DNA（打死不能改的 4 条）

1. **「信息差碾压」作为第一爽点手段** — 全书最高频套路（先知先觉/暗赚反转/钓鱼博弈）。即使引入新元素，不能改为「力量碾压型」。如果有一天主角靠纯力量碾压对手且读者不觉得降智，这本书已经不再是这本书。

2. **穿越者的现代性** — 王平的穿越者身份不是背景设定，是核心叙事工具：吐槽、现代思维解构、资本式算计（贷款/结算/收益最大化）。不能淡化——淡化后王平和其他修仙主角没有区别。

3. **体系化力量设计** — 外功六要素/兵器五等/灵根三档/经卷功能/太平经的贷款机制——**所有力量必须有可理解的规则边界**。不接受的加工：「模糊的悟了」「境界到了自然就懂了」。

4. **阶级批判底色** — 「黔首非人论」「公私论」「再苦一苦」三句金句不是修辞点缀，是全书世界的地基。不能为了爽而洗白压迫系统——王平可以利用系统规则（当捕快、当功曹），但他不能在叙事层面认可「这个制度有道理」。

### 篇幅对 DNA 的影响
- **如果是 20 万字**：DNA 1（信息差碾压）和 DNA 3（体系化设计）仍然不可妥协；DNA 4（阶级批判）可以收窄为暗线；DNA 2（穿越者现代性）可以降频但不消失
- **200 万字（当前）**：四条 DNA 同等不可妥协。200 万字的篇幅要求每卷重新兑现四条 DNA——不能前 50 万字坚持、后 150 万字松懈
- **500 万字**：DNA 1 和 DNA 3 会被篇幅稀释——需要每 3 卷做一次 DNA 审计，检查信息差爽点占比是否低于 35%

### 硬边界

- ❌ **不写龙傲天式碾压** — 主角靠信息差和智力赢，不靠纯力量碾压
- ❌ **不写降智反派** — 反派各有各的聪明和算计。主角赢在信息差更高一层，不是对手蠢
- ❌ **不写后宫/种马** — 感情线不是核心驱动力。可以有感情，不能成为主线发动机
- ❌ **不写机械降神** — 金手指有明确的规则和代价。如果太平经出现新的「无代价能力」，必须提前至少 20 章埋下规则线索
- ❌ **不写无底线轻松的搞笑文** — 黑色幽默是情绪缓冲，不是主基调。基调始终是**黑暗博弈**。任何单章的幽默密度不能超过 30%
- ❌ **不写设定灌水的过渡章** — 每一章必须推进至少一条剧情线或揭示至少一层世界观

## 八、加工矩阵

> **用途**：下游 skill 进行素材注入时，Step 5（PRD 契合度评估）和 Step 7（适配方式判定）直接查此矩阵。

### 冲突模式兼容性

| 冲突模式 | 兼容度 | 加工规则 |
|:---------|:------|:---------|
| **信息差碾压型**（A 知道 B 不知道 C 但 B 以为 D） | ✅ 母语 | 直接使用。本书最高频冲突结构 |
| **智力博弈型**（多方下棋、跨章布局、层层嵌套） | ✅ 可用 | 需至少 3 方参与。禁止纯双人对弈——第三方潜伏是宪法一 |
| **力量碾压型**（主角靠硬实力碾压对手） | ❌ 禁止 | 除非碾压以「信息差」为前提，否则不写 |
| **纯热血型**（无代价的燃） | ❌ 禁止 | 违反「力量有代价」的 DNA。燃可以，必须有代价 |
| **悬疑/推理型**（读者和主角一起解谜） | ⚠️ 有条件 | 可以——但谜底揭示后必须有「暗赚反转」或「阶级讽刺」兑现，不能单纯解谜 |
| **情感羁绊型**（感情驱动冲突） | ⚠️ 有条件 | 可以用——但感情线不是核心驱动力，不能占主线篇幅 20% 以上 |

### 角色原型导入规则

| 原型 | 兼容度 | 变异要求 |
|:-----|:------|:---------|
| **智力型谋士/间谍/奸商** | ✅ 直接导入 | 转化为本世界的博弈参与者。保留智力属性，但行为模式必须符合赤土/仙门的规则体系 |
| **反套路角色**（穿越者/吐槽者/解构者） | ✅ 直接导入 | 可作为情绪缓冲角色。但不能替代主角的穿越者视角——主角是唯一的「现代思维持有者」 |
| **灰色反派**（有合理动机的对手） | ✅ 直接导入 | 反派必须有可理解的动机和各自的聪明 |
| **龙傲天**（无代价碾压） | ❌ 禁止 | 违反 DNA |
| **纯圣母/纯恶人** | ❌ 禁止 | 所有角色必须有灰色层次。即使是压迫者也要有他的「逻辑」 |
| **搞笑担当**（纯喜剧角色） | ⚠️ 需变异 | 不能有「纯为了搞笑存在」的角色。幽默来自主角的穿越者视角，不来自滑稽配角 |
| **修仙者/异人** | ⚠️ 需变异 | 外部修仙素材导入时，必须（1）量化规则边界（2）命名进入本世界命名体系（3）不能模糊化 |

### 力量元素兼容性

| 元素类型 | 兼容度 | 约束 |
|:---------|:------|:-----|
| **有明确规则边界的**（CD/代价/升级路径/量化标尺） | ✅ 兼容 | 必须保留原 CD/代价机制 |
| **模糊的「悟了」** | ❌ 不兼容 | 必须量化为可理解的规则或淘汰 |
| **系统流面板机制** | ✅ 兼容 | 但不写「叮！任务完成」式提示。金手指的结算/锚定/贷款机制以金融隐喻呈现，不以游戏 UI 呈现 |
| **跨域力量体系** | ⚠️ 有条件 | 赤土武道体系与仙门修仙体系各自独立——不同体系的力量不可直接对比强弱。跨体系对抗必须通过「信息差」或「规则漏洞」而非数值对比 |

### 情绪终点约束

| 场景类型 | 允许的情绪终点 | 禁止的情绪终点 |
|:---------|:--------------|:--------------|
| **战斗胜利** | 惨胜（代价可见）/ 暗赚反转（血赚 + 讽刺）/ 蛰伏待机（未全胜，埋下一手）/ 黑色幽默收尾 | 无代价碾压 / 热血欢呼 /「我们赢了！」 |
| **战斗失败 / 死亡** | 理性结算（计算死后收益）/ 黑色幽默转场 / 以死换升级（太平经结算） | 纯绝望哭号 / 主角崩人设的崩溃 |
| **日常 / 过渡章** | 体系展开（世界观揭示）/ 角色关系深化 / 金手指机制探索 | 设定灌水 / 无驱动力的闲聊 |
| **幕终 / 卷终** | 旧身份的彻底终结 + 新维度的开启 | 温柔收束 / 大团圆 |

## 九、跨域素材指引

> 方向性指引——告诉下游 world/plot skill 去哪些领域找素材和灵感。具体采纳/转译规则由 L1 阶段决定。

- **DND 3.5e 体系化设计**：读被遗忘的国度的位面结构、九大职业完整能力树、法术位/复杂度体系。提取「力量体系完全可量化」的设计方法论，而非具体数值。
- **三十六计 + 冷战情报战**：读间谍战史、冷战情报网运作、博弈论经典案例。提取多层信息不对称的叙事结构——不是把「围魏救赵」直接写成情节。
- **金融体系运作**：读银行/信贷/保险/期货的底层逻辑。提取「先享受后付费」的规则骨架——贷款=预支未来力量、抵押=质押灵根/寿命、信用崩盘=劫修群体的次贷危机。
- **阴符经 + 黄巾起义**：读阴符经原文（重点：五贼/三盗/天人关系）+ 黄巾起义的组织结构和太平道教义。提取「盗天为己」的哲学合法性和底层起义的叙事原型。
- **佛教名相体系**：读欲界六天/天魔/菩萨/业力轮回的概念体系。提取「层级化宇宙结构」和「欲望=可修炼的力量来源」的设定逻辑——但方向从「离欲解脱」反转为「炼欲成魔」。

## 十、用户画像

- **核心读者**：25-40 岁男性网文深度读者——厌倦了「龙傲天碾压」「金手指白送」「越级打脸循环」的修仙文套路。想要智力博弈的快感而非暴力发泄的爽感。对历史/军事/金融/哲学有基础兴趣，不排斥信息密度高的叙事。
- **边缘读者**：喜欢三十六计/博弈论的策略游戏玩家；喜欢金融/经济隐喻的读者；喜欢阶级批判/反乌托邦主题的严肃读者。
- **不服务**：追求无脑爽的读者（「主角最强/一路碾压/美女倒贴」）；需要简单直白力量体系的读者（「这个境界比那个境界强」）；期待传统修仙温情叙事的读者（「师徒情深/宗门温暖」）。
