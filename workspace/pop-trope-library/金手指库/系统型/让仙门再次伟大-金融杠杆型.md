> **来源**：拆书实证（D:\workspace\拆书项目\让仙门再次伟大）
> **类型**：系统型（太平经平台）+ 悖论型（透支未来的代价即引擎）+ 道具型（物理经书）
> **赛道**：修仙/穿越/系统流
> **关联元爽点**：绝境反杀/身份跃迁/智斗博弈

## 一、核心创意

「太平经」是一个金融杠杆平台。主角通过死亡复活、身份兑换、借贷未来战力等"资本运作"在两大平行世界（赤土世界武道 / 仙门世界修仙）中撬动资源。力量本质是"借来的"——借命、借身份、借战力、借时间，主角是自己的"基金经理"。

金手指不是"力量来源"，而是"金融杠杆"——把主角变成资本运作人，让每一次力量获取都伴随着债务、期限和利率，用"资本运作"逻辑重构修仙文的成长驱动。

## 二、能力清单

| 模块 | 功能 | 关键设定 |
|:-----|:-----|:---------|
| **死后复活** | 死亡后太平经结算，冷却7天复活 | 死亡不是免费重置，而是有窗口期的稀缺资源 |
| **穿越** | 在赤土世界与仙门世界之间穿越 | 两大平行世界资源互换 |
| **身份兑换** | 消耗势力气运换取身份（白身→皂吏→青士→朱服→紫卿→金爵） | 每次换身份都在"消耗"一方势力的命数 |
| **太平经收集** | 收集物品，品质需逐级递增，不能重复收集低级 | 强制主角不断升级目标，无法躺平 |
| **一诺千金** | 借未来的"战力债"，千日内必须还贷 | 时间压力驱动主角必须在期限内"变现" |
| **诅咒** | 太平经提供的诅咒能力 | 平台功能之一 |
| **灵识** | 跨维度精神感知，可破神意 | ch012觉醒 |
| **锚定身份** | 肉身毁灭后锚定为六欲天魔旗器灵，不受灵根限制 | ch037，器灵化=不可逆"转职" |
| **气禁** | 憋气假死+气血内循环，骗过灵识 | ch007 |

## 三、代价设计

| 代价 | 机制 | 戏剧性来源 |
|:-----|:-----|:-----------|
| **复活冷却7天** | 一场大战中只能死一次 | 死亡是稀缺资源，不是免费重置 |
| **身份兑换消耗气运** | 气运不是无限供应，需消耗一方势力气运 | 每次换身份消耗一方势力的命数，制造道德负担 |
| **一诺千金千日还贷** | 借的是未来战力，有时间限制 | 时间压力驱动主角必须在期限内"变现" |
| **收集品质递增** | 不能重复收集低级物品 | 强制主角不断升级目标，无法躺平 |
| **肉身毁灭不可逆** | 器灵化是单行道 | ch032"英勇就义"后无法回到肉身状态 |

**代价核心**：太平经的每一个功能都是"借"的。主角的力量本质上是"债务"，必须在期限内还清。借命有冷却，借身份消耗气运，借战力有千日倒计时，借时间是穿越世界的代价。

## 四、行动驱动机制

| 驱动维度 | 具体机制 |
|:---------|:---------|
| **死亡驱动** | 死亡触发复活结算=任务重启机制，7天冷却限制滥用 |
| **身份切换** | 消耗气运换取新身份=技能树切换，不同身份解锁不同资源渠道 |
| **收集升级** | 物品品质递增=经验值/升级曲线，强制目标递进 |
| **借贷还贷** | 一诺千金=时间锁，千日倒计时制造紧迫感 |
| **锚定身份** | ch037新机制，器灵化=不可逆的"转职"，开启新成长线 |

**驱动逻辑**：没有传统战宠/血量/经验值——驱动逻辑是"借贷-还贷-投资"的金融循环，主角是自己的"基金经理"。

---

## 七、核心矛盾/两难

**"透支未来"困境**：太平经的每一次赋能都是借来的。复活借的是"下次死亡的窗口"，身份借的是"势力气运"，战力借的是"千日后的自己"。主角越强，债务越深。

更深层矛盾是**身份错位**：主角是"域外天魔"，既不属于赤土武道也不属于仙门修仙，他的力量根基是"外来者"——这意味着他在两个世界都是异类，永远无法真正"归属"。
