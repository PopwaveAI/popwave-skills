# 融合适配清单

> Phase 0 · creative→world 传递协议
> 来源：creative 的 PRD.md v1.1 + 故事引擎.md + 跨域素材蒸馏.md + 拆书融合摘要.md + 素材储备池.md

## 融合决策摘要

| 融合来源 | 影响类型 | 核心决策 |
|:---------|:---------|:---------|
| 《一世之尊》多世界结构 | 结构借鉴 | 上界(仙侠)下界(武侠)的位面关系 + 高阶存在布局低界 |
| 《死人经》黑暗生存感 | 氛围/方法论 | 斩断关系思维 + 多重身份面具 + 信任即弱点 |
| 殖民地世界观 | **核心设定** | 所有功法有毒、奇遇是饵料、飞升=上门送菜 |

## 逐维适配

### L1-01 世界蓝图

| 受融合影响点 | 处理方式 |
|:-------------|:---------|
| 位面结构 — 上界(仙侠)与下界(武侠)的双层结构 | ✅ **保留并强化**：仙侠世界是"收割者"的牧场，武侠世界是殖民地。两层之间有天然屏障，但主角的穿越能力是漏洞 |
| 武侠世界"深水区" — 上古遗留的力量种子 | ✅ **新增维度**：武侠世界并非天生低能，上古曾有强者反抗、留下零散"纯净传承"，作为主角破局的重要资源 |
| 地理层次 — 一世之尊的"真实界+宇宙碎片"模式 | ⚠️ **简化适配**：暂不做复杂多世界，聚焦双界；未来扩展可加入更多碎片世界 |

### L1-02 力量体系

| 受融合影响点 | 处理方式 |
|:-------------|:---------|
| 有毒功法体系 — 殖民者为武侠世界设计的畸变功法 | ✅ **核心设定**：武侠世界的功法都是经过上界筛选/修改过的"安全版本"，修炼越深体内'烙印'越重。这是本书力量体系的基本矛盾 |
| 仙侠世界的正统体系 | ✅ **对位构建**：仙侠有完整的修炼体系（炼气→筑基→金丹→元婴→化神→…），与武侠的"有毒功法"形成对照 |
| 《死人经》"杀人练功"模式 | ✅ **部分采用**：主角可能找到一种"利用死亡净化有毒功法"的法门——每次死亡后复活，功法中的毒被"重置"一次 |
| 一世之尊的"做减求空" | ✅ **抽象映射**：高阶大能通过培养"替身/产物"来承载自己的因果、应劫。主角的穿越机制可能是某个大能的"做减求空"产物 |

### L1-03 历史与驱动力

| 受融合影响点 | 处理方式 |
|:-------------|:---------|
| 殖民历史 — 上界何时/为何殖民了武侠世界 | ✅ **需构建**：上古某次大战后，败者(或某势力)被放逐/封闭在武侠世界；胜者留在仙侠世界，定期"收割" |
| 反抗史 — 武侠世界是否有人反抗过 | ✅ **新增**：历史上数次反抗都被镇压，但留下了"纯净传承"的种子。主角不是第一个觉醒者，但可能是第一个拥有双界穿越能力的 |
| 一世之尊的"彼岸布局" | ✅ **抽象映射**：高阶存在之间的博弈延伸了数千年，主角的出生/穿越能力是某个更大棋局的一部分 |

### L1-04 物种与天赋

| 受融合影响点 | 处理方式 |
|:-------------|:---------|
| 两个世界的人类差异 | ✅ **需构建**：武侠世界的人长期修炼有毒功法，体质有隐性"烙印"。仙侠世界的人天生经脉更完整 |
| 特殊个体 | ✅ **新增**：极少数武侠世界的人天生带有"抗毒体质"——对有毒功法的侵蚀有天然抵抗。主角可能属于此类 |

### L1-05 势力格局

| 受融合影响点 | 处理方式 |
|:-------------|:---------|
| 仙侠世界 — 收割者势力 | ✅ **需构建**：控制殖民通道的宗门/家族，定期从武侠世界"遴选"（实际是收割） |
| 武侠世界 — 表面江湖+深水暗流 | ✅ **双面构建**：表面上是正常江湖（帮派/世家/门派），深层有与上界勾结的"通路者"组织和隐秘的反抗组织 |
| 《死人经》的"金鹏堡"模式 | ✅ **参考**：武侠世界的大势力本身就是殖民地管理的一环——表面争霸，实际共同维持"养殖场"秩序 |

### L1-06 资源与物品

| 受融合影响点 | 处理方式 |
|:-------------|:---------|
| 武侠世界的"资源"本质 | ✅ **核心设定**：武侠世界对仙侠的真正价值——人的生命力/灵魂/肉体经过"修炼提纯"后成为上界丹药/法器的原材料 |
| 灵药/资源的不对称 | ✅ **需构建**：相同药材在武侠和仙侠世界的价值可能天差地别（时间成本/环境因素），这是主角套利的切入点 |
| 有毒功法的"产物" | ✅ **需构建**：修炼有毒功法到高深境界的人，他们的内丹/功力可以被直接提炼为丹药 |

---

## 融合关键决策

1. **双界位面**：暂锁定为双层结构（武侠=下界殖民地，仙侠=上界收割者），未来可扩展
2. **穿越机制定位**：主角的被动穿越+死亡复活能力可能是某个大能的"做减求空"产物——主角是他人的"棋子"，但可凭借信息差摆脱困局
3. **核心破局路径**：利用死亡重置有毒功法的"进度"，在武侠世界修炼"纯净"的体系（从仙侠带回的正确功法）
4. **套利逻辑**：武侠的低阶灵药在仙侠是珍稀品（因为仙侠灵气浓度高导致低阶灵药反而难寻），仙侠的残篇/基础知识在武侠是颠覆性的真知
