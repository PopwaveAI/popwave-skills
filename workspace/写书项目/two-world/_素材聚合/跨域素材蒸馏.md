# 跨域素材蒸馏

> 采集背景：根据 PRD 加工坐标系定向采集
> PRD筛选条件：黑暗博弈/殖民地世界观/死亡驱动/双界信息差/底层耗材视角

---

## 一、殖民地/养殖场世界观素材

### 1.1 核心母题：收割型世界架构

**来源**：网文"养殖场"流派 + 反乌托邦经典

| 要素 | 经典形态 | 本项目的转化方向 |
|:-----|:---------|:----------------|
| **收割延迟** | 主角成长到一定程度才触发收割（如"神魔养殖场"） | 武侠世界的"天劫/飞升"就是收割契约，修为达到某个临界点自动触发 |
| **功法毒化** | 修炼特定功法的人会被标记/操控 | 武侠世界流传的所有**地级以上的功法**都经过上界修改，修炼到高层会逐渐"变质"，修炼者成为特定丹药的"药引" |
| **周期收割** | 每多少年一个收割周期 | 上界每**三百年**一次大收割，通过"天降秘境"来筛选成熟体 |
| **伪秩序** | 殖民地有完整的表面秩序（门派/朝廷/正邪对立） | 武侠世界的正邪对立本身也可能是**被设计的**——让修炼者互相厮杀加速"成熟" |

### 1.2 变体：自上而下的寄生结构

> 上界大能在武侠世界留下了**种子**（功法、血脉、传承），每一个接受者都是"寄生容器"，修炼到一定程度，容器成熟，大能取走"果实"并销毁容器。

**这个设定最强的地方**：它让武侠世界的"江湖"突然有了全新的解释——
- **为什么魔教总是死灰复燃？** 因为魔教的镇教功法是某个上界魔修布下的种子线
- **为什么正道大派总出叛徒？** 因为某些"天才弟子"实际上是被收割目标，悟性越高成熟越快
- **为什么有"三百年一次的大乱"？** 不是历史规律，是收割周期的结果

## 二、死亡驱动系统素材

### 2.1 有代价复活的变体库

| 变体 | 描述 | 与本项目PRD的契合度 |
|:-----|:-----|:-------------------|
| **记忆碎片化** | 每次死亡丢失一段记忆。复活后可能忘了某个重要情报——需要再死一次才能记起来 | ⭐⭐⭐ 高契合——增加信息博弈复杂度 |
| **力量衰退** | 死亡后修为倒退，需要重新修炼。但**领悟不会丢** | ⭐⭐⭐⭐ 非常契合——体现"技术积累"vs"等级重置"的张力 |
| **死亡标记** | 每次复活身上留下一个不可消除的"痕迹"，被某个存在越看越清楚 | ⭐⭐⭐⭐ 非常契合——给被动穿越加一个"被发现"的倒计时 |
| **时间代价** | 复活需要时间（几天到几个月），期间错过关键事件 | ⭐⭐⭐ 高契合——增加战略负担 |
| **灵魂磨损** | 死太多次灵魂变得"稀薄"，最终可能无法再凝聚（真死） | ⭐⭐⭐⭐⭐ 务必采用——给死亡加一个终极上限 |
| **借体重生** | 死亡后随机在武侠/仙侠世界的另一个身体上复活 | ⭐⭐ 低契合——会让世界线太混乱，不利于双界聚焦 |

### 2.2 死亡感知（被动收益）

每次死亡，主角的灵魂在"脱离→复活"的短暂间隙中，可以**窥见世界的一丝真相**：
- 第一次死：看到世界像是一个**被包裹的茧**（暗示世界有边界/有"外面"）
- 第五次死：看到有人**在看**这个茧（第一次感知到上界观察者）
- 第十次死：看到了"功法"中的一条**线**连接着看的人与被看的人（感知到收割连接）
- 越往后：能看到越多世界缝隙中的信息碎片

这个设计把"死"从负面惩罚升级为**信息获取手段**——主角比所有人都"看得多"，但这个红利是用命换来的。

## 三、双界穿越信息差博弈素材

### 3.1 信息不对称套利模型

| 套利方向 | 武侠→仙侠 | 仙侠→武侠 |
|:---------|:-----------|:-----------|
| **功法** | 武侠功法虽"有毒"，但**基本功思路**无可替代（没有上界修改的"纯净版"可能在仙侠世界有特殊价值） | 仙侠世界的**基础理论**（灵气理论/五行生克/经络本质）在武侠世界是失传的上古绝学 |
| **资源** | 武侠世界特有的**凡性材料**（凡铁/百草/普通兽血）在仙侠世界被认为无用，但某些特殊炼制需要"未沾染灵气"的原料 | 仙侠世界不要的**废丹/残料**在武侠世界是逆天神药 |
| **信息** | 武侠世界对"上界"的敬畏和传说是了解收割系统的线索 | 仙侠世界对"下界"的疏忽和傲慢是主角安全的最大屏障 |
| **人脉/身份** | 主角在武侠世界的身份可以光明正大地"消失"而不引起怀疑 | 主角在仙侠世界也以底层身份活动，避免引起上界注意 |

### 3.2 双界的"安全悖论"

> **在武侠世界越强 → 越接近被收割 → 越要尽快逃到仙侠世界**
> **在仙侠世界越活跃 → 越容易被上界发现你来自殖民地 → 越要逃回武侠世界**

这个悖论是主角**长期的核心矛盾**——在两个世界他都无法真正"登顶"，因为登顶就是暴露。

## 四、黑暗博弈叙事素材

### 4.1 "耗材"视角的叙事优势

| 视角特征 | 叙事效果 |
|:---------|:---------|
| 没有退路的信息 | 主角没有家族背景可依赖，每一步决策必须是**信息完全利用** |
| 输得起但不输 | 能复活但不能无限复活（灵魂磨损+力量衰退），"死"仍然是一种策略资源 |
| 没有道德负担 | 在拆穿伪装体系的过程中，不需要遵守"江湖规矩"——规矩本身就是陷阱的一部分 |
| 读者共情锚点 | 所有人都能被"系统压迫"产生共鸣——主角的反抗是**每个人的反抗** |

### 4.2 "死出来的情报"节奏设计

```
第1次死 → 发现穿越的真相（原来死了会穿越）
第2次死 → 在仙侠世界学到一套基础理论
第3次死 → 发现自己在仙侠世界的身份也被人利用了
第4次死 → 发现武侠世界的功法有问题
第5次死 → 在死亡间隙看到"观察者"
第N次死 → 拼凑出完整的世界真相
```

每次死亡不是"又挂了"的挫败——而是**解锁一个关键信息片段**。读者会对"这次能发现什么"产生期待。

## 五、力量体系设计线索

### 5.1 双界力量体系的"错位"

| 武侠体系 | 仙侠体系 | 错位红利 |
|:---------|:---------|:---------|
| 内功/外功/轻功 | 炼气/筑基/金丹 | 武侠的"内劲"对上界灵气有**抗性**（没被污染过） |
| 以"杀死"为胜 | 以"封印/炼化"为胜 | 仙侠大能不理解武侠死士的战术思路 |
| 没有丹田锁定 | 丹田是最关键的要害 | 主角每次死都会重置丹田——别人练一万年的东西，主角可以"死出来" |
| 武者的直觉/预判 | 修仙者的灵识/推演 | 武道直觉在灵识环境里失灵——但灵识在某些武侠位面也可能被压制 |

### 5.2 力量体系的核心矛盾

> **主角的成长曲线是一条"死出来的曲线"——仙侠体系认为修为靠时间堆，主角靠死亡堆。**
>
> 传统修仙者：200 年筑基 → 500 年金丹 → 2000 年元婴
> 主角：死 1 次 = 十年修炼 + 一段关键情报
>
> 但灵魂磨损限制了主角的"死"次数——他必须精打细算每一条命。
