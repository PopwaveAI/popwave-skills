## 四、剧情线

### 主线1 — 世界危机（被动端·收割系统对主角的压迫）

| 字段 | 值 |
|:-----|:----|
| 一句话定义 | 赤土三千年殖民地真相的第一次碎片浮现——有人不想让主角知道什么，也有人想让主角看到什么 |
| 本卷推进 | 从"无异常"到"暗杀→追捕→暴露在系统的视野中"。危机形态升级：暗杀（神秘/单点）→天柱军追捕（组织/区域级）→断命阁介入（组织/跨界级） |
| 预期释放节奏 | 幕1: 暗杀+石刻=第一次释放（"有问题"）/ 幕2: 天柱军追捕+幽渊边缘的警告=第二次释放（"问题很大"）/ 幕3: 碧渊宗看到上界对下界的真实态度=第三次释放（"我不是唯一被利用的"）/ 幕4: 断命阁重新锁定+天柱山异常开启=第四次释放（"问题要升级了"） |
| 活跃幕段 | 幕1 ~ 幕4 全程 |

主线1-子线:
- ①【暗杀之谜】谁下了暗杀令？为什么要杀一个凡人？— 载体角色: 断命阁 "红绳" — 回收预期: 幕4末揭晓：暗杀来自天道院的"观察列表"——主角的因果盲体在因果网上表现为一个"断点"，引起系统自动报警
- ②【天柱军的操作】一个边哨站如何维持"收割秩序"？— 载体角色: 严校尉 — 回收预期: 幕2展示：通过登记修炼进度/监控"飞升候选人"/处理"异常者"来维护流程。主角被归入"异常者"处置程序
- ③【幽渊的秘密】幽渊埋着什么？— 载体角色: 残天人 — 回收预期: 幕2末释放片段信息："上一次有人发现真相，是在幽渊最下面"

### 主线2 — 主角成长（内部端·主角的内在变化）

| 字段 | 值 |
|:-----|:----|
| 一句话定义 | 一个平民少年在三个月内经历认知崩塌、身份重构、第一次做出"不天真"的选择 |
| 本卷推进 | 从"天真的江湖梦乡" → "世界的真相是骗局" → "我知道了但我该怎么办" → "我不逃，我留下来掀桌子" |
| 预期释放节奏 | 幕1: 第一次死亡 → 穿越 → 认知崩塌开始 / 幕2: 追捕中不断验证"功法有毒"的猜想 → 愤怒和恐惧交替 / 幕3: 在尘渊看到上界对下界的真实态度 → 愤怒变成冷静的计算 / 幕4: 决定"不去任何一边" → 第一次真正意义上的成长节点 |
| 活跃幕段 | 幕1 ~ 幕4 全程 |
| 关键成长节点 | 幕1: 第一次死亡后穿越——"世界不是我以为的那样" / 幕2-3过渡: 发现老周是知情者——"连好人都在演" / 幕3: 碧渊宗的杂役生活——"在尘渊我也是底层" / 幕4: 决定留在双界之间——第一次主动选择，不是被动反应 |

### 主线3 — 主角行动（主动端·主角应对主线1的行动轨迹）

| 字段 | 值 |
|:-----|:----|
| 一句话定义 | 从被追杀被动反应，到主动收集信息、建立双界跳板、最后决定主动留在棋盘上 |
| 本卷推进 | 逃亡 → 调查 → 潜伏 → 决定 |
| 预期释放节奏 | 幕1: 从暗杀到逃离柳河镇（被动→第一次主动逃跑）/ 幕2: 被追捕中主动调查老周的线索/幽渊信息（初步主动）/ 幕3: 在碧渊宗伪装杂役并收集尘渊信息（系统化主动）/ 幕4: 利用双界信息差反侦察 → 做出"留下"的决定（战略主动） |
| 活跃幕段 | 幕2-幕4（幕1以被动为主） |
| 关键决策点 | 幕1: "跑还是查" → 选择查 → 去棺材铺 / 幕2: "躲藏还是继续深入" → 选择深入 → 主动去天柱山外围 / 幕3: "在尘渊当杂役安全还是冒险接触上层" → 选择冒险 → 利用废料场接触碧渊宗内部 / 幕4: "留在双界之间 vs 逃到尘渊不回来" → 选择留下 |

### 支线2 — 「赤土的废物就是尘渊的宝贝」（可选·套利逻辑线）
