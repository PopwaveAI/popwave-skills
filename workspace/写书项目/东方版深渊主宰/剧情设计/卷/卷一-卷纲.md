## 本卷定位

| 字段 | 内容 |
|:-----|:------|
| 卷编号 | 卷一 |
| 卷名 | 沧浪沉浮 |
| 本卷命题 | **在吃人的世界里，活下来只是第一步——变成强者才是唯一的路。** |
| 主元爽点 | **权柄争夺（微缩）**——沧浪城街头版权力博弈：疤脸帮会→铁剑门→地下势力三重结构。主角从挨打到反杀到入局 |
| 本卷核心冲突 | 江轩（无力量+有穿越者记忆但失灵）vs 疤脸（暗劲级压迫者）→ 杀疤脸后引出的沧浪城地下势力网。对手不是疤脸一个人，是**疤脸背后那张网** |
| 本卷情绪目标 | **紧张（开局即反杀）→ 爽（变强有反馈）→ 悬疑（老瘸子身份+归墟真相）→ 燃（觉醒）→ 震撼（沈墨登场）** |

## 起点与终点

| 维度 | 卷起点 | 卷终点 |
|:-----|:-------|:-------|
| 主角状态 | 后天·开脉未成。街头苟活的孤儿。不信任何人 | 后天·开脉圆满+练气士第一层血脉解锁。开始有自己的目标——"我要知道我是谁" |
| 关键关系 | 疤脸（压迫者）、老瘸子（同庙乞丐）、阿七（若即若离） | 疤脸已死（ch06）、老瘸子身份爆发（ch22）、沈墨登场上桌（ch25） |
| 力量等级 | 零真气 → 0/12正经全不通 | 12/12正经全通+洗髓前置完成 → 境界垫底但有血脉底牌 |
| 世界认知 | "活下去" → 知道沧浪城地下生态 | 知道了练气士的存在+归墟裂缝+沧浪城有"比疤脸更大的鱼" |
| 阵营/势力 | 无归属。街头的野狗 | 无正式归属，但已被铁剑门注意，沈墨的棋子范围内 |
