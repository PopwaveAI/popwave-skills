## 一、纪元划分

本书历史分五大纪元：

| 纪元 | 跨度 | 特征 |
|:-----|:-----|:------|
| **太古·天道初开** | ∞—万年前 | 天地初分，三千大道自主运转。原生神明诞生（山海古神），无天庭无秩序，各神各管各的 |
| **上古·练气士时代** | 万年前—三千年前 | 大夏/殷商时期，人族练气士文明崛起。以人躯修天道，造出超越神明的法宝阵法。首次威胁到神明的地位 |
| **末古·封神大战** | 三千年前—两千年前 | 天帝与练气士决裂。商周更替表象下，实质是神位大洗牌。天帝陨落，天道崩裂，诸神坠凡，练气士被剿灭 |
| **乱世·诸神割据** | 两千年前—至今 | 封神榜散失，果位碎片散落三界。六大神系互不统属，各自割据洞天争夺香火。归墟裂缝从千年前开始出现并扩张 |
| **将来·归墟将至** | 未来（故事进行时） | 天道裂缝逐年扩大，归墟终将吞没一切。谁能在完全崩坏前重新聚合果位、修复天道——谁就是下一任天帝 |

> 故事开始于**乱世纪元末期**，归墟裂缝已肉眼可见。

## 二、前提大事件

### 2.1 练气士时代的辉煌与覆灭

大夏朝中后期，人族中出现一批能以肉身共鸣天地法则的天才——他们自称**练气士**。

练气士不走神道，不拜神明。他们认为"天道是天地法则的自主运转，不需要神明来代理"。这种理念直接触怒了天帝——神位的合法性建立在"神明是天道在人间的代言人"之上，练气士的存在等于宣告"神明是多余的"。

**覆灭时间线**：
| 阶段 | 事件 |
|:-----|:------|
| 大夏末 | 练气士文明达到巅峰，练气士宗门遍布天下，天帝开始忌惮 |
| 殷商初 | 部分练气士站队商王，试图以人道政权取代天庭统治。天帝震怒 |
| 封神大战导火索 | 天帝设计封神榜——名义上是"敕封正神"，实际上是把所有不听天庭的强者强行绑定到神位上，剥夺其自由意志 |
| 练气士反抗 | 最强的一批练气士识破天帝意图，联合部分不满的古神，掀起对抗天庭的战争 |
| 大战终结 | 天帝陨落，天道崩裂——但练气士也几乎被斩尽杀绝。残存的练气士带着最后的传承隐入秘境，从此人间再无练气士宗门 |

### 2.2 圣者浩劫——诸果位之主坠落

> 这不是"天帝陨落"。**这是整个神权的崩溃。** 所有持有果位的神明——从天帝到最末位的敕封神——在同一个瞬间被打落凡间。天界空了。

封神大战不仅是练气士的覆灭之战，也是诸神的末日。

| 关键节点 | 说明 |
|:---------|:------|
| **天帝陨落** | 最后决战中，天帝与最强的练气士领袖同归于尽。天帝陨落时，天道的维持中枢破碎——果位崩散，封神榜失传 |
| **诸果位之主坠落** | **天道崩裂的冲击波横扫整个天界。** 不只是天帝——**所有果位持有者，无论神力强弱，在同一瞬间被从神位上剥离。** 强大神力直接跌落到天人级，弱小神力连天人都保不住，直接散尽神性沦为凡人。诸神坠凡不是"一些神掉下来了"——是天花板塌了，所有人一起摔 |
| **降格为天人** | "天人"这个境界在今天如此稀缺又如此恐怖的原因：**它不是修炼上去的，是摔下来的。** 现存的每一个天人都曾是更高维度的存在。他们记得自己曾经是神——这才是他们沉默的原因 |
| **天界坠落** | 天庭失去支撑，大片天界碎片坠落人界。原本完整的洞天体系被打散成孤立的碎片 |
| **归墟初现** | 天道裂缝出现后，归墟——万物终结之地——开始通过裂缝向人界渗透。最初只是细微的裂隙，千年后已变成肉眼可见的巨大裂缝 |

### 2.3 圣者浩劫后果（乱世开局前提）

诸果位之主坠落两千年来，世界变成什么样了？

| 层面 | 现状 |
|:-----|:------|
| **神明→天人** | 六大神系各自为政。曾经的神明被困在天人级的躯体中——他们记得自己曾经能掀翻大陆，现在连维持洞天都要精打细算。每个"天人"都是一颗被封印的定时炸弹 |
| **果位** | 散落三界，大部分在洞天中、废墟下、幽冥裂缝边缘。谁拿到就是谁的。但圣者浩劫后，果位的数量只有原来的六成——很多神明的果位在坠落时直接碎裂了 |
| **凡人** | 大多数人"知道有神"，但神已经两千年没有统一显灵过了。江湖宗门的势力反而比神明教会大 |
| **归墟** | 裂缝每十年扩张一次。边荒地区已出现灵气异常的"死域"——草木不生，真气不存 |
| **练气士后代** | 封神大战中被灭门的练气士家族，有些留有血脉但传承尽断。主角就是其中之一——他不知道自己的身世，练气士血脉处于封印状态 |

## 三、核心矛盾

```yaml
core_conflict: "坠落的天神 vs 觉醒的人族 vs 吞噬一切的归墟 — 谁能定义下一个纪元？"
```

本书的根本矛盾不是正邪对决，而是**三个答案的碰撞**。圣者浩劫后，这个问题变得更加尖锐——因为神明不再高高在上，它们就混在人间，跟你我一样会流血：

| 阵营 | 主张 | 代表势力 | 对主角的意义 |
|:------|:------|:---------|:------------|
| **神权回路派** | 修复天道，重建天庭，让诸果位之主归位。神明回到天上，凡人回到地上——这才是秩序 | 封神正统 + 部分山海古神 + 所有想"回去"的坠落天人 | 主角前世记忆知道原版结局是这条路。但世界已经变了——"回去"真的可能吗？还是说诸果位之主的坠落是不可逆的？ |
| **人道路线派** | 天道不需要神。练气士的遗产证明人族可以不靠神明。应该重修练气士文明，建立以人为主的世界秩序 | 练气士后代、天师道系（暧昧中立）、少数人族散修 | ? **主角的家族就是为此而死的**——这是刻在他血脉里的立场。但他前世记忆告诉他人道路线在原版中也没走通 |
| **归墟虚无派** | 秩序本身就是枷锁。让归墟吞噬一切，万物归于无——那才是真正的自由 | 幽冥深处的东西、被归墟侵蚀的天人、堕落者 | 主角的独有天赋让他能接触归墟而不死——这会成为他的诱惑还是责任？何况归墟才是"真正摧毁天界"的那个凶手 |

**三个答案没有绝对的正邪。** 每个阵营都有合理的动机和黑暗面。主角需要在了解世界真相的过程中，找到**自己的答案**——而且他知道"原版答案"，但那个答案正在失效。

## [主角] 标注

- **家族背景**：某支上古练气士望族的最后血脉。封神大战中家族因站队练气士阵营而被诸神联手剿灭。主角的祖辈在灭门前将传承锁入血脉，只留下了"活下来"三个字
- **认知状态**：开局完全不知道自己的身世。以为自己只是沧浪城贫民区的一个普通孤儿，靠偷窃和做零活维生
- **觉醒触发**：练气士传承的第一次解锁需要**真气贯通十二正经**（开脉圆满）。在此之前，主角跟普通江湖少年没有区别——甚至因为血脉封印，他的修行资质看起来比普通人还差（这是保护机制——防止他过早暴露身份被神系发现）
