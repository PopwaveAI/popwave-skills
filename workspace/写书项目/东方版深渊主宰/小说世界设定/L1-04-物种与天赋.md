# L1-04 物种与天赋

> 对齐：PRD §世界观前提 + 融合适配清单 §L1-04 + 主角设定"练气士遗孤·归墟耐受"

## 一、主要种族

### 1.1 人族——天地之灵

人族的优势不是身体强度——他们的身体在三界最弱。人族真正的天赋是**适应性**和**传承性**。

| 特征 | 说明 |
|:-----|:------|
| 经脉结构 | 人人都有完整的经脉系统，理论上人人都可修行武道。但根骨资质决定上限 |
| 修行潜力 | 人族可走武道、练气士、神道三条路径中的任意一条或多条——这是其他种族不具备的灵活性 |
| 平均寿命 | 后天≈80年，先天≈120年，宗师≈150年，大宗师≈300年，天人≈500年 |
| 人口分布 | 人界各处均有，是人界的绝对主导种族 |

### 1.2 妖族——万物有灵

| 特征 | 说明 |
|:-----|:------|
| 起源 | 非人生灵（兽、禽、虫、植）吸收天地灵气后开启灵智，化形为妖 |
| 力量来源 | 血脉+兽形天赋。妖族修的不是经脉，是**妖丹**——体内的一颗能量核心 |
| 化形 | 开脉级即可部分化形（如妖狐保留耳朵尾巴），明劲级以上可完全化人 |
| 寿命 | 妖族寿命普遍比人族长2-5倍，但修行速度慢 |
| 社会地位 | 妖族与人类在边荒共存，人妖混居的城镇并不罕见。但进入中原后妖族会被排斥 |

### 1.3 精怪——非生命启灵

| 特征 | 说明 |
|:-----|:------|
| 起源 | 器物/山川/河流/草木长期受灵气滋养产生灵智 |
| 形态 | 不一定有实体。山精水怪、器灵、灯鬼、石妖均属此类 |
| 力量 | 精怪的力量与其"本体"密切相关——山精可催动山石，河灵可操控水流 |
| 特殊 | 精怪几乎不主动与人冲突，除非被侵犯本体。很多精怪活了几千年，见证了封神大战——它们是活历史书 |

### 1.4 半妖/混血——两界之人

| 特征 | 说明 |
|:-----|:------|
| 起源 | 人族与妖族的混血后代 |
| 特征 | 外表似人，但保留少量妖族特征（兽瞳、尖耳、鳞片等），妖族血脉越浓特征越明显 |
| 力量 | 同时继承人族修武道的天赋和妖族的部分血脉天赋，但两脉都不完整 |
| 社会地位 | 两界都不完全接纳。人族嫌他们"非我族类"，妖族嫌他们"血脉不纯" |

## 二、异兽与怪物

### 2.1 上古异兽（山海经遗脉）

封神大战后大部分上古异兽灭绝或遁入秘境，但仍有少数在人界边缘出没：

| 名称 | 级别 | 特征 | 出现区域 |
|:-----|:-----|:------|:---------|
| 穷奇 | 大宗师级 | 虎身有翼，喜食人，可操控煞气 | 昆仑墟外围 |
| 梼杌 | 宗师级 | 人面虎身，嗜杀 | 南荒深处 |
| 饕餮 | 宗师级~大宗师级 | 可吞噬万物化为己用 | 遗迹深处（沉睡态） |
| 混沌 | 果位金丹级？ | 无面目，无形体，不可直视 | 归墟裂缝 |
| 九尾狐 | 宗师级 | 擅幻术，可读心 | 青丘秘境（已灭） |
| 应龙 | 大宗师级 | 有翼之龙，掌风雨雷电 | 天界碎片区域 |

> 上古异兽不是常见的打怪对象——除了极少数疯子，正常人见到应龙的反应是跑。

### 2.2 幽冥怪物（归墟侵蚀产物）

归墟裂缝渗透人界后，产生了一系列新的怪物——不是天然的，是**被归墟力量扭曲后的生命体**：

| 类型 | 特征 | 威胁级别 |
|:-----|:------|:---------|
| **蚀化兽** | 普通野兽被归墟力量扭曲后的产物。外表溃烂，行动迟缓但不怕痛 | 后天级 |
| **归墟行者** | 人类修行者被归墟侵蚀后的结果。生前实力越强，堕化后越恐怖 | 取决于生前境界 |
| **影孽** | 归墟力量在人间投射的类生命体。无实体，只在阴影中活动 | 先天→宗师 |
| **裂隙守卫** | 归墟裂缝附近的法则具象化产物。不是活的，是"规则"的物理显现 | 不定 |

## 三、特殊血脉与天赋

### 3.1 练气士血脉（? 主角专属）

| 维度 | 内容 |
|:-----|:------|
| **本质** | 上古练气士将传承封印在血脉中，代代遗传的后天改造血脉——类似于基因记忆。不是天生的，是后天炼成的，但一旦炼成就会稳定遗传 |
| **外观特征** | 无。练气士血脉在外观上没有任何标志——这是故意设计的。封神大战后幸存者刻意隐藏了所有血脉外显特征 |
| **核心能力** | ① **天道共鸣**：对天地法则有天然感应，能自行推演功法不足 ② **传承觉醒**：每突破一个境界，自动解锁一段上古记忆 ③ **归墟耐受**：对归墟侵蚀有一定的天然抗性 |
| **封印机制** | 血脉处于封印状态。原因：① 保护机制——凡躯承受不住上古传承的知识量，需逐步解锁 ② 隐蔽机制——防止被神系探测到 |
| **解锁条件** | 第一层解锁：真气贯通十二正经（开脉圆满）。在此之前，主角的修行资质甚至比普通人还差——因为血脉封印消耗了一部分日常修炼产生的真气 |
| **副作用** | 第一次传承觉醒时会引发剧烈的身体反应：经脉暴胀、意识侵入上古记忆碎片、持续数天的高烧。这种反应会在之后每一次觉醒中出现，但烈度逐渐降低 |

### 3.2 归墟耐受——主角的特殊天赋

| 维度 | 内容 |
|:-----|:------|
| **本质** | 练气士血脉中的一个副产物——上古练气士在研究归墟时意外发现的血脉改造 |
| **具体表现** | 主角可以在归墟侵蚀区内停留比普通人长 3-5 倍的时间；被归墟力量直接影响时，恢复速度比常人快；可以"感知"到归墟力量的流动方向（不是看到，是一种直觉） |
| **限制** | ① 不是免疫。时间过长仍然会被侵蚀 ② 刚觉醒时耐受程度很低（≈刚能靠近裂缝边缘10步而不晕厥） ③ 耐受能力随自身境界提升而提高 |
| **叙事功能** | 这是主角在中期进入归墟裂缝探索、获取果位碎片、接触归墟之主的**入场券**。没有这个天赋，宗师以下进入归墟≈自杀 |
| **代价** | 归墟耐受不是单向的天赋——它在保护主角的同时，也在缓慢改变他的存在本质。太多次接触归墟后，主角可能开始失去某些"人之常情"。这一点需要在剧情中逐步揭示 |

### 3.3 其他特殊血脉

| 血脉 | 持有者 | 特征 |
|:-----|:-------|:------|
| **巫觋血脉** | 楚巫神系核心成员 | 天生可与异界存在沟通，精神强度远超常人 |
| **神裔血脉** | 六大神系直系后代 | 天生携带一部分果位亲和力，修行神道比常人快数倍 |
| **天师血脉** | 天师道张氏嫡系 | 对符箓和雷法有天然理解，可跳过学习直接使用部分高阶符箓 |
| **妖皇血脉** | 妖族皇族 | 可强制命令低阶妖族，血脉压制是一切妖族能力的核心 |

## 四、主角初始状态总结

| 项目 | 开局状态 |
|:-----|:---------|
| **种族** | 人族 |
| **血脉** | 练气士血脉（封印态） |
| **修行境界** | 后天·开脉未成——体内有微弱的真气感应，但十二正经一条也没打通 |
| **修行资质** | 看起来比普通人还差（因为血脉封印消耗真气） |
| **实战能力** | 街头打架经验丰富，会一些野路子拳脚。对上正经练过的开脉期武者毫无胜算 |
| **特殊天赋** | 归墟耐受——尚未发现。第一次触发可能在某个意外靠近归墟裂缝的情境下 |
| **信物** | 一块从小佩戴的古玉。不知来历，不知用途。实际上是练气士家族的血脉信物，会在接触到第一块果位碎片时产生共鸣 |
