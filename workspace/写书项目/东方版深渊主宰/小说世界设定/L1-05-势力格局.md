# L1-05 势力格局

> 机制层 | 六大神系 + 凡人势力 + 幽冥势力
> 对齐：PRD §四六大神系设定 + §三神格阶梯 + 融合适配清单 §L1-05 + 素材储备池/01~06-神系素材卡
> 协议概念：果位三级制（弱小→中等→强大神力）

---

## 一、势力总览

封神大战后世界最大的权力格局：**六大神系 + 人间江湖 + 幽冥裂缝**。

> 力量标尺：后天 ≈ 凡间武者 / 先天 ≈ 江湖一流 / 宗师 = 凶门=传奇 / 果位金丹 = 神明

| 势力 | 首领级别 | 洞天 | 立场 | 影响力范围 |
|:-----|:---------|:-----|:-----|:-----------|
| **山海古神** | 强大神力 | 昆仑墟·不周山残骸 | 守旧·藐视后来者 | 山川蛮荒 |
| **楚巫神系** | 中等神力 | 云梦大泽 | 中立神秘·知天命 | 南方沼国·祭祀圈 |
| **封神正统** | 无（天帝缺失） | 天庭残骸 | 天命正统·重建秩序 | 中原势力残余 |
| **天师道系** | 中等偏高 | 龙虎山·洞天福地 | 中立务实·人神双修 | 中原宗门·散修 |
| **藏传密教** | 中等神力 | 须弥山·坛城净土 | 避世·被迫入世 | 西域高原 |
| **民间神祇** | 弱小神力 | 无固定洞天 | 最弱但最接地气 | 遍布人间·乡土 |
| **江湖宗门** | 先天→宗师 | 无洞天 | 各怀心思 | 人界大宗 |
| **归墟幽冥** | 不明（疑似超越果位） | 归墟深处 | 混乱·侵蚀一切 | 裂缝扩张 |

## 二、六大神系详解

### 2.1 山海古神

> 天地初开时的原生神明，与万物同根

| 维度 | 内容 |
|:-----|:------|
| **起源** | 太古天道初开时第一批诞生的存在。兽首人身，掌控山川河流、鸟兽虫鱼的本源权柄 |
| **洞天** | **昆仑墟**——万山之祖，不周山残骸。人界最高的法则交汇点 |
| **当前首领** | 西王母（强大神力）——昆仑墟之主，万妖之宗 |
| **力量风格** | 图腾之力。不需要香火，力量来自自身与天地法则的原始连接 |
| **态度** | 封神大战中古神损失最小（因为天生强大）。他们看不起后来的人造神系和凡人，认为神位本就是古神的，等天道自己修复就好 |
| **对主角的态度** | 古神对人类血脉不感兴趣。练气士血脉对古神来说"不过是模仿神明的人族劣等品"——但只要主角不碰昆仑墟的地盘，古神懒得管他 |
| **弱点** | 守旧僵化。改革和创新对古神来说就是亵渎——这让他们在处理归墟危机上反应最慢 |

### 2.2 楚巫神系

> 第一个"人造神系"——上古巫觋文明感应异界存在创造的神系

| 维度 | 内容 |
|:-----|:------|
| **起源** | 上古练气士时代之前，南方巫觋通过祭祀和感应沟通异界存在，意外造就了第一批非原生神明 |
| **洞天** | **云梦大泽**——迷雾与幻境交织的洞天，里面藏着许多连楚巫自己都说不清的古老存在 |
| **当前首领** | 太一（中等→强大？其真实力量一直是个谜） |
| **力量风格** | 祭祀驱动。巫术体系、预言凶吉、沟通幽冥。楚巫是六大神系中**对归墟了解最深**的 |
| **态度** | 神秘中立。知天命但不逆天。他们认为归墟扩张是天道崩裂后的自然趋势——与其对抗不如"适应"。楚巫对封神大战的立场是"两不相帮" |
| **对主角的影响** | 云梦泽是主角中期的重要地图。楚巫可能第一个感知到主角的练气士血脉——因为他们对"遗落的东西"有天生的感应 |
| **弱点** | 力量依赖祭祀。太一一人的力量压过大半神系，但祭祀体系本身不稳定——一次大瘟疫可以毁掉整个祭祀圈 |

### 2.3 封神正统

> 上古天帝敕封的三百六十五路正神体系——天庭秩序的核心

| 维度 | 内容 |
|:-----|:------|
| **起源** | 天帝亲手建立的官僚神系。神职分周天星斗，职责分工明确——雷公电母、风伯雨师、值日功曹……各司其职 |
| **洞天** | **天庭残骸**——坠落的天界碎片。大块碎片成了封神正统的据点，小块碎片散落三界成为秘境 |
| **当前首领** | **无**。天帝陨落后群龙无首。目前由几位星君（紫微、勾陈等）轮流主事，但谁也压不住谁 |
| **力量风格** | 星官体系。神职即权柄——每位正神的权柄是天帝敕封的，不是自己修炼来的。所以封神正统的神失去天帝后，力量上限被锁死 |
| **态度** | 天命正统。他们是封神大战最大的输家——天帝陨落、天庭崩塌、果位散失。他们最迫切地想重建天庭，找到新天帝 |
| **对主角的态度** | 封神正统是六大神系中**最想抓到练气士后代的神系**。因为练气士的血脉中可能藏着天帝陨落的真相——甚至可能知道封神榜的下落 |
| **弱点** | 群龙无首，内部争权。紫微星君和勾陈星君谁当天帝吵了两千年还没吵完 |

### 2.4 天师道系

> 上古练气士与封神体系碰撞的产物——既要修天道，又不放弃凡人身份

| 维度 | 内容 |
|:-----|:------|
| **起源** | 封神大战末期，部分练气士看到了神明的力量，部分正神看到了人族修行的潜力。双方碰撞后诞生了"人神合修"的第三条路 |
| **洞天** | **龙虎山·洞天福地**——人界与神界的交界，灵气最稳定的区域之一 |
| **当前首领** | 张天师（中等偏高·实力在宗师级，但凭借符箓可短暂触碰天人级） |
| **力量风格** | 符箓、丹鼎、雷法。不完全依赖果位，也不完全依赖肉身修炼——**符箓可以借用天地法则而不需要神明许可** |
| **态度** | 中立务实。"神明不该高于人族"是天师道的核心理念。他们既不支持重建天庭（怕古神重新压人一头），也不支持完全恢复练气士文明（怕人族内部大乱）。他们想要的是**平衡** |
| **对主角的影响** | 天师道可能是主角最早的盟友——因为天师道的传承本身就来自练气士的残篇。主角的练气士血脉解锁后，天师道的功法会变成他最重要的参考 |
| **弱点** | 两头不讨好。古神觉得他们是僭越者，封神正统觉得他们是叛徒。天师道一直走钢丝 |

### 2.5 藏传密教

> 上古佛教东传后与本地神系融合——密续修持通往果位的独特路径

| 维度 | 内容 |
|:-----|:------|
| **起源** | 封神大战后期，外来的佛教修行者与本地神系融合，以密续修持和转世传承开辟了第三条神道路径 |
| **洞天** | **须弥山·坛城净土**——层叠的曼荼罗世界，以密续法理构成的独特洞天 |
| **当前首领** | 持明（中等神力·深不可测） |
| **力量风格** | 转世传承、曼荼罗坛城、护法神体系。核心不是果位碎片而是**续**——通过修持密续，让自己与果位法则"同频" |
| **态度** | 避世修行。两千年不问中原是非。但归墟裂缝扩张到西域后，密教被迫入世——因为归墟直接侵蚀了须弥山的边缘 |
| **对主角的影响** | 密教的转世传承体系可能是主角理解"意识与血脉觉醒"关系的关键参考。密教也可能掌握着部分练气士传承的线索 |
| **弱点** | 人数最少。密教走的是精兵路线，整个神系可能只有几十位核心修行者 |

### 2.6 民间神祇

> 凡人信仰自生之神明。灶神、社神、门神、城隍……香火汇聚成神性

| 维度 | 内容 |
|:-----|:------|
| **起源** | 凡人自发信仰产生的神明。不需要天帝敕封，不需要古老血脉。只要有足够多人真心相信，一个神就诞生了 |
| **洞天** | **无固定洞天**——以乡土庙宇为锚点，祭祀圈即领地。城隍的领地不超过一座城，社神不超过一片乡 |
| **当前首领** | 各自为政，最强的是几位城隍（弱小神力巅峰）和阎罗（中等偏低） |
| **力量风格** | 扎根凡人。香火即命脉。离了祭祀圈力量衰减到不如先天武者 |
| **态度** | 最弱但最接地气。天塌下来最先扛的就是民间神祇——因为他们就住在人中间。归墟裂缝最先威胁的也是他们的祭祀圈 |
| **对主角的影响** | 主角开局在沧浪城底层混饭，最早接触的超自然存在就是民间神祇——可能是某个破败的土地庙、废弃的城隍祠。**民间神祇最贴近普通人的生活，也是向读者释放世界设定的最佳渠道** |

## 三、凡人势力

### 3.1 江湖宗门

| 势力 | 力量级别 | 地域 | 特征 |
|:-----|:---------|:-----|:------|
| **沧浪帮** | 帮主≈先天暗劲 | 沧浪城 | 控制沧浪城九成地下产业。主角开局所在帮会的死对头 |
| **铁剑门** | 掌门≈宗师（凶门） | 边荒 | 边荒最大的武道宗门 |
| **正一盟** | 盟主≈大宗师 | 中原 | 中原江湖正道领袖，与天师道关系密切 |
| **惊蛰阁** | 阁主≈天人 | 南方秘境 | 最神秘的江湖势力，据说是上古练气士后裔建立的秘密组织 |
| **祭祀团** | 取决于服务的神系 | 各区域 | 神明在人间的代言人。有些祭祀团权力大到可以左右一国 |

### 3.2 凡间王朝

| 势力 | 地域 | 特征 |
|:-----|:------|:------|
| **大商残脉（陈国）** | 中原东部 | 自称商王后裔，掌握部分上古练气士时代的遗物 |
| **西岐旧部** | 中原西部 | 封神大战中赢家的后代，与封神正统关系密切 |
| **南荒百越** | 南荒 | 楚巫神系的信徒聚集地，部落联盟制 |
| **西域三十六国** | 西域 | 密教的信徒国家，各自独立但受密教影响 |

## 四、归墟幽冥

| 维度 | 内容 |
|:-----|:------|
| **本质** | 不是"恶"，是"无"。万物终结的回流 |
| **当前状态** | 封神大战后天道出现裂缝，归墟力量通过裂缝开始渗透人界 |
| **已知层级** | 裂缝外围（灵气稀薄的死域）→ 侵蚀带（法则开始紊乱）→ 归墟本体（无规则区——进去后"存在"本身开始瓦解） |
| **归墟之主** | 疑似远古纪元就沉睡在归墟深处的存在，比古神还要古老。无人知道它是什么——甚至不知道它是否真的存在 |
| **对神道的影响** | 归墟力量可以污染果位碎片——被污染的神明会变成"堕神"，丧失理性，力量暴走 |
| **对主角的影响** | 主角的练气士血脉赋予他某种对归墟的天然耐受——但不是免疫。他可以靠意志在侵蚀带多撑一段时间，但不能无限。这是他的天赋，也是他的诅咒 |

## 五、势力关系图谱

```
                    封神正统
                   /   ↑   \
                  /    |    \
                 /     |     \
          山海古神 —— 天师道 —— 楚巫神系
                 \     |     /
                  \    |    /
                   \   ↓   /
                    民间神祇
                        |
                        ↓
                  归墟幽冥 ←——藏传密教（正在被侵蚀边缘）
```

| 关系 | 说明 |
|:-----|:------|
| **封神正统 vs 山海古神** | 互相看不起。古神觉得封神正统是"天帝的走狗"，封神正统觉得古神是"该被取代的旧神" |
| **封神正统 vs 天师道** | 表面上合作（正一盟与天庭势力有往来），暗地里互相防着。天师道不愿意重回天庭管制 |
| **楚巫 vs 归墟** | 楚巫对归墟的理解最深，但这种理解更像是"共处"而非"对抗" |
| **民间神祇 vs 所有神系** | 最大的神系看不上民间神祇（弱小神力在古神面前不如一只蚂蚁），但民间神祇在人间的实际影响力最大 |
| **所有神系 vs 归墟** | 一致敌视归墟扩张——因为归墟侵蚀不分神系。但应对态度各不相同（古神等天道自愈、楚巫想着适应、天师道想主动解决） |

## [主角] 在此格局中的位置

- **开局**：属于最底层——江湖帮会的边缘角色。六大神系不认识他，江湖大宗门不知道他，他甚至不知道自己跟谁有关系
- **中期**：逐渐接触到天师道（功法参考）、楚巫神系（云梦泽）、民间神祇（认知释放渠道）
- **后期**：封神正统和归墟幽冥成为最大的博弈对手。而他身上的练气士血脉让他天然站到了所有神系的对立面——或制高点
- **终局**：他需要决定——站在哪一边？还是走自己的第四条路？
