## 一、修炼资源

### 1.1 灵石——修行界的硬通货

| 品级 | 灵气含量 | 获取难度 | 用途 |
|:-----|:---------|:---------|:------|
| 下品灵石 | 低，杂质多 | 常见，普通灵脉产地皆可采掘 | 后天武者修炼，日常消耗 |
| 中品灵石 | 中等，较纯净 | 较稀缺，需中型灵脉 | 先天→宗师级修炼，阵法驱动 |
| 上品灵石 | 高，极度纯净 | 稀缺，只产于顶级洞天和龙脉节点 | 宗师级以上修炼，神器祭炼 |
| 极品灵石 | 极高，近乎法则实质化 | 罕见，仅存于天界碎片和上古遗迹 | 果位修复，禁咒级阵法 |

**关键设定**：边荒灵气稀薄不是因为灵石稀缺，而是因为**地脉断裂**——灵气流通不畅。所以边荒的下品灵石产量不低（存量矿产），但修炼者吸收效率极差（因为灵气在经脉里走不通）。这才是主角开局"有灵石也升不上去"的根本原因。

### 1.2 丹药

| 类型 | 功效 | 适用层 | 获取 |
|:-----|:------|:------|:------|
| 培元丹 | 固本培元，加速真气积累 | 后天 | 江湖丹坊常见 |
| 洗髓丹 | 淬炼骨髓，辅助洗髓突破 | 后天→洗髓关键 | 较贵，大宗门内供 |
| 凝气丹 | 短时间内大幅提升真气密度 | 先天·明劲 | 战斗型消耗品，有抗药性 |
| 破境丹 | 辅助突破瓶颈（对凶门无效） | 先天→先天内小阶 | 稀有，一颗可抵苦修数年 |
| 续命丹 | 重伤垂死时吊住一口气 | 通用 | 极贵，炼制需数十年药材 |
| 灵髓液 | 灵气液化后的精粹，可替代灵石 | 宗师级以上 | 仅产于顶级洞天 |

**DND 化用**：续命丹 ≈ 高阶复原术；凝气丹 ≈ 爆发药水；破境丹的机制是"临时提升瓶颈处的真气压强"，类似原版的经验值辅助道具，但对凶门无效——因为凶门不是真气问题。

### 1.3 香火愿力（神道专属）

| 特征 | 说明 |
|:-----|:------|
| 来源 | 信徒祈祷、祭祀、供奉 |
| 储存 | 神像/神位牌/庙宇中存放，可调用 |
| 消耗 | 施展神术、维持洞天、对抗归墟侵蚀 |
| 衰减 | 离开祭祀圈越远，可调用香火越少 |
| 与果位关系 | 果位 = 权柄（硬件），香火 = 能源（电力）。二者缺一不可 |

## 二、关键道具

### 2.1 法宝——可成长的装备（化用原版魔法装备/神器）

| 品级 | 特征 | 适用层 | 对标 DND |
|:-----|:------|:------|:---------|
| 凡器 | 比普通武器更锋利坚韧，无特殊效果 | 后天 | +1武器 |
| 法器 | 刻有符文，能增幅真气输出 | 先天 | +2武器，+1防具 |
| 灵器 | 拥有器灵雏形，可认主，随主人成长 | 宗师 | +3武器，特殊附魔 |
| 道器 | 蕴含法则之力，不完整果位碎片嵌入 | 大宗师→天人 | 传奇装备，领域道具 |
| 神器 | 完整果位权柄所化，因果级力量 | 果位金丹 | 神格武装 |

**关键设定**：法宝可升级。一件法器可以通过符文加固、材料熔炼、器灵养成逐步成长为灵器。这是主角的"成长绑定装备线"——对标深渊主宰中索伦的阴影装备不断进化。

### 2.2 功法/术法（可获取的知识资源）

| 品级 | 特征 | 对标 DND |
|:-----|:------|:---------|
| 凡·粗浅 | 江湖大路货，最多练到先天明劲 | 1-2环法术 |
| 灵·正宗 | 宗门核心功法，可到宗师 | 3-5环法术 |
| 玄·秘传 | 仅限少数传承，可到大宗师 | 6-7环 |
| 圣·绝学 | 上古练气士遗留，可到天人 | 8-9环 |
| 神·天书 | 天道残缺法理，涉及果位金丹 | 传奇法术/神级 |

**主角关键**：他的练气士血脉每解锁一阶自动觉醒一段功法/术法记忆——相当于自带圣·绝学级的传承宝库。但问题是：**传承需要当前境界的身体承受力**，过早觉醒高品级功法≈身体先于境界崩掉。

### 2.3 上古遗物

封神大战和练气士覆灭留下的大量不可复制道具：

| 类型 | 用途 | 叙事功能 |
|:-----|:------|:---------|
| **传承玉简** | 一次性读取的功法/知识载体 | 主角解锁练气士传承的落地渠道 |
| **阵盘** | 预刻阵法的便携式道具 | 类似DND卷轴，一次性或有限次 |
| **残缺果位** | 漂浮的果位碎片，可直接吸收 | 权力博弈标的物，吸收后神道修行加速 |
| **封神榜残片** | 封神榜碎裂后的纸片 | 天帝陨落真相的线索，全书最大谜团之一 |
| **归墟石** | 幽冥裂缝附近出产的特殊矿石 | 可吸收归墟力量，用于制造堕化武器 |
| **龙骨/凤羽** | 远古神兽遗骸 | 炼丹炼器的顶级材料 |

## 三、货币体系

### 3.1 凡人界货币

| 货币单位 | 购买力参考 |
|:---------|:----------|
| 铜钱 | 一个馒头约 2 文 |
| 银两 | 1两 ≈ 1000文，普通客栈一晚 ≈ 3钱(0.3两) |
| 金锭 | 1两金 ≈ 10两银 |

### 3.2 修行界货币

| 货币单位 | 兑换参考 | 说明 |
|:---------|:---------|:------|
| 下品灵石 | 1下品 ≈ 100两黄金 | 修行界最基础的交易单位 |
| 中品灵石 | 1中品 ≈ 100下品 | 大宗交易和阵法驱动 |
| 上品灵石 | 1上品 ≈ 100中品 | 有价无市，一般不流通 |
| 功勋/贡献点 | 宗门内部流通 | 大宗门的内部积分体系 |
| 香火愿力 | 不可兑换灵石 | 神道专属，信徒越多产出越高 |

**开局经济状况**：主角在沧浪城底层，银钱勉强糊口。灵石？摸都没摸过。他看到的灵石交易属于"听说过没见过"的层面。

## 四、特殊资源——归墟力量

归墟不是被"采集"的资源——它更接近一种可以被**借用**的力量。靠近裂缝的修行者可以在短时间内获得超出自身境界的力量输出，但代价是身体/神智被逐步侵蚀。

| 利用方式 | 风险等级 | 使用者 |
|:---------|:---------|:-------|
| 吸收归墟石能量战斗 | 高 | 堕落者、亡命徒 |
| 借助归墟裂缝施展禁忌术法 | 极高 | 堕神、疯掉的修行者 |
| 以归墟之力炼制堕化武器 | 中→高 | 黑暗势力 |
| ? **主角的特殊耐受** | 低 | 主角可以比常人多撑数倍时间————但不能无限 |

## [主角] 相关

- **开局资产**：一文不名。住在沧浪城贫民区的破庙，靠偷窃和接小零活维生
- **第一桶金可能性**：帮会跑腿 → 黑市交易 → 某次意外在废弃道观里发现一本残破功法（凡品·入门级）
- **重要伏笔物资**：主角身上有一块从小佩戴的玉佩，他不知道那是上古练气士血脉的信物————实际上是练气士家族留给后代的血脉钥匙之一。玉佩本身没有特殊力量，但它会在主角接触到第一块果位碎片时产生共鸣
