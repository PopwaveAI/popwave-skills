## 二、每次跃迁详表

### 跃迁①：开脉1/12（第一条正经打通）

| 项目 | 内容 |
|:-----|:------|
| **触发事件** | ch13-ch15，第一次反杀疤脸时，濒死状态下偶然冲开第一条正经。真气从"若有若无"变成"有了一条线"——这是他十六年来第一次真正感受到真气 |
| **反馈强度** | ★★ 第一颗糖——"原来真气跑起来是这个感觉" |
| **对手层次变化** | 从"被混混追"到"能打普通帮众" |

### 跃迁②：开脉4/12（获得功法后系统性突破）

| 项目 | 内容 |
|:-----|:------|
| **触发事件** | ch22-ch25，从废弃道观获得残破功法+练气士基础记忆觉醒，一周内连破3-4条 |
| **反馈强度** | ★★★ "有功法就是不一样"——明确感受到变强了 |

### 跃迁③：开脉10/12（归墟石辅助冲击）

| 项目 | 内容 |
|:-----|:------|
| **触发事件** | ch35，用归墟石拼死练功，突破第10条。代价是归墟石碎裂，江轩第一次意识到"归墟力量有代价" |
| **反馈强度** | ★★★ 里程碑——离圆满只差最后2条 |

### 跃迁④：开脉12/12+传承觉醒（全书第一个大质变）

| 项目 | 内容 |
|:-----|:------|
| **触发事件** | 传承觉醒中获得的基础功法替代了凡品功法的粗糙路径，淬炼骨髓→体质飞跃 |
| **社会地位变化** | 小幅度提升。在帮会中不再是"谁都可以踩一脚"的存在，开始被当作"可用的人" |
| **新地图开启** | 沧浪城内城边缘·铁剑门外围区域 |
| **对手层次** | 洗髓级·帮会精英 |
| **练气士觉醒** | 无新觉醒。洗髓是体质层面的变化，传承层仍停留在第一层 |

### 跃迁③：洗髓 → 明劲

| 项目 | 内容 |
|:-----|:------|
| **触发事件** | 某次生死战斗（可能是被疤脸逼到绝路）中，真气突破量变到质变的临界点，第一次外放为劲气 |
| **社会地位变化** | 进入"江湖人"的门槛。不再是小角色，开始被各方势力注意到 |
| **新地图开启** | 边荒更广阔的区域·铁剑门内部 |
| **对手层次** | 明劲~暗劲级 |
| **练气士觉醒** | 无新觉醒。但练气士功法的优势开始显现——他修炼速度比同级快（因为功法品级碾压） |

### 跃迁④：明劲 → 暗劲

| 项目 | 内容 |
|:-----|:------|
| **触发事件** | 持续实战 + 传承功法推演下，明劲穿透性升级为暗劲。过程不需要漫长闭关——他的血脉决定了他是"越打越强"的类型 |
| **社会地位变化** | 成为边荒有头有脸的人物 |
| **新地图开启** | 云梦泽边缘（楚巫神系势力范围）·第一个上古遗迹入口 |
| **对手层次** | 暗劲~宗师门槛 |
| **练气士觉醒** | 第二层传承觉醒：一段关于"练气士与封神大战真相"的记忆碎片——他开始意识到，自己家族的覆灭不是"站错队"那么简单 |

### 跃迁⑤：【凶门】→ 宗师（全书第一大转折）

| 项目 | 内容 |
|:-----|:------|
| **触发事件** | ① 练气士血脉积累的能量足够 ② 接触到了果位碎片/归墟力量/强烈的世界法则冲突 ③ 古玉佩在关键节点产生共鸣，激活了凶门钥匙 |
| **社会地位变化** | **质变。** 不再是江湖人——是"那个怪物"。宗师以下的武者在他面前如同孩童。神系开始注意到他 |
| **新地图开启** | 洞天入口·天界碎片·归墟裂缝边缘 |
| **对手层次** | 宗师~大宗师级·神系使者·上古异兽 |
| **练气士觉醒** | 第三层觉醒——最剧烈的一次。完整的上古练气士功法体系解锁，并包含了"练气士与归墟的关系"的核心信息。江轩开始理解：练气士覆灭的原因不仅是站错队——他们还触碰了归墟的禁忌 |
| **代价** | 踏入宗师的那一刻，他的存在本质发生改变。体内真气→灵力的转化不可逆。从这一刻起，他再也回不到普通人的世界了 |

### 跃迁⑥：宗师 → 大宗师

| 项目 | 内容 |
|:-----|:------|
| **触发事件** | 吸收足够的灵气 + 至少融合一枚完整果位碎片 + 法则之力从借用变为掌控 |
| **社会地位变化** | 进入六大神系博弈的桌面上 |
| **新地图开启** | 各神系洞天·须弥山·龙虎山 |
| **对手层次** | 大宗师~天人级·堕神 |
| **练气士觉醒** | 传承揭示：练气士血脉的本质——它是上古练气士为了"在末日后重建文明"而封印的知识库。江轩的血脉不是唯一的。他可能是最后一个，但不是唯一一个曾经存在的 |

### 跃迁⑦：大宗师 → 天人

| 项目 | 内容 |
|:-----|:------|
| **触发事件** | 开始触碰因果层级的法则 + 可能涉及归墟之主的直接接触 |
| **社会地位变化** | 成为决定世界走向的力量之一 |
| **新地图开启** | 归墟深处·天道核心区域 |
| **对手层次** | 天灾级·神系之主 |
| **练气士觉醒** | 最后一段传承：练气士文明全盛时期的面貌 + 一段跨越时空的留言："如果你读到了这里，那就做你觉得对的事。我们信错了天道——但你们可以重新信点什么。" |

### 跃迁⑧：天人 → 果位金丹

| 项目 | 内容 |
|:-----|:------|
| **触发事件** | 完整的封神榜线索 + 足够多的果位碎片 + 做出三路选择（不是靠打，是靠"决定这个世界走哪条路"） |
| **社会地位变化** | 新天帝候选 / 练气士文明复兴者 / 归墟代言人——取决于选择 |
| **新地图开启** | 天庭中枢·归墟最深处 |
| **对手层次** | 归墟之主·其他天帝候选 |
| **代价** | 成型为果位金丹的那一刻，江轩将失去作为"人"的所有属性——或者保留一部分（？——这取决于他做的是什么选择） |

## 四、防崩机制

| 场景 | 规则 |
|:-----|:------|
| 宗师级战斗波及凡人 | 灵力护体可自动抵消低阶攻击，但大宗师级以上战斗仍会造成大范围环境破坏——这不是设计失误，是叙事需求（让读者感受高武层的重量） |
| 越级战斗（宗师打大宗师） | 几乎不可能正面取胜。但练气士的传承功法 + 古玉佩的共鸣 + 归墟力量可作为临时底牌，创造"拼死一战能跑掉"的局面——过程必重伤 |
| 果位碎片吸收上限 | 同一个人可融合的果位碎片数量有限。不是越多越好——胡乱吸收会导致不同权柄冲突，人格分裂/力量暴走 |
| 归墟力量借用时长 | 宗师级：在侵蚀带可撑 1 炷香（约 30 分钟）；大宗师级：可撑数个时辰；天人级：可短时间自由出入但不驻留；果位金丹：可在归墟中短暂开辟安全区域 |
| 主角不会因传承觉醒实力暴涨 | 每一层传承解锁的是"知识"和"功法"，不是直接的能量灌注。江轩仍然需要自己修炼/吸收灵气/积累真气→灵力→神力的能量。传承只是给了他"知道该往哪走"的路线图——路还是他自己走的 |
