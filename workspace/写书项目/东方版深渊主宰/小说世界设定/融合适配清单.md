# 融合适配清单

> 跨界融合：深渊主宰框架 × 东方多神系 · 上古练气士遗孤主角
> 产出阶段：Phase 0 · 逐维标注"融合决策对 L1 维度的影响"
> 出处：PRD.md §移植声明 + §移植锚点对照 + §主角方向确认

---

## 融合决策摘要

| 原版元素 | 本作处理 | 焊接策略 |
|:---------|:---------|:---------|
| 索伦·10级平民·贫民区孤儿 | 练气士遗孤·江湖底层·宗门边缘人 | 保留"靠自己在黑暗中成长"的核心弧线，替换出身来源 |
| 杀戮经验量化 | 武侠层靠实战+传承觉醒，高武层靠果位碎片 | 保留量化成长感，层间过渡方式不同 |
| 神格阶梯·信仰值·神国 | 果位三级制·香火愿力·洞天经营 | 文化替换，结构保留 |
| 古代奥术帝国遗迹 | 上古练气士遗迹 | 替换包装，保留"超越时代的知识埋藏"定位 |
| 圣者浩劫（诸神坠凡） | 封神大战（天帝陨落·诸神坠凡） | 保留核心矛盾结构 |
| 深渊层·规则侵蚀 | 幽冥/归墟裂缝·幽冥规则侵蚀 | 保留层次化威胁设定 |
| DND 六大属性 | 保留为内层计算模型，对外用东方修行概念 | 不对外暴露，仅做底层一致性保障 |
| 冒险者公会 | 宗门+祭祀团+民间势力组合 | 不照搬单一公会结构 |

## 逐维适配（含主角影响）

### L1-01 世界蓝图

| 影响点 | 变化 |
|:-------|:-----|
| 世界结构 | 原版平面世界+多元位面 → 东方**天·地·人三界 + 洞天福地 + 归墟幽冥** |
| 核心区域 | 原版"琥珀城→白马城→冰雪国度"路线 → 需设计对等的东方地理（江湖边陲→中原→昆仑/幽冥边境） |
| 力量来源 | 原版魔网+奥术 → **天地灵气 + 香火愿力 + 幽冥浊气** 三重来源 |
| ? **主角影响** | 练气士遗孤的初始地：**人界边陲·废弃宗门的遗墟** 或 **某江湖小城的贫民窟** — 对标索伦的琥珀城贫民区 |
| 世界基调 | 暗黑生存 + 史诗冒险（保留原版压抑感，叠加东方神系宏阔色彩） |

### L1-02 力量体系

| 影响点 | 变化 |
|:-------|:-----|
| 等级架构 | 原版 DND 1-20 职业等级 → **武侠·高武·神话 三重大阶梯**，每层内细分 |
| 职业体系 | 原版 游荡者/巫师/战士等 → **东方修行路径**（练气士传承 / 神道果位 / 武道外功 / 巫道祭祀） |
| 法术体系 | 原版魔网+环位法术 → **功法·术法·神通 三层**，释放依赖真气/灵力/神力 |
| 属性模型 | DND 六维 → 保留为底层，对外用**根骨·悟性·心性**等东方概念包装 |
| ? **主角影响** | 练气士遗孤的血脉优势：**传承觉醒**替代原版"升级选专长" — 每突破一层觉醒一段上古记忆/功法。但开局时传承是**锁死的**（因为社会底层接触不到灵气），需要靠实战磨出真气才能触发第一层解锁 |

### L1-03 历史与驱动力

| 影响点 | 变化 |
|:-------|:-----|
| 核心事件 | 原版"圣者浩劫" → **封神大战·天帝陨落** |
| 上古文明 | 原版"奥术帝国" → **上古练气士文明（大夏/殷商遗脉）** |
| 当前威胁 | 原版深渊层扩张 → **幽冥/归墟裂缝扩散** |
| ? **主角影响** | 练气士遗孤的家族在封神大战中被灭门。**上古练气士站错了队**——他们是天帝的反对者，试图以人道取代天道。所以主角的血脉不仅仅是"古代传承"，还是**被六大神系共同追杀的烙印**。开局阶段没人知道他的底细，但一旦身份暴露——所有神系都会想抓住他/消灭他 |

### L1-04 物种与天赋

| 影响点 | 变化 |
|:-------|:-----|
| 主要种族 | DND 精灵/矮人/半身人 → **人族 + 妖族 + 精怪 + 半妖混血** |
| 飞升血脉 | 原版龙脉/阴影血脉等 → **练气士血脉·巫觋血脉·神裔** |
| ? **主角影响** | **练气士血脉**：特征为"天道亲和"——对天地灵气有天然感应，能自行推演功法不足。但开局被封死（原因：凡躯未经淬炼，承受不了上古传承）。解锁条件：**真气贯通十二正经**（武侠层高段） |

### L1-05 势力格局

| 影响点 | 变化 |
|:-------|:-----|
| 神系阵营 | DND 神系阵营 → **六大神系 + 幽冥势力 + 凡人势力** |
| 凡人势力 | 原版冒险者公会/王国 → **宗门·帮会·祭祀团·江湖散修** |
| 上古遗族 | 原版精灵/矮人国度 → **隐世练气士后裔（秘境隐居）** |
| ? **主角影响** | 开局在**江湖底层**——可能是某个小宗门的杂役，或一个帮会的边缘人。与六大神系无关，与上古遗族无关。这正是对标索伦的"从污泥里爬起来"——所有大势力的博弈跟他没关系，他要先解决温饱和生存 |

### L1-06 资源与物品

| 影响点 | 变化 |
|:-------|:-----|
| 修炼资源 | 原版经验值/魔石 → **灵气浓度·灵石·丹药·功法残卷** |
| 神道资源 | 原版信仰值/神性 → **香火愿力·信众·祭祀贡品** |
| 货币 | 原版金德勒 → 对应东方货币体系 |
| ? **主角影响** | 开局一无所有。对标索伦的**从贫民窟偷窃起步**——他可能在废弃的练气士遗迹里捡到第一本残破的功法 |

### L1-07 术语与文明底色

| 影响点 | 变化 |
|:-------|:-----|
| 术语体系 | DND 术语替换为东方神话术语（神格→果位，神国→洞天，法术模型→术法/神通等） |
| 文明底色 | 西方中世纪 → **先秦/商周文明底色 + 六朝志怪质感** |
| 命名要求 | 至少 3 个核心术语为本书原创（区别于类型标配） |
| ? **主角影响** | 主角的初始认知是江湖底层视角——他不知道果位、不知晓六大神系的真正底细。术语通过他的认知逐步释放给读者 |

### 静态金字塔

| 层级 | 人口密度 | 社会地位 | 破坏力 | 战斗张力 |
|:-----|:--------|:---------|:-------|:---------|
| 武侠底层 | 百万人级 | 凡夫·帮众·杂役 | 拳脚兵刃·碎砖裂石 | 巷战·械斗·暗杀 |
| 武侠顶端~高武门槛 | 万人级 | 门派中层·一方豪强 | 剑气十丈·踏水而行 | 门派对决·遗迹争夺 |

## 焊接策略

| 原版机制 | 本作对应 | 焊接方式 |
|:---------|:---------|:---------|
| 杀戮经验 → 升级 | 实战积累 + 传承觉醒 + 果位收集 | 三重并行：武侠层靠实战（硬实力）、高武层靠传承解锁+果位碎片（质变）、神话层靠香火+权柄争夺（量变到质变） |
| 职业专长 | 功法 + 术法 + 神通 | 专长→功法（可学习/可推演），天赋专长→血脉觉醒（不可传授） |
| 法术位 + 魔网 | 真气运转 + 灵力消耗 | 武侠层：真气（内耗型）；高武层：真气+灵气（内外结合）；神话层：神力（香火续航） |
| 神国建设 | 洞天道场经营 | 保留"领地建设→防御→香火聚拢"的种田线 |
| 传说度 | 名望/传说/香火基数 | 三合一：江湖名望（武侠层）→修行界传说（高武层）→香火基数（神话层起点） |
| 深渊层 | 归墟幽冥 | 保留"扩张式威胁 + 可利用但不稳定"的核心矛盾 |
