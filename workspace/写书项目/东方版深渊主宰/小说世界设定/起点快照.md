## 一、主角初始状态

### 基本信息

| 项目 | 内容 |
|:-----|:------|
| 姓名 | 江轩 |
| 年龄 | 约 16-17 岁 |
| 外貌 | 瘦削，因长期营养不良看起来比实际年龄小。黑发黑瞳，五官有几分上古血脉的棱角感（不自知），手掌有常年打杂/偷窃留下的老茧和疤痕 |
| 性格雏形 | 警惕、寡言、不信任何人。在底层活下来的必然后果——不是冷血，是知道"善意是最贵的东西" |
| 人生信条（开局） | "活下去。其他的以后再说。" |

### 穿越者设定（核心新增）

| 项目 | 内容 |
|:-----|:------|
| **本质** | 江轩的灵魂来自现代世界，约 16-17 年前穿越至此，降生在练气士遗孤躯体中。带着前世对"这个世界的原版走向"的记忆 |
| **他知道什么** | 他知道这个世界的"原版设定"——果位体系规则、各神系的底牌、封神大战的"真相"、遗迹藏着什么、谁会在什么时候背叛。他前世至少追读过数本同类设定的小说和设定集 |
| **他不知道什么** | **圣者浩劫发生后，他知道的东西正在集体失效。** 原版中不该陨落的神灵陨落了，该在某个时间点发生的事件提前或消失了。他像一个看完剧透却走进了导演剪辑版电影院的人 |
| **叙事功能** | ① **先知失灵的持续张力**——"我知道接下来会发生什么……等等，不对！" ② 与读者同步探索——他知道一些设定（让读者爽），但世界变了（让读者紧张）③ 凶门的认知门槛——他知道"宗师=传奇"，但亲身经历时才发现知道和经历是两回事 ④ 终局——他必须放弃"我知道剧本"，做出自己的选择 |
| **关键心理** | 他是穿越者这件事不打算告诉任何人。这是他的底牌也是他的诅咒 |

### 身体状态

| 项目 | 状态 | 备注 |
|:-----|:------|:------|
| 武道境界 | 后天·开脉未成 | 体内有微弱的真气感应，十二正经全不通（**0/12**）。打通第1条=正式入门，第6条=可以持续运转真气一炷香，第12条=开脉圆满→洗髓 |
| 根骨资质 | 看起来极差 | 因为血脉封印消耗了一部分日常产生的真气 |
| 体力 | 比普通同龄人弱 | 营养不良的后果 |
| 特殊状态 | 练气士血脉·封印态 | 完全锁死。开脉圆满后触发第一层解锁 |
| 有无暗伤 | 有。左膝旧伤（阴雨天疼）、肋骨曾被打断两根（已愈合但不平整） | 江湖底层的"勋章" |

### 技能与知识

| 项目 | 水平 | 来源 |
|:-----|:------|:------|
| 拳脚功夫 | 野路子一阶。没有正经学过，但是跟人打架的经验非常丰富 | 街头斗殴 |
| 偷窃 | 熟练。能糊口但不精湛 | 生存技能 |
| 认字 | 略懂。认得一些常用字，但读不了完整文章 | 小时候在道观外听老先生念书偷学的 |
| 江湖常识 | 很多。知道谁不能惹、哪条街不能走、什么货不能碰 | 底层生存积累 |
| 世界观 | "这世界就是吃人的。不吃别人就被吃。"——**但他前世知道这个世界真正有多大。** 他知道果位体系、知道六大神系、知道归墟是什么。问题是——他知道的版本和现实对不上 | 经验主义 + 穿越者记忆（未验证） |

### 装备与资产

| 项目 | 状态 | 说明 |
|:-----|:------|:------|
| 钱 | 几乎没有。身上最多十几个铜板 | 一餐饭 2 文，他一天挣不到 20 文 |
| 住处 | 沧浪城贫民区一座废弃破庙 | 漏雨，有老鼠，但至少有个屋顶 |
| 武器 | 无正式武器。一把缺口的短刀（藏在内衣里） | 从死人身上扒来的 |
| **古玉佩** | 挂在脖子上，从小贴身佩戴 | 不知道来历。只知道"捡到我的人说这是我身上唯一的东西" |
| 衣物 | 一身洗得发白的粗布衣，打满补丁 | 仅此一套 |

## 三、世界初始状态

### 表面状态（普通江湖人的认知）

| 层面 | 状态 |
|:-----|:------|
| 沧浪城 | 三不管地带。沧浪帮控制地下产业，铁剑门插了一脚，官府形同虚设 |
| 边荒 | 灵脉断裂，灵气稀薄。真正的强者不会久留 |
| 江湖 | 后封神时代两千年，江湖宗门已经成了人界的实际统治者 |
| 神明 | "听说过没见过。"对凡人来说，神明是传说——跟封神大战一样远的历史 |

### 暗面状态（读者知道+主角逐步发现）

| 层面 | 状态 |
|:-----|:------|
| **圣者浩劫·诸果位之主坠落** | **不是"天帝陨落"那么简单——是全部果位持有者被一起打落凡间，降格为天人。** 天界空了。现存所有"天人"级强者，都曾是高高在上的神明。这就是为什么当前世界的"天人"境界如此稀少又如此恐怖——他们不是"练上去的"，是"摔下来的" |
| 天道 | 裂缝逐年扩大。归墟正在逼近人界 |
| 六大神系 | 各有算盘，互不统属。封神正统最急，古神最稳 |
| 归墟 | 裂缝从百年前开始加速扩张。边荒是最早受到影响的区域 |
| 练气士血脉 | 还存在——但活着的练气士血脉可能不到十人 |

## 四、伏笔初始状态

| 伏笔 | 初始状态 | 触发条件 |
|:-----|:---------|:---------|
| **古玉佩** | 主角戴在身上但不知用途 | 接触到果位碎片时产生共鸣 |
| **练气士血脉封印** | 完全锁死 | 开脉圆满→第一次传承觉醒 |
| **归墟耐受** | 未发现 | 第一次靠近归墟裂缝（可能会在边荒某次意外中触发） |
| **老瘸子的真实身份** | 隐藏 | ? 由剧情触发——江轩前世记忆中"没有这个角色"。他第一次意识到自己带的剧本有缺页 |
| **封神榜下落** | 无人知晓——但江轩"知道"一个可能的位置（对应原版中某处） | 全书中后期核心谜团。他去时会发现——那里没有封神榜 |
| **归墟之主** | 疑似存在。江轩前世记忆中是终局BOSS——但真实世界中对它的描述只有模糊的传说 | "如果连归墟之主都存在——那我还有哪条信息能信？" |
| **圣者浩劫的规模差异** | **第零认知偏差**——江轩开局就发现了 | 他前世记忆："圣者浩劫=天帝陨落，诸神坠凡"; 真实世界：**诸果位之主全部坠落，降格为天人**。"如果圣者浩劫比我知道的严重三倍——那其他事呢？" |
