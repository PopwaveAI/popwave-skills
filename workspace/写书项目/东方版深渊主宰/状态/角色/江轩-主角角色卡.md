# 江轩 · Lv4 主角角色卡

> 对标：深渊主宰·索伦（外冷内暗·从污泥爬起）
> 匹配：世界宪法 C16-C18（练气士血脉）·动态升级表

---

## 设计目的

贯穿全文的灵魂。一个穿越者，带着对这个世界的"原版历史"的记忆，却发现自己降生在一个已经被改变的时间线上。他所知道的"未来"正在一步步失效——读者和他一起经历"原版认知被现实推翻"的反复冲击。核心主题从"从污泥爬起"升级为"知道未来的人，面对一个不按剧本走的世界"。

## 叙事能力

① 穿越者的先知视角：他知道本书世界的"原版走向"（谁该死、谁该活、哪个事件导致哪个结果），但他的每一句"我知道"都在被现实打脸——因为圣者浩劫把因果链砸断了 ② 先知失灵的张力：读者和他一起经历"我以为我知道会发生什么"→"等等，和我知道的不一样"→"那我还有什么用"的弧线 ③ 每次传承觉醒不仅是力量升级，也是"原版记忆vs现实"的认知冲突 ④ 最终，他必须放弃"我知道剧本"，做出真正自己的选择

## 三、心理

### 核心性格特质（3 条）

**特质 1：警惕**
- 表现：不轻易相信任何人，习惯观察后再行动。对面走过来一个人，江轩在两步前就已经在想"他有没有威胁"
- 边界：在经历重大情感转折（如找到可以信任的人）后，警惕性会暂时放松——但这个人的背叛会在全书最痛的地方
- 盲点：过度警惕导致错失善意。有些本来可以成为盟友的人，因为江轩一开始就没给机会而变成敌人

**特质 2：务实求生**
- 表现：不逞英雄。能跑就跑，能谈就谈，下跪也不在话下。活下来是第一要务
- 边界：涉及到原则底线的选择（老瘸子被抓/有人触及他身世真相）时，务实会让位给"必须做的事"
- 盲点：太务实会让人觉得"这个人不靠谱"——中期和天师道合作时，对方对他的第一评价就是"你太精了"

**特质 3：对"真相"的偏执**
- 表现：一旦知道"还有更多内情"，就停不下来。传承觉醒后即使师父说"别深究"他也会继续挖
- 边界：这种偏执在安全环境下是优点（能发现别人忽略的线索），在极端危险下是致命缺陷（明知前面是陷阱也要去）
- 盲点：真相不一定都是好的。有些真相知道之后，就再也回不去了

**特质 4（穿越者专属）：先知失灵的焦虑**
- 表现：他知道太多"不应该知道的事"——果位体系的规则、各神系的底牌、某些遗迹的位置。但他的每一次"我知道"都在被现实打脸。圣者浩劫是他遇到的第一个认知偏差："我记得是天帝陨落——但事实是诸果位之主全部坠落。差太多了。"
- 边界：发现某个关键事件与记忆不符时——他会陷入短暂的自我怀疑，然后更拼命地验证自己还"有用"
- 盲点：过度依赖前世记忆导致忽视当下线索。有些危险在前世故事中不存在，但在这个世界里是致命的

### 决策逻辑

```
日常决策倾向：计算型——不做没把握的事，不做没有退路的选择。宁可多绕两步，不赌命。
危机决策倾向：压力下会反向变形——越危险越冷静，句子越短，行动越快。不是冲动型冒险，是"我算过的，这条生路概率最高"。
决策阈值：当"这条信息关乎我的身世/练气士覆灭的真相"时，日常的计算模式会崩溃。他会选择风险更高的路径。
```

### 核心缺陷

```
缺陷①：无法信任他人。底层生活教会他"信任是送命的捷径"——但一个人不可能靠孤身走完从后天到果位金丹的路。
缺陷②（穿越者专属）：知道太多却无法验证——他不敢暴露自己的先知身份，因为这比练气士血脉更危险。"你知道有人要刺杀你"和"你怎么知道的"是两回事。
对谁最致命：对他自己。他掌握的信息越多，越孤单。中期需要组建势力时，他要么编造理由（撒谎），要么沉默（失去信任）。
整个故事中改善了多少：从"不信任何人"→"信任 2-3 个特定的人"→"愿意为信任的人冒风险"——但永远不会变成"信任世界"。
```

## 四、驱力（四层完整）

| 层级 | 内容 | 故事中何时显露 |
|:-----|:------|:--------------|
| 表层目标 | 活下去，变强，不再任人宰割 | 开局即显——贯穿全书 |
| **深层需要（升级）** | **① 验证自己带来的"剧本"有多少是真实的 ② 搞清楚圣者浩劫为什么比他知道的严重三倍 ③ 如果这个版本的真相才是真的——那练气士覆灭背后又是什么？** | 开局即存在——他是穿越者，这件事从第零章就在驱动他 |
| 黑暗冲动 | 如果知道全部真相后发现自己家族的经历是这个世界的常态——那就毁了这个世界的秩序，让所有人为练气士陪葬 | 中期得知部分封神大战真相时浮现 |
| 恐惧 | ① 自己最后变成怪物（宗师=凶门=非人，归墟耐受=逐步失去人之常情）② 发现自己的血脉根本不该存在 | 中后期逐步暗示 |

### 驱力冲突

表层目标（活下去）与深层需要（追真相）之间的张力最大。追真相意味着冒更大的生命危险——但江轩每次的选择都是"先活下来，带着线索，下次再挖"。这个平衡会在中期某次被迫二选一时被打破。

## 六、演化

### 弧线阶段

**阶段 1：污泥求生（约 ch1-ch40）**
- 起止章节/幕：卷一前半
- 触发事件：无——这就是他的日常生活
- 前心态："活着就行。别的跟我不相干。"
- 后心态："活是活下来了，但我不能一辈子这样。"
- 关键决策：在某个被逼到绝路的关头，选择了"拼命一试"而不是"继续缩着"
- 表象变化 vs 本质变化：表——开始主动寻找变强的机会；里——驱动力从"被动求生"转向"主动掌控"

**阶段 2：追问身世（约 ch41-ch100）**
- 起止章节/幕：卷一中后段→卷二
- 触发事件：开脉圆满，第一次传承觉醒——意识到自己有秘密
- 前心态："我一直以为自己是个没人要的孤儿。"
- 后心态："我的血里有东西。我不知道是什么，但我必须知道。"
- 关键决策：在安全苟活和冒险探索之间，选择了后者
- 表象变化 vs 本质变化：表——开始追查练气士线索；里——被动的好奇心升级为主动的偏执

**阶段 3：凶门（约 ch101-ch150）**
- 起止章节/幕：卷三
- 触发事件：古玉佩共鸣 + 归墟裂缝初接触 + 被迫面对宗师级对手
- 前心态："宗师就是天花板。我想都不敢想。"
- 后心态："我跨过去了。我变了一个物种。"
- 关键决策：在利用归墟力量突破凶门（速度快但危险）和正统修炼突破（安全但耗时）之间选择了前者——这就是他的底色：活着比什么都重要，但没有力量连活着都保证不了
- 表象变化 vs 本质变化：表——实力从先天跳到宗师，质变；里——他开始明显地感觉到自己"不是人了"

**阶段 4：棋手入场（约 ch151-ch230）**
- 起止章节/幕：卷四
- 触发事件：踏入宗师后，神系开始注意到他
- 前心态："变强了，可以报仇了。"
- 后心态："这个世界不是好人坏人的问题。是三个答案里我选哪个的问题。"
- 关键决策：从"个人恩怨"的叙事转向"世界走向"的抉择
- 表象变化 vs 本质变化：表——介入神系博弈；里——从"为自己活"到"为某种理念活"

**阶段 5：最终抉择（约 ch231-终局）**
- 起止章节/幕：卷五→卷六
- 触发事件：得知天道路线三个答案的全部真相
- 前心态："我恨他们。他们灭了我的家族。"——愤怒驱动
- 后心态：取决于他的选择
- 关键决策：人类的最后一位练气士遗孤，要决定天道、神权和人族三者的平衡
- 表象变化 vs 本质变化：表——果位金丹的权力；里——他从逃命者变成护道者（或毁道者）

### 弧线类型

□ 成长弧 — 从一无所有到掌握力量
□ 转变弧 — 从"为自己活"到"为更大的意义活"
□ 揭露弧 — 随着传承觉醒，读者和江轩同步发现世界真相

### 不可逆点清单

**事件 1（约 ch40-50，卷一末）**
- 开脉圆满，第一次传承觉醒。知道自己不是普通孤儿。
- 故事前：他以为自己是沧浪城最普通的底层少年。
- 故事后：他知道自己的血里有秘密——而且这秘密可能会要他的命。

**事件 2（约 ch100-110，卷三初）**
- 踏入宗师·凶门。体内真气→灵力转化完成。
- 他意识到自己"不是人"了——不是贬义，是事实上不再是普通人类的生命形态。
- 旧的驱动力（"变强就不被人欺负"）失效了。因为他已经强到了"欺负"这个词不适用于他的程度。

**事件 3（约 ch230+，卷四/五交界）**
- 被迫选择三路答案之一（或走自己的第四条路）。
- 最黑暗的时刻：得知练气士覆灭的完整真相——可能包括"练气士自己也不干净"。
- 他要决定是重建、推翻、还是让一切归于虚无。

### 关系演化摘要

```
江轩 ? 老瘸子：
  起点：陌生的收留者和被收留者——江轩住他的破庙，分他半块饼
  第一个转折点（开脉圆满觉醒时）：老瘸子在江轩高烧时说了句江轩后来才意识到不对劲的话："你的血在烧。正常人的血不会这样烧。"

  第一个转折点（明劲后）：江轩第一次能把疤脸逼平
  终点：疤脸死于某个更大势力的灭口——不是江轩杀的，但他消失后江轩意识到"我躲了那么久的人不过是别人的一条狗"

江轩 ? 天师道：
  起点：互相试探——天师道通过某种渠道感知到练气士血脉的存在，派出使者接触
  中期：合作——江轩需要天师道的功法知识，天师道需要江轩的练气士血脉去解读某些上古遗迹
  张力点：天师道的"平衡"立场和江轩的"真相偏执"会有冲突
```

## 八、说法风格

| 字段 | 内容 |
|:-----|:------|
| 核心声线 | 冷、短、不漏底。不主动说话，但开口就不废话 |
| 日常语气 | 低语速，句尾偏沉。不熟的人觉得他闷，熟的人知道他在观察 |
| 压力语气 | 越危险越冷静，句子更短。没有"害怕"的语气词。想好怎么说才会开口——但不说谎，因为说谎需要解释，而他在危机中没有多余的解释时间 |
| 对不同对象的分层 | 对不熟的人：能用一个字回答绝不用两个字。对同伴：会给推理过程和结论——"我觉得不行，因为..."。对敌人：要么不说话直接动手，要么说最后一句："你有遗言吗？"对弱者/普通人：收着锋芒，话会稍微软一点 |
| 句式偏好 | 短句。先事实，后判断。少反问（反问太浪费力气）。少感叹号 |
| 高频词与称呼 | "嗯。"、"走。"、"几成把握？"、"听你的。"（前期很少说，后期开始信任后出现） |
| 禁用表达 | 不会说"我叫江轩，请多关照"这种开场白。不会喊口号。不会用"为了正义"这种词。不会抒情 |
| 价值观外露方式 | 不直接说"我信什么"——通过选择暴露。比如选择救一个人而不是夺取一件宝物，读者就明白了：这个人还有底线 |
| 谎言方式 | 几乎不说谎。要隐瞒的时候，会不回答，不会编假话。被逼到必须说谎时，句子会更完整、更像推理——因为他在用逻辑构建一个虚伪但自洽的信息包 |
| 情绪失控的破音点 | ① 涉及练气士覆灭真相和家族之死时——语速会突然变快，措辞变糙 ② 唯一信任的人背叛时——会沉默很久，然后只说一个字："走。" |
| 独白风格 | 自问自答式。"我不知道他在想什么。但我知道他有他的理由。所以我只要知道我要什么就行了。"——极简、工具化、不沉溺情感 |

### 示例台词

- **日常**：（疤脸踹了他一脚）"记住了。"（不是威胁，是真记住了）
- **决策**：（阿七问他走哪条路）"左。右边三个岔口，没法退。左边至少能翻墙。"
- **冲突**：（第一次面对宗师级对手）"...打不过。但能不死。你信我一次。"
- **情感**：（老瘸子失踪后）"...（沉默很久）他走之前说了什么？"
- **黑暗**：（想到家族覆灭的怒火）"他们杀了所有人。那我也不需要留什么情面。"
