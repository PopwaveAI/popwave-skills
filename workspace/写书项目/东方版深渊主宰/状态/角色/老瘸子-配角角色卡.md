# 老瘸子 · Lv3 配角角色卡

> 起源：起点快照·伏笔NPC | 对标：深渊主宰中的神秘老人类角色
> 定位：隐藏导师/观察者——他知道的远比他表现的多

---

## 设计目的

承载"练气士末裔的守护者"这一隐性动机，跨幕存在。读者对这个角色有"他到底是谁"的持续好奇，他的选择会在凶门前后产生决定性后果。

## 叙事能力

① 江轩最早的安全感来源（唯一的"熟人"）② 练气士血脉线索的潜伏释放者③ 凶门危机的隐形引导者④ 消失后的谜团驱动后续剧情

---

## 一、身份

| 字段 | 内容 |
|:-----|:------|
| 姓名 | 老瘸子（真名未知——江轩从没问过，问了也不会说） |
| 年龄 | 看起来 60+，实际可能更老 |
| 与主角的关系 | 住在同一座破庙的"室友"——江轩分他吃的，他偶尔给江轩一句"有用的提醒" |
| 一句话概括 | "一个瘸了一条腿的老乞丐，在沧浪城最破的庙里住了二十年，正好等到一个带着古玉佩的孤儿长大。" |

---

## 二、标签

```
瘸腿 · 半瞎（左眼浑浊） · 话少但偶尔精准 · 看起来风一吹就倒 · 但某些时刻会露出"不像乞丐"的瞬间——比如看到某种符文时瞳孔微缩
```

---

## 三、驱力

| 层级 | 内容 |
|:-----|:------|
| 想要什么 | 保护最后一位练气士血脉安全长大，直到他踏入凶门。之后呢？他说了不算 |
| 最怕什么 | 练气士血脉被神系提前发现——他守了二十年，不能在他手里断送 |
| 什么情况下会脱离主角 | 当神系的追踪逼近到无法隐藏时，他会主动消失——把追踪引走，或留下最后的线索后自己离开 |

---

## 四、叙事功能

```
□ 情感锚点 — 江轩早期唯一的（准）家人
□ 信息管道 — 通过"随口一句话"释放设定（"你的血在烧""那道裂缝三年前还没这么大"）
□ 催化剂 — 推动江轩踏入凶门——他的消失和留下的线索是江轩无法不追的动机
□ 镜像/对照 — 一个守了二十年的人和一个还不知道自己该守什么的人
```

---

## 五、弧线

```
弧线类型：揭露弧——本质不变（他一直都是最后的练气士守门人），但读者逐渐认识他的真实面貌
起点心态："把最后一只幼崽护到能自己走路。"
终点心态（预期终点）：完成使命后，老瘸子可能在大宗师卷再次出现——或者死在某个不为人知的角落里，江轩永远不会知道他的名字
触发事件：江轩开脉圆满、传承觉醒时——老瘸子知道自己"不用再守着秘密了"
```

## 六、登场场景表

| 卷/幕 | 事件 | 该角色的作用 |
|:------|:-----|:------------|
| 卷一·前期 | 日常破庙互动 | 建立"奇怪的老人"形象，埋下伏笔（偶尔的精准发言） |
| 卷一·末 | 江轩开脉圆满·传承觉醒 | 老瘸子在江轩高烧时说出"你的血在烧"——第一个提示 |
| 卷二·中 | 江轩第一次被疤脸追杀到破庙附近 | 老瘸子"随口"指了一条荒废的地道——救了江轩一命 |
| 卷三·前 | 凶门前夕 | 老瘸子失踪。留下一件物品（可能是练气士信物或指向某地的地图） |

## 七、说法风格

| 字段 | 内容 |
|:-----|:------|
| 基础语气 | 松散、忽大忽小——像要散架的老风箱 |
| 节奏 | 慢。中间长停顿，好像在想事情（其实是在算） |
| 常用表达 | "哦。""是吗。""年轻人就是沉不住气。" |
| 回避表达 | 不会直接回答"你是谁"——装作没听见 |
| 对不同对象的口气差 | 对江轩：偶尔带点关心藏在"你死了谁给我带饭"这种话里。对陌生人：装糊涂——"啊？我听不见" |
| 情绪失控时的变形 | 几乎不失控。唯一一次可能在江轩觉醒时——眼神突然锐利了半秒然后恢复 |
| 潜台词习惯 | 所有重要信息都藏在好像不经意的闲聊里 |
| 示例台词 | 日常：—"回来了？今天有饭没？"—"没。""哦。" 伏笔：—"你那块石头……挺好看的。别弄丢了。" 关键：—"你的血在烧。正常人的血不会这样烧。" |

---

## 八、【题材扩展】

| 字段 | 内容 |
|:-----|:------|
| 真实境界 | 疑似大宗师级——至少曾经是 |
| 练气士关联 | 可能是最后一代练气士的守门人/仆从家族后人，非练气士血脉本人，但肩负守护使命 |
| 瘸腿原因 | 可能是在封神大战后的追捕中受的伤——一直在躲避神系的感知 |

---

> 本角色卡对齐起点快照·老瘸子设定。后续 plot 需保证"老瘸子消失"是江轩踏入凶门的核心推动力之一。
