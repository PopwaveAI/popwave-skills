## 二、爽点设计（爽点层）

### 套路识别

**套路①：开局即反杀**
- **铺垫**：疤脸追债→踹飞→按手要切手指——绝望
- **触发**："濒死平静"下真气异动
- **反转**：1/12经脉突开→一刀反捅→反杀
- **释放**：疤脸倒下时不可置信的眼神

**套路②：穿越者第零偏差**
- **铺垫**：前世记忆——"我知道这个世界的剧本"
- **触发**：疤脸第一章就要杀他——按原剧本疤脸不该这么早出手
- **释放**："这个世界不对"——穿越者记忆的第一次被现实打脸
- **读者收益**：开篇就告诉读者——"别以为主角有剧本就无敌，他的剧本是错的"

**套路③：匹夫无罪怀璧其罪（轻埋）**
- **铺垫**：搜出铁剑门腰牌+未写完的信
- **钩子**：疤脸背后有人——而且那人知道疤脸处理不了才派人督战

### 情绪弧线

```
(章首) 压抑(起床就被堵) → 恐惧(暗劲碾压) → 绝望(刀尖对准手指)
    → 平静(濒死时诡异的冷静) → 异动(真气震开) → 爆发(反杀)
    → 初爽(疤脸倒下) → 震惊(搜出腰牌) → 认知颠覆(第零偏差)
    → 冷处理(拖尸入井) → 留白(老瘸子的饼)(章末)
```

### 章末钩子

| 层级 | 内容 |
|------|------|
| L1 情节未决 | 疤脸死了——但他的腰牌是铁剑门的，那封没写完的信是谁写的？铁剑门接下来会怎么做？ |
| L2 世界认知 | 第零偏差确认——穿越者记忆不可靠。圣者浩劫比他知道的严重三倍 |
| L3 人物关系 | 老瘸子看到血迹没问——他是不在乎，还是早就知道江轩会杀人？ |

## 三、角色与人设（角色层）

### 登场角色行为锚定

| 角色 | 出场行为 | 锚定特征 | 动机 |
|------|----------|----------|------|
| 江轩 | 被堵→被踹飞→濒死→反杀→搜线索→处理尸体 | 务实·濒死能爆发·杀完不拖泥带水·有穿越者视角的警觉 | 活下去；验证记忆偏差 |
| 疤脸 | 大清早带人堵门→踹胸→踩碗→拔刀要切手指 | 暴躁·以暴力为手段·不把江轩当人看 | 追债是面子问题；立威给其他人看 |
| 老瘸子 | 回来看到血迹→没问→放下半块饼→走 | 沉默·有边界感·知道不该问的不问 | 可能是知道江轩做了什么，选择不问（伏笔） |
| 打手×2 | 拦门→按住江轩→反杀后逃跑 | 底层打手的正常反应：仗势欺人时狠，见血跑最快 | 听疤脸的，小命最重要 |

### 关键对白

| 角色 | 对白 | 语气 | 潜台词 |
|------|------|------|--------|
| 疤脸 | "又跑？欠的钱什么时候还？" | 狞笑·踩碗·不把对方当人 | 我是来让你记住谁说了算的 |
| 疤脸 | "不还钱，就拿根手指抵。" | 轻松·像在说一件小事 | 切你一根手指对我不是事，对你是一辈子的记号 |
| 江轩(内心) | "疤脸不该死在这个时间点。他才第一章。……这个世界，跟我知道的不一样。" | 震惊·自言自语 | 我的穿越者剧本是错的——这比我欠钱可怕一百倍 |
| 老瘸子 | （沉默）→把饼放下 | 无声 | 我知道你做了什么。我不问是因为我不需要问。你活着就好 |

## 四、感官与画面（感官层）

### 环境基线6维

| 维度 | 内容 |
|:-----|:------|
| 视觉 | 清晨破庙——香案歪倒，供台积灰，屋顶漏光。疤脸右脚踩在碎碗片上，白瓷碴子溅了一地。短刀刃口有锈 |
| 听觉 | 脚步声在庙外石板路上停住→门被踢开→碗碎裂声→江轩被踹飞的闷响→刀背拍脸的轻响→真气震开时体内的"嗡"→刀捅进肋骨的噗声→尸体拖行沙沙声→老瘸子的脚步声停在门槛外→饼放在青砖上的轻响→脚步声远去 |
| 触觉 | 清晨地面的冰凉透过薄裤传到膝盖→胸口挨一脚的钝痛（像被铁锤砸了一下）→后脑撞墙的震荡→刀尖抵着小指根的冰凉刺痛→反扑时肩膀撞到疤脸下巴的骨感→刀捅进去那股阻力和突破感→拖尸体时手抓着布料的粗粝 |
| 嗅觉 | 破庙的霉味+尘土味+疤脸身上的酒气+铁锈味（刀刃）+血味（涌出来时那股腥甜） |
| 味觉 | 嘴里沁出的血沫咸味→混着尘土味的喘气 |
| 体感 | 真气线从若有若无突然变成"通了一条管"的贯通感——不是疼，是全身突然有劲了 |

### 战斗三段落

| 段落 | 内容 |
|:-----|:------|
| 单方面碾压 | 疤脸踹飞→江轩撞墙→被按住→刀尖对准手指 |
| 反扑 | 濒死平静→真气突破→肩膀撞下巴→夺刀 |
| 反杀 | 一刀捅进侧肋→疤脸倒下→打手逃跑 |

### 氛围渲染要点

| 位置 | 氛围 | 手法 |
|:-----|:-----|:------|
| 开场破庙 | 压抑·不安 | 脚步声先于人影出现，读者知道有事要来但不知道什么事 |
| 疤脸踩碗 | 压迫 | 不先打人先踩碗——"你的东西我可以随便毁"的权力展示 |
| 刀尖抵指 | 绝望 | 从"欠债还钱"升级到"要留记号"——读者相信他真的会切 |
| 反杀瞬间 | 爆发 | 短句快节奏，不抒情不断句——真气震开→撞下巴→夺刀→捅进去→疤脸倒 |
| 章末破庙门槛 | 留白 | 老瘸子什么也没问。饼放在门槛上。这个动作比任何台词都重 |
