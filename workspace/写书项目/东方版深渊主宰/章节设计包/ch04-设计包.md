## 二、爽点设计（爽点层）

### 套路识别

**套路①：被围堵·逆境求生**
- **铺垫（压抑）**：两天两夜逃亡，只啃了半块饼，体力见底
- **触发（绝望）**：五个人堵了巷子两头——没路可跑
- **冲突（血战）**：1 vs 5，以夺来的短棍为武器，逐个放倒四个
- **反转（濒死→突破）**：第五个壮汉逼到绝路——濒死中冲开第二条正经
- **释放（碾压逆袭）**：突破后一刀反劈，刀风震脱对方武器

**套路②：战斗中突破**
- **铺垫**：体内真气在濒死时剧烈震荡，接近断裂边缘
- **触发**：前世记忆闪回关于开脉的知识碎片
- **突破**：将残存真气全部压向瓶颈——第二条正经震开
- **释放**：真气翻倍、力量翻倍、一刀决胜负

**套路③：扮猪吃虎·留悬念**
- **铺垫**：全程副手只站在巷口笑，没出手
- **触发**：江轩翻墙后回头看——副手还是不动
- **悬念埋点**："他是在试我"——暗劲级boss没出全力

### 情绪弧线

```
(章首) 疲惫警惕 → 紧张 → 绝望(1vs5) → 挣扎(逐个放倒)
    → 力竭濒死(真气将断) → 拼死一搏 → 突破通电般的爽感
    → 反杀碾压 → 逃脱 → 后怕+警觉(副手没出手)
    → 不确定的隐忧(章末)
```

### 章末钩子

| 层级 | 内容 |
|------|------|
| L1 情节未决 | 疤脸副手（暗劲级）始终没出手——他不是追不到，他是在"看"江轩怎么打 |
| L2 人物关系 | 副手玩味的眼神暗示：他可能不是单纯为疤脸报仇，他有其他目的 |
| L3 力量悬念 | 开脉2/12了仍然不是暗劲级的对手——下一条路在哪？ |

## 四、感官与画面（感官层）

### 环境基线6维

| 维度 | 内容 |
|:-----|:------|
| 视觉 | 清晨贫民区巷子——青石板路面有积水，墙皮剥落露出土坯，矮墙头有几株枯草。五个人的影子从巷口拉长压过来。短棍上的铁钉锈迹斑斑 |
| 听觉 | 小孩嬉闹声突然中断→脚步从两头逼近→短棍破风声→拳脚到肉的闷响→江轩撞墙的撞击声→第二条正经冲开时体内"嗡"的一声低鸣→翻墙落地的闷响→粗喘 |
| 触觉 | 半块饼碴子扎嗓子→夺棍时虎口震麻→左肋挨了一脚的钝痛→撞墙后背火辣→经脉突破时那股气从丹田窜到四肢的"通电感"——不是疼，是麻胀到发烫 |
| 嗅觉 | 雨后巷子的潮气、汗味、生锈铁钉的铁腥味、墙角有股尿骚 |
| 味觉 | 咬饼时混着血沫的咸味→突破后嘴里的铁锈味更重了 |
| 体感 | 体力接近极限的虚脱感→突破后真气灌入四肢的充盈感→翻墙时手臂肌肉的爆发感 |

### 战斗六段式（关键动作）

| 段落 | 内容 |
|:-----|:------|
| 接敌 | 矮身闪避第一拳→卸力反抓→拧腕夺棍 |
| 反击 | 短棍横扫第二个膝盖→骨裂声→倒地 |
| 消耗 | 以一对三游斗→格挡→侧踢→后背撞墙换位→挨一脚闷响 |
| 濒死 | 被打飞撞墙→真气线剧烈颤抖→壮汉举刀 |
| 突破 | 真气压向瓶颈→第二条正经震开→眼中精光 |
| 碾压 | 一刀反劈震脱大刀→补脚踹断胸骨→翻墙脱离 |

### 氛围渲染要点

| 位置 | 氛围 | 手法 |
|:-----|:-----|:------|
| 开场巷子 | 日常→不安 | 小孩嬉闹声突然停滞，"太安静了"的警觉触发 |
| 巷战 | 紧张压迫 | 1vs5的绝望感，逐步放倒但体力也在消耗，伏笔："副手没出手" |
| 墙角濒死 | 绝境 | 真气将断的生理描写 + 前世记忆闪回 = 双重压力 |
| 突破瞬间 | 爽感爆发 | 从气若游丝到真气翻倍的"通电"转换——短句快节奏 |
| 章末翻墙 | 轻松安全→后怕 | 翻墙后靠在院落墙根的粗喘 + 回头看到副手没追的隐忧 |
