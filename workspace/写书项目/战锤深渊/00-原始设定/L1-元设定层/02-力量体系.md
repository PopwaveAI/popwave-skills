# 02-力量体系

> 追溯自：故事引擎「力量层级」·「碰撞点」·「宪法·力量有代价」
> 继承自：DND 3.5e 等级体系 + 深渊主宰神性体系，叠加混沌腐蚀风险
> 核心架构：超凡者 Lv1-20 → 传奇（20级以上）→ 神格等级（独立线·待规画）

---

## 境界总览

```
  超凡者 (Lv1-20)  ← DND 3.5e 标准职业体系
  ├── Lv1-5   低阶超凡  ← 新手冒险者 → 地区佣兵精英
  ├── Lv6-10  中阶超凡  ← 成熟冒险者 → 城市级战力
  ├── Lv11-15 高阶超凡  ← 王国级强者 → 触及位面奥秘
  └── Lv16-20 巅峰超凡  ← 费伦顶战 → 超凡之巅（20级是凡物天花板）

  传奇 20级以上
  ├── 传奇初段  ← 踏入传领域
  ├── 传奇中段  ← 传奇稳固
  ├── 传奇高段  ← 接近半神
  └── 传奇巅峰  ← 准神（封神前夜）

  神格等级（独立等级线）
  ────────────── 待规划
```

### 各阶段详细特征

| 阶段 | 等级 | 战力对应 | 力量来源 | 特征 |
|:-----|:-----|:---------|:---------|:-----|
| **低阶超凡** | Lv1-5 | 精英佣兵→冒险者公会的普通成员 | 基础职业能力、戏法/1-3环法术 | 有职业等级但不超出"雇佣兵"范畴。混沌腐蚀可逆（净化/休息） |
| **中阶超凡** | Lv6-10 | 城市英雄→冒险者公会的中坚力量 | 4-6环法术/职业特性强化/魔法装备 | 已有招牌级能力。施法在亚空间留下微弱痕迹 |
| **高阶超凡** | Lv11-15 | 王国级强者→法师工会高层/将军级 | 6-8环法术/传奇门槛/位面知识 | 开始触碰位面边界。施法会在亚空间留下永久标记。混沌腐蚀开始不可逆 |
| **巅峰超凡** | Lv16-20 | 费伦顶战（伊尔明斯特、凯尔本之流） | 9环法术/史诗专长/终极职业特性 | 凡物天花板。再往上走就必须舍弃"凡人"的身份属*。20级巅峰时，混沌四神能感知到——"值得关注" |
| **传奇初段** | 20+ | 传奇冒险者 | 传奇法术/神性火花雏形 | 正式踏入非人领域。拥有领域碎片的雏形，可在深渊/星界独立生存 |
| **传奇中段** | 20+ | 恶魔领主·下级/古龙级 | 神性火花核心 | 力量辐射影响周围环境。混沌四神记下你的名字 |
| **传奇高段** | 20+ | 恶魔领主·上级/圣者级 | 完整领域雏形/半神格 | 举手投足有天地异象。灰骑士种子进入觉醒窗口。四神必定出手 |
| **传奇巅峰** | 20+ | 准神/封神前夜 | 完整神格雏形+信仰基础 | 只差一步踏入神力领域。此阶段必面对"封神=被外神感知"的悖论 |

## 属性体系（DND 3.5e）

### 六维属性

| 属性 | 缩写 | 影响范围 |
|:-----|:----|:---------|
| **力量** | STR | 近战命中/伤害、负重、冲撞、攀爬等体能检定 |
| **敏捷** | DEX | 远程命中、闪避（AC）、潜行、先攻、反射豁免、巧手 |
| **体质** | CON | 生命值、强韧豁免、耐力、恢复速度、中毒抗性 |
| **智力** | INT | 法术学习/数量、知识检索、炼金术、逻辑推理 |
| **感知** | WIS | 意志豁免、洞察、直觉、野外生存、冥想恢复 |
| **魅力** | CHA | 社交交涉、 intimidation、术士施法、灵能抗性 |

### 关键属性阈值

| 值 | 门槛 | 说明 |
|:---|:-----|:------|
| **10** | 常人标准 | 普通成年人的平均水平 |
| **14** | 精英标准 | 训练有素的职业者水平 |
| **18** | 种族极限 | 普通天赋能达到的上限 |
| **20 → 超凡领域** | ⭐ **超凡门槛** | 属性突破20即进入超凡领域，自动获得一项与该属性对应的特殊能力（如超凡敏捷→阴影掌握、超凡体质→再生初级） |
| **25 → 传奇门槛** | ⭐ **传奇门槛** | 属性突破25触发传奇领域能力。跨过此门槛=在对应的战斗维度达到传奇级（如25敏捷→阴影领域） |
| **30** | 神性之壁 | 凡骨无法达到的属性值，需神性/神格推动 |

### 属性成长规则

- **起点白板**：江轩穿越后属性为标准凡人模板（力10 敏10 体10 智14 感12 魅10）。智力14残留穿越者的知识处理能力
- **每2职业等级获得1点自由属性**：可分配到任意属性，不可超过当前种族/境界上限
- **超凡者阶段属性上限**：20（需突破大门槛才可继续提升）
- **传奇阶段属性上限**：25（需突破传奇门槛才可继续提升）
- **属性差距的实战影响**：双方核心战斗属性差距≥5点时，高属性一方产生**压制效果**（命中/伤害/豁免均获得额外加值）。差距≥10点时低属性方几乎无法造成有效伤害

### 属性 vs 等级的独立维度

属性和职业等级是两个独立的成长维度：
- **等级** = 职业能力、法术位、专长
- **属性** = 基础身体素质、豁免、技能检定
- 两者互相补充但不可互相替代。高属性低等级=天生神力的菜鸟；高等级低属性=技法精湛但身体羸弱的老手

## 核心路径：双线并行

江轩的成长有两条互相关联又互相矛盾的线：

### A线：DND 3.5e 等级线

```
超凡者 Lv1-20
  ↑ 战斗/任务/学习/积累——标准DND升级路径
  
传奇 20级以上
  ↑ 突破「传奇之门」：需要第一枚神性火花 + 意志试炼 + 位面承认
  传奇初段 → 中段 → 高段 → 巅峰
  ↑ 神性火花收集 + 领域领悟 + 意志淬炼

神格等级（独立线·待规划）
```

**风险**：每升一级都在扩大"在亚空间的足迹"。超凡者高阶以上，混沌能追踪你的施法痕迹。传奇阶段，四神能直接感应你的大致位置。

### B线：灰骑士种子线（隐藏成长）

```
休眠态（0%）→ 应激觉醒（20%）→ 半觉醒（50%）→ 完全觉醒（100%）
                         ↓
                    每次濒死/直面混沌时被动触发，不可主动控制
```

**风险**：种子觉醒越快，江轩的人性流失越快。完全觉醒后他是否还能保留自我意志，无人知晓。

**双线关系**：
- A线超凡者 → 传奇突破 → 刺激B线成长（种子成熟度 +5~10%）
- B线半觉醒态 → 给A线提供临时战力加成（对混沌单位等效 +1大阶）
- B线完全觉醒 → 覆盖A线（种子主导人格，江轩不再是"江轩"）

## 突破条件

| 突破 | 条件 | 典型手段 |
|:-----|:------|:---------|
| **Lv1-5 → Lv6-10** | 实战积累+职业里程碑 | 完成3-5个冒险任务，掌握核心职业能力 |
| **Lv11-15** | 英雄事迹+位面认知 | 在重大事件中存活，首次接触异位面信息 |
| **Lv16-20** | 凡人极限淬炼 | 费伦顶级挑战（深渊探险/远古遗迹）中证道 |
| **超凡→传奇** | ⭐ 第一枚神性火花 + 意志试炼 | 击杀圣者/恶魔领主、继承古神遗产、完成传奇仪式 |
| **传奇初→中段** | 第二枚神性火花 + 领域雏形 | 在深渊建立自己的势力碎片 |
| **传奇中→高段** | 第三枚神性火花 + 种子觉醒≥30% | 混沌四神亲自出手后仍活下来 |
| **传奇高→巅峰** | 完整神格雏形 + 信仰基础 | 拥有至少100信徒，体系初具规模 |
| **传奇→神格** | 待规画 | — |

**江轩的特殊突破条件**：
- 每次B线觉醒需要：混沌直接威胁 + 江轩意志检定成功
- 完全觉醒需要：四神之一亲自出手 + 江轩选择"接受种子"而非"封印种子"

## 力量天花板

**超凡者天花板**：Lv20（职业等级上限，凡物能到达的最高）
**传奇天花板**：传奇巅峰（准神）
**故事上限预期**：传奇高段 + 半觉醒种子（等价于能对抗四神化身）
**理论极限**：传奇巅峰 + 种子完全觉醒 + 成功保留人性 → 弑神级

|:-------|:-----------|:-----|

## [江轩] 标签

- 双线并行：超凡者等级 Lv1-20 → 传奇 + 灰骑士种子觉醒度 0%→100%
- 种子觉醒度影响施法稳定性（成熟度高→灵能压制强→法术反噬低）
- 但种子觉醒度越高 → 四神的感知锁定越精确
- 20级后突破传奇之门 = 第一个重大剧情里程碑
