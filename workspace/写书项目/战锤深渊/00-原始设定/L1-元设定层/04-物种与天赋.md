## 主要种族（可玩种族）

| 种族 | 特征 | 可玩 | 势力倾向 |
|:-----|:------|:----:|:---------|
| **人类** | 适应力强，信仰多样，在动荡之年中最灵活 | 是 | 分散各势力 |
| **精灵** | 长寿，奥术亲和，银月城为核心聚集地 | 是 | 倾向于秩序/孤立 |
| **矮人** | 坚韧，抗腐蚀，锻造大师 | 是 | 倾向于秩序 |
| **半身人** | 不起眼，运气好，信息/走私网络 | 是 | 中立 |
| **龙裔** | 龙族血脉，个体战力强 | 是 | 分散 |
| **提夫林** | 恶魔血脉，在动荡之年中受到排挤 | 是 | 倾向于自保 |
| **半精灵** | 人类与精灵混血 | 是 | 中立 |
| **半兽人** | 在剑湾边境外围部落聚居 | 是 | 中立/混乱 |
| **深潜者混血**（⚠️新增） | 与克苏鲁深潜者的混血后代，天生灵能敏感，但精神不稳 | 否（仅NPC） | 倾向于混沌/孤僻 |

## 特殊体质/血脉/天赋

| 名称 | 拥有者 | 效果 |
|:-----|:-------|:-----|
| **灰骑士原体种子** | 仅江轩 | 混沌感知、灵能抗性、种子觉醒时释放反混沌领域 |
| **神性火花亲和** | 封神者 | 能够吞噬/融合神性火花，建立自己的神格 |
| **灵能敏感体质** | 深潜者混血/术士血统 | 天生能在无训练下感知亚空间，但容易发疯 |
| **混沌印记** | 被混沌腐蚀者 | 身体上出现混沌符文，持续侵蚀意志。可逆期短 |

## [江轩] 标签

- 唯一灰骑士种子携带者 → 在费伦没有第二个同类
- 种子让他免疫混沌直接精神控制，但仍是四神的第一优先级猎杀目标
- 深潜者混血/灵能敏感者可以"感知"到他身上的种子——像黑暗中一盏灰白色的灯
