# 05-势力格局

> 追溯自：故事引擎「世界锚点」·「碰撞点·混沌四神入侵」
> 三层势力：费伦本土势力 × 混沌入侵者 × 深渊/外神暗线

---

## 主要势力

### A层：费伦本土

| 势力 | 类型 | 控制区 | 当前状态 |
|:-----|:-----|:-------|:---------|
| **博德之门** | 城邦共和国 | 剑湾南部 | 至上真神教团渗透中，上层议会已被部分腐化 |
| **深水城** | 魔法贵族统治 | 剑湾北部 | 法师工会内部分裂，奸奇教团在高层活动 |
| **银月城** | 精灵王国 | 银月林地 | 被纳垢瘟疫包围，封锁边境 |
| **竖琴手** | 秘密结盟组织 | 全费伦 | 发现了混沌入侵的真相，正在全大陆招募抵抗力量 |
| **散塔林会** | 盗贼工会联盟 | 剑湾商路 | 两线押注：一边与混沌教团交易，一边给抵抗者预留退路 |
| **秩序诸神圣者** | 流浪的堕神 | 全费伦 | 神祇们以圣者形态行走大地，力量受限，各自为战 |

### B层：混沌入侵者

| 势力 | 信奉神 | 战术风格 | 核心区域 |
|:-----|:-------|:---------|:---------|
| **血斧教团** | 恐虐 | 正面强攻，血祭招魔 | 剑湾南部战线 |
| **万变之塔** | 奸奇 | 渗透、阴谋、知识交换 | 深水城法师界 |
| **慈父怀抱** | 纳垢 | 瘟疫扩散、尸体转化 | 银月城周边 |
| **欢愉姐妹会** | 色孽 | 腐蚀贵族、收集灵魂 | 博德之门上流社会 |

**混沌四神在费伦的特殊协议**：四神之间互相敌对，但在费伦达成了罕见的"狩猎协议"——谁能先得到灰骑士种子，谁就获得种子中的弑神之力。因此四神在互相竞争的同时，也不会相互毁灭对方的教团（至少不会公开做）。

### C层：深渊/外神

| 势力 | 类型 | 位置 | 当前状态 |
|:-----|:-----|:-----|:---------|
| **恶魔领主议会** | 深渊统治层 | 深渊 1-400 层 | 观望。部分领主想趁费伦混乱扩大地盘，部分在恐惧外神苏醒 |
| **灵吸怪殖民地** | 异界殖民者 | 星界 | 至上真神的前哨。灵吸怪们在等待一个"大吞噬时刻" |
| **外神沉睡体** | 不可名状 | 深渊 666 层 | 封印开始龟裂。外神在梦中开始影响深渊下层的心灵 |

## 势力关系

```
                ┌──────────┐
                │ 竖琴手   │ ←── 对抗 ──→ 混沌四教团
                └────┬─────┘
                     │ 情报交易
                     ▼
                ┌──────────┐
            ┌──→│散塔林会  │←── 黑市交易 ──→ 混沌四教团
            │   └──────────┘
            │       │
    ┌───────┴┐      │ 军火供应      ┌──────────┐
    │博德之门│      └──────────────→│恶魔领主  │←── 恐惧 ──→ 外神
    └────────┘                      └──────────┘
         │ 渗透                          ↑
         ▼                           ┌──────────┐
    ┌──────────┐    ┌──────────┐      │灵吸怪    │
    │欢愉姐妹会│──→ │深水城   │      │殖民地    │
    └──────────┘    └──────────┘      └──────────┘
                        │ 渗透            ↑
                        ▼                 │ 蝌蚪寄生
                   ┌──────────┐           │
                   │万变之塔  │───────────┘
                   └──────────┘
```

**关键关系**：
1. 竖琴手是唯一意识到"四神入侵只是表层威胁"的势力——他们在寻找关于外神的古老文献
2. 散塔林会在对混沌教团做两面交易——这为江轩提供了信息源和风险
3. 秩序诸神圣者大部分在自保，但也有人意识到灰骑士种子可能是对抗混沌的关键

## [江轩] 标签

- 初始无势力归属 → 一个在夹缝中求生的个体
- 竖琴手是他最可能的早期盟友（理念相近）
- 混沌四教团是他永恒的第一敌人
- 散塔林会和灵吸怪是灰色地带——可以从他们那里获取资源，但不能信任
