> 门禁：❌ 一条不通过 → 退回 Phase 1

## 检验项目

### ✅ 1. 位面架构 ↔ 力量体系

**检查**：位面架构的力量来源、访问方式是否与力量体系中的等级/突破条件一致？

**结论**：✅ 通过
- Lv1-9（凡人期）在主物质位面活动，不上位面风险
- Lv10-14（传奇期）可以进入深渊上层/星界 → 对应力量体系"理解位面"
- Lv15+（半神期）涉及领域碎片 → 对应位面架构中的"深渊递降"
- 亚空间灵能风险等级与力量体系中的混沌腐蚀逻辑一致

### ✅ 2. 历史驱动力 ↔ 势力格局

**检查**：核心矛盾是否有对应的势力体现？混沌入侵的时间线与各势力的状态是否匹配？

**结论**：✅ 通过
- 亚空间大撕裂（30天前）→ 混沌四教团在费伦建立据点
- 神祇失语（15天前）→ 诸神圣者流浪，中立势力失去信仰依靠
- 四神狩猎协议 → 各教团互相竞争但不互灭，逻辑自洽

### ✅ 3. 物种天赋 ↔ 力量体系

**检查**：种族/怪物的威胁等级是否在力量体系的框架内？

**结论**：✅ 通过
- 巴洛炎魔 = ★★★★★ ≈ 微弱神力级，符合DND传统定位
- 旧日支配者化身 = ★★★★★★ ≈ 中等神力级之上，不可正面对抗
- 灵吸怪蝌蚪 → 半神期以下/没有防护者→危险，符合力量体系

### ✅ 4. 资源物品 ↔ 力量体系

**检查**：修炼资源的稀有度是否与力量等级进阶的节奏匹配？

**结论**：✅ 通过
- 神性火花（★★★★★）→ 每一枚都是一个重大剧情节点，支撑半神期突破
- 信仰之力（★★★☆）→ 封神后持续获取，符合DND神祇设定
- 混沌神器碎片（★★★★）→ 仅在高阶战斗中掉落，符合力量体系的双线设计

### ✅ 5. 势力格局 ↔ 历史驱动力

**检查**：各势力的当前状态是否与历史事件的时间线一致？

**结论**：✅ 通过
- 竖琴手"发现混沌真相"→ 在亚空间大撕裂（30天前）后30天内发现问题合理
- 欢乐姐妹会"渗透博德之门上层"→ 7天前四神信使降临后加速渗透合理
- 秩序神圣者"各自为战"→ 神祇失语15天，部分已经开始寻找自己的新角色

### ✅ 6. 主基调一致性

**检查**：六篇的基调是否一致？有没有一篇显得"太轻松"或"太压抑"？

**结论**：✅ 通过
- 世界蓝图：暗黑生存 > 史诗冒险 > 悬疑惊悚
- 力量代价：每级晋升伴随混沌风险
- 历史：双重天灾+外神暗涌，没有救赎色彩
- 物种：混沌阵营和克苏鲁阵营占据了大量篇幅，费伦本土势力处于防守状态

### ✅ 7. [主角] 标签链

**检查**：六篇中所有的 [江轩] 标签是否一致？有没有冲突或矛盾？

**结论**：✅ 通过
- 01：种子休眠 + 四神猎物 → 世界蓝图中的定位
- 02：双线并行设计 → 力量体系中的成长机制
- 03：被选中者 → 历史定位
- 04：唯一种子携带者 → 物种维度独特性
- 05：无势力归属 → 初始状态
- 06：零资产 → 经济状态
- ⚠️ **注意**：主角标签不矛盾，但部分标签偏"被动"（猎物/被选中），建议在剧情推进中有意识转向"主动"（选择者/决定者）
