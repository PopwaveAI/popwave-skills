# 瑟尔镇（Thael）— 卷1起始地点

> 对标原作：琥珀城（Amber City）
> 地点定位：剑湾边缘的矿业/贸易边境小镇，卷1主角穿越点
> 当前状态：已被混沌屠城，沦为空墟

---

## 一、概况

| 字段 | 内容 |
|:-----|:------|
| **全名** | 瑟尔镇（Thael）|
| **别名** | "灰石镇"（因周边灰石矿得名） |
| **位置** | 剑湾南部，距博德之门约7天马程 |
| **建立** | 约180年前，由北地矿工和退役佣兵联合建立 |
| **人口** | 约1,200人（屠城前） |
| **统治者** | 民选镇长 + 镇议会（5名议员，由矿主/商会/神殿代表组成） |
| **经济命脉** | 灰石矿出口（建材/磨石）+ 驿站补给（博德之门-南方商路中间站） |
| **卫兵实力** | 常驻卫兵约30人，平均Lv3-5战士/武者；紧急时可征召民兵约80人 |

### 1. 市场广场 — 镇中心

| 要素 | 描述 |
|:-----|:------|
| **灰石喷泉** | 广场中央的旧矿车轮改造喷泉，是全镇集会地标 |
| **"疲惫矿工"酒馆** | 全镇唯一酒馆，矿工/佣兵/冒险者聚集地。老板"老汤姆"，退役矿工，一条木腿 |
| **告示板** | 冒险者公会任务/镇议会公告/通缉令 |
| **周二集市** | 周边农场和过路商队在此交易 |

### 2. 商业街

| 要素 | 描述 |
|:-----|:------|
| **铁匠铺"硬锤"** | 矮人铁匠杜林经营，主要修农具/矿镐，也能打普通刀剑 |
| **杂货店"塞满的口袋"** | 半身人巴金斯经营，卖干粮/绳索/油灯/低级药水 |
| **驿站"灰马"** | 通往博德之门的定期马车，每周两班 |
| **旅店"旅行者休息"** | 比酒馆楼上干净的住宿，商队常客 |

### 3. 矿工区

| 要素 | 描述 |
|:-----|:------|
| 木屋和石棚混建，道路泥泞，无铺装 |
| 居民以矿工和家属为主，部分失业矿工转做佣兵 |
| 混沌裂缝最先在此区域撕裂——矿工挖穿了不该挖穿的地层 |

### 4. 神殿区

| 要素 | 描述 |
|:-----|:------|
| **晨曦教堂（已失效）** | 供奉晨曦之主的小教堂，一名老牧师主持。动荡之年后圣光熄灭，牧师已殉道 |
| 教堂下方有地下墓穴——混沌屠城时曾是最后的避难所 |

### 5. 镇公所

| 要素 | 描述 |
|:-----|:------|
| 双层石楼，一楼办公/二楼议会 |
| 档案室存有全镇户籍和地契——江轩可能在此找到线索 |

## 三、当地势力（屠城前）

| 势力 | 影响力 | 说明 |
|:-----|:-------|:------|
| **镇长·赫克托** | 镇行政 | 前佣兵队长，务实行事。屠城时组织平民疏散，生死不明 |
| **灰石矿业公会** | 经济命脉 | 控制全镇80%就业。矿主哈罗德·格雷在裂缝撕裂后失踪 |
| **晨曦教堂** | 精神引导 | 老牧师已殉道。动荡之年后教堂无法施展神术，信徒大量流失 |
| **散塔林会线人** | 灰色 | 一名伪装成杂货商人的散塔林会底层线人——提供了混沌裂缝的最早报告但被忽略 |

## 四、毁灭经过（故事前35天）

| 时间线 | 事件 |
|:-------|:------|
| **D-35** | 灰石矿井深度挖掘时触碰古代地脉，引发小规模地震 |
| **D-30** | 矿工报告"闻到硫磺味、听到低语"，矿井封闭 |
| **D-20** | 亚空间裂缝在矿井深处撕裂，混沌能量泄露。首批混沌狂战士出现 |
| **D-15** | 裂缝扩大，血斧教团的战斗祭司从裂缝中带队涌入 |
| **D-7** | 镇长组织疏散，但混沌教团已经封锁主要出口 |
| **D-0** | 全城陷落。镇长组织最后一批平民从地下墓穴撤离 |

**江轩穿越的时间点**：混沌屠城的**最后一夜**。血斧教团正在清理残余、在广场搭建献祭法阵以扩大裂缝。

## 五、与主线的关联

| 关联 | 描述 |
|:-----|:------|
| **江轩穿越锚点** | 在排水沟中苏醒，获取灰骑士种子初次感知 |
| **混沌首次亮相** | 血斧教团屠城为读者展示混沌四神的威胁实感 |
| **亚空间裂缝种子** | 瑟尔镇的裂缝是第一波17处裂缝中的一处，后续会不断扩大 |
| **信息起点** | 江轩从废墟中获得的碎片信息拼接出第一张"世界发生了什么"地图 |
| **地下墓穴** | 逃生的关键路线，里面可能残留晨曦教堂的古老守护 |
