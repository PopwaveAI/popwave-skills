## 一、原作核心架构（继承规律）

| 架构层 | 原文核心 | 继承方式 |
|:-------|:---------|:---------|
| **时代背景** | 动荡之年：诸神失位，圣者降临，力量真空 | ✅ 直接继承，叠加混沌入侵 |
| **力量体系** | 神性火花→神格等级→信仰之力→神职领域 | ✅ 继承，增加"灵能/混沌腐蚀"并行条 |
| **深渊设定** | 无尽深渊，每层一个恶魔领主，深渊意志 | ✅ 继承，底层追加"外神监牢" |
| **DND底层** | 法术位、职业体系、阵营、位面架构 | ✅ 继承，所有魔法增加亚空间风险 |
| **封神路径** | 传奇→半神→微弱神力→中等神力 | ✅ 继承，每级突破都要面对混沌诱惑 |
| **阴谋政治** | 蜘蛛神后/恶魔领主/费伦城邦的权力博弈 | ✅ 继承，层叠加混沌教派 |

## 二、M1/M2 转译

### M1：继承规律

| 规律 | 转译方式 |
|:-----|:---------|
| 动荡创造机会 | 双重入侵让秩序崩坏更快→普通人崛起窗口更大 |
| 神性=权柄+诅咒 | 封神者会被神力污染→混沌/外神更容易盯上你 |
| 深渊=无尽阶梯 | 往下走力量越强、代价越大→通向外神监牢 |
| 信仰=货币 | 信仰之力还能用来锻造对抗混沌的武器/护甲 |

### M2：不继承事件

| 原事件 | 不继承原因 |
|:-------|:-----------|
| 索伦的穿越时间线 | 主角改为江轩，起点和命运不同 |
| 索伦 = 薇薇安的契约关系 | 江轩的身份决定他与蜘蛛神后的博弈不同 |
| 原作具体封神顺序 | 江轩拥有灰骑士种子，封神路径会变形 |
| 洛山达等神祇的圣者事件 | 四神入侵让神祇动向完全不同 |
