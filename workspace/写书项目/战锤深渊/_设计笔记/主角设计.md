# 主角设计 — 江轩

> Phase 0.4 产出 | 基于故事引擎方向锁定
> 碰撞点：DND封神之路 × 灰骑士原体种子 × 混沌四神 × 克苏鲁外神

---

## 一、四维展开

### 1. 身份

**表层身份**：费伦某个边境城镇的阴沟里醒来的无名穿越者，没有身份证件，没有背景靠山，一无所有。

**深层身份**：灵魂中携带了一颗**灰骑士原体基因种子**的穿越者。这颗种子来自战锤40k宇宙，在它还没来得及被植入、成长为真正的原体之前，就跟江轩的灵魂一起穿越到了费伦。它现在处于**休眠态**——像一枚未受精的卵，但江轩的每一次成长/突破/接触神性力量，都会刺激它一分。

**社会身份（发展中的）**：从流浪者 → 冒险者 → 灰骑士的胚芽 → 最终可能成为什么，取决于他的选择。

### 2. 驱力

| 层级 | 驱力 | 强度 | 说明 |
|:-----|:-----|:----:|:-----|
| **表层** | 活下去 | ★★★ | 暴雨夜的阴沟里睁开眼，第一反应就是生存。这个世界太危险了——混沌教团在屠城、诸神失联、原来的秩序在崩坏 |
| **中层** | 找到"为什么是我" | ★★★ | 灰骑士种子为什么会跟他一起穿？这粒种子是谁放的？他只是被随机选中的，还是穿越本身是被刻意安排的？ |
| **深层** | 证明"我可以不一样" | ★★★☆ | 当他发现走上封神之路=一步步靠近混沌/堕落时，真正的驱力浮现了：他要证明一条路——**用混沌的对立面驾驭神性，而不是被神性吞没** |

### 3. 缺陷

| # | 缺陷 | 叙事作用 |
|:-:|:-----|:---------|
| D1 | **肉身凡人** | 他有灰骑士的灵魂/种子，但身体是普通穿越者的。这意味着在种子完全觉醒之前，他可能死在任何一个战斗中。种子不能保护不活的宿主 |
| D2 | **知识诅咒** | 他太了解混沌四神是什么了（穿越者的先知优势）——越了解越恐惧，越恐惧越容易被亚空间听到。奸奇尤其喜欢恐惧的灵魂 |
| D3 | **力量加速异化** | 每次使用灰骑士之力/接触神性火花，种子就成熟一分。但成熟不是无代价的——他会越来越"非人"，血管里流淌银灰色光芒、瞳孔变灰、看见凡人看不见的亚空间低语 |
| D4 | **情感连接的脆弱** | 在这个世界里建立羁绊（队友/盟友/爱人）是活下去的动力，但也是混沌腐蚀的最佳入口——四神会利用他在意的人来腐化他。纳垢最喜欢"为了救他们，你只需要接受一点点……" |

### 4. 唯一性（四级评估）

#### L1 核心差异
江轩是费伦唯一一个同时具备以下三要素的存在：
- **灰骑士原体基因种子**（对抗混沌的天命之器）
- **穿越者知识**（知道DND规则、知道混沌四神、知道外神的传说）
- **封神潜力**（动荡之年里人人都有机会封神，但他的种子给了他独特的起点）

这三者的组合在费伦没有第二个人。他不是"另一个索伦"——他的起点约束和可能性完全不同。

#### L2 约束条件
- 不能主动拥抱混沌力量（否则种子会反噬，灵魂会被灰骑士本质烧毁）
- 不能依赖灰骑士种子解决所有问题（种子会在他准备好之前提前觉醒，导致他失去人类形态）
- 不能走"快封神"路线（神性成长越快，混沌注视越强——他必须在速度和隐蔽之间平衡）

#### L3 不可替代性
混沌四神为什么盯上他？

灰骑士的基因种子是战锤40k宇宙中最纯粹的反混沌武器。在费伦这个被亚空间裂缝撕裂的世界里，一颗活着的灰骑士原体种子意味着：**有人可以在混沌的领域中创造一块"净土"**——一个四神无法触碰的领域。

四神谁都想得到它：
- **奸奇**想解析它 → 制造反灰骑士武器
- **恐虐**想摧毁它 → 最强的敌人值得最强的杀戮
- **纳垢**想感染它 → 让最纯粹的对抗之力变成最毒的瘟疫
- **色孽**想腐蚀它 → 让最坚定的意志在最极致的欢愉中崩塌

江轩的存在本身，就是四神在费伦的第一优先级目标。

#### L4 叙事驱动
- **短程**：江轩活下去 → 发现混沌的渗透 → 第一次使用灰骑士之力保命
- **中程**：在封神路上走 → 卷入四神的阴谋 → 灰骑士种子第一次觉醒
- **长程**：面对"我到底是江轩，还是灰骑士的容器"的哲学拷问 → 在封神与自我毁灭之间做选择
- **终局**：当外神苏醒，当四神的化身降临——江轩必须决定：是成为费伦的新神，还是用灰骑士的种子把自己变成一堵墙，挡在混沌面前

## 二、主角冲突树

```
                    ┌── 表层：如何在四神爪牙下活下去？
                    │
        生存冲突 ───┼── 中层：如何在混乱中收拢盟友/资源？
                    │
                    └── 深层：如何在不动用混沌力量的前提下变强？

                    ┌── 身体：种子觉醒时有多痛苦？
                    │
  内部冲突 ─────────┼── 心理：我在变成人还是武器？
                    │
                    └── 灵魂：如果封神的终点是变成"另一个混沌存在"怎么办？

                    ┌── 四神各自有腐化计划
                    │
  外部冲突 ─────────┼── 费伦诸神的态度（利用？敌视？无视？）
                    │
                    ├── 深渊恶魔领主把他当棋子
                    │
                    └── 外神的苏醒不以任何人的意志为转移
```
