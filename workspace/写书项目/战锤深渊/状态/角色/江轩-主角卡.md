# 江轩 · Lv4 主角角色卡

> 级别判定：Lv4 主角（贯穿全文的灵魂，承载全部情感投射）
> 追溯自：故事引擎 + 主角设计.md + L1 设定层
> 身份：灰骑士原体种子携带者 · 穿越者 · 封神候选

---

## 设计目的

承载故事核心主题——"在混沌横行的宇宙中，力量不是恩赐而是赌注"的具象化身。读者通过江轩体验：被动穿越者的绝望→灰色抉择的挣扎→最终成为"选择者"的蜕变。他的灰骑士种子不是外挂，而是以人性为燃料的武器。

## 叙事能力

- 驱动全篇情节走向——他的成长线 = 故事主线
- 承载读者情感投射：从"他好惨"到"他好强但代价好大"到"他还是人吗"
- 所有关键转折由他的决策触发（用/不用种子力量、信/不信某个盟友、封/不封神）
- 读者最终的认同锚点：在混沌面前，选择不做怪物的人

## 三、心理

### 核心性格特质（3 条）

**特质1：理性计算**
- **表现**：面对突发危险时大脑先过一遍概率，再行动
- **边界**：当涉及他在意的人受伤时，理性立刻崩溃，转为极端保护行动
- **盲点**：过度依赖理性分析，忽视直觉（然而他的直觉往往准确——灰骑士种子的馈赠）

**特质2：被迫的赌徒**
- **表现**：知道力量有代价，但还是会赌最后一次——"就这一次"
- **边界**：赌到种子觉醒度≥60%时会暂退（自我保存本能）
- **盲点**：以为每次赌注都能控制上限

**特质3：深埋的共情**
- **表现**：嘴上说"各管各的"，但行动上会拼死去救队友
- **边界**：不能接受主动伤害无辜者（哪怕为了更大的利益）
- **盲点**：共情是他最大的弱点——四神会利用这一点

### 决策逻辑

```
日常决策倾向：计算优先，延迟满足
危机决策倾向：平时理性，危机时押上一切（灰骑士种子的冲动影响）
决策阈值：当他意识到"不做选择也会有后果"时，从被动转为主动
```

### 核心缺陷

```
缺陷：对力量的恐惧 vs 对力量的渴望之间的撕裂
对谁/什么事最致命？他自己。每次突破都在加速种子觉醒，觉醒越高越不"人"
整个故事中改善了多少？从恐惧种子 → 接受种子 → 学会控制种子 → 最终决定种子的命运（而非种子决定他的）
```

## 四、驱力（四层完整）

| 层级 | 内容 | 故事中何时显露 |
|:-----|:------|:--------------|
| **表层目标** | 活下去，在这个见鬼的世界活过明天 | 开局即显 |
| **深层需要** | 证明"被混沌盯上的人也能走出自己的路，不被腐蚀也不变成怪物" | 卷2中段开始浮出 |
| **黑暗冲动** | 在极端危机下，主动拥抱灰骑士种子的力量——哪怕这意味着加速非人化 | 卷3压力阈值触发 |
| **恐惧** | 最怕自己最终变成一个"打着对抗混沌旗号的怪物"——成为另一个混沌存在 | 全篇伏笔 |

### 驱力冲突

表层目标（活下去）与深层需要（证明路是对的）不矛盾，但黑暗冲动与恐惧之间形成核心张力：**他想用种子力量保护所爱的人，但每次使用都在缩短"他还是江轩"的时间。**

## 五、能力

### 当前能力评估

| 维度 | 描述 | 定位 |
|:-----|:------|:-----|
| **优势项** | 灰骑士种子带来的混沌感知——能"闻到"亚空间的危险浓度 | 逃生/预警能力 |
| **中等项** | 现代人的逻辑思维和分析能力——比费伦人更擅长系统性解决问题 | 智力输出 |
| **弱项** | 这具身体没经过任何战斗训练，体力/格斗/魔法均为零 | 必须快速补足 |

### 成长路径

```
起点能力档位：零战力，只有被动感知（种子休眠 0%）
第一阶段目标（获得第一枚神性火花）：Lv1-5 冒险者级
第二阶段目标（种子半觉醒 50%）：Lv10+ 传奇级
第三阶段目标（种子完全觉醒 100%）：半神级+
终点能力档位（全书结束时）：中等神力 + 种子掌控态（至少 ≈ 四神化身对抗级）
瓶颈预测：
  ① 第一枚火花获取——需要击杀一个有分量的目标
  ② 半觉醒——必须被混沌四神之一亲自盯上，在濒死中激活
  ③ 完全觉醒——需选择：是人，还是武器
```

## 六、演化

### 弧线阶段（4 阶段）

```
阶段 1：「求生者」
  起止章节：卷1
  触发事件：穿越 → 混沌屠城 → 第一次被追猎
  前心态：本能求生，被动反应
  后心态：意识到逃不是出路
  关键决策：选择接受第一枚神性火花（而不是继续逃跑）
  表象变化 vs 本质变化：从弱↔稍微不弱，但心态从"为什么是我"转向"既然是我了，那就走到底"

阶段 2：「深渊使徒」
  起止章节：卷2
  触发事件：第一次下深渊，面对恶魔领主级对手
  前心态：我在这个世界的目标是什么？
  后心态：我在这个世界的角色是什么？
  关键决策：是当秩序的棋子，还是当自己的棋手？
  表象变化 vs 本质变化：从流浪者↔有组织的冒险者；从被动接受天命↔主动探索天命来源

阶段 3：「灰烬觉醒」
  起止章节：卷3
  触发事件：灰骑士种子第一次主动绽放——可能在保护队友的濒死时刻
  前心态：控制种子 = 控制自己
  后心态：控制种子 ≠ 控制自己。种子有自己的意志
  关键决策：继续使用种子力量（加速非人化）还是封印它（失去对抗混沌的核心武器）
  表象变化 vs 本质变化：从用种子的力量↔种子开始用他

阶段 4：「弑神抉择」
  起止章节：卷4
  触发事件：外神开始苏醒 + 四神化身降临
  前心态：我要封神来对抗这一切
  后心态：封神不是答案——成为"混沌的终结者"才是
  关键决策：选择不封神，而是用灰骑士种子把自己变成"一堵墙"
  表象变化 vs 本质变化：从追求力量↔追求代价最小化的解决方案
```

### 弧线类型

```
□ 成长弧  — 从不足到完整（但这里的"完整"不是快乐结局）
□ 转变弧  — 从被动受害者 → 主动选择者
■ 揭露弧  — 本质不变（始终是那个不想当怪物的人），但读者逐渐认识他有多大的代价在扛
```

### 不可逆点清单

```
事件 1（卷1结尾）：拿到第一枚神性火花
  — 从此他被四神正式标注为"目标"，回不去普通冒险者生活
  — 心态：从"也许就是个普通穿越" → "我走上的路没有回头"

事件 2（卷2中段）：在深渊中第一次主动使用种子力量救人
  — 他选择了"使用代价换取他人存活"
  — 旧驱力（只为自己活）失效

事件 3（卷3高潮）：种子半觉醒导致他误伤/误杀队友（或差点）
  — 最黑暗的时刻
  — 被迫面对："我在变成什么？"
```

## 七、网络

| 关系对象 | 关系类型 | 角色给对方什么 | 对方给角色什么 | 张力来源 |
|:---------|:---------|:--------------|:--------------|:---------|
| 蜘蛛神后·萝丝 | 敌/友不明 | 灰骑士种子是对抗混沌的底牌 | 黑色情报、深渊入口 | 萝丝想要种子，江轩不想成为棋子 |
| 竖琴手联络人 | 早期盟友 | 先发制人的混沌预警能力 | 藏身处、情报、治疗 | 信任问题——竖琴手是否知道种子的全貌？ |
| 灵吸怪至上真神 | 终极敌人之一 | 无（他们要他的灵魂） | 蝌蚪力量（诱惑） | 至上真神是四神的工具还是独立势力？ |

### 关系演化摘要

```
江轩 ↔ 萝丝：
  起点：敌不动我不动——萝丝在观察种子的真实价值
  第一个转折点：江轩第一次进入深渊时萝丝"恰巧"出现
  终点：合作/对抗取决于江轩是否愿意接受萝丝的条件

江轩 ↔ 灰骑士种子：
  起点：一个陌生的、发烫的符文
  第一个转折点：第一次主动调用它的力量
  终点：他要么驯服它，要么被它取代
```
