# 蜘蛛神后·萝丝 · Lv3 重要配角卡

> 级别判定：Lv3 重要配角（跨幕存在，独立动机）
> 追溯自：故事引擎 + L1-05 势力格局
> 身份：黑暗精灵神系主神 · 蜘蛛神后 · 坠落凡间的圣者

---

## 设计目的

承载独立动机的灰色角色——她不是纯粹的盟友也不是纯粹的敌人。她代表"费伦旧神在动荡之年中对混沌入侵的复杂态度"，在江轩的成长线上扮演亦师亦敌的催化角色。

## 叙事能力

- 与江轩产生深度互动（宿敌/灰色导师/交易对象）
- 承载读者对"神性在混沌面前有多脆弱"的次级情感投射
- 为卷2——深渊进入提供关键推动力（她掌握深渊入口的秘密）
- 信息管道：她比费伦任何人都更早感知到外神的苏醒

## 四、叙事功能

```
□ 情感锚点 — 承载读者对"旧神试图在新灾难中生存"的复杂情感（敬畏/警惕/同情）
■ 冲突放大 — 她与江轩的交易/博弈推动剧情转折
■ 信息管道 — 向江轩/读者释放关于深渊和外神的古老知识
□ 镜像/对照 — 与江轩的"被动封神"形成对照——一个主动想封神的凡人 vs 一个已经被封神但现在失去神力的神
■ 催化剂 — 推动江轩进入深渊（她掌握入口）
□ 场面注入 — 优雅与恐怖并存的风格
```

---

## 五、弧线

### 弧线类型
```
□ 成长弧  — 从不足到完整
□ 堕落弧  — 从光明到黑暗  
■ 转变弧  — 从"费伦诸神之一"到"混沌时代的幸存者"
□ 揭露弧  — 本质不变，读者逐渐认识其真实面貌
```

### 起终点

```
弧线类型：转变弧
起点心态：作为坠落的旧神，她仍以神的高姿态行事——轻视混沌四神的威胁，认为这只是"动荡之年的常规混乱"
终点心态（或预期终点）：她被迫承认——费伦诸神的时代已经结束，混沌和外神的时代开始了。她的身份需要从"神"转换为"幸存者"
触发事件：混沌四神的信使在费伦展现出比她全盛期更强的力量，她的蜘蛛领域被纳垢瘟疫渗透
```

## 六、登场场景表

| 卷/幕 | 事件 | 该角色的作用 |
|:------|:-----|:------------|
| 卷1后期 | 江轩获得第一枚神性火花后，萝丝第一次接触他 | 信息管道：告诉江轩"你被盯上了"；催化剂：引导他向深渊走 |

| 卷3 | 萝丝的领域被混沌渗透 | 弧线转折：从高傲的神→被迫求援的幸存者 |
| 卷4 | 外神苏醒前，萝丝的选择 | 终局决策：她站在哪边？ |
