## 三、背景

### 时间范围

```
起始: 动荡之年 + 混沌入侵爆发第 35 天，费伦秋季雨夜
结束: 约 45 天后，冬季初
节奏提示: 前期紧凑（每天都是生死），中期有喘息窗口，后期加速压向高潮
```

### 地理范围

| 区域 | 类型 | 叙事功能 | 出现于 |
|:-----|:-----|:---------|:------|
| **边境城镇·废墟** | 残破城镇 | 江轩穿越点，混沌屠城现场，地狱开场 | CH1-3 |
| **剑湾荒野** | 野外 | 逃亡路线，混沌追杀，第一次接触超凡力量 | CH4-8 |
| **博德之门周边** | 城邦外围 | 信息集散地，接触势力，寻找神性火花线索 | CH9-18 |
| **古代精灵遗迹** | 地下城 | 第一枚神性火花的所在地 | CH19-25 |
| **幽暗地域入口** | 地下通道 | 卷末定位——通往深渊的大门 | CH26-30 |

| 维度 | 状态 |
|:-----|:-----|

## 四、剧情线

### 主线1 — 世界危机：混沌渗透

| 字段 | 值 |
|:-----|:----|
| 一句话定义 | 混沌四神在费伦的渗透从暗转明，亚空间裂缝不断扩大 |
| 本卷推进 | 边境城镇被混沌屠城 → 剑湾多处出现裂缝 → 博德之门内教团渗透暴露 |
| 预期释放节奏 | 幕1首章释放（屠城现场）；幕2中段释放（博德渗透暴露）；幕3高潮释放（裂缝扩大至临界） |
| 活跃幕段 | 幕1 / 幕2 / 幕3 |

主线1-子线:
- ① **血斧追杀** — 载体: 猎杀队长 — 回收预期: 卷末CH25反杀，但引来恐虐的直接注视
- ② **亚空间裂缝扩大** — 载体: 环境叙事 — 回收预期: CH30裂缝已从17处增至40+处

### 主线2 — 主角成长：从零到一

| 字段 | 值 |
|:-----|:----|
| 一句话定义 | 江轩从零战力穿越者蜕变为拥有第一枚神性火花的超凡者 |
| 本卷推进 | Lv0→Lv15，种子0%→10%，心态从"为什么是我"→"既然是我了" |
| 预期释放节奏 | 幕1: 首次战斗/scars；幕2: 首个职业能力觉醒；幕3: 第一枚火花获取（大释放） |
| 活跃幕段 | 幕1 / 幕2 / 幕3 |
| 关键成长节点 | 幕1中段（第一次下决心战斗）；幕2末（接受命运）；幕3高潮（火花点燃·种子首次应激反应） |

### 主线3 — 主角行动：求生寻光

| 字段 | 值 |
|:-----|:----|
| 一句话定义 | 江轩在绝境中求生→找到第一个据点→主动寻找神性火花 |
| 本卷推进 | 阴沟逃生→荒野流浪→找到竖琴手→前往遗迹→获得火花 |
| 预期释放节奏 | 全卷渐进——幕1被动→幕2主动→幕3决战 |
| 活跃幕段 | 幕1 / 幕2 / 幕3 |
