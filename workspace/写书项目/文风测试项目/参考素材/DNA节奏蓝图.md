# DNA节奏蓝图

> 从遮天 `combat_epic` + 深渊主宰 `combat_mid_boss` 提取，供A法/B法使用。C法不提供此文件。

---

## 一、遮天 combat_epic 节奏提取

### 核心原则：不写动作，写世界规则的破裂

> "一千五百里外，众多观战者心中惊骇……此时，他们元神欲裂，如刀刮骨，每一寸肌血都要剥落了一样……而这一切，都仅仅是一种波动，并非双方出手。"

**提取**：
- 超高阶战斗不描写"谁打了谁"，转而描写"世界框架被什么力量改变了"
- 手法1：天地异象（日月无光、大道合鸣）
- 手法2：远距离旁观者感受（千里之外如刀刮骨、元神欲裂）
- 手法3：空间破碎描述（虚空裂谷、混沌涌现）
- 战斗描述从"动作"转化为"世界规则被改变"

### 情绪节奏：压→爆→收 三拍

- **压**：被压制/被困/被追杀，累计憋屈感。持续0.5-1章
- **爆**：反杀/升级/揭晓，憋屈一次性释放。0.5-1章
- **收**：简短收束（叹气/感慨/望远方），绝不过度抒情。不超过半页

### 句式模式：短句推进，长句渲染

- 动作/战斗：短句（5-12字），句句有动词，快如鼓点
- 世界观/氛围：长句（20-40字），成语+四字结构，文言厚重感
- 象声词独立成段（轰/锵/噗/咔嚓），作为节奏停顿节点
- 句末多用感叹号和省略号

**原文范例**：
```
"砰！"
叶凡冲天而起，挥动金色的拳头，一拳重重砸了下来。
"咔嚓！"
八步踏过，姬仁浑身骨骼碎裂数十处，形体近乎变形。
```
→ 短句(动作+音效) 与 长句(画面定性) 交替

## 二、深渊主宰 combat_mid_boss 节奏提取

### 核心原则：攻→防→反弹→数据反馈 循环

> "铛！一柄战矛从天而降，风暴领主矫健的身影落地……铛铛铛！风暴领主将战矛舞得狂风肆虐，不过每一次格挡后她便后退一步……"

**提取**：
- boss战采用"攻→防→反弹→反馈"的节奏循环
- 每次碰撞后必有结果反馈（后退/颤抖/武器缺口/皮肤焦黑）
- 终结技/大招作为战斗高潮的标志性收束
- 战前通过旁白/战术说明建立boss威胁等级

### 感官序列：嗅觉→听觉→触觉→视觉

- **嗅觉优先**：气味定位场景情感基调
- **听觉填充**：环境声音建立距离感
- **触觉连接**：温度/质地建立身体连接
- **视觉收束**：局部细节，不做全景扫描

### 段落呼吸

- **独句段**（1-2行）：情感爆发 / 动作转折 / 时间跳跃
- **短句链**（3-5行）：战斗正文，每段一个动作节奏
- **多句段**（8-15行）：设定说明 / 战斗准备
- 全书几乎无超过20行的段落

**原文范例**：
```
铛铛铛！
索伦手中的匕首划过妖异的轨迹，连续接住了对方的劈砍，沉声道："你们不该把主意打到薇薇安的身上。"
"所以只能送你们全部下地狱了！"
```
→ 拟声词独立成段 → 短句链推进 → 对话穿插

### 终结技模式

> "风暴领主发出愤怒的咆哮声，猛地跳跃到了上百米的高度……战矛好似一道闪电划过天际，速度已经快的超出人类的视野捕捉，只能看到天际划过的一道电光……"

**提取**：
- 终结技有蓄力过程（咆哮→跳跃→投掷）
- 速度超出感知能力时，只写"看到的结果"不写"动作过程"
- 终结后必有代价展示（"双手居然好似烧焦了一般"）

## 三、两DNA融合节奏蓝图

将遮天的"世界破裂感"与深渊主宰的"攻防循环"融合：

```
[环境感知] 嗅觉→听觉→触觉→视觉（深渊主宰感官序列）
    ↓
[威胁建立] 通过环境异常暗示力量等级（遮天手法3：空间破碎）
    ↓
[攻防循环] 攻→防→反馈→再攻（深渊主宰节奏循环）
    每轮循环用短句链推进，象声词独立成段
    ↓
[高潮爆发] 不写动作本身，写世界规则改变（遮天核心原则）
    旁观者感受 / 天地异象 / 空间破碎
    ↓
[代价展示] 终结后身体代价（深渊主宰终结技模式）
    ↓
[简短收束] 不超过半页，一叹一望（遮天收拍）
```

### 融合感官配置

| 阶段 | 主导感官 | 辅助感官 | 段落类型 |
|:-----|:---------|:---------|:---------|
| 环境感知 | 嗅觉 | 听觉 | 多句段(8-15行) |
| 威胁建立 | 视觉(局部) | 触觉(温度) | 短句链(3-5行) |
| 攻防循环 | 听觉(碰撞) | 触觉(冲击) | 短句链+独句段交替 |
| 高潮爆发 | 视觉(异象) | 听觉(寂静/轰鸣) | 长句+独句段 |
| 代价展示 | 触觉(疼痛) | 视觉(伤口) | 短句链 |
| 收束 | 视觉(远景) | — | 2-3段 |
