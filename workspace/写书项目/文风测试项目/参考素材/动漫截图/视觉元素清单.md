# Layer 1 视觉参考分析

## 源素材

海贼王第1122集「蜂巢岛拯救战」——卡普vs青雉，4帧关键画面截图。

截图文件位于 `frames/` 目录，分辨率 320×180，JPEG 0.4质量。

## 逐帧分析

### 帧1: frame_120s.jpg (2:00) — 冰水特效全景

| 维度 | 分析 |
|:-----|:-----|
| 构图类型 | 低角度近景，贴水面拍摄 |
| 主色调 | 青蓝(冰)、深蓝(水深)、白(高光) |
| 光效类型 | 锐角光纹切割画面——非柔光，而是几何线条状的光刃；多个圆形高光点散布 |
| 动态线 | 对角线构图，光纹从左上到右下形成视觉冲击方向 |
| 破坏形态 | 水面被切割成锐角碎片，非自然波纹而是暴力碎裂 |
| 色温感 | 冷端主导(>8000K)，高光处接近纯白(过曝) |
| 文字转写锚点 | 光不是"照亮"，是"切开"——光刃有锋利感，水不是"流动"而是"碎裂" |

**分镜功能**：环境威胁建立——展示敌方能力的覆盖范围和压迫感

### 帧2: frame_300s.jpg (5:00) — 角色侧脸特写

| 维度 | 分析 |
|:-----|:-----|
| 构图类型 | 侧脸半身特写，浅景深 |
| 主色调 | 暖色背景(黄/橙/棕沙漠) vs 冷色角色(粉发/蓝白领) |
| 角色状态 | 凝视画外左方——不是看敌人，是在思考/等待/挣扎 |
| 面部细节 | 脸上有疤痕/印记，暗示过往战斗伤痕 |
| 服装符号 | 白色制服+蓝色配饰——权威身份标识 |
| 情绪传递 | 不是恐惧，是"做了决定"的沉默——嘴唇紧闭，目光坚定 |
| 文字转写锚点 | 用疤痕讲故事而非旁白；用"看哪里"暗示内心——不看敌人=不畏惧 or 在保护什么 |

**分镜功能**：情感锚点——在暴力间隙插入角色内心，让观众/读者在乎结果

---

### 帧3: frame_420s.jpg (7:00) — 战后废墟倒地

| 维度 | 分析 |
|:-----|:-----|
| 构图类型 | 俯角全身镜头，角色居中倒地 |
| 主色调 | 紫色(烟尘/暮色) vs 棕色(废墟) vs 蓝(角色服装) |
| 破坏层级 | 点(角色身下碎石)→线(倒塌的建筑)→面(整片废墟)→体(远山+天空) |
| 血/伤 | 面部可见血迹，手臂张开——不是蜷缩，是力竭后的失控 |
| 环境细节 | 棕榈树、地中海式建筑残骸——通过标志性元素锚定地理坐标 |
| 烟尘处理 | 紫色而非灰色——非写实，用色温传递情绪(紫=不自然=超自然力量的余波) |
| 文字转写锚点 | 破坏要给"地质指纹"——棕榈树+白墙碎砖=这是港口城镇；紫色烟尘=这不是普通火灾 |

**分镜功能**：代价展示——胜利或失败的具象化，让力量有重量

---

### 帧4: frame_540s.jpg (9:00) — 背影远眺

| 维度 | 分析 |
|:-----|:-----|
| 构图类型 | 背影全身，三分法构图，人物居左三分之一 |
| 主色调 | 暖色大地(黄/棕) vs 冷色天空海洋(蓝)——冷暖对峙 |
| 服装符号 | 背后"正義"二字——不用旁白说身份，让字说话 |
| 视线引导 | 角色面朝右方大海/远方——不是回看废墟，是看向未来/目标 |
| 环境层次 | 前景悬崖(实)→中景岩层(虚)→远景海平线(更虚)——三段景深 |
| 情绪传递 | 孤独但不弱——背影+大衣+风=坚持者形象 |
| 文字转写锚点 | 用背后的字替代内心独白；用"面朝方向"暗示叙事走向——他不回头 |

**分镜功能**：收尾定格——战斗结束后的情感落地，一个画面定义"这值得吗"

## 分镜结构表

按时间排列4帧的分镜序列：

| 序号 | 时间 | 镜头类型 | 节奏功能 | 段落映射 |
|:-----|:-----|:--------|:---------|:---------|
| 1 | 2:00 | 低角度特效全景 | 建立压迫——展示敌方力量覆盖范围 | 长段(4句+)，慢镜头 |
| 2 | 5:00 | 侧脸特写 | 情感呼吸——在暴力中插入内心 | 独句段或2句短段 |
| 3 | 7:00 | 俯角废墟全身 | 代价展示——力量落地的后果 | 多段级联(8-15段) |
| 4 | 9:00 | 背影远景 | 收尾定格——情感落地 | 2-3段安静收束 |

### 分镜节奏提取

```
全景(慢/长) → 特写(快/短) → 全景后果(爆/胀) → 远景(收/静)
```

这映射到8拍节奏引擎的中间4拍：
```
[慢] → [快] → [爆→胀] → [静→收]
```
