# 三层架构SOP

## 执行流程

```
Layer 1（视觉参考层）
  输入：动漫截图/漫画分镜
  处理：构图分析 → 视觉元素提取 → 分镜序列识别
  输出：视觉元素清单 + 分镜结构表
    ↓
Layer 2（文风DNA层）
  输入：现有文风DNA场景卡
  处理：提取段落呼吸 → 感官序列 → 叙事者距离 → 情绪节拍
  输出：节奏蓝图 + 感官序列 + 情绪节拍
    ↓
Layer 3（渲染指令层）
  输入：Layer 1视觉元素 + Layer 2节奏蓝图 + 渲染技法
  处理：将视觉元素填入节奏骨架，用技法保证打击感
  输出：最终战斗正文
```

## 三层协作关系

| 层 | 提供什么 | 不提供什么 |
|:---|:--------|:----------|
| Layer 1 | 视觉词汇、分镜结构、构图类型 | 时间节奏、重量感、声音、情感 |
| Layer 2 | 段落呼吸、感官序列、情绪节拍 | 具体写什么内容 |
| Layer 3 | 转写技法、禁用词、强制规则 | 素材本身 |

## 执行步骤

### Step 1：视觉参考分析（Layer 1）

1. 逐帧分析截图：构图类型、主色调、光效类型、动态线、破坏形态
2. 提取分镜序列：按时间排列，标注每帧的节奏功能
3. 生成视觉词汇注入器：禁用空泛词，提供替代词汇

### Step 2：DNA节奏提取（Layer 2）

1. 选择匹配的DNA场景卡（如 combat_epic + combat_mid_boss）
2. 提取段落呼吸模式（独句段/短句链/多句段的分布）
3. 提取感官序列（如嗅觉→听觉→触觉→视觉）
4. 提取情绪节拍（如压→爆→收）
5. 融合多源DNA，生成统一节奏蓝图

### Step 3：渲染执行（Layer 3）

1. 按8拍节奏引擎搭建章节骨架
2. 将Layer 1视觉元素填入对应拍位
3. 按Layer 2节奏蓝图控制段落呼吸
4. 按8项渲染技法执行具体转写
5. 逐条检查约束条件

### Step 4：验收检查

- [ ] 8拍是否完整
- [ ] 每次撞击是否有≥3种感官
- [ ] 是否有级联破坏（点→线→面→体）
- [ ] 是否有≥1个安静时刻
- [ ] 是否回扣情感锚点
- [ ] 是否无禁用词
- [ ] 段落呼吸是否符合DNA蓝图
- [ ] 字数是否在3000-5000范围
