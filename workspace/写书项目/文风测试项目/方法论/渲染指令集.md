# 渲染指令集

> Layer 3 技法库。A法/B法/C法共用。

---

## 技法1：压缩-膨胀定律

- 撞击瞬间：≤1句，独立成段（压缩）
- 撞击后果：≥8段，多感官级联展开（膨胀）
- **禁止**在撞击瞬间用形容词修饰力量
- 撞击瞬间的句子里只能有动词和名词

```
✓ 拳落。
✗ 一拳威力惊人地砸了下去。
```

## 技法2：段落=镜头

- 短段（1-2句）= 快切/特写
- 长段（4句+）= 慢镜头/全景
- 空行 = 硬切
- **禁止**在同一段内切换视角
- 连续3个短段后必须接1个长段（呼吸节奏）

## 技法3：感官叠加

- 每次重大撞击至少覆盖3种感官
- 视觉+听觉+触觉/嗅觉/本体觉
- 写"消失"而非"出现"（冲击波抽空空气→写窒息感）
- 优先写"感受者"而非"产生者"

```
✓ 耳膜向内凹陷，眼底只剩白，舌根尝到铁锈味。
✗ 强大的冲击波向四周扩散。
```

## 技法4：级联破坏

- 破坏按"点→线→面→体"扩散
- 用参照物锚定规模（方圆X米/云层消失/从太空可见）
- 给破坏一个可视的"地质指纹"（玻璃化/液化/晶化）
- 每一层破坏至少1段，不允许跳级

```
点：拳落处的冰面向内凹陷
线：凹陷向外扩展成裂纹
面：裂纹连接，整片冰面塌陷
体：塌陷波及地下结构，地牢顶层暴露
```

## 技法5：反差蒙太奇

- 最强的一击，写最轻的动作
- 最大破坏，写最小细节（血珠在碎冰上砸出裂纹）
- 战斗中至少1个"安静时刻"（人物简短对话/微小动作）
- 安静时刻必须卡在两次爆发之间

## 技法6：色温替代形容词

- **禁止**用"绚丽/璀璨/耀眼/震撼/恐怖"
- 用物理概念：色温(K值)、光谱、折射、透镜效应、曝光过度
- 光的后果＞光本身（温度/阴影消失/颜色失真/视网膜灼伤）
- 暗也是一种光效——光被吞噬的暗

```
✓ 色温骤升至白炽，视网膜残留的影像持续了三秒。
✗ 绚丽的光芒照亮了整个战场。
```

## 技法7：8拍节奏引擎

```
慢(绝望铺垫 3-4段)
  → 快(硬切 1句独立成段)
    → 爆(撞击 1字/1句)
      → 胀(后果展开 8-15段，多感官级联)
        → 静(人物特写 2-3段)
          → 升(二次威胁+光效 3-5段)
            → 终(终极爆发，感官过载)
              → 收(安静收尾，情感落地)
```

- 每拍必须完整，不可跳拍
- "爆"和"胀"的篇幅比 = 1:8以上
- "静"拍不可省略——是"胀"和"升"之间的呼吸阀

## 技法8：力量为谁而用

- 每场战斗必须回答：这股力量为谁而用？代价是什么？
- 力量本身不产生张力，张力来自情感前提
- 必须有一个"在乎的人"在战斗中或战斗外
- 在战斗高潮前必须回扣情感锚点（1句即可）

```
✓ 拳落的时候，他想起小沉说"师父我冷"。
✗ 他爆发出全部力量，势不可挡。
```
