## 使用方式

- **查某个参考来源的完整分布** → 开对应 `参考域-*.md`，一章读完
- **查某章有哪些跨域参考** → 翻 `跨域参考索引.md` 的章节标注
- **查哪些是原创的** → 开 `作者原创贡献.md`
- **自查"这个设定可能有原型"** → 按域搜索，`参考域-*.md` 中未列出的可标注「待溯源」
