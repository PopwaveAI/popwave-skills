## 二、完整套路卡片

### 套路1：黑暗祭祀·神祇显化仪式
```
ID: V3-T016 | 强度：⭐⭐⭐⭐⭐ | 来源：ch057
适用场景：需要展示异教/黑暗神祇/献祭仪式
结构：进入禁地→亡灵/阴影生命退散→祭坛→自我献祭（鞭挞/鲜血）→神性显化→神恩治愈→神谕任务
关键要素：暗黑精灵、蛇首鞭、自我鞭挞（痛→快慰转化）、鲜血沿刻痕流动、粉红光芒、神少女显化（灰白皮肤/暗红刺青/蛇缠绕）、"去北方！"
作用：①拓展神祇体系 ②制造宗教神秘感 ③推动主线（神谕任务）④创造世界深度
变异方向：活祭/血祭/苦修/焚香/祈祷；战争神/丰收神/魅惑神/杀戮神
```

### 套路2：垂直压制·峭壁狙击
```
ID: V3-T017 | 强度：⭐⭐⭐⭐⭐ | 来源：ch060
适用场景：主角被数量远超己方的敌人追击时
结构：被追击→寻找地形→发现峭壁→攀爬占领高点→诱饵激怒→远程压制射杀→敌人改变战术包抄→转移→近战冲锋
关键要素：垂直90度峭壁、20米高度、强弩攻击强度20+、爆头秒杀、诱饵（幼崽狼头）、嘲讽成功系统提示
作用：①展示地形利用的战术智慧 ②远程秒杀的快感 ③游荡剑客进阶伏笔 ④"穿林"专长缺失为后续留下学习空间
变异方向：高塔/大树/城墙/屋顶/峡谷上方；不同诱饵（俘虏/尸体/旗帜/神器）
```

### 套路3：战利品拆解·捡漏变体（追加案例到现有套路）
```
ID: V3-T018 | 强度：⭐⭐⭐⭐ | 来源：ch056
结构变体：等待强者离开→搜刮战场→逐一翻找尸体→发现巫师之戒（1200金）→法术书→次元袋→清算总价值（~2000金）
与核心套路差异：不是"自己杀→自己拆"——是"别人杀→自己捡"——纯捡漏——零成本白赚
核心要素：时机把握（等巫师彻底离开）、知识变现（认识法术书和戒指的价值）、受限认知（没有秘法文字辨识专长）
```
