## 一、属性体系

### 六维属性

| 属性 | 说明 | 原文证据 |
|------|------|----------|
| **力量** | 物理攻击力、负重、近战伤害 | 索伦初始12(ch002)，食人魔护腕+2(ch018) |
| **敏捷** | 闪避、命中、潜行、先攻 | 索伦初始19→20超凡(ch006) |
| **体质** | 生命值、抗性、恢复速度 | 索伦15→18(ch014)→19(ch016)→20(ch051) |
| **智力** | 法术学习、学识、记忆能力 | 索伦18→19(ch068)→20(ch068) |
| **感知** | 意志豁免、预感、洞察 | 索伦15(从头到尾) |
| **魅力** | 社交、术士施法、威慑 | 索伦16→18(水精斗篷+2)(ch169) |

### 关键属性节点
- **20点=超凡领域**：20敏捷获得阴影掌握(ch010)；20体质获得再生初级(ch051)；20智力魔网崩溃后不会丧失施法能力(ch068)
- **25点=传奇门槛**：超凡敏捷25觉醒【阴影领域】传奇能力(ch017)
- 每2个职业等级获得1点自由属性(ch006)
- 成年后主属性还有5点以上增幅（未成年角色）(ch028)

> **原文证据**：「必须提升两个职业等级，才可以获得1点自由属性」（ch006）

| 等级段 | 阶位 | 特征 |

- 二阶获得【提气斩】——短时间内爆发力量，持续不超过2分钟(ch065)

#### 术士（Sorcerer）
- 元素能量主动亲近，靠魅力施法，不需要理解法术原理(ch027)
- 觉醒年龄：青春期(13岁左右)魅力约16点；提前1年觉醒基础魅力+1(ch028)
- 薇薇安8岁觉醒，天生魅力21点，获得【施法天才】专长(ch028/ch046)

> **原文证据**：「术士拥有与生俱来的施法能力，在青春期便开始展现原始的法力」（ch046）

#### 牧师（Cleric）
- 感知为施法主属性，依赖神灵赐予神术(ch024)
- 晨曦之主陨落后其牧师将失去神术能力(ch024)

#### 圣武士（Paladin）
- 战士战斗能力+神术（治疗、侦测邪恶）
- 免疫疾病、免疫恐惧
- 可召唤天马（男）/独角兽（女）作为坐骑(ch178)
- 死后灵魂回归上层位面(ch178)

#### 德鲁伊（Druid）
- 中立偏执，平衡高于善恶(ch058)
- 感知施法，可转化为动物形态
- 派系：激进派/温和派/调解派/平衡派(ch099)

#### 武僧（Monk）
- 意志属性最高的职业者之一(ch036-040)
- 传奇能力【超凡入圣】：免疫非超凡物理攻击(ch036-040)

## 三、等级体系

### 阶位划分
| 等级段 | 阶位 | 说明 |
|--------|------|------|
| 1-5级 | 一阶 | 基础职业者，只能城市内小打小闹(ch017) |
| 5-10级 | 二阶 | 熟练职业者，有离开城市冒险的资格(ch017) |
| 10-15级 | 三阶 | 精锐职业者(ch076-080) |
| 15级以上 | 高阶/传奇 | 可入城市统治层(ch080) |
| 20级 | 传奇领域 | 世界顶级战力 |

### 升级曲线
- 二阶→三阶：约需3000杀戮经验(5→6级)(ch034)
- 14级战士≈200万经验，14级巫师同样(ch052-053)
- 9级后经验需求暴增，14级后需10万+经验(ch141-145)
- 20级传奇仅10个自由属性点(ch011-015)

> **原文证据**：「二阶是一个分水岭。巫师开始掌握三级奥术……战士开始掌握提气斩……」（ch017）

## 四、法术体系

### 4.1 法术环位与学习成本

| 环级 | 学习职业经验 | 价格 | 特征 |
|------|-------------|------|------|
| 0环(戏法) | - | 10+金德勒 | 上百种，30节点+1回路(ch070) |
| 1环 | 100点 | 50-100金德勒 | 如奥术飞弹，300+法术节点(ch069) |
| 2环 | 200点 | 150-300金德勒 | 如隐身术，1200+法术节点(ch071-075) |
| 3环 | 400点 | 500-800金德勒 | 如火球术(ch070) |
| 4环 | 800点 | 1000+金德勒 | 如阴影黑触手(ch070) |
| 5环 | 1600点 | 需特殊组织 | - |
| 6环 | 3200点 | - | 如高等轻灵术 |
| 7环 | 6400点 | - | 如死亡一指 |
| 8环 | 12800点 | - | 如高等加速术 |
| 9环 | 更大 | - | 如流星爆、律令震慑 |

### 4.2 三大施法体系
| 属性 | 职业 | 核心逻辑 |
|------|------|----------|
| **智力** | 巫师 | 记忆+法术模型，大脑如生物电脑演算 |
| **感知** | 牧师/德鲁伊 | 意志+直觉+信仰力，借助神力/自然 |
| **魅力** | 术士 | 元素能量主动亲近，本能施法 |

> **原文证据**：「法术模型=子弹，法术位=弹仓，打掉了就消失需重新记忆」（ch056）

### 4.3 法术强度系统
- 公式：基础值(每职业等级+1) + 属性加值(每高出标准值2点+1)
- 【施法天才专长】：所有法术额外+2强度，-1秒施法时间(ch047)

### 4.4 法术位分配原则
- 1/3保命法术（法师护甲、护盾术、传送术、隐身术）
- 1/3控制法术（油腻术、蛛网术、怪物定身术）
- 1/3杀伤法术（火球术、火焰箭、强酸箭）
- 全部记忆杀伤法术的巫师「死得最快」(ch083)

### 4.5 魔网系统
- 魔法女神维系的无形能量网络(ch021)
- 魔网崩溃后：智力18以下永久失去施法能力，20以下只能施展0环戏法(ch021)
- 奥术复兴时代：巫师走向两极——极端废材或极端变态(ch021)

## 六、技能体系

### 基础技能
| 技能 | 说明 | 关键阈值 |
|------|------|----------|
| **潜行** | 暗影移动 | 100点→朦胧术效果；150点→类隐身术(ch059) |
| **躲避(翻滚/腾挪)** | 闪避能力 | 100+可训练寓守予攻 |
| **招架/格挡** | 防御 | 100+可训练寓守予攻 |
| **学识** | 知识储备 | 250点+博闻强记→解锁博学者 |
| **法术辨识** | 预判敌人法术 | - |
| **专注** | 施法成功率/冥想速度 | 泛用性极高(ch068) |
| **使用魔法装置** | 激活魔法物品 | - |
| **交涉/欺诈/威吓** | 社交技能 | - |
| **医疗** | 治疗知识 | - |

### 技能成长机制
- 基础技能可通过实战锻炼提升（战斗中自动获得+1、+2等）(ch034)
- 50点以下好掌握，100点=专业级(ch040)
- 150点潜行配合影武者【躲避视线】可在敌人目光中凭空消失(ch116)

### 传说度系统
- 获取方式：参与传奇战斗自动获得(ch085)
- 当前效果：1点=1点基础技能叠加到交涉/欺诈/威吓中(ch085)
- 未来作用：进入传奇领域后起「决定性作用」(ch085)
- 索伦终卷传说度：10点→+6=16点(ch187)

> **原文证据**：「传说度是封神的核心要素之一」「传说度最少需要200点以上」（ch187）

## 七、兼职系统

| 兼职数量 | 杀戮经验惩罚 | 说明 |
|----------|-------------|------|
| 第一个兼职 | 20% | 索伦兼职巫师 |
| 第二个兼职 | 40% | 需额外5000点 |
| 第三个兼职 | 60% | 需额外10000点 |

### 专职 vs 兼职
| 维度 | 专职 | 兼职 |
|------|------|------|
| 经验消耗 | 集中升级效率高 | 额外50%经验惩罚 |
| 传奇能力 | 快速获得 | 慢，除非弑神 |
| 属性点 | 偏少 | 多3-5个自由属性点 |
| 上限 | 单方向极限 | 多元融合 |

> **原文证据**：「第一个兼职有20%的杀戮经验惩罚，第二个职业就是40%的杀戮经验惩罚」（ch068）
