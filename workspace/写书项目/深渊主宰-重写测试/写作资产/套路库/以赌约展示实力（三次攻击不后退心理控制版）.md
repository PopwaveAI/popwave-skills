# 以赌约展示实力（"三次攻击不后退"心理控制版）

## 定义
实力悬殊时，强者不直接动手，而是通过一个看似狂妄的赌约（让步条件）来展示实力，用极致的技巧碾压后轻描淡写反问，瞬间建立心理控制。

## 适用场景
- 需要让新队友/随从信服主角实力
- 面对质疑者展示实力但不重伤对方
- 建立主角与配角的从属关系
- 快速确立主导地位

## 核心机制
1. **制造张力**：配角公开质疑或轻视主角实力
2. **提出赌约**：主角提出一个看似极其狂妄的条件（"三次攻击中你如何如何就算你赢"）
3. **极致展示**：用夸张的技巧秒杀（"闭眼闪避"+"反手开膛"级别）→让配角连机会都没有
4. **轻描淡写**：完成后不是炫耀，而是淡淡反问"还要再试吗？"或"如何？"
5. **结果**：配角从质疑→震惊→信服→态度180度转变（通常在30秒内完成）
6. **追加心理控制**：可以追加一句"手下留情"的暗示（如"如果不是我手下留情，这一刀已经把你斩成两段"）

## 典型结构
```
①配角质疑/轻视主角
②主角提出夸张赌约（制造悬念）
③配角全力攻击→主角极致闪避/防御
④主角一招制敌（展示碾压级实力）
⑤反问确认（"还要再试吗？"）
⑥配角转变态度→信服/臣服
```

## 案例参考
**ch172**：雅莉安娜质疑索伦"区区一个人类能帮我吗？"→索伦立约"三次攻击不逼我后退算你赢"→雅莉安娜全力扑击→索伦闭眼闪避（九十度后仰以毫厘之差避开）→反手开膛（弯刀浮现寒霜）→"还要再试吗？"→雅莉安娜冷汗直流→彻底信服

## 变体
- **武器对赌版**：赌对方一两招内夺不下武器
- **暗器对赌版**：赌对方在射程内碰不到自己
- **防御版**：赌对方打不破自己的防御法术/护甲
- **非战斗版**：赌学识/辨别毒药/开锁等专业技能

## 注意事项
- 赌约条件必须"看起来夸张但主角确实能做到"→不能太离谱让读者觉得假
- 展示后必须轻描淡写→不能得意洋洋→否则显得主角幼稚
- 赌约的逻辑要有基本合理性（如"闭眼闪避"说明感知已经到了盲斗级别）
- 如果没有后续利用这个心理控制→赌约场景就是浪费
