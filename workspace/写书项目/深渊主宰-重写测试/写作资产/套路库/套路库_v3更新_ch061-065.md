## 二、完整套路卡片

### 套路16：放风筝游斗（远程消耗流）
```
ID: V3-T016 | 强度：⭐⭐⭐⭐ | 来源：ch061
适用场景：速度/体质优于敌人但正面拼不过
结构：利用地形（山壁/屋顶）+速度差保持距离（15m+）→远程武器消耗（强弩/投掷）→利用敌人速度慢/笨重→敌人被消耗后→转入近战收割
关键要素：15m+安全距离、超凡体质（跑得过/恢复快）、铁板桥闪避/翻滚、远程→近战的切换时机
作用：①避免硬碰硬的群战损伤 ②展示战术智商 ③发挥属性优势
变异方向：室内放风筝（利用家具/柱子）/ 森林放风筝（利用树木阻挡）
```

### 套路17：拎头咆哮·斩首威吓
```
ID: V3-T017 | 强度：⭐⭐⭐⭐⭐ | 来源：ch061
适用场景：单挑击杀敌方首领后余敌仍在
结构：终结技斩首（要有"头"这个道具）→单手拎起头颅→震耳咆哮→确认敌人胆怯但未溃→第二次咆哮并高举头颅→敌人士气崩溃→溃逃
关键要素：两头式威吓（第一次测试/第二次击溃）、声音变化（从凶悍嚎叫→胆怯呜咽→溃逃）、拎头的野性气质
作用：①视觉冲击最强 ②不杀完所有敌人也能结束战斗（效率）③展现角色野性面
变异方向：拎其他身体部位（腿/爪）/ 举在火把上烧/ 悬挂在寨门上
```

### 套路18：潜行者对决（同行暗战）
```
ID: V3-T018 | 强度：⭐⭐⭐⭐ | 来源：ch063
适用场景：主角是潜行类职业+敌方也有潜行单位
结构：发现异常痕迹（湿地痕/细微声响/气味）→推理出有潜行者→回忆潜行者设定（潜行100→类隐身术）→遁入阴影搜索→通过光线扭曲/空气波动找位置→暴露瞬间冲刺→一刀斩向"虚空"→鲜血飞溅→敌人显形
关键要素：三感连锁（视觉+触觉+嗅觉）→异常识别、身体预警（寒毛乍起先于大脑判断）、"先见血再见人"的反直觉打击
作用：①展示侦察技能 ②同行对决的职业感 ③隐身vs反隐身的博弈
变异方向：夜间对夜间 / 室内对室内 / 利用反光/水面倒影发现
```

### 套路19：平民动员（恶言激将）
```
ID: V3-T019 | 强度：⭐⭐⭐⭐ | 来源：ch064
适用场景：平民在战斗中惊恐/哭泣/不知所措，需要快速转化为战力
结构：物理中断（耳光/踹倒）打破惊恐状态→恶毒语言描绘最害怕的画面（孩子被吃/家人被杀）→武器踢到面前→指令一个具体动作→第一个被激活→扩散到其他人
关键要素：母性是最易激活的战斗本能、恶毒语言比英勇鼓舞更有效、第一个人的"砍尸体"动作是身份转变仪式
作用：①不写女权说教写求生 ②展示角色不择手段的一面 ③让非战斗人员合理参战
变异方向：对男性用（妻子被侮辱/母亲被杀害）/ 对老人用（看到孙子被杀）/ 用同伴的尸体激活复仇心
```
