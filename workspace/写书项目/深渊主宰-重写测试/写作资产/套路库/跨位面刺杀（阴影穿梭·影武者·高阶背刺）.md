# 跨位面刺杀（阴影穿梭·影武者·高阶背刺）

> 套路类型：⚔️ 游荡者·高阶刺杀·跨位面攻击
> 强度：⭐⭐⭐⭐⭐
> 适用场景：游荡者面对强敌时的"一击必杀"式刺杀——从阴影位面跨越+背刺要害

---

## 核心机制

```
潜行接近（80-90米→可发动阴影穿梭的距离）
→ 发动阴影穿梭（跨越到阴影位面——灰白色/死寂/一秒）
→ 从目标影子中浮现（影子扭曲→朦胧身影→实体化）
→ 背刺命中（弯刀从后背刺入→穿透心脏→154点伤害）
→ 紧急撤离（无论是否致死→不恋战→立刻遁入阴影+躲藏目光）
```

## 变体说明

| 变体 | 结构 | 案例 |
|:----|:-----|:-----|
| 跨位面·影武者·背刺（标准版） | 阴影穿梭→跨越位面→影子浮现→实体化→背后刺入心脏→154伤害 | ch127 |
| 刺杀未死·高等强韧（反转版） | 刺穿心脏→未触发致死判定→紧急撤离→敌人暴走→流干血死 | ch127 |
| 阴影穿梭+躲藏目光（双技能组合版） | 阴影穿梭发动攻击→攻击后立刻躲藏目光→遁入阴影→二次隐形 | ch127 |

## 关键信息

- **阴影穿梭**: 影武者进阶能力——临时跨越阴影位面——从目标影子中浮现
- **阴影位面特征**: 灰白色景物/灰蒙蒙月光/一成不变死寂/一秒钟深刻印象
- **跨位面阴影跳跃**: 阴影穿梭成功后的额外能力——从影子中实体化
- **阴影袭杀**: 获得9点阴影伤害加持
- **背刺伤害**: 154点穿刺伤害——对18级/300血量的目标（半血）
- **使用限制**: 阴影穿梭有冷却——不能短时间内连续使用
- **撤退原则**: 无论是否触发致死判定——→立刻撤退——不贪刀——不恋战
- **后续技能**: 躲藏目光→潜行状态→二次隐形
