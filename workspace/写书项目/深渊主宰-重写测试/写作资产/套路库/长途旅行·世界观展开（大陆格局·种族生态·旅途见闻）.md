# 长途旅行·世界观展开（大陆格局·种族生态·旅途见闻）

> 套路类型：🗺️ 旅行·世界观·种族生态·大陆格局
> 强度：⭐⭐⭐⭐
> 适用场景：角色在长途旅行中通过沿途见闻自然引出世界观设定——不突兀——是"旅行中看到的事物"的延伸

---

## 核心机制

```
长途旅行（体力展示→一天200公里/三天800公里→21超凡体质）
→ 沿途见闻（看到村庄/听到狼嚎/经过丘陵/翻越山谷）
→ 自然联想（"精灵这边城市不多→一共才五六座"→引出城市分布格局）
→ 设定展开（城市总数/沙漠孤城/城邦制度/种族的后勤成本）
→ 旅行收束（看到目的地→笑意→"终于快要到达了"）
```

## 关键信息

- **索伦赶路速度**: 一天200+公里——三天800+公里——部分潜行
- **欧洲城市分布**: 精灵城市5-6座（城邦）/100公里内无其他大城市/大陆南方≈亚洲1/3面积/不到300座城市/集中在河流平原/沙漠仅6座（全在绿洲）
- **兽人座狼骑兵**: 1-2千骑——需要几十万人供养——肉食来自战争尸体+罪犯——冬天减少
- **卓尔石化蜥蜴**: 幽暗地域——喂养成本比座狼更贵——扔尸体喂养
- **野蛮人传承**: 最古老最纯正训练传承——每年进阶野蛮人拜访——索伦尝试过——负巨石山道前进——相当折磨
- **永久隐身设定**: 游荡者25级左右可通过仪式变成永久隐身——但不方便交流

### ch167 案例：财富之城·贸易中心·罪恶双面

#### 结构
索伦离开巫师塔后的场景切换→财富之城全景：络绎不绝的商队→"财富女神在上"口头禅→开拓运河连接内陆→近索罗斯群岛→最大财富神殿→单笔交易千枚金德勒→建筑刷金色颜料→欢愉女神+爱情女神信徒→繁荣与罪恶并存→下水道不缺乏尸体

#### 关键信息
- 财富女神重点关注城市——最大财富神殿
- 运河连接内陆+靠近索罗斯群岛——双线贸易枢纽
- 欢愉女神和爱情女神的牧师也可付费参与宴会
- "下水道不缺乏尸体"——一句话定标城市暗面

### ch168 案例：流沙渡口·蜥蜴人·公正之神·沉船海湾

#### 结构
流沙渡口（南北分界·黄河改道类比·梁山水泊类比）→水土流失（德鲁伊插手制止）→智慧蜥蜴人（懂建寨铸器·弱等神蜥蜴人之神·贸易宝石买粮·武器违禁）→公正之神领地（法律严格·邪恶受排斥·半奴隶制·圣武士打击黑市奴隶）→沉船海湾（藏宝海湾·领航员热门行业·暗潮鱼人+亡灵守护）

#### 关键信息
- 流沙渡口是南北交叉点——湖水奔流而下——曾因过度砍伐造成水土流失
- 蜥蜴人有弱等神庇佑——智慧种——善变狡诈——武器是贸易违禁品
- 公正之神领地法律严格——"王子犯法与庶民同罪"教义受贵族排斥
- 奴隶有身份证明和契约文书——黑市奴隶是圣武士打击重点
- 沉船海湾因地理环境多触礁——领航员热门行业
