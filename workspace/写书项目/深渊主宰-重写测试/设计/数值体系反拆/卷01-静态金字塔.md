# 第一卷宏观数值建模 —— 静态金字塔

> **分析范围：第一卷（ch001-ch187）**
> **⚠️ 本文件所有结论均为第一卷内部观测，不得默认为全书铁律。**
> 后卷可能因主角能级跃迁/地图扩展/叙事阶段变化而打破这些模式。
> 基于卷01全部187章数据聚合
> 反拆目标：揭示作者如何用"力量层级×社会生态"防止力量体系写崩

---

## 一、总览：全书力量金字塔结构

```
          ┌───────────────────┐
          │   真神（不在场）   │  ∅ 全书未直接出场，仅存在于背景
          ├───────────────────┤
          │   魔神/弱等神级   │  1-2位（魅魔女王、寒冬少女）
          ├───────────────────┤
          │   伪神/半神       │  1位（莱茵哈特·封神仪式）
          ├───────────────────┤
          │   传奇领域(20+)   │  6-10位（风暴领主/长公主/白银之手/戈洛尔/传奇武僧/野蛮人/恐怖骑士×8）
          ├───────────────────┤  ← 多数传奇作为"事件触发器"而非日常存在
          │   高阶/准传奇     │  ~20-50位（银月行者/北地女巫/食人魔督军/精灵长老等）
          ├───────────────────┤
          │   三阶(10-15级)   │  ~数百（城市精锐/地区强者）← 主角终卷位置
          ├───────────────────┤
          │   二阶(5-10级)    │  ~数千（城市中坚/熟练冒险者）
          ├───────────────────┤
          │   一阶(1-5级)     │  ~数万（城市卫兵/佣兵/初阶冒险者）
          ├───────────────────┤
          │   凡人/平民(0级)  │  ~百万+（普通市民/农民/手艺人）
          └───────────────────┘
```

**核心反拆结论：** 越往上的层级，存在数量越少，社会存在感越低。
- 传奇 ≈ 战略性威慑力量，日常不可见
- 高阶 ≈ 统治阶层中坚，但"不屑于冒险"
- 中低阶 ≈ 日常可接触的战斗力量
- 凡人 ≈ 世界的分母——战争统计中的"伤亡超万人"

## 二、逐层拆解

### 第1层：凡人/平民

| 维度 | 数值 | 原文证据 |
|:-----|:-----|:---------|
| **HP范围** | 5-15 | combat_capability: 普通人类 |
| **属性范围** | 10-12（标准人类10） | ch002索伦初始属性对比 |
| **全书出现数量** | 城市人口基数——琥珀城~数万人，白马城~数万人，全书涉及城市总人口~几十万 |
| **社会地位** | 底层。贫民区无法无天（ch001-010），贵族区有卫兵巡逻 |
| **对超凡者的无力感** | 完全无力。被传奇法术流星爆一下灭城万余人（ch069），面对二阶以上等于无抵抗 |
| **战斗张力** | 0——凡人对主角级战斗无意义，只作为"消耗品/受害者/背景" |

**关键反拆：作者不把"凡人"当作任何战力标尺。凡人的唯一作用是：** 
- 被杀时制造情绪冲击（琥珀城万人伤亡）
- 被救时制造道德审判（索伦守序邪恶的定位）
- 作为世界真实感的填充物

---

### 第2层：一阶超凡（1-5级）

| 维度 | 数值 | 原文证据 |
|:-----|:-----|:---------|
| **HP范围** | 20-60 | combat_capability: 索伦早期~60 |
| **攻击强度** | 无超凡武器时~20点攻击强度 |
| **社会地位** | 基础战斗力量。城市卫兵水平。狗头人(1级15经验) |
| **全书出现密度** | 琥珀城卫兵≈此级（ch001-005），白马城轻步兵1000≈此级+部分凡人 |
| **战斗张力** | 主角早期的主要对手。ch008杀11人多为凡人+一阶边缘。ch049-050狗头人群战 |

**作者打法规律：**
- 主角1-3级时：一阶是"势均力敌的对手"
- 主角4-5级(二阶)后：一阶变成"经验包/秒杀单位"
- 作用：初期制造紧张→中期制造战斗爽感→后期确认主角成长

---

### 第3层：二阶超凡（5-10级）

| 维度 | 数值 | 原文证据 |
|:-----|:-----|:---------|
| **HP范围** | 60-150 | 索伦HP136(ch122) |
| **升级门槛** | 5→6级约需3000杀戮经验（ch034） |
| **社会地位** | 城市中坚。白马城重步兵500≈此级。可离城冒险（ch017分水岭） |
| **代表对手** | 食人魔(普通) 5级/挑战6级；蜥蜴人(普通)2级 |
| **战斗张力** | 主角全程5-15级的主要战斗等级。势均力敌→小优→碾压的渐进 |

**关键反拆：**
二阶是全书最重要"战斗标尺"——作者用它来定义"正常的战斗该怎么打"：
- 二阶vs二阶：拼技能/专长/战术 → 主角的智力优势被放大
- 二阶vs三阶：需智取/陷阱/环境 → 主角的战术能力被放大
- 二阶vs非凡（食人魔）：需找到弱点 + 特殊打法 → 主角的"有策略的战斗"形象确立

---

### 第4层：三阶超凡（10-15级）

| 维度 | 数值 | 原文证据 |
|:-----|:-----|:---------|
| **HP范围** | 150-300 | 土元素HP~300(ch104) |
| **社会地位** | 城市精英，可担任军队军官或小队领袖 |
| **全书可见数量** | 约百人级。白马城精锐/骑士500中只有部分达到，其他为二阶 |
| **代表对手** | 蛇魔12级/挑战6（实际战力远超等级）；冰霜翼龙17级/挑战14；食人魔督军"三阶以上接近四阶" |
| **战斗张力** | 主角终卷(三阶刺客+微弱神性)的主要对手等级 |

**关键反拆：**
三阶是"准主角级"——这个层级世界足够多能形成冲突，但又不至于多到贬值：
- 索伦终卷是 ~7盗贼/2影武者/5巫师 = 约14级的总战力 = 三阶顶端
- 但他凭神性能力(恐惧目光/吸血鬼之触) + 战术 = 能越阶打准传奇(恐惧神子刺客)
- 这就是"主角越阶"的标准机制：**三阶肉身 + 神性外挂 → 可打17-18级敌人**

### 第5层：高阶/准传奇（15-20级）

| 维度 | 数值 | 原文证据 |
|:-----|:-----|:---------|
| **HP范围** | 300-500 | 冰霜翼龙HP300+(ch147) |
| **社会地位** | "15级以上被称为高阶职业者。可以轻松进入统治阶层——权力和财富都远超普通冒险"（ch080原文引用的等级说明） |
| **全书可见数量** | 约数十人。银月行者18级(ch126-130)、北地女巫16级(ch047)、食人魔督军"三阶接近四阶" |
| **核心机制** | **高阶很少冒险**——这是全书力量体系的第一道防崩墙。因为高阶已经在统治阶层获得权力财富，不会为几十金德勒战斗 |
| **战斗张力** | 主角无法单挑的层级。银月行者一箭射穿索伦手掌(ch127)；食人魔督军需要风暴领主才能击杀 |

**关键反拆："高阶不冒险"规则的防崩意义：**
- 如果高阶满大街跑+见人就打 → 世界无法运转 → 城市早被毁了
- 规则设计：高阶在城里当官→有利益要保护→不会轻易出手
- 索伦能接触到高阶的唯一方式 = "被卷入大事件"（碎骨之战/神子猎杀）
- 这天然把"日常冒险"和"主线高潮"分开——日常对手不超过三阶，主线BOSS才有高阶以上

**关键反拆——传奇的"非对称登场"设计：**

作者对传奇的使用极克制且精确：
1. **不退则进**：传奇集中出现在卷末/大战/危机时刻——但从不日常巡逻
2. **主角不直接对抗传奇**：索伦始终是"旁观/逃脱/被救"——绝不让主角越级打传奇
3. **传奇内战解决问题**：风暴领主杀戈洛尔，长公主放寒冬少女——主角在旁边看着
4. **恐怖骑士×8=8个传奇**：但用"亡灵法术召唤"解释为什么能一次出现这么多——一次性力量

### 第7层：半神/神性级+第8层：魔神级

这两层是"世界天花板展示"——不参与日常，仅作为背景设定和远期目标：

| 层级 | 实体 | HP/属性 | 全书可见次数 | 主角关系 |
|:-----|:-----|:--------|:-----------|:---------|
| 伪神 | 莱茵哈特 | 圣者级 | ch185-187 | 目击/被吸能术打中 |
| 弱等魔神 | 魅魔女王美坎修特 | HP800+, 魅力49 | ch175（仅作为线索引出） | 观察/无直接接触 |
| 神灵化身 | 寒冬少女 | 神力化身 | ch141-145 | 旁观被放逐 |
| 真神级 | 晨曦之主(已陨落)等 | — | 仅背景提及 | 无接触 |

**防崩机制：这些层级只用"展示"不用"战斗"**
- 主角不可能打赢 → 所以不安排直接交手
- 它们的存在目的是：**拉大世界纵深**——让读者知道"上面还有很大很大"
- 到卷末主角才刚"触摸"到神性门槛(3/10微弱神性) → 为今后百章以上保留空间

## 三、核心反拆发现：作者设计这套金字塔的底层逻辑

### 发现1：五层门禁

每一层到下一层都有**硬性门禁**，防止越级接触：

| 门禁 | 跨越方式 | 原文示例 |
|:-----|:---------|:---------|
| 凡人→一阶 | 训练/职业等级提升 | 平民10→盗贼1 |
| 一阶→二阶 | 5级门槛 + 二阶专长 | 盗贼5→阴影袭杀(ch017) |
| 二阶→三阶 | 3000+经验 + 职业等级 | 6级开启 |
| 三阶→高阶 | 15级 + "冒险不再划算" | ch080等级体系说明 |
| 高阶→传奇 | 20级 + 传奇进阶 | 风暴领主、白银之手 |

### 发现2：高阶"不冒险"规则是全书防崩的核心里程碑

ch080原文直接说了：高阶15+可以轻松进入统治阶层，权力财富远超冒险——
这翻译过来就是：**作者在告诉读者，为什么路上的怪物/敌人最多到三阶，而传奇只在特殊事件中登场。**

如果传奇满大街跑 + 主角随便碰到 → 世界力量体系立刻崩掉

### 发现3：每一层的"人口密度"呈指数级递减

| 层级 | 估计数量/城 | 在全书中的功能 |
|:-----|:-----------|:-------------|
| 凡人 | 万人级 | 背景板/炮灰/受害者 |
| 一阶 | 千人级 | 城市卫兵/经验包 |
| 二阶 | 百人级 | 主力战斗对手/队友 |
| 三阶 | 十人级 | 精英对手/配角 |
| 高阶 | 个位数/城 | 统治层/事件触发者 |
| 传奇 | 0-1位/城 | 终极威慑/事件高潮 |

### 发现4：力量层级 = 地图分层

| 地图 | 活跃层级上限 |
|:-----|:------------|
| 琥珀城贫民区 | 一阶（黑帮老大也就二阶边缘） |
| 琥珀城→荒野 | 二阶（蛇魔12级作为关底BOSS） |
| 白马城 | 三阶（军队体系） |
| 碎骨战场 | 传奇（风暴领主vs戈洛尔） |
| 北境→阿伦黛尔 | 神灵级（长公主vs寒冬少女） |
| 财富之城→封神仪式 | 伪神级 |

**结论：地图换一次，金字塔天花板提高一层。主角在新地图底部重新爬。旧地图因为"主角不在那活动了" → 旧金字塔不被破坏。**
