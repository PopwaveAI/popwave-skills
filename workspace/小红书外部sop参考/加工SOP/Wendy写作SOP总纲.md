# Wendy写作SOP总纲

> 汇总Wendy的5期写作教程，按写作学习流程排序：学习路径 → 拆文方法 → 人物塑造 → 剧情冲突 → 灵感到故事

---

## 一、学习路径（新手入门四步）

**核心原则**：不用天赋、不用科班、不用报课，按四步走即可入门。

1. **看书扫榜**：去目标平台看排行榜前十本，每本只看前三章，分析为什么火、读者是谁、开头怎么抓人
2. **拆文模仿**：找一本"我也能写"的爆款，逐章拆开头/结构/人物/对话/节奏，分类思考后模仿
3. **自己原创**：从2-3万字中长篇起步，写完整故事（开头-发展-转折-结尾），发平台看数据
4. **迭代提升**：满足市场需求 × 执行速度 × 作品数量，每篇都用方法循环提升

**三个月计划**：扫榜20天 → 拆文30天 → 原创25天 → 迭代15天

## 二、拆文方法（六维拆文法）

**核心理念**：高手拆骨架和节奏，新手拆皮相和词藻。

### 六个维度

| 维度 | 拆什么 |
|------|--------|
| 人物设定 | 姓名/年龄/身份/欲望/缺陷，配角区分推动剧情型和强化主角型 |
| 关系图谱 | 画连线图，每两人间写动词，标利益同盟/情感羁绊/表面朋友实为敌人 |
| 剧情冲突 | 罗列所有冲突，标记利益冲突/信念冲突，双重冲突=大反派 |
| 正常时间线 | 按真实发生顺序从头到尾看一遍 |
| 成品节奏 | 按作者实际写作顺序逐章标注（每章300-500字叙述） |
| 题材+亮点 | 类型/字数/最扎人的三个点 |

### 最核心方法：正常时间线 vs 成品剧情对照

左边写正常时间线，右边写成品剧情，对照看：
- 剧情被**提前** → 制造悬念
- 剧情被**藏到最后** → 制造反转
- 所有调整都为制造**信息差**

### 剧情曲线 vs 情绪曲线

两张纸叠在一起，深色笔标记着重描写的强剧情（高峰），浅色笔标记一笔带过的缓冲区。对比后知道哪里该**舍得写**、哪里该**舍得跳**。

### 拆透标准

拆完一本书能推出**三本完全不同类型**的新故事——提取底层人物弧光和写作逻辑后重新造骨捏人物，而非换表面元素（那是抄袭）。

## 三、人物塑造体系

**核心理念**：不堆标签，给角色内在驱动力——执念、软肋和底线。

### 人物小传六维度

1. **生理属性**：外貌、身高、特征、健康状态、第一印象
2. **社会属性**：姓名/绰号/家庭/学历/职业/收入/人际圈
3. **性格心理**：内外向/人生态度/紧张反应/**内心恐惧**/**最想要什么**
4. **背景故事**：成长经历/关键事件/转折点/当前困境来源
5. **知识技能**：专业技能/生活技能/爱好/语言风格/口头禅
6. **未来走向**：故事开始时的状态/面临矛盾/可能变化

### 性格弧光（人物灵魂）

角色因经历事件在信念/价值观/行为模式上发生**可辨认的、有因果的变化**。

三个问题：
1. 他原来信什么？
2. 经历了什么事（和谁有关）让观念变化？
3. 观念改变后选择了什么？

**弧光=信念变化曲线**，曲线越大前后对比越强。但微小变化也是弧光。

**高光点**（旧信念切换到新信念的瞬间）= 最亮眼/最感动/最容易爆数据的瞬间。

**写法**：三次后退 + 两次犹豫 + 一次情绪崩盘 + 一次大顿悟。

### 长短篇区别

- **长篇**：像剥洋葱，一层一层露。初次登场只写一两个点，后续慢慢展开
- **短篇**：不铺陈外貌背景，重点写身份/动作/事件选择/语言对话

## 四、剧情冲突挖掘

**核心问题**：写不出剧情是因为只盯主角。故事能讲多久在于主角一路上遇到多少人。

### 两种冲突

| | 利益冲突 | 信念冲突 |
|---|---------|---------|
| 争夺 | 资源/物质 | 对错/输赢 |
| 可分 | 可以分 | 不可分 |
| 解决 | 谈判/补偿 | 说服/碰撞 |
| 调和 | 可调和 | 不可调和 |

### 三种对手

1. **纯利益对手** → 用利益谈判
2. **纯信念对手** → 用说服碰撞
3. **混合型对手**（利益+信念） → **死敌反派**，大结局PK

### 派系冲突（宏观）

1. 画**利益阵营图**（按利益分组，注意身份重叠者）
2. 画**信念阵营图**（价值观相似的站一起）
3. 找派系间矛盾点

**关键**：只有信念冲突没有利益冲突的派系会**结盟对付第三方**。主角每次行动都会改变派系图平衡。

### 钩子技巧（改变呈现顺序制造信息差）

1. **倒序**：紧张事件提到开篇
2. **信息差**：读者知道A不知B在使坏
3. **视角切换**：换配角视角讲同一件事

实操：先按正常顺序写 → 问三个问题（什么做开篇/什么藏到最后/谁的视角信息差最大）→ 打乱重排。**钩子不能乱用，1-2个贯穿即可。**

## 五、从灵感到完整故事（七步流程）

1. **确定主角人设**：用人物事件模板，设定性格弧光（信念变化曲线）
2. **推导反派**：镜子两面，相似但不相同，反派用不正当手段
3. **出人物关系图谱**：梳理谁想要什么、谁和谁利益关系、谁暗中支持谁
4. **正常时间线写大纲**：一句话概括 → 按时间顺序写。短篇3-5千字，长篇3-5万字
5. **打乱时间线**：插入伏笔、切换顺序。**过去是伏笔，现在是战场，未来是钩子，三线合一**
6. **写章纲**：大纲是全景，章纲是近景/中景。**必须写事件，不写状态**
7. **写正文**：从近景到特写，信息密度最大。写完回头查逻辑bug

**大纲的真正目的**：系统化灵感，不依靠灵感持续稳定输出。

## 快速检查清单

- [ ] 人物有没有内在驱动力（不是堆标签）？
- [ ] 有没有性格弧光（信念变化曲线）？
- [ ] 冲突是利益型还是信念型？策略用对了吗？
- [ ] 有没有派系网络？主角在其中的位置是什么？
- [ ] 正常时间线和成品剧情对照了吗？信息差制造了吗？
- [ ] 章纲写的是事件还是状态？
- [ ] 三线合一了吗（过去=伏笔/现在=战场/未来=钩子）？
- [ ] 钩子控制在1-2个了吗？
