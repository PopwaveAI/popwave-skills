# 手把手教你写出第一本小说——主线篇（加工整理）

> 本文整理自小红书"编辑老张讲写作"系列视频第3期，原视频时长4分16秒。

**一句话摘要：先别急着写大纲，用三步法构建主线剧情——确定终点站（主角目标）、建立服务区（制造事故）、用跳方格法推演连锁反应，把灵感变成完整的故事线。**

## 一、先写主线，再写大纲

你写小说是不是想到哪里写到哪里，毫无章法？卡文都是活该。

写大纲的目的是让它作为整个故事的**骨架和指南针**，帮你确定大概的方向，避免中途跑偏。但是——你见过只有一半骨架的正常人吗？像那种不想清楚非要先写大纲的，到后期没有一部不推翻重写的。

所以今天要教你的不是写大纲，而是决定一个故事成败的**主线剧情**：如何把脑子里的灵感变成一条完整的主线。

## 二、三步推出主线情节

### 第一步：确定终点站——主角的欲望和目标

这个在上一期定人设的时候就已经梳理过了。

想让主角的目标更清晰具体，教你一招**先修大法**：

1. 告诉豆包或者Kimi老师你的角色大致的人设
2. 然后问它这个角色会有什么目标
3. AI会给你胡说八道甚至错误的理解
4. 在你纠正的过程当中，主角的目标、欲望甚至人物形象都会越来越清晰

这一步就拿到了路线图的**终点站**，接下来就需要通过中间的**服务区**把整条路线串联起来。

### 第二步：建立服务区——给主角制造事故

学会给主角制造他不想做的事、可以**逆人设**但他非做不可的事，才能推动故事剧情。

**情节推动的本质**是：主角原有的生活被打破后的连锁反应。重点突出**意外性**和**后续连锁影响**。

- **意外性**：突发事件或者人生的转折点事件，一般都是不可逆的变化。

> 实在想不出来这些事情的话，有分类整理好的可以直接套用的**高能事件模板**。

### 第三步：让突发事件产生连锁反应

这一步非常、非常的重要。我们采用**跳方格形式写故事**，分成几个小步骤：

#### 跳方格第一步：核心事件

拿出一张纸，在上面写上：**核心事件 → 角色行动 → 剧情变化**

例如：
- 主角进入了某个区域 → 他发现了一个**惊天秘密**
- 主角打败了某个反派 → 他得到了一个重要物品
- 主角卷入了某个风波 → 他遇到了某个重要的人物

#### 跳方格第二步：意外触发 → 新的境地

再拿一张纸，在上面写上两个点：**①意外触发 ②新的境地**

新的境地又分为三种情况：
- **好的情况**：获得奖励，或者有心的帮手
- **坏的情况**：最终导致危机加深
- **意外的情况**：预料之外、不可想象的方向

在这三个过程当中，就可以引入**男主、配角、反派**的出现，去为主角解决问题，或者制造一些阻碍和转机。

#### 跳方格第三步：阶段性危机解除

拿出第三张纸，写上**阶段性危机解除**——从结果倒推主角解决问题的方法。

**关键技巧**：如果能够在事件结尾处留一个**悬念点**，就会让你下一个情节更有吸引力。比如：
- 事件的疑点只解决了一半，另一半好像近在眼前
- 关键时刻被打断，出现了新变数
- 面临新的抉择
- 用问题推动下一个情节

## 三、完整案例演示

假设你要写"主角误入魔界"的剧情：

### 核心事件

主角意外闯入魔界 → 发现这里正在举办**鉴宝大会** → 被意外认成魔界的某位贵族 → 被迫参加鉴宝开场

### 意外触发 → 新境地

- **好的方向**：发现了一个一直在找的重要宝物
- **坏的方向**：被某个之前打过脸的炮灰认出了身份，遭到众魔追杀
- **意外的**：有一对魔力强大的尊者保驾

### 阶段性危机解除 + 悬念

如果主角成功逃离并拿到了宝物，但是发现宝物上附了魔——

**悬念**：
1. 主角怎么解决附魔问题？
2. 尊者为什么没有一起追杀主角，反而拦下了追兵？

## 四、总结

故事的开始只有一个意外事件，这个意外事件就是**蝴蝶的翅膀**，扇动后引发了后面无数个情节的连锁反应。

只要思考清晰，所有的剧情都可以通过倒推找到逻辑点。
