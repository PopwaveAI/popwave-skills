# 编辑老张写作SOP总纲

> 本文件汇总"编辑老张讲写作"系列前三期视频的核心要点，按写作流程排列为：题材选择 → 人设搭建 → 主线设计，构成新人写小说的完整起步框架。

**总纲摘要：新人写小说的正确顺序是先定题材、再定人设、然后推演主线、最后才写大纲。顺序不能颠倒，否则容易推翻重来。**

## 流程总览

```
题材选择（第1期）→ 人设搭建（第2期）→ 主线设计（第3期）→ 写大纲（后续）
```

> **核心原则**：先定题材 → 再定人设 → 然后定主线 → 最后写大纲。每一步都是下一步的基础，跳步会导致后期推翻重来。

---

## 一、题材选择（第1期）

### 核心原则

新人第一步不是写大纲、不是刷榜，而是**先定题材**。方向没选对，大纲写到一百章都没有用。

### 四步定题材

| 步骤 | 内容 | 要点 |
|------|------|------|
| 第一步：判断大方向 | 搞事业 vs 谈恋爱 | **剧情流**（如《甄嬛传》《斗破苍穹》）vs **感情流**（如纯爱文、《我的26岁女房客》） |
| 第二步：定时代背景 | 6大类 | 洪荒神话、古代、年代、都市、末世、未来科幻 |
| 第三步：叠加设定 | 5项设定 | 剧情设定、性向设定、时空设定、金手指、世界观 |
| 第四步：组合出分类 | 标签直接出来 | 如：古代 + 重生 + 言情 = 古代重生言情 |

### 多题材策略

- 选最热门的
- 全都要 → 写**无限流**或**快穿**（对想象力和脑洞要求高）

### 新手推荐

- **同人文**：省去搭建世界观，自带热度
- **豪门总裁 / 都市日常 / 娱乐圈**：均属言情类，素材来源贴近生活

---

## 二、人设搭建（第2期）

### 核心原则

题材定好后，先定人设再写大纲。否则后期发现剧情不符合人物性格，要么删了重改，要么违和地硬走剧情。

### 三步定人设

1. **列出喜欢的角色特征**：从喜欢的真实/虚拟人物中，提取性格、外貌、人生经历三个点
2. **选出熟悉的人物结合**：选有参考原型且自己了解的，比凭空想象好写得多
3. **扩展形成完整人物**：
   - **丰富性格立体度**：主性格 + 反差面（如人前软萌小白兔，人后心狠白莲花）
   - **扩展外貌特征**：标志性外貌 + 合理理由
   - **扩展人物经历**（最难）：梳理前因后果 → 替换代表事物 → 加成长目标 → 加金手指 → 加反转

### 人设扩展核心技巧

- **替换法**：找到原型的代表事物，替换成自己的（如"眼睛"→"功法秘籍"）
- **反差法**：给主性格加一个对立面，制造反差感
- **反转设计**：梳理所有角色和物品，每个都可以做反转，引出新的故事线
- **金手指叠加**：加入重生、系统、穿越、穿书等热门网文元素增强爽感

---

## 三、主线设计（第3期）

### 核心原则

先写主线剧情，再写大纲。大纲是骨架和指南针，但只有半个骨架的人是走不远的。

### 三步推出主线

1. **确定终点站**：主角的欲望和目标
   - 可用AI辅助法（先修大法）：让AI根据人设推演目标，在纠正过程中逐渐清晰

2. **建立服务区**：给主角制造事故
   - 制造逆人设但非做不可的事
   - 核心是**意外性**（不可逆的变化）+ **后续连锁影响**

3. **推演连锁反应**：跳方格法
   - **格子1**：核心事件 → 角色行动 → 剧情变化
   - **格子2**：意外触发 → 新境地（好/坏/意外三种情况）→ 引入其他角色
   - **格子3**：阶段性危机解除 → 留悬念点 → 推进下一情节

### 跳方格模板

```
核心事件 → 角色行动 → 剧情变化
    ↓
意外触发 → 新境地（好/坏/意外）
    ↓                    ↓
引入男主/配角/反派 → 解决或制造阻碍
    ↓
阶段性危机解除 → 留悬念 → 推进下一情节
```

### 核心认知

> 一个意外事件就是蝴蝶的翅膀，扇动后引发后面无数个情节的连锁反应。所有剧情都可以通过倒推找到逻辑点。

---

## 快速检查清单

- [ ] 题材已确定（大方向 + 时代背景 + 设定 + 分类标签）
- [ ] 人设已完成（主性格 + 反差面 + 外貌特征 + 人物经历 + 成长目标 + 金手指 + 反转点）
- [ ] 主线已推演（终点站 + 服务区 + 连锁反应 + 悬念）
- [ ] 逻辑自洽（每个反转都能引出故事线）

---

完成以上三步后，就可以进入下一步：写出编辑眼中的好大纲。
