# L1-04 · 物种与天赋

> **基于前100章数据推断**

## 人类分类
| 类型 | 特征 | 代表 |
|:-----|:------|:------|
| 普通人 | 无特殊能力，在诡异面前极度脆弱 | 房东、江莺莺父母、多数邻居 |
| 驭诡者 | 体内容纳诡的力量，获得异能但要承受复苏代价 | 苏午(尸陀鬼之手)、医生(融合)、广法 |
| 破限者 | 超越凡人极限，肌肉呈玉质光泽 | 苏午(心猿拳破限)、神打派1号方案成功者 |
| 灵藏命格 | 天然吸引/勾招诡异的特殊体质 | 卓杰 |
| 咒印持有者 | 通过受戒/灌注获得咒印 | 苏午(力士/眼地藏/虎魔) |

## 天赋体系（五色品质）
白色→绿色→蓝色→紫色→橙色（推测）

| 天赋 | 品质 | 效果 | 获取方式 |
|:-----|:-----|:------|:---------|
| 冷静 | 蓝⭐ | 古之名将级洞察力+混乱中推演最佳选择 | 模拟器抽取→升级 |
| 锦鲤附体 | 绿 | 更容易获得神秘物品 | 模拟器兑换+升级 |
| 阴暗之触 | 绿 | 阴影三增益：感知/力量/恢复 | 黑暗行者变异 |
| 肌肉记忆 | 白 | 20%肌肉记忆加成+条件反射 | 格斗家训练领悟 |
| 受感而击 | 被动技能 | 敌意感知+肌肉预判+自动还击 | 心猿拳破限衍生 |
| 疼痛转移 | 白 | 分散注意力转移痛苦 | 模拟中领悟 |
