# L1-07 · 金手指设计

> **基于前100章数据推断**

## 金手指名称：完美人生模拟器
**类型**: 系统/模拟器流 | **载体**: 祖传手表（爷爷→父亲→苏午）

## 核心机制
| 维度 | 内容 |
|:------|:------|
| 模拟未来 | 消耗1元玉可模拟自己的未来人生，死亡=模拟结束，获得评分+奖励 |
| 模拟过去 | 获得他人遗物后可消耗元玉进入对方的过去人生，学习技能/知识 |
| 天赋系统 | 模拟中可抽取/领悟临时天赋，消耗元玉可带出成为永久 |
| 物品带出 | 模拟中获得的物品可消耗元玉带出现实 |
| 安全屋 | 特定地点可自动存档，下次模拟从存档点开始 |
| 亲身体验 | 额外消耗元玉可亲身体验（死亡=现实死亡，但获得永久天赋/提升） |

## 限制
1. 元玉需要模拟评分获得，初期余额极低（初始5元玉）
2. 未来模拟的死亡只是模拟结束，但亲身体验死亡=真死
3. 评分体系：丁<丙<乙<甲（每级分上/中/下），低分奖励极少
4. 遗物一旦损坏（如匕首），该过去人生永久关闭
5. 高价值物品（如尸陀鬼之手）带出需巨额元玉

## 商业逻辑
玩家对模拟器的使用=普通人对RPG游戏的攻略心态：刷分→攒资源→解锁新内容→变强→打更难的副本。不同之处在于模拟器的奖励（天赋/咒印/知识）是真实可以带出到现实的。

> **商业性评分**: ⭐⭐⭐⭐☆（模拟器类金手指读者接受度高，元玉的经济循环设计紧凑）
