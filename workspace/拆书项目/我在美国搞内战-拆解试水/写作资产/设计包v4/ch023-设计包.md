## 2. 爽点设计（爽点层）

| 爽点 | 对应事件 | 机制说明 |
|------|---------|---------|
| 职业选择——信息曝光+决策权 | 事件1-4 | 五种路径逐一展示，读者获得完整信息层面的掌控感；最终选择巫医，符合"有超凡力量才独特"的读者期待 |
| 圣巫之血——神级技能到手 | 事件4-5 | 可治疗"几乎所有已知病症和伤痛"的逆天技能，带来超模金手指的爽感；同时主角清醒分析风险，体现智商在线 |
| 仪式要求——暗黑神秘感 | 事件6 | "恶尸为台，毒婴为祭"台词极具仪式感和冲击力，"美利坚最不缺尸体"的吐槽强化本土化幽默 |
| 技能进化——全科→基础巫医 | 事件7 | 之前靠现代医学忽悠，现在是真的巫医之力，质的飞跃带来升级爽感 |

## 3. 角色与人设（角色层）

| 角色 | 本章表现 | 人设要点 |
|------|---------|---------|
| **韦恩** | 理性分析五个职业路径；有危机意识（打探酒瓶老乔）；主动出击而非被动等待；对圣巫之血的风险有清醒认知 | 冒险者→巫医（2级），新增专属技能圣巫之血；称号圣徒5%→6%；思维方式从"苟"转向"主动出击+包装宗教外壳" |
| **迈克** | 不理解发生了什么但信任兄弟；与有荣焉 | 忠厚老实型兄弟，情绪跟随韦恩起伏 |
| **戴维** | 想买熊猫快餐改善伙食；对食物银行有兴趣 | 注重生活品质改善，具备一定的社区生存知识（知道周三可能发披萨） |
| **众流浪汉** | 自发为韦恩让路，眼中充满敬畏 | 群体角色——已经从"病人-医生"升级为"信徒-圣徒"的宗教式关系 |

## 4. 设定/物品提取区

| 项目 | 分类 | 说明 |
|------|------|------|
| 五大职业路径 | 核心设定 | 吟游诗人/医生/巫医/农夫/剑士——与主角已有技能/称号绑定 |
| 转职道具机制 | 核心设定 | 可转职/双职业，但需使用转职道具，系统会自动提示线索 |
| 【圣巫之血】技能 | 核心设定 | 巫医专属，治疗几乎所有已知病症和伤痛；仪式材料要求极苛刻 |
| 圣巫之血仪式材料 | 核心设定 | 魔晶、灵骨、恶尸、毒婴——四件黑暗材料提示后续收集线 |

| 基础巫医（专精5%） | 技能 | 取代全科医学，具备真实超凡医疗能力 |
| 食物银行（Feeding America） | 场景/设定 | 美国NGO慈善组织，"仓库+前线"模式分发临期食品，政府项目合作 |
| 熊猫快餐 | 物品 | 美国中餐快餐连锁，在此作为"高档食物"的象征 |
| 职业面板升级提示 | 系统UI | 升级后每次获取经验值效率变低（之前几十点，现在1点） |
