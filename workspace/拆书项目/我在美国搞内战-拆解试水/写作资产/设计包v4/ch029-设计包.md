## 4. 设定/物品提取区

| 物品/设定 | 类型 | 说明 |
|----------|------|------|
| **终缚圣事（病人傅油圣事）** | 宗教设定 | 天主教七大圣事之一，为濒死者提供属灵治愈——被韦恩借用为巫术仪式名目 |
| **圣巫祭坛** | 仪式场景 | 结构：魔晶（史莱姆）铺底→骸骨铺层→恶尸为台→毒婴置顶→巫毒灵骨入列 |
| **巫毒教灵骨** | 道具（来自老约翰） | 韦恩从之前剧情中获得的巫毒教法器材料，用于提升祭坛效力 |
| **圣巫之血** | 核心成长物品（已获得） | 第一滴圣巫之血凝结于韦恩食指指尖，有"融开坚冰"般的温热感；可用于后续巫医技能提升 |
| **基础巫医（专精10%）** | 技能 | 原专精7%+仪式奖励3%=10%，系韦恩的主职业技能线之一 |
| **广播体操（仪式替代）** | 设定/笑点 | 主角上辈子唯一会的"舞蹈"，被当作巫医之舞使用且意外有效 |
| **杨慎《西江月·道德三皇五帝》** | 设定/笑点 | 明代词作被当作咒语念诵，仪式后仍生效，暗示仪式真实机制不在语言而在意念/触媒 |
| **光芒涟漪（治愈光环）** | 能力效果 | 圣巫之血凝结时的副产物，对范围生灵产生治愈效果（关节炎、寒湿、强化剂依赖症状等） |
| **尸体拾取规则** | 系统设定 | 仅新死不久（数小时内）的尸体可触发技能拾取，死透多时的尸体无效 |
