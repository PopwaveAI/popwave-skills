## 2. 爽点设计（爽点层）

| 爽点类型 | 触发事件 | 爽感机制 |
|----------|----------|----------|
| ✅ 宝物鉴定快感 | 在垃圾堆中识别出真正的圣遗物 | 从一文不值中发现珍宝的"捡漏"兴奋感 |
| ✅ 历史知识装逼 | 韦恩推理弥赛亚硬币出自洪秀全之手 | 串联东西方宗教史的博学感，知识就是力量 |
| ✅ 现实暴富 | 一枚硬币可一比一兑换商店金币（≈$13000） | 五千美刀成本→超一万三千美刀收益，暴击式投资回报率 |
| ✅ 恐怖战力预期 | 庇护所守卫=复活的八具尸体 | 亡灵军队的震撼脑补，视觉想象力爽点 |
| ✅ 伏笔回收 | 乔治爵士古董店从废墟联想变现实 | 前文埋下的地名/旗帜线索在本章获得完整回应 |
| ✅ 兄弟情义验证 | 迈克为韦恩花光五千刀踩到大运 | 好人有好报的因果爽感，兄弟直觉正确 |
| ✅ 实用知识科普 | 大蒜素制作教程 | 读者可以学到的真·生存技能，增加作品实用价值 |
| ✅ 社会批判 | 揭露美利坚医学专业英文体系为"屎山代码" | 讽刺西方有意制造知识壁垒以获得医疗暴利，解气 |
| ✅ 寻宝冒险预期 | 结尾"以天兄之名"前往古董店 | 勾出读者对下一章的期待，像游戏开新地图 |
| ✅ 人设细节 | 韦恩带着小狗威克出门 | 温和细节让角色更有温度，反差萌 |

| 韦恩 | 主角·天兄继承者 | 敏锐鉴定宝物，冷静规划日程，知识传授追随者，以天兄之名出发 | "弥赛亚……太平天国……"——串联历史链条的智者时刻；"以天兄之名"——宣言式结尾 |

## 4. 设定/物品提取区

| 设定/物品 | 类型 | 说明 |
|-----------|------|------|
| 【弥赛亚硬币】（完整设定） | 圣遗物·货币 | 太平天国圣宝铜钱，被洪秀全触碰过。效果：①一比一兑换商店金币（≈$13000/枚）②可提前雇佣庇护所守卫（1枚/人/10分钟） |
| 庇护所守卫 | 系统设定·隐藏战力 | 八具半跪土中做祷告状的尸体，需弥赛亚硬币激活，推测为死而复生操控尸体模式 |
| 系统商店金币 | 经济设定 | 1金币≈$13000（按当前金价计算），构成韦恩重要经济来源 |
| 史诗级任务【地上天国】 | 任务设定 | 弥赛亚硬币是该任务的圣遗物，暗示主线方向 |
| 乔治爵士古董店 | 场景·关键地点 | 位于海港区旅游区，已开一百多年。地下巷道中有同名的废墟店铺（其中找到太平天国旗帜），地上店面依然正常营业 |
| 太平天国旗帜 | 关键物品（铺垫） | 此前从同一家地下废墟中找到，与弥赛亚硬币同源 |
| 大蒜素（完整设定） | 药品·生存技能 | 大蒜破碎后由蒜氨酸酶分解产生的活性硫化合物，有广谱抗菌、降低耐药性、保护心脑血管、抗癌效果。橄榄油萃取后可冷藏保存约6天 |
| 美利坚识字率数据 | 社会设定 | 21%功能性文盲，54%识字水平低于小学六年级 |
| 西大医疗知识垄断 | 世界观批判 | 专业英文体系被设计为"屎山代码"，普通人无法自我诊疗，必须依赖昂贵医疗体系 |
| 东大大蒜禁令 | 政治讽刺设定 | 西大认定东大大蒜危害国家安全（实际原因是东大大蒜便宜3-4倍） |
