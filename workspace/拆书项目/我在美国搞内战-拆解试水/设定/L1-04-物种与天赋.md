# L1-04 · 物种与天赋（基于前74章数据推断）

> 本书中，**魔域地下城系统**将现实美利坚各人种/阶层映射为黑暗奇幻种族。种族划分本质是**系统滤镜对现实社会的魔幻化翻译**——并非真实存在超自然种族，而是系统将底层群体按种族、阶层、行为特征归类命名。以下为前74章已出现的所有系统种族分类、特殊天赋、怪物数据与人物能力原型。

---

| 种族 | 现实映射 | 社会层级 | 首次出现 | 关键词 |

| **吸血鬼** | 高价尸体买家/研究所 | 上巢→暗世界 | ch019§12 | 吸血鬼培育所 |

> **核心设定**：系统种族标签不等于生物学分类，而是系统根据**社会身份+行为特征**给予的魔幻化标注。本质是「美利坚底层社会观察报告」的游戏化呈现。

## 二、各物种详解

### 2.1 哥布林（Goblin）

| 项目 | 内容 |
|------|------|
| **现实映射** | 黑人/非洲裔流浪汉及街头人员 |
| **系统判定特征** | 肮脏的、领地意识较强、生活在「下巢」 |
| **常见标签** | 流浪哥布林、下巢哥布林、小哥布林 |
| **社会身份** | 底层流浪汉、收尸人产业链底层、强化剂受害者 |
| **已知角色** | 迈克（光头哥布林·生死之交）、戴维（辫子哥布林·生死之交）、老威尔、乔伊（小哥布林·已故） |
| **行为模式** | 领地意识强（尸体=资源）、互助性强（多人一起扎营）、对权威既怕又依赖 |
| **忠诚度体系** | 可发展羁绊/效忠：忠诚度等级有「生死之交」「神恩浩荡」「至死不渝」「狂热信徒」等 |
| **特殊分支** | **奇行种哥布林**——强化剂重度成瘾导致身体/行为异常 | ch012§15 |

**关键出处：**
- ch001§216：「你遇到了两个肮脏的哥布林」
- ch012§15：「【你发现了奇行种哥布林】」
- ch006§12：「【你在下巢哥布林的房屋中度过了一晚，经验值+20】」
