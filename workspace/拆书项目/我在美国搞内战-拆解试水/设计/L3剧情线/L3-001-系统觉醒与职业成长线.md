## 十一、套路分析段（基于当前窗口）

### 11.1 成长节奏范式

本线的核心成长节奏可归纳为 **「积累→瓶颈→突破→展示→整合」** 的五拍循环：

```
积累期（日常/探索/赚钱）→ 瓶颈期（压力/濒死/倒计时）→ 突破期（升级/职业/仪式）→ 展示期（首秀/打脸/神迹）→ 整合期（收束/回血/规划下一阶段）
```

典型应用：
- **L2-001→L2-002→L2-003**：积累（系统认知+经验）→瓶颈（帮派追杀）→突破（智斗逃生+背包解锁+技能融合）
- **L2-004→L2-005**：积累（医疗创业+技能拾取）→瓶颈（枪击濒死）→突破（升级2级+选择巫医）→展示（巫医首秀+战斗首秀）→整合（圣愚人设）
- **L2-008→L2-009**：积累（西蒙驯服+圣遗物）→瓶颈（酒瓶老乔/警察/史蒂夫多重压力）→突破（投名状血契+庇护所建成）→展示（净化之光+史诗任务）→整合（教父宣言+声望系统）

**设计逻辑**：

### 11.4 主角战力维度扩展路径

```
体力维度（身体）: 0.1→1.3（属性点+1）
血量维度（存活）: 5/60→~160/160（属性点+1+升级）
精神力维度（感知）: 0.4→1.8（升级+历史文献发现）
技能维度（能力）: 1个→10+个（拾取+融合+职业转化）
称号维度（身份）: 无→3个（金色1个）
装备维度（外物）: 无→青铜庇护所+圣遗物+手枪
势力维度（人力）: 孤身→二十余人追随+暗世界人脉
经济维度（资源）: 0→17600+刀+弥赛亚硬币
```

**成长哲学**：本线系统成长不是单维度的「数值膨胀」，而是**三维并行**——战力（等级/技能/属性）×资源（经济/装备/材料）×势力（追随者/关系/声望）。三条腿同时走路，使每个阶段的成长感知丰富不单调。
