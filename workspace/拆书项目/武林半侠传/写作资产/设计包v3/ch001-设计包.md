## 2. 爽点设计（爽点层）

### 套路识别

**套路①：穿越者逆境开局**
- **铺垫（压抑）**：现代毕业即失业的文科生，躺在床上辗转反侧
- **触发（意外）**：魂穿醒来被吊在悬崖半空，勒得喘不过气
- **冲突（困境）**：乞丐→生死线边缘→采药童子（最后一条不归路）
- **反转（希望）**：发现演武令金手指+语言通晓→靠能力脱离采药死亡组→升为制药童子
- **叠加反转（隐忧）**：得罪王成→被找茬→冲突未完全化解

**套路②：扮猪吃虎·隐藏底牌**
- **铺垫**：方明在新位置低调少说话
- **触发**：王成多次冷嘲热讽
- **隐忍**：方明"心底有火，却强行忍住了"
- **小反转**：以堂规反制王成→不被情绪裹挟→理性退走

### 情绪弧线

```
(章首) 紧张焦虑 → 化解缓和 → 回忆绝望 → 发现希望 → 振奋 → 怒火暗涌 → 压抑克制(章末)
                                                                              ↑
                                                                       王成挑衅→冷处理
```

### 章末钩子

| 层级 | 内容 |
|------|------|
| L1 情节未决 | 王成约架城东小树林，"是男人的就今晚……敢不敢，给句话"——方明未答应未拒绝 |
| L2 人物关系 | 方明vs王成+背后王印(采药师傅+鹰爪功)的深层仇怨（前身之死）尚未清算 |
| L3 金手指悬念 | 演武令"每个月月圆之时可以开启穿越"——暗示下一章月圆将至 |
