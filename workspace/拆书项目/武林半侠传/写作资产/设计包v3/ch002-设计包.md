## 2. 爽点设计（爽点层）

### 套路识别

**套路①：世界背景铺陈章**
- **铺垫**：前章已暗示回春堂和药王帮
- **展开**：大乾九十九州→玉京→三教五宗→青云宗→大江盟→药王帮→王大善人
- **闭环**：从天下格局落到主角身边的回春堂，层层收束
- **作用**：为后续穿越学武提供清晰的势力版图

**套路②：金手指设定揭秘**
- **铺垫**：演武令在前章仅点明存在和语言通晓
- **揭秘**：月圆穿越、随机抽取、魂穿、时间累积放弃 → 完整规则
- **展示**：已穿越七种武器（失败）→ 拒绝陆小凤 → 第三次抽到鹿鼎记
- **目标**：读者完全理解金手指的机制、限制和风险

**套路③：穿越失败教训**
- **铺垫**：主角期待学会武功纵横
- **反差**：古龙世界水深→三次差点死掉→青龙会见闻→恐怖至极
- **结论**："穿越自有风险，古龙尤须谨慎"
- **对比**：金庸世界"好混多了"——为下章铺垫

### 情绪弧线

```
(章首) 静谧孤寂 → 压抑警惕 → 冷静求知 → 震撼畏惧 → 绝望 → 希望涌现(章末)
                                                    ↑
        偷师识穴→世界观展开→强者回忆=震撼→七种武器=惨痛→鹿鼎记=泪目希望
```

### 章末钩子

| 层级 | 内容 |
|------|------|
| L1 情节未决 | 方明即将穿越鹿鼎记世界——穿越过程未写，留待下章 |
| L2 金手指悬念 | 魂穿规则：三个月时间、不能带物品——如何最大化利用鹿鼎记世界？ |
| L3 情绪钩子 | "鹿鼎记……六个月，足够了！"——读者期待主角在熟悉的世界翻盘 |
