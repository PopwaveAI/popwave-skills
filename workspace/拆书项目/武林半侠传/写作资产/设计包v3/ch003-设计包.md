## 2. 爽点设计（爽点层）

### 套路识别

**套路①：穿越路线取舍决策**
- **铺垫**：鹿鼎记世界已确认，但不知从何入手
- **选项展开**：a) 投靠洪安通(神龙教) b) 加入天地会(陈近南) c) 昆仑山谷挖埋经 d) 拜入武林门派
- **逐项排除**：豹胎易筋丸→造反掉脑袋→太远+绝壁攀不了→只剩门派路线
- **决策**：身家清白+卖田备厚礼→走正规拜师路线
- **爽点**：主角理性不冲动、逻辑闭环、步步为营的计划感

**套路②：入门献祭流**
- **触发**：需要拜师但作为穿越者一无所有
- **献祭**：卖掉父母留下的田地（近百亩）→破财断后路
- **安置**：余田转给老管家作为回报
- **理由包装**："体弱多病，看破红尘"
- **执行**：直奔嵩山少林，剃度出家
- **爽点**：行动果决+断尾求生的魄力+"少林"二字自带安全感+读者期待

**套路③：金手指穿越奇观**
- **五层结构**：青色太阳(主世界)→黄色太阳群→红色火球→白色光河→撞击白色光球(目标世界)
- **色彩递进**：青→黄→红→白，层层减速、落定
- **原文证据**："无尽的虚空当中，耀眼而又灿烂的青色光芒……演武令认准了其中一颗纯白色的光球，猛然撞击了进去"
- **作用**：给金手指赋予仪式感和神秘感，提升逼格

### 情绪弧线

```
(章首) 冷静分析 → 确认可行(微喜) → 穿越奇观(震撼) → 附身清朝(小失落) → 理性推演 → 果决行动 → 剃度尘埃落定(期待)
                                                                                                  ↑
                                                                                  卖地→上路→少林→落发
```

| 层级 | 内容 |
|------|------|
