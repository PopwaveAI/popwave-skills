## 2. 爽点设计（爽点层）

### 套路识别

**套路1：挟持人质强闯禁地套路（延续ch007）**
- 起：以韦小宝为盾牌，用其身份逼迫寺僧让步
- 承：用退开二十丈的距离确保暗器无效，步步紧逼
- 转：一路畅通无阻直达藏经阁，运用对地形的熟悉
- 合：成功进入藏经阁并关闭大门，计划达成

**套路2：虚张声势诈出底牌套路**
- 起：怀疑藏经阁内有隐藏高手
- 承：没有确凿证据，但方明决定讹诈对方
- 转：用铜板断匕的威胁动作逼迫寺方交人
- 合：三个枯瘦老僧竟然真的现身！印证了主角的推测，也埋下了如果没诈出来可能更危险的悬念

### 情绪弧线

轻松拿捏→强硬对峙→步步紧逼→谈判周旋→虚张声势→背后冷汗→虚惊一场→得偿所愿
（紧张感持续攀升，在三个老僧出现的瞬间达到峰值，然后回落为"得逞"的满足感）

### 章末钩子

**L4 成果悬念**：方明套上宝衣、手持匕首、站在藏经阁中，即将开始扫荡秘籍。"方明的眼睛只看武功，自然是一目十行地扫了过去"——读者期待主角会拿到什么绝世武功。

## 4. 感官与画面（感官层）

### 环境基线6维

| 维度 | 设定 |
|------|------|
| 时间 | 白日→进入藏经阁后 |
| 空间 | 迎客亭外（开阔）→寺内路径（随行）→藏经阁外湖（水景）→藏经阁内（封闭书架） |
| 光线 | 室外自然光→藏经阁内偏暗（有驱虫药味的环境） |
| 声音 | 狮子吼、众僧喧哗→机关"咯咧咔嚓"声→匕首入鞘声→门扉关闭声 |
| 气味 | 藏经阁内"一股驱虫的药味" |
| 触感 | 匕首冰凉贴颈、宝衣触感（柔软刀枪不入） |

### 氛围渲染要点

- **对峙压迫感**：方明孤身一人对百余名僧众，但手握关键人质，形成不对称平衡
- **藏经阁的环境意象**：湖心阁楼、木桥连接、匾额高悬——象征知识与禁区的双重属性
- **三个老僧出场**："骨瘦如柴、仿佛下一步就要摔倒"——反差式高手出场法，外表衰朽但方明"背后冷汗直冒"
- **门扉关闭后的松弛感**：从室外紧张对峙到关上门后的复盘，节奏张弛有度
