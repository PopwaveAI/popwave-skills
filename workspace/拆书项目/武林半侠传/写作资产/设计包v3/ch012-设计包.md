## 2. 爽点设计（爽点层）

### 套路识别
1. **【深山闭关/顿悟突破】** — 霍山练功顿悟，白气如剑，实力增长
2. **【自我审视/三要素模型】** — 资质+机缘+毅力的武学成长框架
3. **【追逃博弈/猎犬遭遇】** — 猎犬追踪→杀犬→撤退的经典追逃节奏
4. **【师门回忆/情感驱动】** — 对少林寺生活的怀念，驱动向往更好环境

### 四段式结构

| 阶段 | 对应事件 | 作用 |
|------|----------|------|
| **铺垫** | 练功顿悟（#1），反思缺师长（#2），回忆少林（#3） | 提升实力+心理酝酿 |
| **起势** | 自我审视三要素（#4），演练拳法（#5），啃干粮决心搞钱（#6） | 节奏放缓，积累情感 |
| **转折** | 猎犬追至（#7），杀犬判断追兵近（#8） | 节奏骤变，危机打断日常 |
| **收束** | 封羽率人赶到（#9），放烟花/追入深山（#10） | 追兵逼近，悬念升级 |

### 情绪弧线

```
  ↑紧张感
   |                封羽赶到
   |               / 放烟花追入
   | 猎犬追至    /
   | / 杀犬撤退 /
   |/
   |
   | 练功顿悟——反思少林——自我审视——练习日常
   |____________________________________→ 进度
   平静修炼→危机突降→追兵合围
```

### 章末钩子
- **L4（强烈行动悬念）**：封羽率人追入霍山深山，儿子与徒弟的苦求无效——追兵会追到什么地步？方明能否甩脱？霍山深处有何危险？

## 4. 感官与画面（感官层）

### 环境基线（6维）

| 维度 | 状态 | 原文关键词 |
|------|------|-----------|
| 视觉 | 虫鸣山寂、白雾弥漫、古松虬结、青苔黑石 ↔ 恶犬扑出、血洒长空 | "虫鸣山寂，鸟飞林幽" "根须虬结的巨大古松之下" |

### 战斗动作（六段式）—— 杀犬战

| 段 | 描写 | 效果 |
|----|------|------|
| 预警 | "方明的耳朵突然微微一动" | 内功听觉觉醒的展示 |
| 接敌 | "一道细腰恶犬就从草丛当中扑了出来" | 突然袭击 |
| 格挡 | "右手一记降龙伏虎直接轰击在恶犬头顶" | 伏虎拳杀招 → 克制恶犬 |
| 致命 | "左手的匕首就捅入了恶犬的小腹当中" | 补刀致命 |
| 撤离 | "方明一闪身，血洒长空，之前的恶犬已然毙命" | 快速干净 |
| 善后 | "一把抓起包裹，蜻蜓点水提纵术……飞快地消失不见" | 立即转移，不留战场 |

### 氛围渲染
- **练功氛围**：寂静、幽玄、平和（物我两忘的修炼状态）
- **危机氛围**：草木翻折、恶犬狂吠、血腥味突入——宁静被骤然打破
- **追兵氛围**：寒凉的意志、不达目的不罢休的压迫感
- **全章基调**：宁静→骤然紧张→收束在追兵压境，为下一章铺垫

### 关键画面定格
- 方明白气如剑吐出的修炼画面
- 松鼠被方明吓到、抱松果炸毛逃窜的萌趣画面
- 方明泪流满面啃硬面饼的苦中带乐画面
- 恶犬血溅草丛、方明一闪而过的剪影
- 封羽仰天放烟花，声光信号穿破霍山云雾的画面
