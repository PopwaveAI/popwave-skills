## 2. 爽点设计（爽点层）

### 套路识别
1. **追逃剪羽** —— 方明被追捕，但反向伏击捕快（游击反杀）
2. **碾压打脸** —— 南宫倾城一竹枝秒杀后天大成封羽（实力碾压）
3. **神秘高手登场救场** —— 吊桥效应式引入强者，触发收徒线

### 四段式（对本章）
- **铺垫**：霍山环境压抑，追兵紧迫，生存压力大
- **关键**：方明显露果断狠辣（主动伏击、杀子不悔）
- **高潮**：南宫倾城登场→秒杀封羽（实力展示/身份悬念）
- **回落**：方明与南宫对话，发现麻烦更大，留下勾子

### 情绪弧线
压抑（被追杀）→ 紧张（伏击）→ 快意（反杀成功）→ 震撼（秒杀）→ 警惕（新威胁出现）

### 章末钩子
- **L3（角色悬念）**：南宫倾城这样的大高手看上了方明——是灾是福？方明甩不掉她，接下来会被怎样对待？

## 3. 角色与人设（角色层）

### 登场角色行为锚定

**方明**
- 行为：主动伏击猎杀追兵，搜尸（利刃割喉→擦匕首→搜物资），毫不手软
- 心理：理性评估局势，冷漠权衡生死，"反正把对方得罪死了，再多加一点怨恨也没区别"
- ?? 关键对白："大不了慢慢杀，总是杀得完的" —— 语气：冷静中带狠辣，潜台词：老子不在乎仇恨值，杀到你怕
- ?? 关键对白："感谢姑娘为我除此心头大患"（马上改口示好）—— 语气：识趣，能屈能伸，潜台词：这人太强，先稳住

**封羽**
- 行为：发现儿子尸体时情绪失控（老泪纵横、鹰爪抓树），但随后迅速恢复警觉
- 关键对白："小畜生！！！我封羽从此与你不同戴天！！！"—— 语气：暴怒悲恸
- 临终对白："气与意合……好！没想到我老羽居然死在你这种高手之手，也不算冤枉了……"—— 语气：苦笑认命，大反派也有气度

**南宫倾城**
- 行为：立于树梢上下起伏——"身似轻鸿"轻功展示；一截竹枝杀封羽
- ?? 关键对白："大力鹰爪门？一个三流的门派，很了不起么？"—— 语气：轻蔑冷笑，潜台词：老娘背景比你大得多
- ?? 关键对白："有趣！有趣！你一个十五六岁的孩子，说话怎么好像个小大人一样？"—— 语气：戏谑好奇，潜台词：这小子不简单，姐姐有兴趣了

### 角色关系
- 方明vs封羽 → 不死不休（杀子之仇）
- 南宫vs封羽 → 早已结仇（分舵被破）
- 南宫→方明 → 好奇/兴趣（收徒伏笔）

## 4. 感官与画面（感官层）

| 触觉 | 刀锋冰凉、夜风 |
| 味觉 | — |
| 体感 | 压抑、危险、杀气弥漫 |

### 战斗动作六段式（南宫倾城vs封羽）
1. **对峙发现**：封羽察觉树上有人，瞬间暴退，漆黑经脉蔓延双爪
2. **展示/亮相**：封羽抬头看到一抹"劲装倩影"——颜值描写切入打斗
3. **试探周旋**：封羽报师门求饶，南宫冷笑
4. **出手**：一道洁白剑光飞出（实际是竹枝）
5. **秒杀/终结**：封羽全身脆响、真气外泄、鹰爪铁布衫破
6. **余韵**：封羽苦笑认命→断气，南宫幽幽叹气离去

### 氛围渲染
- 前段：压抑恐怖（霍山禁区描述：百不存一、恐怖的凶兽）
- 中段：冰冷残酷（割喉/搜尸，毫无温情的生存法则）
- 后段：神秘+震撼（绝顶高手登场，声音如清泉/身影如飞仙，与之前的凶险形成巨大反差）
