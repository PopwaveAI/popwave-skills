## 2. 爽点设计（爽点层）

### 套路识别
1. **被动投靠** —— 主角被强制带入组织，没有选择权
2. **遗孤传承** —— 灭门后残存高手寻找火种/收徒重建
3. **宗门内景** —— 新环境展示（道观、弟子、日常）
4. **天敌预告** —— 引入世界观终极威胁（青云宗）

### 四段式（对本章）
- **铺垫**：方明被南宫带走，空中飞行的震撼感
- **关键**：抵达还真观，接触弟子（崔凌/小慕容）——"草台班子"初印象
- **高潮**：南宫坦白——玄真道已灭，仇人是青云宗；方明被迫入伙
- **回落**：方明苦笑接受——"可真是久仰大名了"

### 情绪弧线
震惊（被制住）→ 新鲜（空中飞行）→ 好奇（道观环境）→ 微妙（接触同门）→ 压抑+无奈（得知真相）

### 章末钩子
- **L4（世界观）**：青云宗是天下五宗之一，玄真道与其结仇——主角刚入江湖就面临顶配敌人，未来怎么走？

## 3. 角色与人设（角色层）

### 登场角色行为锚定

**方明**
- 行为：被制住后不反抗、默不作声（理智求生）；内心疯狂吐槽但嘴上乖巧
- ?? 关键对白："总有一天，我也要成就这等武道修为，方不复穿越一场！"—— 语气：内心独白，激昂坚定，潜台词：哪怕现在被像小鸡一样提着，我也不认输
- ?? 关键对白："我可以不加入么？"—— 语气：弱弱试探，明知故问，潜台词：垂死挣扎一下

**南宫倾城**
- 行为：行事利落（说制住就制住）、主导一切（"送一场缘法"）、权威感
- ?? 关键对白："你说呢？（背后的雪白长剑发出嗡鸣）"—— 语气：反问+威胁，潜台词：你有得选？/老娘这把剑不是吃素的

**崔凌**
- 行为：自来熟、笑脸相迎，商人做派，做引言人和"背景板"
- 关键对白："嘻嘻……小弟崔凌……以后还请多多照顾了"—— 语气：亲热随和

**小慕容**
- 行为：蹲地摸额头雪雪呼痛（天然呆），注意力转移极快（小孩心性）
- 关键对白："南宫姐姐说来了个新人，小慕容也想尽早过来见见师兄……"—— 语气：天真无邪

### 角色关系
- 方明→南宫：表面师弟→师姐，实际强者→弱者，被迫顺从
- 方明→崔凌/小慕容：同门，地位平等，但崔凌早入门一年多
- 玄真道vs青云宗：灭门之仇，实力悬殊

## 4. 感官与画面（感官层）

### 环境基线（6维）

| 维度 | 设定 |
|------|------|
| 视觉 | 两耳大风灌入/树木飞速后退/黑夜繁星/峡谷下落/还真观门匾龙飞凤舞/五进院落/红木供桌/青烟中"玄真"画轴 |
| 听觉 | 风声灌耳、两耳大风呼呼、南宫银铃般声音、道观门"吱呀"打开 |
| 嗅觉 | 香风阵阵（南宫）、供桌青烟 |
| 触觉 | 脖子一酸（被制）、手脚被寒风吹得冰凉 |
| 味觉 | 晚饭青菜豆腐+肥鸡 |
| 体感 | 失重（下落）、悬空、微妙（被美女提着走）|

### 空中飞行段落处理（替代战斗）
1. **被制住**：脖子酸麻，动弹不得
2. **升空**：脚下土地化为倒影，大风灌耳
3. **下落**：衣袖被劲风鼓胀，繁星仿佛就在身边
4. **抵达**：手脚冰凉，但站桩稳住
5. **落地观感**：打量环境，看到"还真"二字

### 氛围渲染
- 前段：空中旅行的飘渺/自由感（与ch013的压抑追杀形成对比）
- 中段：神秘感（道观突然出现）→ 日常感（崔凌絮叨介绍、小慕容天然呆）
- 后段：沉重压迫（南宫坦白真相，青云宗四个字砸下来）
