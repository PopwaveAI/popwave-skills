# 设计包 — ch060「妙手」

## 事件链

| 序号 | 事件 | 触发 | 结果 |
|:--:|------|------|------|
| 1 | 方明与色使继续激战：铁指禅功对弹指神通、大摔碑手对腿影 | 上章指法对决延续 | 色使右腿被大摔碑手拍断，倒飞吐血 |
| 2 | 色使死中求活，衣袖射出银芒暗器暴袭 | 方明掌风笼罩 | 方明出弯刀以破戒刀法击溃漫天暗器 |
| 3 | 方明一刀将色使击飞，命令白衣人押入三号秘窟 | 刀法决胜 | 色使被俘——快活王色使落地 |
| 4 | 方明认出现场女子即朱七七，提出为她恢复容貌 | 战斗结束，转向朱七七 | 朱七七睁大双眼半天不敢眨——同意了交易 |
| 5 | 方明以剪刀、布匹、烈酒、干醋等工具为朱七七恢复容貌 | 双方达成交易 | 原本的无盐丑女恢复为体态风流的七七小姐，方明确认继承王怜花易容术 |
| 6 | 方明送朱七七到荒祠，见沈浪、熊猫儿、金无望 | 手术完成，方明好人做到底 | 方明入祠与众人见礼 |
| 7 | 方明与沈浪、金无望、熊猫儿三人一一见礼，言语间略带挑衅 | 沈浪奇怪方明似乎认识他 | 方明以"久仰"之言敷衍，但态度傲然 |
| 8 | 方明掀开车帘展示朱七七 | 称久仰不如见美人 | 熊猫儿大惊——他魂牵梦萦的姑娘居然在此！沈浪掠身上前 |
| 9 | 沈浪解开朱七七穴道，朱七七反手打沈浪耳光，泪诉委屈 | 重逢瞬间 | 朱七七又红着眼指方明——方明轻嗅手指动作令她脸红跑开 |
| 10 | 方明大笑退场，轻功远去，半空躬身行礼如名伶退场 | 目的达成 | 金无望欲追被沈浪拦住，轻功追不上 |
| 11 | 沈浪决定：先去吃饭，再逐个查探方明涉及的地方 | 朱七七讲述经历后 | 以退为进——引而不发 |

## 爽点层

| 爽点 | 强度 | 说明 |
|------|:----:|------|
| 刀法碾压色使 | ★★★★★ | 弯刀出鞘后弧月刀光劈开漫天银芒——"一刀在手，鬼神不留"——武力的具象巅峰展示 |
| 认丑女为朱七七 | ★★★★ | 所有人都认不出，只有方明通过交易中的条件认出——智商/洞察力碾压全场 |
| 妙手回春恢复容貌 | ★★★★ | 手术过程虽一笔带过但"从此丑女变佳人"的反差极具爽感——方明王怜花天赋全盘继承 |
| 掀车帘展示朱七七的重磅炸弹 | ★★★★★ | 熊猫儿大惊、沈浪掠身、朱七七打耳光——高潮连续炸裂，情感冲突集中爆发 |
| 方明摸鼻子暗示暧昧 | ★★★★ | "右手拇指与食指磨蹭，放在鼻下轻嗅"——一指暗示之前手术中的肢体接触，朱七七尖叫跑开——暧昧张力拉满 |
| 名伶退场 | ★★★ | 方明身体被无形丝线牵扯倒飞，半空躬身行礼——轻功炫技+礼仪风度并存，反派潇洒至斯 |
| 沈浪说"先去吃饭" | ★★★ | 暴风雨前的平静——大战之前先吃饭，以退为进的英雄气度 |

## 角色层

| 角色 | 本场定位 | 关键表现 |
|------|----------|----------|
| 方明（王怜花） | 武力碾压者 / 易容修复者 / 退场艺术家 | 弯刀破戒刀法击败色使、以妙手为朱七七恢复容貌、将朱七七送到沈浪面前并留下暧昧暗示扬长而去——全方面的能力展示：武功、医术、风度和布局 |
| 江左司徒（色使） | 败者 / 被俘者 | 被打断腿、射银芒暗器、被弯刀一刀削飞、押入秘窟——快活王座下色使首次落败于方明 |
| 朱七七 | 被救者 / 情绪宣泄者 | 从丑女恢复为七绝女子、重逢沈浪先打后哭、对方明的暧昧暗示羞怒——三种情绪翻转 |
| 沈浪 | 惊愕者 / 克制者 | 初见方明感到惊奇但克制、解朱七七穴道、拦金无望追方明（"光看来时轻功便知追不上"）、以"先去吃饭"作为调查的冷静策略 |
| 熊猫儿 | 震惊者 / 中间人 | 掀车帘发现美人正是梦中人——情绪落差最大；但性格豪爽不记仇，仍介绍方明给沈浪 |
| 金无望 | 观察者 / 行动者 | 全程冷眼旁观，但朱七七提到方千里时一惊；想追方明被沈浪拦下 |
| 白衣人（无名） | 执行者 | 无声无息出现，奉命押色使人入三号秘窟 |
