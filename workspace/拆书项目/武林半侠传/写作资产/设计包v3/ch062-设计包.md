## 二、爽点设计（爽点层）

### 套路识别
**套路A：灭门勘察** → 详见套路库/灭门勘察.md
**套路B：恐怖势力侧写** → 详见套路库/恐怖势力侧写.md

### 情绪弧线
- **起点**：魔刀门江湖传闻铺陈，渲染恐怖神秘 → 压抑好奇
- **推进**：风雪路途 → 寂静庄园 → 血腥惨状 → 逐级升级的恐怖感（朱七七的恐惧作为情绪温度计）
- **高潮**：齐智首级眉心刀痕 + 沈浪"地狱刀法"的评价 → 恐怖+震撼双重爆发
- **终点**：沈浪决意探查魔刀门，熊猫儿慷慨响应 → 压抑后的决心感 + 期待感

### 爽点机制
1. **信息差爽点**：读者早已知魔刀门恐怖，但具体场景（仁义庄惨状）通过沈浪视角逐步呈现，已知与未知叠加放大冲击
2. **数值爽点**：仁义三老（武林名宿）被数招秒杀 → 建立魔刀门战力标尺
3. **轻度打脸**：朱七七"偏要进" → 被惨状吓到呕吐 → 任性受挫
4. **英雄塑造爽点**：沈浪在恐怖现场保持绝对冷静、细致推理、决绝断然 → 智勇双全的人设强化

### 章末钩子
- **类型**：使命感钩子 + 组队钩子
- **内容**：沈浪说"相信最后的真相，一定会令我们大吃一惊的"
- **强度**：L3（中等）
- **预期回收章**：ch063

## 四、感官与画面（感官层）

### 环境基线

| 维度 | 状态 |
|------|------|
| **时间** | 大雪严冬，黄昏偏入夜时分 |
| **天气** | 大雪威寒，天地肃杀 |
| **光线** | 雪地反光明亮 → 庄内晦暗阴沉 |
| **空间** | 开阔雪原 → 死寂庄园大门 → 昏暗大堂 → 血腥后堂内室 |
| **气味** | 雪花清冷 → 熊猫儿嗅出血腥 → 浓稠血滩的尸臭铁锈味 |
| **声音** | 急促马蹄声 → 风雪呼号 → 死寂（无人声） → 朱七七呕吐干呕声 → 沈浪低沉的话语 |

### 动作场景（无正面战斗——勘察节奏）

1. **冻结抵达**：四骑止步，风雪中默默凝视死寂庄园
2. **冲击开门**：朱七七推开庄门，黑血断肢扑面（视觉暴力峰值）
3. **推进侦查**：沈浪缓步向内，逐层发现更多尸体与痕迹
4. **揭示恐怖**：齐智首级 + 眉心刀痕特写（恐怖顶点）
5. **情感收束**：驱赶朱七七的爆发 + 二人组队的决意（情绪回落）

### 氛围渲染要点
1. 风雪与死寂对撞——天地漠然与人世惨剧形成强烈反衬
2. 朱七七从倔强→恐惧→受伤→含泪离去，作为读者情绪的"导体"
3. 刀痕由金无望评价为"地狱带出来的刀法"——将魔刀门恐怖具象化
4. 沈浪说"我本是浪子"时的苦笑——英雄的孤独底色
