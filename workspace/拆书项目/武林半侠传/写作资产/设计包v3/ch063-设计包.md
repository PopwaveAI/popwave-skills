## 二、爽点设计（爽点层）

### 套路识别
**套路A：少女情窦转向与觉醒** → 详见套路库/少女情窦转向与觉醒.md
**套路B：雅士社交博弈（品酒论交）** → 详见套路库/雅士社交博弈.md
**套路C：尸体闯入打断祥和** → 详见套路库/尸体闯入打断祥和.md

### 情绪弧线
- **起点**：朱七七雪中哭泣——失落、伤感、卑微
- **推进①**：抱金无望→被冷拒→被点醒→觉醒（情感从低谷缓慢爬升）
- **推进②**：品酒论交的雅致从容（氛围舒展开来）
- **转折**：沈浪携尸登场（氛围骤变）
- **终点**：七处刀痕的谜团 + 魔刀门压迫感（悬疑高悬）

### 爽点机制
1. **情感成长爽点**：朱七七从"自怨自艾被抛弃"→"我要去帮他"——角色的主动性觉醒
2. **识见展示爽点**：方明品酒辨酒，展现远超常人的知识广度→读者获得"主角强"的满足感
3. **恐怖具象化爽点**："魔刀门"三个字让群雄瞬间清场——恐怖在具体场景中落地
4. **震撼性爽点**：武林七大高手之一的徐若愚被杀——用高位者的死亡来制造冲击

### 章末钩子
- **类型**：悬疑分析钩子（未完成的伤口分析）
- **内容**：徐若愚尸体上七处要害刀痕——但一刀如何同时造成七处伤口？
- **强度**：L3（中等偏高）
- **预期回收章**：ch064（熊猫儿揭示"只挥了一刀"）

## 四、感官与画面（感官层）

### 环境基线

| 维度 | 状态 |
|------|------|
| **时间** | 雪后白天，午时前后 |
| **天气** | 大雪纷飞 → 雪止风寒 |
| **光线** | 雪地刺目反光 → 悦宾楼暖色灯光 |
| **空间** | 开阔雪野 → 繁荣城镇 → 热闹酒楼二层 |
| **气味** | 清冷雪气 → 酒楼菜香酒香 → 尸体血腥突然冲入 |
| **声音** | 风声 + 朱七七哭声 → 酒楼喧哗 → 品酒谈笑声 → "噔噔噔"急促脚步声 → 尖叫惊哗 → 瞬间死寂 |

### 动作场景（无正面战斗）

本章核心为**氛围对撞**：
- **阴→晴**：雪地哭泣（冷/孤独）→ 城镇繁华（暖/热闹）
- **静→动**：品酒对饮的静止优雅（方明与韩伶）→ 沈浪携尸闯入的急促动静
- **满→空**：群雄满座 → "魔刀门"三字 → 整层清空 → 只剩核心角色

### 氛围渲染要点
1. **温度控制链**：朱七七"比雪更冷的是人心" → 金无望"冰冷的大手竟也有了炙热的温度" → 悦宾楼暖酒 → 尸体登场后骤然冰冷
2. **品酒场景的"静止"与尸体闯入的"运动"并置**：方明在尸体旁边继续饮酒——两个世界在同一空间的诡异重叠
3. **"魔刀门"的清场效应**：不写人跑得多快，只写"眨眼便一干二净"
4. **徐若愚的英俊面容**写在与死尸的对比中——"活着时也是一位美男子"
