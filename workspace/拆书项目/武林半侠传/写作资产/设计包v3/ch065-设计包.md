## 二、爽点设计（爽点层）

### 套路识别

**套路1：惊世赌约套路**
- 类型：信息差·打脸·豪气
- 结构：以小博大（十万两黄金）→以人命为筹码（预知死人做不了帮主）→信息碾压对手
- 核心爽点：方明掌握暗杀二老的信息差，范汾阳以为三选一赢面大，实则必输

**套路2：身份反转套路（白飞飞）**
- 类型：社交·揭露
- 结构：白飞飞示弱（怯生生）→实际深不可测（打倒金无望+韩伶）→双方各怀鬼胎
- 核心爽点：善良外表下的阴毒，与方明棋逢对手

### 情绪弧线

起点（镇定）：方明淡然提出赌约，范汾阳自信三选一
推进（悬念）：方明暗示「死人是做不了帮主的」
高潮（惊骇）：沈浪等人意识到单弓、欧阳轮已死
终点（余韵）：方明潇洒离去，留下满堂震惊

### 爽点机制

| 类型 | 内容 | 强度 |
|------|------|------|
| 信息差 | 方明预知方明已杀二老，范汾阳全然不知 | ★★★★ |
| 数值 | 十万两黄金赌注，气魄骇人 | ★★★★ |
| 打脸 | 范汾阳以为三选一必胜，实为必输之局 | ★★★★ |
| 情绪 | 白飞飞外表弱女实则毒女，反差冲击 | ★★★ |
| 语言 | 「天魔七杀式」名字的公布与气势渲染 | ★★★★ |

## 四、感官与画面（感官层）

### 环境基线

| 维度 | 描述 |
|------|------|
| 时间 | 日间（酒楼场景） |
| 天气 | 未见特别描述，推测晴朗 |
| 光线 | 室内明亮（酒楼） |
| 空间 | 酒楼雅间/大厅；切换至荒野（方明驻地） |
| 气味 | 白飞飞袖间磷火味、白纱芬芳 |
| 声音 | 方明掷地有声的对话、白飞飞银铃般的笑声、磷火「嗤嗤」声 |

### 动作场景（方明vs白飞飞）

诱敌→白飞飞媚笑中射出磷火
规避→方明「叹息之间，一抹冷月般的光华绽放」
交锋→刀光斩碎白纱，「片片白纱仿佛蝴蝶一样翩翩乱舞」
收场→白飞飞撤退，留下婚约之言

### 氛围渲染要点

- 赌约时的紧张对峙气氛（方明淡然vs范汾阳自信→惊骇）
- 白飞飞出场时柔弱外表与实际深不可测武功的反差
- 方明驻地夜晚的幽暗氛围，白飞飞如幽灵般出场
- 「天魔七杀式」名字渲染的恐怖神秘色彩
