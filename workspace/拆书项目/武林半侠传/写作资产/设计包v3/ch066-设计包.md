## 二、爽点设计（爽点层）

### 套路识别

**套路1：死而复生证据套路**
- 类型：信息差·打脸·推理
- 结构：隐蔽布局（沈浪盗尸）→极限施压（在最关键时刻打出）→心理击溃（左公龙惊骇自爆）
- 核心爽点：用死者尸体当作「证人」，让凶手自己吓自己

**套路2：凶手自爆套路**
- 类型：打脸·心理战
- 结构：设局逼人→陷入惊骇→心神失守→不打自招
- 核心爽点：沈浪不费一兵一卒，仅靠巧妙布局就让左公龙在千余人面前自曝罪行

**套路3：魔刀反噬套路**
- 类型：设定揭露·武道规则
- 结构：展现恐怖威力→看似无敌→揭示代价（反噬）→拔高设定（此刀非凡人可用）
- 核心爽点：虽有神刀，修炼者必先遭反噬，为「天魔七杀式」设定增添敬畏

### 情绪弧线

起点（压抑）：丐帮大会阴云密布，左公龙看似掌控局面
推进（悬疑）：熊猫儿焦等，沈浪迟迟未到
高潮（惊悚→震撼）：沈浪以尸体作证，左公龙自爆→震撼比武
转折（震惊）：原来刀法有反噬，左公龙是被自己的刀杀死
终点（余思）：沈浪承认「此战是我败了」

### 爽点机制

| 类型 | 内容 | 强度 |
|------|------|------|
| 信息差 | 沈浪盗尸+操控尸体作证，所有人浑然不觉 | ★★★★★ |
| 打脸 | 左公龙前一秒还义正言辞，后一秒自爆凶手 | ★★★★★ |
| 数值 | 乾坤第一指vs天魔七杀式的顶级对决 | ★★★★ |
| 情绪 | 二老尸体「活过来」指认凶手的惊悚反转 | ★★★★★ |
| 设定 | 魔刀反噬规则首次披露 | ★★★★ |

| 类型 | 内容 | 强度 | 预期回收章 |

## 四、感官与画面（感官层）

### 环境基线

| 维度 | 描述 |
|------|------|
| 时间 | 隆冬时分，白日 |
| 天气 | 冬日阳光，积雪未化 |
| 光线 | 日光下的稻田竹棚，明亮中有萧瑟 |
| 空间 | 东郊稻田→碗口粗毛竹搭成的四方竹棚→竹棚顶部（决战地） |
| 气味 | 烈酒（烧刀子）气味、稻田干草味 |
| 声音 | 群丐喧哗→左公龙洪亮唱声→高小虫嬉笑声→竹棚上刀兵交击声→鲜血滴落声 |
| 触觉 | 冬日寒风、积雪冷意、烈酒入喉的灼热感 |

### 动作场景（沈浪vs左公龙）

1. **诱敌**：左公龙飞身上竹棚欲逃，沈浪已等在那里
2. **开局**：左公龙冷喝中黑色长刀出鞘
3. **魔刀展威**：「黑色的闪电般」一化为七，形成刀之炼狱
4. **反制**：沈浪在间不容发之际，两根手指点杀而出
5. **终结**：兵刃断裂声片片响起，左公龙的刀片片断开
6. **败亡**：左公龙如烂泥般倒下，筋骨寸寸断裂
7. **揭露**：沈浪指出「他出的只有半刀」，真正的天魔七杀式如果完整施展，胜负未知

### 氛围渲染要点

- 丐帮大会的粗犷江湖气息：稻草、积雪、毛竹棚、捉虱子的乞丐
- 左公龙登场时道貌岸然、声震全场的压迫感
- 沈浪驾马车入场时全场屏息的紧张感
- 尸体「活过来」指认凶手的惊悚反转
- 竹棚之巅的巅峰对决：魔刀vs乾坤第一指
- 战后熊猫儿摸到左公龙「一具破麻袋」般的尸体时的不寒而栗
