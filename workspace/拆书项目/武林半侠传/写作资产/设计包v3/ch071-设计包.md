## 二、爽点设计（爽点层）

### 套路识别
- **高手对决**：沈浪vs方明（王怜花），碾压式胜负
- **魔刀修炼体系揭秘**：方明降服天魔七杀式的完整心路
- **爱恨纠葛的张力**：白飞飞对沈浪又爱又恨的极端行为
- **刀意顿悟**：战后领悟刀意之境，入画显道

### 情绪弧线
| 阶段 | 情绪 | 内容 |
|------|------|------|
| 起点 | 悬疑 → 敬畏 | 方明揭开魔刀修炼背后"以人试刀"的血腥真相 |
| 推进 | 紧张 | 沈浪被武道之心折服，决斗一触即发 |
| 高潮 | 震撼 | 天魔七杀式一刀碾压沈浪天外飞仙之剑 |
| 终点 | 唏嘘 / 回味 | 白飞飞放走沈浪，婚礼成悲剧 |

### 爽点机制
| 机制 | 说明 |
|------|------|
| 强者揭秘 | 方明以淡然口吻揭露"百余高手试探""七大刀奴"的布局，展示幕后操控者的绝对智商 |
| 碾压式战斗 | 沈浪已使出极限之剑，却被一刀碾压——实力差距带来的压迫感爽点 |
| 武道宣言 | "天生万物，以我为主，为我所用，我又怎会去做区区一招刀法的奴隶"——霸气宣言 |
| 刀意顿悟 | 战后突破+入画，金手指兑现的快感 |
| 爱恨张力 | 白飞飞边打边流泪、婚礼释放——复杂情感带来的虐心爽点 |

### 章末钩子
| 类型 | 内容 | 强度 | 回收章 |
|------|------|------|--------|
| 悬念钩子 | 白飞飞消失于黑暗，沈浪重获自由但武功全失 | L4 | ch072 |

## 四、感官与画面（感官层）

### 环境基线

| 维度 | 描述 |
|------|------|
| 时间 | 白天（无精确时刻） |
| 光线 | 室内光线（古室），日耀于室（决斗时） |
| 空间 | 古室（对话+决斗）→ 牢房（沈浪被囚）→ 兴龙山麓（释放） |
| 声音 | 剑作龙吟 → 弯刀出鞘声（呛！）→ 刀剑交击（轰！）→ 鞭子破空声 |
| 气味 | 硫磺与血腥（七杀修罗图） |
| 温度 | 无直接描写 |
| 触感 | 沾盐水的鞭子抽打、热泪滴落 |

### 动作场景（决斗时刻）

> 「沈浪古铁剑入手，以指弹剑，剑作龙吟。龙吟不绝，长剑也化为神龙，一剑刺了过去。这一剑夭娇如神龙，迅急却如闪电，剑气纵横，日耀于室，剑法正如其人，潇洒，灵秀，不可方物。」

> 「刀光，耀眼到极点的刀光，带着恐怖的杀气呼啸而出，将之前的满堂剑气都盖了下去。如果说沈浪的剑是仙子的话，那方明的刀却是来自地狱九幽的恶鬼！」

> 「一刀挥出，宛若地狱临凡！众鬼索命！摄魂杀魄！」

### 氛围渲染

- **对决前**：剑拔弩张的静谧张力，古铁剑的龙吟划破沉默
- **对决中**：「天外飞仙」与「地狱恶鬼」的意象对立——剑是仙，刀是鬼
- **对决后**：白飞飞鞭笞沈浪时的「热泪缓缓落到脸上」——从刀剑的冷硬切换到情感的炙热
- **释放场景**：喜服如血染，黑暗吞没白飞飞的身影
