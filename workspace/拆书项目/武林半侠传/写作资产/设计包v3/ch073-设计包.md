## 二、爽点设计（爽点层）

### 套路识别
- **运功疗伤（经典武侠桥段）** + 不占便宜的绅士风度
- **同生共死后的信任升级**：从任务关系到战友关系
- **世界观解锁**：武道境界的系统讲解
- **后台升级**：背后从无到有冒出宗师级靠山

### 情绪弧线

| 阶段 | 情绪 | 内容 |
|------|------|------|
| 起点 | 担忧 | 方明赴约，感知危险 |
| 推进 | 专注 | 为南宫倾城疗伤，近距离接触先天之体 |
| 高潮 | 震惊+安心 | 南宫倾城的师父竟是宗师——有靠山了 |
| 终点 | 温暖期待 | 小慕容的回忆与玄真经的许诺，同门情谊加深 |

### 爽点机制

| 机制 | 说明 |
|------|------|
| **疗伤亲密接触** | 方明撕开南宫倾城肩头衣衫运功——经典武侠福利桥段，同时展现医术与菩提心法价值 |
| **情报交换·互有胜负** | 南宫倾城虽重伤但阴老也中了凝霜剑气——谁也没亏 |
| **靠山揭露** | "我师父已是宗师"——从亡命之徒到有宗师撑腰，安全感的飞跃 |
| **功法画饼** | "待诸事完结传你《玄真经》"——给读者未来的期待 |
| **小慕容的彩蛋** | 天然呆的小师妹得了玄真经传承——天才养成的温情 |

## 四、感官与画面（感官层）

### 环境基线

| 维度 | 描述 |
|------|------|
| 时间 | 白天（方明清晨赴约）→傍晚（南宫倾城醒来） |
| 光线 | 洞口昏暗→傍晚暮色 |
| 空间 | 洋河郡城（外界）→悬崖下布满青苔的岩石→荆棘掩盖的漆黑洞口→空无一物的小洞窟 |
| 声音 | 白光破空（叮！）→人体倒地声→烤兔子的滋滋声 |
| 气味 | 食物香气（烤兔子、水壶）→血腥味（渗血的肩头） |
| 温度 | 微凉（洞窟内温度）→南宫倾城身体冰凉（失血+中毒） |
| 触感 | 雪白凝脂（肩头肌肤温软）→冰片（毒物）→血柱喷射的温热 |

### 动作场景（疗伤）

> 「将南宫倾城抱起，方明看了看她还在渗血的肩膀，眉头又是一皱。手上微微用力，已经将她肩头的衣衫撕裂，露出了雪白的凝脂与三个漆黑的血洞。」

> 「一蓬白气在方明头顶蒸腾，已经化气大成的菩提心法运转之下，南宫倾城的肩头突然射出一块碧绿的冰片，一道血柱喷射出来，只是血液已经从漆黑转为鲜红。」

### 氛围渲染

- **城市感知**：整个洋河郡城一片寂静——暴风雨前的平静，危机四伏
- **洞窟相会**：阴冷潮湿的洞穴，青苔，荆棘——逃亡者的藏身处
- **疗伤中**：从高冷仙子到脆弱的少女——「冰肌玉骨宛若月宫仙子」vs「胸前绽放开朵朵红梅」
- **夜晚烤火**：食物的香气、温馨的同门情谊——暴风雨后的宁静
- **温馨回忆**：提到小慕容时，天然呆的形象浮现——点缀温情
