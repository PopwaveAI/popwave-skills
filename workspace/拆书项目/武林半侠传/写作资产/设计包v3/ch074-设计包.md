## 二、爽点设计（爽点层）

### 套路识别
- **套路①：画饼激励** — 南宫倾城以《玄真经》为饵驱动方明
- **套路②：反派布局** — 金面人识破方明武功、设套、通缉、吹捧（捧杀连环计）
- **套路③：伪装过市** — 易容术瞒天过海，混入敌境
- **套路④：故人相逢不相识** — 船上遇到张青松却不出世

### 情绪弧线
```
起点（好奇/渴望）：方明听闻《玄真经》描述，眼睛当中有渴慕之色
  ↓
推进（兴奋/警惕）：已知是诱饵仍应下，随后迅速被反派布网
  ↓
高潮（压力/危机感）：金面人刀法实力碾压展示，方明被全面通缉
  ↓
终点（从容/掌控感）：方明易容出逃，内心已定计反吞棋局
```

### 爽点机制
| 爽点类型 | 内容 | 触发位置 |
|----------|------|----------|
| 信息差爽 | 读者知道方明有演武令金手指，金面人不知道（L99-L103） | 第7事件 |
| 算计反杀爽 | 方明"你们算计我，我也算计你们"的宣言（L139） | 第10事件 |
| 伪装逃脱爽 | 王怜花级易容术骗过所有人（L109-L121） | 第8事件 |
| 武力展示爽（反派） | 金面人展示天魔七杀，刀法在方明之上（L89-L97） | 第6事件 |

### 章末钩子
| 类型 | 内容 | 强度 | 回收章 |

## 四、感官与画面（感官层）

### 环境基线

| 维度 | 描述 |
|------|------|
| 时间 | 昼夜交替 → 三日后；开篇山洞谈玄真经为白天，黑熊洞金面人调查为夜，次日乘船 |
| 光线 | 山洞昏暗→黑熊洞尸身被月光照现→大船日光明媚→第三日辉业城远景 |
| 空间 | △山洞（密室感）→ △黑熊洞（阴森埋尸地）→ △五桅大船（水面开阔）→ △辉业远景 |
| 气味 | 黑熊洞尸臭→江上水汽→辉业城码头繁忙气息 |
| 声音 | 山洞对话低回→金面人魔音贯穴"令整个山穴都来回颤抖"→大江东去水声→码头喧闹 |
| 触感 | 金面人手掌"仿佛爱抚情人一样抚过霍青的尸身"→方明脸上人皮面具冰凉 |

### 动作场景
- **金面人展示刀法**（L89-L97）："呜呜鬼哭当中，一招邪异而恐怖的刀法，已经落在了霍青的尸身之上，痕迹宛然"——刀气绽放、鬼哭魔音、山穴颤抖，反派实力展示极具画面冲击
- **船上易容**：从秀才学子外形到内心独白，易筋换骨的过程虽未细写但留下想象空间

### 氛围渲染
- **压迫感**：金面人的讥诮冷笑、魔音贯穴、刀法碾压，营造反派碾压性优势的氛围
- **从容感**：方明坐在船上品茶观景，内心盘算，与黑暗洞中的凝重形成明暗对比
- **期待感**：结尾"一座宏伟的大城已经在望"，画面拉升，给读者一种"故事正式开始"的开放感

## 五、套路归档

### 套路①：捧杀连环计

**文件**：`套路库/捧杀连环计.md`
**起势**：反派先识破目标身份与武功来历（金面人认出万劫天魔）
**蓄势**：下令通缉 + 暗中助其吹嘘成名，将其推到风口浪尖
**爆发**：等目标成名后被揭穿是"魔道余孽"，身败名裂
**余波**：金面人自身刀法更高，图谋借目标之手完成神教大计

### 套路②：替身夺位

**文件**：`套路库/替身夺位.md`
**起势**：方明需要新身份进入大江盟会（为ch076埋伏笔）
**蓄势**：神刀教安排岳鹏假身份，方明获得潜龙榜排名
**爆发**：以假身份击败真高手，声名鹊起
**余波**：真假身份交织的风险与反制
