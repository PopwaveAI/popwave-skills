## 二、爽点设计（爽点层）

### 套路识别
- **套路①：通缉令打脸** — 刚到新地图就被通缉，方明处境急转直下
- **套路②：情报系统引入** — 金风细雨楼+潜龙榜，世界观追加，爽在信息量
- **套路③：榜单排名爽** — 主角排第十三，读者自动产生"以后要冲第一"的期待
- **套路④：自嘲式幽默** — "花几百两买这些，我都觉得自己是个大傻蛋" + "我要告他诽谤"

### 情绪弧线
```
起点（新地图兴奋）：辉业城繁荣景象，船帆如云，货物如山
  ↓
推进（震惊/警觉）：看到自己被通缉，赏格惊人→迅速推理出神刀教告密
  ↓
高潮（惊叹/兴奋）：金风细雨楼情报爆棚，潜龙榜排名、奇功绝艺榜、高手事迹
  ↓
终点（冷静/计算）：天色昏暗，上街寻找神刀教暗号，进入下一步棋
```

### 爽点机制
| 爽点类型 | 内容 | 触发位置 |

## 四、感官与画面（感官层）

### 环境基线

| 维度 | 描述 |
|------|------|
| 时间 | 早上抵达辉业 → 中午/下午金风细雨楼 → 天色渐渐昏暗（傍晚） |
| 光线 | 码头清晨日光→城门中午明亮→金风细雨楼阁楼内部金碧辉煌→街头暮色苍茫 |
| 空间 | △辉业码头（开阔、船帆如灰色天幕）→△城门（人群围堵）→△金风细雨楼（十余丈高、金碧辉煌阁楼）→△街头（暮色下行色匆匆） |
| 气味 | 码头货物味、汗味→茶楼茗香→街头尘烟 |
| 声音 | 码头工人摩肩接踵、挥汗如雨→城门人群议论嘈杂→金风细雨楼江湖情报声声入耳→街头夜风 |
| 温度 | 秋日微凉，傍晚尤甚 |

### 动作场景
- **挤入人群看榜**（L13-L17）："行人里三层外三层地围了一圈……好奇地挤了进去"——简单动作但画面感强，人群中的方明看到自己的脸被通缉
- **负手入城**（L39）：淡定从容，与通缉令的紧张形成反差
- **翻阅榜单**（L67-L86）：坐在茶楼中，一壶香茗两素点，翻薄册子，画面悠闲中带着专注

### 氛围渲染
- **繁荣感**：辉业城码头"船舶如云，帆布绵延开去几乎形成灰色天幕"，鸿篇巨制的繁荣景象
- **紧张感**：通缉令突然出现在面前，赏格万两，读者跟随方明心跳一起加速
- **信息爆棚**：金风细雨楼内"各种源源不断的江湖情报自动塞入耳朵"，信息量大增
- **暮色沉静**：结尾天色昏暗，方明独立街头寻找暗号，从热闹归于冷静
