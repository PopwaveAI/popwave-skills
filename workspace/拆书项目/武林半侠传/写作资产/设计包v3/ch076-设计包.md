## 二、爽点设计（爽点层）

### 套路识别
- **套路①：假身份替换** — 真正的岳鹏七岁已死，神刀教经营整条身份线十多年
- **套路②：示警暗语** — 方灵月在监视下以掌心写字示警
- **套路③：反向控制** — 方灵月想控制方明，反被方明识破并将计就计
- **套路④：当众挑战** — 主角以新身份在众人面前挑战排名更高的高手
- **套路⑤：信息差碾压** — 方明有云梦仙子迷魂大法+坐忘心经，摄心术无效

### 情绪弧线
```
起点（轻松/重逢喜悦）：见到方灵月，有他乡遇故知的温暖
  ↓
推进（震撼/惊讶）：岳鹏身份大揭秘→神刀教势力远超预期→被下摄心术
  ↓
高潮（反转/掌控）：方明假意中招→走出后揭露自己完全免疫→识破迷心丹
  ↓
终点（兴奋/霸气）：戴上人皮面具，持雁翎刀，当众挑战张青松——大战在即
```

## 四、感官与画面（感官层）

### 环境基线

| 维度 | 描述 |
|------|------|
| 时间 | 傍晚/入夜（接续ch075天暗寻暗号） |
| 光线 | 秘舵内室烛火摇曳→街头暮色→李府灯火通明 |
| 空间 | △神刀教秘舵（地下/密室感）→△街头角落（换装遮蔽）→△李府大门（开阔广场+牌楼） |
| 气味 | 密室陈腐气→人皮面具的冰冷皮革味→街头夜风→李府人群气息 |
| 声音 | 秘舵内低声对话→方灵月天魔吟靡靡之音→街头脚步声→李府门前"侠客豪杰全部沸腾"的喧闹→刀剑出鞘声 |
| 触感 | 方明拉住方灵月冰冷小手（微颤、指甲划字）→人皮面具"脸上一片冰凉"→雁翎刀沉手感 |

### 动作场景
- **掌心写字示警**（L25）：方明握住方灵月的手，后者"指甲微划，似乎写了某个字出来"——紧张而隐蔽的交流
- **戴人皮面具**（L39-L43）：方灵月亲自服侍方明戴上面具，"刀锋般的眉毛，高挺的鼻梁，眉宇间满是自信与狂傲"
- **识破摄心术**（L87-L93）：从看似浑浑噩噩到"脸上的神色就是骤然一变"，反差强烈
- **当众挑战**（L147-L149）：方明持雁翎刀招摇过市→李府门前声如雷霆喊话→纵身而上牌楼顶→刀气吹皱一池春水
- **张青松应战**（L157-L171）：孤鹤般掠出→上牌楼顶→长啸邀战→松木剑鞘嗡鸣→三剑杀招青光闪动

### 氛围渲染
- **重逢的温暖与暗流**：方明与方灵月见面拥抱，但"冰冷小手微微颤抖"的示警让气氛迅速紧张
- **大揭秘的震撼**：岳鹏七岁已死、神刀教经营十多年——信息爆棚，读者与方明同时感到震撼
- **密室压迫感**：方灵月施展摄心术，"香气迷醉，万魔低喃"，方明假装中招的紧张气氛
- **决战前的沸腾**：李府门前"侠客豪杰们全部沸腾"，江湖大战的氛围完全拉起
