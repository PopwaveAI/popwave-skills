## 二、爽点设计（爽点层）

### 套路识别
- **装逼打脸流** — 方明以岳鹏身份藏拙不出刀，最后秒杀张青松
- **英雄救美反转** — 看似英雄救美，实则成全野鸳鸯
- **黑吃黑反杀** — 铜印捕头阴谋被识破，当场被斩

### 情绪弧线
**起点（期待上升）** → 牌楼围观，众人期待岳鹏与青松剑客之战
**推进（蓄势）** → 方明只闪不攻，迟迟不出刀，悬念拉到极致
**高潮（释放）** → 一刀出鞘，秒败潜龙榜第十高手，全场震惊
**过渡（轻松）** → 李如壁推销女儿，丫鬟报小姐被劫，戏剧性反转
**终点（爽感余韵）** → 方明斩铜印捕头，放走野鸳鸯，收银剑潇洒离去

### 爽点机制
| 爽点类型 | 触发点 | 强度 |
|---------|--------|------|
| 实力碾压 | 一刀败青松剑客 | L4 |
| 智谋反杀 | 早已看穿捕头阴谋，轻松斩杀 | L3 |
| 成全之美 | 放走真心相爱的司徒色望与李柔 | L3 |
| 台词逼格 | "该出刀时，我自然会出刀" | L2 |

### 章末钩子
| 类型 | 内容 | 强度 | 回收章 |
|------|------|------|--------|
| 剧情钩子 | 方明手中有银剑，回去如何交代李如壁？野鸳鸯能否安全离去？ | L2 | ch078 |

## 四、感官与画面（感官层）

### 环境基线

| 维度 | 描述 |
|------|------|
| 时间 | 日间（午后偏晚） |
| 天气 | 晴 |
| 光线 | 明亮，刀光与剑光交映 |
| 空间 | 辉业牌楼→李府闺房→城外芦苇水泊 |
| 气味 | 芦苇荡水汽、水草腥味 |
| 声音 | 兵刃交击声、围观者叫好声、芦苇沙沙声 |
| 触感 | 刀柄冰凉、肌肤生寒（银剑） |

### 动作场景
- **牌楼对决**：张青松"人影飘飞，出手如风" → 方明"身影飘飞，偶尔以刀鞘进击" → 刀光"一抹雪白的刀光，迅捷如同闪电般划过" → 咔嚓声中古剑脱手
- **水泊厮杀**：乌篷船上银剑"仿佛蛟龙舞动" → 捕头锁链断折 → 方明"刀光一闪，白光将黑色细针消融" → 铜印捕头"捂着自己的咽喉，掉入了水潭当中"

### 氛围渲染
- **对战前**：牌楼下"翘首以待"的武林中人、风信子"好不停留地记录"——紧张期待
- **对战后**："轰然叫好"、"两眼放光"——崇拜狂热
- **水泊转折**：芦苇"荡漾"、"好一块苟合私会，乃至杀人埋尸之地"——荒诞中带杀机
