## 四、感官与画面（感官层）

### 环境基线

| 维度 | 描述 |
|------|------|
| 时间 | 六月十四（金风细雨楼闲坐时）→ 往前回溯六月初三至六月十二的快闪战斗 |
| 天气 | 各地不同，以夏夜为主 |
| 光线 | 醉红楼灯火（夜战）、白日晴空（路遇百里宇） |
| 空间 | 辉业李府→醉红楼→野地→百兽山庄→金风细雨楼 |
| 气味 | 酒香（金风细雨楼/百兽山庄） |
| 声音 | 刀剑交击声、人群争辩声、酒杯碰撞声 |
| 触感 | 易容面具对脸部的压迫感、酒液入喉的灼热 |

### 氛围渲染
- **快闪战斗蒙太奇**：六月初三→初七→初九→十二→十四，每一天一个场景切换，节奏如鼓点，营造"十日震动康州"的势如破竹感
- **金风细雨楼群戏**："人声鼎沸"、"双方争辩越来越激烈" → 剑拔弩张 → 神秘声音一出"整个金风细雨楼都是一静"——由热闹到冷寂的瞬间切换，形成强烈对比

### 特殊画面点
- **金银辉映幻想**："右手金刀，左手银剑，金银辉映，绝对能亮瞎一片人眼"——骚包的视觉意象，塑造方明的少年心性
