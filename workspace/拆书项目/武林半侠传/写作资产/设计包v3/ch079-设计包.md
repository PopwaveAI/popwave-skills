## 二、爽点设计（爽点层）

### 套路识别
- **危机预警流** — 本性灵光示警，提前察觉伏击
- **智取反追杀** — 不硬扛不求助，反其道而行之，反向追踪
- **穿越流** — 演武令开启新世界（侠客行），新旧世界切换
- **选择权爽点** — 主角有明确的"夺舍选择权"（石破天 or 石中玉）

### 情绪弧线
**起点（知识科普）** → 武道设定讲解：任督二脉是绝顶天堑
**推进（平静→危机）** → 离开酒楼走回客栈，日常感十足
**突然转折（惊悚）** → 本性灵光示警！大难临头的感觉
**高潮1（果断破局）** → 瞬间转身遁走，不陪你玩了
**高潮2（反客为主）** → 反潜伏在伏击地附近，要揪出幕后黑手
**高潮3（系统外挂开启）** → 演武令转动，侠客行世界开启！
**终点（期待拉满）** → 两个选择摆面前：石破天（高回报高代价）还是石中玉（低回报低代价）？

### 爽点机制

| 爽点类型 | 触发点 | 强度 |
|---------|--------|------|
| 六感预警（金手指） | 本性灵光示警，提前避过天罗地网 | L4 |
| 智斗反杀 | 不硬扛、反向潜伏、要揪出幕后黑手 | L4 |
| 系统穿越 | 演武令开启侠客行世界，主角的独特金手指 | L5 |
| 选择权 | 可选石破天或石中玉，信息差带来的主动权 | L4 |
| 吐槽幽默 | "侠客行我早就看过一遍了，你不说我也知道剧情啊，有个毛用？" | L2 |
| 世界观控 | 以原著分析侠客行战力等级，展现主角的知识优势 | L3 |

### 章末钩子

| 类型 | 内容 | 强度 | 回收章 |
|------|------|------|--------|
| 穿越钩子 | 方明选择了石破天还是石中玉？穿越后将如何获取侠客行神功？ | L5 | ch080+（侠客行副本） |

## 四、感官与画面（感官层）

### 环境基线

| 维度 | 描述 |
|------|------|
| 时间 | 六月十四日至六月十五（穿越触发） |
| 天气 | 夏日，炎热 |
| 光线 | 金风细雨楼→街市白日强光→客栈室内幽暗→识海演武令玉色光芒大放 |
| 空间 | 金风细雨楼→街市→客栈→识海/星光长河→侠客行世界外（上帝视角） |
| 气味 | 酒菜香气（金风细雨楼）→街市尘土气息→客栈房间的陈旧木味 |
| 声音 | 人声嘈杂→街市喧嚣→突然的寂静→演武令轰鸣声 |
| 触感 | 本性灵光示警时的"眉心微颤"、"大难临头"的寒意 |

### 关键画面渲染

**危机示警画面（高光）**：
> "刚走到街头，方明的身体便是一颤。一种极为不安的感觉瞬间涌遍全身"
> "四周似乎太过安静，并且隐隐有着危险的气机蛰伏！"
> 环境从喧嚣到死寂的瞬间转换，营造紧张悬疑氛围

**演武令穿越画面（高光）**：
> "玉色的演武令骤然光芒大放！" → 金色铭文流转 → 轰隆声中挟裹真灵跨越星光长河 → 飞向巨大赤色光球
> 视觉层级：识海（微观）→ 星光长河（宏大）→ 侠客行世界（锁定目标）
> 色彩对比：玉色光→金色铭文→赤色光球→白气

**两个可选角色的意象对比**：
> "面目依稀类似的少年…一个老实憨厚，一个却给人油头粉面之感"
> 双子星般的站姿，意象对比强烈——一个如璞玉，一个如涂漆

### 氛围渲染
- **郊外**：金风细雨楼的"热闹"→ 街市的"一切如常"→ 发觉危险时的"过于安静"——由热转冷的递进式降温，让读者的不安同步上升
- **穿越前**：客栈房间内"挂了免打扰牌子，径自躺在床上"→ 识海迸发玉色光芒→ 轰鸣巨响——从极静到极动的爆发
- **上帝视角**：站在世界之外的"近乎神圣仙佛"感——塑造方明超越原世界力量的独特位置
