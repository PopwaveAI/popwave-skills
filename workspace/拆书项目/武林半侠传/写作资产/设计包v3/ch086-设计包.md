## 四、感官与画面（感官层）

### 环境基线

| 维度 | 描述 |
|------|------|
| 🕐 时间 | 傍晚时分抵达凌霄城（原文第53行） |
| 🌤️ 天气 | 雪山严寒，冰雪覆盖，凝水成冰 |
| 💡 光线 | 傍晚昏暗，雪山反光 |
| 📐 空间 | 凌霄城：冲天山峰 + 数百屋宇 + 三丈白墙 + 三丈宽护城冰沟 |
| 👃 气味 | 冰雪清冽（推断） |
| 🔊 声音 | 吊桥轧轧声、铁链呛啷声、长剑出鞘声、方明轻笑声、封万里惊呼声 |

### 动作场景

**场景1：飞跃冰沟（🔒06）**
```
方明纵身一跃 → 身子直似孤鸿雪燕 → 斜飞出四五丈距离 
→ 轻轻巧巧几个转折 → 站到城门之上
```
- 节奏：轻快飘逸，一气呵成
- 视角：侧面仰视，众人惊叹
- 对比：冰沟细滑无从借力 vs 方明如履平地

**场景2：登城震慑（🔒07）**
```
守城弟子长剑刺来 → 方明一挥袖口荡开 → 那人滚出老远 
→ 脚尖挑剑 → 内力灌注意 → 精钢长剑应手而断
```
- 节奏：快准狠，毫不拖泥带水
- 感官：袖风、剑断声、对方惊颤的声音

### 氛围渲染

| 氛围 | 手法 | 效果 |
|------|------|------|
| 雪山孤绝 | 冰沟·白墙·孤峰·三丈高墙 | 与世隔绝的江湖堡垒 |
| 山雨欲来 | 大门无人值守·吊桥高悬·半天才探头 | 异常即阴谋的前兆 |
| 内乱沉重 | 数百弟子包围·剑尖所指·封万里断臂 | 同门相残的悲剧感和压迫感 |

## 五、套路归档

### 套路01：开局铺陈·佛门武学谱

**文件**：`/套路库/套路-开局铺陈-佛门武学谱.md`

**起势**（3-14行）：以旁白介入，梳理金庸系佛门武学体系——金刚不坏神功（防御）、罗汉伏魔神功（调和阴阳）、九阳神功（内息无尽·百毒不侵）、易筋经（佛门神髓）。
**蓄势**（15-20行）：收束至方明自身——其根基纯粹，必得易筋经方能登峰造极，将设定拉回主角线。
**爆发**（21-24行）：明确方明态度——对易筋经的重视甚至超过九阳神功，并引出后续规划（雪山后去少林）。
**余波**：章节背景板，为后期少林线埋下伏笔。

### 套路02：微服出行·势力威仪

**文件**：`/套路库/套路-微服出行-势力威仪.md`

**起势**（35-36行）：方明"有权不用过期作废"，广发长乐帮号令。
**蓄势**（36-37行）：各分舵调快马、沿路好生招待，一路太太平平到西北。
**爆发**：未在本章直接爆，作为背景势力展示。
**余波**：展示方明此时已是江湖一方势力之主，为后续调兵遣将铺垫。

### 套路03：登山显技·围观震惊

**文件**：`/套路库/套路-登山显技-围观震惊.md`

**起势**（39-40行）：雪山奇险，众人艰难攀越。
**蓄势**（41-43行）：方明与石中坚并肩前列，尚有馀力说笑，吐气悠长——第一层对比石清夫妇二线，第二层对比雪山弟子落后。
**爆发**（45行）：白万剑内心独白——"万一那狗贼所说是假……合凌霄城之力，便真的拿得下他一家么？"——真正震惊的是潜在对手的无力感。
**余波**：白万剑信心动摇，铺垫后续方明夺权。

### 套路04：轻功炫技·天堑如夷

**文件**：`/套路库/套路-轻功炫技-天堑如夷.md`

**起势**（75行）：方明来到护城冰沟边上，纵身一跃。
**蓄势**（75行）：身子直似孤鸿雪燕，斜飞出四五丈距离。
**爆发**（75-79行）：轻轻巧巧几个转折站到城门之上——满堂"好"字喝彩，点明冰沟细滑无可借力，更衬轻功绝顶。
**余波**：立定未稳长剑便至，转入下一冲突。

### 套路05：一语破天机

**文件**：`/套路库/套路-一语破天机.md`

**起势**（107行）：方明人影一晃挡住白万剑，"白师傅恐怕去不得"。
**蓄势**（115-119行）：方明连环追问白自在病情，封万里硬着头皮说谎，气场已弱。
**爆发**（121-128行）：方明直刺内心——"可我怎么听说……白老爷子……被你们联手除掉了？"封万里失声："不可能……我之前还去看过……" ——石破天惊！
**余波**（129-140行）：白万剑等拔剑，四支弟子涌入，局势瞬间白热化。
