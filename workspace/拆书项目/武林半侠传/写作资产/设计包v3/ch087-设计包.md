## 四、感官与画面（感官层）

### 环境基线

| 维度 | 描述 |
|------|------|
| 🕐 时间 | 本章延续傍晚→入夜（无明确时间标记） |
| 🌤️ 天气 | 雪山严寒，屋檐有冰雪 |
| 💡 光线 | 大厅内灯火通明或阴暗（推断为火把/烛光） |
| 📐 空间 | 雪山派大厅：宽敞可容数百人，有主位太师椅，门外通广场 |
| 👃 气味 | 冰雪的冷冽、受伤流血的血腥气 |
| 🔊 声音 | 方明爆喝震得冰雪籁籁而下、长剑叮当落地、哀嚎声、梁自进喷血声、史小翠怒骂声 |

### 动作场景

**场景1：爆喝震冰雪（🔒04）**
```
方明爆喝一声 → 屋檐上的冰雪籁籁而下 → 众人被震慑当场
```
- 感官：声波冲击视觉（冰雪落下）→ 震慑力具象化

**场景2：铁指禅夹剑（🔒06）**
```
梁自进长剑倏出，一招'明驼西来' → 方明食指与中指轻轻一夹 
→ 剑尖落入手中如铁钳 → 梁自进涨红了脸拔不出来 
→ 方明两指一松 → 梁自进倒飞撞门框 → 鲜血染白衣襟
```
- 节奏：快快→慢（夹住僵持）→ 快（松手飞出去）
- 对比：梁自进涨红脸的全力 vs 方明云淡风轻

**场景3：群战横扫（🔒07）**
```
方明大笑上前 → 身影如大鹏鸟撞入人群 → 脚下地砖寸寸碎裂 
→ 屈指连弹，长剑不断落地 → 横冲直撞 → 数十名弟子倒飞而出 
→ 胸口凹陷，筋断骨折，哀嚎不已
```
- 节奏：一气呵成，连绵不断
- 感官：地砖碎裂声、长剑落地叮当声、哀嚎声

**场景4：石子飞射败三头领（🔒08）**
```
方明长啸 → 脚下气劲发动 → 板砖碎裂，无数石子飞起 
→ 十指如钩接连弹射 → 石子如飞蝗箭雨 → 三头领举剑横挡 
→ 长剑脱手而飞 → 身上多了几处血洞 → 齐齐倒下
```
- 节奏：蓄力→爆发→收割
- 视觉：飞蝗箭雨、血洞
- 听觉：劲风破空声、长剑脱手声

### 氛围渲染

| 氛围 | 手法 | 效果 |
|------|------|------|
| 内乱紧张 | 数百弟子剑拔弩张、廖自砺喊"乱剑分尸" | 同室操戈的紧迫感 |
| 碾压爽感 | 方明三场打斗递进升级（一对一→群战→远程） | 武力统治的畅快感 |
| 逆袭快感 | "我石中玉也是雪山派弟子，好像也可以来坐一坐" | 身份反转的戏剧性 |
| 长辈威慑 | 史小翠的怒骂如连珠炮，骂得白万剑们只能躬身听着 | 尊卑秩序压制武功 |
| 爱情微光 | 阿绣与石中坚同框，面红过耳 | 柔和插曲，缓冲紧张 |

## 五、套路归档

### 套路06：以武夺帅·四方降伏

**文件**：`/套路库/套路-以武夺帅-四方降伏.md`

**起势**（41-42行）：方明自荐当掌门人，"我石中玉也是雪山派弟子"。
**蓄势**（43-54行）：梁自进出手试探→方明铁指禅夹剑尖→松指震飞→第一场碾压。
**爆发**（55-82行）：群战横扫数十弟子→石子飞射连败三头领→三线收割完成。
**余波**（87-95行）：四支低头尊掌门，方明坐上主位。"我做掌门人，你们服不服？"→"服气！"

### 套路07：一字震全场

**文件**：`/套路库/套路-一字震全场.md`

**起势**（31-32行）：方明"腻歪了这套"，要介入。
**蓄势**（32行）：暗聚内力，声未发而势已成。
**爆发**（33行）："突然爆喝一声，连屋檐上的冰雪都震得籁籁而下"——冰雪落下，全场肃静。
**余波**（35-37行）：方明接着冷冷点破众人龌龊心思，声威并用。

### 套路08：群战如割草

**文件**：`/套路库/套路-群战如割草.md`

**起势**（55-60行）：成自学与廖自砺号令群攻，五支弟子报仇心切挺剑而上。
**蓄势**（61-65行）：方明大笑以武夺帅，身影如大鹏鸟撞入人群，地砖寸寸碎裂。
**爆发**（66-69行）：屈指连弹→长剑不断落地→横冲直撞→数十人倒飞→筋断骨折。
**余波**（70-72行）：其它支脉弟子胆颤心惊，齐自勉握剑颤抖，不敢出声。

### 套路09：一招败高手

**文件**：`/套路库/套路-一招败高手.md`

**起势**（103-105行）：白万剑站出反对，"白老爷子尚在，你算什么东西？"
**蓄势**（106-108行）：白万剑明知不敌但义之所在，准备出手。
**爆发**（109-113行）：方明迅捷扑出→势如闪电→一指禅力侵入经脉→白万剑四肢无法动弹→"说躺下便躺下了"。
**余波**（113行）：雪山众弟子见第一高手一招即败，俱是心服。

### 套路10：长辈阻力·新变量

**文件**：`/套路库/套路-长辈阻力-新变量.md`

**起势**（123-125行）：阿绣掺着史婆婆昂扬进入大厅，雪山弟子悚然动容让路。
**蓄势**（126-131行）：史小翠痛骂所有人——"一身武功全部学到狗身上去了！叫白万死！花万死！"
**爆发**（133行）：史小翠盯着方明，"你这个小畜生，还不快给我下来！！！"
**余波**（145-160行）：方明以阿绣婚姻为筹码交锋，史小翠提出找白自在比武——事件转入更高层冲突。
