### 动作场景

**场景1：第一掌对拼（🔒06）**
```
方明轻飘飘一掌印出 → 掌到中途狂风大作 → 劲气四溢，轰然作响 
→ 白自在肉掌推出 → 大摔碑手相碰 → 轰的一声大响，如旱地惊雷 
→ 白自在倒飞 → 撞在石柱上 → 灰尘籁籁而落
```
- 节奏：慢（起掌）→ 快（加速）→ 极快（碰撞）→ 慢镜头（倒飞）
- 感官：视觉（掌风、灰尘落地）+ 听觉（巨响如雷）
- 反差：方明"轻飘飘" vs 白自在"倒飞"

**场景2：神倒鬼跌三连环失效（🔒07）**
```
白自在目光凛凛如天神下凡 → 狂冲上前 → 右抓后颈 → 左爪揪胸 
→ 脚下连环绊出如幻影 → 三招浑然一体 → 抓住了！ 
→ 方明不闪不避 → 白自在手上仿佛抓到厚鱼鳞皮毫无着力之处 
→ 又似大刺猬，千千万万细针轰然而上 → 不由撒手 → 绊脚徒劳无功
```
- 节奏：快攻（三招连环）→ 僵住（抓住但无效）→ 反伤（细针刺感）
- 感官：触觉（鱼鳞皮、大刺猬细针）+ 热力（蓬勃热力）
- 白自在的震惊转换：大喜→愕然→刺痛撒手

**场景3：终极对攻·观音千手 vs 百拳（🔒08）**
```
白自在怒发如狂 → 像受伤的野兽嚎叫 → 铁掌化为重重虚影 
→ 如乱箭重打 → 拳影掌影纷飞 → 一瞬间数十拳上百掌 
→ 方明轻轻巧巧接下 → 双手结印，观音千手，大慈大悲 
→ 幻化为巨网 → 将白自在囊括进去 → 轰隆隆！！！ 
→ 拳掌不断交接 → 劲力四溢 → 四面墙壁摇晃不断 
→ 石柱锁链轰然作响 → 众人退出石门 → 只能通过风雷声和颤抖感知战斗
```
- 节奏：狂暴攻击（白自在）vs 从容编织（方明）→ 密集对撞
- 视角转换：近景（拳掌细节）→ 全景（墙壁摇晃）→ 退到门外（间接描写）
- 感官：视觉（重重虚影、幻化巨网）+ 听觉（轰隆隆风雷）+ 触觉（地面颤抖）
