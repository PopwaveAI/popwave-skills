## 四、感官与画面（感官层）

### 4.1 环境基线

| 维度 | 设定 |
|------|------|
| 时间 | 清晨→上午（从追击到山洞谈话） |
| 天气 | 晴，野外风轻 |
| 光线 | 路上：自然光 → 战斗：剑光刀影交错 → 山洞：昏暗篝火 |
| 空间 | 野外大路（开阔打斗）→ 马车（狭窄）→ 山洞（封闭、温暖） |
| 气味 | 血战：血腥味 → 山洞：篝火烟味、烤香菇松茸香气 |
| 声音 | 银剑点肉咔嚓声、王振野兽咆哮声、教徒被撕的惨叫声、金刀破风声、篝火噼啪声 |

### 4.2 动作场景

**王振变身·修罗降世（🔒3-🔒4）：**

> 王振猛地咆哮一声，身上肌肉虬结，身形居然又瞬间拔高了两分，变成了一个身高两米，肤色青黑的巨灵神！
> 他原本身形干瘦，但此时骤然发功，却是如暴雨山洪，凶猛狂飙，气魄慑人！
> 王振猛地弹腿，仿佛猛兽般扑击，姿势虽然不雅，但速度却快到了极点。
> 一人惨叫传来，被他抓住两边身子，哗啦一声从中撕裂，内脏泄了一地，死的惨酷无比。

**刀剑双杀·终结王振（🔒6-🔒7）：**

> 方明左剑右刀，金乌刀法与雪山剑法联合而使，刀光剑影连绵不绝，似乎组成了利刃之龙卷，将王振团团包裹在内。
> 两道人影越打越快，中间夹杂着王振野兽般的咆哮，将周围树木扫落一圈。
> 黑色与银色的光芒渐渐盖过了青黑色的兽影。
> 突然间，一道绚丽至极的刀光从战圈中升起，带着王振的一声惨叫。
> 方明刀剑入鞘。王振身上传来一阵爆响，一道刀气炸开，涌出许多青黑色的血液。王振的身体像泄了气的皮球一样瘪缩了下去。

### 4.3 氛围渲染

| 场景段 | 氛围关键词 | 渲染手法 |
|--------|------------|----------|
| 剑法试招 | 从容、游刃有余 | 方明把王振当靶子，"咔嚓"剑点声细腻描写 |
| 修罗变身+屠杀 | 恐怖、血腥、窒息 | 王振身体膨胀的"拔高"描写+手撕活人的视觉冲击 |
| 刀剑决战 | 激烈、绚烂、攻防 | 黑白vs青黑光影对抗、利刃龙卷意象、兽吼人喊交织 |
| 山洞烤串 | 温馨、柔软、烟火气 | 篝火噼啪声、烤香菇香气、方灵月泪光 |
| 真相揭露 | 沉重、秘密、暗流 | 方灵月冷静叙述偷龙转凤，灯火映照下的严肃神情 |
