## 四、感官与画面（感官层）

### 4.1 环境基线

| 维度 | 设定 |
|------|------|
| 时间 | 黄昏至午后→十数天后→七月十三 |
| 天气 | 晴朗/清风送爽 |
| 光线 | 红日西斜→月光 |
| 空间 | 黄龙楼(湖畔酒楼)→八百里太平湖→连碧岛荷花塘 |
| 气味 | 银鱼鲜香、荷香扑鼻 |
| 声音 | 湖水拍岸、清风拂荷、黄莺般的话语声 |
| 温度 | 湖中秋风略带凉意 |

### 4.2 动作场景

暂无高强度武斗场景，但有以下画面鲜明的动作片段：

- **方明泛舟**：雇小船，不要船夫，随波逐流，躺在舟上与自然融为一体
- **摘莲蓬剥莲子**：随手摘莲蓬，剥出白嫩莲子为食，生动闲适
- **小船靠向花船**：方明心念一动，小船缓缓靠拢，抱拳问路

### 4.3 氛围渲染

本章以"宁静→暗流→宁静"的波浪式氛围为主：

1. **黄龙楼段**（氛围：沉静+历史厚重）：
   - 太平银鱼之鲜美，鱼龙道人与青萍剑客决战典故
   - "斯人已去，难以再返"的文人感慨
   - 方明对武道长生的哲学式思考——庄严

2. **噩耗段**（氛围：震惊+阴郁）：
   - 金风细雨楼情报的"峥嵘之感"与"憾甚"转折
   - "我……我日！刘舟师兄，你就这么去了么？"——黑色幽默消解悲伤

3. **荷花塘段**（氛围：清新+闲适）：
   - "接天莲叶无穷碧"的画面感
   - 荷香扑鼻、白嫩莲子——清新自然
   - 黄莺般清脆的话语——生机勃勃
