## 四、感官与画面（感官层）

### 环境基线
| 维度 | 设定 |
|------|------|
| 时间 | 子时（午夜） |
| 天气 | 干燥、无风 |
| 光线 | 油灯一盏，昏黄摇曳，烛火下人影幢幢 |
| 空间 | 破败土地庙，神像破旧，庙内逼仄，屋顶有破洞 |
| 气味 | 油灯燃烧的焦味、灰尘味 |
| 声音 | 夜风掠过屋檐、油灯滋滋燃烧、落针可闻的寂静→突然的瓦片碎裂声→剑鸣声→落地声 |
| 触觉 | 夜间的凉意、袈裟的粗糙布料、剑柄的寒气 |
| 额外维度 | 阴影——岳不群在烛火下的脸孔"隐隐峥嵘" |

### 动作场景（废岳不群武功）
> "方明右掌伸出，与岳不群内力一拼，老岳顿时倒退数步，脸上紫色褪去，变得殷红如血……"

动作链：欺步→一指点出→岳不群举剑横档→精钢长剑断为两截→余势不减点中膻中穴→岳不群大叫一声→真气宣泄而出→萎顿在地→吐出一口鲜血→头发瞬间白了大半

### 氛围渲染
本章氛围由**暗夜的诡异感**（子时土地庙、黑衣人、烛火）→**压迫感**（方明层层揭露，岳不群由淡定到崩溃）→**爆发感**（秒杀二老、废岳不群）→**苍凉感**（宁中则泪流满面带走岳不群）。整体是**暗色调但爽感明亮**的节奏——越黑暗的揭露，越让读者觉得痛快。
