### 环境基线

| 维度 | 描述 | 位置 |

| 气味 | 血腥味（头颅包裹） | L123 |

| 温度/触感 | 刀气如冰、掌风如浪 | L159-L171 |

### 动作场景（岳云书房之战）

```
动作链：岳云双掌微垂凝而不动 → 一掌拍出掌心微缩 → 掌影重叠六重浪 → 方明不动 → 
岳云催发至十二重浪（乱石穿空、惊涛拍岸）→ 方明拔刀一斩 → 黑刀一分为七 → 
天魔七杀式劈波斩浪 → 气爆声接连不断 → 警铃狂响 → 帮众围来 → 岳云喝退众人
```

**画面质感**：
- 怒涛掌：掌力如长江大河，叠浪无穷，书房内充满浪影，视觉上"乱石穿空，惊涛拍岸"
- 天魔七杀式：黑色刀光来自阴间地狱，一分为七，如地狱临凡、众鬼索命
- 对比：岳云的掌法雄浑开阔（水意象），方明的刀法霸道森然（地狱意象）

| 场景段 | 氛围 | 渲染手法 |
