## 二、爽点设计（爽点层）

### 套路识别
1. **"败犬哀啼·瓮中捉鳖"** — 东方玉狼狈躲藏，方明步步围杀
2. **"宿命对决·金刀正统"** — 双金刀少主万劫刀法对决，方明天魔七杀式碾压
3. **"继承一切·夺其所有"** — 杀东方玉→拿金关玉骨诀→接手神刀教→收编岳云→夺大江盟
4. **"双重身份·天下棋局"** — 方明=岳鹏的变脸布局，为夺大江盟主埋线
5. **"天外飞仙·流云降世"** — 白眉少年骑鹤凌空，开启更高武力层级

### 情绪弧线
```
主线：起点（东方玉怨毒+败犬哀啼）→ 推进（四护法反水，绝境围困）
→ 高潮（万劫刀法对决，东方玉死）→ 终点（方明继承一切+化岳鹏登盟主）
副线（章末）：从世俗权谋骤然腾空——白眉少年骑鹤俯瞰，碾压级战力登场
```

## 四、感官与画面（感官层）

### 环境基线

| 维度 | 设定 |
|------|------|
| ⏱时间 | 深夜→白天（分舵→悬崖→大江盟） |
| 🌤天气 | 密林潮湿，悬崖风大 |
| 💡光线 | 分舵黑暗→悬崖开阔→大江盟明亮洞府 |
| 🏛空间 | 密闭分舵→万仞悬崖→开阔密室→高空云层 |
| 👃气味 | 潮湿木料+血腥味+山风清气 |
| 🔊声音 | 野兽低吼、刀兵碰撞、山风呼啸、鹤鸣九皋 |

### 动作场景

**天魔七杀式 vs 万劫刀法（L57-L72）**
> 两点幽火在方明眸中一闪而过→天魔附体→黑色闪电划过虚空→砰！单刀断折→一人倒下

**收信物·金刀教徒跪伏（L89-L98）**
> 金色匕首如小刀，匕身密密麻麻小字→方明手持→所有金刀教徒恭敬跪伏→脸上充满狂热之色

**变脸易容（L135-L148）**
> 脸上肌肉筋骨变化→身形变得更加修长消瘦→眨眼间变成"岳鹏"→岳云心服口服跪拜

**白眉少年登场（L189-L192）**
> 碧空万里→层云之上→双翅三丈铁喙飞鹤→白羽若云，黑爪似钢→羽衣玄冠面白唇红少年→渊渟岳峙

### 氛围渲染

- **前半黑暗压抑**：东方玉躲在黑暗中的密林分舵，"受伤的野兽"——黑暗+怨毒+失败者的狂躁
- **对决肃杀**：扔刀一声脆响+发冠散开黑发乱舞+万劫刀法狠绝狠辣——充满仪式感的宿命对决
- **继承仪式**：金色匕首从尸体中取出→金刀教徒跪伏如见神祇→方明心中吐槽"教派洗脑可怕"——反差感
- **结尾升华**：从地下的厮杀→万仞悬崖→再攀至碧空万里骑鹤而行的仙家画面，视觉层级骤升

## 五、套路归档

### 套路·败犬哀啼·瓮中捉鳖
**起势** → 东方玉藏身分舵，怨毒诅咒方明 → **蓄势** → 自以为神不知鬼不觉，仍在谋划东山再起 → **爆发** → 方明突然现身，携岳云、方灵月、四护法——全部反水 → **余波** → 东方玉"心已彻底沉了下去"

### 套路·宿命对决·万劫归宗
**起势** → 双金刀少主对峙，同修万劫刀法 → **蓄势** → 东方玉全力催发金关玉骨诀，刀法狠绝 → **爆发** → 方明天魔七杀式，黑色闪电一刀定生死 → **余波** → 东方玉临终承认"你果然才是真正的金刀少主"

### 套路·继承一切·夺其所有
**起势** → 东方玉死 → **蓄势** → 岳云搜出金色匕首/金关玉骨诀 → **爆发** → 金刀教徒全部跪伏方明，方明接收五六成神刀教实力 → **余波** → 方明化岳鹏夺大江盟主，双重身份布天下棋局

### 套路·天外飞仙·武力升维
**起势** → 方明以岳鹏身份登大江盟主，大局已定 → **蓄势** → 青云宗/总督府未阻拦，因发生了"更大的事" → **爆发** → 碧空万里，白眉少年骑鹤凌空，俯瞰康州 → **余波** → 悬念拉满：此人究竟是谁？来康州为谁？
