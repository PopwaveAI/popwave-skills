## 二、爽点设计（爽点层）

### 套路识别
1. **"天威降临·蝼蚁齑粉"** — 流云道人骑鹤而来，一爪灭还真观，武力碾压到极致
2. **"宗师陨落·托孤遗言"** — 慕容秀一生修为不堪一击，临终叮嘱"不至天人不可复仇"
3. **"后方坐镇·黑手布局"** — 方明化身岳鹏清洗大江盟，暗布棋子以柔克刚
4. **"藏身蓄势·龙蛰于渊"** — 得知天人存在后，方明果断收收缩，托庇南宫倾城与小慕容
5. **"世界观补全·武力升维"** — 通过对话完整解锁大乾武道体系

### 情绪弧线
```
A线（流云道人）：起点（骑鹤神仙中人）→ 推进（天威笼罩，弟子吓死）→ 高潮（云龙探爪，道观成灰）→ 终点（事了拂衣去）

双线在"天人级力量认知"交汇
```

### 动作场景

**天威降临（L16-L24）**
> 浩大的压力骤然降临→如渊如天→雷雨之前的沉闷×百倍千倍→后天弟子昏厥→冷汗不断→肝胆俱裂气绝而亡→还未动手就吓死

**云龙探爪（L45-L62）**
> 白眉少年伸手成爪，缓缓一抓→一爪出而风云动→层层白云聚成巨大漩涡→云龙隐现→探出夭矫若神龙的一爪→还真观火光一闪旋即熄灭→云龙之爪缓慢而坚定探下→整个还真观化为一片灰烬

**方明端坐主位（L65-L91）**
> 身披九蛟玄色长袍→龙盘虎踞→眉宇开合摄人心魄→六大先天垂手树立→"清洗精简剔除杂鱼"→权威通行无阻

**太平湖地宫（L129-L155）**
> 八百太平湖浩瀚无涯→荒岛别有洞天→走过地洞→干燥整洁地宫→烛火下小慕容犹带泪花→方明扯过锦被盖好

### 氛围渲染

- **开场仙幻震撼**：碧空万里，骑鹤遨游，神仙中人→突然天威如渊如海，弟子吓死→极静到极恐的转化
- **毁灭高潮**：云龙探爪——"武道至此，实已有着惊天动地之能"——视觉奇观+压迫感拉满
- **中段权谋冷静**：从天地大毁灭切回方明端坐总舵的从容清洗，一热一冷对照
- **收尾温情+暗流**：小慕容扑入怀中哭泣，方明盖被子——温柔；南宫倾城复述当日之景，心灵阴影——暗流涌动
- **智力闭合**：武力体系梳理——从感官冲击回归理性认知，为后续升级铺路

## 五、套路归档

### 套路·天威降临·蝼蚁齑粉
**起势** → 白眉少年骑鹤凌空，如神仙中人 → **蓄势** → 天威笼罩还真观，先天弟子被托出百丈，低阶弟子直接吓死 → **爆发** → 一式云龙探爪，风云聚拢化为巨大云龙，一爪之下道观成灰 → **余波** → 事了拂衣去，南宫倾城留下终生心灵阴影

### 套路·宗师托孤·遗言留线
**起势** → 慕容秀已知不敌，留遗言 → **蓄势** → 门派弟子先送走，小慕容抛给倾城 → **爆发** → "记住！不至天人，万万不可兴复仇之念！"→慕容秀战死 → **余波** → 小慕容先天道体，南宫倾城心灵阴影，此仇已刻入后续主线

### 套路·主角隐忍·龙蛰于渊
**起势** → 方明刚统合大江盟，意气风发 → **蓄势** → 听闻流云道人出手灭玄真，内心震骇 → **爆发** → 果断收缩实力，安排后路，接应南宫倾城和小慕容到秘密据点 → **余波** → "光阴是站在我们这边的"——战略性隐忍，躲避天人锋芒

### 套路·武力体系解锁·世界观补全
**起势** → 方明一直对境界有模糊认知 → **蓄势** → 流云天人之威→慕容秀遗言"不到天人" → **爆发** → 方明向南宫倾城请教，结合前世记忆，完整勾勒出先天→宗师→大宗师→天人体系 → **余波** → 世界格局清晰化，主角升级路线明确化
