## 二、爽点设计（爽点层）

### 2.1 套路识别

| 套路ID | 名称 | 本章触发 | 强度 |
|--------|------|----------|------|
| TL-悟道解构 | 主角以更高维认知碾压本土世界观 | 方明用天地元气浓度解释大乾VS黄易武力差异 | L3 |
| TL-藏锋扮猪 | 主角隐藏真实身份，示弱钓敌 | 方明以"岳鹏"身份示人，各方揣测 | L2 |
| TL-强者降临 | 主角以压倒性实力登场震慑 | "盟主驾临"三声炮响，光芒气场全开 | L2 |
| TL-不动如山 | 面对试探，不动声色以实力化解 | 千绝心眼被方明无形挡回 | L2 |
| TL-宝典不得 | 顶级机缘近在眼前却无法参悟 | 玄真经/太玄经看不懂的困境 | L3 |

### 2.2 情绪弧线

```
起点（平静）：方明静参武道，梳理世界体系
  ↓
推进（微沮）：玄真经看不懂，宝典空有
  ↓
转折（豁然）：决定走转战路线，以低维世界汲取神元
  ↓
高潮（张力）：三方势力齐聚，暗中试探交锋
  ↓
终点（悬而未决）：方明以岳鹏身份稳住局面，战略收缩已定
```

## 四、感官与画面（感官层）

### 4.1 环境基线

| 维度 | 描述 |
|------|------|
| 时间 | 白天（中秋之后，新盟主即位不久） |
| 光线 | 静室幽暗→客厅明亮，形成明暗对比 |
| 空间 | 静室（封闭、安静）→客厅（开阔、正式、势力对垒） |
| 声音 | 静室无声→三声炮响、帮众欢呼→茶杯落地脆响 |
| 气味 | 静室檀香/墨香→客厅茶香 |
| 触感/温度 | 筋骨移位的骨节爆响，暗示变身过程的物理感 |

### 4.2 动作场景

**关键场景：方明变装**
> 「身上骨节爆响，筋骨移位当中，已经变成了那个大江盟主『岳鹏』！」

- 画面：骨节啪啪作响、肌肉骨骼重组、面容逐渐变化
- 感官：声音（爆响）、视觉（轮廓改变）、触感（筋骨拉扯）

**关键场景：登场**
> 「穆然三声炮响，帮众齐声欢呼」
> 「眉锋似刀，身穿九蛟黑袍的青年漫步而入，身后六大先天长老相随」

- 画面：黑袍猎猎、六人随行、气势如虹
- 声音：炮声轰鸣、欢呼如潮
- 视觉："眉锋似刀"的锐利形象

**关键场景：暗战**
> 「虚空中一丝极其细微的气机降临下来，无孔不入」

- 抽象化的感官：气机如蛛网般覆盖、被千绝心眼洞穿的感觉

### 4.3 氛围渲染

| 段落 | 氛围 | 手法 |
|------|------|------|
| 开篇悟道 | 深远、超然 | 心理独白+世界观的宏大叙事 |
| 静室困境 | 压抑、焦灼 | "空有宝典"的无奈反衬主角的进取心 |
| 客厅交锋 | 暗流涌动、剑拔弩张 | 表面礼数下的气机对抗，张力 |
| 章末 | 悬念、隐忍 | 一切平静但风雷将至 |
