## 二、爽点设计（爽点层）

### 套路识别
1. **旧日英雄坠落** — 阿飞被林仙儿/上官飞击败，从剑客沦为"死狗"（常规悲剧弧线）
2. **身份碾压** — 方明以神锋楼主身份插入两大高手对峙，气机碾压（装逼打脸变体）
3. **信息差优越** — 方明掌握"绝世之血"任务机制而读者跟随其视角，形成知识优越感
4. **高手收容癖** — 方明收集兵器/高手，类似收藏家/集卡式爽感

### 情绪弧线
**主线（阿飞堕落线）：** 起点(傲气剑客) → 推进(被林铃铃压制) → 转折(林仙儿离去) → 高潮(被上官飞击败+暴雨流泪)
**副线（方明布局线）：** 起点(挂刀的神秘感) → 推进(揭示绝世任务) → 转折(不认可失败) → 高潮(插入两大高手气机，以一敌二)

### 爽点机制
| 爽点 | 触发事件 | 强度 | 机制说明 |
|------|---------|------|---------|
| 知识碾压 | 方明论兵器与人 | L3 | 用哲理高度碾压兵器谱排名体系 |
| 身份碾压 | 方明插入郭嵩阳/吕凤先对峙 | L3 | 硬生生挤入气机领域，制造绝对碾压 |
| 悲剧共情 | 阿飞暴雨流泪 | L4 | 男儿有泪不轻弹的极致悲剧感 |
| 期待感 | 方明同时挑战兵器谱两大高手 | L3 | 下一章决战期待 |

### 章末钩子
| 类型 | 内容 | 强度 | 回收章 |
|------|------|------|--------|
| 战斗期待 | 方明同时挑战郭嵩阳（嵩阳铁剑）与吕凤先（银戟温侯） | L4 | ch132 |

## 四、感官与画面（感官层）

### 环境基线

| 维度 | 详细设定 |
|------|---------|
| 时间 | 白天→黄昏→暴雨夜 |
| 天气 | 晴转暴雨 |
| 光线 | 阁楼灯火忽明忽暗 → 路边黄昏 → 暴雨如注 |
| 空间 | 林间空地 → 红色门槛小楼 → 阿飞草屋 → 山泉边 → 路边 → 枫树林 |
| 气味 | 山泉水草气 → 草屋饭菜余香 → 雨后泥土 |
| 声音 | 竹枝破风声 → 阁楼兵器反光 → 暴雨声 → 子母钢环嗡鸣 → 枫叶沙沙 |

### 关键画面场景

**场景1：林铃铃制阿飞**
> 竹枝如蛇疾刺 → 少女珠钗一闪已到阿飞颈后 → 轻功如鬼魅

**场景2：阁楼挂刀**
> 红门槛星星点点血迹 → 四墙雪白，孤零零四件兵器 → 青手套反光 → 方明挂上黑刀

**场景3：阿飞暴雨流泪**
> 剑折人倒 → 烂泥中如死狗 → 暴雨冲刷 → 两行清泪混着雨水流下

**场景4：枫林对峙**
> 漫天红叶飘零 → 一黑一白两个极端 → 气机布满方圆五丈 → 方明硬生生"挤"入

### 氛围渲染
- **小楼氛围：** 阴森、神秘、收藏者圣殿感
- **阿飞落败氛围：** 凄惨、悲凉、雨夜孤绝
- **枫林对峙氛围：** 肃杀、高手气场、张力满弓
