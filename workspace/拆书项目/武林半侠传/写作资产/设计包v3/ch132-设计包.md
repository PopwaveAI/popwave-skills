## 二、爽点设计（爽点层）

### 套路识别
1. **绝境救场** — 李寻欢出现在大欢喜女菩萨处救阿飞（友情救援经典套路）
2. **幕后黑手曝光** — 方明自承杀五毒童子是为了引大欢喜女菩萨出来（反转）
3. **武力展示碾压** — 方明用小李飞刀技法击败郭嵩阳（借刀打脸的爽感）
4. **世界观降维打击** — 方明讲解如意天魔连环八式，碾压所有在场者的认知（知识碾压）

### 情绪弧线
**主线（魔教揭秘线）：** 起点(方明击败郭嵩阳) → 推进(阿飞被擒的荒诞危机) → 转折(李寻欢救场) → 高潮(方明揭示如意魔刀奥秘)
**副线（阿飞堕落线）：** 起点(行尸走肉) → 推进(被胖女人擒住) → 转折(陷入肉山) → 高潮(被李寻欢所救但已丧失斗志)

### 爽点机制

| 借刀打脸 | 方明用小李飞刀技法击败郭嵩阳 | L2 | 用别人的成名技打败别人，双重羞辱 |

## 四、感官与画面（感官层）

### 环境基线

| 维度 | 详细设定 |
|------|---------|
| 时间 | 白天（枫林）→ 傍晚（草屋）→ 夜晚（大屋） |
| 天气 | 晴（枫林）→ 阴（草屋）→ 无风（大屋） |
| 光线 | 枫林散射光 → 草屋昏暗 → 大屋子烛火通明，食物冒着热气 |
| 空间 | 枫林空地 → 阿飞草屋（破败）→ 大屋子（巨大的空间，堆满食物） |
| 气味 | 枫叶清香 → 草屋霉味 → 炸子鸡等食物浓郁香气 → 肥肉腥味 |
| 声音 | 剑啸 → 断指咔嚓 → 门框垮啦悲鸣 → 食物咀嚼 → 大欢女菩萨笑声如雷鸣震屋 |

### 关键画面场景

**场景1：枫林决战**
> 方明如灵鹫飞起 → 郭嵩阳人剑化黑蛟 → 刀光一闪 → 飞刀插在肩胛 → 仅入两分

**场景2：吕凤先自断手指**
> 左手抓住右手三根金属手指 → 咔嚓掰断 → 扔在泥地 → 无血

**场景3：胖女人掳人**
> 门框挤破 → 两人捆一起还大的身材 → 断剑刺入咽喉却弹出 → 拔剑抹血扛人飞走

**场景4：大欢喜女菩萨本体**
> 一座肉山 → 肥到眼睛脖子都看不见 → 肚皮如海绵 → 大笑时全身肥肉震动 → 整间屋子簌簌发抖

### 氛围渲染
- **枫林决战氛围：** 肃杀、凌厉、一瞬定胜负的寂静
- **草屋氛围：** 破败、绝望、行尸走肉
- **大屋氛围：** 荒诞、窒息、恶心与压迫感并存
- **魔教揭秘氛围：** 庄重、神秘、知识碾压带来的压迫感
