## 二、爽点设计（爽点层）

### 套路识别
1. **境界碾压** — 方明讲神刀斩境界（化繁为简），碾压在场所有人的认知上限
2. **视觉系必杀** — 天魔杀身式将大欢喜女菩萨一刀削成骷髅（暴力美学极致）
3. **身份反转闹剧** — 方明暗示自己是王怜花→再自称开玩笑（先惊吓再解压）
4. **决战宣言** — 方明向李寻欢提出生死对决（上官金虹式的flag）

### 情绪弧线
**主线（魔教揭秘→神刀斩降维）：** 起点(质疑如意刀极限) → 推进(神刀斩设定) → 高潮(天魔杀身式展示) → 余波(转向李寻欢对决)
**决战线（方明vs李寻欢）：** 起点(暗示身份) → 推进(坦白目的) → 转折(对决邀约) → 高潮(气机锁定，生死一线)

### 爽点机制
| 爽点 | 触发事件 | 强度 | 机制说明 |
|------|---------|------|---------|
| 武学设定刷新 | 神刀斩境界设定 | L3 | 三万一千一百零四种变化融于一刀的震撼 |
| 暴力美学 | 天魔杀身式·刮肉为骷髅 | L5 | 极致视觉冲击，千刀万剐血肉成泥 |

| 生死对决预告 | 方明向李寻欢下战书 | L4 | 上官金虹式flag带来的死亡悬念 |

## 四、感官与画面（感官层）

### 环境基线

| 维度 | 详细设定 |
|------|---------|
| 时间 | 夜（大屋封闭时间） |
| 天气 | 无风（室内）→ 方明吐气造成室内飓风 |
| 光线 | 烛火通明 → 刀光耀眼 → 大屋屋顶破洞透光 |
| 空间 | 大屋子（室内拥挤）→ 屋顶破洞（大欢喜女菩萨撞出）→ 室外半空 |
| 气味 | 食物香气（炸鸡等）→ 血腥味（天魔杀身式后）→ 呕吐物 |
| 声音 | 胖女人笑声 → 方明呼吸如龙吟 → 血液流动如长江大河 → 刀光呼啸 → 呕吐声 → 脚步逃散声 |

### 关键画面场景

**场景1：神刀斩讲述**
> 方明立于众人中央，语气庄严 → 所有人屏息 → 阿飞短暂忘记林仙儿

**场景2：大欢喜女菩萨逃跑**
> 胖女人如肉墙堆来 → 杯盘弹起如暗器 → 屋顶破开大洞 → 肉山如气球冲天

**场景3：方明运功**
> 深深吸气 → 吐气成飓风 → 体内血流声如长江大河奔腾 → 身体仿佛拔高数寸 → "简直不是个人"

**场景4：天魔杀身式**
> 黑刀出鞘 → 追上半空的大欢喜女菩萨 → 万千刀芒耀眼 → 刀光后只剩一副红色骷髅 → 肥肉化尽

**场景5：方明暗示是王怜花**
> 方明意味深长地看着阿飞 → "你们凭什么认为……我不是王怜花呢？" → 李寻欢冷汗滑落 → 阿飞喉咙咯咯作响

### 氛围渲染
- **开篇理论氛围：** 庄重、肃穆、仿佛听传道
- **决战氛围（天魔杀身式）：** 暴烈、血腥、视觉震撼到窒息
- **战后氛围：** 死寂 → 呕吐声 → 崩溃逃散 → 诡异的安静
- **最终对峙氛围：** 气机如泥沼，一动不能动，千钧一发
