## 二、爽点设计（爽点层）

### 套路识别
1. **酒楼说书·情报扩散** — 孙小红在公共场合传播宝藏信息，江湖动态全貌
2. **金钱帮压阵·铜钱威仪** — 制钱压人头的细节展示金钱帮的霸道
3. **深藏不露·孙小红惊艳出手** — 一掌击飞上官飞，武力反转
4. **狐假虎威·借神锋楼主之名** — 孙小红用方明的威慑力震住向松等人
5. **压轴登场·神锋楼主现身** — 方明无声无息站在屋内，满堂高手无一人察觉

### 情绪弧线
- **起点**：江湖轰动，宝藏传言四起→江湖混乱仇杀遍地
- **推进**：酒楼说书→胡媚杨承祖吃相狼狈→上官飞率众压阵→杨承祖被秒
- **高潮**：孙小红一掌拍飞上官飞 + "你们就不怕神锋楼主？"→向松冷汗
- **终点**：方明白衣压轴登场——无声无息，全场震撼

### 爽点机制

| 爽点类型 | 触发事件 | 强度 |
|---------|---------|------|
| 势力展示 | 金钱帮五大高手齐出，制钱压人头 | L4 |
| 武力碾压 | 杨承祖被向松十七八个耳光打回来 | L3 |
| 反转暴击 | 孙小红深藏不露，一掌击飞上官飞 | L5 |

| 压轴出场 | 方明无声无息现身，全场无人察觉 | L5 |

### 章末钩子

| 类型 | 内容 | 强度 | 回收章 |

## 四、感官与画面（感官层）

### 环境基线

| 维度 | 描述 |
|------|------|
| 时间 | 夜深——客栈酒楼内 |
| 天气 | 夜色深沉 |
| 光线 | "灯火如豆"——昏暗的油灯，酒楼的微光 |
| 空间 | 客栈酒楼内部——中央说书区+墙角食客区+门口通道 |
| 气味 | 酒菜香气（狼吞虎咽的鸡腿和卤猪蹄）+ 酒馆混浊空气 |
| 听觉（≥4） | ①孙小红清亮的说书声 ②杨承祖/胡媚狼吞虎咽的咀嚼声 ③酒杯摔碎的瓷器声响 ④上官飞长啸"水蛇胡媚"的声势 ⑤向松噼里啪啦的耳光声 ⑥制钱放在人头上的微弱接触声 ⑦方明无声无息出现的死寂 |

### 动作场景

**狼吞虎咽**："呲啦！一只鸡腿已经被那年青人啃了下去，而那女人上嘴皮与下嘴皮一碰，一份卤猪蹄就从桌面上消失了"——画面感极强，反衬其狼狈。
**杨承祖突围**：银枪疾刺→门外伸手→噼里啪啦十七八个耳光→被打晕→手不能招脚不能闪。
**孙小红出手**："轻轻一拍！半空中的子母钢环就猛地下坠，居然被一掌深深嵌进桌子上"→"单掌纷飞，似颠倒五岳"→上官飞撞墙吐血。
**方明现身**：满堂高手皆无察觉→一个白衣人已站在屋子里→"咯咯！向松、高行空、还有燕双飞的手已经在发抖"。

### 氛围渲染

- **江湖混乱氛围**：宝藏消息传出后，数十次仇杀，死伤狼藉的紧张气氛
- **说书氛围**：灯火如豆，孙小红清亮的声音在酒楼回荡，听众聚拢
- **压迫氛围**：上官飞率金钱帮众压境，制钱放人头上的无声威慑
- **反转氛围**：孙小红从"说书小姑娘"到"一掌击飞上官飞"的巨大反差
- **压轴氛围**：方明无声无息出现，满堂高手颤抖——绝对的武力压迫感
