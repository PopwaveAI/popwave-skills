### 文件层级角色

| 层级 | 说明 | 文件 |
|------|------|------|
| **基础设施层** | 被「没钱修什么仙」改造的基础世界观模板 | 经典修仙框架 |
| **世界观层** | 将现代制度投射到修仙世界的核心参考域 | 现代金融体系、赛博朋克反乌托邦 |
| **叙事结构层** | 影响叙事方式、节奏、阅读体验的域 | 教育体制讽刺、黑色幽默荒诞喜剧、游戏系统流 |

### 文件功能

- **参考域(6个)**：每个域独立归档本作中该维度的转化清单，含「参考来源→本作设定→转化程度→chXX证据」的完整溯源链
- **跨域参考索引**：六域的聚合分析，含密度统计、转化亮点（各域最精彩的转化案例）、创意版图（交叉关系图）
- **作者原创贡献**：提取那些无直接参考来源或改编极大的原创设定——这些是本书无法被归类为「缝合怪」的核心创新

## 数据范围声明

> ⚠️ **重要**：所有创意溯源数据均基于**卷1（ch001~ch250）** 推断，不涵盖卷2及后续章节内容。

### 数据来源

| 来源 | 说明 |
|------|------|
| 正文 | 没钱修什么仙 ch001~ch250（熊狼狗著） |
| 设计包 | `设计/setting/` 下的世界观宪法、势力格局、历史驱动力等设定文档 |
| 套路库 | `写作资产/套路库/` 下的结构化写作模板（如桥洞宣言、修仙界牛马经济学等） |
| 卷幕架构 | `设计/卷/卷1-架构.md`、`设计/幕/卷1/` 各幕设计文档 |

### 数据准确性说明

1. **chXX标注**：每条转化记录均标注了来源章节。ch标记后接数字表示章节号，如 `ch001`=第1章，`ch001-87~119`=第1章第87~119段（基于TXT文本行定位）
2. **密度评级**：★★★★★=该书核心设定，贯穿全书/多章节覆盖；★★★★☆=重要设定，有显著戏份；★★★☆☆=单次或零星使用
3. **转化程度**：
   - **直接转化**：现实/经典元素直接移植到修仙世界，核心逻辑不变
   - **概念转化**：保留核心概念但重构了运行机制
   - **细节转化**：将某个细节放大为世界观设定
   - **概念重构**：彻底颠覆原始概念的意义
4. **局限性**：域文件不追求穷尽每条设定，而是侧重于「有参考来源的」「有显著转化的」条目。纯原创设定（无参考来源或改编极大的）归入「作者原创贡献.md」

### 使用建议

- **写作参考**：需要参照某类设定时，先查阅对应参考域文件，再通过chXX跳到正文
- **跨域创作**：参考「跨域参考索引.md」的创意版图，寻找域间交叉的创新空间
- **原创性论证**：需要证明本书原创性时，引用「作者原创贡献.md」
