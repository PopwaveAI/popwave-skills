## 一、人体供应链（备用零件库/活人皮/器官移植）

### 1.1 备用零件库——穷人肉身匹配富人

穷人入学并非为了受教育，而是因肉身与富人的匹配度较高，统一培养后方便作为器官/肉身替换的"备用零件"。

| 条目 | 说明 | 证据 |
|:-----|:-----|:------|
| **核心揭露** | 周天翊临死前揭示：嵩阳高中的穷学生是"备用零件库"——统一培养，肉身匹配富人，方便替换 | ch063 |
| **天魔寄生诀关联** | 天魔寄生法——集合血肉自愈、器官移植、吞噬血肉、气血适配、寄生人体等能力——暗示与备用零件库直接相关 | ch036, ch063 |
| **穷人=零件来源** | 穷人身体=零件来源，天魔寄生法摘取穷人器官 | ch063, L1-01-世界蓝图 |
| **器官供应链** | 穷人=人体零件库，器官移植工业化 | ch063, ch244, L1-03-历史与驱动力 |
| **暗示钩子** | 张羽在功法平台看到"竞争对手"购买天魔寄生法，结合周天翊死前暗示，埋下悬念 | ch036, ch063 |

> **赛博朋克映射**: 穷人=生物器官仓库、富人=零件消费者。身体成为可拆解、可替换的工业零件，人体被彻底物化为商品库存。

### 1.2 活人皮/活衣架——当装备长在别人身上

活人当场剥皮制皮，穿在他人身上作为练功装备，"活衣架"需要每天用精血滋养、运功维护。

| 条目 | 说明 | 证据 |
|:-----|:-----|:------|
| **活人皮真相** | 王总送张羽"真皮"练功皮——由活衣架长期运功维护，当场剥皮制皮 | ch244 |
| **活衣架** | 活人被当作皮具原料架，一生维护一张皮 | ch244 |
| **血神子制皮技术** | 幽冥大学（原幽冥宫）血神子制皮技术支撑活人皮产业 | ch244 |
| **"看宠物的眼神"** | 王总看活衣架剥皮时的眼神——彻底将他人物化 | ch244 |
| **残牛停摆** | 张羽换上活人皮后残牛舍身心诀停摆，象征着穷人本心的消散 | ch244 |

> **赛博朋克映射**: 身体成为可穿戴装备——比"赛博义体"更黑暗的版本：不是机械替换肉体，是将他人肉体直接变作你的装备。

## 二、工业生育（繁育基地/妖裔/开除儿籍）

### 2.1 妖裔繁育基地——从流水线出生的生命

红塔牧业黑暗面：妖兽母体孵化、幼体药物筛选、入职培训幼教版、成品室电击项圈学习——妖裔从出生到死亡全在公司掌控中。

| 条目 | 说明 | 证据 |
|:-----|:-----|:------|
| **工业出生** | 妖裔从出生就有一份工作——入职培训从幼体开始 | ch138 |
| **红塔繁育基地** | 妖兽母体孵化→幼体药物筛选→入职培训幼教版→成品室电击项圈学习 | ch138, L1-04-物种与天赋 |
| **幼体入职培训** | 电击项圈强制学习，最佳员工公司魂洗脑（如安安） | ch138-ch139 |
| **妖裔层层细分** | 繁育基地→孵化室→寝室→学历→种族→层层细分歧视链 | ch174 |
| **研究权vs繁育权** | 繁育权利润大但后代血脉牵扯因果风险——身体作为生产资料 | ch174 |
| **妖裔商品化** | 每个妖裔都有标价，被当作商品买卖 | ch138-ch139, L1-04 |

> **反乌托邦映射**: 类似《美丽新世界》的孵化场——生育完全工业化，生命从诞生前就被设定好社会功能。比《银翼杀手》的复制人更彻底：复制人至少还有"出厂设置"，妖裔从胚胎起就被纳入公司财务表。

### 2.2 开除儿籍——比死亡更可怕的惩罚

被父亲/公司从"儿子"名单中除名，种在流水线上吃喝拉撒到死，还要继续还繁育贷款。

| 条目 | 说明 | 证据 |
|:-----|:-----|:------|
| **开除儿籍惩罚** | 被种在流水线上吃喝拉撒到死，还有繁育贷款要还 | ch075 |
| **虎云涛的恐惧** | 四肢奔跑龙鳞竖起→父亲精血注入→开除儿籍恐惧→工厂大门变巨龙之口意象 | ch112, 幕4 |
| **打工即开除儿籍** | 读不了书只能打工的妖怪会被父亲大人直接开除儿籍 | ch075 (full_text) |
| **开除儿籍恐惧驱动** | 对打工的恐惧、对开除儿籍的恐惧——敦促妖裔拼命修炼 | ch075, ch11295 |

> **反乌托邦映射**: "开除儿籍"是一种比死亡更极端的威胁——不仅剥夺身份、尊严，还让人沦为永不停止的生产机器，且欠着永远还不完的债。这是对《1984》"思想犯"和《我们》"注销"概念的再升级。
