# 套路：信息即武器·符咒改造战

> **定义**: 在昆墟的信息战体系中，符咒不仅是施法工具，更是获取、修改、拦截和伪造信息的手段。高手能自己改造符咒来显示假数据（如威压符伪造存款余额）、隐藏真实信息（如隐藏灵根数据）、反制对手的调查（如「你不配知道」）。核心方法论：「以钱制钱」——用对手的规则对抗对手，而这一切的基础是信息能力。

## 四段式结构

| 段位 | 功能 | 说明 |
|:----|:-----|:-----|
| 起 | 认知突破 | 角色发现符咒「不是固定的」——可以自己改造（如张羽震惊「符咒还能这么改？」） |
| 承 | 方法论传授 | 解释信息战的逻辑：钱驱动人，信息驱动钱——要操控信息就要靠高明的符咒 |
| 转 | 现场演示 | 发动改造后的符咒——展示假数据、隐藏真实信息、反制对手调查 |
| 合 | 信念宣誓 | 「符咒是最有用的」「以钱制钱」——信息战是穷人的核心武器 |

## 核心元素

| 元素 | 本场示例（ch059-ch060） | 作用 |
|:----|:----------------------|:-----|
| 符咒可改 | 张翩翩自己改了威压符——显示假的数据 | 打破「符咒只能按原样用」的认知 |
| 信息操控 | 用改过的威压符展示699分+25亿——实际只有25万 | 以假乱真的信息压制 |
| 反调查 | 何大有天眼符查看——「你不配知道」 | 用高级符咒反制低级符咒 |
| 数据隐藏 | 张翩翩隐藏张羽「???」和白真真「未知新型灵根」 | 保护底牌的保密手段 |
| 方法论 | 「有钱人靠的是钱……最容易被钱驱使，也最容易被钱震」 | 以钱制钱的核心逻辑 |
| 信念 | 「符咒是最有用的！什么道术、武功都没有符咒有用」 | 信息战信仰的宣言 |

## 适用场景
- 主角/同盟在弱势时需要通过信息造假/隐藏来保护自己
- 展示主角团中「信息战专家」角色的能力
- 对抗装备/等级远高于自己的对手——不打架，打信息差

## 变体
- **天眼符版**: 用天眼符获取对方信息的同时，用更高明的符咒反隐藏
- **信用版**: 伪造灵根/天赋/成绩数据来获得资源
- **舆论版**: 修改公告/成绩单/排名来影响学院舆论
- **情报版**: 拦截/篡改神职系统中的信息流——影响神的判断
