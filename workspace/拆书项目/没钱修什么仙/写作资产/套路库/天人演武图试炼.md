# 套路归档：天人演武图试炼

**套路类型**：道心考验 / 试炼机制

---

## 01 套路核心

金丹真人星火亲绘的天人演武图，是一幅特殊的试炼画卷。只有五米内才能看清真容，观看者会直面"心中惊怖"——即心灵最深处的恐惧。试炼不是测试意志力强弱（普通道心测试），而是直接敲打人心灵上的弱点，不给人用心法抵抗的机会。只有灵活调整思想、战胜内心恐惧的人才能通过。

**三关机制**：
1. **克服惊怖**（道心关）→ 直面心中恐惧，蜕变出新的思想
2. **参悟演武图**（悟性关）→ 看穿图画中隐藏的武学
3. **练成武道**（技能关）→ 真正练成其中武功，引起演武图反应

**原文锚定**：
> 「参悟者会直面心中惊怖」
> 「只有坚定精神，战胜内心的弱点，方能参悟画中奥妙」
> 「天人演武图，针对的是心中惊怖，是直接敲打人心灵上的弱点，不给他们用心法去抵抗的机会，测试的就是他们最本真的反应」
> 「挺过第一关，克服心中惊怖之后，接下来才是真正的参悟」
> 「而过了参悟这一关后，最终到底能不能练成，那又是另一件事了」

## 02 应用场景

| 场景类型 | 说明 |
|----------|------|
| 修仙选拔/考试 | 师父选弟子、学校招生、宗门考核均可使用类似机制 |
| 角色深度揭示 | 每个人的幻境不同，直接暴露内心最深的恐惧和弱点 |
| 道心进阶剧情 | 角色通过试炼实现思想蜕变 |
| 群体对比 | 多角色同场试炼，形成高低立判的对比 |

---

## 03 情绪锁定

| 情绪 | 说明 |
|------|------|
| 😰 **恐怖/压迫** | 被内心恐惧直击的感觉——催收天雷、分数巨人、全场嘲笑 |
| 💪 **蜕变爽感** | 从恐惧到"我不怕了"——"有种劈死我""从头开始" |
| 🤔 **悬念/好奇** | 到底谁能练成？到底哪套武功是真的？ |
| 😂 **荒诞幽默** | 每个人的恐惧都有当代社会特色——债务、考试、社交羞辱 |

---

## 04 变体与演化

| 变体 | 描述 | 示例 |
|------|------|------|
| 多重武学解读 | 不同人看到不同的武功 | 拳掌腿指爪刀剑枪棍九种 |
| 集体试炼 | 多人同时受试，互不干扰但也互相影响 | 12名学生同时试炼 |
| 幻境个性化 | 每个人内心不同→幻境不同 | 炼天极(考试)→白真真(催收)→钱深(分数巨人) |
| 思想蜕变路径 | 过关方式不是硬扛，是换个思路 | 妄念/躺平/老赖/摆烂/爆孝 |
