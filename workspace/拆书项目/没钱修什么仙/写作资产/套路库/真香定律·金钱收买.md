# 套路：真香定律·金钱收买

> **定义**: 敌方角色（往往是之前打压主角的权威人物）在验证了主角的真实身份/实力后，果断切换态度——从打压转为送礼/送钱/道歉，完成180度态度大转弯。主角嘴上清高看不起金钱交易但身体很诚实（嘴角压不住）。

## 四段式结构

| 段位 | 功能 | 说明 |
|:----|:-----|:-----|
| 起 | 打压者盘算 | 敌方角色验证后确定主角有真背景（如金丹弟子），决定用"最后手段"——送礼 |
| 承 | 送礼策略纠结 | 送什么？设想A→否定（会打自己脸），设想B→否定（对方不缺），最终选定送钱 |
| 转 | 真香反转 | 主角嘴上说看不起金钱交易，但看到转账数字后嘴角压不住 |
| 合 | 和解达成 | 送礼者道歉+给好处，主角给台阶下，双方达成表面和解 |

## 核心元素

| 元素 | 本场示例（ch033） | 作用 |
|:----|:------------------|:-----|
| 打压者转型 | 王海从孤立张羽→主动送钱求和 | 用极端反差制造爽感 |
| 送礼路线纠结 | 炼体药→不行（自然炼体打脸）→功法→不行（金丹弟子不缺）→送钱 | 衬托主角地位的提升 |
| 一次到位逻辑 | "既然要送钱，那就一次送到位" | 拉高礼金数额 |
| 真香表演 | "看不起金钱交易"→"嘴角压不住"→"这不是钱的关系，是诚意" | 笑点和爽点兼得 |
| 附加好处 | 旷课自由、道歉、"我的体育课就是你的体育课" | 一整套地位认可 |
| 台阶 | "老师你也是为我好" | 给对方体面，自己也体面 |

## 适用场景
- 敌人/反派发现主角有真正的大背景后台后，立即转为送礼求和
- 主角穷但装逼有原则——看到钱又真香的场景
- 主角和某个权威人物从敌对到表面和解的过渡

## 变体
- **资源版**: 送出丹药/灵材/功法（而非金钱）
- **职位版**: 给主角安排肥缺/职务/特权
- **联姻版**: 要把自己女儿/家族女子嫁给主角
- **真心版**: 在送礼过程中被主角的品德打动，由假转真
