# 套路：道心修炼体系

## 定义
系统性的心志/意志力提升方法，通常包含多种路径选择。

## 适用场景
- 角色需要提升精神层面的实力
- 突破瓶颈的关键机制
- 引入新的修炼资源/场所

## 本章应用
- 第155章：问道堂的三种道心修炼方法
  1. 体验强者人生路（接近强者道心）
  2. 极端痛苦刺激（副作用大/见效慢）
  3. 心法场景修炼（最推荐，最高效）

## 心法场景机制详解

```
原理: 根据修炼者心法的类型，生成对应的高强度场景
目的: 在场景中高强度锻炼心法，提振道心
时长: 根据场景复杂度决定
多人模式: 生成同时适应多人心法的综合场景

示例（玉星寒）:
  心法: 天武炼心诀
  场景: 公司保安 vs 破产罢工工人
  身份: 保安队长（代表绝不肯少赚一分钱的公司）
  核心冲突: 极激烈的生死战场
```

## 写作要点
- 道心是软实力的核心，修炼方式要体现精神层面的对抗
- 心法场景要映射角色内心深处的价值观冲突
- 场景设计要有现实讽刺意味（如保安vs工人）
