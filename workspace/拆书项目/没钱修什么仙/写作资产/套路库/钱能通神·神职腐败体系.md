# 套路：钱能通神·神职腐败体系

> **定义**: 八部正神和功曹小神本质上是一个以金钱驱动的官僚体系——「神就是拿钱办事」「没有钱，八部正神如何运转昆墟？」。神的审核、符咒的效果、信仰的等级，全部可以靠充值和打钱来操作。腐败不是个例，而是制度化的运行逻辑。

## 四段式结构

| 段位 | 功能 | 说明 |
|:----|:-----|:-----|
| 起 | 神的傲慢/公正表演 | 神摆出高高在上的姿态，宣称公平公正（如孙甲乙「神卑未敢忘昆忧」），实际在看人下菜碟 |
| 承 | 打钱通道 | 通过特定符咒（好感符）或渠道（私下转账、公开转账）向神账户打钱，自动隐藏转账记录 |
| 转 | 态度180度大转弯 | 到账后神的语气、态度、原则全部翻转——「信仰之心坚定」「敬神」=愿意给钱 |
| 合 | 世界观揭示 | 点明昆墟的钱能通神体系——符=充值，神职部门的运行依赖金钱流量 |

## 核心元素

| 元素 | 本场示例（ch057） | 作用 |
|:----|:------------------|:-----|
| 神的表面公正 | 孙甲乙罢落富豪考生（身价过亿不过就不过）、学霸考生（犹豫太久意志不坚） | 建立神的高傲公正人设——然后打破它 |
| 好感符=打钱 | 「好感符，一种能快速增加神灵好感的符咒……能把钱打进对方的帐户」 | 冠冕堂皇的名字掩盖行贿的本质 |
| 到账就变脸 | 「一个高中生就如此敬神，足见信仰之心坚定」——之前还说不过就不过 | 180度反差的喜剧效果 |
| 腐败不自知 | 孙甲乙不觉得自己腐败——他认为收钱=被敬神 | 神职人员的自我合理化 |
| 底层逻辑揭示 | 「在昆墟，神就是拿钱办事」「符=充值」 | 世界观的核心设定 |

## 适用场景
- 需要揭示修仙世界政治/经济体系黑暗面的情节
- 主角以穷人身份对抗系统不公时的背景设定
- 姐姐/引路人解释「这个世界的真相」的场合

## 变体
- **赛博版**: 贡献点/功勋兑换制度——表面公平，实际被上层操控
- **宗门版**: 长老会/功德殿的「人情世故」——内门弟子靠送礼获得资源
- **科举版**: 考官的腐败——但以神的身份包装成「天意」而非「人意」
