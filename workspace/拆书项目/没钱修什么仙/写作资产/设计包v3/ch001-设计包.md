## 2. 爽点设计（爽点层）

### 2.1 本章套路

#### 套路1：面试修罗场
| 阶段 | 原文描述 |
|:----|:---------|
| 铺垫（期望） | 张羽自恃全校第一，信心赶往面试 |
| 反转（碾压） | 面试官逐一揭露与市区学生的巨大差距 |
| 连环（打击） | 多校面试中条件越来越离谱、越来越绝望 |
| 落点（绝望） | 简历被扔进纸篓，意识到自己什么都不是 |

#### 套路2：穿越开局烂摊子
| 阶段 | 原文描述 |
|:----|:---------|
| 铺垫（穿越） | 天台布娃娃仪式后醒来 |

| 连环（绝望） | 断水断电、母亲离开、身体被药掏空 |
| 落点（悬念） | 掌心符号填充1/10，金手指尚不明朗 |

### 2.2 情绪弧线
**起点→推进→高潮→终点**
> 信心满满（面试前）→ 被碾压（面试打击）→ 连环绝望（多校面试）→ 谷底（回家）→ 虚假希望（补习班）→ 债务深渊（以贷养贷）→ 崩塌（母亲离开）→ 穿越重生（世界观重塑）→ 二次绝望（70万债务）→ 微弱悬念（掌心符号）

### 2.3 爽点机制
- **信息差爽点**：面试官揭示的"学霸真相"——读者和张羽同步受冲击
- **数值爽点**：睡眠时间2h vs 5h、9年差1万h——数字化的残酷
- **打脸爽点**：（延迟满足）面试官轻视张羽→预计后期打脸
- **情绪爽点**：穿越后"这不是中国了"的认知颠覆感

### 2.4 章末钩子
- **类型**：悬念钩子（金手指觉醒）
- **内容**：掌心透明符号被黑色填充了十分之一，意味着什么？
- **强度**：L3
- **预期回收章**：ch004（请神仪式完成）

## 4. 感官与画面（感官层）

### 4.1 环境基线
- **时间**：白天→深夜（面试时段）→次日晨
- **天气/光线**：教室内明亮日光灯 → 出租房昏暗 → 天台霓虹灯光闪烁
- **空间**：面试教室（压抑）→ 街头（穿梭多个学校）→ 出租房（破旧狭小）→ 天台（开阔但诡异）
- **气味**：无特别描写
- **声音**：叫号声、手机震动、小贷短信提示

### 4.2 氛围渲染要点
- 面试教室压迫感——面试官的冷淡语气与张羽的紧张对比
- 天台诡异场景——猩红烛光摇曳、布娃娃纽扣眼睛、街道霓虹投影仙人推荐修仙贷
- 断水断电的出租屋——斑驳红砖墙上的理财广告、黑暗中的寂静
- 穿越瞬间的天旋地转——记忆碎片冲击下的眩晕感
