## 2. 爽点设计（爽点层）

### 2.1 本章套路
**套路: 阶层固化揭露（穷人修仙困境）**
- 通过跳楼事件和白真真的解说，系统性地展现修仙世界的阶层分化
- 核心冲突：天赋vs资源，穷人即使有天赋也追不上富人的资源碾压
- 伏笔铺垫：十几年没有平民子弟考上十大

### 2.2 情绪弧线
```
好奇(羽书功能) → 自信(升级潜力) → 轻松(食堂互怼) → 震惊(跳楼现场) → 恶心(贫穷歧视言论) → 压抑(阶层真相) → 暗惧(自己也可能跳楼) → 投入(体育课练习) → 愤怒(王海打压) → 憋屈(被威胁)
```

### 2.3 爽点机制
| 序号 | 爽点类型 | 触发时刻 | 原文锚点 |
|:----:|:---------|:---------|:---------|
| 1 | 系统探索爽 | 羽书"专精"功能发现 | "一次只能专注于提升一个技能" |
| 2 | 潜力展示爽 | 2天快把健体36式从1级升到3级 | "两天时间就快把一门基础武学从第1级升到第3级" |
| 3 | 认知碾压爽 | 白真真毒舌段子（没绝育、年级导管第一） | "不是年级导管第一" |
| 4 | 反压迫憋屈（蓄力） | 王海的无理打压成为后续爆发引线 | "你的问题我会跟你班主任反应的" |

### 2.4 章末钩子
**短钩**: "你的问题我会跟你班主任反应的，你好自为之吧。"
**长钩**: 王海的敌意+学校阶层真相+张羽欠债背景，三重压力交织，为下一章班主任出场做预告。

### 4.2 氛围渲染要点
- **跳楼现场**："砰的一声"先声夺人，"脑袋开花"画面冲击，"地上溅开的血水"持续刺激感官
- **分数歧视台词**：用极端荒谬的台词制造黑色幽默感（学霸跳十层才死得掉），强化世界观的疯狂底色
- **体育课隔离感**：王海"带头孤立"→同学们"下意识地不再接近张羽"→"无形的无人区"，用空间描写表现人际关系恶化
- **身体极限感**："撕裂般的痛苦""针扎般的巨疼"感官动词持续轰炸，让读者感受修炼的残酷
