## 2. 爽点设计（爽点层）

### 套路名称：金手指滚雪球·挂机流

**情绪弧线**：紧张释放（法力回收成功）→ 价值认知（被动挂机的兴奋）→ 经济压力（还款后穷）→ 被针对（王海的孤立）→ 逆势狂升（被孤立反而更高效修炼）→ 爆发（3天升4级法力+0.5）

**爽点机制**：
- **被动收益爽**：周天采气法"睡觉也在变强"，读者代入获得"挂机升级"的快感
- **认知差爽·双线**：别人以为他被孤立了很惨→实际上他正好全情投入吐纳；王海以为他背后有"卖药大佬"→实际上他纯靠自己
- **数值反馈爽**：每一段都有一个数值变化：1级(11/20)→1级(16/20)→2级(0/40)→4级(5/80)；法力7.7→7.8→8.3——连续、密集、可感知
- **反差爽**：每天高强度修炼还被催收逼债→但仍然守住底线不扎药，建立"穷且自律"的角色形象

**钩子**：
- 王海的怀疑会升级吗？赵天行的跟踪会揭开什么？
- 4级周天采气法的被动效率是主动的四成半——后续升级效率如何？
- 三天无工作机会，经济压力进一步加剧

## 3. 角色与人设（角色层）

### 张羽
- **行为锚定**：极度自律、目标明确、能忍常人所不能忍；被孤立不怒反喜（终于可以专心修炼了）
- **关键对白**（内心）："无法反抗，那便只有享受。"——核心人设总结：在压迫下寻找最优解
- **关键行动**：还款优先（宁可穷也要降低催收风险）→理性决策

### 王海
- **行为锚定**：利益驱动的体育老师，把学生当药品客户；对"市场入侵"高度敏感
- **关键对白**：无直接对白，但从行动可见其控制欲和商业思维——让赵天行传话孤立张羽
- **深层动机**：担心张羽的"进货渠道"影响他的药品销售额

### 赵天行
- **行为锚定**：听话的好学生、王海的执行工具；对张羽有善意但不敢违抗老师
- **关键对白**："张羽同学有点犟啊，跟老师道个歉认个错……"——思维定式是"服从权威"

## 4. 感官与画面（感官层）

### 环境基线
- **地铁**：喧嚣的城市公共交通中，张羽闭目运行周天采气法——**闹中取静的苦修**
- **练功场**：筋肉隆起的学生们扎针、噼啪炸响、野兽嘶吼——**赛博健美房的暴烈感**
- **小房间**：无人、安静、和练功场的喧嚣形成强烈对比——**张羽的"修炼圣地"**
- **教室**：语文/数学课上，张羽闭目吐纳——**表面上课、实则修炼的双重生活**

### 氛围要点
| 氛围 | 触发场景 | 感官关键词 |
|------|----------|-----------|
| 丰收喜悦 | 地铁上法力回归 | 如溪流滚滚向前、微笑、嘴角的弧度 |
| 压抑/紧迫 | 还款场景 | 瞬间降到的数字、催收压力 |
| 暴力热血 | 练功场群像 | 噼啪炸响、野兽嘶吼、汗味、药味 |
| 孤寂专注 | 小房间修炼 | 无声、静坐、只有呼吸的节奏 |
| 狂热冲刺 | 三天不间断修炼 | 只看到数字增长的满足感、时间碎片全利用 |

### 关键画面
1. **地铁神功**：面色苍白踏入地铁→几站后面色红润双目有神→法力在体内自动运转——**"伤势"瞬间恢复的反转画面**
2. **练功场群魔**：小巨人般的学生扎针→体表噼啪炸响→汗水蒸腾——**肉体炼体的暴力美学**
3. **一人在墙角**：宽阔的练功场，学生们热火朝天炼体→角落里张羽独自离开→小房间里独坐吐纳——**从热闹到孤寂的镜头推移**
4. **三个场景混剪**：体育课、语文课、午休、广场——张羽全在吐纳——**蒙太奇式表现"疯狂修炼"**
