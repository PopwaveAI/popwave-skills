## 2. 爽点设计（爽点层）

### 套路名称：扮猪吃虎·低调制霸

**情绪弧线**：荣誉感（走神率第一被表扬）→ 压力（法赛目标10.0）→ 被嘲讽（白真真的"好意打击"）→ 隐藏自信（内心盘算→我有金手指我怕谁）→ 温馨（白真真投喂）→ 警觉（被跟踪）

**爽点机制**：
- **走神率排名爽**：0%对0.03%→第一，且"连续4天0走神率"，张羽的刻苦得到公开认可
- **认知差爽**：所有人都以为他砸钱租灵根才提升这么快（严老师原话："租了灵根还是租了高灵机修炼室"）→只有读者知道是羽书+自律
- **独白自信爽**：内心估算"链气阶段技能最高10级"→"高中三年后也未必能达到的水准"→间接透露周天采气法的强劲上限
- **白真真天才展示爽**：通过白真真"化筷为剑"的高难度操作，设定"天才天花板"，也为后续角色立威

**钩子**：
- 张羽能否在8天内将法力从8.3冲到10.0？
- 赵天行跟踪要干什么？
- 法赛规则是什么？真正的高手什么样？

## 3. 角色与人设（角色层）

### 张羽
- **行为锚定**：全天候苦修（下课也不停）、表面淡定内心狂算（经济账和修为账同时算）、对朋友厚脸皮但要强
- **关键对白**："阿真，你不用考虑我的什么自尊心、面子之类的东西，以我俩的关系，直接给我打钱都没问题。"——对真正朋友完全放开的厚脸皮
- **关键对白**："绝对可以！轻易可以！"——内心狂喜，读者共感到金手指带来的底气

### 白真真
- **行为锚定**：年级第一天才少女，外表清冷高傲，实际对张羽关心（投喂、用玩笑提醒他法赛残酷）
- **关键对白**："被无数强者反复侮辱，成为有钱人们的垫脚石……道心破碎，宛如死狗的模样"——用夸张的毒舌掩盖真实担忧
- **能力展示**：已练成高中基础剑法并化入筷子——天赋远超同龄人
- **经济状态**：打工（在端盘子）维持学业，和张羽是同类人

### 严老师（法力课老师）
- **行为锚定**：严厉但赏识尖子生，借表扬张羽来激励全班，同时也推销静心剂
- **关键对白**："一分钱一分收获，仙道之路上没有免费的东西，不砸钱是走不长远的"——观念输出：这个世界连修炼都要氪金

### 赵天行
- **行为锚定**：被王海派去跟踪张羽的"眼线"，笨拙地执行任务（被发现是迟早的事）
- **关键对白**（上一章的延续）：无本章对白，纯跟踪动作

## 4. 感官与画面（感官层）

### 环境基线
- **法力课静室**：摄像头全方位监控、安静到呼吸可闻、学生盘坐吐纳——**高度规训的修仙教室**
- **食堂**：嘈杂的人声、碗筷碰撞、平价饭菜的朴素气息——**凡俗生活的温暖角落**

### 氛围要点
| 氛围 | 触发场景 | 感官关键词 |
|------|----------|-----------|
| 监控压力 | 静室吐纳 | 摄像头镜头、统计走神率、数据化的修炼 |
| 竞争气氛 | 排名公布 | 惊讶的眼神、微妙的脸色变化、空气中的紧张 |
| 敬佩/羡慕 | 表扬张羽 | 全班目光聚集、闭目不为所动的冷峻 |
| 轻松毒舌 | 食堂对话 | 调侃的语气、筷子碰撞、肉被抢走的郁闷 |
| 温馨关怀 | 投喂场景 | 剩下的半份饭菜、故意留下的肉、不戳破的关心 |
| 悬疑跟踪 | 食堂外 | 高大的身影、黑暗中死死盯着大门 |

### 关键画面
1. **走神率榜单**：大屏幕上第一名张羽0%，第二名白真真0.03%——视觉化的"学霸霸权"
2. **全员注视中的闭目**：全班学生目光齐刷刷看向张羽→张羽仍闭目吐纳——**漠视聚光灯的专注力**
3. **筷子剑法**：白真真的筷子化作剑招，铛铛铛挑飞张羽的筷子，轻松抢走肉——**无声炫技**
4. **跟踪的背影**：高大身影缩在楼道阴影里，凝视食堂大门——**悬疑感拉满**
5. **文具店的误判**：赵天行看到张羽进文具店→心中暗想"难道这里就是张羽进货的地方？"——**认知错位的喜剧**
