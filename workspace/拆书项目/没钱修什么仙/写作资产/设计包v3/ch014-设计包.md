## 2. 爽点设计（爽点层）

### 套路名称：组队·捧哏逗哏模式（张羽×赵天行）

**情绪弧线**：准备冲刺（买药→决心不睡觉）→ 喜剧跟踪（笨拙的赵天行被发现）→ 反差打脸（张羽主动应聘被拒→赵天行被选中）→ 急中生智（捆绑拉赵天行入伙）→ 信息铺垫（金丹真人画展的伏笔）

**爽点机制**：
- **喜剧反差爽**：张羽信心满满举手应聘→司机选了两米高的赵天行→张羽被叫"细狗"——主角出糗但无伤大雅的幽默感
- **急智爽**：被拒后瞬间反应"我们是亲兄弟，发过誓一起上"→强行带自己入场——主角的社交智慧
- **信息铺垫爽**：金丹真人、天人演武图、隐藏绝世武学——铺垫后续大机遇，读者产生期待
- **赵天行人设反转**：从"老师的狗腿子"→被张羽的专注力折服→被拉入安保队伍→角色开始"由黑转粉"

**钩子**：
- 画展会出什么事？天人演武图真的藏有绝世武学吗？
- 赵天行会被张羽"策反"吗？
- 白真真会不会出现？（看ch015知道出现了，这里已是伏笔铺垫）

## 3. 角色与人设（角色层）

### 张羽
- **行为锚定**：为了法赛全力冲刺（买神经填充剂不睡觉）、善于社交谈判（跟老板杀价→发现赵天行→忽悠司机）、务实主义者（细狗不被选中"那就蹭赵天行的车"）
- **关键对白**："我是学生，能不能把贵的便宜点卖我？"——厚脸皮砍价，接地气人设
- **关键对白**："老板，我们两个是亲兄弟，发过誓不论是一个人的工作，还是一群人的工作，全都一起上的。"——街头智慧，随口编话

### 赵天行
- **行为锚定**：笨拙执行任务（跟踪技巧小学生级别）、对王海言听计从但内心矛盾、被张羽的自信所折服
- **关键对白**："我不是来打工的，而且太晚回去我妈会担心的"——本质还是妈宝乖乖男
- **成长暗示**：跟着张羽开始"越界"（打工、吃炸鸡、违背王海的叮嘱）

### 面包车司机
- **行为锚定**：看重外形远超实力、识货（一眼看出赵天行的肌肉底子）、专业素质高
- **关键对白**："细狗不要。做安保缺乏安全感，去挡飞剑都比别人少挡一半啊。"——幽默、专业视角

### 文具店老板
- **行为锚定**：精明的商人，不理张羽的"学生优惠"说辞、但能给出合理低价
- **关键对白**：头也不抬地问"要贵一点的，还是便宜一点的"——见惯了穷学生的从容

## 4. 感官与画面（感官层）

### 环境基线
- **文具店**：展柜像药房一样摆满镇定剂、兴奋剂、专注药剂——**赛博修仙的日常：学校门口卖修炼药**
- **炸鸡店**：油炸香气、暖黄灯光、廉价但温暖——**张羽在这个世界的"奢侈享受"**
- **广场**：污水地面、理发店的吵闹音乐、不好惹的求职者——**底层劳动者的混沌生态**

### 氛围要点
| 氛围 | 触发场景 | 感官关键词 |
|------|----------|-----------|
| 市井交易 | 文具店砍价 | 展柜如药房、扫码付款、肉痛的感觉 |
| 喜剧跟踪 | 路上拆穿 | 巨大身影缩在小巷、被抓包的尴尬、装傻挠头 |
| 炸鸡快乐 | 炸鸡店门口 | 滋滋的声音、油脂香味、口水直流 |
| 专注圣地 | 广场修炼 | 污水味噪音中静坐、纹丝不动的躯干、平稳的呼吸 |
| 反差打脸 | 招聘现场 | 走出面包车的从容→"细狗不要"→震惊的表情 |
| 信息释放 | 面包车路上 | 窗外的城市夜景、金丹真人名字的震撼 |

### 关键画面
1. **文具店"药房"**：摆满镇定剂、兴奋剂、专注药剂的巨大展柜——**世界观画面（学校门口如药店）**
2. **巨大的跟踪**：2米高300多斤的赵天行自以为隐蔽地跟在张羽身后——**喜感的跟踪画面**
3. **吃炸鸡的快乐**：张羽一手炸鸡腿一脸满足在城市夜景下啃着——**底层少年的小确幸**
4. **广场吐纳一镜**：赵天行蹲在旁边观察的两小时时间压缩—赵天行站起走圈→练三十六式→张羽纹丝不动——**对比式画面展示专注力**
5. **震惊脸特写**：司机看着张羽说"细狗不要"→张羽呆滞的表情——**喜剧定格**
6. **面包车上的世界信息**：司机讲述金丹真人→赵天行和张羽的震惊反应——**世界观扩展时刻**
