## 2️⃣ 爽点层

### 套路清单

| 套路 | 触发 | 效果 |
|------|------|------|
| 🌀 **道心试炼奇观** | 每人幻境不同（考试/嘲笑/催收/仪式） | 每个角色的内心弱点被揭露，角色深度瞬间拉满 |
| 🌀 **张羽的反套路操作** | 「不想当弟子就别练成」 | 主角视角的解构，将金丹试炼当作白嫖武功机会 |
| 🌀 **白真真的催收恐怖** | 天雷锁定的压迫感 | 从分数学霸的一面突然展示出巨大的经济压力——角色反转 |
| 🌀 **张羽的两个倒计时** | 真假仪式冲突 | 大悬念——哪个是真的？选错会死 |

### 情绪弧

```
揭开（天人演武图揭幕）
→ 凝重（炼天极被武道意念冲击）
→ 期待（白真真、张羽纷纷上前）
→ 恐怖（白真真幻境：天雷催收）
→ 困惑/悬疑（张羽两个倒计时）
```

### 章末钩子

- **两个仪式倒计时冲突**：一个吐纳后停止，一个参悟后停止，选错=死亡
- **白真真的800万债务**：暴露了角色隐藏的危机
- **赵天行考验失败**：尴尬退场，后续需要解决自尊/自信问题

## 4️⃣ 感官层

### 环境基线

| 维度 | 描述 |
|------|------|
| **视觉** | 李雪莲银发金属光泽+火焰晚礼服；两米的演武图展开；模糊人形逐渐变成清晰中年男子 |
| **听觉** | 众人吹嘘"好画"的尴尬；李雪莲咳嗽打断说话声；"砰"的闷响 |
| **触觉** | 寒意（仪式倒计时）；张羽"一股寒意涌起" |
| **氛围** | 揭幕时的庄重→学生被冲击的紧张→张羽面对两个倒计时的诡异 |

### 氛围要点

- **金丹真人的压迫感**：演武图五米内才有真容，武道意念直接冲击道心，防不胜防
- **众人吹嘘被打脸**：一群人盲目吹图画，李雪莲一句"只有五米内才能看清"让所有人尴尬——讽刺了仙道界阿谀奉承的风气
- **白真真幻境的恐怖感**：从80万到800万的债务累积，从天雷锁定的绝望，画面感极强
- **张羽视角的诡异**：两股同样的寒意、同样的倒计时同时倒数——仪式（金手指）与演武图（试炼）产生未知的冲突
