## 2. 爽点设计（爽点层）

### 2.1 本章套路

**套路1: 魔鬼监工（强制修炼系统）**
- 仪式持续压迫：练体极限后仍被倒计时逼迫继续
- 新层次：炼体的痛苦远超吐纳——"肺喘得像风箱一样，浑身上下的每一寸血肉都传来撕裂般的酸痛"
- 反差：以为适应了→被炼体教做人

**套路2: 贫穷修仙陷阱**
- 炼体=烧钱吃饭：一顿100+，一天几顿
- 时间经济：每个课间都要被压榨
- 核心对比：有钱人扎针吃药 vs 穷人只能靠练和吃

**套路3: 校园卷王竞争（社会比较）**
- 白真真、钱深被张羽卷到精神崩溃
- "连厕所都不怎么去上"的修罗场
- 卷=修仙世界的生存本能

### 2.2 情绪弧线
```
平静(排名第8) → 欣喜(被表扬) → 暗爽(卷到别人) → 果决(策略转型) → 兴奋(发现机制) → 痛苦(炼体极限) → 绝望(还得练) → 崩溃("是人能适应的吗") → 麻木坚持(九天循环)
```
- 整体形态：M形——两次兴奋（排名+发现机制）夹着炼体低谷，最后以麻木的坚持收尾

- **炼体痛苦的递进**：健体三十六式15遍后的极限 → 肺喘如风箱 → 血肉撕裂般的酸痛 → 地板被汗水浸出人形，用身体实感传递压迫感
- **仪式压迫的具象化**：倒计时"10"的数字和"请遵守仪式约定"的公式化语气，与张羽内心的绝望形成反差
- **卷王竞争的视觉化**：白真真瘦了面颊凹陷、钱深的不甘表情、两人先后闭眼吐纳的镜头——无声的修罗场
- **吃饭场面的冲击感**：100多块钱一大桶、十斤饭、整只鸡腿吞下咬碎骨头——贫穷炼体者的生存方式
- **九天蒙太奇**：课间练体→极限→被逼吐纳→循环，用紧凑的节奏带出考前时间感
