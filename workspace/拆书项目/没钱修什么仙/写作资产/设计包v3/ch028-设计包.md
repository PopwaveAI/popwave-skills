## 2. 爽点设计（爽点层）

### 2.1 本章套路

**套路1: 符咒=富人语言（阶层固化揭露）** ★主套路
- 完整展示符咒体系的烧钱结构：篆书购买→会员费→充值祭神→信仰等级
- 八部正神系统：交通部、卫生部等八大神职部门
- 社会剪影：提前交卷的何大有正是这一体系的既得利益者

**套路2: 扮猪吃虎·隐藏实力（冲突铺垫）**
- 何大有以为张羽还是"穷鬼努力"的弱者
- 读者知道张羽散手3级+肉体强度1.25——真正的实力远高于何大有的认知
- 何大有的轻视和先入为主的判断——为下章打脸做准备

**套路3: 世界观展示——分层震撼**
- 八部正神体系的全景展示
- 篆书的运作机制：信仰等级→权限解放→高级符咒
- "穷人和富人之间分界线最明显的一门考试"

### 2.2 情绪弧线
```
愉快(散手升级感悟) → 淡定(道术考试开始) → 懵(题目看不懂) → 烦躁(超纲题) → 压抑(意识到系统不公) → 愤怒(符咒=富人语言) → 恶心(何大有炫富) → 厌恶(何大有内心独白) → 期待(冲突在即)
```

### 2.3 爽点机制

| 序号 | 爽点类型 | 触发时刻 | 原文锚点 |
|:----:|:---------|:---------|:---------|
| 1 | 认知梳理爽 | 符咒体系的全景展示——读者和张羽一起理解"富人语言" | "整个过程就是要用钱堆出来的" |

- **道术考试的压迫感**：鬼画符般的选项（★☆※§之类的符号化题面）——直观呈现"看不懂"的无力感
