## 2. 爽点设计（爽点层）

### 2.1 本章套路

**套路1: 扮猪吃虎·实战碾压** ★主套路
- 何大有以为张羽实力停留在"穷鬼努力"阶段——散手1级水平
- 实际张羽散手3级+肉体1.25级——碾压级的差距
- 第一场秒杀：后发先至，抱摔出界（不到1秒）
- 第二场秒杀：预判轨迹，一脚踹飞
- 第三场硬碰硬：法力恢复碾压

**套路2: 阶级反击（穷打富）**
- 何大有的阶级傲慢：看不起穷人，觉得穷人"是另一个物种"
- 被张羽连秒两场——对傲慢最直接的回应
- 何大有甚至用上了富人家才能练的"小天星掌"——仍然输了
- 输的不只是实力，更是价值观的崩塌

**套路3: 升级质变（技能等级压制）**
- 3级散手 vs 2级散手的实战效果——对招数熟悉度碾压
- 何大有出手的瞬间，张羽已经预判了对方的动作和轨迹
- "只要对手散手等级低于3级，临战反应恐怕都会比我慢上一线"

### 2.2 情绪弧线
```
平静(听规则) → 暗流(对阵何大有) → 紧张(准备开打) → 爽(秒杀!) → 更爽(再秒杀!) → 冷静(对方谨慎了) → 专注(硬碰硬) → 激昂(法力比拼压过对方) → 得意(何大有认输) → 满足(收尾)
```

### 4.2 氛围渲染要点
- **秒杀的速度感**：第一场——"连一秒钟都不到"，"看上去就像是何大有故意配合"——用极短的时间制造视觉冲击
- **何大有的崩溃过程**：震惊→看视频→茫然→不甘→愤怒，情绪层层递进
- **法力比拼的对峙感**：双方掌心相抵、"砰的一声炸响"、额头汗珠、腾腾热气——内家对抗的静态张力
- **阶级对抗的暗线**：何大有钱人底牌（小天星掌）→还是输了，将阶级对抗的胜负具象化到每一次掌力碰撞
- **张羽的从容**：从第一场到第三场，张羽始终控制节奏——读者能感受到"他还有余力"
