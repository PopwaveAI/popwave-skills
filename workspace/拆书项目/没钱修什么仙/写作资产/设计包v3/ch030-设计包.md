## 2. 爽点设计（爽点层）

### 2.1 本章套路

**套路1: 扮猪吃虎·情报战（隐藏实力反向战略）** ★主套路
- 白真真提出隐藏实力的理论：散手等级高不加分，但实战藏一手更有利
- 张羽实战中经历（何大有突然用小天星掌）验证了这一点
- 张羽当晚把散手练到5级——大藏底牌，为下次装逼蓄力

**套路2: 扮猪吃虎·低调装逼**
- 张羽对何大有说"剩一两成"——实际情况远剩更多
- 张羽对白真真说自己赢了——白真真震惊
- "这就是你实力最接近我的时候"——本书最经典的装逼台词之一

**套路3: 世界观展示——道心考试体系**
- 全市统一考试的特殊性
- 灵界面罩的科技感
- 虚拟世界入口的神秘感

### 2.2 情绪弧线
```
得意(打赢何大有) → 警惕(何大有的阶级偏见) → 愉快(装逼赢白真真) → 惊讶(白真真也3级) → 顿悟(情报战) → 满足(夜间练到5级) → 平静(到考场) → 好奇(面罩→虚拟世界)
```

### 4.2 氛围渲染要点
- **战后余韵**：何大有灰白面色的背影vs张羽得意的内心吐槽——画面感极强的胜负对照
- **白真真和张羽切磋的火花**：四只手臂在空中化为道道残影——快节奏的实战对招，默契与竞争并存的友情
- **情报战理论的落地**：张羽听完后当晚就练到散手5级——"想到下次白真真被震惊的模样，都感觉自己的拳头打得更有劲了"——用期待感驱散疲惫
- **考场氛围的转换**：从实战的热闹→道心考场的安静洁白——节奏的冷静过渡
- **道心考试的神秘感**：面罩变形→眼前黑暗→光粒浮现→来到另一个世界——层层递进的异世界入口
