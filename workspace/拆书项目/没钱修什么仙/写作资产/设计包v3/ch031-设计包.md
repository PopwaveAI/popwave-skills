## 2. 爽点设计（爽点层）

### 2.1 本章套路

**套路1: 灵界考试·虚拟实境** ★主套路
- 灵界是十大宗门制造的精神世界，痛觉体验与真实无异
- 疼痛控制器→等级越高越痛→排名越高→晕厥则0分退场
- 硬核考试规则制造压迫感，硬抗刺激

**套路2: 天武觉醒·意志升华**
- 张羽到极限时，天武炼心诀中拳掌腿枪刀剑的意念浮现
- 杀意、斗志、气魄——符合张羽性格的意志出现
- 基础炼心法（静心忍耐）不适合张羽→武功意念才真正契合
- 最后以复仇宣言收尾：「等我爬上去，一定把出题人打得跪地求饶」

**套路3: 内心吐槽·抵抗系统**
- "又是哪个出生想出来的考试题目？" ——经典骂街
- "可别考试考到一半出不去了，来个全市高中生被困虚拟网游"——网文梗
- 从抗拒→自我打气（我爱学习我爱考试）→硬撑→觉醒→狠话——完整情绪弧

### 2.2 情绪弧线
```
好奇(进入灵界) → 吐槽(虚拟网游梗) → 紧张(考试规则宣布) → 抗拒(最讨厌吃苦) → 自我打气(我爱考试) → 痛苦(1级→37级) → 绝望(基础炼心法失效) → 觉醒(天武炼心诀意念) → 振奋(不再难熬) → 狠劲(出题人：等我爬上去就打你)
```

## 4. 感官与画面（感官层）

### 4.1 环境基线

| 场景 | 时间 | 环境特征 | 感官关键词 |

| 精神世界内部 | — | 天武炼心诀意念浮现 | 拳掌腿枪刀剑身影、杀意扑面、武将威武姿态 |

### 4.2 氛围渲染要点
- **灵界的双重性**：既是真实世界又是虚假世界——虚实交替的梦幻感，"看到的、闻到的、摸到的、感觉到的一切都是可以被人为控制的"
- **疼痛等级攀升的具象化**：每升一级=手掌被多一根钢针刺穿，37级=双手双脚37根钢针——用具体数字强化痛感的张力
- **意志与思想的对比**：基础炼心法（静心忍耐/水滴石穿）不适合张羽→天武炼心诀（杀意斗志/一往无前）才匹配——用两种心法的内在冲突展示张羽的性格底色
- **觉醒时的画面感**：拳掌腿枪刀剑的身影浮现，如"跟在绝世武将身后的小兵"——将自己代入战争场面，用画面替代说教，直接振奋精神
- **结局的狠劲**：狞笑+复仇宣言——从被动忍受到主动反击的情绪转折
