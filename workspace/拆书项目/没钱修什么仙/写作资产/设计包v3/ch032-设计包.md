## 2. 爽点设计（爽点层）

### 2.1 本章套路

**套路1: 观察者反转·惊艳打脸** ★主套路
- 李星宇从「按他这个状况撑不下去」→「嗯？撑住了？」→「这个水平的意志力，快2级道心了」
- 情报从1000→10000，标题写"震惊！绝世武道天才"——价值的量化逆转

**套路2: 阶层固化揭露·社会学冷嘲**
- 张羽问跳楼能不能报复出题人→周天翊说有钱人跳楼死不了（付了安保费）
- 「每一个都是付了安保费的」——冷幽默式社会讽刺
- 助理老师看法力效率：「肯定又是个死有钱人」——穷人努力被当成有钱才能做到

**套路3: 教师即销售·黑心盘算** (复用)
- 王海的矛盾：想打低分但又怕错失金丹弟子
- "所有影响自己出货的人，都该被退学"——赤裸裸的利益逻辑
- 想通过体育分数判断张羽身份

### 2.2 情绪弧线
```
惊讶（李星宇：撑住了？）→ 疼痛后遗症(浑身刺痛感) → 决心(不再碰药) → 被误解(有钱人?) → 闲适(午饭吐槽) → 震撼(跳楼的社会真相) → 紧张(王海视线落下) → 不安（王海回忆冲突）→ 压迫（王海最后冷意）
```

## 4. 感官与画面（感官层）

### 4.1 环境基线

| 场景 | 时间 | 环境特征 | 感官关键词 |

| 灵界脱离后 | 上午-临午 | 床位区 | 浑身刺痛的错觉、学生腿软坐倒、有人掏药瓶吞药 |
| 法力考场 | 上午 | 贴片检测区 | 手臂胸口小腹贴片、运转法力、探测仪器 |
| 食堂 | 中午 | 食堂大厅 | 行尸走肉般的学生、哀嚎声、叹气声、骂声 |
| 体育考场 | 下午 | 练功场、评委席 | 筋肉膨胀、野兽嘶吼、青筋暴起、探测贴片数据 |
| 王海的回忆 | — | 过去三周 | 课堂上的冲突、赵天行的报告、金丹真人传闻 |

### 4.2 氛围渲染要点
- **疼痛后遗症的真实感**：明明已脱离灵界但仍浑身刺痛，"一名学生直接腿一软坐倒在地上"——用他人反应强化考试残酷
- **食堂的修罗场**："行尸走肉般的学生"——道心考试后的集体创伤，"道心考试后遗症还没恢复、考试没考好忧心、想着要不要自杀"——三种学生的惨状并排
- **社会真相的冷幽默**：周天翊一本正经说"有钱人在学校跳楼死不了"——用一种旁观者的冷静阐述残酷的社会真相，黑幽感
- **王海的阴冷转向**：从专心打分的体育老师→最后一句"所有影响自己出货的人，都该被退学"——画风骤变，暴露底色
