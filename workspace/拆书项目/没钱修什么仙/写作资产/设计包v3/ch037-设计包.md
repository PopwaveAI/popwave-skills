## 2. 爽点设计（爽点层）

### 2.1 本章套路

**套路1: 黑心合同·卖身契（全面揭露深化）** ★主套路
- 白真真逐一揭穿周澈尘的套路：倒履相迎是固定剧本、代还债务是控制
- 合同陷阱：八部正神见证强制执行（无法逃避）、报考控制、天价违约金
- 没钱就拍卖器官、血肉、丹田——底层人的修仙身体就是最后的资产

**套路2: 阶层固化揭露（穷人修仙困境）** ★主套路
- 富人和穷人不在一个赛道——穷人的对手不是富人
- 十大名校名额被垄断——几十年没有平民子弟考上十大
- 不签合同考十大→面对所有富人的联合绞杀
- 十大宗门方可成仙——普通大学毕业=外包弟子，永远无法成仙

**套路3: 仪式力量驱动（觉醒倒计时）**
- 张羽犹豫时，仪式倒计时降临——恶寒+倒计时
- 仪式力量帮他看清内心："其实我早就没的选了"
- 用仪式倒计时倒逼觉醒选择，串联全书的驱动力

### 2.2 情绪弧线
```
困惑(为什么没签？) → 震惊(白真真逐一揭露) → 恐惧(器官拍卖) → 压抑(阶层真相) → 愤怒(被设计了) → 觉醒(我没得选) → 坚定(一起考十大)
```

### 4.2 氛围渲染要点
- **白真真揭露的节奏感**：每一条揭露都先问张羽一个问题（"真的去了吗？""是不是还说不用急着还债？"），在白真真引导下张羽逐步意识到陷阱——像拆解一个套娃
- **器官拍卖的凉意**：从"身外之物"到"你的器官，你的血肉，你的丹田"——三连递进，从轻到重，读者的寒意逐步加深
- **天约同契大神的威慑感**：八部正神见证→不可逃避→合同不只是纸面→人神共鉴的压迫
- **阶层真相的残酷感**："没有把穷鬼们当竞争对手"——比直接打压更让人绝望的冷漠忽视
- **觉醒瞬间的仪式感**：恶寒+倒计时→内心的坚定→"其实我早就没的选了"——用贯穿全书的仪式力量作为觉醒的催化剂，有宿命感
