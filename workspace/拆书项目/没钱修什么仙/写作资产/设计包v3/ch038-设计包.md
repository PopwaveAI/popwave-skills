## 2. 爽点设计（爽点层）

### 2.1 本章套路

**套路1: 扮猪吃虎·隐藏实力型（意外暴露→实际更强）** ★主套路
- 张羽压制散手威力战斗，但何大有全力防守逼他暴露到4级
- 雷钧一眼看出"散手已经5级了"——实际上张羽早就在隐藏
- "原本张羽并不打算暴露自己进步的散手"——藏了但没藏住

**套路2: 卷王效应·升级质变（周天采气法10级突破）** ★主套路
- 五天苦修，全部精神投入修行
- 10级突破：法力循环澎湃涌动，如长江大河
- 10级效果：被动吐纳=主动吐纳十成效率——24小时自动修炼
- 法力13.3，全年级第二

**套路3: 修仙界牛马经济学（武道赛奖金）**
- 雷钧推荐参加武道实战赛——奖金5万+专家级功法使用权
- 张羽心动："之前报名的法赛就有奖金，这武道实战赛也有奖金"
- 多个竞赛渠道赚钱——穷人靠自己拼出路

### 2.2 情绪弧线
```
惊讶(白真真说法赛见人) → 专注(武道课) → 好战(vs何大有) → 意外暴露(散手4级被发现) → 兴奋(雷钧推荐比赛) → 优越感(何大有不以为意的嘲讽) → 思考(观察富哥) → 被踩(道术课训话) → 满足(静室修行) → 酣畅(天台对练) → 无语(文具店套餐) → 激动(周天采气法突破10级)
```

### 4.2 氛围渲染要点
- **何大有防守战的憋屈与反击**：全力防守的何大有越来越吃力→被一拳轰开双手→被踹飞→"体内法力连番涌动，一时间都没能爬的起来"——动作细节丰富
- **散手暴露的意外感**：张羽刻意压制但一不小心推到4级——"一不小心没控制好散手的威力"——意外比刻意更有说服力
- **天台对练的默契感**：朋友间的全力切磋，互相指导——"不要看我的手，你要看我的腰、肩膀"——信任与竞争并存
- **文具店的荒诞感**：老板卖"张羽原版修炼套餐"——本人来进货却发现自己被当营销素材——成年世界的丑陋
- **突破10级的通感描写**：法力循环如长江大河→耳中呼啸幻音→被动吐纳提升了——从听觉到体感的多层级描写，质变感强烈
- **章末张羽对白真真的怀疑**：营造神秘感——白真真也在飞速进步——"难道她也开了？"
