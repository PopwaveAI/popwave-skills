## 2. 爽点设计（爽点层）

### 2.1 本章套路

**套路1: 功法升级质变——荒牛镇魂心诀10级（期待产生快乐）** ★主套路
- 荒牛心法10级：意守图老牛活了过来→双角仰天低吼
- 效果从"不怕吃苦"升级为"有期待→吃苦也快乐"
- 张羽反思：不是功法有问题，是很多人已经没了期待

**套路2: 学霸秘密反转（张羽轻松碾压法赛内容）** ★主套路
- 白真真搞不定剑气转化——"刚刚学会加法，老师拿了二元一次方程"
- 张羽直接切出剑气——"指尖在笔记本上一阵轻点"
- 白真真："羽子，你这幅陌生的面孔让我感觉到恶心"

**套路3: 世界观展示·法力形态改变+法赛体系**
- 法力形态改变的基础概念：剑气→加热→低温→起电
- 特种法力的概念——奇功绝技带来的特殊效果
- 竞赛内容远超课堂——为后续法赛情节做铺垫

### 2.2 情绪弧线
```
平静修行(赶路上学) → 烦躁(梁勤骚扰) → 狂喜(荒牛10级突破) → 反思(功法哲学) → 被破坏氛围(白真真"溢出") → 紧张(补课) → 发现(只有两人补课) → 好奇(法力形态改变) → 困惑(白真真搞不定) → 得意(轻松搞定剑气) → 好笑(白真真震惊)
```

### 4.2 氛围渲染要点
- **梁勤骚扰的厌烦感**：从"借试卷"→"递水"→"擦汗"→"占位"→"吹捧"→"旁敲侧击"——层层递进的纠缠，读起来越来越烦
- **荒牛心法突破10级的升华感**：意守图老牛活了过来→双角仰天→低吼——从静态意守图到动态活物，质的飞跃
- **快乐心法机制的思辨氛围**：从"能让人在苦修中感受快乐"到"他们没期待了所以感受不到快乐"——略带伤感的哲学思考
- **白真真闯入的破坏性**：严肃思考氛围→"溢出了吧？"→一秒破功——朋友间的互损恰到好处
- **补课教室的冷清对两种阶层的刻画**：空荡荡的座位vs投影仪前的激情讲解——严老师要面子的内心戏和两名学生的努力形成微妙的张力
- **章末的轻松碾压**：白真真头疼→张羽轻松完成→"陌生的面孔让我感觉到恶心"——朋友间的嫉妒+搞笑收尾
