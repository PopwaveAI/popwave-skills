## 2. 爽点设计（爽点层）

### 2.1 本章套路

**套路1: 穷人联盟·幕后真相（白真真的战略）** ★主套路
- 穷人联盟并非表面上的"守望相助"——资源集中在"老大"身上
- 白真真真正的目的：提高话语权→找到老大→问出逃避签约、保留高考资格的方法
- 张羽成为白真真提高话语权的"筹码"——两人形成利益同盟
- 联盟内部也有PUA和资源分配不均——"省吃俭用投了许多资源过去"

**套路2: 赌博诱惑·穷人陷阱（白真真对张羽的警示）**
- 梁勤追到法赛现场下注——赌狗不可救药
- 赌博宣传单直接发到赛场—修仙界全民赌博的氛围
- 张羽一度动心想赌一把→白真真严厉阻止
- 荒牛镇魂心诀压下贪念——金手指帮主角抵抗诱惑
- 白真真说出穷人理论："赢的话问题更大，金钱观会彻底变化"

**套路3: 世界观展示·飞剑赞助乱象（资本介入的荒诞感）**
- "九天剑光寒，一剑震昆墟"——飞剑厂商赞助
- "你们就特幺的法力赛上考飞剑？真是脸都不要了"
- 张羽内心吐槽：合欢宗赞助就考双修？

### 2.2 情绪弧线
```
新奇(墨天逸皮套人) → 警惕(白真真否认拒真人) → 失望(联盟对张羽失去希望) →

```
