## 2. 爽点设计（爽点层）

### 2.1 本章套路

**套路1: 世界观展示·天眼符系统（修仙界大数据评价）** ★主套路
- 天眼符的运作逻辑：恭请万民部的天知方问大神——"从银行存款到学校成绩……综合模糊评价"
- 越详细越贵，越模糊越便宜——"怕一眼把自己看破产了"
- 全场扫视的震撼：不值一提→稀有→精英→特殊评价的分层
- 每个评价标签都揭示角色底色

**套路2: 扮猪吃虎·白真真的复杂成分（表面弱者实际全能）**
- 天眼符下白真真的评价展开："剑批 拳批|掌批 刀批 枪批 药批 分霸"——超长标签
- 但她分数只有34分——"此女不值一提"
- 天眼符看到了潜力，但现场表现暂时跟不上——验证白真真"只是法赛不擅长"
- 复杂成分意味着她有很深的背景（不是实力，而是各方面都碰过）

**套路3: 富人思维·逆向外置灵根认知**
- 宋海龙的外置灵根不是提升法力——而是用来在"汹涌的吐纳灵机"中锻炼控制力
- "弱者才会担心灵机带来的法力暴涨不受控制，对我这样的强者来说，这却是最好的锻炼机会"
- 与何大有/普通人的认知完全相反——富人家的教育不只是有钱，更有方法论

### 2.2 情绪弧线
```
好奇(飞剑规则) → 理解(天眼符设定) → 有趣(各校评价) → 震撼(天眼符的全面性) →
紧张(比赛分数拉锯) → 好笑(白真真超长标签+34分) → 惊讶(外置灵根揭秘) →
羡慕(白真真馋外置灵根) → 反思(宋海龙的富人思维) → 镇定(张羽思考对手) →
期待(张羽的上场策略)
```

### 4.2 氛围渲染要点
- **天眼符的特效**："空气中的金色光粒汇聚到寒星野的双眼"——华丽的视觉特效
- **天眼符扫描的跳跃感**：不断闪过的评价词——"不值一提""稀有""精英""破产赌狗""冰肌玉骨"——节奏感的排列
- **白真真超长评价的喜剧感**：一长串标签念不完——与34分形成的强烈反差
- **宋海龙取下假发/外置灵根的视觉**：以为强者的冲天长发竟然是可拆卸装备——反直觉的视觉冲击
- **宋海龙满分压制**："飞剑如闪电般激射而出""操弄闪电"——动词的力量感
- **张羽思考的孤独感**：场上其他人都在紧张——唯有张羽在冷静思考类比——凸显主角的独立思考
