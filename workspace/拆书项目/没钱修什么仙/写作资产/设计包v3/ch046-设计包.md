## 2. 爽点设计（爽点层）

### 2.1 本章套路

**套路1: 法赛体系·竞赛实战展开（旋转飞剑+化解力气）** ★主套路
- 张羽用另类操作（旋转飞剑）弥补经验不足
- 众人质疑→内行看出真正难度
- 飞剑设计极限决定了分数天花板

**套路2: 竞赛吃药文化（药厂赞助+各校用药全景）** ★新套路归档
- 紫云公司赞助→允许吃药→卖药→完整的黑色幽默链条
- 三校用药的不同风格：紫云拓脉注射液、红塔兽药、白龙康灵针
- "竞赛就是要吃药"——扭曲但真实的竞赛文化
- 服药过量被抬走→"没有服药过量，只有耐药性不足"

**套路3: 天眼符再登场·张羽的？？？标签**
- 苦修士+体育特长生+极度贫困+贷奴+卡奴+？？？
- "这3个问号好嚣张"
- 穷人+嚣张的反差——强者气质藏不住

### 2.2 情绪弧线
```
有趣(旋转飞剑秀) → 好奇(天眼符评价) → 理解(飞剑真难度) →
感叹(张羽法力控制力) → 轻松(过关表扬) → 震惊(第二关允许吃药) →
好笑(各校吃药方式) → 无奈(张羽不敢吃) → 沉重(有人被抬走+老师扭曲价值观)
```

### 4.2 氛围渲染要点
- **旋转飞剑的视觉奇观**："宛如直升机螺旋桨"——巨大的旋转圆盘横扫信号牌
- **天眼符下张羽的标签墙**：苦修士→体育特长生→极度贫困人口→贷奴→卡奴→？？？——层层递进的信息量
- **"？？？"的嚣张感**：穷人但敢用问号——"这不摆明了告诉别人藏着什么吗？"
- **各校用药的全景对比**：紫云打紫色针管→红塔塞兽药→白龙打康灵针——三种不同的"武装到牙齿"
- **竞赛吃药文化的扭曲感**：从"吃药很正常"到"服药过量=耐药性不足"——价值观层层崩坏
- **黑色幽默的节奏**：张羽震惊→各校吃药→有人被抬→"没有服药过量只有耐药性不足"——从震惊到荒诞到此地无银三百两
