## 2. 爽点设计（爽点层）

### 2.1 本章套路

**套路1: 羽书顿悟·功法补全（残缺功法+外功招式=完整功法可学）** ★主套路 ★新套路归档
- 运功图=法力运转之法，但羽书不认（不是完整功法）
- 白真真打的无极云手外功招式=招式补全
- 两者合一=完整功法！羽书浮现无极云手0级
- 练一遍就成的绝对底气

**套路2: 富人vs穷人·感官隔离（穷的声音/钱的力量）**
- 乐沐岚被穷人的痛呼声干扰→戴几万块的隔音耳罩
- "这就是穷的声音"——抽象概念具象化成听觉污染
- "钱的力量可以隔绝穷的声音"——阶级的感官维度

**套路3: 阶层固化·学校投入死循环**
- 嵩阳高中没有投入→没有竞赛成绩→更没有投入动力
- 三大名校天然优势：提前练过护体罡气、有类似题目经验
- "竞赛不只是天赋、素质上的比拼，更是背后财力、势力、底蕴的比拼"

### 2.2 情绪弧线
```
沮丧(张羽差4分出前十) → 轻松(盒饭吐槽) → 好奇(第三关运功图) →
困惑(看不懂/羽书没反应) → 压抑(乐沐岚/宋海龙/熊不凡各自碾压) →
绝望感(张羽承认竞赛是财力比拼) → 死循环(严老师无奈) →
意外(白真真打拳) → 惊醒(和运功图结合！) → 狂喜(无极云手0级！) →
底气(只要练一遍)
```

- **差4分的焦虑**：严老师反复计算分数，张羽吃盒饭心不在焉——无形的压力

- **顿悟瞬间的爆裂感**：张羽看到羽书浮现无极云手0级——"恨不得将白真真抱起来转一圈"
