## 2. 爽点设计（爽点层）

### 2.1 本章套路

**套路1: 穷人进前十·撕开乌云（反向打脸）** ★主套路
- 张羽273总分第四——唯一非名校生
- 在三大名校的包围中撕开一道缝隙
- 钱深认知崩溃：620分vs650+并列——"分数无法完美表达他"
- 何大有：嫉妒但庆幸"他只是个穷鬼"——明褒暗贬

**套路2: 挖人戏码·名校的虚伪（紫云血统歧视+白龙画饼）**
- 紫云：教练想挖→乐沐岚嫌弃第一学历（穷人味重）★学历歧视的极致
- 白龙：寒星野真诚挖人→但考不了十大
- 真实答案：挖过来就不再参加竞赛，因为竞赛是镀金项目，不该让穷人占名额

**套路3: 投资线推进·墨天逸的遗憾（太晚了）**
- 白真真催投资——"第四名你们知道吗？"
- 墨天逸同意加钱——但坦言"太晚了"
- "要能再早两年，我一定选你"——伏笔墨天逸相关的竞赛背景

### 2.2 情绪弧线
```
紧张(等排名) → 狂喜(第四名！) → 自省(真实实力还不如) →
好笑(钱深崩溃) → 讽刺(何大有庆幸穷鬼) → 期待(被挖人) →
心冷(紫云血统歧视) → 愤怒(白龙不能考十大) → 温暖(严老师护犊子) →
兴奋(白真真催投资) → 疑惑(墨天逸说太晚了)
```

### 4.2 氛围渲染要点
- **排名揭晓的视觉：** 原本被三大名校完全占据的前十，突然出现"嵩阳高中，张羽"——像乌云中撕开的缝隙
- **钱深崩溃的喜剧感：** 抱着头喃喃自语——'这个男人……这个无法用分数完美表达的男人'——书呆子的哲学爆发
- **乐沐岚嫌弃的表情：** "穷人味儿太重"——冷漠、傲慢、不屑写在脸上
- **寒星野沉默的沉重感：** 张羽问"能考十大吗？"——寒星野愣住了——"真实答案自然是不能"——无言的压力
- **严老师护犊子的坚定：** 走过来拦在面前——戒备又强硬
- **白真真催投资的兴奋：** 大声炫耀第四名——像推销员一样激情四射
- **墨天逸"太晚了"的遗憾：** "早应该已经冷却的胸膛似乎也微微一热"——旧伤的感觉
