## 2. 爽点设计（爽点层）

### 2.1 本章套路

**套路1: 奖金到手·穷人第一桶金（两万+请客吃饭）** ★主套路
- 两万奖金到账——存款六万三
- 张羽嘴角比AK还难压
- 请白真真吃饭花了四百多——"等有钱了请你吃更好的"

**套路2: 赌狗陷阱续·连免费跳楼都难（梁勤黑色幽默）**
- 跳楼被救→校区禁跳区要赔钱→思考其他跳楼点都赔钱
- "这破世道，连想要免费跳个楼的地方都难找"
- 梁勤跳楼=还债+违法（跳楼也要赔钱）=黑色幽默三连击

**套路3: 功法修炼·羽书升级展示（无相云罡+六招升级）**
- 无相云罡的特性和战斗应用：可刚可柔，借力打力
- 十分钟练十遍→1级→2级
- 飘散的罡气减少=控制力上升——升级可视化

**套路4: 白真真的隐藏天赋（一眼看出运功图有六式）**
- "这不一眼就能看出来？"——白真真满不在乎
- 张羽震惊："竞赛排名100+靠后的学生，亦有可取之处"
- 白真真说要补法力控制——"我白真真一生不弱于人"

```
轻松(签协议) → 好笑(张羽嘴脸vs之前风骨) → 狂喜(奖金到账！) →

```

### 4.2 氛围渲染要点
- **奖金到账的狂喜：** "嘴角比AK还难压"——现代梗的趣味感
- **跳楼被救的荒诞感：** 黑影落下→罡气接住→"校区禁跳区"→"失足也要赔钱"→梁勤思考其他能免费跳的地方——从紧张到荒诞到黑色幽默
- **白真真大吃大喝的快感：** "咔擦咔嚓连肉带骨吞下"——吃得豪爽
- **"一眼看得出来"的反差萌：** 白真真随口说出、满不在乎——张羽震惊、觉得开了挂
- **烂尾楼的归属感：** "好像是回到了家里一样"——穷人连练功场所都充满辛酸
- **无相云罡的视觉展示：** "如同是冲天而起的白色狼烟"——之前比赛场上的一幕重现+此刻升级后的控制
- **升级进步的可见性：** "一缕缕向上飘散的罡气少了许多"——控制力提升具象化
