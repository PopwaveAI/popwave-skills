# 第53章 月考结束风雨欲来 — 设计包 v3

---

## 一、故事脉络层

### 主线
张羽挑战更高难度：在负重+心法状态下练习**10级健体三十六式**，仪式持续逼迫。周澈尘凌晨4点发消息，张羽以"先备战月考"拖延。

### 月考流程
1. **理论考试** — 顺利通过
2. **道术考试** — 满头是汗做完
3. **武道实战** — vs钱深，钱深要求张羽用无极云手全力战斗，被5秒打飞出界，钱深后悔"喊话的样子很靓仔，被打低分很难受"
4. **道心考试（快乐散）** — 灵界虚拟环境，紫云公司赞助，注射致幻药剂"快乐散"，以按按钮次数越少分越高。张羽忍不住按了2次
5. **法力考试+体育考试** — 平平无奇

### 关键转折
- 体育考试后王海申请张羽入竞赛队被拒，原因：**学生会阻挠**
- 张羽终于达到**道心3级**，对肉身掌控更细致，法力更活泼可控
- 迫不及待开启**天武炼心诀**（专家级，包含拳掌腿指爪刀剑枪棍九套武学），第一次完整演练花费14分钟

### 数据变化
- 道心2级(93%) → 3级(0%)
- 法力18.7（高一年级断层第一）
- 肉体强度1.33级

## 四、套路标签层

### 标记套路
- **#月考场景** — 完整的理论→道术→武道→道心→法力→体育考试流程
- **#道心试炼** — 快乐散抗诱惑，以按按钮次数定分
- **#强者对决** — 钱深要求用无极云手全力战斗，5秒爆杀

- **#规则刁难** — 学生会借校方之手阻挠张羽入竞赛队
- **#仪式外挂** — 寒意警告强迫维持心法修炼
- **#紫云赞助广告** — 品牌植入式月考题目设计

- **#荒牛镇魂心诀** — 11级仪式逼迫维持心法
- **#健体三十六式** — 10级后在负重+心法下练习
