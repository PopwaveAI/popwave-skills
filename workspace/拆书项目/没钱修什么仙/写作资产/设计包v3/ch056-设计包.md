## 2. 爽点设计（爽点层）

### 2.1 本章套路

**套路1: 姐姐线·身份揭示（弱鸡弟弟发现姐姐是区域最强）** ★主套路
- 张翩翩平淡说出自己身份——「白龙高中高三第一」「嵩阳市最强高中生」
- 张羽从「听起来不错」到真正理解含金量的震撼
- 姐姐说「去嵩阳高中帮你们撑一撑」——读者知道她是真的有这个分量
- 为后续撑腰场景埋下伏笔

**套路2: 吃空饷·天降好事**
- 神辅工作内容=不用干活每月领钱
- 「这种好事终于轮到他了？」——穷人翻身的第一口甜头
- 时薪不到14块但地位在八部正神体系内
- 关键福利：送篆书（价值近十万）+符咒学习打折
- 穷人对篆书的渴望：「阿真要看到这个还不得跪下来叫我一声爹？」

**套路3: 大药修炼·打药爆发**
- 王海的药「太猛了」——止痛药+专注剂+亢奋剂+体力针剂
- 原本14分钟一套→6分钟一套——效率翻番
- 连续5小时不停歇——「太猛了！王海这药真的是太猛了！这5800花得值啊！」
- 副作用明显但数值增长可观

### 2.2 情绪弧线
```
疑惑(神辅是什么) → 震惊(姐姐是最强高中生) → 窃喜(吃空饷好事) →
期待(考试准备) → 心疼(花钱5800) → 兴奋(打药的爆发感) →
满足(数值成长) → 紧张(进入灵界考试)
```

## 4. 感官与画面（感官层）

### 4.1 环境基线

| 场景 | 时间 | 环境特征 | 感官关键词 |
|:----|:-----|:---------|:-----------|
| 张羽的出租屋/与张翩翩通话 | 白天 | 手机通话场景 | 平静的对话声 |
| 烂尾楼 | 夜晚到凌晨 | 空旷、没人、药味弥漫 | 发烫的身体、气血翻涌、拳风呼啸 |
| 服药后修炼 | 连续5小时不停歇 | 地下碎碎声、人影残影 | 肌肉滚烫、心脏狂跳、针剂扎入 |
| 药效过后的出租屋 | 早上7点后休息 | 安静的出租屋 | 心脏像要飞出来、五脏钝痛、缓缓恢复 |
| 灵界入口 | 9点整 | 视觉闪白转移 | 黑暗中一道光芒闪过 |

### 4.2 氛围渲染要点
- **修炼打破极限的热感**：「浑身都开始发烫发热」「气血宛如是滚滚岩浆」「身体像要烧起来」——药力+体力爆发的热力感
- **疯狂训练的节奏感**：「12分钟→9分钟→6分钟」不断缩短的时间→不断加速的节奏
- **打药副作用的真实感**：「心脏跳得像要飞出来」「五脏六腑一阵阵钝痛」——不是无代价的狂飙
- **穷人心态的微妙感**：算账的冷静（5800→存款4万5→只剩3万9千多）与修炼的热血形成反差
- **灵界面罩的科技感**：稀有装备的质感，戴上的仪式感
- **灵界入门的超现实感**：「黑暗中一道光芒闪过」——瞬间转移的失重感
