## 2. 爽点设计（爽点层）

### 2.1 本章套路

**套路1: 钱能通神·昆墟的本质揭露** ★主套路
- 核心反转：神的审核不是公正的——可以用钱（好感符）买通
- 张羽以为靠道心通过，实际靠姐姐打钱
- 张翩翩的符咒理论：「符=充值」「信仰等级靠充值、学符咒靠充值、请正神辅导靠充值」
- 「神就是拿钱办事」——修仙世界的终极真相之一
- 张羽的震惊逐步升级：花钱买通神？→神就是拿钱办事？→世界观重建

**套路2: 灵界考试·邪神幻境试炼**
- 考试形式：精神世界幻境——模拟高中跳楼绝望→邪神诱惑
- 考验核心：本能的拒绝反应
- 张羽的表现堪称满分：怒砸手机+送中指→把邪神玩偶甩给仇敌→报警抓邪神
- 与孙甲乙视角交叉——他的审核标准（傲慢的高傲神）

**套路3: 孙甲乙·功曹神的傲慢与腐败**
- 神的设定：「神卑未敢忘昆忧」——表面高大上的神职宣言
- 实际做事：看到富豪背景冷笑、看到学霸背景冷笑——「不论什么身份在我这里都一样：不过就不过」
- 收到钱后：「一个高中生就如此敬神，足见信仰之心坚定」——180度大转弯
- 腐败却不觉得腐败——他认为这是「敬神」

### 2.2 情绪弧线
```
压抑(幻境中的高中绝望) → 警惕(邪神推销出现) → 爽(怒砸手机+竖中指拒绝) →
清醒(意识到考核中) → 好奇(孙甲乙视角的考生审核) → 好笑(孙甲乙的傲慢) →
紧张(张羽的幻境表现) → 意外(张翩翩操作) → 震惊(好感符=打钱) →
好笑(会/不会选择题) → 世界观崩塌(钱能通神) → 满足(得到篆书) → 期待(去嵩阳)
```

### 4.2 氛围渲染要点
- **幻境绝望感**: 日复一日的压抑生活——成绩越来越差、排名越来越低、从示范班掉到最差班、同学们的目光像看垃圾、欠贷越来越多——层层递进的绝望堆叠
- **幻境超现实感**: 跳楼边缘接到墓地推销电话——荒诞感与现实绝望的混搭
- **邪神推销的诡异感**: 「一只熊玩偶在屏幕中开口说话」——可爱包装下的危险
- **神视角的冷感**: 孙甲乙的审核——像是在看蚂蚁一样俯视考生，冷漠、高傲
- **打钱变脸的喜剧效果**: 「一个高中生就如此敬神」——前后对比的强烈反差
- **选择题的荒诞感**: 纯白房间里漂浮两个选项——简单到让人怀疑有陷阱
- **篆书的质感**: 「黑白相间，宛如手册」——实物的触感和重量感
- **穷人看到宝贝的兴奋**: 张羽小心翼翼捧起篆书——如获至宝的仪式感
