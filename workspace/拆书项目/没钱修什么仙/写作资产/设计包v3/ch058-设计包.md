## 2. 爽点设计（爽点层）

### 2.1 本章套路

**套路1: 高低年级等级压制·阶层展示** ★主套路
- 核心展示：一年修仙时间差距=绝对的实力鸿沟（肉体1.4级 vs 6级以上）
- 学弟见学长的规矩：主动鞠躬、学长不坐学弟不能坐、上厕所要让学长先尿
- 有人把高一高二高三比作「链气前期、中期、后期」
- 三大名校更离谱——白龙高中可以让低年级下跪磕头
- 赵天行的高壮体型与卑微态度的反差

**套路2: 羞辱战·恶意指导**
- 蓝岭主动点名要「指点」白真真——实际是要趁机打压
- 白真真先手偷袭无用——实力差距太大
- 蓝岭下重手将白真真甩飞撞墙——要不是雷钧救援就重伤了
- 雷钧被蓝岭眼神压制——说明这是学生会的授意
- 「我相信老师你在场，她一定没事的」——表面客气、实际恶意

**套路3: 救场登场·穷人翻身（有篆书了！）**
- 张羽在关键时刻从大门走进来——经典救场时刻
- 「喂，你在欺负我的实习生？」——宣示主权
- 拘役符发动——九只金色巨掌压制蓝岭
- 全场震惊：这穷鬼哪来的篆书？！
- 穷人突然变成了「有装备的人」的反差爽感

### 2.2 情绪弧线
```
平静(进练功房) → 警觉(发现高二学长) → 压抑(等级制度展示) →
尴尬(白真真不问好) → 紧张(点名对战) → 同情(实力碾压) →
愤怒(蓝岭下重手) → 无力(雷钧被压制) → 期待(张羽出现) →
爽(拘役符压制蓝岭) → 震惊(全场看张羽)
```

### 4.2 氛围渲染要点
- **身高对比的视觉冲击**: 赵天行300多斤站在蓝岭面前像柯基——用视觉比例突出阶层差距
- **学长外貌的工业化特征**: 第二性征几乎完全消退、手臂上密密麻麻针孔——残酷修炼的视觉证据
- **白真真的孤立感**: 所有人都在向学长问好→只有她没有→成为全场余光焦点
- **实力碾压的体感**: 一脚踹在要害但对方纹丝不动、脚骨像裂开——用触觉描写实力差
- **被甩飞的镜头感**: 「带着一连串的残影撞向墙壁」「雷钧出现……两人一同撞碎了墙壁」——慢动作般的紧张感
- **救场登场的光影处理**: 大门方向的声音→张羽从大门外走进→抬起右手→篆书滑动——经典登场镜头语言
- **拘役符的金色华丽感**: 「金色光点闪动」「九只金色巨掌」——符咒的视觉震撼
