## 2. 爽点设计（爽点层）

### 2.1 本章套路

**套路1: 威压符·信息即武器** ★主套路
- 威压符展示分数和存款来威压他人——699分 + 25亿
- 分数699分（接近满分）+ 存款25亿——两个维度的绝对压制
- 「威压符的原理：恭请神力展现分数、存款，以威慑他人的符咒」
- 数字逐秒跳出来的节奏感：2万5→25万→250万→2500万→2亿5→25亿
- 配合天眼符的「你不配知道」——滴水不漏的信息压制

**套路2: 大姐姐登场·翻手为云覆手为雨**
- 关键时刻登场——「你要袭击我手下的临时工？」
- 不给任何解释机会——直接上威压符
- 从实力（699分）到财力（25亿）双重碾压——用对手的规则打败对手
- 周澈尘之前的压制→被张翩翩瞬间翻转
- 「不要让我听到任何你对我手下不利的事情……买你的脑花」——霸气收尾

**套路3: 天眼符·身份暴露与反暴露**
- 周澈尘用天眼符看穿张羽身份：外包员工
- 从「神权特许」到「区区一个外包」——被看穿的落差
- 何大有用天眼符看张翩翩——「你不配知道」——用符咒对抗符咒
- 天眼符不是无敌的——可以用更高级的符咒和信息隐藏来反制

### 2.2 情绪弧线
```
紧张(蓝岭挣脱) → 爽(张羽甩身份压制) → 得意(老子今天帅炸) →

逆转(张翩翩登场) → 爽(威压符展示) → 震撼(25亿) → 好笑(数不清) →
爽(「你不配知道」) → 好笑(张羽也想学) → 解气(周澈尘服软) → 霸气(买你的脑花)
```
