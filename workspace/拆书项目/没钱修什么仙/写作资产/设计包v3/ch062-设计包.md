## 2. 爽点设计（爽点层）

### 2.1 本章套路

**套路1: 拜神=签到·神钱通吃·系统黑色幽默** ★世界观核心展示
- 叩拜神一次1000→张羽震惊→还没开通付款账户就从银行卡扣钱
- 张翩翩解释：「这就是神的伟力」「每分每秒都能调动亿万账户里的资金」
- 「欠谁的钱，你也千万不要欠神的钱」——神权=直接扣款权限
- 供奉神=远程供奉→灵界虚拟道具→张羽吐槽：「该不会都是虚拟道具吧！？」
- 签到打卡修仙：每天最少花1000才能维持信仰等级

**套路2: 隐藏实力·武道竞赛策略** ★战略规划
- 张羽的思考：10级无极云手不能暴露→要隐藏底牌
- 对抗思路：多提升基础素质，不依赖高阶功法
- 道术竞赛天克穷人，好在在最后阶段——时间排序的重要性

**套路3: 偷袭反杀·无相云罡护体** ★动作高潮
- 周天翊苦等两小时→抓到"最虚弱"时机→5级狂风腿法偷袭
- 结果：无相云罡自动反弹→周天翊被震飞
- 10级无极云手的罡气护体——"练气入体，罡气外放，劲力贯体，由外及内"的成果展示

### 2.2 情绪弧线
```
震惊(被扣1000) → 愤怒(没确认就扣钱) → 无奈(找不到退款) →
好奇(功能展示) → 吐槽(虚拟道具骗钱) → 焦虑(修仙太烧钱) →
振奋(思考武道策略) → 专注(修炼天武炼心诀) →
紧张(周天翊潜伏) → 警惕(偷袭来临) → 爽(无相云罡反弹！)
```

- **篆书扣款的荒诞感**: 手机一震→扣款1000→没开通账户也能扣——神权的霸道和荒诞
