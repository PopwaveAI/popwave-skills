## 2. 爽点设计（爽点层）

### 2.1 本章套路

**套路1: 校园舆论风暴·校奸标签** ★舆论描写
- 三大阵营的回应：孝子骂叛徒/嫉妒者说手术没做干净/怕事者远离
- 赤裸裸的恶意——「愤怒、嫉妒……以及隐藏的羡慕，最终化为了赤裸裸的恶意」
- 从法赛英雄→全校公敌的转变速度

**套路2: 双幽朋的悲剧·周澈尘的霸凌艺术** ★反派黑历史展示
- 曾经的高一月考总分第一→被周澈尘针对性霸凌
- 霸凌方式：不准写作业/打药/锻炼/通宵——逼着吃炸鸡薯条啤酒、看视频读小说
- 让你眼睁睁看着自己变肥、成绩下降——精神+肉体的双重打击
- 直到彻底求饶才被放过——霸凌的艺术在于摧毁人的意志

**套路3: 赵天行的勇气抉择·怕但还是要坐过去** ★人物成长弧
- 从拒绝邀请→回避目光→羡慕白真真钱深→内心挣扎→回忆张羽的背影→张羽招手→咬牙坐过去
- 又害怕又觉得自己很帅——真实的矛盾心理
- 「他想错了，此刻的赵天行在绝大部分的学生眼中，就像是个小丑」——但勇气本身值得

**套路4: 白真真将计就计·信息隔离** ★战略博弈
- 面对墨天逸的询问——不解释真相
- 让所有人以为他们签约白龙→放弃考十大→不会成为目标
- 「能增加考上十大成功率的选择，才是最优解」——白真真的务实哲学
- 对墨天逸的恩情日后偿还——长远眼光

**套路5: 墨天逸·穷人打工日常+老大暗线** ★底层生态展示
- 灵界做客服——18小时工作制——5分钟休息
- 把钱全部打给老大——「你是嵩阳市唯一可能考上十大的穷人了」
- 老大身份成谜——白真真查不出——天约同契大神的见证合同
- 墨天逸对白真真和张羽的惋惜——「终究还是没能顶住那压力」

### 2.2 情绪弧线
```
心寒(舆论风暴→校奸标签) → 压抑(双幽朋的霸凌回忆) →

好笑(张翩翩拒绝+钱深羞红脸) → 心疼(钱深振作说精神白龙人) →

惋惜(墨天逸的信) → 压抑(墨天逸18小时打工) →

```

## 4. 感官与画面（感官层）

### 4.1 环境基线

### 4.2 氛围渲染要点
- **舆论风暴的恶意感**: 「愤怒、嫉妒……以及隐藏的羡慕，最终化为了赤裸裸的恶意」——情绪逐级递进到"恶意"
- **双幽朋霸凌的窒息感**: 「不准写作业，不准打药，不准锻炼」「逼着吃炸鸡薯条啤酒甜点」「眼睁睁看着肚皮变大」「看着成绩下降」——精神毁灭的四重打击
- **赵天行犹豫的纠结感**: 想过去→怕被排挤→回忆张羽背影→咬牙→坐下——内心斗争的完整呈现
- **坐下后的错位感**: 又怕又觉得自己帅→实际别人看他是小丑——真实的认知偏差
- **钱深羞红脸的喜剧感**: 从被拒的失落→「我最高626分！」的较真→「白龙高中的空气都是甜的」——穷人的精神寄托式幽默
- **白真真决策的冷静感**: 从收到消息到想通——"为什幺要解释？"——将计就计的果断
- **墨天逸18小时打工的窒息感**: 「工作停滞超过1分钟，请立刻返回工作岗位」——资本的无情监控
- **「一定要成功啊老大」的悲壮感**: 所有穷人孤注一掷的希望——老大是唯一的出路
