## 2. 爽点设计（爽点层）

### 2.1 本章套路

**套路1: 白龙高中·分数种姓制度全面展示** ★主套路
- 从大门开始层层歧视：游客门票500→660以上免费→550以下禁止游览
- 校内：660以上走中间白色道，以下走红色两侧
- 电梯：660以下禁止乘坐
- 休息区：贵宾厅（三大名校）→遮阳棚（重点）→草席（普通）
- 空气都是香的——促智剂
- 这不是学校，这是阶级划分的展览馆

**套路2: 穷人三人组的搞笑互助**
- 赵天行只带10块钱——「我妈怕我乱花钱」——妈宝男实锤
- 10块钱买包子→不够吃→想抢便利店
- 钱深出现，300块！——「老钱！你才是我张羽真正的好朋友啊！」
- 张羽反思：「和白真真这种卡奴做朋友？和赵天行这种妈宝男做朋友？」
- 赵天行被拦在门外——「我跟我妈打个电话要钱」
- 穷人三人组各有各的穷法，但穷在一起就有笑点

**套路3: 何大有·傲娇观战**
- 伪装口罩墨镜帽子来看比赛——「不想被发现在看张羽和白真真」
- 被钱深认出→嘴硬：「到不了平均线」「是初中同学邀请」
- 明明想看朋友表现，偏要装不在意——真·傲娇
- 「611的何大有，有资格和他坐一块儿看比赛了」——分数即等级

**套路4: 炼天极的友好与暗讽**
- 主动打招呼：「我就猜你会来」
- 无意间的鄙视——「你们嵩阳只来这几个吗？」「毕竟嵩阳嘛」
- 三大名校各出10人，校内先淘汰一轮——竞赛体制的规模展示
- 炼天极对张羽的态度：有印象、觉得有趣、但不觉得他能赢

### 2.2 情绪弧线
```
饥饿(没钱吃饭) → 绝望(10块只够买包子) → 惊喜(钱深请客300块) → 
满足(12份套餐下肚) → 无语(白龙高中门票500块) → 尴尬(赵天行被拦) →
好奇(香香的空气) → 愤怒(各种分数歧视) → 震撼(巨大运动场) →
渺小(全是三大名校的人) → 意外(炼天极打招呼) → 不爽(被暗讽) →
好笑(何大有伪装被认出) → 期待(评分赛要开始了)
```

### 4.2 氛围渲染要点
- **白龙高中大门前的自我鄙视**: 电子屏冰冷地列出票价和分数门槛——550以下禁止游览——连观赛资格都要按分划分
- **香味的阶级感**: 「空气里这股香甜味」——连呼吸都有档次差——嵩阳的空气没加料
- **地上的红白道**: 白色在中间是「人走的」，红色在两边是「靠边走的」——分数即行走权
- **电梯歧视的荒诞**: 「660分以下禁止乘坐电梯」——物理上的阶层隔离
- **运动场的开幕震撼**: 「高约十米，占地上万平米的巨大运动场」——从嵩阳的破旧到白龙的壮阔
- **三大方阵的压迫感**: 「一个个瞪大眼睛，挺胸抬头，神采奕奕，容光焕发」——像打了鸡血一样的好学生
- **休息区的无声对比**: 三大名校=沙发水果零食，重点=椅子矿泉水，普通=板凳——不需要说话就已经分出三六九等
- **何大有伪装的喜剧**: 帽子+墨镜+口罩——全副武装却被一眼认出——「你是611的何大有吗？」
