## 2. 爽点设计（爽点层）

### 2.1 本章套路

**套路1: 被低估·被无视（白真真赢了却没人在意）** ★副线
- 白真真打败虎云涛——回头看没人关注
- 「一群被分数迷花眼的家伙」
- 同组学生以为她靠阴谋诡计赢
- 「低分的人素质就是差」——三大高中对普通高中的傲慢公式

**套路2: 技巧压制·绝对力量VS绝对技巧（张羽vs楚秋河）** ★主套路
- 楚秋河打药+卡车冲锋拳——绝对力量的宣言
- 张羽不躲——以无相云罡硬接
- 天武炼心诀10级全集中状态→看穿一切
- 牵引卸力→楚秋河中了无数拳
- 「我用纯粹的技巧，压制了力量、速度、体能都更强的我？」——核心爽点

**套路3: 何大有式扭曲心理** ★喜剧副线
- 希望张羽输（讨厌的穷人）但又不想楚秋河赢（看不起自己的同学）
- 「手心手背都是尿，这两个能不能一起淘汰了？」
- 张羽赢了→他痛心；楚秋河输了→他暗爽

**套路4: 打药文化展示（楚秋河赛前补剂）**
- 竞赛教练主动让楚秋河打药
- 「你以为这种学校的学生……可能不吃药吗？」
- 新药「果然够劲」→但技巧碾压一切

**套路5: 校园等级恐惧（楚秋河害怕被全校嘲笑）**
- 输给三大高中的学生=校园地位崩溃
- 「恨不得从30楼楼顶跳下去」
- 被黑暗未来吓晕——黑色幽默

```
狂暴冲锋(卡车冲锋拳) → 震惊(张羽不躲硬接) →
专注(全集中状态) → 畅快(张羽技巧压制) →
憋屈(楚秋河被压着打) → 震动(楚秋河认知崩塌) →

```

- **被无视的反差**: 白真真打赢强敌→回头一看没人——热闹与冷清的强烈对比

- **何大有扭曲喜剧**: 「手心手背都是尿」——纠结的内心独白，读者能代入的扭曲快感
