## 2. 爽点设计（爽点层）

### 2.1 本章套路

**套路1: 张羽狂吃·心脏的饥饿** ★肉体升级的底层需求
- 无味的合成食物——纯营养，难吃但高效
- 心脏如饥似渴——突破后的饥饿感
- 赤髓浑元气加速消化
- 穷人的升级方式：拼命吃+拼命吸收

**套路2: 自助餐区的阶级划分** ★资本差距的直接展示
- 红塔学生大胃王——熊文武做了味觉去除手术
- 玉星寒/宋海龙/乐沐岚吃自带的——有钱人的精致
- 普通高中的混口饭吃——已经被淘汰了还在吃
- 不同的食物代表不同的阶层

**套路3: 周澈尘的警告·高级打工仔的算计** ★精英视角的"善意提醒"
- 「这一关不用那么积极，随便参与一下就行了」
- 「你和白龙高中的勾结，已经被其他公司识破了」
- 「别最后被人卖了，还替人数钱」
- 分析了三大公司的交锋——红塔牧业、紫云药业、仙韵集团
- 「我们不用像张羽那样只有烂命一条，什么都要光膀子上去拼」
- 「一辈子给人当狗，最后也一定会被公司盯上，被张翩翩卖个好价钱」

**套路4: 张翩翩的撑腰·姐姐的霸气** ★强者后盾
- 张羽：下把赢了不会被对付吧？
- 张翩翩：能赢吗？→能赢就去赢
- 「你如果能拿下第一，我帮你和白龙来谈」
- 「然后等着收钱就是」——钱字的魔力

**套路5: 白真真的真灵根·拥抱借根** ★亲密信任
- 「阿羽，真灵根借你吧」
- 一个半小时她突破不了，张羽可以
- 真灵根已习惯张羽的赤髓浑元气
- 灵机灌注五脏六腑、加速消化
- 厕所外的守护——炼天极也被挡住

**套路6: 张羽突破·心脏如大鼓** ★修炼异象
- 浑元丹药力+真灵根灵机→疯狂吸收
- 心脏像黑洞一样吸取营养
- 心跳如大鼓→厕所回荡巨响
- 炼天极感受到了动静

**套路7: 天空柱亮相** ★最终关卡降临
- 五米直径通体银黑的巨柱从天花板降下
- 最低1000公斤，最高10万公斤
- 练气期强者的力量测试
- 选手依次登台撑柱

### 2.2 情绪弧线
```
疯狂进食(咕咚咕咚灌下合成食物) →
恶心但坚持(无味营养→为了比赛) →

```
