# 第152章 张羽的人脉 — 设计包 (v3)

---

## 一、核心概念

> **穷人的竞赛筹码变现 —— "我退出，你给钱"**
> 张羽主动联系三大高中，**将退赛资格当作商品兜售**，一口气拿到230万税后收入。同时激活张翩翩遗留的**人脉资源（冻结符）**，双线操作完成从"被压迫者"到"规则玩家"的转变。

**本章使命：** 
1. 解决张羽/白真真的资金危机（余额仅剩30多万）
2. 建立"穷盟"的商业模式雏形：竞赛名额交易
3. 展示张翩翩留下的人脉资产的威力
4. 为后续买断退赛、道心竞赛缺位、以及周澈尘被震慑做铺垫

---

## 二、情节框架

### 四线并行结构

| 线索 | 内容 | 结果 |
|------|------|------|
| **紫云线** | 张羽→乐沐岚→乐景辰 | 税后60万成交 |
| **红塔线** | 张羽→熊文武→赤身男子(红塔会长) | 税后70万成交 |
| **白龙线** | 张羽→玉星寒→宋海龙 | 税后100万成交，玉星寒吃10万回扣 |
| **人脉线** | 张羽激活冻结符 | 周澈尘银行帐户被冻结，产生恐惧 |

### 关键节点

1. **张羽的主动出击**（第1-3行）
   - 不等对方找上门，自己同时联系三家
   - 核心话术："直接找我们，肯定比他们联系嵩阳高中来的便宜"

2. **乐景辰的审视**（第5-27行）
   - 询问妹妹乐沐岚对张羽道心的评价
   - "忍住不舔汗水"的细节成为道心坚强的证据
   - 乐景辰反问："你收集汗水卖给药企怎么办？"——显示大资本的警觉性

3. **熊文武的利益计算**（第33-46行）
   - 从个人利益出发推荐张羽退赛（自己的兄弟更有机会）
   - 暗线：熊文武希望张羽更有钱→多租孵化室→多赚钱
   - "能从穷人堆里卷出来的，就不可能意志太差"——阶级标签

4. **玉星寒吃回扣**（第47-82行）
   - 绕过甩锅的学长（二楼砸门），直接找新副会长宋海龙
   - 以"高一年级第一的意见"施压，强推100万
   - 宋海龙视角的误解：以为是玉星寒为了自己的名次

5. **总收入结算**（第87-92行）
   - 紫云60万+红塔70万+白龙100万 = 230万
   - 玉星寒回扣10万 → 净得220万
   - 一人110万，完美解决资金危机

6. **冻结符激活**（第99-107行）
   - 张羽决定用张翩翩留下的符咒威慑周澈尘
   - 冻结符：冻结银行帐户，信仰等级越高冻结越短
   - 对应前文张翩翩留给张羽的"人脉资产"

## 三、人物设计

### 张羽 —— 从被压迫者到规则玩家

| 维度 | 分析 |
|------|------|
| **战略转变** | 从"在规则内竞争"变为"把规则本身变现" |
| **商业头脑** | 三家同时询价 → 形成竞价氛围；包税（税后）条款 |
| **人脉资产运用** | 第二次使用张翩翩遗留资源（第一次是万校长退学事件） |
| **心理状态** | 卖名额虽遗憾但务实："反正没有机会考上十大" |

### 玉星寒 —— 吃回扣的中间商

| 维度 | 分析 |
|------|------|
| **动机** | 10万回扣 + 自己也能竞争第一名 |
| **战术** | 绕过不愿干活的学长，直接对新副会长施压 |
| **话术** | "花的又不是我们的钱，都是校委会的钱" |
| **角色功能** | 展现"有资源的人如何通过中间人角色获利" |

### 乐景辰 —— 谨慎的资本玩家

| 维度 | 分析 |
|------|------|
| **评估方式** | 不直接回消息，先通过妹妹收集情报 |
| **警觉性** | 连"汗水被收集卖给药企"的细节都能想到 |
| **决策** | 60万买退赛，认为"划算" |

### 熊文武 —— 利益捆绑者

| 维度 | 分析 |
|------|------|
| **立场** | 希望张羽有钱→多租孵化室→自己多赚 |
| **评价** | "穷人堆里卷出来的，意志不可能差"——阶级视角下的认可 |
| **隐藏情绪** | 对安安被降薪不满（为153章铺垫） |

### 周澈尘（侧写）

| 维度 | 分析 |
|------|------|
| **当前状态** | 未知张羽正在卖名额，但随后被冻结符震慑 |
| **威胁性** | 第100-104行张羽分析：真实实力仍差，需震慑争取时间 |

## 四、设定细节

### 竞赛名额交易机制

| 要素 | 说明 |
|------|------|
| **交易标的** | 张羽和白真真不参加**道心竞赛**（及未来的竞赛） |
| **定价逻辑** | 三大高中评估两人的竞争力 → 出价阻止他们抢名次 |
| **税务处理** | 紫云出**税后60万**（校方代缴税） |
| **回扣文化** | 玉星寒从100万里吃10万回扣，展现中介获利模式 |
| **买断范围** | 仅限这次道心竞赛，明后年不再收——认为两人未来竞争力下降 |

### 冻结符设定

> 张翩翩留给张羽的符咒之一
> 效果：**冻结目标银行帐户**
> 限制：目标信仰等级越高，冻结时间越短
> 本章使用者：嵩阳市巡查队队长（张翩翩人脉）

**战术意义：**
- 不直接伤害肉体，而是从经济层面施压
- 让周澈尘产生"张羽背后有人"的错觉（税务审查的名义）
- 为后续153章周澈尘的退缩做铺垫

### 三大高中竞争力一览

| 高中 | 出价 | 中间人 | 动机 |
|------|------|--------|------|
| 紫云高中 | 税后60万 | 乐沐岚→乐景辰 | 扫除竞争对手 |
| 红塔保护区 | 税后70万 | 熊文武→会长 | 给兄弟腾位置 |
| 白龙高中 | 税后100万 | 玉星寒→宋海龙 | 保第一+吃回扣 |

### 张羽的财务状况变化

```
期初余额: 30多万（即将见底）
竞赛交易: +220万（两人平分，每人110万）
预计使用: 支撑到第一轮筑基考试（下个月）
```
