# 第155章 灵界修行，穷盟现身 — 设计包 (v3)

---

## 一、核心概念

> **灵界 —— 穷人与富人共享的虚拟战场**
> 张羽和白真真首次踏入灵界公共区域，在玉星寒的介绍下进入**问道堂（道心补习机构）**。与此同时，**穷盟（墨天逸为首的神秘组织）**首次正面盯上张羽，在灵界中实时监控他们的道心修炼场景。本章是**新地图开启 + 新势力亮相**的双重转折点。

**本章使命：**
1. **新地图扩张**：正式引入灵界公共区域系统
2. **新机制展示**：道心淬炼场景的构建与运行逻辑
3. **暗线引爆**：穷盟组织首次正式出场，锁定张羽
4. **氛围铺垫**：何大有的态度转变、张羽在校内的"帝王"地位

## 二、情节框架

### 双线并行结构

| 线索 | 内容 | 核心信息 |
|------|------|----------|
| **校内线(A线)** | 张羽/白真真在嵩阳高中的"帝王"地位 + 何大有示好 | 权力格局已变，但未来在灵界 |
| **灵界线(B线)** | 问道堂道心补习 → 三人合用修炼室 → 穷盟监视 | 灵界机制展示 + 势力亮相 |

### 关键节点

1. **嵩阳"帝王"地位的建立**（第21-46行）
   - 食堂中，学生们的敬畏目光 + 各种夸张传言
   - 钱深、赵天行成为"帐下大将"
   - **何大有主动送四盘妖兽肉示好**（代表校董派的妥协）
   - 何大有的内心戏："我和这两个穷鬼已经不是一个赛道的了"
   - **氛围：** 校内已经没有直接威胁，但未来战场在灵界

2. **灵界入口的震撼**（第53-65行）
   - 灵界面罩 → 传送到灵界
   - 无限高大楼 + 无数虚影人潮进进出出、互相穿透
   - "越来越像虚拟世界"的感知
   - 临时权限（花钱问玉星寒买的）—— 显示灵界的门槛

3. **问道堂的设定释放**（第67-83行）
   - 仙韵集团的问道堂：灵界中的补习机构
   - 面向**整个昆墟一层**的有钱人客户
   - 预约制、推荐制 —— 进一步抬高门槛
   - 道心修炼方法介绍：强者人生路、极端痛苦、心法场景

4. **修炼场景的构建**（第83-104行）
   - 三人合用教室 → 生成同时适应三人心法的场景
   - **玉星寒的场景：** 天武炼心诀 → 公司保安 vs 罢工工人
   - **设定释放：** 道心淬炼教室 = 心法需求对应的高强度场景

5. **穷盟现身**（第86-104行）
   - 墨天逸在问道堂工作（卧底）
   - 身材模糊的人影（穷盟老大？）下令监控张羽
   - 实时画面传输：玉星寒保安大战破产工人
   - **伏笔：** 穷盟为什么盯上张羽？

## 三、人物设计

### 张羽 & 白真真 —— 灵界新人

| 维度 | 分析 |
|------|------|
| **灵界状态** | 第一次进入公共区域，被规模震撼 |
| **权限等级** | 临时权限（买来的），还不是正式用户 |
| **道心修炼方向** | 需要根据心法设置场景（本章未展示他们俩的场景） |
| **与玉星寒的关系** | 付费购买灵界补课资源，交易关系 |

### 玉星寒 —— 灵界引路人

| 维度 | 分析 |
|------|------|
| **角色功能** | 带张羽/白真真进入灵界，展示道心修炼机制 |
| **心法场景** | 天武炼心诀 → 保安 vs 罢工工人（极度激烈的生死战场） |
| **背景信息** | 熟门熟路，似乎在问道堂有固定资源 |
| **态度** | 和之前吃回扣的形象一致——做中间人赚钱 |

### 穷盟 —— 新势力

| 成员 | 角色 |
|------|------|
| **模糊人影（疑似老大）** | 发号施令，对张羽的行踪极感兴趣 |
| **墨天逸** | 卧底在问道堂工作，负责监控和情报 |
| **组织性质** | 目前已知能渗透灵界，监控他人修炼场景 |

### 何大有 —— 识时务的新副会长

| 维度 | 分析 |
|------|------|
| **行动** | 主动送四盘妖兽肉给张羽桌 |
| **内心** | "我和他们已经不是一个赛道的了" |
| **立场** | 不再敌对，但也并非盟友——纯粹的势利选择 |
| **暗线** | 担心自己是否还能考上十大（嵩阳和三大高中的差距） |

## 四、设定细节

### 灵界公共区域 —— 新地图

| 设定元素 | 说明 |
|---------|------|
| **入口方式** | 灵界面罩 + 输入网址 |
| **视觉表现** | 无限高大楼，人潮进进出出如虚影（可互相穿透） |
| **访问权限** | 临时权限（需购买） / 预约制 / 推荐制 |
| **类比** | 互联网中的"网站"——时刻存在，接受无数人拜访 |
| **覆盖范围** | 整个昆墟一层都可以通过灵界访问 |

### 问道堂 —— 道心补习机构

| 设定元素 | 说明 |
|---------|------|
| **运营商** | 仙韵集团 |
| **服务** | 道心补习 + 武道补习 |
| **道心修炼方法** | ① 体验强者人生路 ② 极端痛苦刺激 ③ **心法场景修炼（最推荐）** |
| **心法场景机制** | 根据修炼者心法生成对应的高强度场景 |
| **多人场景** | 多人合用时，生成同时适应所有人的综合场景 |
| **限制** | 法力/肉体无法在灵界增长（仍需物质界积累） |

### 道心淬炼场景示例（玉星寒）

```
心法: 天武炼心诀（战场杀伐意念）
场景: 公司保安团队 vs 破产罢工工人
身份: 保安队长（代表绝不肯少赚一分钱的公司）
冲突: 极度激烈，关乎生死存亡的战场
目的: 在高强度杀伐中淬炼天武意念
```

### 穷盟的监控能力

| 能力 | 说明 |
|------|------|
| **实时监控** | 能远程查看他人的道心修炼场景画面 |
| **渗透手段** | 墨天逸在问道堂工作（卧底身份） |
| **监控目标** | 张羽、白真真、玉星寒 |
| **关注点** | 张羽进入灵界了，他们在修炼什么 |
| **未知信息** | 穷盟为什么盯上张羽？组织的目的是什么？ |

### 嵩阳高中权力格局变化

```
旧格局（150章前）:
  学生会（周澈尘/蓝岭） → 压迫 → 穷人学霸（张羽/白真真）

新格局（155章）:
  张羽/白真真（嵩阳"帝王"）
    ├── 钱深/赵天行（帐下大将）
    ├── 何大有（主动示好）
    └── 学生会（不敢招惹）→ 转入地下调查

新战场:
  校内 → 灵界（问道堂/道心竞赛/未知威胁）
  新势力: 穷盟（神秘）
```
