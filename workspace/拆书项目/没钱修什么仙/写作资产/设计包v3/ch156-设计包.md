# 第156章 阿真开厂，张羽打工 — 设计包 (v3)

---

## 一、核心概念

> **穷盟老大宋虚的首次正面登场 —— 邪神信徒盯上张羽**
> 本章通过宋虚（穷盟创立者/邪神信徒）在灵界问道堂中远程观看张羽等人的幻境修炼场景，双线展开：一边是玉星寒的天武炼心诀幻境（保安vs工人暴乱）、白真真的寒魄冰心诀（冷酷老板压榨工人）、张羽的残牛舍身心诀（重病工人忍痛打工）；另一边是宋虚的暗中观察与评价。本章是**穷盟势力核心人物正式亮相 + 邪神线浮出水面**的关键转折。

**本章使命：**
1. **新角色登场**：宋虚（穷盟老大）首次正面出场，揭示其身份背景
2. **幻境修炼展示**：三人各自的修炼场景及心法运行效果
3. **势力暗线点燃**：宋虚作为邪神信徒，计划狩猎张羽背后的邪神来提升潜力
4. **穷人世界观深化**：通过宋虚的身世，展示穷人家族在豪族体系中的挣扎

## 二、情节框架

### 三层嵌套结构

| 层 | 内容 | 核心信息 |
|---|------|----------|
| **现实层(宋虚)** | 宋虚在灵界问道堂中观看幻境 | 穷盟老大+邪神信徒身份→目标锁定张羽 |
| **幻境层(玉星寒/白真真/张羽)** | 工厂场景：保安vs工人、老板压榨、员工忍痛流水线 | 心法淬炼场景的运行逻辑 |
| **背景层(宋虚身世)** | 宋家血脉→爆金币→爷爷墓地变现→遇邪神 | 穷人天才的成长逻辑 = 豪族淘汰者的宿命 |

### 关键节点

1. **宋虚身世披露**（第1-25行）
   - 宋家豪族血脉 · 但已被淘汰几代
   - 太爷爷还能跪宋家主家 → 宋虚连宋海龙好友都不是
   - 爷爷爆爸妈金币 → 供父亲上大学 → 父亲赌博败光家产
   - 宋虚先一步将父亲抵押给宋家主家，变卖爷爷墓地和尸身
   - **转折点**：遇到邪神，获得潜能提升——成为邪神信徒
   - **氛围：** 穷人=豪族淘汰者的宿命循环，所谓"家族羁绊"即互相榨取

2. **幻境中的玉星寒**（第37-50行）
   - 保安队长身份 · 混战中的天武炼心诀
   - 阿真在场外喊话："给我全力出手！不论打死打伤，厂里都会帮你们赔钱！"
   - **心法效果：** 思想越契合心法，心法效果越强
   - 在战场上的天武炼心诀修行效果远胜现实

3. **幻境中的白真真（老板）**（第55-70行）
   - 寒魄冰心诀：剥离感情，冷静计算得失
   - 握玉星寒的手说"干得很好，下个月涨薪"
   - **宋虚点评：** 这门心法最初是为**战斗/刺杀**创造，工厂经营的复杂环境反而更适合淬炼
   - 保持冰心，在复杂环境计算得失

4. **幻境中的张羽（流水线工人）**（第73-97行）
   - 面色苍白、身体虚弱、有肺病
   - 残牛舍身心诀镇压疼痛，前提是**必须有一线希望**
   - 白真真降薪20% → 希望陨灭 → 疼痛反弹
   - 请假被拒、工资拖欠、身份证被借用贷款
   - 张羽心中希望不断走向陨灭

## 三、人物设计

### 宋虚 —— 穷盟老大 / 邪神信徒 / 白龙高中学霸

| 维度 | 分析 |
|------|------|
| **身份** | 白龙高中学霸 · 穷盟创立者（墨天逸口中的"老大"）· 宋家豪族血脉(已被淘汰) |
| **核心矛盾** | 骨子里是豪族淘汰者的后代，但渴望通过考上十大打破宿命 |
| **邪神线** | 已经修行了"仪式"技艺的邪神信徒，目标是找到并狩猎张羽背后的邪神 |
| **对张羽的态度** | 好奇+警惕——想通过张羽找到快速变强的"某个原因" |
| **心法观** | 认为残牛舍身心诀是"废物修炼的废物功法"，寒魄冰心诀才是正道 |
| **价值底色** | "穷人只能靠互相帮助闯出一片天"——但骨子里还是以考上十大为唯一目标 |

### 墨天逸 —— 穷盟成员 / 问道堂卧底

| 维度 | 分析 |
|------|------|
| **角色功能** | 宋虚的眼睛和耳朵，在问道堂工作（卧底），负责监控张羽等人 |
| **态度** | 对老大绝对忠诚，"不拿加班工资免费加班" |
| **对张羽的判断** | "他的想法和我们的很接近"——感性上倾向于信任 |

### 玉星寒 —— 幻境中的保安队长

| 维度 | 分析 |
|------|------|
| **幻境身份** | 工厂保安队长，代表"绝不肯少赚一分钱的公司" |
| **心法匹配度** | 天武炼心诀在战场上如鱼得水，身心契合 |
| **关键设定** | 幻境会让人"暂时忘却现实世界中的一切" |
| **心法效果** | 越战越勇，心法运转越来越顺畅 |

### 白真真 —— 幻境中的冷酷老板

| 维度 | 分析 |
|------|------|
| **幻境身份** | 工厂老板，修行寒魄冰心诀 |
| **核心表现** | 冷漠计算：降薪、压榨、开除、用员工身份证贷款 |
| **心法状态** | "剥离一切感情，冷静计算一切得失"——心法运转越来越流畅 |
| **宋虚评价** | "这门心法在这种场景下理解越来越深刻，未来在考场上也必然能大放异彩" |

### 张羽 —— 幻境中的重病工人

| 维度 | 分析 |
|------|------|
| **幻境身份** | 流水线工人，修行残牛舍身心诀 |
| **身体状态** | 面色苍白、肺病、每次呼吸都钻心地痛 |
| **心法逻辑** | 需要"一线希望"才能镇压疼痛 → 希望不断被压榨断绝 → 疼痛加剧 |
| **宋虚评价** | "这种心法只会练成个只会挨打不会还手的废物" |

## 四、设定细节

### 宋家豪族体系的淘汰逻辑

| 设定元素 | 说明 |
|---------|------|
| **血脉羁绊机制** | 每一代爆爸妈金币 → 投资自己 → 考上大学 → 当公司员工 → 养全家 |
| **风险** | 一旦投资方向错误（如赌博），全家资产被榨干 |
| **底层淘汰者** | 几代以后，子孙连宋家主家的人脉都断了，外人甚至不知道是豪族出身 |
| **宋虚的操作** | ①将父亲抵押给宋家主家赚第一桶金 ②变卖爷爷墓地+尸身赚第二桶金 |
| **爷爷名言** | "与其抱怨环境，不如抱怨家人" |

### 灵界幻境修炼机制

| 设定元素 | 说明 |
|---------|------|
| **核心规则** | 幻境让人暂时忘却现实，全身心投入当前情境 |
| **心法契合度** | 自身思想越是和心法相契合，心法效果便越强 |
| **淬炼原理** | 在情境中加深对心法的理解和契合度 → 道心增强 |
| **场景生成** | 根据修炼者的心法需求自动生成对应场景 |
| **幻境主人** | 玉星寒是场景主人，其他人无法随意调整视角 |

### 寒魄冰心诀（白真真 章节版）

| 设定元素 | 说明 |
|---------|------|
| **原始用途** | 战斗/刺杀心法：出手前后保证心若冰清，天塌不惊 |
| **现代衍生用途** | 股民/投资者修行，辅助投资计算 |
| **10级效果** | 暂时压制情绪，变为冷酷无情，为成功不择手段 |
| **淬炼场景** | 工厂经营的复杂环境（迷雾、骗局、压力、形势变化）比战斗更适合淬炼 |

### 残牛舍身心诀（张羽 章节版）

| 设定元素 | 说明 |
|---------|------|
| **表面功能** | 让穷人"多干点活，少治点病，少要点赔偿款" |
| **运行条件** | 必须有一线希望（哪怕只是提前下班的希望）才能镇压疼痛 |
| **幻境表现** | 希望不断被剥夺 → 疼痛不断反弹加剧 |
| **宋虚的误解** | 认为是"废物修炼的废物功法" |
