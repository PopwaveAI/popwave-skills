# 第158章 张羽发动仪式 — 设计包 (v3)

---

## 一、核心概念

> **邪神对决的前奏 —— 仪式系统的全面展示**
> 本章是**仪式系统的首次系统性设定释放**。宋虚在现实地下室对张羽发动"咒影仪式"——通过灵界接触种下影蛊，窃取情报。福姬第一时间发现影蛊，指导张羽在烂尾楼布置反制仪式。仪式系统与符咒系统的对比（网络基建/神力调用/正神vs邪神）、仪式的四大要素（法坛/祭品/神物/象征）、日常物品充当仪式材料的妙用——完整展示了昆墟的**仪式学世界观**。

**本章使命：**
1. **新系统上线**：仪式系统全面展示（原理/要素/操作）
2. **邪神对抗升级**：福姬 vs 宋虚背后的邪神——第一回合
3. **张羽-福姬信任加深**：福姬展示实用价值，张羽依赖其指导
4. **宋虚成功种蛊**：但不知道福姬已经发现——信息差拉满

## 二、情节框架

### 双线对立结构（施术者 vs 反制者）

| 线索 | 内容 | 核心信息 |
|------|------|----------|
| **宋虚线（施术）** | 地下室中发动咒影仪式 → 种下影蛊 → 等待张羽下次登灵界 | 邪神信徒的仪式操作 |
| **张羽线（反制）** | 福姬发现影蛊 → 指导张羽布置仪式 → 成功控制影蛊 | 邪神vs邪神的对抗 |

### 关键节点

1. **仪式系统设定释放**（第1-10行）
   - 仪式与符咒的相通点：都能通过网络施展
   - **仪式 vs 符咒的区别：** 仪式用途不广，但能逃过正神监管
   - **咒影仪式：** 以邪神力凝练影蛊，寄生于对方影子中窃取情报
   - 发动条件：必须通过**灵界接触**（普通网络不行）

2. **宋虚成功发动咒影仪式**（第12-15行）
   - 在灵界看到张羽 → 立即发动
   - 张羽完全不知道被下了影蛊

3. **福姬发现影蛊**（第50-64行）
   - 张羽刚断开灵界 → 福姬立刻警觉："有人盯上你了"
   - **影蛊特性：** 平时只收集情报、隐匿自身 → 直到再次连上灵界才传递情报
   - **福姬的自信：** "一位曾经屹立于邪神顶点……立刻就发现了这小把戏"
   - 对方以为张羽是"被远程控制的邪神信徒"→想通过张羽找到其背后的邪神

4. **张羽布置反制仪式**（第84-105行）
   - 地点：烂尾楼（安静、无人打扰）
   - **神力网络概念：** 八部正神的网络遍布昆墟→邪神可"借道而行"
   - **四大仪式要素：**
     - **法坛**：神网交接之处（→板凳将就）
     - **祭品**：神力引子（→烧到一半的拜神蜡烛）
     - **灵机保存之神物**：象征灵机充盈（→两只手电筒开到最大）
     - **邪神化身之象征**：代替邪神像（→玩偶熊）
   - **仪式圆环：** 用铜线（神力优良导体）围成

## 四、设定细节

### 仪式系统 —— 全面设定

| 设定元素 | 说明 |
|---------|------|
| **仪式 vs 符咒** | 仪式能逃过正神监管记录，做规则之外的事；符咒依赖八部正神的基建 |
| **施展基础** | 神力网络（与符咒共用）——八部正神管理的网络，邪神可"借道而行" |
| **网络拜神** | 祭拜正神的香烛纸扎渐渐不好找——现在流行网络拜神 |
| **与灵界的关系** | 咒影仪式必须通过灵界接触才能发动（普通网络→灵界→意识） |

### 咒影仪式 —— 具体流程

| 步骤 | 说明 |
|------|------|
| **1. 目标条件** | 目标必须进入灵界 → 通过灵界接触建立连接 |
| **2. 邪神力凝练** | 将邪神力转化为"影蛊"（一种黑色小虫） |
| **3. 种蛊** | 影蛊寄生于目标的影子之中 |
| **4. 收集情报** | 影蛊日夜寄生、暗中收集情报 |
| **5. 传递情报** | 目标再次连上灵界时，影蛊抓住机会传递情报给施术者 |
| **指令机制** | 影蛊体内留有指令：如何搜集情报 + 如何传递情报 |

### 仪式四大要素（福姬版）

| 要素 | 标准形式 | 本章替代（穷凑版） | 原理 |
|------|---------|-------------------|------|
| **法坛** | 正式法坛 | 板凳 | 神网交接之处 |
| **祭品** | 香烛纸扎、雕像木偶 | 烧到一半的拜神蜡烛（神力引子） | 提供神力引子 |
| **灵机保存之神物** | 专业灵物 | 两只手电筒开到最大 | 光照中有灵机，象征灵机充盈 |
| **邪神化身之象征** | 邪神像 | 玩偶熊 | 代替邪神像，防不胜防 |
| **法阵基础** | 专业法阵 | 电线围成圆圈 | 铜=神力优良导体，束缚神力 |
| **台词** | 正式咒文 | "有虫有虫，摄拿出影，天地神力，急急如律令" | 驱动仪式 |

### 影蛊的情报线（信息差）

```
宋虚 → 种蛊成功 → 以为得手
                              ↕ [信息差]
福姬 → 发现影蛊 → 指导反制 → 控制影蛊 → 伪装传递假情报 → 反向探查
```
