# 第159章 参加第一轮筑基考试 — 设计包 (v3)

---

## 一、核心概念

> **筑基考试前夜 —— 穷人的极限冲刺与最终困境**
> 本章是**暴风雨前的沉淀**。主线有三：一是福姬在烂尾楼成功发动反制仪式，吸收影蛊的邪神力，恢复了少许实力，并准备伪装影蛊传递假情报；二是张羽和白真真在八月中旬迎来筑基考试，两人在极端拮据和药物副作用的双重压力下进行最后冲刺；三是乐沐岚公寓的电费危机与张羽的身体警告。本章是**考试前的心态沉淀 + 福姬线的小幅进展 + 经济危机逼近极限**的三线并行。

**本章使命：**
1. **仪式战果回收**：福姬成功吸收影蛊，恢复一点实力
2. **考试前状态交代**：张羽和白真真的实力/经济/身体状态
3. **压力最大化**：钱包即将见底、药物副作用爆发、贷款压力
4. **张羽决心展示**：在绝境中变得更强——残牛舍身心诀效果拔群

## 二、情节框架

### 三线并行结构

| 线索 | 内容 | 核心信息 |
|------|------|----------|
| **仪式线(A线)** | 福姬完成仪式 → 影蛊爆散 → 吸收邪神力 → 准备伪装传递假情报 | 福姬恢复一点实力 + 情报战反转 |
| **考试线(B线)** | 张羽白真真疯狂修行 → 药物副作用 → 钱包见底 → 前往仙都 | 实力：法力87.75/肉体8.06级/道心6级(18%) |
| **生活线(C线)** | 乐沐岚提醒电费8万 + 药物副作用 + 直播间代言仿妖筋肉 | 经济与健康双双危机 |

### 关键节点

1. **仪式完成 · 影蛊爆散**（第1-22行）
   - 福姬发动仪式 → 咒文"有虫有虫，摄拿出影，天地神力，急急如律令"
   - 神力从香烛中浮现，被福姬临时驾驭
   - 黑色小虫从影子中挣扎爬出 → 爆散为大量黑色咒文
   - 福姬吸收咒文 → 恢复一点点实力

2. **情报战反转部署**（第23-39行）
   - 福姬分析影蛊中的指令 → 明白对方传递情报的方式
   - 策略：下次张羽登灵界时，伪装影蛊发假情报
   - 张羽命令：不要惊动对方 → 先搞清对方实力/位置/人手
   - **福姬的野心：** "等你实力更强一些……我们就把我这位同事生吞活剥了"

3. **宋虚的等待**（第40-45行）
   - 宋虚脱灵界面罩，以为影蛊成功种下
   - "接下来就是等待张羽下次登陆灵界"
   - **宋虚的目标：** "若能够掌握你身上的秘密……我考上十大的事情便有九成以上的把握"

4. **直播间仿妖筋肉代言**（第48-64行）
   - 张羽和白真真为仙韵集团的"仿妖筋肉"带货
   - 29万9全套（劣化版的劣化版）
   - 张羽看到20万代言费到账 → 选择什么也不说
   - **暗讽：** 穷人不配用最好的技术

5. **乐沐岚的电费警告**（第66-81行）
   - 张羽擦地（乐沐岚汗水不能泄漏）
   - **电费8万！** → "到底在房间里做什么？种草药吗？"
   - 乐沐岚指出药物副作用："没有超级代谢的你，最近两个月药物用量确实过猛了"
   - 张羽："我的钱包要到极限了"

6. **考试前夕的状态汇总**（第82-112行）
   - 法力: 87.75 | 肉体强度: 8.06级 | 道心: 6级(18%)
   - 钱包: 5万块（白真真4万）
   - **白真真的总结：** "我们已经穷得连下个月贷款都快还不起了"
   - "但……我们的仙道潜力已经被彻底激发了出来"
   - **残牛舍身心诀效果：** 在压力下全身血液加速，身体越来越兴奋
   - **寒魄冰心诀效果：** 巨大风险下筋肉/神经/大脑越来越活跃
   - 男女厕所 → 点击网页进场 → 脚下浮现大洞 → 传送消失

## 三、人物设计

### 张羽 —— 考试前的极限冲刺者

| 维度 | 分析 |
|------|------|
| **实力状态** | 法力87.75 / 肉体强度8.06级 / 道心6级(18%)——远超普通高二生，但离仙都顶尖还有差距 |
| **经济状态** | 仅剩5万块——"再过几天这药是想吃也吃不起了" |
| **身体状态** | 药物副作用初显（胸口抽痛）——但没有超级代谢，用量过猛 |
| **心态** | 目标已经升级：不再满足于超过白龙人，而是对标仙都超级高中生 |
| **心法表现** | 在贷款/考试/余额/药物副作用等强烈压力下→身体越来越兴奋（绝境越强） |

### 白真真 —— 冷静的搭档

| 维度 | 分析 |
|------|------|
| **经济状态** | 仅剩4万块——"穷得连下个月贷款都快还不起了" |
| **心态** | "我们强得可怕，已经不可与两个月前同日而语" |
| **心法表现** | 巨大风险下筋肉/神经/大脑越来越活跃 |
| **角色功能** | 给出最后冲刺的总结陈词 |

### 福姬 —— 恢复一丝实力的邪神

| 维度 | 分析 |
|------|------|
| **实力恢复** | 吸收影蛊后"总算恢复了那么一点点实力" |
| **安全提升** | "下次跟着张羽一起去参加考试的时候，安全性大大提升" |
| **下一步计划** | 伪装影蛊发假情报 + 反向探查对方实力/位置 |
| **野心** | 等时机成熟，"把我这位同事生吞活剥了" |

### 宋虚 —— 不知情的成功者

| 维度 | 分析 |
|------|------|
| **状态** | 以为阴谋成功，等待下次灵界连接 |
| **目标** | 掌握张羽秘密→抓住背后的邪神→激发潜力→考上十大 |
| **信息差** | 不知道福姬已发现影蛊并准备反制 |

## 四、设定细节

### 张羽八月中旬实力汇总

| 属性 | 数值 | 对比 |
|------|------|------|
| **法力** | 87.75个单位 | 远超嵩阳高二生，接近炼气极限（100） |
| **肉体强度** | 8.06级 | 接近白龙高中高三学生水平 |
| **道心** | 6级(18%) | 突破6级时间不长，仍在成长 |
| **经济** | 5万块（白真真4万） | 下个月贷款可能还不上 |
| **药物副作用** | 胸口抽痛 | 没有超级代谢，用量过猛 |

### 仪式系统补充设定

| 设定元素 | 说明 |
|---------|------|
| **仪式发动** | 需沿电线圆环游走 + 诵读咒文 |
| **咒文形式** | "有虫有虫，摄拿出影，天地神力，急急如律令"——道教咒文风格 |
| **香烛作用** | 化为灰飞 → 释放神力 |
| **神力性质** | 本为正神神力 → 被邪神临时驾驭转化为邪神力 |
| **影蛊结构** | 由邪神力凝聚 → 体内含有指令咒文 |
| **爆散形态** | 黑色小虫挣扎 → 爆成大量黑色咒文（烟雾状） |
| **邪神吸收** | 可通过直接吸收咒文来恢复实力 |

### 影蛊情报分析机制

| 步骤 | 说明 |
|------|------|
| 1. 捕获 | 以仪式逼迫影蛊从影子中现身并爆散 |
| 2. 分析 | 研究影蛊体内的指令（如何搜集/传递情报） |
| 3. 伪装 | 下次目标登灵界时，模仿影蛊的信号发送假情报 |
| 4. 反向探查 | 通过对方接收假情报时的反应查明对方身份/位置/实力 |

### 仿妖筋肉技术（劣化版）

| 设定元素 | 说明 |
|---------|------|
| **原版技术** | 真正的仿妖筋肉改造——高端昂贵 |
| **直播间版** | 29万9全套——"劣化版的劣化版" |
| **操作流程** | 去诊所切下筋肉→寄给公司培养→发回安装 |
| **宣传语** | "不需要住院"、"假一赔十"（赔十块肌肉...） |
| **客户群体** | 买不起原版的人——穷人 |
