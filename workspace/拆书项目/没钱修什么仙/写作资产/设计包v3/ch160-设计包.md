# 第160章 神秘张羽 — 设计包 (v3)

---

## 一、核心概念

> **仙都阶层的赤裸展示 —— 道心测试排名引爆悬念**
> 本章是**第一轮筑基考试的开场**。主角张羽和白真真通过传送进入考试地下通道/大厅，第一次正面接触仙都的阶层生态链。核心冲突围绕"仙都至上主义"展开：戴行之（六环内仙都人）对张羽这个"外地人"的鄙视、夜凌霄（二环内顶级学霸）对红包符的沉迷、排名揭晓时"第一名55号估值未知"的悬念。本章揭示了仙都社会的**环套等级体系**，并以**神秘第一名（张羽）**的隐藏实力收尾。

**本章使命：**
1. **新地图展开**：第一轮筑基考试场景（地下通道/大厅）
2. **新角色登场**：戴行之（仙都至上主义者）、夜凌霄（顶级学霸）、云景（最强高中生）
3. **阶层体系展示**：仙都环套等级（十环→二环→中心）
4. **悬念制造**：道心测试第一名估值未知→神秘55号=张羽
5. **暗线延续**：夜凌霄的红包符研究 + 张羽隐藏实力

## 二、情节框架

### 阶层递进式结构

| 层 | 对应人物 | 环位 | 核心特征 |
|----|---------|------|---------|
| **外层** | 张羽/白真真 | 外地人（嵩阳） | 被鄙视的乡下人 |
| **中层** | 戴行之 | 北六环内 | 鄙视外地人，跪舔上城区 |
| **内层** | 夜凌霄 | 上城区二环内 | 追求红包符奥秘，冷漠但礼貌 |
| **顶层** | 云景 | 上城区？ | 最强高中生，法骸眼睛 |

### 关键节点

1. **传送与相遇**（第2-21行）
   - 张羽传送到地下通道
   - 遇到戴行之 → 打招呼被无视
   - **戴行之的仙都至上视角：** "外地人 = 两个物种的区别"
   - "祖上十代都生活在仙都"——仙都至上主义的核心
   - "仙都的上城更是可以称之为昆墟1.5层"

2. **夜凌霄出场**（第23-37行）
   - 戴行之对夜凌霄跪舔："二环的人只跟二环的人做朋友……一辈子都不会下来"
   - 夜凌霄低头练红包符
   - 张羽吐槽："这脏东西你随便乱扔"
   - 夜凌霄认出张羽→意外这个外地人没有被淘汰

3. **大厅中的排名之争**（第40-62行）
   - 仙都爷们逼迫考官公布排名
   - 张羽和白真真的对话："一群唯分数论者" / "真正的仙道哪里是一时的分数和排名能说清的"
   - 戴行之的反击："分数就是一切！排名就是一个人有没有资格在仙道之路上继续前进的标准！"
   - "只有知道自己排名落后的家伙，才会害怕排名"

4. **排名揭晓**（第63-93行）
   - 第一关：推金属大门（无人关心）
   - 第二关：道心测试排名
   - **第一名：55号，估值未知**
   - **第二名：夜凌霄（23号），估值55亿**
   - **第三名：云景（11号），估值50亿**
   - 戴行之的寻找："第一名55号……比云景、夜凌霄更强更有钱的人是谁？"
   - **悬念：55号张羽的身份未被发现**

## 三、人物设计

### 戴行之 —— 仙都至上主义者（新角色）

| 维度 | 分析 |
|------|------|
| **身份** | 仙都六环内居民，祖上十代都生活在仙都 |
| **核心理念** | "仙都=昆墟一层中心，上城区=昆墟1.5层" |
| **行为逻辑** | 鄙视外地人、跪舔二环人、追捧排名 |
| **对待张羽** | 看到是外地人→无视+挺腰→"臭外地的，想来仙都抢我们的筑基证？" |
| **对待夜凌霄** | 低头弯腰讨好→"二环的人只跟二环的人做朋友" |
| **价值观金句** | "分数就是一切！排名就是一个人有没有资格在仙道之路上继续前进的标准！" |
| **祖训** | "来到一群人中，首先认清分最高最有钱的人是谁"——戴家祖训 |
| **角色功能** | 仙都阶层歧视的具象化代表 |

### 夜凌霄 —— 顶级学霸 / 红包符研究者

| 维度 | 分析 |
|------|------|
| **身份** | 上城区二环内，仙都市最有钱的家族之一 |
| **实力** | "被绿洲集团评价为当代高中生里最完美的肉体以及最完美的血脉" |
| **排名** | 道心测试第二名，估值55亿 |
| **当前关注** | 对红包符、好感符到了入迷程度——"九成把握里面藏了不得了的利益" |
| **性格** | 冷漠、自我——对张羽的吐槽只有淡淡回应 |
| **对张羽** | "这个外地来的学生没有在上一次测试中被淘汰"——有轻微注意 |
| **暗线** | 独享红包符秘密，谁也不敢告诉 |

### 云景 —— 昆墟一层最强高中生

| 维度 | 分析 |
|------|------|
| **身份** | 仙云高中学生，被紫云药业称为"昆墟一层的最强高中生" |
| **实力** | "一个月前，除了法力之外，道心和肉体强度都已经达到炼气极限" |
| **排名** | 道心测试第三名，估值50亿 |
| **特征** | "那一双灿若星辰的眼睛……那就是法骸" |
| **角色功能** | 仙都顶尖战力的标杆 |

### 55号（神秘张羽）—— 隐藏实力的主角

| 维度 | 分析 |
|------|------|
| **排名** | 道心测试第一名（估值未知） |
| **为什么"估值未知"** | 张羽的贫穷让考官无法用财富估值 |
| **悬念价值** | 所有人都知道"第一名肯定比云景夜凌霄更强"→但找不到人 |
| **信息差** | 张羽不知道自己排名第一→被列为神秘存在 |

## 四、设定细节

### 仙都环套等级体系

| 等级 | 定义 | 代表 |
|------|------|------|
| **仙都** | 昆墟一层中心 | 最强、最聪明、最有钱的人的集中地 |
| **上城区（浮空城）** | 中心的中心 = 昆墟1.5层 | 二环人一辈子都不会下来 |
| **十环~三环** | 从上城区外到内 | 环数越小越尊贵 |
| **二环内** | 中心的中心的中心 | 夜凌霄——跟二环人结婚繁衍 |
| **六环内** | 上城区中层 | 戴行之——鄙视外地，跪舔上城 |
| **六环外** | 上城区下层 | 戴行之补充强调"是六环内" |
| **外地人（非仙都）** | 乡下地方 | 张羽/白真真——全仙都人都能认出来 |

### 道心测试排名机制

| 设定元素 | 说明 |
|---------|------|
| **测试内容** | 用户在幻境中能抵抗住多少价值的诱惑 |
| **排名依据** | 能坚持到多少亿的报价 |
| **估值含义** | 既是道心坚毅程度的体现，侧面反映选手的身价 |
| **第一名问题** | 55号"估值未知"——因为张羽太穷，无法估值 |
| **排名公布效果** | "像在电影院里找自己的座位"——确认自己排名→确认讨厌的人的排名 |

### 排名前三对比

| 排名 | 选手号 | 姓名 | 估值 | 身份背景 |
|------|-------|------|------|---------|
| 1 | 55号 | 张羽（神秘） | 未知 | 嵩阳穷学生 |
| 2 | 23号 | 夜凌霄 | 55亿 | 上城区二环内/绿洲集团 |
| 3 | 11号 | 云景 | 50亿 | 紫云药业/最强高中生 |

### 红包符研究线延续

| 设定元素 | 说明 |
|---------|------|
| **夜凌霄的沉迷程度** | 走路都在练习——"十分把握里面藏了不得了的利益" |
| **保密状态** | 谁也不敢告诉，只敢自己慢慢研究 |
| **对张羽的态度** | 认出张羽→"这个外地来的学生没有被淘汰"→但没有更多互动 |
| **红包符危险性** | 张羽吐槽"乱扔红包符危害公众安全"——证明红包符确实有杀伤力 |
