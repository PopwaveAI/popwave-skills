# 第161章 小丑位的召唤 — 设计包 (v3)

---

## 一、核心概念

> **被捧上神坛的穷人 —— 张羽的"小丑位"危机**
>
> 张羽因道心测试"未知估值"被夜凌霄、戴行之等人视为隐姓埋名的超级富豪，成为全场焦点，但他内心清楚自己是个下个月贷款利息都快还不上的穷鬼。法力考试中使用统一发放的外置灵根"气海无涯"，他连用都不会用，贫穷底色开始暴露。本章是**身份误会的巅峰 + 实力差距的迎面痛击**，为张羽的绝地反击做铺垫。

**本章使命：**
1. **身份误会顶点**：全场将张羽当成隐姓埋名的超级富豪，误会最大化
2. **实力差距展示**：云景、夜凌霄法力考试一骑绝尘，张羽排第21名
3. **新系统上线**：外置灵根"气海无涯"作为统一考试设备登场
4. **悬念铺垫**：小丑位召唤的恐惧感，悬念下一轮张羽如何破局

---

## 二、情节框架

### 单线集中结构（考场线）

| 阶段 | 内容 | 核心信息 |
|------|------|----------|
| **A 身份误会** | 夜凌霄/戴行之对张羽"未知估值"的猜测 | 误会被放大成"超级富豪"形象 |
| **B 考试规则** | 邓丙丁宣布流程：法力→体育→武功 | 三门考核，总分前20晋级 |
| **C 法力考试** | 使用外置灵根"气海无涯"统一设备 | 张羽不会用，暴露贫穷 |
| **D 主角困境** | 云景夜凌霄双龙争霸，张羽排21名 | 实力差距的直观展示 |

### 关键节点

1. **未知估值的震慑力**（第1-13行）
   - 夜凌霄首问：为何是"未知"？
   - 戴行之推理：撑爆幻境 → 极其有钱
   - 全场学生注目55号，将张羽视为亿万身价的富家子弟
   - **关键心理**：张羽内心恐慌却只能硬撑

2. **白真真的担忧**（第21-25行）
   - "你现在装这么大，一会儿正式比赛开始露了相，会成为老赵嘴里的那种小丑"
   - 交代张羽和白真真只有高三水准，与顶级仙都爷有差距
   - 目标分化：白真真只求通过，追赶是未来几个月的事

3. **外置灵根暴露贫穷**（第68-81行）
   - 统一发放"气海无涯"（中端品牌）
   - 张羽："这个外置灵根怎么用？"
   - **三重误解**：
     - 戴行之："是因为气海无涯对他来说太低端了"
     - 云景："在这里炫富？"
     - 其他学生：羡慕/敌意/好奇加剧
   - 张羽："也算是给我自己先铺垫一下……"（主动搭台阶）
   - 白真真内心点赞

4. **双龙争霸 vs 张羽窘境**（第82-94行）
   - 云景(11号) vs 夜凌霄(23号) = 第一第二激烈纠缠
   - 两根柱子如巨龙，其他柱子显得在"缩小"
   - 张羽的柱子排在20名开外
   - 戴行之最后的疑惑："55号的柱子怎么排这么后面？"
   - **悬念**：以第21名结束，为下一章的爆发留白

## 三、人物设计

### 张羽 —— 被架上神坛的穷鬼

| 维度 | 分析 |
|------|------|
| **表象** | 道心测试第一（未知估值）→ 被当成顶级富家子 |

## 四、设定细节

### 道心测试"未知估值"的连锁反应

| 元素 | 说明 |
|------|------|
| **估值含义** | 只有两种可能：要么撑爆幻境，要么极其有钱 |
| **对张羽的误读** | 所有人都选"极其有钱"这个推理方向 |
| **张羽的真实原因** | 「仪式的影响」——具体什么仪式尚未交代 |
| **后果** | 成为全场焦点，被云景、夜凌霄视为主要对手 |

### 法力考试规则

| 设定元素 | 说明 |
|---------|------|
| **考试内容** | 30分钟内持续输出法力，计总量和速度 |
| **考核维度** | ① 法力爆发力 ② 法力精确控制 ③ 一边输出一边吐纳恢复 |
| **难度定位** | 按课纲是高三内容，但筑基考试不拘泥于课纲 |
| **使用设备** | 外置灵根"气海无涯"（中端品牌）+ 检测设备 + 检测贴片 |
| **评分展示** | 光幕柱状图，按法力值实时排序 |

### 外置灵根"气海无涯"

| 设定元素 | 说明 |
|---------|------|
| **品牌定位** | 中端品牌 |
| **功效** | 增强吐纳，加速灵机吸收 |
| **使用方式** | 注入法力→拉大圆环→套在手上→与身体融为一体 |
| **使用者评价** | 戴行之认为"正常修炼用的灵根"；张羽从未用过 |
| **考试用途** | 统一发放，消除灵根差距（改革内容之一） |

### 嵩阳三人组的考试表现

```
张羽(55号): 第21名（跌落前20）
白真真(56号): 约第20名左右
玉星寒(31号): 第17名
```

### 法力考试排名动态

```
第一梯队: 云景(11号) ↔ 夜凌霄(23号)  双龙争霸
第二梯队: 第3-16名                  仙都顶尖学生
第三梯队: 第17-20名                 玉星寒/白真真在此挣扎
张羽: 第21名                       在前20名之外
```

### 张羽的内心压力链

```
道心测试第一(未知估值)
       ↓
被当成超级富豪
       ↓
法力考试暴露贫穷底色
       ↓
排在第21名(淘汰区)
       ↓
"小丑位正在向我召唤"
```
