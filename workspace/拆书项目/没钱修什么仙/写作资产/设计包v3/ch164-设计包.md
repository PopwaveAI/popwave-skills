# 第164章 暴食 — 设计包 (v3)

---

## 一、核心概念

> **富人的消化也是顶配的 —— 暴食考试中的阶级差距**
>
> 第二场肉体考试"暴食之争"开始，张羽和白真真期待免费大吃大喝，结果上场的是合成食物——他们吃了好几个月吃到想吐的东西。与此同时，云景胸口开洞进食、夜凌霄巨人化吞食、蒙涛暴食灵根依赖症初显。本章是**肉体强度的全面侧面考核**，也是**穷人炼体认知 vs 富人炼体手段的对比**。

**本章使命：**
1. **新考场展示**：暴食之争的规则和实战
2. **阶级对比**：穷人挨饿 vs 富人享受饮食；穷人吃腻合成食物 vs 富人第一次吃
3. **肉体内功概念**：消化能力=内部肉体强度
4. **消化危机铺垫**：为第165章张羽的解法做铺垫

## 二、情节框架

### 单线展开结构

| 阶段 | 内容 | 核心信息 |
|------|------|----------|
| **A 期待与失望** | 张羽白真真期待免费大餐→合成食物 | 穷人日常的讽刺 |
| **B 规则宣讲** | 邓丙丁宣布考试规则 | 不能使用灵根/法力，纯肉体比拼 |
| **C 高手展示** | 云景开洞、夜凌霄巨人化、蒙涛鲸吞 | 不同炼体路线的消化方式 |
| **D 省钱真因** | 黄子丑道破用合成食物的原因 | 正神为了省预算 |
| **E 消化危机** | 蒙涛排名下滑，第二阶段开始 | 消化力才是关键 |

### 关键节点

1. **免费大餐的幻想破灭**（第1-10行）
   - 张羽："自助餐是吧？老子一定要把报名费、交通费、传送费都给吃回本来"
   - 白真真："不会是无限预制菜畅吃吧？那不是爽死了？"
   - 玉星寒："这里可是仙都，下场考试就算吃上龙肉都很正常"
   - 龙肉畅想的破灭 → 合成食物端上来

2. **合成食物的打击**（第22-26行）
   - 桶盖打开 → 合成食物
   - 张羽和白真真吃了好几个月的合成食物，脸都绿了
   - 张羽："特幺的……上次报名的时候交了三万块报名费，就吃这个？"
   - 玉星寒尴尬："这帮正神到底贪了多少？"
   - **阶级讽刺**：对仙都爷来说合成食物是"给外地穷人吃的东西"，对张羽来说是吃吐了的日常

3. **暴食规则**（第27-39行）
   - 合成食物无限量供应
   - 不能使用灵根、法力
   - 可以使用心法（属于肉体力量的一部分）
   - 呕吐/藏食物=作弊→永远取消资格
   - 考的是整体的肉体强度：代谢、消化、肠胃募集力
   - **定义**："肉体内功" vs 奔跑跳跃是"肉体外功"

4. **高手展示**（第40-63行）
   - **蒙涛**：暴食灵根长期使用者，鲸吞一大桶，排名第三
   - **云景**：胸口张开大洞，直接将食物倒入体内，完全放弃味觉和吞咽
   - **夜凌霄**：气血充盈，筋肉拉伸变成小巨人，随意灌入口中
   - **对比**：蒙涛："这两个人的肉体简直就是怪物"
   - 展示了不同炼体体系在消化上的表现差异

5. **省钱真相**（第64-71行）
   - 邓丙丁解释：合成食物最好消化，负担最轻
   - 黄子丑内心道破："真正的原因还是为了节省预算"
   - "这帮高中生一个比一个能吃，真要上点值钱东西，还不被他们吃光预算了？"
   - **财政现实**：正神的预算也是有限制的

6. **阶级食感差异**（第72-89行）
   - 戴行之：合成食物是"给外地打工穷人吃的"
   - 大口吞咽时觉得越来越难吃
   - "恐怕也就只有那种天天吃的穷鬼才习惯了……"
   - 张羽内心："本来连吃这么久……就感觉要吃吐了，今天还要比赛吃这东西？"
   - 张羽问能不能加调料 → 被拒 → 戴行之认为"死有钱人平时吃太好了"
   - **最讽刺的一段**：张羽吃吐了=死有钱人；戴行之认为富人不该这么娇气
   - 蒙涛开始变慢：消化力跟不上，排名下滑

## 四、设定细节

### 暴食之争考试规则

| 设定元素 | 说明 |
|---------|------|
| **食材** | 合成食物（无限量供应） |
| **限制** | 不可使用灵根、法力，不可呕吐/藏食物 |
| **允许** | 心法（属于肉体力量的一部分） |
| **考校维度** | ① 代谢能力 ② 消化能力 ③ 肠胃募集力 |
| **定位** | 肉体的内功——与奔跑跳跃（外功）对应 |
| **作弊后果** | 永远取消筑基资格证考取资格 |
| **排名依据** | 光幕柱状图，按进食份量排序 |

### 合成食物的阶级意义

```
仙都爷视角（戴行之代表）:
  合成食物 = 给外地来仙都打工的穷人们吃的东西
  难以下咽 → 但为了修行可以忍
  评价张羽: 吃相难受 = 平时太舒服了

穷人视角（张羽代表）:
  合成食物 = 吃了好几个月吃到想吐的东西
  本来就要吐了，还要比赛吃这个
  问加调料被拒

讽刺结果:
  张羽吃合成食物难受 → 被戴行之当成"死有钱人娇气"
  实际原因: 他吃了几个月已经吃吐了
```

### 三种炼体路线在消化上的表现

| 选手 | 路线的消化表现 | 背后设定 |
|------|--------------|---------|
| **云景** | 胸口开洞，直接注入体内 | 肉体改造路线，放弃味觉和吞咽 |
| **夜凌霄** | 气血充盈变巨人，随意吞咽 | 完美血脉的肉体拉伸力 |
| **蒙涛** | 暴食灵根依赖，鲸吞快但消化慢 | 灵根依赖路线，脱离灵根就崩 |
| **戴行之** | （铺垫）三秒如厕的超快消化 | 肉体全面的肠胃力量 |
| **李彤云** | （铺垫）消化道改造手术 | 医疗科技路线 |

### 考试阶段二分

```
第一阶段（暴食）:
  谁吃得快 = 比拼吞咽速度和容量
  蒙涛在此阶段强势（暴食灵根练出来的）

第二阶段（消化）:
  谁消化得快 = 比拼代谢和内部控制
  蒙涛开始崩盘，依赖症暴露
  真正的考验开始
```

### 财政现实

```
正神用合成食物的三层真相:

第一层（邓丙丁说出口的）:
  合成食物最好消化，负担最轻，适合考试

第二层（黄子丑心里想的）:
  为了节省预算，学生们太能吃，上值钱东西预算不够

第三层（作者讽刺）:
  报名费收了三万，考试用合成食物——差额被贪了？
  玉星寒："这帮正神到底贪了多少？"
```
