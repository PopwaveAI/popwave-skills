# 第165章 张羽的消化力！ — 设计包 (v3)

---

## 一、核心概念

> **自残式消化 —— 用龙象崩山掌拍自己的穷鬼解法**
>
> 暴食考试进入消化决胜阶段，各种选手展示了不同的消化技巧：戴行之三秒如厕、李彤云拉肚子战术、玉星寒大蟾气、云景火炉炼丹式、夜凌霄完美吸收。张羽在被逼到第19名的绝境下，用龙象崩山掌一掌掌轰向自己的肚子——以掌力辅助压缩、粉碎食物，将肉体控制力发挥到极致。本章是**肉体控制力的巅峰展示**，也是**穷人的创造力最后武器**。

**本章使命：**
1. **消化方式大全**：展示所有选手的不同消化技巧
2. **穷人解法**：张羽用肉体控制力+自残方式突破极限
3. **与法力考试呼应**：上一场用龙象崩山掌轰法力，这一场用龙象崩山掌轰肚子
4. **第20名保卫战延续**：张羽再次在绝境中创新破局

### 关键节点

1. **戴行之的三秒如厕**（第1-13行）
   - 第一秒冲进厕所，第二秒巨响传出，第三秒系着裤子走出来
   - "拉的这么快，这是何等强大的体魄？"
   - 吃完就拉 → 快速清空 → 继续进食
   - 戴行之心得："消化可不止是把食物吃下去这个部分"

2. **蒙涛的暴食灵根依赖症爆发**（第16-27行）
   - 曾经鲸吞第一→因为没有暴食灵根辅助消化→排名一路下滑
   - "甚至因为平日里太过借助暴食灵根的关系……他发现自己肠胃的消化能力缺少了锻炼"
   - 看别人冲厕所，自己肚子毫无感觉→越来越急
   - 从第三名掉到第五名→向第六名掉去

3. **李彤云的拉肚子战术**（第37-49行）
   - 体表浮现乳白色粘稠物质（汗液蛋白质过高→消化不良状态）
   - "在未完全消化的状况下，将废物进行排出" = 拉肚子
   - 邓丙丁裁决："也算是一种肉体控制的表现"
   - 李彤云靠此战术冲到第18位

4. **白真真的极限撑肚**（第52-55行）
   - 雷殛真体的韧性 → 直接往肚子里填食物
   - 放弃高速消化，靠肚子容量硬撑
   - 排名第20位，形势危急

5. **玉星寒的大蟾气**（第56-59行）
   - 肚子一鼓一胀，发出呱呱叫声
   - 专门用来消化的功法
   - 邓丙丁评价："有一门辅助消化的炼体功法，每顿饭都能多获得滋养→日积月累的恐怖差距"

6. **云景的火炉炼丹式消化**（第61-66行）
   - 胸口开洞倒食物 → 全身上下通红 → 空气高温扭曲
   - 体内如炼丹炉，杂质化作青烟从毛孔和口中喷出
   - "可以一直吃到自己体力耗尽为止"

7. **夜凌霄的完美吸收**（第67-72行）
   - 变成小巨人后一直吃，肚子没大一点点
   - 夜氏完美血脉：几乎完美的吸收
   - "一年只需要上一次厕所"
   - 节省下来的体力和时间让无数人羡慕

8. **张羽的龙象崩山掌辅助消化**（第73-100行）
   - 观察夜凌霄：猜测对方在憋着→自己模仿
   - 一开始压缩食物残渣有用→但空间很快又满了
   - **灵光一闪**：龙象崩山掌10级→一掌掌轰向自己肚子
   - **组合技**：龙象崩山掌→穿透肌肤肠胃→轰击体内食物
   - **肉体控制力展示**：全身血肉骨头脏器协作→牵引挪移卸力→掌力绝大部分转移到食物中
   - "肠壁、胃壁上就好像多出了一道又一道掌印"
   - 粉碎→压缩→再粉碎→再压缩
   - **关键台词**："法力控制方面我确实很久没有进步了。但若是比起肉体控制技巧的话，在场哪怕这些仙都爷们……应该也没几人比我更强了"

9. **蒙涛模仿失败**（第86-90行）
   - 学张羽拍肚子→一开始有用→内腑疼到不行
   - "这家伙的脏腑到底强韧到了什么地步？这样还能继续吗？"
   - 凸出张羽肉体控制力的独特性

### 张羽的龙象崩山掌辅助消化系统

```
观察 → 夜凌霄在憋着 → 我也能憋
   ↓
尝试压缩食物残渣 → 空间很快占满
   ↓
龙象崩山掌10级 → 拍向肚子
   ↓
掌力穿透肌肤 → 穿透肠胃 → 轰击内部食物
   ↓
全身协作:
  血肉 ←→ 骨头 ←→ 脏器
     ↓ 牵引、挪移、卸力
  掌力绝大部分转移到食物上
   ↓
肠壁胃壁浮现掌印 → 粉碎 → 压缩
   ↓
再粉碎 → 再压缩（循环）
```

### 肉体控制力 vs 法力控制力的对比（张羽自我评估）

```
法力控制力: 几个月没进步，仅高三水准
           原因：大部分精力放在炼体上

肉体控制力: 在场仙都爷们也没几人比我更强
           原因：数月炼体+多次突破的积累
```

### 暴食灵根依赖症分析（蒙涛）

```
依赖期间:
  暴食灵根 → 高效消化 → 可以鲸吞 → 排名靠前
  肠胃自身消化能力的锻炼 = 0

脱离后:
  失去了灵根的辅助消化
  肠胃自身消化能力不足
  排名一路下滑
  急了但没用

模仿张羽:
  拍肚子 → 一开始有用 → 内腑疼到不行
  原因：同样操作需要极高的肉体控制力
```

### 夜氏完美血脉的消化设定

```
进食后:
  几乎完美吸收
  少量残渣 → 极少
  排泄频率: 一年一次

优势:
  ① 节省体力（不用频繁上厕所）
  ② 节省时间（不用花时间上厕所）
  ③ 营养吸收最大化

羡慕度:
  "每年不知道有多少人想要研究他们血脉中的此等神奇之处"
```

### 张羽两场考试的平行组合技

```
法力考试（第163章）:
  无相云罡（炮管）+ 龙象崩山掌（火药）+ 法力（子弹）
  → 体外输出法力

暴食考试（第165章）:
  肉体控制力（协调）+ 龙象崩山掌（粉碎力）+ 食物（目标）
  → 体内粉碎食物

共同点: 都用了龙象崩山掌，都是被逼入绝境后的创新解法
```
