# 第167章 最后一场，武道考试 — 设计包 (v3)

---

## 一、核心概念

> **暴食之争终局 + 武道考试规则释放**
> 玉星寒在星火真人的教诲下接受失败，放弃了与张羽的死磕。暴食之争结束后，排名定型：云景第一、夜凌霄第二、张羽第14、玉星寒第16、白真真第20。随后第三场武道考试的规则正式释放——学生自选功法在两小时内修炼，按难度和时间评分，练成的功法免费赠送。本章是**筑基考试规则体系的完整构建**，为后续章节的功法修炼大战铺路。

**本章使命：**
1. **角色成长收尾**：玉星寒完成"接受失败"的心态蜕变
2. **规则释放**：武道考试的完整规则——自选功法、星级难度、评分体系
3. **世界观搭建**：仙都学生的竞争格局（夜凌霄vs云景一线）
4. **悬念铺垫**：新规则下，每个学生将如何选择？

## 二、情节框架

### 双段结构：暴食终局 → 规则释放

| 段落 | 内容 | 核心信息 |
|------|------|----------|
| **暴食终局(A段，1-76行)** | 玉星寒放手 → 排名定局 → 学生反应 | 角色成长 + 排名洗牌 |
| **规则释放(B段，77-90行)** | 武道考试规则介绍 → 候选学生思考 | 新机制上线，悬念拉满 |

### 关键节点

1. **玉星寒的放手**（第1-43行）
   - 乳白汗水越来越白，消化系统逼近极限
   - 星火真人的教诲在脑海中响起
   - "该放手时就放手""败不可怕，只要钱还在"
   - 停止死磕，恢复正常进食
   - 张羽超越玉星寒，来到第15位
   - 邓丙丁评价：星火小子坠落1层后自省，把徒弟也调教好了
   - **角色核心：** 优越感崩塌后的成熟——接受失败是强者的能力

2. **正神视角的讽刺**（第44-53行）
   - 黄子丑吹捧"仙道昌盛，天下有福"
   - 小神拍下现场画面向上汇报（学生浑身冒烟、肚子鼓胀、呕吐、殴打自己）
   - 邓丙丁微笑要求挑几张好照片上报
   - **世界建设：** 天庭的官僚主义——只看表面景象

3. **仙都学生的暗线**（第54-76行）
   - 夜凌霄第二、云景第一的宿敌格局
   - 戴行之第7名、蒙涛第10名
   - 戴行之对张羽的疑惑：55号是外地人但只有14名？故意的？
   - 其他学生对张羽的目光带上疑惑
   - 黄子丑评价：仙都学生包揽前四张筑基证
   - 正神疑惑：嵩阳怎么多出三个有潜力的学生？
   - **伏笔：** 张羽的"低调"策略开始引起猜测

4. **武道考试规则释放**（第77-90行）
   - 邓丙丁介绍：武道考试 = 对武道技艺的掌握和理解
   - 黄子丑内心吐槽：实战考核包医疗费太贵，所以改考试
   - 规则核心：每人自选一门功法，2小时内修炼
   - 功法按1-10星难度分级，越高分越高
   - 练不成=0分
   - 练成的功法免费送（否则自己支付费用）
   - **设定释放：** 筑基考试的完整制度

## 四、设定细节

### 筑基考试三轮体系（完整版）

| 轮次 | 考试科目 | 规则要点 |
|------|---------|---------|
| **第一轮（本章）** | 法力 + 肉体 + 武道（综合） | 总分前20名进入第二轮，否则淘汰 |
| **第二轮（10月）** | 道心 / 法力 / 肉体 / 武功（分赛道） | 四条赛道各决出一名冠军 |
| **第三轮（1月）** | 同上（分赛道） | 同上 |
| **最终** | 四张筑基资格证 | 四条赛道各一位得主 |

### 武道考试规则

| 项目 | 内容 |
|------|------|
| **形式** | 学生自选功法，2小时内修炼 |
| **功法库** | 正神提供，按1-10星难度排序 |
| **评分** | 难度系数 × 修炼时间（时间越短分越高） |
| **失败惩罚** | 选中的功法练不成→该关0分 |
| **奖励** | 练成的功法免费赠送（未进前20需自付费用） |
| **限制** | 不能选已学会的功法（作弊处理） |
| **资源** | 练习所需兵器、材料、药物等现场提供 |

### 天庭官僚主义的讽刺

```
表面: 邓丙丁"心系苍生"→ 要"仙道昌盛"
实际: 黄子丑一心省医疗费预算
      小神拍照片向上面汇报"勃勃生机"
      "上面的大人最喜欢看后辈奋力竞争"
深层: 天庭只看表象，基层正神掐预算
```

### 第一轮后各人位次压力评估

|------|--------|--------|-----------|---------|

| 白真真 | 20名后 | 20名 | 必须较高 | 中高 |
