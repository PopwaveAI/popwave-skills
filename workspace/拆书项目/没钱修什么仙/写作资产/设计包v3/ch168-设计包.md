# 第168章 张羽的痛心 — 设计包 (v3)

---

## 一、核心概念

> **穷人的选择痛苦 — 面对三大顶级炼体功法的幸福烦恼**
> 武道考试的功法选择环节中，张羽面临巨大的"痛苦"：155万的春秋无尽禅、154万的钛金炼体要诀、152万的大天龙神力——三门顶尖炼体功法摆在面前，他只能选一门。这是穷人最大的幸福烦恼：**看到一切想要的东西，但只能选一个。** 与此同时，夜凌霄、云景、玉星寒、戴行之等人的选择也在同步展开，展现了不同阶层学生的选择逻辑。

**本章使命：**
1. **功法设定堆料**：释放三门顶级炼体功法的详细设定
2. **穷人思维展示**：张羽的功法筛选逻辑——最贵的必然最强
3. **群像同步推进**：各角色基于自身条件的选择策略对比
4. **悬念拉满**：张羽最终选了哪一门？其他人选了啥？两小时后谁能练成？

## 二、情节框架

### 多线并行：功法选择大乱斗

| 线索 | 角色 | 选择逻辑 | 核心信息 |
|------|------|---------|----------|
| **张羽线(A线)** | 张羽 | 炼体+练气后期+按价格排序→三大功法→犹豫到痛心 | 穷人思维+功法设定 |
| **夜凌霄线(B线)** | 夜凌霄 | 筛选10星→随便选没练过的 | 完美血脉的自信 |
| **云景线(C线)** | 云景 | 10星→直接选最难的森罗万象掌 | 霸道的强者心态 |
| **玉星寒线(D线)** | 玉星寒 | 按师父计划→找最贵的7星天罡气功→"欲练此功挥刀逆生" | 付代价换省钱 |
| **仙都众生(E线)** | 戴行之/蒙涛等 | 5-6星稳妥路线 | 中等阶层的风险控制 |

### 关键节点

1. **张羽的兴奋与计算**（第1-50行）
   - 看到一排排功法，眼中兴奋（辅助羽书修成任何功法轻松）
   - 先想"选最难的拿高分"→但意识到"通过考试就够了"
   - 核心策略：**免费修炼一门最适合的强大功法**
   - 锁定炼体武学+练气后期+按价格排序
   - 三大功法浮现：春秋无尽禅(155万)、钛金炼体要诀(154万)、大天龙神力(152万)
   - **设定释放：** 张羽的穷人筛选法——最贵的必然最强

2. **三大顶级炼体功法的设定**（第51-69行）
   - **春秋无尽禅(155万/8星)**：易筋洗髓，强壮脏腑，排出药毒丹毒，提高打药效果，真正的炼体圣功
   - **钛金炼体要诀(154万/8星)**：熔炼合金进入体内，大幅度增强抗打击能力和肉体承载力，合金品质决定强度上限
   - **大天龙神力(152万/8星)**：炼化龙血成半龙半人之躯，提高全方位的身体能力+潜力，副作用是龙性本淫需配合绝育手术
   - **情感核心：** "我都想要啊""这也太难选了"

3. **夜凌霄的选择**（第73-83行）
   - 完美血脉拥有者：先天发达神经系统+恐怖控制力
   - 学什么都是一看就会、一学就精
   - 许多功法已学会（正版用户）
   - 选了没学过的10星万剑无终诀
   - **角色展示：** 有钱+有天赋=随便选

4. **云景的选择**（第84行）
   - 最简单直接：按难度排序，选排在最前面的森罗万象掌（10星）
   - **角色展示：** 不需要犹豫，强者只需挑最难的

5. **玉星寒的抉择**（第85-103行）
   - 按师父计划走，不额外浪费时间
   - 选计划中最贵的7星天罡气功(90万)
   - "欲练此功，挥刀逆生"——需要修复绝育手术的身体残缺
   - 代价：道心受影响+激素变化+身心改变
   - 咬牙：**"富贵险中求，若为90万都不敢拼，日后又怎攀登仙道？"**
   - **设定释放：** 天罡气功的代价——逆生

| 维度 | 分析 |
|------|------|

## 四、设定细节

### 武道考试功法星级解读

| 星级 | 难度含义（普通高中普通学生） | 适合群体 |
|------|------------------------------|---------|
| 1-4星 | 简单 | 基础学生 |
| 5星 | 很有把握（戴行之标准） | 仙都精英稳妥选择 |
| 6星 | 有失败可能 | 仙都精英冒险选择 |
| 7星 | 挺冒险 | 玉星寒（90万天罡气功） |
| 8星 | 极大概率失败 | 张羽（春秋无尽禅155万） |
| 9星 | 基本没人选 | 李彤云（被逼无奈） |
| 10星 | 三年才能练成 | 云景、夜凌霄（森罗万象掌/万剑无终诀） |

### 三大顶级炼体功法对比

| 功法 | 价格 | 星级 | 核心机制 | 后续成本 |
|------|------|------|---------|---------|
| **春秋无尽禅** | 155万 | 8星 | 易筋洗髓+排毒+提高打药效果 | 无需外物，可自然提升等级 |
| **钛金炼体要诀** | 154万 | 8星 | 熔炼合金入体，增强抗打击和承载力 | 需要持续购买高品质合金 |
| **大天龙神力** | 152万 | 8星 | 炼化龙血成半龙半人，全方位提升 | 需要持续购买龙血，需绝育手术 |

### 张羽的选择逻辑全流程

```
1. 想选10星拿高分 → 但意识到通过考试就够了
2. 赤髓浑元气效率低 → 需要更好的炼体功法
3. 筛选: 炼体+练气后期 → 按价格排序
4. 目标: 155万/154万/152万三大功法
5. 排除逻辑:
   - 钛金炼体要诀 → 合金太贵，后续负担不起
   - 大天龙神力 → 龙血太贵，后续负担不起
   - 春秋无尽禅 → 无需外物就能提升等级 ← 唯一适合穷人的选择
6. 最终: 春秋无尽禅(155万/8星)
```

### 天罡气功的特殊代价

| 代价维度 | 具体内容 |
|---------|---------|
| **身体要求** | 身体的完整性（需修复绝育手术残缺） |
| **道心影响** | 逆生带来的心理冲击 |
| **生理影响** | 激素变化、身心改变 |
| **修行影响** | 对仙道之路有害 |
| **玉星寒的态度** | 咬牙拼了——"富贵险中求" |

## 五、套路归档

### 套路名称：穷人面对天价功法的痛苦选择

**适用场景：** 免费获得顶级资源时，穷人因后续成本不得不放弃其他好选项

**本章应用：**
- 张羽面对三门顶级炼体功法："我都想要啊"
- 但最终只能选春秋无尽禅——因为后续不需要外物
- 钛金炼体要诀需要合金、大天龙神力需要龙血——穷人养不起
- "唉，这也太难选了"
- **核心表达：** 穷人的选择自由是有限的——哪怕免费拿，也要考虑养不养得起

**归档文件：** `写作资产/套路库/穷人面对天价功法的痛苦选择.md`

### 套路名称：功法选择中的阶层对比

**适用场景：** 群像展示不同阶层在相同境遇下的不同决策逻辑

**本章应用：**

- 云景（强者）：直接选最难的，够自信

- 张羽（穷人）：选性价比最高的，考虑后续成本
- **核心表达：** 每个角色的选择都反映了他们的阶层和条件

- 大天龙神力：龙性本淫→需配合绝育手术
