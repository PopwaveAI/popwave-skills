## 一、核心概念

> **第二轮考试规则发布 + 正神分组博弈 + 夜凌霄对张羽下战书**
> 邓丙丁宣布第二轮专项考试规则（道心/法力/体育/武功四选二），正神们以"最大程度挖掘人才"为原则进行分组——将夜凌霄排入体育组、云景排入武道组，其余低分选手（含张羽、白真真）被当作"明年还能再考"的牺牲品填入死亡之组。分组揭晓后，张羽抽到体育组（对夜凌霄），白真真抽到武道组（对云景）。夜凌霄主动走到张羽身旁下战书："下一轮的竞争，你不用全力就不行了。顶峰再见。"

**本章使命：**
1. **规则过渡**：从第一轮综合考过渡到第二轮专项考
2. **正神决策逻辑展示**：以"最大效率输送人才"为名，牺牲低分考生
3. **张羽vs夜凌霄的宿敌关系确立**：夜凌霄主动下战书
4. **白真真vs云景的对位**：武道组同样死亡之组
5. **张羽的心态转变**：被战意点燃 → 但回家路上继续被邪神仪式逼着练功

### 关键节点

1. **第二轮考试规则**（第1-13行）
   - 时间：10月15日，地点相同
   - 专项考试：道心、法力、体育、武功四个赛道
   - 每人报名两个赛道，正神根据第一轮总分进行调整
   - 目的：挖掘人才 + 让四赛道人数均衡 + 避免强手挤在同一赛道
   - **规则特点：** 看似公平的分配机制，实际给了正神极大的操控空间

2. **张羽的报名**（第14-25行）
   - 原计划：体育（第一志愿），白真真武道（第一志愿）
   - 被正神禁止交流 → 凭默契独立报名
   - 第二志愿选择：道心（放弃法力）
   - 理由：发现白真真法力控制已经超过自己
   - **关键：** 双方都在担心撞车

3. **正神的分组决策**（第26-42行）
   - 邓丙丁看到云景和夜凌霄相互错开 → 满意
   - "这两个人放在哪两个赛道，便是哪两条赛道的获胜者"
   - 夜凌霄→体育组（第一志愿），云景→武道组（第一志愿）
   - 前十名尽量与夜凌霄、云景分开
   - 黄子丑："这张羽和白真真有点倒霉啊"
   - **核心逻辑：** 将分数低的选手尽量分到夜凌霄、云景两组
   - 理由：张羽、白真真今年被淘汰了明年还能再考
   - **正神的算盘：** 牺牲低分考生，保全高分选手的晋级机会

4. **分组结果**（第43-51行）
   - 张羽：体育组（vs夜凌霄）——死亡之组
   - 白真真：武道组（vs云景）——死亡之组
   - 玉星寒：体育组（同样死亡之组）
   - 张羽的反应："要跟这家伙竞争吗？"

5. **张羽的内心吐槽**（第49-59行）
   - "特幺的这家伙绝对开了吧？开的挂比我还大吧？"
   - "又有钱，又开挂，23号你真该死啊"
   - 希望夜凌霄出门能被创死（但知道不可能）
   - 抱怨羽书不给力 → "最终还得靠我自己来给你升级"
   - **身份认知变化：** 羽书从外挂→潜力→张羽才是羽书的外挂
   - **核心表达：** 张羽意识到自己得去狩猎邪神升级羽书

6. **夜凌霄下战书**（第68-83行）
   - 夜凌霄主动走近张羽和白真真 → 对视点头打招呼
   - 戴行之看到这一幕："夜凌霄竟然和他们主动打招呼了"
   - "今天的考试，不值得你用全力吗？"
   - 张羽初以为是嘲讽 → 后来意识到不是
   - "下一轮的竞争，你不用全力就不行了"
   - "我会期待到时候和你的较量"
   - "顶峰再见了"（顶峰=昆墟第二层）
   - 张羽接战："你放心，下一次考试，我肯定会让你感觉到惊喜的"
   - **宿敌关系正式确立**

7. **回家途中的仪式倒逼**（第84-95行）
   - 传送到车站厕所 → 不敢在仙都多逗留 → 立刻买车票离开
   - 列车启动，刚放松 → 仪式又响 → 叹息着到走廊练春秋无尽禅
   - 白真真看到心里暗骂"还卷？"
   - 白真真疲惫不想练 → 但能理解张羽的焦虑
   - 白真真收到催款电话（下个月两万多利息）
   - **双重压力：** 考试压力 + 经济压力

## 四、设定细节

### 第二轮专项考试规则

| 维度 | 内容 |
|------|------|
| **时间** | 10月15日（第一轮结束后两个月） |
| **地点** | 同第一轮考场 |
| **赛道** | ①道心 ②法力 ③体育 ④武功 |
| **报名方式** | 每人报两个赛道（第一志愿+第二志愿） |
| **调整机制** | 正神根据第一轮总分进行调整 |
| **调整目的** | ①让四赛道人数均衡 ②避免强者挤在同一赛道 ③最大程度挖掘人才 |
| **限制** | 报名时考生之间禁止交流 |

### 正神分组逻辑

```
第一梯队（绝对核心）:
  夜凌霄 → 体育组（锁定胜者）
  云景   → 武道组（锁定胜者）

第二梯队（高分选手）:
  尽量不与夜凌霄、云景撞车 → 分入道心/法力组 → 保全晋级机会

第三梯队（低分/可牺牲选手）:
  张羽 → 体育组（vs夜凌霄）——牺牲
  白真真 → 武道组（vs云景）——牺牲
  理由: "今年被淘汰了明年还能再考"
```

### 张羽的羽书认知变化

| 阶段 | 认知 | 说明 |
|------|------|------|
| **最初** | 羽书=外挂 | 张羽认为羽书是自己的外挂 |
| **后来** | 羽书=潜力 | 多次靠自己努力提升实力后觉得羽书只是潜力 |
| **现在** | 张羽=羽书的外挂 | 需要自己去狩猎邪神、给羽书升级 |

### 回程细节

| 维度 | 内容 |
|------|------|
| **传送点** | 车站厕所 |
| **行动** | 不敢在仙都多逗留 → 立刻买车票离开 |
| **列车上** | 张羽刚放松→仪式倒逼→走廊练春秋无尽禅 |
| **白真真** | 疲惫→理解张羽→收到催款电话（下月两万多利息） |
