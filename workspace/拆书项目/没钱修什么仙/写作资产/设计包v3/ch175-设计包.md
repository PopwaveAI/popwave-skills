# 第175章 张羽辛苦挣钱 — 设计包 (v3)

---

## 一、核心概念

> **张羽卖种子收入15万 + 黑龙董事后续计划 + 玉星寒的祸根抉择**
> 黑龙董事继续阐述红塔转型逻辑——批量生产调优后代将成为豪族繁衍趋势。张羽再次拒绝义子/借种提议后以"张翩翩合同"为由脱身。龙人虽放行但眼神变冷，命科研组抓紧研究张羽种子——如果真有奥秘"再想办法"。张羽回家后向白真真炫耀15万收入，表示下周可再卖一次。白真真连声叫"爹"给张羽按摩找吃的。张羽意识到钱不长久，决定联系玉星寒去灵界问道堂。玉星寒正被祸根困扰，向星火真人求教后面对两难抉择——留下祸根磨炼意志 vs 祛除祸根逃避问题。

**本章使命：**
1. **龙人世界观延续**：红塔转型的逻辑闭环 + 对张羽起疑的伏笔
2. **启动资金到位**：15万收入 + 下周还可再卖 → 张羽攒够登录问道堂的钱
3. **张羽的紧迫感**：意识到钱不长久 → 下定决心推进狩猎邪神计划
4. **玉星寒的抉择**：祸根的两种处理路径——强者之路 vs 逃避之路
5. **两条线交汇**：张羽联系玉星寒要去问道堂 → 玉星寒产生新想法

## 二、情节框架

### 主视角（张羽）+ 副视角（黑龙董事/玉星寒）

| **张羽的拒绝与脱身(B段，17-35行)** | 拒义子→拒借种→拒变男变女→以张翩翩合同为挡箭牌→只签研究权 | 张羽的警觉性 + 合同挡箭牌 |

### 关键节点

1. **龙人对未来的判断**（第1-11行）
   - "强大的家族将被更强大的家族逐渐吞并"——昆墟一层未来的趋势
   - 豪族竞争越来越激烈 → 批量生产调优后代成为繁衍模式
   - "不断结合血脉中的优势，孕育出一代更比一代强的后裔"
   - 红塔优势：快速占据市场
   - 红塔弱点：过去过于沉浸在妖兽妖裔市场 → 缺乏优秀的人类种子
   - 张羽 = 未来的拳头产品
   - **商业计划：** 提炼血脉优势要素 → 直接用种子改善血脉 → 引起富人兴趣 → 签独家协议

2. **张羽的三连拒**（第12-34行）
   - 拒做义子："我已经签了张翩翩了"
   - 拒借种："这也违反合同的"
   - 拒变男变女："真不行"
   - 龙人试探：从"张翩翩管这么宽？" → 想起古代奴隶制度 → "你也是辛苦了"
   - 最后同意只签研究权
   - **张羽的核心武器：** "张翩翩的合同"成为万能挡箭牌

3. **龙人的后续计划**（第36-71行）
   - 龙人外表和善怜悯 → 实际上只是假象
   - 张羽离开后龙人眼神泛冷
   - 狐妖搂住龙人询问
   - **龙人分析：** 原本只当张羽是普通素材，但"被合同限制得这么死"反而让他更好奇
   - 两个疑问：①张羽说的是真话还是假话？②血脉里是否有秘密？
   - **结论：** "这小子看来都不是那种随便一点诱惑就能哄骗的穷鬼"
   - **指令：** 让科研组抓紧研究种子 → 如果真有奥秘 → "再想办法"
   - **龙人的真实态度：** 放行只是表面，暗地里已经盯上张羽

4. **15万收入**（第73-88行）
   - 张羽回家：雄赳赳气昂昂 → 但有些虚弱
   - "这次我准备不足，量有点少，卖了十五万"
   - "下周如果他们还要的话，那就还可以卖一次"
   - 白真真：一脸羡慕 → "太赚了吧？" → 恨自己不能像张羽一样轻松赚钱
   - 张羽："钱的事情不用着急，有不够的地方你先问我借吧"
   - 白真真：立刻叫"爹！" → 按摩+找吃的
   - **张羽清醒认知：** "这钱可不长久" → "还是得快点提升潜力"
   - 立刻联系玉星寒预约问道堂

5. **玉星寒的祸根抉择**（第91-105行）
   - 白龙高中静室内 → 盘膝吐纳 → 被李雪莲身影困扰
   - "这样下去不行啊"
   - "这祸根的危害太大了"
   - "但若是就这么祛除的话，我又如何修行天罡气功？"
   - 联系星火真人 → 以"我有个朋友（张羽）"为借口
   - **星火真人给出的两个选择：**
     - **选择A（强者之路）：** 留下祸根→磨炼意志→克服欲念→继续天罡气功（但会拖延成绩提升）
     - **选择B（逃避之路）：** 祛除祸根→暂时不修行天罡气功（不影响成绩提升，但未来再练可能更难熬）
   - 两条路各有优劣 → 玉星寒难以抉择
   - 看到张羽来电 → 心中突然涌出一个想法

## 四、设定细节

### 龙人的红塔转型逻辑

```
前提:
  昆墟1层 → 豪族垄断深入方方面面
  绿洲集团体外培养技术 → 红塔畜牧/繁育市场缩水

判断:
  未来的世界 → 没有平民的位置
  未来的平民 → 只来自豪族分支的破落户
  豪族 = 最大的市场 → 后代血脉改造越来越重要

红塔的优势:
  批量生产调优后代的模式
  快速占据豪族血脉市场

红塔的弱点:
  过去过于沉浸在妖兽、妖裔市场
  缺乏足够多、足够优秀的人类种子
  许多豪族的优秀基因不会轻易泄漏

张羽的价值:
  高一天才 + 竞赛获奖 + 高二前达到高三水平
  提炼血脉优势要素 → 直接用种子改善血脉
  引起富人兴趣 → 独家协议 → 更高收益
```

### 张羽的三连拒时间线

| 回合 | 龙人提议 | 张羽回应 | 龙人反应 |
|------|---------|---------|---------|
| 1 | 做义子 + 生育总监（给权力） | "签了张翩翩了，限于合同规定" | "张翩翩管这么宽？" |
| 2 | 借种，每完成一个10万（美女环绕） | 艰难摇头 | "这种高中生气血正旺……抗性就更差" → "我是不是太好说话了？"→施压 |
| 3 | 变男变女/非男非女（突破合同边界） | 连连后退→"真不行" | 变回去→"那行吧，研究权就研究权"→眼神泛冷 |

### 张羽的15万收入明细

| 维度 | 内容 |
|------|------|
| **金额** | 15万 |
| **交易类型** | 研究权（非繁育权） |
| **下次** | 下周可再卖一次 |
| **消耗** | 身体感觉虚弱 |
| **用途** | 攒够登录问道堂的10万 + 升级潜力 |

### 星火真人的两个选择

| 维度 | 选择A：强者之路 | 选择B：逃避之路 |
|------|---------------|---------------|
| **行动** | 留下祸根，磨炼意志，克服欲念 | 祛除祸根，暂时不修行天罡气功 |
| **优点** | 可以继续修行天罡气功 | 不影响近期成绩提升 |
| **缺点** | 拖延接下来一段时间成绩提升 | 逃避问题，未来再练时可能更难熬 |
| **本质** | 直面挑战，长期收益 | 回避挑战，短期收益 |

## 五、套路归档

### 套路名称：红塔的转型逻辑·豪族垄断时代的血脉生意

**适用场景：** 展示大企业/大势力对未来趋势的判断，以及主角在这种趋势中的位置

**本章应用：**
- "未来的世界，没有平民的位置"
- 豪族垄断→平民退出→豪族激斗→后代血脉改造越来越重要
- 红塔转型：从妖兽妖裔市场转向豪族血脉市场
- **核心表达：** 在垄断时代，穷人的基因也是一种可被开发的资源
