# 第177章 狩猎开始 — 设计包 (v3)

---

## 一、核心概念

> **狩猎邪神的序幕 + 白真真的选择 + 邪神祸山的躺平哲学**
> 张羽通过卖种子中介费赚到10万存款近30万，开始为狩猎邪神做最后准备。福姬极力劝说让白真真也进行邪神仪式——张羽最终将选择权交给白真真，而白真真毫不犹豫接受了"开挂"的机会。本章同时引入反派邪神阵营：祸山（躺平混日子型邪神）和他的卷王信徒宋虚，展示狩猎目标的画面。

**本章使命：**
1. **张羽的经济状态**：存款接近30万，以中介费的方式完成资本积累
2. **邪神仪式的规则深化**：愿望不可随意消耗——邪神王会感应
3. **白真真的选择**：穷人面对"开挂机会"时的毫不犹豫
4. **反派阵营亮相**：祸山（躺平邪神）+ 宋虚（卷王信徒）的荒诞组合
5. **狩猎准备**：假情报已通过三重傀儡机送过去

| **福姬劝入伙(C段，23-50行)** | 福姬游说让白真真也进行仪式→张羽顾虑仪式愿望无法随意消耗→最终将选择权交给白真真 | 邪神规则的不可逆性 |

### 关键节点

1. **中介费到账与玉星寒的沉迷**（第1-16行）
   - 熊浩瀚确认种子质量好→张羽10万到账
   - 存款近30万
   - 玉星寒迟迟不回消息→一直在卖种子
   - "这小子不会沉迷了吧？我会不会害了他？"
   - 福姬："总比他在哪里花了钱，还要送蛋白质好吧"
   - 玉星寒豪爽：灵界修炼打五折
   - "好了，五折的钱我已经赚到了"
   - **角色证据：** 玉星寒的仙道潜力变得更可怕了

2. **福姬与张羽的辩论**（第23-44行）
   - 福姬积极推销白真真入伙："合则两利，双赢的事情"
   - 张羽的核心顾虑：仪式愿望不可随意消耗
   - 邪神王的感应机制——消极怠工会遭反噬
   - 福姬的论点：没有白真真助力，狩猎邪神失败一样害了她
   - **关键矛盾：** 张羽不想让白真真被束缚，但福姬说的有道理

3. **白真真的毫不犹豫**（第53-71行）
   - "邪神？这东西不是都市传说吗？还真有？"
   - 确认张羽没被控制后→目光火热
   - "你小子果然开了！开了那么久才带我一起开？"
   - "那还用问吗？我恨不得立刻就提升一下我的潜力"
   - 张羽提醒风险（三个愿望），白真真："我愿意"
   - "这是我自己的选择，就算将来出了事情，也是我自己要付出的代价"
   - 福姬："我现在的力量太虚弱，还干不了这些事"
   - **核心表达：** 穷人从来不怕冒险——怕的是连冒险的机会都没有

4. **反派阵营首次亮相**（第72-97行）
   - 破旧公寓→老旧电脑→影蛊传来情报
   - 李泉：50岁被优化→顶尖外卖员→邪神信徒
   - 祸山（熊玩偶）：打游戏中的邪神
     - 与福姬（奋斗型）不同——祸山只想躺平
     - 信徒打工养活他 → 他在家吃喝玩乐
     - "卷上去干嘛呢？最后大头还不都是让邪神王赚去了？"
   - 宋虚通过影蛊传来的情报让祸山知道被追踪
   - 祸山：不想打打杀杀，现在日子挺好
   - **角色设计：** 荒诞的邪神——比社会牛马还躺平

## 三、人物设计

### 张羽 —— 谨慎的狩猎策划者

| 维度 | 分析 |
|------|------|
| **经济状态** | 存款近30万（含10万中介费） |
| **狩猎心态** | 没有足够把握绝不胡乱动手——内心紧张 |
| **对白真真的保护** | 不想让阿真也被仪式束缚→但福姬的论点有道理 |
| **最终决定** | 将选择权交给白真真——尊重对方的自主权 |
| **对玉星寒的观察** | "这小子仙道潜力变得更可怕了" |

### 福姬 —— 积极的合伙人

| 维度 | 分析 |
|------|------|
| **游说策略** | 从张羽的担忧→白真真的利益→狩猎成功率→层层推进 |
| **规则诚实** | 明确告知愿望不可随意消耗——邪神王感应+反噬 |
| **力量状态** | 太虚弱——无法直接为白真真/张羽提升潜力 |
| **立场** | "我们现在的关系是合伙人" |

### 白真真 —— 毫不犹豫的选择者

| 维度 | 分析 |
|------|------|
| **初始反应** | 震惊→兴奋→火热 |
| **对张羽的反应** | "你小子果然开了！开了那么久才带我一起开？"——带着羡慕和埋怨 |
| **决定逻辑** | 提升潜力→通过筑基考试→考上十大→冒风险值得 |
| **关键台词** | "这是我自己的选择，就算将来出了事情，也是我自己要付出的代价" |
| **压力来源** | 筑基考试后感觉背后一股压力在追着她——怎么进步都觉得不够 |

### 祸山 —— 躺平邪神（反派）

| 维度 | 分析 |
|------|------|
| **外形** | 熊玩偶 |
| **性格** | 懒散、躺平、不想卷 |
| **信条** | "卷上去干嘛呢？最后大头还不都是让邪神王赚去了？" |
| **生活方式** | 信徒打工养他 → 他吃喝玩乐打游戏 |
| **信徒策略** | 专门找社会牛马（外卖/快递/工地）→好骗、死了也没人在意 |
| **与福姬的对比** | 福姬想财富自由、修为自由；祸山觉得混在一层蛮好 |
| **对宋虚的态度** | "宋虚这个家伙……太卷了" |
| **当前状态** | 收到情报后纠结——狩猎同事太麻烦，翻车怎么办？ |

### 宋虚（首次被提及）—— 卷王信徒

| 维度 | 分析 |
|------|------|
| **定位** | 祸山的信徒，但反向推动祸山努力 |
| **行为** | 通过影蛊发来情报、追踪另一邪神 |
| **与祸山的关系** | 一个拼命卷，一个只想躺——荒诞组合 |

### 李泉 —— 忠心信徒

## 四、设定细节

### 邪神仪式规则深化

| 规则 | 说明 |
|------|------|
| **潜能提升前提** | 完成契约仪式 |
| **愿望机制** | 契约后需满足邪神的愿望——邪神王定下的铁律 |
| **愿望消耗** | 不可随意消耗所有愿望→会被视作消极怠工→邪神王感应 |
| **反噬** | 福姬和白真真都会遭到仪式反噬 |
| **张羽的担忧** | 抹除仪式影响比他想象中更麻烦 |
| **当前限制** | 福姬力量太虚弱，必须先狩猎另一邪神才能为任何人提升潜力 |

### 蛋白质经济链条（176-177章串联）

```
张羽介绍玉星寒 → 玉星寒卖种子给红塔集团
     ↓                        ↓
张羽得10万中介费      玉星寒得卖种子收入
     ↓                        ↓
存款近30万             灵界修炼给张羽打五折
                             ↓
                        "好了，五折的钱我已经赚到了"
```

### 三重傀儡机情报传递

| 层级 | 说明 |
|------|------|
| **情报来源** | 张羽/福姬通过灵界发出假情报 |
| **传递方式** | 跳转三重傀儡机→反追踪到祸山位置 |
| **安全等级** | 祸山认为"对方应该没发现我们的位置" |
| **实际效果** | 福姬已成功追踪——敌明我暗 |

### 祸山的信徒管理

| 维度 | 内容 |
|------|------|
| **信徒人数** | 8名 |
| **招募方式** | 快递员、外卖员、工地牛马——好骗、死了没人在意 |
| **管理风格** | 极其松散——信徒各干各的 |
| **信徒待遇** | 自愿打工养祸山——祸山在家吃喝玩乐打游戏 |
| **宋虚加入的影响** | 开始被迫改变（但仍在抗拒） |

## 五、套路归档

### 套路名称：开挂的邀请·穷人从不犹豫

**适用场景：** 重要配角面临"开挂"（邪神仪式/系统等）的选择，穷人和富人的不同反应

**本章应用：**
- 白真真确认安全后："那还用问吗？我恨不得立刻就提升一下我的潜力"
- 风险认知：知道要满足三个愿望→但考上十大的诱惑更大
- "这是我自己的选择，就算将来出了事情，也是我自己要付出的代价"
- **核心表达：** 穷人最不怕的就是冒险——因为不冒险也看不到前途

**归档文件：** `写作资产/套路库/穷人从不犹豫·开挂的邀请.md`

---

### 套路名称：邪神仪式的规则不可逆

**适用场景：** 展示邪神契约的真实代价——不是签了就白嫖

**本章应用：**
- 福姬明确告知愿望不可随意消耗
- 邪神王的感应机制——消极怠工会招来反噬
- 张羽想"让她当场完成所有愿望"——被福姬否决
- "仪式就和符咒一样，里面蕴含的许多规则，不是你想改就能改的"
- **核心表达：** 每次开挂都有代价——世上没有免费的午餐

**归档文件：** `写作资产/套路库/邪神仪式的规则不可逆.md`

---

### 套路名称：躺平邪神·混日子型反派

**适用场景：** 引入一个不想努力、只想躺平的反派角色，制造荒诞感

**本章应用：**
- 祸山的熊玩偶一边打游戏一边处理邪神事务
- "卷上去干嘛呢？最后大头还不都是让邪神王赚去了？"
- 签约社会牛马：好骗、死了也没人在意
- "唉，要不这次狩猎同事的事情就算了吧，感觉太麻烦了"
- **核心表达：** 连邪神都在躺平——这世界到底谁在卷？

**归档文件：** `写作资产/套路库/躺平邪神·不想卷的反派.md`

---

### 套路名称：福姬vs祸山·奋斗型与躺平型对比

**适用场景：** 通过两个同类角色（同为邪神）的对比，展现不同的生存哲学

**本章应用：**
- 福姬：一心往上爬，想财富自由、修为自由
- 祸山：混在一层蛮好，不想费力，信徒养着就行
- 福姬主动出击狩猎同事；祸山觉得"挺麻烦的翻车怎么办"
- **核心表达：** 同样是邪神——选择和活法完全不同

**归档文件：** `写作资产/套路库/邪神对比·奋斗vs躺平.md`
