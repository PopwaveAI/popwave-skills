# 第180章 阻击六大强者 — 设计包 (v3)

---

## 一、核心概念

> **砸到最后一刻的紧迫感 + 白真真的神助攻 + 功亏一篑的撤退**
> 祸山躲在暗室中祈祷，张羽在房间里狂砸半小时仍找不到邪神真身——祸山的杂物堆积反而成了天然屏障。与此同时，宋虚六人在郊区被白真真召来的保安拖住，但最终还是靠花钱脱身。白真真用交通拥堵策略进一步拖延。在宋虚即将回到公寓的最后关头，张羽无奈撤退——第一次突击以"没找到邪神"告终。

**本章使命：**
1. **暗室设定释放**：邪神的末日藏身所真的起作用了
2. **白真真的战术价值**：监控+报警+交通策略——合打辅助
3. **张羽的第一次失败**：计划精准但仍有疏漏——暗室这一手没想到
4. **宋虚团队的逃跑能力**：花钱摆平保安——社会牛马的生存智慧
5. **祸山的求生策略**：躲在暗室中祈祷+手机催促+不敢出声

## 二、情节框架

### 四线并行（祸山/张羽/宋虚/白真真）

| 段落 | 内容 | 核心信息 |
|------|------|----------|
| **祸山暗室求生(A段，1-17行)** | 躲进暗室→庆幸有准备→听张羽砸东西→心痛但不敢出声→手机催宋虚 | 暗室的反直觉效果：杂物越多越难找 |
| **张羽狂砸找邪神(B段，18-35行)** | 无相云罡伪装成大日气海（黑色罡气）→狂砸半小时→祸山心在滴血 | 砸垃圾场半小时=什么都没找到 |
| **宋虚被保安拦截(C段，36-78行)** | 宋虚六人到郊区空手→保安突现→收入/快递/外卖气质被压制→花钱摆平 | 社会牛马的阶级困境：保安=公司=不可抵抗 |
| **白真真监控与拖延(D段，79-101行)** | 白真真监控宋虚→认出了宋虚身份（穷人联盟）→报警引保安→交通拥堵拖时间 | 白真真的战术价值全面展示 |
| **张羽被迫撤退(E段，102-120行)** | 张羽看白真真消息→"他们来了！快撤！"→不甘但必须走 | 第一次出击：没找到邪神但削弱了战力 |

### 关键节点

1. **暗室求生**（第1-17行）
   - 祸山躲在暗室→尺寸极小（十几厘米宽，一米高不到）
   - "当初建造这个小暗室的决定……无比庆幸"
   - 张羽进屋翻找→祸山通过缝隙观察
   - 发信息给宋虚→关机静音→不敢有任何动作
   - **空间叙事：** 暗室的狭小+杂物掩护=完美的藏身点

2. **张羽的狂砸战术**（第18-35行）
   - 张羽变陌生面孔+兜帽
   - 福姬提醒：邪神可能是任何形态（玩具/文具/笔/手套/项链）
   - "一个个找肯定来不及了——砸吧！"
   - 张羽发动无相云罡→伪装成大日气海（吸收光线→黑色罡气）→隐藏出手特征
   - 黑色罡气如乌云滚滚→砸毁一切
   - 祸山在暗室中：心痛电子产品/游戏机/电脑/手办/模型……
   - "该死……砸我家来了？欺负邪神不敢报警是吧？！"
   - **关键困境：** 东西太多+时间不够=找不到

3. **宋虚的郊区遭遇**（第38-77行）
   - 飞剑查探→没有发现→六人从不同位置进入烂尾楼
   - "感觉有点不对劲，先撤出去吧"
   - 保安突现→"你们六个！大晚上过来偷东西是吧？"
   - 万里集团的保安→公司财产→背后有公司
   - 宋虚六人反应：不敢打保安（公司=诉讼=上天入地逃不掉）
   - 外卖/快递/工地气质→看到保安第一反应是气虚
   - 保安趾高气扬→"这几个家伙一看就满身快递外卖工地的气质"
   - 宋虚：先是想好好解释→发现手机震动（祸山求救）→心态转变
   - "邪神内斗，竟然还玩举报？还找捕快？！对方的邪神太不讲规矩了！"
   - 最终花钱摆平保安→但已被拖延
   - **阶级讽刺：** 链气巅峰的强者被两个保安压制——因为保安=公司

4. **白真真的神操作**（第79-101行）
   - 监控拍照→认出了宋虚
   - "是他！"→半年多前调查穷人联盟时筛选出的嫌疑人
   - 白真真手机里存着宋虚的照片和资料
   - **关键信息：** 白真真的情报能力——早就在调查穷人联盟
   - 报警引万里集团保安→"你们效率很快啊"
   - 打开导航→烂尾楼到公寓的路一片赤红
   - "现在正是午夜的上下班高峰，过来的路交通拥堵"
   - 估算：能再拖延一阵
   - **战术价值：** 白真真不仅知道宋虚的身份，还懂交通策略

5. **张羽的撤退**（第102-120行）
   - 看着被砸成垃圾堆的房间→一脸不甘
   - 白真真：他们来了！快撤！速度太快了！
   - 张羽冷哼一声→带着不甘离开
   - 功亏一篑——就差一步没找到暗室
   - **悬念：** 祸山还在暗室里→宋虚即将回援→下章会怎样？

## 三、人物设计

### 祸山 —— 绝境中的等待者

| 维度 | 分析 |
|------|------|
| **暗室状态** | 躲在狭小空间→不敢出声→通过缝隙观察 |
| **心态** | 庆幸→心痛（东西被砸）→害怕被发现→催宋虚 |
| **通讯** | 发消息催宋虚→关静音→不敢多动 |
| **最庆幸的事** | "谁特幺说我乱买东西了？要不是我买了这么多东西，哪能拖延这么久？" |
| **当前处境** | 孤家寡人——两名忠实信徒被抓走，尚未被找到但岌岌可危 |

### 张羽 —— 功亏一篑的突击者

| 维度 | 分析 |
|------|------|
| **进攻方式** | 伪装形态+大日气海伪装无相云罡→黑色罡气狂砸 |
| **困境** | 东西太多→半小时砸不完→找不到邪神 |
| **心态变化** | 从果断到焦躁到不甘 |
| **撤退原因** | 白真真预警→宋虚即将回援→不走会被六人包夹 |
| **收获** | 至少废掉了两名信徒→削弱了祸山的保护力量 |
| **遗憾** | 就差一步没找到暗室——下次再来就没这么容易了 |

### 白真真 —— 战术辅助MVP

| 维度 | 分析 |
|------|------|
| **监控操作** | 提前在烂尾楼安装监控摄像头+电池+网卡（花了好几千） |
| **身份识别** | 一眼认出宋虚——半年多前调查穷人联盟的成果 |
| **报警策略** | 举报烂尾楼有人偷建筑材料→引来万里集团保安 |
| **交通策略** | 选择午夜的上下班高峰→打开导航确认拥堵→拖延宋虚回援时间 |
| **预警** | 实时监控+快速通知张羽撤退 |
| **认知** | "还好昨天放的监控摄像头能用"——提前一天的准备 |

### 宋虚 —— 被拖住的回援者

| 维度 | 分析 |
|------|------|
| **郊区状态** | 发现空手→直觉不对劲→想撤 |
| **保安遭遇** | 被两个保安拦下→气虚（职业病）→先解释→收到祸山求救→怒了 |
| **转折** | "不能拖延了"→花钱摆平 |
| **对邪神内斗的反应** | "太不讲规矩了！竟然玩举报还找捕快！" |
| **性格连续** | 先忍不想花钱→发现不得不走→花钱搞定→延续第178章的克制+务实 |

## 四、设定细节

### 暗室/夹层空间（细节补充）

| 维度 | 内容 |
|------|------|
| **开门方式** | 沙发后墙上的暗门 |
| **内部尺寸** | 十几厘米宽，一米高不到（刚好塞一个熊玩偶） |
| **观察方式** | 留了一道缝隙→能看外面但外面不容易发现 |
| **掩护** | 房间里的各种杂物堆积把暗室位置遮挡得更隐蔽 |
| **通讯** | 可发消息（发完关机+静音） |

### 张羽的伪装攻击手段

| 步骤 | 内容 |
|------|------|
| 1 | 变成另一副陌生面孔（筋肉扭曲） |
| 2 | 穿上兜帽遮掩 |
| 3 | 为隐藏功法特征→先用大日气海吸收光线→使无相云罡变黑 |
| 4 | 发动黑色无相云罡→如乌云滚滚→轰击所有物品 |
| 5 | 目的：找到邪神本体（可能是任何形态的物件） |

### 白真真的监控网

| 维度 | 内容 |
|------|------|
| **设备** | 监控摄像头+电池+网卡 |
| **花费** | 好几千块 |
| **安装时间** | 179-180章前一天 |
| **安装地点** | 烂尾楼内 |
| **功能** | 实时画面传输→白真真手机监控 |
| **用途** | 监视宋虚六人动向→预判回援时间 |
| **附加价值** | 拍到宋虚正脸→确认身份（穷人联盟老大） |

### 午夜上下班高峰（交通策略）

| 维度 | 内容 |
|------|------|
| **时间** | 午夜（所谓的"上下班高峰"——可能是修仙世界的特殊作息） |
| **道路** | 烂尾楼到祸山公寓→导航显示一片赤红（严重拥堵） |
| **预计拖延** | 正常车程→因拥堵至少翻倍 |
| **策略来源** | 张羽和白真真提前计算好的 |
| **效果** | 为张羽争取了更多搜索时间 |

### 社会牛马对公司的敬畏

| 维度 | 内容 |
|------|------|
| **宋虚六人的反应** | 看到保安第一反应是气虚（职业病）→不敢还手→先解释→再花钱 |
| **保安的底气** | "别看对方有六个人，他背后可是有着整个公司" |
| **核心逻辑** | 公司=诉讼=上天入地逃不掉→比邪神可怕一万倍 |
| **阶级讽刺** | 链气巅峰的邪神信徒→被两个保安死死压住 |
| **唯一解法** | 花钱——宋虚掏几千块钱搞定 |

## 五、套路归档

### 套路名称：功亏一篑·第一次突击的遗憾

**适用场景：** 主角精心策划的第一次攻击，效果不错但核心目标没达成——制造悬念

**本章应用：**
- 张羽成功清除了两名信徒→但没找到邪神本体
- 狂砸半小时但错过的暗室
- 不甘但必须撤退
- **核心表达：** 完美的计划也会遇到预想不到的变数——下一次就没这么容易了

### 套路名称：白真真的战术价值·监控+情报+交通

**本章应用：**

### 套路名称：公司保安·链气巅峰也怕

**适用场景：** 展示阶级社会中"公司"的绝对权威——保安=公司=不可抵抗

**本章应用：**
- 六名链气巅峰的邪神信徒→被两个保安压制
- "这几个家伙一看就满身快递外卖工地的气质"
- 不敢打→只能用钱摆平→先忍后掏钱
- 公司诉讼比邪神战斗可怕一万倍
- **核心表达：** 在昆墟层面——公司比武力更强大

**归档文件：** `写作资产/套路库/公司保安·权限碾压武力.md`

### 套路名称：双向拖延·时间赛跑

**适用场景：** 主角和反派同时被时间压力推着走——看谁先到位

**本章应用：**
- 张羽：必须在宋虚回援前找到祸山
- 祸山：必须在张羽找不到之前撑到宋虚回来
- 白真真：用保安+交通拖住宋虚
- 三条时间线同时推进
- **核心表达：** 狩猎的本质是时间竞争——晚一秒钟都可能失败

**归档文件：** `写作资产/套路库/双向拖延·时间赛跑.md`
