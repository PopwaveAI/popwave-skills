# 第181章 狩猎邪神成功 — 设计包 (v3)

---

## 一、核心概念

> **张羽以信息差智胜六名邪神信徒，祸山被福姬镇压吞并**
> 张羽用赤髓浑元气伪装成宋虚混入公寓，以福姬教授的物理震荡法镇压祸山（熊玩偶邪神），随后以信息差戏耍宋虚等六名邪神信徒，最终六人因仪式反噬自燃而死。本章是**邪神线第一阶段收官 + 张羽信息战能力的集中展示**。

**本章使命：**
1. **邪神狩猎成功**：祸山被福姬镇压，为后续福姬恢复位格提供基础
2. **信息战教学**：张羽利用信息差——伪装身份、伪装李泉、假跳窗——全歼六名高手
3. **仪式反噬机制揭示**：邪神信徒泄露身份后的自燃机制再次上演（呼应周天翊之死）
4. **福姬的崛起起点**：吞并祸山后即将恢复位格，福姬从弱鸡开始变有用

---

## 二、情节框架

### 单一主视角（张羽）+ 倒计时紧张感

| 段落 | 内容 | 核心信息 |
|------|------|----------|
| **宋虚赶路(A段，1-9行)** | 宋虚六人以外卖/快递速度赶往公寓 | 邪神信徒的人才结构：外卖员、快递员等底层职业 |
| **伪装进场(B段，10-25行)** | 张羽以赤髓浑元气伪装成宋虚 → 祸山误认 → 张羽抓住祸山 | 赤髓浑元气的伪装功能、邪神本体脆弱 |
| **镇压祸山(C段，26-37行)** | 福姬用物理震荡法搅乱祸山意识 → 暂时镇压 → 搜索仪式记录 | 邪神本体弱+感知力可被震荡干扰 |
| **六人赶到(D段，38-52行)** | 张羽伪装成李泉，责备信徒来得慢 → 宋虚起疑 | 伪装需要改变脸型+熟悉目标信息 |
| **信息战决胜(E段，53-97行)** | 宋虚识破 → 六人欲拼命 → 张羽三连骗：熊玩偶挡刀→无相云罡遮眼→板凳假跳→隐入阴影 | 信息差才是最强武器 |

### 关键节点

1. **张羽伪装进场**（第1-25行）
   - 宋虚六人由外卖员/快递员背负，高速赶路
   - 祸山紧张等待，以为宋虚已到
   - 张羽以赤髓浑元气改变脸型伪装成宋虚
   - 祸山情急之下未识别 → 被张羽一把抓起
   - **设定揭示：** 邪神本体并不强大，仪式中神通广大但本体脆弱

2. **福姬镇压祸山**（第26-37行）
   - 物理震荡法：强烈震荡通过邪神感知力搅乱意识（像晕车）
   - 福姬化为黑珠按在熊玩偶头上 → 暂时镇压
   - 福姬搜索祸山的仪式记录
   - 白真真发消息提醒：\"他们已经上去了！\"

3. **张羽伪装李泉**（第38-52行）
   - 张羽快速切换伪装成李泉
   - 责备宋虚六人来得太晚，让他们跪下认错
   - 六人被熊玩偶的冷哼震慑，纷纷跪下
   - 宋虚越看越不对劲 → 最终识破

4. **信息战三连击**（第53-97行）
   - **第一步：** 以熊玩偶挡刀，逼六人变招（不敢伤邪神）
   - **第二步：** 无相云罡遮蔽视线（六人联手半秒轰散）
   - **第三步：** 板凳砸窗假跳 → 自己隐入大日气海的阴影中
   - 六人跳窗追击 → 已构成\"泄露身份\" → 仪式反噬自燃
   - \"信息差啊信息差。\"

5. **善后**（第95-99行）
   - 张羽捡熊玩偶，转身离开
   - 六道火人化为灰烬
   - 捕快 + 巡查队封锁现场
   - 张羽和白真真回收郊区烂尾楼监控

## 三、人物设计

### 张羽 —— 信息战大师

| 维度 | 分析 |
|------|------|
| **核心策略** | 信息差碾压——用对手所知以外的东西取胜 |
| **伪装能力** | 赤髓浑元气改变脸型+熟悉宋虚资料→成功冒充宋虚→又冒充李泉 |
| **三连骗战术** | ①熊玩偶挡刀(规则压制) ②无相云罡遮眼 ③板凳假跳+阴影隐身 |
| **冷静程度** | 面对六名链气巅峰的搏命攻击，淡然微笑应对 |
| **对邪神的认识** | \"果然是智障仪式\"——理解仪式反噬机制后更加忌惮 |
| **关键成长** | 从被动应付邪神到主动狩猎邪神 |

### 祸山 —— 被狩猎的邪神

| 维度 | 分析 |
|------|------|
| **位格** | 1级下位邪神（与福姬同级） |
| **现状** | 被张羽物理震荡镇压，被福姬吞并中 |
| **仪式设置** | 维持8名信徒的愿望仪式+公寓屏蔽仪式+影蛊监控→已到极限 |
| **临死行为** | 删除所有记录和数据，避免被福姬获取 |
| **结局** | 熊玩偶粉碎，意识被福姬吞噬 |

### 宋虚 —— 穷盟老大 / 穷途末路的邪神信徒

| 维度 | 分析 |
|------|------|
| **赶路阵容** | 两名外卖员+一名快递员背负三人高速前进 |
| **警觉性** | 第一个发现\"李泉\"不对劲 |
| **搏命意志** | 知道泄露身份后，想杀张羽挽回（\"在反噬前杀了他\"） |
| **结局** | 六人齐齐跳窗追击 → 自燃化灰烬 |
| **信息劣势** | 不知道张羽有福姬、无相云罡、大日气海等底牌 |

### 其他五名信徒

| 维度 | 分析 |
|------|------|
| **职业** | 外卖员、快递员等底层职业 |
| **实力** | 全部链气巅峰 |
| **仪式束缚** | 被祸山定下完整仪式：不能伤害祸山、不能透露身份 |
| **结局** | 全数因仪式反噬自燃而亡 |

## 四、设定细节

### 邪神本体脆弱

| 设定元素 | 说明 |
|---------|------|
| **仪式中** | 神通广大 |
| **本体** | 并不强大，不然无法躲避正神探测 |
| **物理攻击** | 能对邪神造成伤害 |
| **震荡干扰** | 通过邪神的感知力搅乱意识（像晕车） |
| **福姬操作** | ①物理打击震荡 ②黑珠按头→镇压 ③搜索仪式记录 |

### 仪式反噬机制

| 设定元素 | 说明 |
|---------|------|
| **触发条件** | 主动或被动泄露邪神信徒身份 |
| **反噬方式** | 体内灼烧，自燃而死 |
| **挽回可能** | \"在反噬前杀了对方就不算泄露\"——有时间窗口 |
| **案例** | 周天翊自燃（前文） → 宋虚六人自燃（本章） |
| **智障仪式的逻辑** | 信息泄露本身已经构成威胁，自燃是防止正神追查的最后手段 |

### 张羽的信息战三连

| 步骤 | 动作 | 目的 |
|------|------|------|
| **①规则压制** | 熊玩偶挡刀 | 六人不敢伤邪神→变招→气血翻涌 |
| **②视觉遮蔽** | 无相云罡吞没全场 | 争取操作空间 |
| **③假跳真藏** | 板凳砸窗+大日气海阴影隐身 | 制造跳窗假象，让六人追出→触发反噬 |

### 善后处理

| 设定元素 | 说明 |
|---------|------|
| **监控回收** | 张羽和白真真回收郊区烂尾楼的监控 |
| **现场** | 大批捕快封锁，巡查队介入调查 |
| **线索** | 宋虚等人焚化后的痕迹 |

## 五、套路归档

### 套路名称：伪装身份的突袭

**适用场景：** 主角用变形/易容能力伪装成敌人信任的目标，实现近身突袭

**本章应用：**
- 张羽用赤髓浑元气伪装成宋虚进入公寓
- 祸山情急之下误认→被张羽一把抓住
- 后又伪装成李泉，训斥宋虚六人
- \"信息差啊信息差\"

**归档文件：** `写作资产/套路库/伪装突袭·信息差碾压.md`

---

### 套路名称：规则反噬杀敌

**适用场景：** 利用对手自身的规则/仪式/契约反噬杀死对方

**本章应用：**
- 六名信徒被定下\"不能泄露身份\"的仪式
- 张羽故意让六人跳窗追击→构成泄露身份→自燃
- 呼应前文周天翊自燃的伏笔
- 规则杀人比硬碰硬更高效

**归档文件：** `写作资产/套路库/规则反噬·契约杀人.md`

---

### 套路名称：信息差碾压高手

**适用场景：** 实力悬殊但主角利用信息不对等碾压强者

**本章应用：**
- 宋虚六人全是链气巅峰，张羽一人难敌
- 但张羽知道他们有仪式束缚，而对方不知道他有福姬
- 三连骗战术：规则压制→视觉遮蔽→假跳真藏
- 六人至死都不知道张羽是怎么操作的

**归档文件：** `写作资产/套路库/信息差碾压·情报就是力量.md`

---

### 套路名称：邪神内斗·吞并晋级

**适用场景：** 一个邪神吞噬另一个邪神，提升自身位格的设定

**本章应用：**
- 福姬镇压祸山后准备吞并其邪神位格
- \"吃了他，恢复位格\"
- 为后续福姬恢复1级邪神、提供仪式能力做铺垫
- 邪神间的狩猎关系

**归档文件：** `写作资产/套路库/邪神内斗·位格吞噬.md`
