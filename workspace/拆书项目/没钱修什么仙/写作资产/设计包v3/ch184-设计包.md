# 第184章 被问询的张羽 — 设计包 (v3)

---

## 一、核心概念

> **张羽的巡查问询攻防战 + 云霓对邪神事件的判断**
> 张羽按照福姬的事前预案，以\"不认识宋虚\"的回答应对云霓的问询，成功通过第一轮审查。但云霓虽然表面放过张羽，内心对邪神事件有自己的判断（怀疑是资深邪神信徒或邪神内斗）。巡查队员私下查张羽购古董的记录被云霓警告终止——涉及筑基考试的保密层级。本章是**张羽的潜伏危机 + 云霓线索心理战 + 巡查队内部权力格局**的三层交织。

**本章使命：**
1. **张羽的问询攻防**：福姬事前预案+张羽的表演→成功规避风险
2. **云霓的内心判断**：认为可能是资深邪神信徒（或邪神内斗）干的
3. **巡查队内部格局**：巡查队=装修精美的房子，大多数墙是假的
4. **张翩翩的线索**：正式员工身份→可跨层联系→但束缚太多
5. **筑基考试的保密层级**：云霓警告手下不要查张羽离开嵩阳的事

## 二、情节框架

### 三层结构：问询面谈篇 + 云霓视角篇 + 巡查队内部篇

| 段落 | 内容 | 核心信息 |
|------|------|----------|
| **福姬预案(A段，1-7行)** | 福姬提前指导张羽如何应对问询 | 事前准备的重要性 |
| **张羽的面谈(B段，8-39行)** | \"不认识宋虚\"→如实说墨天逸的接触→反问出了什么事 | 演技过关 |
| **云霓的内心判断(C段，40-52行)** | 线索全断→认为是资深邪神信徒或邪神内斗 | 云霓的真正判断 |
| **张翩翩的线索(D段，53-61行)** | 正式员工→跨层联系→张羽婉拒 | 姐姐的通道和束缚 |
| **巡查队内部(E段，62-92行)** | 队员查张羽购古董→云霓制止→警告不要查云嵩阳行踪 | 筑基考试保密层级 |

### 关键节点

1. **福姬的事前预案**（第1-7行）
   - 没见过宋虚
   - 公寓监控坏了、张羽和白真真伪装了外貌
   - 邪神信徒保密意识强，个人物品无邪神信息
   - 张羽早有准备——\"卖种子前早准备好容器和纸巾\"的类比

2. **张羽的面谈表演**（第8-39行）
   - 一脸茫然：\"我知道他是学霸，但跟他不认识\"
   - 云霓共享信息：宋虚建立了穷人天才组织
   - 云霓的判断：\"越穷的人越容易被有钱人骗\"→穷人组织=诈骗案件
   - 张羽如实交代和墨天逸的接触
   - 反问：\"他们出了什么事情吗？\"——一脸好奇
   - **结果：** 云霓失望——\"没什么有用的情报啊\"

3. **云霓的内心推理**（第40-52行）
   - 现场有价值的东西都被破坏了
   - 公寓内监控都有问题
   - 周围街道监控没有嫌疑人
   - 外卖员用偷来的手机点的外卖
   - **核心判断：** \"是经验丰富手段利落的邪神信徒干的……或者就有资深邪神在背后指挥\"
   - **结论：** 可能是邪神内斗
   - **关联：** 一年前嵩阳高中出现过邪神痕迹（当时没追到）

4. **张翩翩的线索（第53-61行）**
   - 云霓透露：正式员工身份→可跨层联系
   - 正神监控体系：八部正神掌控各层联系
   - 张羽心动但婉拒
   - 福姬提醒：\"这条路束缚太多，你身上这么多秘密还是算了吧\"

5. **巡查队内部冲突（第62-92行）**
   - 年轻队员查张羽：购买十几万古董
   - 队员暗示：古董可能用来拜神（暗指邪神）
   - 云霓的反击：\"谁让你查他的？不要在我手下自作主张\"
   - **云霓的逻辑：** 张翩翩被正神重点关注→全家是审核对象→不可能和邪神有关
   - **警告：** 不要查张羽离开嵩阳的事情
   - **暗示：** 离开嵩阳和筑基考试有关→保密层级极高→\"捅了马蜂窝\"

### 张羽 —— 守口如瓶的问询高手

## 四、设定细节

### 福姬的事前预案

| 要点 | 逻辑 |
|------|------|
| **没见过宋虚** | 公寓监控坏+张羽伪装外貌→无法证明见过 |
| **真面目未暴露** | 附近街区无真面目监控画面 |
| **邪神信徒的保密** | 个人物品里绝不保留邪神相关信息 |
| **墨天逸接触** | 如实说→巡查队不在乎穷人联盟这种组织 |
| **撒谎原则** | \"在这方面撒谎反而容易引起怀疑\" |

### 云霓的推理链

| 证据 | 推断 |
|------|------|
| 现场被破坏 | 作案者经验丰富 |
| 监控全坏 | 有技术手段 |
| 外卖员用偷来的手机 | 规避追踪很熟练 |
| 六人自燃 | 可能是仪式反噬 |
| **核心结论** | 资深邪神信徒或资深邪神在背后指挥→邪神内斗 |
| **关联老案** | 一年前嵩阳高中出现过邪神痕迹 |

### 跨层联系机制

| 条件 | 说明 |
|------|------|
| **正式员工** | 八部正神下辖的正式员工可跨层联系 |
| **跨层公司贸易** | 相关人员也可跨层联系 |
| **普通人** | 不能轻易跨越层级联系 |
| **张翩翩的状态** | 被正神重点关注的全家审核对象 |
| **张羽的选择** | 婉拒（福姬：束缚太多，秘密太多） |

### 巡查队内部权力结构

| 比喻 | 说明 |
|------|------|
| **装修精美的房子** | 巡查队表面好 |
| **承重墙（两三面）** | 真正掌权的核心 |
| **非承重墙** | 大部分职位只是摆设 |
| **砸掉重装** | 付出代价但可以做 |
| **核心墙** | 云霓自己的判断和底线（如筑基考试保密） |

### 筑基考试的保密层级

| 维度 | 说明 |
|------|------|
| **云霓的警告** | \"谁要是不想干了就查去吧\" |
| **捅的后果** | \"嵩阳市都没人保得住你们\" |
| **逻辑** | 张羽离开嵩阳=参加筑基考试→保密层级极高 |
| **云霓的猜测** | 张翩翩指点弟弟参加筑基考试 |

## 五、套路归档

### 套路名称：问询攻防战——事前预案的艺术

**适用场景：** 主角被官方盘问，通过事前准备成功规避风险

**本章应用：**
- 福姬的事前预案：没见过、伪装了、没证据
- 选择性透露：如实交代墨天逸的接触（避免因撒谎暴露）
- 一脸茫然+好奇反问：扮演无辜者
- 和卖种子前的准备类比——\"早有准备\"

**归档文件：** `写作资产/套路库/问询攻防·事前预案.md`

---

### 套路名称：巡查队的嗅觉——邪神猎人的直觉

**适用场景：** 经验丰富的老手从非直接证据中闻到真相

**本章应用：**
- 现场被破坏→作案者经验丰富
- 监控全坏→有技术手段
- 线索全断→\"越是这样，越觉得有问题\"
- 关联一年前的嵩阳高中邪神事件
- 知道不能打草惊蛇→表面放过，内有盘算

**归档文件：** `写作资产/套路库/老手的直觉·邪神嗅觉.md`

---

### 套路名称：权力的椅子——巡查队的真墙假墙

**适用场景：** 展示一个组织中真正的权力结构和虚假权力的对比

**本章应用：**
- 装修精美的房子=巡查队
- 承重墙=真正的权力核心
- 大部分墙是假的=大部分职位是摆设
- 年轻队员七代巡察人→不能随便处置但也不能随便行动
- \"砸了重装要花不少钱\"

**归档文件：** `写作资产/套路库/巡查队的权力结构.md`

---

### 套路名称：保护伞的边界——张翩翩的影响力

**适用场景：** 主角因某种关系得到大佬的庇护，但庇护有边界

**本章应用：**
- 云霓因为张翩翩的面子照顾张羽
- 但也因此不会怀疑张羽（相信张翩翩的审核）
- 被正神重点关注的全家→反而不可能和邪神有关
- 但庇护有边界：\"想进步可以找我，借钱没有\"
- 正式员工身份可跨层联系但束缚太多

**归档文件：** `写作资产/套路库/保护伞的边界.md`
