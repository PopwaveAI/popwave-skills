# 第185章 真真潜力，仪式上身 — 设计包 (v3)

---

## 一、核心概念

> **白真真潜能觉醒（雷属性特种法力）+ 张羽和白真真获得两大仪式武装**
> 在烂尾楼中，张羽和白真真完成三场仪式：白真真的愿望仪式（激发潜能）、侦测仪式（看穿摄像头）、反检测仪式（隐藏数据）。白真真觉醒雷属性特种法力——充电就能变强。张羽获得30米内透视摄像头的眼力和隐藏实力的反检测能力。两人终于拥有了对抗监控社会的基础装备。

**本章使命：**
1. **白真真潜力觉醒**：雷属性特种法力的完整展示——充电修炼的搞笑亮点
2. **仪式系统正式应用**：侦测仪式+反检测仪式的实际测试
3. **仪式机制补充**：仪式的使用限制（每天时限）+恢复机制（从神力网络窃取）
4. **福姬的后续规划**：还剩4个仪式空位→需要继续提升位格
5. **张羽的春秋无尽禅进度**：9级→即将完成10级目标

### 关键节点

1. **张羽的紧迫感**（第1-11行）
   - 春秋无尽禅8级（36/80）
   - 云霓的关注带来压力——\"巡查队是否在查邪神的事情\"
   - \"掌控信息，掌握信息差……在昆墟往上爬所必备的技能\"
   - →加速仪式准备速度

2. **仪式完整流程**（第12-25行）
   - 烂尾楼房间内布置现场
   - 红烛插满房间、破布娃娃（福姬本体）在黄铜法坛上
   - 两部手机（人神介质）、气海灵根（灵机神物）、福姬自己（邪神象征）
   - 四件古董置于四个方位（香火愿力）
   - **咒文：** \"太极幽暝，暗影浮沉……香火愿力来！\"
   - 古董碎裂→无形力量涌到福姬身上
   - 暗金色神力浮现→白真真被灌体而入

3. **白真真的雷属性觉醒**（第26-35行）
   - 眼中电光暴涨→周身道道雷光
   - 法力带雷霆属性→特种法力
   - 电光步、雷殛真体效果大大提升
   - 反应、速度、灵敏、爆发全面提升
   - 未来修类似功法事半功倍
   - **亮点：** 充电就能补充法力，充电就有炼体效果！

4. **侦测仪式实测**（第36-48行）
   - 发动后直接看到摄像头位置（透视墙后）
   - 30米内有效范围
   - 普通障碍物无法阻挡
   - **限制：** 几分钟消耗近一半力量
   - **恢复：** 每天从神力网络窃取→每天约10分钟使用时间

5. **反检测仪式实测**（第49-55行）
   - 法力突破90但检测出来只有85
   - 继续发动后最低可达50
   - **限制：** 每天1小时持续时间
   - **效果：** 隐藏法力和肉体强度的检测数据
   - 后续月考不用太招摇了

6. **福姬的长期规划**（第56-67行）
   - 总共能维持10个仪式
   - 扣去三场→还剩4个空位
   - 提升位格→维持更多仪式
   - **两种方式：** ①吞其他邪神 ②窃取正神权柄
   - **终极目标：** \"跳出三贷之外，不在五险之中\"

7. **白真真的偷电计划**（第69-86行）
   - 回到公寓后用侦测仪式检查安全
   - 白真真插手指进插座→浑身电光闪烁
   - 张羽发现电表疯狂转动→赶紧阻止
   - 白真真：\"那我明天去学校里偷电修炼\"

8. **春秋无尽禅突破**（第87-89行）
   - 一夜修炼→9级（24/90）
   - 肉身越发纯净

## 四、设定细节

### 仪式完整流程

| 元素 | 说明 |
|------|------|
| **法坛** | 黄铜法坛，摆放破布娃娃（福姬本体） |
| **人神介质** | 两部开了流量的手机 |
| **灵机神物** | 租来的气海灵根 |
| **邪神象征** | 福姬自己 |
| **祭品（香火愿力）** | 四件古董，置于四个方位 |
| **咒文** | \"太极幽暝，暗影浮沉……香火愿力来！\" |
| **过程** | 古董碎裂→香火愿力涌出→暗金色神力浮现→灌体而入 |

### 侦测仪式设定

| 设定元素 | 说明 |
|---------|------|
| **效果** | 直接看到摄像头位置（透视障碍物） |
| **范围** | 方圆30米 |
| **阻挡** | 普通障碍物无法阻挡 |
| **使用限制** | 一次几分钟消耗近半力量 |
| **恢复机制** | 从神力网络窃取→每天约10分钟 |
| **应用场景** | 检查房间是否被监控、躲避摄像头 |

### 反检测仪式设定

| 设定元素 | 说明 |
|---------|------|
| **效果** | 隐藏法力强度/肉体强度检测数据 |
| **最大隐藏** | 90→最低50单位 |
| **持续时间** | 每天1小时 |
| **应用场景** | 月考藏实力、日常降低存在感 |
| **考试策略** | 福姬可临时关闭除愿望外的仪式→不会被正神发现 |

### 白真真雷属性特种法力

| 设定元素 | 说明 |
|---------|------|
| **觉醒方式** | 福姬的愿望仪式（邪神力灌体） |
| **法力变化** | 法力带雷霆属性→特种法力 |
| **功法加成** | 电光步、雷殛真体效果大大提升 |
| **身体变化** | 反应、速度、灵敏、爆发全面提升 |
| **未来潜力** | 修类似功法事半功倍 |
| **充电修炼** | 手指插插座→充电补充法力+炼体效果 |
| **真灵根适应后** | 效率更高（未来可期） |

### 福姬的仪式容量

| 现状 | 说明 |
|------|------|
| **总容量** | 10个仪式（1级邪神） |
| **已占用** | 愿望仪式×2（张羽+白真真）+ 侦测仪式×2 + 反检测仪式×2 = 6个 |
| **剩余** | 4个仪式空位 |
| **提升方式** | ①吞邪神位格(每升2级增多) ②窃取正神权柄 |
| **终极目标** | \"跳出三贷之外，不在五险之中\" |
