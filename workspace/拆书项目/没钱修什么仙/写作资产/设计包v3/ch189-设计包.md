# 第189章 张羽的时间 — 设计包 (v3)

---

## 一、核心概念

> **压迫升级：放学后4小时符咒补课 + 白真真被大学生击败 + 24小时监控 + 被迫蛰伏**
> 张羽的龙象制剂被换成狂龙制剂后，苏海峰又追加"放学后4小时符咒补课"——进一步压缩张羽的自由时间。白真真试图硬闯学生会讨公道，被周家雇佣的72下校大学生周豪轻松击败。周豪提出100万买张羽的试探被白真真用"先打钱再捅刀"的方式挡回。此后白真真和张羽发现被24小时监控跟踪——无法去暗学帮赚钱、无法去保护区修炼。福姬建议：先忍一忍，专注自身成长，等待机会。

**本章使命：**
1. **压迫再升级**：放学后4小时符咒补课（做卷子）→时间被进一步蚕食
2. **白真真的反击失败**：被周豪（72下校大学生，链气巅峰）轻松碾压
3. **100万试探**：周澈尘派人试探白真真是否愿意出卖张羽
4. **监视启动**：张羽和白真真被24小时摄像头跟踪
5. **福姬的战略建议**：先忍——蛰伏——专注成长——等待破绽
6. **忍的条件的确认**：压迫手段未能真正打击到他们→反而有些可被利用

### 线性叙事：药力分析 → 福姬拆解 → 补课通知 → 白真真反击 → 监视压力 → 福姬建议

| **张羽的判断(D段，37-41行)** | 炼体药被换+问真真和其他人的感受不同→确认是冲自己来的→怀疑周澈尘 | 张羽的警觉 |

| **100万试探(G段，78-85行)** | 周豪问100万是否出卖张羽→白真真"先打钱再捅刀"→先签合同再打钱→白真真离去 | 忠诚试探 |

### 关键节点

1. **王海的旁观哲学补刀**（第1-9行）
   - 继续深化：看不懂的事情不参与→几十年金牌体育老师
   - 对林卉的判断："工作不久的新人要趁机大赚一笔的心态"
   - 对张羽的判断："陷入大势力漩涡→最终被吃的一点不剩"
   - **核心警示：** "上面那些家伙，又有哪个是肯让人白占便宜，不要收益的？"

2. **狂龙制剂的实际效果**（第10-27行）
   - 张羽打了药→"像炸药一样在体内爆炸"→恨不得狂舞三天三夜
   - **张羽的发现：** 药很猛→**药好毒**→药毒药渣比普通药物猛烈十倍
   - 春秋无尽禅10级→能控制→但大庭广众下不暴露→顺势练赤髓浑元气
   - **林卉的内心：** "这可不是普通的龙象制剂，而是狂龙制剂"
   - 林卉的预期：张羽第一次→就会有无数次→沉迷→跪求药剂

3. **福姬的压迫拆解**（第28-36行）
   - 福姬总结：三层压迫——通识课不能旷课、武道课只能学基础、体育课用猛药
   - **精准判断：** "这手段简直一个比一个过分"
   - **明显意图：** "在故意浪费你的时间"
   - 张羽的回应：我有我的安排，夯实根基是好事
   - 福姬不理解张羽的真正目的（可能与潜能有关）

4. **放学后4小时补课**（第42-56行）
   - 苏海峰带中年男子周老师→专门为张羽补课
   - 表面理由：发现张羽道术成绩不行→请老师补习
   - **规则陷阱：** "放学后4小时内按嵩阳市规则也算正常教学时间"
   - **补课内容：** 300张符咒试卷→"做卷百遍，其意自现"
   - 张羽的反应："这是要每天再废我4个小时"
   - 周老师的威慑："你想要违逆学校的正常教学计划，想要吃处分吗？"
   - 张羽只能坐下来做卷子

5. **白真真vs周豪——实力差距展示**（第57-77行）
   - 白真真踹开学生会大门→质问何大有→得知是正会长主意
   - 找到周澈尘练功场→"见你一次打吐你一次"
   - 周澈尘头都不抬→"把她赶走"
   - **周豪登场：** "我叫周豪，现在负责周澈尘的安全工作"
   - **秒杀：** 白真真只看到一道残影→双手交叉格挡→如流星般飞出练功场
   - **自我介绍：** "72下校毕业的大学生，当初学的就是安保专业"
   - **实力对比：** 链气巅峰+十年以上修行→"就凭你高中生的水平，在我面前一点机会都没有"
   - **大学等级知识补充：** 10大名校→36上校→72下校→大专及杂牌
   - 白真真知道即使展现全部实力也不可能是对手→果断离去

6. **100万的试探**（第78-85行）
   - 周豪问："周澈尘让我问你，给你100万的话，你愿意出卖张羽吗？"
   - **白真真的回答：** "这不废话？你直接打钱，我现在就去捅他两刀。"
   - 周豪："先签合同再打钱。"
   - 白真真冷哼一声→头也不回地离去
   - **潜含义：** 在"先打钱"vs"先签合同"的博弈中双方都没有真心

7. **24小时监视启动**（第82-90行）
   - 白真真在小花坛修行时→侦测仪式发现摄像头对准自己
   - 不是学校摄像头的位置→是新的埋伏摄像
   - **推理：** 负责跟踪的很可能也是周豪级别的链气巅峰高手
   - 如果是这样→摆脱非常困难
   - **后果：** 无法去暗学帮赚钱→无法去保护区修炼
   - **压迫感的深化：** "一种比对方过去曾经带来的压迫要更加强烈，更加沉重，更加难以抵御的压力"
   - "就像是一个深不见底的深渊，要将她和张羽彻底吞噬"

## 三、人物设计

### 张羽 —— 被层层束缚仍在暗中修行的忍耐者

| 维度 | 分析 |
|------|------|
| **对狂龙制剂的判断** | 药猛+药毒→但春秋无尽禅10级能控→顺势练其他功 |
| **对压迫的认知** | 确认炼体药被换→是冲自己来的→怀疑周澈尘 |
| **对福姬的回应** | "我有我的安排"——不透露凑高中圣体的计划 |
| **对补课的态度** | 极度不满但没办法→只能坐下做卷子（不想吃处分） |
| **信息差** | 福姬和真真都不清楚他凑基础功法的真正目的 |

### 白真真 —— 从雷霆反击到被碾压的清醒者

|------|------|

| **100万试探** | "先打钱再捅刀"→表面答应但实际是反讽 |

| **内心状态** | "嘴角露出一丝不易察觉的微笑"——对自己的计划很有信心 |

## 四、设定细节

### 嵩阳市的补课规则陷阱

| 设定元素 | 说明 |
|---------|------|
| **规则内容** | 放学后4小时内，按嵩阳市规则也算在高中正常教学时间里 |
| **补课安排** | 学校安排→学生不能旷课→否则吃处分 |
| **表面理由** | 张羽道术（符咒）成绩不行→专门补习 |
| **实际目的** | 每天废张羽4个小时（+上课时间→几乎全天无自由时间） |
| **补课内容** | 300张符咒试卷→"做卷百遍，其意自现" |
| **执行方式** | 苏海峰→周老师（专门为张羽安排的补课老师） |

### 大学的等级体系补充

| 等级 | 说明 |
|------|------|
| **10大名校** | 顶尖大学 |
| **36上校** | 36所一等大学 |
| **72下校** | 72所三等大学（周豪毕业于此，链气巅峰+十年以上） |
| **大专** | 更低等级 |
| **杂牌** | 最低等级 |

### 24小时监视系统

| 设定元素 | 说明 |
|---------|------|
| **监视者推测** | 和周豪同一级别→链气巅峰+多年经验 |
| **专业方向** | 可能专精于偷拍、跟踪、监视 |
| **手段** | 摄像头+实体跟踪 |
| **逃脱难度** | 加速/急转/改变方位/窜入地铁→都无法甩脱 |
| **后果** | 无法去暗学帮赚钱→无法去保护区用灵脉修行 |
| **白真真的感受** | "深不见底的深渊，要将她和张羽彻底吞噬" |

### 狂龙制剂vs龙象制剂对比

| 对比项 | 龙象制剂 | 狂龙制剂 |
|--------|---------|---------|
| **表面** | 正常炼体药 | 看起来一样但实际不同 |
| **效力** | 正常 | 强好几倍（炸药感） |
| **药毒** | 正常 | 比普通药物猛烈十倍 |
| **成瘾性** | 无 | **有**——"只要有了第一次就有无数次" |
| **发给对象** | 正常学生 | 只有张羽拿到的是狂龙制剂 |

## 五、套路归档

### 套路名称：时间蚕食·放学后补课陷阱

**适用场景：** 敌人利用"合法"的学校规则来压缩主角的自由时间

**本章应用：**
- "放学后4小时按嵩阳市规则算正常教学时间"→不能旷课否则吃处分
- 补课内容：符咒试卷（做卷百遍）→单纯浪费时间
- 实际效果：每天多废4小时→全天几乎没有自由时间
- 与通识课不能旷、武道课基础功法叠加→三重时间压缩

**归档文件：** `写作资产/套路库/时间蚕食·补课陷阱.md`

### 套路名称：福姬的蛰伏战略·忍的智慧

**适用场景：** 在不可抵抗的对手面前，智囊建议暂避锋芒、专注自身成长

**本章应用：**
- 福姬分析：信息差在我们这边→但我们不了解对方的情报
- "他们拥有的资源、能量都是你们无法媲美的"
- "若是盲目行动，盲目反击，最后往往没什么好结果"
- **建议：** "先忍一忍"→"镇之以静"→"在忍耐中专注自己的成长"
- **等待时机：** "默默寻找破绽，等待反击的机会"
- **忍的条件：** 对方的手段没能真正打击到我们→有些方面反而能被利用
- **核心信息差：** 自身的能力、潜力、仪式力量

**归档文件：** `写作资产/套路库/福姬的蛰伏战略·忍的智慧.md`
