# 第193章 第二轮筑基考试开始 — 设计包 (v3)

---

## 一、核心概念

> **张羽白真真赴考 + 周家止步于筑基考试机密前 + 新BOSS周阳登场——镇压邪神的残疾人**
> 张羽和白真真凭借巡查队证明登上前往仙都的列车。周家突然撤回跟踪——周阳（周家核心人物）因惧怕手下发现筑基考试机密而叫停。张羽由此断定真正对手是周家而非周澈尘。新BOSS周阳完整登场：镇压了周家世传邪神当仪式电池，失去下半身靠换尸体穿戴。考场分组揭晓：张羽在体育赛道（vs夜凌霄、玉星寒、蒙涛等），白真真在武道赛道（vs云景等）。

**本章使命：**
1. **周家撤退与BOSS曝光**：周阳登场——残疾、镇压邪神的长线谋略家
2. **对手层级升级**：从周澈尘→周家上层（周阳）
3. **考场分组揭晓**：四条赛道——体育/武道/道心/法力
4. **体育赛道强敌**：夜凌霄、玉星寒、蒙涛等
5. **武道赛道强敌**：云景（仙都天骄，云家大力投资）
6. **周阳的邪神电池**：镇压邪神换取力量，下半身靠换尸穿戴

## 二、情节框架

### 双线叙事：张羽白真真赴考线 + 周阳线

| 段落 | 内容 | 核心信息 |
|------|------|----------|
| **云霓反应的分析(A段，1-7行)** | 福姬推断云霓知道筑基考试→云霓秒批原因 | 云霓身份的延伸设定 |
| **列车出发与周家撤退(B段，8-23行)** | 司骤雨追问为什么不跟→上层命令撤退 | 周家忌惮筑基考试 |
| **张羽的反向推理(C段，23-30行)** | 真正对手不是周澈尘→是周家 | 对手升级 |
| **周澈尘视角(D段，31-42行)** | 误以为张羽背后是巡查队 | 信息错位 |
| **周阳登场(E段，43-65行)** | 镇压邪神→失去下半身→换尸预算穿 | 新BOSS设定 |
| **周阳的邪神电池细节(F段，67-92行)** | 硬碟被仪式镇压的邪神→养邪神当电池 | 核心设定 |
| **考场分组(G段，93-105行)** | 体育赛道vs夜凌霄、玉星寒、蒙涛；武道赛道vs云景 | 分组揭晓 |

### 关键节点

1. **福姬分析云霓**（第1-7行）
   - 张羽觉得云霓"有点怕我继续说下去"
   - 福姬：云霓猜出你们参加筑基考试
   - 云霓作为代代相传的巡察人→知道筑基考试时间
   - 张羽一下子想通了许多

2. **周家撤退**（第8-17行）
   - 司骤雨追问"一群人陪这俩学生耗了这么久，你说不用跟了？"
   - 对方：公司领导命令，理解要执行不理解也要执行
   - 司骤雨愤怒→无奈离开
   - 张羽白真真确认：他们没跟上来

3. **对手层级升级**（第18-30行）
   - 张羽推理：想对付我们的不是周澈尘
   - 因为周澈尘恐怕不知道筑基考试
   - **福姬确认：** 知道筑基考试信息的恐怕是周家上层
   - **结论：** 真正对手是周家，不是周澈尘

4. **周澈尘的误判**（第31-42行）
   - 蓝岭不满：不该给他们批假
   - 周澈尘以为是巡查队后台
   - 认为张翩翩把关系介绍给了张羽
   - 想不通叔叔（周阳）打算干什么

5. **周阳登场**（第43-65行）
   - 中心大厦333层病床
   - 看到时间点和筑基考试吻合→惊出一身冷汗
   - 选择：不跟（怕手下查出不该查的）
   - 推断：四个月前就出城报名→张羽隐藏实力不小
   - 对邪神越来越感兴趣→"敢打筑基考试主意，位格必然不低"
   - 计划：等他们回来，抓住这条线索→吞邪神位格

6. **周阳的邪神电池系统**（第67-92行）
   - 监控画面：无数线圈、古董、锁链镇压一张硬碟
   - 硬碟剧烈跳动→铜线巨龙般捆缚镇压
   - 硬碟中镇压着周家代代相传的邪神
   - **周阳的核心哲学：** "为什么拜这个东西？应该是人养妖兽一样养着邪神"
   - 把邪神变成了自己的仪式电池
   - **代价：** 失去下半身
   - **解决方案：** 换尸体下半身（穷鬼卖腿→周阳买腿穿）
   - 尸斑问题→用一天问题不大
   - 去各个位置取用放养的下半身素材
   - **暗线：** 张羽是周阳选中的下半身素材——"我的下半身竟然去参加筑基考试了"

7. **考场分组**（第93-105行）
   - 四条赛道：道心、法力、体育、武功
   - 张羽→体育赛道（夜凌霄、玉星寒、蒙涛等）
   - 白真真→武道赛道（云景等）
   - 玉星寒消瘦→"你小子该不会考试前一天还在加班吧？"
   - 夜凌霄充满战意的眼神
   - 张羽心道："现在的我和两个月前的我可是云泥之别了"
