# 第194章 好学生就该打药 — 设计包 (v3)

---

## 一、核心概念

> **体育赛道打药考试规则揭晓 + 医药公司推动"以药代考"的制度讽刺 + 白真真武道擂台赛分组**
> 第二轮筑基考试体育赛道的上半场是"打药考试"——1小时内按药物对身体负担的积分总和排名。23种药物免费供应，张羽狂喜。医药公司推动的"以药代考"政策背后，黄子丑揭露正神被公司绑架的黑暗。白真真在武道赛道面临5人循环擂台赛，最大对手云景得到云家巨额投资。

**本章使命：**
1. **打药考试规则完整展示**：23种药物、积分制、免费供应
2. **医药公司权力揭露**：以药代考背后的利益链条
3. **张羽的贫穷本能**：第一反应是"这破考试收不收费？"
4. **网红评测视频设定**：药物介绍用网红视频→公司营销渗透考试
5. **白真真武道赛道**：5人循环擂台赛，云景是最大阻碍
6. **云景的危机感**：家族投资、挽救颓势的决心

## 二、情节框架

### 双线叙事：体育赛道打药规则线 + 武道赛道分组线

| 段落 | 内容 | 核心信息 |
|------|------|----------|
| **体育赛道规则讲解(A段，1-15行)** | 五名正神考官出现→晋级规则(前2晋)→张羽问有没有额外奖励 | 考试框架 |
| **上半场：打药考试说明(B段，14-66行)** | 23种药，按身体负担积分→1小时计时→不准用灵根/法力 | 核心考试内容 |
| **张羽的贫穷本能(C段，56-65行)** | "这破考试要是收药费"→考官说免费→双眼放光 | 穷人视角笑点 |
| **黄子丑内心吐槽(D段，66-71行)** | 医药公司推动以药代考→天庭缩减开支下的利益交易 | 制度讽刺 |
| **网红评测视频(E段，72-82行)** | 药品评测网红介绍20种药→视频里有开棺系列 | 设定补充 |
| **白真真武道赛道(F段，15-41行)** | 5人循环擂台赛→面对云景→雷霆法力热身 | 武道赛道 |
| **云景的决心(G段，26-32行)** | 云家投资→质的蜕变→必须拿到筑基资格证 | 对手侧写 |

### 关键节点

1. **晋级规则**（第1-15行）
   - 第二轮：每条赛道前2晋级→3个月后（明年1月）最后一轮角逐筑基资格证
   - 第3名：筑基推荐加分
   - 第4名：高考加分
   - 张羽举手问："第一第二名除了争取资格证的机会外就没有奖励了吗？"
   - 邓丙丁：最终轮不论拿没拿到证都有很不错奖励

2. **打药考试规则**（第14-66行）
   - **上半场**：1小时内给药自己打药
   - **药物选择**：23种，按身体负担从高到低排序给不同积分
   - **限制**：不准用灵根、不准动用法力
   - **下半场**：等待揭晓
   - **积分**：负担越大积分越高

3. **张羽的贫穷本能**（第56-65行）
   - 第一反应："这破考试要是收药费——那我就只能找人借钱去了"
   - 问完免费后→"双眼放光"
   - "免费？那就是自助打药了吗？这帮正神总算干了点正事"

4. **黄子丑的内心吐槽——医药公司黑暗**（第66-71行）
   - "医药公司做成功了，说什么以药代考"
   - "最后还不是要报销到神部头上？让神部来付药费？"
   - "亏的不还是天庭？损的不还是昆墟上下的利益？"
   - "还特么的一分钱都没打给我"
   - "公司里那些狗东西，真是不把九品神当神啊"
   - 涉及上面十几层考试的交易→水深不见底→只能心里抱怨

5. **网红评测视频**（第72-82行）
   - 视频介绍20种药物
   - 原身记忆：这个网红以开箱视频出名
   - **开棺系列**：展示不同年份的养户状况→参观地方豪族祖坟
   - **记忆暴击：** "知道有些人的祖坟可以比他的学校还大……哪怕躺棺材里以后每天增值的财富都比他这种穷鬼打工挣得更快"
   - 视频对药物的介绍深入浅出

6. **白真真武道赛道**（第15-41行）
   - 进场后发现没有张羽→不同赛道分开考试
   - 雷霆法力不断激荡跳跃→静不下来
   - 云景是上一场第二名→白真真判断最大阻碍
   - 云景感觉到空气中的静电刺痛→注意到白真真
   - 云景记得上一轮的56号（白真真）

7. **云景的决心**（第26-32行）
   - 两个月来家族投资远超以往→实力质的蜕变
   - 各项道心/法力/肉体已达链气巅峰→继续获得质的飞跃
   - 目标：拿到筑基资格证→挽救家族颓势
   - 庆幸：夜凌霄没在武道赛道
   - 决心："我绝不能败"

## 四、设定细节

### 打药考试规则

| 设定元素 | 说明 |
|---------|------|
| **考试内容** | 1小时内给药自己打药 |
| **药物数量** | 23种 |
| **积分规则** | 按身体负担从高到低给不同积分 |
| **限制** | 不准用灵根、不准动用法力 |
| **身体监控** | 安装检测贴片→超出负荷自动淘汰 |
| **考试时间** | 上半场（下半场待揭晓） |
| **药物来源** | 免费→考场提供 |

### 医药公司权力渗透

| 设定元素 | 说明 |
|---------|------|
| **政策推动** | 医药公司推动"以药代考" |
| **报销链条** | 神部付药费→天庭亏钱→损昆墟利益 |
| **层级** | 涉及上面十几层考试 |
| **黄子丑态度** | 九品神敢怒不敢言 |
| **网红视频** | 公司营销推动的药品评测视频被考试采用 |

### 网红评测视频

| 元素 | 说明 |
|------|------|
| **网红** | 以开箱视频出名，曾拍开棺系列 |
| **内容** | 评测20种常见药物 |
| **深度** | 深入浅出，直白易懂 |
| **开棺系列** | 参观地方豪族祖坟→展示不同年份养户状况 |
| **记忆冲击** | 祖坟比学校大，躺棺材里增值比打工快 |

### 五名正神考官的能力

| 能力 | 说明 |
|------|------|
| **神力化身** | 同时控制4个考场（4具身体/4个视野） |
| **一心多用** | 加班开会、开会送礼、送礼加班→基础操作 |
| **效率** | 早已习惯 |
