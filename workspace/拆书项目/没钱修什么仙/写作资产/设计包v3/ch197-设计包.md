# 第197章 高中圣体？炼气圣体！ — 设计包 (v3)

---

## 一、核心概念

> **张羽发现高中圣体真正含金量＝链气期极限药物代谢 + 正神惊议穷人天才崛起 + 夜凌霄正式应战**
> 张羽运转春秋无尽禅后意识到：自己运功前后的药物代谢能力没有变化——因为高中圣体已经将药物负荷、吸收、代谢能力提升到了链气期极限。这一刻他明白了高中圣体＝炼气圣体。正神们在内部网络震惊于张羽扛下两片仿凤功能素，发现他在月考中隐藏了实力。黄子丑和邓丙丁各自推测张羽背后有人。夜凌霄看到张羽的实力后战意更浓，注射法力脉络针剂。张羽也不甘示弱跟着注射法力脉络针剂。

**本章使命：**
1. **高中圣体真相揭示**：药物代谢能力已到链气期极限（提无可提），因此运转春秋无尽禅也无提升
2. **张羽自信升级**：确认自己能赢夜凌霄——只要对方代谢没到链气巅峰
3. **正神视角·穷人天才的困境**：隐藏实力的必要性、崛起背后需要有人
4. **邓丙丁的态度**：想起张翩翩飞升→决定不插手但考试结束加好友
5. **竞赛正式开始**：夜凌霄注射法力脉络针剂→张羽跟着注射→双线竞赛

### 关键节点

1. **高中圣体＝链气圣体**（第1-23行）
   - 张羽运功后察觉：春秋无尽禅对药物代谢没有提升
   - 推理过程：春秋无尽禅能汇聚药毒 → 非功法问题 → 即代谢已达极限
   - 「这个链气巅峰说的是将我的药物负荷、吸收、代谢能力提升到了链气期的极限」
   - 之前没发现的原因：「前段时间我吃的药太少太一般了」
   - 「仿凤功能素的药性足够强悍汹涌，才让我察觉到了高中圣体的含金量」
   - 瞬间自信：「23号的仙都爷再怎么样厉害，终究也是个高中生，只要他的药物代谢能力没有达到链气期的极限……那这一场考试我必胜！」
   - **核心设定释放：** 高中圣体套装在药物代谢维度的真正含义

2. **张羽趁药炼体**（第24-25行）
   - 虽然不运功也能代谢掉药力，但免费神药不能浪费
   - 借助药力继续磨练筋肉、骨骼、脏腑→长生法力滋养
   - **穷人的精打细算：** 每一份药都要最大化利用

3. **正神对张羽的重新评估**（第26-45行）
   - 「这个张羽一口气扛下了两片仿凤功能素？以他的成绩来说并无可能」
   - 查看检测数据→「在月考中隐藏实力了」
   - 举例穷人隐藏实力的方式：故意打伤/服用副作用药物/修行隐藏功法的功法
   - 但隐藏要付出代价：受伤拖累修行/浪费时间和精力/失去福利援助
   - 「偏偏需要隐藏实力的学生往往又是穷人，本来就缺钱修炼」
   - 黄子丑对张羽警惕：会不会和邪神有关？→检查后松一口气
   - 正神评价：「此子的仙道天赋不可限量」
   - **核心表达：** 穷人隐藏实力是生存策略，代价巨大但不得不为

4. **黄子丑的清醒认知**（第48-54行）
   - 「天赋个鬼！……所有穷人天才能崛起的最重要因素，就是特喵的背后有人」
   - 「光靠天赋和努力就能崛起？也就那些被公司、媒体反复洗脑的凡人才会相信」
   - 长年负责教育行业→见得太多了
   - 「所有能崛起的穷人天才，要么是展露天赋后被强力人物收下，要么是天赋本身便来自强力人物的布局」
   - **世界观深化：** 昆墟一层并非个人努力改变命运的地方——阶层固化，背后有人才可能崛起

5. **邓丙丁的谨慎**（第55-62行）
   - 想起张翩翩飞升→推测张羽背后有神
   - 想交好→立刻想起上次查张翩翩资料后的遭遇→「还是不要多此一举了」
   - 「上面那些大人的一举一动，一言一行都必有深意」
   - 决定考试结束后加个好友
   - **人物刻画：** 邓丙丁能走到今天的位置，靠的就是这份谨慎和边界感

6. **竞赛正式启动**（第63-85行）
   - 夜凌霄看到张羽吃两片药后战意更浓：「这才是你的真实实力吗？」
   - 「果然你上一轮考试隐藏了很多啊。但我可不会输给你。」
   - 夜凌霄的自白：「我可是夜氏一族耗费无数心血……注定要拿下筑基证的完美血脉」
   - 「我的仙道之路可不只是十大」
   - 注射法力脉络针剂→10级春秋无尽禅全力化解
   - 在完美血脉加持下肉体如鲸鱼吞吐药力→感觉还能再打一针
   - 正想再接再厉→瞥见张羽也拿起了法力脉络针剂
   - 「只炼体也太浪费了，顺便再提升一下法力吧」
   - 张羽一针扎下→大日气海猛烈运转，疯狂收摄灵机

## 四、设定细节

### 高中圣体的真正含义（链气巅峰代谢）

| 维度 | 内容 |
|------|------|
| **原始描述** | 「药物负荷、吸收、代谢能力提升至链气巅峰」 |
| **张羽最初理解** | 模糊区间，类似高中毕业多年后的水平 |
| **现在理解** | **链气期的极限**——药物代谢能力提到链气期最大值，提无可提 |
| **为什么之前没发现** | 吃的药太少太一般，不足以触发极限感知 |
| **证据** | 运转春秋无尽禅后代谢无变化 + 春秋无尽禅能汇聚药毒 |
| **实际效果** | 常态状态下就已是链气极限代谢水平 |
| **张羽的称谓** | 不应叫高中圣体，应叫**链气圣体** |

### 穷人隐藏实力的代价

| 维度 | 内容 |
|------|------|
| **为什么隐藏** | 降低存在感，不引起富人注意 |
| **方法1** | 故意打伤自己 |
| **方法2** | 服用副作用巨大的药物降低实力 |
| **方法3** | 修行隐藏实力的功法 |
| **代价** | ①受伤拖累修行 ②浪费大量时间精力 ③失去学校福利和援助 |
| **恶性循环** | 穷人本来就缺钱→隐藏实力失去福利→更缺钱 |
| **正神的评价** | 能在这种情况下还达到争夺筑基证的水平——天赋不可限量 |

### 昆墟一层的崛起规则

| 维度 | 内容 |
|------|------|
| **表面叙事** | 天赋和努力改变命运（被公司和媒体反复洗脑的凡人相信的） |
| **实际规则** | ①展露天赋后被强力人物收下，②天赋本身便来自强力人物的布局 |
| **黄子丑的经验** | 见过5-6个类似水平的穷人天才→没有一个是纯粹靠自己 |
| **核心表达** | 阶层固化，背后有人才可能出头 |
