# 第198章 考试暂停？ — 设计包 (v3)

---

## 一、核心概念

> **张羽vs夜凌霄药物竞赛白热化 + 道解缓释胶囊副作用曝光（降低智商） + 正神为药保密暂停考试**
> 张羽注射法力脉络针剂后意识到不同药物负担身体不同区域，从而采取多药联用策略分担副作用。夜凌霄紧跟其后，使用道解缓释胶囊压制情绪和副作用痛苦。两人积分半小时达到9000分。蒙涛、玉星寒惊叹。张羽注射龙象制剂试图超越，夜凌霄不服再服道解缓释胶囊。张羽观察夜凌霄发现其说话颠三倒四——道解缓释胶囊的副作用是降低智商。黄子丑暂停考试，要求全体签保密合同。张羽放弃该药。

**本章使命：**
1. **多药联用策略释放**：张羽发现不同药物负担身体不同区域→可同时服用多种药物分摊副作用
2. **积分竞赛白热化**：两人半小时齐刷刷9000分，外人惊叹
3. **道解缓释胶囊副作用揭露**：降低智商/认知混乱，说话颠三倒四
4. **考试暂停的真实原因**：正神要为药物公司的副作用保密，不是为考生健康
5. **张羽的观察者智慧**：通过观察夜凌霄服药后的反应来决定自己是否用药

## 二、情节框架

### 关键节点

1. **张羽的多药联用策略**（第1-14行）
   - 注射法力脉络针剂后感觉：呼吸急促、心脏加速、经脉微抽
   - 意识到不同药物负担不同区域→仿凤功能素（消化系统负担）vs法力脉络针剂（呼吸/循环负担）
   - 「所以想要积分更高，想要在这自助餐的过程中修炼得更加全面，那就应该多服用一些不同类型的药物」
   - 道心、法力、肉体强度三者的药物可以同时服用→分担副作用+全面提升
   - 抓紧时间修炼，不浪费任何药效
   - **关键设定释放：** 多药联用＝分散副作用到身体不同系统

2. **两人竞赛的旁观者视角**（第33-36行）
   - 蒙涛：「这才过了一半多点的时间，就吃了六轮神药了？这两个家伙的代谢能力竟然比我强这么多吗？」
   - 「如果把今天的吃药大赛比作吃屎大赛的话，我见这两人，便犹如屎壳郎见化粪池，简直是云泥之别」
   - 玉星寒：「这小子……竟然跟上了顶尖仙都爷的节奏？」
   - **幽默表达：** 用荒诞比喻强化两个人的碾压级代谢能力

3. **龙象制剂的出现**（第37-43行）
   - 张羽注射龙象制剂→夜凌霄解读：「用龙象制剂来暂时超越我？」
   - 夜凌霄自比拍卖场：「就像拍卖场上，已经没有余额的对手加了个100块钱就想超越他一样」
   - 「这家伙到极限了吗？」
   - 夜凌霄决定：再服道解缓释胶囊
   - **竞赛白热化：** 双方都在极限边缘博弈

4. **夜凌霄的决心与家训**（第44-54行）
   - 「第一永远是我，也只能是我」
   - 「屹立于顶点，永远地独占鳌头，永远地抢占第一」→夜氏一族代代相传的祖训
   - 「在这昆墟之中只要不成第一，那不论是世家还是天才，便永远有着被淘汰的那一天」
   - 想到夜氏一族即将突破仙都、向整个昆墟一层扩张→万丈豪情
   - **人设深化：** 夜凌霄的"第一执念"是家族基因和历史责任

5. **道解缓释胶囊副作用的发现**（第55-72行）
   - 张羽一直没服道解缓释胶囊→因为不明确不良反应→所以通过观察夜凌霄来判断
   - 观察：夜凌霄面色发紫、头顶隆起小包→「好像还行」
   - 夜凌霄开始说话颠三倒四：「等我吃了这颗道解缓释胶囊，就会把你彻底压制！」
   - 张羽震惊：「对方刚刚说的是人话？到底是他自己理解有问题，还是对方出了问题？」
   - 「难道这就是道解缓释胶囊的副作用？不会是什么智商降低了吧？」
   - **核心信息：** 道解缓释胶囊的副作用＝认知混乱/智商降低

6. **暂停考试的残酷真相**（第73-78行）
   - 黄子丑叫停考试：「23号选手目前身体出现了一些状况，需要暂停两分钟」
   - 张羽心道：「要治疗夜凌霄吗？这些正神倒还有点良知」
   - 黄子丑接着：「接下来所有考生一同和我签一份合同，要对于23号选手在服用道解缓释胶囊后的反应进行保密」
   - 张羽：「特喵的……暂停考试是因为要给公司这药物副作用保密？」
   - **核心表达：** 正神暂停考试不是为了考生健康，而是为了给药物公司擦屁股——考试规则服从商业利益

## 四、设定细节

### 多药联用策略

| 维度 | 内容 |
|------|------|
| **发现过程** | 法力脉络针剂副作用在呼吸/循环系统→和仿凤功能素（消化系统）不同 |
| **策略本质** | 不同类型药物的负担落在身体不同区域→可同时服用分摊副作用 |
| **三条线** | 道心类药物+法力类药物+肉体强度类药物→同时服用、全面提升 |
| **效果** | 身体承载力最大化利用，积分提升更快 |

### 道解缓释胶囊

| 维度 | 内容 |
|------|------|
| **功能** | 压制负面情绪，降低药物副作用对精神的影响 |
| **副作用** | 智商降低/认知混乱——说话颠三倒四、逻辑自我矛盾 |
| **身体表现** | 面色发紫、头顶隆起小包 |
| **夜凌霄的服用量** | 至少2颗（具体数量未明） |
| **张羽的观察结论** | 通过观察夜凌霄的反应来决定自己是否服用 |
| **最终处理** | 考试暂停后每人限服1颗 → 张羽放弃 |

### 药物分类体系

| 药物类型 | 负担区域 | 功效 |
|---------|---------|------|
| **仿凤功能素** | 消化系统（肠胃腹痛） | 提升肉体强度 |
| **法力脉络针剂** | 呼吸系统/血液循环 | 加速法力运转 |
| **道解缓释胶囊** | 精神/认知系统（副作用） | 压制负面情绪 |
| **龙象制剂** | 未明示 | 暂时性力量提升 |

**适用场景：** 主角发现不同药物负担身体不同区域，从而采取多药联用策略最大化药效

**本章应用：**
- 张羽分别感知仿凤功能素（消化）vs法力脉络针剂（呼吸/循环）的不同负担
- 决定同时服用道心/法力/肉体三类药物分摊副作用
- 修行效率加倍
- **核心表达：** 不是蛮吃药，而是有策略地用身体不同区域的承载能力
