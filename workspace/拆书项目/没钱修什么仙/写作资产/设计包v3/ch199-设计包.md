## 一、核心概念

> **张羽上半场积分超越夜凌霄 + 下半场玄冥重水压力测试开启 + 武道赛场白真真vs戴行之**
> 签完保密合同后续考。张羽放弃道解缓释胶囊，选择其他不良反应明确的药物狂灌。夜凌霄又不甘地连服两片仿凤功能素——面色通红、血管爆出、五脏绞痛——仍被张羽反超。上半场结束：张羽第一、夜凌霄第二。下半场开启——定海柱中的玄冥重水模拟水下压力。张羽决定趁压力还不大的时候继续修炼春秋无尽禅强化肉体。画面切换至武道赛场：戴行之的白真真擂台赛即将开始，戴行之的省钱哲学+速度型战斗风格展示，医保数额设定释放。

**本章使命：**
1. **上半场收尾**：张羽疯狂塞药积分超越夜凌霄，拿下用药第一
2. **夜凌霄的不甘+道解副作用残留**：说话逻辑仍半混乱状态，加重人物弧度
3. **定海柱·玄冥重水设定释放**：压力测试规则，模拟水下1米到10000米
4. **张羽的战略远见**：不在乎一时输赢，趁压力不大继续修炼→长远实力至上
5. **武道赛场引入**：戴行之的省钱哲学+速度战斗风格+医保数额设定
6. **为下章白真真vs戴行之同归于尽铺垫**

### 关键节点

1. **张羽的确定性用药策略**（第9-13行）
   - 「鬼知道这智商要降低多久，万一这破玩意是永久降低智商怎么办？」
   - 放弃道解缓释胶囊，选择不良反应明确的药物
   - 一瓶瓶灌入口中→道心、法力、肉体强度暴涨
   - 分数不断接近夜凌霄
   - **人设特质：** 张羽宁可少拿分也不冒未知风险——穷人经不起永久性损伤

2. **夜凌霄的反击与极限**（第14-21行）
   - 十多分钟内先后服下两片仿凤功能素
   - 面色通红、浑身血管爆出体表、五脏六腑绞痛、「好像肠胃在融化一般」
   - 看到张羽停下来以为到极限了→结果张羽举手问能否上厕所
   - 「饱腹感」——吃完药还能有饱腹感？！
   - 张羽上了厕所继续吃药：仿凤功能素+法力脉络针剂+道心类药物
   - 夜凌霄几次想跟→身体濒临极限→只能咬紧牙关闭眼放弃
   - **核心表达：** 张羽的代谢能力（高中圣体）完胜夜凌霄的完美血脉

3. **夜凌霄的不甘+道解副作用残留**（第23-25行）
   - 「没事……还有下半场，以我的实力，我下半场就能追回来」
   - 但紧接着：「但以我的实力，我下半场如果追不上怎么办？」
   - **逻辑矛盾暴露了道解副作用的残留：** 前半句自信，后半句立刻自我怀疑
   - 这不是正常夜凌霄的心态 → 副作用还在

4. **定海柱·玄冥重水设定**（第26-38行）
   - 考场中央升起五根圆形缸体（定海柱）
   - 玄冥重水：整体5米高，但能模拟水下1米到10000米的压力
   - 规则：承受药物副作用的同时对抗压力→全方位的肉体强大
   - 张羽带上氧气面罩跳入→玄冥重水轻如空气→考试开始后压力递进
   - **核心设定：** 玄冥重水＝可调节压力的特殊水体

5. **张羽的长期主义**（第40-47行）
   - 评估夜凌霄：肉体10级+功法多+肉体控制力强→压力测试劣势
   - 但上半场积分领先→总分前二没问题→一定能挺进下一轮
   - 「既然如此，反正这一轮筑基考试都一定能通过，那我干脆趁着现在的机会变得更强」
   - 趁着模拟深度还不高→在玄冥重水中继续运转春秋无尽禅→强化肉体强度
   - **人设特质：** 张羽不追求短暂的第一，而是追求最大化提升自身实力

6. **戴行之的人设展开**（第53-93行）
   - 仙都1.5层六环内天骄→看不起外地人→但对白真真例外
   - 「速度方面……自然是为了省钱」
   - 选择速度型/闪避方向的原因：医疗费消耗巨大的力量型/横练型太贵
   - 「不伤优于不败，输了还能再来，穷了再难翻身」——戴氏代代传承的思想
   - vs云景直接认输：「如果被打出100万都治不好的伤，那就要带伤进入下一场」
   - **核心人设：** 仙都的"穷人"（相对有钱人而言），靠省钱哲学在仙都立足

7. **医保数额设定**（第55-62行）
   - 所有考生有100万医保额度，每场擂台赛前重置
   - 每一次受创会被神阵瞬间治愈，消耗医保额度
   - 医保耗尽 = 输掉擂台
   - 被击倒/击晕 = 判负（但会被治愈）
   - 10分钟未分胜负 = 医保余额高者胜
   - **设定意义：** 保证擂台战效率和观赏性，同时让战斗策略更多元

## 四、设定细节

### 玄冥重水·定海柱

| 维度 | 内容 |
|------|------|
| **外观** | 五根圆形缸体，高5米 |
| **水体** | 玄冥重水——可模拟水下1米到10000米压力 |
| **初始状态** | 轻如空气，完全无压力 |
| **压力递进** | 水面浮现数字（-1米→-10米→-100米……逐步加深） |
| **氧气** | 氧气面罩不受玄冥重水影响 |
| **考试目的** | 药效下全方位肉体强大的检测：抗压+肉体力量+肉体技巧 |

### 医保数额系统（武道赛场）

| 维度 | 内容 |
|------|------|
| **数额** | 每场100万，每场重置 |
| **治疗方式** | 被创→神阵瞬间治愈→消耗医保额度 |
| **败北条件1** | 医保额度耗尽=输 |
| **败北条件2** | 被击倒/击晕=判负（但会被治愈） |
| **平局机制** | 10分钟未分胜负→医保余额高者胜 |
| **治疗限制** | 被击倒后瞬间治愈 → 可连续作战 |

### 药物极限（上半场数据）

| 选手 | 积分排名 | 状态 |
|------|---------|------|
| **张羽** | 第1名 | 吃喝6+轮神药，但只吃不良反应明确的药，不吃道解 |
| **夜凌霄** | 第2名 | 面色通红、血管爆出、肠胃绞痛、说话逻辑混乱（道解副作用残留） |
