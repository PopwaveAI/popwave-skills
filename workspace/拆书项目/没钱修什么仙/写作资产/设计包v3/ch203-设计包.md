# 第203章 第二轮筑基考试结束 — 设计包 (v3)

## 一、核心概念

> **夜凌霄深度-2581米停止考试 + 第二轮筑基考试结束 + 张羽向邓丙丁举报周家 + 白真真vs云景一招落败**
> 夜凌霄在-2581米处被黄子丑叫停（总分已第一），第二轮筑基考试肉体赛道结束——张羽和夜凌霄获得前两名，将在明年1月参加最后一轮。玉星寒决定回去找师姐要钱加课跟上张羽。张羽主动留下向邓丙丁举报周家跟踪，邓丙丁看破不说破，加了好友。武道考场上白真真对云景发起试探性攻击，被云景一记火焰罡气墙强行推出擂台，一口医保先烧掉二十多万治愈旧伤，发现云景强大且战术聪明。

**本章使命：**
1. **第二轮筑基考试收官**：前两名决出（张羽、夜凌霄），明年1月最终轮
2. **玉星寒动机深化**：决定找李雪莲要钱→加课张羽→明年分赛道→一起考上十大
3. **张羽举报周家**：主动向邓丙丁汇报→加正神好友→为后续破局铺垫
4. **邓丙丁的态度**：识破张羽想借火但不说破，顺势保护以讨好后方的"大人物"
5. **白真真vs云景**：暴露顶级仙都爷碾压级的实力（双丹田火焰罡气墙）
6. **星火真人的十大故事**：十年大学屡败屡战→最终服了→阶层固化主题深化

### 主视角（上帝视角/穿插玉星寒、张羽、白真真三条线）

| **玉星寒的算盘(B段，9-32行)** | 感叹夜凌霄怪物→担心明年还遇到张羽→决定找师姐要钱加课张羽→分赛道策略→星火真人的十大教训 | 富人的学习策略+阶层固化的十年故事 |
| **张羽举报周家(C段，33-61行)** | 张羽主动留下面见黄子丑→邓丙丁截胡→汇报周家跟踪→邓丙丁看破不说破→加好友 | 张羽利用考试信息差布局，邓丙丁顺势结交 |

### 关键节点

1. **夜凌霄的功法储备恐怖**（第2-4行）
   - 除了三重护体功法，还运转了"钛金炼体要诀"
   - 整个人金光闪闪，散发出金属色泽，"犹如一尊矗立在海底深处的神像"
   - -2581米仍然没有达到极限
   - 黄子丑因"总分已排第一"而叫停
   - **核心表达：** 夜凌霄的极限深不可测

2. **玉星寒的十大策略**（第9-32行）
   - 感叹夜凌霄怪物→判断这届筑基证被夜凌霄拿下
   - 发现自己明年还得和张羽同届→决定加课跟上
   - **分赛道策略：** "明年一定要跟他达成默契，在筑基考试中分到不同的赛道去"
   - 如果两人都考上十大→"上了二层，我们的竞争对手就是上面那些爷了"
   - 有朋友能借钱和分担压力很重要
   - **星火真人故事：** 考上万法大学→被上层爷教训→屡败屡战→服了→"有些人你出生的时候追不上，这辈子就都追不上了"
   - **核心表达：** 哪怕考上十大，阶层差距依然是无法逾越的深渊

3. **张羽举报周家·邓丙丁收网**（第39-58行）
   - 张羽主动开口："我有些事情想要报告一下"
   - 黄子丑想趁机结交却被邓丙丁截胡
   - 邓丙丁："子丑，你先去把今天的报告写了吧"
   - 黄子丑内心："不是……这你都要抢？""钱都特幺塞给狗吃了"
   - 张羽汇报被周家跟踪→邓丙丁似笑非笑看破
   - 邓丙丁判断：嵩阳市有家族敢查筑基考试？不太信→更信是张羽想引火烧周家
   - 给张羽加好友→"保护每一个筑基考试的学生的信息安全"
   - **核心设伏：** 张羽得到正神好友，周家毫不知情

4. **白真真蹭医保艺术**（第65-70行）
   - "至少要看一看我和对方的差距到底在哪里"→但还有一个原因："蹭一蹭医保了"
   - 医保额度出现的瞬间就开始减少：100万→减少了二十多万才停
   - 旧伤全部愈合→"通体一阵舒泰"
   - **穷人哲学：** 战斗不只是战斗，还能蹭免费治疗

5. **云景的双丹田火焰罡气墙**（第74-92行）
   - 白真真万剑无终诀8级→雷云风暴般攻击→试探云景反应
   - 云景：法骸双眸看到每一招的数字（3200/500/4899=医保消耗）→但不逐个破解
   - 直接一掌轰出："体内双丹田的悍然爆发，一股股火焰罡气便如同火山喷涌"
   - 罡气如赤红城墙→擂台一分为二→活活把白真真挤出擂台
   - "没有一点点技巧，全都是钱砸出来的底蕴"
   - 白真真分析：云景在发现她擅高速近战后，用大范围力量型招数压制空间和速度
   - **核心表达：** 云景不只是有钱，他还能针对弱点制定战术

## 三、人物设计

### 夜凌霄 —— 深不可测的完美血脉

| 维度 | 分析 |
|------|------|
| **最终深度** | -2581米（未到极限，被叫停） |
| **新增功法** | 钛金炼体要诀（上一轮筑基考试中出现过的功法） |
| **视觉呈现** | 金光闪闪，金属色泽，如海底神像 |
| **总分** | 排第一，黄子丑主动叫停 |

### 玉星寒 —— 从竞争者变成追随者的仙都天才

| 维度 | 分析 |
|------|------|
| **心态变化** | 原本和张羽差不多水平→现在张羽已全面领先 |
| **应对策略** | 找李雪莲要钱→加课张羽→明年分赛道 |
| **分赛道逻辑** | 不与夜凌霄同届（今年）+不与张羽同赛道（明年）→提高拿证概率 |
| **未来规划** | 若一起考上十大，互相照应 |
| **星火真人故事** | 十年大学挑战上层爷→最终服了→阶层固化是铁的 |

### 黄子丑 —— 被截胡的职场人

| 维度 | 分析 |
|------|------|
| **对张羽的判断** | 价值可能比夜凌霄更大→因为他查不出张羽背后是谁→"越是查不出来，便越让黄子丑感觉到一种不得了" |
| **内心落差** | 想趁机结交张羽→被邓丙丁截胡 |
| **内心吐槽** | "钱都特幺塞给狗吃了" |

### 邓丙丁 —— 看破不说破的六等正神

| 维度 | 分析 |
|------|------|
| **识破力** | 立刻看穿张羽想引火烧周家 |
| **策略** | 不拆穿→顺着张羽的意思接住→加好友→"保护每一个筑基考试的学生的信息安全" |
| **深意** | 也想知道张羽背后到底是什么大人物→顺势而为 |
| **行动** | 答应调查周家→给了张羽直接联系通道 |

### 张羽 —— 主动布局的穷人智慧

| 维度 | 分析 |
|------|------|
| **举报动机** | 借助正神力量对付周家 |
| **说话技巧** | "我担心他们是要追查我参加筑基考试的事情"→用考试安全作为理由 |
| **意外收获** | 加了正神好友（之前不知道能加，可能因为信仰等级太低） |

### 云景 —— 碾压级的仙都天骄

| 维度 | 分析 |
|------|------|
| **法骸双眸** | 能看到每一招命中后的医保消耗数值（3200/500/4899） |
| **战术风格** | 不逐个破解招数→一力降十会→大范围碾压 |
| **双丹田** | 比常人多了丹田储备，罡气如火山喷涌 |
| **战斗智商** | 发现白真真擅高速近战→用罡气墙封空间+压力量 |
| **白真真的评价** | "不只是强大和有钱，他还会针对对手的弱点进行战斗" |

### 白真真 —— 试探性的穷斗士

| 维度 | 分析 |
|------|------|
| **战术** | 先蹭医保治愈旧伤→万剑无终诀全力试探→获取情报 |
| **目标的变更** | 已不指望赢→只要知道差距在哪 |
| **分析深度** | 复盘发现云景不是无脑碾压，而是针对弱点的压制 |
| **后续所需** | "还要再多学几门厉害功法进行推演""再花个几百万" |

## 四、设定细节

### 法骸双眸·数据化视力（云景）

| 维度 | 内容 |
|------|------|
| **功能** | 能看到每一招命中后的医保消耗数值 |
| **本章展示** | 白真真一招包含多剑→云景看到：3200/500/4899（不同剑的医保消耗） |
| **战术价值** | 找到破绽→分清楚招数中的虚实变化 |
| **云景的使用方式** | 看得到但不逐一破解→直接一力降十会（说明他自信碾压） |

### 双丹田·火焰罡气（云景）

| 维度 | 内容 |
|------|------|
| **原理** | 体内拥有两个丹田→罡气储备远超常人 |
| **爆发效果** | 火焰罡气如火山喷涌→源源不断泵出→形成赤红城墙 |
| **战术应用** | 擂台一分为二→碾压式推挤→活活将对手挤出擂台 |
| **白真真的感受** | "铺天盖地般压来的罡气""惊人的热浪""被活活挤下了擂台" |

### 医保擂台·蹭医保的艺术

| 维度 | 内容 |
|------|------|
| **白真真的策略** | 上台先烧二十多万医保治愈旧伤 → 然后再全力战斗 |
| **目的** | "至少要看一看我和对方的差距到底在哪里" + "蹭一蹭医保" |
| **核心表达** | 对穷人来说，战斗本身也是一种获取免费治疗的手段 |

### 星火真人·十大教训

| 维度 | 内容 |
|------|------|
| **故事** | 考上万法大学→被上层爷教训→不服→再次挑战→又被教训 |
| **十年结果** | 屡败屡战十整年→最终服了 |
| **核心教训** | "有些人你出生的时候追不上，这辈子就都追不上了" |
| **对玉星寒的影响** | 深以为然+决定找朋友（张羽）一起上十大互相照应 |

## 五、套路归档

### 套路名称：分赛道策略·考试博弈的精细算盘

**适用场景：** 考生评估对手实力后，主动选择错开赛道提高成功率

**本章应用：**
- 玉星寒：不与夜凌霄同届（今年）+不与张羽同赛道（明年）→提高拿证概率
- 如果两人都考上十大→"上了二层互相照应"
- **核心表达：** 聪明的考生不只是提升实力，还会策略性选择赛道
