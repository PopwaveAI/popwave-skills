# 第204章 张羽议计，正神秘辛 — 设计包 (v3)

---

## 一、核心概念

> **考试结束后各方行动 + 福姬揭露正神系统的经济本质 + 张羽谋划借正神之力破局**
> 第二轮筑基考试结束后，各方展开行动：夜凌霄让家族调查药物代谢技术（因张羽的表现引起注意）；玉星寒找李雪莲要钱加课张羽，成功压到3万/小时；张羽和白真真在回程列车上复盘差距，发现接下来三个月需要花几百万甚至上千万才能追上。福姬苏醒后提议走黑暗补课界赚钱，但周家24小时监控是阻碍。张羽指出已有邓丙丁这位正神好友，福姬顺势揭露八部正神系统的经济本质——正神要晋升也需要钱，回收货币是正神的核心考核。

**本章使命：**
1. **多线汇合**：夜凌霄查技术、玉星寒加课、张羽白真真复盘→三条线同时推进
2. **金钱焦虑升级**：三个月需要几百万到上千万花销
3. **福姬补课界提议**：黑暗补课界是穷人唯一的赚钱出路
4. **正神秘辛揭露**：八部正神需要钱才能晋升，赚钱/回收货币是核心考核
5. **借火破局方案**：利用正神从周家身上赚钱的需求，给邓丙丁一个"抓手"
6. **周家持续施压**：压缩张羽白真真的时间，一步步逼入困境

## 二、情节框架

### 主视角（上帝视角/多线并行）

| 段落 | 内容 | 核心信息 |
|------|------|----------|
| **夜凌霄家族调查(A段，1-10行)** | 仙都1.5层→夜凌霄让家族查道解缓释胶囊投资→再让查外地药物代谢技术 | 夜凌霄注意到张羽的秘密但方向偏了 |
| **玉星寒要钱加课(B段，11-33行)** | 找李雪莲→从5万砍到3万/小时→李雪莲要亲自聊→玉星寒斡旋成功 | 张羽的补课涨价到3万/小时 |
| **张羽白真真复盘(C段，34-57行)** | 白真真分析云景战术→张羽对比夜凌霄差距→都需要几百万→高中圣体套装选择分析 | 双主角的修行规划+金钱焦虑 |
| **福姬提议+正神秘辛(D段，58-90行)** | 黑暗补课界提议→周家监控阻碍→张羽提出邓丙丁好友→福姬揭露正神也要钱 | 八部正神体系的经济本质揭示 |

### 关键节点

1. **夜凌霄调查张羽**（第1-10行）
   - 夜凌霄在仙都1.5层打电话通知家族参与道解缓释胶囊投资
   - 又委托家族调查："昆墟一层的其他城市，有没有公司研发出和药物代谢相关的技术？"
   - **核心含义：** 张羽已经引起了夜凌霄的高度警觉，但方向错到"外地公司技术"

2. **玉星寒斡旋砍价**（第11-33行）
   - 玉星寒：找师姐要钱→报价5万/小时→李雪莲嫌贵→玉星寒压到3万/小时
   - 李雪莲："你不是成绩下滑的厉害吗？你和他凑太近，他不会害了你吗？"
   - 玉星寒心想：总不能说"我和他是一起卖种子的交情"
   - **核心幽默：** 卖种子的交情成为两人互相信任的隐秘纽带

3. **双主角金钱焦虑**（第34-57行）
   - 白真真：算来算去要再花几百万
   - 张羽：三个月内要将道心、法力、肉体强度推到炼气极限 + 更多功法 + 凑齐比高中圣体更强的套装
   - 张羽分析：链法图只能拖入十门功法→一个套装六门起效→所以只能同时用一个套装
   - 可选的更强套装：天武战心、大圣传人、武道圣胎、诸龙血裔
   - 但天武战心、大圣传人找不全十门功法→只能选武道圣胎或诸龙血裔
   - "几百万、上千万的修行花销……对张羽、白真真这样的穷人来说，那已经是他们过去想都不敢想的巨大数字了"
   - **核心表达：** 穷人想追上富人，需要的钱是一个天文数字

4. **福姬揭露正神系统本质**（第59-90行）【本章核心设定曝光】
   - "八部的正神若想要提升评级，想要增长神力，想要精进符篆，那也都需要钱"
   - "赚钱，或者说从全昆墟上下回收财富，这是所有正神的一大使命"
   - "当初十大宗门统合昆墟三十六层……深感昆墟管理不易，所以创造了八部正神，塑造了一批为他们管理天下的正神"
   - **核心设定释放：** 整个正神体系是十大宗门发钱→正神收钱的循环系统
   - **对破局的意义：** 邓丙丁作为六等神，也需要赚钱考核→给她一个从周家赚钱的机会=让她咬上周家

5. **破局方案雏形**（第70-77行）
   - 张羽已加邓丙丁好友→周家不知情→信息差
   - 福姬："若能利用好这位正神的力量，足够让周家吃个大亏"
   - 白真真担忧：正神不好利用
   - 福姬："不需要把周家引去调查筑基考试，只需要给这位六等游神一个插手此事的机会就行了"
   - **核心策略：** 不碰考试红线→只给正神赚钱的抓手→让周家自己撞上来

## 三、人物设计

### 张羽 —— 面临金钱焦虑但开始布局的穷人主角

| 维度 | 分析 |
|------|------|
| **短期目标** | 三个月内将道心、法力、肉体强度都推到炼气极限 |
| **功法规划** | 高中圣体不够需要更强套装→筛选天武战心/大圣传人/武道圣胎/诸龙血裔→但找不全十门 |
| **金钱需求** | 几百万到上千万 |
| **破局思考** | 利用邓丙丁好友+信息差→借正神之火烧周家 |
| **对福姬的认知** | 终于遇到能问正神秘辛的渠道 |
| **对卖种子的看法** | 虽然"感觉怪怪的"但自我说服："卖种子的钱不是钱吗？" |

### 白真真 —— 共同面对金钱困境的穷人搭档

| 维度 | 分析 |
|------|------|
| **对云景的复盘** | 从"无脑碾压"到"针对弱点的战术压制"→需要更多功法和推演 |
| **金钱需求** | 再花个几百万 |
| **担忧** | 周家24小时监控→没法去做黑暗补课界的事 |
| **已知敌情** | 周豪（72下校毕业大学生=链气巅峰），跟踪者类似水平 |

### 福姬 —— 揭示系统真相的邪神导师

| 维度 | 分析 |
|------|------|
| **建议路线** | 黑暗补课界是穷人最后的赚钱出路 |
| **对正神的认知** | 清楚正神晋升机制：也需要钱来提升评级、神力、符篆 |
| **核心策略** | 不需要正面冲突→给正神一个赚周家钱的"抓手" |
| **感叹** | "十大宗门确实是深谋远虑" |

### 夜凌霄 —— 谨慎警觉的仙都天骄

| 维度 | 分析 |
|------|------|
| **调查方向** | 查昆墟一层其他城市的药物代谢技术 |
| **谨慎** | 打给家族时不透露筑基考试的具体情报 |
| **优先级** | 道解缓释胶囊投资 > 调查张羽（先办正事） |

### 玉星寒 —— 从竞争者到合作伙伴的转变

| 维度 | 分析 |
|------|------|
| **策略** | 加课张羽→学习对方的修炼诀窍 |
| **关系定位** | 从找陪练→到共同进步→到做生意（3万/小时） |
| **关键纽带** | "卖种子的交情"——不可对外人道的隐秘信任 |

## 四、设定细节

### 链法图·套装规划（本章张羽的选择分析）

| 套装名称 | 状态 | 需求 |
|----------|------|------|
| **高中圣体**（当前启用） | 链气期极限代谢，三倍炼体 | 已激活，但不够用 |
| **天武战心** | 更强但找不全十门 | 需要十门功法集齐 |
| **大圣传人** | 更强但找不全十门 | 需要十门功法集齐 |
| **武道圣胎** | 可选 | 可凑齐十门→能完成 |
| **诸龙血裔** | 可选 | 可凑齐十门→能完成 |

### 链法图规则

| 维度 | 内容 |
|------|------|
| **容量** | 只能拖入十门功法 |
| **套装生效条件** | 一个套装需要六门功法才能起效 |
| **终极效果** | 十门功法才能达到终极效果 |
| **限制** | 同一时间段内，只能起效一个套装 |

### 八部正神·经济本质（本章核心设定释放）

| 维度 | 内容 |
|------|------|
| **起源** | 十大宗门统合昆墟三十六层后，创造八部正神代为管理天下 |
| **晋升条件** | 正神要提升评级、增长神力、精进符篆 → 也需要**钱** |
| **核心使命** | 赚钱/回收全昆墟的财富 |
| **考核目标** | 每年每个正神都有赚钱/回收货币的考核指标 |
| **贫富关系** | 邓丙丁是六等四方游神→也需要钱来晋升→给赚钱机会就会咬 |
| **与张羽的关系** | 给邓丙丁从周家赚钱的"抓手"→她自然会上钩 |
| **与考试的关系** | 不能碰考试红线→赚钱可以碰 |

### 保护区的消费陷阱（周家跟踪者）

| 维度 | 内容 |
|------|------|
| **跟踪手段** | 24小时监控，甩脱困难（周豪级别=链气巅峰） |
| **周家策略** | 不正面打→压缩学习、修行、赚钱时间→一步步心态失衡 |
| **学校层面** | 课程改革+额外补课+坐班要求→环环相扣 |
| **对付方案** | 给邓丙丁创收抓手→让周家无暇顾及 |

## 五、套路归档

### 套路名称：卖种子的交情·隐秘信任纽带

**适用场景：** 两个角色因共同参与灰色/羞耻活动而建立不可对外人道的信任

**本章应用：**
- 玉星寒想加课张羽但无法向李雪莲解释理由→总不能说"我和他是一起卖种子的交情"
- 两人共同卖种子（不可对外言说）→反而成为最牢固的信任基础
- **核心表达：** 一起干过坏事的人，比什么都能互相信任

**归档文件：** `写作资产/套路库/卖种子的交情·隐秘信任纽带.md`

---

### 套路名称：正神也要钱·系统剥削的闭环

**适用场景：** 揭示统治体系中的每一个环节都是经济循环的一部分，正神也不是超脱者

**本章应用：**
- "八部的正神若想要提升评级，想要增长神力，想要精进符篆，那也都需要钱"
- 正神的使命：赚钱/回收财富
- 十大宗门发钱→正神收钱→整个昆墟的利润在掌控中
- **核心表达：** 没有人能逃出货币的循环，连神也要为钱打工

**归档文件：** `写作资产/套路库/正神也要钱·系统剥削的闭环.md`

---

### 套路名称：借正神之火·灰色抓手策略

**适用场景：** 不使用正面规则（考试红线），而是利用正神/官员自身利益制造切入点

**本章应用：**
- 张羽已经有一位正神好友但周家不知道→信息差
- 福姬策略：不需要引周家查考试（他们知道水深）→给正神一个赚周家钱的"抓手"
- 正神也需要钱晋升→赚钱机会送到面前自然会上钩
- **核心表达：** 不要指望正义，要利用利益的齿轮

**归档文件：** `写作资产/套路库/借正神之火·灰色抓手策略.md`

---

### 套路名称：套装轮换·功法规划的策略深度

**适用场景：** 角色需要根据当前需求选择不同的功法套装，功法选择成为战略决策

**本章应用：**
- 链法图只能放十门功法→一套六门起效→同时只能用一套
- 高中圣体不够→需要更强的：天武战心/大圣传人/武道圣胎/诸龙血裔
- 找不全十门的被排除→锁定武道圣胎或诸龙血裔
- 但不管哪套都需要几百万→金钱是最终瓶颈
- **核心表达：** 功法不是越多越好，正确的套装策略比堆数量更重要

**归档文件：** `写作资产/套路库/套装轮换·功法规划的策略深度.md`

---

### 套路名称：金钱焦虑·天文数字的绝望

**适用场景：** 主角算清追上富人所需要的成本后，才发现差距比想象中大得多

**本章应用：**
- 白真真："算来算去，感觉接下来怎么也再花个几百万才行"
- 张羽：三个月内炼气极限+更多功法+更强套装→"起码要花几百万，甚至上千万才比较稳妥"
- "几百万、上千万的修行花销……对张羽、白真真这样的穷人来说，那已经是他们过去想都不敢想的巨大数字了"
- **核心表达：** 穷人就算再努力，追平富人所需的成本仍然是一个天文数字

**归档文件：** `写作资产/套路库/金钱焦虑·天文数字的绝望.md`
