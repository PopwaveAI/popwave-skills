# 第205章 财神第一，大学大专 — 设计包 (v3)

---

## 一、核心概念

> **财税第一·正神体系构建史 + 周家施压坐班 + 跟踪者岳金成与司骤雨 + 保护区消费陷阱**
> 福姬揭露八部正神核心秘密：八部之中财税部最早创立，亿财海生大神（财神）是神中之神。十大宗门发币→正神收钱的循环是控制昆墟的关键。货币规模失衡会引发天地量劫。回到嵩阳市后，巡查队长云霓通知张羽白真真必须坐班——周家进一步施压。跟踪者岳金成（72下校毕业大学生）与司骤雨（20年没升职的大专生）持续监控。张羽白真真利用氧气瓶进入保护区，逼跟踪者一起进入——保护区的呼吸费就是第一波杀伤。

**本章使命：**
1. **财税部体系完整揭露**：八部之中财税第一，亿财海生大神是神中之神
2. **天地量劫概念引入**：货币规模失衡的灾难性后果
3. **周家坐班施压**：系统性压缩张羽白真真的时间
4. **岳金成形象展开**：72下校毕业大学生的优越感vs职场困境
5. **大专生歧视**：大学生vs大专生的微妙鄙视链
6. **保护区消费陷阱**：利用氧气瓶逼跟踪者烧钱

## 二、情节框架

### 主视角（上帝视角/福姬讲述→回到嵩阳→跟踪者视角→保护区行动）

| 段落 | 内容 | 核心信息 |

| **坐班施压(B段，18-33行)** | 云霓通知必须坐班→张羽询问原因→云霓回"周"字→福姬解读施压策略 | 周家不正面打，一步步恶心 |
| **跟踪者C位展开(C段，34-68行)** | 岳金成监控张羽→优越感vs无聊→司骤雨20年没升职→大专生vs大学生鄙视链→岳金成想主动动手被拒 | 反面角色的世界观呈现 |

### 关键节点

1. **福姬讲述八部起源**（第1-17行）【本章核心设定释放】
   - **财税部第一：** "八部之中，第一个被仙人们创立出来的便是财税部"
   - **亿财海生大神（财神）：** "神中之神，神上之神"
   - **发币机制：** 各层各派货币都由财神所发
   - **天地量劫：** "一旦其中出现问题，便可能引发一场天地量劫……不知道多少人跳楼，多少豪门崩塌，多少公司陨落"
   - **不死真仙：** "跳出三贷之外，不在五险之中，甚至能自行发市的无漏真仙们总是能避过"
   - **最终循环：** "一手发钱，一手收钱，整个昆墟三十六层的利润便尽在掌握之中"
   - **福姬的感叹：** "亿兆生灵的生死存亡、福祸穷富都在他们的一念之间"
   - **核心表达：** 货币=终极控制力，修仙世界的金融帝国

2. **周家坐班施压**（第18-33行）
   - 云霓通知："上头突然要查员……下周开始便要每天坐班"
   - 张羽询问原因→云霓回一个"周"字
   - 福姬解读：周家不正面打→一步步压缩学习/修行/赚钱时间→心态失衡→成绩下落
   - **核心表达：** 控制你的时间，就是控制你的人生

3. **跟踪者正面展开**（第34-68行）【本章人物亮点】
   - **岳金成：** 72下校毕业大学生（昆墟一层出身的大学毕业生），想留二层但困难，珍惜工作
   - **日常：** 日夜住公司、自费伙食和过夜费、追随领导前后、不放过加班
   - **心态变化：** 从认真监控到百无聊赖→"我看着你修行，算怎么个事情？"
   - **想主动动手：** 盗走手机/潜入公寓→被司骤雨拒绝（领导安排不可擅作主张）
   - **对司骤雨的鄙视：** "20多年没升职"+"只是大专生"→"我岳金成才是真正的大学生"
   - **司骤雨：** 20年没升职（金州学院大专毕业），听小说监控，懂得不越界
   - **核心表达：** 跟踪者也有自己的职场困境和阶级焦虑

4. **保护区消费陷阱**（第69-92行）
   - 张羽反常坐地铁（平时跑步省钱）→引岳金成跟踪
   - 岳金成换装跟踪（筋肉扭动变样貌+背包换衣物）
   - 张羽白真真双双进保护区→戴氧气面罩（预先准备）
   - 岳金成司骤雨被迫跟进→呼吸就要花钱
   - "看着自己身上的消费记录随着每一口呼吸而不断增长，涨得他们心头滴血"
   - **核心表达：** 利用跟踪者的消费成本反制——让他们每一口呼吸都在花钱

## 三、人物设计

### 福姬 —— 揭示系统终极秘密的邪神

| 维度 | 分析 |
|------|------|
| **讲述内容** | 八部起源、财税第一、财神地位、天地量劫、收币循环 |
| **知识储备** | 连十大宗门创立正神的原始动机都知道 |
| **态度** | 羡慕且向往——"亿兆生灵的生死存亡都在他们的一念之间" |
| **对张羽白真真的指导** | 讲理论的同时服务叙事——为借正神之火提供理论基础 |

### 张羽 —— 开始主动布局的穷人主角

| 维度 | 分析 |
|------|------|
| **应对坐班** | 不去（落入周家节奏）→要按自己的节奏利用信息差 |
| **行动** | 用地铁引开跟踪者→预设氧气瓶→进保护区烧跟踪者的钱 |
| **对正神的理解** | 通过福姬讲述明白了：正神也需要钱 |
| **策略雏形** | 给邓丙丁一个从周家赚钱的机会 |

### 白真真 —— 配合行动的穷人搭档

| 维度 | 分析 |
|------|------|
| **对坐班的态度** | 生气但不慌→"这周家到底想干什么？打又不打，就一步步恶心我们？" |
| **行动** | 配合张羽进保护区，同样戴氧气面罩 |
| **对正神的理解** | "作为一个高中生的她虽然没能听懂全部，但却明白了一件事情——赚钱/货币回收对正神来说很重要" |

### 岳金成 —— 底层大学生的职场困境

| 维度 | 分析 |
|------|------|
| **学历** | 72下校毕业（昆墟一层出身，正经大学生） |
| **工作** | 万星集团安保部，入职一年不到 |
| **心态** | 珍惜工作→认真监控→日复一日→百无聊赖→想主动立功 |
| **优越感** | "大学生"看不起"大专生"司骤雨 |
| **困境** | 二层的大学生越来越多→一层毕业生想留二层千难万难 |
| **具体行为** | 日夜住公司、自费伙食、交过夜费、追随领导、不放过加班——典型的底层打工人 |
| **对张羽的态度** | 观察久了感觉浪费时间→"我这天天看着你修行，又算怎么个事情？" |

### 司骤雨 —— 二十年没升职的大专生

| 维度 | 分析 |
|------|------|
| **学历** | 金州学院毕业（被岳金成归类为"大专"而非"大学"） |
| **工龄** | 20多年没升职加薪 |
| **日常** | 一边听小说一边监控白真真 |
| **职业认知** | 知道领导安排不可擅作主张→拒绝岳金成的主动动手提议 |
| **对保护区的认知** | 知道报销会很慢→但还是要跟进 |

## 四、设定细节

### 八部正神·财税部设定（本章核心设定释放）

| 维度 | 内容 |
|------|------|
| **八部排序** | 财税部为八部之中第一个被仙人们创立出来的部门 |
| **财神名号** | 亿财海生大神 |
| **财神地位** | "神中之神，神上之神" |
| **财富职能** | 昆墟各层各派的货币都由财神发行 |
| **控制逻辑** | 发币→控制各层货币规模和流通→维持昆墟稳定 |
| **天地量劫** | 货币规模失衡→公司战争→生灵涂炭→不知道多少人跳楼 |
| **无漏真仙** | 跳出三贷之外，不在五险之中，能自行发市→总能在量劫后重整昆墟 |
| **收币机制** | 正神回收天下财富→一手发钱一手收钱→尽在掌握 |
| **最终结论** | "亿兆生灵的生死存亡、福祸穷富都在他们的一念之间" |

### 周家施压链

| 维度 | 内容 |
|------|------|
| **第一步** | 学校课程改革 → 影响学习进度 |
| **第二步** | 额外补课要求 → 挤压修行时间 |
| **第三步** | 坐班要求 → 压缩一切可自由支配时间 |
| **目标** | 一步步压缩学习/修行/赚钱的时间 |
| **效果预期** | 心态失衡→成绩下落→修行倒退 |
| **张羽的反制** | 不去坐班（不落入节奏）→按自己节奏用信息差反击 |

### 保护区的消费机制

| 维度 | 内容 |
|------|------|
| **收费方式** | 呼吸就要花钱（空气质量付费） |
| **氧气瓶的作用** | 张羽白真真预先准备好的→在保护区内不给保护区供氧系统赚钱 |
| **跟踪者的困境** | 跟进保护区→必须大口呼吸→每一口都烧钱 |
| **报销问题** | 司骤雨想"公司会报销吧"但又想"多久报销就不好说了" |
| **岳金成的心态** | 看着消费记录"涨得心头滴血" |

### 岳金成的跟踪手段

| 维度 | 内容 |
|------|------|
| **伪装术** | 筋骨扭动改变身形+背包中衣物更换样貌 |
| **每次靠近** | 总能变换一副新样貌 |
| **跟踪路线** | 多条铁路线换乘→最终跟到郊区 |
| **实力** | 链气巅峰（72下校毕业大学生） |

### 大专vs大学的社会歧视（本章社会设定）

| 维度 | 内容 |
|------|------|
| **岳金成的身份** | 72下校毕业=真正的大学生 |
| **司骤雨的身份** | 金州学院毕业=被看不起的"大专" |
| **岳金成的信念** | 过一两年自己职级一定在司骤雨之上（因学历优势） |
| **现实** | 司骤雨20年没升职→能力/机遇问题→大专学历可能也有影响 |
| **社会背景** | 昆墟二层大学生越来越多→一层大学毕业生贬值→竞争白热化 |

## 五、套路归档

### 套路名称：财税第一·货币即权力的世界法则

**适用场景：** 揭示虚构世界中金融/货币体系的终极控制力

**本章应用：**
- 八部之中财税部第一，亿财海生大神是神中之神
- 发币→控制货币规模→维持稳定
- 天地量劫：货币失衡→公司战争→生灵涂炭
- "一手发钱，一手收钱……亿兆生灵的生死存亡都在他们的一念之间"
- **核心表达：** 拳脚功法是表象，货币才是最根本的武器

**归档文件：** `写作资产/套路库/财税第一·货币即权力的世界法则.md`

### 套路名称：天地量劫·经济危机的修仙化

**适用场景：** 用修仙术语描述的周期性经济危机，跳楼的不是真仙而是底层

**本章应用：**
- 十大宗门发币控制不好→货币规模失衡→天地量劫
- "不知道多少人跳楼，多少豪门崩塌，多少公司陨落"
- 无漏真仙（已跳出三贷五险）总能避过→量劫后出来重整昆墟
- **核心表达：** 经济危机中，只有顶层能全身而退

**归档文件：** `写作资产/套路库/天地量劫·经济危机的修仙化.md`

### 套路名称：时间压缩战·系统性的慢性施压

**适用场景：** 反派不正面攻击，而是耗时耗力压缩主角的可支配时间，逐步消耗心态

**本章应用：**
- 周家三步走：课程改革→补课要求→坐班要求
- 目标：压缩学习/修行/赚钱时间→心态失衡→成绩下落
- "不可能老老实实去坐班"→不能落入对方节奏
- **核心表达：** 控制一个人的时间，比直接打他一顿更致命

**归档文件：** `写作资产/套路库/时间压缩战·系统性的慢性施压.md`

**归档文件：** `写作资产/套路库/跟踪者的职场困境·底层反派的职场悲歌.md`
