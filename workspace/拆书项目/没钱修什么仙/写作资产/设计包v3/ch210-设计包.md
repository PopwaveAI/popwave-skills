# 第210章 封锁万星集团 — 设计包 (v3)

---

## 一、核心概念

> **正神封锁万星集团大楼 + 公司人被扣工资恐慌 vs 武装人员飞剑威慑 + 黄子丑审讯岳金成 + 嵩阳市各方对这件事的解读与影响**
> 万星集团大楼被神力光幕封锁。公司人因离岗扣钱而骚动，武装人员飞剑威慑。安保部长安抚员工「算下班时间、不计绩效」。黄子丑投影降临，直接锁定岳金成和司骤雨审讯——揭示整件事因张羽、白真真而起。周澈尘收到父亲消息震惊。云霓分析：六等正神出手≠为张羽撑腰，而是「逮着机会从万星集团和周家身上啃下一块肉」。各方对事件的解读各不同：知情人不敢说、不知情人传「张羽有正神撑腰」。

**本章使命：**
1. **正神封锁大楼**：神力光幕包裹整栋万星集团大楼的壮观场面
2. **公司人的工资焦虑**：离岗扣钱vs飞剑威慑的荒诞对比
3. **黄子丑审讯**：直接点破「监视张羽和白真真多久了」，揭示事件核心
4. **周澈尘的震惊**：从校园霸凌者到意识到卷入六等正神级别事件
5. **嵩阳市各方解读**：知情人沉默vs不知情人传谣，事件在信息差中的变形

## 二、情节框架

### 主视角（上帝视角——多线穿梭）

| 段落 | 内容 | 核心信息 |

| **安保部长安抚(B段，15-32行)** | 部长宣布「算下班时间、不计绩效」，打探消息无果，自己猜测公司出了什么大事 | 公司内部的信息真空与猜测 |
| **黄子丑审讯(C段，33-54行)** | 黄子丑投影降临，锁定岳金成+司骤雨，直接问「监视张羽和白真真多久了」→付款记录揭穿→神力压制 | 正神的精准打击 |
| **正神行动的保密与波及(D段，55-65行)** | 万星集团股东打探消息无果，保密等级极高；从审问信息中发现了张羽、白真真的名字 | 正神信息的严格保密 |

| **周澈尘的震惊(F段，84-106行)** | 周澈尘从修行中接到父亲消息→万星被封锁→家里被调查→六等正神→周豪传话：你叔叔主导的，但有些事需要你去做 | 周家内部的切割与应急 |

### 关键节点

1. **神力封锁下的公司人荒诞剧**（第1-15行）
   - 神力光幕封锁大楼，100多名安保人员被两名武装人员看管
   - 公司人不怕飞剑，怕的是「离开工作岗位太长时间，要扣钱了」
   - 安保部长举起双手安抚：「算下班时间，不计绩效，空缺时长以后加班补」
   - 「被飞剑捅穿了，有钱赔吗？」——公司人连死在飞剑下都关心能不能赔钱
   - **核心表达：** 在昆墟公司人的世界里，工资和绩效考核比武器更有威慑力

2. **安保部长的猜测**（第15-32行）
   - 对武装人员点头哈腰：「我们和嵩阳市巡察总队关系一直很好」
   - 「这几年一直都是版权执法的合作单位」
   - 正神回答：「连我们都不知道是干什么」
   - 安保部长自己猜测：「难道公司暗中盗版其他公司的功法？还是走私？」
   - 自我安慰：「反正肯定是上面大领导为了提高利润作死了」
   - **核心表达：** 公司底层永远不知道上层在干什么——靠猜测和自慰生存

3. **黄子丑的精准审讯**（第45-54行）
   - 「你们两个监视张羽和白真真多久了？」
   - 直接展示付款记录截图——「形象可变，付款记录骗不了人」
   - 神力将两人死死压在墙上
   - 「这次的事情闹得很大，已经引起了六等上神的关注」
   - 岳金成、司骤雨震惊：两个穷鬼学生怎么就引来六等神的插手了？
   - 黄子丑内心：「唉，连好感符都不会，还大学生呢？」
   - **核心表达：** 正神层面的高效——证据确凿、精准锁定、不容抵赖

4. **正神保密等级的信息真空**（第55-65行）
   - 万星集团股东动用一切人脉打探不到消息
   - 「甚至向其他神职人员乃至是正神去打探，也得不到丝毫消息」
   - 保密等级高得「远远出乎了他们的预料」
   - 但能知道：正神一直在查「张羽」「白真真」
   - **核心表达：** 正神体系的保密能力让公司势力完全无法渗透

5. **云霓的政治解读**（第66-83行）
   - 云霓的判断：「是因为筑基考试吧？」
   - 「但周家胆子再大，应该也不敢插手筑基考试」
   - 更深的判断：「那位六等的正神逮着机会，要从万星集团和周家身上啃下一块肉」
   - 对张羽、白真真的新定位：「难道他们两人也和张翩翩一样，背后有某位大神的撑腰？」
   - 对事件影响的预测：
     - 不知情人→六等正神为张羽撑腰
     - 知情人→闭紧嘴巴，最多说「张羽背景很大」
     - 周家→咬死是普通治安事件，喂饱正神，配合张羽要价
   - **核心表达：** 云霓的视角揭示了正神出手的本质——借题发挥的权力游戏

6. **周澈尘的震动与周家切割**（第84-106行）
   - 周澈尘从修行中被电话震醒
   - 父亲消息：「万星集团被正神封锁」「家里也被神部调查」「六等正神」
   - 「张羽、白真真的靠山，竟然是六等正神？」
   - 周豪传话：「他知道这次的事情是您叔叔主导的，和您没有关系」
   - 「但毕竟被人抓了口实，有些事情还是需要您去做」
   - **核心表达：** 周家在正神压力下开始内部切割——叔叔背锅，周澈尘配合善后

## 三、人物设计

### 黄子丑（九等功曹小神） —— 执行高效的正神代表

| 维度 | 分析 |
|------|------|
| **出场** | 神光落下，投影浮现，目光扫过全场→瞬间识海读取资料 |
| **精准锁定** | 直接点出岳金成、司骤雨，跳过所有人 |
| **审讯风格** | 证据直给（付款记录）→神力压制→威胁升级到六等上神 |
| **内心** | 吐槽大学生不会用好感符 |
| **手段** | 现场拜神（众人行礼），神力镇压，多维度攻击 |

### 云霓 —— 成熟的政治分析者

| 维度 | 分析 |
|------|------|
| **判断1** | 正神出手≠为张羽撑腰→是以调查为名放血万星集团 |
| **判断2** | 周家→咬死是普通治安事件，配合正神和张羽 |
| **判断3** | 张羽、白真真可能和张翩翩一样有大神撑腰 |
| **判断4** | 正神保密等级决定信息传播方式 |
| **行动** | 不参与，纯观察分析 |

### 周澈尘 —— 从霸凌者到家族棋子的转变

| 维度 | 分析 |
|------|------|
| **初始状态** | 打坐修行，心想「个人在公司面前如此渺小」 |
| **收到消息** | 父亲电话→六等正神封锁万星→震惊 |
| **认知转变** | 「怪不得两人能这么快崛起」「六等正神恐怕也是张翩翩背后的存在」 |
| **最终** | 周豪传话：叔叔背锅，你去做善后 |
| **身份认知** | 从「周家少爷」→「家族棋子」 |

### 安保部长 —— 公司中层的信息真空者

| 维度 | 分析 |
|------|------|
| **表层** | 向武装人员点头哈腰，维护秩序 |
| **内心** | 不知发生了什么，只能瞎猜 |
| **自我安慰** | 「一定是上面大领导作死了」——中层标准心态 |

### 岳金成、司骤雨 —— 从执行者到被捕者

| 维度 | 分析 |
|------|------|
| **岳金成** | 从傲慢到恐惧→做梦也没想到两个穷学生引出六等正神 |
| **司骤雨** | 同样震惊，但不敢乱动 |
| **共同** | 被神力压制在墙上，完全无法反抗 |

## 四、设定细节

### 神力封锁机制

| 维度 | 内容 |
|------|------|
| **封锁范围** | 万星集团整栋大楼 + 所有进出要道 |
| **封锁形式** | 神力形成警告光幕（闪烁、悬浮半空） |
| **包裹方式** | 将整栋大楼都包裹起来 |

### 正神的保密与信息管控

| 层级 | 可得知信息 |
|------|-----------|
| **外界公司** | 完全不知道发生了什么 |
| **万星集团股东** | 只能查到「正神在查张羽、白真真」 |
| **巡察队队长云霓** | 结合已知信息（离城时间）猜出筑基考试相关 |
| **周家核心** | 知道正神出手，但不知具体由头 |

### 公司人的工资焦虑层次

| 心态 | 内容 |
|------|------|
| **表层恐惧** | 被飞剑捅穿有没有钱赔？ |
| **深层恐惧** | 离开工作岗位扣钱→要借钱上班 |
| **终极恐惧** | 公司会不会没了→失业 |

### 周家的内部应对（切割与善后）

| 维度 | 内容 |
|------|------|
| **定性方向** | 咬死是普通治安事件（不是调查筑基考试） |
| **背锅** | 周阳（叔叔）主导，周澈尘无关 |
| **善后** | 喂饱正神 + 配合张羽要价 + 周澈尘去做某些事 |

## 五、套路归档

### 套路名称：扣钱比飞剑更可怕·公司人的生存逻辑

**适用场景：** 在危险面前公司员工最恐惧的不是死亡，而是被扣工资/绩效

**本章应用：**
- 两名武装人员+两把军用级飞剑看管上百人
- 公司人不怕飞剑怕扣钱：「离开工作岗位太长时间，要扣钱了」→「被飞剑捅穿了，有钱赔吗？」
- 安保部长：「算下班时间，不计绩效」→全体松一口气
- **核心表达：** 在昆墟的公司体系中，经济的惩罚比武力更有效——因为死亡是一时的，债务是一辈子的

**归档文件：** `写作资产/套路库/扣钱比飞剑可怕·公司人生存逻辑.md`

---

### 套路名称：正神拷问·付款记录胜于一切伪装

**适用场景：** 公司人以为改换身形、变化容貌就能隐形，但正神直接通过付款记录锁定

**本章应用：**
- 岳金成、司骤雨一路改换身形变化容貌
- 黄子丑直接展示付款记录——付费记录骗不了人
- 神力压制+证据确凿+不容抵赖
- **核心表达：** 在数字化时代，追踪人的最好方法不是看脸，而是看钱

**归档文件：** `写作资产/套路库/正神拷问·付款记录锁定.md`

---

### 套路名称：信息差的多层解读·权力游戏的舆论泡沫

**适用场景：** 同一件事在不同信息层级中传播出截然不同的版本

**本章应用：**
- 不知情人：六等正神为张羽撑腰，背景极大
- 知情人（筑基考试相关）：闭紧嘴巴不透露丝毫消息
- 云霓：正神出手是在放血万星集团
- 周家：必须咬死普通治安事件
- **核心表达：** 事件本身是什么不重要，重要的是不同层级的人需要它是什么

**归档文件：** `写作资产/套路库/信息差的多层解读·权力游戏舆论泡沫.md`

---

### 套路名称：家族切割·叔叔背锅少爷善后

**适用场景：** 家族势力在外部压力下进行内部切割——让最直接的操作者背锅，核心血脉脱身

**本章应用：**
- 周豪传话：「他知道是您叔叔主导的，和您没有关系」
- 「但毕竟被人抓了口实，有些事情还是需要您去做」
- 周阳从操控者变背锅者，周澈尘从参与者变善后者
- **核心表达：** 家族在面对危机时，第一原则不是共同承担，而是最小化核心损失——牺牲一个人保全家

**归档文件：** `写作资产/套路库/家族切割·叔叔背锅少爷善后.md`

---

### 套路名称：借题放血·正神的权力游戏

**适用场景：** 正神利用高阶权限，以某个合法借口（如筑基考试保密）对公司进行经济掠夺

**本章应用：**
- 云霓解读：正神不是为张羽撑腰，而是「逮着机会从万星集团和周家身上啃下一块肉」
- 正神行动保密等级极高→公司无法打探→只能任人宰割
- 周家的应对：喂饱正神+配合张羽要价
- **核心表达：** 在正神层面，事件本身只是一个切口——真正的目的是利益的再分配

**归档文件：** `写作资产/套路库/借题放血·正神的权力游戏.md`
