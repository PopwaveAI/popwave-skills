# 第212章 张羽的丰厚收获 — 设计包 (v3)

---

## 一、核心概念

> **张羽和白真真向周澈尘展开资源索赔 + 现金150万/人 + 学校全面特权 + 功法全开放 + 怨灵根入手 + 周澈尘被逼出家族灵根清单 + 张羽选中怨灵根为武道圣胎做准备**
> 周澈尘从10万、50万一路加码到150万/人（含岳金成、司骤雨的50万/人），另提供学校全面特权：自由上课、武道老师陪练、灵根免费100小时/月、灵界培训名额。张羽在福姬提醒下索要全部功法版权，并挑中怨灵根——为修怨煞功组武道圣胎做准备。

**本章使命：**
1. **索赔额度层层加码**：10万→50万→100万→150万（含员工赔偿）
2. **学校全面特权**：自由作息、武道陪练、灵根免费、灵界名额
3. **功法全开放**：张羽点名要嵩阳高中所有功法版权——为武道圣胎铺垫
4. **怨灵根入手**：张羽从周家灵根清单中选中怨灵根——因为怨煞功便宜且能组武道圣胎
5. **福姬的连续提醒**：灵根要定好种类、功法版权必拿——战术参谋的全流程指导

## 二、情节框架

### 主视角（张羽/上帝视角）

| 段落 | 内容 | 核心信息 |
|------|------|----------|
| **层层加码的赔偿谈判（A段，1-26行）** | 周澈尘从10万→50万→100万→150万（含员工50万/人）→张羽不动声色内心期待 | 索赔价码一路飙升的过程 |
| **学校全面特权（B段，27-43行）** | 自由上课→武道陪练→灵根免费100小时→灵界名额→周澈尘内心盘算成本 | 张羽和白真真各自看中不同的资源 |
| **索要功法版权（C段，44-52行）** | 福姬提醒→张羽要求嵩阳高中所有功法→含高二高三的专家级功法→为武道圣胎做准备 | 张羽的隐藏目标：组武道圣胎 |
| **灵根挑选（D段，53-82行）** | 周澈尘列出学校天灵根→张羽不满意→周澈尘被迫拿出家族灵根→张羽选中怨灵根→福姬识破周澈尘隐瞒真实用法 | 怨灵根——死者怨念+生者怨念的真正用途 |

### 关键节点

1. **10万到150万的加码过程**（第1-26行）
   - 张羽"别跟我谈钱"的姿态→逼迫周澈尘不断加码
   - 10万→50万→100万+员工各50万→张羽和白真真各150万
   - 周澈尘内心憋屈："他是怎么也没有想到，一年前在他眼中不过是蝼蚁般的两个穷鬼……"
   - 周澈尘的自我安慰："这世道拼的终究是背景，周家还是太弱了"
   - **核心表达：** 拥有了正神撑腰，穷人可以在谈判桌上碾压豪族少爷

2. **学校全面特权清单**（第27-43行）
   - 自由上课时间：不必遵守学校纪律
   - 武道老师一对一旁听：白真真心动"和练气巅峰的武道老师天天实战"
   - 灵根免费100小时/月：张羽动心"各种灵根的加持永远不嫌多"
   - 灵界培训资格：张羽需要"锻造道心"
   - 周澈尘内心盘算：占用蓝岭和自己的灵界名额→转嫁给蓝岭
   - **核心表达：** 学校资源本质上是周家的私人领地——他可以把公共资源当人情送给张羽

3. **福姬的连续战术提醒**（第47-55行）
   - 提醒1："你不是一直想多学几门功法吗？让他把嵩阳高中有教学权的所有功法都交出来。"
   - 张羽内心：其中包含组成武道圣胎和诸龙血裔的功法
   - 提醒2："灵根要规定好是什么灵根……他如果用下级灵根来敷衍你们，那就更对你们完全没用"
   - **核心表达：** 福姬的存在让张羽的谈判从"被动要钱"升级为"主动要资源"

4. **灵根挑选与怨灵根**（第56-82行）
   - 学校天灵根：元阳灵根（处男元阳转化法力）→张羽不满
   - 元阳灵根的恶心点："经过一代代前辈把玩，沾染了成千上万名学生的元阳精气"
   - 周澈尘被迫拿出家族灵根
   - 张羽选中怨灵根→为了修怨煞功（武道圣胎所需）
   - 怨灵根真相：能提取生者怨念，学校/公司是怨念最多的地方
   - 周澈尘隐瞒真实用途→被福姬识破
   - 学校合同细节：学生怨念理论上学校拥有第一处置权
   - **核心表达：** 灵根不只是修炼工具，更是决定你能走哪条路的关键资源

5. **周澈尘的自我觉醒**（第6-9行）
   - "据说上面那些真正的有钱人，他们精心炼制的血脉后裔……一生下来就可以是信仰等级5级的信徒"
   - "这世道拼的终究是背景，周家还是太弱了"
   - **核心表达：** 即使在嵩阳市豪族看来，上面还有更恐怖的力量——阶级永无止境

## 三、人物设计

### 张羽 —— 不动声色的资源猎人

| 维度 | 分析 |
|------|------|
| **表面姿态** | 不动声色、视金钱如粪土——等对方加价 |
| **内心活动** | 150万到手后已经开始期待用途，但脸上完全看不出 |
| **目标明确** | 功法→为武道圣胎、灵根→为怨煞功 |
| **与福姬配合** | 福姬提醒→立即执行→形成高效谈判组合 |
| **谈判成果** | 150万现金+功法全开放+怨灵根+学校全面特权 |

### 周澈尘 —— 被逼到极限的豪族少爷

| 维度 | 分析 |
|------|------|
| **加码过程** | 10万→50万→100万+员工各50万→被迫一步步加价 |
| **内心活动** | "他是怎么也没有想到……蝼蚁般的两个穷鬼"→自我安慰"这世道拼的终究是背景" |
| **让步原因** | 想到父亲的嘱咐、家族资源投入→"昂着的脖子终于是断了" |
| **盘算策略** | 让蓝岭背锅灵界名额→让岳金成司骤雨打工还钱→转移损失 |
| **对张羽的印象** | "这家伙这次的胃口一定很大"——被吃定了 |

### 白真真 —— 精准狙击价格目标

| 维度 | 分析 |
|------|------|
| **谈判风格** | "你以为有钱就能为所欲为，就能犯法了吗？"——义正辞严 |
| **心动目标** | 武道老师一对一陪练——为与云景的战斗做准备 |
| **与张羽配合** | 张羽唱白脸（视金钱如粪土），白真真唱红脸（明确表达不满） |

### 福姬 —— 幕后战术军师

| 维度 | 分析 |
|------|------|
| **连续提醒** | 功法版权→灵根种类→怨灵根真相→逐一点破周澈尘的隐瞒 |
| **对灵根的认知** | 周澈尘说怨灵根只能提取死者怨念→福姬识破："他诓你呢" |
| **战术价值** | 让张羽从被动宰人升级为主动收割资源 |

## 四、设定细节

### 嵩阳高中资源特权清单

| 特权项目 | 内容 | 值多少 |
|----------|------|--------|
| **自由上课时间** | 通识课、武道课、道术课等一律随意进出 | 相当于免修资格 |
| **武道老师陪练** | 任何一位武道老师免费陪练一定时间 | 白真真目标 |
| **灵根免费租赁** | 每月100小时（含周家家族灵根） | 张羽目标 |
| **灵界培训资格** | 学校灵界培训名额 | 张羽需要锻造道心 |
| **功法全开放** | 所有嵩阳高中的功法版权 | 张羽为武道圣胎/诸龙血裔 |

### 天灵根清单

| 灵根 | 效用 | 张羽态度 |
|------|------|----------|
| **元阳灵根** | 炼化处男元阳精气为法力 | 嫌恶心——"一代代前辈把玩" |
| **其他天灵根** | 提升法力/肉体强度为主 | 看不上——张羽快链气巅峰了 |
| **周家死灵根** | 收摄死者死气 | 需要大量尸体资产 |
| **周家冬灵根** | 吸收冻气，改变法力属性 | 需要低温冷冻仓 |
| **怨灵根** | 提取死者怨念→也能提取生者怨念 | 选中——为怨煞功+武道圣胎 |

### 怨灵根详解

| 维度 | 内容 |
|------|------|
| **外形** | 工牌模样——便于存储怨气 |
| **主要用途** | 提取死者怨念——市场上死气更值钱 |
| **隐藏用途（福姬揭穿）** | 也能提取生者怨念——学校/公司才是怨念最多的地方 |
| **价值** | 怨煞功因为门槛高（需要怨气）价格一路下跌至50万——完美配合 |
| **获取代价** | 免费100小时/月中的名额 |

### 学校合同霸王条款

| 条款内容 | 说明 |
|----------|------|
| **学生怨念第一处置权** | 学生在校期间产生的怨念、死气、排泄物等修炼资源，学校拥有第一处置权 |
| **免除方式** | 周澈尘同意免除张羽和白真真的这部分费用 |

### 周澈尘的成本转移策略

| 转移对象 | 内容 |
|----------|------|
| **蓝岭** | 灵界名额让给张羽→周澈尘让蓝岭出80万现金补偿 |
| **岳金成、司骤雨** | 公司垫付50万赔偿→两人打工还钱 |
| **学校公共资源** | "空闲的灵根放着也是放着"→免费借给张羽和白真真 |
| **老师** | "多排几节指导张羽、白真真的课程"→成本由学校承担 |

## 五、套路归档

### 套路名称：索赔层层加码·从十万到百万的谈判艺术

**适用场景：** 掌握对方把柄后，通过谈判逐步提高索赔额度

**本章应用：**
- 周澈尘从10万一路加价到150万/人（含员工赔偿）
- 张羽和白真真一唱一和，一个唱高调一个压价
- **核心表达：** 掌握了权力优势，谈判桌上的数字可以无限膨胀

**归档文件：** `写作资产/套路库/索赔层层加码·谈判艺术.md`

---

### 套路名称：资源特权·学校公共资源的私人化

**适用场景：** 豪族子弟将学校公共资源当作自己的私人礼物送人

**本章应用：**
- 周澈尘用学校空闲灵根、老师排课、灵界名额来给张羽好处
- "空闲的灵根放着也是放着"——公共资源被私人支配
- **核心表达：** 在学校这种豪族控制的地方，公共资源=豪族的私人财富

**归档文件：** `写作资产/套路库/资源特权·学校公共资源私人化.md`

---

### 套路名称：怨灵根·隐藏用途的生者怨念

**适用场景：** 一种被广泛认知为某种用途的资源，实际上有更大的隐藏用途

**本章应用：**
- 表面上怨灵根提取死者怨念——死气精纯值钱
- 实际上也能提取生者怨念——学校/公司怨念量远超墓地
- 周澈尘隐瞒真实用途→福姬识破
- **核心表达：** 信息的价值在于你知道的比别人多一层

**归档文件：** `写作资产/套路库/隐藏用途·怨灵根生者怨念.md`

---

### 套路名称：成本转移·把赔偿转嫁给下属

**适用场景：** 负责人在赔偿他人后，把成本转嫁给更低层的员工

**本章应用：**
- 周澈尘出的100万→让岳金成、司骤雨各出50万还债
- 灵界名额→让蓝岭出80万补偿
- **核心表达：** 最终买单的永远是最底层

**归档文件：** `写作资产/套路库/成本转移·赔偿转嫁底层.md`
