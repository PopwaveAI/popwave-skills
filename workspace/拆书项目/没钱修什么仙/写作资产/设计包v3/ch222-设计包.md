# 第222章 满是观众的筑基考试？ — 设计包 (v3)

---

## 一、核心概念

> **周阳的布局 + 张羽前往仙都 + 玉星寒的期待 + 武道圣胎小成效果详解 + 白真真真灵根功法进展 + 到达仙都 + 筑基考试最后一轮惊现观众席**
> 周阳在病床上分析绿洲集团的垄断趋势，决定筑基考试后对张羽和白真真出手。张羽在列车上收到玉星寒的加油，意识到对方不看好自己这趟能赢。检查羽书，武道圣胎第二层的承载能力和超载能力详解。白真真的真灵根将万剑无终诀推演至突破音障的超音速剑招。到达仙都后进入考场，惊讶发现竟然有大量观众——来自昆墟二层的大学代表。

**本章使命：**
1. **周阳的布局**：决定筑基考试后对付张羽和白真真——"正神可不会再那样护着你们了"
2. **玉星寒的态度**：不看好张羽能赢，想明年合作
3. **武道圣胎第二层详解**：承载能力→超限爆发不伤身；超载能力→更恐怖爆发
4. **白真真的真灵根进展**：万剑无终诀推演→超音速剑招；两门新功法专精擂台战
5. **张羽的弱点揭示**：道心8级(89.5%)——相比法力和肉体强度明显欠缺
6. **观众席重大揭露**：筑基考试最后一轮有观众——来自昆墟二层的大学代表

### 关键节点

1. **周阳的考后猎杀计划**（第1-31行）
   - 观察绿洲大厦超越中心大厦→象征绿洲集团超越嵩阳旧豪族
   - 分析趋势：合成食物→合成肉→未来人体器官合成+血脉技术结合
   - 判断各家族争斗是为在绿洲合作中占更多股份
   - 得出结论：周家排名靠后，不如对付张羽和白真真这两个"弱小的个体"
   - 手段：自己或派人出手 / 用邪神信徒的把柄控制
   - **核心表达：** 大势力争时，小人物反而成了捕猎者的目标——因为好对付

2. **玉星寒的轻视与张羽的自信心**（第39-48行）
   - 玉星寒认为张羽考不上→"和那位仙都爷差了一届，从上一轮的筑基考试来看，相互之间的差距不小"
   - 想明年与张羽搞好关系→"错开赛道"
   - 张羽意识到对方想法→但对考试结果有自己的判断
   - "只要拼尽全力，成功的机会绝对不小"
   - **核心表达：** 外界普遍不看好张羽——这为他用实力打脸做了铺垫

3. **武道圣胎第二层效果·核心机制详解**（第49-63行）
   - 链法图最多挂10门功法→高中圣体和武道圣胎只能二选一
   - **承载能力暴涨：** 可以更超限爆发肉体力量，不用担心筋肉骨骼破损→提高爆发上限和持久力
   - **超载能力：** 让爆发"更加恐怖"
   - **内伤自愈暴涨：** 配合之前的外伤自愈→双重保险
   - 引申：买不起医保→昆墟"脆皮强者"的设定——筑基/金丹强者因没钱赔医疗费而"唯唯诺诺"
   - **核心表达：** 武道圣胎小成意味着战斗风格可以激进到近乎自残——因为伤得起、修得快

4. **白真真真灵根的优势展示**（第71-75行）
   - 道心/法力/肉体强度三项全满→链气极限
   - 不用像张羽凑套装买大量功法→省下的钱投道心
   - 功法数量虽少→但真灵根推演让10级效果远超原版
   - 万剑无终诀推演→雷霆法力加速→第12剑突破音障→超音速剑招
   - 两门新功法专精擂台战
   - **核心表达：** 张羽靠数量堆防御，白真真靠质量堆攻击——两条路线各有所长

5. **观众席的震撼揭露**（第83-88行）
   - 厕所→熟悉的网站→传送入考场
   - 五名正神考官 + 一片片观众席
   - 观众形象光怪陆离→卡通、游戏形象、表情包→通过网络远程观看
   - 来自昆墟二层的各个大学代表
   - **核心表达：** 最后一轮不只是考试，更是人才展示会——考得好能直接获得二层大学的青睐

## 三、人物设计

### 周阳 —— 伺机而动的邪神猎人

| 维度 | 分析 |
|------|------|
| **当前处境** | 下半身空空荡荡→躺在病床看窗外→身体状态不佳但头脑清醒 |
| **对局势的判断** | 绿洲集团推动技术革命→合成食物→合成肉→人体器官→血脉技术 |
| **对家族竞争** | 周家在十六豪族排名靠后→正面竞争不过宋家、乐家 |
| **考后计划** | 筑基考试一结束就对付张羽白真真→派人出手或把柄控制 |
| **心态** | "越发有些迫不及待起来"→看到两人任其揉搓摆弄的样子 |

### 张羽 —— 清醒评估自己实力的考生

| 维度 | 分析 |
|------|------|
| **对玉星寒的态度** | 意识到对方不看好自己→但仍然相信自己有机会 |
| **武道圣胎理解** | 承载能力→超限爆发；超载→更恐怖爆发；内伤自愈→双重保险 |

### 玉星寒 —— 精明算计的旁观者

## 四、设定细节

### 绿洲集团的扩张趋势（周阳视角）

| 维度 | 内容 |
|------|------|
| **当前阶段** | 合成食物→合成肉→市场化成功 |
| **下一阶段** | 人体器官合成（预判） |
| **再下一步** | 血脉技术+人体技术结合（仙都名门望族已在做） |
| **最终结果** | 各地方豪族、公司都与绿洲合作→贯穿昆墟1层的巨无霸 |
| **本质** | 当前的争斗不过是为"在未来的合作中占据更多股份" |

### 武道圣胎第二层·战斗机制详解

| 维度 | 内容 |
|------|------|
| **承载能力暴涨** | 可更超限爆发肉体力量，不用担心筋肉、骨骼破损→提高爆发上限和持久力 |
| **超载能力** | 在承载能力基础上进一步→"爆发起来就更恐怖" |
| **内伤自愈暴涨** | 配合之前的外伤自愈暴涨→医保双保险 |
| **链法图限制** | 最多挂10门→高中圣体和武道圣胎只能二选一 |

### 昆墟的"脆皮强者"设定

| 维度 | 内容 |
|------|------|
| **现象** | 筑基甚至金丹强者因修为高导致保费暴涨→买不起医保 |
| **后果** | "唯唯诺诺，见谁都不敢大声说话"——怕冲突后赔不起医疗费 |
| **称号** | "脆皮强者"——一碰就破产，仙道根基不够扎实 |
| **反讽** | 修为越高越脆弱——因为治疗费也越高 |

### 白真真的真灵根功法推演

| 维度 | 内容 |
|------|------|
| **万剑无终诀** | 雷霆法力加速→第12剑突破音障→超音速剑招 |
| **新功法×2** | 推演后极为擅长擂台战→为争夺筑基资格证做准备 |
| **与张羽的对比** | 张羽靠数量（8门功法堆套装）→白真真靠质量（真灵根推演） |

### 筑基考试最后一轮的观众机制

| 维度 | 内容 |
|------|------|
| **观众来源** | 昆墟二层的各个大学代表 |
| **形象** | 卡通、游戏形象、表情包→通过网络远程观看 |
| **目的** | 观摩比赛→回去向学校报告→给优秀考生开出更好待遇 |
| **正神解释** | "让考生在最终考场崭露头角，登上二层时有更多发展机会" |
| **隐藏内容** | 顺便收门票、开赌局→"回点血"（下章揭露） |

## 五、套路归档

### 套路名称：反派布局·考后猎杀计划

**适用场景：** 反派在主角考试期间提前布局，计划考试结束后对主角出手。为考试后的剧情制造紧张感。

**本章应用：**
- 周阳决定筑基考试后对付张羽白真真→"正神不会再护着你们"
- 手段：自己/派人出手 / 把柄控制
- 因周家在豪族中排名靠后→正面对抗不如对付"弱小的个体"
- **核心表达：** 大树底下好乘凉（考试有保护），但考试一结束，阴影里的猎手就会出手

**归档文件：** `写作资产/套路库/反派布局·考后猎杀计划.md`

---

### 套路名称：三属性差距·偏科的主角

**适用场景：** 展示主角在三个核心属性（道心/法力/肉体）上存在不均衡，埋下后期补强的伏笔。

**本章应用：**
- 法力→链气极限✓
- 肉体强度→链气极限✓
- 道心→8级(89.5%)✗
- 原因：钱都砸在功法上凑套装，没法投道心
- **核心表达：** 主角的成长从来不是匀速的——有所取舍才有爆发式突破

**归档文件：** `写作资产/套路库/三属性差距·偏科主角.md`

---

### 套路名称：考试观众·来自上层的目光

**适用场景：** 重要考试/比赛突然引入高层观众，增加场面对角色的压力，同时为优秀表现提供被上层势力看中的机会。

**本章应用：**
- 从未有观众的筑基考试→突然出现二楼大学代表观众席
- 目标：优秀考生能在二层获得发展机会
- 隐藏条件：也是正神"回血"的手段（卖门票+开赌局）
- **核心表达：** 每一次大考都是展示舞台——上层的眼睛一直在看着下面

**归档文件：** `写作资产/套路库/考试观众·上层目光.md`

---

### 套路名称：同伴对比·数量型vs质量型的功法路线

**适用场景：** 展示主角团内不同角色的修行路线差异——一个靠功法数量堆效果，一个靠灵根推演提质变。

**本章应用：**
- 张羽路线：8门功法凑武道圣胎套装→量大但道心被拖累
- 白真真路线：少量功法+真灵根推演→质高且道心先满
- **核心表达：** 修行没有唯一正解——适合自己的路线才是最好的

**归档文件：** `写作资产/套路库/同伴对比·数量型vs质量型功法路线.md`

---

### 套路名称：医保失灵·越强越脆的讽刺

**适用场景：** 揭示修仙世界中的经济陷阱——修为越高，治疗费越贵，保费越高，反而让人更脆弱。

**本章应用：**
- 张羽因修为增长→保费暴涨→买不起医保
- 引出昆墟"脆皮强者"现象→筑基/金丹强者"见谁都不敢大声说话"
- "一碰就破产"→仙道根基不够扎实
- **核心表达：** 在资本主义化的修仙世界，力量带来的是更高的成本而不是真正的自由

**归档文件：** `写作资产/套路库/医保失灵·越强越脆讽刺.md`
