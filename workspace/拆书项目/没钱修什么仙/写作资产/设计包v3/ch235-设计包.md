# 第235章 奖励到帐，暗流涌动 — 设计包 (v3)

---

## 一、核心概念

> **颁发筑基资格证：四位获胜者登场交流 → 圣杰/谈诗珊的震惊 → 圣杰想当绿洲集团的"狗" → "昆墟是巨大狗场"哲学 → 奖励发放：证书+300万无息贷款+专家功法+指定老师 → 邓丙丁的警告 → 暗流涌动的仙都**
>
> 颁奖典礼上，四位筑基资格证获得者齐聚。法力赛道胜者圣杰、道心赛道胜者谈诗珊得知夜凌霄、云景双双落败后极为震惊。圣杰内心曝光——他想代替夜凌霄成为绿洲集团二层爷的"狗"，并揭示了"昆墟是巨大狗场"的生存哲学。邓丙丁发放奖励：筑基资格证、300万无息贷款、专家级功法、指定大学老师，并警告考生不要轻易接受招揽。邓丙丁独处时推演绿洲集团、神部、上方宗门之间的博弈——这场考试的结果只是更大棋局的一角。

**本章使命：**
1. **四位获胜者亮相**：圣杰（法力）、谈诗珊（道心）——与张羽、白真真同台
2. **圣杰的"狗哲学"**：昆墟是巨大狗场→不是把别人当狗就是当别人的狗→学会当狗和训狗才能往上爬
3. **昆墟一层的等级再现**：1层是1.5层的狗→1.5层是2层的狗→"狗场"的层叠结构
4. **云景的悔恨**：输给白真真是因为不够小心→"如果能再战一次，绝对能胜过对方"
5. **奖励发放**：证书、300万无息贷款、专家功法、指定老师
6. **邓丙丁的警告与善意**："不要轻易答应任何条件"+加好友+压制周家→邓丙丁对张羽有特殊关照
7. **更大的棋局**：绿洲集团用筑基考试做宣传武器→一层各地方豪族被整合→神部无反应→背后可能是幽冥宫的游戏

## 二、情节框架

### 主视角（上帝视角——选手+观众+邓丙丁内心穿插）

| 段落 | 内容 | 核心信息 |

| **仙都学生的震惊(B段，12-50行)** | 圣杰发现夜凌霄未到场→得知夜凌霄输给张羽→云景也输给白真真→震惊 | 外地人把仙都爷的筑基证抢了 |
| **圣杰的"狗哲学"(C段，53-77行)** | 圣杰想当绿洲集团二层爷的狗→"只要钱给够，别说当狗了"→昆墟是巨大狗场→底层爬出来的心态 | 仙都底层的生存哲学 |
| **云景的悔恨(D段，40-51行)** | "如果能再战一次绝对能胜过对方"→粗心大意导致失败 | 输给战术而非硬实力 |
| **颁奖典礼(E段，78-107行)** | 邓丙丁讲话→证书颁发→张羽要"带特效"的证书→300万贷款→专家功法→指定老师 | 奖励的逐项释放 |
| **邓丙丁的警告(F段，101-104行)** | "不要轻易答应任何条件"→信息保护取消→大学/公司提前招揽→张羽感觉对方在对自己说 | 正神的特别关照 |
| **更大的棋局(G段，108-117行)** | 绿洲集团+仙都家族联手变革→考试是宣传武器→神部无反应→是顾忌幽冥宫？→大神们已经落子了？ | 考试结果只是冰山一角 |

### 关键节点

1. **圣杰的"狗哲学"全揭露**（第53-77行）
   - 圣杰的家庭背景：家族旁支→住在仙都地面→进入高中前没见过白天
   - "仙都的穷人比任何城市都更多"
   - **昆墟的狗场结构：**
     - 1层的人是一部分1.5层人的狗
     - 1.5层是2层人的狗
     - "昆墟就像是一个无比巨大的狗场——不是把别人当狗，就是当别人的狗"
   - **圣杰的生存准则：** "只有学会当狗和训狗，才能在昆墟一路往上爬"
   - 他想取代夜凌霄成为绿洲集团二层爷的狗→"当狗这件事情从来没有丝毫负担"
   - **核心表达：** 和夜凌霄的"家族背负"不同，圣杰是从更低处爬上来的人——他早已经放弃了任何尊严幻想，把"当狗"看作正常的生存策略

2. **仙都同学对张羽/白真真的态度转变**（第22-51行）
   - 从"外地人"到"超级外地人"到"身价深不可测的外地富豪"
   - 张羽和白真真的破旧衣服→变成了"高深莫测"
   - 成就改变了认知→贫穷的标签暂时被能力的标签覆盖
   - **核心表达：** 在昆墟这个资本世界，你的出身不重要——你能赢多少人决定了他们怎么看你

3. **邓丙丁的多重关注**（第97-108行）
   - 加好友→压制周家→提醒不要轻易答应条件→"总感觉对方说话的时候一直看着自己和白真真"
   - 话是专门对着他们说的→"我也没有塞好感符啊"
   - 张羽一头雾水，但读者知道邓丙丁一直在暗中照顾
   - **核心表达：** 正神世界也不是铁板一块——邓丙丁有自己的立场和棋局

4. **更大的棋局·邓丙丁的推演**（第108-117行）
   - 绿洲集团用筑基考试做宣传武器→即便夜凌霄失败也会换宣传对象
   - 绿洲大规模干涉一层→整合各地方豪族
   - **神部无反应？** 是在顾忌绿洲集团背后幽冥宫的某位？
   - **最终猜想：** "大神们已经在这场考试中落子了"
   - **核心表达：** 看似是高中生的筑基考试，实则是二层势力、宗门势力的棋盘——一场考试的结果影响的是仙都一层的权力格局

## 三、人物设计

### 圣杰 —— 爬出仙都地底的"训狗师"

| 维度 | 分析 |
|------|------|
| **出身** | 家族旁支→住在仙都地面→高中前没见过白天 |
| **对仙都的评价** | "仙都的穷人比任何城市更多——因为这里是竞争最激烈的地方" |
| **狗哲学** | "昆墟就是一个巨大的狗场——不是把别人当狗，就是当别人的狗" |
| **生存准则** | "只有学会当狗和训狗，才能在昆墟一路往上爬" |
| **当狗的态度** | "只要钱给够→别说当狗了→就算被狗**他都能**到对方破产" |
| **野心** | 想取代夜凌霄→成为绿洲集团二层爷的狗→"
| **对张羽的判断** | 以为是身家深不可测的富家子→脾气可能比夜凌霄还差→自己有机会钻空子 |
| **核心精神** | 从最底层爬上来的实用主义者——尊严是奢侈品，生存才是一切 |

### 白真真 —— 憋着高兴的胜利者

| 维度 | 分析 |
|------|------|
| **表面** | 想装高冷→脸都憋红了 |
| **内心** | "可恶，好想发个朋友圈！" |
| **与张羽互动** | 确认对方也过了→拥抱庆祝→开始规划大学路线 |
| **核心心态** | 穷人的快乐也很朴素——终于拿到那张通往十大的门票了 |

### 张羽 —— 拿到证的喜悦+直觉的正神善意

| 维度 | 分析 |
|------|------|
| **炫耀欲** | 努力控制嘴角不要翘起来→"嗯"→被白真真看穿 |
| **证书的喜悦** | "带特效吗？比如悬在头顶的那种" |
| **对仙都同学** | 觉得他们实力不如自己和阿真→但嘴上不说 |
| **对邓丙丁的直觉** | 感觉对方总是看着自己和阿真说话→"我记得也没塞好感符啊" |
| **核心心态** | 赢了考试、拿到证书、拿了贷款——人生第一次觉得有了希望 |

### 云景 —— 被悔恨折磨的富家子

| 维度 | 分析 |
|------|------|
| **复盘心态** | 越想越后悔→越想越气→"如果能再战一次，绝对能胜过对方" |
| **输的方式** | 粗心大意→不够小心→像考试最后一题做错了 |
| **社交降级** | 没拿到筑基证→和拿到证的人不再是同一桌吃饭的人了 |
| **核心心态** | 不是输在硬实力，是输在轻敌——这才是最痛苦的失败方式 |

### 邓丙丁（六等正神/主考官） —— 棋局的清醒观察者

| 维度 | 分析 |
|------|------|
| **对张羽/白真真的照顾** | 加好友+压制周家+考试提醒 |
| **考试设计意图** | 绿洲集团将考试当宣传武器→夜凌霄输了就换对象 |
| **更大的关注** | 绿洲大规模干涉一层各方→神部没有反应 |
| **怀疑对象** | 绿洲集团背后是幽冥宫的某位 |
| **最终猜想** | "大神们已经在这场考试中落子了" |
| **核心心态** | 她知道这场考试远远不止"颁发四张筑基证"这么简单 |

| 奖励 | 内容 | 备注 |

```
二层（及以上）人
     ↑ 当狗/训狗

     ↑ 当狗/训狗
    1层人
```

- 1层的人是一部分1.5层人的狗
- 1.5层是2层人的狗
- 只有学会当狗和训狗，才能在昆墟一路往上爬

## 五、套路归档

### 套路名称：狗场哲学·昆墟阶级阶梯

**适用场景：** 角色用直白、粗鄙的比喻揭示整个社会的阶级结构——每个人不是被压迫就是压迫别人，不存在真正的平等。

**本章应用：**
- 圣杰："昆墟就像一个无比巨大的狗场——不是把别人当狗，就是当别人的狗"
- "只有学会当狗和训狗，才能在昆墟一路往上爬"
- 1层是1.5层的狗→1.5层是2层的狗→层层叠叠
- "只要钱给够→别说当狗了→就算被狗**他都能**到对方破产"
- **核心表达：** 在绝对不平等的社会里，尊严是最无用的奢侈品——你先活着爬上去，才有资格谈其他的

**归档文件：** `写作资产/套路库/狗场哲学·昆墟阶级阶梯.md`

---

### 套路名称：穷人逆袭后的认知翻转·外地人的身价重估

**适用场景：** 底层/外地出身的天才在重大比赛中战胜当地精英后，周围的人对他的认知从"穷外地"彻底翻转——但本质仍是势利眼的变体。

**本章应用：**
- 从"外地人"到"超级外地人"到"身价深不可测的外地富豪"
- 破旧外衣、布条运动裤→变成"高深莫测"
- "倒反天罡了，外地人竟然跑来把仙都爷的筑基资格证抢了"
- **核心表达：** 在这个阶层固化的世界里，你能赢多少人，就决定了他们怎么看你——但这场认知翻转让底层天才更孤独，因为他们知道这些称赞有多少是对"胜利"而非"你"的献媚

**归档文件：** `写作资产/套路库/穷人逆袭·外地人身价重估.md`

---

### 套路名称：奖励到账·穷人心态的持续反差

**适用场景：** 底层主角在大赛中获胜后面对巨额奖励——富人的不在意 vs 穷人的极度珍惜形成鲜明对比。

**本章应用：**
- 300万无息贷款：张羽/白真真喜滋滋→云景/圣杰无所谓
- 张羽："带特效吗？像悬在头顶的那种"
- "没钱修什么仙"的典型反差：穷人把每一笔钱都当救命稻草
- **核心表达：** 对富家子弟来说只是"又一张卡"，对穷学生来说是改变命运的弹药——同样的奖励对不同出身的人意义完全不同

**归档文件：** `写作资产/套路库/奖励到账·穷人心态反差.md`

---

### 套路名称：考后暗流·棋局的醒者

**适用场景：** 赛后看似平静的颁发奖励场景，只有极少数旁观者知道——这场考试的结果将引发更大的权力博弈。

**本章应用：**
- 邓丙丁独处时推演：绿洲集团整合一层→神部无反应→顾忌幽冥宫
- "大神们已经在这场考试中落子了"
- 考试只是宣传武器→谁是最终赢家尚未可知
- **核心表达：** 底层天才看到的是"我赢了考试"，上层势力看到的是"这局棋该怎么走"——最大的危险不在考场上，在考场外

**归档文件：** `写作资产/套路库/考后暗流·棋局的醒者.md`

---

### 套路名称：邓丙丁的暗中庇护·正神中的善意者

**适用场景：** 在冷酷的权力体系中，仍有个别高位者选择保护底层天才——但他们的善意背后往往也有自己的盘算。

**本章应用：**
- 加好友→压制周家→警告"不要轻易答应条件"
- 张羽：总感觉话是对着自己和白真真说的→"我没塞好感符啊"
- 邓丙丁知道更大的棋局→保护张羽也是保护自己的一步棋？
- **核心表达：** 在人人都是棋子的世界里，遇到愿意拉你一把的上位者是运气——但永远别以为这善意没有代价

**归档文件：** `写作资产/套路库/正神庇护·上位者的善意算盘.md`
