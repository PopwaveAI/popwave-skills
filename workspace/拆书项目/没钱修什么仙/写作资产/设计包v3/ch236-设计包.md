# 第236章 贺张羽 白真真考取筑基资格证成功 — 设计包 (v3)

---

## 一、核心概念

> **张羽和白真真考取筑基证后躲在小城消化收获，各大公司代表涌入嵩阳高中寻人，绿洲集团高调庆祝打破潜规则，周澈尘震惊感受到阶层鸿沟**

> 张羽和白真真离开考场回程，福姬建议先在小城市躲藏消化收获、提升实力，同时让大学代表们在嵩阳高中聚集——人越多周家越不敢动手。与此同时，紫云高中乐景辰发现张羽的汗液样本后大惊，乐沐岚联系张羽未果。嵩阳高中内，各大公司代表蜂拥而至找张羽和白真真，周澈尘被叫去招待却沦为笑柄。最后绿洲集团王总用数百飞剑在空中打出贺词，公开庆祝张羽和白真真考取筑基证，打破一层多年潜规则。

**本章使命：**
1. **张羽/白真真的战略回避**：福姬建议先躲小城消化收获，晾一晾大学代表提升议价空间
2. **周家威胁的意识**：周家可能狗急跳墙，防一手为妙
3. **乐家对张羽的兴趣**：乐景辰发现张羽的汗液纯净度极高，想联系他
4. **各大公司代表涌入嵩阳高中**：仙韵、天武、心创医疗等巨头齐至
5. **周澈尘的众生相**：从学生会陪衬到被公司代表嘲笑
6. **绿洲集团的高调破局**：飞剑贺词打破潜规则，张羽白真真名震嵩阳

### 关键节点

1. **福姬的战略布局：躲小城晾代表·三管齐下**（第24-43行）
   - 大学代表会托公司来联系→先晾一晾让他们相互竞争抬价
   - 周家可能两种反应：缓和关系或直接动手
   - 时间拖得越久，嵩阳聚集的人越多→周家越不敢动手
   - **核心表达：** 贫穷者没有犯错的空间——每一步都必须算到足够远的地方

2. **乐景辰的发现·张羽身体的秘密**（第50-63行）
   - 乐景辰分析张羽+乐沐岚的共同汗水→张羽的汗水纯净度极高
   - 猜测是公司新技术试验品→走神道→成为大神降临的容器
   - 乐沐岚感到可惜：张羽天赋极高却因出身没机会考入十大
   - 紫云药业总部直接联系→要找到张羽
   - **核心表达：** 张羽的身体本身就是一场未解的谜题——开始吸引更高层级的关注

3. **嵩阳高中·大公司代表蜂拥而至**（第79-91行）
   - 仙韵集团、天武集团、心创医疗、万兽牧业、万里集团——全是仙都总部来人
   - 周澈尘完全想不通嵩阳高中有何吸引力
   - 教导主任想推销其他学生→公司代表们哄堂大笑
   - **核心表达：** 整个嵩阳高中，除了张羽和白真真，其他人在公司代表眼中都是空气

4. **周澈尘的阶层冲击·沦为小丑**（第92-114行）
   - 被叫去以学生会会长身份招待→却一问三不知
   - 公司代表的笑容中夹杂着高高在上的嘲弄
   - 从一开始的"同为棋子"对比张羽→到"他们即将跳出棋盘"的觉悟
   - **核心表达：** 当成绩差距巨大到一定程度时，曾经的对手连被正视的资格都已经失去

5. **绿洲集团·飞剑贺词打破潜规则**（第120-122行）
   - 数百飞剑在天空组成贺词：贺张羽、白真真考取筑基资格证成功
   - 原本一层各势力对筑基考试守口如瓶的潜规则被彻底打破
   - 公司代表们意识到：绿洲集团要借此事宣传新技术
   - **核心表达：** 资本的入局不会等待旧的规则改变——它直接碾过去

## 三、人物设计

### 张羽 —— 战略回避的谨慎学霸

| 维度 | 分析 |
|------|------|
| **对周家的警觉** | 推断周家可能狗急跳墙→接受福姬的回避建议 |
| **对大学代表的态度** | 理解福姬的"晾一晾抬价"策略→关手机不联系 |
| **修行规划** | 趁这段时间修行筑基奖励功法+买第十门功法→凑齐武道圣胎 |
| **底层心态** | "周家的存在如鲠在喉"→知道最终只能靠自己解决 |

### 白真真 —— 成长中的战略家

| 维度 | 分析 |
|------|------|
| **附和福姬建议** | 立刻理解晾一晾代表的用意 |
| **对周家的认知** | 经历过帮派斗争→知道斗争的残酷性→"哪怕只有很低概率也要万分谨慎" |
| **隐藏灵根** | 不愿暴露天生灵根→仍需凑钱买天灵根遮掩 |

### 福姬 —— 战略指导者

| 维度 | 分析 |
|------|------|
| **三重目的布局** | 躲藏→消化收获+抬高议价+规避周家 |
| **对周家的判断** | "周家大概率知道邪神信徒身份"→有可能用更激烈手段 |
| **议价策略** | 晾代表→让大学相互察觉→抬价格 |
| **最终解决** | "只能靠你们自己的力量" |

### 乐沐岚 —— 惋惜型观察者

| 维度 | 分析 |
|------|------|
| **对张羽的态度** | 觉得可惜→出身导致没机会考入十大→只能走神道 |
| **行动** | 被乐景辰要求联系张羽→打不通电话→去嵩阳高中找 |
| **内心情绪** | 对穷人的印象被张羽不断粉碎 |

### 乐景辰 —— 商业嗅觉敏锐的研究者

| 维度 | 分析 |
|------|------|
| **猜测** | 张羽是某个公司新技术试验品→走神道→容器 |
| **兴趣** | 张羽的汗水纯净度极高→想得到更多样本 |
| **行动** | 通过乐沐岚联系张羽→联系不上→托乐沐岚去嵩阳 |

### 周澈尘 —— 阶层跌落感的觉醒者

| 维度 | 分析 |
|------|------|
| **初始心态** | 认为自己与张羽都是"被裹挟的棋子" |
| **面对公司代表** | 矮三分→沦为小丑→被嘲弄 |
| **最终觉悟** | "张羽即将跳出棋盘，而我还是一枚棋子"→决心最后冲刺考入十大 |
| **核心矛盾** | 在同龄人面前的优势→在公司代表面前的微不足道 |

## 五、套路归档

### 套路名称：战略回避·晾一晾抬价策略

**适用场景：** 贫穷主角在获得重大成就后，主动回避追捧者以提升议价空间

**本章应用：**
- 福姬建议不要立刻回嵩阳→先躲小城消化收获
- 目的：让大学代表相互察觉→竞争抬价
- "关手机不用急着联系"→主动制造稀缺感
- **核心表达：** 当你手握稀缺资源时，等待本身就是一种谈判武器

**归档文件：** `写作资产/套路库/战略回避·晾一晾抬价策略.md`

---

### 套路名称：周家狗急跳墙·成就带来的反向风险

**适用场景：** 主角成就越高，暴露越多，潜在敌人越可能孤注一掷

**本章应用：**
- 周家可能知道张羽/白真真的邪神信徒身份
- 发现大学代表接触→可能猜通过考试→可能直接动手
- "防一手又不损失什么"
- **核心表达：** 走上坡路时，身后的追兵会跑得更快——因为你的高度意味着你对他们威胁更大

**归档文件：** `写作资产/套路库/成就带来的反向风险·周家狗急跳墙.md`

---

### 套路名称：公司代表围猎·天才被当作投资标的

**适用场景：** 多个资本势力同时争夺一个天才角色——体现资本的逐利本性

**本章应用：**
- 仙韵、天武、心创医疗、万兽牧业、万里集团齐至
- 教导主任推销其他学生→代表们哄堂大笑
- 只看张羽和白真真→其他人都是空气
- **核心表达：** 资本只看得见最顶端的猎物——底下的人连被评估的资格都没有

**归档文件：** `写作资产/套路库/公司代表围猎·天才被当作投资标的.md`

---

### 套路名称：阶层碾压·周澈尘沦为笑柄

**适用场景：** 曾经的优越者被更高层级的势力赤裸裸地蔑视

**本章应用：**
- 周澈尘在嵩阳高中是学生会会长、第一名
- 在公司代表面前"矮了三分"
- 被笑着问话→不知如何回答→教导主任推销被嘲笑
- "那种高高在上的态度和传说中的仙都爷一模一样"
- **核心表达：** 你的优越只在你所在的阶层有效——往上走一步，你什么都不是

**归档文件：** `写作资产/套路库/阶层碾压·周澈尘沦为笑柄.md`

**适用场景：** 资本势力用高调宣传手段打破行业潜规则，强行推动新技术/新规则落地
