## 一、核心概念

> **大学选择的终极纠结 → 张翩翩情报支持 → 各大学开出条件对比 → 金刚宗最优/天剑门最适合阿真/万法大学有姐姐相助 → 王总（王胤）突然现身 → 直接打款500万 × 2 → 邀请成为绿洲集团代言人 → 每人每月400万（总6800万）**

张羽从张翩翩处获取万法大学情报，对比金刚宗、天剑门、万法大学等十大的条件陷入纠结。此时绿洲集团王总——王胤登场，以"我欣赏谁就给谁砸钱"的姿态，先给每人500万见面礼，再开出每人每月400万（总计6800万）的代言费。张羽和白真真被这位"最善良、最优秀、最有气魄"的王总彻底折服。

**本章使命：**
1. **大学信息的全面展开**：张翩翩提供各专业情报+各大学代表开出条件对比
2. **金刚宗 vs 天剑门 vs 万法大学的三角纠结**：张羽应去金刚宗、白真真该去天剑门、一起读万法有姐姐相助
3. **王总（王胤）首次正式登场**：赤发赤袍、清澈友善眼神、半自动无施咒符咒、信仰等级8级以上
4. **现金轰炸建立世界观**：见面先打500万×2、代言费每人每月400万总计6800万
5. **福姬的隐藏风险提示**：张翩翩受正神关注→越接近她越容易暴露福姬

### 关键节点

1. **大学选择的终极纠结**（第1-38行）
   - 张翩翩的情报：专业选择是"第二次投胎"、能帮调查万法大学老师
   - 福姬的情报：万法宗→上三宗之一（与幽冥宫、天魔宗齐名）
   - 金刚宗的条件：直博+天灵根+军队兼职+军用功法权限
   - 天剑门的条件：给白真真最优厚
   - 万法大学的优势：适合两人专业+有姐姐相助
   - 福姬的风险提示：张翩翩被正神关注→越接近越容易暴露福姬
   - **核心表达**：大学选择是人生第二次投胎——每个选择都有优劣，越了解越纠结

2. **公司人代表蜂拥而至**（第19-22行）
   - 绿洲集团捅破筑基证消息后→大学代表直接找张羽
   - 非十大全部淘汰→留下十大+相关专业
   - **核心表达**：筑基证让张羽从"被选择"变成"选择方"

3. **王总·王胤首次亮相**（第39-65行）
   - **外貌**：红袍赤发青年、落地玻璃窗前看夜景、清澈友善眼神
   - **见面就转钱**：每人500万→张羽内心"最善良最优秀最有气魄最适合当他爸爸的人"
   - **半自动无施咒**：金光一闪→符文凭空出现→拦住白真真下跪→信仰等级8级以上
   - **不搞形式主义**："我不喜欢别人下跪，不喜欢搞形式主义"
   - **昆墟道理**："有本事有才能的人，就该多赚"
   - **核心表达**：王总是张羽在昆墟见过的最"善良"的人——不因为敌人而是因为根本不在乎

4. **代言邀请·6800万爆炸**（第74-101行）
   - 绿洲集团产品投放一层市场→原定夜凌霄为主→夜凌霄输了→换成张羽和白真真
   - 张羽秒答→白真真问报酬→张羽嫌弃"阿真还是太看钱了"
   - 每人每月300万→张羽失望→王总最讨厌说"少"→加到400万/月→总计6800万
   - 张羽觉悟："钱多钱少不重要，主要是我们特别喜欢绿洲集团的企业文化"
   - **核心表达**：王总的"善良"是建立在绝对力量+绝对财富之上的从容——他眼中的好就是单纯的"砸钱"

## 三、人物设计

### 张羽 —— 被金钱砸醒的穷人学霸

| 维度 | 分析 |
|------|------|
| **大学纠结** | "大学专业选择简直就是第二次投胎"——意识到选择有多重要 |
| **对比各大学** | 金刚宗（自最优）> 万法大学（有姐助）> 天剑门（阿真最优）→越比越纠结 |
| **第一次见王总** | 被清澈友善眼神震撼→这是昆墟很久没见过的眼神了 |
| **收钱反应** | 看到500万→呼吸粗重→"最适合当他爸爸的人" |
| **嫌弃阿真太看钱** | 自认为不是势利的人→结果发现6800万→"我承认我肤浅了" |
| **最终态度** | "钱多钱少不重要，主要喜欢绿洲企业文化"——经典真香现场 |

### 白真真 —— 更直白的金钱拥抱者

| 维度 | 分析 |
|------|------|
| **赞同读万法** | "能和咱妈上一个大学，那当然是好事情" |
| **收到500万** | 要跪下来喊爹——被王总符咒拦住 |
| **问报酬** | 吞吞吐吐问代言费——比张羽更直接 |
| **核心心态** | 对王总完全信任→有钱就是爹 |

### 王胤（王总） —— 昆墟上层的"善良"富豪

| 维度 | 分析 |
|------|------|
| **外形** | 红袍赤发、清澈友善眼神、纯真善良友好 |
| **信仰等级** | 8级以上——半自动无施咒、意动咒到、正神重点监控对象 |
| **价值观** | "我欣赏谁，就给谁砸钱"——直接打钱没有条件 |
| **"少"字逆鳞** | 最讨厌别人说"少"→被说之后立刻加价 |
| **做事风格** | 直来直往、不搞形式主义、"有本事就该多赚" |
| **对张羽的定位** | 比夜凌霄更优秀→绿洲代言人→投资对象 |
| **本质** | 福姬判断：大概率是下来镀金的 |

### 福姬 —— 谨慎的风险提示者

| 维度 | 分析 |
|------|------|
| **万法宗情报** | 万法宗=万法大学→上三宗之一→同幽冥宫+天魔宗 |
| **暴露风险** | 张翩翩受正神关注→越接近越可能暴露福姬 |
| **对王总的判断** | "此人必然是正神的重点监控对象→要全力隐藏自己了" |
| **退场** | 说完后声音彻底消散→不再发言 |

## 五、套路归档

### 套路名称：大学选择·第二次投胎

**适用场景：** 主角面临多个顶级大学/势力的争夺，每个选择各有优劣，陷入纠结

**本章应用：**
- 金刚宗给张羽最优（直博+天灵根+军队+军用功法）
- 天剑门给白真真最优
- 万法大学有姐姐相助+专业都适合
- 越了解越纠结→福姬提示暴露风险
- **核心表达：** 所谓"选择权"也是一种奢侈品——当真正面临所有选项时，每个选择都意味着放弃同样好的东西

**归档文件：** `写作资产/套路库/大学选择·第二次投胎.md`

---

### 套路名称：我欣赏谁就给谁砸钱·王总的纯粹投资哲学

**适用场景：** 一个上层人物以最直接的方式（现金）表示欣赏，不附带任何条件

**本章应用：**
- 见面第一件事：转每人500万→无任何条件
- "我欣赏谁，就给谁砸钱"
- "有本事有才能的人，就该多赚，这就是昆墟的道理"
- 加价逻辑：被说"少"→立刻加→"我这个人最讨厌被别人说少"
- **核心表达：** 真正的顶层不在乎过程——只在乎"我看好你，你就是我的投资"

**归档文件：** `写作资产/套路库/我欣赏谁就给谁砸钱·王总的纯粹投资哲学.md`

---

### 套路名称：半自动无施咒·信仰等级的实战展现

**适用场景：** 展示上层人物与正神系统紧密连接的符咒能力，无需施法动作即可瞬发

**本章应用：**
- 王胤凭空催动符咒→金光+符文拦住白真真→无画符动作
- 福姬分析：半自动无施咒、信仰8级以上、时刻连接神力网络
- 正神重点监控对象→福姬全力隐藏
- **核心表达：** 当信仰等级达到8级以上，人已经不再是"使用"神力——而是"被神力包裹着存在"

**归档文件：** `写作资产/套路库/半自动无施咒·信仰等级的实战展现.md`

### 套路名称：6800万代言·穷人vs富人的价格认知错位

**适用场景：** 穷人面对巨额金钱时的认知失调——先嫌少、后震惊、最后真香

**本章应用：**
- 张羽：300万/月？→内心"有点失望"→对比刚收的500万觉得"小气"
- 王总加价：400万/月→张羽算过来→每人6800万→"我承认我肤浅了"
- "钱多钱少不重要，主要是我们特别喜欢绿洲集团的企业文化"
- **核心表达：** 穷人对"大钱"的认知会在几秒内被不断刷新——从觉得少到觉得多、从嫌不够到真香

**归档文件：** `写作资产/套路库/6800万代言·穷人vs富人的价格认知错位.md`

### 套路名称：上三宗·十大的内部层级

**适用场景：** 揭示十大内部也有高下之分——上三宗＞其他七宗

**本章应用：**
- 万法宗=万法大学（上三宗之一）
- 幽冥宫=幽冥大学（上三宗之一）
- 天魔宗=天魔大学（上三宗之一）
- "十大之中必然也有高下之分"
- 幽冥大学和天魔大学的名字令人不敢想象内部景象
- **核心表达：** 即使是"十大"，内部也有等级——在昆墟，什么都有等级
