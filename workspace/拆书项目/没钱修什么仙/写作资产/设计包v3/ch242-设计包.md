# 第242章 一层巨变之始 — 设计包 (v3)

---

## 一、核心概念

> **签下代言合同 → 王总推荐幽冥大学 → 张羽轻松安逸的生活开始 → 存款1500万+ → 绿洲集团在各城市扩张 → 红塔保护区大型聚会 → 王总宣布联合豪族推广合成食物/新炼体技术/新血脉技术 → "赏口水"场面 → 妖裔服务生排队接口水 → 绿洲发布会张羽代言 → 血神池泡澡 + 传授气血经 → 夜凌霄等人嫉妒 → 星火真人再次延迟见面**

张羽和白真真签下绿洲集团代言合同后，生活开始安逸。王总推荐幽冥大学并承诺帮提待遇。张羽不再需要拼命修炼——只需全天维持心法运转即可。王总在红塔保护区举办大型聚会，宣布联合各地豪族推广绿洲新技术。聚会上王总向妖裔服务生"吐口水赐造化"的场景让张羽第一次感到不适。随后在发布会上，张羽以"绿色食品改变了我"为绿洲代言，最终被王总拉入血神池修行气血经。星火真人再次因加班推迟见面。

**本章使命：**
1. **王总的幽冥大学背景**：全族幽冥大学毕业→可介绍老师+提待遇
2. **张羽的悠闲生活开端**：法力/肉体已达极限→只修心法→日子轻松到不习惯
3. **绿洲集团的全面扩张**：各城市公司联合→绿洲标志随处可见→豪族们纷纷巴结
4. **"赏口水"的阶级震撼**：王总吐口水=妖裔的"造化"→排队接口水→张羽第一次感受到不适
5. **张羽的代言首秀**：在发布会上背台词→内心犹豫→福姬催促→硬着头皮说
6. **血神池·气血经**：以链气巅峰强者心头血炼制的血池→幽冥大学基本功→提前教学

### 关键节点

1. **王总的幽冥大学背景**（第2-7行）
   - 全族幽冥大学毕业→可帮介绍老师
   - "有事报我名字就行"→找招生办提待遇
   - **核心表达**：王总不是单纯的有钱——是有背景、有门路、有家的"镀金者"

2. **张羽的存款安全感**（第8-14行）
   - 原存款470万+王总500万=970万
   - 每月400万→高中毕业前必然能买天灵根筑基
   - 算上170万贷款+330万无息贷款也肯定够
   - 大学纠结暂时放下→"在高中阶段完成筑基"
   - **核心表达**：钱的增长让张羽第一次有了"选择权"的从容感

3. **红塔保护区的阶级盛宴**（第47-67行）
   - 保护区被包场→自由呼吸不收费（平时要收费）
   - 豪族子弟+公司精英齐聚→各色食物+昂贵药水
   - 王总被众星捧月→"一群狗东西，都在围着王总舔"
   - 夜凌霄等仙都人敌意→为争王总身边的地位
   - **核心表达**：保护区的"自由呼吸"本身就是阶级标志——普通人一辈子都体验不到的"免费呼吸"

4. **"赏口水"的阶级震撼**（第69-86行）
   - 王总酒后看到瘸腿妖裔："可怜啊"→"赏你一场造化"
   - 吐口水→妖裔瘸腿好了→众人恭维
   - "人身如浑金璞玉，经过开发，身上的每一寸便都是神物"
   - 妖裔们排队接口水→张羽感到不舒服但又觉得正常
   - 安安用杯子接口水→为了熊文武→闻着香气一脸满足
   - **核心表达**：这是本章最核心的意象——顶层人物的一口口水，对底层来说就是天大的造化。张羽的不适暗示他还没有完全"适应"有钱人的视角

5. **绿洲发布会·张羽的代言首秀**（第90-99行）
   - 发布会直播覆盖学校/公司食堂/公共交通
   - 主持人：高中筑基考试特殊→张羽/白真真成就→绿色食品改变
   - 张羽背台词→内心犹豫→白真真推他→福姬催促→硬说
   - **核心表达**：张羽第一次意识到自己在为一个他并不完全认同的体系背书——但为了钱，他必须说

6. **血神池·气血经·幽冥大学入门**（第100-110行）
   - 血神池：链气巅峰强者的心头血炼制→改善体质/增长寿数/提升智慧
   - 气血经：幽冥大学基本功→提前学习→王总有教授权免费送
   - 夜凌霄/圣杰在池外嫉妒→"恨不得被教导的是他们"
   - **核心表达**：王总对张羽和白真真的偏爱是独一份的——这既让他们成为众矢之的，也让他们获得了其他人得不到的资源

## 三、人物设计

### 张羽 —— 从拼命到安逸的阶级跃升者

| 维度 | 分析 |
|------|------|
| **存款安全感** | 970万+每月400万→高中毕业前能买天灵根→"肯定够了" |
| **生活状态** | 法力/肉体极限→只修心法→"这种轻松日子好久没过过了"→产生负罪感→去补课界赚钱 |
| **对绿洲扩张的感知** | 注意到街上绿洲标志越来越多→感觉到巨变 |
| **对"赏口水"的反应** | 感到不舒服→"总觉得有些不舒服"→自我说服"好像是正常的" |
| **代言首秀** | 背台词→内心犹豫→被推被催→硬着头皮说 |
| **血神池反应** | 犹豫→王总拉进来→接受传授气血经 |
| **核心心态** | 被动地跟着金钱走——在舒适和良心之间摇摆 |

### 王胤（王总） —— 给"宠物"赏造化的顶层玩家

| 维度 | 分析 |
|------|------|
| **对张羽的态度** | "你们是我的大功臣"→亲切拉进血池→传授气血经 |
| **赏口水的逻辑** | "可怜啊"→"赏你一场造化"→"人身如浑金璞玉"→"今天心情好，我全都赏了" |
| **对人性的理解** | 知道自己的一口口水就是底层最大的恩赐→享受被崇拜的感觉 |
| **豪族聚会中的角色** | 众星捧月→联合各地豪族→宣布大计划 |
| **做事风格** | 直接、大方、亲切但居高临下 |

### 白真真 —— 对王总完全信任的追随者

| 维度 | 分析 |
|------|------|
| **对王总的态度** | 完全信任→"一群狗东西在围着王总舔→我们也赶紧上去" |
| **抢红包** | 直接冲上去抢→不觉得尴尬 |
| **对赏口水的反应** | 没有被特别描写→但也没有表现出不适 |
| **核心心态** | 完全适应了"跟着王总"的生存策略 |

### 夜凌霄、圣杰等仙都人 —— 被抢走位置的嫉妒者

| 维度 | 分析 |
|------|------|
| **见到张羽** | 心中一紧→脸上全是敌意 |
| **对张羽的定位** | "两个家伙这么会舔吗？"→最大竞争对手 |
| **行为策略** | 不去触王总霉头→心中盘算如何讨好王总 |
| **在血池外** | 守在池外→嫉妒→"恨不得被教导的是他们" |

### 福姬 —— 现实主义的幕后军师

| 维度 | 分析 |
|------|------|
| **对王总的判断** | "从上面下来的人"→大概率是下来镀金的 |
| **对局势的态度** | "难得你们走运，遇到这么爱砸钱的爷"→要好工作处好关系 |
| **对降薪问题的态度** | "这种新闻有什么好看的？跟我们又没关系"→"这是天下大势" |
| **本质** | 完全的现实主义者——只在乎自己和宿主的生存/发展 |

## 四、设定细节

### 绿洲集团的扩张策略

## 五、套路归档

### 套路名称：王总的口水·底层的一条造化

**适用场景：** 顶层人物的随意行为（吐一口口水）对底层来说就是改变命运的"造化"

**本章应用：**
- 王总看到瘸腿妖裔→"赏你一场造化"→吐口水→腿好了
- 妖裔们排队争抢口水
- 安安用杯子接→"为了熊文武"
- 张羽不适→但又觉得"这很正常"
- **核心表达：** 当阶层差距大到一定程度，顶层人物身体的每一寸都是神物——底层不仅要接受，还要感恩

**归档文件：** `写作资产/套路库/王总的口水·底层的一条造化.md`

---

### 套路名称：自由呼吸的保护费·阶级的空气墙

**适用场景：** 用"呼吸都要收费"的设定展示物理化的阶层差距——保护区内"免费呼吸"是特权

**本章应用：**
- 红塔保护区被包场→自由呼吸不收钱
- "没想到有朝一日，我们也能在保护区里自由呼吸"
- 平时普通人进保护区呼吸要收费
- **核心表达：** 连呼吸都是阶级——保护区的空气本身就是一种奢侈品

**归档文件：** `写作资产/套路库/自由呼吸的保护费·阶级的空气墙.md`

### 套路名称：镀金的王总·下来做业绩的上层公子

**本章应用：**
- 福姬判断："大概率是下来镀金的"

**归档文件：** `写作资产/套路库/镀金的王总·下来做业绩的上层公子.md`

### 套路名称：血神池·以他人血肉修炼的精英功法

**适用场景：** 上层功法以底层强者的身体精华（心头血）为修炼材料
