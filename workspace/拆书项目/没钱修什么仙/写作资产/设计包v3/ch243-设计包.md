# 第243章 有钱的张羽 — 设计包 (v3)

---

## 一、核心概念

> **张羽达链气极限（道心10级/法力100/肉体10级）→残牛舍身心诀效果下降 → 考虑换有钱人心法 → 墨天逸借钱 → 公司降薪潮开始 → 合成食物涨价 → 赵天行吃原味合成食物 → 绿洲推出食物贷服务 → 王总的目标：让一层有多少人吃饭就有多少人借钱**

张羽终于达到链气全满——道心10级、法力100、肉体强度10级。但随着生活越来越安逸、钱越来越多，残牛舍身心诀效果锐减。他开始考虑换一门"有钱人的心法"。穷人联盟的墨天逸借钱引发张羽对阶级变化的思考，他看到公司降薪、合成食物涨价、同学们生活越来越艰难。绿洲集团推出"食物贷"服务——让人人分期吃饭——王总直言"让一层有多少人吃饭，就有多少人借钱"。张羽第一次感受到自己正在成为自己曾经憎恨的那种"有钱人"。

**本章使命：**
1. **张羽达成链气全满**：道心10级+法力100+肉体强度10级→链气巅峰
2. **残牛舍身心诀失效**：生活安逸→功法效果下降→考虑换心法
3. **墨天逸借钱**：穷人联盟现状→公司降薪→外卖撞人→张羽打10万
4. **公司集体降薪**：豪族联合后第一步→就业环境恶化→员工默默接受
5. **合成食物涨价**：低价挤出市场后→开始涨价→学生吃不起
6. **食物贷的推出**：绿洲新金融产品→分期付费→让人人都有饭吃→实则让人人欠债
7. **王总的终极目标**："以后一层有多少人吃饭，就有多少人借钱"

### 关键节点

1. **张羽链气全满·残牛心法失效**（第1-18行）
   - 道心10级→周身筋肉法力如臂指使→圆润无碍
   - 但残牛变慵懒→断角变钝变粗→效果下降
   - 存款1900万→生活安逸→压力不够
   - 残牛舍身心诀：压力越大越有效→生活越来越好→换个更适合的心法？
   - 张羽心想："我也已经要成为有钱人了……所以要适应有钱人的修行方式"
   - 残牛心法＝底层功法→需要更换
   - **核心表达**：张羽升上来的"穷人心法"不再适应他的新身份——阶级跃迁意味着连功法都要换

2. **墨天逸借钱的阶级对照**（第19-56行）
   - 墨天逸拘谨→"张羽已经和我们不是一个世界的人了"
   - 穷人联盟现状：老大宋虚死亡→降薪→外卖撞人
   - 张羽打10万→两万投资翻了五倍回报
   - 墨天逸：你是所有穷学生的榜样→"狠狠打了那些有钱人的脸"
   - 张羽查看降薪新闻→福姬：没关系→张羽：我就好奇看看
   - 福姬：远离穷鬼接近富豪→"穷人联盟这种货色就应该直接拉黑"
   - 张羽："我知道……我心里就是不太舒坦"
   - **核心表达**：随着阶级跃迁，张羽必须学会"抛弃"过去的人——但良心上过不去

3. **食堂·赵天行的阶级裂缝**（第57-71行）
   - 张羽/白真真：丰盛合成肉 vs 赵天行：原味合成食物
   - 合成食物先低价挤出市场→然后涨价→绿色食物价格飙升
   - 张羽借"难吃"推鸡胸肉给赵天行→赵天言感激又羞耻
   - 数十万高中生报名筑基考试→市场价值巨大
   - **核心表达**：张羽的"施舍"动作，暗示了两人已经不再是同一个阶级——王总对张羽"丢衣服"和张羽对赵天行"丢鸡胸肉"是对称结构

4. **食物贷·绿洲金融的终极设计**（第76-99行）
   - 市民收入降低+食物价格上涨→绿洲决定"改变"
   - 食物贷：分期付费→降低门槛→人人有饭吃
   - 新用户免息+包月优惠→推广目标：每个学生都有一笔备用食物金
   - 张羽想起自己吃不饱时的样子："如果那个饭都吃不饱的张羽遇到了这种情况……"
   - 王总的话："以后一层有多少人吃饭，就有多少人借钱"
   - "关系到集团在一层的布局"
   - 王总承诺：幽冥大学读书→二层金融公司实习→转金融系
   - **核心表达**：食物贷不是慈善——是想让每个人都变成绿洲的"债务人"，彻底垄断底层经济命脉

## 三、人物设计

### 张羽 —— 被动升阶后开始矛盾的有钱人

| 维度 | 分析 |
|------|------|
| **链气全满** | 道心10级+法力100+肉体10级→已到链气极限 |
| **梦中的残牛** | 慵懒不干活→断角变钝→功法效果下降→"是因为太安逸了吗？" |
| **心法思考** | "我也是有钱人了"→要换有钱人心法→找有钱人推荐 |
| **对墨天逸的态度** | 念旧→给了10万→但心里已经开始接受"要远离穷鬼"的价值观 |

## 五、套路归档

### 套路名称：功法的阶级适应·有钱之后要换心法

**适用场景：** 功法与阶级绑定的世界观——底层功法在有钱后就失效，需要换更高级的心法

**本章应用：**
- 残牛舍身心诀：压力越大越有效→生活安逸→功法变差
- "毕竟是适合底层的功法"
- "我也已经要成为有钱人了……要找些有钱人的功法来修炼"
- 观想残牛变得慵懒→没有斗志→功法暗示主角阶级变了
- **核心表达：** 在昆墟，连功法都是有阶级的——穷人的功法靠"抗压"驱动，富人的功法靠"舒适"驱动
