# 第244章 思想落后的张羽 — 设计包 (v3)

---

## 一、核心概念

> **食物贷破十亿庆典 → 王总送活人皮练功皮 → "活衣架"当场剥皮 → 圣杰/谈诗珊筑基成功 → 张羽换上真皮 → 周澈尘被打 → 张羽驱散 → 抵押血肉服务推出 → 王总揭示真相：人肉卖给上层 → "你们思想很落后，要记住是你情我愿的买卖" → 张羽被说得哑口无言**

食物贷规模破十亿，王总在绿洲大厦1111层办庆功宴。王总给张羽买了一张"真皮"练功皮——由一个"活衣架"（活人）长期运功维护的活人皮。张羽看到活衣架当场剥皮后，第一次感受到王总眼中的"善意"其实是"看宠物的眼神"。随后抵押血肉服务推出，张羽追问人肉真相，王总坦然告知：一层的人肉卖给上层做材料。当张羽表达抵触时，王总一句"谁给你们钱，你们就听谁的"让张羽再也说不出话。他的几千万存款、天灵根、筑基、十大——全都来自绿洲。

**本章使命：**
1. **食物贷十亿庆功宴**：红包符轰击、奖金翻倍、药水免费→财富狂欢
2. **活人皮震撼**：活衣架当场剥皮→王总挑皮如挑衣服→张羽看到"看宠物的眼神"
3. **圣杰/谈诗珊筑基成功**：家族支持先筑基→张羽明明更强却落后→财富差距
4. **张羽无奈换皮**：在所有人嫉妒的目光中穿上真皮→残牛直接停摆→自欺欺人
5. **周澈尘的衰落**：周家失去绿洲合作→周澈尘被打→张羽驱散
6. **抵押血肉服务**：用户以自身血肉还债→绿洲卖肉给上层
7. **王总揭示真相**：一层的人体是资源→卖给上层做材料→"你情我愿的买卖"
8. **王总对张羽的"教育"**："你们思想很落后"→"谁给你们钱，你们就听谁的"→张羽被说得哑口无言

## 二、情节框架

### 主视角（张羽视角——庆功宴 → 换皮 → 追问真相）

| 段落 | 内容 | 核心信息 |
|------|------|----------|
| **十亿庆功宴（A段，1-13行）** | 绿洲大厦1111层→神雷烟花组成"十亿"→食物药水免费→奖金翻倍→"未来100个亿万富翁"→红包符如雨洒下 | 食物贷破十亿→财富狂欢 |
| **活人皮震撼（B段，14-53行）** | 王总带张羽看"好东西"→活衣架当场剥皮→真皮由活人长期维护→"干这行二十多年了"→张羽内心：疯了→王总眼神：看财产看宠物的眼神→圣杰谈诗珊筑基成功→张羽无奈换皮→残牛彻底停摆 | 活人皮=高端装备╱看财产的眼神╱残牛停摆 |
| **周澈尘的衰落（C段，54-77行）** | 存款2900万→周澈尘被十多名同学围攻→周家失去绿洲合作→张羽驱散→"关我屁事" | 周家衰落╱张羽已经懒得管小事 |
| **抵押血肉推出（D段，78-82行）** | 食物贷成功后→绿洲推出抵押血肉→允许用户以自身血肉还债→公司人讨论：怎么收肉、评估、变现、存储 | 抵押血肉：抽筋扒肉→送上餐桌 |
| **真相揭示（E段，83-132行）** | 存款3300万→黑暗补课界学员减少（老年学员寿尽）→存款3800万→人肉传闻曝光→张羽追问→王总：合成食物加人肉亏死→卖给上层的→这是你情我愿的买卖→"你们思想很落后"→"谁给你们钱，你们就听谁的"→张羽被说得哑口无言 | 人肉售卖给上层╱一层是材料基地╱张羽无法反驳因为钱都来自王总 |

### 关键节点

1. **十亿庆典·红包符的狂欢**（第1-13行）
   - 神雷烟花组成"十亿"
   - 食物药水免费、奖金翻倍
   - 王总发红包符→无息红包如雨点洒下
   - "未来100个亿万富翁"
   - 白真真冲上去抢红包→张羽不去→王总："有钱了就不跟下面的人抢小钱"
   - **核心表达**：十亿只是开始——绿洲的目标是千亿、万亿

2. **活人皮·"看宠物的眼神"**（第14-41行）
   - 活衣架：像脱外衣一样扒掉自己的皮→露出满是血肉的身躯
   - 张羽问"不疼吗？"→"入行前就屏蔽痛觉神经了"
   - 真皮需要每天精血滋养、秘法炮制、由活衣架运功维护
   - 王总：幽冥大学血神子最擅长剥皮制皮
   - 张羽内心："疯了，这昆墟真特么是疯了"
   - 王总的眼神："那不是看人的眼神，而是看一个财产，一个宠物，一个小动物的眼神……"
   - **核心表达**：对王总来说，活人皮和衣服没有区别——活衣架就是"衣架"，不是人。这是张羽第一次真正看到上层的"善良"是什么样子

3. **张羽无奈换皮·残牛停摆**（第50-60行）
   - 在圣杰/谈诗珊嫉妒的目光中换上真皮
   - 残牛舍身心诀→残牛彻底停摆→不再动弹
   - 张羽内心挣扎："不要输给有钱人"→"我要买天灵根我要筑基"
   - "练功皮嘛，本来就是一种装备，没什么大不了的……"
   - **核心表达**：张羽的自欺欺人——明知不对但为了目标必须做

4. **周澈尘的衰落**（第64-77行）
   - 周家失去绿洲合作机会→日渐衰落
   - 周澈尘被十多名学生围攻→张羽驱散
   - "滚远点，别在我附近乱搞"——不是帮助，是嫌吵
   - 周澈尘说谢谢→转身离去
   - 张羽："关我屁事"
   - **核心表达**：绿洲的扩张就是豪族的"淘汰赛"——周家倒了，下一个倒的会是谁？

5. **王总揭示·人肉出售真相**（第96-128行）
   - 张羽问抵押血肉做食物的事→王总：合成食物没人肉，"亏死"
   - 但卖给上层的时候会用——"你们该不会以为我们做这么多就是为了赚一层的钱吧？"
   - 人体是宝库：血、肉、皮、骨、髓、脑、怨气、魂魄各有妙用
   - 如果还做过改造、修炼够用功→价值更大
   - "他们在这个过程中提高成绩，就算失败了卖肉卖魂也能变现高一点——这是好事"
   - "你们是一层的凡人家庭出身，思想上很落后"
   - "你情我愿的买卖"
   - "谁给你们钱，你们就听谁的"
   - 张羽："发现自己那千锤百炼的肉体，此刻竟施展不出丝毫的力气"
   - **核心表达**：本章的终极爆点——张羽的一切都来自绿洲，他没有任何资格批判绿洲的商业模式

## 三、人物设计

### 张羽 —— 从震撼到沉默的被收买者

| 维度 | 分析 |
|------|------|
| **对活人皮的反应** | 问"不疼吗？"→内心"疯了"→"这昆墟真特么是疯了" |
| **看穿王总的眼神** | "那不是看人的眼神，而是看一个财产，一个宠物，一个小动物的眼神" |
| **换皮时自我说服** | "不要输给有钱人"→"练功皮嘛，就是装备"→残牛停摆 |
| **对周澈尘的态度** | 驱散打人者→但说"关我屁事"→冷漠施舍 |
| **追问人肉真相** | "所以我们在把一层的人卖给上层的人？"——直接问出最扎心的问题 |
| **被王总说服** | 王总说"你情我愿"→"谁给钱就听谁的"→张羽再也说不出话 |
| **最终状态** | 肉体千锤百炼但施展不出丝毫力气→被金钱锁链牢牢捆绑 |
| **核心心态** | 知道不对但无法反抗→因为一切来自王总→自尊和野心在拉扯中 |

### 王胤（王总） —— 用"善良"包装的顶层剥削者

| 维度 | 分析 |
|------|------|
| **对活人皮的态度** | "这张皮是我特意挑选的"→如数家珍→"幽冥大学血神子最擅长剥皮制皮" |
| **对活衣架的态度** | 摆弄人皮和青年→检查质量→像挑衣服 |
| **对抵押血肉的坦白** | "合成食物加人肉亏死"→"只有卖给上面的时候才用" |
| **对人体的认知** | "人体是一个大宝库"→血/肉/皮/骨/髓/脑/怨气/魂魄 |
| **对张羽的教育** | "你们是一层的凡人家庭出身，思想上很落后" |
| **价值观** | "你情我愿"→"谁给你们钱你们就听谁的"→"这在仙道之路上很正常" |
| **层级的逻辑** | 一层成本最低→二层有人体保护法案→一层是完美的材料基地 |
| **根本态度** | 把张羽当宠物——宠物护食时需要教育一下 |

### 活衣架青年 —— 正常的"不正常者"

| 维度 | 分析 |
|------|------|
| **对剥皮的态度** | "入行前就屏蔽痛觉神经了"→"干这行二十多年了" |
| **对真皮的态度** | "皮上绝不会有任何瑕疵"→生怕生意黄了 |
| **核心状态** | 完全不觉得自己被剥削→认为这是"正常的工作" |

### 白真真 —— 完全拥抱上层的有钱人

| 维度 | 分析 |
|------|------|
| **抢红包** | 欢呼→冲上去抢 |
| **对换皮** | 没有质疑→已经完全接受 |
| **对抵押血肉的反应** | 愣神→不知所措→没有表态 |
| **对王总的态度** | 完全服从→不敢说一句话 |

## 四、设定细节

### 食物贷十亿庆功宴

| 维度 | 内容 |
|------|------|
| **地点** | 绿洲大厦1111层（嵩阳市最高处） |
| **庆祝方式** | 神雷烟花组成"十亿"、食物药水免费、奖金翻倍 |
| **王总发言** | 规模扩张到百亿/千亿/万亿→"未来100个亿万富翁" |
| **红包符** | 无息红包符→如雨点撒向全场 |
| **王总对张羽** | "有钱了就要学会不跟下面的人抢这些小钱" |

### 活人皮（高端练功皮）

| 维度 | 内容 |
|------|------|
| **来源** | 出身人身的活人→由秘法炮制+活衣架维护 |
| **维护方式** | 每天以精血滋养+秘法炮制+穿在活衣架身上运功修行 |
| **活衣架工作年限** | 二十多年→入行前屏蔽痛觉神经 |
| **品质差异** | 手工皮≠批量货色→能强化护体功法→突破原有极限 |
| **具体来源** | 横练高手的人皮→适合张羽 |
| **技术背景** | 幽冥大学血神子功法→最擅长剥皮制皮 |
| **价格** | 王总做主买的→张羽不换时被说"以后丢我的脸" |

### 抵押血肉服务

| 维度 | 内容 |
|------|------|
| **产品背景** | 食物贷成功推广后→绿洲集团推出 |
| **运作模式** | 允许用户以自身血肉还债 |
| **公司操作** | 怎么收肉→怎么评估→怎么变现→怎么存储 |
| **最终用途** | 卖给上层（非合成食物） |
| **上层需求** | 幽冥大学、天妖大学、天魔大学、白骨大学需求量高 |
| **价格影响因素** | 做过人体改造→自身修炼够用功→价值更大 |
| **选择权** | "他们想卖，我们就收"→"你情我愿的买卖" |

### 上层的成本逻辑

| 维度 | 内容 |
|------|------|
| **上层用人成本** | 高→禁止凡人养殖→人体保护法案→怨气排放管理 |
| **一层的优势** | 成本低廉→无保护→完美材料基地 |
| **商业模式** | 授权仙道技术→出口仙道产品→换取下层资源 |
| **王总的结论** | "我们也是被逼着来这的" |

### 黑暗补课界的变化

| 维度 | 内容 |
|------|------|
| **绿洲影响** | 联合各公司→加大打击盗版→放弃原本帮派 |
| **学员减少** | 公司降薪→学员收入下滑→减少补课 |
| **老年学员** | 收入下滑后耗尽寿数→走到生命尽头 |
| **张羽收入** | 随之锐减 |

## 五、套路归档

### 套路名称：活人皮·当"装备"长在别人身上

**适用场景：** 展示上层社会将活人身体当作"装备"来制作和使用——赤裸裸的物化

**本章应用：**
- 活衣架当场剥皮→露出血肉身躯→"干这行二十多年了"
- 真皮需每天精血滋养+秘法炮制+活衣架运功维护
- 王总挑皮如挑衣服→"幽冥大学血神子最擅长剥皮制皮"
- 张羽："那不是看人的眼神，是看财产、看宠物、看小动物的眼神"
- **核心表达：** 当你有钱有势到一定地步，别人的身体就是你的"装备"——而你自己对此毫无知觉

**归档文件：** `写作资产/套路库/活人皮·当装备长在别人身上.md`

### 套路名称：残牛停摆·最后一块遮羞布被撕下

**适用场景：** 主角做了违心之事后，他的"穷人心法"彻底失效——功法的运转让主角无法自欺

**本章应用：**
- 穿上活人皮后→残牛彻底停摆→不再动弹
- "什么垃圾心法"→但内心知道是自己变了
- 自我说服："练功皮嘛，本来就是一种装备……"
- **核心表达：** 心法是主角最后的道德指南针——当它停摆，代表主角已经彻底跨过了"穷人"的底线

**归档文件：** `写作资产/套路库/残牛停摆·最后一块遮羞布被撕下.md`

### 套路名称：你们思想很落后·王总的教育课

**适用场景：** 上层人物对底层出身的主角进行"三观教育"——用看似合理的话术消解主角的道德不适

**本章应用：**
- "你们是一层的凡人家庭出身，思想上很落后"
- "身上血肉都是一层的人自己卖的→你情我愿的买卖"
- "他们就算不卖给我，也会卖给别人"
- "这在仙道之路上很正常"
- "谁给你们钱，你们就听谁的"
- **核心表达：** 顶层的话术核心就一句话——"这不关你的事" + "这都是他们自愿的"

**归档文件：** `写作资产/套路库/你们思想很落后·王总的教育课.md`

### 套路名称：千锤百炼的肉体却施展不出力气·金钱的枷锁

**适用场景：** 主角拥有强大的力量，但面对"恩主"的错误行为时发现自己完全无法反抗

**本章应用：**
- 张羽想反驳王总→"发现自己那千锤百炼的肉体，此刻竟施展不出丝毫的力气"
- 想到了各自的几千万存款→天灵根→筑基→十大
- 所有的"未来"都来自王总→没有反抗的资格
- **核心表达：** 当你的所有资本都来自于一个人时，你就成了他的"资产"——强大只是他给你投资的价值，不是你反抗的本钱

**归档文件：** `写作资产/套路库/千锤百炼却施展不出力气·金钱的枷锁.md`

### 套路名称：一层是材料基地·上层的人体供应链

**适用场景：** 揭示一层的真实定位——不是"下层社会"而是"原料生产基地"，人体是核心资源

**本章应用：**
- 人体宝库：血/肉/皮/骨/髓/脑/怨气/魂魄各有妙用
- 做过改造+修炼够用功→价值更大
- 上层成本高：禁止凡人养殖+人体保护法案→一层是完美替代
- "你们该不会以为做这么多就是为了赚一层的钱吧？"
- 技术授权→仙道产品出口→换取下层资源
- **核心表达：** 一层的"修仙教育"和"公司体系"本质上是一条人体供应链——从培养、改造到收割、出售

**归档文件：** `写作资产/套路库/一层是材料基地·上层的人体供应链.md`

### 套路名称：周家的衰落·绿洲的淘汰赛

**适用场景：** 绿洲扩张过程中，不支持/没资格合作的老势力被淘汰——曾经的精英变成下层

**本章应用：**
- 周家竞争力不足→没得到绿洲合作机会→日渐衰落

- **核心表达：** 绿洲不是和所有人合作——它选择和胜利者合作，让失败者更快地变成"材料"

**归档文件：** `写作资产/套路库/周家的衰落·绿洲的淘汰赛.md`
