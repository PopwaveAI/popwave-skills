# 第246章 努力适应，宗门龃龉 — 设计包 (v3)

---

## 一、核心概念

> **给乐沐岚补课→乐沐岚发现张羽未绝育→张羽和白真真与张翩翩通话→"无用阶层"警告→十大宗门龃龉→福姬分析宗门利益→赵天行被父亲抵押肉身→张羽求王总被拒→"要学会吃人"→被赶出→金玉满堂诀倒退→星火真人接近**

张羽继续给乐沐岚做贴身补课，乐沐岚发现张羽未绝育，试图用大数据关键词操控他的手机推荐。张羽与白真真通过巡查队联系张翩翩，得知上层有人认为"一层已是无用阶层"，要将一层改造成养殖场，涉及十大宗门之间的龃龉。福姬分析这是宗门利益之争，非正邪之争。赵天行被父亲抵押肉身给绿洲集团并已出货运往二层，张羽求王总帮忙要回被拒——王总暴怒教训他"要学会吃人"。张羽被赶出，金玉满堂诀效果倒退。章末星火真人已接近。

**本章使命：**
1. **"无用阶层"警告上线**：张翩翩透露上层有人认为一层已废，要改造成养殖场—打破张羽的安逸幻想
2. **十大宗门龃龉浮出水面**：有仙人支持改造、有仙人不支持—但都是为了自身利益
3. **福姬的宗门利益论**：正道魔道之争是伪命题，全是利益博弈
4. **赵天行肉身被卖**：张羽第一次直面"朋友被吃"的残酷现实
5. **王总的真面目**：平和善良只是表象，牵扯公司利益时暴怒羞辱
6. **"吃好了才能变强"**：王总的名言——要学会当人上人就要学会吃人
7. **金玉满堂诀的动摇**：余额再多也无法带来勇气——心法效果出现倒退

## 二、情节框架

### 主视角（张羽视角——补课日常 + 跨层通话 + 拯救赵天行失败）

| 段落 | 内容 | 核心信息 |
|------|------|----------|
| **乐沐岚补课（A段，1-35行）** | 张羽给乐沐岚贴身指导武功→乐沐岚发现张羽未绝育→输入"黑丝美腿大雷乐沐岚"→大数据影响手机→张羽刷到丝袜广告→白真真发现纸篓卫生纸 | 张羽未绝育仍是伏笔╱乐沐岚想获取张羽身上的材料 |
| **张翩翩通话·无用阶层（B段，36-91行）** | 张羽白真真联系张翩翩→张翩翩：只拿报酬别碰生意→一层被上层认为"无用阶层"→要改造成养殖场→有仙人支持→福姬分析：十大宗门利益之争 | 无用阶层概念╱宗门龃龉╱绿洲集团卷入上层斗争 |
| **赵天行肉身被卖（C段，92-153行）** | 赵天行母亲求助→父亲赌博抵押肉身→已出货运往二层→张羽找王总→王总暴怒"生意就是生意"→"要学会吃人"→被赶出→金玉满堂诀倒退 | 王总真面目╱吃人论╱心法动摇 |

### 关键节点

1. **乐沐岚的发现·大数据操控**（第22-35行）
   - 发现张羽没绝育→"没有绝育的学生，特别容易被体内的激素控制"
   - 对着手机重复关键词：黑丝！美腿！写真！曲线！反差！大长腿！大雷！乐沐岚！
   - 结果：张羽刷到丝袜页面→"什么垃圾大数据"
   - 白真真看到纸篓卫生纸→"这次坚持的天数有点短啊"
   - **核心表达**：乐沐岚是个心机girl——不是简单的补课对象，她想从张羽身上获取"筑基材料"

2. **无用阶层·宗门龃龉**（第47-91行）
   - 张翩翩:"上面有人认为，一层已经废了，广大穷人阶层已经变成了无用阶层"
   - "与其这么废下去，不如废物利用，把整个一层改造成养殖场"
   - "甚至有仙人……可能也这么认为"
   - 福姬分析：张翩翩说话很谨慎，有人监控谈话
   - 福姬：有仙人支持绿洲集团，也有仙人不支持→正邪之争是假，利益之争是真
   - **核心表达**：一层的命运不是偶然——是上层宗门博弈的结果。穷人不是活不下去，是被判定为"无用"

3. **赵天行肉身被卖·张羽的无力**（第104-153行）
   - 赵天行父亲失去工作后赌博→把儿子肉身抵押给绿洲集团
   - 张羽找王总帮忙→王总轻松答应→但已出货运往二层
   - 王总翻脸："生意就是生意，发出去的货……你让我问客户要回来？"
   - "穷人一辈子吃苦，只能吃得苦中苦。你如果想和我一样当人上人，就要学会吃人。"
   - 张羽被金光大手拖出，甩出办公室
   - 赤河："让他去做魂修生吧"
   - 金玉满堂诀效果倒退——余额观想不再能带来勇气
   - **核心表达**：有钱了也不够——在昆墟，你是"谁的人"比你有多少钱更重要。王总可以一句话救回，但他选择不救，因为"生意就是生意"

4. **"吃人论"**（第128-136行）
   - "在这个世界，吃好了才能变强"
   - "穷人一辈子吃苦，只能吃得苦中苦"
   - "你如果想和我一样当人上人，就要学会吃人"
   - "不损害公司利益的时候，我可以给你面子"
   - **核心表达**：王总的哲学——富人不是靠吃苦变强的，是靠"吃人"变强的。在公司利益面前，任何情面都是狗屁

## 三、人物设计

### 张羽 —— 在安逸中惊醒的"新有钱人"

| 维度 | 分析 |
|------|------|
| **补课状态** | 自信教学技巧在整个一层名列前茅→给乐沐岚贴身指导 |
| **对无用阶层的反应** | "跟我没关系""我一个高中生又能做什么？"→但心中恼火"我怎么就适应不了呢？" |
| **对赵天行的态度** | 拼命想要救朋友→面对王总的暴怒仍在争取→最终无力 |
| **对王总的认知转变** | 从"好说话的大佬"到"暴怒的巨兽"→认识到王总的另一面 |
| **心法状态** | 金玉满堂诀效果倒退→余额观想不再能带来勇气 |
| **核心心态** | "我的思想还是太落后了吧？"--还在努力适应昆墟，但每一次适应都被现实狠狠打击 |

### 白真真 —— 发现张羽状态下滑的观察者

| 维度 | 分析 |
|------|------|
| **对张羽的观察** | 纸篓卫生纸→"这次坚持的天数有点短啊"→担心安逸生活让他意志下滑 |
| **对流浪汉的态度** | 和张羽一起看到更多流浪汉、老者→但没有多说什么 |
| **对张翩翩通话** | 认真倾听张翩翩的警告→心存不安 |

### 乐沐岚 —— 有心机的补课学生

| 维度 | 分析 |
|------|------|
| **补课状态** | 每周接受张羽辅导→明显感受到进步 |
| **发现** | 通过贴身锻炼确认张羽没有绝育→"张羽很硬" |
| **计划** | 用大数据影响手机推送→"也许就能在将来的锻炼中得到一些张羽身上的材料" |
| **核心心态** | 不是单纯的感恩学生——她有自己的目的（获取筑基材料） |

### 王胤（王总） —— 翻脸无情的驯兽师

| 维度 | 分析 |
|------|------|
| **前期态度** | 轻松答应张羽请求→"小事一桩" |
| **得知已出货** | 气质突变→平和善良→无边寒意→"你……说什么？" |
| **对张羽的训斥** | "搞清楚你自己的身份""你特么拿的是我的钱" |
| **公司利益至上** | "不损害公司利益的时候，我可以给你面子""但你要妨碍公司经营，那就给我滚" |
| **核心哲学** | "吃好了才能变强"→"要学会吃人" |

### 张翩翩 —— 谨慎的跨层传话人

| 维度 | 分析 |
|------|------|
| **警告内容** | 只拿报酬别碰生意→一层要被改造成养殖场 |
| **说话风格** | 非常谨慎→福姬判断有人监控谈话 |
| **建议** | 联系正神（已加好友的那个） |
| **状态** | 似乎很忙→通话时间不长 |

### 福姬 —— 宗门利益的分析师

| 维度 | 分析 |
|------|------|
| **对张翩翩的分析** | 说话谨慎→有人监控 |
| **宗门利益论** | "就算有其他宗门要阻止绿洲集团，必然也是出于自身利益的需求" |
| **对正道魔道的态度** | "什么正道魔道"→全是利益之争 |
| **对局势的判断** | "也许……只是目前还没有阻止"→绿洲集团的行动可能正是另一派想看到的 |

## 四、设定细节

### 无用阶层·养殖场计划

| 维度 | 内容 |
|------|------|
| **提出者** | 上层某些大人物→"有人（仙人）这么认为" |
| **核心论点** | 一层已经废了→广大穷人已经是"无用阶层" |
| **改造方案** | 把整个一层改造成养殖场→纯粹的原材料供应地 |
| **支持方** | 有仙人支持这种改造（幽冥宫一系） |
| **反对方** | 有仙人不支持→但不是因为正邪之分，而是利益之争 |
| **现状** | 绿洲集团动作很大但未被阻止 |

### 十大宗门龃龉机制

| 维度 | 内容 |
|------|------|
| **矛盾实质** | 不是正邪之争→是利益之争 |
| **具体表现** | 上面很多区域禁止凡人养殖、立怨气排放标准→因为其他门派在这方面的仙道技术不行 |
| **绿洲集团的定位** | 卷入双方斗争的棋子 |
| **张翩翩的警告** | 不要牵扯绿洲集团的生意→避免被波及 |

### 赵天行肉身事件

| 维度 | 内容 |
|------|------|
| **起因** | 父亲失去工作→赌博→抵押儿子肉身给绿洲集团 |
| **货物状态** | 已"出货"→运往二层 |
| **买家** | 上层妖裔客户→买血肉作为食材 |
| **王总的理由** | 生意就是生意→发出去的货不能要回 |
| **张羽的尝试** | 想赔钱→想道歉→被逐出 |
| **结局** | 赵天行母亲打来电话→张羽不知如何回应 |

### 金玉满堂诀效果倒退

| 维度 | 内容 |
|------|------|
| **触发原因** | 赵天行事件→第一次直面"朋友被吃"的残酷现实 |
| **具体表现** | 心法效果逐渐放缓→不论现金数字如何观想→都无法再带来之前的底气、勇气 |
| **意义** | 钱不是万能的——当现实底线被突破，余额再多也无法提供心理支撑 |

## 五、套路归档

### 套路名称：无用阶层·养殖场改造计划

**适用场景：** 揭示上层对下层的真实定位——不是"帮助穷人"，而是"废物利用"

**本章应用：**
- "上面有人认为，一层已经废了，特别是这里的广大穷人阶层已经变成了无用阶层"
- "与其这么废下去，不如废物利用，把整个一层改造成养殖场，纯粹的原材料供应地"
- "甚至有仙人……可能也这么认为"
- 张羽看到路边更多流浪汉、缺胳膊断腿的老者→和养殖场计划联系起来
- **核心表达：** 在昆墟，穷人不是"需要帮助的人"，而是"需要被处理的废料"

**归档文件：** `写作资产/套路库/无用阶层·养殖场改造计划.md`

---

### 套路名称：宗门龃龉·正邪之争的利益本质

**适用场景：** 揭示修仙世界中"正邪之争"背后的利益博弈

**本章应用：**
- 福姬分析：有仙人支持绿洲集团，也有仙人不支持
- "什么正道魔道，真有其他宗门的力量来阻止他们，肯定也是为了限制幽冥宫的实力膨胀"
- 其他门派立怨气排放标准→不是因为道德→是因为他们在这方面的技术不行
- **核心表达：** 在昆墟，一切斗争的本质都是利益博弈，"正道"和"魔道"只是标签

**归档文件：** `写作资产/套路库/宗门龃龉·正邪之争的利益本质.md`

### 套路名称：吃人论·生意就是生意的冷酷哲学

**适用场景：** 上层人物对底层的冷酷宣言——利益面前无朋友

**本章应用：**
- "在这个世界，吃好了才能变强"
- "穷人一辈子吃苦，只能吃得苦中苦。你如果想和我一样当人上人，就要学会吃人"
- "不损害公司利益的时候，我可以给你面子"
- "你特么拿的是我的钱"
- 赵天行被卖→王总认为"这种朋友被吃又怎么样？"
- **核心表达：** 上层的逻辑：要么你吃人，要么你被人吃。公司利益高于一切

**归档文件：** `写作资产/套路库/吃人论·生意就是生意的冷酷哲学.md`

### 套路名称：心法动摇·余额无法带来的勇气

**适用场景：** 当现实打击超出心理承受范围，心法效果出现倒退

**本章应用：**
- 走出绿洲大厦后→金玉满堂诀效果逐渐放缓
- 不论如何观想现金数字→都无法再带来之前的底气
- 安逸生活积累的信心→被赵天行事件击碎
- **核心表达：** 心法不是万能的——当你的世界观被打破，再多的余额也无法给你安全感

**归档文件：** `写作资产/套路库/心法动摇·余额无法带来的勇气.md`
