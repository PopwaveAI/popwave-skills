# 第249章 前军用级 — 设计包 (v3)

## 一、核心概念

> **邓丙丁说明现状→杀人证+犯罪豁免证→最强办法：魂魄重创逼回去→让张羽当先锋→张羽讨价还价→福姬提醒：自废修为仍远超链气→谈条件→邓丙丁给2000万+蛟龙灵枢+3门筑基功法→前军用级功法→张羽决定先筑基→白真真转账被冻结→王总电话：要筑基来绿洲大厦我护法**

邓丙丁详细说明绿洲集团的现状和对付王胤的方案：王胤有杀人证和犯罪豁免证，最强办法是魂魄重创逼回上层。邓丙丁想让张羽当先锋，张羽讨价还价要好处。福姬提醒王胤自废修为后仍远超链气，必须筑基后才保险。最终谈成：2000万无息贷款+蛟龙灵枢法骸（租赁）+3门筑基功法（包括一门前军用级）。张羽决定先筑基，让白真真转账凑5800万买天灵根。但白真真转账时被冻结——王总打电话来："要筑基的话，一定要来绿洲大厦，由我为你们护法才行。"

**本章使命：**
1. **王胤的豁免系统揭秘**：杀人证+犯罪豁免证+二层审核→在一层几乎法律豁免
2. **邓丙丁的方案**：不打正面战争→魂魄重创逼他回上层治疗
3. **张羽的讨价还价**：从"拒绝"到"谈条件"→学习在昆墟做生意
4. **福姬的筑基警戒**：自废修为≠变弱→必须筑基才保险
5. **前军用级功法上线**：因仙道技术进步被淘汰为专家级，尚未上市
6. **白真真存款冻结**：王总从一开始就监控账户→只有他同意的筑基才能进行
7. **王总的"护法"邀请**：看似善意，实则枷锁

## 二、情节框架

### 主视角（张羽视角——谈条件 + 决定筑基 + 被冻结）

| 段落 | 内容 | 核心信息 |
|------|------|----------|
| **邓丙丁说明方案（A段，1-30行）** | 正神已调查绿洲违法→但搞不定王胤没用→王胤有杀人证+犯罪豁免证→最佳方案：魂魄重创逼他回上层 | 王胤的法律豁免╱正神的困境 |
| **张羽讨价还价·条件谈判（B段，31-73行）** | 福姬提醒：自废修为仍远超链气→张羽索要军用级物资→拉扯后谈成：2000万+蛟龙灵枢+3门筑基功法（前军用级） | 谈判过程╱前军用级功法 |
| **决定筑基·转账被冻结（C段，74-104行）** | 张羽决定先筑基→让白真真转账凑5800万→白真真账户被冻结→王总电话：来绿洲大厦我护法 | 王总全面控制╱自由筑基不可能 |

### 关键节点

1. **王胤的豁免系统**（第8-14行）
   - "他本人更是有着一层的杀人证和犯罪豁免证"
   - "大部分情况下，他就算是违法犯罪，也都能通过罚款来得到豁免"
   - "极少部分情况罚款也不行的，那也需要二层以上的部门对他的行为进行审核之后，方能实施抓捕"
   - 张羽："……这特么的狗屁昆墟"
   - **核心表达**：在昆墟，法律是给穷人准备的。有钱人买的不是"豁免"，是"提前批准的犯罪权"

2. **魂魄重创方案**（第15-16行）
   - "最简单有效的办法，还是直接将他魂魄重创，逼他回上面治疗"
   - 只要王胤不在→绿洲集团就是一盘散沙→各个击破
   - 因为一层难以治疗魂魄伤势→必须回上层
   - **核心表达**：上层下来的少爷最怕什么？怕昆墟一层的医疗条件治不了他——所以身体受伤还好说，魂魄受伤就得滚回去

3. **福姬的筑基警戒**（第34-40行）
   - "别听她乱吹。筑基期所谓自废修为，就是切断灵根和肉体的融合"
   - "灵根会废掉，法力和肉体强度也会严重倒退"
   - "但就算这样，道心等级能保留一部分，法力和肉体强度倒退再多，也终究还在链气之上"
   - "王胤的实力绝对还是远超任何链气修士"
   - "更何况这种人下来，鬼知道他装备了什么法骸，又带了什么护身法宝"
   - "你要和这个王胤一战的话，必须是筑基之后才保险一点"
   - **核心表达**：自废修为不是清零——金丹修士的底子、道心、装备全部碾压链气。链气打自废筑基≈蚂蚁打蟑螂

4. **谈判·前军用级功法**（第55-75行）
   - 张羽摇头："军用级飞剑、军用级法骸、神灵根、军用级功法……那才能斗一斗"
   - 邓丙丁：实体物资难→电子秘籍、功法版权容易给
   - 最终：2000万无息贷款+蛟龙灵枢法骸（租赁）+3门筑基功法
   - 其中一门：前军用级功法→因仙道技术进步被淘汰为专家级→尚未上市
   - "筑基期的功法，在一层本就难买，这本前军用级功法就更难得了"
   - 另两门：提供链气期功法的筑基后续→可在筑基后修到20级
   - **核心表达**：张羽第一次学习在昆墟"谈生意"——讨价还价不是丢人的事，是必须学会的生存技能

5. **账户冻结·王总的全面控制**（第92-104行）
   - 白真真转账→资金被冻结
   - 王总电话："当初公司给你们转账的时候，我就怕你们骤然得了这么多钱，拿去乱消费"
   - "所以有太大额的资金流动，都会通知我这边，由我用符咒来暂时帮你们锁一锁"
   - "你们要筑基的话，一定要来绿洲大厦，由我为你们护法才行"
   - **核心表达**：这9个月来的"自由"全是假象——王总一直在监控、在控制、在锁链上放长线。所谓"签约自由不用正神见证"——不是给张羽自由，是给王总不受约束的权利

## 三、人物设计

### 张羽 —— 学精了的谈判者

| 维度 | 分析 |
|------|------|
| **对邓丙丁方案的态度** | 不再直接拒绝→而是开始讨价还价 |
| **谈判技巧** | 摇头说"不行不行"→举出所需物资→等待对方加价 |
| **对军用级物资的渴望** | 谁不想要军用级？→前军用级功法让他狠狠心动 |
| **对筑基的急迫** | "筑基的事情不能再等了" |
| **对白真真的钱** | 本不想用她的钱→但局势变化只能先筑基 |
| **核心心态** | 从"被动接受"到"主动谈判"——张羽在昆墟的生存智慧升级了 |

### 邓丙丁 —— 高帽子师太

| 维度 | 分析 |
|------|------|
| **说服策略** | 戴高帽子→"少年英才，有勇力、气魄为一层的亿万苍生做出这番惊天大事" |
| **真实目的** | 找一个不怕被王胤记恨的"刀" |
| **出价底线** | 实体物资难给→电子秘籍、法骸租赁容易→功法版权最愿意给 |
| **对张羽的评估** | 知道对方在讨价还价→"见怪不怪"→"在昆墟找人办事，哪有不要好处的？" |
| **核心心态** | 用最小的代价找到最合适的"刀"——张羽是最佳人选 |

### 福姬 —— 实时提醒的风险顾问

| 维度 | 分析 |
|------|------|
| **对邓丙丁的拆穿** | "别听她乱吹"→王胤自废修为仍远超链气 |
| **对筑基的建议** | "必须是筑基之后才保险一点" |
| **对谈判的指导** | "向她多提点要求……" |
| **对筑基顺序的建议** | "先让白真真把钱转给你，助你买下天灵根" |
| **对羽书利用的建议** | 张羽有羽书→学筑基功法后实力提升更快 |

### 王胤（王总） —— 九个月前就开始布局的驯兽师

| 维度 | 分析 |
|------|------|
| **账户监控** | 从一开始就设置了大额资金流动通知→用符咒锁账户 |
| **表面说辞** | "怕你们乱消费"→"为你们好" |
| **真实控制** | 所有资金的自由都是假象→必须经过他同意才能用 |
| **对筑基的态度** | "一定要来绿洲大厦，由我为你们护法"→必须在眼皮底下进行 |
| **说话语气** | 和善→但却让张羽白真真感到彻骨寒意 |

## 四、设定细节

### 王胤的豁免系统

| 维度 | 内容 |
|------|------|
| **杀人证** | 持证可合法杀人 |
| **犯罪豁免证** | 大部分违法犯罪→罚款豁免 |
| **限制** | 罚款不行的情况→需二层以上部门审核→才能抓捕 |
| **一层实际效果** | 在一层几乎法律豁免→正神也拿他没办法 |

### 自废修为机制

| 维度 | 内容 |
|------|------|
| **机制** | 切断灵根和肉体的融合 |
| **后果** | 灵根废掉→法力和肉体强度严重倒退 |
| **保留项** | 道心等级保留一部分→法力和肉体强度仍在链气之上 |
| **真正威胁** | 法骸+护身法宝+装备→不知道有什么底牌 |
| **福姬评价** | "王胤的实力绝对还是远超任何链气修士" |

### 前军用级功法

| 维度 | 内容 |
|------|------|
| **来历** | 曾是军用级→因仙道技术进步被淘汰→降为专家级 |
| **市场状态** | 尚未在市场中开始销售 |
| **稀缺性** | 一层其他筑基修士花几千万也未必能买到 |
| **对张羽的吸引力** | 狠狠心动 |
| **获取条件** | 必须同意出手对付王胤 |

### 王总的账户控制

| 维度 | 内容 |
|------|------|
| **监控机制** | 大额资金流动通知→用符咒锁账户 |
| **合同设计** | 不找正神见证→对王总约束最小 |
| **表面理由** | 怕乱消费 |
| **真实目的** | 控制张羽白真真的每一步→包括筑基 |
| **持续时间** | 9个月（从第一次打钱开始） |

## 五、套路归档

### 套路名称：法律豁免·有钱人的犯罪许可证

**适用场景：** 揭示昆墟法律体系的双标——法律是给穷人准备的，有钱人买的是"提前批准的犯罪权"

**本章应用：**
- 王胤有杀人证+犯罪豁免证→大部分违法罚款即可豁免
- 少部分需要二层审核才能抓捕
- 正神只能通过网络层面削弱→不敢动王胤本人
- 张羽："……这特么的狗屁昆墟"
- **核心表达：** 在昆墟，法律不是保护弱者的——是强者用来管理弱者的工具

**归档文件：** `写作资产/套路库/法律豁免·有钱人的犯罪许可证.md`

---

### 套路名称：魂魄重创·逼上层少爷回家

**适用场景：** 对付从上层下来的"镀金二代"——最有效的手段是让他们受一层治不好的伤

**本章应用：**
- 正神方案：不打正面战争→魂魄重创→逼王胤回上层治疗
- 因为一层难以治疗魂魄伤势→必须回上层
- 王胤一走→绿洲集团就是一盘散沙
- **核心表达：** 上层下来的少爷最怕什么？怕一层医疗条件治不好他——身体受伤还好说，魂魄受伤就得滚

**归档文件：** `写作资产/套路库/魂魄重创·逼上层少爷回家.md`

### 套路名称：讨价还价·在昆墟谈生意的生存技能

**适用场景：** 主角学会在昆墟"谈生意"——摇头否定→举出所需→等待对方加价

**本章应用：**
- 张羽摇头说"不行不行"→"以我的水平，恐怕要有一口军用级飞剑……"
- 邓丙丁逐渐加价：2000万→蛟龙灵枢→3门筑基功法→前军用级
- 拉扯中试探出邓丙丁的出价底线
- 福姬指导："向她多提点要求……"
- **核心表达：** 在昆墟，不要什么都答应也不要什么都拒绝——谈，才有机会

**归档文件：** `写作资产/套路库/讨价还价·在昆墟谈生意的生存技能.md`

### 套路名称：九个月的假自由·王总的资金锁链

**适用场景：** 看似慷慨的资助其实是精心设计的枷锁——"自由"是假象

**本章应用：**
- 9个月来以为有钱了自由了→其实是王总在放长线
- 大额资金流动通知→符咒锁账户→不找正神见证合同
- "从九个月前开始，从他打钱给我们的那一刻开始，我们就必须经过他的同意"
- 要筑基→"来绿洲大厦，由我为你们护法"
- **核心表达：** 越大的馈赠，越深的控制——王总从一开始就没打算让张羽自由

**归档文件：** `写作资产/套路库/九个月的假自由·王总的资金锁链.md`
