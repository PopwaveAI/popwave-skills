## 2. 爽点设计（爽点层）

### 套路类型
- **反套路救美**：英雄非救美，反而亲手杀死受辱女子以绝其辱——传统期待的颠覆
- **黄雀在后雏形**：海贼内讧被海军打断，暗示海军才是更可怕的猎人
- **规则揭露**：揭穿"兵是兵匪是匪"的二元标签，点明大环境下身份随行为改变

### 情绪弧线
```
压抑（女子惨状）→ 愤怒（海贼闯入）→ 冲击（威廉拔刀杀人）→ 紧绷（对峙）→ 悬念反转（海军来临）
```

### 钩子
- 海军到底来了谁？（追捕他们的那位"灵活变通"的海军将领？）
- 两伙海贼能否一起逃生？巴蒙德日后会否报复？

## 3. 角色与人设（角色层）

### 登场角色
| 角色 | 本章要点 |
|:-----|:---------|
| **威廉** | 冷静果断的决策者；从受辱女子的处境果断判断最佳结局是死亡；剑术精准（来自原主人记忆的剑术技巧清晰浮现）；对标签有超越时代的认知 |
| **瓦隆** | 18岁，矮壮敦厚，大光头泛光，常年在田地劳作力气大；心地善良但缺乏判断力，阻止女子自杀 |
| **阿拉密斯** | 头脑灵活，海贼船生活经验最长；认为"该让她自杀的"，现实主义者 |
| **阿金** | 本章戏份少，与众人一同提武器戒备 |
| **受辱女子** | 年轻漂亮，被海贼强暴虐待后逃出；衣衫不整、披头散发、面目全非；眼神从惊恐转茫然至流泪死去 |
| **巴蒙德（二副）** | 赤上身健硕肌肉，胸膛有浅刀伤；暴躁、精虫上脑、以势压人；对威廉的抱团崛起感到被冒犯 |

### 关键对白
- 威廉："怎么，二副，换口味了，现在喜欢和死人交朋友？" — 冷漠讽刺
- 巴蒙德："威廉，你敢和我顶嘴？你是不是以为自己拉着几个毛都不知道长没长齐的臭小子抱团，在船上就是个人物了？" — 威胁试探

## 4. 设定/物品提取区

### 地点
| 名称 | 描述 | 特征 |
|:-----|:-----|:------|
| 酒馆 | 海贼团的临时落脚点，一天内大门被反复破坏 | 破旧、不安全，位于小镇 |

### 力量体系/身份标签
- 世界的基础逻辑：**"兵是兵，匪是匪"是大众思维**，但威廉穿越者视角认为这只是标签，标签随行为变化
- 海贼等级结构：船长→二副→普通海贼

### 角色/怪物数据
- **巴蒙德（二副）**：单手持刀，力量大，被威廉挡住一击后未继续攻击（保存体力）

### 战斗章数据
- 威廉的剑术：来自穿越前原主人记忆中的剑术技巧，在关键时清晰浮现，精准致命
- 战斗状态：持刀上架挡住劈头砍来的砍刀，格挡反击流畅

### 物品/资源
- 沾血迹的尖刀（女子从厨房拿的）
- 三个酒桶（威廉等人采购的补给）
