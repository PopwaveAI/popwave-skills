## 2. 爽点设计（爽点层）

### 套路类型
- **螳螂捕蝉，黄雀在后**：海军才是真正的黄雀——驱赶海贼去抢劫，跟在后面收赃物
- **双重反转**：读者先以为海军是救星 → 目睹海军黑吃黑 → 发现海军才是最恶毒的猎手
- **伏笔回收**：ch004中威廉推测海军将领"灵活变通"，ch005当场印证

### 情绪弧线
```
紧张逃命（海军追来）→ 庆幸脱身（船启动了）→ 悬念（军舰出现）→ 意外（没追）→ 震惊（海军才是黑手）→ 轻蔑（巴蒙德挑衅被无视）
```

### 钩子
- 海军指挥官的计划会持续到什么时候？下一座村镇是否更惨？
- 船长丹顿一直在码头附近活动"见势不妙开溜"——他对威廉是弃子还是另有打算？
- 威廉在船上夹缝中如何生存？

## 3. 角色与人设（角色层）

### 登场角色
| 角色 | 本章要点 |
|:-----|:---------|
| **威廉** | 早有预感船长可能开溜；逃命时果断放弃补给（"酒哪有命重要"）；对巴蒙德的挑衅选择隐忍、轻蔑回应，展现成熟的政治直觉 |
| **巴蒙德** | 逃命时脸上"一片惶急"不见之前的耀武扬威；回到安全环境立马恢复挑衅，欺软怕硬的流氓本性 |
| **小个子指挥官** | 名字未出现但形象鲜明——个子不高很有派头，白色风衣金色流苏，抽雪茄坐沙发；老谋深算，"驱狼吞虎"的冷血策略家 |
| **镇长** | 头发花白瘦骨嶙峋，在指挥官面前自觉弯腰弓背像罗锅；软弱但尽责，鼓起勇气为镇民讨公道 |
| **丹顿（船长）** | 本章不在场但通过侧面描写呈现形象：在码头附近徘徊准备"开溜"，抛弃来不及上船的部下 |

### 关键对白
- 指挥官："让这帮人再帮我们多搜刮一些战利品，在他们临死之前，多废物利用一下" — 全书迄今最核心的"黄雀"陈述
- 旁白总结："缺乏秩序的大环境下，谁说海贼就只能烧杀掠夺，谁又说海军一定会保境安民了？" — 主题句

## 4. 设定/物品提取区

### 地点
| 名称 | 描述 | 特征 |
|:-----|:-----|:------|
| 小镇码头 | 海贼船停泊、海军军舰靠近的核心地点 | 多桅帆船开动需要时间和人手 |
| 海贼船甲板/船舱 | 破败如幽灵船，人手不足开动缓慢 | 威廉等人操弄缆绳控制风帆 |

### 海军体系
| 项目 | 描述 |
|:-----|:------|
| 制服 | 海员服，系蓝领巾的普通士兵；白色风衣、肩携金色流苏的指挥官 |
| 装备 | 军舰（速度优于破败海贼船），火炮（可远程轰击） |
| 行为准则 | **不保境安民、黑吃黑**——本章核心设定冲击 |
| 战术 | "驱赶"海贼去村镇劫掠，跟在后面收赃物，最后再杀海贼灭口 |

### 海贼世界生态
- 船长有权抛弃来不及登船的海贼
- 破败的帆船在航海士指挥下协调操帆才能开动
- 被抛弃的海贼无船无补给，基本无法生还

### 物品/资源
- 堆积如山的财物——海贼抢来的/从本地刚抢的，全部被海军充公
- 海图——海军搜遍被俘海贼身上未找到，暗示有航海士的海贼察觉了海军的驱虎吞狼策略
