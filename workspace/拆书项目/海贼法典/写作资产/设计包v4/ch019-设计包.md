## 2. 爽点设计（爽点层）

- **对比铺垫，层层递进**：从和道一文字（锋利无阻）→ 破旧打刀（费力但也砍断）→ 刀自己不会砍（活剑vs死剑），三次递进让剑理具象化，读者瞬间明白"人剑关系"。
- **认知错位爽感**：弟子们以为耕四郎在讲"刀要锋利"或"爱护刀具"，只有威廉（和读者）知道没那么简单。当耕四郎对和道一文字说"你去砍"时，认知翻转，俯瞰全场的智力落差带来爽感。
- **作者注破墙**：特意说明"他"不是错别字，强化活剑概念的记忆点。

## 3. 角色与人设（角色层）

| 角色 | 本章亮点 |
|:-----|:---------|
| **耕四郎** | 借两把刀的对比教学展现其"一心流"核心理念——重人不重器；将名刀呼作"他"，暗示剑有灵性是其深信之事；表面笑眯眯，教学内容层层递进引导，节奏掌控力极强。 |
| **爱德蒙** | 对名刀的痴迷（恋物癖嫌疑）、大大咧咧但分场合、基本功扎实（被耕四郎肯定"形"的标准）。行动粗放但认真听指令。 |
| **威廉** | 始终保持旁观者的清醒判断（"没那么简单"），用前世剑道知识解构当下课堂，见识层次远超同龄弟子。 |
| **阿拉密斯** | 插科打诨、好问（"今天不讲大道理了？"）、最后肤浅理解（"我算明白为什么眼馋和道一文字了"），与威廉形成对比。 |
| **瓦隆** | 无具体台词，侧面带过，身份为小山头成员。 |
