## 2. 爽点设计（爽点层）

- **认知落差收束**：上半章"死剑"教学以刀为死物收束"活剑"悬念，借和道一文字自己不会动的直观画面把哲理落地，读者获得"原来如此"的满足感。
- **命运广场系统激活**：首次在醒时触发命运广场信息，暗示系统能力随成长升级——系统文读者获得"变强了"的确认感。
- **命运之线镇黑暗**：阿金的单线+几位伙伴的银线联手镇住道路黑暗，既是伙伴关系的一次具象化展现，也是"羁绊=力量"的浪漫表达。
- **格局拔高**：后半章从剑术修行跳到海贼格局分析（亨利·摩根×德雷克→有情报有渠道的海枭），拉高读者的期待视野——主角不当没头苍蝇，要做幕后操盘手。

## 3. 角色与人设（角色层）

| 角色 | 本章亮点 |
|:-----|:---------|
| **耕四郎** | 以"刀离开人就是死物"完成"活剑vs死剑"完整教学闭环；好老师特质：讲完后期待学员反馈（"你们明白了吗？"），不因全员摇头而失望。 |
| **威廉** | 命运广场系统首次在醒时触发，世界观级别标记；入梦走通【剑术奇才】之路时果断不回头（"也不看身后的广场"），体现其向前看的性格；后半段展示深沉布局意识——要做"狼"而非"没头苍蝇"。 |
| **瓦隆** | 摇头如拨浪鼓，突出头脑简单四肢发达的特质。 |
| **阿拉密斯** | 幸灾乐祸（"嘿嘿直乐"），仍然是插科打诨者定位。 |
| **爱德蒙** | 本章被动工具人——被耕四郎点名举着锈刀傻站，持续被用来证明刀不是根本；并被威廉吩咐做事（搜集报纸信息）。 |
| **阿金**（命运线） | 金线先导，暗示其命运紧密程度高于其他伙伴。 |
| **命运广场** | 新增【逐利者】道路（威廉不准备走）；走通道路条数可能影响命运广场的覆盖范围。 |
