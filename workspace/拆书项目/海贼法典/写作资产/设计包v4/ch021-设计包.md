## 2. 爽点设计（爽点层）

- **开幕反差**：某地禁酒令前的社会狂欢——抬棺放酒瓶、推车放酒瓶、逢人塞酒瓶——荒诞夸张的众生相让读者先"看热闹"，再层层揭秘。
- **上层叙事揭露**：阿尔卡蒂奥的演讲感人肺腑（女儿被害），转脸对儿子说"别让真实性毁掉好故事"，反转干净利落。"美酒如黄金，不只我一个人能看出来"点出禁酒令本质是一场经济/权力博弈——读者获得"看穿权谋"的智识爽感。
- **新角色三人群像**：托尼（疤面/暴躁/率先展示腰间燧发手枪——暗示对威廉有戒备）、曼尼（骚包英俊→被骂）、矮壮手下（无台词但有存在感），三人组出场即立住差异化，铺垫后续戏份。
- **爱德蒙父亲遗作**：两本小说（被刀逼着印刷→孤本），荒诞中有笑点，为爱德蒙角色增加家庭情感底色。

## 3. 角色与人设（角色层）

| 角色 | 本章亮点 |
|:-----|:---------|
| **阿尔卡蒂奥伯爵** | 纳维亚王国财政大臣，布恩迪亚家族家主。表面温文尔雅（燕尾服、八字胡一丝不苟），实则精于编造故事操控舆论；禁酒令的真正动机是"美酒如黄金"——经济利益与权力整合。台词狠辣老练。 |
| **奥雷利亚诺** | 阿尔卡蒂奥之子，年轻、较真（追问码头上低贱水手怎么可能接触妹妹），被父亲训斥"傻小子"。 |
| **萨萨里安** | 闪金商会会长，外表和气有涵养，作为外来视角见证卡尔马的狂欢。商会名字"闪金"="低调不起来"的幽默反差。 |
| **爱德蒙** | 啃指甲看父亲遗作，对儿童文学百看不厌；威尔认为"更离谱的是瓦隆也很喜欢看"——表现爱德蒙情感粗放中的细腻一面。 |
| **瓦隆** | 从爱德蒙肩膀后伸大光头看书，依旧大老粗可爱定位。 |
| **托尼（疤面）** | 新登场角色，二十四五岁，黑色卷发/鹰钩鼻歪断/刀疤划破左眼/左眼习惯性抽动/嘴角下拉——一看就不是好惹的人。性格暴躁、骂手下、出门带两壮汉。一见面就亮燧发手枪暗示警戒，是对威廉的试探+宣示武力。 |
| **曼尼** | 托尼手下，高个英俊长发抹发蜡，被骂臭美，与周围粗犷氛围形成反差。 |
| **矮壮手下** | 无台词，微卷黑发/朝天鼻/小眼睛/微胖壮实，存在感弱但成为三人组构成拼图。 |
