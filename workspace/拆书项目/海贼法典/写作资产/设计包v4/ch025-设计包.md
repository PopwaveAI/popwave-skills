## 2. 爽点设计

| 爽点 | 类型 | 触发位置 | 设计说明 |
|:-----|:----|:---------|:---------|
| 手无缚鸡之力的富家子逆势谈判 | 智斗·反转 | 事件4→5 | 秘书刚被杀、船已离岸、双手被绑——夏洛克却凭一张利嘴将死局扭转成可谈判的局面，呈现"弱者不靠武力靠头脑"的反差爽感 |
| 三倍抬价的心理博弈 | 谋略·压迫感 | 事件5 | 从杀他领一千万 → 绑架换三千万 + 赎船员额外收入，让海贼利益最大化，逼迫对方重新算账。这是"用对方贪婪反噬对方"的经典博弈 |
| 萨萨里安父子的性格对撞 | 人设·铺垫 | 事件1 | 保守谨慎的父亲 vs 大胆冒进的儿子，为后续夏洛克与威廉的走私合作埋下性格伏笔 |

## 3. 角色与人设

| 角色 | 定位 | 本章关键表现 | 人设标签 |
|:-----|:-----|:------------|:---------|
| **夏洛克** | POV+关键人物 | 从回忆父亲引入→通宵研究→遇袭→冷静谈判；全程展现"商人本色" | 手无缚鸡之力、胆大、精于算计、追求效率、不喜排场、大龄中二叛逆期 |
| **萨萨里安**（父） | 背景提及 | 保守谨慎、和气生财型商人；夏洛克叛逆的对象 | 稳重、保守、地方商人领袖 |
| **脏辫男**（海贼首领） | 反派（中层） | 首次登场，持枪杀人、控制全场，但被夏洛克的抬价话术动摇 | 黝黑皮肤、脏辫、凶悍但见识有限的三流海贼 |
| **大嘴男** | 反派（跟班） | 认出"闪金商会"名号，向首领解释 | 大嘴特征、略有见识 |
| **坑洼脸男** | 反派（跟班） | 捆绑夏洛克双手、推搡 | 满脸坑洼、粗暴 |
| **托尼** | 联络人（声音登场） | 传递威廉提议、安排约见 | 中间人、混混头目 |

### 人设揭露

- **夏洛克的核心动机**：骨子里追求"大胆"的利润，信奉"人有多大胆地有多大产"；叛逆期未过但极具商业头脑，生死关头仍优先盘算生意

## 4. 设定/物品提取区

| 设定/物品 | 类别 | 说明 |
|:----------|:-----|:-----|
| **贝利** | 货币 | 本世界统一货币，体量相当于前世日元。秘书被杀订金一千万贝利，夏洛克开价三千万 |
| **闪金商会** | 组织·势力 | 地方商会，萨萨里安任会长，在王国首都推广本地产品，有一定影响力 |
| **电话虫** | 物品·通讯 | 本世界电话设备，能惟妙惟肖模仿通话者的表情神态（文中提到电话虫学习托尼手忙脚乱的模样） |
| **禁酒令** | 设定·政策 | 纳维亚王国的禁酒政策，夏洛克通宵研究的新闻，是走私生意的背景政策 |
| **纳维亚王国** | 地理·国家 | 威廉走私生意涉及的目的地国家，施行禁酒令 |
| **红叶村** | 地理·地点 | 威廉和托尼所在的地点，夏洛克此行目的地 |
| **商会会长职务** | 社会制度 | 由当地德高望重者担任，一举一动受人关注——这解释了夏洛克不向父亲求助的深层原因 |
