## 2. 爽点设计

| 爽点 | 类型 | 触发位置 | 设计说明 |
|:-----|:----|:---------|:---------|
| 世界四大病根的底层逻辑揭示 | 设定·深度 | 事件1→2 | 将海贼王世界混乱表象提炼为四大结构性原因，给读者"原来如此"的阅读满足。本章的爽不在情节张力，而在认知增量 |
| "资本不只包括钱"——武力宣言 | 霸气·反转 | 事件4 | 读者和托尼一样焦虑钱不够时，威廉敲响刀鞘——"资本"被重新定义。从商业线转入武力线的标志性转折 |
| 孤身赴会的鹰隼式登场 | 视觉·帅气 | 事件5 | 威廉独自乘小船→跃身登船→提两箱现金的视觉效果；"像一个英雄走入敌营"的经典爽感 |

| 海贼两难的犹疑状态 | 掌控·智斗 | 事件7 | 威廉虽然孤身、没带武器，但舆论和局势已经被控制住——海贼在纠结时已经输了主动权 |

## 3. 角色与人设

| 角色 | 定位 | 本章关键表现 | 人设标签 |
|:-----|:-----|:------------|:---------|
| **威廉** | 唯一POV | 世界观解剖→现金不足的困境→武力暗示→孤身赴会登船对峙。从思考者到行动者的完美过渡 | 穿越者思维（对比前世）、冷静理性、有勇有谋、一人敌一船的胆色 |
| **托尼** | 情绪对照 | 焦虑、踱步、只能筹到几百万——用普通人的反应反衬威廉的格局 | 现实焦虑型、有局限的好帮手 |
| **阿金** | 信念锚点·武力线 | 一句"资本又不只包括钱而已"引爆全章——他是最懂威廉的人 | 忠诚、行动派、信任威廉到无需解释 |
| **爱德蒙** | 在场（沉默） | 本章零台词，被注为"怪人" | 伏笔角色 |
| **爆炸头/坑洼脸男** | 反派·甲板接应 | 制止暴躁海贼对威廉动武、主动提出先让威廉见夏洛克——有一定的底线和判断力 | 粗犷但未被贪婪完全吞噬、对夏洛克有好感 |
| **暴躁持刀海贼** | 反派·杂兵 | 想抢皮箱、拔刀欲上——代表纯粹的贪婪与暴力 | 冲动、无脑、工具人 |

### 人设揭露

- **威廉的核心特质**：穿越者的对比视角（用前世对照本世界）让他对世界有清醒的判断；能从社会结构高度推导死亡哲学，说明他有超越个人恩怨的思维格局
- **威廉的武力底牌**：本章正式暗示——敲刀鞘+鹰隼登船+孤身赴会，明白告诉读者威廉不是只靠嘴的

## 4. 设定/物品提取区

| 设定/物品 | 类别 | 说明 |
|:----------|:-----|:-----|
| **世界政府四大结构病** | 设定·世界 | ①机构臃肿腐败（天龙人特权+奴隶文化=半殖民式） ②海军基地指挥官军阀化（自行征兵+人事不常调）③封建王朝贵族出身论 ④海贼蝗虫式侵害。逐条列举，形成逻辑闭环 |
| **大海贼时代** | 设定·时代 | 开启后海军重心转向伟大航路，各地基地权力下放，形成军阀土壤 |
| **无风带** | 设定·地理 | 导致各地联络沟通不畅，是地方割据的地理原因之一 |
| **"歌颂死亡"的社会哲学** | 理念·核心 | 死亡在混乱世界中随处可见→人们被迫轻视死亡→反而渴望在有限生命中留下被人牢记的成就→孕育出大批"奇人异士"和豪杰。这是本章的核心理论贡献，为后续所有大胆角色的行为提供世界观层面的合理性 |
| **两千万贝利皮箱** | 物品·道具 | 现金的视觉冲击（一捆捆扎好的贝利）——在贫穷的东海底层海贼眼中极具震撼力 |
| **威廉的灰色衬衫+黑色长裤** | 形象·服饰 | 本章首次具体描绘威廉赴会时的穿着：简洁、利落、不藏武器但也不拖泥带水 |
| **商船甲板场景** | 场景 | 海贼半包围+威廉被围中央+两侧皮箱——西部片式的对峙构图 |
