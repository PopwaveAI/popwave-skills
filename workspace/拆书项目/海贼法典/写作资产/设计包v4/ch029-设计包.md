## 3. 角色与人设（角色层）

| 角色 | 本场表现 | 人设亮点 |
|:----|:--------|:--------|
| **威廉** | 从杀手切换到谈判家的无缝转型 | 1. 冷静分析专长机理（理工思维） 2. 商业谈判中主动控场、抢占心理高地 3. 以战果作为谈判筹码——「墙上那具死尸比什么都能宣示实力」 |
| **夏洛克** | 逐步展现商人胆识和洞察力 | 1. 虽被绑架却不失态，握手从容 2. 迅速从惊吓中恢复，开始讨价还价 3. 敢在屠场谈判——说明胆量极大，但仅限于商业事务 |
| **海贼头领** | 被秒杀，沦为背景道具 | 钉在墙上的尸体成为威廉实力的活招牌 |
| **海贼喽啰** | 被吓破胆的幸存者 | 胆小怯懦，尿裤子、筛糠发抖——衬托威廉的恐怖 |

## 4. 设定/物品提取区

| 类别 | 名称 | 说明 |
|:---|:-----|:-----|
| 🧠 专长 | **闪电反射（深入设定）** | 提供的是「潜力」而非直接达到子弹速度；让威廉能利用预判配合极小幅度的动作躲避子弹——「躲子弹不一定要比子弹快」，关键在于提前观察到枪口指向并做出最小位移 |
| 🧠 战斗哲学 | **感知时间流速变化** | 触发专长时，时间在威廉感知中「仿佛一滞」，对手的动作慢动作化，而实际时间流速不变 |
| 🔫 武器 | **燧发手枪（弹药）** | 单发武器，发射时弹簧带动燧石击打火门产生火星点燃火药 |
| 🧴 道具 | **香水手帕** | 夏洛克随身携带的喷了香水的手帕，用以掩鼻抵御血腥味——体现其精致公子做派 |
| 🧾 商业思维 | **现场谈判法则** | 商务谈判是无所不用其极的战场，实力不只指金钱，还包括护航的武力；乱世中做生意需要能震慑各路海贼劫匪的实实在在武力 |
| 🌍 世界观 | **乱世经商法则** | 和平年代靠政府和官方力量；乱世中商人也必须依靠自身的武力保驾护航 |
