## 3. 角色与人设（角色层）

| 角色 | 本场表现 | 人设亮点 |
|:----|:--------|:--------|
| **威廉** | 谈判家和团队领袖 | 1. 敢要主导权——「你只出钱，我们出命」 2. 懂得进退——同时宣布也会出钱 3. 有商业远见——药坊伪装、低端市场策略 |
| **夏洛克** | 商业天才，胆识过人的合伙人 | 1. 闻「亡命徒」而喜——追求高风险高回报 2. 务实精明——不做无谓的面子之争，要的是利润 3. 家世背景初现——疑似大家族内斗受害者 |
| **瓦隆** | 纯粹的战士 | 光头上沾血但身上无伤，战斗实力可靠；事后酷爱冒险小说——外表憨厚内心闷骚 |
| **阿金** | 伏兵队长 | 带人藏暗舱，执行威廉的潜入计划 |
| **阿拉密斯** | 团队中的「狠人」 | 打断幸存者双手，给其教训但不致命 |
| **爱德蒙** | 务实派+质疑者 | 对走私销路有疑虑；与瓦隆惺惺相惜 |
| **托尼** | 牵线人/本地支持者 | 负责搭桥，对生意细节有观察（对口感有疑虑） |
| **奥马尔** | 村长/老江湖 | 奸诈笑容、洞悉本地下层需求——喝酒是男人的刚需 |

## 4. 设定/物品提取区

| 类别 | 名称 | 说明 |
|:---|:-----|:-----|
| 🚢 战术 | **暗舱伏兵** | 威廉的小木船设有暗舱，阿金等人在船中藏身，通过轻踢护板作为暗号联络 |
| 💰 商业模式 | **低端市场优先战略** | 夏洛克的商业判断：低端市场才是大头；节省运费，高端酒用来贿赂官员更划算 |
| 🏥 伪装 | **药坊伪装卖酒** | 威廉借鉴美国禁酒令时期的做法（源自《了不起的盖茨比》），以医用酒精名义注册药坊，实际售卖酒类 |
| 🍺 商品 | **酒类走私（禁酒令背景）** | 本世界某个区域实施禁酒令，原本几百贝利的酒可卖到几千甚至上万贝利，利润巨大 |
| 📚 人物设定 | **瓦隆的闷骚性格** | 识字后就酷爱冒险小说，向往波澜壮阔的冒险故事，证明他并非单纯的战士，内心有浪漫追求 |
| 👨‍👩‍👧‍👦 背景 | **夏洛克家族内斗** | 夏洛克出身两代富贵，并非真正豪门；有「不成器的亲戚」觊觎家产，疑似策划绑架 |
| 🌍 世界观 | **红叶村·高层关系** | 托尼的叔叔奥马尔是红叶村村长，地方势力与威廉团伙关系密切 |
