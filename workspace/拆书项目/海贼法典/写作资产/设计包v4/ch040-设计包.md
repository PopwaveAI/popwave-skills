## 2. 爽点设计（爽点层）

### 套路清单
- **幕后真相揭秘套路**：从迷宫到广场到雕像到系统，层层剥开世界观核心，满足读者对"为何穿越"的终极好奇
- **力量体系揭秘套路**：命运广场→命运之路→暗能量→命运之线，完整的力量获取逻辑链条
- **风险揭示套路**：探索者高死亡率、弹簧反弹、烈性病毒vs潜伏病毒的对比，制造危机感的同时埋下策略伏笔
- **霸气宣言爽点**："我就是命运之癌"——从被动的实验体到主动的"病毒"自我定位，角色意志第一次宣言

### 情绪弧线
```
好奇(发现广场) → 震惊(得知实验真相) → 绝望(无法回家/高死亡率)
→ 理解(系统规则明朗) → 试探(提出杀主角) → 获得策略(潜伏病毒比喻)
→ 升华("命运之癌"宣言)
```

### 钩子设计
- ☑ **世界观钩子**："零点宇宙"和"841205幻想世界"的设定打开了后续世界观的大门
- ☑ **成长方向钩子**：明确了"潜伏性改变命运"的战略方向，铺垫后续一切操作的底层逻辑
- ☑ **悬念钩子**：高等文明是谁？其他个位数存活的探索者在哪？为中期冲突留白

## 4. 设定/物品提取区

### 世界观核心设定

| 设定 | 类型 | 说明 |
|------|------|------|
| 零点宇宙 | 世界观基石 | 支撑一切幻想世界形成的"土壤"，零点宇宙原住民可在其他幻想世界长期生存 |
| 探索者(实验对象) | 身份设定 | 被高等文明选中投入幻想世界进行穿越实验的人，编号17631号 |
| 841205幻想世界 | 世界编号 | 威廉被投入的海贼王对应幻想世界ID |
| 命运广场 | 系统核心 | 威廉意识深处/绑定空间，大理石广场+雕像组合，随暗能量升级 |
| 命运之路 | 系统机制 | 推动威廉不断冲击既定命运、产生能量的"道路" |
| 暗能量(弹力) | 核心资源 | 幻想世界命运弹簧受挤压后意图回正的作用力，用以升级广场和获取专长 |
| 命运之线 | 系统数据具现 | 受威廉影响而改变既定道路的人们，数量与坚韧度反映影响程度 |
| 弹簧模型 | 核心比喻 | 越激烈挤压→反作用力越强→暗能量越多，但反噬风险也越大 |
| 病毒模型 | 策略比喻 | 烈性病毒(剧烈颠覆)短期内影响大但易被攻克；潜伏性病毒(隐蔽影响)才能长期传播 |

### 世界观规则

| 规则 | 详情 |
|------|------|
| 高等文明与低等文明 | 约束高等文明的只有道德，没有法律——与原著世界政府的"绝对正义"形成对比 |
| 探索者坐标锁定 | 威廉进入后世界坐标自然变动，连送他来的文明也无法再定位 |
| 跨世界力量隔离 | 命运广场的暗能量只能产出本世界(海贼王)体系的力量，无法兑换外世界能力 |
