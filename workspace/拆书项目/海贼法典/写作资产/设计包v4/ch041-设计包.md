## 2. 爽点设计（爽点层）

### 套路清单
- **数值面板爽点**：完整的RPG式角色面板展示——技能、专长、成就、背景分门别类，满足读者"看数据涨"/建立角色认知的爽感
- **兑换兑现爽点**：用辛辛苦苦得到的独特命运线兑换【健壮】，即时获得可感知的属性增长（力量+耐力+免疫力）
- **小伙伴羁绊（具象化）**：阿金的金色雕像成为命运广场第一个"纪念碑"，友情/信任获得系统级确认
- **首脑决策爽点**：威廉在危机中展现领导力——止损（停走私）→保基（作坊转型）→稳住核心战力（让夏洛克专注父亲）

### 情绪弧线
```
理解(系统规则愈发清晰) → 权衡(兑换哪项专长) → 期待(选择健壮)
→ 惊叹(阿金雕像具现) → 满足(面板展示/数字增长)
→ 沉稳(转场至红叶子/决策场景) → 信任(夏洛克困境/威廉承诺)
```

### 钩子设计
- ☑ **成长预演钩子**：逐次叠加的独特命运之线需求(1→2→4→…)，暗示后续需要与越多重要角色建立羁绊
- ☑ **剧情线钩子**："红胡子我们自己处理"——为下章直接行动做铺垫
- ☑ **身世钩子**：夏洛克父亲的狱中处境，闪金商会线待解
