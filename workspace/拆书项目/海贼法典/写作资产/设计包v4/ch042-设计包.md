## 2. 爽点设计（爽点层）

### 套路清单
- **反审讯操作指南**：水刑——中世纪到CIA经久不衰的酷刑技术展示，干净利落无血腥但压迫感拉满
- **智商碾压/心理战爽点**：威廉的审讯节奏——先礼貌问名(软)→对方硬气→水刑(硬)→问简单问题(破防)→打断思路突袭关键提问(致命一击)
- **"抓的就是你"爽点**：沃德搬出红胡子威吓，威廉一句"我笑是因为我没有抓错人"——身份反转，猎人与猎物的位置瞬间锁定
- **情报获取爽点**：多轮水刑层层推进，最终撬出核心战力情报"恶魔果实"——全员震惊的集体反应强化了情报价值
- **伙伴羁绊维护**：爱德蒙复仇情绪失控→阿金拉住安抚→威廉指示——团队内部的分工与信任

### 情绪弧线
```
悠闲(沃德醉酒哼歌) → 突受惊吓(被打晕) → 困惑惊恐(醒来被绑)
→ 虚张声势(搬出红胡子) → 现实打击(水刑开始) → 溺毙恐惧(窒息体验)
→ 心理崩溃(招供基础信息) → 高压认知(再次水刑) → 慌乱泄密("恶魔果实")
```

**读者情绪对照：**
```
兴趣(设伏开始) → 紧张(水刑施压) → 痛快(有效信息提取)
→ 震惊(恶魔果实情报) → 期待(后续怎么打)
```

### 钩子设计
- ☑ **战力钩子**：红胡子是恶魔果实能力者——但具体是什么果实？能力强弱？这是下一段冲突前最后一个情报缺口
- ☑ **名字钩子**：船上有个叫"克洛"的年轻人——熟悉原著的读者会立刻警觉：百计克洛？这是名草帽团前期对手
- ☑ **地名钩子**：卡尔马城附近的私港——下一个行动坐标

### 关键对白

> **沃德（开场）：** "老子可是'红胡子'巴巴罗萨手下的海贼！"
> → 虚张声势的威胁，反而确认了抓的人没错

> **威廉：** "我笑是因为，我没有抓错人。"
> → 同句双重意思，反身定位猎手身份

> **沃德（崩溃后）：** "我只知道他吃过恶魔果实……"
> → 全场屏息的收尾，信息炸弹
