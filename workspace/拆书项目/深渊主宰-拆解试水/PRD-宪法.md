# PRD · 深渊主宰
> 这是一本 DND 3.5e 西幻·黑暗升级流小说。一个从贫民区底层起步的冷酷主角，凭借数据面板的智力优越感和对力量本质的深刻理解，在诸神陨落与回归的深渊边缘，爬出一条弑神之路。

## 〇、核心故事

贫民窟半精灵少年在濒死时觉醒了一块只有他看得见的光屏——六维属性数字跳动，一行冰冷提示浮现：「击杀生物获得经验值以升级」。这个世界的诸神已沉睡六百年，魔网崩溃后法师沦为废人，而一种名为「经验狩猎」的地下活动正在蔓延：有人专门猎杀拥有面板的人。他只想攒够钱带妹妹离开这座吃人的城市，却发现街角乞丐头顶悬浮着等级标签、下水道里的战斗会弹出经验条——世界运行的底层规则，竟然是一套他完全读得懂的 DND 游戏手册。而那些沉睡的神祇，正在陆续醒来。知道这个秘密的人，都在被神追杀。

## 一、爽点锚点

### 锚点①：数据面板的智力优越感
主角知道这个世界力量体系的底层规则（属性阈值、经验获取、兼职惩罚），而绝大多数角色不知道。这种信息差是全书最核心的爽感发动机。读者通过主角的"我懂你不懂"获得智力优越感，每一次借由规则认知实现的越级击杀都是对这种优越感的确认。
- **特征**：信息差驱动型爽感；核心触发场景为"利用敌人不知道的规则取胜"
- **情绪曲线**：信息收集 → 规则认知 → 陷阱布置 → 碾压反转
- **强度**：全书最高频爽点类型，贯穿每个阶段

### 锚点②：阶级攀爬 + 碾碎上位者
主角从社会最底层（贫民区）出发，经历地位重置后重新向上爬升。每个阶段的核心快感在于：让那些俯视主角的高位者，一个个被主角超越、击败、碾碎。这是一条从蝼蚁到弑神的阶级复仇线。
- **特征**：社会地位差驱动型爽感；地图初期被轻视 → 地图末期碾压
- **情绪曲线**：底层蛰伏 → 实力积累 → 地位跃迁 → 碾碎旧上位者
- **强度**：每张地图中后段集中释放，全书级贯穿

### 锚点③：护住唯一在乎的人
主角的一切冷酷行为有一个情感锚点——保护唯一在乎的人（情感锚点角色）。这个锚点让冷血主角拥有可共情的软肋，也让所有"不择手段"有了读者可以接受的动机。情感锚点角色的安危是全书最紧张的悬念线。
- **特征**：亲情驱动型剧情推进器；危机触发主角极限潜能
- **情绪曲线**：守护日常 → 危机威胁 → 拼死爆发 → 暂时安心 → 下一次危机

## 二、核心卖点

### 设定卖点
DND 3.5e 硬核面板体系不是穿越者赠送的金手指——它就是这个世界的物理定律。六维属性、职业等级、法术环位、专长系统、九阵营——这套经过二十年跑团玩家验证的规则体系，直接作为世界观底层架构运转。杀怪得经验、属性阈值质变、兼职惩罚、超凡门槛，每一个数字都是世界法则本身。读者不是在读一个借用游戏设定的小说——他读的就是那本规则书的故事版。

### 世界观/理论体系卖点
诸神黄昏式的神祇陨落与回归框架：神灵可被杀、魔网可崩溃、施法者可瘫痪。多元宇宙位面体系完整呈现——上层位面、下层位面、物质位面、星界之间存在稳定通道。力量不是玄学感悟而是可量化的宇宙常数，智慧而非蛮力才是理解世界本质的钥匙。这套世界观暗含一个理论洞见：如果现实真的是一套可计算的规则系统，那么「知识」本身就是最强大的武器——知道规则的人操纵规则，不知道规则的人被规则操纵。

### 剧情卖点
一条从贫民窟杀到封神仪式的完整阶级攀爬线。主角不是被神明选中的天选之子——他是被意外卷入神魔棋盘的局外人。每一次地图转换都是外部灾难的强加（城市被毁、战争波及、被追杀逃亡），每一次地位重置都意味着从头碾压。读者期待的剧情体验是：看着一个一无所有的底层少年，用规则认知越级击杀、用信息差碾压高位者、用冷酷计算在诸神博弈的夹缝中活下来，直到有一天他站在棋盘上跟神明对弈。

### 主角/金手指卖点
主角是守序邪恶的完美范本——冷酷计算到令人胆寒，但单点情感锚点（唯一在乎的人）让他拥有可共情的软肋。他不滥杀无辜但也不会为救陌生人冒一分风险。他的金手指不是无敌外挂——游戏面板只是「把世界规则翻译成人类可读的语言」，信息才是真正的金手指。金手指的设计哲学是：不给主角任何世界规则之外的超模能力，只给他解读规则的能力。优势来自认知差，而非特权。

### 全书级卖点
完备的防崩体系确保力量体系从开篇到终章不倒：高阶不出现在低阶地图、越级必须付出可感知代价、每张地图重置社会地位、传奇级角色不与主角直接交手、每卷末主角损失战力以降低下卷起点。这套防崩机制不是限制爽感，而是确保每一次升级、每一场越级、每一次碾压的爽感都不会被「力量体系膨胀」稀释。读者可以信任每一个数字的含义——因为在我们的书里，数字真的有重量。

## 三、美学基因 + 命名体系

### 美学基因

**基因①——西幻 DND 硬核**：全书的力量体系、世界观架构和叙事质感完全锚定于 DND 3.5e 的底层逻辑——属性阈值、职业等级、法术环位、专长系统、九阵营、挑战等级。世界有一套可被认知和计算的规则，而非模糊的玄学运作。**排他**：拒绝东方修仙的"积累—突破"成长范式（灵气、境界、功法、炼丹），拒绝日式轻小说的等级膨胀和战斗华丽化（必杀技、觉醒、变身），拒绝一切让力量体系变得模糊或情绪化的叙事处理。

**基因②——底层冷酷视角**：全书叙事始终从社会最底层或地位被打回原形后的重置状态出发。世界被感知为灰的——不是中二的黑，不是少年漫的白，是贫民区泥墙、下水道苔藓、旧伤疤那种灰。主角的每一个选择都首先基于生存计算，而非正义或热血。**排他**：拒绝英雄主义的叙事惯性（主角不会为了别人主动冒险），拒绝热血少年漫的正向情感驱动（战斗不是为了守护而是为了活下去和变强），拒绝虚无主义的全黑无底线（主角有自己的规则和边界）。

### 从基因到世界质感

因为基因①提供了一套精确、可计算的世界规则，而基因②要求叙事始终从底层视角出发，所以世界的物理质感必然呈现为**粗粝的中世纪质感**——不是经过美学提纯的中世纪奇幻（精灵的银白宫殿、法师的浮空塔），而是现实向的中世纪：泥泞的街道、锈蚀的铁器、经过反复修补的皮甲、廉价麦酒与牲畜粪便混合的气味。城墙内外是两个截然不同的世界，同一座城市的不同城区也拥有完全不同的空气味道和地面材质。

因为底层视角意味着主角始终处于感官的劣势位置（看不远、听不清、信息被过滤），所以本书的感官序列遵循**气味先于视觉**的原则。在幽暗的地下道或封闭的街巷中，气味是主角最先获取的环境信息——铁锈与血腥味预示战斗在附近发生过，腐烂的甜味暗示尸体，廉价油脂与汗味暗示人群聚集。视觉在高阶时成为可靠感官（+超凡感知），但在低阶阶段，气味和听觉是主角生存的主要依赖。

因为基因①的力量规则严格、基因②的社会地位差恒定，所以**空间的阶级性**是世界的结构性特征。下层区通向贵族区的道路不仅是景观切换，更是生存法则的切换——主角在上层要遵守的行为规范与在下层完全不同。不同社会阶层的空间不仅在物理上被隔开（石墙、铁门、卫兵岗哨），在感官上也被隔开（上层有香料和鲜花，下层只有污浊的空气）。位面之间的空间转换（主物质位面与下层位面）则是一种更彻底的阶级跨越仪式。

### 从基因到力量美学

因为基因①将力量定义为一套精确计算、可量化的系统（DD3.5e的属性阈值、法术位、专长前置条件），所以战斗的视觉外观是**DND式精确计算的力量**——每一次攻击的命中判定、每一点伤害值、每一次豁免对抗，都在数据面板上有对应的数字变化。战斗不是连续的、流线型的武打表演，而是回合制战术推理的可视化呈现：走位、借机攻击、法术反制、状态叠加。主角在战斗中花在"看面板"和"算概率"上的时间，与花在实际动作上的时间一样多。

因为基因①的世界底层逻辑是"经验从杀戮中来"（宪法②），所以**杀怪得经验的底层逻辑**不仅是数值机制，更是世界观层面的美学表达。击杀不是解放敌人或为民除害，而是"摄取灵魂能量"这一冷血事实的物理表现。怪物死亡时不会有圣光超度或圆寂坐化，而是低贱地倒地，生命值归零，经验条增长。这种力量来源的美学是功利主义的——没有副作用、没有道德光环、没有仪式感的庄严。

因为基因②要求主角从底层计算每一次力量提升的性价比，所以**越级必须设计+代价**不仅是叙事策略，也是美学律令。每一次越级战斗在视觉上都呈现为"以弱胜强的精密差"——主角需要通过地形利用、信息差、环境陷阱、消耗战等多阶段布局将一个不可能战胜的敌人拖入自己的游戏节奏。画面呈现为不对称对抗：笨重的巨力与灵巧的闪避、大范围的AOE与精准的单点突破、蛮横的抗性与恶毒的DEBUFF叠加。获胜后的主角不会无伤收剑，而是必带可感知的代价（断肢、法术位耗尽、关键装备损坏、临时属性损伤）。

### 从基因到情感光谱

因为基因②要求"底层冷酷"、拒绝英雄主义驱动，所以本书**禁用热血欢呼**。主角不会在战斗中暴气吼招，不会在胜利后振臂高呼，不会在关键时刻因"友情/信念/羁绊"爆种翻盘。一切情感的外显都被抑制到最低限度——激烈的情感表达在这个世界中是致命的脆弱。

因为力量提升只能通过杀戮（基因①的底层规则），而杀戮又是冷酷的功利行为，所以情感节奏不可呈现为正弦波式的"紧张—释放"循环。本书的情感节奏是：**压抑 → 冷酷计算 → 短暂释放 → 再压抑**。压抑阶段占比最长（主角在日常中积累无力感），冷酷计算阶段是叙事主体（主角用智力分析替代情绪反应），短暂释放只出现在两个节点——成功保护锚点之后，或完成一次精密越级之后（表现为冷笑、松一口气、沉默的凝视，而非狂喜）。随即进入下一个更深的压抑循环。

因为基因①要求力量体系的严肃性、基因②要求底层视角的冷冽感，所以本书的**情感表达是克制的、间接的、读者侧推理式的**。主角不会说"我很害怕""我很愤怒"，而是通过手部细节、呼吸节奏、战斗中的决策变化来暗示情绪状态。情感浓度不靠台词或独白渲染，而靠环境氛围和叙事留白来让读者自行填充。这种情感质地与同类黑暗升级流的区别在于：它不是"装作冷酷实则有温度"的伪克制（内心OS不停），而是真正的情感熵减——主角懒得对自己解释自己的情绪，因为情绪不提供生存优势。

### 命名体系

从以上所有推导出发，命名语汇方向如下：

- **世界命名语汇**：全部沿用 DND 3.5e 标准西幻语汇。职业名、法术名、属性名、种族名、位面名、神系名均以 SRD 和《被遗忘的国度》为语料库，不做本土化改编。地名采用"形容词+自然元素"结构（如某堡、某镇、某之门、某之地），拒绝东方修仙名相（如"XX山脉"在 DND 语境中应为"XX山"或"XX岭"，"XX洞府"不存在等价物所以直接禁用）。
- **力量命名语汇**：全部沿用 DND 标准术语——职业等级（而非"境界/阶位"）、法术环位（而非"法术等级/层数"）、属性（力量/敏捷/体质/智力/感知/魅力）、专长（Feat，而非"天赋/被动技能"）、阵营（九阵营全称，而非"正道/魔道"）。不用自创名词替换任何 DND 已有术语。
- **神灵命名语汇**：采用西方奇幻多神体系的神名风格——神名由"神职名词+特定后缀"构成，或使用近似人名的简短单词（2-3音节）。拒绝一神教神名风格（如"XX之神"的"-神"后缀滥用），拒绝东方神系语汇（帝、君、仙、圣、王）。
- **角色命名语汇**：采用西幻常见的简短多音节名（2-3音节/ 7-12字母），配合姓氏/称号缀词。拒绝东方风格的姓氏与单字名，拒绝日语风格的多音节轻小说式名，拒绝刻意古雅的希腊/拉丁/希伯来语词根挪用。
- **种族与怪物命名**：完全使用 DND 标准怪物名称（哥布林、豺狼人、食人魔、眼魔、灵吸怪等），不用东方妖异名相替代，不做本土化语感改造。
- **坚决拒绝的命名语汇**：东方修仙等级体系（炼气/筑基/金丹/元婴/化神）、修真功法与丹药名称、东方风格人名与姓氏、日语或轻小说语汇的等级命名（如"下位/中位/上位"代替职业等级描述）、自创玄学名词（如"源气"、"\*\*元"、"\*\*力"等指向模糊的词汇）。

### 视觉锚点

尊严城下水道第三层涵洞。半精灵少年蹲在齐膝深的污水里，左手捂着小腹上的刀口，右手食指悬停在一块只有他能看见的光屏上。光屏边缘在渗血——他的血。屏幕跳动着一行字：「专长选择——你已满足「战斗施法者」前置条件（专注5级、施法者等级3）。是否确认？」涵洞深处传来沉重的脚步声，追猎者的等级标签在他视野边缘闪烁——猩红色的「Lv.9」。他舔了舔嘴唇上的血，默念确认。污水表面突然结出一层薄冰，他的法术位——本来只剩一发——在专长生效的瞬间跳回了三发。他把最后一发灼热射线压在指尖没有释放，转身沉入更深的黑暗里。脚步声近了。他等的就是脚步声。

## 四、用户画像

### 核心读者
- **年龄**：18-35 岁男性为主
- **阅读背景**：了解或热爱 DND/跑团文化、硬核数值体系、CRPG（博德之门、神界原罪）、西幻黑暗升级流
- **核心欲求**：
  - 硬核但可理解的数值体系——看到属性变化能算出意义
  - 智力型主角的爽快越级——"我比你弱但我比你聪明"
  - 合理而不降智的世界运转——高阶不会莫名其妙出现在低阶地盘
  - 黑暗但不压抑——冷酷而非绝望
- **代入路径**：智力 > 武力。读者通过"如果我也会这么算"获得代入，而非"如果我也这么强"

### 不服务谁
- 追求后宫/无脑爽/日常休闲的读者
- 不喜欢数据/数值/面板展示的读者
- 无法接受主角做"不道德但不违法自己规则"之事的读者

## 五、世界宪法

### 宪法①：神力等于意志力
神性本质上是灵魂能量与意志力的结合。融合神性需要与原主人残留意志对抗，意志判定失败则性格被扭曲。只有最坚定的意志才能承载神性而不被反噬。
- **违反后果**：无代价获得神力 → 神性持有者量产 → 神性不再是稀缺资源 → 封神线价值崩塌

### 宪法②：杀戮是唯一的经验来源
这个世界的基础规则之一：击杀具备灵魂能量的生物可获得经验，用于职业升级。不存在"打坐修炼""闭关突破""灵气吸收"等非战斗成长路径。
- **违反后果**：引入非战斗成长路径 → 主角可以选择安全升级 → 战斗驱动力消失 → 叙事节奏崩塌

### 宪法③：超凡属性是质变分水岭
某项属性突破特定阈值（对应 DND 3.5e 的超凡门槛）后，该角色获得特殊能力。属性差距在特定数值范围内产生压制性效果。这一规则约束了越级战斗的可能性边界。
- **违反后果**：属性变成纯数字堆叠 → 阈值失去意义 → 读者不再关心属性增减 → 数值体系的叙事生命力消失

### 宪法④：阵营由行为决定
九阵营（守序善良→混乱邪恶）不是自我宣称的标签，而是角色行为对世界造成的实际影响的客观映射。善/恶/守序/混乱是可通过法术侦测的客观灵光存在。
- **违反后果**：主角可以做坏事但自称善良 → 阵营系统沦为摆设 → 守序邪恶的约束力消失 → 角色行为失去张力

### 宪法⑤：神灵可陨落与回归
这个世界的神灵不是全知全能的抽象存在。神灵可以被杀死（神格被夺取、神职被取代、信徒被掠夺），也可以通过特定条件复活。诸神之间的博弈是世界的顶层驱动力。
- **违反后果**：神灵不可挑战 → 弑神线失去存在前提 → 全书核心目标的合法性崩塌

### 宪法⑥：位面之间存在通道
主物质位面与下层位面、上层位面、元素位面之间存在稳定的自然通道或可开启的临时通道。这些通道是神灵、恶魔、元素生物等超自然存在干涉凡间的物理路径。
- **违反后果**：位面隔绝 → 超自然力量无法干涉凡间 → 神战/恶魔入侵等大事件失去发生前提 → 被动卷入机制无法触发

## 六、跨域素材指引

1. **DND 3.5e SRD**：去这套开源的规则参考书中提取力量体系的底层语法——六属性机制、法术环位、专长与技能、怪物挑战等级等。这是本书的基础架构层，建议直接将其作为"世界运转说明书"使用。
2. **被遗忘的国度设定集**：去这套 DND 官方战役设定中提取多神系博弈、位面架构、魔网系统等世界观要素。重点参考神灵之间的关系网络和动荡之年等世界级事件的结构设计。
3. **博德之门系列叙事结构**：去这套 CRPG 中学习"神子战争"类多势力博弈的叙事组织方式——多方势力同时布局、信息差驱动的勾心斗角、不断反转的盟友关系。注意仅取结构不取具体设定。
4. **克苏鲁神话氛围模式**：去克苏鲁体系的叙事中提取"不可名状的恐惧"和"凡人面对深渊的渺小感"的氛围营造方式。仅取氛围不取具体怪物设定。
5. **冰与火之歌的低魔写实**：去乔治·马丁的作品中学习"高阶不常干预日常"的叙事策略、多视角信息差、以及"任何人随时可能死亡"的真实感。这是防崩体系的叙事范本。

## 七、篇幅规划

- **目标字数**：长篇序列，预估 300-500 万字
- **预估卷数**：多卷结构，预计 8-12 卷
- **单卷范围**：每卷对应一张主要地图的完整叙事弧线，约 25-40 万字
- **爽点密度**：每卷至少 3-5 次核心爽点释放（数据面板优越感 + 阶级碾压 + 情感锚点危机爆发），日常章节以信息差积累和主角布局为主
- **换图节奏**：每卷至少完成一次完整的地图叙事弧线（进入→适应→掌控→被卷入下一张地图）

## 八、核心承诺 + 防崩阈值

### 核心承诺

**承诺①（对应爽点①）：主角永远比别人更懂力量体系的本质**
- 防崩信号 A：主角开始依赖蛮力而非规则认知取胜
- 防崩信号 B：出现其他角色同样了解力量体系底层的设定（稀释信息差优势）

**承诺②（对应爽点②）：每张地图主角都会从底层爬到接近天花板**
- 防崩信号 A：主角在新地图中直接从较高社会地位起步
- 防崩信号 B：连续两张地图不进行社会地位重置

**承诺③（对应爽点③）：情感锚点角色始终是主角行为的核心驱动力**
- 防崩信号 A：超过一卷的篇幅情感锚点角色完全不出场也不被提及
- 防崩信号 B：主角做出重大决策时完全不考虑锚点角色的安危

### 额外承诺

**承诺④：每一次升级都有清晰的叙事对应事件**
- 防崩信号 A：主角突然变强但没有对应的战斗/事件/代价
- 防崩信号 B：属性提升变得例行公事化，失去仪式感

**承诺⑤：越级战斗不是常态而是一卷1-2次的高光时刻**
- 防崩信号 A：越级战斗的频次超过叙事可以赋予仪式感的上限
- 防崩信号 B：越级战斗不再需要付出可感知的代价

**承诺⑥：每卷末主角不会以"最强状态"进入下一卷**
- 防崩信号 A：卷末不安排任何形式的卸能/代价/惩罚
- 防崩信号 B：下一卷起点战力等于或高于上一卷终点战力

## 九、不可妥协 DNA + 边界

### DNA（打死不能改）

1. **DND 3.5e 底层体系**：六属性、职业等级、法术环位、九阵营的基础框架不可替换为其他体系。这是本书"硬核西幻"身份的底线。
2. **杀戮得经验**：力量成长的唯一路径不可改为"修炼/闭关/冥想"等非战斗方式。这条 DNA 决定了主角的处世逻辑和叙事节奏。
3. **局外人主角**：主角不是任何人或神的选民/神子/天命之人。这是本书结构层面的最大差异化。
4. **守序邪恶阵营**：主角的阵营锚定在守序邪恶，不可转向善良或中立。阵营漂移只允许在邪恶轴内部移动（守序邪恶→中立邪恶→混乱邪恶）。
5. **情感锚点单一且唯一**：情感锚点角色只能有一个。多个情感锚点会稀释驱动力浓度、模糊行为动机、拉低冷酷人设的可信度。
6. **传奇不对决主角**：传奇级角色绝不直接与主角交手。传奇的战斗是传奇之间的博弈，主角在旁观察/被救/逃离。这条 DNA 是力量体系可信度的最后防线。
7. **每卷末必须卸能**：每卷结尾必须通过等级损失/重创/封印/丧失关键物品等方式为主角付出可感知的代价，保证下卷起点处于可控低位。

### 硬边界

1. **不后宫**：情感线严格控制在锚点角色一人。任何与之平行的情感线都会稀释"唯一性"。
2. **不降智反派**：高阶角色不会因为主角需要赢就出现在低阶场景里。反派的每一个行为都基于自身动机而非"为主角送经验"。
3. **不引入东方修仙体系词汇**：全书拒绝任何形式的东方修仙名相——功法、境界、炼丹、飞升等语汇不可出现。
4. **不主动换地图**：主角永远不是"想去新地图"的那个推动者。地图转换必须是外部事件的强加结果。
5. **不写无聊的过渡日常**：日常段落必须有叙事价值——要么垒信息差，要么铺垫情感锚点危机，要么展示主角布局。无意义的日常对话和日常副本不可出现。
