## 一、核心冲突模式

### 表层冲突：挂鹅卖鸭（信任欺诈）

```
卖者声称：这是鹅腿（品质承诺）
消费者相信：因为名校背书（权威认证）
实际真相：是鸭腿（成本替代）
暴露方式：群友举报（内部告发）
```

**叙事母题归类**：身份欺诈 / 李代桃僵 / 以次充好
**经典变体**：圣物造假（宗教）、血脉伪造（奇幻）、配方替代（美食）、机芯替换（谍战）

### 中层冲突：优绩主义滤镜下的集体幻觉

```
名校生追捧 → 大众跟风 → 媒体造神 → 光环外溢
     ↓            ↓           ↓           ↓
"清北认可=品质保证" → "16元的鹅腿？不可能有问题"
     ↓
真相揭露时，被骗的不只是金钱，更是"我引以为傲的判断力"
```

**叙事母题归类**：权威崇拜崩塌 / 光环效应反噬 / 集体认知失调
**经典变体**：大师陨落（武侠）、圣子堕落（奇幻）、神谕失效（科幻）

### 深层冲突：信任经济的脆弱性

```
微商/团购模式：无正式契约，纯靠信任链条运转
     ↓
信任链条中的每一个节点都是单点故障
     ↓
一个"自己人"的举报 → 整个生态崩塌
     ↓
问题是：到底是谁背叛了谁？
   - 阿姨背叛了信任（挂鹅卖鸭）
   - 还是群友背叛了"自己人"文化（内部举报）
```

**叙事母题归类**：信任经济的内爆 / 社群契约破裂 / 背叛的镜像
**经典变体**：内鬼告密（谍战）、同门相残（武侠）、背叛者审判（悬疑）

## 二、角色原型识别

### 核心角色矩阵

| 角色 | 现实原型 | 原型标签 | 叙事功能 | 可替换方向 |
|:-----|:---------|:---------|:---------|:-----------|
| **鹅腿阿姨** | 陈秀凤，54岁，底层劳动者 | **"老实人的面具"** | 承受者/伪装者/double agent | 老工匠/老药师/隐世高手/资深夜摊主 |
| **举报者** | "上班精英"群友 | **"清醒的局内人"** | 催化剂/真相揭露者 | 同行/前徒弟/被驱逐的门徒/内部审计者 |
| **清北学子** | 三校学生群体 | **"信徒群像"** | 集体认知的载体/受害者/共谋者 | 学院派/贵族子弟/精英组织/宗教团体 |
| **媒体/网红** | 蜂拥而来的主播 | **"狂欢的寄生虫"** | 放大镜/混乱制造者 | 江湖说书人/情报贩子/街头宣传队 |
| **鹅腿黄牛** | 加价倒卖者 | **"套利的掠食者"** | 系统漏洞的具象化 | 黑市商贩/情报掮客/中间商 |

### 核心张力：鹅腿阿姨的双重身份

陈秀凤有两种相互矛盾的解读方式——这是最值钱的叙事空间：

**解读A：她是受害者**
- 从底层做起（卖水果10年 → 烧烤摊 → 网红）
- 被流量裹挟，被捧上神坛
- 进不到鹅腿（市场客观原因）→ 改用鸭腿
- 保留"鹅腿"名字（品牌惯性）→ 被举报
- 她的眼泪是真的，她的辛苦是真的

**解读B：她是精明的伪装者**
- 早在2011年（走红前12年）就已经改用鸭腿
- 但名字一直叫"鹅腿阿姨"
- 坚持16元的定价既"合理"又"不像真鹅腿"
- 在北大演讲大谈"良心活"
- 被曝光后回应："鹅腿阿姨叫了十几年了，不存在欺诈"

**叙事最值钱的地方：两种解读都对。** 一个人可以同时是卖命干活的劳动者和长期挂羊头卖狗肉的商贩。这不是非黑即白的角色，而是**灰度角色**。

### 时间跳跃的叙事价值

第二幕→第三幕之间有两年的**时间跳跃**。这是叙事上非常值钱的结构：
- 发生在"沉默的两年"里的事，给叙事留下了巨大的填充空间
- 是阿姨逐渐麻痹？还是供应链早就断了？还是她想过坦白但没机会？
- 这两年的空白 = 叙事者的游乐场

## 四、社会寓言层

### 寓言一：优绩主义的双刃剑

清北滤镜能够把16元烤鸭腿捧成"神级鹅腿"，说明**名校认同具有货币般的价值转换能力**。一个人/物一旦被贴上名校标签，自动获得品质溢价。
- 在奇幻世界观中：可以映射为"神选者认证"——某某神殿认证过的物品自动值钱，没人敢质疑
- 在都市世界观中：可以直接复制——清华周边美食图鉴的造神机制
- 在科幻世界观中：行为信用分——被精英节点认证过的商品获得信任溢价

### 寓言二：信任经济的三体问题

```
卖者信用（自称鹅腿）
        ↕
消费者信任（因为清北光环）
        ↕
社群监督（群友举报机制）
```

三个节点中任何一个断裂 → 整个系统崩盘。这个结构可以平移进任何"无正式契约、纯靠声誉运转"的系统：
- 修真界的丹药交易（没有质检局）
- 星际黑市（没有星际法）
- 地下情报网（没有仲裁者）

### 寓言三：真相与名字的博弈

"鹅腿阿姨"这个名字本身已经成为**独立于事实的品牌资产**。即使产品从鹅腿换成了鸭腿，名字也不需要改——因为名字的意义不再是"描述产品"，而是"情感符号"。
- 这在小说的"封号""称号""绰号"设定中可以直接使用
- "屠龙者"其实已经十年没杀过龙
- "天下第一剑客"用的是刀
- "光明圣殿"早已不信光明神

### 寓言四：内部人举报 vs 外部人监督

举报者是"群里某位上班精英"——不是竞争对手，不是消费者协会，而是**自己人**。这触发了社群中最敏感的神经：自己人告自己人。
- 帮派内部：二当家告发帮主
- 组织内部：审计员揭露贪腐
- 家庭内部：子女举报父母
- 最痛的不是外部攻击，是内部爆雷

## 五、可复用的场景模板

### 场景模板A：走红之夜
**核心张力**：一个小人物突然被巨大流量/关注淹没的那一刻
**元素清单**：
- 主角并不知道自己为什么火
- 手机突然涌入几千条消息
- 熟悉的生活节奏被打乱
- 各方势力像秃鹫一样扑来（经纪公司/媒体/黄牛）
- 主角的反应：不是狂喜，是恐惧

### 场景模板B：演讲台上的另一面
**核心张力**：当一个小人物被请上"神坛"讲述成功经验时，台下的人不知道台上的人正在隐藏什么
**元素清单**：
- 演讲主题：诚信/良心/持之以恒
- 主角内心：我知道我在说什么假话吗？
- 听众的反应：虔诚地记录→事后翻出来对照→"她说过这些话"→加倍背叛感
- 戏剧性：她演讲时说的"良心活"恰好是她隐瞒的事情

### 场景模板C：群公告的审判
**核心张力**：在熟悉的社群里，发出一条改变一切的消息
**元素清单**：
- 场景：深夜，手机屏幕的光
- 打字时的手：犹豫/颤抖/决定
- 发出后的寂静（几秒 → 几分钟 → 爆炸）
- "原材料是鸭腿"——五个字瓦解两年的信任
- 回应：她说这不是欺诈——她真的觉得自己没错

### 场景模板D：真相的辩护
**核心张力**：一个被揭穿的人如何为自己辩解——她说的是真话还是谎言？
**元素清单**：
- "鹅腿阿姨叫了十几年了"（品牌历史 = 盾牌）
- "不存在欺诈等行为"（我定义了什么叫欺诈）
- "以后都会给大家写清楚"（我承诺以后诚实）
- "介意请勿下单"（选择权在你们）
- 这不是认错——这是重新划定规则

## 六、各类型适配笔记

| 类型 | 改造方向 | 保留什么 | 替换什么 |
|:-----|:---------|:---------|:---------|
| **都市现实** | 几乎原样搬运 | 全部 | 少量艺术加工 |
| **都市奇幻** | 鹅腿→灵药，阿姨→隐世炼丹师 | 信任崩塌结构 | 产品替换为奇幻道具 |
| **武侠/江湖** | 门派认证的灵药造假 | 清北滤镜→门派光环 | 微信团购→江湖口碑 |
| **科幻** | 信用评分系统下的"挂真肉卖合成肉" | 信任经济内爆 | 高校→基因优化精英学院 |
| **悬疑** | 举报者究竟是谁？这是偶然还是计划？ | 全事件框架 | 加入悬疑线：也许阿姨从第一天就是布局 |
| **美食/治愈** | 阿姨是好人，被迫用鸭腿，最终和解 | 温馨设定 | 降低冲突烈度，转向和解 |
