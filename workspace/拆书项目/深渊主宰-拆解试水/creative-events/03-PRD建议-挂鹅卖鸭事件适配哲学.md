# PRD 建议：挂鹅卖鸭事件 · 跨类型适配加工哲学

> 产出者：pop-writer-creative · 分支B变体
> 说明：轻量级 PRD——不是锁定了一本书，而是为这个素材在不同类型小说中的加工方式提供坐标系。
>
> 使用方式：如果将来某本书要拉取此储备池的素材，先读此 PRD，确定要用哪个适配方向。

---

## 一、素材一句话定位

> 一个靠精英光环运转的信任经济系统，因内部人揭发"挂X卖Y"的真相，引发从狂热追捧到系统崩塌的连锁反应，以及涉事者"两种解读都对"的灰度辩护。

---

## 二、核心承诺（无论什么类型，这个素材必须提供什么）

| 承诺 | 说明 | 不可妥协 |
|:-----|:-----|:---------|
| **真相揭露的时刻感** | "原材料是鸭腿"——五个字摧毁一切的那个瞬间 | ❌ 必须保留 |
| **信任崩塌的连锁反应** | 不是一个人受伤，是整个系统内爆 | ❌ 必须保留 |
| **灰度角色的辩护** | 读者直到最后一刻都无法确定"她是受害者还是骗子" | ❌ 必须保留 |
| **集体幻觉的形成与破碎** | 为什么所有人都相信了不该相信的东西 | ❌ 必须保留 |

---

## 三、加工哲学

### 加工原则

| 原则 | 内容 | 红线 |
|:-----|:-----|:-----|
| **结构先于情节** | 不直接搬运"鸭腿阿姨"的故事，而提取信任经济崩塌的结构 | ❌ 禁止写成新闻纪实文学 |
| **灰度高于审判** | 不写"黑心商贩被惩罚"的道德爽文，保留角色的可争议性 | ❌ 禁止简单善恶二分 |
| **系统大于个人** | 除了核心角色，必须呈现让这件事成为可能的系统/环境/群体 | ❌ 禁止只聚焦单一人物的独角戏 |
| **因果不简化** | 信任构建需要两年，崩塌只需要一天——不对这个节奏做均匀化处理 | ❌ 禁止压缩时间线让崩塌"慢慢发生" |

### 不建议的方向

| ❌ 不建议 | 原因 |
|:----------|:-----|
| 复仇爽文——被坑的消费者联手报复 | 抹杀了灰度，变成了简单的好人打坏人 |
| 法律破案剧——检察官/律师调查真相 | 太依赖制度解决了，失去了信任经济无契约的原始张力 |
| 纯爱情线——两个大学生因为鹅腿相识相爱 | 把社会性事件降格为背景板，浪费了核心冲突 |

## 四、跨类型适配矩阵

### 🏙️ 都市现实（最适合·改动最小）

| 维度 | 设定 |
|:-----|:-----|
| **锚点** | 高校周边摊贩生态、微信群接龙团购、初代网红生存记 |
| **保留** | 几乎所有元素 |
| **改写** | 地点/人名的虚构化处理 |
| **加工重点** | 聚焦"M1信任经济崩塌"的细节刻画——社交媒体截图、群聊记录、"沉默的两年"中发生了什么 |
| **经典对标** | 《漫长的季节》式"小人物命运的微小转折" |
| **避坑** | 不要写成"讽刺社会现象"的论文式写作——故事第一，隐喻是副产品 |

### 🧙 都市奇幻（改动中等·最有想象力空间）

| 维度 | 设定 |
|:-----|:-----|
| **锚点** | 鹅腿=灵药，阿姨=隐世炼丹师，清北=修真宗门，群友=灵修圈子 |
| **保留** | 信任经济结构、内部举报机制、灰度辩护 |
| **替换** | 产品（灵药/符箓/法器）、认证方式（宗门认证）、交易场所（后山集市） |
| **加工重点** | 幻觉是如何形成的：为什么一个炼丹师说"这是筑基丹"，全宗门弟子都信了？因为她在宗门后山卖了十年——从没人想过怀疑 |
| **对标参考** | 《道诡异仙》式"日常中的诡异感" |
| **新加的维度** | 灵药的药性和品级差距（对应鹅腿vs鸭腿的实际差异——普通人吃不出来，但资深丹修一尝就知道） |

### ⚔️ 武侠/江湖（改动中等·江湖经济模型）

| 维度 | 设定 |
|:-----|:-----|
| **锚点** | 门派认证的"正宗"与"伪造"、江湖口碑体系、丐帮/商会的信息网络 |
| **保留** | 权威认证崇拜、信任链条、内部举报 |
| **替换** | 高校→武林圣地（少室山/武当）、团购→江湖口碑、阿姨→老铁匠/老药师 |
| **加工重点** | 江湖版的"优绩主义"：为什么所有人都相信"武当认证的兵器就是好"？因为武当是名门正派——但从没有人真正检验过 |
| **对标参考** | 金庸式的"招牌背后的江湖" |
| **新加的维度** | 举报者可能是其他门派的卧底——"内部举报"的动机从正义感变成派系斗争 |

### 🌌 科幻（改动大·信用分社会）

| 维度 | 设定 |
|:-----|:-----|
| **锚点** | 信用评分社会、精英节点认证、基因优化阶层 |
| **保留** | 信任经济内爆、灰度辩护、集体幻觉 |
| **替换** | 微信群→神经链接群聊、鹅腿→合成蛋白肉vs天然肉、北大→基因优化精英学院 |
| **加工重点** | 未来版的优绩主义：基因优化者的消费选择自带"品质认证光环"——一个未优化的自然人不可能骗过优化者。除非…… |
| **对标参考** | 《黑镜》S3E1「急转直下」+ S1E2「一千五百万的价值」 |
| **新加的维度** | 技术监管（区块链溯源？还是可篡改的数字证书？——如果是后者，信任系统更脆弱） |

### 🕵️ 悬疑/犯罪（改动小·视角重构）

| 维度 | 设定 |
|:-----|:-----|
| **锚点** | 真相调查、多视角叙事、谁的叙述可信 |
| **保留** | 全部事件元素，但用悬疑结构重新排列 |
| **替换** | 不替换世界观——换叙事方式 |
| **加工重点** | 设立一个调查者角色（记者/警察/学生/前群友），从"看到群公告"开始回溯调查：举报者是谁？什么时候开始的？还有谁知道？阿姨从第一天就在骗吗？ |
| **对标参考** | 《看不见的客人》式"多版本真相" |
| **新加的维度** | 每次回溯揭示一个新版本——A说是经营困难被迫改的，B说从2011年就是鸭腿，C说举报者其实是另一个摊主…… |

## 五、素材消费建议（哪个plot阶段拉取什么）

| plot 阶段 | 拉取素材 | 加工方式 |
|:----------|:---------|:---------|
| Step 1 全书主线选择 | M1（信任经济崩塌）→ 作为全书第三幕的核心冲突 | 先构建信任（前三卷读者的熟悉感）→ 崩塌（第四卷引爆） |
| Step 2 第一卷设计 | B1（微商生态）+ B2（大学城文化）+ S1（流量反噬）→ 作为第一幕世界观底色 | 营造"这个信任系统看起来牢固"的错觉 |
| Step 3 主角设计 | M3（双层真相辩护）→ 角色灰度参考 | 主角不是纯粹的善/恶——是在某种条件下，诚实的人也会选择隐瞒 |
| Step 4 第三~四卷冲突 | S4（举报者视角）+ R3（黑吃黑反转）→ 中后期冲突升级 | 让读者怀疑"举报者真的是正义的吗" |
| Step 9 某幕缺冲突 | R1（从头布局的骗局）+ R4（信用分社会版）→ 剩余焊接点救场 | 如果故事中期疲软，这两条可以作为"真正的幕后黑手"反转 |

## 六、边界

### 这个素材不做的事
- ❌ 不做"互联网讽刺小说"的直接搬运——即使写现实题材，也要有强故事线
- ❌ 不做道德教科书——不在结局给出"所以人要诚信"的结论
- ❌ 不做新闻评论——不引用真实采访、不写"笔者按"
- ❌ 不做单一类型锁定——保留跨类型使用的可移植性

### 素材储备池消费时的红线
- ❌ 使用M1时如果选了"都市奇幻"方向，不得同时使用M3的"现实向"辩护口吻（世界观调性冲突）
- ❌ 使用R1（阿姨是布局者）时，不得同时使用M3（灰度辩护）——选了A就不能选B
