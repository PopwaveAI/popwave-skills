## 参考类型定义

| 类型 | 定义 | 判定标准 |
|------|------|---------|
| **直接引用** | 从源头作品中完整沿用的机制/命名/概念 | 术语一致、机制相同、无重大改动 |
| **结构借鉴** | 借用底层框架但予以改编或组合 | 框架相似但实现方式/世界观嵌入有差异 |
| **文化原型** | 源自神话/历史/通用文学母题，非特定作品独创 | 属于公共文化资源池 |
| **存疑** | 相似度高但证据不足，或可能为独立创作中的巧合 | 缺乏直接证据链 |

## 核心原则

- **化用≠照抄**：本索引仅记录参考来源及转化方式，不使用"抄袭""照搬""山寨"等判断性用语
- **每一条目均为具体证据链**：标注来源章号（chXXX），可在对应设计包中交叉验证
- **开放修正**：第二卷167章数据仅为阶段性分析，后续章节补充后可更新
