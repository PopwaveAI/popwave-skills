# 设计包 — ch001「混乱之地」
> 拆书提取 | 章型: 场景引入+氛围渲染 | 事件数: 12

## 基础信息
- **卷**: 第二卷「海外群岛」
- **本卷定位**: 索伦携薇薇安离开大陆，抵达海外群岛开启新地图
- **本章功能**: 引入泰罗港场景，铺垫海外群岛的混乱生态，为海洋神殿任务做引子
- **关键冲突**: 环境冲突（混乱之地vs索伦的适应）+ 潜在人物冲突（亚马逊女人）
- **章末钩子**: 索伦决定去海洋神殿寻求解除吸能术

## 事件链

### 事件1：到达泰罗港
- **类型**: 场景建立
- **原文evidence**: "腥涩的海风迎面吹来。""索伦抱着娇小的薇薇安从甲板上走过"
- **内容**: 索伦抱薇薇安下船，引人注目。两个面色不善者被强壮大汉制止——"一个男人敢这样带一个小姑娘来这里，不是你们这群家伙可以对付的！"

### 事件2：背景信息交代（财富之城消息）
- **类型**: 情报/世界信息
- **原文evidence**: "据说那边发生了很严重的大事，财富之城都被毁掉了三分之一，最起码有数千人死亡"
- **内容**: 吸血鬼神子封神成为伪神，吸引所有目光——教会、恐惧信徒、其他恐惧神子。索伦判断：吸血鬼神子倒下前，他和薇薇安都是安全的。

### 事件3：海外群岛商业逻辑说明
- **类型**: 设定说明
- **原文evidence**: "哪怕是拉一大船的粮食过去，都能在土著人那里卖出来三倍以上的高价"
- **内容**: 金银兑换10:1 vs 6:1的利润差；金矿开采不足十分之一；贸易发达→海盗群体庞大。

### 事件4：目睹小偷受刑
- **类型**: 氛围渲染
- **原文evidence**: "看到被砍下手指的年轻人，以及地面上血淋淋的三根手指头"
- **内容**: 小偷被砍三指，围观者无人同情。规则：偷窃被抓动用私刑卫兵不管，但弄死小偷要罚款。

### 事件5：街道环境描写
- **类型**: 氛围渲染
- **原文evidence**: "大白天都有流莺站街拉客，有时候直接往胡同里面一钻就开始做皮肉生意"
- **内容**: 肮脏地面、鱼腥味、水手客串海盗，整体比大陆混乱很多。

### 事件6：酒馆斗殴
- **类型**: 氛围渲染
- **原文evidence**: "酒馆老板很淡定的在吧台前擦桌子"
- **内容**: 亚马逊女战士获胜，过肩摔扔出男人。规则：打坏东西由输家赔偿。

### 事件7：输家被二次洗劫
- **类型**: 氛围渲染
- **原文evidence**: "被打晕的家伙很快遭到了第二波的洗劫"
- **内容**: 鬼鬼祟祟的小伙子扒光输家只剩裤衩，扔进小巷。

### 事件8：索伦的对比心理
- **类型**: 角色心理
- **原文evidence**: "眼前的混乱在他看来根本就不算什么"
- **内容**: 对比幽暗地域，这里至少克制不出人命；未被通缉/举报的罪犯就是合法平民。

### 事件9：海洋神殿信息
- **类型**: 设定/伏笔
- **原文evidence**: "泰罗港有一座海洋神殿"
- **内容**: 海洋女神与风暴之主关系暧昧又竞争；都是混乱邪恶阵营。

### 事件10：入住亚马逊旅馆
- **类型**: 行动
- **原文evidence**: "一晚上需要付出十五枚金德勒"
- **内容**: 旅馆昂贵但安全，门口有女战士站岗，地下室有高阶职业者，有法术陷阱。索伦安排薇薇安住下。

### 事件11：决定去海洋神殿
- **类型**: 决策
- **原文evidence**: "索伦准备去海洋神殿碰碰运气"
- **内容**: 决定去寻求解除吸能术，即便多付钱也愿意。

### 事件12：遭遇亚马逊行刑者
- **类型**: 人物互动+钩子
- **原文evidence**: "对方身上沾了不少血，还有一点点的肉沫"
- **内容**: 行刑者刚从地下室出来（处理了不守规矩者），调戏索伦——"需要一些愉悦的服务吗？""她们会把你扔下床的！"
- **章末钩子**: 亚马逊女人的彪悍留下印象，为后续互动铺垫。

## 设定提取
- **泰罗港**: 直径30公里左右的岛屿，索罗斯群岛中转站，仅次于财富之城的销赃港口
- **泰罗港规则**: 偷窃不被发现不是犯罪，被发现可动用私刑（砍手指/剁手），但弄死小偷要罚款
- **海外群岛商业**: 粮食三倍价、金银兑换差、矿产开发不足十分之一
- **海盗群体**: 数量庞大，三大海盗王中亚马逊女海盗占一个半
- **神灵背景**: 海洋女神（混乱邪恶）+风暴之主（混乱邪恶），关系暧昧又竞争，数百年前大打出手
- **亚马逊旅馆**: 15金德勒/晚，有高阶职业者镇守，无人敢闹事
