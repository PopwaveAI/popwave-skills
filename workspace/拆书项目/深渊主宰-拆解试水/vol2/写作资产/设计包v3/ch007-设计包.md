## 二、核心事件 Evidence

### E1 【杀戮之心】背景专长获取
- **原文** §6-"杀戮之心【背景专长】：你已经在一次次的杀戮中变得习惯死亡与鲜血……【永久性意志+1】"
- **前置条件**: 终结上百条生命（≈近百人斩累积）
- **进阶路径**: 【杀戮之心】→【屠戮者】（500人头）→【死亡行者】（1000人头，传奇级）
- **独立性**: 三个专长可叠加，不互相取代
- **对应善良阵营**: 【光辉使者】、【自然平衡者】等
- **已获效果**: 永久意志+1，免疫友好术/误导术类影响

### E2 海洋女神的死亡领域权能
- **原文** §22-"出现在索伦面前的是三个扭曲的怪物……死去但因为某些原因转化成了海洋中的亡灵"
- **三仆从展示**:
  1. 鲨鱼人亡灵（腐烂+贝壳海藻）
  2. 娜迦海妖亡灵（单臂变异）
  3. 海洋女妖（前身幽灵，阴冷邪异）
- **原文 §49**: "由传奇牧师出手捕获这些幽灵船，然后在海洋女神的神力下秘密将这些幽灵船活化成低级亡灵生命形态"
- **政治暗示**: 海洋女神染指死亡领域，超出常规海洋神灵职权范围

### E3 幽灵舰队 / 亡语者号属性面板
- **原文** §47-53 — 完整数据：

| 项目 | 数据 |
|------|------|
| 名称 | 亡语者号【活化幽灵船】 |
| 船长 | 索伦【临时】 |
| 水手 | 不溺者亡灵49、海洋怨灵12、海洋幽灵72、娜迦女妖9、鲨鱼人亡灵6、亡语者缚灵872【锁定】 |
| 装备 | 沉没哀嚎（撞角）、亡语者大炮×24、幽灵风帆、回旋重炮×2 |
| 技能 | 召唤迷雾、幽灵船特性、水下航行、缚灵特性、活化生命、活化缆绳、诅咒船身、亡语者特性 |
| 核心特征 | "这艘船是有生命的！它是活的！！！" 相当于"活化城堡/活化塔灵"的舰船版 |

### E4 号角——控制信物
- **原文** §43-44: 娜迦祭司将海螺号角交予索伦，"从今天开始它就交给你来指挥"
- **功能**: 控制幽灵船的核心施法媒介，可能包含精神链接/命令传递等附魔

### E5 海洋女神 vs 风暴之主格局
- **原文** §14-"作为以海洋为核心的神灵，海洋女神也有一个局限性，那就是她无论怎么样都很难把信仰扩大到陆地上。"
- **原文** §8-"索伦突然有一种看见另外一个蜘蛛神后崛起的错觉"
- **战略困境**: 海洋女神局限海域→风暴之主天然优势→暗中的阴谋浮现
- **索伦立场**: 对海洋女神的野心有警觉但仍利用其力量

## 四、写作技法笔记

| 技法 | 用例 | 效果 |
|------|------|------|
| **数据卡片插入** | §47-53 亡语者号面板 | 硬核设定展示，增强真实感 |
| **专长叙事化** | §2-14 杀戮之心专长由复盘自然引出 | 经验结算不打断故事流 |
| **五感调用** | "海水腥涩味"(§16) / "阴冷刺骨寒风"(§38) / "滴滴答答水滴声"(§18) | 强化超自然氛围 |
| **对比联想** | 蜘蛛神后崛起(§11) / 活化城堡(§39) | 提供战斗回忆锚点 |
| **诡异化渲染** | "幽灵船在呼吸，呼吸着死亡的气息"(§42) | 创造恐怖、活著的船的形象 |

## 五、关联伏笔

- **872名亡语者缚灵【锁定】** → 解锁条件/时间点未知，高等级战力预留
- **海洋女神染指死亡领域** → 与后续其他神灵的冲突（含蜘蛛神后路线的可能交叉）
- **"临时船长"** → 索伦与海洋神殿关系的临时性与可消解性
- **不溺者/怨灵分类** → 海洋亡灵生态体系，后续可扩展更多亡灵兵种
