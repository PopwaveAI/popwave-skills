# 设计包 — ch020「返航」

> 拆书提取 | 章型: 收尾·谈判·汇合 | 事件数: 14

## 基础信息
- **卷**: 第二卷「海外群岛」
- **本卷定位**: 索伦携薇薇安离开大陆，抵达海外群岛开启新地图
- **本章功能**: 索伦与娜迦祭司的博弈——成功让风暴教徒背黑锅；护送黄金骨头返航；途中接回女海盗阿黛尔完成招募
- **关键冲突**: 索伦↔娜迦祭司（信任博弈/利益分配）+ 索伦↔阿黛尔（约定兑现）
- **章末钩子**: 阿黛尔如约出现，索伦的新战力加盟

## 智慧实体（本章出场角色状态）

| 角色 | 状态 | 动机/目的 |
|------|------|-----------|
| 索伦 | 已吸收3点神性值，外表平静；趁乱收割人头得3万+杀戮经验+数件超凡物品 | 保全性命/隐藏神性/活下来回岛接阿黛尔 |
| 娜迦祭司 | 跳海找到黄金骨头，发现神性力量"消失"认定为风暴教徒窃取；对索伦有戒心但仍合作 | 护送黄金骨头回海洋神殿，完成任务 |
| 阿黛尔（女海盗） | 红发，在约定海岛等待，倔强表情走向索伦 | 遵守约定，成为索伦的战利品/部下 |

| 深海九头蛇 | 继续与大衮之子战斗 | 拖住大衮之子 |
| 风暴教徒 | 被娜迦祭司认定为窃取了黄金骨头上的神性力量 | 背黑锅 |

## 事件链

### 事件1：黄金骨头无法收入次元袋
- **类型**: 设定验证
- **原文evidence**: "索伦连续尝试了三次，无论是次元袋还是战斗大师手套都无法将这块黄金骨头容纳进去"
- **内容**: 确认次元空间被黄金骨头隔绝，神灵级物质不可被空间装备收纳。

### 事件2：娜迦祭司索要黄金骨头
- **类型**: 对峙/谈判
- **原文evidence**: "娜迦祭司……用冰冷的目光注视着索伦，沉声道：'人类！把它交给我！这是献给海洋女神的祭品！'"
- **内容**: 娜迦祭司手持武器戒备，态度冷峻；如果索伦不给就直接动手。

### 事件3：索伦的让步
- **类型**: 决策/伪装
- **原文evidence**: "索伦的表情露出来一丝犹豫。随即缓缓地收回了手指，稍微后退了一点"
- **内容**: 索伦表现出犹豫后让步，说自己准备交给祭司——表面上配合实则已收下神性。

### 事件4：风暴教徒背黑锅
- **类型**: 剧情转折/幽默
- **原文evidence**: "娜迦祭司……伸手抚摸了一下黄金骨头，然后脸色变得铁青……'那群该死的风暴教徒！他们居然窃取了上面的一部分力量！'"
- **内容**: 娜迦祭司感知到神性力量消失但无法理解索伦的吸收速度，认定是风暴之主的信徒干的；索伦心中松一口气。

### 事件5：娜迦祭司同样无法收纳黄金骨头
- **类型**: 设定确认
- **原文evidence**: "娜迦祭司似乎是想要把黄金骨头收入次元空间，不过她很快就发现这根本不可能"
- **内容**: 即使是娜迦祭司也无法收入空间装备，只能抱着走。

### 事件6：索伦收割战场人头与装备
- **类型**: 拾取/收益
- **原文evidence**: "索伦并没有丝毫的怜悯心，直接控制活化缆绳就开始收割人头和尸体"
- **内容**: 清理昏迷/濒死海盗搜刮财物；从三阶以上敌人身上找到三五件超凡物品（刺剑/弯刀/异种匕首等一阶超凡武器）。

### 事件7：加速戒指——最佳战利品
- **类型**: 物品获取
- **原文evidence**: "最好的超凡装备是一枚加速戒指"
- **内容**: 三阶超凡物品，每天一次【加速术】10分钟效果，移动速度和出手速度提升；不需要充能自动恢复。

### 事件8：娜迦祭司令返航
- **类型**: 命令/决策
- **原文evidence**: "'返回海洋神殿！'这个命令让索伦愣了一下"
- **内容**: 虽九头蛇还在与大衮之子战斗，娜迦祭司却决定先护送祭品回去——黄金骨头比数十万金德勒更珍贵。

### 事件9：娜迦祭司的解释
- **类型**: 对话/设定
- **原文evidence**: "'大衮之子没有那么快被击退！献给海洋女神的祭品不容有任何意外！'"
- **内容**: 祭司用法术接管幽灵船指挥权；鲨鱼人护航返航；索伦确认海洋女神也在谋划什么——深海九头蛇只有神灵才能驱动。

### 事件10：索伦的收益核算
- **类型**: 角色心理
- **原文evidence**: "一路上他捡了不少的人头……居然拿了三万多点杀戮经验"
- **内容**: 正面战场高阶几乎死八成；鲨鱼人幸存不少；索伦觉得没死就很划算。

### 事件11：娜迦祭司对索伦的评估
- **类型**: 潜在伏笔
- **原文evidence**: "一个接近传奇的游荡者有很多用处"
- **内容**: 娜迦祭司盘算从女色方面拉拢索伦（"人类雄性是最喜欢交配的生物"），为后续合作铺垫。

### 事件12：索伦要求停靠接人
- **类型**: 角色行动
- **原文evidence**: "'能不能在这里停一下，我需要接一个人过来。'"
- **内容**: 索伦请示停靠接"一个女人，我的战利品"；娜迦祭司允许但催促快去快回。

### 事件13：海滩等待——阿黛尔出现
- **类型**: 角色登场
- **原文evidence**: "索伦看了一眼空空如也的海滩，脸上不由露出来了一丝失望之色"
- **内容**: 初始海滩无人让索伦失望，但随后红发的阿黛尔出现，脸上挂着一丝倔强表情走来。

### 事件14：索伦的笑意
- **类型**: 温馨收尾
- **原文evidence**: "索伦终于露出一丝笑意！"
- **内容**: 阿黛尔如约等待他，招募成功；章末以温暖的微笑收尾。

## 设定提取
- **神性遗骸不可收纳**: 神灵级/规则级物质无法被次元袋、次元空间容纳
- **索伦的神性吸收速度**: 快得让高阶祭司都无法察觉——这是他的核心优势
- **娜迦祭司的感知**: 可感知神性（祭司中拥有神性的存在有不少——选民/圣者/预备选民/眷顾者）
- **加速戒指（三阶超凡）**: 每天一次【加速术】10分钟，不需充能自动恢复
- **超凡物品等级**: 一阶~三阶不等，从三阶以上敌人尸体搜到
- **杀戮经验获取**: 索伦靠收割人头拿到3万+杀戮经验（正面战场高阶死亡八成）
- **海洋女神谋划**: 神灵才能驱动深海九头蛇；黄金骨头比数十万金德勒更重要
- **索伦的定位**: 被娜迦祭司评估为"接近传奇的游荡者"——有价值但非不可替代
