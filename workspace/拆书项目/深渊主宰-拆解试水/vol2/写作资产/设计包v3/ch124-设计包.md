# ch124 灵魂投影 — 设计包

## 一、本章核心功能
> 世界观扩维章——星界位面初次登场的完整设定输出。通过索伦和歌莉娅的灵魂投影穿越星界之门，展示：(1) 灵魂投影的法术机制与风险；(2) 星界的环境特征（源质/暗能量/半位面/虚空洪流）；(3) 星界对施法者的增幅（法术瞬发/无限能量）；(4) 获得史诗背景专长的数据反馈。功能性上完成了"位面战场"的建立。

## 四、关键设定/物品/能力

| 名称 | 类型 | 作用 | 备注 |
|------|------|------|------|
| 灵魂投影 | 传奇法术 | 灵魂部分离体+半能量化；保留80%实力；死亡=灵魂损伤（可能变白痴/植物人/暴毙） | "散了三魂七魄的一部分"的东方类比 |
| 星界位面 | 位面 | 近乎无极限的平面；银色源质洪流；漂浮半位面；连接所有位面的"放逐之地" | 全卷最重要的位面设定 |
| 源质（暗能量）| 位面能量 | 宇宙基础结构能量；供给半位面成长；让施法者瞬发法术 | 星界的核心机制 |
| 星界旅行者/位面探索者 | 史诗背景专长（部分） | 进入星界自动获取部分信息 | 数据成长回馈 |
| 法术瞬发（星界环境） | 环境效果 | 6环以下全部瞬发；传奇法术半时施法；能量无限 | 星界战斗的核心有利条件 |
| 半位面 | 位面子概念 | 从星界"长出来"的大陆架；连接星界和以太位面；靠源质能量成长 | 施法者最高追求——私人半位面 |

## 十、升级路径/概念衔接
- ch124是整部作品中星界位面的「首次全景介绍」——世界观第二层（位面层）正式展开
- 灵魂投影机制 = 确保角色在星界战死不会直接杀死角色、为后续战斗提供"安全网"但仍有代价
- 法术瞬发设定 = 确保女巫在星界的战斗力不降反升、解释她们敢以投影状态迎战吉斯洋基人的底气
- 史诗背景专长 = 索伦的成长数据回馈——进入星界本身就是一种"修炼"
- 半位面成长设定 = 为vol3索伦建立自己的巫师塔/半位面埋下概念种子
- 结尾的"半位面颤抖"直接衔接ch125的吉斯洋基人+红龙战斗
