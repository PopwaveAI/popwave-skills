## 二、主要势力详解

### 2.1 海洋神殿

| 项目 | 内容 | 证据 |
|------|------|------|
| 信仰神灵 | 海洋女神（混乱邪恶） | ch001 |
| 核心据点 | 泰罗港海洋神殿 | ch001 |
| 头目 | 娜迦祭司（高阶祭司级） | ch001 |
| 武力构成 | 鲨鱼人军团、深海九头蛇、海族战士 | ch020 |
| 政治手段 | 借刀杀人(利用索伦扩张)、坐山观虎斗 | ch030 |

| 南海岸影响力 | 控制在沿海地带举足轻重 | ch090 |
| 神殿规模等级 | 大神殿 > 小神殿（财富需求不同） | ch010 |

**战略立场**：中立偏利用——对索伦保持合作关系，但也算计他
**核心矛盾**：娜迦祭司背信弃义——挑动索伦对付阿斯罗德，开战在即时退缩**（ch167）**
**长远隐患**：与风暴之主的暧昧竞争关系、与大衮的长期敌对

## 三、势力间关系矩阵

### 3.1 同盟/合作链

```
索伦 ←→ 海洋神殿（互相利用）
索伦 ←→ 卓尔精灵（逐步深化的合作）
索伦 ←→ 北地女巫议会（通过歌莉娅建立的情感和政治联系）
索伦 → 猫女露露喵（将来亲自教导）
海洋神殿 → 海族盟友（鲨鱼人/九头蛇等）
卓尔大祭司 ←→ 海寇鱼人（情报联系）
```

### 3.2 敌对/竞争链

```
海洋神殿 ←→ 大衮势力（长期敌对）
海洋神殿 ←→ 风暴教会（暧昧竞争）
索伦 ←→ 海狼（已灭）
索伦 ←→ 沼泽之王（已击杀）
索伦 ←→ 传奇刺客（已反杀）
索伦 ←→ 阿斯罗德（即将决战·第三卷）
恐惧魔神 → 所有恐惧神子（棋子关系）
吸血鬼神子 ←→ 索伦+薇薇安（最终必须面对的敌人）
```

### 3.3 潜在冲突

| 冲突线 | 风险级别 | 预计爆发 |
|-------|---------|---------|
| 娜迦祭司背叛→索伦战后清算 | 高 | 第三卷 |
| 财富教会金牧师两面性 | 中 | 第三卷 |
| 索伦封神→与诸神冲突 | 极高 | 中后卷 |
| 薇薇安恐惧神子身份暴露 | 极高 | 中后期 |
| 圣者浩劫→多方势力洗牌 | 灭世级 | 全卷主线 |
