# 内化方案：「挂鹅卖鸭」→ 深渊主宰

> 产出方式：基于深渊主宰世界宪法（Vol.1 大陆篇10条 + Vol.2 海外群岛篇10条）
> 内化类型：社会心理规则化翻译（非简单怪物替换）
> 状态：🔮 储备 · 未嵌入

## 一、映射策略：四层翻译

深渊主宰的底层规则是 DND 3.5e 的机制翻译成"社会学"（来自拆解结论）。因此"挂鹅卖鸭"不能只翻译成一个怪物伪装卖假货的支线——那太浅了。应该把**社会心理层**翻译成世界规则。

### 层1：表面事件

| 现实 | 深渊主宰内化 |
|:-----|:------------|
| 摊贩宣称卖鹅腿实际卖鸭腿 | 某人宣称出售某物，实际以次充好 |
| 清北学生追捧 | 精英阶层/上级冒险者的集体背书 |
| 16元 vs 25元的价格差距 | 价格远低于市场价——"太便宜一定有鬼" |
| 群友举报 | 内部人揭发 / 竞争对手举报 |

### 层2：社会心理→规则翻译

这是最值钱的层——现实事件的核心社会心理机制，翻译成深渊主宰下的**世界规则**。

**现实机制**：优绩主义迷思 → 清北学生认准的东西=好东西 → "鹅腿阿姨"的品牌价值来自**消费群体的身份光环**而非产品质量本身。

**深渊主宰翻译**：

```
【信誉幻术（Reputation Illusion）】

效果：当一种商品/服务被"高传说度主体"（知名冒险者、
知名公会、高阶施法者）公开认可后，该商品/服务的
市场溢价中产生"非质量溢价"部分。
  
非质量溢价比例 = 认可者的传说度之和 × 0.5%
  即：5位传说度50+的传奇冒险者认可 → 溢价125%
  意味着：10银币的货能卖到22.5银币

深层规则：
  这种溢价本质是"安全感税"——购买者不是在买商品，
  是在买"我跟传奇强者用一样的东西"的身份确认。
```

### 层3：宪法兼容性检查

| 宪法条目 | 冲突吗 | 说明 |
|:---------|:------:|:------|
| ① 神力=意志力 | ✅ 兼容 | 非神性相关，不冲突 |
| ② 杀怪得经验/升级 | ✅ 兼容 | 支线事件，独立于经验体系 |
| ③ 多元宇宙位面通道 | ✅ 兼容 | 不需要位面设定 |
| ④ 法术依赖魔网/神灵 | ✅ 兼容 | 非法术相关事件 |
| ⑤ 神祇可陨落/复活 | ✅ 兼容 | 不涉及神性 |
| ⑥ 超凡属性分水岭 | ✅ 兼容 | 但可作为检测点——超凡属性者不受信誉幻术影响 |
| ⑦ 阵营由行为决定 | ⚠️ 强关联！ | 关键联动！信誉幻术的本质是"虚假宣扬"，在客观阵营判定中会体现 |
| ⑧ 施法者是噩梦 | ✅ 兼容 | 不冲突 |
| ⑨ 经验可弥补等级 | ✅ 兼容 | 非战斗事件 |
| ⑩ 封神需四要素 | ✅ 兼容 | 传说度的侧面验证——虚假传说度可以被积累吗？ |

**关键发现——宪法第7条与"挂鹅卖鸭"的深层共振**：

宪法第7条规定：**阵营由行为对世界的实际影响决定，善恶有客观标准**。那么"鹅腿阿姨"的行为（卖鸭腿但声称鹅腿）在这个世界就是**阵营偏移行为**——不管她心里怎么想，行为本身是欺骗，长期持续会积累邪恶倾向。这是原事件中最能嵌入宪法体系的锚点。

同时宪法第10条的**传说度**机制提供了另一个锚点——如果某人的传说度是"虚假积累"的（众人追捧是因为他们的身份光环而非她的商品质量），那她的封神之路是否会被"虚假传说度"污染？这个设问直接通向世界观的深度。

### 层4：内化后的叙事可能性（按嵌入阶段）

这个事件的叙事结构是**信任经济从建立到崩塌的四幕弧线**。不同阶段嵌入效果完全不同：

#### 选项A：弧线开头嵌入（信任建立期）

**嵌入位置**：大陆篇·白马城时期（ch066-ch085附近）
**形态**：索伦初到一个新城市，发现有一家"全城都在排队"的无名小摊。摊主慈祥热情，顾客发自内心的维护。

**核心叙事功能**：
- 索伦作为外来者，**侦察技能识破诡计**→制度外审视者的优势视角
- 白马城自身的"人间烟火"在最后大战前被温暖化，**埋下"美好的东西可能是假的"的思想伏笔**
- 事件规模小（个人摊贩），不影响主线节奏

**锚点**：宪法第7条（阵营变化）→ 索伦看到顾客慢慢从正常人变成狂热维护者→但**外表正常**→道德模糊带

#### 选项B：弧线中段嵌入（信任运行期）

**嵌入位置**：海外群岛·城市治理期（vol2，索伦建立领地后）
**形态**：索伦自己的领地上，某个街道出现了一个"深受爱戴的老妇人"——她是真心的好，她卖的东西是真的好。居民发自内心信任她。**她确实没有骗人。**

**核心叙事功能**：
- 反转——不是骗子，是真的好人。**让读者（和索伦）产生自我怀疑**：是不是被白马城的经历影响了判断？
- 但深层危机在于：这种"深度的信任"本身可能成为群体的盲点——如果将来这个老妇人被污蔑，居民会暴力捍卫她（对应现实中"第一个揭露者被网暴"）
- **索伦面临治理难题**：如何处理"被信任的人"和"可能被污蔑的人"之间的张力

**锚点**：宪法第2条（力量即话语权）→ 索伦作为领主，不能只靠"武力推翻"

#### 选项C：弧线结尾嵌入（信任崩塌期）

**嵌入位置**：海外群岛·封神酝酿期（vol2后期，索伦接近封神）
**形态**：索伦自己积累了大量传说度。但他开始思考——这些传说度有多少是"真实的"？有多少是因为他是"恐惧神子/海盗王/斩首者"这些头衔的光环？

**核心叙事功能**：
- 宏观上的自我反思——"鹅腿阿姨"的故事**镜像映射到索伦自己身上**
- 索伦问自己：如果我的传说度大部分来自头衔而非实际，那我跟那个挂鹅卖鸭的阿姨有什么区别？
- 直接通向宪法第10条：**封神时传说度是核心要素——如果传说度有水分，封神会出什么问题？**

**锚点**：宪法第10条（封神四要素）+ 第1条（神力=意志力）→ 如果索伦带着自我怀疑去封神...？

## 四、与现有设计包的差异说明

已有设计包 `鬼婆·诅咒肉摊` 走的是**表层翻译路线**（怪物伪装+诅咒肉+打怪支线），这是标准的"借事件外壳做支线"。

本方案走的是**规则翻译路线**（社会心理→世界规则→宪法共振→角色自我反思→封神元叙事），目标是让这个事件**穿透到深渊主宰的世界运行规则层面**，不止做支线素材。

两者可以并存——支线包提供具体的3章战斗内容，规则方案提供主线叙事燃料。但如果你只能选一个，**规则方案的价值期更长**（贯穿到封神阶段），支线包的回收期更短（3章打完就结束）。
