## 【新套路】T013 · 主角职业路线推演章

**定义**：用一整章的内心独白来展露游戏化进阶体系，同时塑造主角的焦虑感和决策力

**适用场景**：
- 等级/经验到达关键节点时
- 新地图/新阶段开启前
- 面临多重职业路线选择时

**结构模板**：
```
当前资源盘点（经验值/属性/技能）
→ 列出可选路径（A vs B）
→ 深入分析路径A（专长需求/属性门槛/前置条件）
→ 深入分析路径B
→ 交差距离计算（还差多少）
→ 挫败感（永远差很多）
→ 极端底牌浮现（阴影巫妖类）
→ 仪式性收尾（撕纸/燃烧）
```

**经典案例**：ch111《决心与推演》——索伦用整夜时间推演刺客→魔影刺客→阴影主宰路线，最后立下阴影巫妖底线

**复用提示**：避免纯数据堆砌，数据必须包裹在"主角情绪反应"中（叹气/敲额头/瞳孔变化）

## 【新套路】T014 · 教学式战斗

**定义**：战斗进行中，强者向新人同时讲解技巧，实现"战斗+设定展露+角色成长"三合一

**适用场景**：
- 高级角色带低级角色时
- 需要展露某职业/体系特色时
- 想用实战自然传递设定时

**经典句式**（直接可套用）：
- "XX看清楚！XX（技巧名）是很重要的技巧"
- "你要时刻记住一点，对于XX而言，XX比XX更加重要"
- "法术不一定要瞄准敌人才有效！"

**经典案例**：ch114歌莉娅在激战中向薇薇安讲解"运动计算""法术预判""控制比伤害更重要"

**三层效果**：
1. 战斗层：推动剧情，解决危机
2. 教学层：传递世界观/战斗知识给读者
3. 角色层：塑造教导者"游刃有余"的强者形象，展示被教导者的成长空间

## 【新套路】T015 · 爆发式能力觉醒

**定义**：角色在情绪极限/生死边缘触发全新能力，通常之前只伏笔未展示

**触发条件**（推荐模板）：
```
环境压力积累（多方向围攻/资源耗尽）
→ 关键触发事件（同伴受伤/重要之人被威胁/情绪崩溃）
→ 身体异变描述（充血/瞳孔变化/影子扭曲）
→ 能力首次展露（名称+加粗/引号标注）
→ 消耗代价暗示（杀戮经验消耗/受伤恢复/不稳定）
```

**经典案例**：ch115索伦在薇薇安爆发+被围杀愤怒充血状态下首次激活【阴影位面穿梭】

**铁律**：新能力必须在前文有伏笔（哪怕很隐晦），不能凭空出现

## 【新套路】T016 · 双神子对照法

**定义**：用同一神性来源的两个不同化身/子嗣做对照，一个展示堕落/邪恶路径，暗示主角色能走向另一条路

**结构**：
1. 核心道具/身份（神子）
2. 负面样板出现（邪恶神子）
3. 负面样板展示该身份的黑暗面（利用身份害人/自保/出卖）
4. → 让读者产生"主角绝对不能变成那样"的情绪
5. → 为主角获得同类力量时如何选择提供参照

**经典案例**：ch113-ch115的精灵游侠恐惧神子 vs 薇薇安恐惧神子，同为恐惧神子，一个恶毒自私出卖他人，一个纯真无辜被动觉醒

**进阶用法**：可以用在同一人身上（主角差点堕落的危机时刻）

## 【新套路】T017 · 战力标杆时刻

**定义**：让队伍中的次要强者在某场战斗中全力展露实力，建立该角色的"上限质感"，同时为主角的成长设定参照系

**推进方式**：
```
其他角色陷入苦战/被压制
→ 配角出手秒杀/控制场面
→ 用数据/旁白侧写其实力（"XXX4阶职业者一击毙命"）
→ 用其他角色的内心OS/神情变化强化震撼效果
→ 收尾金句（"这才是XXX的风采！"）
```

**经典案例**：ch114北地女巫歌莉娅——死亡一指秒混种兽人、火球术预判炸变异怪物、阴影黑触手控制全场

**注意**：战力标杆角色不能频繁全力作战，否则标杆贬值；一场华丽的战斗后应让其回到"节省法术位/需要休息"的限制状态

## 【新套路】T018 · 恶毒聪明的反派揭穿

**定义**：反派在危机中通过揭穿主角一方的核心秘密（而此秘密本应逐步揭露），将矛盾提前引爆

**核心动作**：
1. 反派被追杀途中遇到主角团
2. 故意将灾难引向主角团方向
3. 喊出/揭露主角团某成员的隐藏身份
4. 面对证据，身份无法再隐瞒
5. → 矛盾提前爆发，局势复杂化

**经典案例**：ch115 精灵游侠将猎杀者引向索伦队伍，并大喊"那边的小姑娘也是一位恐惧神子"

**效果**：强迫本应慢慢展开的身份线在读者毫无准备时炸开，制造震撼感

## 【存量套路更新】T003 · 设定展露

**新增技法**：游戏化UI风格的拼装式设定展露

在ch111中，作者使用了类似游戏UI的数据屏显方式来展露职业设定：
```
游荡者进阶的分支极多:
偏向偷窃能力的进阶是【神偷】
偏向战斗能力的进阶是【游荡剑客】
偏向追踪能力的进阶是【赏金猎人】
偏向暗杀能力的进阶是【刺客】
偏向潜行能力的进阶是【潜行者】
偏向闪躲能力的进阶是【影武者】
```

建议在需要快速传递大量分支信息时使用此格式，用"偏向X的是Y"的句式做快速扫读。

## 【存量套路更新】T007 · 世界危机倒计时

**新增机制**：通过多个独立势力的同步行动来展露世界级危机的到来

ch115演示了极佳结构：
```
1. 正文危机（猎杀者围攻）
2. 薇薇安爆发引发波纹
3. 大地行者德鲁伊感知（中立观察者视角）
4. 绝望之手组织出动（第三方势力行动）
```
→ 同一事件从多个视角/势力同步响应，暗示该事件具有世界级影响力

## 【存量套路更新】T002 · 战斗描写

**新增技巧**：法术预判定位法

面对高速目标时，法术不应瞄准敌人当前位置，而应预判其移动轨迹投放到前方路径上。ch114火球术打冲刺变异猎豹即为范例。

写法模板：
```
"时间倒回一秒前，急速冲刺的怪物来不及停下来，完全是惯性力一般，
在它冲过来的同时，也恰好进入了XX（法术）的爆炸核心区域。"
```
