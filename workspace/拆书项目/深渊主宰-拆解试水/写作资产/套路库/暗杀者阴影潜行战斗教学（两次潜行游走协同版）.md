# 暗杀者阴影潜行战斗教学（两次潜行游走协同版）

## 定义
完成整的游荡者/暗杀者战斗流程展示：潜伏等待→一次突袭秒杀→打完就走重新潜行→利用队友牵制→二次突袭改变道具→清理战场。核心是"不硬拼+利用环境+协同队友"的组合战斗方式。

## 适用场景
- 需要展示主角的暗杀者战斗风格
- 队友协同战斗的第一次实操
- 游荡者职业特色教学场景
- 通过战斗展示道具效果（如圣水）

## 核心机制
1. **潜伏等待**：选最佳切入时机（敌人注意力最集中=也是最松懈对其他方向的警惕时）
2. **一次突袭**：必须秒杀→背刺要害+属性/附魔伤害
3. **即刻转移**：打完就走→重新遁入潜行→不等敌人反应
4. **队友牵制**：利用队友持续吸引敌人注意力→自己不暴露
5. **二次突袭**：改变武器/道具→这次加入针对性克制（如圣水/银器）
6. **清理战场**：确认敌人死亡→处理痕迹→回收道具

## 典型结构
```
①主角潜伏等待（利用环境遮蔽）
②等待队友牵制/制造混乱（创造时机）
③第一次突袭→秒杀→收刀→潜行消失
④敌人恐慌→锁定不到目标
⑤队友持续缠斗（维持注意力分散）
⑥第二次突袭→改变策略（使用克制道具）
⑦清理战场→回收材料
```

## 案例参考
**ch174**：索伦潜伏等待→雅莉安娜暴走吸引全部注意力→索伦从阴影突刺杀第一个吸血鬼（弯刀刺心脏+霜冻）→收刀急退→遁入潜行消失→第二个吸血鬼无法锁定目标产生恐惧→雅莉安娜持续缠斗堂姐→索伦二次突袭（圣水浇身+弯刀刺杀）→两个仆从全灭→进入单挑

## 变体
- **道具侧重版**：主要靠陷阱/毒药/圣水等消耗品→战斗动作较少
- **环境侧重版**：利用地形（高低差/阴影/障碍物）创造优势
- **远程版**：弓箭/弩/飞刀暗杀→打完换位置→持续远程压制

## 注意事项
- 两次突袭必须有差异化（不同的敌人/不同的道具/不同的角度）→不能重复
- 队友的牵制要写清楚→让读者理解为什么敌人没有发现主角
- 潜行的使用要有逻辑限制（不能无限隐身→要有"在阴影中"的前提条件）
- 战斗中的道具使用可以有"效果展示"（如圣水腐蚀皮肤）→让读者感受到道具的针对性
