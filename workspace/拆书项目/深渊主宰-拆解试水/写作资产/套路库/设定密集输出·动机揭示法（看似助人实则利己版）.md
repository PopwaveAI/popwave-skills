# 设定密集输出·动机揭示法（"看似助人实则利己"版）

## 定义
先用"主角帮助弱小"建立正面印象，在密集输出的设定信息中逐步释放主角真实动机，让读者自己拼图出"原来主角不是在施舍、是在猎杀/利用"的反转。

## 适用场景
- 需要在一章内完成大量体系设定输出
- 主角有灰色/实用主义的道德立场
- 需要建立主角"不是圣人但读者能理解他"的人设
- 让设定输出本身成为情节推进的一环

## 核心机制
1. **正面引子**：主角先帮助一个弱者（建立读者对"好人"的预期）
2. **设定信息轰炸**：在看似"解释世界观"的过程中密集输出数据/规则
3. **关键词植入**：在输出设定时植入一个"奇怪的关键词"（如"吸血鬼脊椎骨"）→让读者觉得"这个东西好像很重要"
4. **拼图提示**：继续用设定解释暗示"为什么这个东西很重要"（如"这是血脉纯化仪式的关键材料"）
5. **动机翻转**：最后一句点破→"索伦的目的就是这样→他帮雅莉安娜是为了脊椎骨"
6. **读者恍然大悟**：之前的"知识科普"全部变成了"主角的情报储备"→重新审视主角人设

## 典型结构
```
①主角帮助弱者（建立正面印象）
②在解释世界观中输出大量设定（A设定/B设定/C设定）
③在某一个设定中植入"核心材料/道具"的关键词
④围绕该道具继续输出相关设定（制造悬念：这东西好像很值钱）
⑤最后揭示：主角帮弱者→就是为了这个道具
⑥读者：原来如此！（反转成功）
```

## 案例参考
**ch172**：索伦说要帮雅莉安娜杀追捕者（正面印象）→输出吸血鬼生物模板（属性/弱点/等级）→"吸血鬼身上有一个很重要的部件——脊椎骨"→"这是邪恶向血脉纯化的关键道具"→继续输出血脉纯化仪式的原理→"索伦的目的就是这样"→"他需要把生命力的潜能通过仪式催发出来"→"他帮雅莉安娜不是为了善心→是为了吸血鬼脊椎骨"→读者重新评价索伦

## 变体
- **任务奖励版**：主角接任务→输出怪物设定+材料价值→最后揭示主角是为了材料
- **护送版**：主角护送某人→输出其血脉/身世设定→最后揭示是为了接近某个目标
- **情报版**：主角帮忙打探消息→输出势力设定→最后揭示是为了自己的计划

## 注意事项
- 设定输出的数据量要有上限→否则变成数据堆砌→打断叙事节奏
- 动机反转不能太生硬→最好在设定输出时埋下伏笔（如反复强调"脊椎骨"很重要）
- 反转后不能让读者觉得"之前的帮助是假的"→应该是"帮助是真的+但也有自己的私心"
- 适合灰色主角→纯善良主角不适合（因为会让读者觉得虚伪）
