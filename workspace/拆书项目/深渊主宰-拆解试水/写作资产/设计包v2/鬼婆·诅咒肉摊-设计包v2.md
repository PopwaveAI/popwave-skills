## 2. 第Ⅰ章·老街肉香 — 事件链（骨架层）

### 2.1 故事背景

白马城旧城区偏巷尽头，有一家没有招牌的烤肉摊。只在傍晚出摊，掀开铁锅的瞬间，整条街都是香味。排队的人从巷口排到巷尾——老主顾说"艾薇婆婆在这儿卖了十年了，白马城最好吃的烤肉。"

没有人知道肉从哪里来。没有人问。

### 2.2 事件链

| # | 事件 | 类型 | POV | 字数估 | 说明 |
|:-:|:-----|:----|:---|:-----:|:-----|
| 1 | 索伦在旧城区办事（买情报/补给/等人），被香味吸引 | 日常 | 索伦 | ~300 | 香味描述——温暖、浓郁、混合了某种奇异的甜味（读者不知道那是诅咒材料的特征） |
| 2 | 索伦看到排队的人群——平民、商人、甚至两个低阶卫兵也在队伍里 | 日常·观察 | 索伦 | ~250 | 排队的秩序感、说说笑笑的氛围——这不是一个普通的摊，这是一个社区中心 |
| 3 | 卫兵甲闲聊："艾薇婆婆的肉，我吃了八年了——一天不吃浑身不得劲" | 日常→信息 | 路人/卫兵 | ~200 | 轻度暗示——"一天不吃浑身不得劲"既可以是忠实粉丝的夸张，也可以是诅咒的早期提示 |
| 4 | 索伦排队→见到老艾薇——外貌描写 | 建置 | 索伦 | ~350 | 老妇人形象：驼背、笑容慈祥、围裙干净、动作麻利——但索伦的【侦察】技能捕捉到微小的异常（指甲缝里的暗红/手指比普通人长/某个瞬间眼神不是老人的眼神）→ 触发战斗警觉 |
| 5 | 索伦买了一份——系统提示 | 披露·规则的超自然 | 索伦 | ~200 | 【系统提示：您正在食用【诅咒材料腌制的肉块（？？？）】】→ 索伦不动声色吃下去 → 【意志豁免：成功。你感觉到了微弱的意志侵蚀——来自食物的长期积累。单人单次摄入不足以产生效果。】 |
| 6 | 索伦离开——留下悬念 | 悬念 | 索伦 | ~200 | 索伦没有揭穿。回到住处后记录：白马城·旧城区·无名肉摊→疑似诅咒材料。摊主→疑似非人生物。 |

### 2.3 本章核心爽点

| 套路类型 | 结构拆解 |
|:---------|:---------|
| **日常表象下的超自然暗流** | 烟火气的排队场景→索伦的侦察技能捕捉细节→系统提示直接打破日常→读者从"好香"到"这是什么鬼"的认知反转 |
| **信息不对等** | 读者跟着索伦发现真相——但排队的人不知道。索伦看着他们说说笑笑→读者的焦虑感 |

### 2.4 世界观数据嵌入

| 嵌入内容 | 方式 | 证据 |
|:---------|:-----|:------|
| 【侦察】技能的运用 | 技能展示 | 索伦用侦察捕捉到艾薇的异常细节 |
| 【意志豁免】机制 | 系统提示展示 | "单人单次摄入不足以产生效果" |

## 3. 第Ⅱ章·圣骑士的遗物 — 事件链

### 3.1 故事背景

索伦开始调查。他不急着动手——先搞清楚这是什么东西。

在白马城的流浪者营地/废弃岗哨/下水道入口处，他发现了一面残破的银色战旗——白银之手圣骑士团的标志。

战旗上有干涸的血迹和零乱的爪痕。营地附近有被翻过的泥土——三具尸体被草草掩埋。

### 3.2 事件链

| # | 事件 | 类型 | POV | 字数估 | 说明 |
|:-:|:-----|:----|:---|:-----:|:-----|
| 1 | 索伦追查线索——问旅店老板/地下情报贩子/歌莉娅（如果她在） | 调查 | 索伦 | ~400 | 间接信息：老艾薇大约十年前出现在白马城→没人知道她从哪来→有人质疑过她的肉→质疑者后来消失了→"自己离开白马城了"（官方说法） |
| 2 | 索伦在城外废弃岗哨发现白银之手圣旗 | 发现·转折 | 索伦 | ~350 | 银徽章已经发黑——圣旗上的血迹呈喷射状（战斗暴起，非处决）→ 地面有焚烧痕迹 → 索伦做战斗推断："至少三个人。战斗持续不到一分钟。对方人多。" |
| 3 | 索伦挖掘尸体→确认圣骑士身份→死因分析 | 调查·信息 | 索伦 | ~400 | 🚨关键对白（系统提示）：【鉴定：白银之手圣骑士团·第三巡察小队——全员战死。死亡时间：约六个月前。致命伤：钝器打击+撕咬伤。】→ 撕咬伤提示"攻击者是人形生物——但攻击模式像野兽"→ 这就是伥鬼 |
| 4 | 索伦回城——路上看到吃烤肉的人回家——他们有说有笑 | 情绪累积 | 索伦 | ~250 | 索伦沉默地看著欢快的人群——读者知道真相、索伦知道真相、人群不知道——情绪剪刀差 |
| 5 | 索伦决定动手——准备战斗 | 决断 | 索伦 | ~200 | 索伦暗杀流程准备：淬毒弩失、阴影潜伏、战斗法术预备 → 章末悬念 → "明天——关店。永远的。" |

### 3.3 本章核心爽点

| 套路 | 结构 |
|:-----|:------|
| **调查→推理→决断** | 系统鉴定给出客观事实→索伦分析战斗痕→还原事件→决定动手——完整推理过程给读者参与感 |
| **情绪剪刀差** | 索伦知道圣骑士已死→他走在欢快的街道上→人群一无所知→读者的焦虑随情绪累积 |
| **惩罚系统** | ⭐ 圣骑士不是被鬼婆单杀——是被**伥鬼群围攻致死**——对应现实中"揭发者被网暴"——这一层的保留让故事从"打怪升级"变成"社会系统的暴力反映" |

### 3.4 世界观数据嵌入

| 嵌入内容 | 方式 | 说明 |
|:---------|:-----|:------|
| 白银之手圣骑士团 | 战旗鉴定 | 与本作已有设定（ch177-185）联动——白银之手曾出现在财富之城 |

## 4. 第Ⅲ章·百年老店 — Boss战设计

### 4.1 鬼婆·老艾薇

**伪装身份**：白马城旧城区·无名烤肉摊摊主——"艾薇婆婆"
**真实种族**：夜鬼婆（Night Hag）/ 荒野鬼婆（Annis Hag）混合设定

**属性估算（适配本作DnD体系）**：

| 属性 | 数值 | 说明 |
|:-----|:----|:------|
| 挑战等级 | CR 8~10 | 三阶职业者可单挑；索伦在本窗口约8级，配合战术可胜 |
| 力量 | 20（超凡） | 鬼婆的蛮力——徒手可撕开锁子甲 |
| 敏捷 | 14 | 不算快，但步伐诡异 |
| 体质 | 20（超凡） | 再生能力——普通武器伤口会自行愈合 |
| 智力 | 16 | 狡猾——不是蠢货 |
| 感知 | 16 | 危险感知——索伦的潜行需要150+才能骗过 |
| 魅力 | 8 | 伪装时提升至14（伪装术加成），真身暴露后降至8 |

**核心能力**：

| 能力 | 效果 | 类型 | 说明 |
|:-----|:-----|:----|:------|
| **伪装术（已常驻）** | 外表为慈祥老妇人——人类形态下侦察/潜行对抗难度+5 | 超自然能力 | 只有索伦级的高侦察才能日常识破 |
| **诅咒烹煮** | 用地狱/沼泽材料腌制的肉——食用者每餐做一次意志豁免（DC 14）→累积失败3次→被动转化开始 | 诅咒 | ⭐ 核心规则：只有"有身份焦虑需求"者豁免劣势 |
| **伥鬼转化** | 完全转化后→失去自我意志→成为鬼婆的护卫→对外表现为"狂热捍卫者" | 诅咒·长期 | 转化者外表不变——这是最阴险的——白天正常生活，晚上鬼婆召唤时变成护卫 |
| **夜嚎** | 夜鬼婆的招牌能力——尖啸后召唤范围内的所有伥鬼 | 召唤能力 | 第Ⅱ章埋伏的线索回收——圣骑士就是这么被群攻致死 |
| **利爪** | 徒手攻击——2d6+5（超凡力量加成）——带诅咒伤害 | 近战能力 | 物理+诅咒双属性 |
| **变化外形** | 变形成猪/狼/乌鸦——用于逃跑/侦查 | 变形能力 | 如果索伦打草惊蛇→鬼婆可能变形逃跑 |
| **强韧再生** | 每轮回血5点——需火焰/神圣伤害才能抑制 | 再生能力 | 🚨 需要利用火焰/圣水/神圣武器——索伦需要提前准备 |

### 4.2 伥鬼（被诅咒转化的受害者）

**属性**：

| 属性 | 数值 | 说明 |
|:-----|:----|:------|
| 挑战等级 | CR 1~3（个体） | 单个弱——群攻致命 |
| 力量 | 14~16 | 被诅咒强化——比普通人力气大 |
| 敏捷 | 12 | 正常 |
| 体质 | 14 | 受伤不知痛 |
| 智力 | 4~6 | 大幅降低——只剩基础语言能力和鬼婆的命令理解 |
| 感知 | 10 | 正常 |
| 魅力 | 6 | 眼神空洞（仔细看能发现） |

**核心特征**：

| 特征 | 说明 |
|:-----|:------|
| **外表正常** | 白天看起来完全是普通人——说话、走路、工作——**只有夜间或鬼婆召唤时才显露异常** |
| **狂热捍卫** | 如果有人质疑鬼婆→伥鬼会主动攻击质疑者→事发后不记得自己做过（记忆模糊） |
| **群体行动** | 鬼婆的夜嚎召唤范围内所有伥鬼→群攻效果 |
| **可逆性** | ⚠️ 转化初期（1个月内）可被【移除诅咒】+【复原术】恢复→超过3个月不可逆 |
| **死亡后** | 灵魂回归自由——但肉体的死亡不可挽回 |

### 4.3 Boss战·事件链

| # | 事件 | 类型 | 说明 |
|:-:|:-----|:----|:------|
| 1 | 索伦夜间突袭——潜行接近摊位→鬼婆正在后屋处理"材料"→索伦目睹真实外貌 | 战斗·开场 | 鬼婆不再伪装——驼背消失、手指变成利爪、围裙下的皮肤是灰绿色的——索伦看到案板上的"材料"（非人，但足以让读者寒毛竖起） |
| 2 | 鬼婆发现索伦——"你跟他们不一样。你吃得出来。"——第一句对白定调 | 对话 | 🚨 鬼婆**知道**有少数人不会被诅咒影响。她不在乎——"反正就你一个，管什么用。" |
| 3 | 索伦vs鬼婆 第一阶段：近身格斗 | 战斗·高潮 | 鬼婆的利爪+再生——索伦用弯刀+淬毒弩失打消耗——索伦需要制造火焰/神圣伤害来抑制再生 |
| 4 | 鬼婆夜嚎→召唤伥鬼 | 战斗·转折 | 街巷阴影中走出七八个面目呆滞的人影——他们曾经是白马城的居民——现在拿着菜刀/铁锹/木棍围过来 |
| 5 | 索伦vs伥鬼群+鬼婆 第二阶段：混战 | 战斗·高潮 | 🚨 索伦面临选择：全力攻击鬼婆（无视伥鬼→会被包围）or先清理伥鬼（给鬼婆恢复时间） |
| 6 | 关键抉择：索伦击杀鬼婆的方式 | 战斗·收尾 | 可以选择：A. 弯刀斩首（快速解决）→但伥鬼失去控制后怎么办；B. 用圣水/火焰法术后击杀（抑制再生）→击杀速度慢但干净；C. 留活口→问出诅咒的解咒方法→但鬼婆可能骗人 |
| 7 | 战后处理·伥鬼 | 抉择·余韵 | 伥鬼失去控制后→茫然站在原地或倒地→可尝试【移除诅咒】+【复原术】→但只有初期转化者可救→长期转化者……只能杀死释放灵魂 |

### 4.4 Boss战爽点

| 爽点 | 设计方式 |
|:-----|:---------|
| 🩸 **信息差转化为战力差** | 第Ⅰ章索伦获得了"诅咒材料"的信息→第Ⅱ章获得了"圣骑士被伥鬼围攻"的情报→第Ⅲ章利用这些信息做针对性准备（圣水/火焰武器）→信息差带来的战斗优势 |
| 🩸 **双阶段boss** | 近身格斗→夜嚎召唤→混战——两个阶段打断战斗节奏，避免"站桩输出"的单调 |
| 🩸 **道德抉择** | 伥鬼曾经是人。杀还是不杀？→索伦的灰色立场在面对"被诅咒的无辜者"时被逼到角落——这比打一个纯恶鬼婆更有后劲 |
| 🩸 **规则内嵌** | 圣骑士团的覆灭不是背景故事——是伥鬼群攻的实战预演→读者在战斗前就已经知道"这群东西一起上有多危险"→紧张感天然存在 |

## 5. 诅咒触发条件的规则化·核心设计

### 5.1 诅咒规则

```
【鬼婆的诅咒肉食】

类型：诅咒·渐进·认知侵蚀
豁免：意志（DC 14）
触发条件：
  ┌─────────────────────────────────────────────┐
  │  ① 食用诅咒材料制作的肉食                   │
  │  ② 自身的身份认同存在"外部依赖倾向"           │
  │     （豁免时自动劣势——DC不变但投骰两次取低）  │
  │                                              │
  │  "外部依赖倾向"定义：                         │
  │  - 需要通过消费特定商品/服务来确认社会身份    │
  │  - 需要"跟别人不一样"来获得安全感             │
  │  - 对外部符号（品牌/店名/摊主的人设）的依赖   │
  │    超过对产品本身质量的判断                   │
  └─────────────────────────────────────────────┘

累积机制：
  每餐判定一次。
  失败累积到3次 → 开始转化（意志降至10，魅力降至8）
  失败累积到5次 → 完全转化（智力降至4，彻底成为伥鬼）
  成功累积到3次 → 获得临时免疫（7天内不再受影响）

豁免优势条件（即不受影响人群）：
  - 职业者等级≥5（身份由战斗力和等级锚定）
  - 传说度≥3（身份由传说/声望锚定）
  - 拥有超凡属性（20+者自我认知极度稳固）
```

## 6. 嵌入检查

### 6.1 推荐嵌入窗口

| 维度 | 评估 |
|:-----|:------|
| **最佳嵌入章号** | ch070-ch080（白马城烽火期——主线大战之前/间隙期插入） |
| **前置条件** | 索伦已抵达白马城+已建立旧城区情报网络（ch044已铺垫科凡地头蛇） |
| **避让冲突** | 不与碎骨食人魔部落主线（ch080-085冲突）、风暴领主之战时间重叠 |
| **触发方式** | 索伦在城内买情报时"不经意"闻到香味→自然的日常触发，不打断主线节奏 |

### 6.2 不建议做的事情

- ❌ 把鬼婆设为主线的核心反派（她是支线，削弱了恐惧神子主线的压迫感）
- ❌ 让索伦救回所有伥鬼（必须有一部分不可救→灰色代价）
- ❌ 让歌莉娅直接鉴定出诅咒（需要索伦自己发现→读者参与调查）

### 6.3 如果需要缩短（1章版本）

如果只有1章篇幅，压缩为：
```
① 索伦偶然发现摊位异常→② 系统提示→③ 索伦直接夜间突袭→④ boss战
→ 圣骑士队的设定改为通过尸体身上搜到的文件来揭示（不独立成章）
→ 降低伥鬼的设定权重——只出现在boss战中作为阶段转换
```
