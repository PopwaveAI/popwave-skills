# ch003 · 墓园区

雨后的道路一片泥泞。

索伦抱着希斯的尸体走在前面，薇薇安跟在身后，两个人在贫民区的街道中穿行。低矮的屋棚密密麻麻地挤在道路两侧，屋檐还在滴水，地面上到处都是积水坑，一脚踩下去，浑浊的泥浆溅上裤腿。

空气中永远存在着一股恶臭味。腐烂的垃圾、陈年的污水、说不清来源的酸腐气息，混杂在一起，成为这片区域特有的味道。

路过的人投来目光。

"索伦！那是索伦！他醒过来了！"

窃窃私语在两侧响起，像风吹过水面泛起的波纹。

"这下子有好戏看了。那些打他妹妹主意的人，怕是要吃点苦头。"

索伦没有回头。他继续走，抱紧怀里冰冷的老狗。

"哥哥。"薇薇安在后面小声唤道。

"去墓园。"索伦道，"希斯应该被安葬在那里。"

小女孩愣了一下，没有多问，跟了上来。

--

从出生长大到如今，索伦从来没有像今天这样认真地看过这座城。

琥珀城。

这个名字让他感觉到一丝熟悉。曾经在游戏中，他也听说过这个名字——那是《诸神之战》里北地的一座中型城市，以琥珀矿脉和女巫议会闻名。但游戏中的琥珀城富丽堂皇，街道铺着青石板，城中有高耸的法师塔和繁华的集市，和眼前这片被雨水泡烂的贫民窟完全是两个世界。

不过那个时候，已经是史诗剧情开启之后了。

他的脚步顿了顿。脑海中浮现出一段段数据——不是穿越前的游戏画面，而是那个世界的历史。

万神殿。

那是无尽世界中的至高殿堂，诸神议事的地方。在那个世界里，神灵不是虚无缥缈的信仰，而是真实存在的、可以干涉凡间事务的存在。他们掌管着秩序、死亡、生命、战争、魔法、阴影——所有你能想象到的领域，都有对应的神灵在注视着。

然后某一天，一切都变了。

诸神黄昏。

没有人知道起因是什么。有人说是一位古老的神祇从星界深渊中苏醒，有人说是一场跨越位面的神战，也有人说是因为凡人的信仰崩塌——真相已经湮没在历史中。但结果所有人都知道：万神殿内的诸神陨落了三分之一。

魔法女神陨落，秩序之主陨落，阴影主宰不知所踪。神国从外层位面坠入主物质位面，天空中出现了一道道裂痕，像是一面镜子被打碎后的纹路。

那是圣者的时代。也是属于诸神的时代。更是天界诸神最后的一场狂欢。当浩劫结束，属于凡人的时代也即将开启。

所有人在浩劫中都至少死过一次。

索伦的脑海中闪过这句话，脚步微微一顿。

薇薇安抬起头看他，他没有说话，继续往前走。

--

贫民区没有墓园。只有乱葬岗。

在那片低洼的荒地上，尸体被随意堆放，风吹日晒，无人问津。捡垃圾的人会在夜间剥去死者身上值钱的东西，野狗在更深的时候出现，拖走骨头。有时候连尸体都会消失——被不知名的施法者带走，不知道用于什么用途。

索伦知道这些。因为原身的记忆中有这些画面。

但他不会把希斯葬在那里。

他穿过贫民区最肮脏的那条巷子，走上了通往城区的大路。薇薇安跟在他身后，有些不安地望着周围陌生的环境——她很少离开贫民区，这里的街道宽阔得让她不适应。

"我们要去哪里？"她问道。

"墓园区。"

索伦记得最开始进入游戏的时候。没有光芒四射的装备，没有震撼全服的首杀公告。他只是一个在训练场里连木桩都砍不利索的平民角色，和成千上万的玩家一样，从零开始。

那时候他觉得这个游戏太难了。怎么会有这么逼真的世界，物价这么高，npc这么刻薄，连学个基础技能都要攒半个月的金币。

但现在他真的活在这个世界里了。

每一口呼吸都是真实的。每一条命都只有一次。

他低头看了看怀里冰冷的希斯。它活着的时候，用这条命守护了薇薇安半个月。

--

墓园区到了。

这里和贫民区完全不同。没有恶臭，没有泥泞，没有拥挤的屋棚。一片开阔的土地，绿草如茵，白色的墓碑整齐排列，像一排排沉默的守卫。空气中弥漫着雨后青草和泥土的气息，和贫民区那种腐烂的味道判若两个世界。

一位穿着旧袍的老牧师站在入口处。他望着索伦和他怀中的老狗，目光平静，没有惊讶，也没有鄙夷。

"你想把它葬在这里？"

"是。"

老牧师看了看他身后的薇薇安，又看了看索伦怀中已经僵硬的老狗，沉默了片刻。

"按照神殿的规定，安葬一具尸体需要缴纳供奉。"

索伦从口袋里摸出一枚银德勒，放在掌心，递了过去。那枚银币被他的体温捂得温热，边缘还有一些没擦干净的泥渍。

"这是为了安葬它献给神殿的供奉。"

老牧师低头看着那枚银币，又抬头看了看索伦的脸。少年很瘦，脸色苍白，衣服破旧，但眼神很平静。那种平静里有种说不出的分量——像是一个已经想好了所有后果、并且愿意接受的人。

老牧师接过银币，点了点头。

"也许它是一只忠诚的猎犬，才值得你这么做。"

他指了指墓园东侧的一角："那边还有空位。去罢。"

--

墓园的泥土是松软的。

索伦找了一处安静的位置，用手和一根捡来的木棍，在地上挖出一个不大不小的坑。薇薇安蹲在旁边，没有哭，只是安静地看着他把希斯的尸体放进坑中，一捧一捧地盖上泥土。

她忽然开口道："希斯不会觉得疼吧？"

"不会。"索伦道，"它已经走了。"

"那就好。"

泥土全部盖上了，和周围的地面齐平。没有墓碑，只有一小堆新土，在阳光下泛着湿润的光。

"希斯！……"小女孩蹲坐在那座不起眼的坟墓前，低声说了几句话。声音太小了，索伦听不清说了什么。然后她站了起来，表情很平静。

她走到索伦身边，伸出手，握住了他的手。

"哥哥。我们回去罢。"

她的手很小，还有些凉。但握得很紧。

索伦点了点头，握紧她的手，带着她离开墓园。

身后的墓园重新恢复了宁静。阳光落在那堆新土上，像是镀了一层金。
