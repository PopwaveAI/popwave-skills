# ch004 · 一切都会好起来

花了三个铜德勒，买到了一根手臂粗长的黑面包。

索伦掰下一块递给薇薇安，自己也咬了一口。那东西冷掉之后硬得像根棒槌，咬下去嘎嘣一声，满嘴都是粗糙的麦麸碎屑，嚼起来像是在嚼锯末。唯一的办法是泡进热水里，等它软成一团糊状，才能勉强咽下去。

薇薇安蹲在灶台边，捧着碗小口小口地喝，热气扑在她脸上，熏得鼻尖红红的。

索伦坐在旁边，慢慢嚼着自己那份。他的动作比薇薇安更慢，像是在用咀嚼的时间思考什么。

从穿越到现在，他已经越来越清楚地感觉到一个变化——这个身体里的灵魂不再是两个。前世《诸神之战》的经验数据和今世原身的记忆，正在以一种诡异的方式融合在一起。他现在想事情的时候，已经分不清哪些是前世的游戏知识，哪些是原身的生存经验。它们混在一起，成了他的一部分。

他必须要对自己的行为负责。更要对身边小小的薇薇安负责。

“哥哥。”薇薇安放下碗，小声唤道。

“嗯？”

“明天我们还会挨饿吗？”

索伦看着她那张被热气熏得微红的小脸，忽然笑了一下。他伸出手，轻轻揉了揉她的头发。

“明天就给你买好吃的！一切都会好起来的！”

薇薇安没有说话。她低下头，继续喝那碗泡软的黑麦糊，嘴角微微翘起。

她记得，从很小的时候开始，哥哥就总是这样说。每一次她害怕的时候、哭的时候、饿得睡不着的时候，他就会蹲下来，用那双还带着少年气的眼睛看着她，说一句“一切都会好起来的”。然后下一次，真的就会好起来。

所以她相信这句话。

入夜后，薇薇安睡下了。索伦听着她的呼吸声均匀起来，才慢慢站起身，走到墙角，翻开了床板下的暗格。

一柄寒光闪烁的匕首躺在那里。

他握住刀柄，轻轻提起，在昏暗的火光下端详了片刻。刀刃上没有任何锈迹，被原身保养得很好。

他嘴角露出一丝冷笑。

那些敢于打薇薇安主意的杂碎，也应该送他们下地狱了。

--

火炉酒馆。

贫民区唯一的酒馆，入夜后喧哗一片。劣质麦酒的味道从门口一直飘到街对面，里面挤满了各种面孔——苦力、小贩、赌徒，还有一些揣着匕首坐在角落里的灰色人物。

索伦混在人群中，披着一件深灰色的旧斗篷，坐在最暗的一张桌子旁。

“索伦醒过来了。”一个刀疤瘦子喝了口酒，大声道，“有人要倒霉了。”

“巴拉斯和卡诺波之前收了一个大人的定金，要把那小丫头绑走……”旁边的人压低声音，“索伦知道这事，能善罢甘休？”

“知道又怎么样？他刚醒，身体还没恢复。”

索伦没有动。他静静地坐在那里，像一截枯木。

又过了一会儿，酒馆的门被推开，一道冷风吹进来。两个男人并肩走了出去——一个高壮，一个中等身材。巴拉斯和卡诺波。

索伦放下手中的空杯，起身跟了出去。

--

夜色灰蒙蒙的。月亮被薄云遮住，只在街巷间投下一层模糊的光。

巴拉斯拐进了一条窄巷，边走边解腰带。他走到墙根下，正准备放水，忽然感觉全身的汗毛乍起——一种说不清的危险感从脊椎骨一直窜到后脑勺。

他下意识地伸手去摸后腰的匕首。

晚了。

一道身影从阴影中凭空出现。寒光一闪，匕首精准地刺入他的后心。与此同时，一只手从身后捂住了他的嘴，把那声即将冲出的惨叫死死地按在喉咙里。

“噗。”

匕首拔出。鲜血涌出。

巴拉斯的尸体软软地倒在地上，发出一声沉闷的轻响。

索伦蹲在尸体旁，看了看刀刃上的血迹，皱了一下眉头。他站起来，三两下把血迹擦在死者的衣服上，然后重新隐入阴影中。

巷外，卡诺波走了一段才发现巴拉斯没跟上来。

“巴拉斯？你好了没有？”

没有人回答。他骂了一句，提着裤子，犹豫着往回走，朝黑漆漆的巷子里探头。

“巴——”

月光下，一道身影躬身在墙壁上一蹬腿，整个人像一支箭一样射了出来。卡诺波只来得及看到一片模糊的黑影在眼前迅速放大，随即一只手已经掐住了他的脖子，巨大的冲击力把他整个人扑倒在地。

他的声音被生生按回了喉咙里。

寒芒划过咽喉，然后刺入了胸口。

刀刃切入的角度精准而果断，像是做过千百次一样。卡诺波的四肢抽搐了几下，不动了。

索伦站起来，低头看了看自己身上的血迹，眉头皱得更深了。

“弄脏了。”

--

一阵暖流在体内浮现。

脑海中，数据面板自动弹出。

“阴影躲藏成功！背刺发动！成功发动背刺*2，对目标造成12点伤害！目标死亡！”
“阴影躲藏成功！背刺发动！成功发动背刺*2，对目标造成16点伤害！目标死亡！”
“抽取目标灵魂能量，获得杀戮经验30点。”

他沉默了片刻。然后蹲下身，在两人身上摸索了一番，摸出五枚银德勒、一把铜德勒，还有一小包劣质毒药。

贫民区的夜依然安静。没有人注意到两条小巷中发生了什么。没有人理会，也没有人追究。

在这里，死一个流浪汉和死一条狗差不多。

索伦把匕首收回腰间，走回夜色中。他的身影渐渐融入了阴影，像是从未出现过一样。
