# ch007 · 十字巷

码头东侧的仓库已经不能去了。索伦换了一个方向，沿着河岸往南走，在另一片货区找到了新的目标——一间半废弃的木材仓库，锁具老旧，护卫松散。

他在里面待了一整个上午。

撬锁、拆陷阱、翻箱倒柜。不是为了偷东西，是为了刷经验。每一把锁被撬开时面板上跳出的数字，每一个陷阱被解除时系统传来的暖流——他在用这些微小的进步，一寸一寸地提升自己。

等到他从仓库里出来时，盗贼技能的数值已经重新分配过了。所有的技能点都被投到了潜行上——从45点加到了68点。这是他当前最需要的能力。在这个世界里，能看见你的敌人，才叫敌人；看不见你的，那叫死人。

剩下的点数，他留着给盗窃和陷阱。下一步提升盗贼等级，需要先把陷阱技能提高到30点。

他在码头区转了一圈，身影在人群中穿梭。20点敏捷带来的不止是战斗力的提升，还有身体对细微动作的绝对控制力——手指的灵活性、步幅的精确度、身体重心的微调，全部进入了另一个层次。

等到他走出人群时，口袋里已经多了几枚银德勒。

没有人察觉。没有人回头。

--

他靠在码头边一根木桩上，在阳光下打开了自己的面板。

专长栏里，两个醒目的条目：

灵巧左手【个人专长】：你的左手和右手一样灵活。双持武器时副手攻击不受减值。撬锁、盗窃、精密操作时，你的有效技巧等级+2。

过目不忘【个人专长】：你的记忆能力远超常人。你见过一次的景象可以在脑海中精确重现，你听过一次的名字不会忘记。学习和阅读技能书时，你的有效智力判定+2。

他看着这两个专长，沉默了片刻。

前世游戏里的初始天赋，在这个世界成了他最大的资本。没有帮派来压榨抽成，没有老大在上面等着分赃。他偷到的每一枚铜德勒、刷到的每一点经验，全部属于他自己。

这种感觉很好。

--

当索伦回到自己住的那间低矮屋棚时，立刻听到了里面传出的说话声。

是一个陌生的声音。女性的，年轻，柔和，带着一种说不出的温和——像是春天里的风。

他推开门。

屋里多了一个人。一个看起来十七八岁的少女，穿着白色的简朴长袍，腰间系着一条细绳，领口处绣着一轮淡金色的晨曦图案。她正蹲在灶台前帮薇薇安烧水，听到门响，转过头来。

看到索伦的一瞬间，她的表情从温和变成了冷淡。

她站起来，拍了拍裙子上沾到的灰。

“既然他回来了，那我便离开了。”她对薇薇安说了一句，然后看了一眼索伦，微微点了点头，算是打过招呼，径直朝门口走去。

经过索伦身边时，她停了一下。

“你自己一个人在这里要多多小心。”这话是对薇薇安说的，但她的目光落在索伦身上，“有什么事情可以去晨曦神殿找我。”

说完她便走了。门在她身后轻轻关上。

索伦没有拦她。晨曦神殿的见习牧师——她的长袍领口的晨曦图案已经说明了身份。在诸神黄昏的历史中，掌管晨曦的那位神灵，在混乱年代开启后不久就陨落了。

但那还早。现在还早。

“哥哥。”薇薇安小步跑过来，“刚才那个姐姐人很好，她帮我烧了热水。”

索伦揉了揉她的头发。

“安亚丽。”

“咦？哥哥怎么知道她的名字？”

“……猜的。”

他不确定自己是在哪个碎片记忆中看到过这个名字。原身的记忆太杂了，像是被打碎后重新拼起来的玻璃，有些地方清晰，有些地方模糊。

但那个女孩的长袍和徽章，他认得。

--

晚上的时候，薇薇安坐在床边，晃着小腿，忽然道：“今天有人鬼鬼祟祟地躲在附近。好像是科尔的手下。”

索伦正在切熏肉的手停住了。

停顿很短，短到薇薇安没有注意到。但他手中的匕首在那一瞬间转了半个圈——不是切肉的角度，是捅人的角度。

然后他又恢复了切肉的动作。

“什么时候？”

“下午。那个姐姐走了之后没多久。幸好安亚丽姐姐过来，这才把他们给吓跑了。”

索伦没有接话。他把切好的熏肉放进锅里，用木铲翻了翻，油花溅起，发出滋滋的声响。

科尔的眼中迸发出一丝杀机。

但他背对着薇薇安，她没有看到。

“明天我就带你去城内，先找一个旅店住下来。”他平静地说道，“你呆在那里安全些。”

薇薇安没有多问。她只是点了点头。

--

晚餐后，索伦从口袋里摸出一样东西，放在掌心里递给薇薇安。

一块褐色的麦芽糖。用劣质糖浆熬的，颜色发暗，表面还有些黏手。码头区小摊上卖的那种，一个铜德勒能买三块。

薇薇安的眼睛一下子亮了。

“哥哥！好甜好好吃！你也吃一块吧。”

她掰下一半，踮起脚尖，往索伦嘴边塞。索伦张嘴接住，嚼了嚼。

涩涩的。甜甜的。

贫民区只有这个了。

两个人在昏暗的火光中嚼着糖，谁也没有说话。灶台上的锅里还在翻腾着热水和熏肉片，蒸汽模糊了他们的脸。

“哥哥你油放多了。煎鸡蛋只要放一点点油就可以了。”

“……知道了。”

--

深夜。贫民区彻底安静了下来。蟋蟀的叫声从墙缝里渗进来，偶尔远处传来一两声不知道是狗叫还是人嚎的声音，然后再次归入沉寂。

薇薇安已经睡着了。她侧躺着，一只手紧紧攥着索伦的衣角，像是在梦里也不愿意放开。

索伦躺着没有动。

过了很久。

他一直闭目躺着，呼吸平稳，像是睡熟了。然后，黑暗中，他睁开了眼睛。

他的动作极轻。先是慢慢地把被薇薇安攥住的衣角抽出来，然后一寸一寸地从她身边移开。她翻了个身，嘴里含糊地嘟囔了一声，没有醒来。

他无声地下了床，赤着脚踩在冰冷的地面上，摸黑穿上了衣服。然后他走到墙角，从暗格中摸出那柄匕首，握在手里掂了掂，放入了后腰的腰带中。

他走回到床边，低头看了一眼床上那个蜷缩着的小小身影。

眼神从温柔变成了冷冽。

然后他转身，推开门，走进了夜色中。

门在他身后轻轻关上。

咔嗒。

黑夜是杀戮的时刻。
