# ch008 · 杀人夜

十字巷。

科尔帮派的据点在一间老旧的院子里。三间连排的砖木屋，围墙矮得能翻过去，门口挂着两盏快熄灭的油灯。光线昏暗，阴影稠密——对索伦而言，这样的环境再好不过了。

他在黑暗中站了一会儿，没有急着行动。先是听。风的方向，屋内的说话声，脚步声的位置和频率。

然后他动了。

--

第一个走出来的是个鹰钩鼻的恶棍。他打着哈欠，摇摇晃晃地走到巷子的墙根下，解开裤子就开始放水。水流声在安静的巷子里格外清晰，掩盖了背后的脚步声。

索伦像一道影子一样贴了上去，左手捂住他的嘴，右手的匕首划过一道寒光。

一刀。干净利落。

鲜血从喉咙的切口涌出，顺着指尖往下淌。恶棍的身体猛地抽搐了几下，然后瘫软下来。索伦卡住他的脖子，等了几秒，确认不再挣扎，然后无声地把尸体拖进墙角的阴影中。

“阴影躲藏成功！背刺发动！成功发动背刺*2，对目标造成30点伤害！目标死亡！”

数据面板弹出又消失。

他直起身，走向第二个。

--

第二个恶棍发现第一个恶棍消失了。他喊了一声，没有人应。他骂骂咧咧地走出来，往巷子里探头——

一把匕首刺穿了他的后心。

强有力的手臂勒住了他的脖子，把他的惨叫按死在了喉咙里。他的身体僵硬了不到一秒，便重重地倒在了地上。

“阴影躲藏成功！背刺发动！成功发动背刺*2，对目标造成32点伤害！目标死亡！”

三十秒。两具尸体。

索伦低头看了看自己身上的血迹。巷子里弥漫着新鲜的血腥味，浓郁得像是打翻了一桶铁锈水。

继续隐藏下去已经毫无用处。

他不再掩饰身形，脚掌发力在地面上一蹬，整个人腾空而起，在墙壁上一接力，稳稳地落在了屋顶上。

--

院子里剩下的打手们已经察觉到了不对。一个声音从屋内传来：“外面怎么那么安静？”

索伦捡起一片瓦片，朝院子的对面扔了过去。

瓦片落地的响声立刻吸引了所有人的注意——“那边！”“过去看看！”

门被推开，三个打手冲了出来，朝瓦片落地的方向跑去。

索伦从屋顶一跃而下。

他的身影在空中划过一条弧线，落在了最后一个人的背上。匕首从后颈刺入，那人连惨叫都没来得及发出，直接扑倒在地。剩下的两个人猛地转身，只看到一道黑影在地上翻滚后迅速起身——

索伦的左手一甩，一柄飞刀脱手而出。

“噗。”

飞刀钉入了其中一人的咽喉。他双手捂着脖子，喉咙里发出咯咯的声音，跪倒在地。

最后一个人终于拔出了武器，但索伦已经近身。匕首从他的肋骨缝隙中刺入，直入心脏。

三个。四秒。

--

屋内，加里斯已经抓起了靠在床边的长剑。

他穿着皮甲睡觉——这是一个老兵的警觉。当他听到外面接二连三的声响时，他知道来的人不止一个，而且来的人不是普通的盗贼。

他冲出屋子，正好看到索伦从最后一具尸体上拔出匕首。

“你——”

“铛！”

加里斯抢先出剑。他的军用剑术大开大合，每一剑都带着沉重的力道，逼得索伦连连后退。匕首与长剑在空中碰撞三次，火星迸发，在黑暗中格外刺眼。

铛！铛！铛！

索伦的手腕被震得发麻。盗贼绝对不能跟战士打持久战——力量差距太大了。

但他没有退。他侧步滑踢，身体像一条蛇一样侧身绕过加里斯的长剑攻击范围，匕首的刀尖划过加里斯的腰部。没有刺穿皮甲，但刀尖上涂抹的毒液已经顺着刀势渗进了被划破的皮革缝隙。

几秒后，加里斯感觉到腰侧传来一阵麻痹感。

有毒。

他的动作出现了一个极其短暂的迟滞。

索伦抓住了这个迟滞。他的脚步忽然变得诡异——不是后撤，不是侧移，而是用一种难以预测的轨迹绕到了加里斯的背后。那是《诸神之战》中最基础的战斗技巧之一，被称为“暗影步”的残缺版本。真正的暗影步可以穿越阴影瞬间移动到敌人背后，而索伦现在只能靠步法和敏捷来模拟那个效果。

但对于一个已经中了毒的人而言，够了。

匕首刺入了加里斯的颈骨。然后翻转刀柄，转了一圈。

加里斯的眼睛瞪得很大。他张了张嘴，想要说什么，但最终只吐出了一口血沫，然后身体直挺挺地倒下。

索伦站在他的尸体旁，微微喘息了片刻。

他低头看了看加里斯的尸体。

“能够死在暗影步下，也算对得起你职业者的身份。”

--

他提起加里斯的尸体，从大门撞了进去。

“嗦！”

弩矢破空而来。索伦把尸体往前一挡——弩矢钉入了加里斯的后背。他借着尸体挡箭的掩护，连续三步蹬踏墙壁，整个人在墙面上横着助跑了三米，然后翻身落地。

落地时他已经贴近了弩手。匕首划过，弩手捂着喉咙倒下。

剩下的人彻底崩溃了。有人想要翻墙逃跑，有人跪地求饶。

索伦没有停。

他的身影在院子中穿梭——每一刀都精准，每一刀都致命。不到三分钟，院子里已经没有站着的人了。

他穿过满地的尸体，推开了最后一扇门。

科尔瘫坐在墙角，手中握着弓弩，但弩弦已经没有上箭了。他看到索伦走进来，脸色惨白，嘴唇哆嗦着。

“索……索伦！……不要杀我！……我让你做头儿！……以后你就是老大！……”

索伦走到他面前，低头看着他。

没有回答。没有表情。

一道寒光。

科尔的话戛然而止。他的喉咙上多了一条细细的红线，鲜血从红线中缓缓渗出。他双手捂住喉咙，身体软软地滑倒在地上。

索伦面无表情地割开了他的喉咙。

--

一共是十一个人。其中有三个进阶成为职业者，可惜都是一级。

他蹲下身，开始在尸体上翻找财物。搜出了十几枚银德勒、一把备用的匕首、还有一些零碎的杂物。

他扯起房间里的一块床单，把值钱的东西卷了进去，系成一个结实的包袱，扛在肩上。

走到门口时，他回头看了一眼满地的尸体。

然后他转身，走进了夜色中。

他离开后不久，几条黑影鬼鬼祟祟地靠近了院子。那些是住在附近的贫民，他们等打斗声平息后，像食腐动物一样溜进现场，翻找尸体上残留的值钱东西——靴子、腰带的铜扣、甚至加里斯那件被刺穿的皮甲。

没有人理会那十一个人是怎么死的。

这里是贫民区。发生这样的事情再正常不过了。
