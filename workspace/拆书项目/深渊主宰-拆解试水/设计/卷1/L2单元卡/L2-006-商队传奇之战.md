### 阶段详解

#### 阶段一：旅途启程与危机前兆（ch030）

| 节拍 | 事件 | 不可逆变化 |
|:-----|:-----|:-----------|
| 权力博弈·A线 | 琥珀城警备队连续失踪十多人→卡拉斯爵士暴怒→神殿骑士要求出动圣武士→贵族拒绝（灰色地带必被清洗）→最后通牒（天黑前不出权则教会自行处理） | **琥珀城权力格局展示**；圣武士vs贵族的深层冲突建立；倒计时压迫感 |
| 旅途启程·B线 | 离开3小时→农田减少→进入野地森林→道路变窄→泥泞路面→商队护卫全面运转 | 从文明到荒野的渐变；30公里=野外警戒线 |
| 女巫初接触 | 北地女巫主动邀请薇薇安上马车→"马车里面很舒适，也很安全"→薇薇安凭18感知判断"她不是坏人"→接受邀请 | **神秘角色与天赋者的首次接触**；女巫对薇薇安的特别关注 |
| 侦查·发现 | 护卫队长征召索伦→发现地精脚印（几十只/可能有术士）→发现掘地虫痕迹（新鲜泥土/血迹/五尺宽/群居）→"麻烦大了" | **旅途第一道危机预兆**；多层侦查法（先发现轻威胁→再发现重威胁） |

| 预警指挥 | 索伦识别→命令撤离→"所有人不要分散！"→挑战面板弹出（三阶/挑战8/B+）→前世事件对比（掘地虫暴走事件未触发） | 前世经验首次大规模运用；指挥权试探 |

## 六、节奏控制

| 节奏模式 | 覆盖章节 | 比例 | 说明 |
|:---------|:---------|:-----|:------|
| 旅途建置/权力博弈 | ch030 A线+B线、ch035温情段 | ~15% | 信息密集但节奏舒缓，承担世界观展示功能 |
| 怪物遭遇/战斗 | ch031（首杀）+ ch033（团战） | ~18% | 快节奏·感官密集·高潮战斗 |
| 决策/整军/准备 | ch032（整军出战） | ~9% | 中速·人情+战术的过渡 |
| 战后休整/医疗/盘点 | ch034（休整）+ ch035前半 | ~15% | 慢节奏·温情+知识变现+医疗硬核 |
| 法爷威慑/不战屈人 | ch035后半（狼群退散） | ~5% | 中速·惊艳·压迫与释放 |
| 传奇之战（远观） | ch036-ch037 | ~18% | 极快·数据震撼·连续高潮·断章悬念 |
| 过渡/科普/日常 | ch038（剑齿虎/红龙）+ ch039（拳法大师） | ~15% | 慢节奏·世界观填充·温情 |
| 小镇/观摩/系统科普 | ch040 | ~9% | 中速·教学感·设定输出·幽默收尾 |

**节奏特征**：
1. **三波脉冲递进**：第一波（ch031首杀）为单人越级击杀，节奏快但篇幅短；第二波（ch032-ch033团战）为团队全员作战，节奏从决策→指挥→牺牲→高潮，篇幅更长、层次更丰富；第三波（ch036-ch037）为传奇之战远观，虽然索伦不直接参战，但通过天象/声音/数据面板制造最强震撼
2. **「女巫暗线」贯穿全单元**：从ch030邀请薇薇安→ch032正式登场→ch034让利→ch035光亮术威慑→ch036反制预言→ch038找薇薇安交谈——女巫的层次递进式展示贯穿11章，为后续做铺垫
3. **战斗后必有沉降**：ch031首杀后接ch032人情+决策；ch033团战后接ch034全章休整——战斗高潮后的沉降期至少1章，绝不连续两章高密度战斗
4. **远观高潮的独特节奏**：ch036-ch037的传奇之战不走主角战斗节奏（主角在湖边远观），而是通过「天象（火烧云）+ 声音（轰隆倒塌）+ 读者脑补 + 数据面板」制造高潮——降低了正面大战的描写成本，但悬念感极强
