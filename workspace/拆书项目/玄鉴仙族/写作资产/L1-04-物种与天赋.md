## 一、主要种族

### 1.1 人族

- **分布**：望月湖区各家族（李/卢/费/郁/万/邓等）、青池宗等修仙宗门、黎夏郡/密林郡凡人
- **修炼体系**：轮数体系→练气→筑基→紫府→金丹
- **社会结构**：宗门（顶尖）→ 修仙家族（中层）→ 凡人（底层）
- **特性**：灵窍的有无决定能否修仙——「千人仅三人的开窍率」（ch005韩文许讲述）

### 1.2 山越

- **定义**：北麓地区的蛮族势力，非人族？
- **特征**：
  - 伽泥奚被描述为「人与狼交合所诞」（ch080阿会剌回忆）
  - 山越内部有完整的权力体系（王、部族、巫觋）
  - 使用「杂气法力」而非正宗功法（ch095李通崖的分析）
- **重要角色**：
  - 伽泥奚（死亡，ch105）：北麓山越王者，筑基修为，自碎大道之基而死
  - 阿会剌（被俘，ch080）：山越部将，被李家夜袭俘获
  - 木焦蛮（存活，ch095）：山越巫觋，练气四层，施巫觋咒术者
- **社会结构**：
  - 大厥庭为最高祭坛/权力中心
  - 各部落有各自首领（阿会部等）
  - 巫觋掌握咒术和祭祀权
  - 大祭祀具有最高权威（ch105血祭仪式）
- **经济**：劫掠为主，非农耕文明

## 二、天赋与资质体系

### 2.1 灵窍

| 项目 | 内容 | 出处 |
|------|------|------|
| 定义 | 修仙资质的核心指标，决定能否修炼 | ch005 |
| 比例 | 「仙缘难得——千人仅三人的开窍率」 | ch005 |
| 灵窍传承 | ch100李通崖立家规——灵窍为唯一硬通货 | ch100 |
| 大宗 | 嫡系+庶出有灵窍者 | ch100 |
| 小宗 | 庶出无灵窍者+支系有灵窍者（止步小宗） | ch100 |
| 支系 | 三代无灵窍降为支系 | ch100 |
| 探查 | 练气以上修士可用神识探查对方灵窍（ch125陆江仙探查刘长迭「六轮三窍」） | ch125 |

**灵窍决定论**：家族制度的根基——「灵窍为唯一硬通货，保证大宗人才密度」。没有灵窍的家族成员三代后降为支系，脱离核心。

### 2.2 玄珠符种

- **来源**：法鉴颁布的修仙资质增强
- **功能**：不仅是「有修炼资质」，更能建立与法鉴的绑定关系
- **死后回馈**：符种拥有者死后，其气运、香火、修为信息回归法鉴（ch100李项平之死验证）
- **已分配**：李木田、李长湖、李通崖、李项平、李尺泾（ch009）

### 2.3 箓气天赋

| 类型 | 描述 | 出处 |
|------|------|------|
| 箓气加持 | 通过祭祀法鉴获得「箓」的加持，提升战力/生存能力 | 贯穿全书 |
| 重海长鲸箓 | 李通崖——真元恢复速度惊人 | ch110, ch115 |
| 避死延生箓 | 李项平——能预警致命危险 + 吞噬咒术（异变） | ch100 |
| 大衍天玄箓 | 刘长迭（残存）——超脱于常规修炼体系之外的存在 | ch125 |

### 2.4 剑意天赋

- **月阙剑意**（李尺泾）：自悟剑意，白光如月光——非传授获得，为「自悟」型天赋
- 剑意可击穿筑基级别防御（ch119-ch120）
- 李尺泾以月阙剑意闻名，刘长迭的「前世记忆」也称李通崖为「月阙剑李通崖」（ch125）

## 三、天赋图谱（前150章已知）

```
                    灵窍（先天资质）—— 修仙入门门槛
                     /
修仙资质 —— 玄珠符种（法鉴绑定）—— 死后回馈机制
                 \
                  箓气（后天加持）—— 法鉴赋予
                       \
                        剑意（自悟型）—— 最罕见的天赋
```

**天赋层级**（基于前150章推断）：
1. **最常见**：有灵窍 → 可修仙
2. **较稀有**：有灵窍 + 获得符种 → 绑定法鉴
3. **极稀有**：有灵窍 + 符种 + 获得箓气 → 战力/生存提升
4. **最罕见**：自悟剑意（如李尺泾的月阙剑意）

## 四、巫觋术（非人族的特殊路径）

- **源头**：法鉴箓气体系的劣化/人化版（ch095陆江仙分析）
- **原理**：将《牲祭法》中祈求的对象从法鉴改换成了施法者本人
- **代价**：消耗自身精气为燃料（ch095木焦蛮「一身精气顿时衰落下来几分」）
- **效果**：黑气追踪、咒术攻击——但效果远弱于正宗箓气
- **使用者**：山越巫觋一脉（木焦蛮为代表）
