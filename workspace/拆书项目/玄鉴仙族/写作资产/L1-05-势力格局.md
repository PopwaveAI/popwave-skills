## 二、主要势力详析

### 2.1 李家（主角家族）

| 维度 | 内容 |
|------|------|
| **定位** | 青池宗附属修仙家族，从黎泾村农户起步（ch001） |
| **核心领导** | 李通崖（练气六层） |
| **核心成员** | 李玄宣（练气）、李玄锋（胎息五层玉京轮）、李玄岭（胎息）、李景恬（内史记录者） |
| **前150章终点修为** | 李通崖练气六层（重海长鲸箓）、李玄宣练气、李玄锋胎息巅峰 |
| **资产** | 灵田、眉尺山洞府、储物袋、丹药、符箓、阵旗、丹炉、青乌矿（合作）、日仪玄光阵 |
| **外部关系** | 青池宗附庸（名义）；与卢家矿脉合作；与费家潜在联盟；与郁家表面服从；与狐族情报交换 |
| **动机** | 生存 → 发展 → 仇恨（对青池宗） |
| **ch150危机** | 长霄门追杀灵岩子至家门口——被迫选择帮/不帮/杀 |
| **战力评估** | 练气六层家主+数名练气/胎息族人 → 望月湖中层家族（在郁家面前需示弱） |

### 2.2 青池宗

| 维度 | 内容 |
|------|------|
| **定位** | 三宗七门之一，望月湖区域名义上的统治宗门 |
| **领袖** | 迟尉（掌门，闭关维持生机，秘密猎杀天才炼丹「铺垫道途」）|
| **已知高层** | 元素真人（青衣紫府，执行者但有良知）、司元白（青穗峰主，李尺泾的师父）、萧初庭（已入紫府）、迟灸云 |
| **内部派系** | 迟尉派（炼丹模式）vs 温和派（元素真人同情天才但不反抗） |
| **与李家关系** | 名义附庸 → 实际为猎杀对象（李尺泾被炼丹）→ 前150章末仇恨暗线已建 |
| **战力** | 有紫府级存在（迟尉、元素真人、萧初庭），有灵阵（天元灵阵），有大蛟（独立存在，交易关系） |
| **注** | 前150章对青池宗内部的全貌揭示有限 |

### 2.3 郁家

| 维度 | 内容 |
|------|------|
| **定位** | 密林郡统治者，望月湖区的新秩序建立者 |
| **领袖** | 郁萧贵（筑基修士，单手杀练气巅峰，ch135） |
| **核心成员** | 郁慕高（外交）、郁慕仙、郁慕剑、郁家子在元乌峰 |
| **崛起过程** | 从众家族中脱颖而出（郁家五杰只剩其一，ch135卢思嗣回忆）→ ch135宣布密林郡成立 |
| **统治方式** | 暴力展示（湖中洲屠杀 ch134、单手杀练气巅峰 ch135）+ 阳谋（坊市操控 ch136） |
| **对李家的态度** | 表面客气（郁慕高称呼「通崖前辈」），但实际上压迫——将卢家压得喘不过气，李家是下一个目标 |
| **战力** | 筑基（郁萧贵）+ 数名练气（九位练气围剿散修 ch135） |
| **潜在敌人** | 费家+李家可能形成反郁联盟（ch135眼神交流） |

### 2.4 紫烟门

| 维度 | 内容 |
|------|------|
| **定位** | 青池宗的同盟宗门 |
| **首登场** | ch150——灵岩子被长霄门追杀至李家 |
| **传承** | 紫气峰传承（灵岩子「紫气峰的传承，不要断在我手上」） |
| **法器特色** | 紫烟灵罩（淡紫色玉珠法器，撑三天三夜） |
| **与李家关系** | 潜在盟友（李通崖决定救灵岩子伏杀长霄门） |
| **战力** | 灵岩子练气六层 |

### 2.5 长霄门

| 维度 | 内容 |
|------|------|
| **定位** | 与紫烟门敌对的宗门 |
| **首登场** | ch150——玉云子率众追杀灵岩子 |
| **行为特征** | 嚣张、傲慢、贪婪——「得罪了我长霄门……谁知道哪天就被人灭了门扬长而去」 |
| **战力** | 玉云子练气七层+一名练气六层 |
| **前150章末状态** | 被李通崖设计诱入大榕树伏杀圈（结果未明） |

### 2.6 镗金门

| 维度 | 内容 |
|------|------|
| **定位** | 与青池宗在望月湖-蕈林原有默契/对峙（ch070） |
| **线索** | 少门主在南边「打秋风」（ch070老道内心独白） |
| **出场** | 仅被提及，未正面登场 |
| **前150章此维度数据极少** |  |

### 2.7 萧家（萧初庭）

| 维度 | 内容 |
|------|------|
| **定位** | 青池宗内部的特殊存在——已入紫府（ch110情报） |
| **萧初庭** | 明面是筑基巅峰→实际已入紫府；赐明神散给李通崖（ch050）、在青池宗内有谋划 |
| **萧元思** | 炼丹师，李尺泾的师兄，蛇蛟果炼丹者，情感上同情李尺泾 |
| **萧雍灵** | 在冠云峰别院提醒李玄锋注意刘长迭（ch125） |
| **与李家关系** | 友善（萧元思的桥梁作用、萧初庭的善意）——但未超越宗门体制 |
| **战力** | 萧初庭已入紫府 |

### 2.9 卢家

| 维度 | 内容 |
|------|------|
| **定位** | 华芊山家族，与李家平级的望月湖势力 |
| **领袖** | 卢思嗣（练气修为） |
| **关键事件** | ch110提议青乌矿三方合作；ch131被郁家欺压；ch135臣服于郁家 |
| **新生代** | 卢远陆（练气，正法，傲慢）、卢远平（已故，杂气） |
| **与李家关系** | 从平等竞争（ch110打脸）→ 合作（矿脉）→ 同为被压迫者 |
| **战力** | 练气级家族，无筑基 |

### 2.10 费家

| 维度 | 内容 |
|------|------|
| **定位** | 望月湖区的另一家族，有筑基战力 |
| **领袖** | 费家家主（锦衣风流，筑基？ch135被与李通崖对视） |
| **关键事件** | ch135密林郡宣布——费家家主与李通崖眼神交流「两人心中皆有了底」 |
| **与李家关系** | 潜在反郁联盟——双方都是郁家压迫下的筑基级势力 |
| **战力** | 至少练气巅峰以上 |

### 2.11 山越

| 维度 | 内容 |
|------|------|
| **定位** | 北麓蛮族，李家最大生存威胁（前期）→ 伽泥奚死后崩溃（ch105后） |
| **领袖** | 伽泥奚（已死，ch105自碎大道之基） |
| **核心战力** | 木焦蛮（练气四层巫觋，存活）、阿会剌（被俘） |
| **战力体系** | 山越杂气 < 正宗功法（ch095李通崖一压三）；巫觋咒术 = 箓气劣化版 |
| **前150章末状态** | 权力真空，军事威胁解除——但大厥庭被毁后可能有残余势力 |
| **注** | 巫觋秘术线的伏笔仍在（陆江仙决定研究巫觋术，ch095） |

### 2.12 大黎山妖族

| 维度 | 内容 |
|------|------|
| **定位** | 独立于人族修仙体系之外的势力，非敌对非盟友 |
| **已知成员** | 白榕狐（练气，合作）、狐狸精（练气，情报）、神秘狐（高阶，ch150）、筑基猪妖（威胁） |
| **与李家关系** | 情报交换（白榕狐提供于羽楔真相、雄鹿情报）→ 友好中立 |
| **战力** | 有紫府级以上狐妖（神秘狐暗示），但未介入人族争斗 |

## 三、势力博弈的核心矛盾

### 3.1 明面矛盾

| 矛盾 | 对立面 | 状态 |
|------|--------|------|
| 青池宗 vs 紫烟门 vs 长霄门 | 三宗七门内部博弈 | 前150章末升温 |
| 郁家 vs 诸家族 | 新势力的压迫 vs 旧家族的生存 | 持续压制中 |
| 李家 vs 山越 | 家族生存 vs 蛮族入侵 | 山越已崩，威胁解除 |

### 3.2 暗面矛盾（前150章末已建立）

| 矛盾 | 核心 | 状态 |
|------|------|------|
| 李家 vs 青池宗 | 李尺泾被炼丹→仇恨种子 | 潜伏期 |
| 陆江仙 vs 未知存在 | 法鉴真相、大衍天玄箓来源 | 待探索 |
| 迟尉 vs 大蛟 | 交易关系下的潜在博弈 | 伏笔 |

### 3.3 前150章势力格局的关键转折

1. **ch105**：伽泥奚死亡 → 山越威胁解除
2. **ch110**：青池宗紫府陨落 → 权力真空
3. **ch120**：李尺泾被炼丹 → 仇恨奠基
4. **ch135**：密林郡成立 → 新压迫者降临
5. **ch150**：长霄门追杀至李家 → 外部宗门压力直接入侵
