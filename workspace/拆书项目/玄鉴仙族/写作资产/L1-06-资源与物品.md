## 三、经济体系规则

### 3.1 资源匮乏的铁律

- 李家从凡人起步——修仙界最基本的**功法/丹药/灵石**都需要拼命获取
- 建家族之初仅靠灵稻+白元果维持（ch021-ch045）
- 后来有灵石收入但仍然紧巴巴——「五年积蓄」只有五枚灵石（ch050）

### 3.2 以物易物 → 灵石交易 → 人情交易

| 交易类型 | 例证 | 出处 |
|----------|------|------|
| 以物易物 | 遂元丹换功法（李尺泾操作） | ch076 |
| 灵石交易 | 坊市购买符笔/储物袋（60灵石以上？） | ch060 |
| 人情交易 | 刘长迭100灵石布阵 + 「今后往来望月湖皆可在我李家落脚」 | ch125 |
| 武力获取 | 李通崖截杀乌家女修→丹炉 | ch115 |
| 战利品 | 老道被反杀→25灵石+丹药+符箓+长剑 | ch075 |

### 3.3 家族资产不完全清单（前150章末）

| 资产类别 | 内容 | 价值评估 |
|----------|------|----------|
| 灵石 | 约数十枚（具体数量不明） | 中等 |
| 功法 | 《江河一气诀》《清灵元法》《采气诀》《玄水剑诀》等 | 核心资产 |
| 丹药 | 蛇元丹（部分已用）、玉芽丹 | 有限 |
| 法器 | 灰白长剑、金庚弓、储物袋、阵旗、丹炉 | 关键资产 |
| 灵田 | 若干（受山越劫掠影响） | 基础 |
| 矿脉 | 青乌矿（三分之一合作权） | 年收入几十灵石 |
| 法阵 | 日仪玄光阵（两峰覆盖） | 防御升级 |
| 特殊 | 法鉴、宝珠、青尺剑（托管） | 无价 |

## 四、经济体系的特点

1. **资源极度匮乏**：李家从零起步，每一块灵石都是拼出来的
2. **暴力是经济升级最快的手段**：战利品（ch075）+截杀（ch115）是两次最大的经济跃升
3. **技术短板制约**：有蛇蛟果但无炼丹传承（ch093）→ 有丹炉但还没发展出炼丹能力（ch115后）
4. **人情/关系也是经济资源**：萧初庭赐丹、萧元思炼丹、刘长迭优惠布阵——关系的价值不亚于灵石
5. **稳定收入源的建立**：青乌矿脉是李家第一个稳定年收入来源（年产几十灵石）
6. **郁家的经济战**：ch136郁家坊市阳谋——新格局下经济被压制的风险
