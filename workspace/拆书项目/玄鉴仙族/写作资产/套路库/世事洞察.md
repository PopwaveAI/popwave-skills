# 世事洞察

## 定义
成熟角色通过长期阅历，对年轻角色进行关于人情世故、权力场的点拨与指导。

## 特征
| 维度 | 描述 |
|------|------|
| 讲述者 | 老练的家族/宗门成员 |
| 接受者 | 天才但涉世未深的年轻人 |
| 内容 | 趋炎附势的本质、速速抽身不浪费时间、戒备有心人 |
| 形式 | 私下点拨、警醒之言、旁敲侧击 |
