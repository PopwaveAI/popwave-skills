# 套路：世家·吞并

## 定义
一个势力（家族/宗门）以武力或权势为后盾，要求另一方缴纳供奉、臣服听令，逐步蚕食对方的独立性。本质是**以实力差距为借口的资源掠夺**。

## 适用场景
- 新崛起的势力整合周边小家族
- 筑基家族对练气家族的压迫
- 宗门对散修/小族的系统性盘剥

## 结构模板
1. **势力出现**：新势力派遣使者/先锋拜访各地
2. **实力展示**：或明或暗地展示武力威慑
3. **要求供奉**：以"保护""整合"之名索要资源
4. **受迫方选择**：或屈服（当前）或反抗（后续）
5. **礼制包装**：用"资源整合""共建坊市"等话术美化掠夺

## 变体
| 变体名 | 说明 |
|--------|------|
| 温和吞并 | 以合作之名行吞并之实，给足面子（如郁家对李家） |
| 强硬吞并 | 赤裸裸的武力压迫，不给选择（如郁家对卢家） |
| 间接吞并 | 通过姻亲、债务、物资垄断等软性手段 |
| **"两个选择"吞并** | 吞并方给出"内附"与"离开"两个选型，表面仁慈，实则"离开"者在半路被截杀（如李家对卢家华芊山） |
| **"典例"型吞并** | 首次吞并为后世立规矩，既要体面（不落口实）又要彻底（不留后患），需要精心设计流程（如李家对卢家） |

## 案例
**ch131-ch132**：郁家派郁慕高到访望月湖周边诸家，要求"每五年按青池宗标准向郁家缴纳供奉"。对卢家态度傲慢，对李家因忌惮剑仙而客气——同一势力根据对象实力不同展现不同面貌。

**ch142-ch144**：李家对卢家华芊山的吞并——卢思嗣死后一夜之间完成。李通崖先杀卢远陆稳住局面，再给出"内附"与"礼送出境"两个选择。选择离开的卢家人（以胎息修士为首）被李玄锋、李玄岭在半路截杀殆尽。内附者则被监视安置，失去族名（"男丁入赘、女子改嫁、不得留有卢姓"）。全过程以"体面"为外壳（李通崖对卢安宇客气、惋惜），以斩草除根为内核——既为后世做典例，又不留后患。

## 注意
- 吞并过程是展示势力格局和人物立场的绝佳机会
- 对不同对象的不同态度能暴露势力首领的判断力与性格
- "两个选择"的陷阱模式尤其适合用于展示**吞并方的政治虚伪性**——表面仁慈与背后杀心的反差是叙事张力来源
- 对姻亲家族的吞并难度更高：过于绝情招致不信任，过于仁慈留下后患，需要精细平衡
