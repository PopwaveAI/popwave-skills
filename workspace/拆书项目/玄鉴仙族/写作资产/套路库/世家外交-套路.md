# 套路：外交拜访/世家外交

## 定义

新晋家族（李家）初次以独立身份拜访老牌世家（萧家）的标准化流程。涉及山门礼仪、主客寒暄、信息交换、求取资源等环节。

## 功能

- 建立/巩固上下级或平等关系的家族间联系
- 释放世界观信息（地区势力图、资源价格体系、修炼常识）
- 展示小家族在大家族面前的姿态与智慧
- 通过接待方态度折射大家族对该家族的定位

## 关键要素

| 要素 | 说明 |
|------|------|
| **山门礼仪** | 下马威/接待规格——是否禁飞、何人迎接、等候时长 |
| **主客寒暄** | 问候长辈、修为、家事——测试关系亲疏的试探环节 |
| **需求提出** | 客方向主方提出资源/信息需求（阵法、矿石、人脉等） |
| **信息反馈** | 主方给出回答，同时释放自己掌握的情报 |
| **赠礼** | 主方可能主动赠予书籍/资源，作为长期关系的投入 |
| **送客** | 以文化话术暗示结束（如"可以去看看郡中风物"） |
| **后场独白** | 主方视角的评论——揭示真实态度 |

## 适用章节

- ch116（拜访萧雍灵）：完整流程
- 后续李家更大规模外交可参照此结构升级

## 变体

- **平等外交**（本案例）：双方实力差距不大，以礼节和人情维系的平等交往
- **乞求外交**：弱势方求资源，姿态更低
- **威慑外交**：强势方展示武力，压迫对方
