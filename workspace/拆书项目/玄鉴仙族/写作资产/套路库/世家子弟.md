# 套路：世家子弟

## 核心公式

```
出身名门大族 → 见多识广/谦逊有礼（对同道）→ 成为信息源/人脉资源 → 提供世界观信息+未来可兑换的人情
```

## 典型特征

| 维度 | 特点 | 示例（萧雍灵） |
|------|------|-------------|
| 出身 | 世家嫡系、族中有筑基 | 萧家·萧初庭之孙 |
| 修为 | 中等（不是最强也不是最弱） | 承明轮 |
| 性格 | 对同道谦逊/对凡人疏离 | 见李通崖礼貌、对管事不理 |
| 知识 | 读过族史、知道秘辛 | 三宗十门、紫府大战 |
| 装备 | 有宝物（识别修为） | 能识别人修为的宝物 |
| 人脉 | 认识各大势力人物 | 族叔萧元思、知道宁婉 |

## 叙事功能

1. **信息水管**：通过其渊博知识自然输出世界观
2. **社会定位标尺**：其态度反映主角家族在修仙界的地位
3. **未来资源**：留有人情可后续兑换

## 注意事项

- 世家子弟的"友好"要有范围限定——对同道友好+对凡人不屑，这样更真实
- 知识面要合理——读族史所以知道三百年前大战，不是凭空而来
- 避免写成"移动百科全书"——信息要通过对话互动自然引出
