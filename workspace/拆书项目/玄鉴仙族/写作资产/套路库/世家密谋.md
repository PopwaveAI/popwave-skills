# 世家密谋

---

## 定义

修仙家族内部或家族之间的秘密会谈场景，通常以双线或多线对照呈现，展示各方势力的战略博弈和真实意图。

## 公式

```
公开场合（面子）→ 私下密会（里子）→ 双方/多方谋划 → 揭示真实意图 → 筹码/威胁/结盟
```

## 核心要素

| 要素 | 说明 |
|------|------|
| **私密空间** | 小洲、船舱、密室——远离公开视野 |
| **话术分层** | 表面话（面子）→ 试探话（里子）→ 利益话（底牌） |
| **信息不对称** | 密谋者知道的事，外人不知道 |
| **双线对切** | 甲乙双方的密谋同时呈现，让读者看到博弈全貌 |

## 玄鉴仙族实例

**ch137 双线密谋**（经典实例 #1）
- A线：费望白 × 李通崖（小洲密会）
  - 试探→敞亮→达成默契（费李联盟隐性形成）
  - 信息交换：古黎道盗贼、郁家实力
  - 结果：双方心照不宣，各自得益
- B线：郁萧贵 × 郁慕高（船舱密谈）
  - 凌驾 vs 商议（父子间的权力让渡）
  - 核心决策：温水煮蛙对付李家
  - 结果：制定断代策略，复制万华芊案

**ch136 湖上宴会**（变形）
- 虽是公开场景，但本质是"半密谋"——郁家在公开场合展示力量，逼迫诸家表态
- 李通崖与费望白在人群中眼神交流——密谋的雏形

## 技法要点

1. **双线对切**：A线密谋 → B线密谋，让两方的谋划互相映照
2. **话术设计**：密谋中的对话不该过分直白——要有一点"你知我知"的默契
3. **父子/兄弟密谋**：家族内部的密谋往往带有"传承"意味（老一代教新一代）
4. **盟约不说破**：有时最好的盟约是不签盟约——心有灵犀胜过一纸文书（费李关系）
