# 世家晋升

## 定义
修仙家族通过族中杰出人物的成就（修为突破、名声传播、宗门认可）获得社会等级跃升。

## 特征
| 维度 | 描述 |
|------|------|
| 等级体系 | 胎息小族 → 大族 → 世家（需筑基） |
| 驱动因素 | 战力、宗门地位、名望 |
| 实际收益 | 灵物种子分配、供奉额度、社会尊重 |
| 象征意义 | 家族从边缘进入主流视野 |
