# 套路：世界揭秘

## 核心公式

```
新角色入队/偶遇 → 闲聊/问答 → 自然引出世界观重大设定 → 读者获得认知升级
```

## 用途

通过角色对话自然揭露世界背景设定，避免信息灌入式的"旁白说明"。让读者像主角一样"发现"世界的秘密。

## 信息投放技巧

1. **分层投放**：先浅后深
   - 浅层：当前人物/地区的信息（宁婉是谁→月湖峰）
   - 中层：区域历史（万家先祖→阵法大师）
   - 深层：世界格局（三宗十门→紫府大战）

2. **问题驱动**：让主角（代表读者）主动提问
   - "不知这场大战……"（ch048 李通崖追问）
   - "那松林朔风乃何物？"（ch048 继续追问）

3. **信息提供者人设**：信息源要合理
   - 萧雍灵：世家嫡系 + 读过族史 + 爱说话 → 合理的"信息水管"

## 信息量控制

| 级别 | 信息密度 | 处理方式 | 示例 |
|------|---------|---------|------|
| ★★★ | 核心世界观 | 放慢节奏，给读者消化时间 | 紫府修士大战、月华元府 |
| ★★ | 区域知识 | 穿插在对话中 | 松林朔风、小清灵气 |
| ★ | 风土人情 | 一笔带过 | 宁婉面纱、霞光云船 |

## 注意事项

- 不要让角色像百科一样背书——信息应因对话需要而出现
- 信息应该可被后续情节验证/利用（松林朔风→未来练气需要）
- 真相与传言之间可以有差异（为后续反转留空间）

## 适用场景

- 世界观扩展的中期阶段
- 主线需要补充背景信息时
- 新角色需要展示其知识渊博时
