# 套路：为家求法

## 核心公式

```
发现家族功法短板（缺少系统性传承/没有练气功法） → 借着宗门/外界渠道开口 → 谈判换取功法 → 因地制宜选择 → 家族传承升级
```

## 关键阶段

1. **认知觉醒**：家族弟子意识到家族缺少功法传承
   - "功法尤为珍贵"——不是在坊市随便能买到的
   - 功法是家族传承的核心——没有功法就永远是小族
   - 破族灭门时功法会被销毁——获取渠道极其有限

2. **关系借用**：利用自身在宗门/外界的关系开口
   - 李尺泾利用与萧元思的师兄弟关系（ch076）
   - "下次前往冠云峰，将老道的这些物什卖了"（李通崖通过坊市购买）

3. **资源交换**：功法不会白给——需要等价交换
   - 功法换遂元丹（李尺泾路线）
   - 灵石/灵物购买（坊市路线）

4. **因地制宜**：选功法时考虑家族领地特征
   - 黎泾山在望月湖旁 → 选湖中金秋相关功法
   - 芦苇河网众多 → 选江中清气相关功法
   - 地块特征 → 功法选择 → 最大修炼效率

5. **家族传承落地**：功法带回后需要在家族内传承
   - 谁适合练什么功法
   - 功法在家族中的管理机制

## 功法获取渠道

| 渠道 | 难度 | 费用 | 风险 | 示例 |
|------|------|------|------|------|
| 宗门关系 | 需要人脉 | 交换代价 | 低（正法可靠） | 李尺泾×萧元思 |
| 坊市购买 | 需要灵石 | 高 | 中（怕假货/古法） | 李通崖去冠云峰 |
| 战利品 | 需要击杀 | 随机 | 高（可能不匹配） | 反杀老道所得 |
| 家族血脉传承 | 需要祖先 | 无 | 低 | 各家祖传功法 |

## 正法vs古法vs秘法体系

| 类型 | 特点 | 选择偏好 |
|------|------|---------|
| 正法 | 经过宗门验证，修行稳妥 | 首选（风险最低） |
| 古法 | 古老传承，可能有缺陷但也可能有隐藏好处 | 次选 |
| 秘法 | 不公开的特殊传承 | 特定情况使用 |

## 适用场景

- 家族从"胎息初期"向"练气/系统性修仙"阶段过渡
- 家族在外有子弟在宗门修炼
- 家族获得大量灵石/资源后需要转化为传承资本

## 注意事项

- 功法是仙族小说的核心资产——获取过程要有"艰辛/谈判/交换"的叙事重量
- 选功法时"地理匹配"是《玄鉴仙族》的特色要求——不要忽略
- 正法价格/价值高于古法——因为这代表"确定性"
- 功法获取后要考虑传承管理——谁练什么、如何保管
