# 套路库：人丹阴谋

> 归类：宗门黑幕 / 栽培式牺牲

---

## 定义

上宗以培养弟子为名，实际上将特定弟子/家族作为"药材"培养，在达到成熟期后献祭给更高层存在（妖王/魔修/突破瓶颈），换取资源或政治交易的黑暗套路。

---

## 本卷实例（ch087）

| 项目 | 内容 |
|------|------|
| 执行方 | 青池宗 |
| 目标 | 南疆妖王（突破所需） |
| 药引A | 修炼太阴月华者（李尺泾→《青烟照月诀》） |
| 药引B | 特殊体质者（邓求之→邓家代代修炼秘法为补品） |
| 操作机制 | ①送到倚山城轮防 → ②兽潮喂养消耗 → ③收网送入口 |
| 历史先例 | 二十五年前倚山城破→筑基修士于羽楔死（练太阴月华）|
| 辅助手段 | 宗门发放的灵药灵丹是"用药深远的配子" |
| 受害方知情状态 | 李尺泾看穿五年，邓求之几百年装疯卖傻 |

## 写作要点

1. **必须留历史痕迹**：于羽楔=二十五年前的"李尺泾"——说明这不是第一次
2. **受害者不能太蠢**：李尺泾早早看穿，沉默五年才有揭底质感
3. **辅助手段合理化**：兽潮"恰到好处"的退去节奏——看似帮忙实为磨刀
4. **双受害结构**：两条不同路径（功法型/体质型）汇于一个结局——叙事效率翻倍
5. **外部世界存在参照**：蕈林原供奉木像的案例——暗示这不是孤立现象
