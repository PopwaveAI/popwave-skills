# 套路：仙凡之别

## 定义

描写修仙者与凡人之间社会地位、权力关系、生存条件之间的巨大鸿沟。在《玄鉴仙族》中，这种鸿沟是故事世界的重要底色。

## 功能

- 建立世界观的残酷真实感
- 为李家的"朴素家风"提供对比参照
- 推动角色成长（如李玄锋第一次直观感受仙凡差距）
- 为后续"凡人线"埋下情感基础

## 关键要素

| 要素 | 说明 |
|------|------|
| **跪拜仪式** | 凡人见修仙者的标准反应——"战战兢兢地磕头，也不敢说话" |
| **无意之害** | 修仙者日常动作（飞行、降落）对凡人生活造成的物理破坏 |
| **秩序崩溃** | 修仙者出现→凡人正常秩序瞬间崩塌（商队溃散、儿童啼哭） |
| **角色震惊** | 来自相对平等背景（李家）的角色对此感到不适/迷茫 |
| **制度性压迫** | 驻扎数百年的世家建立了一整套尊卑体系 |

## 适用章节

- ch117（江渔女）：城门风波 + 醉春楼沦落
- 可作为全书反复出现的背景图层

## 变体

- **近距离接触**（本案例）：修仙者进入凡人生活区
- **制度性压迫**：世家治下凡人的世代奴役
- **凡人反抗/挣扎**：极少数情况，通常以悲剧收场
