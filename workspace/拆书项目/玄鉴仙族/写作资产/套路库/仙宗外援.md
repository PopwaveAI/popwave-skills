# 套路：仙宗外援

## 核心公式

```
家族子弟在仙宗修炼 → 偶遇负责家族区域的上使/同门 → 托付物品/信息/人情 → 上使到达家族区域时给予关照
```

## 用途

修仙家族体系中的"外援人脉"套路。家族与仙宗之间通过子弟的个人关系搭桥，获得信息、资源或保护层面的额外关照。

## 关键节点

1. **搭桥**：家族子弟在仙宗与目标人物建立联系（ch046 李尺泾遇宁婉）
2. **托付**：托付灵石/丹药/信件等物（ch046 五枚灵石）
3. **送达**：上使到达家族区域，当众/私下交付（ch049-050 宁婉递锦囊）
4. **回响**：家族因此受关注→获得额外机会（ch050 萧初庭因此赏识李通崖）

## 效果

- **对内**：读者感受到"家族培养子弟→子弟反哺"的闭环快感
- **对外**：其他家族意识到该家族有仙宗背景，不敢轻视
- **对角色**：上使形象软化（从公事公办到有私情）

## 注意事项

- 托付物不宜过重（以心意为主，否则上使行为不合理）
- 上使的关照程度要在合理范围内（不能因私费公）
- 最好与上使本身的性格呼应（宁婉本就性格温婉）

## 适用场景

- 家族初期缺乏外部声望，需要仙宗背书
- 想展示家族子弟虽是仙宗新人但已积累人脉
- 需要让某个强者对主角家族"凭空"产生好感
