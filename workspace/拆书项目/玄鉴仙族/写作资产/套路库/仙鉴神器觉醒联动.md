# 套路：仙鉴/神器觉醒联动

## 适用范围
主角使用的神器（法鉴）发生重大变化/使用时，引发同源其他碎片/神器的反应。

## 经典范例
- ch085：陆江仙凝聚灰箓→老道玉佩吸引→金羽宗镜片暴动

## 结构要素
1. **操作端**：主角使用神器（献祭/敕封）
2. **感应端**：同源碎片/关联法器产生共鸣
3. **风险暴露**：操作过程可能暴露位置
4. **收敛规避**：主角立即中断操作、收敛气息
5. **追索方反应**：第三方势力发现异常但未能定位

## 写作要点
- 感应要从"冥冥之中"写起——模糊、直觉式的感知
- 暴露风险要具体（老道的玉佩是标记物）
- 回避动作要有技术细节（收敛气息、断联系、收回神识）
- 第三方视角补充（金羽宗三紫府锁镜片）
- 关联但不挂钩——保持悬疑
