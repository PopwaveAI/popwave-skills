# 以退为进

---

## 定义

在冲突/谈判/博弈中，表面上选择退让、低调、不争，实际是为了在更大的格局上获得优势。退一步不是认输，而是为进两步蓄力。

## 公式

```
面临压力/挑衅 → 表面的退让/拒绝/不回应 → 观察局势发展 → 在更重要的节点上展现力量 → 获得比正面硬刚更大的收益
```

## 核心要素

| 要素 | 说明 |
|------|------|
| **表面姿态** | 谦让、低调、不接招、不出头 |
| **内在主动权** | 退让的一方仍然掌控着自己的节奏 |
| **时机选择** | 该退时退，该进时进（判断力是核心） |
| **收益最大化** | 保存实力，让对手消耗，最终得利 |

## 玄鉴仙族实例

**实例 #1：李通崖不接受挑战（ch136）**
- 退：面对郁慕剑的剑术挑战，李通崖谢绝"不知兄台哪里听来的谣言"
- 再逼：郁萧贵开口"给个面子"→ 李通崖才应战
- 进：一招月阙剑意震慑全场，收剑教诲
- 结果：既展现了剑仙传承的实力，又不失风度，还挣得尊敬

**实例 #2：李家不参与坊市生事（ch140）**
- 退：诸家都在趁火打劫时，李家"勒令弟子不得外出"
- 进：坐看郁家吃亏，保存实力，积累声望
- 结果：郁家无法抓住李家把柄，李家的风险降至最低

**实例 #3：李通崖推脱助阵邀请（ch137）**
- 退：费望白暗示李通崖参与对付郁家 → 李通崖用"一身剑术太显眼"推脱
- 实际：暗示费望白"支持"但不公开站队
- 进：避免了被卷入直接冲突，既不得罪郁家，又维系了费家关系

## 与其他套路的关系

| 套路 | 交叉方式 |
|------|---------|
| 藏锋纳锐 | 以退为进的"退"就是藏锋的实践表达 |
| 阳谋制衡 | 退让之后坐观对手被阳谋消耗，是收益 |
| 世家密谋 | 密谋的本质就是以退为进的集体版本 |

## 叙事功能

1. **延缓冲突**：不急于爆发，给剧情留出更多发展空间
2. **塑造人物智慧**：退让需要更大的勇气和判断力，是智者的标志
3. **制造爽感延迟**：读者知道主角有实力，看他隐忍→等待→终于出手，积累期待

## 变形

- **以退为退**：有时退就是退，不是计策，是真的不想卷入（李通崖的生存哲学）
- **以退为责**：家主选择退让，也是为了保护家族（"切莫节外生枝"）
