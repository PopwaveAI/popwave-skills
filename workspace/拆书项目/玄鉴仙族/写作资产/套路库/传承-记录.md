# 套路：传承·记录

## 定义
家族/势力将核心功法、战斗心得、修行经验整理成册，供后辈学习。核心是**知识的物化与延续**——即使用户逝去，留下的文字可为家族提供长期的战力保障。

## 适用场景
- 角色掌握独门技艺，考虑家族传承
- 角色即将远行/闭关/可能牺牲
- 家族功法库的充实
- 长辈对后辈才能的认可与培养

## 结构模板
1. **才能展示**：角色通过战斗/实务展现某种过人天赋
2. **认可提议**：长辈/领导提出"写成书册留档"
3. **讨论门槛**：角色指出传承的难度/限制条件（如需要灵窍）
4. **决定执行**：虽然有限制，仍决定留下底子
5. **意义升华**：这对家族的长期意义

## 变体
| 变体名 | 说明 |
|--------|------|
| 功法传承 | 完整修炼体系记录（如《玄水剑诀》） |
| 心得传承 | 个人战斗经验汇总（如李玄锋弓法心得） |
| 精神传承 | 家族历史/训诫记录 |
| 开山立派 | 从零创建新传承体系 |

## 案例
**ch133**：李通崖见李玄锋箭法犀利，提议"将心得整理整理，撰写成册，好叫今后族中的后辈也能学一学"。李玄锋表示弓法依赖灵窍和罡气，没有灵窍的人练了威力十不存一，但仍答应"给家中留一份底子"。

## 注意
- "有门槛的传承"比"人人都能练"更有真实感——体现修仙界的天赋分化
- 传承的内容应与角色性格/修行路线一致
