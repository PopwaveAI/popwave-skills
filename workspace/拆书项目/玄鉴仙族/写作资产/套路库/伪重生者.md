# 伪重生者

## 定义
角色以为自己拥有前世记忆、重生回到了过去，但实际上这些"记忆"并非真正的时间穿越，而是外力推演制造的虚假记忆。

## 特征
| 维度 | 描述 |
|------|------|
| 自我认知 | "我是重生者，知道未来" |
| 实际真相 | 大衍天玄箓的推演记忆植入 |
| 记忆内容 | 大灾大蝗、仙族诛灭、魔灾肆虐等合理剧情 |
| 破绽 | 对当事人（李家）知道得太多太细 |
| 情感卷入 | 以为自己在"投资"熟悉的家族，其实只是信息差 |
