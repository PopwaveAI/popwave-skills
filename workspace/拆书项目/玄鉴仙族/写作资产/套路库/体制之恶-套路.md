# 套路：上界黑暗/体制之恶

## 定义

修仙世界顶层的制度性/结构性暴力——宗门/大势力将其成员的个体价值彻底工具化，以"大局"为名掩盖对天才的掠夺与消耗。

## 功能

- 将个人悲剧（李尺泾之死）提升为制度批判
- 建立"外部压力源"——李家未来反抗的正当性基础
- 解释"为何世家都要小心翼翼"（萧家/卢家的行为逻辑）
- 塑造复杂反派——迟尉不是传统意义上邪恶，而是"正常"的规则执行者

## 关键要素

| 要素 | 说明 |
|------|------|
| **制度性冷血** | 不是个别恶人的行为，而是整个体系的运行法则 |
| **执行者的良知** | 元素真人："一个个送掉宗内的天才，为他一人铺垫道途"——明知不公仍执行 |
| **顶层受益者** | 迟尉——闭关维持生机，他人生命为其续命 |
| **底层工具** | 大蛟——非人存在，无善恶判断，只做交易 |
| **被牺牲者的认知** | 李尺泾：到死前才"识破青池宗真面目" |
| **幸存者的仇恨** | 萧元思的愤怒——"十多年不曾动怒过的他神色一恼" |

## 适用章节

- ch119-ch120（倚山城线）：紫府真人揭晓真相、李尺泾被投入丹炉
- ch116（萧家老祖的隐忍策略）：作为体制阴暗面的应对案例
- ch118（陵峪门灭门）：同一个体制下的不同牺牲品

## 变体

- **献祭式**（本案例）：定期献祭天才给大蛟炼丹
- **清洗式**：大规模清洗门派内的不安定因素
- **资源垄断式**：控制关键资源，确保下位者无法翻身
- **附庸式**：通过联姻/人质控制下属家族

## 叙事技巧

在读者被李尺泾的剑意震撼、爱上这个角色之后，再用体制之恶将其毁灭——先建立情感投资再收割情绪回报。
