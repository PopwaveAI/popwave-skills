# 套路：供奉仪式

## 核心公式

```
筹备 → 同行 → 抵达供奉地 → 见大佬/各家齐聚 → 仙宗上使降临 → 缴纳供奉 → 社交/交易 → 额外奖励
```

## 用途

用于修仙家族社会中定期向仙宗上缴供奉的集体活动。既是经济行为，也是社交场合——各家族借此碰面、展示实力、攀谈交际。

## 变种

| 变种 | 差异 | 示例 |
|------|------|------|
| 大佬坐镇 | 供奉地有筑基长辈压场 | ch049 萧初庭坐镇冠云峰 |
| 上使带礼 | 上使与某家族有私交，增加关照 | ch049-050 宁婉—李家 |
| 供奉后坊市 | 缴纳后附带交易市场 | ch050 冠云坊市 |
| 压制羞辱 | 其他家族借机炫耀/打压 | ch049 卢平远羞辱万元凯 |

## 情绪曲线

```
期待(上路) → 敬畏(见大佬) → 震撼(上使登场) → 紧张(缴供) → 满足/压抑(社交结果)
```

## 角色功能

- **主办方（萧家）**：提供场地+秩序担保
- **上使（宁婉）**：代表仙宗权威+可附加私交线
- **各家代表**：展示+攀谈+交易
- **被关注者（李家）**：因各种原因成为焦点

## 适用场景

- 家族需要获得修炼资源时（通过供奉建立稳定关系）
- 需要展示世界格局——各家实力排列
- 需要让主角家族进入公开社交圈
