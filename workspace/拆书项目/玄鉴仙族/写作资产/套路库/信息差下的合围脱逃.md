# 套路库：信息差下的合围脱逃

> 归类：情节驱动 / 紧迫逃生

---

## 定义

当强敌以碾压级兵力合围家族核心据点时，家族成员依靠信息优势（法器/内应/灵识）提前警觉，在合围完成前一刻突破封锁、藏匿或撤离。关键在于"时间差"和"信息差"——不是硬碰硬，而是在夹缝中利用对手的认知盲区。

---

## 本卷实例（ch089-ch090 — 黎泾山逃生）

| 项目 | 内容 |
|------|------|
| 合围方 | 伽泥奚山越军（练气中期，2000+精兵） |
| 被围方 | 李玄岭（胎息二层）+ 李景恬（凡人）+ 山上女眷 |
| 信息源 | 法鉴（玄明妙法鉴）→ 陆江仙托梦 → 李玄岭获得高空视角 |
| 合围战术 | 一路正面进镇 / 一路绕后山封锁 → 防止逃往泾阳村 |
| 逃脱手段 | ①神行术抱人下山 → ②法鉴实时导航规避包抄 → ③换装翻墙入旧院 |
| 合围方盲区 | 伽泥奚不知李家有法鉴这种外挂级信息工具 |
| 胜负结论 | 李家核心血脉逃脱，普通村民儿童被掳 |

## 套路结构

```
┌─────────────┐
│ 预警触发     │ ← 陆江仙神识扫描 → 托梦 → 李玄岭获情报
└──────┬──────┘
       ▼
┌─────────────┐
│ 合围执行     │ ← 山越军兵分两路，悄声包抄
└──────┬──────┘
       ▼
┌─────────────┐
│ 时间差       │ ← 伽泥奚押陈三水行军 → 拖出三个时辰误差
└──────┬──────┘
       ▼
┌─────────────┐
│ 逃脱路线     │ ← 法鉴查包抄→改道→下山→换装→藏匿
└──────┬──────┘
       ▼
┌─────────────┐
│ 扑空         │ ← 伽泥奚上山时只剩"有温度的被褥"
└──────┬──────┘
       ▼
┌─────────────┐
│ 二次搜捕     │ ← 伽泥奚下令搜捕"面容姣好者"
└──────┬──────┘
       ▼
┌─────────────┐
│ 余波         │ ← 等待伽泥奚撤走的"一天一夜"
└─────────────┘
```

---

## 写作要点

1. **逃脱不能太轻松**：要付出代价——陈三水的牺牲换来了时间差
2. **信息源必须有局限**：法鉴只能在黎泾镇范围，李玄岭胎息二层修为低
3. **追捕方不能太笨**：伽泥奚发现"有温度的被褥"→推理"娃子敏锐"→立即二次搜捕
4. **逃脱后的危机不退场**：虽然逃了但镇门被封，躲藏的紧张感持续
5. **信息差是核心悬念工具**：读者知道法鉴存在，但伽泥奚不知道——双重视角张力
