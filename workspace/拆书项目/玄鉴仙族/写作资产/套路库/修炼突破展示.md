# 套路：修炼突破展示

## 核心公式

```
突破境界 → 测试新能力 → 直观展示效果 → 内心自省 → 对比预期
```

## 关键节点

1. **突破宣告**：角色或旁白告知境界突破
2. **能力验证**：使用新能力做测试（最佳方式：有直观效果）
3. **效果呈现**：让读者"看到"突破带来的变化
4. **自省**：角色比较实际效果与预期/功法记载

## 效果呈现技巧

| 方式 | 示例（ch046） | 效果 |
|------|-------------|------|
| 一剑断树 | "那棵大榕树顿时轰然倒地" | 视觉冲击力 |
| 与功法对比 | "和那《青元养轮法》中描述的玉京轮修为相近了" | 暗示功法不凡 |
| 技能命名 | "玄水剑芒" | 给能力命名=建立记忆点 |

## 功能

- **对内**：确认修炼体系的有效性、让角色有成就感
- **对读者**：看得见的成长——爽感的来源
- **对叙事**：为后续冲突提供武力基础

## 注意事项

- 展示要有新意——同样境界突破如果每次都"一剑断树"就腻了
- 最好与功法/家族特色绑定
- 突破后适当体现"与功法描述的差异"为后续设伏
