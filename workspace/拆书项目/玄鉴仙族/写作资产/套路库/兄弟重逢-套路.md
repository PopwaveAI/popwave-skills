# 套路：兄弟重逢

## 核心公式

```
生死未卜（一方失踪/参战） → 归来的狼狈与惊喜 → 互损/吐槽的情感表达 → 温情（含蓄的关怀）
```

## 关键阶段

1. **悬念积累**：一方在冒险/参战中生死未卜
   - 李项平被练气追杀死活不知（ch071-074）
   - 李通崖带着李玄锋回家却发现项平未归
2. **归来的姿态**：归来时一身狼狈，但带着笑容
   - "李项平一身狼狈，笑盈盈地站在门前"
3. **第一句话**：兄弟间的第一句话——含蓄的情感表达
   - "就知道你小子狡猾，准死不了"（李通崖）
   - 不是"你没事吧"——而是带着调侃的放心
4. **吐槽/诉苦**：用吐槽化解危险经历的紧张感
   - "这叫什么事！被一练气修士撵了一天两夜，差点被人一掌打死"（李项平）
5. **回到正事**：马上转入战利品清点等事务性对话
   - 兄弟之间不需要过多情感表达，行动代替话语

## 情感表达特点

| 表达方式 | 示例 | 说明 |
|---------|------|------|
| 反向表达 | "准死不了"——正话反说 | 不直接说"我很担心你" |
| 吐槽代替诉苦 | "这叫什么事……" | 把恐惧转化成喜剧 |
| 行动代替话语 | 坐下喝茶、清点物品 | 正常化——你回来了，该干嘛干嘛 |
| 肢体语言 | 笑盈盈/嘴角微翘 | 克制但可察觉的喜悦 |

## 情绪曲线

```
隐忧（悬念期） → 喜悦（看到对方活着） → 放松(调侃) → 日常化(回到正事)
```

## 兄弟关系的类型模板

| 类型 | 互动模式 | 示例 |
|------|---------|------|
| 吐槽型 | 互损中表达关心 | 李项平x李通崖 |
| 沉默型 | 不说话但默契 | 某些兄弟组合 |
| 保护型 | 一人挺身在前 | — |

## 适用场景

- 冒险线结束后一方归来
- 长距离分离后的重逢
- 大战/劫难后的家庭团聚

## 注意事项

- 兄弟情感表达要含蓄——直接说"我想死你了"不符合《玄鉴仙族》的男性叙事风格
- 吐槽/调侃是最好的情感载体——"准死不了"比"你没事太好了"更有力量
- 重逢后不要拖太久情感戏——马上转入下一件事（战利品/局势讨论）
