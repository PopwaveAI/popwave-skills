# 套路库：决心突破/资源传承

> 归类：情节驱动 / 人物成长

---

## 定义

家族年轻一辈的武力担当因外部危机（入侵/被俘/战败）而产生强烈的自己太弱的自责，在愤怒/不甘的情绪驱动下，宣布闭关突破。家族核心管理者评估后，将家族仅存的稀缺修炼资源（最后一枚丹药、最后一株灵草）交给这位突破者，完成一次"资源向战斗力倾斜"的战略分配。突破者带着家族全体的期望进入闭关——读者的期待也随之转向"出关后会变多强"。关键词：**愤怒→决心→资源倾斜→闭关悬念**。

---

## 本卷实例（ch093 — 李玄锋拿蛇元丹突破）

| 项目 | 内容 |
|------|------|
| 角色 | 李玄锋——胎息修为，长弓手，性格冲动/杀性重 |
| 触发事件 | 父亲李项平带兵在山越境内失踪+家族沦陷的无力感 |
| 自责表达 | "还是实力不够，若是我有炼气修为……左右不过是一箭的事！" |
| 决心宣言 | "我这就去闭关突破胎息五层玉京轮！"——想就去做的冲动性格 |
| 资源来源 | 蛇元丹——萧前辈留下的三枚之一（两枚已被长辈突破使用，最后一枚） |
| 资源背景 | 另有三枚蛇蛟果但无炼丹传承——暴露家族技术短板的窘境 |
| 交付仪式 | 李玄宣取出玉瓶给李玄锋——信任+期望的传递 |
| 后续去向 | 前往眉尺山洞府闭关 |
| 平行价值讨论 | 李玄锋"杀散修抢传承" vs 李玄宣"积蓄灵石买"——路线分歧 |

## 套路结构

```
┌─────────────┐
│ 危机刺激     │ ← 家族被入侵/亲人被俘→自己无力改变
└──────┬──────┘
       ▼
┌─────────────┐
│ 愤怒自责     │ ← "还是实力不够"→对自身修为的强烈不满
└──────┬──────┘
       ▼
┌─────────────┐
│ 决心宣布     │ ← "我这就去闭关突破XX层"——冲动但真诚的宣言
└──────┬──────┘
       ▼
┌─────────────┐
│ 资源分配     │ ← 管理者评估后，将最后一枚丹药交给突破者
└──────┬──────┘
       ▼
┌─────────────┐
│ 信任传递     │ ← 交付时的动作/对话——"这是家族的希望"
└──────┬──────┘
       ▼
┌─────────────┐
│ 进入闭关     │ ← 前往洞府/密室——读者期待提升
└──────┬──────┘
       ▼
┌─────────────┐
│ 出关悬念     │ ← 能否突破成功？突破后战力提升多少？——悬置答案
└─────────────┘
```

---

## 写作要点

1. **愤怒必须有因**：李玄锋的愤怒不是性格缺陷——父亲失踪+自己无力营救是合理的刺激源
2. **资源要显得"珍贵"**：最后一枚蛇元丹+三枚蛇蛟果无法使用——让资源的稀缺感具体化
3. **交付场景要有仪式感**：李玄宣"从怀中取出一个玉瓶"——动作本身就暗示了珍重
4. **闭关不是退场**：李玄锋去闭关不等于这条线结束——读者的期待感被转移到"他出关时会怎样"
5. **价值观冲突作为调味**：李玄锋"抢" vs 李玄宣"买"的争论暗示两人成长路径的不同
6. **联动家族技术短板**：蛇蛟果三枚但无炼丹传承——让"资源传承"的同时展示"还有更紧迫的资源瓶颈"

---

## 注意陷阱

| 陷阱 | 说明 |
|------|------|
| 突破太容易 | 如果每次愤怒→闭关→必然成功，突破失去含金量。要有突破失败的可能性 |
| 资源太充足 | 如果家族不缺资源，突破就只是时间问题——没有"最后一枚"的稀缺感 |
| 闭关时间太短 | 如果下一章就出关成功，读者觉得突破不值钱——需要合理的闭关时长 |
| 冲动变成莽撞 | 李玄锋的冲动要有理智支撑——他知道自己到了突破的边缘，不是凭空喊闭关 |
