# 剑中传承

---

## 定义

看似普通的武器/器物中暗藏高阶功法或传承，需要特定条件才能激活。激活方式与功法本体相关（如需要同源真元），形成"传承筛选器"。

## 公式

```
获得器物（看似普通）→ 察觉异样（熟悉感/特殊标记）→ 尝试激活（注入法力/特定手法）→ 触发传承→ 获得功法和/或背景信息
```

## 核心要素

| 要素 | 说明 |
|------|------|
| **伪装层** | 器物外表普通，材质无奇 |
| **触发机制** | 需特定功法/境界才能激活（形成门槛） |
| **揭示层** | 传递的不止功法，还有传承者的信息（姓名、顿悟场景、留言） |
| **配套性** | 新功法和已有功法形成配套体系 |

## 玄鉴仙族实例

**ch136-ch138 郁慕剑之剑**（经典实例 #1）
- 伪装：练气级别法器，"材质平平无奇"，刻"峪"字
- 触发：需《江河一气诀》真元灌注
- 门槛：五成真元（远超练气巅峰）才能彻底点亮——本是为筛选筑基修士的
- 揭示：三品步法《越河湍流步》+ 附篇《血遁术》
- 传承者：萧衔忧，在大陵江上顿悟
- 配套：与《江河一气诀》《江河大陵经》同源

## 变形

- **玉简传承**（ch138提及）：未解密的《江河大陵经》玉简，也是同体系的传承载体
- **水光投影法**：剑身飞跃，在水光中投射文字——**以水为媒的传承形式**与《江河一气诀》主题呼应

## 用法建议

1. 传承的获取最好有跨度（跨章节），增加获得感
2. 触发门槛让传承更有分量（不是随便什么人就能得到）
3. 传承内容不限于功法——可附带人物背景、世界观信息
4. 赠与式的传承（既得者留信给后人）比打怪掉落更有温度
