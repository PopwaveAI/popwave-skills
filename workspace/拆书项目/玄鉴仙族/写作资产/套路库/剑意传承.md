# 剑意传承

## 定义
修士的剑道修为/经验凝为实体（剑形），留待有缘人继承。

## 特征
| 维度 | 描述 |
|------|------|
| 凝形条件 | 领悟剑意的剑修在死后/远走前可凝形 |
| 物质形态 | 通体淡青色的宝剑 |
| 内涵 | 毕生剑道经验与感悟 |
| 放置方式 | 插于巨石之上（公开等有缘） |
| 寓意 | "剑在人在，传承不绝" |
