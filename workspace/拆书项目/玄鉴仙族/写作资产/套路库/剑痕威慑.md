# 剑痕威慑

## 定义
通过一道无法愈合的伤痕，侧面烘托施剑者的恐怖战力，即使其人已死/不在此地，余威仍在。

## 特征
| 维度 | 描述 |
|------|------|
| 手法 | 不写剑客，只写伤痕 |
| 效果 | 伤者修为远高于被伤时——随时间推移更显恐怖 |
| 情感 | 恐惧 + 后怕 + 隐晦的敬意 |
