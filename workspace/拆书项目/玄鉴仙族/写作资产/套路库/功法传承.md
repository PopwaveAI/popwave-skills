# 套路：功法传承

## 定义
修士在突破关键节点时，向家族/师门求取功法口诀的完整流程，展示知识传递的规则与伦理。

## 本套路适用场景
- 角色达到修为瓶颈，准备突破
- 需要展示家族/门派的功法管理制度
- 新人加入核心体系的仪式性时刻

## 典型案例（ch149 - 陈冬河求法）
**情境**：陈冬河胎息巅峰，前来求取练气法诀。

**七步传承流程**：
```
第一步：求法者就位
  "晚辈陈冬河……前来求法！"（正式行礼）

第二步：家族评估
  李通崖询问年龄（二十八虚岁）
  → 对标"郁慕锋三十岁练气"（天赋比肩）

第三步：功法展示
  目前李家可用三品练气诀：
  ①《江河一气诀》
  ②《洞泉澈灵诀》
  （优缺点+采气之所详细介绍）

第四步：授法者决定
  李通崖决定授予《江河一气诀》

第五步：资源配套
  先提供"江中清气"用于突破

第六步：偿还义务
  练气后去江上采一份清气还回家族

第七步：灵誓锁定
  发玄景灵誓 → 阅读功法 → 传承完成
```

## 家族传承伦理

| 原则 | 含义 |
|------|------|
| 不藏私 | 李家对核心成员不隐瞒功法资源 |
| 有偿传承 | 先借后还（清气偿还模式） |
| 关系优先 | 从小看着长大+与李项平如父子 |
| 对标引导 | 用外部天才（郁慕锋）作为评估标尺 |

## 功法资源分类
```
功法（知识）
  └── 三品练气诀：《江河一气诀》《洞泉澈灵诀》

采气之所（外景资源）
  └── 各功法对应特定自然环境+清气

气体资源（消耗品）
  └── 江中清气（一次性使用，需偿还）
```

## 心法口诀
"功法传内不传外，传贤不传庸，传信不传疑。"
