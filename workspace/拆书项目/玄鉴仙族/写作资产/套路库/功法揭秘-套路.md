# 套路：功法揭秘

## 基本信息

| 项目 | 内容 |
|------|------|
| 套路编号 | R003 |
| 套路名称 | 功法揭秘 |
| 核心功能 | 通过一个冲突事件引出功法/修炼体系的核心设定 |
| 首次出现 | ch052-ch053（邓求之→夜月凛气） |
| 出现频率 | 中 |

## 套路结构

### 1. 触发条件
- 发生了某个冲突/异常事件（如纨绔挑衅）
- 有知情人（师兄/师姐/师傅）来解释

### 2. 标准流程

```
冲突事件发生 → 主角疑惑(为什么对方如此行为) → 
知情人解释 → 展开功法设定 → 揭示更深层关联(太阴月华/望月湖)
```

**ch052-ch053应用实例：**
- 冲突：邓求之阴晴不定，赢了认输，又打随从
- 疑惑：李尺泾不明白邓求之为什么这样
- 解释者：袁湍（师姐）
- 设定展开：
  - 夜月凛气：五品功法
  - 缺一气（太阴月华）
  - 副作用：影响心智→阴晴不定
  - 来源：月华元府（月华仙府）
  - 青池宗也有同类功法《月湖映秋诀》
- 深层关联：太阴月华皆出自望月湖→李尺泾联想到自家鉴子

### 3. 关键元素

| 元素 | 说明 |
|------|------|
| **冲突先行** | 先有异常行为/事件，后有解释——不直接堆设定 |
| **知情人身份** | 解释者需要有合理的信息渠道 |
| **品级锚定** | 给出功法品级，建立参照系（五品>四品>三品） |
| **连锁揭示** | 一层层往外推：功法→功法问题→来源→更大的世界观 |
| **主角联结** | 主角必须从揭秘中找到与自身的关联（李尺泾联想到鉴子） |

### 4. 注意事项
- 信息密度大但不枯燥——通过"对话+震惊反应"来引导读者
- 不要一次性把所有信息都倒出来，留一些悬念
- 解释的语气要自然（闲聊/谈天，不是上课）
- 揭秘内容必须与剧情主线相关，不能是孤立的设定
