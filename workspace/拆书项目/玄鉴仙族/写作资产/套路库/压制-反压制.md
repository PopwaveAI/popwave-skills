# 套路：压制·反压制

## 核心公式

```
角色A（强者/胜利者）压制角色B（弱者/失意者） → 制造压抑情绪 → 
第三方介入/展示 → 角色B的"隐性优势"被揭示 → 情绪反转（从压抑→释然/自豪）
```

## 示例（ch049-ch050）

### 压制阶段（ch049）
- 压制者：卢平远（突破练气期）
- 被压制者：万元凯（青元轮）
- 压制方式：倨傲态度、爱搭不理、"恭喜道友了"（阴阳怪气）
- 情绪效果：压抑、羡慕、失落

### 反击阶段（ch049-ch050）
- 反击方式：萧雍灵主动提及万家先祖万华芊（隐性底蕴）
- 追加反击：萧初庭单独留李通崖（李家获得筑基大佬青眼）
- 情绪效果：自豪、解气

## 情绪公式

```
压抑 → 隐性优势揭示 → 释然/自豪
```

注意：这不是直接的打脸，而是通过"被第三方提及/认可"来实现的反压制

## 变种

| 变种 | 差异 | 效果 |
|------|------|------|
| 直接打脸 | 当面击败压制者 | 爽快直接 |
| 隐性反击 | 通过第三方提及底蕴 | 更真实、留有余地 |
| 时间差反击 | 日后的修炼超过压制者 | 延迟满足 |
| 格局压制 | 压制者看不起的人被更高层认可 | 格局差带来的爽感 |

## 适用场景

- 展现家族间的竞争关系
- 为后续冲突埋下伏笔
- 制造情绪起伏

## 注意事项

- 压制要有合理依据（卢平远确实突破了练气）
- 反击不能太刻意（通过第三方提及往事比当面回击更自然）
- 被压制者的反应要真实（万元凯的羡慕和失落写得很好）
