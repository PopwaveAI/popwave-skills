# 套路：发现遗迹/阵法

## 基本信息

| 项目 | 内容 |
|------|------|
| 套路编号 | R004 |
| 套路名称 | 发现遗迹/阵法 |
| 核心功能 | 在探索中发现古老的遗迹/阵法，驱动寻宝/探险线 |
| 首次出现 | ch053「眉尺山阵法」 |
| 出现频率 | 中 |

## 套路结构

### 1. 触发条件
- 主角在野外勘探/巡查/迁徙
- 到达某个看似普通的地点
- 发现异常（灵气异常、地貌异常、动物反应异常）

### 2. 标准流程

```
日常勘探 → 发现异常 → 初步试探(射箭/法术) → 
确认有阵法 → 判断等级 → 决策(独吞/共享/求助) →
安排守卫/开发 或 准备破阵
```

**ch053应用实例：**
- 日常：李项平勘探眉尺山，寻找灵田
- 发现异常：某个山坡灵气富足
- 试探：李项平用青乌弓射箭——箭撞上虚空涟漪
- 确认：有幻阵+防御灵阵，≥胎息境
- 决策：不找万家共享（避免分一杯羹和反目风险）
- 安排：先修山路造庭院圈地，等二哥突破玉京

### 3. 关键元素

| 元素 | 说明 |
|------|------|
| **日常起点** | 从平常的勘探/赶路开始，制造意外发现感 |
| **试探手段** | 需要已有的装备/技能来试探（如ch053青乌弓射箭） |
| **效果展示** | 箭撞上涟漪、凭空阻隔——视觉化表现隐形屏障 |
| **等级评估** | 判断阵法强度，决定下一步行动 |
| **决策逻辑** | 是否分享情报——体现家族/个人利益判断 |

### 4. 注意事项
- 发现过程要合理——不能"走两步就撞上光幕"
- 试探过程要展示已有装备的用途（ch053青乌弓射箭印证了ch051购买的合理性）
- 决策过程要体现角色性格（李项平谨慎、家族本位）
- 不能立刻破解——为后续破阵/探索制造时间差
