# 套路：古法秘法体系

## 核心公式

```
功法体系科普 → 正法/古法/秘法三分法 → 优缺点对比 → 为后续剧情铺垫
```

## 关键阶段

1. **提出疑问**：角色主动询问某种知识
   - 李通崖问"古法秘法？"（ch072）
2. **解释正法**：基准参照系
   - 有配套天地灵气，可正常修炼的二品以上功法
   - 价格：几十至上百灵石
3. **解释古法**：失传的法诀
   - 原配套天地灵气消失/找不到，法诀变成无用之物
   - 特点：不是不能练，是缺了关键条件
4. **解释秘法**：人为改造的古法
   - 后人用其他气代替原版的气
   - 缺点：弱于原版→修炼慢/有缺陷/须丹药吊命
5. **角色反应**：角色对此知识的态度——是渴望、畏惧还是无所谓？
   - 李通崖："管他什么古法秘法正法，我囊中空空，却也什么也买不起"（自嘲化解）

## 知识结构表格

| 类型 | 定义 | 优点 | 缺点 | 价格 |
|------|------|------|------|------|
| **正法** | 有完整配套天地灵气的功法 | 稳定可靠、前人千百年锤炼 | 价格昂贵 | 几十~上百灵石 |
| **古法** | 原配套灵气消失的法诀 | 可能威力不俗 | 无法修炼（缺气） | 因人而异 |
| **秘法** | 从古法改造替代灵气而成 | 能用（替代方案） | 弱于原版，有缺陷 | 可有 |

## 情绪曲线

```
好奇(提问) → 认知拓展(科普中) → 理解(消化) → 现实对照(囊中羞涩/反差)
```

## 适用场景

- 需要扩展修仙界功法体系时
- 主角面临突破/选择功法阶段前
- 需要展示"有些功法虽然存在但已失传"的世界设定
- 为后续"主角获得古法/秘法"做知识铺垫

## 叙事功能

1. **世界观填充**：使修仙界的知识体系更丰厚
2. **剧情铺垫**：为后续的古法发现/秘法改造做准备
3. **经济体系**：展示功法市场的价格层级
4. **角色塑造**：角色对知识的态度塑造其性格

## 注意事项

- 科普段落不宜过长——ch072通过坊市闲逛+对话自然输出
- 三分法的关键是"区别"要清晰——否则读者记不住
- 最好有角色反应来消化知识（李通崖的自嘲是范本）
