# 名门投效

## 定义
宗门/世家/散修因新兴势力的潜力或名声，主动释放善意、建立关系。

## 特征
| 维度 | 描述 |
|------|------|
| 动机 | 结交潜力股、剑仙名声、投资未来 |
| 方式 | 主动攀谈、送人情、优惠报价 |
| 风险 | 若对方势弱可能押错注 |
| 时点 | 家族刚刚晋升/出现明星人物之时 |
