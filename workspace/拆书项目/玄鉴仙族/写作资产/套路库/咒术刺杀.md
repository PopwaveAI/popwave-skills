# 套路：咒术刺杀

## 定义
通过远程咒杀/祈咒之术，无视距离和物理防御直接夺人性命的暗杀方式。

## 功能
- 制造"防不胜防"的恐怖感
- 绕开常规战斗的叙事（不给正面应战机会）
- 引入新的力量体系（箓巫/咒术体系）

## 本卷范例（ch099-ch100 木焦蛮求巫杀项平）

### 流程
1. **无法用常规手段击败** → 木焦蛮被李项平打崩溃
2. **寻找非常规手段** → 巫山箓巫
3. **献祭/付出代价** → 女子+灵物+李家血脉孩子
4. **箓巫内部决策** → 从"不愿出手"到"祈咒之术最安全"
5. **施法** → 血池+祭品+咒辞
6. **生效** → 无视距离降临李项平身上

### 咒术优缺点
| 优势 | 劣势 |
|------|------|
| 无视距离 | 需祭品/血池 |
| 难以防范 | 需目标信息（姓名/血脉关联物） |
| 无需正面对抗 | 可能被反噬/被箓气克制 |
| 制造恐惧 | 师门有限制（"不可干扰地上进程"） |

### 关键细节
- **箓气克制咒术**: 黑气被项平体内箓气吞噬——暗示箓气vs咒术的位阶关系
- **咒术祷文**: "祈告玄明正箓……却其目，化起骨，去一身血肉"
- **施法材料**: 练气巅峰妖物（肥大如猪）、血池、人骨

## 对叙事结构的改变
- 咒术使"战斗"从正面交锋变成"施法vs防御"的隐蔽斗争
- 死亡的不可预测性加强
- 引入"因果/气运"维度的对抗
