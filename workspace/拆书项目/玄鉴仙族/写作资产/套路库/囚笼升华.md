# 囚笼升华

## 定义
将物理空间的扩大与实际自由未变的落差，用于表达角色的孤独与无力感。

## 特征
| 维度 | 描述 |
|------|------|
| 手法 | 从"小囚笼"到"大囚笼"——范围升级但本质不变 |
| 情感 | 自嘲式的苦笑 |
| 功能 | 提醒读者/角色：力量提升 ≠ 自由提升 |

## 使用记录
| 章节 | 事件 |
|------|------|
| ch121（暗） | 之前鉴中空间极小（只能容纳神识）→ 现在方圆千里 |
| ch121 | 陆江仙想穿回黎泾山，心头大恐怖升起 |
| ch121 | "好歹这座监狱大了许多"——苦笑收尾 |
