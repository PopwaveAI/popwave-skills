# 套路：坊市探索

## 核心公式

```
进入坊市 → 观察摊位/店铺 → 询价 → 认知升级（发现价格体系/层级差异） → 确认/修正过往认知
```

## 关键阶段

1. **过渡**：从外部环境进入坊市——通过幻阵/禁制等展现修仙与凡俗的隔绝
2. **观察**：摊主商品浏览（展示修仙经济物种多样性）
3. **询价**：询问特定物品价格
4. **认知冲击**：通过价格发现自己的认知偏差
5. **闭环**：对比过往经历→得出新结论

## 信息投放

| 信息类型 | 方式 | 示例（ch050） |
|---------|------|-------------|
| 价格基准 | 直接询问 | 胎息阵旗8灵石 |
| 层级差异 | 摊主说明 | "练气材料请去东边店中" |
| 历史验证 | 主角自省 | "萧元思卖我家的价格算是便宜" |

## 情绪曲线

```
期待(进入) → 好奇(浏览) → 认知刷新(询价) → 满足(认知闭环)
```

## 功能

- 展现修仙世界的经济体系与阶层
- 让主角（和读者）重新理解过往经历
- 为未来的交易/采购做世界观铺垫
- 过渡章节的平缓收束

## 注意事项

- 价格体系要前后一致（白纹鲫鱼皮=练气材料要维持）
- 询价后的"认知闭环"很重要——让读者获得"原来如此"的满足感
