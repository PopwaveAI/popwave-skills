# 套路：坊市购物

## 基本信息

| 项目 | 内容 |
|------|------|
| 套路编号 | R001 |
| 套路名称 | 坊市购物 |
| 核心功能 | 主角在修仙坊市中购物，展示经济状况、世界观、获取新装备 |
| 首次出现 | ch051「购买」 |
| 出现频率 | 低（前期） |

## 套路结构

### 1. 触发条件
- 主角进入坊市/集市
- 需要购买物资/装备
- 有经济压力（灵石有限）

### 2. 标准流程

```
盘点财物 → 初看高价物(买不起) → 寻找目标 → 议价 → 交易/放弃 → 偶遇额外收获
```

**ch051应用实例：**
- 盘点：5灵石、5白元果、90斤灵米，负债10灵石
- 初看：符箓（3张1灵石）、法剑（10灵石）——均买不起
- 寻找：逛摊位，寻找修仙百艺传承（未果，引爆世界观解说）
- 议价：青乌弓，4灵石→2灵石+5白元果（≈2.5灵石）
- 偶遇：吴柞虫+虫书，2灵石

### 3. 关键元素

| 元素 | 说明 |
|------|------|
| **经济窘迫** | 主角必须精打细算，每一块灵石都是珍贵的 |
| **信息展示** | 通过摊主/路人自然展开世界观 |
| **物品种类** | 分级：看看就好(买不起)→犹豫对比→最终下单 |
| **议价细节** | 必须给出具体的议价逻辑，不能简单"讲价成功" |
| **混合支付** | 灵石+灵物混合支付是常见模式 |

### 4. 注意事项
- 不要写成爽文式的"买买买"，要体现穷家难当的经济逻辑
- 议价过程要合理：指出货物缺陷、给出替代方案、时间压力
- 每一笔消费都要体现主角的决策逻辑——为什么选这个不选那个
- 后续要有呼应：买的物品要在后续章节中用上
