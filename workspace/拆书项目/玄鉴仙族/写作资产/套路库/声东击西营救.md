# 套路库：声东击西营救

> 归类：情节驱动 / 战术配合

---

## 定义

营救方派出正面战力（通常是一位实力强劲的修行者）主动挑战敌营，通过高调的战斗吸引敌营全部高端战力+大量士兵的注意力，营救目标则趁敌方认知聚焦于空中/正面战场之时，从敌方防守薄弱的方向完成逃脱。关键在于"正面的吸引力足够强"和"逃脱者足够快"——不是暗杀，而是光明正大的"我打我的，你跑你的"。

---

## 本卷实例（ch095 — 李通崖牵制×李玄岭逃脱）

| 项目 | 内容 |
|------|------|
| 正面牵制方 | 李通崖（练气二层）——单骑闯寨，踏空质问，一打三 |
| 逃脱方 | 李玄岭（胎息二层）+ 李景恬（凡人） |
| 牵制方式 | 上空与木焦蛮+两练气大战——玄水剑诀压制全场 |
| 牵制效果 | 整个寨子的注意力被吸引——士兵抬头上望，胎息境去平乱，练气灵识仅十余丈 |
| 逃脱窗口 | 开战到撤退之间的十几回合——李通崖边打边等，确认两人脱险后撤退 |
| 逃脱手段 | 断枷→杀二人→越栏→金光术→神行术——十几息完成 |
| 撤退方式 | 李通崖"再斗上十几回便撤走"——且战且退，不恋战 |

## 套路结构

```
┌─────────────┐
│ 情报确认     │ ← 营救方确认目标位置+敌营战力配置
└──────┬──────┘
       ▼
┌─────────────┐
│ 正面出击     │ ← 高调杀到→公开宣示→吸引所有注意力
└──────┬──────┘
       ▼
┌─────────────┐
│ 高端牵制     │ ← 与敌方最强战力交战→压制/对峙→让战场"黏住"
└──────┬──────┘
       ▼
┌─────────────┐
│ 目标逃脱     │ ← 趁乱执行脱逃方案（此时敌营低端战力无人关注）
└──────┬──────┘
       ▼
┌─────────────┐
│ 掩护撤退     │ ← 正面方确认目标安全后→且战且退→全身而退
└──────┬──────┘
       ▼
┌─────────────┐
│ 会后汇合     │ ← 正面方与逃脱方在预设地点/方向重逢
└─────────────┘
```

---

## 写作要点

1. **正面战力要够强**：必须有能力在多人围攻下支撑——李通崖能一压三才能制造窗口。如果几下被打败，营救失败
2. **牵制需要"理由"**：不能无缘无故去挑战——李通崖的质问（"山越为何袭我家"）让战斗有正当性
3. **逃脱执行必须快**：十几息的窗口——用短促的动词堆叠制造"争分夺秒"的紧迫感
4. **正面方必须确认目标安全**：李通崖边打边焦虑"两个孩子逃出来没有"——而不是直接跑了不管
5. **撤退同样需要技术**：且战且退——不是逃跑而是控制节奏，防止敌方追击发现逃脱
6. **双方不需要提前沟通**：李通崖不知道李玄岭的计划，李玄岭不知道李通崖会来——"巧合式的默契"比安排好更有故事感

---

## 注意陷阱

| 陷阱 | 说明 |
|------|------|
| 正面战力太无敌 | 如果李通崖轻松碾压一切，战斗失去紧张感——设计为"压得住但耗不起" |
| 逃脱太顺利 | 如果逃跑没有任何风险，营救方显得多余——木焦蛮的巫术追击提供了风险 |
| 双方提前约定 | 如果两人提前商量好里应外合，失去"巧合救援"的戏剧效果 |
| 敌营高端战力太多 | 如果敌方有超过正面对手能制约的练气数量（如伽泥奚也在场），计划失败 |
