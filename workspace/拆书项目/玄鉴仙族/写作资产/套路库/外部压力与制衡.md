# 套路：外部压力与制衡

## 定义
家族/势力面临外部威胁（敌对势力、妖物、大势力压迫）时，利用多方势力之间的矛盾来维持自身安全与发展的策略。

## 叙功能
- 扩展世界地缘政治格局
- 为主角势力提供成长压力和动力
- 制造"危机感"让读者持续关注

## 本卷实例（ch044-ch045）

### 大黎山区域势力格局

```
        北 → 郁家（万家旧敌，未详）
        
  西 → 汲家（练气期修士汲登齐，崛起中）
         ↑
  西 ← 万家（玉京轮巅峰家主+华芊山大阵，已衰落）
         ↓
  东 → 李家（黎泾山，新兴势力，表面无练气）

  南 → 大黎山（妖族地盘，狼妖来源地）
```

### 关键逻辑链

```
汲家崛起 → 围攻万家 → 万家寻求强力手段 → 找李家交易 → 李家无法兑现
                                                                  ↓
                            利用汲家牵制万家 ← 决定"拖"字诀 ← 暴露困境
```

### 李木田的战略洞察

> "且先拖着……让那汲登齐替我等压着万家"

策略拆解：
1. **短期**：万家在汲家压力下自顾不暇，无力在灵稻/地界上对李家施压
2. **中期**：李家利用这个窗口期发展（培养玉京轮修士）
3. **长期**：等李家实力足以与万家对等时，再重启交易或重新谈判
4. **核心前提**：汲家存在就是李家的保护伞——外部敌人压制了潜在合作方

### 写作要点
| 要素 | 说明 |
|------|------|
| 多方博弈 | 不能只有两方对立，至少引入第三方形成制衡 |
| 势力实力透明 | 让读者清楚了解各方的修为/资源/底牌 |
| 动态平衡 | 制衡是临时的——李家成长会打破平衡 |
| 战略合理性 | 家族决策要有前瞻性，不能仅为推进剧情而降低智商 |
