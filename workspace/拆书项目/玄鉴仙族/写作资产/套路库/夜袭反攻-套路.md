# 套路：夜袭反攻

## 核心公式

```
被袭击/受辱（白天/昨夜） → 决定复仇（"有仇不过夜"） → 情报获取（法鉴侦察/俘虏招供） → 清除哨兵 → 衔枚夜袭 → 火光为信四面杀出 → 敌军溃散/主将被擒
```

## 关键阶段

1. **复仇意志**：经受袭击后不选择"等明天"——而是要"今晚就干"
   - "这事就不能这么算了"（李项平，ch079）
   - "此间不须有隔夜仇"（ch079）
   - 体现主角家族的**血性**——区别于普通家族的"忍气吞声"

2. **情报准备**：夜的黑暗是双刃剑——需要信息优势
   - 法鉴扫山——"将整座山的情况扫得一清二楚"（ch079）
   - 知晓敌军驻扎位置/人数/哨兵部署
   - 情报是夜袭成功的前提——没有情报优势的夜袭=自杀

3. **哨兵清除**：消灭敌军的"眼睛"
   - "数道流光飞越入山间，挨个点射死了山越留在林中监视的哨兵"（ch079）
   - 狙杀要安静/精准——暴露则整个计划失败
   - 这是**技术活**——需要远程精准射手

4. **衔枚夜袭**：无声接近——在敌军毫无察觉时完成包围
   - "取横枚衔在口中，用衣巾紧紧缚在面上防止其掉出"（ch079）
   - 夜色的肃杀感——全部人沉默行军
   - 指令通过手势/传令兵传递——不发出声音

5. **信号/引爆**：以一道流光/火光为信号——同时发动
   - 李项平一箭射入营地为信（ch080）
   - "四周的山林间升起一道道火光"——包围圈合拢
   - 从极静到极乱的瞬间转换

6. **分割包围**：不是混战——是有组织的围剿
   - 山越被"分割成了四五片"——各自为战
   - 主将（李通崖）直取敌军首领——擒贼先擒王

7. **迫降收尾**：主将投降后，传令"降者不杀"
   - 阿会剌大叫投降后——"降者不杀！降者不杀！"
   - "成片成片的部众丢了武器跪下"——高效收尾
   - 夜袭的终点是秩序恢复——不是继续杀戮

## 战术要素对比

| 要素 | 夜袭方（李家） | 被袭方（山越） |
|------|--------------|--------------|
| 情报 | 法鉴扫山——全知视角 | 哨兵被清——瞎子 |
| 准备 | 衔枚裹面——完全静默 | 饮酒庆功——毫无防备 |
| 指挥 | 统一号令——配合严密 | 被分割——各自为战 |
| 主将 | 李通崖正面碾压 | 阿会剌被一剑震脱武器 |
| 收尾 | "降者不杀"——秩序控制 | 熟练投降——习以为常 |

## 叙事节奏

```
复仇决定[决断] → 请出法鉴[神圣] → 扫山侦察[全知] → 
清除哨兵[精准/安静] → 衔枚行军[肃杀] → 
箭入营地[爆发] → 四面火光[爽快] → 
主将决战[激烈] → 敌军投降[秩序恢复]
```

## 适用场景

- 主角家族遭遇突袭后——需要一次"漂亮的反击"展示实力
- 主角家族拥有情报优势（法鉴/无孔不入的探子）
- 需要展示主角家族"有仇不过夜"的性格底色

## 注意事项

- 夜袭成功的前提是**情报碾压**——没有法鉴级别的侦察能力，夜袭风险极高
- 哨兵清除是夜袭的"前菜"——要单独写清楚
- 夜袭的最高潮不是"杀更多的人"——而是"活捉主将"
- 收尾要快——不要让夜袭变成混战
