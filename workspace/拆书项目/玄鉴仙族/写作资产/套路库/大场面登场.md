# 套路：大场面登场

## 核心公式

```
铺垫（众人期待/氛围造势） → 远观（视觉奇观） → 近景（细节展示） → 人物亮相（身份揭示）
```

## 关键阶段

1. **铺垫**：通过围观者的反应/对话造势
   - "好大的架势""这便是那霞光云船？"（ch049）
2. **远观**：宏大视觉奇观
   - "东边缓缓升起一片灿烂的霞光"（ch049）
3. **近景**：细节呈现
   - "这是一艘架着金色霞光的云船，两侧插着一对云雾般翅膀"（ch049）
4. **亮相**：人物正式登场，揭示身份
   - "飘出几道青色的流光……显出个身着青玉色长裙的女子"（ch049）

## 情绪曲线

```
期待(铺垫) → 敬畏(远观) → 震撼(近景) → 肃然(亮相)
```

## 视觉效果三要素

| 要素 | 作用 | 示例 |
|------|------|------|
| 光线 | 氛围/神圣感 | 金色霞光覆盖 |
| 动态 | 速度/力量感 | "破开一层层的云雾，几息之间便停靠" |
| 距离 | 层次感 | 先"缓缓升起"→再"缓缓靠近"→最后"停靠山顶" |

## 适用场景

- 重要人物首次登场
- 重大事件开幕
- 需要提升章节情绪到高点时

## 注意事项

- 大场面的调度感要清晰——先远景后近景
- 周围人的反应是大场面成立的关键（众修士躬身行礼）
- 不要过度使用——会降低震撼力
