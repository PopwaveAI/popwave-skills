# 套路：大宗小宗制度

## 适用范围
修仙家族内部的权力/血脉等级制度设计。

## 经典范例
- ch082：李木田遗言确立：嫡系四脉→大宗，余李姓→小宗，再远→支脉
- ch083/ch085：祭祀站位体现了大宗/小宗/旁支的区别

## 结构要素
1. **层级设计**：大宗（至尊）→ 小宗（普通）→ 支脉（待发展）
2. **上下流通**：允许晋升/降级（"你等自为之"）
3. **制度执行者**：族正（李玄宣）——管束李氏子弟
4. **空间体现**：祭祀站位、宅邸等级
5. **外部配套**：陈许柳诸大姓迁入黎泾镇，防坐大

## 写作要点
- 制度要在对话/遗言中提出，而非旁白解释
- 一句话点明核心（"上下流通，你等自为之"）
- 制度的执行者明确（玄宣为族正）
- 制度与外部施压配套（迁大姓、防坐大）
