# 套路：大棋局揭秘

## 定义
前期铺垫的种种事件被证明是某个高阶存在/大势力的统一布局，主角发现自身只是棋盘的棋子。

## 功能
- 将故事格局从"家族级冲突"升级到"紫府级/天下级博弈"
- 让主角/读者感到细思极恐
- 为角色行动提供新的动力源（打破棋局or顺应棋局）

## 本卷范例（ch098 汲登齐密信→大巫阴谋）

### 揭示流程
1. **铺垫**: 山越入侵（前文多次出现）
2. **疑点积累**: 青池宗下令"权且忍让"不合理
3. **关键线索**: 密信内容——"大巫等了一百年……"
4. **拼接真相**: 
   - 大巫→培育伽泥奚→统一北麓→祭品
   - 各支山越入侵→掩盖伽泥奚一人
   - 所有战争都是烟雾
5. **情绪冲击**: "头顶的天空一片阴暗"——棋子的绝望感

### 结构公式
> 事件分散（前文）→ 零散疑点 → 一份关键信息（密信/密档/遗言）→ 拼图完成 → 细思极恐

### 大棋局的参数
| 参数 | 本卷值 |
|------|--------|
| 棋手 | 大巫（紫府级？） |
| 棋子 | 伽泥奚（天命之子） |
| 棋盘 | 北麓山越全境 |
| 时间跨度 | 一百年 |
| 旁支 | 各支山越入侵=烟幕 |
| 隐蔽手段 | 坐视劫掠，牺牲其他部落 |
| 最终目标 | 祭品（炼制/突破/仪式） |

## 变体
- **镗金门紫府陨落**: 另一盘棋（青池宗借机扩张）
- **月华元府线索**: 法鉴可能的来源（伏线更长的棋）

## 节奏要点
1. 揭示前需有足够"异常信号"堆叠
2. 信息集中释放时不宜过长（200-300字为佳）
3. 应有"震惊→回溯→感受"三阶段情绪曲线
4. 揭示后必须给角色一个动作反应（李通崖"抬头望天"）
