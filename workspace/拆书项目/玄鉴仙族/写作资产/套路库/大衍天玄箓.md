# 大衍天玄箓

## 定义
一种高级箓气，拥有推演未来、测算玄机、预测前缘后续的能力。可制造极其逼真的"未来记忆"植入载体神魂。

## 特征
| 维度 | 描述 |
|------|------|
| 箓名 | 大衍天玄箓 |
| 形态 | 通体亮银的篆字，银色影子 |
| 能力 | 衍未来、算玄机、测前缘后续、主未定之事 |
| 级别 | 至少金丹期以上 |
| 定位方式 | 在升阳府中留下银色影子 |
| 局限性 | 不涉时空之力（非真正穿越） |
| 箓气残留 | 即使本体不在，接触过的人会留有箓气痕迹 |

## 工作原理
"大衍天玄箓依据眼前之事推演而出"——基于当前现实推演未来发展的可能性，将其包装为"亲身经历过的记忆"。
