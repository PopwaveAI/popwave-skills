# 套路：天外来祸

## 定义
更高层级的势力（宗门、筑基级家族）之间的冲突突然降临到小家族头上，迫使小家族在夹缝中做出生死抉择。

## 本套路适用场景
- 小家族处于多方势力交汇的敏感地带
- 需要制造外部危机以测试家族韧性
- 引入上层世界观（三宗七门等）
- 推动主角做出高难度决策

## 典型案例（ch150）
**情境**：紫烟门灵岩子被长霄门追杀至李家上空。

**四阶段叙事**：

### 第一阶段：危机降临
```
灵岩子（狼狈逃窜）→ 李家大阵外求救
  长霄门两人（追杀）→ 威胁"多管闲事就灭门"
```

### 第二阶段：两难困局
```
选择一：公开帮助灵岩子 → 得罪长霄门 → 灭门风险
选择二：拒绝帮助灵岩子 → 紫烟门追责 → 青池宗压力
选择三：隐身不救 → 坐视灵岩子死在门前 → 一样被追查
```

### 第三阶段：破局之策
李通崖的"最优解"：
```
公开示弱（"老祖不在"）→ 欺骗长霄门
     +
暗设伏杀（引至大榕树下）→ 无声杀人
     =
既救灵岩子 + 不得罪长霄门（无人知晓）
```

### 第四阶段：执行
```
灵岩子面不变色配合表演 → 飞向湖上诱敌
李通崖+李玄锋 → 大榕树下设伏
狐狸（神秘存在）→ 不确定变量
章末悬念：伏杀即将开始
```

## 关键技巧
- **皮球效应**：灵岩子一路被小家族踢皮球踢到李家（"剑仙世家"名声所累）
- **两难表出**：帮不帮的利弊要清晰罗列，让读者感受抉择的重量
- **第三选择**：不选A不选B，创造C方案（"悄悄的杀了"）
- **示弱伪装**：公开示弱+暗地行动的配合

## 风险矩阵

| 风险源 | 可能性 | 影响 | 缓解措施 |
|--------|--------|------|----------|
| 长霄门后续追查 | 中 | 灭门 | 蒙面+隐秘地点+不留活口 |
| 伏杀失败 | 低 | 全面暴露 | 李玄锋+金庚弓加入 |
| 狐狸介入不可控 | 未知 | 变量 | 狐狸已认识李通崖 |

## 心法口诀
"福祸无门，惟人自召。天外来祸，既要能接，也要能藏。"
