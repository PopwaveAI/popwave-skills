# 套路：天才陨落/人材为药

## 定义

宗门/顶级势力将绝世天才作为"炼丹材料"（或其他形式的消耗品）投入修炼/炼药过程。是《玄鉴仙族》中揭示上界黑暗体制的核心套路。

## 功能

- 塑造世界观的残酷底色——最美好的天赋被最丑陋的方式消耗
- 建立读者对体制（青池宗/迟尉）的仇恨与不满
- 为家族/幸存者角色提供复仇/反抗的动力
- 讨论"天才悖论"：天才因才华而被选中，才华正是其致死原因

## 关键要素

| 要素 | 说明 |
|------|------|
| **选中理由** | 天赋过高，符合炼丹的特殊要求（道基、功法、年龄） |
| **掩饰手段** | 宗门以任务/征召名义调集天才，外部不知情 |
| **执行者分层** | 顶层决策者（迟尉）→中层执行者（元素真人/迟灸云）→底层工具（大蛟） |
| **被牺牲者态度** | 从反抗到认命到最后的尊严（李尺泾的从容） |
| **旁观者反应** | 有人惋惜（元素真人）、有人得意（迟灸云）、有人麻木 |
| **物化语汇** | "药龄"、"药材"、"美味"——人被定义为一味原料 |

## 适用章节

- ch120（入炉）：完整流程——李尺泾+邓求之投入丹炉
- ch119：为陨落做铺垫——越强大越可惜

## 变体

- **直接杀戮型**：因太强被忌惮而灭杀
- **工具用尽型**：榨干价值后抛弃
- **炼丹型（本案例）**：以天才的肉身+修为为炼丹原料
- **夺舍型**：老怪夺取天才躯壳

## 世界观效应

这是修仙界最黑暗的规则之一——"天赋"不一定带来好事，可能带来最可怕的命运。萧家老祖"五年不得突破"的命令正是对此的应对。
