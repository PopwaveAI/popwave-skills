## 套路结构

### 1. 触发条件
- 有无法用常规手段解决的障碍（阵法/强敌）
- 已经建立祭祀制度（口诀+祭品+仪式）

### 2. 标准流程

```
评估方案(常规手段不可行) → 准备祭祀(灵米瓜果) →
捧法器到目标地点 → 恭请(念承法要令) →
太阴玄光降下 → 对抗/僵持 → 以质取胜(灵力质＞量) →
障碍被摧毁 → 新通道/洞口/遗迹浮现 → 
法器篆符熄灭(消耗) → 谨慎回收法器
```

**ch055应用实例：**
- 评估：阵法≥胎息境，李家无人懂阵，常规方法无法破解
- 准备：安排祭祀灵米瓜果，告罪后捧鉴子
- 地点：眉尺山山坡前
- 恭请：李通崖下拜念口诀
- 降下：白光刺破黑夜，精准降落在山坡上
- 对抗：太阴玄光撞上薄膜，僵持
- 以质取胜：阵法的灵力"量虽大，质也差"——金光触白光如残雪融化
- 结果：阵法闪烁熄灭，石质洞口浮现，灵气喷涌
- 消耗：十二道篆符逐一熄灭，鉴子落回手中
- 回收：先送归法鉴，再探洞府

### 3. 关键元素

| 元素 | 说明 |
|------|------|
| **前置条件** | 需要经过祭祀仪式，不能随意使用 |
| **视觉对比** | 太阴玄光(纯白/月光) vs 敌方金光/黑气——色彩对立 |
| **抗衡过程** | 不能秒杀——需要僵持片刻，体现力量交锋 |
| **质 vs 量** | 输出灵力品质高（太阴玄光）胜于数量大（阵法）——体现境界优势 |
| **有代价** | 每次使用消耗一道篆符——防止无限制使用 |
| **谨慎收尾** | 使用后妥善回收法器，体现敬畏之心 |

### 4. 注意事项
- 每次使用都要有"代价"——不能变成随意乱放的大招
- 破阵过程要有层次：接触→僵持→突破→碎裂
- 阵法被破的后果要合理：不一定爆炸，可能是缓缓熄灭
- 战后要安排回收/恢复环节，不能打完就走
- 太阴玄光对其他目标的杀伤效果已单独设定（≈杀炼气初期）
