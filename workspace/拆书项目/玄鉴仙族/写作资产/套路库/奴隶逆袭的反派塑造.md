# 套路库：奴隶逆袭的反派塑造

> 归类：人物弧 / 反派立体化

---

## 定义

反派角色拥有底层/奴隶出身，通过暴力手段突破阶级限制成为一方霸主。他的残暴不是天生的，而是被压迫历史塑造的创伤反应。核心矛盾：他反抗了压迫，却成了新的压迫者——但内心深处仍保留对"奴隶"二字的极端敏感。

---

## 本卷实例（ch090 — 伽泥奚）

| 项目 | 内容 |
|------|------|
| 曾为 | 奴隶（颈间纹有奴隶横纹） |
| 翻身方式 | 杀尽买下他的部族，胁迫族巫洗去横纹 |
| 创伤后遗症 | 此后不在身体上涂抹任何花纹（纹身=奴隶标记） |
| 对奴役话题的反应 | 木焦蛮说"带回去充当奴隶"时，伽泥奚"眼中闪过一丝暴虐的杀意"（但克制未发作） |
| 对奴隶的投射 | 严令搜捕"面容姣好者"——精准捕捉可能的高价值俘虏 |
| 当前处境 | 统一北麓十万山越，正是人生巅峰，但木焦蛮劝其考虑子嗣问题 |

### 人物层次分析

```
表层：冷酷、高效、智谋过人
         ↓
中层：细致入微的观察力（被褥温度/面容姣好逻辑）
         ↓
深层：奴隶创伤——对奴隶话题极端敏感，但不流露（克制=可怕）
         ↓
底色：统一北麓后寻求正统化（子嗣问题被提出）
```

## 写作要点

1. **创伤细节要具体**：不要只说"他曾是奴隶"——要给出"颈间横纹""洗去花纹""杀尽买主"的具体意象
2. **暴露而不解释**：伽泥奚的暴虐杀意只"闪过一丝"——读者自己解读，比旁白解释更有张力
3. **奴隶出身的反派不能只有工具性**：他攻打李家不单是任务——背后的身份挣扎是隐线
4. **创伤不能取代残暴**：他对陈三水的刑讯和拧头是真实的冷酷——过去不能洗白现在
5. **埋下可被针对的弱点**：伽泥奚对"奴隶"的高度敏感——未来可能被利用
