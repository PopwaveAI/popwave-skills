# 套路：妖物袭村

## 定义
妖兽/妖物袭击人类聚居地（村落、城镇）引发冲突的经典套路。用于制造外部威胁、展示角色能力、引入战斗系统。

## 叙功能
- 制造危机，激活主角团
- 展示角色性格（遇事态度、领导力、战术思维）
- 引入修仙世界的力量层级体系

## 本卷实例（ch041-ch043）

### 结构模板
```
死讯传来 → 村民恐慌 → 核心角色到场 → 组织防御 → 妖物现身 → 对峙/交锋 → 援军/底牌 → 斩杀 → 善后
```

### 变体与创新
1. **妖物有灵智**：非单纯野兽，会轻蔑、会回忆、有旧伤、有族群被灭的历史
2. **双线利益绑定**：利用妖物迫使两家（李+万）合作，不靠交情靠利益
3. **以退为进诱敌**：非原地死守，而是主动撤退将妖物引向伏击点
4. **心理博弈**：李项平利用妖物"惜命谨慎"的弱点制定战术，而非纯武力对抗

### 写作要点
| 要素 | 说明 |
|------|------|
| 妖物设定 | 给出具体形象（银灰巨狼、黄牛大小、幽绿瞳孔）、修为（玉京轮）、能力（吐息、扑击） |
| 凡人视角 | 通过村民恐惧/伤亡来衬托危机真实感 |
| 层次推进 | 从消息→目击→交锋→不敌→反击，逐级加压 |
| 收尾方式 | 非主角亲手斩杀→展示底牌，留有余地 |
