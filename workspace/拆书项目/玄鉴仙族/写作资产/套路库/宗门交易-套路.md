# 套路：宗门交易

## 核心公式

```
宗门弟子发现需求（家族功法短板/丹药需求） → 借助同门/师尊的关系 → 提出交易意向 → 讨价还价 → 以物易物达成双赢 → 兑现交易
```

## 关键阶段

1. **需求发现**：弟子在宗门中意识到家族需要某种稀缺资源（功法/丹药/法器）
   - 李尺泾发现李家缺乏系统性功法传承（ch076）
   - 多为练气/胎息初期的需求——大资源需要更高级别的关系

2. **关系利用**：借助同门/师尊的人脉打开交易渠道
   - 萧元思与李尺泾是师兄弟——有信任基础
   - "你替我萧家向宗内要枚遂元丹作抵"——宗门贡献换家族资源

3. **谈判博弈**：双方各有所求，讨价还价
   - 李尺泾："区区一门三品功法我可不换！"
   - 最终：两门三品/二品功法+采气诀 换 遂元丹
   - 关键：**双方都觉得自己赚了**（萧家得了稀有丹药，李家得了传承功法）

4. **资源筛选**：根据家族地缘特征选择最合适的功法
   - "黎泾山就在望月湖旁"——选择能利用望月湖/芦苇河网的功法
   - 因地制宜——不是随便功法都要，要贴合家族实际

5. **兑现等待**：交易达成但需要时间兑现——悬念留到后续

## 交易谈判技巧（玄鉴仙族版）

| 技巧 | 示例 | 说明 |
|------|------|------|
| 先提高价留降价空间 | "我任选两门三品" | 狮子大开口，等对方还价 |
| 抬升对方筹码价值 | "遂元丹珍贵得很" | 强调己方付出的稀有性 |
| 指定品质（正法vs古法） | "我要正法，才不要古法秘法" | 正法是经过宗门验证的标准化功法 |
| 地理匹配 | "采用湖中金秋的练气诀" | 结合家族领地特征选功法 |

## 交易类型

| 类别 | 示例章节 | 内容 |
|------|---------|------|
| 功法换丹药 | ch076 | 萧家功法 + 采气诀 → 遂元丹 |
| 宗门任务换奖励 | 李尺泾南下任务 | 任务完成后向宗内提要求 |
| 家族资源换宗门资源 | 后续可能 | — |

## 适用场景

- 家族弟子在宗门中有一定地位，有能力为家族谋利
- 家族进入"资源拓展期"——需要系统性功法传承
- 同门/师尊关系厚实——有信任基础才可能进行高级别交易

## 注意事项

- 交易的核心是"双赢"——不是单方面的索求
- 谈判过程要有"你来我往"的张力——不能一方说了算
- 功法选择要考虑地理环境——《玄鉴仙族》强调"修仙要因地制宜"
- 交易结果不能马上兑现——要有等待期保持悬念
