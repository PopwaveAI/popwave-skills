# 套路库：家族器物传承

> 归类：情感驱动 / 精神锚点

---

## 定义

家族中的某件日常器物（不是法器，不是法宝——而是一件粗糙、使用痕迹明显、与某位逝去长辈紧密相关的旧物）在某个特定情绪节点，被核心角色无意中使用/触碰，器物上的文字/刻痕/特定位置触发角色的回忆和情感共鸣，从而完成一次跨越代际的精神传递。器物本身没有神力，但承载的情感重量足以成为角色重新振作、坚定方向的"精神锚点"。关键词：**粗糙的器物 × 深厚的情感**。

---

## 本卷实例（ch093 — 李木田的破木椅子）

| 项目 | 内容 |
|------|------|
| 器物 | 一把破木椅子——李木田生前"年年坐着"的旧椅 |
| 主人 | 李木田——李家上一代掌门人，"因为一句筑基谣传在山上困了整整半生" |
| 触发者 | 李玄宣——李家少家主，正在经历山越入侵+姐弟失踪+法鉴丢失的三重打击 |
| 触发情境 | 失落中颓然坐下→看到椅子腿上歪歪扭扭刻着八个字 |
| 刻字 | "宜室宜家，尔炽尔昌"——家族从安定到繁盛的终极愿景 |
| 情感反应 | "脑海中猛然浮现出那个老人"→"抹了抹眼角"→"神色坚定地出门去了" |
| 叙事功能 | 从情绪低谷（失落/担忧/无奈）→转换为行动（稳定家族/继续工作） |

## 套路结构

```
┌─────────────┐
│ 危机冲击     │ ← 角色遭遇重大打击（失去/被俘/战败/决策失误）
└──────┬──────┘
       ▼
┌─────────────┐
│ 情绪低谷     │ ← 失落/愤怒/无奈/自我怀疑的状态
└──────┬──────┘
       ▼
┌─────────────┐
│ 触碰器物     │ ← 无意中接触某件与逝去长辈相关的日常物品
└──────┬──────┘
       ▼
┌─────────────┐
│ 触发回忆     │ ← 器物上的刻痕/使用痕迹/特定位置→激活长辈画面
└──────┬──────┘
       ▼
┌─────────────┐
│ 精神传递     │ ← 长辈的未竟之志/精神遗嘱通过器物被感知
└──────┬──────┘
       ▼
┌─────────────┐
│ 重新振作     │ ← 从低谷中走出→坚定方向→继续前行
└─────────────┘
```

---

## 写作要点

1. **器物必须"平常"**：不是传家法宝，不是灵器——越普通越真挚。李木田的椅子"破损"但被保留——说明家族记得他
2. **刻字要"歪歪扭扭"**：不是工整的铭文，而是手工刻上去的——质朴的人味是最强的感染力
3. **情感触发不能太长**：从见字→抹泪→离开——一次情感的快速冲击，不要拖沓抒情
4. **器物贯穿前后文**：这把椅子如果前面出现过（至少暗示过），再在危机中出现会更自然
5. **长辈的遗憾要在前文铺垫**："因为一句筑基谣传在山上困了整整半生"——李木田的悲剧底色已经在ch093前建立
6. **不要直接说教**：八字本身就是传世格言——不需要额外解释，让读者自己体会

---

## 注意陷阱

| 陷阱 | 说明 |
|------|------|
| 器物太"完美" | 如果椅子是精雕细琢的灵木宝椅，失去"破木"的质朴感 |
| 抒情过度 | 不要让角色对着椅子哭半天——抹泪即止，转为行动 |
| 器物功能性过强 | 如果椅子里藏着地图/功法/灵石——那就变成"寻宝"而非"精神传承" |
| 需要前文铺垫 | 如果没有提前建立长辈与晚辈的情感连接，单次触发会显得生硬 |
