# 套路：家族训诫

## 核心公式

```
错误/不作为发生 → 长辈发现 → 训斥（指出问题） → 输出家族哲学（核心台词） → 晚辈受教/成长
```

## 关键阶段

1. **错误或不作为**：晚辈在处理事务时暴露出问题
   - 李玄宣掌家一月：没探查万家情报、让玄锋独自跑去汲家、只顾与掌事周旋（ch074）
2. **长辈发现**：归来的长辈一针见血点出问题
   - 李通崖："万家覆灭了足足一月有余，你竟然连华芊山下空无一人都不知"
3. **训斥**：直接严厉的批评
   - "李玄宣！你当真做的好事！"
4. **输出家族哲学**：核心台词——点明权力/生存的本质
   - "我李家的权力来源于刀兵与修为"
   - "要果决，要心狠，要敢杀人"
   - "我李家无须抱业守成之主，要的是狼虎进取之君！"
5. **给出解决方案/出路**：训斥后给出具体方向
   - "莫要陪那几个掌事试探…敢跟你磨磨蹭蹭一刀杀了就是"
   - "去眉尺山洞府好生想一想，突破胎息第四层再出来"

## 训斥的层次设计

| 层次 | 内容 | 情绪 |
|------|------|------|
| 指责（What） | 你做了什么／没做什么 | 愤怒 |
| 分析（Why） | 为什么这是错的（一刀见血） | 冰冷/清醒 |
| 警示（So What） | 不改正的后果是什么 | 震慑 |
| 方向（How） | 应该怎么做 | 引导 |
| 惩戒（Penalty） | 罚去修炼/反省 | 教育 |

## 核心台词设计

家族训诫的"核心台词"需要：
1. **可传播性**（读者能记住、转述）
2. **哲学高度**（不止于单个事件，而是上升到原则层面）
3. **符合人物性格**（李通崖是务实/果决/冷峻的性格）
4. **可成为家族/作品的标志性话语**

## 适用场景

- 晚辈掌家失败/犯错时
- 家族治理哲学需要正式确立时
- 角色的成长节点

## 注意事项

- 训斥不是单纯的骂——必须同时输出有价值的世界观/方法论
- 训斥后需要给出出路（方向+惩戒），否则显得长辈只会发火
- 训斥的强度要与错误的严重程度匹配
- "核心台词"的节奏需要卡在情绪的最高点
