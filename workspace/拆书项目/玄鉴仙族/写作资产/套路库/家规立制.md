# 套路：家规立制（宗法制度）

## 定义
家族在关键转型期制定/修订宗法规则，确立嫡庶、大宗小宗、灵窍继承等制度。

## 功能
- 确立作品内部的社会运行规则
- 体现家主的治理思维
- 为后续家族矛盾的埋下制度性伏笔

## 本卷范例（ch100 李通崖立家规）

### 规则内容
```
嫡系 + 庶出有灵窍者 → 大宗
庶出无灵窍者 → 降为小宗
小宗三代无灵窍者 → 降为支系
支系有灵窍者 → 回认祖系为大宗（但止步小宗）
叶氏（旁支）有灵窍者 → 止步小宗
```

### 制度设计的思维
1. **灵窍为唯一硬通货**: 无论出身，有灵窍可上升
2. **三代过滤机制**: 保证大宗人才密度
3. **支系天花板**: 防止旁支过度膨胀动摇大宗根基
4. **叶氏（秋阳支系）的限制**: 边缘化的隐性歧视

### 立法时机
- 家族的第三代（渊修）诞生时——标志着家族从"二代创业者"向"三代守成者"转型
- 战后危机期——制度变革往往在危机后更容易推行

## 叙事功能
- **世界观层面**: 让修仙家族的社会结构更真实
- **角色层面**: 展现李通崖的治理者思维
- **剧情层面**: 此制度将在后续引发大宗vs小宗/支系的矛盾（伏笔）

## 变体
- **分家立制**（如ch093万家分家）
- **联姻盟约**（无新家规，但重塑血缘秩序）
- **族老会议立规**（多角色博弈版）
