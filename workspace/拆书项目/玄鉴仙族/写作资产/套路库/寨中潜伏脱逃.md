# 套路库：寨中潜伏脱逃

> 归类：情节驱动 / 敌后求生

---

## 定义

角色（通常是低阶修士或凡人）被敌方俘虏后，伪装成普通奴隶/劳力潜入敌营内部，暗中侦察布局、寻找同伴、等待时机，最终利用环境变化或外部干扰，趁乱执行逃脱方案。关键在于"隐忍等待"和"外部机遇的捕捉"——不是硬闯，而是等待最薄弱的窗口一击脱离。

---

## 本卷实例（ch091-ch095 — 山越寨中逃脱）

| 项目 | 内容 |
|------|------|
| 被俘方 | 李玄岭（胎息二层）+ 李景恬（凡人） |
| 敌营 | 山越前线寨子——三位练气驻守，数百山越士兵 |
| 伪装方式 | 李玄岭装成普通苦工奴隶，颈画横纹，戴木枷，表演疲劳 |
| 潜伏时长 | 十几天——耐心等待窗口 |
| 信息收集 | 用法鉴定位李景恬位置，观察寨子布局，听部巫对话获取明日启程情报 |
| 外部机遇 | 李通崖正面杀到→空中大战吸引所有人注意→趁乱执行 |
| 逃脱手段 | 踢断枷锁→杀侍卫→越栏→金光术杀兵→神行术抱人钻林 |
| 应急处置 | 若被木焦蛮追上——用太阴玄光抵抗，最坏方案"先杀你再自杀" |

## 套路结构

```
┌─────────────┐
│ 被俘入营     │ ← 被敌方抓捕→押送→进入封闭环境
└──────┬──────┘
       ▼
┌─────────────┐
│ 伪装潜伏     │ ← 降低存在感/假装无害/融入奴隶群体
└──────┬──────┘
       ▼
┌─────────────┐
│ 等待窗口     │ ← 同时进行：信息收集+联系同伴+观察守卫规律
└──────┬──────┘
       ▼
┌─────────────┐
│ 倒计时触发   │ ← 某种不可逆事件（启程/转移/处刑）逼迫必须在时限内行动
└──────┬──────┘
       ▼
┌─────────────┐
│ 外部机遇     │ ← 友方正面牵制/意外混乱/天象变化→注意力被转移
└──────┬──────┘
       ▼
┌─────────────┐
│ 逃脱执行     │ ← 多动作连续执行（杀哨兵+破障碍+加速术+钻林）
└──────┬──────┘
       ▼
┌─────────────┐
│ 接应/脱离    │ ← 友方掩护撤退 / 利用地形隐蔽
└─────────────┘
```

---

## 写作要点

1. **潜伏不能太轻松**：要付出压抑的代价——李玄岭装累装苦十几天，忍受屎尿堆积的环境
2. **等待必须有理由**：为什么要等？——不是被动而是主动选择：等李景恬消息、等最佳时机
3. **外部机遇不能太刻意**：李通崖的杀到不是安排好的，而是家族营救的巧合——让逃脱有偶发感
4. **执行要快**：十几息完成整套动作——用动词堆叠制造节奏，不给读者和追兵思考空间
5. **要有最坏预案**："先杀你再自杀"——底线方案让人物的决心变得具体而沉重
6. **逃脱不是终点**：逃脱成功后的去向（汇合、继续躲藏、回程）要留悬念

---

## 注意陷阱

| 陷阱 | 说明 |
|------|------|
| 被俘方太容易逃脱 | 如果没有任何阻碍地逃脱，前期的"潜伏"失去意义——必须有实质风险 |
| 敌人太蠢 | 追兵需要展示合理的追捕能力，否则逃脱缺乏成就感 |
| 信息获取手段太外挂 | 法鉴的存在是合理外挂，但需要限制（三位练气在上时李玄岭不敢用） |
| 外部接应太及时 | 恰到好处的救援会降低紧张感——李通崖是来挑战的，不是专程来救的 |
