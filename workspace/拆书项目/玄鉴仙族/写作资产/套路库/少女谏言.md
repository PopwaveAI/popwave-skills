# 套路库：少女谏言

> 归类：人物弧 / 智识代偿

---

## 定义

年轻女性（通常未成年）以超越年龄的政治/军事/外交洞察力，在家族危机时向族长提供关键谏言。核心张力在于——她的智慧和判断力超越了所有在场成年男性，但由于性别/年龄/修为限制，仍需借"谏"的形式传达，而非直接决策。

---

## 本卷实例（ch086 — 李景恬）

| 项目 | 内容 |
|------|------|
| 角色 | 李景恬，十二岁 |
| 身份 | 李项平长女 |
| 修为 | 无（凡人）——无灵窍 |
| 洞察力表现 | ①一眼看穿青池宗与山越的交易逻辑 ②预判家族已成"交割的货物" |
| 谏言内容 | 献地求和——（实际是测试父亲底线） |
| 父亲反应 | 断然拒绝——李景恬"脸上绽放笑容"（测试通过） |

### 谏言分析

| 层次 | 内容 | 深度 |
|------|------|------|
| 第1层 | "我家恐成他人鱼肉，还是要早作打算" | 危机意识 |
| 第2层 | "青池宗与山越一族的大巫达成了共识" | 信息推断 |
| 第3层 | "不愿我李家作出反应" | 动机分析 |
| 第4层 | "有伤财劳民之险，却无灭族之患" | 风险评估 |
| 第5层 | 献地求和（实际测试） | 策略试探 + 价值确认 |

## 写作要点

1. **智慧必须合乎年龄**：十二岁可用敏锐、逻辑清晰来描述，但不能用几十年的经验智慧——李景恬的分析是逻辑推导而非阅历沉淀
2. **谏言需有层级**：从"有危险"→"原因"→"动机"→"风险评估"→"策略"，步步递进
3. **可以是测试而非建议**：献地求和不是李景恬真实主张，而是测试父亲是否足够刚烈
4. **谏言之后有行动呼应**：ch090李景恬冷静配合逃窜——证明她不只是会说
5. **性别/修为的前提限制不可忽略**："要是我也有灵窍"——智识无法弥补修炼的硬差距
