# 套路：少年立誓

## 核心公式

```
目睹/经历仇恨（触发事件） → 内心激荡（情绪积累） → 立下誓言（标志性台词） → 奠定人物基调（为后续复仇线做铺垫）
```

## 关键阶段

1. **触发事件**：少年目睹/经历亲人被害、家族蒙难等仇恨
   - 李玄锋在尸体中寻找父亲（ch074）
   - 看到万家五个村子全被屠杀
2. **情绪积累**：愤怒+无力感的混合
   - 少年还不足够强大，内心充满不甘
3. **立下誓言**：标志性的"等小爷长大了……"宣言
   - "待小爷长大了，定要杀光这镗金门，射下那少主的脑袋来当球踢！"（ch074）
4. **色调定调**：誓言的方向决定了人物后续的剧情走向
   - 李玄锋的誓言是全体仇杀——意味着他后续是一个"铁血复仇者"路线

## 少年誓言的层次

| 层次 | 内容 | 说明 |
|------|------|------|
| 对象 | "杀光镗金门" | 明确仇恨对象 |
| 目标 | "射下少主的脑袋" | 具体化复仇目标 |
| 方式 | "当球踢" | 带有少年气的暴力色彩 |
| 语气 | 自言自语/内心独白 | 真实、不加掩饰 |

## 情绪曲线

```
沉默（目睹惨状） → 内心压抑（听长辈讲话） → 内心独白（独处时的自语） → 誓言（情绪爆发）
```

## 叙事功能

1. **人物定性**：一句话定调这个角色后续的剧情线——读者对其有预期
2. **情感投资**：读者记住"这个少年以后要复仇"，形成期待的钩子
3. **代际对比**：展示李玄锋与李玄宣（守成/谨慎）的性格差异
4. **复仇线铺陈**：为长篇剧情中的远期中场爆发做铺垫

## 适用场景

- 介绍新角色/下一代角色时
- 需要给复仇线埋主线钩子时
- 角色需要"性格高光时刻"时

## 注意事项

- 誓言要符合年龄——李玄锋说"当球踢"是孩子的语言（不是"血债血偿"这种成年人的表达）
- 誓言不能太公式化——要有角色的个性烙印
- 立誓后需要后续章节的呼应——不能立了就忘
- 少年立誓需有后续成长（能力增长）来支撑——否则变成"白喊"
