# 套路库：山越入侵

> 归类：冲突驱动 / 战争推进

---

## 定义

以异文明（山越）对中原家族领地的大规模入侵为叙事引擎，推动家族应对、撤退、觉醒与成长。核心冲突不是善恶，而是生存——野蛮对文明的碾压侵入。

---

## 本卷实例（ch086-ch090）

| 项目 | 内容 |
|------|------|
| 入侵方 | 伽泥奚（北麓山越之王，练气中期，统一十万山越） |
| 被侵方 | 李家（青池宗治下家族，胎息巅峰族长） |
| 触发条件 | 青池宗与山越大巫达成秘密交易——默许山越入境劫掠 |
| 入侵规模 | 2000-3000精锐山越，四位练气人巫听命，几十位胎息部将 |
| 战术特征 | 声东击西（不打正面军阵→直掏黎泾山）、夜袭+合围、脚踝符文无声行军 |
| 时间窗口 | 青池宗给出的劫掠时间有限——“我们的时间不多” |
| 破坏维度 | 梨川口沦陷 → 黎泾村被占 → 黎泾山被搜 → 全族逃散 → 儿童被掳 |

## 写作要点

1. **入侵必须有成本**：青池宗的窗口期说明上层博弈存在——入侵不是无代价的
2. **引入方不能是纯恶**：伽泥奚是奴隶翻身，有合理动机，细节使反派立体
3. **损失必须可感**：陈三水之死+景恬玄岭逃难+儿童被掳——每个损失都有名字和面孔
4. **平行叙事**：入侵导致家族分成两路（西进vs藏匿），天然制造悬念剪辑
5. **时间压力**：窗口期=倒计时=叙事紧迫感
