# 套路：山越劫掠

## 核心公式

```
山越内部权力斗争（被更强部族驱逐） → 被迫迁移东逃 → 发现目标（中小型家族/村落） → 抓探子获情报 → 夜袭劫掠 → 遭遇抵抗 → 抢够就走/被击退
```

## 关键阶段

1. **困境根源**：山越入侵往往不是因为穷，而是因为被更强的部族驱逐
   - 阿会剌部被伽泥奚驱逐——三千人大寨被毁（ch078）
   - "回头你去和伽泥奚那条恶狼拼命？"——没有退路
   - 山越不是单纯的"蛮子"——有他们自己的权力结构和生存逻辑

2. **目标选择**：挑软柿子捏
   - 阿会剌从青池宗/魔门的地盘一路避开，找到李家
   - "管他娘的！先抢了再说！"——先试探再决定力度
   - 需要情报支持——抓探子/严刑拷打

3. **劫掠方式**：散开劫掠、速战速决、不恋战
   - "散入村落中四处劫掠，已经没有任何队形可言"
   - 吹号收兵——抢不够就走
   - 核心原则：**保存部众 > 抢到财物**

4. **遭遇抵抗**：被目标家族的武装力量反击
   - 李项平族兵赶到，山越被咬下近百人（ch079）
   - 鸣金收兵——不入山林追击

5. **后果**：可能被反攻——如果目标家族有"有仇不过夜"的性格
   - Ch079-ch080的夜袭反攻——从劫掠者变为被围剿者

## 山越设定要素

| 要素 | 描述 | 示例 |
|------|------|------|
| 首领纹身 | 脸颊花纹代表修为等级 | 阿会剌胎息四层=四道纹 |
| 装饰 | 兽牙/玉石——石器时代的质感 | "身上的兽牙与玉石相互碰撞" |
| 战斗风格 | 赤裸上身、臂上纹图案、挥舞长刀 | "赤裸着上身，臂上纹着图案" |
| 信仰系统 | 族巫——负责祭祀、拷问 | "族巫拖着这人下去，毒虫蜂针一一安排上" |
| 组织 | 大寨（三千人级）→小寨（九百人级） | 阿会部从三千人被毁到九百残部 |
| 内部环境 | 弱肉强食——王者统一中 | "伽泥奚生来就是一统北麓山越的狼王" |

## 叙事功能

- **制造危机**：打破主角家族的和平日常——进入战斗周期
- **展示家族武力**：通过与山越的实战展示家族军事实力
- **扩展世界**：引入山越世界的设定——伽泥奚等大反派
- **角色成长契机**：战争逼迫角色成长（李玄宣的首次杀戮）

## 适用场景

- 家族需要一次"战斗经验"但不宜引入宗门级大冲突
- 需要引入外族视角丰富世界观
- 需要"非内耗"的战斗——抗击外族可以团结内部

## 注意事项

- 山越不是单纯的"蛮子"——要有自己的文化和生存逻辑
- 要有山越内部的权力斗争线——入侵只是结果，不是起因
- 山越的"散开劫掠/不成队形"和主角家族"组织严密的族兵"形成战术对比
- 山越线的反派（伽泥奚）可以成长为后续章节的大Boss
