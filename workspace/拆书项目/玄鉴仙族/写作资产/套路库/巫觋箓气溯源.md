# 套路库：巫觋箓气溯源

> 归类：世界观设定 / 力量体系揭秘

---

## 定义

在修仙世界观中，法鉴类神器发现异族/敌对势力的某种法术（巫术、咒术、祭祀术等）与自己掌握的箓气/符种体系存在同源性——表现为"弱化版"、"魔改版"或"山寨版"的箓气运用。这揭示了两个重要信息：①该法术体系真正的源头是法鉴（或类法鉴的更高存在）；②法鉴对这类"衍生法术"拥有天然的权限压制/控制权。这种发现往往推动主角（陆江仙型法鉴持有者）从"被动观测"转向"主动研究"自身法器。

---

## 本卷实例（ch094-ch095 — 陆江仙发现巫觋术与法鉴箓气同源）

| 项目 | 内容 |
|------|------|
| 契机 | 李玄岭用法鉴定位时，陆江仙感应到大厥庭方向有黄箓（金丹级）气息 |
| 首次接触 | 木焦蛮施巫觋咒术——黑气顶着硕大头骨追袭李通崖 |
| 陆江仙反应 | "黑气却熟悉得很，隐隐约约有种法鉴出品的味道"——本能识别 |
| 控制能力 | "勾一勾手指便能把这黑气牵引过来吃掉"——法鉴对箓气衍生品的绝对权限 |
| 干预决策 | 选择"装死"但偷控——黑气被压制，李通崖几乎不受影响 |
| 结构分析 | "像箓气，却又差了太多"——注意到劣化特征 |
| 本质总结 | "将《牲祭法》中祈求的对象从法鉴改换成了施法者本人"——源流关系的理论化 |
| 后续规划 | "有空倒也可以研究研究这东西"——从被动→主动探索 |
| 连带发现 | 大厥庭方向的金丹黄箓——指向更高层级的势力背景 |

┌─────────────┐
│ 偶然接触     │ ← 法鉴持有者目睹/感应到异族法术
└──────┬──────┘
       ▼
┌─────────────┐
│ 本能识别     │ ← "熟悉的味道"——看似不同的法术与箓气产生共鸣
└──────┬──────┘
       ▼
┌─────────────┐
│ 权限验证     │ ← 尝试控制/牵引→发现该法术可以被法鉴完全压制
└──────┬──────┘
       ▼
┌─────────────┐

       ▼
┌─────────────┐

       ▼
┌─────────────┐
│ 世界观扩展   │ ← 发现更高层级的存在（本卷：金丹黄箓/背后势力）

       ▼
┌─────────────┐
│ 主动研究     │ ← 从被动观测者→主动探索者
└─────────────┘

## 写作要点

1. **识别要有自然触发条件**：陆江仙不是主动搜索——是在"看热闹"中自然发现黑气"熟悉"
2. **权限压制要有视觉表现**：黑气被控制后"附着在李通崖衣物上动也不动"——简单的画面描述胜过效果说明
3. **"山寨版"要有逻辑**：巫术为何比箓气弱？因为失去了法鉴作为转化中心，施法者只能消耗自身精气
4. **发现不能一次性揭秘**：ch094发现黄箓但决定不动→ch095分析黑气但只做初步推理——信息逐步释放
5. **主角价值观要一致**：陆江仙选择"装死"而不是暴露——符合他谨慎/低调的一贯性格
6. **溯源要带出更大的世界**：黄箓的出现意味着金丹级势力——法鉴的来历/李江群疑案/三宗七门之上的力量——世界观从"家族恩怨"扩展到"天地棋局"

## 注意陷阱

| 陷阱 | 说明 |
|------|------|
| 解释过于学术化 | 不要大段引用设定文本——陆江仙的分析是通过"感觉+简练推理"完成的，读者消化不累 |
| 法鉴太万能 | 权限压制不能太夸张——本卷中陆江仙只能控黑气，不能直接击溃木焦蛮 |
| 一次性揭秘 | 不要在一两章内把源流关系说完——留白和悬念比全盘交代更有档次 |
| 主角失去谨慎 | 发现金丹黄箓后如果陆江仙立刻冲过去——性格断裂+找死 |
