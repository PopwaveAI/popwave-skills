# 套路：战利品清点

## 核心公式

```
战斗/反杀结束 → 倒出战利品 → 逐件检查/评估价值 → 讨论用途 → 家族储备增强（经济闭环）
```

## 关键阶段

1. **回归安全地点**：战斗结束后回到安全环境
   - 李项平回到黎泾村李家正院（ch075）
2. **倾倒清点**：将储物袋内容全部倒出，展示"收获时刻"
   - "李通崖点点头，灵识探入储物袋中，哗啦啦地倒出一小堆物品来"（ch075）
3. **逐件评估**：每件物品的来历、价值、用途讨论
   - 灵石：25枚→家族储备（ch075）
   - 玉芽丹：练气期丹药，比蛇元丹更强（ch075）
   - 符箓：24张护盾符，≈8灵石（ch075）
   - 长剑：胎息法器，3-5灵石（ch075）
   - 金属块：未知，待鉴定（ch075）
4. **特殊物品**：其中一件触发新的事件/悬念
   - 玉佩→法鉴吞噬（ch075）
5. **用途规划**：分配/规划战利品的使用方向
   - "可以留些给玄宣玄锋防身"
6. **经济闭环**：完成从"战斗消耗→战利品补偿→家族增强"的经济循环

| 物品 | 数量 | 价值评估 | 用途/去向 | 说明 |

## 叙事功能

1. **经济闭环**：战斗的投入（符箓消耗、受伤）→ 产出（战利品）的核算
2. **奖励感**：读者跟着体验"数钱"的快感
3. **展示角色性格**：对不同物品的态度展现角色特质
   - 李通崖："爱不释手地拿起那长剑仔细研究"（装备控）
   - 李项平："老道十二道符箓唤出大盾，连被我骗了两次，难怪恼羞成怒"（得意/幽默）
4. **世界观信息**：通过物品的价值展示修仙经济体系
5. **埋设悬念**：特殊物品（金属块/玉佩）为后续做铺垫

## 适用场景

- 战斗/冒险后的奖励章（节奏需要慢下来）
- 需要展示家族/角色的经济状况时
- 引入新装备/新物品时

## 注意事项

- 清点应有主次——最重要的物品/最有趣的物品放在中间（ch075玉佩放中间）
- 物品数量不宜过多（5-7件为宜）——太多会像流水账
- 至少一件物品需要触发新的事件/悬念（玉佩→法鉴吞噬）
- 清点后的讨论可以展示角色关系和未来规划
