# 套路：战后穿过

## 核心公式

```
战争/屠杀发生后 → 主角穿越战场残骸 → 感官冲击（视觉/嗅觉） → 内心拷问 → 对系统/体制的讽刺
```

## 关键阶段

1. **战争残留**：大屠杀后的战场场景
   - 蕈林原——镗金门南下后的惨状（ch074）
2. **感官冲击**：用最直接的感官描写（尤其是嗅觉）让读者身临其境
   - 尸臭：雨水浸泡+太阳暴晒，浓烈到牛不干活（ch074）
   - 视觉：肿胀青黑的尸体、鼓起的气球状皮肤
3. **孤独穿行**：主角独自在尸体间行走，无人陪伴
   - 李通崖丢下牛车徒步，在尸体身上走着（ch074）
4. **幸存者相遇**：与战场幸存者的互动
   - 不会说话的村丁从尸堆中爬出
   - 三三两两的孩子缀在后面
5. **体制讽刺**：主角内心对造成屠杀的系统/权力结构进行批判
   - "青池宗会怎么记载？"（ch074）
6. **接续叙事**：穿过后发现与主线相关的线索/人物
   - 发现李玄锋在尸体中扒拉（ch074）

## 感官描写策略

| 感官 | ch074示例 | 效果 |
|------|----------|------|
| 嗅觉 | "好浓烈的尸臭""牛儿也被臭得不干活了" | 最直接、最具冲击力 |
| 视觉 | "皮肤上鼓起一个个大气球""呈青黑色，血肉在里头溃烂" | 画面感强烈 |
| 触觉 | "搬开道上的积尸" | 近距离接触死亡的沉重 |
| 听觉 | "抱着头哭泣" | 幸存者的绝望 |

## 情绪曲线

```
平静(进入战场) → 恶心/不适(感官冲击) → 麻木(持续穿行) → 愤怒/讽刺(内心独白) → 心疼/紧张(发现幸存者) → 进入新阶段
```

## 叙事功能

1. **具象化代价**：将"战争残酷"从抽象转化为具体的感官体验
2. **道德拷问**：主角/读者都要面对"这些惨案是谁造成的"的质问
3. **体制批判**：通过主角的内心独白输出世界观批判（ch074的"青池宗会怎么记载"）
4. **情感铺垫**：为后续角色行动（立誓/复仇）提供情感基础
5. **氛围切换**：从"紧张行动"切换到"沉重反思"的节奏缓冲

## 适用场景

- 大屠杀/战争后的过渡章节
- 需要强制主角（和读者）面对战争真实代价时
- 为主角的"黑化"/"坚定"提供情感trigger

## 注意事项

- 感官描写要以嗅觉为核心——视觉可以闭眼不看，味道无法逃避
- 长度要控制——过长会让读者感到疲劳，ch074控制在开局段落
- 战后穿过不能只是"惨状展示"——必须有叙事推进（发现线索/人物/转折）
