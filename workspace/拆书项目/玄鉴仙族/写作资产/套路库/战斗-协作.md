# 套路：战斗·协作

## 定义
多人协同作战，通过明确的分工和战术配合，以弱胜强或高效击杀强敌。核心是**1+1>2的协同效应**——各司其职、互相掩护、时机配合。

## 适用场景
- 多角色共同面对强敌
- 家族/队伍的战斗配合展示
- 远程+近战、牵制+输出的经典组合

## 常见协作组合
| 组合 | 说明 |
|------|------|
| 近战牵制 + 远程狙击 | 一人正面吸引注意，一人远程精确打击 |
| 主攻 + 辅助 | 一人主攻，一人法术加持/控制 |
| 阵地 + 骚扰 | 一人守住关键位置，一人灵活穿插 |
| 诱饵 + 陷阱 | 一人引诱敌人进入预设陷阱区域 |

## 结构模板
1. **战术部署**：战前明确各自位置和职责
2. **开战执行**：各角色到位，开始配合
3. **节奏变化**：根据敌人反应调整战术
4. **致命一击**：把握对方破绽，协同完成击杀
5. **复盘总结**：战后讨论配合的优劣

## 案例
**ch133**：李通崖+李玄锋猎熊：
- 李通崖（正面/近战）：拔剑引诱熊罴，用剑气消耗和牵制
- 李玄锋（远程/狙击）：在对面山头上拉弓蓄力，等待李通崖创造机会
- 配合节点：熊罴被李玄锋箭吸引注意力时，李通崖近身斩出月阙剑弧
- 追击配合：熊罴欲逃，李通崖追击牵制，李玄锋第二箭命中

## 注意
- 协作战斗需要清晰的"空间关系"描述（谁在哪儿、距离多远）
- 不同角色的战斗风格应通过配合形成节奏（快慢交替）
- 避免写成各自单挑——真正的配合是互相创造时机
