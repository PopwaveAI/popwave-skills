# 套路：战略拖延（拖字诀）

## 定义
当实力不足或时机未到时，通过虚与委蛇、缓兵之计、以商议为由等方式推迟决策/行动，争取成长时间。

## 叙功能
- 解决"当前困境无法立即解决"的叙事难题
- 建立角色的战略思维和耐心
- 为后续突破做预告（"等我方变强后再处理"）

## 本卷实例（ch045）

### 困境回顾
李家无法满足万家的交易条件（提供练气巅峰/筑基符箓），因为：
1. 法鉴太阴玄光只能"诛邪除妖"，无法主动攻击人类修士（汲登齐）
2. 李家没有人具备画制这种符箓的能力
3. 编造"符箓"之说已用在一处，无法重复使用

### 拖延执行细节

| 步骤 | 话术/行动 | 效果 |
|------|----------|------|
| 1. 口头婉拒 | "非是通崖不愿交换，实是不能" | 万天仓不信，追加筹码 |
| 2. 留有余地 | 面对心动的条件，不再直接拒绝 | 万天仓认为有希望 |
| 3. 推迟决策 | "还须同父亲弟弟一同商榷" | 万天仓满意而去，认为稳了 |
| 4. 内部定策 | 决定"拖"，不拒绝也不给出明确时间 | 争取2-3年发展窗口 |

### 拖延的战略价值
> "待到我李家出了玉京轮……在此之前，便先让那汲登齐替我等压着万家"

1. **为自己的成长争取时间**：2-3年内尝试培养玉京轮修士
2. **维持外部压力平衡**：让汲家继续压制万家，防止万家调头来找李家麻烦
3. **保护秘密**：不交易就不用暴露"符箓是假的"
4. **保留选择权**：等到李家实力对等时，谈判地位完全不同

### 写作要点
| 要素 | 说明 |
|------|------|
| 合理性 | 拖延必须有理有据，不能是角色不想干 |
| 对手反应 | 被拖延方不会永远等下去，要给读者一种"拖但只能拖一阵"的紧迫感 |
| 明确目标 | 拖到什么时候/拖到达到什么条件——ch045明确"玉京轮" |
| 风险提示 | 拖延不是万能的——如果汲家吞并万家，李家就失去了制衡的棋子 |

### 变体与创新
- 非被动躲避，而是**主动利用外部势力（汲家）牵制交易对手（万家）**
- 拖延的同时制定清晰的下一步目标（玉京轮解锁法鉴能力）
- 章题即是核心策略（"拖"），高度凝练
