# 套路库：战略西进（断尾求生）

> 归类：情节驱动 / 军事决策

---

## 定义

当家族主力被敌军截断后路、困于险地时，族长不选择率兵回救（大概率中伏），而选择反直觉方向——向更危险的未知地带"西进/撤退"，以空间换时间，以撤退求生存。

---

## 本卷实例（ch089 — 李项平西进）

| 项目 | 内容 |
|------|------|
| 决策者 | 李项平（胎息巅峰，李家第三代家主） |
| 局势 | 驻军河对岸→梨川口被控→后路被断→两千山越在对岸 |
| 敌人兵力 | 伽泥奚2000-3000 + 自身练气中期 + 4位练气人巫 |
| 我方兵力 | 800族兵 + 200山越兵 = 1000人 |
| 选项A | 向东渡河夺回梨川口（正面决战=大概率被灭） |
| 选项B | 原地固守（补给断绝=等死） |
| 选项C（选中） | 西进密林（敌人后方不稳+可能出现转机） |

### 决策逻辑

```
伽泥奚大破敌军 → 立刻奔袭而来 → 
推测他后方必定不稳（补给线+老家防卫空虚）→
西进可避其锋芒+有可能找到机会 → 
每在原地待一刻就越危险
```

---

## 套路结构

```
┌─────────────┐
│ 陷入绝地     │ ← 后路被断，正面不可敌
└──────┬──────┘
       ▼
┌─────────────┐
│ 选项排除     │ ← A正面=死 / B死守=死
└──────┬──────┘
       ▼
┌─────────────┐
│ 反直觉决策   │ ← C.向看似更危险的方向前进
└──────┬──────┘
       ▼
┌─────────────┐
│ 战术支撑     │ ← 装粮入储物袋（后勤准备）
└──────┬──────┘
       ▼
┌─────────────┐
│ 拔营西进     │ ← 头也不回，全军向西
└──────┬──────┘
       ▼
┌─────────────┐
│ 对手落空     │ ← 伽泥奚到黎泾山扑空
└──────┬──────┘
       ▼
┌─────────────┐
│ 后效待定     │ ← 李项平西进的结果待后续展开
└─────────────┘
```

---

## 写作要点

1. **决策必须有紧迫性**：后路被断的消息一传来，李项平脑子里飞速排除了两个选项
2. **不能是无脑跑**：要有储物袋装粮的细节说明——这是有组织的战略撤退
3. **西进要有理由**：伽泥奚"后方必定不稳"——不是逃跑，是战术判断
4. **与另一线形成平行悬念**：李项平向西 vs 伽泥奚向东——两条线交叉好看
5. **决策者要有心理刻画**：火盆倒映在眼中→心中漏跳一拍→攥刀捏得吱吱作响——小细节暴露内心紧张
