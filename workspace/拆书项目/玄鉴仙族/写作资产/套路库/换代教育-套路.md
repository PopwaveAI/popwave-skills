# 套路：换代教育

## 核心公式

```
晚辈犯错/不足 → 长辈发现 → 指出问题 + 施加惩戒 → 给方向/出路 → 晚辈受教/成长
```

## 关键阶段

1. **犯错发现**：长辈发现晚辈在治理/修炼/行为上犯了错
   - 李通崖发现李玄宣掌家期间不作为（ch074）
2. **当面训斥**：指出具体错误，不绕弯子
   - "万家覆灭了足足一月有余……你叔父失踪月余……让锋儿独自溜到了汲家地界上！"
3. **权力本质教育**：输出治理/生存的核心原则
   - "我李家的权力来源于刀兵与修为"
4. **惩戒**：一定程度上的惩罚，推动成长
   - "去眉尺山洞府好生想一想，突破胎息第四层青元轮再出来"
5. **受教/成长**：晚辈接受教育（或后续表现展现成长）
   - 李玄宣"反倒像是松了口气般恭恭敬敬地退下了"

## 惩戒的类型

| 类型 | 特点 | ch074示例 |
|------|------|----------|
| 闭门修炼 | 不惩罚而是"放你去修炼"——将错误转化为成长动力 | 罚去眉尺山洞府突破 |
| 剥夺权力 | 暂时收回管理权限 | 让项平/通崖重新接手 |
| 边事磨砺 | 派到风险边缘去实战锻炼 | — |

## 换代教育的心理机制

```
晚辈：犯错愧疚 → 面临训斥 → 恐惧（但不怨恨）→ 释然（给了出路）
长辈：发现错误 → 愤怒 → 压住情绪输出教育 → 给出方向
```

关键：训斥后必须**给出路**——让错误变成成长的机会，而不是单纯的惩罚。

## 适用场景

- 换代期（老一代传位给新一代的过渡阶段）
- 晚辈第一次独立处理事务失败
- 家族/宗门治理哲学的传承

## 注意事项

- 训斥要够狠（否则没有教育效果），但后续要有建设性（否则只制造仇恨）
- 惩戒要与错误相关——李玄宣的问题是不作为/不探查，惩罚是去修炼（反向：让他变强）
- 长辈的家族哲学输出需要真实可信——不能是说教套话
