# 套路：摊牌博弈/核威慑谈判

## 定义

两个势力在相互依存且相互威胁的状态下，通过"亮底牌"式的极限谈判达成脆弱的平衡。特点是将潜在冲突的表面平静打破，直接讨论最坏结果。

## 功能

- 打破长期拖延/暧昧状态，推动剧情进入新阶段
- 展示双方智谋、底牌、承受力的较量
- 为读者提供"信息差快感"——底牌一张张翻开的节奏控制
- 揭示双方的底线与恐惧

## 关键要素

| 要素 | 说明 |
|------|------|
| **突破表态** | 一方以"话说到这份上了"的态度跳出客套（卢思嗣："明人不说暗话！"） |
| **B计划展示** | 提出如果谈判破裂的替代方案（"将华芊山献给郁家"） |
| **暴力反制** | 对方也提出极端方案（"不怕我将你留在这"） |
| **反击底牌** | 展示让对方投鼠忌器的筹码（月湖峰香火情） |
| **交易确认** | 从"要不要谈"进入"怎么谈"阶段 |
| **情感和解** | 谈判完成后的一方致谢/感慨——人性化的收束 |
| **内心保留** | 双方私下都知道这只是权宜之计（李通崖暗道"整垮卢家有太多办法"） |

## 适用章节

- ch118（交底订亲）：李通崖 vs 卢思嗣的完整谈判

## 变体

- **正面碾压**：强势方不给谈判空间，直接提出最终条件
- **暗中布局**：表面友好，私下做最坏准备
- **三方博弈**：引入第三方势力作为谈判筹码（本案例中的"郁家"作为窗外大象）
