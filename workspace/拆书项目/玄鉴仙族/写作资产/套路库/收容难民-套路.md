# 套路：收容难民

## 核心公式

```
强族覆灭 → 遗孤逃亡（一群孩子/残部）→ 来投/被收容 → 家主决定 → 打散安置 → 纳入家族体系
```

## 关键阶段

1. **覆灭背景**：一个家族/部落被摧毁——通常是突袭或宗门争斗
   - 万华芊山被镗金门攻破，万家大阵轰然破灭（ch077）
   - 阿会部三千人大寨被伽泥奚摧毁（ch080回忆）
   - 难民是"失根"的一群人——对新的归属有很强的依附需求

2. **逃亡姿态**：难民的心态——悲伤/绝望/仇恨
   - 万天景"心如死灰地走着"（ch077）
   - 孩子们"不吵也不闹"——被巨大变故压得失去童真
   - 改名明志——"万天景"→"万天仇"（仇恨驱动）

3. **收容决策**：家主的决定——接纳还是驱逐？
   - 李项平的决定：收容、打散编入各村、孤儿寄养
   - 仁厚气度的展现——但也有实用考量（人力、阵法知识）
   - **关键分寸**：不能直接纳入核心——打散安置防止反噬

4. **分类安置**：不同年龄段不同处理方式
   - 十岁以下 → 寄养书院/找人家
   - 十几岁 → 分地独自成户
   - 为首修士 → 单独面见、评估、使用

5. **人才评估**：家主对难民中修士的判断
   - "懦弱无谋，但有几分情义，可以一用"（李项平评万天仇）
   - "远不如万天仓"——与已死的精英对比
   - 决定难民在家族中的位置

## 难民类型

| 类型 | 心态 | 使用方式 | 示例 |
|------|------|---------|------|
| 仇恨驱动型 | 改名明志，复仇为动力 | 实用但需警惕（可能引祸） | 万天仇 |
| 认命型 | 接受现实，安心归附 | 可直接编入生产 | 普通难民 |
| 卧薪尝胆型 | 表面归顺，心怀异志 | 需隔离/监控 | — |

## 叙事功能

- **壮大家族**：人口/劳动力/知识（万天仇的阵法知识）的补充
- **展露气度**：收容难民体现家族仁厚——区别于"趁火打劫"
- **引入新角色**：难民可能成为未来的重要配角
- **埋下隐患**：仇恨/复仇/未来风险——收容的人可能带来麻烦

## 适用场景

- 家族发展壮大期——有余力也有需求吸纳外部人口
- 附近强族被灭——"捡漏"吸收其残部
- 家族需要非血缘修士（阵法师/符箓师等专业人才）

## 注意事项

- 收容要"打散"——不能让难民抱团形成独立势力
- 为首的修士要单独面对、单独评估——不能混入群众
- 难民的心理状态要写出来——不是工具人，有属于自己的悲剧
- 改名是重要的心理标志——"旧我已经死了，新我诞生"
