# 套路：政治婚姻/联姻交易

## 定义

两个修仙家族之间以婚姻为形式达成的政治交易。本质是利益交换（土地、安全、传承）而非情感结合。

## 功能

- 解决地缘/安全危机（本案例：阻止郁家南进）
- 建立/加固家族联盟
- 展示角色在家族责任与个人意愿之间的取舍
- 为后续剧情埋线（联姻后的权力分化）

## 关键要素

| 要素 | 说明 |
|------|------|
| **摊牌环节** | 一方以"要联姻/要翻脸"的极端选择逼对方表态 |
| **筹码展示** | 正反双方亮出底牌（香火情、势力威胁、宗门关系） |
| **妥协条件** | 双方各退一步，达成可接受的交易条款 |
| **人选确定** | 哪一房的子女出嫁/入赘？嫡出还是庶出？——反应家族内部价值排序 |
| **见面确认** | 联姻对象登场，相貌/资质/器量评估 |
| **情感注脚** | 角色内心对"委屈两人了"的无奈与愧疚 |

## 适用章节

- ch118（交底订亲）：完整流程——李玄岭娶卢婉容
- 后续可复用此结构（不同家族、不同力量对比下的变体）

## 变体

- **强方压弱方**（本案例）：李家略强于卢家，联姻是卢家求自保
- **平等联姻**：实力相当的两个家族联合对抗第三方
- **弱方联姻强方**：小家族嫁女入大家族以获取庇护
- **反类型**：联姻最终破裂，引发更激烈的冲突
