# 套路：旅途中继

## 核心公式

```
出发在路上 → 途径驿站/酒肆/城镇 → 偶遇新人物/获取信息/遭遇事件 → 补充后继续上路
```

## 用途

用于长途旅程中的中间站。核心功能是在不推进主线的情况下丰富世界观，同时可以通过巧遇引入新角色或信息。

## 关键元素

| 元素 | 功能 | 示例（ch047） |
|------|------|-------------|
| 特色美食 | 记忆锚点+展现风土 | 雪花酒 |
| 市井人物 | 展示社会底层视角 | 吴老汉、壮汉们 |
| 偶遇贵人 | 引入新角色/信息 | 萧雍灵 |
| 反差场景 | 贵人与破店的碰撞制造趣味 | 萧雍灵进破旧酒肆 |

## 情绪节奏

```
旅途疲惫(慢) → 到达驿站的舒适(中) → 品美食的满足(慢) → 
旁听的轻松(中) → 偶遇的意外(快) → 交谈的信息获取(慢) → 告别再上路(中)
```

## 变种

| 变种 | 差异 | 效果 |
|------|------|------|
| 纯休息 | 无偶遇，侧重美食和环境 | 放松节奏 |
| 信息中继 | 偶遇带来重要世界观信息 | 为下阶段铺垫 |
| 冲突中继 | 在驿站发生矛盾/打斗 | 制造紧张 |
| 人情中继 | 驿站主人与主角有旧情 | 温情调剂 |

## 适用场景

- 长距离旅行中需要节奏变换
- 需要在不推进主线的情况下填充内容
- 要为后续引入新角色做铺垫
