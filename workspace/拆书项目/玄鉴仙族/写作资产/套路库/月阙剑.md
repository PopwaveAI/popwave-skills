# 月阙剑

## 定义
李通崖的剑道称号与形象符号。

## 特征
| 维度 | 描述 |
|------|------|
| 外形 | 灰袍，缓且长眉毛，两颊削瘦，肩膀宽大 |
| 佩剑 | 腰悬一剑，藏鞘中看不见形态 |
| 气度 | 雄远、稳重、成熟 |
| 性格初现 | 谨慎、温和但严谨 |
| 在家族中地位 | 李家实际掌舵人（仲父） |
