# 套路：法器认可/祭祀

## 基本信息

| 项目 | 内容 |
|------|------|
| 套路编号 | R007 |
| 套路名称 | 法器认可/祭祀 |
| 核心功能 | 家族法器展现神异，建立祭祀体系，强化法器与家族的联系 |
| 首次出现 | ch054-ch055 |
| 出现频率 | 低（关键节点） |

## 套路结构

### 1. 触发条件
- 使用者修为达到某个门槛（如玉京轮诞生灵识）
- 法器有灵/有器灵（如陆江仙）

### 2. 标准流程

```
使用者修为达标 → 尝试接触/探查法器(用灵识) →
法器有反应(白芒/月华/震动) → 展现神异场景 →
建立祭祀制度(仪式/祭品/口诀) → 法器与家族深度绑定
```

**ch054-ch055应用实例：**
- 门槛：李通崖突破玉京轮，诞生灵识
- 接触：灵识投入镜中
- 反应：陆江仙被"戳了一下"，配合放出白光
- 神异场景：月华喷涌、弥漫小屋、从窗户溢出
- 建立制度：
  - 口诀：承法要令
  - 祭品：灵米瓜果
  - 操作：下拜恭请
  - 效果：太阴玄光
- 深度绑定：玄珠符种共鸣，李通崖李项平李玄宣同时感应

### 3. 关键元素

| 元素 | 说明 |
|------|------|
| **修为门槛** | 法器不会在主角太弱时完全展现——维持神秘感 |
| **器灵视角** | 如果有器灵，要给出器灵的内心活动 |
| **视觉奇观** | 神异展现需要强烈的视觉描写（月华流淌、白光） |
| **仪式感** | 祭祀过程不能太随意，要有庄严的仪式 |
| **成本设计** | 使用法器需要消耗（灵物/篆符/时间）——防止滥用 |
| **长远规划** | 器灵/设计者要思考"再过几代会怎样"——神圣化过程 |

### 4. 注意事项
- 法器的神异展现要有渐进性——不是一次性全部暴露
- 祭祀制度的设计要有合理性：为什么是灵米瓜果？为什么要有口诀？
- 器灵的内心活动要有深度——不能只是"好厉害"
- 要有成本意识：每次使用消耗什么？恢复需要什么？
