# 套路：法鉴反杀

## 核心公式

```
绝境（力量对比悬殊） → 诱敌/示弱（制造触发条件） → 触碰禁忌（碰法鉴） → 法鉴觉醒（十二篆符齐亮） → 太阴玄光（反杀）
```

## 关键阶段

1. **绝境建立**：力量对比悬殊，主角被碾压
   - 李项平胎息境 vs 老道练气境
   - 符箓用尽，内腑受伤（ch073）
2. **示弱诱敌**：假装不支/失败，引诱敌人接近
   - 李项平假装无力再攻击
   - 法鉴脱手而出（故意/意外？半真半假）
3. **触碰禁忌**：敌人对法鉴使用不当的手段（拂尘抓取/直接触碰），触发禁忌
   - 老道用拂尘去捉鉴子（ch073）
   - 法鉴有灵性——"如同有个不自量力的小孩拿起了桌面上的筷子"
4. **法鉴觉醒**：青灰色鉴子浮空，十二道篆符逐一闪亮，威压释放
   - 震慑在场所有人（包括主角）动弹不得
5. **太阴玄光**：白色光芒倾泻而下，敌人被化尽
   - 老道只来得及发出半个音节

## 触发条件

| 条件 | 说明 | 反例（不会触发） |
|------|------|------------------|
| 直接触碰鉴体 | 触碰鉴身才触发 | 法术攻击不触发？ |
| 用外物抓取 | 拂尘/绳索等外物接触鉴体 | 李项平正常持握不触发 |
| 具有攻击意图 | 抓取时的心态有"征服/夺取"意味 | 正常请鉴无问题 |
| 鉴主允许范围外 | 非授权者的不当操作 | 持鉴者本人持有安全 |

## 情绪曲线

```
绝望(力量悬殊) → 蓄势(诱敌) → 悬疑(法鉴脱手) → 禁忌触发(紧张) → 觉醒(震撼) → 反杀(爽)
```

## 与"太阴玄光破阵"的区别

| 维度 | 太阴玄光破阵（ch050） | 法鉴反杀（ch073） |
|------|-------------------|-----------------|
| 触发方式 | 主动使用太阴玄光 | 被动觉醒（禁忌触发） |
| 节奏 | 蓄力→释放 | 绝境→触碰→觉醒→释放 |
| 威力 | 破阵级 | 震慑练气+化尽 |
| 角色参与度 | 陆江仙主动控制 | 陆江仙发现新能力+观察升级 |

## 适用场景

- 主角（或主角势力）实力不足时需要反杀强敌
- 需要展示法鉴的"神异"特性时
- 需要回收之前铺垫的"法鉴忌讳"设定时

## 注意事项

- 禁忌触发需要提前铺垫（ch070狼妖夜铺垫了"鉴子忌讳被触碰"）
- 法鉴觉醒的视觉描写要足够震撼（十二篆符+如小太阳）
- 反杀后要留下物质收益（战利品）——否则显得白打了
