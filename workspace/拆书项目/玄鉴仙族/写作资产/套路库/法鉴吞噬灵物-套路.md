# 套路：法鉴吞噬灵物

## 核心公式

```
灵物出现 → 法鉴感应（陆江仙警觉） → 主动吸引（灵物自行飞向法鉴） → 吞噬/吸收（白气涌入） → 篆符齐亮 → 陆江仙升级
```

## 关键阶段

1. **灵物出现**：某种与法鉴有渊源/共鸣的灵物被带入鉴子附近
   - 泛光玉佩从老道储物袋中掉出（ch075）
   - 距离鉴子大约一寸
2. **陆江仙感应**：在鉴子中半睡半醒的陆江仙突然精神一振
   - "原本在鉴子中半睡半醒的陆江仙便打起了精神"
3. **主动吸引**：法鉴顺着感应"轻轻一引"，灵物自行飞向鉴身
   - "那枚玉佩顿时像一只欢快的飞鸟般跃起，一头向青灰色的法鉴上撞去"（ch075）
   - 注意：是灵物**主动**撞向鉴子，不需要外力
4. **吞噬过程**：白气从灵物中蓬勃而出，没入镜中
   - "一道道白气蓬勃而出，如同倦鸟归林般没入镜中"（ch075）
5. **成长期象**：十二道篆符逐一闪亮，神识范围扩大
   - "镜身边缘的十二道篆符逐一闪亮，陆江仙只觉得通体舒畅，神识范围不断扩大"（ch075）
6. **升级完成**：灵物变为普通状态（功能被抽离），落回桌面

## 吞噬视觉描写要素

| 要素 | 作用 | ch075示例 |
|------|------|----------|
| 灵物自主性 | 显示法鉴的"吸引力"而非"攻击性" | "像一只欢快的飞鸟般跃起" |
| 白气涌入 | 能量转移的视觉化 | "一道道的白气蓬勃而出，如同倦鸟归林般没入镜中" |
| 篆符齐亮 | 升级的仪式感——视觉高潮 | "十二道篆符逐一闪亮" |
| 陆江仙感受 | 主观视角的升级体验 | "通体舒畅""神识范围不断扩大" |

## 升级效果

| ch075 | 效果 |
|-------|------|
| 神识范围 | 不断扩大 |
| 信息获取 | "一道道讯息流淌而出" |
| 后续潜力 | 为法鉴解锁更多功能铺垫 |

## 适用场景

- 法鉴需要升级/觉醒新能力时
- 主角获得与法鉴起源相关的灵物时
- 需要展示法鉴与某些灵物的"渊源"时

## 注意事项

- 吞噬过程的描写要有"仪式感"——不能太随意
- 灵物的"主动"意象很重要——不是法鉴在"吃"，是灵物"归巢"
- 每次吞噬后法鉴的变化需要有所不同（神识范围/新能力/信息解锁）
- 灵物的来源需要合理——不能随便什么灵物都能被吞噬（需要有"渊源"）
