# 套路：法鉴赐丹

## 核心公式

```
法鉴获得新材料/条件（《祭萃夺元法》·箓气） → 觉醒新能力（凝聚箓丹） → 设定使用规则（每大境界一枚/留箓印/死后回收） → 提升家族修炼上限 → 等待条件完善（需练气修为才能采箓气）
```

## 关键阶段

1. **材料/条件满足**：法鉴之前不能做的事——现在可以做了
   - 《祭萃夺元法》其实早就获得——但条件苛刻一直不能采箓气（ch077）
   - "前几年便得了这《祭萃夺元法》，却因条件苛刻迟迟不能采出箓气"（ch077）
   - 等待关键角色突破（李通崖突破练气后才能采箓气）

2. **能力觉醒**：法鉴在某个契机下解锁/展示新能力
   - 陆江仙梳理脑海信息时发现——鉴身亮起白莹莹的线
   - 新能力不是突发的——是"早就知道有这个可能，现在条件成熟了"
   - 法鉴的能力觉醒通常伴随视觉变化（篆符亮起/白线出现）

3. **规则设定**：法鉴的"物尽其用"风格——每个能力都有代价/限制
   - 每大境界仅可服食一枚（限制服用频次）
   - 在升阳府留下箓印（监控宿主）
   - 死后竭尽其精气转化为箓气回馈鉴身（回收机制）
   - **法鉴风格**：给予力量，但永远留后门

4. **叙事功能**：提升家族修炼上限/为角色突破铺垫
   - 箓丹可破关障精进修为——突破瓶颈的利器
   - 对此筑基紫府亦有奇效——连高阶修士也有用
   - 但需要先采出箓气——需要李通崖突破练气

## 法鉴能力觉醒模式

| 阶段 | 表现 | 判定标准 |
|------|------|---------|
| 条件前置 | 需要某种灵物/修为条件 | 尚未满足 |
| 发现可能 | 陆江仙知道法鉴有这个能力 | 信息解锁 |
| 条件满足 | 关键人物突破/灵物到手 | 可激活 |
| 能力启用 | 凝聚/灌注/赐予 | 正式使用 |

## 对比：法鉴吞噬灵物 vs 法鉴赐丹

| 维度 | 法鉴吞噬灵物 | 法鉴赐丹 |
|------|-------------|---------|
| 方向 | 灵物→法鉴（能量向内） | 法鉴→修士（能量向外） |
| 主体 | 陆江仙受益 | 家族修士受益 |
| 效果 | 法鉴升级/神识扩大 | 修士突破瓶颈 |
| 代价 | 灵物被吞噬 | 箓印留下+死后回收 |
| 风格 | 法鉴在"吃" | 法鉴在"投资" |

## 适用场景

- 法鉴需要展示新能力——不要总靠"吞噬灵物"
- 家族修炼线遇到瓶颈——需要外挂突破
- 铺垫主角/重要配角的突破节点

## 注意事项

- 法鉴赐丹能力的觉醒要有"条件前置"——不能凭空获取
- 箓印机制是核心——体现法鉴的"物尽其用"暗黑风格
- 箓丹不是万能的——"多则对筑基紫府亦有奇效"说明效果由箓气量决定
- 能力觉醒后不一定马上使用——还有条件（需采箓气）
