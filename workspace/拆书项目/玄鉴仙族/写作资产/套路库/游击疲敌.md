# 套路：游击疲敌

## 定义
弱势一方在敌方腹地通过灵活机动、避强击虚、不断骚扰的方式消耗敌方资源与意志。

## 功能
- 展现智谋型角色的战略才华
- 制造喜剧/荒诞效果（敌方被折腾到疯癫）
- 以小博大，营造"一人牵制一军"的英雄叙事

## 本卷范例（ch099 李项平×木焦蛮）

### 李项平游击战术清单
| 战术 | 效果 |
|------|------|
| 武装千人打垮 | 瓦解敌方基层 |
| 杀长老分粮 | 瓦解敌方统治结构 |
| 在练气修士抵达前撤离 | 规避高端战力 |
| 到处制造假消息 | 信息战 |
| 组织流民逼近大厥庭 | 迫使回援 |
| 裹挟近万流民 | 制造人道危机 |
| 传递假首级 | 心理战 |

### 敌方崩溃过程
1. 愤怒（第一次被打败）→ 2. 困惑（到处都有消息）→ 3. 失眠（无法定位）→ 4. 怀疑（其自身修为真实性）→ 5. 恐惧（无所不在的蝗虫）→ 6. 绝望（求巫山箓巫）

### 荒诞细节
- 木焦蛮连续收到"李项平首级"→ 都是山越的脑袋
- 部下拿着假首级邀功
- "这鸟货就像屁股和头顶都长了眼睛"

## 核心心法
1. 游击方需要地形/民心支持（李项平利用山越流民）
2. 必须展现敌方的"破防"过程
3. 游击战的结局注定是悲壮的（实力悬殊），但过程务必精彩
4. 可适度幽默——木焦蛮的崩溃有喜剧色彩
