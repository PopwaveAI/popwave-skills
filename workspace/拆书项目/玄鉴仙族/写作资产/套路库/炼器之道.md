# 套路：炼器之道

## 定义
法器从构思、选材、炼制到交付的完整叙事流程，展示修仙世界的工艺与经济体系。

## 本套路适用场景
- 角色需要专属/升级装备
- 需要展示修仙世界的经济与工艺设定
- 为后续关键战斗预设武器

## 典型案例（ch148-ch149）
**情境**：李玄锋定制练气级法器长弓。

**五步流程**：
```
第一步：需求提出
  李玄锋 → 冠云峰坊市 → 选法器店

第二步：方案讨论
  楚明炼 ←→ 李玄锋
  - 弓身：金方屏石（玄火煅烧，承受五千斤之力）
  - 弓弦：青乌金精（金属性罡气传导）
  - 符文：聚气凝气

第三步：定价立契
  材料80+工费40=120灵石
  (自备青乌矿→折价30→实付90)
  立契→玄景灵誓

第四步：炼制周期
  玄火锻烧126天（半年）

第五步：验收交付
  满月拉弓（威力测试）→ 命名 → 解契
```

## 关键技巧
- **炼器师角色塑造**：楚明炼——粗犷外貌+柔和声音，有历史渊源
- **价格压力制造**：超预算→自备材料折价→建立经济真实感
- **交付场面高潮**：满月拉弓震撼全场，彰显武器威力
- **命名仪式**：赋予法器姓名→确立持有者身份

## 经济模型参考
```
练气级法器：100-150灵石区间
  材料占比：~65%（自备可省~25%）
  工费占比：~35%
  立契担保：第三方（萧家）+ 灵誓
```
