# 套路：神兵出世

## 定义
一件顶级法器/武器从交付到首次展示威力的完整场面序列，强调"器与人"的匹配与震撼效应。

## 本套路适用场景
- 角色获得专属标志性武器
- 需要展示战力跃升的里程碑
- 铺垫后续重大战斗

## 典型案例（ch149 - 金庚弓）
**情境**：李玄锋半年后取弓、试射。

**三幕结构**：

### 第一幕：出世
| 元素 | 内容 |
|------|------|
| 外观 | 暗金色流光、长七尺六寸、沉重 |
| 触感 | 嘭然压桌（碧玉白光抵消） |
| 在场反应 | 学徒气喘吁吁、汗流浃背 |

### 第二幕：人弓合一
| 元素 | 内容 |
|------|------|
| 拉弓场景 | 右手持弓、左手提弦、沉气咬牙 |
| 力量展示 | 满月——嗡嗡大作 |
| 旁观者反应 | 学徒惊掉下巴、楚明炼面如刀割后退 |
| 楚明炼反应 | "莫非是妖兽化形！" |

### 第三幕：试射（威力验证）
| 元素 | 内容 |
|------|------|
| 蓄力 | 金庚弓亮起金色符文，弓弦透出浓烈火光 |
| 射击 | "着！" |
| 效果 | 炸倒五六棵树，地面大坑，黄土飞溅 |
| 附加效果 | 淡金色罡气打落整片树叶 |
| 评价 | "筑基修士猝不及防也要吃亏" |

## 命名仪式
```
法器名 ← 持有者命名（"金庚"）
    ↓
契约解除 + 灵誓解除（财货两讫）
    ↓
人器绑定完成
```

## 反高潮设计（高阶技巧）
**李玄锋自省**："却没能同这弓配合起来……"
→ 从得意到自省 → 闭关五年写弓法
→ 暗示：神兵只是起点，真正的力量在人与器的融合

## 心法口诀
"神兵出世，先惊四座；人器交融，方见真章。"
