# 禁制秘传套路

## 标签
#禁制 #秘传 #保密 #功法传承 #灵誓

## 适用范围
修仙家族/宗门中保护功法传承、防止外泄的秘密手段。与\"反套路禁制\"（不可言传）互补，\"玄景灵誓\"是一种主动的、可选择的保密手段。

## 核心要素
1. **传承困境**：家族需要保密功法又需要传授给可靠后辈的矛盾
2. **灵誓机制**：以玄景轮为誓，违誓者玄景破碎、修为尽废、应誓而死
3. **获取门槛低**：\"不值钱的小技巧，人人皆会\"——技术不贵，但价值高
4. **局限性明确**：只能保密，不能防歹意——\"天道尚难分清，区区灵誓又能懂什么\"
5. **应用场景**：培养核心子弟时使用——先修行到玄景轮，再立誓，再传秘法
6. **补充体系**：与功法本身禁制（白丸禁言）形成双重保护

## 关键事件
1. 家族成员询问是否有保密秘法
2. 外援传授灵誓机制与原理
3. 说明灵誓的局限性
4. 家族制定选拔+立誓+传法的流程
5. 核心子弟按流程被培养（立誓后才放心使用）

## 爽点设计
- 技术补全：\"补上最后一环\"的满足感
- 透明展示：灵誓效果明确、局限性也明确，没有\"刀枪不入\"的虚假安全感
- 实用导向：解决问题而非渲染神秘
- 信任边界：灵誓不能防恶意，体现了人性与制度之间的界限

## 变体
- 更高阶的灵誓可增加约束力（如天道誓言、道心誓言）
- 可设置\"破解灵誓\"的剧情（代价极高但可行）
- 家族可发展自己的灵誓变体
