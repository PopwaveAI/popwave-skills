# 套路：突破境界

## 基本信息

| 项目 | 内容 |
|------|------|
| 套路编号 | R006 |
| 套路名称 | 突破境界 |
| 核心功能 | 主角修为突破，获得新能力，拓展世界观 |
| 首次出现 | ch054「玉京」（李通崖突破玉京轮） |
| 出现频率 | 高（修炼文核心套路） |

## 套路结构

### 1. 触发条件
- 积累足够（修炼、丹药、顿悟）
- 闭关/入定

### 2. 标准流程

```
从入定中醒来 → 感知新境界 → 展示新能力 →
用新能力验证已有物品(阵旗/法器) → 
新境界的战力参照 → 世界观的拓展
```

**ch054应用实例：**
- 醒来：李通崖从入定中缓缓醒来，心中喜悦
- 感知：灵识诞生，在周身游荡，看到灵气游走
- 新能力展示1：操控阵旗——雾里迷阵如臂使指
- 新能力展示2：抹除血印——掌控家族权限
- 新能力展示3：捏碎青石、测算对凡人的战力（击溃百千军队）
- 世界观拓展：灵识探测修为——看到李项平身上四个灵轮亮起

### 3. 关键元素

| 元素 | 说明 |
|------|------|
| **感官更新** | 境界突破后感知世界的方式变化（如灵识看到灵气） |
| **能力具象** | 新能力要用具体操作展示（不是"感觉更强了"） |
| **战力锚定** | 给出相对于凡人/低阶修士的战力参照 |
| **验证反馈** | 用新能力验证已有物品，提供熟悉与掌控感 |
| **境界知识** | 自然带出修炼体系的知识（玉京→灵初轮→炼气） |

### 4. 注意事项
- 避免"打坐→光华一闪→突破了"的模板化写法
- 突破后要有"世界焕然一新"的感觉——用感官描写体现
- 新能力的展示要合理设置场景，不能凭空生造
- 要给出明确的能力上限和限制，不能无限开挂
