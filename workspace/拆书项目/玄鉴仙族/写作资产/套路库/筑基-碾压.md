# 套路：筑基·碾压

## 定义
筑基修士对练气期修士的**单方面、压倒性压制**。核心是展示修仙境界间的绝对鸿沟——无论练气巅峰积累多深厚，在筑基面前都如蝼蚁一般。

## 适用场景
- 新筑基修士的首次亮相战力展示
- 势力间实力对比的直观呈现
- 为读者建立"境界压制"的直观概念

## 结构模板
1. **境界铺垫**：受害者的修为信息（如练气巅峰，真元通透但不够高明）
2. **身份反差**：受害者态度极尽卑微（谄媚、自称晚辈、自扇耳光）
3. **出手动作**：轻描淡写，不费吹灰之力
4. **一击致命**：对方无法做出任何有效反应就被击杀
5. **旁观者反应**：见证者的震惊与内心评估

## 碾压的视觉化
| 细节 | 作用 |
|------|------|
| "抬手间袖口滑落，露出光洁如玉的手臂" | 筑基修士的肉身已超凡 |
| "轻轻一挥" | 动作轻易，与杀伤力形成反差 |
| "老者脑袋顿时如同西瓜般炸开" | 残忍直白的结果展示 |
| "来不及哼一声" | 反应时间趋近于零 |
| "直直地坠下去" | 尸体坠落的无力感 |

## 案例
**ch135**：郁萧贵（筑基）击杀老年散修（练气巅峰）。老年散修"谄媚地叫着郁萧贵前辈""朝着自己脸上啪啪两巴掌"，卑微到极点。郁萧贵"脸色和蔼""轻轻一挥"——老者脑袋炸开。卢思嗣感慨"一如既往！"

## 注意
- 碾压场景的力量在于**受害者越卑微，碾压感越强**
- 不要浪费碾压场面——它同时展示了施暴者的实力、性格和风格
- "和蔼微笑"的同时杀人，比咆哮杀人的反派更令人不寒而栗
