# 箓丹体系

## 定义
鉴身利用祭祀中多余的香火箓气，结合太阴月华凝聚而成的丹药。修仙者每个大境界仅可服食一枚，可破关障、精进修为，同时在升阳府留下箓印，死后将修为转化为箓气回馈鉴身。

## 与箓气（灰箓/箓气）对比

| 维度 | 箓气（灰箓） | 箓丹 |
|------|------------|------|
| **形态** | 无形的符文/光点，直接注入升阳府 | 白莹莹的实体丹药，刻有神秘纹路 |
| **产出** | 香火凝聚的直接产物 | 香火+太阴月华+秘法再加工 |
| **效果** | 赋予受箓者特殊能力（如力贯千钧） | 破关障、精进修为、死后反馈 |
| **消耗** | 一次性消耗箓气 | 可服食，修为反馈可持续 |
| **适用范围** | 某一代核心弟子（赠予特殊能力） | 可流通到市面上给高阶修士 |

## 特征

| 维度 | 描述 |
|------|------|
| **外观** | 白莹莹、亮晶晶，刻画神秘曲折的线条纹路，颇为美丽 |
| **服用限制** | 每个大境界仅可服食一枚，且建议在突破瓶颈时使用（如练气巅峰突破筑基） |
| **附加效果** | 在升阳府留下箓印，增加吐纳灵气速度 |
| **死后反馈** | 宿主死亡时，竭尽其精气转化为箓气回馈鉴身，可取修为十之二三 |
| **效率** | 比不上玄珠符珠（十之二三 vs 更高），但胜在可量产、可流通 |
| **秘法来源** | 李项平当年杀死练气老修士获得的玉佩，被鉴身修复后得到 |
