# 套路：纨绔挑衅

## 基本信息

| 项目 | 内容 |
|------|------|
| 套路编号 | R002 |
| 套路名称 | 纨绔挑衅 |
| 核心功能 | 制造冲突，展示主角战力，引入配角和势力背景 |
| 首次出现 | ch052「邓求之」 |
| 出现频率 | 中 |

## 套路结构

### 1. 触发条件
- 主角（通常是低阶/新弟子）在宗门/公共场合
- 对方是世家大族/高阶弟子
- 有表面理由（看不顺眼、试探成色）

### 2. 标准流程

```
偶遇/被盯上 → 言语挑衅 → 包围/逼墙角 → 越级战斗 →
反杀/逼平 → 对方认输 → 留伤药/示好 → 铺垫对方背景
```

**ch052应用实例：**
- 偶遇：李尺泾走在青穗峰附近小路，邓求之迎面而来
- 挑衅："师弟别慌，试试你的成色罢了"
- 包围：随从上前逼至角落
- 战斗：灵初轮(胎息巅峰)vs练气期
  - 邓求之金珠法器远攻
  - 李尺泾灵网缠金珠→硬挨一击→欺身近→拔剑月白剑光
  - 平安符挡下剑光
- 认输："却是你赢了！" → 赠伤药
- 铺垫背景：邓求之阴晴不定→暗示功法有问题

### 3. 关键元素

| 元素 | 说明 |
|------|------|
| **力量悬殊** | 对手必须明显强于主角（至少一个大境界） |
| **战术智慧** | 反杀靠的不是开挂，而是战术选择（远攻对近身、舍小换大） |
| **对方留手** | 最终对方主动收手，而非被主角击败——维持战力体系 |
| **背景钩子** | 挑衅后必须引出对方的背景/功法/家族设定 |
| **性格展示** | 战后对方的表现揭示其性格（阴晴不定/磊落/记仇） |

### 4. 注意事项
- 避免写成纯粹的"打脸爽文"——主角应该赢得艰难且有代价（受伤）
- 对方的"认输"要有合理性：可能是不想闹大、可能是被主角表现震慑
- 要留下伏笔：对方记住了主角，后续可能再遇
- 必须通过这场冲突展开世界观，而不是为打而打
