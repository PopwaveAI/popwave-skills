# 套路：结盟·暗线

## 定义
两个或多个势力在正式的外交场合中，通过**眼神、微表情、简短的寒暄**等非语言信号，传达"我理解你的处境""或许我们可以合作"的信息。核心是"在公开场合下的秘密默契"。

## 适用场景
- 面对共同威胁时，两个势力尚未建立正式联盟
- 在压迫者眼皮底下交换信息的风险场景
- 为后续联盟关系做铺垫

## 结构模板
1. **共同处境**：两个势力同时成为第三方压迫的对象/潜在对象
2. **公开相遇**：在压迫者主导的场合中见面（如宴席）
3. **非语言交流**：一个眼神、一个表情、一句简单的话
4. **心照不宣**：双方理解了彼此的态度，但不进一步摊牌
5. **分开记忆**：各自离开后，在心中记下这个瞬间

## 可用的暗号交流形式
| 形式 | 说明 |
|------|------|
| 眼神对视 | "我理解你"的基本信号 |
| 座位/距离的选择 | 表明站队态度 |
| 简短的应和 | 在会议中的搭腔 |
| 共同的沉默 | 面对同一要求时都不表态 |
| 举杯/动作微调 | 向对方示意 |

## 案例
**ch135**：李通崖落座后与对面的费家家主（同为筑基家族）"对视一眼"，李通崖见费家家主"一脸希冀之色"——双方都从郁家的强势中感到了威胁，都需要盟友。两人"心中皆有了底"，但没有进一步交流。

## 注意
- 暗线交流不能过于直白，否则失去"暗"的意味
- 眼神交流后的独立反应（各怀心思）增加了真实感
- 这种默契需要后续章节的验证——是真结盟还是各自为战？
