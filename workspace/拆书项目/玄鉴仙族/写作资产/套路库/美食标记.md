# 套路：美食标记

## 核心公式

```
引入地方特色美食/饮品 → 详细描写制作/口感 → 成为该地标和旅途的记忆锚点 → 可复用/形成系列
```

## 关键元素

| 元素 | 说明 | 示例（雪花酒） |
|------|------|-------------|
| 命名 | 有特色、易记 | "雪花酒" |
| 工艺 | 详细描写让人有画面感 | 羊肉批片→酒煮烂→细切研膏→羊髓熔油→拌入→温酒浸饮 |
| 口感 | 多用感官词 | "美酒的醇香和羊肉的鲜美混合着淡淡的奶味" |
| 角色反应 | 初尝的体验感 | 李通崖不知不觉喝完 |

## 叙事功能

1. **记忆锚点**：读者记住这个地方的特色 → 后续再提"东口雪花酒"立刻唤醒记忆
2. **节奏调节**：在紧张冒险中插入美食描写=放慢节奏+调节情绪
3. **角色人情**：万元凯介绍雪花酒=展示他热情好客的一面
4. **风土呈现**：通过美食展现地域文化（古黎道的酒文化）

## 注意事项

- 美食描写要恰到好处，不宜过长打断节奏
- 可以与其他套路组合使用（如旅途中继中的美食标记）
- 同一地区可以形成"美食系列"增加沉浸感
