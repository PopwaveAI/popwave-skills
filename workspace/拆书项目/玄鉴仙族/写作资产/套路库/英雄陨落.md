# 套路：英雄陨落

## 定义
重要正面角色在悲壮情境下死亡，其死亡方式传递某种价值（牺牲/反抗/传承/宿命）。

## 功能
- 建立作品的悲剧深度与情感厚度
- 为其他角色提供成长/觉醒的动力
- 制造读者情感冲击，强化记忆点

## 本卷范例（ch100 李项平之死）

### 死亡前的"完美一天"（反差设计）
- 火边烤芋头，谈笑风生
- 调侃陈冬河与景恬婚事
- 众人崇敬的目光

### 死亡预警（避死延生箓气）
- 预见"跪倒在地，口中吐血"
- "避无可避，延无可延"
- 为死亡赋予宿命感

### 死亡过程（诗化处理）
- **不直接描写惨状**: 从"眼中的世界已经黑了下来"到"嘴角微微翘起"
- **三个阶段**: 身体枯萎→黑气被箓气吞噬→尸体化蝗虫飞散
- **最后的声音**: 低低的笑声、"好低的手段"
- **最后的豪言**: "这辈子老子已然活赚了！"
- **最后的回忆**: 少年捉鱼+青灰色鉴子+朝阳灿烂（首尾呼应）

### 死亡后的余响
1. **蝗虫传讯**: 飞越千里撞死李通崖窗上
2. **法鉴感知**: 陆江仙怅然若失
3. **气运反哺**: 玄珠符种因李项平夺伽泥奚气运而回馈大增
4. **新生对照**: 李渊修诞生在同章——生死的并置

### 英雄陨落的要素清单
| 要素 | 本卷做法 |
|------|---------|
| 预兆 | 避死延生箓气预警 |
| 诀别 | 托付宝珠+路线 |
| 遗言 | "活赚了" |
| 遗物 | 宝珠、青乌弓 |
| 死亡场景奇观 | 蝗虫飞散 |
| 死亡余响 | 窗上撞死的蝗虫 |
| 对照生命 | 渊修诞生 |
| 传承标记 | 陈冬河背弓率众东归 |

## 变体
- **沉默的陨落**（无遗言/无场面）
- **盛大的陨落**（惊天动地，众人皆知）
- **孤寂的陨落**（无人知晓，尸骨无存）——李项平介于盛大的蝗虫异象与孤寂之间

## 禁忌
- 不可为死而死——死亡必须推动剧情
- 不可拖沓——死亡场面宜短促有力
- 不可廉价——只有与读者建立了情感连接的角色才值得此套路
