# 套路：薪火相传

## 定义
通过长辈的离世或凋零，推动下一代接班成长的叙事模式。死亡既是终结，也是交接。

## 本套路适用场景
- 老一辈骨干自然凋零，家族代际更替节点
- 葬礼提供群像展示窗口
- 需要推进角色从受庇护到独立承担

## 典型案例（ch147-ch148）
**情境**：陈二牛（最早一镇之长，李家骨干）去世。

**多层传递结构**：
```
陈二牛（已逝）── 言传身教 → 陈冬河（子，入赘李家）
                        ↓
李项平（养父）── 教"狠厉出击" → 陈冬河（修炼突破）
                        ↓
李通崖（家主）── 授予《江河一气诀》→ 陈冬河（练气）
                        ↓
陈冬河（新一代战力）── 拱卫家族
```

**辅助传递线**：
```
陈二牛之死 → 李通崖感叹"老一辈已然萧条"（自觉为辈分最大者）
           → 李平逸经历葬礼（接触死亡与礼法）
           → 李渊修哭泣（情感共鸣）
           → 两少年友谊深化
```

## 关键技巧
- **仪式作为叙事节点**：葬礼提供群像汇聚的合理性
- **旧人遗言设定基调**：陈二牛"大胆去做，父亲不会怪你"
- **多层伤感递进**：个人（陈冬河）→家族（李通崖）→社会（镇民、孩子）
- **传承物化**：功法口诀/清气/法器作为传递载体
