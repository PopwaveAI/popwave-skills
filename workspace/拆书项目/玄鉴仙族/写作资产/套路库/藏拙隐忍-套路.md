# 套路：藏拙/隐忍天才

## 定义

天赋极高的角色刻意隐藏实力/才华，只在最关键（通常是绝境）时刻显露全部实力。是《玄鉴仙族》中"天赋者之殇"主题的核心构成。

## 功能

- 制造"扮猪吃老虎"式的震撼效果
- 强化角色的悲剧色彩——才华越耀眼，陨落越遗憾
- 为其他角色提供警示（"萧家老祖"的"五年不得突破"原则）
- 展现残酷世界中的生存智慧

## 关键要素

| 要素 | 说明 |
|------|------|
| **隐藏时长** | 藏得越久，爆发越震撼（李尺泾藏了五年剑意） |
| **对外姿态** | 配合、谦逊、从不张扬——烟雾弹 |
| **逼出条件** | 必须是在绝境中才能展现（李尺泾将死） |
| **亮相场面** | 全城青锋躁动、天地异象——视觉化的震撼 |
| **观者评价** | 同伴震惊、敌人惋惜/恐惧 |
| **代价** | 藏拙的代价是持续忍受羞辱/压制；亮相的代价是再无退路 |

## 适用章节

- ch119-ch120（李尺泾）：剑意藏拙五年，在比试中完全释放
- ch116（萧家老祖的"五年不得突破"命令）：作为藏拙理论被阐述

## 变体

- **修为藏拙**（本案例）：隐藏剑意/真正战力
- **智力藏拙**：隐藏谋略/心计
- **身份藏拙**：隐藏真实出身/背景

## 世界观效应

"这世道，最天才与最愚钝的往往最凄惨。"——萧雍灵。藏拙不仅是个人选择，更是残酷环境中的生存策略。
