# 藏锋纳锐

---

## 定义

强者在完全有能力压制对手时，不采用暴力摧毁，而是以"收剑/收招"的方式化解冲突，将进攻转化为教诲，让对手心服口服。

## 公式

```
对方挑衅/挑战 → 轻松化解 → 不伤人而"收其器" → 赠言点化 → 对手恍然大悟/心生敬佩
```

## 核心要素

| 要素 | 说明 |
|------|------|
| **优势方** | 有能力碾压但不碾压 |
| **器物象征** | 被收走的不只是武器，更是锐气的载体 |
| **点化之言** | 一句哲理（通常是剑道/武道哲学） |
| **收束方式** | 定下未来归还的契约（你来取剑/等你悟了再来） |

## 玄鉴仙族实例

**ch136 郁慕剑**（经典实例 #1）
- 李通崖原地防守化解全部攻势 → 月阙剑意点到即止 → 收剑 → "其进锐者其退速，青锋出鞘之前在于藏"
- 约定：等郁慕剑能将锐气收放自如，来黎泾山取剑
- 郁慕剑反应：浑身一震，神色复杂 → ch139中"李通崖真君子也！我真是小人一个！"

## 变形

- **藏锋于家族**（ch137/ch140）：李家整体低调不出头，"局势越平静越好"——家族层面的藏锋
- **器物藏器**（ch138）：郁慕剑之剑本身也是"藏锋"——剑中藏传承，只有懂藏的人才能发现

## 效果

1. 化敌为友（至少降低敌意）
2. 立住"君子"人设，让旁观者生敬
3. 创造后续伏笔（归还之约可延展为剧情钩子）
