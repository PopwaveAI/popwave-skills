# 血脉疑虑

---

## 定义

家族成员因出身（混血/非纯血/外来血统）而在家族内部受到隐性的审视、猜疑或偏见，影响其继承权、资源分配和情感归属。

## 公式

```
家族内部存在"非纯血"成员 → 某事件触发对他/她身世的重新审视 → 检查/试探 → 没有定论但芥蒂已生 → 采取预防措施（隐瞒/删改/补偿）
```

## 核心要素

| 要素 | 说明 |
|------|------|
| **"非纯"的源头** | 父母一方非本族/外族/敌对族系 |
| **触发事件** | 家族成员死亡（尤其是被外族所杀），引发牵怒 |
| **检查行为** | 特征检查（眸色/体质/血脉反应） |
| **情感矛盾** | 理智上知道孩子无辜，情感上无法做到完全无芥蒂 |
| **处理方式** | 或隐瞒、或补偿、或从族史中抹去 |

## 玄鉴仙族实例

**ch139-ch140 李渊蛟血脉案**（经典实例 #1）
- 源头：李渊蛟之母木芽鹿为木焦蛮族妹；李项平死于山越（木焦蛮/巫山所为）
- 触发：李玄宣一月不见李渊蛟 → 被李渊修点醒 → 前往看望
- 检查：阳光下眸子微褐 → 阴影中褪去 → 松一口气但"猜疑终究不曾说出来"
- 处理方法：找李景恬讨论"看能不能不记上"（从族史删改）

## 深度分析

**李玄宣的情感光谱**
| 维度 | 描述 |
|------|------|
| 愧疚感 | 知道不该迁怒于孩子 |
| 防备心 | 害怕血脉中藏有隐患 |
| 保护欲 | 仍想保护渊蛟的未来（删族史） |
| 距离感 | "嘴上不说，心中多少有些芥蒂" |

## 变形

- **血脉优势**（相反方向）：安景明14岁练气——显示纯血天才的威胁
- **血脉偏见**（社会层）：郁家对诸家的歧视——血统论的社会背景

## 用法建议

1. 不要给确定答案——血脉疑虑的张力在于"不确定"
2. 处理方式暗示人物的道德取向（李玄宣选择保护而非排斥）
3. 血脉问题可以跨卷发展，成为角色的核心成长驱动力
