# 套路库：装疯卖傻求生术

> 归类：人物策略 / 伪装设定

---

## 定义

角色长期以疯癫/愚钝/白痴的面目示人，实则拥有清醒的思维和深远的谋划。伪装不是性格缺陷，而是面对高压环境（宗门监控、家族迫害、体质秘密）的唯一生存策略。

---

## 本卷实例（ch087 — 邓求之）

| 项目 | 内容 |
|------|------|
| 公开形象 | 疯言疯语、死皮赖脸、口无遮拦——"又开始发疯了" |
| 真实面目 | 清醒无比，第一个看穿兽潮节奏，洞察人丹真相 |
| 伪装时长 | 几百年（邓家代代如此） |
| 伪装原因 | 邓家体质特殊→被强迫修炼秘法→炼成补品→必须装疯以免引人注目 |
| 伪装效果 | 成功降低威胁度，周围人"用异样的眼光去看"但不提防 |
| 破防条件 | 遇到李尺泾——五年并肩作战+生死相依→终于坦白 |

### 伪装技巧
| 技巧 | 实例 |
|------|------|
| 用疯癫语气说真话 | "要吃你与我，他们却通通要为你陪葬" |
| 用抱怨包装警告 | "哪有这么贴心的事"——指兽潮退却节奏异常 |
| 找理性派分流注意力 | 李尺泾的理性反驳→邓求之看似无理的追问→实则步步逼近真相 |

## 套路结构

```
┌─────────────┐
│ 被迫伪装     │ ← 体质/家族/使命原因
└──────┬──────┘
       ▼
┌─────────────┐
│ 表层崩溃     │ ← 社交上被边缘化/被嘲笑
└──────┬──────┘
       ▼
┌─────────────┐
│ 内层清醒     │ ← 超常洞察+暗中分析
└──────┬──────┘
       ▼
┌─────────────┐
│ 真假切换     │ ← 关键时刻用疯话传递真相
└──────┬──────┘
       ▼
┌─────────────┐
│ 找到知己     │ ← 极端环境下的信任建立
└──────┬──────┘
       ▼
┌─────────────┐
│ 卸下伪装     │ ← 坦白一切（往往是生死关头）
└─────────────┘
```

---

## 写作要点

1. **前期不揭示**：前文邓求之出场时让读者也以为他是疯子——才有反转效果
2. **疯话要有信息密度**：疯话不是乱话，每句都可做双重解读
3. **伪装有代价**：几百年装疯——邓求之虽清醒，精神状态已受影响
4. **卸伪需要阈值**：五年同生共死才达到信任阈值——避免随意破防
5. **给伪装的逻辑理由**：不是"性格古怪"，而是家族被胁迫炼补品——求生不是选择
