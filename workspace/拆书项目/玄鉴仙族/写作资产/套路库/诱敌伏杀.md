# 套路：诱敌伏杀

## 定义
通过佯败、撤退、引诱等方式将敌人引入预设伏击圈/阵法/陷阱中予以歼灭的战术套路。

## 叙功能
- 展示角色的战略思维（非纯武力型角色）
- 利用地形/阵法/道具取胜，体现"智取"美学
- 将悬念（伏击点是什么？）转化为高潮（伏击生效）

## 本卷实例（ch041-ch043）

### 结构模板
```
分析敌我差距 → 判断不可力敌 → 设定诱敌路线 → 部署伏击点（阵法/法鉴） → 边打边撤诱敌 → 触发伏击 → 一击斩杀
```

### 本卷执行细节
1. **诱饵设计**：李项平自己+万天仓（修仙者对大补）为诱饵
2. **退路规划**：黎泾山有阵法庇护 → 实际上法鉴在黎泾山顶
3. **信息差利用**：妖物不知黎泾山法鉴存在，万萧华也不知道（对外称"符箓"）
4. **心理利用**：李项平精准利用妖物"惜命谨慎"的特点，确定它不会冒进

### 写作要点
| 要素 | 说明 |
|------|------|
| 合理性 | 必须有合理的"为什么要诱敌"（实力不足）和"为什么能诱成功"（妖物盯上修仙者） |
| 代价展示 | 沿途村丁伤亡 → 诱敌有代价，不廉价 |
| 伏击预期管理 | 先铺垫"黎泾山有阵法"，后展示法鉴，层层递进 |
| 收束确认 | 妖物确实被击杀，不留后患 |
