# 套路：资源规划

## 基本信息

| 项目 | 内容 |
|------|------|
| 套路编号 | R005 |
| 套路名称 | 资源规划 |
| 核心功能 | 对新发现的资源进行开发利用规划，展示家族经营思维 |
| 首次出现 | ch054「玉京」 |
| 出现频率 | 中 |


## 套路结构

### 1. 触发条件
- 发现了新的资源（灵田/矿产/灵脉）
- 获得了新的资产（法器/灵物）
- 家族需要发展壮大

### 2. 标准流程

```
资源评估(产量/价值) → 用途规划(种植/养殖/售卖) →
人员安排(谁负责谁监督) → 控制机制(防止做大) →
长期预期(几年后能达到什么规模)
```

**ch054应用实例：**
- 资源：眉尺山山坡周围灵气富足，好大一片灵田
- 产量评估：每五年两茬，至少100斤灵米
- 人员安排：李秋阳管束，李叶生筹备人手，任平安协助
- 控制机制：李秋阳的父兄在黎泾山脚下，交叉管束
- 长期预期：原有300斤+新100斤=400斤，每五年2-3灵石结余

### 3. 关键元素

| 元素 | 说明 |
|------|------|
| **具体数据** | 必须给出产量/价值的具体数字，不能模糊 |
| **人员链** | 谁负责执行 → 谁负责监督 → 谁承担风险 |
| **控制逻辑** | 放权与控权的平衡——为什么放心用这个人 |
| **长期视角** | 不是一次性的安排，要有后续预期 |
| **家族本位** | 所有规划服务于家族整体利益 |

### 4. 注意事项
- 数据要合理、有参照系（原有产量+新增产量的对比）
- 人员安排要体现家族内的人际关系和制衡
- 控制机制不能太阴暗（全盘监控）也不能太天真（完全放权）
- 要体现"穷家创业"的经济逻辑——每一斤灵米都重要
