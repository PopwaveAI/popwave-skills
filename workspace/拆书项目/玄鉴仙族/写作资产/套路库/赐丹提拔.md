# 套路：赐丹提拔

## 核心公式

```
长辈/强者看中后辈 → 以"粗糙/强硬"的理由给予资源 → 不容推辞 → 后辈感激/惶恐 → 实力获得提升基础
```

## 关键节点

1. **视察**：强者观察后辈的修为/天赋/品性
2. **判断**：确认后辈值得投资（或出于人情/面子）
3. **赠与**：以不容拒绝的方式给予
   - 关键技巧：给一个"不容推辞的理由"而非直接宣示善意
   - 示例（ch050）："家中连个玉京轮都没有，丢了青穗峰的脸"
4. **离开**：不给后辈推辞/道谢的机会
   - "拿上丹药走人，莫要聒噪" → "送客！"

## 强者心理分析

- 表面：不耐烦/嫌弃
- 潜台词：我看好你/你值得我培养/这是人情
- 用意：保全对方面子+不让对方有负担

## 效果

- **对后辈**：实力提升的机会+人情债
- **对读者**：爽感——主角获得大佬赏识
- **对强者**：种下善因（未来可能的回报或人情）
- **对世界观**：展现修仙世界的"提携"文化

## 变种

| 变种 | 差异 | 示例 |
|------|------|------|
| 硬塞（霸总式） | 不容拒绝、不给推辞机会 | ch050 萧初庭 |
| 交易式 | 等价交换，各取所需 | 更商业化 |
| 考验式 | 先考验再赐予 | 标准师门套路 |
| 战利品式 | 从敌人处缴获后赐予 | 战斗后奖励 |

## 注意事项

- 赐予物品要与强者身份匹配（筑基赐胎息级丹药合理）
- 赐予理由要合理（不能无缘无故）
- 赐予后后辈的反应要真实（感恩+惶恐+决心回报）
