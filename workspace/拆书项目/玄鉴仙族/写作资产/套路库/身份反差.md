# 套路：身份反差

## 核心公式

```
出场排场极大（外表高贵） → 行为与预期的反差 → 揭示真实身份/意图 → 制造悬念或笑点
```

## 常见类型

| 类型 | 预期 | 实际 | 效果 |
|------|------|------|------|
| 谦逊贵族 | 嚣张跋扈 | 礼貌谦逊 | 好感+悬念 |
| 朴素高手 | 平平无奇 | 实力超群 | 震撼 |
| 恶相善人 | 面目可憎 | 心地善良 | 温情 |

## 示例（ch047 ch048）

| 要素 | 内容 |
|------|------|
| 预期 | 萧雍灵排场极大（仆从簇拥、雪白狐裘、金色锦囊）→应是嚣张跋扈的世家子 |
| 实际 | "恭敬""礼貌""毫不在意地坐在破木凳"→对同道极为谦逊 |
| 悬念 | 中年管事恐惧他——说明他平时并非如此，为何对李通崖例外？ |
| 回收 | 管事的对话揭示：在修士眼中凡人不算人→萧雍灵的"礼貌"只对同道 |

## 效果

- 打破读者预判 → 角色立体
- 层层揭示性格 → 阅读趣味
- 为后续信息揭示留空间

## 注意事项

- 反差不能凭空——需要有内在逻辑（修士对同道客气≠对凡人客气）
- 反差的方向要有新意（高手扮猪吃是常见套路，贵族谦逊更少见）
- 反差背后最好有世界观逻辑支撑
