# 套路：轻敌致死

## 核心公式

```
力量占优（自信建立） → 连续被骗（轻敌升级） → 触碰禁忌/错误决策（致命错误） → 突然反转（死）
```

## 关键阶段

1. **力量优势建立**：强者展示明显优势
   - 老道练气vs李项平胎息（ch073）
2. **第一次被骗**：受害者吃小亏但不以为意
   - 第一次太阴玄光蒸发左膀——认为只是没防备
3. **轻敌归因**：为对方的成功找合理化的低估理由
   - "胎息境持这鉴子便有如此威力…想必消耗极大，一时用不出来了"（ch073）
4. **第二次被骗**：再次上当，被示弱引诱
   - 李项平假装无力再战，举鉴诱敌（老道被耍了一次）
5. **判断失误**：基于错误归因做出致命决策
   - 用拂尘去捉法鉴——以为已经安全
6. **触发禁忌**：触碰了不应碰的东西
   - 法鉴觉醒——十二篆符齐亮
7. **死**：来不及反应就被反杀

## 轻敌者的心理模型

```
"我是强者" → "他侥幸/消耗大" → "我是对的！" → "他在骗我但我能应对" → "完了！"
```

## 情绪曲线

```
自信(优势) → 轻视(教训不够) → 懊恼但不反思(找借口) → 贪婪(趁胜追击) → 傲慢(致命决策) → 后悔(已晚)
```

## 适用场景

- 反派/配角具备碾压实力但死于轻敌
- 为"弱者借助外力反杀"提供合理性
- 展示有灵性的法器/武器需要敬畏

## 关键要点

| 要素 | 作用 | ch073示例 |
|------|------|----------|
| 两次以上的被骗 | 坐实"轻敌/不吸取教训"的性格 | 第一次蒸发左膀→第二次转身拿鉴→第三次捉拂尘 |
| 合理化的轻敌归因 | 让读者觉得"这人该死" | 老道自我安慰"消耗大" |
| 触碰明确禁忌 | 符合设定而非运气 | 拂尘捉鉴=犯忌讳 |
| 突然到无法反应的死亡 | 轻敌的直接惩罚 | 半个音节都发不出 |

## 注意事项

- 不能让轻敌显得太蠢——需要有一定合理性
- 反面教材：如果敌人第一次被骗后立刻警觉，就不能再用同一招
- 禁忌的设置需要有前文铺垫（ch070的法鉴忌讳设定）
