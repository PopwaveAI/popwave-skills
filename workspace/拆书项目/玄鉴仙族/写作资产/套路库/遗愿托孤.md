# 套路：遗愿托孤

## 定义
人物预感/明知自己将死，将自己的遗物、知识、责任托付给可信之人，并嘱托其照顾家族/后人。

## 功能
- 塑造将死之人的慷慨悲壮
- 埋下大量伏笔（托付的物品必然在后续使用）
- 强化"传承"这一核心主题

## 本卷范例（ch097 李尺泾托孤萧元思）

### 典型结构
1. **预知死亡**: 李尺泾深知自己是炼丹"主药"，突破筑基即赴死
2. **告别亲友**:
   - 邓求之（同为辅材，酒醉诀别）
   - "不就是被吃嘛？"
   - "这酒不够烈"
3. **交付遗物**:
   - 遂元丹 → 家族可用筑基丹
   - 炼丹传承 → 家族炼丹体系开端
   - 弓法秘术 → 玄锋可用
   - 灵石百余 → 家族资金
   - 符箓若干 → 家族物资
4. **托付誓言**:
   - "只要我萧元思不死，李家绝不会断了传承！"
5. **最终心愿**: 诅咒紫府老祖突破金丹失败（反向祝福）

### 托付对象的选择逻辑
- 萧元思：忠厚、在宗门内有一定地位、与李家有情谊
- 托付内容以"传承"为核心——功法/知识 > 财物

## 变体（ch100 李项平托付陈冬河）
- 项平未托付功法财物，而是**托付宝珠+撤退路线**
- 符号物（宝珠）的传递意义 > 实用物品
- 时间紧迫下的"极简版托孤"

## 节奏要点
1. 托付过程不宜一次完成，应分段释放
2. 每件遗物都应在后续找到用武之地
3. 托付者情绪宜"笑对死亡"而非嚎啕大哭
4. 受托者需给出坚定承诺，形成情感锚点
