# 重生者布局

---

## 定义

带有前世记忆的角色（重生者/穿越者）利用对未来大事件的预知，试图提前布局抢占先机，但受限于前世信息的模糊性（不知道细节、不知道路径），只能像"盲人摸象"般运作。

## 公式

```
重生/穿越 → 知道某件大事即将发生 → 知道"谁"会崛起但不知道"怎么"崛起 → 尝试寻找/结交潜在强者 → 因信息不全屡屡碰壁 → 与不知情者产生认知差异
```

## 核心要素

| 要素 | 说明 |
|------|------|
| **信息优势** | 知道大事件的时间轴（十几年后魔灾） |
| **信息盲区** | 不知道具体细节（强者如何崛起、机缘在哪） |
| **行动方式** | 广撒网式结交——从现在还不起眼的人里寻找未来强者 |
| **认知负担** | "事事都计算着，累得头昏脑胀"——知晓未来的独行者很累 |
| **身份隐匿** | 不能暴露自己是重生者 |

## 玄鉴仙族实例

**ch138 刘长迭**（经典实例 #1）
- 重生状态：从头修炼更快 + 阵道修为带回来
- 知道什么：魔灾还有十几年（青池宗与镗金门受灾）
- 不知道什么：细节——只知道"江雁"这个名字
- 当前行动：问李玄锋是否认识江雁 → 失败
- 现有关系：已与李玄锋建立友谊（"跟几个潜力股扯上些关系"）

**刘长迭的困境**
> "前世那些强者此时尚未崛起，可前世他们也从未大肆宣扬是如何崛起的，让刘长迭想先人一步抢人机缘都无从开始"

这是重生者套路的核心痛点：**知道结果不知道路径**——这比什么都不知道更折磨人。

## 与其他套路的关系

| 套路 | 交叉方式 |
|------|---------|
| 世家密谋 | 重生者成为隐性的第三方势力，影响世家博弈 |
| 阳谋制衡 | 重生者预估郁家/诸家的行动结果，提前站队 |
| 以退为进 | 重生者知道"退"才是最佳选择（因为知道未来的大势） |

## 用法建议

1. 重生者不需要全知全能——限制信息的模糊度是关键
2. 重生者的孤独感是情感抓手（"累得头昏脑胀"）
3. 重生者的行为可以影响但不主导剧情，保持故事的开放性
4. 长期伏笔：随着时间推移，重生者的预言逐一验证/被颠覆
