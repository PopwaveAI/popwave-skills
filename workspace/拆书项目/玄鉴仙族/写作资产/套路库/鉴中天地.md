# 鉴中天地

## 定义
鉴中自成天地。法鉴内部开辟独立的洞天世界，可容纳神魂/化身活动。

## 特征
| 维度 | 描述 |
|------|------|
| 类型 | 洞天/秘境 |
| 状态 | 残破（灰败、废墟、雾霭） |
| 范围 | 方圆千里 |
| 核心地貌 | 巍峨大山，山顶有白色大树+石桌石椅 |
| 建筑遗迹 | 月白色碎石、月华元府建筑残骸 |

## 功能
- 容纳陆江仙的太阴化身
- 太阴月华可在此凝聚物质
- 从鉴中穿梭回现实需克服"大恐怖"
