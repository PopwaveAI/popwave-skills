# 镜中造物

---

## 定义

主角（陆江仙）在镜中天地使用不同于修仙界常规体系的力量，从无到有创造灵性造物/修复世界。核心特征是"随心所欲"——没有固定公式，全凭意念把控。

## 公式

```
研究/拆解已有力量体系 → 获得灵感 → 尝试造物 → 造物获得灵性 → 自主运转 → 对造物主产生依赖/跟随
```

## 核心要素

| 要素 | 说明 |
|------|------|
| **力量差异** | 与修仙界法诀"规规矩矩"不同，镜中力量"随心所欲" |
| **材料来源** | 镜中天地的特有资源（月牙白色碎石、太阴月华） |
| **造物特征** | 有形有手有脚，"呆呆傻傻"但能协作 |
| **造物功能** | 修复破损废墟（掘土、搬砖） |
| **对修仙界的感应** | 能听到镜外求箓者的声音 |

## 玄鉴仙族实例

**ch140 陆江仙造月牙石人**（经典实例 #1）
- 研究基础：从山越巫术中拆解灵感
- 造物过程：白光术法如雨般落下 → 月牙碎石跳动 → 堆砌成人形 → 太阴月华点亮脑袋
- 造物结果：亮晶晶脑袋的小人 → 自动协作修复废墟
- 关键区分：与李家法诀不同，"没有固定的施法结果，全凭他心念来把控"
- 结尾信号："原来是求箓来了"——镜内镜外的连接

## 与其他套路的关系

| 套路 | 交叉方式 |
|------|---------|
| 剑中传承 | 镜像线提供"箓气"（传承的另一种形式） |
| 以退为进 | 陆江仙的存在本身就是李家"退"的底气——有隐藏力量 |

## 叙事功能

1. **神秘性维持**：陆江仙的力量体系不解释，保持"不可知"的敬畏
2. **节奏调节**：人间线的政治博弈/家族矛盾之后，切换至镜中线的超现实场景，打破阅读疲劳
3. **世界观锚点**：提醒读者——这个世界不仅有修仙家族和大宗，还有镜中这一层更高的存在
4. **箓气供应线**：陆江仙是李家"箓气"的源头，直接关系家族战斗力

## 用法建议

1. 力量体系不要解释太清楚——保持造物的"非逻辑感"
2. 造物最好有拟人化特征（呆呆的、协作的），增加亲近感
3. 镜中线和人间线的连接点（如求箓）要自然，不要生硬
4. 造物过程从粗糙到精细，展现陆江仙的实力成长
