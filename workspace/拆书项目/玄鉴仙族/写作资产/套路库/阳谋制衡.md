# 阳谋制衡

---

## 定义

弱势方无法正面抵抗强势方，于是利用对方布局上的必然破绽，以"合法合规"的手段制造打击。所有行动都在规则之内（或至少无法追责），让强势方吃哑巴亏。

## 公式

```
强势方布局扩张 → 弱势方发现其结构弱点 → 借"外力"打击弱点 → 强势方明知是对方所为却无法追责 → 损失坐实
```

## 核心要素

| 要素 | 说明 |
|------|------|
| **表面合法** | 所有打击者身份都是"盗贼/流寇"，非诸家正式力量 |
| **结构弱点** | 盯住对方两线作战（坊市+本家）时的人手不足 |
| **时间差** | 先攻击甲处逼对方分兵 → 趁乙处空虚再攻击 |
| **无从追责** | 无法证明幕后主使，只能"闷声吃下这亏" |

## 玄鉴仙族实例

**ch137-ch140 郁家坊市阳谋**（经典实例 #1）
- 准备：费望白向李通崖透露"古黎道上有盗贼"——既是找盟友表态，也是暗示计划
- 执行1：有人攻击密林郡本家阵法 → 郁家分兵回援
- 执行2：坊市空虚 → 两个筑基修士（"强盗"）攻击坊市
- 追加打击：诸家安排的"趁火打劫者"生乱偷盗
- 结果：客户逃走大半，人心惶惶，烂账一堆，郁家闷声吃大亏

## 变形

- **温水煮蛙**（ch137）：郁家对李家的断代策略也是阳谋的变形——偷偷做，不留证据
- **派系平衡**（ch139）：郁家派郁慕剑道歉+送请帖——软硬兼施也是一种微妙的阳谋

## 用法建议

1. 阳谋之前必有**铺垫环节**（费望白透露信息）
2. 阳谋过程中强势方的反应很重要——郁萧贵的震怒+无能为力，产生爽感
3. 阳谋之后需有**复盘环节**（ch140李通崖复盘）让读者看到全貌
