# 阵法大师投资

## 定义
顶尖阵法师以低于市场的价格、出色的技术为新兴家族布阵，换取未来的友谊与利益回报。

## 特征
| 维度 | 描述 |
|------|------|
| 投资者 | 阵法大师（一般修为不高但技术顶尖） |
| 投资方式 | 减免灵石费用、额外服务（一次覆盖两座山） |
| 回报 | 家族落脚权、后续布阵合作、长期关系 |
| 风险 | 家族可能势衰导致投资失败 |
| 时机 | 家族尚未崛起时介入（回报最高） |
