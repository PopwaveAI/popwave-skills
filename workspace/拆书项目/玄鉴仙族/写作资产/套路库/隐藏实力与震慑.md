# 套路：隐藏实力与震慑

## 定义
角色/势力/器物隐藏真实力量，在关键时刻展示以达成震慑、拉拢或威吓效果。

## 叙功能
- 制造剧情高潮（"底牌"的爆发）
- 构建信息不对称（读者知/角色不知/部分角色知）
- 确立世界观中的力量层级

## 本卷实例（ch043）

### 核心：法鉴的第一次出手

| 维度 | 内容 |
|------|------|
| 隐藏方 | 黎泾山顶的法鉴（实际有器灵陆江仙） |
| 被震慑方 | 万萧华（玉京轮巅峰）、李通崖/李木田（自己人也震惊） |
| 展示方式 | 太阴玄光——白色月华喷涌而出，如流星破空 |
| 击杀效果 | 从玉京轮狼妖脖颈上"轻巧地越过"，秒杀 |
| 对外说辞 | 李通崖谎称"父亲自制的符箓" |

### 震慑的层次感

1. **先抑**：万萧华全程被压制→狼狈逃命→底牌不出就要死
2. **再震**：白芒"理也不理地越过他"—秒杀妖物→万萧华"目瞪口呆"
3. **后隐**：给"符箓"说辞收尾，将真正力量重新隐藏

### 隐藏实力的策略价值
- **陆江仙的动机**：怕被紫府金丹修士控制，必须伪装为无意识器物
- **李家的需要**：防止其他势力觊觎法鉴；维持"李木田符箓"的外部形象
- **叙事埋点**：为后续"为何不直接用此法鉴对付人类敌人"做铺垫（只能诛邪除妖）

### 变体与创新
- 非"主角扮猪吃虎"，而是**器物有智但伪装无意识**——更稀有、更有潜力
- 不是一次性展示，而是"展示即收回"——保持神秘感
