# 青穗剑仙传说

## 定义
剑仙李尺泾的名声在修仙界传播，成为李家最重要的无形资产。

## 特征
| 维度 | 描述 |
|------|------|
| 剑仙本名 | 李尺泾 |
| 称号 | 青穗剑仙（剑意领悟者） |
| 剑法特征 | 拔剑式起手——或大若白帆，或飘忽如星 |
| 本命剑 | 青尺剑（青白一体，颇为瑰丽） |
| 归属 | 青池宗，青穗峰 |
| 现状 | 被派去南疆深处寻药，七十年未归 |
| 隐忧 | 落在青迟魔门手上，修了神妙道基，迟早成老魔资粮 |
