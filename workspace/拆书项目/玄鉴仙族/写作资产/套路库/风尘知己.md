# 风尘知己

## 定义
修仙者在世俗中为风尘女子赎身，虽因身份差距不能明媒正娶，但留下真挚情感牵连。

## 特征
| 维度 | 描述 |
|------|------|
| 赎身价 | 二十两银（对修仙者不值一提） |
| 安置方式 | 买宅院安置，定期探望 |
| 情感性质 | 同情→怜爱→责任→羁绊 |
| 冲突点 | 家规森严 vs 私情难舍 |
