# 套路：首次杀戮·心理突破

## 核心公式

```
少年第一次上战场 → 武力碾压/杀得尽兴（狂暴状态）→ 撞上一个"不该杀"的人（老人/孩子/无辜者） → 目光对视 → 心理冲击/停滞 → "原来……如此……"的冷静觉醒 → 平静地继续杀戮
```

## 关键阶段

1. **上场与保护**：少年第一次上战场——通常有护卫/保护
   - 李玄宣周围有十二名族兵保护（ch080）
   - 不是孤立无援——但保护圈的收紧也意味着战斗的烈度有限
   - 初次上阵的少年通常处于"亢奋/好奇/恐惧"混杂的状态

2. **狂暴状态**：杀得尽兴——被战斗激发出本性中的狠厉
   - "杀得尽兴，仿佛也激起了骨子里的狠厉，双目通红"
   - 修士武力对凡人/低级修士的碾压——"一扫三四个"
   - 这种状态下的杀戮是没有心理负担的——因为"他们是敌人"

3. **撞击时刻**：遇见一个"特殊"的敌人——触发心理机制
   - 一个老山越——"满头白发，深深的皱纹，惊恐的目光"
   - "年龄够当他爷爷"——触发同理心/伦理想象
   - 目光对视——不是砍一个"敌人"，是杀一个"人"
   - 这个时刻是"面纱被揭开"的瞬间——战争的美好滤镜碎了

4. **心理崩塌**：被对方的目光/表情/反应击穿了心理防线
   - "吓得李玄宣手脚一麻"——条件反射般的恐惧
   - 老人咬他——对他而言是"他在反抗/他在恨我"
   - 条件反射般贯杀老人——不是主动的"杀"，是被动的"杀"

5. **死亡之丑陋**：对死亡过程的不加修饰描写——剥离浪漫化
   - "骨头尽裂、鲜血搅和的粪便、不似人声的惨叫"
   - 不是"优雅地死去"——是真实的、丑陋的、生理性的死亡
   - 眼睛里的仇恨目光——死亡不是终结，仇恨会延续

6. **"原来……如此……"觉醒**：从狂暴到平静的突变
   - "原来……如此……"——不是崩溃，是**理解**
   - 世界的面纱揭开——"原来这就是战争""原来杀人就是这样"
   - 不是"我受不了了"——而是"我知道了"
   - 从**情绪驱动的杀戮**（杀得尽兴/双目通红）到**认知驱动的杀戮**（平静、一言不发）

## 心理阶段变化

```
亢奋/好奇 → 狂暴/屠杀快感 → 撞击/停滞 → 崩溃/恶心 → 
冷静/理解 → 平静/接纳 → 继续杀戮（但不再是"杀得尽兴"）
```

## 描写要点

| 阶段 | 描写要素 | 作用 |
|------|---------|------|
| 狂暴 | 双目通红、一扫三四个、热血沸腾 | 建立战斗的"快感"假象 |
| 撞击 | 老人的目光、白发、皱纹 | 触发同理心的视觉细节 |
| 崩塌 | 手脚一麻、泪光闪闪、寒意钻脊柱 | 生理反应——身体先于大脑做出反应 |
| 死亡 | 粪便、惨叫、仇恨的目光 | 死亡的丑陋——消除浪漫化 |
| 觉醒 | "原来……如此……"、平静、一言不发 | 认知升级——不是崩溃而是坚硬 |

## 评价维度

| 维度 | 优秀处理 | 糟糕处理 |
|------|---------|---------|
| 对死亡的态度 | 认识到丑陋但接纳 | 崩溃/呕吐/留下心理阴影 |
| 心理走向 | 变硬/变冷静 | 变软弱/变恐惧 |
| 叙事重心 | 认识世界的"面纱被揭开" | 停留在"杀人好可怕" |
| 后续状态 | 可以正常战斗（但不同了） | 无法继续战斗 |

## 适用场景

- 温室中长大的天才少年第一次经历实战
- 主角/主要配角从"和平状态"进入"战争状态"的转折点
- 家族叙事中需要完成"下一代传承者"的成人礼

## 注意事项

- 不要走"崩溃→吐→哭→被安慰→振作"的俗套——太软
- 走"撞击→停滞→觉醒→变硬"的路线——更符合《玄鉴仙族》的冷峻风格
- 老人的设置很关键——年龄、外貌、反应都决定冲击力
- "原来……如此……"四个字是核心——简洁、有力、有余韵
- 觉醒后的杀戮要"平静"——比狂暴更可怕的冷静
