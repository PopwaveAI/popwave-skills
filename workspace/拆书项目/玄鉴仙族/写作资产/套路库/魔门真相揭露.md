# 套路：魔门真相揭露

## 核心公式

```
仙门表象（表层认知） → 知情者/当事人揭露（黑暗真相） → 历史旧名/起源揭示（世界观颠覆）
```

## 关键阶段

1. **表象建立**：通过日常叙事展示某势力的"仙门正道"形象
   - 青池宗：三宗七门之一，收取供奉，表面维护秩序（ch049-050）
2. **知情者揭露**：由某个看透局势的角色（通常为同阵营的局内人）揭露黑暗本质
   - 汲登齐："你以为我们是什么？我们不过是这两家养的猪狗"（ch071）
   - 萧初庭："花费几百年种出个大世家，然后将其连骨带肉吃得干干净净"（ch071）
3. **证据佐证**：用具体案例来证明揭露内容
   - 月明琉璃树种在李江群的骨肉中（ch071）
4. **历史旧名**：揭露该势力的前身/旧名，彻底颠覆认知
   - 青池宗→青迟魔门（ch071）

## 情绪曲线

```
平静(表层认知) → 震惊(被揭露) → 怀疑(内心挣扎) → 接受并认知颠覆(升华)
```

## 揭露者必要条件

| 条件 | 说明 | 示例 |
|------|------|------|
| 身份合理 | 必须是有资格知晓真相的人 | 汲登齐(执行者)/萧初庭(世家老祖) |
| 动机合理 | 为什么愿意揭露？ | 汲登齐：对自身处境的愤怒；萧初庭：教育后代 |
| 话语可信 | 用具体案例支撑（不空谈） | "月明琉璃树种在李江群骨肉中" |

## 双重揭露结构（ch071）

本章使用了"平行揭露"：汲登齐揭露"具体操作层面"（祭杀凡人）+ 萧初庭揭露"制度分析层面"（养蛊逻辑）

两条线相互印证：

```
汲登齐线：具体行动（祭杀） → 残酷在场
萧初庭线：制度分析（养蛊） → 哲学揭示
```

## 适用场景

- 需要对已有认知的世界观进行颠覆时
- 需要提升作品思想深度时
- 主角势力需要认清所处环境的真实规则时

## 注意事项

- 揭露不能停留在"空喊黑暗"——必须有具体的案例/证据支撑
- 揭露者的身份必须合理——不能是随便一个路人甲
- 旧名揭示带来的冲击力远大于"直接说这个宗门坏"
