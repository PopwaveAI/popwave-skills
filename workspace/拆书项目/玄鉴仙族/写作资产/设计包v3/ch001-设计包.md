## 2. 爽点设计(爽点层)

### 套路识别：穿越异界·镜灵觉醒
**四段式：**
- **起势**：陆江仙猝死，残魂附于铜镜，坠落修仙世界
- **蓄势**：河底孤独漂浮，观察环境，确认自身处境（"我不做人了"）
- **爆发**：月光入体→吸收月华→凝气成功→内视镜身→镜子发光⭐
- **余波**：确认修炼可行，制定长期计划（藏身、学语言、探索世界）

### 情绪弧线
- **起点**：困惑、迷茫（"好美"→"我不做人了？"→"我好像挂了？"）
- **推进**：接受、好奇、压抑（河底飘荡，观察鱼蟹，无聊）
- **高潮**：惊喜、狂喜（月光入体，凝气成功，内视成功）
- **终点**：希望、警惕、规划（"先好好吞吐月华"）

### 爽点机制
| 机制 | 说明 | 触发点 |
|------|------|--------|
| 🎯 **信息差** | 读者知道这是穿越+镜灵，视角新奇；凡人世界不知铜镜有灵智 | 第7-8事件 |
| 📊 **数值成长** | 从0到1：无→凝气成功→体内气流 | 事件7 |
| 😤 **情绪释放** | 从"不做人了"的绝望到"修炼有望"的希望反转 | 事件7-8 |
| 💥 **新奇感** | 铜镜视角看世界、内视自身镜体 | 事件3/8 |

## 4. 感官与画面(感官层)

### 环境基线
| 维度 | 设定 |
|------|------|
| ⏰ **时间** | 从黄昏→夜晚→晨曦（跨越一整夜） |
| 🌤️ **天气** | 晴朗，月光皎洁 |
| 💡 **光线** | 黑暗→白光爆发→蔚蓝天空→夕阳→月光→晨曦 |
| 🏞️ **空间** | 河底（1米限制视角）、河面倒影、俯瞰全景 |
| 👃 **气味** | 略（河底无特殊气味描写） |
| 👂 **声音** | "叮叮咚咚的水花撞击声"、鱼游声、玻璃破碎声 |

### 环境变化时序
| 时段 | 光线描写 | 情绪 |
|------|----------|------|
| 初始 | "光怪陆离的色彩""浓密的黑暗" | 混沌、混乱 |
| 白光爆发 | "灿烂的白光划破黑暗""金色咒文如星辰" | 震撼、美 |
| 俯瞰世界 | "蔚然如大海的天空""波光粼粼" | 惊奇 |
| 夕阳 | "灿烂的夕阳爬满天空" | 落寞 |
| 月夜⭐ | "清亮的月光柔和地飘在河面上""淡白色的月晕" | 希望、惊喜 |
| 晨曦 | "暖暖的晨曦撒在河面上" | 新生 |

### 氛围渲染要点
1. **视觉奇观**：从黑暗到咒文爆发再到俯瞰世界的镜头拉远，营造穿越的文学质感
2. **孤独感**：鱼蟹环绕却无人交流的河底画面，衬托镜灵的孤独处境
3. **修炼仪式感**：月光→月晕→凝气→内视，节奏由缓转急，营造突破感
4. **身体化意象**：铜镜视角的"胸闷气短""磕在青石上"——用人类身体感受描写非人形态

## 5. 套路归档

### 套路：穿越异界·镜灵觉醒

**归档文件**：`套路库/穿越异界·镜灵觉醒.md`

**四段式模板：**
1. **起势**：主角穿越/死亡→灵魂附着于器物
2. **蓄势**：器物视角观察新世界，处境孤独而危险
3. **爆发**：首次利用新身份获得成长/能力（吸收月华凝气）
4. **余波**：确认变强路径，制定生存策略

**触发条件：**
- 主角从原有世界死亡
- 灵魂转移至非人形态（器物/动物等）
- 新世界存在可吸收的能量/灵气

**核心机制：**
- 器物视角限制（视觉/听觉/行动受限）
- 利用环境自然能量（月华）实现第一次突破
- 内心独白驱动（无法与他人直接交流）
- 迫切的隐藏需求（避免被修仙者发现）

**原文案例：**
> "清亮的月光柔和地飘在河面上，陆江仙惊喜地感觉到一股清凉的气息渗透到河底……月光一点点汇聚在身体上方，仿佛活过来般形成了一抹淡白色的月晕……他只觉得周身一凉，陷入一种似睡非睡的冥想之中。"
> "体内果然有一股淡淡的气流在徘徊，围绕着身体的圆边做圆周运动。"
