## 2. 爽点设计(爽点层)

### 套路识别：凡俗家族·天降异宝
**四段式：**
- **起势**：凡人李家在修仙者争斗阴影下生活，李木田忧心忡忡
- **蓄势**：李项平日常抓鱼→偶然摸到河底异物→发现残破铜镜
- **爆发**：陆江仙首次接触凡人，学习方言，观察李家
- **余波**：铜镜被带入村庄，陆江仙计划融入人类世界

### 情绪弧线
- **起点**：压抑、忧虑（李木田担心仙人打架殃及村子）
- **推进**：平淡→好奇（李项平抓鱼→发现镜子）
- **高潮**：惊奇、希望（陆江仙发现能探知心理、学语言）
- **终点**：谨慎期待（"走一步看一步"）

## 4. 感官与画面(感官层)

### 环境基线
| 维度 | 设定 |
|------|------|
| ⏰ **时间** | 寅时(凌晨)→清晨→上午 |
| 🌤️ **天气** | 清晨薄雾→晴朗 |
| 💡 **光线** | 黑漆漆→清晨薄雾→朝阳 |
| 🏞️ **空间** | 李家破屋→村路→眉尺河滩涂/河底→筐中 |
| 👃 **气味** | "炊烟袅袅"（烟火气） |
| 👂 **声音** | "鸡鸣狗吠"、水流声、鱼跃声、李木田叫喊声 |

### 关键场景画面
**李木田的破屋：**
> "黑漆漆的透出一点点辉光"——贫困的农村氛围

**眉尺河：**
> "眉尺河又浅又宽，河岸边大片滩涂和芦苇，村里成几十几百养的鹅鸭"——田园牧歌感

**陆江仙的筐内视角：**
> "与青尾大眼瞪小眼地看着"——器物视角的荒诞喜剧感

### 氛围渲染要点
1. **乡村生活质感**：寅时醒、破屋顶、鸡鸣狗吠、炊烟、抓鱼——细节堆叠出一个真实的山村
2. **双重视角差异**：李项平的粗糙动作 vs 陆江仙的精密感知，形成认知反差
3. **紧张感的隐藏**：李木田的开篇忧虑（仙人争斗）为后续暗线埋下伏笔
4. **语言学习的巧思**：用"探知心理活动"的能力解决跨语言难题，不落俗套

## 5. 套路归档

### 套路：凡俗家族·天降异宝

**归档文件**：`套路库/凡俗家族·天降异宝.md`

**四段式模板：**
1. **起势**：凡人/底层家族在修仙世界边缘艰难生存
2. **蓄势**：家族成员偶然发现/拾取一件看起来普通的物品
3. **爆发**：物品显露出非凡特性（吸收月华/发光等异象）
4. **余波**：家族面临选择——是上交还是私藏，风险与机遇并存

**触发条件：**
- 修仙世界观中存在仙凡鸿沟
- 家族处于修仙界的边缘/底层
- 家族成员（通常是最不起眼的那个）偶然获得超凡物品

**核心机制：**
- 凡人对超凡物品的认知偏差（"以为是破烂鉴子"）
- 家族利益 vs 个人贪婪的张力
- 异宝的隐蔽性 vs 暴露风险
- 双线叙事（器物灵智视角+家族视角）

**原文案例：**
> "李项平把这手里物什一看。这东西大概巴掌大小，中心青灰色的圆形小盘……正面碎成了七八块……倒是有点像姨母的那枚鉴子。但自己手上这枚鉴子也太模糊了"
> "眉尺河里的鱼儿决计没有这般蠢，这青尾十有八九是上游窜下来的野鱼儿"
