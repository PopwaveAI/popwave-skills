## 2. 爽点设计(爽点层)

### 套路识别：家族底牌·祖辈遗泽
**四段式：**
- **起势**：李家面临风险——李叶盛可能泄露铜镜秘密
- **蓄势**：通崖提议灭口 → 木田反问驳回 → 家族内部观点碰撞
- **爆发**⭐：李木田纵身取木盒 → 展示珍藏（越兵战法/符箓/碎银）→ 讲述从军经历
- **余波**：制定应对策略（碎琉璃掩护），分散风险，锁定下一步

### 情绪弧线
- **起点**：紧张、恐惧（李叶盛被抓、泄密风险）
- **推进**：争辩、肃杀（通崖提出杀人、木田反问）
- **高潮**：震撼、敬畏（木盒打开、越兵战法曝光、从军经历披露）
- **终点**：希望、谋划（对策制定成功）

### 爽点机制
| 机制 | 说明 | 触发点 |
|------|------|--------|
| 🎯 **信息差(家族底牌)** | 读者和孩子们都不知道李木田藏了什么——木盒打开时产生信息释放的爽感 | 事件7 |
| 💥 **打脸(性格反差)** | 李木田一直低调务农→实际上是杀人从军的老兵，有越兵战法和符箓 | 事件7-8 |
| 📊 **数值(家族底蕴)** | 李家不只是普通农户——有军伍传承、有符箓遗物、有战略眼光 | 事件7-9 |
| 😤 **情绪(家族认同)** | 通崖"我去杀了他"的狠厉→木田反问的逻辑→最后统一策略，家族凝聚力提升 | 事件5-9 |

## 5. 套路归档

### 套路1：家族底牌·祖辈遗泽

**归档文件**：`套路库/家族底牌·祖辈遗泽.md`

**四段式模板：**
1. **起势**：家族面临外部威胁/决策困境
2. **蓄势**：家族内部观点碰撞、看似孤立无援
3. **爆发**⭐：长辈打开隐藏的底牌（从军经历/祖传武学/遗物/秘密关系）
4. **余波**：利用底牌制定应对策略，家族凝聚力提升

**触发条件：**
- 家族看似普通，但长辈有不为人知的经历
- 家族面临危机需要底牌破局
- 有合适的"展示场景"（家族会议/危机时刻）

**核心机制：**
- 利用藏宝/隐藏空间制造惊喜感（屋梁木盒）
- 底牌与当前困境形成逻辑闭环（琉璃掩护鉴子）
- 底牌展示同时也是角色背景披露

**原文案例：**
> "李木田摇摇头，抬手将木桌移正，纵身一跃，在桌上轻轻一点，从屋梁上抽出一块木盒来。将木盒放在桌面上……盒中躺着一道木简、一张符箓、几块碎银琉璃之类的杂物"
> "我十三岁离乡，沿着古黎道进了安黎县。杨将军接了朝廷命令攻打山越，在古黎道征兵，当时你爹我无处容身，只好当了兵"

### 套路2：家族决策·风险管控

**归档文件**：`套路库/家族决策·风险管控.md`

**四段式模板：**
1. **起势**：某成员发现/持有危险物品或信息
2. **蓄势**：家族成员评估泄密风险，辩论应对方案
3. **爆发**：最极端的方案被提出（灭口/弃宝/逃亡）→被否决或通过
4. **余波**：共识达成，分工执行，统一口径

**触发条件：**
- 家族持有不能被外界知晓的秘密
- 有第三方（无意或有意）接触到秘密边缘
- 家族中有不同性格的决策者

**核心机制：**
- 家族会议+观点碰撞（宽厚vs狠厉vs老练）
- 风险评估梯度（杀/藏/换/逃/瞒）
- 统一口径的对外策略

**原文案例：**
> "李通崖：'我去杀了他！'……李木田：'那若是伱幼弟吃里扒外，嘴上不严，你也一并杀了？'……李通崖：'咱家没有这样的孬种'"
> "李木田：'明天我会拿这块琉璃，对你二伯说这是项平从河里捡来的，不管李叶盛见没见着，我们一口咬定就是一枚碎琉璃，不过在月光下闪烁罢了。'"
