## 2. 爽点设计（爽点层）

### 套路识别

**套路1：联姻结盟（世俗联姻稳固根基）**
- 起：李木田提雁纳采
- 承：田芸羞涩答"稀罕"——两情相悦
- 转：田守水忆兄弟情——两代交好
- 合：婚事定下，李家通过联姻巩固世俗根基

**套路2：血色往事伏笔（屠戮余孽危机）**
- 起：田守水提及元家旧事
- 承：怀疑有未杀尽的族人偷偷祭祀
- 转：李木田询问"还有未杀尽的族人？"——眼神阴郁
- 合：望着大黎山若有所思——暗流涌动

**套路3：家训警示（仙路危机意识）**
- 起：李项平婚事开心，气氛轻松
- 承：李木田放下茶杯，严肃开口
- 转：指出仙路恐怖、法鉴是祸事
- 合：叮嘱速留子嗣——未虑胜先虑败，李家生存哲学

### 情绪弧线
温馨轻快（提亲→田芸答"稀罕"）→ 暗流微涌（元家往事→望向大黎山）→ 日常祥和（三兄弟研习）→ 喜讯欢乐（提亲成功→李项平跃起）→ 严肃沉重（李木田警示仙路危机）→ 满怀希望（李尺泾报喜→李通崖内心独白）

### 章末钩子
**L4** — 李木田的警示"这法鉴即是我家的机缘，也会是我家的祸事"与元家余孽的伏笔叠加。李通崖"修仙之道，李通崖来了"的内心独白与李尺泾即将凝聚玄景轮的消息形成双重收束——既有希望又有危机。夏至即将到来，剩下的五枚符种将如何分配？

## 4. 感官与画面（感官层）

### 环境基线6维

| 维度 | 基线描述 |

| 视觉 | 田家院前择菜的田芸→李木田背着手提大雁→田芸从耳根红到脖颈→田守水出迎→李家院中三兄弟各忙各的（翻布帛、刻木简、算田契）→李尺泾冲出屋 |
| 听觉 | 择菜哼歌声→院中对话→"稀罕！"→田守水大笑→院中李项平跃起→李通崖刻刀声→李尺泾清脆叫声→兄弟们的欢笑 |
| 触觉 | 大雁羽毛触感→清茶温热→刻刀在木简上刻划→李木田抹鬓角白发的触感 |

| 味觉 | 清茶回甘 |
| 体感 | 提亲时的轻松愉悦→谈及元家时的冷意→李木田警示时的凝重→李尺泾报喜时的振奋温暖 |

### 氛围渲染
全章以"结亲"为表、"结忧"为里。前段提亲画面温馨：李木田背手提雁、田芸择菜哼歌、"稀罕"答得又羞又直。中段田守水提及元家时，阳光下的对话骤然蒙上阴影——"未杀尽的族人"、"偷偷祭祀"、"望着大黎山"三个短语将危机感拉满。后段院中兄弟三人的日常（翻布帛、刻木简、算田契）是全书最平静的画面之一——仙法已经成为日常一部分。李木田坐在台阶上喝茶说话的画面：一个老兵出身的父亲，在院中向四个儿子交代后事般的叮嘱。章末李尺泾冲出的活泼与李通崖内心独白的克制形成对比——全家都在向前走，但前路既有霞光也有暗影。
