## 2. 爽点设计(爽点层)

### 套路识别

**套路①: 怀璧其罪**
- 起: 夏至祭祀, 法鉴赐符种, 一家四子皆可修仙
- 承: 李尺泾率先凝聚玄景轮, 全家欣喜
- 转: 检测灵窍发现四人皆无天生灵窍, 全凭法鉴符种
- 合: 李尺泾顿悟法鉴是灭门祸根, 下令绝对保密

**套路②: 天赋觉醒**
- 起: 李尺泾是四兄弟中最小的, 一直受兄长庇护
- 承: 他率先完成玄景轮, 成为李家第一位修仙者
- 转: 发现只有自己因符种可修行, 但同样无灵窍
- 合: 原本稚嫩的少年骤然成熟, 承担起守护家族秘密的重任

### 情绪弧线
```
庄严(夏至祭祀) → 期待(白丸入体) → 赞叹(玄景功成) → 欣喜(兄弟交流) → 震惊(查无灵窍) → 恐惧(法鉴秘密) → 凝重(家族警示) → 压迫(走向未知)
```

### 章末钩子
**L4 — 世界真相揭露钩**
"我已炼成玄景轮...那些练气士也无法用法力探知你灵窍所在" —— 揭示这个世界存在"练气士"这一更高阶存在, 李家在修仙界中如履薄冰。

### 关键对白

**李尺泾 → 李木田:**
> "父亲！您说的对, 这法鉴亦是我家灭门之祸。"
语气: 颤抖而坚定, 从震惊中迅速冷静, 带有少年骤然成熟的庄重。
潜台词: "我明白了我背负的重量, 我准备好承担了。"

**李木田 → 众子:**
> "行了, 你们修炼去吧, 我去田上看一看。"
语气: 表面平淡, 实则内心沉重——背手去田上是李木田的习惯性排解动作。
潜台词: "我需要时间消化这个秘密, 你们先做好自己的事。"

## 4. 感官与画面(感官层)

### 环境基线6维

| 维度 | 设定值 | 原文依据 |

| 时间 | 夏至清晨→入夜 | "天色方才灰灰暗" → "月光下" |

| 声音 | 雨声→虫鸣→寂静 | 开篇雨景, 后场虫鸣 |
| 气味 | 泥土味/新麦香 | "地里光秃秃的, 散发着泥土味" / 新麦收割 |

### 氛围渲染要点

1. **开篇田园**: 雨点打过田地, 新麦扎束插在田野, 农户在田埂上招呼 —— 以宁静乡村烘托即将到来的超凡事件
2. **修行场景**: 螭龙纹镜架支法鉴, 案上摆新麦瓜果三杯夏雨茶 —— 朴素祭祀与仙家法器的反差
3. **玄景轮功成**: "一道亮银色的光弧宛若水中月般从清潭中浮现, 晶莹剔透, 仿若极细的月牙" —— 视觉上的晶莹美感
4. **绝望转折**: "大滴大滴的汗水从他俊俏的脸庞上滑落, 沾湿了他淡青色的衣袍" —— 从美感转入艰难的肉身感受
5. **终场凝重**: 李木田"背着手到田上去了" —— 以日常生活动作承载巨大压力
