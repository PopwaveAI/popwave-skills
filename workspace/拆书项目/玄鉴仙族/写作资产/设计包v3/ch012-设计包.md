## 2. 爽点设计(爽点层)

### 套路识别

**套路①: 天赋困境与分流**
- 起: 李长湖修行三月无进展, 自卑愧疚
- 承: 鼓起勇气向父亲坦白, 请求另寻出路
- 转: 李木田安排他负责俗务(婚礼筹备), 家族人才分流
- 合: 任屏儿怀孕消息传来, 长湖在世俗血脉延续中找到新价值

**套路②: 法术初显威**
- 起: 李项平从入定中醒来, 展开《金光术》木简
- 承: 读罢两个晚上便领悟诀窍, 掐诀凝聚金光
- 转: 抱青石做靶, 金光术劈斩留刀痕——展示攻击力
- 合: 兄弟评价法术好用, 暗示李家有了战斗手段

### 情绪弧线
```
压抑(长湖入定失败) → 自卑(自觉无天赋) → 决绝(向父坦白) → 释然(被安排俗务) → 欣喜(得知有孕) → 期待(项平展卷) → 惊叹(金光初显) → 忧虑(谈论大哥) → 警觉(发现窥视)
```

### 章末钩子
**L5 — 危机引爆钩**
"贼子好胆！" —— 后山有人窥视, 李家秘密面临暴露危机, 下一章直接爆发冲突。

## 3. 角色与人设(角色层)

### 登场角色行为锚定

| 角色 | 本场行为 | 性格锚点 |
|------|----------|----------|
| 李长湖 | 七次入定失败 / 向父坦白放弃修仙 / 得知妻子怀孕大喜过望 | 宽厚仁善但自卑, 重情重家; 原文"眉眼柔和了不少"体现对妻子的温柔; 从压抑到狂喜的对比 |

| 任屏儿 | 在正院等夫君 / 羞涩告知怀孕 | 端庄大气; 原文"容貌端庄大气, 眉眼带着点喜色" |

| 李通崖 | 旁观项平施法 / 与项平论大哥天赋 | 沉稳冷静; 原文"早已从入定中醒来, 望着李项平笑道" |

### 关键对白

**李长湖 → 李木田:**
> "父亲。/ 我恐非修仙那块料。"
语气: 羞愧、底气不足, 知道会让父亲失望。
潜台词: "我试过了, 我真的不行, 我不想再拖累弟弟们。"

**李木田 → 李长湖(得知怀孕后):**
> "我去请先生, 你好生看着她。"
语气: 喜不自禁却强行压住, 立刻转入行动模式。
潜台词: "你是要当爹的人了, 修仙不行就在家族血脉上出力。"

**李项平(喝出):**
> "贼子好胆！"
语气: 厉声、愤怒、警觉, 不含丝毫犹豫。
潜台词: "果然有人觊觎我李家秘密, 找死。"

## 4. 感官与画面(感官层)

### 环境基线6维

| 维度 | 设定值 | 原文依据 |
|------|--------|----------|
| 时间 | 夜间(多个月夜推进) | "修炼了三月有余" | 后场"皎洁的月光下" |

| 光线 | 月光(时明时暗) | "皎洁的月光下" / "月色稍微不好的夜里" |
| 声音 | 虫鸣→对话→法术破空 | 后院虫鸣 / 铁器碰撞声 / 金光劈石声 |
| 气味 | 磨刀铁腥味/泥土味 | "当锵…"铁器碰撞 |

### 氛围渲染要点

1. **磨刀开场**: 李木田在后院弯着腰打磨长刀, "右手握着刀背, 左手拇指与食指捏住刀身" —— 具象化的手部动作传递出李木田的务实与肃杀
2. **月光下的木简**: "皎洁的月光下, 卷首的几个大字赫然在目: 《金光术》" —— 简洁的画面感
3. **金光劈石**: "亮色的火星子一闪而过, 青石表面多了一道不深不浅的刀痕, 倒是足足后退了一掌之距" —— 视觉与力量的结合
4. **孕妇静坐**: 任屏儿坐在院中缝制衣服, 与田芸聊天 —— 日常生活的宁静与即将到来的新生形成对照
5. **终场警觉**: "李项平却是一眯眼, 骤然站起" —— 静态突然转入动态, 悬念拉满
