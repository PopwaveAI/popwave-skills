## 2. 爽点设计(爽点层)

### 套路识别

**套路①: 斩草除根**
- 起: 李叶盛因嫉妒起贪念, 认定李家有宝, 趁夜窥视
- 承: 李叶盛翻后山找到了视野开阔处, 看到了不该看的
- 转: 李项平不知何时已站在他身后, 无声无息
- 合: 金光术一击毙命, 碎尸喂兽, 不留痕迹

**套路②: 初次杀人·冷静型**
- 起: 李项平发现窥视者, 没有警告没有谈判
- 承: 毫不犹豫施展金光术, 一击斩首
- 转: 嫌弃地退一步避开血, 有条不紊地碎尸
- 合: 对野兽说"诸君慢用", 谈笑自若——与李木田"初杀人, 收刀饮酒, 谈笑自若"如出一辙

### 情绪弧线
```
烦躁嫉妒(叶盛) → 贪念滋生(叶盛) → 得意潜入(叶盛) → 疑神疑鬼(叶盛) → 恐惧(叶盛回头见人) → 震撼(金光闪过) → 冰冷(项平碎尸) → 震撼揭示(通崖点题)
```

### 章末钩子
**L3 — 人物揭露钩**
"是因为他足够残忍。" —— 李通崖用这句话点明了李项平的本性, 也暗示了李家未来的掌舵风格。读者对李项平的认识被颠覆——原来那个开朗懒散的少年, 骨子里是头狼。

### 关键对白

**李项平(杀完人后):**
> "诸君慢用。"
语气: 轻描淡写, 甚至带一丝调侃, 仿佛在宴请宾客。
潜台词: "处理干净了, 不会有人发现。"

**李通崖 → 李长湖:**
> "是因为他足够残忍。"
语气: 压低声音, 轻轻地说, 但字字千钧。
潜台词: "项平才是父亲选定的接班人, 不是你, 也不是我。"

## 4. 感官与画面(感官层)

### 环境基线6维

| 维度 | 设定值 | 原文依据 |
|------|--------|----------|
| 时间 | 深夜 | "夜色已晚" |
| 天气 | 多云, 月光被遮挡 | "明月又受乌云遮挡, 大地一片乌黑" |
| 温度 | 山间寒凉 | "凉风从李叶盛的裤衩子底下穿过, 冻得他双腿一夹" |
| 光线 | 昏暗→金芒一闪→极暗 | 大地乌黑 / "一道璀璨的金芒" / 过后血迹在黑暗中 |
| 声音 | 猿啸豺嚎→法术声→寂静 | "高猿长啸, 豺吠狼嚎" / "噗嗤噗嗤"碎尸声 |
| 气味 | 山间草木味→血腥味 | "热腾的血花染红了青石、土路、枯叶" → 浓烈血腥 |

### 氛围渲染要点

1. **开篇无赖气**: 李叶盛"叼着狗尾巴草"的吊儿郎当, 和不学无术的嫉妒心 —— 底层无赖的典型画像
2. **潜伏视角**: 从后山高处俯视李家大院, "模模糊糊的人影" —— 窥视者的不安与好奇
3. **夜山恐怖**: "高猿长啸, 豺吠狼嚎"与"凉风从裤衩子底下穿过" —— 冻出来的实感, 增强环境的冷酷
4. **致命转身**: "一扭头, 却见着山间小径上静静地站着个人影" —— 从窥视者到被监视者的反转, 恐怖氛围满格
5. **斩首瞬间**: "一会儿望见亮盈盈的明月, 一会儿又看见月光下模糊的小院" —— 飞起的头颅的残余视角, 画面冲击力极强
6. **诸君慢用**: 绿油油的眼睛(野兽)在林中阴影处闪烁, 李项平拍手告辞 —— 冰冷诗意般的残酷

### 战斗章: 环境强化
后山的黑暗(视线遮蔽) + 项平的无声潜行(信息差) + 金光的突然爆发(视觉冲击) = 战斗效果远强于正面对抗。项平利用地形和夜色完成了完美的伏击。
