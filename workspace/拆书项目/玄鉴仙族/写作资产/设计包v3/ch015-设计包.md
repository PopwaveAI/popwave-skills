## 2. 爽点设计(爽点层)

### 套路识别

**套路①: 伪装潜入——元家余孽归来**
- 起: 难民到村, 李长湖与柳林峰商议收留
- 承: 难民中一可疑青年盯上李长湖, 跪地哭泣博取同情
- 转: 徐老汉认出青年脚踝黑痣——二十多年前元家婴儿的标记
- 合: 李长湖上前去扶, 徐老汉与主事人同时喝止, 青年抬头露出野兽般凶眸

**套路②: 老人识破——旧日痕迹**
- 起: 徐老汉在人群中观察难民, 被青年锐利的眸子吸引
- 承: 青年跪下时裤管上缩露出脚踝, 黑痣被徐老汉捕捉
- 转: 记忆闪回二十多年前, 元家妇女抱婴儿、徐老汉讨好说吉祥话的场景
- 合: "慢着！"——这个躬了二十多年腰的老人, 第一次挺直了腰杆怒喝

### 情绪弧线
```
平静(陆江仙内心) → 疑惑(难民到来) → 谨慎(长湖处理) → 压抑(难民惨状) → 同情(柳林峰收留) → 不安(徐老汉疑心) → 惊觉(黑痣认出) → 危机爆发(喝止+凶眸)
```

### 章末钩子
**L5 — 生死危机钩**
"那双凶残锐利, 仿若野兽般的眸子。" —— 徐老汉与难民主事人同时喝止"慢着!", 但李长湖已经俯身去扶。青年猛然抬头的那双眸子, 宣告杀机已至。本章在生死一线间戛然而止。

### 关键对白

**徐老汉(喝止):**
> "慢着！！"
语气: 用尽全身力气, 嘶吼出从未有过的气势。
潜台词: "别碰他！他是元家的人！你父亲杀了元家满门, 他是来报仇的！"

**难民主事人(同时喝止):**
> "慢着！"
语气: 惊疑, 带着不确定但确定有问题的警觉。
潜台词: "这人不对, 我同路三个月竟没见过他, 他不是难民中的一员。"

## 4. 感官与画面(感官层)

### 环境基线6维

| 维度 | 设定值 | 原文依据 |
|------|--------|----------|
| 时间 | 黄昏/傍晚 | 李长湖忙完项平婚事"几日"后, 天色将晚 |
| 天气 | 晴 | 无雨, 可见度高 |
| 温度 | 夏末秋初, 温和 | 适宜户外活动 |
| 光线 | 日暮→渐暗 | 村口对峙, 光线充足到可辨认人脸 |
| 声音 | 难民嘈杂→哭声震天→喝止声 | "纷纷号啕大哭起来, 一时间哭声遍地" / "慢着！！" |
| 气味 | 难民身上的尘土汗味 / 田野气息 | 逃难多日的气味, 与村中田野气息形成对比 |

### 氛围渲染要点

1. **陆江仙内心**: 用"如芒在背"形容感受古黎道上磅礴气息, "将神识缩成一团"的卑微姿态——与法鉴"仙器"身份形成反差萌
2. **难民登场**: "衣衫褴褛"的难民与黎泾村的宁静形成对比; "刀兵四起"的信息打破了之前世外桃源般的村庄氛围
3. **可疑青年的眸子**: "眼睛如同带着火焰般灼灼地盯着"—— 反复用"火焰"形容眼神, 隐喻仇恨的燃烧
4. **黑痣的记忆闪回**: "金黄色的稻田在秋风中起伏, 那妇女抱着孩子到田上来" —— 温暖的秋收画面, 与现下的冰冷杀机形成对照
5. **老人的腰**: "躬了二十多年的腰"第一次"用力伸直" —— 身体动作的象征意义: 一个从未抬起头的佃户, 为了主人挺直了脊梁
6. **野兽般的眸子**: "凶残锐利, 仿若野兽" —— 与人类的外表形成对照, 暗示元家余孽已非正常人心态
