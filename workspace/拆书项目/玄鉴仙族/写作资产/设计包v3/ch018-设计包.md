# 第十八章「三年」设计包

---

## 一、骨架层 · 叙事结构

### 1.1 章节定位
- **卷轴位置**：第一卷「起始」第三段「家族筑基」过渡章
- **章节功能**：时间跳跃（三年），完成代际过渡；确立家族制度；推进修仙进展
- **核心矛盾**：时间冲刷伤痛 vs 家族必须向前；修仙与世俗治理的平衡
- **叙事节奏**：收尸善后→白事→时间跳跃→李玄宣出场→辈分确立→家族治理

### 1.2 事件链

| # | 事件 | 场景 | POV | 证据/原文标记 |
|---|------|------|-----|--------------|
| 1 | 徐老汉拖着青年尸体与头颅回到村口 | 村口山路 | 全知→难民视角 | "衣上臂上血迹斑斑，左手勉力拖行着一具血淋淋的尸体" |
| 2 | 徐老汉将尸首交与李家人，不肯受恩 | 李家院前 | 徐老汉 | "老汉没有几年可活了……便待那孩子出生，携来看看我" |

### 1.3 三幕结构

| 幕 | 起止事件 | 情绪曲线 | 功能 |
|----|---------|---------|------|
| 第一幕（收束） | 徐老汉拖尸→长湖下葬→通崖突破 | 悲凉→希望微升 | 收束前两章的死亡情节，开启新希望 |
| 第二幕（新生） | 两年后→李玄宣玩耍→辈分确立 | 温暖→传承 | 新生代登场，家族延续的象征 |
| 第三幕（治理） | 李叶生培养→家族治理讨论 | 理智→谋划 | 从情感过渡到务实治理 |

### 1.4 章节开关
- **钩子（开）**：徐老汉拖尸出现，作为前两章的血腥收尾
- **悬念（关）**：李尺泾即将突破，修仙十年为引——"修仙十年，真是叫人好生期待"

## 二、爽点层 · 情绪体验

### 2.1 核心情绪
- **伤痛愈合**：白事后李通崖突破，悲痛转化为家族前进动力
- **血脉延续**：李玄宣的活泼天真冲散死气
- **制度确立**：辈分诗创造仪式感与归属感

### 2.2 关键爽点

| 爽点类型 | 触发方式 | 强度 |
|---------|---------|------|
| 收尸祭奠 | 徐老汉拖尸进村 | ★★★ |
| 突破成仙 | 李通崖凝聚玄景轮 | ★★★★★ |
| 萌娃治愈 | 李玄宣捧花跑入后院 | ★★★★ |
| 辈分确立 | 八字辈分诗 | ★★★★★ |
| 家族治理 | 恩威并重的政治智慧 | ★★★★ |
| 修仙期待 | 尺泾说"修仙十年" | ★★★ |

### 2.3 情绪曲线
```
拖尸(惊悚→悲凉) → 白事(压抑) → 突破(希望↑) → 两年后(平静) → 萌娃(温暖) → 辈分(仪式感) → 治理(理智)
```

## 三、角色层 · 人物驱动

### 3.1 本章角色

| 角色 | 出场状态 | 角色功能 | 关键台词/动作 |
|------|---------|---------|-------------|
| 徐老汉 | 复仇→归隐 | 完成使命后退场 | "受了湖儿的恩德，就替他杀了这人" |
| 李通崖 | 突破成修仙者 | 家族首位正式修士 | "凝聚出玄景轮，踏入修仙之门" |
| 李尺泾 | 修炼即将突破 | 家族武力天花板预备 | "今夜我便可着手凝聚周行轮" |
| 李玄宣 | 两岁幼童（首秀） | 血脉延续象征，情感锚点 | "叔……抱……" |
| 李木田 | 祖父/家主 | 慈爱祖父+严厉族长 | 从茶饭不思到重新振作 |
| 李项平 | 家族管理者 | 理性统筹者 | "治人还须恩威并重" |
| 李叶生 | 被提及（未出场） | 旁系血脉培养对象 | "孑然一身，与我家格外亲近" |
| 任屏儿（任氏） | 母亲 | 为遗腹子取名 | 寻思一夜取名"宣" |

### 3.2 人物弧光
- **李木田**：从丧子的绝望中恢复，将情感转移至孙儿，同时展现家族治理的冷峻面
- **李通崖**：从兄弟转化为修仙者，家族的新一代武力支柱
- **李玄宣**：初登场即承载家族希望，李长湖的生命以另一种方式延续

### 3.3 对话节奏
- 前半沉重（徐老汉告别人世）→ 中段温暖（李玄宣咿呀）→ 后半务实理性（家族治理讨论）
- 辈分诗的公布带有仪式感，节奏放慢

## 四、感官层 · 沉浸体验

### 4.1 视觉意象
- **血迹拖痕**：徐老汉衣臂血迹斑斑，尸体拖过村口
- **锄头挂头**：锄头上挂着的残破人头，血腥震撼
- **白巾素缟**：满院白色丧事布置
- **李玄宣捧花**：两岁孩童捧着几朵野花蹦跳，象征新生
- **树荫院落**：阳光透过稀疏小树投射细碎树荫

### 4.2 听觉设计
- 村口难民骚动的议论声
- 白事的哭泣声
- 李玄宣的咿呀童语"叔……抱……"
- 李尺泾爽朗的笑声
- 李木田"胡说"的故作恼怒声

### 4.3 触觉与温度
- 徐老汉手脚冰凉、端不住茶碗的颤抖
- 清晨明朗温暖的阳光
- 李玄宣扭动的小身体
- 木简上的灰尘

### 4.4 环境氛围
| 场景 | 氛围关键词 | 营造手法 |
|------|-----------|---------|
| 村口拖尸 | 血腥、震慑 | 视觉冲击先于解释 |
| 白事 | 悲恸、克制 | 白巾素缟，家家有泣声 |
| 后院清晨 | 温暖、宁静 | 明朗阳光，稀疏树荫 |
| 辈分确立 | 仪式感、庄重 | 八字三句，节奏放慢 |
