# ch022 仙宗 · 设计包 v3

## 一、骨架（情节结构）

### 核心冲突
李尺泾入青池宗，以凡人视角沉浸式体验仙家气象，同时直面修仙界的残酷法则——温情下的师门关怀与"大争之道"的血淋淋真相形成巨大落差。

### 情节分段
| 段落 | 内容 | 篇幅占比 |
|------|------|----------|
| 1 | 李项平、李通崖等检视各村人口，部署掌事人与灵田种植 | 20% |
| 2 | 飞梭旅途→青池山门→护山大阵→灵气震撼 | 20% |
| 3 | 青穗峰认师→见三师兄萧元思、四师姐袁湍→收见面礼 | 25% |
| 4 | 祭拜已故同门→司元白揭示修仙残酷法则 | 25% |
| 5 | 两种视角并行：家族在凡间种田管理，子弟在宗门修行 | 10% |

### 核心情节点
- **A点（激励事件）**：飞梭腾空→李尺泾告别凡间，进入修仙界
- **B点（关键转折）**：司元白揭晓同门死亡真相→从温情转为残酷
- **C点（主题深化）**："争不赢便死"的训话→全章主题升华，也为李家命运定调

## 二、爽点设计

### 核心爽点

1. **仙宗奇观，视觉震撼**
   - 极高山群、亭台楼榭连绵不绝
   - 仙鹤白鸟穿梭群山之间
   - 护山大阵透明光幕的震撼感
   - "好高的灵气浓度"——体感上的升华

2. **师门温情，资源降临**
   - 三师兄赠剑诀（练气修士给的见面礼）
   - 四师姐赠安神玉佩（胎息境好用具）
   - 师徒关系的确立，靠山降临
   - 李尺泾内心："这阵法能不能学得，给我家后山也布上一个"

3. **反转震撼，认知升级**
   - 前半段：仙气飘飘、逍遥自在
   - 后半段：大师兄筑基失败身死道消、其他同门皆死于各种劫难
   - 从"大丈夫当乘风揽月"到"争不赢便死了"的认知觉醒

### 爽点节奏
| 阶段 | 情绪 | 手段 |
|------|------|------|
| 飞梭途中 | 兴奋/期待 | 俯瞰大地雄城，空间飞跃 |
| 入仙门 | 惊叹/震撼 | 护山大阵、灵鹤、灵气 |
| 见师门 | 温暖/安心 | 师兄师姐热情赠礼 |
| 祭拜同门 | 沉重/震惊 | 真相揭晓的冲击 |
| 司元白训话 | 震撼/觉醒 | 世界法则赤裸裸摊开 |

## 四、感官设计

### 视觉
- 飞梭上俯瞰：广阔大地、蚂蚁般的房屋、雄俊的城池
- 青池群山：极高山群、亭台楼榭连绵不绝
- 仙鹤白鸟在群山之间穿梭
- 护山大阵：几近透明的大罩子，光幕打开缺口
- 青穗峰：低矮的山峰（与其他峰对比）
- 萧元思：浓眉大眼，颇为潇洒大方

### 听觉
- 风声呼啸（飞梭飞行）
- 仙鹤开口说话的声音（奇妙震撼）
- 司元白骄傲介绍青穗峰以剑法闻名→尴尬改口
- 司元白冷峻训话："争不赢便死了"

### 触觉/体感
- 踏在飞梭上的失重感
- 进入山门后浑身一阵舒爽（灵气浓度暴增）
- 安神玉佩的微凉触感

### 味觉/嗅觉
- 山间清新空气
- 灵气充裕时的气息变化

### 环境氛围
| 场景 | 氛围 |
|------|------|
| 凡间李家院 | 务实、琐碎、家族经营 |
| 飞梭高空 | 自由、辽阔、向上 |
| 青池山门 | 神秘、震撼、仙气 |
| 青穗峰见师兄 | 温暖、善意、家族感 |
| 祭拜墓冢 | 沉重、肃穆、惨烈 |

| 飞梭旅途 | 亮蓝/金色 | 自由向往 |
