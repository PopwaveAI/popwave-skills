## 二、爽点设计

### 核心爽点

1. **悬念拉满，千中无一**
   - 一个个孩子被唱名上台
   - "将手在那孩子肩膀上一搭……"→"下去吧"
   - 梨川口全员落选的绝望感
   - 压得越紧，中选时的爆发越爽

2. **寒门逆袭，血脉回归**
   - 叶家（原李氏支脉）因灵窍得以归宗
   - 叶承福从忐忑到喜极而泣
   - 叶秋阳→李秋阳：姓氏改变的象征意义
   - "我叶家好歹也算李家支脉，怎么也有些仙缘在身上"

3. **人间百态，众生相绘**
   - 陈二牛的患得患失（恨自己孩子年纪错过）
   - 陈长户哀求的眼神
   - 陈二牛翻白眼摊手的无奈
   - 柳柔绚被选中时呆呆的反应
   - 叶承福一屁股坐地上闭目倾听的细节

4. **权力制衡，管理智慧**
   - 柳柔绚（异姓）调黎泾修行
   - 李木田的深谋远虑
   - "不够狠毒"——对自我管理风格的反思，为未来更冷酷手段埋伏笔

### 爽点节奏
| 阶段 | 情绪 | 手段 |
|------|------|------|
| 灵雨术种田 | 平静 | 日常劳作引入 |
| 孩童集结 | 期待/紧张 | 排队的压抑感 |
| 梨川口全员落选 | 失望/沉重 | 打击感 |
| 柳柔绚中选 | 惊喜/欢呼 | 第一次释放 |
| 李秋阳中选 | 双喜临门 | 感动+归属感 |
| 归宗仪式 | 温情/满足 | 家族认同 |
| 权力制衡密谈 | 冷峻/智慧 | 理性冷静 |

## 四、感官设计

### 视觉
- 灵雨术：细细密密的灵雨飘落在乌黑的泥土中
- 大槐树：枝繁叶茂，树下各色人群
- 柳柔绚："眉眼弯弯，眼睛明亮，胆怯地咬着唇"
- 李通崖搭脉检测的重复动作
- 祠堂上香、磕头、敬茶的仪式过程
- 叶承福"一屁股坐在沙地上，愣愣地倾听"
