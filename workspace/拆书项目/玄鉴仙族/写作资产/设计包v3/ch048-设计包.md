### 四段式结构

- **起**：三人互报家门，萧雍灵认出李家（与萧元思的关联）
- **承**：萧雍灵介绍宁婉情况、万家先祖万华芊
- **转**：三百年前紫府大战的信息重磅抛出 + 松林朔风修炼法
- **合**：告别 + 管事训随从镜头（讽刺收尾，点明修士与凡人的差距）

### 章节定位

**世界观揭秘章**——全章核心功能就是通过萧雍灵之口，向读者（和李通崖）揭示世界观的核心设定：
1. 三宗十门的格局与渊源
2. 三百年前大战的来龙去脉（月华元府→紫府修士战死）
3. 胎息晋练气的关键：天地灵气（以小清灵气为例）

### 章节节奏

```
报家门(慢) → 认出李家(中) → 修为误会(快/幽默) → 
宁婉信息(中) → 万家渊源(慢) → 三百年前大战(慢/重要) → 
修炼信息(慢/科普) → 告别(快) → 管事训人(讽刺)
```

## 三、角色层（人物弧光）

### 核心角色：萧雍灵（深入刻画）

| 维度 | 本章揭示内容 |
|------|------------|
| 修为 | 承明轮，靠宝物识别修士（非玉京灵识）→ "原来是靠宝物"这一设定降低了他的压迫感 |
| 性格深化 | 对同道热忱、自信、见多识广、喜欢炫耀情报 |
| 世界观地位 | 萧家嫡系——知道三宗秘辛，读过族史 |
| 价值 | 信息提供者（本章6个新信息点全部出自他口） |

### 角色三视角

- **李通崖视角**：求知若渴——每个信息点都追问
- **万元凯视角**：复杂心态——既因万家被提及而自豪，也因卢家练气而失落
- **中年管事视角**：恐惧——"向来嚣张跋扈"的族兄今日极为反常

### 本章人物出场公式

```
萧雍灵 = 信息带路人（世家子弟 + 见多识广 + 乐于分享）
```
