### 四段式结构

- **起**：悬念回收——宁婉交付灵石（"你便是那黎泾李家的人？"→解释清楚）
- **承**：供奉收尾 + 各家散去（场面过渡）
- **转**：萧初庭单独谈话+赐丹药（高光时刻）
- **合**：逛坊市+阵旗报价感受修仙经济体系（落回地面，为下阶段做准备）

### 章节功能定位

```
回收伏笔（灵石交付）→ 高光奖励（萧初庭赐丹）→ 扩展世界（坊市体系）
```

### 章节节奏

```
点名悬念回收(快) → 众目焦点(中/压抑) → 供奉收尾(慢) →
萧初庭留人(快/意外) → 长辈谈话(慢/温情) → 
赐丹高潮(快/霸气) → 坊市入口(中) → 
阵旗询价(慢/科普) → 呼应萧元思(中/闭环)
```

3. **认知奖励**：坊市见闻——修仙界的经济体系

## 三、角色层（人物弧光）

### 核心角色：萧初庭（本章高光）

| 维度 | 内容 | 原文证据 |
|------|------|---------|
| 外貌 | 老者 | "低眉读着书" |
| 修为 | 筑基修士 | "货真价实的筑基前辈" |
| 行事风格 | 说一不二、霸气但不失温情 | "拿上丹药走人，莫要聒噪" |
| 与李家关系 | 萧元思在青穗峰→知道李尺泾天赋→爱屋及乌 | "我那孙儿和你兄弟一同在青穗峰修炼数年" |
| 赐丹动机 | 不露骨的好意，用"丢了青穗峰的脸"当理由 | "家中连个玉京轮都没有，丢了青穗峰的脸" |
| 对李通崖 | 看穿他的拘谨，主动给他台阶下 | "送客！"（假装不耐烦） |

### 关键对话分析

**萧初庭台词三层解读**：
> "拿上丹药走人，莫要聒噪，家中连个玉京轮都没有，丢了青穗峰的脸。"

1. 表面：不耐烦，嫌弃
2. 潜台词：我给你丹药是合情合理的（因为你是青穗峰的人）
3. 用意：不给李通崖拒绝的机会+保全对方面子

### 李通崖的角色弧

| 本章状态 | 变化 |
|---------|------|
| 入场：被宁婉点名→紧张+受宠若惊 | → 灵石到手→感动酸楚 |
| → 被萧初庭留下→忐忑 | → 获得赐丹→惊喜惶恐 |
| → 逛坊市→回归冷静 | → 最后得出结论（萧家确实好心） |

心态弧线：紧张 → 感动 → 惶恐 → 感激 → 冷静反思

**清茶**——与酒肆的雪花酒对照：
- "一阵清甜从舌尖升起，脑海中一片清明"
- 暗示：这是灵茶？功能性的饮品

### 触觉

- 小茶杯的"捏着有些不适"——出身农户的细节
- 怀中丹药紧了紧——珍惜
- 摊位前"谨慎地没有用手去碰"阵旗——交易礼仪
