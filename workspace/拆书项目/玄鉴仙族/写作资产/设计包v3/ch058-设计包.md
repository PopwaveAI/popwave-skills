## 四、物品 / 设定

| 物品 | 性质 | 备注 |
|------|------|----------|
| 太阴月华 | 天地灵气（极品） | 三百年五道；可助筑基紫府；炼器（大雪绝锋） |
| 月湖映秋诀 | 五品功法 | 需太阴月华修炼；属青池宗；需立玄景灵誓 |
| 青玉瓶（法器） | 储存法器 | 从洞府所得→装入太阴月华→作为"洞府机缘" |
| 白玉瓶 | 储存法器 | 接收火中煞气→留家中使用 |
| 火中煞气 | 地火灵气（中等） | 采自地火口；用于锻造灵器精炼品质；不算珍贵 |
| 大雪绝锋 | 青池宗主法器 | 以太阴月华炼成 |
| 月湖峰主 | 历史人物（紫府修士） | 胎息时得太阴月华，修月湖映秋诀→紫府 |

## 五、核心冲突 / 策略

| 冲突 | 李家的困境 | 解决方案 |
|------|-----------|----------|
| 怀璧之罪 | 太阴月华太珍贵，一旦泄露必遭灭门 | 用洞府火中煞气作掩护→营造"从旧洞府获得机缘"的假象 |
| 功法门禁 | 月湖映秋诀需玄景灵誓，无法给全族用 | 短期：仅李尺泾修炼；长期：未提及 |
| 仙宗疑虑 | 青池宗会派员探查洞府 | 洞府历史三百年以上→可信度高；司元白筑基都未发现法鉴 |

- **瞒天过海计**的核心逻辑：用一个真东西（洞府、火中煞气）来掩护另一个更珍贵的东西（太阴月华、法鉴）

## 六、章节主题与写作要点

- **主题**：危机应对——如何保护超出家族承受能力的宝藏
- **写作亮点**：李项平教导李玄宣那段（"对仙宗来说我们就是一只蚂蚁"）——家族智慧的代际传承
- **节奏**：激动（太阴月华的价值）→恐慌（怀璧之罪）→冷静分析→定计→执行
- **伏笔**：司元白当年没发现鉴子→说明鉴子的隐藏能力极强
