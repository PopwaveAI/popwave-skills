## 一、骨架（Framework）

### 1.1 本章定位
- **卷轴位置**：青穗峰坊市+黎泾山日常线
- **时间跨度**：李尺泾归家前至离家前一整天
- **空间跨度**：望月湖坊市→大船→黎泾山后院/前院

### 1.2 情节结构

| 序段 | 内容 | 字数占比 |
|------|------|----------|
| 开端 | 李尺泾、李通崖、李项平逛坊市后返回 | ~15% |
| 转场 | 陆江仙梦境获《祭萃夺元法》 | ~15% |
| 发展 | 李尺泾演示符墨技法，传授符箓省钱秘诀 | ~20% |

| 尾声 | 李项平倾诉家族责任，兄弟情深 | ~20% |

| 编号 | 类型 | 爽点描述 | 触发时刻 |
|------|------|----------|----------|
| S1 | **系统爽** | 陆江仙梦中获得新的法诀《祭萃夺元法》，系统功能再次进化 | 梦醒段落 |
| S2 | **技法爽** | 箓气功能一览：修为/六识/根骨/资质/品相/补缺——全方位强化 | 法诀说明 |

## 三、角色（Characters）

### 3.1 角色弧线

| 角色 | 本章状态 | 变化/发展 |
|------|----------|-----------|
| **陆江仙** | 镜灵态 | 获得新法诀，依旧受限于镜身无法自愈；对符种（李尺泾）的监控持续 |
| **李尺泾** | 归家弟子 | 从依赖家族→成为家族的技术输出者（符术+剑术）；口称"死在外头"埋下长期flag |
| **李通崖** | 二弟/习剑者 | 练成剑芒但无法剑气离体，主动求教，勤奋用功 |
| **李项平** | 三弟/家主 | 坦诚自身天资有限（止步玉京轮），全力培养弟弟和下一代 |
| **李木田** | 老祖父 | 短暂出场，状态尚可 |

### 3.2 关系棋盘

```
         李木田（老祖父）
         /        \
   李通崖（二哥）  李项平（三弟/代家主）
       |              |
   习剑者         操持家业
       \            /
      李尺泾（四弟/青穗峰弟子）
       /            \
  符术传承      剑法传承
```

### 3.3 关键对话
- **李项平→李尺泾**："我和你二哥能死，唯独你不能。"——奠定了李尺泾作为家族最高潜力者的地位，也是本章最核心的情感锚点。
