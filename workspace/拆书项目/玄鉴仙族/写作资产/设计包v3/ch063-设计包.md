## 一、骨架（Framework）

### 1.1 本章定位
- **卷轴位置**：家族内政线/外部局势线
- **时间跨度**：送走李尺泾当日→李通崖发现牲祭法→与李项平商议
- **空间跨度**：黎泾山前院→后院（鉴室）→眉尺山洞府

### 1.2 情节结构

| 序段 | 内容 | 字数占比 |
|------|------|----------|
| 序曲 | 李木田的身体衰败与田守水之死回忆 | ~15% |
| 主线A | 李通崖入后院，从法鉴中获得《牲祭法》 | ~20% |
| 主线B | 李项平分析三家格局（万/汲/李） | ~25% |
| 主线C | 兄弟商议：操练族兵以备战事 | ~25% |
| 收尾 | 柳柔绚报喜——怀孕 | ~15% |

## 二、爽点（Satisfaction Points）

| 编号 | 类型 | 爽点描述 | 触发时刻 |
|------|------|----------|----------|
| S1 | **系统升级** | 陆江仙主动修改裁剪法诀，从《祭萃夺元法》中提炼出《牲祭法》——鉴子功能更灵活 | 信息传递 |
| S2 | **成长爽** | 李项平分析三家格局头头是道——从武夫到战略家的成长 | 洞府独白 |

## 三、角色（Characters）

### 3.1 角色弧线

| 角色 | 本章状态 | 变化/发展 |
|------|----------|-----------|
| **李木田** | 衰老中的老祖父 | 身体机能明显下降（走路扶墙、腿无知觉、声音沙哑），但意志不减（"他娘的，老子也要拄上拐杖了"） |
| **李通崖** | 二哥/鉴者 | 再次获得法鉴赐法，成为家族与鉴子的主要接口；同意操练族兵 |
| **李项平** | 代家主/战略家 | 展示出成熟的战略分析能力（三家格局+山越威胁），推动族兵正规化 |
| **陆江仙** | 镜灵（后台） | 主动修改法诀，展现对鉴子功能的控制力越来越强 |
| **柳柔绚** | 李通崖之妻 | 报喜怀孕，为李通崖一脉延续 |
| **田守水** | 已故老人（回忆） | 塑造为对李玄锋有深情的老人，留下木玩具遗产 |

### 3.2 三代人对比

```
第一代（创业）：李木田、田守水、任平安
    → 田守水已死，李木田衰老，任平安病弱
    
第二代（守成）：李通崖、李项平
    → 正值壮年，掌握鉴子，战略决策
    
第三代（萌芽）：李玄锋（6岁+）、李玄宣（文）、李景恬、李玄岭（未出生）
    → 接棒在即
```

### 3.3 关键对话
- **李项平→自己**："三家如今各自蛰伏，汲家努力消化夺来的土地，万家则默默积蓄力量四处求援，我李家正缓缓增长到应有的体量……三家之间必有一战" —— 展示十三年的磨练让李项平从一个义愤少年成长为能分析地缘政治的家主。
