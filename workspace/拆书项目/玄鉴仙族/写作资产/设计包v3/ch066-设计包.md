# 第六十六章「急援」设计包 V3

---

## 一、剧情概要

### 表层剧情
李项平处理家族日常事务，安排李谢文去黎道口当差。万天仓紧急求援——万家连供奉都拿不出，已到背水一战的地步，请求李家按约定出手伏击汲登齐。李项平答应出手，但觉察到其中蹊跷：汲登齐故意施压逼迫后援现身，是阳谋。

### 深层剧情
- 李家与万家唇齿相依——若万家灭，李家孤悬山中，事事隔绝
- 汲登齐有恃无恐，必有底牌
- 万萧华也在用阳谋——明知危险也要拉李家下水
- 李项平决定：不求击杀汲登齐，只替万家解围，探明对方底牌

### 信息增量
- 李家内部权力结构：李项平当家主、李玄宣为少家主继承人
- 万家形势恶化至日常供奉都交不出
- 伏击约定条款：李家出手攻击汲登齐，万家交付阵道传承
- 通往郡中的道路仅此一条——地缘战略位置

- 万天仓：从冷静求援 → 激动失态 → 后知后觉发现阳谋 → 恐惧

## 三、世界观/设定

### 修为体系
- **胎息境**：四层青元轮（李项平）、三层周行轮（李秋阳三年未突破）、巅峰灵初轮（万萧华）
- **丹药**：明神散（值20枚灵石，极难求得）
- **灵窍检测**：近十年唯一查出身具灵窍者是陈东河

### 地缘势力
- **李家**：眉尺山/黎泾山，靠古黎道通郡中
- **万家**：华芊山，与李家相邻
- **汲家**：骅中山，与万家敌对
- **地缘死结**：通往郡中仅一条道路，过万家地界

### 阵道传承
- 万家先祖万华芊留下的大阵（地锁华芊阵）
- 约定条件：若李家出手，万家交付阵道传承

### 设定新增
- 阳谋结构：汲登齐故意长期施压，逼迫万家搬救兵，准备一网打尽
- 家族生存逻辑：即使明知陷阱也不得不跳

## 四、技巧/结构

### 叙事结构
```
万家求援（危机输入）
    ↓
李项平权衡（决策节点）
    ↓
觉察不对劲（悬念升起）
    ↓
双阳谋揭示（信息反转）
    ↓
必须出手（困境锁定）
```

### 技法分析
1. **信息不对称**：读者与李项平同步获知万家困境，共同推理
2. **双层阳谋**：汲登齐的阳谋（逼后援）→ 万萧华的阳谋（拉李家下水）→ 李项平两头看透
3. **铺垫与预兆**：「箭在弦上不得不发」预兆下一章的重大危机
4. **日常→危机的节奏**：开篇处理日常事务（李谢文安排）→ 突然接入万家求援 → 急转直下到生死决策
5. **用细节暗示格局**：一枚明神散20灵石买不到、族中近十年只有一个有灵窍的孩子——李家的窘迫不言自明

### 场景链

| 场景 | 地点 | 功能 |
|------|------|------|
| 眉尺山洞府 | 李项平读书 | 建立日常节奏 |
| 李叶生汇报 | 洞府中 | 引出万家问题、安排李谢文 |
| 万天仓求援 | 洞府中 | 核心冲突输入 |
| 双阳谋对话 | 洞府中 | 信息反转 |
| 李玄宣入府分析 | 洞府中 | 决策锁定 |
