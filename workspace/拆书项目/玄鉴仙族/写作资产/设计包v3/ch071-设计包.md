### 四段式结构

- **起**：双线并行——李项平逃亡（紧张线）+ 汲登齐杀万萧华（残酷线）+ 李通崖拜访萧初庭（信息线）
- **承**：汲登齐揭露"猪狗论"——世家与散修的真实处境
- **转**：萧初庭解释"养蛊"体系——三宗七门为何鼓励家族厮杀
- **合**：惊天揭秘——青迟魔门（青池宗前身）

### 章节功能定位

```
追杀危机（紧张开篇）→ 魔门真相（世界观颠覆）→ 双线交汇铺垫（为后续章节蓄势）
```

### 章节节奏

```
追杀(快/紧张) → 万萧华之死(慢/残酷) → 汲登齐独白(中/压抑) → 
李通崖拜访(慢/稳重) → 萧初庭世界观输出(中/震撼) → 青迟魔门揭露(高潮/冲击)
```

### 情绪曲线

```
追杀紧张 → 万萧华之死[残酷] → 汲登齐独白[寒意] →
```

### 本章双重揭露

1. **汲登齐线**：魔宗手段——祭杀凡人收集怨气血气（具体操作层面）
2. **萧初庭线**：体制真相——三宗七门的养蛊逻辑（制度层面）

两条线相互印证，揭露同一个黑暗真相。

## 三、角色层（人物弧光）

### 核心角色：萧初庭（本章智识输出）

| 维度 | 内容 | 原文证据 |
|------|------|---------|
| 外貌 | 老者形象，眯眼饮茶 | "眯着眼睛饮着茶" |
| 修为 | 筑基修士 | "萧家老祖筑基修士萧初庭" |
| 行事风格 | 洞察世故，直言不讳 | 直接揭露青池宗本质 |
| 与李家关系 | 因李尺泾天赋而看重 | "同这李家多打点好关系" |
| 核心台词 | "青迟魔门"——修仙界体制批判者 | |
| 对萧雍灵 | 传授真实世界观的教育者 | "你当青池峰上的月明琉璃树是怎么种出来的" |

### 关键对话分析：萧初庭独白

> "你以为我们是什么，我们不过是这两家养的猪狗，是他们交易中投上天平的一小颗砝码"

1. **表面**：骂世家散修都是猪狗
2. **潜台词**：修仙界的底层逻辑是残酷的资源掠夺
3. **用意**：打破萧雍灵的天真幻想，交代世界观底色

### 汲登齐的角色弧

| 阶段 | 状态 |
|------|------|
| 登场 | 斩杀万萧华的冷酷刽子手 |

| 本质 | 被系统异化的复仇者——"猪牛聪明了，所以要交换着宰杀" |

### 本章世态温度计

> **本包脉络**：世界观信息密度最高的章节之一。通过汲登齐和萧初庭两条线，从"实际操作层面"和"制度分析层面"双重揭露修仙界的黑暗本质。萧初庭的"养蛊理论"和"青迟魔门"的旧名揭示，彻底颠覆了此前"三宗七门正道"的表象认知。本章是《玄鉴仙族》世界观输出的关键节点——从此读者知道：这不是一个简单的修仙故事，而是关于系统压迫与反抗的叙事。
