### 四段式结构

- **起**：萧初庭谈话的后续思考 + 故人重逢（萧雍灵）
- **承**：闲聊修炼进度，李通崖交代家族窘境
- **转**：古法秘法知识输出（世界观补充）
- **合**：回到内心——担忧李尺泾在青池宗的处境

### 章节功能定位

```
前章信息消化 → 知识科普（古法秘法） → 经济困窘展现 → 内心担忧收尾
```

### 章节节奏

```
思考回顾(慢) → 故人重逢(中/欣喜) → 万家消息(低/哀伤) →
闲聊修炼(中/平稳) → 经济困境(慢/压抑) →
```

## 三、角色层（人物弧光）

### 核心角色：萧雍灵（本章知识输出者）

| 维度 | 内容 | 原文证据 |
|------|------|---------|
| 外貌 | 锦衣少年 | "屏风后走出个锦衣少年来" |
| 修为 | 世家子弟，天赋不错 | 与李通崖同龄，练气在望 |
| 性格 | 热心、好为人师 | 主动追上来讲解知识 |
| 社会阶层 | 萧家嫡系——世家子弟 | — |
| 与李通崖关系 | 平等交往的朋友 | "通崖兄""雍灵兄"互称 |

### 古法秘法体系知识输出

| 类型 | 定义 | 问题 |
|------|------|------|
| **正法** | 有配套天地灵气，可正常修炼的二品以上功法 | 价格昂贵 |
| **古法** | 原配套天地灵气消失/找不到，法诀成为无用之物 | 无法修炼 |
| **秘法** | 后人改气替代修成的功法（从古法中改造而来） | 弱于原版，有缺陷（修炼慢/须丹药吊命） |

### 关键对话分析：李通崖回应

> "谈不上家族所累，反倒是家族所成。"
> "管他什么古法秘法正法，我囊中空空，却也什么也买不起。"

1. **格局展示**：不抱怨家族拖累，反而感恩
2. **自嘲化解**：用幽默化解经济困境的尴尬
3. **人物塑造效果**：责任感强、豁达、务实

### 李通崖的角色弧

| 阶段 | 状态 |
|------|------|
| 入场：忧心局势 | 为李项平+局势担忧 |
| 见萧雍灵 | 获得友谊+信息 |
| 聊修炼 | 暴露经济困境（不卑不亢） |
| 学习古法秘法 | 认知提升 |
| 离场 | 转为担忧李尺泾——延续家族命运 |
