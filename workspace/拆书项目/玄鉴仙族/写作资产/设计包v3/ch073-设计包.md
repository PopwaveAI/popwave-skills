# ch073 反杀 — v3 设计包

## 一、骨架层（叙事结构）

### 事件链

| # | 场景(Scene) | 章节定位 | POV | 事件概要 | 原文证据 |
|---|------------|---------|-----|---------|---------|
| 1 | 汲家境内/果园 | 起（逃亡延续） | 李项平 | 李项平逃入汲家境内，符箓用尽，老道缀在后面 | "算算脚程，却已经深入汲家境内" |
| 2 | 果园/金桔树篱 | 承A（诱敌） | 李项平 | 李项平转身举起法鉴，老道畏惧不敢上前；李项平拍金刚符，拼死一搏 | "老道差点吐出一口老血，硬生生扭转飞梭" |
| 3 | 金桔树下 | 承B（近身肉搏） | 李项平/老道 | 老道金光术打到李项平身上，法鉴脱手而出 | "李项平顿时五腑俱震，喷出一口鲜血" |
| 4 | 金桔树下 | 转（法鉴觉醒） | 陆江仙 | 老道用拂尘去捉法鉴，触发鉴子自我防御——十二道篆符齐亮，白光滔天 | "青灰色的鉴子仿佛受了什么羞辱一般赫然漂浮而起" |
| 5 | 金桔树下 | 合A（太阴玄光） | 陆江仙 | 陆江仙发现太阴玄光可震慑练气期动弹不得，一道玄光将老道化尽 | "纯白色的太阴玄光倾泄而下，老道只来得及从咽喉中发出半个音节" |
| 6 | 金桔树下 | 合B（农户偷袭） | 李项平 | 农户少年见财起意，用石头砸李项平想"捡仙修漏" | "他娘的！好大的胆子，仙修的漏也敢捡？" |
| 7 | 树旁 | 结局（反杀） | 李项平 | 李项平用树枝附金光术捅穿少年脖颈，捡起鉴子和锦囊逃离 | "捂着喉咙倒地挣扎起来" |

### 四段式结构

- **起**：逃亡继续——李项平深入汲家，符箓用尽
- **承**：诱敌+近身——李项平与老道拼死一搏（B级搏杀）
- **转**：法鉴觉醒——老道犯了法鉴"忌讳"，触发自我防御，太阴玄光震慑练气
- **合**：反杀+意外——老道化尽，少女偷袭反杀，捡走战利品

### 章节功能定位

```
追杀紧张持续 → 博命诱敌（蓄势） → 法鉴觉醒（高潮） → 反杀+意外（收尾）
```

### 章节节奏

```
逃亡疲惫(慢) → 查看老道位置(中/紧张) → 
举鉴诱敌(快/惊险) → 被击中吐血(低/绝望) → 
法鉴脱手(极低/危机) → 拂尘触鉴(静默/蓄势) → 
十二道篆符齐亮(极高/高潮) → 
太阴玄光化尽老道(顶级/爽) → 
农户偷袭(意外/惊吓) → 反杀少年(解气) → 
捡物逃离(余韵)
```

## 二、爽点层（情绪驱动）

### 爽点清单

| 爽点类型 | 具体内容 | 强度 | 触发位置 |
|---------|---------|------|---------|
| **法鉴觉醒·太阴玄光** | 法鉴自发觉醒，十二道篆符齐亮，震慑练气 | ★★★★★ | 第55-71段 |
| **化尽老道** | 太阴玄光倾泄而下，老道如雪化尽 | ★★★★★ | 第87段 |
| **打脸老道** | "不该用拂尘去捉那鉴子的"——后悔莫及 | ★★★★★ | 第63-69段 |
| **农户反杀** | "仙修的漏也敢捡？"——李项平反杀偷袭者 | ★★★★ | 第101-105段 |
| **实力新认知** | 陆江仙意识到自己可以震慑练气 | ★★★★ | 第79-81段 |

### 情绪曲线

```
逃亡[紧张] → 示弱诱敌[蓄势] → 被击中重伤[低谷] →
法鉴脱手[危机] → 拂尘触鉴[悬疑] → 
十二道篆符齐亮[震撼] → 
太阴玄光化尽老道[顶级爽] → 
农户偷袭[惊吓] → 反杀[解气] → 
逃离[余韵/轻松]
```

### 本章情感公式

```
绝望蓄势(重伤+法鉴脱手) + 禁忌触发(拂尘捉鉴) = 顶级爆发(太阴玄光)
```

关键是"法鉴被拂尘捉取"触发禁忌——这是之前铺垫过的设定（ch070狼妖夜李项平父亲和二哥的描述），此处回收，读者能理解逻辑。

## 三、角色层（人物弧光）

### 核心角色：陆江仙（本章第一次正经战力的展示）

| 维度 | 内容 | 原文证据 |
|------|------|---------|
| 能力认知 | 太阴玄光可震慑练气动弹不得 | "原来凝聚起太阴玄光，只要距离够近，甚至可以震慑住练气" |
| 性格特征 | 冷静观察，等待时机 | "陆江仙看着动弹不得的老道" |
| 感知升级 | 神识扫到农户少年靠近 | "神识扫动之下，陆江仙发现这家农户已经有人醒来" |
| 法鉴特性 | 有灵性，触碰有"忌讳" | "如同有个不自量力的小孩拿起了桌面上的筷子，直直地指着自己的眼睛" |

### 老道的角色分析

| 维度 | 内容 |
|------|------|
| 修为 | 练气期 |
| 性格 | 贪欲主导，执着但不沉稳 |
| 致命错误 | 用拂尘捉取法鉴（犯了法器忌讳） |
| 结局 | 被太阴玄光化尽，只留下锦囊和金属块 |
| 叙事功能 | 1.验证法鉴的威力 2.为陆江仙战力认知升级 3.提供战利品 |

### 李项平的角色弧

| 阶段 | 状态 |
|------|------|
| 逃亡 | 疲惫/绝望 |
| 诱敌 | 冷静/决绝 |
| 被击中 | 重伤 |
| 法鉴觉醒 | 解脱/如释重负 |
| 反杀农户 | 凶狠/果断 |
| 离场 | 踉跄/警惕 |

### 农户少年的设定特征

| 特征 | 分析 |
|------|------|
| 身份 | 汲家治下的农户少年，穿破麻衣 |
| 动机 | 见李项平重伤，想捡仙修的漏 |
| 动作细节 | 用石头砸——底层人的简陋武器 |

| 叙事功能 | 展示修真世界底层凡人为了修仙资源可以多疯狂；也展示李项平在绝境下的果断狠辣 |

### 听觉

- "老道差点吐出一口老血"（夸张）——气急败坏
- "喉咙中发出半个音节"——来不及惨叫
- "他娘的！好大的胆子"——农户少年的粗口
