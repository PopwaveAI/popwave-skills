### 四段式结构

- **起**：训诫后续（李叶生的小九九）+ 击鼓调兵（为应对做准备）
- **承A**：李项平归来（月余逃亡，各种惨状）
- **承B**：兄弟重逢 + 倒出战利品（奖励高潮前奏）
- **转**：法鉴吞噬玉佩（陆江仙升级）
- **合**：战利品清点 + 用途讨论（收尾落定）

### 章节功能定位

```
悬念回收（项平归来） + 奖励时刻（战利品） + 法鉴升级（世界观推进）
```

### 章节节奏

```
训诫收尾(快/余韵) → 击鼓调兵(中/决断) → 
项平逃亡(慢/憔悴) → 谨慎绕行(中/警惕) → 
兄弟重逢(快/温情) → 
倒出战利品(中/期待) → 
法鉴噬玉佩(慢/震撼神圣) → 
清点讨论(慢/满足)
```

### 情绪曲线

```
李叶生被点破[紧张] → 击鼓调兵[决断] → 
项平逃亡中[压抑/孤单] → 谨慎回乡[警惕] → 
兄弟重逢[温暖] → 项平吐槽[轻松] → 
倒出战利品[期待/兴奋] → 
法鉴噬玉佩[神圣/震撼] → 
丹药讨论[满足] → 符箓分配规划[实用/安心]
```

### 本章结构特征

**奖励章**：ch073的反杀事件，收益在ch075正式清点兑现。是ch071-ch075五章序列的收束章——把所有悬念一口气回收：
1. 李项平安危→归来（安全回收）
2. 老道的战利品→清点（经济回收）
3. 玉佩→被法鉴吞噬（神秘回收）
4. 陆江仙→实力增长（成长回收）
