### 四段式结构

- **起**：师兄弟切磋收功 → 镗金门消息（外部压力）
- **承A**：南下御敌任务（行动线触发）
- **承B**：为家族求法（核心冲突：功法珍贵如何获取）
- **转**：以遂元丹换功法的交易谈判 + 选定功法（解决冲突）
- **合**：家中对话——家族策略与后辈培养计划（双线并进，内外呼应）

### 章节功能定位

```
双线叙事——宗门线（求法交易）+ 家族线（战略谋划）
对外拓展资源渠道 + 对内布局后辈培养
```

### 章节节奏

```
切磋收功(快/流畅) → 忧心家事(慢/忧虑) → 
南下任务信息(中/决断) → 提出求法(中/期待) → 
交易谈判(快/你来我往) → 功法选定(慢/满足) → 
家中对话(慢/沉稳) → 后辈计划(中/远见)
```

### 情绪曲线

```
切磋收功[平静/满足] → 忧心家事[忧虑] → 
南下消息[决断] → 求法提议[期待] → 
讨价还价[紧张/有趣] → 功法选定[圆满] → 
家中谈话[沉稳/安心] → 后辈计划[远见/希望]
```

### 本章结构特征

**资源拓展章**：本章核心功能是为家族获取功法传承——通过李尺泾在青池宗的人脉关系，以遂元丹为筹码换取三品《江河一气诀》和二品《清灵元法》。这是李家从"靠战利品和坊市零散交易"到"通过宗门关系系统获取功法"的质变节点。家族线的李项平与李通崖对话则呼应ch074-075的战略规划，形成"对外拓展+对内巩固"的双线结构。
