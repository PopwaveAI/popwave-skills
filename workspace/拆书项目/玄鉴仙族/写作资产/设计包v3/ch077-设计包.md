### 四段式结构

- **起**：难民消息 → 收容决定（接纳万家人，展现家族气度）
- **承A**：万天景视角——逃亡的心路历程（悲哀/仇恨/绝望）
- **承B**：面见李项平（改名明志，"万天仇"诞生）
- **转**：家族实力盘点（"已与当年万家相差无几"——成长确认）+ 西部隐患（山越威胁）
- **合**：法鉴升级（箓丹能力——提升家族修炼上限）

### 章节功能定位

```
收容线（万家遗孤融入）+ 家族实力盘点（成就展示）+ 法鉴觉醒（新能力预告）
壮大与预警并存——家族在变强的同时也面临新的威胁
```

### 章节节奏

```
难民消息(中/意外) → 收容决定(快/果断) → 
万天景逃亡(慢/悲哀) → 入山见李项平(中/紧张) → 
改名万天仇(快/决绝) → 
二李评价万家(慢/分析) → 
家族实力盘点(中/满足) → 
西部隐患(慢/隐隐不安) → 
法鉴箓丹(慢/神圣/希望)
```

## 二、爽点层（情绪驱动）

### 爽点清单

| 爽点类型 | 具体内容 | 强度 | 触发位置 |
|---------|---------|------|---------|
| **李家气度** | 接纳万家难民，不打散不奴役，编入各村分地——仁厚 | ★★★★★ | 第17-18段 |
| **改名明志** | "万天仇！我叫万天仇！"——名字从"景"变"仇"，仇恨驱动 | ★★★★★ | 第61段 |
| **家族实力飞跃** | "不算上泾儿，我李家的实力已经同当年的万家相差无几"——从附庸到一方势力 | ★★★★★ | 第82-83段 |
| **李秋阳的小心思** | 见万天景十四五岁才玄景轮，心中暗喜——普通人视角的家族自豪 | ★★★★ | 第36-37段 |
| **法鉴新能力** | 箓丹——可以助破关精进修为，对筑基紫府亦有奇效 | ★★★★★ | 第105-106段 |
| **二李精准点评** | "万家也就万萧华和万天仓值得忌惮"——战略眼光 | ★★★★ | 第67-69段 |
| **李项平的气场** | "那双宛若鹰隼的眸子直直撞上万天景的目光"——家主威仪 | ★★★★ | 第45段 |

### 情绪曲线

```
难民到来[意外] → 安置决定[仁厚/果断] → 
万天景逃亡记忆[悲哀/绝望] → 
入李家[警惕/陌生] → 见李项平[被震慑] → 
改名万天仇[决绝/仇恨] → 
李家盘点[满足/自豪] → 
西部隐患[隐隐不安] → 
法鉴箓丹[神圣/希望]
```

### 本章结构特征

**壮大与预警章**：本章完成两个功能——（1）展示李家的壮大（收容难民、实力盘点、法鉴升级）；（2）铺垫新的威胁（西部山越）。"名字叙事"是本章暗线——万天景改名万天仇（仇恨驱动），与李秋阳的名字（秋阳=温和/秋日阳光）形成对照。

法鉴的箓丹能力是本章的隐藏高潮——这是李家修炼体系的重大升级，从"靠天赋自己突破"到"有丹药辅助破关"，且箓印的设计保留了法鉴一贯的"物尽其用"风格（人死后箓气回归法鉴）。

## 三、角色层（人物弧光）

### 核心角色：万天景→万天仇（从难民到棋子）

| 维度 | 内容 | 原文证据 |
|------|------|---------|
| 出场形象 | 十四五岁，带着一群孩子，心如死灰 | "掩着泪磕磕绊绊地向西走" |
| 原名寓意 | 万天景——"景"为光明，但万家已毁 | — |
| 改名心理 | 万天仇——"仇"为仇恨，以复仇为动力 | "万天仇！我叫万天仇！" |
| 修为 | 胎息一层玄景轮（最底层修士） | "玄景轮的修为" |

| 维度 | 内容 | 原文证据 |

### 李秋阳（庶出修士的视角）

| 维度 | 内容 | 原文证据 |

### 陆江仙（法鉴新能力觉醒）

| 维度 | 内容 | 原文证据 |
|------|------|---------|
| 状态 | 神识游走而出，见熟悉的石台和天窗 | — |
| 新能力 | 凝聚箓丹——结合箓气和太阴月华 | "鉴身可以采用《祭萃夺元法》中所得箓气结合太阴月华凝聚出一枚箓丹" |
| 箓丹效果 | 可破关障精进修为，大境界仅可服食一枚 | "修仙者一个大境界仅可服食一枚" |

### 角色关系升级

- **万天景→李家**：从"亡国遗孤"→"李家的新成员（以仇恨为驱动）"
- **李家×万家旧部**：收容意味着势力扩张，也意味着承担更多责任
- **陆江仙×法鉴**：法鉴的"物尽其用"风格被再次强调——箓印机制延续了"谁用谁被标记"的暗黑设定

> **本包脉络**：本章是"收容与预警"章。万家覆灭后的难民涌入，既展示了李家的仁厚气度，也为家族增加了人力（万天仇的阵盘知识、劳动力）。家族实力盘点是本章的成就展示时刻——李家经过数年的发展，已经从一个村寨小族成长到与当年万家相差无几的规模。但西边山越的阴影开始浮现——与李项平"汲家一二十年腾不出手"的判断形成对照，真正的威胁不是从东边来，而是从西边来。法鉴觉醒的箓丹能力是本作的修炼体系升级——标志着李家从"纯靠天赋突破"进入"丹药辅助破关"的新阶段。
