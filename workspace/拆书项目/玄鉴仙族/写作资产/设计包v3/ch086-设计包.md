## 四、主题／意象层

### 核心意象
| 意象 | 文本出处 | 象征意义 |
|------|----------|----------|
| 月光下的山林 | 结尾独对月光 | 前路未明，暗流涌动 |
| 剪灯芯 | 李景恬反复剪灯芯 | 暗中维持微弱光明，女性在家族中的守护角色 |
| 六枚符种 | 存世的遗产分配 | 上一代遗留的有限资源，新一代各自命运 |

- **信息不对称**：李家对青池宗与山越的交易一无所知，"山越小卒都知道"

## 六、本卷写作技法

| 技法 | 应用 |
|------|------|
| **信息分层释放** | 先给"第六波"→再给伽泥奚全貌→最后给青池宗默许 |
| **少女谏言** | 用小孩口说道理，反差感强化冲击力 |
| **拒绝收买人心** | 项平拒绝后李景恬"绽放笑容"——双重测试设计 |
| **感官闭环** | 从"剪灯芯"的视觉 + "月光"的氛围 → 灵识感知的修行者视角 |
