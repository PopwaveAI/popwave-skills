### 修炼体系细节
| 条目 | 说明 |

| 青烟照月诀 | 练就太阴月华的功法，传承已断 |
| 人丹 | 以修士为药材炼制丹药，喂养妖王突破用 |

### 势力格局
| 势力 | 关系 | 备注 |
|------|------|------|
| 青池宗 | 幕后操盘手 | 以弟子为祭品与南疆妖王交换灵物 |
| 南疆妖王 | 交易方 | 吞食太阴月华修者为突破材料 |
| 邓家 | 青池宗附属家族 | 体质特殊，代代修炼秘法为补品 |
| 三宗七门 | 共同驻防倚山城 | 多方势力犬牙交错 |

### 关键物品／功法
- **太阴月华**：高阶灵物，可辅助突破，传承已断
- **邓家秘法**：代代修炼后用作补品；早期不全→发疯→近代改进但仍需装疯
- **各色灵药灵丹**：表面是宗门补给，实为练人丹的"用药深远的配子"

### 人丹背景设定
| 元素 | 内容 |
|------|------|
| 目的 | 南疆妖王突破所需 |
| 主要材料 | 太阴月华修炼者（李尺泾）/体质特殊者（邓求之） |
| 操作手法 | 送弟子到倚山城轮防→兽潮配合喂养→耗尽搜刮→收入妖王口中 |

## 六、本卷写作技法

| 技法 | 应用 |
|------|------|
| **信息差控制** | 邓求之先试探→李尺泾沉默→李尺泾反问"你又是为何"→邓被迫亮底牌 |
| **兽潮为叙事节拍器** | 每次关键对话精准卡在兽潮退去后的空隙，节奏天然紧凑 |
| **装疯为叙事诡计** | 前文邓求之"疯癫"印象被本话彻底颠覆，人物深度瞬间拉满 |
| **历史镜像** | 于羽楔=二十五年前的"李尺泾"——揭示循环悲剧结构 |
| **反话正说** | "平地晋级紫府"——用最荒诞的表达反衬最真实的绝境 |
| **泡茶仪式** | 在揭露人丹真相的同时保持动作：泡茶、倒茶——日常动作放大荒诞感 |
