## 二、设定层

### 修炼体系细节
| 条目 | 说明 |
|------|------|
| 入定困难 | 修士心中不安则无法入定——灵识预警机制 |
| 灵识 | 胎息五层玉京轮——但仅限周围感知 |
| 神识 | 陆江仙的法鉴神识——远超灵识，可覆盖整个黎泾镇 |
| 幻术入梦 | 陆江仙以幻术接管李玄岭梦境 |
| 太阴玄光 | 法鉴攻击咒语："李家弟子某某仰荷玄泽，恭请玄明妙法，谨出太阴玄光，诛斩恶逆，破秽摄妖" |

### 空间格局
```
  梨川口（伽泥奚占）
      │
      │ 河（浮桥被毁）
      │
  对岸军帐（李项平，1000兵）
      │
      │ 西进 →
      ▼
  西边密林（撤退路线）
      
      vs
      
  伽泥奚另一路 →
      ▼
  黎泾村 → 黎泾山（陆江仙/李玄岭/景恬）
```

### 关键物品
- **储物袋（老道的）**：一院子大小，可装大量米粮→远征补给
- **青乌弓**：李项平标志性武器
- **法鉴（玄明妙法鉴）**：供奉在后院石台，流转淡白色月华，温润手感

### 陆江仙操作机制
| 步骤 | 说明 |
|------|------|
| ①感知 | 练气修为入境→神识自动扫描 |
| ②搜索 | 找到李家各人位置（李通崖/柳柔绚→江边；李玄宣/李玄锋→黎道口；田芸/任氏→山火炉旁；李景恬/李玄岭→山上）|
| ③局限 | 神识只能在黎泾镇范围内游走 |
| ④干预 | 选李玄岭（胎息二层）→幻术入梦 |
| ⑤权限开放 | 李玄岭请咒后得到高空视角 |

| 火盆倒映 | 李项平眼中倒映红色火光 | 燃烧的家族命运 |

- **神鉴觉醒**：正式主动干预剧情

## 六、本卷写作技法

| 技法 | 应用 |
|------|------|
| **平行剪辑** | A线李项平决策西进 / B线伽泥奚逼近黎泾山 / C线陆江仙神识扫描 → 三条线同期发生，交叉呈现 |
| **神鉴视角** | 第一次用神识扫描全景展示李家所有人物的生活切片——家族群像的宏观快照 |
| **托梦机制** | 用小儿梦境传递急报——既保留神秘感，又符合低修为角色的接收限制 |
| **道具接力** | 法鉴从供奉物→被李玄岭取出→成为主动观测工具 → 为ch090战斗做前置 |
| **粮食细节** | "堆积如山的米粮和豆子"——写实的后勤关切，增加战场真实感 |
| **悬念钩子** | 结尾停在"山越兵直指黎泾镇"——章节停在高潮前一刻 |
