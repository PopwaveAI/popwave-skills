## 二、设定层

### 修炼体系细节
| 条目 | 说明 |
|------|------|
| 神行术 | 低阶辅助法术，李玄岭用此抱景恬下山 |
| 灵窍 | 修仙资质识别——李景恬遗憾"要是我也有灵窍" |
| 雾里迷阵 | 山上防御措施，对练气期伽泥奚"不过是雕虫小技" |

### 山越军战术
| 条目 | 说明 |
|------|------|
| 合围战术 | 一路正面进村+一路绕后山包抄防止逃往泾阳村 |
| 军纪特点 | "纪律严明"——抢劫不是散乱，而是系统性搜捕 |
| 掳掠目标 | 七到十五岁少年儿童 + "衣着华丽面容姣好者" |
| 时间窗口 | "时间不多了" ——青池宗窗口即将关闭 |

### 伽泥奚个人背景（本话披露）
| 条目 | 说明 |
|------|------|
| 颈间横纹 | 曾为奴隶——纹身是奴隶标记 |
| 遭遇 | 杀尽买下他的部族，胁迫族巫洗去横纹 |
| 性格后果 | 从此不在身体上涂抹任何花纹——对纹身/标记极度敏感 |
| 子嗣问题 | 木焦蛮劝其早做子嗣考虑——暗示统一北麓后要建立正统 |
| 对奴隶的敏感 | 木焦蛮谈"带回去充当奴隶"时伽泥奚眼中闪过暴虐杀意——但未发作 |

### 法鉴功能延伸
| 功能 | 说明 |
|------|------|
| 实时观察 | 李玄岭多次用来看伽泥奚动向 |
| 预警机制 | 发现山越从另一侧包抄——避免正面撞上 |
| 温润触感 | 揣在怀中"温温热热的触感"——灵性连接 |
| 战术价值 | 信息战层面的碾压——但李玄岭修为过低，只能被动观测 |

## 四、主题／意象层

### 核心意象
| 意象 | 出处 | 象征 |
|------|------|------|
| 换装裁衣 | 两人换素服、裁短衣 | 贵族之家的坠地伪装，文明退回到求生本能 |
| 空荡石台 | 供奉法鉴的石台已空 | 家族核心遗产逃亡中 |
| 有温度的被褥 | 伽泥奚手触有余温 | 死里逃生的惊险感——只差一步 |
| 颈间横纹 | 伽泥奚曾为奴隶的印记 | 强者也曾是最底层的被压迫者 |
| 七到十五岁的孩童 | 搜捕目标 | 一个族群未来被连根拔起的威胁 |
| 乌云 | 陈三水死时的乌云延续至此 | 黑暗时代的笼罩 |

### 主题词
- **逃与追**：全章是猫鼠游戏的节奏——法鉴查→改变路线→换装→躲藏
- **藏匿生存**：文明社会的秩序被野蛮闯入，退回最原始的生存斗争
- **天才的盲区**：伽泥奚算无遗策，但未预料到法鉴的信息外挂
- **奴隶主循环**：被压迫者的复仇往往不是解放，而是制造新的被压迫者

## 六、本卷写作技法

| 技法 | 应用 |
|------|------|
| **猫鼠游戏** | 整章是"用法鉴查动向→规避→再查→再躲"的循环，节奏紧迫 |
| **死角缺陷取胜** | 李玄岭下山→换装→躲藏→伽泥奚搜山→扑空→李玄岭赢在信息差 |
| **配角细节杀** | "面容姣好"四字看似随意描写→实际是伽泥奚逻辑推演的结果——筛出李家直系 |
| **暴君层次** | 伽泥奚听木焦蛮说"奴隶"时一闪而过的杀机——暴虐背后的伤疤，人物立体化 |
| **道具升级** | 法鉴从"供奉品"→"观测工具"→ch089→ch090成为生存外挂 |
| **结句钩子** | "去陪那家伙玩一玩"——指向未知第三方势力，为下一卷铺悬念 |
