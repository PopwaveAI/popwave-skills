### 时间／空间坐标
- **时间**：紧接ch090之后，伽泥奚西归的同一拨行动——奴隶队伍押送过程
- **空间**：野外猎场→黎泾镇旧院→大鱼溪→山越寨子（链式移动：从李家领地深入敌境）
- **核心冲突**：两条线索并行——李通崖等前方战情研判 vs 李玄岭/李景恬被俘入寨

### 双线叙事结构
```
A线（家族战场线）：猎场狩猎 → 急报 → 通崖侦查 → 青池玉印
                            ↓
B线（被俘线）：  旧院被捕 → 混入奴隶队伍 → 伪装前行 → 入寨/画纹
```
两条线在同时间轴并进，最后在"山越寨中"汇合——A线知道B线被俘但无法干预，形成读-角色信息不对称的张力。

## 四、主题／意象层

### 核心意象

| 意象 | 出处 | 象征 |
|------|------|------|
| 猪妖尸体上的箭孔 | 开头猎猪之战 | 李家尚存的力量（武德）与即将到来的落差 |
| 碗口大的血洞 | 李玄锋射杀猪妖 | 冷兵器的暴力美学——"武器好但皮子废了" |
| "权且忍让"四字 | 青池玉印回复 | 被抛弃的棋子的绝望感 |
| 颈间奴隶横纹 | 族巫画在李玄岭颈上 | 从"少家主族人"沦落为"畜牲奴隶"的符号化标记 |
| 木质枷锁 | 寨中使用的木制刑具 | 山越冶铁能力的匮乏——落后的暴力 |
| 大鱼溪 | 黎泾与山越的分界线 | 不可逆的越界——过了此溪便入敌境 |
| 火烧黎泾山/满点火把 | 山越攻占后的黎泾镇 | 家园沦陷的视觉化冲击 |

### 主题词
- **沦陷与潜伏**：核心叙事动力——家族领地沦陷、子弟被迫进入敌境潜伏
- **求援无门**：青池宗的冷漠——"权且忍让"四个字决定了李家只能靠自己
- **身份消解**：从李家少家主→奴隶——符号上的降级
- **信息不对称**：读者/李通崖知道法鉴存在，但山越不知道——而法鉴此刻也被压制
- **姐弟羁绊**：被分开的威胁——姐姐被单独带走对李玄岭形成情感压力

## 六、本卷写作技法

| 技法 | 应用 |
|------|------|
| **双线并行** | A线（家族战场：猎场→通崖侦查→青池回复）与B线（被俘线：旧院→押送→入寨）交替推进，时间同步，互相不知道对方的处境 |
| **从日常到危机的骤降** | 开头打猎夸箭法的轻松→"山越攻占黎泾镇"的急报——一瞬间切换氛围 |
| **政治孤立感的刻画** | "权且忍让"四个字高度浓缩——宗门的冷酷、家族的孤立，一笔带过但信息量大 |
| **伪装逻辑的合理性** | 李玄岭不是"主动装傻"，而是有清晰理由：①山上常下受众识 ②景恬深居简出风险小——each决策有依据 |
| **道具功能展示** | 法鉴在这一话被"压制"——三位练气在上，李玄岭不敢用——展示法器的局限性，避免成为万能解 |
| **符号化标记** | 奴隶横纹不仅是剧情道具——颈间画纹成为"身份降级"的视觉符号，呼应伽泥奚颈间曾被画纹的伏笔 |
| **结句钩子** | ①李景恬被木焦蛮审视结尾 ②"青池不援"的政治信号 ③李项平行踪不明——三重悬念收尾 |
