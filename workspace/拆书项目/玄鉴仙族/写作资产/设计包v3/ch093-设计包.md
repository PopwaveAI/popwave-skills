- **时间**：山越撤走后的第一天——李景恬李玄岭被俘的同一时期

- **核心冲突**：家内秩序重建（行政线）vs 情感失落（精神线）vs 实力焦虑（力量线）——三线交织

## 二、设定层

### 家族损失统计

| 条目 | 说明 |
|------|------|
| 人口损失 | 黎泾镇适龄少年被裹挟西进——但李家查过灵窍的子弟不在损失之列 |
| 伤亡 | 杀人不多—几百村丁受伤；梨川口死人多（陈家嫡长子阵亡） |
| 陈家损失 | "只要陈冬河不死，陈家死了谁都不要紧"——实用主义的冷酷判断 |
| 财务损失 | 灵田受劫掠——李通崖专门过问 |
| 法器遗失 | 法鉴被李玄岭带走（主动而非被抢）——不是损失而是转移 |

### 修炼境界补充

| 条目 | 说明 |
|------|------|
| 胎息五层玉京轮 | 李玄锋要突破的目标——从对话可知修炼有层级名称 |
| 胎息四层青元轮 | 李玄宣卡在这一层两年多 |
| 突破条件 | 李玄宣认为需"契机"，李玄锋认为"想突破就去闭关"——两种突破观 |
| 蛇元丹 | 三枚——两枚已用（长辈各一），最后一枚给李玄锋 |
| 蛇蛟果 | 保存三枚但无炼丹传承——无法利用，家族技术短板的体现 |
| 炼丹传承 | "找个散修杀了抢走" vs "积蓄灵石去买"——两种道路选择的争论 |

### 法鉴保管细节

| 条目 | 说明 |
|------|------|
| 平时位置 | 偏院小屋案台上供奉 |
| 失踪发现 | 李玄宣看到"倒塌的案台和空荡荡的石台" |
| 李通崖推理 | ①若被抓→山越会来勒索→无勒索证明未被抓 ②玄岭机灵→提前取走法鉴 ③玄岭应是与景恬一同被裹挟西进 |
| 法鉴价值 | 对李家是"核心遗产"——李玄宣见空台时"失落与愤怒一齐涌上心头" |

### 李通崖战力评估

| 条目 | 说明 |
|------|------|
| 练气二层 | 自身修为 |
| 玄水剑气 | 最大杀招——快而锐 |
| 同阶压制 | "没有三五个留不下我"——自信但适度 |
| 对比李尺泾 | "五个绑起来还不够他一剑的"——暗示李尺泾练气顶峰的战斗力 |
| 实战策略 | 遇到硬茬就撤——"惧对方有援手"（ch091） |

## 三、人物弧层

### 李玄宣（本章——情绪的起伏线）

| 维度 | 刻画 |
|------|------|
| 担忧状态 | "还是不曾找到景恬和玄岭？"——开场就是追问 |
| 务实决策 | 以"夺回孩子""向山越报仇"为名招募——实则只是重建管束各村 |
| 失控瞬间 | 推门见石台空——"该死"——失落到愤怒到无奈 |
| 情感触发 | 坐李木田旧椅→见"宜室宜家，尔炽尔昌"八字→情绪涌上心 |

| 管理风格 | 温和务实——"不会做傻事"（区别于李玄锋的冲动） |

### 李通崖（本章——家族定海神针）

| 维度 | 刻画 |
|------|------|
| 侦查结果 | 向西两个时辰被逼退——山越西边扎寨，四个练气 |
| 乐观推断 | "我若是找不到，那山越练气同样找不到"——逻辑推导式的安慰 |

| 战斗哲学 | 能打就打，打不过就跑——务实而非死战 |

### 李玄锋（本章——愤怒化动力）

| 维度 | 刻画 |
|------|------|

## 四、主题／意象层

### 核心意象

| 意象 | 出处 | 象征 |
|------|------|------|
| 空荡荡的石台 | 供奉法鉴的案台 | 家族核心遗产的转移——从"保险柜"进入了"危险世界" |
| 破木椅子 | 李木田生前年年坐着的椅子 | 三代人的精神传承——极简但厚重 |
| "宜室宜家，尔炽尔昌" | 椅腿上刻的八字 | 家族从生存到繁盛的终极愿景 |
| 蛇元丹的传递 | 最后一枚丹药给李玄锋 | 资源有限下的战略分配——培养武力优先 |
| 蛇蛟果的窘境 | 有三枚但无炼丹传承 | 家族技术短板的象征——有原料无工艺 |
| "找个散修杀了抢走" | 李玄锋的发言 | 丛林逻辑 vs 秩序逻辑的路线分歧 |
| 倒塌的案台 | 偏院小屋的狼藉场景 | 山越入侵留下的破坏痕迹——秩序被打破后的一切 |

### 主题词
- **从恐慌到安心**：全章的情绪弧——从"人失踪+法器失踪"到"人可能安全+法器也在手中"
- **秩序重建**：山越撤走不代表危机结束——家族需要从被动转向"安内"
- **两代价值观差异**：李玄锋的"抢" vs 李玄宣的"买"——家族崛起路线的内部张力
- **精神传承的力量**：李木田的八个字——在绝望时刻给予后辈力量和方向
- **技术短板的焦虑**：有资源但无技术——家族底蕴不足的自我认知

## 六、本卷写作技法

| 技法 | 应用 |
|------|------|
| **解谜式对话** | 李通崖从"山越无动静"反推"法鉴被玄岭取走"——一个逻辑推理段落化解了前半章的失落后怕 |
| **器物触发情感** | 破木椅子上的八字——物件的叙事功能：将李木田的精神从死者传递到生者 |
| **两代对话中的路线分歧** | 李玄锋"杀散修抢传承" vs 李玄宣"积蓄灵石去买"——轻描淡写但反映了价值观差异 |
| **资源管理的张力** | 蛇元丹只剩最后一枚、蛇蛟果三枚用不了——家族底蕴不足的焦虑感 |
| **侧面展示人物** | 李木田只通过"一句筑基谣传困了半生"和椅子上的八字刻画——极简但感人 |
| **"弱者决策"** | 李玄宣不是最高战力也不是最有天赋——但他是最顾家的管理者——在危机中做出了正确的行政决策 |
| **高潮锚点** | 全章的情绪高潮不在战斗，而在李玄宣看见"宜室宜家，尔炽尔昌"的那一刻——以静制动的情感爆发 |
