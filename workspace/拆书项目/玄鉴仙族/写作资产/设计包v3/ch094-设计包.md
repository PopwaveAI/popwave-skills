## 二、设定层

### 箓气体系重大披露

| 条目 | 说明 |
|------|------|
| 箓气等级 | 灰→白→青→黄（对应练气→筑基→紫府→金丹） |
| 灰箓 | 李项平身上那枚——对应练气期 |
| 黄箓 | 大厥庭方向出现——**金丹期**的祭品和香火凝结 |
| 祇封条件 | 需要活祭妖物+海量香火——不是随便可造 |
| 黄箓来源 | 法鉴受了金丹期的祭品+香火才能凝结——意味着大厥庭存在金丹级存在或曾与法鉴有过联系 |
| 陆江仙的震惊 | "怎么可能？"——法鉴是他自己的法器，不应该有其他箓气存在 |

### 世界观升级

| 条目 | 说明 |
|------|------|
| 李江群疑案 | 陆江仙怀疑黄箓与仙府传人李江群有关——但修为对不上（紫府不可能敕封黄箓） |
| 三宗七门 | 当年围杀李江群的主力——但金丹级的势力不在其中 |
| 更大势力推演 | "针对法鉴的绝不是区区三宗七门，背后应该有更大的势力"——**世界观扩展的提示性句子** |
| 陆江仙的谨慎 | "无论如何都不能轻易前往这大厥庭"——主角意识到自身风险的主动规避 |

### 大厥庭设定

| 条目 | 说明 |
|------|------|
| 位置 | 山越腹地大城市——"山越的大城" |
| 建设计划 | 建大石台祭祀诸巫，印证一统北麓的功业 |
| 李玄岭的恐惧 | "进了那山越的大城，才真是上天无路下地无门" |
| 威胁等级 | 疑似有金丹级箓气——远超李家当前应对能力 |

### 脱逃计划细节

| 条目 | 说明 |
|------|------|
| 时机 | 傍晚弦月正中之时 |
| 地点 | 寨外东边林中汇合 |
| 手段 | 李玄岭先出寨采伐木料→回身林中制造声响→李景恬假意如厕→杀侍从→带人走 |
| 工具 | 法鉴（已存太阴玄光可用） |
| 应变方案 | 若被抓回——李景恬可安全（木焦蛮不会伤她）；若木焦蛮亲追——李玄岭用法鉴太阴玄光 |
| 最坏方案 | "先杀你再自杀" |

## 四、主题／意象层

### 核心意象

| 意象 | 出处 | 象征 |
|------|------|------|
| "如同羊圈般"的住宿 | 寨中奴隶的关押地 | 非人化的待遇——文明人退回畜牲境 |
| 屎尿堆成山 | 奴隶住宿环境 | 极端的卑污——环境压垮意志的设计 |
| 明日启程回大厥庭 | 部巫对话 | 倒计时——逃脱窗口即将关闭 |
| 黄箓如"黑夜中的火炬" | 陆江仙的感受 | 金丹级存在的强烈信号——未知的巨大威胁 |
| 弦月正中 | 约定的时间信号 | 黑夜中的希望标志——微弱但确定 |
| "先杀你再自杀" | 李玄岭的承诺 | 最极端的家族忠诚——宁可同死不愿受辱 |
| 大厥庭的大石台 | 山越的祭祀工程 | 一个文明（山越）的自我正当化——祭祀即权力 |

### 主题词
- **隐忍的极限**：十几天装苦力等待——被动的忍耐终于走到行动的临界点
- **倒计时紧迫感**："明日启程"——整章从"还可以再等"到"不得不走"的节奏转化
- **世界观扩展**：从家族→山越→法鉴箓气→金丹级势力——视野逐层放大
- **极端家族忠诚**：宁可同死不愿受辱——修仙世界的道德底线（生不如死）
- **姐弟协作的默契**：十几天后的第一次接触——无需多言就能定下计划
- **主角的谨慎**：陆江仙主动选择不冒进——展示修行世界中的"苟"哲学

## 六、本卷写作技法

| 技法 | 应用 |
|------|------|
| **倒计时叙事** | "明日启程"——整章从悠哉侦察→紧迫行动，一个时间信息打破节奏 |
| **世界观扩展的插叙** | 在紧凑的逃脱线中间插入陆江仙的黄箓发现——文脉断裂但信息密度提升 |
| **法器功能展示+限制** | 法鉴可以定位、感应到远处的黄箓——但陆江仙选择不主动干预（隐蔽原则） |
| **双视角并行** | 李玄岭的寨中行动（细致/微观）vs 陆江仙的法鉴感测（宏大/抽象）——视角交替制造层次感 |
| **底线谈判** | "先杀你再自杀"——用极端条款制造戏剧张力，远比"我们一起走"有冲击力 |
| **侧面描写李项平** | 通过俘虏李景恬的消息展示李项平的行动——"集结山越流民"——暗示他开始从被动转为主动 |
| **情报作为"倒计时触发器"** | 部巫的对话不是闲聊——是作者精心放置的剧情推进器 |
| **"大城恐惧"** | "上天无路下地无门"——用城来制造心理压迫，为逃脱增加情感的紧迫系数 |
