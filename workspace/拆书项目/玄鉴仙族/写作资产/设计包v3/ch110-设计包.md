# 第一百一十章·矿脉 — 设计包 (v3)

## 一、骨架 (Framework)

### 1.1 章节定位
- **卷轴位置**：第一卷·家族草创期·中期（信息爆发·矿脉合作开端）
- **核心事件**：李通崖出关应对卢家 + 紫府攻青池山情报 + 矿脉合作
- **情感基调**：遮掩修为的紧张 → 被轻视的有趣 → 情报震惊 → 务实谈判

### 1.2 事件链表格

| # | 触发 | 事件 | 人物 | 结果/转折 |
|---|------|------|------|-----------|
| 1 | 时间推进 | 李通崖闭关一年有余，卢家第三次来访 | 李通崖、李玄岭 | 被迫出关——再躲恐遭疑 |
| 2 | 背景交代 | 李通崖解释为何出关：练气不能闭死关数年 | 叙述 | 修炼体系设定细化 |
| 3 | 情节过渡 | 李通崖问李玄岭：卢思嗣可曾说什么 | 李通崖、李玄岭 | 卢远陆的态度初现 |
| 4 | 内心独白 | 李通崖回顾与萧初庭的差距：筑基巅峰交好胎息 | 李通崖 | 格局/胸襟对比 |
| 5 | 萧初庭信息 | 据金羽宗张允所说，萧初庭已成紫府 | 李通崖(思考) | **重磅信息**：萧初庭紫府 + 暗中谋划 |
| 6 | 会面开始 | 李通崖入正院见卢思嗣、卢远陆 | 李通崖、卢思嗣、卢远陆 | 关键人物聚集 |
| 7 | 震惊 | 卢思嗣见李通崖练气五层，失声 | 卢思嗣 | 进度震惊：三年从三层到五层 |
| 8 | 圆谎 | 李通崖以"灵果突破"解释，卢思嗣接受 | 李通崖、卢思嗣 | 掩饰成功 |
| 9 | 新角登场 | 卢远陆登场——二十出头，态度傲慢 | 卢远陆 | 卢家新生代形象的展示 |
| 10 | 对比冲突 | 卢远陆炫耀正法真元 → 李通崖戳"卢远平是兄" | 卢远陆、李通崖 | 打脸——兄用杂气，弟用正法 |
| 11 | 信息断层 | 卢思嗣支走卢远陆，尴尬道歉 | 卢思嗣 | 家教缺失的窘迫 |
| 12 | 重磅情报 | 卢思嗣告知：紫府修士攻打青池山，天元灵阵被祭起 | 卢思嗣 → 李通崖 | **世界格局剧变信号** |
| 13 | 深度爆料 | 青池宗紫府"仙去" → 几年必有变动 | 卢思嗣 | **核心情报**：紫府死亡造成权力真空 |
| 14 | 矿脉提案 | 卢思嗣告知青乌矿发现，邀请三方合作 | 卢思嗣、李通崖 | 家族经济新篇章 |
| 15 | 行动确认 | 李通崖同意前去探查，之后再议分润 | 李通崖、卢思嗣 | 矿脉合作启动 |

### 1.3 结构节奏
```
出关(被动) → 对比萧初庭(格局) → 会面(遮掩) → 小冲突(打脸) → 情报(震惊) → 矿脉(务实)
   ↑紧张         ↑视野扩展           ↑机智        ↑爽          ↑震撼        ↑落地
```

## 五、套路归档 (Troop Archive)

### 套路1： 闭关出关
- **路径**：`套路库/闭关出关.md`
- **触发标签**：#修炼周期 #闭关设计 #进步展示
- **本回实例**：李通崖闭关一年有余，修为从练气三层→五层(实际因重海长鲸箓)，以"灵果"遮掩
- **重现策略**：修仙文的经典"显示器"——每次出关都是实力成长的外在证明；关键是要给出合理的进步幅度和遮掩理由

### 套路2： 纨绔打脸
- **路径**：`套路库/纨绔打脸.md`
- **触发标签**：#傲慢晚辈 #身份反转 #实力打脸
- **本回实例**：卢远陆炫耀正法真元 → 李通崖提起其兄卢远平用杂气，话语中蕴含对比，让对方羞窘
- **重现策略**：不需要直接动手，一个轻描淡写的信息差就足以打脸；温和的"以子之矛攻子之盾"比暴力碾压更高明

### 套路3： 世界格局信号（高层震荡）
- **路径**：`套路库/世界格局信号.md`
- **触发标签**：#紫府陨落 #势力洗牌 #高维事件
- **本回实例**：紫府修士攻青池山→天元灵阵被祭起→青池宗紫府"仙去"→几年必有变动
- **重现策略**：家族文需要定期插入高层震荡信号——即使不直接参与，也让读者感受到"风雨欲来"的紧迫感，为家族选择提供格局背景

### 套路4： 资源谈判（跨势力合作）
- **路径**：`套路库/资源谈判.md`
- **触发标签**：#矿产合作 #三方分润 #谈判开局
- **本回实例**：青乌矿横跨三家地界，卢思嗣心不甘情不愿地提议共同开发
- **重现策略**：天降资源必然伴随谈判；关键是把谈判过程写出真实博弈感——对方的不甘、己方的筹码、各方的底线
