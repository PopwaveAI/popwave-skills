## 三、深层：主题与象征

### 核心主题
1. **"世家名声"的虚幻性**：乌家女修最后的底牌是"我是乌家五世孙女"——但在荒郊野外、蒙面人的剑下，这名字一文不值。在实力面前，背景只是软弱的托词
2. **丹炉的象征意义**：一枚品相好的丹炉——既是战利品中最大的惊喜，也预示着李家未来的炼丹可能性（资源加工升级）
3. **重海长鲸箓的实战优势**：真元恢复速度惊人——打完一场追逐+战斗，调息片刻几近回满——持续作战能力的保证
4. **谨慎即生存法则**：蒙面、处理痕迹、遁入安全洞府——每一步都是李通崖"谨慎"标签的具象化

### 意象系统
- **灵布蒙面（灰白）**：身份的消隐——"做这件事的不是我"
- **青碧色法剑（乌家女修）**：武器即身份——好法器被抢走，其主已死
- **火雀符箓五道爆裂**：保命符的奋力一搏——但仍是徒劳
- **丹炉上的血迹（净衣术洗去）**：器物上附着的血腥历史——每一件法器背后都有死人
- **蛇妖洞府·淡蓝色水晶照明**：安身遁入世外的净土——战利品的清点仪式在安全的洞穴中进行

### 埋设伏笔
- 丹炉入手 → 李家未来可能发展炼丹能力
- 乌家女修之死 → 郡南乌家是否追查？
- 储物袋中的乌家令牌（青乌打造，刻"乌"字） → 若后续被发现，可作为线索
- 李通崖对自身战力的确认（练气六层正法修士不是对手） → 更高目标的底气
- 望月湖坊市的禁断法最终结果 → ch114陈涛平的生死未决

## 四、基底层：叙事逻辑与工具

### 叙事机制
- **"掠食者"的完整行动链**：侦查→锁定→拦截→威胁→战斗→击杀→善后→清点→休整——每一步都清晰呈现
- **信息差制造的道德灰色地带**：李通崖蒙面、杀人不留名——读者知道他是为了家族利益，但行为本身是"黑吃黑"
- **战力校准公式**：练气五层（李通崖）≈ 压着练气六层（乌女）打 + 硬抗练气级符箓 + 追得上 + 杀得掉——给读者一个清晰的速度/力量参照系
- **收获清单仪式**：战后清点战利品是"主角暴富"的**仪式化呈现**——逐件检查、估值、分类——让读者一同体验满足感

### 情绪曲线

```
情感强度
   ▲
   │  蒙面等待（紧绷/专注）
   │    ↓
   │  拦截成功（爆发）——→威胁被拒（张力上升）
   │                              ↓
   │          战斗（激烈/快节奏）←——符箓爆发（意外/危机）
   │                                        ↓
   │  追杀（紧迫）——→斩杀（残酷/利落）——→善后（冷静）
   │                                                 ↓
   │            清点战利品（放松/喜悦）——→丹炉惊喜（高潮满足）
   │                                                      ↓
   │    盘膝修炼·箓气恢复（安宁/满足收束）
   └──────────────────────────────────────────────────────► 时间
```

### 章节功能定位
- **获利章**：李通崖通过截杀获得丹炉——李家资产的重要增量
- **战力校准章**：实战验证李通崖"练气五层可杀练气六层正法修士"的战力水平
- **道德灰色章**：主角通过"黑吃黑"获利——修仙世界的丛林法则赤裸裸呈现
- **坊市劫难的尾声**：ch114-ch115是完整的"大事件+获益"两章连缀
