# 第一百二十章 入炉 — 设计包 v3

---

## 一、章旨

| 维度 | 内容 |
|------|------|
| **叙事功能** | 李尺泾线终结：剑意亮相→紫府真人对话→被大蛟投入丹炉炼化。悲剧完成 |
| **核心冲突** | 个体天才 vs 修仙体制——李尺泾用全部才华和生命无法抗衡的宿命 |
| **情感基调** | 悲壮到凄凉：从剑气冲天的激昂到炉中静待死亡的沉寂 |
| **信息密度** | 高（紫府修士心态揭露、青池宗内斗、大蛟炼丹规则） |

> 本章是全书第一个核心角色的陨落，也是李家"天赋者之殇"主题的第一次完整呈现。

| **写法要点** | 以第一人称内心独白收束，从宏大叙事回归到最朴素的亲情牵挂 |

## 三、人物弧光

### 李尺泾

| 维度 | 变化 |
|------|------|
| **出场状态** | 剑意全开的巅峰强者 |
| **与紫府周旋** | 不卑不亢，直面死亡 |
| **临终心态** | 从"紫府修士当真快意"的洒脱，到对家人的牵挂 |
| **最终形象** | 被投入丹炉前仍保持冷静与尊严——"口不能言，动弹不得"的身体束缚 vs "静静地想着"的精神自由 |
| **悲剧本质** | 不是死于战斗或意外，而是被当做人形药材——最彻底的物化与屈辱 |

### 青衣紫府（元素真人）

| 维度 | 内容 |
|------|------|
| **角色定位** | 执行者，有良知但无力改变 |
| **关键态度** | 同情李尺泾（"你还真对我脾性"），但仍然坚决执行 |
| **底层逻辑** | "紫府也有紫府的窘迫之处，只是大多数时候好歹不用充当棋子罢了" |
| **复杂性** | 不是纯粹的恶人，而是体制下的妥协者 |

### 大蛟

| 维度 | 内容 |
|------|------|
| **角色定位** | 非人存在，超脱宗门体制的力量 |
| **关键特征** | 少年外貌+竖瞳，对迟尉的不屑，将人类视为"药材" |
| **关键台词** | "你这人真美味，若不是答应了迟老鬼，我还真想吃了你" |

### 邓求之

| 维度 | 内容 |
|------|------|
| **角色定位** | 与李尺泾一同入炉的"药引" |
| **最终状态** | 被剖腹取物，惨死——与李尺泾形成对比（练气 vs 筑基） |
| **写法要点** | 以李尺泾视角写邓求之之死，增加悲剧冲击力 |

## 四、技法分析

| 技法 | 应用 | 效果 |
|------|------|------|
| **先扬后抑** | ch119的剑意璀璨 → ch120的炉中沉寂 | 极致的情绪反差 |
| **对话揭露** | 紫府修士坦言宗门内幕 | 在角色将死时释放最黑暗的信息 |
| **留白与想象** | 入炉过程不写痛苦，只写内心 | 让读者自行填补恐怖感 |
| **悲喜剧对位** | "紫府修士当真快意！"——笑对死亡 | 用苦中作乐强化悲剧感 |
| **收束技法** | 李尺泾临终想起家人/木田/父亲/兄长 | 从史诗叙事回归家庭温情 |
| **物化技法** | 人被当做药材、"药龄"概念 | 冷酷的制度性暴力 |
| **章末注脚** | "好多人羊了……"（作者附注） | 现实与虚构的偶然呼应 |
