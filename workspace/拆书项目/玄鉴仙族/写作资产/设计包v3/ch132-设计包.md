# 第一百三十二章 郁家拜访 — 设计包 v3

---

## 一、核心定位

| 维度 | 内容 |
|------|------|
| **章节类型** | 外交·世家交往·势力博弈 |
| **叙事节奏** | 中速（接待、对话、分析） |
| **核心冲突** | 郁家"以势压人"的扩张方式 vs 李家"以力为凭"的观望态度 |
| **情绪基调** | 表面客气→暗流涌动→理性分析 |
| **章旨** | "拜访"是世家外交的表面仪式，底层是实力的博弈与站队的试探——郁家客气是因为忌惮"剑仙"，李家退让是因为看穿对方走不远 |

## 二、四层结构

### Ⅰ 设定层（World）

| 设定项目 | 内容 |
|---------|------|
| **时间点** | 郁家拜访卢家之后，前往李家的当天 |
| **地理空间** | 华芊山（卢家地界）→ 黎泾李家（阵前迎接→院内落座） |
| **势力格局** | 郁家（两位筑基，子弟入元乌峰，意图做望月湖之首）、李家（有"青穗剑仙"之名震慑）、费家（北岸筑基家族，未出场但被提及） |
| **修行体系线索** | 李尺泾"三十出头筑基"→"紫府有望"的传言是李家最大的威慑资本 |

### Ⅱ 人物层（Character）

| 角色 | 本场功能 | 状态变化 | 关键台词/动作 |
|------|---------|---------|-------------|
| **郁慕高（郁公子）** | 郁家外交使节 | 傲慢（卢家）→ 谨慎恭敬（李家），切换自如 | "都给我放谨慎些！到了李家山头上轻易不要说话！" |
| **胖子（郁家随从）** | 郁家莽撞角色 | 从不服气→被训斥→闭嘴 | "不就是那青穗剑仙么！" |
| **高瘦男子（郁家随从）** | 观察者/智囊 | 仔细观察李家地界人文 | "这李家地界上的人倒是多，看来治理凡人花了不少心思" |
| **李玄锋** | 迎客者/守护者 | 从警惕→灵识确认安全→礼貌引路 | 身背漆黑长弓迎风而起 |
| **李通崖** | 决策者/李家之主 | 看出郁家来意→表面支持→暗藏不看好 | "诸家奉贵族为首，这好处与供奉贵族想必也吃足了吧！" / "你且看着，倒要看看他怎么收场！" |

### Ⅳ 技法层（Technique）

| 技法 | 应用 | 效果 |
|------|------|------|
| **对比叙事** | 郁家对卢家的傲慢 vs 对李家的恭敬 | 同一人物在不同实力对象前的两面性，揭示修仙世界的实力法则 |
| **对话潜台词** | 郁慕高"不过是诸家感谢郁家庇护"→ 李通崖"好处与供奉吃足了吧" | 表面客气下的权力博弈 |
| **智囊角色配置** | 三人组：公子（决策）+ 高瘦者（观察）+ 胖子（冲动） | 通过不同角色视角完整展现郁家对李家的评估 |
| **信息差** | 郁家以为李家有剑仙靠山（李尺泾）→ 读者知道李尺泾已死/被炼成丹 | 制造戏剧张力——郁家的忌惮建立在虚假信息上 |
| **预见性对话** | 李通崖"你且看着，倒要看看他怎么收场" | 为后续郁家行动埋下点评基调 |
| **空间层次感** | 飞在湖上→阵前通报→入阵→院中落座 | 用空间递进展现世家交往的礼仪层级 |
| **细节暗示** | 李通崖"泾儿的事至少还能瞒上几十年" | 点出李家当前威慑的脆弱性与时间压力 |
