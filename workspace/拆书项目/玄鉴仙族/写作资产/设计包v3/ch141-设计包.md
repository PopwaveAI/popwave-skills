# 第一百四十一章 求箓 — 设计包 v3

---

## 一、核心定位

| 维度 | 内容 |
|------|------|
| **章节类型** | 世界展示 · 仪式叙事 · 体系奠基 |
| **叙事节奏** | 中速（祭祀准备→仪式进行→箓丹凝聚→收尾） |
| **核心冲突** | 无直接冲突——以世界体系的展演为主，为后续的家族崛起注入神秘力量 |
| **情绪基调** | 庄严肃穆的祭祀氛围 → 神秘力量的显现 → 陆江仙的野心布局 |
| **章旨** | "求箓"不仅是祭祀仪式的高潮，更是李家获得特殊力量体系（箓气+箓丹）的起点——从此李家与普通修仙家族的差异开始显现 |

## 二、四层结构

### Ⅰ 设定层（World）

| 设定项目 | 内容 |
|---------|------|
| **时间点** | ch140之后，祭祀大典当天 |
| **地理空间** | 黎泾镇（四镇镇民聚集）→ 黎泾山青石圆台（祭祀场所）→ 镜中（陆江仙视角） |
| **势力格局** | 李家统治四镇已近四十年，四镇居民富足安定，祭祀成最高礼仪 |
| **新设定** | **灰箓体系**：香火凝聚的箓气可赋予受箓者特殊能力（力贯千钧、追云逐月、延年长命）；**箓丹体系**：多余箓气+太阴月华可凝聚箓丹，每个大境界服食一枚，可破关障精进修为，死后修为反馈鉴身 |

### Ⅳ 技法层（Technique）

| 技法 | 应用 | 效果 |
|------|------|------|
| **外人视角评价** | 通过万天仇的感慨来评述李家的气象 | 让读者通过"外人"的惊讶来感受李家的不同寻常，比赘述更有说服力 |
| **仪式三段式** | 准备（镇民布置）→高潮（杀鹿凝箓）→余韵（箓丹凝聚） | 完整的仪式感，节奏张弛有度 |
| **"选项展示"叙事** | 三箓能力逐一展示，陆江仙逐一权衡 | 既展示了世界观设定，又制造了"选择"的悬念 |
| **镜外/镜内视点切换** | 祭祀场面（镇民视角）→ 镜中（陆江仙神识视角） | 双重视角，神凡分离感——陆江仙是"下棋的人" |
| **经济算计的插入** | 陆江仙设想"流通到市面上给筑基/紫府服用后回收" | 在神秘体系中加入经济理性——角色不是圣徒，是有私欲的"棋手" |
| **历史回声** | 箓丹秘法来自李项平当年杀死的练气老修士的玉佩 | 过往的积累在新时代派上用场，增强世界连续感 |
