## 一、人物弧光层

### 李通崖
| 维度 | 内容 |
|------|------|
| 状态 | 练气六层，突破一年，修为进展缓慢 |
| 心态 | 清醒务实，深谋远虑，善于借势 |
| 弧光 | 从被动应对到主动布局——借丁西定之口播下抗郁种子 |
| 关键台词 | "得此一地，我李家便可再有三十年的安心发展，你们且先斗着吧……" |
| 决策逻辑 | 外交诱导 > 武力对抗；隐忍布局 > 当场表态 |

### 丁西定
| 维度 | 内容 |
|------|------|
| 定位 | 蕈林原丁家主事人，练气巅峰 |
| 性格 | 粗中有细，好面子，易受引导 |
| 处境 | 丁家两百年无领土变化，刚得迢云山，缺外部理解 |
| 弧光 | 从客套拜访到被成功引导为郁家制衡棋子 |
| 关键台词 | "看来若是要在这湖边闯出一番事业，还须动乱这郁家啊！" |

### 李玄岭
| 维度 | 内容 |
|------|------|
| 定位 | 李通崖之子，族中决策参与者 |
| 成长 | 从战术性思考（均分符种）到接受战略性思维（每代一麒麟儿） |
| 弧光 | 在符种分配制度上被父亲纠正认知——从"分"到"合" |
| 关键认知 | 认识到"不患寡而患不均"是家族内乱的根源 |

## 二、情节推进层

### 事件链

| 节拍 | 事件 | 推动力 | 悬念/效果 |
|------|------|--------|----------|
| 1 | 李通崖调息完毕，感知修为进境缓慢 | 修为瓶颈 | 揭晓练气后期需要五六年积累 |
| 2 | 丁西定来访，通报身份 | 外交事件 | 新势力入局，望月湖格局改变 |
| 3 | 丁西定攀亲（袁家与青穗峰渊源） | 拉关系 | 李通崖内心不以为然，点明"姻亲不如利益" |
| 4 | 李通崖大吐苦水，渲染郁家威胁 | 诱导策略 | 成功引导丁西定产生抗郁念头 |
| 5 | 丁西定沉思，决定联络外援打击郁家 | 决策形成 | 李通崖不战而屈人之兵 |
| 6 | 李玄岭提议符种分配方案（四脉均分） | 制度建设 | 父子对话，揭示核心家族治理理念 |
| 7 | 李通崖纠正：每代只授一枚，四脉浑一 | 长远战略 | 确立"选麒麟儿 + 法鉴授种"制度雏形 |

### 章节结构：双层叙事
- **外层**：外交博弈（丁西定来访 → 诱导制衡郁家）
- **内层**：家族治理（符种分配制度的辩论与确立）

## 三、世界观层

### 新增/深化设定

| 设定 | 内容 | 类型 |
|------|------|------|
| 蕈林原丁家 | 两百年无领土变化，被袁家视为"守门之犬" | 势力扩展 |
| 练气后期突破规律 | 练气六→七→八→九层，尤其注重积累 | 修为体系深化 |
| 符种分配制度 | 每代只授一枚，四脉浑一，法鉴前蒙眼选麒麟儿 | 家族制度 |
| 袁家-丁家关系 | 世代姻亲，但丁家实际受袁家压制 | 势力关系 |
| 练气巅峰 | 丁西定的修为层级 | 战力标杆 |
| 法鉴授种 | 蒙眼跪法鉴前，由法鉴自行选择符种归属 | 家族仪式 |

### 望月湖势力格局（更新）

```
安家 ——+— 制衡 —→ 郁家（筑基家族）
       |
       +— 引入丁家 → 迢云山（新势力）
       |
       +— 毗邻 ——→ 李家（剑仙家族）

李家 ←—— 外交诱导 ——→ 丁家 ←—— 共同制衡 ——→ 郁家
```
