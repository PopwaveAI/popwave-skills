## 三、世界观层

### 新增/深化设定

| 设定 | 内容 | 类型 |
|------|------|------|
| 黄玉灵米 | 练气修士的修仙资粮，价格是普通灵稻4-5倍 | 修仙经济 |
| 黄玉灵米种植条件 | 需灵雨、梳理地脉；玉京轮修士才能独自栽种 | 农业体系 |
| 李家灵田产出 | 每五年450斤黄玉灵米 + 3000多斤普通灵稻 | 家族经济 |
| 宗门供奉标准 | 每五年上交300斤黄玉灵米 + 2000斤普通灵稻 | 势力义务 |
| 胎息修士分工原则 | 天赋差修为低者种田，玉京轮以上学符术/突破练气 | 人才管理 |
| 华芊山下三村 | 五千余人口，卢家统治下民生凋敝 | 社会对照 |
| 四镇人口红利 | 十万出头的镇民，每年一千婴幼儿出一两个灵窍者 | 人口/修行基础 |
| 李平逸人设 | 李谢文嫡长子，身无灵窍，安排为李渊修从属 | 凡人路线 |

### 家族管理哲学
```
天赋修士管理分层模型：
  胎息低层（低级灵田）↔ 种灵稻、做杂务
  玉京轮（中高级灵田）↔ 种黄玉灵米、学符术
  练气（战略资源）    ↔ 突破修为、战力、外交
```
