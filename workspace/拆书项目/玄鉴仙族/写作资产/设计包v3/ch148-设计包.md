## 三、世界观层

### 新增/深化设定

| 设定 | 内容 | 类型 |
|------|------|------|
| 法器定制流程 | 找店主 → 谈规格 → 定材料 → 定价格 → 立契发灵誓 | 修仙经济体系 |
| 法器定价模型 | 原材料80灵石 + 工费40灵石 = 120灵石（练气级长弓） | 经济设定 |
| 青乌金精 | 以普通青乌矿提炼，适合传导金属性罡气 | 材料体系 |
| 金方屏石 | 弓身材料，需玄火煅烧，承受五千斤之力 | 材料体系 |
| 玄景灵誓 | 坊市交易的契约担保机制 | 规则设定 |
| 冠云峰坊市 | 具备法器定制能力的坊市节点 | 地图/经济 |
| 练气级法器价格区间 | 120灵石级的长弓为练气上品 | 战力/经济标尺 |
| 萧家（坊市立契方） | 第三方契约担保势力 | 势力设定 |
| 李家法器弓设计 | 五千斤拉力，金属性罡气传导 | 战斗风格设定 |

### 法器定制产业链
```
修士需求 → 方案设计 → 材料采购（青乌矿自备/外购）
  → 玄火锻烧（126天） → 符文刻录 → 组装调试 → 立契灵誓 → 交付
```
