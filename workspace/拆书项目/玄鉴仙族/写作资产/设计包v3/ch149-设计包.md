## 一、人物弧光层

### 李玄锋
| 维度 | 内容 |
|------|------|
| 状态 | 练气修为，刚取得金庚弓 |
| 弧光核心 | 从"玩家/使用者" → "开创者/武道奠基人" |
| 关键节点 | 1) 取弓试拉满月震撼全场 → 2) 试射威力惊人 → 3) 自省"未能与弓配合" → 4) 决定闭关五年撰写弓法 |
| 关键台词 | "却没能同这弓配合起来……准备闭关五年，将一身所学整理一番，推陈出新" |
| 成长意义 | 从依赖蛮力与天赋 → 追求技法/道法融合的系统化 |
| 命名"金庚" | 赋予法器姓名，确立持有者身份 |

### 陈冬河
| 维度 | 内容 |
|------|------|
| 状态 | 胎息巅峰，二十八岁（虚岁） |
| 弧光 | 从丧父沉痛 → 奋发求法 → 获得《江河一气诀》 |
| 关键台词 | "晚辈陈冬河，修为已至胎息巅峰，前来求法！" |
| 家族认同 | 以练气诀+采气偿还模式融入家族体系 |
| 天赋定位 | 比肩郁慕锋（郁家三十岁练气） |

### 楚明炼
| 维度 | 内容 |
|------|------|
| 深化 | 技术高手，对作品有自豪感 |
| 弧光 | 从怀疑（是否有人拉得动这弓） → 震惊（李玄锋拉至满月） |
| 关键台词 | "这怎么可能！此人莫不是妖兽化形！！" |
| 功能 | 技术权威的认证——证明金庚弓和李玄锋的双重含金量 |

### 李通崖
| 维度 | 内容 |
|------|------|
| 角色 | 导师/父亲/家主三重身份 |

## 二、情节推进层

### 事件链

| 节拍 | 事件 | 推动力 | 悬念/效果 |
|------|------|--------|----------|
| 1 | 半年后，李玄锋赴坊市取弓 | 时间过渡 | 季节变化（雪化绿林） |
| 2 | 楚明炼展示金庚弓 | 交付 | 法器规格、品级展示 |
| 3 | 李玄锋拉弓至满月，震撼全场 | 实力展示 | 五千斤之力，暗金流光 |
| 4 | 命名"金庚"，签约解契 | 仪式完成 | 法器与主人绑定 |
| 5 | 回山后与诸人试射 | 威力验证 | 炸倒五六棵树+大坑 |
| 6 | 李玄锋自省，决定闭关五年写弓法 | 战略升级 | 从蛮力到道法融合 |
| 7 | 李通崖督促玄宣玄岭 | 家族规划 | 十年内两人要突破练气 |

### 章节结构：三段式
- **第一段**：金庚弓出世（取弓→命名→试射）
- **第二段**：战力评估与未来规划（自省→督促）
- **第三段**：功法传承（陈冬河求法→授予）

## 三、世界观层

### 新增/深化设定

| 设定 | 内容 | 类型 |
|------|------|------|
| 金庚弓 | 长七尺六寸，角长二尺五寸，暗金色流光，练气上品 | 法器设定 |
| 金庚弓材料 | 金方屏石（弓身）+ 青乌金精（弓弦） | 材料体系 |
| 符文系统（弓） | 金庚弓身刻满聚气凝气符文 | 符文体系 |
| 练气上品法器价格 | 120灵石（80材料+40工费） | 经济标准 |
| 法器命名传统 | 赋予法器以姓名的实践 | 修仙文化 |
| 李家三品练气诀 | 目前有《江河一气诀》《洞泉澈灵诀》两套 | 功法体系 |
| 江中清气 | 练气所需的外景采气资源 | 修行资源 |
| 采气偿还制度 | 家族提供清气突破，练气后采回一份还回家族 | 家族制度 |
| 郁慕锋标杆 | 三十岁练气，郁家称天赋好 | 战力标尺 |
| 陈冬河定位 | 二十八岁胎息巅峰（比肩郁慕锋） | 天赋标尺 |

### 李家功法传承流程
```
胎息巅峰修士 → 求法 → 家族评估 → 选择法诀
  → 家族提供清气（先借） → 突破练气
  → 练气后采一份清气还回家族（偿还模式）
```
