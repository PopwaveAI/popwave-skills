## 一、人物弧光层

### 灵岩子（首次登场）
| 维度 | 内容 |
|------|------|
| 出身 | 三宗七门中的紫烟门，紫气峰传承 |
| 修为 | 练气六层 |
| 初始状态 | 被长霄门追杀三天三夜，狼狈不堪 |
| 性格 | 有韧性（撑三天）、懂得求救、知进退 |
| 弧光 | 从绝望逃命 → 得计脱险 → 进入伏杀圈 |
| 关键道具 | 紫烟灵罩（淡紫色玉珠法器，撑起紫色护盾） |
| 关键信息 | "紫气峰的传承，不要断在我手上" → 身负传承使命 |

### 长霄门玉云子（首次登场）
| 维度 | 内容 |
|------|------|
| 定位 | 追杀者，练气七层+练气六层组合 |
| 性格 | 嚣张、傲慢、贪婪 |
| 关键台词 | "得罪了我长霄门……谁知道哪天就被人灭了门扬长而去" |
| 致命弱点 | 轻敌→以为李家老祖不在→可勒索 |

### 李通崖
| 维度 | 内容 |
|------|------|
| 状态 | 练气六层，面临上层宗门压力 |
| 决策弧光 | 快速权衡→风险判断→制定伏杀计 |
| 关键认知 | "声名所庇，也受声名所累" |
| 策略选择 | 明面示弱（"老祖不在"）→ 暗设伏杀（引至大榕树下） |
| 关键台词 | "最好能悄悄的将这两人杀了，最为稳妥" |

### 狐（大榕树下的狐狸）
| 维度 | 内容 |
|------|------|
| 状态 | 睡觉被惊醒，认出李通崖为"晚辈" |
| 功能 | 1) 神秘存在感 2) 战力伏笔 3) 李通崖的伏杀地点选择 |
| 关键台词 | "哟，来了个晚辈！" |

## 二、情节推进层

### 事件链

| 节拍 | 事件 | 推动力 | 悬念/效果 |
|------|------|--------|----------|
| 1 | 陈冬河求法后，李通崖独叹年近五十修为缓慢 | 时间/修为压力 | 主角自我认知 |
| 2 | 灵岩子被追至李家上空 | 外部事件入侵 | "天外来祸"模式启动 |
| 3 | 灵岩子求救，长霄门威胁 | 危机升级 | 李家被夹在三宗七门之间 |
| 4 | 李通崖快速权衡（帮/不帮/杀） | 决策点 | "悄悄的杀了两人的最稳妥" |
| 5 | 李通崖出面示弱（"老祖不在"） | 策略实施 | 欺骗长霄门 |
| 6 | 传音灵岩子，制定诱敌+伏杀计划 | 暗线联动 | 两全其美之策形成 |
| 7 | 灵岩子飞往湖上，长霄门追击 | 诱敌执行 | 李通崖带李玄锋前往大榕树 |
| 8 | 李通崖到大榕树下与狐狸沟通 | 伏杀准备 | 神秘力量介入可能 |
| 9 | 李玄锋埋伏，李通崖静待 | 伏杀就绪 | 章末悬念——大战将起 |

### 章节结构：五段推进
- **引子**：李通崖独白（50岁危机感）
- **危机触发**：灵岩子求援至李家
- **权衡**：明暗双线思考——帮/杀/不帮
- **布局**：示弱+传音+诱敌
- **就绪**：大榕树下伏杀阵成型

## 三、世界观层

### 新增/深化设定

| 设定 | 内容 | 类型 |
|------|------|------|
| 三宗七门 | 紫烟门（灵岩子所属）、长霄门（玉云子所属）、镗金门（方位参考） | 宗门体系 |
| 紫烟门定位 | 青池宗同盟 | 势力关系网 |
| 紫烟灵罩 | 淡紫色玉珠法器，撑起紫光护盾，撑三天三夜 | 法器体系 |
| 长霄门 | 与紫烟门敌对，追杀出门办事的紫烟门弟子 | 势力冲突 |
| 大黎山外围 | 宗门修士的活动区域，李家地处边缘 | 地图层次 |
| 练气七层+六层组合 | 长霄门追杀者的战力配置 | 战力标尺 |
| 传音入密 | 李通崖以法力传音，灵岩子配合表演（面不变色） | 法术细节 |
| 修士皮球效应 | 小家族互相推诿，将麻烦踢给下一个家族 | 修仙生态 |
| 狐狸（大榕树） | 疑似高阶存在，认识李通崖为"晚辈" | 神秘生物设定 |

### 三宗七门与李家的关系网
```
青池宗（宗主级）── 同盟 ──→ 紫烟门
     │                              │
     │                              ├─ 紫气峰传承（灵岩子）
     │                              │
     │ 名义管辖                      └─ 与长霄门敌对
     │
     └── 李家（边缘小家族，实际被压迫）

长霄门（敌对紫烟门）
     └── 玉云子等追杀灵岩子
         └── 途经李家威胁
```

### 李家面临的风险分析
```
帮灵岩子（公开） → 得罪长霄门 → 灭门风险
不帮灵岩子       → 紫烟门追责 → 青池宗压力
杀长霄门（秘密） → 无人知晓  → 最稳妥（李通崖选择）
```
