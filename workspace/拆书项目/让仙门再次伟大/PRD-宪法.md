# PRD · 让仙门再次伟大
> 黑暗修仙穿越 · 信息差爽文。穿越者在一个双世界（凡人界 + 仙门）的阶级固化世界中冷酷爬升。

## 〇、核心故事

他叫陈玄——白莲教永不兑现的承诺里，最不值钱的那张底牌。一个被当做耗材的卧底，被塞进了一座永远不可能爬出去的凡人界县城。官府拿他当兔，白莲教拿他当柴，他唯一的遗产是一本没人看得懂的《太平经》。那是上一位被耗干的造反者留下的火种——不是金手指，是一份借据，上面写满了前代人的血债和利息。每一次他被榨干、被杀死、被埋葬，这本破书就会在下一具尸体里重新翻开，把他扔进一个更大的棋盘。凡人界→仙门→炼狱→道体，死亡不是终点，是证明他还没死透的下一跳。当所有下棋的人都以为他只是个死掉的棋子时，棺材板下的那双手已经开始算账了。

## 一、爽点锚点

### 1.1 信息差碾压
读者获得「我知道主角知道而反派不知道」的认知不对称愉悦。不是力量碾压，是信息碾压。所有冲突本质是信息不对等的博弈。

| 特征参数 | 描述 |
|:--------|:-----|
| 触发条件 | 至少三方参与的多层博弈结构；主角掌握至少一层对手不知道的信息 |
| 强度峰值 | 单次闭环完成时，读者收获「看穿全局」的智力优越感 |
| 衰减曲线 | 连续同类变体超过三次后递减，需切换变体或升级信息层级 |

### 1.2 阶级革命 / 身份跃迁
每一次阶级跨越 = 旧身份的彻底毁灭。变强不是变美——是变得更不像人。底层唯一的上升路径是「杀死旧身份」。

| 特征参数 | 描述 |
|:--------|:-----|
| 触发条件 | 必须有旧身份可供毁灭；必须有具象的压迫者（不是抽象体制） |
| 强度峰值 | 身份毁灭伴随读者认知的颠覆——熟悉的关系网络被彻底切断 |
| 衰减曲线 | 连续两次身份跃迁无「旧身份毁灭」伴随 → 退化为龙傲天模式 |

### 1.3 穿越者解构 / 黑色幽默
不是搞笑——是绝望中的不屈以荒诞形式呈现。读者永远在高潮后喘不过气时被一个穿越者梗逗笑，然后发现幽默背后更大的危机正在逼近。

| 特征参数 | 描述 |
|:--------|:-----|
| 触发条件 | 现代思维对古代世界的落差点；必须在紧张场景之后 |
| 强度峰值 | 生死关头与荒诞吐槽重叠的瞬间——笑是对死亡的最后一次嘲讽 |
| 衰减曲线 | 同一种梗重复三次后失效 → 需更新吐槽素材库 |

## 二、核心卖点

### 1. 设定卖点

- **双世界独立体系**：凡人界与仙门各有独立的规则体系和力量标尺，不可直接对比。凡人界的铁器和仙门的极光不在同一张评分表上——跨域博弈只能靠信息差和规则漏洞，不能靠战力数值对轰。双世界不是两张地图，是两套游戏规则，读者同时知道两套规则而角色只知其一。
- **死亡驱动的维度跳转**：死亡不是惩罚，是叙事节点的强制性跃迁。每一次死亡对应一次世界维度或力量体系的切换——从卧底到捕快，从天魔解体到器灵重生。不是线性升级，是「套盒子」：每次以为到了顶，上面还有更大的棋盘。

### 2. 世界观/理论体系卖点

- **制度化的阶级压迫**：功法被刻意剥离炼法、秋税制度设计为吸血循环、仙门的「修仙」本质是另一种剥削体系。不是有个别坏人作恶，是整个系统从底层逻辑上就是吃人的。金句贯穿全书——黔首非人论、公私论、再苦一苦——每一条都是世界运转的底层法令，不是角色的牢骚。
- **力量的金融化代价模型**：修仙的本质是「超前享受 + 事后还账」的借贷循环。力量获取的底层语言是金融术语——贷款、结算、抵押、利息、信用。读者在享受升级爽感的同时，永远被「下次怎么办」的紧张感追赶。爽感与焦虑在同一个心跳中震荡。

### 3. 剧情卖点

- **多层嵌套博弈结构**：任何看似双人对弈的局面都至少有第三方（或更多方）潜伏。每个对手的动机都合理，每个盟友的背叛都有前因——没有无脑反派，所有冲突都是信息不对称下的智力博弈。读者永远在「原来还有一层」的震撼中翻页。
- **四阶段叙事脉冲**：压抑→智力满足→血腥快感→黑色幽默→蛰伏希望，按W型情感曲线交替运转。每一轮高潮后不是休憩，是更大危机逼近的信号——螺旋上升的压迫让读者永远无法喘息。

### 4. 主角/金手指卖点

- **穿越者的现代解构视角**：他不是阴谋家，他是被卷入棋局的棋子——靠灵识系统的信息差把棋盘看穿。现代思维对古代世界的解构是黑色幽默的天然来源。每句吐槽都是底层视角的自然产物：在两个荒诞世界的并置中，唯一清醒的只有他。
- **金手指的造反者血统**：《太平经》不是天选恩赐、不是上古传承，是前代造反者留下的火种。金手指的身份来源决定了这本书的叙事立场——它站在底层这边。它不是让你变强的工具，是一份让你在算清旧账时发现利息已经欠了几百年的借据。

### 5. 全书级卖点

- **信息差碾压作为第一爽点手段**：读者获得「我知道主角知道而反派不知道」的认知不对称愉悦。主角赢不是因为他更强，是因为他看到了别人没看到的棋盘。力量碾压可以爽三章，信息碾压能让读者记三百章。
- **每一次翻身都是旧身份的彻底毁灭**：变强不是变帅——是变得更不像人。每一次阶级跨越伴随一段记忆的消失、一个名字的剥落、一段关系的强制断联。读者想看他赢，但每次赢了之后要付出的代价让读者既爽又痛。

## 三、美学基因 + 命名体系

### 美学基因

| 基因 | 定义 | 排他性约束 |
|:----|:-----|:----------|
| **东方黑暗仙侠** | 根植仙侠传统而非日轻/西幻，基调为黑暗博弈而非热血/温馨/搞笑 | 如果叙事基调滑向纯热血对轰或轻小说氛围，基因身份自动丧失 |
| **底层视角** | 主角从凡人界最底层出发，世界运行的基本规则是阶级压迫而非神仙斗法 | 如果叙事重心从底层撕开缺口转移为上层争霸，底层基因被覆盖 |

### 从基因到世界质感

**凡人界的质感**：寒冬的铁灰色天空下，泥泞的官道冻成硬壳，踩上去发出碎裂声。铁器铺里的炭火是唯一的热源，火光映在铁砧上，红的不是铁水是血垢。赤土、锈迹、草木灰——这个世界的美学拒绝任何装饰性元素。每一样东西都指向同一个目的：活着，或者让别人死。空气里有猪粪味、铁锈味、汗酸味，没有香炉、没有灵植、没有「仙气」。底层视角要求世界的第一印象是粗粝的触感，而非观赏性的「沉浸感」。

**仙门的质感**：凡人仰望仙门时看到的是五彩极光撕裂夜空，但极光之下是无边黑暗中的高台——没有祥云、没有仙鹤，只有悬浮在虚无中的祭坛。仙门的力量显现从来不是光芒万丈的「大能降临」，而是诡谲的、不祥的奇观：黑暗中的符文像伤口一样裂开，极光在血色中徐徐转动。仙门的美学本质是「令人不安的壮丽」——它美，但美得让人想逃。

**两个世界质感相反，本质相同**：凡人界的粗粝和仙门的诡谲，共同指向同一件事——把底层压榨到极致。凡人界用寒冷和匮乏压榨肉体，仙门用恐惧和不可知压榨灵魂。两个世界的美学差异不是风格选择，是压迫手法的两种显现。

### 从基因到力量美学

**力量的外观**：凡人界的力量没有任何观赏性——它是铁锹劈入泥土的闷响、是拳头砸碎牙齿的骨碎声、是拼命时握刀滑脱的血印。这是功能性暴力，每一分力都指向生存本身，多余的美感是奢侈。赤土的强者不在乎出手是否漂亮，只在乎对面是否还能站起来。仙门的力量则是诡谲的谜面——你看到五彩极光在黑暗中凝结成某种几何形状，但你不知道它意味着什么；你看到符文在虚空中自行燃烧，但你猜不到点燃它的代价。仙门的力量美学拒绝「帅气的打斗」——它应该是让人脊背发凉的未知。

**变强的代价**：每一次突破对应的不是更帅的招数，而是旧身份的毁灭。从凡人到修士不是加冕，是剥皮——剥掉你的姓氏、你的面孔、你记得的所有人的名字。力量美学上这意味着：突破场景不能是「金光沐浴、经脉舒张」，必须是某种不可逆的丧失：一只手在突破后化成了颗粒，一段记忆在晋升后永远消失，一个关系被规则强制断联。读者需要看到「获得」与「失去」在同一个画面中同时发生。

**力量的来源**：这不是关于天赋或努力的叙事——主角变强是因为信息不对称带来的认知套利、是因为找到了系统规则中的漏洞、是因为愿意下死亡赌注而对手不敢。力量的获取场景在美学上应该呈现为：一场精密的算计（智力不对称）、一次危险的操弄（规则漏洞）、一局赌上性命的押注（死亡赌注）。这些场景的美学质感不是修炼而是赌博——火光下的眼神、颤抖的指尖、落子无悔的仪式感。力量的来源不是汗水和天赋，是胆量和脑子。

### 从基因到情感光谱

**禁用情感**：纯热血（「我要守护这一切！」）、纯温馨（岁月静好的日常）、纯绝望（从头丧到尾的压抑）——全部禁用。热血温情属于龙傲天叙事，纯绝望属于纯黑暗文，它们都不属于这本书。情感光谱的根本约束是：任何情感都必须同时携带「不对」的杂质。

**情感节奏：脉冲式W型**。四个波峰按固定顺序排列——压抑（世界施加的压力，让读者喘不过气）→ 智力满足（主角用信息差破解困局的爽感）→ 血腥快感（冲突兑现的暴力或残酷释放）→ 黑色幽默（穿越者视角的荒诞消解，把紧绷的弦突然松开一个齿距）→ 蛰伏希望（在压抑中埋下一条细若游丝的线索，不保证兑现，但给读者一个「也许还有机会」的念头）。五个节点构成一个W型周期，每个周期在高潮后指向更大的危机——不是循环，是螺旋上升的压迫。

**情感的特殊质地**：黑色幽默不是调味料，不是「在压抑文里加点搞笑」，它是底层视角的自然产物。一个从现代穿越到修仙世界底层的人，当他看到仙门的宏伟秩序和凡人界的血污泥泞时，他天然会说出一句荒诞的吐槽——不是因为作者想搞笑，是因为在这个人的坐标系里，两个世界的并置本身就是荒诞的。黑色幽默是角色的自然反应，不是作者的设计。如果黑色幽默让读者感到「作者在硬逗我笑」，那就失败了。

### 命名体系

- **凡人界命名法则**：民间直白语汇。铁器就叫铁器，血就叫血，炭火就叫炭火。不使用任何佛教术语、道家玄词、仙侠常见词汇进入凡人界。拒绝「灵器」「法器」「丹药」「灵石」。外号来自市井特征（瘸三、麻婶、铁头），不来自修行等级。
- **仙门命名法则**：以佛教名相改造为骨架（欲界天层级结构的挪用与篡改、天魔/人道/炼欲成力的逆向使用），以阴符经盗天哲学为底层逻辑（「天人合发，万变定基」反用为「天人相盗，万变我基」）。拒绝正统修仙画风（剑道/丹道/御剑飞行），也拒绝日系轻小说感（XX者/XX使/公会/技能树）。
- **核心金手指命名法则**：名称必须来自民间反抗传统——造反者的圣经（太平道、黄巾起义、方腊、白莲教的遗骸）。金手指不是上古传承，不是天选恩赐，是底层在历史黑暗中自己凿出来的反噬工具。金融隐喻用于代价结构（贷款/结算/抵押/信用/杠杆）而非命名本身——名称是身份标识，代价结构是规则骨架，两者不混淆。
- **禁止穿越者现代名词入侵叙事语境**：主角可以心里吐槽，但对话和叙事旁白中不出现「系统」「任务」「升级」「属性面板」「充值」等现代游戏术语。穿越者的现代性只通过行为模式和思维结构体现，不通过词汇感染世界观。
- **禁忌命名清单（不完整举例，仅作方向约束）**：不含「灵」「仙」「道」「丹」「剑」等传统仙侠高频词根作为凡人界事物命名；不含英文音译/日式汉字组合；不含「圣」「神」「帝」「尊」用于任何正派命名；不含「天」「苍」「玄」「昊」等中二高频汉字用于主角能力命名。

### 视觉锚点

雪夜。城外乱葬岗。陈玄的胸口被白莲教的符钉穿了个对穿，血在雪地上洇成一朵正在凝固的红花——像一枚刚盖下去的印章。天地之间只剩两种颜色：死的白，和正在死的红。他听见自己的心跳变成钟摆，一下一下，越来越慢。然后他笑了一声。不是因为疼。是因为他在最后一秒想明白了一件事——白莲教、官家、那位高高在上的仙门执事，他们以为自己在下一盘大棋，但棋盘是从《太平经》里那页破纸上撕下来的。他在想，等下一局开始的时候，他们最好把牌洗得干净点。

## 五、世界宪法

### 宪法一：信息差驱动一切核心冲突
所有核心冲突的本质是信息不对称。胜负不取决于谁更强，取决于谁看到了对手没看到的棋盘。**违反后果**：如果核心冲突不再由信息差驱动，这本书不再是这本书。

### 宪法二：阶级压迫不可逆
凡人界的阶层是绝对固化的——底层没有真实的上升通道，所有「希望」（修行路径、官场晋升、宗教救赎）都是压迫系统设计的陷阱。**违反后果**：如果叙事开始合理化或美化阶级压迫系统，核心承诺崩塌。

### 宪法三：死亡是穿越节点
死亡不是终点，是叙事维度的强制性跳转。每次死亡对应一次力量体系代际跃迁或世界维度切换。**违反后果**：如果出现无死亡的升级路径，金手指底层设定被违反。

### 宪法四：力量必须有代价
所有金手指都不是白给的。超前享受必须事后还账——借贷模型贯穿整个力量系统。**违反后果**：出现无代价的突然升级且无规则线索 → 机械降神 → 核心信任崩塌。

### 宪法五：黑色幽默是减压阀不是调味料
在紧张场景之后必须留有情绪缓冲空间。黑色幽默是结构性组件，不是可选的调味料。**违反后果**：连续过长篇幅无黑色幽默 → 读者被压抑逼走；黑色幽默瓦解黑暗基调 → 读者失去紧张感。

### 宪法六：禁止纯双人对弈
任何重要冲突必须有至少三方参与或潜伏。读者不能相信任何「看似简单的对抗」——每个表面上的双人对弈，都有未被揭露的旁观者或操盘手。**违反后果**：冲突张力自动下降，叙事滑向标准二元对抗模式。

## 六、跨域素材指引

- **阴符经（盗天哲学）**：为力量体系提供哲学底盘——从「顺天」反转为「盗天」。只进入力量体系架构层，不进入角色对话。
- **佛教名相（层级宇宙）**：提供层级化宇宙结构（欲界天）和对抗性概念（天魔 vs 正道）。用于仙门命名和身份体系，不侵入凡人界命名。
- **三十六计（博弈结构）**：提供信息差冲突的博弈原型。计谋转化为情节行动，不写成说教或注释。
- **金融体系（代价模型）**：提供精确的代价-收益模型和不对称收益框架。用于金手指规则骨架（贷款/结算/信用），不侵入功法命名。
- **民间反抗传统（太平道 / 黄巾起义）**：提供造反者的叙事原型。金手指的身份来源和力量体系的社会后果由此决定。

## 七、篇幅规划

- **目标总字数**：~200 万字区间
- **预估卷数**：5-6 卷，每卷对应一次力量体系代际跃迁或世界维度切换
- **爽点密度**：每章至少嵌入信息差类 + 阶级类 / 幽默类交替脉冲；每幕至少一个高密度高潮区
- **升级节奏**：每卷 2-3 次力量质变节点，每次伴随旧身份毁灭

## 八、核心承诺 + 防崩阈值

### 承诺 1：智力碾压的暗爽——「我比所有人都多看一步」
每场冲突让读者感受到主角掌握对手不知道的信息。主角赢不是因为更强，是因为看到了对手没看到的棋盘。
- **防崩信号**：连续较长篇幅无信息差碾压结构 → 叙事滑向力量碾压。反派不能降智——每个反派各有各的聪明，主角赢在信息差更高一层。

### 承诺 2：阶级压迫下的反抗快感——每一次阶级跨越伴随身份毁灭
底层用智力、金手指和「死亡—重生」机制掀翻压迫系统。主角可以利用系统规则，但不能在叙事层面认可制度有道理。
- **防崩信号**：连续较长篇幅未触及阶级压迫议题 → 叙事脱离世界观地基。主角开始说「这个制度也有它的道理」→ 核心承诺崩。

### 承诺 3：穿越者的荒诞笑声——死前还在算收益
在最黑暗的场景中突然被穿越者梗逗笑——这个笑不是脱节，是主角对死亡的最后一次嘲讽。
- **防崩信号**：幽默不能瓦解主基调（决死场景的黑色幽默是「笑对死亡」，不是「这只是一场游戏」）。过长篇幅无黑色幽默 → 读者被压抑逼走。

### 承诺 4：有代价的力量——先享受后付费
每次力量获取都是一笔「贷款」——读者在享受升级爽感的同时保持紧张：「他这次用了，下次怎么办？」
- **防崩信号**：出现无代价的突然升级且无提前规则线索 → 机械降神 → 核心信任崩塌。

## 九、不可妥协 DNA + 边界

### 不可妥协 DNA

**DNA 1：信息差碾压作为第一爽点手段**
全书最高频的爽点兑现方式。如果有一天主角靠纯力量碾压对手且读者不觉得降智，这本书不再是这本书。

**DNA 2：穿越者的现代性**
穿越者身份不是背景设定——是核心叙事工具：吐槽、现代思维解构、资本式算计。淡化后主角和其他修仙主角没有区别。

**DNA 3：体系化力量设计**
所有力量必须有可理解的规则边界。不接受「模糊的悟了」或「境界到了自然就懂了」。

**DNA 4：阶级批判底色**
世界运行的基本规则是阶级压迫。不能为了爽而洗白压迫系统。主角可以利用系统规则，但不能在叙事层面认可制度有道理。

### 硬边界（禁区）

- ❌ 不写龙傲天式碾压——主角靠信息差和智力赢，不靠纯力量碾压
- ❌ 不写降智反派——每个反派各有各的聪明，主角赢在信息差更高一层
- ❌ 不写后宫 / 种马——感情线不是核心驱动力
- ❌ 不写机械降神——新能力必须有提前的规则线索
- ❌ 不写无底线轻松的搞笑文——黑色幽默是缓冲，黑暗是主基调
- ❌ 不写设定灌水的过渡章——每一章必须推进剧情或揭示世界观
- ❌ 不写纯双人对弈——任何冲突至少有 3 方参与或潜伏
- ❌ 不写无代价的升级——每次力量获取必须伴随可量化的代价
- ❌ 不写「制度有道理」的妥协——主角可以利用体制但不能认同体制
