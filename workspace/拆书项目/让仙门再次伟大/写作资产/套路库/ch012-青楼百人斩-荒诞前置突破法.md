## ch012 — 青楼百人斩（荒诞前置突破法）

**标签:** `荒诞` `爽文` `突破前置` `反差爽`

### 四段式拆解

| 阶段 | 内容 |
|------|------|
| **引发** | 主角在青楼以一敌百，用香艳战绩构建荒诞期待（"都进去了，一个也没出来"） |
| **发展** | 侧面视角不断强化（凤姑娘视死如归→谢石憋屈→丫鬟惊呆），叠加第三人称震惊 |
| **高潮** | 百人斩完成→立刻盘膝修炼→吸收灵气→肉身异变→欲界天人身成就 |
| **落地** | 灵识觉醒 → 刀法圆满 → 发现楼下谢石（章末钩子，故事继续推进） |

### 原文锚点

> 「都进去了，一个也没出来，全被拿下了。」——丫鬟
> 「这是刚练【铁牛功】？哪怕大圆满了也就这个水准吧，可能还要更强！」——凤姑娘
> 「第一百零八个！」——王平

### 复用规则

| 维度 | 说明 |
|------|------|
| **适用场景** | 主角在非正统修炼场所、用非正统方式完成某个前置条件，再转入正式突破 |
| **核心机制** | 荒诞铺垫（笑点） + 严肃突破（爽点） = 反差叠加，1+1>2 |
| **替换变量** | 场所（青楼/赌场/市井/斗兽场）、前置条件（百人斩/万人敌/千杯不醉/赌神）、突破内容（功法/体术/兵刃） |
| **风险提示** | 荒诞度控制在读者能接受的范围内，不能过度低俗，建议用侧面视角（丫鬟/旁观者）做缓冲 |
| **情绪节奏** | 前1/3荒诞好笑→中1/3严肃紧张（异变）→后1/3爽快爆发→章末悬念 |
