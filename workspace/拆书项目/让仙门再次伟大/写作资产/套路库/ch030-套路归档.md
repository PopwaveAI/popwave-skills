# ch030「武道之心」— 套路归档

## 本章使用的写作套路

| 套路 | 用法 | 效果 |
|------|------|------|
| **传功成果兑现型** | ch029 传授的【叩仙门】→ ch030 开篇突破【穿云震天弓大成】 | 跨章连续爽——让读者感受到「付出了就有回报」的正向激励 |
| **良心抉择·自我牺牲型** | 王平在逃亡和牺牲之间选择后者——「我有无数次机会，别人只有一次」 | 锁定主角的核心价值观，让读者产生强烈角色认同 |
| **芳心彻底沦陷型** | 苏夫人对比陈浩彦后确认王平真心——主动献身+传授家族秘术 | 情感线高光时刻，之前所有铺垫在此刻爆发 |
| **禁忌秘术入手型** | 天魔解体大法——自废武功换数倍内劲 | 获得大杀器底牌，制造「将在何时使用」的期待 |
| **英雄有苦说不出型** | 王平内心尴尬「用力过猛，复活的话说不出口了」 | 喜剧反差——苏夫人真情流露 vs 王平的窘迫秘密 |
| **装备套装化型** | 六件凡兵整合为利器套装，龙虎风云开路 | 游戏化的装备升级仪式，给读者「练级收获」的满足感 |
| **缩地成寸·降维碾压型** | 灵识模拟神意，三步登城墙，全城震慑 | 本章最高爽点——实力碾压带来的「非人」恐惧 |
| **妖魔化名声的反向兑现型** | 王平在县城被妖魔化，本人出现时全城震慑 | 先建立传说再兑现，让读者有「果然如此」的满足感 |
| **确认敌人虚弱·落井下石型** | 幻视大鹏蜷缩含恨，「老弱病残占三，痛打落水狗」 | 反攻号角——从逃命到主动追杀的角色逆转，极具爽感 |
