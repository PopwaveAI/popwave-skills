# ch032「立谈中，死生同，一诺千金重」— 套路归档

## 本章使用的写作套路

| 套路 | 用法 | 效果 |
|------|------|------|
| **以弱胜强（金手指反杀型）** | **（复用）** 王平以死相搏，一箭穿四卫伤徐秉正——虽未能杀敌，但以蝼蚁之躯撼动参天大树 | 虽败犹荣——造成的威慑和人心瓦解远超物理伤害 |
| **诛心战术（心理战型）** | ⭐ **新套路**——王平那一箭的根本目的不是杀人，而是诛心：暴露徐秉正"用下属挡箭"的作风，瓦解县衙人心 | 把死后余波变成杀伤力——徐秉正赢了战斗，输了人心 |
| **慷慨赴死（太平经结算型）** | ⭐ **新套路**——王平笑着赴死，将尸体交给太平经结算 | 独特的"主角死亡"处理方式——不是真死，而是"以命换能力"的循环 |
| **阶级批判（反派真心独白型）** | **（复用·强化）** 徐秉正内心独白「有些人的命就是比其他人贵重」 | 阶级主体的核心价值观赤裸表达，比任何辩论都有力 |
| **丹药博弈（真假混杂型）** | **（复用）** 王平在天意回魂丹外包裹断肠丸——徐秉正用人命试药才能剥出真丹 | 遗计的力量——即使死了，还在给对手制造麻烦 |
| **尸身消失（诡异钩型）** | ⭐ **新套路**——主角死后尸体凭空消失，引发多方猜疑 | 制造悬念——读者和剧中角色都在困惑，为复活/结算制造期待 |
| **守丧落泪（情感延迟兑现型）** | ⭐ **新套路**——苏夫人的情感在失去后才真正觉醒 | 延迟的情感宣泄比即时反应更有力量 |
| **长生不老药（朝廷阴谋型）** | **世界观推进**——天意回魂丹=长生不老药的真相首次揭露 | 将江湖争斗升格为朝堂阴谋，扩展故事格局 |
| **章标题点题（诗词收官型）** | 章标题「立谈中，死生同，一诺千金重」在终场被轻声念出 | 标题与内容形成闭环——全章气势凝在这一句里 |

## 可提取的新套路模板

### 套路名称：诛心战术（心理战型）

**核心公式：** 战斗的主要目标不是物理杀伤，而是通过牺牲/攻击行为暴露对手的价值观/弱点，瓦解对手的团队/人心

**关键要素：**
1. 主角明知物理击杀成功率低
2. 选择一种"展示对手本质"的攻击方式
3. 对手虽然赢了战斗，但输了人心/声望
4. 余波的效果在战斗后才显现

**用法示例：**
- 王平死前一箭，穿透四老捕快之躯——暴露徐秉正让下属挡箭的本质
- 徐秉正赢得战斗，但县衙人心尽失

### 套路名称：慷慨赴死（太平经结算型）

**核心公式：** 主角主动赴死 → 尸体被金手指回收 → 转换为资源/能力 → 为后续复活埋下伏笔

**关键要素：**
1. 主角在死前安排好了所有后事（因果了结）
2. 赴死时没有恐惧，反而期待（因为知道金手指会运作）
3. 尸体消失，制造诡异悬念
4. 对手/配角以为主角已死，情绪各有不同

**用法示例：**
- 王平大笑赴死，尸体消失
- 苏夫人守丧落泪，徐秉正误判是守冲所为

### 套路名称：尸身消失（诡异钩型）

**核心公式：** 关键角色死后尸体凭空消失 → 所有人都在猜疑 → 悬念贯穿后续多章

**关键要素：**
1. 死亡被多人目击确认
2. 尸体在众目睽睽下消失
3. 各方角色做出不同的猜测（都与真相有偏差）
4. 读者知道主角还有金手指，但不知道具体机制

### 套路名称：守丧落泪（情感延迟兑现型）

**核心公式：** 角色A在关键事件中表现出超常的冷静/坚强 → 在后续的安静时刻才崩溃/流泪 → 情感延迟释放更有力量

**关键要素：**
1. 事件当时：角色不哭不闹，甚至大笑/冷静
2. 事后安静时刻：无人注意时，情感才涌出
3. 触发的不是事件本身，而是某个具体的回忆/细节（如一句承诺）
4. 呈现出"坚强者的脆弱"的感人效果

**用法示例：**
- 苏夫人听到陈浩彦被杀——大笑（ch029）
- 苏夫人确认王平身死——七日后在桃林落泪（ch032）
- 触发点：回忆中王平说的「此生定然护你周全」
