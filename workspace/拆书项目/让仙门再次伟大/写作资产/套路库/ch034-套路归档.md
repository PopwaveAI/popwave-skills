# ch034「渡劫，炼炁，修真而登仙」— 套路归档

## 本章使用的写作套路

| 套路 | 用法 | 效果 |
|------|------|------|
| **反差幽默型** | 七天叩仙门失败吐槽「什么垃圾修仙！」 | 用幽默化解读者对主角受挫的潜在失望 |
| **世界观大展开型** | 通过玉简一次性交代修仙全体系——渡劫三境→炼炁→修真登仙 | 从武道转入修仙的关键转折章，让读者对后续升级路径一目了然 |
| **瓶颈突破·解惑型** | 王平悟出气禁不能大成是因为「没有提炼出法力，硬体不达标」 | 用新设定解答旧伏笔，给读者「原来如此」的满足感 |
| **受挫-补偿节奏型** | 七天叩仙门失败→立刻获得功法玉简；寇豹突袭受挫→立刻使用一诺千金 | 不让负面情绪持续太久，每次受挫都紧跟正反馈 |
| **金手指实战展示型** | 一诺千金的完整操作流程——写承诺→验证可行性→提前获得效果→事后还贷 | 新金手指机制首次完整展示，比口头介绍更有说服力 |
| **贷款/透支修炼型** | 用未来时间换当下能力——千日内可达成→贷→还贷→再贷→加速螺旋 | 「左脚踩右脚飞升」的独特修炼体系 |
| **修炼蒙太奇/加速型** | 「半个月还完→贷款到第三层→两个半月还完→贷款到圆满」跳跃叙事 | 省去详尽日常修炼描写，直接展示成果 |
| **欲兽驯服·克制碾压型** | 紫光瞳术让四头欲兽瞬间匍匐——功法存在天生克制 | 从被攻击到掌控者的瞬间反转 |
| **身份跨越型** | 「从今以后我就是修仙者了」——从武者到修仙者的定性转变 | 重大身份升级带来的读者满足感 |
| **喜剧金句定锚型** | 「绝贷天骄，舍我其谁？」——贷款梗给修炼体系打上主角个人烙印 | 让严肃的修仙升级带上幽默色彩 |
