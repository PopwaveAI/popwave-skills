# ch035「盗天贼」— 套路归档

## 本章使用的写作套路

| 套路 | 用法 | 效果 |
|------|------|------|
| **世界观揭秘（二十四节炁型）** | ⭐ **新套路**——通过六欲浑天魔的视角展示修仙者眼中的真实世界：二十四道极光般的炁 | 视觉化呈现「修仙开天眼」的概念，把抽象世界观转化为可感的视觉奇观 |
| **脱胎换骨（蜕皮新生型）** | ⭐ **新套路**——融体重铸+蜕皮新生的痛苦与畅快并存的突破体验 | 让读者「感受」到修炼的代价和回报——痛苦越大，突破越震撼 |
| **物种碾压（修仙vs武道型）** | **（复用·强化）** 一口灵气=十年苦功，武功彻底散功 | 确立修仙体系的绝对优越性——不是等级差异，是物种差异 |
| **开源革命（盗天贼型）** | ⭐ **新套路**——仙门弃徒将自己的功法在仙门内部「开源」，造就劫修群体 | 现代概念映射仙侠——「开源」=最早的劫修诞生根源，极具创意 |
| **制度批判（劫修土壤型）** | **（复用·深化）** 王平内心：只要仙门限制上升通道，劫修就永远存在 | 将「劫修为什么杀不干净」的根源归结为制度问题——反派的诞生是因为系统本身 |
| **笑容下的寒意（美艳威胁型）** | ⭐ **新套路**——尤道姑美艳笑容中许诺「惊喜」，同时强调「人牲价值」 | 美丽与恐怖的反差——用最甜的语气说最冷的话，令人不寒而栗 |
| **隐藏实力的焦虑（苟道生存型）** | **（复用）** 王平渴望隐藏修为的秘法，决定在仙门继续苟住 | 保持主角清醒的人设——不因战力飞跃而膨胀 |
| **战力盘点（跨界对比型）** | 王平评估本体+六欲浑天魔能否碰封号武师 | 保持读者对战力体系的认知——用旧体系（封号武师）衡量新体系（修仙） |
| **武功散尽的伏笔（破而后立型）** | 天鹏擒龙功体崩解，但暗示可以换更强的功体 | 为未来武道+修仙双修埋下潜力钩子 |

## 可提取的新套路模板

### 套路名称：世界观揭秘（二十四节炁型）

**核心公式：** 主角获得新视角/新能力 → 看到凡人/普通修士看不到的世界真实面貌 → 揭示世界观深层次设定

**关键要素：**
1. 需要一个「转换器」（如六欲浑天魔）来切换视角
2. 新视角下的世界与常识认知有巨大差异
3. 信息密度大——一次性揭示大量世界观设定
4. 同时揭示主角当前修炼路径的下一步指引

**用法示例：**
- 王平通过六欲浑天魔看到二十四节炁——五彩极光般的世界
- 了解「二十四节炁」与炼炁修行的关系
- 发现其中一道炁与自身功法产生共鸣

### 套路名称：脱胎换骨（蜕皮新生型）

**核心公式：** 修炼突破 → 身体被彻底重塑（剧痛）→ 蜕去旧皮/旧壳 → 获得远超过去的全新体魄

**关键要素：**
1. 突破过程包含强烈痛苦——用具体比喻让读者感受到（如头顶开口灌水银）
2. 旧身体被「蜕去」——有视觉化的剥落过程（腐烂皮肤、血水）
3. 新生体魄有明显的外在神异（金色肌肤）
4. 力量展示——用简单动作（踩碎青砖）展示质的飞跃
5. 修炼前后的力量对比——用旧体系的巅峰（天魔解体）作为参照物

**用法示例：**
- 王平呼吸灵气→融体重铸→蜕皮→金色真身
- 一脚踩碎青砖，媲美天魔解体状态
- 体魄直接跨入「修仙者」等级，碾压一切武道成就

### 套路名称：开源革命（盗天贼型）

**核心公式：** 体系中不得志的天才 → 将自己的核心功法「开源」散播 → 形成不受掌控的地下势力 → 揭露体制产生反叛者的必然性

**关键要素：**
1. 反叛者不是恶人——是被体制拒绝的天才
2. 「开源」= 降低修行门槛，人人都能学
3. 所有修炼者天然成为同门/盟友
4. 形成制度外的平行生态（劫修）
5. 揭示核心矛盾：只要上升通道被垄断，地下势力就永远存在

**用法示例：**
- 盗天贼将功法散播到全仙门 → 劫修遍地
- 王平内心判断：这是制度问题

### 套路名称：笑容下的寒意（美艳威胁型）

**核心公式：** 角色用最美丽/温柔的表情说最可怕的话 → 美丽与威胁的反差制造惊悚效果 → 让读者和主角同时不寒而栗

**关键要素：**
1. 笑容必须是「美艳倾倒众生」级别的
2. 台词用温和/赞赏的语气说出「惊喜」「奖励」等正面词汇
3. 在关键位置用特定词汇暗示危险（如「人牲价值」）
4. 给读者留白思考的空间——「惊喜」到底是什么，让读者自己脑补
5. 主角的恐惧反应作为印证——让读者确认那确实是威胁

**用法示例：**
- 尤道姑露出美艳笑容：「给你一个惊喜……让你充分实现自己的人牲价值」
- 王平立刻想起死人税和仙门力士的制作——后背发凉
