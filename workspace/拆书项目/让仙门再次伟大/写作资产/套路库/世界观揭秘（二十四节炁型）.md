# 世界观揭秘（二十四节炁型）

**套路类型**：世界观拓展型 / 视角转换
**首次出现**：ch035
**关联套路**：升维对比、底层逻辑暴露

## 四段式拆解

```
【起】主角获得新能力/新视角（六欲浑天魔/灵识切换）
      ——视角转换的「触发装置」亮相
【承】新视角下的世界与常识认知完全不同——
     「在此之前，他眼中的世界和灵识感应到的世界并无区别……然而现在不同了」
     世界呈现出凡人看不到的「真实面貌」（二十四道极光）
【转】这个真实世界存在某种秩序/规律（二十四节炁节制万象）
     ——揭示世界观的核心设定原则（炼炁修行、二十四节炁）
     同时暗示主角修炼的下一步方向（与其中一道炁共鸣）
【合】视角的获得不是一次性的——它永久改变了主角的认知
     ——主角从此用「双重视野」看世界
```

## 核心效果
- 视觉化呈现抽象世界观——把「灵气」这种设定变为可感可观的极光奇观
- 制造「开天眼」的认知升级感——读者随着主角一起「看到」真实世界
- 一次性收获大量设定信息——单场景完成世界观深度拓展
- 为后续修炼体系提供具象化指引（哪一道炁、如何共鸣）

## 复用条件
- 故事存在凡人看不到的「第二层世界」（灵气、炁、法则、规则等）
- 主角即将从「普通人」晋升为「修行者」
- 需要一种视觉震撼的方式来展示世界观设定（而不是说明文）

## 变体可能
- **法则视角版**：主角看到天地间的法则/规则之线
- **因果视角版**：主角看到人与人之的因果线/缘分线
- **灵力视角版**：主角看到灵气/法力的流动轨迹
- **阴阳视角版**：主角看到阴阳二气的运转
