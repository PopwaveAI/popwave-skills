# 套路：世界观解密（武功体系建立型）

## 元信息

- **套路名称**: 世界观解密（武功体系建立型）
- **首次出现章节**: ch009
- **所属母套路**: 世界观建立/设定展示类套路
- **复用性**: 高——适用于任何需要系统地介绍完整世界观体系的场景

## 触发条件

一个掌握全局信息的大佬（师傅/前辈/官方权威）在公开场合进行「讲课式」信息释放 → 体系化知识从顶层到底层逐层展开 → 听众（包括主角）以凡人的视角接收震撼 → 知识释放完毕后自然引出出正在发生的剧情或任务

## 套路公式

```
公开讲课场景建立世界观 →
体系化知识逐层展开（五等→六要素→品级→升华机制）→
用「凡人」的震撼反应强化体系落差（众捕快绝望）→
体系内「更高层级」的存在制造向往（中乘武功可一门多淬炼）→
自然引出该体系在本章的核心用途（为本剧情做背景铺垫）
```

## 核心张力

读者跟随主角一起接收完整的武道世界观体系——这不是零散的知识碎片，而是一张完整的「武功科技树」。在 ch009 中，兵器五等=装备进度条，外功六要素=角色属性面板，武功品级=技能等级——读者瞬间获得了该世界的「底层规则」。相比于碎片式信息投放，集中式世界观解密能让读者更快建立对世界的信任感和掌控感。

## 关键证据

- ch009: 「天下兵器，共有五等。凡兵，利器，名珍，神物，天工」
- ch009: 「皮如牛，肉如象，筋如龙，骨如虎，血如火，髓如霜」
- ch009: 「【阎罗三魔刀】主攻的就是筋，只要入门就能做到筋如龙」
- ch009: 「更高明的中乘武功……就可以一门武功同时淬炼多个，甚至全身部位」
- ch009: 「六门武功就是六件兵器，合著要凑齐六神装才算得上外功圆满？」

## 情绪收益

认知刷新快感 + 对世界底层规则的掌控感 + 对更高层级的向往 + 科技树式的清晰成长路径 + 从「什么都不知道」到「知道体系」的充实感

## 变体方向

| 变体方向 | 示例 |
|---------|------|
| 宗师讲课型：由地位最高的人公开讲授体系知识 | 宗门大长老讲修仙境界 |
| 秘籍自学型：主角从秘籍/书籍中自己获取体系知识 | 捡到《天地境界总纲》 |
| 战斗悟道型：在实战中逐步发现体系规则 | 打贏才知道原来修为分这些等级 |
| 反派自曝型：反派为了装逼/炫耀主动揭示体系 | 「你可知我为何能碾压你？因为境界之差！」 |

## 下游期待

- 该体系内的「更高层存在」何时登场？
- 主角需要多久才能填满体系内的所有位置？
- 体系内的「异常值」（超出体系的存在）是否存在？

## 复用提示

- 体系化解密最好在一个场景内完成，不要中断——断开的体系信息会大幅降低读者的整理效率
- 用「凡人」的反应作为基准线——让读者知道正常人应该怎么想，再对比主角的差异
- 体系揭秘的结尾处一定要「自然的引出本段剧情的用途」——否则就成了「百科知识」而不是「剧情信息」
- 六要素可以用「游戏化翻译」（六神装/属性面板/技能树）进行本土化表达，降低阅读门槛

### ch018 新增实例：世界观「翻倍式」扩展——武道→神佛+妖兽+异人三连击

**触发条件**: ch018三连击世界观升维——
① 守冲自曝「很多大德都有死而复生的本事」+自己被怜生菩萨复活（神佛体系存在）
② 水缸粗细/十丈长的漆黑巨蛇出现（妖兽/灵兽体系存在）
③ 游神真人——朝廷异人/走出堂口/「不值得所以不走出来」（异人体系存在）

**与 ch009 对比**:

| 维度 | ch009（讲课式体系建立） | ch018（三连击世界观升维） |
|:----|:----------------------|:-------------------------|
| 信息方式 | 大佬公开讲课——体系化知识从顶层到底层逐层展开 | 剧情自然触发——三个世界观信息分别由三个不同事件展示 |
| 涉及体系 | 武道体系（兵器五等/外功六要素/武功品级/升华机制） | 神佛体系（大德复活/怜生菩萨）+ 妖兽体系（巨蛇灵兽）+ 异人体系（朝廷异人/堂口/香火） |
| 预设颠覆 | 建立「原来武道是这样运转的」认知 | 打破「原来世界只有武道」的预设 |
| 读者情绪 | 世界规则清晰化的掌控感 + 成长路径明确 | 「卧槽还有？」的三次震撼 + 世界突然变大的兴奋 |
| 叙事功能 | 为后续修炼成长提供体系框架 | 为后续剧情提供世界观扩展——更大的世界在等着 |

**关键对白/证据**（新增）:
- 🔒「这天下，死而复生也不是什么稀奇之事，很多大德都有这本事，为师当年也同样经历过一次。」——神佛体系（连击①）
- 🔒「这是自然界能养出来的生物！？」——妖兽体系（连击②）
- 🔒「朝廷异人主动走出堂口，真人这次可是犯了莫大的禁忌。」——异人体系入口（连击③）
- 🔒「从来没有什么禁忌，我等之所以不离开堂口，不踏足红尘，仅仅是因为不值得，也没有必要而已。」——异人体系的傲慢姿态

**核心变化**: ch018不是建立新体系，而是**在已有武道体系基础上「翻倍」**——让读者意识到之前看到的武道世界只是冰山一角。这种翻倍不是靠「讲课」完成的，而是通过**三个独立的剧情事件自然触发**——守冲自曝（对话级）、巨蛇登场（视觉级）、游神真人（人物登场级）。每个事件都同时服务于当前剧情和世界观扩展，避免了专门的「设定课」。

**情绪收益**: 世界不断被打破的震撼感 + 后续发展空间突然变大的期待 + 从「我觉得我懂了」到「我其实什么都不懂」的认知冲击

**下游期待**:
- 怜生菩萨是哪一体系的神佛？还有其他大德吗？
- 妖兽体系还有哪些种类？巨蛇是什么级别？
- 异人体系内部的等级和势力分布？朝廷与异人的关系？
- 武道 vs 修仙 vs 神佛——这三条力量路线之间有什么关系？
