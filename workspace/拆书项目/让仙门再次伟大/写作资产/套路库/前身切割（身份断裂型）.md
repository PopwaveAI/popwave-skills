# 前身切割（身份断裂型）

## 套路概述

**核心机制**：主角主动将前身记忆（或当前身份的记忆）识别为"不属于自己"的内容，并主动献祭/销毁这些记忆，以完成不可逆的身份断裂。

**关键词**：记忆切割 · 身份断裂 · 前身献祭 · 不可逆过渡

## 触发条件

1. **身份识别**：主角必须首先意识到某些记忆/情感/关系不属于真正的自己——通常是穿越者/转世者发现前身的记忆浮现
2. **切割动机**：存在一个外部压力迫使主角做出选择——销毁记忆可以换取生存机会/获得力量/解决危机
3. **不可逆性**：切割行为必须是不可逆的——记忆一旦被销毁/献祭，前身彻底消失

## 执行节奏

```
① 前身记忆浮现（走马灯）→ ② 主角识别非我（"这不是我啊"）
→ ③ 决策切割（"切割！必须切割！"）→ ④ 记忆献祭/销毁
→ ⑤ 前身彻底消失（怨气爆发/能量获取）
→ ⑥ 只剩主角一人——真正的"从此以后只有我"
```

## 适用场景

| 场景类型 | 示例 |
|---------|------|
| 穿越/转生后身份整合 | 主角发现前身记忆影响自己的判断——需要彻底切割 |
| 夺舍/附身体系 | 被夺舍者的残魂威胁主角的神魂——主动销毁以绝后患 |
| 前身因果偿还 | 前身欠下的因果牵扯主角——切割后可以不再为别人的命运买单 |
| 人格整合治疗 | 角色有双重人格或多重记忆需要整合——销毁不必要的人格 |


## 变体

1. **被动型**：主角并非主动切割，而是外力（搜魂/魔头入侵）导致记忆破碎——事后主角坦然接受
2. **仪式型**：有专门的仪式或法门来完成切割（如本章的魔头吞吃记忆）
3. **寄生型**：主角将前身记忆寄存在某个容器/器灵中，让另一个自己承担
