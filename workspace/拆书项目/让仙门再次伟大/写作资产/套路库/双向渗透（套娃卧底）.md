# 套路：双向渗透（套娃卧底）

## 基本信息

| 项目 | 内容 |
|:----|:-----|
| **套路名称** | 双向渗透（套娃卧底） |
| **首次出现章节** | ch003「家里养了鬼都不知道？」 |
| **套路类型** | 结构型 / 悬疑型 |
| **可复用性** | 高（可适用于任何有组织对抗的世界观） |

## 定义与触发条件

**触发条件**：主角是A组织派往B组织的卧底 → 发现B组织在A组织内部也有卧底 → 双方互不知晓对方卧底身份 → 任何一方的行动都可能触发另一方的反制

**核心张力**：卧底发现自己不是唯一的内鬼——你查别人，别人也在查你；双方都在同一张间谍网里

## 首次出现原文证据

- 🔒「有内鬼，终止行动！」——白莲教在县衙的卧底传回消息
- 🔒「不只是白莲教里有自己这个卧底，县衙里也有白莲教的卧底！」
- 🔒 守冲展示纸条 + 刀抵眉心 = 双向渗透的后果爆发

## 情绪收益

- 信息不对称反转爽感（读者以为王平安全了，瞬间又陷绝境）
- 危机感持续升级
- 悬疑谜题（县衙里的白莲教卧底是谁？）

## 变体与扩展方向

| 变体 | 说明 |
|:----|:-----|
| **三方渗透** | 引入第三方势力，形成三向卧底网络 |
| **镜像渗透** | 两边卧底身份同级别、同时对等暴露 |
| **反间计变体** | 利用对方卧底反向传递假情报 |

## 使用注意事项

1. 需要提前铺设「双方都有渗透能力」的世界观合理性
2. 卧底身份揭秘应在中后期，不宜过早完全揭晓
3. 需注意复杂度管理——卧底层级过多会让读者困惑
