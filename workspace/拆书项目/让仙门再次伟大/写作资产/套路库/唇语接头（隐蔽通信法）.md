# 套路：唇语接头（隐蔽通信法）

## 基本信息

| 项目 | 内容 |
|:----|:-----|
| **套路名称** | 唇语接头（隐蔽通信法） |
| **首次出现章节** | ch003「家里养了鬼都不知道？」 |
| **套路类型** | 技巧型 / 操作型 |
| **可复用性** | 中高（需要合适的掩护场景） |

## 定义与触发条件

**触发条件**：需要传递情报但无法使用纸条/信物等传统方式 → 接头人利用环境掩护（如与第三人对话）→ 用唇语/口型传递真实信息 → 旁观者只看到正常对话

**核心张力**：明面一套说辞，暗地一套信息——双重对话的结构本身就有张力

## 首次出现原文证据

- 🔒「别回头。」——接头信号
- 🔒「王平张嘴，一边和铁匠铺老板沟通，一边做着口型，令人惊异的是，口型和说出的话竟是截然不同」
- 🔒「这个技巧他当初可是学了大几个月才学会的」

## 情绪收益

- 技术感/专业感（有训练门槛的技巧、增加角色能力深度）
- 智力操作爽感
- 悬疑氛围（读者知道接头在发生，但旁观者一无所知）

## 变体与扩展方向

| 变体 | 说明 |
|:----|:-----|
| **手语接头** | 用特定手势在人群中传递信息 |
| **咳嗽摩斯码** | 用咳嗽节奏传递编码信息 |
| **双关语接头** | 看似正常的对话，关键词语有约定含义 |
| **物品暗号** | 通过摆放/取用特定物品传递信号 |

## 使用注意事项

1. 需预先建立「角色训练过该技能」的设定（否则会显得突兀）
2. 每次用法的环境掩护需不同，避免重复模板
3. 需设置反制手段（懂唇语的人、近距离、光线不足、面具遮挡等）
4. 频率不宜过高——每3-5章出现一次即可，且需引入风险

## 下游期待（本小说内）

- 唇语接头是否会被第三方识破（距离过近、有懂唇语的人在旁）？
- 后续是否会出现反唇语侦测手段（故意噪音干扰、面具遮挡）
