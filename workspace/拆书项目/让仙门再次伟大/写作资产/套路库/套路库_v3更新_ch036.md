### 卡片1：🎁 赠礼迷惑（先甜后苦型）

**来源**：ch036 - 尤道姑赠送六欲天魔旗

**核心公式**：
```
A方赠送B方重礼[真诚表现] → B方感动/信任上升[「难道她是好人？」] → A方翻脸[显露真实目的] → B方被更深重地伤害[信任崩塌]
```

**效果**：
1. 情感振幅大——从惊喜到愤怒的情绪跨度制造阅读张力
2. 反派塑造深度——不是单纯的恶，是有层次有演技的真实人物
3. 强化读者的代入感——读者跟随主角一起被骗，加深被背叛的共感

**使用条件**：
- 赠礼必须真的有用/珍贵（不能是假货，否则降低反派档次）
- 翻脸前需要有足够中性的善意表现（不能一翻脸就OOC）
- 翻脸后的揭示需要逻辑自洽——不能是临时起意

**变体**：
- 美女陷阱：「对你好→骗你感情→收割你的一切」（美人计变体）
- 师门恩情：师尊宠信→泄露宗门机密→真相是养肥再杀
- 合作骗局：合作伙伴给足利益→核心资产归属权暗藏陷阱

**反制方法**：
- 主角需要预置「防人之心」——如王平的「域外天魔冷却完毕=保底思维」
- 主角需要观察「过度关怀」的不自然——王平「直觉告诉我还是不太对」

**本章应用演示**：
```
尤道姑赠六欲天魔旗[明确说"专门为你炼制的护身法器"] 
  → 王平感动[「难道是我误会了？尤道姑其实是好人？」]
  → 尤道姑带其至静室[用教学遮掩陷阱]
  → 翻脸[「乖乖站好」+移景定身]
  → 真相[「买个人材回来自己炼」]
```

### 卡片2：📖 设定震撼·知识即毒药（功法污染型·变体）

**来源**：ch036 - 【五贼盗天万化神符】玉简诱惑

**核心公式**：
```
设定揭示[知识本身是污染] → 行动限制[不能碰/不能了解] → 逻辑闭环[为何放在明面无人敢取] → 反派实力背书[弃徒以此反向钓鱼]
```

**效果**：
1. 颠覆传统「秘籍争夺战」套路——不是谁抢到谁赢，是谁碰谁死
2. 完美解决「高级功法为何没人抢」的逻辑漏洞
3. 增加世界观的神秘感和危险度——知识不再安全

**使用条件**：
- 需要提前铺垫「某个东西没人敢碰」的异常现象
- 需要有一个「受害者现身说法」（如尤道姑、盗天贼弃徒）
- 污染的具体机制需要逻辑自洽（接触→理解→被影响）

**变体**：
- 诅咒信息：特定文字/图案本身蕴含诅咒，看到即中招
- 模因污染：功法/知识具有模因特性，理解即传播
- 共鸣追踪：功法与特定人物的命格共鸣——了解者会被原主感知

**反制方法**：
- 主角拥有特殊体质/被动免疫（如王平的可能走向）
- 通过间接方式获取（碎片化+不完整理解）
- 在安全环境下由高阶者过滤后再传递

**本章应用演示**：
```
玉简摆在静室正中（无人敢碰的异常）
  → 王平被吸引差点中招（验证危险性）
  → 尤道姑警告「知识本身是剧毒」
  → 揭示弃徒「盗天贼」公开功法的反向钓鱼逻辑
  → 闭环：好东西摆在面前也不能拿
```
