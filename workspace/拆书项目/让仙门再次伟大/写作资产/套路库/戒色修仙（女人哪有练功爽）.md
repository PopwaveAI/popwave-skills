# 戒色修仙（女人哪有练功爽）

**套路类型**：人设反差型 / 黑色幽默
**首次出现**：ch024
**关联套路**：温水煮青蛙、忍辱负重

## 四段式拆解

```
【起】主角身处充满情欲诱惑的场景（美女在侧/青楼/后宫机会），
     读者预期主角会「把持不住」或至少心动。
【承】主角内心OS出现180度反转——
     「至于苏夫人，早就被他抛到九霄云外去了。」
     「开玩笑，女人哪有练功爽？」
【转】用理性/功利性的动机解释自己的克制：
     「陈浩彦手里的武功才是他此行的目标，岂能浪费时间在这种事情上？」
【合】读者获得「反差爽」——传统桥段中男主该做的事被彻底颠覆，
     突出了修仙主角的「力量至上」价值观。
```

## 核心效果
- 打破读者惯性期待，制造意外笑点
- 强化主角「修仙脑」的人设——所有行为以实力增长为第一优先级
- 让主角在「情欲场景」中保持道德制高点（不是不想，而是没空想）

## 复用条件
- 主角是务实型/力量至上型人设
- 场景中出现美色诱惑但主角有更优先的目标
- 正在实施「温水煮青蛙」等需要理性克制的策略

## 变体可能
- **仇女版**：主角有过被女人坑的经历，形成心理阴影
- **佛系版**：主角修的是童子功/无欲功，不能破戒
- **眼光太高版**：「就这？」——不是不近女色，是看不上
