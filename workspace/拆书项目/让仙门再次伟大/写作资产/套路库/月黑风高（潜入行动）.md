# 套路：月黑风高·潜入行动

## 基本信息
- **名称**: 月黑风高·潜入行动
- **类型**: 场景套路 / 行动型
- **首次出现章节**: ch010
- **嵌套型**: 否

## 触发条件
需要进入高警戒区域获取情报/物品/刺杀目标 → 选择夜间行动 → 利用环境（月光/阴影/风声）和特殊技能（匿息/轻功）绕过守卫 → 接近目标

## 核心张力
- **技术细节的沉浸感**: 呼吸、心跳、脚步声的控制+巡逻路线的预判+阴影中的穿行——让读者身临其境
- **关键心理**: "会不会被发现"的悬疑感贯穿全程，每一次擦肩而过都是一个小高潮

## 关键对白/证据
- 🔒「夜风呼啸，大日隐匿，三轮明月高悬，却被乌云遮住了月光，只投射下了一片难以测度的深沉阴影」
- 🔒「呼吸，心跳，乃至脚步，所有声音都无限趋近于无」
- 🔒「他就这样默默地跟着巡逻队伍，身影隐匿在黑暗中，从县衙外走到县衙内，还和好几队巡逻捕快打了照面，却愣是没有一个人察觉到他的存在」

## 情绪收益
**沉浸式紧张 + 技术感爽**：读者通过细腻的感官描写进入"第三视角"的潜入体验

## 成功要素
1. **环境铺垫充分**: 天气、月色、守卫的对话——构建可信的潜入场景
2. **技能展示具体**: 不是模糊的"他潜入了"，而是"呼吸心跳脚步趋近于无"
3. **意外元素**: 巡逻队伍的对话提供世道背景信息，同时增加擦肩而过的紧张感
4. **反转收束**: 纯粹的潜入不够——目标是陷阱才有冲击力（ch010中黄雀在后）

### ch013 新增实例：雨夜追杀战

**参与角色**: 谢石（被追杀方）、刘烨（追杀方）、王平（终结方）
**场景**: 暴雨夜 → 丹凤楼 → 龙兴县街巷

**特征分析**:

本实例不是「潜入」而是「追杀/遭遇战」，但核心场景元素一致——黑暗、大雨、夜间，天气和光线成为战斗的核心要素，而非简单背景。

**雨的使用——从背景到武器**:

| 用途 | 内容 | 原文证据 |
|:----|:-----|:---------|
| **掩体** | 素月莲华刀的飞刀藏在雨幕下，让人难以察觉 | 「飞刀藏在雨幕之下，眨眼即逝，等到众人看到的时候，这一刀已经掠过参与围杀的一位捕快的脖颈」 |
| **画面对比增强** | 刀光/拳印在雨中化作肉眼可见的轨迹，视觉冲击力MAX | 「无形的刀气在雨水的映衬下化作肉眼可见的刀光浪潮」/「拳印穿过雨幕，印出一道显赫轨迹」 |
| **氛围渲染** | 雷声重叠战斗声，雨幕染红——史诗感 | 「撞击声和天上的雷鸣重叠，滚动的气劲在雨幕下炸出了一朵又一朵水花」/「血雾冲天而起，给周围的雨幕染上了一层刺目的红」 |
| **意象闭环** | 从开篇「好大的雨」到收尾「今晚雨大，正好洗刀」——雨→刀→血的意象链完成 | 开篇「好大的雨啊。」→收尾「今晚雨大，正好洗刀。」 |

**与 ch010 对比**:

| 维度 | ch010（月黑风高·潜入） | ch013（雨夜追杀） |
|:----|:----------------------|:-----------------|
| 行动模式 | 主动潜入（匿息、穿行） | 被动追杀/遭遇战 |
| 天气功能 | 乌云遮月提供阴影 | 暴雨作为战斗要素（掩体+衬景+意象） |
| 空间 | 从县衙外→县衙内（室内外结合） | 丹凤楼→县城街道（全室外） |
| 光线 | 月光被乌云遮住，深沉阴影 | 电闪雷鸣提供间歇性光明 |
| 核心情绪 | 潜伏的紧张感 | 追杀的紧迫感 + 斩杀的爽感 |
| 天气与战斗的关系 | 天气为背景（提供阴影） | **天气为战斗工具（雨幕藏刀、雨衬刀光、雨洗刀）** |

**变体说明**:
「月黑风高」的核心不只是「潜入」，而是「利用夜间+恶劣天气作为战斗优势」。ch013 展示了这个套路的另一种可能——不潜入，而是把恶劣天气作为战斗工具。「雨」的三层用法（掩体、画面对比、意象闭环）值得在后续章节中复用或升级——比如在雾中、雪中、沙暴中战斗。
