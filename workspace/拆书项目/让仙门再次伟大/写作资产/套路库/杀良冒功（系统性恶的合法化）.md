# 套路：杀良冒功（系统性恶的合法化）

## 基本信息

| 项目 | 内容 |
|:----|:-----|
| **套路名称** | 杀良冒功（系统性恶的合法化） |
| **首次出现章节** | ch004「太平经」 |
| **套路类型** | 世界观型 / 社会批判型 |
| **可复用性** | 低（全卷1-2次足够，过度使用会让世界观显得过度黑暗） |

## 定义与触发条件

**触发条件**：官方/执法力量为完成任务指标或赏金 → 杀平民冒充贼寇 → 将屠杀合法化/合理化 → 展示体系运作的本质

**核心张力**：系统性的恶不是个别人的恶念，而是整个体制运转的逻辑产物——每个人的理性选择累积成非理性的残酷结果

## 首次出现原文证据

- 🔒「都杀了。」——陈皓彦三字令
- 🔒「我们这里有一个白莲教，隔壁县还有一个罗生教，这个世道，反贼是杀不完的」「是为了让县城里的豪族，士子，让所有人都相信我们正在保境安民」
- 🔒「人命关天，指的是那些住在县城里的人……至于城外的那些黔首……他们也算人？」
- 🔒「于公于私，都是大节无亏，至于其他，正所谓小节不拘，小事也无需挂怀。」

## 情绪收益

- 世界观深度（对封建社会本质的真实刻画）
- 认知冲击（读者被迫思考「善恶的边界由谁来定义」）
- 对主角处境的强化同情（主角也是同样的黔首出身）

## 变体与扩展方向

| 变体 | 说明 |
|:----|:-----|
| **战争屠城** | 军队杀平民冒充敌军首级领赏——历史原型更残酷 |
| **宗教审判变体** | 宗教组织以「清除异端」名义屠杀非信众 |
| **官僚KPI变体** | 现代版本——基层为完成考核指标造假数据 |

## 使用注意事项

1. 必须有道德代言人（如本章的林晟）在现场提出异议，否则变成单纯卖惨
2. 合理化理论必须有至少一定逻辑——完全无理的反派反而削弱深度
3. 不宜在同卷内重复展示（失去新鲜感），但可作为闪回/他人口述复用
4. 杀良冒功的受害者最好与主角有关联（强化共情）

## 下游期待（本小说内）
