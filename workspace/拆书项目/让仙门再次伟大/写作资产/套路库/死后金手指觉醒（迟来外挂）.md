# 套路：死后金手指觉醒（迟来外挂）

## 元信息
- **套路名称**: 死后金手指觉醒（迟来外挂）
- **首次出现章节**: ch005
- **所属母套路**: 穿越者苟且生存（隐忍发展型）的续集
- **复用性**: 中——特定于外挂首次激活场景，但「延迟满足」哲学可迁移至其他情境

## 触发条件
主角死亡 → 进入太平经内部空间 → 金手指(太平经)正式激活

## 套路公式
```
主角长期无外挂（铺垫）→ 死亡（触发）→ 金手指激活（满足）→ 
但金手指有重大限制（吊胃口）→ 短暂满足后进入新的困境（延续动力）
```

## 核心张力
穿越二十年后终于等到外挂，但代价是已经死过一次；外挂虽强但冷却太长，属于「有但不够用」的吊胃口设定。

## 情绪收益
迟到满足感 + 期待感（金手指终于来了） + 不完美的遗憾（七天冷却、只有一次）+ 金手指持续扩展的成长感
