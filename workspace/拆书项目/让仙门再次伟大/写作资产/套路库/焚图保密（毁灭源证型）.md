# 套路：焚图保密（毁灭源证型）

## 元信息

- **套路名称**: 焚图保密（毁灭源证型）
- **首次出现章节**: ch026
- **所属母套路**: 风险管理类套路
- **复用性**: 高——任何「获得秘密/秘籍后需防追踪」的场景均可复用

## 触发条件

角色获得某种机密信息（武功秘籍、地图、情报等）的实体载体 → 角色判断携带实体载体有风险（被搜查、被跟踪、被抢夺）→ 角色将实体载体彻底销毁，仅依靠自身能力（灵识/记忆/背诵）保存内容

## 套路公式

```
获得机密载体（秘籍/图纸/信件）
    ↓
判断「保留实体=风险」
    ↓
彻底销毁实体（焚烧/粉碎/溶解）
    ↓
仅凭自身能力保留内容
    ↓
「这才是最保险的」
```

## 核心张力

**谨慎 x 决断力** — 主角在获得宝贵信息后，不贪不留，瞬间做出销毁的决定，展现其理性与克制。读者会经历「心疼」（这么好的秘籍烧了？）→「理解」（对，这样才安全）→「敬佩」（这主角够稳）的情绪曲线。

## 关键证据

- ch026: 「这就是陈浩彦隐藏最深的秘密，此刻却被他毫不犹豫地烧毁，当然，所有内容都被他用灵识收录了。」
- ch026: 「『这才是最保险的。』」

## 情绪收益

- 主角形象加分（谨慎、果断、不贪小利）
- 悬念张力（秘密内容只有主角知道了 → 后续任何相关事件只有主角能解密）
- 降低读者对「实体物会落入他人之手」的焦虑

## 变体方向

| 变体方向 | 示例 |
|---------|------|
| **多人合谋变体** | 多人各持碎片，各自销毁自己的部分，只有集合全部记忆才能还原 |
| **被骗销毁变体** | 反派诱导主角销毁重要文件，主角后来发现被骗——销毁变成了错误 |
| **修真记忆变体** | 不是普通记忆，而是用灵识/神识烙印，记忆本身就是载体——「我就是秘籍」 |

## 使用注意事项

1. 销毁前让读者看到内容——否则销毁没有价值，读者只会觉得「你倒是告诉我写了什么啊」
2. 销毁后要为「主角如何运用这个秘密」留空间——否则就成了「知道了也白知道」
3. 可以制造「他人以为主角已丢失秘密」的信息差——他人以为主角没有 = 主角可以在关键时刻使用

## 下游期待

- 有人搜索/搜查主角住处时，找不到任何证据（「果然搜不到，因为他烧了」）
- 反派知道有秘籍存在，但不知道已被销毁 × 主角已经掌握内容
- 可与其他信息差套路联用——「你以为我没有 = 其实我有」
