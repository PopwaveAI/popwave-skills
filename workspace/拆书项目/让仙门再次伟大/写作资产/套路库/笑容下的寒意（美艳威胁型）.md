# 笑容下的寒意（美艳威胁型）

**套路类型**：人物塑造型 / 反差式恐怖
**首次出现**：ch035
**关联套路**：反派内心戏、黑色幽默、双面人设

## 四段式拆解

```
【起】高位角色（上级/导师/长辈）对主角展现
     出反常的温柔/满意——「露出了一抹倾倒众生的美艳笑容」
     ——氛围瞬间从紧张/冷漠转为「温暖」
【承】用正面词汇包裹威胁——
     「给你一个惊喜」「让你的价值充分实现」
     ——没有一个字是负面词汇
【转】在关键位置插入一个特定的「暗语词」——
     这个词本身无害，但在故事语境中有恐怖含义
     （如「人牲价值」中的「牲」字）
【合】主角的脑海里闪过相关记忆/既视感——
     从「被表扬」到「被盯上」的认知转变
     主角后背发凉，读者同步感受到寒意
```

## 核心效果
- 用「美」和「善」的表象包装「恶」——比直接的威胁更令人不寒而栗
- 没有直接的暴力或愤怒——但让读者和主角一起感受「被掌控」的恐惧
- 利用信息差——读者和主角一样不知道「惊喜」是什么，但知道绝对不是好事
- 留下长期悬念——读者会持续期待这个「惊喜」的兑现

## 复用条件
- 存在一个对主角有生杀大权的高位角色
- 该角色表面上对主角「很好」（培养、奖励、提携）
- 需要在和平表象下埋藏危机的暗线

## 变体可能
- **长辈关怀版**：师父慈祥微笑——「好徒儿，让为师看看你的修为」——背后藏着夺舍/转生的意图
- **上级嘉奖版**：皇帝/宗主赞扬——「爱卿果然不负朕望」——背后是监视/利用
- **恋人温柔版**：恋人甜蜜一笑——「你对我真好」——背后是算计/利用
- **同伴信赖版**：同伴拍肩微笑——「有你真好」——背后是背刺/出卖的前奏
