# 套路：过河拆桥（卧底抛弃）

## 基本信息

| 项目 | 内容 |
|:----|:-----|
| **套路名称** | 过河拆桥（卧底抛弃） |
| **首次出现章节** | ch004「太平经」 |
| **套路类型** | 身份型 / 情绪驱动型 |
| **可复用性** | 中（适度使用增强世界观真实感，过度使用削弱信任感） |

## 定义与触发条件

**触发条件**：卧底完成/即将完成任务 → 上司为自保/利益否认卧底身份 → 卧底被抛弃 → 死于自己人之手

**核心张力**：你以为是战友的人，亲手杀了你——最大的背叛来自最信任的方向

## 首次出现原文证据

- 🔒「什么卧底？县衙从来没有往白莲教安插什么卧底」——陈皓彦当众否认王平身份
- 🔒「总捕大人，刚刚有一个反贼说是县衙的卧底……」「只是一个反贼想要活命，信口开河」
- 🔒「至于那个卧底……区区乡野出生的黔首，性命比尘埃还贱，死了也就死了，反正他又不可能活过来找自己复仇」——陈皓彦内心独白

## 情绪收益

- 强烈的愤怒/憋屈驱动（为后续复仇线建立情感锚）
- 对体制黑暗面的认知震撼
- 对主角命运的同情 + 复仇期待的累积

## 变体与扩展方向

| 变体 | 说明 |
|:----|:-----|
| **顶级弃子** | 身居高位的卧底被无情抛弃，筹码越大落差越大 |
| **多重弃子** | 卧底牺牲后发现自己不是第一个被抛弃的——同类悲剧多处 |
| **主动跳反** | 被抛弃的卧底如果存活，可能直接黑化跳反到对面阵营 |

## 使用注意事项

1. 抛弃方的理由必须合理（利益）、不能是纯粹的脸谱化恶（会降低故事可信度）
2. 被抛弃的卧底需要有后续利用价值（复仇线/揭露线），否则只是虐主
3. 建议与「复活转生」或「绝境反杀」搭配使用，避免主角长期处于憋屈状态
4. 抛弃方如果位高权重，后续可以是全卷级反派

## 下游期待（本小说内）

- 陈皓彦是王平的第一复仇目标——复活后如何寻找时机报复
- 是否还有其他被陈皓彦抛弃的卧底（潜在盟友或受害者家属）
- 王平未来成为上位者后是否面临类似选择——这是检验他是否黑化的测试
