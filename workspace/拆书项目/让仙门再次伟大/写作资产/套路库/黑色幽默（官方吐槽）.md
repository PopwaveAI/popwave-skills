# 套路：黑色幽默·官方吐槽

## 基本信息
- **名称**: 黑色幽默·官方吐槽
- **类型**: 母套路 / 语言风格型
- **首次出现章节**: ch003（雏形）→ ch010（成熟）
- **嵌套型**: 否（语言风格层套路，可叠加在任何场景上）

## 触发条件
角色用一本正经的态度/冠冕堂皇的理由 → 包装实际上low/尴尬/充满槽点的行为 → 形成"正经过头"和"内容低俗"之间的强烈反差

## 核心张力
- **读者知道真相，角色假装不知道**: 主角嘴里说"洁身自好"，手上已经在拿纸巾——读者爆笑
- **关键心理**: 全知视角+反差笑点——读者有"我懂你在装"的智力优越感

## 关键对白/证据
- 🔒「归根结底，还是我本人洁身自好，绝非好色之徒，岂能无缘无故去那种乌烟瘴气的地方作践自己？」（言罢拿纸巾）
- 🔒前一秒刘爷后一秒刘哥——「翻脸不认人的速度比自己这个老江湖都快」
- 🔒「守冲的赏银都特么没有这么多！」——用武侠世界的赏金吐槽物价

## 情绪收益
**幽默爆笑 + 角色亲和力**：让严肃/紧张的剧情间隙插入轻松笑意，缓解阅读疲劳，同时强化主角的"俗人"亲近感

## 使用要点
1. **不能滥用**: 每章最多1-2次，否则会稀释效果
2. **黄金位置**: 紧张场面结尾/高潮前后的间隙使用效果最佳
3. **语言风格**: 现代感吐槽的"翻译腔"要和古代背景形成错位喜感
4. **搭配场景**: 最佳搭配——主角做双标/自我欺骗的事情时（嘴上说一套，手上做一套）

## 变体
| 变体 | 说明 | 示例 |
|------|------|------|
| 双标高帽型 | 嘴上宏大叙事，实际自私算计 | ch010"洁身自好" |
| 社会死亡型 | 被当场拆穿后的尴尬补救 | — |
| 翻脸型 | 称呼随利益变化（刘爷→刘哥→刘爷） | ch010 |
| 打工人加钱型 | 面对无法拒绝的任务以加钱来保留尊严 | ch014「得加钱！」 |

### ch014 变体扩展：打工人加钱型

**触发条件**: 上级以不可拒绝的方式布置必死任务 → 主角内心清楚无选择余地 → 但以讨价还价的方式（「得加钱！」）回应 → 在服从框架内保留反制姿态

**核心张力**: 与ch010的「双标高帽型」不同——ch014的黑色幽默来自于「知道无法拒绝，但还要皮一下」的反抗姿态。王平从咬牙切齿到突然冒出「得加钱」，是在极度压迫下（三天+九死一生）用现代打工人的槽点来解构严肃的生死抉择。这种幽默的效果来自：读者完全理解王平面临的是必死的陷阱——所以他的「得加钱」不是真正的讨价还价，而是一种「我知道我被坑了，但我面对你不能显得太好欺负」的姿态。

**关键对白/证据**:
- 🔒「练了，就是九死一生，不练，就是十死无生，该怎么选你应该明白。」（陈浩彦——不给选择的压迫）
- 🔒「这个任务属下可以接，但是.....得加钱！」（王平——以加钱保留尊严的反抗）

**情绪收益**: 对王平「即便被压制也不示弱」的认同感 + 用裁员梗/打工人梗带入现代语境的跨越时代的幽默

### ch017 变体扩展：上位者双标·当面拆台型

**触发条件**: 守冲洗白自己要杀捕快替谢石报仇（「算是替他报仇，也算成全了我和他师徒间的情义」）→ 教众当面吐槽「教主您可谓是天底下第一薄情寡义之人了」→ 守冲义正言辞反驳「为师乃是顶天立地的好男儿，素来重情重义」→ 教众再怼「谢石没练出内劲，不是因为师傅你不传真功吗？」→ 守冲表情顿时一僵 → 阴恻恻反驳 → 自曝传功真相

**核心张力**: 与之前的黑色幽默变体（双标高帽型/翻脸型/打工人加钱型）不同——ch017的变体是 **「上位者双标 vs 下属当面拆台」** 。守冲作为教主，他的双标是「一本正经地自我美化」（我要替谢石报仇→重情重义），但教众作为底层的吐槽是「当场打脸」（你薄情寡义还装什么）。这种「教主被自己人拆台」的反差，在传统严肃小说中是「犯上」——但这里的处理是轻松吐槽，守冲的反应也不是暴怒而是无奈辩解，进一步强化了喜剧效果。

**关键特点**:
- **被拆台者**是反派而非主角——这是该套路首次应用在反派身上
- **吐槽者**是反派的下属——教众可以当面吐槽教主，说明守冲的双标已经到了「全组织都知道」的程度
- **对话节奏**：守冲自夸 → 教众拆台 → 守冲反驳 → 教众灵魂拷问 → 守冲表情一僵 → 阴恻恻解释 → 自曝真相——攻防转换节奏极快

**关键对白/证据**:
- 🔒「为师乃是顶天立地的好男儿，素来重情重义」——双标自夸
- 🔒「咱们都知道，教主您可谓是天底下第一薄情寡义之人了。」——教众当面吐槽（力度最大的一句）
- 🔒「怎么，不说话了？」/「不敢说。」/「放心，为师向来闻过则喜。」/「那我们就说了。」——对话节奏制造幽默
- 🔒「谢石师兄没有练出内劲，不是因为师傅你不传真功吗？」——灵魂拷问
- 🔒「守冲表情顿时一僵。」——被打脸瞬间的微表情

**情绪收益**: 反派被自己人拆台的爆笑效果 + 颠覆「教主威严」的松弛感 + 守冲这个反派被进一步立体化 + 「即使反派也有喜剧时刻」的趣味性

**复用提示**:
- 这种变体的关键在于拆台者必须是与被拆台者有长期关系的「自己人」——外人拆台只是冲突，自己人拆台才是喜剧
- 守冲的「闻过则喜」是点睛之笔——嘴上说欢迎批评，实际被批评后立刻黑脸，反差拉满
- 与ch010的王平双标不同——ch017的黑色幽默更「轻松」，因为读者对反派没有代入感，可以纯粹欣赏喜剧效果
