## 2. 爽点设计（爽点层）

### 2.1 本章套路
以**死后金手指觉醒**切入，经历**虚假希望（修仙世界）→残酷真相（成为力士傀儡）** 的反差打击，转入**精神控制的生存博弈**。属于「金手指觉醒 + 世界观跳转 + 底层挣扎」复合型套路。王平从死到生、从人到工具，在极度不利的境地下确立七天生死赌局。

### 2.2 情绪弧线（起点→推进→高潮→终点）
```
起点：茫然（死后在太平间苏醒，面对未知金手指）
  ↓
推进①：狂喜（发现金手指可复活，穿越者最大梦想成真）
  ↓
推进②：纠结（选择困难：原地复活 vs 随机穿越）
  ↓
推进③：希望（穿越后听到修仙世界对话——赌对了！）
  ↓
推进④：落差（发现自己是力士傀儡，被烙思想钢印）
  ↓
高潮：惊悚（差点被洗脑成忠仆——人牲陷阱的危机感）
  ↓
终点：紧迫（确立七天生死倒计时，必须苟住）
```

| 爽点 | 触发节点 | 机制说明 | 强度 |

| 随机赌命 | 事件5-6（弃原地复活→赌随机穿越） | 以命为注的冒险抉择，代入感强 | L3 |

### 4.2 动作场景（本章无战斗，但有动作节点）

1. **查看书册**：缓步向前，目光落在字迹上一动不动——大量讯息强制性灌入脑海的被动体验
2. **穿越推门**：咬牙跺脚转悠半小时→大步向前推门而入——犹豫后的决断动作，有画面张力
3. **穿越过程**：金光淹没视野→微风拂面→如泡温泉→五感逐渐消失——慢镜头式的抽象感官描写
4. **思想钢印烙刻**：清凉感从头顶蔓延向四肢百骸——被动的、不可抗拒的精神入侵
5. **人格挣扎**：身心逐渐被「感恩」念头同化→猛然一哆嗦挣脱→头皮发麻——从顺从到惊醒的生理反应链

## 5. 套路归档

检测到以下套路，均在修仙宗门+穿越者框架内运行：

### 套路 1：死后金手指觉醒

**名称**: 死后金手指觉醒（迟来外挂）
**嵌套型**: 是（嵌套在「穿越者二十年苟且」的母结构续集中）
**所在章节**: ch005

**触发条件**: 主角死亡 → 进入太平经内部空间 → 金手指(太平经)正式激活
**核心张力**: 穿越二十年后终于等到外挂，但代价是已经死过一次；外挂虽强但冷却太长，属于「有但不够用」的吊胃口设定
**关键对白/证据**:
- 🔒「合著我的金手指要等死了才能激活，也不给我点提示。」
- 🔒「【卷一：域外天魔】」「死后复活一次，冷却时间七天」
- 🔒「穿越至今二十年，庸碌无为了半辈子」
**情绪收益**: 迟到满足感 + 期待感（金手指终于来了）+ 不完美的遗憾（七天冷却）

**下游期待**:
- 域外天魔冷却结束后如何使用第一次复活
- 太平经收集到不同物品后会开出什么经卷

### 套路 2：希望→落差双重打击

**名称**: 希望→落差双重打击（虚假希望法）
**嵌套型**: 是
**所在章节**: ch005

**触发条件**: 主角穿越到修仙世界狂喜 → 发现自己是力士傀儡（工具/奴仆）→ 希望破灭 → 面临精神控制危机
**核心张力**: 先给读者最高期待（修仙世界）→再狠狠摔碎（沦为工具人）→才发现处境比想象中更可怕（人格洗脑）
**关键对白/证据**:
- 🔒「修仙世界？王平瞬间激动，毕竟谁没有一个修炼成仙，长生不老的梦？」
- 🔒「终于炼制成功了，这一批的【力士】成功出炉，可以拿去交货了」→ 幻想被残忍打破
- 🔒「我是，护法院的【力士】」「仙门特地炼制出来服务一众弟子的傀儡，奴仆」
**情绪收益**: 强烈情绪反差 + 对主角处境的深度共情 + 对仙门黑暗面的好奇

**下游期待**:
- 力士在仙门的日常待遇到底有多惨
- 这个仙门到底是什么性质的势力（正/邪/中间态）
- 其他力士是否也有独立意识？还是王平是唯一的特例

### 套路 3：思想钢印/精神控制人格战

**名称**: 思想钢印（精神控制对抗）
**嵌套型**: 是
**所在章节**: ch005

**触发条件**: 作为力士被烙印 → 清凉感中植入效忠人格 → 穿越者原有人格与其冲突 → 在感恩与人牲之间挣扎
**核心张力**: 这不是肉体威胁，而是人格层面的慢性自杀——主角每时每刻都在被同化成忠仆，清醒与沉沦拉锯
**关键对白/证据**:
- 🔒「头顶传来一股清凉感……竟是突然生出了要一辈子效忠仙门的念头」
- 🔒「所有力士在刚出炉的时候都是一片空白……这种清凉感会直接生成他们的人格，甚至形成思想钢印」
- 🔒「我是仙门一块砖，哪里需要往哪……操！」
- 🔒「只要我还是【力士】，精神就会持续被其影响」
**情绪收益**: 精神恐怖 + 紧迫感 + 对主角意志力的钦佩

**下游期待**:
- 王平能抵抗钢印多久？是否会有被同化的临界时刻？
- 是否会有其他力士作为「已被同化」的对照样例出现？
- 太平经或未来经卷是否提供对抗精神控制的手段？

### 套路 4：品质递增收集体系

**名称**: 品质递增收集体系（逐级兑换法）
**嵌套型**: 是
**所在章节**: ch005

**触发条件**: 需要增加太平经卷数量 → 揭示收集机制 → 物品品质必须逐级递增
**核心张力**: 每件收集物品的品质必须高于上次，导致收集策略必须精打细算——不能贪快，必须从最低级开始稳步爬升
**关键对白/证据**:
- 🔒「俗世凡物，地蕴精粹，天养神华，宇内奇珍，唯一真品」
- 🔒「收集以上类型的物件……每收集一件可添一道经卷，每次收集的物件品质必须高于上次」
- 🔒「利用率最高的方法，就是从【俗世凡物】开始，一步步慢慢提升」
**情绪收益**: 策略性 + 成长期待 + 收集癖满足

**下游期待**:
- 王平在当前力士的处境下如何获取最低品级的【俗世凡物】
- 每次收集的间隔和难度会如何递增
- 鉴定功能在剧情中的实际用途
