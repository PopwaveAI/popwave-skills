## 一、事件链（骨架层）

### E1｜库房苏醒，身份困境
- **内容**: 王平随金色光晕散去后发现自己身处库房，周围有十具青铜人像（力士），他是其中之一
- **原文证据**: 「入目所见赫然是一间库房。库房之中，灯火幽幽。空地上摆放着足足十具栩栩如生的青铜人像，而王平就站在他们之中，如同混在了狼群中的哈士奇。」
- **🔒 关键对白**: 『不妙啊。』

### E2｜力士体系认知——有意识即死罪
- **内容**: 回顾穿越记忆，得知力士不允许拥有意识，被发现独立人格会灭顶之灾；必须在【域外天魔】恢复前绝对隐藏
- **原文证据**: 「在仙门，【力士】是不被允许拥有意识的，只掌握必要的知识，一切行动完全遵照既定的指令。」
- **🔒 关键对白**: 『换而言之，【力士】就是一种人形的死物，我现在这种具备独立人格和思考能力的情况无疑是有问题的，一旦被发现，恐怕立刻就是灭顶之灾，所以在【域外天魔】恢复前我绝对不能暴露。』

### E3｜发现【气禁】法术
- **内容**: 从力士记忆中找到一门完整法术【气禁】，可提炼真气，止血生肌、续骨连筋、刀枪不入；心痒难耐但环境不允许演练
- **原文证据**: 「哪怕是最低级的力士，也不是呆板死硬的傀儡，在力士的记忆中，王平竟然找到了一门完整的法术！法术名为【气禁】。」
- **🔒 关键对白**: 『真的有！』 / 『无妨，反正我如今是修炼工具，而且好像已经被人订购了，到时候肯定不愁没有演练法术的机会。』

### E4｜库门开启——尤道姑登场
- **内容**: 巨响后大门打开，刺眼光芒映出道观景象；三道人影——矮胖弟子、马脸弟子、尤道姑（尤师姐）
- **原文证据**: 「「咣当！」是大门开启的声音。」 / 「那是一位道姑，一身蓝白相间的道袍，及腰长发，趁着夜色，在月下仿佛宛若都会乘风而去的仙子。」
- **🔒 关键对白**: 「尤师姐，这是你要的十具【力士】。」

### E5｜邪念突生→强行压制
- **内容**: 王平看到尤道姑的瞬间本能生出欲望和蹂躏幻想，意识到是力士体系遗留的精神影响，强行将心神集中回力士记忆
- **原文证据**: 「王平自问不是一个好色的人，然而在看到这位道姑的瞬间，心底却没来由地生出了几分欲望，几乎就要出于本能的，在脑海中构建出将她打落云端，按在地上，撕开那身出尘气质的蹂躏景象。」
- **🔒 关键对白**: 『.....不对！』

### E6｜尤道姑审视——惊险过关
- **内容**: 尤道姑察觉异样锁定仓库方向，视线有「重量」感；王平全力维持心神，最终未被看穿
- **原文证据**: 「原本正在和两位送货弟子交谈的道姑也突然绣眉一蹙，转过身，视线随之锁定在了仓库的方向。」
- **🔒 关键对白**: 「怎么了？难道这一批【力士】不合你的心意？」 / 「......师弟误会了。」「【力士】的品质相当不错，用来修炼【斩仙葫芦真解】已是绰绰有余了。」

### E7｜【斩仙葫芦真解】信息揭露
- **内容**: 得知【斩仙葫芦真解】是从【三道，八法，七十二术】中拆出的八法之一，需要力士充当「磨剑石」（类似双修）
- **原文证据**: 「仙门治下，各类法门数以百万计，却无一列外，都是从【三道，八法，七十二术】中拆解出来的，其中【七十二术】不做限制，人人可学，但是【三道八法】却只有真传弟子才有资格阅览。」 / 『而【斩仙葫芦真解】就是八法之一。』
- **🔒 关键对白**: 『有点像是双修。』 / 『日尼玛，居然连平日的保养都要工具自己动？』

### E8｜修炼开始——真相揭露：采阳补阴
- **内容**: 尤道姑运功，力场扩散包裹王平；真相——磨剑石靠力士自身修为反哺修者，是精神入侵+修为吞噬
- **原文证据**: 「运功开始。」「四周的空气为之一滞。紧接着，一道无形力场立刻以尤道姑为中心轰然扩散开来，将他彻底包裹在内！」 / 「所谓用【力士】做磨剑石，就是以力士来修者运功，造成彼此气息共鸣，然后凭借自身精神力入侵力士体内，以力士的自身修为作为养料，来反哺修者自身！」
- **🔒 关键对白**: 「卧槽！」 / 『妈的！原来是真的被当成牛马了啊！』 / 『没有耕坏的田，只有累死的牛！』 / 『这他妈哪是什么磨剑石，分明就是采阳补阴，一次性的充电宝啊！』

### E9｜精神侵蚀——发现唯一活路
- **内容**: 被精神入侵，痛楚中意识到穿越者精神力远超常人暂时撑得住，但最终会被榨干；发现力士记忆中监控持有者的禁忌术法【真灵傀儡术】
- **原文证据**: 「剧烈的痛楚，撕裂身体的蹂躏感，王平只能咬牙硬撑」 / 「三道八法，不能修炼。但是七十二术是可以的。而七十二术中，就有一门法门，乃是天下一切仙人道士的克星。」
- **🔒 关键对白**: 『必须想办法反抗！』 / 『但以我现在的实力，反抗就是死路一条。』 / 『所以，留给我的路只有一条！』

### E10｜冒险修炼【真灵傀儡术】
- **内容**: 尤道姑精神力已入侵王平身体导致无法停下；王平利用力士权限操控修炼，反向吸收精神力量，修炼真灵傀儡术
- **原文证据**: 「要么，彻底沉沦，成为尤道姑修炼的养料。要么，铤而走险，拼死一搏。」 / 「操控力士进行修炼，不断吸收着场域内的精神力量，将它们转化为自己成长的养分。」
- **🔒 关键对白**: 「要么，彻底沉沦……要么，铤而走险。」

### E11｜反杀——真灵傀儡术成功
- **内容**: 王平成功修炼真灵傀儡术，反客为主，在尤道姑体内植入傀儡术；尤道姑瘫软在地，从修炼者变成被控制者
- **原文证据**: 「直到片刻之后，王平猛然睁开双眼。但见修炼室中，尤道姑已然瘫软在地，浑身香汗淋漓，面色潮红，眼神迷离」
- **🔒 关键对白**: 「你...你到底是什么东西！？」 / 「我是你的新主人，我已经通过你的力量，在你体内植入了《真灵傀儡术》。从今天起，我生，你生，我死，你也死。」

### E12｜惩罚立威 + 收奴完成
- **内容**: 王平用傀儡术惩罚尤道姑，尤道姑彻底屈服称主人；王平获得第一个可靠奴仆，终结牛马生活
- **原文证据**: 「王平的意念一动，尤道姑的面色顿时就变得无比古怪，犹如吞了什么脏东西一样，脸色铁青。」 / 「「够，够了！我都听你的！我都听你的！」」
- **🔒 关键对白**: 「主人！主人！求求你放过我吧！以后你就是我的主人，我就是你的奴隶，你让我做什么都行！」 / «——他娘的，终于可以结束这该死的牛马生活了！»

### 套路识别

| 套路 | 类型 | 本第出现位置 | 功能 |

| **思想钢印（精神控制对抗）** | 母套路（延续） | E5（邪念干扰） | 力士体系的钢印效果表现为「非内源性欲望」——王平对尤道姑的邪念实为力士体系的残留影响 |

## 四、感官与画面（感官层）

### 环境基线（≥4维）

| 维度 | 描写 | 原文证据 |
|------|------|---------|
| **视觉** | 库房-灯火幽幽-青铜人像；道观-旭日骄阳-绿野仙踪-祥云朵朵-遍地流光；尤道姑-蓝白道袍-及腰长发-月下仙子 | 「库房之中，灯火幽幽」/「头顶是旭日骄阳，脚下是绿野仙踪，入目所见尽是鸟语花香，祥云朵朵，遍地流光」/「一身蓝白相间的道袍，及腰长发，趁着夜色，在月下仿佛宛若都会乘风而去的仙子」 |
