## 2. 爽点设计（爽点层）

### 2.1 本章套路
以**金手指新功能解锁**（身份兑换体系）为核心驱动，辅以**残缺功法获取**（欲界天生法/阎罗三魔刀）和**身份跃升**（流民→捕快），再穿插**升维对比**（乡野vs县城）和**信息差爽感**（读者同步看到守冲损失气运，王平不知）来强化阅读体验。章末以**师徒互坑**（守冲让谢石强闯县衙）制造悬念钩子。

### 2.2 情绪弧线（起点→推进→高潮→终点）
```
起点：惊喜（太平间苏醒，意外获得经文秘法）
  ↓
推进①：兴奋（欲界天生法效果——可凝聚灵识，好货！）
  ↓
推进②：务实（决定先回大顺，练功为主）
  ↓
推进③：期待（发现域外天魔新功能——可换身份！）
  ↓
推进④：好奇（身份兑换后定格为捕快——命运大反转）
  ↓
推进⑤：惊喜（回到龙兴县看到武功秘籍和捕快福利）
  ↓
高潮：对比震撼（县城捕快吃沃焦米妖兽肉——乡野生活简直吃屎）
  ↓
终点①：自嘲笃定（开挂才是硬道理）
  ↓
终点②：悬念悬起（交织支线——守冲吐血，要谢石强闯县衙）
```

### 4.2 动作场景

1. **经文灌入**：猛然捂头，瞳孔剧震——被动的信息涌入体验，视觉/听觉冲击
2. **舞刀试法**：铿锵一声拔刀出鞘→随手挥舞牛尾刀→分析刀法优劣——从兴奋到冷静分析的过程动作链
3. **开门接饭**：持刀警惕推门→小厮点头哈腰递饭盒→接过饭盒→关门——从戒备到放松的微动作
4. **吃饭场景**：怒从心头起→抓起勺子狠狠挖饭→塞进嘴里→真香——从愤慨到真香的反差动作链（未来可补充详细描写）
5. **守冲吐血**：眼角抽搐→赤红脸色→仰面朝天喷血→面色瞬间转白——从隐忍到爆发的生理表现链
6. **师徒对话**：守冲歪头看谢石→谢石面露敬佩→被告知真相→谢石？？？——从期待到懵逼的微表情变化

## 5. 套路归档

检测到以下套路，均在金手指解锁+身份跃升+世界观回归框架内运行：

### 套路 1：死后金手指觉醒（续）

**名称**: 死后金手指觉醒（迟来外挂）
**嵌套型**: 是（域外天魔新功能解锁——身份兑换体系）
**所在章节**: ch008（续 ch005）

**触发条件**: 王平在太平间决定返回大顺王朝 → 太平经域外天魔浮现新字迹 → 身份/气运/兑换功能首次亮相
**核心张力**: 金手指不是一次性给出，而是在不同情境下逐步解锁新功能——每次解锁都带来全新可能性。这次身份兑换体系让王平可以从底层流民跃升为官方身份，且消耗的是别人的气运（白莲教），爽感加倍
**关键对白/证据**:
- 🔒「当前身份：流民（白身）。当前气运：一（白莲教堂主）」
- 🔒「身份有六个等级。白身，皂吏，青士，朱服，紫卿，金爵」
- 🔒「还记得这个功能能在特定世界用」
**情绪收益**: 金手指持续成长感 + 身份跃迁爽感 + 用敌人气运不心疼的快感

**下游期待**:
- 身份还能继续兑换到更高等级吗？（皂吏→青士→朱服...）
- 气运如何再次获取？是否可以通过太平经的其他功能积攒气运？
- 身份系统的其他功能（除兑换外）是否还有未揭示的机制？

### 套路 2：品质递增收集体系（续）

**名称**: 品质递增收集体系（逐级兑换法）
**嵌套型**: 是（身份等级体系是收集体系在身份维度的映射）
**所在章节**: ch008（续 ch005）

**触发条件**: 域外天魔显示身份等级——白身→皂吏→青士→朱服→紫卿→金爵，消耗气运可兑换对应等级
**核心张力**: 与太平经的物品收集品级体系（俗世凡物→地蕴精粹→天养神华→宇内奇珍→唯一真品）形成平行映射——身份也有逐级提升的阶梯，每次消耗对应等级的气运
**关键对白/证据**:
- 🔒「身份有六个等级。白身，皂吏，青士，朱服，紫卿，金爵」
- 🔒「我的气运每高一级，就可以兑换与之对应等级的身份」
**情绪收益**: 体系感 + 成长路径清晰 + 策略性选择

**下游期待**:
- 不同等级身份在剧情中有什么实际好处？
- 气运的获取方式——是否与主角的行为/经历/功绩直接挂钩？
- 金爵身份是什么级别的存在？

### 套路 3：身份跃升（阶级跨越型爽点）

**名称**: 身份跃升（阶级跨越法）
**嵌套型**: 否（新套路）
**所在章节**: ch008

**触发条件**: 流民（白身）→ 消耗白莲教气运 → 皂吏级身份 → 随机定格为捕快 → 从被斩杀者变成斩杀者
**核心张力**: 主角从最底层（流民/人牲/被杀者）一跃成为体制内（捕快），且是穿越回同一地点（龙兴县）——命运反转的宿命感极强。更妙的是消耗的是敌人的气运，零成本爽
**关键对白/证据**:
- 🔒「上一世，我被龙兴县的官衙捕快们斩杀，这一世，我成官衙捕快了？」
- 🔒「反正用的是白莲教的气运，没了就没了」
- 🔒「三个捕快世家，在衙门里是有名录的」
**情绪收益**: 阶级跃迁爽感 + 命运反转宿命感 + 零成本白嫖的快感

**下游期待**:
- 王平以捕快身份如何面对昔日白莲教同门？（谢石/守冲即将强闯县衙）
- 身份能否继续升级？（皂吏→青士...）
- 捕快权力在龙兴县到底有多大？

### 套路 4：升维对比（物质降维打击）

**名称**: 升维对比（物质落差法）
**嵌套型**: 否（新套路）
**所在章节**: ch008

**触发条件**: 王平作为捕快吃到沃焦米+妖兽肉+药田蔬菜 → 对比自己在乡野的饮食（自认不错）→「和吃屎有什么区别」
**核心张力**: 同一位面不同阶级之间的物质生活落差被极度放大——读者跟王平一起经历「原来这才是人过的日子」的认知刷新
**关键对白/证据**:
- 🔒「沃焦米？」「别说是吃了，就是站在旁边闻一闻，王平都能感觉到体内的气血在沸腾」

**情绪收益**: 代入王平的震撼体验 + 对更高阶层的向往 + 对世界观的立体认知

**下游期待**:
- 更高等级的官老爷（青士/朱服以上）的物质生活是什么样？

- 王平是否能在捕快体系中继续往上爬？
