## 一、事件链（骨架层）

### E1｜铁牛功之秘——清明心境破贪刀
- **内容**: 刘烨透露【阎罗三魔刀】第一刀【贪】的克星是事后的「清明心境」；【铁牛功】能多次进入该心境且不伤身
- **原文证据**: 「那段时间里的男子可谓是无欲无求，正好是【贪】之刀意的克星。」
- **🔒 关键对白**: 「可别小看了我的铁牛功。」 / 「心境，和铁牛功相关.....莫非是做了那事之后，短暂的清明心境？」

### E2｜欲界天生法的希望
- **内容**: 王平意识到【铁牛功】可用于辅助【欲界天生法】；揭露【欲界天生法】的修炼机制——108种姿势=小周天，108人=大周天，完成一次大周天即练成
- **原文证据**: 「这门功法，或许还有助于我修行【欲界天生法】！」/「秘法共有108种姿势，全部做完就是一次小周天循环」
- **🔒 关键对白**: 『我若是将【铁牛功】练至圆满，凝聚出来的兵法【金枪不倒】能不能抗住【欲界天生法】的消耗？』

### E3｜反向讨价——从刘爷到刘哥
- **内容**: 王平没钱；刘烨主动送秘籍换未来中乘武功；王平看穿后立刻改口叫刘哥，开价五百两
- **原文证据**: 「难道在你心中，我是那种见钱眼开之人？」/「行了，拿着吧。」/「刘老哥，您要这么说.....可就不是现在这个价钱了。」
- **🔒 关键对白**: 「五百两，一分都不能少！」 / 「抢钱啊你！守冲的赏银都特么没有这么多！」

### E4｜丹凤楼门路——白嫖机会
- **内容**: 刘烨退让，提供丹凤楼白嫖门路（报刘烨名号+练铁牛功即可实战）；王平秒懂刘烨有份，拱手道谢
- **原文证据**: 「钱是没有的，但如果只是想要实战.....我在城里的丹凤楼有一条门路。」
- **🔒 关键对白**: 「多谢刘爷！」 / 「适才相戏耳！」

### E5｜战略忍耐——以手装逼先练功
- **内容**: 王平决定先用手练至铁牛功圆满获得【金枪不倒】兵法，再去丹凤楼一鼓作气修成欲界天生法
- **原文证据**: 「最好的方法，还是先将【铁牛功】悄无声息地练到大圆满境界。」/「有了【金枪不倒】的兵法加持后，再去白嫖。」
- **🔒 关键对白**: 「归根结底，还是我本人洁身自好，绝非好色之徒，岂能无缘无故去那种乌烟瘴气的地方作践自己？」（边说边拿纸巾）

### E6｜月黑风高——谢石夜潜县衙
- **内容**: 场景切换至深夜县衙；谢石用匿息武功潜入，跟巡逻队擦肩而过而不被发现；抵达内衙正堂前看见【莲华刀】
- **原文证据**: 「夜风呼啸，大日隐匿，三轮明月高悬，却被乌云遮住了月光」/「呼吸，心跳，乃至脚步，所有声音都无限趋近于无」
- **🔒 关键对白**: 『找到了.....』 / 『老东西只是拿我探路。』

### E7｜陈皓彦伏击——一箭落贼
- **内容**: 陈皓彦从正堂阴影中现身；谢石跺地崩碎方寸、借反冲疾驰逃遁；陈皓彦开弓一箭——白光撕夜幕、雷音在前电光在后，将谢石从半空击落
- **原文证据**: 「这就想走？」/「外功圆满？老东西还挺会教徒弟。」/「轰隆隆！.....一道白光，撕开了夜幕。」
- **🔒 关键对白**: 「白莲贼子，夜袭县衙，以下犯上，人人得而诛之！」

### E8｜全城搜捕令
- **内容**: 谢石中箭坠入县城；陈皓彦拔刀指天，下令大索全城——线索十两、斩杀百两、窝藏族诛
- **原文证据**: 「传我命令，大索全城，能提供此贼线索者，赏银十两，斩杀此贼者，赏银百两，窝藏此贼者族诛！」
- **🔒 关键对白**: 「锵锵！」（拔刀声）

## 二、爽点设计（爽点层）

### 套路识别

| 套路 | 类型 | 本章出现位置 | 功能 |
|------|------|------------|------|
| **功法联动·资源整合** ★新 | 爽点套路 | E1-E2, E5 | 将铁牛功、阎罗三魔刀、欲界天生法三条线串联，一石三鸟的规划感 |
| **反向讨价·坐地起价** ★新 | 爽点套路 | E3-E4 | 对方主动送→看穿图谋→立刻抬价→最终获利（秘籍+白嫖门路） |
| **黑色幽默·官方吐槽** | 母套路（延续） | E3（刘爷→刘哥）、E5（以手装逼+洁身自好） | 翻脸不认人的速度、边说洁身自好边拿纸巾——反差幽默 |
| **忍辱负重·战略忍耐** | 母套路（延续） | E5 | 先用手练功→待兵法圆满→再去白嫖一波流，布局长远 |
| **月黑风高·潜入行动** | 场景套路 | E6-E7 | 夜色+匿息武功+巡逻擦肩而过，经典刺客潜入桥段 |
| **螳螂捕蝉·黄雀在后** ★新 | 爽点套路 | E7-E8 | 谢石以为潜入成功→陈皓彦早已等候→一箭射落→反转为猎物 |
| **武力展示·降维打击** ★新 | 爽点套路 | E7（一箭） | 外功圆满的谢石被陈皓彦一箭射落——实力差距碾压式展示 |

**情绪阶段变化：**
- **E1-E2**: 好奇心+希望感（功法联动的新知输入 + 修炼前景展望）
- **E3-E4**: 爽感（反向讨价——从叫刘爷到叫刘哥到白嫖门路，层层得利）
- **E5**: 反差幽默+智商感（「洁身自好+拿纸巾」的黑色幽默 → 战略规划的理性感）
- **E6**: 紧张+沉浸（夜色潜入、匿息、巡逻擦肩——悬疑感拉升）
- **E7**: 震撼+爽（一箭射落——武力碾压展示）
- **E8**: 收束+期待（全城搜捕令——之后谢石是死是活？王平如何卷入？）
