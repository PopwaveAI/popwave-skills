## 2. 爽点设计（爽点层）

### 2.1 本章套路
以**战果展示**（王平斩杀谢石后的校场拖尸）为开篇爽点，直接展示主角的战斗结果。以**内心复盘**（灵识+真龙筋战术总结）提供战斗爽感的回味和延展。以**人情博弈**（刘烨退银→王平拒绝→深化友情）提供社交层面的智商/情商爽感。中间以**身份认可+钓鱼任务揭露**为章节核心——陈浩彦认可真龙筋→推荐穿云震天弓→布置九死一生押运任务。三层的权谋设计（陈浩彦→游神真人→守冲的鹬蚌相争）提供信息差解密爽感。末章以**黑色幽默**（「得加钱！」+三天vs三十年的荒诞反差）为收尾，以「加钱」这个现代感槽点和任务的压迫感形成强烈反差。

### 2.2 情绪弧线（起点→推进→高潮→终点）
```
起点：战果展示的爽快（王平拖尸校场，众捕快震惊→王平冷静复盘）
  ↓
推进①：人情温暖（刘烨退银示好→王平拒绝收钱→「我们不是朋友么？」——友情升温）
  ↓
推进②：压迫感重塑（陈浩彦目击之术——紧张感瞬间拉满）

  ↓

```

## 5. 套路归档

检测到以下套路，均在战果复盘 + 身份认可 + 钓鱼任务 + 黑色幽默框架内运行：

### 套路 1：身份跃升（阶级跨越法）— 续

**名称**: 身份跃升（阶级跨越法）
**嵌套型**: 是（续 ch008→ch011）
**所在章节**: ch014

**触发条件**: 王平斩杀白莲反贼谢石 → 以战果获得陈浩彦的正式认可 → 陈浩彦不但认可真龙筋，还公开表示「我当年也是如此」→ 将家族秘传武功穿云震天弓传授给王平

**核心张力**: 从ch008的「刚获得捕快身份」到ch011的「以捕快身份执法」再到ch014的「以捕快身份获得上级认可」——身份跃升的三个阶段：获取身份→使用身份→被权威承认该身份的价值。陈浩彦的「我当年也是如此」是身份体系内最高级别的认可——比当街执法的爽感更强烈，因为这是来自体系内既得利益者的背书。

**关键对白/证据**:
- 🔒「天生神力？……毕竟我当年也是如此。」
- 🔒「我家秘传的【穿云震天弓】，最适合你！」

**情绪收益**: 权威同类认同的快感 + 职业路径明确的向往感 + 「被挑选为继承人/徒弟」的满足感 + 阶级跃升的阶段性里程碑

**下游期待**:
- 王平修炼穿云震天弓后的战力提升
- 能否成功拜入神捕门？
- 陈浩彦对王平的「同类认可」到底是真心还是利用工具的一部分？

### 套路 2：先扬后抑（情绪陷阱法）

**名称**: 先扬后抑（情绪陷阱法）
**嵌套型**: 是（套娃结构，扬中有抑，抑中有扬）
**所在章节**: ch014

**触发条件**: 王平被陈浩彦热情招待（赐座、赏金、夸赞「一表人才」）→ 认可真龙筋 → 推荐穿云震天弓 → 利诱职业路径 → **立刻揭露真相**：押送莲华刀到游神观，九死一生，三天学会三十年武功

**套路公式**:
```
领赏/被认可/被夸赞（扬——心情上升到顶点）→
功法诱惑+职业路径（扬——更高收益的期待）→
任务揭晓（转折点——从扬到抑的急转弯）→
九死一生定性（抑——任务必死）→
三天限时训练（抑上加抑——时间极限压制）→
「练了九死一生，不练十死无生」（抑到谷底——没有选择）→
「得加钱！」（小幅度回扬——用幽默缓解绝望）
```

**核心张力**: 与ch003（内心畅想→背叛）和ch011（权威预言→突破）不同——ch014是先扬后抑的「复合版本」：扬的是身份认可+功法诱惑+职业路径（三重叠加的快乐），抑的是任务不可拒绝+时间极限压迫+任务必死性质（三重叠加的绝望）。但抑到谷底后再用黑色幽默的方式回扬，不是完全绝望。

**关键对白/证据**:
- 🔒「不错，果然是一表人才。」（扬——夸赞）
- 🔒「这是你斩杀反贼的赏钱。」（扬——赏金）
- 🔒「我家秘传的【穿云震天弓】，最适合你！」（扬——功法诱惑）
- 🔒「我要你帮我押送一件东西。」（转折——进入正题）
- 🔒「这就是九死一生的地方了。」（抑——任务定性）
- 🔒「三天.....那还不是九死一生！」（抑——时间压迫）
- 🔒「这个任务属下可以接，但是.....得加钱！」（小幅度回扬——幽默破局）

**情绪收益**: 从天堂到地狱的落差感（强化任务的残酷程度）+ 黑色幽默的意外松口气 + 对主角「即便被压迫也不失幽默」的亲近感

### 套路 3：黑色幽默（官方吐槽）

**名称**: 黑色幽默·官方吐槽
**嵌套型**: 是（续 ch003→ch010）
**所在章节**: ch014

**触发条件**: 陈浩彦用三十年与三天的极限对比制造压迫 → 王平被逼到无路可退 → 咬牙切齿接受任务 → 但以「得加钱！」这一极具现代感的打工人口号来收尾

**核心张力**: 本章的黑色幽默内核不是「双标自欺」（如ch010的洁身自好），而是「体系内打工人的抗争」——面对一个无法拒绝的九死一生任务，王平最后的反抗姿态不是拒绝，而是「可以接，但加钱！」。这种「以退为进」的幽默感，来自于将严肃的生死抉择转化为一场讨价还价的场景——既保留了尊严（我不是无脑服从的炮灰），又保持了松弛感（再绝境也能皮一下）。

**关键对白/证据**:
- 🔒「练了，就是九死一生，不练，就是十死无生，该怎么选你应该明白。」——压迫方的话术
- 🔒「这个任务属下可以接，但是.....得加钱！」——反抗方的反制

**情绪收益**: 在极度压迫的剧情中突然跳出「打工人加钱梗」——读者会因为「这人怎么在这种情况下还敢讲价」而爆笑 + 角色亲和力的强化（不是苦大仇深的主角）

### 套路 4：金手指冷却倒计时（蓄力爆发型）— 续

**名称**: 金手指冷却倒计时（蓄力爆发型）
**嵌套型**: 是（续 ch006→ch007→ch009）
**所在章节**: ch014

**触发条件**: 王平在龙筋问询前内心计算 → 决定不藏拙（展示真龙筋）→ 核心原因：域外天魔「冷却结束了」→ 有底气面对陈浩彦可能的过河拆桥

**核心张力**: 与ch006-ch007的「CD等待」和ch009的「CD未到，克制欲望」不同——ch014是金手指**冷却完毕后的第一次心理暗示**。主角的心态从「等CD→忍」变成了「CD已过→我有底牌→可以冒险」。这一转变标志着王平的行为模式从「谨慎保守」向「有底气地激进」转变。

**关键对白/证据**:
- 🔒「反正【域外天魔】冷却结束了……」
