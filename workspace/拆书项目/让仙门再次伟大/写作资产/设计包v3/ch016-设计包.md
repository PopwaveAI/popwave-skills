## 2. 爽点设计（爽点层）

### 2.1 本章套路
本章以**晨练突破**（弓法入门+血如火练成）为开篇修炼爽点，随后以**草市采购**为主线，展现刘烨的草莽智慧（收例钱退多收、仗官势砍价）和王平对此的观察学习。以**异铁渠道揭露**（赃物销赃链、谢石飞刀来源）提供世界观延展。以**金手指升级**（太平经第二页开启）为章节核心爽点——从获取素材到升级成功的完整闭环。末段以**任务紧急加速**（守冲入城→陈浩彦提前召见）收束，将ch014种下的钓鱼任务推向执行阶段。

### 2.2 情绪弧线（起点→推进→高潮→终点）
```
起点：修炼爽感（清晨练箭→弓法突破→血如火练成）
  ↓
推进①：日常温馨（刘烨送血玉包子→询问效果→人情温馨）
  ↓
推进②：需求提出（问铁→刘烨秒懂→计划去草市——信任关系的展示）
  ↓
推进③：草市百态（刘扒皮骂名→商贩惊恐→刘烨骂骂咧咧但守规矩——人物立体化）
  ↓
推进④：交易博弈（摊主开价50两→刘烨砍到10两——以官压民爽感）
  ↓
高潮：外挂升级（太平经第二页开启——【卷二：空】——金手指里程碑）
  ↓
推进⑤：展望新挂（期待更强的第二经卷——未来期待感）
  ↓
终点：风云突变（守冲入城→计划加速→立刻行动——悬念拉起）
```

**嵌套型**: 否（首次出现）
**所在章节**: ch016

**触发条件**: 王平从黑市购买百炼元铜（异种铁）→ 以太平经吞噬素材 → 太平经轰然震动 → 第二页开启：【卷二：空】【待收录】

**核心张力**: 本章首次完整展示「太平经的升级机制」——之前只在ch002中简单提及太平经的能力（收录功法/能力），ch016首次展示「通过吞噬素材来升级本身」。这种「注入素材→消耗→页面扩容」的机制为后续「寻找素材→升级太平经→收录新挂」的循环提供了完整范式。域外天魔作为第一挂（卷一）珠玉在前，第二挂的期待值极高。

**关键对白/证据**:
- 🔒「【百炼元铜（俗世凡物）】」— 太平经识别素材
- 🔒「【卷二：空】【待收录】」— 升级成功标志
- 🔒「新的【经卷】只要自己选的素材足够好，相信应该能给自己开出一个更强的外挂吧？」

**情绪收益**: 金手指成长的正反馈感 + 「系统升级了」的满足感 + 对未来新能力的强烈期待

### 套路 2：逛街见闻（世界观延展型）— 新

**名称**: 逛街见闻（世界观延展型）
**嵌套型**: 否（首次出现）
**所在章节**: ch016

**触发条件**: 王平跟随刘烨前往草市 → 目睹刘烨收例钱的全过程（喧嚣→退回多收→对骂但不动手）→ 接触到赃物黑市交易链（谢石飞刀+捕快佩刀）→ 以官压民砍价成功

```

场景进入（草市喧嚣）→
人物展示（刘烨的规矩/骂名/原则）→

```

**核心张力**: 本章的「逛街」并非纯粹的水字数，而是通过草市的见闻完成了三项功能：(1) 刘烨人设的立体化——从单纯的「净街虎」到「有规矩的老油条」（退回多收的钱）；(2) 世界观的延展——赃物交易链的揭露（谢石飞刀+战死捕快刀 → 县衙有人参与销赃）；(3) 太平经升级素材的获取——这是逛街的「实质产出」。
