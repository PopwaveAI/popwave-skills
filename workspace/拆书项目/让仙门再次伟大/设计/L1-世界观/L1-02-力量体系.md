# L1-02 力量体系

⚠️ **基于前37章数据推断**：当前只读到了全书的前37章（共约5-6卷）。所有结论（境界总览、核心路径、突破条件等）均基于此有限窗口，不代表全书终态。后续章节可能补充或修订以下内容。

---

## 一、境界总览

本书存在**两大平行世界对应两套独立力量体系**——赤土世界的武道体系与仙门世界的修仙体系，二者互不统属，但存在跨位面的战力碾压关系。

### 赤土武道完整等级链

```
凡人（黔首/平民）→ 练武入门（铁牛功等）→ 外功六要素修炼期
→ 皮如牛 → 肉如象 → 筋如龙 → 骨如虎 → 血如火 → 髓如霜
→ 外功圆满（六要素全部大成）
→ 内劲武师（修功体图→开内劲）
→ 封号武师（觉醒神意，如徐秉正「大鹏金翅鸟」）
→ 见神宗师（当前世已知最高成就）
→ 武神（传说级，武道史上第一位创造神意体系的存在）
```

外功六要素**无严格顺序约束**，可并行修炼（如王平筋如龙后跳过骨髓直接修血如火），每要素分四品（如血分：烈火/炽炎/真焱/至阳四品）。兵器五等体系平行存在：凡兵→利器→名珍→神物→天工（ch009）。武功品级独立分类：下乘→中乘→上乘→绝世（ch009）。

### 仙门修仙完整等级链

```
凡人（无灵根→人牲/力士/随侍童子）
→ 渡劫三境：人劫 → 天劫 → 命劫  ← 有灵根者正统路径
→ 炼炁 → 修真 → 登仙
                                ← 另类分支
废灵根者 → 六欲天自在成就法（以身豢兽炼废灵根）→ 废灵根大成（无炼炁资格）
无灵根者 → 五贼盗天功法（命/物/时/功/神）→ 盗天炼炁（劫修路径，无需真灵根）
```

灵根分三档：废灵根（万中无一但仍大道无望）→ 真灵根（修仙正选）→ 异灵根（ch033）。灵根测试是天尊传下的铁则——「天尊言出法随，九字而定天下法」：「渡劫，炼炁，修真而登仙」（ch034）。

### 其他修炼体系（线索级）

其他体系前37章仅展示片段，未形成完整等级链：
- **大德体系**：死而复生——守冲被「怜生菩萨」复活（ch018）
- **异人/方仙道体系**：游神真人（≈方仙道异人），独立于武道和修仙的修炼路径（ch018/ch029）
- **妖兽体系**：妖兽横行（ch018展示巨型蛇妖），自有修炼等级但前37章未展开
- **梵修体系**：他化自在统摄欲界天书（ch007），独立修行路径
- **域外天魔系统**：王平金手指——死后复活+穿越+身份兑换，非传统修炼体系（ch004+）

## 二、核心路径

### 赤土武道核心路径（主路径）

**途径一：外功六要素锤炼（底层散修/军户/捕快主流）**

最通用的修炼方式，不需要特殊天赋。通过药浴、淬体、功法演练逐步强化皮肉筋骨血髓六要素。每关分四品，越往后代价越高——「血如火」需要神引药（名珍级灵药，价值连城，ch015）、「骨如虎」需要山君骨（ch021梁上机关取宝）。王平从铁牛功入门到外功七次质变圆满（ch025），走的是典型的资源驱动型路径——灵识抄全本武功（ch015）、截胡夺宝莲华刀（ch018）、入苏家取丹（ch024-ch025），每一步都是信息差换资源。

**途径二：功体图→内劲→封号武师（中层/高层主流）**

外功圆满后，核心步骤是修习功体图（如天鹏擒龙功体图，ch025-ch026），建立内劲循环。功体图分「天、地、人」三等，天鹏擒龙图为顶尖级别（「伥」与「食」双用，ch025）。内劲大成后有一定概率觉醒神意，晋升封号武师（徐秉正即此境界）。封号武师拥有神意——意念压制/监牢具象化，相当于精神层面的领域技（ch027）。封号武师之上是见神宗师，武道史上还有武神（创造神意体系的传说），但前37章未出现宗师级实际战斗。

**途径三：军阵协同（官府/军队系统）**

百人捕快军阵以气血共鸣驱动，可爆发出≈封号武师级的战力（ch031）。这不是个人修炼路径，而是组织化战力放大手段。

### 仙门修仙核心路径

**途径一：灵根正统修仙（仙门主流）**

以灵根资质为门槛，走「渡劫→炼炁→修真→登仙」的天尊法旨路径。渡劫三境中「人劫」可具体描述为四项特质：「通灵识念/无漏真身/周天吐纳/借炁显法」（ch034）。这条路径需要真灵根或异灵根——废灵根者只能止步于渡劫境底层，「大道无望」（ch034）。

**途径二：六欲天自在成就法（废灵根旁门）**

以身豢兽——通过「兽房采集欲兽胚胎→体内炼化欲兽→集齐六欲兽→六欲浑天魔」的路径，将废灵根修炼大成（ch034-ch036）。王平在3个月内完成四头欲兽炼化，废灵根大成。可解锁六大神异：飞天/万钧/卸甲/吐纳/神足/蛟钺（ch034）。但这只是给了修炼资格，无法炼炁，上限有限。

**途径三：五贼盗天功法（劫修路径）**

「五贼者，命、物、时、功、神」（ch037）——盗取天地五贼之炁，无需真灵根也可炼炁入道。这是盗天贼创立的叛逆功法，已开源，形成劫修群体（ch035）。前37章仅揭示五贼真意和功法存在，具体修炼法在玉简中，被尤道姑打伤后陷入沉寂（ch037）。

**途径四：器灵修仙（主角自创路径）**

王平肉身毁灭后成为六欲天魔旗器灵，推演出「成为器灵不受灵根限制」（ch037）这条从未有人走过的路径。目前只有理论框架，无完整修炼方法。

### 两条体系之间的核心门槛差异

武道突破的核心瓶颈是**资源**——没有名珍级灵药、异铁神兵、功体图，天赋再高也卡在某个要素上不去（王平若非截胡夺宝，仅凭正常捕快薪水三辈子也凑不齐髓如霜的材料）。修仙突破的核心瓶颈是**灵根资质**——灵根是先天决定的硬门槛，废灵根者无论如何努力也「大道无望」，除非走五贼盗天或器灵化这种制度外的旁门路线。

## 四、力量天花板

### 赤土武道天花板

**已知最高：封号武师（神意层次）——徐秉正级。**

封号武师已是龙兴县/长乐郡级别的武力顶峰。徐秉正一人即可镇压全县、可单人碾压王平（王平需靠与日偕亡+一念花开合击才勉强对抗，ch027-ch028）。百人军阵爆发才「≈封号武师级」战力（ch031），说明封号武师之上一级（见神宗师）才是真正的地方性战略力量。

**传说级：武神。** 武道史上第一位创造神意体系的传说——「一位武神级的存在……创造出了【神意】体系」（ch028）。前37章未出现武神级实际战力，其战力规模不可考。

**跨位面参照**：力士（仙门最低级的人形傀儡/奴仆）放到大顺可「轻轻松松打遍天下无敌手」（ch007）。寇豹（欲兽，仙门妖兽）≈内劲武师（ch034）。这意味着修仙体系的起步级即可碾压赤土武道的中上层，武道巅峰（见神宗师/武神）可能勉强对应修仙体系的中下层。「把仙门的寇豹放到大顺，起码也是内劲武师级别」（ch034）是前37章唯一的跨体系战力标尺。

### 仙门修仙天花板

**当前视野极限：登仙（＞修真＞炼炁＞渡劫）。**

天尊法旨明确的终极目标：「渡劫，炼炁，修真而登仙」（ch034）。前37章中出现的最高修士为尤道姑（极乐仙峰修士），但她的具体境界未量化给出（ch005-ch037）。师姐尤雪心在ch006展示的「采阳补阴」与ch036的「移景定身」术法属于修仙体系的手段，前37章未出现劫修之上的实战对抗，修仙天花板的具体战力无法量化。

### 太平经系统

太平经本身不是力量体系，而是**金手指平台**——提供死后复活/穿越/身份兑换/诅咒/贷款等功能，不属于「修炼升级」范畴，但它解锁的「域外天魔系统」和「锚定身份」是主角突破力量上限的核心杠杆（ch004+）。

## 五、跨级约束

### 武道内部的跨级压制

**外功六要素每阶四品差距即见胜负**：刘烨展示的铁牛功圆满兵法「金枪不倒」（ch009）说明不入流武功也能有奇效，但面对中乘武功/利器级兵器即被碾压。ch013王平一刀斩杀谢石（同为白莲教出身但兵器/功体差距），体现了装备+功体图对同级战的绝对优势。

**神意对非神意的碾压**：ch027王平首战徐秉正时，徐秉正的神意（意念压迫）让王平「感觉身体被一座无形大山压住」，灵识才是破神意的唯一手段——这是认知维度级的差距。没有灵识（修仙级精神感知）的武者面对神意基本无解。

**军阵可跨级但代价高**：百人军阵可爆发封号武师级战力（ch031），但代价是「每一轮冲击都有减员」，且需要训练有素的军官维持阵型。

### 修仙内部的跨级压制

**灵根三档之间是天堑**：废灵根「大道无望」（ch034），无论如何努力无法渡劫，无法炼炁。真灵根是修仙正选，异灵根万中无一——废灵根对真灵根修士的战斗力差距可能比凡人vs武者还大。

**渡劫三境每差一境判若云泥**：人劫期四大特质（通灵识念/无漏真身/周天吐纳/借炁显法）是基础，天劫和命劫的超凡能力前37章未展示但暗示差距巨大（ch034）。

### 武道 vs 修仙的跨体系压制

**修仙起步碾压武道中层**（最关键的跨级约束）：

- 寇豹（欲兽/仙门底层战斗单位）≈ 内劲武师级战力（ch034）
- 力士（最低级人形傀儡）可「轻轻松松打遍天下无敌手」（ch007），≈ 封号武师以下全部
- 这意味着：赤土世界的武道修炼者在面对修仙世界的正规战力时，最低级的单位就相当于武道中层（内劲武师）。武道巅峰（见神宗师/武神）可能勉强跟修仙的渡劫中期对标，但炼炁级以上的修士对赤土世界就是降维打击

### 金手指系统的约束

- 域外天魔复活：冷却**7天**（ch005）——一场大战中只能死一次
- 身份兑换：需要消耗一方**势力气运**（ch008），气运不是无限供应
- 一诺千金：千日内还贷（ch033-ch034），借的是未来的「战力债」
- 太平经收集：物品品质需逐级递增，不能重复收集低级物品（ch005）
