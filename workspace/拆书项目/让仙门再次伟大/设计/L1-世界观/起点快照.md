# 起点快照 · 王平（ch001之前状态）

> ⚠️ **基于前37章推断**：以下快照基于ch001开场时的状态描述和ch001-ch037中的倒叙/记忆补全。不包括金手指觉醒后的任何信息。

---

## 三维快照

### 力量

| 子维度 | 状态 | 证据 |
|:-------|:-----|:-----|
| **功法** | 无功法凡人 | 穿越二十年只会白莲教传功时获得的基本武艺（ch001「苟且偷生了二十年」），没有完整功法体系 |
| **武艺** | 粗浅功夫 | 基本拳脚/刀法水平被谢石压制（ch001穿越回忆中谢石是监视者，王平实力弱于谢石） |
| **金手指** | **无（自以为）** | 以为穿越没有带外挂（ch001「穿越至今二十年，苟且偷生了二十年」→ch005「原来开挂是这种感觉……草泥马二十年啊！」） |
| **内力/气血** | 无系统修炼 | 未修炼任何内功/气血功，仅靠白莲教基层传功的粗浅吐纳维持基本体力 |
| **兵器** | 普通牛尾刀 | 使用的只是底层教众的基础兵器，无特殊属性 |
| **身份力** | 白身 | 黔首阶层，无任何体制背书（ch008「当前身份：流民（白身）」定位了ch001之前的身份等级） |

**一句话**：王平是在大顺王朝最底层的黔首，只靠基本武艺和二十年隐忍苟活的穿越者。

---

### 关系

| 子维度 | 状态 | 证据 |
|:-------|:-----|:-----|
| **白莲教内** | 潜伏伪装 | 表面是前村白莲教堂口的忠诚教众（ch001参加集会行礼「莲华降世，普度众生」），实则是受命卧底 |
| **与谢石** | 同门/被监视 | 谢石是守冲派来盯梢王平的人（ch002守冲命令「盯着王平，我要知道他未来几天的所有行踪」），表面上以师兄师弟相称 |
| **与守冲** | 表面师徒 | 守冲是白莲教高层（老道士/前村堂口实际掌舵人），王平拜师学飞刀但关系核心是「监视与被监视」（ch002收徒+同时安排谢石盯梢） |
| **与县衙上线** | 单线接头 | 有县衙内某位上线（ch003揭示是陈浩彦），但上线的身份信息对王平不对称（王平只知道联系人方式，不知道上线全貌） |
| **与普通人** | 边缘化存在 | 作为白莲教众（在官府眼中是逆贼）+穿越者（心理上疏离），在底层社会中也处于边缘地位 |
| **亲密关系** | **无** | 二十年中未建立任何真正的亲密关系（ch029穿越者乡愁段暗示王平在赤土孤独求生，父母记忆在另一个世界） |
| **信息管道** | 老鼠传信 | 特殊的卧底联络方式——通过老鼠传信（ch001）而非直接见面，说明他的联络层级极低且被严格保密 |

**一句话**：王平是一个被白莲教和县衙双向操纵的「双面间谍」——两边都不信任他，两边都在用他。

### 认知

| 子维度 | 状态 | 证据 |
|:-------|:-----|:-----|
| **世界观认知** | 以为世界是「单纯武道世界」 | 不知道修仙、异人、大德、妖兽的存在（ch018「这破地方难道不是单纯的武道世界吗？怎么听上去水还挺深！」——以ch018的震惊反推ch001前的无知） |
| **对穿越的认知** | 认命穿越无外挂 | 二十年穿越生涯没有任何金手指觉醒，已经放弃开挂期待（ch001「苟且偷生了二十年」→ch005金手指觉醒前的绝望） |
| **对白莲教的认知** | 知道表面价值（反抗封建）但怀疑其真相 | 知道白莲教是底层反抗组织，但作为卧底自然不相信其教义——内心认为白莲教也是压迫工具之一 |
| **对县衙的认知** | 作为工具利用但又警惕 | 知道县衙是压迫机构，但作为卧底选择了「弃暗投明」——实际上是选择了「投靠更强大的压迫者」（ch002「当官府暗中联系上他的时候……王平立刻选择了弃暗投明」） |
| **对自身处境的认知** | 清晰但不甘 | 完全清楚自己是双面间谍随时可能被抛弃（ch001的「全体自裁」提议正是这种危机感的直接反应），但无力量改变 |
| **对阶级的认知** | 亲身体验但尚未提炼为思想 | 作为底层黔首生存二十年，亲身感受过阶级压迫（ch019回放婴儿时吃糙米+沙砾分三天煮粥的记忆——上上世被饿死），但尚未形成「阶级批判」的系统认知 |
| **对未来的认知** | 绝望中带着麻木的坚持 | 没有明确的长期目标，只有「活下去」的本能——二十年苟活本身就已经是最好的策略 |
| **黑色幽默能力** | 已存在 | 穿越者思维+现代梗储备已经存在（ch001「草泥马」式吐槽在ch005才开口喷出，但二十年间积累的吐槽素材早已存在） |

**一句话**：王平以「武道世界+无外挂」的认知框架在赤土底层生存了二十年——他以为自己对世界的理解是完整的，实际上只看到了冰山一角。
