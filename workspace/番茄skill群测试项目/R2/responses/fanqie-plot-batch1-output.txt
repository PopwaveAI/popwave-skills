## ch001 · 高考开场白

**概念变体**：V001 - 高考考场0点觉醒系统，获得灰色天赋【一目十行（弱化版）】，靠它抄完试卷并震惊全班。

**核心爽点**：未知→揭示（从绝望到开挂）

**爽点触发位**：章末（成绩公布全班震惊）

**钩子**：系统面板消失前留下一句话：“明天0点，你还能获得新的天赋，但槽位只有一个——你留得住今天的神技吗？”

**节奏约束**：事件密度10 / 概念交互≥2次（激活面板+使用能力） / 前300字出冲突（高考卡题）

**边界**：写到成绩公布全班震惊为止 / 不能写后续分析成绩和回家场景

**前章衔接**：无（第一章）

**章型**：opening_shift

## ch002 · 吃垮自助餐

**概念变体**：V002 - 第二天0点获得灰色天赋【超强消化】，槽位已满，在保留【一目十行】备战复试和换用【超强消化】赚钱之间博弈，最终选择后者并去自助餐测试。

**核心爽点**：弱→强（从穷学生到靠吃赚钱的转变）

**爽点触发位**：前1/3（决定换天赋） + 中段（自助餐碾压）

**钩子**：章末，在自助餐店门口被一个穿西装的男人拦住：“小朋友，有没有兴趣参加三天后的大胃王比赛？奖金十万。”

**节奏约束**：事件密度9 / 概念交互≥2次（换天赋决策+吃自助测试） / 前300字出冲突（0点开奖发现灰色废柴，但想到合成又犹豫）

**边界**：写到被邀请为止 / 不能深入写比赛内容

**前章衔接**：ch1结尾全班震惊，放学回家路上思考明天要换什么天赋

**章型**：exploration_gain

**节奏约束**：事件密度10 / 概念交互≥3次（使用超强消化+获得新天赋+面对威胁） / 前300字出冲突（比赛开始对手下黑手）

**边界**：写到混混围堵为止 / 不能写正式交手

**前章衔接**：ch2结尾被西装男邀请，第二天来到比赛现场

**章型**：confrontation_pressure

## ch005 · 雷法初现

**概念变体**：V005 - 获得白色天赋【初级雷法】，需要覆盖【痛觉迟钝】，犹豫后覆盖，并用雷法震慑混混同伙，同时引出觉醒者世界线索。

**核心爽点**：未知→揭示（新天赋的强大效果） + 弱→强（从被追打到一击震慑）

**钩子**：远处高楼上一个黑衣人放下望远镜：“初级雷法……有意思，派人接触一下。”同时系统提示：“解锁第二天赋槽需要10个白色能量，当前进度0/10。”

**节奏约束**：事件密度8 / 概念交互≥2次（覆盖决策+雷法使用） / 前300字出冲突（混混同伙再次包围）

**边界**：写到惊退众人和系统提示解锁条件为止 / 不能写与黑衣人直接接触

**前章衔接**：ch4结尾获得【初级雷法】，本章开始决定是否覆盖【痛觉迟钝】

**章型**：reveal_hook
