# 幕001 候选方案

## 候选A："暗夜掘墓人"——隐秘觉醒路线

**核心概念**：林远穿越后三天内，将面板当作"产品原型"进行灰度测试，每晚偷偷到村外坟地练习盗贼基础动作。面板在授名仪式前夜自动解锁盗贼职业，他利用盗贼的"察觉"加成，在仪式当天发现祭坛下藏有阵营晶石——仪式不是恩赐而是灵魂权限写入。他必须在仪式上伪装成"零神眷的普通农夫"过关，同时暗中发现真相。仪式后一只威胁农田的低级怪物给了他首杀机会——用农夫式战斗（地形+农具+盗贼技能）完成。

**弧线构成**：面板发现与测试（发现弧）→ 坟地秘密练习（成长弧）→ 仪式真相揭露（悬念弧）→ 首杀实战验证（战斗弧）

**素材侧重**：方向05（低等级战斗写法：环境利用+AC对抗）、方向07（弱者的武器：日常抵抗+信息不对称恐惧）、方向00（面板=世界异常点+农夫式战斗体系）

**资源路径**：盗贼技能（隐匿/察觉）→ 零资源起步 → 农具即兴武器 → 面板解锁的"实践次数"机制

**挫折设计**：练习第三夜差点被巡夜守卫发现，被迫中断并改变练习地点（信息不对称恐惧具象化）

**爽点节点**：面板激活→盗贼解锁→发现仪式真相→首杀完成→老巴克递出第一把武器

**优势**：最贴合主角引擎设定的第一章主动动作，"坟地练盗贼"画面感极强，信息不对称恐惧贯穿全幕
**劣势**：前三分之一偏隐秘行动，缺乏正面冲突张力，需要村庄日常做缓冲

## 候选B："温水锅中的人"——村庄生态路线

**核心概念**：以灰麦村日常性苦难为主舞台，林远穿越后逐步体验村庄的压迫光谱——村长的善意压迫、守卫的冷漠执行、父亲的麻木接受、村民的闰土式驯化。面板在日常观察中浮现，林远以运营思维分析村庄权力结构（画组织架构图）。授名仪式是"温水煮沸"的高潮时刻——他亲眼看到一个少年被阵营晶石"写入"职业，灵魂被绑定。仪式后一只哥布林斥候袭击村庄外围，林远利用农夫环境知识完成首杀，但战斗方式异常引起老巴克注意。

**弧线构成**：村庄压迫体验（社会弧）→ 面板浮现与分析（发现弧）→ 仪式目击与真相初探（悬念弧）→ 哥布林首杀（战斗弧）

**素材侧重**：方向08（村庄NPC功能分化+动态反馈系统+三级威胁生态）、方向07（鲁迅农民形象光谱+日常性苦难+全景敞视恐惧）、方向00（灰麦村=温水锅+灰麦村NPC=压迫光谱活体标本）

**资源路径**：村庄情报网络（酒馆/铁匠铺/告示板）→ 农夫环境知识 → 面板被动响应 → 老巴克的冒险经验

**挫折设计**：林远试图帮一个被征用粮食的农夫说话，反被村长"善意"教训"你不懂神的安排"——善意压迫比暴力更难反抗

**爽点节点**：画出村庄组织架构图→面板首次响应→仪式上看穿晶石本质→首杀农夫式战斗→老巴克"你小子不简单"

**优势**：村庄生态最丰满，NPC群像最丰富，打工人共情最强（"这不就是公司吗"）
**劣势**：前期节奏偏慢，"温水锅"需要大量日常描写，首杀来得偏晚

## 候选C："MVP测试者"——产品经理路线

**核心概念**：林远穿越后以互联网运营思维对待面板——先跑通最小可行产品（MVP）。他将面板功能拆解为"用户行为→系统响应"的因果链，设计实验验证面板规则（做什么能推进进度条？实践次数怎么算？）。授名仪式被他视为"竞品分析"——观察教会如何执行授权流程。仪式上他利用面板的"察觉"被动加成，发现阵营晶石的灵魂写入机制。首杀是他"验证面板战斗功能"的测试——他主动出击消灭威胁农田的野狼，用数据复盘战斗效率。

**弧线构成**：面板MVP测试（发现弧）→ 竞品分析（仪式悬念弧）→ 战斗功能验证（首杀战斗弧）→ 系统复盘与迭代（成长弧）

**素材侧重**：方向03（异常法则"日常中单点偏差"+番茄黄金三章"危机+金手指"+打工人身份焦虑）、方向01（多职业"广度换深度"+骰子判定不确定性）、方向00（面板=反996系统+前代持有者失败记忆）

**资源路径**：数据分析思维 → 面板规则逆向工程 → 农夫知识+地形利用 → 实验迭代式成长

**挫折设计**：第一次"测试"面板时过度暴露，被村中一个敏感的村民（C-2型：羡慕的底层青年）注意到异常，不得不花精力消除隐患

**爽点节点**：面板规则逆向破解→画出"面板功能逻辑图"→仪式上看穿晶石→首杀数据复盘→记忆碎片首次涌入的震撼

**优势**：打工人思维浓度最高，运营方法论贯穿始终，"用产品经理方式玩DND"反差极强
**劣势**：过度理性化削弱情感张力，"测试者"视角让角色显得疏离，村庄温度感不足

## 方向选择

**选定方向：候选A"暗夜掘墓人"，整合B的村庄生态丰富度与C的运营思维内核。**

**选择理由**：

1. **最贴合主角引擎**：主角引擎明确指定第一章主动动作是"坟地练习盗贼隐匿/察觉"，候选A以此为骨架，最忠实于角色设定
2. **画面感最强**："少年在坟地月光下练习隐匿步伐，面板进度条缓缓推进"——这个画面既是开局钩子也是成长隐喻
3. **张力结构最完整**：隐秘练习（暗线）+ 村庄日常（明线）+ 仪式真相（高潮）+ 首杀验证（爽点），四线交织
4. **整合空间大**：B的村庄NPC群像可作为日常场景填充（老巴克、村长、父亲、村民），C的运营思维可作为林远的内心独白和面板测试方法论
5. **信息不对称恐惧最浓**："随时可能被发现"的恐惧贯穿前三章，制造持续追读动力
6. **首杀设计最合理**：盗贼技能（隐匿+察觉）+ 农夫环境知识 + 农具即兴武器 = "农夫式战斗"的独特美学，区别于一切同类作品
