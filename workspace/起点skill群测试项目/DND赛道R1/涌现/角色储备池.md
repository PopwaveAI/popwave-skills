# 角色储备池（Step 9）

> 分级原则：S级=贯穿全书的核心角色（含主角），A级=卷级主要角色，B级=功能性槽位（非完整人物），C级=路人反馈模板。
> 原型迁移红线：只迁移气质/反应模式/处境张力/冲突方式，禁止照搬身份/专有设定/具体剧情。

### S-1 林远（主角）

| 字段 | 内容 |
|------|------|
| 来源依据 | 穿越者设定 + 互联网运营职业思维 + 金手指（不受神权限制的职业面板）的载体 |
| 角色原型参考 | 详见下方 |
| 稳定锚点 | 996受害者的共情——看到任何"系统压榨个体"的场景都会触发强烈反应，无论多冷静都会血压升高 |
| 欲望与压力 | 想让所有人自由发展（欲望）vs 记忆碎片侵蚀自我认同+分享面板削弱自身（压力） |
| 反应模式 | 遇事先收集信息→分析系统漏洞→制定AB计划→选择风险最低的路径执行；危机中优先保护信息源和退路而非正面应对 |
| 说话方式 | 现代人思维但刻意压低词汇复杂度，常用比喻把DND概念翻译成自己能理解的框架（"授权仪式就是入职体检加竞业协议二合一"）；紧张时会在脑内默念"运营SOP第一步：确认问题边界" |
| 可制造事件类型 | 发现系统漏洞/建立地下信息网络/职业面板泄露引发追杀/分享面板引发阵营冲突/记忆碎片中前主人秘密曝光 |
| 主角可利用点 | N/A（是主角本身） |

**角色原型参考（3个）：**

1. **《诡秘之主》克莱恩·莫雷蒂** — 迁移气质：谨慎到偏执的系统化思维者，在危险中保持理性分析，习惯用"流程"对抗"未知"。林远和克莱恩一样会在行动前做大量准备工作，但林远的"系统"不是神秘学而是互联网运营方法论。
2. **《全职猎人》小杰·弗里克斯** — 迁移处境张力：从底层突破壁垒的执着，面对"不可能"时不是退缩而是找方法。林远和小杰一样有"我偏要试试"的倔劲，但林远的驱动是理性分析而非直觉。
3. **《三体》罗辑** — 迁移冲突方式：普通人被推上改变世界的位置，内心挣扎于"我不想要这个使命"和"只有我能做"之间。林远和罗辑一样会经历从逃避到承担的心路，但林远的承担更早、更主动。

### S-2 塞拉斯·光明之手（核心对手/神选之子）

| 字段 | 内容 |
|------|------|
| 来源依据 | 对立面设计——主角是"反抗系统者"，他必须是"系统最大受益者且真心捍卫系统者"。不是反派，是"另一种正义"的代表 |
| 角色原型参考 | 详见下方 |
| 稳定锚点 | "我见过没有秩序的世界是什么样"——幼年亲历边陲无序区的屠杀（匪帮屠村），被光铸者教会救起。他不是被洗脑，是用创伤说服了自己：授权系统是必要的善 |
| 欲望与压力 | 想维护世界秩序让所有人安全（欲望）vs 越来越多证据证明神在利用凡人、授权系统并非"保护"而是"控制"（压力） |
| 反应模式 | 先评估威胁等级→按教会规章选择应对方式→规章不足时以"神的旨意"为名自行裁决→事后向教会报备；面对主角时会有微妙犹豫——因为主角的"人人自由"理念让他想起幼年屠杀前"如果人人都有力量自保"的幻想 |
| 说话方式 | 正式、有教养、引用经文但不是教条式而是论述式（"光铸者的教谕第三章说'秩序是自由的基石'，这不是命令，这是观察结论"）；常用看似有道理的话，让主角难以简单反驳 |
| 可制造事件类型 | 追查异端主角/教廷内部政治斗争/与主角的正面交锋/信仰危机后的立场动摇/最终站队选择 |
| 主角可利用点 | 1)他的"善意"——他不会用卑鄙手段，给主角留下了利用规则的空间 2)他对神的盲信使他在看到真相时会产生动摇，动摇期是主角争取他的窗口 3)他受教会规章约束，规章的漏洞就是主角的逃生通道 |

**角色原型参考（3个）：**

1. **Fate系列 吉尔伽美什** — 迁移气质：天选之人的傲慢与真诚并存。塞拉斯和金闪闪一样真心相信自己的秩序观是正确的，不是虚伪的——但塞拉斯比金闪闪多了一份"我见过混乱的代价"的创伤驱动。
2. **《火影忍者》宇智波鼬** — 迁移冲突方式：为了大局牺牲个人情感，站在制度一方做痛苦的选择。塞拉斯和鼬一样会为了"秩序"做出残酷的决定，但他不像鼬那样沉默——他会解释，会辩论，会试图说服主角他是错的。
3. **《冰与火之歌》琼恩·雪诺** — 迁移处境张力：被命运选中的人，认真对待责任，但内心有自我怀疑。塞拉斯和琼恩一样被推上"拯救者"的位置，但塞拉斯的"拯救"方向是维护现有秩序，而琼恩是打破——这个镜像让两人的对峙有悲剧感。

**完整人物卡：**
- 全名：塞拉斯·冯·黎明阶（被光铸者教会收养后赐姓"黎明阶"）
- 年龄：22岁
- 授权职业：圣武士（光铸者授权，守序善良）+ 牧师（光铸者授权）+ 法师（律法者授权，守序中立）——三重授权，诸神联合选定的"完美容器"
- 等级：起始12级（英雄期），全书随主角成长同步提升
- 阵营：守序善良（表面）/ 守序中立（实际——他优先维护秩序而非善良）
- 外貌：金发蓝眼，面容端正但不是柔美的英俊而是刚毅的端正，右手背有光铸者的圣印烙痕
- 核心矛盾：他越接近真相（神在利用凡人），越需要做出选择——是维护"善意的谎言"还是面对"痛苦的真相"

### S-3 渊者·梅琳达（神秘导师/后期关键角色）

| 字段 | 内容 |
|------|------|
| 来源依据 | 需要一个"先行者"角色——50年前就尝试过打破神权限制的人，她的失败是主角的警告，她的知识是主角的钥匙 |
| 角色原型参考 | 详见下方 |
| 稳定锚点 | "我差一点就成功了"——这句话她对自己说了50年。失败不是因为方向错，是因为她一个人走得太快。她看到林远时第一反应不是"终于有人来接班"而是"别让他重蹈覆辙" |
| 欲望与压力 | 想亲眼看到神权被推翻（欲望）vs 害怕林远重蹈自己覆辙、害怕自己给的情报反而害了他（压力） |
| 反应模式 | 先测试对方的心性和决心（设局观察对方在压力下的选择）→以谜语和考验代替直接帮助→确认对方值得信任后才给关键信息→给出信息后立刻拉开距离（怕自己的感情影响判断） |
| 说话方式 | 简短、带刺、常用反问句（"你觉得呢？""你确定？"）；话只说一半，留余味；偶尔在关键时刻说出完整而残酷的真相，像刀子一样精准 |
| 可制造事件类型 | 考验主角的心性/揭露"沉默之冬"历史真相/以自身封印为代价提供关键帮助/临终托付核心秘密/她的过去引发教会最高级别追查 |
| 主角可利用点 | 1)她掌握50年前对抗神权的第一手经验 2)她知道诸神的弱点——神力与信徒数量的链接 3)她自己就是一面镜子，她的失败路径是主角的避坑指南。但获取这些需要通过她的考验，而她的考验往往涉及道德困境 |

**角色原型参考（3个）：**

1. **《指环王》甘道夫** — 迁移角色功能：引导者，在关键时刻指路但不替代主角做选择。梅琳达和甘道夫一样是"知道更多但不说全部"的导师，但梅琳达比甘道夫更破碎、更犹豫——因为她失败过，而甘道夫没有。
2. **克苏鲁神话体系中 知晓禁忌知识的老者** — 迁移处境张力：知道太多，被知识本身伤害。梅琳达和这类角色一样，她的力量和知识是被封印的"代价"——她知道诸神的秘密，但说出那些秘密会加速她自己封印的崩溃。
3. **《进击的巨人》艾尔文·史密斯** — 迁移反应模式：为更大目标牺牲一切的执念，但在关键时刻会面对"我的执念是不是自私"的自我审判。梅琳达和艾尔文一样会为了"看到真相被揭露"而牺牲自己，但她比艾尔文多了一层"我已经牺牲了一次但失败了"的创伤。

**完整人物卡：**
- 全名：梅琳达·灰烬（原名梅琳达·银叶，被封印后自取"灰烬"为姓）
- 年龄：物理年龄约70岁（外观），实际灵魂年龄超过200岁（封印期间意识仍在运转）
- 原授权职业：德鲁伊（生命母授权）→ 后自行突破获得游侠+野蛮人+术士的非授权职业
- 当前状态：大部分力量被封印，仅保留3级德鲁伊能力和残缺的游侠感知
- 等级：封印前18级（传奇期），封印后约5级（凡人期）
- 外貌：灰发灰眼，面容枯瘦但眼神锐利如鹰，左手始终戴着手套（手套下是封印灼烧的痕迹）
- 核心矛盾：她想帮助林远，但每一次帮助都会消耗她残存的力量；她需要在"帮多少"和"留多少给自己保命"之间反复权衡

| 主角可利用点 | 1)老巴克的冒险经验（他曾是"野武士"——未授权自学战士职业的人）能教林远基础战斗 2)他在灰麦村的人望可以帮林远获得村民的初步信任 3)他被教会断臂的经历是"授权系统残酷性"的最佳活教材 |

**完整人物卡：**

- 年龄：52岁
- 原职业：未授权的"野武士"——年轻时离开村庄试图自学战士职业，成功达到4级后被教会捕获，右臂被"阵营审判"强制剥离职业能力时废掉
- 当前身份：灰麦村铁匠，瘸了一条腿（年轻时冒险受的伤）+ 断了右臂
- 等级：封顶4级战士（被封印后降为普通人）
- 核心功能：第一卷的情感锚点+世界观解说者+主角离开村庄后的牵挂

### A-2 灰鸽·艾琳（教会中的灰色角色）

| 字段 | 内容 |
|------|------|
| 来源依据 | 需要"体制内的良心"角色，让教会不沦为脸谱化反派，同时为中期"教会内部分裂"埋线 |
| 角色原型参考 | 1)《悲惨世界》沙威 — 迁移冲突方式：制度执行者，内心正义感与制度冲突时的崩溃。艾琳和沙威一样相信制度，但她的崩溃不是跳河而是"开始在制度内留余地" 2)《魔戒》法拉米尔 — 迁移气质：善良的体制内人，面对诱惑/压力时选择不那么做。艾琳和法拉米尔一样有能力但不受重用，因为她"不够狠" 3)《 hunger games 》普鲁塔克 — 迁移反应模式：表面服从体制，内心有更长远的计划，在体制内悄悄布局 |
| 稳定锚点 | 弟弟被剥夺职业时眼中的绝望——她执行过无数次"阵营审判"，但只有弟弟的那次让她在夜里失眠。从那以后她开始"留余地" |
| 欲望与压力 | 想在教会体制内从内部改变不公（欲望）vs 体制不允许任何改变，且她的"留余地"行为一旦被发现就是异端（压力） |
| 反应模式 | 严格按照程序行事但在程序允许的范围内留余地（该搜的地方"搜过了没找到"、该报告的细节"遗漏了"）→内心冲突时主动寻找"合规的理由"放水→放水后自我怀疑（"我是在救人还是在纵容异端？"） |
| 说话方式 | 公事公办的语气，措辞精准如法律文书；关键处会加重语气暗示（"我明天会来搜查你的住处，上午十点"——暗示你最好在那之前离开）；独处时会对着弟弟的画像自言自语 |
| 可制造事件类型 | 追查主角时的"意外放水"/教会内部清洗时被迫站队/弟弟事件被政敌利用勒索/最终倒戈提供教会核心情报 |
| 主角可利用点 | 1)她的弟弟——主角如果能帮弟弟恢复（或找到替代方案），就能获得一个教会内部的高级别盟友 2)她的"程序内留余地"模式——林远可以精确计算她的行为模式，预判她的"放水"时机 3)她掌握教会审判官的内部通讯密码和巡查路线 |

**完整人物卡：**
- 全名：艾琳·灰鸽（"灰鸽"是教会审判官的代号，意为"灰色地带的信使"）
- 年龄：28岁
- 授权职业：牧师（暗影母授权，中立邪恶标签——讽刺的是她本人并不邪恶，但暗影母偏好"秘密与暗影"的审判官）
- 等级：8级牧师（超凡期）
- 阵营：表面中立邪恶（授权标签）/ 实际中立善良
- 外貌：黑发短发，教会审判官黑色制服，面容冷淡但眼睛在看到受审者时会微微颤动
- 弟弟：艾尔文，14岁，原为见习法师（律法者授权），因"阵营偏移"被审判剥夺，现为废人
- 核心功能：中期"教会内部分裂"剧情的引爆点+主角进入教会核心层的通道

### A-3 燎原·凯尔（底层反抗者领袖）

| 字段 | 内容 |
|------|------|
| 来源依据 | 需要"反抗军"势力代表，且与主角形成"推翻方式"的路线分歧——凯尔主张暴力革命，林远主张系统性瓦解 |
| 角色原型参考 | 1)《V字仇杀队》V — 迁移气质：地下革命者的个人魅力与偏执。凯尔和V一样有强大的感召力，但凯尔比V更务实也更冷酷 2)《星球大战外传》卡西安·安多 — 迁移反应模式：务实的反抗者，为了目标什么都做得出来包括牺牲同伴。凯尔和安多一样会做"必要的恶" 3)《三国演义》曹操 — 迁移冲突方式："宁教我负天下人"的实用主义，在乱世中选择效率最高而非道德最优的路径 |
| 稳定锚点 | "我们不是异端，我们是人"——这句话是他建立反抗军的初心，但随着反抗军壮大，这句话逐渐变成口号而非信念 |
| 欲望与压力 | 想推翻神权让人人自由（欲望）vs 手段越来越极端、开始不择手段牺牲无辜者、反抗军内部开始出现独裁倾向（压力） |

## C级路人反馈模板（8类）

> 以下为路人对主角/事件的不同反应模板，用于快速生成群体反应场景。

### C-1 恐惧的村民
**触发条件**：亲眼看到主角使用非授权职业能力
**反应链条**：惊恐 → 看圣人一样看主角（"你是神派来的吗？"）→ 被教会告知主角是异端 → 恐惧压过敬畏 → 偷偷向教会报告（出于恐惧而非恶意）
**对话模板**："你……你怎么没有仪式就能……不不不，我什么都没看到，别找我……"
**叙事功能**：展示底层人对"异常"的本能恐惧——他们不是坏，是被系统吓怕了

### C-2 羡慕的底层青年
**触发条件**：听说有人不用授权就获得了力量
**反应链条**：羡慕 → 向往（"如果我也行就好了"）→ 犹豫（"但被发现了会死"）→ 二选一：鼓起勇气加入主角的地下网络 / 因恐惧而退缩并嫉妒
**对话模板**："真的不用仪式就能……？我……我也能行吗？不，算了我什么都没说。"
**叙事功能**：展示"渴望自由"的种子在底层萌芽——这是后期革命的基础

### C-3 鄙夷的授权者
**触发条件**：看到主角的"野路子"职业能力
**反应链条**：鄙夷（"没有神的恩赐终究是野狗"）→ 嘲讽（"你那点花招也配叫职业？"）→ 被主角实力打脸后震惊 → 要么更新认知要么加倍仇恨
**对话模板**："无授权的野路子？呵，别侮辱'职业'这个词了。等真正的冒险者来了你就知道……"
**叙事功能**：展示授权系统对既得利益者的洗脑——他们真心相信"授权=正统"

### C-4 狂热的信徒
**触发条件**：听到任何质疑神权的声音
**反应链条**：狂怒 → 以"异端"之名发起攻击/举报 → 如果打不过就哭嚎"神会惩罚你的" → 永不悔改
**对话模板**："亵渎！这是亵渎！光铸者在上，这个异端必须被净化！来人啊！异端在这里！"
**叙事功能**：展示授权系统的"精神控制"效果——信徒不是被强迫的，是被洗脑到真心维护系统

### C-5 冷漠的旁观者
**触发条件**：都市中看到与反抗/镇压相关的事件
**反应链条**：看热闹 → 无所谓谁赢（"反正跟我没关系"）→ 事件影响到自己生活时才开始有态度 → 态度取决于"谁让我的日子更好过"
**对话模板**："又闹了？上次是盗贼公会火拼，这次又是什么？算了，面包涨了两个铜板才是正经事。"
**叙事功能**：展示"沉默大多数"——革命不是靠说服所有人，是靠足够多的人不再沉默

### C-6 投机的商人
**触发条件**：看到局势变化可能影响利益
**反应链条**：立刻计算利益 → 两头下注（同时卖情报给教会和反抗军）→ 谁赢帮谁 → 永远站在"赢面大"的一边
**对话模板**："这位大人说笑了，我一个做小本生意的哪敢站队……不过如果您需要什么物资，我倒是有个渠道。价格嘛……好商量。"
**叙事功能**：展示利益驱动的中立力量——他们的存在让局势更复杂

### C-7 绝望的被剥夺者
**触发条件**：曾有职业但被教会"阵营审判"剥夺的人，看到主角的自由职业能力
**反应链条**：麻木（已经放弃了希望）→ 看到主角时眼中闪过光 → 要么奋起（"让我也试试"）要么因绝望太深而疯狂（"凭什么你行我不行！"）
**对话模板**："职业……我曾经也是法师……他们把我的法师拿走了……你说你能给人职业？你骗人……你骗人的吧？"
**叙事功能**：展示授权系统最残酷的后果——被剥夺者比从未拥有者更痛苦

### C-8 好奇的学者
**触发条件**：从学术角度得知主角的"无授权职业"现象
**反应链条**：学术好奇（"这在理论上不可能"）→ 请求研究/观察主角 → 提供理论分析但缺乏实战经验 → 可能无意中泄露主角信息给教会
**对话模板**："太不可思议了……你的灵魂结构没有授权晶石的烙印，这在现行理论框架下是不可能的。我能记录一下吗？我保证不写名字——学术伦理嘛。"
**叙事功能**：展示"知识本身的中立性"——学者的好奇不是恶意但可能造成严重后果
