# 方向00：内部涌现

## 危机维度扫描
| 危机维度 | 本幕是否有？ | 推导 |
|---------|------------|------|
| 生存危机 | 有 | 1级角色仅6-12HP，首杀面对低级怪物仍有生命危险；毒素溅射无体质豁免险些丧命。低等级战斗的"一击致命"特性（方向05）制造持续生存紧张感。 |
| 身份危机 | 有 | 记忆碎片涌入导致"我是程远还是某个死去战士"的认知混淆（方向06）；穿越者身份与农夫身份的割裂；面板持有者身份与未授权者身份的矛盾（方向03+04）。 |
| 阵营危机 | 有 | 多职业阵营撕裂首次显现——战士的守序倾向与潜在的混乱倾向冲突（方向01+02）。第一幕高潮处"创伤触发立场转换"导致阵营条剧烈摇摆（方向02）。 |
| 社会危机 | 有 | 未授权者拥有职业能力=死罪。授权者巡视制度制造"全景敞视"恐惧（方向07）。首杀后战斗方式暴露异常，为"被发现"埋下伏笔（方向08）。 |
| 道德危机 | 有 | "只有我有面板"vs"人人自由发展"的核心矛盾初现（方向04）。程远使用面板时产生的记忆碎片可能操控行为，造成意外伤害后的道德自责（方向06）。 |
| 信息危机 | 有 | 程远不知道授权系统的完整规则，不知道面板的来源和代价上限，不知道村庄里谁是眼线（方向07）。限制性视角让读者与主角共享信息不对称的焦虑。 |
| 成长危机 | 有 | 多职业的"分享削弱"——升级速度被分散到多个职业，每个职业等级提升缓慢（方向01）。1-3级需要实打实的战斗经验，无法速成（方向05）。 |
| 关系危机 | 有 | 程远的多职业面板让他与村民产生隔阂——他拥有了不该拥有的东西，这种"异类感"影响他与朋友、家人的关系（方向07+08）。 |

## 金手指巧思扫描
| 检查项 | 达标？ |
|--------|-------|
| 代价是否具体可见 | 达标——记忆碎片干扰（视觉/听觉/行为层面可感知）、阵营撕裂（面板上阵营条可视化摇摆）、分享削弱（升级经验需求翻倍可量化）三重代价均有具体表现。 |
| 代价是否递增 | 达标——每多一个职业，记忆碎片增多+阵营冲突加剧+升级速度更慢。代价随金手指使用深度递增，形成"越用越强但越危险"的螺旋。 |
| 代价是否与爽点形成张力 | 达标——每解锁新职业是爽点（获得新能力），但伴随记忆碎片涌入（恐惧）+阵营撕裂（痛苦）+升级减速（焦虑）。爽与痛同步递增。 |
| 金手指是否与世界观深度绑定 | 达标——面板的存在直接否定"神权授权系统"的合法性，它是体制的"系统级漏洞"，不是外挂而是"世界观内的反体制工具"。 |
| 金手指是否有成长空间 | 达标——从1职业到多职业，从1级到高等级，从被动接受记忆到主动管理碎片，面板本身有清晰的成长曲线。 |
| 金手指是否有被发现的风险 | 达标——第一幕的核心风险就是"被发现"。授权者巡视制度、村民眼线、战斗方式异常都是暴露途径。被发现=死刑。 |
| 金手指是否有独特性（区别于同类作品） | 达标——不同于"无系统猎杀诸神"（方向04），面板是"对神权系统的系统级破解"，本质是"开源超凡技术"。多职业+三重代价的组合在同类作品中差异化明显。 |
| 金手指是否与角色弧光绑定 | 达标——程远的"打工人反抗"动机与面板的"打破垄断"功能天然契合。面板不是给他的"礼物"，而是他"死过一次后获得的第二次机会的工具"（方向03）。 |

## 燃料扩展推导
| 已有设定/外部素材 | 碰撞产生的新燃料 | 来源 |
|---------|----------------|-----------|
| 多职业"广度换深度"（方向01） + 打工人996猝死背景（方向03） | **"职业面板=反996系统"**：程远在前世被系统压榨至死，穿越后获得的面板本质上是"绕过压榨系统"的工具。他的反抗不是为了当英雄，而是"老子死过一次了，绝不再被系统压榨"。这让金手指与角色动机形成完美的因果关系。 | 方向01×方向03 |
| 阵营撕裂"创伤触发转换"（方向02） + 记忆碎片"认知不可靠性"（方向06） | **"阵营撕裂的记忆触发机制"**：每次阵营倾向剧烈摇摆时，不是简单的"行为犹豫"，而是触发对应阵营职业的前代持有者记忆碎片涌入——守序面触发时涌入圣武士的誓言记忆，混乱面触发时涌入游荡者的背叛记忆。阵营撕裂与记忆碎片不是两个独立代价，而是因果联动的一体两面。 | 方向02×方向06 |
| "弱者的武器"日常抵抗（方向07） + DND低等级环境战术（方向05） | **"农夫式战斗体系"**：程远初期的战斗风格完全不同于正规冒险者——他利用农具(锄头/镰刀/草叉)作为即兴武器，利用农田地形(沟渠/田埂/谷仓)制造优势，利用农夫知识(天气/土壤/作物)预判怪物行为。这种"农夫式战斗"既是低等级的权宜之计，也是主角独特的战斗美学——不华丽，但致命；不专业，但聪明。 | 方向05×方向07 |
| 守序邪恶"温水煮蛙"（方向02） + 神权本质=阶级利益博弈（方向04） | **"灰麦村=温水锅"**：灰麦村的压迫不是暴力恐怖，而是"日常性苦难"——农夫多干活少吃饭被视为"天经地义"，偷学技能被视为"亵渎神明"。村长不是恶人，而是真心相信神权秩序的信徒。这种"善意压迫"比暴力压迫更难反抗，因为你无法把一个"真心为你好"的人定义为敌人。程远的第一步不是"打败村长"，而是"认清温水正在煮沸"。 | 方向02×方向04×方向07 |
| 异常法则"日常中的单点偏差"（方向03） + DND村庄动态反馈系统（方向08） | **"面板=世界中的异常点"**：面板的出现本身就是一个"异常"——它存在于一个不该存在的地方(未授权者身上)，做着不该做的事(自主分配职业)。第一幕的叙事节奏可以完全套用异常法则：Ch1面板浮现但不解释，Ch2面板功能显现但主角不确定是否真实，Ch3面板影响行为但世界仍未察觉。异常法则的"不解释、不升级、世界保持稳定"三原则完美适配第一幕的克制节奏。 | 方向03×方向08 |
| 记忆碎片"碎片拼接信息释放"（方向06） + 神权推翻叙事"身份反转"（方向04） | **"前代持有者的失败记忆=全书悬念种子"**：第一幕中涌入的记忆碎片拼合出一个模糊图景——曾经有人持有过这个面板，那个人很强，但最终失败了。他是谁？为什么失败？面板为什么传给了程远？这个悬念不在第一幕解答，但碎片的存在让读者知道"面板有历史，有前任，有失败的先例"——这比"面板是无敌外挂"更有深度，也为后续卷的叙事埋下了种子。 | 方向04×方向06 |
| "全景敞视"信息不对称恐惧（方向07） + DND优劣势战术机制（方向05） | **"信息不对称=战术劣势的可视化"**：程远不知道授权者的侦测范围和手段，这种信息不对称不仅制造心理恐惧，还直接影响战术选择——他必须假设"随时可能被发现"，因此每一次练习面板技能都要精心设计撤退路线和掩护故事。这种"如履薄冰"的练级过程本身就是高密度叙事素材。 | 方向05×方向07 |
| DND村庄NPC功能分化（方向08） + 鲁迅农民形象光谱（方向07） | **"灰麦村NPC=压迫光谱的活体标本"**：村庄中每个NPC代表压迫光谱上的一个点位——村长(压迫者/守序邪恶)、守卫队长(执行者/工具人)、神殿牧师(体制维护者/可能动摇)、铁匠(羡慕者/潜在盟友)、麻木村民(被驯化者/闰土型)、有反抗意识的村民(觉醒者/祥林嫂型)。程远在村庄中的每一次互动都在这个光谱上滑动，不同NPC对他的态度变化反映了"面板存在"对整个村庄权力结构的冲击。 | 方向07×方向08 |
