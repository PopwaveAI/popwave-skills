## 燃料速览

### 方向00-内部涌现
- **危机维度扫描**：8个危机维度全部命中（生存/身份/阵营/社会/道德/信息/成长/关系），第一幕危机密度极高
- **金手指巧思扫描**：8项检查全部达标，三重代价（记忆碎片+阵营撕裂+分享削弱）形成递增螺旋
- **核心新燃料**：
  - "职业面板=反996系统"——金手指与角色动机的因果绑定
  - "阵营撕裂的记忆触发机制"——两大代价因果联动
  - "农夫式战斗体系"——低等级战斗美学
  - "灰麦村=温水锅"——善意压迫比暴力压迫更难反抗
  - "前代持有者的失败记忆=全书悬念种子"——碎片拼接式信息释放
  - "灰麦村NPC=压迫光谱的活体标本"——每个NPC代表一个压迫维度

### 方向01-DND职业体系深度
- 12职业功能分化 → 神权授权体系的等级阶梯映射
- 多职业"广度换深度" → 金手指代价的规则根基
- 骰子判定+豁免机制 → 低等级"命运不确定性"
- 战斗三分体系 → 30-50%失误概率的凡人赌命感
- 职业与阵营关联 → 阵营撕裂的设定根基

### 方向02-DND阵营系统与道德困境
- 九宫格哲学深度 → 阵营撕裂=角色弧光核心引擎
- 阵营流动性"创伤触发" → 第一幕高潮的转折机制
- 权力关系维度"谁制定规则" → 全书道德张力根源
- 守序邪恶"温水煮蛙" → 灰麦村统治者的角色定位
- 阵营倾向光谱 → 面板可视化界面的设计依据

### 方向03-打工人穿越异界开局模式
- 异常法则"日常中单点偏差" → 面板浮现的叙事节奏
- 番茄黄金三章"危机+金手指" → 第一章开篇模板
- 打工人身份焦虑 → 程远的核心反抗动力
- 情绪过山车"好奇→着急→满足" → 三章结构设计
- "反铺垫"原则 → 世界观通过体验流出而非解说

### 方向04-神权统治推翻叙事
- 《牧神记》残老村模式 → 灰麦村=被神权边缘化的村庄
- 《溯回青空》身份反转 → 程远在神权叙事中="异端/魔种"
- 番茄"猎杀诸神"差异化 → 面板=系统级破解而非暴力推翻
- 神权本质=阶级利益博弈 → 授权系统=技术垄断工具
- 《道诡异仙》认知不确定性 → 记忆碎片的恐惧体验来源

### 方向05-DND低等级战斗写法
- AC对抗机制 → "一击致命"的凡人赌命紧张感
- d20优劣势机制 → 弱者胜强者的规则支撑
- 豁免脆弱性 → "赢了战斗但死于副作用"的代价设计
- 1-3级成长曲线 → 第一幕等级区间控制+3级子职业质变节点
- 环境利用+道具战术 → "农夫式战斗"的规则基础

### 方向06-记忆碎片/多重身份叙事
- 《记忆碎片》双线逆向 → 记忆可篡改性的叙事应用
- 诺兰记忆哲学 → "我还是我吗"的身份焦虑
- 多重视角身份重构 → 不同职业=不同人格切面
- 非线性碎片拼接 → 记忆碎片的信息释放节奏
- 记忆行为操控伦理 → "这不是我做的"的道德困境

### 方向07-底层革命/农民起义写法
- 《弱者的武器》日常抵抗 → 程远初期的"偷懒+装糊涂+暗中互助"
- 弱者叙事限制性视角 → 读者与主角共享信息不对称焦虑
- 鲁迅农民形象光谱 → 灰麦村村民的麻木→觉醒光谱
- 日常性苦难 → 灰麦村压迫的质感来源
- 全景敞视信息不对称 → "随时可能被发现"的持续恐惧

### 方向08-DND世界村庄生态
- 动态世界反馈系统 → 村庄态度随程远行为变化
- NPC功能分化 → 灰麦村6大NPC角色定位
- 新手村任务机制 → 神权体制对"新手村"功能的扭曲
- NPC/冒险者边界 → 程远"越界"对体制的威胁
- 三级威胁生态 → 首杀来源+战斗方式暴露的伏笔
