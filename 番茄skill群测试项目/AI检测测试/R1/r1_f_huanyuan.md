# 换元法版 · 从《我的诡异人生》第一章换元

窗外淅淅沥沥下着小雨。

陈三更望着门外的雨线，默默地叹了口气，他的心情就像外面的天色一样沉郁。*

三年前的今天一样下着小雨，父亲的死讯就在那天送到了铺子里。*

坐在柜台后面，他拿起一块手表。*

手表是老物件了，小时候见爷爷一直戴着这块手表，后来传到父亲手里。再后来，父亲故去，这块手表就到了陈三更手中。他一直把这块表带在身边，聊作念想。*

表盘镜面有些泛黄模糊，金属表链倒仍然发着亮光，此刻镜面下的表针一动不动，固执地指向两点十五分。*

铺子里很安静。座钟的摆停了有一阵了，他懒得去上弦。这两年老街的人越来越少，连买香的人都少了。上个月把纸钱涨了五毛就被人指着鼻子骂了三条街。

"怎么不转了？"*

陈三更皱眉看着停滞的表针，他从父亲的遗物里翻出这块表以后，它就一直转动着，这三年来这还是第一次停止转动。*

是不是坏掉了？

掐住自己的思维，他手指拧在表盘右侧的表把上，尝试拧转表把。*

然而，表把却像是焊死在了表盘上一样，任凭陈三更手指发力，它都纹丝不动。*

他有些着急，更加大了力道。*

冷不防手指抹过手表侧面的棱角，原本不会伤人的棱角在他用力过甚下，直接在他指腹上割出一个小小的创口。*

鲜血立时渗出，染红表盘玻璃。*

陈三更连忙抽纸去擦镜面上的鲜血——但不等他擦拭，血液就在镜面上凝成一滴血珠，这滴血珠倏忽间渗进表内，落在表针上。

"哒哒，哒哒，哒哒……"*

原本停滞不动的表针，当下再度转动了起来。像是得了那一滴鲜血的助力。*

棺材间里的长明灯这时候自己亮了。陈三更没有去点它，它自己亮的。油灯的火苗跳了三下，然后稳住了，跟表针的节奏一模一样——哒，哒，哒。

整块表盘都变作鲜艳的红色，而后鲜红色染透了表链，牢牢缠绕在陈三更的手腕上。*

他看到血色浸染下的秒针以肉眼难及的速度，顺时针转动着，过了一圈又一圈。分针、时针的指向也跟着飞快变化。*

表针尤在转动，陈三更所处的整个铺子陷入无边的黑暗里。*

一块巨大的表盘悬浮在黑暗中，所有的表针统统指向了数字"0"。*

不是机械的电子女声，是一个很老的声音，像收音机里搜到的短波电台，刺啦刺啦的，在他脑海里响起来：

"陈记白事。第三十七代。香火接上了。"*

他愣在那里。黑暗里什么都看不见。柜台上那枚老太太留下的铜钱开始在口袋里发烫，隔着一层布，烫得他大腿生疼。他把它掏出来——铜钱在黑暗里发光，不是火焰，是那种老怀表表面的夜光，暗绿色的，一圈一圈往外扩散。

铜钱正面的符号在动。不是他眼花。那枚他不认识的符号像是一条虫子，在铜面上游动，从正中心游到边缘，然后停住了。停住的位置正对着棺材间的方向。

陈三更的意识被吸入表盘之中。*

他眼前的屏幕崩碎开来，恍恍惚惚间，像是真的坠入了另一个世界，四周一片漆黑。*

他尝试着活动身体，发现自己坐在一张椅子上，周围什么也看不见。

不像宿舍。不像铺子。这里是——他闻了闻——湿土和腐烂木头的味道。像古墓。脚下很凉。水泥地面变成了夯土，踩上去鞋底往下陷一截。

铜钱还在发光。那枚符号停在的位置，正前方。

黑暗里，有人的呼吸声。
