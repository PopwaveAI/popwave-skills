# R2 测试包

## 测试目标
基于M70朱雀报告的4段分析，大胆猜想朱雀检测机制，设计5个假设验证实验。

## M70报告回顾
| 片段 | AIGC值 | 内容类型 |
|------|--------|---------|
| 片段1 | 89.92% | 环境描写（纯换元） |
| 片段2 | 30.55% | 画皮鬼独白（换元） |
| 片段3 | 81.18% | 合同对话（原创） |
| 片段4 | 24.59% | 系统【】格式（混合） |

## 5个假设与测试

### R2-A 环境碎片化 — 测H3
- **假设H3**：环境描写天然判AI，碎片化无法改变
- **做法**：只把片段1改为3-6字碎片，片段2-4保持M70原样
- **预期**：如果H3正确→碎片化无效（片段1仍89%）；如果错误→碎片化有效
- **文件**：r2_a_env_fragment.md（1946字）

### R2-B 对话独白化 — 测H4
- **假设H4**：重复/断裂/省略的独白体天然像人类
- **做法**：只把片段3合同对话改为独白体（重复/省略/断裂/自言自语），片段1/2/4保持M70原样
- **预期**：如果H4正确→片段3从81%降到30%以下
- **文件**：r2_b_dialogue_monologue.md（2196字）

### R2-C 信息密度稀释 — 测H1
- **假设H1**：朱雀检测句间信息密度，密度高判AI
- **做法**：全文每2句信息句后插入1句零信息句（废话/重复/无意义动作）
- **预期**：如果H1正确→全文AI分整体下降
- **文件**：r2_c_density_dilution.md（2156字）

### R2-D 可预测性打断 — 测H2
- **假设H2**：朱雀检测上下文可预测性，可预测判AI
- **做法**：全文每5-8句插入1处不可预测打断（跑题/意外反应/答非所问）
- **预期**：如果H2正确→全文AI分整体下降
- **文件**：r2_d_unpredictable.md（2200字）

### R2-E 多源换元 — 测H5
- **假设H5**：M90不如M70是因为单源换元同质化，多源换元可解决
- **做法**：70%换元，从第一章和第二章交替换元
- **预期**：如果H5正确→多源换元AI分低于单源换元
- **文件**：r2_e_multi_source.md（2200字）

## 测试优先级
1. **R2-B（对话独白化）** — 最可能验证，因为片段2已证明独白30%有效
2. **R2-A（环境碎片化）** — 直接针对89%的重灾区
3. **R2-C（信息密度稀释）** — 测试最核心假设
4. **R2-D（可预测性打断）** — 测试第二核心假设
5. **R2-E（多源换元）** — 验证同质化问题

## 对照基线
M70（56%人工 / 43%疑似 / 0%实锤）作为对照基线，R2各版本只改变一个变量。
