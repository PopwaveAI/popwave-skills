# R3：质量突破法

## 核心原则
不破坏文本质量，而是用AI无法达到的写作质量冲破检测上限。

## AI写作的特征（要避免的）
1. 每句只传递信息，不制造体验
2. 感官描写泛化（"阴冷""潮湿"等通用词）
3. 叙述者无人格，像摄像机而非人
4. 情节逻辑驱动，情感逻辑缺席
5. 句式节奏均匀，无情绪起伏
6. 比喻用常见配对（冷→冰窖，害怕→心跳加快）

## 人类优秀写作的特征（要做到的）
1. 每句制造体验——读到一个句子，你在身体里有所感受
2. 感官具体到引发生理反应——不是"冷"，是那种让你的后槽牙发酸的冷
3. 叙述者有人格——有态度、有判断、有情绪、有偏见
4. 情感逻辑驱动——场景服务于情感，而非情感服务于场景
5. 节奏随情绪变——紧张时短促，回忆时绵长，爆发时一个长句砸下来
6. 比喻是新鲜的——从角色的生活经验中长出来的比喻，不是通用比喻库

## 写作方法
1. 感官锚定法：每个场景找一个"锚定感官"——不是泛泛的五官全开，而是抓住一个最有记忆点的感官细节，深挖到底
2. 情感前置法：先确定这个场景的情感核心是什么（恐惧？荒诞？心酸？），所有描写服务于这个情感
3. 叙述者人格法：叙述者不是摄像机，是一个有态度的人。他对眼前的一切有看法，有评价，有联想
4. 生活经验比喻法：比喻从角色的职业/经历中长出来（产品经理的比喻用互联网术语，不用文学术语）
5. 情绪节奏法：句长跟随情绪——焦虑时句子越来越短，回忆时句子拉长，爆发时一个短句砸断
