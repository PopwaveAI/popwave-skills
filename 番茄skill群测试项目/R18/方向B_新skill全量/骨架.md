# 骨架（plot v6.1.0）

## 一句话核心
散修沈劫能看见并剪断别人命数线夺取修为，但每剪一条自己命线细一分——杀到最强时命如蛛丝，一阵风就能断。

---

## 全书三段走向

**第一段·矿场蛰伏（卷一）**
沈劫是青冥宗黑矿场的底层挖矿散修，灵石被抢、命如草芥。一次濒死中觉醒断命者体质——能看见所有人头顶的命数线，伸手一剪，对方修为尽数涌入。他用这能力反杀矿场管事赵屠，逃出矿场进入青冥宗外门。但第一次主动剪线后，他摸到自己脉搏变虚，命线肉眼可见地细了一截。

**第二段·宗门崛起（卷二至卷三）**
沈劫进入青冥宗外门，在宗门等级体系里往上爬。每一步晋升都伴随着"杀谁、不杀谁"的选择。他发现宗门本身就是一台命数收割机——内门长老的修为靠吸取外门弟子的命元维持。命线越来越细，陈青衣劝他收手，但停手就是死。他查出断命者体质是被人为植入的，幕后有人需要一个"替天剪线"的工具。

**第三段·逆天对决（卷四至终章）**
沈劫发现整个修真界的灵气体系是一张巨型命数网，天道本身就是最大的收割者。所有修士都是被养肥的羊，断命者是唯一能剪断天道之线的存在。终极抉择：剪断天道=自己命线彻底碎裂。沈劫选择以命换命，在巅峰时刻消亡，换修真界挣脱收割。

---

## 第一卷终点与Boss

**第一卷终点**：沈劫剪断赵屠命线逃出矿场，被陈青衣带入青冥宗外门，但第一次感受到命线变细的恐惧——代价已经刻进骨髓。

**第一卷Boss**：赵屠——青冥宗黑矿场管事，筑基中期。靠压榨散修灵石和血气修炼，手下养了一帮打手，矿场散修死在他手里的不下百人。沈劫第一次主动剪筑基修士的命线，修为暴涨但也第一次看到自己命线变细。

## 角色人设（5人）

### 1. 沈劫（主角）
- **身份**：散修，青冥宗黑矿场矿工，断命者体质
- **核心欲望**：站着活下去。不跪不舔，不被这吃人世道吞掉
- **内在矛盾**：每杀一人变强一分，但离死亡近一步。越强越脆弱，越想活越得杀人
- **功能**：道德灰度主角，每章做"杀不杀"的选择。读者代入的不是无敌爽，是"代价感"
- **性格底色**：闷，狠，认死理。不废话，不说狠话，动手前不预告

### 2. 陈青衣（师兄/情感锚点）
- **身份**：青冥宗外门弟子，筑基初期，来矿场采购灵石时撞见沈劫
- **核心欲望**：保全自己安稳日子，但良心过不去——他想拉沈劫一把又怕被拖下水
- **功能**：提醒代价。每次沈劫剪线后他是最先看到命线变细的人，代表"正常人视角"
- **性格底色**：碎嘴，怕事，但关键时刻不掉链子。跟沈劫形成一动一静的搭子感

### 3. 赵屠（第一卷Boss）
- **身份**：黑矿场管事，筑基中期
- **核心欲望**：靠矿场散修的血汗修炼，攒够资源冲击筑基后期进内门
- **致命弱点**：贪婪不知收手。把所有散修当灵石矿，逼到绝路的人多了，恨他的人比怕他的人多
- **功能**：第一个被剪的反派，展示金手指的爽感和代价的双重冲击
- **性格底色**：粗中有细，笑面虎。打你之前先笑，笑完下手不留活口

### 4. 苏映雪（中后期关键角色）
- **身份**：青冥宗内门弟子，背景成谜
- **核心欲望**：查清父亲在矿场"意外死亡"的真相
- **功能**：信息差载体。她手里的线索指向宗门高层，是沈劫后期掀桌子的引线
- **性格底色**：冷，硬，有耐心。不信任任何人但会合作

### 5. 莫老（神秘老者/设定揭示者）
- **身份**：矿场角落里一个捡矿渣的瘸腿老头，实为被废修为的元婴修士
- **核心欲望**：找一个断命者终结天道收割——他曾是上一个断命者，失败被废
- **功能**：设定揭示。他知道断命者体质的来历、代价的真正含义、天道的秘密，但每次只透露一点
- **性格底色**：话少，眼神比嘴毒。看沈劫像看自己年轻时候

## 幕序列（5幕）

### 第一幕：矿场血路（ch01-10）— 黄金三章法则
**功能位**：开篇钩子 → 金手指激活 → 短期目标 → 第一次打脸 → 绑定助攻 → 第一卷Boss战
**一句话弧线**：从被踩进泥里的矿工，到剪断管事命线杀出血路——代价是命线第一次变细
**章节预算**：10章

- **ch01·冲突爆发**：沈劫挖到的灵矿被赵屠手下强抢，反抗被打到吐血，濒死时眼前出现一根根光线——命数线第一次显形
- **ch02·金手指激活**：沈劫本能伸手剪断打他矿工的命线，对方倒地气绝，一股热流涌入——他获得了练气一层的修为，同时摸到自己脉搏发虚
- **ch03·短期目标明确**：赵屠发现矿工死因蹊跷，怀疑沈劫，放话三天后"处理"他。沈劫必须变强逃出去，莫老告诉他"你能看见别人看不见的东西"
- **ch04-07·第一次打脸**：沈劫暗中剪掉矿场几个恶霸打手的命线，每次修为涨一截。赵屠发现手下接连暴毙开始恐慌追查，沈劫从被追的猎物变成暗处的猎人
- **ch08-10·绑定助攻+Boss战**：陈青衣来矿场采购灵石，撞见沈劫异常。ch09赵屠锁定沈劫设下死局，ch10沈劫正面硬刚赵屠，剪断他的筑基命线——修为暴涨到练气圆满，但命线细了一圈，陈青衣看见他脉搏跳得像蜂鸟

### 第二幕：外门求生（ch11-25）
**功能位**：新环境立足 → 体系规则建立 → 中层反派登场 → 代价深化
**一句话弧线**：进入青冥宗外门，发现宗门本身是更大的矿场——外门弟子是内门的耗材
**章节预算**：15章

### 第三幕：内门暗流（ch26-50）
**功能位**：权力斗争 → 信息差引爆 → 体质真相揭露 → 中期Boss战
**一句话弧线**：查明断命者体质被人植入，宗门长老想把他当工具——他成了所有人的猎物和棋子
**章节预算**：25章

### 第四幕：天下棋局（ch51-80）
**功能位**：跨宗门格局 → 天道线索 → 命线危机爆发 → 终极抉择前置
**一句话弧线**：发现整个修真界是天道的命数网，所有修士都是待收割的羊
**章节预算**：30章

### 第五幕：断天之战（ch81-终章）
**功能位**：终极对决 → 代价兑现 → 主题升华
**一句话弧线**：沈劫以最后的命线剪断天道之线，最强即消亡，换修真界挣脱收割
**章节预算**：20章

## 信息差（3条）

1. **沈劫不知道自己的断命者体质是被人刻意植入的**——莫老知道（他是上一个失败的断命者），但只在关键时刻透露碎片。读者跟着沈劫一起发现真相，制造"被操控"的恐惧感。

2. **赵屠不知道沈劫就是杀他手下的人**——读者看着沈劫一条条剪掉打手命线，赵屠还在矿场里暴跳如雷追查"凶手"。读者拥有上帝视角的爽感，等着看赵屠发现真相那一刻。

3. **沈劫不知道每剪一条命线，不只是变细——还在吸引天道的注意**。命线变细不是单纯的减寿，是让自己在天道的命数网上"显形"。剪得越多，越容易被天道锁定。这个信息差在中后期引爆，让前期的"爽"变成后期的"恐"。

## 核心高潮点（3个）

1. **第一卷高潮·剪赵屠**（ch10）：沈劫正面剪断筑基中期赵屠的命线，修为暴涨到练气圆满。但他低头看见自己命线细了一圈，脉搏虚快如蜂鸟。陈青衣抓住他的手腕，脸色煞白："你的脉……怎么跳成这样？"——爽感与恐惧同时降临，金手指的代价第一次具象化。

2. **中期高潮·宗门大比暴露**（约ch40）：沈劫在宗门大比上被逼入绝境，失控之下全场修士的命数线同时显形——他看见每一个对手头顶的线，忍不住伸手。全场数百修士命线尽收眼底，所有人第一次知道"断命者"是什么。他成了所有人心中的怪物，也是所有势力想抢夺的工具。

3. **终极高潮·剪天道**（终章）：沈劫站在天道命数网前，看见那根贯穿天地的巨线。剪断它=修真界挣脱收割，但自己的命线会彻底碎裂。他最后看了一眼变细如蛛丝的命线，伸手——最强的一剪，也是最后一剪。

## 分层反派设计（3层）

### 第一层·低级反派：赵屠（矿场管事，筑基中期）
- **核心欲望**：靠压榨矿场散修修炼，攒资源冲筑基后期，进青冥宗内门翻身
- **致命弱点**：贪婪不知收手。把所有人当矿挖，逼到绝路的人太多，恨意积累成反噬。他不知道矿场里藏着一个能剪他命的人
- **被击败方式**：被沈劫正面剪断命线，修为被吸干。死前才知道杀他手下的人就是他瞧不起的矿工
- **叙事功能**：展示金手指——爽感（吸筑基修为暴涨）和代价（命线变细）的第一次双重冲击

### 第二层·中期反派：周长青（青冥宗内门长老，金丹期）
- **核心欲望**：利用宗门体系垄断灵脉，把外门弟子当耗材吸取命元修炼。维持"宗门长老"的体面，实际是最大的吸血鬼
- **致命弱点**：他的修为来源见不得光（吸取弟子命元），最怕断命者的存在被上层知道——因为断命者能看见所有人的命线，他的秘密会暴露。所以他发现沈劫后不是直接杀，而是想控制利用
- **被击败方式**：沈虹查清他的命元吸取机制，在宗门大殿上当众剪断他的命线——所有被他吸过命的弟子命线同时松动，真相大白
- **叙事功能**：揭示宗门本质（更大的矿场），把"剪线"从个人恩仇上升到体制对抗

### 第三层·终极反派：天道（命数网的意志）
- **核心欲望**：收割所有修士的命数维持自身存在。整个修真界的灵气体系就是它编织的命数网，修士修炼=被养肥=待收割
- **致命弱点**：断命者是唯一能剪断它命数线的存在。它害怕的不是沈劫的强，而是沈劫的"剪"。所以它一直在追杀所有断命者——莫老就是被它废的
- **被击败方式**：沈劫以最后的命线为代价剪断天道之线。不是打败，是同归于尽式的终结
- **叙事功能**：主题升华。最强=最脆弱，终极一剪=自我消亡。把"越强越脆弱"的核心设定推到极致

## 黄金三章细化（第一幕前3章）

**ch01·冲突爆发**
- 前3句扔炸弹：沈劫被按在矿渣里，嘴里全是血和碎石，头顶有人在笑
- 300字内：冲突（灵矿被抢+被打半死）+主角（底层散修矿工沈劫）+金手指苗头（濒死时看见光线）+钩子（那根光线是什么）
- 章末钩子：沈劫伸手碰了一下那根从打手头顶延伸出的光线，光线断了，打手倒了

**ch02·金手指激活**
- 沈劫剪断第一个人的命线，热流涌入，获得练气一层修为
- 同时发现自己的脉搏变虚——代价的第一次暗示
- 莫老远远看着，说了一句意味不明的话
- 章末钩子：赵屠的手下发现矿工尸体，赵屠下令彻查

**ch03·短期目标明确**
- 赵屠放话三天后处理沈劫（怀疑他跟矿工死亡有关）
- 沈劫需要在这三天内变强到能逃出去
- 莫老点破："你能看见别人看不见的东西"
- 章末钩子：沈劫看向赵屠头顶——那里有一根比所有人都粗的命线，他咽了口唾沫
