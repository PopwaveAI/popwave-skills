# 第三章 · 巷战教学

垃圾堆里的腐臭味钻进鼻腔。

陈锋蹲在两栋楼之间的夹缝里，后背贴着潮湿的砖墙，手里的碎玻璃片已经染成暗红色。左臂的刀口还在渗血，刚才那一刀割得不深，但一直在流，像是故意提醒他还活着。

面板浮在视网膜左下方，半透明，数字安静地闪烁。

```
【杀戮神子系统·击杀结算】
击杀目标：2/3
当前等级：Lv.0
经验条：0/100
```

零。

他杀了那个拿刀的混混，又补了赶来的第二个人，但面板上的经验条纹丝不动。刚才那场死巷里的搏命割喉，系统判定连一点经验都不给。

陈锋扯了下嘴角。

——敌意判定。

他记得那个提示。偷袭没有明确杀意的目标，只给1点。那两个人冲过来的时候，脸上写的是“干他妈的东亚佬”，不是“我要杀了你”。系统不认这种情绪。

但第一个人——那个卡在窄巷口、用砍刀指着他、嘴里骂着“你他妈的找死”的壮汉——那是真的想砍死他。

系统认了。

```
【击杀结算】
目标等级：普通人 Lv.1
击杀方式：割喉（要害击杀）
获得经验：35 KEP（基础20 + 要害加成15）
经验条：35/100
```

三十五点。杀一个想要你命的人，够你升三分之一的级。

脚步声从左边的巷口传来，不是一个人，是两个。踩在碎砖和塑料袋上的声音，沉且稳，职业打手的步伐。

陈锋屏住呼吸，右手摸向兜里的折叠刀。

折叠刀没开刃，刀锋钝得像块铁皮。他之前用它拆过纸箱、削过木棍、撬过锁芯，从来没砍过活人。但在死巷里，这把刀跟着碎玻璃一起，划开了第一个人的喉咙。

他现在只剩这把刀了。

碎玻璃还在手里，但握久了手心出汗，会滑。折叠刀至少有握柄，不会掉。

脚步声在巷口停住了。

“这边有血。”一个沙哑的声音说，语气里带着烦躁，“妈的，这小子跑不远，肯定在哪个缝里缩着。”

“老大说你活要见人死要见尸。”另一个声音，年轻些，嗓门大，“他一个送外卖的华裔，能翻出什么浪？刚才要不是库纳尔大意——”

“闭嘴。”沙哑声音打断他。

陈锋在角落里一动不动。

两个人。

他的目光扫过面板——死亡嗅觉的感知范围是50米，但他现在不需要系统告诉他哪里有杀意。这两个人站的位置在左边巷口，宽约两米，地面有碎玻璃、破砖块、一根锈蚀的铁管。

铁管。

他慢慢把视线移到那根铁管上。长约一米五，拇指粗细，一头嵌在垃圾堆里，另一头露在外面。如果把它拔出来，就是个简陋但可用的武器。

比他手里的折叠刀强。

“搜。”沙哑声音说，“你去右边那条巷子，我走这边。”

脚步声分开了。

陈锋的脑子飞速运转。

两个人搜索，一个向左一个向右，意味着他们不是一起行动。如果他先处理掉左边这个沙哑声音——这是一个一对一的机会。但问题是，他动手的时候，右边的年轻人听到动静会冲过来。

他有三十秒。

最多三十秒。

陈锋开始算——不是靠情绪算，是靠经验和逻辑。

左手边的沙哑声音走得慢，脚步沉稳，说明有经验，不容易被骗。右手边的年轻人脚步轻快，容易被声音吸引。

三十秒够他做一件事。

陈锋无声地站起身，把那把折叠刀别在腰后，然后弯腰抓起那根铁管。铁管比想象中沉，生锈的表面硌手，但不影响握持。

他深吸一口气，然后——

“这边！”

他喊了一声，声音里带着慌乱，像个慌不择路的猎物。

喊完，他往右边的巷子跑。

不是直线跑，是贴着墙边跑，脚步声故意放得重但带着慌乱的气口。他需要那个年轻人听到他、追上他、以为他在逃。

而左边的沙哑声音听到喊声后，一定会掉头往这边赶。

制造时间差，制造信息差。

陈锋跑到右边的巷子尽头——是一个死胡同，三面是墙，墙上有一扇钉死的铁门。

他站定了，喘了两口气，不是真的累，是为了让呼吸声传出去。

果然，年轻人追到了巷口。

“找到了！这狗日的在这儿！”

年轻人冲进来，手里拎着一根甩棍。他看见陈锋背对着他、站在死胡同尽头，咧开了嘴。

“你跑啊，你他妈的——”

他朝陈锋冲过去。

陈锋等了三秒。

三秒——听年轻人脚步声的节奏判断距离。三米、两米、一米——

他猛地转身，身体向下一缩，铁管横向扫出。

年轻人完全没有料到这个动作。他的视觉追踪是“背对自己的逃跑者”，脑子里预设的是追上去从背后给他一棍子。

铁管砸在他的膝盖上。

咔嚓一声，位置对，发力对。膝盖骨碎了的触感从铁管传回来，年轻人惨叫着翻倒，甩棍掉在地上，双手去捂膝盖。

陈锋没给他机会。

铁管抡起，落下，一记闷响，声音不大但足够有效。

年轻人不动了。

面板弹出：

```
【击杀结算】
目标等级：普通人 Lv.1
击杀方式：钝器击倒（非致命）
敌意判定：明确杀意
获得经验：30 KEP

当前经验：65/100
```

六十五。

还差三十五。

脚步声从巷口逼近。

沙哑声音赶到了——他听到年轻人在喊“找到了”，然后听到惨叫声和闷响，现在正朝这边跑过来。但他显然比年轻人聪明，没有直接冲进巷子，而是在巷口停下了。

“阿杰？”他喊了一声年轻人的名字。

没有回应。

沙哑声音沉默了三秒。

陈锋能感觉到他的警惕——这个人不莽撞，不像库纳尔那样一上来就动手。他是个有经验的打手，知道判断局势。

但他判断的是“阿杰被袭击了，袭击者还在里面”。

他没想到的是——袭击者已经在巷口等他。

陈锋贴墙站在巷口右边，铁管握在手里，屏住呼吸。

沙哑声音慢慢靠近巷口，手里拎着一把砍刀——陈锋能听到刀锋刮到墙砖的声音，是窄巷里不好挥舞，刀尖蹭到了墙壁。

他在巷口探头了。

不是整张脸探出来，是微微侧身，先看半只眼睛和刀尖。

陈锋没动。

沙哑声音看到了巷子里的情况——阿杰躺在地上不动，死胡同里没有人。他的第一反应是袭击者翻墙跑了，或者藏在某个角落里等他进去搜。

他开始迈步走进巷子。

陈锋的呼吸控制得极稳——不是憋着气，是慢慢地小口呼吸，保证身体不动。他的眼睛盯着沙哑声音的后脑勺，听着他的脚步从身边走过。

在他完全走进巷子、背对自己的一瞬间。

陈锋动了。

铁管砸在后脑勺。

位置准、力量够。

沙哑声音往前踉跄了一步，但没有倒地——他回身就是一刀。

这一刀是最纯粹的本能反应，不瞄准，直接往身后横砍。陈锋侧身闪避，刀锋擦过他的胸口，割破了他的T恤，在皮肤上留下一条浅浅的血线。

好险。

陈锋退后半步，重新握紧铁管。

沙哑声音晃了晃脑袋，后脑勺挨的那一下让他有点发懵，但他没有倒下。他转过身，用砍刀指着陈锋，眼神里带着怒火和警惕。

“你他妈的是什么东西？”

陈锋没回答。

他在算。

面板上这个人头顶没有等级显示——系统只在击杀后结算，战斗中看不到对方的具体数据。但从刚才那一棍子的手感和反应速度来看，这个人比刚才那两个都强。

一个打手头目，大概在库纳尔手下干了好几年。

但也只是打手。

陈锋握着铁管，往前迈了一步。

沙哑声音也往前迈了一步——他不退，不退就说明他有自信。再加上知道自己今晚必须干掉这小子才能回去交差，他咬牙硬上。

两个人之间的距离缩短到两米。

沙哑声音先动手——他往前冲步，砍刀从右下方往上撩。一个很典型的街头劈砍动作，目标是陈锋的胸腹。

陈锋没躲。

他往左边横跨一步，铁管斜着往沙哑声音的前臂上砸。

砍刀落空，铁管砸中了手腕。

咣当——砍刀脱手。

沙哑声音惨叫了一声，下意识去捡刀。但陈锋不给他机会，铁管往他肩膀上猛砸一下、砸在锁骨上——咔嚓一声。

沙哑声音往后退了两步，捂着手腕，瞪大眼睛看着他。

“你……你怎么……”

陈锋没说话，他走过去，捡起掉在地上的砍刀，试了试刀柄的手感。比折叠刀好多了，重心还行，适合街头搏命，虽然是劣质货但锋利度够用。

他转身看着沙哑声音，举起砍刀。

“谁让你们来的？”

沙哑声音没回答，盯着他，嘴角在抽搐。

陈锋没有犹豫——他问了，对方不答，那就没必要继续问了。砍刀往下，干净利落。

面板弹出：

```
【击杀结算】
目标等级：精英打手 Lv.2
击杀方式：冷兵器致命（要害击杀）
获得经验：65 KEP（基础40 + 等级差+5 + 要害加成15）

当前经验：130/100

等级提升：Lv.0 → Lv.1
获得属性点：5
获得专长槽：1
```

经验条清零，数字跳动，一个全新的界面浮现在眼前。

陈锋站在巷子里，听着面板的提示音，看着那个升级动画一闪而过——界面上的字从灰色变成金色，边框泛起暗红色的光晕，像火焰在燃烧。

一股温热的力量从胸口涌出，像血液被点燃了一样。不是错觉，是真实的身体变化——他能感觉到力量在增长，肌肉、骨骼、神经都在被某种东西强化。

他低头看了看自己的手。

刚才还因为失血而发白的指尖，现在恢复了颜色。左臂上的刀伤口，居然在以肉眼可见的速度止住流血，周围的皮肤向中心收拢。

升级带来的身体修复。

面板弹出进阶显示：

```
【等级突破完成】
陈锋 Lv.0 → Lv.1
职业：游荡者（觉醒）
生命值：恢复100%
状态：清洁

【属性点分配】
力量：8 → 9
敏捷：12 → 14（+2）
体质：10 → 11
智力：12
感知：11 → 12（+1）
魅力：7

剩余属性点：0

【专长获取】
游荡者入门：阴影亲和
效果1：在低光环境下隐身判定+15%
效果2：暗影中的第一次攻击增加额外1d4伤害
效果3：在屋顶/墙角/阴影等区域移动速度+10%

【职业能力解锁】
暗影斗篷（未激活）
冷却时间：120秒
效果：进入阴影状态，持续3秒，期间移动速度+50%，下一次攻击必定暴击
```

陈锋看着面板上的数字，嘴角不自觉地扯了一个笑容。

力量加了1点，敏捷加了2点，体质1点，感知1点。12岁的身体原本不算强，但14点敏捷加上阴影亲和的判定加成，足以让他在街头巷战中占据优势。

他试了试握刀的手感——刀柄在手里更稳了，手腕的转动也更快了。甚至能感觉到周围空气的流动，耳朵里能听清三条街外轮胎碾过碎石的细响。

升级带来的变化是全方位的。

陈锋弯腰在沙哑声音的尸体上搜了搜——找到了一卷钞票，约莫三千卢比；一个小包，里面装着止血粉和绷带；还有一部翻盖手机。

翻盖手机的通讯记录里，最近三条都是同一个号码打来的，备注是“库纳尔老大”。

陈锋把手机揣进兜里，把钞票和药品收好，那把砍刀绑在腰后，然后打量了一下地面上的尸体。

他现在需要处理痕迹。

不是专业级的处理，没那个条件。但至少不能让这三个人大白天躺在巷子里被人发现，他需要时间，需要空间去查清楚库纳尔为什么对他这么上心。

陈锋把三具尸体拖到死胡同最里面，用垃圾堆里的破布和纸板盖上。血迹能清理的部分用水管冲了冲（巷尾有个破水龙头），掩不掉的部分就让自己记住——明天天黑之前，必须换位置。

做完这一切，他背上砍刀，翻过巷尾的铁门，进了另一片街区。

走了大约十五分钟，他在一栋废弃的住宅楼前停下——楼体外面写着拆迁编号，窗户全部钉死，三层以上没有门。这是阿米尔之前告诉过他的一个藏身处。

他用折叠刀撬开门锁，进了一楼。

电梯早就拆了，他走楼梯上到四楼，找了一个窗户朝南的房间，关上门，靠着墙坐下。

面板还在那里，安静地浮在左下角。

```
等级：Lv.1
职业：游荡者
经验条：30/250
六维：力9 敏14 体11 智12 感12 魅7
专长：阴影亲和（入门）
能力：暗影斗篷（CD 120s）
特性：死亡嗅觉（半径50m）
```

他伸手在空中点了点“特性”两个字，系统弹出更详细的说明：

```
死亡嗅觉（被动）
感知周围半径50米内对你有明确杀意的目标
数量：0（当前范围内无敌对目标）
精确度：可定位大致方向，无法获取距离
提示方式：面板左上角闪烁红点 + 轻微刺痛感
```

零个。

至少在这栋楼里，没有人想要他的命。

但出了这栋楼呢？库纳尔派了三个人来找他，现在三个人都没回去，库纳尔一定会派更多的人来。达亚尔家族不缺打手，他只是一个Lv.1的游荡者——还远远不够。

陈锋靠在墙上，闭着眼，让呼吸慢慢平静下来。

面板上的数字在黑暗中微弱发光。三十点经验，还需要杀多少人才能到Lv.2？

他睁开眼，看了看自己的手心。那把折叠刀的刀柄印还在，握久了留下了一道痕。

三个。

他杀了三个人。不是因为仇恨，不是因为愤怒，只是因为系统让他算了一笔账——杀他们能拿到经验、能活下去、能让库纳尔知道他不好惹。

理智的选择，但每杀一个人，他离“普通人”这个标签就更远一步。

陈锋把刀收好，站起身，走到窗边。

外面是新德里的夜景——远处高架桥上亮着灯光，近处的贫民窟里烧着垃圾堆，烟升起来，遮住了天上的星。

他要在库纳尔找到他之前，先找到库纳尔。

翻盖手机还揣在兜里，通讯录里库纳尔的号码安静地躺着。

陈锋想了想，拨了一个电话。

电话接通了。

“谁？”那头的声音粗哑，带着不耐烦。

陈锋开口了，声音比平时沉了一度，内容简短——“库纳尔，你的人回不去了。下一个轮到谁？”

他挂断了电话，把手机卡抽出来，掰断，扔出窗外。

然后他等着面板上那个红点亮起。

——死亡嗅觉有冷却的，但敌意判定没有。

现在，游戏开始了。
