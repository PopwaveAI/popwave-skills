# Step 4: 四图叠加推演剧情白描

剧情白描是plot层最重要的产出。必须用四图叠加推演法推演——禁止凭空一口气写。

## 推演方法
剧情白描 = L2单元卡 × 四张地图叠加 × 因果链推理

### 推演流程
1. 从Step 3幕序列提取L2单元卡（每幕拆成3-5个L2单元，每个单元覆盖2-3章）
2. 对每个L2单元卡执行6步推演：
   - 查地理地图 → 确定摄像头位置
   - 查力量地图 → 确定主角等级变化
   - 查势力地图 → 确定碰撞和演变
   - 查危机地图 → 确定倒计时推进
   - 因果链串联 → 生成叙事段
   - 标注信息差 → 增加"读者知道但角色不知道"
3. ch01已由seed的黄金首章产出，不重新推演。从ch01的章末钩子开始推演ch02。

## 写作要求
1. 整卷一口气写完：不拆章节/不拆标签
2. 以"给朋友讲故事"的方式：自然流畅，确保故事本身好看
3. 融入叙事：画面/情绪/内心活动都融入叙事，不单独标注
4. 消费世界构筑：把四张地图+人物包的关键信息融入故事流

## 内容要求
每段故事流必须包含：
- 事件推进：主角做了什么，发生了什么变化
- 信息差标注：(读者知道但角色不知道：XXX)
- 伏笔标注：(伏笔：XXX，chXX回收)
- 钩子标注：(钩子：XXX)

## 世界构筑消化规则
- 力量地图：战斗时融入等级/技能/金手指的具体表现（写感受不列数值）
- 地理地图：场景迁移时自然带出地标特征

#### **势力地图**

**势力碰撞和演变**

南海市的异常世界，并非简单的“人类 vs 诡异”，而是一个复杂的生态系统。

- **势力金字塔（4层）**

  - **中层（规则层）**：**市异常局·档案科**。掌握着所有异常案卷与历史真相，但选择中立，只记录不干预。局长办公室是真正的权力中心，但对普通人可见度极低。
  - **上层（超然层）**：**深夜茶馆的中立派异常者**。他们看清了世界真相，但选择共存或旁观，不参与人类势力纷争。**告死之诗（幕后金主组织）的触角**。隐藏在人类社会中，渗透高层，意图利用异常现象进行“世界重塑”。
  - **顶层（管理层/世界意志）**：**世界裂缝本身产生的“世界记忆”**。一种无人格但遵循规则（如：不产生大规模恐慌）的具象化力量，是“共情”能力的源头。

- **势力关系图**
  - **宣传科** **vs** **收容科**：表面合作，内部分歧。宣传科想“安抚”，收容科倾向“物理收容”。
  - **市异常局** **vs** **中立派异常者**：不合作、不敌对、默认共存。但一旦中立派插手人类内部事务，即刻转为敌对。
  - **告死之诗（渗透者）** **vs** **市异常局（底层）**：渗透者利用规则漏洞和AI系统，给底层制造麻烦，测试“世界裂缝”的反应。
  - **主角（陈渡）** **vs** **自己世界的常识**：最大的对抗来源于他需要在“体制内做事”与“理解并尊重异常者”之间找到平衡。

- **势力变化轨迹（第一卷结束时）**
  - 陈渡靠连续优秀KPI引起**局长办公室**注意，获得一次正式谈话资格。

  - **告死之诗**意识到“人类方出现了一个能听懂世界声音的年轻人”，将其列为重点观察/清除目标。
  - **中立派异常者**第一次主动向陈渡传递情报（通过茶馆老板娘柳姐）。
  - **收容科**从怀疑陈渡转为有限度合作，因为陈渡的悼词能降低收容物维护成本。

- **主角位置与迁移**
  - **初始位置**：金字塔底层的执行者，被各方势力视为棋子/工具人。
  - **位置迁移**：通过撰写悼词积累的声誉和认知深度，他在系统中获得“不可替代性”，并撬动了势力间的平衡。他不再是棋子，而是各方都想争取或除掉的“变量”。

    *   **低谷**（ch13）：陈渡被锁进收容科绝密室，面前是暴走的秤砣，AI分身给他下达“24小时写完整篇悼词”的死亡命令。但他同时接到叶晚的纸条：“账本被人毁了，秤砣执念无法化解。”

    *   **高潮**（ch19-20）：陈渡突然冷静下来，利用秦昭与AI分身配合时暴露的“后门规则”，写下一篇没有主语、没有固定形式的“声诉”——不是悼词，而是《关于本次暴走事件的责任认定说明》，他用共情翻译的血与泪，将秦昭的作案痕迹、被篡改的案卷记录、路灯案里的线索、地契交易的真相，全部转化为一段可以被AI系统识别的“事实”。局长AI检测到这篇“自洽性真相”无法被AI分身的后门程序过滤，直接启动全局审查。秦昭的身份败露。
    *   **变化**：空间裂缝被从未被写过的“真相”所闭合。陈渡从“宣传科科员”变成“可以跨界调用案卷的特级协调员”。他的存在被“告死之诗”确认锁定，成为组织的直接目标。

# 悼词第一句就惹怒了三百年老鬼

我，陈渡，考公上岸了。

从街道办调到市里，只高兴了三小时——人事科那位戴眼镜的大姐把我领到一栋灰色小楼前，指了指门牌：“市异常事件应急管理局宣传科，以后你就在这儿。”我盯着“异常事件”四个字，以为是个冷衙门，直到推开门看见墙上贴着《怨灵舆情处置流程图》和《高危收容物形象美化KPI考核表》。

“新来的？坐。”一个穿格子衬衫的中年男人头也不抬，把一沓案卷推过来，“我叫周海，你领导。今天有一单急活——康大爷，死在隔壁菜市场三百年了，怨气不散，把整条街的秤都搞失灵了。你给他写篇悼词，用官方话术安抚一下，争取让他今天配合收容。”

我刚张嘴想问问流程，他已经转身走了。墙角的风扇嗡嗡转着，吹得案卷边角往上翻。我翻开第一页，一张惨白的老脸照片贴在上面，眼睛直勾勾盯着我。

“陈渡，”我对自己说，“这活儿不难，写个悼词还能难到哪去？跟街道办写退休干部追悼词差不多。”

【警告！系统激活】

【检测到宿主首次接触怨灵级宣传任务，正在扫描……】

【核心能力：共情翻译——能理解怨灵执念，并转化为有效话术】

【当前任务：为“康大贵”（枉死类怨灵）撰写悼词】
【任务时限：24小时】
【失败后果：怨灵将入侵宿主梦境，强制体验执念原貌】

我眼皮跳了一下，盯着这串凭空跳出来的蓝色字体。手机没收到任何通知，但字体就浮在我眼前，风吹不散。

“系统？”我小声试了试。

没回应。但这玩意儿像个倒计时炸弹，已经挂在上方角落：【23:58:21】。

算了，先干活。我打开电脑，建了个Word文档，把照片扫描进去。然后从案卷里翻出康大贵的生平资料——康熙年间杀猪匠，因为缺斤短两跟买主吵架，被菜刀砍死在摊位前。三百年来一直在菜市场转悠，谁短了他的秤他就闹谁。

我扶了扶眼镜，盯着那张惨白的老脸。

“康大爷，”我清了清嗓子，对着空气开口，“我是市异常局宣传科的，叫陈渡。您的案子现在由我负责，我今天中午之前把悼词写完，您先别激动，行么？”

办公室没动静。窗外的蝉叫得正响。

我点点头，把自己当成在街道办写讣告，开始敲第一段话。

“尊敬的康大贵先生，您一生勤勉，敬职敬责，虽因意外离世，但您的精神将永远激励后人……”

噼里啪啦敲完第一句，电脑屏幕突然黑了。

灰白色的光从屏幕里透出来，越来越亮，一堆字符像是被什么力量撑得往外涌——那个惨白的老脸直接浮在屏幕表面，像是泡在水里的死人脸，鼓胀、模糊，眼眶里全是黑色的汁液。

“我死得冤啊！！”

整栋楼仿佛震动了一下。墙上挂的文件哗啦啦掉了下来，头顶的日光灯管开始闪。

“勤勉？敬职敬责？”那张脸扭曲着，“我缺斤短两怎么了？那个买主先骂我黑心肠，我气不过才拿刀吓唬他——结果他抢了我的刀，把我抹了脖子！三百年了，没人给我写过一句公道话！你们这些当官的，就知道欺负老百姓！”

我后背的汗湿透了衬衫领子。

但我也当过三年街道办的人，最擅长的就是跟老同志打交道。

“康大爷，”我深吸一口气，把声音压得平和，“对不住，是我写得不好。您别急，咱重来。”

我伸手把显示器的光调暗了些——虽然他这张脸看着确实瘆人，但咱得保持专业。

“要不这样，您说一句，我记一句。把来龙去脉说清楚，我给您写一个能上报的版本。”

那张脸愣住了。

黑色汁液慢慢往回渗，恢复成一团模糊的肉色。“你……你不是来敷衍我的？”

“我是官方工作人员，”我从抽屉里翻出工作证，对着屏幕晃了晃，“这是我的证件编号，你随时可以查。今天这篇悼词，我一定写得让你满意。”

【怨灵情绪指数：暴怒（87%）→警惕（62%）】
【你的第一轮共情触发，对方愿意配合】

我松了口气，赶紧把打印机的纸抽出一张垫在键盘下面。康大贵沉默了半晌，用那种又沙又哑的声音开始讲他的“冤屈”——

买主姓刘，是个杀鸡的贩子，那天他卖了只瘟鸡给康大贵的老婆，康大贵气不过才在秤上做手脚。两人从骂战打到夺刀，康大贵被刘一把推倒，后脑勺正好磕在自家砧板上，当场毙命。

“所以你杀鸡贩子拿刀抹你脖子是假的。”我说。

“我……我是被磕死的啊！但这很丢脸！我一个大老爷们被一只瘟鸡气死在菜市场，说出去谁愿意听？”怨灵语气里全是恼羞成怒，“我编个‘被菜刀抹脖子’的故事，至少听起来壮烈点！”

我忍住没笑。老同志的虚荣心，这我懂。

“成，我就把这个真实经过写进去——但是，”我抬眼看他，“您得答应我，悼词写完，就去收容科报到，别在菜市场闹了。”

“你写得好再说。”

我盯着屏幕，把这六段话从头到尾捋了一遍。办公室静得只剩主机风扇在喘。康大贵那张脸忽明忽暗，像没调好的老电视。

敲下最后一行时，屏幕恢复正常。Word文档自动生成标准公文字体，宋体小四，行间距1.5倍。

【悼词完成度：93%（优秀）】
【“康大贵”怨灵狂暴指数：72%→23%】
【KPI结算：宣传工作资质积分+45，当前总分：45/100】

我把文档打印出来，去隔壁茶水间接了杯水。回来的时候，桌上多了一张光盘——周海放的，上面贴了张便签：“行啊，新来的，第一个就搞定了。下个月收容科要腾床位，明天还有三个——一个会跳舞的路灯，一个收件必死的快递柜，还有一个在火锅里骂人的鬼。”我盯着便签的手指在发抖。

就在这时候，手机突然响了。屏幕上跳出来一条短信——档案室发的。

“陈渡同志，你下午调取的康熙年间菜市场秤砣校准记录，有个情况需要补充告知：那个秤砣……今天早上被收容科抬进来了。它成精了，嘴里一直在念‘康老头欠我七百八十五斤肉，什么时候还？’”

我握着手机，看向墙角。

一台半透明的AI分身正飘在那儿——是周海的电子监察系统。它用那种完全没有起伏的电子音说：“陈渡，明天早上八点，悼词采访。你还有十四个小时做案卷预习。”

AI分身的轮廓渐渐消散在日光灯的白光里。

我蹲下来抱起我的猫。那团软乎乎的黑毛家伙正趴在办公桌下面，用它那双金黄色的眼睛望着我，喵了一声。

“完了，”我苦笑，“明天采访的那盏路灯，它给我发什么歌？”

话音未落，办公室的电话响了。

来电显示：异常事件应急管理局·档案科。

我犹豫了三秒，接起来。

“喂？”

电话那头是一个女声，稳定得像是录音：“陈渡同志，给你提个醒——你说的那盏路灯，它在收容科待了两个月，刚刚学会放《最炫民族风》。而且，”她顿了顿，“它的下一个目标是学会唱《死了都要爱》。”

我沉默了五秒钟。

“这有什么问题？”

“问题在于，它每一首歌都是以高强度电能脉冲的形式向外广播的。昨天晚上，整个收容区的走廊灯跟着它一块儿亮了，值班科的同志跳了四小时的广场舞才被关掉。”

我挂断电话，盯着天花板看了一会儿。

“行吧，”我对康大贵的照片说，“至少你比路灯正常。”

照片里那张惨白的老脸，嘴角竟然微微扬起了一点。
