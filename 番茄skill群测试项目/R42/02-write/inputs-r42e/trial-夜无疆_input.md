# 夜无疆 · ch03 剧情梗概（中性·不带笔触）

> 说明：本梗概只描述发生了什么，不含任何笔触特征（句式/感官/叙事距离）。用于测试DNA驱动笔触的变量隔离。

## 本章核心事件
秦铭在浅夜独自前往野外山林，经历人面鹫袭击后，从树洞中捕获变异红松鼠并掏空存粮，满载而归，在山顶望向群山深处的未知地带。

## 场景序列

### 场景1：出发（浅夜·村外雪地）
- 浅夜到来，秦铭全副武装（猎叉、短刀、弓箭、兽皮袋）出门
- 目标是山林边缘，距离约四里路
- 在积雪中艰难跋涉

### 场景2：人面鹫袭击（野外·积雪林地）
- 一只人面鹫（食腐猛禽）从空中袭击秦铭
- 秦铭用弓箭反击，人面鹫受伤后飞走
- 秦铭判断它不会善罢甘休，保持警惕

### 场景3：第一个树洞（山林边缘）
- 到达目的地，找到一个树洞
- 树洞口结着冰花，说明里面有活物（冰霜是生物气息凝结）
- 秦铭判断这个树洞的生物不在家

### 场景4：地光与发现（密林深处）
- 地光突然出现，照亮密林
- 秦铭趁机观察，发现另一个结着冰花的树洞
- 树洞里传出窸窣声，有活物在内

### 场景5：捕获变异红松鼠（树洞）
- 秦铭攀上树，用猎叉堵住洞口
- 从洞中拖出一只变异红松鼠，两斤多重，皮毛发光
- 红松鼠挣扎但被制服

### 场景6：掏空存粮（多个树洞）
- 树洞内有野核桃、栗子、红枣等存粮
- 秦铭连找四个树洞，掏空全部存粮，共八斤多干果
- 变异红松鼠看到巢穴被搬空，气得僵挺不动

### 场景7：山顶远眺（山顶）
- 秦铭来到山顶
- 望向群山深处的明灿地带（未知危险区域）
- 内心有所触动，但决定先回去

### 场景8：村民议论（双树村）
- 双树村村民看到秦铭独自外出冒险归来
- 村民议论秦铭胆大，也议论野外危险
- 有人对秦铭表示担忧

## 关键人物状态
- 秦铭：主角，少年，十六七岁，孤身一人生活，刚大病一场，弓箭高手
- 陆泽：秦铭的朋友/邻居，关心秦铭
- 梁婉清：陆泽的妻子，担心家里食物不够
- 周阿婆：已去世的老人，曾接济秦铭
