# 玄鉴仙族 · ch03 剧情梗概（中性·不带笔触）

> 说明：本梗概只描述发生了什么，不含任何笔触特征（句式/感官/叙事距离）。用于测试DNA驱动笔触的变量隔离。

## 本章核心事件
李项平发现一面古镜（鉴子），带回家后引发父亲李木田的警觉，夜间布防时从瓜田捞出一个偷瓜的族人。

## 场景序列

### 场景1：打渔归来（日间·村口→家）
- 李项平打渔归来，在村口遇到女孩田芸
- 李项平送鱼给田芸
- 回家路上，怀里揣着白天捡到的一面古镜（鉴子）

### 场景2：家庭介绍（日间·家中）
- 李家有四兄弟：李长湖（长子）、李通崖（次子）、李项平（三子，主角）、李尺泾（四子）
- 父亲李木田，村里唯一出过远门的人，从过军、杀过人，回村后买地成为大户
- 李木田不满足于种田生活，心气高

### 场景3：镜中视角（日间·田间）
- 鉴子内有一个存在（陆江仙），附在镜子里
- 陆江仙感受到村口大槐树方向有牵引力
- 陆江仙判断这是一个普通村落，没有修仙者
- 陆江仙观察李家四兄弟的气质

### 场景4：夜间献镜（夜间·家中）
- 夜晚，李项平把鉴子拿给父亲李木田看
- 月光汇聚在镜面上，形成月晕
- 李木田大惊失色，杀意浮现
- 李木田让李项平叫哥哥们拿刀出来

### 场景5：布防抓人（夜间·瓜田）
- 李通崖判断是仇家上门，分发布防
- 李木田走到屋前瓜田，弯腰从瓜田里捞出一个人来
- 是族人李叶盛，只想偷个瓜解馋
- 李木田冷脸处置，收刀走人

## 关键人物状态
- 李项平：主角，少年，打渔为生，捡到鉴子
- 李木田：父亲，从过军，杀过人，警觉性高
- 李通崖：次子，有判断力，负责布防
- 陆江仙：镜中存在，观察者视角
- 田芸：村中女孩，与李项平有交情
- 李叶盛：偷瓜的族人
