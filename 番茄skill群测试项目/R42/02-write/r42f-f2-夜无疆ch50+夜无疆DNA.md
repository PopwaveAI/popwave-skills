# 第五十章 天光

浅夜降临，秦铭坐在房间中，铜盆里的太阳石散发出最明亮的光，整间屋子都被照得通透。他盘坐在炕上，周身覆盖一层明灿灿的光晕。

瓷坛里的淡金色液体泛着微光，蚁巢精华散发出一股说不清的气息，像是草木的清香，又像矿石的冷冽。秦铭将瓷坛打开，倒进白碗中，液体在碗底轻轻晃动，泛起细密的波纹。

他端起来，一饮而尽。

液体入喉的瞬间，秦铭全身迅速发烫，五脏六腑像是在燃烧，每一块骨骼都发出细微的咔嚓声，筋络在体内绷紧又放松，血液奔涌如江河。他很快冲出房间，站到院中。

寒风吹过来，落在身上却变成了暖风。

雪花飘落，触到他的皮肤时如同春雨般温柔，没有一丝寒意。

秦铭站在雪地里，浑身开始发光。先是微弱的银光浮现在体表，而后银色光晕渐渐凝实，变成一层银色的泥浆般的东西，覆盖全身。紧接着，无数细密的金色针芒从体内刺出，在银泥中游动穿梭，像是有灵性的金线在织衣。

金针穿线，在银泥中不断游走，最终形成一幅完整的“金缕玉衣”。

秦铭动了。

他开始演法，手印跟着帛书上记载的新生法不断地推进，动作行云流水，没有一丝停滞。

轰——

心海中有一道光照出来。

那是一道纯粹的光，穿透了五脏六腑，穿透了筋络骨骼，穿透了他整个人。他的眼睛最先承受不住，两道神秘的眸光激射而出，射在院墙的青石上，留下两个焦黑的孔洞。

全身上下，无处不生天光。

秦铭的身体像是变成了一盏灯，从内到外都在发光，照亮了整个院子。雪地被映得通明，连远处树木的阴影都被驱散。

光芒渐渐内敛。

秦铭盘坐在雪地上，天光一寸寸收回体内，最终归于平静。他睁开眼，双眼明亮清澈，整个人散发出一种平和自然的气息。

体悟如泉水般涌来：坚持练帛书，就能诞生天光。帛书上的新生法没有骗他，从头到尾都是真实的。

但天光劲呢？

秦铭皱眉。天光有了，可如何将天光转化成劲，他完全摸不着头脑。帛书的前两页只有新生法，没有提及具体的攻击法门。

只能等今夜入睡后，在梦中翻看帛书的第三页。

秦铭起身，走到屋中，将铜盆端到院中，抓起那柄长柄乌金锤。这柄锤子太沉，他用着不顺手，不如砍柴刀来得利索。

他丢掉乌金锤，重新握住砍柴刀。

雪地上，他开始演练搏杀法。刀光在夜色中闪动，每一刀都带着风声，刀锋划破空气，发出尖锐的嘶鸣。

秦铭感觉自己血肉间有光在跳动，像是有东西要喷薄出来。他疑心那就是天光劲，但还差临门一脚，始终没能彻底激发。

他一遍一遍地练，汗水浸透了内衫，热气从头顶蒸腾而起。

直到浅夜结束，秦铭才停下。

他喘着气，看着手中的砍柴刀，刀身上已经出现了细微的缺口。这柄刀太普通，根本承受不住他现在的力量。

敲门声响起。

“小秦？在不在？”

刘老头的声音，带着几分焦急。

秦铭打开院门，刘老头站在门外，裹着厚厚的兽皮袄，脸上是那种说不清道不明的表情。

“你……成功了？”刘老头上下打量着秦铭，眼珠子都快瞪出来，“你就这么弄成了？一点准备都不做？”

秦铭点头：“成了。”

“三次新生！”刘老头声音发颤，“你小子是真的能折腾，我活了这么大岁数，头一次见人这么快就完成三次新生的。不用准备什么灵药辅助，不用找什么特殊地点，就这么在自家院子里……”

秦铭打断他：“你为什么还不喝那灵性物质？”

刘老头沉默了一会儿，叹了口气：“我怕。”

“怕什么？”

“我年纪大了，身体底子本来就薄。”刘老头低声道，“初次新生就拦住了九成人，到了多次新生，每多一次，难度翻倍。我怕我喝了，扛不住，死在当场……”

秦铭没说话，等着。

刘老头在院子里走了几步，像是在回忆什么：“我年轻的时候进过山，运气不好，被一头变异兽咬伤了腿，差点死在里头。后来拜了个师父，学了些本事，结果没几年师父就横死了，说是惹上了不该惹的东西。再后来，我找了几个朋友，准备联手弄些神秘物质，结果遇上那头名兽，除了我，全死了。”

他苦笑：“我这辈子总是一波三折，该成的事，总在关键时刻掉链子。”

秦铭道：“那是以前。”

刘老头抬起头，看着他。

“以前总掉链子，是因为没有真正的机遇。”秦铭认真道，“现在机遇就在眼前，你还在怕什么？”

刘老头的眼神变了变，过了好一会儿，才轻声道：“你还记得……染血的风筝吗？”

秦铭的瞳孔骤缩。

“当年我在村口的火泉边，亲眼看到一只风筝飘到了双树村的中心。”刘老头压低声音，“那只风筝上染满了血，红得刺眼。后来我就开始倒霉，一直倒霉到现在。我总觉得，那只风筝是不祥的兆头，只要它出现过，我这辈子就好不了。”

秦铭盯着他：“你确定是在村口的火泉看到的？”

刘老头点头：“就在东边的火泉旁，我记得很清楚。”

秦铭没有再追问，而是走进屋中，将那坛蚁巢精华端出来。他倒了一碗，递到刘老头面前：“喝下去。”

刘老头看着碗中淡金色的液体，手在发抖。

“我……”他嘴唇哆嗦，“我真的可以？”

“你有潜力。”秦铭说，“你只是没试过。”

刘老头深吸一口气，接过碗，一饮而尽。

片刻后，他的全身开始滚烫，体内有光雾蒸腾而起，整个人像是被架在火上烤。他咬着牙，硬生生没有叫出声。

过了将近半炷香的时间，光雾渐渐散去。

刘老头睁开眼，眼眶发红，声音沙哑：“我……我成功了……”

秦铭点头，没有多说什么。

刘老头裹紧兽皮袄，转身往院外跑，跑了几步又回头：“小秦，谢了！”

他奔走如飞，整个人像是变了一个人。

秦铭站在院中，目送他消失在夜色里。

深夜。

秦铭半梦半醒，眼前出现一道模糊的身影——那是幼年的自己，正站在一座破旧的房间中，看着一张帛书。帛书被一双粗大的手翻开，手指上全是茧子，衣服也打了不少补丁。

那只手的主人，五十岁出头，两鬓斑白。

是爷爷。

秦铭的心跳加快，他想看清那张脸，但画面始终模糊。帛书翻到了第三页，上面密密麻麻写满了字迹，内容都是新生法的延续，提到了五感和本能直觉，是第四次新生的方向。

但秦铭最关心的天光劲，一个字都没有提。

他差点惊醒。

就在他快要放弃的时候，帛书页面最下角，有一行小字批注，字迹潦草，像是随手写的：

“天光都出来了还练不出天光劲？一个字，融。”

秦铭猛然睁开眼。

他坐在炕上，浑身是汗，脑中不断回响着那个字——融。

融。

天光在体内，用力，将它融入每一寸血肉、每一块骨骼、每一条筋络中。

秦铭翻身下炕，连鞋都没穿，直接冲到院中。他开始演练帛书新生法，动作比之前更加流畅，更加迅疾。

体内的天光开始躁动，像是被激活的猛兽，在血肉间冲撞。

秦铭屏住呼吸，将意念集中在指尖。

指端的皮肤开始出现神秘涟漪，一圈一圈的波纹向外扩散，像是石子投入水面的涟漪，带着微弱的光芒。

嗡——

一道细微的震动从指尖发出，天光劲崩现。

秦铭的指端触及刀柄，那柄砍柴刀像是被黏住一样，牢牢贴在他的手掌上。他微微发力，天光劲顺着刀身传导出去，刀身上出现一道缺口。

咔——

砍柴刀裂开了。

秦铭看着手中的断刀，嘴角露出一丝笑意。

成了。

天光劲，练成了。

他捏着半截刀身，站在院中，整个人气宇轩昂，和之前判若两人。

远处传来嘈杂声，村口方向有马蹄声响起。

秦铭抬眼，看到金鸡岭的人来了，正在村里招新生者。

他想了想，转身回屋，从床底摸出一块拳头大小的羊脂玉铁，掂了掂分量。

是时候打造一柄真正的兵器了。
