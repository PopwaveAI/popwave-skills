# 第十四章 夜猎

浅夜刚至，秦铭背起兽皮袋，猎叉斜挎腰间，短刀别在腿侧，箭壶挂满二十支箭。

推开门，冷风灌进来。

积雪没过脚踝，每踩一步都陷进去大半条腿。秦铭压低身子，踩着昨晚踩实的旧脚印往村外走。

四里路，走得浑身冒热气。

山林边缘的积雪更深，快到膝盖。枯树在月光下投出扭曲的影子，树枝上的积雪偶尔簌簌落下。

秦铭停下，耳朵动了动。

风里有股腐臭味。

【生物感知：食腐猛禽残留在空中的气味分子】

他抬头，夜空里盘旋着一道影子。

人面鹫。

这种猛禽翅膀展开有两米，喙能撕开野猪皮，最恶心的是——它长着一张皱缩的人脸，眼珠浑浊，盯着活物就像盯腐肉。

秦铭拉弓搭箭。

箭尖对准天空，手指搭在弓弦上，没有立刻松。

人面鹫盘旋两圈，俯冲。

翅膀撕裂空气，带起一阵腥风。

秦铭眯眼，箭头跟着黑影移动。三十步、二十步、十五步——

松弦。

箭矢扎进人面鹫的翅膀根部，带出一蓬血花。

猛禽发出一声嘶哑的尖叫，扑腾着往远处飞走，落了一地黑羽。

秦铭没追击。

“麻烦了。”

他重新搭箭，箭尖跟着天空转了两圈。确认没人面鹫回来，才往前继续走。

树洞在密林深处，被藤蔓挡住一半。

秦铭拨开藤蔓，蹲下看树洞口。

冰花。

灰白色的冰花结在洞口边缘，像霜打的蜘蛛网。这种冰花是活物呼吸凝结的气息，树洞里面一定有东西。

他伸手探进去，摸到干草和兽毛。

空的。

“出门了。”

秦铭拍拍手，站起来。走五步，又找到一个树洞。

同样结着冰花。

再伸手探，还是空的。

这不对。

方圆五十步内至少有七八个树洞，如果都是空的，那说明这片区域的猎食者太凶猛，小动物全吓跑了。

或者——

地面震动。

非常轻的震动，轻到几乎感觉不到。但秦铭趴在地上，耳朵贴雪，听到地底传来沉闷的声响。

地光。

这片区域的地光提前来了。

他爬起来，往声音传来的方向跑。

密林深处，地面裂开一道缝隙，淡蓝色的光芒从裂缝中透出来，把周围的树干和积雪映成诡异的蓝白色。

秦铭盯着那道裂缝看了三秒，转身往另一棵树跑去。

这棵树比别的大一圈，树干要三人合抱，树皮上爬满老藤。树洞在半人高的位置，洞口结着厚厚一层冰花。

里面传出窸窣声。

秦铭抽出猎叉，猫腰凑近洞口。

窸窣声停了。

停顿两秒，又响起来。

他握紧猎叉，猛地捅进洞口，用叉齿抵住洞内。洞里传出“吱吱吱”的叫声，有什么东西在挣扎，爪子挠树洞壁，发出刺耳的声响。

秦铭一只手按住猎叉，另一只手摸进洞。

热。

洞里比外面暖和至少十度，空气里有股干果和兽毛混合的气味。手指碰到毛绒绒的一团，那东西想缩回去，但被猎叉抵着，只能让他抓。

抓住后颈皮，往外拖。

一只变异的红松鼠被他拽出洞。

那松鼠比普通松鼠大三倍，体长超过两尺，尾巴蓬松得像个火把，皮毛在月光下泛着淡淡的红光。四颗门牙露在外面，又尖又长，能一口咬断普通人的指骨。

【变异红松鼠：普通松鼠受到地光辐射产生的变异种，皮毛可入药，肉量中等，爪子与门牙具有弱穿透性】

秦铭把红松鼠按在地上，用膝盖压住身体，麻利地从腰间抽出一根细麻绳，将松鼠的嘴巴和前爪捆在一起。

松鼠挣扎了两下，不动了。

得手。

他扔进兽皮袋，重新看向洞口。

里面还有东西。

伸手进去一掏，是一把野核桃。再掏，是一把板栗。继续掏，红枣、山核桃、杏干，一捧接一捧。

秦铭眼睛亮了。

这是红松鼠的存粮。

他找了半天，确定了这只红松鼠的巢穴分布——整整四个树洞，全被这畜生填满了干果。

秦铭一个树洞一个树洞地掏。

第一棵树洞：野核桃两斤半，板栗一斤二两。

第二棵树洞：红枣一斤七两，山核桃八两。

第三棵树洞：杏干一斤，松子六两。

第四棵树洞：板栗三斤，红枣一斤四两。

掏完最后一个树洞时，那只被捆着的红松鼠醒了，看到他的兽皮袋鼓得满满当当，再看到那四个树洞里空空如也，顿时浑身僵住，四只小腿蹬了两下，不动了。

气得僵了。

秦铭笑了声，把兽皮袋扎好口，扛上肩。

八斤多干果，换算成银两，至少能卖二十两。

他爬上山顶，站在最高处，望向群山深处。

那里有一片明灿灿的光带，像流淌的星河，在黑暗中格外刺眼。

秦铭看了很久。

那片光带是天陨石的辐射区，里面藏着宝物，也藏着危险。据说每晚，都有人在那片光带里升一阶，也有人永远留在那里。

他握了握拳。

又松开。

“还不到时候。”

扛起兽皮袋，下山。

双树村的灯火在夜色中显得昏黄，秦铭走到村口时就听到有人在说话。

“这小子又出去打猎了？”是陆泽的声音。

“可不是。”另一个人接话，“大半夜一个人往山林跑，真是不要命。”

“他刚病好，这身子骨也折腾得起？”

“他家那个情况，不吃肉怎么养身体？”

秦铭走到人群前时，几个人都噤了声。

陆泽看到他肩上的兽皮袋鼓鼓囊囊，愣了愣:“真打着了？”

“嗯。”

“什么？”

“红松鼠，八斤干果。”

陆泽嘴张了张，半晌才道:“你小子胆子真大。”

秦铭点点头，从袋子里掏出两把板栗，塞给陆泽:“带回去给嫂子尝尝。”

陆泽想推，但看到秦铭的眼神，还是接下了。

秦铭转身走回自己家。

推开门，屋子里黑漆漆的，只有灶台里的余烬还在闪着一丝暗红。

他放下兽皮袋，坐在门槛上，望着远处群山深处那片明灿灿的地带。

今晚只是开始。

明天，他要往更深的地方去。
