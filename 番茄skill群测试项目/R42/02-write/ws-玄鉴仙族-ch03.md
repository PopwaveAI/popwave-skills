# 第二十三章 夜镜

日头还没落尽，李项平走在村道上，竹篓里几条鲫鱼甩着尾巴。

田芸坐在自家门槛上择菜。

“给你。”李项平从篓子里拎出两条最大的，扔过去。

鱼在空中划过一道银光，落在田芸脚边。

“又不要钱的？”

“多得很。”

田芸捡起鱼，抬眼看他:“你今儿捡到宝了？”

李项平眉头一动:“啥？”

“看你那眼神，”田芸拍拍裙子站起来，“跟偷了鸡似的。”

李项平没接话，摇了摇头，大步朝家走。

怀里的东西硌着胸口——那面青铜鉴子，巴掌大，表面长满铜绿，可擦干净的地方照出人影，清晰得有点邪门。

他没跟任何人说。

这鉴子是在河滩石缝里捡到的。掀开石头时，底下没有淤泥，没有水草，就躺着这面镜子。太阳照上去，镜面没有反光，像把所有光线吞了进去。

李项平直觉这东西不一般。

推开院门，院子里热气腾腾。

大嫂坐在灶台边添柴，二嫂端着盆从井边走过来，三嫂在屋檐下晾衣裳。

“三弟回来了。”大嫂朝堂屋努努嘴，“爹正念叨你呢。”

堂屋里，李木田坐在主位上喝茶。

李长湖站着，李通崖坐着，李尺泾捧着本书看。

四个儿子。

李长湖三十出头，精瘦结实，一张脸晒成古铜色，常年在地里泡出来的。他是李家的顶梁柱，地里的活一人顶仨。

李通崖二十五六，身板比大哥壮一圈，脚上常年绑着沙袋。这人性子沉稳，遇事不慌，村里人都说像李木田年轻时候。

李尺泾最小，十七八岁，村里唯一的读书人。去年院试没过，整天抱着圣贤书，想明年再考。

李项平是老三，二十岁，高瘦，手上的茧子是拉渔网磨出来的。

“回来了？”李木田放下茶碗，“今儿收成咋样？”

“够吃。”李项平把竹篓放在墙角。

李木田盯着他看了两眼，没再说啥。

晚饭吃得安静。

李木田吃饭不爱说话，一大家子人都不吭声，只听见碗筷碰撞声。

李项平扒了几口饭，往嘴里塞了两块红烧肉，烫得嘶了口气。

“慢点吃。”李通崖递了碗凉水过来。

【胃里温热，食物落进空荡荡的肠胃】

饭吃完，大嫂们收拾碗筷，几个男人各干各的。

李项平走到院子里，月亮已经升起来。

他坐在井沿上，掏出那面鉴子。

月光照在镜面上，镜面轻轻一震。

刘项平看见自己的脸映在里面，可那表情不像他自己——嘴角微微上翘，像在笑，又像在看。

“哐当！”

李项平手一哆嗦，鉴子掉在青石板上。

他捡起来再看，镜子里又恢复了他的脸，面无表情。

屋门推开，李木田走出来:“干啥呢？”

李项平犹豫了一下，还是把鉴子递过去:“爹，你瞧瞧这个。”

李木田接过来，借着月光一看。

月光落在镜面上，不再是反射出去，而是像水滴一样被吸进去。

镜面上浮起一圈月晕。

淡淡的，像雾里透出的光。

李木田脸色猛地一变。

他的手攥紧鉴子，指节发白，双眼死死盯着那圈月晕，瞳孔微缩。

“这东西哪儿来的？”

“河滩捡的。”

李木田深吸一口气，转身大步走进屋:“老大老二老四，都出来！拿刀！”

堂屋里一阵乒乒乓乓。

李长湖拎着砍柴刀冲出来，李通崖从墙缝里抽出把七寸匕首，李尺泾抓着把鱼叉，嘴哆嗦着:“爹，啥事？”

李木田没说话，把鉴子揣进怀里，推开后门。

后门通向一片瓜田。

月光照在瓜藤上，叶子泛着银灰色。远处蛙鸣一阵接一阵。

李木田弯下腰，贴着瓜田走。

【脚步声在泥土上很轻，只听见衣服摩擦藤叶的声音】

李项平跟在后面，心跳咚咚响。

李通崖从侧面包抄过来，匕首反握。

李长湖守在瓜田另一头，砍柴刀横在身前。

李尺泾蹲在墙角，鱼叉抖着。

李木田走到瓜田中间，停下来。

他弯腰，伸手。

手探进瓜藤深处，摸到一个软乎乎的东西。

那东西一颤，想要跑。

李木田手一紧，揪着衣领提了起来。

是个人。

瘦小的年轻男人，穿着青布短褂，脸上糊着泥土，嘴角还粘着瓜瓤，吓傻了。

“李...李叶盛？”李尺泾叫出声。

李木田松开手。

李叶盛扑通跪在地上，腮帮子鼓着，嘴里的瓜还没咽下去。

“木田叔，我，我就是想尝尝...”

李木田蹲下来，盯着他的眼睛:“偷瓜？”

“饿...饿得慌...”李叶盛声音发颤，“家里揭不开锅了，我就...”

“饿了不来要？”

李叶盛头低下去，不敢说话。

李木田站起来，手按在腰间的刀柄上。

刀柄被握得发白。

李叶盛吓得脸色发白，鼻涕眼泪一起流:“木田叔，我下次不敢了，我真不敢了...”

李通崖走过来，看了老爹一眼:“爹，是自家人。”

“自家人？”李木田冷哼一声，转头看向李叶盛，“你知道我今天为啥出来？”

李叶盛愣住。

“我闻到你身上的味了。”李木田蹲下来，“你在瓜田蹲了多久了？”

“两......两个时辰。”

“听着。”李木田压低声音，“你今天来偷瓜，我不杀你。”

李叶盛拼命点头。

“但你要记住。”李木田凑近他，声音变了调，“李家往后，不会再有这种事。”

他站起来，拍拍手:“滚。”

李叶盛连滚带爬跑了。

院子里又安静下来。

李木田掏出鉴子，月光照上去，那圈月晕还没散。

“爹，这到底啥东西？”李长湖问。

“不晓得。”李木田低头看着自己的影子倒映在镜面上，“但沾了这玩意儿，麻烦会来。”

李通崖问:“那咋办？”

李木田沉默了一会儿，把鉴子塞回怀里:“先留着。”

他转身看着四个儿子:“今晚起，轮流守夜。一有动静就叫。”

李项平站在院子里，夜风吹过来。

【风从领口灌进来，凉意顺着脊背往下滑】

他握紧拳头。

这镜子，到底什么东西？

镜子里，陆江仙正注视着这一切。

【凡人村落，灵气稀薄，无修炼者迹象】

【但村口大槐树方向，有微弱的牵引力】

【是某种法器的残留波动】

【这面鉴子，不简单】

陆江仙的目光穿透夜色，落在李木田身上。

这凡人警觉性极高，杀过人，闻得出活人的气味。

李家这门，不好混。

可村口那棵大槐树，藏着东西。

必须去看看。

陆江仙等待着——

等李家人都睡了，就去村口。

月光照进堂屋，照在那面鉴子上。

镜面上浮起淡淡的人影，一闪而逝。
