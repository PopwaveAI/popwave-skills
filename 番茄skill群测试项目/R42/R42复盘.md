## 三、笔触一致性详细分析

### 玄鉴仙族（8.5/10）

| 笔触维度 | DNA描述 | 试写表现 | 一致性 |
|:---------|:--------|:---------|:-------|
| 句式节奏 | 短句为主，对话引导词省略 | "嗯。"李项平应了声 / "拿着。" | ✓ 高度一致 |
| 叙事距离 | 中近距，跟随主角视角 | 全程跟随李项平视角，未跳POV | ✓ 一致 |
| 静态蓄力 | "缓缓""慢慢"延缓时间流速，蓄力→爆发 | "一步一步走向瓜田，脚步很慢，很稳，像一头伺机而动的野兽" | ✓ 高度一致 |
| 对话风格 | 引导词后置/省略，对话即动作 | "堂弟？" / "偷你家瓜来了！" | ✓ 一致 |
| 情绪外化 | 通过动作（青筋暴起/指节发白）而非心理描写 | "手上青筋暴起""指节捏得发白" | ✓ 一致 |
| 段落呼吸 | 独句段制造停顿 | "是李叶盛。" / "他堂哥。" | ✓ 高度一致 |

**差异点**：
- 原文ch03有大量"鉴子视角"（陆江仙观察村落），试写弱化了——试写更聚焦李项平视角，陆江仙的内心活动被压缩
- 原文李木田的家庭介绍更铺陈（四兄弟+从军经历展开），试写更紧凑
- 试写自动补了"这东西谁也不能跟外人提"的章末决策——DNA影响了内容节奏填充

**结论**：读试写正文，确实像季越人的笔触。短句节奏、静态蓄力、对话引导词省略三大特征都被准确复制。

### 夜无疆（7.5/10）

| 笔触维度 | DNA描述 | 试写表现 | 一致性 |
|:---------|:--------|:---------|:-------|
| 感官顺序 | 触觉→视觉→听觉 | "寒风灌进领口""积雪咯吱咯吱""翅膀扑腾" | ✓ 一致 |
| 动作链 | 短促节奏感，动作连贯 | "扔掉猎叉→取弓搭箭→拉成满月→箭尖闪冷光→嗖地飞出" | ✓ 一致 |
| 环境氛围 | 密林/雪地/夜色，感官进入 | "黑糊糊一片""灰蒙蒙天幕""黑黢黢群山" | ✓ 基本一致 |
| 段落呼吸 | 长段叙事，信息密度高 | 段落偏长，单段信息量大 | ✓ 一致（但过度） |
| 内心独白 | 通过观察引发思考 | "他知道那些地方很危险" | △ 偏直白，原文更含蓄 |
| 细节铺陈 | 习性/原理/机制说明 | 人面鹫习性、冰花原理、树洞温度 | ✓ 一致 |

**差异点**：
- **字数超标（3775字 vs 2500字上限）**——核心问题。辰东DNA的长段风格和番茄字数红线冲突时，API跟着DNA走，忽略字数约束
- 内心独白比原文更直白——原文辰东的"沉重感""压迫感"更依赖语感和词藻，DNA难以完全捕捉
- 试写自动补了"煮粥喝粥"日常场景——梗概里没有，是API按DNA节奏自行填充
- 感官描写更"清晰直白"，原文的感官更"沉重有压迫感"——这是词藻层面的差异，DNA的观察字段无法传递

**结论**：感官顺序和动作链抓得不错，但长段失控导致字数超标。辰东的"沉重感"更难复制——这是DNA方法的固有局限。

## 四、核心发现

### 发现1：DNA驱动笔触可行，两本书笔触区分明显
玄鉴仙族试写呈现"短句+静态蓄力+对话省略引导词"的季越人风格；夜无疆试写呈现"感官铺陈+动作链+长段叙事"的辰东风格。**同一剧情梗概，换不同DNA，笔触确实不同**——DNA可插拔方案验证通过。

### 发现2：短句风格DNA比长段风格DNA更"可执行"
玄鉴仙族（短句）笔触一致性8.5 > 夜无疆（长段）7.5。原因：
- 短句DNA的"观察"更具体可操作（"引导词省略""独句段"）
- 长段DNA的"观察"更依赖语感（"沉重感""压迫感"），API难以复制

### 发现3：DNA风格与字数红线冲突时，API优先服从DNA
夜无疆3775字超标，说明当DNA的段落风格（长段）和write约束（≤2500字）冲突时，API倾向于跟着DNA走。**DNA的字数影响需要显式处理**——要么在DNA提取时标注"长段需压缩"，要么在write约束里加强"字数优先于DNA风格"。

### 发现4：DNA不仅影响笔触，还影响内容节奏填充
两篇试写都自动补充了梗概外的内容（玄鉴仙族补了章末决策，夜无疆补了煮粥场景）。说明DNA不仅传递笔触特征，还传递了"作者在什么节点放什么内容"的节奏感。这符合预期，但需要注意**消费规则要更严格**——DNA应该只影响"怎么写"，不应该影响"写什么"。

### 发现5：DNA提取的采样受限标注有效
玄鉴仙族只有18章采样（中段缺失），但DNA质量依然可用。v4模板的"采样受限"标注没有导致DNA质量崩塌，说明20章以内的采样对于笔触提取是够用的。

## 五、对pop-dna-style设计的启示

| 发现 | 对pop-dna-style的启示 |
|:-----|:---------------------|
| 短句DNA更可执行 | 模板应强调"观察字段必须具体可操作"，避免抽象描述（如"沉重感"） |
| 长段DNA导致字数超标 | DNA提取时应增加"章型节拍绑定"维度的字数约束——标注参考书每章字数区间，write消费时以此为字数锚点 |
| DNA影响内容填充 | 消费规则需强化：DNA只影响笔触层（句式/感官/距离），禁止影响内容层（事件选择/节奏填充） |
| 采样20章够用 | 正式版可维持20章采样（前10+中5+后5），不必强制30章 |
| 两本书笔触区分明显 | DNA提取流程稳定可复用，可固化进pop-dna-style skill |

## 六、下一步建议

1. **pop-dna-style骨架可开建**——R42验证了DNA提取+消费的完整链路可行，模板用pop-shared-dna v4，采样20章，API调用方式已跑通
2. **字数冲突需处理**——在write的DNA消费规则里增加"字数红线优先于DNA段落风格"的硬约束，或让pop-dna-style提取时标注参考书字数区间
3. **观察字段具体化**——pop-dna-style的模板指引应要求"观察字段必须给出可执行的特征"（如"引导词省略"），而非抽象感受（如"沉重感"）
4. **可扩展测试**——下一轮R43可测试"换同一本书的不同章节试写"，验证DNA的一致性是否稳定
