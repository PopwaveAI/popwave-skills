# 第1章正文 | 为第2章铺垫：面板觉醒与首杀

## 夜雨

雨滴打在破旧的木板上，发出沉闷的声响。

空气中弥漫着腐烂的臭味，混杂着泥土和垃圾的气息。贫民窟的夜晚永远是这样——潮湿，阴冷，充满绝望。

索伦睁开眼睛。

他的视线模糊了一瞬，然后逐渐清晰。头顶是低矮的木棚顶，雨水从缝隙中渗下来，滴在地面上积成一小滩。耳边是雨声，还有——一个微弱的呼吸声。

他侧过头。

一个瘦弱的小女孩趴在床边，蜷缩成一团，身上盖着一件破旧的麻布外套。她的头发乱糟糟的，脸上沾着灰，睡得很沉。

薇薇安。

索伦的记忆开始回涌。不是那种清晰的、有序的回忆，而是碎片式的——贫民窟的街道，饿着肚子的日子，被欺负的场景，还有眼前这个女孩，他的妹妹。

但他脑子里还有别的东西。

密密麻麻的数据流，像瀑布一样从眼前划过。0和1组成的洪流，在他意识深处奔腾。那不是幻觉，也不是梦境——那是真实存在的，正在他脑内运转的东西。

索伦闭上眼睛，深吸一口气。

当他再次睁眼时，那些数据流消失了，取而代之的是一个半透明的面板悬浮在视野中。

“系统初始化完成。”

“检测到宿主灵魂绑定。”

“是否查看当前属性？”

索伦的瞳孔微微收缩。

他知道这是什么。这不是他这个世界的东西，而是来自另一个世界的记忆——那个世界的人们把这些叫做“游戏数据面板”。他曾经在某个夜晚、某个梦境中见过这套系统，而现在它真实地出现在他眼前。

他默念：查看。

面板展开：

```
姓名：索伦
种族：半精灵
属性：力量12、敏捷19（+1）、体质15、智力18（+1）、感知15、魅力16
阵营：守序邪恶
职业：5级平民/1级盗贼【一阶】
生命值：12/12
经验值：0（未分配）
```

索伦盯着那个“敏捷19（+1）”的数值看了两秒。

他知道这个数值意味着什么——普通人的敏捷是10点，码头苦力工人的力量是12点。敏捷19，这是超凡的门槛，意味着他可以做到普通人做不到的事。跳跃、翻滚、潜行——这些将成为他的本能。

智力18（+1）同样不低。这意味着他的大脑运算速度远超常人，能记住更多细节，更快分析局势。

但此刻，更让他关注的是另一条信息：

“职业：5级平民/1级盗贼【一阶】”

平民5级——这是他这些年摸爬滚打混出来的底子。盗贼1级——这是他刚刚觉醒的职业。

索伦缓缓从床上坐起身。

木板床发出吱呀声，薇薇安动了动，没有醒来。

他低头看着自己的双手。手指修长，骨节分明，指腹上有薄茧。那是常年干粗活留下的痕迹。但现在，他能感觉到一些不一样的东西——一种轻盈感，一种随时可以爆发的力量。

他想起了一些事情。

那是在另一个世界，在上辈子。他死过一次，然后重生了。他带着所有的记忆，带着那个游戏系统的数据流，重新活了过来。

那个世界的东西——副本、等级、装备、技能——全部转化成了他脑内的数据面板。

而这个世界，是一个真正的魔幻世界。这里有神，有魔法，有怪物，有战争。他记得很清楚——即将到来的诸神黄昏，圣者浩劫，百年血战。

这座城市叫做琥珀城。它会在未来的某一天被毁灭，所有居民被屠杀。而现在，距离那一天还有一段时间。

索伦的目光落在薇薇安身上。

这个瘦弱的小女孩，是他的妹妹。在这个世界，他是她唯一的亲人。

他把手轻轻放在她头上。

薇薇安猛地惊醒，抬起头。

“哥哥！”她的眼睛亮了起来，扑进他怀里，“你终于醒了！你睡了两天了！”

索伦的身体僵硬了一瞬。

他不习惯这种亲密接触。但他没有推开她。

“我没事。”他声音沙哑。

薇薇安抬起头，脸上带着泪痕，眼睛红肿。她瘦得厉害，锁骨凸出，胳膊细得像枯枝。

索伦的目光落在她的后背。

她的外套破了一个口子，露出了里面的肌肤——青紫色的淤伤，大片大片地连在一起。

“怎么回事？”他的声音变冷。

薇薇安赶紧拉紧外套，低下头：“没事……我不小心摔的。”

“谁打的？”

“真的没事……”薇薇安的声音越来越小。

索伦没有再问。

他知道答案。贫民窟里，一个八岁的小女孩独自照顾昏迷的哥哥，没有大人的庇护，会遭遇什么。那些混混，那些觊觎他们的目光，那些压在头上的恶霸。

他捏紧拳头。

然后松开。

“饿了吧？”他问。

薇薇安点点头。

索伦起身，扫视了一眼这个破旧的木棚。

这就是他们住的地方——四壁漏风，屋顶漏水，地上铺着一层干草，角落里堆着几块发霉的黑面包和一小袋米。

他走到火炉边，蹲下身子。

炉子里的火已经熄灭了，灰烬是冷的。旁边放着一个残缺的铁锅，里面盛着半锅水，水面漂着一层灰色的浮沫。

薇薇安跟过来，小声说：“我去借了点米，想给你煮粥……但是……”

索伦看到铁锅旁边的地上撒着一小撮米粒。

“外面那些小孩把米抢走了。”薇薇安的声音很轻，像在说一件很平常的事，“我抢不过他们。”

索伦闭上眼睛。

两秒后，他睁开眼，转身走出木棚。

雨还在下。

他站在门口，看着昏暗的街道。贫民窟的夜没有灯火，只有远处几个火把发出微弱的火光。空气里弥漫着垃圾和排泄物的臭味，老鼠在角落里爬行。

他看到不远处有几个人影。

三个，都是十几岁的男孩。他们蹲在一栋破房子的屋檐下，手里拿着什么东西在分。

其中一个抬起头，看到了索伦。

“哟，醒了啊？”那个男孩咧开嘴笑了，露出一口黄牙，“还以为你直接死了呢。”

旁边两个男孩也笑起来。

索伦没有说话。

他转身走回木棚，捡起地上的一根木棍。那是平时用来挑柴火的，一头很尖。

然后他再次走出木棚，朝那个方向走去。

“哥哥？”薇薇安探出头。

“待在里面，别出来。”索伦头也不回。

雨声很大。

索伦走到那三个男孩面前，停下脚步。

雨滴顺着他的脸颊滑落，他的表情没有任何波动。

“就是你？”那个为首的男孩站起来，比索伦高半个头，“怎么，来找弟弟的麻烦？”

他伸手推了索伦的胸口一下。

手指刚触碰到衣服，索伦就动了。

他的身体以一种不可思议的角度侧转，避开了那只手，同时右手握着木棍猛地向上刺出。尖头刺进那个男孩的下颚，从口腔穿透，直达大脑。

噗——

鲜血混着雨水溅开。

另外两个男孩还没反应过来，索伦已经收回木棍，转身一棍砸在一人的太阳穴上。骨头碎裂的声音在雨中清晰可闻。

第三人终于反应过来，转身要跑。

索伦跨出一步，木棍从他身后刺入后颈。

噗通。

三具尸体倒在泥水里，雨水冲刷着他们的血迹。

索伦站在原地，看着自己的手。

他的手在发抖。

不是恐惧——是兴奋。

面板跳了出来：

“击杀目标：人类男性。”
“获得杀戮经验：10点。”
“击杀目标：人类男性。”
“获得杀戮经验：10点。”
“击杀目标：人类男性。”
“获得杀戮经验：10点。”
“当前经验值：30点。”

三十点。

索伦盯着面板看了两秒。

他的手还在发抖。他盯着自己的手指，看着它们在雨中微微颤动。

这是第一次。

第一次杀人。

他的心跳得很快，呼吸变得急促，胃里翻涌着一股恶心感。但很快，那股感觉就消散了。

他曾经在梦中杀过无数人，在数据流里模拟过无数场战斗。但真实世界的第一次，感觉还是不一样。

索伦深吸一口气。

然后蹲下身子，开始翻尸体。

第一个男孩口袋里掏出了半块黑面包。第二个口袋里有一枚铜币。第三个口袋里什么都没有。

他面无表情地把黑面包和铜币收好，然后站起身。

回到木棚的时候，薇薇安站在门口，眼睛瞪得很大。

“哥哥……你……”

“把门关上。”索伦走进木棚，把湿透的外套脱下来，“明天早上给你煮粥。”

薇薇安没有追问。

她只是乖乖地关上门，然后蹲在火炉旁边，试图生火。

索伦看着她的背影，目光落在她后背的淤伤上。

他需要治疗药剂。

贫民窟里没有药店，但码头区有。不过那些药都很贵，一枚金德勒才能买最低级的治疗药水。

他拿出来刚才获得的战利品：一枚铜币，半块发霉的黑面包。

不够。远远不够。

索伦走到木棚角落，从一块松动的木板下面掏出一个布包。里面有三枚铜币——那是他昏迷前藏的，最后的积蓄。

他把那枚铜币也放进去，一共四枚。

四枚铜币。连一块干面包都买不起，更何况是治疗药剂。

索伦皱了皱眉。

他需要钱。需要更多。

面板又弹了出来：

“提示：宿主可查看当前技能。”

他默念：查看技能。

面板切换：

```
盗贼技能：
- 潜行（1级）：在阴影中移动时，有40%概率不被发现。
- 背刺（1级）：从背后攻击时，造成2倍伤害。
- 开锁（1级）：可以打开普通级别的锁，有一定的失败概率。

平民技能：
- 街头求生（5级）：在贫民窟的环境中，有50%概率找到食物和水。
- 抗打击（3级）：受到普通攻击时，减免1点伤害。
```

潜行。背刺。开锁。

索伦眼中闪过一丝精光。

他可以偷。

但他很快压下了这个念头。偷意味着风险，被抓到就没有以后了。他要做的是更稳妥的事——找一些不会反抗的目标。

比如那些守夜的码头守卫。

比如那些喝醉酒的商人。

比如那些——落单的科尔帮派成员。

索伦的记忆里，贫民窟的恶霸就是科尔帮派。他们控制着这片区域，定期向住户收取保护费。如果有人不交，他们就会把那人打一顿，或者把他们的女儿带走。

薇薇安背后的淤伤，很可能就是他们造成的。

索伦的瞳孔微微收缩。

他需要情报。

“薇薇安。”他开口。

“嗯？”

“那些欺负你的人，是科尔帮的？”

薇薇安沉默了一会儿，点了点头。

“他们收了保护费，我们没有钱……”她的声音越来越小，“所以他们……要我……”

索伦捏紧拳头。

“他们有多少人？”

“白天的时候可以看到七八个人在街上晃，但听说他们的老巢在北边的巷子里，有二十多人。”薇薇安小声说，“领头的叫巴拉斯，是一个四十多岁的独眼男人。他有两个手下很厉害，一个叫卡诺波，一个叫……”

“够了。”索伦打断她，“你在这里待着，锁好门，谁叫都不要开。”

“哥哥，你要去哪里？”

“去给你找药。”

索伦站起身，把湿透的外套重新披上。

“可是外面在下雨……”

“今晚雨大，正好办事。”

他走到门口，回头看了一眼薇薇安。

“我很快回来。”

索伦推开门，走进雨中。

外面，三具尸体还在泥水里泡着。雨水冲刷着血迹，沿着街道的沟渠流向下水道。

他扫了一眼，没有停留。

沿着街道向北走。脚步轻盈，几乎不发出声响。

雨声很大，掩盖了他所有的脚步声。

他在黑暗中穿行，目光扫视着街道两侧。两边都是低矮的木屋，有的已经塌了，有的还有人住。灯烛透出来的光微弱得像萤火虫。

他听到了声音。

从前方传来——两个人的交谈声，夹杂着笑声和酒瓶碰撞的声音。

索伦停下脚步。

他贴着墙根，慢慢靠近。

巷子里有一个破旧的雨棚，下面站着两个人。他们靠在墙角，一个人手里拿着酒瓶，另一个人嘴里叼着烟。

“巴拉斯说了，今晚守好这一片，不要让那些穷人闹事。”叼烟的人说。

“能闹什么事？那些穷鬼连饭都吃不上，还能造反？”拿酒瓶的人笑了。

“也是。不过那个老头说的那件事，你听说没有？”

“什么事？”

“那个小丫头，就是那个叫薇薇安的，她哥醒了。”

索伦的瞳孔骤缩。

“醒了又怎么样？一个半死不活的废物，能干什么？”

“巴拉斯说要找天再去，把那小丫头带走。听说是有人出价买的，一个处女能卖不少钱呢。”

“啧啧啧。那丫头才几岁啊……”

“管她几岁，能卖钱就成。”

索伦的手指微微颤抖。

不是恐惧。是愤怒。

但他没有立刻冲出去。

他在数——两个人，酒瓶，香烟。一个靠在墙上，一个站着聊。都在雨棚下，视野开阔，但注意力分散。

潜行。

索伦的身体融入阴影。

他贴着墙根移动，脚步轻得像猫。雨水打在他的身上，冰冷刺骨，但他感觉不到。

他在接近。

近了。

更近了。

当他的距离缩短到五步时，叼烟的人突然抬起头。

“嗯？”

他还没反应过来。

索伦的身影从阴影中猛地窜出，右手握着那根木棍的尖头，直接刺进那个人的喉咙。

噗呲——

鲜血喷溅。

叼烟的人瞪大了眼睛，想要喊叫，喉咙里只能发出咕噜咕噜的声音。他伸手想要抓住索伦，但身体已经开始失去力量，瘫软下去。

拿酒瓶的人猛地回头，酒瓶脱手砸在地上。

“你——”

索伦没有给他说话的机会。

他收回木棍，一个侧步转到那人身后，左手捂住他的嘴，右手握着木棍从后颈刺入。

噗嗤——

骨头碎裂的声音沉闷地响起。

拿酒瓶的人的身体抽搐了两下，然后软了下来。

索伦松开手，两人的尸体噗通倒在地上。

面板弹出：

“击杀目标：人类男性。”
“获得杀戮经验：10点。”
“击杀目标：人类男性。”
“获得杀戮经验：10点。”
“当前经验值：50点。”

索伦蹲下身子，开始翻尸体。

第一个口袋里摸出了五枚铜币和一小袋烟草。第二个口袋里有两枚银币和一封信。

银币。

索伦把那两枚银币收好，展开信纸。

雨水打湿了纸张，字迹有些模糊，但他还是能看出来，上面写的是——一个地址。

“码头区，仓库街，第三号仓库。”

索伦皱了皱眉。

他不知道这封信意味着什么，但他知道，这一定有用。

他把信折好放进怀里，然后站起身。

巴拉斯。

卡诺波。

二十多个手下。

索伦看向黑暗的巷子深处，目光冰冷。

他知道这些人现在大概在什么地方——北边巷子里的老巢。那里有一个废弃的铁匠铺，被改造成了他们的据点。

他需要潜进去，杀光他们。

但二十多个人，他一个人做不到。

至少现在做不到。

他需要等级，需要技能，需要装备。

索伦低头看了一眼自己的手。

握木棍的位置已经磨破了皮，鲜血渗出来。

他很弱。现在的他，对付三四个普通人已经是极限。面对二十多个有武器的帮派成员，他只会被反杀。

所以他要先变强。

面板再次弹出：

“提示：宿主已积累50点杀戮经验，可消耗30点提升【潜行】至2级。”

索伦毫不犹豫：提升。

面板闪烁：

“潜行提升至2级。”
“消耗30点杀戮经验，剩余20点。”
“潜行成功率提升至55%。”

索伦活动了一下身体。

他能感觉到一丝变化——身体的肌肉记忆似乎更精准了。他闭上眼睛，感受着四周的声音。

雨的落下声。

远处狗吠声。

还有——木棚里，薇薇安均匀的呼吸声。

他睁开眼。

今晚的行动到此为止。

他转身，沿着原路返回。

回到木棚的时候，薇薇安已经生了火。火光照亮了她瘦弱的脸庞，她坐在炉边，双手抱着膝盖，盯着跳动的火焰发呆。

“哥哥？”她抬起头。

索伦走进来，把湿透的外套挂在门边。

“我回来了。”

薇薇安站了起来，快步走到他面前，抱住他。

索伦的身体再次僵硬了一瞬。

“你身上有血的味道。”薇薇安闷闷地说。

“嗯。”

“你杀人了吗？”

索伦沉默了两秒。

“嗯。”

薇薇安抱得更紧了。

“谢谢你。”

她说完，退后一步，抬头看着索伦。她的眼睛里没有恐惧，没有恶心，只有信任。

“谢谢你保护我。”

索伦看着她的眼睛，不知怎么，心里的那股愤怒消散了一些。

“明天早上，我去码头区给你买药。”他说。

薇薇安摇摇头：“我不需要药……”

“你需要。”索伦打断她，“你的后背全是淤青。”

薇薇安低下头，没再说话。

索伦蹲下身子，伸手轻轻摸了摸她的头。

“去睡吧。明天还要早起。”

薇薇安点点头，走回她睡觉的那个角落里，蜷缩着躺下。

索伦坐在火炉边，盯着跳动的火焰。

他的脑海里在飞速运转。

两枚银币和五枚铜币，加上今天的收获，一共是两枚银币和九枚铜币。这够买一瓶最劣质的治疗药水，但那种药水的效果很差，只能止住表面的伤口，对严重的淤伤起不了什么作用。

他需要更好的药。

或者——他需要更多的钱。

面板又弹了出来：

“提示：宿主可学习新技能。是否消耗10点经验值学习【街头求生】的进阶技能【情报收集】？”

索伦盯着面板。

情报收集——这技能听起来很实用。他可以在贫民窟里打听消息，找到值得下手的目标。

他点了确定。

“技能学习成功：情报收集（1级）。”
“消耗10点经验值，剩余10点。”
“在贫民窟环境中，有30%概率获得有用的情报。”

索伦的脑海中涌入了一些信息——关于贫民窟的地图，关于哪些巷子能走，关于哪些地方有暗哨，关于哪些人知道哪些秘密。

他闭上眼睛，记下了所有信息。

明天。

明天开始，他要找钱。

他要治好薇薇安，然后带她离开这里。

离开这个贫民窟，离开琥珀城。

因为索伦记得——那些来自另一个世界的记忆告诉他，未来会发生什么。

诸神陨落。魔网崩溃。圣者浩劫。百年血战。

这座城市，会在三年后被毁灭。

所有居民，全部屠杀。

而他，必须在那之前，带薇薇安离开。

火光照在他脸上，映出他冰冷的眼神。

“哥哥？”薇薇安的声音从角落里传来。

“嗯？”

“你刚才在想什么？”

索伦沉默了一会儿。

“我在想……明天吃什么。”

薇薇安笑了：“骗子。”

索伦没有说话。

他站起来，走到木棚的门口，推开一条缝，看着外面漆黑的夜。

雨还在下。

风中夹着泥土的腥味。

远处，有狗叫声传来。

索伦的目光变得深远。

诸神黄昏即将降临。

而他，将会在这片混沌中，找到自己的路。

火炉中的火焰噼啪作响，映出他的影子——一个半精灵少年，站在黑暗与光明的交界处，手里握着匕首，准备迎接即将到来的风暴。

“哥哥？”薇薇安又叫他。

“嗯。”

“你什么时候变得这么厉害了？”

索伦愣了一下。

他低头看着自己的手。

“刚刚。”他说。

“刚刚？”

“嗯。”

索伦关上木门，走回薇薇安身边。

“我刚才想明白了一件事。”

“什么事？”

“这个世界，不是你死就是我活。”

薇薇安沉默了一会儿。

“所以你要杀很多人吗？”

索伦低下头，看着她的眼睛。

“如果需要。”

薇薇安把脑袋靠在他肩膀上。

“那我要一直跟着你。”

索伦没有说话。

他伸手抱住她，把她搂在怀里。

窗外，雨还在下。

黑暗笼罩着整个贫民窟。

但木棚里，火光照亮了两人的身影。
