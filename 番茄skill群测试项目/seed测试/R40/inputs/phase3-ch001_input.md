## 红线

1. **发散prompt必须纯自由**——不注入搜索数据/格式模板/结构引导/借鉴点/避雷点，只给一句话方向。R9验证：纯自由模式（11字符system+一句话user）产出的创意质量远高于注入搜索数据/格式模板/结构引导的模式
2. **市场调研必须做**——发散前WebSearch 3轮+产出借鉴点/避雷点清单。发散后用清单做事后校准（踩避雷点=淘汰，用到借鉴点=加分）。R13/R14《诡编》级别好创意的另一个关键就是市场校准
3. **Phase 1必须产出≥5个差异化方向**——不得只给3个，不得同赛道，每句话读完脑子里立刻浮现场景
4. **Phase 2只在用户选定后启动**——不在发散阶段做结构化
5. **金手指三要素齐全**——创意×机制×限制缺一不可，禁止只加限制不加代价
6. **四眼法四项全过才放行**——画面/限制/场景量/行为引擎
7. **只产出创意.md+ch001.md**——不写骨架/世界规则/节奏框架/笔触风格/四张地图/敌人梯度（这些移交plot）
8. **首章必须好看**——不是事件展示，是让读者想看第二章的开篇。必须有一个"生活化场景×金手指"的反差画面，纯战斗展示/纯穿越事件不算合格首章
9. **番茄简介必须是散文体**——带对话、带情绪弧线、带钩子，不是概述/设定说明书
10. **创意不得与番茄Top20核心机制雷同**——市场校准的避雷点校验必须执行

## 3b. 黄金开篇法则

首章必须遵循番茄黄金开篇法则：

1. **前3句扔炸弹** — 第一句话就制造冲突/悬念/反差，禁止铺垫环境/背景/天气
2. **300字内完成4件事** — 冲突爆发+主角出场+金手指激活+章末钩子
3. **第一章必须激活金手指** — 不能等到第二章才让读者看到金手指
4. **第一章必须有1个爽感触发** — 压力突破/弱→强/未知揭示/被欺→反击
5. **章末必须有钩子** — 让读者想看第二章的悬念/威胁/转折
6. **必须有1个"生活化场景×金手指"的反差画面** — R13/R14好开篇的核心特质。纯战斗展示/纯穿越事件不算合格首章

### 法则6详解：反差画面（R13/R14核心特质）

**反差画面 = 生活化场景 × 金手指**

R13/R14的好开篇之所以好看，是因为金手指触发在一个**日常可感的生活场景**里，制造了荒谬反差：

| 样本 | 生活化场景 | 金手指 | 反差画面 |
|------|----------|--------|---------|
| R14《诡编》 | 推开凶宅大门（日常场景） | 篡改诡异属性面板 | 女鬼扑来，我把她"暴躁"改成"正在织毛衣"，她真的开始织毛衣 |
| R13张平凡 | 加班骑车回家（日常场景） | 公文语成真 | 黄鼠狼拦路讨封，用红头文件格式回答，它必须照办 |
| 反面R37 | 跳恒河穿越（非日常） | 系统看忠诚度 | 无生活化场景，无反差 |
| 反面R38 | 沙滩苏醒杀怪（非日常） | 基因种子烧蝌蚪 | 无生活化场景，无反差 |

**反差画面设计方法**：
- 找一个主角金手指能触发的**日常可感场景**（上班/回家/买菜/相亲/加班/收快递）
- 让金手指在这个场景里**以最荒谬的方式触发**
- 画面要有**具体动作+具体台词+具体物象**（毛线球/红头文件/属性面板）

**禁止的开篇类型**：
- 纯战斗展示（苏醒→杀怪→杀怪）— 无生活化
- 纯穿越事件（跳河→机场→被抢）— 无反差
- 纯设定铺陈（介绍世界观→介绍金手指→介绍主角）— 无画面

## 3d. 笔触层（三态协议，同步 write v8.0.0 减法版）

> write v8.0.0 做减法移除了 8 项内置风格倾向（短句为主/道引导词/【】面板/动作链必选/多感官必选/感官优先级表/情绪外化/碎片化思考），笔触层交给 DNA。首章作为 write 的 ch001，遵循同一三态协议。

| 状态 | 条件 | 首章笔触行为 |
|:--|:--|:--|
| 启用态 | `涌现/文风锚定.md` 存在 | 笔触层从 DNA 取，禁止凭空发挥 |
| 缺失态 | `涌现/文风锚定.md` 不存在 | 提示用户部署 DNA，用户拒绝则进 trial 模式 |
| trial 模式 | 用户拒绝部署 | 不内置任何默认风格，用户须显式声明基础风格倾向 |

**启用态**：加载 `涌现/文风锚定.md`，首章的句式/感官/叙事距离/对话/情绪外化/段落呼吸从 DNA 取。DNA 无系统面板格式设定时，首章不得擅自加面板。

**trial 模式**：首章笔触由用户显式声明（如"简洁短句+无面板+动作驱动"），不内置 write v7.2.0 的默认风格。如用户不声明，接受 API 自由发挥的质量波动。

> 注：首章是 seed 产出，不是 write 产出。但首章的笔触协议与 write v8.0.0 一致——write v7.2.0 的动作链/多感官/感官优先级表/【】面板/碎片化思考等已移除，不再作为首章硬约束。

## 3e. 爽感闭环（必须）

首章必须有1个爽感触发，且必须是**触发→爆发→后果**三段式：

- **触发**：主角面对压力/被欺/未知/弱者处境
- **爆发**：金手指激活，反转/突破/揭示/反击
- **后果**：爆发后的可见变化（诡异被改写/敌人失态/主角获得反馈）

三段缺一=废章。

## 创意.md

# 创意名

## 一句话简介
26岁被裁员的古董鉴定师沈渡，在爷爷留下的破旧古董店里，意外激活一台民国留声机——只要触碰旧物，就能“听到”前主人一生中最刻骨铭心的那一秒。旧物不是物品，是未解的心结。

## 创意
情绪侦探×旧物通灵——主角不是鉴宝，是替陌生人听完他们生命中最痛的那一秒。

## 创意来源
王道1「二手古董店的留声机」。Phase 1市场验证后用户选定，Phase 2打磨了行为引擎、金手指限制、四眼法验证，并补全了故事纲领。

## 金手指
- **创意**：触碰旧物能“播放”前主人最刻骨铭心的一秒——包含声音、情绪、画面残影
- **机制**：每次完成“代人传话”（把这一秒里未了的心愿转达给相关者），留声机就会解锁一段关于爷爷失踪的线索碎片
- **限制**：播放不可逆、不可暂停、不可跳过——触发后必须完整经历那致命一秒。而且每播放一次，主角自己的某段记忆会被随机“覆盖”一格（他也在被遗忘）
- **合成**：沈渡触碰爷爷留下的民国留声机，听见了1943年一个女人的尖叫。从那以后，每碰一件旧物，就被迫听一次别人的绝唱——他越听，自己越少。他必须在被清空之前找到爷爷的秘密：留声机到底是谁的遗物？

## 行为引擎
**接触旧物 → 触发记忆 → 解读一秒 → 找到活人 → 代人传话 → 解锁线索**

每章固定行为：沈渡拿到一件新旧物→闭眼听那致命一秒→从中提取关键信息（人名/地点/未说完的话）→找到对应的人或地点→把那一秒里的心意转达出去→解锁留声机的新碎片。

## 轻量主角轮廓
- **名字**：沈渡
- **身份**：前古董鉴定师，现旧货店老板，26岁，负债8万
- **核心动机**：找到爷爷的下落——爷爷失踪前最后碰过的就是那台留声机
- **记忆点**：他每次听完记忆后会短暂失神，像是被什么东西抽走一部分，然后若无其事地继续修钟表。别人觉得他怪，只有他自己知道——他在拿自己的记忆换别人的秘密

## 番茄简介

我叫沈渡，一个欠了八万块、守着爷爷旧货店等死的倒霉蛋。

那台破留声机是我从仓库角落里扒出来的——民国货，铜喇叭裂了道缝，唱片槽里卡着一张没标签的黑胶。我没当回事，顺手拨了一下唱针。

然后我听见了尖叫。

一个女人在哭，声音像从井底传上来的。还有个男人在喘，在喊“别过来”。然后是铁器砸碎骨头的声音——嘭，嘭，嘭。

我睁开眼，发现自己在发抖。后背湿透了，嘴里有一股铁锈味。而那张黑胶唱片上，多了一行字——“1987年，红旗巷3号，最后一次见他。”

我把它放回去，告诉自己别碰了。第二天又有客人拿了件旧物来修。我告诉自己别碰。但当指尖碰到那只银手镯的一瞬间——

我听见另一个女人在笑。

她在说：“囡囡，等我回来给你做红烧肉啊。”

那天的菜市场很吵，有辆卡车刹车的声音，然后一切安静了。

我睁开眼，那只手镯上多了一行字：“她没等到那顿肉。”

我把它包好，骑上电动车，导航到手上多出来的那个地址。

后来我才发现——这破玩意儿每听一次，就会从我脑子里偷走一样东西。先是我初恋的名字，再是我妈的生日，然后是今天早上我吃了什么。

我不知道再听下去还剩下什么。

但那只手镯上的女人还在等她的红烧肉。

而我的爷爷——他最后一次碰这台留声机时，听见了什么？

## 十、故事纲领（五维框架）

### 世界底色
喧闹的市井烟火气——街角的旧货店、修表摊、老茶馆、菜市场的讨价还价。每个人看起来都很普通，但每个人心里都有一件不愿提起的旧物。这座城市的灰尘里，藏着一千万个未说完的秘密。

### 危机基调
每一件旧物都是一颗未爆弹——那些被压在最深处的情感，正在通过留声机反向侵蚀沈渡。他听见别人的绝唱，自己的记忆就被抹去一格。爷爷失踪的真相或许就藏在最后一段记忆里——但那段记忆，可能需要沈渡拿命去换。

### 主角一生弧线
- **起点**：一个觉得自己不会再被任何东西打动的人——被裁员、欠债、对什么都提不起兴趣
- **拐点**：被迫听完一段段陌生人的死亡瞬间，发现那些“一秒”里藏着的情感比他自己整个人生还要浓烈
- **终点**：他用自己全部的记忆，换回爷爷那最后一秒的秘密——然后发现爷爷当年也是这么选的。他不是在救别人，他是在完成一场持续八十年的、父子之间沉默的告白

### 最大钩子
触摸旧物，听见生死之间最后的秘密

### 即时兑现感
第1章激活留声机+第一个记忆碎片（寻人线索）→第2章接到第一单“代人传话”→第3章发现碎片与爷爷失踪有关→第4章第一次被记忆反噬，失去一段自己的回忆，意识到代价→第5章接到一个更危险的遗物，记忆覆盖推进核心悬念。每章一个独立小故事+每三章推进一步主线，不让读者等太久。
