## 红线

1. **发散prompt必须纯自由**——不注入搜索数据/格式模板/结构引导/借鉴点/避雷点，只给一句话方向。R9验证：纯自由模式（11字符system+一句话user）产出的创意质量远高于注入搜索数据/格式模板/结构引导的模式
2. **市场调研必须做**——发散前WebSearch 3轮+产出借鉴点/避雷点清单。发散后用清单做事后校准（踩避雷点=淘汰，用到借鉴点=加分）。R13/R14《诡编》级别好创意的另一个关键就是市场校准
3. **Phase 1必须产出≥5个差异化方向**——不得只给3个，不得同赛道，每句话读完脑子里立刻浮现场景
4. **Phase 2只在用户选定后启动**——不在发散阶段做结构化
5. **金手指三要素齐全**——创意×机制×限制缺一不可，禁止只加限制不加代价
6. **四眼法四项全过才放行**——画面/限制/场景量/行为引擎
7. **只产出创意.md+ch001.md**——不写骨架/世界规则/节奏框架/笔触风格/四张地图/敌人梯度（这些移交plot）
8. **首章必须好看**——不是事件展示，是让读者想看第二章的开篇。必须有一个"生活化场景×金手指"的反差画面，纯战斗展示/纯穿越事件不算合格首章
9. **番茄简介必须是散文体**——带对话、带情绪弧线、带钩子，不是概述/设定说明书
10. **创意不得与番茄Top20核心机制雷同**——市场校准的避雷点校验必须执行

## 3b. 黄金开篇法则

首章必须遵循番茄黄金开篇法则：

1. **前3句扔炸弹** — 第一句话就制造冲突/悬念/反差，禁止铺垫环境/背景/天气
2. **300字内完成4件事** — 冲突爆发+主角出场+金手指激活+章末钩子
3. **第一章必须激活金手指** — 不能等到第二章才让读者看到金手指
4. **第一章必须有1个爽感触发** — 压力突破/弱→强/未知揭示/被欺→反击
5. **章末必须有钩子** — 让读者想看第二章的悬念/威胁/转折
6. **必须有1个"生活化场景×金手指"的反差画面** — R13/R14好开篇的核心特质。纯战斗展示/纯穿越事件不算合格首章

### 法则6详解：反差画面（R13/R14核心特质）

**反差画面 = 生活化场景 × 金手指**

R13/R14的好开篇之所以好看，是因为金手指触发在一个**日常可感的生活场景**里，制造了荒谬反差：

| 样本 | 生活化场景 | 金手指 | 反差画面 |
|------|----------|--------|---------|
| R14《诡编》 | 推开凶宅大门（日常场景） | 篡改诡异属性面板 | 女鬼扑来，我把她"暴躁"改成"正在织毛衣"，她真的开始织毛衣 |
| R13张平凡 | 加班骑车回家（日常场景） | 公文语成真 | 黄鼠狼拦路讨封，用红头文件格式回答，它必须照办 |
| 反面R37 | 跳恒河穿越（非日常） | 系统看忠诚度 | 无生活化场景，无反差 |
| 反面R38 | 沙滩苏醒杀怪（非日常） | 基因种子烧蝌蚪 | 无生活化场景，无反差 |

**反差画面设计方法**：
- 找一个主角金手指能触发的**日常可感场景**（上班/回家/买菜/相亲/加班/收快递）
- 让金手指在这个场景里**以最荒谬的方式触发**
- 画面要有**具体动作+具体台词+具体物象**（毛线球/红头文件/属性面板）

**禁止的开篇类型**：
- 纯战斗展示（苏醒→杀怪→杀怪）— 无生活化
- 纯穿越事件（跳河→机场→被抢）— 无反差
- 纯设定铺陈（介绍世界观→介绍金手指→介绍主角）— 无画面

## 3d. 笔触层（三态协议，同步 write v8.0.0 减法版）

> write v8.0.0 做减法移除了 8 项内置风格倾向（短句为主/道引导词/【】面板/动作链必选/多感官必选/感官优先级表/情绪外化/碎片化思考），笔触层交给 DNA。首章作为 write 的 ch001，遵循同一三态协议。

| 状态 | 条件 | 首章笔触行为 |
|:--|:--|:--|
| 启用态 | `涌现/文风锚定.md` 存在 | 笔触层从 DNA 取，禁止凭空发挥 |
| 缺失态 | `涌现/文风锚定.md` 不存在 | 提示用户部署 DNA，用户拒绝则进 trial 模式 |
| trial 模式 | 用户拒绝部署 | 不内置任何默认风格，用户须显式声明基础风格倾向 |

**启用态**：加载 `涌现/文风锚定.md`，首章的句式/感官/叙事距离/对话/情绪外化/段落呼吸从 DNA 取。DNA 无系统面板格式设定时，首章不得擅自加面板。

**trial 模式**：首章笔触由用户显式声明（如"简洁短句+无面板+动作驱动"），不内置 write v7.2.0 的默认风格。如用户不声明，接受 API 自由发挥的质量波动。

> 注：首章是 seed 产出，不是 write 产出。但首章的笔触协议与 write v8.0.0 一致——write v7.2.0 的动作链/多感官/感官优先级表/【】面板/碎片化思考等已移除，不再作为首章硬约束。

## 3e. 爽感闭环（必须）

首章必须有1个爽感触发，且必须是**触发→爆发→后果**三段式：

- **触发**：主角面对压力/被欺/未知/弱者处境
- **爆发**：金手指激活，反转/突破/揭示/反击
- **后果**：爆发后的可见变化（诡异被改写/敌人失态/主角获得反馈）

三段缺一=废章。

## 创意.md

# 创意名

## 一句话简介
急诊科医生林深能看见病人体内的“病灵”，他和这些寄生在疾病中的灵体讨价还价，把绝症治成一场交易。

## 创意
一个急诊科医生×“听见疾病声音并和它谈判”的本事——不是用手术刀切病灶，是用话术切动机。

## 创意来源
用户选定方向：【王道4：我拿病历号脉】+ Phase 2打磨：行为框架碰撞（接诊流程→谈判流程）、四眼法验证通过、故事纲领落盘。

## 金手指
- **创意**：主角能看见并对话病灵——疾病背后的灵体，它们寄生在病人病灶里，靠病人的生命力活着。
- **机制**：谈判框架——病灵有诉求（活下去/换宿主/完成执念），主角用条件交换让病灵主动离开宿主。每成功一次，病灵的部分“权柄”会残留在他体内，可用来诊断或治疗同类型疾病。
- **限制**：一次谈判消耗主角半天的精气神——谈完必须休息，连续谈判会把自己谈进ICU。病灵等级越高，反噬越狠（高等级病灵失败后会记住他的气息，追踪报复）。
- **合成**：急诊科医生把问诊变成了和病灵的跨国谈判——他能听见疾病在说话，但说错一句话，搭进去的不只是病人，还有他自己。

## 行为引擎
接诊→感知病灵→谈判（条件博弈）→履约或反制→休息恢复→接诊。

## 轻量主角轮廓
- **名字**：林深
- **身份**：市三甲医院急诊科住院医师，轮转第三年，还没定科
- **核心动机**：不想再送走没法治的病人——能多救一个是一个
- **记忆点**：急诊科里最累最穷的大夫，说话带东北碴子，谈判时像在菜市场砍价

## 番茄简介

我叫林深，急诊科大夫。

凌晨三点，120拉进来一个男的在抢救台上翻白眼。我冲上去按压，手碰到他胸口的一瞬间，脑门里炸开一个声音，干巴巴的，像从骨头缝里挤出来：

“别按了，我在这儿住了三十年了。”

我手一抖——我没有疯，我真的听见了。一团灰蒙蒙的影子趴在那人左肺上，歪着头打量我。它说它叫肺积，活了三十年，搬过三次家，从肺结核到肺癌，越住越舒坦。

我咽了口唾沫，压低声音问它：“……那你走不走？”

它愣了一下：“你谁啊？”

“我叫林深，他的大夫。”我一边抢救一边跟一团影子谈判，“你在他这儿待着，他也活不成，你也迟早饿死。我给你找个新宿主——重症监护室有个刚送来的，更年轻，血氧更低，你搬过去还能再住二十年。”

肺积沉默了一会儿：“……有空调吗？”

“ICU，恒温恒湿。”

“成交。”

它松爪的那一刻，监护仪的波形恢复了。我看着病人的血氧从67拉到95，深吸一口气——门口护士推门问我：“林大夫，你刚才跟谁说话呢？”

“没事，”我擦了把汗，“跟一个肺商量搬家的事。”

护士的嘴张成了一个O型。我还不知道，这只是开始。肿瘤病房那个肝癌，听说我这儿能谈搬迁补偿，托人来找我递话——

它想住进院长脑袋里。

## 十、故事纲领（三核心+营销层）

> world的核心输入。三核心每项必须给world推导锚点。金手指融入主角核心，不独立。

### 什么样的世界
一座所有疾病背后都有“病灵”暗中寄生的现代都市。病灵是疾病的意识体，它们在病灶中孕育、成长、繁衍——能闻到绝望、爱慕恐惧、贪婪地争夺宿主。隐藏规则：病灵只在宿主濒死或情绪崩溃时现身，医院是阴阳交界最薄的地方。world可生长设定方向：病灵来源链（负面情绪→小病灵→大病魔→上古疫源）、病灵生态位（寄生级→共生级→领地级→灾祸级）、医学界的地下势力（知道病灵并利用其力量的势力/被病灵腐化的医生）。

### 什么样的舞台·世界危机
全书危机本质：病灵在进化。从“寄生人体”开始向“寄生社会关系”蔓延——恐癌症让健康人得癌、失独症传染全家。危机源头：一个被遗忘的“零号病灵”在百年前被镇压在某家医院地下，封印随着城镇化被慢慢磨蚀。world可推导——敌人梯度：寄生级（感冒/胃炎级别的小病灵，仅影响宿主个体）→共生级（糖尿病/高血压，和宿主相互依存）→领地级（癌症/渐冻症，有领地意识能影响病房气场）→灾祸级（瘟疫/集体癔症，能跨宿主动员）→零号（危机本体，抽象概念化的疾病意识）。压力源：零号每溢出一次，全球范围内就会出现新的“不治之症”；瘟疫降生的倒计时随着主角每次谈判中泄漏的气息加速。敌人矩阵：①表层：病灵→宿主→愿意牺牲一切的病人家属；②中层：利用病灵做器官买卖的地下医团→反向研究的药物公司分公司；③深层：释放零号的邪道医者→被病灵腐化的同行→零号本身。

### 什么样的主角
- **起点**：急诊科轮转第三年的住院医师林深，穷、累、女朋友跑了、主任不待见他。初始能力：体质特殊能感知病灵但控制不了——从小总说“这人身上有味道”，被当怪胎，学会闭嘴。
- **拐点**：一晚连续送走三个肺炎老人后崩溃，在太平间里被一只快饿死的病灵搭话——“你闻得到我，你救救我……我能帮你救人。”他答应帮病灵找新宿主，代价是获得它的“权柄残留”——从此能主动看见病灵。他第一次知道自己是“听灵体质”。
- **终点**：林深必须成为活体封印——用他的身体把零号的权柄锁在自己体内。他最终活着退役，但代价是再也不能做高危手术（谈判副作用），三十年后病灵因他的衰老而再次骚动。特殊性成熟：他从“靠谈判救单个病人”成长为“能规划一场瘟疫级的谈判”——不是消灭病灵，是让它们和自己谈判达成和平协议。
- **力量路径**：单病灵谈判（用病历号脉求小病灵离开→获得其权柄碎片）→多病灵协同（利用权柄碎片诊断同类型疾病→学习病灵语言→介入领地级谈判）→灾祸级超大型谈判（一次谈一支病灵军团→推导病灵社会契约→参与零号封印）。world据此推导幕序列：第一幕（1-30章）急诊室黑诊所谈判练手+建立江湖威望；第二幕（31-60章）介入医院政治+发现地下医团+病灵大佬出场；第三幕（61-90章）与病灵高层建交+零号封印松动；第四幕（91-100+）零号之战+代价支付。

### 最大钩子
