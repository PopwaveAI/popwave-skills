# 我跟一个肺商量了搬家的事

凌晨两点四十七分，急诊室门口撞进来一个肺。

不对，是一辆平车推着的病人，但他的肺已经先到了。灰蒙蒙的一团影子趴在那人胸口，像个赖着不走的老租户，听见脚步声连眼皮都没抬一下。

“林深！接诊！”护士长宋姐的声音从走廊那头劈过来。

我跑过去的时候扫了一眼平车上的男人，五十来岁，脸憋成了猪肝色，嘴唇紫得发黑，监护仪上的血氧一蹦一蹦往下掉——67、62、58。三个实习医生围着做气囊辅助通气，但没用，他的左肺跟死了一样没动静。

手按上他胸骨的一瞬间，我的脑子里炸开一个声音。

干巴巴的，像从骨头缝里挤出来的：

“别按了，三十年没让人这么按过。”

我手一抖。

旁边的实习医生小刘吓了一跳：“林哥？”

“没事。”我压低声音，盯着那团灰蒙蒙的影子。

它趴在病人左侧胸腔的位置，像一只瘦骨嶙峋的老猫，爪子嵌进肺叶里。这时我才看清它长了张脸——皱巴巴的，全是褶子，眼珠子浑浊发黄，正歪着头打量我。

“你……看得见我？”它说。

我咽了口唾沫，没搭话。

“林哥，血氧掉到52了！”小刘喊。

“准备插管。”

我嘴上说着，脑子里已经炸开了锅。我能看见它。这个趴在这人左肺上的灰影子，它是活的，它在跟我说话。

“你叫肺积？在这儿住了三十年？”

它愣了一下：

“你谁啊？怎么知道我叫什么？”

我一边手上忙着准备气管插管，一边压低声音跟它说话：

“我是他大夫。你在他这儿住了三十年，搬过三次家，从肺结核到肺癌，对吧？”

肺积的眼神变了，它弓起背，那团灰蒙蒙的雾气朝我涌过来：

“你想干什么？”

“想跟你谈谈。”

我用最快的速度扫了一眼病历上的信息：陈某某，男，54岁，左侧肺不张伴大量胸腔积液，既往史肺鳞癌术后两年——两年，就是两年前做的手术。病灵还在，说明手术没切干净，它换了个位置继续长。

小刘已经把喉镜递到我手边了。我没接。

“你们先做高流量吸氧，推一支肾上腺糖皮质激素。”

“林哥，不插管吗？”

“等一下。”我说，“我先谈一下。”

小刘的表情像是在怀疑我疯了。我没理他，压低声音对那团灰蒙蒙的影子说：

“肺积，你在他这儿住着，他活不成。他活不成你也得饿死——宿主死了，你们病灵也得消散，对吧？”

肺积眯着眼看我。它的爪子从肺叶里拔出来一点：

“你懂的不少。你到底是什么人？”

“林深，急诊科大夫。我说真的，你换个地方住行不行？”

“我住这儿三十年了！”它的嗓门一下子提起来，“我从一个结核杆菌开始住起，一点一点把地盘扩到现在这么大，我容易吗？你让我搬我就搬？”

“我给你找更好的。”

“你上哪儿找？”

我瞟了一眼对面的抢救室——120刚送来的一个年轻人，车祸外伤，左肺挫裂伤伴活动性出血，正在止血，血氧一样低，且随时可能不行。

“那边有一个，二十七岁，左肺挫裂伤，按年龄能让你再住五十年。”我说，“比你现住的这个强，他的本体还年轻。”

肺积沉默了一会儿：

“……那你有补偿吗？”

“补偿？”

“搬家费！我在老宅住了三十年，里头有我沉淀的权柄——你让我搬，得等价置换。”

我脑袋飞速运转。它说的“权柄”是什么？谈判成功后的残留部分？

“你要什么补偿？”

肺积的眼珠子转了两圈：

“隔壁二床那老太太，肺里有颗结节。我要是搬过去，那个结节就归我了。”

二床老太太的肺结节我看了片子，6mm，磨玻璃影，随访观察名单上的。现在还在良性阶段，但如果有病灵入驻，半年内就能翻成早期浸润癌。

我咬了咬牙：

“那不行。换一个。”

“你他妈事儿真多。”肺积啐了一口，“那——那个结节我不动，但我要一个夜班大夫的右肺。那个戴眼镜的，老咳嗽那个。”

我回头看了一眼，说的是小刘。他支气管炎拖了两周没好，右肺下叶确实有片感染灶正在吸收。

“他不行，他是我的人。”

“那你总得给点好处吧？我三十年积蓄搬一次家，一分钱不拿就走，你当我傻啊？”

我想起来一件事。每个病灵谈判成功的代价——我要消耗半天的精气神。但它刚才说“权柄”这个词：

“你把权柄留给我。”

“什么？”

“你搬走后，把你的权柄残留留在我这儿。你的诊断权——我能看出下一个病人有没有和你一样的肺部病变。这样我以后能少误诊几个。”

肺积像看傻子一样看了我三秒，然后爆出一阵干巴巴的笑：

“你他妈真是个大夫啊。行吧，你要是能接住老子的权柄，那是你的本事。”

“那成交？”

“成交。”

它最后看了一眼身下的病人，爪子从肺叶里拔了出来——抽出来的一瞬间，左肺的积液像被捅破的水袋一样涌出来，但随即又被周围的肺组织吸收了回去。灰影从病人胸口跳了出去，像一滩水一样流进走廊，朝年轻病人的抢救室去了。

监护仪的波形抖了一下，然后开始恢复。

血氧：67、72、78、85、95。

男人的脸色从猪肝色变成了暗红色，再到正常的肤色。他猛地干咳了一声，睁开眼，茫然地看着天花板。

“这……这是在哪儿？”

小刘的嘴张成了一个O型。

“林哥……你刚才跟谁说话呢？”

“没事。”我擦了把头上的汗，感觉到一阵天旋地转——不是头晕，是身体里某个地方被抽空了一块，“跟一个肺商量了搬家的事。”

“搬家？”

“它搬到对面那个车祸的年轻人身上去了。”

宋姐推门进来：“林深，你脸色怎么这么差？”

“没事，休息一下就好。”

我靠在墙上，手指还在发抖。刚才发生的事情——我看见了一只病灵，跟它谈判，它搬走了，病人活了。这是我第一次做到。从小到大我一直说自己能“闻到病人的味道”，被当怪胎骂了二十年，以为自己是幻觉太严重，今天才知道，那是真的。

不是幻觉。

“林哥，你手机在响。”

我低头一看，是住院总办公室的号码。

“林深，过来一趟。肿瘤病房那边有个肝癌，听说了你的事，托人来找你递话。”

“什么话？”

“它说——它想住进院长脑袋里。”

我挂掉电话，靠在墙上深吸一口气。

看来这活儿，才刚刚开始。
