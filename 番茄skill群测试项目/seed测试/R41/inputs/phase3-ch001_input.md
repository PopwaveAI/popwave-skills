## 红线

1. **发散prompt必须纯自由**——不注入搜索数据/格式模板/结构引导/借鉴点/避雷点，只给一句话方向。R9验证：纯自由模式（11字符system+一句话user）产出的创意质量远高于注入搜索数据/格式模板/结构引导的模式
2. **市场调研必须做**——发散前WebSearch 3轮+产出借鉴点/避雷点清单。发散后用清单做事后校准（踩避雷点=淘汰，用到借鉴点=加分）。R13/R14《诡编》级别好创意的另一个关键就是市场校准
3. **Phase 1必须产出≥5个差异化方向**——不得只给3个，不得同赛道，每句话读完脑子里立刻浮现场景
4. **Phase 2只在用户选定后启动**——不在发散阶段做结构化
5. **金手指三要素齐全**——创意×机制×限制缺一不可，禁止只加限制不加代价
6. **四眼法四项全过才放行**——画面/限制/场景量/行为引擎
7. **只产出创意.md+ch001.md**——不写骨架/世界规则/节奏框架/笔触风格/四张地图/敌人梯度（这些移交plot）
8. **首章必须好看**——不是事件展示，是让读者想看第二章的开篇。必须有一个"生活化场景×金手指"的反差画面，纯战斗展示/纯穿越事件不算合格首章
9. **番茄简介必须是散文体**——带对话、带情绪弧线、带钩子，不是概述/设定说明书
10. **创意不得与番茄Top20核心机制雷同**——市场校准的避雷点校验必须执行

## 3b. 黄金开篇法则

首章必须遵循番茄黄金开篇法则：

1. **前3句扔炸弹** — 第一句话就制造冲突/悬念/反差，禁止铺垫环境/背景/天气
2. **300字内完成4件事** — 冲突爆发+主角出场+金手指激活+章末钩子
3. **第一章必须激活金手指** — 不能等到第二章才让读者看到金手指
4. **第一章必须有1个爽感触发** — 压力突破/弱→强/未知揭示/被欺→反击
5. **章末必须有钩子** — 让读者想看第二章的悬念/威胁/转折
6. **必须有1个"生活化场景×金手指"的反差画面** — R13/R14好开篇的核心特质。纯战斗展示/纯穿越事件不算合格首章

### 法则6详解：反差画面（R13/R14核心特质）

**反差画面 = 生活化场景 × 金手指**

R13/R14的好开篇之所以好看，是因为金手指触发在一个**日常可感的生活场景**里，制造了荒谬反差：

| 样本 | 生活化场景 | 金手指 | 反差画面 |
|------|----------|--------|---------|
| R14《诡编》 | 推开凶宅大门（日常场景） | 篡改诡异属性面板 | 女鬼扑来，我把她"暴躁"改成"正在织毛衣"，她真的开始织毛衣 |
| R13张平凡 | 加班骑车回家（日常场景） | 公文语成真 | 黄鼠狼拦路讨封，用红头文件格式回答，它必须照办 |
| 反面R37 | 跳恒河穿越（非日常） | 系统看忠诚度 | 无生活化场景，无反差 |
| 反面R38 | 沙滩苏醒杀怪（非日常） | 基因种子烧蝌蚪 | 无生活化场景，无反差 |

**反差画面设计方法**：
- 找一个主角金手指能触发的**日常可感场景**（上班/回家/买菜/相亲/加班/收快递）
- 让金手指在这个场景里**以最荒谬的方式触发**
- 画面要有**具体动作+具体台词+具体物象**（毛线球/红头文件/属性面板）

**禁止的开篇类型**：
- 纯战斗展示（苏醒→杀怪→杀怪）— 无生活化
- 纯穿越事件（跳河→机场→被抢）— 无反差
- 纯设定铺陈（介绍世界观→介绍金手指→介绍主角）— 无画面

## 3d. 笔触层（三态协议，同步 write v8.0.0 减法版）

> write v8.0.0 做减法移除了 8 项内置风格倾向（短句为主/道引导词/【】面板/动作链必选/多感官必选/感官优先级表/情绪外化/碎片化思考），笔触层交给 DNA。首章作为 write 的 ch001，遵循同一三态协议。

| 状态 | 条件 | 首章笔触行为 |
|:--|:--|:--|
| 启用态 | `涌现/文风锚定.md` 存在 | 笔触层从 DNA 取，禁止凭空发挥 |
| 缺失态 | `涌现/文风锚定.md` 不存在 | 提示用户部署 DNA，用户拒绝则进 trial 模式 |
| trial 模式 | 用户拒绝部署 | 不内置任何默认风格，用户须显式声明基础风格倾向 |

**启用态**：加载 `涌现/文风锚定.md`，首章的句式/感官/叙事距离/对话/情绪外化/段落呼吸从 DNA 取。DNA 无系统面板格式设定时，首章不得擅自加面板。

**trial 模式**：首章笔触由用户显式声明（如"简洁短句+无面板+动作驱动"），不内置 write v7.2.0 的默认风格。如用户不声明，接受 API 自由发挥的质量波动。

> 注：首章是 seed 产出，不是 write 产出。但首章的笔触协议与 write v8.0.0 一致——write v7.2.0 的动作链/多感官/感官优先级表/【】面板/碎片化思考等已移除，不再作为首章硬约束。

## 3e. 爽感闭环（必须）

首章必须有1个爽感触发，且必须是**触发→爆发→后果**三段式：

- **触发**：主角面对压力/被欺/未知/弱者处境
- **爆发**：金手指激活，反转/突破/揭示/反击
- **后果**：爆发后的可见变化（诡异被改写/敌人失态/主角获得反馈）

三段缺一=废章。

# 暗巷仲裁者

## 一句话简介
一个开在凶宅三楼的破律师事务所，老板是我，我能看见每份委托合同上缠绕的因果线——我为活人辩护，也为怨鬼翻案。

## 创意
一个濒临倒闭的律师行老板，发现自己能“看见”委托合同上的因果线，接下别人不敢接的都市灵异纠纷案，在法庭上为鬼魂辩护。

## 创意来源
用户选定方向：都市异能赛道。Phase 1发散产出10个方向（5个王道+5个猎奇），用户选定【王道1：暗巷仲裁者】。该创意在律政辩论与都市异能结合上具有稀缺性，且主角成长路径清晰、剧情张力足。

## 金手指
- **创意**：能通过触摸合同看见因果线，每条线都记录着签约方的前因后果和背叛代价。
- **机制**：发现合同中的文字陷阱，让灵异法律条文“吃掉”违约方的能量作为诉讼费。
- **限制**：只能看见双方签过字、按过手印的“有效合同”。口头约定和未成文的因果看不见，必须想方设法让对方在自己的文书上签字。
- **合成**：主角作为濒临倒闭的律师行老板，能通过触摸合同看见缠绕其上的因果线（创意），发现文字陷阱后利用灵异法律条文“吃掉”违约方的能量作为律所运转的诉讼费（机制）。但他只能看见已经双方签字的有效合同，对口头约定和未成文的因果关系毫无办法，必须不断想办法诱使对手签字画押（限制）。

## 行为引擎
承接灵异纠纷案→翻阅合同/证据→寻找因果线漏洞→开庭/谈判→结案善后（每个案子循环一次）

## 轻量主角轮廓
- **名字**：林深
- **身份**：律所“深巷事务所”老板，前民商法律师，因得罪人被同行排挤，只剩一间开在凶宅三楼的破事务所。
- **核心动机**：找出当年那个让他在律师圈身败名裂的幕后黑手——他怀疑那人根本不是活人，只是一团披着人皮的因果。
- **记忆点**：永远穿洗得发白的旧西装，衬衫领口磨到起毛，但手上那只老式派克钢笔从来不出墨——是他父亲临终前握过的遗物，也是他最初看见因果线的“钥匙”。

## 番茄简介

我叫林深，在凶宅三楼开了家快倒闭的律所。

接到的第一个案子是个凶宅试睡员——他推开门就尖叫着跑出来，说天花板上有东西在写血书。我走进那间屋子，盯着租约上渐渐渗出血迹的字迹看了三秒。

“房东，你这合同有重大瑕疵。”我把租约拍在桌上，“当事人签署时你未告知目标房产内有‘怨鬼再就业培训计划’。根据灵异纠纷调解条例第42条第6款，乙方有权立即解除合同不用赔偿。”

天花板上的血字顿了顿，开始重新排版成一份解约协议。

紧接着第二个案子来了。一个老头非要把自己的骨灰盒卖给我当镇纸，说他活着的时候是个书记员，死前签署了份遗体捐赠合同，结果死后被告知要把他轮回的资格都捐出去。我翻出他死后单位和家属补签的那份《追加协议》，笑了。

“追加协议没有死者本人的亲笔签名，手印是缝合上去的，非法，还请您拿回去重签。”

我打了一场又一场这样的官司，在死人堆里接活，在合同上找活路。直到有一天，我收到一个没有寄件人的包裹。里面是一份与我父亲死亡有关的旧遗嘱。

末页赫然签着：因果审核委员会。

## 十、故事纲领（三核心+营销层）

> world的核心输入。三核心每项必须给world推导锚点。金手指融入主角核心，不独立。

### 什么样的世界
深夜法庭的法槌敲响时，空气中渗出淡淡的香灰味——这是一个灵异事件被都市规则表层掩盖的现代都市。隐藏规则：“所有超自然事件背后都有合法合同或协议支撑，只要能找到合同的漏洞就能破解，反之就要承受代价。”这个世界里，鬼魂和妖精都讲法律，只是它们签合同用的是血，违约后果是命。

### 什么样的舞台·世界危机
第一句给读者压迫感：林深父亲死亡那天签过一份保密协议，签完第二天就死在办公室——法医说是心源性猝死，但那份协议至今下落不明。全书危机本质是“因果规则正在被改写”，有人想把灵异世界的底层法律改成一套用来收割寿命的霸王条款。敌人梯度从底层（利用合同敲诈的恶鬼）到中层（违规经营的灵异律师所）到顶层（因果审核委员会）。压力源矩阵包括每打输一场官司林深律所的“因果信誉度”就会下降，信誉度归零时他会被从律师公会除名，再也接不了案。

### 什么样的主角
- **起点**：破产律师，开在凶宅三楼的律所只剩他一人，欠债六十万，每天靠泡面和日结咨询费续命。初始能力是能隐约感知合同上有东西“不对劲”，但说不出为什么。
- **拐点**：一个鬼魂跑进他的律所非要签代理合同——那个鬼说自己的房子被活人霸占了，而那个活人手上那份租房合同上签的不是人名，是“甲乙双方同意”。林深第一次清晰看见整份合同的因果线从纸面渗出，像蛛网一样缠住他的神经。
- **终点**：成为因果规则链条上真正的话事人，但不是替谁改写规则，而是把被篡改的契约一条条钉回原位。特殊性成熟：他的钢笔能主动在合同上留下一条不可叛变的因果律条款——一旦写下，任何人（包括他自己）都必须在字面意义上执行。
- **力量路径**：从“只能看已经签好的合同” → “能通过自己的钢笔在合同上添加一条因果律条款（对方签字即生效）” → “能看见未签合同的因果，在对方落笔前预判后果” → “最终能签下一条规则级别的因果契约，改写底层的灵异法律框架”。
