# 第001章 凶宅租约上的因果线

“这官司我接了。”

我把印泥盒往桌上一推，对面的试睡员小张吓得从椅子上弹起来。

“林、林律师，那房子真有鬼——”

“有鬼才接。”我翻开那份被揉成一团的租房合同，领口的线头蹭着脖子，旧西装袖口磨得发亮，但手上那只派克钢笔在日光灯下泛着一层冷光，“房东说违约要赔三个月租金？”

“对，押金全扣，还得倒贴八千——”

“不用赔。”

我盯着租房合同第三页第七行。

纸面上多出一条暗红的线，像干涸的血迹，从“丁字口街道77号3单元602室”那几个字里渗出来，顺着纸张纹路蛇一样蜿蜒。线的另一头绕在“合同期限”那一栏，末端打了个死结——死结里缠着几根头发，黑色的，还在微微颤动。

这是因果线。

我看得见。

打我父亲死后第七天开始，我就能从签过字的合同上看见这种东西。不是每次都能看见，得碰运气，有时盯着十分钟什么也没有，有时低头扫一眼就满纸血痕。

今天运气不差。

“房东有没有告诉你，那房子去年死过人？”

小张脸色发白：“说、说是正常死亡——”

“正常死亡会在合同里写‘目标房产内无异常能量波动’？”

我把合同翻到最后一页，指了指附则那一栏。在密密麻麻的打印条款末尾，手写加了一行小字——字迹很淡，像是写完后又被人用橡皮擦过一遍，但我凑近了还能认出来：

*“乙方确认已实地勘查并知悉该房产结构及现状，甲方已完成瑕疵告知。”*

那行小字上面，飘着一团灰色的烟雾。

不是真的烟，是因果线散逸出来的气息。

我眯起眼，看见那团烟慢慢聚成一张脸——皱巴巴的，颧骨突出，嘴唇发紫，像是被什么东西扼住了喉咙窒息死的。那张脸张口说了句什么，我听不见声音，但口型我看懂了：

*“我不走。”*

“林律师？”小张声音发颤，“您在看什么？”

“看你合同上多出来的那行字。”我把合同拍平，从抽屉里翻出一本薄薄的册子——封面上印着“灵异纠纷调解条例【2024年第一版】”，是我花三十块钱在孔夫子旧书网淘的，书页发黄，边角卷起，“房东在你签合同之前，有没有让你签过什么告知确认书？”

“没、没有。”

“那就对了。”

我翻开条例第42条——那条的页角被人折过，书脊处还有几道指甲印，我查过太多遍了。

“根据《灵异纠纷调解条例》第四十二条第六款，出租方在签订租赁合同前，应当书面向承租方告知目标房产内存在的灵异现象及历史事件。如果出租方未能履行告知义务——”

我把条例推到小张面前让他看。

“承租方有权在发现灵异现象后三十日内单方解除合同，且不承担任何违约责任。”

小张眼睛亮了：“那我不用赔钱了？”

“不仅不用赔。”我笑了笑，从笔筒里抽出一支红色记号笔，在合同最后一页的条款旁画了个圈，“房东为了让你签这份合同，隐瞒了房屋内存在‘非人类房屋共居者’的重大事实。根据条例第四十三条，乙方有权要求甲方返还全部押金，并承担因隐瞒事实导致的精神损失费用。”

我顿了顿，把钢笔从西装口袋里拔出来。

那只派克钢笔——笔杆上漆都磨掉了，露出下面发黄的铜胎。笔帽上刻着一行小字：“勘误为善”。是我爸的字迹。

我爸当了一辈子书记员，退休前还在法院的民事审判庭帮忙整理卷宗。他死那天没去上班，一个人坐在书房里，面前摊着一份打了马赛克的判决书。法医说心源性猝死，收拾遗物时我在他右手虎口处发现一个针尖大的红点，像是签字时被什么尖东西扎了一下。

那只派克钢笔就搁在判决书上。

我从那天起开始看得见因果线。

“您要干什么？”小张看着我拔开笔帽。

“帮那位‘非人类房屋共居者’重写一份合同。”

我用笔尖轻轻碰了碰那张灰色烟雾的脸。

笔尖触到纸面的瞬间，那团烟雾猛地收缩，像是被针扎了的章鱼，所有的灰色线条往回卷，在笔尖下聚成一滴暗红色的墨珠。墨珠在纸面上滚动，然后慢慢渗开，化作一行手写体：

*“甲方（出租方）确认：目标房产内存在一名——不，一名不愿透露姓名的怨鬼再就业培训计划学员。该学员目前因培训未完成，暂无法前往地府报到。作为善意出租方，应当为学员提供安全、稳定的居住环境。”*

写完这些字的瞬间，我虎口一阵刺痛。

因果线从我握住笔的指尖流出，和那团灰色的烟雾绞在一起，像两条蛇互相缠绕。我的钢笔在多出来的那行字末尾自动签上了字——

*“因果公证人：林深”*

字迹很淡，很快就被纸面吸收了。但我知道它已经生效。

我从口袋里掏出手机，拨了合同上的房东电话。

“喂，哪位？”

“你好，我是深巷律师事务所的负责人林深。”我用肩膀夹着手机，一边把那张合同翻到正面，“关于你租给张先生的丁字口77号3单元602室的合同，我有几个问题想跟你核实一下。”

“什么问题？”

“请问，你在与张先生签订租赁合同前，是否已经向张先生书面告知该房屋内存在‘灵异现象’？”

电话那头沉默了三秒。

“……你是什么人？”

“律师。”我笑了笑，“你是不是觉得那行写在附则里的小字，已经算尽了告知义务？那行‘乙方确认已实地勘查并知悉该房产结构及现状’——司法实践中，‘结构及现状’不包含‘肉眼不可见的超自然存在’，这个判例去年就有了。你这个条款没有法律效力。”

“……”

“另外，你在合同第三页第七行的‘合同期限’条款下，手写加了一句‘如因非乙方原因导致房屋无法正常使用，本合同自动顺延’——”我顿了顿，盯着那段蠕动起来的血字，“这句话的意思是，只要那位‘非人类共居者’不搬走，你就能一直收租金，张先生永远住不进来，也永远解不了约。”

我听见电话那头传来什么东西碎裂的声音。

“但你忘了写一条。”我的钢笔在纸上又划了一下，“作为出租方，你有义务确保房屋的‘正常使用’——如果房子里有个闹得人没法住的东西，承租方依然可以主张该房屋‘不适合居住’，从而撤销合同。”

这一次，笔尖过处，那团灰色的烟雾剧烈震动起来。

我看见了。

在那张皱巴巴的鬼脸后面，有一整页密密麻麻的契约条款——那是房东和某个我不能看见名字的存在之间签的协议。那份协议的核心只有一条：只要602室能「持续产生租金」，对方就不得干预。

但现在，这份租房合同被撕裂了。

我笔尖落下的地方，那张鬼脸露出了一个几乎称得上“感激”的表情。灰色烟雾化作一股细流，沿着笔杆慢慢爬进我的手指。指尖涌上一股暖意，像是喝了口陈年的热酒——这是因果线里剥离出来的能量，是我这场官司的律师费。

“我给你两个选择。”我对电话那头说，“第一，无条件解约，退还押金，赔偿张先生精神损失费三千元。第二，我去灵异纠纷调解委员会立案，按条例开听证会——”

“退钱。”

电话那头的声音像是从牙缝里挤出来的。

“全额退，赔偿金一周内到账。那份合同……你处理掉。”

“爽快。”我挂了电话。

小张愣在原地，嘴巴张得能塞进一枚鸡蛋：“就、就这样？”

“就这样。”我把那份合同叠好，塞进碎纸机，“明天去收押金和赔偿金。以后租房前先查一下房产的‘案底’，别只看产权证。”

“林律师……您刚才说那房子有‘非人类共居者’——那到底是什么？”

我想了想。

“鬼。”

“……”

“它在参加一个再就业培训——听起来像是有组织的那种。不过不关你事了。”我拉开抽屉，把那份灵异纠纷调解条例塞进去。

抽屉合上的瞬间，我看见最底下压着一封没有寄件人的快递。

白色信封，边缘泛黄，像是放了很久。

信封上没写名字，没写地址，只有一行字——

*“林深律师亲启”*

我盯着那行字看了三秒。

和所有签过字的合同不一样，这封信上没有因果线，干干净净的，像是刚从印刷厂出来。

但我记得这个字体。

那封放在父亲遗物里的判决书上，也有同样的字体。
