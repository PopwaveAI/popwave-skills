# 我的客户不是人

林深第三次从水杯里捞出一只蟑螂的时候，事务所的门被风吹开了。

没有风。窗户用胶带封着，门锁是新换的防盗锁，保险栓还在原位——但门就这么自己开了。走廊灯灭了半截，暗红色的应急灯把墙上的水渍照得像干涸的血痕。

他推了推无框眼镜，把蟑螂弹进水槽，头也没抬：“门在左边，自己关上。咨询费三百，现金或者上坟烧纸都行。”

一团黑雾卷了进来。

准确说，是人在雾里的形状——四十来岁，发际线靠后，穿着一件洗得发白的工装夹克，整个身体从腰部以下全是半透明的。他站在门口看了看天花板上的漏水斑，又看了看林深桌上那台积灰的二手打印机，表情里带点“这地方比我家还破”的嫌弃。

“林律师？”黑雾问。

“说事。”

“我叫张建国。”对方搓了搓手，“死三年了，一直住在前房东的床底下。昨天房东说要装修，把我剩下的东西全扔了，连我在床底下画的粉笔线都擦了。林律师，你说这公平吗？我租那床底十七年，房租一分没欠过。”

林深抬起眼皮。他看见那团黑雾头顶飘着一根细线——暗红色的，微微发着光，从天花板穿出去，消失在楼上的某个方向。那是因果线？他脑子里闪过一个念头。三个月前输了那场官司之后，他就开始陆续看见这些奇怪的东西，起初以为是幻觉，后来发现只有盯着某些特别的人和事物时才会出现。

而且那根线在动。

它在往张建国的方向收拢，像钓鱼线被慢慢回拉，每收一寸，张建国的身体就透明一分。

“你签合同了？”林深问。

“签了。”张建国从怀里掏出一张皱巴巴的纸，纸张边缘发黄发脆，页脚有一个红色的拇指印，按上去还在渗血。“十年前租房的时候签的，当时想着床底下放点杂物，就没仔细看。”

林深说：“我先做几个假设，你回答。房东知道你已经死了吗？”

“不知道。我死之后继续住，他没发现。说真的，只要我不在半夜出来走动，正常人也看不见我。”

“最近三个月，房东忽然说要装修？”

“对。而且特别急，上周就贴了通知，说三天内搬完。”

林深从抽屉里抽出一支漏墨的钢笔——“漏墨”不是形容词，笔尖是真的裂了一道缝，墨水顺着笔杆往下淌，滴在桌面上一滩暗蓝。他也不知道为什么要带这支笔，这是当年第一个案子时用的，官司输了之后就没能换掉，口袋里装着它，像某种自虐式的纪念。

他把合同摊开。

然后他看见了。

那张纸上的字像是有生命的——每一个字都缠绕着细如蚕丝的线，暗红色的在条件条款周围绕了三圈，代表租户的真实意愿；墨黑色的在页脚那行免责声明底部扎了根，蜿蜒向上，像一条藏在纸缝里的蛇。金线？林深找了找，没见到。金线代表灵魂绑定，这种小案子还不至于上那种规模。

但那根黑线的走向让他后背一凉。

它从第几条条款下面绕出来，穿过“甲方有权收回租用空间”的纸面，用非常不起眼的笔法，在纸张底部的空白处刺了一排小字——字小到肉眼根本看不清，只有通过因果线才能发现它的存在。

林深把纸翻过来，对着天花板上的灯看了半分钟，直到眼睛发酸，才终于辨认出那行字的内容：

“当租户死亡后，床底空间自动归属于房东。该条款不受时间限制，不因租户知情与否而改变。”

他把纸拍在桌上。

张建国凑过来看，什么也没看出来：“怎么啦？”

“你签合同的时候，房东给你看的是正面三页，背面这一行你没签。”

“废话，我又不是签卖身契——谁会翻过来看啊！”

林深拿起漏墨的钢笔，在办公桌上敲了两下。墨水又从笔尖滴下来，拉成一条细线，落在合同正面的空白处，像一只活的海星一样自己张开，摊平，变成一个个歪歪扭扭的方块字。

他只写了一条。

“床底使用权可转让，与租户的灵魂存在方式无关。转让不通知甲方，亦不构成违约。”

张建国瞪大了眼睛：“这……有用吗？我是鬼啊，法律管我吗？”

“法律管不了你。”林深把笔放下，钢笔的裂口处冒出一缕白烟，像什么东西被烧糊了。“但合同管。只要这份合同上写明了你拥有使用权，房东就不能赶你，除非他签字同意放弃。”

“你确定？”

“他签的字比你多，他自己加的那条免责声明也是他自己写的。”林深顿了顿，“法律面前，人鬼平等。”

话音刚落，他的视野突然模糊了一下。

不，不是视野——是感觉。就像有人用橡皮擦从他记忆里抹了一小块东西。他下意识地回忆刚才干了什么：接了案子、看了合同、补了条款。那昨天呢？他去哪里吃的午饭？他眨了眨眼，怎么也想不起来了，脑海里关于“昨天中午”的内容像被漂白过一样，空白一片。

代价。每用一次这支笔，他就会失去一段记忆。

林深闭了一下眼，再睁开时发现那根暗红的因果线已经变了颜色——它现在不是连着张建国，而是从合同上的钢笔字迹那里分出一支，绕了一个大弯，穿过窗台的裂缝，缠在了他自己的手腕上。那线很细，几乎看不见，但林深知道它已经挂上了。

每写一次，自己就从这个世界淡出一分。

要么接受，要么不接案子。不接案子？他瞥了一眼办公桌上那张被退回来的上个月的房租催缴单。房东说得客气，但“再不交租请搬走”几个字用了加大加粗，血红色的打印体。

他咽了口唾沫，把钢笔收进口袋。

“好了。”他冲张建国说，“你可以回去看看，房东如果找麻烦，让他来找我。”

张建国喜笑颜开，刚想再说句感谢的话，门外忽然传来一个声音。

“请问……这里是林律师吗？”

声音闷闷的，像隔着一层棉被说话。

林深抬头，看见门缝里伸进来一只青灰色的手，手指捏着一叠发黄的借条，每张借条的落款都有不同的名字，有的还在往下滴暗红色的液体。

门被推开了。

一具无头的清朝僵尸站在门口，脖子上是一道整齐的切口，空荡荡的领口里飘出一股淡淡的香烛味。它的另一只手提着一盏白色的纸灯笼，灯笼上写着两个字：“借寿”。

僵尸冲林深咧嘴笑了。它没有嘴，没有头，但林深就是知道它在笑，而且它那叠借条上缠绕的因果线——

是金线。

密密麻麻的金线。
