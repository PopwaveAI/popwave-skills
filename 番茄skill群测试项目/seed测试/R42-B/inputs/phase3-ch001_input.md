## 红线

1. **发散prompt必须纯自由**——不注入搜索数据/格式模板/结构引导/借鉴点/避雷点，只给一句话方向。R9验证：纯自由模式（11字符system+一句话user）产出的创意质量远高于注入搜索数据/格式模板/结构引导的模式
2. **市场调研必须做**——发散前WebSearch 3轮+产出借鉴点/避雷点清单。发散后用清单做事后校准（踩避雷点=淘汰，用到借鉴点=加分）。R13/R14《诡编》级别好创意的另一个关键就是市场校准
3. **Phase 1必须产出≥5个差异化方向**——不得只给3个，不得同赛道，每句话读完脑子里立刻浮现场景
4. **Phase 2只在用户选定后启动**——不在发散阶段做结构化
5. **金手指三要素齐全**——创意×机制×限制缺一不可，禁止只加限制不加代价
6. **四眼法四项全过才放行**——画面/限制/场景量/行为引擎
7. **只产出创意.md+ch001.md**——不写骨架/世界规则/节奏框架/笔触风格/四张地图/敌人梯度（这些移交plot）
8. **首章必须好看**——不是事件展示，是让读者想看第二章的开篇。必须有一个"生活化场景×金手指"的反差画面，纯战斗展示/纯穿越事件不算合格首章
9. **番茄简介必须是散文体**——带对话、带情绪弧线、带钩子，不是概述/设定说明书
10. **创意不得与番茄Top20核心机制雷同**——市场校准的避雷点校验必须执行

## 3b. 黄金开篇法则

首章必须遵循番茄黄金开篇法则：

1. **前3句扔炸弹** — 第一句话就制造冲突/悬念/反差，禁止铺垫环境/背景/天气
2. **300字内完成4件事** — 冲突爆发+主角出场+金手指激活+章末钩子
3. **第一章必须激活金手指** — 不能等到第二章才让读者看到金手指
4. **第一章必须有1个爽感触发** — 压力突破/弱→强/未知揭示/被欺→反击
5. **章末必须有钩子** — 让读者想看第二章的悬念/威胁/转折
6. **必须有1个"生活化场景×金手指"的反差画面** — R13/R14好开篇的核心特质。纯战斗展示/纯穿越事件不算合格首章

### 法则6详解：反差画面（R13/R14核心特质）

**反差画面 = 生活化场景 × 金手指**

R13/R14的好开篇之所以好看，是因为金手指触发在一个**日常可感的生活场景**里，制造了荒谬反差：

| 样本 | 生活化场景 | 金手指 | 反差画面 |
|------|----------|--------|---------|
| R14《诡编》 | 推开凶宅大门（日常场景） | 篡改诡异属性面板 | 女鬼扑来，我把她"暴躁"改成"正在织毛衣"，她真的开始织毛衣 |
| R13张平凡 | 加班骑车回家（日常场景） | 公文语成真 | 黄鼠狼拦路讨封，用红头文件格式回答，它必须照办 |
| 反面R37 | 跳恒河穿越（非日常） | 系统看忠诚度 | 无生活化场景，无反差 |
| 反面R38 | 沙滩苏醒杀怪（非日常） | 基因种子烧蝌蚪 | 无生活化场景，无反差 |

**反差画面设计方法**：
- 找一个主角金手指能触发的**日常可感场景**（上班/回家/买菜/相亲/加班/收快递）
- 让金手指在这个场景里**以最荒谬的方式触发**
- 画面要有**具体动作+具体台词+具体物象**（毛线球/红头文件/属性面板）

**禁止的开篇类型**：
- 纯战斗展示（苏醒→杀怪→杀怪）— 无生活化
- 纯穿越事件（跳河→机场→被抢）— 无反差
- 纯设定铺陈（介绍世界观→介绍金手指→介绍主角）— 无画面

## 3d. 笔触层（三态协议，同步 write v8.0.0 减法版）

> write v8.0.0 做减法移除了 8 项内置风格倾向（短句为主/道引导词/【】面板/动作链必选/多感官必选/感官优先级表/情绪外化/碎片化思考），笔触层交给 DNA。首章作为 write 的 ch001，遵循同一三态协议。

| 状态 | 条件 | 首章笔触行为 |
|:--|:--|:--|
| 启用态 | `涌现/文风锚定.md` 存在 | 笔触层从 DNA 取，禁止凭空发挥 |
| 缺失态 | `涌现/文风锚定.md` 不存在 | 提示用户部署 DNA，用户拒绝则进 trial 模式 |
| trial 模式 | 用户拒绝部署 | 不内置任何默认风格，用户须显式声明基础风格倾向 |

**启用态**：加载 `涌现/文风锚定.md`，首章的句式/感官/叙事距离/对话/情绪外化/段落呼吸从 DNA 取。DNA 无系统面板格式设定时，首章不得擅自加面板。

**trial 模式**：首章笔触由用户显式声明（如"简洁短句+无面板+动作驱动"），不内置 write v7.2.0 的默认风格。如用户不声明，接受 API 自由发挥的质量波动。

> 注：首章是 seed 产出，不是 write 产出。但首章的笔触协议与 write v8.0.0 一致——write v7.2.0 的动作链/多感官/感官优先级表/【】面板/碎片化思考等已移除，不再作为首章硬约束。

## 3e. 爽感闭环（必须）

首章必须有1个爽感触发，且必须是**触发→爆发→后果**三段式：

- **触发**：主角面对压力/被欺/未知/弱者处境
- **爆发**：金手指激活，反转/突破/揭示/反击
- **后果**：爆发后的可见变化（诡异被改写/敌人失态/主角获得反馈）

三段缺一=废章。

## 创意.md

# 最后的火泉守夜人

## 一句话简介
永夜降临三年，主角是守护最后一口火泉的守夜人，每天用配额分配维持脆弱的生存秩序，直到他发现火泉底下藏着一个秘密——他不是在守护火泉，火泉是在守护他。

## 创意
一个守夜人管理最后一口火泉的日常（配额分配/维修火泉罩/调解争抢），发现火泉有生命特征——它在“呼吸”，每次呼吸匹配一次资源危机。

## 创意来源
用户选定【猎奇1：最后的火泉守夜人】。Phase 1发散自永夜末世+资源管理赛道的交叉点，市场调研确认“修仙+末世”赛道在番茄有稳定读者盘（100-500W区间），但“资源管理型末世”供给偏少，竞品多走战斗生存流。选定此方向。

## 金手指
- **创意**：主角能感知火泉的“心跳”——火泉不是死资源，是有生命周期的地脉心脏。
- **机制**：火泉的“心跳”频率=火泉剩余寿命/火泉状态/火泉喜恶。主角通过对火泉心跳的感知和调配（今天多给A区配额，明天少给B区），能“哄”火泉多产出/延迟衰竭/发现隐藏层。
- **限制**：感知火泉心跳消耗主角的体温和寿命——每一次“心跳共鸣”都会让主角体表温度下降0.5℃，累积超过15℃（即30次）主角会低温昏迷甚至冻死。且火泉有嫉妒心——发现主角对其他火泉或人太亲近会“发脾气”（产出骤降/脉动紊乱）。
- **合成**：通过对火泉心跳的感知和配额调配来“哄”火泉多产出或发现隐藏资源，但每次使用共鸣都会消耗主角的体温和寿命，且火泉会嫉妒主角分心。

## 行为引擎
每章主角在做一个具体的“守夜人”行为：分配配额/维修火泉罩/调解冲突/感知心跳/勘探附近是否有新火泉/与抢火泉的人谈判——不是打怪升级，是资源管理+政治博弈。

## 轻量主角轮廓
- **名字**：苏远
- **身份**：永夜三年幸存者，原城市供暖系统工程师，现在是最后一口“母亲火泉”的守夜人
- **核心动机**：火泉衰竭他会死——但他发现更可怕的是，火泉衰竭之前，他先得活下去
- **记忆点**：永远穿着左臂缝了七道疤的橙色工装棉袄，铁壶不离手——那是他死去的搭档留下的

## 番茄简介

我叫苏远，永夜第三年，我是最后一口火泉的守夜人。

工作内容挺简单的：每天给十七个聚居区分配火泉配额，修一修防护罩上的裂缝，再跟那些想插队的家伙说“排队”。

直到今天，我把耳朵贴上火泉地面，听见了“咚——咚——咚——”。

三年来第一次。火泉有心脏。

我跟我搭档——铁壶里的热水说：“它在跳。”

热水没理我。

我试着多给A区配了半斤火泉配额，火泉的心跳变快了——它在高兴？

我又试着少给B区配了一两——心跳慢了一点，但更沉了——它在生气？

我盯着那团拳头大的火光，突然想到一个问题：

永夜三年，所有火泉都灭了，就剩这一口。

这玩意儿不是我自己找到的——是它选的。它在三年前从地下喷出来，炸了我搭档的帐篷，把我震晕过去，然后用一根火线拖着我走了三公里。

等我醒来，我已经是守夜人了。

我以为是运气。现在我知道了——不是我在守护火泉。是火泉在守护我。它要干什么？

我当然查了。地下三千米，四十七具骸骨，全部穿着和我一样的橙色工装棉袄。骨头上刻着同一行字——“守夜人序列·编号遗忘”。

最底下那具骸骨手里握着一本日记，第一页写着：

“第一百三十七代守夜人留。火泉需要心跳，如果没有人给它心跳，它就会自己长出一个。”

## 十、故事纲领（三核心+营销层）

### 什么样的世界
- **差异层**：永夜降临三年，太阳不再升起，世界被冻土和黑暗覆盖。火泉——偶尔从地裂缝中喷出的光热竖井——是唯一能让人类在零下五十度生存的能源。火泉是有生命周期的：出生→旺盛→衰竭→熄灭，每一口火泉平均寿命是五到十年。
- **日常层**：普通人靠火泉配额活命。火泉每产生一份光热，就会消耗地底的某种“地脉”。聚居区根据火泉大小分配配额，配额不够的人就在外围冻死。没有火泉的地方是绝对死域——一小时内冻成冰雕。普通人日常在干的事：排队领配额、修保温棚、找柴火（柴火能多扛半小时）、祈祷明天的配额还有。
- **主角层**：主角是“母亲火泉”的守夜人——这是最后一口还在旺盛期的火泉，其他火泉都在衰竭。他能听到火泉的“心跳”，这个能力让他比任何人都了解火泉的状态，但也让他成为所有势力的目标——有人想抢火泉，有人想毁灭火泉，还有人想通过火泉找到永夜的原因。

### 什么样的舞台·世界危机
- **危机源头**：永夜不是自然现象，是“地脉枯竭”——整个大陆的地下热能被某远古规则抽走用于封印一个名为“冻土之核”的东西。火泉是封印破碎前溢出的最后热量。当所有火泉熄灭，封印完全破碎，冻土之核会苏醒——世界将变成永恒冻结，连火泉都不会再有。
- **敌人梯度（三层+隐藏层）**：
  - 表层：聚居区抢火泉的幸存者/想偷袭守夜人的小团伙（为了生存）
  - 中层：暗夜商会（垄断其他火泉资源，想吞并母亲火泉扩势力）/守夜工会黑手套（有人背叛了守夜人序列，和永夜教会勾结）
  - 深层：永夜教会（不是信永夜，是信“冻土之核是救赎”——他们想让封印完全破碎，让世界进入永恒冻结）
  - 隐藏层：母亲火泉本身（它的心跳是有意识的——它想做什么？为什么选苏远？它和冻土之核是什么关系？）
- **压力源/倒计时**：①母亲火泉的心跳在变慢——每次减速说明衰竭进度+5%，最快三个月后衰竭→熄灭；②苏远每次感知心跳体温下降0.5℃，累积≤15℃（30次极限）；③永夜教会每三个月献祭一位守夜人来加速封印破碎。

### 什么样的主角
- **起点**：永夜三年的守夜人，穿着搭档留下的橙色工装棉袄，每天管理十七个聚居区的火泉配额。表面上是冷静的资源管理员，骨子里还困在三年前搭档被火泉吞没的夜晚——他不敢离火泉太近，也不敢离火泉太远。初始能力：只能模糊感知火泉有无心跳，无法理解心跳在说什么。
- **拐点**：发现火泉心跳=火泉寿命不止是数值，还是“活的”。铁壶里的热水（搭档的骨灰/意识残留）告诉他：“你不是听火泉，你是和它对话。每说一句，你就冷一分。但如果你不说，它会冷二十年。”——苏远开始主动感知心跳，被卷入“火泉选守夜人”的千年博弈。
- **终点**：在母亲火泉熄灭之前，要么找到让火泉恢复的方法（打破封印/唤醒冻土之核的真身），要么成为新的“火泉之源”——将自己的体温和生命反向注入火泉，变成活着的火泉守夜人（不死不灭，代价是永远不能离开火泉三公里）。最终选择：他成为了百年未有的“共振型守夜人”——他能和火泉对谈，也让火泉能通过他说话。
- **力量路径**（三级映射三幕）：
  - 第一幕（1-30章）：**听心跳**。能力：能听出火泉的“情绪”（高兴/生气/疲倦/饥渴等四种状态）→对应场景：苏远通过听心跳发现母亲火泉最近“高兴”的时候是有人送它一块地脉石，“生气”的时候是有人往它附近埋金属垃圾——他开始用投喂/清理来哄火泉多产出，解决定居区的配额危机。
  - 第二幕（31-60章）：**引导心跳**。能力：能用配额的增减和投喂物的选择来“引导”火泉的脉动节奏，让火泉主动产出更多热量/减缓衰竭/甚至在地底“嗅”到隐藏矿脉→对应场景：暗夜商会派人潜伏在聚居区想挖母亲火泉的地脉，苏远通过引导火泉的深层脉动，让火泉在地底三千米的某个位置“打了一个喷嚏”——震塌了商会的挖道，把敌人活埋。同时感知到地底有异常——那是永夜教会的据点。
  - 第三幕（61-100章）：**共振心跳**。能力：苏远的体温和火泉的心跳完全同步——他能通过自己的身体感知火泉感知到的一切，也能让火泉通过他“说话”——但每次共振消耗等价于24小时的寿命。→对应场景：永夜教会推进最后献祭，苏远必须进入火泉内部进行“深层对话”，和冻土之核（被封印的远古存在）谈判。最终发现母亲火泉是“锁”的内核——冻土之核被锁在火泉里，火泉不是生命体，是容器。

### 最大钩子
“不是我在守护火泉，是火泉在守护我。”

### 即时兑现感
第1章：苏远日常分配配额，发现火泉心跳异常（第一声“咚”）→第2章：用配额调配验证心跳规律，发现火泉有喜恶（可以哄）→第3章：暗夜商会派人抢火泉，苏远用火泉“愤怒”的脉动震死人→第4章：铁壶热水（搭档意识）揭示真相：火泉在选守护者，苏远是第一百三十七代→第5章：火泉内部探索，发现冻土之核的线索——永夜不是偶然。
