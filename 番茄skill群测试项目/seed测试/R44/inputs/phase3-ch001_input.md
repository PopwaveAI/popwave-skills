## 红线

1. **发散prompt必须纯自由**——不注入搜索数据/格式模板/结构引导/借鉴点/避雷点，只给一句话方向。R9验证：纯自由模式（11字符system+一句话user）产出的创意质量远高于注入搜索数据/格式模板/结构引导的模式
2. **市场调研必须做**——发散前WebSearch 3轮+产出借鉴点/避雷点清单。发散后用清单做事后校准（踩避雷点=淘汰，用到借鉴点=加分）。R13/R14《诡编》级别好创意的另一个关键就是市场校准
3. **Phase 1必须产出≥5个差异化方向**——不得只给3个，不得同赛道，每句话读完脑子里立刻浮现场景
4. **Phase 2只在用户选定后启动**——不在发散阶段做结构化
5. **金手指三要素齐全**——创意×机制×限制缺一不可，禁止只加限制不加代价
6. **四眼法四项全过才放行**——画面/限制/场景量/行为引擎
7. **只产出创意.md+ch001.md**——不写骨架/世界规则/节奏框架/笔触风格/四张地图/敌人梯度（这些移交plot）
8. **首章必须好看**——不是事件展示，是让读者想看第二章的开篇。必须有一个"生活化场景×金手指"的反差画面，纯战斗展示/纯穿越事件不算合格首章
9. **番茄简介必须是散文体**——带对话、带情绪弧线、带钩子，不是概述/设定说明书
10. **创意不得与番茄Top20核心机制雷同**——市场校准的避雷点校验必须执行

## 定位

黄金首章是**试读章**——目的是验证"这个创意能不能抓住读者"。不是精雕细琢的正式版（正式版由write按plot的剧情白描产出），但必须是一篇**能让读者想看第二章**的开篇。

**首章不好看=创意死刑**。R37/R38测试发现的首章退化根因：纯事件展示（穿越→被抢劫→找大D）或纯战斗展示（苏醒→杀怪→杀怪），没有"生活化场景×金手指"的反差画面。首章必须好看。

## 3b. 黄金开篇法则

首章必须遵循番茄黄金开篇法则：

1. **前3句扔炸弹** — 第一句话就制造冲突/悬念/反差，禁止铺垫环境/背景/天气
2. **300字内完成4件事** — 冲突爆发+主角出场+金手指激活+章末钩子
3. **第一章必须激活金手指** — 不能等到第二章才让读者看到金手指
4. **第一章必须有1个爽感触发** — 压力突破/弱→强/未知揭示/被欺→反击
5. **章末必须有钩子** — 让读者想看第二章的悬念/威胁/转折
6. **必须有1个"生活化场景×金手指"的反差画面** — R13/R14好开篇的核心特质。纯战斗展示/纯穿越事件不算合格首章

### 法则6详解：反差画面（R13/R14核心特质）

**反差画面 = 生活化场景 × 金手指**

R13/R14的好开篇之所以好看，是因为金手指触发在一个**日常可感的生活场景**里，制造了荒谬反差：

| 样本 | 生活化场景 | 金手指 | 反差画面 |
|------|----------|--------|---------|
| R14《诡编》 | 推开凶宅大门（日常场景） | 篡改诡异属性面板 | 女鬼扑来，我把她"暴躁"改成"正在织毛衣"，她真的开始织毛衣 |
| R13张平凡 | 加班骑车回家（日常场景） | 公文语成真 | 黄鼠狼拦路讨封，用红头文件格式回答，它必须照办 |
| 反面R37 | 跳恒河穿越（非日常） | 系统看忠诚度 | 无生活化场景，无反差 |
| 反面R38 | 沙滩苏醒杀怪（非日常） | 基因种子烧蝌蚪 | 无生活化场景，无反差 |

**反差画面设计方法**：
- 找一个主角金手指能触发的**日常可感场景**（上班/回家/买菜/相亲/加班/收快递）
- 让金手指在这个场景里**以最荒谬的方式触发**
- 画面要有**具体动作+具体台词+具体物象**（毛线球/红头文件/属性面板）

**禁止的开篇类型**：
- 纯战斗展示（苏醒→杀怪→杀怪）— 无生活化
- 纯穿越事件（跳河→机场→被抢）— 无反差
- 纯设定铺陈（介绍世界观→介绍金手指→介绍主角）— 无画面

## 3d. 笔触层（三态协议，同步 write v8.0.0 减法版）

> write v8.0.0 做减法移除了 8 项内置风格倾向（短句为主/道引导词/【】面板/动作链必选/多感官必选/感官优先级表/情绪外化/碎片化思考），笔触层交给 DNA。首章作为 write 的 ch001，遵循同一三态协议。

| 状态 | 条件 | 首章笔触行为 |
|:--|:--|:--|
| 启用态 | `涌现/文风锚定.md` 存在 | 笔触层从 DNA 取，禁止凭空发挥 |
| 缺失态 | `涌现/文风锚定.md` 不存在 | 提示用户部署 DNA，用户拒绝则进 trial 模式 |
| trial 模式 | 用户拒绝部署 | 不内置任何默认风格，用户须显式声明基础风格倾向 |

**启用态**：加载 `涌现/文风锚定.md`，首章的句式/感官/叙事距离/对话/情绪外化/段落呼吸从 DNA 取。DNA 无系统面板格式设定时，首章不得擅自加面板。

**trial 模式**：首章笔触由用户显式声明（如"简洁短句+无面板+动作驱动"），不内置 write v7.2.0 的默认风格。如用户不声明，接受 API 自由发挥的质量波动。

> 注：首章是 seed 产出，不是 write 产出。但首章的笔触协议与 write v8.0.0 一致——write v7.2.0 的动作链/多感官/感官优先级表/【】面板/碎片化思考等已移除，不再作为首章硬约束。

## 3e. 爽感闭环（必须）

首章必须有1个爽感触发，且必须是**触发→爆发→后果**三段式：

- **触发**：主角面对压力/被欺/未知/弱者处境
- **爆发**：金手指激活，反转/突破/揭示/反击
- **后果**：爆发后的可见变化（诡异被改写/敌人失态/主角获得反馈）

三段缺一=废章。

## 创意.md

# 宗门公务员：修仙界底层改革

## 一句话简介
灵根废柴的陈渡被踢到濒临倒闭的外门杂物科当科长。他不会修仙，但会用现代管理学原理和绩效考核表，对腐朽的宗门体制进行一场自上而下的降维打击改革。

## 创意
凡人管理×修仙宗门——一个灵根废柴用绩效考核、资源整合、人事制度，把一盘散沙的外门杂物科改造成宗门最赚钱的部门，逼着各峰长老们主动内卷。

## 创意来源
Phase 1发散方向：王道1——宗门公务员：修仙界底层改革（荒诞轻喜剧+权谋博弈）。用户选定该创意后进入Phase 2打磨。

## 金手指
- **创意**：陈渡拥有一本“宗门管理手册”——不是修炼功法，而是一套动态更新的现代管理学工具书，包括KPI制定工具、资源整合算法、人事博弈模型、财务审计模板。
- **机制**：手册根据他正在处理的管理问题自动生成对应方案（如“处理旷工弟子→生成《积分考勤制度》”、“谈判丹方提成→生成《利益分配博弈模型》”）。方案执行成功后，手册解锁更高级的管理工具（从部门管理升到战略管理）。
- **限制**：每一项管理改革都会产生“阻力值”和“信任值”。阻力值来自既得利益者（长老、受贿管事），达到阈值会引发弹劾/暗杀/罢工。信任值来自手下执行者，低于阈值时弟子集体摸鱼/架空/告密。主角必须在改革的同时平衡这两条红线，否则改革提前崩盘。
- **合成**：陈渡手中的《宗门管理手册》能根据管理问题生成现代管理方案，每次成功执行解锁更高级工具——但每一个决策都会在宗门权力棋局中增加“阻力值”或“信任值”，他不只要解决问题，还要在对立势力反扑之前稳住基本盘。

## 行为引擎
陈渡每章在做一件具体的管理事务：开分工会、制定规章、资源盘点、谈判拉赞助、人事任免、绩效面谈、跨部门撕资源、写工作总结应付上峰检查。每章解决一个管理难题，同时为下一章的难题埋下伏笔。

## 轻量主角轮廓
- **名字**：陈渡
- **身份**：被踢到外门杂物科的凡人科长，宗门最底层的“养老”职位
- **核心动机**：本只想混吃等死攒够钱下山，结果发现再不动手杂物科三个月后就要撤编，全员发配矿洞——被迫反抗
- **记忆点**：推不存在的眼镜。说话喜欢用“我们把这个项目……”、“我建议按流程走……”、“根据季度考核结果……”。喝劣质灵茶时会皱眉，因为茶水自带铁锈味。

## 番茄简介

我叫陈渡，灵根资质测试是下下等——下到连测灵石都懒得发光那种。

别的弟子在练剑、炼丹、闭关突破，我被一脚踢到外门杂物科当科长。管一群被各峰淘汰的老弱病残，负责扫厕所、修水管、喂灵兽。

杂物科快倒闭了。

我去找丹峰借点灵石发工资，长老连门都没开，隔着门板说：“没钱，你们自己想办法。哦对了，下个月的丹方渣滓也得你们一块打包带走。”

我站在门外，看着手里快要转不起来的预算表，叹了口气。

然后我打开了《宗门管理手册》——我穿越时带过来的管理学教材。

一个月后，杂物科的弟子开始了三班倒流水线：第一班修水管兼卖茶水，第二班用丹方渣滓炼废丹转售，第三班承包了全宗的卫生审计。

丹峰长老瞪着我的月报，上面写着：“贵峰丹方损耗率达37%，建议引入我们的精益炼制方案。分成比例六四，我们六。”

他把茶杯摔了。

再过两个月，外门弟子人手一份《KPI考核指南》；执法堂的兄弟悄咪咪问我能不能给他们做一套稽查流程优化；藏经阁的扫地老头递来一本残卷，说只要我答应把后勤系统改革方案给他徒弟，这残卷就是我的。

我说：“行，先签合同。”

他愣了：“修仙界不流行签合同。”

我推了推不存在的眼镜：“现在流行了。”

三个月后，各峰长老联名上书要罢免我。理由是——杂物科太赚钱了，影响了宗门权力平衡。

但他们不知道，我已经把考勤系统悄悄接入了宗门核心阵法。

我坐在杂物科的破椅子上，翘着二郎腿，对手下说：“传我话。告诉各位长老——要么接受外门改革方案，全员KPI引入；要么我关掉阵法，让全宗连口水都没得喝。”

门外的风，吹进来一股灵茶的香气。

——铁锈味儿，终于没了。

## 十、故事纲领（三核心+营销层·≤1000字）

### 什么样的世界

**差异层**：一个以“修仙天赋”为唯一晋升标准的宗门社会。灵根决定一切——灵根差的弟子只能做杂役，灵根好的弟子自动获得资源倾斜。但隐藏规则是：宗门运转依赖的不是修士的个人战力，而是无数底层杂役维持的后勤体系——灵谷种植、丹坊废料处理、灵石矿脉维护、基础设施修缮。这些“凡人之事”被高层视为理所当然，实则决定宗门存亡。

**日常层**：外门弟子每天三班倒维持宗门运转。灵根差的干杂活，灵根好的只管修炼。杂役没有晋升通道，干到死只能领三份灵石工资。偶尔抗议会被执法堂以“荒废修炼”为由罚去矿洞。弟子们最大的愿望是攒够灵石下山开个铺子，而不是在宗门里熬到死。

**主角层**：陈渡是灵根废柴中的废柴，被宗门当做包袱踢到最垫底的杂物科。但他带来的不是修仙天赋，而是一套完整的现代管理学知识体系——他对接的不是“天地灵气”，而是“人心、资源、制度、博弈”。他的生存法则和宗门原有规则完全不在一个维度上：别人看灵根资质，他看人力成本；别人看境界突破，他看绩效考核；别人看宗门气运，他看着财务赤字流汗。

### 什么样的舞台·世界危机

**全书危机本质**：宗门表面光鲜，实则财政连年亏损。灵脉枯竭、丹方低效、弟子怠工、长老山头林立、外敌势力渗透——宗门气运池处于历史低位。如果不改革，少则十年多则二十年，沧云宗将从一流宗门跌入末流，被周边势力吞并。

**危机源头**：宗门开山祖师建立的“灵根至上”体制在繁荣期效率极高，但几百年下来制度僵化。长老们垄断资源，谁动谁就会被弹劾；底层弟子没有上升通道，混吃等死；中层管事吃拿卡要，腐败成风。宗门像一个生锈的机器，零件还在转，但效率逐年下跌。最致命的是——没人愿意承认问题出在体制上，所有人都在说“下一代灵根好的弟子会救宗门的”。

**敌人梯度**：
- **表层敌人（1-30章）**：杂物科内部问题——懒散弟子、贪污的老管事、不配合的跨部门同事。是“人性之恶”层面的对手。
- **中层敌人（31-60章）**：外门既得利益群体——贪腐的丹峰管事、偷灵石的执法堂小队长、靠关系占着闲职的家族子弟。是“体制漏洞”层面的对手。
- **深层敌人（61-90章）**：内门长老院——他们不是坏人，但改革会削他们的权、断他们的财路。他们动动嘴皮子，陈渡的提案就过不去。是“制度设计漏洞”层面的对手。
- **顶层敌人（91-100章+）**：宗门规则本身——当陈渡的改革触及到“灵根决定论”的根基时，他将面对的是几百年沉淀下来的宗门意识形态和既得利益集团的反扑。同时外敌势力（其他宗门/魔道）趁机渗透，内外夹击。

**压力源/倒计时**：杂物科三个月后撤编——陈渡和他手下全员发配黑风矿洞（死亡率30%+）。这是陈渡必须改革的直接压力。后续压力升级为宗门财政红线（每月赤字超支就会被内门问责）、每季度一次的内门巡察、不定期的宗门内斗外敌入侵。

### 什么样的主角

- **起点**：陈渡，灵根下下等的外门杂物科科长。手下二十多个老弱病残，预算赤字，没有修炼资源，连宗门食堂的杂役都看不起他。初始态度是“混完三年存够钱就下山”，能不动就不动，能用嘴皮子解决的事绝不动手。

- **拐点**：发现杂物科三个月后即将撤编，全员发配矿洞。同时意外触发《宗门管理手册》（他穿越时带过来的），手册第一个任务弹出：“请在三日内完成部门资源盘点与人员结构评估报告，作为改革第一份数据基础。”——这是他第一次意识到，自己带来的东西，在这个世界真的有效。

- **终点**：从一个只想躺平的废灵根，变成能够撼动宗门体制的改革者。他不是在修仙，而是在修“体制”。最终目标不是突破境界，而是用一套新的制度，把宗门从崩溃边缘拉回来——哪怕这个过程会让他成为所有既得利益者的众矢之的。

- **力量路径**（每级=一幕）：
  - **第一幕（1-30章）：部门自治级管理**。能力：资源盘点、KPI制定、流程再造。场景：完成杂物科内部改革（人员重新分工→制定考勤制度→建立积分奖惩→谈判丹方渣滓采购→实现部门盈利）。幕终点：杂物科扭亏为盈，陈渡初获手下的信任和上层的注意。
  - **第二幕（31-60章）：外门整合级管理**。能力：跨部门谈判→利益博弈模型→人才发掘→财务审计。场景：陈渡的管理权限被扩大到整个外门后勤系统。他开始整合灵谷种植、灵兽饲养、丹坊废料处理三个板块，建立外门后勤一体化。幕终点：外门后勤体系效率翻倍，陈渡触怒了丹峰和灵兽峰的利益链，正式进入长老院的视野。
  - **第三幕（61-90章）：宗门体制级管理**。能力：战略规划→制度改革→权力博弈→危机应对。场景：长老院开始明面阻挠改革。陈渡需要对抗制度惯性、既得利益集团的反扑、外部势力的渗透。幕终点：陈渡的提案获得宗主的暗中支持，但内门长老联合抵制，改革陷入拉锯战。
  - **第四幕（91-100章+）：棋局级博弈**。能力：规则重构→顶层博弈→内外夹击应对。场景：外敌势力借机渗透宗门，陈渡的改革被误解为“通敌”。他必须在自证清白的同时，稳住改革成果，最终面对宗门体制的核心问题。

### 最大钩子
用KPI卷死修仙界——废灵根管理活棋局。

### 即时兑现感
第1章陈渡接手杂物科，一群人瘫在破院子里晒太阳，老管事嘲笑他“这位置就是个养老坑” → 第2章他用《资源盘点表》发现大家私藏灵谷，一纸“多劳多得积分制”让躺平弟子抢着干活 → 第3章他拿着丹方渣滓和丹峰管事谈合作，对方被他的“精益炼制方案”整懵了，第一次觉得这个凡人有点意思 → 第4章手下弟子开始主动找他要活干，陈渡看着月报表上终于不飘红的数字，笑了 → 第5章内门巡察使路过杂物科，看到满墙的KPI进度板和整洁的仓库，皱了皱眉，说：“这事要上报。”
