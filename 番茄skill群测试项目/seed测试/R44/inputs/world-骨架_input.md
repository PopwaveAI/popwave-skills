# Step 2 · 从创意生长世界（内化）

> **核心原则：世界不是填表填出来的，是从创意核心要素反向推导出来的。**
>
> **力量体系是骨架，地图/势力/危机是血肉——血肉塞入骨架。**（详见 pop-decon-setting v3.3.0 的骨架/血肉两层结构设计）

## 推导链

```
创意.md核心要素 + 赛道已验证模式 + 参考书DNA
        ↓
    力量体系 ──→ 地图 ──→ 势力 ──→ 危机
```

## 力量体系设计起点判定

| 本项目类型 | 设计起点 | 例 |
|:--|:--|:--|
| 系统流 | 金手指机制是主角优势，体系来自世界规则或赛道惯例 | 深渊主宰：DND体系是世界观自带 |
| 修炼流 | 世界观自带的修炼体系，金手指只是加速器 | 凡人修仙传：炼气→筑基→金丹→元婴 |
| 规则系 | 世界规则就是力量体系，金手指只是让你看规则 | 诡舍：规则怪谈世界的规则本身 |
| 无金手指 | 世界观/赛道惯例直接决定 | — |

## 2a. 三源合流设计力量体系

### 第一源：赛道参考（`写作参考/市场校准.md`，如有）
- top作品的力量体系类型（等级制/修炼境/规则系/百分比/职业树）
- 读者疲劳的模式（什么已被写烂了，必须避开）
- 正在冒头的变体（赛道演变方向）

### 第二源：参考书DNA（`涌现/文风锚定.md` + `涌现/剧情DNA-brief.md`，如有）
- 文风锚定：参考书的力量体系风格
- 剧情DNA-brief·Boss梯度：每层boss的强度分布+出场节奏
- 剧情DNA-brief·主角成长曲线：破局手段演变（前期靠什么/中期靠什么/后期靠什么）

**消费规则**：学设计思路和强度分布节奏，不抄具体等级名。

### 第三源：创意.md
- 金手指（如有）：机制和限制决定主角在体系中的**优势方式和成长节奏**（不是决定体系本身）
- 行为引擎：决定**触发升级的条件**（不是靠刷怪，靠对赌/经营/探索？）
- 前期成长路线：决定每卷的力量跨度
- 世界观规则（如有）：规则系优先于金手指

### 三源合流产出：力量层级表

展开5-7个力量层级，每个层级必须：

| 字段 | 含义 | 要求 |
|:--|:--|:--|
| 名称 | ≤6字，贴合本项目世界观 | 不套"DND/炼气筑基"模板 |
| 这个世界长什么样 | MMO式的等级区描述 | 新手区→城镇区→高级区 |
| 主角在这个层级面对什么 | 不是"解锁什么技能" | 这个层级的世界是什么样的 |
| 升级触发条件 | 从行为引擎推导 | 不是"打够怪就升级" |
| 参考源 | 标注参考了哪个源 | 赛道·X / 参考书·X / 创意·X |

**力量跨度**：第一卷从阶位1到阶位N（对齐创意.md前期成长路线的街头期上限）

## 2b. 从力量体系+行为引擎生长地图

推导逻辑（MMO式）：
```
力量层级 → 每个阶位的"区域主题" → 城市/世界地图
  阶位1："新手区" → 贫民区+赌场
  阶位2："城镇区" → 码头+商业街+公会
  阶位3："高级区" → 贵族区+神殿区+地下城
```

每个区域回答：
- 对应哪个力量阶位？
- 主角的**核心行为**是什么？（从行为引擎推导）
- 有什么**信息差**可以利用？

输出：城市结构表（区域/对应阶位/主角行为/信息差来源/势力归属/出现幕）

## 2c. 从力量体系+地图生长势力

推导逻辑：
```
力量阶位1→贫民区→谁控制这里的资源？→街头帮派
力量阶位2→码头+商业街→谁控制贸易？→地方商会+走私网
力量阶位3→贵族区→谁控制制度？→贵族家族+执法系统
力量阶位4→隐藏区域→谁在幕后？→神秘组织
```

每层势力必须回答：
1. **为什么主角必须和这个势力冲突？**（不解决它就无法进入下一阶位）
2. **主角靠什么赢？**（阶位1靠硬刚→阶位2靠信息差→阶位3靠制度漏洞）
3. **赢了得到什么？**（地盘/人脉/信息/身份——进入下一阶位的钥匙）

## 2d. 危机体系设计（四源）

| 危机来源 | 说明 | 必选 |
|:--|:--|:--|
| **世界固有危险** | 创意.md设定的世界级威胁——全书暗线 | ✓ |
| **力量体系内压** | 进入更高阶位必然面对的压力（强者注目/势力反弹/制度反噬） | ✓ |
| **金手指反噬** | 金手指限制条款触发的危机 | 如有金手指 |
| **赛道惯例压力** | 赛道读者期待的特定压力类型（如诡异流必然有"规则杀"） | 可选 |

**倒计时设计**：从危机四源推导2-3条倒计时（触发条件→到期章→后果）

## 3b. 幕序列（3-4幕）

从Step 2的敌人梯度+危机倒计时推导幕序列。每幕=一个"对赌难度升级"的弧段。

```markdown
### 第一幕：「幕名」
- **功能位**：建置（建立世界+第一次小对赌+埋大对赌的伏笔）
- **对赌难度**：LV1-2，街头级的公平对赌
- **弧线**：（一句话——从学会出千到赢下走私线，付出了什么代价）
- **章节预算**：ch001-006（ch001已由seed产出，实际写ch002-006）
- **幕末钩子**：第一幕结束时读者在想什么？

### 第二幕：「幕名」
- **功能位**：冲突升级（对赌难度升级——地方势力开始不跟你公平对赌了）
- **对赌难度**：LV2-4，需要筹码/信息差才能诱使对赌
- **弧线**：（一句话）
- **章节预算**：ch007-014

### 第三幕：「幕名」
- **功能位**：高潮（对赌难度最高——和制度权力+神秘力量对赌）
- **对赌难度**：LV4-6+，制度保护+不确定性
- **弧线**：（一句话）
- **章节预算**：ch015-022

### 第四幕：「幕名」
- **功能位**：余波+伏笔（揭示更大的对赌——世界危机）
- **对赌难度**：暗示LV8+
- **弧线**：（一句话）
- **章节预算**：ch023-025
```

每幕的设计原则：**对赌难度必须逐幕升级**。不是"等级提升"，是"公平对赌的难度提升"——街头级用小筹码就能诱使对赌，系统级需要制度缺陷/信息不对称，幕后级需要更大的赌注。

# 宗门公务员：修仙界底层改革

## 一句话简介
灵根废柴的陈渡被踢到濒临倒闭的外门杂物科当科长。他不会修仙，但会用现代管理学原理和绩效考核表，对腐朽的宗门体制进行一场自上而下的降维打击改革。

## 创意
凡人管理×修仙宗门——一个灵根废柴用绩效考核、资源整合、人事制度，把一盘散沙的外门杂物科改造成宗门最赚钱的部门，逼着各峰长老们主动内卷。

## 创意来源

- **限制**：每一项管理改革都会产生“阻力值”和“信任值”。阻力值来自既得利益者（长老、受贿管事），达到阈值会引发弹劾/暗杀/罢工。信任值来自手下执行者，低于阈值时弟子集体摸鱼/架空/告密。主角必须在改革的同时平衡这两条红线，否则改革提前崩盘。
- **合成**：陈渡手中的《宗门管理手册》能根据管理问题生成现代管理方案，每次成功执行解锁更高级工具——但每一个决策都会在宗门权力棋局中增加“阻力值”或“信任值”，他不只要解决问题，还要在对立势力反扑之前稳住基本盘。

## 番茄简介

我叫陈渡，灵根资质测试是下下等——下到连测灵石都懒得发光那种。

别的弟子在练剑、炼丹、闭关突破，我被一脚踢到外门杂物科当科长。管一群被各峰淘汰的老弱病残，负责扫厕所、修水管、喂灵兽。

杂物科快倒闭了。

我去找丹峰借点灵石发工资，长老连门都没开，隔着门板说：“没钱，你们自己想办法。哦对了，下个月的丹方渣滓也得你们一块打包带走。”

我站在门外，看着手里快要转不起来的预算表，叹了口气。

然后我打开了《宗门管理手册》——我穿越时带过来的管理学教材。

一个月后，杂物科的弟子开始了三班倒流水线：第一班修水管兼卖茶水，第二班用丹方渣滓炼废丹转售，第三班承包了全宗的卫生审计。

丹峰长老瞪着我的月报，上面写着：“贵峰丹方损耗率达37%，建议引入我们的精益炼制方案。分成比例六四，我们六。”

他把茶杯摔了。

再过两个月，外门弟子人手一份《KPI考核指南》；执法堂的兄弟悄咪咪问我能不能给他们做一套稽查流程优化；藏经阁的扫地老头递来一本残卷，说只要我答应把后勤系统改革方案给他徒弟，这残卷就是我的。

我说：“行，先签合同。”

他愣了：“修仙界不流行签合同。”

我推了推不存在的眼镜：“现在流行了。”

三个月后，各峰长老联名上书要罢免我。理由是——杂物科太赚钱了，影响了宗门权力平衡。

但他们不知道，我已经把考勤系统悄悄接入了宗门核心阵法。

我坐在杂物科的破椅子上，翘着二郎腿，对手下说：“传我话。告诉各位长老——要么接受外门改革方案，全员KPI引入；要么我关掉阵法，让全宗连口水都没得喝。”

门外的风，吹进来一股灵茶的香气。

## 十、故事纲领（三核心+营销层·≤1000字）

### 什么样的世界

**差异层**：一个以“修仙天赋”为唯一晋升标准的宗门社会。灵根决定一切——灵根差的弟子只能做杂役，灵根好的弟子自动获得资源倾斜。但隐藏规则是：宗门运转依赖的不是修士的个人战力，而是无数底层杂役维持的后勤体系——灵谷种植、丹坊废料处理、灵石矿脉维护、基础设施修缮。这些“凡人之事”被高层视为理所当然，实则决定宗门存亡。

**日常层**：外门弟子每天三班倒维持宗门运转。灵根差的干杂活，灵根好的只管修炼。杂役没有晋升通道，干到死只能领三份灵石工资。偶尔抗议会被执法堂以“荒废修炼”为由罚去矿洞。弟子们最大的愿望是攒够灵石下山开个铺子，而不是在宗门里熬到死。

**主角层**：陈渡是灵根废柴中的废柴，被宗门当做包袱踢到最垫底的杂物科。但他带来的不是修仙天赋，而是一套完整的现代管理学知识体系——他对接的不是“天地灵气”，而是“人心、资源、制度、博弈”。他的生存法则和宗门原有规则完全不在一个维度上：别人看灵根资质，他看人力成本；别人看境界突破，他看绩效考核；别人看宗门气运，他看着财务赤字流汗。

### 什么样的舞台·世界危机

**全书危机本质**：宗门表面光鲜，实则财政连年亏损。灵脉枯竭、丹方低效、弟子怠工、长老山头林立、外敌势力渗透——宗门气运池处于历史低位。如果不改革，少则十年多则二十年，沧云宗将从一流宗门跌入末流，被周边势力吞并。

**危机源头**：宗门开山祖师建立的“灵根至上”体制在繁荣期效率极高，但几百年下来制度僵化。长老们垄断资源，谁动谁就会被弹劾；底层弟子没有上升通道，混吃等死；中层管事吃拿卡要，腐败成风。宗门像一个生锈的机器，零件还在转，但效率逐年下跌。最致命的是——没人愿意承认问题出在体制上，所有人都在说“下一代灵根好的弟子会救宗门的”。

**敌人梯度**：
- **表层敌人（1-30章）**：杂物科内部问题——懒散弟子、贪污的老管事、不配合的跨部门同事。是“人性之恶”层面的对手。
- **中层敌人（31-60章）**：外门既得利益群体——贪腐的丹峰管事、偷灵石的执法堂小队长、靠关系占着闲职的家族子弟。是“体制漏洞”层面的对手。
- **深层敌人（61-90章）**：内门长老院——他们不是坏人，但改革会削他们的权、断他们的财路。他们动动嘴皮子，陈渡的提案就过不去。是“制度设计漏洞”层面的对手。
- **顶层敌人（91-100章+）**：宗门规则本身——当陈渡的改革触及到“灵根决定论”的根基时，他将面对的是几百年沉淀下来的宗门意识形态和既得利益集团的反扑。同时外敌势力（其他宗门/魔道）趁机渗透，内外夹击。

**压力源/倒计时**：杂物科三个月后撤编——陈渡和他手下全员发配黑风矿洞（死亡率30%+）。这是陈渡必须改革的直接压力。后续压力升级为宗门财政红线（每月赤字超支就会被内门问责）、每季度一次的内门巡察、不定期的宗门内斗外敌入侵。

### 什么样的主角

- **起点**：陈渡，灵根下下等的外门杂物科科长。手下二十多个老弱病残，预算赤字，没有修炼资源，连宗门食堂的杂役都看不起他。初始态度是“混完三年存够钱就下山”，能不动就不动，能用嘴皮子解决的事绝不动手。

- **拐点**：发现杂物科三个月后即将撤编，全员发配矿洞。同时意外触发《宗门管理手册》（他穿越时带过来的），手册第一个任务弹出：“请在三日内完成部门资源盘点与人员结构评估报告，作为改革第一份数据基础。”——这是他第一次意识到，自己带来的东西，在这个世界真的有效。

- **终点**：从一个只想躺平的废灵根，变成能够撼动宗门体制的改革者。他不是在修仙，而是在修“体制”。最终目标不是突破境界，而是用一套新的制度，把宗门从崩溃边缘拉回来——哪怕这个过程会让他成为所有既得利益者的众矢之的。

- **力量路径**（每级=一幕）：
  - **第一幕（1-30章）：部门自治级管理**。能力：资源盘点、KPI制定、流程再造。场景：完成杂物科内部改革（人员重新分工→制定考勤制度→建立积分奖惩→谈判丹方渣滓采购→实现部门盈利）。幕终点：杂物科扭亏为盈，陈渡初获手下的信任和上层的注意。
  - **第二幕（31-60章）：外门整合级管理**。能力：跨部门谈判→利益博弈模型→人才发掘→财务审计。场景：陈渡的管理权限被扩大到整个外门后勤系统。他开始整合灵谷种植、灵兽饲养、丹坊废料处理三个板块，建立外门后勤一体化。幕终点：外门后勤体系效率翻倍，陈渡触怒了丹峰和灵兽峰的利益链，正式进入长老院的视野。
  - **第三幕（61-90章）：宗门体制级管理**。能力：战略规划→制度改革→权力博弈→危机应对。场景：长老院开始明面阻挠改革。陈渡需要对抗制度惯性、既得利益集团的反扑、外部势力的渗透。幕终点：陈渡的提案获得宗主的暗中支持，但内门长老联合抵制，改革陷入拉锯战。
  - **第四幕（91-100章+）：棋局级博弈**。能力：规则重构→顶层博弈→内外夹击应对。场景：外敌势力借机渗透宗门，陈渡的改革被误解为“通敌”。他必须在自证清白的同时，稳住改革成果，最终面对宗门体制的核心问题。

# 外门杂物科，今天开始改革

“陈渡，杂物科科长。你前任上个月发了疯，说看见灵兽会说话了。那边那间房就是你的办公室，哦对了，茅房堵了三天了，你先带人通一下。”

我不推那扇门都知道——门板斜挂着，上面的漆斑驳得像走了三十年的炼丹炉底。院子里横七竖八躺着二十来号人，有人拿帽子盖着脸打鼾，有人在石桌上抠脚，还有两个老头在墙角下棋，棋子是捡来的石头，棋盘画在地上用脚一抹就没了。

我从屋里翻出上月报表，看了第一行就叹了口气。

杂物科上月支出：三百四十块灵石。其中一百块是修水管，一百二买拖把扫帚，剩下全赔给了灵兽峰——因为咱们喂猪的混小子把饲料搅错了，把灵兽峰三头灵猪喂成了废猪，赔了人家一百二。

上月收入：十八块灵石。来自帮内门弟子代领快递的小费。

净亏损：三百二十二。

我把报表折好，塞回抽屉。抽屉里还有一封长老院上周发来的公函，语气很客气——“杂物科如本季度无法实现收支平衡，建议撤销编制。所有人发配黑风矿洞，听候差遣。”

我脑子嗡嗡响。

不是说三个月后才撤编吗？怎么这信上写的是“本季度”？

我翻出日历——今天是季末最后一天。

也就是说，再不做点什么，明天太阳升起来之前，我和屋外这二十号人就该收拾铺盖滚去矿洞了。

我他妈才穿越过来三个小时。

“科长，你杵在门口干嘛？赶紧的，茅房要漫出来了。”抠脚的管事抬起头，不耐烦地喊我，“别在这站着充大爷，咱们杂物科没那闲工夫——唉不对啊，你好歹动动手，不然月底真把咱发配矿洞去了。”

我走回屋里，从怀里掏出一本旧书。这不是什么修仙功法，是我穿越之前随手塞包里的一本《企业行政管理实务》。铅印的，定价二十八块八，封面还有半个咖啡渍。

书页哗啦啦自己翻了起来。

“正在检测当前管理场景……场景识别：濒临倒闭的边缘部门。现有人员22人，平均工作年限11年，人均月产出不足灵盖峰一名杂役弟子的1/5。部门负债累计：780灵石。距离撤编倒计时：1天。”

我愣了一下，推了推鼻梁上那副不存在的眼镜。

书又翻了一页：“建议方案：《全员积分考勤制度》——根据完成的工作量分配灵石奖励，多劳多得，不劳不得。同时建立岗位责任制，将现有人员按技能重新分组。是否需要生成详细执行模板？”

我想了想说：“生成。”

就一眨眼的工夫，书的空白页上浮现出一排排清晰的字迹，像打印机扫描出来的，连页码都标好了。

我盯着那几页纸看了一会儿，忽然有了底气。

我走到院子里，拍了拍手：“把所有能动的都叫过来，开个短会。”

抠脚的老管事叫郝大壮，在这干了十六年，据说以前是执法堂的，因为偷喝灵酒被贬下来的。他慢悠悠站起身，上下打量我：“陈科长，你这啥意思？”

“开分工会。”

院里二十号人面面相觑，然后有三三两两地围过来，但还是有几个躺着的没动。我补了一句：“今天到场参加分工会的，每人记两个积分。积分月底兑换灵石，一个积分三块灵石。”

那几个躺着的唰地坐起来了。

效率这东西，从来不是靠觉悟提上来的，是靠价钱。

我用书页上生成的方案，对着糊墙上的门板把人员重新捋了一遍：

“王婶，你带三个人，把库房里攒了三年多的废铁废木头拖出来，能修的修，不能修的分拣卖去坊市。”

“李胖子，你继续负责灵兽饲料。但是这次不用咱们自己买粮，你去丹峰讨他们的丹方渣滓——炼丹剩下那些废料，按他们的规矩是要花灵石雇人处理的，你跟他们说免费帮他们拉走，顺便收他们一点搬运费意思意思。”

李胖子瞪大了眼睛：“丹方渣滓？那不都是毒渣子吗？”

“不全是。我查了——三品以下的丹渣里还含一成多的灵气残留，磨成粉掺在粗粮里喂灵兽，比纯粗粮营养高两成。”我说，“我已经算过成本了。”

李胖子张了张嘴，居然什么也说不出来。

后面我又排了七八个人的岗位：巡逻用具登记员、快递统收分发员、灵田灌溉系统维护员、甚至还有一个“宗务大厅窗口答疑员”——说白了就是帮那些不知道各部门在哪的内门弟子指路，顺便收点问路费。

所有人听完之后，院子里安静了五秒钟。

郝大壮挠了挠头：“科长，你这套东西……哪来的？”

“看书看的。”我推了推不存在的眼镜。

“试试。”

但我已经算到这一步了。

我把李胖子叫到一边，从怀里掏出《企业行政管理实务》，翻到夹着书签的那一页：“丹峰管事有个儿子在外门当杂役，资质不好，在执法堂做文书。你去找这个人，跟他说杂物科可以帮他儿子转到后勤考核系统，干满两年，晋升通道明确，不比在执法堂打杂差。”

“我怎么跟他说？”

“你就说，这是杂物科科长的条件。他家的孩子，可以优先转岗。”

李胖子张了张嘴，到底还是去了。

傍晚时分，他一脸古怪地回来了。后面跟着两个丹峰的杂役，用灵宠车拉着整整三大车丹方渣滓。那管事远远站着看了我一眼，没说话，转身走了。

李胖子凑过来小声道：“科长，他还真给。说以后每三天送一次，每次给咱们三十块灵石的运输费。”

我说：“记着，这三十块灵石拿出一半来给咱们的人加餐。”

从傍晚到入夜，整个杂物科的院子难得地亮起了几盏油灯。

王婶那边的手下拆了半面山似的废料，挑出几把还能用的铁锹。修厕所的几个人在门板上贴了第一张登记牌——歪歪扭扭写着“已使用”三个字，下面留了一行横线，方便人打勾。

我坐在那张连靠背都缺了一角的破椅子上，面前摊着那本书。书页上新浮现出一行字：

“部门改革第一阶段已完成。阻力值：15（丹峰管事感到不安）。信任值：55（手下对你的执行力开始认可）。警告：阻力值每上升10点，请做好应对反弹的准备。”

我端起那杯劣质灵茶。茶水还是带着那股铁锈味。

但我居然觉得没那么难喝了。

余光瞥见院门外闪过一道人影，匆匆而去，方向是丹峰的方向。

我挑了挑眉。

是去报信的吗？

还是去通风报信，说外门杂物科今天搞了一场“改革”，连那个管事的孩子都被拉进来了？

不知道。

但不管是谁，接下来有得玩了。
