# Step 2 · 从创意生长世界（内化）

> **核心原则：世界不是填表填出来的，是从创意核心要素反向推导出来的。**
>
> **力量体系是骨架，地图/势力/危机是血肉——血肉塞入骨架。**（详见 pop-decon-setting v3.3.0 的骨架/血肉两层结构设计）

## 推导链

```
创意.md核心要素 + 赛道已验证模式 + 参考书DNA
        ↓
    力量体系 ──→ 地图 ──→ 势力 ──→ 危机
```

## 力量体系设计起点判定

> **v1.1.0核心改动：world不再自己发明力量体系，消费seed已明确的力量体系。** seed v13.13.0+的故事纲领"什么样的世界"必须包含"力量体系层"，world拿到后直接展开为力量层级表，不再自己猜。

| 本项目类型 | 设计起点 | 例 |
|:--|:--|:--|
| seed已明确力量体系 | **直接消费seed的力量体系**，展开为力量层级表。不自己发明，不替换，不修改 | seed说力量体系是修仙境界→world展开炼气→筑基→金丹→元婴的层级表 |
| seed未明确力量体系（v13.13.0前） | 从赛道惯例+创意.md要素推导，标注"seed缺失，world推导" | — |

**硬约束**：world的力量层级表必须和seed故事纲领的力量体系对齐。如果seed说力量体系是修仙境界，world不能展开成"管理权限层级"——不对齐=world错误，不是seed错误。

## 2a. 三源合流设计力量体系

### 第一源：赛道参考（`写作参考/市场校准.md`，如有）
- top作品的力量体系类型（等级制/修炼境/规则系/百分比/职业树）
- 读者疲劳的模式（什么已被写烂了，必须避开）
- 正在冒头的变体（赛道演变方向）

### 第二源：参考书DNA（`涌现/文风锚定.md` + `涌现/剧情DNA-brief.md`，如有）
- 文风锚定：参考书的力量体系风格
- 剧情DNA-brief·Boss梯度：每层boss的强度分布+出场节奏
- 剧情DNA-brief·主角成长曲线：破局手段演变（前期靠什么/中期靠什么/后期靠什么）

**消费规则**：学设计思路和强度分布节奏，不抄具体等级名。

### 第三源：创意.md
- 金手指（如有）：机制和限制决定主角在体系中的**优势方式和成长节奏**（不是决定体系本身）
- 行为引擎：决定**触发升级的条件**（不是靠刷怪，靠对赌/经营/探索？）
- 前期成长路线：决定每卷的力量跨度
- 世界观规则（如有）：规则系优先于金手指

### 三源合流产出：力量层级表

展开5-7个力量层级，每个层级必须：

| 字段 | 含义 | 要求 |
|:--|:--|:--|
| 名称 | ≤6字，贴合本项目世界观 | 不套"DND/炼气筑基"模板 |
| 这个世界长什么样 | MMO式的等级区描述 | 新手区→城镇区→高级区 |
| 主角在这个层级面对什么 | 不是"解锁什么技能" | 这个层级的世界是什么样的 |
| 升级触发条件 | 从行为引擎推导 | 不是"打够怪就升级" |
| 参考源 | 标注参考了哪个源 | 赛道·X / 参考书·X / 创意·X |

**力量跨度**：第一卷从阶位1到阶位N（对齐创意.md前期成长路线的街头期上限）

## 2b. 从力量体系+行为引擎生长地图

推导逻辑（MMO式）：
```
力量层级 → 每个阶位的"区域主题" → 城市/世界地图
  阶位1："新手区" → 贫民区+赌场
  阶位2："城镇区" → 码头+商业街+公会
  阶位3："高级区" → 贵族区+神殿区+地下城
```

每个区域回答：
- 对应哪个力量阶位？
- 主角的**核心行为**是什么？（从行为引擎推导）
- 有什么**信息差**可以利用？

输出：城市结构表（区域/对应阶位/主角行为/信息差来源/势力归属/出现幕）

## 2c. 从力量体系+地图生长势力

推导逻辑：
```
力量阶位1→贫民区→谁控制这里的资源？→街头帮派
力量阶位2→码头+商业街→谁控制贸易？→地方商会+走私网
力量阶位3→贵族区→谁控制制度？→贵族家族+执法系统
力量阶位4→隐藏区域→谁在幕后？→神秘组织
```

每层势力必须回答：
1. **为什么主角必须和这个势力冲突？**（不解决它就无法进入下一阶位）
2. **主角靠什么赢？**（阶位1靠硬刚→阶位2靠信息差→阶位3靠制度漏洞）
3. **赢了得到什么？**（地盘/人脉/信息/身份——进入下一阶位的钥匙）

## 2d. 危机体系设计（四源）

| 危机来源 | 说明 | 必选 |
|:--|:--|:--|
| **世界固有危险** | 创意.md设定的世界级威胁——全书暗线 | ✓ |
| **力量体系内压** | 进入更高阶位必然面对的压力（强者注目/势力反弹/制度反噬） | ✓ |
| **金手指反噬** | 金手指限制条款触发的危机 | 如有金手指 |
| **赛道惯例压力** | 赛道读者期待的特定压力类型（如诡异流必然有"规则杀"） | 可选 |

**倒计时设计**：从危机四源推导2-3条倒计时（触发条件→到期章→后果）

## 3a. 第一卷终点

**核心问题**：第一卷结束时，主角在世界中的位置发生了什么质变？

从Step 2的势力结构和危机倒计时推导：

| 从Step 2取 | 第一卷终点的答案 |
|:--|:--|

## 3b. 幕序列（3-4幕）

从Step 2的敌人梯度+危机倒计时推导幕序列。每幕=一个"对赌难度升级"的弧段。

```markdown
### 第一幕：「幕名」
- **功能位**：建置（建立世界+第一次小对赌+埋大对赌的伏笔）
- **对赌难度**：LV1-2，街头级的公平对赌
- **弧线**：（一句话——从学会出千到赢下走私线，付出了什么代价）
- **章节预算**：ch001-006（ch001已由seed产出，实际写ch002-006）
- **幕末钩子**：第一幕结束时读者在想什么？

### 第二幕：「幕名」
- **功能位**：冲突升级（对赌难度升级——地方势力开始不跟你公平对赌了）
- **对赌难度**：LV2-4，需要筹码/信息差才能诱使对赌
- **弧线**：（一句话）
- **章节预算**：ch007-014

### 第三幕：「幕名」
- **功能位**：高潮（对赌难度最高——和制度权力+神秘力量对赌）
- **对赌难度**：LV4-6+，制度保护+不确定性
- **弧线**：（一句话）
- **章节预算**：ch015-022

### 第四幕：「幕名」
- **功能位**：余波+伏笔（揭示更大的对赌——世界危机）
- **对赌难度**：暗示LV8+
- **弧线**：（一句话）
- **章节预算**：ch023-025
```

每幕的设计原则：**对赌难度必须逐幕升级**。不是"等级提升"，是"公平对赌的难度提升"——街头级用小筹码就能诱使对赌，系统级需要制度缺陷/信息不对称，幕后级需要更大的赌注。

# 创意.md

## 一句话简介
灵根废柴陈渡被踢到濒临倒闭的杂物科当科长，他不会修仙，但会用KPI、绩效、资源整合，对腐朽的修仙宗门搞一场“凡人管理学”的改革起义。

## 创意
一个灵根废柴被迫管理一群被各峰淘汰的杂役，别人修仙靠天赋，他修仙靠KPI——用现代职场管理手段，把修仙宗门里占着茅坑不拉屎的长老们逼得主动内卷。

## 创意来源
用户选定方向：王道1-宗门公务员：修仙界底层改革（荒诞轻喜剧 + 权谋博弈）。

## 金手指
- **创意**：穿越者带现代管理思维，在修仙宗门搞“凡人管理学”。
- **机制**：人心、制度效率、资源匹配度——这些东西在修仙世界具有真实的“强化反馈”。当陈渡把杂物科改制为有战斗力的小团体，那种“组织红利”会以类似修炼突破的形态反馈给他和团队（不是修为提升，而是更直接的东西：灵石、人脉、情报、垄断资源）。
- **限制**：陈渡本人灵根为零，无法修炼，灵力感知为零。他的所有管理决策必须依赖执行者（别人替他跑腿、打架、谈判），一旦被架空或执行者背叛，他毫无修为自保。
- **合成**：穿越者用KPI和资源整合，让被宗门抛弃的废灵根爆发出不亚于内门弟子的组织效率，但本人是修为真空的凡人，所有权力根基依赖“人心”和“制度信任”——一旦团队反噬或高层直接用武力碾压，他连一张护身符都捏不碎。

## 行为引擎
**主角每章在做同一件事：用管理手段解决一个修仙世界特有的难题。**
- 具体行为：写制度 / 算账 / 拉人谈判 / 摆平内耗 / 考核下属 / 做资源置换 / 向上汇报争取权限。
- 每章对应一个“管理对象”——小到分配灵石，大到摆平丹峰的材料垄断，再大到和权谋长老做组织对赌。

## 轻量主角轮廓
- **名字**：陈渡
- **身份**：修仙宗门·杂物科科长（管仓库、清洁、杂役调度的边缘岗位），灵根为零的凡人。
- **核心动机**：杂物科再不盈利明年整科裁撤，所有废灵根杂役会被赶出宗门喂妖兽——他必须用管理手段让垃圾科室活下来，顺便往上爬。
- **记忆点**：“你们修仙，我修KPI。”——一个不戴眼镜却总爱推眼镜的凡人，手里攥着《杂物科第四季度业绩考核表》，在漫天剑光里跟人聊绩效考核。

## 番茄简介

我叫陈渡，灵根为零，被发配到宗门最没用的杂物科当科长。

第一天报到，门框上挂着一个歪了的木牌：“杂物科，正在倒闭中，有事请烧纸。”

会议室里，五個杂役，三个在睡觉，两个在打牌，桌上积的灰能种菜。

我说：“从明天起，咱们按KPI干活。扫地的，每天定量；理仓库的，每件物品录入；跑腿的，按单计件。”

一个杂役抬头看我：“科长，你又不懂修炼，能拿我们怎么样？”

我指了指门口那块牌子：“这个月科室要是再亏灵石，明年咱们一起去北山喂妖兽。你们想去，我不拦着。”

那之后，我一个人熬了三夜，算账、列需求、做表格。

第二天我走进丹峰，跟那个鼻孔朝天的掌炉长老说：“长老，你这些材料如果从杂物科走单，我保证你丹峰的丹药产出速度提三成——条件是分我科室5%的返点。”

长老眯眼：“你一介凡人，凭什么？”

我推了推不存在的眼镜：“凭我能让你丹峰弟子，不用再花两个时辰自己去领材料、自己扛回炉、自己清库存。”

他沉默了三秒，说：“签合同，我让弟子明天来对接。”

一个月后，杂物科扭亏为盈。三个月后，执法堂、器峰、药园全都来找我做资源整合。

我以为我要起飞了。

直到今天，掌门把我叫去：“陈渡，杂务堂主事病逝了，宗门上下都觉得你该上位——但二长老说，你一个凡人管杂物科就够了，再往上，就是‘以下犯上’。”

他看着我，递过一封任命书：“你自己填吧，填什么，就是什么。”

我低头一看——任命书上，是空的。

他等着我自己写一个“止步于此”的职位。

而我，慢慢拿起笔——

【情绪锚点台词】“你们修仙，我修KPI。修到你们连拒绝的勇气都没有。”

## 故事纲领（三核心+营销层·≤1000字）

### 核心一：什么样的世界（3-4句）

> **力量体系层**：修仙境界体系——炼气→筑基→金丹→元婴→化神→（每级分初、中、后、圆满四阶）。修为决定权力、地位、寿命。灵根决定能否修炼，灵根越差，上限越低。
> **差异层**：这个修仙世界有一条隐藏的“组织法则”——当团队（科室/分堂/宗门）内部制度效率高、人心凝聚、资源配置合理时，会自然产生一种“组织红利”——类似灵气富集的“管理场”。凡人是感受不到的，但修炼者能隐约感应——这个制度有“势”。
> **日常层**：底层弟子每天为灵石艰难打拼；灵根差的做杂役，灵根好的进内门。平庸者被系统压榨，天才被资源倾斜。宗门财政常年被各峰吃空，底层弟子连点灯用的灵石都要省着花。
> **主角层**：主角陈渡是凡人，灵根为零，无法感知灵力、无法修炼。但他的“组织红利”法则，能让他用管理手段为团队创造类似“修炼资源”的收益——灵石、情报、人脉、宗门话语权。代价是自己永远站在修为真空处，手无缚鸡之力，一旦被武力碾压就是死局。

### 核心二：什么样的舞台·世界危机（3-4句）

> **全书危机源头**：宗门库府空虚到了极限——再不改革，三年内会被周边三个小宗门联手吞并。掌门是知道真相的少数人之一，所以他默许了陈渡这个凡人的“改革实验”，因为他已经没有更好的棋子可以用。
> **敌人梯度**：
> - 第一层（基层阻碍）：各峰管库房的老油子弟子、垄断资源的中层执事——陈渡的“管理改革”直接动他们的灰色利益。
> - 第二层（中层反扑）：丹峰掌炉长老、器峰主事——他们是既得利益者，被陈渡的“整合方案”破了垄断，会发动宗门政治围剿。
> - 第三层（高层博弈）：二长老（保守派核心）、大长老（中立观望者）——他们不痛不痒，但一旦陈渡的“凡人改革”触及宗门权力根基，他们会以“以下犯上”灭杀。
> - 第四层（外部威胁）：周边三个虎视眈眈的小宗门——宗门衰弱，他们已经在等分尸。
> - 第五层（隐藏层）：“组织红利”法则的真正源头——这个法则不是自然法则，是远古某个大能留下的宗门管理传承，有人在暗中观测陈渡的“改革成果”，等待果实成熟后收割。
> **压力源/倒计时**：宗门库府还能撑在三年。每过一个大剧情，剩余时间缩短。陈渡必须在有限时间内，用凡人手段撬动足够多的组织红利，让宗门整体实力提升到自保线以上——同时，那个“收割者”也在逼近他改革的尽头。

### 核心三：什么样的主角（4-5句·含力量路径）

> **起点**：灵根为零的凡人，被踢到濒临倒闭的杂物科当科长。初始能力为零——不会修仙、不会打架、没有资源、没有后台。性格：表面温和，骨子里是偏执的实用主义者。
> 
> **拐点**：发现“组织红利”法则——当自己用管理手段提升科室效率时，科室的灵石收益、弟子忠诚度、宗门地位都获得了超越预期的增幅。他发现这个法则不是错觉，是真实存在的“制度修炼”。危机和机遇同时到来——改革动了太多人的蛋糕，二长老已经盯上他了。
> 
> **终点**：从一个凡人科长，成长为打破修仙阶级天花板的“组织大能”——不靠修为，靠制度之力。他最终要对抗的不是某个长老，而是这个宗门整个腐朽体制本身，以及那个隐藏在制度法则背后的古老收割者。
> 
> **力量路径（对齐世界力量体系）**：
> 第一级（炼气期层级·对应1-40章）：科室改制——把杂物科改造成有效率的小团队，发挥相当于炼气期弟子的组织战斗力。场景：摆平丹峰材料垄断后，杂物科获得第一笔返点灵石，科室弟子集体突破一个境界。
> 
> 第二级（筑基期层级·对应41-80章）：资源整合——打通丹峰、器峰、药园的资源通道，建立跨部门的“宗门效率联盟”，组织红利相当于筑基期修士的号召力。场景：执法堂主动来找陈渡做绩效考核联合方案，陈渡被掌门破格提拔为杂务堂代理主事。
> 
> 第三级（金丹期层级·对应81-120章）：制度博弈——用制度设计对抗内部保守派和外部入侵者，组织红利达到金丹期的威慑力——不是打，而是让所有人不敢对自己动手。场景：二长老联手外部宗门设局，陈渡用一份《宗门改革三年规划》在掌门前反杀了二长老的弹劾。
> 
> 第四级（元婴期层级·对应121-160章）：规则书写——以制度之力改写宗门底层规则，组织红利达到元婴期的掌控力——可以决定宗门资源的分配规则。场景：外部联军压境，陈渡用一套“战时资源调配制度”让宗门撑过三年危局，成为事实上的权力核心。
> 
> 第五级（化神期之上·对应161-200章）：改造体制——打破阶级天花板，建立“制度修仙”新体系，面对并击败那个隐藏的古老收割者。场景：最终boss不是外部宗门，而是那个创造了“组织红利”法则并准备收割成果的远古宗门管理传承者。

**力量路径对齐自检**：每级力量路径=对应修仙境界的一个层级（炼气→筑基→金丹→元婴→化神）。主角陈渡不修炼，但他的“组织红利”能达到对应境界的威慑/资源/号召力/控制力——这是对齐，不是偏移。力量路径不是在说“主角修为升级”，而是在说“主角制度之力的对抗层级和修仙境界是对应的”。
