# Step 2 · 从创意生长世界（内化）

> **核心原则：世界不是填表填出来的，是从创意核心要素反向推导出来的。**
>
> **力量体系是骨架，地图/势力/危机是血肉——血肉塞入骨架。**（详见 pop-decon-setting v3.3.0 的骨架/血肉两层结构设计）

## 推导链

```
创意.md核心要素 + 赛道已验证模式 + 参考书DNA
        ↓
    力量体系 ──→ 地图 ──→ 势力 ──→ 危机
```

## 力量体系设计起点判定

> **v1.1.0核心改动：world不再自己发明力量体系，消费seed已明确的力量体系。** seed v13.13.0+的故事纲领"什么样的世界"必须包含"力量体系层"，world拿到后直接展开为力量层级表，不再自己猜。

| 本项目类型 | 设计起点 | 例 |
|:--|:--|:--|
| seed已明确力量体系 | **直接消费seed的力量体系**，展开为力量层级表。不自己发明，不替换，不修改 | seed说力量体系是修仙境界→world展开炼气→筑基→金丹→元婴的层级表 |
| seed未明确力量体系（v13.13.0前） | 从赛道惯例+创意.md要素推导，标注"seed缺失，world推导" | — |

**硬约束**：world的力量层级表必须和seed故事纲领的力量体系对齐。如果seed说力量体系是修仙境界，world不能展开成"管理权限层级"——不对齐=world错误，不是seed错误。

## 2a. 三源合流设计力量体系

### 第一源：赛道参考（`写作参考/市场校准.md`，如有）
- top作品的力量体系类型（等级制/修炼境/规则系/百分比/职业树）
- 读者疲劳的模式（什么已被写烂了，必须避开）
- 正在冒头的变体（赛道演变方向）

### 第二源：参考书DNA（`涌现/文风锚定.md` + `涌现/剧情DNA-brief.md`，如有）
- 文风锚定：参考书的力量体系风格
- 剧情DNA-brief·Boss梯度：每层boss的强度分布+出场节奏
- 剧情DNA-brief·主角成长曲线：破局手段演变（前期靠什么/中期靠什么/后期靠什么）

**消费规则**：学设计思路和强度分布节奏，不抄具体等级名。

### 第三源：创意.md
- 金手指（如有）：机制和限制决定主角在体系中的**优势方式和成长节奏**（不是决定体系本身）
- 行为引擎：决定**触发升级的条件**（不是靠刷怪，靠对赌/经营/探索？）
- 前期成长路线：决定每卷的力量跨度
- 世界观规则（如有）：规则系优先于金手指

### 三源合流产出：力量层级表

展开5-7个力量层级，每个层级必须：

| 字段 | 含义 | 要求 |
|:--|:--|:--|
| 名称 | ≤6字，贴合本项目世界观 | 不套"DND/炼气筑基"模板 |
| 这个世界长什么样 | MMO式的等级区描述 | 新手区→城镇区→高级区 |
| 主角在这个层级面对什么 | 不是"解锁什么技能" | 这个层级的世界是什么样的 |
| 升级触发条件 | 从行为引擎推导 | 不是"打够怪就升级" |
| 参考源 | 标注参考了哪个源 | 赛道·X / 参考书·X / 创意·X |

**力量跨度**：第一卷从阶位1到阶位N（对齐创意.md前期成长路线的街头期上限）

## 2b. 从力量体系+行为引擎生长地图

推导逻辑（MMO式）：
```
力量层级 → 每个阶位的"区域主题" → 城市/世界地图
  阶位1："新手区" → 贫民区+赌场
  阶位2："城镇区" → 码头+商业街+公会
  阶位3："高级区" → 贵族区+神殿区+地下城
```

每个区域回答：
- 对应哪个力量阶位？
- 主角的**核心行为**是什么？（从行为引擎推导）
- 有什么**信息差**可以利用？

输出：城市结构表（区域/对应阶位/主角行为/信息差来源/势力归属/出现幕）

## 2c. 从力量体系+地图生长势力

推导逻辑：
```
力量阶位1→贫民区→谁控制这里的资源？→街头帮派
力量阶位2→码头+商业街→谁控制贸易？→地方商会+走私网
力量阶位3→贵族区→谁控制制度？→贵族家族+执法系统
力量阶位4→隐藏区域→谁在幕后？→神秘组织
```

每层势力必须回答（设定视角，不回答剧情怎么赢）：
1. **这个势力控制什么资源？**（资源=进入下一阶位的钥匙）
2. **这个势力有什么弱点/裂缝？**（弱点=world给plot提供的可利用设定，不是剧情怎么利用）
3. **和上层势力什么关系？**（从属/同盟/对抗——决定主角打这层时上层什么态度）

## 2d. 危机体系设计（四源）

| 危机来源 | 说明 | 必选 |
|:--|:--|:--|
| **世界固有危险** | 创意.md设定的世界级威胁——全书暗线 | ✓ |
| **力量体系内压** | 进入更高阶位必然面对的压力（强者注目/势力反弹/制度反噬） | ✓ |
| **金手指反噬** | 金手指限制条款触发的危机 | 如有金手指 |
| **赛道惯例压力** | 赛道读者期待的特定压力类型（如诡异流必然有"规则杀"） | 可选 |

**倒计时设计**：从危机四源推导2-3条倒计时（触发条件→到期章→后果）

## 3a. 第一卷设定范围

**核心问题**：第一卷在世界中的设定范围是什么？主角在哪些区域活动、遇到哪些势力、危机覆盖什么程度？

从Step 2的世界构筑推导：

| 从Step 2取 | 第一卷设定范围的答案 |
|:--|:--|
| 力量层级表 | → 第一卷覆盖到第几个力量阶位？（通常前2-3个阶位） |
| 城市结构表 | → 第一卷在哪些区域发生？（通常前2-3个区域） |
| 势力结构表 | → 第一卷碰到哪几层势力？（通常前2-3层，深层势力只暗示） |
| 危机倒计时表 | → 第一卷危机揭示到什么程度？（通常只揭示表层+暗示深层） |

产出格式：
```markdown
### 第一卷设定范围
- **力量范围**：第1阶位到第N阶位（主角从XX到XX）
- **地理范围**：区域A+区域B（主角活动的地标）
- **势力范围**：第1层势力+第2层势力（主角直接碰撞的势力），第3层势力暗示
- **危机范围**：表层危机揭示（XXX），深层危机暗示（XXX），终极危机不揭示
- **卷末设定边界**：第一卷结束时主角触及但未进入的设定（给第二卷留的舞台）
```

# 创意.md

## 一句话简介
灵根废柴的陈渡被踢到濒临倒闭的杂物科当科长，他没法修仙，但他用KPI、绩效考核和资源整合，对腐朽的宗门体制进行一场自上而下的"降维打击改革"。

## 创意
一个被宗门淘汰的废灵根，用现代企业管理学改革修仙宗门——当别人在修炼时，他在写《杂物科第四季度业绩考核表》；当别人突破金丹时，他在跟丹峰谈"百草订单"的返点。修仙靠灵根，管理靠脑子——而他脑子够用。

## 创意来源
用户选定方向：王道1——宗门公务员：修仙界底层改革。情绪基调：荒诞轻喜剧+权谋博弈。

## 金手指
- **创意**（形式）：宗门《管理条例》——陈渡发现修仙界的"制度文件"对灵根废柴也有效，只要文件格式规范、流程合规、领导签字盖章，就能产生类似法术的效果。
  - 具体表现：他写的公文贴在门上能当阵法用，签的调令能让人瞬间移动，盖了章的采购单能凭空变出灵石——前提是格式绝对规范、流程一个不落、签字盖章齐全。
- **机制**（动力）：制度即力量——在宗门体系内，文件的效力不依赖灵力，而依赖流程的完整性和权力层级的背书。陈渡不需要灵气，他只需要知道"这个文件该找谁签字、用什么格式、走什么流程"。
  - 核心逻辑：一个章=一个小法术，两个章连签=一个中阶法术，宗门大印+掌门签字=高阶禁咒级。制度越完善，文件威力越大。
- **限制**（张力）：
  1. **格式反噬**：公文格式错一个字、少一个章、走错一个流程，法术就会失控——申请宿舍调令写错名字，申请者会被传送到粪坑；采购单盖错章，采购物会变成攻击性物品攻击陈渡本人。每次用金手指都有"格式焦虑"。
  2. **职权范围限制**：陈渡签发的文件只对"杂物科编制内"或"签字盖章流程中涉及的对象"有效。他想用文件攻击宗主？不行——宗主不在他的管辖范围内，文件没效力。他必须先把对方纳入自己的"职权射程"。
  3. **制度反噬代价**：每成功签发一份"越级文件"（如绕过峰主直接跟宗主汇报的调令），陈渡的"制度反噬值"上升。反噬值到临界点，宗门天道会自动生成针对他的"制度漏洞"——比如安排一个职权完全重叠的副科长来制衡他，或突然空降一个"制度监察使"。每一份文件的背后，都在加固或松动他的权力根基。
  4. **签字背书透支**：陈渡可以替别人签字背书（帮下属请假、替外门弟子担保物资），每背一次书，他的个人"制度信誉度"就消耗一分。信誉度归零时，他签发的任何文件都会自动失效——包括他手上所有已生效的管理制度。所以背书的对象和数量必须精准控制。

## 行为引擎
主角每章在做一件具体的制度事务：写文件、走流程、签合同、做审计、搞KPI考核、协调跨部门资源、跟各峰长老谈合作条件。不是"修炼升级"，是"办成事"。
