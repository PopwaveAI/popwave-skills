## 3个梗候选

### 梗候选1：【差评返现】——专治双标

**来源**：消费者维权（写差评解决问题）+ 女拳赛道
**反差点**：给行为写"差评"→差评成真；写"好评"→反噬自己
**画面**：
> 女生在食堂骂打饭阿姨手抖："你们男人做的饭就是不行！"
> 主角拿出手机，给这条"女拳言论"写了个5星好评："说得好，格局打开！"
> 下一秒，那个女生自己手一抖，把汤全洒在自己裙子上——"好评反噬"
 
**行为引擎**：
- 每章遇到一个"双标行为"→主角判断写好评还是差评
- 写差评：对方遭遇对应恶果（必须真实因果，不能凭空造伤害）
- 写好评：反噬自己（但下次遇到同类型事件时免疫力更强）

**可扩展方向**：

## Step 2 合成金手指（选梗候选1：差评返现）

### 选梗理由：
- 梗候选1画面最快、行为引擎最清晰（写好评/差评/判断，每章一个选择）
- 差评返现本身自带梗记忆点（消费者维权梗天然有台词："好评返现？行啊，那差评也得兑现吧？"）
- 限制天生存在（好评反噬自己，差评有因果链条约束）

### 金手指合成

```
金手指 = 梗（差评返现——专治双标）
       × 机制（写出评语即生效——写差评对方遭遇对应恶果，写好评分担对应善果）
       × 限制（1.好评反噬自己——善意被践踏 2.差评必须真实因果——凭空造伤害不生效 3.每天只能写3条评价）
```

**合成金手指：**
> 主角有"差评返现"能力：用文字评价一个人或一件事，评价就会成真。
> - **差评生效**：评价的内容以因果方式发生在评价对象身上（"做事双标"→ta双标行为被公开惩罚）
> - **好评反噬**：评价的善意以相同形式发生在自己身上（"她是个好人"→自己接下ta的霉运）
> - **每天3条，不能撤回**

| ✅ **限制天然存在** | ✅ | 好评反噬自己 + 差评需真实因果 + 每天3条——天然张力 |
| ✅ **能想出10+场景** | ✅ | 见梗候选1的可扩展方向表，已列表10个 |
| ✅ **行为引擎** | ✅ | 主角每章做：观察→判断→写评价→承受后果，不是抽卡/升级 |

## 金手指

- **梗**：差评返现——专治双标
- **梗来源**：用户方向（女拳问题）+ 消费者维权行为框架碰撞（写差评→问题解决，写好评分担）
- **行为引擎**：每章主角遇到一个"双标行为"→判断写好评还是差评→承受后果/影响因果
- **机制**：写评语即生效——差评内容以真实因果落在对象身上，好评对应反噬自己
- **限制**：1. 好评反噬自己（善意被利用） 2. 差评必须有真实因果依据（不能凭空造雷） 3. 每天3条，不可撤回
- **合成金手指**：主角每天能动用3次"差评返现"——对双标行为写评价就成真；差评能揭穿对方，但好评会反噬自己；越公正越安全，越情绪化越危险
