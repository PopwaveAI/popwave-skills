# 3个创意方向

## 创意1：《我的AI助手，专治各种不服》
**一句话描述**：男主获得一个能看穿所有双标话术的AI助手，但他从不直接怼人，而是默默记录、拆解，最后用对方自己的逻辑反杀。
**画面**：出租屋里，男主对着手机屏幕笑，屏幕上是AI自动生成的“双标话术图谱”，他在相亲群里发了一句：“我同意你刚才的观点，但根据你的逻辑，接下来你应该这么反驳才对。”然后群里炸了。
**为什么有意思**：不靠“怼”靠“拆”，更具智斗感和爽感。主角的武器是逻辑而非情绪，更能吸引理性读者。AI助手可以作为系统替代，不踩“系统逆袭”的红线。

---

## 创意2：《我在民政局，专门给“拳师”办理离婚》
**一句话描述**：男主穿越到平行世界，成了一名民政局婚姻登记员，每天面对各种极端男女对立案例，但他不站队，只负责“执行规则”，结果成了全网最火的“人间清醒”。
**画面**：办公桌前，一个女拳师咆哮：“男人没一个好东西！”男主头也不抬地递出表格：“根据《婚姻法》第XX条，你丈夫的出轨证据已认定，请签字。”弹幕：“这操作，我直呼专业。”
**为什么有意思**：以“职场公务员”视角切入，既有现实感又有制度幽默。通过办理一个又一个充满争议的婚姻案件，展现社会观念碰撞，安全系数高。每案一个反转，适合做单元剧。

## 创意3：《全城男女互换，女拳崩溃了》
**一句话描述**：一场神秘事件让城市里男女的身体、社会角色互换，原本在网络上叫嚣的女性突然发现自己成了“弱势方”，而原本沉默的男性开始体验女性的日常困境。
**画面**：女拳群主在群里崩溃：“凭什么我现在要穿着高跟鞋挤地铁？我昨天骂男人就该这样？”底下一堆人沉默。另一边，一个普通男人第一次体验到了被骚扰的感觉，他沉默了。
**为什么有意思**：用“交换”这个设定，批判双标，但效果不是“打脸”，而是“共情”。让读者通过体验差异，自己得出结论，价值观更正向，不易引战。而且交换后的冲突天然带有喜剧和戏剧张力。

# 最终种子

## 核心创意
- **创意名**：非接触式拆解：我靠AI话术图谱，专治双标
- **灵感来源**：市场现有“怒怼流”同质化严重，且直接对抗易踩红线。从“逻辑博弈/认知差”角度切入更安全、更有深度。
- **机制**：主角获得一个AI助手，能将任何“双标话术”自动拆解为“标准句式+破绽坐标”，主角只需在关键节点说一句“引导性反问”，对方就会自己掉进逻辑陷阱，暴露双标本质。
- **限制**：AI不能直接给出怼人话术，只能提供“逻辑拆解图”和“对方话术根源”，主角必须自己借助现实场景“设局”才能生效。且主角每用一次，现实中就会消耗一种“社会隐形资源”（如信用值、好感度），必须巧妙平衡。
- **一句话简介**：嘴炮时代，我靠AI逻辑拆解+现实布局，让所有极端言论在公共场合自曝其短，成为全网第一“人间清醒”。

## 番茄简介
【话术剖开者】+【双标终结者】+【逻辑控场王】+【现实辩手】

我叫陈默，一个普通人，直到脑子里响起一个声音：
“检测到极端话术，目标ID：娇娇女王。话术类型：权利扩大化+受害者叙事。破绽坐标：第3、7、12句。建议操作：在对方使用‘你根本不理解女性困境’时，反问‘那如果我是一个结扎的男性，你还会说这句话吗？’”
我没有直接怼，而是打开了一个相亲群，把这份话术图谱匿名发给了两个“对线”的人。
第二天，群里炸了：“她说我直男癌，但我用她自己的逻辑反驳回去了！她怎么破防了？！”
AI又响起：
“恭喜宿主，首次触发电竞式反拆。当前社会信用值：70%。下次操作将消耗‘陌生好感度’，请谨慎布局。”
我笑了。比愤怒更可怕的，是让你亲口毁掉自己的“人设”。而我，只用递上一把逻辑的手术刀。
