## 定位

pop-research 是管线的**独立调研 skill**，三种主流使用场景：

| 调用方 | 触发时机 | 档位 | 产出 |
|:--|:--|:--|:--|
| **pipeline Phase 0** | 项目初始化，自动触发 | **赛道定位调研** | `素材/赛道调研.md` |
| **pipeline Phase 0** | 用户给了力量体系参考书 | **decon-lite** | `素材/decon-lite-{书名}.md` |
| write | 写作质感缺口 | **采风** | `素材/知识沉淀/{主题}-采风.md` |
| seed/plot（深层需求） | 需要世界观/压力矩阵/行业深度 | **三界框架**（可选） | `素材/知识沉淀/{主题}.md` |

**为什么重构**：旧版 v1.4.0 310 行中 309 行是三界框架，但它被调用的频次远低于赛道定位调研和 decon-lite。新版把高频场景提到前面，三界框架降为"有需要才用"的深层选项。

# 主流程 · 赛道定位调研

> **触发**：pipeline Phase 0 Stage 2 自动派发子agent。所有项目必有。
> **产出**：`素材/赛道调研.md`。消费方：seed 1d 双轨发散（王道赛道借鉴+避雷）+ plot 三源合流（赛道参考第一源）。

## S1 · 接收需求

输入：
- 赛道方向（来自 `素材/用户意图.md`）
- 用户已给的参考书信息（如有，需要在调研中标注避免撞车）
- 用户指定对标书（如有）

## S2 · 四轮搜索

**每轮不是"搜一下"，是搜到底（至少看前 5 条结果的摘要+内容）：**

### 第1轮：赛道格局

```
目标：摸清这个赛道当前什么在火、什么在衰落

搜索词：
1. "{赛道} 番茄小说 排行榜 热销 2026" — 找 top 5-8 + 核心卖点
2. "{赛道} 起点 精品 万订 2026" — 补起点 top 3（如果番茄不够）

对每本 top 书提取可迁移核心卖点（1句话，不抄设定）：
| 排名 | 书名 | 核心卖点（为什么读者买单） | 可迁移方向 |
```

### 第2轮：读者偏好

```
目标：这个赛道的读者在找什么、骂什么

搜索词：
1. "{赛道} 小说 读者 推荐 出圈 2026" — B站/抖音/知乎上的推荐+讨论
2. "{赛道} 小说 不爽 雷点 审美疲劳 2026" — 读者喷什么、避雷什么
3. "{赛道} 爽点 名场面 好评 2026" — 名场面 = 核心爽点的具体化

对每轮至少看前 5 条的内容摘要，产出借鉴点和避雷点：
- 借鉴：什么爽点/机制/节奏在读者中形成了共识好评
- 避雷：什么设定/金手指/开局模式已经被喷烂了
```

### 第3轮：赛道演变

```
目标：这个赛道正在往哪个方向变

搜索词：
1. "{赛道} 新题材 新趋势 2026" — 冒头的新方向
2. "{赛道} 传统题材 衰落 无人问津 2026" — 已经在走下坡的老模式
3. "{赛道} 细分 垂直 蓝海 2026" — 大品类下的细分赛道

产出：细分赛道格局
| 细分方向 | 代表作品 | 热度 | 饱和度 | 机会判断 |
```

### 第4轮：对标分析（如有用户指定对标书）

```
目标：用户指定的对标书在赛道中的位置

对每本用户指定的对标书：
1. 搜 "{书名} 作者 风格 读者评价"
2. 判断这本书在赛道的细分位置（是哪个方向的标杆？读者的核心认可点？）
3. 确认本项目不会和它撞车（设定/金手指/开局模式）
```

## S3 · 产出落盘

落盘到 `素材/赛道调研.md`：

```markdown

## 三、细分赛道格局
| 细分方向 | 代表作品 | 热度 | 饱和度 | 机会判断 |
```

## S4 · 质量门禁

| 检查项 | 合格标准 |
|:--|:--|
| top作品≥5本 | 有书名列卖点 |
| 借鉴点≥3条 | 每条有来源 |
| 避雷点≥3条 | 每条有来源 |
| 细分格局≥3个方向 | 含热度+饱和度判断 |
| 落地建议有具体内容 | 不是"注意差异化"这种废话 |

# 流程 · decon-lite 增强版

> **触发**：pipeline Phase 0 Stage 2。用户给了力量体系参考书。
> **产出**：`素材/decon-lite-{书名}.md`。消费方：plot 三源合流（第二源）+ seed 1d-0改编策略层 + seed 故事纲领（节奏参考）。

**v2.1.0升级**：旧版5表拆解深度不足——表1只到"描述级"（这本书有什么），没到"规则级"（规则怎么运转），下游world消费时只有骨架没有规则。新增表6-8补势力/金手指/危机三个维度，服务seed 1d-0改编策略层。所有表字段用**抽象结构**（"这本书有几层X"），不用具体内容模板（"4层势力"）。

## DL1 · 接收需求

输入：书名 + 本地txt路径 + 采样范围（默认前 300 章）

## DL2 · 八表拆解

### 表1：力量体系结构（规则级深度）

> **v2.1.0核心升级**：从描述级升级到规则级。引入"主养成线+子养成线"抽象——主养成线=位阶（区分境界的核心刻度），子养成线=位阶内的平行成长维度（每本书的子养成线数量和内容不同，由书决定，不固定字段）。

| 字段 | 说明 | 必须给出具体内容 | 抽象要求 |
|:--|:--|:--|:--|
| 力量体系类型 | 等级制/修炼境/规则系/百分比/职业树/... | 有明确判断理由 | 通用字段 |
| **主养成线（位阶）** | 这本书的位阶名称/数量/每位阶的章节范围/每位阶的质变标志 | 至少覆盖前4阶，每阶标注质变标志 | 抽象结构：列位阶名+章范围+质变标志 |
| **子养成线清单** | 这本书有几条平行的子养成线？每条叫什么？（数量由书决定） | 列出所有子养成线名称 | 抽象结构：先识别数量再列名称 |
| **子养成线1规则** | 第1条子养成线的获取规则+消耗规则+上限+与主养成线的关系 | 有具体数值/条件+原文chXX出处 | 逐条拆，不固定字段名 |
| **子养成线2规则** | 第2条... | 同上 | 同上 |
| **子养成线N规则** | 第N条...（N由这本书决定） | 同上 | 同上 |
| **子养成线交叉规则** | 子养成线之间怎么互相影响？有没有兼职惩罚/互相制约/互相增益？ | 有具体规则+原文出处 | 抽象结构：列交叉关系 |
| 升级触发 | 主角靠什么行为升级？和子养成线什么关系？ | 写具体行为，不写"打怪升级" | 通用字段 |
| 强度膨胀 | 位阶间的实力差距多大？线性/跳跃/指数？ | 有具体对比（如LV1可以1v1街头混混，LV5可以1v15） | 通用字段 |
| 破局手段演变 | 不同位阶主角各靠什么赢？ | 至少覆盖3个位阶 | 通用字段 |
| 设计亮点 | 最值得本项目借鉴的结构性设计是什么？ | 1句话讲清楚 | 通用字段 |

**拆解要点**：
- 子养成线数量不固定——深渊主宰可能有5条（职业等级/属性点/专长/技能点/兼职），修仙书可能有4条（法力/神识/肉身/法宝），系统流可能有3条（技能树/属性点/装备）。先识别这本书有几条，再逐条拆
- 每条子养成线的规则字段不固定——职业等级拆"等级上限/进阶前置/兼职方向"，法力拆"增长方式/容量上限/恢复机制"。由这条子养成线的性质决定拆什么
- **数值锚点**：每条规则标注原文chXX出处，便于world消费时查证

| 钩子分布 | 章末钩子以什么类型为主？分前/中/后期变化 | 分类型占比 |

### 表4：人称与视角策略（新增）

| 字段 | 说明 |
|:--|:--|

### 表5：开篇解剖（新增）

| 字段 | 说明 |
|:--|:--|
| 第1章第1句 | 原句摘录→分析：这一句建立了什么期待？ |
| 前3章埋了什么 | 悬念/钩子/核心矛盾各是什么？哪个最早被读者接收到？ |

### 表6：势力格局（v2.1.0新增）

> 服务seed 1d-0改编策略层的"势力替换"维度。字段用抽象结构——拆"这本书的势力分层"，不写死"4层"。

| 字段 | 说明 | 抽象要求 |
|:--|:--|:--|
| 势力分层结构 | 这本书的势力分几层？每层叫什么？（层数由书决定） | 先识别层数再列名称 |
| 每层势力具名 | 每层有哪些具名势力/组织？主角和每层什么关系？ | 列具名势力+主角关系 |
| 层间矛盾 | 层与层之间的核心矛盾是什么？同层内的矛盾是什么？ | 列矛盾关系 |
| 势力博弈关系 | 势力之间怎么博弈？同盟/对抗/利用/制衡？ | 列博弈类型 |
| 主角在势力格局中的位置 | 主角起始在哪层？终点到哪层？靠什么往上爬？ | 标注起点+终点+路径 |
| 设计亮点 | 最值得借鉴的势力结构设计是什么？ | 1句话 |

**拆解要点**：
- 势力层数不固定——深渊主宰可能是帮派/工会/教会/神殿/诸神5层，修仙可能是家族/宗门/王朝/天道4层，都市可能是街头/帮派/商会/政界4层。先识别这本书有几层，再逐层拆
- 服务1d-0策略层：用户选"势力替换"策略时，这张表提供"借什么势力结构、换什么势力"的素材

### 表7：金手指深度（v2.1.0新增）

> 服务seed 1d-0改编策略层的"金手指形式"维度+world力量体系设计的金手指耦合。

| 字段 | 说明 | 抽象要求 |
|:--|:--|:--|
| 金手指类型 | 这本书的金手指属于什么类型？（重生记忆/系统面板/血脉觉醒/神器绑定/...） | 有明确判断 |
| 机制 | 金手指具体能做什么？核心功能是什么？ | 列核心功能 |
| 限制 | 金手指有什么使用限制？（冷却/消耗/条件/副作用） | 列所有限制条件+原文出处 |
| 代价 | 使用金手指要付出什么代价？（身体/精神/时间/因果） | 列代价+原文出处 |
| 成长路径 | 金手指本身会成长吗？成长条件是什么？ | 列成长阶段+条件 |
| 与主养成线耦合 | 金手指加速哪条主养成线/子养成线？还是开了一条新养成线？ | 标注耦合关系 |
| 与世界观耦合 | 金手指和世界的力量体系/规则有什么关系？是世界内的还是外挂？ | 标注耦合程度 |
| 设计亮点 | 最值得借鉴的金手指设计是什么？ | 1句话 |

**拆解要点**：
- "与主养成线耦合"是关键——深渊主宰的金手指（重生记忆）不直接给属性，而是"知道怎么练"，加速的是职业等级这条子养成线。修仙书的金手指可能直接给法力，加速主养成线。耦合方式不同，设计思路不同
- 服务1d-0策略层：用户选"金手指形式"策略时，这张表提供"借什么金手指机制、改什么限制/代价"的素材

### 表8：世界危机引擎（v2.1.0新增）

> 服务seed 1d-0改编策略层的"世界危机"维度+world危机设计的结构参考。

| 字段 | 说明 | 抽象要求 |
|:--|:--|:--|
| 危机分层结构 | 这本书的危机分几层？每层叫什么？（层数由书决定） | 先识别层数再列名称 |
| 终极危机 | 全书最终的大危机是什么？驱动全书的核心矛盾？ | 1-2句话概括 |
| 阶段性危机 | 每卷/每阶段的危机是什么？和终极危机什么关系？ | 列阶段危机+与终极危机关系 |
| 危机驱动机制 | 危机怎么驱动剧情持续运转？倒计时/衰减/惩罚/递进？ | 列驱动机制类型 |
| 危机与主角关系 | 主角和每层危机什么关系？是被动卷入还是主动对抗？ | 标注关系类型 |
| 危机升级路径 | 危机怎么升级？从小危机到终极危机的 escalation 路径？ | 列升级路径 |
| 设计亮点 | 最值得借鉴的危机引擎设计是什么？ | 1句话 |

**拆解要点**：
- 危机层数不固定——深渊主宰可能是剥皮案(卷1)/上古红龙(卷2)/诸神黄昏(全书)3层，修仙可能是宗门大比(卷1)/域外天魔(卷2)/天道崩塌(全书)3层，末日流可能是丧尸潮(卷1)/变异体(卷2)/文明覆灭(全书)3层。先识别这本书有几层危机，再逐层拆
- 服务1d-0策略层：用户选"世界危机"策略时，这张表提供"借什么危机引擎、换什么危机"的素材

## DL3 · 质量门禁

| 检查项 | 合格标准 |
|:--|:--|
| 8张表全部完整 | 每张表字段齐全，不缺项 |
| 表1力量体系规则级深度 | 主养成线≥4阶 + 子养成线≥2条（每条有规则+chXX出处）+ 交叉规则有 |
| 表6势力格局分层 | ≥2层具名势力 + 层间矛盾有 |
| 表7金手指深度 | 机制/限制/代价/成长路径四项齐全 + 与主养成线耦合关系明确 |
| 表8世界危机引擎 | ≥2层危机 + 终极危机明确 + 驱动机制有 |
| 爽点间隔有章数 | 不是"快/慢" |

    “手术费的事情不用担心，妈找了个新兼职，一定会帮你凑齐的，你踏踏实实参加面试。”

    看着屏幕上母亲发来的消息，张羽默默收起手机，深吸一口气，闭上了眼睛，静静地等待着。

    良久之后，前方传来一阵叫号声。

    “989号考生，张羽。”

    张羽站了起来，走进了面试的教室内。

    他看向三位面试官，露出了练习许久的礼貌笑容：“三位好，我是东阳初级中学的张羽。”

    中间的面试官望着他，淡淡道：“你为什么要报考我们的学校？”

    张羽：“贵校悠久的历史，深厚的底蕴，丰富的教学资源，一直以来为社会培养出了许多杰出人才……”

    面试官皱了皱眉，打断道：“别说这些场面话。”

    张羽老实道：“我想考上名牌大学，嵩阳高中是我能报考的学校中，大学录取率最高的一个。”

    面试官微微一笑，看着手上的资料说道：“嗯，全科满分，全校第一？怪不得被推荐过来。”

    “你成绩没有问题，但想要进嵩阳高中，光靠校内考试成绩是远远不够的。”

    他想了想，随意问道：“你现在每天睡多久？”

    张羽：“五个小时。”

    面试官意外道：“五小时？”

    “我们学校的学生从小学开始，每天平均睡眠时长都不超过两个小时。而像历届的优秀毕业生基本上都是不睡觉的。”

    “你每天竟然要睡五个小时，那就是每天要比别人少学三个小时，九年就差了近一万个小时……”

    张羽微微一愣，没想到本以为已经非常努力的自己，在努力上竟然还和他人有这么大的差距。

    张羽连忙说道：“我会努力追赶上他们的。”

    左边的面试官问道：“高中课程你学了多少？”

    张羽稍稍镇定下来，自信道：“我已经自学完了高一课程。”

    就在他不知所措时，中间的面试官又问出了下一个问题。

    “本校为了提升学习效率，防止早恋，要求所有学生在入学前要完成把相关器官摘除的绝育手术，从此专心修行。”

    面试官不置可否地点了点头：“行了，今天的面试结束了，你先出去吧。”

    一旁的女面试官笑了笑：“我看他什么都不知道就来了，该有的测试报告和课外考试成绩一个也没有，只能说下面这些普通初中的推荐生质量是一年不如一年，要不是有扶持政策，哪有资格见我们。”

    中间的面试官点了点头：“唉，我本以为穷人会更努力一点，也许是我对他们期望过高了。”

    “这个就暂时备选吧。”

    说罢，他便将张羽的简历扔进了一旁的纸篓里，和另外几百份备选简历挤在了一起。

    虽然张羽感觉自己的这趟面试似乎不太成功，但没有多余的时间给他思考这些，他已经开始准备起了一趟又一趟的面试，来到了一个个不同的高中。

    “张羽同学，我们了解到你的家庭情况恐怕不足以负担这边的学费。不过我们为贫困生提供了优惠的贷款服务，只要你愿意抵押一些不重要的器官就行……”

    “放心，你没来错地方，我们知道你是男的，而我校虽是女校，但从来不歧视男性，只要你完成了性别转换手术，不但能够入学，更是会被认为道心坚毅的种子学生，有机会进入重点班学习元阴炼气之术……”

    “可惜了，你距离我们的入学标准还是差上了一点点。不过今年为了照顾贫困学生，我们推出了一个特长生政策。你如果愿意放弃肉身的话，可以凭借魂修特长生的身份，在校长的万魂幡里学习……”

    要么是高不可攀，让他完全看不到希望的门槛。

    要么就是陷阱重重，让他感觉自己会被一口吞下的合同。

    此刻的他骤然发现，在郊区普通初中有着全校第一成绩的他，和市中心的那些学生们比起来便早已经有了天壤之别。

    过去这些年的努力学习似乎都成了笑话。

    到头来他和那些不学无术的同学们并没有差别，同样也上不了高中。

    回到家中，张羽如一尊石像般默默坐着。

    桌上的手机连连震动了起来。

    母亲：这次面试怎么样？

    母亲：今天要加班，冰箱里的饭菜你记得自己热了吃。

    母亲：儿子，你不要担心，不论你选绝育还是变性，手术的钱妈妈一定会帮你凑齐的。

第3章 体育

    仙道修行自从经过十大宗门的归纳整合之后，如今早已经拥有了系统性的境界和实力划分，其中包罗万象，覆盖了仙道技术的方方面面，可以说是浩如烟海。

    而决定一个人能登上哪个仙道境界的关键指标便是道心等级和法力水平。

    像是对高中生来说，唯有道心跨越10级门槛，法力超过60点，才能从炼气境界突破到筑基境界。

    如果说道心、法力是仙道的基础，那么体育课上锻炼的肉体强度，便是基础的基础。

    与此同时，一阵阵光波扫过了在场众多学生们的身体，将他们的肉体强度数据都显示在了不远处的大屏幕上从高到低排列。

    第1名钱深肉体强度1.15级

    第2名白真真肉体强度1.13级

    第3名何大有肉体强度1.12级

    第19名张羽肉体强度0.82级

    像是肉体强度不但被量化，更是每节体育课都会被重新检测出来。

    而这个指标和道心一样，在炼气境界最多只能达到10级。

    虽然肉体强度不是突破仙道境界的关键指标，却关系到修炼者的效率和战力，是仙道的重要基础，在仙道科目的650分中占了150分。

    就在两人窃窃私语的时候，一道冷冽的目光突然扫了过来，只听王海喝道：“上体育课别在下面说小话。”

    “炼气阶段的炼体，主要就是锤炼筋肉、骨骼，通过一次次的练习来打破上限，最终提升肉体强度等级。”

第13章 法赛

    这天的法力课上，示范班的学生正盘坐在静室之中运转着吐纳法。

    前方的摄像头则监控着学生们的一举一动，将学生们有没有睁眼，有没有挠痒，有没有动手动脚，有没有屁股扭来扭去……任何轻微的异状都会被一一记录下来。

    而所有这些数据会被统一记录为走神率。

    每周走神率最高的班级，会被全校通报批评。

    示范班的学生对此不以为意，反正不论怎么考核，他们班也必然是第一。

    而每周走神率最低的学生，不但会被通报表扬，更是会被奖励500块钱，以及一小时天灵根的免费租用权。

    这就让示范班的学生们一个个提起了神来。

    而张羽第一次在脑海里知道走神率这玩意的存在，便感叹如果前世有这个东西，学生们必然会骂学校大傻逼。

    但这个世界却不一样，示范班的学生们个个像是发情的公牛一样盯着每天的走神率，整个班级里都充满了一股激烈的卷气。

    此时此刻，伴随着一阵铃响，静室内默默运功的众人都停了下来，将目光看向了前方的大屏幕，期待着这节课的走神率数据。

    第1名张羽走神率0%法力8.3

    第2名白真真走神率0.03%法力11.2

    第3名钱深走神率0.12%法力11.1

    ……

    看着这走神率的排名，教室内的学生们虽然没有出声，但从他们眼神和表情的细微变化中，能感觉到一丝丝的惊讶在蔓延。

    台上法力课的指导老师姓严，是一名盘着头带着黑框眼镜的女老师。

    此刻便听这位严老师看着大屏幕上的成绩说道：“走神率超过0.3%的几位同学，我今天也不点名了。”

    “已经开学三个月了，我说了多少次冥心于一，合气于淡？”

    “走神率这么高，你们的吐纳效率怎么可能比得上别人？”

    “大家都是一天修炼24个小时，你们不会比别人多出时间来，吐纳效率低了以后，一天落后个一点点，高中三年后你们在法力上的差距可能就是10点、20点甚至30点。”

    “还有我说了多少次了，走神率降不下来的同学，可以来我办公室买静心剂，昨天我下午在办公室坐了三个小时，没一个人来问我的……是都觉得自己一点问题没有了？！”

    一番批评后，严老师又微微一笑，看着屏幕上的前几名说道：“当然，也有表现很好的同学。”

    “这里点名表扬一下走神率最低的三名同学，特别是张羽，已经连续4天走神率0%了，对比以前，可谓是质的飞跃。”

    “还有法力这三天也提升了足足0.5点，说明他在课外时间也一定下足了苦功，不是租了灵根，就是租了高灵机的修炼室，是吧张羽同学？”

    所有学生们目光立刻全都看了过去。

    却见张羽仍旧双目紧闭，看上去似睡非睡，还在继续默默运转周天采气法。

    严老师赞赏道：“你们看，这就是真正学霸的态度，下课了都不停，抓紧每一分每一秒的时间去修炼，去变强。”

    赵天行看着还在认真吐纳的张羽，发现自己越来越看不懂这位同学了：“明明是这么用功的人，为什么体育课上反而要偷懒不扎药呢？”

    白真真看着仍旧紧闭双目的张羽，心道：“羽子这几天除了吃饭拉屎，就差走路的时候没吐纳了，用功到了这种地步……难道是被70万的催债电话逼出来的？。”

    台上，严老师继续说道：“当然了，所谓一分钱一分收获，仙道之路上没有免费的东西，不砸钱是走不长远的，最近学校里的天灵根租借有了空位，感兴趣的同学可以来找我。”

    而在严老师离开之前，她说的一件事情却是引起了张羽的注意。

    “对了，市里的法赛下个月要开始了，高一组基本上全市所有高中都会参加，第一名的奖金有10万……”

    “因为比赛下周五报名，所以高一年级里，学校打算在下周五挑选法力水平前十的学生参加法赛……”

    ……

    当天晚上，食堂。

    周天翊今天要提前赶去补习班，餐桌上此刻只留下了张羽和白真真。

    被白真真左看右看，看得有些受不了了，张羽开口说道：“阿真，我知道自己的颜值很高，在这只知道学习的嵩阳高中里犹如是鹤立鸡群，但你也不用这么死盯着看吧。”

    白真真问道：“你说你也想参加法赛？”

    “可惜了，阿真。”张羽摇摇头道：“你这位年级第一的绝世天才如果不是我的朋友，接下来就该向我挑衅一番，接着在赛场上被我的惊人表现连番打脸，最后瞳孔地震几十级了。”

    白真真摇了摇头：“羽子，你想参加的可是全市法力竞赛啊。”

    “那是整个嵩阳市所有高中，派出自己最优秀，法力最雄厚，家里最有钱的学生来参赛，汇聚了无数的死有钱人。”

    白真真一脸感慨地说道：“像你这样的普通人去参加会是怎么样的结果？”

    她的双眼似乎已经看穿了未来的绝景，一脸悲哀道：“被无数强者反复侮辱，成为有钱人们的垫脚石，嵩阳高中的小丑，学校风评降到尘埃！”

    “太惨啦，我都已经看到你被其他高中的妖孽狠狠踩在脚底，道心破碎，宛如死狗的模样了。”

    张羽翻了个白眼：“你别太夸张了，法力赛又不是擂台赛，输了也不丢人。”

    “而且第一名可是有十万块钱的。”

    “哪怕只是前十也有一万块钱啊。”

    白真真看着张羽一脸贪财的模样，心想羽子果然是被催收公司逼得不行，已经魔怔到妄图参加竞赛去了。

    张羽确实是经济困难，虽然前几天才在仙韵公司赚了一万三，但转眼间就被他拿了一万去还贷了，又被他花了一千五补上房租，后来也就剩下两千多块钱。

    “想参赛，你先想想怎么把法力水平提升到年级前十吧，你现在排名才16吧？想要冲到前十还得借多少钱？”

    张羽也在心里快速算了算。

    他如今法力8.3，排名年级第16。

    而现在年级第十名的学生，法力是9.5。

    那么想要超越对方的话，张羽就必须在下周五之前，将法力提升到9.5的水平……

    “不对，要考虑到所有人都会进步的情况。”

    “那么保险一点的话，最好能在下周五之前把法力提升到10.0的水平。”

    “距离下周五还有八天，八天时间内，将法力从8.3提升到10.0，足足提升1.7吗？”

    “每天要提升0.2以上……可能吗？”

    绝对可以！轻易可以！

    在其他老师和同学的眼中，张羽这几天的迅猛提升一定是砸下重金，租了优质的灵根和场地才获得的。

    但张羽知道，这一切都源自于他的惊人潜力和极度的自律。

    4级的周天采气法就可以让他在短短3天内提升法力0.5之多，他相信接下来随着周天采气法继续升级，他的法力提升也一定会越来越迅猛。

    “毕竟炼气阶段，技能的最高等级也就是10级。”

    “一门4级的吐纳法，对绝大多数高一新生来说，都是不可能在第一个学期内达到的。”

    “而我接下来继续修行，短时间内恐怕就能将周天采气法一路提升到8级、9级甚至是10级，那就是许多人在高中三年后也未必能达到的水准了。”

    想到这里，张羽正振作精神的时候，突然发现白真真一筷子朝着他碗里伸来。

    “好胆！”张羽一声呵斥，抬起筷子便挡了上去。

    接着铛铛铛的撞击声中，他的筷子已经被白真真轻松搅下。

    感受着白真真动作中的熟悉感，张羽惊讶道：“高中基础剑法？你已经学会了？还化用进了筷子里？”

    迅速从张羽碗中夹了几块肉，白真真轻笑道：“这很难吗羽子？不是有手就行？”

    张羽心道：“干，又被她装到了。”

    白真真混着夹来的肉又迅速扒了几口饭，意外道：“想不到5元套餐的排骨也挺好吃的，下次我也买一份。”

    拍了拍微微有些鼓涨的肚子，白真真打了个饱嗝说道：“唉，再吃就影响补课了。”

    说着，她将餐盘推向了张羽：“给你加餐了，羽子。”

    看着白真真留下的整整齐齐半份没动过的‘剩饭剩菜’，张羽明白这是对方故意留给他的。

    似乎是因为了解到张羽的经济困难，这段时间的白真真便经常在晚饭的时候投喂他。

    张羽一把将餐盘拿了过来，一边吃一边说道：“阿真，你不用考虑我的什么自尊心、面子之类的东西，以我俩的关系，直接给我打钱都没问题。”

    白真真翻了个白眼：“我要去补课了，你慢慢吃吧。”

    ……

    食堂外。

    一道高大的身影正站在不远的楼道中，双眼死死盯着食堂大门的方向。

    片刻后，当看到张羽从食堂大门里走出来的时候，这道高大身影精神一震，立刻便跟了上去。

    跟着张羽走出学校大门，当看到对方进了街对面的文具店时，赵天行目光一动，心中想到：“难道这里就是张羽进货的地方？”

第15章 中心大厦

    听到金丹真人这四个字，车上的赵天行便是微微一惊，接着又不信道：“金丹真人的女儿怎么会来嵩阳？”

    张羽也在一旁微微点头，他知道炼气之上是筑基，筑基之上才是金丹。

    想要成就金丹，必然是重点高中毕业，顺利考入十大名校，还进入大宗门一路发展，也不知道要爬上昆墟的第几层了。

    这种人的女儿，一出生起码也是昆墟二层往上，怎么会来到一层的嵩阳市？

    司机闻言微微一笑，却像是起了卖弄的心思，缓缓说道：“岂止是李雪莲女士要来我嵩阳市？”

    “据说其父星火真人就是从我们嵩阳市出来的，现在是要退休了，这才打算回到嵩阳市颐养天年……”

    张羽听着却觉得怪怪的，金丹大能回新手村养老？

    司机这边则是接着说道：“这次画展可不止是李女士的画，据说还有一副星火真人亲绘的天人演武图，其中据说隐藏了一套绝世武学。”

    “来这次画展的人里，除了想要结交李女士的之外，便还有许多都想要参悟一番这演武图，想要从中悟出那一套绝世武学。”

    ……

    嵩阳中心大厦。

    第999层。

    张羽正站在落地窗前，有些震撼地看着脚下的滚滚云海，这还是他第一次站在如此高空去俯瞰整座城市。

    整了整身上新换上的保安服，张羽心中有些感叹。

    自从进入中心大厦以来，张羽便感觉自己像是来到了另一个梦幻世界。

    如果说他住的郊区环境还有点类似前世的话，那么嵩阳高中的基建环境就已经隐隐在各方面都超越了上一世的世界，而眼前的中心大厦……

    不论那999层的恐怖高度，还是大厦内充满了法力流转的种种仙道技术，都称得上是梦幻般的场景。

    回想自己在道术加持下，几秒钟内就被阵法牵引至999层，却又毫发无伤的过程，张羽心中便忍不住再生感慨。

    “同样都是嵩阳市，凡人住在郊区，修仙者汇聚于市中心，而这座中心大厦则是真正有钱人的地盘了。”

    “三个地方，虽然都在一座城市之中，却犹如三个不同的分层世界……”

    就在这时，张羽身后响起了赵天行的声音：“张羽，队长让我们过来巡逻的，得走了吧？”

    张羽转过头去，看向身后同样穿着保安服的赵天行。

    只不过和张羽穿了保安服后的坦然不同，赵天行却是十分局促，畏畏缩缩……一副屎拉在裤子里的衰样。

    他总觉得穿上了这套保安服后像个小丑，周围的人看向他的目光中都像是带着三分鄙夷、三分蔑视、三分轻贱。

    但其实是赵天行想多了，穿上这套保安服后，他就再也不是凡人，就像是成为了大厦里的背景板，在大多部分人眼中的存在感也就略高于绿化。

    张羽一拍他的肩膀，说道：“挺胸抬头，咱们现在是中心大厦的保安，又不是来送外卖的，你怕什么。”

    感觉到四周围的路人似乎都被张羽的说话声吸引了过来，赵天行连忙说道：“我知道了，张羽你说话声音轻点，别影响到别人。”

    这次画展的规模不小，占据了上千平米的场地，还供应了酒水和自助用餐，在张羽看来与其说是来看画，更像是一种聚会。

    而那传说中的天人演武图此刻便被一层幕布覆盖，据说要等李雪莲女士亲自来揭开。

    张羽便和赵天行被分到这片场地里较靠边的一处，作为保安来回巡逻。

    而此刻画展都尚未正式开始，这巡逻也显得轻松无比，更没有引起仪式反噬。

    毕竟张羽身上的仪式力量针对的是完成愿望的事情。

    而他要完成愿望的方法则是学习，修炼，努力考上大学，工作赚钱……最终来完成邪神的愿望。

    像是临时工作这一类的事情也是工作赚钱乃至学习修炼的一部分，仪式力量并不会逼迫着他转去做别的事情。

    反倒是此刻没事瞎转悠的状态，对张羽来说反而比这几天的苦修轻松了太多太多。

    就在这时，两人靠近餐桌这边的时候，便看到一群侍者正在摆放餐盘，整理桌面装饰。

    张羽瞄了一眼，看着那一碟子一碟子被摆放出来的菜色，惊讶道：“这是……海鲜？”

    赵天行也望了过去，看了看餐盘里的鲍鱼龙虾……微微点头：“是海洋生物，那个带壳的我跟我妈出去的时候看到过，好像一斤就要三、四万呢。”

    “不过这里是中心大厦，有这些也很正常吧。”

    海鲜，张羽在上一世并不少见，也没有少吃。

    但在这昆墟之中，至少在昆墟第一层里，海洋生物是一种极度稀有的存在，甚至原来的张羽只在网上见过，现实里别说吃了，连看都没看到过。

    只因为整个昆墟第一层的2000多万平方公里中……是没有大海的。

    据说这市面上的海洋生物，要么是从更上层的昆墟引进过来，要么就是一些机构自己制造人工海洋环境来养殖的。

    而听到赵天行嘴里那一斤几万块的价格，张羽心中更是暗暗咋舌：“这么贵？”

    “这东西吃了有什么好处？是不是能增长法力？”

    无怪乎张羽这么想，不论是原来的张羽，还是此刻的张羽，他所接触到的仙道世界中，每一个人都在不择手段地前进，拼尽全力地提升自己的法力、武功、道术、肉体、道心……

    衣、食、住、行……生活中的各种东西都无不和提升修为有关。

    “好处？”赵天行闻言想了想，最终摇了摇头：“没听说过吃这个有什么好处，好像就是为了好吃。”

    “就是为了好吃？”张羽看着桌上的一盘盘佳肴微微一愣。

    就在张羽和赵天行两人一同偷偷打量餐桌的时候，一道清冷的声音突然在他们背后响起。

    “你们两个保安，不好好巡逻，在这里偷偷摸摸干什么？”

    听到这道声音，赵天行吓了一跳，转过身来赶紧解释道：“我们没干什么……”

    但看到背后之人的模样，赵天行却是愣在了当场。

    张羽却是施施然地转过身来，看着白真真说道：“咦？竟然是阿真，这么巧你也来看画展啊？”

    而一旁的赵天行发现来人是同班同学后，只觉得身上的保安服越发难看丑陋，身体尴尬地扭来扭去，就像是憋得要原地大便一样。

    特别是想到自己干保安的事情如果被传去班里，贫穷、打工、缺钱等等标签被同学们轮流贴上来，在圈子里的地位、风评便要一落千丈，赵天行就扭得更厉害了。

    白真真面无表情地看着张羽，说道：“想不到你们年纪轻轻，就少走几十年弯路，来这干保安了。”

    赵天行连忙摆摆手，解释道：“不是的，白真真同学，我是陪张羽过来的。”

    张羽却是猛地一掌拍在赵天行的肩膀上：“你慌什么？你可是时薪800，一小时购买力就能够满足凡人十日吃喝的临时保安，说出去不知道多少人要羡慕。”

    “啊？”赵天行闻言微微一愣，从没想过时薪800有什么厉害的。

    但他看着面前突然不说话的白真真，就感觉到一股冷意从对方身上冒了出来。

    对于这位年级总分第一的天才少女，赵天行一直都不太熟悉，只知道对方一直以来都冷冷清清、生人勿进的样子，只跟张羽、周天翊他们一起吃饭。

    事实上赵天行所在的圈子也试过邀请对方，却始终没有得到回应。

    在他心中，白真真就是一名天资纵横，却非常高傲和难接近的天才少女。

    此刻看着对方面无表情，沉默无声的样子，赵天行只觉得对方好像是生气了。

    “一定是张羽乱说话惹她生气了。”
