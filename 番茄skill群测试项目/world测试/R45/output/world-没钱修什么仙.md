# 力量体系四层结构设计

## 第一层：世界源力量

### 源力量名称
**灵矶（灵机沉淀之基）**

### 源力量来源
天地间弥漫着名为“灵矶”的微尘状能量粒子。灵矶并非均匀分布——它像河流一样流动、沉淀于特定的地域（灵脉、古迹、宗门驻地）。灵矶浓度决定了修士的修炼效率上限。灵矶不是天然可被直接吸收的——需要经过“转化通道”（灵根、功法、阵法）才能变为修士可用的灵力。灵矶的存量有限，且被宗门和财阀垄断——穷人只能依赖高利贷租借灵根、购买低效转化设备。

### 源力量→主养成线转化链
灵矶（外界能量粒子）→ 灵根/阵法转化 → 灵力（修士体内基础能量）→ 吐纳法运转 → 法力（进阶位阶的核心指标）→ 位阶突破
- 灵矶浓度高→转化效率高→法力增长快
- 灵根越优质→转化率越高→同样时间收益更大
- 功法等级→决定转化过程的损耗率

### 源力量→敌/怪分级标准
以敌人对灵矶的驾驭能力分级：
- **灵聚级**（炼气期）：体内灵力来自外界灵矶，靠借/买灵根维持——无灵矶区域等于凡人
- **灵生级**（筑基期）：已种入自身灵根，可自主从环境中缓慢吸引灵矶——断供后仍能生存
- **灵核级**（金丹期）：体内凝聚灵核，可主动吸收消耗灵矶，甚至可抽取周边区域灵矶——开始影响环境
- **灵域级**（元婴期）：灵核进化为灵域，可划定区域改造灵矶流动——改变战场规则
- **灵矶源级**（化神及以上）：自身即为灵矶源头之一，可创造灵矶——世界改造者级别

| 阶位5 | **化神期** | 传说层次——触及世界本源“灵矶”的诞生与崩坏。不再依赖环境中的灵矶——自身可创造灵矶（微型灵矶源）。拥有“界域”能力——在界域内，敌人无法吸取灵矶，而创造者法则优先 | 揭开了“灵矶”和“古法炼器”的真正秘密——这是上一轮仙道文明毁灭的源头。主角意识到，自己身处一个巨大的“棋盘”中：自己被选中为古法传人，不是为了变强，而是为了重启某个上古阵法/修复某件关键法器。他必须做出选择：顺从还是反抗？ | 灵魂→神识→心神完成“道心合一”。不再是由外部资源堆积而成，而是对“灵矶本源”的理解达到了哲学/法则层面的顿悟——必须亲历过“灵矶枯竭”的危险，洞察灵矶的“生灭法则”，然后才可能与之共振 | **世外传说、隐世真仙级**。全大陆已知数量<10人，普通民众视为传说。他们不再参与世俗权力游戏，但他们的存在本身就是大陆的“神秘威慑”。社会地位≈神话人物——偶尔显灵，但不被凡俗所知 | 第五卷 |
| 阶位6 | **渡劫期** | 终章舞台——世界正在“灵矶衰竭”（源力量危机）。大陆上的灵矶浓度持续下滑，高阶修士开始感受“灵力退化”。修士之间的力量依赖度下降，阴谋取代正面战斗成为主流。主角需要利用炼器修复世界核心，同时对抗预谋已久的幕后黑手 | 最终的抉择：①修复灵矶来源（可能牺牲自己作为新灵矶源）②带领少数人逃离这块垂死的世界（寻找新大陆）③用禁忌之法强行唤醒上一轮文明的残骸，用古法炼制一个新世界。同时面临最强大敌人——那些不愿世界改变、想靠最后一点灵矶苟延残喘的保守派 | 元婴→化神→渡劫的每一步之前已铺垫。终极突破条件：①完成上古法器“世界之心”的修复（主线目标）②与灵矶的本源共鸣——理解“灵矶”是一种有意识的能量，它有自己的“生命周期”③在道心动摇的最后关头，接受“自我毁灭以重造新灵矶源”的献祭式代价 | **仅存传说、世界末日的最后希望**。数量≤3人（敌对方）。社会地位≈“救世主”或“灭世者”。不再是阶层——他们是改变阶层本身结构的力量 | 最终卷（第六或第七卷） |
| 阶位7 | **大乘期**（隐藏） | 超脱层次——主动“终结”了旧的灵矶循环，成为新世界的“规则设计者之一”。不再生活于旧的修仙世界，而在新创造的“第二天地”中构建规则循环 | 实现了从“夺灵者”到“造灵者”的转变。不再需要修炼——因为修炼的规则由他设计。最终敌人（可能是上一轮文明残骸、或世界本身的意志）也在此层面被解决。主角须决定自己在新世界的角色：隐居、主导、或彻底隐去 | 必须在渡劫+修复世界核心时完成“献祭”与“重生”两个仪式，跳出旧世界的灵矶依附关系。不再依赖“灵矶量”，而靠“规则制定权”。这是全书唯一一次触及，最终章揭示 | **不再属于“社会阶层”——属于“创世纪话”** | 最终卷（尾声） |

| **涉及哪些子养成线** | 法力线 + 肉体强度线（两条线均受影响） |
| **协同/制约类型** | 外部强制协同（仪式压力↑→双线均被迫提升） |
| **具体规则** | 邪神仪式/传承契约（如上界羽书）会24小时监控修炼意愿。一旦偷懒/懈怠，即用死亡倒计时（10、9、8...）施加心理压迫。强制效果：不限定“做什么”，但强制“向着完成愿望”——修炼/打工/做任务。这导致主角无法松懈→被迫持续修炼，法力和肉体强度均在高压环境下被迫共同提升 |
| **参考源** | decon·表1·交叉规则·仪式压迫机制（本项目中机制为“古法炼器传承”的烙印监控器——主角每修复一件高阶法器，会短暂激活，持续监控“是否在向修复更高阶法器努力”） |

### 兼修平衡总结
- 两条子养成线均受金钱、符箓/补剂/药方的消费驱动
- 双修需面对：①资金不足 ②时间分散 ③肉体损伤累积
- 单线专精：①成长更快但另一面拖后腿 ②阶位突破时若另一线太弱可能卡住（如道心不足不能突破炼气期）
- 世界设计倾向：金钱驱动+债务陷阱+时间挤压→迫使主角必须在“短期爆发”和“长期均衡”间不断做抉择→制造剧情张力
