# 力量体系四层结构设计

## 第一层：世界源力量

| 字段 | 内容 |
|:--|:--|
| **源力量名称** | **器脉灵韵** |
| **源力量来源** | 上古炼器文明崩毁后，破碎的器灵碎片融入天地灵气，形成一种特殊灵力——器脉灵韵。凡有法器埋藏、遗迹留存之地，器脉灵韵浓度更高。宗门立派常选址于古器战场遗址，借此地利修炼。 |
| **源力量→主养成线转化链** | 器脉灵韵→被修士吸入炼化→转化为体内灵力→修为增长。普通修士只能被动吸收天地间的器脉灵韵（效率极低）；拥有古法炼器传承的主角可通过修复法器、接触器灵残影，主动吸收高纯度器脉灵韵（效率提升数十倍）。 |
| **源力量→敌分级标准** | 敌人（妖兽/邪修）按法器灵压等级划分：<br>· 凡器级（对应胎息境修士）<br>· 法器级（对应练气境修士）<br>· 灵器级（对应筑基境修士）<br>· 法宝级（对应金丹境修士）<br>主角的阶位对应能修复/对抗的敌级上限 |

## 第三层：子养成线

### 子养成线1：古法炼器传承（对应decon·玄珠符种体系）

| 字段 | 内容 |
|:--|:--|
| **子养成线名称** | **古法炼器传承** |
| **自身阶位** | 初觉→通窍→铭心→融魂→化道（5级） |
| **自身阶位说明** | · 初觉：能感知法器中的器灵残影<br>· 通窍：能看清器灵记忆碎片的画面<br>· 铭心：能完整阅读器灵记忆中的功法/法术<br>· 融魂：能与器灵残影短暂共鸣（获得该法器原主人的实战技巧）<br>· 化道：能将器灵记忆碎片的规则刻入自身道基 |
| **和主养成线挂钩关系** | 主养成线阶位决定子养成线上限：<br>· 熔火淬体→上限初觉<br>· 铸灵化元→上限通窍<br>· 铭纹凝识→上限铭心<br>· 玄器通灵→上限融魂<br>· 器魂合道→上限化道<br>· 每次修复法器获得器灵记忆碎片→累积→提升子养成线 |
| **等级提升规则** | 获取：修复法器时，从器灵残影中提取记忆碎片，累积足够碎片数量+选择关键碎片融入自身意识<br>消耗：每次融合碎片消耗灵力，越高级的碎片消耗越大<br>上限：主养成线阶位硬上限 |
| **参考源** | decon·表1·子养成线1·玄珠符种体系 |

### 子养成线2：法术/神通学习线（对应decon·法术/神通学习线）

| 字段 | 内容 |
|:--|:--|
| **子养成线名称** | **法诀修习** |
| **自身阶位** | 初窥→小成→贯通→融会→入道（5级） |
| **自身阶位说明** | 每门法术/功法的熟练度等级，不同法术难度不同，上限不同 |
| **和主养成线挂钩关系** | 主养成线阶位决定可学习的法诀上限等级：<br>· 熔火淬体→只能学习凡级法诀（基础控火术、净衣术），上限初窥<br>· 铸灵化元→可学习黄级法诀（炼器术入门、金光术），上限小成<br>· 铭纹凝识→可学习玄级法诀，上限贯通<br>· 玄器通灵→可学习地级法诀，上限融会<br>· 器魂合道→可学习天级法诀，上限入道 |
| **等级提升规则** | 获取：通过宗门传承/修复法器获得的器灵记忆碎片（内含原主人的法术修行感悟）/坊市购买<br>消耗：学习施展消耗灵力，高级法诀需在特殊环境中修炼<br>上限：主养成线位阶上限 + 法诀自身难度上限 |
| **参考源** | decon·表1·子养成线2·法术/神通学习线 |

### 子养成线3：灵根资质（对应decon·灵窍资质线）

| 字段 | 内容 |
|:--|:--|
| **子养成线名称** | **灵根资质** |
| **自身阶位** | 废灵根→杂灵根→三灵根→双灵根→单灵根→天灵根（6级，**主角固定废灵根**） |
| **自身阶位说明** | 先天资质，主角是废灵根——这是设定硬约束，不成长<br>但通过古法炼器传承，主角**绕过了资质限制**：修复法器时接触器脉灵韵，吸收效率可比拟单灵根（甚至更高） |
| **和主养成线挂钩关系** | 完全绕过——灵根资质决定常规修士的修炼上限（废灵根终身止步熔火淬体），但古法炼器传承为主角打开一条无视灵根的路径。<br>**这是本项目核心张力**：灵根资质被废，但炼器传承成为替代方案 |
| **等级提升规则** | 获取：先天固定，不可提升<br>消耗：无<br>上限：固定，主角为废灵根<br>**备注**：此线在本项目中作为“被绕过的障碍”存在，不消耗主角资源 |
| **参考源** | decon·表1·子养成线3·灵窍资质线（本项目改动：保留其制约性，但通过传承绕过） |

### 子养成线4：外物/丹药/灵植线（对应decon·外物/丹药/灵植线）

| **自身阶位** | 匮乏→自足→充裕→富足→豪奢（5级） |
| **自身阶位说明** | 主角可支配的灵石/丹药/炼器材料的等级和数量<br>· 匮乏：穷得叮当响，只够维生<br>· 自足：完成日常修复任务后略有盈余<br>· 充裕：能主动囤积中阶炼器材料<br>· 富足：灵脉洞府+稳定丹药供应<br>· 豪奢：可批量采购稀有材料，炼制高阶法器 |

### 子养成线5：炼器百艺（对应decon·修仙百艺线）

| 字段 | 内容 |
|:--|:--|
| **子养成线名称** | **炼器百艺** |
| **自身阶位** | 凡品→黄品→玄品→地品→天品（5级） |
| **自身阶位说明** | 能修复/炼制的法器最高品阶<br>· 凡品：修复凡器（凡级法器的各个部件）<br>· 黄品：修复黄级法器<br>· 玄品：修复玄级法器<br>· 地品：修复地级法器<br>· 天品：修复天级法器 |

| **参考源** | decon·表1·子养成线5·修仙百艺线 |

## 第四层：交叉规则

### 交叉规则1：传承×资质替代（对应decon·玄珠符种×灵窍资质）

| 字段 | 内容 |
|:--|:--|
| **规则名称** | **传承替代资质** |
| **涉及子养成线** | 古法炼器传承 × 灵根资质 |
| **协同/制约类型** | **协同**（传承替代/覆盖资质） |
| **具体规则** | 灵根资质决定常规路径上限，但古法炼器传承提供替代通道。主角通过修复法器吸收器脉灵韵，效率相当于单灵根修士，完全绕开灵根限制。<br>**代价**：无法通过常规修炼（打坐）获得灵力，必须持续修复法器才能推进主养成线。 |
| **参考源** | decon·表1·交叉规则1·玄珠符种×灵根资质 |

### 交叉规则2：传承×主养成线（对应decon·玄珠符种×主养成线）

| 字段 | 内容 |
|:--|:--|
| **规则名称** | **修复推动修为** |
| **涉及子养成线** | 古法炼器传承 × 主养成线 |
| **协同/制约类型** | **协同**（传承是主养成线的核心加速器） |
| **具体规则** | 每次修复法器，主角从器脉灵韵中吸收相当于自己打坐3-30天的灵力（取决于法器品阶）。修复高阶法器还能获得器灵记忆碎片→记忆碎片融合→主养成线突破。<br>**代价**：修复失败反噬受伤，损失灵力且可能短期无法修复。 |
| **参考源** | decon·表1·交叉规则2·玄珠符种×主养成线 |

### 交叉规则3：主养成线×法诀修习（对应decon·主养成线×法术线）

| 字段 | 内容 |
|:--|:--|
| **规则名称** | **位阶锁法诀** |
| **涉及子养成线** | 主养成线 × 法诀修习 |
| **协同/制约类型** | **制约**（位阶不够→无法学习高阶法诀） |
| **具体规则** | 主养成线阶位是法诀修习的硬门槛。熔火淬体期只能学基础法诀；需到铭纹凝识（三阶）才能学习高阶炼器术。<br>但主角可通过器灵记忆碎片获得“原主人的法诀修炼经验”→一定程度上降低学习门槛（如三阶时可学本需四阶的法诀初篇）。 |
| **参考源** | decon·表1·交叉规则3·主养成线×法术线 |

### 交叉规则4：资源积累×主养成线（对应decon·外物线×主养成线）

| 字段 | 内容 |
|:--|:--|
| **规则名称** | **资源加速修炼** |
| **涉及子养成线** | 资源积累 × 主养成线 |
| **协同/制约类型** | **协同**（资源越丰富→修炼越快） |
| **具体规则** | 丹药可直接转化为灵力（非器脉灵韵），辅助主养成线推进。灵石可用于购买更高阶的法器残件→修复→获取更多器脉灵韵。<br>资源匮乏时，主角只能靠基础修复维持修炼，速度极慢。这是主角行为引擎的核心动力——必须持续接单赚钱买材料。 |
| **参考源** | decon·表1·交叉规则4·外物线×主养成线 |

### 交叉规则5：资源积累×炼器百艺（对应decon·外物线×百艺线）

| 字段 | 内容 |
|:--|:--|
| **规则名称** | **以炼养炼** |
| **涉及子养成线** | 资源积累 × 炼器百艺 |
| **协同/制约类型** | **协同**（炼器产出→换取资源→支撑更高级炼器） |
| **具体规则** | 炼器百艺等级越高→可修复的法器越高级→报酬越高→灵石+稀有材料越多→可购买更好的修复工具和丹药→支撑继续提升炼器等级。<br>**这是核心循环**：收法器→鉴定→修复→交付→获得报酬/记忆碎片→买材料→修更好的法器。 |
| **参考源** | decon·表1·交叉规则5·外物线×百艺线 |

### 交叉规则6：主养成线×炼器百艺（对应decon·主养成线×百艺线）

| 字段 | 内容 |
|:--|:--|
| **规则名称** | **以修证道** |
| **涉及子养成线** | 主养成线 × 炼器百艺 |
| **协同/制约类型** | **制约~协同双向**（先制约后协同） |
| **具体规则** | 低阶时（阶位1-2）：炼器百艺等级不能超过主养成线阶位（灵力总量不足，无法支撑高阶炼器的消耗）。<br>高阶时（阶位3+）：两者相互促进——主养成线提升→可修更高级法器→获取更精纯的器脉灵韵/记忆碎片→反哺主养成线突破。最后一个阶位（万器朝宗）甚至要求主角炼制出本命法器才能突破。 |
| **参考源** | decon·表1·交叉规则6·主养成线×百艺线 |

