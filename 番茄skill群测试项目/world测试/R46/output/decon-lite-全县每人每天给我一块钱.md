# 表9:动力引擎提取表 —— 《全县每人每天给我一块钱》(我还是一根葱)

> 拆书人:pop | 拆书日期:2026-07-21 | 采样范围:前30章
> 赛道:都市金手指流(重生 + 人口绑定系统)
> 范式归类判据:飞轮型=正反馈闭环+外部输入维持;压迫型=外部压力+倒计时驱动;探索型=靠发现新区域/新规则驱动;对抗型=靠与同位竞争者博弈驱动;混合型=两种范式叠加

---

## 9.1 驱动逻辑(为什么攀登)

- **世界级驱动力**:经济振兴——青泽县是国家级贫困县(常住人口不到三十万,人均月收入不到三千),年轻人大规模外出务工,县城经济凋敝(ch1"常住人口三十万都不到,年轻人都跑光了,留下的全是大爷大妈")。系统规则将主角收益与全县人口直接绑定,迫使主角振兴县城经济以提升自身收入。这不是个人动机,而是系统强制将"全县经济"变成主角的核心利益。
- **驱动力覆盖范围**:仅主角攀登。系统只绑定主角一人,全县其他人并不知道系统存在。但主角的攀登行为(开厂、高福利)客观上带动全县人口共同富裕,形成"一人攀登→全县受益"的辐射结构。
- **驱动力来源**:制度强制(系统规则强制绑定人口与收益) + 资源稀缺(贫困县人口流失严重)
- **原文证据**:
  - ch1 "青泽县每有一个人,系统就会每天给他发一块钱。"
  - ch1 "系统的规则是按当日人口给钱。"
  - ch1 "青泽县目前的常住人口是28万7千4百21人……每天什么都不干,只要睁开眼,账户里就会多出将近二十九万的现金。"
  - ch25 "系统的核心机制绑定的是人口。服装厂只是手段,是他在这个穷县城制造就业、留住人口的第一块砖。"

## 9.2 运转机制(靠什么攀登)

- **闭环结构**:输入(系统日收入 = 当日青泽县人口数 × 1元) → 转化(将系统收入投资产业:开服装厂、高福利高单价) → 输出(提供就业岗位 → 留住本地人口 + 吸引外出务工人口回流) → 反馈(县域人口增加 → 系统日收入增加 → 可投入更多产业建设 → 吸引更多人口)
- **加速器**:
  1. 系统日收入无上限——人口越多收入越高,且每日自动到账,无需主动操作(ch1"每天什么都不干,只要睁开眼,账户里就会多出将近二十九万的现金")。
  2. 高福利高单价策略——主动将利润让给工人(净利润不超过10%,底薪3000另贴),制造口碑传播效应,比招聘广告更有效(ch25"高福利、高单价,这种消息在工人圈子里传得比任何招聘广告都快")。
  3. 重生信息优势——主角知道哪些工人有真本事(如周桂兰的手艺),提前锁定关键人才。
- **减速器**:
  1. 适龄劳动力有限——青泽县常住适龄女性劳动力本就稀少,第一波招工后急剧下降(ch25"昨天只来了两个,今天到现在一个都没有")。
  2. 前老板李建国欠薪跑路的信任危机——工人对"新服装厂"普遍不信任,招工阻力极大(ch20"被李建国那事儿伤过的人,身上都有一种味道,叫作'被背叛过的警觉'")。
  3. 人口流出惯性——年轻人大规模南下打工的趋势难以逆转(ch1"年轻人都跑光了")。
- **闭环是否自洽**:是。系统收益→投资产业→留住人口→系统收益增加的正反馈闭环逻辑成立。前30章已展示完整链条:重生获得系统(ch1)→开服装厂(ch7-10)→招工遇阻(ch11-14)→高福利策略突破(ch25-26)→口碑传播吸引更多工人(ch26)。
- **原文证据**:
  - ch1 "青泽县每有一个人,系统就会每天给他发一块钱……系统的规则是按当日人口给钱。"
  - ch25 "留人、引人、筛人。三步棋,第一步已经落下了。"
  - ch25 "他真正在意的,不是养得起这五十个人,而是这五十个人能帮他吸引更多的人。"
  - ch26 "这帮女人今晚回到家,会跟老公说、跟邻居说、跟娘家妈说……一个人回去说一句'我们厂一件衣服能挣多少钱',比他在厂门口贴十张招工启事都管用。"
  - ch26 "在开发区B12厂房,踩缝纫机,底薪三千,计件另算,一个月能挣八九千,手艺好的过万。"

## 9.3 代价结构(攀登代价)

- **代价清单**:金钱(大量投入产业建设) / 精力(管理工厂) / 隐蔽性风险(系统存在不能暴露)
- **主要代价**:金钱代价——主角必须将系统收入大量投入产业建设(设备采购、工人工资、厂房租金),且主动放弃利润(净利润不超过10%),本质上是用系统收益补贴全县经济。但这一代价被系统日收入(近29万/天)完全覆盖,不构成真实压力。
- **代价是否真实可感**:否。系统日收入近29万(ch1),而工人月工资支出约15万(ch25"光人员工资每个月就要十五万"),系统一天收入即可覆盖两天工人支出。主角明确表示"我有钱烧的"(ch25),金钱代价被系统收益消解,不构成核心张力。真正的张力来自招工困难(信任危机)和人口增长瓶颈,而非代价本身。
- **原文证据**:
  - ch1 "每天什么都不干,只要睁开眼,账户里就会多出将近二十九万的现金。"
  - ch25 "刚才那张成本表里没算固定支出,光人员工资每个月就要十五万……但现在,系统一天的收益就能覆盖。对他而言,无伤大雅。"
  - ch25 "让他们骂,我的买卖我愿意,我有钱烧的。"
  - ch25 "利润薄不要紧。他要的从来就不是加工费里那点差价。"

## 9.4 范式归类

- **主范式**:飞轮型
- **范式依据**:正反馈闭环 + 外部输入维持。系统日收入(外部输入)→投资产业→留住人口→系统日收入增加(正反馈),构成典型的飞轮结构。飞轮的外部输入是"青泽县人口",转化器是"产业建设与高福利策略",输出是"人口增长",反馈是"系统收益增长→更多投资"。
- **是否存在范式演化**:否(前30章保持飞轮型一致)
- **演化触发事件**:无。前30章主角始终围绕"留住人口→增加系统收入"这一飞轮运转,策略从"开厂"升级到"高福利"但飞轮结构未变。

## 9.5 动力引擎与力量体系的关系

- **引擎与骨架关系**:引擎是骨架本身。本书无传统修炼/战力体系,系统的"人口绑定收益"既是动力引擎,也是唯一"力量体系"——主角的"力量"就是钱(系统收益),钱的多少直接由人口决定。引擎即骨架,无平行关系。
- **引擎与世界观同构度**:高。系统规则(人口→收益)与现实世界的经济逻辑(人口→劳动力→经济产出)高度同构,系统只是将现实经济逻辑压缩为"一个人=一块钱/天"的简化模型,主角的产业建设行为完全符合现实经济规律。
- **引擎与赛道的关系**:互相塑造。都市金手指赛道决定了系统绑定"县域人口"这一现实变量,而系统规则反过来塑造了赛道走向——主角必须走"产业振兴"路线而非"都市打脸"路线,系统将传统的都市重生文改造为"县域经济建设文"。

## 9.6 三组成合一性评估

- **驱动逻辑+运转机制+代价结构是否合一为"力量体系+社会结构+爽感引擎"**:完全合一
- **合一性证据**:驱动力(系统绑定人口)、运转机制(投资产业→留住人口→收益增长)、代价结构(金钱投入,被系统覆盖)三者完全合一于"人口经济飞轮"。力量体系(系统收益=钱)、社会结构(县域经济+工人群体)、爽感引擎(高福利让工人月薪过万,情感爽感拉满)三者高度耦合——系统收益直接转化为工人的高工资,高工资直接产生情感爽感,情感爽感直接驱动口碑传播,口碑传播直接增加人口,人口直接增加系统收益。闭环完全闭合。
  - ch26 "在青泽县,月薪过万是什么概念?县中学的骨干教师,月薪四千二……而一个踩缝纫机的女工,过万了。"——力量体系(钱)、社会结构(县域经济)、爽感引擎(底层逆袭)三者合一的典型场景

## 9.7 辨识度评估

- **只看位阶结构,这本书有辨识度吗**:无。本书无数值化等级、无修炼境界、无战力位阶。主角的"等级"只有银行余额数字,不构成传统意义上的位阶结构。
- **加入动力引擎后,辨识度**:显著提升。系统规则"县域人口=日收入"是本书核心辨识点,这一引擎将"都市重生打脸"改造为"县域经济飞轮",在同赛道中具有较高辨识度。高福利策略(净利润不超10%、底薪另贴)与工人情感线(周桂兰、王小慧的逆袭)结合,构成独特的"共同富裕爽感"范式。
- **反例风险**:中低。本书等级系统(系统收益)虽非传统数值化等级,但系统规则明确、闭环自洽、引擎核心性强,不属于"等级系统不核心"的反例。唯一风险是代价结构过弱(系统收益完全覆盖支出),可能导致后期张力不足,但前30章通过招工困难(信任危机)弥补了这一缺陷。
