# 表9:动力引擎提取表 —— 《廓晋》(榴弹怕水)

> 拆书人:pop | 拆书日期:2026-07-21 | 采样范围:前30章
> 赛道:历史穿越(东晋永和年间)
> 范式归类判据:飞轮型=正反馈闭环+外部输入维持;压迫型=外部压力+倒计时驱动;探索型=靠发现新区域/新规则驱动;对抗型=靠与同位竞争者博弈驱动;混合型=两种范式叠加

---

> **特别标注:本书无典型等级系统。**
> 无数值化面板、无修炼境界、无系统任务/奖励机制。主角金手指为"历史知识"(穿越者携带的后世记忆),非系统化养成框架。以下分析以"剧情驱动力"替代"等级系统引擎"进行拆解,作为边界案例供范式覆盖参考。

---

## 9.1 驱动逻辑(为什么攀登)

- **世界级驱动力**:乱世生存——东晋永和年间,衣冠南渡三十年,南北对立,石虎死后北方大乱,淮北士族百姓纷纷南渡逃难,所有人都在为活下去而挣扎(ch1"淮北地区处于南北之间,战乱频仍,起圩、抛荒、逃难、屠城,屡见不鲜")。同时存在第二层驱动力:门阀等级制度——士族与寒门之间存在不可逾越的鸿沟,社会地位决定生存资源分配(ch17"若是我真与他们说了话,必会被其他高门耻笑")。
- **驱动力覆盖范围**:全员攀登。流民为生存攀登(找粮食、找庇护),士族为门第巩固攀登(维持权势、北伐建功),主角为"坞堡主"梦想攀登(ch1"难道还不能打起精神,奋斗个坞堡主做做?")。
- **驱动力来源**:生存压力 + 制度强制(门阀等级制度)
- **原文证据**:
  - ch1 "石虎死了!……北方风起云涌,羯人、鲜卑人、氐人、羌人、汉人……你来我往,他死彼亡"
  - ch1 "天下正朔所在,大晋朝廷之任大都督青、扬、徐、兖、豫五州诸军事褚裒正式率王师北伐"
  - ch1 "都到了东晋十六国了……难道还不能打起精神,奋斗个坞堡主做做?……这是天胡开局啊!"
  - ch17 "若是我真与他们说了话,必会被其他高门耻笑"

## 9.2 运转机制(靠什么攀登)

- **闭环结构**:输入(穿越者的历史知识 + 流民劳动力) → 转化(利用历史预判做正确决策 + 产业建设 + 人际攀附) → 输出(物资积累 + 社会地位提升) → 反馈(接触更高层级人物 → 获得更多资源与机遇 → 进一步提升地位)
- **加速器**:历史知识金手指——主角知道褚裒北伐必败(ch1"必然是这位大都督北伐一败涂地了"),知道淝水之战时间线(ch1"淝水之战还有多少年?"),知道王谢桓等历史人物走向,由此做出精准决策:冒姓彭城混入流民帅队伍、提前南渡避祸、用草屩产业获得刘任公信任、猎虎事件接触陈郡谢氏。
- **减速器**:门阀等级制度(寒门与士族的鸿沟,ch17谢泉不愿与彭城刘氏对话);资源匮乏(流民一无所有,ch1"靠着帮人背行李、地里捡残留庄稼");身份隐患(冒姓彭城随时可能暴露,ch17刘吉利追问"你父祖到底是谁")。
- **闭环是否自洽**:是。历史知识→正确决策→地位提升→更多机遇的闭环逻辑成立,前30章已展示完整链条:冒姓入队(ch1)→织屩获信(ch2-14)→猎虎接触谢氏(ch15-17)→谒见大都督(ch15预告)。
- **原文证据**:
  - ch1 "必然是这位大都督北伐一败涂地了……自己得赶紧离开彭城,最好学着那部分渡泗水的人,继续往南逃啊!"
  - ch1 "刘阿乘决定冒姓彭城!"——历史知识驱动下的核心决策
  - ch15 "刘任公亲口说了,官府的安置救济要到了"——产业建设获得官方认可
  - ch16-17 猎虎事件:主角指挥善后("把虎抬出去,大张旗鼓的抬出去!"),接触陈郡谢氏,获得帷帐物资
  - ch17 "人得活下去,才能说别的"——闭环的底层逻辑

## 9.3 代价结构(攀登代价)

- **代价清单**:尊严 / 安全 / 人际 / 身份隐患
- **主要代价**:尊严代价——主角必须忍受门阀歧视,甘心自污下贱以换取生存资源。这是核心张力来源:一个现代白领的灵魂困在乱世最底层,必须向门阀低头。
- **代价是否真实可感**:是。主角的自污行为(索要谢家脏污帷帐)被同伴明确指出并引发内心讨论,代价有具体场景与情感重量。
- **原文证据**:
  - ch17 "偏偏你竟忍住了,甚至于甘心自污下贱,为营地里的妇孺取那些冬日衣料"——刘吉利对主角尊严代价的直观评价
  - ch17 "那又如何呢?大家穷的朝不保夕,若是这类自污下贱便能取这么多好布,我巴不得多来几次"——主角对代价的理性接受
  - ch1 "刘阿乘决定冒姓彭城!"——身份造假的安全代价(随时可能暴露)
  - ch16 "这老虎被他们撵的明显要闯入一家甲门士族的野餐帷帐里,但凡死伤一两个,刘任公怕是要立即提桶跑路"——行动的安全风险

## 9.4 范式归类

- **主范式**:混合型(探索型 + 对抗型)
- **范式依据**:探索型——主角靠发现历史机遇(预判北伐失败、预判谢氏崛起)和发现社会规则(门阀运作逻辑)驱动攀登;对抗型——主角在门阀等级制度中与高门博弈(猎虎事件中与谢氏的接洽、冒姓彭城的身份博弈),同时与同位竞争者(刘虎子、刘吉利等流民内部角色)争夺话语权。
- **是否存在范式演化**:否(前30章保持混合型一致)
- **演化触发事件**:无。前30章范式稳定,历史知识驱动+门阀博弈贯穿始终。

## 9.5 动力引擎与力量体系的关系

- **引擎与骨架关系**:无骨架(本书无超自然力量体系),引擎就是剧情本身。所谓"力量"是社会地位与政治权力,引擎(历史知识驱动决策)直接产出这种"软力量"。
- **引擎与世界观同构度**:高。东晋门阀政治的世界观本身就是等级制度,主角的历史知识引擎与这一世界观高度契合——知道历史走向就等于知道等级制度的运作方向。
- **引擎与赛道的关系**:赛道决定引擎。历史穿越赛道决定了金手指是"历史知识"而非"系统面板",引擎必须在真实历史框架内运转。

## 9.6 三组成合一性评估

- **驱动逻辑+运转机制+代价结构是否合一为"力量体系+社会结构+爽感引擎"**:部分合一
- **合一性证据**:驱动力(乱世生存)、运转机制(历史知识驱动决策)、代价结构(尊严妥协)三者围绕"乱世底层求生"这一核心主题合一,但缺少独立的力量体系维度——社会结构(门阀等级)本身就是力量体系,爽感引擎(历史知识开挂)直接嵌入社会结构中运作,三者耦合度高但并非完全合一,因为"力量体系"这一维度被社会结构替代而非独立存在。
  - ch17 "人得活下去,才能说别的"——三组成的合一基点:生存驱动、知识运转、尊严代价统一于"活下去"

## 9.7 辨识度评估

- **只看位阶结构,这本书有辨识度吗**:无。本书无等级系统、无数值化位阶、无修炼境界。社会地位的"软等级"(流民→寒门士族→高门士族)是历史赛道通用结构,非本书独创。
- **加入动力引擎后,辨识度**:略提升。历史知识金手指+坞堡主目标构成一定辨识度,但"穿越者靠历史知识在古代生存"是历史穿越赛道的常见模式,辨识度提升有限。
- **反例风险**:高。本书等级系统不核心,甚至无等级系统。作为男频等级系统验证项目的边界案例,本书验证了"历史赛道可无典型等级系统,靠剧情驱动力(历史知识+门阀博弈)维持攀登感"。若项目目标是验证"等级系统核心性",本书是反例;若项目目标是"范式覆盖",本书补充了"无等级系统的剧情驱动型"范式。
