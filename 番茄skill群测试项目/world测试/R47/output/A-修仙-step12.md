#### 2a. 力量体系设计（广义坐标系·修仙赛道）

**设计宣言**：本项目采用**修仙位阶**作为核心坐标系。此坐标系划出舞台，定义了“强弱的刻度”和“位置的空间”。主角通过“炼器”这一非主流但合法的路径在这个坐标系内攀爬。

##### 第一层：世界源力量

| 字段 | 内容 |
|:--|:--|
| **源力量名称** | **炉火真意** & **器灵残念** |
| **源力量来源** | 古法炼器时代，器道宗师与天地交感，将自身道途印刻于所铸法器之中，形成“器灵”。器灵承载着原主人的一缕修炼感悟（器灵残念）。这股力量游离于天地灵气之外，是炼器师才能调动的特殊力量。古代“器道之争”后，古法炼器传承断绝，天地间的“炉火真意”（器灵产生的本源）逐渐稀薄，导致现代炼器只能炼制无灵的法器。 |
| **源力量→主养成线转化链** | 主角通过修复古法器，接触“炉火真意”→提取“器灵残念”（修炼感悟碎片）→以此为资粮，一边提升炼器水平，一边将感悟炼化为自身修为，加速主养成线（胎息→练气→筑基…）的突破。 |
| **源力量→诡/怪/敌分级标准** | 本世界“诡/敌”主要为**失控的法器、被邪修控制的器灵、以及觊觎古法的妖兽/魔修**。分级标准按其原法器品阶决定：凡器→法器（对应胎息-练气）→宝器（对应筑基）→灵器（对应紫府）→道器（对应金丹以上）。失控品阶越高，危险越大。 |

#### 2b. 动力引擎设计（众生攀登系统）

##### 组成1：驱动逻辑（众生为什么爬）

| 字段 | 内容 |
|:--|:--|
| **驱动力类型** | **生存压力 + 资源稀缺** |
| **驱动力描述** | 这是一个**修仙资源被顶级宗门和世家彻底垄断**的世界。没有灵根，连门都入不了；有灵根，也会被卡死在功法和资源上。修士的每一次突破，都意味着对稀缺资源的残酷争夺。散修如同野狗，为了一颗下品灵石就能搏命；宗门弟子为了一个筑基名额，不惜手足相残。**不攀登，就是被剥削、被奴役、最终被时代抛弃，连凡人都不如。** |
| **主角欲望对齐** | 主角陆沉的欲望非常朴素：**活下去，并且活得有尊严**。他没有宏大的拯救世界理想，只想靠着炼器手艺，让自己一个灵根废柴也能看到向上的希望，不再任人欺凌。这种欲望，是残酷世界生存压力的个人化体现。 |

##### 组成2：运转机制（众生怎么爬的闭环）

| 字段 | 内容 |
|:--|:--|
| **输入** | 投入**时间、灵力、精力、知识、人脉、资源（灵石/材料）**。普通修士打坐苦修，接宗门任务赚钱；世家子弟用灵石买丹药，请名师指点。 |
| **转化** | **功法修炼**（打坐炼气）；**战斗历练**（在生死边缘激发潜力）；**资源消耗**（吞服丹药、炼化灵石）；**知识学习**（领悟功法、研究阵法）。这是一个低效的、通用的、所有人都能走的路径。 |
| **输出** | **境界提升**、**神识增强**、**法术威力变大**、**寿命延长**。 |
| **反馈 (反哺)** | **正反馈**：境界越高→能获取的资源越多（接高级任务/占据灵脉）→修炼越快。**负反馈**：境界越高→突破需要的资源指数级增长→必须面对更危险的挑战、卷入更大的势力纷争。 |
| **闭环图 (主角/炼器师特化)** | 主角的攀登闭环是这个世界法则下的一个**高效特化分支**：<br>`修复法器 (输入: 废品+灵力+知识)` → `提取器灵残影 (转化: 独有捷径)` → `获得感悟/资源/声望 (输出)` → `炼化感悟提升修为/人脉 (反馈)` → `接更高阶的修复委托 (正反馈加速)` |

#### 2f. 金手指定位（助力定位）

| 字段 | 内容 |
|:--|:--|
| **金手指类型** | **觉醒型/知识传承型** |
| **加速机制** | 金手指（古法炼器传承）提供了 **“资源转化和知识获取”** 的双重加速器：<br>1. **将废品转化为财富**：别人看不上的废品法器，主角能完美修复并转卖，迅速积累初始财富。<br>2. **将死物转化为知识**：能提取器灵记忆碎片（修炼感悟、功法片段、秘境地图），这是用钱都买不到的知识。这相当于让主角在攀登坐标系时，同时拥有“经验加速”和“地图攻略”双重优势。 |
| **限制** | ①灵力消耗巨大（限制修复频率）②碎片排斥且有时效（限制知识吸收率）③有失败爆炸风险（增加攀登风险）④修复时无感知（致命弱点）⑤仅对“有灵法器”有效（有边际效应） |
| **代价** | 每一次修复，都是一次对自己**生命力和神识的献祭**，并在体内留下一丝不属于自己的“器灵残念”，为日后“器灵反噬”（发动机代价危机）埋下伏笔。爬得越快，代价积累越深。 |
| **与引擎的关系** | 金手指是众生通用引擎（打坐、战斗、嗑药）上的一个高效能**涡轮增压器**。主角的攀登逻辑仍然遵循“投入→转化→产出→反馈”的通用闭环，但金手指让他把投入（废品+灵力）转化成了别人无法想象的产出（感悟+财富+人脉），从而实现提速。 |
| **喧宾夺主检查** | ✅ **通过**。<br>1. **坐标系意义未消失**：主角依然要突破胎息、练气的境界壁垒，每个阶位的质变依然需要海量灵力和经验。金手指只是让他更快地获得资源，但不能让他躺着升到金丹。<br>2. **众生攀登意义未消失**：主角每次修复都是冒着生命危险拼出来的，不是凭空获取。他依然要遵循世界的丛林法则，去讨价还价、去依附势力、去卖命。金手指没有让他脱离众生的逻辑。 |
