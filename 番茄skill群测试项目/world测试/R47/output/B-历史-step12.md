### 2b. 动力引擎设计（众生攀登系统）

#### 组成1：驱动逻辑（众生为什么爬）

| 字段 | 含义 |
|:--|:--|
| **驱动力类型** | **资源稀缺 + 生存压力 + 地位争夺**（三叠加重压） |
| **驱动力描述** | 九品中正制下，社会资源（土地、官职、教育、法律豁免权）被高门士族垄断。不上爬，寒门世代为吏，阶下为囚；不与高门联姻，士族将被排挤出权力圈。**不爬，就会被时代的车轮碾过，沦为他人耗材。** |
| **主角欲望对齐** | 主角沈默的核心欲望是 **“为家族挣一个士族身份”** ，这恰好是“生存压力”（家族未来）和“地位争夺”（打破门阀垄断）的**个人化表达**。他爬，是因为他看到了“不爬”的悲惨未来（被灭门）。 |

#### 组成2：运转机制（众生怎么爬的闭环）

| 字段 | 内容 |
|:--|:--|
| **输入** | 投入：**时间、金钱、学识、婚姻、忠诚、鲜血（军功）**。 |
| **转化** | 通过**科举 / 军功 / 联姻 / 依附名士**，获得**品级评定**，得到出仕资格。 |
| **输出** | 获得**官品**（权力）、**土地**（经济）、**免役权**（特权）、**人脉**（政治资本）。 |
| **反馈** | [正反馈] 官位越高，掌握资源越多，越容易获得**更高评价（上品）**，形成飞轮。 [负反馈] 官位越高，站队风险越大，敌人越强，**树大招风**。 |
| **闭环图** | **站队或立功 → 获得品级 → 升官掌权 → 资源更多 → 更高品级**。 这个过程中，寒门靠“知识”入局，士族靠“血统”开局。 |

#### 组成3：众生攀登方式分层（势力/人物的来源）

| 群体 | 攀登方式 | 对应势力层 | 例 |
|:--|:--|:--|:--|
| **没资源的人（寒门/庶民）** | **靠知识（经学造诣）**进入体制，靠**拼死立功**换取升迁。 | **底层/中层势力** | 如主角、贫苦出身的名士、边疆立功的军户。 |
| **有资源的人（低品士族/豪强）** | 靠 **家族人脉** 举荐，靠 **联姻** 扩大势力。 | **中层势力** | 地方豪强、郡望世家。 |
| **爬到顶层的人（高门士族）** | **垄断教育资源**（不让寒门读真经）、**垄断举荐权**（相互吹捧）。 | **高层/顶层势力** | 中原五大高门、京畿世家。 |
| **掉下来的人/爬不动的人** | 成为**背景板**（吏员世家）、**触底反弹者**（造反的黄巾军）、**怨气凝成的绊脚石**（被主角踩下去的对手）。 | 散人/反派 | 无能狂怒的纨绔、被排挤出权力中心的老臣。 |

#### 组成4：代价结构（众生付出什么）

| 字段 | 含义 |
|:--|:--|
| **通用代价** | **时间**（十年寒窗）、**尊严**（对上级卑躬屈膝）、**人身自由**（被差遣）。 |
| **群体差异代价** | **寒门**: 付出**尊严**和**生命**（被贬斥、被灭门风险高）。<br>**高门**：付出**家族信用**（站错队，家族覆灭）和**内部斗争**。<br>**宗王**：付出**血脉**（参与夺嫡，失败就是死）。 |
| **代价累积效应** | 越往上爬，付出的**政治代价越沉重**。到了三品以上，**站队就是生死**，不再有“隐性”选择。代价呈现指数级增长。 |

#### 组成5：范式归类

### 2e. 危机体系（三层面）

#### 终极危机（全书终卷爆发）

*   **层面**：**坐标系内压** （门阀制度走向崩解）
*   **来源**：力量体系（九品中正制）与王朝发展（人口爆炸、土地兼并、边疆战乱）的不可调和矛盾。门阀拒绝任何改革，导致社会生产力崩溃，王朝面临亡国危机。
*   **触发条件**：主角在推行“科举改制”时，遭遇**所有门阀联合+皇族猜忌**，形成“改革即死，不改也亡”的死局。
*   **预计爆发卷**：第七卷末-第八卷
*   **后果**：要么主角成功打破制度，建立新的选官体系（代价是血洗一大半高门，引起天下大乱）；要么主角失败，王朝覆灭，主角成为乱世枭雄。

#### 阶段性危机1：权力塌方（引擎代价）

*   **层面**：**引擎代价** （站队失败，被连根拔起）
*   **来源**：动力引擎的“人脉/派系”代价累积。
*   **触发条件**：主角升为郡守（第三卷末），选择效忠的皇子在“夺嫡”中失败，或自己站错队。
*   **预计爆发卷**：第三卷末-第四卷初
*   **后果**：主角的派系被清洗，官位被一撸到底，家族面临抄家灭族风险。不得不转向“压迫型”模式（引擎演化触发）。

#### 阶段性危机2：预知失效（金手指限制）

*   **层面**：**金手指限制** （蝴蝶效应+平行偏差全面爆发）
*   **来源**：金手指限制 ①、②、④。
*   **触发条件**：主角在第五卷进入京城后，频繁动用记忆改变历史，导致记忆全面崩溃。他“知道的历史”与“正在发生的历史”出现巨大偏差，甚至完全相反。
*   **预计爆发卷**：第五卷中后期
*   **后果**：主角赖以制胜的“预知”失效，沦为普通能臣。他必须在没有金手指庇护的情况下，凭真实的谋略硬刚门阀，完成从“穿越者”到“时代弄潮儿”的蜕变。

### 2f. 金手指定位（助力定位）

| 字段 | 要求 |
|:--|:--|
| **金手指类型** | **穿越知识**（前世历史记忆+过目不忘） |
| **加速机制** | • **科举碾压**：过目不忘让背书无敌，快速获得入仕资格（跨越阶位1）。<br>• **政治预判**：知道未来三十年的政治走向，可以**精准站队**，在派系斗争中极大降低跨层风险（跨越阶位2-4）。<br>• **情报优势**：知道关键人物的弱点，可以快速拉拢或击垮对手（加速跨越阶位5-6）。 |
| **限制（防止喧宾夺主）** | ①**时效性**：30年内，让主角不能无限躺赢，后期必须靠自己。<br>②**蝴蝶效应**：改变越多越不准，逼主角从“被动预言家”转向“主动操盘手”，增加了决策风险。<br>③**平行偏差**：防止金手指变成精确开挂，制造意外和困境。<br>④**使用代价**：头痛、昏厥，使用一次必须有战略级收益，不能随便用。<br>⑤**不能改大势**：可以夺一城一池之胜利，但无法阻止国本级的危机（如一场注定要输的战争），保持了“世界观”的严肃性。 |
| **代价** | 身心俱疲、信息混乱、蝴蝶效应导致不可预测的后果。 |
| **与引擎的关系** | 金手指是主角在“飞轮型”引擎下的**超级加速器**。他攀登的每一步（科考、立军功、站队）都因金手指而更高效、更安全。但攀登的**底层逻辑**（获取资源、构建人脉、应对站队风险）没有变。他没有因为能预测未来就直接当上皇帝，他仍然需要走完所有流程。 |

#### 不喧宾夺主检查

| 检查项 | 判定 | 理由 |
|:--|:--|:--|
| 金手指是否让坐标系失去意义？ | **否** | 即使预知未来，主角也受限于**官品/社会地位**，他不能无视“士族对寒门的歧视”这种坐标系规则。他需要一步步获得官身才能说话。 |
| 金手指是否取消了众生攀登的意义？ | **否** | 众生攀登的代价（家族、尊严、生命）依然沉重。金手指只是让主角**选对了路**，而不是不需要走路。所有门阀势力依然在按自己的逻辑攀登，主角只是提前知道了结局，但**过程依然凶险**。 |

→ **金手指设计通过红线检查，仅作为“助力”，未喧宾夺主。**
