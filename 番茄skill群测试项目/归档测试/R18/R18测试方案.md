# R18 AB test方案

## 测试目标

验证skill v5.0.0迭代效果——新加入的心法/代入感/避免流水账/高疲劳词/情绪节奏公式是否提升了正文质量。

## 前置条件

R17验证结论：方向2（燃料+心法）在5章连续写作中完胜。心法累积优势放大。

本轮skill迭代：
- write v5.0.0：新增5条心法 + 代入感技法 + 避免流水账 + 高疲劳词 + 情绪节奏公式
- plot v6.1.0：新增黄金三章法则参考 + 分层反派设计
- review v3.1.0：新增高疲劳词检测 + 代入感检查
- seed v8.1.0：新增核心爆点字段

## 测试设计

3方向AB test，全链路 seed→plot→write→review 到ch05。

| 方向 | seed | plot | write | review | 验证目标 |
|------|------|------|-------|--------|---------|
| A_旧skill心法 | v8.0.0 | v6.0.0 | v4.1.0+手动5条心法 | v3.0.0 | R17最优配置复现（对照组） |
| B_新skill全量 | v8.1.0 | v6.1.0 | v5.0.0 | v3.1.0 | 完整迭代效果 |
| C_新skill仅心法 | v8.1.0 | v6.1.0 | v5.0.0去掉代入感/去油/节奏 | v3.1.0 | 外部PE参考技法增量价值 |

### 方向A：旧skill心法（对照组）
- 复现R17方向2的最优配置
- write使用v4.1.0，但手动注入5条心法（与R17方向2完全一致）
- 作为baseline，衡量新skill的增量

### 方向B：新skill全量（实验组）
- 使用全部最新skill版本
- write v5.0.0包含心法+代入感+避免流水账+高疲劳词+情绪节奏
- 验证完整迭代效果

### 方向C：新skill仅心法（控制变量组）
- write v5.0.0但去掉代入感技法/避免流水账/高疲劳词/情绪节奏公式
- 只保留5条心法（与方向A的心法部分一致）
- 验证外部PE参考技法的增量价值

## 关键验证点

1. **B vs A**：新skill整体是否优于旧skill+心法？
2. **B vs C**：外部PE参考技法（代入感/避免流水账/高疲劳词/情绪节奏）是否有增量？
3. **心法累积优势是否持续**：5章连续写作中，心法效果是否与R17一致？

## 共享创意

新选一个玄幻题材种子（不复用R17的"二手灵魂当铺"）。由seed v8.1.0产出，三个方向共享同一创意+骨架。

## 执行流程

1. seed v8.1.0 产出共享创意.md
2. 3个方向各自独立执行plot→write→review到ch05
3. 方向A用旧skill（v8.0.0/v6.0.0/v4.1.0/v3.0.0）+手动心法
4. 方向B用新skill全量（v8.1.0/v6.1.0/v5.0.0/v3.1.0）
5. 方向C用新skill但write去掉外部PE参考技法（只保留心法）
6. 写到ch05后复盘对比

## 评估维度

| 维度 | 评估方法 |
|------|---------|
| 代入感 | 画面具体度/五感落地/熟悉感/情绪共鸣 |
| 流畅度 | 是否有流水账/信息空转/节奏断裂 |
| 语言质量 | 高疲劳词频率/句式多样性/对话自然度 |
| 爽感闭环 | 触发→爆发→后果三段式完整度 |
| 钩子追踪 | 埋设/回收/待回收闭环 |
| 连续性 | 5章质量是否稳定/心法累积优势 |
