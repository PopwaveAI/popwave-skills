# ch001「夜雨」

雨下了一整夜。

八岁女孩蹲在漏雨的棚屋角落里，用捡来的枯枝拨弄着火堆。铁罐里的粥咕嘟咕嘟冒着泡，米香混着柴烟飘出去，很快就被贫民窟的腐臭吞没。她舀起一勺，吹凉了，小心翼翼送到草席上躺着的人嘴边。

哥哥已经昏迷四天了。

“哥，吃饭了。”薇薇安压低声音，像怕吵醒谁似的，“你不吃，我就倒给老希斯了。”

草席上的人没动。

蹲在门口的老狗希斯竖起耳朵，喉咙里发出低沉的呜咽。它是条半瞎的杂种狗，皮毛上满是打架留下的疤，但嗅觉还在。刚才有三个醉鬼在巷口晃悠，想往这边摸，老狗亮出牙，他们骂了句“晦气”就走了。

薇薇安把粥喂进哥哥的嘴里，看他喉结动了动，松了口气。她又舀了一勺，自己吹凉了喝掉。粥很稀，米粒沉在底下，得用勺搅着才能捞起来。

她想起三天前那个卖杂货的大婶说过的话：“小丫头，你哥要是再不醒，你就去码头区找老杰克。他收小孩，管饭，还给银币哩。”

薇薇安知道那是干什么的。

她没见过自己妈——听说生她的时候难产死了。爸死在两年前的帮派火并里，尸体被扔在臭水沟，是哥哥把他背回来埋的。从那以后，哥哥就扛起了这个家。他给码头卸货，给旅店跑腿，偶尔还会去市场“顺”点东西回来。每次回来都笑嘻嘻的，说“薇薇看，哥给你带啥了”。

可四天前，哥哥回来的时候浑身是血。

薇薇安不知道发生了什么。她只记得那晚哥哥倒在门口，背上被什么利器豁开一条大口子，肉往外翻着，血把她的裙子都浸透了。她用偷来的布和烧酒帮哥哥止血，把家里最后的钱买了伤药，然后坐在草席边等。

等了一天，两天，三天。

今天第四天了。

薇薇安把最后一口粥喂进哥哥嘴里，放下铁罐，摸了摸老希斯的头。“老希斯，”她小声说，“要是哥哥还不醒，我就去找老杰克。”

老狗舔了舔她的手。

薇薇安站起来，走到门口，扒着门缝往外看。雨雾里，隐约能看到巷口的篝火和晃动的人影。那是帮派的人。自从哥哥倒下后，这条巷子就换了主人，三天两头有人打架，前天夜里还死了人。她听见惨叫声，然后重物落地的闷响。早上推开门，巷子口有一摊黑褐色的东西，被雨水冲淡了，但没完全冲掉。

薇薇安缩回屋子，把门闩上，抱着老狗缩在床脚。

窗外的雨声越来越大。铁皮屋顶漏水，滴答滴答溅在泥地上。薇薇安听着这个声音，眼皮越来越沉。她已经三天没怎么睡了。每次合眼都会做噩梦，梦见哥哥倒在血泊里，梦见自己被拖走卖给人贩子，梦见老杰克那双浑浊的眼睛。

“最多再去求求铁匠铺的叔叔，”她闭着眼睛喃喃，“他人好，上次给了我们两个铜板……”

“薇薇。”

声音很低，像从喉咙深处挤出来的。

薇薇安猛地睁开眼。

草席上，那双闭了四天的眼睛睁开了。

“哥……哥！”薇薇安扑过去，一头扎进那人的怀里，眼泪哗地流下来，“你醒了！你终于醒了！我还以为——我还以为你要丢下我了——”

她哭得上气不接下气，死死抱着不松手。

被她抱着的人愣了一下，缓缓抬起手，摸了摸她的头发。

“……没事。”

他的声音很哑，像干裂的河床。

“哥回来了。”

薇薇安哭着，没看见她哥的眼睛。

那双眼睛里没有刚刚苏醒的迷茫，没有劫后余生的庆幸。

只有一种冰冷得不像活人的、审视的目光——像在扫描，在确认，在计算。

【数据模板初始化完成。】

【宿主：索伦·影歌】

【种族：半精灵】

【职业：平民LV1 / 盗贼LV1】

【当前状态：濒死虚弱——生命值3/12，属性全项降至60%。】

【警告：当前环境极度危险，建议立即获取医疗资源。】

【主线任务已解锁：活下去。】

他闭上眼，深吸一口气。

这个世界——他来过。

在游戏里。

不。

这就是游戏。

他把怀里哭累了的薇薇安轻轻推开，撑着床板坐起来。肋骨断了两根，背上那条伤口还火烧火燎地疼。但他的眼神已经完全清醒了。

“哥，你躺着——”薇薇安急道。

“我饿了。”索伦说，“还有粥吗？”

薇薇安愣了一下，赶紧去拿铁罐。罐底还有些剩粥，已经凉了。她要去热，被索伦拦住：“凉的就行。”

他三口两口喝完粥，力气恢复了一些。生命值跳到5/12。

够了。

索伦站起来，走到门口，从门缝往外看。雨雾里，三个壮汉正靠在对面屋檐下喝酒。他们腰里别着刀，身上文着骷髅图案——这是贫民窟的地头蛇，科尔帮派的人。

门口的老希斯竖起尾巴，发出低沉的吼声。

索伦盯着那三个人，手慢慢摸向腰间。那里别着一把匕首，是他以前用来拆包裹的。刀刃钝了，但捅人够用。

【检测到威胁：目标生物等级平均LV3，当前宿主状态为濒死。】

【建议战术：潜行背刺，逐个击破。】

索伦眯起眼。

这个游戏，他记得。

每一张地图，每一个副本，每一个Boss。

包括这个世界即将迎来什么。

夜雨还在下。

巷口的醉汉骂了一声，其中一个站起来，朝这边晃过来。

老希斯龇出牙。

索伦握紧了匕首。

“哥……”薇薇安在身后小声说，“你要干什么？”

索伦回头，看了她一眼。

“哥去弄点药，”他说，“你在屋里待着，别出来。”

他推开门，走进雨里。

薇薇安站在门口，看着哥哥的背影消失在雨雾中。

她总觉得，哥哥醒过来之后，有什么东西不一样了。

但她说不上来。

老希斯舔了舔她的手。

“没事，”薇薇安抱着老狗，小声说，“哥哥回来了。”

她没看到的是——

雨雾里，索伦的右手上，一道数据面板正在缓缓浮现。

【技能「背刺」准备就绪。】

【目标距离：7米。】

他走向那三个醉汉，脚步没有声音。
