# ch003「墓园」

晨雾还没散尽，贫民窟的泥巴路被昨夜的雨泡了一晚上，踩上去又黏又滑。

索伦抱着老希斯的尸体，走在前面。

狗的身体已经冷了。皮毛上残留着雨水的潮气，四条腿僵直地垂着，尾巴拖下来，扫过泥泞的路面。

薇薇安跟在他身后，没哭，但眼睛红得像兔子。

路上的人看见他们，伸着脖子议论。

“那不是老瘸子家的娃吗？”

“抱的啥玩意？”

“狗吧？死了还抱着。”

“穷成这样，还有心思埋狗。”

索伦没搭理他们。

他脑壳里还在转昨晚杀人的事。

科尔帮还剩十几个人。要是反应快的话，今天下午就能发现那三个醉鬼的尸体。到时候无非两种可能——要么他们觉得是其他帮派动的手，要么他们会直接找到他门口来。

杀三个人从醉汉身上摸了六枚银币十几个铜板。这点钱够薇薇安吃两天肉，但不够她吃一个月。而诸神黄昏还有八个月就要来了。

时间不够。

太不够了。

他抱紧老狗的尸体，加快脚步。

墓园在贫民窟最边上。与其说是墓园，不如说是一块四面围了圈破篱笆的荒地。没人管，也没人收钱——谁想埋就往里埋，反正下面早不知道挖了多少层了。

索伦走到篱笆门口的时候，看到一个老头弯着腰在修篱笆。

那老头穿着一件打了不知道多少补丁的旧袍子，头发花白，背佝偻成了一张弓。

“埋人？”老头抬起头，看了他一眼。

“埋狗。”

老头沉默了一会儿，没说话，继续修篱笆。

索伦蹲下来，开始用手挖坑。

泥巴很松软。昨夜的雨水把土泡透了，挖起来不费劲。他一边挖一边感觉到手上沾满烂泥，指甲嵌进去，凉意往骨头缝里钻。

薇薇安也想蹲下来帮忙，被索伦用眼神制止了。

“别弄脏手。”

她只好站在旁边，看着那只老狗的尸体被一点点放进土里。

老希斯躺在坑底，眼睛闭着，尾巴蜷在身体旁边，像睡了一样。

索伦往它身上盖了第一把土。

土落在皮毛上，发出沉闷的声响。

就在这时，他脑子里那条剧情线又开始跳了。

再过几个月，诸神会从天山上坠落。

然后是魔网崩溃，魔法女神陨落。

再然后——圣者浩劫。

满地死人和怪物。神祇在凡间行走，魔鬼在阴影里猎杀平民。

他记得游戏里有个NPC说过一句话：

“诸神之战打完后，世界人口少了一半。”

一半。

那还是保守估计。

索伦往下又挖了两下，突然碰到什么硬东西。

他把土扒开，看到半截人骨头。不是很老，上面还挂着褐色的皮肉碎末。

这就是贫民窟的墓园。

你不挑位置，下面躺着谁也不知道。

他把狗往旁边挪了挪，尽量不去碰那截骨头。

“哥，”薇薇安的声音从身后传来，“老希斯……它会去哪？”

“狗不去哪。”索伦把土填回去，“死了就是死了。”

薇薇安咬着嘴唇，没吭声。

“但是我记住了，”索伦低着头，填土的动作没停，“它守了门口三天，一直到能站起来的人才闭眼。”

他拍了拍填平的土，站起来。

那枚银德勒还放在他怀里，是他偷来的其中一个。

他掏出来看了看。

一枚银币。对普通人来说不值钱。但对贫民窟的乞丐来说，够买一周的黑面包了。

索伦握着这枚银币，朝那老头走过去。

“大爷。”

老头抬起头，看着他手里的钱。

“这是白橡木河对面的价格，”老头说，不带什么感情，“贫民窟的墓园，不收费。”

“我知道。”索伦把钱递过去，“但我不是埋人，我是埋狗。要点安葬费。”

老头盯着他看了好一会儿。

“你是昨晚醒过来的那个小子吧？”

索伦没说话。

“你妹妹，”老头说，“听说这几天一直在煮粥喂你。”

“嗯。”

老头沉默了一会儿，接过钱，“那这钱我收了。狗埋在那些人的角落里，没人会动。”

索伦点了下头。

他转身走回薇薇安身边。

“走吧。”

薇薇安看着那个小土包，眼眶又红了。

“它不会再疼了吧？”

“不会了。”

索伦伸手，握住她的手。

三秒钟后，薇薇安收紧了手指。

她什么都没说，只是死死攥着他的拇指，像被风吹得没处抓的藤蔓，终于缠上了一根能拽住的杆子。

索伦低下头，看着她的手。

小小的，又脏又黑，骨节凸出来，指甲缝里嵌着泥。

他想起那条剧情线里的灾难。

想起老狗的血，地上一小滩，浅得快要干。

想起刚才那截人骨头，褐色的肉渣。

这个世界不会因为他是穿越者就对他手下留情。

但是他怀里还有那枚银德勒换来的安葬凭证。

他还有薇薇安。

她握着他的手。

他攥住她的手。

够了。

两个人踩着泥巴路，一步一步往回走。

晨雾在身后缓缓合拢，将墓园吞进光晕里。

薇薇安走在前面，踢开一块小石头，头也不回地说：“哥，回去我给你煮粥。”

“你有米吗？”

“刚才买的呀！你给我的钱！”

索伦愣了一下，然后从兜里摸出剩下的五个银币和铜板。

那一枚银德勒已经花出去了。

剩下的钱，够吃五天。

五天之后，要么挣到钱，要么饿着。

而诸神黄昏，还有八个月。

他站在贫民窟的泥巴路中央，抬头看了看天。

阳光穿过雾，光线像碎了一样砸在地上。

系统面板还在他眼前闪烁。

【任务提示：生存（短期）。剩余时间：八个月（至圣者浩劫）。建议：积累初始资源、提升职业等级、确保核心人员安全。】

八个月。

索伦踢开脚边的小石头，跟上薇薇安的脚步。

系统面板在他视线边缘缓缓隐没。

他什么都没对自己说。

但系统没告诉他——

下一个要死的，

是谁。
