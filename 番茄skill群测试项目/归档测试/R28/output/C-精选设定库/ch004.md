# ch004「一切都会好起来的」

索伦带着薇薇安回到贫民区时，太阳已经升起来了。

泥巴路上的人多了起来。小贩蹲在路口卖发霉的面包，几个光着脚的小孩蹲在地上弹石子，破布帐篷门口坐着眼神空洞的女人。

薇薇安走在他前面，攥着他给的钱。

一路上她不说话，但脚步明显比昨天轻。

她跑到一个卖黑面包的摊子前，从兜里摸出三个铜板。

“要这个。”

面包硬的能砸死人。薇薇安把它装进衣服里，拍了两下，像揣着什么宝贝。

两人钻进一间漏雨的棚屋。说是屋，其实就是几块破木板搭起来的窝棚，顶上盖着发黑的稻草。地面是夯实的泥巴，中间挖了个坑当炉灶。

薇薇安蹲下来生火。她动作麻利，一看就是干惯了活的人。火柴划了两下就着，枯枝丢进去，火苗噼里啪啦响起来。

她把黑面包掰碎，丢进一个小陶罐里，加水煮。

索伦靠在墙边看着。

他脑子里还在转昨晚的剧情线。

八个月。

圣者浩劫。

他得在这个时间里让薇薇安有自保能力，还得搞定金手指激活。

但眼前最紧要的，是那三个帮派。

科尔帮。

他杀了他们三个人，尸体还在那条巷子里。天黑之前肯定会被发现。

到时候不会有人追查死因——贫民窟死几个人跟死几只老鼠没区别——但他们一定会找到他头上。

因为这片区域的人都知道，他妹妹之前被打过。

索伦低头看了看自己的手。

一级盗贼，敏捷19。数据上够用。

但实战是另一回事。

“哥。”

薇薇安的声音把他拉回来。

她端着陶罐走过来，里面是泡软的面包糊。热气腾腾的，闻起来有麦子味。

“你喝。”

索伦接过罐子。

薇薇安蹲在他面前，双手撑着脸看他。

那表情像一只等投食的小猫，眼睛亮晶晶的。

索伦喝了一口。

热的。麦子味很淡，但很暖。他的胃蠕动了一下，缓慢接住了这份温度。

“哥，好吃吗？”

“还行。”

“嘿嘿。”

薇薇安笑了。

她一笑，眼睛弯起来，露出缺了一颗的门牙。

索伦又喝了一口。

之后把罐子递给她。

“你也吃。”

“你先吃，我不饿。”

“吃了。”

他语气没变，但薇薇安已经听话地接过去了。

她捧着陶罐，小口小口地喝着，像一只咕噜咕噜的猫。

喝着喝着，她突然抬头说：“哥，以后都会好起来的，对吧？”

索伦顿了一下。

他看着她眼睛里的火光，说：“会。”

“你保证？”

“我保证。”

薇薇安没再追问，低下头继续喝粥。

老狗昨天死了。

但这碗粥是热的。

她把碗底舔干净了。

等薇薇安睡着了，索伦才站起来。

他从床底暗格里抽出那把匕首。

长十六公分，铁质的，刀刃上锈斑点点。唯一的优点是——开过刃。

他试了一下，在手指上划了一道浅浅的口子。

血珠子渗出来。

疼。真实的那种。

他把伤口舔干净。

脑子里已经在模拟了。

火炉酒馆。北墙外那条小巷。踩进去两步半，左手边有个暗处，刚好够一个人蹲伏藏身。从那个位置能看到酒馆后门，十五步的距离，土路上有石子。

巴拉斯每次喝完酒会从后门出来，先靠墙吐一会儿，然后沿巷子往东走。

卡诺波跟他一起，但比他慢一步。

前后差三米。

先杀巴拉斯，然后转身杀卡诺波。

三分钟。

索伦把最后的干粮塞进嘴里，咬碎，吞下去。

他把匕首插进腰带里。

外面天已经黑了。

火炉酒馆在贫民窟的东边。

说是酒馆，其实就是一间大一点的棚子，门口挂着一块破布当招牌。

里面闹哄哄的，传出一股劣质麦酒的气味，混着汗臭和烟味。

索伦蹲在巷口的暗影里。

他没有从正面靠近。小巷两侧是板墙，到处是漏缝，他挑没有光的地方走，影子落在碎砖和烂泥之间，脚步轻得像猫。

后门吱呀一声开了。

一个醉醺醺的身影跌出来。

巴拉斯。

他靠在墙上，弯着腰干呕了两声。嘴里含糊不清地骂着什么，大概是嫌酒不好，或者嫌钱太少。

索伦没有动。他在等卡诺波。

但卡诺波没出来。

巴拉斯一个人。

那就先杀一个。

索伦贴着墙摸过去。

【阴影躲藏判定: 成功】

【背刺发动: 自动确认暴击】

匕首从巴拉斯的腰椎往上扎进去——左肾，斜刺，角度刚好够刀尖探到肺。

巴拉斯身体猛地一僵。他想喊，但肺已经漏气了，只发出一声漏风的气音。血从他嘴里涌出来。

索伦一脚踹到他膝盖弯上，让他跪倒。

【发动背刺*1，目标受重伤】

【目标士气判定: 失败】

巴拉斯挣扎着想掏武器。但匕首在体内转了一下，他的力气就跟着肺里的风一起漏光了。

他倒下去，后脑勺砸在泥地上，发出一声闷响。

索伦没有停。

匕首拔出来，再扎进去。一直到他脑子里跳出一行字：

【目标死亡！】

【抽取目标灵魂能量，获得杀戮经验15点】

巷子里安静下来。

可就在这时，后门砰一声被踢开了。

卡诺波冲了出来。

他手里提着半截酒瓶，赤裸的上身全是刺青，右眼上有一块刀疤。

他看见巴拉斯倒在血里，又看见索伦手里滴血的匕首。

“操——”

他扑上来。

索伦没退。

两个人之间的距离不到两步半。他往前一踏，匕首从一个诡异的角度捅向卡诺波的脖子。

卡诺波侧身躲开了第一刀，酒瓶砸过来。玻璃碴擦过索伦的额头，带出一条血线。他眼前一红，但没有停——第二刀从腋下递了进去。

刀尖扎进心脏。

【发动背刺*2】

【暴击确认！】

【目标死亡！】

【抽取目标灵魂能量，获得杀戮经验15点】

卡诺波的眼神在五秒内就从凶狠到茫然再到空白。

他张了张嘴，什么都没说出来，就顺着一堵墙滑下去了。

索伦站在两具尸体中间喘了口气。

然后蹲下身开始搜刮。

两个尸体加起来摸了六枚银币、十几个铜板，还有一把半开的折叠刀。

他给折叠刀别进裤腰，又翻了翻衣兜，确认没有遗漏。

然后拖起两具尸体丢进巷子深处那堆垃圾底下。

他低头看了一眼自己。

脚上没有溅到血，但衣服前襟染了一大片。现在是晚上，贫民窟的路灯几乎没有，但明天天亮之前，这身衣服不能穿了。

索伦转身往棚屋的方向走。

路上没遇到任何人。

夜风冷，月亮被云遮了一半。他踩着影子走在贫民窟的窄巷里，匕首上的血已经干了。

回到棚屋，他在水桶边蹲下洗了洗手。

冷水。沁骨的凉。

他甩了甩手上的水珠，听见自己的声音在脑子里响起来。

【累计获得杀戮经验: 30点】

【当前杀戮经验: 30点】

三十点。够升一级。

但他没点升级。

他先走到床前。

薇薇安还在睡。

她缩成一小团，被子拉到下巴，呼吸均匀。

索伦站在床边看了一会儿。

然后他在暗格旁边坐了下来。

匕首搁在膝盖上，折叠刀放在触手可及的地方。

他没点灯。

就这么在黑暗里坐着。

过了几分钟，他轻轻说了一句——

“一切都会好起来的。”

风从板墙的缝隙里灌进来，吹动了他额前染血的碎发。

棚屋外面，黑黢黢的贫民窟开始沉入更深也更冷的夜。

但这间屋子的火坑灰烬里，还留着一丝余温。
