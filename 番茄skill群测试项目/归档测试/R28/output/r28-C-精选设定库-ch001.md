# ch001「夜雨（上）」

雨砸在屋顶的破木板上，像是有人在头顶砸碎了水缸。

薇薇安缩在灶台边，用湿透的木柴搭了个小小的三角堆。火柴划了三根才着，火苗舔着柴皮发出噼啪声，细烟在潮湿的空气里飘了不到三尺就被雨打散了。

锅里剩了半碗冷粥。

她小心地把粥倒进小陶罐，架在火上。火苗舔着罐底，粥面慢慢冒起细泡。

薇薇安回头看了一眼床上的哥哥。

索伦躺在草席上，脸色白得像教堂里的蜡烛。他已经昏迷三天了，额头烫得吓人。昨天她拿仅剩的两枚铜币找街口的巫婆买了退烧草药，可他现在还是没醒。

“哥，粥好了。”

她端着小陶罐爬到床沿边。哥哥的嘴唇干裂起皮，她用小木勺舀了一勺，吹凉了，小心翼翼地送到他嘴边。

粥顺着嘴角流下来。

她又试了一次，还是没喂进去。

外面传来老狗希斯的低吼声。薇薇安立刻竖起耳朵，手不自觉地握紧了木勺。

“汪——”

希斯的声音低沉、粗哑，像是从嗓子眼里挤出来的威胁。紧接着是人的脚步声，踩在泥水里啪嗒啪嗒响。

“老不死的畜生，再叫老子炖了你！”

男人的声音从门外传来。薇薇安听出那是这条巷子收保护费的铁头——他昨天就来过，说要是哥哥死了，就把她卖到妓院抵债。

希斯没有退。老狗佝偻的身子挡在门口，嘴角的獠牙露出来。

脚步声远了。

希斯才慢慢趴下来，浑浊的眼睛还盯着门缝的方向。

薇薇安松了口气，回头看着哥哥的脸。

她想起母亲临死前拉着她的手说的话——“照顾好哥哥。”那是去年冬天的事，母亲得了肺痨，没钱治，咳了半个月血就死了。父亲？她没见过父亲，母亲说他是冒险者，死在某个地下城了。

贫民窟的孩子活不过十二岁，这是琥珀城穷人都知道的事。

巷口那个卖面包的老奶奶上个月死了，尸体被收尸人拖走。隔壁家的小男孩得了痢疾，三天就没了。母亲撑了半个月，已经算是命硬的。

薇薇安伸手摸了摸哥哥的额头。

还很烫。

她把沾湿的布巾叠好，放在他额头上。哥哥的睫毛动了动，眼睛却还是紧紧闭着。

“哥，你要是再不醒……”薇薇安咬着嘴唇，“我……我就把自己卖给那个驼背人。”

驼背人三天前来过这条巷子，专门收小孩。他给的价格不高，一个孩子十枚银币，但他说他有药，能救垂死的人。

薇薇安当时把他赶走了。

但她知道，如果哥哥再不醒，她就只剩这个选择了。

外面突然响起喊杀声。

“杀！杀了那个混账！”

“操他妈的，给我上！”

金属碰撞声、惨叫声、怒骂声混在一起，像有什么东西砸在了墙上。薇薇安吓得缩到床脚，抱住了希斯。

老狗伏在她身边，尾巴夹得紧紧的。

火并声持续了约莫一炷香的功夫，慢慢平息了。

雨还在下。

薇薇安不敢出门去看。她只是抱着希斯，把脸埋进它粗糙的皮毛里。

不知过了多久，她感觉手心有什么东西动了动。

她猛地抬头。

哥哥的手指勾住了她的衣角。

“哥？！”薇薇安扑到床沿边。

索伦的眼睛慢慢睁开了。

那双眼睛不像昏迷的人——它们很清醒，像是刚从一场大梦里醒过来，瞳孔里映着跳跃的炉火。他微微张了张嘴，喉咙里发出一个低沉的声音：“……薇薇安。”

“哥！你醒了！你终于醒了！”

薇薇安扑进他怀里大哭起来。

索伦的手慢慢抬起来，轻轻拍了拍她的后背。他的目光越过妹妹的肩膀，盯着屋顶漏雨的缝隙，像是在看什么东西。

他的眼睛里有什么东西闪过——
