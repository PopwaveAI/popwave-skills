# ch002「夜雨（下）」

索伦睁开眼的第一个念头不是疼，是杀。

锈匕首还在墙角。窗外打斗声渐歇。薇薇安趴在他胸口睡着，眼角泪痕未干，小手死死攥着他的衣角，像怕他再死一回。老狗希斯趴在门口，浑浊的眼睛半闭，呼吸很浅很浅。

索伦没有吵醒妹妹。他闭眼，脑中数据流重新浮现。

【职业：平民LV1 / 盗贼LV1】

【未分配经验：150点】

他手指轻颤。150点——上一世这是新手村杀三只野猪的经验。但这个世界没有野猪，只有贫民窟里觊觎他妹妹的恶徒。

分配。全部。

【消耗150点经验——平民LV1→LV10（满级）】

【体力槽+20%】【生命值上限+10%】【基础负重+50%】

【解锁被动：平民坚韧——轻度伤势恢复速度+30%】

一股暖流从胸口扩散开来。四天滴水未进的虚弱感像被一只手从体内抽走。他握了握拳，指节咔咔作响。肌肉还是那些肌肉，但某种更深层的东西已经被激活。

“还不够。”他低声道。

【提示：平民职业满级，可进阶——战士/游荡者/猎人】

【推荐：以当前属性（敏捷19）优先解锁游荡者系进阶】

索伦动了动脖子，骨骼发出轻微爆响。窗外的天已经蒙蒙亮了，雨声渐止。铁皮屋顶滴落的水珠慢下来，从急促的雨线变成有节奏的鼓点。

薇薇安梦里不知道梦见了什么，眉头皱紧，加了句“哥哥别走”。

索伦摸了摸她的头发。

老狗希斯突然抬起头。它浑浊的眼睛看向门外，喉咙里发出一声低沉的呜咽——不是警告，是某种更放松的东西。好像确认了这具身体里的灵魂已经回来，它缓缓把脑袋搁回前爪上。

那口气松得太彻底了。

索伦察觉到什么，猛地转头。老狗希斯的眼睛还睁着，但那层浑浊已经凝固了。它的胸腔不再起伏，尾巴最后动了动，像在回应什么只有它听到的呼唤。

“希斯。”索伦轻喊了一声。

狗没动。

薇薇安在睡梦中蹭了蹭他的胸口，嘟囔了一句“别吵”。她不知道。索伦看着那条守护了四天四夜的老狗，胸口某个地方被轻轻刺了一下。

系统没有弹出击杀提示。这是自然死亡，不是战斗。

【侦测到死亡生物……确认：老狗希斯】

【心灵羁绊已记录】

索伦沉默了很久。

天彻底亮了。贫民窟的早晨和夜晚一样脏、一样乱、一样嘈杂。远处传来铁匠铺的开炉声，小孩哭闹声，女人骂街声。巷口夜里的尸体已经被拖走了，只剩下一块暗褐色的血迹，在晨光里格外扎眼。

“哥哥？”薇薇安醒了。

她揉着眼睛坐起来，看到索伦已经站在门口，背影逆光。她刚要说话，目光越过他看到了门口的老狗——希斯保持趴着的姿势，但那双一直看着她的眼睛，再也不会动了。

“希斯……？”她的声音很小。

索伦转头：“它走了。今天早上。”

薇薇安的眼泪无声地淌下来。她没有嚎哭，只是走过去，蹲下，把脸埋进老狗灰白的毛里。肩膀轻轻抖动。

索伦没有安慰她。他蹲下来，用一块破布把老狗裹好。

“我去埋掉它。”

“我也去。”

“不行。”索伦的语气没有商量的余地，“你待在家里。锁好门。”

“可是——”

“听话。”

薇薇安咬着嘴唇看他，眼睛红红的，但终究没再说话。她看着索伦抱起裹好的老狗，看着哥哥的背影消失在巷口的晨光里，然后默默把门关上，上好门闩。

她蹲在灶台边，盯着昨晚剩下的锅底发呆。灶台里的火星已经灭了，她小心翼翼地吹了几下，扔进去一小截木柴，想让火重新烧起来。

哥哥回来了。她不用去西街找老鲍勃了。但希斯死了。

她的眼泪又掉了下来。

另一边。

索伦抱着老狗穿过三条街，来到贫民窟边缘的一小块空地。这里长满杂草，散落着破陶罐和生锈的铁片。他捡了块尖石头，开始挖坑。

泥土是湿润的，昨夜的大雨让地面松软了许多。尖石头插入泥土，撬开碎石，一下又一下。索伦的手很快磨出了水泡，但他没有停。坑挖到半米深时，他的脑海里突然闪过一个念头——

【提示：发现可回收场景——平民墓园】

【墓园在当前世界中属于盗贼公会/帮派管控范畴】

【挖掘行为已引起附近巡逻人员的注意】

索伦的动作顿了一下。

他微微侧头，眼角余光扫到五十米外巷口有个瘦长的人影——穿灰色旧风衣，戴着兜帽，正靠在墙边抽烟。对方没有刻意躲藏，也没有靠近，只是站在那里，像一个偶然路过的流浪汉。

但索伦知道，这个时间点出现在这片空地的人，不会是偶然。

他没有抬头，继续挖坑。坑挖好后，他把老狗裹好的尸体轻轻放进去，再一捧一捧地把泥土填回去。没有墓碑，没有祭品，他只是在坟头压了块平整的石头。

然后站起来，拍了拍手上的土。

转身，离开。

那灰衣人还在墙边抽烟，烟头的火星在晨光里明明灭灭。索伦从他身边走过时，视线没有交汇，脚步没有停顿。但他能感觉到，那道目光一直跟着他，直到他拐进另一条巷子。

回木屋的路上，索伦的手无意识地摸了摸腰侧。匕首还在——那把锈钝的匕首，是整间木屋里唯一的武器。他沿着墙根走，脑子里在快速计算。

【当前战力评估：】

【职业：平民LV10 / 盗贼LV1——总等级11】

【状态：轻度虚弱（已解除）】

【装备：锈钝匕首（攻击力-3）】

【待解决问题：食物、武器、金钱、帮派威胁】

他停下脚步。

巷子前面三米处的墙根下，蹲着两个半大孩子。一个胖子，一个瘦子，穿得比他好不了多少，但腰里都别着匕首——铁制的，没锈的。

胖子正把一个七八岁男孩按在地上搜身。男孩哭着说“我真的没钱了”，胖子一拳砸在他后脑上，瘦子在旁边笑。

索伦在巷口站了两秒。

他认得那个胖子——科尔的手下。科尔是这条街的地头蛇，专收“保护费”，其实就是每隔几天来刮一次。而上辈子记忆里，三个月后，科尔会因为投靠魔鬼教会被城卫军吊死在中央广场。

但现在，科尔还是这条街的王。

而那孩子脑后被砸了一拳，已经不动了。

索伦没有管那孩子——不是不想，是他知道这个时间点出手的下场。科尔的人不止两个，巷子外还有望风的，只要他动了手，三天内就会有人来敲他的门。

但那个胖子站起来时，恰好看到了他。

“哟——这不是死了四天的那个杂种吗？”胖子咧嘴笑，露出黄牙，“你妹妹呢？鲍勃那边开价了，五枚银币一个小丫头，细皮嫩肉的——”

索伦看着他。

目光很平静。像看一个死人。

胖子被那目光看得心里一毛，但嘴上不饶人：“咋了？你他妈还想动手？”

索伦没说话。他转身，走进另一条巷子。

胖子在背后骂骂咧咧了几句，但没追上来。身后传来瘦子的声音：“别管他，那杂种早晚要死——”

索伦没有回头。他走回木屋门口，推开门，薇薇安正蹲在灶台前，锅里的粥重新冒了热气。看到他回来，她眼睛亮了一下，随即又暗下去。

“埋好了？”

“嗯。”

“希斯……能去天堂吗？”

索伦沉默了一会儿：“能。”

那顿饭，厨房里只有她给他盛了碗粥，然后自己坐在一旁。索伦把粥喝完，放下碗：“今晚我要出去一下。”

“去哪里？”

“找钱。”

薇薇安想问“去偷吗”，但话到嘴边咽了回去。她只是点点头：“我锁好门。”

索伦看着她，忽然伸手拍了拍她的头。然后站起来，走向墙角那把锈钝的匕首。

月光从破屋顶漏下来，照在刀刃上。

索伦握住刀柄。

这一次，他知道了自己该做什么。

【任务更新：24小时内清除贫民区帮派威胁】

【目标：击杀帮派成员≥3人】

【奖励：经验值+90 / 解锁盗贼进阶任务线】

索伦把匕首插进腰间的破布带里。

“哥哥。”薇薇安在身后唤道。

他回头。

“你要回来吃饭。”女孩认真地说。

索伦嘴角动了动——算是笑了一下。

“好。”

门在他身后关上。屋外，月亮刚从云层后露出半个脸——残月，像一把生锈的匕首。

巷口那个灰衣人已经不见了。
