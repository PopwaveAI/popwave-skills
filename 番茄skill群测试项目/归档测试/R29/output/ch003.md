# ch003「墓园区」

索伦推开门时，晨光正从破屋顶的缝隙漏进来。

薇薇安坐在灶台边，锅里的粥冒着热气。她看到他回来，眼睛先是一亮，然后注意到他裤腿上沾的泥土。

“埋好了？”

“嗯。”

她没再问。默默盛了一碗粥递过来。索伦接过，两口喝完。粥很稀，米粒少得可怜，但至少是热的。薇薇安自己也盛了一碗，小口小口地喝。

锅底还剩一层。

“你不吃？”薇薇安问。

“你吃。”

她没有推让，把那层粥底刮进自己碗里。索伦看着她——八岁的女孩，手腕细得像两根枯枝。前世记忆里，三个月后她会因为营养不良发高烧，而那时他已经死在巴拉斯刀下。

但那不一样了。

墙角的锈匕首还在。他昨晚没用上——不是因为不想，是因为出巷口时，灰衣人还在那棵枯树下抽烟。烟头的火星在暗处一亮一灭，像一个警告。

索伦退了回来。

不是怕，是不能打没准备的仗。两把锈匕首对二十把铁匕首，他就算有职业等级也扛不住。他需要装备，需要信息，还需要一个没人盯梢的时间窗口。

但今天，他有更重要的事。

“走。”他站起来。

“去哪？”

“墓园。”

薇薇安愣了一下，然后默默地站起来，牵住他的手。

索伦抱起裹好的老狗尸体，用一块洗得发白的旧布包着。尸体已经硬了，重量比活着的时候轻得多。他一只胳膊夹着，另一只手牵着薇薇安，推开木门。

贫民窟的早晨一如既往地脏乱。

污水横流的泥泞街道，穿着破衣的小孩在垃圾堆里翻找，酒鬼栽倒在巷口，鼾声如雷。几个女人坐在门口择菜，看到索伦抱着个布包走过，目光黏在他身上。

“那不是南街那个死过一回的小子吗？”

“怀里抱的啥？”

“不会是死人吧？”

议论声不高不低，刚好让他听见。索伦面无表情地走过去，像没听见。薇薇安低着头，紧紧攥着他的衣角。

路过一个铁匠铺时，一个光头汉子正在砸铁。看到索伦，他放下锤子，擦了把汗：“小子，你怀里是啥？”

“狗。”

“死了？”

“……嗯。”

光头沉默了两秒：“埋南边荒地那片？”

“去墓园。”

“墓园？”光头皱眉，“你哪有钱？墓园那老头要一个银币才肯埋，你拿得出？”

索伦没回答。

光头看了他两眼，叹了口气：“算了，你走吧。”

索伦继续走。身后传来光头嘀咕的声音：“这世道，埋条狗都比埋人贵。”

穿过三条泥泞的街，墓园的铁门出现在视线尽头。

说是墓园，其实只是一圈垮了半边的石头围墙，里面散落着歪歪扭扭的墓碑。杂草丛生，几只乌鸦蹲在墙头，黑豆似的眼睛盯着来人。

铁门旁边有个小木屋，门半掩着。一个白发老头坐在门槛上晒太阳，手里捧着个陶碗，碗里是黑乎乎的东西——大概是麦粥。

索伦走近，站定。

老头抬眼看他，目光落在他怀里的布包上：“人？”

“狗。”

“狗？”老头放下碗，“狗来什么墓园？后头荒地刨个坑不就埋了？”

索伦沉默了两秒，手探进怀里，摸出一枚银德勒。硬币在晨光里闪着暗淡的光——这是他前天晚上从巴拉斯尸体旁顺来的，沾着干涸的血迹。

他递过去。

老头看着那枚银币，眼神变了。他没有立刻接，而是盯着索伦看了好一会儿。

“你偷的？”

“捡的。”

“捡的？”老头哼了一声，“贫民窟能捡到银币？你当我老糊涂？”

索伦没辩解，只是举着那枚银币。手很稳。

老头沉默。乌鸦在墙头嘎嘎叫了两声。远处的铁匠铺又响起打铁声，叮叮当当。

“行吧。”老头接过银币，用指甲弹了一下，放在耳边听了听，“分量够。埋哪儿？”

“角落里。”

“还有要求？”老头抬头看了他一眼，但没多话，站起来推开铁门，“进来吧。”

墓园不大，转一圈都用不了两分钟。老头带着他走到围墙最角落的位置——一棵歪脖子的老槐树下，树根拱起一块地皮，草都长不旺。

“就这吧。树荫好，凉快。”

索伦蹲下，把裹着老狗的布包放在地上。他没有让老头帮忙，自己找了块尖石，开始挖坑。

泥土比他想象中硬。昨夜的大雨只浇透了表面，下面一层还是硬的。尖石撬开草皮，刨开土块，碎石硌得手指生疼。水泡破了，渗出血丝。

索伦没停。

薇薇安蹲在旁边，想帮忙又不知道做什么，小手攥着裙角，眼泪在眼眶里打转。她咬着嘴唇，没让眼泪掉下来。

索伦挖了十几分钟。坑不大，刚好够放下老狗。他把布包小心地放进去，一捧一捧地把土填上。最后一捧土压实，他找了块平整的石头压在坟头。

没有墓碑。没有祭品。

只有那棵歪脖子槐树，和树荫下的一小片新土。

索伦站起来，膝盖上沾满泥土。他看着那堆土包，什么也没说。

老头站在几步外，叼着烟斗：“完了？”

“嗯。”

“那就走吧。”老头吐了口烟，“待久了被人看见，还以为我这儿真埋了条狗——传出去谁还来买墓？”

索伦没有立刻走。他站在槐树下，背对着薇薇安，沉默了很久。

然后他开口了，声音很轻：

“这世界要完了。”

老头抽烟的动作顿了一下：“你说啥？”

“诸神黄昏。圣者浩劫。神灵陨落，恶魔降临，亡灵漫山遍野地跑。”索伦的声音像在念一份报告，不带感情，一字一句，“还有三个月。第一批神灵开始坠落。然后是魔法女神的魔网崩溃，法术失效，死者从坟里爬出来。”

老头握着烟斗的手僵住了。

烟灰掉在地上。

他盯着索伦，眼神从疑惑变成警惕，再变成某种说不清道不明的东西。好一会儿，他才哑着嗓子开口：

“……你到底是什么人？”

索伦没有回答。

他转过身，牵起薇薇安的手：“走。”

薇薇安被他拉着，一边走一边回头。她看到老头的背影站在槐树下，烟斗的烟雾在晨光里飘散。老槐树的叶子哗哗响，像在说什么。

老头最后说了一句，声音被风吹得断断续续：

“小子……别死太早。”

索伦没有回头。

他们走出墓园，铁门在身后吱呀关上。薇薇安低着头走了一会儿，忽然问：

“哥哥，刚才你说的是真的吗？”

“……嗯。”

“那我们会死吗？”

索伦停下脚步。

他低头看着薇薇安，她仰着脸，眼睛红红的，但没有哭。八岁的女孩，瘦得像根豆芽菜，但那双眼睛里有种不属于这个年龄的倔强。

他不会让她死。

“不会。”他说得很轻，但很稳，“我们不会死。”

他蹲下来，拉起薇薇安的手。她的小手冰凉，指节突出，像一把细骨头上包了一层皮。这双手不该承受这个世界的重量。但他知道，在这个世界，没有人能替他承受。

“哥哥。”薇薇安轻声说，“希斯会去天堂吗？”

索伦看着她的眼睛，沉默了一会儿。

“会。”

“你怎么知道？”

“因为它是条好狗。”他站起来，“好人上天堂，好狗也是。”

薇薇安没笑，但她用力点了点头。

墓园的铁门在身后关上。街巷还是那些街巷，泥泞、肮脏、臭气熏天。但索伦的步子变了，他牵着妹妹走在街道的阴影里，阳光越来越亮，把影子拉得越来越短。

时间不多了。

三个月后神灵开始坠落，诸神黄昏的第一批信号会从天空撕裂。他必须在风暴来临之前，让自己和妹妹站到一个足够高的地方。

不是高处不胜寒的高。

是没人能伸手够到的高。

索伦看了一眼天空——碧蓝如洗，一只乌鸦飞过，拖出一道黑色的影子。

它落在墓园的槐树上，嘎嘎叫了两声。

然后安静了。
