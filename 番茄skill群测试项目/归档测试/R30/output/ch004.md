# 第四章：磨坊

凌晨的风裹着腐烂的麦秸味，从废弃磨坊的裂缝里钻出来。

埃德温蹲在五十米外的土坡后面，盯着那座摇摇欲坠的三层木楼。月光透过破洞照进去，能看到一楼地面散落着碎木板和干涸的血迹。

蒂奇蹲在他旁边，正往一根铁管里塞火药和铁砂。

“情报没问题，”蒂奇压低声音，“三天前有人看到三只地精进了磨坊，后来又有商人说里面有绿皮在晒肉干——最少四只。”

“你说三只。”埃德温没转头。

“我说的是‘最少’。地精这玩意儿繁殖快，要是有一只是母的，可能已经多了一窝崽子。”蒂奇把铁管口封死，塞进背包里，“不过崽子的经验值估计不够你塞牙缝。”

埃德温没接话。他盯着磨坊二楼的窗口，那里有一团模糊的暖色光影——是心跳的热量。左眼的胀感又来了，但比前天轻了些。

【侦测目标：地精（LV1）】
【数量：1】
【要害：左颈暴露】

面板上浮出一行字，又快速消失。

蒂奇没看到。他正忙着检查腰间那些瓶瓶罐罐。

“你怎么知道地精凌晨不睡觉？”蒂奇问。

“它们的作息和人不一样。”埃德温站起来，把短刀别在腰后，“我从正面进去。你封锁后门和二楼窗口，用陷阱断退路。”

“喂喂，正面是你负责？我一个人包三面？”

“你负责的是‘封锁’，不是‘杀’。”埃德温看了他一眼，“你只要不让它们逃出去就行。”

蒂奇嘴角抽了抽，但还是点了点头。他从背包里掏出两枚圆滚滚的金属球，塞进手里的铁管：“行。那我就在外面等你——你被砍出来的时候记得喊救命，我好收尸。”

埃德温没理他，猫着腰摸向磨坊的正门。

磨坊的木门已经腐朽了大半，门轴锈得发出吱呀声。他没有推门，而是侧身从门板上的破洞里钻进去——

一楼的空气中弥漫着浓烈的腥臭味，像是腐烂的肉和发霉的麦子混在一起发酵。地面上散落着碎裂的陶罐、生锈的农具和黑褐色的血迹。

暖色光影——两只，在东侧的墙角处。

【侦测目标：地精（LV1）】
【数量：2】
【警戒状态：睡眠中】

埃德温屏住呼吸，贴着墙壁移动。他的脚踩在碎木上，发出细微的咔嚓声——但那两只睡梦中的地精只是翻了个身，没有醒来。

他走到距离它们三米的位置，蹲下来，握着短刀的手没有颤抖。

第一只——左颈暴露。

短刀在空中划出一条干净的弧线，精准地从左颈刺入，贯穿咽喉。

【击杀：地精（LV1）】
【经验值+15】

第二只被铁链拖着爬行的声音惊醒了——它唰地睁开眼睛，迎上的是埃德温已经递到咽喉前的刀锋。

“噗嗤——”

【击杀：地精（LV1）】
【经验值+15】

干净利落。两只地精甚至没发出一声完整的惨叫。

埃德温甩掉短刀上的血，抬头看向通往二楼的木梯。木梯的木板已经腐朽大半，踩上去可能会断裂——但他不需要走楼梯。

【侦测目标：地精萨满（LV2）】
【数量：1】
【位置：二楼中央区域】
【警戒状态：被惊动】

埃德温心头一紧——刚才那两只地精被击杀的瞬间，萨满已经感知到了。

木梯上传来急促的脚步声和叽叽喳喳的嘶吼——萨满正在往下冲。

埃德温没有后退。他退后两步，贴着墙根，隐入阴影中。短刀横在身前，左眼的金色流光一闪而过。

脚步声越来越近——

萨满冲下来的那一刻，埃德温找到了那个机会——

萨满的身体暴露在破洞透进来的月光下——这是一只比其他地精高半个头的绿色怪物，皮肤上浮现着暗红色的符文纹路，手里攥着一根镶着黑色晶体的木杖。它的瞳孔在黑暗中发出绿光，搜寻着闯入者的踪迹。

它的视线扫过墙角——埃德温藏身的阴影正对着它的左侧盲区。

埃德温没有犹豫，从阴影中爆发出来，短刀直刺萨满左颈——

【侦测弱点已激活】
【要害标记：左颈（暴露）+ 左锁骨下方黑色晶体（关键）】

萨满的反应比普通地精快了不止一倍——它侧身一闪，木杖横扫过来，砸在埃德温的肩膀上。一股酸麻感从肩膀蔓延到半边身体——那不是单纯的物理伤害。

【受到攻击：深渊能量冲击（微弱）】
【生命值-15（剩余42/57）】
【杂念+1】

埃德温咬紧牙关，不退反进。短刀在萨满挥杖的空隙刺向它的左颈——但萨满那只握着木杖的手突然抓上来，一把攥住了刀刃。

血从它的指缝里渗出来，但它没有松手。

萨满的嘴角咧开，露出一口黄色的尖牙，喉咙里发出嘶哑的笑声：“人类...你的灵魂...都是深渊的味道...你是被选中的...”

埃德温的瞳孔骤然收缩。

这句话——和海伦娜的警示一样。

但他没有停下思考。他左手拔出腰后的第二把匕首——那是附魔匕首——从下往上，刺进萨满的下巴，贯穿上颚。

【击杀：地精萨满（LV2）】
【经验值+60】
【掉落：深渊木杖（未鉴定）+ 黑色晶体】

萨满的尸体抽搐了两下，倒在地上。

埃德温拔回短刀，喘着粗气。肩膀上的伤口还在渗血，但他没时间处理。他蹲下来，捡起那根木杖——触手冰凉，木杖上的黑色符文和之前那把附魔匕首上的一模一样。

他抬头看了一眼二楼的阴影——还有几只地精崽子，但已经构不成威胁。

他转身走向门口。

——门外传来脚步声。

蒂奇的声音从墙根传来：“打完了？我刚才好像听到里面在说话？”

埃德温没回答。他把那把附魔匕首和木杖塞进怀里，走出磨坊。

月光洒在他身上，他左眼的金色又亮了一瞬，但很快就熄灭了。

蒂奇从墙角探出头来：“你肩膀在流血。”

“不碍事。”埃德温拉紧了胸口的衣领，“萨满掉了点东西，回去能换钱。”

蒂奇吹了声口哨：“好活儿。明天去公会交任务，够你买一星期的干粮了。”

埃德温没说话。他侧过头，目光落在远处的屋顶上——那里隐约有一个模糊的黑影，一闪而逝。

他的左眼眶胀得更厉害了。

但那不是萨满的深渊冲击留下的后遗症。

是提醒——

有人在看着他们。
