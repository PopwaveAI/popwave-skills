# 深渊主宰 · 完整立项包

## 1. 一句话故事

底层半精灵穿越者，凭借杀戮升级的DND数据面板，在深渊入侵的混乱世界中不断变强，只为守护妹妹不被命运吞噬，最终直面世界背后的真实抉择。

## 2. 草纲

- **主角**：半精灵穿越者 埃德温（前世中文名 林辰），贫民窟孤儿，妹妹 莉莉 是他唯一的亲人
- **想做什么**：保护妹妹，让她过上好日子；同时在这个深渊侵染的世界活下去，寻找回归地球的可能
- **为什么**：前世父母早逝，妹妹因病离世，这一世穿越后再次获得妹妹，决不允许悲剧重演；同时发现自己被“杀戮升级”系统绑定，不断杀戮会逐渐失去人性，必须找到金手指的真相
- **如何做**：利用数据面板点出最适合刺客/游侠的属性和技能；从贫民窟混混到冒险者公会注册，承接清剿深渊生物的任务；暗中积累力量，建立情报网络，逐步接触世界核心秘密
- **阻力**：①等级压制：高等级敌人经验衰减，越级挑战风险极大；②杀戮“副作用”：每次升级都会轻微侵蚀理性，积累过量后可能触发“狂暴”或“人格分裂”；③对妹妹的保护成为情感漏洞，敌人可能绑架/利用莉莉；④古老神祇和深渊领主视他为“棋子”，暗中操控
- **结局方向**：主角选择“吸收深渊规则，以自身意志镇守世界裂缝”，不再执着回归地球，而是成为新秩序的守护者；妹妹最终成长为独立的女法师，与主角并肩作战

## 3. 金手指（含深度设计）

### 3.1 金手指6要素（基础）

- **创意**：DND3.5E数据化面板，击杀生物获得经验值，升级时自由分配属性、学习技能、觉醒专长
- **功能（入口能力）**：实时显示视野内生物等级/生命/要害标记，且可感知经验值预估（击杀后能获得多少）；能无消耗使用一次“侦测弱点”（5秒内下一次攻击无视目标50%护甲/防御等级）
- **触发条件**：必须由主角亲手造成致死一击（包括远程/陷阱，但必须由主角本人最后一击）；组队时经验值大幅衰减（只有50%），且队友只能获得10%残余经验
- **限制边界**：
  - 初始只能看到等级不超过自身5级的生物面板，超过则显示“？？？”且无法侦测弱点
  - 经验值每升一级所需经验指数级增长，5级后必须越级击杀才能高效升级（同级或低级敌怪经验值低到可忽略）
  - 每天最多使用3次“侦测弱点”，且两次使用间隔至少1小时
  - 杀戮获得的经验值蕴含“深渊杂念”，每升2级需消耗特殊物品（“净化药剂”或“心灵试炼”）才能安全提升属性，否则累积杂念会导致失控
- **进化方向**：经历7个质变型阶段（见3.2）
- **本质**：金手指是上古深渊本源意志的“试验品”碎片——它故意赋予部分智慧物种杀戮升级的能力，制造强者互食的“进化沙箱”，最终筛选出最强的容器来承载深渊核心。主角若走完全部进化路线，将面临“被深渊意志同化为新神”还是“切断源流成为凡人之王”的抉择。

### 3.2 金手指深度设计

| 维度 | 内容 |
|------|------|
| **类型判定** | 组合型：面板流70% + 杀戮流30% |
| **功能分类** | 信息类（侦测/弱点标记）+ 成长类（属性专长分配）+ 战斗类（侦测弱点爆发） |
| **升级路径（7阶段）** | ①**凡人阶**（1-5级）：面板基础功能，只能分配6项基础属性（力量/敏捷/体质/智力/感知/魅力）；②**职业定轨**（6-10级）：触发一次强制转职任务（如“游侠之试炼”），获得职业特异化技能树；③**专精觉醒**（11-15级）：可觉醒职业专属传奇专长（如“精准射击”让远程攻击必中）；④**领域展开**（16-20级）：解锁“领域”——以主角为中心一定范围获得临时增幅，可同时影响队友；⑤**传奇突破**（21-30级）：超越凡俗数据框架，获得血统天赋（如半精灵血统完全觉醒，变成高等半精灵），属性上限突破常规；⑥**规则干涉**（31-40级）：能短暂改写局部世界规则（例如让“重力方向改变”或“元素火焰转化为圣光”）；⑦**本源共鸣**（41-?级）：与深渊意志底层规则同步，可有限度召唤深渊法则实体，但每次使用都会永久丧失一部分人性 |
| **限制条件深度** | **使用前提**：必须亲手终结目标生命；**冷却时间**：侦测弱点有限制；**副作用**：累积“深渊杂念”值——每升1级积累1点，积累到阈值（10/20/50）会触发“意志检定”失败则陷入狂暴1轮，并永久损失1点感知；**成长上限**：超过20级后深渊意志开始主动“召唤”主角，若不回应则面板开始崩坏（出现乱码、经验无效），逼迫主角寻找“净化仪式”或“晋升仪式” |
| **与剧情驱动关系** | ①被逼选择职业带出“血统使命”剧情（父母遗留线索）；②传奇突破时妹妹被深渊魔神掳走，触发“解救与协议”主线；③规则干涉阶段引出“世界是由多个深渊碎片拼接”的世界观真相；④本源共鸣阶段面临“毁灭世界重塑”还是“封印自己”的终极抉择 |

## 4. 主角设定卡 + 角色深度设计

### 4a. 主角设定卡（6要素）

| 要素 | 内容 |
|------|------|
| **职业/身份** | 半精灵穿越者，前世名林辰（25岁，游戏策划），现名埃德温·晨星（假姓，真实姓名未知）；贫民窟拾荒儿，后注册冒险者公会游侠 |
| **主性格** | 外冷内热：对外人警惕、精于算计、独行侠作风；对妹妹则无微不至、过度保护 |
| **记忆点** | 左耳戴一枚精灵风格的骨制耳环（原本属早逝的母亲），战斗时习惯先用左眼锁定目标（左眼瞳孔微带金色，半精灵血统觉醒时会发光） |
| **缺点** | ①依赖杀戮升级方式解决一切问题，不擅长外交/谈判；②对妹妹的过度保护导致她缺乏成长机会，实际限制了她的自立；③杀伐太多后下意识用“收益评估”思考人际关系，显得冷漠无礼 |
| **主角危机** | **存在性危机**：金手指本质是深渊本源试验品，升到传奇后必然会被深渊意志“回收”——他不升级则会因力量不足导致妹妹被深渊军团杀害；升级则一步步丧失自我，最终成为深渊的分身。要么妹妹死，要么自己变成怪物。两种结局都是失败，他必须找到第三条路：夺取深渊本源的控制权。 |
| **成长弧线** | 初始“被金手指驱动” (1-5级，追着经验值杀怪) → 中期“驾驭金手指”（6-15级，选择性升级，规避杂念） → 后期“超越金手指”（16-30级，发现本质后尝试切断源流，并利用规则改造金手指） |

### 4b. 角色深度设计

**主角深度（S级）**

- **表层欲望**：攒够金币给妹妹买一栋城里的安全别墅，让她永远不愁吃穿
- **深层需要**：真正需要面对自己的懦弱——前世妹妹病逝时他因为没钱没能去更大的医院，这一世他要靠杀戮获得力量来绝对掌控命运，但这种掌控本身却在复刻前世的失败（过度保护导致妹妹缺乏能力，一旦自己不在她就会死）
- **致命缺陷**：**过度保护（控制欲）** ——害怕失去妹妹，所以不允许她冒险、不让她学习战斗、阻挠她接触任何可能危险的事务，导致妹妹对自己产生逆反心理，性格变得孤僻且渴望证明自己，最终在一次私自外出中被深渊教徒绑架，成为敌人要挟主角的筹码。这个缺陷促使他必须在“放手让妹妹成长”与“继续圈养”之间做选择。
- **行为模式**：面对敌人：先隐匿侦测弱点，然后用致命一击解决（最有效率）；面对弱者：冷漠但会顺手帮一把（只要不耽误升级）；面对背叛：愤怒但迅速计算对方剩余价值，利用完再铲除；面对选择：优先选择战斗力提升最大的那条；面对资源：节省所有可用资金存起来买房产/装备，很少自己享乐
- **人物弧光（5阶段）**：
  - **起点（1-5级）**：穿越后充满不安全感，疯狂杀怪升级，把妹妹关在地下室里；性格偏执，只相信力量和金币
  - **转变期（6-10级）**：在冒险者公会遇到第一个真正的朋友——侏儒游荡者“蒂奇”，被他逐步引导认识到“人不是只有力量”；同时妹妹偷偷帮人配药发现她有草药学天赋，主角不得不让步
  - **成长期（11-20级）**：妹妹被绑走后暴怒，第一次对敌人动用深层面板能力（超出自身极限），导致杂念值爆满差点失控；随后开始学习团队合作，组建一支小队；学会信任他人
  - **巅峰期（21-40级）**：逐渐解开金手指真相，面临“成为深渊继承者与妹妹为敌”还是“自我牺牲切断深渊通道”的终极抉择；此时妹妹已成长为高阶女术士，主角开始依赖她而非单纯保护她
  - **终极期（41-?级）**：选择第三条路——不完全切断源流，而是以自身灵魂为锚点，将深渊意志转化为世界规则一部分，成为类似“虚空守望者”的半神存在；妹妹则成为大陆上最年轻的传奇法师议会成员，最后兄妹联手稳定位面裂缝

**核心羁绊角色（S/A级）**

1. **妹妹·莉莉**（S级，200-300字）
   - 关系：穿越后相依为命的亲妹妹（二人均为半精灵，但莉莉精灵血统更纯，有精灵尖耳朵）
   - 核心能力/功能：感知极高，天生亲和自然元素，后期成为元素术士（专注风/水两系）
   - 表层欲望：不想再做被保护的累赘，想帮上哥哥的忙
   - 深层需要：被哥哥真正地认可和信任，渴望平等而不是被施舍的亲情
   - **致命缺陷**：**逆反赌气**——为了证明自己故意做危险的事，有时会掉入简单的陷阱
   - 关系变化轨迹：被保护者→暗中跟随的学徒→被绑架后觉醒能力→并肩作战的同伴→最终成为主角的精神支柱
   - **人物弧光（3阶段）**：①（1-10级）天真懂事的拖油瓶→②（11-25级）叛逆觉醒期，被绑架后经历生死领悟到哥哥的苦心，决定用正确的方式变强→③（26级后）成熟的女法师，在主角动摇时反过来指引他

2. **侏儒游荡者·蒂奇**（A级，120-200字）
   - 关系：主角在冒险者公会的第一个队友兼损友
   - 核心能力/功能：机械制作（陷阱/暗器/辅助装备），情报贩卖
   - 表层欲望：多赚金币，出人投地
   - 深层需要：渴望被真正的朋友接纳（侏儒家族被灭门，他一直在寻找归属感）
   - **致命缺陷**：**过度贪婪**——为了金币可能会坑队友，但关键时候又会良心发现
   - 关系变化轨迹：相互利用的交易者→被迫组队后成为生死之交→即使主角力量超越他后仍保持平等朋友关系→最后成为主角情报网的代理人
   - **人物弧光（3阶段）**：①自私贪财的市侩侏儒→②经历一次被抛弃的阴影后选择救主角放弃巨额赏金→③成长为靠谱的“铁三角”一角

**终极对手（S级，200-300字）**

- **深渊意识投影·“折光者”**（真实身份是上古精灵皇族后裔，因研究深渊规则而被污染，成为深渊意志在这个世界的代言人）
- **威胁等级**：主角在20级左右首次接触其低级分身（约25级战力），最终形态超过70级（半神级）
- **对抗阶段**：
  - 第一阶段（10-15级）：折光者派遣低级邪教徒骚扰主角所在城镇，主角误以为是普通邪教
  - 第二阶段（16-25级）：通过绑架莉莉，折光者迫使主角进入“试炼迷宫”，逼他疯狂升级（加速杂念积累）
  - 第三阶段（26-35级）：折光者现身，与主角谈判——愿意将深渊本源让渡一半给主角，要求主角协助他打开世界屏障，迎接真正的深渊大军入侵
  - 第四阶段（36-50级）：折光者计划失败，暴怒下与主角正面决战，他的“真实形态”是一团不断蠕动的规则符文，能操控周围敌人/友方的经验值流动，强行让低等级生物升级后自爆
- **敌对原因**：折光者认为世界终将被深渊毁灭，与其苟延残喘不如早点投入深渊怀抱，成为新世界的种子；主角则坚持人类/精灵应有自己的道路，不愿成为深渊的祭品。他们是理念的碰撞，并非纯粹邪恶
- **弱点**：折光者自身也受到深渊杂念侵蚀，每一次使用大范围规则干涉都会导致他的一部分记忆消散；他的“核心”是一个半透明的精灵灵魂结晶，只要摧毁它就能净化折光者——主角最终选择将其净化而不是杀死，让他恢复成上古精灵贤者形态

## 5. 世界观骨架（6要素，含具体数值）

### 5.1 世界格局

希斯大陆（中世纪西幻背景）共分为四大区域：
- **西北人类帝国**（安卡尔帝国）：人口最多，科技和魔法混用，但内部贵族斗争激烈，对异族（半精灵/兽人）态度苛刻
- **东南精灵王国**（银月森林）：封闭保守，纯血精灵主导，排斥混血，半精灵在王国边境会受到歧视
- **中央矮人城邦**（铁峰联盟）：中立商贸枢纽，矮人、侏儒和工匠聚集地，地下有丰富的秘银和奥金矿脉
- **南方深渊裂缝带**（恶魔夷平区）：三年前深渊裂缝爆发，巨大裂隙横跨数百公里，不断涌出低级恶魔，周围建立了一系列“防御堡垒”和“冒险者猎杀区”

### 5.2 力量体系

**六大基础属性**（与DND3.5E一致，范围1-50，普通人平均10，传奇上限30，半神上限45，神力可达50+）：
- **力量**：物理伤害、负重、近战命中（每点+5%近战伤害）
- **敏捷**：远程命中、闪避、先攻、潜行（每点+2%闪避率）
- **体质**：生命值、耐力、强韧检定（每点+10生命值+1%基础生命恢复）
- **智力**：法术位、法术强度、智力检定（每点+2%法术伤害+1%法术位）
- **感知**：意志检定、觉察范围、治疗强度（每点+1%意志豁免）
- **魅力**：交涉、威吓、召唤强度、领导力（每点+1%召唤物属性）

**等级阶位**（共6阶）：
| 阶位 | 等级范围 | 称号 | 特征 |
|------|----------|------|------|
| 凡人阶 | 1-5 | 学徒 | 可自由分配属性，觉醒基础战斗本能 |
| 启灵阶 | 6-10 | 职业者 | 完成职业转职任务，获职业专属技能树 |
| 超凡阶 | 11-20 | 精英 | 觉醒职业专长，属性成长阈值提升至30；可组建冒险团 |
| 传奇阶 | 21-30 | 传奇 | 突破人类/精灵自然寿命极限，属性上限至30；解锁领域 |
| 半神阶 | 31-40 | 半神 | 可短暂干涉世界规则，需建立信徒/领地来维持存在 |
| 神阶 | 41-? | 真神 | 拥有创造/毁灭领域的权柄，不受凡间身体限制 |

**升级机制**：
- 每升一级获得1个属性点和1个技能点（可强化已有技能或学习新技能）
- 从凡人阶到启灵阶只需累计经验，但必须完成转职任务才能晋升启灵阶
- 从超凡阶到传奇阶需要获得一枚“传奇圣印”（可通过挑战传奇级生物、完成传奇任务或获得神恩）
- 从传奇阶至半神阶需要“点燃神火”仪式：收集5种不同属性的深渊结晶核，并完成一次“自我净化”
- 半神阶之后不再有固定经验需求，而是遵循“神力池”系统——通过信徒祈祷、领域扩张、击杀敌对半神来积累神力，积累足够后可尝试晋升

### 5.3 核心规则

1. **经验值与等级压制规则**：击杀等级低于自身5级以上的敌人经验值固定1点；等级低于自身10级以上的敌人不获得任何经验值。越级击杀（比自身高5级以上）经验值获得200%，且有机会掉落稀有材料。
2. **深渊杂念规则**：任何通过杀戮获得的经验都自带微量深渊污染。累积超过5点杂念进入“暴躁”状态（意志检定-5）；10点触发“深渊耳语”（每夜被噩梦纠缠，睡眠恢复效率-50%）；20点可能随机狂暴；30点以上可能精神分裂，出现第二人格。只有纯净的“圣水”或完成“心灵净化任务”可清零杂念。
3. **金手指与世界观对接**：主角的DND面板本质是上古“深渊本源碎片”的具现化，符合“世界法则可被量化”的深层设定；面板提供的数据辅助是规则层级的，不违反任何物理定律。无代价只有限制：升级必须有足够多同等级敌人存在，但深渊入侵正好提供了大量怪。

### 5.4 世界危机

**深渊第四波侵蚀**：在主角穿越这一年，深渊裂缝正在以一种不可逆的方式扩大。前三次侵蚀被各族联军镇压，但第四次侵蚀强度远超以往——裂缝中开始出现“规则扭曲”（特定区域内魔法失效、重力异常或时间流速改变），且每次击杀恶魔留下的尸体如果不及时处理会自行孵化出新魔物。一种名为“凋零雾”的瘟疫正从裂缝向南扩散，凡是被雾笼罩超过3天的生物会转化为中立魔物。世界面临“被深渊同化重塑”的终极威胁。

### 5.5 势力格局

- **安卡尔帝国**（人类）：国内贵族分成亲精灵派与仇外派，军力最强但不稳定；掌握帝国中央魔法学院和大型铁矿山
- **银月精灵议会**（精灵）：保守，对外援助有限，但拥有最高深的知识（知道深渊秘密）；与半精灵关系紧张
- **铁峰城邦联盟**（矮人+侏儒）：中立势力，垄断高端装备制造和地脉网络，在首次深渊战争后建立“深渊守卫者”组织
- **深渊裂隙势力**：包括低阶恶魔（炮灰）、高阶恶魔领主（半神级，各自拥有领域）、以及堕落的原住民祭司（如折光者）；内部也有派系斗争，部分恶魔领主希望吞噬裂缝本源，而非完全打开通道
- **玩家公会**（冒险者公会，自由）：中立松散组织，由各国出资共建，在裂缝区域设置前哨站；主角初始加入此公会
- **隐秘势力·血脉教会**（半精灵/混血种）：少数半精灵秘密结社，希望找到“纯血制造”的方法来终结种族歧视，其中部分被深渊利用（折光者出身于此）

### 5.6 进度框架（3段式）

1. **保命与积累期**（1-10级）：主角在贫民窟苟发育，注册冒险者，接取低级清剿任务获得资源，为妹妹攒钱买安全屋。位置：贫民窟→帝国边境小镇。
2. **冲突与势力接触期**（11-30级）：妹妹被绑架后主动介入主线，发现深渊阴谋；接触血脉教会，了解金手指真相；组建自己的小队（包括蒂奇、精灵游侠、人类牧师）。位置：边境→中央铁峰城→银月森林边缘→深渊前线堡垒。
3. **重塑与抉择期**（31级以后）：正面挑战折光者，夺取深渊本源控制权，面临“个人意志”与“世界存亡”的终极抉择；最终改写规则，把深渊裂缝从“入侵源”变成“试炼场”，并成为新秩序的守望者。

## 6. 长线线索

### 6.1 主线悬念：金手指的终极秘密是什么？

- **表层**：金手指是远古精灵帝国制造的“升神工具”，用于批量制造英雄对抗深渊。
- **中层**：金手指本质是深渊本源碎裂后残留在现实侧的“世界规则编辑器”，任何一个使用者都有潜力改写世界规则。
- **深层真相**：世界本身就是一个巨大的“深渊本源监狱”——现实侧是囚笼，深渊侧是狱卒的通道。金手指的制造者（上古精灵神）为了打破囚笼，故意散布碎片让凡人染上深渊力量，最终收集足够多的“半深渊者”作为钥匙冲击世界边界。主角如果走完全部进化，就会被当作活体钥匙献祭。

### 6.2 长线伏笔

1. **主角左眼金色**：当他情绪激动时左眼会闪烁金光——这是精灵先祖“本源共鸣”的标记，只有能完全掌控深渊规则的血统才会显现。这条线在主角升到传奇时彻底觉醒，赋予他“无视深渊杂念增幅”的天赋，但也让深渊意志将他锁定为“最优容器”。
2. **母亲的身份**：主角一直以为母亲是普通半精灵妓女，但母亲留下的那枚骨制耳环实际上是精灵王庭的“封印之印”。当主角到达银月森林时，精灵族大长老会认出耳环，揭示母亲其实是精灵族最后一任“虚空守门人”——她的职责是对抗深渊污染，但因生下混血儿而被流放。这解释为什么主角能以穿越者身份获得金手指（血脉使然）。
3. **妹妹的预言能力**：妹妹莉莉在中期觉醒元素能力时，偶然说出几句上古语，预言了深渊裂缝的自动关闭——但条件是需要“最纯粹的半血者献祭”。莉莉一直以为主角就是那个献祭者，会在最后关头主动牺牲自己，但被主角反转剧情。
