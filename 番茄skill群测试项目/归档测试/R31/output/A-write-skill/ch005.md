# 第五章 一个清晨

索伦在木屋的阴影里待到天亮。

阳光从门缝漏进来，照在地面的干草上。外面传来鸡叫声，狗叫声，还有早起女人的骂街声。

他站起来，活动了一下僵硬的肩膀。

一晚上没睡好。身上的伤还在疼，生命值只恢复到7/12。濒死虚弱的状态还没完全解除，动作稍大，肋下就传来一阵抽痛。

但没时间休息了。

他看了一眼床上的薇薇安。她还睡着，蜷缩成一团，像只小猫。面包包布还抱在怀里，脸上带着安心。

索伦走过去，想给她盖好被子——然后停住了。

他看到薇薇安转身时，后背的衣服被掀起来了一点。

露出的皮肤上，有一片青紫。

黑紫色，像淤血。拳头大小，在后背偏下的位置。不是摔倒的那种伤。

是棍棒打的。

索伦的呼吸停了半秒。

他蹲下来，轻轻掀开她后背的衣服。那片淤伤很大，从后腰延伸到脊椎附近，边缘泛着黄绿色——说明已经有两三天了。

不是昨晚的伤。

是之前被打的。

他猛地想起前天晚上，他把薇薇安抱回家时，她一直弯着腰，说是肚子疼。不是肚子。是后背。

她怕他担心，没说真话。

索伦放下衣服，坐在床边。

拳头握紧了。

不是愤怒——是冰冷的理智。他知道贫民区的孩子挨打是常事，但没有父母的孩子更惨。薇薇安没有父母，没有保护者，只有他这个半死不活的哥哥。

“谁打的？”

索伦低声问。声音平静得可怕。

薇薇安醒了。

她揉揉眼睛，看到索伦的脸色，愣了一下，然后笑了：“哥，你醒啦？”

“谁打的？”

薇薇安的笑容僵住。

“没……没人。”

“薇薇安。”

她低下头，手指攥着面包布。

“三天前。那个杂货铺的老板娘。我……我去拿剩菜，她说我偷东西，就用扫把打我。”

索伦没说话。

“我不疼的。”她抬起脸，努力笑了一下，“真的不疼。”

索伦看着她眼角的泪痕。

“我会找药。”

“嗯。”

“两天内。你等着。”

薇薇安点点头，像往常一样，无条件相信。

---

索伦站起身，走到门口的破镜子前，脱下上衣。

镜子里的身体——瘦，苍白，布满了新旧伤疤。胸口的伤是穿越时留下的，已经结痂；肋下的刀伤还没全好，绷带上渗出血迹；手臂上、背上、腰上，到处都是。

他不是什么大侠。他只是个在贫民区杀过人的半精灵。

【属性面板——当前状态】

【姓名：索伦】
【种族：半精灵】
【职业：平民LV1 / 盗贼LV1】
【生命值：7/12（濒死虚弱）】
【属性：力量12 敏捷19 体质15 智力18 感知15 魅力16】
【特征：夜视、左手灵活、半精灵血统】
【杀戮经验：30】
【升级所需：100点（盗贼LV1→LV2）】

力量12，比普通人的10点强一点。不算大，但够。

敏捷19——接近超凡门槛20点。这才是他最大的底牌。

他握了握拳。

杀死巴拉斯给了30点经验。加上昨晚那两个喽啰，一共60点。要升到盗贼2级，还需要40点。

两个敌人。

索伦把衣服穿好。

薇薇安已经把房间收拾了一遍。她把稻草铺平，把空罐子摆好，用破布擦干净桌子。看到索伦出来，她抬起头，笑了：“哥，你看，干净了！”

屋里确实干净了一些——虽然还是破，但有了点家的样子。

“好。”

索伦走过去，从口袋里摸出昨晚顺来的十二个铜币，放在桌上。

“米还有吗？”

薇薇安摇头：“只有昨天那块面包了。”

索伦看了一眼布袋。空的。

“我会找吃的。”

“嗯！”

她没问怎么找。她从来不问。

索伦拿出一块黑面包——昨晚藏起来的，掰了一半递给她。

“吃。”

“哥也吃。”

“我吃过了。”

薇薇安看着他，没说话。她接过面包，掰了一小块，剩下的又递给他。

“你吃。”

索伦看着她倔强的眼睛，接过面包。

两人坐在破桌子两边，默默地吃。

面包很硬，有馊味。但这是他们两人份的早餐。

吃到一半，薇薇安突然问：“哥，我们会搬走吗？”

索伦停下咀嚼。

“搬？”

“你昨天说，过两天搬到城里住。”

索伦想起来了。他从乱葬岗回来后，和薇薇安说过要离开贫民区。但那是空话——他不知道怎么赚到足够的钱，不知道去哪，不知道怎么带着一个八岁的小女孩在陌生的城市活下去。

现在不同了。

他有了方向。

“会。”

他放下面包。

“明天。明天我带你去城里。”

“真的？”

“真的。”

薇薇安的眼睛亮了。

她不知道索伦的计划有多疯狂——一个濒死的半精灵，带着八岁的妹妹，要去贫民区边缘的乱葬岗找宝藏。

她只知道，哥哥说了，就一定行。

索伦看着她信任的眼神，心里有块石头松了一点。

他从口袋里摸出那柄匕首，放在桌上。

匕首的刃上还有昨晚的血迹，已经干了，变成暗褐色。

“这个，你拿着。”

“哥不用的吗？”

“我还有一把。”

其实没有。但他要让薇薇安有安全感。

薇薇安接过匕首，握紧。她的手很小，只能握住一半柄，但她点了点头。

“我会保护自己的。”

“不。”

索伦纠正她。

“是有人想伤害你的时候，用它扎他。扎喉咙，扎眼睛，扎大腿根。扎完就跑。找亮的地方跑。找人多的地方跑。”

薇薇安认真点头。

“我记住了。”

索伦站起来，走到门口。

外面，贫民区的街道已经热闹起来。小贩推着车叫卖，女人蹲在门口洗衣裳，孩子们光着脚在泥地里跑。

一切都像昨天一样。

但索伦知道，不一样了。

他从口袋摸出昨晚从胖子身上顺来的劣质匕首。

【物品：劣质匕首（普通）】
【攻击：1d3】
【耐久：5/10】

不如他那把好。但够用了。

他需要更多的经验值。需要更多的钱。

要去乱葬岗，需要一把像样的武器，需要治疗药水，还需要食物和饮水。

而他现在的全部家当是——十二个铜币，一块黑面包，两柄破匕首。

不够。

远远不够。

索伦站在门口，看着远处乱葬岗的方向。

那里埋着巴拉斯。那里还有几个在贫民区混饭吃的小角色。

杀了他们，能拿到经验值。

杀了他们，能拿到钱。

杀了他们，贫民区的帮派会混乱几天。

而他需要的就是这“几天”的时间窗口。

他转头看了一眼屋里的薇薇安。

她在吃面包，小口小口地嚼，像只仓鼠。

“等我。”

他低声说。

然后，他走进阳光里。
