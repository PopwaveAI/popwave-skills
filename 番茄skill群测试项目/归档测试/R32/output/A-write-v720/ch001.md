# 当我的哥哥在雨夜中睁眼时，贫民窟的命运就已经注定

雨砸在破木板上，噼啪响。

八岁的薇薇安跪在灶台前，用冻僵的手添柴。三块破砖头架着一口缺了口的铁锅，里面是最后一把米——她翻遍整个屋子才找到的。

米粒在沸水里翻滚，白气升腾，混着木柴燃烧的烟味。她的肚子咕咕叫，但她只是舔了舔干裂的嘴唇，继续盯着锅里的粥。

床上传来微弱的呼吸声。

薇薇安回头看了一眼——哥哥蜷缩在破棉被里，脸色惨白，嘴唇发紫，已经昏迷七天了。她不知道他为什么昏迷，只知道那天他回来时浑身是血，衣服烂成布条，手里握着一把断掉的匕首。

她把匕首藏在了床底下。

【任务：救治哥哥——未完成】

这行字浮现在视野里的时候，她已经习惯了。三天前开始出现的，像是什么东西嵌进了脑子里。她看不懂，但知道这跟哥哥有关。

粥煮好了。她舀了一碗，小心翼翼地端到床边。

“哥，”她道，“吃饭了。”

哥哥没反应。

她用勺子舀了一勺，吹了几口气，凑到他嘴边。他的嘴唇紧闭着，粥顺着嘴角流下来，滴在枕头上。

薇薇安的眼泪一下子就涌出来了。

“哥，你醒醒啊……”她声音发抖，“你不醒，我就——”

她就得把自己卖给人口贩子了。那个满脸横肉的男人已经来过两次了，说像她这样的小女孩能卖个好价钱，能给哥哥买药。

她擦掉眼泪，又舀了一勺粥。

老狗希斯趴在门口，耳朵竖着，灰色的眼睛里透着警惕。这条老狗跟了他们四年，是父亲死后留下的唯一东西。

屋外传来脚步声。

希斯站起来，喉咙里发出低沉的咆哮。

薇薇安放下碗，跑到门边，贴着门板听。脚步声很重，是成年男人。她在心里数着——一个，两个，三个。脚步声在门口停了一下，又走远了。

她松了口气，靠门滑坐在地上。

贫民窟就是这样。白天是贼，晚上是匪，半夜还有杀人的。一个月前，隔壁的老约翰半夜被捅了十三刀，尸体扔在巷子里，三天没人收。第四天，他的女儿失踪了。

薇薇安抱紧了希斯，把脸埋进它的毛里。

“哥，”她小声道，“你要是不醒，我就真的撑不下去了。”

锅里还有半碗粥。她给自己盛了一勺，吹凉了喝下去。米粒在嘴里慢慢嚼烂，咽下去，胃里暖了一瞬。

喝完粥，她把锅碗收拾好，又坐到床边。

哥哥的手指动了动。

薇薇安瞪大了眼睛。

又是一下。

她的手抖了起来，抓住哥哥的手，“哥？哥！”

哥哥的眼睛缓缓睁开。

那双眼睛——比昏迷前更亮，像是有东西在里面流动。

“哥！”薇薇安扑进他怀里，放声大哭，“你终于醒了，我好怕……”

她没注意到，哥哥的瞳孔里，倒映着密密麻麻的数据流。

【宿主苏醒完成】

【生命体征稳定】

【系统初始化进度：100%】

【欢迎回到——你的世界】

索伦的喉咙动了动，想说点什么。

他看了看怀里的妹妹，又看了看床上破旧的棉被，和墙角漏雨的屋顶。

时间线对上了。

他记得自己死了，死在血色城堡的最后一战。无数刀剑刺穿他的身体，然后有光吞没了他，再然后——

他回到了十五年前。

回到了琥珀城的贫民窟。

回到了妹妹还活着的时候。

他的手摸向腰间——匕首还在。

这一次，没有人能从他手里夺走任何东西。

“我没事了，”他道，声音沙哑，“没事了。”

薇薇安哭着笑着，在他身上蹭来蹭去。希斯也凑过来，用脑袋蹭他的手。

索伦摸了摸老狗的耳朵，眼睛却盯着窗外。

雨还在下，黑暗中有什么东西在动。

是食物链底层的掠食者，在寻找下一个猎物。

但他不再是猎物了。

他站起来，腿还有点软，但已经能走了。他走到墙角，从床底下摸出那柄断掉的匕首——上面还残留着干掉的血迹。

“哥？”薇薇安看着他的背影，“你要做什么？”

索伦转过身，笑了笑。

“做饭，”他道，“你先睡，我把粥热一热。”

薇薇安盯着他看了几秒，点了点头。

她确实困了。连日来担惊受怕，哥哥一醒，紧绷的神经一下子就松了。她爬到床上，蜷在还留着哥哥体温的棉被里，很快就睡着了。

索伦等她睡熟了，才从墙角的破柜子里翻出半截蜡烛，点燃了，插在木桌上的酒瓶里。

他从靴子里抽出另一把匕首——比断掉的那把长，也更锋利。

老狗希斯抬起脑袋，看着他。

索伦蹲下来，拍了拍它的头。

“守着妹妹，”他低声道，“我去去就回。”

希斯呜了一声，算是答应了。

索伦推开门，走进雨里。

这是圣者浩劫前十五年，深渊恶魔还未大规模入侵，诸神还在万神殿里安坐，而他有十五年的时间，把这个世界搅得天翻地覆。

但一切要从雨夜的贫民窟开始。

从那些，打他妹妹主意的人开始。
