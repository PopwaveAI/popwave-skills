# 妹妹背上的淤青，和一条老狗的葬礼

门外的雨已经小了。

索伦踏进雨夜，破木屐踩在泥水里，发出吧嗒的声响。他没带伞——贫民窟的人也不配用伞。雨水顺着他的脖子往下淌，浸透了破烂的单衣。

【力量12·敏捷19·体质15·智力18·感知15·魅力16】

【职业：平民LV5 / 盗贼LV1】

【生命值：12/12】

【未分配经验：150】

数据流在视野边缘跳动，像嵌进骨头的纹身。他记得前世的自己花了整整三个月才习惯这个东西——而这次醒来，不到一刻钟就已经完全接受了。

【注：150点杀戮经验可分配至平民职业】

他选了个岔路口的方向——通往码头区仓库的捷径。昨晚打妹妹主意的那两个混混，名叫巴拉斯和卡诺波，就住在更北边的废木棚里。但他不急着动手，刚醒的虚弱还在，得先确认自己这具身体的状态。

走到巷口时，他停下脚步。左边第三间木屋的窗口透出昏黄的油灯光——那是巴拉斯的老巢，门虚掩着。

他的心跳快了几分。不是恐惧，是兴奋。

索伦猫着腰靠近，身体贴着墙壁，每一步都踩在屋檐滴水的位置——雨水冲掉了脚步声。他侧头从门缝里看进去：两张破床上躺着人，一壮一瘦，鼾声粗重。桌上搁着半瓶劣酒，旁边的油灯快燃尽了。

【潜行判定：21（基础）】→【成功】

他推开门，吱呀声被鼾声盖过。匕尖探出来，月光从窗外洒进，照在那把匕首上——是他床底下那柄断掉后又被他磨快的旧货，刃长不到四寸，但足够深了。

巴拉斯翻了个身，嘴里嘟囔着梦话。

索伦走到他床边，一只手捂住了他的嘴，另一只手的刀锋从颈侧刺进去。不是割喉，是斜捅——刀尖穿过软肉，刺入颈椎骨之间的缝隙，干净利落地切断脊髓。

巴拉斯身体猛地绷紧，然后软了。

床上的卡诺波还没醒。索伦转身，同样的手法解决了他。血沿着床沿往下淌，滴在地板上，汇成一小摊。

【击杀巴拉斯（平民LV6）——获得15点杀戮经验】

【击杀卡诺波（平民LV4）——获得10点杀戮经验】

【累计击杀经验：25点】

索伦甩了甩匕首上的血，在死者的衣服上擦干净。他的手指有点抖——不是体力问题，是这具身体太生涩了。敏捷属性很高，但肌肉没有记忆，动作总是差半拍。

“得适应几天，”他低声自语。

回到破屋时天色还没亮。薇薇安还在睡，缩成一团像只小兽。老狗希斯趴在床边，听到动静竖起耳朵，看到是他后又趴下去，尾巴摇了摇。

索伦在门槛上坐下来，把靴子脱了，倒掉里面的积水。然后他开始分配经验值——150点全加在平民职业上。

【平民LV5→LV6·消耗35点杀戮经验】

【平民LV6→LV7·消耗40点杀戮经验】

【平民LV7→LV8·消耗45点杀戮经验】

【平民LV8→LV9·消耗55点杀戮经验——失败，需求60点】

他皱了皱眉。平民职业虽是最底层的职业，升级难度却一点不低。前世他穿越到这个世界的第三年才搞清楚这个系统——平民职业每10级带来的基础属性增益，是所有高阶职业的根基。

【平民LV8】

【职业专长：无】（平民职业无特殊专长）

【备注：平民升至10级后将开启第一职业进阶路线】

还剩15点经验，不够再升一级。他收好面板，靠在门框上闭眼了片刻。雨已经停了，夜风吹过来，带着泥土和腐木的腥味。

天蒙蒙亮的时候，薇薇安醒了。

她揉了揉眼睛，看到门口坐着的索伦，咧嘴笑了，“哥！”

“给你弄点吃的，”索伦站起来，活动了下发僵的肩膀。

锅里还剩下半碗凉粥。他热了热，加了点盐，端给薇薇安。她接过来，喝了一口，抬起头，“哥你不吃？”

“我不饿。”

你饿的。他咽了口口水，胃里火烧火燎，但这粥必须留给妹妹。他转头走到床边收拾——昨夜从巴拉斯尸体上摸回来的一袋铜币和几枚银币，一共十三个银德勒。

“够买几斤粗麦了，”他掂了掂钱袋，“还能剩点买药。”

薇薇安放下碗，朝门口走过去，“希斯，吃饭啦——咦？”

她的声音突然僵住了。

索伦转过头。老狗希斯还趴在他刚才坐过的门槛旁，一动不动，嘴巴微张，灰色的眼睛半睁着，像在看着什么。

“希斯？”薇薇安蹲下来，推了推它的脑袋。

老狗的身体是僵硬的。

“希斯！”薇薇安的声音尖了起来，她使劲摇头，“哥，哥它怎么了——”

索伦走过去，摸了摸老狗的脖子——脉搏已经没了。身体已经凉了，至少死了有两三个时辰。

昨夜他出门的时候，老狗还在守门。

他回来的时候，老狗轻轻摇了尾巴。

那是希斯最后一次见到他，然后它就倒下了。

一条十五岁的老狗，守护过他们的父亲，又守护了他们四年。它太老了，昨晚的雨夜和寒冷，耗尽了它最后一口气。

薇薇安的眼泪一颗颗掉下来，落在希斯的灰毛上。

索伦蹲在旁边，伸手把老狗的眼睛合上。

“它不疼的，”他哑着嗓子，“老死的，不是被打的。”

薇薇安扑进他怀里，肩膀颤抖着。

晨曦从破屋顶的缝隙里漏进来，照在这间狭窄的屋子里。一个八岁的小女孩，抱着她刚醒来的哥哥，为一条老狗哭泣。

索伦拍了拍她的背，看向窗外。

天亮了。但他知道，这个世界的黑暗才刚刚开始。

【提示：老狗希斯的死亡触发剧情支线——守护者的遗志】

【奖励：体质+1（永久·来自守护者的馈赠）】

他的身体微不可察地颤了下。属性面板上，体质数值从15跳到了16。这条老狗到死都在守护他们。

“等我回来安葬它，”索伦道，“我送你到晨曦神殿去，那里有教会的修士阿姨。”

薇薇安抬起头，眼睛红红的，“我不去。”

“你必须去——”

“我不去！”她喊了一声，又缩了缩身子，“哥你要去哪儿？”

索伦沉默了片刻。

“去找吃的，”他道，“和药。”

他没说实话。他要去找那具昨晚解决的第二具尸体——卡诺波身上也许还有值钱的东西。他还得摸清这具身体的战斗极限，贫民窟不会给他太多适应时间。

薇薇安盯着他看了好一会儿，终于点了点头。

索伦背起希斯的尸体，带着薇薇安走到屋后的小土坡。他用偷来的那枚银德勒找守墓老头租了块地，亲手挖了个坑。泥土又硬又潮湿，混着碎石头和草根，一铲子下去嗡嗡响。

薇薇安在旁边摘了一把野花，放在希斯的身体旁边。索伦把土填上，用力踩实。没有墓碑，他在土堆上捡了块石头，划了个“犬”字。

“走吧，”他道。

“哥，”薇薇安拉了拉他的袖子，“你看。”

他顺着她的手指看过去——土堆旁边，一株不知名的幼苗正在晨光里冒出来。

索伦愣了一下。

【守护者的遗志——已激活】

【你的身边永远有一个沉默的影子】

他牵起妹妹的手，走向贫民窟的方向。

老狗的尸体已经冰冷，但贫民窟里盯着他们的眼睛，从昨夜开始已经少了两双。

而这条街上的肮脏生物，很快就会知道——

这条街，换主人了。
