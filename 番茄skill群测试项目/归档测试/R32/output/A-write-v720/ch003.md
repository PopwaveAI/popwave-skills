# 墓园里，一条老狗的葬礼和八十银币的承诺

索伦扛着铁锹走回破屋的时候，薇薇安还蹲在老狗身边，用脏兮兮的小手梳理希斯的灰毛。看到哥哥回来，她站起身，脸上挂着一道道泪痕，但没有哭出声。

“走吧，”索伦哑着嗓子道。

他弯下腰，双手搂住老狗的躯干。希斯的身体已经僵硬了，四条腿直挺挺伸着，像一根冻僵的木头。他使劲往怀里一收，膝盖顶在狗肚子下面，摇晃了两下才把这个四十多斤的老家伙扛起来。

尸体贴着脖子，冷硬，没有一丝温度。

薇薇安紧跟在后面，一只手拉着他的衣角，另一只手攥着从希斯身上捋下来的一撮灰毛，指甲攥得发白。

泥泞的街道上，两旁的破烂木屋里探出几张灰扑扑的脸。一个胡子拉碴的光头靠在门框上，嗤笑一声，“哦哟，捡破烂的小子扛着条死狗去哪儿？”

索伦没搭话，步子没停。

光头的笑声干巴巴的，像钝刀刮骨头。

薇薇安攥着他衣角的手紧了紧。索伦侧过头看了她一眼——那张小脸绷着，没有回头，没有哭。

他忘了从前这丫头是什么时候学会的——不搭理那些苍蝇。

穿过三条巷子，路过几个冒着恶臭的垃圾堆，拐过倒塌的烂墙根，他们终于到了墓园的铁栅栏门口。

说是墓园，其实就是城西边一片荒坡。坡上东倒西歪插着几块木板，下面埋着没亲没故的穷鬼。杂草齐腰深，铁栅栏烂了一半，剩下的锈得发红。

守墓的老头住在门口的木棚里。

索伦推开门走进去。老牧师正坐在门槛上啃干饼——浑浊的眼睛抬起来，扫了一眼索伦肩膀上扛着的狗，又落在他身后的薇薇安身上。

“埋人五个银的，埋狗——”老头顿了顿，“活物，三个银。”

索伦把希斯放在地上，从口袋里掏出钱袋。早上从巴拉斯和卡诺波尸体上搜出来的钱，一共有十三个银德勒，十七个铜子。

他数出三枚，迟疑了一下，又添了两枚。

“五枚，”他双手递过去，“算人的。”

老牧师没接钱，盯着他看了好一会儿。

“你爹妈呢？”

“死了，”索伦道，“四年前。”

“那这狗——”

“也死了。昨晚。”

老牧师沉默了片刻，接过钱掂了掂，“角落那块地，树底下，自己挖。”

索伦点头，转身扛起老狗往墓园里走。薇薇安一直没松他的衣角。

泥土干了，硬得像石头。他往下挖了半个时辰，铁锹磕在碎石上溅出火星，掌心的皮磨破了三道口子。

薇薇安蹲在坑边，把手里那撮灰毛放进去。

索伦把希斯放进坑里，开始填土。第一铲土砸在狗肚子上的时候，他的动作顿了一下，然后继续。

填平，实压，拍平。

矮胖子墓园管理人走过来，递给他一块刻着狗头的木牌。索伦接过来，用铁锹敲进土里。

他蹲下来，在薇薇安面前半跪着，擦了擦她眼角的泪，“走吧，回家。”

薇薇安抬起头，拉住他的手指，“哥，希斯去哪儿了？”

索伦沉默了片刻，道，“它守完了我们，也该去守它自己了。”

他的手指在口袋里摩挲着剩下的八枚银币，盘算着该买药还是买粮食。但看着眼前瘦弱的妹妹，他忽然明白了——这两件事根本容不得选择。

他牵着薇薇安的走，走出墓园。

天色还早。雨停后，天边露出一片灰白的光。空气里有股潮乎乎的泥腥味。

走在街上，索伦的脑海里突然闪过了《诸神之战》的未来——黄昏时的腥风血雨，圣者降临，诸神陨落，魔网崩溃，亡灵天灾席卷大地，无数生灵化为枯骨。

这些记忆碎片像刀子一样扎进他的骨髓。

“哥，你怎么了？”薇薇安问。

索伦没说话，只是低头看着她。那张八岁的小脸上还挂着泪痕，但眼睛里有股倔强的劲儿。她刚失去了一条守护她四年的老狗，却还能拉紧他的手，仰起头问他怎么了。

“没什么，”索伦咧嘴笑了笑，“走吧，去买点吃的。”

他捏紧了口袋里那八枚银币。

索伦拉着薇薇安穿过一条窄巷，拐进早晨的集市口。摊贩们刚支起板子，卖菜的胖女人在那抖索着摊开绿叶子，卖鱼的老头把昨晚没卖完的死鱼又摆上来了，苍蝇嗡嗡围着他转。

一个卖烤饼的妇人看他走来，皱着眉道，“小孩一边去！别站这儿挡客人！”

索伦从口袋里掏出两枚银德勒，“两张饼。”

妇人的表情马上变了，麻利地用油纸包了两张烤饼递过来。索伦接过来，塞给薇薇安一张。

薇薇安接过来，咬了一口，又抬头看他，“哥你呢？”

“我吃过了，”索伦咬了一口自己那张，嚼了几口，嗓子眼干涩，咽下去费劲。

他确实饿。胃里火烧火燎的，但他还不能让薇薇安看出来。

两人蹲在巷口的石阶上啃完了饼。索伦抹了抹嘴，盘算着接下来的计划。

“哥，”薇薇安忽然道，“希斯的葬礼花了五个银，我们还剩多少？”

“八个银，”索伦道，“五十七个铜子。”

他想了一下，决定先不买药了。昨晚杀那两个混混让他攒了25点杀戮经验，加上之前的150点，他给自己规划了一条清晰的升级路径。

先升平民到10级，积累足够的基础属性，然后转职盗贼，正式踏入超凡领域。

但在那之前——

他得先活过这几天。

贫民窟里不缺饿鬼。一条老狗死了，两只眼睛盯着他们的，不会只有昨晚那两个巴拉斯和卡诺波。那些肮脏生物，很快就会知道这条街上少了两双盯着猎物的眼睛。

得让他们知道，这地方换主人了。

他捏紧了口袋里那枚银币。

“走，”他站起来，“我带你去看看，这条街上还有多少不长眼的货色。”

薇薇安仰起头，看着他。那双眼睛里没有恐惧，只有信任。

她握紧了他的手指，“好。”

两人牵着手，拐进更深的巷子。

市场里的喧嚣被身后的建筑物吞没了。索伦的脚步声在巷子里回荡，一下，两下，三下——

他的目光落在巷子尽头那扇虚掩的木门上。

里面传来男人的粗嗓门，“昨晚上巴拉斯和卡诺波那两个废物不管用了——”

门板忽然被人一脚踹开。

索伦侧过头，嘴角微微勾起来。

他的手捏紧了铁锹把。

他松开了妹妹的手，“站在这儿别动。等我回来。”

薇薇安点了点头。

索伦提着铁锹，走进了那扇门。

晨光从破屋顶漏下来，照在院子里的积水滩上。三个男人围着一个火堆，喝着昨晚剩下的劣酒，看到索伦走进来，眼神不善地落在他身上。

领头那个光头咧嘴露出一口黄牙，“小孩，你走错地方了——”

索伦没废话。

操起铁锹就往他脸上拍。

刀锋般的锹沿正正拍在光头的鼻梁上，血花四溅，光头惨叫一声栽倒在地。左右两个人还没反应过来，索伦已经翻身横踢，一脚踹在第二个人的膝盖窝，那人跪倒闷响，他一锹背又拍在对方后脑勺上。

不到十秒，三个全躺了。

索伦甩了甩铁锹上的血，扫了一圈地上呻吟的男人，吐了口唾沫，“告诉你们老大，这条街——换主人了。”

他提着铁锹走出院子。

薇薇安还站在那里，一动不动。

索伦走过去，牵起她的手，“走吧。”

“哥，”她小声道，“你受伤了吗？”

“没有。”

“那他们呢？”

“他们该倒霉了。”

他捏紧了口袋里那八枚银币。这个贫民窟藏着的，远远不止这些。他知道——这个世界的黑暗，才刚刚开始。

而他那把磨快的匕首，也才刚刚沾血。
