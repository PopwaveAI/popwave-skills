# 第三章「墓园区」

索伦的手指碰到希斯的身体，冰凉。

他蹲在门口，煤油灯的光映在老狗的身上。希斯的肚子不再起伏，眼睛半睁着，瞳孔已经涣散。它的嘴边还留着昨晚薇薇安喂它的小块黑面包，没咽下去。

索伦伸手把希斯的眼睛合上。手指触到那层薄薄的眼皮时，他才发现自己的手在抖。

不是怕。是愤怒。

“哥……”

薇薇安站在他身后，声音很小，像是怕吵醒谁。她手里还攥着那条破毛巾，昨晚用来给希斯擦嘴巴的那条。

“它睡着了？”她问。

索伦没回答。他站起身，把希斯抱起来。老狗的身体比想象中轻，骨头撑着一层皮，像是抱着一捆干柴。它活着的时候还能走能叫，死了才知道，这狗早就瘦成了这样。

“换鞋。”他说，“跟我走。”

薇薇安没有追问。她低头套上那双大了两号、鞋头磨出洞的破皮靴，又看了眼希斯趴过的地方——那里只剩下碎木板上一个压扁的印子。

贫民区的街道在蒙蒙亮的光线里显得更脏。泥浆混着垃圾被露水泡软，踩上去又滑又黏。索伦抱着希斯走在前头，薇薇安跟在他身后，一步一踩，溅起的泥点子落在她小腿上。

街道边有人探头看。

两个蹲在屋檐下烤火的流浪汉瞥了一眼，又低下头去，装作没看见。一个正在倒水的中年女人看见索伦怀里的死狗，皱了下眉，转身进屋关了门。

没人说话。但那种看垃圾一样的眼神，索伦都记着。

他低头看怀里的希斯。这条老狗跟了他母亲多久，他记不清了。母亲死后，它没有走，守在门口，守着他和薇薇安。它不吃别人的东西，只吃薇薇安递来的。它不看别人，只等他的脚步声。

然后它死了。饿死的。

索伦的嘴角绷成一条直线。

昨晚那些敲门的、想拖走薇薇安的脸，一个一个在脑子里过。布拉格，那个收保护费的。巷子口的混混头子卡诺波。还有那个笑得很假的胖子，巴拉斯。

索伦把他的脸刻进了骨头里。

“哥哥，墓园到了。”

薇薇安的声音把他拉回来。他抬头，面前的铁栅栏门歪在一个快要垮掉的水泥墩子上。门上的铁锈一层盖一层，写着“晨曦墓园”的木牌斜挂着，字迹已经模糊不清。

门后是一片荒地和几排歪斜的墓碑。野草从裂缝里长出来，有的比人还高。晨雾还没散，让这片地方看起来更像被世界抛弃的角落。

索伦推开门。铁门发出吱呀一声，惊起一只乌鸦，它从墓碑上飞走，翅膀拍碎雾气。

墓园角落里有一间小木屋，烟囱正冒着烟。索伦走过去，抬手敲了三下门。

门开了。

开门的老人穿着破旧的灰袍子，露出来的手腕上全是皱纹。他的眼睛浑浊，胡须很久没刮，一张脸像是晒干的橘子皮。

“什么事？”他的声音也像橘子皮，干巴巴的。

“埋狗。”索伦说。

老人扫了他一眼，又看到他怀里的希斯，目光落在薇薇安身上停了两秒。“埋狗要钱。”

“多少？”

“五个铜德勒，挖个坑。十个铜德勒，立块木板。一个银德勒，给你念段经文。”

索伦把希斯放下，从最里层的口袋里摸出一枚银币。银德勒在他手里磨蹭了一下才递过去——这是他身上最值钱的东西，藏了两天，连买黑面包时都没舍得花。

老人接过银币，用牙咬了一下，然后塞进腰带。

“跟我来。”

他带着他们走到墓园最角落。那里有几棵枯树，树根裸露在外，周围全是野草。老人用手比划了一下：“就这儿，挖。”

索伦没有说话，从木屋旁边捡起一把铁锹。铲子生锈了，柄上还沾着干透的泥。

他用力踩下第一铲。

土很硬，底下全是碎石头。每一铲下去，铁锹都发出沉闷的声响。索伦的手臂酸得发抖，但他没停下来。他一下一下挖，汗从额头滴到土里，衣领湿了，后背也湿了。

薇薇安站在旁边，抱着希斯的头。她没有哭，也没有说话，就那样蹲着，看着哥哥挖坑。

老人搬来一块巴掌大的木板，靠在树上，又回屋端出一碗浑浊的酒水，泼在坑里。

“好了。”他说，“放进去。”

索伦弯腰，从薇薇安怀里接过希斯。老狗的身体已经彻底凉透了，僵硬得像一块木头。他把它放下去的时候，手指不小心碰到那双半闭的眼睛。

他停了一下。

然后，他开始填土。

一铲。一铲。泥土盖住希斯的腿，盖住它的肚子，盖住它的脸，直到最后一个土堆鼓起来，和一排简陋的坟墓一样高。

老人清了清嗓子，对着土堆说了几句含糊不清的话。他的口音很重，像是夹杂着古老的语言，索伦只听懂了一个词——“安息”。

然后老人回了木屋。

晨雾散尽，阳光透过枯树枝照下来。墓园里安静得像没有活着的东西。

薇薇安的眼泪终于没忍住。她咬着嘴唇，肩膀一抽一抽的，但没有哭出声。她蹲在土堆前，伸手摸着那块巴掌大的木板，她刚才用石头在上面划了几个字——“希斯”。

“它还会回来吗？”她问。

“不会。”

“那我能梦到它吗？”

索伦沉默了一下，“也许会。”

薇薇安站起来，用袖子狠狠蹭了一下眼睛。她没有再哭，而是转过身，把手伸向索伦。她的手很小，手指上全是泥。

“哥，我们走吧。”

索伦看着她。那双眼睛红红的，但里面有一点亮光——不是绝望，是坚定。

他抓住她的手，握紧。

“走。”

她的小手抓紧他的手指，两个人转身，踩着野草，踩过碎石子，一步步走出墓园。

铁门在他们身后关上，发出吱呀一声。

薇薇安忽然停住了脚步。

“哥。”

“嗯？”

“那些人，昨晚想拖我走的人……”

索伦低头看她，从她眼睛里看到了比刚才更亮的光。

“他们还会来吗？”

索伦的嘴角动了动。

“他们不来找我们，我也会去找他们。”

薇薇安没有再问。她拉着他的手，跟着他走出那道生锈的铁门。

阳光照在他们身后的墓园里。角落那个新土堆上的木板还没有干透，“希斯”两个字歪歪扭扭，但刻得很深。
