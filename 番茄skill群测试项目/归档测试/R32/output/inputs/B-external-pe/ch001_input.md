# 深渊主宰 · 卷1剧情白描（ch001-ch187）

八岁女孩薇薇安在贫民窟的夜雨中煮粥喂食昏迷的哥哥，老狗希斯驱赶着觊觎他们的目光。当哥哥索伦终于睁眼时，薇薇安扑进他怀里大哭——但她不知道，哥哥的灵魂已经融合了游戏数据流，眼前浮现密密麻麻的0和1。第二天清晨，老狗希斯力竭而死，索伦抱着它在墓园安葬，用偷来的一枚银德勒支付安葬费。他知道这不是游戏，而是真实世界，而他预知这个世界即将迎来诸神黄昏与圣者浩劫。

索伦先用匕首在夜色中背刺了打妹妹主意的巴拉斯和卡诺波，获得30点杀戮经验。他查看属性：半精灵，敏捷19，智力18，职业是平民和一级盗贼。看到薇薇安后背的青黑淤伤后，他决定尽快找治疗药剂。潜入码头区仓库撬锁拆陷阱，盗贼升到2级，敏捷突破20点超凡门槛。他计划前往藏宝洞获取超凡装备，同时顺手清除了贫民区的科尔帮派，一夜杀光核心成员，赢得“割喉者”的绰号。

将薇薇安托付给晨曦神殿的见习牧师安亚丽后，索伦独自前往乱葬岗下的藏宝洞。他先后击杀骷髅、用圣水消灭幽灵，在密室内发现一个铁木盒。打开后是奥术帝国遗物——万象无常牌。第一张“神偷”把他全身物品偷光，第二张“苦行者”将体质提升到18点并觉醒坚毅专长，第三张直接召唤出三阶深渊蛇魔。索伦利用地道陷阱重创蛇魔，在濒死时将盗贼升至4级，最终击杀蛇魔获得1800点经验。打扫战场时发现两把蛇魔长剑+1，并找到食人魔力量护腕。

离开藏宝洞后，索伦出售武器，用100金德勒购买低级次元袋。他预知魔法女神将陨落、魔网崩溃，决定带薇薇安前往冰雪国度避开圣者浩劫。在冒险者工会注册后，他见识了琥珀城剥皮事件的升级，认出这是混乱时代来临的征兆。准备远行时，薇薇安拒绝成为牧师的建议，坚定表示要在哥哥身边。旅途中，索伦发现妹妹只看一次就学会光亮术，测试后确认她是天生术士——魅力21点，感知18点。他推测神子身份，决心用生命守护她。

商队遭遇掘地虫袭击，索伦凭记忆判断是三阶群居生物，指挥击杀后获得大量经验。他将战利品收益最大化分配，赢得商队尊重。北地女巫歌莉娅主动接近，索伦察觉到她觊觎薇薇安天赋，始终警惕。女巫占卜看到血与火的未来，预言薇薇安将伴随杀戮与战争。索伦坦言自己在害怕和逃避即将到来的浩劫，但绝不让她离开。

琥珀城被毁灭的消息传来，诸神不再回应信徒祈祷。索伦意识到必须提升实力，他猎杀狗头人群，将体质突破20点超凡领域，获得再生天赋。又击杀豺狼人首领，领悟剑势重劈。在村寨抵抗蜥蜴人时，他兼职巫师，撕开四环阴影黑触手卷轴逆转战局。他用数据模板永久储存法术模型，突破传统巫师对法术书的依赖。

白马城军队清扫食人魔部落，索伦参与侦查。他与半精灵游侠亚斯配合，在碎骨之战中击杀食人魔督军，获得2400点经验，巫师升至5级，领悟法术反制。亚斯阵亡后，索伦为他立碑，获得1点传说度。回归白马城后，歌莉娅决定同行。途中，索伦揭开薇薇安可能是深渊魔神神子的真相，歌莉娅通过侦测谎言确认预言大体真实。

索伦一行穿越冰封谷，前往精灵国度。在落叶城遇到暴雪提前降临，他决定背着薇薇安穿越冰雪路径。途中他通过自虐式训练获得寒冷抗性专长，薇薇安则展示出三级术士二级巫师的实力——无咒语瞬发八颗奥术飞弹。进入元素祭坛时，索伦击杀土元素，临阵突破领悟反射闪避和灵活移动，获得影武者进阶信息。祭坛塌陷后岩石巨人追杀，他们带着价值五六千金德勒的宝石逃离。

# 深渊主宰卷五 · 恶魔领主的崛起（ch560-ch711）

索伦的恶魔分身索隆抵达古伦巴竞技场时，第一眼看到的是被钉死在石柱上的巴洛炎魔尸体——这是恶魔领主米亚米德-灾币的威慑。索隆支付三枚灾币给夜巫购买情报，无视周围恶魔的嘲讽，径直向额头印有奴隶印记的魅魔侍女报名参加死亡竞技挑战赛。与此同时，索伦本体在摩多城领悟了专长【移动施法】，探视薇薇安时发现她额头浮现高等恶魔刺青，恐惧魔女的力量正在加速苏醒。索伦向歌莉娅告别，接过她赠予的改良吉斯银刃，承诺半个月后返回，随后前往无底深渊第三层遗忘之地。
