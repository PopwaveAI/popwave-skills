# 旧屋新主、小巷匕首，以及那两个必须死的人

新住处比之前的破屋好不了多少——至少不漏雨。

索伦把薇薇安安顿在墙角最干燥的位置，翻出半条毯子铺在地上。

“坐这里。”

薇薇安顺从地坐下，膝盖蜷起来，背靠着墙。索伦从墙角摸出一块黑面包，掰成几块，放到一个豁口的碗里。

“我出去找点水。”

他拎起铁皮桶，走出门。

井绳断了一截，他弯腰拉了拉，够长。绞上水来，拎着桶往回走。

回到屋里，他把水倒进碗里，面包块浮起来，慢慢泡软。

“吃吧。”

薇薇安接过碗，低头用指尖夹起一块泡软的面包塞进嘴里，慢慢嚼着，腮帮子鼓起来又扁下去。嚼完一口，她又夹起第二块，蘸了蘸碗底的水，再塞进嘴里。

她吃到第四块的时候停下来，抬头看他。

“哥哥不吃吗？”

“我吃过了。”他说。

薇薇安盯着他看了一会儿，低下头，把碗里最后两小块面包囫囵吞进去，然后端起碗，把泡面包的水也喝干净了。

“我吃饱了。”

“躺下睡。”

薇薇安挪了挪身子，半躺在毯子上。索伦脱下外衣叠了叠，垫在她头底下。

她闭上眼。

索伦坐在她身边，背靠着墙。听着薇薇安的呼吸从急促慢慢变深，变成均匀的轻鼾——她睡熟了。

他等了一盏茶的时间，确认她不会醒。

然后他站起身，摸到床板底下那个暗格。

匕首还在。

他握在手里掂了掂，刀刃在昏暗里泛起一线白光。不够快，但够用。

他从暗格里又摸出一个铁片——三寸长，磨了一条刃边——塞进左袖里。然后从墙角拎出一根短铁棍，掂了掂重量，插在后腰上。

他把外褂往身上拉了拉，盖住铁棍和匕首。

看了一眼地上睡着的薇薇安。

她蜷缩着，呼吸很轻，眉眼舒展着。

索伦转身，推门出去，反手把门带上。

夜色已经落透了。

巷子里没有灯，只有远处几间酒馆窗口透出的昏黄光晕。他贴着墙根走，步子压得很轻，鞋子踩在湿漉漉的石板上几乎无声。

他拐过第三个弯，火炉酒馆就在前面。

酒馆的招牌挂在一根生锈的铁杆上，被风吹得嘎吱响。门口站着两个人——一个胖子，靠在门框上剔牙；另一个瘦子蹲在台阶上，手里捏着一个酒壶，正往嘴里灌。

索伦没看他们。他贴着墙，拐进了酒馆对面的小巷。

巷子很窄，两个人并肩都走不过去。他踩到一块骨头——咔嚓一声断了。

他蹲下来。

酒馆里传来摔酒杯的声音和哄笑声。

然后，后门吱呀一声开了。

一个人影晃出来——肥胖，秃顶，走路时重心在左右两脚之间来回倒换。他嘴里含着什么东西，嚼了几下，吐在地上。

巴拉斯。

他往前走了三步，步子虚浮，像喝了酒。

然后他停下来。

索伦动了。

他从墙上弹出去，三步落地的动作极为连贯——第一步蓄在左脚，第二步重心前移到右脚，第三步膝弯弹出整个人蹿出去，匕首握在身体右侧，刀刃朝上，从下巴斜着往耳根的方向刺。

巴拉斯还没反应过来。

匕首刺进去了。

刃尖破开皮肉，穿过下颌，往上斜走，刺进颅腔底部。

巴拉斯喉咙发出一声含混的气音。他的身体僵了一瞬间——腿在抖，手指攥紧又松开——然后整个人往侧面倒下去，嘭的一声砸在地上，地上的泥水溅了一墙。

【击杀提示：巴拉斯死亡。获得杀戮经验：15点。】

索伦拔刀，血从伤口喷出来，溅在他手腕和袖口上。

他没停。

他转身，沿来路跑出小巷，拐过一个弯，步子踩得重了些，故意弄出声响。他冲过酒馆门口，冲进另一边的大路。

卡诺波还在酒馆门口——那个脸上有疤的男人刚站起来，正准备往菜市场的方向走。

他看到索伦冲过来，愣了一下。

然后他看到索伦袖口的血迹。

他的眼神变了——不是愤怒，是警觉。他伸手要摸腰间的短刀，但索伦已经冲到他面前五步的距离。

卡诺波没有后退，他拔出短刀，横在身前，左脚后撤半步摆出防御姿态，膝盖微弯压低重心。

“你他妈——”

索伦没有减速。

他冲进卡诺波的防御范围，左手的钉子在卡诺波还没反应过来的瞬间，已经抵住对方腰侧软肋——那根尖钉贴着肋骨边缘戳进去。卡诺波浑身一颤，左手本能地往下按，短刀歪斜。

索伦的匕首同时到了——从下往上，挑开卡诺波右手的护肘，锋刃顺着小臂内侧的肌理一路拉开。

血涌出来。

卡诺波惨叫。

短刀从手里滑落，啪地掉在石板上。

索伦没有收手。他踏前一步，身体撞进卡诺波怀里，左手的钉子拔出来，又一下送进对方的右腿根，穿过肌肉的缝隙卡在髋骨的边缘。

卡诺波的右腿撑不住了，膝盖一软，整个人往地上跪倒。

他跪在地上，双手撑地，血顺着小臂淌下来，在石板的缝隙间蜿蜒流淌。

“你——你杀了巴拉斯——”

索伦蹲下来，匕首平平地搁在他眼前。

“你知道我为什么杀他。”

卡诺波喘着粗气，额头上全是汗，嘴唇发白。

“你妹妹——”

“还打她主意吗？”

卡诺波没说话。

索伦伸手，把匕首转了半圈，刀尖朝下，稳稳地刺进卡诺波左肩锁骨上方的沟壑里，穿过三角肌的边缘，刀尖嵌在肩胛骨和锁骨之间的缝隙中。

卡诺波惨叫，声音在巷子里回荡。

索伦拔刀。

【击杀提示：卡诺波死亡。获得杀戮经验：15点。】

索伦把刀在卡诺波的衣服上擦了擦，站起身。

他弯腰，翻了两具尸体的口袋。巴拉斯身上有六个银德勒和一把铜币，卡诺波腰袋里有一个刻着粗糙的圣徽的木牌和十三枚银德勒。

他把钱收好，把圣徽扔进旁边的水沟。

站起身，他转身离开。

夜色如旧。

他没有直接回住处——拐了两个弯，把匕首在水桶里洗了一遍，又用手掬水把袖口搓了搓，搓掉表面的血渍。

他等了一盏茶的时间，确认没人跟上来。

然后他绕回住处。

推开门。

薇薇安还在熟睡，姿势几乎没有变过——蜷缩着，头枕着他的外衣，呼吸均匀。

索伦把匕首放回暗格，铁棍靠在墙角，然后坐下来，背靠着墙。

他看了一眼窗外。

天快亮了。

【杀戮经验：30点】

【职业经验+30，平民LV1→平民LV1（30/100）】

他闭上眼，休息了一会儿。然后他睁开眼，看了一眼薇薇安。

她睡得正香，嘴角微微翘着——大概在做梦。

一切都会好起来的。
