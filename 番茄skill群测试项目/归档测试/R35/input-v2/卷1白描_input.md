## 地理地图的设计规律

### 1. 每幕都有1个"枢纽地标"

- Act1枢纽：德鲁伊树林（所有势力交汇）
- Act2枢纽：月出之塔（教派基地）
- Act3枢纽：下城区（所有支线收束）

### 2. 每个区域都有"明暗两条路径"

- Act1：地面路径（荒野）+ 地下路径（幽冥地域）
- Act2：塔外路径（诅咒之地）+ 塔内路径（渗透）
- Act3：城内路径（政治）+ 地下路径（夺心魔巢穴）

### 3. 摄像头移动遵循"枢纽辐射式"

```
枢纽地标 → 辐射探索周边 → 回枢纽触发主线 → 前往下一幕
```

不是线性A→B→C，而是"枢纽驻扎+辐射探索"。

### 4. 区域难度梯度

| 区域 | 主角推荐等级 | 怪物强度 |
|------|----------|---------|
| 荒野 | LV1-4 | 地精/强盗/低阶亡灵 |

## 反派强度梯度（决定成长gap）

### 三层反派（对应三层力量阶梯）

### 三大反派的关系

```

    ↕ 竞合关系

```

**设计精髓**：反派强度不是线性递增，而是"小反派是大反派的附庸"，每次击败小反派都能揭示大反派的一层阴谋。

## 金手指的力量加成（索伦注入BG3的对应）

### BG3原生金手指：夺心魔幼体

- **能力**：能操控感染者、汲取宿主记忆、变形改造
- **限制**：会被主脑召唤回去、会逐渐控制宿主意志
- **成长路径**：感染→抵抗→反向利用→吞噬/驱逐

### 深渊主宰金手指映射（索伦注入BG3时）

| BG3金手指 | 索伦金手指 | 映射逻辑 |
|----------|----------|---------|
| 夺心魔幼体感染 | 游戏数据流融合 | 都是"外来寄生意识" |
| 幼体带来的战术感知 | 系统面板【】内联 | 都是"信息层加成" |
| 可选择吞噬/驱逐幼体 | 可选择升级/保持职业 | 都是"自主选择成长路径" |
| 幼体被主脑召唤的威胁 | 圣者浩劫的末日倒计时 | 都是"时间压力源" |

### 索伦金手指的核心机制

```
杀戮经验值（可分配） → 升级职业/属性/专长
   ↓
属性突破20点 → 超凡门槛（再生/超凡反射等天赋）
   ↓
击杀神性生物 → 融合神性值（逐步向神级靠拢）
   ↓
神职融合 → 最终成为死神
```

---

## 力量地图的设计规律

### 1. 每个力量节点都绑定剧情事件

不是"升级到LV5就学会三阶法术"，而是"在地精营地战役中杀出经验，学会三阶法术"。

**力量节点 = 事件节点 + 属性变化**

### 2. 反派强度决定主角成长速度

主角从LV1到LV12的成长速度，取决于反派的强度梯度：
- 小反派LV5 → 主角必须LV5才能打
- 中反派LV8 → 主角必须LV8才能打
- 大反派LV12+ → 主角必须LV12+才能打

**每个反派就是主角力量曲线上的一个"检查点"**。

### 3. 金手指提供的"作弊"是缩短gap的关键

- 普通职业者：LV1→LV12需要数十年训练
- 索伦（有金手指）：LV1→LV12通过杀戮经验可在数月内完成

**金手指的本质是"加速力量地图的推进速度"**。

主角从势力金字塔的底层，逐步爬到顶层。

## 危机总览

BG3的危机设计是"四层叠加+时间倒计时"：

```
终极危机：主脑觉醒/夺心魔种族入侵多元宇宙
   ↑
深层危机：至上真神教派操控博德之门
   ↑  
表层危机：月出之塔教派扩张+幼体感染扩散
   ↑
个人危机：主角被感染+妹妹需要保护
```

**主角从个人危机开始，逐层揭示更深的危机，最终面对终极危机**。

## 危机时间线（三幕式倒计时）

### Act 1：感染扩散期（倒计时启动）

| 时间节点 | 危机事件 | 主角感知 |
|---------|---------|---------|
| 第1天 | 鹦鹉螺坠毁，主角被幼体感染 | "我脑子里有东西在动" |
| 第3天 | 感染者开始出现异变征兆 | "其他感染者变成了夺心魔" |
| 第7天 | 感染进入第二阶段，可以感知主脑 | "主脑能召唤我" |
| 第14天 | 感染完成倒计时，即将转化 | "必须找到解药" |

**个人危机**：主角自己被感染，倒计时启动。

### Act 2：教派扩张期（危机升级）

| 时间节点 | 危机事件 | 主角感知 |
|---------|---------|---------|

**终极危机**：主脑如果完全觉醒，整个费伦大陆将被夺心魔种族吞噬。

## 危机地图的设计规律

### 1. 每层危机都有"时间倒计时"

- 个人危机：14天转化倒计时
- 表层危机：教派扩张速度
- 深层危机：戈塔什的政变计划
- 终极危机：主脑觉醒时刻

**倒计时给主角施加"必须行动"的压力**。

### 2. 危机之间有"因果链"

```
主脑觉醒（终极）
   ↑ 戈塔什为它准备宿主
戈塔什政变（深层）
   ↑ 凯瑟里克为他在抓人
月出之塔抓人（表层）
   ↑ 地精教派为塔楼服务
地精抓难民（个人危机放大）
   ↑ 主角介入难民冲突
主角被感染（起点）
```

**每层危机都是下一层危机的原因，也是上一层危机的结果**。

### 3. 危机推进主角成长

- 感染倒计时 → 必须找解药 → 接触德鲁伊/月出之塔
- 教派扩张 → 必须对抗月出之塔 → 联合最后之光
- 戈塔什政变 → 必须去博德之门 → 联合盗贼公会
- 主脑觉醒 → 必须决战 → 吞噬/驱逐幼体

**危机是主角行动的核心驱动力**。

### 4. 危机揭示"更大的世界"

```
Act1揭示：月出之塔存在（世界比想象中大）
Act2揭示：戈塔什是幕后黑手（势力比想象中深）
Act3揭示：主脑才是终极敌人（敌人比想象中强）
```

每次揭示都让主角（和读者）意识到"世界比想象中危险"。
