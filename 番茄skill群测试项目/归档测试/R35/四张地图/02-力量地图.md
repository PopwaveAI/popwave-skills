# 力量地图 · 博德之门3

> R35测试 | 第二张地图
> 来源：基于博德之门3游戏结构反向拆解
> 作用：决定主角成长曲线，反派的强度gap

---

## 力量阶梯总览

BG3采用D&D 5e规则，等级LV1-LV12，力量成长分5个阶梯：

```
LV1-3：起步期（凡人期）
  ↓
LV4-5：超凡门槛（首次觉醒职业特性/专长）
  ↓
LV6-7：中阶（多职业解锁/三阶法术）
  ↓
LV8-10：高阶（传奇装备/五环法术/宿敌对决）
  ↓
LV11-12：巅峰（传奇专长/六环法术/封神之战）
```

## 反派强度梯度（决定成长gap）

### 三层反派（对应三层力量阶梯）

| 反派层级 | 代表角色 | 力量等级 | 出现时机 | 致命弱点 |
|---------|---------|---------|---------|---------|
| **低层** | 明萨拉（地精首领） | LV5 | Act1 | 过度自信，单兵作战 |
| **中层** | 凯瑟里克·索恩（月出之塔教主） | LV8-10 | Act2 | 莎尔背叛+复活仪式破绽 |
| **高层** | 戈塔什公爵（大恶魔化身） | LV12+ | Act3 | 契约绑定+野心膨胀 |
| **隐藏终极** | 主脑（夺心魔女王） | 神级 | 最终战 | 依赖宿主+银剑破盾 |

### 三大反派的关系

```
戈塔什公爵（人类契约大恶魔）
    ↕ 竞合关系
凯瑟里克·索恩（死亡骑士复活）
    ↕ 上下级关系  
明萨拉（地精教派首领，月出之塔附庸）
```

**设计精髓**：反派强度不是线性递增，而是"小反派是大反派的附庸"，每次击败小反派都能揭示大反派的一层阴谋。

## 金手指的力量加成（索伦注入BG3的对应）

### BG3原生金手指：夺心魔幼体

- **能力**：能操控感染者、汲取宿主记忆、变形改造
- **限制**：会被主脑召唤回去、会逐渐控制宿主意志
- **成长路径**：感染→抵抗→反向利用→吞噬/驱逐

### 深渊主宰金手指映射（索伦注入BG3时）

| BG3金手指 | 索伦金手指 | 映射逻辑 |
|----------|----------|---------|
| 夺心魔幼体感染 | 游戏数据流融合 | 都是"外来寄生意识" |
| 幼体带来的战术感知 | 系统面板【】内联 | 都是"信息层加成" |
| 可选择吞噬/驱逐幼体 | 可选择升级/保持职业 | 都是"自主选择成长路径" |
| 幼体被主脑召唤的威胁 | 圣者浩劫的末日倒计时 | 都是"时间压力源" |

### 索伦金手指的核心机制

```
杀戮经验值（可分配） → 升级职业/属性/专长
   ↓
属性突破20点 → 超凡门槛（再生/超凡反射等天赋）
   ↓
击杀神性生物 → 融合神性值（逐步向神级靠拢）
   ↓
神职融合 → 最终成为死神
```

## 力量地图的设计规律

### 1. 每个力量节点都绑定剧情事件

不是"升级到LV5就学会三阶法术"，而是"在地精营地战役中杀出经验，学会三阶法术"。

**力量节点 = 事件节点 + 属性变化**

### 2. 反派强度决定主角成长速度

主角从LV1到LV12的成长速度，取决于反派的强度梯度：
- 小反派LV5 → 主角必须LV5才能打
- 中反派LV8 → 主角必须LV8才能打
- 大反派LV12+ → 主角必须LV12+才能打

**每个反派就是主角力量曲线上的一个"检查点"**。

### 3. 金手指提供的"作弊"是缩短gap的关键

- 普通职业者：LV1→LV12需要数十年训练
- 索伦（有金手指）：LV1→LV12通过杀戮经验可在数月内完成

**金手指的本质是"加速力量地图的推进速度"**。
