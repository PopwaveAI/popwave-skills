# 势力地图 · 博德之门3

> R35测试 | 第三张地图
> 来源：基于博德之门3游戏结构反向拆解
> 作用：决定主角和谁打、和谁结盟，势力演变推动剧情

---

## 势力全景图

BG3的势力结构是"金字塔式三层"：

```
顶层：至上真神教派（戈塔什+凯瑟里克+主脑）
   ↓ 操控
中层：各区域地头蛇（地精三首领/月出之塔教徒/钢铁卫士）
   ↓ 附庸
底层：难民/流民/被感染者（主角也是其中一员）
```

## 势力地图的设计规律

### 1. 势力三阶结构：顶层-中层-底层

每个区域都有：
- **顶层势力**：掌控大局的反派（地精教派为月出之塔服务）
- **中层势力**：地头蛇（德鲁伊集会/最后之光酒馆）
- **底层势力**：流民/难民（提夫林/被感染者）

主角从底层起步，逐层向上碰撞。

### 2. 每个势力都有"双重身份"

- 地精教派：表面抓难民，实为月出之塔服务
- 钢铁卫士：表面维护秩序，实为戈塔什的私兵
- 戈塔什：表面公爵，实为大恶魔化身

**每个势力都有"读者知道但角色不知道"的隐藏信息**。

### 3. 势力演变推动剧情

```
Act1：主角介入地精-难民冲突 → 揭示月出之塔阴谋
Act2：主角联合最后之光对抗塔楼 → 揭示戈塔什幕后
Act3：主角联合盗贼公会推翻戈塔什 → 揭示主脑才是终极敌人
```

每次势力碰撞都揭示一层更深的阴谋。

### 4. 主角的身份转变

```
Act1：被感染的难民（被动）
Act2：抵抗者的盟友（半主动）
Act3：博德之门的拯救者/新公爵（主动）
```

主角从势力金字塔的底层，逐步爬到顶层。
