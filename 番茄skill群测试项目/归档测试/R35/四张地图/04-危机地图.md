# 危机地图 · 博德之门3

> R35测试 | 第四张地图
> 来源：基于博德之门3游戏结构反向拆解
> 作用：决定世界怎么推着主角走，时间压力源

---

## 危机总览

BG3的危机设计是"四层叠加+时间倒计时"：

```
终极危机：主脑觉醒/夺心魔种族入侵多元宇宙
   ↑
深层危机：至上真神教派操控博德之门
   ↑  
表层危机：月出之塔教派扩张+幼体感染扩散
   ↑
个人危机：主角被感染+妹妹需要保护
```

**主角从个人危机开始，逐层揭示更深的危机，最终面对终极危机**。

---

## 危机时间线（三幕式倒计时）

### Act 1：感染扩散期（倒计时启动）

| 时间节点 | 危机事件 | 主角感知 |
|---------|---------|---------|
| 第1天 | 鹦鹉螺坠毁，主角被幼体感染 | "我脑子里有东西在动" |
| 第3天 | 感染者开始出现异变征兆 | "其他感染者变成了夺心魔" |
| 第7天 | 感染进入第二阶段，可以感知主脑 | "主脑能召唤我" |
| 第14天 | 感染完成倒计时，即将转化 | "必须找到解药" |

**个人危机**：主角自己被感染，倒计时启动。

### Act 2：教派扩张期（危机升级）

| 时间节点 | 危机事件 | 主角感知 |
|---------|---------|---------|
| 月出之塔战役 | 凯瑟里克复活亡妻的仪式失败 | "这个教派在搞什么？" |
| 阴影诅咒扩散 | 诅咒之地扩大，吞噬周边 | "世界在腐坏" |
| 钢铁卫士集结 | 机甲军团向博德之门移动 | "有什么大事要发生" |
| 月亮坠落异象 | 天上月亮出现裂痕（诸神异象） | "更高层的力量在动" |

**深层危机**：至上真神教派的阴谋逐渐浮现。

### Act 3：主脑觉醒期（终极危机）

| 时间节点 | 危机事件 | 主角感知 |
|---------|---------|---------|
| 博德之门被围 | 戈塔什发动政变，自封大公 | "城市沦陷" |
| 夺心魔巢穴暴露 | 地下发现主脑培育基地 | "他们一直在我们脚下" |
| 主脑觉醒 | 主脑突破封印，开始召唤夺心魔大军 | "整个城市都要被感染" |
| 终极决战 | 主角必须选择：吞噬幼体对抗主脑/驱逐幼体保持人性 | "我是什么？" |

**终极危机**：主脑如果完全觉醒，整个费伦大陆将被夺心魔种族吞噬。

## 危机地图的设计规律

### 1. 每层危机都有"时间倒计时"

- 个人危机：14天转化倒计时
- 表层危机：教派扩张速度
- 深层危机：戈塔什的政变计划
- 终极危机：主脑觉醒时刻

**倒计时给主角施加"必须行动"的压力**。

### 2. 危机之间有"因果链"

```
主脑觉醒（终极）
   ↑ 戈塔什为它准备宿主
戈塔什政变（深层）
   ↑ 凯瑟里克为他在抓人
月出之塔抓人（表层）
   ↑ 地精教派为塔楼服务
地精抓难民（个人危机放大）
   ↑ 主角介入难民冲突
主角被感染（起点）
```

**每层危机都是下一层危机的原因，也是上一层危机的结果**。

### 3. 危机推进主角成长

- 感染倒计时 → 必须找解药 → 接触德鲁伊/月出之塔
- 教派扩张 → 必须对抗月出之塔 → 联合最后之光
- 戈塔什政变 → 必须去博德之门 → 联合盗贼公会
- 主脑觉醒 → 必须决战 → 吞噬/驱逐幼体

**危机是主角行动的核心驱动力**。

### 4. 危机揭示"更大的世界"

```
Act1揭示：月出之塔存在（世界比想象中大）
Act2揭示：戈塔什是幕后黑手（势力比想象中深）
Act3揭示：主脑才是终极敌人（敌人比想象中强）
```

每次揭示都让主角（和读者）意识到"世界比想象中危险"。

## 对应到深渊主宰的危机地图（注入索伦时）

| BG3危机层级 | 深渊主宰对应 | 映射逻辑 |
|------------|------------|---------|
| 主角被感染（个人） | 索伦融合游戏数据流 | 都是"外来意识入侵" |
| 月出之塔抓人（表层） | 琥珀城剥皮事件 | 都是"异常事件频发" |
| 戈塔什政变（深层） | 吸血鬼神子封神 | 都是"势力格局巨变" |
| 主脑觉醒（终极） | 圣者浩劫/诸神陨落 | 都是"世界级末日" |

### 深渊主宰的危机倒计时

```
倒计时1：圣者浩劫倒计时（15年→逐步缩短）
倒计时2：薇薇安魔神觉醒倒计时（沉睡→逐步恶化）
倒计时3：恐惧魔神复活倒计时（神力回流→逐渐累积）
```

**三个倒计时同时运行，主角必须同时应对**。

## 索伦注入BG3的危机起点

如果索伦被放入BG3世界，他的危机起点是：

### 个人危机（最紧迫）
- 索伦被夺心魔幼体感染（替代游戏数据流）
- 薇薇安需要治疗药剂（保留原设定）
- 索伦预知"主脑将在3个月内觉醒"（游戏数据流提供预警）

### 表层危机（中等紧迫）
- 月出之塔教派开始抓难民
- 琥珀城（对应BG3的荒野）出现剥皮事件
- 地精教派向德鲁伊树林推进

### 深层危机（隐藏）
- 戈塔什在博德之门布局政变
- 钢铁卫士在地下集结
- 主脑的意志开始在感染者的梦境中浮现

### 终极危机（伏笔）
- 主脑觉醒将导致费伦大陆沦陷
- 诸神不会干预（圣者浩劫的BG3版本）
- 只有主角能阻止（因为有游戏数据流预知）
