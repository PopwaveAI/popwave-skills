# 推演过程 · 四张地图叠加法

> R35测试 | 推演过程文件
> 作用：展示剧情白描是如何从四张地图推演出来的，每个L2单元卡都标注四张地图的坐标

---

## 推演方法论

### 核心公式

```
剧情白描 = L2单元卡（事件骨架） × 四张地图叠加 × 因果链推理
```

### 推演步骤（对每个L2单元卡执行）

1. **取L2单元卡**：一句话单元描述
2. **查地理地图**：这事件发生在哪？摄像头在哪？
3. **查力量地图**：主角此时等级多少？反派强度多少？gap多大？
4. **查势力地图**：主角和谁碰撞？和谁结盟？势力格局如何演变？
5. **查危机地图**：此时世界在发生什么？倒计时推进到哪了？
6. **因果链串联**：把5层信息用因果链串成叙事
7. **标注信息差**：哪些是"读者知道但角色不知道"

## L2单元卡推演（BG3-ACT1，对应深渊主宰卷1）

### 单元1：苏醒与起步（ch001-ch010）

**L2单元卡**：索伦在贫民窟苏醒，融合游戏数据流，为保护妹妹，以一级盗贼之身连续击杀贫民区威胁，获得初始生存资本。

**地理坐标**：琥珀城贫民窟郊外→坠毁点破屋→科尔帮据点→晨曦神殿避难所

**力量坐标**：
- 起点：平民LV1/盗贼LV1，4HP，敏捷19
- 终点：盗贼LV2，敏捷突破20超凡门槛，HP恢复
- 反派强度：巴拉斯LV2，卡诺波LV3，科尔帮主LV4

**势力坐标**：
- 起点：索伦无势力，科尔帮在附近搜索
- 碰撞：索伦 vs 科尔帮（低层势力碰撞）
- 终点：索伦清除科尔帮，获得"割喉者"绰号，进入月出之塔教派视野

**危机坐标**：
- 个人：感染倒计时14天开始
- 表层：科尔帮为月出之塔抓难民
- 深层：戈塔什在博德之门布局（主角不知道）
- 终极：主脑在地下沉睡（主角不知道）

**信息差标注**：
- 读者知道：戈塔什是幕后黑手，主脑是终极敌人
- 索伦知道：感染倒计时+主脑3个月后觉醒（数据流预知）
- 索伦不知道：戈塔什的存在、薇薇安的魔神神子身份

**因果链串联**：
索伦苏醒（地理：坠毁点）→ 发现自己被感染+融合数据流（力量：LV1，预知激活）→ 发现薇薇安被科尔帮欺负（势力：低层碰撞）→ 击杀巴拉斯+卡诺波（力量：升至LV2）→ 清除科尔帮（势力：低层势力清除，"割喉者"名号建立）→ 察觉感染在扩散（危机：倒计时启动）→ 前往晨曦神殿安置薇薇安（地理：转移到避难所）

### 单元2：藏宝洞与蛇魔（ch011-ch016）

**L2单元卡**：索伦为获取超凡装备独闯乱葬岗下藏宝洞，击杀骷髅僵尸后，不幸召唤出三阶深渊蛇魔，以弱胜强，浴血搏杀将其斩杀。

**地理坐标**：乱葬岗入口→藏宝洞第一层（骷髅区）→藏宝洞第二层（幽灵密室）→藏宝洞最深处（蛇魔祭坛）

**力量坐标**：
- 起点：盗贼LV2，敏捷20（刚突破超凡）
- 终点：盗贼LV4，获得1800点经验
- 反派强度：骷髅LV1，幽灵LV3，深渊蛇魔LV6（三阶）

**势力坐标**：
- 起点：索伦独行
- 碰撞：索伦 vs 奥术帝国遗物（万象无常牌）vs 深渊蛇魔
- 终点：索伦获得万象无常牌+蛇魔战利品，但暴露了位置给"无底深渊"势力

**危机坐标**：
- 个人：感染倒计时第10天
- 表层：月出之塔教派察觉"割喉者"的存在
- 深层：戈塔什的间谍开始追踪索伦
- 终极：蛇魔召唤仪式意外激活了主脑的一个分身（主角不知道）

**信息差标注**：
- 读者知道：万象无常牌是奥术帝国遗物，背后有更大秘密
- 索伦知道：蛇魔是三阶生物，挑战等级远超自己（数据流提示）
- 索伦不知道：蛇魔召唤与主脑有关联，薇薇安的魔神意志在蠢动

**因果链串联**：
索伦安置薇薇安（地理：晨曦神殿）→ 独自前往乱葬岗（地理：地下入口）→ 击杀骷髅+幽灵（力量：LV2稳定）→ 发现万象无常牌（势力：获得奥术帝国遗物）→ 触发蛇魔召唤（危机：意外激活主脑分身）→ 濒死突破LV4（力量：超凡+1）→ 击杀蛇魔（力量：获得1800经验）→ 离开藏宝洞（地理：返回地面）

### 单元3：商队之行（ch017-ch024）

**L2单元卡**：索伦带薇薇安加入北地女巫歌莉娅的商队，途中遭遇怪物、结识强者，并获得命运卡牌等关键物品。

**地理坐标**：晨曦神殿→荒野商路→荒野营地→遭遇掘地虫的峡谷→商队目的地

**力量坐标**：
- 起点：盗贼LV4
- 终点：兼职巫师LV1，获得四环法术卷轴
- 反派强度：掘地虫LV5（三阶群居）

**势力坐标**：
- 起点：索伦+薇薇安独行
- 结盟：北地女巫歌莉娅（潜在盟友，但动机不明）
- 碰撞：索伦 vs 掘地虫（自然威胁）
- 终点：商队抵达目的地，歌莉娅开始关注薇薇安

**危机坐标**：
- 个人：感染倒计时第7天，开始能感知主脑的意志
- 表层：荒野出现异常生物（掘地虫不在本地生态中）
- 深层：歌莉娅在占卜薇薇安的未来（主角不知道）
- 终极：薇薇安在商队途中第一次魔力失控（主角看到但不知道原因）

**信息差标注**：
- 读者知道：歌莉娅察觉到薇薇安的异常
- 索伦知道：感染在恶化，需要尽快找到解药
- 索伦不知道：薇薇安的魔力失控与魔神有关

**因果链串联**：
索伦决定离开琥珀城（地理：转移）→ 加入歌莉娅商队（势力：结盟）→ 商队遭遇掘地虫（危机：表层异常）→ 索伦指挥击杀获得经验（力量：升级）→ 兼职巫师（力量：解锁新职业）→ 薇薇安魔力失控（危机：隐藏炸弹开始滴答）→ 歌莉娅占卜看到血与火（信息差：读者知道薇薇安危险）

## 推演总结：白描的生成逻辑验证

### 三层信息叠加

每段白描叙事都包含三层信息叠加：
1. **地理层**：摄像头位置明确（坠毁点→藏宝洞→商路）
2. **力量层**：主角数值变化清晰（LV1→LV2→LV4→兼职巫师）
3. **势力层**：势力碰撞有起因后果（科尔帮→奥术遗物→北地女巫）
4. **危机层**：世界在同步推进（感染倒计时+月出之塔+戈塔什+主脑）

### 因果链的完整闭环

```
个人危机驱动行动 → 行动触发势力碰撞 → 势力碰撞揭示深层危机 → 深层危机推动主角成长 → 成长解锁新地理区域
```

### 信息差的节奏控制

每个单元卡都有"读者知道但角色不知道"的信息差：
- 单元1：读者知道戈塔什存在，索伦不知道
- 单元2：读者知道蛇魔与主脑有关，索伦不知道
- 单元3：读者知道薇薇安是魔神神子，索伦不知道

**信息差是白描的张力来源**。
