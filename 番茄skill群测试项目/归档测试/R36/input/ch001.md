# ch001 场景卡

## 章型：opening_shift（开篇启程章）

**核心事件**：索伦在鹦鹉螺号坠毁后苏醒，融合游戏数据流，击杀新生夺心魔，追踪妹妹脚印，击杀三只地精斥候，抵达德鲁伊树林难民营地，得知妹妹被地精抓走。

**主角状态**：
- 索伦：LV1，4/12HP，被夺心魔幼体感染（倒计时13天23小时）
- 金手指激活：属性面板、经验条、战斗分析、弱点提示

**场景序列**：

1. **坠毁苏醒**（鹦鹉螺号坠毁点，荒野沙滩）
   - 索伦在沙滩上醒来，视野浮现淡蓝色光字
   - 【状态更新：夺心魔幼体感染——倒计时13天23小时47分】
   - 看到鹦鹉螺号残骸燃烧，周围有尸体
   - 发现妹妹薇薇安失踪

2. **新生夺心魔遭遇**（沙滩）
   - 一个人类法师被感染转化，新生夺心魔扑向索伦
   - 【战斗分析：未成熟期夺心魔 | LV4 | 弱点：穿刺伤害针对幼体本体】
   - 索伦用淬毒匕首击杀，获得240点经验
   - 系统面板弹出一条提示：【击杀LV4夺心魔 | 获得经验值240点】

3. **追踪脚印**（沙滩→树林）
   - 发现薇薇安的鞋印被拖向东北方向树林
   - 数据流展开荒野地图：【地理识别：费伦大陆-剑湾-荒野区域】
   - 索伦跟着鞋印跑进树林

4. **击杀地精斥候**（树林中）
   - 三只地精正拖着一个昏迷的提夫林女孩，没有薇薇安
   - 【目标锁定：地精斥候x3 | LV2-LV3 | 弱点：听力敏锐但夜盲 | 建议战术：潜行背刺+远程清理】
   - 数据流给出战术路径，索伦利用19点敏捷潜行背刺，三分钟解决战斗
   - 获得战利品：德鲁伊树林地图

5. **抵达难民营地**（德鲁伊树林外围）
   - 找到了提夫林难民聚居地
   - 瓦洛（提夫林斥候）告知：地精教派在抓难民，送到月出之塔
   - 今天早上有地精拖着一群孩子往营地北边走了
   - 【任务更新：寻找薇薇安 | 线索指向：德鲁伊树林东北方向的地精营地】

**爽感闭环**：
- 压力：索伦被感染，妹妹失踪，倒计时紧迫
- 突破：LV1击杀LV4夺心魔，三分钟反杀三只地精
- 后果：获得经验值+地图+情报，明确下一步目标

**章末钩子**：北边是地精营地，那里有三个首领，妹妹很可能就在那里——而索伦只有一把匕首和一块伪造的身份牌。
