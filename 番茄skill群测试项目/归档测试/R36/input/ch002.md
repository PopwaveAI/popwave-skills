# ch002 场景卡

## 章型：reveal_hook（揭秘升级章）

**核心事件**：索伦伪装成月出之塔教徒混入地精营地，潜伏三天，发现哈尔辛被关押，摧毁咒文图腾释放哈尔辛，利用数据流战术击杀明萨拉，发现妹妹已被转送至幽暗地域灰矮人据点。

**主角状态**：
- 索伦：LV3→LV4，装备淬毒匕首+1
- 金手指：伪造文书自动成功、BOSS弱点分析、陷阱触发时间线

**场景序列**：

1. **伪装渗透**（地精营地外围）
   - 数据流提供三条路线：正面强攻/潜行刺杀/伪装渗透
   - 索伦用数据流伪造月出之塔身份牌：【技能检定：伪造文书 | 智力18+工具熟练 | 自动成功】
   - 混入地精营地，营地里腥臭冲天

2. **潜伏侦查**（地精营地内部）
   - 三天潜伏，收集情报，搜索薇薇安踪迹
   - 在大帐中见到明萨拉：【目标：明萨拉·班科 | 地精教派首领 | LV5 | 弱点：过度依赖神术护盾，真身暴露时移动力为零】
   - 祭坛上绑着一个半精灵男孩，但不是薇薇安

3. **解救哈尔辛**（地牢）
   - 地牢里发现被关押的德鲁伊哈尔辛，被诅咒变成熊形态
   - 数据流给出解咒方法：【解咒条件：摧毁三处咒文图腾 → 释放哈尔辛 → 获得盟友】
   - 潜行干掉看守，炸掉西侧/南侧/东侧三座图腾
   - 哈尔辛恢复半精灵形态，加入

4. **击杀明萨拉**（地精大帐）
   - 哈尔辛吸引正面火力，索伦利用数据流BOSS战时间线
   - 引明萨拉进入祭坛中央，触发隐藏陷阱【陷阱伤害：神圣能量爆发 | 对明萨拉造成3秒眩晕】
   - 哈尔辛一拳砸碎祭坛，神术护盾破碎的瞬间，索伦匕首穿过明萨拉后颈
   - 【击杀LV5明萨拉 | 获得经验值800点 | 等级提升：盗贼LV3 → LV4 | 解锁专长：健壮】

5. **发现密函**（大帐战场）
   - 明萨拉临死前说：“凯瑟里克……会替我复仇的……月出之塔……不会放过你们……”
   - 搜到转运记录：“7号容器——一个半精灵女童——已转送至幽暗地域的灰矮人据点，交由灰矮人之王格里姆波看管”
   - 信息差揭示：地精教派只是月出之塔的先锋军，凯瑟里克在幕后

**爽感闭环**：
- 压力：三天潜伏，随时可能暴露，妹妹不在营地
- 突破：成功释放哈尔辛，击杀LV5明萨拉，突击升级
- 后果：获得盟友+情报，但妹妹已被转送更危险的地方

**章末钩子**：幽暗地域——那片被真菌覆盖的黑暗地下世界，灰矮人之王格里姆波在等着，而月出之塔的审判官也在那里。
