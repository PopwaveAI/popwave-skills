# ch004 场景卡

## 章型：reveal_hook（揭秘升级章）

**核心事件**：索伦利用熔炉环境击杀格里姆波，与月出之塔审判官对决，从审判官尸体搜出关键情报——戈塔什公爵是幕后黑手，抵达月出之塔踏桥。

**主角状态**：
- 索伦：LV4→LV5，解锁兼职巫师路线
- 哈尔辛：LV6德鲁伊，薇薇安+老狗希斯安全

**场景序列**：

1. **击杀格里姆波**（熔炉核心区）
   - 格里姆波举战锤追击，索伦与哈尔辛边打边退
   - 数据流提示：【机会攻击窗口：熔炉顶部的挂钩链条可破坏→砸落熔铁桶→引爆酒桶区域】
   - 索伦甩出飞爪勾住链条，一刀砍断
   - 装满熔铁的桶砸落，溅起的火花点燃灰矮人随身携带的烈酒桶
   - 连环爆炸把格里姆波炸飞
   - 【击杀LV6格里姆波 | 获得经验值1200点 | 等级提升：LV4→LV5 | 解锁：兼职巫师路线】

2. **对决审判官**（据点外围）
   - 月出之塔审判官在爆炸中幸存，释放神术锁链缠住索伦的腿
   - 数据流显示：【神术锁链弱点：祈唤词"莎尔的阴影"——每5秒重复一次】
   - 四秒后审判官开口："莎尔的阴影……"
   - 索伦在第五秒准确掷出匕首，打断祈唤，锁链崩碎
   - 审判官被抓时还在喊："你对抗不了凯瑟里克大人！"
   - 索伦割断他的喉咙

3. **搜出关键情报**（据点废墟）
   - 从审判官尸体搜出一卷羊皮纸
   - 内容：月出之塔详细地图、凯瑟里克·索恩的复活仪式计划、以及一个名字——戈塔什公爵
   - 信息差揭示：月出之塔教主凯瑟里克只是棋子，真正的大BOSS是博德之门城的戈塔什公爵——至上真神教派的真正领袖，大恶魔契约的持有者
   - 薇薇安靠在索伦肩上，攥着他的衣角

4. **抵达月出之塔踏桥**（幽暗地域出口）
   - 幽暗地域出口通向月出之塔的踏桥
   - 索伦站在地下世界边缘，看着远处那座被阴影笼罩的黑塔
   - 数据流警告：【区域切换：即将进入月出之塔势力范围 | 建议：完成Act1准备 → 提升至LV6-7 → 招募足够盟友 → 确认薇薇安的安全】
   - 倒计时：6天19小时

**爽感闭环**：
- 压力：熔炉区爆炸混战，审判官神术锁链束缚
- 突破：LV6格里姆波被击杀，晋升LV5解锁巫师路线，打断审判官祈唤反杀
- 后果：获得关键情报（戈塔什），抵达月出之塔，倒计时紧迫

**章末钩子**：塔桥尽头，月出之塔的门缝里透出绿色邪光，一个低沉的声音在索伦脑中响起："你身体里的幼体……很有意思……凯瑟里克大人会喜欢你的。"——数据流同时刷新出一条红色警告：【主脑意识入侵检测中 | 抵抗检定：感知15】。
