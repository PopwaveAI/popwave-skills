# ch005 场景卡

## 章型：growth（成长蓄力章）

**核心事件**：索伦在月出之塔踏桥前做最后的准备，分配升级点数兼职巫师，哈尔辛讲述月出之塔的内部情报，薇薇安第一次展现出术士天赋（魔力波动），索伦做出决定——进入月出之塔。

**主角状态**：
- 索伦：LV5，解锁兼职巫师路线
- 哈尔辛：LV6德鲁伊
- 薇薇安：8岁，天生术士（魅力21），魔力开始波动

**场景序列**：

1. **战后休整**（月出之塔踏桥前，临时营地）
   - 索伦在幽暗地域出口的安全区域休整
   - 分配经验值，将盗贼升至LV5，申请兼职巫师
   - 数据流展示职业分支：【可选兼职：巫师→法术列表：一环法术x3/二环法术x1 | 咒法/防护/预言学派】
   - 索伦选择预言学派（与数据流预知能力协同）
   - 学习法术：护盾术、魔法飞弹、侦测魔法、镜影术

2. **哈尔辛的情报**（营地篝火旁）
   - 哈尔辛讲述月出之塔的内部结构：三层塔楼、地下监狱、顶层是凯瑟里克的研究室
   - 凯瑟里克·索恩——曾经是死亡骑士，现在被莎尔复活，在搞一个"复活仪式"
   - 戈塔什公爵的名字再次出现：哈尔辛说"戈塔什在博德之门城有钢铁卫士军团，月出之塔只是他的一个前哨站"
   - 信息差揭示：至上真神教派是一个跨区域组织，月出之塔只是冰山一角

3. **薇薇安的魔力波动**（营地）
   - 薇薇安在篝火边烤火，突然手心冒出一团蓝白色的微光
   - 她吓了一跳，但很快控制住，光芒消失
   - 哈尔辛察觉异常，询问索伦
   - 索伦用数据流扫描薇薇安：【检测到异常魔力波动 | 魔力源：未知 | 强度：超凡级 | 警告：与宿主体质不匹配】
   - 索伦意识到妹妹不是普通术士，但没告诉哈尔辛全部真相

4. **做出决定**（踏桥起点）
   - 索伦看着远处的黑塔，数据流显示任务路径
   - 【可选路线：穿越塔桥进入月出之塔→潜入营救剩余俘虏→对抗凯瑟里克→获取主脑坐标→前往博德之门城】
   - 薇薇安说："哥，后面的坏人还在追我们吗？"
   - 索伦把薇薇安放在老狗希斯背上，低声说："他们追不上。哥哥很快就能治好你。"
   - 哈尔辛拍了拍他的肩膀："月出之塔在前面，我在后面。你救了我的命，我现在还给你。"
   - 索伦没有回头，迈步走上塔桥

**爽感闭环**：
- 压力：倒计时6天19小时，凯瑟里克LV8-10在塔里等着，戈塔什藏在暗处
- 突破：兼职巫师，获得预言学派法术，薇薇安魔力觉醒（隐藏力量）
- 后果：获得月出之塔情报，明确下一步目标，但薇薇安的秘密越来越危险

**章末钩子**：塔桥尽头，月出之塔的门缝里透出绿色邪光，一个低沉的声音在索伦脑中响起："你身体里的幼体……很有意思……凯瑟里克大人会喜欢你的。"——而数据流同时刷新出一条红色警告：【主脑意识入侵检测中 | 抵抗检定：感知15 | 建议：立即撤退或接受恩泽】。
