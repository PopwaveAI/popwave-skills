# ch001 场景卡

## 章型：opening_shift（开篇启程章）

**核心事件**：阿米尔在恒河边跳河时穿越到纽约肯尼迪机场，系统激活，看到纽约势力地图，在哈莱姆被黑帮混混抢劫，用50美元买命，混混指引他去Bed-Stuy找大D要工作。

**主角状态**：
- 阿米尔：25岁，达利特，口袋200美元，系统LV1（派系面板激活）
- 普丽雅：22岁妹妹，跟着穿越，攥着哥哥衣角
- 倒计时：7天

**场景序列**：

1. **恒河边→肯尼迪机场**（穿越瞬间）
   - 阿米尔在恒河边闭眼准备跳河，睁开眼闻到柴油味
   - 普丽雅死死攥着他的衣角——妹妹跟着穿越了
   - 系统面板弹出，势力地图展开：哈莱姆标红/布鲁克林标黄/曼哈顿上东区标紫
   - 紫色代表"终极威胁"，系统没告诉他是什么

2. **哈莱姆街头遭遇**（第一个危险）
   - 阿米尔在哈莱姆找便宜旅馆
   - 三个黑帮混混拦路抢劫
   - 【派系面板已激活。哈莱姆毒枭"黑桃"的手下——忠诚度85%，实力LV2，矛盾点：对黑桃的分成不满】
   - 阿米尔用仅剩50美元买命
   - 混混嗤笑着说"去Bed-Stuy找大D要工作"——系统第一条路

3. **7天倒计时启动**
   - 阿米尔带着普丽雅在街头坐着，系统弹出警告
   - 【7天倒计时启动：必须在7天内挑起一场有理由的局部冲突，否则系统反噬】
   - 阿米尔看着陌生城市的霓虹灯，第一次意识到自己被系统绑定了

**爽感闭环**：
- 压力：异国他乡+语言不通+被抢劫+系统绑定+7天倒计时
- 突破：系统激活，看到势力地图，获得第一条路径
- 后果：明确下一步目标——去找大D

**章末钩子**：阿米尔看着系统地图上Bed-Stuy的黄色光点，普丽雅问他"哥，我们去哪？"阿米尔说："去一个叫大D的人那里找工作。"但系统面板闪烁着一行小字：【Bed-Stuy·大D·忠诚度72%，女儿患白血病，每月医药费1.5万美元】。
