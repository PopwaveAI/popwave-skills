# ch002 场景卡

## 章型：reveal_hook（揭秘升级章）

**核心事件**：阿米尔抵达Bed-Stuy找到大D，被大D当成FBI差点崩了，阿米尔打开系统展示大D帮派内部矛盾面板（四个小头目密谋背叛），阿米尔用"解决医药费+清除叛徒"的承诺策反大D成为第一个代理人。

**主角状态**：
- 阿米尔：LV1，资产0，7天倒计时还剩5天
- 大D：LV2帮派头目，80人，女儿白血病

**场景序列**：

1. **Bed-Stuy的破旧酒吧**（大D据点）
   - 阿米尔带着普丽雅走进大D的地下赌场
   - 大D的手下把他按在地上，大D拿枪指着他
   - "FBI派来的？你他妈知道这是什么地方吗？"

2. **系统面板展示**（反转时刻）
   - 阿米尔说："我不是FBI，我是来帮你的。你手下有四个人要背叛你。"
   - 大D不信，阿米尔让大D看系统面板
   - 【大D帮派·内部面板：忠诚度分布——二把手杰克68%/三把手托尼55%/四把手雷65%/五把手萨姆60%——四人均有叛变意图】
   - 大D震惊，因为杰克是他发小

3. **摊牌与策反**
   - 阿米尔直接摊牌："黑桃想吞你的地盘，马库斯想取代黑桃，你女儿医院账单一个月后就到期——你只有一个选项。"
   - "我能帮你解决医药费，但你要跟我干。"
   - 大D沉默了很久，问："你到底是什么人？"
   - 阿米尔说："一个在印度被你们这种人踩到骨子里的人。"

4. **签下代理人契约**
   - 大D成为阿米尔第一个代理人
   - 系统提示：【代理人部署完成·大D·忠诚度85%·代价：抽取10%赌场收入】
   - 系统升级提示：【LV2解锁·矛盾洞察功能——能看到势力内部真实矛盾】

**爽感闭环**：
- 压力：7天倒计时5天，被枪指着头
- 突破：系统展示+反向策反，获得第一个代理人
- 后果：获得LV2升级，但暴露了自己的存在

**章末钩子**：大D收起枪，递给阿米尔一杯威士忌："欢迎加入纽约，小子。"但系统面板上，曼哈顿上东区的紫色光点突然闪烁了一下——【检测到高层势力活动·阿什利家族·正在关注Bed-Stuy区域异常】。阿米尔没注意到这个提示，他正在喝第一口威士忌。
