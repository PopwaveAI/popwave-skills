# ch003 场景卡

## 章型：combat_reversal（战斗反击章）

**核心事件**：阿米尔利用黑桃二把手马库斯的背叛欲望，在黑桃的交易中故意泄露错误时间，让马库斯抢了黑桃的货。黑桃内部火并，马库斯带30人叛逃投奔大D。阿米尔吞并马库斯势力，LV3升级。

**主角状态**：
- 阿米尔：LV2，资产0，7天倒计时还剩3天
- 大D：代理人，80人
- 马库斯：黑桃二把手，忠诚度62%（被黑桃绿了）

**场景序列**：

1. **黑桃内部火并布局**
   - 阿米尔通过系统看到黑桃二把手马库斯忠诚度62%（黑桃睡了马库斯女人）
   - 阿米尔让大D派线人接触马库斯，透露"黑桃要干掉你"
   - 阿米尔故意泄露黑桃交易错误时间给马库斯

2. **截货行动**（哈莱姆仓库）
   - 马库斯带人抢了黑桃一批货
   - 黑桃发现后大怒，内部火并
   - 马库斯带30人叛逃，投奔布鲁克林大D

3. **阿米尔吞并马库斯势力**
   - 阿米尔在Bed-Stuy地下室接见马库斯
   - 马库斯不信这个印度小子，但大D在场压制
   - 阿米尔说："你女人被黑桃睡了，你地盘被黑桃抢了，你现在只有30个人和一箱货——跟我干，我帮你杀黑桃。"
   - 马库斯归顺
   - 系统提示：【吞并马库斯势力+30人·LV3升级·战利品分配功能解锁·资产：5万美元】

4. **第一次尝到甜头**
   - 阿米尔站在屋顶看纽约夜景
   - 回忆印度高种姓家族怎么碾碎他人生
   - 现在他成了那个操纵碾碎的人
   - 普丽雅问他："哥，你刚才让马库斯和黑桃火并，死了3个人，你不难过吗？"
   - 阿米尔说："在印度，我们这种人每天死成百上千，没人问过我们难不难过。"

**爽感闭环**：
- 压力：倒计时3天，必须挑起冲突
- 突破：LV3升级，资产5万，第一次吞并外部势力
- 后果：尝到"用矛盾杀人"甜头，心态开始黑化

**章末钩子**：阿米尔看着系统面板上黑桃的忠诚度暴跌至70%，但警察局长沃克的面板突然亮起——【沃克·警察局长·忠诚度面板：对老钱家族92%/对黑桃67%/对主角隐藏威胁感】。阿米尔第一次意识到，纽约的警察系统已经被上层势力渗透了。
