# ch004 场景卡

## 章型：reveal_hook（揭秘升级章）

**核心事件**：阿米尔通过大D的关系网接触到警察局长沃克，在一家意大利餐厅与沃克会面，用系统看到沃克的私生子在布朗克斯，阿米尔用这个把柄把沃克变成代理人。沃克告知阿米尔：黑桃的真正老板是华尔街的理查德，理查德在为曼哈顿老钱家族洗钱。

**主角状态**：
- 阿米尔：LV3，资产5万美元
- 沃克：LV5警察局长，60岁，私生子

**场景序列**：

1. **意大利餐厅会面**
   - 大D安排阿米尔和沃克在布鲁克林意大利餐厅会面
   - 沃克坐下："听说印度小子想跟我做生意？"
   - 阿米尔用系统扫描沃克
   - 【沃克·警察局长·LV5·忠诚度面板：对老钱家族92%（私生子保护费）/对黑桃67%（分红）/对主角隐藏威胁感——评估：阿米尔是外来变量，有威胁】

2. **摊牌私生子**
   - 沃克："你想要什么？"
   - 阿米尔："我在布朗克斯有个朋友，他住在145号街的3楼公寓，他叫汤米，今年12岁，他在上私立学校。"
   - 沃克脸色一变——汤米是他的私生子
   - 阿米尔："别紧张，我不是来威胁你的。我是来帮你解决后顾之忧的。汤米的学费我来付，但你得跟我干。"

3. **沃克归顺**
   - 沃克沉默良久，问："你到底是什么人？"
   - 阿米尔："一个想把纽约搅得天翻地覆的人。"
   - 沃克归顺，系统提示：【代理人部署完成·沃克·忠诚度78%·解锁LV4情报网络功能·资产：50万美元（警察局保护费分成）】

4. **信息差揭示**
   - 沃克告诉阿米尔一个情报：黑桃的真正老板不是哈莱姆人，而是华尔街的理查德
   - 理查德在为曼哈顿的老钱家族洗钱
   - "阿什利家族——你听过吗？他们控制纽约半个世纪了。"
   - 阿米尔第一次听到阿什利的名字
   - 系统面板上曼哈顿上东区的紫色光点开始闪烁

**爽感闭环**：
- 压力：LV3对LV5，力量差距
- 突破：策反警察局长，LV4升级，获得情报网络
- 后果：黑桃的幕后是华尔街，华尔街的幕后是老钱家族——揭示更深的敌人

**章末钩子**：阿米尔回到Bed-Stuy地下室，打开系统面板。屏幕上，阿什利家族的紫色光点第一次展开——【阿什利家族·曼哈顿上东区·势力评级LV8·家族成员：阿什利三世68岁（家主）/阿什利四世42岁（继承人）/理查德45岁（代理人）——真实控制力：华尔街17%+地产23%+政客人脉】。阿米尔盯着屏幕，普丽雅问他："哥，紫色是什么意思？"阿米尔说："那是我最终要干掉的人。"
