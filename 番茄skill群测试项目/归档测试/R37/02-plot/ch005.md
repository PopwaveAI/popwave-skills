# ch005 场景卡

## 章型：reveal_hook（揭秘升级章）

**核心事件**：阿米尔策划清除黑桃——通过沃克警察局内部行动+马库斯内部接应，突袭黑桃的哈莱姆总部。黑桃死亡前说出理查德的名字，阿米尔从黑桃尸体上找到华尔街密钥。系统升级LV5，解锁代理人部署功能。但阿米尔意识到，他刚踏入纽约时被抢劫的三个混混——他们的刺青指向阿什利家族的暗影部队。

**主角状态**：
- 阿米尔：LV4，资产50万美元
- 大D+马库斯：代理人，120人
- 沃克：警察局长代理人

**场景序列**：

1. **突袭黑桃总部**（哈莱姆褐石公寓）
   - 阿米尔协调三方力量：大D+马库斯正面突击，沃克警察局"意外"不巡逻
   - 系统给出精确时间窗口：【黑桃总部防御薄弱窗口：23:00-01:00·值班人员12人·火力等级中】
   - 突击队从后门进入，马库斯一枪打爆黑桃保镖的头
   - 阿米尔不在前线，他在2公里外的屋顶通过系统实时看面板

2. **黑桃之死**
   - 黑桃被围堵在豪华公寓顶层
   - 黑桃临死前对马库斯说："你以为你赢了？理查德会让你比死还惨。"
   - 马库斯一枪爆头，但黑桃临死前最后一句："理查德...阿什利..."
   - 阿米尔通过马库斯的耳机听到这两个名字

3. **搜出华尔街密钥**
   - 在黑桃保险箱找到：一本瑞士银行账户密码+一张华尔街私人俱乐部会员卡
   - 会员卡上印着"Wall Street Club·Richard·Membership #0451"
   - 系统面板弹出：【发现关键道具：华尔街私人俱乐部会员卡·持有者：理查德·阿什利·阿什利家族代理人】
   - 信息差揭示：理查德不只是一个代理人，他是阿什利家族的成员

4. **LV5升级+暗影部队揭示**
   - 系统提示：【LV5升级·解锁：代理人部署（在华尔街扶植代理人）·资产：500万美元（黑桃遗产转化）】
   - 阿米尔第一次有资格在曼哈顿设据点
   - 但系统同时弹出一条警告：【警告：追溯分析——阿米尔抵达纽约第一天遭遇的抢劫者·刺青分析·对应阿什利家族暗影部队·编号D-003·结论：阿米尔穿越到纽约第一天就被阿什利家族盯上了】

5. **阿米尔的震惊**
   - 阿米尔呆住
   - 他以为自己是猎人，其实他从第一天起就是猎物
   - 普丽雅问他怎么了
   - 阿米尔说："我们从头到尾都在被监视。"
   - 普丽雅："那我们怎么办？"
   - 阿米尔看着窗外的曼哈顿天际线："继续。既然他们盯着我们，那我就让他们看看什么叫真正的内战。"

**爽感闭环**：
- 压力：黑桃是LV3，阿米尔是LV4——不能亲自下场，必须靠代理人
- 突破：清除第一个大反派，LV5升级，资产500万
- 后果：发现自己从第一天就是猎物，猎人变成猎物的反转

**章末钩子**：阿米尔站在Bed-Stuy屋顶，系统面板上曼哈顿上东区的紫色光点越来越大。系统刷新出一条新提示：【阿什利家族·家主阿什利三世·正在召开紧急会议·议题：Bed-Stuy异常变量·处置方案：启动理查德专案组】。阿米尔第一次感到真正的压力——他不是在跟一个老钱家族斗，他是在跟整个美国老钱体系斗。
