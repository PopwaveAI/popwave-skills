# 警察局长听完一个名字，脸色比他制服还白

风还灌在阿米尔领口。

那行字浮在视野角落没散——【沃克·纽约警察局长·忠诚度面板：对老钱家族·92% / 对黑桃·67% / 对主角·隐藏威胁感。】

倒计时跳了一下：5天01小时57分。

阿米尔盯着那个深蓝色光点看了十秒。光点落在曼哈顿下城警察总部那栋灰楼的位置，像一颗钉子钉在地图上。92%。他在印度被主管扇耳光那天，他给那个主管的忠诚度也是90%往上——因为饭碗。

"对主角·隐藏威胁感"。阿米尔把这几个字又看了一遍。系统从来不给无用的标签。这意味着沃克已经注意到了他——一个外来变量。他要么先动手，要么被动手。

系统面板只能告诉他数字，不能告诉他怎么把数字变成一把刀。但阿米尔在瓦拉纳西的棚户区学过一件事——刀不在你手里的时候，你就得先让对方以为刀在他手里。

他把面板往大D那边推了半寸。

"认识？"大D凑过来，金链子晃了一下。

"警察局长。"阿米尔道，"黑桃背后的人。"

大D的脸沉了下去。他往地上吐了口痰，又用鞋底碾了碾，烟头在水泥地上蹭出一道黑印。

"沃克这老东西，"大D道，"我女儿去医院那天，他手下把我拦在急诊门口两个钟头。两个钟头。"

阿米尔没接话。他抬手把面板合上，转身往楼梯下走。屋顶的铁皮在他脚下嘎吱响，像踩在旧铁罐上。

"安排个饭局。"他边走边道，"我要见他。"

大D在身后停了一下。

"他不会见你的。"大D道。

"他会。"阿米尔道，"告诉他，我知道汤米。"

---

饭局是两天后。布鲁克林第八大道一家意大利餐厅，门口挂着褪色的绿白红三色旗，玻璃窗蒙着一层油烟。大D把阿米尔带进去时，厨房那边传来橄榄油烧蒜的焦香，混着老木头的潮味。沃克的人先到的——门口站着两个穿夹克的白人，腰间鼓着。

沃克已经坐在最里面的卡座。六十岁，灰西装，警徽别在胸口，手边一杯黑咖啡没动。他抬眼看了阿米尔一下，没站起来。

"听说印度小子想跟我做生意？"沃克道。声音很平，像在念一份报告。

阿米尔在他对面坐下。皮椅陷下去半寸，凉。他没有先开口。他抬手在桌面下虚点了一下，系统面板展开——蓝白色的光浮在桌布的阴影里，只有他能看见。

【沃克·警察局长·LV5·忠诚度面板：对老钱家族·92%（私生子保护费）/对黑桃·67%（分红）/对主角·隐藏威胁感——评估：阿米尔是外来变量，有威胁。】

LV5。比他高两级。阿米尔的后背绷了一下，又松开。在印度，高他两级的人他连正眼看的资格都没有。

"你想要什么？"沃克道。

阿米尔没看他。他盯着桌布上一道咖啡渍看了三秒，然后开口。

"我在布朗克斯有个朋友，"阿米尔道，"他住在145号街的3楼公寓，他叫汤米，今年十二岁，他在上私立学校。"

沃克的手停住了。那只握着咖啡杯的手指节发白，杯子在桌面上磕了一下，发出一声很轻的咚。他的另一只手在桌下握成了拳，又松开。他的目光往门口那两个白人保镖那边飘了一下，又收回来。

"你——"沃克道。

"别紧张。"阿米尔道，"我不是来威胁你的。我是来帮你解决后顾之忧的。"

他把面板往沃克那边转了半寸——当然沃克看不见，但这个动作让对面那个六十岁的老人看见了阿米尔眼睛里的东西。

"汤米的学费我来付。"阿米尔道，"但你得跟我干。"

---

餐厅里很安静。厨房的油烟机嗡嗡响，远处有辆垃圾车倒车，嘟嘟嘟的警报声透过玻璃传进来。沃克坐着没动，像一截被钉在卡座里的木头。

阿米尔等他。他端起自己面前那杯没动的水，喝了一口。水是凉的，杯壁有点滑，他的拇指在杯沿上蹭了一下。

很久之后，沃克开口。

"你到底是什么人？"他道。

阿米尔放下杯子。杯底在桌面上留下一圈水印。

"一个想把纽约搅得天翻地覆的人。"他道。

沃克的喉结上下动了一下。他抬手去解西装领口那颗扣子，解了两次才解开，手指在扣子上抖了一下。然后他把自己那杯黑咖啡端起来，一口喝干。咖啡杯落回桌面时，声音比刚才重。

"成交。"沃克道。

【代理人部署完成·沃克·忠诚度78%。】

【LV4解锁。新功能：情报网络——可调取NYPD内部数据库与街头线人网络。】

【资产：$500,000（警察局保护费分成）。】

面板在阿米尔眼前跳了三下。LV4。他从LV3到LV4只用了两天。五万美元到五十万美元，也只用了两天。他攥了一下拳头，指节咯咯响，又松开。

但沃克没让他高兴完。

"小子，"沃克道，他往前倾了倾身子，压低声音，"你以为黑桃是你最大的敌人？"

阿米尔的手指在桌下停了一下。

"黑桃背后还有人。"沃克道，"不是哈莱姆的人，是华尔街的。一个叫理查德的白人，穿三件套西装，开阿斯顿马丁。黑桃每月的货款，有三成是直接打给理查德的。"

阿米尔没动。

"理查德给谁洗钱？"他道。

沃克沉默了一下。他低头看了一眼自己胸口的警徽，手指在那枚金属上摩了一下，又抬头看阿米尔。

"阿什利家族。"沃克道，"你听过吗？曼哈顿上东区，他们控制纽约半个世纪了。我当警察三十二年，前二十六年都是给他们看门的。"

他顿了一下，又压低声音。

"小子，你听我一句劝。黑桃这种人，死了就死了，没人会查。但阿什利不一样。你要是动他们一根手指，第二天你的尸体会在东河里泡着，警察连报告都不会写。"

阿米尔没动。但系统面板在他眼前炸了一下——不是红光，不是黄光，是紫色。

曼哈顿上东区那块紫色光斑第一次开始闪烁，像一颗心脏在地图上跳。

---

饭局散的时候天已经黑了。

阿米尔回到Bed-Stuy的地下室。普丽雅坐在那盏灯泡下写作业，铅笔在纸上沙沙响。她抬头看了阿米尔一眼，没说话，又低头继续写。地下室还是那股霉味混着铁锈味，水管在滴水，嗒，嗒，嗒。

阿米尔把系统面板整个展开，铺在半空里。

紫色光斑在他面前慢慢展开，像一个被折叠了很久的纸团，一层一层摊开。

【阿什利家族·曼哈顿上东区·势力评级LV8。】

【家族成员：阿什利三世·68岁·家主 / 阿什利四世·42岁·继承人 / 理查德·45岁·代理人。】

【真实控制力：华尔街17% + 地产23% + 政客人脉。】

LV8。阿米尔的喉咙发干。他是LV4。黑桃是LV3。沃克是LV5。这家人是LV8。中间隔着四级，每一级在系统里都是十倍以上的资源差。

在印度，他见过的最高级别的存在是地主家门口那块铜牌。那个地主碾死他父亲用了三个月——拖欠工资、断水、最后放狗。阿什利家族碾死他，可能只需要一个电话。

他盯着那行字看了很久。灯泡在他头顶嗡嗡响，普丽雅的铅笔停了。

"哥，"她小声道，"紫色是什么意思？"

阿米尔没回头。他抬手，在紫色光斑上虚虚按了一下，光斑颤了颤，又稳住。

"那是我最终要干掉的人。"他道。
