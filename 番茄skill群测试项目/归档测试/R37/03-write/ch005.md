# 黑桃临死吐出那个名字，我才明白自己一直是猎物

紫色光斑还在半空里跳。

【阿什利家族·曼哈顿上东区·势力评级LV8。】

那行字浮着没散，像一块烧红的铁印在阿米尔眼前。他盯着它看了很久，久到头顶灯泡嗡嗡响了两轮，久到普丽雅的铅笔又沙沙写了一行。他刚说出口的那句话还挂在喉咙里——"那是我最终要干掉的人。"

干掉。他说得很轻。可LV4对LV8，中间隔着四级，每一级在系统里都是十倍以上的资源差。黑桃是LV3，沃克是LV5，这家人是LV8。他连沃克都是昨天才啃下来的，阿什利家族碾死他，可能只需要一个电话。

阿米尔抬手，把紫色光斑往边上一推，光斑颤了颤，缩回地图角落。他不能现在动紫色。他得先把眼前这颗黑子吃掉——黑桃还活着一天，他在哈莱姆就站不稳。

倒计时在面板角落跳了一下：3天04小时22分。

【系统任务·黑桃清除·倒计时。逾期惩罚：暴露主角位置与全部情报。】

他攥了一下拳头，指节咯咯响，又松开。

"叫大D和马库斯。"阿米尔道，声音压得很低，"今晚动手。"

突袭的局是三天前就布好的。沃克那条线，他让警察局第七分局今晚23点到凌晨1点"意外"不巡逻哈莱姆东区——线人举报有一批货走布朗克斯，警力全调过去了。马库斯带二十人从褐石公寓后门进，负责堵人；大D带六十人压前街，负责收尾。阿米尔不往前线，他没那个命去拼。他的位置在两公里外一栋废弃水塔的顶上，通过系统面板看整场仗。

夜风从水塔的铁皮缝里灌进来，刮在阿米尔脖子上，凉。他蹲在水泥台上，面前摊开的系统面板蓝光照亮了他半张脸，耳机里传来马库斯那边断断续续的电流声，夹杂着远处哈莱姆的犬吠。

【黑桃总部防御薄弱窗口：23:00-01:00·值班人员12人·火力等级中·建议突入点：后门消防梯。】

23:14。耳机里马库斯的声音压下来："到位。"

阿米尔盯着面板。哈莱姆东区那栋褐石公寓在地图上是一个发黄的光点，十二个小红点在里面游动。

"动。"阿米尔道。

耳机里炸开一声闷响——马库斯一脚踹开后门消防梯的铁栅，铁皮撞墙的咣当声顺着电流传过来，紧接着三声短促的枪响，啪啪啪，像撕布。面板上后门的两个红点同时熄灭。

"保镖清了。"马库斯道，喘着气，"两个。"

阿米尔的拇指在面板边缘蹭了一下，指腹发凉。耳机里传来脚步踩上木楼梯的咚咚声，一层，两层，杂着马库斯手下压低的喝骂和金属撞门的哐啷。每隔几秒，面板上就有一个红点变灰。十二个，十一个，九个。

褐石公寓顶层那颗最亮的红点开始往角落里缩——黑桃。

"他往上跑了。"马库斯道。

"让他跑。"阿米尔道，"堵顶层。"

23:41。顶层。

黑桃被堵在豪华公寓最里那间房。马库斯的人踹开卧室门时，黑桃正往后窗退，西装下摆翻起来，手里攥着一把镀金的玩意儿，手在抖。他背抵着窗玻璃，霓虹从他身后照进来，把他的脸切成一半红一半黑。

"马库斯。"黑桃道，声音哑了，"你这狗，回来咬主人。"

马库斯没接话。他抬枪，枪口稳稳压在黑桃眉心。

"你以为你赢了？"黑桃道，喉结上下滚了一下，他咧嘴笑，笑得很难看，"理查德会让你比死还惨。你们这帮街头猴子，在他眼里连人都算不上。"

马库斯的手指在扳机上顿了半秒。

耳机里，阿米尔的声音冷冷插进来："问清楚理查德是谁。"

黑桃听见了。他的目光往马库斯耳朵那边瞟——那是一只无线耳机。他突然笑得更响，笑到咳嗽，血沫从嘴角溢出来，他不知什么时候已经中了一枪，西装左胸洇开一片黑。

"理查德……"黑桃道，声音弱下去，"阿什利……"

马库斯扣了扳机。

砰。黑桃的脑袋往后一仰，撞在窗玻璃上，玻璃裂出一道蛛网纹，没碎。他的身体顺着玻璃滑下去，镀金手枪当啷掉在地板上，滚了半圈。

耳机里安静了两秒。

"死了。"马库斯道。

阿米尔盯着面板上那颗最亮的红点熄灭，变成灰色。他没有松气。

"搜他保险箱。"阿米尔道，"卧室衣柜后面，系统标了位置。"

耳机里传来撬锁的金属声，哐，哐，再是咔嗒一声。马库斯的声音再响起来时带着喘："找到了。一个本子，一张卡。"

"念。"

"一本瑞士银行账户密码……"马库斯翻着，"还有一张卡，硬的，印着——Wall Street Club，Richard，Membership 0451。"

阿米尔的呼吸停了一下。

系统面板在他眼前炸开，不是红光，不是黄光，是一道冷白色的光，把整片夜空都照得发青。

【发现关键道具：华尔街私人俱乐部会员卡·持有者：理查德·阿什利·阿什利家族成员（非代理人）。】

【信息修正：理查德·阿什利三世·阿什利家族二公子·45岁·实际负责家族地下资金通道。原评级"代理人"修正为"家族核心成员"。】

阿什利。不是代理人。是家里的人。

沃克告诉他理查德是"代理人"，系统之前也标的是代理人。现在这张卡把那个标签撕掉了——理查德姓阿什利，是那个LV8家族的二公子，亲自在给黑桃这种街头毒枭对接洗钱通道。

阿米尔的后背贴着水塔铁皮，铁皮冰凉，透过衬衫渗进脊背。他一直以为黑桃背后是个穿三件套的中间人，现在才明白，黑桃背后是阿什利家族的一只手，直接伸到了哈莱姆街角。

他咽了口唾沫，喉咙发干。

面板又跳了。

【LV5升级·解锁：代理人部署（可在曼哈顿/华尔街扶植代理人）·资产：$5,000,000（黑桃遗产转化）·情报网络范围扩展至华尔街层。】

五百万。他从五十万到五百万，用了一颗子弹的时间。阿米尔的手指在面板上停了一下，没有像上次那样攥拳。五百万在阿什利家族面前，还是个零头。

他正要把面板收起来，一条红色警告弹了出来，盖住了LV5的升级提示。

【警告·追溯分析启动。】

【对象：主角抵达纽约肯尼迪机场第一天，哈莱姆区遭遇的三名抢劫者。】

【刺青图谱比对：三人颈部"黑桃A"倒置刺青·对应——阿什利家族暗影部队·编号D-003。】

【结论：主角穿越抵达纽约第一天，即被阿什利家族暗影部队锁定。首次抢劫非随机事件，为试探性接触。】

阿米尔的手僵在半空。

他想起那天。恒河边闭眼，睁开是肯尼迪机场的柴油味，兜里两百美元，普丽雅攥着他的衣角。三个混混拦路，他花五十美元买命，混混嗤笑着指他去Bed-Stuy找大D。他一直以为那是纽约街头的随机恶意，是系统给他指的第一条路。

不是。

他从踏上纽约的第一秒起，就是被盯着的猎物。他以为自己是猎人，一路布局，策反沃克，吃掉黑桃——可这些动作，阿什利家族的暗影部队在第一天就看着了。

"哥？"普丽雅的声音从楼梯口传来，她爬上水塔顶，手里还捏着那支铅笔，"你脸色好难看。"

阿米尔没回头。他盯着那条红色警告看了很久，然后抬手，把它和紫色光斑并排摆在半空里。一个LV8的家族，一支编号D-003的暗影部队，一张理查德的会员卡。三条线，全指向同一个方向。

"我们从头到尾都在被监视。"阿米尔道。

普丽雅的铅笔停在半空。"那……那我们怎么办？"

阿米尔站起身，水塔铁皮在他脚下嘎吱响。他望向远处的曼哈顿天际线，那片灯海里，上东区有一团紫色的雾，比昨天更浓。

"继续。"他道，"既然他们盯着我们，那我就让他们看看，什么叫真正的内战。"

他话音没落，面板又跳了。

曼哈顿上东区那块紫色光斑，在他眼前一点一点膨胀，从一颗星变成一团云，边缘开始向Bed-Stuy这边蔓延。

【阿什利家族·家主阿什利三世·正在召开紧急会议。】

【议题：Bed-Stuy异常变量（黑桃资产非正常转移）。】

【处置方案：启动"理查德专案组"。】

阿米尔盯着那行字，夜风灌进他领口，他没觉得冷。他第一次清楚地感到，自己脚下踩的不是哈莱姆的街头，是一张早就铺好的网。他不是在跟一个老钱家族斗——他是在跟整张网斗。

紫色光斑又大了一圈。
