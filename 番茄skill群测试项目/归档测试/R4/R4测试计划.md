# R4 控制变量测试计划

## �试目标

用同一个种子（渣女税务局），通过不同skill配置的子agent并行产出第一章，隔离skill链路中哪个环节开始限制agent输出。

## 前置数据（R3已测）

| 组 | 种子 | skill配置 | 结论 |
|----|------|----------|------|
| C | 渣女税务局 | 完整write skill(章意图4问+去AI味6条+验收) | 有底但灵气被框住 |
| D | 渣女税务局 | 无skill，纯自由发挥 | 四版最强，灵感自然冒出 |

## R4测试矩阵

### T1: 纯燃料型（最激进——零规则+外部素材注入）

**配置**: 种子 + 真实PUA案例 + 番茄爆款开头分析 + 字数控制 + "自由发挥"
**去掉**: 所有skill规则（章意图4问、去AI味6条、验收）
**加入**: 外部燃料（真实事件+爆款分析）
**假设**: 外部燃料 > 内部规则。给素材比给规则更能提升质量
**文件**: T1_纯燃料型_第一章.txt

### T2: 正向心法型（激进——用情感引导替代规则约束）

**配置**: 种子 + 写作心法（情感引导）+ 字数控制
**去掉**: 所有skill规则
**加入**: 5条正向心法（代入感/读者体验/情绪爆发/避免无聊/不想规则想好看）
**假设**: 正向情感引导 > 负向规则约束。"想想什么好看"比"不能这样写"更有效
**文件**: T2_正向心法型_第一章.txt

### T3: 砹章意图型（诊断——保留笔触规则但砍掉"先想后写"流程）

**配置**: 种子 + 去AI味6条（作为参考提示，不用填表）+ 字数控制 + "直接写，不要思考过程"
**去掉**: 章意图4问 + 验收检查表
**保留**: 去AI味6条核心精神
**假设**: "先想4个问题再写"这个流程本身在杀灵气。保留笔触规则但砍掉流程，质量会回升
**文件**: T3_砹章意图型_第一章.txt

## 对比维度

1. **开场冲击力**: 前3句话是否抓人？
2. **人物立体度**: 主角和配角有没有记忆点？
3. **叙事维度**: 是单线推进还是多线交叉？
4. **灵气自然度**: 有没有"写着写着冒出来的"灵感？
5. **节奏密度**: 信息推进够快吗？
6. **章末钩子**: 读者想翻下一章吗？

## 预期结论

- T1 > D → 燃料注入确实提升质量
- T1 ≈ D → 燃料没用，种子才是天花板
- T2 > C → 正向引导比规则约束有效
- T2 ≈ D → 引导和自由发挥效果一样
- T3 > C → "先想后写"流程在杀灵气
- T3 ≈ C → 笔触规则也在限制，不只是流程问题
