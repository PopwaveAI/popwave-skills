## 表1：力量体系结构（坐标系）
### 1.1 力量体系类型
**民俗志怪传承型异术体系（禁忌行当 + 师徒/血脉传承 + 走江功德制）**

《捞尸人》构建了一个以长江水乡为核心舞台的民俗灵异力量体系。力量来源不是"修炼升级"，而是"禁忌行当传承"——捞尸人世代游走于江河流域，掌握走江规则、阴水秘术、水下禁忌等隐秘知识，以"职业传承"的方式获得对抗邪祟的能力。后期发展为以"走江功德"为核心的天道授权体制。

体系特征：
- **职业传承型**：力量绑定特定禁忌行当（捞尸人/走阴人/扎纸匠/风水师），非通用修炼体系，而是家族/师徒传承的"行当秘术"（ch1-2李三江捞尸+扎纸双行当；ch2刘金霞走阴算命）
- **禁忌驱动**：力量使用受严格禁忌约束（三不捞、走江规则），触碰禁忌必遭反噬，"禁忌"既是限制也是张力来源（ch1直立死倒为禁忌特例）
- **走江功德制**：走江者代表天道镇杀邪祟，积累功德，形成"浪（邪祟挑战）→镇杀→功德"的循环体制（ch101"代表天道镇杀邪祟"；ch621"一场走江，大量功德"）
- **阴阳双线**：世界观构建"人间-水底-阴界"三重空间，力量在阴阳两界交叉运作（走阴/入梦能力贯穿阴阳）
- **写实志怪**：以冷静白描笔法，将诡异现象锚定于民俗认知与物理逻辑（如直立溺尸的水流涡旋成因、"祟"的驱邪仪式），兼具真实感与神秘感（ch1-2刘金霞灶台香灰驱祟仪式）
- **人性博弈底色**："人知鬼恐怖，鬼晓人心毒"，力量体系服务于"比水鬼更可怕的是人心"的核心命题

### 1.3 子养成线
1. **走阴/入梦能力（感知线·核心金手指）**：主角被动走阴→主动走阴→走阴操控死倒。能进入梦境/特殊环境，进入他人内心世界，读取死倒生前记忆，催眠引诱操控行为（ch3710"拥有被动走阴的能力"；ch5169-5172"通过走阴达成联系，再进行意识融入，读取生前记忆，催眠操控"；ch7312"走阴成功的最后一步"）
2. **知识自学体系（认知线）**：通过阅读藏书自学异术。《江湖志怪录》（死倒百科）→《阴阳相学精解》（面相/算命）→《正道伏魔录》（伏魔法门）→各家传承典籍。学习能力惊人，三天看完七本《阴阳相学精解》（ch3580；ch7136"阴家十二法门的造诣，比正统传人都高深得多"）
3. **阵法术法（术法线）**：布置阵法（鬼打墙困敌）、符针/符纸制作、慑术（精神威慑）等。阵法可阻止动静传播、阻止无关人员靠近（ch9684-9685；ch9826骨戒增幅慑术效果）
4. **捞尸体术（体术线）**：背功、摔跤、擒拿、腿绞——专门针对死倒设计的近身搏击功夫（ch3299-3302《江湖志怪录》标注死倒弱点及攻击法门）
5. **功德/因果系统（天道线）**：走江镇杀邪祟积累功德，功德可转移/消耗；因果感知（金线指引因果深重者将死）；天道监督所有行道中人（ch621"一场走江，大量功德"；ch619"金线自少年身上释出，指向因果牵扯极深的人"；ch7893"天道肯定会拿着计算器算自己头上"）
6. **团队工具人系统（组织线）**：组建走江团队，每个成员有独立成长路径：
   - 谭文彬：《新五官图》镇压体内四灵兽，船头吆喝兼少君府管家，向地府报备（ch6226-6250）
   - 阿璃（秦璃）：秦柳绝学+《邪书》+血瓷瓶召唤梦中邪祟傀儡，利用诅咒阴影增强实力（ch6241-6242）
   - 润生：秦叔邪路归正统，九条蛟影趋于一合，熔炉锤炼死倒本能（ch6291-6294）
   - 阴萌：自封鬼气术法，搬运酆都大帝鬼气（ch6269-6277）
   - 林书友：官将首阴神系统，吸收恶鬼献祭之力（ch6270；ch6284-6288）
7. **龙王之灵（传承守护线）**：龙王之灵庇护门庭，走江走到极致可成龙王（ch619"柳清澄龙王之灵复燃"；ch66122"我当年就算不被围攻，也成不了龙王，心境不够"）

**涌现规律**：
- **初期**（门外汉→入门）：涌现"感知维度"——入梦/走阴能力觉醒，是进入超自然世界的钥匙；同步涌现"知识维度"——自学藏书打开认知边界
- **中期**（入门→正式走江）：涌现"合法性维度"——天道编制+功德系统，从野路子升级为正规军；同步涌现"组织维度"——团队工具人系统
- **后期**（资深→家主）：涌现"宿命维度"——龙王之灵、因果感知、家主权柄，从个人技艺跃迁为家族传承守护

- **特色**：本体系的"涌现"不是数值膨胀，而是"认知边界"的逐步打开+合法性层级的提升+团队职能的分化——走阴看见更多、知识理解更深、功德积累更厚、团队分工更细

### 1.5 交叉规则
1. **走阴×知识交叉**：走阴能力提供实践经验，知识自学体系提供理论支撑。走阴中遇到的死倒类型可在《江湖志怪录》中查到对应解法（ch3385"《江湖志怪录》对尸妖的描述"）；知识反过来指导走阴操作（ch5169-5172走阴操控死倒四步法）
2. **功德×因果交叉**：走江镇杀邪祟积累功德，但每个镇杀行为都牵扯因果。功德可转移消耗，因果则会以金线形式预警（ch619金线指引；ch621功德转移）；"天道不管苦衷缘由"——因果清算冷酷无情（ch7231）
3. **天道编制×禁忌反噬交叉**：官方编制身份保护走江者免受部分邪祟报复（ch9670-9672"秦柳两家传承在身，代表天道镇杀邪祟""天道，在看着呢"），但触碰禁忌仍遭反噬。编制是保护伞也是约束链
4. **传承×天赋交叉**：各家传承需要"门里蹦出个天才"才能拓展上限（ch9744"想要将传承进一步研究拓展，更需要门里忽然蹦出个天才"）。阴长生一人贡献让阴家啃两千年老本（ch9746-9747）。李追远的天赋让柳玉梅不惜打破规矩让他秦柳两家一肩挑（ch9748-9749）
5. **团队职能×个体代价交叉**：每个团队成员的成长都伴随巨大代价——谭文彬修初代本诀副作用将沦为灵兽寄体（ch6266）；阿璃加深诅咒阴影可能无法回头（ch6243）；润生九蛟合一将"变得不再像人"（ch6294）；阴萌搬运大帝鬼气"无法想象有多辛苦"（ch6273）。团队越强，个体代价越重
6. **阴阳交叉**：力量在"人间-水底-阴界"三重空间交叉运作。走阴沟通阴界，官将首本质是"阴间派出所"（ch9701），酆都大帝鬼气可从丰都搬运（ch6269-6270）

## 表9：动力引擎提取（众生攀登系统）
### 9.1 驱动逻辑
**走江功德驱动 + 禁忌行当传承驱动 + 诡秘事件解谜驱动 + 宿命抗争驱动**

体系运转的多重引擎：
- **功德驱动**：走江镇杀邪祟积累功德，功德是走江者的核心收益（ch621"一场走江，大量功德"）。功德可转移、可消耗，是体系运转的"货币"
- **传承驱动**：捞尸人/走阴人/风水师等技艺通过家族血脉与师徒传承，驱动主角学习掌握。传承出现天才能拓展上限（ch9744）
- **事件驱动**：邪祟以"浪"的形式一波波来袭（ch9806-9808"每一浪刚过去时，都会有一段平静期"），驱动主角团队持续作战
- **宿命驱动**：血脉宿命与家族恩怨驱动主角逐步揭开真相、对抗终极势力（仙姑/大乌龟/西王母）
- **天道驱动**：天道是最高仲裁者，所有行道中人都在天道监督下运作（ch7893"天道肯定会拿着计算器算自己头上"）

### 9.2 运转机制
1. **传承学习**：太爷/师父传授捞尸秘术、走江规则；看书自学拓展知识边界（ch13-14读《江湖志怪录》；ch3580三天看七本《阴阳相学精解》）
2. **走阴感知**：主动/被动走阴进入梦境/特殊环境/他人内心，获取信息、操控死倒（ch5169-5172走阴四步法）
3. **天道授权**：正式走江后获得编制，可合法镇杀邪祟并积累功德（ch101"官方编制身份"）
4. **浪式挑战**：邪祟以"浪"的形式来袭，走江者"自选题"迎战（ch9810-9813"利用了规则，把它们列为题目后，它们就不得不来""它们就是自己的题库"）
5. **团队分工**：团队成员各司其职——船头吆喝（谭文彬）、诅咒操控（阿璃）、体术先锋（润生）、鬼气搬运（阴萌）、官将首系统（林书友）（ch6226-6294）
6. **因果清算**：所有行为牵扯因果，天道以金线形式预警因果深重者将死（ch619"金线自少年身上释出"）

### 9.3 非主角变强路径
- **捞尸人路线**：各地捞尸人恪守规则、镇压邪祟，以行当资历积累力量（李三江为老一辈代表）
- **走阴人路线**：刘金霞拥有走阴能力，可沟通阴界，是独立异术路径（ch2；ch3710）
- **民间术士路线**：白事班子、风水师、扎纸匠等民间势力，各掌握部分民俗术法
- **宗门传承路线**：各大玄门家族（阴家自东汉传承两千年）靠代代传承积累，偶出天才拓展上限（ch9743-9747）
- **邪道路线**：修邪玄门人以自身为载体封养，求另一种方式兵解成仙，具备生前道法神通（ch3376）
- **官将首路线**：阴间执法体系，官将首阴神为"阴间派出所"（ch9701）
- **龙王路线**：走江走到极致可成龙王，龙王之灵可庇护门庭（ch66122；ch619）
- **点灯路线**：通过"点灯"获得力量（如王霖体内"功德照纸"快车道），但代价是沦为他人体附庸（ch6196；ch6202）

### 9.5 代价结构
- **走阴迷失代价**：走阴走多了容易迷失（ch5300"柳玉梅曾提醒过他，走阴走多了对人不好，容易迷失"）
- **禁忌反噬代价**：触碰禁忌必遭反噬，轻则厄运缠身，重则丧命
- **天煞孤星代价**：捞尸人长期接触不祥之物，被世俗隔绝，背负孤独宿命
- **邪水/死倒侵蚀代价**：接触邪水/死倒有被侵蚀风险
- **团队个体代价**：团队成员变强伴随巨大代价——沦为灵兽寄体、诅咒加深不可逆、变得不像人、极限痛苦（ch6266；ch6243；ch6294；ch6273）
- **二次点灯代价**：点灯者二次点灯后可退出走江，但所有本事付之一炬，沦为普通人（ch6196"体内的那张纸被烧成灰，所有本事也跟着付之一炬"）
- **因果清算代价**：所有行为牵扯因果，天道冷酷清算，"天道是不管的"苦衷缘由（ch7231）

### 9.6 范式归类
**禁忌传承型民俗志怪范式（走江功德制变体）**

核心特征：力量源于禁忌行当传承，受天道编制约束，以"认知边界打开"（走阴觉醒、知识自学、因果感知）替代"数值膨胀"。力量体系服务于"人性博弈"核心命题，"比水鬼更可怕的是人心"。后期发展为"走江功德制"——天道授权+浪式挑战+团队分工+龙王进阶的复合体制。这是"都市志怪+民俗悬疑"题材的典型范式，创新点在于将传统民俗行当升级为有编制、有功德、有因果的天道治理体系。

### 9.7 演化节点
- **觉醒期**（ch1-14）：被祟→入梦觉醒→自学《江湖志怪录》，认知边界打开

- **终局期**（ch622+）：对抗西王母等神话级存在，全员付出极限代价

## 表7：金手指深度
### 7.1 类型
**入梦/走阴能力（超自然感知+操控） + 自学天才（知识降维打击） + 天道编制（合法性外挂）**

主角李追远的金手指为三重结构：
1. **入梦/走阴能力**：被死倒祟到后觉醒入梦能力，可进入梦境/特殊环境/他人内心世界。从被动走阴进化为主动走阴，可读取死倒生前记忆、催眠引诱操控行为（ch1-2被祟后开始入梦；ch3710"拥有被动走阴的能力"；ch5169-5172走阴操控四步法；ch7312"走阴成功的最后一步"）
2. **自学天才**：学习能力惊人，三天看完七本《阴阳相学精解》，阴家十二法门造诣超越正统传人。看书自学即可掌握各家家传秘术，且能拓展传承上限（ch3580；ch7136；ch9748-9749柳玉梅因天赋让他两家一肩挑）
3. **天道编制**：正式走江后获得官方编制身份，代表天道镇杀邪祟，受天道保护（ch101"官方编制身份，多宝贵啊""天道，在看着呢"）

### 7.2 加速机制
- **走阴信息优势**：走阴可进入他人内心/梦境，获取常人无法获得的信息维度，在诡秘事件中拥有降维打击（ch5169读取死倒记忆；ch9790-9791进入阿璃内心世界对抗邪祟）
- **自学降维打击**：看书速度极快且理解深刻，能在极短时间内掌握别家数代积累的传承，并将不同传承融会贯通拓展上限（ch9744"门里忽然蹦出个天才，以极短时间完全掌握现有传承后，将传承体系进一步拔高，打开上限"）
- **天才聪慧**：设定为"天才聪慧、理性冷静"，认知与推理能力加速诡秘事件解谜（ch3384-3392面对尸妖幻觉保持镇定破局）
- **天道编制保护**：编制身份使邪祟不敢轻易报复，"天道在看着呢"（ch9670-9672）
- **题库机制**：利用规则将邪祟列为"题目"强制其现身，化被动为主动（ch9810-9813"把它们列为题目后，它们就不得不来""它们就是自己的题库"）

### 7.3 限制
- **走阴迷失风险**：走阴走多了容易迷失自我（ch5300）
- **无直接战斗力**：走阴/自学提供信息与知识优势，但不提供直接战斗力，对抗诡秘生物仍需体术+术法+团队配合
- **编制约束**：天道编制是保护也是约束，所有行为受因果清算，"天道不管苦衷缘由"（ch7231）
- **身体续航限制**：灵魂强大但身体生病会严重制约续航（ch66171-66173"灵魂强大，各种手段并不会因为身体生病而无法施展，但这会严重制约续航"）
- **无数值升级外挂**：金手指不提供"数值升级"，力量成长靠传承学习+走江功德+实战经验积累
- **自学弊端**：没有老师教，没有成套教学方案，"高处复杂问题能解决，简单概念却抓耳挠腮"（ch5185）；"各家都有传授之法，写在书上没师傅领门至多看点皮毛"（ch5210）

### 7.4 代价
- **走阴迷失代价**：走阴过多容易迷失（ch5300）
- **孤独宿命代价**：捞尸人/走江者传承伴随孤独宿命
- **因果清算代价**：所有走江行为牵扯因果，天道冷酷清算
- **身体消耗代价**：走江消耗巨大，身体生病制约续航（ch66171-66173）
- **团队代价转嫁**：主角的强大部分建立在团队成员的极限牺牲上——谭文彬将沦为灵兽寄体、阿璃诅咒加深不可逆、润生变得不像人（ch6266；ch6243；ch6294）
- **情感代价**：游走生死边缘，与阿璃的宿命绑定伴随巨大风险

### 7.5 耦合关系
**认知驱动型耦合：金手指打开"感知+知识+合法性"三重维度，驱动传承成长**

- **走阴×知识**：走阴提供实践经验，知识自学提供理论支撑，二者交叉驱动主角快速成长。走阴遇到的死倒类型可在书中查到解法，知识反过来指导走阴操作
- **天道编制×功德**：编制是获取功德的合法性前提，功德是编制的价值体现。编制保护走江者，功德积累反哺实力
- **金手指不提供"力量本身"**：走阴/自学/编制均为"进入体系"的钥匙，而非"绕过体系"的外挂。力量成长仍严格遵循"传承学习→走阴实践→天道授权→功德积累→团队分工→龙王进阶"的体系规则
- **耦合总结**：金手指=感知维度钥匙（走阴）×知识降维（自学天才）×合法性外挂（天道编制），三者均为"进入体系并加速成长"的工具，而非"凌驾体系"的作弊器。主角再强，也在天道、因果、禁忌之内。后期团队系统的出现，更将个人金手指转化为组织化力量——每个成员分担一种代价， collectively 实现跨越式提升
