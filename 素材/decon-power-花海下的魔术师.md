## 表1：力量体系结构（坐标系）

### 1.1 力量体系类型

**司命天物体系（天物觉醒+命师八境+奉天极武三轨制）**

世界观基底：千年前次元裂缝开启，异族进军，诡兽肆虐。老天降下灵气和"司命天物"让人类抗衡。每个人在18岁时通过仪式觉醒独属于自己的司命天物。司命天物蕴藏在人体内，司掌五脏六腑和筋脉。主角尽飞尘是穿越者，前世是舞台魔术师，觉醒寰级天物"终序花园"。

核心特征：
- **三轨并行**：司命天物等级（天赋）+ 命师境界（修为）+ 奉天极武（技能）三条独立成长线
- **天物决定起点**：天物等级从五级到寰级，决定起点高度和资源获取（ch5、ch11）
- **境界决定上限**：命师八境决定灵力量级和能力解锁（ch8）
- **极武决定战斗方式**：奉天极武分四阶十二级，结合天物使用（ch8）
- **无功法**：世界没有"功法"概念，极武是唯一技能体系（ch8）
- **穿越者+魔术师**：主角前世是魔术师，以魔术思维运用战斗（ch25：前世热爱魔术）
- **国际视角**：大夏（中国）、雾都（英国）、法黎塔（法国）等多国设定（ch230）

### 1.5 交叉规则

1. **天物×境界×极武三轨协同**：天物决定起点，境界决定上限，极武决定战斗方式——三者独立成长但互相影响（ch8、ch17）
2. **刻印×极武交叉**：刻印系统可复制任何极武，使主角不依赖学习极武——卡牌释放比原版更方便且节省灵力（ch17、ch23）
3. **全属性×极武交叉**：全属性使主角可学习任何属性极武，配合刻印系统形成全能战斗体系（ch23）
4. **秘境×传承交叉**：秘境探索→道诡暮场→遗骨→极武传承，形成资源获取循环（ch53→ch58→ch64）
5. **天物等级×社会资源交叉**：寰级天物拥有者获得正星保护、书院资源、社会特权（ch21-26）
6. **魔术思维×战斗交叉**：前世魔术师经验影响战斗风格——欺骗、误导、布局（ch1003：整场战斗骗过所有人）
7. **灵气×诡气交叉**（后期）：同时操控两种能量体系，领域内可自由切换（ch1251）

## 表9：动力引擎提取（天物攀登系统）

### 9.1 驱动逻辑

**寰级觉醒驱动 + 刻印金手指驱动 + 道诡战场外患驱动**

- **起点驱动（穿越+觉醒）**：穿越者前世魔术师，18岁觉醒寰级天物（ch1、ch5、ch11）
- **内驱（变强+自由）**：寰级带来变强压力，同时主角想保持自由、毁坏名声避免人设束缚（ch16）
- **外驱（异族威胁）**：异族肆虐千年，道诡战场持续战争，反派势力追杀（ch233、ch529）
- **正星驱动（月明一）**：尊境正星月明一的保护和教导，半帝级别的强者作为后盾（ch22、ch68）

### 9.2 运转机制

1. 穿越觉醒 → 寰级天物"终序花园" → 获得刻印能力（ch1、ch5、ch17）
2. 正星月明一安排 → 天策书院入学 → 修炼资源+极武学习（ch21-25）
3. 大荒实战 → 秘境探索 → 道诡暮场获得传承（ch24、ch53、ch64）
4. 刻印系统复制极武 → 卡牌战斗体系 → 无限制释放（ch17、ch30）
5. 国际任务 → 接触异族势力 → 反派交锋（ch206、ch233）
6. 境界突破 → 新能力解锁 → 更高维度战斗（ch8各境界）
7. 帝境领域 → 诡气操控 → 终极战斗形态（ch1251、ch541）

### 9.3 非主角变强路径

- 普通觉醒者：五级~一级天物，正常修炼路线，受限于天物等级和资源（ch5）
- 特级天物拥有者：书院天骄，如莲雨楼（特级莲华弓，崭境九转）（ch23）
- 寰级天物拥有者：八位寰级，各有正星保护，资源充足（ch71）
- 异族：种族之力为支撑，主上/信徒体系（ch1251-1254）
- 人造人：死人DNA+导铁+芯片，流境以下限制（ch214）
- 禁忌修炼者：吞噬天物的异咒经路线（ch243）

### 9.5 代价结构

- **名声代价**：寰级天物拥有者需保持完美英雄人设，主角选择主动毁坏名声（ch16）
- **正星限制**：正星除生命危险不可出手，主角需独立面对大部分威胁（ch26）
- **极武消耗**：飞行类极武是一次性消耗品，万金难求（ch23）
- **秘境风险**：道诡暮场中有古境以上诡尸，极其危险（ch58、ch63）
- **至暗代价**：手套下的至暗力量可能带来未知风险（ch548）
- **人性代价**：反派利用人性弱点，身边的人成为目标（ch233："人最可怕的是人性"）

### 9.6 范式归类

**天物觉醒+三轨并行+刻印复制型**
- 司命天物为天赋基底，命师八境为修为框架，奉天极武为技能体系
- 刻印系统是核心金手指——无限制复制极武，使主角不依赖学习
- 全属性+刻印形成全能战斗体系
- 魔术师思维影响战斗风格——欺骗、布局、误导
- 每个境界突破涌现新子系统，是长篇持续供给新鲜感的关键
- 国际视角和道诡战场提供了宏大的世界观舞台

### 9.7 演化节点

1. **穿越觉醒**：前世魔术师穿越，18岁觉醒寰级天物（ch1、ch5、ch11）

6. **共读计划**：全球天才联合培养（ch250）
7. **道诡战场**：人族与异族正面战场（ch532-551）
8. **帝境领域**：领域系统、诡气操控、契约诡兽（ch541、ch1251）

## 表7：金手指深度

### 7.1 金手指类型

**寰级天物"终序花园"——刻印复制型**

三大核心能力：
1. **防御点满**：天物自带防御能力，极其强悍（ch17）
2. **修炼点满**：天物自带修炼加速，灵力精纯（ch17）
3. **刻印**：所看到的物品及极武，都可被卡牌刻印，然后用灵力催发释放——无副作用、无次数限制、可抹除重刻（ch17）

附加金手指：
4. **全属性**：可学习任何属性极武，完全不排斥（ch23）
5. **前世魔术师经验**：穿越者带来的魔术思维，影响战斗风格——欺骗、布局、误导（ch25、ch1003）

### 7.2 加速机制

- **刻印复制加速**：不需要学习极武，刻印即可使用，节省大量修炼时间（ch17）
- **卡牌战斗加速**：卡牌释放比原版极武更方便且节省灵力（ch23）
- **全属性加速**：可学习任何属性极武，不存在"不适合"（ch23）
- **正星资源加速**：尊境正星月明一提供修炼资源和专业指导（ch22、ch25）
- **书院资源加速**：天策书院赠与灵金、高级极武（ch21-25）
- **秘境传承加速**：道诡暮场遗骨可获得高阶极武传承（ch64-68）

### 7.3 限制

- **灵力限制**：刻印释放需要灵力催发，灵力不足时无法使用（ch30）
- **修为限制**：部分极武有境界要求，如无始印需沉境才能完整使用（ch69）
- **天物秘密**：刻印能力的来源不完全透明，可能涉及更深层的秘密（ch24）
- **正星限制**：月明一除生命危险不可出手（ch26）
- **至暗风险**：手套下的至暗力量可能带来未知风险（ch548）
- **社会压力**：寰级天物拥有者被社会关注，需维持形象（ch16）

### 7.4 代价

- **名声代价**：寰级天物带来社会关注，主角主动毁坏名声以获得自由（ch16）
- **孤立代价**：不选择扮猪吃虎路线，身份公开后面临持续挑战（ch17）
- **人性代价**：身边的人成为反派目标（ch233）
- **至暗代价**：手套下的至暗力量是未知的负担（ch548）
- **责任代价**：寰级意味着被寄予厚望，需参与道诡战场等（ch532）

### 7.5 与主养成线耦合关系

- **天物×刻印耦合**：寰级天物"终序花园"提供刻印能力——天物等级决定金手指强度（ch5→ch17）
- **境界×刻印耦合**：刻印释放需要灵力催发，境界提升灵力→更多卡牌同时释放→更强战斗力（ch30、ch244）
- **极武×刻印耦合**：刻印可复制任何极武，使主角不依赖极武学习——但高阶极武仍需境界支撑（ch17、ch69）
- **全属性×刻印耦合**：全属性使主角可刻印任何属性极武，形成全能战斗体系（ch23）
- **魔术师×战斗耦合**：前世魔术师经验影响战斗风格——欺骗、布局、误导与刻印系统配合（ch25、ch1003）
- **终局耦合**：帝境领域+诡气操控+至暗力量，金手指与主养成线在终点汇合（ch541、ch548、ch1251）
