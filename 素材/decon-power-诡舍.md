## 表1：力量体系结构

### 1.1 力量体系类型

**恐怖生存/诅咒副本型力量体系**

《诡舍》并非传统修仙升级体系，而是以"诅咒→副本生存→收集脱出"为核心闭环的恐怖生存型力量体系。其核心特征：

- **力量来源非内生**：主角无法自主修炼变强，力量来源于外部获取（鬼器道具）、知识积累（隐藏规则掌握）、社交网络（老带新/组织资源）和身份能力（主角真实身份为顶级杀手"棺材"）
- **成长维度为"生存能力"而非"战斗力"**：变强=存活率提升，而非攻击力增强。鬼器是消耗品而非装备，知识（规则）是最核心的"力量"
- **副本制驱动**：九扇血门构成主养成线，每扇门是独立的恐怖副本，有独立的杀戮规则、生路、NPC和鬼怪
- **知识即力量**：隐藏规则的掌握程度直接决定存活率，信息差是核心博弈维度（ch28"血门有两个很奇怪的隐藏规则"，ch53"血门上的提示只是重要提示，它并不会将副本世界所有规则全部都告诉我们"）
- **反力量递进**：敌方（鬼怪）的力量随 Door 数递增而质变——从受规则束缚的地缚灵，到追踪型厉鬼，到每杀一人觉醒新能力的恶灵

证据：ch1（诅咒引入/大巴/血门）、ch14（诡舍规则解释"是诅咒"）、ch15（鬼分两种/血门机制）、ch53（九扇血门体系/死亡率统计）

### 1.2 主养成线位阶表

主养成线 = **九扇血门（Nine Blood Doors）**

每个被诅咒者必须经历九扇血门。可通过两条路径脱出诅咒：(A) 活过全部九扇血门；(B) 收集12块拼图碎片，前往迷雾世界终点。（ch53）

| 阶段 | 血门编号 | 间隔时间 | 核心机制 | 死亡率 | 证据章 |
|------|---------|---------|---------|--------|--------|
| **新手期** | 第1-3扇 | 1周/扇 | 鬼一次只杀一人（1-kill rule）；1/10法则全效；鬼器初获；无拼图碎片产出 | 较低但有翻车风险 | ch53, ch57 |
| **进阶期** | 第4-6扇 | 6月/扇 | 拼图碎片出现（1/10概率）；NPC能力获取开放；秽土/旧址系统；特殊拼图碎片→特殊鬼器 | 显著上升 | ch53, ch344 |
| **深渊期** | 第7-9扇 | 1年/扇 | 1/10法则大幅削弱；鬼觉醒能力系统（每杀一人觉醒一种能力）；规则束缚减少；几乎必死 | ch7=95.796%；ch8-9≈100% | ch53, ch180, ch384 |

**位阶说明**：
- 新手期的"老带新"机制允许资深诡客带新人刷低难度门，获取鬼器（ch59"血门代练"）
- 进阶期是资源积累窗口期，6个月间隔给予充分准备时间（ch53"到那个时候，时间就充裕很多了"）
- 深渊期死亡率极高，绝大多数诡客选择在第七扇门前凑齐拼图碎片脱出（ch53"每个诡舍的成员为了能够在第七扇血门来临之前离开这个地方，逃脱诅咒，他们都会想尽一切办法去收集拼图碎片"）

## 表9：动力引擎提取

### 9.1 驱动逻辑

《诡舍》的核心驱动力是**被动卷入+死亡高压+资源稀缺+时间倒计时**的四重压迫：

| 驱动要素 | 机制 | 证据章 |
|---------|------|--------|
| **诅咒强制** | 被诅咒者无法拒绝进入血门；不去则血门后"有东西出来找你"，无论逃到哪里都会被找到 | ch1 |
| **死亡永久** | 血门内死亡=真实死亡，无复活机制 | ch1, ch14 |
| **时间倒计时** | 前三扇每周一扇，4-6扇每半年一扇，7-9扇每年一扇——不可拖延 | ch53 |
| **资源稀缺** | 鬼器使用次数有限（强力≤3次），拼图碎片1/10概率，不可无限刷 | ch55, ch53 |
| **难度递增** | 每扇门难度递增，第七扇死亡率95.796%——不进则死，进亦可能死 | ch53 |
| **PVP威胁** | 罗生门组织杀人夺信，其他诡客可能利用规则害人 | ch164, ch146 |
| **终局赌注** | 最终一人直面"祂"，输了人类文明消失——终极压力 | ch991 |

### 9.2 运转机制闭环

```
诅咒激活 → 大巴接送 → 诡舍安置 → 血门开启 → 副本生存
    ↑                                              ↓
    ← ← ← 鬼器消耗/获取 ← ← ← 生路发现 ← ← ← 规则掌握
    ↓
拼图碎片收集（Door 4+） → 12块凑齐 → 迷雾终点 → 终局挑战
    ↓                                    ↓
未凑齐 → 继续血门（难度递增）         失败=文明消失
    ↓
第七扇前未脱出 → 95.796%死亡率 → 几乎必死
```

**闭环关键节点**：
1. **入口闭环**：诅咒→大巴→诡舍→血门，不可拒绝不可逃避（ch1, ch14）
2. **生存闭环**：规则掌握→生路发现→鬼器使用→存活→鬼器消耗→需更多鬼器→刷门/代练（ch28, ch33, ch59）
3. **脱出闭环**：Door 4+拼图碎片→12块→终点资格→终局挑战（ch53, ch991）
4. **失败闭环**：鬼器耗尽/规则误判→死亡→鬼器封印→队友更弱→连锁死亡（ch33）

## 表7：金手指深度

### 7.1 类型

**信息预知型金手指——神秘信件系统**

主角宁秋水在进入每扇血门前，会收到一封无法追踪来源的神秘信件，内含：
- **照片**：与即将遭遇的关键人物/事件相关的照片（如刘承峰的照片，ch12）
- **杀戮规则提示**：以诗句/暗语形式给出鬼的杀戮法则（如"风不点灯，雨不燃烛。日不登楼，夜不瞑目"，ch12）
- **关键人物保护提示**：如"白潇潇不能够死在他的第二扇血门里"（ch40）

信件特征：
- 邮局查无记录，"那封信就这样凭空地出现了"（ch60）
- 字迹逐渐潦草，寄信人时间紧迫（ch79"第三封信的字迹，已经有一些说不出的潦草"）
- 收信人不止宁秋水一人（ch60"其实，收到神秘来信的……不止你一人"）
- 信件内容全部应验（ch12"那封神秘的信件，到目前为止，上面所有的内容都应验了"）

### 7.2 加速机制

| 加速维度 | 机制 | 效果 | 证据章 |
|---------|------|------|--------|
| **生路发现加速** | 信件提供杀戮规则→直接推断生路 | 跳过大量试错/探索时间；他人靠命试规则，主角靠信件知规则 | ch12 |
| **人员识别加速** | 信件附带照片→提前识别关键盟友 | 第一扇血门即识别刘承峰为"值得一交的朋友" | ch12 |
| **避险加速** | 信件提示保护对象→避免关键人物死亡 | 知道白潇潇不能死于第二扇→主动保护 | ch40 |
| **规则验证加速** | 信件规则可直接验证→快速构建规则模型 | 比其他诡客更快掌握隐藏规则体系 | ch12 |

### 7.3 限制

| 限制类型 | 具体表现 | 证据章 |
|---------|---------|--------|
| **信息不完整** | 信件仅给出杀戮规则/照片，不直接给出生路——仍需自主推理 | ch12（规则是暗语，非直接答案） |
| **来源不明** | 无法确认寄信人身份/目的——可能存在被操控风险 | ch12, ch60 |
| **不可控** | 信件自动出现，主角无法主动请求/选择内容 | ch1, ch40 |
| **时效递减** | 第三封信字迹潦草，寄信人"时间已经不多了"——金手指可能在衰减/消失 | ch79 |
| **非唯一性** | "收到神秘来信的……不止你一人"——其他人也有类似金手指 | ch60 |
| **不提供战斗力** | 信件仅提供信息，不提供鬼器/能力/战斗力——面对鬼仍需自保 | ch12 |
| **被抢风险** | 罗生门组织"疯狂杀人夺信"——金手指本身是PVP目标 | ch164 |

### 7.4 代价

| 代价类型 | 具体表现 | 证据章 |
|---------|---------|--------|
| **监视代价** | "一直有人在他看不见的地方盯着他"——被未知势力监视 | ch12 |
| **身份暴露风险** | 信件内容暗示有人了解主角的一切——隐私/安全威胁 | ch12 |
| **依赖风险** | 过度依赖信件→若信件停止/出错→失去核心优势 | ch79（字迹潦草暗示即将停止） |
| **被操控风险** | 信件引导主角保护特定人/做特定事——可能在执行他人计划 | ch40, ch690 |
| **因果反噬** | "这一切的一切，背后一定有一只我们看不见的手在操控"——金手指可能是更大棋局的一部分 | ch12 |
| **终局代价** | 大计划揭示——"他如今扮演的，是当年自己赋予的宁秋水的身份"——金手指可能源于主角自身的终局布局 | ch690 |
