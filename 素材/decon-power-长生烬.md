> 采样方式：三阶段算法 v2.0（前10章全读 + 11-100章关键词扫描 + 100章后每25章+境界过渡期必采）

---

> **与《系统赋我长生》区分说明**
> 《长生烬》与《系统赋我长生，活着终会无敌》同为紫灵风雪所著，主角同为陈浔，核心设定（长生系统、黑牛伴生）一致。但《长生烬》在txt采样中展现出远比网络公开信息更丰富的力量体系：
> - 五维属性加点系统（力量/速度/防御/法力/万物精元），每项有20点桎梏机制
> - 万物精元的独特功能（加速植物生长，培育灵药）
> - 修仙百艺系统（炼器/炼丹/阵法/符箓）的详细展开
> - 本命法宝系统（黑棺死气、开山斧等）
> - 五颗金丹（五行相生相克）的特殊设定
> - 自创功法体系（《五行仙经》）
> - 垃圾回收厂经济帝国
> - 灵兽家族/弟子体系（小鹤、小赤等）
> 以下拆解基于《长生烬》txt采样数据，标注证据章。

## 表1：力量体系结构（坐标系）

### 1.1 力量体系类型

**长生系统流（五维加点+传统修仙双轨制）**

世界观基底：主角陈浔穿越到浩瀚无垠的修仙界，觉醒长生系统，寿元无限，每年获得一点属性可加在五维属性上（力量/速度/防御/法力/万物精元）。系统还赠送一头长生灵兽（断角黑牛）为伴，每年也可独立加点。以长生者视角跨越修仙界多个大时代和地域：小山村→宁云山脉→五蕴宗→御虚城→大离→禁海→海外→三千大世界。

核心特征：
- **五维属性加点制**：力量/速度/防御/法力/万物精元，每项20点桎梏（筑基前），筑基后突破（ch42、ch188）
- **传统修仙境界制**：炼气→筑基→金丹→元婴→化神→炼虚→合道→大乘→渡劫
- **双轨并行**：长生点加点和修仙境界是两条独立的成长线，互相影响但有各自的上限
- **苟道生存制**：以"苟"为核心策略，靠时间碾压一切（ch14："遇事莫冲动"；ch51："跟着你浔哥混"）
- **灵兽伴生制**：长生黑牛伴生，独立成长线，同样拥有五维加点和修仙境界
- **万物精元制**：独特的第五维属性，可加速植物生长，培育灵药（ch45：一滴加速两个月生长）

### 1.5 交叉规则

1. **五维加点×修仙境界协同**：加点强化肉身基能，修仙境界强化法力/神识；两者各有上限，筑基后加点桎梏解除（ch42、ch188）
2. **万物精元×灵药培育×境界突破**：万物精元培育灵药→灵药炼丹→丹药辅助突破境界（ch45→ch180→ch188）
3. **长生×苟道规则**：无限寿命使"不战斗"成为可行策略，时间本身成为武器（ch14、ch51）
4. **黑牛×主角协同**：双长生组合，分工明确（主角炼器/炼丹，黑牛阵法）（ch176）
5. **灵根限制×加点绕过**：五系杂灵根导致修炼慢，但万物精元培育灵药可绕过灵根限制（ch50、ch180）
6. **死气×本命法宝**：死气材质可提升法宝至十八万年，死气可污浊万物（ch399）
7. **经济×修炼**：灵石可购买功法/丹药/留影石讲道，经济实力直接决定修炼速度（ch1151）
8. **时代变迁×修炼规则**：不同地域/时代的修炼规则不同（大离→禁海→海外）（ch762-779）

## 表9：动力引擎提取（长生攀登系统）

### 9.1 驱动逻辑

**时间碾压驱动 + 五维加点驱动 + 苟道生存驱动**

- **起点驱动（长生系统）**：穿越觉醒长生系统，寿元无限+每年加点（ch1）
- **内驱（变强）**：五维属性加点带来可感知的变强反馈（ch1："浑身突然涌起一股力量"）
- **外驱（生存压力）**：修仙者可随意杀凡人，生存压力驱动变强（ch5-6）
- **时间驱动（终极武器）**：不需要天才资质，只需要"活着"，时间是最强武器（ch50）

### 9.2 运转机制

1. 穿越觉醒长生系统 → 获得无限寿命+五维加点+黑牛伴生（ch1）
2. 凡人期苟活 → 加点力量/速度 → 超越凡人（ch1-14）
3. 获得修仙功法 → 修炼炼气决 → 进入修仙体系（ch41）
4. 万物精元培育灵药 → 绕过灵根限制 → 准备筑基丹药（ch45、ch180）
5. 筑基突破 → 长生点桎梏解除 → 炼器/阵法/炼丹系统开启（ch188）
6. 金丹突破 → 本命法宝 → 五颗金丹 → 死气系统（ch395-399）
7. 元婴突破 → 传送阵 → 灵兽家族 → 禁海探索（ch762-779）
8. 化神突破 → 自创功法 → 垃圾回收厂 → 传道（ch1151-1179）
9. 炼虚突破 → 脱离灵气仙道 → 三千大世界（ch1166+）

### 9.3 非主角变强路径

- 同时代修士：标准修炼路线，受限于有限寿命和灵根天赋（ch50）

### 9.5 代价结构

- **无限寿命的孤独代价**：见证所有人离世，熟悉的人慢慢消失（ch3、ch9）
- **苟道的机会成本**：错过机缘和资源，修炼速度极慢（ch50）
- **灵根限制**：五系杂灵根导致修炼效率低，需要外物辅助（ch50）

- **经济消耗**：突破境界需要海量灵药/丹药/灵石（ch1151）

### 9.6 范式归类

**系统加点+时间碾压+苟道生存型**
- 长生系统为核心金手指，传统修仙境界为基础框架

- "时间"作为终极武器，"苟"从负面行为升华为体系核心策略
- 万物精元是独特的第五维属性，连接了加点系统和修仙资源系统
- 每个境界突破涌现新子系统，是超长篇持续供给新鲜感的关键

## 表7：金手指深度

### 7.1 金手指类型

**长生系统（寿元无限+五维加点+万物精元型）**

三大核心金手指：
1. **寿元无限**：天地法则无法限制寿元，不会自然死亡（ch1）
2. **五维加点**：每年1点可分配到力量/速度/防御/法力/万物精元（ch1）
3. **灵兽赠送**：长生黑牛伴生，独立加点，独立修炼（ch1）

独特金手指：
4. **万物精元**：第五维属性，一滴加速两个月植物生长，可培育灵药绕过灵根限制（ch45）
5. **丹田拓宽**：筑基时筑基丹拓宽丹田，法力储存量翻倍（ch188）

### 7.2 加速机制

- **加点加速**：每年1点属性提升，可感知的变强（ch1）
- **时间加速**：无限寿命可花数百年修炼一个境界（ch50）
- **万物精元加速**：培育灵药→炼丹→辅助突破（ch45→ch180）
- **黑牛协同加速**：双长生组合，分工修炼（ch176）
- **无死亡压力**：不需要冒险争夺资源（ch51）
- **经济加速**：垃圾回收厂年赚200万中品灵石，购买资源/留影石（ch1151）

### 7.3 限制

- **每年仅1点**：加点速度极慢，一年只能提升一项属性1点（ch1）
- **20点桎梏**：筑基前每项属性上限20点，无法继续提升（ch42）
- **灵根限制**：五系杂灵根导致修仙效率低，境界可能卡死（ch50）
- **可被杀死**：长生不等于无敌，仍然可以被杀死（ch1："咱们虽然长生，但却能被杀死"）
- **无技能**：有法力但无技能时毫无用处（ch42："他理解为蓝量，但是他们没有技能"）
- **经济依赖**：高境界突破需要海量灵石（ch1151）

### 7.4 代价

- **万古孤独**：见证所有人离世，熟悉的人慢慢消失（ch3、ch9）
- **苟道的机会成本**：错过机缘和资源（ch50）
- **时间消耗**：每个境界需要数百年修炼（ch395：百年计划）
- **被同时代修士轻视**：修炼速度慢，被看作药农/杂役（ch176）
- **身份隐蔽压力**：需要不断隐藏身份、销赃跑路（ch153）

### 7.5 与主养成线耦合关系

- **时间耦合**：金手指（长生）与主养成线（修仙境界）的关系是"时间换空间"——花数百年磨一个境界（ch50）
- **加点耦合**：五维加点独立于修仙境界，但筑基突破后加点桎梏解除（ch42→ch188）
- **万物精元耦合**：万物精元→培育灵药→炼丹→辅助境界突破，形成正反馈循环（ch45→ch180→ch188）
- **黑牛耦合**：黑牛独立成长但与主角协同，分工修炼（主角炼器/炼丹，黑牛阵法）（ch176）
- **终局耦合**：化神后自创《五行仙经》，金手指（长生系统）与主养成线（修仙境界）在自创功法中汇合——脱离灵气仙道，建立独立修炼体系（ch1166）
