## 表1：力量体系结构（坐标系）

### 1.1 力量体系类型

**民俗灵异体系（风水术数+出马仙+术法三轨制）**

世界观基底：主角吴劫出生时十九道天雷围绕家门，妖狐渡劫化形时天谴将其神魂落在婴儿身上。妖狐（八尾白狐）至少千年道行，神魂寄宿主角体内，一体双魂。主角命犯十八劫，每三年一劫，每劫必伤人命。从小被黄皮子和狐狸护卫（妖狐的亲友后代），只喝血不吃饭。后拜师风水王李玄通，学风水术数和各种术法，行走江湖。

核心特征：
- **三轨并行**：风水术数（环境操控）+ 出马仙/仙家（灵体协作）+ 术法/法器（战斗手段）三条独立成长线
- **一体双魂**：主角与八尾狐共用一个身体，狐妖可赋予主角妖力（ch1698）
- **劫数驱动**：十八劫每三年一劫，是被动成长动力（ch1、ch6）
- **民俗基底**：尸鬼、黄大仙、狐仙、出马弟子、风水先生等中国民俗灵异元素
- **宗门江湖**：茅山宗（华夏第一宗门）、龙虎山、崆峒派、塔云山等修行宗门
- **官方管理**：特调组（官方超自然管理机构），镇守燕北的高人（ch1681）
- **苟中带邪**：主角自称"老六"/"邪修"，以非常规手段取胜（ch1695）

### 1.5 交叉规则

1. **风水×术法交叉**：风水布局影响灵气环境，术法在风水宝地威力增强（ch2477、ch2465）
2. **八尾狐×主角协同**：狐妖妖力赋能主角，主角保护狐妖恢复神魂——共生关系（ch1698）
3. **天罡印×鬼物养成交叉**：天罡印既是法器也是鬼物容器，鬼物团队在战斗中吞噬提升（ch1695）
4. **劫数×成长交叉**：每次应劫后主角变强，劫数是被动成长动力（ch1、ch6）
5. **玄真悟元功×寒冰九剑交叉**：功法境界决定剑法催动能力（ch1700）
6. **修为×法器交叉**：修为提升增强法器威力，法器也需要修为催动（ch1694-1695）
7. **正邪交叉**：主角被称为"邪修"（养鬼物），但行正道之事——正邪边界模糊（ch1695）
8. **宗门×江湖交叉**：宗门背景影响社会地位和资源获取，江湖恩怨驱动剧情（ch2465-2470）

## 表9：动力引擎提取（民俗灵异攀登系统）

### 9.1 驱动逻辑

**劫数被动驱动 + 师承成长驱动 + 江湖恩怨驱动**

- **起点驱动（妖狐夺舍+劫数）**：出生即妖狐神魂寄宿，命犯十八劫（ch1）
- **内驱（变强保护自己和身边的人）**：每次劫数都是生死考验，必须变强才能活下去（ch6）
- **外驱（江湖恩怨）**：白弥勒/一关道的终极威胁，宗门之间的恩怨（ch663、ch2467）
- **师承驱动**：风水王李玄通八年倾囊相授，学风水术数和各种术法（ch95、ch693）

### 9.2 运转机制

1. 妖狐夺舍出生 → 十八劫诅咒 → 黄皮子/狐狸护卫（ch1-5）
2. 三岁劫 → 尸鬼战斗 → 八尾狐元神显现 → 刘婆子出马仙（ch5-16）
3. 拜师风水王 → 学风水术数+术法+法器 → 八年倾囊相授（ch95）
4. 出师行走江湖 → 接案赚钱 → 天眼通+天蓬尺+招魂咒（ch86-92）
5. 万年雪莲果 → 八尾狐恢复 → 修为提升 → 玄真悟元功突破（ch662-1694）

### 9.5 代价结构

- **劫数代价**：每三年一劫，必伤人命，不是主角死就是别人死（ch1、ch6）
- **仙家代价**：八尾狐挡雷劫致道行尽毁，元神显现后更加虚弱（ch14-16）
- **孤独代价**：从小被村民排斥，"怪胎"，没有玩伴（ch3）
- **因果代价**：横死阴灵飘荡致无辜丧命→沾染因果（ch92）
- **正邪代价**：养鬼物被视为"邪修"，名声受损（ch1695）
- **经济代价**：师父经常坑钱，需要不断接案赚钱（ch2465）
- **人情代价**：欠下各种人情债，需要还（ch662：万年雪莲果分配）

### 9.6 范式归类

**民俗灵异+师承成长+劫数驱动型**
- 以中国民俗灵异为基底（出马仙/风水/尸鬼/黄大仙等）
- 师承制是核心成长路径（风水王李玄通八年倾囊相授）
- 劫数体系是独特的被动成长动力（十八劫每三年一劫）
- 八尾狐共生+天罡印鬼物团队构成独特的"一人成军"战斗体系
- 正邪边界模糊——主角被称为"邪修"但行正道之事
- 每个大阶段突破涌现新子系统，是超长篇持续供给新鲜感的关键
- 宗门江湖+特调组提供了社会化的力量体系框架

## 表7：金手指深度

### 7.1 金手指类型

**八尾狐仙家神魂+天罡印鬼物团队+劫数体系（三重金手指组合）**

三大核心金手指：
1. **八尾狐仙家神魂**：千年道行的八尾白狐神魂寄宿体内，一体双魂
   - 妖力赋能：将妖力扩散全身，临时增强战斗力（ch1698）
   - 元神显现：危急时刻元神可显现击退邪祟（ch14）
   - 恢复潜力：吞噬妖物稳固神魂，目标重铸法身（ch1698）
2. **天罡印鬼物团队**：可收容/释放鬼物战斗的法器
   - 鬼物团队：娜姐、刘欣霖、鬼夫妻、小婴灵等（ch1695）
   - 吞噬成长：鬼物可吞噬其他鬼物提升道行（ch1695）
   - 一人成军：释放鬼物团队相当于拥有军队（ch1695）
3. **劫数体系**：每三年一劫的被动成长动力
   - 每次应劫后变强（ch1、ch6）
   - 黄皮子/狐狸护卫自动保护（ch1-5）

附加金手指：
4. **师承风水王**：八年倾囊相授，风水术数+术法+法器+医术+解蛊等全科技能（ch95、ch693）
5. **全属性术法**：风水/术法/医术/解蛊/针灸等综合能力（ch676、ch693）

- **八尾狐妖力加速**：妖力扩散全身，临时大幅增强战斗力（ch1698）

- **洞天福地加速**：茅山宗洞天福地灵气充裕十几倍（ch2465）

- **劫数必伤人命**：每三年一劫，不是主角死就是别人死（ch6）

- **八尾狐×修为耦合**：狐妖妖力赋能主角战斗力，主角保护狐妖恢复神魂——共生关系（ch1698）
- **天罡印×鬼物养成耦合**：天罡印既是法器也是鬼物容器，鬼物团队在战斗中吞噬提升——正反馈循环（ch1695）
- **劫数×成长耦合**：每次应劫后主角变强，劫数是被动成长动力——压力驱动（ch1、ch6）

- **玄真悟元功×寒冰九剑耦合**：功法境界决定剑法催动能力——功法是剑法的基础（ch1700）
- **正邪×战斗耦合**：被称"邪修"但行正道之事，正邪边界模糊——以非常规手段取胜（ch1695）
- **终局耦合**：八尾狐重铸法身+白弥勒终极对决，金手指与主养成线在终点汇合（ch1698、ch2467）
