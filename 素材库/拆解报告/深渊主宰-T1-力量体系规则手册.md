## 一、力量体系全景图

### 1.1 三大力量体系交互关系

《深渊主宰》采用**三层力量体系架构**，三者可相互转换、交织升级：

`
┌─────────────────────────────────────────────────────┐
│                    力量体系架构                       │
├──────────┬──────────────────┬───────────────────────┤
│ DND职业系 │    神性体系       │      恶魔体系         │
│ (凡人之道)│   (神灵之道)      │    (深渊之道)         │
├──────────┼──────────────────┼───────────────────────┤
│ 平民      │ 凡人/神子         │ 怯魔                  │
│ ↓        │ ↓                │ ↓                    │
│ 职业者    │ 半神              │ 中阶恶魔              │
│ ↓        │ ↓                │ ↓                    │
│ 进阶职业  │ 伪神(圣者形态)     │ 高阶恶魔              │
│ ↓        │ ↓                │ ↓                    │
│ 传奇领域  │ 微弱神力          │ 巴洛炎魔              │
│ ↓        │ ↓                │ ↓                    │
│ 大奥术师  │ 弱等→中等→强大神力 │ 恶魔领主/恶魔王子     │
└──────────┴──────────────────┴───────────────────────┘
`

- **DND职业系**：基础骨架，通过职业等级提升获得属性点、技能点、专长
- **神性体系**：高维力量，通过神性值→神格值→神力值逐层解锁
- **恶魔体系**：主角恶魔化身的专属成长线，通过血战杀戮快速升级

### 1.2 三体系交汇点

| 交汇类型 | 表现方式 | 小说章节 |
|---------|---------|---------|
| 职业+神性 | 传奇职业者获得半神能力 | 第4卷 封神之战 |
| 恶魔+神性 | 恶魔化身通过杀戮积累神性 | 第5卷 恶魔领主的崛起 |
| 职业+恶魔 | 恶魔化身保留前世战斗技巧 | 第3卷 屠神之战 |
| 三线合一 | 本体与恶魔化身融合为死亡之主 | 第10卷第81章 |

### 3.2 通胀驱动因子

1. **杀戮经验积累**：从初期几十点一场到后期数百万一场
2. **神性/神格/神力三值叠加**：每获得一个层次都是指数增长
3. **领域数量扩张**：从1个到22个领域，每个领域提供独立法则加成
4. **兼职叠加效应**：盗贼+影武者+巫师+奥术师四线叠加
5. **装备加成**：霜巨人之力(+6)等传奇装备提供属性突破

### 4.4 兼职规则

- **兼职上限**：无硬性上限，但经验需求按职业等级递增
- **兼职门槛**：需要对应属性达标（如巫师要求智力15+）
- **经验稀释**：200万杀戮经验可把单一职业升到20级传奇，但兼职只能将两个职业升到14级
- **性价比**：专职优先获得传奇能力，兼职适应复杂环境
