# 网文创作方法论 PRD

> 本文档从12张思维导图截图沉淀而来，覆盖网文创作全链路：创意筛选 → 总纲搭建 → 金手指设计 → 书名 → 开篇 → 读者情绪与爽点 → 简介 → 高潮桥段 → 细纲 → 正文写作（对话/外貌/打斗/视角/排版） → 感情线。

---

## 1. 创作方法论

从创意到主角设定的完整流程，四个阶段依次推进。

### 1.1 筛想法

从以下5个维度筛选创意素材：

- **人物**：有张力的人物关系
- **场景**：有画面感的空间设定
- **台词**：有记忆点的金句
- **情节**：有转折的事件链条
- **脑洞设定**：有新意的世界观规则

### 1.2 提炼核心卖点

核心问题：**能给读者带来哪些情绪/共鸣？**

- 新颖设定
- 核心梗
- 人设

### 1.3 确定故事类型

根据核心卖点匹配故事类型（都市/玄幻/科幻/历史/言情等）。

### 1.4 设定主角

| 维度 | 内容 |
|------|------|
| **人物初定** | 职业、主性格 |
| **记忆点** | 外貌特征、口头禅、习惯动作、神态、金手指 |
| **完善性格** | 小缺点、成长变化 |
| **描写方法** | 肖像、语言、动作、神态、心理 |

**主角设定技巧**：

- 人设稳定（性格前后一致，不崩塌）
- 有争议点（读者能讨论、站队）
- 有记忆点（一个特征就能想起这个角色）
- 有合理的行为理由（每个行动都有动机支撑）

## 2. 总纲框架

从一句话到完整总纲的三级递进。

### 2.1 一句话故事

公式：**谁 + 想做什么 + 为什么**

### 2.2 草纲

公式：**谁 + 想做什么 + 为什么想做 + 如何做 + 阻力 + 结局如何**

### 2.3 总纲

| 模块 | 内容 |
|------|------|
| **人物** | 主角 + 配角 |
| **背景** | 世界地图 |
| **分阶段纲要** | 地图、等级、进度 |
| **感情戏大框架** | 暗恋 → 表白 → 经历磨难 → 修成正果 |

---

## 3. 金手指设计

### 3.1 作用

- 增加吸引力
- 推动情节
- 让作品有特色
- 代入感强
- 合乎逻辑
- 增加爽点

### 3.2 设置方向

| 方向 | 说明 |
|------|------|
| 主角成长路线 | 金手指随主角成长而进化 |
| 背景环境设置 | 金手指与世界规则挂钩 |
| 主角目的 | 金手指服务于主角核心目标 |
| 题材缺乏因素 | 金手指填补赛道空白 |

### 3.3 注意点

- 数量少胜过数量多
- 衍生性越强越好
- 适度，不能过强或过弱
- 要有一定成长空间
- 紧跟潮流

### 3.4 常见类型

| 类型 | 说明 |
|------|------|
| 技能 | 主角先天或后天拥有的能力 |
| 空间 | 随身空间/异空间 |
| 系统 | 系统面板/任务/兑换 |
| 外挂 | 宝物/萌宠 |

## 4. 取书名

9种书名命名策略：

| 策略 | 示例 |
|------|------|
| 直接抛题材 | 《重生...》《穿越...》 |
| 相关亮点/展现人物关系 | 《回到明朝当王爷》《女总裁的上门女婿》 |
| 核心亮点/矛盾 | 《表白失败就变强》《地铁误连别人热点后》 |
| 脑洞书名 | 《骗男孩子睡觉的100种方法》《这膝盖我收下了》 |
| 主副标题结合 | 《重生八零：媳妇有点辣》《人在综漫，开局一个模拟器》 |
| 文艺向书名 | 《寂寞空庭春欲晚》《有匪》 |
| 数字式书名 | 《玄学直播，十亿人围观》 |
| 热点式书名 | 《禁止套娃》《剑修风评被害后》 |
| 问句式书名 | 《这不是钓鱼游戏吗》《小说家多开几个马甲怎么了》 |

## 5. 吸引力开篇

### 5.1 冲突开篇

- **开局展现乱**：让主角陷入麻烦中
- **设置第一个有进无回关口**
- **设置第二个关口推动主角前进**

**冲突类型**：

| 类型 | 适用场景 |
|------|---------|
| 与自然环境 | 末世、荒年、冒险 |
| 与社会环境 | 权谋 |
| 与人 | 职场、后宫、家庭 |
| 与自身 | 身体残疾/内心缺陷 |

### 5.2 悬念开篇

| 悬念层级 | 说明 |
|---------|------|
| 核心悬念 | 主线结果 |
| 超级悬念 | 未知的势力/东西 |
| 场景悬念 | 主角在一个场景中遇到了障碍 |
| 段落悬念 | 一连串场景中的一个共同的未知 |

### 5.3 设定开篇

| 设定类型 | 内容 |
|---------|------|
| 背景设定 | 世界观、金手指、特殊经历 |
| 人物设定 | 主角核心人设展示 |

## 6. 调动读者情绪与爽点

### 6.1 调动读者情绪

| 策略 | 方法 |
|------|------|
| **读者共鸣** | 了解读者群体，写其熟悉的经历 |
| **市场搭配** | 改进市场验证的套路化人物 |
| **自我感知** | 写自己熟悉的，换位思考，代入角色 |
| **人物技巧** | 对比：刻画反派，对比主角；铺垫：利用厉害人物衬托主角 |
| **情节技巧** | 普通事件：回忆过往某个时间点发生的事情+真情流露；冲突事件：矛盾激化、绝境 |

### 6.2 爽点

**马斯洛需求层次**：从生理需求到自我实现，不同层次的需求对应不同爽点类型。

**悬念三要素**：

| 要素 | 作用 |
|------|------|
| 限制时间 | 给故事设置紧张感和节奏感 |
| 提高赌注 | 提高人物、情节、社会的戏剧冲突 |
| 微观障碍 | 不起眼小事件或人物，进场后带来巨大复杂的旁生枝节变化 |

## 7. 写简介

### 7.1 四大要素

环境、人物、冲突、悬念

### 7.2 创作技巧

- 从主角人设出发
- 抛剧情核心亮点
- 直给大爽点

### 7.3 避免雷区

- 写成大纲式简介
- 风格不统一
- 内容与正文不统一

---

## 8. 高潮桥段

### 8.1 前期准备：设置低谷期

- 重重困难
- 情绪压制

### 8.2 高潮设置

- 真相揭秘
- 主角光环
- 剧情反转

### 8.3 高潮结束

- 主角有所变化（成长/心态转变/身份变化）

---

## 9. 细纲

### 9.1 人物详细信息

为每个出场人物建立详细信息卡。

### 9.2 剧情拆解

将情节拆成**时间、地点、人物、事件、结果**五大要素，将主线大情节分成很多小情节。

### 9.3 剧情推导

1. 拟定故事初始状态及最终结果
2. 推演过程丰满情节、补全所需人物
3. 从结果逆推

### 9.4 拉长故事

| 维度 | 方法 |
|------|------|
| 时代背景 | 扩展世界观时间跨度 |
| 故事桥段 | 增加支线/单元剧 |

### 9.5 戏剧冲突

冲突四要素：**需求 → 阻碍 → 行动 → 结果**

### 9.6 引发情绪波动

方法：**找对比**（人物前后对比、角色之间对比、预期与结果对比）

### 9.7 注意点

- 细纲分配到章节中
- 剩余细纲自动调整到下一章
- 不要规定死自己一章必须写哪些内容，断在哪里
- **只写重要事件**

## 10. 正文写作

### 10.1 对话描写

**作用**：

| 作用 | 具体内容 |
|------|---------|
| 传达信息 | 人物关系、故事背景、人物观点 |
| 塑造人设 | 性格、情绪、世界观、价值观 |
| 推动情节 | 铺垫、伏笔 |

**写作要点**：

- 避免直接回答
- 压缩对白
- 提前安排对话关系
- 巧用沉默替代

**如何写**（四种提示语位置）：

1. 提示语前，内容后
2. 提示语在中间
3. 提示语在后，内容在前
4. 无提示语

### 10.2 故事流畅方法

| 维度 | 方法 |
|------|------|
| **结局** | 结局先行，所有铺垫指向结局 |
| **排版** | 段落：每段最多50个字；句子：将长句子分割成短句 |
| **描写逻辑顺序** | 动态：描绘动作轨迹；静态：远→近、上→下 |

### 10.3 视角转换

| 场景类型 | 转换方法 |
|---------|---------|
| 不同场景 | 直接切 |
| 同一场景·对手戏 | 直接切，但不要来回横跳 |
| 同一场景·非对手戏 | 先转人，再切视角；聚焦物品，再切视角 |

### 10.4 外貌描写

- 着重一个特点
- 符合观察顺序
- 贴合人物性格、身份

### 10.5 打斗描写

**作用**：

- 突出主角强（爽点）
- 让主角体会到弱小（为心态转变做铺垫）
- 突出主角经过某件事后的情绪（为获得奖励打斗/为其他情节铺垫）

**种类**：

| 流派 | 代表作 |
|------|--------|
| 写实派 | 金庸、梁羽生 |
| 意境派 | 古龙 |
| 虚实结合 | 《寻秦记》 |

**如何写**：

| 阶段 | 要点 |
|------|------|
| 打斗前 | 描写起因、导火索，做好前期矛盾铺垫 |
| 打斗中 | 想好切入点 |
| 打斗后 | 结果反馈 |

**画面感描写技巧**：

- 以点概面
- 练习电影拉片
- 侧面描写
- 增设细节
- 增加感官描写

## 11. 感情线

### 11.1 类型

- 暗恋
- 明恋
  - 一见钟情
  - 日久生情

### 11.2 设定误区

- 一来就一见钟情
- 为了恋爱而恋爱
- 为了矛盾而矛盾
- 情节俗套
- 感情太过平淡或者顺利
- 感情转变生硬

### 11.3 模版

```
初相识 → 暗恋（感情升温+降温） → 契机（感情爆破） → 表白 → 经历磨难 → 结局
```

> 创作限制：每段最多50个字（段落排版要求）。
