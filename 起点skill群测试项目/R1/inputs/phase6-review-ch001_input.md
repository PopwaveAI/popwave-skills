## 做什么

输入：本章施工卡（plot-design 产出） + 本章正文（prose-render 产出） + DNA 笔触要求 + 骨架.md（seed 产出） + 主角设计.md（seed 产出）
输出：审核报告（四维审核结论 + gap 分析 + 决策） + 本章摘要 + 状态快照 + current-state 更新

核心原则：**不是强制正文服从大纲，也不是放任正文偏离大纲，而是作为编辑判断"谁更好"。**
审核优先级：**符合性 > 笔触 > 好看度 > 沉淀。前面不通过，后面不审。**

## Step 1: 符合性检查（正文 vs 施工卡 + 骨架 + 设定）

> 第一道闸门：正文是否兑现了施工卡承诺的故事，是否遵守骨架（坐标系 + 动力引擎）的约束。
> 符合性是第一闸门——偏离核心事件（主线丢失）=废章，骨架一致性不通过=废章，天赋约束取消坐标系意义=废章。

### Step 1.0：加载审核上下文

读取：
- 本章施工卡（来自 plot-design 的幕纲）
- 本章正文（来自 prose-render 的渲染产出）
- 骨架.md（来自 seed——力量体系坐标系 + 动力引擎众生攀登系统）【v3.0.0 新增】
- 主角设计.md（来自 seed——金手指设计 + 爽感矛盾公式 + 攀登方式）【v3.0.0 新增】
- 种子文档（风格关键词/不可牺牲项/禁区）
- 流派 skill 的笔触 DNA（如 `prose-render-dndlike/dna/深渊主宰-笔触DNA.md`——叙事距离/面板格式/章节过渡/战斗模式分层/POV 规则等）
- 种子文档的不可牺牲项与禁区（用于红线检查）
- 前一章状态快照（如有——用于衔接检查）
- 世界圣经活跃层（本幕的 NPC/地点/组织/风俗/资源——检查正文是否与活跃层一致）

### 1a. 核心事件对照（gap 逐 beat 对比）——保留 v2.0.1

将施工卡的 7 节拍硬锚点和微 beat 施工表，与正文实际内容逐项对比。

对比方法：
1. 施工卡的每个节拍/beat，在正文中找到对应段落
2. 标注匹配状态：✅匹配 / ⚠️偏移 / ❌缺失 / ➕新增

输出 gap 清单：
```
| beat# | 施工卡内容 | 正文实际 | 状态 | gap类型 |
|-------|-----------|---------|------|---------|
| 1 | 催收员隔门喊话 | 催收员隔门喊话 | ✅ | — |
| 7 | 决定今晚动手→磨刀 | 暗杀执行→翻尸体→面板结算 | ⚠️ | 提速（吞并ch04） |
| + | — | 索伦去十字巷获取情报 | ➕ | 新增（大纲无） |
```

#### gap 分类（保留 v2.0.1 的 5 种类型）

| 类型 | 含义 | 典型场景 |
|------|------|---------|
| **提速** | 正文把后续章节的内容提前写了 | ch03 写了 ch04 的暗杀执行 |
| **延迟** | 正文在一个 beat 上花了过多篇幅，没写完本章该写的 | ch02 花 3000 字写面板激活，没写到门上催收刻痕 |
| **偏离** | 正文走了和施工卡不同的方向 | 施工卡写"暗杀"，正文写了"谈判" |
| **遗漏** | 正文跳过了施工卡中的某个 beat | 施工卡有"翻尸体发现纸条"，正文没写 |
| **新增** | 正文写了施工卡中没有的内容 | 正文新增了一条"格里芬势力线" |

#### 质量判断（保留 v2.0.1）

对每个 gap，回答一个问题：**正文版本更好还是大纲版本更好？**

判断维度：
1. **叙事自然度**：正文版本是否比大纲分章更自然？
2. **读者体验**：正文版本是否给读者更好的阅读体验？
3. **爽感完整性**：正文版本是否保持了爽感的完整闭环？
4. **后续影响**：如果采纳正文版本，后续施工卡需要怎么调整？调整成本是否可接受？
5. **字数合理**：正文是否在 2000-2500 字范围内且不超过 3000 字？

判断结论（每个 gap 一个）：
- **正文更好**：采纳正文，调整后续大纲
- **大纲更好**：标记正文需要修改，附修改建议
- **各有优劣**：标注优劣，建议人工裁决

**核心事件判定**：
- 全部兑现 → 进 1b
- 有偏离但主线未丢 → 标注偏离，进 1b（下游需评估偏离是否可接受）
- 主线丢失（核心事件未写/被替换成别的） → **废章，打回重写**

### 1b. 骨架一致性检查（新增）

打开骨架.md，对照本章正文检查骨架约束是否被遵守。骨架（力量体系 + 动力引擎）是硬约束，不通过不得放行。

#### 1b-1. 力量体系一致性
- 坐标系是否被遵守：主角/配角的能力表现是否在骨架.md 力量体系四层结构（世界源力量/主养成线/子养成线/交叉规则）定义的阶位范围内？
- 越级检查：有没有越级但不付代价？（越级必须通过金手指加速 + 代价兑现，不能凭空跨越坐标系阶位）
- 属性数值检查：面板数值/等级是否与坐标系阶位刻度一致？有没有属性偷跑？

#### 1b-2. 动力引擎一致性
- 众生攀登逻辑有没有断裂：本章的攀登行为是否符合骨架.md 动力引擎组成 3（众生攀登方式分层）？主角的攀登方式是否与众生通用闭环一致（输入→转化→输出→反馈，只是更快）？
- 驱动逻辑连续性：本章行为动机是否与骨架.md 组成 1（驱动逻辑——世界级驱动力）对齐？
- 代价结构兑现：本章的攀登是否兑现了骨架.md 组成 4（代价结构——通用代价 + 群体差异代价 + 累积效应）？

**判定**：
- 通过 → 进 1c
- 不通过 → **废章，打回重写**（骨架一致性是硬约束，越级不付代价/攀登逻辑断裂/驱动逻辑脱钩=废章）

### 1c. 天赋约束检查（新增）

打开主角设计.md 的金手指设计（2b），对照本章正文检查：

| 检查项 | 判定标准 | 不合格处理 |
|--------|---------|-----------|
| 金手指不喧宾夺主 | 金手指是否让坐标系失去意义？是否取消了众生攀登的意义？ | **废章**（金手指取消坐标系意义=废章） |
| 限制兑现 | 金手指限制（主角设计.md 2b 声明的"不能取消坐标系本身的意义"等限制）是否被遵守？有没有突破声明的限制？ | 标注越界点，打回修改 |
| 代价兑现 | 金手指使用是否付出了代价（爽感矛盾公式的"代价约束"因子）？有没有无代价白嫖？ | 标注未付代价点 |

**判定**：
- 通过 → 进 1d
- 金手指取消坐标系意义 → **废章**
- 限制/代价未兑现 → 标注越界点，打回修改

### 1d. 剧情映射检查（新增）

检查正文中的血肉（地图/势力/人物/剧情）是否映射骨架坐标系，有没有脱离坐标系的无效填充：

| 检查项 | 判定标准 | 不合格处理 |
|--------|---------|-----------|
| 地图映射 | 本章出现的地点是否在坐标系中有位置意义？（不是随便一个地名，而是坐标系中的某个阶位/势力范围） | 标注脱离坐标系的地点 |
| 势力映射 | 本章涉及的势力是否在坐标系中有层级定位？势力关系是否与坐标系一致？ | 标注脱钩势力 |
| 人物映射 | 本章出场人物是否在坐标系中有位置？（不是路人甲，而是坐标系某阶位/某攀登方式的代表） | 标注无效人物 |
| 剧情映射 | 本章事件是否推动坐标系内的攀登/博弈？有没有与坐标系无关的无效剧情填充？ | 标注无效剧情 |

**判定**：
- 通过 → 进 1e
- 不通过 → 标注脱离坐标系的血肉清单，建议删改

### 1e. 角色一致性检查

对照角色库 + 主角设计.md 的攀登方式，检查：

| 检查项 | 判定标准 | 不合格处理 |
|--------|---------|-----------|
| 角色性格一致性 | 角色行为是否符合角色库中的性格描述 | ✓/✗（如偏离，写明哪句不符合） |
| 攀登方式一致性 | 角色行为是否符合其攀登方式类型（对应骨架.md 组成 3 众生攀登方式分层中的某一种） | ✓/✗（如偏离，标注哪句不符合） |
| 角色出场连续性 | 分幕设计标注该出场的角色是否全部出场 | ✓/✗（如缺失，写明谁没出场） |
| 角色状态连续性 | 角色力量层级/关系状态是否前后衔接 | ✓/✗（如断裂，写明哪里不衔接） |
| 面板使用检查 | 战斗/升级场景是否使用了 DNA 面板格式 | ✓/✗（如缺失，标注哪个场景缺面板） |

**角色不一致 → 废章标记，打回修正**。

### 1f. 爽感闭环检查

#### 三段式检查
- 本章主推爽感类型：______
- 触发段：有/无（位置）
- 爆发段：有/无（位置）
- 后果段：有/无（位置）
- 三段缺一 → 废章标记，打回补全

#### 爽感因子检查（新增）
爽感矛盾公式（坐标系门槛 × 天赋加速 × 代价约束）三因子，本章是否触发了至少两个？
- 坐标系门槛因子：是否体现了跨越坐标系阶位的难度？
- 天赋加速因子：是否体现了金手指加速跨越？
- 代价约束因子：是否体现了金手指代价/限制带来的张力？
- 触发因子数：__ / 3（<2 → 爽感强度不足，标注建议补足哪个因子）

### 红线检查（保留 v2.0.1）

检查正文是否违反硬性红线（不管 gap 判断结果，红线违反必须修改）：

| 红线 | 检查方法 |
|------|---------|
| 隔离红线 | 正文中是否出现原作专有元素（非同人测试模式下） |
| 篇幅红线 | 正文字数是否在 DNA 要求的范围内 |
| 主动行为红线 | 本章是否有 ≥1 个主角主动行为 |
| 面板格式红线 | 面板是否用引号包裹系统消息嵌入正文（非代码块） |
| 设定强调纪律 | 同一设定约束是否在一章中提及 >1 次 |
| 战斗多样性红线 | 是否连续 3 章使用同一种战斗模式 |
| 三层世界红线 | 正文中出现的地名/NPC/组织/风俗是否在活跃层有记录？正文是否现编了未落盘的世界细节？ |

## Step 2: 笔触检查

### 2a. AI 味 7 项检查（从番茄 review 移植）

定性判断，不打分。逐条检查：

| 检查项 | 判定标准 | 不合格处理 |
|--------|---------|-----------|
| 句式重复 | 同一句式连续使用 2 次以上？ | 标注位置 + 建议替换 |
| 情绪直给 | "他感到 XX""心里一 XX""不由得"？ | 标注位置 + 建议用动作展示 |
| 结尾收束 | 章末停在总结性旁白？ | 标注 + 建议停在动作或对话 |
| 描写堆叠 | 连续 3 行以上纯感官描写无动作推进？ | 标注 + 建议插入动作 |
| 信息重复 | 同一信息一章内出现 2 次以上？ | 标注 + 建议删除 |
| 对话死板 | 连续 6 行以上纯对话无动作？ | 标注 + 建议穿插动作 |
| 高疲劳词 | 冷笑/蝼蚁/轰然炸裂/倒吸凉气/瞳孔骤缩/满场死寂等单章 ≥2 次？ | 标注 + 建议替换 |

### 2b. 笔触 DNA 一致性检查（保留 v2.0.1 的 8 维度）

对照 DNA 笔触要求，检查正文的笔触符合度：

| 维度 | 检查内容 |
|------|---------|
| 叙事距离 | 是否以主角视角为主体（≥60%）？镜头切换是否有标志且快速切回？ |
| 对话引导词 | 是否使用 DNA 指定的引导动词？（如深渊主宰 DNA 用"道"不用"说"） |
| 面板弹出方式 | 是否作为意识流数据流嵌入正文？是否不停顿动作节奏？ |
| 章节过渡 | 是否使用了正确的过渡模式？是否连续两章使用相同过渡？ |
| 感官序列 | 新场景首次出场是否使用了感官序列？非新场景是否避免了感官序列？ |
| 情绪收束 | 杀完人后是否有冷评收束（或首次杀人的递减式心理波动）？ |
| 参照物梯子 | 面板数值弹出时是否用参照物建立认知标尺？ |
| 主角定位 | 面对威胁时是否先算后动？情绪是否通过瞳孔/呼吸/手指表达而非面部？ |

## Step 3: 好看度检查（定性 4 问，新增）

> 这一步最主观也最重要。不要被技法 checkboxes 蒙蔽——技法全 ✓ 不代表好看。

逐条回答，必须给出具体证据，不能泛泛说"还行"：

1. **有没有劲？**（读完这章有没有兴奋感）
   - 有 → 引用最抓人的 1-2 句正文
   - 没有 → 问题在哪？（节奏太慢/信息太少/主角太怂/废话太多/场景太常见）

2. **记忆点？**（读完能记住什么具体画面/台词/反转）
   - 找得到 → 引用原文
   - 找不到 → 本章缺乏记忆点，标注"无记忆点"

3. **哪里无聊？**（哪段想跳过）
   - 标注具体段落位置（如"第 3-5 段屋顶夜景描写偏长"）
   - 建议：删/压缩/插入动作/换场景

4. **代入感？**（有没有跟主角共情的瞬间）
   - 读者能想象出画面吗？（哪段画面感强/弱）
   - 主角的选择和读者会重叠吗？（主角是否主动决策）
   - 配角像活人吗？（有没有独立动机/小动作）

## 红线

1. **gap 必须逐 beat 检查**：不能只看总体感觉，必须逐个 beat 对比。跳过 beat 对比=不合格。
2. **质量判断必须有理由**：每个"正文更好"或"大纲更好"的判断必须附理由（哪个维度更好、为什么）。无理由的判断=不合格。
3. **红线违反不可妥协**：即使正文在叙事上更好，如果违反了硬性红线（如面板用代码块而非引号），仍需修改。
4. **骨架一致性检查不通过=废章**（新增）：骨架（力量体系 + 动力引擎）是硬约束，正文越级不付代价/动力引擎攀登逻辑断裂/驱动逻辑脱钩=废章，必须打回重写。
5. **天赋约束检查发现金手指取消坐标系意义=废章**（新增）：金手指喧宾夺主，让坐标系失去意义/取消众生攀登意义=废章。金手指是引擎加速器，不是引擎本身。
6. **摘要必须客观**：摘要只记录"发生了什么"，不评价"写得好不好"。
7. **状态快照必须准确**：状态快照的数据必须和正文中的面板数据一致。如果正文有属性偷跑，快照标注"正文数据异常"。
8. **主角变化必须每章沉淀**（新增）：位置/能力/资产/心态/关系五项缺一不可，不沉淀=状态断裂。
9. **钩子追踪必须列待回收列表**（新增）：超出 3 章未回收的钩子必须标红预警。
10. **不打分**（新增）：定性判断"通过/需修改/废章"，不给百分比或星数。

# 第一章：觉醒业火

残阳把妖兽山脉的碎石坡染成暗红色。

沈劫蹲在一块巨岩后面，手里的猎刀已经握了半个时辰，掌心全是汗。他盯着前方那个灰袍散修的背影——那人正蹲在三十步外的一处灌木丛旁，往地上撒一种暗绿色的粉末。

引兽粉。

沈劫七岁起就在落星城街头摸爬滚打，见过太多这种套路：散修用凡人当诱饵，引妖兽出来，然后坐收渔利。被骗进山的凡人，十个里活不下来两个。

他知道眼前这人嘴里没一句真话。昨晚在酒馆，这人拍着桌子吹嘘“山脉深处有座古墓，宝贝多得装不下”，满屋子的人都在笑，只有沈劫注意到他说“古墓”时眼神往旁边飘了一下。

撒谎的人在编细节时会不自觉看别处。

所以当这人今天找上他，笑眯眯地说“小兄弟，帮我望风，得了宝贝分你三成”时，沈劫心里已经有了答案。

但他还是来了。

因为他在赌——赌自己在关键时刻能反杀。

“小子，你还在磨蹭什么？”灰袍散修回头喊道，脸上挂着不耐烦的笑，“再不走天就黑了，山里晚上可不是闹着玩的。”

沈劫没动。他的目光落在散修腰间的玉佩上——劣质货色，最多值三两银子。一个敢进山寻宝的散修，就这点家当？

骗子的装备越好，骗术越大。装备越差，越急着干一票就跑。

这人的玉佩连好货色都算不上，说明他不是刚入行，就是快混不下去了——无论是哪一种，都意味着他会毫不犹豫地拿人喂妖兽。

“来了。”沈劫站起身，猎刀别在腰间，低着头跟上去。

他刻意落后三步，保持距离。

散修瞥了他一眼，没说话。

两人一前一后沿着山道往深处走。越往里，树越密，光线越暗。空气里开始弥漫一股腥臭味——那是大型妖兽的领地气味。

沈劫后颈的汗毛竖起来。他闻得出来，这不是外围那些低阶妖兽的味道。至少是灵级中期。

淬血开脉级的猎物，引的却是灵级妖兽？

——这人要的不是“寻宝”，是拿他喂妖兽，等妖兽进食时偷袭。

沈劫握紧刀柄，心跳加快，但呼吸没乱。

他等的就是这个。

“就在前面了。”散修指着前方一处被藤蔓覆盖的石壁，“古墓入口就在那后面，我之前探过，里面有个机关我一个人打不开……”

散修的话突然停住了。

因为草丛里亮起了一双琥珀色的竖瞳。

赤纹豹。

灵级初期，肩高四尺，体长一丈有余，浑身覆盖着赤红色的斑纹，正压低身体，尾巴缓缓摆动。

它被引兽粉的气味引来了。

“操，这么快……”散修骂了一声，回头朝沈劫喊，“小子，快往那边跑！把它引开！”

沈劫没跑。

他站在原地，看着那头赤纹豹，又看了看散修。

散修脸上的慌张太假了——一个能在妖兽山脉活下来的散修，见到妖兽的第一反应是为它准备好了法器，而不是让一个凡人去送死。

沈劫往前迈了一步。

不是往赤纹豹的方向，而是往散修的方向。

“你干什么？！”散修脸色变了，“你往我这边——”

沈劫没等他说完。

他转身就跑。

但不是往山下跑，而是直接冲向了散修的背后。

散修下意识想拔刀，但他低估了一个在街头摸爬滚打十年的孤儿对“时机”的判断力——沈劫冲到他面前时，他刚摸到刀柄，腰还没转过来。

沈劫横刀架上了他的脖子。

“别动。”

散修僵住了。赤纹豹还在三十步外缓缓逼近，低吼声在山谷里回荡。

“小子，你知道我是谁吗？”散修的声音发紧，但还在强装镇定，“我是落星城——”

“淬血开脉·第二轮，业力刚满两年。”沈劫的刀紧贴着他的喉咙，声音平静得不像一个十七岁的少年，“你在酒馆吹牛的时候说的。”

散修的笑容僵住了。

“你还说过，六个月前你单杀了一只灵级中期的铁背熊。”沈劫继续说，“但铁背熊只生活在妖兽山脉中圈，外围根本不会有。一个第二轮散修孤身进中圈？你活着出来的概率不到一成。”

散修的脸色彻底沉下来：“你怎么知道这些？”

“我做了功课。”

——金手指“九世杀生忆”在昨晚第一次触发了。

沈劫在酒馆角落里睡觉时，脑海里突然闪过一段画面：一个男人跪在这片碎石坡上，胸口被赤纹豹的利爪撕开，血从指缝渗出来，用尽最后力气说：“兄弟……别相信散修……他们拿凡人当饵……”

画面戛然而止。沈劫惊醒时浑身冷汗，那个男人的脸他已经记不清了，但那股绝望和愤怒却烙在了骨头里。

那是前世死在同一个散修陷阱里的亡魂。

他不知道这是金手指，也不知道为什么能看到这些。但他知道一点：眼前这人，手上不干净。

“放下刀，我们可以谈谈。”散修的声音软下来，“那古墓是真的，我用道心发誓——”

“你有道心吗？”

沈劫一句话堵住了他的嘴。

散修的眼中闪过一丝狠厉，右手无声地摸向腰间——那里藏着一枚符篆。

淬血开脉级的散修随身带着的应急法器，大多是劣质火符，激活后能炸开方圆五步的范围。够把凡人炸死，但对修士自己也是两败俱伤。

散修赌的是沈劫一个凡人，不会知道符篆的用法。

但他赌错了。

沈劫在落星城混了十年，见过的散修来来去去，大部分人的“底牌”就那么几种——符篆是其中最常用的。他早就注意到散修腰间那个不起眼的布袋里装的是符纸。

所以在散修摸到符篆的那一刻，沈劫已经出刀了。

不是割喉，而是用刀背重重砸在散修的右肩关节上。

“咔嚓”。

骨裂声清晰可闻。散修发出一声惨叫，右手软软垂下，符篆滚落在地。

赤纹豹被吼声刺激，猛地扑过来。

沈劫一脚将散修踢倒在地，自己往侧面翻滚。

赤纹豹的目标不是沈劫——它被引兽粉的味道吸引，而散修身上沾了大量的引兽粉。

这本来就是散修自己的陷阱。

扑来的赤纹豹一口咬住了散修的小腿，骨头碎裂的声音在寂静的山谷中格外刺耳。

“啊啊啊——！”

散修的惨叫声撕破天际。他拼命拍打豹头，用左手拔刀乱砍，但灵级初期的妖兽皮糙肉厚，几刀下去只砍出几道白痕。

沈劫没有逃走。

他蹲在十步外的巨岩旁，冷静地看着这一幕。

散修和赤纹豹之间的战斗大约会持续二十息——赤纹豹会先啃碎散修的左腿，然后在他挣扎时咬断他的脖子。散修大约还能活十五息。

要出手，就要在这十五息之内。

沈劫算过：赤纹豹的头部是它防御最硬的地方，但它的左后腿根部有一小块没有鳞甲覆盖的区域，那里是经脉交汇点，一刀刺入可以废掉它的后肢。

但那是“理想情况”。

实战中，一只受伤发狂的赤纹豹会在三息之内转身反击，他只有不到一息的出手时机。

沈劫深吸一口气。

他打开了猎刀上的绳扣，将刀柄缠在手腕上。

“就是现在。”

赤纹豹正低头咬住散修的左肩用力撕扯，整个左后腿暴露在外，没有防护。

沈劫冲了出去。

他的脚步很轻，落地的节奏卡在赤纹豹撕扯时发出的低吼声中——这是小时候在树林里追野兔时练出来的本事。

四步。

三步。

两步。

赤纹豹的左后腿根部，那块巴掌大的无鳞区，已经近在眼前。

沈劫出刀。

刀尖精准地刺入了经脉交汇点，从下往上斜挑，切断了三条主要经脉。

赤纹豹发出一声震耳欲聋的咆哮，猛地转身，一爪拍向沈劫。

沈劫来不及拔刀，只能双手交叉挡在身前。

“砰——”

他被拍飞出去，重重撞在树干上，喉头一甜，一口血喷出来。

赤纹豹的后腿已经废了，但它还有前爪和獠牙，它一瘸一拐地朝沈劫挪过来，眼中泛着嗜血的凶光。

沈劫靠着树干挣扎起身，右臂被豹爪抓出三道深可见骨的伤口，血沿着手指滴落。

他没有刀了。

猎刀还插在赤纹豹的左后腿上。

赤纹豹张开满是腥臭涎水的大口，朝他扑来——

一瞬间，第二段记忆碎片猛地炸开：

一双手在翻动一本泛黄的册子，上面写着：“赤纹豹·灵级初期·弱点：左后腿经脉交汇点（需一击切断至少三条经脉才能瘫痪行动力）。若第一次攻击未造成瘫痪，它会进入狂暴状态，速度提升三成，持续十息。”

沈劫瞳孔微缩。

十息。

他必须在十息之内，从这只发狂的赤纹豹身上，把那把刀拔出来。

赤纹豹再次扑来。

沈劫没有后退。他身体往左侧倾斜，险之又险地擦过豹爪，然后一脚踩在废掉的左后腿上，借力腾空翻滚，双手准确无误地抓住了刀柄——

用力一拔！

血花溅了他一脸。

赤纹豹吃痛，回头咬向他，但左后腿的瘫痪让它的动作慢了半拍。

沈劫没有犹豫。

刀锋从下颚斜刺入脑干。

赤纹豹的身体猛地僵硬，然后轰然倒下，抽搐了两下，不动了。

山谷安静了下来。

只有沈劫粗重的喘息声和散修越来越微弱的呻吟。

沈劫走过去。

散修还活着，但左腿已经被赤纹豹啃得只剩白骨，血泊越扩越大。他看见沈劫走过来，眼中闪过一丝哀求：“救……救我……”

沈劫看着他的眼睛，没有动。

十几息之后，散修的眼睛失去了焦距。

就在这时，沈劫体内猛然一震——一股无形的力量从散修的尸体中涌出，像一条温热的气流，穿过空气，钻入他的毛孔。

那是一种从未体验过的感觉。

像全身的血液都在沸腾，又像浸在温水中，每一个细胞都在贪婪地吸收着这股力量。沈劫的身体开始发烫，皮肤表面浮现出淡红色的纹路，从心脏位置向外蔓延。

业火。

这就是业火。

淬血开脉·第一轮，觉醒。

沈劫跪在地上，双手撑着碎石，身体里的能量像野马一样奔涌，他体内的“灵池”——那个一直被封印的地方——在这一刻被冲开了。

他也看到了第三段记忆碎片：

一只手掌摊开，掌心有一滴暗红色的液体在旋转。

“第一滴业火，是你的‘根’。它会记住你第一次杀人时的情绪——恐惧、愤怒、快感。每一滴业火都对应一段记忆。你杀的人越多，业火越浓，你的记忆就越模糊。”

“这就是代价。”

沈劫猛地睁开眼。

他低头看着自己的右手——指尖还残留着未干的豹血，虎口被刀柄磨出了血泡，掌心的纹路里渗着丝丝黑色。

那是浊气残渣。

他忽然意识到一个问题——

刚才那三段记忆碎片里的“人”，是谁？

那个跪在赤纹豹面前的男人，那个翻看妖兽册子的修士，那个说“业火会记住你第一次杀人情绪”的苍老声音——

他们，都死过。

金手指“九世杀生忆”的第一个代价，已经降临。

他发现自己怎么也想不起母亲的脸了。

那个藏在尸堆中时，最后看见的母亲的面容，在这场战斗后，变得模糊了。

好像隔着一层磨砂玻璃，能感觉到她的存在、她的温度、她最后的拥抱，但具体的长相——五官的位置、眼睛的颜色、嘴角的弧度——

想不起来了。

沈劫跪在血泊里，眼泪无声地滑落。

但他没有哭出声。

因为他知道，这只是开始。

今后他还会失去更多。

而他能做的，就是趁还来得及，记住自己是谁。

夕阳西沉，妖兽山脉的影子越拉越长。

沈劫站起身，从散修的尸体上摸出两本残破的功法册子和几枚劣质灵石，又从赤纹豹的体内取出一枚指甲盖大小的妖丹，用布包好收进怀里。

他回头看了一眼这片被血浸透的碎石坡。

然后转身，朝着山下那座烟火缭绕的城市走去。

那座城市，叫业火城。

那是散修的聚集地，是整个九州南部最混乱、最残酷、也最有机会的地方。

他不知道自己会在那里遇到什么人、经历多少场生死。

但有一件事他无比确信：

他一定要活下去。

活到足够强为止。

## 2b 金手指设计

### 金手指类型+名称
- **类型**：穿越知识（结合重生信息差——不完全是重生，是获得了未来某条时间线的记忆碎片）
- **名称**：**九世杀生忆**

### 加速机制
- **加速点**：骨架.md组成2的“反馈”环节——主角在杀戮→吸收浊气→晋升的循环中，金手指不增加浊气量，而是让主角知道“杀谁收益最大、怎么杀风险最低”，同时提供“前世记忆中的关键节点”作为情报信息差。
- **加速方式**：金手指让主角在每次遭遇强大敌人/机缘时，触发“记忆碎片”——可能是一段未来某条时间线的影像（某个已被杀死的合道修士的击杀录像）、一个隐藏的弱点（这条龙的心口有一块逆鳞）、一个危险地点（这个秘境里面有陷阱）。本质上是一条只有他能看的“攻略”。
- **与众生攀登方式关系**：主角的攀登方式（散修的野路子）不变——他依然要靠自己找猎杀目标、自己生存、自己承担风险。金手指只是放大了“选目标”这个环节的效率，让他少踩坑、多踩点关键收益。这就如同一把好刀让猎人效率更高，但猎人还是得自己进山。

### 限制（红线级）
- **使用限制**：记忆碎片并非随叫随到——只有在主角遇到“前世时间线中的关键节点”（如某个曾经存在过的强大修士的陨落地、某个秘境的开启前兆）时，金手指才被动触发回忆片段。平时无法主动查询。
- **成长限制**：金手指记忆不全——主角只能看到“某条时间线中某个人的视角”，视角有限（不是全知全能的“系统提示”），且存在视角盲区（被杀的修士看不到自己被杀的真相）。因此他的“攻略”总有残缺，要靠自己补全。
- **坐标系意义保留声明**：**金手指不取消坐标系本身的意义——主角仍然需要一步步淬血开脉→业火炼神→煞灵化形……每层都需要亲手积累足够的杀戮浊气，金手指只提供“打哪儿更快”的情报，不提供“打完后直接晋级”的捷径。**

### 代价
- **单次代价**：动用记忆碎片后，主角自身会暂时失去对应的一段记忆（以“遗忘”为代价——可能是忘了某个不重要的人的脸、某个学过的低阶神通口诀、某个去过的地方的路）。这让他每次动用金手指都“丢失一部分自己”。
- **累积代价**：记忆碎片用得越多，丢失的记忆越多。到后期，他可能连自己为什么要变强（保护重要的人）都忘了，变成一个只知道杀戮任务的机器。他必须在变得更强和保持人性之间做抉择——每次使用金手指，都可能是在“抹除自己的一部分”。
- **代价来源对齐**：与骨架.md组成4（代价结构）对齐——通用代价是“心魔侵蚀”（杀戮导致人性流失），金手指的代价是“记忆丢失”（放大心魔侵蚀的具象化表现——你不是单纯变成业兽，你是慢慢忘了自己是谁）。代价是对通用代价的放大和戏剧化。

### 与引擎关系
- **金手指在引擎中的定位**：组成2（运转机制）的加速器——是“反馈”环节的增强（主角能从死亡修士的记忆中看到攻略，避免走弯路），不是引擎本身。
- **引擎闭环是否改变**：不改变——主角仍然依照“杀戮→吸收浊气→晋升”的通用闭环修炼。金手指不改变这个闭环，只让他在闭环内走得更准更快。
