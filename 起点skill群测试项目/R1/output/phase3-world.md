# 地图.md（第一卷区域）

## 九州南部·业火城周边三维地图

```
三维坐标：(业位层 × 空间层 × 时间层)
- 业位层：淬血开脉(1) → 业火炼神(2) → 煞灵化形(3)
- 空间层：地表(凡人城/散修窝) → 地下(妖兽巢穴) → 裂隙(秘境入口)
- 时间层：白天(相对安全) → 黄昏(狩猎高峰期) → 夜晚(妖兽活跃期)
```

---

### 区域1：业火城（散修市场·核心枢纽）

| 字段 | 内容 |
|:--|:--|
| **对应阶位** | 业火炼神（第2业位）——但城内混杂淬血开脉（占60%）到煞灵化形（占5%），是底层散修的集散地 |
| **攀登资源** | ①任务公示板（悬赏妖兽/寻宝/杀人，获得业火榜积分+浊气）②黑市（妖兽材料/残缺神通/情报交易）③散修血战擂台（同阶厮杀，胜者获败者部分浊气） |
| **空间法则** | ①**业火结界**：结界内不能使用高阶神通（＞煞灵化形级），违规者被业火灼烧→逐出城；②**城主豁免**：业火榜排名前10的散修可在城内动武，不受结界限制。 |
| **主角行为** | ①在任务板接第一单妖兽猎杀；②用金手指记忆碎片预判某悬赏妖兽的弱点/位置；③在血战擂台利用信息差碾压同阶获取第一桶金。 |
| **信息差** | 前世记忆中该城将在一个月后遭遇妖兽潮（乘着灵脉波动的空隙突破结界），主角可利用这个时间窗口提前布局。 |

### 区域2：妖兽山脉（主要猎场·死亡边缘）

| 字段 | 内容 |
|:--|:--|
| **对应阶位** | 淬血开脉→业火炼神初期（山脉外围）；业火炼神中期→后期（山脉中圈）；煞灵化形→结丹初（核心区域） |
| **攀登资源** | ①妖兽命数浊气（优先级最高）②妖兽灵骨/妖丹/材料（换钱/换神通）③山脉中的隐藏秘境入口（金手指可提示坐标） |
| **空间法则** | ①**圈层禁制**：核心区域被天然屏障阻断，没有煞灵化形以上修为无法硬闯（强行进入者被煞气腐蚀业位）；②**潮汐灵脉**：每15天水脉波动一次，高阶妖兽活动范围外扩，低阶妖兽躲入巢穴，是“捡漏”黄金期。 |
| **主角行为** | ①以淬血开脉期猎杀同阶妖兽积累第一轮业火；②灵脉潮汐期冒险深入中圈猎杀高阶落单妖兽→金手指提示弱点+关键一击；③躲避“猎杀团”追杀——被散修团伙当目标盯上。 |
| **信息差** | 前世记忆中有一条“灵脉潮汐当日的峡谷捷径”，中圈妖兽会涌入一个密闭洞穴躲避潮汐——主角可提前布置陷阱，一网打尽数十只中阶妖兽。 |

### 区域3：旧战场遗迹（秘境入口·高收益高风险）

| 字段 | 内容 |
|:--|:--|
| **对应阶位** | 业火炼神中期以上——但淬血开脉期也可通过金手指信息差“摸进去” |
| **攀登资源** | ①上古修士遗骸→炼化获得“神通真意”残印（比猎杀掉落高5倍概率）②秘境内部有未开采的灵脉矿（≈直接吸收+300%浊气的稀罕物）③某位陨落金丹的洞府（含业位晋升关键情报） |
| **空间法则** | ①**记忆残影**：遗迹内残存上古修士的杀意碎片——低阶修士触碰会直接触发幻境（心魔冲击）。必须用金手指记忆碎片规避这些“回忆陷阱”。②**时间裂隙**：遗迹内时间流速不同（外界1天=遗迹内3天），可加速练级但加速使用金手指代价（每次记忆碎片触发后，丢失2倍的记忆量）。 |
| **主角行为** | ①金手指提示“遗迹入口将在7天后因灵脉波动短暂开启”→提前准备进入；②在遗迹内利用记忆碎片规避致命幻境，获得炼体材料+神通残印；③在核心洞府遭遇其他“有情报”的散修（小boss级人物），利用信息差反杀。 |
| **信息差** | 前世记忆中该遗迹的核心宝藏是一枚“煞灵卵”（可提前炼化为第二煞灵，跳级至煞灵化形初期），但炼化需要牺牲一段关键记忆（忘了父亲的脸）。 |

### 区域4：凡人城·落星城（低风险·零收益）

| 字段 | 内容 |
|:--|:--|
| **对应阶位** | 淬血开脉以下（凡人级）——修士入城会被压制到淬血开脉初期 |
| **攀登资源** | ①凡人城的“情报网络”（佣兵工会、暗杀组织）提供修士世界的情报中转；②黑市低阶材料（妖兽灵骨/残破法器）可换金银，金银可换修士的低阶丹药/材料 |
| **空间法则** | **业位压制**：九州天道禁令禁止修士在凡人城大规模屠杀凡人——杀一个凡人获得的浊气≤1点，但屠城会触发天道清缴（业火降世烧死屠城者）。这是保护底层凡人的最后防线。 |
| **主角行为** | ①用金手指寻找落星城中混迹的“散修卧底”（前世记忆中此人背后是一个小型猎杀组织，主角可先一步接触获取合作机会）；②购买进山装备/药材；③穿越到业火城中转。 |
| **信息差** | 落星城的暗杀组织“夜鸦”首领是前世记忆中的某个潜力散修（后来成为煞谷战场的中坚），可提前交好建立人脉。 |

# 势力.md（第一卷势力·标注攀登方式类型）

## 底层势力：淬血散修群（攀登方式类型：没资源的人·野路子）

| 字段 | 内容 |
|:--|:--|
| **攀登方式** | **野路子型（没资源的人）**——无宗门/无家族，靠潜伏凡人城暗杀/偷渡进山脉/赌命进秘境的亡命徒。不修炼杀戮技巧，而是杀戮本能。死亡率高达70%。 |
| **代表人物** | ①**刀疤刘**（淬血开脉第9轮）：业火城附近最大的散修团伙头目，控制妖兽山脉外围5条猎杀路线，经营“情报换命”的黑市交易。②**哑女**（淬血开脉第4轮）：靠伪装成猎物引散修入陷阱反杀的独行浪客。③**老三**（淬血开脉第2轮）：主角进入业火城后的第一个“盟友”，实际是刀疤刘的探子。 |
| **控制资源** | ①妖兽山脉外围的猎杀路线（对新手散修收取“过路费”10%的猎杀收益）②凡人城的低阶情报网络（哪个区域有落单妖兽/哪个秘境偷偷开了入口）③少量初阶妖兽灵骨和残缺法器（从尸体上扒的） |
| **弱点** | 内部不团结——刀疤刘和一众中小团伙有利益冲突（猎杀路线分级收费、新手容易被讹诈），动辄血拼不死不休。小说第一卷末因为主角的“攻略”被刀疤刘盯上，但底层散修之间缺乏凝聚力，容易被挑拨离间。 |
| **与上层关系** | 被业火城的中层势力**城主府**视为可有可无的“韭菜”——每个月派管事来收一次“保护费”（妖兽材料抵资），不收就灭。底层散修不敢还手，因为城主府背后有铁山宗（煞灵化形以上）。 |

---

## 底层→中层过渡势力：业火城猎杀团（攀登方式类型：没资源的人·伪组织化转型）

| 字段 | 内容 |
|:--|:--|
| **攀登方式** | **伪组织化型（没资源的人转型）**——几个各有专长的散修组成小队，分工合作（坦克控怪/刺客背刺/治疗救护），通过“围猎”击杀更高阶妖兽，收益按贡献分配。是底层散修中唯一能存活超过3年的群体。 |
| **代表人物** | ①**雷老虎**（煞灵化形初期）：业火榜排名第3，公开组队模式的首创者，经营着“猎杀训练营”培养新手散修。②**阿土**（业火炼神第7轮）：雷老虎的副队长，负责侦察和陷阱布置。③**老烟鬼**（业火炼神第5轮）：临时加入主角团队的散修，擅长阵法辅助。 |
| **控制资源** | ①妖兽山脉中圈的部分猎杀路线（雷老虎带队打通7条，每条月供30只中阶妖兽）②业火城的“猎杀许可”（城主府批准的官方猎杀证，免上缴保护费）③猎杀训练营的学员资源（新手散修加入团队可快速提升战力） |
| **弱点** | 依赖性太强——雷老虎一旦战死或者离开业火城，猎杀团立刻分裂。内部有矛盾：老人（雷老虎嫡系）压榨新人（主角这种临时加入的）。主角中期和雷老虎团队结盟，但后期因为利益分配产生裂痕。 |
| **与上层关系** | 城主府与之合作——猎杀团为城主府提供稳定的高阶妖兽材料（城主府用来开发神通残印），城主府为猎杀团提供装备和庇护。本质是城主的“白手套”，高层（铁山宗）默许这种寄生关系。 |

## 中层势力：铁山宗外门（攀登方式类型：有资源的人·分配型）

| 字段 | 内容 |
|:--|:--|
| **攀登方式** | **分配型（有资源的人）**——宗门子弟精挑细选，有长老传授功法、有妖兽繁殖区提供批量猎杀机会、有丹药辅助修炼。效率是散修的5-10倍，安全是散修的100倍（有人帮忙疗伤/守护）。但修炼成果70%上交给宗门，只剩30%自己用。 |
| **代表人物** | ①**铁山宗外门执事·赵无极**（煞灵化形第9轮）：业火城的实际掌控者（城主是傀儡），负责铁山宗在九州南部的情报和矿脉采集。②**外门弟子·林婉儿**（业火炼神第8轮）：铁山宗外门首席，赵无极的利刃，负责清剿敢反抗的散修势力。③**外门炼器师·老邹**（煞灵化形第2轮）：给铁山宗提供法器，在业火城经营法器铺子。 |
| **控制资源** | ①业火城的**灵脉采集权**（城内唯一能公开修炼的灵脉来源——铁山宗布下了“业火大阵”过滤浊气，让宗门弟子修炼效率远超散修）②妖兽山脉中圈至核心区域的**猎杀许可证**管理权（散修必须付费才能获得许可证）③低阶丹药/法器的**垄断供应**（散修只有黑市和高价两条路） |
| **弱点** | 傲慢——铁山宗外门看不起散修，认为散修只是“会走路的资源”，不把底层散修放在眼里。主角可以利用这种傲慢，“先示弱→收集情报→一击反杀”。 |
| **与上层关系** | 铁山宗总部（煞灵化形至结丹初期）在九州南部腹地的**铁山城**，赵无极只是外门执事。总部对南部业火城的态度是“放养”——只要每年上缴20%的灵脉采集收益和保护费就行。这意味着：①赵无极必须在业火城维持表面秩序（否则铁山宗会换人）；②赵无极其实也怕散修大规模叛乱（会被总部问责）。 |

## 顶层势力（第一卷只暗示不碰撞）：铁山宗核心+九天道场（攀登方式类型：爬到顶层的人·战争型）

| 字段 | 内容 |
|:--|:--|
| **攀登方式** | **战争型（爬到顶层的人）**——元婴以上不再亲自猎杀低阶妖兽，而是发动宗战争/灭国级冲突，批量收割高阶浊气。一次宗门战争=10年猎杀的总和。消耗的“代价”是战死的小势力修士的心魔。 |
| **暗示方式** | ①通过赵无极的言行：提到“铁山宗内门长老在准备一场大战”（不说明目标）②通过雷老虎的感慨：“九天道场一个命令，南部所有宗门都得动起来——为什么动？因为天道在上头压着。” |
| **第一卷不碰撞原因** | 主角第一主力(淬血→业火初期)不可能和铁山宗内门（煞灵→金丹）碰撞，更不可能和合道（九天道场）有任何交集。第一卷的任务是：让读者知道“这世界塔顶还有人”，但不让主角触碰到那层。 |

## 坐标系内压（力量体系→主角面对的门槛性压迫）

### 危机1：业火榜的“猎杀配额”危机（淬血→业火晋升驱动）

| 字段 | 内容 |
|:--|:--|
| **危机来源** | 骨架.md力量体系：淬血开脉第1轮→业火炼神第1轮的晋升条件是“业火榜积分达标”+“猎杀同阶≥10人”。但业火城的业火榜积分被铁山宗外门和雷老虎等猎杀团暗中操控——新手散修很难直接拿到积分。 |
| **触发条件** | 主角在第1轮淬血开脉后尝试接业火榜任务，发现任务被猎杀团垄断；或主角击败某散修被动进入榜上有名，招惹猎杀团。 |
| **预计爆发卷** | 卷一中段（第10-20章）：主角在妖兽山脉第一次大规模猎杀后，业火榜积分暴涨，引起雷老虎团队注意。 |
| **后果** | ①雷老虎警告主角“新人别太急，慢慢来”——暗示主角要么加入他们，要么被踢出业火榜；②若主角拒绝加入，雷老虎安排手下在猎杀中“意外”陷害主角（如故意放走高阶妖兽让主角被围攻）。 |
| **对齐骨架检查** | ✅ 坐标系内压——业位晋升必须过业火榜门槛（骨架.md力量体系：业火炼神晋升条件含“业火榜积分”）、有势力垄断屏障（骨架.md力量体系：散修晋升难度大因为资源被垄断） |
