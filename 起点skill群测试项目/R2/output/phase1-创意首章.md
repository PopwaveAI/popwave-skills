# 创意发散+故事纲领+黄金首章

## Step 1g：双轨发散

### 王道轨（3个）

#### 王道1：数据面板·情报猎手

| 字段 | 内容 |
|------|------|
| **一句话卖点** | D&D数据面板的底层流浪者靠“看穿一切冷却和弱点”的金手指，在黑森林里精准越级猎杀，一路杀穿复仇路 |
| **核心爽感机制** | 数据面板碾压+精准卡冷却窗口的越级击杀 |
| **主角驱动** | 主角必须在野区高效猎杀升级，否则仇人会越走越远。数据之眼让他从“碰运气乱杀”变成“科学狩猎” |
| **使用的燃料（改造方式）** | 借鉴深渊主宰的数据面板体系（燃料A），融合D&D的战斗节奏与弱点暴露窗口机制（燃料B），加强代价约束——精神力枯竭时会被反杀（燃料C） |
| **最容易写崩的点** | 数据之眼用多了变机械操作，失去紧张感。需要通过精神力上限永久下降来维持张力 |
| **三眼法判断** | ✅ 画面（主角躲在废墟后，盯着4级魔物的技能倒计时，数到1然后冲出去）→ ✅ 限制（精神力有限，高等级敌人信息模糊）→ ✅ 场景（越级猎杀/反杀赏金猎人/潜入领主领地/收集灵魂碎片升级/鉴定敌人属性） |

### 猎奇轨（2个）

#### 猎奇1：数据面板·黑魂遗产

| 字段 | 内容 |
|------|------|
| **一句话卖点** | 主角每次死亡都会变成“数据残响”重新活过来，但每次复活会遗忘一段记忆——复仇的记忆也在遗忘 |
| **核心爽感机制** | 死亡惩罚=记忆丢失+数据重置，必须和遗忘赛跑 |
| **主角驱动** | 主角必须一边在深渊世界里猎杀升级，一边和“遗忘”对抗——他每次死亡都会丢一段记忆，包括仇人的脸、仇人的名字。他得在彻底忘记复仇目标前完成复仇 |
| **使用的燃料（改造方式）** | 借鉴黑魂的死亡轮回+记忆碎片机制（燃料B），融合“遗忘”作为代价（燃料C——遗忘比死亡更可怕），改造深渊主宰的死亡规则：主角不会真死，但会丢失随机记忆片段 |
| **最容易写崩的点** | 读者会觉得“反正主角不会死”，紧张感消失；或者遗忘机制太复杂，读者跟不上 |
| **三眼法判断** | ✅ 画面（主角死在第5次猎杀中，醒来后盯着手上的石板——仇人的名字从石板边缘开始模糊，他拼命想记住，但名字还是碎了一角）→ ✅ 限制（遗忘不可逆，一旦忘记关键信息可能永远找不到仇人）→ ✅ 场景（复活后检查还剩多少记忆/为了记住名字冒死挑战高等级/在遗迹里找到记忆碎片/发现仇人利用自己的遗忘在布局/在忘记一切前完成最后一击） |

#### 猎奇2：数据面板·反杀公式

| 字段 | 内容 |
|------|------|
| **一句话卖点** | 主角的数据之眼看到的不是弱点——而是“如果你不退，你会怎么死”。每一次战斗都是必死预言的破解游戏 |
| **核心爽感机制** | 预知死亡+反常识破局 |
| **主角驱动** | 主角每次面对强者，数据之眼会弹出“必死结局”——比如“如果他正面冲锋，会被对方在第3秒击杀”。主角必须在有限时间内找到唯一能活下来的战术 |
| **使用的燃料（改造方式）** | 借鉴未来预知/死亡回档类题材（燃料B），融合“预知=更重代价”的逻辑（燃料C——预知越精准，精神力消耗越大），改造为“只能看到自己的死法，看不到别人的” |
| **最容易写崩的点** | 如果每次结局都被预知，会变成完美通关剧情，失去真实感 |
| **三眼法判断** | ✅ 画面（主角的数据之眼弹出红色警告：“如果正面对抗，35秒后左胸被刺穿”——他必须找到那条唯一活下来的路线）→ ✅ 限制（精神力消耗极大，过度使用后预知会变模糊甚至错误）→ ✅ 场景（破解必死预言/在绝境中找到唯一活路/预知失效时的绝望一战/反杀后对手的震撼/发现仇人也有预知能力） |

### 推荐方案

| 维度 | 王道1（情报猎手） | 猎奇1（黑魂遗产） | 猎奇2（反杀公式） |
|------|------------------|------------------|------------------|
| **核心爽感机制** | 数据面板碾压+精准卡冷却 | 死亡惩罚=遗忘+数据重置 | 预知必死+反常识破局 |
| **主角主动性** | 主动选择目标，高效猎杀 | 被动死亡+主动寻找记忆 | 主动找唯一活路 |
| **骨架对齐度** | ⭐⭐⭐⭐⭐ 完全契合 | ⭐⭐⭐ 死亡机制需要微调 | ⭐⭐⭐⭐ 战斗部分契合 |
| **长线续航（250章+）** | ⭐⭐⭐⭐ 等级+复仇双线 | ⭐⭐⭐ 遗忘机制易疲劳 | ⭐⭐⭐ 预知易变套路 |
| **独特性（同赛道100本）** | ⭐⭐⭐⭐ 情报猎手角度新 | ⭐⭐⭐⭐ 遗忘代价有深度 | ⭐⭐⭐ 预知不算新鲜 |
| **执行难度** | ⭐⭐⭐ 中等 | ⭐⭐⭐⭐ 较高（记忆管理复杂） | ⭐⭐⭐ 中等 |
| **张力强度** | ⭐⭐⭐⭐ 每一场战斗都像赌博 | ⭐⭐⭐⭐⭐ 遗忘是终极恐惧 | ⭐⭐⭐⭐ 预知增加紧迫 |

**推荐：王道1（情报猎手）**

理由：
1. **骨架契合度最高**：D&D数据面板体系下，数据之眼的弱点暴露+冷却倒计时是最自然、最契合世界观的设定，不需要额外打补丁
2. **爽感兑现最快**：首章就能让主角完成一次越级击杀，读者立刻感受到金手指的价值
3. **长线续航有保障**：等级线（升级节奏）+复仇线（仇人身份逐步揭示）+消耗线（精神力上限下降→寻找灵魂净水）三条线交织
4. **追加吸收败者强点**：
   - 吸收猎奇1的“遗忘代价”增强精神力消耗的紧迫感（不仅是精神力枯竭，还会短暂遗忘关键信息）
   - 吸收猎奇2的“必死预言”局部使用——面对仇人时，数据之眼第一次弹出“必死结局”，主角首次面临金手指也救不了的绝境

**淘汰项**：
- 王道2（副本探索者）：骨架的“杀戮飞轮”引擎不支持纯解密，战斗和探索必须5:5配比——副本探索线更适合中期演化后（第四卷后）
- 王道3（以战养战）：逆风局不足，读者会疲劳
- 猎奇1（黑魂遗产）：遗忘机制对长篇管理要求太高，且受众面窄
- 猎奇2（反杀公式）：预知能力在同赛道已有代表作，独特性和新鲜感不足

**推荐创意的最终一句话卖点**：
> **“D&D数据面板世界的底层流浪者，靠看穿敌人技能冷却和弱点窗口，在黑森林里用脑子和刀片越级猎杀，在精神力枯竭前杀出一条复仇路。”**

## Step 1h：故事纲领（三核心+营销层）

### 什么样的世界（4层）

#### 第1层：源力量——深渊法则·灵魂石板

旧神陨落后，世界法则崩坏，残留的原始规则以数据化的方式烙印在所有生灵的灵魂中。每个生灵拥有一块灵魂石板，上面记录着自己的职业等级、六围属性、技能和专长。世界遵循“杀戮→经验→进化”的底层法则——这是一场无法退出的游戏，不杀戮就无法变强，不升级就会被更高阶的猎杀者吞噬。

#### 第2层：坐标系形态——20级五阶位金字塔

职业等级1-20级，分为五个深渊阶位：碎骨者（1-5级）、噬魂者（6-10级）、深渊领主（11-15级）、主宰者（16-19级）、至高主宰（20级）。每个阶位对应世界中的社会地位：碎骨者是底层炮灰，噬魂者是可被招揽的战士，深渊领主拥有领地，主宰者接近神明。

**主角当前坐标**：碎骨者·3级——最底层的流亡者，距离仇人（15级深渊领主）还有12个等级+2个阶位的差距。

#### 第3层：社会结构（众生攀登方式）

**有资源的人**（贵族、商会、神殿）：圈养低阶魔物或奴隶，花钱雇高等级护卫带刷，升级速度稳但依赖外力。

**没资源的人**（平民、流浪者、逃兵）：必须亲自冒险进入野区猎杀野怪或落单同行，死亡风险极高，但收益全部归自己。主角属于这一类。

**爬到顶层的人**（领主·主宰者）：垄断高等级禁区（深渊中层），禁止低等级者进入，从领地内的杀戮行为中抽取灵魂碎片税。

**爬不动/掉下来的人**：成为高阶者的移动经验包，被迫躲在边缘区域，或者加入“拒绝杀戮”的异教派（但常被当作低风险猎物猎杀）。

#### 第4层：信息分层

- **经济系统**：灵魂碎片=通用货币；高等级碎片（噬魂者级以上）可兑换属性源质、技能之书、专长传承石；灵魂净水（修复灵魂磨损）只有领主级以上势力稳定产出。主角的赚钱方式：猎杀→贩卖碎片→购买装备/补给。
- **政治系统**：统治力=等级压制。高阶者制定领地规则，低阶者要么服从（交“经验税”）、要么离开（失去安全区）、要么反抗（被猎杀）。主角没有政治资本，只能游离在规则之外。
- **文化规范**：杀戮被默认为生存法则。拒绝杀戮会被当作“资源浪费”——因此异教派和隐居者经常被当作“无用经验包”猎杀。主角的底线（不杀平民）在这个世界是稀缺品质，会被嘲“幼稚”，也为后期势力建立埋下伏笔。
- **信息不对称**：
  - 高阶者知道“法则碎片”的存在（第15级后的晋升关键），低阶者不知道
  - 旧神遗迹的位置被领主级势力垄断
  - **主角的优势**：数据之眼让他能在战场上实时看到敌人的技能冷却和弱点暴露窗口，这是其他人不具备的战场感知能力

### 什么样的舞台·世界危机（3要素）

#### 要素1：终极危机——法则崩溃+旧神回归

深渊法则正在缓慢崩坏：第16级以上的主宰者发现“杀戮飞轮”的转化效率开始下降——高阶敌人死后灵魂碎片极难吸收。旧神陨落后的原始法则正在消退，旧神有回归的趋势。如果旧神回归，深渊法则被改写，所有依赖法则升级的生灵（包括主角）都可能被剥夺力量。在旧神回归之前达到至高主宰（20级），才有资格参与“法则继承”仪式，决定世界的未来秩序。

#### 要素2：阶段性危机

| 阶段 | 敌人梯度 | 具体对象 | 量级 |
|------|----------|----------|------|
| 第1阶段（1-5级） | 底层猎杀者 | 野区魔物（4-5级）、落单的流浪者、赏金猎人小队（5-6级） | 同级或越1-2级 |
| 第2阶段（6-10级） | 地方势力 | 黑森林匪帮首领（7级）、小型领主亲卫队（8-9级） | 越2-3级，复数敌人 |
| 第3阶段（11-15级） | 仇敌势力 | 仇人（15级深渊领主）+其麾下的领地军队（10-14级） | 越4级+军团围剿 |
| 第4阶段（16-20级） | 世界级敌人 | 其他主宰者（16-19级）、旧神残留意志（20级级） | 同级或越级+法则层面战斗 |

#### 要素3：危机驱动机制

**压力源**：第3级时主角的数据之眼探测到“灵魂碎片转化效率开始下降”——他的精神力永久上限每降低5%，转化效率下降1%。如果他不找到提升方法，到第10级时转化效率将跌破50%，升级速度爆炸性衰减。

**倒计时**：仇人（15级深渊领主）正在筹划晋升主宰者——一旦他成功，主角永远追不上复仇。主角必须在他晋升前达到等级+战力足以挑战他的水平。

**危机升级**：主角达到第7级时，仇人发现他活着，开始派出赏金猎人追杀——被动猎杀变为主动逃亡+反击。

### 什么样的主角（5要素）

#### 要素1：起点位置

**碎骨者·3级**——黑森林边缘废墟中的流亡者。没有资源、没有队友、没有势力背景。身上只有一把从尸体上扒下来的生锈匕首和一块破旧的灵魂石板。

#### 要素2：核心欲望

**复仇**——灭门仇人是15级深渊领主。主角的核心欲望与世界经济级驱动力（生存压力+零和资源争夺）完全对齐：不杀仇人，心念不通；要杀仇人，就必须遵循深渊法则杀戮升级，否则死在仇人手下只是时间问题。

#### 要素3：特殊天赋（含金手指）

**深渊数据之眼**——每次战斗可主动激活，持续5-15秒，能看到：
- 敌人的主等级+属性概览（同阶敌人完整面板，高5级以上模糊化）
- 技能冷却倒计时（精确到秒）
- 弱点暴露窗口（持续1-2秒）
- 击杀后灵魂碎片品质预估

**限制**：
- 每次激活消耗5%-10%精神力
- 连续使用3场高烈度战斗→眼压过载（视野模糊、呕吐）
- 过度使用→精神力上限永久下降→灵魂石板出现裂纹→需要灵魂净水修复
- 等级高于主角5级以上的敌人→只能看到模糊信息（“力量？？？”“技能？？？”）

**代价**：精神力是金手指的燃料，也是主角的生命线。精神力见底=任人宰割。过度使用=永久损伤=升级速度永久变慢。

#### 要素4：代价

**通用代价**：每次战斗都有死亡风险；灵魂磨损（精神力上限下降）→需灵魂净水修复（领主级以上势力才能稳定产出，底层难以获取）；仇恨累积（击杀敌人后其亲友追踪主角）。

**金手指的额外代价**：精神力消耗加剧了灵魂磨损的速度——主角比其他无资源者更依赖高效率猎杀，但也更早面临“灵魂石板裂纹”的门槛。

#### 要素5：与众生关系

主角是“没资源的人”中的一员——没有贵族的圈养场，没有神殿的训练营，只有一个人、一把刀、一片野区。

但他的攀登方式比其他无资源者多了一个优势（数据之眼），也多了一个劣势（精神力消耗带来的灵魂磨损更快）。他不会因为金手指就改变身份归属——他还是那个“死了没人收尸的流浪者”，只是数据之眼让他在同阶战斗中存活率更高。

**与仇人关系**：仇人是高阶者（15级深渊领主），主角是底层复仇者。跨越12个等级的复仇之路是主角与最顶层势力之间的第一道鸿沟。

### 营销层

#### 最大钩子（≤20字）

> **数据之眼，暴君之躯，暗黑复仇。**

#### 即时兑现感

**第1章**（本步目标）：主角用数据之眼预判4级魔物的技能冷却，卡1秒弱点窗口完成越级击杀，从3级升到4级——读者立刻看到金手指的价值+等级提升的快感。

**第2-3章**：被赏金猎人盯上，主角边逃边猎杀，在精神力即将枯竭时完成反杀；第3章末尾发现仇人的线索方向。

**第4-5章**：进入更深的野区猎杀5级噬魂者级敌人，第一次用数据之眼看到“精神力上限下降”的警告；第5章末尾精神力上限永久下降了2%，主角开始意识到代价，决定寻找灵魂净水的线索。

**前5章节奏定位**：生存猎杀→第一次越级→被追杀→反杀→发现复仇线索→接触代价机制。读者前5章就能感受到“世界残酷+金手指有用+代价真实+复仇有方向”。

# 正文/ch001.txt

# 第一章 数据之眼

黑森林的夜是活的。

腐烂的树根在地表下蠕动，像一条条被斩断又活过来的蛇。灰色的雾气贴地漫走，盖住那些被啃得只剩骨架的尸体，又在一阵风吹过后露出下面泛着磷光的骨头。

陈野靠在一块断裂的石碑后面，用袖子擦掉匕首上的血迹。

这把匕首是他三天前从一个死人手里扒来的——那个人的喉咙被什么东西咬穿了，死的时候眼睛还睁着，灵魂石板上写着“碎骨者·2级”。陈野当时也是2级。

现在他是3级了。

三天，三只森林灰狼，一场差点被反杀的夜袭——30点灵魂碎片，刚好够升到3级。

灵魂石板在他眼前微微闪烁。

```
【陈野】
种族：人类（深渊腐化未显）
职业：无职（通用）
等级：3（碎骨者）
经验：30/100
属性——力量：4，敏捷：6，体质：4，感知：5，智力：7，魅力：2
技能——潜行（1级），匕首精通（1级），感知危机（未激活）
专长——无
```

魅力2。

陈野看了一眼这个数字，没什么表情。在这个世界里长得好看没有任何意义——没有人会因为你的脸而少吃你一块肉。相反，低魅力意味着你在深渊法则的“社交判定”中更容易被人怀疑、排斥、攻击。

也就是说，他天生就长着一张“好欺负的脸”。

但这几天他已经发现，3级以下的人确实不敢再主动惹他了。3级在黑森林的边缘废墟算个门槛——炮灰里排中上游，至少不会被人当成随手宰掉的经验包。

陈野收起石板，从石碑后面探出头。

前方五十米处是一片开阔地废墟，只有几堵半倒的石墙和一个干涸的喷水池。水池中央蹲着什么东西——灰色的、像石头一样的轮廓。如果不是它微微起伏的脊背，陈野会以为那是废墟的一部分。

他的眼睛微微发酸。

数据之眼。

那东西从昨天开始就出现在他视野里了——他不知道这是不是所有灵魂石板解锁后的隐藏功能，还是只有他一个人有。当他把注意力集中在一个活物上超过三秒，他的视线会像被什么东西撬开一样，看到那活物的“内部数据”。

昨晚他就是靠这个躲过了那只3级灰狼的伏击——数据之眼显示那只灰狼的技能冷却倒计时还有7秒才到爆发期，他只需要蹲在原地等7秒，然后趁它技能真空期一刀解决。

7秒，没有灰狼，他死得很惨。7秒，有数据之眼，他活着升到了3级。

但现在他的头已经有点发晕了——昨晚连续用了两次数据之眼，精神力消耗还没完全恢复。

他咬咬牙，集中注意力。

三秒后，水池里的那个轮廓在他视野里裂开了一层灰色的数据纹路：

```
【岩脊猎食者】
等级：4
状态：休眠中（警戒范围15米）
属性——力量：8，敏捷：3，体质：9，感知：4，智力：2
技能——岩石皮肤（被动/冷却：∞/持续：永远），重击（主动/冷却：12秒/伤害修正：+150%），潜伏突刺（主动/冷却：30秒/伤害修正：+200%）
弱点：腹部（护甲减半），敏捷型技能命中后硬直3秒
推荐战术：引诱重击→闪避→针对腹部攻击
```

4级。

陈野的手指在匕首柄上收紧了一下，又松开。

4级的野怪，理论上3级的他单挑赢面不大。属性差摆在那里——对方力量8，他力量4，正面硬拼就像拿树枝去砸石头。而且这只岩脊猎食者还有岩石皮肤，普通攻击估计只能擦出火星。

但数据之眼给他看到了一些东西。

“重击冷却12秒，潜伏突刺冷却30秒……它打完一套爆发后有至少12秒的技能真空期。”

“腹部护甲减半——如果能把它诱导到侧身位，捅到那片软肉……”

“敏捷型技能命中后硬直3秒——意味着我只要先打中一次，它就会僵住3秒，这3秒内我能多捅至少两刀。”

陈野舔了舔干裂的嘴唇。

计划有了，剩下的问题是：他能不能扛住第一下。

如果是满精神力状态，他敢赌。但现在他只能再用最多一次数据之眼，用完如果没打死这只岩脊猎食者，他会在精神力枯竭+体力加速消耗的状态下被反杀。

不能犹豫。

他站起身，踩碎一根枯枝。

咔嚓——声音在夜里格外刺耳。

水池中央的岩脊猎食者猛地抬起头。那是一张扁平的、爬行动物般的脸，两只灰色眼睛像两颗脏透了的玻璃珠，正中央的瞳孔缩成一条竖线，死死锁住了陈野。

15米警戒范围。

它动了。

岩脊猎食者的四条腿撑起那将近两米长的身躯，像一块突然活了过来的灰色巨石。它没发出任何叫声——这种猎食者从不在捕猎前浪费体力在嗓子上。它的脚步沉重，每一步都在废墟石板上留下一个浅浅的爪印。

陈野没跑。

他站在原地看着它冲过来，手中的匕首转了半圈，握紧。

10米。

他激活了数据之眼。

视野里那个灰色的轮廓炸开大片的红色数字。它的技能面板在闪烁——

```
潜伏突刺：可用（冷却0/30）
重击：可用（冷却0/12）
```

它用的是第一手潜伏突刺。

陈野往左偏了半步。

岩脊猎食者的身体猛地压低——那是潜伏突刺的前摇动作。它的后腿肌肉泵成两块钢板，整个身躯像一枚炮弹一样弹射过来！

速度快得离谱。

即使是预判过的陈野，也只来得及侧过半个身位——那只布满岩甲的头颅擦着他的左肩撞过去，裹着一股腥臭的风。砰的一声，他身后的石墙被撞出一个碗口大的坑，碎石飞溅。

陈野的耳朵嗡嗡响，但他没停。

数据之眼的数字还在跳动——

```
潜伏突刺命中：否
潜伏突刺状态：冷却中（0/30）
剩余技能：重击（冷却0/12）
```

岩脊猎食者一击落空，四条腿在碎石堆里转了个身，动作比刚才明显慢了一点——潜伏突刺的后摇让它需要半秒重新稳重心。

半秒。

够了。

陈野冲上去，匕首刺向它的左眼！

匕首的尖端刺进了它的眼眶——但也只刺进去半寸。

岩脊猎食者发出第一声嘶吼，猛地甩头，把陈野连人带刀甩飞出去。陈野在地上滚了三圈才稳住身体，左手撑地时摸到了一截带钉子的木板。

他抬头看了一眼——

数据之眼还在运转，但视野的边缘已经开始发红。精神力过载的警告。

只剩10%了。

岩脊猎食者的左眼眶插着那把匕首，深色的血从伤口涌出来，滴在地上嘶嘶冒烟。它的技能面板在刷新——

```
重击：可用（冷却0/12）
潜伏突刺：冷却中（21/30）
```

12秒的真空期，还剩不到8秒。

它还有重击。

重击的伤害修正+150%，如果挨上一下，以他4点的体质，大概率直接进入濒死状态。

但数据之眼还给了另一个信息——腹部。

“重击的前摇是直立上半身+前爪高举——这个姿势会让它整个腹部暴露出来。”

陈野扔掉木板，右手摸到腰间另一把短匕——这是他真正防身的家伙，从来没拿出来用过，因为一旦被发现他还有第二把刀，敌人就会防他这个后手。

但现在必须赌最后一次。

岩脊猎食者冲过来了。

它的上半身果然开始抬高——那是重击的前兆！

陈野没有躲。

他朝着岩脊猎食者迎面冲过去。

一人一兽的距离在三步之内缩短到零。岩脊猎食者的前爪带着风声拍下来——

陈野往下一蹲，膝盖滑进它的腹下！

它的爪子拍在他原来的位置，震得地面一颤。

陈野仰面躺在岩脊猎食者的腹部下方——那片护甲减半的软肉位置。他看到那层灰白色的皮肤表面有一条条裂缝般的纹路，那是岩石皮肤覆盖不完全的地方。

短匕狠狠捅进去。

噗嗤。

匕首没根而入。

岩脊猎食者发出一声震耳欲聋的惨嚎，整个身体像触电一样抽搐起来——它的腹部就是它最脆弱的命门，被捅穿之后血液和内脏液喷涌而出，淋了陈野一身。

他拔出匕首，又捅了一刀。第三刀。

岩脊猎食者的身体终于软下来，四条腿再也撑不住那两米长的躯体，轰地一声压下来。陈野蠕动着从下面爬出来的时候，浑身都是黑色的血和粘液。

他瘫坐在废墟里，大口喘气。

灵魂石板弹出了提示：

```
击杀：岩脊猎食者（4级）
获得灵魂碎片：40点
越级击杀加成：额外10点
属性源质掉落：无
技能之书碎片掉落：无
专长传承石掉落：无
当前经验：30/100 → 80/120
```

距离4级只剩40点。

陈野看着那个数字，想笑，但喉咙里翻上来的只有血腥味和酸水。

精神力过载的后劲上来了——他的视线开始发花，眼前的废墟像是泡在水里一样扭曲变形。他用力眨了几下眼睛，视野才勉强恢复正常。

数据之眼自动关闭了。视野边缘那行红色文字还在闪烁：

```
警告：精神力剩余不足10%。连续使用超过推荐阈值可能导致精神力上限永久下降（当前上限：100/100，未见裂纹）。

永久下降。

陈野记住了这个词。

他喘匀了气，站起来，走到岩脊猎食者的尸体前，把插在眼眶里和腹部的那两把匕首拔出来，用尸体的皮草擦干净血迹，收好。

然后他看向北方。

黑森林的尽头，有一片被灰色雾气笼罩的区域——那片区域的深处，住着一个人。

一个他必须在精神力彻底耗尽之前，追得上的人。

陈野转身，迈过岩脊猎食者的尸体，朝黑森林更深处走去。

他的灵魂石板在黑暗中闪烁了一下，新的信息弹出来：

```
警告：精神力上限当前为98/100——轻微下降（2%）。建议休整至少12小时后再进行下一次战斗。

陈野脚步停了一瞬，随即继续往前走。

12小时。

他可能没有12小时。
