| **格杀镇**（安全区·交界点） | 碎骨者2-5级 | 交易灵魂碎片（黑市价：10枚灰色碎片=1枚白色碎片）；情报购买（每份情报5-10枚灰色碎片）。 | 镇内有“领主法则”覆盖（由格杀镇的实际控制者——一位7级噬魂者领主布置），在镇内不允许械斗，违规者会被领主亲卫斩杀。 | 出售多余灵魂碎片换装备/情报/消耗品（灵魂净水、属性源质）；接触其他独立猎手，获得“谁在黑森林猎杀效率最高”的情报。 | 黑市上有人出售“数据之眼的克星”——一种短时间屏蔽数据探测的消耗品（主角需要提防别人用）。 | 格杀镇领主（7级噬魂者，姓氏“马拉”）+ 黑手套商会分舵 + 血斧佣兵团据点。 |

## 势力具体设计（第一卷主要碰撞势力）

### 1. 铁骨帮（底层势力）

| 字段 | 内容 |
|:--|:--|
| **攀登方式类型** | 没资源型（平民/流浪者/逃兵） |
| **势力规模** | 约40-50人，核心成员10人（3-5级碎骨者），普通成员（1-2级碎骨者）近30人，附庸10余个。 |
| **控制资源** | 边缘废墟的“铁骨街”（3条街巷）及其周边的低级魔物刷新点（1-2级变异鼠、腐化野狗）。帮会成员靠轮流看守猎杀路线维持基本升级节奏。 |
| **内部结构** | 帮主“铁骨”（7级噬魂者——年迈，战斗力已衰退至6级水平）。副帮主A“黑牙”（6级噬魂者，激进派，想推翻铁骨）；副帮主B“疤脸”（6级噬魂者，保守派，想维持现状）。两股势力暗斗激烈。 |
| **资源控制方式** | 成员每周必须上交50%的灵魂碎片（灰色/白色）给帮会，帮会统一管理“猎杀路线分配”和“保护费缴纳”。 |
| **弱点/裂缝** | 1.内部分裂：黑牙和疤脸相互仇视，帮主铁骨无力控制。<br>2.信息被垄断：黑手套商会控制了铁骨帮的情报渠道（猎杀路线刷新时间、势力动向），帮会严重依赖商会提供的情报（价格昂贵）。<br>3.血斧压迫：30%的年保护费是沉重的负担，铁骨帮每年剩余碎片仅够维持核心成员消耗，无法扩张。 |
| **与血斧的关系** | 铁骨帮是血斧佣兵团的“附庸势力”（半控制/半压榨关系）。血斧定期派一名5级噬魂者来收保护费，铁骨帮提供低级猎杀路线供血斧新成员练级。 |
| **主角原因** | 主角初期在边缘废墟猎杀时会偶遇铁骨帮成员（他们也想猎杀那些低级魔物）。冲突来源：铁骨帮认为主角闯入他们的猎杀路线（“这条街的猎物是我们的”）。主角第一次越级击杀（3级主角杀4级铁骨帮成员）就发生在这里。 |
| **本卷结局** | 铁骨帮内部分裂在黑牙和疤脸的冲突中暴露：黑牙暗中联络血斧想取代铁骨，疤脸则想投靠黑手套商会。主角在冲突中利用数据之眼帮疤脸击杀了黑牙（疤脸欠主角人情）。铁骨帮在本卷末被血斧吞并，疤脸成为血斧的附庸。 |

### 2. 孤狼猎团（中层势力·独立精英）

| 字段 | 内容 |
|:--|:--|
| **攀登方式类型** | 没资源型（个体精英模式） |
| **势力规模** | 约15-20人，全是独立猎手，无固定上下级关系。核心“召集者”只有团长“黑鸦”（6级噬魂者）。 |
| **控制资源** | 黑森林外围的高收益猎杀路线（3-4级魔物刷新时间表）+ 格杀镇内的情报网络。 |
| **内部结构** | 团长黑鸦负责“信息汇总”——他把每个成员收集的情报整合后，按贡献度分享给全体成员。无固定指挥权，团长不强迫任何人执行任务。 |
| **资源控制方式** | 每个成员自行猎杀，猎获归个人所有。但需每月缴纳“情报费”（5枚白色碎片/人）。团长负责维护情报系统的准确性。 |
| **弱点/裂缝** | 1.成员间互不信任：今天的情报可能偏袒了某个人，明天就可能被其他人背刺。<br>2.团长黑鸦有隐藏野心：他想把猎团变成自己的私军，目前正在暗中积累力量（已囤积了30枚白色碎片）。<br>3.猎团缺乏高端战力：全体成员等级最高为6级，一旦遇到血斧的精英队（全员5-7级），只能跑。 |
| **与血斧的关系** | 独立关系。黑鸦团长采取“不主动挑衅、不透露内部情报、必要时雇佣血斧当保镖”的谨慎策略。目前血斧并未把猎团当成威胁（因为等级太低）。 |
| **与黑手套商会的关系** | 交易关系：猎团用情报（魔物刷新时间、其他势力动向）换取黑手套的装备/消耗品。黑手套很重视猎团的情报能力，但商会也在暗中挖猎团的人。 |
| **主角原因** | 主角在黑森林外围高效猎杀（数据之眼+越级击杀）的动静太大，被猎团成员发现。团长黑鸦主动接触主角（提供情报换取主角的“越级猎杀技巧”分享）。主角可以利用猎人情报网提升猎杀效率，但也要警惕猎团的内部问题。 |
| **本卷结局** | 黑鸦试图收编主角为正式成员（提供“情报不用交费”的承诺），但主角拒绝。黑鸦被激怒，暗中向血斧透露了主角的越级猎杀能力（想让血斧干掉主角）。本卷末主角与黑鸦发生冲突：黑鸦发现主角是“越级猎杀者”（情报价值极高），试图联手血斧抓主角。 |

### 3. 血斧佣兵团（上层势力·本卷主要反派）

| 字段 | 内容 |
|:--|:--|
| **攀登方式类型** | 有资源型（小领主/小贵族控制的佣兵系统） |
| **势力规模** | 核心战力约50人（全员4-7级），附庸铁骨帮（约40人）+ 其他低级势力（约20人）。总控制人数约110人。 |
| **控制资源** | 格杀镇30%战力（拥有自己的角斗场训练营）；高等级猎杀路线（4-6级魔物区，由5级以上的团员轮流看守）；圈养低级魔物的控制权（在镇外设陷阱捕获后/运送至角斗场）；3个黑森林内的小型据点。 |
| **内部结构** | 团长“血斧”（7级噬魂者，残暴型，战斗力在同级中属于上流）。副团长A“毒蛇”（6级噬魂者，谋略型，掌管情报和刺客行动）；副团长B“铁蹄”（6级噬魂者，武力型，掌管正面战和训练营）。两副团长关系紧张：毒蛇想通过黑手套商会的支持取代血斧的团长位置；铁蹄忠诚于血斧（因为血斧救过他命）。 |
| **资源控制方式** | 团员按等级分成三档：5级以下为“炮灰队”（负责低级猎杀路线和守据点），5-6级为“精英队”（负责高等级猎杀路线和对外作战），7级血斧本人负责总体调度和对外威慑。每周猎杀所得按军功分配（血斧拿40%，副团长各拿15%，精英队共享15%，炮灰队共享30%——炮灰队几乎吃不饱）。 |
| **弱点/裂缝** | 1.团长血斧性格残暴：对铁骨帮和附庸势力的压榨越来越重，导致附庸势力忠诚度下降（铁骨帮内部已有人想反水）。<br>2.毒蛇与黑手套勾结：黑手套商会承诺帮毒蛇上位，毒蛇则许诺上位后让血斧佣兵团“依附”于商会（放弃独立性）。<br>3.过度依赖精英：血斧本人一倒，整个组织会瞬间瘫痪（铁蹄不够聪明，毒蛇会趁机夺权）。 |
| **与领主马拉的关系** | 马拉领主对血斧很不满——他需要格杀镇的稳定，但血斧的压榨行为已引发低级猎手的不满（可能导致低级猎手离开格杀镇，影响领主的税收和安全）。马拉领主已经暗中寻找替代血斧的势力（比如扶持孤狼猎团的团长黑鸦）。 |
| **主角原因** | 主角的越级猎杀能力引起血斧注意（“3级杀4级，这种战斗力在野区很值钱”）。血斧想收编主角，先是拉拢（提供装备和庇护），遭到拒绝后转为追杀（“不能让这种潜力威胁留在野区，会成为其他势力的武器”）。 |
| **本卷结局** | 毒蛇的实权阴谋在卷尾爆发：毒蛇勾结黑手套商会刺杀血斧（用黑手套提供的数据屏蔽符让血斧的数据之眼失效），但主角在冲突中被卷了进来。数据之眼看到了毒蛇的阴谋全过程：毒蛇在刺杀时使用了“违规手段”（黑手套商会提供的进阶版灵魂碎片——加持了法则干扰），主角用数据之眼记录了这个证据。血斧重伤，毒蛇逃离加入黑手套，血斧残余势力（铁蹄+20名精英队）决定投靠领主马拉。主角成为“知道真相但保持沉默”的关键人物。 |

### 4. 黑手套商会（上层势力·本卷幕后的隐形之手）

| 字段 | 内容 |
|:--|:--|
| **攀登方式类型** | 有资源型（大商会——控制资源+雇佣高等级护卫） |
| **势力规模** | 格杀镇分舵约15人（全员5-8级：分舵主“黑手套”本人8级噬魂者+4名7级亲卫+10名6级执行员），总部未知（暗示位于深渊裂隙某处）。 |
| **控制资源** | 黑市交易链：灵魂碎片兑换（灰色↔白色↔蓝色）、属性源质买卖、技能之书交换、专长传承石（极其稀有，仅在拍卖行出现）。情报网络：覆盖格杀镇及周边所有势力（包括铁骨帮、孤狼猎团、血斧佣兵团）。消耗品供应：灵魂净水（修复精神力上限的唯一药剂——黑手套垄断了格杀镇的供应）。 |
| **内部结构** | 分舵制：每个城镇/据点设一个分舵，分舵主对商会总部负责（但总部的地址和人员全部保密）。分舵任务：赚取灵魂碎片差价、发展附庸势力、收集深渊法则核心情报。 |
