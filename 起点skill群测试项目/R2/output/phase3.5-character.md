### 陆渊

#### 基本信息
- 姓名：陆渊
- 定位：主角
- 阵营：正派
- 攀登方式类型：没资源型
- 等级坐标：碎骨者3级（起点）→碎骨者4级（第一章末）
- 性别 / 年龄：男 / 19岁
- 引入章：ch001
- 退场章：全书

#### 性格描述
> 3-5个标签+具体行为表现

| 标签 | 行为表现 |
|:--|:--|
| **冷默寡言** | 用眼神和行动代替回答。被铁骨帮成员拦住时，只说了一句“让开”，对方继续挑衅就直接动手。 |
| **计算型狠辣** | 战斗前先用数据之眼默默计算对手属性差和技能冷却，确认胜率后才出手。出手就是杀招，不留活口。 |
| **蓄力爆发** | 平时保持低调不显山露水，但在绝境时会极限压榨精神力，打出超越面板数据的爆发伤害。 |
| **孤独成性** | 拒绝任何组队邀请，认为“今天并肩作战的人，明天可能为了我的灵魂碎片背后捅刀”。 |
| **复仇执念** | 每一场战斗的潜意识动力都是“我要变强，然后找到那个人”。升级不是为了变强本身，而是为了复仇。 |

#### 背景故事
> ≥200字小传

陆渊原本生活在深渊裂隙边缘的一座小镇——灰石镇。8岁那年，镇子被一位15级以上的深渊领主屠灭（具体身份未知）。他躲在母亲的尸体下才逃过一劫，但母亲临死前用身体替他挡住了一道魂火打击。从那天起，陆渊的世界只剩下两件事：活下来，然后变强到足以杀死那个人。

他在边缘废墟流浪了11年，靠着捡拾腐肉和逃避野狗勉强活到18岁。直到一年前，他在一座废弃神庙中触摸了一块残破的“灵魂石板碎片”，灵魂石板因此被激活——他成了那个“10万人中才有1个”的觉醒者。但他觉醒得太晚：其他人6-8岁觉醒时就开始猎杀升级，而他19岁时等级只有3级，远远落后同龄人。他所拥有的唯一优势是觉醒时附带的金手指——“深渊数据之眼”，这让他看到了越级猎杀的可能性。

陆渊从不提起自己的过去。别人问“你家人呢”，他只回答“死了”。别人问“你从哪来”，他只回答“废墟”。他那双眼睛深处，始终燃烧着一团冰冷的火——那是复仇的执念，也是支撑他每天凌晨4点起来猎杀的唯一动力。

#### 口头禅
- “让开。”（对人时最常说的话）
- “这是唯一的办法。”（做风险最高的决定时，自言自语）

#### 成长数据
- 起点：碎骨者3级
- 终点（全书）：至高主宰20级
- 关键转折：
  - 第一章：首次使用数据之眼越级击杀4级变异魔物→升级至4级
  - 第四章：在废弃矿洞越级击杀5级蜘蛛母皇→升级至5级+获得仇人线索

### 疤脸

#### 基本信息
- 姓名：铁骨帮副帮主·疤脸
- 定位：配角
- 阵营：中立偏正（前期与主角冲突，后期合作）
- 攀登方式类型：没资源型
- 等级坐标：噬魂者6级
- 性别 / 年龄：男 / 34岁
- 引入章：ch002
- 退场章：ch005（存活——被血斧吞并后成为附庸）

#### 性格描述
> 3-5个标签+具体行为表现

| 标签 | 行为表现 |
|:--|:--|
| **保守实干** | 反对铁骨帮内耗，坚持“先把猎杀路线守住再谈权力”。每次开会都说“等这季的魔物刷新稳定了再谈换帮主的事”。 |
| **粗中有细** | 外表粗犷（满脸刀疤），但做事有分寸：知道铁骨帮打不过血斧就乖乖上缴保护费，不逞强不硬刚。 |
| **倔强忠诚** | 对铁骨帮的底层成员有保护欲。曾自己掏腰包替3个欠保护费的新人垫付碎片，以维持帮会稳定。 |
| **面对弱小警惕但不主动欺凌** | 看到主角（3级流浪者）时没有直接赶尽杀绝，而是警告“这条街的猎物归铁骨帮，你再猎杀一次我就动手”。 |

#### 背景故事
> ≥200字小传

疤脸原名陈铁柱，来自格杀镇东边的一个矿工家庭。父亲在废弃矿洞中被魔物咬死后，母亲改嫁，他14岁就进了铁骨帮当打手。那时候铁骨帮还在边缘废墟最穷的一条街上混，帮主铁骨头是个30岁的壮汉，带着十几个兄弟抢猎杀路线。疤脸跟着铁骨头打了7年架，从1级碎骨者一路杀到5级碎骨者，脸上被刀疤盖了半张脸——所以后来都叫他疤脸。

疤脸对铁骨帮的感情很深：这是他唯一待过的地方，帮里的底层成员都是他一起挨饿的兄弟。他不喜欢黑牙那种“想干掉帮主自己上位”的激进做法，但也不喜欢帮主铁骨越来越懦弱的控制方式。他一直想在中间维持平衡，直到主角出现——这个3级的流浪者竟然在黑夜里连杀了帮里几个4级成员。疤脸第一反应是震惊，第二反应是“这小子是个好刀，不能让他死在内斗中”。

他选择和主角私下交易：你帮我把黑牙除掉，我放你继续在黑森林猎杀，还给你铁骨帮的情报网使用权。这是他这辈子第一次做“背叛帮主”的决定——但他告诉自己，这是在救帮会。

- “别急，先把地盘守稳了再说。”
- “小子，你这是在找死……不，等等，你这打法有点意思。”

#### 成长数据
- 起点：噬魂者6级
- 终点（本卷末）：噬魂者6级（被血斧吞并后仍保持等级，但地位下降）
- 关键转折：第四章与主角联手击杀黑牙，从此与主角建立合作关系

### 黑牙

#### 基本信息
- 姓名：铁骨帮副帮主·黑牙
- 定位：反派·街头层（第一卷前半段主要对手）
- 阵营：反派
- 攀登方式类型：没资源型（激进转卖型——想通过攀附上层势力跳过资源积累阶段）
- 等级坐标：噬魂者6级
- 性别 / 年龄：男 / 28岁
- 引入章：ch002
- 退场章：ch004（被疤脸+主角联手击杀）

#### 性格描述
> 3-5个标签+具体行为表现

| 标签 | 行为表现 |
|:--|:--|
| **激进冒险** | 认为铁骨帮应该停止缴纳血斧的保护费，转而投靠黑手套商会。理由是“血斧太弱，黑手套才是真正的势力”。 |
| **暴戾嗜血** | 处理帮内异议者时手段极其残忍——曾把偷猎低级魔物的成员打残后丢到黑森林里喂魔物。 |
| **对上层势力盲目信任** | 相信黑手套商会承诺的“只要当了铁骨帮帮主，就给你晋升7级的资源”。完全没意识到商会只是把他当消耗品。 |
| **急于证明自己** | 因为自己是从外部加入铁骨帮的（不是帮主亲信），所以格外想靠激进举动证明“我比疤脸强”。 |

#### 背景故事
> ≥200字小传

黑牙在成为“黑牙”之前，有一个名字——李狼。他出生在格杀镇最烂的街区，父亲是个酒鬼赌徒，母亲在他6岁时就离开了。他8岁觉醒灵魂石板时父亲已经欠下一堆赌债，最后被血斧佣兵团的人活活打死——李狼那时候躲在柜子里，透过缝隙看着父亲的血溅到墙上。他不是为父亲悲伤，他是害怕：如果自己也死了，那就什么都没了。

从那以后，李狼变得极其疯狂——他相信只有爬到上面才是唯一的活路。他13岁加入铁骨帮，15岁升到4级碎骨者，18岁到6级噬魂者。他的升级速度在铁骨帮里最快，因为他敢作敢当、敢拼敢杀。但帮主铁骨并不完全信任他，因为他“太急、太不安分”。

黑牙的愤怒积累了很多年：他想当帮主，想获得晋升7级的资源，想离开这个泥潭。当他听说黑手套商会愿意帮他实现这件事时，他毫不犹豫地接受了——他从来没想过商会的承诺可能是假的。他只是太想离开底层了，连代价都不愿意看清。

#### 口头禅
- “等老子当了帮主，你们这些人全都要跪着叫爷。”
- “铁骨帮？呵，就是一群杀老鼠的小混混。”

#### 成长数据
- 起点：噬魂者6级
- 终点：噬魂者6级（阵亡）
- 关键转折：第四章中暗中串联黑手套商会，想要暗杀帮主铁骨，被疤脸设局+主角数据之眼暴露行动轨迹，最终被击杀。

| **对过去的荣耀念念不忘** | 经常对底层成员吹嘘自己年轻时“一个人打退5个血斧成员”的故事。但实际上那些故事已经无法验证了。 |
| **对衰老和衰退极度恐惧** | 偷偷花大价钱从黑手套商会购买一种名为“灵魂合剂”的药水，相传能延缓等级衰退，但副作用是会导致灵魂石板裂纹。 |
| **对主角既害怕又好奇** | 听说一个3级流浪者能连杀4级铁骨帮成员后，第一反应是“杀掉他”——但见面后发现主角不对劲，又犹豫了。 |

#### 背景故事
> ≥200字小传

铁骨年轻时是格杀镇一带有名的狠人。25岁觉醒灵魂石板，35岁升到噬魂者7级，靠着一把铁骨刀打下铁骨帮的半壁江山——边缘废墟的3条街黑森林的几个猎杀点。那时候铁骨帮是边缘废墟最大的帮会，连血斧佣兵团都得给几分面子（因为铁骨能打）。

但一切在铁骨45岁那年变了。他在一次越级猎杀（想挑战8级黑森林巡逻者）中被打成重伤，灵魂石板出现了第一条裂纹。从那以后，他的等级不再增长，反而开始缓慢衰退——7级的面板背后，实际的战力已经下滑到6级。他知道这件事不能让任何人知道，否则铁骨帮立刻会分崩离析。

所以他选择藏起来，躲在帮主的位子上不敢退位。他花了大价钱从黑手套商会买药、买情报、买一切能掩盖衰退的东西。他甚至知道黑牙和疤脸在暗斗——但他不敢处理。如果处理黑牙，黑牙勾结血斧的事会闹大；如果扶持疤脸，黑牙会立刻反扑。他只能看着铁骨帮一点点烂掉。

他不恨主角。他甚至有点羡慕主角——这么年轻、等级这么低、但这么敢拼敢杀。如果自己年轻时有主角的勇气，也许现在已经是噬魂者8级了。

| **树敌过多但不在乎** | 知道黑手套商会在挖他的人，也知道马拉领主在盯着他，但他认为“只要我够强，这些人就不敢动我”。 |

#### 背景故事
> ≥200字小传

血斧不是格杀镇人。他来自深渊裂隙更深处的一个叫作“血牙镇”的地方，那个镇子比格杀镇还要混乱——每天都有猎杀者死了直接被路边人开膛取灵魂碎片。他在那种地方杀到7级，手下沾了至少50条人命。30岁那年他厌倦了底层互杀，决定“上个层次”——于是他以“提供武装护卫”的名义来到了格杀镇，建立了血斧佣兵团。

血斧佣兵团只花了3年就从一个5人小队发展到30多人，控制了格杀镇30%的高阶猎杀路线（4-6级区）。在这个过程中，血斧杀了至少2个竞争者、吞并了1个小帮会、逼走了1个敌对势力。他现在是格杀镇的“第二号人物”——仅次于马拉领主。但他不满足：他还想抢马拉的地盘。

血斧有一个不为人知的秘密：他身上有一块奇特的“暗影器官碎片”——从深黑森林的BOSS身上掉落的，能让他的灵魂石板临时提升一次爆发力（持续约30秒）。这是他敢和马拉叫板的原因之一。但他不知道的是，马拉其实也知道这件事。
