# 正文：第贰章 酒馆里的未来人

码头上晒着的咸鱼在日头下翻着白肚皮，腥臭味混着海水的气息从每一块甲板的缝隙里渗出来。博德兰的码头区永远是这样——鱼腥、汗水、隔夜的麦酒味，以及一些更隐秘的、见不得光的东西的气味。

尘心的脚步踩在湿漉漉的石板路上。

“酒馆二楼最里头那个卡座。”这是巴鲁克打听到的信息，“那个姓尤里的情报贩子，每天晚上都在那儿喝酒。”

姓尤里。尘心在记忆里翻找着这个名字——前世动荡之年，码头区的一个小人物，靠着卖情报活过了第一次大清洗。不是英雄，不是强者，但有用。

“站住。”

两个穿灰鼠帮外套的混混拦住了路，其中一个下巴上有道被拳头打出来的旧疤，目光上下打量着尘心：“生面孔啊。过路费五个铜币。”

“三个铜币就够了？”尘心的目光扫过两人——没系统面板弹出，只是凭经验估算，两人都是一阶水准，体态松垮，气感涣散。连威胁都算不上。

“？？？”疤脸愣了一瞬，“你说什么？”

“你们心里其实没底。”尘心跨出一步，挤到两人之间，“码头区的规矩是收保护费不是收过路费，你们俩连这点都分不清楚。第一次干这活吧？上个月血牙帮刚收编了你们，训练还没到位。”

疤脸脸色一变。他旁边那个瘦高个儿下意识后退了半步。

“你怎么——”

“你左边腰间鼓出来的那一块不是武器，是中午买的干粮还没吃完。你右边袖口沾着墨水，”尘心看向瘦高个儿，“不是手上的活计沾的，是抄账簿蹭上去的。你们俩是灰鼠帮新编的内务。”他的目光落在疤脸喉结上，“想知道我怎么知道的？”

疤脸已经说不出话了。

“走吧。”尘心绕过两人，声音落在身后，“告诉瘸腿芬恩，武馆那个人今天晚上会来找他。他可以选择听我说话，或者等我打完再说。”

疤脸和瘦高个儿对望一眼，头也不回地跑了。

酒馆的门吱呀一声推开。昏暗的灯光在那瞬间照出了柜台后面老板娘冷淡的脸，以及角落里几个醉倒在桌子上、拿酒杯当枕头的中年人。空气里弥漫着廉价的麦酒味儿和油烟。

二楼，最里头那个卡座。

坐着一个黑瘦的男人，中等身材，三十岁上下，手里端着一杯麦酒却一口没喝，像是在等什么人。看到尘心上楼，他的眼睛眯了一下。

“你是巴鲁克的师弟？”尤里放下了酒杯，语气里带着警惕，“我以为来的是巴鲁克本人。”

“巴鲁克被你约过来，是想买灰鼠帮的情报。”尘心在对面坐下来，没有接他递过来的酒杯，“但他不知道的事情是——你真正想卖的不是灰鼠帮的情报。”

尤里的警惕又深了一层。

目标：尤里。等级：1级（一阶·凡人）。威胁等级：极低。
核心资产：情报网（覆盖码头区/冒险者公会/部分城卫队）。当前状态：饥饿（手头紧）。

“我想卖什么，得由买方来决定。”尤里挤出一个笑脸，“你师兄说要买灰鼠帮的驻地、人手、头目的习惯。我手上确实有这些，价格公道——”

“三天后。”

尘心突然打断他，眼神平静。

“下周三午后，血牙帮会袭击最后一批抵达博德兰的商会车队。押送目标是三箱铁锭、两箱药剂和一箱——嗯，价值不高但很惹眼的东西。全城一半的目光都会被那场袭击吸引过去。”

尤里的表情僵住了。

“……你怎么知道血牙帮的计划？”

“我不仅知道这个。”尘心向前倾了一寸——极其微小的动作，却让尤里感到整个人被压住了，“我还知道三天后血牙帮动手的时候，灰鼠帮的精锐会全部窝在码头区的地下仓库里。芬恩以为这是我师兄巴鲁克的天，他自己倒是藏得挺好。”

“……”

“你会问我为什么知道这些。”尘心的嘴角动了动，像一个节制的笑，“你不用知道。”

尤里沉默了。

他反复端详着眼前这个年轻武僧的脸——轮廓还带着十几岁的模样，但那双眼睛，那种眼神，不是这个年纪该有的。

“你说完之后，我该怎么做？”

“把你的手伸出来。”

尤里迟疑了一下，还是伸出了右手。尘心没有握住，只是看了一眼——手掌上没有酒癍，指腹有茧，握笔的姿势偏一侧，说明他抄过记过，不是纯武人。

“三天后灰鼠帮窝藏的时候，码头区地下仓库东侧第三根柱子后面有个暗格。里面的东西你拿去卖了，够你吃一年。”

尤里的呼吸急促了一点点。

“我凭什么相信你？”

“因为你现在除了‘情报换钱’之外没有别的路子，而我可以给你一条更好的。”尘心的语气没有起伏，像在说一件已经发生的事，“你卖给我的情报，我可以溢价收。但要快，我只在博德兰待二十天。”

“你一个武僧——”

“你想说的不是这个。”尘心的目光微沉，“你想说的是‘你凭什么有这么多钱’。好，那我换个问题——你猜我是怎么知道血牙帮的计划和灰鼠帮暗格的？”

尤里嘴唇动了动，没能说出话来。

尘心没有等他回答。他从怀里掏出一张羊皮纸，放在桌子上码平，推了过去。

“五天后，码头区第三座栈桥下面，白天退潮的时候，水底三丈深处有一箱沉船遗货。箱子已经松了，扒开礁石就能捞上来。里面的东西你自己处理。”

“……”

尤里的瞳孔微微放大——那是货真价实的未来信息。

“我只要灰鼠帮的情报，驻地从哪儿进、人手分布、芬恩的起居习惯。今晚就能拿到。”

沉默继续了几秒。

然后尤里忽然笑了一声，把酒杯推开了，看着尘心的眼睛，声音低了很多：“你不是普通的武僧。”

“你没说错。”

尘心站起来，把一枚银币放在桌角压住羊皮纸，转身走向楼梯。

“情报，今晚送到武馆后院。你知道怎么避开眼线的。”

脚步声消失在楼梯口。

尤里坐在卡座里，看着桌角的银币，不自觉地捏紧了羊皮纸。他的目光扫过第一行字——“沉船遗货 退潮时 水下三丈 第三座栈桥下”——手指微微颤抖了一下。

这个小武僧凭什么知道五天后的沉船？

不对，他是怎么知道的？

尤里缓缓把羊皮纸折好，塞进怀里，端起那杯早已凉透的麦酒，一饮而尽。

“有意思……”他低声说。

---

酒馆门外的冷风吹散了最后一点烟火气。

尘心走出酒馆的时候，夕阳已经把码头染成了橙红色。海鸟在天空中划着弧线，叫声混杂着潮水的呼吸声。他路过那根自己站过的柱子，上面还有两道浅到几乎看不出的刀痕——是他前世逃亡来到这座城市宰杀第一个追杀者时留下的。

他站住了。

没有回头。

“第一个人头，路上那帮强盗已经收过了。现在是尤里——情报网的第一环。”

他在心里默默盘算着：
- 燃料：情报源✔
- 萌芽中的人脉点：尤里的交情+5（初步信任）
- 灰鼠帮：已完成情报收集，进入打击窗口
- 主线压力：距离影之王降临还有十七天，面板进度——经验值条192/300，约64%。

还不够。

他攥了攥拳头，气脉运行了一圈。气池7/7，状态平稳。三阶的门还远，但LV6的门已经能看见了——再杀三四批地下城的怪物，或者搞定灰鼠帮那帮人头，就能摸到突破的边。

“快一点，再快一点。”他对自己说。

海风把武袍下摆吹得猎猎作响。夕阳尽头，博德兰城墙外的密林深处，有什么东西正悄无声息地翻涌着——它还没有来，但它的脚步已经在倒计时里越来越近。

尘心知道，只有抢在它前面，才能活下来。

### 爽感层·坐标系跃迁检查

- **天赋加速体现**：情报交易中使用重生记忆的三层未来信息——不是靠战斗硬实力，而是靠信息差碾压。第83行信息断奏（“你猜我是怎么知道的？”→沉默）制造阅读爽感。
- **代价约束体现**：尤里追问“我凭什么相信你？”——主角用信息本身证明自己，这段对话构成信息侧的代价交换（不给尤里实际验证前，信息本身就是主角赌的信用成本）。
- **坐标系位置变化**：本章虽然无战斗升级，但尘心在信息世界中的位阶跃升了——从一个码头区的生面孔变成了“知道未来的人”。在博德兰的情报圈坐标系中，他从“零权重”变成了“不可忽略者”。第110-120行的足印刀痕回溯完成情感锚点收束。
