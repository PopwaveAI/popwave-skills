# ch003 · 保护费

武馆的大门被一脚踹开。

木门砸在地上，扬起一阵灰尘。三个壮汉鱼贯而入，为首的光头肩上扛着一根铁棍，脖子上露着一道狰狞的疤痕。

“老王呢？让他出来说话。”

巴鲁克从练功房走出来，两米高的大块头挡在院子中间：“师傅不在。有事跟我说。”

光头上下打量了他一眼，嗤笑一声：“跟你说？你一个臭练武的，能替老王做主？”

“武馆的事，我能做主。”

“行。”光头把铁棍往地上一杵，砸出一声闷响，“这个月的保护费，翻倍。一百枚银币，今天就要。”

巴鲁克眉头一皱：“上个月不是五十吗？怎么翻倍了？”

“这是码头的规矩。你们武馆最近不是接了不少活儿吗？有钱了，就该多交点。”光头咧开嘴，露出满口黄牙，“要不这样——你跪下来给我磕三个头，这个月我给你打个折，收八十。”

巴鲁克的拳头攥紧了。

但他没动。

尘心靠在练功房的门框上，冷眼看着这一幕。

*目标评估：灰鼠帮打手。等级：1-2级。威胁等级：低。*
*三人中光头最强，估计力量14-15，敏捷10-11。*
*正面对抗胜率：100%。*

他的目光扫过巴鲁克的后背。

大师兄在前世替他挡过刀，他知道这个人的底色——憨厚、忠诚、关键时刻从不退缩。

但现在，巴鲁克在忍。

他不是怕打架。他是怕给武馆惹麻烦。

光头走到巴鲁克面前，拿铁棍戳了戳他的胸口：“怎么？不服气？不服气就动手啊。你们武馆不是教拳的吗？让我看看你们有多能打。”

巴鲁克的身体绷紧了。

“动手啊！”

铁棍砸在巴鲁克的肩膀上——啪！

巴鲁克没躲。

光头愣了一下，随即大笑：“还真是个怂包。你们这些练武的，全他妈是花架子。”

他转过身，对手下挥了挥手：“把练功房给我砸了。给老王长长记性。”

两个打手朝练功房走去。

巴鲁克终于动了——他伸手拦住其中一个：“别动武馆的东西。”

“哟？还敢拦？”

光头抡起铁棍，直接朝巴鲁克的脑袋砸下来。

铛！

沉重的碰撞声在院子里炸开。

巴鲁克举起左臂硬接了这一棍，手臂上的皮甲被打得裂开一道口子，鲜血顺着小臂往下淌。

但他没退。

“我说了。别动武馆的东西。”

光头皱起眉头，重新握紧了铁棍：“你他妈找死——”

“够了。”

尘心的声音从练功房门口传来。

三个人同时转头。

尘心缓步走进院子，脚步不急不慢。他在光头脑前三步处停下，平静地看着他。

“滚。”

光头愣了一下，随即笑了：“你谁啊？毛都没长齐，让老子滚？”

“三秒。”

“什么？”

“三秒之内，你不滚——”尘心眼神一沉，“我就把你的腿打断。”

光头脸上的笑容僵住了。

不是因为尘心的话有多狠。而是因为他看到了尘心的眼神——那双眼睛里没有愤怒、没有恐惧、没有任何情绪波动。平静得像一潭死水。

那是只有真正杀过人的眼睛。

光头本能地向后退了一步，但随即又停住了。他在心里骂了自己一句——怕什么？对面就是个二十岁不到的毛头小子，自己带了两个人，手里还有铁棍。

他重新挺直腰板：“小兔崽子，你他妈——”

话音未落，尘心动了。

没有预兆、没有蓄力、没有任何多余的动作。

他的右脚在地面上一蹬，身体骤然前压，在光头举起铁棍的瞬间已经突进到了他的身前三寸。

右掌推出。

震击掌。

手掌结结实实地印在光头的胸口。

那一瞬间，光头感觉自己的心脏像是被一柄铁锤砸中了。胸腔里的空气被一口气挤了出去，他整个人弯着腰向后飞出两米多远，重重地摔在地上，铁棍脱手滚出三米。

“哇——”

光头张嘴吐出一口酸水，胸口疼得几乎喘不上气。

“老大！”

另外两个打手冲上来——一个挥拳，一个从腰间拔出匕首。

*气池消耗：震击掌-1点。当前气池：6/7。*

尘心身体微侧，避开迎面一拳，左手顺势扣住对方的腕关节——喀吧一声轻响，对方的惨叫声还没出口，膝盖已经撞在了他的腹部。

第三个打手的匕首刺到面前。

尘心不退反进，身体下沉让匕首从头顶划过，右手五指如钩扣住对方的手腕，一拧——

铛！

匕首掉落地面。

打手跪倒在地，手腕以一个不正常的角度扭曲着，疼得浑身颤抖。

院子里安静了。

光头趴在地上，捂着胸口拼命喘气。两个打手一个抱着手腕惨叫，一个蜷缩在地吐着酸水。

巴鲁克愣在原地，看着尘心——这个平时不怎么说话、练功时总是一个人闷头打拳的师弟，刚才用了不到三息的时间放倒了三个人。

“你……你什么时候——”

“二阶。”尘心打断他，语气平淡，像是在说今天天气不错。

巴鲁克的嘴张得更大了：“二、二阶？！”

一阶到二阶的差距有多大？巴鲁克跟老王练了五年，还是卡在一阶巅峰。整个武馆只有老王是二阶，但他旧伤复发后战力已经跌到了一阶极限。

现在他师弟，二十岁不到，已经是二阶了。

*目标评估：灰鼠帮打手三人均已失去战斗力。气池剩余：6/7。战斗中消耗：震击掌-1点。*

尘心蹲下身，看着光头的脸：“回去告诉芬恩。武馆的保护费，一毛钱都不会给。”

光头捂着胸口，艰难地喘气：“你……你等着……老大不会放过你的……”

“让他来。”

尘心站起来，拍了拍手上不存在的灰尘：“我在这等他。”

光头咬着牙爬起来，看了尘心一眼，满眼怨毒，但一个字都不敢多说。两个手下挣扎着跟在他身后，三个人夹着尾巴出了武馆大门。

巴鲁克走上去，把门板重新装上。他的左臂还在流血，但他似乎完全没在意：“师弟，你什么时候突破二阶的？”

“几天前。”

“几天前？！”巴鲁克的嗓门拔高了，“你怎么不早说？师傅知道吗？”

“还没来得及告诉他。”

巴鲁克愣了几秒，随即咧嘴笑了，笑得像个孩子：“好啊，好啊！我们武馆终于又出二阶了！”

他用力拍了拍尘心的肩膀——那是他表达高兴的唯一方式，每次都是这么大力。

尘心的肩膀纹丝不动。

巴鲁克拍了两下，忽然想起什么：“对了，你说芬恩……那个灰鼠帮的瘸子，会不会真的带人来？”

“会。”

尘心的回答干脆利落。

他转身走回练功房，经过门槛时停顿了一下：“所以我需要更快的变强。”

巴鲁克目送他走进房内，看着那个背影——师弟今天说的话，比以前一个月说的都多。但他总觉得，师弟说的话里藏着什么他没听懂的东西。

练功房内。

尘心坐在蒲团上，闭上眼。

“气池剩余：6/7。恢复中……预计完全恢复时间：1小时。”
“震击掌熟练度：+3点（155/200）。”
“当前经验值：LV5→LV6进度22%。剩余经验需求：78%。”

太慢了。

按照这个速度，二十天内根本来不及升到三阶。今天暴露实力之后，芬恩会更快来报复，整个武馆都会被拖进冲突里。

我需要更极端的刷级方式。

前世在冒险者公会，他听说过地下城深处的密道——那里有被遗忘的废弃祭坛，盘踞着远超地表难度的怪物群。前世他不敢去，因为够不够强。

这一世——

他睁开眼。

“地下城深处，明天就去。”

倒计时：十九天。
