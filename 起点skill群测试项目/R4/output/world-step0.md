**底牌消费**：骨架.md（7阶武道阶位+混合型引擎：探索型前期+压迫型后期）+ 主角设计.md（冷硬理性型·武力碾压路线·据点型移动模式）+ 地图.md（博德兰10区域分区）+ 势力.md（4层8势力结构）

基于7阶武道阶位体系+冷硬理性型主角（武力碾压路线），提供3个世界结构方案：

---

**A) 据点型——博德兰城为核心，从内向外扩张**

→ 地图布局：以博德兰城（10个区域：贫民窟/码头区→冒险者公会→商业街→城卫队驻地→神殿区→武馆→地下城→城墙→郊外→边境要塞废墟）为第一卷核心舞台，卷2向外扩展至剑湾北部，卷3扩展至深水城，逐卷外扩
→ 势力：4层8势力——底层(灰鼠帮/血牙帮) → 中层(城卫队/冒险者公会/商会联盟) → 中上层(城主府/贵族派系/法师塔残余) → 顶层(落凡圣者·影之王)
→ 危机类型：递进型——灰鼠帮试探→贵族打压→林克斯渗透→影之王投影降临，每幕升级一层威胁
→ 空间叙事策略：据点推进式——主角在博德兰建立根据地（武馆+城卫队盟约），从城内向外辐射，每解锁一个新区域=解锁新阶位门槛+新势力碰撞
→ 与力量体系的匹配：地图.md的10个区域已按阶位壁垒设计（贫民窟=一阶区→神殿区=三阶+区），区域阶位与主角成长曲线同步
→ 与主角性格的匹配：冷硬理性型+据点型=先在根据地稳扎稳打发育，不盲目扩张，符合"先评估→再决策"的行动偏好
→ 影响标注：世界结构定下后，plot的分幕走向以博德兰城内4幕为框架（ch001-010武馆→ch011-025城内势力→ch026-035影之王降临→ch036-040撤离），write的空间尺度从城内巷战逐步升级到城墙级战场。

---

**B) 游击型——多城+道路，移动叙事**

→ 地图布局：博德兰(卷1前10章) → 剑湾北部道路(卷1后30章) → 废弃法师塔 → 边境要塞废墟，主角从开局就在移动中，不在任何一个城市建立长期据点
→ 势力：3层5势力——精简为底层(盗匪) → 中层(冒险者公会各城分部) → 顶层(落凡圣者追兵)，去掉城卫队/贵族/商会等本地势力
→ 危机类型：追逐型——影之王追兵不断追杀，主角在移动中应对，每到一个新地点就遇到新危机
→ 空间叙事策略：公路片式——主角带着巴鲁克在路上流浪，每到一处短暂停留→冲突→离开
→ 与力量体系的匹配：移动叙事中难以建立稳定的修炼场所（武馆/冥想室），气功/禅定的修炼节奏会被打乱
→ 与主角性格的匹配：冷硬理性型的"先评估→再决策"需要稳定的信息输入，游击型的不确定性会削弱理性决策的优势
→ 风险：与骨架.md地图.md的博德兰10区域详细设计冲突——如果主角不在博德兰停留，地图.md的80%内容浪费
→ 影响标注：此方案需要重构地图.md（从城内分区改为道路+驿站布局），势力.md的本地势力（灰鼠帮/城卫队/贵族）全部失效。

---

**C) 争霸型——剑湾大区域多子区域，棋盘式多势力交叉**

→ 地图布局：开局即展示剑湾全景（博德兰+深水城+其他3-4个城邦），主角在多个城邦间穿梭，每城邦有独立势力格局
→ 势力：5层12+势力——在原有8势力基础上增加深水城势力、其他城邦势力、跨城邦联盟，形成棋盘式多势力交叉
→ 危机类型：棋局型——多势力同时博弈，主角在多方势力间合纵连横
→ 空间叙事策略：棋盘式——主角同时面对多个区域的多个势力，需要在不同城邦间分配精力
→ 与力量体系的匹配：第一卷主角仅二阶→三阶，在多城邦棋盘上过于弱小，无法参与跨城邦博弈
→ 与主角性格的匹配：冷硬理性型适合棋盘博弈，但第一卷的主角实力不足以支撑争霸叙事——争霸需要四阶+
→ 风险：第一卷展开多城邦全景会分散读者注意力，与"爽文"的聚焦式叙事冲突；且骨架.md引擎演化节点4（传奇·卷3-4）才进入世界级博弈，提前到卷1会打乱引擎节奏
→ 影响标注：此方案与骨架.md的演化节点冲突——引擎节点3（入局·卷2）才首次与落凡神祇交手，争霸型第一卷就需要多势力博弈，节奏超前。

→ 阶段性危机：
  - 卷1：小范围蝴蝶效应——救了前世该死的人→引发连锁反应
  - 卷2-3：中范围蝴蝶效应——改变的城市命运引来不同的圣者关注
  - 卷4-5：大范围蝴蝶效应——前世的盟友变成敌人，前世的敌人变成盟友
  - 卷6-7：终极蝴蝶效应——时间线完全偏离，主角必须靠自己的判断而非前世记忆登神
→ 危机驱动机制：蝴蝶效应为主+代价累积为辅，倒计时弱化
→ 风险：蝴蝶效应型危机在前期的"爽感"不足——读者更想看"我知道你不知道"的信息差碾压，而不是"我的信息越来越不准"的焦虑
