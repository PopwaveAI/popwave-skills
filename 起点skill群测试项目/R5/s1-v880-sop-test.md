## S2 力量体系+动力引擎设计

基于S1选定的世界（睡眠剥夺·清醒者世界），这个世界允许什么干预方式？

> 设计前提：力量体系 = 干预这个世界的能力。力量≠危机（清醒病是被干预对象，不是力量来源）。
> 世界矛盾：清醒病改变了所有人的生存规则——不睡→看到里层→里层渗入现实
> 世界危机：清醒病每月+2%扩散，5年倒计时，渗入口随清醒者人数指数增长

### A) 里层锚定体系（最适配世界矛盾）

→ 干预方式：清醒者通过主动接触里层实体，建立"锚定关系"——把里层实体锚定在现实中，既防止它自由扩散，又能借用它的能力干预世界
→ 主养成线：6阶——
  - 第一阶「窥见」：被动看到里层实体（所有清醒者自动获得，不算真正的"干预"）
  - 第二阶「触碰」：主动接触里层实体而不被吞噬（需要练习——里层实体对清醒者的精神有侵蚀性，触碰时间过长会"模糊"——分不清现实和里层）
  - 第三阶「锚定」：将一个里层实体锚定在现实中——锚定后该实体无法自由移动/扩散，但会持续消耗锚定者的"清醒度"（加速清醒病恶化）。锚定者可以借用锚定实体的能力
  - 第四阶「契约」：与里层实体建立双向契约——实体提供稳定能力，锚定者提供"现实锚点"（实体需要锚定者才能稳定存在于现实）。契约实体可以离开锚定者一定范围行动
  - 第五阶「折叠」：主动操控现实与里层的边界——把现实的一小块区域"折叠"进里层（制造临时隔离区），或把里层的一小块"拉"进现实（召唤实体）。折叠是最高风险的操作——每次折叠都会在现实留下"裂缝"，裂缝不愈合就成为新的渗入口
  - 第六阶「穿行」：肉身进入里层——在里层中，物理法则不同，清醒病不恶化（因为里层没有"睡眠"的概念），但里层会试图"同化"闯入者。穿行者需要在被同化前完成目标并返回

→ 子养成线：5条——
  - ①锚定术（锚定/契约/解除锚定的技巧——不同实体需要不同锚定方式）
  - ②里层感知（感知范围/精度/能看到的里层深度——高级感知能看到"里层地形"）
  - ③现实折叠（操控边界的技巧——折叠面积/持续时间/裂缝修复）
  - ④清醒者网络（人脉资源——其他清醒者的情报/庇护/交易/联合锚定）
  - ⑤助眠术（延缓清醒病恶化的手段——冥想/药物/里层能量反哺。这是"续命"线，不是"变强"线，但不点就会死）

→ 引擎范式：压迫型——越深入里层交互，清醒病恶化越快。每次锚定/折叠/穿行都消耗"清醒度储备"，储备耗尽=彻底不清醒=变成里层的一部分。主角在"变强vs续命"间被迫选择

→ 与世界的适配度：干预什么=里层渗入现实（世界危机的直接表现）；怎么干预=锚定实体防止扩散+借用实体能力+折叠边界。力量体系直接干预世界危机的运作机制（渗入口），但每次干预都加速危机对自身的侵蚀（清醒病恶化）。干预和代价一体两面但不混淆

→ 代价：干预世界要付出什么——①直接代价：每次锚定/折叠消耗清醒度储备，加速清醒病恶化（不可逆）②间接代价：折叠留下的裂缝会成为新渗入口，主角在"消灭渗入口"和"制造新渗入口"之间两难③终极代价：穿行里层时有被同化的风险——不是死亡，而是变成"不属于现实也不属于里层"的东西

### B) 清醒共鸣体系（变体）

→ 干预方式：不锚定实体，而是利用清醒者之间的"共鸣"——多个清醒者可以共振放大干预能力，但需要至少一人承受"模糊"副作用
→ 主养成线：5阶——独自感知→双人共鸣→小队共鸣→群体共鸣→全域共鸣（全域共鸣=所有清醒者同时共鸣，可以暂时"关闭"一个城市的里层渗入口，但参与者集体清醒病恶化一级）
→ 引擎范式：混合型——前期探索（寻找共鸣伙伴），后期压迫（共鸣规模越大副作用越大）
→ 与世界的适配度：干预方式偏向"集体对抗"而非"个人英雄"，适合群像叙事；但干预能力依赖人数，清醒者越多=共鸣越强=副作用越大（正反馈陷阱）
→ 代价：共鸣的"模糊"副作用——参与者暂时分不清现实和里层，持续时间随共鸣规模增长。群体共鸣后参与者需要72小时恢复，期间完全无法干预

### C) 梦境反攻体系（跨类型）

→ 干预方式：清醒者进入"反向睡眠"状态——身体在睡觉（延缓清醒病恶化），意识进入里层的"梦境层"进行干预。在梦境层可以改变里层实体的行为逻辑，但无法直接影响现实
→ 主养成线：6阶——浅眠入梦→深眠探索→梦境塑形→梦境植入（在里层实体脑中植入"行为指令"）→梦境构建（在里层建造永久结构）→梦境置换（将里层的一片区域与现实置换——现实的一块进入里层，里层的一块进入现实）
→ 引擎范式：探索型——主角在梦境层探索里层的运作规律，用知识而非力量干预
→ 与世界的适配度：干预方式最反直觉——"用睡眠对抗不眠症"。身体在睡（延缓恶化），意识在里层干预。但梦境层的行为只能间接影响现实（改变实体行为→实体在现实的表现变化），不如锚定体系直接
→ 代价：梦境层停留时间有限（身体睡眠时长），且梦境层有"梦魇"——里层的防御机制，会追杀闯入者。被梦魇抓住=意识困在里层永远醒不来

## S3 主角设计

基于S1选定的世界（睡眠剥夺·清醒者世界）和S2力量体系（里层锚定体系），这个世界会催生什么样的主角？

> 设计前提：主角从世界×力量体系中长出。欲望来自世界矛盾，行动方式来自力量体系，金手指是专属干预手段。
> 用户S2追问结果：b) 干预和代价的关系（变强=加速自我毁灭+制造新裂缝）

### A) 睡眠科护士·治愈者（与世界矛盾最匹配）

→ 身份底色：医院睡眠科护士，还没感染清醒病。每天接触清醒病患者。她是全城唯一的"治愈者"——触碰能让清醒者入睡1小时
→ 欲望来源：世界矛盾（清醒病在夺走所有人的睡眠）→ 想找到彻底治愈清醒病的方法，不只是暂时的1小时
→ 行动方式：锚定体系第三阶「锚定」——她还没感染清醒病，看不到里层实体，但她的"治愈"触碰本质上是一种反向锚定（把清醒者的意识锚定回现实，暂时拉离里层）。她在尝试将治愈能力发展为正向锚定——不靠清醒病也能锚定里层实体
→ 金手指：治愈触碰——能让清醒者暂时入睡（全城唯一），且触碰里层实体时可以"安抚"实体（降低其侵蚀性，使锚定更安全）。这是她独有的干预手段——别人锚定要硬抗侵蚀，她锚定可以先把实体"安抚"再锚定
→ 金手指限制：①她还没感染清醒病，正常情况下看不到里层实体——只有触碰清醒者时，能通过清醒者的眼睛"借用"视野②每次治愈/安抚都消耗自身的"睡眠储备"——她的睡眠时间在缩短，她在用自己的睡眠换别人的。治愈者终将变成感染者
→ 金手指代价：自身睡眠时间持续缩短——治愈别人=透支自己。当她的睡眠时间归零，她会成为最后一个清醒者，而且是"因为治愈别人而感染"的清醒者
→ 与世界的切入点：她被全城追猎——政府想研究她的治愈能力，黑市想买卖她的触碰，Pharma公司想复制她的能力。她必须在逃亡中寻找彻底治愈清醒病的方法，而不是只能暂时缓解

### B) 第30天清醒者·便利店夜班（反差型）

→ 身份底色：大学生，便利店夜班店员，感染清醒病第30天。还在坚持上班——因为夜班全是清醒者，他是唯一能正常和顾客交流的店员（其他清醒者多少都有点"模糊"）。他盯着监控看了一整年，发现了里层实体进入便利店的规律
→ 欲望来源：世界矛盾（清醒病改变生存规则）→ 最初只想活下来，后来发现便利店是里层渗入口的"节点"——每个深夜便利店的监控里都有看不到的"顾客"
→ 行动方式：锚定体系第二阶「触碰」→第三阶「锚定」——他通过长期观察掌握了里层实体的行为规律，比大多数清醒者更擅长"触碰"而不被侵蚀。他的锚定方式很独特：用便利店的商品作为锚定媒介（实体进入便利店后，他可以用特定商品"锚定"实体——比如用过期牛奶锚定一个液态实体，用打火机锚定一个热态实体）
→ 金手指：监控视野——通过任何监控摄像头都能看到里层实体（不需要自己看到）。这让他在清醒者中拥有最广的感知范围——整个城市的监控都是他的眼睛。而且他看了一年监控，对里层实体的行为模式了如指掌
→ 金手指限制：①只能通过监控看到，没有监控的地方是盲区②监控视野是"平面"的，无法判断实体的深度/距离③需要实时盯着监控画面才能看到，不能同时做其他事
→ 金手指代价：长期盯着监控看里层实体，他的"模糊"程度在加速——不是清醒病恶化（他的睡眠时间稳定在0小时），而是现实感在模糊。他开始分不清监控画面里的里层实体和现实里的顾客。某天他可能对着一个真实顾客说"你不是人"
→ 与世界的切入点：他发现便利店是里层渗入口的节点——每个24小时便利店都是一个微型渗入口。他白天送货时发现，全市的便利店形成了一个网络，里层实体通过便利店网络在现实中移动

### C) 退休刑警·清醒病家属（成长型·被迫卷入）

→ 身份底色：退休刑警，60岁，没感染清醒病。老伴感染了，第45天，已经"模糊"到认不出他了。他每天去睡眠科陪老伴，同时用刑警的老办法调查清醒病的起源——他不信这是"病"，他觉得这是"案件"
→ 欲望来源：老伴——世界矛盾从"社会新闻"变成了"我的家庭"。欲望是找到清醒病的源头，治好老伴
→ 阻碍来源：世界危机（清醒病不可逆）+自身局限（60岁退休刑警，没感染看不到里层，体力不如年轻人）+官方阻力（政府不想让民间调查清醒病起源）
→ 行动方式：锚定体系第一条子养成线"清醒者网络"——他不做锚定（看不到里层），但他在清醒者中建立情报网。他的刑警技能（审讯/侧写/证据链）让他成为清醒者中最会"查案"的人。他指挥年轻清醒者去锚定/感知，自己负责分析和推理
→ 金手指：刑警直觉——他对"异常"的敏感度极高，即使看不到里层实体，也能通过现实中的蛛丝马迹推断出里层实体的位置和行为。本质上是一种"间接触发"的里层感知——别人的感知是"看到"，他的感知是"推理出来"
→ 金手指限制：①推理需要时间和线索，不像直接看到那么快②推理可能有误，特别是在面对完全未知类型的里层实体时③他没感染清醒病，无法直接锚定/折叠/穿行，所有干预都依赖年轻清醒者执行
→ 金手指代价：他越深入调查，越接近清醒病的起源——而起源可能在里层。调查的终点可能需要他主动感染清醒病才能看到真相。治好老伴的答案和感染清醒病的风险是同一个东西
→ 与世界的切入点：他在老伴的"模糊"呓语中发现了一条线索——老伴反复念叨一个地名。他追查这个地名，发现它是20年前一个被废弃的睡眠实验室。清醒病不是自然产生的，是人造的

## S3选择后的金手指深化（模拟Agent根据追问结果细化）

用户被"碰撞方式"吸引，金手指深化方向：

**监控视野 → 便利店网络 → 渗入口地图**

主角发现便利店是渗入口节点后，他的金手指从"单店监控"进化为"全市便利店监控网络"——他可以接入任何便利店的监控系统，看到全市里层实体的移动路径。这让他成为唯一能绘制"渗入口地图"的人。

但代价同步升级：接入越多监控，现实感模糊越严重。他开始用便利店的广播系统自言自语——因为分不清是在对真实顾客说话还是在对里层实体说话。同事以为他疯了，只有清醒者知道他"看到的"是真实的。

## 测试复盘：力量体系 v8.7.0 vs v8.8.0

| 维度 | v8.7.0（旧·有问题的） | v8.8.0（新·干预世界的能力） |
|:--|:--|:--|
| 定义 | 清醒度阶梯（睡眠时间做阶位） | 里层锚定体系（干预里层渗入现实的能力） |
| 力量和危机的关系 | 混为一谈（都是清醒病） | 分离（力量=锚定/折叠/穿行，危机=清醒病扩散） |
| 主角怎么变强 | 等病情加重（被动） | 主动探索里层+练习锚定+建立契约（主动） |
| 代价机制 | 变强=病情恶化（但变强本身被动） | 变强=主动锚定里层，代价=消耗清醒度储备+折叠制造新裂缝（主动选择付出代价） |
| 干预什么 | 无（力量体系没回答"干预什么"） | 里层渗入现实（直接干预世界危机的运作机制） |
| 怎么干预 | 无 | 锚定实体防扩散+借用实体能力+折叠边界+穿行里层 |
| 跨赛道通用性 | 不通用（只适用清醒病设定） | 通用（"干预世界的能力"定义适用于任何赛道） |

### 金手指归入S3的效果

| 维度 | 旧SOP（金手指单独S4） | 新SOP（金手指归入S3） |
|:--|:--|:--|
| 设计连贯性 | 主角和金手指分开设计，可能不匹配 | 主角+金手指一体设计，金手指=主角的专属干预手段 |
| 与力量体系的关系 | 金手指独立于力量体系 | 金手指在力量体系框架内（监控视野=锚定体系的感知子线增强版） |
| 限制和代价 | 单独设计，可能和力量体系矛盾 | 和力量体系一致（监控视野的模糊代价=锚定体系的清醒度消耗） |
| 交互轮次 | 5轮（多一轮） | 4轮核心+1轮可选（更紧凑） |

### 三个主角的镜头换人验证（反向验证S1世界质量）

| 主角 | 金手指 | 故事方向 | 精彩度 |
|:--|:--|:--|:--|
| 睡眠科护士·治愈者 | 治愈触碰+安抚实体 | 被追猎的治愈者，用睡眠换生命 | 高（追猎+自我牺牲） |
| 便利店夜班·第30天 | 监控视野+商品锚定 | 渗入口地图绘制者，日常物×里层 | 高（日常质感+悬疑） |
| 退休刑警·家属 | 刑警直觉+间接感知 | 追查清醒病起源的老年侦探 | 高（推理+情感） |

三个完全不同风格的故事在同一个世界里都精彩——**反向验证了S1世界的镜头换人检验**。好舞台就是换什么主角都能好看。
