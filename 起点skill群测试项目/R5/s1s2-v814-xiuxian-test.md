## S0 世界偏好收集（前置收集·非选项轮）

> 在给世界方案之前，先了解用户对"世界"的偏好。4个问题，快速过。

```
在给你世界方案之前，先了解你的偏好（回答即可，不需要详细展开）：

   b) 衰减型（资源/环境持续恶化——温水煮青蛙式绝望）
   c) 扩散型（危机从一个点不断蔓延——恐惧来自不可控范围）

4. 力量体系落点偏好+篇幅预期：

   b) 低武→中武——跨第一个质变点，篇幅50-200万字
```

## S1 世界构筑（核心轮·第一轮）

> 好的小说舞台 = 完善表世界 + 危机引擎（机遇轴+危险轴+咬合机制）+ 增量吸引力。
> 世界先于主角存在——矛盾不需要主角就已存在，主角是被扔进去的人。

### Agent 输出

基于你的赛道（修仙）、增量吸引力偏好（经典类型框架·凡人流/苟道流）、危机节奏偏好（衰减型）和落点偏好（低武→中武，100万字），提供3个世界方案：

---

#### A) 末法残局

→ **世界矛盾性质**：资源垄断 + 阶级压迫——灵气衰退导致灵脉枯竭，大仙门垄断最后的灵脉资源，散修/低灵根者被彻底锁死在底层

→ **表世界**：灵气衰退已持续三百年，曾经遍地灵泉的世界如今灵气稀薄如秋末残阳。大仙门盘踞最后的灵脉矿脉，以"护道"之名封锁资源；散修沦为矿工、药农、符匠，在灵气稀薄的外围城区讨生活。茶馆里说书人讲着三百年前"一剑开天"的旧事，灵药铺的柜台后摆着越来越淡的丹药，坊市里散修们为一株三年份的灵草讨价还价——曾经这是杂灵根弟子都不屑一顾的垃圾。镇子的护城阵法每年来仙门弟子修补一次，但修补间隔越来越长，去年那次甚至没人来。

→ **危机引擎**：
  - **机遇轴**：秩序崩塌型——灵气持续衰退，大仙门为争夺最后的灵脉彼此开战，外围防线收缩，曾经被仙门垄断的矿脉、洞府、秘境逐渐被放弃。
    - 打破了什么：仙门对灵脉资源的绝对垄断秩序
    - 空出了什么：被放弃的废弃灵矿、无人看守的荒废洞府、仙门撤防后暴露的远古秘境入口
    - 主角怎么借力：作为灵根低劣被仙门拒之门外的散修，趁仙门收缩防线，进入被放弃的区域搜刮残灵、残阵、残药——正统修士看不上的残渣，却是散修的全部
    - 没有危机能上升吗：**不能**——灵气衰退前，仙门垄断一切灵脉，散修连入门资格都没有，低灵根者一生只能做杂役；正是衰退导致仙门收缩，才在防线外围撕开了散修可以渗透的缝隙

  - **危险轴**：
    - 动力=不攀登会怎样：灵气衰退导致体内灵力自然流失（"灵枯"现象），不持续修炼补充就会灵力枯竭、修为倒退，最终沦为凡人甚至灵枯而死。这不是"不变强就原地踏步"，而是"不变强就倒退乃至死亡"
    - 机会=危机中出现了什么新资源/位置/关系：废弃灵矿中残留的"灵髓"（灵脉枯竭后凝结的精华，正统修士从未关注）；仙门撤防后无主的荒废洞府成为散修据点；因仙门收缩而流落坊市的仙门弃徒带来功法碎片和情报；灵兽因栖息地灵气枯竭而迁徙，沿途暴露出被它们守护的灵药
    - 道义=灰色——主角不是正道侠客，他在废墟中搜刮、在仙门与散修的夹缝中求生、有时利用他人的困境。但有强力危险前置支撑：灵枯倒退的生存压力让他"被逼无奈"，灰色行为有正当性基础

  - **咬合机制**：螺旋上升——灵气进一步衰退→仙门收缩更多防线→更多废区暴露（机遇扩大）→同时灵枯加速（危险加深）→主角必须抓住新机遇更快攀升→攀升后触及更深层的仙门利益（危险升级）→需要更高层级的机遇→灵气继续衰退暴露更深秘境……机遇与危险互相驱动，形成螺旋

→ **增量吸引力**：经典类型框架 — 凡人流（散修在末法中从废墟捡残渣起家，每一分收获都来之不易），蹭"凡人流"读者已有的认知资产（低灵根/散修/苟道发育/资源稀缺），读者一看就知道是什么味

→ **表世界自检**：规则自洽[✓] 生活质感[✓] 结构性矛盾[✓] 力量嵌入社会[✓]
  - 规则自洽：灵气衰退→灵脉枯竭→仙门收缩→散修渗透废区→残灵搜刮→灵枯倒退，子系统间有因果链
  - 生活质感：坊市讨价还价/灵药铺越来越淡的丹药/说书人讲旧事/护城阵修补间隔变长/散修做矿工药农符匠
  - 结构性矛盾：仙门垄断vs散修求生、灵气衰退vs修士需要灵气——不依赖主角就有张力
  - 力量嵌入社会：灵力=社会地位硬通货，灵根=阶级，灵脉=生产资料，散修=底层劳工

→ **危机引擎自检**：运作机制[✓] 不可逃避[✓] 多层次[✓] 持续恶化[✓] 渗透日常[✓] 机遇-危险咬合[✓]
  - 运作机制：灵气衰退→仙门收缩→废区暴露→散修渗透→灵枯倒退→必须攀升，体系化循环
  - 不可逃避：灵枯是生理现象，不修炼就倒退死亡，主角绕不过去
  - 多层次：个人（灵枯）→社区（坊市衰落/阵法失修）→世界级（仙门大战/灵脉枯竭）
  - 持续恶化：灵气持续衰减，有明确的方向性和加速趋势
  - 渗透日常：丹药越来越淡、阵法越来越弱、灵兽迁徙、物价飞涨——危机是生活底色
  - 机遇-危险咬合：螺旋上升，机遇扩大与危险加深同步

→ **增量吸引力**：命中1个增量源（经典类型框架·凡人流）

→ **镜头换人检验**：
  - 换主角A（仙门弃徒，被逐出后沦为散修）+金手指（残缺功法自行推演）→ 一个被仙门抛弃的天才在废区中重建修为、暗中积蓄力量最终复仇仙门的故事
  - 换主角B（灵药铺掌柜，低灵根但精通药理）+金手指（能感知灵药残留年份和药性）→ 一个药商在末法中靠灵药生意崛起、建立散修药材联盟、以商道对抗仙门垄断的故事
  - 判定：**通过**——两个换主角的故事都精彩，世界有同人创作空间

#### B) 灵潮枯竭

→ **世界矛盾性质**：资源垄断 + 制度压迫——灵气以"灵潮"形式周期涌动，如今灵潮越来越弱、间隔越来越长，修仙世家垄断仅存的"潮井"（灵潮涌出之地），外姓修士需缴重税才能修炼

→ **表世界**：这个世界的灵气以"灵潮"形式存在——曾经每隔数十年灵潮涌动一次，灵潮过处万物生长、修士突破。如今灵潮越来越弱、间隔越来越长，上一次灵潮已是五十年前。修仙世家垄断了仅存的"潮井"（灵潮涌出之地），以潮井为中心建立家族领地，外人需缴纳重税才能在潮井辐射区修炼。坊间流传着"等潮人"的说法——散修们守在荒野中等待不知何时会来的微弱灵潮余波，像等待甘霖的旱灾灾民。镇子上的灵潮预报旗已经三年没动过了，曾经热闹的灵潮节如今冷冷清清，只有老人还在按照旧历筹备。

→ **危机引擎**：
  - **机遇轴**：制度压迫型——修仙世家对潮井的垄断本身就是危机。灵潮枯竭导致家族内部争夺潮井控制权的内战爆发，垄断体系从内部瓦解。
    - 打破了什么：世家对潮井的世袭垄断制度
    - 空出了什么：内战中失败的旁系子弟流落在外带来功法和潮井情报；被战火破坏的潮井辐射区防线出现缺口；家族秘密供奉的"养潮阵法"残片在战乱中流出
    - 主角怎么借力：作为在潮井辐射区边缘交税修炼的底层修士，趁家族内战时渗透进被战火波及的辐射区核心，接触到本被世家垄断的潮井本源
    - 没有危机能上升吗：**不能**——灵潮枯竭前，世家垄断一切潮井，外姓修士只能交重税在辐射区边缘修炼，一生天花板不过炼气后期；正是内战打破了垄断，才让外姓修士有了接触潮井核心的可能

  - **危险轴**：
    - 动力=不攀登会怎样：灵潮枯竭导致辐射区灵气浓度持续下降，交不起税的修士被逐出辐射区→灵气断绝→修为停滞甚至灵力倒退→沦为凡人。而且灵潮枯竭引发灵兽暴动（栖息地灵气消失，灵兽向外迁徙攻击人类聚落），不攀登连自保都做不到
    - 机会=危机中出现了什么新资源/位置/关系：内战中流出的功法残篇和潮井情报；被破坏的潮井辐射区防线缺口；失势旁系子弟带来的人脉和资源；灵兽暴动时击杀灵兽获得的灵核和兽材；被战火暴露的家族地下秘库
    - 道义=中间带——主角不是正邪任何一方，他在家族内战的夹缝中寻找最优解，既不帮世家维护垄断，也不加入叛军推翻秩序，而是趁乱获取资源壮大自己。正当性来自"世家垄断本就不义，主角不过是拿回本该属于所有人的灵潮使用权"

  - **咬合机制**：周期型——灵潮枯竭是周期性的（每次灵潮越来越弱但仍有余波），每次灵潮余波到来时既是机遇（灵气短暂充盈可以突破/采集灵材）也是危险（灵兽暴动/潮井争夺战加剧）。100万字篇幅设计3-4次灵潮余波周期，每次余波之间穿插家族内战和散修崛起剧情

→ **增量吸引力**：经典类型框架 + 现实映射 — 苟道流（低调发育，在世家内战中不站队，闷声捡漏），同时灵潮枯竭+世家垄断映射"资源垄断/阶级固化"的现实焦虑

→ **表世界自检**：规则自洽[✓] 生活质感[✓] 结构性矛盾[✓] 力量嵌入社会[✓]
  - 规则自洽：灵潮枯竭→潮井争夺→家族内战→垄断瓦解→外姓渗透→灵兽暴动，因果链清晰
  - 生活质感："等潮人"/灵潮预报旗/灵潮节冷清/潮井税/辐射区边缘修炼——有独特的生态细节
  - 结构性矛盾：世家垄断vs外姓修士、灵潮枯竭vs修士依赖灵潮
  - 力量嵌入社会：潮井=生产资料、潮井税=地租、世家=封建领主、等潮人=流民

→ **危机引擎自检**：运作机制[✓] 不可逃避[✓] 多层次[✓] 持续恶化[✓] 渗透日常[✓] 机遇-危险咬合[✓]
  - 运作机制：灵潮周期性枯竭→家族争夺潮井→内战→垄断瓦解→外姓渗透→灵兽暴动→需要更强力量→等待下次灵潮
  - 不可逃避：辐射区灵气下降+灵兽暴动，主角无处可逃
  - 多层次：个人（修为停滞）→社区（坊市凋敝/等潮人流民化）→世界级（家族内战/灵潮断绝）
  - 持续恶化：灵潮越来越弱、间隔越来越长、灵兽暴动越来越频繁
  - 渗透日常：潮井税越来越重/灵潮节越来越冷清/预报旗不动/等潮人的日常
  - 机遇-危险咬合：周期型，每次灵潮余波同时带来机遇和危险

→ **增量吸引力**：命中2个增量源（经典类型框架·苟道流 + 现实映射·资源垄断/阶级固化）

→ **镜头换人检验**：
  - 换主角A（世家旁系子弟，被嫡系排挤后流落外围）+金手指（能感应潮井脉动方向）→ 一个失势子弟利用对潮井的了解，在内战中暗中操控灵潮走向、最终颠覆嫡系的故事
  - 换主角B（灵兽猎人，以猎杀暴动灵兽为生）+金手指（灵兽灵核提纯）→ 一个猎人在灵潮枯竭中以灵兽为核心资源崛起、建立灵兽材料商业帝国的故事
  - 判定：**通过**——两个换主角的故事都精彩

#### C) 天道锈蚀

→ **世界矛盾性质**：规则崩坏——天道法则本身正在"锈蚀"，修炼功法、渡劫飞升、阵法符箓赖以运行的天道法则越来越不可靠

→ **表世界**：这个世界的修仙依赖"天道法则"——修炼功法、渡劫飞升、阵法符箓，一切都在天道法则的框架内运行。但天道本身正在"锈蚀"。功法有时运转顺畅有时突然逆行，渡劫天雷时有时无偶尔还会劈错人，阵法偶尔自行激活偶尔彻底失灵，符箓成功率从九成跌到六成还在降。修士们称之为"天道锈蚀"——法则的铁律开始生锈，可靠性逐年下降。大仙门花费巨资维护"法则稳定区"（用上古阵法锚定天道法则，使其在一定范围内正常运作），稳定区外则是"锈蚀区"，法术不稳定但也不受天道约束。坊市里"稳定符"价格飞涨，修士出门前要先卜算今日法术成功率，茶馆里有人讲某某散修在锈蚀区用"不稳定功法"反而突破了的奇闻。

→ **危机引擎**：
  - **机遇轴**：宿命诅咒型——天道锈蚀是周期性的（上古记载每隔万年天道锈蚀一轮，上一次毁灭了一个纪元），锈蚀过程本身=成长通道。
    - 打破了什么：天道法则对修炼路径的绝对约束——功法必须按天道法则运转、渡劫必须按天道法则考验、阵法必须按天道法则构建，这些"铁律"在锈蚀中失效
    - 空出了什么："法则锈蚀区"中不受天道约束的修炼方式——逆行功法、变异阵法、不需要渡劫的突破路径；天道法则失效后暴露的"法则骨架"（法则的本源结构，平时被法则运行遮蔽，锈蚀时才能观测到）
    - 主角怎么借力：作为灵根低劣无法通过正统功法修炼的散修，进入锈蚀区寻找不受天道约束的"野路子"——在法则失灵处，正统修士的功法失效，而主角发现了一种利用法则锈蚀本身的修炼方式
    - 没有危机能上升吗：**不能**——天道未锈蚀时，修炼路径被天道法则严格规定，灵根低劣者连炼气都进不去；正是法则失效，才在锈蚀区中撕开了正统路径之外的"野路子"

  - **危险轴**：
    - 动力=不攀登会怎样：天道锈蚀是渐进的，法则越来越不稳定。不攀登进入更高层级（掌握法则骨架、利用锈蚀而非被锈蚀毁灭），就会在法则完全崩解时被波及——法术失灵=失去自保能力，阵法失灵=失去庇护所，渡劫失灵=无法突破甚至被紊乱天雷劈杀。天道锈蚀是倒计时，不攀升就等死
    - 机会=危机中出现了什么新资源/位置/关系：锈蚀区中法则失效暴露的"法则骨架"（法则的本源结构，可观测可利用）；法则失灵后正统功法逆运转产生的"逆灵"（一种新的能量形态）；因天雷紊乱而未被劈毁的"漏劫者"（渡劫失败但未死的修士，掌握独特的劫雷感悟）；法则锈蚀导致上古封印松动，释放出被封印纪元的遗产
    - 道义=灰色——主角利用天道锈蚀（法则失灵）来修炼，在正统修士眼中是"逆天道而行"的异端。但天道本身在崩坏，"顺应天道"已不能保证生存，"利用锈蚀"是被逼出来的生存策略。正当性来自"天道已不可靠，不利用锈蚀就是等死"

  - **咬合机制**：一体两面——天道锈蚀本身同时是危险来源和机遇来源。同一场锈蚀：让法术失灵（危险）=让法则骨架暴露（机遇）；让渡劫天雷紊乱（危险）=让漏劫者存活带来劫雷感悟（机遇）；让阵法失灵（危险）=让上古封印松动释放遗产（机遇）。最高级咬合形式——危机本身的双面性驱动故事，不需要额外设计"机遇出现"的情节，因为机遇和危险是同一件事的两面

→ **大质变点事件**：蚀灵化形——主角在废区猎杀一头垂死灵兽，将灵兽内丹喂食给体内蚀灵，蚀灵吸收内丹精华后完全觉醒，从寄生状态化为独立灵体。这一刻，主角从"蚀灵寄生于体内增强自身"（低武·量化强化区）跨越到"蚀灵独立化形、人灵协同战斗"（中武·规则干预区）。干预方式质变：从"内部共生增强"变成"独立灵体外放协同"。

  2. 共生协调线——提升与蚀灵的共生效率，解锁新特性（侵蚀/感知/防护/攻击），形成"协调训练→解锁特性→实战验证→新特性"的循环

→ **引擎范式**：压迫型——蚀灵饥饿压力驱动（不喂养就反噬，反噬就死），核心张力是"蚀灵越来越强→食量越来越大→需要更危险的食料→更危险的废区→更强的蚀灵"的正反馈压迫循环

→ **与世界的适配度**：高度适配——蚀灵是末法世界的产物（灵兽/灵地枯竭后寄生其中的蚀灵失去宿主四处流浪），主角与蚀灵共生本身就是"在衰退中求存"的隐喻。干预方式是"利用末法催生的蚀灵作为力量来源"，回应"灵气衰退后还有什么可用"的问题。力量与世界危机同源但分离。用户被"危机紧迫感"吸引→蚀灵饥饿反噬的压迫机制让每一阶都有生存压力，战斗对抗感极强。

→ **代价**：
  1. 反噬风险——蚀灵饥饿时会侵蚀宿主灵力甚至肉身，必须持续喂养，一旦断粮就面临反噬
  2. 非人化——共生越深，身体和精神越偏离人类：蚀纹蔓延全身、感官异变（夜视/兽性嗅觉）、情感淡漠，在散修社群中被视为异类
  3. 道德灰色——蚀灵本质是寄生掠夺，喂养蚀灵需要猎杀灵兽（甚至其他修士的灵宠），在散修中是异端行为，招致敌意和追杀

#### C) 碎阵铭纹道

→ **力量源**：阵纹残片——末法世界中古阵法崩裂后残留的阵纹碎片。每一枚阵纹残片都蕴含浓缩的灵力和法则规则，散落在废区、古战场、崩塌秘境中。主角学会辨识、收集并铭刻阵纹残片于自身，激活其中的力量。

→ **干预方式**：收集阵纹残片铭刻于体——不同阵纹组合产生不同效果（护体阵纹/增力阵纹/感知阵纹/攻击阵纹）。主角能对世界的矛盾做什么：用旧体系崩坏的遗产（阵纹残片）重组为自身力量，在正统功法垄断之外开辟"铭纹修炼"的第三条路。

→ **三境界落点**：低武→中武 — 质变点跨越：跨第一个大质变点

→ **大质变点事件**：阵图自衍——主角在一处崩塌的古大阵遗址中找到古大阵的核心残片，铭刻入体后，全身阵纹网络在核心残片的驱动下凝聚为完整的"内阵图"。阵图不再是被动铭刻的碎片集合，而是可以自主运转、自主衍化新阵纹的活体系统。这一刻，主角从"铭刻碎片获得单一阵效"（低武·量化强化区）跨越到"内阵图自主运转、外投布阵"（中武·规则干预区）。干预方式质变：从"铭刻固定碎片"变成"阵图自主衍化与外投"。

→ **子境界分层**（低武5阶+中武5阶=10阶）：

**低武·拾阵铭体篇**（量化强化区·收集铭刻为主）：

| 2 | **铭纹境** | 阵纹铭体，获得单一阵效（护体/增力/感知等） | →双纹共鸣：两枚阵纹共鸣，效果叠加质变（干预方式：单一阵效→复合阵效） |

→ **与世界的适配度**：高度适配——末法世界中阵法崩裂、阵纹残片遍地，正是衰退的"遗产"。干预方式是"收集别人看不见的阵纹碎片，重组为自身力量"，回应"旧体系崩坏后散修还能用什么"的问题。阵纹残片是末法的产物（阵法锈蚀/崩裂后残留），力量与世界危机同源但分离。用户被"危机紧迫感"吸引→纹蚀反噬风险让铭刻每枚阵纹都有代价博弈，战斗中阵纹组合的临场应变感强。

→ **代价**：
  1. 来源不稳定——阵纹残片靠运气收集，无法量产，强力的阵纹残片往往在极危险的废区深处
  2. 纹蚀风险——铭刻过多阵纹会侵蚀肉身（阵纹占用肉身"空间"），阵纹失控时反噬剧烈（阵纹冲突导致自爆）
  3. 悟性门槛——阵纹组合需要极高悟性，组合错误会"阵纹冲突"导致反噬甚至自爆；且不同阵纹残片来自不同古阵，体系不兼容，需要主角自行推演融合规则

## 验证总结

### 验证1：危机引擎三轴是否完整设计

**结论：通过。** 三个世界方案的危机引擎均完整设计了三轴：

| 世界 | 机遇轴（机制） | 危险轴（动力+机会+道义） | 咬合机制 |
|:--|:--|:--|:--|
| A) 末法残局 | 秩序崩塌型——仙门垄断瓦解，废区暴露 | 动力=灵枯倒退 机会=灵髓/弃徒/灵兽迁徙 道义=灰色 | 螺旋上升 |
| B) 灵潮枯竭 | 制度压迫型——世家垄断内战，防线缺口 | 动力=辐射区灵气降+灵兽暴动 机会=功法残篇/秘库/灵核 道义=中间带 | 周期型 |
| C) 天道锈蚀 | 宿命诅咒型——天道法则锈蚀，铁律失效 | 动力=法则崩解倒计时 机会=法则骨架/逆灵/漏劫者 道义=灰色 | 一体两面 |

三轴设计覆盖了框架定义的全部三种机遇机制（秩序崩塌/制度压迫/宿命诅咒）、全部三种道义光谱（灰色/中间带/灰色）、全部三种咬合形式（螺旋上升/周期型/一体两面）。

### 验证2：机遇-危险咬合是否成立

**结论：通过。** 三个世界的机遇-危险咬合均逻辑自洽：

- **A) 螺旋上升**：灵气衰退→仙门收缩（机遇扩大）+灵枯加速（危险加深）→主角攀升→触及仙门利益（危险升级）→需要更深废区（更高机遇）→灵气继续衰退。机遇与危险在同一个方向上同步扩大，互相驱动形成螺旋。✓
- **B) 周期型**：灵潮余波周期到来→灵气短暂充盈（机遇：可突破/采集灵材）+灵兽暴动/潮井争夺（危险：生存威胁）→周期间隔越来越长（持续恶化）→每次周期都需要更强力量应对。机遇与危险在同一个时间节点同时出现，周期型咬合。✓
- **C) 一体两面**：同一场天道锈蚀→法术失灵（危险）=法则骨架暴露（机遇）；天雷紊乱（危险）=漏劫者存活（机遇）；阵法失灵（危险）=封印松动（机遇）。机遇和危险是同一事件的正反两面，最高级咬合形式。✓

每个世界的"没有危机能上升吗"答案均为"不能"，满足框架硬约束。

### 验证3：质变点分级法是否正确使用（大质变点+小质变点）
