## S1 世界构筑

基于你的赛道（西幻）、标签（D&D/克苏鲁/中世纪奇幻/魔法/骑士）和世界偏好（奇幻世界 + IP世界 + 侵蚀型危机 + 全程三境界200万字+），提供3个世界方案：

### A) 深渊裂隙世界（Abyss Rift World）

→ 世界矛盾性质：**深渊侵蚀**——混沌深渊之力持续向上扩张，裂隙不断撕裂大陆，旧秩序在深渊的吞噬下瓦解

→ 表世界：一个受深渊裂隙侵蚀困扰的中世纪大陆。大陆沿海分布着诸王国，内陆逐渐被深渊裂隙占据。深渊前哨城是冒险者的聚集地，有黑市交易深渊晶石、佣兵酒馆接取猎杀委托、深渊学家研究裂隙规律。普通人在恐惧中度日，商队需要雇佣"裂隙守望者"护送穿越危险区，贵族在沿海安全区享乐但必须缴纳"深渊税"维持屏障。城市有教堂、行会、市集，空气中偶尔弥漫深渊的气息——铁锈味和低频嗡鸣。深渊晶石是硬通货，猎人的社会地位高于普通贵族。

→ 危机引擎：
  · **机遇轴**（秩序崩塌型）：深渊裂隙不断扩张，摧毁了旧王国的领土划分和封建秩序→深渊周边的无人区变成了冒险者的"深渊猎场"→深渊晶石（新资源）从裂隙中析出，深渊猎人组织（新体系）崛起，深渊前哨城（新组织）成为新势力中心→主角通过猎杀深渊生物获取深渊晶石提升实力，在无人区建立据点→**没有深渊扩张，内陆无人区不会开放，深渊晶石不会析出，旧秩序不会瓦解——主角无法上升**
  · **危险轴**：
    - 动力：不攀登就会被深渊吞噬——裂隙每年扩张数里，城市会沦陷，家人会变成深渊畸变体。深渊侵蚀不可逆，被侵蚀的人会逐渐异化
    - 机会：深渊晶石（新资源——可锻造抗侵蚀装备/提升力量）、裂隙守望者/深渊猎人公会（新体系——全新的职业体系和晋升通道）、深渊前哨城（新组织——不受旧王国管辖的自由地）
    - 道义：正道——对抗深渊扩张=保护人类，猎杀深渊生物=正义
  · **咬合机制**（螺旋上升）：深渊扩张逼主角猎杀深渊生物→获取深渊晶石提升实力→实力提升后探索更深的裂隙→发现更深裂隙连接着深渊源头→暴露更大的威胁（深渊领主正在苏醒）→需要更强的力量→更深入裂隙获取更稀有的深渊晶石……每次深入都既是危险升级也是机遇升级，螺旋攀升

→ 增量吸引力：**IP世界** — 借用D&D深渊（Abyss）设定认知基础。读者看到"深渊裂隙""深渊领主""深渊晶石"就能联想到D&D的深渊位面体系，无需额外学习设定。同时融入克苏鲁式的"不可名状的深渊低语"元素，双IP叠加

→ 表世界自检：规则自洽[✓] 生活质感[✓] 结构性矛盾[✓] 力量嵌入社会[✓]
  - 规则自洽闭环：深渊扩张→裂隙析出晶石→晶石可锻造抗侵蚀装备→抗侵蚀装备使人能深入裂隙→深入裂隙获取更多晶石。同时深渊税→维持屏障→屏障延缓扩张→税收不足则屏障衰减。子系统交叉规则，牵一发动全身
  - 生活质感：深渊前哨城的黑市、佣兵酒馆、深渊税、商队护送、晶石交易——读者能"住进去"
  - 结构性矛盾：沿海贵族vs深渊前哨城的权力争夺、深渊税的负担、猎人与普通人的阶层分化——不依赖主角就有张力
  - 力量嵌入社会：深渊晶石是经济硬通货，猎人等级决定社会地位，深渊税是财政支柱——力量体系嵌入经济/阶层/组织

→ 危机引擎自检：运作机制[✓] 不可逃避[✓] 多层次[✓] 持续恶化[✓] 渗透日常[✓] 机遇-危险咬合[✓]
  - 运作机制：深渊裂隙系统性扩张（有规律可研究，深渊学家在研究裂隙周期），非随机事件
  - 不可逃避：深渊扩张覆盖整个大陆，主角绕不过去，不猎杀就等死
  - 多层次展开：个人（被侵蚀的家人）→社区（前哨城的安全）→世界级（深渊领主苏醒）
  - 持续恶化（②侵蚀/污染扩散）：深渊裂隙每年扩张，不可逆恶化，主角不做事世界也在变糟——侵蚀型，坏东西（深渊）在扩大
  - 渗透日常：深渊税、深渊气息、商队护送、猎人职业——危机是生活底色
  - 机遇-危险咬合：深渊扩张=危险来源+机遇来源，螺旋上升

→ 增量吸引力：命中2个增量源（IP世界[D&D深渊+克苏鲁低语] + 经典类型框架[冒险者公会猎杀体系]）

→ 镜头换人检验：
  · 换主角A（深渊猎人·孤儿）+金手指（深渊感知——能感知裂隙深处的稀有晶石位置）→ 一个在深渊猎场中寻找珍稀晶石、逐渐揭开深渊源头秘密的猎人成长故事
  · 换主角B（落魄贵族子弟）+金手指（深渊契约——能与深渊生物签订契约借力）→ 一个利用深渊力量对抗深渊的贵族堕落与救赎故事，在权力与堕落间挣扎
  · 判定：**通过**——两个换主角的故事方向截然不同且都精彩，世界有同人创作空间

### B) 诡秘纪元世界（Eldritch Era World）

→ 世界矛盾性质：**诡秘侵蚀**——不可名状的真知正在从现实的裂缝中渗出，每一条被泄露的诡秘知识都在扭曲周围的现实，日常被诡秘一点点吞噬

→ 表世界：一个由"真理教会"统治的城邦联合体世界。教会垄断一切超自然知识，将诡秘真知列为最高禁忌。城市按"知识等级"分区——平民区禁止接触任何诡秘事物，学者区有有限的研究许可，核心区只有教会高层能进入。城市街角有"净化检查站"，空气中弥漫着说不清的不安。黑市流通被禁的诡秘手稿，隐修会秘密研究诡秘真知。普通人生活中偶尔会听到"低语"——那是诡秘真知在试图渗透。教堂的钟声每天定时响起，据说是"净化仪式"，但实际上只是延缓诡秘扩散。人们生活在"认知安全"的幻象中，不知道真相。

→ 危机引擎：
  · **机遇轴**（制度压迫型）：诡秘真知持续渗出，教会的知识垄断正在瓦解→旧垄断瓦解=知识从禁忌变为可获取→研究诡秘真知可以获得超自然力量（旧垄断瓦解+新维度开放）→主角通过研究被泄露的诡秘手稿获取力量，利用教会管控的缝隙上升→**没有诡秘渗出，教会垄断不会松动，诡秘知识不会泄露，主角无法获取力量——无法上升**
  · **危险轴**：
    - 动力：不研究诡秘就会被诡秘侵蚀吞噬——诡秘扩散不可逆，城市会逐渐被诡秘笼罩，不掌握对抗知识就会被低语逼疯。接触诡秘越多越危险但也越强
    - 机会：诡秘手稿（新资源——蕴含超自然力量的知识）、隐修会（新体系——全新的研究组织和晋升体系）、诡秘研究者网络（新组织——跨越城邦的地下网络）
    - 道义：灰色——利用诡秘=必要之恶，研究诡秘本身有被侵蚀的风险，但不研究就等死。"被逼无奈"有强力危险前置支撑（不研究=被诡秘吞噬）
  · **咬合机制**（一体两面）：诡秘真知既是力量来源也是侵蚀来源——越深入研究越强，但越接近疯狂。力量与疯狂一体两面，主角必须在力量与理智间走钢丝。每次获取力量都意味着更接近诡秘的真相，而真相本身会侵蚀认知

→ 增量吸引力：**IP世界** — 借用克苏鲁神话体系（不可名状之真知、知识即诅咒、理智值机制）认知基础。读者看到"诡秘真知""不可名状""低语""认知侵蚀"就能联想到克苏鲁神话，无需额外学习。同时借用D&D的"知识即力量"设定（密斯特拉的知识女神体系），双IP叠加

→ 表世界自检：规则自洽[✓] 生活质感[✓] 结构性矛盾[✓] 力量嵌入社会[✓]
  - 规则自洽闭环：诡秘渗出→教会垄断知识→垄断松动→知识泄露→研究者获取力量→力量增强→接触更深诡秘→更深诡秘加速渗出。同时净化仪式→延缓扩散→但消耗教会资源→资源不足则净化减弱→扩散加速。子系统交叉规则
  - 生活质感：知识等级分区、净化检查站、黑市手稿交易、低语侵扰、教堂钟声净化仪式——读者能"住进去"
  - 结构性矛盾：教会垄断vs学术自由、平民vs学者的知识鸿沟、安全区的阶层分化——不依赖主角就有张力
  - 力量嵌入社会：知识等级决定社会地位，隐修会研究等级=权力等级，诡秘手稿是地下硬通货

→ 危机引擎自检：运作机制[✓] 不可逃避[✓] 多层次[✓] 持续恶化[✓] 渗透日常[✓] 机遇-危险咬合[✓]
  - 运作机制：诡秘真知系统性渗出（有扩散规律，隐修会在研究扩散周期），非随机事件
  - 不可逃避：诡秘扩散覆盖所有城邦，主角绕不过去，不研究就等死
  - 多层次展开：个人（低语侵扰）→社区（城邦的净化体系崩溃）→世界级（不可名状的存在正在苏醒）
  - 持续恶化（②侵蚀/污染扩散）：诡秘真知持续渗出扩散，不可逆恶化——侵蚀型，坏东西（诡秘真知）在扩大
  - 渗透日常：低语、净化检查站、知识等级分区、手稿黑市——危机是生活底色
  - 机遇-危险咬合：诡秘真知一体两面=力量来源+侵蚀来源，最高级咬合

→ 增量吸引力：命中2个增量源（IP世界[克苏鲁+D&D知识体系] + 经典类型框架[禁忌知识研究/秘密组织体系]）

→ 镜头换人检验：
  · 换主角A（教会异端审判官）+金手指（诡秘免疫——能安全接触诡秘不被侵蚀）→ 一个能安全研究诡秘的审判官逐渐发现教会本身就是诡秘侵蚀源的故事
  · 换主角B（黑市手稿贩子）+金手指（诡秘翻译——能解读任何诡秘手稿）→ 一个贩卖禁忌知识的商人逐渐被诡秘真知改变、在利益与疯狂间挣扎的故事
  · 判定：**通过**——两个换主角的故事方向截然不同且都精彩，世界有同人创作空间

### C) 枯萎诅咒世界（Withering Curse World）

→ 世界矛盾性质：**腐化侵蚀**——一种源自远古黑魔法的腐化诅咒正在从大陆中心向外扩散，所到之处生命枯萎、大地腐朽、魔法扭曲，腐化区域不可逆地扩大

→ 表世界：一个魔法王国的封建大陆，腐化诅咒从大陆中心的"陨魔坑"向外扩散，形成不断扩大的"腐化荒原"。王国领主们获得国王授予的"净化权"来开疆拓土——净化腐化区域可获得封地。净化者行会是受尊敬的职业组织，有严格的等级制度。边境净化城是前沿阵地，有净化结晶交易市场、净化者训练场、腐化监测站。普通人生活在"安全区"内，靠净化者维持的"圣光屏障"隔绝腐化。安全区有农田、市集、教堂，但每个人都知道腐化在逼近——田地边缘有时会出现枯黄的草，那是腐化的前兆。贵族争抢净化权，净化者行会内部有派系斗争，普通人在恐惧中耕作。

→ 危机引擎：
  · **机遇轴**（宿命诅咒型）：腐化诅咒周期性加速扩散，每次扩散波都带来新的应对需求→新需求出现=净化权授予=新的封地和权力→净化腐化区域可获得净化结晶（新资源）和封地（新权力）→主角通过净化腐化区域获取结晶和封地，建立自己的领地→**没有腐化扩散，国王不会授予净化权，腐化区域不会开放，净化结晶不会产生——主角无法上升**
  · **危险轴**：
    - 动力：不净化腐化就会扩散到自己的领地——腐化会侵蚀魔法本身，导致圣光屏障失效，屏障失效则安全区沦陷。腐化扩散波每隔数年加速一次，不做好准备就会被吞噬
    - 机会：净化结晶（新资源——可强化圣光/锻造抗腐化装备/提升魔法）、净化者行会（新体系——全新的职业晋升通道）、边境净化城（新组织——不受旧贵族管辖的前沿势力）
    - 道义：正道——净化腐化=保护世界，对抗诅咒=正义
  · **咬合机制**（周期型）：腐化诅咒每隔数年爆发一次扩散波，每次扩散波既是威胁（腐化区域扩大、屏障受压）也是成长契机（新的腐化区域=新的净化机会=新的结晶和封地）。天然适合超长篇——每次扩散波都是一个完整的叙事周期

→ 增量吸引力：**IP世界** — 借用D&D的黑魔法腐化（Darklands/Shadowfell的腐化设定）和魔兽世界的枯萎/腐化区域设定认知基础。读者看到"腐化荒原""净化结晶""圣光屏障"就能联想到D&D的腐化区域和WoW的瘟疫之地。同时借用经典奇幻的"领主开疆"框架

→ 表世界自检：规则自洽[✓] 生活质感[✓] 结构性矛盾[✓] 力量嵌入社会[✓]
  - 规则自洽闭环：腐化扩散→国王授予净化权→净化者净化→获取净化结晶→结晶强化屏障→屏障延缓扩散→但腐化加速→屏障受压→需要更多净化者→扩大净化权授予。同时净化结晶=经济/军事硬通货→结晶争夺→行会内部派系斗争→影响净化效率。子系统交叉规则
